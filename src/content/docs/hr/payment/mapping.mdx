Ako ste integrator, poput Winka, i želite mapirati svoje vanjske primatelje plaćanja za korištenje s TripPayom, to možete učiniti na jedan od dva načina:

* Ručno mapiranje
* Korištenje[API za plaćanje](/developers/apis/#payment-api)(*Vidi`Account Mappings`*)

:::tip
Osim ako se ne bavite samo s nekoliko istih svojstava, preporučujemo da koristite API za automatizaciju ovog koraka. :::

U nastavku vas vodimo kroz ručni postupak mapiranja vanjskog primatelja plaćanja.*Koristi API ispod haube.*

<Steps>
  1. Prijavi se [TripPay](https://pay.wink.travel).
  2. Odaberite svoj integracijski račun.
  3. Klik `Settings > Mappings` iz glavne navigacijske trake.
  4. Kliknite na `Create mapping` gumb. Vidjet ćete obrazac za mapiranje.
  5. **Lokalni račun** Počnite tipkati naziv postojećeg TripPaya koji želite mapirati.
  6. **Vanjski identifikator** Zalijepite svoj vanjski ID za ovaj entitet onako kako postoji u udaljenom sustavu.
  7. Klik `Save` za nastavak. Bit ćete preusmjereni natrag na svoj popis mapiranja.
</Steps>


S vašim postavljenim mapiranjima, TripPay sada zna koje račune treba pripisati kao korisnike rezervacije kada ih navedete pomoću svojih vanjskih identifikatora.

## API

Razvojni programeri koji žele upravljati`Payments`, idi na[Razvojni programeri > API > Plaćanja](/developers/apis/#payment-api).
