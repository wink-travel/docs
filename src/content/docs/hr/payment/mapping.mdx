Ako ste integrator, poput Wink, i želite mapirati svoje vanjske primatelje za korištenje s TripPay, to možete učiniti na jedan od dva načina:

- Ručno mapiranje
- Korištenje [Payment API](/hr/developers/apis/#payment-api) (*Pogledajte `Account Mappings`*)

:::tip
Osim ako ne radite samo s nekoliko istih objekata, preporučujemo da koristite API za automatizaciju ovog koraka.
:::

Ispod vas vodimo kroz ručni način mapiranja vanjskog primatelja. *Ispod haube koristi API.*

<Steps>
1. Prijavite se u [TripPay](https://pay.wink.travel).
2. Odaberite svoj integratorski račun.
3. Kliknite `Settings > Mappings` u glavnoj navigacijskoj traci.
4. Kliknite gumb `Create mapping`. Vidjet ćete obrazac za mapiranje.
5. **Local account** Počnite upisivati naziv postojećeg TripPay računa koji želite mapirati.
6. **External identifier** Zalijepite svoj vanjski ID za ovu entitetu kako postoji u udaljenom sustavu.
7. Kliknite `Save` za nastavak. Bit ćete preusmjereni natrag na svoj popis mapiranja.
</Steps>

S mapiranjima na mjestu, TripPay sada zna koje račune treba kreditirati kao korisnike rezervacije kada ih spominjete koristeći svoje vanjske identifikatore.

## API

Razvojni programeri koji žele upravljati `Payments`, idu na [Developers > API > Payments](/hr/developers/apis/#payment-api).
