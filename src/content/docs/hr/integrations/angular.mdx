Wink komponente same po sebi su Angular Elements — prirodno se integriraju u Angular aplikacije. Jedino što je potrebno je reći Angularovom kompajleru da dopušta nepoznate nazive elemenata.

## Instalacija

```bash
npm install @wink/elements
```

## Bootstrap — učitaj CDN jednom

Pozovite `load()` prije `bootstrapApplication` (ili unutar `APP_INITIALIZER`):

```ts
// src/main.ts
import { bootstrapApplication } from '@angular/platform-browser';
import { AppComponent } from './portal/app.component';
import { load } from '@wink/elements';

load({ clientId: import.meta.env['NG_APP_WINK_CLIENT_ID'] });

bootstrapApplication(AppComponent).catch(console.error);
```

## Dopuštanje naziva prilagođenih elemenata

Dodajte `CUSTOM_ELEMENTS_SCHEMA` svakom standalone komponentom (ili modulu) koji renderira `<wink-*>` oznake:

```ts
// src/portal/hotels/hotels.component.ts
import { Component, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

@Component({
  selector: 'app-hotels',
  standalone: true,
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
  template: `
    <wink-content-loader layout="HOTEL" [id]="layoutId" />
    <wink-lookup />
  `,
})
export class HotelsComponent {
  layoutId = 'YOUR_LAYOUT_ID';
}
```

<Aside type="tip" title="NgModule aplikacije">
Za aplikacije temeljene na NgModule, umjesto toga dodajte `CUSTOM_ELEMENTS_SCHEMA` u niz `schemas` dekoratora `@NgModule`.
</Aside>

## Povezivanje atributa

Angularova sintaksa predložaka radi izravno s atributima prilagođenih elemenata kao stringovima:

```html
<!-- Statički -->
<wink-content-loader layout="HOTEL" id="abc123" />

<!-- Dinamičko povezivanje -->
<wink-content-loader [attr.layout]="layout" [attr.id]="layoutId" />
```

Koristite `[attr.name]` povezivanje kada je vrijednost dinamična — Angularovo uobičajeno `[property]` povezivanje cilja DOM svojstva, a ne HTML atribute, na kojima se prilagođeni elementi oslanjaju.

## Varijable okoline

Za `@angular/cli` 16+ projekte, koristite `environment.ts` ili prefiks `NG_APP_*` s `@ngx-env/builder`:

```ts
// src/environments/environment.ts
export const environment = {
  winkClientId: 'YOUR_CLIENT_ID',
};
```

```ts
// src/main.ts
import { environment } from './environments/environment';
load({ clientId: environment.winkClientId });
```

## TypeScript tipovi

`@wink/elements` paket izlaže tipizirane sučelja za sve atribute komponenti:

```ts
import type { WinkContentLoaderAttributes, WinkLayout } from '@wink/elements';

const attrs: WinkContentLoaderAttributes = {
  layout: 'HOTEL' as WinkLayout,
  id: 'abc123',
};
```
