**Wink Drupal modul** dodaje Wink travel inventarne komponente na vašu Drupal stranicu kao nativne blokove. Urednici ih mogu postaviti bilo gdje u rasporedu putem standardnog sustava blokova.

<LinkCard
  title="wink-travel/wink-drupal-module na GitHubu"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Izvorni kod, vodič za postavljanje za developere i upute za doprinos."
/>

## Zahtjevi

- Drupal 10 ili 11
- PHP 8.1+
- Composer

## Instalacija

<Steps>
1. Instalirajte putem Composera:
   ```bash
   composer require drupal/wink
   ```
2. Omogućite modul:
   ```bash
   drush en wink -y
   ```
3. Idite na **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Unesite svoj **Client ID** i **Client Secret**, odaberite svoje okruženje i kliknite **Save**.
   Posjetite [Applications](/hr/portal/settings/applications) za dohvat svojih vjerodajnica.
</Steps>

## Dodavanje bloka na stranicu

<Steps>
1. Idite na **Admin → Structure → Block Layout**.
2. Kliknite **Place block** u regiji gdje želite Wink sadržaj.
3. Pretražite **Wink** da pronađete dostupne blokove:
   - **Wink Content** — prikazuje karticu hotela, mrežu ili kartu
   - **Wink Lookup** — traka za pretraživanje destinacija
   - **Wink Search** — gumb za odabir itinerera
   - **Wink Account** — gumb za prijavu
   - **Wink Itinerary** — gumb za status itinerera
   - **Wink Cart** — gumb za košaricu
4. Za **Wink Content**, konfigurirajte **Layout Type** i **Layout ID** u postavkama bloka.
5. Spremite konfiguraciju bloka.
</Steps>

## Dostupni blokovi

| Blok | Namjena |
|---|---|
| **Wink Content** | Prikazuje kartice hotela, rangirane mreže, karte i više |
| **Wink Lookup** | Traka za pretraživanje destinacija i hotela |
| **Wink Search** | Ikona gumba koja otvara odabir itinerera |
| **Wink Account** | Gumb za prijavu / korisnički račun |
| **Wink Itinerary** | Gumb itinerera koji prikazuje trenutni put |
| **Wink Cart** | Gumb sa sažetkom košarice |
