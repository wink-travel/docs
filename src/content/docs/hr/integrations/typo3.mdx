**Wink TYPO3 Extension** integrira Wink travel inventar u TYPO3 13 kao nativne sadržajne elemente. Urednici dodaju Wink komponente na stranice koristeći standardni TYPO3 uređivač stranica — nije potrebna promjena predložaka.

## Zahtjevi

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instalacija

<Steps>
1. Instalirajte putem Composera:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivirajte ekstenziju:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. U TYPO3 backendu idite na **Admin Tools → Settings → Extension Configuration → wink**.
4. Unesite svoj **Client ID**, **Client Secret** i odaberite svoje okruženje. Spremite.
   Posjetite [Applications](/hr/portal/settings/applications) za dohvat vaših vjerodajnica.
5. Dodajte statički TypoScript predložak u svoj root predložak:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Dodavanje sadržajnog elementa na stranicu

<Steps>
1. Otvorite stranicu u modulu **Page**.
2. Kliknite na gumb **+** u području sadržaja za dodavanje novog elementa.
3. Odaberite jedan od **Wink Travel** sadržajnih elemenata:
   - **Wink Content** — hotelske kartice, rangirane mreže, karte
   - **Wink Lookup** — traka za pretraživanje destinacija
   - **Wink Search** — gumb za odabir itinerera
   - **Wink Account** — gumb za prijavu
   - **Wink Itinerary** — gumb za status itinerera
   - **Wink Cart** — gumb za košaricu
4. Za **Wink Content**, koristite FlexForm polja za odabir **Layout Type** i unos **Layout ID**.
5. Spremite i objavite stranicu.
</Steps>

## Dostupni sadržajni elementi

| Element | Namjena |
|---|---|
| **Wink Content** | Hotelske kartice, rangirane mreže, karte i još mnogo toga |
| **Wink Lookup** | Traka za pretraživanje destinacija |
| **Wink Search** | Ikona gumba koja otvara odabir itinerera |
| **Wink Account** | Gumb za prijavu / korisnički račun |
| **Wink Itinerary** | Gumb itinerera koji prikazuje trenutni put |
| **Wink Cart** | Gumb sa sažetkom košarice |
