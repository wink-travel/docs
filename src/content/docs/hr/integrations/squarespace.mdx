:::note
Značajka Code Injection u Squarespaceu dostupna je na **Business planu i višim**. Prilagođeni HTML blokovi dostupni su na svim plaćenim planovima.
:::

Squarespace podržava prilagođeni HTML i JavaScript, što znači da možete ugraditi bilo koji Wink web komponentu izravno na svoju stranicu bez dodatka. Ovaj vodič pokazuje kako učitati Wink skripte na cijeloj stranici, a zatim postaviti komponente na pojedinačne stranice.

## Učitaj Wink skripte na cijeloj stranici

Dodavanjem skripti u globalni footer i header vaše stranice, Wink je dostupan na svakoj stranici bez ponavljanja postavki.

<Steps>
1. U Squarespace editoru idite na **Settings → Advanced → Code Injection**.
2. U polje **Header** dodajte stilsku datoteku:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. U polje **Footer** dodajte skriptu i učitavač aplikacije (zamijenite `YOUR_CLIENT_ID` sa svojim Wink Client ID-jem):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Kliknite **Save**.
</Steps>

:::tip
Vaš Client ID dostupan je u vašem Wink računu pod [Applications](/hr/portal/settings/applications).
:::

## Dodajte komponentu na stranicu

Nakon što su skripte učitane globalno, možete dodati bilo koju Wink komponentu na bilo koju stranicu koristeći **Code Block**.

<Steps>
1. Otvorite stranicu koju želite uređivati.
2. Kliknite **Edit** i dodajte novi **Code** blok na mjesto gdje želite da se komponenta prikaže.
3. Zalijepite HTML komponente, na primjer content loader:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Kliknite **Apply** zatim **Save**.
</Steps>

## Dostupne komponente

| Komponenta | HTML oznaka | Namjena |
|---|---|---|
| Content loader | `<wink-content-loader>` | Prikaz hotelskih kartica, mreža, karata |
| Lookup | `<wink-lookup>` | Tražilica destinacija |
| Search button | `<wink-search-button>` | Otvori odabir itinerera |
| Account button | `<wink-account-button>` | Prijava / korisnički izbornik |
| Itinerary button | `<wink-itinerary-button>` | Prikaži trenutni itinerer |
| Itinerary picker | `<wink-itinerary-picker>` | Cijeli obrazac itinerera |
| Shopping cart | `<wink-shopping-cart-button>` | Gumb sažetka košarice |

Pogledajte [Web Components](/hr/integrations/web-components) za potpuni popis atributa za svaku komponentu.
