Ubrzajte vrijeme izlaska na tržište koristeći jedan od naših službeno podržanih jezičnih Software Development Kitova.
Ako trenutno ne podržavamo vaš omiljeni jezik, idite na API koji želite koristiti i preuzmite OpenAPI specifikaciju koju nudimo za svaki API na vrhu stranice. Možete generirati vlastiti SDK bilo forkanjem jednog od naših javnih SDK repozitorija ili korištenjem jednog od OpenAPI generatora.

## Java

Ova biblioteka je kompajlirana za Java 24. Konfigurirana je za korištenje reaktivnog WebClienta i optimizirana za upotrebu sa Spring Frameworkom. Međutim, Spring nije obavezan, ali vam pruža OAuth2 podršku odmah iz kutije.
