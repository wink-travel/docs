Ova stranica prikazuje svaki webhook događaj na koji se možete pretplatiti na Wink-u. Za upute o postavljanju, pogledajte [Webhook Integration Guide](/hr/guides/webhook-integration) ili [Webhooks management page](/hr/my-account/webhooks).

## Kako se pretplatiti

1. Idite na **Applications > Webhooks** u vašem portalu.
2. Kreirajte webhook i odaberite događaje koje želite primati.
3. Wink šalje HTTP POST na vašu krajnju točku svaki put kada se događaj dogodi.

Za potpune korake postavljanja i najbolje prakse, pogledajte [Webhook Integration](/hr/guides/webhook-integration).

## Događaji računa

Događaji vezani uz životni ciklus i postavke vašeg TripPay računa.

| Događaj | Ključ | Opis |
|---|---|---|
| Račun kreiran | `account.create` | Novi račun je kreiran. |
| Račun na pregledu | `account.under_review` | Račun je na čekanju za pregled. |
| Račun ažuriran | `account.updated` | Podaci računa su promijenjeni. |
| Status računa ažuriran | `account.status.updated` | Status računa je promijenjen (npr. odobren, suspendiran). |
| Adresa računa ažurirana | `account.address.updated` | Adresa računa je izmijenjena. |
| Detalji računa ažurirani | `account.details.updated` | Polja detalja računa su promijenjena. |
| Informacije o računu ažurirane | `account.information.update` | Opće informacije o računu su ažurirane. |
| Upravljatelji računa ažurirani | `account.managers.update` | Popis upravitelja računa je promijenjen. |
| Profilna slika ažurirana | `account.profile_picture.updated` | Profilna slika računa je promijenjena. |
| Agent ažuriran | `account.agent.updated` | Informacije o agentu na računu su izmijenjene. |
| URL ime ažurirano | `account.url-name.update` | Vanity URL računa je promijenjen. |
| Jedinstveni ID ažuriran | `account.unique-id.update` | Jedinstveni identifikator računa je izmijenjen. |

### Zadaci računa

| Događaj | Ključ | Opis |
|---|---|---|
| Zadatak kreiran | `account.task.create` | Novi zadatak je dodan na račun. |
| Zadatak ažuriran | `account.task.update` | Zadatak na računu je izmijenjen. |
| Zadatak uklonjen | `account.task.remove` | Zadatak na računu je uklonjen. |
| Zadatak dovršen | `account.task.complete` | Zadatak na računu je dovršen. |
| Zadatak verifikacije poreznog ID-a dovršen | `account.task.tax_id.complete` | Zadatak verifikacije poreznog ID-a je dovršen. |

### Primalac računa

| Događaj | Ključ | Opis |
|---|---|---|
| Primalac kreiran | `account.acquirer.created` | Novi primalac je dodan na račun. |
| Primalac ažuriran | `account.acquirer.updated` | Primalac računa je izmijenjen. |
| Primalac uklonjen | `account.acquirer.removed` | Primalac je uklonjen s računa. |

## Događaji ugovora o rezervaciji

| Neuspjeh ugovora | `booking.contract.failed` | Plaćanje ugovora o rezervaciji nije uspjelo. |
| Ugovor otkazan | `booking.contract.cancelled` | Ugovor o rezervaciji je otkazan. |
| Djelomični povrat | `booking.contract.refund.partial` | Izdana je djelomična povratna uplata na ugovor. |
| Potpuni povrat | `booking.contract.refund.full` | Izdana je potpuna povratna uplata na ugovor. |
| Zahtjev za povrat | `booking.contract.refund.request` | Podnesen je zahtjev za povrat. |

## Događaji upravljačkog subjekta

Događaji za upravljačke subjekte (objekte, partnere, tvrtke) i njihovu administraciju.

### Pretplata

| Događaj | Ključ | Opis |
|---|---|---|
| Pretplata osigurana | `managing_entity.subscription.provisioned` | Nova pretplata je osigurana. |
| Pretplata promijenjena | `managing_entity.subscription.changed` | Plan pretplate je promijenjen. |
| Račun pretplate promijenjen | `managing_entity.subscription.invoice.changed` | Račun pretplate je ažuriran. |

### Status i administracija

| Događaj | Ključ | Opis |
|---|---|---|
| Status ažuriran | `managing_entity.status.updated` | Status subjekta je promijenjen. |
| Subjekt izbrisan | `managing_entity.deleted` | Upravljački subjekt je izbrisan. |
| Agent ažuriran | `managing_entity.agent.updated` | Agent subjekta je izmijenjen. |
| Email obavijest agentu | `managing_entity.agent.email.notification` | Poslana je email obavijest agentu. |
| Email obavijest upravitelju | `managing_entity.manager.email.notification` | Poslana je email obavijest upravitelju. |

### Životni ciklus upravitelja

| Događaj | Ključ | Opis |
|---|---|---|
| Upravitelj dodan | `managing_entity.manager.added` | Novi upravitelj je dodan. |

| Upravitelj uklonjen | `managing_entity.manager.removed` | Upravitelj je uklonjen. |
| Upravitelj prihvaćen | `managing_entity.manager.accepted` | Pozivnica upravitelju je prihvaćena. |
| Upravitelj odbijen | `managing_entity.manager.rejected` | Pozivnica upravitelju je odbijena. |
