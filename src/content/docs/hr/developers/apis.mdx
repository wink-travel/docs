Taxonomy krajnje točke su za programere koji žele konzumirati i proizvoditi turistički inventar i trebaju taksonomije standardnih i nestandardnih kodova za vrste inventara, klase, statuse itd.

</LinkButton>

## Analytics API

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
Saznajte više

## Payment API

Payment krajnje točke su za programere koji žele kupiti turistički inventar. To se može učiniti putem API-ja kao registrirani Travel Agent ili koristeći naš API u kombinaciji s našim PCI usklađenim widgetom za plaćanje za sve ostale entitete.
Payment API izlaže krajnje točke za partnere i hotele za praćenje rezervacija, analitike i dostupnosti sredstava. Također mogu odabrati povlačenje dostupnih sredstava na svoj bankovni račun.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Saznajte više
</LinkButton>
