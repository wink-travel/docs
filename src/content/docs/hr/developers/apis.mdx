## API za plaćanje

Krajnje točke plaćanja namijenjene su programerima koji žele kupiti putničku opremu. To se može učiniti putem API-ja kao registrirani putnički agent ili korištenjem našeg API-ja u kombinaciji s našim PCI kompatibilnim widgetom za plaćanje za sve ostale subjekte.
API za plaćanje pruža krajnje točke partnerima i hotelima za praćenje rezervacija, analitike i dostupnosti sredstava. Također mogu odabrati podizanje dostupnih sredstava na svoj bankovni račun.

  Saznajte više
</LinkButton>
