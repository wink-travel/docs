Wink se integrira s vodećim channel managerima, sustavima za upravljanje nekretninama (PMS) i centralnim rezervacijskim sustavima (CRS) kako bi vaše cijene i dostupnost bile sinkronizirane na svim vašim prodajnim kanalima.

## Kako to funkcionira

Integracije channel managera omogućuju dvosmjerni protok podataka između vašeg sustava za upravljanje nekretninama i Winka:

1. **Slanje cijena** — Vaš channel manager šalje ažuriranja cijena Winku. Kada ažurirate cijene u svom PMS-u, te promjene se automatski prikazuju na Winku.
2. **Sinkronizacija dostupnosti** — Vaš channel manager šalje ažuriranja dostupnosti Winku. Kada se sobe rezerviraju ili oslobode u vašem PMS-u, zalihe na Winku se ažuriraju u stvarnom vremenu.
3. **Obavijesti o rezervacijama** — Kada se rezervacija izvrši na Winku, vaš channel manager biva obaviješten kako bi se rezervacija prikazala u vašem PMS-u.

## Podržani channel manageri

Wink se trenutno integrira sa sljedećim sustavima:

| Channel Manager | Opis |
|---|---|
| **Channex** | Globalna platforma za distribuciju i upravljanje kanalima. |
| **TravelClick** | Hotelska rješenja i upravljanje kanalima (Amadeus). |
| **Omnibees** | Distribucija i upravljanje prihodima za hotele. |
| **D-Edge** | Tehnologija za ugostiteljstvo i centralna rezervacijska rješenja. |
| **Allotz** | Alokacija inventara i upravljanje kanalima. |
| **Ezee / Yanolja** | Cloud PMS i upravljanje kanalima fokusirano na azijsko tržište. |
| **RateGain** | Tehnologija za inteligenciju cijena i distribuciju. |
| **RateTiger** | Distribucija cijena i dostupnosti u stvarnom vremenu. |
| **CloudBeds** | Cloud-based PMS, channel manager i booking engine. |
| **SiteMinder** | Vodeći hotelski channel manager i platforma za distribuciju. |
| **Exely** | Sinkronizacija cijena i dostupnosti za hotele. |

:::note
Ako vaš channel manager nije na popisu, obratite se Wink timu. Nove integracije mogu se dodati na temelju OTA (Open Travel Alliance) protokol standarda koje Wink podržava.
:::

## Za developere

Ako razvijate integraciju channel managera ili PMS-a s Winkom, koristite Channel Manager API endpointe:

- **Ažuriranja cijena** — Pošaljite promjene cijena Winku.
- **Ažuriranja dostupnosti** — Pošaljite promjene dostupnosti Winku.
- **Preuzimanje rezervacija** — Povucite rezervacije iz Winka u svoj sustav.

Channel Manager API slijedi OTA protokol standarde (SOAP/XML) za kompatibilnost s postojećim ugostiteljskim sustavima.

## Webhook obavijesti

Možete se pretplatiti na webhook događaje channel managera kako biste primali obavijesti u stvarnom vremenu:

- `channel-manager.update.rate` — Primljeno ažuriranje cijena.
- `channel-manager.update.availability` — Primljeno ažuriranje dostupnosti.
- `channel-manager.update` — Opće ažuriranje channel managera.

Pogledajte [Katalog Webhook Događaja](/hr/developers/webhook-events#channel-manager-events) za detalje.
