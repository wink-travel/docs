Booking engine dolazi u dvije varijante kako bi podržao različite potrebe affiliate partnera, tvrtke za upravljanje nekretninama i hotela.

## Online putnička agencija

Za affiliate partnera najlogičnije je koristiti verziju online putničke agencije. Razlog je što je stranica postavljena da zadovolji putnike koji žele pregledavati više nekretnina.

Početnu stranicu ove aplikacije možete pronaći u nastavku.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Internet booking engine

Za hotel ili tvrtku za upravljanje nekretninama najlogičnije je koristiti verziju internet booking enginea. Ova vrsta korisnika želi istaknuti jednu nekretninu ili više nekretnina pod upravljanjem.

Početnu stranicu ove aplikacije možete pronaći u nastavku.

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">

</LinkButton>

## Konfiguracija

Većinu vremena nema potrebe za direktnim dijeljenjem bilo koje od ovih aplikacija. Kreirali smo različite načine za dijeljenje inventara putovanja, koje možete otkriti u [Wink Studio](/studio/what-is-studio) ili putem [WinkLinks](/link-manager/wink-links).

Međutim, ako želite ručno povezati jednu od ovih web stranica, obavezno dodajte svoj client ID i customization ID u URL kao parametre.

Svoj client ID možete pronaći na sljedeći način:

<Steps>
1. Prijavite se u [Wink Extranet](/extranet/what-is-extranet) ili [Wink Studio](/studio/what-is-studio).
2. Kliknite na ikonu svog profila u gornjem desnom kutu i kliknite na [Applications](/developers/applications).
3. Koristite aplikaciju koja je povezana s Wink računom s kojim želite raditi.
4. Kopirajte Client ID iz te aplikacije i zalijepite ga u URL kao parametar zahtjeva. Primjer: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Svoj customization ID kao nekretninu možete pronaći na sljedeći način:

<Steps>
1. Prijavite se u [Wink Extranet](/extranet/what-is-extranet).
2. Odaberite nekretninu s kojom želite raditi.
3. Iz glavne navigacijske trake kliknite `Account > Branding`
4. Kopirajte Customization ID i zalijepite ga u URL kao parametar zahtjeva. Primjer: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Svoj customization ID kao affiliate partner možete pronaći na sljedeći način:

<Steps>
1. Prijavite se u [Wink Studio](/studio/what-is-studio).
2. Odaberite račun s kojim želite raditi.
3. Iz glavne navigacijske trake kliknite `Tools > Customization`
4. Kopirajte Customization ID iz prilagodbe koju želite koristiti i zalijepite ga u URL kao parametar zahtjeva. Primjer: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Dobit ćete URL koji izgleda otprilike ovako:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Ako želite ići direktno na stranicu hotela, URL bi izgledao otprilike ovako:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Ako ne uključite client i/ili configuration ID, Wink OTA će koristiti zadanu konfiguraciju.
:::

## IBE

Ako ste tvrtka za upravljanje nekretninama, želite povezati na početnu stranicu Wink IBE. Početna stranica će prikazati vašu dodijeljenu temu zajedno s [nekretninama koje predstavljate](/studio/agency-representation).

Dobit ćete URL koji izgleda otprilike ovako:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Ako ste nekretnina, želite se povezati direktno na početnu stranicu nekretnine u Wink IBE.

Taj URL bi izgledao otprilike ovako:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
Ako ne uključite client i/ili configuration ID, Wink IBE se NEĆE učitati. Ovi parametri su obavezni.
:::
