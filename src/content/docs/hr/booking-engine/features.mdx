Rezervacijski sustav dolazi u dvije varijante kako bi podržao različite potrebe partnera, tvrtke za upravljanje nekretninama i hotela.

Za partnerski program, najlogičnije je koristiti online verziju putničke agencije. Razlog tome je što je stranica postavljena kako bi se prilagodila putnicima koji žele pregledati više nekretnina.

Početna stranica ove aplikacije može se pronaći u nastavku.

## Konfiguracija

Većinom vremena nema potrebe izravno dijeliti bilo koju od ovih aplikacija. Osmislili smo razne načine za dijeljenje putnog inventara, koje možete otkriti u[Wink Studio](/studio/what-is-studio)ili kroz[WinkLinks](/link-manager/wink-links).

Međutim, ako želite ručno povezati jednu od ovih web-lokacija, obavezno dodajte svoj ID klijenta i ID prilagodbe u URL kao parametre.

Svoj ID klijenta možete pronaći na sljedeći način:

<Steps>
  1. Prijavite se u bilo koji od [Wink Extranet](/extranet/what-is-extranet) ili [Wink Studio](/studio/what-is-studio).
  2. Kliknite na ikonu svog profila u gornjem desnom kutu i kliknite na [Primjene](/developers/applications).
  3. Koristite aplikaciju koja je povezana s Wink računom s kojim želite raditi.
  4. Kopirajte ID klijenta iz te aplikacije i zalijepite ga u URL kao parametar zahtjeva. Primjer: ?client-id=MY\_CLIENT\_ID
</Steps>


### Ekstranet

Svoj ID prilagodbe kao svojstvo možete pronaći na sljedeći način:

<Steps>
  1. Prijavi se [Wink Extranet](/extranet/what-is-extranet).
  2. Odaberite svojstvo s kojim želite raditi.
  3. Na glavnoj navigacijskoj traci kliknite `Account > Branding`
  4. Kopirajte ID prilagodbe i zalijepite ga u URL kao parametar zahtjeva. Primjer: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Studio

Svoj ID prilagodbe kao pridruženi član možete pronaći na sljedeći način:

<Steps>
  1. Prijavi se [Wink Studio](/studio/what-is-studio).
  2. Odaberite račun s kojim želite raditi.
  3. Na glavnoj navigacijskoj traci kliknite `Tools > Customization`
  4. Kopirajte ID prilagodbe iz prilagodbe koju želite koristiti i zalijepite ga na URL kao parametar zahtjeva. Primjer: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

Ono što na kraju dobijete je URL koji izgleda otprilike ovako:

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Ako želite ići izravno na stranicu hotela, URL bi izgledao ovako:

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::napomena
Ako ne uključite ID klijenta i/ili konfiguracije, Wink OTA će se prema zadanim postavkama vratiti na zadanu konfiguraciju.
:::

## IBE

Ako ste tvrtka za upravljanje nekretninama, želite se povezati s početnom stranicom Wink IBE. Na početnoj stranici prikazat će se vaša odabrana tema zajedno s[nekretnine koje predstavljate](/studio/agency-representation).

Ono što na kraju dobijete je URL koji izgleda otprilike ovako:

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Ako ste vlasnik nekretnine, želite se izravno povezati s odredišnom stranicom nekretnine Wink IBE.

Taj URL bi izgledao otprilike ovako:

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::napomena
Ako ne uključite ID klijenta i/ili konfiguracije, Wink IBE se NEĆE učitati. Ovi parametri su obavezni.
:::
