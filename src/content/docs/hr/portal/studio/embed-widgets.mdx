Studio pruža nekoliko ugradivih widgeta koje možete dodati na svoju web stranicu kako biste potaknuli rezervacije.

Za pristup ugradivim widgetima, idite na `Tools > Miscellaneous` u bočnoj traci.

## Dostupni widgeti

| Widget | Opis |
| -- | -- |
| **Account button** | Brendirani gumb koji vodi na vašu partnersku stranicu |
| **Itinerary button** | Gumb koji otvara selektor itinerera |
| **Itinerary form** | Inline obrazac za odabir datuma i gostiju |
| **Search button** | Gumb koji otvara pretraživanje objekata |
| **Lookup button** | Gumb za pretraživanje specifičnog inventara |

## Kako ugraditi

Svaki widget pruža isječak koda koji možete kopirati i zalijepiti u HTML svoje web stranice. Isječak uključuje vaš **Client ID** i **Customization ID** tako da je widget automatski povezan s vašim partnerskim računom.

:::note
Ako koristite WordPress, također nudimo WordPress dodatak za lakšu integraciju.
:::
