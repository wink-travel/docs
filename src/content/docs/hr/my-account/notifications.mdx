Ponekad želimo stupiti u kontakt s vama. Kada se to dogodi, zvono u gornjem desnom kutu će se promijeniti.
Da biste pročitali što smo vam poslali:

<Steps>
1. Kliknite na `Račun > Obavijesti` u glavnoj navigacijskoj traci.
2. Vidjet ćete nove i prethodne obavijesti.
3. Odaberite `Označi kao pročitano` ili `Izbriši` obavijesti nakon što ih pročitate.
</Steps>

## API

Programeri koji žele upravljati `Affiliates` mogu otići na [Developers > API > Affiliate](/hr/developers/apis/#affiliate-api).
