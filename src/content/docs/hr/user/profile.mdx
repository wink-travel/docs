Ako želite ažurirati svoj korisnički profil za svoj Wink korisnički račun, slijedite ove korake:

<Steps>
  1. Morate biti prijavljeni na jednu od naših web stranica.
  2. Kliknite ikonu svog profila u gornjem desnom kutu i pojavit će se izbornik.
  3. Kliknite na `Profile` link.
  4. Slijedite upute za ažuriranje `e-mail`, `first` i `last name`.
  5. Također možete prenijeti novi `profile picture` od sebe samog.
  6. Klik `Save`.
</Steps>


:::napomena
Vaša nova profilna slika prikazat će se sljedeći put kada se potpuno odjavite s Winka.
:::
