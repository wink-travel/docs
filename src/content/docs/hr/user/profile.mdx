Ako želite ažurirati svoj korisnički profil za svoj Wink korisnički račun, slijedite ove korake:

<Steps>
1. Morate biti prijavljeni na jednu od naših web stranica.
2. Kliknite na ikonu svog profila u gornjem desnom kutu i pojavit će se izbornik.
3. Kliknite na poveznicu `Profil`.
4. Slijedite upute za ažuriranje svog `e-maila`, `imena` i `prezimena`.
5. Također možete učitati novu `profilnu sliku` sebe.
6. Kliknite `Spremi`.
</Steps>

:::note
Vaša nova profilna slika prikazat će se sljedeći put kada se potpuno odjavite iz Wink.
:::
