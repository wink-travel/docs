The content loader is at the heart of our web component library. It's responsible for displaying your inventory (cards, grids, maps).

## Usage

```html {3-6}
<html>
  <body>
    <wink-content-loader
       layout="GUEST_ROOM"
       id="2de7ee9c-61c9-11ef-9722-42004e494300"
    ></wink-content-loader>
  </body>
</html>
```

### Attributes

| Attribute | Required | Description |
|-----------|----------|-------------|
| `layout` | Yes | The layout type (see below) |
| `id` | No | The inventory identifier (omit for `RANKED` with custom `sort`) |
| `sort` | No | Sort type for `RANKED` layout only |

### Layout types

- `AD_BANNER`
- `MAP`
- `HOTEL`
- `GUEST_ROOM`
- `MEETING_ROOM`
- `SPA`
- `RESTAURANT`
- `ACTIVITY`
- `ATTRACTION`
- `PLACE`
- `ADD_ON`
- `LIST`
- `SEARCH`
- `RANKED`

### Sort types

Available when layout is `RANKED` and you don't want to use an existing ranked grid:

- `MEMBER`
- `PRICE_LOW_TO_HIGH`
- `PRICE_HIGH_TO_LOW`
- `PRICE`
- `POPULARITY`
- `ECO`
- `EXPERIENCE`
- `PERK`
- `LOYALTY`
- `PACKAGE`

Samples can be found for [cards](/studio/cards), [grids](/studio/grids) and [maps](/studio/maps).
