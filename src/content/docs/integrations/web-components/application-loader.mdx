The loader is responsible for keeping state and managing interaction between our components. It must be included on every page that uses Wink web components.

## Usage

Include it below the Wink Javascript tag:

```html {3-6}
<html>
  <body>
    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

### Attributes

| Attribute | Required | Description |
|-----------|----------|-------------|
| `client-id` | Yes | Your Wink application client ID |
| `configuration-id` | Yes | Your customization ID from the Wink portal |
