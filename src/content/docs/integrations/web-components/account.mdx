The account button allows you to add Wink authentication to your site.

## Usage

```html {3}
<html>
  <body>
    <wink-account></wink-account>
  </body>
</html>
```

Clicking on the button, when the user is un-authenticated, will forward the user to get authenticated.
When the user is authenticated, it display the user's profile icon.
