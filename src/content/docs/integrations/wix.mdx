The **Wink Wix App** adds Wink web components to your Wix site as native widgets you can drag and drop in the Wix Editor. No coding required for site owners.

## Install

<Steps>
1. Go to the [Wix App Market](https://www.wix.com/app-market) and search for **Wink Travel**.
2. Click **Add to Site** and follow the prompts to authorize the app.
3. Once installed, open **Dashboard → Wink Settings** and enter your **Client ID** and **Client Secret**.
   Go to [Applications](/my-account/applications) to retrieve your credentials.
4. Click **Save** and then **Test Connection** to confirm Wink is connected.
</Steps>

## Add a component to a page

<Steps>
1. Open the **Wix Editor** and go to the page where you want to add Wink content.
2. Click **Add (+) → Apps → Wink Travel** to see all available widgets.
3. Drag a widget onto the page — for example, **Wink Content** to display a hotel grid.
4. In the widget settings panel, select the **Layout Type** and paste the **Layout ID** from Wink Studio.
5. Click **Publish** to make the changes live.
</Steps>

## Available widgets

| Widget | Purpose |
|---|---|
| **Wink Content** | Displays hotel cards, ranked grids, maps, and more |
| **Wink Lookup** | Search bar for destinations and hotels |
| **Wink Search** | Icon button that opens the itinerary picker |
| **Wink Account** | Sign-in / user account button |
| **Wink Itinerary** | Itinerary button showing the current trip |
| **Wink Cart** | Shopping cart summary button |
