Wink 旅遊平台擁有源自需求的功能。人們不知道如何：

- 快速開始在線銷售旅遊庫存。
- 輕鬆透過社群媒體或線上銷售設施和體驗。
- 在訂房過程中向旅客進行加購。
- 找到專為旅遊設計且能直接賺取佣金的聯盟網絡。
- 利用聯盟網絡無需額外努力即可跨越數千個銷售渠道銷售，從而降低成本和風險。
- 保留客戶所有權以便重新行銷。
- 利用社交影響力獲利。
- 整合不需冗長保密協議和長時間等待的旅遊平台。
- 與聯盟合作時正確追蹤投資報酬率。
- …等等

隨著時間推移，新增了新功能，擴展了現有功能，也移除了一些功能。如果你在旅遊領域工作，或只是喜歡／需要旅遊，我們有充分理由讓你加入 Wink。繼續閱讀，了解原因。

以下功能構成 Wink 的核心。

## 訂房引擎

Wink 的核心是一個強大、白標且高度可自訂的訂房引擎。它具備你對訂房引擎的所有期待功能，如即時價格、物業資訊、設施、位置等。它支援不同的排名類別，如環保、生活風格和評論。旅客可以建立詳細的旅客檔案，分享給飯店，告知偏好、過敏資訊、緊急聯絡號碼等。

我們的訂房引擎以美觀的介面讓每個物業和目的地呈現出色。但真正讓它與眾不同的是能夠向旅客加售任何物品，無論是在物業內或周邊。就像廉價航空鼓勵乘客升級更寬敞座位、增加行李額度和餐點一樣，Wink 讓物業清楚知道加售應該如何進行。從房內附加服務、會議室預訂、當地餐廳訂位，到博物館、自然健行和水上樂園… Wink 讓物業有能力讓整個鄰近區域透過優惠和體驗活起來，讓旅客驚艷。

我們的重點是讓旅客體驗更完整的旅遊體驗，並在抵達前以更高價格點連結物業與客人。

## 社群分享

我們的社群分享功能旨在讓製作、分享和消費旅遊優惠在社群媒體或其他線上平台變得簡單。最廣泛使用的線上分享方式是 `連結`。我們的連結附帶動態預覽，會在你選擇的美麗圖片上疊加價格。此功能加速旅客發現和訂房流程，並提升從社群動態到訂房的轉換率​​。

## 連結管理器

連結管理器讓你保存並組織喜歡的網址以便日後使用。瀏覽器書籤就是一種連結管理器。隨著社群媒體興起，讓連結能被朋友和觀眾輕鬆取得的需求更大。你的 Facebook 動態就是一種社群友善的連結管理器。

我們的連結管理工具 WinkLinks 更進一步。

- 它提供你一個專屬網址，例如 https://i.trvl.as/bob，適合像 IG 這類只允許分享單一網址的平台，並讓你管理來自網路各處的連結。此外，如果網站支援 `豐富內容`，如 Spotify，WinkLinks 會讓播放清單可播放，或讓 Facebook 影片可觀看。
- WinkLinks 讓你免費加入所有現有的聯盟連結，如 CJ 和 Amazon。
- WinkLinks 也是任何人只需兩次點擊就能銷售旅遊庫存的最簡便方式。它是一個精簡版訂房引擎，專注於將你的觀眾轉換為訂房。

WinkLinks 高度可自訂，支援你的品牌需求。

## 聯盟網絡

訂房引擎是我們的核心產品，但聯盟網絡讓我們的平台活起來。Wink 讓物業與聯盟互相連結。雙方可以協商交易、佣金和折扣。物業不必猜測合作是否合理，能看到聯盟過去的表現及其帶來的旅客族群。即使沒有直接連結，聯盟也能取得一般可用的價格和優惠。

Wink 以多種方式向聯盟提供庫存。我們追蹤每筆訂房的分析、你的投資報酬率，並負責支付所有相關人員。你可以專注於你最擅長的事：銷售給你的觀眾。

## 物業入口網站

Wink 的 Extranet 是最先進的內容管理系統，讓物業能在線上銷售房型、設施、體驗和各種附加服務。我們設計此系統時，首重易用性，其次是飯店比以往更需要銷售更多。我們在整個使用旅程中運用 AI，協助並加速物業的 `上市時間`。過去需要數週的工作，現在不到一小時即可完成。

我們的 Extranet 是物業自助系統，讓物業能連結聯盟並觸及全球旅客。
