Wink 旅行平台所配備的功能都是出於需求而誕生的。人們不知道如何：

* 快速開始在線銷售旅遊庫存。
* 透過社群媒體或線上輕鬆銷售設施和體驗。
* 在預訂過程中向旅客追加銷售。
* 尋找一個適合旅行的聯盟網絡，您可以從中賺取直接佣金。
* 利用聯盟網絡，無需額外努力即可透過數千個銷售管道進行銷售；從而降低成本和風險。
* 保留客戶所有權以便重新定位。
* 利用他們的社會影響力賺錢。
* 與不需要冗長保密協議和長時間等待的旅遊平台整合。
* 與關聯公司合作時正確追蹤投資報酬率。
* ....以及更多

隨著時間的推移，新功能被添加，現有功能擴展，一些功能被刪除。如果您在旅遊領域工作，或者只是喜歡/需要旅行，我們有一個令人信服的理由讓您加入 Wink。請繼續閱讀以了解原因。

以下功能構成了 Wink 的核心。

## 預訂引擎

Wink 的核心是一個功能強大、白標且高度可自訂的預訂引擎。它具有您期望從預訂引擎獲得的所有功能，例如即時定價、有關酒店、其便利設施、設備、位置等的資訊。它包括對不同排名類別的支持，例如環保、生活方式和評論。它允許旅行者創建一份詳細的旅行者檔案，並向酒店分享他們的喜好、過敏類型、緊急電話號碼等資訊。

我們的預訂引擎透過其精美設計的介面使每個房產和目的地看起來都令人驚嘆。但是，它真正與眾不同之處在於，它能夠向遊客推銷其場所內外的任何產品。就像廉價航空公司激勵乘客升級到擁有更多腿部空間、更多行李配額和更多餐食的更好的座位一樣......Wink 讓酒店清楚地了解追加銷售應該是什麼樣的。從客房內輔助設施、預訂會議室或在當地餐廳預訂餐桌，到博物館、自然健行和水上樂園…Wink 讓飯店能夠透過讓旅客驚嘆的優惠和體驗，讓整個鄰居都充滿活力。

我們的重點是讓旅客獲得更豐富的體驗，並在客人抵達之前以更高的價格將飯店與客人聯繫起來。

## 社群分享

我們的社交分享功能旨在讓您輕鬆在社群媒體或任何其他地方線上製作、分享和使用旅遊優惠。在網路上分享任何內容最常用的方法是`the link`。我們的連結帶有動態連結預覽，可在您選擇的精美圖片上顯示價格。此功能加快了旅客發現和預訂的方式，並增加了從社交資訊流到預訂的轉換路徑。

## 連結管理器

連結管理員可讓您儲存和組織您喜歡的 URL 以供日後使用。您的瀏覽器書籤就是連結管理員的一個範例。隨著社群媒體的出現，向朋友和觀眾展示和提供連結的需求越來越大。您的 Facebook 動態就是社群友善連結管理員的一個範例。

我們的連結管理工具 WinkLinks 更進一步。

* 它會給你一個虛榮網址，例如<https://i.trvl.as/bob>，當 IG 等網站只允許您共享單一 URL 時，它可以讓您管理來自整個網路的連結。此外，如果網站支持`rich content`與 Spotify 一樣，WinkLinks 使播放清單可播放或使 Facebook 影片可觀看。
* WinkLinks 讓您免費添加所有現有的附屬鏈接，如 CJ 和 Amazon。
* WinkLinks 也是任何人銷售旅遊庫存的最簡單方式，只需點擊兩次即可。它是一個精簡的預訂引擎，專注於將您的受眾轉化為預訂。

WinkLinks 高度可自訂並支援您的品牌需求。

## 聯盟網絡

預訂引擎是我們的核心產品，但聯盟網絡才是讓我們的平台活躍起來的關鍵。 Wink 讓飯店與附屬機構建立聯繫，反之亦然。雙方可以協商交易、佣金和折扣。飯店無需猜測這種合作關係是否合理，而是可以了解聯營酒店的過往表現以及其帶來的旅客群體。即使沒有直接連接，聯屬公司也可以獲得普遍可用的價格和交易。

Wink 為附屬公司提供多種庫存。我們追蹤每筆預訂的分析結果、您的投資回報率，並負責向所有相關人員付款。你可以專注於你最擅長的事情；即向你的受眾銷售。

## 房地產入口網站

Wink 的 Extranet 是最先進的內容管理系統，可供飯店在線上銷售任何類型的房間、設施、體驗和輔助設備。當我們設計這個系統時，首先考慮的是易於使用，其次才是滿足飯店比以往任何時候都更需要銷售更多產品的需求。我們在整個用戶旅程中利用人工智慧來協助和加速酒店的`time-to-market`。過去需要數週才能完成的工作現在只需不到一小時就能完成。

我們的外部網路是一個自助服務系統，可供酒店與附屬機構聯繫並接觸世界各地的旅客。
