在 Wink，我們尊重您的隱私並全力致力於保護您的個人資料。我們遵守**一般資料保護規則 (GDPR)**及全球其他適用的資料保護法律。

本頁說明您如何請求刪除我們可能透過平台收集的任何個人資料——包括但不限於透過 Facebook 或 Instagram 等社交登入提供者取得的資料。

## 我們對資料隱私的承諾

我們僅收集提供服務所需的資料，且絕不販售您的個人資訊。這些資料可能包括：

- 您的姓名和電子郵件地址
- 公開的個人資料（例如來自 Facebook 或 Google）
- 您在我們平台上的使用或活動資料
- 您明確授權的任何權限

我們採取適當措施保障您的資料安全，並依據 GDPR 及 Meta 的資料政策處理所有刪除請求。

## 請求資料刪除

您有權請求我們刪除持有的任何個人資訊。

請求資料刪除的方法：

- 📧 **發送電子郵件至**：[support@wink.travel](mailto:support@wink.travel)
- 📝 主旨請使用：`Data Deletion Request`  
- 🔍 請包含識別資訊，例如：
  - 您在本平台使用的電子郵件地址
  - 您的 Facebook 或 Google ID 或類似資訊（如適用）

我們將於收到請求後的**30天內**回覆，符合 GDPR 要求。

## 有任何問題或疑慮？

若您對我們如何收集、使用或刪除您的資料有任何疑問，歡迎隨時聯絡我們的隱私團隊：[support@wink.travel](mailto:support@wink.travel)。
