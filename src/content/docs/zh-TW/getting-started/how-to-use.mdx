## 核心概念

> 你看到一個... 就等於看過全部

當您熟悉 Wink 的所有功能時，會發現許多操作方式有重疊。
理解核心概念將讓您更熟悉並更有信心使用我們的功能。

開始吧：

- 每個區塊通常都有列出、新增、更新及刪除項目的方式。
- 編輯項目時，表單欄位旁的紅色星號 <Badge text="*" variant="danger" size="large" /> 表示該欄位為必填。
- 若某欄位資料錯誤，該欄位會出現紅色邊框，並在欄位下方以紅字說明問題。
- 只有當表單有效且有變更時，您才能 `儲存` 表單。
- 有些表單包含多個分頁，例如客房表單。
- 所有分頁都必須有效才能繼續。
- 含有缺漏或無效資料的分頁，名稱旁會顯示黃色三角形 <Badge text="⚠️" variant="caution" size="large" />。
- 提交表單時，大多數表單會顯示 `確認` 對話框，詢問您是否要繼續。
