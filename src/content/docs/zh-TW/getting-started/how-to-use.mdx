## 核心概念

> 你看到一個...你就看到了所有

當您熟悉 Wink 的所有功能時，您會發現它們的運作方式有許多重疊之處。
理解核心概念將使您在使用我們的功能時更加熟悉並更有信心。

開始了：

* 每個部分通常都有列出、新增、更新和刪除條目的方法。
* 當你編輯條目時，紅色星號<Badge text="*" variant="danger" size="large" />
  &#x20;在表單欄位上表示它是必填的。
* 如果特定表單欄位不正確，您將看到其周圍有紅色邊框，並在表單欄位下方看到紅色文字，解釋問題所在。
* 您只能`Save` 一旦表格生效並且對其進行了一些更改。
* 有些表格包含多個標籤，例如客房表格。
* 所有標籤都必須有效，您才能繼續。
* 包含缺少或無效表單資料的選項卡將顯示黃色三角形<Badge text="⚠️" variant="caution" size="large" />
  &#x20;在其名稱旁邊。
* 提交表單時，大多數表單都會顯示`Confirmation` 對話框詢問您是否要繼續。
