## 創建應用程式

由於您無法訪問`secret key` 我們為您創建的應用程序，您需要創建自己的應用程式。

：：：警告
在創建應用程式之前，您需要在 Wink 上建立聯盟帳戶或在 TripPay 上建立帳戶，因為您將將您的應用程式綁定到特定帳戶。
:::

以下是建立應用程式的步驟：

<Steps>
  1. 在應用程式頁面上，按一下 `Create new application` 按鈕。
  2. ***姓名*** 為你的應用程式命名。 *例如酷炫應用*
  3. **實體名稱** 將您的應用程式與您的一個現有帳戶關聯。 *例如酷帳戶*
  4. **重定向 URI** 至少需要一個重定向 URI。它用於驗證成功後的重定向網域。 *例如https://www.cool-site.com*
  5. 點選 `Save` 按鈕繼續。
</Steps>


您將被重新導向回您的應用程式清單。您將在清單中看到您的新應用程序，並且只能存取一次應用程式金鑰。確保複製它並將其保存在安全的地方。

## 更新應用程式

如果您的某個重新導向 URI 發生變更或您想要新增或刪除 URI，則需要更新您的應用程式。

若要更新現有應用程序，請執行以下操作：

<Steps>
  1. 點選 `Actions` 您要更新的應用程式的連結。
  2. 點選 `Update` 操作下的按鈕。
  3. 對應用程式進行更改。
  4. 點選 `Save` 按鈕繼續。
</Steps>


您將被重新導向回您的應用程式清單。

## 刪除應用程式

：：：警告
請不要刪除我們為您自動建立的應用程序，以確保所有功能都適用於您的帳戶。
:::

如果您不再使用某個應用程序，您可以直接將其刪除。

若要刪除應用程序，請執行以下操作：

<Steps>
  1. 點選 `Actions` 您要更新的應用程式的連結。
  2. 點選 `Remove` 操作下的按鈕。
  3. 點選 `OK` 按鈕確認刪除。
</Steps>


您將被重新導向回您的應用程式清單。
