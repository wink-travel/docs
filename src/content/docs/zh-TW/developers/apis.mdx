我們所有的 REST API 都是語言無關且符合 Open API 規範。開始測試，無需撰寫任何程式碼。

如果您還沒有建立 Wink 帳號， [請從這裡開始](/user/register) 。

單一端點用於取得訂房引擎的白標與自訂化資訊。

</LinkButton>

### Lookup API

### Property API

## Taxonomy API

Taxonomy 端點適用於想消費及生產旅遊庫存，並需要標準及非標準代碼分類（如庫存類型、等級、狀態等）的開發者。

<LinkButton href="https://developer.wink.travel/reference" variant="secondary" icon="external" target="_blank">
了解更多
</LinkButton>

## Analytics API

Analytics API 提供端點給任何想要洞察排行榜排名及深入分析時間序列、平台層級資料的使用者，涵蓋多種指標。

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
了解更多
</LinkButton>

## Payment API

Payment 端點適用於想購買旅遊庫存的開發者。註冊旅行社可透過 API 完成購買，其他實體則可結合我們符合 PCI 規範的付款元件使用 API。
Payment API 提供端點給聯盟與飯店追蹤預訂、分析及資金可用性，並可選擇將可用資金提領至銀行帳戶。

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
了解更多
</LinkButton>
