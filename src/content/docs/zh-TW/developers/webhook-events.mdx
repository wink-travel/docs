此頁面列出您可以在 Wink 訂閱的所有 webhook 事件。設定說明請參閱 [Webhook Integration Guide](/zh-TW/guides/webhook-integration) 或 [Webhooks 管理頁面](/zh-TW/my-account/webhooks)。

## 如何訂閱

1. 在您的入口網站中，導航至 **Applications > Webhooks**。
2. 建立 webhook 並選擇您想接收的事件。
3. 每當事件發生時，Wink 會向您的端點發送 HTTP POST。

完整設定步驟與最佳實踐，請參閱 [Webhook Integration](/zh-TW/guides/webhook-integration)。
