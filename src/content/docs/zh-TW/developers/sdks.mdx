使用我們官方支援的語言軟體開發工具包之一來加快產品上市時間。
如果我們目前不支援您喜歡的語言，請前往您想要使用的 API 並下載我們在頁面頂部為每個 API 提供的 OpenAPI 規格。您可以透過分叉我們的一個公共 SDK 儲存庫或使用其中一個 OpenAPI 產生器來產生您自己的 SDK。

## Java

該函式庫是針對 Java 24 編譯的。它配置為使用反應式 WebClient，並針對與 Spring Framework 的使用進行了最佳化。然而，Spring 不是必需的，但它確實為您提供了開箱即用的 OAuth2 支援。
