透過使用我們官方支援的程式語言軟體開發套件，加快產品上市時間。
如果我們目前尚未支援您喜愛的語言，請前往您想使用的 API，並下載我們在頁面頂端提供的 OpenAPI 規格。您可以透過分叉我們的公開 SDK 倉庫之一，或使用 OpenAPI 的產生器自行生成 SDK。

## Java

此函式庫為 Java 24 編譯。它配置為使用反應式 WebClient，並針對 Spring Framework 進行優化。不過，Spring 並非必須，但它可讓您開箱即用地支援 OAuth2。
