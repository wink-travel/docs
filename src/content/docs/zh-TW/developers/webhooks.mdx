Webhook 是一種讓您接收並回應來自 Wink 和 TripPay 的有關您帳戶的即時事件的方式。

## 例子

<Steps>
  1. 在 Wink 上預訂。
  2. Wink 活動 `booking.created` 已派遣。
  3. 您的系統正在積極監聽此事件。
  4. 您的系統會追蹤您自己的一組分析並使用此事件來更新其時間序列資料庫。
</Steps>

：：：筆記
Wink webhook 與 TripPay webhook 不同。

* 如果你想監聽 Wink 事件，你可以建立一個 webhook[外部網路](https://extranet.wink.travel) 或者[工作室](https://studio.wink.travel)。
* 如果您想監聽 TripPay 事件，您可以建立一個 webhook[支付](https://pay.wink.travel)。
  :::

*對於此範例，我們將透過以下方式進行身份驗證[工作室](https://studio.wink.travel)。*

<Steps>
  1. 一旦通過身份驗證，請點擊右上角的個人資料圖標，然後會出現一個選單。
  2. 點選 `Webhooks` 關聯。
  3. 您將被重定向到您的 webhook 頁面。
</Steps>


## 創建 webhook

：：：警告
在創建 webhook 之前，您需要在 Wink 上建立一個附屬帳戶或在 TripPay 上建立帳戶，因為您將綁定 webhook 到特定帳戶。
:::

以下是建立 webhook 的步驟：

<Steps>
  1. 在 webhook 頁面中，按一下 `Create new webhook` 按鈕。
  2. ***姓名*** 為你的 webhook 命名。 *例如眨眼追蹤者*
  3. ***使能夠*** 保持原樣即可啟用。
  4. **實體名稱** 將您的 webhook 與您的一個現有帳戶連結。 *例如酷帳戶*
  5. **活動** 聆聽您感興趣的事件。 *例如 `booking.created`*
  6. **Webhook URL** 輸入您將收聽我們活動的 URL。 *例如 `https://my.system.com/webhook/wink`*
  7. 點選 `Save` 按鈕繼續。
</Steps>


您將被重定向回您的 webhook 列表，並且您的新 webhook 已準備就緒。

## 更新 webhook

如果您想要執行以下操作，則需要更新您的應用程式：

* 禁用你的 webhook。
* 收聽更多/更少事件。
* 更改您的監聽 webhook URL。

若要更新現有的 webhook，請執行下列操作：

<Steps>
  1. 點選 `Actions` 您想要更新的 webhook 連結。
  2. 點選 `Update` 操作下的按鈕。
  3. 對 webhook 進行更改。
  4. 點選 `Save` 按鈕繼續。
</Steps>


您將被重定向回您的 webhook 清單。

## 刪除 webhook

如果您不再需要該 webhook，您可以繼續將其刪除。

若要刪除 webhook，請執行以下操作：

<Steps>
  1. 點選 `Actions` 您想要更新的 webhook 連結。
  2. 點選 `Remove` 操作下的按鈕。
  3. 點選 `OK` 按鈕確認刪除。
</Steps>


您將被重定向回您的 webhook 清單。
