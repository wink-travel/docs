對於任何使用 WordPress 開展業務或部落格的人，我們強烈建議使用 [Wink Affiliate WordPress插件](https://wordpress.org/plugins/wink2travel/) 。
該外掛程式讓您可以輕鬆存取您的 Wink 卡、網格、地圖等，並且可以使用您最喜歡的頁面建立器或我們的自訂短代碼將內容拖曳到您的貼文和頁面中。

## 安裝

這是安裝方法`Wink Affiliate WordPress Plugin` 進入您的 WordPress 網站。

<Steps>
  1. 當您以管理員身分登入儀表板時。
  2. 點選 `Plugins > Add New Plugin`。
  3. 搜尋 `Wink Affiliate WordPress Plugin`。
  4. 點選 `Install` 下載插件並將其安裝到您的網站。
  5. 點選 `Activate` 在您的網站上啟用該外掛程式。
  6. 啟動後會轉發 `Wink Settings` 頁。 *您也可以透過導航前往 `Appearance > Customize > Wink Settings`。*
  7. 它會問你 `Client ID` 和 `Client Secret`。前往 [應用](/developers/applications) 了解如何檢索您的憑證。
  8. 輸入您的憑證並點擊 `Publish`。您現在已連線到您的 Wink 帳戶。
</Steps>

## 頁面建立器

我們支援 WordPress 上最受歡迎的頁面建立器，讓任何想要使用 Wink 的人都可以輕鬆使用。

### 古騰堡

Gutenberg 是 WordPress 的原生編輯器。

### 阿瓦達

[阿瓦達](https://avada.com) 以其大量精美的模板以及具有高級功能集的強大頁面建立器而聞名。

### Elementor

[Elementor](https://elementor.com) 是\[目前]最受歡迎的頁面建立器，有免費版和專業版可供選擇。

### WPBakery

[WPBakery](https://wpbakery.com) 是市場上第一個先進的頁面建立器。

## 短代碼

### \[眨眼查找]

與排名內容網格一起使用。輸入您想造訪的地方並顯示庫存。

### \[眨眼搜尋]

簡單的按鈕即可開啟行程表。

### \[winkaccount]

帳戶按鈕可讓您進行身份驗證。一旦通過身份驗證，它就會變成一個帶有帳戶選項的下拉式選單。

### \[眨眼日記]

與搜尋按鈕相同，只是它包含行程資訊作為按鈕文字。

### \[眨眼內容]

允許您將您在 Wink 上選擇的庫存直接嵌入到頁面或貼文中。此短代碼還需要一個佈局 ID。

佈局

```
[winkcontent layoutid=""]
```
