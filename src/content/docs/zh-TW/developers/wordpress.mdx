對於使用 WordPress 經營商務或部落格的用戶，我們強烈推薦使用 [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/)。
此外掛讓您輕鬆存取 Wink 卡片、網格、地圖等，並可透過您喜愛的頁面編輯器或我們的自訂短碼，簡單地將內容拖曳至文章和頁面中。

## 安裝

以下說明如何將 `Wink Affiliate WordPress Plugin` 安裝到您的 WordPress 網站。

<Steps>
1. 以管理員身份登入並進入儀表板。
2. 點選 `外掛 > 安裝外掛`。
3. 搜尋 `Wink Affiliate WordPress Plugin`。
4. 點選 `安裝`，將外掛下載並安裝到您的網站。
5. 點選 `啟用`，在您的網站啟用外掛。
6. 啟用後會導向 `Wink 設定` 頁面。*您也可以透過 `外觀 > 自訂 > Wink 設定` 前往該頁面。*
7. 系統會要求您輸入 `Client ID` 和 `Client Secret`。請前往 [Applications](/zh-TW/developers/applications) 了解如何取得您的憑證。
8. 輸入憑證後點選 `發布`。您現在已連接至您的 Wink 帳號。
</Steps>

## 頁面編輯器

我們支援 WordPress 上最受歡迎的頁面編輯器，讓任何想使用 Wink 的人都能輕鬆上手。

### Gutenberg

Gutenberg 是 WordPress 內建的編輯器。

### Avada

[Avada](https://avada.com) 以豐富的精美範本及功能強大的頁面編輯器聞名。

### Elementor

[Elementor](https://elementor.com) 是目前最受歡迎的頁面編輯器，提供免費及專業版。

### WPBakery

[WPBakery](https://wpbakery.com) 是市場上第一款進階頁面編輯器。

## 短碼

### [winklookup]

適用於排名內容網格。輸入您想造訪的地點，並顯示相關庫存。

### [winksearch]

簡單的按鈕，用於開啟行程表單。

### [winkaccount]

帳號按鈕，讓您進行身份驗證。驗證後會變成帶有帳號選項的下拉選單。

### [winkitinerary]

與搜尋按鈕相同，但按鈕文字包含行程資訊。

### [winkcontent]

讓您將在 Wink 上選擇的庫存直接嵌入頁面或文章中。此短碼也需要一個版面配置 ID。

版面配置

```
[winkcontent layoutid=""]
```
