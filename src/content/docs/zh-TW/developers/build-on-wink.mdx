> 「如果我看得更遠，那是因為我站在巨人的肩膀上。」

-- *艾薩克·牛頓爵士*

這樣的平台不是孤立建造的，而是許多運作部件的總和；有舊的也有新的。我們希望繼續服務後來者。

> 「沒必要重新發明破損的輪子。」

旅遊科技分散且多為舊有系統。我們嘗試採用有效的舊系統精華。我們整合了許多渠道管理系統、物業管理系統等。我們也加入了一些新元素。並且，我們將所有這些包裝在業界標準和協議中，這些標準和協議是眾所周知且易於使用的……這樣您就不必自己做。

:::note[API 驅動方法]
大致上意味著，我們為自己建置的功能應該是：

- 易於理解
- 靈活
- 可擴展
- 可重用，且...
- 可擴充

……供其他也想使用這些 API 的人使用。
:::

Wink 是使用這種 `API 驅動方法` 建構的。基於這種設計理念，所有 Wink 的功能都可以像系統是您自己的那樣，讓外部開發者互動。此外，平台還有其他面向，您在整合時可以加以利用。
