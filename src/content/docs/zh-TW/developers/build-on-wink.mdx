> “如果說我看得更遠，那是因為我站在巨人的肩膀上。”

\--*艾薩克·牛頓爵士*

這樣的平台並不是孤立建立的。相反，它是許多活動部件的總和；新與舊。我們希望繼續為我們的後代服務。

> “沒有必要重新發明破舊的輪子。”

旅遊技術較為分散且遺留。我們盡力吸取舊方法的精華。我們整合了一批通路經理、物業管理系統++。我們添加了一些新內容。而且，我們已將所有內容納入已知且易於使用的行業標準和協議中……因此您不必這樣做。

:::note\[API 驅動程式的方法]
大致意思是，我們為自己建立的任何功能都應該是：

* 可以理解
* 靈活的
* 可擴充
* 可重複使用且...
* 可擴充

……其他也想使用這些 API 的人。
:::

Wink 是使用這樣的`API-driven approach`。由於這種設計理念，外部開發人員可以與 Wink 的所有功能進行交互，就像系統是您自己的系統一樣。與我們整合時，您還可以利用該平台的其他方面。
