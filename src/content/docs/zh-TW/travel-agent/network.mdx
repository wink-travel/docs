當您首次以旅遊代理身份加入 Wink，您會找到一些想在平台上銷售的商品，並分享到社群媒體、您的旅遊部落格、應用程式等。
某個時候，您會覺得自己已經相當熟悉我們的工具，並且累積了一些訂單。
接下來您可以做的，就是尋求與供應商建立直接合作關係，提升您的銷售實力。

:::note
您可以透過點擊主選單的 `Account > Network` 來管理您目前的活躍及待審核合作關係。
:::

本文說明何時以及如何向供應商申請直接合作關係，這對雙方都有利。

## 何時申請

請記住，對大多數旅遊代理來說，直接與供應商連結並不一定有意義。大多數供應商已經向平台上的所有人提供了最佳優惠和加購服務。
不過，我們建議您在符合以下任一條件時，嘗試與我們的供應商建立直接連結：

- 您擁有大量潛在訂房客群，且與特定供應商的產品特別契合。
*例如：您是滑雪愛好者，擁有豐富的粉雪、野外滑雪內容，且所有追蹤者都是滑雪迷。與滑雪旅館連結，申請獨特的平日優惠或類似方案，是明智之舉。*
- 您帶著自己的旅遊團隊前往目的地。
*例如：您組織瑜伽靜修團，已有 20 位瑜伽學員確認參加。透過直接連結，靜修中心可以提供您獨家促銷代碼，讓您的團員享有團體折扣。*
- 您是處理大量訂單的旅遊代理或企業。
*旅遊代理和企業在 Wink 上皆被視為合作夥伴，但他們也擁有專屬網站。*

## 申請條件

您必須在 Wink 上至少完成 5 筆過去訂單，才能提出申請。
我們設置此條件是為避免供應商收到過多訊息。我們建議您先專注於在 Wink 建立良好業績，供應商自然會主動聯繫您。

## 運作方式

當您找到想合作的供應商（請參考 [搜尋](/zh-TW/studio/search/)），並想提出合作申請時，流程如下：

<Steps>
1. 您在該供應商的搜尋結果中點擊 `Request a direct connection`。
2. 系統會呈現一份表單，讓您向供應商自我介紹。請保持禮貌並切中要點，說明您是誰、為何聯繫、您能為他們帶來什麼價值，以及您目前的具體需求。
3. 提交表單後，等待供應商回覆。回覆時間可能從一天到一週不等。供應商工作繁忙，請耐心等候。在等待回覆期間，您無法對同一供應商提出另一申請。
</Steps>

供應商會回覆：

- **接受** 您的申請已被接受，雙方建立直接銷售通路，且可依需求進行多種設定。
- **拒絕** 您的申請被拒絕，您仍可使用原有的庫存。最早可在拒絕後兩個月再次申請。
- **無回應** 供應商未回覆。此時建議您尋找其他供應商。

## 優勢

一旦申請被接受，供應商可對您新建立的銷售通路進行以下設定：

- 為您的客群設定不同的會員折扣。
- 設定不同的佣金比例，讓您賺取更多。
- 新增僅限您客群享有的獨家優惠。
- 新增僅限您客群使用的獨家庫存及加購服務。

## API

想管理您的網絡的開發者，可前往 [Developers > API > Sales Channel](/zh-TW/developers/apis/#sales-channel-api)。
