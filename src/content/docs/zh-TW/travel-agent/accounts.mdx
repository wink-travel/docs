[Wink Agent](https://agent.wink.travel) 連結您與全球旅遊庫存。本文將引導您如何在 Wink 上建立和管理您的旅行社帳戶。

## 建立帳戶

驗證後，您將被導向您的[帳戶頁面](https://agent.wink.travel/company/list)，該頁面會顯示您所有的旅行社帳戶。

點擊 `Create new account` 按鈕繼續。您將被導向我們的[建立帳戶](https://agent.wink.travel/company/create)頁面。

填寫所有必填欄位。

<Steps>
1. **名稱** 您帳戶的名稱。*這不必是您的真實姓名，但將會是使用者在查看您的 WinkLinks 頁面或供應商想直接與您連繫時看到的名稱。*
2. **實體類型** 您是公司還是個人。*主要用於判斷您如何有資格接收款項。*
3. **類型** 選擇最適合您帳戶的類型。*例如：影響者*
4. **描述** 為您的帳戶撰寫描述。*這將顯示在您的 WinkLinks 頁面上，並在供應商想直接與您連繫時看到。*
5. **網站** 輸入您的主要網站。*這可以是您的 IG 帳號網址或公司網站。應該是您大部分線上工作的地方。供應商會透過這些連結判斷是否與您連結。*
6. **城市** 您的所在地？
</Steps>

### 付款條款

:::note
如果您選擇自行負責付款，您將只能存取與您有直接連結的物業。
:::

:::caution
在此情況下，Wink 不承擔支付飯店的責任。
:::

:::note
您只能在建立帳戶時設定此選項。大多數旅行社應保持此設定關閉 🛑，並使用此入口網站。
:::

已註冊並使用我們 API 整合的旅行社，可以選擇收款並將款項支付給我們的物業。

<Steps>
1. **我想收款** 旅行社負責收取款項。
2. **我想支付飯店** 旅行社負責支付飯店。
</Steps>

想要直接與您連結的飯店會看到警告，並必須點擊確認 Wink 不負責支付飯店，才能與您的帳戶建立關係。

### 協議

<Steps>
1. **協議** 接受我們的[聯盟協議](/zh-TW/travel-agent/terms-of-service)及[付款條款](/zh-TW/travel-agent/payment-terms)以繼續。
2. 點擊 `Save` 按鈕。
</Steps>

恭喜 🎉 ...您已在 Wink 上建立了第一個聯盟帳戶。

建立後，您可以進一步自訂帳戶，填寫地址並新增個人資料圖片。

<Steps>
1. 點擊您的新旅行社帳戶。*這會帶您到帳戶儀表板。*
2. 在主選單點擊 `Account > Profile`。
3. 在個人資料頁面，您可以更新現有資料並新增更多資訊。
</Steps>

:::note
您的旅行社帳戶個人資料圖片不應與您的 Wink 使用者個人資料圖片混淆。旅行社帳戶個人資料圖片會顯示在您的 WinkLinks 頁面，並被想與您連結的供應商看到。

您的使用者個人資料圖片主要供內部使用。
:::

## 管理者

您可以邀請其他使用者協助管理您的旅行社帳戶。使用者接受邀請後，將擁有帳戶的完整存取權，但不包含付款權限。您將始終是唯一掌控您資金的人。

<Steps>
1. 從[帳戶頁面](https://agent.wink.travel/company/list)選擇您想管理的旅行社帳戶。
2. 在主選單點擊 `Account > Managers`。
3. 輸入您想設為帳戶管理者的使用者電子郵件。
4. 點擊電子郵件旁的 `Invite` 連結。
</Steps>

Wink 會寄送電子郵件給該使用者，邀請他們接受或拒絕您的邀請。

## 通知

有時我們會想與您聯繫。當發生時，右上角的鈴鐺圖示會改變。
要查看我們發送的訊息：

<Steps>
1. 在主選單點擊 `Account > Notifications`。
2. 您會看到新的和過去的通知。
3. 閱讀後，您可以選擇 `標記為已讀` 或 `刪除` 通知。
</Steps>

## API

旅行社仍被視為聯盟。想管理 `Affiliates` 的開發者可以前往 [Developers > API > Affiliate](/zh-TW/developers/apis/#affiliate-api)。
