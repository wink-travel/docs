[眨眼代理](https://agent.wink.travel) 將您與世界各地的旅行庫存聯繫起來。本文將指導您在 Wink 上建立和管理您的旅行社帳戶。

## 建立帳戶

一旦通過身份驗證，您將被重定向到您的[帳戶頁面](https://agent.wink.travel/company/list) 它會顯示您所有的旅行社帳戶。

點選`Create new account` 按鈕繼續。您將被重定向到我們的[建立帳戶](https://agent.wink.travel/company/create) 頁。

填寫所有必填欄位。

<Steps>
  1. **姓名** 您的帳戶名稱。 *這不需要是您的真實姓名，但當用戶查看您的 WinkLinks 頁面或供應商想要直接與您聯繫時，就會看到它。*
  2. **實體類型** 無論您是公司還是個人。 *主要用於確定您是否有資格獲得資金。*
  3. **類型** 選擇最適合您帳戶的類型。 *例如影響者*
  4. **描述** 為您的帳戶寫一個描述。 *這將在您的 WinkLinks 頁面上顯示，當供應商想要直接與您聯繫時也會顯示。*
  5. **網站** 輸入您的主網站。 *這可以是您的 IG 帳戶 URL 或公司網站。它應該是您進行大部分在線工作的地方。供應商在決定是否與您聯繫時會查看這些連結。*
  6. **城市** 你們在哪裡？
</Steps>


### 付款條款

：：：筆記
如果您選擇負責付款，您將只能造訪與您有直接聯繫的房產。
:::

：：：警告
Wink 不承擔在這種情況下支付飯店費用的責任。
:::

：：：筆記
您只能在建立帳戶期間與此選項進行互動。大多數旅行社應該停用此設定🛑並改用此入口網站。
:::

使用我們的 API 與我們合作的註冊旅行社可以選擇收取付款並將這些資金支付給我們的飯店。

<Steps>
  1. **我想要付款** 旅行社負責收取資金。
  2. **我想支付飯店費用** 旅行社負責支付飯店費用。
</Steps>


想要與您直接聯繫的飯店將看到一條警告，並且必須點擊確認，Wink 不負責向飯店付款，以便與您的帳戶建立關係。

### 協定

<Steps>
  1. **協定** 接受我們的 [聯營協議](/travel-agent/terms-of-service) 和 [付款條件](/travel-agent/payment-terms) 繼續。
  2. 點選 `Save` 按鈕。
</Steps>


恭喜🎉...您已經在 Wink 上建立了您的第一個聯盟帳戶。

## 管理

  <figcaption>*旅行社帳戶輸入*</figcaption>

創建後，您可以透過填寫地址和添加個人資料圖片來進一步自訂您的帳戶。

<Steps>
  1. 點擊您的新旅行社帳戶。 *這將帶您進入您的帳戶儀表板。*
  2. 點選 `Account > Profile` 在主導覽列中。
  3. 在個人資料頁面上，您將能夠更新您現有的個人資料並添加其他資訊。
</Steps>


：：：筆記
您的旅行社帳戶個人資料圖片不應與您的 Wink 用戶個人資料圖片混淆。您的旅行社帳戶個人資料圖片將顯示在您的 WinkLinks 頁面上，並可供您想要與您聯繫的供應商看到。

您的使用者資料圖片主要供內部使用。
:::

## 主管

您可以邀請其他使用者來協助管理您的旅行社帳戶。一旦用戶接受您的邀請，他們將擁有您帳戶的完全存取權（付款除外）。您將永遠是您自己所擁有的資金的唯一負責人。

<Steps>
  1. 從中選擇您想要使用的旅行社帳戶 [帳戶頁面](https://agent.wink.travel/company/list)。
  2. 點選 `Account > Managers` 在主導覽列中。
  3. 將您想要設為帳戶管理員的使用者的電子郵件加入。
  4. 點選 `Invite` 電子郵件旁邊的連結。
</Steps>


Wink 將向用戶發送電子郵件，以接受或拒絕您的邀請。

## 通知

有時，我們想與您聯繫。當發生這種情況時，您會看到右上角的鈴鐺發生變化。
要閱讀我們發送給您的內容：

<Steps>
  1. 點選 `Account > Notifications` 在主導覽列中。
  2. 您將看到新的和過去的通知。
  3. 選擇 `Mark as read` 或者 `Delete` 閱讀完通知後，請立即通知。
</Steps>
