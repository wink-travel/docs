當在 Wink 上完成預訂時，會發生以下情況：

<Steps>
1. 您的渠道管理系統、CRS 或 PMS 會收到通知。
2. 您會收到一封確認電子郵件 *(訂房櫃台聯絡人)*，內含管理新預訂的連結。
3. 旅客會收到一封確認電子郵件，並包含如何聯絡您的訂房櫃台的資訊。
4. 旅行社會收到與旅客相同的預訂確認。
4. 開發人員可以監聽 `booking.created` webhook 事件，並即時接收我們提供的詳細預訂資訊。
</Steps>

您可以點擊電子郵件中的連結，或從主導覽列進入 `Account > Bookings` 開始操作。

## 預訂列表

預訂列表會顯示您所有過去及未來的預訂，依預訂日期升冪排序。

### 篩選條件

您可以使用篩選條件來尋找所需的預訂。

- 預訂日期 [範圍]
- 抵達日期 [範圍]
- 離開日期 [範圍]
- 預訂代碼
- 旅客姓氏
- 旅客名字
- 主費率名稱
- 物業名稱

### 匯出

您可以將符合目前篩選條件的預訂匯出為 CSV 檔。

匯出預訂步驟：

<Steps>
1. 確認您已啟用所需的篩選條件。
2. 等待預訂列表刷新。
2. 點擊列表右下角的 🗂️ (*檔案匯出*) 圖示按鈕。
</Steps>

## 預訂詳情

從預訂列表中選擇您要操作的預訂，系統會導向 `Booking` 頁面。
此頁面包含您在旅客抵達當天歡迎他們所需的所有詳細資訊。

頁面上也包含一些重要操作，協助您妥善服務旅客。您可以在頁面頂端的 `Actions` 下拉選單中找到所有可用操作。

### 重新寄送電子郵件

若要重新寄送確認電子郵件給旅客，請點選 `Actions > Resend confirmation email`。

### 取消預訂

若要以物業身份取消預訂，請點選 `Actions > Cancel booking`。

系統會跳出一個視窗，讓您：

<Steps>
1. 選擇取消類型。*例如：日期錯誤*
2. 輸入取消原因。
</Steps>

*此按鈕會依照您的取消政策規則顯示，因此不一定隨時可用。*

## API

想以旅行社身份管理 `Bookings` 的開發人員，可前往 [Developers > API > Travel Agent](/zh-TW/developers/apis/#travel-agent-api)。
