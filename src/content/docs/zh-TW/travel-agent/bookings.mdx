在 Wink 上預訂時，會發生以下情況：

<Steps>
  1. 您的通路經理、CRS 或 PMS 已收到通知。
  2. 您將收到一封確認電子郵件 *（預訂台聯絡方式）* 並附有管理新預訂的連結。
  3. 旅客會收到一封確認電子郵件，其中還包括如何聯絡預訂處的資訊。
  4. 旅行社收到與旅客相同的預訂確認。
  5. 開發人員可以聽取 `booking.created` webhook 事件並即時接收我們的詳細預訂資訊。
</Steps>


請點擊電子郵件中的鏈接，或前往`Account > Bookings` 從主導覽列開始。

## 預訂網格

預訂網格顯示您所有過去和未來的預訂，按預訂日期升序排列。

### 過濾器

您可以套用篩選器來尋找您正在尋找的預訂。

* 預訂日期\[範圍]
* 到達日期\[範圍]
* 出發日期\[範圍]
* 預訂代碼
* 旅行者姓氏
* 旅行者的名字
* 主費率名稱
* 屬性名稱

### 出口

您可以將符合目前篩選條件的預訂匯出為 CSV。

要匯出您的預訂：

<Steps>
  1. 確保您想要的過濾器處於活動狀態。
  2. 等待預訂網格刷新。
  3. 點選🗂️（*文件匯出*) 圖標按鈕。
</Steps>


## 預訂

從預訂網格中選擇您想要處理的預訂，您將被重新導向到`Booking` 頁。
本頁包含您需要了解的所有詳細信息，以便在客人抵達當天妥善接待他們。

該頁面還包含為客人提供妥善服務所需的一些重要操作。您可以在`Actions` 頁面頂部的下拉式選單。

### 重新發送電子郵件

若要向旅客重新發送確認電子郵件，請點選`Actions > Resend confirmation email`。

### 取消預訂

若要取消預訂，請點擊`Actions > Cancel booking`。

將顯示一個彈出窗口，您可以：

<Steps>
  1. 選擇取消類型。 *例如日期不正確*
  2. 輸入取消的原因。
</Steps>


*此按鈕遵循您的取消政策規則，因此並非總是可用。*

## API

想要管理的開發人員`Bookings`作為旅行社代理人，可以前往 [開發人員 > API > 旅行社](/developers/apis/#travel-agent-api) 。
