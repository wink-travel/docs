我們為註冊旅行社提供的解決方案是 [Wink Agent](https://agent.wink.travel)。這是一個自助式入口網站，代理人可以用來代表客戶搜尋和預訂，並從每筆訂單中賺取佣金。

更進階的旅行社可以選擇在 API 層級整合，以便在 Wink 平台外處理付款和支付。

旅行社擁有與 [Wink Studio](/zh-TW/studio/what-is-studio) 部分相同的入口網站功能，因此這裡不再重複說明。
