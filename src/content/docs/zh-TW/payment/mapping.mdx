如果您是像 Wink 這樣的整合商，並且想要將您的外部收款人映射以用於 TripPay，您可以透過以下兩種方式之一完成：

- 手動映射
- 使用 [Payment API](/zh-TW/developers/apis/#payment-api) (*請參閱 `Account Mappings`*)

:::tip
除非您只處理少數相同的物業，否則我們建議您使用 API 來自動化此步驟。
:::

以下，我們將引導您完成手動映射外部收款人的流程。*此流程底層使用 API。*

<Steps>
1. 登入 [TripPay](https://pay.wink.travel)。
2. 選擇您的整合商帳戶。
3. 從主導航列點選 `Settings > Mappings`。
4. 點擊 `Create mapping` 按鈕。您將看到映射表單。
5. **Local account** 開始輸入您想映射的現有 TripPay 帳戶名稱。
6. **External identifier** 貼上此實體在遠端系統中的外部 ID。
7. 點擊 `Save` 繼續。系統會將您導回映射列表。
</Steps>

完成映射後，TripPay 現在知道當您使用外部識別碼參照時，應將預訂的受益人款項入帳至哪些帳戶。

## API

想管理 `Payments` 的開發者，請前往 [Developers > API > Payments](/zh-TW/developers/apis/#payment-api)。
