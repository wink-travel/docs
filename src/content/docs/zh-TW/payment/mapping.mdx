如果您是像 Wink 這樣的整合商，並且想要映射您的外部收款人以便與 TripPay 一起使用，您可以透過以下兩種方式之一來實現：

* 手動映射
* 使用[支付API](/developers/apis/#payment-api) (*看`Account Mappings`*)

：：：提示
除非您僅處理少數相同的屬性，否則我們建議您使用 API 來自動執行此步驟。
:::

下面，我們將引導您完成手動映射外部收款人的方法。 *它使用底層的 API。*

<Steps>
  1. 登入 [旅費](https://pay.wink.travel)。
  2. 選擇您的整合商帳戶。
  3. 點選 `Settings > Mappings` 從主導覽列。
  4. 點選 `Create mapping` 按鈕。您將看到映射表單。
  5. **本地帳戶** 開始輸入您想要映射的現有 TripPay 的名稱。
  6. **外部標識符** 貼上此實體在遠端系統中的外部 ID。
  7. 點選 `Save` 繼續。您將被重新導向回您的映射清單。
</Steps>


透過您的映射，當您使用外部識別碼引用哪些帳戶時，TripPay 現在知道要將哪些帳戶記為預訂的受益人。

## API

想要管理的開發人員`Payments`，前往[開發者 > API > 付款](/developers/apis/#payment-api)。
