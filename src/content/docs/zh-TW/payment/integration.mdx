如果您是旅遊公司，並且想將您的業務中的付款和撥款部分外包給 TripPay，以下是開始的 6 個基本步驟：

<Steps>
1. 在 Wink 上建立一個 [使用者帳號](/zh-TW/user/register)。
2. 在 [TripPay](https://pay.wink.travel) 建立一個帳號。
3. 在 TripPay 上建立一個 [應用程式](/zh-TW/developers/applications)，讓您能與我們的 API 互動。
4. 在 TripPay 建立並對應您的受益人。*例如，大多數情況下，這表示為您平台上想讓 TripPay 付款的物業建立帳戶。*
5. 將 [付款網頁元件](/zh-TW/developers/web-components/#payment) 整合到您的網站中。
6. 將旅客的 `購物車` 傳送給 TripPay 並啟動付款流程。
</Steps>

:::tip
欲取得使用 TripPay 為您的旅遊網站提供服務的完整指南，請參閱指南 [與 TripPay 整合](/zh-TW/guides/integrating-with-trip-pay)。
:::

## API

想管理 `付款` 的開發者，請前往 [開發者 > API > 付款](/zh-TW/developers/apis/#payment-api)。
