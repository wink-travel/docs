如果您是旅遊公司，並且希望將業務的付款和支出部分外包給 TripPay，請按照以下 6 個基本步驟開始操作：

<Steps>
  1. 創建一個 [使用者帳戶](/user/register) 在 Wink 上。
  2. 建立帳戶 [旅費](https://pay.wink.travel)。
  3. 創建一個 [應用](/developers/applications) 在 TripPay 上，您可以與我們的 API 對話。
  4. 使用 TripPay 建立並對應您的受益人。 *例如對大多數人來說，這意味著在您的平台上為您希望 TripPay 支付的房產建立帳戶。*
  5. 整合 [付款網路元件](/developers/web-components/#payment) 進入您的網站。
  6. 向 TripPay 發送旅行者的 `shopping cart` 並啟動付款流程。
</Steps>


：：：提示
有關在您的旅遊網站上使用 TripPay 的完整指南，請閱讀指南[與 TripPay 集成](/guides/integrating-with-trip-pay)。
:::
