:::note
除非您是整合商，否則您不必擔心在 TripPay 建立帳戶。整合商會在您與他們建立帳戶時為您完成此步驟。
:::

收款人使用 TripPay 來：

- 追蹤他們賺了多少錢。
- 目前帳戶中有多少金額。
- …以及可提領的金額。

## 使用方式

一般收款人通常會這樣使用 TripPay：

<Steps>
1. TripPay 從 3 個不同旅遊平台的 10 筆訂單中保留資金，針對 Hotel X，且資金現已可用。
2. Hotel X 收到 TripPay 寄來的電子郵件，內容為 `您有可用資金`，並附有管理帳戶的連結。
3. Hotel X 登入 [TripPay](https://pay.wink.travel)。
4. Hotel X 查看上個月的訂單及帳戶分析。
5. Hotel X 選擇提款。
6. Hotel X 選擇 `Pay-out` 付款方式。
7. Hotel X 按照指示完成資金處理步驟。
8. Hotel X 提交轉帳請求並點擊 `OK`。
</Steps>

*資金通常在所有地區 24 小時內可用。*

:::note
當您達到一定的交易量和金額時，我們可能會聯繫您，要求提供更多文件以驗證您的身份及/或公司資料。
:::
