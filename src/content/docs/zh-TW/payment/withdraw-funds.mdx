收款人使用 TripPay 來：

* 追蹤他們賺了多少錢。
* 他們的帳戶目前有多少錢。
* ……以及可以提領的金額是多少。

## 用法

一般收款人通常會與 TripPay 進行以下互動：

<Steps>
  1. TripPay 保留了來自 3 個不同旅遊平台的 10 個預訂的資金，用於 Hotel X，這些資金現在可用。
  2. X 飯店收到 TripPay 寄來的電子郵件，內容如下 `You have funds available` 並提供一個連結來管理他們的帳戶。
  3. 飯店 X 登入 [旅費](https://pay.wink.travel)。
  4. X 飯店可以查看上個月的預訂情況以及帳戶分析。
  5. X 酒店選擇提取資金。
  6. X 飯店選擇 `Pay-out` 方法。
  7. X 飯店依照所需步驟處理資金。
  8. X 飯店提交轉移請求並點擊 `OK`。
</Steps>


*所有地區的資金通常可在 24 小時內到帳。*
