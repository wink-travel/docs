訂房引擎有兩種版本，以支援聯盟行銷、物業管理公司及飯店的不同需求。

## 線上旅遊代理商

對於聯盟行銷來說，使用線上旅遊代理商版本最為合適。原因是該網站設計用於讓旅客瀏覽多個物業。

您可以在下方找到此應用程式的首頁。

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## 網路訂房引擎

對於飯店或物業管理公司來說，使用網路訂房引擎版本最為合適。這類使用者希望強調單一物業或多個管理物業。

您可以在下方找到此應用程式的首頁。

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE
</LinkButton>

## 自訂化

飯店和聯盟行銷可以透過自訂色彩調色盤來調整訂房引擎的 *外觀與感受*。

請在我們的開發者沙盒中測試自訂您的主題。

<LinkButton href="https://developer.wink.travel/web-component" variant="secondary" icon="external" target="_blank">
  主題設定
</LinkButton>


## 設定

大多數情況下，您不需要直接分享這兩個應用程式。我們已創建多種方式讓您分享旅遊庫存，您可以在 [Wink Studio](/studio/what-is-studio) 或透過 [WinkLinks](/link-manager/wink-links) 發現。

不過，如果您想手動連結到這些網站，請務必在 URL 中加入您的客戶 ID 和自訂 ID 作為參數。

您可以透過以下方式找到您的客戶 ID：

<Steps>
1. 登入 [Wink Extranet](/extranet/what-is-extranet) 或 [Wink Studio](/studio/what-is-studio)。
2. 點擊右上角的個人資料圖示，然後點選 [Applications](/developers/applications)。
3. 使用與您想操作的 Wink 帳戶相關聯的應用程式。
4. 複製該應用程式的 Client ID，並將其作為請求參數貼到 URL 中。範例：?client-id=MY_CLIENT_ID
</Steps>

### Extranet

您可以透過以下方式找到作為物業的自訂 ID：

<Steps>
1. 登入 [Wink Extranet](/extranet/what-is-extranet)。
2. 選擇您想操作的物業。
3. 從主選單點擊 `Account > Branding`
4. 複製 Customization ID，並將其作為請求參數貼到 URL 中。範例：?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

您可以透過以下方式找到作為聯盟行銷的自訂 ID：

<Steps>
1. 登入 [Wink Studio](/studio/what-is-studio)。
2. 選擇您想操作的帳戶。
3. 從主選單點擊 `Tools > Customization`
4. 複製您想使用的自訂項目的 Customization ID，並將其作為請求參數貼到 URL 中。範例：?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

最終您會得到類似以下的 URL：

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

如果您想直接前往某個飯店頁面，URL 會像這樣：

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
如果您未包含客戶 ID 和/或自訂 ID，Wink OTA 將會使用預設設定。
:::

## IBE

如果您是物業管理公司，您會想連結到 Wink IBE 首頁。首頁會顯示您指定的主題以及您所代表的 [物業](/studio/agency-representation)。

最終您會得到類似以下的 URL：

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

如果您是物業，您會想直接連結到 Wink IBE 物業登陸頁面。

該 URL 會像這樣：

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
如果您未包含客戶 ID 和/或自訂 ID，Wink IBE 將無法載入。這些參數為必填。
:::

## 範例：物業頁面

Wink 上的每個物業在任一網站上都有自己獨特的登陸頁面。所有外部連結都會導向這些登陸頁面之一。

該登陸頁面包含旅客做出明智選擇是否入住該物業所需的所有資訊。
