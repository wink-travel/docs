## 配置

大多數時候，不需要直接共享這兩個應用程式中的任何一個。我們創建了多種方式供您分享旅行清單，您可以在[Wink工作室](/studio/what-is-studio) 或透過[WinkLinks](/link-manager/wink-links)。

但是，如果您想手動連結到其中一個網站，請確保將您的客戶端 ID 和自訂 ID 作為參數新增至 URL。

您可以透過以下方式找到您的客戶端 ID：

<Steps>
  1. 登入 [Wink 外部網路](/extranet/what-is-extranet) 或者 [Wink工作室](/studio/what-is-studio)。
  2. 點擊右上角的個人資料圖標，然後點擊 [應用](/developers/applications)。
  3. 使用與您想要使用的 Wink 帳戶綁定的應用程式。
  4. 從該應用程式複製客戶端 ID，並將其貼上到 URL 上作為請求參數。例如：?client-id=MY\_CLIENT\_ID
</Steps>


### 外部網路

您可以透過以下方式找到您的自訂 ID 作為屬性：

<Steps>
  1. 登入 [Wink 外部網路](/extranet/what-is-extranet)。
  2. 選擇您想要處理的屬性。
  3. 在主導覽列中，按一下 `Account > Branding`
  4. 複製自訂 ID 並將其貼上到 URL 上作為請求參數。例如：?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### 工作室

您可以透過以下方式找到您的附屬自訂 ID：

<Steps>
  1. 登入 [Wink工作室](/studio/what-is-studio)。
  2. 選擇您想要使用的帳戶。
  3. 在主導覽列中，按一下 `Tools > Customization`
  4. 從您想要使用的自訂中複製自訂 ID，並將其貼上到 URL 上作為請求參數。例如：?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## 空中交通管制

最終得到的 URL 類似以下內容：

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

如果您想直接進入飯店頁面，URL 將如下所示：

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

：：：筆記
如果您不包含用戶端和/或設定 ID，Wink OTA 將預設使用預設配置。
:::

## 國際教育局

如果您是一家物業管理公司，您希望連結到 Wink IBE 主頁。主頁將顯示您指定的主題以及[您所代表的財產](/studio/agency-representation)。

最終得到的 URL 類似以下內容：

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

如果您是房地產公司，您希望直接連結到 Wink IBE 房地產登陸頁面。

該 URL 看起來類似以下內容：

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

：：：筆記
如果您不包含客戶端和/或設定 ID，Wink IBE 將不會載入。這些參數是必需的。
:::
