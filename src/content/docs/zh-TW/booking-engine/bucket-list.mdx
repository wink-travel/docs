在任何您看到 Wink 庫存 [卡片](/zh-TW/studio/cards) 的地方，卡片右上角都會有一個愛心，讓您將該庫存加入願望清單。

請依照以下步驟管理您的願望清單：

<Steps>
1. 點擊您想加入的庫存的 ♡ `愛心圖示`。
2. 如果您已登入，愛心圖示會變成紅色。
3. 如果您尚未登入，系統會引導您登入。
</Steps>

:::tip
再次點擊 ♥️ `愛心圖示` 可將其從清單中移除。
:::

您可以點擊右上角的個人資料圖示，然後點選 `願望清單` 按鈕來管理所有願望清單項目。
