如果您想更新 Wink 使用者帳戶的個人資料，請依照以下步驟操作：

<Steps>
1. 前往 [my.wink.travel](https://my.wink.travel)。
2. 點擊右上角的 `Sign-in / Register` 進行登入。
3. 登入後，點擊右上角的個人資料圖示，會出現選單。
4. 點擊 `Profile` 連結。
5. 按照指示更新您的 `電子郵件`、`名字` 和 `姓氏`。
6. 您也可以上傳新的 `個人頭像`。
7. 點擊 `Save`。
</Steps>

<Steps>
1. 您必須先登入我們的任一網站。
2. 點擊右上角的個人資料圖示，會出現選單。
3. 點擊 `Profile` 連結。
4. 按照指示更新您的 `電子郵件`、`名字` 和 `姓氏`。
5. 您也可以上傳新的 `個人頭像`。
6. 點擊 `Update`。
</Steps>

:::note
您新的個人頭像將會在您完全登出 Wink 後顯示。
:::
