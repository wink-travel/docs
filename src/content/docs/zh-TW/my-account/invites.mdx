Wink 使用者可以選擇讓您管理他們的物業、聯盟或旅行社帳戶。當發生此情況時，您將收到一封包含邀請請求的電子郵件。

import { Steps } from '@astrojs/starlight/components';

要查看您目前的 Wink 邀請，請依照以下步驟操作：

<Steps>
 1. 登入 [my.wink.travel](https://my.wink.travel)。
 2. 從左側邊欄點擊 `Invites`。
 3. 您將看到所有目前邀請的清單。
 4. 此區段會列出您所有的申請。
 5. 選擇 `Accept` 或 `Reject` 該邀請。
 6. 若您接受邀請，該帳戶將顯示在 `Accounts` 區段。
</Steps>
