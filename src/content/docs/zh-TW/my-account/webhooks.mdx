Webhooks 是一種讓您接收並回應來自 Wink 帳戶的即時事件的方式。

<Steps>
 1. 從 [Accounts](https://my.wink.travel) 選擇您想操作的帳戶。
 2. 點擊帳戶卡片底部的 `Actions`，並選擇 `Manage account`。
 3. 在接下來的頁面中點擊 `Webhooks` 標籤。
 4. 此區域會列出您所有的 webhooks。
 5. 選擇編輯現有的 webhook 或建立新的 webhook。
 </Steps>

## 範例

<Steps>
1. 在 Wink 上發生一筆訂單。
1. Wink 事件 `booking.created` 被觸發。
2. 您的系統正在積極監聽此事件。
3. 您的系統追蹤自己的分析數據，並使用此事件更新時間序列資料庫。
</Steps>

## 建立 webhook

:::caution
您需要先在 Wink 建立一個聯盟帳戶或在 TripPay 建立一個帳戶，才能建立 webhook，因為您將會把 webhook 綁定到特定帳戶。
:::

建立 webhook 的步驟如下：

<Steps>
1. 在 webhooks 頁面，點擊 `Create new webhook` 按鈕。
2. ***名稱*** 為您的 webhook 命名。*例如 Wink Stalker*
3. ***啟用*** 保持預設啟用狀態。
4. **實體名稱** 將您的 webhook 連結到您現有的帳戶之一。*例如 Cool Account*
5. **事件** 監聽您感興趣的事件。*例如 `booking.created`*
6. **Webhook URL** 輸入您將用來接收我們事件的 URL。*例如 `https://my.system.com/webhook/wink`*
7. 點擊 `Save` 按鈕繼續。
</Steps>

系統會將您導回 webhook 清單，您的新 webhook 已準備就緒。

## 更新 webhook

如果您想要：

- 停用您的 webhook。
- 監聽更多或更少的事件。
- 更改監聽的 webhook URL。

您需要更新您的應用程式。

更新現有 webhook 的步驟如下：

<Steps>
1. 點擊您想更新的 webhook 的 `Actions` 連結。
2. 點擊 Actions 下的 `Update` 按鈕。
3. 對 webhook 進行修改。
4. 點擊 `Save` 按鈕繼續。
</Steps>

系統會將您導回 webhook 清單。

## 移除 webhook

如果您不再需要該 webhook，可以將其移除。

移除 webhook 的步驟如下：

<Steps>
1. 點擊您想移除的 webhook 的 `Actions` 連結。
2. 點擊 Actions 下的 `Remove` 按鈕。
3. 點擊 `OK` 按鈕確認刪除。
</Steps>

系統會將您導回 webhook 清單。
