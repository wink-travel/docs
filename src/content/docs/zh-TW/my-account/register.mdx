請依照以下步驟操作：

<Steps>
1. 前往 [my.wink.travel](https://my.wink.travel)
2. 點擊右上角的 `登入 / 註冊`
3. 您將被導向登入表單。點擊表單欄位下方的 `註冊` 連結。
4. 填寫註冊表單並點擊 `註冊`。系統會將您重新導向至登入頁面。
5. 登入後，您將再次被導向「我的帳戶」，在此您可以建立您的第一個帳戶。
</Steps>

建立 Wink 使用者帳號可讓您存取平台上的所有功能。您可以建立無限的物業和聯盟帳戶。
