若要管理您的 WinkLinks 標籤，請導覽至`WinkLinks` 在主導覽列中，點選`Tags` 選項卡。

隨著內容的增長，您將需要開始組織其中的一些內容。我們創造了`tags` 為了這個目的。您可以根據需要使用任意數量的標籤對內容進行分組。

## 建立標籤

<Steps>
  1. 透過點擊建立新標籤 `Add a new tag` 按鈕。
  2. 輸入新標籤的名稱。 *例如柏林頂級咖啡館*
  3. 點選 `Save` 關聯。
</Steps>


## 更新標籤

<Steps>
  1. 首先點擊 `Edit` 按一下要修改的標籤旁邊的連結。
  2. 更新標籤的名稱。 *例如柏林最好的咖啡館*
  3. 點選 `Save` 關聯。
</Steps>


## 刪除標籤

：：：筆記
刪除標籤時，您也將從所有相關內容中刪除它。
:::

<Steps>
  1. 點選 `Remove` 您想要刪除的標籤旁邊的連結。
</Steps>


## API

想要管理的開發人員`WinkLinks` 可以前往[開發者 > API > WinkLinks](/developers/apis/#winklinks-api)。
