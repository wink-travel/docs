要管理您的 WinkLinks 標籤，請在主導航列中前往 `WinkLinks`，然後點擊 `Tags` 分頁。

隨著內容的增加，您會想開始組織部分內容。我們為此創建了 `tags`。您可以使用任意多的標籤來分組內容。

## 建立標籤

<Steps>
1. 點擊 `Add a new tag` 按鈕來建立新標籤。
2. 輸入新標籤的名稱。*例如：柏林頂級咖啡館*
3. 點擊 `Save` 連結。
</Steps>

## 更新標籤

<Steps>
1. 先點擊您想修改的標籤旁的 `Edit` 連結。
2. 更新標籤名稱。*例如：柏林最佳咖啡館*
3. 點擊 `Save` 連結。
</Steps>

## 移除標籤

:::note
移除標籤時，該標籤也會從所有相關內容中被移除。
:::

<Steps>
1. 點擊您想移除的標籤旁的 `Remove` 連結。
</Steps>

## API

想管理 `WinkLinks` 的開發者可以前往 [Developers > API > WinkLinks](/zh-TW/developers/apis/#winklinks-api)。
