若要管理 WinkLinks 上的內容，請確保您已選擇要在 Wink Studio 上使用的聯盟帳戶，然後導航至`WinkLinks` 在主導覽列中。

您的內容可在`Entries` 選項卡。

我們支援三種主要類型的內容：

* 連結到網頁的公開 URL。
* 透過 WinkLinks 上傳的檔案。
* 透過我們的創建的文本內容`Quotes` 特徵。

## 添加內容

以下是為 WinkLinks 新增內容的方法。根據`Entries` 選項卡，首先按一下`Add entry` 按鈕。

### 正常連結

：：：提示
該連結必須指向網頁。例如，您不能直接連結到 PDF。如果您想分享 PDF，您需要使用`Upload file` 特徵。
:::

以下是新增普通連結的步驟：

<Steps>
  1. 輸入您想要分享的連結。 *例如https://trvl.as/s/3xWCH*
  2. 一旦您輸入鏈接，Wink 就會尋找該網站的詳細資訊並填充您接下來看到的表單。
  3. 您可以選擇修改一些數據，例如網站名稱、標題和描述。
  4. 您也可以選擇上傳自訂圖像。
  5. 點選 `Save` 準備好繼續時，請按下按鈕。
</Steps>


### 上傳文件

:::警告\[要求]

* 影像大小不能超過 10Mb。
* 影片大小不能超過 50Mb。
* 我們支援所有可識別的圖像和視訊格式，包括 PDF。
  :::

:::note\[建議]

* 您的特色圖片尺寸為 2560w x 1440h（16x9 寬高比）。
* 所有其他影像均為 2560w x 1920h（4:3 寬高比）。
* 影像大小不應大於 5Mb。請在將圖像上傳至 Wink 之前對其進行壓縮。
  :::

以下是新增文件的步驟：

<Steps>
  1. 可以新增媒體：
     * 透過將圖像從電腦拖曳到視窗或點擊 `Browse`。
     * 透過貼上指向圖像的外部 URL。
     * 透過使用筆記型電腦攝影機。
     * 來自您的 Google Drive 帳戶。
     * 來自您的 Dropbox 帳號。
     * 來自 Shutterstock
     * 圖片來源：Gettyimages
     * 來自 iStock
     * 來自 Unsplash
  2. 文件上傳完成後，您可以添加更多元資料來描述您的文件，例如標題和描述。
  3. 點選 `Save` 準備好繼續時，請按下按鈕。
</Steps>


### 撰寫報價

：：：提示
引用是一種很好的方式，可以透過添加分隔符號來分隔長列表 \[...而且它還能讓你聽起來很聰明，知識淵博 😉]。
:::

以下是撰寫報價單的步驟：

<Steps>
  1. 輸入引文 *例如玫瑰是紅色的。紫羅蘭是藍色的…*。
  2. 輸入原作者的姓名。 *例如埃德蒙·史賓塞*
  3. 輸入您獲得報價的網站名稱。 *例如每日行情*
  4. 點選 `Save` 準備好繼續時，請按下按鈕。
</Steps>


## 更新內容

所有內容都可以透過點擊`Actions` 連結並選擇`Update`。您可以透過點擊`Remove` 按鈕。

## 刪除內容

所有內容都可以輕鬆修改選擇`Update`。您可以透過類似的方式刪除內容，先點擊`Actions` 鏈接，然後點擊`Remove` 按鈕。

## 內容排序

您可以透過點擊每個條目左下角的圖示並將其拖曳到您想要的位置來對內容進行排序。

## 分組內容

：：：提示
在建立條目之前建立標籤。否則，您將失去進度並必須重新開始。
:::

編輯條目時，您可以選擇新增一個或多個標籤以進行分組。捲動到表單底部，找到`Tags` 表單欄位。

當使用者在查看您的 WinkLinks 頁面時，可以選擇如何使用您的內容（清單或網格），並可以選擇一個或多個標籤來進一步過濾您的內容。

## API

想要管理的開發人員`WinkLinks` 可以前往[開發者 > API > WinkLinks](/developers/apis/#winklinks-api)。
