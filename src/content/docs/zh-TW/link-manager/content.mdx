您的內容可在 `Entries` 標籤下找到。

我們支援三種主要內容類型：

- 公開可用的 URL，連結至網頁。
- 透過 WinkLinks 上傳的檔案。
- 透過我們的 `Quotes` 功能建立的文字內容。

## 新增內容

以下說明如何新增內容至 WinkLinks。在 `Entries` 標籤下，先點擊 `Add entry` 按鈕。

### 一般連結

:::tip
連結必須指向網頁。例如，您不能直接連結到 PDF。如果您想分享 PDF，請使用 `Upload file` 功能。
:::

新增一般連結的步驟如下：

<Steps>
1. 輸入您想分享的連結。*例如 https://trvl.as/3xWCH*
2. 輸入連結後，Wink 會自動擷取該網站的詳細資料並填入接下來的表單。
3. 您可以選擇修改部分資料，如網站名稱、標題和描述。
4. 您也可以選擇上傳自訂圖片。
5. 準備好後，點擊 `Save` 按鈕繼續。
</Steps>

### 上傳檔案

:::caution[需求]
- 圖片大小不可超過 10Mb。
- 影片大小不可超過 50Mb。
- 我們支援所有認可的圖片和影片格式，包括 PDF。
:::

:::note[建議]
- 特色圖片建議尺寸為 2560w x 1440h（16:9 長寬比）。
- 其他圖片建議尺寸為 2560w x 1920h（4:3 長寬比）。
- 圖片大小應不超過 5Mb。請在上傳至 Wink 前先壓縮圖片。
:::

新增檔案的步驟如下：

<Steps>
1. 媒體可透過以下方式新增：
    - 從電腦拖曳圖片至視窗或點擊 `Browse`。
    - 貼上指向圖片的外部 URL。
    - 使用筆記型電腦的相機。
    - 從您的 Google Drive 帳戶。
    - 從您的 Dropbox 帳戶。
    - 從 Shutterstock。
    - 從 gettyimages。
    - 從 iStock。
    - 從 Unsplash。
2. 檔案上傳完成後，您可以新增更多描述檔案的元資料，如標題和描述。
3. 準備好後，點擊 `Save` 按鈕繼續。
</Steps>

### 撰寫引言

:::tip
引言是分隔長列表的好方法 [...而且還能讓您聽起來聰明且博學 😉]。
:::

撰寫引言的步驟如下：

<Steps>
1. 輸入引言 *例如 Roses are red. Violets are blue...*。
2. 輸入原作者名稱。*例如 Edmund Spenser*。
3. 輸入您取得引言的網站名稱。*例如 Daily Quotes*。
4. 準備好後，點擊 `Save` 按鈕繼續。
</Steps>

## 更新內容

所有內容都可以透過點擊 `Actions` 連結並選擇 `Update` 來輕鬆修改。您也可以透過點擊 `Remove` 按鈕以類似方式刪除內容。

## 刪除內容

所有內容都可以透過選擇 `Update` 來輕鬆修改。您也可以先點擊 `Actions` 連結，再點擊 `Remove` 按鈕以類似方式刪除內容。

## 內容排序

您可以點擊每個條目左下角的圖示，並拖曳至您想要的位置來排序內容。

## 內容分組

:::tip
請先建立標籤，再建立條目。否則，您將失去進度並必須重新開始。
:::

編輯條目時，您可以選擇新增一個或多個標籤以作分組用途。向下捲動表單至底部即可找到 `Tags` 表單欄位。

瀏覽您 WinkLinks 頁面的使用者，可以選擇他們想要的內容呈現方式（列表或網格），並可選擇一個或多個標籤以進一步篩選內容。

## API

想管理 `WinkLinks` 的開發者可以前往 [Developers > API > WinkLinks](/developers/apis/#winklinks-api) 。
