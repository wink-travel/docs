連結管理器讓你可以儲存並整理你喜歡的網址以便日後使用。你的瀏覽器書籤就是一種連結管理器。隨著社群媒體的興起，讓連結能夠被朋友和觀眾看到並分享的需求也越來越大。你的 Facebook 動態消息就是一種社群友善的連結管理器。

我們打造了 [WinkLinks](https://i.trvl.as/) 作為旅遊產業的連結管理器。它能提供資訊、激發興趣、做廣告……並讓你從全球各地的飯店預訂超值優惠。

## 功能

- WinkLinks 讓你免費加入所有現有的聯盟行銷連結，如 Commission Junction 和 Amazon。
- 你會獲得一個酷炫的專屬網址，例如 [https://i.trvl.as/bob](https://i.trvl.as/bob)，適合只能分享單一網址的時候使用。
- 你可以整理並分享連結，還能上傳檔案。
- 飯店、連鎖品牌可以使用 WinkLinks 作為進階訂房引擎：
    - 支援多個物業。
    - 支援多個目的地。
    - 支援多種貨幣。
    - 支援多種語言。
- WinkLinks 支援 `豐富內容` *(見下方)*。
- 從 IG 到訂房只需 2 次點擊 🚀

## 豐富內容

:::note
在你的 [設定](/zh-TW/link-manager/settings) 中將 `Intelligent` 設為 🟢 即可使用此功能。
:::

如果你分享的連結來自支援 `rich` / `oEmbed` 內容的網站，該連結在 WinkLinks 上會變得生動活潑。

支援的網站：

- Facebook - *可互動貼文並播放影片*。
- Instagram - *可互動貼文並播放影片*。
- YouTube - *播放影片*。
- X (前身為 Twitter) - *可互動貼文*。
- Spotify - *播放你喜愛的播放清單*。
- Wink - *請參見 `進階`*。

## 進階

WinkLinks 也是展示你透過 [Wink Studio](https://studio.wink.travel) 找到的旅遊庫存，並想向觀眾推廣的絕佳方式。

使用 WinkLinks 銷售你的：

- [可分享連結](/zh-TW/studio/shareable-links) - 可選擇顯示為：
    - [廣告橫幅](/zh-TW/developers/web-components/#content-loader)。
    - 一般連結，帶有即時價格更新。
- [卡片](/zh-TW/studio/cards) - 可訂房的卡片，帶有即時價格與可用性。
- [格子](/zh-TW/studio/grids) - 含可訂房卡片的格子。
- [地圖](/zh-TW/studio/maps) - 含標記並可開啟可訂房卡片的地圖。
