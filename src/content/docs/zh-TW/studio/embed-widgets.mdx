Studio 提供多種可嵌入的小工具，您可以將它們加入自己的網站以促進預訂。

要存取嵌入小工具，請從側邊欄導航至 `Tools > Miscellaneous`。

## 可用的小工具

| 小工具 | 說明 |
| -- | -- |
| **Account button** | 連結至您的聯盟頁面的品牌按鈕 |
| **Itinerary button** | 開啟行程選擇器的按鈕 |
| **Itinerary form** | 內嵌的日期與旅客選擇表單 |
| **Search button** | 開啟物業搜尋的按鈕 |
| **Lookup button** | 用於查詢特定庫存的按鈕 |

## 如何嵌入

每個小工具都提供一段程式碼片段，您可以複製並貼到網站的 HTML 中。該片段包含您的 **Client ID** 和 **Customization ID**，讓小工具自動連結到您的聯盟帳戶。

:::note
如果您使用 WordPress，我們也提供 WordPress 外掛以便更輕鬆整合。
:::
