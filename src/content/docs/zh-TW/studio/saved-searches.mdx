已儲存的搜尋正如其名。在搜尋旅遊庫存時，您可以選擇將搜尋查詢儲存以便日後使用。

## 儲存您的搜尋查詢

當您開始搜尋庫存（請參閱 [搜尋](/zh-TW/studio/search)）時，您可以微調搜尋篩選條件。
在查詢表單底部有一個選項是 `儲存搜尋`。

輸入名稱後，點擊 `儲存搜尋` 按鈕。之後，您可以從主選單導覽至 `庫存 > 已儲存的搜尋` 查看您新建立的紀錄。

## 清單操作

您的已儲存搜尋可用來以網格或地圖形式向使用者展示旅遊庫存，並讓您的搜尋結果以您想要的方式顯示。

    <figcaption>*未套用篩選條件的已儲存搜尋查詢*</figcaption>

每個已儲存的搜尋查詢都有可執行的操作：

- 將搜尋查詢轉換為 [地圖](/zh-TW/studio/maps)。
- 將搜尋查詢轉換為 [網格](/zh-TW/studio/grids)。
- 移除已儲存的搜尋。

## API

想管理已儲存搜尋的開發者可以前往 [開發者 > API > 瀏覽](/zh-TW/developers/apis/#browse-api)。
