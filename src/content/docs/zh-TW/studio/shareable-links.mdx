可共享連結是 Wink 的核心功能，它源自於透過網路上最簡單的方式以動態方式傳達庫存和定價的需求；即超連結。

<h4 align="center">[https://trvl.as/s/3xWCH](https://trvl.as/s/3xWCH)</h4>


以上是我們的一個連結的範例。它簡短且便於攜帶...甚至短到可以記住。雖然很短，但這個連結包含**霹靂** 資訊量，例如：

* 您的應用程式 ID。
* 您的會員 ID。
* 向使用者顯示的圖像和文字。
* 外觀和感覺（*例如主題、顏色等等*）。
* 用於價格查詢的行程（*2 位成人，1 晚*）。
* 顯示價格的貨幣。
* 顯示文字的語言。

本文的其餘部分將引導您了解如何建立、自訂和分享您的連結。

## 建立連結

<figure>
  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="行動" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*搜尋結果操作*</figcaption>
</figure>


上面的圖片取自[搜尋](/studio/search) 並向您展示您可以對搜尋結果執行的一些操作。其中一項行動是`Make a link`。
點擊該按鈕，它將把您重定向到我們的可共享連結表單頁面，您可以在其中開始自訂您的連結。

## 自訂連結

該表單允許您透過以下方式自訂連結：

<Steps>
  1. 給它一個名字，以便您記住該連結的內容。
  2. 選擇 [客製化](/studio/customization) 您想申請此連結。
  3. 新增關鍵字，以逗號分隔，用於 [網路爬蟲](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)。
  4. 可選擇輸入您的 X 帳號句柄或 Facebook App ID。
  5. 使用您希望用戶看到的語言添加標題和描述。
  6. 選擇一張或多張將在 Facebook 或 LinkedIn 等平台上分享連結時顯示的圖片。
  7. 選擇一個主題/覆蓋，在您選擇的圖像上添加品牌設計。 *注意：這是讓價格顯示在圖像頂部的方法。*
  8. 點選 `Save` 按鈕繼續。
</Steps>

儲存連結後，您將被重新導向到可分享連結頁面，現在可以與全世界分享您的連結了。

## 分享連結

<figure>
  <CloudinaryImage width="350" height="641" resize="scale" quality="best" alt="行動" src="academy/shareable-link-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*可共享連結操作*</figcaption>
</figure>


上面顯示了您的連結的所有可用操作的圖像：

* **預覽** 向您展示連結的配置方式以及連結在使用者看來的樣子的簡要概述。
* **更新** 更新您的連結配置。
* **加入 WinkLinks** 將連結新增至您的 WinkLinks 帳戶。
* **複製** 將連結複製到剪貼板，以便您可以輕鬆地在任何您想要的地方分享它（*包括[WinkLinks](/link-manager/wink-links)*）。
* **分享** 打開一個新窗口，其中提供在許多社交網路和訊息平台上共享的選項。
* **QR 圖碼** 產生一個二維碼，您可以在 IG 等更注重圖像的平台上分享。
* **嵌入** 向您展示如何將此連結嵌入為[廣告橫幅](/developers/web-components/#content-loader) 進入您的網站。
* **與 WordPress 一起使用** 向您展示如何使用我們的[WordPress 外掛](/developers/wordpress) 將此連結嵌入到您的網站。

下面我們將更詳細地介紹其中一些選項。

### 預覽

<figure>
  <CloudinaryImage width="700" height="373" resize="scale" quality="best" alt="預覽" src="academy/shareable-link-preview" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*可共享連結預覽*</figcaption>
</figure>


當你點擊`Preview`，您將看到應用了您的主題的圖像。當您在社交媒體（例如 Facebook）上分享連結時，您可以期待在您的動態中看到相同的內容以及標題和描述。

### 社群分享

<figure>
  <CloudinaryImage width="350" height="231" resize="scale" quality="best" alt="社群分享" src="academy/shareable-link-share" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*社交分享選項*</figcaption>
</figure>


如果您點擊`Share` 按鈕，將打開一個新窗口，您可以透過該窗口方便地直接分享到您最喜歡的社交網路和訊息應用程式。
點擊任何圖示按鈕即可繼續在該網路上發布您的連結。 *注意：某些按鈕僅當您從行動裝置共用時才會顯示。*

### QR 圖碼

<figure>
  <CloudinaryImage width="350" height="440" resize="scale" quality="best" alt="社群分享" src="academy/shareable-link-qr-code" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*透過二維碼分享*</figcaption>
</figure>


如果您想分享二維碼而不是鏈接，請點擊`QR code` 按鈕和新視窗將會出現您的二維碼。

如何儲存您的二維碼：

* 如果您使用的是筆記型電腦，請右鍵單擊圖像並選擇`Save image`。
* 如果您正在使用手機：
  * 按下並按住影像上的手指，直到出現將影像儲存到照片的選項。
  * 截取二維碼的螢幕截圖，裁剪以適應大小，然後儲存到照片中。

### 嵌入

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

以下是如何將您的連結作為廣告橫幅嵌入到您的網站：

* 第 3 行向您展示如何將 Wink 樣式嵌入到您的網站。
* 第 6 至 9 行展示如何使用[wink-內容載入器](/developers/web-components/#content-loader) Web 元件並告訴它為您的程式碼取得廣告橫幅。
* 第 11 行向您展示如何將我們的 Javascript 嵌入到您的網站。
* 第 13 行展示如何嵌入[wink-app-loader](/developers/web-components/#application-loader) Web 元件並告訴它要取得您的頁面層級配置首選項。
