可分享連結是 Wink 的核心功能之一，源自於需要以網路上最簡單的方式——超連結，動態地傳達庫存和價格資訊。

<h4 align="center">[https://trvl.as/3xWCH](https://trvl.as/3xWCH)</h4>

上方是我們其中一個連結的範例。它短小且便於攜帶……甚至短到容易記憶。雖然短小，這個連結卻包含了**爆炸性**的資訊量，例如：

- 您的應用程式 ID。
- 您的聯盟 ID。
- 顯示給使用者的圖片和文字。
- 外觀風格（*例如主題、顏色等*）。
- 用於價格查詢的行程（*2 位成人，1 晚*）。
- 顯示價格的貨幣。
- 顯示文字的語言。

本文接下來將引導您如何建立、自訂及分享您的連結。

## 自訂連結

該表單允許您以以下方式自訂連結：

<Steps>
 1. 為連結命名，方便您記住連結內容。
 2. 選擇您想套用於此連結的[自訂設定](/studio/customization)。
 3. 新增以逗號分隔的關鍵字，供[網路爬蟲](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)使用。
 4. 選擇性輸入您的 X 帳號名稱或 Facebook 應用程式 ID。
 5. 以您希望使用者看到的語言新增標題和描述。
 6. 選擇一張或多張圖片，當您在 Facebook 或 LinkedIn 等平台分享連結時會顯示。
 7. 選擇一個主題／覆蓋層，為您選擇的圖片加上品牌設計。*注意：這是讓價格顯示在圖片上的方式。*
 8. 點擊 `儲存` 按鈕繼續。
 </Steps>

上圖展示了您連結可用的所有操作：

- **預覽** 快速查看連結設定及使用者看到的樣貌。
- **更新** 更新您的連結設定。
- **加入 WinkLinks** 將連結加入您的 WinkLinks 帳號。
- **複製** 將連結複製到剪貼簿，方便您在任何地方分享（*包括 [WinkLinks](/link-manager/wink-links)*）。
- **分享** 開啟新視窗，提供多種社群網路和通訊平台的分享選項。
- **QR 碼** 產生 QR 碼，方便您在以圖片為主的平台如 IG 分享。
- **嵌入** 顯示如何將此連結以[廣告橫幅](/developers/web-components/#content-loader)形式嵌入您的網站。
- **WordPress 使用** 顯示如何使用我們的[WordPress 外掛](/developers/wordpress)將此連結嵌入您的網站。

如果您想分享 QR 碼而非連結，點擊 `QR 碼` 按鈕，會開啟新視窗顯示您的 QR 碼。

如何儲存您的 QR 碼：

- 如果您使用筆記型電腦，右鍵點擊圖片並選擇 `儲存圖片`。
- 如果您使用手機：
    - 長按圖片直到出現選項，選擇將圖片儲存到相簿。
    - 截圖 QR 碼，裁切合適後儲存到相簿。

### 嵌入

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

以下說明如何將您的連結以廣告橫幅形式嵌入網站：

- 第 3 行示範如何將 Wink 樣式嵌入您的網站。
- 第 6 至 9 行示範如何使用 [wink-content-loader](/developers/web-components/#content-loader) Web 元件，並告訴它抓取您的廣告橫幅。
- 第 11 行示範如何將我們的 Javascript 嵌入您的網站。
- 第 13 行示範如何使用 [wink-app-loader](/developers/web-components/#application-loader) Web 元件，並告訴它抓取您的頁面層級設定偏好。
