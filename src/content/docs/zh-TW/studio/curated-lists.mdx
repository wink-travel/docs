精選清單是一項功能，可讓您對在搜尋旅行庫存時找到的物品中選擇的庫存進行分組和排序。
創建後，您可以將列表作為[網格](/studio/grids) 或[地圖](/studio/maps) 透過[WinkLinks](/link-manager/wink-links)， 我們的[WordPress 外掛](/developers/wordpress/) 或嵌入我們的[Web 元件](/developers/web-components) 進入您的網站。

預設情況下，每個聯盟帳戶都以`Favorites` 您可以立即使用的清單。 `Favorites` 是特殊的並且不能被刪除。
所有精選清單都允許您執行以下操作：

## 創建列表

<figure>
  <CloudinaryImage width="350" height="219" resize="scale" quality="best" alt="建立精選列表" src="academy/create-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*建立新的精選清單表單*</figcaption>
</figure>


* 在儀表板上，點擊`Inventory > Curated Lists` 在導覽列中。
* 透過點擊建立一個新的精選列表`Make a new list` 按鈕。
* 您將被要求命名您的清單。 *例如：紐約市前 3 名精選*。
* 點選`Save` 按鈕。

## 新增列表項

您可以將清單項目從搜尋結果新增到任何清單。

* 前往[搜尋](/studio/search)
* 搜尋您想要新增到清單中的內容。
* 找到後，點擊`Actions` 搜尋結果卡左下角的連結。
* 點選`Add to...` 然後選擇要新增此庫存的精選清單。
* 返回精選清單頁面以驗證該項目是否已新增至您的清單。

## 重新排序列表項

<figure>
  <CloudinaryImage width="350" height="183" resize="scale" quality="best" alt="重新排序精選列表項" src="academy/reorder-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*重新排序精選列表項*</figcaption>
</figure>


您可以透過點擊並拖曳清單中每個項目左側的垂直點來對清單項目進行排序。

## 複製列表項

<figure>
  <CloudinaryImage width="450" height="178" resize="scale" quality="best" alt="複製到精選列表項" src="academy/copy-to-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*複製精選列表項*</figcaption>
</figure>


您可以透過點擊複製圖示並選擇要將項目複製到的清單來複製清單項目。

## 移動列表項

<figure>
  <CloudinaryImage width="450" height="173" resize="scale" quality="best" alt="移動精選列表項" src="academy/move-to-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*移動精選列表項*</figcaption>
</figure>


您可以透過點擊複製圖示並選擇要將項目移至的清單來移動清單項目。

## 刪除清單項目

<figure>
  <CloudinaryImage width="350" height="216" resize="scale" quality="best" alt="刪除精選清單項目" src="academy/remove-curated-list-item" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*刪除精選清單項目*</figcaption>
</figure>


您可以透過點擊垃圾桶圖示並確認刪除來刪除清單項目。

## 列出操作

<figure>
  <CloudinaryImage width="350" height="183" resize="scale" quality="best" alt="精選列表操作" src="academy/curated-list-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*精選列表操作*</figcaption>
</figure>


每個清單都有一些您可以執行的操作。

* 將列表轉換為[地圖](/studio/maps)。
* 將列表轉換為[網格](/studio/grids)。
* 刪除清單（*除了收藏夾*）。

## API

想要管理精選清單的開發者可以前往[開發人員 > API > 瀏覽](/developers/apis/#browse-api)。
