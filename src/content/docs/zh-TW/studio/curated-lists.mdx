精選清單是一項功能，讓您將從旅遊庫存搜尋中挑選出的項目進行分組和排序。
建立後，您可以透過 [WinkLinks](/zh-TW/link-manager/wink-links)、我們的 [WordPress 外掛](/zh-TW/developers/wordpress/) 或將我們的 [Web Components](/zh-TW/developers/web-components) 嵌入您的網站，將清單以 [網格](/zh-TW/studio/grids) 或 [地圖](/zh-TW/studio/maps) 形式提供給使用者銷售。

預設情況下，每個聯盟帳號都會有一個 `Favorites` 清單，您可以立即使用。`Favorites` 是特殊清單，無法刪除。
所有精選清單都允許您執行以下操作：

## 建立清單

<CloudinaryImage
      width="350"
      height="219"

    <figcaption>*建立新精選清單表單*</figcaption>
</figure>

- 在儀表板中，點選導覽列的 `Inventory > Curated Lists`。
- 點擊 `Make a new list` 按鈕建立新的精選清單。
- 系統會要求您為清單命名。*範例：Top 3 NYC Picks*。
- 點擊 `Save` 按鈕。

## 新增清單項目

您可以從搜尋結果中將項目新增到任何清單。

- 前往 [Search](/zh-TW/studio/search)
- 搜尋您想加入清單的項目。
- 找到後，點擊搜尋結果卡片左下角的 `Actions` 連結。
- 點擊 `Add to...`，然後選擇您想加入該庫存的精選清單。
- 回到精選清單頁面，確認項目已加入清單。

每個清單都可以執行以下操作：

- 將清單轉換為 [地圖](/zh-TW/studio/maps)。
- 將清單轉換為 [網格](/zh-TW/studio/grids)。
- 移除清單（*Favorites 除外*）。

## API

想管理精選清單的開發者可以前往 [Developers > API > Browse](/zh-TW/developers/apis/#browse-api)。
