代理機構是您可以指派給自己帳戶以進行代表的普通 Wink 帳戶。以下是您可能希望由代理機構代表您的幾個原因：

- 由第三方對您的 Wink 帳戶進行全部或部分管理。
- 藝人與其代理機構已有現有協議，並希望 Wink 處理通過 Wink 進入資金的財務部分。
- 藝人代理機構負責所有線上工作並管理 Wink 上的所有帳戶。

## 指派代理機構

以下是如何將代理機構指派給您的帳戶。

<Steps>
 1. 登入您的使用者帳戶，進入 [Wink Studio](https://studio.wink.travel)。
 2. 選擇您想合作的附屬帳戶。
 3. 從主導覽列點選 `Account > Managers`。
 4. 在 `Agency` 區塊，輸入您想指派代表您的代理機構名稱。
 5. 從下拉選單中選擇代理費用。
 5. 可選擇新增到期規則。
 7. 點擊 `Set agency` 按鈕繼續。
 </Steps>

 :::note
您必須是該帳戶的擁有者才能指派代理機構。
 :::

 :::tip[範例]
代理費用是從附屬佣金中計算得出。

訂單金額 = $100，佣金 = 10%，代理費用 = 10% => 100 * 0.1 * 0.1 = $1
 :::

## 取消指派代理機構

以下是如何從您的帳戶取消指派代理機構。

<Steps>
 1. 登入您的使用者帳戶，進入 [Wink Studio](https://studio.wink.travel)。
 2. 選擇您想合作的附屬帳戶。
 3. 從主導覽列點選 `Account > Managers`。
 4. 在 `Agency` 區塊，點擊代理機構下拉選單右側的 <Badge text="x" variant="danger" size="small" /> 圖示。
 5. 點擊 `Set agency` 按鈕繼續。
 </Steps>
