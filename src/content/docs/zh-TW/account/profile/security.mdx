**Security** 分頁位於 [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security)，您可以在此管理 **通行金鑰** — WebAuthn 憑證，讓您無需密碼即可登入。

## 什麼是通行金鑰？

通行金鑰是一種綁定於您的裝置（或密碼管理器）的加密憑證，用以證明您的身份。通行金鑰具備防釣魚功能，且比輸入密碼更快速。

## 註冊通行金鑰

1. 使用現有憑證登入 [iam.wink.travel](https://iam.wink.travel)。
2. 前往 **我的帳戶 → 安全性**。
3. 點擊 `新增通行金鑰`。
4. 您的瀏覽器會提示您使用裝置進行驗證（Touch ID、Face ID、Windows Hello、硬體金鑰或已同步的密碼管理器）。
5. 為通行金鑰命名（例如 *工作用 MacBook*），以便日後辨識。

## 使用通行金鑰登入

在 [iam.wink.travel](https://iam.wink.travel) 登入畫面輸入您的電子郵件 — 若該電子郵件已註冊通行金鑰，系統將提示您使用通行金鑰驗證，而非輸入密碼。

## 移除通行金鑰

在同一個安全性分頁中，每個已註冊的通行金鑰旁都有 `移除` 按鈕。若您遺失裝置、淘汰硬體金鑰或想清理未使用的憑證，請使用此功能。

:::caution
移除所有通行金鑰後，密碼將成為唯一的登入方式。請確保在移除最後一把通行金鑰前記得您的密碼。
:::
