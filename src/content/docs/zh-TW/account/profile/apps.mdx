在 [iam.wink.travel/my-account/apps](https://iam.wink.travel/my-account/apps) 的 **Apps** 分頁中，列出您授權存取 Wink 帳戶的所有 OAuth2 應用程式，並提供撤銷該存取權的方式。

## 應用程式如何出現在此處

當您在整合 Wink 的第三方工具（或 Wink 自家的應用程式）上點選 `Authorize` 時，授權流程會記錄該授權。從此之後，該應用程式即可在您核准的範圍內代表您操作。

## 檢視存取權

每筆資料會顯示：

- **應用程式名稱** 及簡短描述。
- 應用程式可存取的 **範圍**（例如：讀取庫存、管理訂單）。
- **最後使用時間** — 該應用程式最近一次代表您呼叫 Wink API 的時間。

## 撤銷應用程式

點選任一項目旁的 `Revoke`。該應用程式的權杖會立即失效，且它針對您帳戶執行的任何背景工作將開始出現 401 錯誤。

:::tip
撤銷不具破壞性 — 您可隨時重新授權該應用程式，只需再次完成其授權流程。當應用程式行為異常、您要更換憑證，或不再使用該整合時，請使用此功能。
:::

## 您自行開發的 OAuth 應用程式

如果您想要 *建立* OAuth 應用程式（不只是授權），這是組織帳戶的功能 — 請參閱 [設定 → 應用程式](/zh-TW/portal/settings/applications)。
