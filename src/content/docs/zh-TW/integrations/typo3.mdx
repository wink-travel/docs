**Wink TYPO3 擴充功能** 將 Wink 旅遊庫存整合到 TYPO3 13 作為原生內容元素。編輯者可使用標準 TYPO3 頁面編輯器將 Wink 元件新增至頁面 — 無需更改範本。

## 系統需求

- TYPO3 13.x
- PHP 8.2+
- Composer

## 安裝

<Steps>
1. 使用 Composer 安裝：
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. 啟用擴充功能：
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. 在 TYPO3 後台，前往 **管理工具 → 設定 → 擴充功能設定 → wink**。
4. 輸入您的 **Client ID**、**Client Secret**，並選擇您的環境。儲存設定。
   請至 [應用程式](/zh-TW/portal/settings/applications) 取得您的憑證。
5. 將靜態 TypoScript 範本加入您的根範本：
   **範本 → 編輯 → 包含 → 可用項目 → Wink Travel**。
</Steps>

## 新增內容元素至頁面

<Steps>
1. 在 **頁面** 模組中開啟一個頁面。
2. 點擊內容區的 **+** 按鈕新增元素。
3. 選擇其中一個 **Wink Travel** 內容元素：
   - **Wink Content** — 飯店卡片、排名網格、地圖
   - **Wink Lookup** — 目的地搜尋列
   - **Wink Search** — 行程選擇按鈕
   - **Wink Account** — 登入按鈕
   - **Wink Itinerary** — 行程狀態按鈕
   - **Wink Cart** — 購物車按鈕
4. 對於 **Wink Content**，使用 FlexForm 欄位選擇 **版面類型** 並輸入 **版面 ID**。
5. 儲存並發佈頁面。
</Steps>

## 可用內容元素

| 元素 | 功能說明 |
|---|---|
| **Wink Content** | 飯店卡片、排名網格、地圖等 |
| **Wink Lookup** | 目的地搜尋列 |
| **Wink Search** | 開啟行程選擇器的圖示按鈕 |
| **Wink Account** | 登入 / 使用者帳戶按鈕 |
| **Wink Itinerary** | 顯示目前行程的行程按鈕 |
| **Wink Cart** | 購物車摘要按鈕 |
