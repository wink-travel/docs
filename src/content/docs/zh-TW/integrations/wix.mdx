**Wink Wix App** 將 Wink 網頁元件作為原生小工具添加到您的 Wix 網站，您可以在 Wix 編輯器中拖放使用。網站擁有者無需編碼。

## 安裝

<Steps>
1. 前往 [Wix App Market](https://www.wix.com/app-market) 並搜尋 **Wink Travel**。
2. 點擊 **Add to Site** 並依照提示授權應用程式。
3. 安裝完成後，打開 **Dashboard → Wink Settings**，輸入您的 **Client ID** 和 **Client Secret**。
   前往 [Applications](/zh-TW/portal/settings/applications) 取得您的憑證。
4. 點擊 **Save**，然後點擊 **Test Connection** 以確認 Wink 已連接。
</Steps>

## 在頁面中新增元件

<Steps>
1. 開啟 **Wix Editor**，前往您想新增 Wink 內容的頁面。
2. 點擊 **Add (+) → Apps → Wink Travel**，查看所有可用的小工具。
3. 將小工具拖曳到頁面上 — 例如，使用 **Wink Content** 顯示飯店網格。
4. 在小工具設定面板中，選擇 **Layout Type** 並貼上來自 Wink Studio 的 **Layout ID**。
5. 點擊 **Publish** 使變更生效。
</Steps>

## 可用小工具

| 小工具 | 功能說明 |
|---|---|
| **Wink Content** | 顯示飯店卡片、排名網格、地圖等 |
| **Wink Lookup** | 目的地與飯店搜尋欄 |
| **Wink Search** | 開啟行程選擇器的圖示按鈕 |
| **Wink Account** | 登入 / 使用者帳戶按鈕 |
| **Wink Itinerary** | 顯示當前行程的行程按鈕 |
| **Wink Cart** | 購物車摘要按鈕 |
