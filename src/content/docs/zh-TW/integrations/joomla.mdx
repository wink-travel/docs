**Wink Joomla 擴充套件** 是一個套件，安裝系統外掛（載入 CDN 腳本）、內容外掛（處理文章中的短碼標籤）及元件（管理設定頁面）。

/>

## 需求

- Joomla 4 或 5
- PHP 8.1+

## 安裝

<Steps>
1. 從 [GitHub 版本發行頁面](https://github.com/wink-travel/wink-joomla-extension/releases) 下載最新的 `pkg_wink-x.x.x.zip`。
2. 在 Joomla 管理後台，前往 **Extensions → Manage → Install**。
3. 上傳 zip 檔案並點擊 **Upload & Install**。
4. 兩個外掛會自動啟用。前往 **Components → Wink Settings**。
5. 輸入您的 **Client ID** 和 **Client Secret**，然後點擊 **Save**。
   請至 [Applications](/zh-TW/portal/settings/applications) 取得您的憑證。
</Steps>

## 在文章中嵌入元件

在文章內容中任何位置使用短碼風格標籤：

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

內容外掛會在渲染時將這些標籤替換成正確的 `<wink-*>` HTML。系統外掛會自動載入 CDN 腳本並在頁腳注入 `<wink-app-loader>`。

## 可用短碼

| 短碼 | 輸出 | 說明 |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | 飯店卡片、格線、地圖 |
| `{wink-lookup}` | `<wink-lookup>` | 目的地搜尋列 |
| `{wink-search}` | `<wink-search-button>` | 行程選擇按鈕 |
| `{wink-account}` | `<wink-account-button>` | 登入 / 使用者帳戶 |
| `{wink-itinerary}` | `<wink-itinerary-button>` | 行程狀態按鈕 |
| `{wink-cart}` | `<wink-shopping-cart-button>` | 購物車按鈕 |
