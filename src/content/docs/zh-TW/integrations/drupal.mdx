**Wink Drupal 模組** 將 Wink 旅遊庫存元件作為原生區塊新增到您的 Drupal 網站。編輯者可以透過標準區塊系統將它們放置在版面中的任何位置。

<LinkCard

## 系統需求

- Drupal 10 或 11
- PHP 8.1+
- Composer

## 安裝

<Steps>
1. 使用 Composer 安裝：
   ```bash
   composer require drupal/wink
   ```
2. 啟用模組：
   ```bash
   drush en wink -y
   ```
3. 前往 **管理 → 設定 → 網路服務 → Wink 設定** (`/admin/config/wink`)。
4. 輸入您的 **Client ID** 和 **Client Secret**，選擇您的環境，然後點擊 **儲存**。
   請至 [應用程式](/zh-TW/portal/settings/applications) 取得您的憑證。
</Steps>

## 新增區塊到頁面

<Steps>
1. 前往 **管理 → 結構 → 區塊佈局**。
2. 在您想放置 Wink 內容的區域點擊 **放置區塊**。
3. 搜尋 **Wink** 以找到可用的區塊：
   - **Wink Content** — 顯示飯店卡片、格狀或地圖
   - **Wink Lookup** — 目的地搜尋列
   - **Wink Search** — 行程選擇按鈕
   - **Wink Account** — 登入按鈕
   - **Wink Itinerary** — 行程狀態按鈕
   - **Wink Cart** — 購物車按鈕
4. 對於 **Wink Content**，在區塊設定中配置 **版面類型** 和 **版面 ID**。
5. 儲存區塊設定。
</Steps>

## 可用區塊

| 區塊 | 功能說明 |
|---|---|
| **Wink Content** | 顯示飯店卡片、排名格狀、地圖等 |
| **Wink Lookup** | 目的地與飯店搜尋列 |
| **Wink Search** | 開啟行程選擇器的圖示按鈕 |
| **Wink Account** | 登入 / 使用者帳戶按鈕 |
| **Wink Itinerary** | 顯示目前行程的行程按鈕 |
| **Wink Cart** | 購物車摘要按鈕 |
