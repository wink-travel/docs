當您在 Extranet 為物業新增庫存 — 包括客房、餐廳、水療中心、會議室、活動等 — 該庫存即會在 Wink 生態系統中可用。本指南說明各部分如何連接。

## 庫存流程

```
Extranet (您建立) → 訂房引擎 (旅客瀏覽) → Studio (聯盟銷售)
```

1. **您在 Extranet 建立庫存** — 客房類型、設施和體驗。
2. **旅客在訂房引擎瀏覽** 該庫存，透過可分享連結、卡片或直接搜尋。
3. **聯盟在 Studio 發現** 您的庫存，建立卡片和網格，並銷售給他們的受眾。

## 客房類型

客房類型是您庫存的核心。每個客房類型需要：

- 名稱和描述。
- 床型配置。
- 最大入住人數（成人和兒童）。
- 客房專屬設施。
- 照片。
- 至少一個含主價的價格方案。

若無價格，客房類型存在但無法預訂。

## 設施和體驗

除了客房，您還可以列出：

| 庫存類型 | 定義 | 範例 |
| -- | -- | -- |
| **餐廳** | 現場用餐 | 「屋頂酒吧與燒烤」 |
| **水療中心** | 健康服務 | 「寧靜水療 - 60分鐘按摩」 |
| **會議室** | 活動與會議空間 | 「董事會議室 A - 20人」 |
| **活動** | 物業內或附近的活動 | 「導覽浮潛之旅」 |
| **景點** | 重要附近地點 | 「國家博物館 - 500公尺」 |
| **地點** | 有趣地點 | 「當地市場」 |
| **附加項目** | 可與客房一同購買的額外服務 | 「機場接送」 |

每種設施類型在 Extranet 有獨立區塊。聯盟可單獨銷售這些項目 — 旅客可能透過水療中心列表而非客房搜尋發現您的物業。

## 聯盟如何使用您的庫存

在 Studio，聯盟可以：

- **搜尋** 您的物業或單一庫存項目。
- **建立卡片** — 單一可預訂的客房、水療療程或活動卡片。
- **建立網格** — 一組卡片一起展示。
- **建立地圖** — 以位置為基礎的標記，顯示可預訂庫存。
- **分享連結** — 直接連結到您的物業或特定項目。

您的庫存越完整多元，聯盟推廣物業的方式就越多。

## 小提示

- **詳盡列出** — 列出物業提供的所有項目。每個項目都是潛在的預訂入口。
- **使用優質照片** — 每個庫存項目應至少有一張高品質圖片。
- **保持描述更新** — 當您更改服務或季節性調整時，更新描述。
- **為所有項目定價** — 無價格的庫存可瀏覽但無法預訂。
