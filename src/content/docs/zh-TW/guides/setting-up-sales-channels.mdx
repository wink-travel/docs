銷售通路是您控制庫存分銷的方式。銷售通路定義了誰可以銷售您的房間、他們賺取的佣金以及可用的房價方案。

## 銷售通路類型

在 Wink 上有三種主要的物業分銷方式：

- **Wink Network** — 您的物業對平台上的所有合作夥伴可見。這是最廣泛的分銷方式。
- **直接通路** — 您為特定合作夥伴或夥伴建立私人通路，並設定自訂的佣金比例。
- **您自己的網站** — 使用可分享連結或嵌入卡片，直接從您的網站銷售，無需中介。

## 建立銷售通路

<Steps>
1. 前往 https://extranet.wink.travel 並選擇您的物業。
2. 導航至銷售通路區域。
3. 點擊 `建立新的銷售通路`。
4. 為通路命名（例如：「網站直銷」、「合作夥伴 - Kim Travel」）。
5. 設定 **佣金百分比** — 合作夥伴每筆訂單可獲得的佣金比例。
6. 選擇此通路可用的 **房價方案**。
7. 選擇性限制特定的 **房型** 或 **庫存**。
8. 點擊 `儲存`。
</Steps>

## Wink Network

Wink Network 是預設的分銷通路。啟用後，您的物業會出現在 [Studio](/zh-TW/studio/search)，任何合作夥伴都能找到並銷售您的物業。

啟用方式：

<Steps>
1. 前往您的物業網路設定。
2. 啟用 `Wink Network`。
3. 為網路合作夥伴設定預設佣金比例。
</Steps>

:::tip
Wink Network 是獲得廣泛曝光的最簡單方式。您也可以稍後為表現最佳的合作夥伴建立直接通路，並設定不同的佣金比例。
:::

## 直接合作夥伴通路

對於您想密切合作的合作夥伴，建立具有自訂佣金的直接通路。

<Steps>
1. 透過網路區找到該合作夥伴。詳見 [Network](/zh-TW/extranet/network)。
2. 為該合作夥伴建立直接銷售通路。
3. 設定自訂佣金比例（可高於或低於網路預設）。
4. 選擇此通路可用的庫存。
</Steps>

當您想為網紅或旅行社提供優惠價格或專屬房間時，這非常有用。

## 佣金結構

佣金以訂單總額的百分比表示。佣金會在相關方之間分配：

- **合作夥伴佣金** — 銷售者（合作夥伴、旅行社、網紅）所賺取的部分。
- **平台費用** — Wink 的處理費用。
- **物業收入** — 剩餘部分歸您所有。

您可控制合作夥伴佣金。平台費用會自動扣除。

## 監控績效

通路啟用後，使用 [Analytics](/zh-TW/my-account/analytics) 追蹤哪些通路帶來最多訂單和收入。這有助於您優化佣金比例並決定投資哪些合作關係。
