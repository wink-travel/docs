自訂控制您的庫存如何呈現給旅客 — 主題色彩、預設貨幣、語言、行程等。它們適用於可分享連結、嵌入卡片、網格、地圖和訂房引擎。

## 什麼是自訂？

自訂是一個設定檔，定義：

- **顯示貨幣** — 價格顯示的貨幣。
- **顯示語言** — 內容的預設語言。
- **預設行程** — 預填的入住/退房日期和旅客人數。
- **主題** — 套用於嵌入元件的色彩與品牌風格。
- **設定** — 庫存的呈現方式（版面配置、排序）。

您可以為不同市場或使用情境建立多個自訂。

## 建立自訂

<Steps>
1. 前往 https://studio.wink.travel（或您的 https://my.wink.travel 帳戶）。
2. 從側邊欄導航至 `Customization`。
3. 點擊 `Create a new customization`。
4. 給它一個描述性名稱（例如「Europe - EUR」、「Japan Market」、「Default」）。
5. 設定顯示貨幣、語言和預設行程。
6. 配置主題色彩以符合您的品牌。
7. 點擊 `Save`。
</Steps>

## 自訂的使用範圍

建立後，自訂會在整個平台中被引用：

| 功能 | 自訂的應用方式 |
| -- | -- |
| **可分享連結** | 連結使用自訂的貨幣、語言和行程作為預設 |
| **嵌入卡片** | 卡片以您的主題色彩和顯示設定呈現 |
| **網格和地圖** | 庫存網格和地圖標記遵循自訂設定 |
| **訂房引擎** | 整個訂房流程使用自訂設定 |
| **社交貼文** | AI 生成的貼文可使用特定自訂來顯示價格 |

## 多重自訂

常見做法是為每個目標市場建立一個自訂：

- **「US Market」** — USD、英文、預設 2 晚住宿。
- **「Europe」** — EUR、英文、預設 3 晚住宿。
- **「Japan」** — JPY、日文、預設 1 晚住宿。

當您建立可分享連結或嵌入卡片時，會選擇使用哪個自訂。這表示同一物件可依不同受眾呈現不同方式。

## 透過 URL 覆寫

您也可以使用 URL 參數在可分享連結上覆寫自訂預設：

- `l=es` — 將語言覆寫為西班牙文。
- `c=EUR` — 將貨幣覆寫為 EUR。
- `sd=2024-08-24&n=2` — 將行程覆寫為特定日期和住宿晚數。

請參閱 [The Book Now Button](/zh-TW/guides/the-book-now-button) 了解完整的 URL 參數列表。
