這**立即預訂** 按鈕是酒店業中一個常見的術語，指的是酒店與供應商之間的整合，該供應商了解酒店的空房情況和價格，並可以讓旅客預訂房間。該按鈕是酒店在自己的網站上按照自己的條款銷售客房的最簡單的方式。

*許多飯店使用其通路經理提供的捆綁預訂引擎。我們確實建議酒店在選擇任何特定預訂引擎之前進行一些研究並比較本地預訂引擎。*

## 觀眾

本文主要針對那些想要在其網站上安裝我們免費、高轉換率預訂引擎的飯店。
但是...對於任何想在其網站上使用我們的技術的人來說，這也是有益的閱讀材料。

## 透過連結

為了最簡單的集成，我們建議使用[可共享連結](/studio/shareable-links)。

以下是執行此操作的步驟：

<Steps>
  1. 前往 https://extranet.wink.travel
  2. 點選 `Sign-In / Register` 在右上角並使用您的使用者帳戶登入。
  3. 從左上角的下拉式選單中選擇您想要使用的屬性。
  4. 點選 `Booking engine` 在導覽列中。
  5. 該頁面將顯示您的連結。
  6. 將此連結提供給您的網站管理員，並將其用於您的 **立即預訂** 按鈕。
  7. 你是✅。
</Steps>


### 先進的

該連結已包含嵌入 URL 的行程，源自於您選擇的`customization`。
您可以透過以下方式進一步自訂連結：

* 更改選定的自訂或
* 覆蓋行程

在本文中，我們將重點介紹後者，因為它可以讓您更自由地讓旅客在您的網站上選擇行程並將其轉發到 Wink 的預訂引擎。

以下是如何將您自己的行程與 URL 一起傳遞。

有效的請求參數：

* **標準差** 若要新增開始日期：*sd=2024-08-24*
* **n** 新增住宿晚數：*n=1* 或者
* **編輯** 使用結束日期：*編輯=2024-08-25*
* **rc** 嘉賓：*rc=a2* （成人：2）
* **公升** 請求顯示語言：*l=es*
* **c** 請求顯示貨幣：*c=歐元*
* **頁** 新增促銷：*p=ABC123*

URL 看起來如下：

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2\&l=es\&c=EUR

對於更進階的用例，您還可以使用以下方式要求多個房間配置`+` 符號：

例子：

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2-ca16q1-ca8q1+a2\&l=es\&c=EUR

* 房間類型 1 = 成人：2 名，兒童：2 名 = 16 歲 + 8 歲。
* 房間類型 2 = 成人：2 人。

### 例子

樣本**立即預訂** 按鈕HTML：

*本範例假設您使用[引導](https://getbootstrap.com) 作為您的主要 CSS 庫。*

```html
<a 
  href="https://trvl.as/abc123" 
  target="_blank" 
  class="btn btn-lg btn-primary" 
  title="Book a room"
>
BOOK NOW
</a>
```

### 其他福利

使用我們的連結作為您的`book now` 按鈕。

* **社交友善** 該連結可以輕鬆在社交媒體上分享，並顯示`dynamic pricing` 在您選擇的圖像之上。
* **短的** 該 URL 非常易於共享，您可以在 Instagram Bio URL 等地方使用它。

## 透過嵌入

一些飯店希望其網站能夠具有更多的功能和更深層的整合。本範例將介紹嵌入[卡片](/studio/cards) 進入他們的網站。

以下是執行此操作的步驟：

<Steps>
  1. 前往 https://studio.wink.travel
  2. 點選 `Sign-In / Register` 在右上角並使用您的使用者帳戶登入。
  3. 從左上角選擇您想要處理的屬性。
  4. 點選 `Search` 從主導覽列。
  5. 輸入您的房產名稱並點擊搜尋按鈕。
  6. 點選 `Actions` 搜尋結果卡上的連結並選擇 `Create > Make a Card`。
  7. 這將創建一個 `Card`。
  8. 右上角會出現一條訊息，告訴您卡片已準備就緒，並詢問您是否要進一步自訂它。
  9. 點選 `Tools > Cards` 從主導覽欄，您將看到您的新卡。
  10. 點選 `Actions` 連結卡上的鏈接，然後點擊 `Embed`。
  11. 這將打開一個新窗口，其中包含您可以複製的範例 HTML 程式碼。
  12. 要檢索您的 `Client-ID`，前往 [應用](/developers/applications)。
  13. 要檢索您的 `Configuration-ID`，前往 [客製化](/studio/customization)
  14. 你是✅。
</Steps>


嵌入式卡片 HTML 範例：

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*上面的 HTML 範例不包含所需的 Wink CSS 和 Javascript；只是卡片。*
