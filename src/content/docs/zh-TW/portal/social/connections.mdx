您已經連結到 WinkLinks，我們的連結管理工具。點擊 `Connect a social network` 以連結更多您的社交網路。
此流程會將您導向您的社交網路，批准 `Wink Social` 應用程式代表您發布內容。

連結成功後，我們會開始為該網路產生下個排程月份的內容。

:::note
免費方案包含 WinkLinks。Facebook、Instagram 及其他平台則包含於付費方案中。
:::

## 支援的平台

| 平台 | 說明 | 需求 |
| -- | -- | -- |
| **WinkLinks** | 自動發布至我們的進階連結管理器 | 所有方案皆包含 |
| **Facebook** | 自動發布貼文，回覆留言與私訊 | 需有 Facebook 商業專頁 |
| **Instagram** | 自動發布貼文，回覆留言與私訊 | 需有連結 Facebook 帳號的 Instagram 商業帳號 |

每個連結會顯示目前狀態：**啟用中**、**已斷開** 或 **已暫停**。

:::note
我們會為每個社交網路建立並發布最佳化的內容。
:::
