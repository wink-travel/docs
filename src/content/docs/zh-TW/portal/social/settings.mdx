# 設定

當您首次使用 Social 時，我們會詢問您希望如何建立內容。這些選項隨時可用，讓您輕鬆在之後更改。

請從側邊欄導覽至 `Preferences` 以管理您的設定。

偏好設定區包含您的基本內容產生設定：

1. **偏好自訂** 選擇產生內容時要使用的自訂設定。自訂設定控制主題色彩、預設行程、貨幣、語言等。您可在 [my.wink.travel](https://my.wink.travel) 帳戶中建立自訂設定。
2. **自動核准** 啟用後，產生的貼文會自動發布，無需您的核准。當您對我們產出的內容品質有信心時，請啟用此功能。
3. **核准者姓名** 將收到核准請求電子郵件的人員或團隊名稱。
4. **核准者電子郵件** 將寄送即將發布貼文核准請求的電子郵件地址。

Social 提供多種內容創作者可供選擇。每位創作者專注於使用 AI 為您的物業創造獨特內容的特定方式。每位內容創作者都有一個 **權重** 屬性 — 權重越高，該創作者在產生貼文時被選中的機率越大。

以下是所有可用的內容創作者：

| 創作者 | 說明 |
| -- | -- |
| **目的地** | 利用您的地點資訊創造有用的區域介紹文字 |
| **房型** | 產生可預訂的貼文，包含 AI 生成的文字與您的房型圖片 |
| **活動** | 產生可預訂的貼文，介紹您的活動 |
| **附加服務** | 產生可預訂的貼文，介紹您的附加服務 |
| **景點** | 產生可預訂的貼文，介紹您的景點 |
| **地點** | 產生可預訂的貼文，介紹您的地點 |
| **會議室** | 產生可預訂的貼文，介紹您的會議室 |
| **餐廳** | 產生可預訂的貼文，介紹您的餐廳 |
| **水療** | 產生可預訂的貼文，介紹您的水療設施 |
| **物業** | 產生可預訂的貼文，介紹您價格最優的房型，搭配 AI 生成文字與物業圖片 |
| **用戶情感** | 彙整網路上的旅客評論，產生情感導向的貼文，突顯旅客喜愛您的物業的原因 |
| **趣味** | 產生輕鬆有趣的貼文，讓您的動態保持吸引力 |

:::tip[專家提示]
請確保您擁有所選內容創作者所需的庫存類型。例如，若您選擇「餐廳」，則您的物業需至少有一家餐廳登錄。
:::

## 語氣風格

<CldImage

  <figcaption>*圖 3 - 語氣風格*</figcaption>

語氣風格設定 AI 產生內容時使用的語言風格。選擇符合您品牌個性的語氣。與內容創作者一樣，每種語氣都有一個 **權重**，用以控制使用頻率。

以下是可用的語氣：

| 語氣 | 風格 |
| -- | -- |
| **中性** | 冷靜、權威、事實性。無誇張、無笑話 — 只有可靠且結構良好的知識 |
| **親切** | 溫暖、親近、文字中帶笑意，使用口語縮寫與偶爾表情符號 |
| **奢華** | 精緻、優雅。使用「精美」和「訂製」等詞彙，讓讀者感受尊貴 |
| **美食家** | 感官豐富、令人垂涎的描述。美食誘惑語言讓讀者食指大動 |
| **冒險** | 大膽、無畏、探險家氣息。生動的感官描述讓一切聽起來壯麗 |
| **家庭** | 健康、適合兒童的興奮感。溫暖的父母氛圍，強調安全與樂趣 |
| **浪漫** | 詩意、憧憬、心動。夕陽、柔和語言與愛情氛圍 |
| **幽默** | 諷刺、快速一語雙關、巧妙文字遊戲。讀起來像單口喜劇表演 |
| **環保** | 誠懇且正義感強。讚揚綠色替代方案與永續選擇 |
| **在地達人** | 極具地方特色的參考，提醒避開觀光陷阱，使用在地俚語 |
| **包容** | 歡迎、慶祝且肯定。讓每個人都感受到被看見與重視 |
| **神秘** | 空靈、靈性、宇宙真理。溫柔邀請內省 |
| **未來感** | 流線型、前瞻思維。喜愛「範式轉移」等詞彙與樂觀未來願景 |
| **經濟實惠** | 精明、足智多謀、驕傲地節儉。讚揚實惠發現與免費體驗 |

## 貼文類型

貼文類型決定產生內容的目標與結構。每種貼文類型都有一個 **權重**，用以控制使用頻率。

| 貼文類型 | 說明 |
| -- | -- |
| **交易型** | 清晰、簡潔、以轉換為目標。以事實、價格與後續步驟推動預訂與直接行動 |
| **認知型** | 建立品牌形象的內容，旨在介紹您的物業並提升潛在旅客的能見度 |
| **情感型** | 具情感共鳴的內容，創造正面聯想並與受眾建立情感連結 |

您可能想將行銷重點放在特定語言與地區。告訴我們您想為哪些語言創造內容，以及您關注的時區，我們會負責其餘工作。貼文會在您選擇的時區預定時間發布。

*大多數物業使用單一語言，並預設為當地語言。*

透過連結您的 Google 商家資料，我們能擷取更豐富的物業細節，以產生更有效的內容。您的物業需有經過驗證的 Google 商家資料，才能讓我們從您的列表中產生內容。部分內容創作者**需要**您擁有 Google 商家帳戶。

:::tip[專家提示]
從小開始

首次嘗試時不要太過雄心壯志。我們建議您先從單一語言為物業創造內容開始。
:::
