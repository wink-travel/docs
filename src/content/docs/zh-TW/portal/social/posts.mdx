貼文編輯器讓您可以建立無限的手動貼文，同時也是管理自動為您建立貼文的地方。

編輯器有 4 個區塊：

1. 內容區塊。
2. 媒體區塊。
3. 連結區塊。
4. 設定區塊。

## 內容

內容區塊允許您在發佈前修改文字內容。您可以建立多語言的在地化內容 — 每種語言都有自己的標題和描述。

## 媒體

媒體區塊允許您修改與文字搭配的圖片和影片。

## 連結

連結區塊允許您設定使用者閱讀貼文後要導向的連結。*例如，這可以是一個特殊優惠的連結或更多資訊的頁面。* 您可以新增連結 URL、連結標題和連結描述。

## 設定

設定區塊讓您可以針對每篇貼文修改／覆寫所有預設設定。您可以設定：

- **平台** — 選擇此貼文將發佈到哪個社群網路。
- **貼文類型** — 選擇交易型、認知型或情感型。
- **語氣** — 覆寫此特定貼文的語氣。
- **自訂** — 使用不同的顯示貨幣、語言、行程等自訂設定。
- **時區** — 覆寫發佈的時區。
- **排程日期** — 自動產生的貼文會在最佳時間自動排程。若您偏好其他時段，可覆寫日期和時間。

## 貼文狀態生命週期

每篇貼文都遵循以下狀態生命週期：

| 狀態 | 說明 |
| -- | -- |
| **草稿** | 貼文已建立但尚未準備好審核 |
| **待審核** | 貼文等待人工審核後排程 |
| **已核准** | 貼文已核准並將排程發佈 |
| **已拒絕** | 貼文經審核後被拒絕 — 不會發佈 |
| **已排程** | 貼文已排隊等待在排定的日期和時間發佈 |
| **已發佈** | 貼文已成功發佈到社群網路 |
| **暫停** | 貼文發佈已暫時暫停 |
| **失敗** | 貼文因錯誤無法發佈 |

## 核准貼文

貼文可在貼文列表中核准，您可以選擇一篇或多篇貼文，並選擇核准或拒絕。

:::note
每個語言和每個社群網路都會有一筆條目。

例如，若您選擇了 3 個社群網路和 3 種語言，日曆上會顯示 9 筆條目。
:::
