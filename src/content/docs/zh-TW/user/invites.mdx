Wink 使用者可以選擇讓您管理他們的一個財產、附屬機構或旅行社帳戶。當這種情況發生時，您將收到一封包含邀請請求的電子郵件。

import { Steps } from '@astrojs/starlight/components';

若要查看您目前的 Wink 邀請，請依照下列步驟操作：

<Steps>
  1. 您必須登入我們的某個網站。
  2. 點擊右上角的個人資料圖標，將出現一個選單。
  3. 點選 `Invites` 關聯。
  4. 透過以下方式與您的邀請互動 `Accepting` 或者 `Rejecting` 他們。
</Steps>
