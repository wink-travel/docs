如果您想更改您的 Wink 使用者帳戶密碼，請依照以下步驟操作：

## 更新密碼

<Steps>
1. 您必須先登入我們的任一網站。
2. 點擊右上角的個人資料圖示，會出現選單。
3. 點擊 `更改密碼` 連結。
4. 按照指示更改您的密碼。
5. 點擊 `儲存`。
</Steps>

下次登入時，您需要使用新密碼。

## 忘記密碼

有時候需要重設使用者密碼。

以下是重設使用者密碼的步驟：

<Steps>
1. 在登入頁面點擊 `忘記密碼` 連結。
2. 輸入您想重設的帳戶使用者名稱／電子郵件。
3. 點擊 `忘記密碼` 按鈕。*您將被導向重設密碼表單。*
4. 檢查您的電子郵件。
</Steps>

1. 將您在電子郵件中收到的代碼貼到新表單中，並輸入新密碼。
2. 確認您的新密碼。
3. 點擊 `重設密碼` 按鈕。*您將被導回登入頁面。*
</Steps>
