如果您想要變更 Wink 使用者帳號的密碼，請依照下列步驟操作：

## 更新密碼

<Steps>
  1. 您必須登入我們的某個網站。
  2. 點擊右上角的個人資料圖標，將出現一個選單。
  3. 點選 `Change password` 關聯。
  4. 按照說明更改您的密碼。
  5. 點選 `Save`。
</Steps>


下次登入時，您將需要新密碼。

## 忘記密碼

有時需要重置使用者密碼。

以下是重置使用者密碼的步驟：

<Steps>
  1. 點選 `Forgot password` 來自登入頁面的連結。
  2. 輸入您想要重設的帳戶的使用者名稱/電子郵件。
  3. 點選 `Forgot password` 按鈕。 *您將被重新導向到重設密碼表單。*
  4. 檢查你的電子郵件。
</Steps>


<figure>
  <CloudinaryImage width="350" height="354" resize="scale" quality="best" alt="忘記密碼" src="academy/forgot-password" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*忘記密碼表格*</figcaption>

<Steps>
  1. 將您在電子郵件中收到的代碼與新密碼一起貼到新表單中。
  2. 確認您的新密碼。
  3. 點選 `Reset password` 按鈕。 *您將被重新導向回登入頁面。*
</Steps>


<figure>
  <CloudinaryImage width="350" height="467" resize="scale" quality="best" alt="重設密碼" src="academy/reset-password" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*重設密碼表格*</figcaption>
</figure>


您的密碼已成功重設。您可以使用新的憑證登入 Wink。
