如果您想更新 Wink 使用者帳戶的個人檔案，請依照以下步驟操作：

<Steps>
1. 您必須先登入我們的任一網站。
2. 點擊右上角的個人檔案圖示，會出現選單。
3. 點擊 `Profile` 連結。
4. 按照指示更新您的 `電子郵件`、`名字` 和 `姓氏`。
5. 您也可以上傳新的 `個人照片`。
6. 點擊 `儲存`。
</Steps>

:::note
您新的個人照片將會在您完全登出 Wink 後顯示。
:::
