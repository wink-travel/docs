如果您想要更新 Wink 使用者帳號的使用者個人資料，請依照下列步驟操作：

<Steps>
  1. 您必須登入我們的某個網站。
  2. 點擊右上角的個人資料圖標，將出現一個選單。
  3. 點選 `Profile` 關聯。
  4. 按照說明更新您的 `e-mail`， `first` 和 `last name`。
  5. 您也可以上傳新的 `profile picture` 你自己。
  6. 點選 `Save`。
</Steps>


：：：筆記
當您下次完全退出 Wink 時，將顯示新的個人資料圖片。
:::
