您可以透過我們任何一個入口網站建立使用者帳號。帳號建立後，可用於 Wink 所有應用程式和網站。

## 旅客

請依照以下步驟操作：

<Steps>
1. 前往 [https://ota.wink.travel](https://ota.wink.travel)
2. 點擊右上角的 `Sign-In / Register`
3. 系統會要求您輸入基本使用者資料。填寫所有必填欄位後，點擊 `Save`。
4. 現在您可以使用您的使用者名稱（電子郵件）和密碼登入 Wink。
5. 登入後，系統會將您導回原本所在的頁面。
</Steps>

## 註冊

建立 Wink 使用者帳號可讓您使用平台上的所有功能。您可以建立無限數量的物業和聯盟帳號。

<Steps>
1. 第一次被導向我們的授權伺服器時，系統會要求您輸入使用者名稱和密碼。
2. 請點擊 `Create account` 按鈕。
3. 填寫所有必填欄位後，點擊 `Register` 繼續。系統會帶您回到登入表單。
4. 輸入您的電子郵件／使用者名稱和密碼以登入。
5. 您會被導回您原本所在的網站。
</Steps>
