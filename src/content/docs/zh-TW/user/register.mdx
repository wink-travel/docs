您可以透過我們的任何入口網站建立使用者帳戶。建立後，此帳戶可用於 Wink 的所有應用程式和網站。

## 旅行者

請依照以下步驟操作：

<Steps>
  1. 前往 [https://ota.wink.travel](https://ota.wink.travel)
  2. 點選 `Sign-In / Register` 在右上角
  3. 您將被要求輸入基本使用者詳細資料。填寫所有必填欄位並點擊 `Save`。
  4. 現在您可以使用您的使用者名稱（電子郵件）和密碼登入 Wink。
  5. 一旦登錄，系統就會將您重新導向回原來的位置。
</Steps>

## 登記

建立 Wink 使用者帳戶可讓您存取平台上的所有功能。您可以建立無限的財產和附屬帳戶。

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="登入" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*登入表單*</figcaption>
</figure>


<Steps>
  1. 第一次將您引導至我們的授權伺服器時，系統會要求您輸入使用者名稱和密碼。
  2. 相反，點擊 `Create account` 按鈕。
  3. 填寫所有必填欄位並點擊 `Register` 繼續。這將帶您返回登入表單。
  4. 輸入您的電子郵件/使用者名稱和密碼登入。
  5. 您將被重新導向回原來的網站。
</Steps>
