## 銷售通路

要管理您的銷售通路，請從主選單點選 `Distribution > Sales channels`。

銷售通路是您現有的聯盟夥伴關係，並以他們可存取的庫存、優惠及折扣加以規範。

要更新現有的銷售通路，請點選您想更新的銷售通路下方的 `Actions` 連結。

- **會員折扣** 透過此銷售通路預訂的旅客可享有會員價格。
- **佣金** 每筆成功訂單會分配給該銷售通路的佣金。
- **單一促銷** 新增您希望此銷售通路擁有的促銷活動。
- **組合促銷** 新增您希望此銷售通路擁有的組合促銷活動。
- **啟用** 切換啟用開關以完全啟用或停用該銷售通路。
- **黑名單** 將銷售通路標記為黑名單，將無法透過該通路進行預訂。

設定完成後，點選 `Save` 按鈕繼續。

:::tip
將銷售通路列入黑名單後，該通路將完全無法取得您的任何價格，甚至包括您的 Wink Network 價格。
:::

## 探索網路

要尋找新的聯盟夥伴，請從主選單點選 `Distribution > Explore network`。

在此區域，頁面上方會顯示待處理的聯盟請求，下方則列出可連結的聯盟夥伴清單。

### 回應請求

您可以選擇拒絕聯盟請求。若想繼續處理，系統會帶您至 `affiliate details` 頁面，您可在此查看該聯盟的詳細資訊及 Wink 上的過往績效指標。

### 搜尋聯盟夥伴

在聯盟夥伴列表中，您可以依名稱、類型及地點搜尋聯盟夥伴。看到喜歡的夥伴時，點選即可進入 `affiliate details` 頁面。

### 與聯盟夥伴連結

若您選擇與某聯盟夥伴建立直接關係，系統會要求您以管理銷售通路的方式來規範此關係。

儲存新的銷售通路後，系統會帶您回到銷售通路清單。

## API

想管理您的 `Network` 的開發者，可前往 [Developers > APIs > Distribution](/zh-TW/developers/apis/#distribution-api)。
