本文向您展示如何在 Wink 上與熱切的聯盟會員建立聯繫並管理現有的聯盟會員。

## 銷售通路

若要管理您的銷售管道，請點選`Distribution > Sales channels` 從主導覽列。

  <CloudinaryImage width="350" height="402" resize="scale" quality="best" alt="銷售通路" src="academy/sales-channel" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*銷售通路進入*</figcaption>
</figure>


銷售管道是您現有的聯盟關係，透過他們可以存取的庫存、交易和折扣進行編碼。

若要更新現有的銷售管道，請點選`Actions` 您希望更新的銷售管道下的連結。

* **會員折扣** 透過此銷售管道預訂的旅客可享有會員價格。
* **委員會** 每筆成功預訂都會分配佣金給銷售管道。
* **單次促銷** 新增您希望此銷售管道擁有的促銷活動。
* **捆綁促銷** 新增您希望此銷售管道擁有的捆綁促銷。
* **已啟用** 切換啟用開關以完全啟用/停用銷售管道。
* **列入黑名單** 將銷售管道標記為黑名單將導致無法透過該銷售管道進行預訂。

對設定滿意後，點擊`Save` 按鈕繼續。

：：：提示
將銷售管道列入黑名單，您將無法從您那裡獲得任何價格；甚至您的 Wink Network 費率。
:::

## 探索網絡

要在 Wink 上尋找新的聯盟會員，請點擊`Distribution > Explore network` 從主導覽列。

在此部分中，您可以在頁面頂部看到來自關聯公司的待處理請求，並在下方看到要連接的關聯公司的清單。

### 回應請求

您可以選擇拒絕聯盟請求。如果您想繼續請求，您將被帶到`affiliate details` 您可以在此頁面上獲取有關關聯屬會員的更多資訊以及 Wink 的過去績效指標。

### 搜尋關聯公司

在聯盟會員網格中，您可以按名稱、類型和位置搜尋聯盟會員。當你看到你喜歡的一個時，點擊它，它會帶你到`affiliate details` 頁。

### 與聯盟會員聯繫

如果您選擇與關聯公司建立直接關係，您將被要求以與管理上述銷售管道相同的方式對這種關係進行編纂。

儲存新的銷售管道後，您將返回銷售管道清單。

## API

想要管理其`Network` 可以前往[開發人員 > API > 分發](/developers/apis/#distribution-api)。
