本文說明如何在 Wink 上與積極的聯盟合作夥伴連接並管理現有的合作夥伴。

## 銷售渠道

要管理您的銷售渠道，請從主導航欄點擊 `Distribution > Sales channels`。

銷售渠道是您現有的聯盟合作關係，並以他們可存取的庫存、優惠和折扣進行編碼。

要更新現有的銷售渠道，請點擊您想更新的銷售渠道下方的 `Actions` 連結。

- **會員折扣** 會員價格適用於透過此銷售渠道預訂的旅客。
- **佣金** 每筆成功預訂，該銷售渠道將獲得佣金。
- **單一促銷** 新增您希望此銷售渠道擁有的促銷活動。
- **組合促銷** 新增您希望此銷售渠道擁有的組合促銷活動。
- **啟用** 切換啟用開關以完全啟用或停用銷售渠道。
- **黑名單** 將銷售渠道標記為黑名單，將無法透過該銷售渠道進行預訂。

設定完成後，點擊 `Save` 按鈕繼續。

:::tip
將銷售渠道列入黑名單後，該銷售渠道將完全無法從您這裡取得任何價格；甚至包括您的 Wink Network 價格。
:::

## 探索網絡

要在 Wink 上尋找新的聯盟合作夥伴，請從主導航欄點擊 `Distribution > Explore network`。

在此區域，您可以在頁面上方看到來自聯盟的待處理請求，下方則列出可連接的聯盟清單。

### 回應請求

您可以選擇拒絕聯盟請求。如果您想繼續該請求，系統將帶您前往 `affiliate details` 頁面，您可以在該頁面查看聯盟的更多資訊以及在 Wink 上的過往績效指標。

### 搜尋聯盟

在聯盟列表中，您可以依名稱、類型和地點搜尋聯盟。當您看到喜歡的聯盟時，點擊它即可前往 `affiliate details` 頁面。

### 與聯盟連接

如果您選擇與聯盟建立直接關係，系統會要求您以管理銷售渠道的方式編碼該關係，如上所述。

儲存新的銷售渠道後，系統會帶您回到銷售渠道清單。

## API

想要管理其 `Network` 的開發者，可以前往 [Developers > APIs > Distribution](/zh-TW/developers/apis/#distribution-api)。
