附加組件是設施和體驗的高級形式。附加元件可讓您在場內和場外展示和銷售您的附加元件。若要建立附加元件，請點選`Create add-on` 按鈕。

以下部分將引導您完成管理附加元件的每個部分。

## 配置

這`Configurations` 當您開始時，選項卡是預設選擇的。

本節介紹附加元件的基礎知識。

* **內部名稱** 附加元件的名稱，您在​​內部引用它。 *例如鬼屋*
* **單位數量** 設定可用數量。 *例如10*
* **強制的** 預訂房間時是否必須新增此附加服務。 *例如不*
* **費率計劃** 可選擇設定費率計劃以利用其限制。

## 行程

此部分可讓您管理附加元件何時可用。

### 營業時間

* **打開** 附加元件可用的時間。 *例如09:00*
* **關閉** 附加元件不可用的時間。 *例如21:00*

### 營運天數

此部分可讓您設定附加元件可用的星期幾。

*將開關切換到✅以指示該天可用。*

### 季節性

：：：筆記
沒有漂亮、原生的日/月表單欄位。在這種情況下，我們僅使用此日期選擇器中的月份和日期。
:::

此部分可讓您設定附加元件全年可用的日期。

* **開始日期** 附加元件可用的日期。 *例如2024年9月11日*
* **開始日期** 附加元件不可用的日期。 *例如2024年7月11日*

## 便利設施

：：：筆記
此部分為可選。
:::

點選`Amenities` 選項卡繼續。

與物業設施類似，附加設施僅適用於特定的附加設施。檢查每個適用的便利設施。

## 描述

：：：筆記
您需要至少寫一個描述，並且需要使用英文。
:::

點選`Descriptions` 選項卡繼續。

與屬性歡迎文字類似，您也可以為附加元件建立本地化描述。
您可以使用任何多種語言進行書寫。我們將添加所有最受歡迎語言的翻譯。
這是旅行者在查看您的附加元件時將看到的文字。

* **姓名** 為附加元件命名。 *例如水上飛機運輸*
* **描述** 用一兩段文字描述該附加元件。 *例如我們的島嶼只能透過水上飛機到達…*

## 社群媒體

：：：筆記
此部分為可選。
:::

點選`Social media` 選項卡繼續。

您可能會想，「為什麼附加元件有自己的社群媒體部分？」。附加設施也可以是靠近物業但在場外的設施，擁有自己的員工和自己的 IG 資料。

社群媒體可以讓您與新舊旅行者保持長期聯繫。
將您的社交網路帳戶新增至您的個人資料中，以便客人可以輕鬆地與您聯繫並了解此附加元件。

<Steps>
  1. 切換您想要啟用的社交網路的開關。
  2. 輸入您在該網路上的帳戶名稱。
</Steps>


## 名聲

：：：筆記
此部分為可選。
:::

點選`Reputation` 選項卡繼續。

類似`Social media` 選項卡，如果此附加元件具有單獨的聲譽分數或您想要在此處突出顯示的額外榮譽，則可以使用此選項卡。

您的聲譽就是您的數位貨幣，您希望無論走到哪裡都能隨身攜帶它。
聲譽條目可以是您的酒店想要獲得的任何榮譽，例如第三方評級或您的廚師贏得金牌的烘焙比賽。
在此部分，您可以添加來自互聯網各地的分數和排名。

點選`Add reputation score` 按鈕。

<Steps>
  1. **類別** 選擇您想要新增的分數類型。 *例如第三方審核*
  2. **提供者** 輸入誰給你評分。 *例如BigOTA.com*
  3. **評級類型** 選擇您獲得分數的形式。 *例如數字*
  4. **日期** （可選）輸入您收到分數的日期。
  5. **等級** 輸入您獲得的分數。 *例如8.5*
  6. **最高評級** 或者，輸入此類獎項可能獲得的最高分數。 *例如10*
</Steps>


## 基於屬性的銷售

：：：筆記
此部分為可選。
:::

點選`Attribute Based Selling` 選項卡繼續。

我們的所有設施都配有自己的 ABS 部分。此部分可讓您在主庫存上建立追加銷售機會或將非交易庫存類型變成交易類型。
我們強烈建議您創造這些機會，以便在預訂過程中以更高的價格與預訂者建立聯繫。

：：：提示
附加組件可透過 ABS 出售。它可以透過不同的方式出售：

* 緩慢但自由。

* 速度快但成本高。

* +++ 讓您的想像自由馳騁並賺取一些額外的錢！ ！
  :::

* **價格點** 讓旅客快速了解這個附加組件的價格。

* **積極的** 如果您希望此附加元件在所有銷售管道中都無法預訂，請切換到🛑。

* **可收取佣金** 表示關聯公司是否對該庫存收取佣金。

* **精選** 如果該選項出現，則指示該附加元件是否應該更加突出。

* **基於屬性的銷售** 將開關切換到✅以在此庫存上啟用 ABS。 *啟用後，您需要添加至少 1 個 ABS 商品進行銷售*

### 附屬品

當您啟用 ABS 時，您可以開始新增輔助物品並使其可供出售。

*預設情況下，已經為您建立一個空項目。您可以透過點擊`Add item for sale` 按鈕。*

* **姓名** 給你的輔助設備取個名字。 *例如緩慢但自由*
* **定價類型** 指出您想要如何計算價格。 *例如設定為`Per person` 表示價格是基於編號的。乘客*
* **基價** 輸入該商品的正常價格。 *例如200 / 人*
* **折扣價** 輸入 Wink 用戶預訂商品時所獲得的價格。 *例如150 / 人*

### 限制

您可以透過設定最小/最大數量限制來限制此商品的可用性。

* **最低數量** 限制需要預訂的最低數量。 *例如1 位出席者*
* **最大數量** 限制需要預訂的最大數量。 *例如4名參加者*

### 描述

：：：筆記
您需要至少寫一個描述，並且需要使用英文。
:::

* **姓名** 給你的輔助設備取個名字。 *例如緩慢但自由*
* **描述** 用一兩段話描述輔助內容。 *例如使用我們非常古老的螺旋槳飛機帶您到達那裡。*

## 接近度

點選`Proximity` 選項卡繼續。

並非所有庫存都位於現場。如果您的庫存位於場外，請在此處告知旅行者，您將有機會設定單獨的地圖位置以及可選的聯絡資訊。

*當附加設施距離酒店有幾公里並且旅行者想向計程車司機解釋如何到達那裡時，這會很有幫助。*

點選`Save` 按鈕繼續。
