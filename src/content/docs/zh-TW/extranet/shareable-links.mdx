:::note
這是 Wink Studio 中 [可分享連結](/studio/shareable-links) 功能的子集。

物件將自動提供其所有庫存作為可分享連結，無需前往 Wink Studio 生成。
:::

可分享連結是 Wink 的核心功能之一，源自於需要以網路上最簡單的方式——超連結，動態地傳達庫存和價格資訊。

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

上方是我們其中一個連結的範例。它短小且便於攜帶……甚至短到容易記憶。雖然簡短，這個連結包含了**爆炸性**的資訊，例如：

- 您的應用程式 ID。
- 您的聯盟 ID。
- 顯示給使用者的圖片和文字。
- 外觀風格（*例如主題、顏色等*）。
- 用於價格查詢的行程（*2 位成人，1 晚*）。
- 顯示價格的貨幣。
- 顯示文字的語言。

本文接下來將引導您如何管理指向您庫存的連結。

## 自訂連結

表單讓您以以下方式自訂連結：

<Steps>
 1. 預設情況下，連結會以您的庫存英文名稱命名。您可以給它任何讓您記得連結內容的名稱。
 2. 預設會套用您的主要品牌／自訂設定。您也可以選擇 Wink Studio 中產生的其他 [自訂設定](/studio/customization) 。
 3. 新增以逗號分隔的關鍵字，供 [網路爬蟲](https://developer.mozilla.org/en-US/docs/Glossary/Crawler) 使用。
 4. 選填您的 X 帳號名稱或 Facebook App ID。
 5. 以您希望使用者看到的語言新增標題和描述。
 6. 選擇一張或多張圖片，當您在 Facebook 或 LinkedIn 等平台分享連結時會顯示。
 7. 選擇一個主題／覆蓋層，為您選擇的圖片加上品牌設計。*注意：這是讓價格顯示在圖片上的方式。*
 8. 點擊 `Save` 按鈕繼續。
 </Steps>

上圖展示了連結的所有可用操作：

- **預覽** 顯示連結設定的快速概覽，以及使用者看到的樣貌。
- **更新** 更新您的連結設定。
- **加入 WinkLinks** 將連結加入您的 WinkLinks 帳戶。
- **複製** 將連結複製到剪貼簿，方便您在任何地方分享（*包括 [WinkLinks](/link-manager/wink-links) *）。
- **分享** 開啟新視窗，提供多個社群網路和通訊平台的分享選項。
- **QR 碼** 產生 QR 碼，方便在以圖片為主的平台如 IG 分享。
- **嵌入** 顯示如何將此連結作為 [廣告橫幅](/developers/web-components/#content-loader) 嵌入您的網站。
- **WordPress 使用** 顯示如何使用我們的 [WordPress 外掛](/developers/wordpress) 將此連結嵌入您的網站。

點擊任一圖示按鈕即可繼續在該網路發布連結。*注意：部分按鈕僅在行動裝置分享時顯示。*

若您想分享 QR 碼而非連結，點擊 `QR code` 按鈕，將開啟新視窗顯示您的 QR 碼。

如何儲存您的 QR 碼：

- 若您使用筆電，右鍵點擊圖片並選擇 `儲存圖片`。
- 若您使用手機：
    - 長按圖片直到出現選項，選擇將圖片儲存到相簿。
    - 截圖 QR 碼，裁切合適後儲存至相簿。

```html {3, 6-9, 11, 13-16}
<html>

       layout="AD_BANNER"

    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

以下說明如何將您的連結作為廣告橫幅嵌入網站：

- 第 3 行示範如何將 Wink 樣式嵌入網站。
- 第 6 至 9 行示範如何使用 [wink-content-loader](/developers/web-components/#content-loader) Web 元件，並告訴它抓取您的廣告橫幅。
- 第 11 行示範如何將我們的 Javascript 嵌入網站。
- 第 13 行示範如何使用 [wink-app-loader](/developers/web-components/#application-loader) Web 元件，並告訴它抓取您的頁面級設定偏好。
