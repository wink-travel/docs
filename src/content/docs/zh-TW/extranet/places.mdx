場所可讓您在場內和場外展示和銷售您的場所。若要建立地點，請點擊`Create place` 按鈕。

以下部分將引導您完成管理場所的每個部分。

## 配置

這`Configurations` 當您開始時，選項卡是預設選擇的。

本節介紹您所在地的基本情況。

* **內部名稱** 該地點的名稱，就像您在內部所指的那樣。 *例如鬼屋*
* **生活方式** 可選擇適合這個地方的生活方式。 *例如家庭*
* **類型** 可選擇最適合的地點類型。 *例如博物館*

## 行程

此部分可讓您管理該地點何時可用。

### 營業時間

* **打開** 一天中何時有空位。 *例如09:00*
* **關閉** 一天中某個地點不可用的時間。 *例如21:00*

### 營運天數

此部分可讓您設定該地點在一周中的哪幾天可用。

*將開關切換到✅以指示該天可用。*

### 季節性

：：：筆記
沒有漂亮、原生的日/月表單欄位。在這種情況下，我們僅使用此日期選擇器中的月份和日期。
:::

此部分可讓您設定該地點全年可用的日期。

* **開始日期** 位置可用的日期。 *例如2024年9月11日*
* **開始日期** 位置不可用的日期。 *例如2024年7月11日*

## 便利設施

：：：筆記
此部分為可選。
:::

點選`Amenities` 選項卡繼續。

與物業設施類似，地點設施僅適用於特定地點。檢查每個適用的便利設施。

## 描述

：：：筆記
您需要至少寫一個描述，並且需要使用英文。
:::

點選`Descriptions` 選項卡繼續。

與飯店歡迎文字類似，您也可以為該地點建立本地化描述。
您可以使用任何多種語言進行書寫。我們將添加所有最受歡迎語言的翻譯。
這是旅行者在查看您的地點時會看到的文字。

* **姓名** 給這個地方取個名字。 *例如鬼屋*
* **描述** 用一兩段話描述這個地方。 *例如想知道鬼魂是否真實存在嗎？*

## 社群媒體

：：：筆記
此部分為可選。
:::

點選`Social media` 選項卡繼續。

您可能會想，「為什麼一個地方有自己的社群媒體部分？」。地點也可以是靠近房產但在場外的設施，擁有自己的員工和自己的 IG 資料。

社群媒體可以讓您與新舊旅行者保持長期聯繫。
將您的社交網路帳戶添加到您的個人資料中，以便客人可以輕鬆與您聯繫並了解這個地方。

<Steps>
  1. 切換您想要啟用的社交網路的開關。
  2. 輸入您在該網路上的帳戶名稱。
</Steps>


## 名聲

：：：筆記
此部分為可選。
:::

點選`Reputation` 選項卡繼續。

類似`Social media` 選項卡，如果該地點具有單獨的聲譽分數或您想要在此突出顯示的額外榮譽，則可以使用此選項卡。

您的聲譽就是您的數位貨幣，您希望無論走到哪裡都能隨身攜帶它。
聲譽條目可以是您的酒店想要獲得的任何榮譽，例如第三方評級或您的廚師贏得金牌的烘焙比賽。
在此部分，您可以添加來自互聯網各地的分數和排名。

點選`Add reputation score` 按鈕。

<Steps>
  1. **類別** 選擇您想要新增的分數類型。 *例如第三方審核*
  2. **提供者** 輸入誰給你評分。 *例如BigOTA.com*
  3. **評級類型** 選擇您獲得分數的形式。 *例如數字*
  4. **日期** （可選）輸入您收到分數的日期。
  5. **等級** 輸入您獲得的分數。 *例如8.5*
  6. **最高評級** 或者，輸入此類獎項可能獲得的最高分數。 *例如10*
</Steps>


## 基於屬性的銷售

：：：筆記
此部分為可選。
:::

點選`Attribute Based Selling` 選項卡繼續。

我們的所有設施都配有自己的 ABS 部分。此部分可讓您在主庫存上建立追加銷售機會或將非交易庫存類型變成交易類型。
我們強烈建議您創造這些機會，以便在預訂過程中以更高的價格與預訂者建立聯繫。

：：：提示
可以透過 ABS 出售某個位置。它可以透過不同的方式出售：

* 單張票。

* 集體旅行。

* 遇見鬼。

* +++ 讓您的想像自由馳騁並賺取一些額外的錢！ ！
  :::

* **價格點** 讓旅行者快速了解這個地方有多貴。

* **積極的** 如果您希望此地點在所有銷售管道上都無法預訂，請切換至🛑。

* **可收取佣金** 表示關聯公司是否對該庫存收取佣金。

* **精選** 如果有選項，請指出是否應將此位置放置得更顯眼。

* **基於屬性的銷售** 將開關切換到✅以在此庫存上啟用 ABS。 *啟用後，您需要添加至少 1 個 ABS 商品進行銷售*

### 附屬品

當您啟用 ABS 時，您可以開始新增輔助物品並使其可供出售。

*預設情況下，已經為您建立一個空項目。您可以透過點擊`Add item for sale` 按鈕。*

* **姓名** 給你的輔助設備取個名字。 *例如鬼屋門票*
* **定價類型** 指出您想要如何計算價格。 *例如設定為`Per day` 表示價格是基於編號的。天。*
* **基價** 輸入該商品的正常價格。 *例如200 /天*
* **折扣價** 輸入 Wink 用戶預訂商品時所獲得的價格。 *例如150 /天*

### 限制

您可以透過設定最小/最大數量限制來限制此商品的可用性。

* **最低數量** 限制需要預訂的最低數量。 *例如1 位出席者*
* **最大數量** 限制需要預訂的最大數量。 *例如4名參加者*

### 描述

：：：筆記
您需要至少寫一個描述，並且需要使用英文。
:::

* **姓名** 給你的輔助設備取個名字。 *例如團體旅行*
* **描述** 用一兩段話描述輔助內容。 *例如嚇壞你所有的朋友。*

## 接近度

點選`Proximity` 選項卡繼續。

並非所有庫存都位於現場。如果您的庫存位於場外，請在此處告知旅行者，您將有機會設定單獨的地圖位置以及可選的聯絡資訊。

*當某個地方距離酒店有幾公里，而旅客想向計程車司機解釋如何到達那裡時，這會很有幫助。*

點選`Save` 按鈕繼續。
