活動讓您展示並銷售您的活動，無論是在場內或場外。要建立活動，請點擊 `Create activity` 按鈕。

以下章節將逐步引導您管理活動的每個部分。

## 設定

預設會選擇 `Configurations` 標籤開始。

本節涵蓋您的活動基本資訊。

- **內部名稱** 活動的名稱，供內部參考。*例如：Fly fishing*
- **生活風格** 可選擇符合此活動的生活風格。*例如：Sports*
- **年齡** 可選擇此活動建議的最低年齡層。*例如：Adult*
- **類型** 可選擇最符合的活動類型。*例如：Fishing*
- **是否需預約** 此活動是否需要預約。*例如：Yes*

## 時間表

本節讓您管理活動的可用時間。

### 營業時間

- **開放時間** 活動開始可用的時間。*例如：09:00*
- **關閉時間** 活動結束不可用的時間。*例如：21:00*

### 營業日

本節讓您設定活動每週可用的日期。

*切換開關至 ✅ 表示該日可用。*

### 季節性

:::note
此處沒有漂亮的原生日/月欄位。在此情況下，我們只使用日期選擇器中的月份和日期。
:::

本節讓您設定活動全年可用的日期。

- **開始日期** 活動開始可用的日期。*例如：2024年9月11日*
- **結束日期** 活動結束不可用的日期。*例如：2024年7月11日*

## 設施

:::note
此部分為選填。
:::

點擊 `Amenities` 標籤繼續。

類似於物業設施，活動設施僅適用於特定活動。勾選適用的設施。

## 描述

:::note
您需要至少撰寫一則描述，且必須為英文。
:::

點擊 `Descriptions` 標籤繼續。

類似於物業歡迎文字，您也可以為活動建立多語系描述。
您可以用多種語言撰寫，我們會為所有熱門語言新增翻譯。
這是旅客在查看您的活動時會看到的文字。

- **名稱** 為活動命名。*例如：Fly fishing*
- **描述** 用一兩段文字描述活動。*例如：只有你…森林…河流…瓶中寧靜。*

## 社群媒體

:::note
此部分為選填。
:::

點擊 `Social media` 標籤繼續。

您可能會想，「為什麼活動會有自己的社群媒體區塊？」活動也可能是物業附近但場外的設施，擁有自己的員工和 IG 帳號。

社群媒體讓您隨時與新舊旅客保持聯繫。
將您的社群帳號加入個人檔案，讓旅客輕鬆聯絡您並了解此活動。

<Steps>
1. 切換您想啟用的社群網路開關。
2. 輸入您在該網路上的帳號名稱。
</Steps>

## 評價

:::note
此部分為選填。
:::

點擊 `Reputation` 標籤繼續。

類似於 `Social media` 標籤，若此活動有獨立的評價分數或額外榮譽，您可在此強調。

您的評價是您的數位貨幣，您希望隨時攜帶。
評價條目可以是物業想要認可的任何項目，例如第三方評分或廚師在烘焙比賽中獲得金牌。
在此可新增來自網路各處的分數與排名。

點擊 `Add reputation score` 按鈕。

<Steps>
1. **類別** 選擇您要新增的分數類型。*例如：第三方評論*
2. **提供者** 輸入給予您分數的單位。*例如：BigOTA.com*
3. **評分類型** 選擇您收到分數的形式。*例如：數字*
4. **日期** 可選，輸入您收到分數的日期。
5. **評分** 輸入您收到的分數。*例如：8.5*
6. **最高分** 可選，輸入此獎項的最高分數。*例如：10*
</Steps>

## 屬性基礎銷售

:::note
此部分為選填。
:::

點擊 `Attribute Based Selling` 標籤繼續。

我們所有設施都有自己的 ABS 區塊。本節讓您在主庫存上建立加購機會，或將非交易型庫存轉為交易型。
我們強烈建議您建立這些機會，以便在訂房過程中以較高價格與訂房者連結。

:::tip
活動可透過 ABS 方式銷售。可用不同方式銷售：

- 一日遊。
- 三日遊。
- +++ 讓您的想像力奔馳，賺取額外收入！！
:::

- **價格指標** 快速告知旅客此活動的價格層級。
- **啟用** 若您想讓此活動在所有銷售通路中不可預訂，切換開關至 🛑。
- **可抽成** 表示此庫存是否會給予聯盟佣金。
- **精選** 表示若有此選項，是否要讓此活動更顯眼。
- **屬性基礎銷售** 切換開關至 ✅ 啟用此庫存的 ABS。*啟用後，您需新增至少一個 ABS 銷售項目*

### 附加項目

啟用 ABS 後，您可以開始新增附加項目並開放銷售。

*預設已為您建立一個空白項目。您可點擊 `Add item for sale` 按鈕新增更多。*

- **名稱** 為附加項目命名。*例如：一日遊*
- **定價類型** 表示您如何計算價格。*例如：設定為 `Per day` 表示價格依天數計算。*
- **基本價格** 輸入此項目的常規價格。*例如：200 / 天*
- **折扣價格** 輸入 Wink 用戶預訂此項目時的價格。*例如：150 / 天*

### 限制

您可以透過設定數量的最小/最大限制來限制此項目的可用性。

- **最小數量** 限制必須預訂的最小數量。*例如：1 位參加者*
- **最大數量** 限制可預訂的最大數量。*例如：4 位參加者*

### 描述

:::note
您需要至少撰寫一則描述，且必須為英文。
:::

- **名稱** 為附加項目命名。*例如：一日遊*
- **描述** 用一兩段文字描述附加項目。*例如：休息一天，前往河邊。*

## 鄰近位置

點擊 `Proximity` 標籤繼續。

並非所有庫存都位於場內。若您有場外庫存，請在此告知旅客，並可設定獨立地圖位置及選填聯絡資訊。

*當活動距離飯店數公里，旅客想向計程車司機說明如何抵達時，這會很有幫助。*

點擊 `Save` 按鈕繼續。
