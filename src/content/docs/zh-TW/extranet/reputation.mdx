您的聲譽是您的數位貨幣，您希望能隨時攜帶它。
聲譽條目可以是您的物業想要認領的任何項目，例如第三方評分或您的主廚在烘焙比賽中獲得金牌。
在本節中，您可以新增來自網路各處的分數和排名。

要管理您的聲譽，請點擊主導航標籤中的 `Property > Reputation`。

點擊 `Add reputation score` 按鈕。

<Steps>
1. **Category** 選擇您想新增的分數類型。*例如 第三方評論*
2. **Provider** 輸入授予您分數的機構。*例如 BigOTA.com*
3. **Rating type** 選擇您獲得分數的形式。*例如 數值型*
4. **Date** （選填）輸入您獲得分數的日期。
5. **Rating** 輸入您獲得的分數。*例如 8.5*
6. **Max rating** （選填）輸入此類獎項的最高分數。*例如 10*
</Steps>

點擊 `Save` 按鈕繼續。
