[Wink Extranet](https://extranet.wink.travel) 是我們的物業入口網站。它是為飯店、連鎖、品牌及飯店管理公司設計，用來在 Wink 平台上建立及管理其物業的數位檔案。

我們的 Extranet 經過精心設計，盡可能簡單，但仍支援進階使用者期望的所有功能。舉幾個例子：

1. 簡單：使用我們智慧的上線流程，讓您的物業在不到一小時內完成上線並開始營運。
2. 進階：Wink 支援 8500 種不同的取消政策變化。
3. 酷炫：讓地球上某個人享有 10% 折扣，且使用者無需登入或輸入促銷代碼。

讓我們的精靈引導您完成整個流程。當您感到足夠熟悉時……就可以自己操作。

Wink 是一個先進的物業及體驗供應商內容管理系統。您在我們任一應用程式中能做的事，都可以透過 API 完成。這讓您能與 Wink 深度整合，甚至可以在完全不登入我們任何應用程式的情況下透過 Wink 銷售。

:::tip[專業提示]
您是否正在透過 Leonardo International 支付昂貴的媒體資產託管費用？您可以透過 Wink 做完全相同的事，且還能獲得更多操作及微調圖片和影片的選項。
:::

所有這些功能對您完全免費。只有當您完成訂單時，我們才會收取少量費用。沒有損失——只有收穫。
