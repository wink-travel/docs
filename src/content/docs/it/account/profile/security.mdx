La scheda **Sicurezza** su [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) è dove gestisci le **passkey** — credenziali WebAuthn che ti permettono di accedere senza password.

## Cos'è una passkey?

Una passkey è una credenziale crittografica legata al tuo dispositivo (o gestore di password) che dimostra che sei tu. Le passkey sono resistenti al phishing e più veloci della digitazione di una password.

## Registrare una passkey

1. Accedi a [iam.wink.travel](https://iam.wink.travel) con le tue credenziali esistenti.
2. Vai su **Il mio account → Sicurezza**.
3. Clicca su `Aggiungi passkey`.
4. Il tuo browser ti chiederà di autenticarti con il tuo dispositivo (Touch ID, Face ID, Windows Hello, una chiave hardware o un gestore di password sincronizzato).
5. Dai un nome amichevole alla passkey (es. *Work MacBook*) così potrai riconoscerla in seguito.

## Usare una passkey per accedere

Nella schermata di accesso di [iam.wink.travel](https://iam.wink.travel), inserisci la tua email — se hai una passkey registrata per quell’email, ti verrà chiesto di autenticarti con essa invece di digitare una password.

## Rimuovere una passkey

Nella stessa scheda Sicurezza, ogni passkey registrata ha un pulsante `Rimuovi`. Usalo se perdi un dispositivo, ritiri una chiave hardware o vuoi eliminare credenziali inutilizzate.

:::caution
Rimuovere tutte le passkey lascia la password come unico metodo di accesso. Assicurati di ricordare la tua password prima di rimuovere l’ultima passkey.
:::
