Wink offre un sistema di reportistica che ti consente di generare ed esportare dati relativi al tuo account. I report sono disponibili in formato CSV per un facile import in fogli di calcolo, strumenti di analisi o nei tuoi sistemi.

## Tipi di report

Puoi generare report per le seguenti categorie di dati:

| Tipo di report | Cosa include |
|---|---|
| **Proprietà** | Dettagli della proprietà, stato dell'annuncio, riepilogo dell'inventario. |
| **Utente** | Account utente, ruoli, riepilogo attività. |
| **Prenotazione** | Storico prenotazioni, stati, ricavi, dettagli ospiti. |
| **Account** | Informazioni sull'account, saldi, storico transazioni. |

## Come generare un report

1. Vai alla sezione reportistica nel pannello di controllo del tuo account.
2. Seleziona il **tipo di report** che vuoi generare.
3. Configura eventuali **filtri** per restringere i dati (es. intervallo di date, stato, proprietà).
4. Scegli i **campi** da includere nell'esportazione. Ogni tipo di report offre un set di campi disponibili da selezionare.
5. Clicca su **Esporta** per scaricare il report in formato CSV.

## Campi personalizzati

I report supportano la selezione personalizzata dei campi. Puoi scegliere quali colonne includere nell’esportazione, così ottieni esattamente i dati di cui hai bisogno — né più né meno.

## Casi d’uso

- **Riconciliazione dei ricavi** — Esporta report di prenotazioni con dettagli finanziari per riconciliare con il tuo sistema contabile.
- **Analisi delle performance** — Esporta report delle proprietà per confrontare le performance degli annunci nel tuo portfolio.
- **Approfondimenti sugli ospiti** — Esporta dati di prenotazione per comprendere la demografia dei viaggiatori e i modelli di prenotazione.
- **Audit degli account** — Esporta dati degli account per conformità o revisione interna.
