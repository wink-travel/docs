Un utente Wink può scegliere di permetterti di gestire uno dei suoi account di struttura, affiliato o agente di viaggio. Quando ciò accade, riceverai un'email con una richiesta di invito.

import { Steps } from '@astrojs/starlight/components';

Per vedere i tuoi inviti Wink attuali, segui questi passaggi:

<Steps>
 1. Accedi a [my.wink.travel](https://my.wink.travel).
 2. Clicca su `Inviti` nella barra laterale sinistra.
 3. Vedrai una lista di tutti gli inviti attuali.
 4. Questa sezione elencherà tutte le tue richieste.
 5. Scegli di `Accettare` o `Rifiutare` l'invito.
 6. Se hai accettato l'invito, l'account apparirà nella sezione `Account`.
</Steps>
