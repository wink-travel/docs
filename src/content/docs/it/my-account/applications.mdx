Le applicazioni ti danno accesso a tutte le funzionalità di Wink come sviluppatore. Puoi accedere a tutti gli stessi dati e persino estendere le funzionalità di Wink.
Puoi scegliere di utilizzare il server OAuth2 di Wink per autenticare il tuo sito di viaggi allo stesso modo in cui useresti Google o Facebook.
In questo modo puoi estendere le funzionalità di Wink nel tuo sito web o applicazione.

:::note
La tua prima app è già creata quando [crei un nuovo account](/it/my-account/create-account). Utilizziamo questa app per alcuni scenari di prenotazione.

Non puoi riutilizzare questa app. Crea nuove app secondo le tue necessità.
:::

Un'applicazione generata su Wink può essere utilizzata per tutti gli endpoint API.

<Steps>
 1. Seleziona l'account con cui vuoi lavorare da [Accounts](https://my.wink.travel).
 2. Clicca su `Actions` in fondo alla scheda dell'account e scegli `Manage account`.
 3. Clicca sulla scheda `Applications` nella pagina successiva.
 4. Questa sezione elencherà tutte le tue applicazioni.
 5. Scegli di modificare una esistente o crearne una nuova.
 </Steps>

## Crea app

Poiché non puoi accedere alla `secret key` dell'app che abbiamo creato per te, dovrai crearne una tua.

:::caution
Devi creare un account affiliato su Wink o un account su TripPay prima di creare un'app, poiché legherai la tua applicazione a un account specifico.
:::

Ecco i passaggi per creare un'applicazione:

<Steps>
1. Dalla pagina delle app, clicca sul pulsante `Create new application`.
2. ***Nome*** Dai un nome alla tua app. *es. Cool App*
3. **Nome entità** Collega la tua app a uno dei tuoi account esistenti. *es. Cool Account*
4. **Redirect URI** È richiesto almeno un redirect URI. Viene usato per convalidare il dominio di reindirizzamento dopo un'autenticazione riuscita. *es. https://www.cool-site.com*
5. Clicca sul pulsante `Save` per continuare.
</Steps>

Verrai reindirizzato alla lista delle tue applicazioni. Vedrai la tua nuova app nell'elenco e avrai accesso alla secret key dell'app solo questa volta. Assicurati di copiarla e conservarla in un luogo sicuro.

## Aggiorna app

Dovrai aggiornare la tua app se cambia uno dei tuoi redirect URI o se vuoi aggiungere o rimuovere un URI.

Per aggiornare un'app esistente, procedi così:

<Steps>
1. Clicca sul link `Actions` per l'app che vuoi aggiornare.
2. Clicca sul pulsante `Update` sotto Actions.
3. Apporta le modifiche all'app.
4. Clicca sul pulsante `Save` per continuare.
</Steps>

Verrai reindirizzato alla lista delle tue applicazioni.

## Rimuovi app

:::caution
Per favore, non eliminare l'app che abbiamo creato automaticamente per te per assicurarti che tutte le funzionalità funzionino per il tuo account.
:::

Se non hai più bisogno di un'app, puoi procedere a rimuoverla.

Per rimuovere un'app, procedi così:

<Steps>
1. Clicca sul link `Actions` per l'app che vuoi aggiornare.
2. Clicca sul pulsante `Remove` sotto Actions.
3. Clicca sul pulsante `OK` per confermare l'eliminazione.
</Steps>

Verrai reindirizzato alla lista delle tue applicazioni.
