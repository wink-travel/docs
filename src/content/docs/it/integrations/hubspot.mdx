HubSpot CMS supporta moduli HTML personalizzati e codice per header/footer a livello di sito, rendendo semplice incorporare componenti Wink nelle pagine marketing e landing page.

:::note
L'inserimento di codice a livello di sito richiede l'accesso a **HubSpot CMS Hub** (Starter o superiore). I moduli HTML personalizzati sono disponibili in tutti i livelli CMS.
:::

## Carica gli script Wink a livello di sito

<Steps>
1. In HubSpot, vai su **Marketing → Files and Templates → Design Tools**.
2. Apri il `main.css` del **tema** attivo oppure usa le impostazioni globali dei contenuti.
3. In alternativa, vai su **Settings → Website → Pages → Custom Code**.
4. Nel campo **Head HTML**, aggiungi il foglio di stile:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
5. Nel campo **Footer HTML**, aggiungi lo script e il loader dell'applicazione (sostituisci `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
6. Salva e pubblica.
</Steps>

:::tip
Il tuo Client ID è disponibile nel tuo account Wink sotto [Applications](/it/portal/settings/applications).
:::

## Aggiungi un componente tramite un modulo HTML personalizzato

<Steps>
1. In Design Tools, crea un nuovo **modulo** di tipo **Rich Text** o **HTML**.
2. Nel contenuto predefinito del modulo, incolla l'HTML del componente:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
3. Salva il modulo e trascinalo in qualsiasi template di pagina o pagina usando l'editor di pagina.
</Steps>

## Aggiungi un componente direttamente nell'editor di pagina

Se preferisci non creare un modulo, puoi usare una sezione **Custom HTML** nell'editor drag-and-drop:

<Steps>
1. Apri la pagina nell'editor di pagina HubSpot.
2. Clicca su **Add** e scegli una sezione **Custom HTML** o **Embed**.
3. Incolla il tag del componente nel campo HTML.
4. Clicca su **Apply** e pubblica la pagina.
</Steps>

## Componenti disponibili

| Componente | Tag HTML | Scopo |
|---|---|---|
| Content loader | `<wink-content-loader>` | Mostrare schede hotel, griglie, mappe |
| Lookup | `<wink-lookup>` | Barra di ricerca per destinazioni |
| Search button | `<wink-search-button>` | Aprire il selettore itinerario |
| Account button | `<wink-account-button>` | Accesso / menu utente |
| Itinerary button | `<wink-itinerary-button>` | Mostrare l'itinerario corrente |
| Itinerary picker | `<wink-itinerary-picker>` | Modulo completo itinerario |
| Shopping cart | `<wink-shopping-cart-button>` | Pulsante riepilogo carrello |

Consulta [Web Components](/it/integrations/web-components) per il riferimento completo degli attributi.
