L'**Estensione Wink per Joomla** è un pacchetto che installa un plugin di sistema (carica gli script CDN), un plugin di contenuto (elabora i tag shortcode negli articoli) e un componente (pagina delle impostazioni admin).

<LinkCard
  title="wink-travel/wink-joomla-extension su GitHub"
  href="https://github.com/wink-travel/wink-joomla-extension"
  description="Codice sorgente, guida per sviluppatori e istruzioni per contributi."
/>

## Requisiti

- Joomla 4 o 5
- PHP 8.1+

## Installazione

<Steps>
1. Scarica l'ultima versione `pkg_wink-x.x.x.zip` dalla [pagina delle release GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. Nell'Admin di Joomla, vai su **Estensioni → Gestisci → Installa**.
3. Carica il file zip e clicca su **Carica & Installa**.
4. Entrambi i plugin vengono abilitati automaticamente. Vai su **Componenti → Wink Settings**.
5. Inserisci il tuo **Client ID** e **Client Secret** e clicca su **Salva**.
   Vai su [Applicazioni](/it/portal/settings/applications) per recuperare le tue credenziali.
</Steps>

## Inserire componenti negli articoli

Usa i tag in stile shortcode ovunque nel contenuto degli articoli:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Il plugin di contenuto sostituisce questi tag con il corretto HTML `<wink-*>` al momento del rendering. Il plugin di sistema carica automaticamente gli script CDN e inserisce `<wink-app-loader>` nel footer della pagina.

## Shortcode disponibili

| Shortcode | Output | Descrizione |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Schede hotel, griglie, mappe |
| `{wink-lookup}` | `<wink-lookup>` | Barra di ricerca destinazioni |
| `{wink-search}` | `<wink-search-button>` | Pulsante selezione itinerario |
| `{wink-account}` | `<wink-account-button>` | Accesso / account utente |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Pulsante stato itinerario |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Pulsante carrello |
