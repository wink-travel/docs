L'**Estensione Wink TYPO3** integra l'inventario viaggi Wink in TYPO3 13 come elementi di contenuto nativi. Gli editor aggiungono i componenti Wink alle pagine utilizzando l'editor di pagina standard di TYPO3 — non sono necessarie modifiche ai template.

<LinkCard
  title="wink-travel/wink-typo3-extension su GitHub"
  href="https://github.com/wink-travel/wink-typo3-extension"
  description="Codice sorgente, guida all'installazione per sviluppatori e istruzioni per contribuire."

## Requisiti

- TYPO3 13.x
- PHP 8.2+
- Composer

## Installazione

<Steps>
1. Installa tramite Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Attiva l'estensione:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Nel backend di TYPO3, vai su **Admin Tools → Settings → Extension Configuration → wink**.
4. Inserisci il tuo **Client ID**, **Client Secret** e seleziona l'ambiente. Salva.
   Vai su [Applications](/it/portal/settings/applications) per recuperare le tue credenziali.
5. Aggiungi il template TypoScript statico al tuo template root:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Aggiungere un elemento di contenuto a una pagina

<Steps>
1. Apri una pagina nel modulo **Page**.
2. Clicca sul pulsante **+** nell'area contenuti per aggiungere un nuovo elemento.
3. Scegli uno degli elementi di contenuto **Wink Travel**:
   - **Wink Content** — schede hotel, griglie classificate, mappe
   - **Wink Lookup** — barra di ricerca destinazioni
   - **Wink Search** — pulsante selezione itinerario
   - **Wink Account** — pulsante di accesso
   - **Wink Itinerary** — pulsante stato itinerario
   - **Wink Cart** — pulsante carrello
4. Per **Wink Content**, usa i campi FlexForm per selezionare un **Tipo di Layout** e inserire l'**ID Layout**.
5. Salva e pubblica la pagina.
</Steps>

## Elementi di contenuto disponibili

| Elemento | Scopo |
|---|---|
| **Wink Content** | Schede hotel, griglie classificate, mappe e altro |
| **Wink Lookup** | Barra di ricerca destinazioni |
| **Wink Search** | Pulsante icona che apre il selettore itinerario |
| **Wink Account** | Pulsante di accesso / account utente |
| **Wink Itinerary** | Pulsante itinerario che mostra il viaggio corrente |
| **Wink Cart** | Pulsante riepilogo carrello |
