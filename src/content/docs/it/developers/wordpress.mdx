Per chiunque utilizzi WordPress per la propria attività o blog, consigliamo vivamente di utilizzare[Plugin WordPress Wink Affiliate](https://wordpress.org/plugins/wink2travel/)Il plugin ti consente di accedere facilmente alle tue schede Wink, griglie, mappe ecc. e semplifica il trascinamento dei contenuti nei tuoi post e nelle tue pagine utilizzando il tuo page builder preferito o i nostri shortcode personalizzati.

## Installare

Ecco come installare il`Wink Affiliate WordPress Plugin`nel tuo sito web WordPress.

<Steps>
  1. Quando hai effettuato l'accesso come amministratore e sei sulla Dashboard.
  2. Clic `Plugins > Add New Plugin`.
  3. Cercare `Wink Affiliate WordPress Plugin`.
  4. Clic `Install` per scaricare e installare il plugin sul tuo sito.
  5. Clic `Activate` per attivare il plugin sul tuo sito.
  6. L'attivazione ti inoltrerà il `Wink Settings` pagina. *Puoi anche andarci navigando verso `Appearance > Customize > Wink Settings`.*
  7. Ti chiederà il tuo `Client ID` E `Client Secret`Vai a [Applicazioni](/developers/applications) per scoprire come recuperare le tue credenziali.
  8. Inserisci le tue credenziali e clicca `Publish`Ora sei connesso al tuo account Wink.
</Steps>

## Costruttori di pagine

Supportiamo i page builder più popolari su WordPress, rendendo semplice l'utilizzo di Wink per chiunque.

### Gutenberg

Gutenberg è l'editor nativo di WordPress.

### Avada

[Avada](https://avada.com)è famoso per la sua ampia selezione di bellissimi modelli insieme a un potente page builder dotato di un set di funzionalità avanzate.

### Elementor

[Elementor](https://elementor.com)è il page builder più popolare \[attualmente], disponibile sia in versione gratuita che professionale.

### WPBakery

[WPBakery](https://wpbakery.com)è stato il primo page builder avanzato sul mercato.

## Codici brevi

### \[occhiolino]

Funziona con una griglia di contenuti classificati. Digita un luogo che desideri visitare e visualizza l'inventario.

### \[winksearch]

Semplice pulsante per aprire il modulo dell'itinerario.

### \[occhiolino account]

Pulsante Account per l'autenticazione. Una volta effettuata l'autenticazione, si aprirà un menu a discesa con le opzioni dell'account.

### \[winkitinerary]

Uguale al pulsante di ricerca, solo che contiene informazioni sull'itinerario sotto forma di testo.

### \[winkcontent]

Ti permette di incorporare l'inventario selezionato su Wink direttamente in una pagina o in un post. Questo shortcode richiede anche un ID layout.

La disposizione

```
[winkcontent layoutid=""]
```
