Questa pagina elenca tutti gli eventi webhook a cui puoi iscriverti su Wink. Per le istruzioni di configurazione, consulta la [Guida all'integrazione Webhook](/it/guides/webhook-integration) o la [pagina di gestione Webhook](/it/my-account/webhooks).

## Come iscriversi

1. Vai su **Applicazioni > Webhook** nel tuo portale.
2. Crea un webhook e seleziona gli eventi che vuoi ricevere.
3. Wink invia una richiesta HTTP POST al tuo endpoint ogni volta che si verifica l'evento.

Per i passaggi completi di configurazione e le migliori pratiche, consulta [Integrazione Webhook](/it/guides/webhook-integration).

## Eventi account

Eventi relativi al ciclo di vita e alle impostazioni del tuo account TripPay.

| Evento | Chiave | Descrizione |
|---|---|---|
| Account creato | `account.create` | È stato creato un nuovo account. |
| Account in revisione | `account.under_review` | L'account è in attesa di revisione. |
| Account aggiornato | `account.updated` | I dettagli dell'account sono cambiati. |
| Stato account aggiornato | `account.status.updated` | Lo stato dell'account è cambiato (es. approvato, sospeso). |
| Indirizzo account aggiornato | `account.address.updated` | L'indirizzo dell'account è stato modificato. |
| Dettagli account aggiornati | `account.details.updated` | I campi dei dettagli dell'account sono cambiati. |
| Informazioni account aggiornate | `account.information.update` | Le informazioni generali dell'account sono state aggiornate. |
| Gestori account aggiornati | `account.managers.update` | La lista dei gestori dell'account è cambiata. |
| Foto profilo aggiornata | `account.profile_picture.updated` | La foto profilo dell'account è stata modificata. |
| Agente aggiornato | `account.agent.updated` | Le informazioni dell'agente sull'account sono state modificate. |
| URL nome aggiornato | `account.url-name.update` | L'URL personalizzato dell'account è cambiato. |
| ID univoco aggiornato | `account.unique-id.update` | L'identificatore univoco dell'account è stato modificato. |

### Attività account

| Evento | Chiave | Descrizione |
|---|---|---|
| Attività creata | `account.task.create` | Una nuova attività è stata aggiunta all'account. |
| Attività aggiornata | `account.task.update` | Un'attività dell'account è stata modificata. |
| Attività rimossa | `account.task.remove` | Un'attività dell'account è stata rimossa. |
| Attività completata | `account.task.complete` | Un'attività dell'account è stata completata. |
| Attività verifica codice fiscale completata | `account.task.tax_id.complete` | L'attività di verifica del codice fiscale è stata completata. |

### Acquirente account

| Evento | Chiave | Descrizione |
|---|---|---|
| Acquirente creato | `account.acquirer.created` | Un nuovo acquirente è stato aggiunto all'account. |
| Acquirente aggiornato | `account.acquirer.updated` | Un acquirente dell'account è stato modificato. |
| Acquirente rimosso | `account.acquirer.removed` | Un acquirente è stato rimosso dall'account. |

## Eventi contratto di prenotazione

Eventi che tracciano il ciclo di vita di un contratto di prenotazione tramite TripPay.

| Evento | Chiave | Descrizione |
|---|---|---|
| Inizializzazione richiesta | `booking.contract.initialize.request` | È stata richiesta l'inizializzazione di un contratto di prenotazione. |
| Inizializzazione completata | `booking.contract.initialize.complete` | L'inizializzazione di un contratto di prenotazione è stata completata. |
| Contratto creato | `booking.contract.created` | È stato creato un nuovo contratto di prenotazione. |
| Contratto in elaborazione | `booking.contract.processing` | Il pagamento del contratto di prenotazione è in elaborazione. |
| Contratto fallito | `booking.contract.failed` | Il pagamento del contratto di prenotazione è fallito. |
| Contratto annullato | `booking.contract.cancelled` | Il contratto di prenotazione è stato annullato. |
| Rimborso parziale | `booking.contract.refund.partial` | È stato emesso un rimborso parziale sul contratto. |
| Rimborso totale | `booking.contract.refund.full` | È stato emesso un rimborso totale sul contratto. |
| Richiesta di rimborso | `booking.contract.refund.request` | È stata inviata una richiesta di rimborso. |
| Errore rimborso | `booking.contract.refund.error` | Si è verificato un errore durante l'elaborazione di un rimborso. |
| Pagamento creato | `booking.contract.payout.create` | È stato creato un pagamento per il contratto. |
| Pagamento completato | `booking.contract.payout.complete` | Un pagamento è stato completato con successo. |

## Eventi entità gestore

Eventi per entità gestori (proprietà, affiliati, aziende) e la loro amministrazione.

### Abbonamento

| Evento | Chiave | Descrizione |
|---|---|---|
| Abbonamento fornito | `managing_entity.subscription.provisioned` | È stato fornito un nuovo abbonamento. |
| Abbonamento modificato | `managing_entity.subscription.changed` | Il piano di abbonamento è stato modificato. |
| Fattura abbonamento modificata | `managing_entity.subscription.invoice.changed` | La fattura dell'abbonamento è stata aggiornata. |

### Stato e amministrazione

| Evento | Chiave | Descrizione |
|---|---|---|
| Stato aggiornato | `managing_entity.status.updated` | Lo stato dell'entità è cambiato. |
| Entità eliminata | `managing_entity.deleted` | Un'entità gestore è stata eliminata. |
| Agente aggiornato | `managing_entity.agent.updated` | L'agente dell'entità è stato modificato. |
| Notifica email agente | `managing_entity.agent.email.notification` | È stata inviata una notifica email all'agente. |
| Notifica email manager | `managing_entity.manager.email.notification` | È stata inviata una notifica email a un manager. |

### Ciclo di vita manager

| Evento | Chiave | Descrizione |
|---|---|---|
| Manager aggiunto | `managing_entity.manager.added` | È stato aggiunto un nuovo manager. |
| Manager invitato | `managing_entity.manager.invited` | È stato inviato un invito a un manager. |
| Manager rimosso | `managing_entity.manager.removed` | Un manager è stato rimosso. |
| Manager accettato | `managing_entity.manager.accepted` | Un invito a manager è stato accettato. |
| Manager rifiutato | `managing_entity.manager.rejected` | Un invito a manager è stato rifiutato. |
| Proprietario impostato | `managing_entity.manager.owner.set` | La proprietà dell'entità è stata trasferita. |

### Informazioni entità

| Evento | Chiave | Descrizione |
|---|---|---|
| Informazioni aggiornate | `managing_entity.information.updated` | Le informazioni dell'entità sono state aggiornate. |
| Indirizzo aggiornato | `managing_entity.city.updated` | L'indirizzo/città dell'entità è stato modificato. |
| Rappresentante account aggiornato | `managing_entity.account_representative.updated` | Il rappresentante dell'account è stato cambiato. |
| Foto profilo aggiornata | `managing_entity.profile_picture.updated` | La foto profilo dell'entità è stata aggiornata. |
| Logo aggiornato | `managing_entity.logo.updated` | Il logo dell'entità è stato aggiornato. |
| URL nome aggiornato | `managing_entity.url_name.updated` | L'URL personalizzato dell'entità è cambiato. |
| ID univoco aggiornato | `managing_entity.unique_id.updated` | L'identificatore univoco dell'entità è cambiato. |
| Nome aggiornato | `managing_entity.name.updated` | Il nome dell'entità è stato modificato. |
