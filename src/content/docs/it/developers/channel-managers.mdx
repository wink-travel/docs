Wink si integra con i principali channel manager, sistemi di gestione della proprietà (PMS) e sistemi di prenotazione centralizzati (CRS) per mantenere sincronizzati tariffe e disponibilità su tutti i tuoi canali di vendita.

## Come funziona

Le integrazioni con i channel manager consentono un flusso bidirezionale di dati tra il tuo sistema di gestione della proprietà e Wink:

1. **Invio tariffe** — Il tuo channel manager invia aggiornamenti delle tariffe a Wink. Quando aggiorni i prezzi nel tuo PMS, tali modifiche si riflettono automaticamente su Wink.
2. **Sincronizzazione disponibilità** — Il tuo channel manager invia aggiornamenti di disponibilità a Wink. Quando le camere vengono prenotate o liberate nel tuo PMS, l’inventario di Wink si aggiorna in tempo reale.
3. **Notifiche di prenotazione** — Quando viene effettuata una prenotazione su Wink, il tuo channel manager viene notificato affinché la prenotazione appaia nel tuo PMS.

## Channel manager supportati

Wink si integra attualmente con i seguenti sistemi:

| Channel Manager | Descrizione |
|---|---|
| **Channex** | Piattaforma globale di distribuzione e gestione canali. |
| **TravelClick** | Soluzioni per hotel e gestione canali (Amadeus). |
| **Omnibees** | Distribuzione e gestione ricavi per hotel. |
| **D-Edge** | Tecnologia per l’ospitalità e soluzioni di prenotazione centralizzata. |
| **Allotz** | Allocazione inventario e gestione canali. |
| **Ezee / Yanolja** | PMS cloud e gestione canali focalizzati sul mercato asiatico. |
| **RateGain** | Tecnologia per intelligenza tariffe e distribuzione. |
| **RateTiger** | Distribuzione in tempo reale di tariffe e disponibilità. |
| **CloudBeds** | PMS cloud, channel manager e motore di prenotazione. |
| **SiteMinder** | Principale channel manager e piattaforma di distribuzione per hotel. |
| **Exely** | Sincronizzazione tariffe e disponibilità per hotel. |

:::note
Se il tuo channel manager non è elencato, contatta il team di Wink. Nuove integrazioni possono essere aggiunte in base agli standard del protocollo OTA (Open Travel Alliance) supportati da Wink.
:::

## Per sviluppatori

Se stai sviluppando un’integrazione channel manager o PMS con Wink, utilizza gli endpoint API del Channel Manager:

- **Aggiornamenti tariffe** — Invia modifiche tariffe a Wink.
- **Aggiornamenti disponibilità** — Invia modifiche disponibilità a Wink.
- **Recupero prenotazioni** — Estrai prenotazioni da Wink nel tuo sistema.

L’API Channel Manager segue gli standard del protocollo OTA (SOAP/XML) per la compatibilità con i sistemi di ospitalità esistenti.

## Notifiche webhook

Puoi iscriverti agli eventi webhook del channel manager per ricevere notifiche in tempo reale:

- `channel-manager.update.rate` — Aggiornamento tariffe ricevuto.
- `channel-manager.update.availability` — Aggiornamento disponibilità ricevuto.
- `channel-manager.update` — Aggiornamento generale channel manager.

Consulta il [Catalogo Eventi Webhook](/it/developers/webhook-events#channel-manager-events) per i dettagli.
