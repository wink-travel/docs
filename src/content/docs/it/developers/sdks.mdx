Accelera il time-to-market utilizzando uno dei nostri Software Development Kit (SDK) per linguaggi ufficialmente supportati. Se al momento non supportiamo il tuo linguaggio preferito, accedi all'API che desideri utilizzare e scarica la specifica OpenAPI che mettiamo a disposizione per ciascuna API in cima alla pagina. Puoi generare il tuo SDK effettuando un fork di uno dei nostri repository SDK pubblici o utilizzando uno dei generatori OpenAPI.

## Giava

Questa libreria è stata compilata per Java 24. È configurata per utilizzare il WebClient reattivo ed è ottimizzata per l'uso con Spring Framework. Tuttavia, Spring non è un requisito, ma offre il supporto OAuth2 pronto all'uso.
