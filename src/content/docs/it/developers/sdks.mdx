Accelera il time-to-market utilizzando uno dei nostri Software Development Kit ufficialmente supportati per linguaggi specifici.
Se attualmente non supportiamo il tuo linguaggio preferito, vai all'API che vuoi utilizzare e scarica la specifica OpenAPI che rendiamo disponibile per ogni API in cima alla pagina. Puoi generare il tuo SDK forchettando uno dei nostri repository SDK pubblici o utilizzando uno dei generatori di OpenAPI.

## Java

Questa libreria è stata compilata per Java 24. È configurata per usare il WebClient reattivo ed è ottimizzata per l'uso con il Spring Framework. Tuttavia, Spring non è un requisito ma offre il supporto OAuth2 out of the box.
