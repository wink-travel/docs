Tutte le nostre API REST sono indipendenti dal linguaggio e conformi a Open API. Inizia a testare senza scrivere una singola riga di codice.

Se non hai ancora creato il tuo account Wink, [inizia qui](/user/register).

## Payment API

Gli endpoint Payment sono per sviluppatori che vogliono acquistare inventario di viaggio. Questo può essere fatto tramite API come agente di viaggio registrato o usando la nostra API in combinazione con il nostro widget di pagamento conforme PCI per tutte le altre entità.
La Payment API espone endpoint per affiliati e hotel per tracciare prenotazioni, analisi e disponibilità di fondi. Possono anche scegliere di prelevare fondi disponibili sul loro conto bancario.

</LinkButton>
