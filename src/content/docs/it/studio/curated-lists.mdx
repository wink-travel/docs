Le liste curate sono una funzionalità che ti permette di raggruppare e ordinare l'inventario che hai selezionato tra le cose trovate durante la ricerca di inventario di viaggio.
Una volta creata, puoi rendere la lista disponibile ai tuoi utenti mettendola in vendita come [griglia](/it/studio/grids) o come [mappa](/it/studio/maps) tramite [WinkLinks](/it/link-manager/wink-links), il nostro [plugin WordPress](/it/developers/wordpress/) o incorporando uno dei nostri [Web Components](/it/developers/web-components) nel tuo sito.

Di default, ogni account affiliato inizia con una lista `Preferiti` che puoi usare subito. `Preferiti` è speciale e non può essere rimossa.
Tutte le liste curate ti permettono di fare quanto segue:

## Crea una lista

- Dalla dashboard, clicca su `Inventario > Liste curate` nella barra di navigazione.
- Crea una nuova lista curata cliccando sul pulsante `Crea una nuova lista`.
- Ti verrà chiesto di dare un nome alla tua lista. *Esempio: Top 3 Scelte NYC*.
- Clicca sul pulsante `Salva`.

## Aggiungi un elemento alla lista

Puoi aggiungere un elemento a qualsiasi lista dai risultati di ricerca.

- Vai su [Ricerca](/it/studio/search)
- Cerca qualcosa che vuoi aggiungere alla tua lista.
- Una volta trovato, clicca sul link `Azioni` in basso a sinistra nella scheda del risultato di ricerca.
- Clicca su `Aggiungi a...` e poi seleziona la lista curata a cui vuoi aggiungere questo inventario.
- Torna alla pagina delle liste curate per verificare che l'elemento sia stato aggiunto alla tua lista.

Ogni lista ha alcune azioni che puoi eseguire su di essa.

- Converti la lista in una [mappa](/it/studio/maps).
- Converti la lista in una [griglia](/it/studio/grids).
- Rimuovi la lista (*eccetto Preferiti*).

## API

Gli sviluppatori che vogliono gestire le liste curate possono andare su [Developers > API > Browse](/it/developers/apis/#browse-api).
