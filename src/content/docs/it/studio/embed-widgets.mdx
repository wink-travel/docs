Studio offre diversi widget incorporabili che puoi aggiungere al tuo sito web per aumentare le prenotazioni.

Per accedere ai widget da incorporare, vai su `Tools > Miscellaneous` dalla barra laterale.

## Widget disponibili

| Widget | Descrizione |
| -- | -- |
| **Account button** | Un pulsante brandizzato che collega alla tua pagina affiliata |
| **Itinerary button** | Un pulsante che apre il selettore di itinerario |
| **Itinerary form** | Un modulo inline per selezionare date e ospiti |
| **Search button** | Un pulsante che apre la ricerca delle proprietà |
| **Lookup button** | Un pulsante per cercare un inventario specifico |

## Come incorporare

Ogni widget fornisce uno snippet di codice che puoi copiare e incollare nell’HTML del tuo sito web. Lo snippet include il tuo **Client ID** e **Customization ID** così il widget è automaticamente collegato al tuo account affiliato.

:::note
Se usi WordPress, offriamo anche un plugin per WordPress per un’integrazione più semplice.
:::
