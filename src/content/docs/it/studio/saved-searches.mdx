Le ricerche salvate fanno esattamente ciò che suggerisce il nome. Durante la ricerca di un inventario di viaggio, hai la possibilità di salvare la query di ricerca per un utilizzo successivo.

## Salvataggio della query di ricerca

Quando inizi a cercare l'inventario (vedi[Ricerca](/studio/search)), hai la possibilità di perfezionare i tuoi filtri di ricerca.
In fondo al modulo di ricerca c'è l'opzione per`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Salva una ricerca" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Mostra il collegamento per salvare la query di ricerca*</figcaption>
</figure>


Cliccando su questo link avrai la possibilità di dare un nome a questa ricerca (*ad esempio la mia biblioteca di hotel magici*) qualcosa che ricorderai più tardi.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Dai un nome alla tua ricerca" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Dai un nome alla tua ricerca*</figcaption>
</figure>


Dagli un nome e clicca su`Save search`pulsante. Dopodiché, puoi navigare verso`Inventory > Saves Searches`dalla barra di navigazione principale per visualizzare il record appena creato.

## Elenca le azioni

La ricerca salvata può essere utilizzata per presentare ai tuoi utenti l'inventario dei viaggi sotto forma di griglia o mappa e per visualizzare i risultati della ricerca nel modo desiderato.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Azioni di ricerca salvate" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Query di ricerca salvata senza filtri applicati*</figcaption>
</figure>


Ogni query di ricerca salvata ha delle azioni che puoi eseguire su di essa:

* Convertire la query di ricerca in un[mappa](/studio/maps).
* Convertire la query di ricerca in un[griglia](/studio/grids).
* Rimuovi la ricerca salvata.

## API

Gli sviluppatori che desiderano gestire le ricerche salvate possono andare su[Sviluppatori > API > Sfoglia](/developers/apis/#browse-api).
