Le ricerche salvate fanno esattamente ciò che suggerisce il nome. Durante la ricerca dell'inventario di viaggio, hai la possibilità di salvare la tua query di ricerca per un uso futuro.

## Salvare la tua query di ricerca

Quando inizi a cercare l'inventario (vedi [Search](/it/studio/search) ), puoi affinare i filtri di ricerca.
In fondo al modulo di ricerca c'è l'opzione `Save search`.

Cliccando su questo link avrai la possibilità di dare un nome a questa ricerca (*es. La mia libreria di hotel magici*), qualcosa che ricorderai per dopo.

<figure>
<CloudinaryImage
      width="350"
      height="329"
      resize="scale"
      quality="best"
      alt="Dai un nome alla tua ricerca"
      src="academy/name-save-search"
      errorImage="noimage_opaque_nyrtl0.png"
    />
    <figcaption>*Dai un nome alla tua ricerca*</figcaption>
</figure>

Dagli un nome e clicca sul pulsante `Save search`. Dopo di che, puoi navigare su `Inventory > Saves Searches` dalla barra di navigazione principale per vedere il tuo nuovo record creato.

## Azioni nella lista

La tua ricerca salvata può essere utilizzata per mostrare l'inventario di viaggio ai tuoi utenti sotto forma di griglia o mappa e far visualizzare i risultati della ricerca nel modo che preferisci.

<figure>

Ogni query di ricerca salvata ha azioni che puoi eseguire su di essa:

- Converti la query di ricerca in una [mappa](/it/studio/maps).
- Converti la query di ricerca in una [griglia](/it/studio/grids).
- Rimuovi la ricerca salvata.

## API

Gli sviluppatori che vogliono gestire le ricerche salvate possono andare su [Developers > API > Browse](/it/developers/apis/#browse-api).
