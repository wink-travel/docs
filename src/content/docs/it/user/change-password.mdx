Se vuoi cambiare la password del tuo account utente Wink, segui questi passaggi:

## Aggiorna password

<Steps>
  1. Devi aver effettuato l'accesso a uno dei nostri siti web.
  2. Fai clic sull'icona del tuo profilo nell'angolo in alto a destra e apparirà un menu.
  3. Fare clic su `Change password` collegamento.
  4. Segui le istruzioni per cambiare la tua password.
  5. Clic `Save`.
</Steps>


La prossima volta che effettuerai l'accesso, ti servirà la nuova password.

## Ha dimenticato la password

A volte è necessario reimpostare la password utente.

Ecco i passaggi per reimpostare la password utente:

<Steps>
  1. Fare clic su `Forgot password` collegamento dalla pagina di accesso.
  2. Digita il nome utente/indirizzo e-mail dell'account che desideri reimpostare.
  3. Clic `Forgot password` pulsante. *Verrai reindirizzato al modulo per reimpostare la password.*
  4. Controlla la tua posta elettronica.
</Steps>


<figure>
  <CloudinaryImage width="350" height="354" resize="scale" quality="best" alt="Ha dimenticato la password" src="academy/forgot-password" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Modulo password dimenticata*</figcaption>
</figure>


<Steps>
  1. Incolla il codice ricevuto via e-mail nel nuovo modulo insieme alla nuova password.
  2. Conferma la tua nuova password.
  3. Clic `Reset password` pulsante. *Verrai reindirizzato alla pagina di accesso.*
</Steps>


<figure>
  <CloudinaryImage width="350" height="467" resize="scale" quality="best" alt="Reimposta password" src="academy/reset-password" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Modulo di reimpostazione della password*</figcaption>
</figure>


La tua password è stata reimpostata correttamente. Puoi accedere a Wink con le tue nuove credenziali.
