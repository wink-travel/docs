Se vuoi cambiare la password del tuo account utente Wink, segui questi passaggi:

## Aggiorna password

<Steps>
1. Devi essere connesso a uno dei nostri siti web.
2. Clicca sull'icona del profilo, nell'angolo in alto a destra, e apparirà un menu.
3. Clicca sul link `Cambia password`.
4. Segui le istruzioni per cambiare la tua password.
5. Clicca su `Salva`.
</Steps>

La prossima volta che effettuerai l'accesso, ti servirà la nuova password.

## Password dimenticata

A volte è necessario reimpostare la password utente.

Ecco i passaggi per reimpostare la password utente:

<Steps>
1. Clicca sul link `Password dimenticata` dalla pagina di accesso.
2. Inserisci il nome utente / e-mail dell'account che desideri reimpostare.
3. Clicca sul pulsante `Password dimenticata`. *Verrai reindirizzato al modulo di reimpostazione della password.*
4. Controlla la tua e-mail.
</Steps>

2. Conferma la tua nuova password.
3. Clicca sul pulsante `Reimposta password`. *Verrai reindirizzato di nuovo alla pagina di accesso.*
</Steps>
