Un utente Wink può scegliere di lasciarti gestire uno dei suoi account di struttura, affiliato o agente di viaggio. Quando ciò accade, riceverai un'e-mail con una richiesta di invito.

import { Steps } from '@astrojs/starlight/components';

Per vedere i tuoi inviti Wink attuali, segui questi passaggi:

<Steps>
1. Devi aver effettuato l'accesso a uno dei nostri siti web.
2. Clicca sull'icona del tuo profilo, nell'angolo in alto a destra, e apparirà un menu.
3. Clicca sul link `Inviti`.
4. Interagisci con i tuoi inviti accettandoli o rifiutandoli.
</Steps>
