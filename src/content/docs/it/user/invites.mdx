Un utente Wink può scegliere di affidarti la gestione di uno dei suoi account di proprietà, affiliati o agenzie di viaggio. In tal caso, riceverai un'e-mail con una richiesta di invito.

import { Steps } from '@astrojs/starlight/components';

Per visualizzare i tuoi inviti Wink attuali, segui questi passaggi:

<Steps>
  1. Devi aver effettuato l'accesso a uno dei nostri siti web.
  2. Fai clic sull'icona del tuo profilo nell'angolo in alto a destra e apparirà un menu.
  3. Fare clic su `Invites` collegamento.
  4. Interagisci con i tuoi inviti tramite `Accepting` O `Rejecting` loro.
</Steps>
