Rezervačný systém je dostupný v dvoch verziách, aby podporil rôzne potreby affiliate partnera, správcovskej spoločnosti a hotela.

## Online cestovná agentúra

Pre affiliate partnera je najvhodnejšie použiť verziu online cestovnej agentúry. Dôvodom je, že stránka je nastavená tak, aby vyhovovala cestovateľom, ktorí chcú prezerať viacero nehnuteľností.

Domovskú stránku tejto aplikácie nájdete nižšie.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Internetový rezervačný systém

Pre hotel alebo správcovskú spoločnosť je najvhodnejšie použiť verziu internetového rezervačného systému. Tento typ používateľa má záujem zvýrazniť jednu nehnuteľnosť alebo viacero nehnuteľností v správe.

Domovskú stránku tejto aplikácie nájdete nižšie.

Hotely a affiliate partneri môžu prispôsobiť `vzhľad a dojem` rezervačného systému úpravou farebnej palety.

## Konfigurácia

Väčšinou nie je potrebné priamo zdieľať žiadnu z týchto aplikácií. Vytvorili sme rôzne spôsoby, ako môžete zdieľať cestovný inventár, ktoré môžete objaviť v [Wink Studio](/studio/what-is-studio) alebo cez [WinkLinks](/link-manager/wink-links).

Ak však chcete manuálne prepojiť na niektorú z týchto stránok, uistite sa, že do URL pridáte svoje ID klienta a ID prispôsobenia ako parametre.

Svoje ID klienta nájdete takto:

<Steps>
1. Prihláste sa do [Wink Extranet](/extranet/what-is-extranet) alebo [Wink Studio](/studio/what-is-studio).
2. Kliknite na ikonu svojho profilu v pravom hornom rohu a vyberte [Aplikácie](/developers/applications).
3. Použite aplikáciu, ktorá je prepojená s Wink účtom, s ktorým chcete pracovať.
4. Skopírujte ID klienta z tejto aplikácie a vložte ho do URL ako parameter požiadavky. Príklad: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Svoje ID prispôsobenia ako nehnuteľnosti nájdete takto:

<Steps>
1. Prihláste sa do [Wink Extranet](/extranet/what-is-extranet).
2. Vyberte nehnuteľnosť, s ktorou chcete pracovať.
3. V hlavnom navigačnom paneli kliknite na `Účet > Branding`
4. Skopírujte ID prispôsobenia a vložte ho do URL ako parameter požiadavky. Príklad: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Svoje ID prispôsobenia ako affiliate partner nájdete takto:

<Steps>
1. Prihláste sa do [Wink Studio](/studio/what-is-studio).
2. Vyberte účet, s ktorým chcete pracovať.
3. V hlavnom navigačnom paneli kliknite na `Nástroje > Prispôsobenie`
4. Skopírujte ID prispôsobenia z prispôsobenia, ktoré chcete použiť, a vložte ho do URL ako parameter požiadavky. Príklad: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Výsledkom je URL, ktoré vyzerá približne takto:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Ak chcete ísť priamo na stránku hotela, URL by vyzerala približne takto:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Ak nezadáte ID klienta a/alebo ID prispôsobenia, Wink OTA použije predvolené nastavenie.
:::

## IBE

Ak ste správcovská spoločnosť, chcete prepojiť na domovskú stránku Wink IBE. Domovská stránka zobrazí vašu určenú tému spolu s [nehnuteľnosťami, ktoré zastupujete](/studio/agency-representation).

Výsledkom je URL, ktoré vyzerá približne takto:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Ak ste nehnuteľnosť, chcete prepojiť priamo na vstupnú stránku nehnuteľnosti Wink IBE.

Táto URL by vyzerala približne takto:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
Ak nezadáte ID klienta a/alebo ID prispôsobenia, Wink IBE sa NENAČÍTA. Tieto parametre sú povinné.
:::

## Príklad: Stránka nehnuteľnosti

Každá nehnuteľnosť na Wink má svoju vlastnú jedinečnú vstupnú stránku v oboch webových verziách. Všetky externé odkazy smerujú cestovateľov na jednu z týchto vstupných stránok.

Vstupná stránka obsahuje všetky informácie, ktoré cestovateľ potrebuje na to, aby sa mohol informovane rozhodnúť, či chce zostať v danej nehnuteľnosti.
