Ak ste integrátor, ako napríklad Wink, a chcete namapovať svojich externých príjemcov platieb na použitie s TripPay, môžete to urobiť jedným z dvoch spôsobov:

* Manuálne mapovanie
* Používanie[Platobné rozhranie API](/developers/apis/#payment-api)(*Pozri`Account Mappings`*)

:::tip
Pokiaľ nepracujete len s niekoľkými rovnakými vlastnosťami, odporúčame vám použiť API na automatizáciu tohto kroku. :::

Nižšie vás prevedieme manuálnym postupom mapovania externého príjemcu platby.*Používa API pod kapotou.*

<Steps>
  1. Prihlásiť sa [TripPay](https://pay.wink.travel).
  2. Vyberte si svoj integračný účet.
  3. Kliknite `Settings > Mappings` z hlavného navigačného panela.
  4. Kliknite na `Create mapping` tlačidlo. Zobrazí sa vám formulár mapovania.
  5. **Lokálny účet** Začnite písať názov existujúceho TripPay, ktorý chcete namapovať.
  6. **Externý identifikátor** Vložte svoje externé ID pre túto entitu tak, ako existuje vo vzdialenom systéme.
  7. Kliknite `Save` Ak chcete pokračovať, budete presmerovaní späť na zoznam mapovaní.
</Steps>


Po zavedení mapovaní teraz TripPay vie, ktoré účty má pripísať ako príjemcov rezervácie, keď sa na ne odvoláte pomocou svojich externých identifikátorov.

## API

Vývojári, ktorí chcú spravovať`Payments`, prejdite na[Vývojári > API > Platby](/developers/apis/#payment-api).
