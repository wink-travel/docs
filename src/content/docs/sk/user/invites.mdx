Používateľ služby Wink si môže zvoliť, že vám umožní spravovať jeden zo svojich účtov nehnuteľností, partnerských spoločností alebo cestovných kancelárií. Keď sa tak stane, dostanete e-mail so žiadosťou o pozvánku.

import { Steps } from '@astrojs/starlight/components';

Ak chcete zobraziť svoje aktuálne pozvánky Wink, postupujte podľa týchto krokov:

<Steps>
  1. Musíte byť prihlásený/á na jednej z našich webových stránok.
  2. Kliknite na ikonu svojho profilu v pravom hornom rohu a zobrazí sa vám ponuka.
  3. Kliknite na `Invites` odkaz.
  4. Reagujte so svojimi pozvánkami buď `Accepting` alebo `Rejecting` ich.
</Steps>
