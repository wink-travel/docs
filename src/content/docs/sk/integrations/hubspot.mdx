HubSpot CMS podporuje vlastné HTML moduly a kód hlavičky/pätky pre celú stránku, čo uľahčuje vloženie komponentov Wink na marketingové stránky a vstupné stránky.

:::note
Vkladanie kódu pre celú stránku vyžaduje prístup k **HubSpot CMS Hub** (Starter alebo vyšší). Vlastné HTML moduly sú dostupné vo všetkých úrovniach CMS.
:::

## Načítanie skriptov Wink pre celú stránku

<Steps>
1. V HubSpot prejdite na **Marketing → Súbory a šablóny → Nástroje na dizajn**.
2. Otvorte aktívny **motív** `main.css` alebo použite globálne nastavenia obsahu.
3. Alternatívne prejdite na **Nastavenia → Webová stránka → Stránky → Vlastný kód**.
4. Do poľa **Head HTML** pridajte štýlový súbor:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
5. Do poľa **Footer HTML** pridajte skript a načítač aplikácie (nahraďte `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
6. Uložte a publikujte.
</Steps>

:::tip
Váš Client ID nájdete vo svojom účte Wink v sekcii [Applications](/sk/portal/settings/applications).
:::

## Pridanie komponentu cez vlastný HTML modul

<Steps>
1. V Nástrojoch na dizajn vytvorte nový **modul** typu **Rich Text** alebo **HTML**.
2. Do predvoleného obsahu modulu vložte HTML komponentu:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
3. Uložte modul a pretiahnite ho do ľubovoľnej šablóny stránky alebo stránky pomocou editora stránok.
</Steps>

## Pridanie komponentu priamo v editore stránok

Ak nechcete vytvárať modul, môžete použiť sekciu **Custom HTML** v drag-and-drop editore:

<Steps>
1. Otvorte stránku v editore stránok HubSpot.
2. Kliknite na **Pridať** a vyberte sekciu **Custom HTML** alebo **Embed**.
3. Vložte značku komponentu do HTML poľa.
4. Kliknite na **Použiť** a publikujte stránku.
</Steps>

## Dostupné komponenty

| Komponent | HTML značka | Účel |
|---|---|---|
| Content loader | `<wink-content-loader>` | Zobrazenie hotelových kariet, mriežok, máp |
| Lookup | `<wink-lookup>` | Vyhľadávací panel pre destinácie |
| Search button | `<wink-search-button>` | Otvoriť výber itinerára |
| Account button | `<wink-account-button>` | Prihlásenie / používateľské menu |
| Itinerary button | `<wink-itinerary-button>` | Zobraziť aktuálny itinerár |
| Itinerary picker | `<wink-itinerary-picker>` | Kompletný formulár itinerára |
| Shopping cart | `<wink-shopping-cart-button>` | Tlačidlo súhrnu košíka |

Pozrite si [Web Components](/sk/integrations/web-components) pre úplný zoznam atribútov.
