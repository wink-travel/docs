**Wink TYPO3 Extension** integruje inventár Wink travel do TYPO3 13 ako natívne obsahové prvky. Redaktori pridávajú komponenty Wink na stránky pomocou štandardného editora stránok TYPO3 — nie sú potrebné žiadne zmeny šablóny.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"

  description="Zdrojový kód, príručka pre vývojárov a pokyny na prispievanie."

## Požiadavky

- TYPO3 13.x
- PHP 8.2+
- Composer

## Inštalácia

<Steps>
1. Nainštalujte cez Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivujte rozšírenie:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. V administrácii TYPO3 prejdite na **Admin Tools → Settings → Extension Configuration → wink**.
4. Zadajte svoj **Client ID**, **Client Secret** a vyberte prostredie. Uložte.
   Pre získanie prihlasovacích údajov prejdite na [Applications](/sk/portal/settings/applications).
5. Pridajte statickú šablónu TypoScript do svojej koreňovej šablóny:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Pridanie obsahového prvku na stránku

<Steps>
1. Otvorte stránku v module **Page**.
2. Kliknite na tlačidlo **+** v oblasti obsahu pre pridanie nového prvku.
3. Vyberte jeden z obsahových prvkov **Wink Travel**:
   - **Wink Content** — hotelové karty, zoradené mriežky, mapy
   - **Wink Lookup** — vyhľadávací panel destinácií
   - **Wink Search** — tlačidlo výberu itinerára
   - **Wink Account** — tlačidlo prihlásenia
   - **Wink Itinerary** — tlačidlo stavu itinerára
   - **Wink Cart** — tlačidlo nákupného košíka
4. Pre **Wink Content** použite polia FlexForm na výber **Layout Type** a zadajte **Layout ID**.
5. Uložte a publikujte stránku.
</Steps>

## Dostupné obsahové prvky

| Prvok | Účel |
|---|---|
| **Wink Content** | Hotelové karty, zoradené mriežky, mapy a ďalšie |
| **Wink Lookup** | Vyhľadávací panel destinácií |
| **Wink Search** | Ikonové tlačidlo otvárajúce výber itinerára |
| **Wink Account** | Tlačidlo prihlásenia / používateľského účtu |
| **Wink Itinerary** | Tlačidlo itinerára zobrazujúce aktuálnu cestu |
| **Wink Cart** | Tlačidlo so súhrnom nákupného košíka |
