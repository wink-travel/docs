Studio poskytuje niekoľko vložiteľných widgetov, ktoré môžete pridať na svoju webovú stránku na podporu rezervácií.

Pre prístup k vloženým widgetom prejdite v bočnom paneli do `Nástroje > Rôzne`.

## Dostupné widgety

| Widget | Popis |
| -- | -- |
| **Tlačidlo účtu** | Značkové tlačidlo, ktoré odkazuje na vašu partnerskú stránku |
| **Tlačidlo itinerára** | Tlačidlo, ktoré otvorí selektor itinerára |
| **Formulár itinerára** | Inline formulár na výber dátumu a hostí |
| **Tlačidlo vyhľadávania** | Tlačidlo, ktoré otvorí vyhľadávanie nehnuteľností |
| **Tlačidlo vyhľadávania v inventári** | Tlačidlo na vyhľadávanie konkrétneho inventára |

## Ako vložiť

Každý widget poskytuje kódový útržok, ktorý môžete skopírovať a vložiť do HTML vašej webovej stránky. Útržok obsahuje vaše **Client ID** a **Customization ID**, takže widget je automaticky prepojený s vaším partnerským účtom.

:::note
Ak používate WordPress, poskytujeme aj WordPress plugin pre jednoduchšiu integráciu.
:::
