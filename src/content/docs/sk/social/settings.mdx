# Nastavenia

Keď začnete používať Social, opýtame sa vás, ako chcete, aby sme vytvárali váš obsah. Tieto možnosti máte k dispozícii kedykoľvek, takže ich môžete neskôr jednoducho zmeniť.

Pre správu nastavení prejdite do `Preferences` v bočnom paneli.

Sekcia preferences obsahuje základné nastavenia generovania obsahu:

1. **Preferred customization** Vyberte, ktorú prispôsobenú šablónu chcete použiť pri generovaní obsahu. Prispôsobenia ovplyvňujú farby témy, predvolený itinerár, menu, jazyk a ďalšie. Prispôsobenia vytvárate vo svojom účte na [my.wink.travel](https://my.wink.travel).
2. **Auto-approve** Ak je zapnuté, generované príspevky sa automaticky publikujú bez potreby vášho schválenia. Zapnite túto možnosť, keď budete mať dôveru v kvalitu obsahu, ktorý pre vás vytvárame.
3. **Approver name** Meno osoby alebo tímu, ktorý bude dostávať e-maily so žiadosťou o schválenie.
4. **Approver e-mail** E-mailová adresa, na ktorú budú zasielané žiadosti o schválenie nadchádzajúcich príspevkov.

## Content creators

Social ponúka mnoho tvorcov obsahu na výber. Každý tvorca sa zameriava na špecifický spôsob vytvárania jedinečného obsahu pre vašu nehnuteľnosť pomocou AI. Každý tvorca má atribút **Weight** — vyššia hodnota zvyšuje pravdepodobnosť, že bude vybraný pri generovaní príspevkov.

Tu sú všetci dostupní tvorcovia obsahu:

| Creator | Popis |
| -- | -- |
| **Destination** | Využíva vašu lokalitu na vytvorenie užitočného textu o okolí |
| **Room type** | Vytvára rezervovateľný príspevok s jedným z vašich typov izieb s AI generovaným textom a obrázkami izieb |
| **Activity** | Vytvára rezervovateľný príspevok s jednou z vašich aktivít |
| **Add-On** | Vytvára rezervovateľný príspevok s jedným z vašich doplnkov |
| **Attraction** | Vytvára rezervovateľný príspevok s jednou z vašich atrakcií |
| **Place** | Vytvára rezervovateľný príspevok s jedným z vašich miest |
| **Meeting room** | Vytvára rezervovateľný príspevok s jednou z vašich zasadacích miestností |
| **Restaurant** | Vytvára rezervovateľný príspevok s jednou z vašich reštaurácií |
| **Spa** | Vytvára rezervovateľný príspevok s jedným z vašich spa zariadení |
| **Property** | Vytvára rezervovateľný príspevok s vašou najlepšie ohodnotenou izbou s AI generovaným textom a obrázkami nehnuteľnosti |
| **User sentiment** | Zhrňuje recenzie hostí z celého webu do príspevku založeného na sentimentoch, ktorý zdôrazňuje, čo hostia na vašej nehnuteľnosti milujú |
| **Funny** | Vytvára ľahké, zábavné príspevky, ktoré udržujú váš feed zaujímavý |

:::tip[Profesionálny tip]
Uistite sa, že máte typ inventára potrebný pre vybraných tvorcov obsahu. Napríklad, ak vyberiete "Restaurant", musíte mať vo svojej nehnuteľnosti aspoň jednu reštauráciu.
:::

## Tone of voice

Tón hlasu určuje štýl jazyka, ktorý AI používa pri generovaní obsahu. Vyberte tóny, ktoré zodpovedajú osobnosti vašej značky. Rovnako ako tvorcovia obsahu, aj každý tón má **Weight**, ktorý ovplyvňuje, ako často sa používa.

Tu sú dostupné tóny:

| Tone | Štýl |
| -- | -- |
| **Neutral** | Pokojný, autoritatívny, faktický. Žiadny prehnaný nadšený tón, žiadne vtipy — len spoľahlivé, dobre štruktúrované informácie |
| **Friendly** | Teplý, priateľský, s úsmevom v texte, s neformálnymi skratkami a občasnými emotikonmi |
| **Luxury** | Sofistikovaný, vytvorený. Slová ako "exkluzívny" a "na mieru", ktoré čitateľom dodávajú pocit výnimočnosti |
| **Foodie** | Zmyselné, lákavé opisy. Jazyk jedla, ktorý čitateľov hladí chuťové poháriky |
| **Adventurous** | Odvážny, nebojácny, energický prieskumník. Živé zmyslové opisy, ktoré robia všetko epickým |
| **Family** | Rodinne prívetivý, plný radosti. Teplá rodičovská energia o bezpečných a zábavných zážitkoch |
| **Romantic** | Poetický, nostalgický, srdce rozbúrené. Západy slnka, jemný jazyk a láska |
| **Humorous** | Sarkastický, rýchle vtipné poznámky, šikovné slovné hry. Číta sa ako stand-up vystúpenie |
| **Eco** | Úprimný a spravodlivý. Oslavuje zelené alternatívy a udržateľné voľby |
| **Local insider** | Veľmi špecifické miestne odkazy, varuje pred turistickými pascami, používa miestny slang |
| **Inclusive** | Vítajúci, oslavujúci a potvrdzujúci. Každý sa cíti viditeľný a cenený |
| **Mystical** | Éterický, duchovný, kozmické pravdy. Jemné pozvania na hľadanie vnútra |
| **Futuristic** | Elegantný, s pohľadom do budúcnosti. Miluje výrazy ako "paradigmatický posun" a optimistické vízie budúcnosti |
| **Budget** | Šikovný, vynaliezavý, hrdý na úspornosť. Oslavuje dostupné nálezy a bezplatné zážitky |

## Post types

Typy príspevkov určujú cieľ a štruktúru generovaného obsahu. Každý typ príspevku má **Weight**, ktorý ovplyvňuje, ako často sa používa.

| Post type | Popis |
| -- | -- |
| **Transactional** | Jasný, stručný, zameraný na konverziu. Podporuje rezervácie a priame akcie pomocou faktov, cien a ďalších krokov |
| **Awareness** | Obsah na budovanie značky, ktorý má za cieľ predstaviť vašu nehnuteľnosť a zvýšiť jej viditeľnosť medzi potenciálnymi hosťami |
| **Sentiment** | Emocionálne rezonujúci obsah, ktorý vytvára pozitívne asociácie a buduje emocionálne spojenie s vaším publikom |

Možno chcete zamerať svoje marketingové úsilie na konkrétny jazyk a región. Povedzte nám, pre ktoré jazyky chcete vytvárať obsah a aké časové pásmo vás zaujíma, a my urobíme zvyšok. Príspevky sa publikujú v plánovanom čase vo vybranom časovom pásme.

*Väčšina nehnuteľností používa jeden jazyk a predvolene svoj miestny jazyk.*

:::
