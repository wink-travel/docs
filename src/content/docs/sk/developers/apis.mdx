## Platobné rozhranie API

Platobné koncové body sú určené pre vývojárov, ktorí si chcú zakúpiť cestovné vybavenie. Toto je možné vykonať prostredníctvom rozhrania API ako registrovaný cestovný agent alebo pomocou nášho rozhrania API v spojení s naším platobným widgetom kompatibilným s PCI pre všetky ostatné subjekty.
Platobné rozhranie API sprístupňuje koncové body pre partnerov a hotely na sledovanie rezervácií, analýz a dostupnosti finančných prostriedkov. Môžu sa tiež rozhodnúť vybrať dostupné finančné prostriedky na svoj bankový účet.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Zistite viac
</LinkButton>
