Jeden endpoint na získanie informácií o whitelabel a prispôsobení pre rezervačný engine.

</LinkButton>

## Taxonomy API

Taxonomy endpointy sú pre vývojárov, ktorí chcú konzumovať a vytvárať cestovný inventár a potrebujú taxonómie štandardných a nestandardných kódov pre typy inventára, triedy, stavy a pod.

<LinkButton href="https://developer.wink.travel/reference" variant="secondary" icon="external" target="_blank">
Zistiť viac
</LinkButton>

## Analytics API

Analytics API poskytuje endpointy pre kohokoľvek, kto chce získať prehľad o rebríčkoch lídrov, ako aj hlboký pohľad na časové rady, údaje na úrovni platformy; obsahujúce širokú škálu metrík.

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
Zistiť viac
</LinkButton>

## Payment API

Payment endpointy sú pre vývojárov, ktorí chcú nakupovať cestovný inventár. Toto je možné vykonať cez API ako registrovaný cestovný agent alebo použiť naše API v spojení s naším PCI-kompatibilným platobným widgetom pre všetky ostatné entity.
Payment API poskytuje endpointy pre partnerov a hotely na sledovanie rezervácií, analytiku a dostupnosť finančných prostriedkov. Môžu si tiež zvoliť výber dostupných prostriedkov na svoj bankový účet.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Zistiť viac
</LinkButton>
