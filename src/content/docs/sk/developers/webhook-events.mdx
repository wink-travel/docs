Táto stránka uvádza všetky udalosti webhookov, na ktoré sa môžete prihlásiť na Wink. Pre inštrukcie na nastavenie si pozrite [Sprievodcu integráciou webhookov](/sk/guides/webhook-integration) alebo [stránku správy webhookov](/sk/my-account/webhooks).

## Ako sa prihlásiť na odber

1. Prejdite v portáli do **Aplikácie > Webhooky**.
2. Vytvorte webhook a vyberte udalosti, ktoré chcete prijímať.
3. Wink pošle HTTP POST na váš endpoint vždy, keď nastane daná udalosť.

Pre kompletné kroky nastavenia a najlepšie postupy si pozrite [Integráciu webhookov](/sk/guides/webhook-integration).

## Udalosti účtu

Udalosti súvisiace s životným cyklom a nastaveniami vášho TripPay účtu.

| Udalosť | Kľúč | Popis |
|---|---|---|
| Účet vytvorený | `account.create` | Bol vytvorený nový účet. |
| Účet v revízii | `account.under_review` | Účet je v čakaní na kontrolu. |
| Účet aktualizovaný | `account.updated` | Údaje účtu sa zmenili. |
| Stav účtu aktualizovaný | `account.status.updated` | Stav účtu sa zmenil (napr. schválený, pozastavený). |
| Adresa účtu aktualizovaná | `account.address.updated` | Adresa účtu bola upravená. |
| Detaily účtu aktualizované | `account.details.updated` | Polia detailov účtu sa zmenili. |
| Informácie o účte aktualizované | `account.information.update` | Všeobecné informácie o účte boli aktualizované. |
| Správcovia účtu aktualizovaní | `account.managers.update` | Zoznam správcov účtu sa zmenil. |
| Profilová fotografia aktualizovaná | `account.profile_picture.updated` | Profilová fotografia účtu bola zmenená. |
| Agent aktualizovaný | `account.agent.updated` | Informácie o agentovi na účte boli upravené. |
| URL názov aktualizovaný | `account.url-name.update` | Vanity URL účtu sa zmenila. |
| Unikátny ID aktualizovaný | `account.unique-id.update` | Unikátny identifikátor účtu bol upravený. |

### Úlohy účtu

| Udalosť | Kľúč | Popis |
|---|---|---|
| Úloha vytvorená | `account.task.create` | Do účtu bola pridaná nová úloha. |
| Úloha aktualizovaná | `account.task.update` | Úloha účtu bola upravená. |
| Úloha odstránená | `account.task.remove` | Úloha účtu bola odstránená. |
| Úloha dokončená | `account.task.complete` | Úloha účtu bola dokončená. |
| Úloha overenia daňového ID dokončená | `account.task.tax_id.complete` | Úloha overenia daňového ID bola dokončená. |

### Získavateľ účtu

| Udalosť | Kľúč | Popis |
|---|---|---|
| Získavateľ vytvorený | `account.acquirer.created` | Do účtu bol pridaný nový získavateľ. |
| Získavateľ aktualizovaný | `account.acquirer.updated` | Získavateľ účtu bol upravený. |
| Získavateľ odstránený | `account.acquirer.removed` | Získavateľ bol z účtu odstránený. |

## Udalosti spravujúcej entity

Udalosti pre spravujúce entity (nehnuteľnosti, pridružené subjekty, spoločnosti) a ich správu.

### Predplatné

| Udalosť | Kľúč | Popis |
|---|---|---|
| Predplatné zriadené | `managing_entity.subscription.provisioned` | Bolo zriadené nové predplatné. |
| Predplatné zmenené | `managing_entity.subscription.changed` | Plán predplatného bol zmenený. |
| Faktúra predplatného zmenená | `managing_entity.subscription.invoice.changed` | Faktúra predplatného bola aktualizovaná. |

### Stav a správa

| Udalosť | Kľúč | Popis |
|---|---|---|
| Stav aktualizovaný | `managing_entity.status.updated` | Stav entity sa zmenil. |
| Entita vymazaná | `managing_entity.deleted` | Spravujúca entita bola vymazaná. |
| Agent aktualizovaný | `managing_entity.agent.updated` | Agent entity bol upravený. |
| Emailové upozornenie agenta | `managing_entity.agent.email.notification` | Agentovi bol odoslaný email s upozornením. |
| Emailové upozornenie manažéra | `managing_entity.manager.email.notification` | Manažérovi bol odoslaný email s upozornením. |

### Životný cyklus manažéra

| Udalosť | Kľúč | Popis |
|---|---|---|
| Manažér pridaný | `managing_entity.manager.added` | Bol pridaný nový manažér. |
| Manažér pozvaný | `managing_entity.manager.invited` | Manažérovi bola zaslaná pozvánka. |
| Manažér odstránený | `managing_entity.manager.removed` | Manažér bol odstránený. |
| Manažér prijatý | `managing_entity.manager.accepted` | Manažér prijal pozvánku. |
| Manažér odmietnutý | `managing_entity.manager.rejected` | Manažér odmietol pozvánku. |
| Vlastník nastavený | `managing_entity.manager.owner.set` | Vlastníctvo entity bolo prevedené. |

### Informácie o entite

| Udalosť | Kľúč | Popis |
|---|---|---|
| Informácie aktualizované | `managing_entity.information.updated` | Informácie o entite boli aktualizované. |
| Adresa aktualizovaná | `managing_entity.city.updated` | Adresa/mesto entity sa zmenilo. |
| Zástupca účtu aktualizovaný | `managing_entity.account_representative.updated` | Zástupca účtu bol zmenený. |
| Profilová fotografia aktualizovaná | `managing_entity.profile_picture.updated` | Profilová fotografia entity bola aktualizovaná. |
