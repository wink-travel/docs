Zrýchlite čas uvedenia na trh pomocou jednej z našich oficiálne podporovaných súprav na vývoj softvéru pre jazyky.
Ak momentálne nepodporujeme váš obľúbený jazyk, prejdite na rozhranie API, ktoré chcete použiť, a stiahnite si špecifikáciu OpenAPI, ktorú pre každé rozhranie API sprístupňujeme v hornej časti stránky. Svoju vlastnú sadu SDK si môžete vygenerovať buď vytvorením rozvetvenia jedného z našich verejných repozitárov SDK, alebo použitím jedného z generátorov OpenAPI.

## Jáva

Táto knižnica bola skompilovaná pre Java 24. Je nakonfigurovaná na používanie reaktívneho WebClienta a je optimalizovaná na použitie so Spring Frameworkom. Spring však nie je povinný, ale poskytuje podporu OAuth2 hneď po vybalení.
