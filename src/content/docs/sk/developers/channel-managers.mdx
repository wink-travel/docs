Wink sa integruje s poprednými channel managermi, systémami na správu nehnuteľností (PMS) a centrálnymi rezervačnými systémami (CRS), aby boli vaše ceny a dostupnosť synchronizované vo všetkých predajných kanáloch.

## Ako to funguje

Integrácie channel managerov umožňujú obojsmerný tok dát medzi vaším systémom na správu nehnuteľností a Winkom:

1. **Push cien** — Váš channel manager posiela aktualizácie cien do Winku. Keď aktualizujete ceny vo vašom PMS, tieto zmeny sa automaticky prejavia vo Winku.
2. **Synchronizácia dostupnosti** — Váš channel manager posiela aktualizácie dostupnosti do Winku. Keď sú izby rezervované alebo uvoľnené vo vašom PMS, inventár vo Winku sa aktualizuje v reálnom čase.
3. **Notifikácie o rezerváciách** — Keď je rezervácia vytvorená vo Winku, váš channel manager je o tom informovaný, aby sa rezervácia zobrazila vo vašom PMS.

## Podporovaní channel manageri

Wink momentálne integruje nasledujúce systémy:

| Channel Manager | Popis |
|---|---|
| **Channex** | Globálna platforma pre distribúciu a správu kanálov. |
| **TravelClick** | Hotelové riešenia a správa kanálov (Amadeus). |
| **Omnibees** | Distribúcia a riadenie príjmov pre hotely. |
| **D-Edge** | Technológie pre pohostinstvo a centrálne rezervačné riešenia. |
| **Allotz** | Alokácia inventára a správa kanálov. |
| **Ezee / Yanolja** | Cloudové PMS a správa kanálov zameraná na ázijský trh. |
| **RateGain** | Technológia pre inteligenciu cien a distribúciu. |
| **RateTiger** | Distribúcia cien a dostupnosti v reálnom čase. |
| **CloudBeds** | Cloudové PMS, channel manager a rezervačný engine. |
| **SiteMinder** | Vedúci hotelový channel manager a distribučná platforma. |
| **Exely** | Synchronizácia cien a dostupnosti pre hotely. |

:::note
Ak váš channel manager nie je na zozname, kontaktujte tím Wink. Nové integrácie môžu byť pridané na základe štandardov protokolu OTA (Open Travel Alliance), ktoré Wink podporuje.
:::

## Pre vývojárov

Ak vytvárate integráciu channel managera alebo PMS s Winkom, použite API endpointy Channel Managera:

- **Aktualizácie cien** — Posielajte zmeny cien do Winku.
- **Aktualizácie dostupnosti** — Posielajte zmeny dostupnosti do Winku.
- **Získavanie rezervácií** — Sťahujte rezervácie z Winku do vášho systému.

API Channel Managera dodržiava štandardy protokolu OTA (SOAP/XML) pre kompatibilitu s existujúcimi systémami v pohostinstve.

## Notifikácie webhookov

Môžete sa prihlásiť na odber udalostí webhookov channel managera, aby ste dostávali notifikácie v reálnom čase:

- `channel-manager.update.rate` — Prijatá aktualizácia cien.
- `channel-manager.update.availability` — Prijatá aktualizácia dostupnosti.
- `channel-manager.update` — Všeobecná aktualizácia channel managera.

Podrobnosti nájdete v [Katalógu udalostí webhookov](/sk/developers/webhook-events#channel-manager-events).
