Studio pruža nekoliko ugradivih vidžeta koje možete dodati na svoj sajt kako biste povećali broj rezervacija.

Da biste pristupili ugradivim vidžetima, idite na `Tools > Miscellaneous` u bočnoj traci.

## Dostupni vidžeti

| Vidžet | Opis |
| -- | -- |
| **Account button** | Brendirano dugme koje vodi do vaše affiliate stranice |
| **Itinerary button** | Dugme koje otvara selektor itinerera |
| **Itinerary form** | Inline forma za izbor datuma i gostiju |
| **Search button** | Dugme koje otvara pretragu smeštaja |
| **Lookup button** | Dugme za pretragu specifičnog inventara |

## Kako ugraditi

Svaki vidžet pruža kod koji možete kopirati i zalepiti u HTML vašeg sajta. Kod uključuje vaš **Client ID** i **Customization ID** tako da je vidžet automatski povezan sa vašim affiliate nalogom.

:::note
Ako koristite WordPress, takođe nudimo WordPress dodatak za lakšu integraciju.
:::
