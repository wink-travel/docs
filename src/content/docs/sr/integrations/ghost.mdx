Ugrađena Ghost funkcija **Code Injection** omogućava vam da dodate skripte u globalni `<head>` i `</body>` vaše publikacije. Svi Ghost planovi podržavaju code injection, što olakšava ugradnju Wink komponenti na bilo koju stranicu ili post.

## Učitajte Wink skripte na celom sajtu

<Steps>
1. U Ghost Admin panelu, idite na **Settings → Code Injection**.
2. U polje **Site Header** dodajte stilsku datoteku:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. U polje **Site Footer** dodajte skriptu i učitavač aplikacije (zamenite `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Kliknite na **Save**.
</Steps>

:::tip
Vaš Client ID je dostupan u vašem Wink nalogu pod [Applications](/sr/portal/settings/applications).
:::

## Dodajte komponentu u post ili stranicu

Ghost editor podržava **HTML karticu** koja vam omogućava da ubacite sirovi HTML bilo gde u vaš sadržaj.

<Steps>
1. Otvorite post ili stranicu u Ghost editoru.
2. Ukucajte `/` da otvorite izbor kartica i izaberite **HTML**.
3. Nalepite HTML komponente, na primer:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Kliknite van kartice da biste videli pregled, zatim objavite.
</Steps>

## Code injection po stranici

Za komponentu koja treba da se pojavi samo na jednoj stranici, Ghost takođe podržava code injection po stranici:

<Steps>
1. Otvorite podešavanja posta ili stranice (ikona zupčanika u editoru).
2. Skrolujte do **Code Injection**.
3. Dodajte tag komponente u polje **Footer** (globalni app-loader treba biti samo u footeru na nivou celog sajta).
4. Sačuvajte.
</Steps>

## Dostupne komponente

| Komponenta | HTML tag | Svrha |
|---|---|---|
| Content loader | `<wink-content-loader>` | Prikaz hotelskih kartica, mreža, mapa |
| Lookup | `<wink-lookup>` | Pretraživač destinacija |
| Search button | `<wink-search-button>` | Otvaranje izbora itinerera |
| Account button | `<wink-account-button>` | Prijava / korisnički meni |
| Itinerary button | `<wink-itinerary-button>` | Prikaz trenutnog itinerera |
| Itinerary picker | `<wink-itinerary-picker>` | Kompletan formular itinerera |
| Shopping cart | `<wink-shopping-cart-button>` | Dugme sa pregledom korpe |

Pogledajte [Web Components](/sr/integrations/web-components) za kompletnu referencu atributa.
