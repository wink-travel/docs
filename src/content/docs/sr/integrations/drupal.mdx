**Wink Drupal modul** dodaje Wink travel inventarske komponente na vaš Drupal sajt kao nativne blokove. Urednici ih mogu postaviti bilo gde u rasporedu preko standardnog sistema blokova.

<LinkCard
  title="wink-travel/wink-drupal-module na GitHub-u"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Izvorni kod, vodič za podešavanje za developere i uputstva za doprinos."
/>

## Zahtevi

- Drupal 10 ili 11
- PHP 8.1+
- Composer

## Instalacija

<Steps>
1. Instalirajte preko Composera:
   ```bash
   composer require drupal/wink
   ```
2. Omogućite modul:
   ```bash
   drush en wink -y
   ```
3. Idite na **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Unesite vaš **Client ID** i **Client Secret**, izaberite okruženje i kliknite **Save**.
   Posetite [Applications](/sr/portal/settings/applications) da preuzmete svoje akreditive.
</Steps>

## Dodavanje bloka na stranicu

<Steps>
1. Idite na **Admin → Structure → Block Layout**.
2. Kliknite **Place block** u regionu gde želite Wink sadržaj.
3. Pretražite **Wink** da pronađete dostupne blokove:
   - **Wink Content** — prikazuje karticu hotela, mrežu ili mapu
   - **Wink Lookup** — traka za pretragu destinacija
   - **Wink Search** — dugme za izbor itinerera
   - **Wink Account** — dugme za prijavu
   - **Wink Itinerary** — dugme za status itinerera
   - **Wink Cart** — dugme korpe za kupovinu
4. Za **Wink Content**, konfigurišite **Layout Type** i **Layout ID** u podešavanjima bloka.
5. Sačuvajte konfiguraciju bloka.
</Steps>

## Dostupni blokovi

| Blok | Namena |
|---|---|
| **Wink Content** | Prikazuje kartice hotela, rangirane mreže, mape i još mnogo toga |
| **Wink Lookup** | Traka za pretragu destinacija i hotela |
| **Wink Search** | Ikona dugmeta koja otvara izbor itinerera |
| **Wink Account** | Dugme za prijavu / korisnički nalog |
| **Wink Itinerary** | Dugme itinerera koje prikazuje trenutni put |
| **Wink Cart** | Dugme sa pregledom korpe za kupovinu |
