**Wink TYPO3 Extension** integriše Wink travel inventar u TYPO3 13 kao nativne sadržajne elemente. Urednici dodaju Wink komponente na stranice koristeći standardni TYPO3 uređivač stranica — nije potrebna promena šablona.

## Zahtevi

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instalacija

<Steps>
1. Instalirajte preko Composera:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivirajte ekstenziju:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. U TYPO3 backend-u, idite na **Admin Tools → Settings → Extension Configuration → wink**.
4. Unesite vaš **Client ID**, **Client Secret** i izaberite okruženje. Sačuvajte.
   Posetite [Applications](/sr/portal/settings/applications) da preuzmete vaše akreditive.
5. Dodajte statički TypoScript šablon u vaš root šablon:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Dodavanje sadržajnog elementa na stranicu

<Steps>
1. Otvorite stranicu u modulu **Page**.
2. Kliknite na dugme **+** u delu za sadržaj da dodate novi element.
3. Izaberite jedan od **Wink Travel** sadržajnih elemenata:
   - **Wink Content** — hotelske kartice, rangirane mreže, mape
   - **Wink Lookup** — traka za pretragu destinacija
   - **Wink Search** — dugme za izbor itinerera
   - **Wink Account** — dugme za prijavu
   - **Wink Itinerary** — dugme za status itinerera
   - **Wink Cart** — dugme korpe za kupovinu
4. Za **Wink Content**, koristite FlexForm polja da izaberete **Layout Type** i unesete **Layout ID**.
5. Sačuvajte i objavite stranicu.
</Steps>

## Dostupni sadržajni elementi

| Element | Namena |
|---|---|
| **Wink Content** | Hotelske kartice, rangirane mreže, mape i još mnogo toga |
| **Wink Lookup** | Traka za pretragu destinacija |
| **Wink Search** | Ikona dugmeta koja otvara izbor itinerera |
| **Wink Account** | Dugme za prijavu / korisnički nalog |
| **Wink Itinerary** | Dugme itinerera koje prikazuje trenutni put |
| **Wink Cart** | Dugme sa pregledom korpe za kupovinu |
