Webflow podržava prilagođeni HTML i JavaScript na tri mesta: u `<head>` delu za ceo sajt, u `</body>` delu za ceo sajt i u kodovima za ugradnju po elementu. Ovo olakšava integraciju Wink web komponenti.

:::note
Prilagođeni kod je dostupan na **Webflow Basic planu i višim**. Ugradnja u `</body>` za ceo sajt zahteva plaćeni hosting plan.
:::

## Učitajte Wink skripte za ceo sajt

<Steps>
1. U Webflow Designer-u otvorite **Project Settings → Custom Code**.
2. U sekciji **Head Code** dodajte stilsku datoteku:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. U sekciji **Footer Code** dodajte skriptu i učitavač aplikacije (zamenite `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Kliknite na **Save Changes**.
</Steps>

:::tip
Vaš Client ID je dostupan u vašem Wink nalogu pod [Applications](/sr/portal/settings/applications).
:::

## Dodajte komponentu na određeni element

<Steps>
1. U Designer-u izaberite **Div** ili **Section** gde želite komponentu.
2. U desnom panelu kliknite na ikonu **`</>` Embed** (ili dodajte **HTML Embed** element iz Add panela).
3. Nalepite HTML komponente, na primer:
   ```html
   <wink-content-loader
     layout="RANKED"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Kliknite na **Save & Close**.
</Steps>

## Prilagođeni kod po stranici

Ako vam je Wink potreban samo na određenim stranicama, koristite ugradnju koda po stranici umesto za ceo sajt:

<Steps>
1. Otvorite **Page Settings** za ciljnu stranicu (ikona zupčanika u Pages panelu).
2. Skrolujte do **Custom Code → Before `</body>` tag**.
3. Nalepite istu skriptu i učitavač aplikacije kao gore.
4. Sačuvajte podešavanja stranice.
</Steps>

## Dostupne komponente

| Komponenta | HTML tag | Namena |
|---|---|---|
| Content loader | `<wink-content-loader>` | Prikaz hotelskih kartica, mreža, mapa |
| Lookup | `<wink-lookup>` | Pretraživač destinacija |
| Search button | `<wink-search-button>` | Otvaranje izbora itinerera |
| Account button | `<wink-account-button>` | Prijava / korisnički meni |
| Itinerary button | `<wink-itinerary-button>` | Prikaz trenutnog itinerera |
| Itinerary picker | `<wink-itinerary-picker>` | Kompletan formular itinerera |
| Shopping cart | `<wink-shopping-cart-button>` | Dugme za pregled korpe |

Pogledajte [Web Components](/sr/integrations/web-components) za kompletnu referencu atributa.
