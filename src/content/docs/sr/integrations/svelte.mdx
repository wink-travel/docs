Svelte prosleđuje atribute direktno DOM-u bez virtuelnog DOM sloja, što čini prilagođene elemente prirodnim izborom. Za SvelteKit (SSR), učitajte CDN unutar `onMount` ili koristite proveru `browser`.

<LinkCard
  title="@wink/elements npm paket"
  href="/integrations/npm"
  description="Instalirajte i učitajte CDN paket sa TypeScript tipovima."
/>

## Instalacija

```bash
npm install @wink/elements
```

## SvelteKit — root layout

```ts
// src/routes/+layout.ts
import { browser } from '$app/environment';
import { load as loadWink } from '@wink/elements';

export function load() {
  if (browser) {
    loadWink({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  }
}
```

Alternativno, učitajte unutar `onMount` u `+layout.svelte`:

```svelte
<!-- src/routes/+layout.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<slot />
```

## Korišćenje u komponentama

Svelte prosleđuje sve nepoznate atribute direktno DOM-u, tako da nije potrebna dodatna konfiguracija:

```svelte
<script>
  export let layoutId = 'YOUR_LAYOUT_ID';
</script>

<main>
  <wink-content-loader layout="HOTEL" id={layoutId} />
  <wink-lookup />
</main>
```

## Svelte 5 (runes sintaksa)

```svelte
<script lang="ts">
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  let { layoutId }: { layoutId: string } = $props();

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id={layoutId} />
```

## Environment promenljive

SvelteKit izlaže samo promenljive koje počinju sa `PUBLIC_` u pregledaču:

```
# .env
PUBLIC_WINK_CLIENT_ID=your-client-id
```

<Aside type="caution" title="SSR renderovanje">
Nikada ne renderujte `<wink-*>` elemente tokom server-side renderovanja — prilagođeni elementi se registruju u pregledačevom `customElements` registru koji ne postoji na serveru. Omotajte u `{#if browser}` ili pozovite `load()` samo unutar `onMount`.
</Aside>

```svelte
<script>
  import { browser } from '$app/environment';
</script>

{#if browser}
  <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
{/if}
```

## Čisti Svelte (Vite, bez SvelteKit)

```svelte
<!-- src/App.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.VITE_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
```
