Qwik-ov model nastavka rada znači da se JavaScript izvršava lenjo — samo kada korisnik interaguje. Koristite `useVisibleTask$` da pokrenete kod koji mora da se izvrši u pregledaču kada komponenta postane vidljiva.

/>

## Instalacija

```bash
npm install @wink/elements
```

## Učitavanje sa useVisibleTask$

`useVisibleTask$` je Qwik-ov životni ciklusni hook za kod koji se izvršava samo u pregledaču, odmah. Postavite ga u vaš root layout:

```tsx
// src/routes/layout.tsx
import { component$, useVisibleTask$ } from '@builder.io/qwik';
import { load } from '@wink/elements';

export default component$(() => {
  useVisibleTask$(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });

  return (
    <main>
      <Slot />
    </main>
  );
});
```

<Aside type="tip" title="Strategija useVisibleTask$">
Po defaultu `useVisibleTask$` se pokreće kada je komponenta vidljiva u viewport-u. Za root layout koji je uvek vidljiv, ovo se pokreće odmah pri učitavanju stranice — što je ekvivalentno `onMount` u drugim framework-ovima.

Prosledite `{ strategy: 'document-ready' }` ako želite da se skripta učita što ranije:
```ts
useVisibleTask$({ strategy: 'document-ready' }, () => {
  load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
});
```
</Aside>

## Korišćenje na stranici

```tsx
// src/routes/hotels/index.tsx
import { component$ } from '@builder.io/qwik';

export default component$(() => {
  return (
    <main>
      <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
      <wink-lookup />
    </main>
  );
});
```

## TypeScript — deklarisanje JSX intrinzičnih elemenata

Qwik koristi sopstveni JSX namespace. Proširite ga u deklaracionom fajlu:

```ts
// src/wink-elements.d.ts
import type {
  WinkContentLoaderAttributes,
  WinkLookupAttributes,
  WinkSearchButtonAttributes,
  WinkAccountButtonAttributes,
  WinkItineraryButtonAttributes,
  WinkShoppingCartButtonAttributes,
  WinkAppLoaderAttributes,
} from '@wink/elements';

declare module '@builder.io/qwik' {
  namespace JSX {
    interface IntrinsicElements {
      'wink-content-loader': WinkContentLoaderAttributes;
      'wink-lookup': WinkLookupAttributes;
      'wink-search-button': WinkSearchButtonAttributes;
      'wink-account-button': WinkAccountButtonAttributes;
      'wink-itinerary-button': WinkItineraryButtonAttributes;
      'wink-shopping-cart-button': WinkShoppingCartButtonAttributes;
      'wink-app-loader': WinkAppLoaderAttributes;
    }
  }
}
```

## Environment promenljive

Qwik (baziran na Vite) izlaže promenljive koje počinju sa `PUBLIC_` u pregledaču:

```
# .env
PUBLIC_WINK_CLIENT_ID=your-client-id
```

<Aside type="caution" title="Server-side rendering">
Qwik renderuje HTML na serveru. `<wink-*>` tagovi se ispisuju kao statički markup. CDN skripta i registracija custom elemenata se dešavaju na klijentskoj strani nakon što se `useVisibleTask$` pokrene. Elementi će kratko biti u nerešenom stanju dok se CDN paket ne učita — ovo je očekivano ponašanje.
</Aside>
