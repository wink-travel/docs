Wink se integriše sa vodećim channel manager-ima, sistemima za upravljanje nekretninama (PMS) i centralnim rezervacionim sistemima (CRS) kako bi vaše cene i dostupnost bili sinhronizovani na svim vašim prodajnim kanalima.

## Kako funkcioniše

Integracije sa channel manager-ima omogućavaju dvosmerni protok podataka između vašeg sistema za upravljanje nekretninama i Wink-a:

1. **Slanje cena** — Vaš channel manager šalje ažuriranja cena Wink-u. Kada ažurirate cene u vašem PMS-u, te promene se automatski prikazuju na Wink-u.
2. **Sinhronizacija dostupnosti** — Vaš channel manager šalje ažuriranja dostupnosti Wink-u. Kada se sobe rezervišu ili oslobode u vašem PMS-u, inventar na Wink-u se ažurira u realnom vremenu.
3. **Obaveštenja o rezervacijama** — Kada se rezervacija napravi na Wink-u, vaš channel manager biva obavešten kako bi rezervacija bila vidljiva u vašem PMS-u.

## Podržani channel manager-i

Wink trenutno integriše sledeće sisteme:

| Channel Manager | Opis |
|---|---|
| **Channex** | Globalna platforma za distribuciju i upravljanje kanalima. |
| **TravelClick** | Hotelska rešenja i upravljanje kanalima (Amadeus). |
| **Omnibees** | Distribucija i upravljanje prihodima za hotele. |
| **D-Edge** | Tehnologija za ugostiteljstvo i centralna rezervaciona rešenja. |
| **Allotz** | Alokacija inventara i upravljanje kanalima. |
| **Ezee / Yanolja** | Cloud PMS i upravljanje kanalima fokusirano na azijsko tržište. |
| **RateGain** | Inteligencija cena i tehnologija distribucije. |
| **RateTiger** | Distribucija cena i dostupnosti u realnom vremenu. |
| **CloudBeds** | Cloud-based PMS, channel manager i booking engine. |
| **SiteMinder** | Vodeći hotelski channel manager i platforma za distribuciju. |
| **Exely** | Sinhronizacija cena i dostupnosti za hotele. |

:::note
Ako vaš channel manager nije na listi, obratite se Wink timu. Nove integracije mogu biti dodate na osnovu OTA (Open Travel Alliance) protokol standarda koje Wink podržava.
:::

## Za developere

Ako pravite integraciju channel manager-a ili PMS-a sa Wink-om, koristite Channel Manager API endpoint-e:

- **Ažuriranja cena** — Šaljite promene cena Wink-u.
- **Ažuriranja dostupnosti** — Šaljite promene dostupnosti Wink-u.
- **Preuzimanje rezervacija** — Povlačite rezervacije iz Wink-a u vaš sistem.

Channel Manager API prati OTA protokol standarde (SOAP/XML) radi kompatibilnosti sa postojećim ugostiteljskim sistemima.

## Webhook obaveštenja

Možete se pretplatiti na webhook događaje channel manager-a da biste primali obaveštenja u realnom vremenu:

- `channel-manager.update.rate` — Primljeno ažuriranje cena.
- `channel-manager.update.availability` — Primljeno ažuriranje dostupnosti.
- `channel-manager.update` — Opšte ažuriranje channel manager-a.

Pogledajte [Katalog Webhook Događaja](/sr/developers/webhook-events#channel-manager-events) za detalje.
