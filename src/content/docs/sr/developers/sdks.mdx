Ubrzajte vreme izlaska na tržište korišćenjem jednog od naših zvanično podržanih jezičkih Software Development Kit-ova.
Ako trenutno ne podržavamo vaš omiljeni jezik, idite na API koji želite da koristite i preuzmite OpenAPI specifikaciju koju nudimo za svaki API na vrhu stranice. Možete generisati sopstveni SDK tako što ćete ili napraviti fork jednog od naših javnih SDK repozitorijuma ili koristiti jedan od OpenAPI generatora.

## Java

Ova biblioteka je kompajlirana za Java 24. Konfigurisana je za korišćenje reaktivnog WebClient-a i optimizovana je za upotrebu sa Spring Framework-om. Međutim, Spring nije obavezan, ali vam pruža OAuth2 podršku odmah iz kutije.
