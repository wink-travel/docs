</LinkButton>

  Сазнајте више

## API за плаћање

Крајње тачке плаћања су намењене програмерима који желе да купе инвентар за путовања. То се може урадити путем API-ја као регистровани туристички агент или коришћењем нашег API-ја у комбинацији са нашим PCI компатибилним виџетом за плаћање за све остале ентитете.
API за плаћање пружа крајње тачке за партнере и хотеле како би пратили резервације, аналитику и доступност средстава. Такође могу да изаберу да повуку расположива средства на свој банковни рачун.
