[Portal](https://app.wink.travel) adalah aplikasi web terpadu tempat setiap bagian dari ekosistem Wink kini berada. Sebelumnya, setiap area fitur (Extranet, Studio, Travel Agent, Payment, Social, Link Manager) memiliki URL sendiri. Mulai rilis ini, semuanya menjadi bagian dari satu Portal di bawah `app.wink.travel`, yang diorganisir sebagai sub-path.

## Bagian

- **`/extranet`** — Fitur properti untuk hotel, jaringan, merek, dan perusahaan manajemen.
- **`/studio`** — Fitur afiliasi untuk mempromosikan inventaris perjalanan dan mendapatkan komisi.
- **`/travel-agent`** — Fitur agen perjalanan untuk agen terdaftar guna mencari, memesan, dan mendapatkan penghasilan.
- **`/payment`** — Fitur pembayaran untuk mengelola pembayaran dan penarikan.
- **`/social`** — Social Otomatis untuk posting media sosial berbasis AI.
- **`/link-manager`** — Link Manager untuk membuat halaman WinkLinks.
- **`/corporate`** — Corporate (segera hadir).
- **`/settings`** — Pengaturan per akun: branding, manajer, aplikasi, webhook, pembayaran, dan lainnya.
- **`/plan`** — Kelola langganan Wink Anda.
- **`/invite`** — Terima undangan untuk mengelola akun lain.
- **`/analytics`** — Dasbor kinerja di seluruh akun Anda.
- **`/leaderboard`** — Lihat peringkat properti Anda secara geografis.
- **`/claims`** — Ubah lead properti menjadi listing lengkap.

## Portal vs. Pengaturan Pengguna

Portal menangani semua yang Anda lakukan sebagai organisasi atau pemegang akun. Manajemen profil pengguna pribadi (email, kata sandi, passkey, persetujuan aplikasi OAuth) berada terpisah di layanan **IAM** di `iam.wink.travel/my-account`. Lihat [Pengaturan Pengguna](/id/account/overview) untuk informasi lebih lanjut.

## Berpindah akun

Anda dapat mengelola beberapa akun — properti, akun afiliasi, akun agen perjalanan, akun korporat — dari satu login. Gunakan pengalih akun di navigasi atas untuk berpindah antar akun. Pengaturan, analitik, dan sebagian besar fitur berlaku per akun.
