Editor postingan memungkinkan Anda membuat postingan manual tanpa batas dan juga merupakan tempat Anda mengelola postingan yang dibuat secara otomatis untuk Anda.

Editor memiliki 4 bagian:

1. Bagian konten.
2. Bagian media.
3. Bagian tautan.
4. Bagian pengaturan.

## Konten

Bagian konten memungkinkan Anda memodifikasi isi teks sebelum dipublikasikan. Anda dapat membuat konten yang dilokalisasi dalam berbagai bahasa — setiap bahasa memiliki judul dan deskripsi sendiri.

## Media

Bagian media memungkinkan Anda memodifikasi gambar dan video yang menyertai teks.

## Tautan

Bagian tautan memungkinkan Anda menautkan ke tempat yang Anda inginkan pengguna kunjungi setelah membaca postingan Anda. *Misalnya, ini bisa berupa tautan ke penawaran khusus atau tempat untuk informasi lebih lanjut.* Anda dapat menambahkan URL tautan, keterangan tautan, dan deskripsi tautan.

## Pengaturan

Bagian pengaturan memungkinkan Anda memodifikasi / menimpa semua pengaturan default per postingan. Anda dapat mengatur:

- **Platform** — Pilih jaringan sosial mana postingan ini akan dipublikasikan.
- **Jenis postingan** — Pilih antara Transaksional, Kesadaran, atau Sentimen.
- **Nada suara** — Timpa nada suara untuk postingan spesifik ini.
- **Kustomisasi** — Gunakan kustomisasi berbeda untuk mata uang tampilan, bahasa, rencana perjalanan, dll.
- **Zona waktu** — Timpa zona waktu untuk publikasi.
- **Tanggal terjadwal** — Postingan yang dihasilkan dijadwalkan secara otomatis pada waktu optimal. Anda dapat menimpa tanggal dan waktu jika menginginkan slot berbeda.

## Siklus status postingan

Setiap postingan mengikuti siklus status:

| Status | Deskripsi |
| -- | -- |
| **Draft** | Postingan telah dibuat tetapi belum siap untuk ditinjau |
| **Pending approval** | Postingan menunggu persetujuan manusia sebelum dijadwalkan |
| **Approved** | Postingan telah disetujui dan akan dijadwalkan |
| **Rejected** | Postingan ditinjau dan ditolak — tidak akan dipublikasikan |
| **Scheduled** | Postingan dalam antrean untuk publikasi pada tanggal dan waktu yang dijadwalkan |
| **Published** | Postingan telah berhasil dipublikasikan ke jaringan sosial |
| **Paused** | Publikasi postingan sementara dihentikan |
| **Failed** | Postingan gagal dipublikasikan karena kesalahan |

## Menyetujui postingan

Postingan disetujui di grid postingan di mana Anda dapat memilih satu atau lebih postingan dan memilih untuk menyetujui atau menolaknya.

:::note
Akan ada satu entri untuk setiap bahasa dan setiap jaringan sosial.

Misalnya, jika Anda memiliki 3 jaringan sosial dan 3 bahasa yang dipilih, Anda akan melihat 9 entri di kalender.
:::
