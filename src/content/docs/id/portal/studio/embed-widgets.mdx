Studio menyediakan beberapa widget yang dapat disematkan yang bisa Anda tambahkan ke situs web Anda untuk meningkatkan pemesanan.

Untuk mengakses widget sematan, navigasikan ke `Tools > Miscellaneous` dari sidebar.

## Widget yang tersedia

| Widget | Deskripsi |
| -- | -- |
| **Account button** | Tombol bermerek yang mengarah ke halaman afiliasi Anda |
| **Itinerary button** | Tombol yang membuka pemilih rencana perjalanan |
| **Itinerary form** | Formulir pemilih tanggal dan tamu secara inline |
| **Search button** | Tombol yang membuka pencarian properti |
| **Lookup button** | Tombol untuk mencari inventaris tertentu |

## Cara menyematkan

Setiap widget menyediakan potongan kode yang dapat Anda salin dan tempel ke HTML situs web Anda. Potongan kode tersebut menyertakan **Client ID** dan **Customization ID** Anda sehingga widget secara otomatis terhubung ke akun afiliasi Anda.

:::note
Jika Anda menggunakan WordPress, kami juga menyediakan plugin WordPress untuk integrasi yang lebih mudah.
:::
