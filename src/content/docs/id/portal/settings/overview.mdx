Setelah Anda membuat akun, berikut hal-hal yang dapat Anda lakukan lebih lanjut saat berada di [my.wink.travel](https://my.wink.travel).

- Perbarui perwakilan akun
- Perbarui informasi akun
- Perbarui ID sistem unik *cara menemukan akun Anda di seluruh sistem*
- Perbarui alamat
- Perbarui foto profil *[ditampilkan di WinkLinks dan saat terhubung dengan properti dan afiliasi di Wink]*
- Perbarui logo *[ditampilkan di e-mail]*

Anda juga dapat...

- Mengelola [kehadiran online](/id/portal/settings/online-presence).
- Menunjuk [perwakilan agen khusus](/id/portal/settings/agency-representation) untuk mengelola akun Anda [dengan biaya].
- Menunjuk [manajer](/id/portal/settings/managers) yang Anda inginkan untuk memiliki akses ke akun Anda.
- Mengelola [kustomisasi](/id/portal/settings/branding) untuk membantu Anda mengontrol pengalaman pengguna saat melihat inventaris Anda.
- Mengelola [aplikasi](/id/portal/settings/applications) untuk memberi Anda akses ke Wink sebagai pengembang dan menambahkan / memperluas fitur Anda sendiri ke Wink.
- Mengelola [webhook](/id/portal/settings/webhooks) untuk mendapatkan pemberitahuan tentang kejadian platform Wink secara real-time. *Misalnya Dapatkan pemberitahuan tentang pemesanan baru*.
