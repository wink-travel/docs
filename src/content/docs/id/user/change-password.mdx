Jika Anda ingin mengganti kata sandi untuk akun pengguna Wink Anda, ikuti langkah-langkah berikut:

## Perbarui kata sandi

<Steps>
1. Anda harus masuk ke salah satu situs web kami.
2. Klik ikon profil Anda, di pojok kanan atas dan menu akan muncul.
3. Klik tautan `Ganti kata sandi`.
4. Ikuti petunjuk untuk mengganti kata sandi Anda.
5. Klik `Simpan`.
</Steps>

Saat Anda masuk berikutnya, Anda akan memerlukan kata sandi baru Anda.

## Lupa kata sandi

Terkadang perlu untuk mengatur ulang kata sandi pengguna.

Berikut adalah langkah-langkah untuk mengatur ulang kata sandi pengguna Anda:

<Steps>
1. Klik tautan `Lupa kata sandi` dari halaman masuk.
2. Ketik nama pengguna / e-mail untuk akun yang ingin Anda atur ulang.
3. Klik tombol `Lupa kata sandi`. *Anda akan diarahkan ke formulir pengaturan ulang kata sandi.*
4. Periksa e-mail Anda.
</Steps>

<Steps>
1. Tempelkan kode yang Anda terima di e-mail ke formulir baru bersama dengan kata sandi baru Anda.
2. Konfirmasi kata sandi baru Anda.
3. Klik tombol `Atur ulang kata sandi`. *Anda akan diarahkan kembali ke halaman masuk.*
</Steps>
