Jika Anda ingin mengubah kata sandi untuk akun pengguna Wink Anda, ikuti langkah-langkah berikut:

## Perbarui kata sandi

<Steps>
  1. Anda harus masuk ke salah satu situs web kami.
  2. Klik ikon profil Anda di sudut kanan atas, dan sebuah menu akan muncul.
  3. Klik tombol `Change password` link.
  4. Ikuti petunjuk untuk mengubah kata sandi Anda.
  5. Klik `Save`.
</Steps>


Saat Anda masuk berikutnya, Anda akan memerlukan kata sandi baru.

## Lupa kata sandi

Kadang-kadang perlu untuk mengatur ulang kata sandi pengguna.

Berikut langkah-langkah untuk mengatur ulang kata sandi pengguna Anda:

<Steps>
  1. Klik pada `Forgot password` tautan dari halaman masuk.
  2. Ketikkan nama pengguna/e-mail untuk akun yang ingin Anda atur ulang.
  3. Klik `Forgot password` tombol. *Anda diarahkan ke formulir pengaturan ulang kata sandi.*
  4. Periksa email Anda.
</Steps>

<Steps>
  1. Tempelkan kode yang Anda terima melalui e-mail ke formulir baru bersama dengan kata sandi baru Anda.
  2. Konfirmasikan kata sandi baru Anda.
  3. Klik `Reset password` tombol. *Anda diarahkan kembali ke halaman masuk.*
</Steps>


<figure>
  <CloudinaryImage width="350" height="467" resize="scale" quality="best" alt="Setel ulang kata sandi" src="academy/reset-password" errorImage="noimage_opaque_nyrtl0.png" />

Kata sandi Anda berhasil direset. Anda dapat masuk ke Wink dengan kredensial baru Anda.
