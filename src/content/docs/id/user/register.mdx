Anda dapat membuat akun pengguna melalui portal mana pun. Setelah dibuat, akun ini dapat digunakan di semua aplikasi dan situs web Wink.

## Pelancong

Ikuti langkah-langkah berikut:

<Steps>
  1. Kunjungi [https://ota.wink.travel](https://ota.wink.travel)
  2. Klik di sini `Sign-In / Register` di pojok kanan atas
  3. Anda akan diminta untuk memasukkan rincian dasar pengguna. Isi semua kolom yang diperlukan dan klik `Save`.
  4. Anda sekarang dapat masuk ke Wink dengan nama pengguna (e-mail) dan kata sandi Anda.
  5. Setelah masuk, sistem akan mengarahkan Anda kembali ke tempat asal Anda.
</Steps>

## Pendaftaran

Membuat akun pengguna Wink memberi Anda akses ke semua fitur pada platform. Anda dapat membuat properti dan akun afiliasi tanpa batas.

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="Masuk" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Formulir masuk*</figcaption>
</figure>


<Steps>
  1. Saat pertama kali Anda diarahkan ke server otorisasi kami, Anda akan diminta memasukkan nama pengguna dan kata sandi.
  2. Sebagai gantinya, klik `Create account` tombol.
  3. Isi semua kolom yang diperlukan dan klik `Register` untuk melanjutkan. Ini akan membawa Anda kembali ke formulir masuk.
  4. Masukkan e-mail / nama pengguna dan kata sandi Anda untuk masuk.
  5. Anda diarahkan kembali ke situs asal Anda.
</Steps>
