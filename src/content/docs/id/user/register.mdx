Anda dapat membuat akun pengguna melalui salah satu portal kami. Setelah dibuat, akun ini dapat digunakan di semua aplikasi dan situs web Wink.

## Wisatawan

Ikuti langkah-langkah berikut:

<Steps>
1. Kunjungi [https://ota.wink.travel](https://ota.wink.travel)
2. Klik `Sign-In / Register` di pojok kanan atas
3. Anda akan diminta untuk memasukkan detail pengguna dasar. Isi semua kolom yang diperlukan dan klik `Save`.
4. Sekarang Anda dapat masuk ke Wink dengan nama pengguna (e-mail) dan kata sandi Anda.
5. Setelah masuk, sistem akan mengarahkan Anda kembali ke tempat asal Anda.
</Steps>

## Pendaftaran

Membuat akun pengguna Wink memberi Anda akses ke semua fitur di platform. Anda dapat membuat properti dan akun afiliasi tanpa batas.

<Steps>
1. Pertama kali Anda diarahkan ke server otorisasi kami, Anda akan diminta memasukkan nama pengguna dan kata sandi.
2. Sebagai gantinya, klik tombol `Create account`.
3. Isi semua kolom yang diperlukan dan klik `Register` untuk melanjutkan. Ini akan membawa Anda kembali ke form masuk.
4. Masukkan e-mail / nama pengguna dan kata sandi untuk masuk.
5. Anda akan diarahkan kembali ke situs asal Anda.
</Steps>
