Jika Anda ingin memperbarui profil pengguna untuk akun pengguna Wink Anda, ikuti langkah-langkah berikut:

<Steps>
  1. Anda harus masuk ke salah satu situs web kami.
  2. Klik ikon profil Anda di sudut kanan atas, dan sebuah menu akan muncul.
  3. Klik tombol `Profile` link.
  4. Ikuti petunjuk untuk memperbarui `e-mail`Bahasa Indonesia: `first` Dan `last name`.
  5. Anda juga dapat mengunggah yang baru `profile picture` dari dirimu sendiri.
  6. Klik `Save`.
</Steps>


:::catatan
Foto profil baru Anda akan ditampilkan saat Anda keluar sepenuhnya dari Wink.
:::
