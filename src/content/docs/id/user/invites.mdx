Pengguna Wink dapat memilih untuk mengizinkan Anda mengelola salah satu akun properti, afiliasi, atau agen perjalanan mereka. Saat itu terjadi, Anda akan menerima email berisi permintaan undangan.

import { Steps } from '@astrojs/starlight/components';

Untuk melihat undangan Wink Anda saat ini, ikuti langkah-langkah berikut:

<Steps>
  1. Anda harus masuk ke salah satu situs web kami.
  2. Klik ikon profil Anda di sudut kanan atas, dan sebuah menu akan muncul.
  3. Klik tombol `Invites` link.
  4. Berinteraksi dengan undangan Anda dengan `Accepting` atau `Rejecting` mereka.
</Steps>
