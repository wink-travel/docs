Seorang pengguna Wink dapat memilih untuk membiarkan Anda mengelola salah satu akun properti, afiliasi, atau agen perjalanan mereka. Ketika itu terjadi, Anda akan menerima email dengan permintaan undangan.

import { Steps } from '@astrojs/starlight/components';

Untuk melihat undangan Wink Anda saat ini, ikuti langkah-langkah berikut:

<Steps>
1. Anda harus masuk ke salah satu situs web kami.
2. Klik ikon profil Anda, di sudut kanan atas dan menu akan muncul.
3. Klik tautan `Invites`.
4. Berinteraksilah dengan undangan Anda dengan cara `Menerima` atau `Menolak` mereka.
</Steps>
