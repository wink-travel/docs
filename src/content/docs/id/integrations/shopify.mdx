**Wink Shopify App** memungkinkan operator tur, penjual pengalaman, dan pedagang perhotelan menampilkan inventaris Wink secara langsung di toko Shopify mereka — berdampingan dengan produk, di halaman arahan, atau di mana saja dalam tema.

## Instalasi

<Steps>
1. Buka [Shopify App Store](https://apps.shopify.com) dan cari **Wink Travel**.
2. Klik **Add app** dan ikuti alur OAuth untuk mengotorisasi.
3. Di pengaturan aplikasi Wink, masukkan **Client ID** dan **Client Secret** Anda.
   Buka [Applications](/id/portal/settings/applications) untuk mengambil kredensial Anda.
4. Klik **Verify & Save**. Pesan sukses akan mengonfirmasi koneksi.
</Steps>

## Aktifkan App Embed Block

App Embed Block secara otomatis menyisipkan skrip dan stylesheet Wink ke setiap halaman — Anda tidak perlu mengedit kode tema.

<Steps>
1. Di Shopify Admin, buka **Online Store → Themes → Customize**.
2. Klik **App embeds** di panel kiri.
3. Aktifkan **Wink Travel**.
4. Klik **Save**.
</Steps>

:::note
App Embed Block membaca Client ID Anda dari metafield Shopify yang diatur saat instalasi. Client Secret Anda tidak pernah terekspos ke browser.
:::

## Tambahkan komponen ke halaman

<Steps>
1. Di editor tema, buka halaman atau bagian tempat Anda ingin menampilkan konten Wink.
2. Klik **Add block** dan pilih dari blok Wink:
   - **Wink Content** — kartu hotel, grid, atau peta
   - **Wink Lookup** — bilah pencarian destinasi
   - **Wink Search** — tombol pemilih itinerary
   - **Wink Account** — tombol masuk
   - **Wink Itinerary** — tombol status itinerary
   - **Wink Cart** — tombol keranjang belanja
3. Untuk **Wink Content**, pilih **Layout Type** dan masukkan **Layout ID** dari Wink Studio.
4. Klik **Save**.
</Steps>

## Blok tema yang tersedia

| Blok | Tujuan |
|---|---|
| **Wink Content** | Menampilkan kartu hotel, grid berperingkat, peta, dan lainnya |
| **Wink Lookup** | Bilah pencarian untuk destinasi dan hotel |
| **Wink Search** | Tombol ikon yang membuka pemilih itinerary |
| **Wink Account** | Tombol masuk / akun pengguna |
| **Wink Itinerary** | Tombol itinerary yang menampilkan perjalanan saat ini |
| **Wink Cart** | Tombol ringkasan keranjang belanja |
