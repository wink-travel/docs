**Wink Drupal Module** menambahkan komponen inventaris perjalanan Wink ke situs Drupal Anda sebagai blok native. Editor dapat menempatkannya di mana saja dalam tata letak melalui sistem blok standar.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"

/>

## Persyaratan

- Drupal 10 atau 11
- PHP 8.1+
- Composer

## Instalasi

<Steps>
1. Instal melalui Composer:
   ```bash
   composer require drupal/wink
   ```
2. Aktifkan modul:
   ```bash
   drush en wink -y
   ```
3. Buka **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Masukkan **Client ID** dan **Client Secret** Anda, pilih lingkungan Anda, lalu klik **Save**.
   Buka [Applications](/id/portal/settings/applications) untuk mengambil kredensial Anda.
</Steps>

## Menambahkan blok ke halaman

<Steps>
1. Buka **Admin → Structure → Block Layout**.
2. Klik **Place block** di wilayah tempat Anda ingin konten Wink.
3. Cari **Wink** untuk menemukan blok yang tersedia:
   - **Wink Content** — menampilkan kartu hotel, grid, atau peta
   - **Wink Lookup** — bilah pencarian destinasi
   - **Wink Search** — tombol pemilih itinerary
   - **Wink Account** — tombol masuk
   - **Wink Itinerary** — tombol status itinerary
   - **Wink Cart** — tombol keranjang belanja
4. Untuk **Wink Content**, konfigurasikan **Layout Type** dan **Layout ID** di pengaturan blok.
5. Simpan konfigurasi blok.
</Steps>

## Blok yang tersedia

| Blok | Tujuan |
|---|---|
| **Wink Content** | Menampilkan kartu hotel, grid peringkat, peta, dan lainnya |
| **Wink Lookup** | Bilah pencarian untuk destinasi dan hotel |
| **Wink Search** | Tombol ikon yang membuka pemilih itinerary |
| **Wink Account** | Tombol masuk / akun pengguna |
| **Wink Itinerary** | Tombol itinerary yang menampilkan perjalanan saat ini |
| **Wink Cart** | Tombol ringkasan keranjang belanja |
