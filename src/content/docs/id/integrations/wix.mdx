The **Wink Wix App** menambahkan komponen web Wink ke situs Wix Anda sebagai widget native yang dapat Anda seret dan lepas di Wix Editor. Tidak perlu coding untuk pemilik situs.

/>

## Instal

<Steps>
1. Buka [Wix App Market](https://www.wix.com/app-market) dan cari **Wink Travel**.
2. Klik **Add to Site** dan ikuti petunjuk untuk mengotorisasi aplikasi.
3. Setelah terpasang, buka **Dashboard → Wink Settings** dan masukkan **Client ID** dan **Client Secret** Anda.
   Buka [Applications](/id/portal/settings/applications) untuk mengambil kredensial Anda.
4. Klik **Save** lalu **Test Connection** untuk memastikan Wink terhubung.
</Steps>

## Tambahkan komponen ke halaman

<Steps>
1. Buka **Wix Editor** dan pergi ke halaman tempat Anda ingin menambahkan konten Wink.
2. Klik **Add (+) → Apps → Wink Travel** untuk melihat semua widget yang tersedia.
3. Seret widget ke halaman — misalnya, **Wink Content** untuk menampilkan grid hotel.
4. Di panel pengaturan widget, pilih **Layout Type** dan tempel **Layout ID** dari Wink Studio.
5. Klik **Publish** untuk menerapkan perubahan secara langsung.
</Steps>

## Widget yang tersedia

| Widget | Tujuan |
|---|---|
| **Wink Content** | Menampilkan kartu hotel, grid peringkat, peta, dan lainnya |
| **Wink Lookup** | Bar pencarian untuk destinasi dan hotel |
| **Wink Search** | Tombol ikon yang membuka pemilih itinerary |
| **Wink Account** | Tombol masuk / akun pengguna |
| **Wink Itinerary** | Tombol itinerary yang menampilkan perjalanan saat ini |
| **Wink Cart** | Tombol ringkasan keranjang belanja |
