Nuxt 3 merender halaman di server secara default. Skrip CDN dan elemen kustom harus dimuat hanya di sisi klien. Plugin klien Nuxt adalah tempat paling bersih untuk melakukan ini.

<LinkCard
  title="@wink/elements npm package"
  href="/integrations/npm"
  description="Pasang dan muat bundel CDN dengan tipe TypeScript."
/>

## Pasang

```bash
npm install @wink/elements
```

## Buat plugin klien

Suffix `.client.ts` memberi tahu Nuxt untuk menjalankan plugin ini hanya di browser:

```ts
// plugins/wink.client.ts
import { load } from '@wink/elements';

export default defineNuxtPlugin(() => {
  load({ clientId: useRuntimeConfig().public.winkClientId });
});
```

## Konfigurasikan variabel runtime

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  runtimeConfig: {
    public: {
      winkClientId: process.env.NUXT_PUBLIC_WINK_CLIENT_ID,
    },
  },
});
```

Tambahkan ke `.env`:

```
NUXT_PUBLIC_WINK_CLIENT_ID=your-client-id
```

## Konfigurasikan Vue untuk elemen kustom

Tambahkan opsi compiler `isCustomElement` agar Vue tidak memperingatkan tentang tag `<wink-*>` yang tidak dikenal:

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  vue: {
    compilerOptions: {
      isCustomElement: (tag) => tag.startsWith('wink-'),
    },
  },
  runtimeConfig: {
    public: {
      winkClientId: process.env.NUXT_PUBLIC_WINK_CLIENT_ID,
    },
  },
});
```

## Gunakan di template

Bungkus komponen Wink dengan `<ClientOnly>` untuk mencegah ketidaksesuaian hidrasi SSR:

```vue
<template>
  <main>
    <ClientOnly>
      <wink-content-loader layout="HOTEL" :id="layoutId" />
    </ClientOnly>
  </main>
</template>

<script setup lang="ts">
const layoutId = 'YOUR_LAYOUT_ID';
</script>
```

<Aside type="tip">
`<ClientOnly>` merender placeholder di server dan memasang elemen asli di klien — ini adalah pola yang direkomendasikan untuk semua embed pihak ketiga yang hanya untuk browser di Nuxt.
</Aside>

## Tipe TypeScript

Perluas `HTMLElementTagNameMap` global (sama seperti Vue 3) untuk mendapatkan autocompletion atribut. Lihat [halaman integrasi Vue 3](/id/integrations/vue#typescript--augment-global-element-types) untuk file deklarasi lengkap.
