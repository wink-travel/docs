Jika Anda adalah perusahaan perjalanan dan ingin mengalihdayakan bagian pembayaran dan pencairan bisnis Anda ke TripPay, berikut adalah 6 langkah dasar untuk memulai:

<Steps>
  1. Membuat sebuah [akun pengguna](/user/register) pada Wink.
  2. Buat akun dengan [Pembayaran TripPay](https://pay.wink.travel).
  3. Membuat sebuah [Aplikasi](/developers/applications) di TripPay yang memungkinkan Anda berbicara dengan API kami.
  4. Buat dan petakan penerima manfaat Anda dengan TripPay. *Misalnya, bagi sebagian besar orang, itu berarti membuat akun untuk properti di platform Anda yang ingin Anda bayarkan melalui TripPay.*
  5. Integrasikan [komponen web pembayaran](/developers/web-components/#payment) ke situs Anda.
  6. Kirim TripPay ke pelancong `shopping cart` dan memulai proses pembayaran.
</Steps>


:::tip
Untuk panduan lengkap tentang penggunaan TripPay untuk situs perjalanan Anda, baca panduannya[Integrasi dengan TripPay](/guides/integrating-with-trip-pay).
:::

## API

Pengembang yang ingin mengelola`Payments`, pergi ke[Pengembang > API > Pembayaran](/developers/apis/#payment-api).
