Jika Anda adalah perusahaan perjalanan dan ingin mengalihdayakan bagian pembayaran dan pencairan bisnis Anda ke TripPay, berikut 6 langkah dasar untuk memulai:

<Steps>
1. Buat [akun pengguna](/id/user/register) di Wink.
2. Buat akun dengan [TripPay](https://pay.wink.travel).
3. Buat [Aplikasi](/id/developers/applications) di TripPay yang memungkinkan Anda berkomunikasi dengan API kami.
4. Buat dan petakan penerima manfaat Anda dengan TripPay. *misalnya Untuk kebanyakan, itu berarti membuat akun untuk properti di platform Anda yang ingin Anda bayar melalui TripPay.*
5. Integrasikan [komponen web pembayaran](/id/developers/web-components/#payment) ke dalam situs Anda.
6. Kirim `keranjang belanja` pelancong ke TripPay dan mulai proses pembayaran.
</Steps>

:::tip
Untuk panduan lengkap menggunakan TripPay untuk situs perjalanan Anda, baca panduan [Integrasi dengan TripPay](/id/guides/integrating-with-trip-pay).
:::

## API

Pengembang yang ingin mengelola `Pembayaran`, kunjungi [Developers > API > Payments](/id/developers/apis/#payment-api).
