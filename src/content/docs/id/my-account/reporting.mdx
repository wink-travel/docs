Wink menyediakan sistem pelaporan yang memungkinkan Anda menghasilkan dan mengekspor data di seluruh akun Anda. Laporan tersedia dalam format CSV untuk memudahkan impor ke spreadsheet, alat analitik, atau sistem Anda sendiri.

## Jenis laporan

Anda dapat menghasilkan laporan untuk kategori data berikut:

| Jenis laporan | Apa yang termasuk |
|---|---|
| **Properti** | Detail properti, status listing, ringkasan inventaris. |
| **Pengguna** | Akun pengguna, peran, ringkasan aktivitas. |
| **Pemesanan** | Riwayat pemesanan, status, pendapatan, detail tamu. |
| **Akun** | Informasi akun, saldo, riwayat transaksi. |

## Cara menghasilkan laporan

1. Navigasikan ke bagian pelaporan di dasbor akun Anda.
2. Pilih **jenis laporan** yang ingin Anda hasilkan.
3. Konfigurasikan **filter** untuk mempersempit data (misalnya, rentang tanggal, status, properti).
4. Pilih **kolom** yang ingin Anda sertakan dalam ekspor. Setiap jenis laporan menawarkan kumpulan kolom yang tersedia untuk dipilih.
5. Klik **Ekspor** untuk mengunduh laporan sebagai file CSV.

## Kolom kustom

Laporan mendukung pemilihan kolom yang dapat disesuaikan. Anda memilih kolom mana yang akan disertakan dalam ekspor, sehingga Anda mendapatkan data yang tepat — tidak lebih, tidak kurang.

## Kasus penggunaan

- **Rekonsiliasi pendapatan** — Ekspor laporan pemesanan dengan detail keuangan untuk direkonsiliasi dengan sistem akuntansi Anda.
- **Analisis kinerja** — Ekspor laporan properti untuk membandingkan kinerja listing di seluruh portofolio Anda.
- **Wawasan tamu** — Ekspor data pemesanan untuk memahami demografi pelancong dan pola pemesanan.
- **Audit akun** — Ekspor data akun untuk kepatuhan atau tinjauan internal.
