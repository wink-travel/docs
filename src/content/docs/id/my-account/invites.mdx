Seorang pengguna Wink dapat memilih untuk membiarkan Anda mengelola salah satu akun properti, afiliasi, atau agen perjalanan mereka. Ketika itu terjadi, Anda akan menerima email dengan permintaan undangan.

import { Steps } from '@astrojs/starlight/components';

Untuk melihat undangan Wink Anda saat ini, ikuti langkah-langkah berikut:

<Steps>
 1. Masuk ke [my.wink.travel](https://my.wink.travel).
 2. Klik pada `Invites` dari sidebar kiri.
 3. Anda akan melihat daftar semua undangan saat ini.
 4. Bagian ini akan menampilkan semua aplikasi Anda.
 5. Pilih untuk `Accept` atau `Reject` undangan tersebut.
 6. Jika Anda menerima undangan, akun akan muncul di bagian `Accounts`.
</Steps>
