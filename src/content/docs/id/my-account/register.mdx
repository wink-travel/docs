Ikuti langkah-langkah ini:

<Steps>
1. Buka [my.wink.travel](https://my.wink.travel)
2. Klik `Masuk / Daftar` di pojok kanan atas
3. Anda akan dibawa ke formulir login. Klik tautan `Daftar` di bawah kolom formulir.
4. Lengkapi formulir pendaftaran dan klik `Daftar`. Anda akan diarahkan ke halaman masuk.
5. Setelah masuk, Anda akan diarahkan kembali ke Akun Saya di mana Anda dapat membuat akun pertama Anda.
</Steps>

Membuat akun pengguna Wink memberi Anda akses ke semua fitur di platform. Anda dapat membuat properti dan akun afiliasi tanpa batas.
