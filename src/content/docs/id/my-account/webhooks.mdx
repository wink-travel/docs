Webhooks adalah cara bagi Anda untuk menerima, dan merespons, event realtime dari Wink untuk akun Anda.

<Steps>
 1. Pilih akun yang ingin Anda gunakan dari [Accounts](https://my.wink.travel).
 2. Klik `Actions` di bagian bawah kartu akun dan pilih `Manage account`.
 3. Klik tab `Webhooks` pada halaman berikutnya.
 4. Bagian ini akan menampilkan semua webhook Anda.
 5. Pilih untuk mengedit yang sudah ada atau membuat yang baru.
 </Steps>

## Contoh

<Steps>
1. Terjadi pemesanan di Wink.
1. Event Wink `booking.created` dikirim.
2. Sistem Anda secara aktif mendengarkan event ini.
3. Sistem Anda melacak analitik sendiri dan menggunakan event ini untuk memperbarui database time-series-nya.
</Steps>

## Membuat webhook

:::caution
Anda perlu membuat akun afiliasi di Wink atau akun di TripPay sebelum membuat webhook karena Anda akan mengaitkan webhook dengan akun tertentu.
:::

Berikut langkah-langkah membuat webhook:

<Steps>
1. Dari halaman webhooks, klik tombol `Create new webhook`.
2. ***Name*** Beri nama webhook Anda. *misal Wink Stalker*
3. ***Enable*** Biarkan seperti semula agar aktif.
4. **Entity name** Kaitkan webhook Anda dengan salah satu akun yang sudah ada. *misal Cool Account*
5. **Events** Dengarkan event yang Anda minati. *misal `booking.created`*
6. **Webhook URL** Masukkan URL tempat Anda akan mendengarkan event dari kami. *misal `https://my.system.com/webhook/wink`*
7. Klik tombol `Save` untuk melanjutkan.
</Steps>

Anda akan diarahkan kembali ke daftar webhook dan webhook baru Anda siap digunakan.

## Memperbarui webhook

Anda perlu memperbarui aplikasi jika ingin:

- Menonaktifkan webhook Anda.
- Mendengarkan lebih banyak/lebih sedikit event.
- Mengubah URL webhook yang didengarkan.

Untuk memperbarui webhook yang sudah ada, lakukan hal berikut:

<Steps>
1. Klik tautan `Actions` untuk webhook yang ingin diperbarui.
2. Klik tombol `Update` di bawah Actions.
3. Lakukan perubahan pada webhook.
4. Klik tombol `Save` untuk melanjutkan.
</Steps>

Anda akan diarahkan kembali ke daftar webhook Anda.

## Menghapus webhook

Jika Anda tidak lagi membutuhkan webhook, Anda dapat menghapusnya.

Untuk menghapus webhook, lakukan hal berikut:

<Steps>
1. Klik tautan `Actions` untuk webhook yang ingin dihapus.
2. Klik tombol `Remove` di bawah Actions.
3. Klik tombol `OK` untuk mengonfirmasi penghapusan.
</Steps>

Anda akan diarahkan kembali ke daftar webhook Anda.
