Jika Anda ingin memperbarui profil pengguna untuk akun pengguna Wink Anda, ikuti langkah-langkah berikut:

<Steps>
1. Buka [my.wink.travel](https://my.wink.travel).
2. Masuk dengan mengklik `Sign-in / Register` di pojok kanan atas.
3. Setelah Anda masuk, klik ikon profil Anda di pojok kanan atas, dan menu akan muncul.
4. Klik tautan `Profile`.
5. Ikuti petunjuk untuk memperbarui `e-mail`, `nama depan` dan `nama belakang`.
6. Anda juga dapat mengunggah `foto profil` baru.
7. Klik `Save`.
</Steps>

<Steps>
1. Anda harus masuk ke salah satu situs web kami.
2. Klik ikon profil Anda di pojok kanan atas dan menu akan muncul.
3. Klik tautan `Profile`.
4. Ikuti petunjuk untuk memperbarui `e-mail`, `nama depan` dan `nama belakang`.
5. Anda juga dapat mengunggah `foto profil` baru.
6. Klik `Update`.
</Steps>

:::note
Foto profil baru Anda akan ditampilkan saat Anda keluar sepenuhnya dari Wink.
:::
