Konten Anda tersedia di bawah tab `Entries`.

Kami mendukung tiga jenis konten utama:

- URL yang tersedia untuk umum yang mengarah ke halaman web.
- File yang diunggah melalui WinkLinks.
- Konten teks yang dibuat melalui fitur `Quotes` kami.

## Tambah konten

Berikut cara menambahkan konten ke WinkLinks. Di bawah tab `Entries`, mulai dengan mengklik tombol `Add entry`.

### Tautan biasa

:::tip
Tautan harus mengarah ke halaman web. Misalnya, Anda tidak dapat menautkan langsung ke PDF. Jika Anda ingin membagikan PDF, Anda perlu menggunakan fitur `Upload file`.
:::

Berikut langkah-langkah menambahkan tautan biasa:

<Steps>
1. Masukkan tautan yang ingin Anda bagikan. *misal https://trvl.as/3xWCH*
2. Setelah memasukkan tautan, Wink akan mencari detail situs web ini dan mengisi formulir yang Anda lihat berikutnya.
3. Anda dapat memilih untuk mengubah beberapa data, seperti nama situs, judul, dan deskripsi.
4. Anda juga dapat memilih untuk mengunggah gambar khusus.
5. Klik tombol `Save` saat Anda siap melanjutkan.
</Steps>

### Unggah file

:::caution[Persyaratan]
- Gambar tidak boleh melebihi ukuran 10Mb.
- Video tidak boleh melebihi ukuran 50Mb.
- Kami mendukung semua format gambar dan video yang diakui, termasuk PDF.
:::

:::note[Rekomendasi]
- 2560w x 1440h (rasio aspek 16x9) untuk gambar unggulan Anda.
- 2560w x 1920h (rasio aspek 4:3) untuk semua gambar lainnya.
- Ukuran gambar tidak boleh lebih besar dari 5Mb. Harap kompres gambar Anda sebelum mengunggah ke Wink.
:::

Berikut langkah-langkah menambahkan file:

<Steps>
1. Media dapat ditambahkan:
    - Dengan menyeret gambar dari komputer Anda ke jendela atau klik `Browse`.
    - Dengan menempelkan URL eksternal yang mengarah ke gambar.
    - Dengan menggunakan kamera laptop Anda.
    - Dari akun Google Drive Anda.
    - Dari akun Dropbox Anda.
    - Dari Shutterstock
    - Dari gettyimages
    - Dari iStock
    - Dari Unsplash
2. Setelah file selesai diunggah, Anda dapat menambahkan metadata untuk mendeskripsikan file Anda seperti judul dan deskripsi.
3. Klik tombol `Save` saat Anda siap melanjutkan.
</Steps>

### Menulis kutipan

:::tip
Kutipan adalah cara yang bagus untuk menambahkan pembatas agar daftar panjang terpecah [...dan juga membuat Anda terdengar pintar dan berpengetahuan ğŸ˜‰].
:::

Berikut langkah-langkah menulis kutipan:

<Steps>
1. Ketik kutipan *misal Roses are red. Violets are blue...*.
2. Masukkan nama penulis asli. *misal Edmund Spenser*
3. Masukkan nama situs tempat Anda mendapatkan kutipan. *misal Daily Quotes*
4. Klik tombol `Save` saat Anda siap melanjutkan.
</Steps>

## Perbarui konten

Semua konten dapat dengan mudah diubah dengan mengklik tautan `Actions` dan memilih `Update`. Anda dapat menghapus konten dengan cara serupa dengan mengklik tombol `Remove`.

## Hapus konten

Semua konten dapat dengan mudah diubah dengan memilih `Update`. Anda dapat menghapus konten dengan cara serupa dengan terlebih dahulu mengklik tautan `Actions` lalu mengklik tombol `Remove`.

## Mengurutkan konten

Anda dapat mengurutkan konten dengan mengklik ikon yang terletak di sudut kiri bawah setiap entri dan menyeretnya ke posisi yang Anda inginkan.

## Mengelompokkan konten

:::tip
Buat tag Anda sebelum membuat entri. Jika tidak, Anda akan kehilangan kemajuan dan harus memulai ulang.
:::

Saat Anda mengedit entri, Anda memiliki opsi untuk menambahkan satu atau lebih tag untuk tujuan pengelompokan. Gulir ke bagian bawah formulir untuk menemukan kolom `Tags`.

Pengguna yang melihat halaman WinkLinks Anda dapat memilih bagaimana mereka ingin mengonsumsi konten Anda (baik daftar atau grid) dan dapat memilih satu atau lebih tag untuk memfilter konten Anda lebih lanjut.

## API

Pengembang yang ingin mengelola `WinkLinks` dapat menuju ke [Developers > API > WinkLinks](/id/developers/apis/#winklinks-api).
