Untuk mengelola tag WinkLinks Anda, navigasikan ke `WinkLinks` di bilah navigasi utama dan klik tab `Tags`.

Seiring pertumbuhan konten Anda, Anda akan ingin mulai mengaturnya. Kami membuat `tags` untuk tujuan ini. Anda dapat mengelompokkan konten menggunakan sebanyak mungkin tag yang Anda inginkan.

## Buat tag

<Steps>
1. Buat tag baru dengan mengklik tombol `Add a new tag`.
2. Masukkan nama tag baru Anda. *misalnya Top coffee shops in Berlin*
3. Klik tautan `Save`.
</Steps>

## Perbarui tag

<Steps>
1. Mulailah dengan mengklik tautan `Edit` di sebelah tag yang ingin Anda ubah.
2. Perbarui nama tag Anda. *misalnya Best coffee shops in Berlin*
3. Klik tautan `Save`.
</Steps>

## Hapus tag

:::note
Saat menghapus tag, Anda juga akan menghapusnya dari semua konten yang terkait.
:::

<Steps>
1. Klik tautan `Remove` di sebelah tag yang ingin Anda hapus.
</Steps>

## API

Pengembang yang ingin mengelola `WinkLinks` dapat menuju ke [Developers > API > WinkLinks](/id/developers/apis/#winklinks-api).
