Untuk mengelola tag WinkLinks Anda, navigasikan ke`WinkLinks`di bilah navigasi utama dan klik`Tags`tab.

Saat konten Anda berkembang, Anda akan ingin mulai mengatur sebagiannya. Kami membuat`tags`untuk tujuan ini. Anda dapat mengelompokkan konten menggunakan tag sebanyak yang Anda inginkan.

## Buat tag

<Steps>
  1. Buat tag baru dengan mengklik `Add a new tag` tombol.
  2. Masukkan nama tag baru Anda. *misalnya Kedai kopi terbaik di Berlin*
  3. Klik tombol `Save` link.
</Steps>


## Perbarui tag

<Steps>
  1. Mulailah dengan mengklik `Edit` tautan di samping tag yang ingin Anda ubah.
  2. Perbarui nama tag Anda. *misalnya Kedai kopi terbaik di Berlin*
  3. Klik tombol `Save` link.
</Steps>


## Hapus tag

:::catatan
Saat menghapus tag, Anda juga akan menghapusnya dari semua konten terkait.
:::

<Steps>
  1. Klik tombol `Remove` tautan di samping tag yang ingin Anda hapus.
</Steps>


## API

Pengembang yang ingin mengelola`WinkLinks`bisa menuju ke[Pengembang > API > WinkLinks](/developers/apis/#winklinks-api).
