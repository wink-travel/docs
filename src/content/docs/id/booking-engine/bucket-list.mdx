Di mana pun Anda melihat [kartu](/id/studio/cards) inventaris Wink, Anda akan melihat ikon hati di sudut kanan atas kartu yang memungkinkan Anda menambahkan inventaris tersebut ke daftar keinginan Anda.

Ikuti langkah-langkah ini untuk mengelola daftar keinginan Anda:

<Steps>
1. Klik pada ikon ♡ `hati` untuk inventaris yang ingin Anda tambahkan.
2. Jika Anda sudah masuk, ikon hati akan berubah menjadi merah.
3. Jika Anda belum masuk, Anda akan diarahkan untuk melakukannya.
</Steps>

:::tip
Klik ikon ♥️ `hati` sekali lagi untuk menghapusnya dari daftar Anda.
:::

Anda dapat mengelola semua item daftar keinginan Anda dengan mengklik ikon profil Anda di sudut kanan atas lalu klik tombol `Daftar keinginan`.
