Di mana pun Anda melihat inventaris Wink[kartu-kartu](/studio/cards), Anda akan melihat hati di sudut kanan atas kartu yang memungkinkan Anda menambahkan inventaris tersebut ke daftar keinginan Anda.

Ikuti langkah-langkah berikut untuk mengelola daftar keinginan Anda:

<Steps>
  1. Klik pada ♡ `heart icon` untuk inventaris yang ingin Anda tambahkan.
  2. Jika Anda sudah masuk, ikon hati akan berubah menjadi merah.
  3. Jika Anda belum masuk, Anda akan diarahkan untuk melakukannya.
</Steps>


:::tips
Klik ♥️`heart icon`kedua kalinya untuk menghapusnya dari daftar Anda.
:::

Anda dapat mengelola semua item daftar keinginan Anda dengan mengklik ikon profil Anda di sudut kanan atas lalu klik`Bucket list` tombol.
