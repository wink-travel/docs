Mesin pemesanan hadir dalam dua varian untuk mendukung kebutuhan berbeda dari afiliasi, perusahaan manajemen properti, dan hotel.

## Agen perjalanan online

Untuk afiliasi, paling masuk akal menggunakan versi agen perjalanan online. Alasannya adalah situs ini disiapkan untuk mengakomodasi pelancong yang ingin melihat beberapa properti.

Anda dapat menemukan halaman utama aplikasi ini di bawah.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Mesin pemesanan internet

Untuk hotel atau perusahaan manajemen properti, paling masuk akal menggunakan versi mesin pemesanan internet. Jenis pengguna ini tertarik menonjolkan satu properti atau beberapa properti yang dikelola.

Anda dapat menemukan halaman utama aplikasi ini di bawah.

## Konfigurasi

Sebagian besar waktu, tidak perlu membagikan salah satu aplikasi ini secara langsung. Kami telah membuat berbagai cara untuk Anda membagikan inventaris perjalanan, yang dapat Anda temukan di [Wink Studio](/id/studio/what-is-studio) atau melalui [WinkLinks](/id/link-manager/wink-links).

Namun, jika Anda ingin menautkan secara manual ke salah satu situs web ini, pastikan Anda menambahkan ID klien dan ID kustomisasi ke URL sebagai parameter.

Anda dapat menemukan ID klien Anda dengan:

<Steps>
1. Masuk ke [Wink Extranet](/id/extranet/what-is-extranet) atau [Wink Studio](/id/studio/what-is-studio).
2. Klik ikon profil Anda di pojok kanan atas dan klik [Applications](/id/developers/applications).
3. Gunakan aplikasi yang terkait dengan akun Wink yang ingin Anda gunakan.
4. Salin Client ID dari aplikasi tersebut dan tempelkan pada URL sebagai parameter permintaan. Contoh: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Anda dapat menemukan ID kustomisasi Anda sebagai properti dengan:

<Steps>
1. Masuk ke [Wink Extranet](/id/extranet/what-is-extranet).
2. Pilih properti yang ingin Anda kelola.
3. Dari bilah navigasi utama, klik `Account > Branding`
4. Salin Customization ID dan tempelkan pada URL sebagai parameter permintaan. Contoh: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Anda dapat menemukan ID kustomisasi Anda sebagai afiliasi dengan:

<Steps>
1. Masuk ke [Wink Studio](/id/studio/what-is-studio).
2. Pilih akun yang ingin Anda kelola.
3. Dari bilah navigasi utama, klik `Tools > Customization`
4. Salin Customization ID dari kustomisasi yang ingin Anda gunakan dan tempelkan pada URL sebagai parameter permintaan. Contoh: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Yang Anda dapatkan adalah URL yang terlihat seperti ini:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Jika Anda ingin langsung ke halaman hotel, URL akan terlihat seperti ini:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Jika Anda tidak menyertakan ID klien dan/atau ID konfigurasi, Wink OTA akan menggunakan konfigurasi default.
:::

## IBE

Jika Anda adalah perusahaan manajemen properti, Anda ingin menautkan ke halaman utama Wink IBE. Halaman utama akan menampilkan tema yang ditentukan bersama dengan [properti yang Anda wakili](/id/studio/agency-representation).

Yang Anda dapatkan adalah URL yang terlihat seperti ini:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Jika Anda adalah properti, Anda ingin menautkan langsung ke halaman landing properti Wink IBE.

URL tersebut akan terlihat seperti ini:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
Jika Anda tidak menyertakan ID klien dan/atau ID konfigurasi, Wink IBE TIDAK akan dimuat. Parameter ini wajib.
:::
