## Konfigurasi

Sebagian besar waktu, tidak perlu membagikan salah satu dari aplikasi ini secara langsung. Kami telah membuat berbagai cara bagi Anda untuk membagikan inventaris perjalanan, yang dapat Anda temukan di[Studio Kedipan](/studio/what-is-studio)atau melalui[Tautan Wink](/link-manager/wink-links).

Namun, jika Anda ingin menautkan secara manual ke salah satu situs web ini, pastikan Anda menambahkan ID klien dan ID penyesuaian ke URL sebagai parameter.

Anda dapat menemukan ID klien Anda dengan:

<Steps>
  1. Masuk ke salah satu [Ekstranet Wink](/extranet/what-is-extranet) atau [Studio Kedipan](/studio/what-is-studio).
  2. Klik ikon profil Anda di pojok kanan atas dan klik [Aplikasi](/developers/applications).
  3. Gunakan aplikasi yang terkait dengan akun Wink yang ingin Anda gunakan.
  4. Salin ID Klien dari aplikasi tersebut dan tempel di URL sebagai parameter permintaan. Contoh: ?client-id=MY\_CLIENT\_ID
</Steps>


### Jaringan ekstranet

Anda dapat menemukan ID kustomisasi Anda sebagai properti dengan:

<Steps>
  1. Masuk ke dalam [Ekstranet Wink](/extranet/what-is-extranet).
  2. Pilih properti yang ingin Anda kerjakan.
  3. Dari bilah navigasi utama, klik `Account > Branding`
  4. Salin ID Kustomisasi dan tempel di URL sebagai parameter permintaan. Contoh: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Studio

Anda dapat menemukan ID kustomisasi Anda sebagai afiliasi dengan:

<Steps>
  1. Masuk ke dalam [Studio Kedipan](/studio/what-is-studio).
  2. Pilih akun yang ingin Anda gunakan.
  3. Dari bilah navigasi utama, klik `Tools > Customization`
  4. Salin ID Kustomisasi dari kustomisasi yang ingin Anda gunakan dan tempel di URL sebagai parameter permintaan. Contoh: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## Tidak ada data

Hasilnya adalah URL yang terlihat seperti ini:

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Jika Anda ingin langsung menuju ke halaman hotel, URL-nya akan terlihat seperti ini:

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::catatan
Jika Anda tidak menyertakan ID klien dan/atau konfigurasi, Wink OTA akan menggunakan konfigurasi default.
:::

## saya menjadi

Jika Anda adalah perusahaan manajemen properti, Anda ingin menautkan ke beranda Wink IBE. Beranda akan menampilkan tema yang Anda tentukan beserta[properti yang Anda wakili](/studio/agency-representation).

Hasilnya adalah URL yang terlihat seperti ini:

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Jika Anda seorang pemilik properti, Anda ingin menautkan langsung ke laman arahan properti Wink IBE.

URL tersebut akan terlihat seperti ini:

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::catatan
Jika Anda tidak menyertakan ID klien dan/atau konfigurasi, Wink IBE TIDAK akan dimuat. Parameter ini wajib diisi.
:::
