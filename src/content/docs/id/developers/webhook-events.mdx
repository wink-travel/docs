Halaman ini mencantumkan setiap event webhook yang dapat Anda langgani di Wink. Untuk instruksi pengaturan, lihat [Panduan Integrasi Webhook](/id/guides/webhook-integration) atau [halaman pengelolaan Webhooks](/id/my-account/webhooks).

## Cara berlangganan

1. Navigasikan ke **Applications > Webhooks** di portal Anda.
2. Buat webhook dan pilih event yang ingin Anda terima.
3. Wink mengirim HTTP POST ke endpoint Anda setiap kali event terjadi.

Untuk langkah pengaturan lengkap dan praktik terbaik, lihat [Integrasi Webhook](/id/guides/webhook-integration).

## Event akun

Event yang terkait dengan siklus hidup dan pengaturan akun TripPay Anda.

| Event | Key | Deskripsi |
|---|---|---|
| Akun dibuat | `account.create` | Akun baru telah dibuat. |
| Akun dalam peninjauan | `account.under_review` | Akun sedang menunggu peninjauan. |
| Akun diperbarui | `account.updated` | Detail akun telah berubah. |
| Status akun diperbarui | `account.status.updated` | Status akun telah berubah (misalnya, disetujui, ditangguhkan). |
| Alamat akun diperbarui | `account.address.updated` | Alamat akun telah diubah. |
| Detail akun diperbarui | `account.details.updated` | Kolom detail akun telah berubah. |
| Informasi akun diperbarui | `account.information.update` | Informasi umum akun telah diperbarui. |
| Manajer akun diperbarui | `account.managers.update` | Daftar manajer akun telah berubah. |
| Foto profil diperbarui | `account.profile_picture.updated` | Foto profil akun telah diubah. |
| Agen diperbarui | `account.agent.updated` | Informasi agen pada akun telah diubah. |
| Nama URL diperbarui | `account.url-name.update` | URL khusus akun telah berubah. |
| ID unik diperbarui | `account.unique-id.update` | Identifier unik akun telah diubah. |
