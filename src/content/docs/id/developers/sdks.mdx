Percepat waktu ke pasar dengan menggunakan salah satu Software Development Kit bahasa yang secara resmi kami dukung.  
Jika saat ini kami belum mendukung bahasa favorit Anda, buka API yang ingin Anda gunakan dan unduh spesifikasi OpenAPI yang kami sediakan untuk setiap API di bagian atas halaman. Anda dapat membuat SDK Anda sendiri dengan cara mem-fork salah satu repositori SDK publik kami atau dengan menggunakan salah satu generator OpenAPI.

## Java

Perpustakaan ini dikompilasi untuk Java 24. Ini dikonfigurasi untuk menggunakan WebClient reaktif dan dioptimalkan untuk digunakan dengan Spring Framework. Namun, Spring bukanlah persyaratan tetapi memberikan dukungan OAuth2 secara langsung.
