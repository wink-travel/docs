Percepat waktu pemasaran dengan menggunakan salah satu Kit Pengembangan Perangkat Lunak bahasa yang didukung secara resmi.
Jika saat ini kami tidak mendukung bahasa favorit Anda, buka API yang ingin Anda gunakan dan unduh spesifikasi OpenAPI yang kami sediakan untuk setiap API di bagian atas halaman. Anda dapat membuat SDK Anda sendiri dengan melakukan forking pada salah satu repositori SDK publik kami atau dengan menggunakan salah satu generator OpenAPI.

## Jawa

Pustaka ini dikompilasi untuk Java 24. Pustaka ini dikonfigurasi untuk menggunakan WebClient reaktif dan dioptimalkan untuk digunakan dengan Spring Framework. Namun, Spring bukanlah persyaratan, tetapi menyediakan dukungan OAuth2 secara langsung.
