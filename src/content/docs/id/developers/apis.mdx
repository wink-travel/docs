## API Pembayaran

Titik akhir pembayaran diperuntukkan bagi pengembang yang ingin membeli inventaris perjalanan. Ini dapat dilakukan melalui API sebagai Agen Perjalanan terdaftar atau menggunakan API kami bersama dengan widget pembayaran yang sesuai dengan PCI untuk semua entitas lainnya. API Pembayaran memaparkan titik akhir bagi afiliasi dan hotel untuk melacak pemesanan, analitik, dan ketersediaan dana. Mereka juga dapat memilih untuk menarik dana yang tersedia ke rekening bank mereka.
