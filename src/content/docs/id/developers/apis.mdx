## Payment API

Endpoint Payment ditujukan untuk pengembang yang ingin membeli inventaris perjalanan. Ini dapat dilakukan melalui API sebagai Travel Agent terdaftar atau menggunakan API kami bersama dengan widget pembayaran yang sesuai PCI untuk entitas lainnya.
Payment API menyediakan endpoint bagi afiliasi dan hotel untuk melacak pemesanan, analitik, dan ketersediaan dana. Mereka juga dapat memilih untuk menarik dana yang tersedia ke rekening bank mereka.
