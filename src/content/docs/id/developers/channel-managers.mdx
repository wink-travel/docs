Wink terintegrasi dengan channel manager terkemuka, sistem manajemen properti (PMS), dan sistem reservasi pusat (CRS) untuk menjaga tarif dan ketersediaan Anda tetap sinkron di semua saluran penjualan Anda.

## Cara kerjanya

Integrasi channel manager memungkinkan aliran data dua arah antara sistem manajemen properti Anda dan Wink:

1. **Dorongan tarif** — Channel manager Anda mengirimkan pembaruan tarif ke Wink. Ketika Anda memperbarui harga di PMS Anda, perubahan tersebut secara otomatis tercermin di Wink.
2. **Sinkronisasi ketersediaan** — Channel manager Anda mengirimkan pembaruan ketersediaan ke Wink. Ketika kamar dipesan atau dibebaskan di PMS Anda, inventaris Wink diperbarui secara real time.
3. **Notifikasi pemesanan** — Ketika pemesanan dilakukan di Wink, channel manager Anda diberi tahu sehingga reservasi muncul di PMS Anda.

## Channel manager yang didukung

Wink saat ini terintegrasi dengan sistem berikut:

| Channel Manager | Deskripsi |
|---|---|
| **Channex** | Platform distribusi global dan manajemen channel. |
| **TravelClick** | Solusi hotel dan manajemen channel (Amadeus). |
| **Omnibees** | Distribusi dan manajemen pendapatan untuk hotel. |
| **D-Edge** | Teknologi perhotelan dan solusi reservasi pusat. |
| **Allotz** | Alokasi inventaris dan manajemen channel. |
| **Ezee / Yanolja** | PMS berbasis cloud dan manajemen channel yang fokus pada pasar Asia. |
| **RateGain** | Teknologi intelijen tarif dan distribusi. |
| **RateTiger** | Distribusi tarif dan ketersediaan secara real-time. |
| **CloudBeds** | PMS berbasis cloud, channel manager, dan mesin pemesanan. |
| **SiteMinder** | Channel manager hotel terkemuka dan platform distribusi. |
| **Exely** | Sinkronisasi tarif dan ketersediaan untuk hotel. |

:::note
Jika channel manager Anda tidak tercantum, hubungi tim Wink. Integrasi baru dapat ditambahkan berdasarkan standar protokol OTA (Open Travel Alliance) yang didukung Wink.
:::

## Untuk pengembang

Jika Anda membangun integrasi channel manager atau PMS dengan Wink, gunakan endpoint Channel Manager API:

- **Pembaruan tarif** — Dorong perubahan tarif ke Wink.
- **Pembaruan ketersediaan** — Dorong perubahan ketersediaan ke Wink.
- **Pengambilan pemesanan** — Tarik pemesanan dari Wink ke sistem Anda.

Channel Manager API mengikuti standar protokol OTA (SOAP/XML) untuk kompatibilitas dengan sistem perhotelan yang ada.

## Notifikasi webhook

Anda dapat berlangganan ke event webhook channel manager untuk menerima notifikasi secara real-time:

- `channel-manager.update.rate` — Pembaruan tarif diterima.
- `channel-manager.update.availability` — Pembaruan ketersediaan diterima.
- `channel-manager.update` — Pembaruan umum channel manager.

Lihat [Katalog Event Webhook](/id/developers/webhook-events#channel-manager-events) untuk detailnya.
