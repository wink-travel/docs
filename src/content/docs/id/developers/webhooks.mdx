Webhooks adalah cara bagi Anda untuk menerima, dan merespons, event realtime dari Wink dan TripPay yang berkaitan dengan akun Anda.

## Contoh

<Steps>
1. Terjadi pemesanan di Wink.
1. Event Wink `booking.created` dikirimkan.
2. Sistem Anda secara aktif mendengarkan event ini.
3. Sistem Anda melacak analitik sendiri dan menggunakan event ini untuk memperbarui database time-series-nya.
</Steps>

:::note
Webhook Wink berbeda dengan webhook TripPay.

- Jika Anda ingin mendengarkan event Wink, Anda dapat membuat webhook dari [Extranet](https://extranet.wink.travel) atau [Studio](https://studio.wink.travel).
- Jika Anda ingin mendengarkan event TripPay, Anda dapat membuat webhook dari [Payment](https://pay.wink.travel).
:::

*Untuk contoh ini, kami akan melakukan autentikasi melalui [Studio](https://studio.wink.travel).*

<Steps>
1. Setelah autentikasi, klik ikon profil Anda di pojok kanan atas, dan menu akan muncul.
2. Klik tautan `Webhooks`.
3. Anda akan diarahkan ke halaman webhooks Anda.
</Steps>

## Membuat webhook

:::caution
Anda perlu membuat akun afiliasi di Wink atau akun di TripPay sebelum membuat webhook karena webhook Anda akan dihubungkan ke akun tertentu.
:::

Berikut langkah-langkah membuat webhook:

<Steps>
1. Dari halaman webhooks, klik tombol `Create new webhook`.
2. ***Name*** Beri nama webhook Anda. *misal Wink Stalker*
3. ***Enable*** Biarkan seperti semula agar aktif.
4. **Entity name** Hubungkan webhook Anda dengan salah satu akun yang sudah ada. *misal Cool Account*
5. **Events** Pilih event yang ingin Anda dengarkan. *misal `booking.created`*
6. **Webhook URL** Masukkan URL tempat Anda akan mendengarkan event dari kami. *misal `https://my.system.com/webhook/wink`*
7. Klik tombol `Save` untuk melanjutkan.
</Steps>

Anda akan diarahkan kembali ke daftar webhook dan webhook baru Anda siap digunakan.

## Memperbarui webhook

Anda perlu memperbarui aplikasi jika ingin:

- Menonaktifkan webhook Anda.
- Mendengarkan lebih banyak/lebih sedikit event.
- Mengubah URL webhook yang didengarkan.

Untuk memperbarui webhook yang sudah ada, lakukan hal berikut:

<Steps>
1. Klik tautan `Actions` pada webhook yang ingin Anda perbarui.
2. Klik tombol `Update` di bawah Actions.
3. Lakukan perubahan pada webhook.
4. Klik tombol `Save` untuk melanjutkan.
</Steps>

Anda akan diarahkan kembali ke daftar webhook Anda.

## Menghapus webhook

Jika Anda tidak lagi membutuhkan webhook, Anda dapat menghapusnya.

Untuk menghapus webhook, lakukan hal berikut:

<Steps>
1. Klik tautan `Actions` pada webhook yang ingin Anda hapus.
2. Klik tombol `Remove` di bawah Actions.
3. Klik tombol `OK` untuk mengonfirmasi penghapusan.
</Steps>

Anda akan diarahkan kembali ke daftar webhook Anda.
