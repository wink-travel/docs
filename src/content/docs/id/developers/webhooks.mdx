Webhook adalah cara bagi Anda untuk menerima, dan bereaksi terhadap, peristiwa waktu nyata dari Wink dan TripPay terkait akun Anda.

## Contoh

<Steps>
  1. Pemesanan terjadi pada Wink.
  2. Acara Wink `booking.created` dikirim.
  3. Sistem Anda secara aktif mendengarkan peristiwa ini.
  4. Sistem Anda melacak rangkaian analitik Anda sendiri dan menggunakan peristiwa ini untuk memperbarui basis data deret waktu.
</Steps>

:::catatan
Webhook Wink berbeda dengan webhook TripPay.

* Jika Anda ingin mendengarkan acara Wink, Anda dapat membuat webhook dari[Jaringan ekstranet](https://extranet.wink.travel) atau [Studio](https://studio.wink.travel).
* Jika Anda ingin mendengarkan acara TripPay, Anda dapat membuat webhook dari[Pembayaran](https://pay.wink.travel).
  :::

*Untuk contoh ini, kami akan melakukan autentikasi melalui[Studio](https://studio.wink.travel).*

<Steps>
  1. Setelah diautentikasi, klik ikon profil Anda di sudut kanan atas, dan menu akan muncul.
  2. Klik tombol `Webhooks` link.
  3. Anda akan diarahkan ke halaman webhook Anda.
</Steps>


## Buat webhook

:::perhatian
Anda perlu membuat akun afiliasi di Wink atau akun di TripPay sebelum membuat webhook karena Anda akan mengikat webhook Anda ke akun tertentu.
:::

Berikut langkah-langkah untuk membuat webhook:

<Steps>
  1. Dari halaman webhook, klik `Create new webhook` tombol.
  2. ***Nama*** Beri nama pada webhook Anda. *misalnya Wink Stalker*
  3. ***Memungkinkan*** Biarkan apa adanya untuk diaktifkan.
  4. **Nama entitas** Hubungkan webhook Anda dengan salah satu akun Anda yang ada. *misalnya Akun Keren*
  5. **Acara** Dengarkan acara yang menarik minat Anda. *misalnya `booking.created`*
  6. **URL kait web** Masukkan URL tempat Anda akan mendengarkan acara dari kami. *misalnya `https://my.system.com/webhook/wink`*
  7. Klik tombol `Save` untuk melanjutkan.
</Steps>


Anda diarahkan kembali ke daftar webhook Anda dan webhook baru Anda siap digunakan.

## Perbarui webhook

Anda perlu memperbarui aplikasi Anda jika Anda ingin:

* Nonaktifkan webhook Anda.
* Dengarkan lebih banyak/lebih sedikit acara.
* Ubah URL webhook pendengaran Anda.

Untuk memperbarui webhook yang ada, lakukan hal berikut:

<Steps>
  1. Klik tombol `Actions` tautan untuk webhook yang ingin Anda perbarui.
  2. Klik tombol `Update` tombol di bawah Tindakan.
  3. Lakukan perubahan pada webhook.
  4. Klik tombol `Save` untuk melanjutkan.
</Steps>


Anda diarahkan kembali ke daftar webhook Anda.

## Hapus webhook

Jika Anda tidak lagi memerlukan webhook, Anda dapat menghapusnya.

Untuk menghapus webhook, lakukan hal berikut:

<Steps>
  1. Klik tombol `Actions` tautan untuk webhook yang ingin Anda perbarui.
  2. Klik tombol `Remove` tombol di bawah Tindakan.
  3. Klik `OK` untuk mengonfirmasi penghapusan.
</Steps>


Anda diarahkan kembali ke daftar webhook Anda.
