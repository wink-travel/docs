Aplikasi memberi Anda akses penuh ke fitur di Wink dan TripPay serta memungkinkan pengguna Anda mengautentikasi dan masuk ke situs Anda menggunakan server otorisasi OAuth2 kami; sama seperti Facebook dan Google.  
Anda dapat membangun OTA Anda sendiri dengan Wink. Semua dimulai dengan sebuah aplikasi.

*Secara default, Anda mendapatkan sebuah aplikasi saat mendaftar akun afiliasi Wink baru atau akun TripPay. Kami menggunakan `client ID` untuk beberapa skenario.*

:::note
Aplikasi yang dibuat di Wink atau TripPay dapat digunakan untuk semua endpoint API.
:::

*Untuk contoh ini, kami akan mengautentikasi melalui [Wink Studio](https://studio.wink.travel).*

<Steps>
1. Setelah terautentikasi, klik ikon profil Anda di pojok kanan atas, dan menu akan muncul.
2. Klik tautan `Applications`.
3. Anda akan diarahkan ke halaman aplikasi Anda.
</Steps>

## Buat aplikasi

Karena Anda tidak dapat mengakses `secret key` dari aplikasi yang kami buat untuk Anda, Anda perlu membuat aplikasi Anda sendiri.

:::caution
Anda harus membuat akun afiliasi di Wink atau akun di TripPay sebelum membuat aplikasi karena aplikasi Anda akan terikat pada akun tertentu.
:::

Berikut langkah-langkah membuat aplikasi:

<Steps>
1. Dari halaman aplikasi, klik tombol `Create new application`.
2. ***Name*** Beri nama aplikasi Anda. *misalnya Cool App*
3. **Entity name** Hubungkan aplikasi Anda dengan salah satu akun yang sudah ada. *misalnya Cool Account*
4. **Redirect URIs** Minimal satu redirect URI diperlukan. Ini digunakan untuk memvalidasi domain redirect setelah autentikasi berhasil. *misalnya https://www.cool-site.com*
5. Klik tombol `Save` untuk melanjutkan.
</Steps>

Anda akan diarahkan kembali ke daftar aplikasi Anda. Anda akan melihat aplikasi baru Anda dalam daftar dan hanya kali ini Anda dapat mengakses secret key aplikasi. Pastikan untuk menyalinnya dan menyimpannya di tempat yang aman.

## Perbarui aplikasi

Anda perlu memperbarui aplikasi jika salah satu redirect URI Anda berubah atau Anda ingin menambah atau menghapus URI.

Untuk memperbarui aplikasi yang sudah ada, lakukan hal berikut:

<Steps>
1. Klik tautan `Actions` untuk aplikasi yang ingin Anda perbarui.
2. Klik tombol `Update` di bawah Actions.
3. Lakukan perubahan pada aplikasi.
4. Klik tombol `Save` untuk melanjutkan.
</Steps>

Anda akan diarahkan kembali ke daftar aplikasi Anda.

## Hapus aplikasi

:::caution
Harap jangan hapus aplikasi yang kami buat otomatis untuk Anda agar semua fitur dapat berfungsi untuk akun Anda.
:::

Jika Anda tidak lagi membutuhkan sebuah aplikasi, Anda dapat menghapusnya.

Untuk menghapus aplikasi, lakukan hal berikut:

<Steps>
1. Klik tautan `Actions` untuk aplikasi yang ingin Anda hapus.
2. Klik tombol `Remove` di bawah Actions.
3. Klik tombol `OK` untuk mengonfirmasi penghapusan.
</Steps>

Anda akan diarahkan kembali ke daftar aplikasi Anda.
