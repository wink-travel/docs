Aplikasi memberi Anda akses penuh ke berbagai fitur di Wink dan TripPay dan memungkinkan pengguna Anda mengautentikasi dan masuk ke situs Anda menggunakan server otorisasi OAuth2 kami; sama seperti Facebook dan Google.
Anda dapat membuat OTA Anda sendiri dengan Wink. Semuanya dimulai dengan sebuah aplikasi.

*Secara default, Anda mendapatkan aplikasi saat Anda mendaftar akun afiliasi Wink atau akun TripPay baru. Kami menggunakan`client ID`untuk beberapa skenario.*

:::catatan
Aplikasi yang dibuat di Wink atau TripPay dapat digunakan untuk semua titik akhir API.
:::

*Untuk contoh ini, kami akan melakukan autentikasi melalui[Studio Kedipan](https://studio.wink.travel).*

<Steps>
  1. Setelah diautentikasi, klik ikon profil Anda di sudut kanan atas, dan menu akan muncul.
  2. Klik tombol `Applications` link.
  3. Anda akan diarahkan ke halaman aplikasi Anda.
</Steps>


## Buat aplikasi

Karena Anda tidak dapat mengakses`secret key`dari aplikasi yang kami buat untuk Anda, Anda perlu membuatnya sendiri.

:::perhatian
Anda perlu membuat akun afiliasi di Wink atau akun di TripPay sebelum membuat aplikasi karena Anda akan mengikat aplikasi Anda ke akun tertentu.
:::

Berikut langkah-langkah untuk membuat aplikasi:

<Steps>
  1. Dari halaman aplikasi, klik `Create new application` tombol.
  2. ***Nama*** Beri nama pada aplikasi Anda. *misalnya Aplikasi Keren*
  3. **Nama entitas** Tautkan aplikasi Anda dengan salah satu akun yang ada. *misalnya Akun Keren*
  4. **Pengalihan URI** Diperlukan setidaknya satu URI pengalihan. URI ini digunakan untuk memvalidasi domain pengalihan setelah autentikasi berhasil. *misalnya https://www.cool-site.com*
  5. Klik tombol `Save` untuk melanjutkan.
</Steps>


Anda akan diarahkan kembali ke daftar aplikasi Anda. Anda akan melihat aplikasi baru Anda dalam daftar dan memiliki akses ke kunci rahasia aplikasi hanya sekali ini. Pastikan untuk menyalinnya dan menyimpannya di tempat yang aman.

## Perbarui aplikasi

Anda perlu memperbarui aplikasi Anda jika salah satu URI pengalihan Anda berubah atau Anda ingin menambah atau menghapus URI.

Untuk memperbarui aplikasi yang sudah ada, lakukan hal berikut:

<Steps>
  1. Klik tombol `Actions` tautan untuk aplikasi yang ingin Anda perbarui.
  2. Klik tombol `Update` tombol di bawah Tindakan.
  3. Lakukan perubahan pada aplikasi.
  4. Klik tombol `Save` untuk melanjutkan.
</Steps>


Anda diarahkan kembali ke daftar aplikasi Anda.

## Hapus aplikasi

:::peringatan
Harap jangan hapus aplikasi yang kami buat secara otomatis untuk Anda guna memastikan semua fitur berfungsi pada akun Anda.
:::

Jika Anda tidak lagi membutuhkan aplikasi tersebut, Anda dapat menghapusnya.

Untuk menghapus aplikasi, lakukan hal berikut:

<Steps>
  1. Klik tombol `Actions` tautan untuk aplikasi yang ingin Anda perbarui.
  2. Klik tombol `Remove` tombol di bawah Tindakan.
  3. Klik `OK` untuk mengonfirmasi penghapusan.
</Steps>


Anda diarahkan kembali ke daftar aplikasi Anda.
