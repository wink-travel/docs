Bagi siapa pun yang menggunakan WordPress untuk bisnis atau blog mereka, kami sangat menyarankan untuk menggunakan[Plugin Afiliasi WordPress Wink](https://wordpress.org/plugins/wink2travel/).
Plugin ini memberi Anda akses mudah ke kartu Wink, kisi, peta, dsb., dan membuatnya semudah menyeret konten ke dalam posting dan halaman Anda menggunakan pembuat halaman favorit Anda atau kode pendek khusus kami.

## Memasang

Beginilah cara Anda menginstalnya`Wink Affiliate WordPress Plugin`ke situs web WordPress Anda.

<Steps>
  1. Saat Anda masuk sebagai administrator dan berada di Dasbor.
  2. Klik `Plugins > Add New Plugin`.
  3. Pencarian untuk `Wink Affiliate WordPress Plugin`.
  4. Klik `Install` untuk mengunduh dan memasang plugin ke situs Anda.
  5. Klik `Activate` untuk mengaktifkan plugin di situs Anda.
  6. Aktivasi akan meneruskan Anda `Wink Settings` halaman. *Anda juga dapat pergi ke sana dengan menavigasi ke `Appearance > Customize > Wink Settings`.*
  7. Ini akan meminta Anda untuk `Client ID` Dan `Client Secret`Pergi ke [Aplikasi](/developers/applications) untuk mempelajari cara mengambil kredensial Anda.
  8. Masukkan kredensial Anda dan klik `Publish`Anda sekarang terhubung ke akun Wink Anda.
</Steps>

## Pembuat halaman

Kami mendukung pembuat halaman terpopuler di WordPress yang memudahkan siapa saja yang ingin menggunakan Wink.

### Bahasa Inggris Gutenberg

Gutenberg adalah editor asli dari WordPress.

### Ada

[Ada](https://avada.com)terkenal karena banyaknya pilihan templat yang indah beserta pembuat halaman yang tangguh dengan serangkaian fitur yang canggih.

### Elemen

[Elemen](https://elementor.com)adalah pembuat halaman \[saat ini] paling populer dengan versi gratis dan pro yang tersedia.

### WPBakery

[WPBakery](https://wpbakery.com)adalah pembuat halaman canggih pertama di pasaran.

## Kode pendek

### \[tampilan kedip]

Berfungsi dengan kisi konten berperingkat. Ketik tempat yang ingin Anda kunjungi dan inventaris akan ditampilkan.

### \[pencarian kedip]

Tombol sederhana untuk membuka formulir rencana perjalanan.

### \[akun kedip]

Tombol akun untuk memungkinkan Anda melakukan autentikasi. Setelah diautentikasi, tombol tersebut akan berubah menjadi menu tarik-turun dengan opsi akun.

### \[catatan perjalanan winkitinerary]

Sama seperti tombol pencarian, hanya saja tombol ini berisi informasi rencana perjalanan sebagai teks tombol.

### \[konten kedip]

Memungkinkan Anda menyematkan inventaris yang telah Anda pilih di Wink langsung ke halaman atau posting. Kode pendek ini juga memerlukan ID tata letak.

Tata letak

```
[winkcontent layoutid=""]
```
