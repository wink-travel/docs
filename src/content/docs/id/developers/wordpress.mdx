Untuk siapa saja yang menggunakan WordPress untuk bisnis atau blog mereka, kami sangat merekomendasikan menggunakan [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
Plugin ini memberi Anda akses mudah ke kartu, grid, peta Wink Anda, dan membuatnya semudah menyeret konten ke dalam posting dan halaman menggunakan pembuat halaman favorit Anda atau shortcode khusus kami.

##  Instalasi

Berikut cara menginstal `Wink Affiliate WordPress Plugin` ke dalam situs WordPress Anda.

<Steps>
1. Saat Anda masuk sebagai administrator dan berada di Dashboard.
2. Klik `Plugins > Add New Plugin`.
3. Cari `Wink Affiliate WordPress Plugin`.
4. Klik `Install` untuk mengunduh dan menginstal plugin ke situs Anda.
5. Klik `Activate` untuk mengaktifkan plugin di situs Anda.
6. Aktivasi akan mengarahkan Anda ke halaman `Wink Settings`. *Anda juga dapat pergi ke sana dengan menavigasi ke `Appearance > Customize > Wink Settings`.*
7. Anda akan diminta memasukkan `Client ID` dan `Client Secret`. Kunjungi [Applications](/id/developers/applications) untuk mempelajari cara mendapatkan kredensial Anda.
8. Masukkan kredensial Anda dan klik `Publish`. Anda sekarang terhubung ke akun Wink Anda.
</Steps>

## Pembuat halaman

Kami mendukung pembuat halaman paling populer di WordPress yang memudahkan siapa saja yang ingin menggunakan Wink.

### Gutenberg

Gutenberg adalah editor bawaan dari WordPress.

### Avada

[Avada](https://avada.com) terkenal dengan pilihan template yang banyak dan indah serta pembuat halaman yang kuat dengan fitur canggih.

### Elementor

[Elementor](https://elementor.com) adalah pembuat halaman yang [saat ini] paling populer dengan versi gratis dan pro yang tersedia.

### WPBakery

[WPBakery](https://wpbakery.com) adalah pembuat halaman canggih pertama di pasar.

## Short codes

### [winklookup]

Bekerja dengan grid konten berperingkat. Ketik tempat yang ingin Anda kunjungi dan inventaris akan ditampilkan.

### [winksearch]

Tombol sederhana untuk membuka formulir rencana perjalanan.

### [winkaccount]

Tombol akun untuk mengautentikasi Anda. Setelah terautentikasi, tombol ini berubah menjadi dropdown dengan opsi akun.

### [winkitinerary]

Sama seperti tombol pencarian, hanya saja berisi informasi rencana perjalanan sebagai teks tombol.

### [winkcontent]

Memungkinkan Anda menyematkan inventaris yang telah Anda pilih di Wink langsung ke halaman atau posting. Shortcode ini juga mengharapkan ID tata letak.

Tata letak

```
[winkcontent layoutid=""]
```
