Keterampilan AI adalah paket instruksi portabel yang mengajarkan agen pengkodean cara bekerja dengan Wink. Alih-alih mengandalkan data pelatihan atau dokumentasi yang usang, setiap keterampilan mengarahkan agen untuk menggunakan [server MCP Wink](/id/developers/mcp) sebagai sumber kebenaran langsung untuk kontrak API dan dokumen platform.

Keterampilan ini bekerja dengan klien agen apa pun yang mendukungnya — Claude Code, Codex, Cursor, dan lainnya.

## Keterampilan yang Tersedia

### wink-api-builder

Mengimplementasikan, meninjau, atau men-debug integrasi API Wink. Keterampilan ini membimbing agen melalui alur kerja terstruktur:

1. Cari operasi API yang tepat melalui `api_search`
2. Buka kontrak permintaan/respons langsung dari server MCP
3. Ambil pengaturan otentikasi dari dokumen
4. Tulis kode berdasarkan kontrak yang terverifikasi
5. Tandai ketidaksesuaian antara kode yang ada dan API saat ini

Berguna untuk:
- Membangun metode klien baru untuk API Wink apa pun
- Men-debug ketidaksesuaian kontrak (misalnya permintaan yang mengembalikan HTTP 400)
- Meninjau integrasi yang ada terhadap spesifikasi langsung

## Instalasi

Salin folder keterampilan ke direktori keterampilan agen Anda.

:::tip
Server MCP Wink harus terhubung di lingkungan yang sama. Lihat [Server MCP](/id/developers/mcp) untuk petunjuk pengaturan.
:::

**Claude Code**

```bash
# repo-local
cp -r skills/wink-api-builder .claude/skills/

# atau user-level
cp -r skills/wink-api-builder ~/.claude/skills/
```

Panggil dengan:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repo-local
cp -r skills/wink-api-builder .agents/skills/

# atau user-level
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Panggil dengan:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Agen lain**

Tempatkan folder keterampilan di mana pun agen Anda mengambil keterampilan. Titik masuknya adalah `SKILL.md`.
