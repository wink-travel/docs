Menambahkan informasi GM membantu menciptakan keakraban lebih dengan tempat tersebut.

Untuk memperbarui informasi Manajer Umum, klik `Property > General Manager` di tab navigasi utama.

Anda dapat menambahkan nama dan foto GM Anda saat ini beserta pesan pribadi untuk tamu.

Perbarui formulir dan klik tombol `Save` untuk melanjutkan.
