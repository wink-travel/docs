Pengumuman adalah cara untuk memberi tahu tamu Anda tentang informasi penting dengan cara yang mudah diperhatikan.
Jika sebuah pengumuman `dipicu`, itu akan muncul sebagai jendela baru di atas halaman utama Anda.

Untuk melihat pengumuman Anda, klik `Properti > Pengumuman` di bilah navigasi utama.

Klik tombol `Buat pengumuman` untuk membuat entri baru.

<Steps>
1. Pilih apakah ingin menyertakan judul atau tidak.
2. Pengumuman dapat dipicu dengan salah satu dari dua cara:
    - Selalu tampil, setiap kali seseorang mengunjungi halaman Anda.
    - Tampilkan hanya ketika rencana perjalanan cocok dengan rencana perjalanan pengumuman.
3. Jika Anda memilih untuk menampilkan pengumuman hanya untuk rentang tanggal tertentu, tambahkan tanggalnya di kolom formulir di bawah.
4. Klik tombol `Simpan` dan Anda akan dibawa kembali ke daftar pengumuman Anda.
</Steps>
