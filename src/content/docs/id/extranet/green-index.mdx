Indeks hijau Anda diukur dari dampak properti Anda terhadap lingkungan lokal dan dalam keputusan pembelian Anda.

Untuk mengelola skor Indeks Hijau Anda, klik `Property > Green Index` di tab navigasi utama.

Lalui setiap tab dan jawab kuesioner sebaik mungkin.

Klik tombol `Save` untuk melanjutkan.
