Profil properti adalah langkah wajib dalam wizard onboarding kami dan memberi kami kesempatan untuk mengenal properti Anda lebih baik menggunakan kriteria, seperti:

- Jumlah bintang.
- Jenis hotel.
- Tipe lokasi.
- ...dan lainnya

Untuk mengelola profil properti Anda, klik `Property > Profile` di tab navigasi utama.

Perbarui formulir dan klik tombol `Save` untuk melanjutkan.
