Anda akan diminta untuk menulis deskripsi panjang dan pendek tentang properti Anda selama pengaturan.

Untuk mengelola teks sambutan Anda, klik `Property > Welcome Text` di tab navigasi utama.

Perbarui formulir dan klik tombol `Save` untuk melanjutkan.

:::note
Setidaknya, kami membutuhkan Anda untuk menyelesaikan teks sambutan dalam bahasa Inggris.

Buat teks sambutan untuk semua bahasa lain yang ingin Anda kelola.
Kami akan menggunakan teks bahasa Inggris Anda untuk menerjemahkan ke semua bahasa populer lainnya.
:::
