Pencarian tersimpan melakukan apa yang namanya maksudkan. Saat mencari inventaris perjalanan, Anda memiliki opsi untuk menyimpan kueri pencarian Anda untuk digunakan nanti.

## Menyimpan kueri pencarian Anda

Saat Anda mulai mencari inventaris (lihat [Search](/id/studio/search) ), Anda dapat menyempurnakan filter pencarian Anda.
Di bagian bawah formulir kueri terdapat opsi `Save search`.

Mengklik tautan ini akan memberi Anda kesempatan untuk memberi nama pencarian ini (*misalnya Perpustakaan Hotel Ajaib Saya*) sesuatu yang akan Anda ingat untuk nanti.

Beri nama dan klik tombol `Save search`. Setelah itu, Anda dapat menavigasi ke `Inventory > Saves Searches` dari bilah navigasi utama untuk melihat catatan yang baru dibuat.

## Tindakan daftar

Pencarian tersimpan Anda dapat digunakan untuk menampilkan inventaris perjalanan kepada pengguna Anda dalam bentuk grid atau peta dan membuat hasil pencarian Anda tampil sesuai keinginan.

<figure>

Setiap kueri pencarian tersimpan memiliki tindakan yang dapat Anda jalankan:

- Ubah kueri pencarian menjadi [map](/id/studio/maps).
- Ubah kueri pencarian menjadi [grid](/id/studio/grids).
- Hapus pencarian tersimpan.

## API

Pengembang yang ingin mengelola pencarian tersimpan dapat menuju ke [Developers > API > Browse](/id/developers/apis/#browse-api).

## Bacaan lanjutan

- Baca tentang fitur [Map](/id/studio/maps) kami.
- Baca tentang fitur [Grid](/id/studio/grids) kami.
