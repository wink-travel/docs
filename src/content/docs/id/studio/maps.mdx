Sisa artikel ini memandu Anda cara membuat, menyesuaikan, dan membagikan peta Anda.

## Membuat peta

Peta dapat dibuat dengan dua cara:

- **Peta penanda tunggal**: Buat peta dari sesuatu yang Anda temukan saat [mencari](/id/studio/search).
- **Peta penanda ganda**: Buat peta dari [daftar kurasi](/id/studio/curated-lists) atau [pencarian tersimpan](/id/studio/saved-searches).

Klik tombol itu dan Anda akan diarahkan ke halaman formulir peta kami di mana Anda dapat mulai menyesuaikan peta Anda.

### Peta penanda ganda

Baik [daftar kurasi](/id/studio/curated-lists#list-actions) maupun [pencarian tersimpan](/id/studio/saved-searches#list-actions) memiliki tindakan yang memungkinkan Anda membuat peta. Peta ini akan berisi penanda peta untuk semua inventaris dalam daftar tersebut.

## Menyesuaikan peta

Formulir memungkinkan Anda menyesuaikan peta dengan cara berikut:

<Steps>
 1. Beri nama agar Anda ingat tentang apa peta tersebut.
 2. Pilih apakah Anda ingin pengguna diizinkan untuk bergerak di sekitar peta.
 3. Atur tinggi peta dalam piksel sesuai keinginan Anda.
 4. Pilih apakah Anda ingin pengguna diizinkan untuk memperbesar dan memperkecil peta.
 5. Pilih gaya peta yang sesuai dengan situs web tempat Anda akan menyematkan peta.
 6. Pilih warna penanda.
 7. Pilih tampilan kartu awal yang ingin dilihat pengguna terlebih dahulu. Secara default, ini adalah tampilan asli dari inventaris tersebut.
 8. Anda dapat menambahkan penanda, lingkaran, persegi panjang, dan poligon Anda sendiri untuk menunjukkan lokasi lain di peta yang mungkin menarik bagi pengguna Anda.
 9. Klik tombol `Simpan` untuk melanjutkan.
</Steps>

Setelah menyimpan peta Anda, Anda akan diarahkan ke halaman peta Anda dan peta Anda sekarang siap untuk dibagikan ke dunia.

Di atas menunjukkan gambar dengan semua tindakan yang tersedia untuk peta Anda:

- **Perbarui** Memperbarui konfigurasi peta Anda.
- **Tambahkan ke WinkLinks** Menambahkan peta ke akun WinkLinks Anda.
- **Sematkan** Menunjukkan cara menyematkan peta ini sebagai [Map](/id/developers/web-components/#content-loader) ke dalam situs web Anda.
- **Gunakan dengan WordPress** Menunjukkan cara menggunakan [plugin WordPress](/id/developers/wordpress) kami untuk menyematkan peta ini ke dalam situs web Anda.

Kami membahas beberapa opsi ini secara lebih rinci di bawah.

### Sematkan

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Berikut cara menyematkan peta Anda ke situs Anda:

- Baris 3 menunjukkan cara menyematkan gaya Wink ke situs Anda.
- Baris 6 sampai 9 menunjukkan cara menggunakan Komponen Web [wink-content-loader](/id/developers/web-components/#content-loader) dan memberitahunya untuk mengambil banner iklan untuk kode Anda.
- Baris 11 menunjukkan cara menyematkan Javascript kami ke situs Anda.
- Baris 13 menunjukkan cara menyematkan Komponen Web [wink-app-loader](/id/developers/web-components/#application-loader) dan memberitahunya untuk mengambil preferensi konfigurasi tingkat halaman Anda.

## API

Pengembang yang ingin mengelola peta dapat menuju ke [Developers > API > Inventory](/id/developers/apis/#inventory-api).

## Bacaan lanjutan

- Pelajari lebih lanjut tentang koleksi [Komponen Web](/id/developers/web-components).
- Pelajari lebih lanjut tentang [plugin WordPress Wink](/id/developers/wordpress).
