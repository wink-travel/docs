Agensi adalah akun Wink biasa yang dapat Anda tetapkan ke akun Anda untuk tujuan representasi. Berikut ini adalah beberapa alasan mengapa Anda menginginkan agensi yang mewakili Anda:

* Pengelolaan penuh/sebagian akun Wink Anda oleh pihak ke-3.
* Talent sudah memiliki perjanjian dengan agensi mereka dan ingin Wink menangani aspek keuangan dari pengaturan dana yang masuk melalui Wink.
* Agensi bakat melakukan semua pekerjaan daring dan mengelola semua akun di Wink.

## Menetapkan agensi

Berikut ini cara menetapkan agensi pada akun Anda.

<Steps>
  1. Masuk ke akun pengguna Anda di [Studio Kedipan](https://studio.wink.travel).
  2. Pilih akun afiliasi yang ingin Anda gunakan.
  3. Klik di sini `Account > Managers` dari bilah navigasi utama.
  4. Di dalam `Agency` bagian, ketik nama lembaga yang ingin Anda wakili.
  5. Pilih biaya agensi dari menu dropdown.
  6. Tambahkan aturan kedaluwarsa secara opsional.
  7. Klik tombol `Set agency` untuk melanjutkan.
</Steps>


:::catatan
Anda harus menjadi pemilik akun untuk menetapkan agensi.
:::

:::tip\[Contoh]
Biaya afiliasi berasal dari komisi afiliasi.

Pemesanan = $100, Komisi = 10%, Biaya agensi = 10% => 100 \* 0,1 \* 0,1 = $1
:::

## Batalkan penetapan agensi

Berikut ini cara membatalkan penetapan agensi dari akun Anda.

<Steps>
  1. Masuk ke akun pengguna Anda di [Studio Kedipan](https://studio.wink.travel).
  2. Pilih akun afiliasi yang ingin Anda gunakan.
  3. Klik di sini `Account > Managers` dari bilah navigasi utama.
  4. Di dalam `Agency` bagian, klik <Badge text="x" variant="danger" size="small" /> ikon di sisi kanan menu dropdown agensi.
  5. Klik tombol `Set agency` untuk melanjutkan.
</Steps>


## API

Pengembang yang ingin mengelola agensi melalui API dapat menuju ke[Pengembang > API > Afiliasi](/developers/apis/#affiliate-api).
