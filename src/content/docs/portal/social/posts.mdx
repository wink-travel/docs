The post editor lets you create unlimited manual posts and it's also the place you go to manage posts that have been created for you automatically.

The editor has 4 sections:

1. The content section.
2. The media section.
3. The link section.
4. The settings section.

## Content

The content section allows you to modify the text content before it gets published. You can create localized content in multiple languages — each language gets its own title and description.

## Media

The media section allows you to modify the images and videos that go along with the text.

## Link

The link section allows you to link to somewhere you wish the user to go after reading your post. *E.g. This can be a link to a special deal or a place to go to for more information.* You can add a link URL, link caption, and link description.

## Settings

The settings section lets you modify / override all your default settings per post. You can set:

- **Platform** — Choose which social network this post will be published to.
- **Post type** — Choose between Transactional, Awareness, or Sentiment.
- **Tone of voice** — Override the tone of voice for this specific post.
- **Customization** — Use a different customization for display currency, language, itinerary, etc.
- **Timezone** — Override the timezone for publishing.
- **Scheduled date** — Generated posts are automatically scheduled at the optimal time. You can override the date and time if you prefer a different slot.

## Post status lifecycle

Every post follows a status lifecycle:

| Status | Description |
| -- | -- |
| **Draft** | Post has been created but is not yet ready for review |
| **Pending approval** | Post is waiting for human approval before scheduling |
| **Approved** | Post has been approved and will be scheduled |
| **Rejected** | Post was reviewed and rejected — it will not be published |
| **Scheduled** | Post is queued for publication at the scheduled date and time |
| **Published** | Post has been successfully published to the social network |
| **Paused** | Post publication has been temporarily paused |
| **Failed** | Post could not be published due to an error |

## Approving posts

Posts are approved in the post grid where you can select one or more posts and choose to approve or reject them.

:::note
There will be one entry for every language and every social network.

E.g. If you have 3 social networks and 3 languages selected, you will see 9 entries on the calendar.
:::
