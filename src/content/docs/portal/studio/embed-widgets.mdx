Studio provides several embeddable widgets you can add to your own website to drive bookings.

To access embed widgets, navigate to `Tools > Miscellaneous` from the sidebar.

## Available widgets

| Widget | Description |
| -- | -- |
| **Account button** | A branded button that links to your affiliate page |
| **Itinerary button** | A button that opens the itinerary selector |
| **Itinerary form** | An inline date and guest selector form |
| **Search button** | A button that opens property search |
| **Lookup button** | A button to look up specific inventory |

## How to embed

Each widget provides a code snippet you can copy and paste into your website's HTML. The snippet includes your **Client ID** and **Customization ID** so the widget is automatically linked to your affiliate account.

:::note
If you use WordPress, we also provide a WordPress plugin for easier integration.
:::
