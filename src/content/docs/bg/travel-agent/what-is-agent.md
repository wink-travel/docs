Нашето решение за регистрирани туристически агенти е [Wink Agent](https://agent.wink.travel). Това е самообслужващ се портал, който агентите могат да използват, за да търсят и резервират от името на своите клиенти и да печелят комисионна за себе си при всяка резервация.

По-напредналите туристически агенти могат да изберат интеграция на ниво API с цел обработка на плащания и изплащания извън платформата Wink.

Туристическите агенти имат подмножество от същите функции на портала като [Wink Studio](/bg/studio/what-is-studio), затова няма да ги разглеждаме отново тук.

## Търсене

Отидете на [Search](/bg/studio/search), за да научите как да търсите и резервирате хотели.

В противен случай продължете към следващия раздел.

## API

Разработчиците, които искат да управляват `Travel Agent`, могат да отидат на [Developers > API > Travel Agent](/bg/developers/apis/#travel-agent-api).
