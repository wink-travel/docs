Тази страница изброява всички webhook събития, на които можете да се абонирате в Wink. За инструкции за настройка вижте [Ръководството за интеграция на Webhook](/bg/guides/webhook-integration) или [страницата за управление на Webhooks](/bg/my-account/webhooks).

## Как да се абонирате

1. Отидете в **Applications > Webhooks** в портала си.
2. Създайте webhook и изберете събитията, които искате да получавате.
3. Wink изпраща HTTP POST към вашия крайна точка всеки път, когато събитието се случи.

За пълни стъпки за настройка и добри практики вижте [Интеграция на Webhook](/bg/guides/webhook-integration).

## Събития за акаунт

Събития, свързани с жизнения цикъл и настройките на вашия TripPay акаунт.

| Събитие | Ключ | Описание |
|---|---|---|
| Акаунт създаден | `account.create` | Създаден е нов акаунт. |
| Акаунт под преглед | `account.under_review` | Акаунтът е в процес на преглед. |
| Акаунт актуализиран | `account.updated` | Детайлите на акаунта са променени. |
| Статус на акаунта актуализиран | `account.status.updated` | Статусът на акаунта е променен (напр. одобрен, спрян). |
| Адрес на акаунта актуализиран | `account.address.updated` | Адресът на акаунта е променен. |
| Детайли на акаунта актуализирани | `account.details.updated` | Полетата с детайли на акаунта са променени. |
| Информация за акаунта актуализирана | `account.information.update` | Общата информация за акаунта е обновена. |
| Мениджъри на акаунта актуализирани | `account.managers.update` | Списъкът с мениджъри на акаунта е променен. |
| Профилна снимка актуализирана | `account.profile_picture.updated` | Профилната снимка на акаунта е променена. |
| Агент актуализиран | `account.agent.updated` | Информацията за агента на акаунта е променена. |
| URL име актуализирано | `account.url-name.update` | Персонализираният URL на акаунта е променен. |
| Уникален идентификатор актуализиран | `account.unique-id.update` | Уникалният идентификатор на акаунта е променен. |

### Задачи за акаунт

| Събитие | Ключ | Описание |
|---|---|---|
| Задача създадена | `account.task.create` | Добавена е нова задача към акаунта. |
| Задача актуализирана | `account.task.update` | Задача в акаунта е променена. |
| Задача премахната | `account.task.remove` | Задача в акаунта е премахната. |
| Задача завършена | `account.task.complete` | Задача в акаунта е завършена. |
| Задача за данъчен идентификатор завършена | `account.task.tax_id.complete` | Задачата за проверка на данъчен идентификатор е завършена. |

### Приемател на плащания за акаунт

| Събитие | Ключ | Описание |
|---|---|---|
| Приемател създаден | `account.acquirer.created` | Добавен е нов приемател към акаунта. |
| Приемател актуализиран | `account.acquirer.updated` | Приемател на акаунта е променен. |
| Приемател премахнат | `account.acquirer.removed` | Приемател е премахнат от акаунта. |

## Събития за договор за резервация

Събития, проследяващи жизнения цикъл на договор за резервация чрез TripPay.

| Събитие | Ключ | Описание |
|---|---|---|
| Инициализацията е заявена | `booking.contract.initialize.request` | Заявена е инициализация на договор за резервация. |
| Инициализацията е завършена | `booking.contract.initialize.complete` | Инициализацията на договор за резервация е завършена. |
| Договор създаден | `booking.contract.created` | Създаден е нов договор за резервация. |
| Обработка на договора | `booking.contract.processing` | Плащането по договора се обработва. |
| Договорът е неуспешен | `booking.contract.failed` | Плащането по договора е неуспешно. |
| Договорът е анулиран | `booking.contract.cancelled` | Договорът за резервация е анулиран. |
| Частично възстановяване | `booking.contract.refund.partial` | Извършено е частично възстановяване по договора. |
| Пълно възстановяване | `booking.contract.refund.full` | Извършено е пълно възстановяване по договора. |
| Заявка за възстановяване | `booking.contract.refund.request` | Подадена е заявка за възстановяване. |
| Грешка при възстановяване | `booking.contract.refund.error` | Възникна грешка при обработка на възстановяване. |
| Плащане създадено | `booking.contract.payout.create` | Създадено е плащане по договора. |
| Плащане завършено | `booking.contract.payout.complete` | Плащането е успешно завършено. |

## Събития за управляващи субекти

Събития за управляващи субекти (имоти, филиали, компании) и тяхната администрация.

### Абонамент

| Събитие | Ключ | Описание |
|---|---|---|
| Абонамент активиран | `managing_entity.subscription.provisioned` | Нов абонамент е активиран. |
| Абонамент променен | `managing_entity.subscription.changed` | Планът на абонамента е променен. |
| Фактура за абонамент променена | `managing_entity.subscription.invoice.changed` | Фактурата за абонамента е обновена. |

### Статус и администрация

| Събитие | Ключ | Описание |
|---|---|---|
| Статус актуализиран | `managing_entity.status.updated` | Статусът на субекта е променен. |
| Субект изтрит | `managing_entity.deleted` | Управляващ субект е изтрит. |
| Агент актуализиран | `managing_entity.agent.updated` | Агентът на субекта е променен. |
| Известие по имейл до агент | `managing_entity.agent.email.notification` | Изпратено е имейл известие до агента. |
| Известие по имейл до мениджър | `managing_entity.manager.email.notification` | Изпратено е имейл известие до мениджър. |

### Жизнен цикъл на мениджър

| Събитие | Ключ | Описание |
|---|---|---|
| Мениджър добавен | `managing_entity.manager.added` | Добавен е нов мениджър. |
| Мениджър поканен | `managing_entity.manager.invited` | Изпратена е покана към мениджър. |
| Мениджър премахнат | `managing_entity.manager.removed` | Мениджър е премахнат. |
| Мениджър приел | `managing_entity.manager.accepted` | Поканата към мениджър е приета. |
| Мениджър отказал | `managing_entity.manager.rejected` | Поканата към мениджър е отказана. |
| Собственик зададен | `managing_entity.manager.owner.set` | Собствеността на субекта е прехвърлена. |

### Информация за субекта

| Събитие | Ключ | Описание |
|---|---|---|
| Информация актуализирана | `managing_entity.information.updated` | Информацията за субекта е обновена. |
| Адрес актуализиран | `managing_entity.city.updated` | Адресът/градът на субекта е променен. |
| Представител на акаунта актуализиран | `managing_entity.account_representative.updated` | Представителят на акаунта е променен. |
| Профилна снимка актуализирана | `managing_entity.profile_picture.updated` | Профилната снимка на субекта е обновена. |
| Лого актуализирано | `managing_entity.logo.updated` | Логото на субекта е обновено. |
| URL име актуализирано | `managing_entity.url_name.updated` | Персонализираният URL на субекта е променен. |
| Уникален идентификатор актуализиран | `managing_entity.unique_id.updated` | Уникалният идентификатор на субекта е променен. |
