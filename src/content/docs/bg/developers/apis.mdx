## Консумиране на крайни точки

Крайните точки за потребление са предназначени за разработчици, които искат да намерят съществуващ инвентар за пътувания и да го резервират или да го използват за реклама чрез един от своите партньорски акаунти в Wink.

### API за конфигурация

Единна крайна точка за извличане на информация за whitelabel + персонализиране за системата за резервации.

<LinkButton href="https://docs.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
  Научете повече
</LinkButton>


### API за търсене

Удобен за програмисти начин за търсене на персонализирани туристически предложения в различни географски региони.

<LinkButton href="https://docs.wink.travel/lookup" variant="secondary" icon="external" target="_blank">
  Научете повече
</LinkButton>


### API за инвентаризация

Удобен за програмисти начин за извличане на известни туристически наличности, както са били официално локализирани, чрез Lookup API.

<LinkButton href="https://docs.wink.travel/inventory" variant="secondary" icon="external" target="_blank">
  Научете повече
</LinkButton>


### API за резервации

## Създаване на крайни точки

Крайните точки за производство са предназначени за разработчици, които искат да създават и управляват инвентаризация на пътувания.

### API за регистрация

Като продуцент, това често е мястото, откъдето започвате своето пътешествие. Тези крайни точки ви позволяват да създавате свойства в Wink.

<LinkButton href="https://docs.wink.travel/extranet/property/register" variant="secondary" icon="external" target="_blank">
  Научете повече
</LinkButton>


### API за свойства

## API за плащане

Крайните точки за плащане са предназначени за разработчици, които искат да закупят туристически инвентар. Това може да се направи чрез API като регистриран туристически агент или чрез използването на нашия API във връзка с нашия PCI съвместим платежен уиджет за всички останали организации.
API за плащане предоставя крайни точки на партньори и хотели за проследяване на резервации, анализи и наличност на средства. Те могат също така да изберат да теглят налични средства в банковата си сметка.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Научете повече
</LinkButton>
