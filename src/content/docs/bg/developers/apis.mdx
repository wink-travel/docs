## Consume endpoints

Една крайна точка за извличане на информация за брандиране и персонализация на резервационния двигател.

</LinkButton>

## Taxonomy API

Taxonomy крайните точки са за разработчици, които искат да консумират и произвеждат туристически инвентар и имат нужда от таксономии на стандартни и нестандартни кодове за типове инвентар, класове, статуси и др.

## Payment API

Платежните крайни точки са за разработчици, които искат да закупят туристически инвентар. Това може да се направи чрез API като регистриран Travel Agent или чрез нашия API в комбинация с нашия PCI съвместим платежен уиджет за всички други обекти.
Payment API предоставя крайни точки за афилиейти и хотели за проследяване на резервации, анализи и наличност на средства. Те също могат да изберат да изтеглят наличните средства към своята банкова сметка.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Научете повече
</LinkButton>
