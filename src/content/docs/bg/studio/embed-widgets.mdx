Studio предоставя няколко вградими джаджи, които можете да добавите към собствения си уебсайт, за да увеличите резервациите.

За да получите достъп до вградените джаджи, отидете на `Tools > Miscellaneous` от страничната лента.

## Налични джаджи

| Джаджа | Описание |
| -- | -- |
| **Account button** | Брандиран бутон, който води към вашата афилиейт страница |
| **Itinerary button** | Бутон, който отваря селектора на маршрути |
| **Itinerary form** | Вградена форма за избор на дати и гости |
| **Search button** | Бутон, който отваря търсене на имоти |
| **Lookup button** | Бутон за търсене на конкретен инвентар |

## Как да вградите

Всяка джаджа предоставя кодов фрагмент, който можете да копирате и поставите в HTML на вашия уебсайт. Фрагментът включва вашия **Client ID** и **Customization ID**, така че джаджата автоматично да е свързана с вашия афилиейт акаунт.

:::note
Ако използвате WordPress, ние също предоставяме WordPress плъгин за по-лесна интеграция.
:::
