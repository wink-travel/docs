Svelte предава атрибутите директно към DOM без слой виртуален DOM, което прави потребителските елементи естествено подходящи. За SvelteKit (SSR) заредете CDN вътре в `onMount` или използвайте проверка с `browser`.

<LinkCard

/>

## Инсталиране

```bash
npm install @wink/elements
```

## SvelteKit — основен layout

```ts
// src/routes/+layout.ts
import { browser } from '$app/environment';
import { load as loadWink } from '@wink/elements';

export function load() {
  if (browser) {
    loadWink({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  }
}
```

Като алтернатива, заредете вътре в `onMount` в `+layout.svelte`:

```svelte
<!-- src/routes/+layout.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<slot />
```

## Използване в компоненти

Svelte предава всички непознати атрибути директно към DOM, така че не е необходима допълнителна конфигурация:

```svelte
<script>
  export let layoutId = 'YOUR_LAYOUT_ID';
</script>

<main>
  <wink-content-loader layout="HOTEL" id={layoutId} />
  <wink-lookup />
</main>
```

## Svelte 5 (runes синтаксис)

```svelte
<script lang="ts">
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  let { layoutId }: { layoutId: string } = $props();

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id={layoutId} />
```

## Променливи на средата

SvelteKit предоставя на браузъра само променливи с префикс `PUBLIC_`:

```
# .env
PUBLIC_WINK_CLIENT_ID=your-client-id
```

<Aside type="caution" title="SSR рендериране">
Никога не рендерирайте `<wink-*>` елементи по време на сървърно рендериране — потребителските елементи се регистрират в браузърния регистър `customElements`, който не съществува на сървъра. Обвийте в `{#if browser}` или извикайте `load()` само вътре в `onMount`.
</Aside>

```svelte
<script>
  import { browser } from '$app/environment';
</script>

{#if browser}
  <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
{/if}
```

## Чист Svelte (Vite, без SvelteKit)

```svelte
<!-- src/App.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.VITE_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
```
