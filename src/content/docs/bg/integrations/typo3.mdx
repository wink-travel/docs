**Wink TYPO3 Extension** интегрира Wink travel инвентар в TYPO3 13 като нативни съдържателни елементи. Редакторите добавят Wink компоненти към страниците чрез стандартния редактор на страници в TYPO3 — не са необходими промени в шаблона.

## Изисквания

- TYPO3 13.x
- PHP 8.2+
- Composer

## Инсталация

<Steps>
1. Инсталирайте чрез Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Активирайте разширението:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. В бекенда на TYPO3 отидете на **Admin Tools → Settings → Extension Configuration → wink**.
4. Въведете своя **Client ID**, **Client Secret** и изберете своята среда. Запазете.
   Отидете на [Applications](/bg/portal/settings/applications), за да получите своите данни за достъп.
5. Добавете статичния TypoScript шаблон към основния си шаблон:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Добавяне на съдържателен елемент към страница

<Steps>
1. Отворете страница в модула **Page**.
2. Кликнете върху бутона **+** в областта за съдържание, за да добавите нов елемент.
3. Изберете един от съдържателните елементи **Wink Travel**:
   - **Wink Content** — хотелски карти, подредени мрежи, карти
   - **Wink Lookup** — лента за търсене на дестинации
   - **Wink Search** — бутон за избор на маршрут
   - **Wink Account** — бутон за влизане
   - **Wink Itinerary** — бутон за статус на маршрута
   - **Wink Cart** — бутон за пазарска количка
4. За **Wink Content** използвайте полетата FlexForm, за да изберете **Layout Type** и въведете **Layout ID**.
5. Запазете и публикувайте страницата.
</Steps>

## Налични съдържателни елементи

| Елемент | Цел |
|---|---|
| **Wink Content** | Хотелски карти, подредени мрежи, карти и други |
| **Wink Lookup** | Лента за търсене на дестинации |
| **Wink Search** | Икона бутон, който отваря избора на маршрут |
| **Wink Account** | Бутон за влизане / потребителски акаунт |
| **Wink Itinerary** | Бутон за маршрут, показващ текущото пътуване |
| **Wink Cart** | Бутон за обобщение на пазарската количка |
