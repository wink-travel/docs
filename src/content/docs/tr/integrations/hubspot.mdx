HubSpot CMS, özel HTML modülleri ve site genelinde header/footer kodu desteği sunar, bu da Wink bileşenlerini pazarlama sayfalarında ve açılış sayfalarında kolayca gömmenizi sağlar.

:::note
Site genelinde kod eklemek için **HubSpot CMS Hub** (Starter veya üzeri) erişimi gereklidir. Özel HTML modülleri tüm CMS katmanlarında mevcuttur.
:::

## Wink scriptlerini site genelinde yükleyin

<Steps>
1. HubSpot'ta **Marketing → Files and Templates → Design Tools** bölümüne gidin.
2. Aktif **temanızın** `main.css` dosyasını açın veya global içerik ayarlarını kullanın.
3. Alternatif olarak, **Settings → Website → Pages → Custom Code** bölümüne gidin.
4. **Head HTML** alanına stil dosyasını ekleyin:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
5. **Footer HTML** alanına script ve uygulama yükleyicisini ekleyin (`YOUR_CLIENT_ID` ile değiştirin):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
6. Kaydedin ve yayınlayın.
</Steps>

:::tip
Müşteri Kimliğiniz, Wink hesabınızda [Applications](/tr/portal/settings/applications) altında bulunabilir.
:::

## Özel HTML modülü ile bileşen ekleyin

<Steps>
1. Design Tools'ta, **Rich Text** veya **HTML** türünde yeni bir **modül** oluşturun.
2. Modülün varsayılan içeriğine bileşen HTML'sini yapıştırın:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
3. Modülü kaydedin ve sayfa şablonuna veya sayfa düzenleyicisi ile herhangi bir sayfaya sürükleyin.
</Steps>

## Bileşeni doğrudan sayfa düzenleyicide ekleyin

Modül oluşturmak istemiyorsanız, sürükle-bırak düzenleyicide bir **Özel HTML** bölümü kullanabilirsiniz:

<Steps>
1. Sayfayı HubSpot sayfa düzenleyicisinde açın.
2. **Add** tıklayın ve bir **Custom HTML** veya **Embed** bölümü seçin.
3. Bileşen etiketini HTML alanına yapıştırın.
4. **Apply** tıklayın ve sayfayı yayınlayın.
</Steps>

## Mevcut bileşenler

| Bileşen | HTML etiketi | Amaç |
|---|---|---|
| İçerik yükleyici | `<wink-content-loader>` | Otel kartları, ızgaralar, haritalar gösterir |
| Arama | `<wink-lookup>` | Destinasyonlar için arama çubuğu |
| Arama butonu | `<wink-search-button>` | Seyahat planlayıcıyı açar |
| Hesap butonu | `<wink-account-button>` | Giriş / kullanıcı menüsü |
| Seyahat butonu | `<wink-itinerary-button>` | Mevcut seyahati gösterir |
| Seyahat seçici | `<wink-itinerary-picker>` | Tam seyahat formu |
| Alışveriş sepeti | `<wink-shopping-cart-button>` | Sepet özeti butonu |

Tam özellik referansı için [Web Components](/tr/integrations/web-components) sayfasına bakın.
