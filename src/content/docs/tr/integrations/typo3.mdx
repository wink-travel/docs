**Wink TYPO3 Uzantısı**, Wink seyahat envanterini TYPO3 13'e yerel içerik öğeleri olarak entegre eder. Editörler, standart TYPO3 sayfa düzenleyicisini kullanarak sayfalara Wink bileşenleri ekler — şablon değişikliği gerekmez.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"

  description="Kaynak kodu, geliştirici kurulum rehberi ve katkı talimatları."
/>

## Gereksinimler

- TYPO3 13.x
- PHP 8.2+
- Composer

## Kurulum

<Steps>
1. Composer ile yükleyin:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Uzantıyı etkinleştirin:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. TYPO3 arka ucunda, **Admin Tools → Settings → Extension Configuration → wink** bölümüne gidin.
4. **Client ID**, **Client Secret** bilgilerinizi girin ve ortamınızı seçin. Kaydedin.
   Kimlik bilgilerinizi almak için [Applications](/tr/portal/settings/applications) sayfasına gidin.
5. Statik TypoScript şablonunu kök şablonunuza ekleyin:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Bir sayfaya içerik öğesi ekleyin

<Steps>
1. **Page** modülünde bir sayfa açın.
2. İçerik alanındaki **+** butonuna tıklayarak yeni bir öğe ekleyin.
3. **Wink Travel** içerik öğelerinden birini seçin:
   - **Wink Content** — otel kartları, sıralı ızgaralar, haritalar
   - **Wink Lookup** — destinasyon arama çubuğu
   - **Wink Search** — seyahat planı seçici butonu
   - **Wink Account** — giriş butonu
   - **Wink Itinerary** — seyahat durumu butonu
   - **Wink Cart** — alışveriş sepeti butonu
4. **Wink Content** için, FlexForm alanlarını kullanarak bir **Layout Type** seçin ve **Layout ID** girin.
5. Sayfayı kaydedin ve yayınlayın.
</Steps>

## Mevcut içerik öğeleri

| Öğeler | Amaç |
|---|---|
| **Wink Content** | Otel kartları, sıralı ızgaralar, haritalar ve daha fazlası |
| **Wink Lookup** | Destinasyon arama çubuğu |
| **Wink Search** | Seyahat planı seçiciyi açan simge butonu |
| **Wink Account** | Giriş / kullanıcı hesabı butonu |
| **Wink Itinerary** | Mevcut seyahati gösteren seyahat butonu |
| **Wink Cart** | Alışveriş sepeti özet butonu |
