**Wink Drupal Modülü**, Wink seyahat envanteri bileşenlerini Drupal sitenize yerel bloklar olarak ekler. Editörler, standart blok sistemi aracılığıyla bunları düzen içinde istedikleri yere yerleştirebilirler.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Kaynak kodu, geliştirici kurulum rehberi ve katkı talimatları."
/>

## Gereksinimler

- Drupal 10 veya 11
- PHP 8.1+
- Composer

## Kurulum

<Steps>
1. Composer ile yükleyin:
   ```bash
   composer require drupal/wink
   ```
2. Modülü etkinleştirin:
   ```bash
   drush en wink -y
   ```
3. **Yönetici → Yapılandırma → Web Hizmetleri → Wink Ayarları** (`/admin/config/wink`) sayfasına gidin.
4. **Client ID** ve **Client Secret** bilgilerinizi girin, ortamınızı seçin ve **Kaydet** butonuna tıklayın.
   Kimlik bilgilerinizi almak için [Uygulamalar](/tr/portal/settings/applications) sayfasına gidin.
</Steps>

## Bir sayfaya blok ekleme

<Steps>
1. **Yönetici → Yapı → Blok Düzeni** sayfasına gidin.
2. Wink içeriğini eklemek istediğiniz bölge için **Blok yerleştir** butonuna tıklayın.
3. Mevcut blokları bulmak için **Wink** araması yapın:
   - **Wink Content** — otel kartı, ızgara veya harita gösterir
   - **Wink Lookup** — destinasyon arama çubuğu
   - **Wink Search** — seyahat planı seçici butonu
   - **Wink Account** — giriş butonu
   - **Wink Itinerary** — seyahat durumu butonu
   - **Wink Cart** — alışveriş sepeti butonu
4. **Wink Content** için blok ayarlarında **Düzen Türü** ve **Düzen ID** yapılandırmasını yapın.
5. Blok yapılandırmasını kaydedin.
</Steps>

## Mevcut bloklar

| Blok | Amaç |
|---|---|
| **Wink Content** | Otel kartları, sıralı ızgaralar, haritalar ve daha fazlasını gösterir |
| **Wink Lookup** | Destinasyonlar ve oteller için arama çubuğu |
| **Wink Search** | Seyahat planı seçiciyi açan simge butonu |
| **Wink Account** | Giriş / kullanıcı hesabı butonu |
| **Wink Itinerary** | Mevcut seyahati gösteren seyahat butonu |
| **Wink Cart** | Alışveriş sepeti özet butonu |
