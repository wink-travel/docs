:::note
Squarespace'in Code Injection özelliği **Business plan ve üzeri** planlarda mevcuttur. Özel HTML blokları tüm ücretli planlarda kullanılabilir.
:::

Squarespace, özel HTML ve JavaScript'i destekler; bu da herhangi bir Wink web bileşenini eklenti olmadan doğrudan sitenize gömebileceğiniz anlamına gelir. Bu rehber, Wink scriptlerini site genelinde nasıl yükleyeceğinizi ve ardından bileşenleri tek tek sayfalara nasıl yerleştireceğinizi gösterir.

## Wink scriptlerini site genelinde yükleyin

Scriptleri sitenizin global footer ve header alanlarına eklemek, Wink'in her sayfada kurulumu tekrarlamadan kullanılmasını sağlar.

<Steps>
1. Squarespace editöründe **Settings → Advanced → Code Injection** bölümüne gidin.
2. **Header** alanına stil dosyasını ekleyin:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. **Footer** alanına script ve uygulama yükleyicisini ekleyin (`YOUR_CLIENT_ID` yerine kendi Wink Client ID'nizi yazın):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. **Save** butonuna tıklayın.
</Steps>

:::tip
Client ID'niz Wink hesabınızda [Applications](/tr/portal/settings/applications) bölümünde bulunabilir.
:::

## Bir sayfaya bileşen ekleyin

Scriptler global olarak yüklendikten sonra, herhangi bir Wink bileşenini istediğiniz sayfaya **Code Block** kullanarak ekleyebilirsiniz.

<Steps>
1. Düzenlemek istediğiniz sayfayı açın.
2. **Edit**e tıklayın ve bileşenin görünmesini istediğiniz yere yeni bir **Code** bloğu ekleyin.
3. Örneğin bir içerik yükleyici HTML'sini yapıştırın:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. **Apply** ardından **Save** butonlarına tıklayın.
</Steps>

## Mevcut bileşenler

| Bileşen | HTML etiketi | Amaç |
|---|---|---|
| İçerik yükleyici | `<wink-content-loader>` | Otel kartları, ızgaralar, haritalar gösterir |
| Arama | `<wink-lookup>` | Destinasyonlar için arama çubuğu |
| Arama butonu | `<wink-search-button>` | Seyahat planlayıcıyı açar |
| Hesap butonu | `<wink-account-button>` | Giriş / kullanıcı menüsü |
| Seyahat butonu | `<wink-itinerary-button>` | Mevcut seyahati gösterir |
| Seyahat seçici | `<wink-itinerary-picker>` | Tam seyahat formu |
| Alışveriş sepeti | `<wink-shopping-cart-button>` | Sepet özeti butonu |

Her bileşenin tam özellik referansı için [Web Components](/tr/integrations/web-components) sayfasına bakabilirsiniz.
