Yapay zeka becerileri, kodlama ajanlarına Wink ile nasıl çalışacaklarını öğreten taşınabilir talimat paketleridir. Eğitim verilerine veya güncel olmayan belgelere güvenmek yerine, her beceri ajanı API sözleşmeleri ve platform dokümanları için canlı gerçek kaynak olarak [Wink MCP sunucusunu](/tr/developers/mcp) kullanmaya yönlendirir.

Beceriler, Claude Code, Codex, Cursor ve diğerleri gibi onları destekleyen herhangi bir ajan istemcisi ile çalışır.

## Mevcut Beceriler

### wink-api-builder

Wink API entegrasyonlarını uygulayın, gözden geçirin veya hata ayıklayın. Bu beceri ajanı yapılandırılmış bir iş akışıyla yönlendirir:

1. `api_search` ile doğru API işlemini arayın
2. MCP sunucusundan canlı istek/yanıt sözleşmelerini açın
3. Belgelerden kimlik doğrulama kurulumunu alın
4. Doğrulanmış sözleşmeye göre kod yazın
5. Mevcut kod ile güncel API arasındaki uyumsuzlukları işaretleyin

Faydaları:
- Herhangi bir Wink API için yeni istemci yöntemleri oluşturmak
- Sözleşme uyumsuzluklarını hata ayıklamak (örneğin, HTTP 400 dönen istek)
- Mevcut entegrasyonları canlı spesifikasyona karşı gözden geçirmek

## Kurulum

Beceri klasörünü ajanınızın beceri dizinine kopyalayın.

:::tip
Wink MCP sunucusu aynı ortamda bağlı olmalıdır. Kurulum talimatları için [MCP Server](/tr/developers/mcp) sayfasına bakınız.
:::

**Claude Code**

```bash
# depo yerel
cp -r skills/wink-api-builder .claude/skills/

# veya kullanıcı düzeyi
cp -r skills/wink-api-builder ~/.claude/skills/
```

Şu şekilde çağırın:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# depo yerel
cp -r skills/wink-api-builder .agents/skills/

# veya kullanıcı düzeyi
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Şu şekilde çağırın:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Diğer ajanlar**

Beceri klasörünü, ajanın becerileri çözdüğü herhangi bir yere yerleştirin. Giriş noktası `SKILL.md` dosyasıdır.
