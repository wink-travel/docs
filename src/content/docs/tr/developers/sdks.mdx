Resmi olarak desteklenen dil Yazılım Geliştirme Kitlerimizden birini kullanarak pazara sunma süresini hızlandırın.
Şu anda en sevdiğiniz dili desteklemiyorsak, kullanmak istediğiniz API'ye gidin ve sayfanın en üstünde her API için kullanıma sunduğumuz OpenAPI spesifikasyonunu indirin. Genel SDK depolarımızdan birini çatallayarak veya OpenAPI'nin oluşturucularından birini kullanarak kendi SDK'nızı oluşturabilirsiniz.

## Cava

Bu kütüphane Java 24 için derlenmiştir. Reaktif WebClient'ı kullanmak üzere yapılandırılmıştır ve Spring Framework ile kullanım için optimize edilmiştir. Ancak Spring bir gereklilik değildir ancak size kutudan çıktığı haliyle OAuth2 desteği sağlar.
