Resmi olarak desteklenen dil Yazılım Geliştirme Kitlerimizden birini kullanarak pazara çıkış sürenizi hızlandırın.  
Şu anda favori dilinizi desteklemiyorsak, kullanmak istediğiniz API'ye gidin ve sayfanın üst kısmında her API için sunduğumuz OpenAPI spesifikasyonunu indirin. Kendi SDK'nızı, halka açık SDK depolarımızdan birini çatallayarak veya OpenAPI'nin jeneratörlerinden birini kullanarak oluşturabilirsiniz.

## Java

Bu kütüphane Java 24 için derlenmiştir. Reaktif WebClient kullanacak şekilde yapılandırılmıştır ve Spring Framework ile kullanım için optimize edilmiştir. Ancak, Spring bir gereklilik değildir; fakat kutudan çıktığı gibi OAuth2 desteği sağlar.
