## Ödeme API

Ödeme uç noktaları, seyahat envanteri satın almak isteyen geliştiriciler içindir. Bu, kayıtlı bir Seyahat Acentesi olarak API üzerinden veya diğer tüm varlıklar için PCI uyumlu ödeme widget'ımızla birlikte API kullanılarak yapılabilir.
Ödeme API, bağlı hesaplar ve oteller için rezervasyonları, analitikleri ve fon kullanılabilirliğini takip etmek için uç noktalar sunar. Ayrıca kullanılabilir fonları banka hesaplarına çekmeyi seçebilirler.
