Bir ajans, temsil amaçlarıyla hesabınıza atayabileceğiniz normal bir Wink hesabıdır. Aşağıda, bir ajansın sizi temsil etmesini isteyebileceğiniz bazı nedenler bulunmaktadır:

- Wink hesabınızın tam / kısmi yönetimi üçüncü taraf tarafından yapılır.
- Yetenek, ajansıyla mevcut bir anlaşmaya sahiptir ve Wink'in, Wink üzerinden gelen fonların finansal yönünü yönetmesini ister.
- Yetenek ajansı tüm çevrimiçi işleri yapar ve Wink üzerindeki tüm hesapları yönetir.

## Ajans atama

Bir ajansı hesabınıza nasıl atayacağınız aşağıda açıklanmıştır.

<Steps>
 1. [Wink Studio](https://studio.wink.travel) üzerinde kullanıcı hesabınıza giriş yapın.
 2. Çalışmak istediğiniz bağlı hesabı seçin.
 3. Ana navigasyon çubuğundan `Account > Managers` seçeneğine tıklayın.
 4. `Agency` bölümünde, sizi temsil etmesini istediğiniz ajansın adını yazın.
 5. Açılır menüden ajans ücretini seçin.
 5. İsterseniz sona erme kuralları ekleyin.
 7. Devam etmek için `Set agency` butonuna tıklayın.
 </Steps>

 :::note
Ajans atamak için hesabın sahibi olmanız gerekir.
 :::

 :::tip[Örnek]
Ajans ücreti, bağlı komisyonundan türetilir.

Rezervasyon = 100$, Komisyon = %10, Ajans ücreti = %10 => 100 * 0.1 * 0.1 = 1$
 :::

## Ajans kaldırma

Bir ajansı hesabınızdan nasıl kaldıracağınız aşağıda açıklanmıştır.

<Steps>
 1. [Wink Studio](https://studio.wink.travel) üzerinde kullanıcı hesabınıza giriş yapın.
 2. Çalışmak istediğiniz bağlı hesabı seçin.
 3. Ana navigasyon çubuğundan `Account > Managers` seçeneğine tıklayın.
 4. `Agency` bölümünde, ajans açılır menüsünün sağ tarafındaki <Badge text="x" variant="danger" size="small" /> simgesine tıklayın.
 5. Devam etmek için `Set agency` butonuna tıklayın.
 </Steps>


## API

Ajansı API üzerinden yönetmek isteyen geliştiriciler, [Developers > API > Affiliate](/tr/developers/apis/#affiliate-api) sayfasına gidebilir.
