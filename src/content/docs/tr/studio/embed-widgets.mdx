Studio, rezervasyonları artırmak için kendi web sitenize ekleyebileceğiniz birkaç gömülebilir widget sunar.

Gömme widget'lara erişmek için, kenar çubuğundan `Tools > Miscellaneous` yolunu izleyin.

## Mevcut widget'lar

| Widget | Açıklama |
| -- | -- |
| **Hesap butonu** | Ortaklık sayfanıza bağlantı veren markalı bir buton |
| **Güzergah butonu** | Güzergah seçiciyi açan bir buton |
| **Güzergah formu** | Satır içi tarih ve misafir seçici formu |
| **Arama butonu** | Konaklama aramasını açan bir buton |
| **Arama butonu** | Belirli envanteri aramak için bir buton |

## Nasıl gömülür

Her widget, web sitenizin HTML'sine kopyalayıp yapıştırabileceğiniz bir kod parçası sağlar. Kod parçası, widget'ın otomatik olarak ortaklık hesabınıza bağlanması için **Client ID** ve **Customization ID** içerir.

:::note
WordPress kullanıyorsanız, daha kolay entegrasyon için bir WordPress eklentisi de sağlıyoruz.
:::
