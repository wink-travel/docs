Eğer Wink gibi bir entegratörseniz ve TripPay ile kullanmak üzere harici alacaklılarınızı eşlemek istiyorsanız, bunu iki yoldan biriyle yapabilirsiniz:

- Manuel eşleme
- [Payment API](/tr/developers/apis/#payment-api) kullanarak (*Bkz. `Account Mappings`*)

:::tip
Aynı mülklerden sadece birkaç tanesiyle ilgilenmiyorsanız, bu adımı otomatikleştirmek için API kullanmanızı öneririz.
:::

Aşağıda, harici bir alacaklıyı manuel olarak eşleme adımlarını gösteriyoruz. *Bu işlem arka planda API kullanır.*

<Steps>
1. [TripPay](https://pay.wink.travel) hesabınıza giriş yapın.
2. Entegratör hesabınızı seçin.
3. Ana navigasyon çubuğundan `Settings > Mappings` seçeneğine tıklayın.
4. `Create mapping` butonuna tıklayın. Eşleme formu açılacaktır.
5. **Local account** Eşlemek istediğiniz mevcut TripPay hesabının adını yazmaya başlayın.
6. **External identifier** Bu varlığın uzak sistemdeki harici kimliğini yapıştırın.
7. Devam etmek için `Save` butonuna tıklayın. Eşleme listenize geri yönlendirilirsiniz.
</Steps>

Eşlemeleriniz yapıldıktan sonra, TripPay artık harici kimliklerinizi kullanarak bir rezervasyonun yararlanıcıları olarak hangi hesapların alacaklandırılacağını bilir.

## API

`Payments` yönetmek isteyen geliştiriciler, [Developers > API > Payments](/tr/developers/apis/#payment-api) sayfasına gidin.
