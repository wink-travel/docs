Bazen sizinle iletişime geçmek isteriz. Bu olduğunda, sağ üst köşedeki zil simgesinin değiştiğini göreceksiniz.  
Size gönderdiğimiz mesajı okumak için:

<Steps>
1. Ana navigasyon çubuğunda `Hesap > Bildirimler`e tıklayın.  
2. Yeni ve geçmiş bildirimleri göreceksiniz.  
3. Bildirimleri okuduktan sonra `Okundu olarak işaretle` veya `Sil` seçeneklerinden birini seçin.  
</Steps>

## API

`Affiliate`leri yönetmek isteyen geliştiriciler [Geliştiriciler > API > Affiliate](/tr/developers/apis/#affiliate-api) sayfasına gidebilirler.
