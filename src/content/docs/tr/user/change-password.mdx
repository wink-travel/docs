Wink kullanıcı hesabınızın şifresini değiştirmek istiyorsanız şu adımları izleyin:

## Şifreyi güncelle

<Steps>
  1. Web sitelerimizden birinde oturum açmış olmanız gerekmektedir.
  2. Sağ üst köşedeki profil simgenize tıkladığınızda bir menü açılacaktır.
  3. Tıkla `Change password` bağlantı.
  4. Şifrenizi değiştirmek için talimatları izleyin.
  5. Tıklamak `Save`.
</Steps>


Bir dahaki sefere giriş yaptığınızda yeni şifrenize ihtiyacınız olacak.

## Parolanızı mı unuttunuz

Bazen kullanıcı şifresini sıfırlamak gerekebilir.

Kullanıcı şifrenizi sıfırlamak için aşağıdaki adımları izleyin:

<Steps>
  1. Üzerine tıklayın `Forgot password` Giriş sayfasındaki bağlantıya tıklayın.
  2. Sıfırlamak istediğiniz hesabınızın kullanıcı adını / e-posta adresini yazın.
  3. Tıklamak `Forgot password` düğme. *Şifre sıfırlama formuna yönlendirileceksiniz.*
  4. E-postanızı kontrol edin.
</Steps>

<Steps>
  1. E-postanıza gelen kodu yeni şifrenizle birlikte yeni forma yapıştırın.
  2. Yeni şifrenizi onaylayın.
  3. Tıklamak `Reset password` düğme. *Oturum açma sayfasına yönlendirileceksiniz.*
</Steps>


<figure>
  <CloudinaryImage width="350" height="467" resize="scale" quality="best" alt="Şifreyi sıfırla" src="academy/reset-password" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Şifre sıfırlama formu*</figcaption>
</figure>


Şifreniz başarıyla sıfırlandı. Wink'e yeni kimlik bilgilerinizle giriş yapabilirsiniz.
