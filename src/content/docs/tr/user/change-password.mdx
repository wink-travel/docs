Wink kullanıcı hesabınızın şifresini değiştirmek istiyorsanız, şu adımları izleyin:

## Şifreyi güncelle

<Steps>
1. Web sitelerimizden birine giriş yapmış olmalısınız.
2. Sağ üst köşedeki profil simgenize tıklayın, bir menü açılacaktır.
3. `Şifre değiştir` bağlantısına tıklayın.
4. Şifrenizi değiştirmek için talimatları izleyin.
5. `Kaydet` butonuna tıklayın.
</Steps>

Bir sonraki girişinizde yeni şifrenizi kullanmanız gerekecektir.

## Şifremi unuttum

Bazen kullanıcı şifresini sıfırlamak gerekebilir.

Kullanıcı şifrenizi sıfırlamak için adımlar şunlardır:

<Steps>
1. Giriş sayfasından `Şifremi unuttum` bağlantısına tıklayın.
2. Sıfırlamak istediğiniz hesabın kullanıcı adı / e-posta adresini yazın.
3. `Şifremi unuttum` butonuna tıklayın. *Şifre sıfırlama formuna yönlendirilirsiniz.*
4. E-postanızı kontrol edin.
</Steps>

1. E-postada aldığınız kodu yeni formdaki ilgili alana ve yeni şifrenizi yapıştırın.
2. Yeni şifrenizi onaylayın.
3. `Şifreyi sıfırla` butonuna tıklayın. *Giriş sayfasına geri yönlendirilirsiniz.*
</Steps>
