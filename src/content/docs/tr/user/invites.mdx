Bir Wink kullanıcısı, mülk, bağlı kuruluş veya seyahat acentesi hesaplarından birini sizin yönetmenize izin verebilir. Bu olduğunda, bir davet isteği içeren bir e-posta alırsınız.

import { Steps } from '@astrojs/starlight/components';

Mevcut Wink davetlerinizi görmek için şu adımları izleyin:

<Steps>
1. Web sitelerimizden birine giriş yapmış olmalısınız.
2. Sağ üst köşedeki profil simgenize tıklayın, bir menü açılacaktır.
3. `Davetler` bağlantısına tıklayın.
4. Davetlerinizi `Kabul ederek` veya `Reddederek` yanıtlayın.
</Steps>
