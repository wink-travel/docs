Bir Wink kullanıcısı, mülklerinden, iştiraklerinden veya seyahat acentesi hesaplarından birini yönetmenize izin vermeyi seçebilir. Bu gerçekleştiğinde, bir davet isteği içeren bir e-posta alacaksınız.

import { Steps } from '@astrojs/starlight/components';

Mevcut Wink davetlerinizi görmek için şu adımları izleyin:

<Steps>
  1. Web sitelerimizden birinde oturum açmış olmanız gerekmektedir.
  2. Sağ üst köşedeki profil simgenize tıkladığınızda bir menü açılacaktır.
  3. Tıkla `Invites` bağlantı.
  4. Davetlerinizle şu şekilde etkileşime geçin: `Accepting` veya `Rejecting` onlara.
</Steps>
