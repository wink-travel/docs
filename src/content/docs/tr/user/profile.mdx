Wink kullanıcı hesabınız için kullanıcı profilinizi güncellemek istiyorsanız şu adımları izleyin:

<Steps>
  1. Web sitelerimizden birinde oturum açmış olmanız gerekmektedir.
  2. Sağ üst köşedeki profil simgenize tıkladığınızda bir menü açılacaktır.
  3. Tıkla `Profile` bağlantı.
  4. Güncellemek için talimatları izleyin `e-mail`, `first` Ve `last name`.
  5. Ayrıca yeni bir tane de yükleyebilirsiniz `profile picture` Kendinin.
  6. Tıklamak `Save`.
</Steps>


:::not
Yeni profil resminiz Wink'ten tamamen çıkış yaptığınızda görüntülenecektir.
:::
