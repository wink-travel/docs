Wink kullanıcı hesabınız için kullanıcı profilinizi güncellemek istiyorsanız, şu adımları izleyin:

<Steps>
1. Web sitelerimizden birine giriş yapmış olmalısınız.
2. Sağ üst köşedeki profil simgenize tıklayın, bir menü açılacaktır.
3. `Profil` bağlantısına tıklayın.
4. `e-posta`, `ad` ve `soyad` bilgilerinizi güncellemek için talimatları izleyin.
5. Ayrıca kendinizin yeni bir `profil fotoğrafını` yükleyebilirsiniz.
6. `Kaydet` butonuna tıklayın.
</Steps>

:::note
Yeni profil fotoğrafınız, Wink'ten tamamen çıkış yaptığınızda bir sonraki girişinizde görüntülenecektir.
:::
