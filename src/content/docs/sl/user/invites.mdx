Uporabnik Wink lahko izbere, da vam dovoli upravljanje enega od svojih računov za nepremičnino, podružnico ali potovalno agencijo. Ko se to zgodi, boste prejeli e-pošto z zahtevo za povabilo.

import { Steps } from '@astrojs/starlight/components';

Če želite videti svoja trenutna povabila Wink, sledite tem korakom:

<Steps>
1. Prijaviti se morate v eno od naših spletnih strani.
2. Kliknite ikono svojega profila v zgornjem desnem kotu, da se prikaže meni.
3. Kliknite povezavo `Povabila`.
4. Ukrepajte glede svojih povabil tako, da jih bodisi `Sprejmete` ali `Zavrnete`.
</Steps>
