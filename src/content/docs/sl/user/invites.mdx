Uporabnik Wink se lahko odloči, da vam dovoli upravljanje enega od njegovih računov nepremičnin, partnerskih podjetij ali potovalnih agencij. Ko se to zgodi, boste prejeli e-pošto z zahtevo za povabilo.

import { Steps } from '@astrojs/starlight/components';

Če si želite ogledati svoja trenutna povabila Wink, sledite tem korakom:

<Steps>
  1. Prijavljeni morate biti na enem od naših spletnih mest.
  2. Kliknite ikono svojega profila v zgornjem desnem kotu in prikazal se bo meni.
  3. Kliknite na `Invites` povezava.
  4. S povabili komunicirajte tako, da `Accepting` ali `Rejecting` njih.
</Steps>
