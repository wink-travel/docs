**Wink Drupal Module** doda Wink potovalne inventarne komponente na vašo Drupal stran kot izvorne bloke. Uredniki jih lahko postavijo kjerkoli v postavitvi preko standardnega sistema blokov.

<LinkCard
  title="wink-travel/wink-drupal-module na GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Izvorna koda, navodila za nastavitev za razvijalce in navodila za prispevanje."
/>

## Zahteve

- Drupal 10 ali 11
- PHP 8.1+
- Composer

## Namestitev

<Steps>
1. Namestite preko Composerja:
   ```bash
   composer require drupal/wink
   ```
2. Omogočite modul:
   ```bash
   drush en wink -y
   ```
3. Pojdite na **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Vnesite svoj **Client ID** in **Client Secret**, izberite okolje in kliknite **Save**.
   Pojdite na [Applications](/sl/portal/settings/applications) za pridobitev svojih poverilnic.
</Steps>

## Dodajanje bloka na stran

<Steps>
1. Pojdite na **Admin → Structure → Block Layout**.
2. Kliknite **Place block** v regiji, kjer želite vsebino Wink.
3. Poiščite **Wink**, da najdete razpoložljive bloke:
   - **Wink Content** — prikaže kartico hotela, mrežo ali zemljevid
   - **Wink Lookup** — iskalna vrstica za destinacije
   - **Wink Search** — gumb za izbiro poti
   - **Wink Account** — gumb za prijavo
   - **Wink Itinerary** — gumb za status poti
   - **Wink Cart** — gumb za nakupovalno košarico
4. Za **Wink Content** nastavite **Layout Type** in **Layout ID** v nastavitvah bloka.
5. Shrani konfiguracijo bloka.
</Steps>

## Razpoložljivi bloki

| Blok | Namen |
|---|---|
| **Wink Content** | Prikazuje kartice hotelov, rangirane mreže, zemljevide in več |
| **Wink Lookup** | Iskalna vrstica za destinacije in hotele |
| **Wink Search** | Ikona gumba, ki odpre izbirnik poti |
| **Wink Account** | Gumb za prijavo / uporabniški račun |
| **Wink Itinerary** | Gumb poti, ki prikazuje trenutno potovanje |
| **Wink Cart** | Povzetek nakupovalne košarice |
