Wink komponente so brskalniku namenjeni custom elementi — CDN skripte je treba naložiti na strani odjemalca. Spodnji vzorci zajemajo tako **App Router** (Next.js 13+) kot tudi zastareli **Pages Router**.

## Namestitev

```bash
npm install @wink/elements
```

## App Router (Next.js 13+)

Ustvarite klientno komponento, ki enkrat pokliče `load()` in jo upodobite znotraj vaše root postavitve:

```tsx
// components/WinkLoader.tsx
'use client';
import { useEffect } from 'react';
import { load } from '@wink/elements';

export function WinkLoader() {
  useEffect(() => {
    load({ clientId: process.env.NEXT_PUBLIC_WINK_CLIENT_ID! });
  }, []);
  return null;
}
```

```tsx
// app/layout.tsx
import { WinkLoader } from '@/components/WinkLoader';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <WinkLoader />
        {children}
      </body>
    </html>
  );
}
```

Uporabite Wink komponente v kateri koli **Client Component**:

```tsx
// app/hotels/page.tsx
'use client';

export default function HotelsPage() {
  return (
    <main>
      <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
    </main>
  );
}
```

<Aside type="caution" title="Server Components">
`<wink-*>` oznak ni mogoče upodabljati v Server Components, ker je register custom elementov na voljo samo v brskalniku. Vedno dodajte `'use client'` v katerokoli komponento, ki upodablja Wink elemente.
</Aside>

## Pages Router

Naložite enkrat v `_app.tsx`:

```tsx
// pages/_app.tsx
import type { AppProps } from 'next/app';
import { useEffect } from 'react';
import { load } from '@wink/elements';

export default function App({ Component, pageProps }: AppProps) {
  useEffect(() => {
    load({ clientId: process.env.NEXT_PUBLIC_WINK_CLIENT_ID! });
  }, []);
  return <Component {...pageProps} />;
}
```

Nato uporabite Wink komponente na kateri koli strani:

```tsx
// pages/hotels.tsx
export default function HotelsPage() {
  return <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />;
}
```

## Okoljske spremenljivke

Dodajte v `.env.local`:

```
NEXT_PUBLIC_WINK_CLIENT_ID=your-client-id
```

Predpona `NEXT_PUBLIC_` je potrebna, da se spremenljivka izpostavi brskalniškemu paketu.
