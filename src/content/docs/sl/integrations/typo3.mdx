**Wink TYPO3 Extension** integrira Wink inventar potovanj v TYPO3 13 kot nativne vsebinske elemente. Uredniki dodajo Wink komponente na strani z uporabo standardnega urejevalnika strani TYPO3 — brez potrebe po spremembah predloge.

## Zahteve

- TYPO3 13.x
- PHP 8.2+
- Composer

## Namestitev

<Steps>
1. Namestite preko Composerja:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivirajte razširitev:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. V TYPO3 backendu pojdite na **Admin Tools → Settings → Extension Configuration → wink**.
4. Vnesite svoj **Client ID**, **Client Secret** in izberite okolje. Shrani.
   Pojdite na [Applications](/sl/portal/settings/applications) za pridobitev vaših poverilnic.
5. Dodajte statični TypoScript predlogo v vašo korensko predlogo:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Dodajanje vsebinskega elementa na stran

<Steps>
1. Odprite stran v modulu **Page**.
2. Kliknite gumb **+** na območju vsebine za dodajanje novega elementa.
3. Izberite enega od vsebinskih elementov **Wink Travel**:
   - **Wink Content** — hotelske kartice, razvrščene mreže, zemljevidi
   - **Wink Lookup** — iskalna vrstica za destinacije
   - **Wink Search** — gumb za izbiro poti
   - **Wink Account** — gumb za prijavo
   - **Wink Itinerary** — gumb za status poti
   - **Wink Cart** — gumb za nakupovalno košarico
4. Za **Wink Content** uporabite polja FlexForm za izbiro **Layout Type** in vnesite **Layout ID**.
5. Shrani in objavi stran.
</Steps>

## Na voljo vsebinski elementi

| Element | Namen |
|---|---|
| **Wink Content** | Hotelske kartice, razvrščene mreže, zemljevidi in več |
| **Wink Lookup** | Iskalna vrstica za destinacije |
| **Wink Search** | Ikona gumba, ki odpre izbirnik poti |
| **Wink Account** | Gumb za prijavo / uporabniški račun |
| **Wink Itinerary** | Gumb poti, ki prikazuje trenutno potovanje |
| **Wink Cart** | Povzetek nakupovalne košarice |
