Webflow podpira prilagojen HTML in JavaScript na treh mestih: v celotnem spletnem mestu v `<head>`, v celotnem spletnem mestu pred `</body>` in v vdelavah kode na posameznih elementih. To omogoča enostavno integracijo Wink spletnih komponent.

:::note
Prilagojena koda je na voljo v **Webflow Basic načrtu in višje**. Vdelava kode pred `</body>` na celotnem spletnem mestu zahteva plačljiv gostiteljski načrt.
:::

## Naložite Wink skripte za celotno spletno mesto

<Steps>
1. V Webflow Designerju odprite **Project Settings → Custom Code**.
2. V razdelku **Head Code** dodajte slogovno datoteko:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. V razdelku **Footer Code** dodajte skripto in nalagalnik aplikacije (zamenjajte `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Kliknite **Save Changes**.
</Steps>

:::tip
Vaš Client ID je na voljo v vašem Wink računu pod [Applications](/sl/portal/settings/applications).
:::

## Dodajte komponento na določen element

<Steps>
1. V Designerju izberite **Div** ali **Section**, kamor želite dodati komponento.
2. V desnem panelu kliknite ikono **`</>` Embed** (ali dodajte element **HTML Embed** iz Add panela).
3. Prilepite HTML komponente, na primer:
   ```html
   <wink-content-loader
     layout="RANKED"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Kliknite **Save & Close**.
</Steps>

## Prilagojena koda na posamezni strani

Če potrebujete Wink samo na določenih straneh, uporabite vdelavo kode na posamezni strani namesto za celotno spletno mesto:

<Steps>
1. Odprite **Page Settings** za ciljno stran (ikona zobnika v Pages panelu).
2. Pomaknite se do **Custom Code → Before `</body>` tag**.
3. Prilepite isto skripto in nalagalnik aplikacije kot zgoraj.
4. Shrani nastavitve strani.
</Steps>

## Na voljo komponente

| Komponenta | HTML oznaka | Namen |
|---|---|---|
| Content loader | `<wink-content-loader>` | Prikaz hotelskih kartic, mrež, zemljevidov |
| Lookup | `<wink-lookup>` | Iskalna vrstica za destinacije |
| Search button | `<wink-search-button>` | Odpri izbirnik poti |
| Account button | `<wink-account-button>` | Prijava / uporabniški meni |
| Itinerary button | `<wink-itinerary-button>` | Prikaži trenutno pot |
| Itinerary picker | `<wink-itinerary-picker>` | Celoten obrazec poti |
| Shopping cart | `<wink-shopping-cart-button>` | Gumb za povzetek košarice |

Oglejte si [Web Components](/sl/integrations/web-components) za celoten seznam atributov.
