Vgrajena funkcija **Code Injection** v Ghostu vam omogoča dodajanje skript v globalni `<head>` in `</body>` vaše publikacije. Vsi Ghost načrti podpirajo code injection, kar omogoča enostavno vdelavo Wink komponent na katerokoli stran ali objavo.

## Naložite Wink skripte za celotno spletno mesto

<Steps>
1. V Ghost Admin panelu pojdite na **Settings → Code Injection**.
2. V polje **Site Header** dodajte slogovno datoteko:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. V polje **Site Footer** dodajte skripto in nalagalnik aplikacije (zamenjajte `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Kliknite **Save**.
</Steps>

:::tip
Vaš Client ID je na voljo v vašem Wink računu pod [Applications](/sl/portal/settings/applications).
:::

## Dodajte komponento v objavo ali stran

Ghostov urejevalnik podpira **HTML kartico**, ki vam omogoča vstavljanje surovega HTML kjerkoli v vaši vsebini.

<Steps>
1. Odprite objavo ali stran v Ghost urejevalniku.
2. Vtipkajte `/`, da odprete izbirnik kartic, in izberite **HTML**.
3. Prilepite HTML komponente, na primer:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Kliknite izven kartice za predogled, nato objavite.
</Steps>

## Code injection za posamezno stran

Za komponento, ki naj se prikaže samo na eni strani, Ghost podpira tudi code injection za posamezno stran:

<Steps>
1. Odprite nastavitve objave ali strani (ikona zobnika v urejevalniku).
2. Pomaknite se do **Code Injection**.
3. Dodajte oznako komponente v polje **Footer** (globalni app-loader mora biti le v globalnem footerju).
4. Shrani.
</Steps>

## Na voljo komponente

| Komponenta | HTML oznaka | Namen |
|---|---|---|
| Content loader | `<wink-content-loader>` | Prikaz hotelskih kartic, mrež, zemljevidov |
| Lookup | `<wink-lookup>` | Iskalna vrstica za destinacije |
| Search button | `<wink-search-button>` | Odpri izbirnik poti |
| Account button | `<wink-account-button>` | Prijava / uporabniški meni |
| Itinerary button | `<wink-itinerary-button>` | Prikaži trenutno potovanje |
| Itinerary picker | `<wink-itinerary-picker>` | Celoten obrazec poti |
| Shopping cart | `<wink-shopping-cart-button>` | Gumb za povzetek košarice |

Oglejte si [Web Components](/sl/integrations/web-components) za celoten seznam atributov.
