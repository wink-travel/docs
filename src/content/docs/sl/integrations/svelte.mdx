Svelte posreduje atribute neposredno v DOM brez plasti virtualnega DOM, zaradi česar so prilagojeni elementi naravna izbira. Za SvelteKit (SSR) naložite CDN znotraj `onMount` ali uporabite preverjanje `browser`.

<LinkCard
  title="@wink/elements npm package"
  href="/integrations/npm"
  description="Namestite in naložite CDN paket z TypeScript tipi."
/>

## Namestitev

```bash
npm install @wink/elements
```

## SvelteKit — osnovna postavitev

```ts
// src/routes/+layout.ts
import { browser } from '$app/environment';
import { load as loadWink } from '@wink/elements';

export function load() {
  if (browser) {
    loadWink({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  }
}
```

Alternativno, naložite znotraj `onMount` v `+layout.svelte`:

```svelte
<!-- src/routes/+layout.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<slot />
```

## Uporaba v komponentah

Svelte posreduje vse neznane atribute neposredno v DOM, zato dodatna konfiguracija ni potrebna:

```svelte
<script>
  export let layoutId = 'YOUR_LAYOUT_ID';
</script>

<main>
  <wink-content-loader layout="HOTEL" id={layoutId} />
  <wink-lookup />
</main>
```

## Svelte 5 (runes sintaksa)

```svelte
<script lang="ts">
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  let { layoutId }: { layoutId: string } = $props();

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id={layoutId} />
```

## Okoljske spremenljivke

SvelteKit izpostavi samo spremenljivke, ki se začnejo z `PUBLIC_`, v brskalnik:

```
# .env
PUBLIC_WINK_CLIENT_ID=your-client-id
```

<Aside type="caution" title="SSR upodabljanje">
Nikoli ne upodabljajte `<wink-*>` elementov med strežniškim upodabljanjem — prilagojeni elementi so registrirani v brskalnikovem registru `customElements`, ki na strežniku ne obstaja. Zavijte jih v `{#if browser}` ali pokličite `load()` samo znotraj `onMount`.
</Aside>

```svelte
<script>
  import { browser } from '$app/environment';
</script>

{#if browser}
  <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
{/if}
```

## Čisti Svelte (Vite, brez SvelteKit)

```svelte
<!-- src/App.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.VITE_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
```
