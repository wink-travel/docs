Ta stran navaja vse dogodke webhook, na katere se lahko naročite na Wink. Za navodila za nastavitev si oglejte [Vodnik za integracijo webhook](/sl/guides/webhook-integration) ali [Stran za upravljanje webhookov](/sl/my-account/webhooks).

## Kako se naročiti

1. Pojdite na **Applications > Webhooks** v vašem portalu.
2. Ustvarite webhook in izberite dogodke, ki jih želite prejemati.
3. Wink pošlje HTTP POST na vašo končno točko vsakič, ko se dogodek zgodi.

Za celotne korake nastavitve in najboljše prakse si oglejte [Integracija webhook](/sl/guides/webhook-integration).

## Dogodki računa

Dogodki, povezani z življenjskim ciklom in nastavitvami vašega TripPay računa.

| Dogodek | Ključ | Opis |
|---|---|---|
| Račun ustvarjen | `account.create` | Ustvarjen je bil nov račun. |
| Račun v pregledu | `account.under_review` | Račun je v postopku pregleda. |
| Račun posodobljen | `account.updated` | Podatki računa so se spremenili. |
| Status računa posodobljen | `account.status.updated` | Status računa se je spremenil (npr. odobren, začasno onemogočen). |
| Naslov računa posodobljen | `account.address.updated` | Naslov računa je bil spremenjen. |
| Podrobnosti računa posodobljene | `account.details.updated` | Polja podrobnosti računa so se spremenila. |
| Informacije o računu posodobljene | `account.information.update` | Splošne informacije o računu so bile posodobljene. |
| Upravljavci računa posodobljeni | `account.managers.update` | Seznam upravljavcev računa se je spremenil. |
| Profilna slika posodobljena | `account.profile_picture.updated` | Profilna slika računa je bila spremenjena. |
| Agent posodobljen | `account.agent.updated` | Podatki o agentu na računu so bili spremenjeni. |
| URL ime posodobljeno | `account.url-name.update` | Prilagojeni URL računa se je spremenil. |
| Edinstvena ID posodobljena | `account.unique-id.update` | Edinstveni identifikator računa je bil spremenjen. |

### Naloge računa

| Dogodek | Ključ | Opis |
|---|---|---|
| Naloga ustvarjena | `account.task.create` | Novi nalogi so bili dodani računu. |
| Naloga posodobljena | `account.task.update` | Naloga računa je bila spremenjena. |
| Naloga odstranjena | `account.task.remove` | Naloga računa je bila odstranjena. |
| Naloga dokončana | `account.task.complete` | Naloga računa je bila dokončana. |
| Naloga preverjanja davčne številke dokončana | `account.task.tax_id.complete` | Naloga preverjanja davčne številke je bila dokončana. |

### Prevzemnik računa

| Dogodek | Ključ | Opis |
|---|---|---|
| Prevzemnik ustvarjen | `account.acquirer.created` | Nov prevzemnik je bil dodan računu. |
| Prevzemnik posodobljen | `account.acquirer.updated` | Prevzemnik računa je bil spremenjen. |
| Prevzemnik odstranjen | `account.acquirer.removed` | Prevzemnik je bil odstranjen z računa. |

| Pogodba ni uspela | `booking.contract.failed` | Plačilo pogodbe o rezervaciji ni uspelo. |
| Pogodba preklicana | `booking.contract.cancelled` | Pogodba o rezervaciji je bila preklicana. |
| Delni povračilo | `booking.contract.refund.partial` | Na pogodbi je bilo izdano delno povračilo. |
| Polno povračilo | `booking.contract.refund.full` | Na pogodbi je bilo izdano polno povračilo. |

## Dogodki upravljavskih entitet

Dogodki za upravljavske entitete (objekti, podružnice, podjetja) in njihovo upravljanje.

### Naročnina

| Dogodek | Ključ | Opis |
|---|---|---|
| Naročnina zagotovljena | `managing_entity.subscription.provisioned` | Nova naročnina je bila zagotovljena. |
| Naročnina spremenjena | `managing_entity.subscription.changed` | Načrt naročnine je bil spremenjen. |
| Račun za naročnino spremenjen | `managing_entity.subscription.invoice.changed` | Račun za naročnino je bil posodobljen. |

### Status in uprava

| Dogodek | Ključ | Opis |
|---|---|---|
| Status posodobljen | `managing_entity.status.updated` | Status entitete se je spremenil. |
| Entiteta izbrisana | `managing_entity.deleted` | Upravljavska entiteta je bila izbrisana. |
| Agent posodobljen | `managing_entity.agent.updated` | Agent entitete je bil spremenjen. |
| Obvestilo agenta po e-pošti | `managing_entity.agent.email.notification` | Agent je prejel obvestilo po e-pošti. |
| Obvestilo upravitelja po e-pošti | `managing_entity.manager.email.notification` | Upravitelj je prejel obvestilo po e-pošti. |

### Življenjski cikel upravitelja

| Dogodek | Ključ | Opis |

| Upravitelj odstranjen | `managing_entity.manager.removed` | Upravitelj je bil odstranjen. |
