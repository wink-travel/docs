Pospešite čas do trga z uporabo enega izmed naših uradno podprtih jezikovnih Software Development Kitov.
Če trenutno ne podpiramo vašega najljubšega jezika, pojdite na API, ki ga želite uporabljati, in prenesite OpenAPI specifikacijo, ki jo za vsak API omogočamo na vrhu strani. Svoje SDK lahko ustvarite tako, da forkate enega izmed naših javnih SDK repozitorijev ali pa uporabite enega izmed OpenAPI generatorjev.

## Java

Ta knjižnica je prevedena za Java 24. Konfigurirana je za uporabo reaktivnega WebClienta in je optimizirana za uporabo s Spring Frameworkom. Vendar Spring ni zahteva, vendar vam omogoča podporo za OAuth2 takoj po namestitvi.
