## API za plačila

Končne točke plačila so namenjene razvijalcem, ki želijo kupiti potovalni inventar. To lahko storijo prek API-ja kot registrirani potovalni agenti ali z uporabo našega API-ja v povezavi z našim plačilnim pripomočkom, ki je skladen s PCI, za vse druge subjekte.
API za plačila razkriva končne točke za podružnice in hotele za sledenje rezervacijam, analitiki in razpoložljivosti sredstev. Prav tako se lahko odločijo za dvig razpoložljivih sredstev na svoj bančni račun.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Več informacij
</LinkButton>
