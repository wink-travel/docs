## Payment API

Plačilni konektorji so za razvijalce, ki želijo kupiti turistični inventar. To je mogoče preko API-ja kot registrirani potovalni agent ali z uporabo našega API-ja v povezavi z našim PCI skladnim plačilnim vtičnikom za vse druge entitete.
Payment API omogoča partnerjem in hotelom sledenje rezervacijam, analitiko in razpoložljivosti sredstev. Prav tako lahko izberejo dvig razpoložljivih sredstev na svoj bančni račun.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Izvedite več
</LinkButton>
