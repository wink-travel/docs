Egy Wink felhasználó eldöntheti, hogy engedélyezi-e számodra egy ingatlan-, partner- vagy utazási ügynöki fiókjának kezelését. Amikor ez megtörténik, e-mailben kapsz egy meghívókérelmet.

import { Steps } from '@astrojs/starlight/components';

A jelenlegi Wink meghívóid megtekintéséhez kövesd az alábbi lépéseket:

<Steps>
 1. Jelentkezz be a [my.wink.travel](https://my.wink.travel) oldalra.
 2. Kattints a bal oldali menüben az `Invites` menüpontra.
 3. Meg fog jelenni az összes aktuális meghívó listája.
 4. Ez a szakasz az összes jelentkezésedet felsorolja.
 5. Válaszd az `Accept` vagy `Reject` lehetőséget a meghívóval kapcsolatban.
 6. Ha elfogadtad a meghívót, a fiók megjelenik az `Accounts` szakaszban.
</Steps>
