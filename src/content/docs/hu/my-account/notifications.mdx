Előfordul, hogy kapcsolatba szeretnénk lépni veled. Ilyenkor a jobb felső sarokban lévő csengő ikon megváltozik.
Az elküldött üzenetek elolvasásához:

<Steps>
1. Kattints a fő navigációs sávban az `Fiók > Értesítések` menüpontra.
2. Megjelennek az új és a korábbi értesítések.
3. Olvasás után válaszd a `Megjelölés olvasottként` vagy a `Törlés` lehetőséget.
</Steps>

## API

Azok a fejlesztők, akik az `Affiliate`-eket szeretnék kezelni, látogassanak el a [Fejlesztők > API > Affiliate](/hu/developers/apis/#affiliate-api) oldalra.
