A mentett keresések azt teszik, amit a nevük sugall. Utazási készletek keresése közben lehetőséged van a keresési lekérdezés elmentésére későbbi felhasználás céljából.

## A keresési lekérdezés mentése

Amikor elkezdi keresni a készletet (lásd[Keresés](/studio/search)), lehetőséged van a keresési szűrők finomhangolására.
A lekérdezési űrlap alján található a lehetőség, hogy`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Keresés mentése" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Megjeleníti a keresési lekérdezés mentésére mutató linket*</figcaption>
</figure>


Erre a linkre kattintva nevet adhatsz a keresésnek (*pl. A varázslatos szállodák könyvtára*) valami, amire később emlékezni fogsz.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Adj nevet a keresésednek" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Adj nevet a keresésednek*</figcaption>
</figure>


Adj neki egy nevet, és kattints a`Save search`gombra. Ezután navigálhat a következőhöz:`Inventory > Saves Searches`a fő navigációs sávon az újonnan létrehozott rekord megtekintéséhez.

## Listaműveletek

A mentett keresés segítségével az utazási készleteket rács vagy térkép formájában jelenítheti meg a felhasználóknak, és a keresési eredményeket a kívánt módon jelenítheti meg.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Mentett keresési műveletek" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Mentett keresési lekérdezés szűrők alkalmazása nélkül*</figcaption>
</figure>


Minden mentett keresési lekérdezéshez tartozik egy művelet, amelyet végrehajthat rajta:

* Keresési lekérdezés konvertálása[térkép](/studio/maps).
* Keresési lekérdezés konvertálása[rács](/studio/grids).
* Távolítsa el a mentett keresést.

## API

A mentett kereséseket kezelni kívánó fejlesztők a következő címen érhetik el:[Fejlesztők > API > Böngészés](/developers/apis/#browse-api).
