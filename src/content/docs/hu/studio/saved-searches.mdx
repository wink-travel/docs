A mentett keresések azt teszik, amit a nevük sugall. Amikor utazási készletet keresel, lehetőséged van elmenteni a keresési lekérdezésed későbbi használatra.

## A keresési lekérdezés mentése

Amikor elkezdesz készletet keresni (lásd [Keresés](/hu/studio/search) ), finomhangolhatod a keresési szűrőket.
A lekérdezési űrlap alján megtalálod a `Keresés mentése` opciót.

Erre a linkre kattintva lehetőséged nyílik elnevezni ezt a keresést (*pl. A varázslatos szállodák könyvtára*), amit később könnyen fel tudsz idézni.

Adj neki nevet, majd kattints a `Keresés mentése` gombra. Ezután a fő navigációs sávból az `Inventory > Mentett keresések` menüpont alatt megtekintheted az újonnan létrehozott rekordot.

## Lista műveletek

A mentett keresésed segítségével megjelenítheted az utazási készletet felhasználóid számára rács vagy térkép formájában, és úgy jelenítheted meg a keresési eredményeket, ahogy szeretnéd.

Minden mentett keresési lekérdezéshez tartoznak végrehajtható műveletek:

- Átalakíthatod a keresési lekérdezést [térképpé](/hu/studio/maps).
- Átalakíthatod a keresési lekérdezést [rácssá](/hu/studio/grids).
- Törölheted a mentett keresést.

## API

A fejlesztők, akik szeretnék kezelni a mentett kereséseket, a [Fejlesztők > API > Böngészés](/hu/developers/apis/#browse-api) menüpontban találják meg a szükséges eszközöket.
