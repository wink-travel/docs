## Fizetési API

A Fizetési végpontok fejlesztőknek szólnak, akik utazási készletet szeretnének vásárolni. Ez megtehető regisztrált utazási ügynökként az API-n keresztül, vagy az API-t használva PCI-kompatibilis fizetési widgetünkkel minden más entitás számára.
A Fizetési API végpontokat biztosít partnereknek és szállodáknak foglalások, analitika és pénzeszközök elérhetőségének nyomon követésére. Emellett dönthetnek úgy is, hogy elérhető pénzeszközeiket bankszámlájukra utalják.

Tudjon meg többet
