## Fizetési API

A fizetési végpontok azoknak a fejlesztőknek szólnak, akik utazási felszerelést szeretnének vásárolni. Ez megtehető az API-n keresztül regisztrált utazási ügynökként, vagy az API-nkat használva, PCI-kompatibilis fizetési widgetünkkel együtt, minden más entitás számára.

A fizetési API végpontokat tesz elérhetővé a partnerek és szállodák számára a foglalások, az elemzések és az összegek elérhetőségének nyomon követéséhez. Dönthetnek úgy is, hogy a rendelkezésre álló összeget bankszámlájukra utalják.

  Tudj meg többet
</LinkButton>
