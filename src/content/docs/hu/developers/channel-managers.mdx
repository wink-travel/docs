A Wink integrálódik vezető csatornakezelőkkel, ingatlankezelő rendszerekkel (PMS) és központi foglalási rendszerekkel (CRS), hogy az árait és elérhetőségét minden értékesítési csatornáján szinkronban tartsa.

## Hogyan működik

A csatornakezelő integrációk kétirányú adatáramlást tesznek lehetővé az ingatlankezelő rendszer és a Wink között:

1. **Árküldés** — A csatornakezelője árfrissítéseket küld a Winknek. Amikor frissíti az árait a PMS-ben, ezek a változások automatikusan megjelennek a Winkben.
2. **Elérhetőség szinkronizálás** — A csatornakezelője elérhetőségi frissítéseket küld a Winknek. Amikor szobákat foglalnak vagy szabadítanak fel a PMS-ben, a Wink készlete valós időben frissül.
3. **Foglalási értesítések** — Amikor foglalás történik a Winkben, a csatornakezelője értesítést kap, így a foglalás megjelenik a PMS-ben.

## Támogatott csatornakezelők

A Wink jelenleg a következő rendszerekkel integrálódik:

| Csatornakezelő | Leírás |
|---|---|
| **Channex** | Globális disztribúciós és csatornakezelő platform. |
| **TravelClick** | Szállodai megoldások és csatornakezelés (Amadeus). |
| **Omnibees** | Disztribúció és bevételkezelés szállodák számára. |
| **D-Edge** | Vendéglátóipari technológia és központi foglalási megoldások. |
| **Allotz** | Készletallokáció és csatornakezelés. |
| **Ezee / Yanolja** | Felhő alapú PMS és csatornakezelés, az ázsiai piacra fókuszálva. |
| **RateGain** | Árintelligencia és disztribúciós technológia. |
| **RateTiger** | Valós idejű ár- és elérhetőség disztribúció. |
| **CloudBeds** | Felhő alapú PMS, csatornakezelő és foglalási motor. |
| **SiteMinder** | Vezető szállodai csatornakezelő és disztribúciós platform. |
| **Exely** | Ár- és elérhetőség szinkronizálás szállodák számára. |

:::note
Ha az Ön csatornakezelője nincs a listán, lépjen kapcsolatba a Wink csapatával. Új integrációk hozzáadhatók az OTA (Open Travel Alliance) protokoll szabványai alapján, amelyeket a Wink támogat.
:::

## Fejlesztőknek

Ha csatornakezelő vagy PMS integrációt épít a Winkkel, használja a Channel Manager API végpontjait:

- **Árfrissítések** — Küldje el az árváltozásokat a Winknek.
- **Elérhetőség frissítések** — Küldje el az elérhetőségi változásokat a Winknek.
- **Foglalások lekérése** — Húzza le a foglalásokat a Winkből a rendszerébe.

A Channel Manager API az OTA protokoll szabványait (SOAP/XML) követi a meglévő vendéglátóipari rendszerekkel való kompatibilitás érdekében.

## Webhook értesítések

Feliratkozhat csatornakezelő webhook eseményekre, hogy valós idejű értesítéseket kapjon:

- `channel-manager.update.rate` — Árfrissítés érkezett.
- `channel-manager.update.availability` — Elérhetőség frissítés érkezett.
- `channel-manager.update` — Általános csatornakezelő frissítés.

Részletekért lásd a [Webhook Események Katalógusát](/hu/developers/webhook-events#channel-manager-events).
