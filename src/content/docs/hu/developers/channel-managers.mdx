A Wink integrálódik vezető channel managerekkel, ingatlankezelő rendszerekkel (PMS) és központi foglalási rendszerekkel (CRS), hogy árai és elérhetőségei szinkronban legyenek az összes értékesítési csatornán.

## Hogyan működik

A channel manager integrációk kétirányú adatáramlást tesznek lehetővé az ingatlankezelő rendszer és a Wink között:

1. **Árküldés** — A channel manager elküldi az árfrissítéseket a Winknek. Amikor frissíti árait a PMS-ben, ezek a változások automatikusan megjelennek a Wink felületén.
2. **Elérhetőség szinkronizálás** — A channel manager elküldi az elérhetőségi frissítéseket a Winknek. Amikor szobákat foglalnak vagy szabadítanak fel a PMS-ben, a Wink készlete valós időben frissül.
3. **Foglalási értesítések** — Amikor foglalás történik a Wink-en, a channel managert értesítik, így a foglalás megjelenik a PMS-ben.

## Támogatott channel managerek

A Wink jelenleg az alábbi rendszerekkel integrálódik:

| Channel Manager | Leírás |
|---|---|
| **Channex** | Globális disztribúciós és channel management platform. |
| **TravelClick** | Szállodai megoldások és channel management (Amadeus). |
| **Omnibees** | Disztribúció és bevételkezelés szállodák számára. |
| **D-Edge** | Vendéglátóipari technológia és központi foglalási megoldások. |
| **Allotz** | Készletallokáció és channel management. |
| **Ezee / Yanolja** | Felhő alapú PMS és channel management, az ázsiai piacra fókuszálva. |
| **RateGain** | Árintelligencia és disztribúciós technológia. |
| **RateTiger** | Valós idejű ár- és elérhetőség disztribúció. |
| **CloudBeds** | Felhő alapú PMS, channel manager és foglalási motor. |
| **SiteMinder** | Vezető szállodai channel manager és disztribúciós platform. |
| **Exely** | Ár- és elérhetőség szinkronizálás szállodák számára. |

:::note
Ha az Ön channel managere nincs a listán, vegye fel a kapcsolatot a Wink csapatával. Új integrációk hozzáadhatók az OTA (Open Travel Alliance) protokoll szabványai alapján, amelyeket a Wink támogat.
:::

## Fejlesztőknek

Ha channel manager vagy PMS integrációt épít a Winkkel, használja a Channel Manager API végpontokat:

- **Árfrissítések** — Küldje el az árváltozásokat a Winknek.
- **Elérhetőség frissítések** — Küldje el az elérhetőségi változásokat a Winknek.
- **Foglalások lekérése** — Húzza le a foglalásokat a Winkből a rendszerébe.

A Channel Manager API az OTA protokoll szabványait (SOAP/XML) követi, hogy kompatibilis legyen a meglévő vendéglátóipari rendszerekkel.

## Webhook értesítések

Feliratkozhat channel manager webhook eseményekre, hogy valós idejű értesítéseket kapjon:

- `channel-manager.update.rate` — Árfrissítés érkezett.
- `channel-manager.update.availability` — Elérhetőségi frissítés érkezett.
- `channel-manager.update` — Általános channel manager frissítés.

Részletekért lásd a [Webhook Események Katalógusát](/hu/developers/webhook-events#channel-manager-events).
