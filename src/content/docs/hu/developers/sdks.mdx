Gyorsítsa fel a piacra jutási időt az egyik hivatalosan támogatott nyelvi Software Development Kit használatával.
Ha jelenleg nem támogatjuk a kedvenc nyelvét, lépjen arra az API-ra, amelyet használni szeretne, és töltse le az OpenAPI specifikációt, amelyet minden API oldal tetején elérhetővé teszünk. Saját SDK-t generálhat úgy, hogy vagy fork-olja valamelyik nyilvános SDK-tárházunkat, vagy az OpenAPI generátorainak egyikét használja.

## Java

Ez a könyvtár Java 24-re lett fordítva. Reaktív WebClient használatára van konfigurálva, és a Spring Framework-kel való használatra van optimalizálva. A Spring azonban nem kötelező, de alapból OAuth2 támogatást nyújt.
