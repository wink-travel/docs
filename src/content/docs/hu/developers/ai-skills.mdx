Az AI készségek hordozható utasításcsomagok, amelyek megtanítják a kódoló ügynököket, hogyan dolgozzanak a Wink-kel. Ahelyett, hogy képzési adatokra vagy elavult dokumentációra támaszkodnának, minden készség az ügynököt a [Wink MCP szerver](/hu/developers/mcp) használatára irányítja, mint az API szerződések és a platform dokumentációjának élő igazságforrására.

A készségek bármely olyan ügynök klienssel működnek, amely támogatja őket — Claude Code, Codex, Cursor és mások.

## Elérhető készségek

### wink-api-builder

Wink API integrációk megvalósítása, áttekintése vagy hibakeresése. A készség egy strukturált munkafolyamaton vezeti végig az ügynököt:

1. Megfelelő API művelet keresése `api_search` segítségével
2. Élő kérés/válasz szerződések megnyitása az MCP szerverről
3. Hitelesítési beállítások lekérése a dokumentációból
4. Kód írása a hitelesített szerződés alapján
5. Bármilyen eltérés jelzése a meglévő kód és az aktuális API között

Hasznos:
- Új kliens metódusok építése bármely Wink API-hoz
- Szerződés eltérések hibakeresése (pl. HTTP 400 választ adó kérés)
- Meglévő integrációk áttekintése az élő specifikáció alapján

## Telepítés

Másold a készség mappát az ügynököd készség könyvtárába.

:::tip
A Wink MCP szervernek ugyanabban a környezetben kell csatlakoznia. Lásd a [MCP Server](/hu/developers/mcp) oldalt a beállítási útmutatóért.
:::

**Claude Code**

```bash
# repo-local
cp -r skills/wink-api-builder .claude/skills/

# vagy felhasználói szinten
cp -r skills/wink-api-builder ~/.claude/skills/
```

Indítás:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repo-local
cp -r skills/wink-api-builder .agents/skills/

# vagy felhasználói szinten
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Indítás:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Egyéb ügynökök**

Helyezd a készség mappát oda, ahonnan az ügynököd a készségeket betölti. A belépési pont a `SKILL.md`.
