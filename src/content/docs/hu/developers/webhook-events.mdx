Ez az oldal felsorolja az összes webhook eseményt, amelyre feliratkozhatsz a Wink-en. A beállítási útmutatóért lásd a [Webhook integrációs útmutatót](/hu/guides/webhook-integration) vagy a [Webhook kezelőoldalt](/hu/my-account/webhooks).

## Hogyan iratkozz fel

1. Navigálj a portálodban az **Alkalmazások > Webhookok** menüpontra.
2. Hozz létre egy webhookot, és válaszd ki azokat az eseményeket, amelyeket fogadni szeretnél.
3. A Wink minden esemény bekövetkeztekor HTTP POST kérést küld a végpontodra.

A teljes beállítási lépésekért és legjobb gyakorlatokért lásd a [Webhook integráció](/hu/guides/webhook-integration) oldalt.

## Fiók események

Események a TripPay fiók életciklusával és beállításaival kapcsolatban.

| Esemény | Kulcs | Leírás |
|---|---|---|
| Fiók létrehozva | `account.create` | Új fiók jött létre. |
| Fiók felülvizsgálat alatt | `account.under_review` | A fiók felülvizsgálatra vár. |
| Fiók frissítve | `account.updated` | A fiók adatai megváltoztak. |
| Fiók státusz frissítve | `account.status.updated` | A fiók státusza megváltozott (pl. jóváhagyva, felfüggesztve). |
| Fiók cím frissítve | `account.address.updated` | A fiók címe módosult. |
| Fiók részletek frissítve | `account.details.updated` | A fiók részletei megváltoztak. |
| Fiók információ frissítve | `account.information.update` | Általános fiókinformáció frissítve. |
| Fiók kezelők frissítve | `account.managers.update` | A fiók kezelőinek listája megváltozott. |
| Profilkép frissítve | `account.profile_picture.updated` | A fiók profilképe megváltozott. |
| Ügynök frissítve | `account.agent.updated` | Az ügynök adatai módosultak a fiókban. |
| URL név frissítve | `account.url-name.update` | A fiók egyedi URL-je megváltozott. |
| Egyedi azonosító frissítve | `account.unique-id.update` | A fiók egyedi azonosítója módosult. |

### Fiók feladatok

| Esemény | Kulcs | Leírás |
|---|---|---|
| Feladat létrehozva | `account.task.create` | Új feladat került hozzáadásra a fiókhoz. |
| Feladat frissítve | `account.task.update` | Egy fiókhoz tartozó feladat módosult. |
| Feladat eltávolítva | `account.task.remove` | Egy fiókhoz tartozó feladat törölve lett. |
| Feladat teljesítve | `account.task.complete` | Egy fiókhoz tartozó feladat befejeződött. |
| Adószám feladat teljesítve | `account.task.tax_id.complete` | Az adószám ellenőrzési feladat befejeződött. |

### Fiók beszedő

| Esemény | Kulcs | Leírás |
|---|---|---|
| Beszedő létrehozva | `account.acquirer.created` | Új beszedő került hozzáadásra a fiókhoz. |
| Beszedő frissítve | `account.acquirer.updated` | Egy fiókhoz tartozó beszedő módosult. |
| Beszedő eltávolítva | `account.acquirer.removed` | Egy beszedő törölve lett a fiókból. |

## Foglalási szerződés események

Események a foglalási szerződés életciklusának nyomon követésére a TripPay-en keresztül.

| Esemény | Kulcs | Leírás |
|---|---|---|
| Inicializálás kérése | `booking.contract.initialize.request` | Foglalási szerződés inicializálására irányuló kérés érkezett. |
| Inicializálás befejezve | `booking.contract.initialize.complete` | A foglalási szerződés inicializálása befejeződött. |
| Szerződés létrehozva | `booking.contract.created` | Új foglalási szerződés jött létre. |
| Szerződés feldolgozás alatt | `booking.contract.processing` | A foglalási szerződés fizetése feldolgozás alatt áll. |
| Szerződés sikertelen | `booking.contract.failed` | A foglalási szerződés fizetése sikertelen volt. |
| Szerződés törölve | `booking.contract.cancelled` | A foglalási szerződés törölve lett. |
| Részleges visszatérítés | `booking.contract.refund.partial` | Részleges visszatérítés történt a szerződésen. |
| Teljes visszatérítés | `booking.contract.refund.full` | Teljes visszatérítés történt a szerződésen. |
| Visszatérítés kérése | `booking.contract.refund.request` | Visszatérítési kérelem lett benyújtva. |
| Visszatérítési hiba | `booking.contract.refund.error` | Hiba történt a visszatérítés feldolgozása közben. |
| Kifizetés létrehozva | `booking.contract.payout.create` | Kifizetés jött létre a szerződéshez. |
| Kifizetés befejezve | `booking.contract.payout.complete` | A kifizetés sikeresen befejeződött. |

| Előfizetési számla módosítva | `managing_entity.subscription.invoice.changed` | Az előfizetési számla frissült. |
