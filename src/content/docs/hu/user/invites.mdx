Egy Wink felhasználó választhatja, hogy átengedi neked az egyik ingatlan-, partner- vagy utazási irodai fiókját. Amikor ez megtörténik, kapni fogsz egy meghívót tartalmazó e-mailt.

import { Steps } from '@astrojs/starlight/components';

A jelenlegi Wink-meghívóid megtekintéséhez kövesd az alábbi lépéseket:

<Steps>
  1. Be kell jelentkezned valamelyik weboldalunkra.
  2. Kattints a profilod ikonjára a jobb felső sarokban, és megjelenik egy menü.
  3. Kattintson a `Invites` link.
  4. Interakcióba léphetsz a meghívóiddal az alábbiak szerint: `Accepting` vagy `Rejecting` őket.
</Steps>
