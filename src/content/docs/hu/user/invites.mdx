Egy Wink felhasználó dönthet úgy, hogy engedélyezi számodra egy ingatlan-, partner- vagy utazási ügynöki fiókjának kezelését. Amikor ez megtörténik, kapsz egy e-mailt egy meghívó kéréssel.

import { Steps } from '@astrojs/starlight/components';

A jelenlegi Wink meghívóid megtekintéséhez kövesd az alábbi lépéseket:

<Steps>
1. Be kell jelentkezned valamelyik weboldalunkra.
2. Kattints a profil ikonodra a jobb felső sarokban, ekkor megjelenik egy menü.
3. Kattints a `Meghívók` hivatkozásra.
4. Kezeld a meghívóidat azzal, hogy vagy `Elfogadod`, vagy `Elutasítod` őket.
</Steps>
