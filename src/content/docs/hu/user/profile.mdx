Ha frissíteni szeretné a Wink felhasználói fiókjához tartozó profilját, kövesse az alábbi lépéseket:

<Steps>
1. Be kell jelentkeznie valamelyik weboldalunkra.
2. Kattintson a profilikonjára a jobb felső sarokban, ekkor megjelenik egy menü.
3. Kattintson a `Profil` hivatkozásra.
4. Kövesse az utasításokat az `e-mail`, `keresztnév` és `vezetéknév` frissítéséhez.
5. Feltölthet egy új `profilképet` is magáról.
6. Kattintson a `Mentés` gombra.
</Steps>

:::note
Az új profilképe a következő teljes kijelentkezés után fog megjelenni a Winkben.
:::
