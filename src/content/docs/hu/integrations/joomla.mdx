A **Wink Joomla Extension** egy csomag, amely telepít egy rendszerbővítményt (betölti a CDN szkripteket), egy tartalombővítményt (feldolgozza a shortcode címkéket a cikkekben), és egy komponenst (admin beállítási oldal).

## Követelmények

- Joomla 4 vagy 5
- PHP 8.1+

## Telepítés

<Steps>
1. Töltse le a legfrissebb `pkg_wink-x.x.x.zip` fájlt a [GitHub kiadások oldaláról](https://github.com/wink-travel/wink-joomla-extension/releases).
2. A Joomla admin felületen lépjen a **Bővítmények → Kezelés → Telepítés** menüpontra.
3. Töltse fel a zip fájlt, majd kattintson a **Feltöltés és telepítés** gombra.
4. Mindkét bővítmény automatikusan engedélyezve lesz. Lépjen a **Komponensek → Wink Beállítások** oldalra.
5. Adja meg a **Client ID** és **Client Secret** értékeket, majd kattintson a **Mentés** gombra.
   A hitelesítő adatok megszerzéséhez látogasson el az [Alkalmazások](/hu/portal/settings/applications) oldalra.
</Steps>

## Komponensek beágyazása cikkekbe

Használjon shortcode-stílusú címkéket a cikk tartalmában bárhol:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

A tartalombővítmény ezeket a címkéket a megfelelő `<wink-*>` HTML elemekre cseréli a megjelenítéskor. A rendszerbővítmény automatikusan betölti a CDN szkripteket és beszúrja a `<wink-app-loader>` elemet az oldal láblécébe.

## Elérhető shortcode-ok

| Shortcode | Kimenet | Leírás |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Szállodai kártyák, rácsok, térképek |
| `{wink-lookup}` | `<wink-lookup>` | Úti cél keresőmező |
| `{wink-search}` | `<wink-search-button>` | Útvonalválasztó gomb |
| `{wink-account}` | `<wink-account-button>` | Bejelentkezés / felhasználói fiók |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Útvonal állapot gomb |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Kosár gomb |
