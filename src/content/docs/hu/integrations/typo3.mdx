A **Wink TYPO3 Extension** integrálja a Wink utazási készletet a TYPO3 13 rendszerbe natív tartalmi elemekként. A szerkesztők a szabványos TYPO3 oldalszerkesztő segítségével adhatják hozzá a Wink komponenseket az oldalakhoz — sablonmódosítás nem szükséges.

## Követelmények

- TYPO3 13.x
- PHP 8.2+
- Composer

## Telepítés

<Steps>
1. Telepítés Composerrel:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. A bővítmény aktiválása:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. A TYPO3 backendben lépjen az **Admin Tools → Settings → Extension Configuration → wink** menüpontra.
4. Adja meg a **Client ID**-t, a **Client Secret**-et, és válassza ki a környezetet. Mentse el.
   A hitelesítő adatok lekéréséhez látogasson el a [Applications](/hu/portal/settings/applications) oldalra.
5. Adja hozzá a statikus TypoScript sablont a gyökérsablonhoz:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Tartalmi elem hozzáadása egy oldalhoz

<Steps>
1. Nyisson meg egy oldalt a **Page** modulban.
2. Kattintson a **+** gombra a tartalomterületen egy új elem hozzáadásához.
3. Válasszon egyet a **Wink Travel** tartalmi elemek közül:
   - **Wink Content** — szállodakártyák, rangsorolt rácsok, térképek
   - **Wink Lookup** — úti cél keresősáv
   - **Wink Search** — útiterv választó gomb
   - **Wink Account** — bejelentkezés gomb
   - **Wink Itinerary** — útiterv állapot gomb
   - **Wink Cart** — bevásárlókosár gomb
4. A **Wink Content** esetén használja a FlexForm mezőket a **Layout Type** kiválasztásához és a **Layout ID** megadásához.
5. Mentse el és tegye közzé az oldalt.
</Steps>

## Elérhető tartalmi elemek

| Elem | Cél |
|---|---|
| **Wink Content** | Szállodakártyák, rangsorolt rácsok, térképek és egyéb |
| **Wink Lookup** | Úti cél keresősáv |
| **Wink Search** | Ikongomb, amely megnyitja az útiterv választót |
| **Wink Account** | Bejelentkezés / felhasználói fiók gomb |
| **Wink Itinerary** | Útiterv gomb, amely a jelenlegi utazást mutatja |
| **Wink Cart** | Bevásárlókosár összefoglaló gomb |
