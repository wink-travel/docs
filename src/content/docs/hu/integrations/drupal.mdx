A **Wink Drupal Modul** natív blokkoként adja hozzá a Wink utazási készlet komponenseit Drupal oldalához. A szerkesztők a szabványos blokk rendszer segítségével bárhol elhelyezhetik őket az elrendezésben.

<LinkCard

## Követelmények

- Drupal 10 vagy 11
- PHP 8.1+
- Composer

## Telepítés

<Steps>
1. Telepítés Composerrel:
   ```bash
   composer require drupal/wink
   ```
2. A modul engedélyezése:
   ```bash
   drush en wink -y
   ```
3. Menjen a **Admin → Konfiguráció → Webszolgáltatások → Wink beállítások** oldalra (`/admin/config/wink`).
4. Adja meg a **Client ID** és **Client Secret** értékeket, válassza ki a környezetet, majd kattintson a **Mentés** gombra.
   A hitelesítő adatok megszerzéséhez látogasson el a [Alkalmazások](/hu/portal/settings/applications) oldalra.
</Steps>

## Blokk hozzáadása egy oldalhoz

<Steps>
1. Menjen a **Admin → Struktúra → Blokk elrendezés** menüpontra.
2. Kattintson a **Blokk elhelyezése** gombra abban a régióban, ahol a Wink tartalmat szeretné megjeleníteni.
3. Keressen rá a **Wink** kifejezésre az elérhető blokkok megtalálásához:
   - **Wink Content** — hotelkártya, rács vagy térkép megjelenítése
   - **Wink Lookup** — úti cél keresősáv
   - **Wink Search** — útiterv választó gomb
   - **Wink Account** — bejelentkezés gomb
   - **Wink Itinerary** — útiterv állapot gomb
   - **Wink Cart** — bevásárlókosár gomb
4. A **Wink Content** blokk esetén állítsa be az **Elrendezés típusa** és az **Elrendezés azonosítója** értékeket a blokk beállításaiban.
5. Mentse el a blokk konfigurációját.
</Steps>

## Elérhető blokkok

| Blokk | Célja |
|---|---|
| **Wink Content** | Hotelkártyák, rangsorolt rácsok, térképek és egyéb megjelenítése |
| **Wink Lookup** | Úti célok és hotelek keresősávja |
| **Wink Search** | Ikongomb, amely megnyitja az útiterv választót |
| **Wink Account** | Bejelentkezés / felhasználói fiók gomb |
| **Wink Itinerary** | Az aktuális utazást mutató útiterv gomb |
| **Wink Cart** | Bevásárlókosár összefoglaló gomb |
