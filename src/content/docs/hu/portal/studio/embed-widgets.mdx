A Studio több beágyazható widgetet kínál, amelyeket hozzáadhatsz a saját weboldaladhoz a foglalások ösztönzésére.

A beágyazható widgetek eléréséhez navigálj a `Tools > Miscellaneous` menüpontra az oldalsávból.

## Elérhető widgetek

| Widget | Leírás |
| -- | -- |
| **Account button** | Egy márkázott gomb, amely az affiliate oldaladra mutat |
| **Itinerary button** | Egy gomb, amely megnyitja az útitervválasztót |
| **Itinerary form** | Egy beágyazott dátum- és vendégválasztó űrlap |
| **Search button** | Egy gomb, amely megnyitja az ingatlankeresőt |
| **Lookup button** | Egy gomb, amellyel konkrét készletet kereshetsz |

## Hogyan ágyazd be

Minden widget egy kódrészletet biztosít, amelyet kimásolhatsz és beilleszthetsz a weboldalad HTML-jébe. A részlet tartalmazza a **Client ID** és a **Customization ID** értékeket, így a widget automatikusan az affiliate fiókodhoz kapcsolódik.

:::note
Ha WordPress-t használsz, egy WordPress bővítményt is biztosítunk a könnyebb integráció érdekében.
:::
