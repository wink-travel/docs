# Beállítások

Amikor először kezded el használni a Social-t, megkérdezzük, hogyan szeretnéd, hogy létrehozzuk a tartalmaidat. Ezek az opciók mindig elérhetőek számodra, így később könnyedén módosíthatod őket.

A beállításaid kezeléséhez navigálj a `Preferences` menüpontra az oldalsávból.

A preferenciák szekció tartalmazza az alapvető tartalomgenerálási beállításaidat:

1. **Előnyben részesített testreszabás** Válaszd ki, melyik testreszabást szeretnéd használni a tartalom generálásakor. A testreszabások szabályozzák a témaszíneket, az alapértelmezett útitervet, a pénznemet, a nyelvet és még sok mást. Testreszabásokat a fiókod alatt hozhatsz létre a [my.wink.travel](https://my.wink.travel) oldalon.
2. **Automatikus jóváhagyás** Ha engedélyezve van, a generált bejegyzések automatikusan közzé lesznek téve, jóváhagyásod nélkül. Ezt akkor kapcsold be, ha már megbízol a számunkra előállított tartalom minőségében.
3. **Jóváhagyó neve** Az a személy vagy csapat neve, aki megkapja a jóváhagyási kérelmeket tartalmazó e-maileket.
4. **Jóváhagyó e-mail címe** Az az e-mail cím, ahová a közelgő bejegyzések jóváhagyási kérelmei érkeznek.

## Tartalomkészítők

A Social számos tartalomkészítőt kínál, amelyek közül választhatsz. Minden készítő egy adott módra fókuszál, hogy egyedi tartalmat hozzon létre az ingatlanod számára AI segítségével. Minden tartalomkészítő rendelkezik egy **Súly** attribútummal — a magasabb súly növeli annak esélyét, hogy az adott készítő kerüljön kiválasztásra a bejegyzések generálásakor.

Íme az összes elérhető tartalomkészítő:

| Készítő | Leírás |
| -- | -- |
| **Úti cél** | A helyszíned alapján készít hasznos szöveget a környékről |
| **Szobatípus** | Foglalható bejegyzést hoz létre az egyik szobatípusodról AI által generált szöveggel és a szobatípus képeivel |
| **Tevékenység** | Foglalható bejegyzést készít az egyik tevékenységedről |
| **Kiegészítő szolgáltatás** | Foglalható bejegyzést készít az egyik kiegészítő szolgáltatásodról |
| **Látnivaló** | Foglalható bejegyzést készít az egyik látnivalódról |
| **Helyszín** | Foglalható bejegyzést készít az egyik helyszínedről |
| **Tárgyalóterem** | Foglalható bejegyzést készít az egyik tárgyalótermedről |
| **Étterem** | Foglalható bejegyzést készít az egyik éttermedről |
| **Spa** | Foglalható bejegyzést készít az egyik spa szolgáltatásodról |
| **Ingatlan** | Foglalható bejegyzést készít a legjobb árú szobádról AI által generált szöveggel és az ingatlan képeivel |
| **Felhasználói vélemény** | Összegzi a vendégértékeléseket a weben, és érzelmi alapú bejegyzést készít, amely kiemeli, mit szeretnek a vendégek az ingatlanodban |
| **Vicces** | Könnyed, szórakoztató bejegyzéseket készít, hogy a hírfolyamod érdekes maradjon |

:::tip[Pro tipp]
Győződj meg róla, hogy rendelkezel a kiválasztott tartalomkészítők által igényelt készlettípussal. Például, ha az "Étterem" opciót választod, legalább egy éttermet kell szerepeltetned az ingatlanodban.
:::

A hangnem határozza meg azt a nyelvi stílust, amelyet az AI használ a tartalom generálásakor. Válaszd ki azokat a hangnemeket, amelyek illenek a márkád személyiségéhez. Ahogy a tartalomkészítőknél, itt is minden hangnemhez tartozik egy **Súly**, amely szabályozza, milyen gyakran használódik.

Az elérhető hangnemek:

| Hangnem | Stílus |
| -- | -- |
| **Semleges** | Nyugodt, tekintélyes, tényszerű. Nincs túlzás, nincs viccelődés — csak megbízható, jól strukturált tudás |

| **Luxus** | Kifinomult, elegáns. Olyan szavak, mint az „exkluzív” és „egyedi”, amelyek különlegessé teszik az olvasót |
| **Gasztronómiai** | Érzéki, étvágygerjesztő leírások. Ételpornó nyelvezet, amely éhesvé teszi az olvasót |
| **Kalandvágyó** | Merész, bátor, felfedező energia. Élénk érzékszervi leírások, amelyek mindent epikussá tesznek |
| **Családbarát** | Egészséges, gyerekbarát izgalom. Meleg szülői energia a biztonságos és szórakoztató élményekről |
| **Romantikus** | Költői, vágyakozó, szívdobogtató. Naplementék, lágy nyelvezet és szerelem |
| **Humoros** | Szarkasztikus, gyors egy soros poénok, szellemes szóviccek. Olyan, mint egy stand-up előadás |
| **Öko** | Komoly és igazságos. Ünnepli a zöld alternatívákat és a fenntartható választásokat |
| **Helyi bennfentes** | Nagyon specifikus helyi utalások, figyelmeztet a turistacsapdákra, helyi szleng használata |
| **Befogadó** | Barátságos, ünneplő és megerősítő. Mindenkit láthatónak és értékeltnek éreztet |
| **Misztikus** | Éteri, spirituális, kozmikus igazságok. Gyengéd meghívások a belső szemlélődésre |

## Bejegyzéstípusok

A bejegyzéstípusok határozzák meg a generált tartalom célját és szerkezetét. Minden bejegyzéstípushoz tartozik egy **Súly**, amely szabályozza, milyen gyakran használódik.

| Bejegyzéstípus | Leírás |
| -- | -- |
| **Tranzakciós** | Egyértelmű, tömör, konverzióközpontú. Foglalásokat és közvetlen lépéseket ösztönöz tényekkel, árakkal és következő lépésekkel |
| **Ismertség növelő** | Márkaépítő tartalom, amely bemutatja az ingatlanodat és növeli a láthatóságot a potenciális vendégek körében |
| **Érzelmi** | Érzelmileg rezonáló tartalom, amely pozitív asszociációkat teremt és érzelmi kapcsolatot épít a közönségeddel |
