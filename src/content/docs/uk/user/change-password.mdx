Якщо ви хочете змінити пароль для свого облікового запису користувача Wink, виконайте ці кроки:

## Оновлення пароля

<Steps>
1. Ви повинні бути увійшли на один із наших вебсайтів.
2. Натисніть на іконку профілю у верхньому правому куті, і з’явиться меню.
3. Натисніть на посилання `Змінити пароль`.
4. Дотримуйтесь інструкцій для зміни пароля.
5. Натисніть `Зберегти`.
</Steps>

Наступного разу, коли ви увійдете, вам знадобиться новий пароль.

## Забули пароль

Іноді необхідно скинути пароль користувача.

Ось кроки для скидання вашого пароля:

<Steps>
1. Натисніть на посилання `Забули пароль` на сторінці входу.
2. Введіть ім’я користувача / електронну пошту для облікового запису, який хочете скинути.
3. Натисніть кнопку `Забули пароль`. *Вас буде перенаправлено на форму скидання пароля.*
4. Перевірте свою електронну пошту.
</Steps>

1. Вставте код, який ви отримали на електронну пошту, у нову форму разом із новим паролем.
2. Підтвердіть свій новий пароль.
3. Натисніть кнопку `Скинути пароль`. *Вас буде перенаправлено назад на сторінку входу.*
</Steps>
