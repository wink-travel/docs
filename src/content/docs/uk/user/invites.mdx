Користувач Wink може дозволити вам керувати одним із їхніх облікових записів нерухомості, афілійованих осіб або туристичних агентів. Коли це відбувається, ви отримаєте електронного листа з запитом на запрошення.

import { Steps } from '@astrojs/starlight/components';

Щоб переглянути ваші поточні запрошення Wink, виконайте ці кроки:

<Steps>
1. Ви повинні увійти в один із наших вебсайтів.
2. Натисніть на іконку профілю у верхньому правому куті, і з’явиться меню.
3. Натисніть на посилання `Invites`.
4. Взаємодійте зі своїми запрошеннями, або `Приймаючи`, або `Відхиляючи` їх.
</Steps>
