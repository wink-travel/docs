Користувач Wink може дозволити вам керувати одним зі своїх облікових записів нерухомості, партнера або туристичного агента. Коли це станеться, ви отримаєте електронного листа із запитом на запрошення.

import { Steps } from '@astrojs/starlight/components';

Щоб переглянути свої поточні запрошення Wink, виконайте такі дії:

<Steps>
  1. Ви повинні бути авторизовані на одному з наших вебсайтів.
  2. Натисніть на значок свого профілю у верхньому правому куті, і з’явиться меню.
  3. Натисніть на `Invites` посилання.
  4. Взаємодійте зі своїми запрошеннями одним із `Accepting` або `Rejecting` їх.
</Steps>
