Додатки надають вам доступ до всіх функцій Wink як розробнику. Ви можете отримувати доступ до тих самих даних і навіть розширювати власні функції Wink.
Ви можете обрати використання OAuth2 сервера Wink для автентифікації вашого туристичного сайту так само, як ви використовуєте Google або Facebook.
Таким чином ви можете розширити функції Wink на ваш власний вебсайт або додаток.

:::note
Ваш перший додаток вже створений, коли ви [створюєте новий акаунт](/uk/my-account/create-account). Ми використовуємо цей додаток для деяких сценаріїв бронювання.

Ви не можете повторно використовувати цей додаток. Будь ласка, створюйте нові додатки за потребою.
:::

:::note
Додаток, створений на Wink, можна використовувати для всіх API кінцевих точок.
:::

<Steps>
 1. Виберіть акаунт, з яким хочете працювати, на сторінці [Accounts](https://my.wink.travel).
 2. Натисніть `Actions` внизу картки акаунта та оберіть `Manage account`.
 3. Натисніть вкладку `Applications` на наступній сторінці.
 4. У цьому розділі буде перелік усіх ваших додатків.
 5. Оберіть редагування існуючого або створення нового додатку.
 </Steps>

## Створення додатку

Оскільки ви не маєте доступу до `secret key` додатку, який ми створили для вас, вам потрібно створити власний.

:::caution
Вам потрібно створити афілійований акаунт на Wink або акаунт на TripPay перед створенням додатку, оскільки ви будете прив’язувати додаток до конкретного акаунта.
:::

Ось кроки для створення додатку:

<Steps>
1. На сторінці додатків натисніть кнопку `Create new application`.
2. ***Назва*** Дайте вашому додатку назву. *наприклад Cool App*
3. **Назва організації** Прив’яжіть додаток до одного з ваших існуючих акаунтів. *наприклад Cool Account*
4. **Redirect URIs** Потрібен принаймні один redirect URI. Він використовується для перевірки домену перенаправлення після успішної автентифікації. *наприклад https://www.cool-site.com*
5. Натисніть кнопку `Save`, щоб продовжити.
</Steps>

Вас перенаправлять назад до списку додатків. Ви побачите свій новий додаток у списку і матимете доступ до секретного ключа додатку лише один раз. Обов’язково скопіюйте його та збережіть у безпечному місці.

## Оновлення додатку

Вам потрібно оновити додаток, якщо змінюється один із redirect URI або ви хочете додати чи видалити URI.

Щоб оновити існуючий додаток, виконайте наступне:

<Steps>
1. Натисніть посилання `Actions` для додатку, який хочете оновити.
2. Натисніть кнопку `Update` під Actions.
3. Внесіть зміни до додатку.
4. Натисніть кнопку `Save`, щоб продовжити.
</Steps>

Вас перенаправлять назад до списку додатків.

## Видалення додатку

:::caution
Будь ласка, не видаляйте додаток, який ми створили автоматично, щоб усі функції працювали для вашого акаунта.
:::

Якщо додаток вам більше не потрібен, ви можете його видалити.

Щоб видалити додаток, виконайте наступне:

<Steps>
1. Натисніть посилання `Actions` для додатку, який хочете видалити.
2. Натисніть кнопку `Remove` під Actions.
3. Натисніть кнопку `OK`, щоб підтвердити видалення.
</Steps>

Вас перенаправлять назад до списку додатків.
