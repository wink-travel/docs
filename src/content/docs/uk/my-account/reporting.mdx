Wink надає систему звітності, яка дозволяє генерувати та експортувати дані по всьому вашому акаунту. Звіти доступні у форматі CSV для зручного імпорту у таблиці, аналітичні інструменти або ваші власні системи.

## Типи звітів

Ви можете генерувати звіти за такими категоріями даних:

| Тип звіту | Що включає |
|---|---|
| **Об’єкт** | Деталі об’єкта, статус оголошення, підсумок інвентарю. |
| **Користувач** | Акаунти користувачів, ролі, підсумок активності. |
| **Бронювання** | Історія бронювань, статуси, дохід, дані гостей. |
| **Акаунт** | Інформація про акаунт, баланси, історія транзакцій. |

## Як згенерувати звіт

1. Перейдіть до розділу звітності у вашій панелі керування акаунтом.
2. Виберіть **тип звіту**, який хочете згенерувати.
3. Налаштуйте будь-які **фільтри** для звуження даних (наприклад, діапазон дат, статус, об’єкт).
4. Оберіть **поля**, які хочете включити до експорту. Кожен тип звіту пропонує набір доступних полів для вибору.
5. Натисніть **Експорт** для завантаження звіту у форматі CSV.

## Користувацькі поля

Звіти підтримують вибір полів за вашим бажанням. Ви обираєте, які колонки включати до експорту, щоб отримати саме ті дані, які вам потрібні — нічого зайвого.

## Варіанти використання

- **Звірка доходів** — Експортуйте звіти бронювань з фінансовими деталями для звірки з вашою бухгалтерською системою.
- **Аналіз ефективності** — Експортуйте звіти об’єктів, щоб порівняти ефективність оголошень у вашому портфелі.
- **Інформація про гостей** — Експортуйте дані бронювань, щоб зрозуміти демографію мандрівників та патерни бронювань.
- **Аудит акаунтів** — Експортуйте дані акаунтів для відповідності або внутрішнього перегляду.
