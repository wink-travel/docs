Для всіх, хто використовує WordPress для свого бізнесу або блогу, ми настійно рекомендуємо використовувати [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
Цей плагін дає вам легкий доступ до ваших карток Wink, сіток, карт тощо, і робить це так просто, як перетягування контенту у ваші пости та сторінки за допомогою улюбленого конструктора сторінок або наших власних шорткодів.

##  Встановлення

Ось як встановити `Wink Affiliate WordPress Plugin` на ваш сайт WordPress.

<Steps>
1. Коли ви увійшли як адміністратор і перебуваєте на Панелі керування.
2. Натисніть `Плагіни > Додати новий плагін`.
3. Знайдіть `Wink Affiliate WordPress Plugin`.
4. Натисніть `Встановити`, щоб завантажити та встановити плагін на ваш сайт.
5. Натисніть `Активувати`, щоб активувати плагін на вашому сайті.
6. Після активації вас перенаправить на сторінку `Wink Settings`. *Ви також можете перейти туди, вибравши `Зовнішній вигляд > Налаштувати > Wink Settings`.*
7. Вас попросять ввести `Client ID` та `Client Secret`. Перейдіть до [Applications](/uk/developers/applications), щоб дізнатися, як отримати свої облікові дані.
8. Введіть свої облікові дані та натисніть `Опублікувати`. Тепер ви підключені до свого облікового запису Wink.
</Steps>

## Конструктори сторінок

Ми підтримуємо найпопулярніші конструктори сторінок для WordPress, що робить використання Wink простим для кожного.

### Gutenberg

Gutenberg — це вбудований редактор WordPress.

### Avada

[Avada](https://avada.com) відомий своїм великим вибором красивих шаблонів та потужним конструктором сторінок з розширеним набором функцій.

### Elementor

[Elementor](https://elementor.com) — це [наразі] найпопулярніший конструктор сторінок з безкоштовною та професійною версіями.

### WPBakery

[WPBakery](https://wpbakery.com) був першим просунутим конструктором сторінок на ринку.

## Шорткоди

### [winklookup]

Працює з рейтингованою сіткою контенту. Введіть місце, яке хочете відвідати, і відобразиться доступний інвентар.

### [winksearch]

Проста кнопка для відкриття форми маршруту.

### [winkaccount]

Кнопка облікового запису для автентифікації. Після автентифікації вона перетворюється на випадаюче меню з опціями облікового запису.

### [winkitinerary]

Те саме, що кнопка пошуку, але містить інформацію про маршрут як текст кнопки.

### [winkcontent]

Дозволяє вбудувати вибраний інвентар Wink безпосередньо на сторінку або пост. Цей шорткод також очікує ID макета.

Макет

```
[winkcontent layoutid=""]
```
