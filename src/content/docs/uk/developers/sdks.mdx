Прискорте час виведення на ринок, використовуючи один з наших офіційно підтримуваних мовних комплектів розробки програмного забезпечення.
Якщо ми наразі не підтримуємо вашу улюблену мову, перейдіть до потрібного API та завантажте специфікацію OpenAPI, яку ми надаємо для кожного API у верхній частині сторінки. Ви можете створити власний SDK, створивши форк одного з наших публічних репозиторіїв SDK або скориставшись одним із генераторів OpenAPI.

## Ява

Цю бібліотеку було скомпільовано для Java 24. Вона налаштована на використання реактивного WebClient та оптимізована для використання з Spring Framework. Однак Spring не є обов'язковим, але вона забезпечує підтримку OAuth2 "з коробки".
