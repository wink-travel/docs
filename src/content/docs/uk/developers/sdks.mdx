Прискорте вихід на ринок, використовуючи один із наших офіційно підтримуваних мовних Software Development Kits.
Якщо ми наразі не підтримуємо вашу улюблену мову, перейдіть до API, який хочете використовувати, і завантажте OpenAPI-специфікацію, яку ми надаємо для кожного API у верхній частині сторінки. Ви можете згенерувати власний SDK, або форкнувши один із наших публічних репозиторіїв SDK, або використовуючи один із генераторів OpenAPI.

## Java

Ця бібліотека скомпільована для Java 24. Вона налаштована для використання реактивного WebClient і оптимізована для роботи зі Spring Framework. Однак Spring не є обов’язковим, але він надає підтримку OAuth2 "з коробки".
