## API оплати

Кінцеві точки оплати призначені для розробників, які хочуть придбати туристичний інвентар. Це можна зробити через API як зареєстрований туристичний агент або використовуючи наш API разом із нашим платіжним віджетом, сумісним зі стандартом PCI, для всіх інших організацій.
API оплати надає кінцеві точки для партнерів та готелів для відстеження бронювань, аналітики та наявності коштів. Вони також можуть вибрати зняття доступних коштів на свій банківський рахунок.
