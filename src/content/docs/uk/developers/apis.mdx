Якщо ви ще не створили свій обліковий запис Wink, [почніть тут](/uk/user/register).

## Payment API

Платіжні кінцеві точки призначені для розробників, які хочуть придбати туристичний інвентар. Це можна зробити через API як зареєстрований Travel Agent або використовуючи наш API разом із нашим PCI-сумісним платіжним віджетом для всіх інших сутностей.
Payment API надає кінцеві точки для афілійованих осіб і готелів для відстеження бронювань, аналітики та доступності коштів. Вони також можуть обрати виведення доступних коштів на свій банківський рахунок.
