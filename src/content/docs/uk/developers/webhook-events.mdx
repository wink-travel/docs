На цій сторінці наведено всі події webhook, на які ви можете підписатися у Wink. Для інструкцій з налаштування дивіться [Посібник з інтеграції Webhook](/uk/guides/webhook-integration) або [сторінку керування Webhook](/uk/my-account/webhooks).

## Як підписатися

1. Перейдіть у **Applications > Webhooks** у вашому порталі.
2. Створіть webhook і виберіть події, які хочете отримувати.
3. Wink надсилатиме HTTP POST на вашу кінцеву точку щоразу, коли відбуватиметься подія.

Для повних кроків налаштування та найкращих практик дивіться [Інтеграція Webhook](/uk/guides/webhook-integration).

## Події облікового запису

Події, пов’язані з життєвим циклом та налаштуваннями вашого облікового запису TripPay.

| Подія | Ключ | Опис |
|---|---|---|
| Обліковий запис створено | `account.create` | Створено новий обліковий запис. |
| Обліковий запис на перевірці | `account.under_review` | Обліковий запис очікує на перевірку. |
| Обліковий запис оновлено | `account.updated` | Дані облікового запису змінено. |
| Статус облікового запису оновлено | `account.status.updated` | Статус облікового запису змінено (наприклад, затверджено, призупинено). |
| Адреса облікового запису оновлена | `account.address.updated` | Адресу облікового запису змінено. |
| Деталі облікового запису оновлено | `account.details.updated` | Поля деталей облікового запису змінено. |
| Інформація облікового запису оновлена | `account.information.update` | Загальна інформація облікового запису оновлена. |
| Менеджери облікового запису оновлені | `account.managers.update` | Список менеджерів облікового запису змінено. |
| Фото профілю оновлено | `account.profile_picture.updated` | Фото профілю облікового запису змінено. |
| Агент оновлений | `account.agent.updated` | Інформацію про агента облікового запису змінено. |
| URL-ім’я оновлено | `account.url-name.update` | Змінено унікальне URL-ім’я облікового запису. |
| Унікальний ідентифікатор оновлено | `account.unique-id.update` | Унікальний ідентифікатор облікового запису змінено. |

### Завдання облікового запису

| Подія | Ключ | Опис |
|---|---|---|
| Завдання створено | `account.task.create` | До облікового запису додано нове завдання. |
| Завдання оновлено | `account.task.update` | Завдання облікового запису змінено. |
| Завдання видалено | `account.task.remove` | Завдання облікового запису видалено. |
| Завдання виконано | `account.task.complete` | Завдання облікового запису виконано. |
| Завдання з перевірки податкового ідентифікатора виконано | `account.task.tax_id.complete` | Завдання з перевірки податкового ідентифікатора виконано. |

### Еквайринг облікового запису

| Подія | Ключ | Опис |
|---|---|---|
| Еквайринг створено | `account.acquirer.created` | До облікового запису додано нового еквайра. |
