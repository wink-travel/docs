Wink інтегрується з провідними channel manager, системами управління нерухомістю (PMS) та центральними системами бронювання (CRS), щоб підтримувати синхронізацію ваших тарифів і доступності на всіх каналах продажу.

## Як це працює

Інтеграції channel manager забезпечують двонаправлений потік даних між вашою системою управління нерухомістю та Wink:

1. **Передача тарифів** — Ваш channel manager надсилає оновлення тарифів до Wink. Коли ви оновлюєте ціни у вашому PMS, ці зміни автоматично відображаються у Wink.
2. **Синхронізація доступності** — Ваш channel manager надсилає оновлення доступності до Wink. Коли номери бронюються або звільняються у вашому PMS, інвентар Wink оновлюється в режимі реального часу.
3. **Сповіщення про бронювання** — Коли бронювання здійснюється через Wink, ваш channel manager отримує сповіщення, щоб резервування з’явилося у вашому PMS.

## Підтримувані channel manager

Наразі Wink інтегрується з такими системами:

| Channel Manager | Опис |
|---|---|
| **Channex** | Глобальна платформа розподілу та управління каналами. |
| **TravelClick** | Готельні рішення та управління каналами (Amadeus). |
| **Omnibees** | Розподіл та управління доходами для готелів. |
| **D-Edge** | Технології гостинності та рішення для центрального бронювання. |
| **Allotz** | Розподіл інвентарю та управління каналами. |
| **Ezee / Yanolja** | Хмарний PMS та управління каналами, орієнтовані на азіатський ринок. |
| **RateGain** | Технології інтелектуального управління тарифами та розподілу. |
| **RateTiger** | Розподіл тарифів і доступності в режимі реального часу. |
| **CloudBeds** | Хмарний PMS, channel manager та система бронювання. |
| **SiteMinder** | Провідний готельний channel manager та платформа розподілу. |
| **Exely** | Синхронізація тарифів і доступності для готелів. |

:::note
Якщо ваш channel manager не вказаний у списку, зверніться до команди Wink. Нові інтеграції можуть бути додані на основі стандартів протоколу OTA (Open Travel Alliance), які підтримує Wink.
:::

## Для розробників

Якщо ви створюєте інтеграцію channel manager або PMS з Wink, використовуйте кінцеві точки Channel Manager API:

- **Оновлення тарифів** — Надсилайте зміни тарифів до Wink.
- **Оновлення доступності** — Надсилайте зміни доступності до Wink.
- **Отримання бронювань** — Завантажуйте бронювання з Wink у вашу систему.

Channel Manager API відповідає стандартам протоколу OTA (SOAP/XML) для сумісності з існуючими системами гостинності.

## Сповіщення через webhook

Ви можете підписатися на події webhook channel manager, щоб отримувати сповіщення в режимі реального часу:

- `channel-manager.update.rate` — Отримано оновлення тарифів.
- `channel-manager.update.availability` — Отримано оновлення доступності.
- `channel-manager.update` — Загальне оновлення channel manager.

Деталі дивіться у [Каталозі подій Webhook](/uk/developers/webhook-events#channel-manager-events).
