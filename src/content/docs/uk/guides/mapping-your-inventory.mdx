Коли ви додаєте інвентар до вашої власності в Extranet — кімнати, ресторани, спа, конференц-зали, активності та інше — цей інвентар стає доступним у всій екосистемі Wink. Цей посібник пояснює, як ці елементи пов’язані.

## Потік інвентарю

```
Extranet (ви створюєте) → Booking Engine (мандрівники бачать) → Studio (афілійовані продають)
```

1. **Ви створюєте інвентар** в Extranet — типи кімнат, послуги та враження.
2. **Мандрівники переглядають** цей інвентар у системі бронювання через посилання для спільного доступу, картки або прямі пошуки.
3. **Афілійовані особи знаходять** ваш інвентар у Studio, створюють картки та сітки і продають його своїй аудиторії.

## Типи кімнат

Типи кімнат — це основа вашого інвентарю. Кожен тип кімнати потребує:

- Назви та опису.
- Конфігурації ліжок.
- Максимальної місткості (дорослі та діти).
- Зручностей, специфічних для кімнати.
- Фотографій.
- Принаймні одного тарифного плану з основними тарифами.

Без тарифів тип кімнати існує, але не буде доступним для бронювання.

## Послуги та враження

Окрім кімнат, ви можете додати:

| Тип інвентарю | Що це | Приклад |
| -- | -- | -- |
| **Ресторан** | Харчування на місці | "Rooftop Bar & Grill" |
| **Спа** | Послуги оздоровлення | "Serenity Spa - 60хв масаж" |
| **Конференц-зал** | Простір для подій і конференцій | "Boardroom A - 20 осіб" |
| **Активність** | Розваги на території або поруч | "Екскурсія з снорклінгом" |
| **Пам’ятка** | Відомі місця поблизу | "Національний музей - 500м" |
| **Місце** | Цікаві локації | "Місцевий ринок" |
| **Додатково** | Додаткові послуги, що купуються з кімнатою | "Трансфер з аеропорту" |

Кожен тип послуг має власний розділ в Extranet. Афілійовані можуть продавати будь-який із них окремо — мандрівник може знайти вашу власність через спа, а не через пошук кімнат.

## Як афілійовані використовують ваш інвентар

У Studio афілійовані можуть:

- **Шукати** вашу власність або окремі елементи інвентарю.
- **Створювати картки** — окремі бронювані картки для кімнати, спа-послуги або активності.
- **Створювати сітки** — колекції карток, що відображаються разом.
- **Створювати карти** — маркери на основі локації, що показують бронюваний інвентар.
- **Ділитися посиланнями** — прямі URL на вашу власність або конкретні елементи.

Чим повніший і різноманітніший ваш інвентар, тим більше способів афілійовані можуть просувати вашу власність.

## Поради

- **Будьте ретельними** — вкажіть усе, що пропонує ваша власність. Кожен елемент — потенційна точка входу для бронювання.
- **Використовуйте якісні фото** — кожен елемент інвентарю повинен мати принаймні одне якісне зображення.
- **Оновлюйте описи** — оновлюйте описи при зміні пропозицій або сезонно.
- **Цінуйте все** — інвентар без цін можна переглядати, але не можна забронювати.
