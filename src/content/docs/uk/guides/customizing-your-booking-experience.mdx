Налаштування контролюють, як ваш інвентар відображається мандрівникам — кольори теми, валюта за замовчуванням, мова, маршрут і багато іншого. Вони застосовуються до посилань для спільного доступу, вбудованих карток, сіток, карт і движка бронювання.

## Що таке налаштування?

Налаштування — це профіль конфігурації, який визначає:

- **Відображувана валюта** — Валюта, в якій показуються ціни.
- **Відображувана мова** — Мова за замовчуванням для контенту.
- **Маршрут за замовчуванням** — Попередньо заповнені дати заїзду/виїзду та кількість гостей.
- **Тема** — Кольори та брендинг, застосовані до вбудованих компонентів.
- **Конфігурація** — Як представлений інвентар (макет, сортування).

Ви можете створювати кілька налаштувань для різних ринків або випадків використання.

## Створення налаштування

<Steps>
1. Перейдіть на https://studio.wink.travel (або у свій акаунт на https://my.wink.travel).
2. Відкрийте розділ `Customization` у бічному меню.
3. Натисніть `Create a new customization`.
4. Дайте описову назву (наприклад, "Europe - EUR", "Japan Market", "Default").
5. Встановіть валюту відображення, мову та маршрут за замовчуванням.
6. Налаштуйте кольори теми відповідно до вашого бренду.
7. Натисніть `Save`.
</Steps>

## Де використовуються налаштування

Після створення налаштування використовуються по всій платформі:

| Функція | Як застосовується налаштування |
| -- | -- |
| **Посилання для спільного доступу** | Посилання використовує валюту, мову та маршрут налаштування за замовчуванням |
| **Вбудовані картки** | Картки відображаються з кольорами теми та налаштуваннями відображення |
| **Сітки та карти** | Сітки інвентарю та маркери на карті відповідають налаштуванню |
| **Движок бронювання** | Весь процес бронювання використовує налаштування |
| **Соціальні пости** | AI-генеровані пости можуть використовувати конкретне налаштування для відображення цін |

## Кілька налаштувань

Поширена практика — створювати одне налаштування для кожного цільового ринку:

- **"US Market"** — USD, англійська, стандартне перебування 2 ночі.
- **"Europe"** — EUR, англійська, стандартне перебування 3 ночі.
- **"Japan"** — JPY, японська, стандартне перебування 1 ніч.

Коли ви створюєте посилання для спільного доступу або вбудовуєте картку, ви обираєте, яке налаштування використовувати. Це означає, що одна й та сама власність може бути представлена по-різному залежно від аудиторії.

## Перезапис через URL

Ви також можете перезаписати налаштування за замовчуванням за допомогою параметрів URL у посиланнях для спільного доступу:

- `l=es` — Перезаписати мову на іспанську.
- `c=EUR` — Перезаписати валюту на EUR.
- `sd=2024-08-24&n=2` — Перезаписати маршрут на конкретну дату та кількість ночей.

Дивіться [The Book Now Button](/uk/guides/the-book-now-button) для повного списку параметрів URL.
