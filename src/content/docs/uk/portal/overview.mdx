[Portal](https://app.wink.travel) — це єдиний веб-додаток, де тепер розміщені всі розділи екосистеми Wink. Раніше кожна функціональна зона (Extranet, Studio, Travel Agent, Payment, Social, Link Manager) мала власну URL-адресу. Починаючи з цього релізу, вони всі є розділами одного Порталу під `app.wink.travel`, організованими як підшляхи.

## Розділи

- **`/extranet`** — Функції для готелів, мереж, брендів і управляючих компаній.
- **`/studio`** — Функції для афілійованих партнерів для просування туристичного інвентарю та отримання комісій.
- **`/travel-agent`** — Функції для зареєстрованих турагентів для пошуку, бронювання та заробітку.
- **`/payment`** — Функції для управління виплатами та зняттям коштів.
- **`/social`** — Автоматизований Social для публікацій у соціальних мережах на основі ШІ.
- **`/link-manager`** — Link Manager для створення сторінок WinkLinks.
- **`/corporate`** — Корпоративний розділ (скоро).
- **`/settings`** — Налаштування для кожного акаунта: брендинг, менеджери, додатки, вебхуки, виплати та інше.
- **`/plan`** — Керування вашою підпискою Wink.
- **`/invite`** — Прийом запрошень для керування іншими акаунтами.
- **`/analytics`** — Панелі продуктивності по ваших акаунтах.
- **`/leaderboard`** — Перегляд рейтингу вашої власності за географією.
- **`/claims`** — Перетворення лідів власності у повні оголошення.

## Портал vs. Налаштування користувача

Портал керує всім, що ви робите як організація або власник акаунта. Особисте керування профілем користувача (електронна пошта, пароль, ключі доступу, дозволи OAuth-додатків) знаходиться окремо у сервісі **IAM** за адресою `iam.wink.travel/my-account`. Детальніше дивіться у [Налаштування користувача](/uk/account/overview).

## Перемикання акаунтів

Ви можете керувати кількома акаунтами — власностями, афілійованими акаунтами, акаунтами турагентів, корпоративними акаунтами — з одного входу. Використовуйте перемикач акаунтів у верхній навігації, щоб переходити між ними. Налаштування, аналітика та більшість функцій працюють у межах конкретного акаунта.
