Studio надає кілька вбудованих віджетів, які ви можете додати на свій вебсайт для збільшення кількості бронювань.

Щоб отримати доступ до вбудованих віджетів, перейдіть у `Інструменти > Різне` в бічному меню.

## Доступні віджети

| Віджет | Опис |
| -- | -- |
| **Кнопка акаунта** | Фірмова кнопка, що посилається на вашу афілійовану сторінку |
| **Кнопка маршруту** | Кнопка, що відкриває селектор маршруту |
| **Форма маршруту** | Вбудована форма вибору дати та гостей |
| **Кнопка пошуку** | Кнопка, що відкриває пошук об’єктів |
| **Кнопка пошуку в інвентарі** | Кнопка для пошуку конкретного інвентарю |

## Як вбудувати

Кожен віджет надає фрагмент коду, який ви можете скопіювати та вставити у HTML вашого вебсайту. Фрагмент містить ваш **Client ID** та **Customization ID**, щоб віджет автоматично був пов’язаний з вашим афілійованим акаунтом.

:::note
Якщо ви використовуєте WordPress, ми також пропонуємо плагін для WordPress для легшої інтеграції.
:::
