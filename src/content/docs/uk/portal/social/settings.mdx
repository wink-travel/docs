# Налаштування

Коли ви вперше починаєте працювати з Social, ми запитаємо, як ви хочете, щоб ми створювали ваш контент. Ці опції доступні вам у будь-який час, тож ви можете легко змінити їх пізніше.

Перейдіть до `Preferences` у бічному меню, щоб керувати своїми налаштуваннями.

Розділ preferences містить ваші базові налаштування генерації контенту:

1. **Preferred customization** Виберіть, яку кастомізацію використовувати при генерації контенту. Кастомізації контролюють кольори теми, стандартний маршрут, валюту, мову та інше. Ви створюєте кастомізації у своєму акаунті на [my.wink.travel](https://my.wink.travel).
2. **Auto-approve** Якщо увімкнено, згенеровані пости публікуються автоматично без вашого схвалення. Увімкніть це, коли будете впевнені у якості контенту, який ми для вас створюємо.
3. **Approver name** Ім’я особи або команди, яка отримуватиме електронні листи з запитами на схвалення.
4. **Approver e-mail** Електронна адреса, на яку надсилатимуться запити на схвалення майбутніх постів.

## Content creators

<CldImage

Social пропонує багато контент-кріейторів на вибір. Кожен кріейтор зосереджений на одному конкретному способі створення унікального контенту для вашої власності за допомогою AI. Кожен контент-кріейтор має атрибут **Weight** — більша вага збільшує ймовірність вибору цього кріейтора при генерації постів.

Ось усі доступні контент-кріейтори:

| Creator | Description |
| -- | -- |
| **Destination** | Використовує ваше розташування для створення корисного тексту про місцевість |
| **Room type** | Створює бронюваний пост з одним із ваших типів номерів із AI-згенерованим текстом та зображеннями типу номера |
| **Activity** | Створює бронюваний пост з однією з ваших активностей |
| **Add-On** | Створює бронюваний пост з одним із ваших додаткових сервісів |
| **Attraction** | Створює бронюваний пост з однією з ваших визначних пам’яток |
| **Place** | Створює бронюваний пост з одним із ваших місць |
| **Meeting room** | Створює бронюваний пост з однією з ваших конференц-залів |
| **Restaurant** | Створює бронюваний пост з одним із ваших ресторанів |
| **Spa** | Створює бронюваний пост з одним із ваших спа |
| **Property** | Створює бронюваний пост з вашим найкращим номером за ціною з AI-згенерованим текстом та зображеннями власності |
| **User sentiment** | Агрегує відгуки гостей з усього інтернету у пост, що відображає позитивні враження гостей про вашу власність |
| **Funny** | Створює легкі, розважальні пости, щоб підтримувати цікавість вашої стрічки |

:::tip[Порада експерта]
Переконайтеся, що у вас є потрібний тип інвентарю для обраних контент-кріейторів. Наприклад, якщо ви обираєте "Restaurant", у вашій власності має бути принаймні один ресторан.
:::

## Tone of voice

Тон голосу визначає стиль мови, який AI використовує при генерації контенту. Оберіть тони, що відповідають особистості вашого бренду. Як і у випадку з контент-кріейторами, кожен тон має **Weight** для контролю частоти використання.

Ось доступні тони:

| Tone | Style |
| -- | -- |
| **Neutral** | Спокійний, авторитетний, фактичний. Без перебільшень і жартів — лише надійні, структуровані знання |
| **Friendly** | Теплий, доступний, енергія усмішки через текст із розмовними скороченнями та іноді емодзі |
| **Luxury** | Вишуканий, витончений. Слова на кшталт "вишуканий" і "індивідуальний", що створюють відчуття ексклюзивності |
| **Foodie** | Чуттєві, апетитні описи. Мова, що викликає апетит |
| **Adventurous** | Сміливий, відважний, енергія дослідника. Яскраві сенсорні описи, що роблять усе епічним |
| **Family** | Доброзичливий, дружній до дітей. Тепла батьківська енергія про безпечні та веселі враження |
| **Romantic** | Поетичний, мрійливий, що зворушує серце. Захід сонця, ніжна мова і любов |
| **Humorous** | Саркастичний, швидкі жарти, дотепна гра слів. Читається як стендап |
| **Eco** | Щирий і праведний. Підкреслює зелені альтернативи та сталий вибір |
| **Local insider** | Дуже конкретні місцеві посилання, попереджає про туристичні пастки, використовує місцевий сленг |
| **Inclusive** | Привітний, святковий і підтримуючий. Робить так, щоб кожен почувався поміченим і цінним |
| **Mystical** | Ефірний, духовний, космічні істини. Легкі запрошення заглянути всередину себе |
| **Futuristic** | Стильний, орієнтований у майбутнє. Любить терміни на кшталт "парадигмальна зміна" та оптимістичні бачення майбутнього |
| **Budget** | Практичний, винахідливий, гордо ощадливий. Підкреслює доступні знахідки та безкоштовні враження |

## Post types

Типи постів визначають мету та структуру згенерованого контенту. Кожен тип поста має **Weight** для контролю частоти використання.

| Post type | Description |
| -- | -- |
| **Transactional** | Чіткий, лаконічний, орієнтований на конверсію. Сприяє бронюванням і прямим діям за допомогою фактів, цін і наступних кроків |
| **Awareness** | Контент для побудови бренду, призначений для представлення вашої власності та підвищення видимості серед потенційних гостей |
| **Sentiment** | Емоційно резонансний контент, що створює позитивні асоціації та будує емоційний зв’язок з аудиторією |

Підключивши свій Google Business Profile, ми можемо отримувати більш детальну інформацію про власність для ефективнішого контенту.
