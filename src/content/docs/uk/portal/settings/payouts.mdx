Щоб керувати банківськими рахунками, натисніть `Account > Bank Accounts` у бічному меню.

## Додавання банківського рахунку

Натисніть `Add bank account`, щоб зареєструвати новий рахунок. Форма адаптується залежно від вибраної країни:

- **Країна платежу** — Країна, де розташований ваш банк.
- **Валюта** — Валюта, в якій ви хочете отримувати виплати.
- **Ім’я власника рахунку** — Ім’я, вказане на банківському рахунку.
- **Реквізити маршрутизації** — Поля, специфічні для країни, такі як Routing number (США), Sort code (Великобританія), SWIFT/BIC код або IBAN.
- **Номер рахунку** — Номер вашого банківського рахунку.

## Основний рахунок

Ви можете призначити один банківський рахунок як **Основний**. Ваш основний рахунок отримує всі автоматичні виплати. Додаткові рахунки можна використовувати для ручних зняттів.

Щоб змінити основний рахунок, натисніть `Make primary` на будь-якому рахунку у вашому списку.

## Керування рахунками

У списку банківських рахунків ви можете:

- Редагувати деталі рахунку.
- Видалити рахунок, який більше не потрібен.
- Змінити, який рахунок є основним.
