:::note
Створення вашого першого основного тарифу є частиною майстра налаштування.
:::

:::note
`Основний тариф` — це унікальна пара `номер для гостей / тарифний план`.

*Більшість постачальників туристичних тарифів використовують лише номер для гостей і тарифний план у цьому контексті. Коли ви бачите свій номер для гостей, виставлений на продаж на OTA або системах бронювання, мандрівники порівнюватимуть тарифні плани.*

Wink додав термін `основний тариф`, щоб надати унікальній комбінації більше гнучкості.
:::

:::note
Основні тарифи створюються / видаляються автоматично при додаванні або видаленні номера для гостей або тарифного плану.
:::

Щоб керувати своїми основними тарифами, натисніть `Монетизація > Основний тариф` у головному меню.

У власності буде основний тариф для кожної комбінації номеру для гостей / тарифного плану.

*Приклад: якщо у вашій власності один номер для гостей і два тарифні плани, гість побачить два основні тарифи, доступні для продажу (якщо всі умови виконані).*

Запис основного тарифу має два параметри для керування:

- Увімкнути або вимкнути доступність основного тарифу.
- Встановити переваги.

## Доступність

Коли ви вмикаєте або вимикаєте основний тариф, це впливає на доступність у всіх ваших каналах продажу. Цей перемикач — швидкий спосіб відключити конкретний основний тариф скрізь, негайно.

## Переваги

Коли мандрівники шукають номер для гостей, вони не взаємодіють безпосередньо з номером. Натомість вони взаємодіють з основним тарифом цього номера.
Основний тариф може містити унікальні переваги, які доступні лише для конкретної комбінації номеру для гостей / тарифного плану; таким чином власники власності отримують ще один спосіб стимулювати цю комбінацію.

Дві речі, які слід враховувати при виборі переваги:

- Чи є вона гарантованою або залежить від наявності.
- Її числове значення.

Гарантована перевага має вищу числову оцінку, оскільки вона представляє більшу цінність для клієнтів.
Кожен готель отримує `бал переваг`, який базується на сумарному числовому значенні всіх переваг у всіх основних тарифах.
Мандрівники, які шукають вигідні пропозиції, можуть сортувати готелі за високим балом переваг.

Переваги — це спосіб:

- Відрізнити комбінацію номеру для гостей / тарифного плану.
- Досягти більшої видимості. *Тобто отримати вищий бал переваг, ніж інші готелі у вашому регіоні.*

:::tip
Переваги виникли з бажання готелів виділитися та `обіграти систему`.
Останнього разу, коли вони це робили, вони фактично завдали собі шкоди, віддаючи всі свої прибутки OTA за кращі позиції через вищі комісії.
Історично комісії були єдиним способом для готелів маніпулювати системою.

Переваги — це спосіб для готелів винагороджувати мандрівників, які не обов’язково мають реальні витрати для готелів.

Якщо у вас є пропозиції щодо інших типів переваг, які можна надавати, [зв’яжіться з нами](mailto:hi@wink.travel).
:::

Існує два способи увімкнути інвентар для каналу продажу:

- Увімкнути доступність інвентарю на картці. *Див. зображення вище*
- Перейти до `Розповсюдження > Інвентар` у головному меню, присвяченому керуванню інвентарем у всіх каналах продажу.

## API

Розробники, які хочуть керувати `Основними тарифами`, можуть перейти до [Розробники > API > Монетизація](/uk/developers/apis/#monetize-api).
