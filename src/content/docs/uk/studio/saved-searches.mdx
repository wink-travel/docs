Збережені пошуки відповідають назві. Під час пошуку туристичного інвентарю ви маєте можливість зберегти свій пошуковий запит для подальшого використання.

## Збереження вашого пошукового запиту

Коли ви починаєте пошук інвентарю (див.[Пошук](/studio/search)), ви маєте можливість точно налаштувати фільтри пошуку.
Внизу форми запиту є опція`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Зберегти пошук" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Показує посилання для збереження пошукового запиту*</figcaption>
</figure>


Натиснувши на це посилання, ви зможете дати назву цьому пошуку (*наприклад, Моя бібліотека чарівних готелів*) щось, що ви запам'ятаєте на потім.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Дайте назву своєму пошуку" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Дайте назву своєму пошуку*</figcaption>
</figure>


Дайте йому назву та натисніть на`Save search`кнопка. Після цього ви можете перейти до`Inventory > Saves Searches`з головної панелі навігації, щоб переглянути щойно створений запис.

## Список дій

Збережений пошук можна використовувати для відображення туристичних пропозицій користувачам у вигляді сітки або карти, а результати пошуку відображатимуться так, як вам потрібно.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Збережені дії пошуку" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Збережений пошуковий запит без застосованих фільтрів*</figcaption>
</figure>


Кожен збережений пошуковий запит має дії, які ви можете виконати над ним:

* Перетворити пошуковий запит на[карта](/studio/maps).
* Перетворити пошуковий запит на[сітка](/studio/grids).
* Видалити збережений пошук.

## API

Розробники, які хочуть керувати збереженими пошуками, можуть перейти до[Розробники > API > Огляд](/developers/apis/#browse-api).
