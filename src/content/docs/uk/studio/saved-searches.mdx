Збережені пошуки роблять саме те, що вказано в назві. Під час пошуку туристичного інвентарю у вас є можливість зберегти свій пошуковий запит для подальшого використання.

## Збереження пошукового запиту

Коли ви починаєте шукати інвентар (див. [Пошук](/uk/studio/search)), у вас є можливість точно налаштувати фільтри пошуку.
Внизу форми запиту є опція `Зберегти пошук`.

Натискання на це посилання дасть вам можливість назвати цей пошук (*наприклад, Моя бібліотека чарівних готелів*), щоб ви могли його запам’ятати для подальшого використання.

<figure>
<CloudinaryImage
      width="350"
      height="329"
      resize="scale"
      quality="best"
      alt="Дайте пошуку назву"

      errorImage="noimage_opaque_nyrtl0.png"
    />
    <figcaption>*Дайте пошуку назву*</figcaption>
</figure>

Дайте йому назву та натисніть кнопку `Зберегти пошук`. Після цього ви можете перейти до `Інвентар > Збережені пошуки` у головному меню, щоб побачити створений запис.

## Дії зі списком

Ваш збережений пошук можна використовувати для відображення туристичного інвентарю вашим користувачам у вигляді сітки або карти та налаштувати відображення результатів пошуку так, як вам потрібно.

Кожен збережений пошуковий запит має дії, які ви можете виконати:

- Перетворити пошуковий запит на [карту](/uk/studio/maps).
- Перетворити пошуковий запит на [сітку](/uk/studio/grids).
- Видалити збережений пошук.

## API

Розробники, які хочуть керувати збереженими пошуками, можуть перейти до [Розробники > API > Перегляд](/uk/developers/apis/#browse-api).
