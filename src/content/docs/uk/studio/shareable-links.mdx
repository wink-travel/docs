Посилання для спільного використання — це основна функція Wink, яка виникла з потреби динамічно передавати інформацію про інвентар та ціни найпростішим способом в Інтернеті; а саме через гіперпосилання.

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

Вгорі наведено приклад одного з наших посилань. Воно коротке і зручне для перенесення... навіть настільки коротке, що його легко запам’ятати. Хоча посилання коротке, воно містить **величезну** кількість інформації, такої як:

- Ваш App ID.
- Ваш Affiliate ID.
- Зображення(я) та текст, які відображатимуться користувачу.
- Зовнішній вигляд (*тема, кольори та інше*).
- Маршрут, який використовується для запиту ціни (*2 дорослих, 1 ніч*).
- Валюта для відображення ціни.
- Мова для відображення тексту.

Далі в цій статті ми розповімо, як створювати, налаштовувати та ділитися вашими посиланнями.

Вгорі зображення з [Пошуку](/studio/search), яке показує деякі дії, які ви можете виконати з результатами пошуку. Одна з цих дій — `Створити посилання`.
Натисніть цю кнопку, і вас перенаправить на сторінку форми для створення посилання для спільного використання, де ви зможете почати налаштовувати посилання.

## Налаштувати посилання

Форма дозволяє налаштувати посилання наступним чином:

<Steps>
 1. Дайте йому назву, щоб пам’ятати, про що це посилання.
 2. Виберіть [Налаштування](/studio/customization), які хочете застосувати до цього посилання.
 3. Додайте ключові слова, розділені комою, які будуть використовуватися [веб-краулерами](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. За бажанням введіть ваш X-аккаунт або Facebook App ID.
 5. Додайте заголовки та описи мовами, якими хочете показувати користувачу.
 6. Виберіть одне або кілька зображень, які відображатимуться при поширенні посилання на платформах, таких як Facebook або LinkedIn.
 7. Оберіть тему / накладку, яка додає брендований дизайн поверх вибраних зображень. *Примітка: саме так відображаються ціни поверх зображень.*
 8. Натисніть кнопку `Зберегти`, щоб продовжити.
 </Steps>

Вгорі зображення з усіма доступними діями для вашого посилання:

- **Попередній перегляд** Показує швидкий огляд того, як налаштоване посилання і як воно виглядатиме для користувача.
- **Оновити** Оновлює конфігурацію вашого посилання.
- **Додати до WinkLinks** Додає посилання до вашого облікового запису WinkLinks.
- **Копіювати** Копіює посилання у буфер обміну, щоб ви могли легко поділитися ним будь-де (*включно з [WinkLinks](/link-manager/wink-links) *).
- **Поділитися** Відкриває нове вікно з опціями для поширення у багатьох соціальних мережах та месенджерах.
- **QR-код** Генерує QR-код, який можна поширювати на платформах, таких як IG, орієнтованих на зображення.
- **Вбудувати** Показує, як вбудувати це посилання як [рекламний банер](/developers/web-components/#content-loader) на ваш сайт.
- **Використовувати з WordPress** Показує, як використовувати наш [плагін для WordPress](/developers/wordpress) для вбудовування цього посилання на ваш сайт.

Деякі з цих опцій ми розглянемо детальніше нижче.

Якщо ви хочете поділитися QR-кодом замість посилання, натисніть кнопку `QR code`, і з’явиться нове вікно з вашим QR-кодом.

Як зберегти QR-код:

- Якщо ви на ноутбуці, клацніть правою кнопкою миші по зображенню і виберіть `Зберегти зображення`.
- Якщо ви на мобільному телефоні:
    - Натисніть і утримуйте палець на зображенні, доки не з’явиться опція зберегти зображення у Фотографії.
    - Зробіть скріншот QR-коду, обріжте його за потреби і збережіть у Фотографії.

### Вбудувати

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Ось як вбудувати ваше посилання як рекламний банер на ваш сайт:

- Рядок 3 показує, як вбудувати стилі Wink на ваш сайт.
- Рядки 6–9 показують, як використовувати Web Component [wink-content-loader](/developers/web-components/#content-loader) і вказати йому завантажити рекламний банер для вашого коду.
- Рядок 11 показує, як вбудувати наш Javascript на ваш сайт.
- Рядок 13 показує, як вбудувати Web Component [wink-app-loader](/developers/web-components/#application-loader) і вказати йому завантажити налаштування конфігурації сторінки.
