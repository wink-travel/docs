Отримувачі платежів використовують TripPay, щоб:

- Відстежувати, скільки грошей вони заробили.
- Скільки наразі на їхньому рахунку.
- ...і яка сума доступна для виведення.

## Використання

Звичайний отримувач платежів зазвичай взаємодіє з TripPay так:

<Steps>
1. TripPay утримує кошти з 10 бронювань на 3 різних туристичних платформах для Hotel X, і кошти тепер доступні.
2. Hotel X отримує електронного листа від TripPay з повідомленням `У вас є доступні кошти` та посиланням для керування своїм рахунком.
3. Hotel X входить у [TripPay](https://pay.wink.travel).
4. Hotel X бачить бронювання за минулий місяць разом з аналітикою рахунку.
5. Hotel X обирає виведення коштів.
6. Hotel X вибирає метод `Виплата`.
7. Hotel X виконує необхідні кроки для обробки коштів.
8. Hotel X надсилає запит на переказ і натискає `OK`.
</Steps>

*Кошти зазвичай доступні протягом 24 годин у всіх регіонах.*

:::note
Якщо ви досягнете певного обсягу та суми, ми можемо звернутися до вас за додатковими документами, що підтверджують вашу особу та/або особу вашої компанії.
:::
