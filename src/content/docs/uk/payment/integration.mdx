Якщо ви туристична компанія і хочете передати на аутсорсинг частину свого бізнесу, пов’язану з оплатою та виплатами, TripPay, ось 6 основних кроків для початку:

<Steps>
1. Створіть [користувацький акаунт](/uk/user/register) на Wink.
2. Створіть акаунт у [TripPay](https://pay.wink.travel).
3. Створіть [Додаток](/uk/developers/applications) у TripPay, який дозволить вам взаємодіяти з нашим API.
4. Створіть і зв’яжіть своїх бенефіціарів з TripPay. *наприклад, для більшості це означає створення акаунтів для об’єктів на вашій платформі, яким ви хочете, щоб TripPay здійснював виплати.*
5. Інтегруйте [веб-компонент оплати](/uk/developers/web-components/#payment) на свій сайт.
6. Надішліть TripPay `кошик покупок` мандрівника та ініціюйте процес оплати.
</Steps>

:::tip
Для повного посібника з використання TripPay на вашому туристичному сайті прочитайте керівництво [Інтеграція з TripPay](/uk/guides/integrating-with-trip-pay).
:::

## API

Розробники, які хочуть керувати `Платежами`, переходьте до [Розробники > API > Платежі](/uk/developers/apis/#payment-api).
