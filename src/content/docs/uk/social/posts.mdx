Редактор постів дозволяє створювати необмежену кількість ручних постів, а також це місце, куди ви заходите для керування постами, створеними для вас автоматично.

Редактор має 4 секції:

1. Секція контенту.
2. Секція медіа.
3. Секція посилань.
4. Секція налаштувань.

## Контент

Секція контенту дозволяє змінювати текстовий вміст перед публікацією. Ви можете створювати локалізований контент кількома мовами — для кожної мови є свій заголовок і опис.

## Медіа

Секція медіа дозволяє змінювати зображення та відео, що супроводжують текст.

## Посилання

Секція посилань дозволяє додати посилання, куди ви хочете, щоб користувач перейшов після прочитання вашого поста. *Наприклад, це може бути посилання на спеціальну пропозицію або місце для отримання додаткової інформації.* Ви можете додати URL посилання, підпис до посилання та опис посилання.

## Налаштування

Секція налаштувань дозволяє змінювати / перевизначати всі ваші стандартні налаштування для кожного поста. Ви можете встановити:

- **Платформа** — Виберіть, в якій соціальній мережі буде опубліковано цей пост.
- **Тип поста** — Виберіть між Транзакційним, Інформативним або Емоційним.
- **Тон голосу** — Перевизначте тон голосу для цього конкретного поста.
- **Налаштування** — Використовуйте інші налаштування для валюти відображення, мови, маршруту тощо.
- **Часовий пояс** — Перевизначте часовий пояс для публікації.
- **Запланована дата** — Автоматично створені пости заплановані на оптимальний час. Ви можете змінити дату і час, якщо бажаєте інший слот.

## Життєвий цикл статусу поста

Кожен пост проходить життєвий цикл статусу:

| Статус | Опис |
| -- | -- |
| **Чернетка** | Пост створено, але він ще не готовий до перегляду |
| **Очікує затвердження** | Пост чекає на затвердження людиною перед плануванням |
| **Затверджено** | Пост затверджено і він буде запланований |
| **Відхилено** | Пост переглянуто і відхилено — він не буде опублікований |
| **Заплановано** | Пост поставлено в чергу на публікацію у запланований час і дату |
| **Опубліковано** | Пост успішно опубліковано в соціальній мережі |
| **Призупинено** | Публікація поста тимчасово призупинена |
| **Помилка** | Пост не вдалося опублікувати через помилку |

## Затвердження постів

Пости затверджуються у сітці постів, де ви можете вибрати один або кілька постів і обрати затвердити або відхилити їх.

:::note
Для кожної мови і кожної соціальної мережі буде окремий запис.

Наприклад, якщо у вас вибрано 3 соціальні мережі і 3 мови, ви побачите 9 записів у календарі.
:::
