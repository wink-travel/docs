Ви вже підключені до WinkLinks, нашого інструменту керування посиланнями. Підключайте більше своїх соціальних мереж, натискаючи на `Підключити соціальну мережу`.
Процес перенаправить вас до вашої соціальної мережі для підтвердження додатку `Wink Social` на публікацію контенту від вашого імені.

Після підключення ми починаємо генерувати контент для цієї мережі на наступний запланований місяць.

:::note
Безкоштовний тариф включає WinkLinks. Facebook, Instagram та інші платформи доступні на платних планах.
:::

## Підтримувані платформи

| Платформа | Опис | Вимоги |
| -- | -- | -- |
| **WinkLinks** | Автоматична публікація через наш розширений менеджер посилань | Включено у всі плани |
| **Facebook** | Автоматизація публікацій, відповідей на коментарі та особисті повідомлення | Потрібна бізнес-сторінка Facebook |
| **Instagram** | Автоматизація публікацій, відповідей на коментарі та особисті повідомлення | Потрібен бізнес-акаунт Instagram, пов’язаний з акаунтом Facebook |

Кожне підключення показує свій поточний статус: **Активний**, **Відключений** або **Призупинений**.

:::note
Ми створюємо та публікуємо контент, оптимізований для кожної соціальної мережі.
:::
