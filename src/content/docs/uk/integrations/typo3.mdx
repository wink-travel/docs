**Розширення Wink TYPO3** інтегрує інвентар Wink для подорожей у TYPO3 13 як нативні елементи контенту. Редактори додають компоненти Wink на сторінки за допомогою стандартного редактора сторінок TYPO3 — без необхідності змін у шаблонах.

## Вимоги

- TYPO3 13.x
- PHP 8.2+
- Composer

## Встановлення

<Steps>
1. Встановіть через Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Активуйте розширення:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. У бекенді TYPO3 перейдіть до **Admin Tools → Settings → Extension Configuration → wink**.
4. Введіть ваші **Client ID**, **Client Secret** та оберіть середовище. Збережіть.
   Перейдіть до [Applications](/uk/portal/settings/applications), щоб отримати свої облікові дані.
5. Додайте статичний шаблон TypoScript до кореневого шаблону:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Додавання елемента контенту на сторінку

<Steps>
1. Відкрийте сторінку у модулі **Page**.
2. Натисніть кнопку **+** у області контенту, щоб додати новий елемент.
3. Оберіть один із елементів контенту **Wink Travel**:
   - **Wink Content** — картки готелів, рейтингові сітки, карти
   - **Wink Lookup** — панель пошуку напрямків
   - **Wink Search** — кнопка вибору маршруту
   - **Wink Account** — кнопка входу
   - **Wink Itinerary** — кнопка статусу маршруту
   - **Wink Cart** — кнопка кошика покупок
4. Для **Wink Content** використовуйте поля FlexForm, щоб обрати **Layout Type** та ввести **Layout ID**.
5. Збережіть і опублікуйте сторінку.
</Steps>

## Доступні елементи контенту

| Елемент | Призначення |
|---|---|
| **Wink Content** | Картки готелів, рейтингові сітки, карти та інше |
| **Wink Lookup** | Панель пошуку напрямків |
| **Wink Search** | Іконка-кнопка, що відкриває вибір маршруту |
| **Wink Account** | Кнопка входу / облікового запису користувача |
| **Wink Itinerary** | Кнопка маршруту, що показує поточну поїздку |
| **Wink Cart** | Кнопка з підсумком кошика покупок |
