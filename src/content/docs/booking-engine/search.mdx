The search page is where travelers browse properties by location.

## Browsing by location

Properties can be browsed by location type — cities, islands, countries, and more. Each location page shows a grid of available hotels with pricing.

Each property card displays:

- Hotel name and main image.
- Location.
- Lowest available price.
- Guest rating.

## Itinerary selector

Travelers can refine their search using the itinerary selector at the top of the page:

- **Check-in / Check-out dates** — Set the travel dates.
- **Guests** — Number of adults and children.
- **Promo code** — Apply a promotional code for discounts.

The property grid updates in real time as the itinerary changes.

## No results

If no properties match the search criteria, a helpful message is shown suggesting the traveler adjust their dates or location.
