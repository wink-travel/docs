Si ets un integrador, com Wink, i vols assignar els teus pagadors externs per utilitzar-los amb TripPay, ho pots fer d'una de dues maneres:

- Assignació manual
- Utilitzant la [Payment API](/ca/developers/apis/#payment-api) (*Consulta `Account Mappings`*)

:::tip
A menys que només gestionis un nombre reduït de les mateixes propietats, recomanem que utilitzis l'API per automatitzar aquest pas.
:::

A continuació, t'expliquem el procés manual per assignar un pagador extern. *S'utilitza l'API en segon pla.*

<Steps>
1. Inicia sessió a [TripPay](https://pay.wink.travel).
2. Selecciona el teu compte d'integrador.
3. Fes clic a `Settings > Mappings` des de la barra de navegació principal.
4. Fes clic al botó `Create mapping`. Veureu el formulari d'assignació.
5. **Local account** Comença a escriure el nom del TripPay existent que vols assignar.
6. **External identifier** Enganxa el teu ID extern per a aquesta entitat tal com existeix en el sistema remot.
7. Fes clic a `Save` per continuar. Seràs redirigit a la teva llista d'assignacions.
</Steps>

Amb les teves assignacions configurades, TripPay ja sap quins comptes acreditar com a beneficiaris d'una reserva quan els refereixes utilitzant els teus identificadors externs.

## API

Els desenvolupadors que vulguin gestionar `Payments`, han d'anar a [Developers > API > Payments](/ca/developers/apis/#payment-api).
