El motor de reserves està disponible en dues versions per donar suport a les diferents necessitats d’un afiliat, una empresa de gestió de propietats i un hotel.

## Agència de viatges en línia

Per a un afiliat, té més sentit utilitzar la versió d’agència de viatges en línia. El motiu és que el lloc està configurat per acomodar viatgers que volen veure diverses propietats.

Podeu trobar la pàgina d’inici d’aquesta aplicació a continuació.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Motor de reserves per Internet

Per a un hotel o una empresa de gestió de propietats, té més sentit utilitzar la versió del motor de reserves per Internet. Aquest tipus d’usuari està interessat a destacar una sola propietat o diverses propietats sota gestió.

Podeu trobar la pàgina d’inici d’aquesta aplicació a continuació.

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE
</LinkButton>

## Configuració

La majoria de vegades, no cal compartir cap d’aquestes aplicacions directament. Hem creat diverses maneres perquè pugueu compartir l’inventari de viatges, que podeu descobrir a [Wink Studio](/studio/what-is-studio) o a través de [WinkLinks](/link-manager/wink-links).

Tanmateix, si voleu enllaçar manualment a un d’aquests llocs web, assegureu-vos d’afegir el vostre ID de client i l’ID de personalització a l’URL com a paràmetres.

Podeu trobar el vostre ID de client fent:

<Steps>
1. Inicieu sessió a [Wink Extranet](/extranet/what-is-extranet) o a [Wink Studio](/studio/what-is-studio).
2. Feu clic a la icona del vostre perfil a la cantonada superior dreta i feu clic a [Aplicacions](/developers/applications).
3. Utilitzeu l’aplicació vinculada al compte Wink amb què voleu treballar.
4. Copieu l’ID de client d’aquesta aplicació i enganxeu-lo a l’URL com a paràmetre de sol·licitud. Exemple: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Podeu trobar el vostre ID de personalització com a propietat fent:

<Steps>
1. Inicieu sessió a [Wink Extranet](/extranet/what-is-extranet).
2. Seleccioneu la propietat amb què voleu treballar.
3. A la barra de navegació principal, feu clic a `Compte > Marca`
4. Copieu l’ID de personalització i enganxeu-lo a l’URL com a paràmetre de sol·licitud. Exemple: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Podeu trobar el vostre ID de personalització com a afiliat fent:

<Steps>
1. Inicieu sessió a [Wink Studio](/studio/what-is-studio).
2. Seleccioneu el compte amb què voleu treballar.
3. A la barra de navegació principal, feu clic a `Eines > Personalització`
4. Copieu l’ID de personalització de la personalització que voleu utilitzar i enganxeu-lo a l’URL com a paràmetre de sol·licitud. Exemple: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

El que obtindreu és un URL que s’assembla a això:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Si voleu anar directament a la pàgina d’un hotel, l’URL seria similar a aquest:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Si no incloeu un ID de client i/o de configuració, Wink OTA utilitzarà una configuració per defecte.
:::

## IBE

Si sou una empresa de gestió de propietats, voleu enllaçar a la pàgina d’inici de Wink IBE. La pàgina d’inici mostrarà el vostre tema designat juntament amb les [propietats que representeu](/studio/agency-representation).

El que obtindreu és un URL que s’assembla a això:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Si sou una propietat, voleu enllaçar directament a la pàgina d’aterratge de la propietat a Wink IBE.

Aquest URL seria similar a aquest:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
Si no incloeu un ID de client i/o de configuració, Wink IBE NO s’iniciarà. Aquests paràmetres són obligatoris.
:::
