# Configuració

Quan comences a utilitzar Social, et preguntarem com vols que creem el teu contingut. Aquestes opcions estan disponibles en tot moment perquè les puguis canviar fàcilment més endavant.

Navega a `Preferències` des de la barra lateral per gestionar la teva configuració.

## Preferències

<figure>

  <figcaption>*Fig. 1 - Preferències*</figcaption>

La secció de preferències conté la configuració bàsica per a la generació de contingut:

1. **Personalització preferida** Tria quina personalització vols utilitzar per generar contingut. Les personalitzacions controlen els colors del tema, l’itinerari per defecte, la moneda, l’idioma i més. Pots crear personalitzacions al teu compte a [my.wink.travel](https://my.wink.travel).
2. **Aprovar automàticament** Quan està activat, les publicacions generades es publiquen automàticament sense necessitat de la teva aprovació. Activa-ho quan confiïs en la qualitat del contingut que produïm per a tu.
3. **Nom de l’aprovador** El nom de la persona o equip que rebrà els correus electrònics de sol·licitud d’aprovació.
4. **Correu electrònic de l’aprovador** L’adreça de correu electrònic on s’enviaran les sol·licituds d’aprovació per a les publicacions futures.

## Creadors de contingut

<figure>
<CldImage
  src="academy/social_content_creators"

Social inclou molts creadors de contingut per triar. Cada creador se centra en una manera específica de crear contingut únic per a la teva propietat utilitzant IA. Cada creador de contingut té un atribut de **Pes** — un pes més alt augmenta la probabilitat que aquest creador sigui seleccionat quan es generen publicacions.

Aquí tens tots els creadors de contingut disponibles:

| Creador | Descripció |
| -- | -- |
| **Destinació** | Utilitza la teva ubicació per crear textos útils sobre la zona |
| **Tipus d’habitació** | Crea una publicació reservable amb un dels teus tipus d’habitació amb text generat per IA i imatges del tipus d’habitació |
| **Activitat** | Crea una publicació reservable amb una de les teves activitats |
| **Addició** | Crea una publicació reservable amb un dels teus add-ons |
| **Atracció** | Crea una publicació reservable amb una de les teves atraccions |
| **Lloc** | Crea una publicació reservable amb un dels teus llocs |
| **Sala de reunions** | Crea una publicació reservable amb una de les teves sales de reunions |
| **Restaurant** | Crea una publicació reservable amb un dels teus restaurants |
| **Spa** | Crea una publicació reservable amb un dels teus spas |
| **Propietat** | Crea una publicació reservable amb l’habitació millor preuada amb text generat per IA i imatges de la propietat |
| **Sentiment d’usuari** | Agrega ressenyes de clients d’arreu de la web en una publicació basada en sentiments que destaca què els agrada als clients de la teva propietat |
| **Divertit** | Crea publicacions lleugeres i entretingudes per mantenir el teu feed atractiu |

:::tip[Consell professional]
Assegura’t de tenir el tipus d’inventari necessari per als creadors de contingut que triïs. Per exemple, si selecciones "Restaurant", has de tenir almenys un restaurant registrat a la teva propietat.
:::

## To de veu

<CldImage

El to de veu estableix l’estil de llenguatge que utilitza la IA quan genera contingut. Tria els tons que coincideixin amb la personalitat de la teva marca. Igual que amb els creadors de contingut, cada to té un **Pes** per controlar la freqüència d’ús.

Aquí tens els tons disponibles:

| To | Estil |
| -- | -- |
| **Neutral** | Calmat, autoritari, factual. Sense exageracions ni acudits — només coneixement fiable i ben estructurat |
| **Amistós** | Càlid, proper, amb energia de somriure a través del text amb contraccions informals i emojis ocasionals |
| **Luxe** | Sofisticat, refinat. Paraules com "exquisit" i "personalitzat" que fan sentir exclusivitat als lectors |
| **Gastronòmic** | Descripcions sensorials i apetitoses. Llenguatge que fa venir gana als lectors |
| **Aventurer** | Valent, intrèpid, energia d’explorador. Descripcions sensorials vívides que fan que tot soni èpic |
| **Familiar** | Emoció sana i apta per a nens. Energia càlida i parental sobre experiències segures i divertides |
| **Romàntic** | Poètic, nostàlgic, que fa bategar el cor. Capvespres, llenguatge suau i amor |
| **Humorístic** | Sarcàstic, frases ràpides, jocs de paraules intel·ligents. Llegeix com un monòleg d’humor |
| **Eco** | Sincer i just. Celebra alternatives verdes i opcions sostenibles |
| **Local expert** | Referències locals molt específiques, alerta sobre trampes per a turistes, utilitza argot local |
| **Inclusiu** | Acollidor, celebratori i afirmatiu. Fa que tothom se senti vist i valorat |
| **Místic** | Etèric, espiritual, veritats còsmiques. Invitacions suaus a mirar cap a dins |
| **Futurista** | Elegant, visionari. Li agraden termes com "canvi de paradigma" i visions optimistes del futur |
| **Econòmic** | Ingeniós, amb recursos, orgullosament frugal. Celebra troballes assequibles i experiències gratuïtes |

## Tipus de publicacions

Els tipus de publicacions determinen l’objectiu i l’estructura del contingut generat. Cada tipus de publicació té un **Pes** per controlar la freqüència d’ús.

| Tipus de publicació | Descripció |
| -- | -- |
| **Transaccional** | Clar, concís, enfocat a la conversió. Impulsa reserves i accions directes amb fets, preus i passos següents |
| **De conscienciació** | Contingut per construir marca dissenyat per presentar la teva propietat i augmentar la visibilitat entre possibles clients |
| **De sentiment** | Contingut emocionalment resonant que crea associacions positives i construeix una connexió emocional amb la teva audiència |

Potser vols centrar els teus esforços de màrqueting en un idioma i regió particulars. Indica’ns els idiomes per als quals vols crear contingut i la zona horària que t’interessa i nosaltres farem la resta. Les publicacions es publiquen a l’hora programada a la zona horària seleccionada.

*La majoria de propietats utilitzen un sol idioma i per defecte el seu idioma local.*

En connectar el teu perfil de Google Business, podem obtenir detalls més complets de la propietat per a un contingut més efectiu. La teva propietat necessita un perfil de Google Business verificat perquè puguem generar contingut a partir del teu llistat. Alguns dels nostres creadors de contingut **requereixen** que tinguis un compte de Google Business.

:::tip[Consell professional]
Comença petit

No siguis massa ambiciós en el teu primer intent. Et recomanem que comencis creant contingut per a la teva propietat en un sol idioma.
:::
