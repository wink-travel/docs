## API de pagament

Els punts finals de pagament són per a desenvolupadors que volen comprar inventari de viatges. Això es pot fer a través de l'API com a agent de viatges registrat o utilitzant la nostra API juntament amb el nostre widget de pagament compatible amb PCI per a totes les altres entitats.
L'API de pagament exposa punts finals per a afiliats i hotels per fer un seguiment de reserves, anàlisis i disponibilitat de fons. També poden optar per retirar els fons disponibles al seu compte bancari.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Més informació
</LinkButton>
