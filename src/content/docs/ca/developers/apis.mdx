Totes les nostres APIs REST són independents de l'idioma i compatibles amb Open API. Comença a provar sense escriure ni una sola línia de codi.

Si encara no has creat el teu compte Wink, [comença aquí](/user/register).

## Analytics API

La Analytics API exposa endpoints per a qualsevol que vulgui obtenir informació sobre rànquings de classificació així com una anàlisi profunda de sèries temporals, dades a nivell de plataforma; oferint una gran varietat de mètriques.

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
Més informació
</LinkButton>

## Payment API

Els endpoints de Payment són per a desenvolupadors que volen comprar inventari de viatge. Això es pot fer via API com a Agent de Viatges registrat o utilitzant la nostra API conjuntament amb el nostre widget de pagament compatible amb PCI per a totes les altres entitats.
La Payment API exposa endpoints per a afiliats i hotels per fer seguiment de reserves, anàlisis i disponibilitat de fons. També poden triar retirar fons disponibles al seu compte bancari.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Més informació
</LinkButton>
