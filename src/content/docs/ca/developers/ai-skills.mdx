Les habilitats d'IA són paquets d'instruccions portàtils que ensenyen als agents de codificació com treballar amb Wink. En lloc de dependre de dades d'entrenament o documentació obsoleta, cada habilitat dirigeix l'agent a utilitzar el [servidor MCP de Wink](/ca/developers/mcp) com a font viva de veritat per als contractes d'API i la documentació de la plataforma.

Les habilitats funcionen amb qualsevol client agent que les suporti — Claude Code, Codex, Cursor i altres.

## Habilitats disponibles

### wink-api-builder

Implementa, revisa o depura integracions d'API de Wink. L'habilitat guia l'agent a través d'un flux de treball estructurat:

1. Cerca l'operació d'API correcta mitjançant `api_search`
2. Obre els contractes de sol·licitud/resposta en viu des del servidor MCP
3. Extreu la configuració d'autenticació de la documentació
4. Escriu codi segons el contracte verificat
5. Senyala qualsevol discrepància entre el codi existent i l'API actual

Útil per a:
- Construir nous mètodes de client per a qualsevol API de Wink
- Depurar discrepàncies de contracte (p. ex., una sol·licitud que retorna HTTP 400)
- Revisar integracions existents segons l'especificació en viu

## Instal·lació

Copia la carpeta de l'habilitat al directori d'habilitats del teu agent.

:::tip
El servidor MCP de Wink ha d'estar connectat en el mateix entorn. Consulta [Servidor MCP](/ca/developers/mcp) per a les instruccions de configuració.
:::

**Claude Code**

```bash
# a nivell de repositori
cp -r skills/wink-api-builder .claude/skills/

# o a nivell d'usuari
cp -r skills/wink-api-builder ~/.claude/skills/
```

Invoca amb:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# a nivell de repositori
cp -r skills/wink-api-builder .agents/skills/

# o a nivell d'usuari
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Invoca amb:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Altres agents**

Col·loca la carpeta de l'habilitat on el teu agent resol les habilitats. El punt d'entrada és `SKILL.md`.

## Requisits

- Un [servidor MCP de Wink](/ca/developers/mcp) connectat
- Un client agent amb suport per MCP i habilitats
