Wink s'integra amb els principals channel managers, sistemes de gestió de propietats (PMS) i sistemes centrals de reserves (CRS) per mantenir sincronitzades les teves tarifes i disponibilitat a tots els teus canals de venda.

## Com funciona

Les integracions amb channel manager permeten un flux bidireccional de dades entre el teu sistema de gestió de propietats i Wink:

1. **Enviament de tarifes** — El teu channel manager envia actualitzacions de tarifes a Wink. Quan actualitzes els preus al teu PMS, aquests canvis es reflecteixen automàticament a Wink.
2. **Sincronització de disponibilitat** — El teu channel manager envia actualitzacions de disponibilitat a Wink. Quan les habitacions es reserven o es alliberen al teu PMS, l'inventari de Wink s'actualitza en temps real.
3. **Notificacions de reserves** — Quan es fa una reserva a Wink, el teu channel manager en rep la notificació perquè la reserva aparegui al teu PMS.

## Channel managers compatibles

Actualment, Wink s'integra amb els següents sistemes:

| Channel Manager | Descripció |
|---|---|
| **Channex** | Plataforma global de distribució i gestió de canals. |
| **TravelClick** | Solucions hoteleres i gestió de canals (Amadeus). |
| **Omnibees** | Distribució i gestió d'ingressos per a hotels. |
| **D-Edge** | Tecnologia hotelera i solucions de reserves centrals. |
| **Allotz** | Assignació d'inventari i gestió de canals. |
| **Ezee / Yanolja** | PMS al núvol i gestió de canals enfocada al mercat asiàtic. |
| **RateGain** | Intel·ligència de tarifes i tecnologia de distribució. |
| **RateTiger** | Distribució en temps real de tarifes i disponibilitat. |
| **CloudBeds** | PMS al núvol, channel manager i motor de reserves. |
| **SiteMinder** | Principal channel manager hoteler i plataforma de distribució. |
| **Exely** | Sincronització de tarifes i disponibilitat per a hotels. |

:::note
Si el teu channel manager no està a la llista, contacta amb l'equip de Wink. Es poden afegir noves integracions basades en els estàndards del protocol OTA (Open Travel Alliance) que Wink suporta.
:::

## Per a desenvolupadors

Si estàs desenvolupant una integració de channel manager o PMS amb Wink, utilitza els endpoints de l'API de Channel Manager:

- **Actualitzacions de tarifes** — Envia canvis de tarifes a Wink.
- **Actualitzacions de disponibilitat** — Envia canvis de disponibilitat a Wink.
- **Recuperació de reserves** — Extreu reserves de Wink al teu sistema.

L'API de Channel Manager segueix els estàndards del protocol OTA (SOAP/XML) per garantir la compatibilitat amb els sistemes hotelers existents.

## Notificacions per webhook

Pots subscriure't als esdeveniments webhook del channel manager per rebre notificacions en temps real:

- `channel-manager.update.rate` — S'ha rebut una actualització de tarifa.
- `channel-manager.update.availability` — S'ha rebut una actualització de disponibilitat.
- `channel-manager.update` — Actualització general del channel manager.

Consulta el [Catàleg d'Esdeveniments Webhook](/ca/developers/webhook-events#channel-manager-events) per a més detalls.
