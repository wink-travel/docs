Accelera el temps de comercialització utilitzant un dels nostres kits de desenvolupament de programari (SAD) per a idiomes oficialment compatibles.
Si actualment no admetem el vostre idioma preferit, aneu a l'API que voleu utilitzar i descarregueu l'especificació d'OpenAPI que posem a disposició per a cada API a la part superior de la pàgina. Podeu generar el vostre propi SDK creant una bifurcació d'un dels nostres repositoris públics d'SDK o utilitzant un dels generadors d'OpenAPI.

## Java

Aquesta biblioteca s'ha compilat per a Java 24. Està configurada per utilitzar el WebClient reactiu i està optimitzada per al seu ús amb Spring Framework. Tanmateix, Spring no és un requisit, però sí que ofereix compatibilitat amb OAuth2 des del primer moment.
