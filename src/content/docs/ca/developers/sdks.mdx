Accelera el temps de sortida al mercat utilitzant un dels nostres Kits de Desenvolupament de Programari oficialment suportats per llenguatge.
Si actualment no suportem el teu llenguatge preferit, ves a l'API que vols utilitzar i descarrega l'especificació OpenAPI que posem a disposició per a cada API a la part superior de la pàgina. Pots generar el teu propi SDK fent un fork d'un dels nostres repositoris públics de SDK o utilitzant un dels generadors d'OpenAPI.

## Java

Aquesta biblioteca es va compilar per a Java 24. Està configurada per utilitzar el WebClient reactiu i està optimitzada per a l'ús amb el Spring Framework. Tot i això, Spring no és un requisit però et proporciona suport OAuth2 de sèrie.
