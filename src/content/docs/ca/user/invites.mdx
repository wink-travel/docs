Un usuari de Wink pot triar que gestionis un dels seus comptes de propietat, afiliat o agent de viatges. Quan això passa, rebràs un correu electrònic amb una sol·licitud d'invitació.

import { Steps } from '@astrojs/starlight/components';

Per veure les teves invitacions actuals de Wink, segueix aquests passos:

<Steps>
1. Has d'estar connectat a un dels nostres llocs web.
2. Fes clic a la icona del teu perfil, a la cantonada superior dreta, i apareixerà un menú.
3. Fes clic a l'enllaç `Invitacions`.
4. Interactua amb les teves invitacions acceptant-les o rebutjant-les.
</Steps>
