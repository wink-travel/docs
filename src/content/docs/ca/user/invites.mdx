Un usuari de Wink pot triar que gestionis un dels seus comptes de propietat, afiliat o agent de viatges. Quan això passi, rebràs un correu electrònic amb una sol·licitud d'invitació.

import { Steps } from '@astrojs/starlight/components';

Per veure les teves invitacions actuals de Wink, segueix aquests passos:

<Steps>
  1. Heu d'haver iniciat la sessió en un dels nostres llocs web.
  2. Feu clic a la icona del vostre perfil, a la cantonada superior dreta, i apareixerà un menú.
  3. Feu clic a `Invites` enllaç.
  4. Interactua amb les teves invitacions mitjançant qualsevol de les dues opcions: `Accepting` o `Rejecting` ells.
</Steps>
