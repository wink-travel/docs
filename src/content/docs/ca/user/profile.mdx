Si vols actualitzar el teu perfil d'usuari per al teu compte d'usuari de Wink, segueix aquests passos:

<Steps>
1. Has d'estar connectat a un dels nostres llocs web.
2. Fes clic a la icona del teu perfil, a la cantonada superior dreta, i apareixerà un menú.
3. Fes clic a l'enllaç `Perfil`.
4. Segueix les instruccions per actualitzar el teu `correu electrònic`, `nom` i `cognoms`.
5. També pots pujar una nova `foto de perfil` teva.
6. Fes clic a `Desa`.
</Steps>

:::note
La teva nova foto de perfil es mostrarà la propera vegada que tanquis sessió completament a Wink.
:::
