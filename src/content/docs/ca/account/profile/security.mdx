La pestanya **Seguretat** a [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) és on gestiones les **passkeys** — credencials WebAuthn que et permeten iniciar sessió sense contrasenya.

## Què és una passkey?

Una passkey és una credencial criptogràfica vinculada al teu dispositiu (o gestor de contrasenyes) que demostra que ets tu. Les passkeys són resistents al phishing i més ràpides que escriure una contrasenya.

## Registrar una passkey

1. Inicia sessió a [iam.wink.travel](https://iam.wink.travel) amb les teves credencials existents.
2. Ves a **El meu compte → Seguretat**.
3. Fes clic a `Afegeix passkey`.
4. El teu navegador et demanarà que t'autentiquis amb el teu dispositiu (Touch ID, Face ID, Windows Hello, una clau de maquinari o un gestor de contrasenyes sincronitzat).
5. Dona un nom amigable a la passkey (per exemple, *MacBook de feina*) perquè la puguis reconèixer més endavant.

## Utilitzar una passkey per iniciar sessió

A la pantalla d'inici de sessió de [iam.wink.travel](https://iam.wink.travel), introdueix el teu correu electrònic — si tens una passkey registrada per a aquest correu, se't demanarà que t'autentiquis amb ella en lloc d'escriure una contrasenya.

## Eliminar una passkey

A la mateixa pestanya Seguretat, cada passkey registrada té un botó `Elimina`. Utilitza'l si perds un dispositiu, retires una clau de maquinari o vols netejar credencials no utilitzades.

:::caution
Eliminar totes les teves passkeys deixa la contrasenya com a únic mètode d'inici de sessió. Assegura't de recordar la teva contrasenya abans d'eliminar l'última passkey.
:::
