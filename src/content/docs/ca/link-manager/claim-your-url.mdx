Creant un [compte d’afiliat](/ca/studio/what-is-studio), ja disposes d’una URL personalitzada de WinkLinks. Les funcions de publicador de WinkLinks existeixen a la nostra aplicació Studio, juntament amb altres funcions per a afiliats.

### Exemple

Si el nom del teu compte és Queen Bee, la teva URL personalitzada de WinkLinks serà:

<h4 align="center">https://i.trvl.as/queen-bee</h4>

Els noms de compte estan disponibles per ordre de `qui arriba primer, serveix primer` - Aconsegueix la teva URL única mentre puguis!
