Per gestionar les teves etiquetes de WinkLinks, ves a `WinkLinks` a la barra de navegació principal i fes clic a la pestanya `Etiquetes`.

A mesura que el teu contingut creix, voldràs començar a organitzar-ne una part. Hem creat les `etiquetes` per a aquest propòsit. Pots agrupar contingut utilitzant tantes etiquetes com vulguis.

## Crear etiqueta

<Steps>
1. Crea una nova etiqueta fent clic al botó `Afegeix una nova etiqueta`.
2. Introdueix el nom de la teva nova etiqueta. *p. ex. Millors cafeteries de Berlín*
3. Fes clic a l'enllaç `Desa`.
</Steps>

## Actualitzar etiqueta

<Steps>
1. Comença fent clic a l'enllaç `Edita` al costat de l'etiqueta que vols modificar.
2. Actualitza el nom de la teva etiqueta. *p. ex. Millors cafeteries de Berlín*
3. Fes clic a l'enllaç `Desa`.
</Steps>

## Eliminar etiqueta

:::note
Quan elimines una etiqueta, també l'eliminaràs de tot el contingut associat.
:::

<Steps>
1. Fes clic a l'enllaç `Elimina` al costat de l'etiqueta que vols eliminar.
</Steps>

## API

Els desenvolupadors que vulguin gestionar `WinkLinks` poden anar a [Developers > API > WinkLinks](/ca/developers/apis/#winklinks-api).
