El **Mòdul Wink per Drupal** afegeix components de l'inventari de viatges de Wink al teu lloc Drupal com a blocs nadius. Els editors poden col·locar-los a qualsevol lloc del disseny mitjançant el sistema estàndard de blocs.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Codi font, guia de configuració per a desenvolupadors i instruccions per a contribucions."
/>

## Requisits

- Drupal 10 o 11
- PHP 8.1+
- Composer

## Instal·lació

<Steps>
1. Instal·la mitjançant Composer:
   ```bash
   composer require drupal/wink
   ```
2. Activa el mòdul:
   ```bash
   drush en wink -y
   ```
3. Ves a **Administració → Configuració → Serveis web → Configuració de Wink** (`/admin/config/wink`).
4. Introdueix el teu **Client ID** i **Client Secret**, selecciona el teu entorn i fes clic a **Desa**.
   Ves a [Aplicacions](/ca/portal/settings/applications) per obtenir les teves credencials.
</Steps>

## Afegir un bloc a una pàgina

<Steps>
1. Ves a **Administració → Estructura → Disseny de blocs**.
2. Fes clic a **Col·loca un bloc** a la regió on vulguis el contingut de Wink.
3. Cerca **Wink** per trobar els blocs disponibles:
   - **Wink Content** — mostra una targeta d’hotel, quadrícula o mapa
   - **Wink Lookup** — barra de cerca de destinacions
   - **Wink Search** — botó per seleccionar l’itinerari
   - **Wink Account** — botó d’inici de sessió
   - **Wink Itinerary** — botó d’estat de l’itinerari
   - **Wink Cart** — botó del carret de la compra
4. Per a **Wink Content**, configura el **Tipus de disseny** i l’**ID de disseny** a la configuració del bloc.
5. Desa la configuració del bloc.
</Steps>

## Blocs disponibles

| Bloc | Propòsit |
|---|---|
| **Wink Content** | Mostra targetes d’hotel, quadrícules ordenades, mapes i més |
| **Wink Lookup** | Barra de cerca per a destinacions i hotels |
| **Wink Search** | Botó icona que obre el selector d’itineraris |
| **Wink Account** | Botó d’inici de sessió / compte d’usuari |
| **Wink Itinerary** | Botó d’itinerari que mostra el viatge actual |
| **Wink Cart** | Botó resum del carret de la compra |
