L'**Extensió Wink per TYPO3** integra l'inventari de viatges de Wink a TYPO3 13 com a elements de contingut natius. Els editors afegeixen components Wink a les pàgines utilitzant l'editor de pàgines estàndard de TYPO3 — no cal canviar cap plantilla.

<LinkCard
  title="wink-travel/wink-typo3-extension a GitHub"
  href="https://github.com/wink-travel/wink-typo3-extension"
  description="Codi font, guia de configuració per a desenvolupadors i instruccions per a contribucions."
/>

## Requisits

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instal·lació

<Steps>
1. Instal·la mitjançant Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Activa l'extensió:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Al backend de TYPO3, ves a **Admin Tools → Settings → Extension Configuration → wink**.
4. Introdueix el teu **Client ID**, **Client Secret** i selecciona el teu entorn. Desa.
   Ves a [Applications](/ca/portal/settings/applications) per obtenir les teves credencials.
5. Afegeix la plantilla estàtica TypoScript a la teva plantilla arrel:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Afegeix un element de contingut a una pàgina

<Steps>
1. Obre una pàgina al mòdul **Page**.
2. Fes clic al botó **+** a l'àrea de contingut per afegir un nou element.
3. Tria un dels elements de contingut **Wink Travel**:
   - **Wink Content** — targetes d'hotels, graelles ordenades, mapes
   - **Wink Lookup** — barra de cerca de destinacions
   - **Wink Search** — botó selector d'itineraris
   - **Wink Account** — botó d'inici de sessió
   - **Wink Itinerary** — botó d'estat de l'itinerari
   - **Wink Cart** — botó del carret de la compra
4. Per a **Wink Content**, utilitza els camps FlexForm per seleccionar un **Tipus de disseny** i introdueix el **ID del disseny**.
5. Desa i publica la pàgina.
</Steps>

## Elements de contingut disponibles

| Element | Propòsit |
|---|---|
| **Wink Content** | Targetes d'hotels, graelles ordenades, mapes i més |
| **Wink Lookup** | Barra de cerca de destinacions |
| **Wink Search** | Botó icona que obre el selector d'itineraris |
| **Wink Account** | Botó d'inici de sessió / compte d'usuari |
| **Wink Itinerary** | Botó d'itinerari que mostra el viatge actual |
| **Wink Cart** | Botó resum del carret de la compra |
