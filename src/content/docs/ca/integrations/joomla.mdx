L'**Extensió Wink per Joomla** és un paquet que instal·la un plugin de sistema (carrega scripts CDN), un plugin de contingut (processa les etiquetes shortcode als articles) i un component (pàgina de configuració d'administració).

<LinkCard
  title="wink-travel/wink-joomla-extension on GitHub"
  href="https://github.com/wink-travel/wink-joomla-extension"
  description="Codi font, guia de configuració per a desenvolupadors i instruccions per a contribucions."
/>

## Requisits

- Joomla 4 o 5
- PHP 8.1+

## Instal·lació

<Steps>
1. Descarrega l'últim `pkg_wink-x.x.x.zip` des de la [pàgina de llançaments de GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. A l'administració de Joomla, ves a **Extensions → Gestiona → Instal·la**.
3. Pugeu el fitxer zip i fes clic a **Puja i instal·la**.
4. Ambdós plugins s'activen automàticament. Ves a **Components → Wink Settings**.
5. Introdueix el teu **Client ID** i **Client Secret** i fes clic a **Desa**.
   Ves a [Applications](/ca/portal/settings/applications) per obtenir les teves credencials.
</Steps>

## Inserir components als articles

Utilitza etiquetes en estil shortcode a qualsevol lloc del contingut de l'article:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

El plugin de contingut substitueix aquestes etiquetes pel correcte HTML `<wink-*>` en el moment de renderitzar. El plugin de sistema carrega automàticament els scripts CDN i injecta `<wink-app-loader>` al peu de pàgina.

## Shortcodes disponibles

| Shortcode | Sortida | Descripció |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Targetes d'hotel, graelles, mapes |
| `{wink-lookup}` | `<wink-lookup>` | Barra de cerca de destinacions |
| `{wink-search}` | `<wink-search-button>` | Botó per seleccionar l'itinerari |
| `{wink-account}` | `<wink-account-button>` | Inici de sessió / compte d'usuari |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Botó d'estat de l'itinerari |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Botó del carretó |
