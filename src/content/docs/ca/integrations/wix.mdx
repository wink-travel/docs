L'**aplicació Wink Wix** afegeix components web de Wink al teu lloc Wix com a widgets nadius que pots arrossegar i deixar anar a l'Editor de Wix. No es requereix programació per als propietaris del lloc.

<LinkCard
  title="wink-travel/wink-wix-app on GitHub"
  href="https://github.com/wink-travel/wink-wix-app"

## Instal·lar

<Steps>
1. Ves al [Wix App Market](https://www.wix.com/app-market) i cerca **Wink Travel**.
2. Fes clic a **Afegeix al lloc** i segueix les instruccions per autoritzar l'aplicació.
3. Un cop instal·lada, obre **Tauler → Configuració de Wink** i introdueix el teu **Client ID** i **Client Secret**.
   Ves a [Aplicacions](/ca/portal/settings/applications) per recuperar les teves credencials.
4. Fes clic a **Desa** i després a **Prova la connexió** per confirmar que Wink està connectat.
</Steps>

## Afegeix un component a una pàgina

<Steps>
1. Obre l'**Editor de Wix** i ves a la pàgina on vols afegir contingut de Wink.
2. Fes clic a **Afegeix (+) → Aplicacions → Wink Travel** per veure tots els widgets disponibles.
3. Arrossega un widget a la pàgina — per exemple, **Wink Content** per mostrar una graella d'hotels.
4. Al panell de configuració del widget, selecciona el **Tipus de disseny** i enganxa el **Layout ID** de Wink Studio.
5. Fes clic a **Publica** per fer efectius els canvis.
</Steps>

## Widgets disponibles

| Widget | Propòsit |
|---|---|
| **Wink Content** | Mostra targetes d'hotels, graelles classificades, mapes i més |
| **Wink Lookup** | Barra de cerca per a destinacions i hotels |
| **Wink Search** | Botó d'icona que obre el selector d'itinerari |
| **Wink Account** | Botó d'inici de sessió / compte d'usuari |
| **Wink Itinerary** | Botó d'itinerari que mostra el viatge actual |
| **Wink Cart** | Botó resum del carret de la compra |
