Studio ofereix diversos widgets per inserir que pots afegir al teu propi lloc web per impulsar les reserves.

Per accedir als widgets per inserir, ves a `Eines > Diversos` des de la barra lateral.

## Widgets disponibles

| Widget | Descripció |
| -- | -- |
| **Botó de compte** | Un botó amb marca que enllaça a la teva pàgina d’afiliat |
| **Botó d’itinerari** | Un botó que obre el selector d’itinerari |
| **Formulari d’itinerari** | Un formulari en línia per seleccionar dates i convidats |
| **Botó de cerca** | Un botó que obre la cerca d’allotjaments |
| **Botó de consulta** | Un botó per consultar inventari específic |

## Com inserir

Cada widget proporciona un fragment de codi que pots copiar i enganxar al HTML del teu lloc web. El fragment inclou el teu **Client ID** i **Customization ID** perquè el widget s’enllaci automàticament amb el teu compte d’afiliat.

:::note
Si utilitzes WordPress, també oferim un plugin de WordPress per a una integració més fàcil.
:::
