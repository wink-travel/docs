Un usuari de Wink pot triar que gestionis un dels seus comptes d’allotjament, afiliat o agent de viatges. Quan això passa, rebràs un correu electrònic amb una sol·licitud d’invitació.

import { Steps } from '@astrojs/starlight/components';

Per veure les teves invitacions actuals de Wink, segueix aquests passos:

<Steps>
 1. Inicia sessió a [my.wink.travel](https://my.wink.travel).
 2. Fes clic a `Invitacions` al menú lateral esquerre.
 3. Veureu una llista de totes les invitacions actuals.
 4. Aquesta secció mostrarà totes les teves sol·licituds.
 5. Tria `Acceptar` o `Rebutjar` la invitació.
 6. Si has acceptat la invitació, el compte apareixerà a la secció `Comptes`.
</Steps>
