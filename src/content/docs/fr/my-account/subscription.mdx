La page d'abonnement vous permet de gérer votre plan et les propriétés licenciées sous votre compte.

## Plans disponibles

Parcourez les niveaux d'abonnement disponibles affichés via notre tableau des tarifs. Chaque plan inclut différents ensembles de fonctionnalités et limites de propriétés. Sélectionnez le plan qui correspond le mieux à vos besoins.

## Abonnement actuel

Si vous avez déjà un abonnement actif, cette section affiche les détails et le statut de votre plan actuel.

## Licences

Le tableau des licences montre toutes les propriétés provisionnées sous votre abonnement. Vous pouvez :

- Voir quelles propriétés sont actuellement licenciées.
- Ajouter de nouvelles licences de propriété en utilisant le bouton `Add license`.
- Gérer la facturation via le lien du portail client Stripe.

:::note
La gestion de la facturation (moyens de paiement, factures, modifications de plan) se fait via le portail client Stripe.
:::
