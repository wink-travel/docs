*Temps approximatif pour compléter : ~2 minutes*

D'autres membres peuvent également [inviter votre propriété](/fr/studio/search) à rejoindre Wink. Vous recevrez un e-mail d'invitation de notre part avec une demande de `lead` que vous pouvez réclamer et utiliser pour finaliser la création de la propriété.

Pour voir vos réclamations Wink actuelles, suivez ces étapes :

<Steps>
 1. Connectez-vous à [my.wink.travel](https://my.wink.travel).
 2. Cliquez sur `Réclamations` dans la barre latérale gauche.
 3. Vous verrez une liste de toutes les réclamations en cours.
 4. Choisissez la réclamation sur laquelle vous souhaitez travailler et complétez les champs manquants du formulaire pour créer votre compte.
</Steps>
