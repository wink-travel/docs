La connaissance est le moteur de Wink. Elle vous indique comment tout le monde, et tout, fonctionne sur Wink, jusqu’à l’heure près. Avant Wink, les hôtels s’abonnaient à un fournisseur tiers, et lui payaient beaucoup d’argent, pour accéder à ce que l’équipe de gestion des revenus appelle `le comp set` ; ce qui se traduit par : `Comment ma propriété se compare-t-elle à une autre propriété ?`.

Sur Wink, vous avez accès gratuitement aux analyses au niveau de la plateforme. Cela vous permet de suivre la performance de n’importe quel affilié, agent de voyage ou propriété selon n’importe quelle métrique que vous souhaitez... autant que vous voulez. Vous pouvez aussi accéder à des tendances plus larges par région pour voir quel type de produit est vendu où et quand. Entre de bonnes mains, cette connaissance peut améliorer les compétences de vente de tout utilisateur Wink.

## Accéder aux graphiques

Vous pouvez accéder à vos graphiques depuis n’importe lequel de nos sites web. Pour accéder à vos graphiques analytiques, suivez ces étapes :

<Steps>
1. Vous devez être connecté à l’un de nos sites web.
2. Cliquez sur votre icône de profil, en haut à droite, un menu apparaîtra.
3. Cliquez sur le lien `Analytics`.
4. Vous serez redirigé vers vos graphiques.
</Steps>

:::note
Par défaut, nous créons des graphiques simples pour chaque propriété, agent de voyage et compte affilié que vous créez afin que vous puissiez facilement suivre la performance de vos propres comptes.
:::

## Créer un graphique

Lorsque vous souhaitez commencer à créer vos propres graphiques, allez à vos graphiques, comme décrit ci-dessus, et cliquez sur le bouton `Create chart`.

Voici les étapes simples pour créer un graphique analytique, suivies d’instructions plus détaillées ci-dessous.

<Steps>
1. **Série temporelle** Choisissez une série temporelle pour votre graphique. *Ex. Quotidien*.
2. **Type de suivi** Choisissez si vous souhaitez suivre une plage de dates fixe ou continue. *Ex. `Continuous`*.
    - Continu : Sélectionnez les unités de temps. *Ex. 7 jours*
    - Fixe : Entrez la plage de date / heure *Ex. 22 sept. 2023 16:00 - 29 sept. 2023 16:00*.
3. **Afficher la devise** Si vous traitez des montants dans votre graphique, choisissez dans quelles devises vous souhaitez les voir. *Ex. USD*.
4. **Suivre une propriété** Suivez une propriété spécifique. *Optionnel*
5. **Suivre un canal de vente** Suivez la localisation du canal de vente par continent, pays ou ville. *Optionnel*
6. **Filtrer le type de canal de vente** Filtrez sur un type spécifique de canal de vente. *Ex. Influenceur*. *Optionnel*
7. **Suivre un canal de vente spécifique** Suivez un canal de vente spécifique. *Optionnel*
8. **Suivre un voyageur** Suivez la localisation du voyageur par continent, pays ou ville. *Optionnel*
9. **Choisir les points de données** Choisissez les points de données que vous souhaitez suivre. *Ex. Réservations & Annulations*
10. **Grouper les données** Au lieu de filtrer sur des propriétés ou des canaux de vente, vous pouvez aussi grouper pour agréger les points de données. *Ex. Grouper par canaux de vente*
11. **Nom du graphique** Donnez un nom descriptif à votre graphique. *Ex. Réservations Travelbug à l’heure*
12. Cliquez sur le bouton `Save`.
</Steps>

## Comment ça fonctionne

Les graphiques sont traités en 4 étapes. Ces étapes sont décrites ci-dessous.

### Type de série temporelle

En choisissant une plage de dates fixe ou une tranche de temps continue, vous commencez à vous concentrer sur le QUAND vous souhaitez voir les données.

### Suivi / Filtrage

En choisissant une propriété, un canal de vente, etc., vous commencez à vous concentrer sur QUI vous souhaitez voir dans les données. Vous filtrez effectivement tout ce que vous ne voulez pas voir.

### Points de données

En sélectionnant des points de données, vous commencez à vous concentrer sur QUOI vous souhaitez voir dans les données.

:::note
Les points de données doivent être « synergiques » de manière à pouvoir s’afficher ensemble sur un graphique. Par exemple, vous n’incluriez pas « Pages vues » et « Revenu net total » sur le même graphique.
:::

### Regroupement <Badge text="Nouveau" variant="note" />

Le regroupement est un moyen puissant d’agréger et de se concentrer sur les données.

:::tip
Par exemple, si vous êtes une propriété et que vous souhaitez suivre combien de réservations chacun de vos canaux de vente a réalisé, vous filtreriez sur votre propriété (ci-dessus), sélectionneriez de suivre les « Réservations » (ci-dessus) et vous grouperiez par « Canal de vente » dans la section Regroupement.
:::

## Supprimer un graphique

Pour supprimer un graphique, suivez ces étapes :

<Steps>
1. Cliquez sur le lien `Actions` qui se trouve sous chaque graphique.
2. Cliquez sur le bouton `Remove`.
</Steps>

Votre graphique a été supprimé.
