Si vous souhaitez mettre à jour votre profil utilisateur pour votre compte utilisateur Wink, suivez ces étapes :

<Steps>
1. Rendez-vous sur [my.wink.travel](https://my.wink.travel).
2. Connectez-vous en cliquant sur `Sign-in / Register` en haut à droite.
3. Une fois connecté, cliquez sur votre icône de profil, en haut à droite, un menu apparaîtra.
4. Cliquez sur le lien `Profile`.
5. Suivez les instructions pour mettre à jour votre `e-mail`, `prénom` et `nom`.
6. Vous pouvez également télécharger une nouvelle `photo de profil` de vous-même.
7. Cliquez sur `Save`.
</Steps>

<Steps>
1. Vous devez être connecté à l’un de nos sites web.
2. Cliquez sur votre icône de profil, en haut à droite, un menu apparaîtra.
3. Cliquez sur le lien `Profile`.
4. Suivez les instructions pour mettre à jour votre `e-mail`, `prénom` et `nom`.
5. Vous pouvez également télécharger une nouvelle `photo de profil` de vous-même.
6. Cliquez sur `Update`.
</Steps>

:::note
Votre nouvelle photo de profil s’affichera la prochaine fois que vous vous déconnecterez complètement de Wink.
:::
