Une agence est un compte Wink classique que vous, en tant que propriétaire, pouvez assigner à votre compte à des fins de représentation. Voici quelques raisons pour lesquelles vous voudriez qu’une agence vous représente :

- Gestion complète / partielle de votre compte Wink par un tiers.
- La propriété a un accord existant avec sa société de gestion et souhaite que Wink gère l’aspect financier de leur arrangement des fonds entrants via Wink.
- L’agence de gestion immobilière effectue tout le travail en ligne au nom de ses propriétés.
- Agence de talents représentant un influenceur / créateur de contenu.

## Assigner une agence

Voici comment assigner une agence à votre compte.

<Steps>
 1. Sélectionnez le compte avec lequel vous souhaitez travailler depuis [Accounts](https://my.wink.travel).
 2. Cliquez sur `Actions` en bas de la carte du compte et choisissez `Gérer le compte`.
 3. Cliquez sur l’onglet `Agency` sur la page suivante.
 4. Dans la section `Agency`, saisissez le nom de l’agence que vous souhaitez vous représenter.
 5. Choisissez les frais d’agence dans le menu déroulant.
 5. Ajoutez éventuellement des règles d’expiration.
 7. Cliquez sur le bouton `Save` pour continuer.
 </Steps>

 :::note
Vous devez être le propriétaire de la propriété pour assigner une agence.
 :::

 :::tip[Exemple]
Les frais d’agence sont calculés à partir de la réservation.

Réservation = 100 $, Frais d’agence = 1 % => 100 * 0,01 = 1 $
 :::

## Désassigner une agence

Voici comment désassigner une agence de votre compte.

<Steps>
 1. Sélectionnez le compte avec lequel vous souhaitez travailler depuis [Accounts](https://my.wink.travel).
 2. Cliquez sur `Actions` en bas de la carte du compte et choisissez `Gérer le compte`.
 3. Cliquez sur l’onglet `Agency` sur la page suivante.
 4. Dans la section `Agency`, cliquez sur l’icône <Badge text="x" variant="danger" size="small" /> à droite du menu déroulant de l’agence.
 5. Cliquez sur le bouton `Save` pour continuer.
 </Steps>
