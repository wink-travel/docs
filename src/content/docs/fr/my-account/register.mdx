Suivez ces étapes :

<Steps>
1. Rendez-vous sur [my.wink.travel](https://my.wink.travel)
2. Cliquez sur `Sign-In / Register` en haut à droite
3. Vous arriverez sur un formulaire de connexion. Cliquez sur le lien `Register` sous les champs du formulaire.
4. Remplissez le formulaire d'inscription et cliquez sur `Register`. Vous êtes redirigé vers la page de connexion.
5. Une fois connecté, vous serez redirigé vers Mon Compte où vous pourrez créer votre premier compte.
</Steps>

Créer un compte utilisateur Wink vous donne accès à toutes les fonctionnalités de la plateforme. Vous pouvez créer un nombre illimité de propriétés et de comptes affiliés.
