Si vous souhaitez changer votre mot de passe, ou si vous l'avez oublié, suivez ces étapes :

## Mettre à jour le mot de passe

<Steps>
1. Rendez-vous sur [my.wink.travel](https://my.wink.travel).
1. Connectez-vous en cliquant sur `Sign-in / Register` en haut à droite.
2. Cliquez sur votre icône de profil, en haut à droite, un menu apparaîtra.
3. Cliquez sur le lien `Change password`.
4. Suivez les instructions pour changer votre mot de passe.
5. Cliquez sur `Change password` pour continuer.
</Steps>

La prochaine fois que vous vous connecterez, vous devrez utiliser votre nouveau mot de passe.

## Mot de passe oublié

Parfois, il est nécessaire de réinitialiser le mot de passe utilisateur.

Voici les étapes pour réinitialiser votre mot de passe utilisateur :

<Steps>
1. Cliquez sur le lien `Forgot password` depuis la page de connexion.
2. Saisissez le nom d'utilisateur / e-mail du compte que vous souhaitez réinitialiser.
3. Cliquez sur le bouton `Forgot password`. *Vous êtes redirigé vers le formulaire de réinitialisation du mot de passe.*
4. Vérifiez votre e-mail.
</Steps>

<Steps>
1. Collez le code que vous avez reçu par e-mail dans le nouveau formulaire avec votre nouveau mot de passe.
2. Confirmez votre nouveau mot de passe.
3. Cliquez sur le bouton `Reset password`. *Vous êtes redirigé vers la page de connexion.*
</Steps>
