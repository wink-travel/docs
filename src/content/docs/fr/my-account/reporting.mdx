Wink propose un système de rapports qui vous permet de générer et d’exporter des données de votre compte. Les rapports sont disponibles au format CSV pour une importation facile dans des tableurs, des outils d’analyse ou vos propres systèmes.

## Types de rapports

Vous pouvez générer des rapports pour les catégories de données suivantes :

| Type de rapport | Ce qu’il inclut |
|---|---|
| **Propriété** | Détails de la propriété, statut de l’annonce, résumé de l’inventaire. |
| **Utilisateur** | Comptes utilisateurs, rôles, résumé des activités. |
| **Réservation** | Historique des réservations, statuts, revenus, détails des clients. |
| **Compte** | Informations sur le compte, soldes, historique des transactions. |

## Comment générer un rapport

1. Accédez à la section rapports dans le tableau de bord de votre compte.
2. Sélectionnez le **type de rapport** que vous souhaitez générer.
3. Configurez les **filtres** pour affiner les données (par exemple, plage de dates, statut, propriété).
4. Choisissez les **champs** que vous souhaitez inclure dans l’export. Chaque type de rapport propose un ensemble de champs disponibles à sélectionner.
5. Cliquez sur **Exporter** pour télécharger le rapport au format CSV.

## Champs personnalisés

Les rapports prennent en charge la sélection personnalisée des champs. Vous choisissez les colonnes à inclure dans votre export, pour obtenir exactement les données dont vous avez besoin — ni plus, ni moins.

## Cas d’utilisation

- **Rapprochement des revenus** — Exportez les rapports de réservation avec les détails financiers pour les rapprocher avec votre système comptable.
- **Analyse de performance** — Exportez les rapports de propriété pour comparer la performance des annonces dans votre portefeuille.
- **Informations sur les clients** — Exportez les données de réservation pour comprendre la démographie des voyageurs et les tendances de réservation.
- **Audit de compte** — Exportez les données de compte pour la conformité ou la revue interne.
