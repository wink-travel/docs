Un utilisateur Wink peut choisir de vous laisser gérer l’un de ses comptes de propriété, d’affilié ou d’agent de voyage. Lorsque cela se produit, vous recevrez un e-mail avec une demande d’invitation.

import { Steps } from '@astrojs/starlight/components';

Pour voir vos invitations Wink actuelles, suivez ces étapes :

<Steps>
 1. Connectez-vous à [my.wink.travel](https://my.wink.travel).
 2. Cliquez sur `Invites` dans la barre latérale gauche.
 3. Vous verrez une liste de toutes les invitations en cours.
 4. Cette section affichera toutes vos demandes.
 5. Choisissez d’`Accepter` ou de `Refuser` l’invitation.
 6. Si vous avez accepté l’invitation, le compte apparaîtra dans la section `Accounts`.
</Steps>
