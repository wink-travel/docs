La connaissance est l'élément vital de Wink. Elle vous informe des performances de chacun et de chaque élément sur Wink, heure par heure. Avant Wink, les hôtels s'abonnaient à un fournisseur tiers et le payaient très cher pour accéder à ce que l'équipe de gestion des revenus appelle…`the comp set`; ce qui se traduit par :`How does my property compare with another property?`.

Sur Wink, vous avez accès gratuitement à des analyses à l'échelle de la plateforme. Vous pouvez ainsi suivre les performances de vos affiliés, agences de voyages ou établissements selon les indicateurs de votre choix… autant que vous le souhaitez. Vous pouvez également accéder à des tendances plus larges par région pour identifier les produits vendus, où et quand. Entre de bonnes mains, ces informations peuvent améliorer les compétences commerciales de tout utilisateur de Wink.

## Accéder aux graphiques

Vous pouvez accéder à vos graphiques depuis n'importe lequel de nos sites web. Pour accéder à vos graphiques analytiques, suivez ces étapes :

<Steps>
  1. Vous devez être connecté à l'un de nos sites Web.
  2. Cliquez sur l’icône de votre profil, dans le coin supérieur droit, et un menu apparaîtra.
  3. Cliquez sur le `Analytics` lien.
  4. Vous serez redirigé vers vos cartes.
</Steps>


:::note
Par défaut, nous créons des graphiques simples pour chaque établissement, agence de voyages et compte affilié que vous créez, afin que vous puissiez facilement suivre les performances de vos comptes.
:::

## Créer un graphique

Lorsque vous souhaitez commencer à créer vos propres graphiques, accédez à vos graphiques, comme décrit ci-dessus, et cliquez sur le bouton`Create chart` bouton.

Vous trouverez ci-dessous les étapes simples pour créer un graphique d’analyse, suivies d’instructions plus détaillées ci-dessous.

<Steps>
  1. **Séries chronologiques** Choisissez une série chronologique pour votre graphique. *Par exemple, quotidiennement*.
  2. **Type de suivi** Choisissez de suivre une plage de dates fixe ou en continu. *Par exemple `Continuous`*.
     * Continu : sélectionnez les unités de temps. *Par exemple, 7 jours*
     * Corrigé : Saisir une plage de dates/heures *Par exemple, du 22 septembre 2023 à 16h00 au 29 septembre 2023 à 16h00*.
  3. **Afficher la devise** Si vous traitez des montants dans votre graphique, choisissez dans quelles devises vous souhaitez les afficher. *Par exemple, USD*.
  4. **Propriété de la piste** Suivre une propriété spécifique. *Facultatif*
  5. **Suivre le canal de vente** Suivez l'emplacement du canal de vente par continent, pays ou ville. *Facultatif*
  6. **Filtrer le type de canal de vente**Filtrer sur un type spécifique de canal de vente. *Par exemple, un influenceur*. *Facultatif*
  7. **Suivre un canal de vente spécifique** Suivre un canal de vente spécifique. *Facultatif*
  8. **Voyageur sur piste** Suivez l'emplacement du voyageur par continent, pays ou ville. *Facultatif*
  9. **Choisir des points de données** Choisissez les points de données que vous souhaitez suivre. *Par exemple, réservations et annulations*
  10. **Données de groupe** Au lieu de filtrer sur les propriétés ou les canaux de vente, vous pouvez également regrouper pour agréger sur des points de données. *Par exemple, groupe sur les canaux de vente*
  11. **Nom du graphique** Donnez à votre graphique un nom descriptif. *Par exemple, les réservations Travelbug sur l'horaire*
  12. Cliquez `Save` bouton.
</Steps>


## Comment ça marche

Le traitement des graphiques se déroule en quatre étapes, décrites ci-dessous.

### Type de série chronologique

En choisissant une plage de dates fixe ou une tranche de temps continue, vous commencez à vous concentrer sur le moment où vous souhaitez voir les données.

### Suivi / Filtrage

En choisissant une propriété, un canal de vente, etc., vous commencez à vous concentrer sur les personnes dont vous souhaitez consulter les données. Vous filtrez ainsi efficacement tout ce que vous ne souhaitez pas voir.

### Points de données

En sélectionnant des points de données, vous commencez à vous concentrer sur CE pour quoi vous voulez voir les données.

:::note
Les points de données doivent être « synergiques » afin de pouvoir s'intégrer les uns aux autres dans un graphique. Par exemple, vous ne devez pas inclure les « Pages vues » et le « Revenu net total » dans le même graphique.
:::

### Regroupement<Badge text="New" variant="note" />


Le regroupement est un moyen puissant d’agréger et de se concentrer sur les données.

:::astuce
Par exemple, si vous êtes un établissement et que vous souhaitez suivre le nombre de réservations effectuées par chacun de vos canaux de vente, filtrez votre établissement (ci-dessus), sélectionnez « Suivi des réservations » (ci-dessus) et regroupez les résultats par « Canal de vente » dans la section « Regroupement ».

## Supprimer un graphique

Pour supprimer un graphique, suivez ces étapes :

<Steps>
  1. Cliquez sur le `Actions` lien qui existe sous chaque graphique.
  2. Cliquez sur le `Remove` bouton
</Steps>


Votre graphique a été supprimé.
