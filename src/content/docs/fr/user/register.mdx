Vous pouvez créer un compte utilisateur via n’importe lequel de nos portails. Une fois créé, ce compte peut être utilisé sur toutes les applications et sites web de Wink.

## Voyageurs

Suivez ces étapes :

<Steps>
1. Rendez-vous sur [https://ota.wink.travel](https://ota.wink.travel)
2. Cliquez sur `Sign-In / Register` en haut à droite
3. Il vous sera demandé de saisir les informations de base de l’utilisateur. Remplissez tous les champs obligatoires et cliquez sur `Save`.
4. Vous pouvez maintenant vous connecter à Wink avec votre nom d’utilisateur (e-mail) et votre mot de passe.
5. Une fois connecté, le système vous redirige vers la page d’où vous venez.
</Steps>

## Inscription

Créer un compte utilisateur Wink vous donne accès à toutes les fonctionnalités de la plateforme. Vous pouvez créer un nombre illimité de propriétés et de comptes affiliés.

<Steps>
1. La première fois que vous êtes dirigé vers notre serveur d’autorisation, il vous sera demandé de saisir votre nom d’utilisateur et votre mot de passe.
2. Cliquez plutôt sur le bouton `Create account`.
3. Remplissez tous les champs obligatoires et cliquez sur `Register` pour continuer. Cela vous ramène au formulaire de connexion.
4. Saisissez votre e-mail / nom d’utilisateur et votre mot de passe pour vous connecter.
5. Vous êtes redirigé vers le site d’où vous venez.
</Steps>
