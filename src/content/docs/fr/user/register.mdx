Vous pouvez créer un compte utilisateur via n'importe lequel de nos portails. Une fois créé, ce compte est utilisable sur toutes les applications et tous les sites web Wink.

## Les voyageurs

Suivez ces étapes :

<Steps>
  1. Rendez-vous sur [https://ota.wink.travel](https://ota.wink.travel)
  2. Cliquez sur `Sign-In / Register` dans le coin supérieur droit
  3. Vous serez invité à saisir vos informations d'utilisateur. Remplissez tous les champs obligatoires et cliquez sur `Save`.
  4. Vous pouvez désormais vous connecter à Wink avec votre nom d'utilisateur (e-mail) et votre mot de passe.
  5. Une fois connecté, le système vous redirige vers l'endroit d'où vous venez.
</Steps>

## Inscription

Créer un compte utilisateur Wink vous donne accès à toutes les fonctionnalités de la plateforme. Vous pouvez créer un nombre illimité de propriétés et de comptes affiliés.

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="Se connecter" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Formulaire de connexion*</figcaption>
</figure>


<Steps>
  1. La première fois que vous êtes dirigé vers notre serveur d'autorisation, il vous sera demandé de saisir votre nom d'utilisateur et votre mot de passe.
  2. Au lieu de cela, cliquez sur `Create account` bouton.
  3. Remplissez tous les champs obligatoires et cliquez sur `Register` pour continuer. Cela vous ramène au formulaire de connexion.
  4. Entrez votre e-mail / nom d'utilisateur et votre mot de passe pour vous connecter.
  5. Vous êtes redirigé vers le site d'où vous venez.
</Steps>
