Si vous souhaitez mettre à jour votre profil utilisateur pour votre compte utilisateur Wink, suivez ces étapes :

<Steps>
  1. Vous devez être connecté à l'un de nos sites Web.
  2. Cliquez sur l'icône de votre profil, dans le coin supérieur droit et un menu apparaîtra.
  3. Cliquez sur le `Profile` lien.
  4. Suivez les instructions pour mettre à jour votre `e-mail`, `first` et `last name`.
  5. Vous pouvez également télécharger un nouveau `profile picture` de toi-même.
  6. Cliquez `Save`.
</Steps>


:::note
Votre nouvelle photo de profil s'affichera la prochaine fois que vous vous déconnecterez complètement de Wink.
:::
