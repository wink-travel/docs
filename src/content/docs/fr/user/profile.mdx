Si vous souhaitez mettre à jour votre profil utilisateur pour votre compte Wink, suivez ces étapes :

<Steps>
1. Vous devez être connecté à l’un de nos sites web.
2. Cliquez sur votre icône de profil, en haut à droite, un menu apparaîtra.
3. Cliquez sur le lien `Profil`.
4. Suivez les instructions pour mettre à jour votre `e-mail`, `prénom` et `nom`.
5. Vous pouvez également télécharger une nouvelle `photo de profil` de vous-même.
6. Cliquez sur `Enregistrer`.
</Steps>

:::note
Votre nouvelle photo de profil s’affichera la prochaine fois que vous vous déconnecterez complètement de Wink.
:::
