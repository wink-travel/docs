Un utilisateur Wink peut choisir de vous confier la gestion de l'un de ses comptes d'établissement, d'affilié ou d'agent de voyages. Vous recevrez alors un e-mail de demande d'invitation.

import { Steps } from '@astrojs/starlight/components';

Pour voir vos invitations Wink actuelles, suivez ces étapes :

<Steps>
  1. Vous devez être connecté à l'un de nos sites Web.
  2. Cliquez sur l'icône de votre profil, dans le coin supérieur droit et un menu apparaîtra.
  3. Cliquez sur le `Invites` lien.
  4. Interagissez avec vos invitations soit en `Accepting` ou `Rejecting` eux.
</Steps>
