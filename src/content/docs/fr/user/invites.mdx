Un utilisateur Wink peut choisir de vous laisser gérer l'un de ses comptes de propriété, d'affilié ou d'agent de voyage. Lorsque cela se produit, vous recevrez un e-mail avec une demande d'invitation.

import { Steps } from '@astrojs/starlight/components';

Pour voir vos invitations Wink actuelles, suivez ces étapes :

<Steps>
1. Vous devez être connecté à l'un de nos sites web.
2. Cliquez sur votre icône de profil, en haut à droite, un menu apparaîtra.
3. Cliquez sur le lien `Invites`.
4. Interagissez avec vos invitations en les `Acceptant` ou en les Rejetant`.
</Steps>
