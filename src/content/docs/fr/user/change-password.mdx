Si vous souhaitez modifier votre mot de passe pour votre compte utilisateur Wink, suivez ces étapes :

## Mettre à jour le mot de passe

<Steps>
  1. Vous devez être connecté à l'un de nos sites Web.
  2. Cliquez sur l'icône de votre profil, dans le coin supérieur droit et un menu apparaîtra.
  3. Cliquez sur le `Change password` lien.
  4. Suivez les instructions pour changer votre mot de passe.
  5. Cliquez `Save`.
</Steps>


La prochaine fois que vous vous connecterez, vous aurez besoin de votre nouveau mot de passe.

## Mot de passe oublié

Parfois, il est nécessaire de réinitialiser le mot de passe de l'utilisateur.

Voici les étapes pour réinitialiser votre mot de passe utilisateur :

<Steps>
  1. Cliquez sur le `Forgot password` lien depuis la page de connexion.
  2. Saisissez le nom d'utilisateur / e-mail du compte que vous souhaitez réinitialiser.
  3. Cliquez `Forgot password` bouton. *Vous êtes redirigé vers le formulaire de réinitialisation du mot de passe.*
  4. Vérifiez votre e-mail.
</Steps>

<Steps>
  1. Collez le code que vous avez reçu dans l'e-mail dans le nouveau formulaire avec votre nouveau mot de passe.
  2. Confirmez votre nouveau mot de passe.
  3. Cliquez `Reset password` bouton. *Vous êtes redirigé vers la page de connexion.*
</Steps>

Votre mot de passe a été réinitialisé avec succès. Vous pouvez vous connecter à Wink avec vos nouveaux identifiants.
