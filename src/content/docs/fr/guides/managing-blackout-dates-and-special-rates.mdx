Tous les jours ne sont pas tarifés de la même manière. Les événements, les jours fériés, les saisons creuses et les dernières disponibilités nécessitent des tarifs différents. Ce guide vous montre comment utiliser les tarifs spéciaux et les restrictions des plans tarifaires pour gérer ces situations.

## Tarification événementielle

Lorsqu’une conférence, un festival ou un événement local génère une forte demande, vous souhaiterez augmenter les tarifs pour ces dates.

<Steps>
1. Allez dans `Monetize > Special Rates` dans votre Extranet.
2. Cliquez sur `Create a new special rate`.
3. Sélectionnez le tarif principal et le type de chambre à remplacer.
4. Définissez la plage de dates couvrant la période de l’événement.
5. Saisissez les prix journaliers ajustés (plus élevés).
6. Cliquez sur `Save`.
</Steps>

Le tarif spécial remplace le tarif principal uniquement pour ces dates. Votre tarification habituelle reste inchangée en dehors de la période de l’événement.

## Ajustements saisonniers

Pour des périodes plus longues (haute saison, saison intermédiaire, basse saison), vous pouvez soit :

- **Utiliser des tarifs spéciaux** — Créez des remplacements pour chaque saison avec des tarifs ajustés.
- **Mettre à jour les tarifs principaux** — Ajustez directement les tarifs de base pour ces périodes.

Les tarifs spéciaux sont mieux adaptés aux ajustements temporaires, tandis que la mise à jour des tarifs principaux est pertinente pour des schémas annuels prévisibles.

## Tarifs de dernière minute

Pour remplir les chambres invendues à l’approche de la date d’arrivée :

<Steps>
1. Créez un nouveau **plan tarifaire** avec une restriction de réservation anticipée (par exemple, réservation au plus tard 3 jours à l’avance).
2. Associez-le à un **tarif spécial** offrant une réduction pour les dates ciblées.
3. Ce plan tarifaire n’apparaîtra qu’aux voyageurs réservant dans la fenêtre de dernière minute.
</Steps>

## Dates d’exclusion

Pour bloquer certaines dates d’un plan tarifaire spécifique (par exemple, pas de tarifs promotionnels en haute saison) :

<Steps>
1. Allez dans `Monetize > Rate Plans` et sélectionnez le plan tarifaire.
2. Rendez-vous dans la section **Restrictions**.
3. Ajoutez une restriction **Sell date** pour limiter les dates de réservation du tarif.
4. Ou utilisez les restrictions **Stay date** pour limiter les dates de séjour couvertes par le tarif.
</Steps>

Par exemple, vous pouvez restreindre votre plan tarifaire « Early Bird » pour qu’il ne soit pas disponible pour les réservations en décembre (haute saison), tandis que votre plan tarifaire régulier reste ouvert.

## Combinaison des restrictions

Les restrictions peuvent être combinées pour un contrôle précis :

| Restriction | Ce qu’elle contrôle |
| -- | -- |
| **Sell dates** | Quand le tarif peut être réservé |
| **Stay dates** | Quelles dates de séjour le tarif couvre |
| **Available days** | Quels jours de la semaine le tarif apparaît |
| **Arrival days** | Quels jours le client doit arriver |
| **Departure days** | Quels jours le client doit partir |
| **Required days** | Le client doit séjourner au moins un de ces jours |
| **Advance booking** | Combien de temps à l’avance la réservation doit être faite |
| **Length of stay** | Nombre minimum et maximum de nuits |
