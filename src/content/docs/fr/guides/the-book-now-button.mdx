Le **RÉSERVEZ MAINTENANT**Le bouton est un terme omniprésent dans le secteur de l'hôtellerie. Il désigne l'intégration de l'hôtel avec un fournisseur qui connaît les disponibilités et les tarifs de l'établissement et permet aux voyageurs de réserver une chambre. Le bouton est le moyen le plus simple pour les hôtels de vendre des chambres sur leur site web et selon leurs propres conditions.

*De nombreux hôtels utilisent le moteur de réservation intégré proposé par leur channel manager. Nous recommandons aux hôtels de se renseigner et de comparer les moteurs de réservation natifs avant de choisir un moteur en particulier.*

## Public

Cet article s'adresse principalement aux hôtels souhaitant installer notre moteur de réservation gratuit et performant sur leur site web.

Mais… il est également utile à quiconque souhaite utiliser nos technologies sur son site web.

## via le lien

Pour une intégration plus simple, nous recommandons d'utiliser un[Lien partageable](/studio/shareable-links).

Voici les étapes à suivre pour y parvenir :

<Steps>
  1. Accédez à https://extranet.wink.travel
  2. Cliquez `Sign-In / Register` dans le coin supérieur droit et connectez-vous avec votre compte utilisateur.
  3. Sélectionnez la propriété avec laquelle vous souhaitez travailler dans la liste déroulante en haut à gauche.
  4. Cliquez sur `Booking engine` dans la barre de navigation.
  5. La page vous montrera votre lien.
  6. Donnez ce lien à votre webmaster et utilisez-le pour votre **RÉSERVEZ MAINTENANT** bouton.
  7. Tu es ✅.
</Steps>


### Avancé

Le lien est déjà accompagné d'un itinéraire intégré à l'URL, qui provient de votre choix`customization`.
Vous pouvez personnaliser davantage votre lien en :

* Modification de la personnalisation sélectionnée OU
* Remplacer l'itinéraire

Dans cet article, nous nous concentrerons sur ce dernier, car il vous donne plus de liberté pour permettre aux voyageurs de sélectionner un itinéraire sur votre site Web et de le transmettre au moteur de réservation de Wink.

Voici comment transmettre votre propre itinéraire avec l'URL.

Paramètres de requête valides :

* **SD**Pour ajouter une date de début :*sd=2024-08-24*
* **n**Pour ajouter le nombre de nuits :*n=1* OU
* **éd.**&#x55;tiliser la date de fin :*éd.=2024-08-25*
* **rc**Invités :*rc=a2*(Adultes : 2)
* **l**Demander une langue d'affichage :*l=es*
* **c**Demander une devise d'affichage :*c=EUR*
* **p**Ajouter une promotion :*p=ABC123*

L'URL ressemblerait à ceci :

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2\&l=es\&c=EUR

Pour des cas d'utilisation plus avancés, vous pouvez également demander plusieurs configurations de salles à l'aide du`+` signe:

Exemple:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2-ca16q1-ca8q1+a2\&l=es\&c=EUR

* Type de chambre 1 = Adultes : 2, Enfants : 2 = 16 ans + 8 ans.
* Type de chambre 2 = Adultes : 2.

### Exemple

Échantillon **RÉSERVEZ MAINTENANT**bouton HTML :

*Cet exemple suppose que vous utilisez[Bootstrap](https://getbootstrap.com)comme votre bibliothèque CSS principale.*

```html
<a 
  href="https://trvl.as/abc123" 
  target="_blank" 
  class="btn btn-lg btn-primary" 
  title="Book a room"
>
BOOK NOW
</a>
```

### Autres avantages

Il y a d’autres avantages à utiliser l’un de nos liens comme`book now` bouton.

* **Social-friendly**Le lien peut être facilement partagé sur les réseaux sociaux et il s'affiche`dynamic pricing`au-dessus de l'image que vous avez sélectionnée.
* **Court**L'URL est très partageable et vous pouvez l'utiliser dans des endroits comme l'URL de votre bio Instagram.

## via Embed

Certains hôtels souhaitent plus de fonctionnalités et une intégration plus poussée pour leur site web. Cet exemple illustre l'intégration d'un[Carte](/studio/cards)sur leur site Web.

Voici les étapes à suivre pour y parvenir :

<Steps>
  1. Accédez à https://studio.wink.travel
  2. Cliquez `Sign-In / Register` dans le coin supérieur droit et connectez-vous avec votre compte utilisateur.
  3. Sélectionnez la propriété avec laquelle vous souhaitez travailler dans le coin supérieur gauche.
  4. Cliquez sur `Search` depuis la barre de navigation principale.
  5. Tapez le nom de votre propriété et cliquez sur le bouton de recherche.
  6. Cliquez sur le `Actions` lien sur votre carte de résultats de recherche et sélectionnez `Create > Make a Card`.
  7. Cela créera un `Card`.
  8. Un message apparaîtra dans le coin supérieur droit vous indiquant que votre carte est prête et si vous souhaitez la personnaliser davantage.
  9. Cliquez sur `Tools > Cards` depuis la barre de navigation principale et vous verrez votre nouvelle carte.
  10. Cliquez sur le `Actions` lien sur votre carte de lien et cliquez sur `Embed`.
  11. Cela ouvre une nouvelle fenêtre avec un exemple de code HTML que vous pouvez copier.
  12. Pour récupérer votre `Client-ID`, aller à [Applications](/developers/applications).
  13. Pour récupérer votre `Configuration-ID`, aller à [Personnalisation](/studio/customization)
  14. Tu es ✅.
</Steps>


Exemple de carte HTML intégrée :

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*L'exemple HTML ci-dessus n'inclut pas le CSS et le Javascript Wink requis ; juste la carte.*
