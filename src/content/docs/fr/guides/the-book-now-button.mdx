Le bouton **RÉSERVER MAINTENANT** est un terme omniprésent dans l'industrie hôtelière qui fait référence à l'intégration propre de l'hôtel avec un fournisseur qui connaît la disponibilité et les tarifs de l'établissement et permet aux voyageurs de réserver une chambre. Ce bouton est le moyen le plus simple pour les hôtels de pouvoir vendre des chambres sur leur propre site web et selon leurs propres conditions.

*De nombreux hôtels utilisent le moteur de réservation intégré proposé par leur gestionnaire de canaux. Nous recommandons aux hôtels de faire des recherches et de comparer les moteurs de réservation natifs avant de s'engager avec un en particulier.*

## Public cible

Cet article s'adresse principalement aux hôtels qui souhaitent installer notre moteur de réservation gratuit et à fort taux de conversion sur leur site web.
Mais... il est également utile pour toute personne souhaitant utiliser nos technologies sur leurs sites web.

## via un lien

Pour l'intégration la plus simple, nous recommandons d'utiliser un [lien partageable](/fr/studio/shareable-links).

Voici les étapes à suivre :

<Steps>
1. Rendez-vous sur https://extranet.wink.travel
2. Cliquez sur `Connexion / Inscription` en haut à droite et connectez-vous avec votre compte utilisateur.
3. Sélectionnez l'établissement avec lequel vous souhaitez travailler dans le menu déroulant en haut à gauche.
4. Cliquez sur `Moteur de réservation` dans la barre de navigation.
5. La page vous affichera votre lien.
6. Donnez ce lien à votre webmaster et utilisez-le pour votre bouton **RÉSERVER MAINTENANT**.
7. Vous êtes ✅.
</Steps>

### Avancé

Le lien contient déjà un itinéraire intégré dans l'URL, qui provient de votre `personnalisation` choisie.
Vous pouvez personnaliser davantage votre lien en :

- Changeant la personnalisation sélectionnée OU
- En remplaçant l'itinéraire

Dans cet article, nous nous concentrerons sur la seconde option car elle vous donne plus de liberté pour permettre aux voyageurs de sélectionner un itinéraire sur votre site web et de le transmettre au moteur de réservation Wink.

Voici comment transmettre votre propre itinéraire avec l'URL.

Paramètres de requête valides :

- **sd** Pour ajouter une date de début : *sd=2024-08-24*
- **n** Pour ajouter le nombre de nuits : *n=1* OU
- **ed** Utiliser la date de fin : *ed=2024-08-25*
- **rc** Invités : *rc=a2* (Adultes : 2)
- **l** Demander une langue d'affichage : *l=es*
- **c** Demander une devise d'affichage : *c=EUR*
- **p** Ajouter une promotion : *p=ABC123*

L'URL ressemblerait à ceci :

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2&l=es&c=EUR

Pour des cas d'utilisation plus avancés, vous pouvez également demander plusieurs configurations de chambre en utilisant le signe `+` :

Exemple :

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2-ca16q1-ca8q1+a2&l=es&c=EUR

- Type de chambre 1 = Adultes : 2, Enfants : 2 = 16 ans + 8 ans.
- Type de chambre 2 = Adultes : 2.

### Exemple

Exemple de bouton **RÉSERVER MAINTENANT** en HTML :

*Cet exemple suppose que vous utilisez [Bootstrap](https://getbootstrap.com) comme bibliothèque CSS principale.*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Réserver une chambre"
>
RÉSERVER MAINTENANT
</a>
```

### Autres avantages

Il y a d'autres avantages à utiliser l'un de nos liens comme bouton `réserver maintenant`.

- **Compatible réseaux sociaux** Le lien peut être facilement partagé sur les réseaux sociaux et affiche un `tarif dynamique` au-dessus de l'image que vous avez sélectionnée.
- **Court** L'URL est très facile à partager et vous pouvez l'utiliser dans des endroits comme l'URL de votre bio Instagram.

## via intégration

Certains hôtels souhaitent un peu plus de fonctionnalités et une intégration plus poussée pour leur site web. Cet exemple couvrira l'intégration d'une [Carte](/fr/studio/cards) dans leur site.

Voici les étapes à suivre :

<Steps>
1. Rendez-vous sur https://studio.wink.travel
2. Cliquez sur `Connexion / Inscription` en haut à droite et connectez-vous avec votre compte utilisateur.
3. Sélectionnez l'établissement avec lequel vous souhaitez travailler dans le coin supérieur gauche.
4. Cliquez sur `Recherche` dans la barre de navigation principale.
6. Tapez le nom de votre établissement et cliquez sur le bouton de recherche.
7. Cliquez sur le lien `Actions` sur la carte de résultat de recherche et sélectionnez `Créer > Créer une carte`.
8. Cela créera une `Carte`.
9. Un message apparaîtra en haut à droite vous indiquant que votre carte est prête et si vous souhaitez la personnaliser davantage.
10. Cliquez sur `Outils > Cartes` dans la barre de navigation principale et vous verrez votre nouvelle carte.
11. Cliquez sur le lien `Actions` sur votre carte et cliquez sur `Intégrer`.
12. Cela ouvre une nouvelle fenêtre avec un code HTML d'exemple que vous pouvez copier.
13. Pour récupérer votre `Client-ID`, rendez-vous sur [Applications](/fr/developers/applications).
14. Pour récupérer votre `Configuration-ID`, rendez-vous sur [Personnalisation](/fr/studio/customization)
14. Vous êtes ✅.
</Steps>

Exemple de code HTML pour la carte intégrée :

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*L'exemple HTML ci-dessus n'inclut pas le CSS et le Javascript Wink requis ; uniquement la carte.*

Un client voit maintenant les tarifs dynamiques, peut modifier l'itinéraire et cliquer pour réserver directement sur votre site web. Le bouton de réservation vous conduit à la page d'accueil de votre établissement Wink pour finaliser le paiement.
