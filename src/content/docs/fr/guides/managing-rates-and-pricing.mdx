La gestion des tarifs est l’une des parties les plus importantes — et les plus complexes — de la gestion d’un établissement sur Wink. Ce guide décompose la hiérarchie des prix pour que vous puissiez la configurer en toute confiance.

## La hiérarchie des tarifs

Wink utilise une structure tarifaire à trois niveaux :

1. **Tarif de base** — Le tarif journalier de base pour un type de chambre.
2. **Plan tarifaire** — Un ensemble de règles qui encadrent le tarif de base (politique d’annulation, repas, restrictions, modificateurs).
3. **Tarif spécial** — Une dérogation qui ajuste le tarif de base pour des plages de dates spécifiques, des promotions ou des conditions particulières.

Comprendre cette hiérarchie est essentiel. Les tarifs de base posent les fondations, les plans tarifaires définissent la manière dont le tarif est présenté et vendu, et les tarifs spéciaux vous permettent de lancer des promotions ciblées sans modifier le prix de base.

## Étape 1 — Configurer les tarifs de base

Les tarifs de base sont vos prix journaliers de référence par type de chambre.

<Steps>
1. Rendez-vous sur https://extranet.wink.travel et sélectionnez votre établissement.
2. Cliquez sur `Monetize > Master Rates` dans la barre de navigation principale.
3. Cliquez sur `Créer un nouveau tarif de base`.
4. Sélectionnez le type de chambre, le plan tarifaire et la plage de dates que vous souhaitez tarifer.
5. Saisissez les tarifs journaliers pour chaque jour de la plage.
6. Cliquez sur `Enregistrer`.
</Steps>

:::tip
Utilisez la vue calendrier pour repérer rapidement les lacunes dans votre tarification. Un type de chambre ne sera pas réservable pour les dates sans tarif de base.
:::

## Étape 2 — Configurer les plans tarifaires

Les plans tarifaires contrôlent les conditions dans lesquelles un tarif est vendu.

<Steps>
1. Cliquez sur `Monetize > Rate Plans` dans la barre de navigation principale.
2. Créez un nouveau plan tarifaire ou modifiez-en un existant.
3. Configurez les paramètres clés :
   - **Politique d’annulation** — Remboursable, non remboursable ou conditionnelle.
   - **Repas** — Petit-déjeuner, tout compris, etc.
   - **Modificateurs d’occupation** — Remise pour occupation simple, frais supplémentaires pour adultes/enfants.
   - **Restrictions** — Durée du séjour, réservation anticipée, dates de vente, jours d’arrivée, etc.
4. Cliquez sur `Enregistrer`.
</Steps>

Un même type de chambre peut avoir plusieurs plans tarifaires. Par exemple, vous pouvez avoir un plan « Flexible » (remboursable, petit-déjeuner inclus) et un plan « Économique » (non remboursable, chambre seule) — tous deux utilisant le même tarif de base mais présentés différemment aux voyageurs.

## Étape 3 — Créer des tarifs spéciaux (optionnel)

Les tarifs spéciaux remplacent les tarifs de base pour des scénarios spécifiques — promotions, ajustements saisonniers ou tarification événementielle.

<Steps>
1. Cliquez sur `Monetize > Special Rates` dans la barre de navigation principale.
2. Cliquez sur `Créer un nouveau tarif spécial`.
3. Sélectionnez le tarif de base à remplacer et définissez la plage de dates.
4. Saisissez les prix journaliers ajustés.
5. Cliquez sur `Enregistrer`.
</Steps>

Les tarifs spéciaux priment sur les tarifs de base pour les dates qu’ils couvrent.

## Étape 4 — Vérifier avec le testeur de tarifs

Avant la mise en ligne de vos tarifs, utilisez le [Rate Tester](/fr/extranet/rate-tester) pour vérifier que tout est calculé correctement.

<Steps>
1. Cliquez sur `Monetize > Rate Tester` dans la barre de navigation principale.
2. Sélectionnez un canal de vente, une plage de dates et une configuration de chambre.
3. Cliquez sur `Rechercher`.
4. Passez en revue les résultats — vérifiez que les prix, restrictions et politiques d’annulation s’affichent comme prévu.
</Steps>

## Modèles courants

- **BAR (Best Available Rate)** — Un plan tarifaire flexible unique avec votre politique d’annulation standard. La configuration la plus courante.
- **Remise non remboursable** — Un second plan tarifaire avec un prix plus bas et une annulation non remboursable. Attire les voyageurs sensibles au prix.
- **Tarification événementielle** — Utilisez des tarifs spéciaux pour augmenter les prix pendant les périodes de forte demande (conférences, vacances, festivals).
- **Remise pour long séjour** — Ajoutez une restriction de durée de séjour (par exemple, minimum 7 nuits) à un plan tarifaire dédié avec un tarif de base plus bas.
