## Trouver des choses à vendre

Lorsque vous commencez en tant qu’affilié sur Wink, la première chose que vous vous dites est :

> Faisons de l’argent !

Juste après, vous vous demandez :

> Comment ? Quoi ? Quand ? Où ?

Certains affiliés savent déjà ce qu’ils veulent vendre via leurs canaux. Si vous êtes nouveau et souhaitez déterminer quels fournisseurs promouvoir... c’est là que les analyses peuvent être utiles.

Voici un exemple de démarche pour votre toute première fois :

<Steps>
1. Allez sur [Wink Studio](/fr/studio/search) pour rechercher des fournisseurs.
2. Créez un graphique qui suit les réservations dans plusieurs pays. *Vous savez maintenant quels pays performent le mieux.*
3. Créez un autre graphique pour certaines villes dans ces pays performants. *Vous savez maintenant quelles villes performent le mieux.*
4. Recherchez des fournisseurs dans ces villes et trouvez-en quelques-uns qui vous plaisent.
5. Suivez ces fournisseurs en cliquant sur le bouton `Actions > Détails du fournisseur > Suivre la performance` sur leur carte de résultat de recherche *Voir image ci-dessous*.
6. Cela crée un nouveau graphique pour ce fournisseur que vous pouvez suivre.
7. Comparez les performances passées de chaque fournisseur ainsi que le prix moyen et la commission versée.
8. Cela devrait vous donner une poignée de fournisseurs pour commencer à produire.
</Steps>

Maintenant, changez de contexte et commencez à suivre les voyageurs.

<Steps>
1. Suivez quels pays et quelles villes réservent le plus de voyages.
2. Suivez à quels moments ces voyages sont réservés.
</Steps>

Avec ces informations, vous avez une bonne compréhension de ce qui se vend, qui l’achète et quand.

Vous pouvez aller encore plus en détail et découvrir exactement ce qui est réservé. *Exemple : les séances de spa se vendent bien à Berlin, Allemagne, pendant le week-end.*

## Ensemble concurrentiel

Dans le jargon de la gestion hôtelière, un « *comp set* » (abréviation de competitive set) désigne un groupe d’hôtels qu’un hôtel particulier considère comme ses principaux concurrents.
Ces hôtels sont sélectionnés selon des critères tels que l’emplacement, le marché cible, les équipements et les tarifs.
Le comp set est utilisé pour le benchmarking et la comparaison de performance, aidant un hôtel à évaluer sa propre position sur le marché.

Les comp sets sont disponibles pour tous les hôtels, sur la plateforme Wink, gratuitement. Vous pouvez suivre la performance de n’importe quel hôtel sur notre plateforme en utilisant ces indicateurs :

- Visites de page
- Visites de carte
- Visites de marqueur sur la carte
- Réservations
- Nuits de chambre
- Salles de réunion
- Restaurants
- Spas
- Activités
- Attractions
- Lieux
- Services en chambre
- Extras
- Revenu chambre
- Revenu moyen par chambre
- Revenu salles de réunion
- Revenu restaurants
- Revenu spas
- Revenu activités
- Revenu attractions
- Revenu lieux
- Revenu services en chambre
- Revenu extras
- Revenu total des services annexes
- Revenu net

Pour créer un `comp set`, suivez ces étapes :

<Steps>
1. Allez sur [Wink Studio](/fr/studio/search) pour rechercher des fournisseurs.
2. Recherchez d’autres propriétés avec lesquelles vous souhaitez vous comparer.
3. Suivez la propriété en cliquant sur le bouton `Actions > Détails du fournisseur > Suivre la performance` sur leur carte de résultat de recherche *Voir image ci-dessous*.
4. Cela crée un nouveau graphique pour ce fournisseur que vous pouvez suivre.
5. Allez dans vos [Graphiques](https://extranet.wink.travel/analytics/list).
6. Pour mettre à jour le graphique afin de suivre les indicateurs souhaités, cliquez sur le bouton `Actions > Mettre à jour` pour votre nouveau graphique.
6. Cliquez sur le bouton `Enregistrer` pour continuer.
</Steps>
