Les canaux de vente sont la manière dont vous contrôlez la distribution de votre inventaire. Un canal de vente définit qui peut vendre vos chambres, quelle commission ils gagnent, et quels plans tarifaires sont disponibles.

## Types de canaux de vente

Il existe trois principales façons de distribuer votre établissement sur Wink :

- **Wink Network** — Votre établissement est visible par tous les affiliés sur la plateforme. C’est la distribution la plus large.
- **Canaux directs** — Vous créez un canal privé pour un affilié ou partenaire spécifique avec un taux de commission personnalisé.
- **Votre propre site web** — Utilisez des liens partageables ou des cartes intégrées pour vendre directement depuis votre site sans intermédiaire.

## Création d’un canal de vente

<Steps>
1. Rendez-vous sur https://extranet.wink.travel et sélectionnez votre établissement.
2. Accédez à la section des canaux de vente.
3. Cliquez sur `Créer un nouveau canal de vente`.
4. Donnez un nom au canal (par exemple, "Site Web Direct", "Partenaire - Kim Travel").
5. Définissez le **pourcentage de commission** — combien l’affilié gagne par réservation.
6. Sélectionnez les **plans tarifaires** disponibles sur ce canal.
7. Facultativement, restreignez aux **types de chambres** ou **inventaire** spécifiques.
8. Cliquez sur `Enregistrer`.
</Steps>

## Wink Network

Le Wink Network est le canal de distribution par défaut. Lorsqu’il est activé, votre établissement apparaît dans [Studio](/fr/studio/search) où tout affilié peut le trouver et le vendre.

Pour l’activer :

<Steps>
1. Allez dans les paramètres réseau de votre établissement.
2. Activez `Wink Network`.
3. Définissez un taux de commission par défaut pour les affiliés du réseau.
</Steps>

:::tip
Le Wink Network est le moyen le plus simple d’obtenir une large visibilité. Vous pouvez toujours créer des canaux directs plus tard pour vos partenaires les plus performants avec des taux de commission différents.
:::

## Canaux directs pour affiliés

Pour les affiliés avec lesquels vous souhaitez travailler étroitement, créez un canal direct avec une commission personnalisée.

<Steps>
1. Trouvez l’affilié via la section Réseau. Voir [Network](/fr/extranet/network).
2. Créez un canal de vente direct pour cet affilié.
3. Définissez un taux de commission personnalisé (qui peut être supérieur ou inférieur à votre taux réseau par défaut).
4. Sélectionnez l’inventaire disponible sur ce canal.
</Steps>

Cela est utile lorsque vous souhaitez offrir à un influenceur ou agent de voyage un tarif préférentiel ou un accès exclusif à certaines chambres.

## Structure des commissions

Les commissions sont exprimées en pourcentage du total de la réservation. La commission est répartie entre les parties impliquées :

- **Commission affilié** — Ce que gagne le vendeur (affilié, agent de voyage, influenceur).
- **Frais de plateforme** — Les frais de traitement de Wink.
- **Revenu de l’établissement** — Le reste vous revient.

Vous contrôlez la commission affilié. Les frais de plateforme sont déduits automatiquement.

## Suivi des performances

Une fois vos canaux actifs, utilisez [Analytics](/fr/my-account/analytics) pour suivre quels canaux génèrent le plus de réservations et de revenus. Cela vous aide à optimiser les taux de commission et à décider dans quels partenariats investir.
