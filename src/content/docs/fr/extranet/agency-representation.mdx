Une agence est un compte Wink classique que vous pouvez assigner à votre compte dans un but de représentation. Voici quelques raisons pour lesquelles vous voudriez qu’une agence vous représente :

- Gestion totale / partielle de votre compte Wink par un tiers.
- La propriété a un accord existant avec sa société de gestion et souhaite que Wink gère l’aspect financier de leur arrangement des fonds entrants via Wink.
- L’agence de gestion immobilière effectue tout le travail en ligne au nom de ses propriétés.

## Assigner une agence

Voici comment assigner une agence à votre compte.

<Steps>
 1. Connectez-vous à votre compte utilisateur sur [Wink Extranet](https://extranet.wink.travel).
 2. Sélectionnez la propriété avec laquelle vous souhaitez travailler.
 3. Cliquez sur `Compte > Gestionnaires` dans la barre de navigation principale.
 4. Dans la section `Agence`, saisissez le nom de l’agence que vous souhaitez vous représenter.
 5. Choisissez les frais d’agence dans le menu déroulant.
 5. Ajoutez éventuellement des règles d’expiration.
 7. Cliquez sur le bouton `Définir l’agence` pour continuer.
 </Steps>

 :::note
Vous devez être le propriétaire de la propriété pour assigner une agence.
 :::

 :::tip[Exemple]
Les frais d’agence sont calculés à partir de la réservation.

Réservation = 100 $, Frais d’agence = 1 % => 100 * 0,01 = 1 $
 :::

## Désassigner une agence

Voici comment désassigner une agence de votre compte.

<Steps>
 1. Connectez-vous à votre compte utilisateur sur [Wink Extranet](https://extranet.wink.travel).
 2. Sélectionnez la propriété avec laquelle vous souhaitez travailler.
 3. Cliquez sur `Compte > Gestionnaires` dans la barre de navigation principale.
 4. Dans la section `Agence`, cliquez sur l’icône <Badge text="x" variant="danger" size="small" /> à droite du menu déroulant de l’agence.
 5. Cliquez sur le bouton `Définir l’agence` pour continuer.
 </Steps>
