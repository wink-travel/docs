Une agence est un compte Wink classique que vous pouvez assigner à votre compte pour vous représenter. Voici quelques raisons pour lesquelles vous pourriez souhaiter être représenté par une agence :

* Gestion complète / partielle de votre compte Wink par un tiers.
* Property a un accord existant avec sa société de gestion et souhaite que Wink gère l'aspect financier de son organisation des fonds entrant via Wink.
* L'agence de gestion immobilière effectue tous les travaux en ligne pour le compte de ses propriétés.

## Affecter une agence

Voici comment attribuer une agence à votre compte.

<Steps>
  1. Connectez-vous à votre compte utilisateur sur [Extranet Wink](https://extranet.wink.travel).
  2. Sélectionnez la propriété avec laquelle vous souhaitez travailler.
  3. Cliquez sur `Account > Managers` depuis la barre de navigation principale.
  4. Dans le `Agency` section, tapez le nom de l'agence que vous souhaitez représenter.
  5. Choisissez les frais d'agence dans la liste déroulante.
  6. Ajoutez éventuellement des règles d’expiration.
  7. Cliquez sur le `Set agency` bouton pour continuer.
</Steps>


:::note
Vous devez être propriétaire du bien pour attribuer une agence.
:::

:::tip\[Exemple]
Les frais d'agence sont calculés sur la réservation.

Réservation = 100 $, Frais d'agence = 1 % => 100 \* 0,01 = 1 $
:::

## Annuler l'attribution de l'agence

Voici comment désattribuer une agence de votre compte.

<Steps>
  1. Connectez-vous à votre compte utilisateur sur [Extranet Wink](https://extranet.wink.travel).
  2. Sélectionnez la propriété avec laquelle vous souhaitez travailler.
  3. Cliquez sur `Account > Managers` depuis la barre de navigation principale.
  4. Dans le `Agency` section, cliquez sur le <Badge text="x" variant="danger" size="small" /> icône sur le côté droit de la liste déroulante de l'agence.
  5. Cliquez sur le `Set agency` bouton pour continuer.
</Steps>
