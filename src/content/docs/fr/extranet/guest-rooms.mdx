Chaque réservation commence par une chambre. Les chambres d'hôtes présentent de nombreuses caractéristiques, et il peut être difficile de s'y retrouver au début. Cet article vous expliquera tout ce que vous devez savoir pour devenir un spécialiste des chambres d'hôtes Wink.

Chambres d'hôtes, ou types de chambres comme on dit dans`the biz`, vous permet de présenter et de vendre vos chambres, sur place et à l'extérieur. Pour créer une chambre d'hôtes, cliquez sur le bouton`Create a new guest room` bouton.

Les sections suivantes vous guident à travers chaque étape de la gestion d'une chambre d'hôtes.

## Configuration

Le `Configurations`L'onglet est sélectionné par défaut lorsque vous démarrez.

Cette section couvre les bases de votre chambre d'amis afin que nous sachions à qui nous pouvons vendre cette chambre.

* **Nom du type de chambre**Le nom de la pièce, comme vous l'appelez en interne.*par exemple Supreme Deluxe*
* **Non-fumeur**Que la chambre soit non-fumeur ou non.
* **Tarif standard**Le tarif publié est pour une nuitée sans aucune réduction ni prime incluse.*par exemple 100. Notez que nous utilisons uniquement ce nombre pour présenter le prix attendu aux canaux de vente ou pour pré-remplir certains champs lorsque vous remplissez des formulaires.*
* **Meilleur tarif disponible**Le meilleur tarif disponible pour un séjour d'une nuit sans aucune réduction ni prime incluse.*par exemple 75. Notez que nous utilisons uniquement ce numéro pour présenter le prix attendu aux canaux de vente ou pour pré-remplir certains champs lorsque vous remplissez des formulaires.*
* **Quantité**Le nombre d'unités de ce type de chambre dont vous disposez à un jour donné.*par exemple 40*

### Occupation

:::Attention
L'occupation joue un rôle important dans la disponibilité de votre chambre pour un itinéraire spécifique. Nombre de ces champs seront automatiquement renseignés lors de la création de la chambre. Veillez à les adapter à votre type de chambre.
:::

* **Occupation minimale**Le nombre minimum de personnes autorisées à séjourner dans cette chambre.*par exemple 1*
* **Occupation adulte incluse**Nombre d'adultes inclus dans le prix d'une nuitée.*par exemple 2. Remarque : si la pièce peut accueillir plus de personnes que ce nombre, elles sont calculées à l'aide du`Extra adult`modificateur sur votre forfait.*
* **Occupation maximale**Le nombre maximum de personnes (adultes + enfants) autorisées à séjourner dans cette chambre.*par exemple 4*
* **Occupation maximale par adulte**Le nombre maximum d'adultes autorisés à séjourner dans cette chambre.*par exemple 3*
* **Occupation enfant incluse**Nombre d'enfants inclus dans le prix d'une nuitée.*par exemple 1. Remarque : si la pièce peut accueillir plus de personnes que ce nombre, elles sont calculées à l'aide du`Extra child`modificateur sur votre forfait.*
* **Occupation maximale des enfants**Le nombre maximum d'enfants autorisés à séjourner dans cette chambre.*par exemple 2*

### Autre

Cette section est fournie à titre informatif uniquement. Les voyageurs recherchant des chambres et des suites plus spacieuses trouveront ces informations intéressantes.

* **Nombre de salles de bain**Le nombre de salles de bains pour ce type de chambre.*par exemple 1*
* **Nombre de pièces à vivre**Le nombre de pièces à vivre pour ce type de chambre.*par exemple 1*
* **Nombre maximal de lits pliants**Le nombre de lits d'appoint pris en charge par ce type de chambre.*par exemple 1*
* **Max berceaux**Le nombre de lits bébé pris en charge par ce type de chambre.*par exemple 1*

### Classification des chambres

* **Segment**Le segment dans lequel la chambre d'hôtes est classée.*par exemple, Budget*
* **Emplacement**Le type d'emplacement sous lequel la chambre d'hôtes est classée.*par exemple, étage élevé*
* **Voir**Le type de vue à attendre de cette chambre d'hôtes.*par exemple, vue sur la ville*
* **Classification**La classification sous laquelle cette chambre d'hôtes est classée.*par exemple Économie*
* **Architecture**Le style de construction autour de cette chambre d'hôtes.*par exemple asiatique*
* **Sol**Un ou plusieurs étages optionnels où se trouve cette chambre d'hôtes*par exemple, 20e étage et plus*

### Composition

La composition ne s'applique à vous que si la chambre d'hôtes est composée de plusieurs chambres d'hôtes.

* **Composition**Basculez l'interrupteur sur ✅ si la chambre d'hôtes est composée de plusieurs pièces.*par exemple Oui*
* **Compter**Indiquez le nombre de chambres d'hôtes qui composent cette chambre d'hôtes.*par exemple 2*

### Détails de l'auberge

Cette section s'applique uniquement si vous gérez un dortoir, une auberge ou un autre service de chambre partagé.

* **Chambre partagée**Basculez l'interrupteur sur ✅ si la chambre d'amis est une chambre partagée.*par exemple Oui*
* **Genre**Indiquez le sexe de cette chambre d'hôtes.*par exemple, Homme*

## Chambres

:::note
Il existe une infinité d'agencements de chambres possibles. Nous avons fait en sorte que ce soit aussi personnalisable que possible pour s'adapter à votre propriété.
:::

Cliquez sur le`Bedrooms`onglet pour continuer.

La section chambres vous permet de préciser l'agencement de votre/vos chambre(s) et d'indiquer ce qu'il y a dans chacune d'elles.

Les chambres d'hôtes du même type peuvent avoir plusieurs configurations, ou`Bedding configurations`Une chambre d'amis nécessite au moins un aménagement.

* **Nom de la configuration**Donnez à la mise en page un nom que vous comprendrez*par exemple Principal*

Vous pouvez avoir plusieurs chambres dans un même aménagement.

* **Type de chambre**Chaque chambre est dotée d'un type.*Le premier est`Master`par défaut, mais vous pouvez le modifier dans votre configuration.*

Dans une chambre, vous pouvez avoir plusieurs lits de différents types et quantités.

* **Type de lit**Choisissez le type de lit dans votre chambre.*par exemple, le roi*
* **Quantité**Choisissez le nombre de lits de ce type de lit dans cette chambre.*par exemple 1*

## Équipements

:::note
Cette section est facultative.
:::

Cliquez sur le`Amenities`appuyez sur l'onglet pour continuer.

Tout comme les équipements de l'établissement, les équipements des chambres s'appliquent uniquement à la chambre concernée. Cochez chaque équipement concerné.

## Descriptions

:::note
Vous devez rédiger au moins une description en anglais.
:::

Cliquez sur le`Descriptions`appuyez sur l'onglet pour continuer.

Tout comme pour le texte d'accueil de votre propriété, vous pouvez également créer des descriptions localisées pour vos chambres.
Vous pouvez rédiger dans autant de langues que vous le souhaitez. Nous ajouterons des traductions pour toutes les langues les plus courantes.
Voici le texte que les voyageurs verront lors de la visite de vos chambres.

* **Nom**Donnez un nom à la chambre d'amis.*par exemple Supreme Deluxe*
* **Description**Décrivez la chambre d’amis en un ou deux paragraphes.*Par exemple, c'est la meilleure chambre d'hôtes du monde entier...*

## Réseaux sociaux

:::note
Cette section est facultative.
:::

Cliquez sur le`Social media`appuyez sur l'onglet pour continuer.

Vous vous demandez peut-être pourquoi une chambre d'hôtes possède sa propre section sur les réseaux sociaux. Une chambre d'hôtes peut également être une villa située à proximité de la propriété, mais hors site, avec son propre personnel et son propre profil Instagram.

Les réseaux sociaux vous permettent de rester en contact avec vos voyageurs, nouveaux ou existants, au fil du temps.
Ajoutez vos comptes de réseaux sociaux à votre profil pour que vos clients puissent facilement vous contacter et en savoir plus sur votre chambre.

<Steps>
  1. Basculez le commutateur d'un réseau social que vous souhaitez activer.
  2. Entrez le nom de votre compte sur ce réseau.
</Steps>


## Réputation

:::note
Cette section est facultative.
:::

Cliquez sur le`Reputation`appuyez sur l'onglet pour continuer.

Similaire au`Social media`onglet, vous utiliseriez cet onglet si cette chambre d'hôtes a un ou plusieurs scores de réputation distincts ou des distinctions supplémentaires que vous souhaitez mettre en évidence ici.

Votre réputation est votre monnaie numérique et vous souhaitez pouvoir l'emporter partout avec vous.
Une entrée de réputation peut être tout ce dont votre établissement souhaite s'attribuer le mérite, comme une évaluation par un tiers ou le concours de pâtisserie où votre chef a remporté l'or.
Dans cette section, vous pouvez ajouter vos notes et classements provenant de partout sur Internet.

Cliquez sur le`Add reputation score` bouton.

<Steps>
  1. **Catégorie** Choisissez le type de score que vous souhaitez ajouter. *par exemple, examen par un tiers*
  2. **Fournisseur** Tapez qui vous a attribué la note. *par exemple BigOTA.com*
  3. **Type de notation** Sélectionnez le formulaire dans lequel vous avez reçu le score. *par exemple numérique*
  4. **Date** Si vous le souhaitez, indiquez la date à laquelle vous avez reçu le score.
  5. **Notation** Entrez le score que vous avez reçu. *par exemple 8,5*
  6. **Note maximale** En option, saisissez le score maximum possible pour ce type de récompense. *par exemple 10*
</Steps>


## Vente basée sur les attributs

:::note
Cette section est facultative.
:::

Cliquez sur le`Attribute Based Selling`appuyez sur l'onglet pour continuer.

Toutes nos installations disposent de leur propre section ABS. Cette section vous permet de créer des opportunités de vente incitative sur l'inventaire principal ou de transformer un inventaire non transactionnel en transactionnel.
Nous vous recommandons vivement de créer ces opportunités pour potentiellement toucher le client à un prix plus élevé lors du processus de réservation.

:::conseil
Une chambre d'amis comprend de nombreux équipements annexes :

* Matelas supérieur.

* Kit de salle de bain de luxe.

* Enceinte Bluetooth Marshal.

* Bureau de travail avec chaise extra confortable.

* Petit-déjeuner en chambre

* Massage en chambre

* Pack de 12 bières

* Bouteille de vin

* Chocolat de qualité supérieure

* +++ Laissez libre cours à votre imagination et gagnez un peu d'argent !
  :::

* **Niveau de prix**Donnez au voyageur une idée rapide du prix de cette chambre. Exactement… pour la chambre, pas pour les services annexes.

* **Actif**Basculez le commutateur sur 🛑 si vous souhaitez que cette chambre d'hôtes ne soit plus disponible à la réservation sur tous les tarifs principaux et canaux de vente.

* **Commissionnable**Indiquez si les affiliés perçoivent une commission sur cet inventaire.

* **En vedette**Indiquez si cette chambre d'amis devrait être placée plus en évidence si l'option se présente.

* **Vente basée sur les attributs**Basculez le commutateur sur ✅ pour activer l'ABS sur cet inventaire.*Une fois activé, vous devez ajouter au moins 1 article ABS à vendre*

### Article auxiliaire

Lorsque vous activez l'ABS, vous pouvez commencer à ajouter des éléments auxiliaires et les rendre disponibles à la vente.

*Par défaut, un élément vide a déjà été créé pour vous. Vous pouvez en ajouter d'autres en cliquant sur le bouton`Add item for sale` bouton.*

* **Nom**Donnez un nom à votre auxiliaire.*par exemple, truffes au chocolat au champagne de Royce*
* **Type de tarification**Indiquez comment vous souhaitez calculer le prix.*Par exemple, location de paddle. Définir sur`Per hour`pour indiquer que le prix est basé sur le nombre d'heures louées.*
* **Prix de base**Entrez le prix régulier de cet article.*par exemple 20 / heure*
* **Prix réduit**Saisissez le prix que les utilisateurs de Wink obtiennent lors de la réservation de l'article à côté de la chambre.*par exemple 15 / heure*

### Limites

Vous pouvez limiter la disponibilité de cet article en définissant des quantités minimales et maximales.

Si nous reprenons notre exemple de location de paddle à l'heure, mais que nous souhaitons exiger du client qu'il réserve au moins deux planches simultanément, nous procéderons comme suit :

* **Quantité minimale**Limitez la quantité minimale qui doit être réservée.*par exemple 2 planches à pagaie*
* **Quantité maximale**Limitez la quantité maximale qui doit être réservée.*par exemple, laisser vide pour aucune restriction maximale*

### Descriptions

:::note
Vous devez rédiger au moins une description en anglais.
:::

* **Nom**Donnez un nom à votre auxiliaire.*par exemple, Stand Up Paddleboard*
* **Description**Décrivez l’accessoire en un ou deux paragraphes.*Par exemple : Ne restez pas assis sans rien faire – sortez votre pagaie !*

## Proximité

Cliquez sur le`Proximity`appuyez sur l'onglet pour continuer.

Tous les stocks ne sont pas situés sur place. Si vous avez des stocks hors de vos locaux, informez-en le voyageur ici ; vous pourrez ainsi indiquer un emplacement distinct sur la carte, ainsi que ses coordonnées facultatives.

*Cela peut être utile dans les moments où une villa appartient à l'hôtel mais est située à quelques kilomètres de l'hôtel et que le voyageur souhaite expliquer au chauffeur de taxi comment s'y rendre.*

Cliquez sur le`Save`bouton pour continuer.
