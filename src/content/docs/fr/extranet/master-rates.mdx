:::note
La création de votre premier tarif maître fait partie de l'assistant d'intégration.
:::

:::note
Un `tarif maître` est une paire unique `chambre d'hôte / plan tarifaire`.

*La plupart des fournisseurs de tarifs de voyage utilisent uniquement chambre d'hôte et plan tarifaire dans ce contexte. Lorsque vous voyez votre chambre d'hôte listée à la vente sur les OTA ou moteurs de réservation, les voyageurs comparent les plans tarifaires.*

Wink a ajouté le terme `tarif maître` pour pouvoir donner plus de flexibilité à cette combinaison unique.
:::

:::note
Les tarifs maîtres sont créés / supprimés automatiquement lorsqu'une chambre d'hôte ou un plan tarifaire est ajouté ou supprimé.
:::

Pour gérer vos Tarifs Maîtres, cliquez sur `Monétiser > Tarif Maître` dans la barre de navigation principale.

Un établissement aura un tarif maître pour chaque combinaison chambre d'hôte / plan tarifaire.

*Exemple : Si votre établissement a une chambre d'hôte et deux plans tarifaires, le voyageur verra deux tarifs maîtres disponibles à la vente (si toutes les conditions sont remplies).*

Une entrée de tarif maître propose deux options de gestion :

- Activer ou désactiver la disponibilité du tarif maître.
- Définir des avantages.

## Disponibilité

Lorsque vous activez / désactivez un tarif maître, cela affecte la disponibilité sur tous vos canaux de vente. Ce bouton est un moyen rapide de désactiver un tarif maître spécifique partout, immédiatement.

## Avantages

Lorsque les voyageurs cherchent à réserver une chambre d'hôte, ils n'interagissent pas directement avec la chambre d'hôte. Ils interagissent plutôt avec le tarif maître de cette chambre.
Un tarif maître peut être chargé avec des avantages uniques disponibles uniquement pour la combinaison spécifique chambre d'hôte / plan tarifaire ; offrant ainsi aux propriétaires une autre façon d'inciter cette combinaison.

Deux points à garder à l'esprit lors du choix d'un avantage :

- S'il est garanti ou basé sur la disponibilité.
- Sa valeur numérique.

Un avantage garanti a une valeur numérique plus élevée car il représente une plus grande valeur pour les clients.
Chaque hôtel recevra un `score d'avantages` basé sur la valeur numérique agrégée de tous les avantages sur tous les tarifs maîtres.
Les voyageurs à la recherche d'une bonne affaire peuvent trier les hôtels selon leur score d'avantages élevé.

Les avantages sont un moyen de :

- Différencier la combinaison chambre d'hôte / plan tarifaire.
- Obtenir une meilleure visibilité. *c.-à-d. Obtenir un score plus élevé sur l'échelle des avantages que les autres hôtels de votre région.*

:::tip
Les avantages sont nés du désir des hôtels de se différencier et de `manipuler le système`.
La dernière fois qu'ils ont fait cela, ils se sont littéralement tués en donnant tous leurs bénéfices aux OTA pour un meilleur placement via des commissions plus élevées.
Historiquement, les commissions étaient le seul levier que les hôtels pouvaient manipuler.

Les avantages sont un moyen pour les hôtels de récompenser les voyageurs sans nécessairement avoir un coût réel pour les hôtels.

Si vous avez des suggestions pour d'autres types d'avantages à offrir, [contactez-nous](mailto:hi@wink.travel).
:::

Il y a deux façons d'activer l'inventaire pour un canal de vente :

- Activer la disponibilité de l'inventaire sur la carte. *Voir figure ci-dessus*
- Naviguer vers `Distribution > Inventaire` dans la barre de navigation principale dédiée à la gestion de l'inventaire sur tous les canaux de vente.

## API

Les développeurs qui souhaitent gérer les `Tarifs Maîtres` peuvent se rendre sur [Développeurs > APIs > Monétiser](/fr/developers/apis/#monetize-api).
