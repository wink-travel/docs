Cet article vous montre comment vous connecter avec des affiliés enthousiastes sur Wink et gérer ceux déjà existants.

## Canaux de vente

Pour gérer vos canaux de vente, cliquez sur `Distribution > Canaux de vente` dans la barre de navigation principale.

Les canaux de vente sont vos relations d’affiliation existantes, codifiées avec l’inventaire, les offres et les remises auxquels ils ont accès.

Pour mettre à jour un canal de vente existant, cliquez sur le lien `Actions` sous le canal de vente que vous souhaitez modifier.

- **Remise membre** Le tarif membre est accordé aux voyageurs qui réservent via ce canal de vente.
- **Commission** La commission est attribuée au canal de vente pour chaque réservation réussie.
- **Promotions simples** Ajoutez les promotions que vous souhaitez attribuer à ce canal de vente.
- **Promotions groupées** Ajoutez les promotions groupées que vous souhaitez attribuer à ce canal de vente.
- **Activé** Basculez l’interrupteur activé pour activer / désactiver complètement le canal de vente.
- **Liste noire** Marquer un canal de vente comme en liste noire rend impossible toute réservation via ce canal.

Une fois que vous êtes satisfait de vos réglages, cliquez sur le bouton `Enregistrer` pour continuer.

:::tip
En mettant un canal de vente en liste noire, vous empêchez ce canal d’obtenir le moindre tarif de votre part ; y compris vos tarifs Wink Network.
:::

## Explorer le réseau

Pour trouver de nouveaux affiliés sur Wink, cliquez sur `Distribution > Explorer le réseau` dans la barre de navigation principale.

Dans cette section, vous pouvez voir les demandes en attente des affiliés en haut de la page et une liste d’affiliés à connecter en dessous.

### Répondre à une demande

Vous pouvez choisir de rejeter la demande d’affiliation. Si vous souhaitez poursuivre la demande, vous serez redirigé vers la page `détails de l’affilié` où vous pourrez obtenir plus d’informations sur l’affilié ainsi que des indicateurs de performance passés sur Wink.

### Rechercher des affiliés

Dans la grille des affiliés, vous pouvez rechercher des affiliés par nom, type et localisation. Lorsque vous en trouvez un qui vous intéresse, cliquez dessus pour accéder à la page `détails de l’affilié`.

### Se connecter avec un affilié

Si vous choisissez d’établir une relation directe avec un affilié, il vous sera demandé de codifier la relation de la même manière que pour gérer un canal de vente ci-dessus.

Une fois que vous avez enregistré le nouveau canal de vente, vous revenez à votre liste de canaux de vente.

## API

Les développeurs qui souhaitent gérer leur `Réseau` peuvent se rendre sur [Développeurs > APIs > Distribution](/fr/developers/apis/#distribution-api).
