Il vous sera demandé de rédiger une description longue et une description courte de votre établissement lors de la configuration.

Pour gérer votre texte de bienvenue, cliquez sur `Property > Welcome Text` dans l'onglet de navigation principal.

Mettez à jour le formulaire et cliquez sur le bouton `Save` pour continuer.

:::note
Nous avons besoin, au minimum, que vous complétiez votre texte de bienvenue en anglais.

Créez un texte de bienvenue pour toutes les autres langues que vous souhaitez gérer.
Nous utiliserons votre texte en anglais pour le traduire dans toutes les autres langues populaires.
:::
