Vous serez invité à compléter les politiques de votre établissement lors de la configuration.

Pour gérer les politiques de votre établissement, cliquez sur `Property > Policies` dans l'onglet de navigation principal.

Informez vos clients de :

- Heures d'arrivée/départ.
- Si vous autorisez les animaux domestiques et les enfants dans l'établissement.
- Si vous proposez internet et un parking sur place.

Mettez à jour le formulaire et cliquez sur le bouton `Save` pour continuer.
