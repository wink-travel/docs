Le profil de l'établissement est une étape obligatoire dans notre assistant d'intégration et nous permet de mieux connaître votre établissement grâce à des critères tels que :

- Nombre d'étoiles.
- Type d'hôtel.
- Type d'emplacement.
- ...et plus encore

Pour gérer le profil de votre établissement, cliquez sur `Property > Profile` dans l'onglet de navigation principal.

Mettez à jour le formulaire et cliquez sur le bouton `Save` pour continuer.
