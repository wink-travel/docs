Votre réputation est votre monnaie digitale et vous souhaitez pouvoir l’emporter partout avec vous.  
Une entrée de réputation peut être tout ce dont votre établissement veut se prévaloir, comme une note d’un tiers ou pour le concours où votre chef a remporté l’or.  
Dans cette section, vous pouvez ajouter vos scores et classements provenant de toute la toile.

Pour gérer votre réputation, cliquez sur `Property > Reputation` dans l’onglet de navigation principal.

Cliquez sur le bouton `Add reputation score`.

<Steps>
1. **Category** Choisissez le type de score que vous souhaitez ajouter. *ex. Avis tiers*
2. **Provider** Saisissez qui vous a attribué le score. *ex. BigOTA.com*
3. **Rating type** Sélectionnez la forme sous laquelle vous avez reçu le score. *ex. Numérique*
4. **Date** Facultatif, saisissez la date à laquelle vous avez reçu le score.
5. **Rating** Entrez le score que vous avez reçu. *ex. 8.5*
6. **Max rating** Facultatif, saisissez le score maximal possible pour ce type de récompense. *ex. 10*
</Steps>

Cliquez sur le bouton `Save` pour continuer.
