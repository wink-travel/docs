Ajouter les informations du DG permet de créer plus de familiarité avec l’établissement.

Pour mettre à jour les informations du Directeur général, cliquez sur `Property > General Manager` dans l’onglet de navigation principal.

Vous pouvez ajouter le nom et la photo de votre DG actuel ainsi qu’un message personnel aux clients.

Mettez à jour le formulaire et cliquez sur le bouton `Save` pour continuer.
