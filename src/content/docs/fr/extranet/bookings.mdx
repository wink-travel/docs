Lorsqu’une réservation est effectuée sur Wink, les actions suivantes se produisent :

<Steps>
1. Votre channel manager, CRS ou PMS est notifié.
2. Vous recevez un e-mail de confirmation *(contacts du bureau de réservation)* avec un lien pour gérer votre nouvelle réservation.
3. Le voyageur reçoit un e-mail de confirmation incluant également les coordonnées de votre bureau de réservation.
4. Les développeurs peuvent écouter l’événement webhook `booking.created` et recevoir des informations détaillées sur la réservation en temps réel.
</Steps>

Suivez soit le lien dans l’e-mail, soit rendez-vous dans `Compte > Réservations` depuis la barre de navigation principale pour commencer.

## Grille des réservations

La grille des réservations affiche toutes vos réservations passées et futures, triées par date de réservation dans l’ordre croissant.

### Filtres

Vous pouvez appliquer des filtres pour trouver les réservations que vous recherchez.

- Date de réservation [plage]
- Date d’arrivée [plage]
- Date de départ [plage]
- Code de réservation
- Nom de famille du voyageur
- Prénom du voyageur
- Nom du tarif principal
- Canal de vente / Source

### Export

Vous pouvez exporter les réservations correspondant à vos critères de filtre actuels au format CSV.

Pour exporter vos réservations :

<Steps>
1. Assurez-vous que les filtres souhaités sont actifs.
2. Attendez que la grille des réservations se rafraîchisse.
2. Cliquez sur l’icône 🗂️ (*Export fichier*) en bas à droite de la grille.
</Steps>

## Réservation

Sélectionnez la réservation avec laquelle vous souhaitez travailler dans la grille des réservations et vous serez redirigé vers la page `Réservation`.
Cette page contient tous les détails nécessaires pour bien accueillir votre client le jour de son arrivée.

La page contient également quelques actions importantes pour bien servir vos clients. Vous trouverez toutes les actions disponibles dans le menu déroulant `Actions` en haut de la page.

### Renvoyer l’e-mail

Pour renvoyer l’e-mail de confirmation au voyageur, cliquez sur `Actions > Renvoyer l’e-mail de confirmation`.

### Annuler la réservation

Pour annuler la réservation en tant qu’établissement, cliquez sur `Actions > Annuler la réservation`.

Une fenêtre pop-up s’affichera et vous permettra de :

<Steps>
1. Sélectionner le type d’annulation. *ex. Dates incorrectes*
2. Saisir un motif d’annulation.
</Steps>

*Ce bouton respecte votre politique d’annulation et ne sera donc pas toujours disponible.*

### Demander un remboursement

Pour demander un remboursement, quelle qu’en soit la raison, cliquez sur `Actions > Demander un remboursement`.

Une fenêtre pop-up s’affichera et vous permettra de :

<Steps>
1. Demander un remboursement total ou partiel.
    - Si vous choisissez partiel, il vous sera demandé le montant.
2. Sélectionner un motif.
3. Saisir un motif descriptif pour la demande.
4. Indiquer si vous souhaitez également annuler la réservation, que la demande soit approuvée ou non.
</Steps>

 *Cette action est disponible tant que les fonds de cette réservation n’ont pas été traités.*.

## Synchronisation du calendrier

Pour commencer à synchroniser vos réservations Wink avec votre logiciel de calendrier préféré, rendez-vous dans `Distribution > Synchroniser les réservations avec votre calendrier` depuis la barre de navigation principale.

La première fois, il vous sera demandé de générer un code d’accès que vous pourrez utiliser avec votre logiciel de calendrier pour authentifier la connexion.

</figure>

Cliquez sur le bouton `Oui` pour continuer ou sur `Non` si vous avez changé d’avis.

Vous verrez 3 champs que vous devez utiliser pour connecter Wink à votre logiciel de calendrier :

1. L’URL à utiliser pour s’abonner aux réservations.
2. Le nom d’utilisateur pour l’authentification.
3. La clé d’accès pour l’authentification.

:::tip
CalDAV est un protocole standard de calendrier pris en charge par la plupart des applications de calendrier, y compris Google Calendar, Apple Calendar et Outlook. Il suffit d’ajouter l’URL comme nouvel abonnement de calendrier dans votre application préférée.
:::

Une fois cette étape terminée et l’URL CalDAV ajoutée à votre logiciel de calendrier, votre calendrier commencera à afficher les réservations provenant de Wink.

3 jours après le départ du client, celui-ci recevra un e-mail de notre part lui demandant de laisser un avis sur votre établissement.
L’avis consiste en des questions à choix multiples simples avec des valeurs allant de 1 à 5. Les catégories sont établissement, services et équipements.

Les clients peuvent également vous laisser un message auquel vous pouvez répondre et choisir de rendre public pour que d’autres clients le voient.
