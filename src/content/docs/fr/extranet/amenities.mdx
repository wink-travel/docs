Les équipements de la propriété sont les services et installations disponibles dans votre établissement.

Pour gérer les équipements de votre propriété, cliquez sur `Property > Amenities` dans l'onglet de navigation principal.

Cochez toutes les cases correspondant à votre propriété.

Cliquez sur le bouton `Save` pour continuer.
