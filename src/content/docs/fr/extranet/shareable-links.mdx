:::note
Ceci est un sous-ensemble de la fonctionnalité [Liens partageables](/studio/shareable-links) disponible dans Wink Studio.

Les propriétés auront automatiquement tout leur inventaire disponible sous forme de liens partageables sans avoir à passer par Wink Studio pour les générer.
:::

Les liens partageables sont une fonctionnalité centrale de Wink et sont nés du besoin de communiquer l’inventaire et les tarifs de manière dynamique via le moyen le plus simple possible sur Internet ; à savoir le lien hypertexte.

<h4 align="center">[https://trvl.as/3xWCH](https://trvl.as/3xWCH)</h4>

Ci-dessus un exemple de l’un de nos liens. Il est court et portable... même assez court pour être mémorisé. Bien que court, ce lien contient une quantité **explosive** d’informations, telles que :

- Votre ID d’application.
- Votre ID d’affilié.
- L’image(s) et le texte à afficher à l’utilisateur.
- L’apparence (*c’est-à-dire le thème, les couleurs et plus*).
- L’itinéraire utilisé pour notre requête de prix (*2 adultes, 1 nuit*).
- La devise dans laquelle afficher le prix.
- La langue dans laquelle afficher le texte.

Le reste de cet article vous guide pour gérer les liens vers vos inventaires.

## Personnaliser le lien

 Le formulaire vous permet de personnaliser votre lien de la manière suivante :

<Steps>
 1. Par défaut, le lien reçoit le nom de votre inventaire en anglais. Vous pouvez lui donner n’importe quel nom qui vous aide à vous souvenir de ce à quoi le lien correspond.
 2. Par défaut, votre personnalisation / branding principal est appliqué au lien. Vous pouvez sélectionner d’autres [Personnalisations](/studio/customization) que vous avez créées dans Wink Studio.
 3. Ajoutez des mots-clés, séparés par une virgule, qui seront utilisés par les [robots d’indexation](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Facultativement, saisissez votre identifiant de compte X ou votre ID d’application Facebook.
 5. Ajoutez des titres et descriptions dans les langues que vous souhaitez afficher à l’utilisateur.
 6. Sélectionnez une ou plusieurs images qui s’afficheront lorsque vous partagerez le lien sur des plateformes telles que Facebook ou LinkedIn.
 7. Choisissez un thème / superposition qui ajoute un design de marque par-dessus l’image(s) sélectionnée(s). *Note : C’est ainsi que vous affichez les prix par-dessus l’image.*
 8. Cliquez sur le bouton `Save` pour continuer.
 </Steps>

L’image ci-dessus vous montre toutes les actions disponibles pour votre lien :

- **Aperçu** Vous montre un aperçu rapide de la configuration du lien et de son apparence pour l’utilisateur.
- **Mettre à jour** Met à jour la configuration de votre lien.
- **Ajouter à WinkLinks** Ajoute le lien à votre compte WinkLinks.
- **Copier** Copie le lien dans votre presse-papiers pour que vous puissiez facilement le partager où vous voulez (*y compris sur [WinkLinks](/link-manager/wink-links)*).
- **Partager** Ouvre une nouvelle fenêtre avec des options pour partager sur de nombreux réseaux sociaux et plateformes de messagerie.
- **Code QR** Génère un code QR que vous pouvez partager sur des plateformes comme IG qui sont plus axées sur l’image.
- **Intégrer** Vous montre comment intégrer ce lien en tant que [bannière publicitaire](/developers/web-components/#content-loader) dans votre site web.
- **Utiliser avec WordPress** Vous montre comment utiliser notre [plugin WordPress](/developers/wordpress) pour intégrer ce lien dans votre site.

Nous détaillons certaines de ces options ci-dessous.

Si vous souhaitez partager un code QR au lieu d’un lien, cliquez sur le bouton `QR code` et une nouvelle fenêtre apparaîtra avec votre code QR.

Comment enregistrer votre code QR :

- Si vous êtes sur votre ordinateur portable, faites un clic droit sur l’image et sélectionnez `Enregistrer l’image`.
- Si vous êtes sur votre téléphone mobile :
    - Appuyez longuement sur l’image jusqu’à ce qu’une option apparaisse pour enregistrer l’image dans vos Photos.
    - Faites une capture d’écran du code QR, recadrez-la pour l’ajuster, et enregistrez-la dans Photos.

### Intégrer

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Voici comment intégrer votre lien, en tant que bannière publicitaire, dans votre site :

- La ligne 3 vous montre comment intégrer les styles Wink dans votre site.
- Les lignes 6 à 9 vous montrent comment utiliser le composant Web [wink-content-loader](/developers/web-components/#content-loader) et lui indiquer de récupérer une bannière publicitaire pour votre code.
- La ligne 11 vous montre comment intégrer notre Javascript dans votre site.
- La ligne 13 vous montre comment intégrer le composant Web [wink-app-loader](/developers/web-components/#application-loader) et lui indiquer de récupérer vos préférences de configuration au niveau de la page.
