Les annonces sont un moyen d'informer vos clients d'informations importantes de manière visible.  
Si une annonce est `déclenchée`, elle apparaît sous forme de nouvelle fenêtre au-dessus de votre page d'accueil.

Pour voir vos annonces, cliquez sur `Propriété > Annonce` dans la barre de navigation principale.

Cliquez sur le bouton `Créer une annonce` pour créer une nouvelle entrée.

<Steps>
1. Choisissez d’inclure ou non un titre.
2. Les annonces peuvent être déclenchées de deux manières :
    - Toujours afficher, à chaque visite de votre page.
    - Afficher uniquement lorsque l’itinéraire correspond à l’itinéraire de l’annonce.
3. Si vous avez choisi d’afficher l’annonce uniquement pour une certaine plage de dates, ajoutez les dates dans les champs du formulaire ci-dessous.
4. Cliquez sur le bouton `Enregistrer` et vous serez redirigé vers vos annonces.
</Steps>
