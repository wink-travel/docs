La publication fonctionne ainsi :

<Steps>
1. Le contenu est généré à partir de vos préférences une semaine avant le mois de publication.
2. Vous recevez un e-mail vous demandant d’approuver le contenu nouvellement créé [ou vous informant que le contenu a été créé pour le mois suivant et qu’il n’y a rien d’autre à faire car vous avez choisi d’approuver automatiquement le contenu].
3. Vous vous connectez à [social.wink.travel](https://social.wink.travel) pour approuver les publications.
4. Les publications sont programmées pour être publiées. (Vous pouvez encore arrêter la tâche de publication si vous le faites avant qu’elle ne soit publiée. Sinon, vous devrez aller sur le réseau social et supprimer ou modifier manuellement la publication.)
5. Les publications sont diffusées au moment le plus opportun pour votre fuseau horaire sélectionné.
</Steps>

Une fois le contenu approuvé pour publication, il est ajouté à votre calendrier.

:::note
Il y aura une entrée pour chaque langue et chaque réseau social.

Par exemple, si vous avez 3 réseaux sociaux et 3 langues sélectionnés, vous verrez 9 entrées dans le calendrier.
:::
