# Paramètres

Lorsque vous commencez avec Social, nous vous demandons comment vous souhaitez que nous créions votre contenu. Ces options sont disponibles à tout moment afin que vous puissiez facilement les modifier plus tard.

Accédez à `Préférences` depuis la barre latérale pour gérer vos paramètres.

La section préférences contient vos paramètres de base pour la génération de contenu :

1. **Personnalisation préférée** Choisissez la personnalisation à utiliser lors de la génération de contenu. Les personnalisations contrôlent les couleurs du thème, l’itinéraire par défaut, la devise, la langue, et plus encore. Vous créez des personnalisations dans votre compte sur [my.wink.travel](https://my.wink.travel).
2. **Approbation automatique** Lorsqu’elle est activée, les publications générées sont publiées automatiquement sans nécessiter votre approbation. Activez cette option une fois que vous êtes confiant dans la qualité du contenu que nous produisons pour vous.
3. **Nom de l’approbateur** Le nom de la personne ou de l’équipe qui recevra les e-mails de demande d’approbation.
4. **E-mail de l’approbateur** L’adresse e-mail où les demandes d’approbation pour les publications à venir seront envoyées.

Social propose de nombreux créateurs de contenu parmi lesquels choisir. Chaque créateur se concentre sur une manière spécifique de créer un contenu unique pour votre établissement en utilisant l’IA. Chaque créateur de contenu possède un attribut **Poids** — un poids plus élevé augmente la probabilité que ce créateur soit sélectionné lors de la génération des publications.

Voici tous les créateurs de contenu disponibles :

| Créateur | Description |
| -- | -- |
| **Destination** | Utilise votre emplacement pour créer un texte utile sur la région |
| **Type de chambre** | Crée une publication réservable mettant en avant un de vos types de chambre avec un texte généré par IA et vos images de type de chambre |
| **Activité** | Crée une publication réservable mettant en avant une de vos activités |
| **Option supplémentaire** | Crée une publication réservable mettant en avant une de vos options supplémentaires |
| **Attraction** | Crée une publication réservable mettant en avant une de vos attractions |
| **Lieu** | Crée une publication réservable mettant en avant un de vos lieux |
| **Salle de réunion** | Crée une publication réservable mettant en avant une de vos salles de réunion |
| **Restaurant** | Crée une publication réservable mettant en avant un de vos restaurants |
| **Spa** | Crée une publication réservable mettant en avant un de vos spas |
| **Établissement** | Crée une publication réservable mettant en avant votre chambre au meilleur prix avec un texte généré par IA et vos images d’établissement |
| **Sentiment utilisateur** | Agrège les avis des clients sur le web dans une publication basée sur le ressenti qui met en avant ce que les clients aiment dans votre établissement |
| **Drôle** | Crée des publications légères et divertissantes pour garder votre fil engageant |

:::tip[Astuce pro]
Assurez-vous de disposer du type d’inventaire requis pour vos créateurs de contenu choisis. Par exemple, si vous sélectionnez « Restaurant », vous devez avoir au moins un restaurant répertorié dans votre établissement.
:::

Le ton de voix définit le style de langage que l’IA utilise lors de la génération de contenu. Choisissez les tons qui correspondent à la personnalité de votre marque. Comme pour les créateurs de contenu, chaque ton a un **Poids** pour contrôler la fréquence d’utilisation.

Voici les tons disponibles :

| Ton | Style |
| -- | -- |
| **Neutre** | Calme, autoritaire, factuel. Pas de battage, pas de blagues — juste un savoir fiable et bien structuré |
| **Amical** | Chaleureux, accessible, énergie souriante à travers le texte avec des contractions familières et des emojis occasionnels |
| **Luxe** | Sophistiqué, raffiné. Des mots comme « exquis » et « sur mesure » qui font sentir exclusif |
| **Gourmand** | Descriptions sensuelles et appétissantes. Un langage de « food porn » qui donne faim |
| **Aventurier** | Audacieux, intrépide, énergie d’explorateur. Des descriptions sensorielles vives qui rendent tout épique |
| **Familial** | Excitation saine et adaptée aux enfants. Énergie parentale chaleureuse sur des expériences sûres et amusantes |
| **Romantique** | Poétique, nostalgique, qui fait battre le cœur. Couchers de soleil, langage doux et amour |
| **Humoristique** | Sarcastique, répliques rapides, jeux de mots intelligents. Se lit comme un spectacle de stand-up |
| **Écolo** | Sincère et engagé. Célèbre les alternatives vertes et les choix durables |
| **Initié local** | Références locales très spécifiques, met en garde contre les pièges à touristes, utilise l’argot local |
| **Inclusif** | Accueillant, festif et affirmatif. Fait que tout le monde se sente vu et valorisé |
| **Mystique** | Éthéré, spirituel, vérités cosmiques. Invitations douces à regarder à l’intérieur de soi |
| **Futuriste** | Élégant, tourné vers l’avenir. Aime les termes comme « changement de paradigme » et les visions optimistes du futur |
| **Économique** | Débrouillard, ingénieux, fièrement frugal. Célèbre les bonnes affaires abordables et les expériences gratuites |

## Types de publication

Les types de publication déterminent l’objectif et la structure du contenu généré. Chaque type de publication a un **Poids** pour contrôler la fréquence d’utilisation.

| Type de publication | Description |
| -- | -- |
| **Transactionnel** | Clair, concis, axé sur la conversion. Encourage les réservations et les actions directes avec des faits, des prix et les étapes suivantes |
| **Sensibilisation** | Contenu de construction de marque conçu pour présenter votre établissement et augmenter la visibilité auprès des clients potentiels |
| **Sentiment** | Contenu émotionnellement résonnant qui crée des associations positives et construit une connexion émotionnelle avec votre audience |

  <figcaption>*Fig. 4 - Fuseaux horaires*</figcaption>

Vous pouvez vouloir concentrer vos efforts marketing sur une langue et une région particulières. Indiquez-nous les langues pour lesquelles vous souhaitez créer du contenu ainsi que le fuseau horaire qui vous intéresse, et nous nous occupons du reste. Les publications sont publiées à l’heure prévue dans le fuseau horaire sélectionné.

*La plupart des établissements utilisent une seule langue et par défaut leur langue locale.*

En connectant votre profil Google Business, nous pouvons extraire des détails plus riches sur votre établissement pour un contenu plus efficace. Votre établissement doit avoir un profil Google Business vérifié afin que nous puissions générer du contenu à partir de votre fiche. Certains de nos créateurs de contenu **exigent** que vous disposiez d’un compte Google Business.

:::tip[Astuce pro]
Commencez petit

Ne soyez pas trop ambitieux lors de votre première tentative. Nous vous recommandons de commencer par créer du contenu pour votre établissement dans une seule langue.
:::
