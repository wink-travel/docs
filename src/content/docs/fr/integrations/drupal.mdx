Le **module Wink Drupal** ajoute des composants d'inventaire de voyages Wink à votre site Drupal sous forme de blocs natifs. Les éditeurs peuvent les placer n'importe où dans la mise en page via le système standard de blocs.

<LinkCard

/>

## Exigences

- Drupal 10 ou 11
- PHP 8.1+
- Composer

## Installation

<Steps>
1. Installez via Composer :
   ```bash
   composer require drupal/wink
   ```
2. Activez le module :
   ```bash
   drush en wink -y
   ```
3. Allez dans **Administration → Configuration → Services Web → Paramètres Wink** (`/admin/config/wink`).
4. Saisissez votre **Client ID** et **Client Secret**, sélectionnez votre environnement, puis cliquez sur **Enregistrer**.
   Rendez-vous sur [Applications](/fr/portal/settings/applications) pour récupérer vos identifiants.
</Steps>

## Ajouter un bloc à une page

<Steps>
1. Allez dans **Administration → Structure → Disposition des blocs**.
2. Cliquez sur **Placer un bloc** dans la région où vous souhaitez afficher le contenu Wink.
3. Recherchez **Wink** pour trouver les blocs disponibles :
   - **Wink Content** — affiche une carte d'hôtel, une grille ou une carte
   - **Wink Lookup** — barre de recherche de destination
   - **Wink Search** — bouton de sélection d’itinéraire
   - **Wink Account** — bouton de connexion
   - **Wink Itinerary** — bouton d’état de l’itinéraire
   - **Wink Cart** — bouton du panier d’achat
4. Pour **Wink Content**, configurez le **Type de mise en page** et l’**ID de mise en page** dans les paramètres du bloc.
5. Enregistrez la configuration du bloc.
</Steps>

## Blocs disponibles

| Bloc | Fonction |
|---|---|
| **Wink Content** | Affiche des cartes d’hôtels, des grilles classées, des cartes, et plus |
| **Wink Lookup** | Barre de recherche pour destinations et hôtels |
| **Wink Search** | Bouton icône qui ouvre le sélecteur d’itinéraire |
| **Wink Account** | Bouton de connexion / compte utilisateur |
| **Wink Itinerary** | Bouton d’itinéraire affichant le voyage en cours |
| **Wink Cart** | Bouton résumé du panier d’achat |
