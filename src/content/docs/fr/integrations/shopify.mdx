L’**application Wink Shopify** permet aux voyagistes, vendeurs d’expériences et commerçants en hôtellerie d’afficher l’inventaire Wink en temps réel directement sur leur vitrine Shopify — aux côtés des produits, sur les pages d’accueil ou n’importe où dans le thème.

## Installation

<Steps>
1. Rendez-vous sur le [Shopify App Store](https://apps.shopify.com) et recherchez **Wink Travel**.
2. Cliquez sur **Ajouter l’application** et suivez le processus OAuth pour autoriser.
3. Dans les paramètres de l’application Wink, saisissez votre **Client ID** et **Client Secret**.
   Allez dans [Applications](/fr/portal/settings/applications) pour récupérer vos identifiants.
4. Cliquez sur **Vérifier & Enregistrer**. Un message de succès confirme la connexion.
</Steps>

## Activer le bloc d’intégration de l’application

Le bloc d’intégration de l’application injecte automatiquement le script et la feuille de style Wink dans chaque page — vous n’avez pas besoin de modifier le code du thème.

<Steps>
1. Dans l’interface d’administration Shopify, allez dans **Boutique en ligne → Thèmes → Personnaliser**.
2. Cliquez sur **Intégrations d’applications** dans le panneau de gauche.
3. Activez **Wink Travel**.
4. Cliquez sur **Enregistrer**.
</Steps>

:::note
Le bloc d’intégration lit votre Client ID depuis un métachamp Shopify configuré lors de l’installation. Votre Client Secret n’est jamais exposé au navigateur.
:::

## Ajouter un composant à une page

<Steps>
1. Dans l’éditeur de thème, ouvrez la page ou la section où vous souhaitez afficher le contenu Wink.
2. Cliquez sur **Ajouter un bloc** et choisissez parmi les blocs Wink :
   - **Wink Content** — cartes d’hôtels, grilles ou cartes
   - **Wink Lookup** — barre de recherche de destinations
   - **Wink Search** — bouton de sélection d’itinéraire
   - **Wink Account** — bouton de connexion
   - **Wink Itinerary** — bouton d’état de l’itinéraire
   - **Wink Cart** — bouton du panier d’achat
3. Pour **Wink Content**, sélectionnez un **Type de mise en page** et saisissez l’**ID de mise en page** depuis Wink Studio.
4. Cliquez sur **Enregistrer**.
</Steps>

## Blocs de thème disponibles

| Bloc | Fonction |
|---|---|
| **Wink Content** | Affiche des cartes d’hôtels, des grilles classées, des cartes et plus |
| **Wink Lookup** | Barre de recherche pour destinations et hôtels |
| **Wink Search** | Bouton icône qui ouvre le sélecteur d’itinéraire |
| **Wink Account** | Bouton de connexion / compte utilisateur |
| **Wink Itinerary** | Bouton d’itinéraire affichant le voyage en cours |
| **Wink Cart** | Bouton résumé du panier d’achat |
