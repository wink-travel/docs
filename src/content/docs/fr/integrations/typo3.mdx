L'**extension Wink TYPO3** intègre l'inventaire de voyages Wink dans TYPO3 13 sous forme d'éléments de contenu natifs. Les éditeurs ajoutent des composants Wink aux pages via l'éditeur de pages TYPO3 standard — aucune modification de template requise.

<LinkCard
  title="wink-travel/wink-typo3-extension sur GitHub"

## Prérequis

- TYPO3 13.x
- PHP 8.2+
- Composer

## Installation

<Steps>
1. Installez via Composer :
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Activez l'extension :
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Dans le backend TYPO3, allez dans **Admin Tools → Settings → Extension Configuration → wink**.
4. Saisissez votre **Client ID**, **Client Secret**, et sélectionnez votre environnement. Enregistrez.
   Rendez-vous sur [Applications](/fr/portal/settings/applications) pour récupérer vos identifiants.
5. Ajoutez le template TypoScript statique à votre template racine :
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Ajouter un élément de contenu à une page

<Steps>
1. Ouvrez une page dans le module **Page**.
2. Cliquez sur le bouton **+** dans la zone de contenu pour ajouter un nouvel élément.
3. Choisissez l'un des éléments de contenu **Wink Travel** :
   - **Wink Content** — cartes d'hôtels, grilles classées, cartes
   - **Wink Lookup** — barre de recherche de destination
   - **Wink Search** — bouton de sélection d'itinéraire
   - **Wink Account** — bouton de connexion
   - **Wink Itinerary** — bouton d'état de l'itinéraire
   - **Wink Cart** — bouton du panier d'achat
4. Pour **Wink Content**, utilisez les champs FlexForm pour sélectionner un **Type de mise en page** et saisir l'**ID de mise en page**.
5. Enregistrez et publiez la page.
</Steps>

## Éléments de contenu disponibles

| Élément | Objectif |
|---|---|
| **Wink Content** | Cartes d'hôtels, grilles classées, cartes, et plus |
| **Wink Lookup** | Barre de recherche de destination |
| **Wink Search** | Bouton icône ouvrant le sélecteur d'itinéraire |
| **Wink Account** | Bouton de connexion / compte utilisateur |
| **Wink Itinerary** | Bouton d'itinéraire affichant le voyage en cours |
| **Wink Cart** | Bouton résumé du panier d'achat |
