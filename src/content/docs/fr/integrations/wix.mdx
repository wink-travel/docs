L'**application Wink Wix** ajoute des composants web Wink à votre site Wix sous forme de widgets natifs que vous pouvez glisser-déposer dans l'éditeur Wix. Aucune programmation requise pour les propriétaires de site.

/>

## Installation

<Steps>
1. Rendez-vous sur le [Wix App Market](https://www.wix.com/app-market) et recherchez **Wink Travel**.
2. Cliquez sur **Ajouter au site** et suivez les instructions pour autoriser l'application.
3. Une fois installée, ouvrez **Tableau de bord → Paramètres Wink** et saisissez votre **Client ID** et **Client Secret**.
   Rendez-vous sur [Applications](/fr/portal/settings/applications) pour récupérer vos identifiants.
4. Cliquez sur **Enregistrer** puis sur **Tester la connexion** pour confirmer que Wink est connecté.
</Steps>

## Ajouter un composant à une page

<Steps>
1. Ouvrez l'**éditeur Wix** et allez sur la page où vous souhaitez ajouter du contenu Wink.
2. Cliquez sur **Ajouter (+) → Applications → Wink Travel** pour voir tous les widgets disponibles.
3. Glissez un widget sur la page — par exemple, **Wink Content** pour afficher une grille d'hôtels.
4. Dans le panneau des paramètres du widget, sélectionnez le **Type de mise en page** et collez le **Layout ID** depuis Wink Studio.
5. Cliquez sur **Publier** pour rendre les modifications visibles en ligne.
</Steps>

## Widgets disponibles

| Widget | Objectif |
|---|---|
| **Wink Content** | Affiche des cartes d'hôtels, des grilles classées, des cartes et plus encore |
| **Wink Lookup** | Barre de recherche pour destinations et hôtels |
| **Wink Search** | Bouton icône qui ouvre le sélecteur d’itinéraire |
| **Wink Account** | Bouton de connexion / compte utilisateur |
| **Wink Itinerary** | Bouton d’itinéraire affichant le voyage en cours |
| **Wink Cart** | Bouton résumé du panier d’achat |
