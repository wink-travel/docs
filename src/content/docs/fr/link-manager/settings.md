Pour gérer vos paramètres WinkLinks, rendez-vous dans `WinkLinks` dans la barre de navigation principale et cliquez sur l’onglet `Paramètres`.

Voici les façons dont vous pouvez configurer le comportement de vos WinkLinks avec les `Paramètres` :

- **Intelligent** En activant l’intelligence, nous essayons de récupérer le contenu `riche` disponible sur des sites comme `Wink`, `Spotify`, `Instagram`, `Facebook`, `YouTube`, `X (anciennement Twitter)` et d’autres. *Par exemple, si vous partagez un post IG sur WinkLinks, le post s’affichera comme vous le voyez sur IG, avec les likes, les commentaires et une possibilité d’interagir avec le post.*
- **Affichage initial** Contrôlez la manière dont votre audience interagit d’abord avec votre site et voit votre contenu. Trois options s’offrent à vous :
    - Design `basé sur des colonnes`. C’est notre design préféré et si vous n’affichez pas trop de contenu, il sera très esthétique sur votre page. Cependant, il peut devenir `lourd` si vous partagez beaucoup de contenu riche.
    - Design `basé sur des lignes`. C’est une façon de rendre votre contenu un peu moins lourd à charger pour les navigateurs, mais nous recommandons que votre contenu ait une quantité de texte similaire pour que les cartes s’alignent mieux.
    - Design `basé sur une liste`. La plupart des autres gestionnaires de liens affichent une liste simple avec une URL et un contenu minimal. Choisissez ce mode d’affichage si vous avez beaucoup de contenu et souhaitez l’afficher le plus rapidement possible.
- **Photo de profil** En haut de chaque page WinkLinks se trouve la photo de profil de votre compte ainsi que la description de votre compte. Vous pouvez afficher votre photo de profil de deux façons :
    - `Cercle` Nous recommandons d’utiliser un style circulaire si votre photo est de vous-même ou de tout autre élément qui s’intègre dans un conteneur rond.
    - `Rectangle` Nous recommandons d’utiliser un style rectangulaire si votre logo d’entreprise est votre photo de profil, car la plupart des logos d’entreprise sont de forme rectangulaire.
- **Personnalisation** Choisissez la personnalisation que vous souhaitez utiliser pour votre page WinkLinks. Cela affecte les couleurs de votre thème ainsi que votre itinéraire lors de l’affichage des cartes d’inventaire Wink, des cartes ou des grilles.
