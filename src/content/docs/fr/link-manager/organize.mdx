Pour gérer vos tags WinkLinks, rendez-vous dans `WinkLinks` dans la barre de navigation principale et cliquez sur l’onglet `Tags`.

Au fur et à mesure que votre contenu grandit, vous voudrez commencer à l’organiser. Nous avons créé les `tags` à cet effet. Vous pouvez regrouper le contenu en utilisant autant de tags que vous le souhaitez.

## Créer un tag

<Steps>
1. Créez un nouveau tag en cliquant sur le bouton `Add a new tag`.
2. Saisissez le nom de votre nouveau tag. *ex. Top coffee shops in Berlin*
3. Cliquez sur le lien `Save`.
</Steps>

## Mettre à jour un tag

<Steps>
1. Commencez par cliquer sur le lien `Edit` à côté du tag que vous souhaitez modifier.
2. Mettez à jour le nom de votre tag. *ex. Best coffee shops in Berlin*
3. Cliquez sur le lien `Save`.
</Steps>

## Supprimer un tag

:::note
Lorsque vous supprimez un tag, vous le supprimez également de tout le contenu associé.
:::

<Steps>
1. Cliquez sur le lien `Remove` à côté du tag que vous souhaitez supprimer.
</Steps>
