Pour gérer vos balises WinkLinks, accédez à`WinkLinks`dans la barre de navigation principale et cliquez sur le`Tags`onglet.

À mesure que votre contenu se développe, vous souhaiterez peut-être commencer à l'organiser. Nous avons créé`tags`À cette fin, vous pouvez regrouper le contenu à l'aide d'autant de balises que vous le souhaitez.

## Créer une balise

<Steps>
  1. Créez une nouvelle balise en cliquant sur le `Add a new tag` bouton.
  2. Entrez le nom de votre nouvelle balise. *Par exemple, les meilleurs cafés de Berlin*
  3. Cliquez sur le `Save` lien.
</Steps>


## Mettre à jour la balise

<Steps>
  1. Commencez par cliquer sur le `Edit` lien à côté de la balise que vous souhaitez modifier.
  2. Mettez à jour le nom de votre tag. *Par exemple, les meilleurs cafés de Berlin*
  3. Cliquez sur le `Save` lien.
</Steps>


## Supprimer l'étiquette

:::note
La suppression d'une balise entraîne sa suppression de tout le contenu associé.
:::

<Steps>
  1. Cliquez sur le `Remove` lien à côté de la balise que vous souhaitez supprimer.
</Steps>
