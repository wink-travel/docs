En créant un [compte affilié](/fr/studio/what-is-studio), vous disposez déjà d’une URL personnalisée WinkLinks. Les fonctionnalités éditeur WinkLinks sont disponibles dans notre application Studio, aux côtés de nos autres fonctionnalités affiliées.

### Exemple

Si le nom de votre compte est Queen Bee, votre URL personnalisée WinkLinks sera :

<h4 align="center">https://i.trvl.as/queen-bee</h4>

Les noms de compte sont attribués selon le principe du `premier arrivé - premier servi` - Réservez votre URL unique tant qu’elle est disponible !
