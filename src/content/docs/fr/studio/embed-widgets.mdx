Studio propose plusieurs widgets intégrables que vous pouvez ajouter à votre propre site web pour générer des réservations.

Pour accéder aux widgets intégrables, allez dans `Outils > Divers` depuis la barre latérale.

## Widgets disponibles

| Widget | Description |
| -- | -- |
| **Bouton de compte** | Un bouton de marque qui renvoie à votre page d’affilié |
| **Bouton d’itinéraire** | Un bouton qui ouvre le sélecteur d’itinéraire |
| **Formulaire d’itinéraire** | Un formulaire en ligne pour sélectionner dates et invités |
| **Bouton de recherche** | Un bouton qui ouvre la recherche de propriétés |
| **Bouton de recherche d’inventaire** | Un bouton pour rechercher un inventaire spécifique |

## Comment intégrer

Chaque widget fournit un extrait de code que vous pouvez copier-coller dans le HTML de votre site web. L’extrait inclut votre **ID client** et **ID de personnalisation** pour que le widget soit automatiquement lié à votre compte affilié.

:::note
Si vous utilisez WordPress, nous proposons également un plugin WordPress pour une intégration plus facile.
:::
