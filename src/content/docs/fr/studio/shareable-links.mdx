Les liens partageables sont une fonctionnalité essentielle de Wink et sont nés du besoin de communiquer l'inventaire et les prix de manière dynamique par le biais du moyen le plus simple possible sur Internet, à savoir le lien hypertexte.

<h4 align="center">[https://trvl.as/s/3xWCH](https://trvl.as/s/3xWCH)</h4>


Ci-dessus, un exemple de l'un de nos liens. Il est court et facile à utiliser... suffisamment court pour être mémorisé. Malgré sa brièveté, ce lien est riche en informations.**explosif**quantité d'informations, telles que :

* Votre identifiant d'application.
* Votre identifiant d'affilié.
* L'image(s) et le texte à afficher à l'utilisateur.
* L'apparence et la sensation (*c'est-à-dire le thème, les couleurs et plus encore*).
* L'itinéraire à utiliser pour notre demande de prix (*2 adultes, 1 nuit*).
* La devise dans laquelle afficher le prix.
* La langue dans laquelle afficher le texte.

Le reste de cet article vous explique comment créer, personnaliser et partager vos liens.

## Créer un lien

Ci-dessus, une image prise à partir de[Recherche](/studio/search)et vous présente quelques actions possibles avec vos résultats de recherche. L'une de ces actions est`Make a link`Cliquez sur ce bouton et vous serez redirigé vers notre page de formulaire de lien partageable où vous pourrez commencer à personnaliser votre lien.

## Personnaliser le lien

Le formulaire vous permet de personnaliser votre lien des manières suivantes :

<Steps>
  1. Donnez-lui un nom pour vous rappeler de quoi parle le lien.
  2. Sélectionnez le [Personnalisation](/studio/customization) vous souhaitez postuler sur ce lien.
  3. Ajoutez des mots-clés, séparés par une virgule, qui seront utilisés par [Robots d'exploration Web](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
  4. Saisissez éventuellement votre identifiant de compte X ou l'ID de l'application Facebook.
  5. Ajoutez des titres et des descriptions dans les langues que vous souhaitez que l’utilisateur voie.
  6. Sélectionnez une ou plusieurs images qui s'afficheront lorsque vous partagerez le lien sur des plateformes telles que Facebook ou LinkedIn.
  7. Choisissez un thème/une superposition qui ajoute un design de marque sur les images que vous avez sélectionnées. *Remarque : voici comment afficher les prix au-dessus de l’image.*
  8. Cliquez sur le `Save` bouton pour continuer.
</Steps>

## Partager le lien

<figure>
  <CloudinaryImage width="350" height="641" resize="scale" quality="best" alt="Actes" src="academy/shareable-link-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Actions de lien partageables*</figcaption>
</figure>


Ci-dessus, vous trouverez une image avec toutes les actions disponibles pour votre lien :

* **Aperçu**Vous montre un aperçu rapide de la façon dont le lien a été configuré et à quoi il ressemblera pour l'utilisateur.
* **Mise à jour**Met à jour votre configuration de lien.
* **Ajouter à WinkLinks**Ajoute le lien à votre compte WinkLinks.
* **Copie**Copie le lien dans votre presse-papiers afin que vous puissiez facilement le partager où vous le souhaitez (*y compris [Liens Wink](/link-manager/wink-links)*).
* **Partager**Ouvre une nouvelle fenêtre avec des options de partage sur de nombreux réseaux sociaux et plateformes de messagerie.
* **code QR**Génère un code QR que vous pouvez partager sur des plateformes comme IG qui sont davantage axées sur l'image.
* **Intégrer**Vous montre comment intégrer ce lien en tant que[Bannière publicitaire](/developers/web-components/#content-loader)dans votre site Web.
* **Utiliser avec WordPress**Vous montre comment utiliser notre[plugin WordPress](/developers/wordpress)pour intégrer ce lien dans votre site Web.

Nous couvrons certaines de ces options, plus en détail, ci-dessous.

### Aperçu

<figure>
  <CloudinaryImage width="700" height="373" resize="scale" quality="best" alt="Aperçu" src="academy/shareable-link-preview" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Aperçu du lien partageable*</figcaption>
</figure>


Lorsque vous cliquez sur`Preview`Vous verrez une image avec votre thème appliqué. Lorsque vous partagerez le lien sur les réseaux sociaux, comme Facebook, vous verrez probablement la même image apparaître dans votre fil d'actualité, avec votre titre et votre description.

### Partage social

<figure>
  <CloudinaryImage width="350" height="231" resize="scale" quality="best" alt="Partage social" src="academy/shareable-link-share" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Options de partage social*</figcaption>
</figure>


Si vous cliquez sur le`Share`Cliquez sur le bouton pour ouvrir une nouvelle fenêtre vous permettant de partager directement votre lien sur vos réseaux sociaux et applications de messagerie préférés.
Cliquez sur l'une des icônes pour continuer à publier votre lien sur ce réseau.*Remarque : certains boutons ne s'afficheront que lorsque vous partagez depuis votre appareil mobile.*

### code QR

<figure>
  <CloudinaryImage width="350" height="440" resize="scale" quality="best" alt="Partage social" src="academy/shareable-link-qr-code" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Partager via un code QR*</figcaption>
</figure>


Si vous souhaitez partager un code QR au lieu d'un lien, cliquez sur le`QR code`bouton et une nouvelle fenêtre apparaîtra avec votre code QR.

Comment enregistrer votre code QR :

* Si vous êtes sur votre ordinateur portable, faites un clic droit sur l'image et sélectionnez`Save image`.
* Si vous êtes sur votre téléphone portable :
  * Appuyez et maintenez votre doigt sur l'image jusqu'à ce qu'une option apparaisse pour enregistrer l'image dans vos photos.
  * Prenez une capture d'écran du code QR, recadrez-la pour l'ajuster et enregistrez-la dans Photos.

### Intégrer

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Voici comment intégrer votre lien, sous forme de bannière publicitaire, dans votre site :

* La ligne 3 vous montre comment intégrer les styles Wink dans votre site.
* Les lignes 6 à 9 vous montrent comment utiliser le[chargeur de contenu wink](/developers/web-components/#content-loader)Composant Web et dites-lui de récupérer une bannière publicitaire pour votre code.
* La ligne 11 vous montre comment intégrer notre Javascript dans votre site.
* La ligne 13 vous montre comment intégrer le[chargeur d'application wink](/developers/web-components/#application-loader)Composant Web et dites-lui de récupérer vos préférences de configuration au niveau de la page.
