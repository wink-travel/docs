Les listes sélectionnées sont une fonctionnalité qui vous permet de regrouper et trier l'inventaire que vous avez choisi parmi les éléments trouvés lors de vos recherches d'inventaire de voyage.
Une fois créée, vous pouvez rendre la liste accessible à vos utilisateurs en la mettant en vente sous forme de [grille](/fr/studio/grids) ou de [carte](/fr/studio/maps) via [WinkLinks](/fr/link-manager/wink-links), notre [plugin WordPress](/fr/developers/wordpress/) ou en intégrant l'un de nos [Web Components](/fr/developers/web-components) sur votre site.

Par défaut, chaque compte affilié commence avec une liste `Favoris` que vous pouvez utiliser immédiatement. `Favoris` est spéciale et ne peut pas être supprimée.
Toutes les listes sélectionnées vous permettent de faire ce qui suit :

## Créer une liste

<CloudinaryImage

    <figcaption>*Formulaire de création d'une nouvelle liste sélectionnée*</figcaption>

- Depuis le tableau de bord, cliquez sur `Inventaire > Listes sélectionnées` dans la barre de navigation.
- Créez une nouvelle liste sélectionnée en cliquant sur le bouton `Créer une nouvelle liste`.
- Vous serez invité à nommer votre liste. *Exemple : Top 3 des choix à NYC*.
- Cliquez sur le bouton `Enregistrer`.

## Ajouter un élément à une liste

Vous pouvez ajouter un élément à une liste depuis les résultats de recherche.

- Allez sur [Recherche](/fr/studio/search)
- Recherchez un élément que vous souhaitez ajouter à votre liste.
- Une fois trouvé, cliquez sur le lien `Actions` en bas à gauche de la carte du résultat de recherche.
- Cliquez sur `Ajouter à...` puis sélectionnez la liste sélectionnée à laquelle vous voulez ajouter cet inventaire.
- Retournez à la page des listes sélectionnées pour vérifier que l'élément a bien été ajouté à votre liste.

Chaque liste propose plusieurs actions que vous pouvez exécuter.

- Convertir la liste en [carte](/fr/studio/maps).
- Convertir la liste en [grille](/fr/studio/grids).
- Supprimer la liste (*sauf pour Favoris*).

## API

Les développeurs souhaitant gérer les listes sélectionnées peuvent se rendre dans [Développeurs > API > Parcourir](/fr/developers/apis/#browse-api).
