Les recherches enregistrées font ce que leur nom indique. Lors de la recherche d’inventaire de voyage, vous avez la possibilité d’enregistrer votre requête de recherche pour une utilisation ultérieure.

## Enregistrer votre requête de recherche

Lorsque vous commencez à rechercher de l’inventaire (voir [Search](/fr/studio/search) ), vous pouvez affiner vos filtres de recherche.
En bas du formulaire de requête se trouve l’option `Save search`.

Cliquer sur ce lien vous donnera la possibilité de nommer cette recherche (*par exemple Ma bibliothèque d’hôtels magiques*), un nom que vous retiendrez pour plus tard.

<figure>
<CloudinaryImage
      width="350"

    />
    <figcaption>*Donnez un nom à votre recherche*</figcaption>
</figure>

Donnez-lui un nom et cliquez sur le bouton `Save search`. Ensuite, vous pouvez naviguer vers `Inventory > Saves Searches` depuis la barre de navigation principale pour voir votre nouvel enregistrement.

## Actions sur la liste

Votre recherche enregistrée peut être utilisée pour exposer l’inventaire de voyage à vos utilisateurs sous forme de grille ou de carte et faire afficher vos résultats de recherche comme vous le souhaitez.

<figure>
<CloudinaryImage
      width="700"

Chaque requête de recherche enregistrée dispose d’actions que vous pouvez exécuter :

- Convertir la requête de recherche en [carte](/fr/studio/maps).
- Convertir la requête de recherche en [grille](/fr/studio/grids).
- Supprimer la recherche enregistrée.

## API

Les développeurs souhaitant gérer les recherches enregistrées peuvent se rendre dans [Developers > API > Browse](/fr/developers/apis/#browse-api).
