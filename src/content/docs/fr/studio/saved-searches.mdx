Les recherches enregistrées font ce que leur nom indique. Lorsque vous recherchez des voyages, vous avez la possibilité d'enregistrer votre requête pour une utilisation ultérieure.

## Sauvegarder votre requête de recherche

Lorsque vous commencez à rechercher un inventaire (voir[Recherche](/studio/search)), vous avez la possibilité d'affiner vos filtres de recherche.
En bas du formulaire de recherche se trouve l'option`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Enregistrer une recherche" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Affiche le lien pour enregistrer la requête de recherche*</figcaption>
</figure>


En cliquant sur ce lien, vous aurez la possibilité de nommer cette recherche (*par exemple Ma bibliothèque d'hôtels magiques*) quelque chose dont vous vous souviendrez plus tard.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Donnez un nom à votre recherche" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Donnez un nom à votre recherche*</figcaption>
</figure>


Donnez-lui un nom et cliquez sur le`Save search`bouton. Ensuite, vous pouvez naviguer vers`Inventory > Saves Searches`depuis la barre de navigation principale pour voir votre enregistrement nouvellement créé.

## Liste des actions

Votre recherche enregistrée peut être utilisée pour exposer l'inventaire de voyage à vos utilisateurs sous la forme d'une grille ou d'une carte et faire en sorte que vos résultats de recherche s'affichent comme vous le souhaitez.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Actions de recherche enregistrées" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Requête de recherche enregistrée sans aucun filtre appliqué*</figcaption>
</figure>


Chaque requête de recherche enregistrée comporte des actions que vous pouvez exécuter :

* Convertir la requête de recherche en un[carte](/studio/maps).
* Convertir la requête de recherche en un[grille](/studio/grids).
* Supprimer la recherche enregistrée.

## API

Les développeurs qui souhaitent gérer les recherches enregistrées peuvent se rendre sur[Développeurs > API > Parcourir](/developers/apis/#browse-api).
