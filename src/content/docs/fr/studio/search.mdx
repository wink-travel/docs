## Commencez votre recherche

Voici comment commencer votre recherche de fournisseurs et d’inventaire sur Wink.

1. Choisissez le compte affilié avec lequel vous souhaitez travailler depuis votre [liste de comptes](https://studio.wink.travel/company/list)
2. Accédez à la page de recherche de deux manières :
    1. Vous pouvez faire une recherche rapide de propriétés directement depuis le tableau de bord ou...
    2. Depuis la barre de navigation principale, cliquez sur Inventory → Search
3. La page de recherche vous montrera les derniers fournisseurs et inventaires sur Wink ainsi que les catégories à choisir.
4. Dans notre exemple, cliquez sur la carte qui indique `Hotels`.
5. Vous êtes redirigé vers la page des résultats de recherche. Ici, vous pouvez faire défiler les fournisseurs disponibles dans la catégorie `Hotels` et appliquer / modifier les filtres de recherche à l’aide des filtres situés à gauche de l’écran.

Les résultats de recherche contiennent des informations agrégées sur un fournisseur ou un inventaire ainsi que des actions que vous pouvez exécuter.

## Affiner les résultats de recherche

Voici toutes les façons dont vous pouvez filtrer l’inventaire et les fournisseurs. Cliquez sur l’icône de filtre dans le coin supérieur droit de vos résultats de recherche.

- Nom de la propriété (*ex. Centara*)
- Nom de l’inventaire (*ex. Villa*)
- Types d’inventaire (*ex. Spas*)
- Classement par étoiles (*ex. propriétés 5 étoiles*)
- Note des avis utilisateurs (*ex. 8 / 10*)
- Géolocalisation (*ex. sélectionner un emplacement sur la carte*)
- Continents (*ex. Amérique du Nord*)
- Pays (*ex. Allemagne ou France*)
- Villes (*ex. Tokyo*)
- Modes de vie (*ex. LGBTQ*)
- Types d’emplacement (*ex. Aéroport*)
- Types de segments (*ex. Modéré*)
- Types de catégories (*ex. Lodge*)
- Styles (*ex. Art déco*)
- Relations directes uniquement (*ex. afficher l’inventaire direct*)
- Acceptant les animaux (*ex. afficher l’inventaire acceptant les animaux*)
- Écologique (*ex. afficher l’inventaire écologique*)
- Adapté aux enfants (*ex. afficher l’inventaire adapté aux enfants*)
- Populaire (*ex. afficher l’inventaire à forte valeur*)

## Actions

Les actions vous permettent d’interagir avec l’inventaire ou le fournisseur. Nous avons 3 catégories principales qui vous permettent de faire ce qui suit :

1. **Détails du fournisseur** Comment puis-je en savoir plus sur le fournisseur ?
2. **Créer...** Comment puis-je convertir cet inventaire en quelque chose que je peux vendre ?
3. **Ajouter à...** Comment puis-je enregistrer cet inventaire dans des listes que je peux vendre en même temps ?

Lorsque vous cliquez sur `Supplier details`, vous pouvez :

- **Réserver** Aller directement à la propriété et réserver en utilisant votre ID affilié pour gagner une commission.
- **Demander une connexion directe** Demander au fournisseur s’il souhaite vous donner une connexion directe.
- **Suivre la performance** Créer un graphique analytique pour suivre la performance du fournisseur dans le temps.

Lorsque vous cliquez sur `Add to...`, un menu s’ouvre affichant vos [listes sélectionnées](/fr/studio/curated-lists) existantes et une option pour créer une nouvelle liste. Vous pouvez :

- **Ajouter à une liste existante** Cela ajoute le fournisseur ou l’inventaire à une liste que vous avez déjà créée.
- **Nouvelle liste sélectionnée** Cela vous demandera de créer une nouvelle liste et l’élément y sera ajouté.

## Résultats vides

Si vous recherchez un fournisseur précis mais ne le trouvez pas, un champ vous sera proposé pour compléter votre recherche afin de trouver ce que vous cherchez sur Google Places. S’il existe sur Google Places, vous pouvez générer un `lead` sur Wink.
Cela nous indique que quelqu’un souhaite que nous contactions ce fournisseur pour lui proposer de rejoindre Wink. Nous contacterons le fournisseur et lui demanderons de rejoindre. Vous serez notifié lorsqu’il le fera.
