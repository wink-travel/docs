Si vous êtes une entreprise de voyage et que vous souhaitez externaliser la partie paiement et versement de votre activité à TripPay, voici 6 étapes de base pour commencer :

<Steps>
1. Créez un [compte utilisateur](/fr/user/register) sur Wink.
2. Créez un compte chez [TripPay](https://pay.wink.travel).
3. Créez une [Application](/fr/developers/applications) sur TripPay qui vous permet de communiquer avec notre API.
4. Créez et associez vos bénéficiaires avec TripPay. *par exemple, pour la plupart, cela signifie créer des comptes pour les établissements sur votre plateforme que vous souhaitez que TripPay paie.*
5. Intégrez le [composant web de paiement](/fr/developers/web-components/#payment) à votre site.
6. Envoyez à TripPay le `panier` du voyageur et lancez le processus de paiement.
</Steps>

:::tip
Pour un guide complet sur l’utilisation de TripPay pour votre site de voyage, consultez le guide [Intégration avec TripPay](/fr/guides/integrating-with-trip-pay).
:::

## API

Les développeurs qui souhaitent gérer les `Paiements` peuvent se rendre dans [Développeurs > API > Paiements](/fr/developers/apis/#payment-api).
