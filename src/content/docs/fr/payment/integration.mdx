Si vous êtes une agence de voyages et que vous souhaitez externaliser la partie paiement et décaissement de votre activité à TripPay, voici 6 étapes de base pour commencer :

<Steps>
  1. Créer un [compte utilisateur](/user/register) sur Wink.
  2. Créez un compte avec [TripPay](https://pay.wink.travel).
  3. Créer un [Application](/developers/applications) sur TripPay qui vous permet de communiquer avec notre API.
  4. Créez et cartographiez vos bénéficiaires avec TripPay. *Par exemple, pour la plupart, cela signifie créer des comptes pour les propriétés sur votre plateforme que vous souhaitez que TripPay paie.*
  5. Intégrer le [composant Web de paiement](/developers/web-components/#payment) dans votre site.
  6. Envoyer TripPay au voyageur `shopping cart` et lancer le processus de paiement.
</Steps>


:::conseil
Pour un guide complet sur l'utilisation de TripPay pour votre site de voyage, consultez le guide.[Intégration avec TripPay](/guides/integrating-with-trip-pay).
:::
