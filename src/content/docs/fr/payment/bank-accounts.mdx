Pour gérer vos comptes bancaires, cliquez sur `Compte > Comptes bancaires` dans la barre latérale.

## Ajouter un compte bancaire

Cliquez sur `Ajouter un compte bancaire` pour enregistrer un nouveau compte. Le formulaire s’adapte en fonction du pays sélectionné :

- **Pays de paiement** — Le pays où se trouve votre banque.
- **Devise** — La devise dans laquelle vous souhaitez recevoir les paiements.
- **Nom du titulaire du compte** — Le nom figurant sur le compte bancaire.
- **Détails de routage** — Champs spécifiques au pays tels que le numéro de routage (US), le code Sort (UK), le code SWIFT/BIC ou l’IBAN.
- **Numéro de compte** — Votre numéro de compte bancaire.

## Compte principal

Vous pouvez désigner un compte bancaire comme **Principal**. Votre compte principal reçoit tous les paiements automatiques. Les comptes supplémentaires peuvent être utilisés pour des retraits manuels.

Pour changer votre compte principal, cliquez sur `Définir comme principal` sur n’importe quel compte de votre liste.

## Gestion des comptes

Depuis la liste des comptes bancaires, vous pouvez :

- Modifier les détails du compte.
- Supprimer un compte dont vous n’avez plus besoin.
- Changer le compte principal.
