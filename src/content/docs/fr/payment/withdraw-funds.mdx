:::note
Sauf si vous êtes un intégrateur, vous n'avez pas à vous soucier de créer un compte sur TripPay. L'intégrateur le fera pour vous au moment où vous créez un compte chez lui.
:::

Les bénéficiaires utilisent TripPay pour :

- Suivre combien d'argent ils ont gagné.
- Combien est actuellement sur leur compte.
- ...et quel montant est disponible pour retrait.

## Utilisation

Un bénéficiaire classique interagit généralement avec TripPay de cette manière :

<Steps>
1. TripPay retient des fonds provenant de 10 réservations, issues de 3 plateformes de voyage différentes, pour l'Hôtel X et les fonds sont maintenant disponibles.
2. L'Hôtel X reçoit un e-mail de TripPay indiquant `Vous avez des fonds disponibles` avec un lien pour gérer leur compte.
3. L'Hôtel X se connecte à [TripPay](https://pay.wink.travel).
4. L'Hôtel X voit les réservations du mois dernier ainsi que les analyses du compte.
5. L'Hôtel X choisit de retirer des fonds.
6. L'Hôtel X sélectionne la méthode `Paiement`.
7. L'Hôtel X suit les étapes requises pour traiter les fonds.
8. L'Hôtel X soumet la demande de transfert et clique sur `OK`.
</Steps>

*Les fonds sont généralement disponibles sous 24 heures dans toutes les régions.*

:::note
Une fois que vous atteignez un certain volume et montant, nous pouvons vous contacter pour obtenir plus de documents vérifiant votre identité et/ou celle de votre entreprise.
:::
