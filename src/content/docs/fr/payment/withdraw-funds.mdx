:::note
Sauf si vous êtes un intégrateur, vous n'avez pas à vous soucier de créer un compte sur TripPay. L'intégrateur s'en chargera pour vous lors de la création de votre compte.

Les bénéficiaires utilisent TripPay pour :

* Suivez combien d’argent ils ont gagné.
* Combien y a-t-il actuellement sur leur compte ?
* ...et quel montant est disponible pour être retiré.

## Usage

Un bénéficiaire normal interagit généralement avec TripPay comme ceci :

<Steps>
  1. TripPay conserve les fonds de 10 réservations, provenant de 3 plateformes de voyage différentes, pour l'hôtel X et les fonds sont désormais disponibles.
  2. L'hôtel X reçoit un e-mail de TripPay indiquant `You have funds available` et avec un lien pour gérer leur compte.
  3. L'hôtel X se connecte à [TripPay](https://pay.wink.travel).
  4. L'hôtel X voit les réservations du mois dernier ainsi que les analyses de compte.
  5. L'hôtel X choisit de retirer des fonds.
  6. L'Hôtel X choisit le `Pay-out` méthode.
  7. L'hôtel X suit les étapes requises pour traiter les fonds.
  8. L'hôtel X soumet une demande de transfert et clique `OK`.
</Steps>


*Les fonds sont généralement disponibles dans les 24 heures dans toutes les régions.*

:::note
Une fois que vous aurez atteint un certain volume et un certain montant, nous pourrons vous contacter pour obtenir des documents supplémentaires confirmant votre identité et/ou celle de votre entreprise.
:::
