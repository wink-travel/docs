Votre contenu est disponible sous l’onglet `Entries`.

Nous supportons trois principaux types de contenu :

- URL accessibles publiquement qui renvoient à des pages web.
- Fichiers qui ont été téléchargés via WinkLinks.
- Contenu textuel créé via notre fonctionnalité `Quotes`.

## Ajouter du contenu

Voici comment ajouter du contenu à WinkLinks. Sous l’onglet `Entries`, commencez par cliquer sur le bouton `Add entry`.

### Lien normal

:::tip
Le lien doit pointer vers une page web. Par exemple, vous ne pouvez pas lier directement un PDF. Si vous souhaitez partager un PDF, vous devez utiliser la fonctionnalité `Upload file`.
:::

Voici les étapes pour ajouter un lien normal :

<Steps>
1. Saisissez le lien que vous souhaitez partager. *ex. https://trvl.as/3xWCH*
2. Une fois le lien saisi, Wink recherche les détails de ce site web et remplit le formulaire que vous voyez ensuite.
3. Vous pouvez choisir de modifier certaines données, comme le nom du site, le titre et la description.
4. Vous pouvez également choisir de télécharger une image personnalisée.
5. Cliquez sur le bouton `Save` lorsque vous êtes prêt à continuer.
</Steps>

### Télécharger un fichier

:::caution[Exigences]
- Les images ne doivent pas dépasser 10 Mo.
- Les vidéos ne doivent pas dépasser 50 Mo.
- Nous supportons tous les formats reconnus d’images et de vidéos, y compris le PDF.
:::

:::note[Recommandations]
- 2560w x 1440h (format 16:9) pour votre image mise en avant.
- 2560w x 1920h (format 4:3) pour toutes les autres images.
- La taille de l’image ne doit pas dépasser 5 Mo. Veuillez compresser vos images avant de les télécharger sur Wink.
:::

Voici les étapes pour ajouter un fichier :

<Steps>
1. Les médias peuvent être ajoutés :
    - En faisant glisser des images depuis votre ordinateur vers la fenêtre ou en cliquant sur `Browse`.
    - En collant une URL externe pointant vers une image.
    - En utilisant la caméra de votre ordinateur portable.
    - Depuis votre compte Google Drive.
    - Depuis votre compte Dropbox.
    - Depuis Shutterstock
    - Depuis gettyimages
    - Depuis iStock
    - Depuis Unsplash
2. Une fois le fichier téléchargé, vous pouvez ajouter plus de métadonnées pour décrire votre fichier, comme le titre et la description.
3. Cliquez sur le bouton `Save` lorsque vous êtes prêt à continuer.
</Steps>

### Écrire une citation

:::tip
Une citation est un excellent moyen d’ajouter un séparateur pour aérer une longue liste [...et cela vous donne aussi l’air intelligent et cultivé 😉].
:::

Voici les étapes pour écrire une citation :

<Steps>
1. Tapez la citation *ex. Roses are red. Violets are blue...*.
2. Saisissez le nom de l’auteur original. *ex. Edmund Spenser*
3. Saisissez le nom du site d’où vous avez obtenu la citation. *ex. Daily Quotes*
4. Cliquez sur le bouton `Save` lorsque vous êtes prêt à continuer.
</Steps>

## Mettre à jour le contenu

Tout le contenu peut être facilement modifié en cliquant sur le lien `Actions` et en sélectionnant `Update`. Vous pouvez supprimer du contenu de manière similaire en cliquant sur le bouton `Remove`.

## Supprimer du contenu

Tout le contenu peut être facilement modifié en sélectionnant `Update`. Vous pouvez supprimer du contenu de manière similaire en cliquant d’abord sur le lien `Actions` puis sur le bouton `Remove`.

## Trier le contenu

Vous pouvez ordonner votre contenu en cliquant sur l’icône située en bas à gauche de chaque entrée et en la faisant glisser à la position souhaitée.

## Grouper le contenu

:::tip
Créez vos tags avant de créer vos entrées. Sinon, vous perdez votre progression et devez recommencer.
:::

Lorsque vous modifiez une entrée, vous avez la possibilité d’ajouter un ou plusieurs tags à des fins de regroupement. Faites défiler jusqu’en bas du formulaire pour trouver le champ `Tags`.

Les utilisateurs, en consultant votre page WinkLinks, peuvent choisir comment ils souhaitent consommer votre contenu (en liste ou en grille) et peuvent sélectionner un ou plusieurs tags pour affiner davantage le filtrage de votre contenu.
