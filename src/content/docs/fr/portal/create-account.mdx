Wink vous connecte à l'inventaire de voyages dans le monde entier. Cet article vous guide pas à pas pour créer votre premier compte sur Wink.

<Steps>
1. Rendez-vous sur [app.wink.travel](https://app.wink.travel).
2. Connectez-vous en cliquant sur `Sign-in / Register` en haut à droite.
3. Une fois connecté, cliquez sur `Accounts` dans le menu latéral gauche.
4. Cliquez sur le bouton `Create` pour commencer à créer un nouveau compte.
5. Choisissez d'abord le type de compte que vous souhaitez créer.
6. **Nom** Le nom de votre compte. *Ce n’est pas nécessairement votre vrai nom, mais ce sera ce que les utilisateurs verront lorsqu’ils consulteront votre page WinkLinks ou lorsque les fournisseurs voudront vous contacter directement.*
7. **Type d’entité** Indiquez si vous êtes une entreprise ou un particulier. *Principalement utilisé pour déterminer votre éligibilité à recevoir des fonds.*
8. **Type** Choisissez un type qui correspond le mieux à votre compte. *ex. Influenceur*
9. **Description** Rédigez une description pour votre compte. *Celle-ci sera visible sur votre page WinkLinks et lorsque les fournisseurs voudront vous contacter directement.*
10. **Site web** Saisissez votre site principal. *Cela peut être l’URL de votre compte IG ou le site de votre entreprise. Ce doit être l’endroit où vous effectuez la majeure partie de votre travail en ligne. Les fournisseurs consulteront ces liens pour décider s’ils veulent se connecter avec vous.*
11. **Ville** Où êtes-vous basé ?
12. Acceptez nos Conditions générales et Conditions de paiement.
13. Cliquez sur le bouton `Save`.
14. Vous serez redirigé vers la page `Manage account` où vous pourrez ajouter plus de détails à votre compte.
</Steps>

Félicitations 🎉 Vous venez de créer votre premier compte sur Wink.

:::tip[Astuce pro]
Un compte propriété agit également comme un compte affilié. Si vous êtes une propriété, vous n’avez pas besoin de créer deux comptes...
Vous pouvez gérer et vendre votre inventaire avec le même compte.
:::
