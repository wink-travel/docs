Cet article suppose que vous avez déjà un compte utilisateur Wink.
:::

[Wink Agent](https://agent.wink.travel) vous connecte à l'inventaire de voyages dans le monde entier. Cet article vous guide à travers la création et la gestion de vos comptes d'agent de voyage sur Wink.

## Créer un compte

Une fois authentifié, vous serez redirigé vers votre [page des comptes](https://agent.wink.travel/company/list) qui affiche tous vos comptes d'agent de voyage.

Cliquez sur le bouton `Create new account` pour continuer. Vous serez redirigé vers notre page de [création de compte](https://agent.wink.travel/company/create).

Remplissez tous les champs obligatoires.

<Steps>
1. **Nom** Le nom de votre compte. *Ce n'est pas nécessairement votre vrai nom mais ce sera ce que les utilisateurs verront lorsqu'ils consulteront votre page WinkLinks ou lorsque les fournisseurs voudront se connecter directement avec vous.*
2. **Type d'entité** Si vous êtes une entreprise ou un particulier. *Principalement utilisé pour déterminer votre éligibilité à recevoir des fonds.*
3. **Type** Choisissez un type qui correspond le mieux à votre compte. *ex. Influenceur*
4. **Description** Rédigez une description pour votre compte. *Celle-ci sera visible sur votre page WinkLinks et lorsque les fournisseurs voudront se connecter directement avec vous.*
5. **Site web** Entrez votre site web principal. *Cela peut être l'URL de votre compte IG ou le site web de votre entreprise. Ce doit être l'endroit où vous effectuez la majeure partie de votre travail en ligne. Les fournisseurs consulteront ces liens pour décider s'ils veulent se connecter avec vous.*
6. **Ville** Où êtes-vous basé ?
</Steps>

### Conditions de paiement

:::note
Si vous choisissez d'être responsable du paiement, vous n'aurez accès qu'aux propriétés avec lesquelles vous avez une connexion directe.
:::

:::caution
Wink décline toute responsabilité pour le paiement des hôtels dans ce scénario.
:::

:::note
Vous ne pouvez interagir avec cette option que lors de la création du compte. La plupart des agents de voyage devraient laisser ce paramètre désactivé 🛑 et utiliser ce portail à la place.
:::

Les agents de voyage enregistrés qui s'intègrent avec nous via notre API ont la possibilité de prendre en charge le paiement et de verser ces fonds à nos propriétés.

<Steps>
1. **Je veux prendre en charge le paiement** L'agent de voyage prend la responsabilité de collecter les fonds.
2. **Je veux payer les hôtels** L'agent de voyage est responsable du paiement des hôtels.
</Steps>

Les hôtels qui souhaitent se connecter directement avec vous verront un avertissement et devront cliquer pour reconnaître que Wink n'est pas responsable du paiement des hôtels, afin d'établir une relation avec votre compte.

### Accord

<Steps>
1. **Accord** Acceptez notre [contrat d'affiliation](/fr/portal/travel-agent/terms-of-service) et nos [conditions de paiement](/fr/portal/travel-agent/payment-terms) pour continuer.
2. Cliquez sur le bouton `Save`.
</Steps>

Félicitations 🎉 ...vous avez créé votre premier compte affilié sur Wink.

Une fois créé, vous pouvez personnaliser davantage votre compte en remplissant votre adresse et en ajoutant une photo de profil à votre compte.

<Steps>
1. Cliquez sur votre nouveau compte d'agent de voyage. *Cela vous mènera à votre tableau de bord de compte.*
2. Cliquez sur `Account > Profile` dans la barre de navigation principale.
3. Sur la page Profil, vous pourrez mettre à jour votre profil existant et y ajouter des informations supplémentaires.
</Steps>

:::note
La photo de profil de votre compte d'agent de voyage ne doit pas être confondue avec votre photo de profil utilisateur Wink. La photo de profil de votre compte d'agent de voyage sera affichée sur votre page WinkLinks et visible par les fournisseurs souhaitant se connecter avec vous.

Votre photo de profil utilisateur est principalement destinée à un usage interne.
:::

## Gestionnaires

Vous pouvez inviter d'autres utilisateurs à vous aider à gérer votre compte d'agent de voyage. Une fois qu'un utilisateur accepte votre invitation, il aura un accès complet à votre compte à l'exception des paiements. Vous resterez toujours le seul responsable des fonds que vous possédez.

<Steps>
1. Sélectionnez le compte d'agent de voyage avec lequel vous souhaitez travailler depuis la [page des comptes](https://agent.wink.travel/company/list).
2. Cliquez sur `Account > Managers` dans la barre de navigation principale.
3. Ajoutez l'e-mail de l'utilisateur que vous souhaitez nommer gestionnaire de compte.
4. Cliquez sur le lien `Invite` à côté de l'e-mail.
</Steps>

Wink enverra un e-mail à l'utilisateur pour accepter ou refuser votre invitation.

## Notifications

Parfois, nous souhaitons vous contacter. Lorsque cela se produit, vous verrez la cloche en haut à droite changer.
Pour lire ce que nous vous avons envoyé :

<Steps>
1. Cliquez sur `Account > Notifications` dans la barre de navigation principale.
2. Vous verrez les notifications nouvelles et passées.
3. Choisissez de `Marquer comme lu` ou de `Supprimer` les notifications une fois que vous les avez lues.
</Steps>

## API

Les agents de voyage sont toujours considérés comme des affiliés. Les développeurs qui souhaitent gérer les `Affiliates` peuvent se rendre dans [Developers > API > Affiliate](/fr/api).
