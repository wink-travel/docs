Lorsqu'une réservation est effectuée sur Wink, les actions suivantes se produisent :

<Steps>
1. Votre channel manager, CRS ou PMS est notifié.
2. Vous recevez un e-mail de confirmation *(contacts du bureau de réservation)* avec un lien pour gérer votre nouvelle réservation.
3. Le voyageur reçoit un e-mail de confirmation qui inclut également les coordonnées de votre bureau de réservation.
4. L'agent de voyage reçoit la même confirmation de réservation que le voyageur.
4. Les développeurs peuvent écouter l'événement webhook `booking.created` et recevoir des informations détaillées sur la réservation en temps réel.
</Steps>

Suivez soit le lien dans l'e-mail, soit allez dans `Compte > Réservations` depuis la barre de navigation principale pour commencer.

## Grille des réservations

La grille des réservations vous montre toutes vos réservations passées et futures, triées par date de réservation dans l'ordre croissant.

### Filtres

Vous pouvez appliquer des filtres pour trouver les réservations que vous recherchez.

- Date de réservation [plage]
- Date d'arrivée [plage]
- Date de départ [plage]
- Code de réservation
- Nom de famille du voyageur
- Prénom du voyageur
- Nom du tarif principal
- Nom de l'établissement

### Export

Vous pouvez exporter les réservations correspondant à vos critères de filtre actuels au format CSV.

Pour exporter vos réservations :

<Steps>
1. Assurez-vous que les filtres souhaités sont actifs.
2. Attendez que la grille des réservations se rafraîchisse.
2. Cliquez sur l'icône 🗂️ (*Export fichier*) en bas à droite de la grille.
</Steps>

## Réservation

Sélectionnez la réservation avec laquelle vous souhaitez travailler depuis la grille des réservations et vous serez redirigé vers la page `Réservation`.
Cette page contient tous les détails dont vous avez besoin pour bien accueillir votre client le jour de son arrivée.

La page contient également quelques actions importantes pour bien servir vos clients. Vous pouvez trouver toutes les actions disponibles dans le menu déroulant `Actions` en haut de la page.

### Renvoyer l’e-mail

Pour renvoyer l’e-mail de confirmation au voyageur, cliquez sur `Actions > Renvoyer l’e-mail de confirmation`.

### Annuler la réservation

Pour annuler la réservation en tant qu’établissement, cliquez sur `Actions > Annuler la réservation`.

Une fenêtre pop-up s’affichera et vous permettra de :

<Steps>
1. Sélectionner le type d’annulation. *ex. Dates incorrectes*
2. Saisir un motif pour l’annulation.
</Steps>

*Ce bouton suit les règles de votre politique d’annulation et ne sera donc pas toujours disponible.*

## API

Les développeurs souhaitant gérer les `Réservations`, en tant qu’agent de voyage, peuvent se rendre dans [Développeurs > API > Agent de voyage](/fr/api).
