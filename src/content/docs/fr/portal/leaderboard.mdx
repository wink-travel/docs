Le classement montre comment votre établissement se positionne par rapport aux autres sur la plateforme Wink.

## Classements géographiques

Les classements sont disponibles à quatre niveaux géographiques :

- **Ville** — Votre classement dans votre ville.
- **Pays** — Votre classement dans votre pays.
- **Continent** — Votre classement sur votre continent.
- **Monde** — Votre classement à l’échelle mondiale.

## Filtrage

Utilisez le menu déroulant de recherche de ville pour changer le focus géographique. Les classements se mettent à jour automatiquement en fonction de la localisation sélectionnée. Les classements sont spécifiques à la devise, reflétant la performance sur le marché choisi.
