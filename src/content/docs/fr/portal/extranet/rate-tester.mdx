Le testeur de tarifs vous permet de vérifier que vos tarifs de chambre sont correctement configurés avant leur mise en ligne pour les voyageurs.

Pour accéder au testeur de tarifs, cliquez sur `Monetize > Rate Tester` dans la barre de navigation principale.

## Comment ça fonctionne

Saisissez un scénario de recherche et le testeur de tarifs vous montrera exactement comment les tarifs sont calculés :

- **Canal de vente** — Sélectionnez le canal à tester (par exemple, votre moteur de réservation, un affilié ou une intégration spécifique).
- **Dates d’arrivée / de départ** — Les dates de voyage à tester.
- **Configuration de la chambre** — Nombre d’adultes et d’enfants.

Cliquez sur `Search` pour voir les résultats.

## Résultats

La page des résultats affiche les tarifs calculés pour chaque type de chambre et plan tarifaire correspondant à vos critères de recherche. Utilisez cela pour vérifier que :

- Les modificateurs de prix (occupation simple, adultes/enfants supplémentaires) sont appliqués correctement.
- Les politiques d’annulation s’affichent comme prévu.
- Les restrictions sont correctement filtrées.
- Les promotions et remises sont bien prises en compte.
