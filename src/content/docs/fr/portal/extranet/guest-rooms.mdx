Chaque parcours de réservation commence par une chambre d'hôtes. Il existe de nombreuses fonctionnalités autour des chambres d'hôtes et cela peut être un peu déroutant au début.
Cet article couvrira tout ce que vous devez savoir pour devenir un spécialiste des chambres d'hôtes sur Wink.

Les chambres d'hôtes, ou types de chambres comme on dit dans `le métier`, vous permettent de présenter et de vendre vos chambres, sur place ou hors site. Pour créer une chambre d'hôtes, cliquez sur le bouton `Créer une nouvelle chambre d'hôtes`.

Les sections suivantes vous guident à travers chaque partie de la gestion d'une chambre d'hôtes.

## Configuration

L'onglet `Configurations` est sélectionné par défaut lorsque vous commencez.

Cette section couvre les bases de votre chambre d'hôtes afin que nous sachions à qui nous pouvons vendre cette chambre.

- **Nom du type de chambre** Le nom de la chambre, tel que vous le désignez en interne. *ex. Supreme deluxe*
- **Non-fumeur** Indique si la chambre est non-fumeur ou non.
- **Tarif affiché** Le tarif publié pour une nuitée sans aucun rabais ni supplément inclus. *ex. 100. Notez que nous utilisons ce chiffre uniquement pour afficher le prix attendu aux canaux de vente ou pour pré-remplir certains champs lors de la saisie de formulaires.*
- **Meilleur tarif disponible** Le meilleur tarif disponible pour une nuitée sans aucun rabais ni supplément inclus. *ex. 75. Notez que nous utilisons ce chiffre uniquement pour afficher le prix attendu aux canaux de vente ou pour pré-remplir certains champs lors de la saisie de formulaires.*
- **Quantité** Le nombre d'unités que vous avez de ce type de chambre à une date donnée. *ex. 40*

### Occupation

:::caution
L'occupation joue un rôle important dans la disponibilité de votre chambre d'hôtes pour un itinéraire spécifique. Beaucoup de ces champs seront pré-remplis lors de la création initiale de la chambre. Assurez-vous de les modifier pour qu'ils correspondent à votre type de chambre.
:::

- **Occupation min.** Le nombre minimum de personnes autorisées à séjourner dans cette chambre. *ex. 1*
- **Occupation adulte incluse** Nombre d'adultes inclus dans le prix d'une nuitée. *ex. 2. Note : Si la chambre supporte plus que ce nombre, ils sont calculés via le modificateur `Adulte supplémentaire` sur votre plan tarifaire.*
- **Occupation max.** Le nombre maximum de personnes (adultes + enfants) autorisées à séjourner dans cette chambre. *ex. 4*
- **Occupation adulte max.** Le nombre maximum d'adultes autorisés à séjourner dans cette chambre. *ex. 3*
- **Occupation enfant incluse** Nombre d'enfants inclus dans le prix d'une nuitée. *ex. 1. Note : Si la chambre supporte plus que ce nombre, ils sont calculés via le modificateur `Enfant supplémentaire` sur votre plan tarifaire.*
- **Occupation enfant max.** Le nombre maximum d'enfants autorisés à séjourner dans cette chambre. *ex. 2*

### Autres

Cette section est à titre informatif uniquement. Les voyageurs cherchant des chambres plus grandes et des suites trouveront ces informations intéressantes.

- **Nombre de salles de bain** Le nombre de salles de bain pour ce type de chambre. *ex. 1*
- **Nombre de salons** Le nombre de salons pour ce type de chambre. *ex. 1*
- **Nombre maximum de lits d'appoint** Le nombre de lits d'appoint supportés par ce type de chambre. *ex. 1*
- **Nombre maximum de lits bébé** Le nombre de lits bébé supportés par ce type de chambre. *ex. 1*

### Classification de la chambre

- **Segment** Le segment sous lequel la chambre d'hôtes est catégorisée. *ex. Budget*
- **Emplacement** Le type d'emplacement sous lequel la chambre d'hôtes est catégorisée. *ex. Étage élevé*
- **Vue** Le type de vue à attendre de cette chambre d'hôtes. *ex. Vue sur la ville*
- **Classification** La classification sous laquelle cette chambre d'hôtes est classée. *ex. Économique*
- **Architecture** Le style architectural autour de cette chambre d'hôtes. *ex. Asiatique*
- **Étage** Un ou plusieurs étages optionnels où cette chambre d'hôtes peut se trouver *ex. À partir du 20e étage*

### Composition

La composition s'applique uniquement si la chambre d'hôtes est composée de plusieurs chambres.

- **Composition** Interrupteur à bascule à ✅ si la chambre d'hôtes est composée de plusieurs chambres. *ex. Oui*
- **Nombre** Indiquez combien de chambres composent cette chambre d'hôtes. *ex. 2*

### Détails de l'auberge

Cette section s'applique uniquement si vous gérez un dortoir, une auberge ou un autre service de chambre partagée.

- **Chambre partagée** Interrupteur à bascule à ✅ si la chambre d'hôtes est une chambre partagée. *ex. Oui*
- **Genre** Indiquez le genre pour cette chambre d'hôtes. *ex. Masculin*

## Chambres

:::note
Il existe un nombre infini de configurations possibles pour les chambres. Nous avons essayé de rendre cela aussi personnalisable que possible pour s'adapter à votre établissement.
:::

Cliquez sur l'onglet `Chambres` pour continuer.

La section chambres vous permet de spécifier la configuration de votre ou vos chambres et d'indiquer ce qu'elles contiennent.

Les chambres du même type peuvent avoir plusieurs configurations, ou `Configurations de literie`. Une chambre d'hôtes doit avoir au moins une configuration.

- **Nom de la configuration** Donnez un nom à la configuration que vous comprendrez *ex. Principal*

Vous pouvez avoir plusieurs chambres dans une configuration.

- **Type de chambre** Chaque chambre a un type. *La première est `Maître` par défaut, mais vous pouvez la modifier dans votre configuration.*

Dans une chambre, vous pouvez avoir plusieurs lits de types et quantités différentes.

- **Type de lit** Choisissez le type de lit dans votre chambre. *ex. King*
- **Quantité** Choisissez le nombre de lits de ce type dans cette chambre. *ex. 1*

## Équipements

:::note
Cette section est optionnelle.
:::

Cliquez sur l'onglet `Équipements` pour continuer.

Comme pour les équipements de l'établissement, les équipements de la chambre d'hôtes s'appliquent uniquement à cette chambre spécifique. Cochez chaque équipement applicable.

## Descriptions

:::note
Vous devez écrire au moins une description et elle doit être en anglais.
:::

Cliquez sur l'onglet `Descriptions` pour continuer.

Comme pour le texte d'accueil de l'établissement, vous pouvez créer des descriptions localisées pour la chambre d'hôtes également.
Vous pouvez écrire dans autant de langues que vous le souhaitez. Nous ajouterons des traductions pour toutes les langues les plus populaires.
C'est le texte que les voyageurs verront lorsqu'ils consulteront vos chambres d'hôtes.

- **Nom** Donnez un nom à la chambre d'hôtes. *ex. Supreme deluxe*
- **Description** Décrivez la chambre d'hôtes en un ou deux paragraphes. *ex. C'est la meilleure chambre d'hôtes du monde entier...*

## Réseaux sociaux

:::note
Cette section est optionnelle.
:::

Cliquez sur l'onglet `Réseaux sociaux` pour continuer.

Vous vous demandez peut-être, "Pourquoi une chambre d'hôtes aurait-elle sa propre section réseaux sociaux ?". Une chambre d'hôtes peut aussi être une villa proche de l'établissement mais hors site avec son propre personnel et son propre profil IG.

Les réseaux sociaux vous permettent de rester en contact avec les voyageurs nouveaux et existants au fil du temps.
Ajoutez vos comptes de réseaux sociaux à votre profil pour que les clients puissent facilement vous contacter et en savoir plus sur votre chambre d'hôtes.

<Steps>
1. Activez l'interrupteur du réseau social que vous souhaitez activer.
2. Saisissez le nom de votre compte sur ce réseau.
</Steps>

## Réputation

:::note
Cette section est optionnelle.
:::

Cliquez sur l'onglet `Réputation` pour continuer.

Comme pour l'onglet `Réseaux sociaux`, vous utiliserez cet onglet si cette chambre d'hôtes a un ou plusieurs scores de réputation distincts ou des distinctions supplémentaires que vous souhaitez mettre en avant ici.

Votre réputation est votre monnaie digitale et vous voulez pouvoir l'emporter partout avec vous.
Une entrée de réputation peut être tout ce que votre établissement souhaite valoriser, comme une note tierce ou pour le concours où votre chef a remporté une médaille d'or.
Dans cette section, vous pouvez ajouter vos scores et classements provenant de toute la toile.

Cliquez sur le bouton `Ajouter un score de réputation`.

<Steps>
1. **Catégorie** Choisissez le type de score que vous souhaitez ajouter. *ex. Avis tiers*
2. **Fournisseur** Indiquez qui vous a attribué ce score. *ex. BigOTA.com*
3. **Type de notation** Sélectionnez la forme sous laquelle vous avez reçu le score. *ex. Numérique*
4. **Date** Facultatif, saisissez la date à laquelle vous avez reçu le score.
5. **Note** Saisissez la note que vous avez reçue. *ex. 8.5*
6. **Note maximale** Facultatif, saisissez la note maximale possible pour ce type de distinction. *ex. 10*
</Steps>

## Vente basée sur les attributs

:::note
Cette section est optionnelle.
:::

Cliquez sur l'onglet `Vente basée sur les attributs` pour continuer.

Toutes nos installations disposent de leur propre section ABS. Cette section vous permet de créer des opportunités de vente additionnelle sur l'inventaire principal ou de rendre un type d'inventaire non transactionnel transactionnel.
Nous vous recommandons vivement de créer ces opportunités pour potentiellement toucher le client à un prix plus élevé lors du processus de réservation.

:::tip
Une chambre d'hôtes offre de nombreuses possibilités d'inclure des services annexes :

- Matelas supérieur.
- Kit de salle de bain deluxe.
- Enceinte Bluetooth Marshal.
- Bureau avec chaise extra confortable.
- Petit-déjeuner en chambre
- Massage en chambre
- Pack de 12 bières
- Bouteille de vin
- Chocolat premium
- +++ laissez libre cours à votre imagination et gagnez un peu d'argent en plus !!
:::

- **Niveau de prix** Donnez au voyageur une indication rapide du coût de cette chambre d'hôtes. Exactement... pour la chambre d'hôtes. Pas pour les services annexes.
- **Actif** Interrupteur à bascule à 🛑 si vous souhaitez que cette chambre d'hôtes devienne indisponible à la réservation sur tous les tarifs principaux et canaux de vente.
- **Commisssionnable** Indiquez si les affiliés touchent une commission sur cet inventaire.
- **Mis en avant** Indiquez si cette chambre d'hôtes doit être mise en avant si l'option se présente.
- **Vente basée sur les attributs** Interrupteur à bascule à ✅ pour activer l'ABS sur cet inventaire. *Une fois activé, vous devez ajouter au moins 1 article ABS à la vente*

### Article annexe

Lorsque vous activez l'ABS, vous pouvez commencer à ajouter des articles annexes et les rendre disponibles à la vente.

*Par défaut, un article vide a déjà été créé pour vous. Vous pouvez en ajouter d'autres en cliquant sur le bouton `Ajouter un article à la vente`.*

- **Nom** Donnez un nom à votre article annexe. *ex. Truffes au chocolat et champagne par Royce*
- **Type de tarification** Indiquez comment vous souhaitez calculer le prix. *ex. Location de paddle. Réglez sur `Par heure` pour indiquer que le prix est basé sur le nombre d'heures louées.*
- **Prix de base** Saisissez le prix régulier de cet article. *ex. 20 / heure*
- **Prix remisé** Saisissez le prix que les utilisateurs Wink obtiennent lorsqu'ils réservent l'article avec la chambre. *ex. 15 / heure*

### Limites

Vous pouvez restreindre la disponibilité de cet article en définissant des limites min / max sur la quantité.
Si nous continuons notre exemple de location de paddle à l'heure mais que nous voulons exiger que le client réserve au moins 2 paddles en même temps, nous ferions :

- **Quantité minimale** Restreindre la quantité minimale à réserver. *ex. 2 paddles*
- **Quantité maximale** Restreindre la quantité maximale à réserver. *ex. laisser vide pour aucune restriction max*

### Descriptions

:::note
Vous devez écrire au moins une description et elle doit être en anglais.
:::

- **Nom** Donnez un nom à votre article annexe. *ex. Stand Up Paddleboard*
- **Description** Décrivez l'article annexe en un ou deux paragraphes. *ex. Ne restez pas assis - Prenez votre pagaie !*

## Proximité

Cliquez sur l'onglet `Proximité` pour continuer.

Tout l'inventaire n'est pas situé sur place. Si vous avez un inventaire situé hors site, informez-en le voyageur ici et vous aurez la possibilité de définir un emplacement de carte distinct ainsi que des informations de contact optionnelles.

*Cela peut être utile lorsque, par exemple, une villa appartenant à l'hôtel est située à quelques kilomètres de l'hôtel et que le voyageur veut expliquer au chauffeur de taxi comment s'y rendre.*

Cliquez sur le bouton `Enregistrer` pour continuer.
