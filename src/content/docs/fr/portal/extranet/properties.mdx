Sur cette page, vous apprendrez tout ce que vous devez savoir sur la création et la gestion des propriétés sur Wink.

## Créer une propriété

Une fois authentifié, vous serez redirigé vers votre [page des propriétés](https://extranet.wink.travel/property/list) qui affiche toutes vos propriétés existantes.
Si vous en avez beaucoup, vous pouvez filtrer vos propriétés en tapant un nom.

Cliquez sur le bouton `Créer une nouvelle propriété` pour continuer. Vous serez redirigé vers notre page de [création de propriété](https://extranet.wink.travel/property/create).

Il y a trois chemins possibles pour commencer l’intégration de votre propriété.

1. **Manuel** L’approche du bricoleur pour `Tout faire vous-même !`
2. **Réclamer** Prenez possession d’un `lead` et réclamez-le comme votre propriété.
3. **Intelligent** Utilisez l’IA pour automatiser certaines étapes du processus d’intégration.

### Manuel

*Temps approximatif pour compléter : ~5 minutes*

Notre processus d’intégration manuel vous permet de créer une propriété avec le minimum de champs nécessaires pour commencer.
Notre assistant facile à utiliser vous guide à travers tout ce qui est requis pour activer votre profil et le mettre en ligne.

<Steps>
1. **Ville** Dites-nous où se trouve votre propriété *Ex. Estepona, Espagne*.
2. **Nom** Le nom de votre propriété. *Ex. Santiva Resort*
3. **Nom local** Si votre propriété se trouve dans un pays qui n’utilise pas l’alphabet latin, tapez le nom local de votre propriété. *Ex. ซานทิวา รีสอร์ท en thaï*
4. **Adresse** Remplissez l’adresse de la propriété
5. **Bureau de réservation** Remplissez les détails du bureau de réservation.
6. **Accord** Acceptez nos [conditions d’utilisation](/fr/portal/extranet/terms-of-service) et [conditions de paiement](/fr/portal/extranet/payment-terms) pour continuer.
7. Cliquez sur le bouton `Enregistrer`.
</Steps>

Vous êtes redirigé vers votre page des propriétés.

Félicitations 🎉 ...vous avez créé votre première propriété sur Wink de manière manuelle.

### Réclamer

*Temps approximatif pour compléter : ~2 minutes*

D’autres membres peuvent également [vous inviter](/fr/portal/studio/search#empty-results) à rejoindre Wink. Vous recevrez un e-mail d’invitation de notre part avec une demande de `lead` que vous pouvez réclamer et utiliser pour finaliser la création de la propriété.

Dans ces cas, vous devez :

<Steps>
1. Cliquez sur l’onglet Réclamer depuis la [page des propriétés](https://extranet.wink.travel/property/list).
2. Cliquez sur la propriété que vous souhaitez réclamer.
3. Continuez le flux de travail [intelligent](#intelligent).
</Steps>

Vous pouvez aussi supprimer la réclamation et créer votre propriété de zéro, si vous le souhaitez.

### Intelligent

*Temps approximatif pour compléter : ~2 minutes*

:::note
Votre propriété doit être disponible sur Google Places pour continuer.
:::

L’intégration intelligente élimine certaines des étapes que vous auriez à faire manuellement. Vous obtenez :

- Géolocalisation
- Adresse
- Images
- Texte de bienvenue généré automatiquement.
- ...et quelques autres éléments gratuits

Cliquez sur le bouton `Commencer` pour l’intégration intelligente.

<Steps>
1. **Nom** Commencez à taper le nom de votre propriété tel qu’il apparaît sur Google. *Ex. Santiva Resort*.
2. **Adresse** Parfois, nous ne pouvons pas correctement localiser une adresse Google et vous demanderons votre aide.
3. Cliquez sur le bouton `Enregistrer`.
</Steps>

Vous êtes redirigé vers votre page des propriétés.

Félicitations 🎉 ...vous avez créé votre première propriété sur Wink de manière intelligente.

## Tableau de bord

Une fois votre propriété créée, vous pouvez finaliser la création de votre profil ou commencer à gérer votre propriété en cliquant sur le lien `Gérer` dans votre liste de propriétés.
Cela vous amène au tableau de bord de la propriété.

Le tableau de bord est la base de votre propriété. Il se compose de sections pour vous aider à naviguer et à tirer le meilleur parti de nos outils :

### Comment ça va ?

Cette section vous donne un aperçu rapide de la santé de votre propriété. Elle vous indique :

- Nombre total de réservations
- Valeur moyenne des réservations
- Montant total des ventes

Ces chiffres incluent toutes les réservations effectuées via n’importe quel canal de vente du réseau Wink.

### Vendez votre inventaire

Cette section vous informe sur les types d’outils disponibles pour vendre votre inventaire via Wink.
Cliquer sur l’une des cases vous mènera à [Wink Studio](/fr/portal/studio/what-is-studio).
Que vous soyez un hôtel ou un affilié, toutes les ventes s’y font et les hôtels peuvent être leurs propres affiliés ; vendant via leur propre canal de vente natif.

### Demandes du réseau Wink

Vous pouvez trouver ici les demandes en attente des affiliés qui souhaitent se connecter avec vous.
Il y a aussi un lien qui vous emmène directement à `Distribution > Explorer le réseau` pour gérer tous vos canaux de vente actuels et en attente.

### Tâches

La section des tâches vous montre des actions optionnelles, mais recommandées, que vous pouvez faire pour rendre votre profil de propriété plus attractif.

*Nous recommandons d’ajouter vos scores de réputation en ligne provenant de tout site externe où vous avez une note, comme les évaluations à jour des grandes OTA.*

### Analyses

Une nouvelle propriété reçoit automatiquement ses analyses de base au moment de la création. Par défaut, vous verrez :

- Revenu net des 12 derniers mois.
- Valeur moyenne des réservations des 12 derniers mois.
- Nombre total de vues de page des 12 derniers mois.
- Nombre total de réservations des 12 derniers mois.
- Nombre total d’annulations des 12 derniers mois.

Ces graphiques peuvent être facilement modifiés. Cependant, ils contiennent toutes les analyses dont votre propriété aura besoin pendant un certain temps.

## Annonces

Les annonces sont un moyen d’informer vos clients d’informations importantes de manière visible.
Si une annonce est `déclenchée`, elle s’affiche sous forme de nouvelle fenêtre au-dessus de votre page d’accueil.

Pour voir vos annonces, cliquez sur `Propriété > Annonce` dans la barre de navigation principale.

Cliquez sur le bouton `Créer une annonce` pour créer une nouvelle entrée.

<Steps>
1. Choisissez si vous souhaitez inclure un titre ou non.
2. Les annonces peuvent être déclenchées de deux manières :
    - Toujours afficher, à chaque visite de votre page.
    - Afficher uniquement lorsque l’itinéraire correspond à l’itinéraire de l’annonce.
3. Si vous avez choisi d’afficher l’annonce uniquement pour une certaine période, ajoutez les dates dans les champs du formulaire ci-dessous.
4. Cliquez sur le bouton `Enregistrer` et vous serez ramené à vos annonces.
</Steps>

## Profil

Le profil de la propriété est une étape obligatoire dans notre assistant d’intégration et nous permet de mieux connaître votre propriété selon des critères tels que :

- Nombre d’étoiles.
- Type d’hôtel.
- Type d’emplacement.
- ...et plus encore

Pour gérer le profil de votre propriété, cliquez sur `Propriété > Profil` dans l’onglet de navigation principal.

Mettez à jour le formulaire et cliquez sur le bouton `Enregistrer` pour continuer.

## Adresse

Vous serez invité à fournir l’adresse de votre propriété lors de la configuration.

Pour gérer votre adresse, cliquez sur `Propriété > Adresse` dans l’onglet de navigation principal.

:::note
Dans certains pays, comme la Thaïlande, il est recommandé de remplir également l’adresse locale en thaï, car cela peut être très utile pour les clients qui doivent la donner au chauffeur de taxi.
:::

Mettez à jour le formulaire et cliquez sur le bouton `Enregistrer` pour continuer.

:::note
Notre liste de villes provient directement de [geonames.org](https://www.geonames.org). Si votre ville ou île n’apparaît pas dans la liste déroulante des villes, veuillez nous en informer à support@wink.travel.
:::

## Bureau de réservation

Vous serez invité à fournir les coordonnées de votre bureau de réservation lors de la configuration.

Pour gérer votre bureau de réservation, cliquez sur `Propriété > Bureau de réservation` dans l’onglet de navigation principal.

Mettez à jour le formulaire et cliquez sur le bouton `Enregistrer` pour continuer.

:::note
Il n’est pas obligatoire d’utiliser le nom spécifique de la personne principale à la réception. Cela ajoute simplement une touche de familiarité.
Les clients entrants aiment pouvoir se connecter avec une vraie personne à leur arrivée et pouvoir demander quelque chose comme « Bonjour... Lisa est-elle là ? »
:::

## Texte de bienvenue

Vous serez invité à rédiger une description longue et courte de votre propriété lors de la configuration.

Pour gérer votre texte de bienvenue, cliquez sur `Propriété > Texte de bienvenue` dans l’onglet de navigation principal.

Mettez à jour le formulaire et cliquez sur le bouton `Enregistrer` pour continuer.

:::note
Au minimum, nous avons besoin que vous complétiez votre texte de bienvenue en anglais.

Créez un texte de bienvenue pour toutes les autres langues que vous souhaitez gérer.
Nous utiliserons votre texte anglais pour traduire dans toutes les autres langues populaires.
:::

## Directeur général

Ajouter les informations du DG aide à créer plus de familiarité avec le lieu.

Pour mettre à jour les informations du Directeur Général, cliquez sur `Propriété > Directeur Général` dans l’onglet de navigation principal.

Vous pouvez ajouter le nom et la photo de votre DG actuel ainsi qu’un message personnel aux clients.

Mettez à jour le formulaire et cliquez sur le bouton `Enregistrer` pour continuer.

## Politiques

Vous serez invité à compléter les politiques de votre propriété lors de la configuration.

Pour gérer les politiques de votre propriété, cliquez sur `Propriété > Politiques` dans l’onglet de navigation principal.

Informez vos clients sur :

- Les heures d’arrivée/départ.
- Si vous autorisez les animaux et les enfants sur place.
- Si vous offrez internet et parking sur place.

Mettez à jour le formulaire et cliquez sur le bouton `Enregistrer` pour continuer.

## Localisation sur la carte

Vous serez invité à définir la localisation sur la carte lors de la configuration.

Pour gérer la localisation sur la carte de votre propriété, cliquez sur `Propriété > Localisation sur la carte` dans l’onglet de navigation principal.

Vous pouvez définir la localisation de votre propriété de deux manières :

1. Recherchez votre propriété par nom dans le champ de recherche au-dessus de la carte et sélectionnez-la.
2. Utilisez la carte, zoomez et déplacez-vous jusqu’à l’emplacement de votre propriété et cliquez sur la carte lorsque vous la trouvez.

Un marqueur apparaîtra avec vos coordonnées mises à jour.

:::note
Votre propriété doit être enregistrée sur Google Pages pour apparaître dans le champ de recherche.
:::

Mettez à jour le formulaire et cliquez sur le bouton `Enregistrer` pour continuer.

## Équipements

Les équipements de la propriété sont les services et installations disponibles sur votre propriété.

Pour gérer les équipements de votre propriété, cliquez sur `Propriété > Équipements` dans l’onglet de navigation principal.

Cochez toutes les cases qui s’appliquent à votre propriété.

Cliquez sur le bouton `Enregistrer` pour continuer.

## Photos & Vidéos

:::note
Cette section est uniquement pour les images et vidéos de vos locaux, comme le hall, l’extérieur, etc. Tous les autres inventaires ont leur propre section média.
:::

Vous serez invité à ajouter des photos de votre propriété lors de la configuration.

Pour gérer les médias de votre propriété, cliquez sur `Propriété > Photos & Vidéos` dans l’onglet de navigation principal.

Pour ajouter une nouvelle photo ou vidéo :

### Métadonnées

Vous pouvez ajouter des métadonnées supplémentaires à vos images et vidéos en cliquant sur l’icône ✏️ à côté de votre image.

Les métadonnées peuvent être :

- **Style de vie** Définissez-le si une image représente un style de vie spécifique.
- **Catégorie** Définissez-le si elle correspond à une catégorie spécifique. *ex. Vue piscine*
- **Légendes** Décrivez l’image dans la langue de votre choix.

:::note[Pourquoi utiliser le style de vie ?]
Si les voyageurs filtrent selon un style de vie spécifique et que votre propriété apparaît, nous affichons l’image qui correspond au style de vie choisi par le voyageur comme image principale ; rendant ainsi votre propriété plus pertinente.
:::

Cliquez sur le bouton `Enregistrer` pour continuer.

## Styles de vie

Vous serez invité à associer certains styles de vie à votre propriété lors de la configuration.

Pour gérer vos styles de vie, cliquez sur `Propriété > Styles de vie` dans l’onglet de navigation principal.

Activez l’interrupteur à côté des styles de vie que vous souhaitez utiliser.

Cliquez sur le bouton `Enregistrer` pour continuer.

:::caution[Exigences]
Vous pouvez choisir un maximum de 6 styles de vie. Choisissez ceux qui correspondent le mieux à votre propriété.
:::

## Réputation

Votre réputation est votre monnaie digitale et vous voulez pouvoir l’emporter partout avec vous.
Une entrée de réputation peut être tout ce dont votre propriété veut se vanter, comme une note tierce ou pour le concours où votre chef a remporté une médaille d’or.
Dans cette section, vous pouvez ajouter vos scores et classements provenant de partout sur Internet.

Pour gérer votre réputation, cliquez sur `Propriété > Réputation` dans l’onglet de navigation principal.

Cliquez sur le bouton `Ajouter un score de réputation`.

<Steps>
1. **Catégorie** Choisissez le type de score que vous souhaitez ajouter. *ex. Avis tiers*
2. **Fournisseur** Tapez qui vous a attribué le score. *ex. BigOTA.com*
3. **Type de notation** Sélectionnez la forme sous laquelle vous avez reçu le score. *ex. Numérique*
4. **Date** Facultatif, saisissez la date à laquelle vous avez reçu le score.
5. **Note** Entrez la note que vous avez reçue. *ex. 8.5*
6. **Note maximale** Facultatif, saisissez la note maximale possible pour ce type de récompense. *ex. 10*
</Steps>

Cliquez sur le bouton `Enregistrer` pour continuer.

## Réseaux sociaux

Les réseaux sociaux vous permettent de rester en contact avec les voyageurs nouveaux et existants au fil du temps.
Ajoutez vos comptes de réseaux sociaux à votre profil pour que les clients puissent facilement vous contacter et en savoir plus sur votre propriété.

Pour gérer vos comptes de réseaux sociaux, cliquez sur `Propriété > Réseaux sociaux` dans l’onglet de navigation principal.

<Steps>
1. Activez l’interrupteur du réseau social que vous souhaitez activer.
2. Entrez le nom de votre compte sur ce réseau.
</Steps>

Cliquez sur le bouton `Enregistrer` pour continuer.

## Indice vert

Votre indice vert est mesuré par l’impact de votre propriété sur l’environnement local et dans vos décisions d’achat.

Pour gérer votre score d’Indice Vert, cliquez sur `Propriété > Indice Vert` dans l’onglet de navigation principal.

Parcourez chaque onglet et répondez au questionnaire du mieux possible.

Cliquez sur le bouton `Enregistrer` pour continuer.

## Branding

Vous pouvez personnaliser l’expérience du voyageur lorsqu’il est sur votre page d’accueil Wink et lorsqu’il reçoit des e-mails de confirmation de réservation.

Pour gérer votre Branding, cliquez sur `Compte > Branding` dans l’onglet de navigation principal.

Pour en savoir plus sur les options disponibles, rendez-vous sur [Personnalisation](/fr/portal/settings/branding).

## Gestionnaires

Vous pouvez inviter d’autres utilisateurs à vous aider à gérer votre compte affilié. Une fois qu’un utilisateur accepte votre invitation, il aura un accès complet à votre compte à l’exception des paiements. Vous resterez toujours le seul responsable des fonds que vous possédez.

<Steps>
1. Cliquez sur `Compte > Gestionnaires` dans la barre de navigation principale.
2. Ajoutez l’e-mail de l’utilisateur que vous souhaitez nommer gestionnaire de compte.
3. Cliquez sur le lien `Inviter` à côté de l’e-mail.
</Steps>

## Notifications

Parfois, nous voulons vous contacter. Lorsque cela arrive, vous verrez la cloche en haut à droite changer.
Pour lire ce que nous vous avons envoyé :

<Steps>
1. Cliquez sur `Compte > Notifications` dans la barre de navigation principale.
2. Vous verrez les notifications nouvelles et passées.
3. Choisissez de `Marquer comme lu` ou de `Supprimer` les notifications une fois que vous les avez lues.
</Steps>

## API

Les développeurs qui souhaitent gérer les `informations de propriété` peuvent se rendre sur [Développeurs > API > Propriété](/fr/api/#property-api).
