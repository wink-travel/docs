[Wink Studio](https://studio.wink.travel) vous connecte à l'inventaire de voyages dans le monde entier. Cet article vous guide à travers la création et la gestion de vos comptes affiliés sur Wink.

## Créer un compte

Une fois authentifié, vous serez redirigé vers votre [page des comptes](https://studio.wink.travel/company/list) qui affiche tous vos comptes affiliés.

Cliquez sur le bouton `Create new account` pour continuer. Vous serez redirigé vers notre page de [création de compte](https://studio.wink.travel/company/create).

Remplissez tous les champs obligatoires.

<Steps>
1. **Nom** Le nom de votre compte. *Ce n’est pas nécessairement votre vrai nom mais ce sera ce que les utilisateurs verront lorsqu’ils consulteront votre page WinkLinks ou lorsque les fournisseurs voudront vous contacter directement.*
2. **Type d’entité** Indiquez si vous êtes une entreprise ou un particulier. *Principalement utilisé pour déterminer votre éligibilité à recevoir des fonds.*
3. **Type** Choisissez un type qui correspond le mieux à votre compte. *ex. Influenceur*
4. **Description** Rédigez une description pour votre compte. *Celle-ci sera visible sur votre page WinkLinks et lorsque les fournisseurs voudront vous contacter directement.*
5. **Site web** Entrez votre site web principal. *Cela peut être l’URL de votre compte IG ou le site web de votre entreprise. Ce doit être l’endroit où vous effectuez la majeure partie de votre travail en ligne. Les fournisseurs consulteront ces liens pour décider s’ils veulent se connecter avec vous.*
6. **Ville** Où êtes-vous basé ?
7. **Dépenses annuelles de voyage** Combien vous ou votre entreprise dépensez en voyages chaque année.
7. **Accord** Acceptez notre [contrat d’affiliation](/fr/portal/studio/terms-of-service) et nos [conditions de paiement](/fr/portal/studio/payment-terms) pour continuer.
8. Cliquez sur le bouton `Save`.
</Steps>

Félicitations 🎉 ...vous avez créé votre premier compte affilié sur Wink.

## Présence en ligne

Il est recommandé à tous les affiliés de renseigner leur présence en ligne. La présence en ligne permet aux hôtels de voir où vous êtes trouvés par les utilisateurs et votre niveau de portée.

<Steps>
1. **Type** Entrez le type de présence en ligne. *ex. Facebook*
2. **Compte** Entrez l’URL de votre compte. *ex. https://facebook.com/wink2travel*
3. **Taille de l’audience** Indiquez la taille de votre audience. *ex. Cela peut être vos abonnés ou le nombre d’employés*
4. **Type d’audience** Indiquez le type d’audience que vous avez. *ex. Famille car votre blog de voyage est axé sur les voyages en famille*
</Steps>

Une fois créé, vous pouvez personnaliser davantage votre compte en renseignant votre adresse et en ajoutant une photo de profil à votre compte.

<Steps>
1. Cliquez sur votre nouveau compte affilié. *Cela vous mènera à votre tableau de bord de compte.*
2. Cliquez sur `Account > Profile` dans la barre de navigation principale.
3. Sur la page Profil, vous pourrez mettre à jour votre profil existant et y ajouter des informations supplémentaires.
</Steps>

:::note
La photo de profil de votre compte affilié ne doit pas être confondue avec la photo de profil de votre compte utilisateur Wink. La photo de profil de votre compte affilié sera affichée sur votre page WinkLinks et visible par les fournisseurs souhaitant se connecter avec vous.

Votre photo de profil utilisateur est principalement destinée à un usage interne.
:::

## Gestionnaires

Vous pouvez inviter d’autres utilisateurs à vous aider à gérer votre compte affilié. Une fois qu’un utilisateur accepte votre invitation, il aura un accès complet à votre compte à l’exception des paiements. Vous resterez toujours le seul responsable des fonds que vous possédez.

<Steps>
1. Sélectionnez le compte affilié avec lequel vous souhaitez travailler depuis la [page des comptes](https://studio.wink.travel/company/list).
2. Cliquez sur `Account > Managers` dans la barre de navigation principale.
3. Ajoutez l’e-mail de l’utilisateur que vous souhaitez nommer gestionnaire de compte.
4. Cliquez sur le lien `Invite` à côté de l’e-mail.
</Steps>

Wink enverra un e-mail à l’utilisateur pour qu’il accepte ou refuse votre invitation.

## Notifications

Parfois, nous souhaitons vous contacter. Lorsque cela se produit, vous verrez la cloche en haut à droite changer.
Pour lire ce que nous vous avons envoyé :

<Steps>
1. Cliquez sur `Account > Notifications` dans la barre de navigation principale.
2. Vous verrez les notifications nouvelles et passées.
3. Choisissez de `Marquer comme lu` ou `Supprimer` les notifications une fois que vous les avez lues.
</Steps>
