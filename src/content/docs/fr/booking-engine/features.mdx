Le moteur de réservation se décline en deux versions pour répondre aux besoins différents d’un affilié, d’une société de gestion immobilière et d’un hôtel.

## Agence de voyage en ligne

Pour un affilié, il est plus logique d’utiliser la version agence de voyage en ligne. La raison en est que le site est conçu pour accueillir des voyageurs souhaitant consulter plusieurs propriétés.

Vous pouvez trouver la page d’accueil de cette application ci-dessous.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Moteur de réservation internet

Pour un hôtel ou une société de gestion immobilière, il est plus logique d’utiliser la version moteur de réservation internet. Ce type d’utilisateur souhaite mettre en avant une seule propriété ou plusieurs propriétés sous gestion.

Vous pouvez trouver la page d’accueil de cette application ci-dessous.

## Configuration

La plupart du temps, il n’est pas nécessaire de partager directement l’une ou l’autre de ces applications. Nous avons créé plusieurs moyens pour vous permettre de partager l’inventaire de voyages, que vous pouvez découvrir dans [Wink Studio](/fr/studio/what-is-studio) ou via [WinkLinks](/fr/link-manager/wink-links).

Cependant, si vous souhaitez créer un lien manuel vers l’un de ces sites, assurez-vous d’ajouter votre ID client et votre ID de personnalisation dans l’URL en tant que paramètres.

Vous pouvez trouver votre ID client en :

<Steps>
1. Vous connecter à [Wink Extranet](/fr/extranet/what-is-extranet) ou [Wink Studio](/fr/studio/what-is-studio).
2. Cliquer sur votre icône de profil en haut à droite et cliquer sur [Applications](/fr/developers/applications).
3. Utiliser l’application liée au compte Wink avec lequel vous souhaitez travailler.
4. Copier l’ID client de cette application et le coller dans l’URL en tant que paramètre de requête. Exemple : ?client-id=MON_ID_CLIENT
</Steps>

### Extranet

Vous pouvez trouver votre ID de personnalisation en tant que propriété en :

<Steps>
1. Vous connecter à [Wink Extranet](/fr/extranet/what-is-extranet).
2. Sélectionner la propriété avec laquelle vous souhaitez travailler.
3. Dans la barre de navigation principale, cliquer sur `Compte > Branding`
4. Copier l’ID de personnalisation et le coller dans l’URL en tant que paramètre de requête. Exemple : ?configuration-id=MON_ID_PERSONNALISATION
</Steps>

### Studio

Vous pouvez trouver votre ID de personnalisation en tant qu’affilié en :

<Steps>
1. Vous connecter à [Wink Studio](/fr/studio/what-is-studio).
2. Sélectionner le compte avec lequel vous souhaitez travailler.
3. Dans la barre de navigation principale, cliquer sur `Outils > Personnalisation`
4. Copier l’ID de personnalisation de la personnalisation que vous souhaitez utiliser et le coller dans l’URL en tant que paramètre de requête. Exemple : ?configuration-id=MON_ID_PERSONNALISATION
</Steps>

## OTA

Vous obtenez une URL qui ressemble à ceci :

https://ota.wink.travel?client-id=MON_ID_CLIENT&configuration-id=MON_ID_PERSONNALISATION

Si vous souhaitez accéder directement à une page d’hôtel, l’URL ressemblera à ceci :

https://ota.wink.travel/hotel/hotel-x?client-id=MON_ID_CLIENT&configuration-id=MON_ID_PERSONNALISATION

:::note
Si vous n’incluez pas un ID client et/ou un ID de configuration, Wink OTA utilisera une configuration par défaut.
:::

## IBE

Si vous êtes une société de gestion immobilière, vous souhaitez créer un lien vers la page d’accueil de Wink IBE. La page d’accueil affichera votre thème désigné ainsi que les [propriétés que vous représentez](/fr/studio/agency-representation).

Vous obtenez une URL qui ressemble à ceci :

https://book.wink.travel?client-id=MON_ID_CLIENT&configuration-id=MON_ID_PERSONNALISATION

Si vous êtes une propriété, vous souhaitez créer un lien direct vers la page d’atterrissage de la propriété sur Wink IBE.

Cette URL ressemblera à ceci :

https://book.wink.travel/hotel/hotel-x?client-id=MON_ID_CLIENT&configuration-id=MON_ID_PERSONNALISATION


:::note
Si vous n’incluez pas un ID client et/ou un ID de configuration, Wink IBE NE se chargera PAS. Ces paramètres sont obligatoires.
:::

## Exemple : page de propriété

Chaque propriété sur Wink dispose de sa propre page d’atterrissage unique sur l’un ou l’autre des sites. Tous les liens externes dirigent les voyageurs vers l’une de ces pages d’atterrissage.

La page d’atterrissage contient toutes les informations dont un voyageur a besoin pour faire un choix éclairé quant à son séjour dans cette propriété.
