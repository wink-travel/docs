Le moteur de réservation est disponible en deux versions pour répondre aux différents besoins d'un affilié, d'une société de gestion immobilière et d'un hôtel.

## Agence de voyages en ligne

Pour un affilié, il est plus judicieux d'utiliser la version agence de voyages en ligne. En effet, le site est conçu pour accueillir les voyageurs souhaitant consulter plusieurs établissements.

Vous trouverez ci-dessous la page d'accueil de cette application.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Clin d'œil OTA
</LinkButton>


## moteur de réservation Internet

Pour un hôtel ou une société de gestion immobilière, il est plus judicieux d'utiliser la version du moteur de réservation en ligne. Ce type d'utilisateur souhaite mettre en avant un ou plusieurs établissements sous sa gestion.

## Configuration

La plupart du temps, il n'est pas nécessaire de partager directement ces applications. Nous avons créé différentes façons de partager votre inventaire de voyages, que vous pouvez découvrir dans[Wink Studio](/studio/what-is-studio)ou par[Liens Wink](/link-manager/wink-links).

Cependant, si vous souhaitez créer un lien manuel vers l'un de ces sites Web, assurez-vous d'ajouter votre ID client et votre ID de personnalisation à l'URL en tant que paramètres.

Vous pouvez trouver votre identifiant client en :

<Steps>
  1. Connectez-vous à l'un ou l'autre [Extranet Wink](/extranet/what-is-extranet) ou [Wink Studio](/studio/what-is-studio).
  2. Cliquez sur l'icône de votre profil dans le coin supérieur droit et cliquez sur [Applications](/developers/applications).
  3. Utilisez l’application liée au compte Wink avec lequel vous souhaitez travailler.
  4. Copiez l'identifiant client de cette application et collez-le dans l'URL comme paramètre de requête. Exemple : ?client-id=MY\_CLIENT\_ID
</Steps>


### Extranet

Vous pouvez trouver votre ID de personnalisation en tant que propriété en :

<Steps>
  1. Connectez-vous à [Extranet Wink](/extranet/what-is-extranet).
  2. Sélectionnez la propriété avec laquelle vous souhaitez travailler.
  3. Dans la barre de navigation principale, cliquez sur `Account > Branding`
  4. Copiez l'ID de personnalisation et collez-le dans l'URL comme paramètre de requête. Exemple : ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Studio

Vous pouvez trouver votre identifiant de personnalisation en tant qu'affilié en :

<Steps>
  1. Connectez-vous à [Wink Studio](/studio/what-is-studio).
  2. Sélectionnez le compte avec lequel vous souhaitez travailler.
  3. Dans la barre de navigation principale, cliquez sur `Tools > Customization`
  4. Copiez l'ID de personnalisation de la personnalisation souhaitée et collez-le dans l'URL comme paramètre de requête. Exemple : ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

Vous obtenez alors une URL qui ressemble à ceci :

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Si vous souhaitez accéder directement à la page d'un hôtel, l'URL ressemblerait à ceci :

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::note
Si vous n'incluez pas d'identifiant client et/ou de configuration, Wink OTA utilisera une configuration par défaut.
:::

## BIE

Si vous êtes une société de gestion immobilière, vous souhaitez créer un lien vers la page d'accueil de Wink IBE. Celle-ci affichera votre thème et les informations de votre choix.[propriétés que vous représentez](/studio/agency-representation).

Vous obtenez alors une URL qui ressemble à ceci :

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Si vous êtes une propriété, vous souhaitez créer un lien direct vers la page de destination de la propriété Wink IBE.

Cette URL ressemblerait à ceci :

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::note
Si vous n'incluez pas d'ID client et/ou de configuration, Wink IBE ne se chargera pas. Ces paramètres sont obligatoires.
:::
