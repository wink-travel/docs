Le moteur de réservation existe en deux versions pour répondre aux besoins différents d’un affilié, d’une société de gestion immobilière et d’un hôtel.

## Agence de voyage en ligne

Pour un affilié, il est plus logique d’utiliser la version agence de voyage en ligne. La raison est que le site est conçu pour accueillir des voyageurs souhaitant consulter plusieurs établissements.

Vous pouvez trouver la page d’accueil de cette application ci-dessous.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Moteur de réservation internet

Pour un hôtel ou une société de gestion immobilière, il est plus logique d’utiliser la version moteur de réservation internet. Ce type d’utilisateur souhaite mettre en avant un seul établissement ou plusieurs établissements sous gestion.

Vous pouvez trouver la page d’accueil de cette application ci-dessous.

  Wink IBE
</LinkButton>

## Configuration

La plupart du temps, il n’est pas nécessaire de partager directement l’une ou l’autre de ces applications. Nous avons créé plusieurs moyens pour vous permettre de partager l’inventaire de voyages, que vous pouvez découvrir dans [Wink Studio](/studio/what-is-studio) ou via [WinkLinks](/link-manager/wink-links).

Cependant, si vous souhaitez créer un lien manuel vers l’un de ces sites, assurez-vous d’ajouter votre ID client et votre ID de personnalisation dans l’URL en tant que paramètres.

Vous pouvez trouver votre ID client en :

<Steps>
1. Connectez-vous soit à [Wink Extranet](/extranet/what-is-extranet) soit à [Wink Studio](/studio/what-is-studio).
2. Cliquez sur votre icône de profil en haut à droite et cliquez sur [Applications](/developers/applications).
3. Utilisez l’application liée au compte Wink avec lequel vous souhaitez travailler.
4. Copiez l’ID client de cette application et collez-le dans l’URL en tant que paramètre de requête. Exemple : ?client-id=MON_ID_CLIENT
</Steps>

### Extranet

Vous pouvez trouver votre ID de personnalisation en tant qu’établissement en :

<Steps>
1. Connectez-vous à [Wink Extranet](/extranet/what-is-extranet).
2. Sélectionnez l’établissement avec lequel vous souhaitez travailler.
3. Dans la barre de navigation principale, cliquez sur `Compte > Branding`
4. Copiez l’ID de personnalisation et collez-le dans l’URL en tant que paramètre de requête. Exemple : ?configuration-id=MON_ID_PERSONNALISATION
</Steps>

### Studio

Vous pouvez trouver votre ID de personnalisation en tant qu’affilié en :

<Steps>
1. Connectez-vous à [Wink Studio](/studio/what-is-studio).
2. Sélectionnez le compte avec lequel vous souhaitez travailler.
3. Dans la barre de navigation principale, cliquez sur `Outils > Personnalisation`
4. Copiez l’ID de personnalisation de la personnalisation que vous souhaitez utiliser et collez-le dans l’URL en tant que paramètre de requête. Exemple : ?configuration-id=MON_ID_PERSONNALISATION
</Steps>

## OTA

Vous obtenez une URL qui ressemble à ceci :

https://ota.wink.travel?client-id=MON_ID_CLIENT&configuration-id=MON_ID_PERSONNALISATION

Si vous souhaitez accéder directement à la page d’un hôtel, l’URL ressemblera à ceci :

https://ota.wink.travel/hotel/hotel-x?client-id=MON_ID_CLIENT&configuration-id=MON_ID_PERSONNALISATION

:::note
Si vous n’incluez pas un ID client et/ou un ID de personnalisation, Wink OTA utilisera une configuration par défaut.
:::

## IBE

Si vous êtes une société de gestion immobilière, vous souhaitez créer un lien vers la page d’accueil de Wink IBE. La page d’accueil affichera votre thème désigné ainsi que les [établissements que vous représentez](/studio/agency-representation).

Vous obtenez une URL qui ressemble à ceci :

https://book.wink.travel?client-id=MON_ID_CLIENT&configuration-id=MON_ID_PERSONNALISATION

Si vous êtes un établissement, vous souhaitez créer un lien direct vers la page d’accueil de l’établissement sur Wink IBE.

Cette URL ressemblera à ceci :

https://book.wink.travel/hotel/hotel-x?client-id=MON_ID_CLIENT&configuration-id=MON_ID_PERSONNALISATION


:::note
Si vous n’incluez pas un ID client et/ou un ID de personnalisation, Wink IBE NE se chargera PAS. Ces paramètres sont obligatoires.
:::

## Exemple : page d’établissement

Chaque établissement sur Wink dispose de sa propre page d’accueil unique dans l’un ou l’autre des sites. Tous les liens externes dirigent les voyageurs vers l’une de ces pages d’accueil.

La page d’accueil contient toutes les informations dont un voyageur a besoin pour faire un choix éclairé quant à son séjour dans cet établissement.
