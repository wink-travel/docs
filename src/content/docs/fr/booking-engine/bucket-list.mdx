Partout où vous voyez des [cartes](/fr/studio/cards) d'inventaire Wink, vous verrez un cœur dans le coin supérieur droit de la carte qui vous permet d'ajouter cet inventaire à votre liste de souhaits.

Suivez ces étapes pour gérer votre liste de souhaits :

<Steps>
1. Cliquez sur l’icône ♡ `cœur` pour l’inventaire que vous souhaitez ajouter.
2. Si vous êtes connecté, l’icône cœur deviendra rouge.
3. Si vous n’êtes pas connecté, vous serez redirigé pour le faire.
</Steps>

:::tip
Cliquez une seconde fois sur l’icône ♥️ `cœur` pour le retirer de votre liste.
:::

Vous pouvez gérer tous les éléments de votre liste de souhaits en cliquant sur votre icône de profil dans le coin supérieur droit, puis en cliquant sur le bouton `Liste de souhaits`.
