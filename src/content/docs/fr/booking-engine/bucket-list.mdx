Partout où vous voyez l'inventaire Wink[cartes](/studio/cards), vous verrez un cœur dans le coin supérieur droit de la carte qui vous permet d'ajouter cet inventaire à votre liste de choses à faire.

Suivez ces étapes pour gérer votre liste de choses à faire :

<Steps>
  1. Cliquez sur le ♡ `heart icon` pour l'inventaire que vous souhaitez ajouter.
  2. Si vous êtes connecté, l'icône en forme de cœur deviendra rouge.
  3. Si vous n'êtes pas connecté, vous serez redirigé pour le faire.
</Steps>


:::astuce
Cliquez sur le ♥️`heart icon`une deuxième fois pour le supprimer de votre liste.
:::

Vous pouvez gérer tous les éléments de votre liste de choses à faire en cliquant sur l'icône de votre profil dans le coin supérieur droit, puis en cliquant sur le bouton`Bucket list` bouton.
