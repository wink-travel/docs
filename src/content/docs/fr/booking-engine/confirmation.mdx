Après un paiement réussi, le voyageur arrive sur la page de confirmation de réservation.

## Page de remerciement

La page de confirmation affiche :

- Un message de remerciement confirmant la réservation.
- Le **code de réservation** pour référence.
- Une note indiquant qu’un e-mail de confirmation a été envoyé.

Si la réservation est encore en cours de traitement, un bref message « Confirmation de votre réservation... » est affiché jusqu’à ce que le serveur confirme.

## Détails de la réservation

Une fois confirmée, le voyageur peut consulter les détails complets de la réservation :

- Nom et emplacement de l’hôtel.
- Type et configuration de la chambre.
- Dates d’arrivée et de départ.
- Informations sur les invités.
- Détail des prix.
- Politique d’annulation.
- Détails sur le paiement et le mode de paiement.

## Actions des invités

Depuis la page des détails de la réservation, les voyageurs authentifiés peuvent :

- **Annuler la réservation** — Si la politique d’annulation le permet.
- **Laisser un avis** — Noter et commenter l’établissement.
- **Envoyer un message** — Contacter directement l’hôtel.
- **Partager** — Partager la réservation sur les réseaux sociaux.
- **Imprimer** — Imprimer la confirmation de réservation.

Les invités non authentifiés voient une vue en lecture seule des détails de leur réservation.
