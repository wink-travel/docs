L’onglet **Sécurité** sur [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) est l’endroit où vous gérez les **passkeys** — des identifiants WebAuthn qui vous permettent de vous connecter sans mot de passe.

## Qu’est-ce qu’un passkey ?

Un passkey est un identifiant cryptographique lié à votre appareil (ou gestionnaire de mots de passe) qui prouve que vous êtes bien vous. Les passkeys sont résistants au phishing et plus rapides que la saisie d’un mot de passe.

## Enregistrer un passkey

1. Connectez-vous à [iam.wink.travel](https://iam.wink.travel) avec vos identifiants existants.
2. Allez dans **Mon compte → Sécurité**.
3. Cliquez sur `Ajouter un passkey`.
4. Votre navigateur vous invitera à vous authentifier avec votre appareil (Touch ID, Face ID, Windows Hello, une clé matérielle ou un gestionnaire de mots de passe synchronisé).
5. Donnez un nom convivial au passkey (par exemple *Work MacBook*) pour pouvoir le reconnaître plus tard.

## Utiliser un passkey pour se connecter

Sur l’écran de connexion de [iam.wink.travel](https://iam.wink.travel), saisissez votre e-mail — si un passkey est enregistré pour cet e-mail, vous serez invité à vous authentifier avec celui-ci au lieu de taper un mot de passe.

## Supprimer un passkey

Dans le même onglet Sécurité, chaque passkey enregistré dispose d’un bouton `Supprimer`. Utilisez-le si vous perdez un appareil, mettez hors service une clé matérielle ou souhaitez nettoyer les identifiants inutilisés.

:::caution
Supprimer tous vos passkeys laisse le mot de passe comme seule méthode de connexion. Assurez-vous de vous souvenir de votre mot de passe avant de supprimer le dernier passkey.
:::
