[Wink IAM](https://iam.wink.travel) est le service d'identité pour votre **compte utilisateur** Wink — distinct des **comptes organisationnels** (établissements, comptes affiliés, etc.) que vous gérez dans [Portal](/fr/portal/overview).

## Ce qui se trouve ici

- **`/my-account/general`** — Votre profil utilisateur : prénom, nom, nom d’utilisateur, e-mail, photo de profil, mot de passe.
- **`/my-account/security`** — Passkeys (WebAuthn) pour une connexion sans mot de passe.
- **`/my-account/apps`** — Applications OAuth2 que vous avez autorisées à accéder à votre compte Wink. Révoquez l’accès à tout moment.

## Paramètres utilisateur vs. Portal : quelle est la différence ?

| Concept | Se trouve sur | Exemples |
|---|---|---|
| **Identité utilisateur** (vous, la personne) | `iam.wink.travel` | E-mail, mot de passe, passkeys, photo de profil |
| **Comptes organisationnels** (établissements, organisations affiliées, etc. que vous gérez) | `app.wink.travel` | Image de marque, gestionnaires, paiements, candidatures, webhooks |

Un utilisateur peut gérer plusieurs comptes organisationnels ; un compte organisationnel peut être géré par plusieurs utilisateurs (via les [gestionnaires](/fr/portal/settings/managers)).

## Processus de connexion

La connexion commence toujours sur IAM. Une fois authentifié, vous êtes redirigé vers Portal, où vous choisissez le compte organisationnel dans lequel vous souhaitez travailler.
