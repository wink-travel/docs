Les applications vous donnent un accès complet aux fonctionnalités de Wink et TripPay et permettent à vos utilisateurs de s'authentifier et de se connecter à votre site via notre serveur d'autorisation OAuth2, comme Facebook et Google.

Vous pouvez créer votre propre OTA avec Wink. Tout commence par une application.

*Par défaut, vous recevez une application lors de la création d'un nouveau compte affilié Wink ou TripPay. Nous utilisons`client ID`pour certains scénarios.*

:::note
Une application générée sur Wink ou TripPay peut être utilisée pour tous les points de terminaison de l'API.
:::

*Pour cet exemple, nous allons nous authentifier via[Wink Studio](https://studio.wink.travel).*

<Steps>
  1. Une fois authentifié, cliquez sur l'icône de votre profil, dans le coin supérieur droit, et un menu apparaîtra.
  2. Cliquez sur le `Applications` lien.
  3. Vous serez redirigé vers la page de vos applications.
</Steps>


## Créer une application

Comme vous ne pouvez pas accéder au`secret key`de l'application que nous avons créée pour vous, vous devrez créer la vôtre.

:::Attention
Vous devez créer un compte affilié sur Wink ou un compte TripPay avant de créer une application, car vous lierez votre application à un compte spécifique.
:::

Voici les étapes pour créer une application :

<Steps>
  1. Depuis la page des applications, cliquez sur le `Create new application` bouton.
  2. ***Nom*** Donnez un nom à votre application. *par exemple, Cool App*
  3. **Nom de l'entité** Liez votre application à l’un de vos comptes existants. *par exemple, Cool Account*
  4. **Redirection des URI** Au moins un URI de redirection est requis. Il sert à valider le domaine de redirection après une authentification réussie. *par ex. https://www.cool-site.com*
  5. Cliquez sur le `Save` bouton pour continuer.
</Steps>


Vous serez redirigé vers votre liste d'applications. Votre nouvelle application y apparaîtra et vous n'aurez accès qu'une seule fois à sa clé secrète. Assurez-vous de la copier et de l'enregistrer en lieu sûr.

## Mettre à jour l'application

Vous devrez mettre à jour votre application si l'un de vos URI de redirection change ou si vous souhaitez ajouter ou supprimer un URI.

Pour mettre à jour une application existante, procédez comme suit :

<Steps>
  1. Cliquez sur le `Actions` lien pour l'application que vous souhaitez mettre à jour.
  2. Cliquez sur le `Update` bouton sous Actions.
  3. Apportez vos modifications à l'application.
  4. Cliquez sur le `Save` bouton pour continuer.
</Steps>


Vous êtes redirigé vers votre liste d'applications.

## Supprimer l'application

:::Attention
Veuillez ne pas supprimer l'application que nous avons créée automatiquement pour vous afin de garantir le bon fonctionnement de toutes les fonctionnalités de votre compte.
:::

Si vous n’avez plus besoin d’une application, vous pouvez la supprimer.

Pour supprimer une application, procédez comme suit :

<Steps>
  1. Cliquez sur le `Actions` lien pour l'application que vous souhaitez mettre à jour.
  2. Cliquez sur le `Remove` bouton sous Actions.
  3. Cliquez `OK` bouton pour confirmer la suppression.
</Steps>


Vous êtes redirigé vers votre liste d'applications.
