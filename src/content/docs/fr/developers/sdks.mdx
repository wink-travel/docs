Accélérez le time-to-market en utilisant l’un de nos kits de développement logiciel officiellement pris en charge.
Si nous ne supportons pas encore votre langage préféré, rendez-vous sur l’API que vous souhaitez utiliser et téléchargez la spécification OpenAPI que nous mettons à disposition pour chaque API en haut de la page. Vous pouvez générer votre propre SDK en forkant l’un de nos dépôts publics de SDK ou en utilisant l’un des générateurs OpenAPI.

## Java

Cette bibliothèque a été compilée pour Java 24. Elle est configurée pour utiliser le WebClient réactif et est optimisée pour une utilisation avec le Spring Framework. Cependant, Spring n’est pas une exigence mais il vous offre un support OAuth2 prêt à l’emploi.
