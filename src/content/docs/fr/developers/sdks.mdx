Accélérez la mise sur le marché en utilisant l'un de nos kits de développement logiciel (SDK) officiellement pris en charge.
Si votre langage préféré n'est pas actuellement pris en charge, accédez à l'API que vous souhaitez utiliser et téléchargez la spécification OpenAPI disponible pour chaque API en haut de la page. Vous pouvez générer votre propre SDK en créant un fork de l'un de nos dépôts SDK publics ou en utilisant l'un des générateurs OpenAPI.

## Java

Cette bibliothèque a été compilée pour Java 24. Elle est configurée pour utiliser le client Web réactif et optimisée pour Spring Framework. Spring n'est pas obligatoire, mais offre une prise en charge OAuth2 prête à l'emploi.
