Cette page liste tous les événements webhook auxquels vous pouvez vous abonner sur Wink. Pour les instructions de configuration, consultez le [Guide d'intégration Webhook](/fr/guides/webhook-integration) ou la [page de gestion des Webhooks](/fr/my-account/webhooks).

## Comment s'abonner

1. Accédez à **Applications > Webhooks** dans votre portail.
2. Créez un webhook et sélectionnez les événements que vous souhaitez recevoir.
3. Wink envoie un HTTP POST à votre point de terminaison chaque fois que l'événement se produit.

Pour les étapes complètes de configuration et les bonnes pratiques, consultez [Intégration Webhook](/fr/guides/webhook-integration).

## Événements de compte

Événements liés au cycle de vie et aux paramètres de votre compte TripPay.

| Événement | Clé | Description |
|---|---|---|
| Compte créé | `account.create` | Un nouveau compte a été créé. |
| Compte en cours de révision | `account.under_review` | Le compte est en attente de révision. |
| Compte mis à jour | `account.updated` | Les détails du compte ont changé. |
| Statut du compte mis à jour | `account.status.updated` | Le statut du compte a changé (ex. : approuvé, suspendu). |
| Adresse du compte mise à jour | `account.address.updated` | L'adresse du compte a été modifiée. |
| Détails du compte mis à jour | `account.details.updated` | Les champs de détails du compte ont changé. |
| Informations du compte mises à jour | `account.information.update` | Les informations générales du compte ont été mises à jour. |
| Gestionnaires du compte mis à jour | `account.managers.update` | La liste des gestionnaires du compte a changé. |
| Photo de profil mise à jour | `account.profile_picture.updated` | La photo de profil du compte a été modifiée. |
| Agent mis à jour | `account.agent.updated` | Les informations de l'agent du compte ont été modifiées. |
| Nom d'URL mis à jour | `account.url-name.update` | L'URL personnalisée du compte a changé. |
| Identifiant unique mis à jour | `account.unique-id.update` | L'identifiant unique du compte a été modifié. |

## Événements de contrat de réservation

Événements suivant le cycle de vie d'un contrat de réservation via TripPay.

| Événement | Clé | Description |
|---|---|---|
| Initialisation demandée | `booking.contract.initialize.request` | Une initialisation de contrat de réservation a été demandée. |
| Initialisation terminée | `booking.contract.initialize.complete` | L'initialisation du contrat de réservation est terminée. |
| Contrat créé | `booking.contract.created` | Un nouveau contrat de réservation a été créé. |
| Contrat en traitement | `booking.contract.processing` | Le paiement du contrat de réservation est en cours de traitement. |
| Contrat échoué | `booking.contract.failed` | Le paiement du contrat de réservation a échoué. |
| Contrat annulé | `booking.contract.cancelled` | Le contrat de réservation a été annulé. |
| Remboursement partiel | `booking.contract.refund.partial` | Un remboursement partiel a été effectué sur le contrat. |
| Remboursement complet | `booking.contract.refund.full` | Un remboursement complet a été effectué sur le contrat. |
| Demande de remboursement | `booking.contract.refund.request` | Une demande de remboursement a été soumise. |
| Erreur de remboursement | `booking.contract.refund.error` | Une erreur est survenue lors du traitement d'un remboursement. |
| Paiement créé | `booking.contract.payout.create` | Un paiement a été créé pour le contrat. |
| Paiement terminé | `booking.contract.payout.complete` | Un paiement a été effectué avec succès. |

## Événements d'entité gestionnaire

Événements pour les entités gestionnaires (propriétés, affiliés, entreprises) et leur administration.

### Abonnement

| Événement | Clé | Description |
|---|---|---|
| Abonnement provisionné | `managing_entity.subscription.provisioned` | Un nouvel abonnement a été provisionné. |
| Abonnement modifié | `managing_entity.subscription.changed` | Le plan d'abonnement a été modifié. |
| Facture d'abonnement modifiée | `managing_entity.subscription.invoice.changed` | La facture d'abonnement a été mise à jour. |

### Statut et administration

| Événement | Clé | Description |
|---|---|---|
| Statut mis à jour | `managing_entity.status.updated` | Le statut de l'entité a changé. |
| Entité supprimée | `managing_entity.deleted` | Une entité gestionnaire a été supprimée. |
| Agent mis à jour | `managing_entity.agent.updated` | L'agent de l'entité a été modifié. |
| Notification email à l'agent | `managing_entity.agent.email.notification` | Une notification par email a été envoyée à l'agent. |
| Notification email au gestionnaire | `managing_entity.manager.email.notification` | Une notification par email a été envoyée à un gestionnaire. |

### Cycle de vie du gestionnaire

| Événement | Clé | Description |
|---|---|---|
| Gestionnaire ajouté | `managing_entity.manager.added` | Un nouveau gestionnaire a été ajouté. |
| Gestionnaire invité | `managing_entity.manager.invited` | Une invitation a été envoyée à un gestionnaire. |
| Gestionnaire supprimé | `managing_entity.manager.removed` | Un gestionnaire a été supprimé. |
| Gestionnaire accepté | `managing_entity.manager.accepted` | Une invitation de gestionnaire a été acceptée. |
| Gestionnaire rejeté | `managing_entity.manager.rejected` | Une invitation de gestionnaire a été rejetée. |
| Propriétaire défini | `managing_entity.manager.owner.set` | La propriété de l'entité a été transférée. |

|---|---|---|

| Représentant du compte mis à jour | `managing_entity.account_representative.updated` | Le représentant du compte a été modifié. |
| Photo de profil mise à jour | `managing_entity.profile_picture.updated` | La photo de profil de l'entité a été mise à jour. |
