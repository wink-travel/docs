## API de paiement

Les points de terminaison de paiement sont destinés aux développeurs souhaitant acheter des voyages. Vous pouvez effectuer ces achats via l'API en tant qu'agent de voyages enregistré ou en utilisant notre API en conjonction avec notre widget de paiement conforme aux normes PCI pour toutes les autres entités.
L'API de paiement propose des points de terminaison permettant aux affiliés et aux hôtels de suivre les réservations, les analyses et la disponibilité des fonds. Ils peuvent également choisir de retirer les fonds disponibles sur leur compte bancaire.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Apprendre encore plus
</LinkButton>
