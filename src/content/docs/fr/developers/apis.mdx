</LinkButton>

</LinkButton>

## Analytics API

L'API Analytics expose des points de terminaison pour toute personne souhaitant obtenir des informations sur le classement des leaders ainsi qu'une analyse approfondie des séries temporelles, des données au niveau de la plateforme ; hébergeant une grande variété de métriques.

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
En savoir plus
</LinkButton>

## Payment API

Les points de terminaison Payment sont destinés aux développeurs qui souhaitent acheter de l'inventaire de voyage. Cela peut être fait via l'API en tant qu'agent de voyage enregistré ou en utilisant notre API conjointement avec notre widget de paiement conforme PCI pour toutes les autres entités.
L'API Payment expose des points de terminaison pour les affiliés et les hôtels afin de suivre les réservations, les analyses et la disponibilité des fonds. Ils peuvent également choisir de retirer les fonds disponibles sur leur compte bancaire.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
En savoir plus
</LinkButton>
