Pour tous ceux qui utilisent WordPress pour leur entreprise ou leur blog, nous recommandons vivement d'utiliser le [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
Le plugin vous donne un accès facile à vos cartes Wink, grilles, cartes, etc., et rend aussi simple que de glisser du contenu dans vos articles et pages en utilisant votre constructeur de pages préféré ou nos shortcodes personnalisés.

##  Installation

Voici comment installer le `Wink Affiliate WordPress Plugin` sur votre site WordPress.

<Steps>
1. Connectez-vous en tant qu'administrateur et accédez au Tableau de bord.
2. Cliquez sur `Extensions > Ajouter`.
3. Recherchez `Wink Affiliate WordPress Plugin`.
4. Cliquez sur `Installer` pour télécharger et installer le plugin sur votre site.
5. Cliquez sur `Activer` pour activer le plugin sur votre site.
6. L'activation vous redirigera vers la page `Paramètres Wink`. *Vous pouvez aussi y accéder en naviguant vers `Apparence > Personnaliser > Paramètres Wink`.*
7. Il vous sera demandé votre `Client ID` et `Client Secret`. Rendez-vous sur [Applications](/fr/developers/applications) pour apprendre comment récupérer vos identifiants.
8. Entrez vos identifiants et cliquez sur `Publier`. Vous êtes maintenant connecté à votre compte Wink.
</Steps>

## Constructeurs de pages

Nous supportons les constructeurs de pages les plus populaires sur WordPress, ce qui facilite l'utilisation de Wink pour tous.

### Gutenberg

Gutenberg est l'éditeur natif de WordPress.

### Avada

[Avada](https://avada.com) est célèbre pour sa large sélection de magnifiques modèles ainsi que pour son constructeur de pages robuste avec un ensemble de fonctionnalités avancées.

### Elementor

[Elementor](https://elementor.com) est le constructeur de pages le plus populaire [actuellement], avec une version gratuite et une version pro disponibles.

### WPBakery

[WPBakery](https://wpbakery.com) a été le premier constructeur de pages avancé sur le marché.

## Shortcodes

### [winklookup]

Fonctionne avec la grille de contenu classée. Tapez un lieu que vous souhaitez visiter et affichez l'inventaire.

### [winksearch]

Bouton simple pour ouvrir le formulaire d'itinéraire.

### [winkaccount]

Bouton de compte pour vous authentifier. Une fois authentifié, il se transforme en menu déroulant avec des options de compte.

### [winkitinerary]

Identique au bouton de recherche, mais il contient les informations d'itinéraire comme texte du bouton.

### [winkcontent]

Vous permet d'intégrer l'inventaire que vous avez sélectionné sur Wink directement dans une page ou un article. Ce shortcode attend également un ID de mise en page.

La mise en page

```
[winkcontent layoutid=""]
```
