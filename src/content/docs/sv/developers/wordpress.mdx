För alla som använder WordPress för sitt företag eller sin blogg rekommenderar vi starkt att använda[Wink Affiliate WordPress-plugin](https://wordpress.org/plugins/wink2travel/).
Plugin-programmet ger dig enkel åtkomst till dina Wink-kort, rutnät, kartor etc. och gör det så enkelt som att dra innehåll till dina inlägg och sidor med hjälp av din favoritsidbyggare eller våra anpassade kortkoder.

## Installera

Så här installerar du`Wink Affiliate WordPress Plugin`in på din WordPress-webbplats.

<Steps>
  1. När du är inloggad som administratör och på instrumentpanelen.
  2. Klick `Plugins > Add New Plugin`.
  3. Leta efter `Wink Affiliate WordPress Plugin`.
  4. Klick `Install` för att ladda ner och installera plugin-programmet på din webbplats.
  5. Klick `Activate` för att aktivera plugin-programmet på din webbplats.
  6. Aktivering vidarebefordrar dig `Wink Settings` sida. *Du kan också gå dit genom att navigera till `Appearance > Customize > Wink Settings`.*
  7. Den kommer att be dig om din `Client ID` och `Client Secret`Gå till [Applikationer](/developers/applications) för att lära dig hur du hämtar dina inloggningsuppgifter.
  8. Ange dina inloggningsuppgifter och klicka `Publish`Du är nu ansluten till ditt Wink-konto.
</Steps>

## Sidbyggare

Vi stöder de mest populära sidbyggarna på WordPress vilket gör det enkelt för alla som vill använda Wink.

### Gutenberg

Gutenberg är den inbyggda redigeraren från WordPress.

### Avada

[Avada](https://avada.com)är känt för sitt stora urval av vackra mallar tillsammans med en robust sidbyggare med en avancerad funktionsuppsättning.

### Elementor

[Elementor](https://elementor.com)är den \[för närvarande] mest populära sidbyggaren med både en gratis- och en proversion tillgänglig.

### WPBakery

[WPBakery](https://wpbakery.com)var den första avancerade sidbyggaren på marknaden.

## Korta koder

### \[winklookup]

Fungerar med rutnät för rankat innehåll. Skriv in en plats du vill besöka så visas annonsutrymmet.

### \[blinksökning]

Enkel knapp för att öppna resplanformuläret.

### \[winkkonto]

Konto-knapp för att låta dig autentisera. När den är autentiserad förvandlas den till en rullgardinsmeny med kontoalternativ.

### \[winkitinerary]

Samma som sökknappen, bara att den innehåller resplaninformation som knapptext.

### \[blinkinnehåll]

Låter dig bädda in det inventarium du har valt på Wink direkt på en sida eller ett inlägg. Denna shortcode förväntar sig också ett layout-ID.

Layouten

```
[winkcontent layoutid=""]
```
