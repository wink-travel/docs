För alla som använder WordPress för sin verksamhet eller blogg rekommenderar vi starkt att använda [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
Pluginet ger dig enkel tillgång till dina Wink-kort, rutnät, kartor med mera, och gör det lika enkelt som att dra innehåll in i dina inlägg och sidor med din favorit-sidbyggare eller våra anpassade kortkoder.

## Installera

Så här installerar du `Wink Affiliate WordPress Plugin` på din WordPress-webbplats.

<Steps>
1. När du är inloggad som administratör och på instrumentpanelen.
2. Klicka på `Plugins > Add New Plugin`.
3. Sök efter `Wink Affiliate WordPress Plugin`.
4. Klicka på `Install` för att ladda ner och installera pluginet på din webbplats.
5. Klicka på `Activate` för att aktivera pluginet på din webbplats.
6. Aktiveringen skickar dig till sidan `Wink Settings`. *Du kan också gå dit genom att navigera till `Appearance > Customize > Wink Settings`.*
7. Den kommer att be dig om ditt `Client ID` och `Client Secret`. Gå till [Applications](/sv/developers/applications) för att lära dig hur du hämtar dina uppgifter.
8. Ange dina uppgifter och klicka på `Publish`. Du är nu ansluten till ditt Wink-konto.
</Steps>

## Sidbyggare

Vi stödjer de mest populära sidbyggarna på WordPress som gör det enkelt för alla som vill använda Wink.

### Gutenberg

Gutenberg är den inbyggda editorn från WordPress.

### Avada

[Avada](https://avada.com) är känd för sitt stora urval av vackra mallar tillsammans med en robust sidbyggare med avancerade funktioner.

### Elementor

[Elementor](https://elementor.com) är den [för närvarande] mest populära sidbyggaren med både en gratis- och pro-version tillgänglig.

### WPBakery

[WPBakery](https://wpbakery.com) var den första avancerade sidbyggaren på marknaden.

## Kortkoder

### [winklookup]

Fungerar med rankat innehållsrutnät. Skriv in en plats du vill besöka och få visat tillgängligt innehåll.

### [winksearch]

Enkel knapp för att öppna resplanformuläret.

### [winkaccount]

Kontoknapp för att låta dig autentisera. När du är autentiserad blir den en dropdown med kontoval.

### [winkitinerary]

Samma som sökknappen, men innehåller resplaninformation som knapptext.

### [winkcontent]

Låter dig bädda in det inventarium du valt på Wink direkt i en sida eller ett inlägg. Denna kortkod förväntar sig också ett layout-ID.

Layouten

```
[winkcontent layoutid=""]
```
