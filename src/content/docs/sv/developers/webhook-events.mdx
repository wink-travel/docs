Den här sidan listar alla webhook-händelser du kan prenumerera på i Wink. För installationsinstruktioner, se [Webhook Integration Guide](/sv/guides/webhook-integration) eller [Webhooks management page](/sv/my-account/webhooks).

## Hur man prenumererar

1. Navigera till **Applications > Webhooks** i din portal.
2. Skapa en webhook och välj de händelser du vill ta emot.
3. Wink skickar en HTTP POST till din endpoint varje gång händelsen inträffar.

För fullständiga installationssteg och bästa praxis, se [Webhook Integration](/sv/guides/webhook-integration).

## Kontohändelser

Händelser relaterade till din TripPay-kontos livscykel och inställningar.

| Event | Key | Description |
|---|---|---|
| Konto skapat | `account.create` | Ett nytt konto har skapats. |
| Konto under granskning | `account.under_review` | Konto väntar på granskning. |
| Konto uppdaterat | `account.updated` | Kontodetaljer har ändrats. |
| Kontostatus uppdaterad | `account.status.updated` | Kontostatus har ändrats (t.ex. godkänt, avstängt). |
| Kontoadress uppdaterad | `account.address.updated` | Kontoadress har ändrats. |
| Kontodetaljer uppdaterade | `account.details.updated` | Kontodetaljfält har ändrats. |
| Kontoinformation uppdaterad | `account.information.update` | Allmän kontoinformation har uppdaterats. |
| Kontohanterare uppdaterade | `account.managers.update` | Lista över kontohanterare har ändrats. |
| Profilbild uppdaterad | `account.profile_picture.updated` | Profilbild för kontot har ändrats. |
| Agent uppdaterad | `account.agent.updated` | Agentinformation på kontot har ändrats. |
| URL-namn uppdaterat | `account.url-name.update` | Kontots vanity-URL har ändrats. |
| Unikt ID uppdaterat | `account.unique-id.update` | Kontots unika identifierare har ändrats. |

## Bokningskontrakts-händelser

Händelser som spårar livscykeln för ett bokningskontrakt via TripPay.

|---|---|---|
| Initiering begärd | `booking.contract.initialize.request` | Initiering av bokningskontrakt har begärts. |

| Kontrakt avbrutet | `booking.contract.cancelled` | Bokningskontrakt har avbrutits. |
| Partiell återbetalning | `booking.contract.refund.partial` | En partiell återbetalning har utfärdats för kontraktet. |
| Full återbetalning | `booking.contract.refund.full` | En full återbetalning har utfärdats för kontraktet. |
| Återbetalning begärd | `booking.contract.refund.request` | En begäran om återbetalning har skickats in. |
| Återbetalningsfel | `booking.contract.refund.error` | Ett fel uppstod vid hantering av återbetalning. |
| Utbetalning skapad | `booking.contract.payout.create` | En utbetalning har skapats för kontraktet. |
| Utbetalning slutförd | `booking.contract.payout.complete` | En utbetalning har slutförts framgångsrikt. |
