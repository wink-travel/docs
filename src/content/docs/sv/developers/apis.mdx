## Betalnings-API

Betalningsslutpunkter är för utvecklare som vill köpa resematerial. Detta kan göras via API:et som registrerad resebyrå eller med hjälp av vårt API tillsammans med vår PCI-kompatibla betalningswidget för alla andra enheter.
Betalnings-API:et exponerar slutpunkter för affiliates och hotell för att spåra bokningar, analyser och tillgänglighet av medel. De kan också välja att ta ut tillgängliga medel till sitt bankkonto.

  Läs mer
</LinkButton>
