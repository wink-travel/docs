## Payment API

Payment endpoints är för utvecklare som vill köpa reseinventarium. Detta kan göras via API:et som registrerad Travel Agent eller använda vårt API i kombination med vår PCI-kompatibla betalningswidget för alla andra enheter.
Payment API exponerar endpoints för affiliates och hotell att spåra bokningar, analyser och tillgängliga medel. De kan också välja att ta ut tillgängliga medel till sitt bankkonto.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">

</LinkButton>
