Wink integreras med ledande channel managers, fastighetshanteringssystem (PMS) och centrala reservationssystem (CRS) för att hålla dina priser och tillgänglighet synkroniserade över alla dina försäljningskanaler.

## Hur det fungerar

Integrationer med channel managers möjliggör en tvåvägsdataflöde mellan ditt fastighetshanteringssystem och Wink:

1. **Prisuppdatering** — Din channel manager skickar prisuppdateringar till Wink. När du uppdaterar dina priser i ditt PMS speglas dessa ändringar automatiskt på Wink.
2. **Synkronisering av tillgänglighet** — Din channel manager skickar tillgänglighetsuppdateringar till Wink. När rum bokas eller släpps i ditt PMS uppdateras Winks lager i realtid.
3. **Bokningsnotifikationer** — När en bokning görs på Wink meddelas din channel manager så att reservationen visas i ditt PMS.

## Stödda channel managers

Wink integreras för närvarande med följande system:

| Channel Manager | Beskrivning |
|---|---|
| **Channex** | Global distributions- och channel management-plattform. |
| **TravelClick** | Hotellösningar och channel management (Amadeus). |
| **Omnibees** | Distribution och intäkthantering för hotell. |
| **D-Edge** | Hotellteknologi och centrala reservationslösningar. |
| **Allotz** | Lagerallokering och channel management. |
| **Ezee / Yanolja** | Molnbaserat PMS och channel management med fokus på den asiatiska marknaden. |
| **RateGain** | Prisintelligens och distributionsteknologi. |
| **RateTiger** | Distribution av priser och tillgänglighet i realtid. |
| **CloudBeds** | Molnbaserat PMS, channel manager och bokningsmotor. |
| **SiteMinder** | Ledande hotell channel manager och distributionsplattform. |
| **Exely** | Synkronisering av priser och tillgänglighet för hotell. |

:::note
Om din channel manager inte finns med i listan, kontakta Wink-teamet. Nya integrationer kan läggas till baserat på OTA (Open Travel Alliance) protokollstandarder som Wink stödjer.
:::

## För utvecklare

Om du bygger en integration för channel manager eller PMS med Wink, använd Channel Manager API-endpoints:

- **Prisuppdateringar** — Skicka prisändringar till Wink.
- **Tillgänglighetsuppdateringar** — Skicka tillgänglighetsändringar till Wink.
- **Bokningshämtning** — Hämta bokningar från Wink till ditt system.

Channel Manager API följer OTA-protokollstandarder (SOAP/XML) för kompatibilitet med befintliga hotellssystem.

## Webhook-notifikationer

Du kan prenumerera på webhook-händelser från channel manager för att få notifikationer i realtid:

- `channel-manager.update.rate` — Prisuppdatering mottagen.
- `channel-manager.update.availability` — Tillgänglighetsuppdatering mottagen.
- `channel-manager.update` — Allmän channel manager-uppdatering.

Se [Webhook Events Catalog](/sv/developers/webhook-events#channel-manager-events) för detaljer.
