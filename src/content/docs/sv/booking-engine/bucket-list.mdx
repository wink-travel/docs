Varje gång du ser Wink-inventarie [kort](/sv/studio/cards), ser du ett hjärta i det övre högra hörnet av kortet som låter dig lägga till det inventariet i din bucket list.

Följ dessa steg för att hantera din bucket list:

<Steps>
1. Klicka på ♡ `hjärtikonen` för det inventarie du vill lägga till.
2. Om du är inloggad blir hjärtikonen röd.
3. Om du inte är inloggad kommer du att omdirigeras för att logga in.
</Steps>

:::tip
Klicka på ♥️ `hjärtikonen` en andra gång för att ta bort det från din lista.
:::

Du kan hantera alla dina bucket list-objekt genom att klicka på din profilikon i det övre högra hörnet och sedan klicka på knappen `Bucket list`.
