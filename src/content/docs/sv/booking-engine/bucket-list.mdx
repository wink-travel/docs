Överallt där du ser Wink-inventering[kort](/studio/cards), ser du ett hjärta i kortets övre högra hörn som låter dig lägga till det inventariet på din bucketlist.

Följ dessa steg för att hantera din bucketlist:

<Steps>
  1. Klicka på ♡ `heart icon` för det lager du vill lägga till.
  2. Om du är inloggad blir hjärtikonen röd.
  3. Om du inte är inloggad kommer du att omdirigeras för att göra det.
</Steps>


:::tips
Klicka på ♥️`heart icon`en andra gång för att ta bort den från din lista.
:::

Du kan hantera alla dina bucket list-punkter genom att klicka på din profilikon i det övre högra hörnet och sedan klicka på`Bucket list`knappen.
