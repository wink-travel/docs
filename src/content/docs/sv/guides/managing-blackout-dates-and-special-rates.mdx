Inte alla dagar har samma pris. Evenemang, helgdagar, lågsäsong och sista minuten-platser kräver olika prissättning. Den här guiden visar hur du använder specialpriser och begränsningar i prisplaner för att hantera dessa situationer.

## Prissättning vid evenemang

När en konferens, festival eller lokalt evenemang ökar efterfrågan vill du höja priserna för dessa datum.

<Steps>
1. Gå till `Monetize > Special Rates` i ditt Extranet.
2. Klicka på `Create a new special rate`.
3. Välj masterpriset och rumstypen som ska åsidosättas.
4. Ange datumintervallet som täcker evenemangsperioden.
5. Fyll i de justerade (högre) dagliga priserna.
6. Klicka på `Save`.
</Steps>

Specialpriset åsidosätter masterpriset endast för dessa datum. Din ordinarie prissättning förblir oförändrad utanför evenemangsperioden.

## Säsongsjusteringar

För längre perioder (högsäsong, mellansäsong, lågsäsong) kan du antingen:

- **Använda specialpriser** — Skapa åsidosättningar för varje säsong med justerad prissättning.
- **Uppdatera masterpriser** — Justera baspriserna direkt för dessa perioder.

Specialpriser är bättre för tillfälliga justeringar, medan uppdatering av masterpriser passar för förutsägbara årliga mönster.

## Sista minuten-priser

För att fylla osålda rum nära ankomstdatum:

<Steps>
1. Skapa en ny **prisplan** med en begränsning för förhandsbokning (t.ex. måste bokas senast 3 dagar i förväg).
2. Koppla den till ett **specialpris** som ger rabatt för de aktuella datumen.
3. Denna prisplan visas endast för resenärer som bokar inom sista minuten-fönstret.
</Steps>

## Blackout-datum

För att blockera vissa datum från en specifik prisplan (t.ex. inga kampanjpriser under högsäsong):

<Steps>
1. Gå till `Monetize > Rate Plans` och välj prisplanen.
2. Navigera till avsnittet **Restrictions**.
3. Lägg till en **Sell date**-begränsning för att begränsa när priset kan bokas.
4. Eller använd **Stay date**-begränsningar för att begränsa vilka datum priset gäller för.
</Steps>

Till exempel kan du begränsa din "Early Bird"-prisplan så att den inte är tillgänglig för bokningar under december (högsäsong), medan din ordinarie prisplan förblir öppen.

## Kombinera begränsningar

Begränsningar kan kombineras för exakt kontroll:

| Begränsning | Vad den styr |
| -- | -- |
| **Sell dates** | När priset kan bokas |
| **Stay dates** | Vilka vistelsedatum priset gäller för |
| **Available days** | Vilka veckodagar priset visas |
| **Arrival days** | Vilka dagar gästen måste ankomma på |
| **Departure days** | Vilka dagar gästen måste avresa på |
| **Required days** | Gästen måste stanna över minst en av dessa dagar |
| **Advance booking** | Hur långt i förväg bokningen måste göras |
| **Length of stay** | Minsta och högsta antal nätter |
