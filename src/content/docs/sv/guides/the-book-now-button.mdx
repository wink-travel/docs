**BOKA NU**-knappen är ett allmänt förekommande begrepp inom hotellbranschen som syftar på hotellets egen integration med en leverantör som känner till fastighetens tillgänglighet och priser och kan låta resenärer boka ett rum. Knappen är det enklaste sättet för hotell att kunna sälja rum på sin egen webbplats och på sina egna villkor.

*Många hotell använder den inbyggda bokningsmotorn som erbjuds av deras kanalhanterare. Vi rekommenderar att hotell gör lite research och jämför inhemska bokningsmotorer innan de binder sig till någon särskild.*

## Målgrupp

Denna artikel är främst för hotell som vill installera vår kostnadsfria, högkonverterande bokningsmotor på sin webbplats.
Men... den är också värdefull läsning för alla som vill använda våra teknologier på sina webbplatser.

## via Länk

För den enklaste integrationen rekommenderar vi att använda en [Delbar Länk](/sv/studio/shareable-links).

Här är stegen för att göra detta:

<Steps>
1. Gå till https://extranet.wink.travel
2. Klicka på `Sign-In / Register` uppe till höger och logga in med ditt användarkonto.
3. Välj den fastighet du vill arbeta med från rullgardinsmenyn uppe till vänster.
4. Klicka på `Booking engine` i navigeringsfältet.
5. Sidan visar din länk.
6. Ge denna länk till din webbansvarige och använd den för din **BOKA NU**-knapp.
7. Du är ✅.
</Steps>

### Avancerat

Länken kommer redan med en resplan inbakad i URL:en, som härstammar från din valda `customization`.
Du kan anpassa din länk ytterligare genom att:

- Ändra den valda anpassningen ELLER
- Åsidosätta resplanen

I denna artikel fokuserar vi på det senare eftersom det ger dig större frihet att låta resenärer välja en resplan på din webbplats och vidarebefordra den till Winks bokningsmotor.

Så här skickar du med din egen resplan tillsammans med URL:en.

Giltiga förfrågningsparametrar:

- **sd** För att lägga till startdatum: *sd=2024-08-24*
- **n** För att lägga till antal nätter: *n=1* ELLER
- **ed** Använd slutdatum: *ed=2024-08-25*
- **rc** Gäster: *rc=a2* (Vuxna: 2)
- **l** Begär visningsspråk: *l=es*
- **c** Begär visningsvaluta: *c=EUR*
- **p** Lägg till en kampanj: *p=ABC123*

URL:en skulle se ut så här:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2&l=es&c=EUR

För mer avancerade användningsfall kan du också begära flera rumskonfigurationer med hjälp av `+`-tecknet:

Exempel:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2-ca16q1-ca8q1+a2&l=es&c=EUR

- Rumstyp 1 = Vuxna: 2, Barn: 2 = 16 år + 8 år gamla.
- Rumstyp 2 = Vuxna: 2.

### Exempel

Exempel på **BOKA NU**-knapp i HTML:

*Detta exempel förutsätter att du använder [Bootstrap](https://getbootstrap.com) som ditt primära CSS-bibliotek.*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Boka ett rum"
>
BOKA NU
</a>
```

### Andra fördelar

Det finns andra fördelar med att använda en av våra länkar som din `boka nu`-knapp.

- **Socialvänlig** Länken kan enkelt delas på sociala medier och visar `dynamisk prissättning` ovanpå den bild du valt.
- **Kort** URL:en är mycket delbar och du kan använda den på platser som din Instagram Bio URL.

## via Inbäddning

Vissa hotell vill ha lite mer funktionalitet och en djupare integration för sin webbplats. Detta exempel täcker inbäddning av ett [Card](/sv/studio/cards) på deras webbplats.

Här är stegen för att göra detta:

<Steps>
1. Gå till https://studio.wink.travel
2. Klicka på `Sign-In / Register` uppe till höger och logga in med ditt användarkonto.
3. Välj den fastighet du vill arbeta med från uppe till vänster.
4. Klicka på `Search` i huvudnavigeringsfältet.
6. Skriv in namnet på din fastighet och klicka på sökknappen.
7. Klicka på länken `Actions` på ditt sökresultatskort och välj `Create > Make a Card`.
8. Detta skapar ett `Card`.
9. Ett meddelande visas uppe till höger som berättar att ditt kort är klart och om du vill anpassa det ytterligare.
10. Klicka på `Tools > Cards` i huvudnavigeringsfältet så ser du ditt nya kort.
11. Klicka på länken `Actions` på ditt kort och klicka på `Embed`.
12. Detta öppnar ett nytt fönster med exempel på HTML-kod som du kan kopiera.
13. För att hämta ditt `Client-ID`, gå till [Applications](/sv/developers/applications).
14. För att hämta ditt `Configuration-ID`, gå till [Customization](/sv/studio/customization)
14. Du är ✅.
</Steps>

Exempel på inbäddad kort-HTML:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*HTML-exemplet ovan inkluderar inte den nödvändiga Wink CSS och Javascript; bara kortet.*

En bokare ser nu dynamisk prissättning, kan ändra resplan och klicka för att boka direkt på din webbplats. Bokningsknappen tar dig till din Wink-fastighets landningssida för att slutföra betalningen.
