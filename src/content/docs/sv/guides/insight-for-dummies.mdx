## Hitta saker att sälja

När du börjar som affiliate på Wink säger du först till dig själv:

> Nu ska vi tjäna pengar!.

Omedelbart efter frågar du dig:

> Hur? Vad? När? Var?

Vissa affiliates vet redan vad de vill sälja genom sina kanaler. Om du är ny i spelet och vill ta reda på vilka leverantörer du ska marknadsföra... då kan analys vara till stor hjälp.

Här är ett exempel på hur du kan göra detta för allra första gången:

<Steps>
1. Gå till [Wink Studio](/sv/studio/search) för att söka efter leverantörer.
2. Skapa ett diagram som spårar bokningar över flera länder. *Nu vet du vilka länder som presterar bäst.*
3. Skapa ett annat diagram för några av städerna i de länder som presterar bäst. *Nu vet du vilka städer som presterar bäst.*
4. Sök efter leverantörer i dessa städer och hitta några du gillar.
5. Spåra dessa leverantörer genom att klicka på knappen `Actions > Supplier details > Track performance` på deras sökresultatskort *Se bild nedan*.
6. Detta skapar ett nytt diagram för den leverantören som du kan följa.
7. Jämför tidigare prestation för varje leverantör tillsammans med genomsnittligt pris och den provision som betalas.
8. Detta bör ge dig ett par leverantörer att börja producera för.
</Steps>

Byt nu kontext och börja spåra resenärer.

<Steps>
1. Spåra vilka länder och städer som bokar flest resor.
2. Spåra vid vilka tider dessa resor bokas.
</Steps>

Med den kunskapen har du en ganska bra förståelse för vad som säljer, vem som köper och när.

Du kan gå ännu mer på djupet och ta reda på exakt vad som bokas också. *Exempel: Spabehandlingar säljer bra i Berlin, Tyskland under helgerna.*

## Konkurrensgrupp

Inom hotellbranschen kallas en ”*comp set*” (förkortning för competitive set) en grupp hotell som ett visst hotell betraktar som sina främsta konkurrenter.
Dessa hotell väljs ut baserat på faktorer som läge, målgrupp, bekvämligheter och prissättning.
Comp set används för benchmarking och prestationsjämförelse, vilket hjälper ett hotell att bedöma sin egen position på marknaden.

Comp sets är tillgängliga för alla hotell på Wink-plattformen utan kostnad. Du kan spåra prestationen för vilket hotell som helst på vår plattform med dessa mått:

För att skapa en `comp set`, följ dessa steg:

<Steps>
1. Gå till [Wink Studio](/sv/studio/search) för att söka efter leverantörer.
2. Sök efter andra fastigheter du vill jämföra dig med.
3. Spåra fastigheten genom att klicka på knappen `Actions > Supplier details > Track performance` på deras sökresultatskort *Se bild nedan*.
4. Detta skapar ett nytt diagram för den leverantören som du kan följa.
5. Gå till dina [Diagram](https://extranet.wink.travel/analytics/list).
6. För att uppdatera diagrammet till att spåra de mått du vill, klicka på knappen `Actions > Update` för ditt nya diagram.
6. Klicka på knappen `Save` för att fortsätta.
</Steps>
