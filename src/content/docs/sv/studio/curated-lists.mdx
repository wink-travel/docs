Kuraterade listor är en funktion som låter dig gruppera och sortera inventarier som du har valt från saker du hittat när du sökt efter reseinventarier.  
När listan är skapad kan du göra den tillgänglig för dina användare genom att erbjuda den till försäljning som ett [grid](/sv/studio/grids) eller en [karta](/sv/studio/maps) via [WinkLinks](/sv/link-manager/wink-links), vår [WordPress-plugin](/sv/developers/wordpress/) eller genom att bädda in en av våra [Web Components](/sv/developers/web-components) på din webbplats.

Som standard startar varje affiliatekonto med en `Favorites`-lista som du kan använda direkt. `Favorites` är speciell och kan inte tas bort.  
Alla kuraterade listor tillåter dig att göra följande:

## Skapa en lista

- Från instrumentpanelen, klicka på `Inventory > Curated Lists` i navigeringsfältet.  
- Skapa en ny kuraterad lista genom att klicka på knappen `Make a new list`.  
- Du kommer att bli ombedd att namnge din lista. *Exempel: Topp 3 NYC-val*.  
- Klicka på knappen `Save`.

## Lägg till ett listobjekt

Du kan lägga till ett listobjekt i vilken lista som helst från sökresultaten.

- Gå till [Search](/sv/studio/search)  
- Sök efter något du vill lägga till i din lista.  
- När du hittar det, klicka på länken `Actions` längst ned till vänster på sökresultatskortet.  
- Klicka på `Add to...` och välj sedan den kuraterade lista du vill lägga till detta inventarie i.  
- Gå tillbaka till sidan för kuraterade listor för att verifiera att objektet lades till i din lista.

Varje lista har några åtgärder du kan utföra på den.

- Konvertera listan till en [karta](/sv/studio/maps).  
- Konvertera listan till ett [grid](/sv/studio/grids).  
- Ta bort listan (*utom Favorites*).

## API

Utvecklare som vill hantera kuraterade listor kan gå till [Developers > API > Browse](/sv/developers/apis/#browse-api).
