Studio erbjuder flera bäddbara widgets som du kan lägga till på din egen webbplats för att öka bokningarna.

För att komma åt bäddningswidgets, navigera till `Tools > Miscellaneous` i sidomenyn.

## Tillgängliga widgets

| Widget | Beskrivning |
| -- | -- |
| **Account button** | En varumärkesknapp som länkar till din affiliate-sida |
| **Itinerary button** | En knapp som öppnar resplanväljaren |
| **Itinerary form** | Ett inbäddat formulär för datum- och gästval |
| **Search button** | En knapp som öppnar fastighetssökning |
| **Lookup button** | En knapp för att söka specifikt inventarium |

## Hur man bäddar in

Varje widget tillhandahåller en kodsnutt som du kan kopiera och klistra in i din webbplats HTML. Snutten inkluderar ditt **Client ID** och **Customization ID** så att widgeten automatiskt kopplas till ditt affiliate-konto.

:::note
Om du använder WordPress erbjuder vi även ett WordPress-plugin för enklare integration.
:::
