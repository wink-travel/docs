Ett kort representerar en enskild leverantör (t.ex. hotell eller upplevelseleverantör) eller inventarie (dvs. rumstyp, mötesrum, spa etc.) och visar detaljerna via ett interaktivt, bokningsbart kortgränssnitt.
En användare kan interagera med ett kort på följande sätt:

- Klicka på knappen `Boka`.
- Klicka på länken `Galleri` för att se alla foton och videor för denna inventarie.
- Klicka på länken `Bästa erbjudande`, om tillgängligt, för att se det bäst prissatta rummet.
- Klicka på länken `Hotellinformation`, om tillgängligt, för att se fastighetsinformation.

Ett kort har flera sidor för att minimera kortets storlek och kategorisera data; vilket gör det lättillgängligt för användaren och ger dig större chans att konvertera användaren till en bokning.

Ett kort har antingen två eller tre sidor:

- Ett hotellkort visar fastighetsdata på framsidan och den bästa rumstypen på baksidan. Priset, på båda sidor, visar det bästa priset för hotellet.
- Ett rumstypkort visar rumstypdata på framsidan och fastighetsdata på baksidan. Priset visar priset för rumstypen på framsidan och det bästa priset för hotellet på baksidan.
- Alla andra kort (t.ex. spa, restaurang etc.) har 3 sidor. Exempel: Spa-data visas på framsidan med det bästa priset för spaet. Rumstypdata för det bäst prissatta rummet finns på den andra sidan. Fastighetsdata tillsammans med det bäst prissatta rummet finns på den tredje kortsidan.

Klicka på den knappen så omdirigeras du till vårt kortformulär där du kan börja anpassa ditt kort.

## Anpassa kort

Formuläret låter dig anpassa ditt kort på följande sätt:

<Steps>
 1. Ge det ett namn så att du kommer ihåg vad kortet handlar om.
 2. Välj den [Anpassning](/sv/studio/customization) du vill tillämpa på detta kort.
 3. Välj den initiala kortsidan du vill att användarna ska se först. Som standard är det den ursprungliga sidan för den inventarien.
 4. Välj det <Badge text="märke" variant="tip" size="small" /> du vill använda på kortet. Ett märke låter användare jämföra inventarier baserat på ett samlat mått som t.ex. `miljövänligt`.
 5. Lägg till nyckelord, separerade med kommatecken, som kommer att användas av [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 6. Lägg till titlar och beskrivningar på de språk du vill att användaren ska se.
 7. Välj en eller flera bilder som visas när användare klickar på länken `Galleri`.
 8. Klicka på knappen `Spara` för att fortsätta.
 </Steps>

 Efter att du sparat ditt kort omdirigeras du till din kortsida och ditt kort är nu redo att delas med världen.

- **Använd med WordPress** Visar hur du använder vårt [WordPress-plugin](/sv/developers/wordpress) för att bädda in detta kort på din webbplats.

Vi går igenom några av dessa alternativ mer i detalj nedan.

### Bädda in

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="GUEST_ROOM"
       id="2de7ee9c-61c9-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Så här bäddar du in ditt kort på din webbplats:

- Rad 3 visar hur du bäddar in Wink-stilarna på din webbplats.
- Raderna 6 till 9 visar hur du använder [wink-content-loader](/sv/developers/web-components/#content-loader) Web Component och ber den hämta ett gäst-rumskort för din kod.
- Rad 11 visar hur du bäddar in vår Javascript på din webbplats.
- Rad 13 visar hur du bäddar in [wink-app-loader](/sv/developers/web-components/#application-loader) Web Component och ber den hämta dina konfigurationsinställningar på sidnivå.
