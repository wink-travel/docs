Sparade sökningar gör vad namnet antyder. När du söker efter resekataloger har du möjlighet att spara din sökfråga för senare användning.

## Sparar din sökfråga

När du börjar söka efter lager (se[Söka](/studio/search)), har du möjlighet att finjustera dina sökfilter. Längst ner i frågeformuläret finns alternativet att`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Spara en sökning" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Visar länk till Spara sökfråga*</figcaption>
</figure>


Om du klickar på den här länken får du möjlighet att namnge sökningen (*t.ex. Mitt bibliotek med magiska hotell*) något du kommer att minnas senare.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Ge din sökning ett namn" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Ge din sökning ett namn*</figcaption>
</figure>


Ge den ett namn och klicka på`Save search`knappen. Efter det kan du navigera till`Inventory > Saves Searches`från huvudnavigeringsfältet för att se din nyskapade post.

## Lista åtgärder

Din sparade sökning kan användas för att visa dina användare reseinformation i form av ett rutnät eller en karta och få dina sökresultat att visas på det sätt du vill.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Sparade sökåtgärder" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Sparad sökfråga utan filter*</figcaption>
</figure>


Varje sparad sökfråga har åtgärder som du kan utföra på den:

* Konvertera sökfrågan till en[karta](/studio/maps).
* Konvertera sökfrågan till en[rutnät](/studio/grids).
* Ta bort den sparade sökningen.

## API

Utvecklare som vill hantera sparade sökningar kan gå vidare till[Utvecklare > API > Bläddra](/developers/apis/#browse-api).

## Vidare läsning
