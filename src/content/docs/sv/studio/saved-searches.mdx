Sparade sökningar gör precis vad namnet antyder. När du söker efter reseinventarier har du möjlighet att spara din sökfråga för senare användning.

## Spara din sökfråga

När du börjar söka efter inventarier (se [Search](/sv/studio/search)) kan du finjustera dina sökfilter.
Längst ner i sökformuläret finns alternativet `Save search`.

Genom att klicka på denna länk får du möjlighet att namnge denna sökning (*t.ex. Min samling av magiska hotell*) något du kommer ihåg till senare.

<figure>

</figure>

Ge den ett namn och klicka på knappen `Save search`. Därefter kan du navigera till `Inventory > Saves Searches` från huvudmenyn för att se din nyligen skapade post.

## Liståtgärder

Din sparade sökning kan användas för att visa reseinventarier för dina användare i form av ett rutnät eller en karta och göra så att dina sökresultat visas på det sätt du vill.

<figure>

    <figcaption>*Sparad sökfråga utan tillämpade filter*</figcaption>

Varje sparad sökfråga har åtgärder du kan utföra på den:

- Konvertera sökfrågan till en [karta](/sv/studio/maps).
- Konvertera sökfrågan till ett [rutnät](/sv/studio/grids).
- Ta bort den sparade sökningen.

## API

Utvecklare som vill hantera sparade sökningar kan gå till [Developers > API > Browse](/sv/developers/apis/#browse-api).
