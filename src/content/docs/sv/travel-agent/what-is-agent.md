Vår lösning för registrerade resebyråer är [Wink Agent](https://agent.wink.travel). Det är en självbetjäningsportal som agenter kan använda för att söka och boka åt sina kunder och tjäna en provision för sig själva på varje bokning.

Mer avancerade resebyråer kan välja att integrera på API-nivå för att hantera betalningar och utbetalningar utanför Wink-plattformen.

Resebyråer har en delmängd av samma portalfunktioner som [Wink Studio](/sv/studio/what-is-studio) så vi kommer inte att täcka dem igen här.

## Sök

Gå till [Search](/sv/studio/search) för att lära dig hur du söker efter och bokar hotell.

Annars, fortsätt till nästa avsnitt.

## API

Utvecklare som vill hantera `Travel Agent` kan gå till [Developers > API > Travel Agent](/sv/developers/apis/#travel-agent-api).
