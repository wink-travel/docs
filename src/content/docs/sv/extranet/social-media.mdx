Sociala medier håller dig i kontakt med nya och befintliga resenärer över tid.
Lägg till dina konton för sociala nätverk i din profil så att gäster enkelt kan kontakta dig och lära sig mer om din fastighet.

För att hantera dina konton för sociala nätverk, klicka på `Property > Social media` i huvudnavigeringsfliken.

<Steps>
1. Växla på knappen för det sociala nätverk du vill aktivera.
2. Ange namnet på ditt konto på det nätverket.
</Steps>

Klicka på `Save`-knappen för att fortsätta.
