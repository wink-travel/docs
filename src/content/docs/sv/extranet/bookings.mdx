När en bokning görs på Wink händer följande:

<Steps>
1. Din channel manager, CRS eller PMS meddelas.
2. Du får ett bekräftelsemail *(bokningsdiskens kontakter)* med en länk för att hantera din nya bokning.
3. Resenären får ett bekräftelsemail som också innehåller hur man kontaktar din bokningsdisk.
4. Utvecklare kan lyssna på webhook-händelsen `booking.created` och få detaljerad bokningsinformation från oss i realtid.
</Steps>

Följ antingen länken i mailet eller gå till `Konto > Bokningar` från huvudmenyn för att komma igång.

## Bokningsöversikt

Bokningsöversikten visar alla dina tidigare och framtida bokningar, sorterade efter bokningsdatum i stigande ordning.

### Filter

Du kan använda filter för att hitta de bokningar du söker.

- Bokningsdatum [intervall]
- Ankomstdatum [intervall]
- Avresedatum [intervall]
- Bokningskod
- Resenärens efternamn
- Resenärens förnamn
- Masterpriset namn
- Försäljningskanal / Källa

### Exportera

Du kan exportera bokningar som matchar dina aktuella filterkriterier till CSV.

För att exportera dina bokningar:

<Steps>
1. Se till att dina önskade filter är aktiva.
2. Vänta på att bokningsöversikten uppdateras.
2. Klicka på 🗂️ (*Fil-export*) ikonknappen längst ner till höger i översikten.
</Steps>

## Bokning

Välj den bokning du vill arbeta med från bokningsöversikten så omdirigeras du till sidan `Bokning`.
Denna sida innehåller all information du behöver för att välkomna din gäst på ankomstdagen på rätt sätt.

Sidan innehåller också några viktiga åtgärder du behöver för att ge dina gäster god service. Du hittar alla tillgängliga åtgärder i rullgardinsmenyn `Åtgärder` högst upp på sidan.

### Skicka om e-post

För att skicka om bekräftelsemailet till resenären, klicka på `Åtgärder > Skicka om bekräftelsemail`.

### Avboka bokning

För att avboka bokningen som anläggning, klicka på `Åtgärder > Avboka bokning`.

Ett popup-fönster visas där du kan:

<Steps>
1. Välja avbokningstyp. *t.ex. Felaktiga datum*
2. Ange en anledning till avbokningen.
</Steps>

*Denna knapp följer dina avbokningsregler och är därför inte alltid tillgänglig.*

### Begär återbetalning

För att begära återbetalning, oavsett anledning, klicka på `Åtgärder > Begär återbetalning`.

Ett popup-fönster visas där du kan:

<Steps>
1. Begära full eller delvis återbetalning.
    - Om du valde delvis, kommer du att bli ombedd att ange beloppet.
2. Välja en anledning.
3. Skriv en beskrivande anledning för begäran.
4. Ange om du även vill avboka bokningen, oavsett om begäran godkänns eller inte.
</Steps>

 *Denna åtgärd är tillgänglig tills medlen för denna bokning har behandlats.*.

## Kalender-synkronisering

För att börja synkronisera dina Wink-bokningar med din favoritkalenderprogramvara, gå till `Distribution > Synkronisera bokningar till din kalender` från huvudmenyn för att komma igång.

Första gången kommer du att bli ombedd att generera en passkod som du kan använda med din kalenderprogramvara för att autentisera anslutningen.

    />
    <figcaption>*Frågar om du vill generera en CalDAV-passkod*</figcaption>
</figure>

Klicka på knappen `Ja` för att fortsätta eller 'Nej' om du ändrat dig.

Du kommer att se 3 fält som du behöver använda för att koppla Wink till din kalenderprogramvara:

1. URL:en som används för att prenumerera på bokningar.
2. Användarnamnet för autentisering.
3. Passnyckeln för autentisering.

:::tip
CalDAV är ett standardprotokoll för kalendrar som stöds av de flesta kalenderapplikationer inklusive Google Kalender, Apple Kalender och Outlook. Lägg helt enkelt till URL:en som en ny kalenderprenumeration i din föredragna app.
:::

När du har slutfört detta steg och lagt till CalDAV-URL:en i din kalenderprogramvara, kommer din kalender börja visa bokningar från Wink.

3 dagar efter att gästen checkat ut får gästen ett mail från oss där vi ber dem att recensera din anläggning.
Recensionen består av enkla flervalsfrågor med värden från 1 till 5. Kategorierna är anläggning, tjänster och faciliteter.

Gäster kan också lämna ett meddelande som du kan svara på och välja att göra offentligt för andra gäster att se.
