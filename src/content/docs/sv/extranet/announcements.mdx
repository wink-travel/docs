Meddelanden är ett sätt att informera dina gäster om viktig information på ett sätt som uppmärksammas.
Om ett meddelande `utlöses` visas det som ett nytt fönster ovanpå din landningssida.

För att se dina meddelanden, klicka på `Property > Announcement` i huvudnavigeringsfältet.

Klicka på knappen `Create announcement` för att skapa en ny post.

<Steps>
1. Välj om du vill inkludera en titel eller inte.
2. Meddelanden kan utlösas på två sätt:
    - Visa alltid, varje gång någon besöker din sida.
    - Visa endast när resplanen matchar meddelandets resplan.
3. Om du valde att endast visa meddelandet under ett visst datumintervall, lägg till datumen i formulärfälten nedan.
4. Klicka på knappen `Save` så tas du tillbaka till dina meddelanden.
</Steps>
