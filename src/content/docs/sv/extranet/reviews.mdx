Recensionssidan visar alla gästrecensioner som lämnats för din fastighet.

För att se dina recensioner, klicka på `Distribution > Reviews` i huvudnavigeringsfältet.

## Bläddra bland recensioner

Recensionslistan kan filtreras för att hjälpa dig hitta specifik feedback:

- **Datumintervall** — Filtrera efter när recensionen lämnades.
- **Gästnamn** — Sök på för- eller efternamn.
- **Gästemail** — Sök recensioner via e-postadress.
- **Besvarad** — Filtrera efter om du har svarat på recensionen.

Resultaten är paginerade med 25 recensioner per sida.

## Recensionsdetaljer

Klicka på en recension för att se fullständiga detaljer, inklusive gästens skriftliga feedback och betyg. Använd detta för att förstå gästernas känslor och identifiera förbättringsområden.
