Att lägga till information om GM hjälper till att skapa mer igenkänning för platsen.

För att uppdatera information om General Manager, klicka på `Property > General Manager` i huvudnavigationsfliken.

Du kan lägga till namn och foto på din nuvarande GM tillsammans med ett personligt meddelande till gästerna.

Uppdatera formuläret och klicka på `Save`-knappen för att fortsätta.
