Du kommer att bli ombedd att koppla vissa livsstilar till din fastighet under uppsättningen.

För att hantera dina livsstilar, klicka på `Property > Lifestyles` i huvudnavigationsfliken.

Växla knappen bredvid de livsstilar du vill använda.

Klicka på `Save`-knappen för att fortsätta.

:::caution[Krav]
Du kan välja högst 6 livsstilar. Välj de som bäst matchar din fastighet.
:::
