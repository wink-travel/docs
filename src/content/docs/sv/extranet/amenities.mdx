Fastighetsfaciliteter är tjänster och bekvämligheter som finns tillgängliga på din fastighet.

För att hantera din fastighets faciliteter, klicka på `Property > Amenities` i huvudnavigationsfliken.

Markera alla rutor som gäller för din fastighet.

Klicka på knappen `Save` för att fortsätta.
