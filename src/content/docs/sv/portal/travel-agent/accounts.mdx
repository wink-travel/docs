[Wink Agent](https://agent.wink.travel) kopplar dig till reseinventarier världen över. Den här artikeln guidar dig genom att skapa och hantera dina reseagentkonton på Wink.

## Skapa konto

När du är autentiserad kommer du att omdirigeras till din [kontosida](https://agent.wink.travel/company/list) som visar alla dina reseagentkonton.

Klicka på knappen `Create new account` för att fortsätta. Du kommer att omdirigeras till vår [kontoskapande](https://agent.wink.travel/company/create) sida.

Fyll i alla obligatoriska fält.

<Steps>
1. **Namn** Namnet på ditt konto. *Detta behöver inte vara ditt riktiga namn men kommer att vara något användare ser när de tittar på din WinkLinks-sida eller när leverantörer vill koppla upp sig direkt med dig.*
2. **Företagstyp** Om du är ett företag eller en privatperson. *Används mest för att avgöra hur du är berättigad att ta emot betalningar.*
3. **Typ** Välj en typ som bäst passar ditt konto. *t.ex. Influencer*
4. **Beskrivning** Skriv en beskrivning för ditt konto. *Detta kommer att synas på din WinkLinks-sida och när leverantörer vill koppla upp sig direkt med dig.*
5. **Webbplats** Ange din huvudsakliga webbplats. *Det kan vara din IG-kontos URL eller företagswebbplats. Det bör vara platsen där du gör mest av ditt onlinearbete. Leverantörer kommer att gå till dessa länkar när de avgör om de ska koppla upp sig med dig.*
6. **Stad** Var är du baserad?
</Steps>

### Betalningsvillkor

:::note
Om du väljer att vara ansvarig för betalning kommer du endast att ha tillgång till fastigheter som du har en direkt koppling till.
:::

:::caution
Wink tar inget ansvar för att betala hotell i detta scenario.
:::

:::note
Du kan endast interagera med detta alternativ under kontoskapandet. De flesta reseagenter bör lämna denna inställning avstängd 🛑 och använda denna portal istället.
:::

Registrerade reseagenter som integrerar med oss via vårt API har möjlighet att ta emot betalning och vidarebefordra dessa medel till våra fastigheter.

<Steps>
1. **Jag vill ta emot betalning** Resagenten tar ansvar för att samla in medel.
2. **Jag vill betala hotell** Resagenten ansvarar för att betala hotellen.
</Steps>

Hotell som vill koppla upp sig direkt med dig kommer att se en varning och måste klicka för att bekräfta att Wink inte ansvarar för att betala hotellen, för att etablera en relation med ditt konto.

### Avtal

<Steps>
1. **Avtal** Acceptera vårt [affiliateavtal](/sv/portal/travel-agent/terms-of-service) och [betalningsvillkor](/sv/portal/travel-agent/payment-terms) för att fortsätta.
2. Klicka på knappen `Save`.
</Steps>

Grattis 🎉 ...du har skapat ditt första affiliatekonto på Wink.

När kontot är skapat kan du anpassa det ytterligare genom att fylla i din adress och lägga till en profilbild på ditt konto.

<Steps>
1. Klicka på ditt nya reseagentkonto. *Detta tar dig till din kontopanel.*
2. Klicka på `Account > Profile` i huvudnavigeringsfältet.
3. På profilsidan kan du uppdatera din befintliga profil och lägga till ytterligare information.
</Steps>

:::note
Din reseagentkontos profilbild ska inte förväxlas med din Wink-användarprofilbild. Din reseagentkontos profilbild visas på din WinkLinks-sida och ses av leverantörer som vill koppla upp sig med dig.

Din användarprofilbild är mest för internt bruk.
:::

## Chefer

Du kan bjuda in andra användare att hjälpa till att hantera ditt reseagentkonto. När en användare accepterar din inbjudan får de full tillgång till ditt konto med undantag för betalningar. Du kommer alltid vara den enda som ansvarar för de medel du äger.

<Steps>
1. Välj det reseagentkonto du vill arbeta med från [kontosidan](https://agent.wink.travel/company/list).
2. Klicka på `Account > Managers` i huvudnavigeringsfältet.
3. Lägg till e-postadressen till den användare du vill göra till kontochef.
4. Klicka på länken `Invite` bredvid e-postadressen.
</Steps>

Wink skickar ett e-postmeddelande till användaren för att acceptera eller avvisa din inbjudan.

## Notiser

Ibland vill vi kontakta dig. När det händer kommer du att se att klockikonen uppe till höger ändras.
För att läsa vad vi skickat:

<Steps>
1. Klicka på `Account > Notifications` i huvudnavigeringsfältet.
2. Du kommer att se nya och tidigare notiser.
3. Välj att `Mark as read` eller `Delete` notiserna när du har läst dem.
</Steps>

## API

Reseagenter räknas fortfarande som affiliates. Utvecklare som vill hantera `Affiliates` kan gå till [Developers > API > Affiliate](/sv/api).
