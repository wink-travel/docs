Prenumerationssidan låter dig hantera din plan och de fastigheter som är licensierade under ditt konto.

## Tillgängliga planer

Bläddra bland tillgängliga prenumerationsnivåer som visas via vår prissättningstabell. Varje plan inkluderar olika funktioner och fastighetsgränser. Välj den plan som bäst passar dina behov.

## Aktuell prenumeration

Om du redan har en aktiv prenumeration visar denna sektion detaljer och status för din nuvarande plan.

## Licenser

Licenstabellen visar alla fastigheter som tilldelats under din prenumeration. Du kan:

- Se vilka fastigheter som för närvarande är licensierade.
- Lägga till nya fastighetslicenser med knappen `Add license`.
- Hantera fakturering via länken till Stripe kundportal.

:::note
Faktureringshantering (betalningsmetoder, fakturor, planändringar) sker via Stripe kundportal.
:::
