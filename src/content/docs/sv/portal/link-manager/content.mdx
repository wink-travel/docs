Ditt innehåll finns tillgängligt under fliken `Entries`.

Vi stödjer tre huvudtyper av innehåll:

- Offentligt tillgängliga URL:er som länkar till webbsidor.
- Filer som laddats upp via WinkLinks.
- Textinnehåll skapat genom vår funktion `Quotes`.

## Lägg till innehåll

Så här lägger du till innehåll i WinkLinks. Under fliken `Entries`, börja med att klicka på knappen `Add entry`.

### Vanlig länk

:::tip
Länken måste gå till en webbsida. Du kan till exempel inte länka direkt till en PDF. Om du vill dela en PDF måste du använda funktionen `Upload file`.
:::

Här är stegen för att lägga till en vanlig länk:

<Steps>
1. Ange länken du vill dela. *t.ex. https://trvl.as/3xWCH*
2. När du har angett länken söker Wink efter detaljer om denna webbplats och fyller i formuläret du ser härnäst.
3. Du kan välja att ändra vissa av uppgifterna, såsom webbplatsens namn, titel och beskrivning.
4. Du kan också välja att ladda upp en egen bild.
5. Klicka på knappen `Save` när du är redo att fortsätta.
</Steps>

### Ladda upp fil

:::caution[Krav]
- Bilder får inte överstiga 10 Mb i storlek.
- Videor får inte överstiga 50 Mb i storlek.
- Vi stödjer alla erkända bild- och videoformat, inklusive PDF.
:::

:::note[Rekommendationer]
- 2560w x 1440h (16x9 bildförhållande) för din utvalda bild.
- 2560w x 1920h (4:3 bildförhållande) för alla andra bilder.
- Bildstorleken bör inte vara större än 5 Mb. Komprimera dina bilder innan du laddar upp dem till Wink.
:::

Här är stegen för att lägga till en fil:

<Steps>
1. Media kan läggas till:
    - Genom att dra bilder från din dator till fönstret eller klicka på `Browse`.
    - Genom att klistra in en extern URL som pekar på en bild.
    - Genom att använda din laptopkamera.
    - Från ditt Google Drive-konto.
    - Från ditt Dropbox-konto.
    - Från Shutterstock
    - Från gettyimages
    - Från iStock
    - Från Unsplash
2. När filen har laddats upp kan du lägga till mer metadata för att beskriva din fil, såsom titel och beskrivning.
3. Klicka på knappen `Save` när du är redo att fortsätta.
</Steps>

### Skriv ett citat

:::tip
Ett citat är ett utmärkt sätt att lägga till en avgränsare för att bryta upp en lång lista [...och det får dig att låta smart och kunnig också 😉].
:::

Här är stegen för att skriva ett citat:

<Steps>
1. Skriv citatet *t.ex. Rosor är röda. Viol är blå...*.
2. Ange namnet på den ursprungliga författaren. *t.ex. Edmund Spenser*
3. Ange namnet på webbplatsen där du hämtade citatet. *t.ex. Daily Quotes*
4. Klicka på knappen `Save` när du är redo att fortsätta.
</Steps>

## Uppdatera innehåll

Allt innehåll kan enkelt ändras genom att klicka på länken `Actions` och välja `Update`. Du kan ta bort innehåll på liknande sätt genom att klicka på knappen `Remove`.

## Ta bort innehåll

Allt innehåll kan enkelt ändras genom att välja `Update`. Du kan ta bort innehåll på liknande sätt genom att först klicka på länken `Actions` och sedan klicka på knappen `Remove`.

## Sortera innehåll

Du kan ordna ditt innehåll genom att klicka på ikonen som finns i nedre vänstra hörnet av varje post och dra den till den position du vill ha den i.

## Gruppera innehåll

:::tip
Skapa dina taggar innan du skapar dina poster. Annars förlorar du ditt arbete och måste börja om.
:::

När du redigerar en post har du möjlighet att lägga till en eller flera taggar för gruppering. Scrolla till botten av formuläret för att hitta fältet `Tags`.

Användare som tittar på din WinkLinks-sida kan välja hur de vill konsumera ditt innehåll (antingen lista eller rutnät) och kan välja en eller flera taggar för att ytterligare filtrera ditt innehåll.

## API

Utvecklare som vill hantera `WinkLinks` kan gå till [Developers > API > WinkLinks](/sv/api).
