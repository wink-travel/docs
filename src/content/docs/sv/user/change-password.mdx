Om du vill byta lösenord för ditt Wink-användarkonto, följ dessa steg:

## Uppdatera lösenord

<Steps>
1. Du måste vara inloggad på någon av våra webbplatser.
2. Klicka på din profilikon uppe till höger så visas en meny.
3. Klicka på länken `Byt lösenord`.
4. Följ instruktionerna för att byta ditt lösenord.
5. Klicka på `Spara`.
</Steps>

Nästa gång du loggar in behöver du ditt nya lösenord.

## Glömt lösenord

Ibland är det nödvändigt att återställa användarlösenordet.

Här är stegen för att återställa ditt användarlösenord:

<Steps>
1. Klicka på länken `Glömt lösenord` på inloggningssidan.
2. Skriv in användarnamnet / e-postadressen för det konto du vill återställa.
3. Klicka på knappen `Glömt lösenord`. *Du omdirigeras till formuläret för återställning av lösenord.*
4. Kontrollera din e-post.
</Steps>

<Steps>
1. Klistra in koden du fick i e-posten i det nya formuläret tillsammans med ditt nya lösenord.
2. Bekräfta ditt nya lösenord.
3. Klicka på knappen `Återställ lösenord`. *Du omdirigeras tillbaka till inloggningssidan.*
</Steps>
