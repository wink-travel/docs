En Wink-användare kan välja att låta dig hantera ett av sina boende-, affiliate- eller resebyråkonton. När det händer får du ett e-postmeddelande med en inbjudningsförfrågan.

import { Steps } from '@astrojs/starlight/components';

För att se dina nuvarande Wink-inbjudningar, följ dessa steg:

<Steps>
  1. Du måste vara inloggad på en av våra webbplatser.
  2. Klicka på din profilikon i det övre högra hörnet så visas en meny.
  3. Klicka på `Invites` länk.
  4. Interagera med dina inbjudningar genom att antingen `Accepting` eller `Rejecting` dem.
</Steps>
