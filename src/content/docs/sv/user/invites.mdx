En Wink-användare kan välja att låta dig hantera ett av deras fastighets-, affiliate- eller resebyråkonton. När det händer får du ett e-postmeddelande med en inbjudningsförfrågan.

import { Steps } from '@astrojs/starlight/components';

För att se dina aktuella Wink-inbjudningar, följ dessa steg:

<Steps>
1. Du måste vara inloggad på någon av våra webbplatser.
2. Klicka på din profilikon uppe till höger så visas en meny.
3. Klicka på länken `Invites`.
4. Hantera dina inbjudningar genom att antingen `Acceptera` eller `Avvisa` dem.
</Steps>
