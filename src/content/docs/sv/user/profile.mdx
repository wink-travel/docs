Om du vill uppdatera din användarprofil för ditt Wink-användarkonto, följ dessa steg:

<Steps>
1. Du måste vara inloggad på någon av våra webbplatser.
2. Klicka på din profilikon uppe till höger så visas en meny.
3. Klicka på länken `Profil`.
4. Följ instruktionerna för att uppdatera din `e-post`, `förnamn` och `efternamn`.
5. Du kan också ladda upp en ny `profilbild` på dig själv.
6. Klicka på `Spara`.
</Steps>

:::note
Din nya profilbild visas nästa gång du loggar ut helt från Wink.
:::
