Du kan skapa ett användarkonto via vilken som helst av våra portaler. När kontot väl är skapat kan det användas i alla Winks appar och webbplatser.

## Resenärer

Följ dessa steg:

<Steps>
  1. Gå vidare till [https://ota.wink.travel](https://ota.wink.travel)
  2. Klicka på `Sign-In / Register` i det övre högra hörnet
  3. Du kommer att bli ombedd att ange grundläggande användaruppgifter. Fyll i alla obligatoriska fält och klicka på `Save`.
  4. Du kan nu logga in på Wink med ditt användarnamn (e-postadress) och lösenord.
  5. När du är inloggad omdirigerar systemet dig tillbaka till var du kom ifrån.
</Steps>

## Registrering

Att skapa ett Wink-användarkonto ger dig tillgång till alla funktioner på plattformen. Du kan skapa ett obegränsat antal fastigheter och affiliate-konton.

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="Logga in" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Inloggningsformulär*</figcaption>
</figure>


<Steps>
  1. Första gången du dirigeras till vår auktoriseringsserver kommer du att bli ombedd att ange ditt användarnamn och lösenord.
  2. Klicka istället på `Create account` knapp.
  3. Fyll i alla obligatoriska fält och klicka på `Register` för att fortsätta. Detta tar dig tillbaka till inloggningsformuläret.
  4. Ange din e-postadress/användarnamn och lösenord för att logga in.
  5. Du omdirigeras tillbaka till webbplatsen du kom ifrån.
</Steps>
