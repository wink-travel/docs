Du kan skapa ett användarkonto via någon av våra portaler. När kontot är skapat kan det användas i alla Wink:s appar och webbplatser.

## Resenärer

Följ dessa steg:

<Steps>
1. Gå till [https://ota.wink.travel](https://ota.wink.travel)
2. Klicka på `Sign-In / Register` uppe i högra hörnet
3. Du kommer att bli ombedd att ange grundläggande användaruppgifter. Fyll i alla obligatoriska fält och klicka på `Save`.
4. Du kan nu logga in på Wink med ditt användarnamn (e-post) och lösenord.
5. När du är inloggad omdirigerar systemet dig tillbaka till där du kom ifrån.
</Steps>

## Registrering

Att skapa ett användarkonto på Wink ger dig tillgång till alla funktioner på plattformen. Du kan skapa obegränsat med fastigheter och affiliate-konton.

<figure>

    />
    <figcaption>*Inloggningsformulär*</figcaption>
</figure>

<Steps>
1. Första gången du dirigeras till vår auktoriseringsserver kommer du att bli ombedd att ange ditt användarnamn och lösenord.
2. Klicka istället på knappen `Create account`.
3. Fyll i alla obligatoriska fält och klicka på `Register` för att fortsätta. Detta tar dig tillbaka till inloggningsformuläret.
4. Ange din e-post / användarnamn och lösenord för att logga in.
5. Du omdirigeras tillbaka till sidan du kom ifrån.
</Steps>
