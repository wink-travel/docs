För att hantera dina bankkonton, klicka på `Konto > Bankkonton` i sidomenyn.

## Lägga till ett bankkonto

Klicka på `Lägg till bankkonto` för att registrera ett nytt konto. Formuläret anpassas efter det land du väljer:

- **Betalningsland** — Landet där din bank är belägen.
- **Valuta** — Valutan du vill ta emot utbetalningar i.
- **Kontoinnehavarens namn** — Namnet på bankkontot.
- **Routingdetaljer** — Landspecifika fält som Routingnummer (US), Sortkod (UK), SWIFT/BIC-kod eller IBAN.
- **Kontonummer** — Ditt bankkontonummer.

## Primärt konto

Du kan ange ett bankkonto som **Primärt**. Ditt primära konto tar emot alla automatiska utbetalningar. Ytterligare konton kan användas för manuella uttag.

För att ändra ditt primära konto, klicka på `Gör till primärt` på något konto i din lista.

## Hantera konton

Från listan över bankkonton kan du:

- Redigera kontouppgifter.
- Ta bort ett konto du inte längre behöver.
- Byta vilket konto som är primärt.
