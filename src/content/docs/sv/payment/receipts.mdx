Varje bokning på TripPay genererar ett kvitto som du kan komma åt från bokningsdetaljsidan.

## Vad som finns på ett kvitto

Ett kvitto innehåller:

- **Betalningsuppgifter** — Betalningsmetod, transaktions-ID:n och tidsstämplar.
- **Beloppsfördelning** — Rumsavgifter, tilläggstjänster och totalbelopp.
- **Avbokningspolicy** — Policyn som gällde vid bokningstillfället.
- **Återbetalningsinformation** — Eventuella återbetalningar som genomförts, med belopp och datum.

## Kvittoformat

TripPay erbjuder två kvittoformat:

- **Gästvänligt TripPay-kvitto** — Ett rent, varumärkesanpassat kvitto som är lämpligt att dela med gästen.
- **Betalningsgateway-kvitto** — Den fullständiga transaktionsrapporten från betalningsprocessorn.

Båda är tillgängliga via länkar på bokningsdetaljsidan.
