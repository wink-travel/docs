Om du är ett reseföretag och vill outsourca betalnings- och utbetalningsdelen av din verksamhet till TripPay, här är 6 grundläggande steg för att komma igång:

<Steps>
1. Skapa ett [användarkonto](/sv/user/register) på Wink.
2. Skapa ett konto hos [TripPay](https://pay.wink.travel).
3. Skapa en [Applikation](/sv/developers/applications) på TripPay som låter dig kommunicera med vårt API.
4. Skapa och koppla dina mottagare med TripPay. *t.ex. För de flesta innebär det att skapa konton för de boenden på din plattform som du vill att TripPay ska betala till.*
5. Integrera [betalningswebbkomponenten](/sv/developers/web-components/#payment) på din webbplats.
6. Skicka TripPay resenärens `shopping cart` och initiera betalningsprocessen.
</Steps>

:::tip
För en komplett guide till att använda TripPay för din resewebbplats, läs guiden [Integrating with TripPay](/sv/guides/integrating-with-trip-pay).
:::

## API

Utvecklare som vill hantera `Payments`, gå till [Developers > API > Payments](/sv/developers/apis/#payment-api).
