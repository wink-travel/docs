# Anslutningar

Du är redan ansluten till WinkLinks, vårt verktyg för länkhantering. Anslut till fler av dina sociala nätverk genom att klicka på `Anslut ett socialt nätverk`.
Processen kommer att omdirigera dig till ditt sociala nätverk för att godkänna `Wink Social`-appen att posta innehåll åt dig.

När du är ansluten börjar vi generera innehåll för det nätverket för nästa schemalagda månad.

:::note
Vi skapar och publicerar innehåll optimerat för varje socialt nätverk.
:::
