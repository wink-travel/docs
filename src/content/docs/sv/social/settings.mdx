# Inställningar

När du först börjar använda Social kommer vi att fråga hur du vill att vi ska skapa ditt innehåll. Dessa alternativ är tillgängliga för dig hela tiden så att du enkelt kan ändra dem senare.

Navigera till `Preferences` i sidomenyn för att hantera dina inställningar.

Avsnittet Preferences innehåller dina grundläggande inställningar för innehållsgenerering:

1. **Preferred customization** Välj vilken anpassning som ska användas vid generering av innehåll. Anpassningar styr temafärger, standardresplan, valuta, språk och mer. Du skapar anpassningar under ditt konto på [my.wink.travel](https://my.wink.travel).
2. **Auto-approve** När detta är aktiverat publiceras genererade inlägg automatiskt utan att kräva din godkännande. Aktivera detta när du är säker på kvaliteten på det innehåll vi producerar för dig.
3. **Approver name** Namnet på personen eller teamet som kommer att få e-post med godkännandeförfrågningar.
4. **Approver e-mail** E-postadressen dit godkännandeförfrågningar för kommande inlägg skickas.

## Content creators

<figure>
<CldImage
  src="academy/social_content_creators"
  width="700"
  height="355"
  alt="Social content creators"
/>
  <figcaption>*Fig. 2 - Content creators*</figcaption>

Social har många content creators att välja mellan. Varje creator fokuserar på ett specifikt sätt att skapa unikt innehåll för din anläggning med hjälp av AI. Varje content creator har en **Weight**-attribut — en högre vikt ökar chansen att den creatorn väljs vid generering av inlägg.

Här är alla tillgängliga content creators:

| Creator | Beskrivning |
| -- | -- |
| **Destination** | Använder din plats för att skapa hjälpsam text om området |
| **Room type** | Skapar ett bokningsbart inlägg med en av dina rumstyper med AI-genererad text och bilder på rumstypen |
| **Activity** | Skapar ett bokningsbart inlägg med en av dina aktiviteter |
| **Add-On** | Skapar ett bokningsbart inlägg med ett av dina tillägg |
| **Attraction** | Skapar ett bokningsbart inlägg med en av dina attraktioner |
| **Place** | Skapar ett bokningsbart inlägg med en av dina platser |
| **Meeting room** | Skapar ett bokningsbart inlägg med ett av dina mötesrum |
| **Restaurant** | Skapar ett bokningsbart inlägg med en av dina restauranger |
| **Spa** | Skapar ett bokningsbart inlägg med ett av dina spa |
| **Property** | Skapar ett bokningsbart inlägg med ditt bäst prissatta rum med AI-genererad text och bilder på anläggningen |
| **User sentiment** | Sammanställer gästrecensioner från hela webben till ett känslodrivet inlägg som lyfter fram vad gäster älskar med din anläggning |
| **Funny** | Skapar lättsamma, underhållande inlägg för att hålla ditt flöde engagerande |

:::tip[Pro tip]
Se till att du har den inventarietyp som krävs för dina valda content creators. Om du till exempel väljer "Restaurant" behöver du ha minst en restaurang listad på din anläggning.
:::

## Tone of voice

<figure>
<CldImage
  src="academy/social_personas"

  height="396"

  <figcaption>*Fig. 3 - Tone of voice*</figcaption>

Tone of voice bestämmer språkstilen AI använder när innehåll genereras. Välj de toner som matchar din varumärkespersonlighet. Precis som med content creators har varje ton en **Weight** för att styra hur ofta den används.

Här är de tillgängliga tonerna:

| Tone | Stil |
| -- | -- |
| **Neutral** | Lugnt, auktoritativt, faktabaserat. Ingen hype, inga skämt — bara pålitlig, välstrukturerad kunskap |
| **Friendly** | Varmt, tillgängligt, med en leende-igenom-text-energi med avslappnade sammandragningar och ibland emojis |
| **Luxury** | Sofistikerat, förfinat. Ord som "exquisite" och "bespoke" som får läsaren att känna sig exklusiv |
| **Foodie** | Sinnliga, aptitretande beskrivningar. Matporrspråk som gör läsaren hungrig |
| **Adventurous** | Djärvt, orädd, upptäckarenergi. Levande sinnesintryck som får allt att låta episkt |
| **Family** | Hälsosamt, barnvänligt engagemang. Varm föräldraenergi om säkra och roliga upplevelser |
| **Romantic** | Poetiskt, längtansfullt, hjärtfladdrande. Solnedgångar, mjukt språk och kärlek |
| **Humorous** | Sarkastiskt, snabba one-liners, smart ordlek. Låter som en stand-up-show |
| **Eco** | Ärlig och rättfärdig. Hyllar gröna alternativ och hållbara val |
| **Local insider** | Mycket specifika lokala referenser, varnar för turistfällor, använder lokal slang |
| **Inclusive** | Välkomnande, firande och bekräftande. Får alla att känna sig sedda och värderade |
| **Mystical** | Eteriskt, andligt, kosmiska sanningar. Mjuka inbjudningar att se inåt |
| **Futuristic** | Snyggt, framåtblickande. Älskar termer som "paradigm shift" och optimistiska framtidsvisioner |
| **Budget** | Slitstarkt, resursstarkt, stolt sparsam. Hyllar prisvärda fynd och gratisupplevelser |

## Post types

Posttyper bestämmer målet och strukturen för det genererade innehållet. Varje posttyp har en **Weight** för att styra hur ofta den används.

| Post type | Beskrivning |
| -- | -- |
| **Transactional** | Klart, koncist, konverteringsfokuserat. Driver bokningar och direkta åtgärder med fakta, priser och nästa steg |
| **Awareness** | Varumärkesbyggande innehåll utformat för att introducera din anläggning och öka synligheten bland potentiella gäster |
| **Sentiment** | Känslomässigt resonant innehåll som skapar positiva associationer och bygger en emotionell koppling med din publik |

Du kanske vill rikta dina marknadsföringsinsatser mot ett särskilt språk och en region. Berätta för oss vilka språk du vill skapa innehåll för och vilken tidszon du är intresserad av så sköter vi resten. Inlägg publiceras vid schemalagd tid i din valda tidszon.

*De flesta anläggningar använder ett enda språk och standardiserar på sitt lokala språk.*

Genom att koppla ditt Google Business Profile kan vi hämta rikare anläggningsdetaljer för mer effektivt innehåll. Din anläggning behöver en verifierad Google Business Profile för att vi ska kunna generera innehåll från din lista. Vissa av våra content creators **kräver** att du har ett Google Business-konto.

:::tip[Pro tip]
Börja smått

Var inte för ambitiös vid första försöket. Vi rekommenderar att du börjar med att skapa innehåll för din anläggning på ett enda språk.
:::
