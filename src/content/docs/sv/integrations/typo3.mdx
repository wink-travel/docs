**Wink TYPO3 Extension** integrerar Wink reseinventarie i TYPO3 13 som inbyggda innehållselement. Redaktörer lägger till Wink-komponenter på sidor med den vanliga TYPO3 sidredigeraren — inga malländringar krävs.

<LinkCard
  title="wink-travel/wink-typo3-extension på GitHub"
  href="https://github.com/wink-travel/wink-typo3-extension"

/>

## Krav

- TYPO3 13.x
- PHP 8.2+
- Composer

## Installera

<Steps>
1. Installera via Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivera tillägget:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Gå i TYPO3-backend till **Admin Tools → Settings → Extension Configuration → wink**.
4. Ange din **Client ID**, **Client Secret** och välj din miljö. Spara.
   Gå till [Applications](/sv/portal/settings/applications) för att hämta dina uppgifter.
5. Lägg till den statiska TypoScript-mallen i din rotmall:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Lägg till ett innehållselement på en sida

<Steps>
1. Öppna en sida i **Page**-modulen.
2. Klicka på **+**-knappen i innehållsområdet för att lägga till ett nytt element.
3. Välj ett av **Wink Travel**-innehållselementen:
   - **Wink Content** — hotellkort, rankade rutnät, kartor
   - **Wink Lookup** — sökfält för destinationer
   - **Wink Search** — knapp för att välja resplan
   - **Wink Account** — inloggningsknapp
   - **Wink Itinerary** — knapp för resplanstatus
   - **Wink Cart** — knapp för kundvagn
4. För **Wink Content**, använd FlexForm-fälten för att välja en **Layout Type** och ange **Layout ID**.
5. Spara och publicera sidan.
</Steps>

## Tillgängliga innehållselement

| Element | Syfte |
|---|---|
| **Wink Content** | Hotellkort, rankade rutnät, kartor med mera |
| **Wink Lookup** | Sökfält för destinationer |
| **Wink Search** | Ikonknapp som öppnar resplansväljaren |
| **Wink Account** | Inloggnings- / användarkontoknapp |
| **Wink Itinerary** | Resplanskknapp som visar aktuell resa |
| **Wink Cart** | Sammanfattningsknapp för kundvagn |
