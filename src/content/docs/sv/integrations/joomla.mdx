**Wink Joomla Extension** är ett paket som installerar ett systemplugin (laddar CDN-skript), ett innehållsplugin (bearbetar shortcode-taggar i artiklar) och en komponent (admininställningssida).

<LinkCard
  title="wink-travel/wink-joomla-extension on GitHub"
  href="https://github.com/wink-travel/wink-joomla-extension"
  description="Källkod, utvecklaruppsättningsguide och instruktioner för bidrag."
/>

## Krav

- Joomla 4 eller 5
- PHP 8.1+

## Installera

<Steps>
1. Ladda ner senaste `pkg_wink-x.x.x.zip` från [GitHub releases-sidan](https://github.com/wink-travel/wink-joomla-extension/releases).
2. I Joomla Admin, gå till **Extensions → Manage → Install**.
3. Ladda upp zip-filen och klicka på **Upload & Install**.
4. Båda plugins aktiveras automatiskt. Gå till **Components → Wink Settings**.
5. Ange din **Client ID** och **Client Secret** och klicka på **Save**.
   Gå till [Applications](/sv/portal/settings/applications) för att hämta dina uppgifter.
</Steps>

## Bädda in komponenter i artiklar

Använd shortcode-liknande taggar var som helst i artikelinnehållet:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Innehållspluginet ersätter dessa taggar med korrekt `<wink-*>` HTML vid renderingen. Systempluginet laddar automatiskt CDN-skripten och injicerar `<wink-app-loader>` i sidfoten.

## Tillgängliga shortcodes

| Shortcode | Output | Beskrivning |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Hotellkort, rutnät, kartor |
| `{wink-lookup}` | `<wink-lookup>` | Sökfält för destination |
| `{wink-search}` | `<wink-search-button>` | Knapp för att välja resplan |
| `{wink-account}` | `<wink-account-button>` | Inloggning / användarkonto |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Knapp för resplanstatus |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Kundvagnsknapp |
