**Wink Wix App** lägger till Wink webbkomponenter på din Wix-sida som inbyggda widgets som du kan dra och släppa i Wix Editor. Ingen kodning krävs för webbplatsägare.

## Installera

<Steps>
1. Gå till [Wix App Market](https://www.wix.com/app-market) och sök efter **Wink Travel**.
2. Klicka på **Lägg till på webbplats** och följ anvisningarna för att auktorisera appen.
3. När den är installerad, öppna **Dashboard → Wink Settings** och ange din **Client ID** och **Client Secret**.
   Gå till [Applications](/sv/portal/settings/applications) för att hämta dina uppgifter.
4. Klicka på **Spara** och sedan **Testa anslutning** för att bekräfta att Wink är anslutet.
</Steps>

## Lägg till en komponent på en sida

<Steps>
1. Öppna **Wix Editor** och gå till sidan där du vill lägga till Wink-innehåll.
2. Klicka på **Lägg till (+) → Appar → Wink Travel** för att se alla tillgängliga widgets.
3. Dra en widget till sidan — till exempel **Wink Content** för att visa ett hotellgalleri.
4. I widgetens inställningspanel, välj **Layouttyp** och klistra in **Layout ID** från Wink Studio.
5. Klicka på **Publicera** för att göra ändringarna live.
</Steps>

## Tillgängliga widgets

| Widget | Syfte |
|---|---|
| **Wink Content** | Visar hotellkort, rankade gallerier, kartor med mera |
| **Wink Lookup** | Sökruta för destinationer och hotell |
| **Wink Search** | Ikonknapp som öppnar resplanväljaren |
| **Wink Account** | Inloggnings- / användarkonto-knapp |
| **Wink Itinerary** | Resplan-knapp som visar aktuell resa |
| **Wink Cart** | Sammanfattningsknapp för kundvagn |
