**Wink Shopify App** låter researrangörer, upplevelseförsäljare och hotellhandlare visa live Wink-inventarium direkt på sin Shopify-butiksida — tillsammans med produkter, på landningssidor eller var som helst i temat.

## Installera

<Steps>
1. Gå till [Shopify App Store](https://apps.shopify.com) och sök efter **Wink Travel**.
2. Klicka på **Add app** och följ OAuth-flödet för att auktorisera.
3. I Wink-appens inställningar, ange din **Client ID** och **Client Secret**.
   Gå till [Applications](/sv/portal/settings/applications) för att hämta dina uppgifter.
4. Klicka på **Verify & Save**. Ett framgångsmeddelande bekräftar anslutningen.
</Steps>

## Aktivera App Embed Block

App Embed Block injicerar automatiskt Wink-skriptet och stilmallen på varje sida — du behöver inte redigera temakoden.

<Steps>
1. I Shopify Admin, gå till **Online Store → Themes → Customize**.
2. Klicka på **App embeds** i vänstra panelen.
3. Slå på **Wink Travel**.
4. Klicka på **Save**.
</Steps>

:::note
App Embed Block läser din Client ID från ett Shopify-metafält som sätts vid installation. Din Client Secret exponeras aldrig i webbläsaren.
:::

## Lägg till en komponent på en sida

<Steps>
1. I temaredigeraren, öppna sidan eller sektionen där du vill ha Wink-innehåll.
2. Klicka på **Add block** och välj bland Wink-blocken:
   - **Wink Content** — hotellkort, rutnät eller kartor
   - **Wink Lookup** — sökfält för destinationer
   - **Wink Search** — knapp för att välja resplan
   - **Wink Account** — inloggningsknapp
   - **Wink Itinerary** — knapp för resplanstatus
   - **Wink Cart** — knapp för kundvagn
3. För **Wink Content**, välj en **Layout Type** och ange **Layout ID** från Wink Studio.
4. Klicka på **Save**.
</Steps>

## Tillgängliga temablock

| Block | Syfte |
|---|---|
| **Wink Content** | Visar hotellkort, rankade rutnät, kartor med mera |
| **Wink Lookup** | Sökfält för destinationer och hotell |
| **Wink Search** | Ikonknapp som öppnar resplansväljaren |
| **Wink Account** | Inloggnings- / användarkontoknapp |
| **Wink Itinerary** | Resplanskknapp som visar aktuell resa |
| **Wink Cart** | Sammanfattningsknapp för kundvagn |
