**Wink Drupal Module** lägger till Wink reseinventariekomponenter på din Drupal-sida som inbyggda block. Redaktörer kan placera dem var som helst i layouten via det vanliga blocksystemet.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Källkod, utvecklaruppsättningsguide och instruktioner för bidrag."
/>

## Krav

- Drupal 10 eller 11
- PHP 8.1+
- Composer

## Installera

<Steps>
1. Installera via Composer:
   ```bash
   composer require drupal/wink
   ```
2. Aktivera modulen:
   ```bash
   drush en wink -y
   ```
3. Gå till **Admin → Konfiguration → Webbtjänster → Wink-inställningar** (`/admin/config/wink`).
4. Ange din **Client ID** och **Client Secret**, välj din miljö och klicka på **Spara**.
   Gå till [Applications](/sv/portal/settings/applications) för att hämta dina uppgifter.
</Steps>

## Lägg till ett block på en sida

<Steps>
1. Gå till **Admin → Struktur → Blocklayout**.
2. Klicka på **Placera block** i den region där du vill ha Wink-innehåll.
3. Sök efter **Wink** för att hitta tillgängliga block:
   - **Wink Content** — visar ett hotellkort, rutnät eller karta
   - **Wink Lookup** — sökfält för destinationer
   - **Wink Search** — knapp för att välja resplan
   - **Wink Account** — inloggningsknapp
   - **Wink Itinerary** — knapp för resplanstatus
   - **Wink Cart** — knapp för kundvagn
4. För **Wink Content**, konfigurera **Layouttyp** och **Layout-ID** i blockinställningarna.
5. Spara blockkonfigurationen.
</Steps>

## Tillgängliga block

| Block | Syfte |
|---|---|
| **Wink Content** | Visar hotellkort, rankade rutnät, kartor med mera |
| **Wink Lookup** | Sökfält för destinationer och hotell |
| **Wink Search** | Ikonknapp som öppnar resplansväljaren |
| **Wink Account** | Inloggnings- / användarkontoknapp |
| **Wink Itinerary** | Resplanskknapp som visar aktuell resa |
| **Wink Cart** | Sammanfattningsknapp för kundvagn |
