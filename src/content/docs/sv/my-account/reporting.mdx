Wink erbjuder ett rapporteringssystem som låter dig generera och exportera data från ditt konto. Rapporter finns tillgängliga i CSV-format för enkel import till kalkylblad, analysverktyg eller dina egna system.

## Rapporteringsformat

Du kan generera rapporter för följande datakategorier:

| Rapporttyp | Vad den innehåller |
|---|---|
| **Fastighet** | Fastighetsdetaljer, liststatus, inventariesammanfattning. |
| **Användare** | Användarkonton, roller, aktivitetsöversikt. |
| **Bokning** | Bokningshistorik, statusar, intäkter, gästinformation. |
| **Konto** | Kontoinformation, saldon, transaktionshistorik. |

## Hur man genererar en rapport

1. Navigera till rapporteringssektionen i din kontopanel.
2. Välj den **rapporttyp** du vill generera.
3. Konfigurera eventuella **filter** för att begränsa datan (t.ex. datumintervall, status, fastighet).
4. Välj de **fält** du vill inkludera i exporten. Varje rapporttyp erbjuder ett urval av tillgängliga fält att välja mellan.
5. Klicka på **Exportera** för att ladda ner rapporten som en CSV-fil.

## Anpassade fält

Rapporter stödjer anpassad fältval. Du väljer vilka kolumner som ska ingå i din export, så att du får exakt den data du behöver — inte mer, inte mindre.

## Användningsområden

- **Intäktsavstämning** — Exportera bokningsrapporter med finansiella detaljer för att stämma av mot ditt redovisningssystem.
- **Prestandaanalys** — Exportera fastighetsrapporter för att jämföra listningsprestanda i din portfölj.
- **Gästinsikter** — Exportera bokningsdata för att förstå resenärers demografi och bokningsmönster.
- **Kontorevision** — Exportera kontodata för efterlevnad eller intern granskning.
