Om du vill byta ditt lösenord, eller om du har glömt ditt lösenord, följ dessa steg:

## Uppdatera lösenord

<Steps>
1. Gå till [my.wink.travel](https://my.wink.travel).
1. Logga in genom att klicka på `Sign-in / Register` uppe i högra hörnet.
2. Klicka på din profilikon, uppe i högra hörnet, så visas en meny.
3. Klicka på länken `Change password`.
4. Följ instruktionerna för att byta ditt lösenord.
5. Klicka på `Change password` för att fortsätta.
</Steps>

Nästa gång du loggar in behöver du ditt nya lösenord.

## Glömt lösenord

Ibland är det nödvändigt att återställa användarens lösenord.

Här är stegen för att återställa ditt lösenord:

<Steps>
1. Klicka på länken `Forgot password` från inloggningssidan.
2. Skriv in användarnamnet / e-postadressen för det konto du vill återställa.
3. Klicka på knappen `Forgot password`. *Du omdirigeras till formuläret för att återställa lösenordet.*
4. Kontrollera din e-post.
</Steps>

2. Bekräfta ditt nya lösenord.
3. Klicka på knappen `Reset password`. *Du omdirigeras tillbaka till inloggningssidan.*
</Steps>
