En Wink-användare kan välja att låta dig hantera ett av deras fastighets-, affiliate- eller resebyråkonton. När det händer får du ett e-postmeddelande med en inbjudningsförfrågan.

import { Steps } from '@astrojs/starlight/components';

För att se dina aktuella Wink-inbjudningar, följ dessa steg:

<Steps>
 1. Logga in på [my.wink.travel](https://my.wink.travel).
 2. Klicka på `Invites` i vänstra sidofältet.
 3. Du kommer att se en lista över alla aktuella inbjudningar.
 4. Den här sektionen listar alla dina ansökningar.
 5. Välj att `Accept` eller `Reject` inbjudan.
 6. Om du accepterade inbjudan visas kontot i sektionen `Accounts`.
</Steps>
