Ein Wink-Nutzer kann Ihnen erlauben, eines seiner Unterkunfts-, Partner- oder Reisebürokonten zu verwalten. In diesem Fall erhalten Sie eine E-Mail mit einer Einladung.

import { Steps } from '@astrojs/starlight/components';

Um Ihre aktuellen Wink-Einladungen zu sehen, folgen Sie diesen Schritten:

<Steps>
 1. Melden Sie sich bei [my.wink.travel](https://my.wink.travel) an.
 2. Klicken Sie in der linken Seitenleiste auf `Invites`.
 3. Sie sehen eine Liste aller aktuellen Einladungen.
 4. Dieser Bereich zeigt alle Ihre Anfragen an.
 5. Wählen Sie, ob Sie die Einladung `Akzeptieren` oder `Ablehnen` möchten.
 6. Wenn Sie die Einladung akzeptiert haben, wird das Konto im Bereich `Accounts` angezeigt.
</Steps>
