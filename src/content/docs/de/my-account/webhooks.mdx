Webhooks sind eine Möglichkeit für Sie, Echtzeit-Ereignisse von Wink für Ihr Konto zu empfangen und darauf zu reagieren.

<Steps>
 1. Wählen Sie das Konto aus, mit dem Sie arbeiten möchten, unter [Accounts](https://my.wink.travel).
 2. Klicken Sie unten auf der Kontokarte auf `Actions` und wählen Sie `Manage account`.
 3. Klicken Sie auf der folgenden Seite auf den Reiter `Webhooks`.
 4. In diesem Abschnitt werden alle Ihre Webhooks aufgelistet.
 5. Wählen Sie, ob Sie einen bestehenden bearbeiten oder einen neuen erstellen möchten.
 </Steps>

## Beispiel

<Steps>
1. Eine Buchung erfolgt auf Wink.
1. Das Wink-Ereignis `booking.created` wird ausgelöst.
2. Ihr System hört aktiv auf dieses Ereignis.
3. Ihr System verfolgt eigene Analysen und verwendet dieses Ereignis, um seine Zeitreihendatenbank zu aktualisieren.
</Steps>

## Webhook erstellen

:::caution
Sie müssen ein Partnerkonto bei Wink oder ein Konto bei TripPay erstellen, bevor Sie einen Webhook anlegen, da Sie Ihren Webhook an ein bestimmtes Konto binden.
:::

Hier sind die Schritte zum Erstellen eines Webhooks:

<Steps>
1. Klicken Sie auf der Webhooks-Seite auf die Schaltfläche `Create new webhook`.
2. ***Name*** Geben Sie Ihrem Webhook einen Namen. *z. B. Wink Stalker*
3. ***Enable*** Belassen Sie die Einstellung aktiviert.
4. **Entity name** Verknüpfen Sie Ihren Webhook mit einem Ihrer bestehenden Konten. *z. B. Cool Account*
5. **Events** Wählen Sie die Ereignisse aus, die Sie interessieren. *z. B. `booking.created`*
6. **Webhook URL** Geben Sie die URL ein, unter der Sie unsere Ereignisse empfangen möchten. *z. B. `https://my.system.com/webhook/wink`*
7. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
</Steps>

Sie werden zurück zu Ihrer Webhook-Liste geleitet und Ihr neuer Webhook ist einsatzbereit.

## Webhook aktualisieren

Sie müssen Ihre App aktualisieren, wenn Sie:

- Ihren Webhook deaktivieren möchten.
- Mehr oder weniger Ereignisse empfangen möchten.
- Die URL Ihres Webhooks ändern möchten.

So aktualisieren Sie einen bestehenden Webhook:

<Steps>
1. Klicken Sie auf den Link `Actions` für den Webhook, den Sie aktualisieren möchten.
2. Klicken Sie unter Actions auf die Schaltfläche `Update`.
3. Nehmen Sie Ihre Änderungen am Webhook vor.
4. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
</Steps>

Sie werden zurück zu Ihrer Webhook-Liste geleitet.

## Webhook entfernen

Wenn Sie den Webhook nicht mehr benötigen, können Sie ihn entfernen.

So entfernen Sie einen Webhook:

<Steps>
1. Klicken Sie auf den Link `Actions` für den Webhook, den Sie entfernen möchten.
2. Klicken Sie unter Actions auf die Schaltfläche `Remove`.
3. Klicken Sie auf die Schaltfläche `OK`, um die Löschung zu bestätigen.
</Steps>

Sie werden zurück zu Ihrer Webhook-Liste geleitet.
