Wenn Sie Ihr Benutzerprofil für Ihr Wink-Benutzerkonto aktualisieren möchten, befolgen Sie diese Schritte:

<Steps>
1. Gehen Sie zu [my.wink.travel](https://my.wink.travel).
2. Melden Sie sich an, indem Sie oben rechts auf `Anmelden / Registrieren` klicken.
3. Sobald Sie angemeldet sind, klicken Sie auf Ihr Profilbild oben rechts, und es erscheint ein Menü.
4. Klicken Sie auf den Link `Profil`.
5. Befolgen Sie die Anweisungen, um Ihre `E-Mail`, Ihren `Vor-` und `Nachnamen` zu aktualisieren.
6. Sie können auch ein neues `Profilbild` von sich hochladen.
7. Klicken Sie auf `Speichern`.
</Steps>

<Steps>
1. Sie müssen auf einer unserer Websites angemeldet sein.
2. Klicken Sie auf Ihr Profilbild oben rechts, und es erscheint ein Menü.
3. Klicken Sie auf den Link `Profil`.
4. Befolgen Sie die Anweisungen, um Ihre `E-Mail`, Ihren `Vor-` und `Nachnamen` zu aktualisieren.
5. Sie können auch ein neues `Profilbild` von sich hochladen.
6. Klicken Sie auf `Aktualisieren`.
</Steps>

:::note
Ihr neues Profilbild wird beim nächsten vollständigen Abmelden von Wink angezeigt.
:::
