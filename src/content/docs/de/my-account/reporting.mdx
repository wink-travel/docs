Wink bietet ein Berichtssystem, mit dem Sie Daten aus Ihrem Konto generieren und exportieren können. Berichte sind im CSV-Format verfügbar, um sie einfach in Tabellenkalkulationen, Analysetools oder Ihre eigenen Systeme zu importieren.

## Berichtstypen

Sie können Berichte für die folgenden Datenkategorien erstellen:

| Berichtstyp | Was enthalten ist |
|---|---|
| **Objekt** | Objektdetails, Listenstatus, Inventarübersicht. |
| **Benutzer** | Benutzerkonten, Rollen, Aktivitätsübersicht. |
| **Buchung** | Buchungshistorie, Status, Umsatz, Gästedetails. |
| **Konto** | Kontoinformationen, Salden, Transaktionshistorie. |

## So erstellen Sie einen Bericht

1. Navigieren Sie zum Berichtsbereich in Ihrem Kontodashboard.
2. Wählen Sie den **Berichtstyp** aus, den Sie erstellen möchten.
3. Konfigurieren Sie beliebige **Filter**, um die Daten einzuschränken (z. B. Datumsbereich, Status, Objekt).
4. Wählen Sie die **Felder** aus, die im Export enthalten sein sollen. Jeder Berichtstyp bietet eine Auswahl verfügbarer Felder.
5. Klicken Sie auf **Exportieren**, um den Bericht als CSV-Datei herunterzuladen.

## Benutzerdefinierte Felder

Berichte unterstützen eine anpassbare Feldauswahl. Sie wählen aus, welche Spalten in Ihrem Export enthalten sein sollen, sodass Sie genau die Daten erhalten, die Sie benötigen — nicht mehr und nicht weniger.

## Anwendungsfälle

- **Umsatzabstimmung** — Exportieren Sie Buchungsberichte mit Finanzdetails, um diese mit Ihrem Buchhaltungssystem abzugleichen.
- **Leistungsanalyse** — Exportieren Sie Objektberichte, um die Leistung der Einträge in Ihrem Portfolio zu vergleichen.
- **Gästeeinblicke** — Exportieren Sie Buchungsdaten, um Reisendemografien und Buchungsmuster zu verstehen.
- **Kontoüberprüfung** — Exportieren Sie Kontodaten für Compliance- oder interne Prüfungen.
