## Nutzer einladen

Sie können andere Nutzer einladen, um bei der Verwaltung Ihres Affiliate-Kontos zu helfen. Sobald ein Nutzer Ihre Einladung annimmt, hat er vollen Zugriff auf Ihr Konto mit Ausnahme der Zahlungsfunktionen. Sie bleiben immer die einzige Person, die für die von Ihnen verwalteten Gelder verantwortlich ist.

<Steps>
1. Wählen Sie das Konto aus, mit dem Sie arbeiten möchten, unter [Accounts](https://my.wink.travel).
2. Klicken Sie unten auf der Kontokarte auf `Actions` und wählen Sie `Manage account`.
3. Klicken Sie auf der folgenden Seite auf den Reiter `Managers`.
4. Geben Sie die E-Mail-Adresse der Person ein, die Sie zur Verwaltung Ihres Kontos einladen möchten.
5. Klicken Sie neben der E-Mail auf den Link `Invite`.
</Steps>

Wink sendet der Person, die Sie zu Ihrem Konto hinzufügen möchten, eine E-Mail, und diese kann Ihre Einladung annehmen oder ablehnen.

## Eigentümerschaft übertragen

Wenn Sie derzeit der Kontoinhaber sind, haben Sie die Möglichkeit, die Eigentümerschaft des Kontos an einen bestehenden Manager zu übertragen.

Folgen Sie den untenstehenden Schritten, um die Eigentümerschaft zu ändern:

<Steps>
1. Wählen Sie das Konto aus, mit dem Sie arbeiten möchten, unter [Accounts](https://my.wink.travel).
2. Klicken Sie unten auf der Kontokarte auf `Actions` und wählen Sie `Manage account`.
3. Klicken Sie auf der folgenden Seite auf den Reiter `Managers`.
4. Klicken Sie neben einem Ihrer bestehenden Manager auf den Link `Make owner`.
5. Sie sind nun Manager des Kontos, aber nicht mehr der Eigentümer.
</Steps>
