Anwendungen geben Ihnen als Entwickler Zugriff auf alle Funktionen von Wink. Sie können auf alle dieselben Daten zugreifen und sogar die Funktionen von Wink erweitern.
Sie können den OAuth2-Server von Wink verwenden, um Ihre Reise-Website genauso zu authentifizieren, wie Sie es mit Google oder Facebook tun würden.
So können Sie die Funktionen von Wink in Ihre eigene Website oder Anwendung integrieren.

:::note
Ihre erste App wird bereits erstellt, wenn Sie ein [neues Konto erstellen](/de/my-account/create-account). Wir verwenden diese App für einige Buchungsszenarien.

Sie können diese App nicht erneut verwenden. Bitte erstellen Sie neue Apps, wenn Sie sie benötigen.
:::

<Steps>
 1. Wählen Sie das Konto aus, mit dem Sie arbeiten möchten, unter [Accounts](https://my.wink.travel).
 2. Klicken Sie unten auf der Kontokarte auf `Actions` und wählen Sie `Manage account`.
 3. Klicken Sie auf der folgenden Seite auf den Tab `Applications`.
 4. In diesem Abschnitt werden alle Ihre Anwendungen aufgelistet.
 5. Wählen Sie eine bestehende Anwendung zur Bearbeitung aus oder erstellen Sie eine neue.
 </Steps>

## App erstellen

Da Sie keinen Zugriff auf den `secret key` der für Sie erstellten App haben, müssen Sie Ihre eigene erstellen.

:::caution
Sie müssen ein Affiliate-Konto bei Wink oder ein Konto bei TripPay erstellen, bevor Sie eine App erstellen, da Sie Ihre Anwendung an ein bestimmtes Konto binden.
:::

Hier sind die Schritte zur Erstellung einer Anwendung:

<Steps>
1. Klicken Sie auf der Apps-Seite auf die Schaltfläche `Create new application`.
2. ***Name*** Geben Sie Ihrer App einen Namen. *z.B. Cool App*
3. **Entity name** Verknüpfen Sie Ihre App mit einem Ihrer bestehenden Konten. *z.B. Cool Account*
4. **Redirect URIs** Mindestens eine Redirect-URI ist erforderlich. Sie wird verwendet, um die Redirect-Domain nach einer erfolgreichen Authentifizierung zu validieren. *z.B. https://www.cool-site.com*
5. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
</Steps>

Sie werden zurück zu Ihrer Liste der Anwendungen weitergeleitet. Sie sehen Ihre neue App in der Liste und haben nur dieses eine Mal Zugriff auf den geheimen Schlüssel der App. Stellen Sie sicher, dass Sie ihn kopieren und an einem sicheren Ort speichern.

## App aktualisieren

Sie müssen Ihre App aktualisieren, wenn sich eine Ihrer Redirect-URIs ändert oder Sie eine URI hinzufügen oder entfernen möchten.

Um eine bestehende App zu aktualisieren, gehen Sie wie folgt vor:

<Steps>
1. Klicken Sie auf den Link `Actions` für die App, die Sie aktualisieren möchten.
2. Klicken Sie unter Actions auf die Schaltfläche `Update`.
3. Nehmen Sie Ihre Änderungen an der App vor.
4. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
</Steps>

Sie werden zurück zu Ihrer Liste der Anwendungen weitergeleitet.

## App entfernen

:::caution
Bitte löschen Sie nicht die App, die wir automatisch für Sie erstellt haben, um sicherzustellen, dass alle Funktionen für Ihr Konto funktionieren.
:::

Wenn Sie eine App nicht mehr benötigen, können Sie sie entfernen.

Um eine App zu entfernen, gehen Sie wie folgt vor:

<Steps>
1. Klicken Sie auf den Link `Actions` für die App, die Sie entfernen möchten.
2. Klicken Sie unter Actions auf die Schaltfläche `Remove`.
3. Klicken Sie auf die Schaltfläche `OK`, um die Löschung zu bestätigen.
</Steps>

Sie werden zurück zu Ihrer Liste der Anwendungen weitergeleitet.
