Eine Agentur ist ein normales Wink-Konto, das Sie als Eigentümer Ihrem Konto für Vertretungszwecke zuweisen können. Nachfolgend einige Gründe, warum Sie eine Agentur als Vertreter wünschen könnten:

- Vollständige / teilweise Verwaltung Ihres Wink-Kontos durch einen Dritten.
- Die Immobilie hat eine bestehende Vereinbarung mit ihrer Verwaltungsgesellschaft und möchte, dass Wink den finanziellen Aspekt der Vereinbarung bezüglich der über Wink eingehenden Gelder übernimmt.
- Die Immobilienverwaltungsagentur erledigt alle Online-Arbeiten im Namen ihrer Immobilien.
- Talentagentur, die einen Influencer / Content Creator vertritt.

## Agentur zuweisen

So weisen Sie eine Agentur Ihrem Konto zu.

<Steps>
 1. Wählen Sie das Konto aus, mit dem Sie arbeiten möchten, unter [Accounts](https://my.wink.travel).
 2. Klicken Sie unten auf der Kontokarte auf `Actions` und wählen Sie `Manage account`.
 3. Klicken Sie auf der folgenden Seite auf den Reiter `Agency`.
 4. Geben Sie im Abschnitt `Agency` den Namen der Agentur ein, die Sie vertreten soll.
 5. Wählen Sie die Agenturgebühr aus dem Dropdown-Menü.
 5. Optional können Sie Ablaufregeln hinzufügen.
 7. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
 </Steps>

 :::note
Sie müssen Eigentümer der Immobilie sein, um eine Agentur zuzuweisen.
 :::

 :::tip[Beispiel]
Die Agenturgebühr wird aus der Buchung berechnet.

Buchung = 100 $, Agenturgebühr = 1 % => 100 * 0,01 = 1 $
 :::

## Agentur entfernen

So entfernen Sie eine Agentur von Ihrem Konto.

<Steps>
 1. Wählen Sie das Konto aus, mit dem Sie arbeiten möchten, unter [Accounts](https://my.wink.travel).
 2. Klicken Sie unten auf der Kontokarte auf `Actions` und wählen Sie `Manage account`.
 3. Klicken Sie auf der folgenden Seite auf den Reiter `Agency`.
 4. Klicken Sie im Abschnitt `Agency` auf das <Badge text="x" variant="danger" size="small" /> Symbol rechts neben dem Agentur-Dropdown.
 5. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
 </Steps>
