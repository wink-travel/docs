Um Ihre WinkLinks-Tags zu verwalten, navigieren Sie zu`WinkLinks`in der Hauptnavigationsleiste und klicken Sie auf das`Tags`aus.

Wenn Ihr Inhalt wächst, möchten Sie ihn besser organisieren. Wir haben`tags`zu diesem Zweck. Sie können Inhalte mit beliebig vielen Tags gruppieren.

## Tag erstellen

<Steps>
  1. Erstellen Sie ein neues Tag, indem Sie auf das `Add a new tag` Taste.
  2. Geben Sie den Namen Ihres neuen Tags ein. *z.B. Top Cafés in Berlin*
  3. Klicken Sie auf das `Save` Link.
</Steps>


## Tag aktualisieren

<Steps>
  1. Klicken Sie zunächst auf das `Edit` Link neben dem Tag, das Sie ändern möchten.
  2. Aktualisieren Sie den Namen Ihres Tags. *z.B. Die besten Cafés in Berlin*
  3. Klicken Sie auf das `Save` Link.
</Steps>


## Tag entfernen

:::Hinweis
Wenn Sie ein Tag entfernen, wird es auch aus allen zugehörigen Inhalten entfernt.
:::

<Steps>
  1. Klicken Sie auf das `Remove` Link neben dem Tag, das Sie entfernen möchten.
</Steps>
