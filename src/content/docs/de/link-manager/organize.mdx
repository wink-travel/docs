Um Ihre WinkLinks-Tags zu verwalten, navigieren Sie in der Hauptnavigationsleiste zu `WinkLinks` und klicken Sie auf den Reiter `Tags`.

Mit wachsendem Inhalt möchten Sie diesen möglicherweise organisieren. Dafür haben wir `Tags` erstellt. Sie können Inhalte mit so vielen Tags gruppieren, wie Sie möchten.

## Tag erstellen

<Steps>
1. Erstellen Sie einen neuen Tag, indem Sie auf die Schaltfläche `Add a new tag` klicken.
2. Geben Sie den Namen Ihres neuen Tags ein. *z. B. Top-Kaffeeläden in Berlin*
3. Klicken Sie auf den Link `Save`.
</Steps>

## Tag aktualisieren

<Steps>
1. Beginnen Sie, indem Sie auf den Link `Edit` neben dem Tag klicken, den Sie ändern möchten.
2. Aktualisieren Sie den Namen Ihres Tags. *z. B. Beste Kaffeeläden in Berlin*
3. Klicken Sie auf den Link `Save`.
</Steps>

## Tag entfernen

:::note
Beim Entfernen eines Tags wird dieser auch aus allen zugehörigen Inhalten entfernt.
:::

<Steps>
1. Klicken Sie auf den Link `Remove` neben dem Tag, den Sie entfernen möchten.
</Steps>

## API

Entwickler, die `WinkLinks` verwalten möchten, können zu [Developers > API > WinkLinks](/de/developers/apis/#winklinks-api) wechseln.
