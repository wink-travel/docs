Diese Seite listet alle Webhook-Ereignisse auf, die Sie auf Wink abonnieren können. Für Einrichtungshinweise siehe den [Webhook-Integrationsleitfaden](/de/guides/webhook-integration) oder die [Webhook-Verwaltungsseite](/de/my-account/webhooks).

## So abonnieren Sie

1. Navigieren Sie im Portal zu **Anwendungen > Webhooks**.
2. Erstellen Sie einen Webhook und wählen Sie die Ereignisse aus, die Sie erhalten möchten.
3. Wink sendet bei jedem Auftreten des Ereignisses eine HTTP POST-Anfrage an Ihren Endpunkt.

Für vollständige Einrichtungsschritte und bewährte Verfahren siehe [Webhook-Integration](/de/guides/webhook-integration).

## Kontoereignisse

Ereignisse im Zusammenhang mit dem Lebenszyklus und den Einstellungen Ihres TripPay-Kontos.

| Ereignis | Schlüssel | Beschreibung |
|---|---|---|
| Konto erstellt | `account.create` | Ein neues Konto wurde erstellt. |
| Konto in Prüfung | `account.under_review` | Konto befindet sich in der Überprüfung. |
| Konto aktualisiert | `account.updated` | Kontodetails wurden geändert. |
| Kontostatus aktualisiert | `account.status.updated` | Kontostatus wurde geändert (z. B. genehmigt, gesperrt). |
| Kontoadresse aktualisiert | `account.address.updated` | Kontoadresse wurde geändert. |
| Kontodetails aktualisiert | `account.details.updated` | Kontodetailfelder wurden geändert. |
| Kontoinformationen aktualisiert | `account.information.update` | Allgemeine Kontoinformationen wurden aktualisiert. |
| Kontomanager aktualisiert | `account.managers.update` | Liste der Kontomanager wurde geändert. |
| Profilbild aktualisiert | `account.profile_picture.updated` | Profilbild des Kontos wurde geändert. |
| Agent aktualisiert | `account.agent.updated` | Agenteninformationen im Konto wurden geändert. |
| URL-Name aktualisiert | `account.url-name.update` | Vanity-URL des Kontos wurde geändert. |
| Eindeutige ID aktualisiert | `account.unique-id.update` | Eindeutiger Kontobezeichner wurde geändert. |

### Kontenaufgaben

|---|---|---|
| Aufgabe erstellt | `account.task.create` | Eine neue Aufgabe wurde dem Konto hinzugefügt. |
| Aufgabe aktualisiert | `account.task.update` | Eine Kontenaufgabe wurde geändert. |
| Aufgabe entfernt | `account.task.remove` | Eine Kontenaufgabe wurde entfernt. |
| Aufgabe abgeschlossen | `account.task.complete` | Eine Kontenaufgabe wurde abgeschlossen. |
| Steuer-ID-Aufgabe abgeschlossen | `account.task.tax_id.complete` | Steuer-ID-Verifizierungsaufgabe wurde abgeschlossen. |

### Konto-Akquisiteur

| Ereignis | Schlüssel | Beschreibung |

## Ereignisse der Verwaltungseinheit

Ereignisse für Verwaltungseinheiten (Objekte, Partner, Unternehmen) und deren Verwaltung.

| Ereignis | Schlüssel | Beschreibung |
|---|---|---|
| Abonnement bereitgestellt | `managing_entity.subscription.provisioned` | Ein neues Abonnement wurde bereitgestellt. |
| Abonnement geändert | `managing_entity.subscription.changed` | Das Abonnement wurde geändert. |
| Abonnementrechnung geändert | `managing_entity.subscription.invoice.changed` | Die Abonnementrechnung wurde aktualisiert. |

### Status und Verwaltung

| Ereignis | Schlüssel | Beschreibung |
|---|---|---|
| Status aktualisiert | `managing_entity.status.updated` | Der Status der Einheit wurde geändert. |
| Einheit gelöscht | `managing_entity.deleted` | Eine Verwaltungseinheit wurde gelöscht. |
| Agent aktualisiert | `managing_entity.agent.updated` | Der Agent der Einheit wurde geändert. |

| Manager E-Mail-Benachrichtigung | `managing_entity.manager.email.notification` | Eine E-Mail-Benachrichtigung wurde an einen Manager gesendet. |

### Manager-Lebenszyklus

| Ereignis | Schlüssel | Beschreibung |
|---|---|---|

| Manager entfernt | `managing_entity.manager.removed` | Ein Manager wurde entfernt. |
