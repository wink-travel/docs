Wink integriert sich mit führenden Channel Managern, Property Management Systemen (PMS) und zentralen Reservierungssystemen (CRS), um Ihre Preise und Verfügbarkeiten über alle Vertriebskanäle hinweg synchron zu halten.

## So funktioniert es

Channel Manager Integrationen ermöglichen einen bidirektionalen Datenaustausch zwischen Ihrem Property Management System und Wink:

1. **Preisübermittlung** — Ihr Channel Manager sendet Preisaktualisierungen an Wink. Wenn Sie Ihre Preise in Ihrem PMS ändern, werden diese Änderungen automatisch auf Wink übernommen.
2. **Verfügbarkeits-Synchronisation** — Ihr Channel Manager sendet Verfügbarkeitsaktualisierungen an Wink. Wenn Zimmer in Ihrem PMS gebucht oder freigegeben werden, aktualisiert Wink das Inventar in Echtzeit.
3. **Buchungsbenachrichtigungen** — Wenn eine Buchung über Wink erfolgt, wird Ihr Channel Manager benachrichtigt, sodass die Reservierung in Ihrem PMS erscheint.

## Unterstützte Channel Manager

Wink integriert derzeit die folgenden Systeme:

| Channel Manager | Beschreibung |
|---|---|
| **Channex** | Globale Vertriebs- und Channel Management Plattform. |
| **TravelClick** | Hotel-Lösungen und Channel Management (Amadeus). |
| **Omnibees** | Vertrieb und Revenue Management für Hotels. |
| **D-Edge** | Hospitality-Technologie und zentrale Reservierungslösungen. |
| **Allotz** | Inventarzuweisung und Channel Management. |
| **Ezee / Yanolja** | Cloud-PMS und Channel Management mit Fokus auf den asiatischen Markt. |
| **RateGain** | Preisintelligenz und Vertriebstechnologie. |
| **RateTiger** | Echtzeit-Verteilung von Preisen und Verfügbarkeiten. |
| **CloudBeds** | Cloud-basiertes PMS, Channel Manager und Buchungsmaschine. |
| **SiteMinder** | Führender Hotel-Channel Manager und Vertriebsplattform. |
| **Exely** | Preis- und Verfügbarkeits-Synchronisation für Hotels. |

:::note
Wenn Ihr Channel Manager nicht aufgeführt ist, wenden Sie sich an das Wink-Team. Neue Integrationen können basierend auf den OTA (Open Travel Alliance) Protokollstandards hinzugefügt werden, die Wink unterstützt.
:::

## Für Entwickler

Wenn Sie eine Channel Manager- oder PMS-Integration mit Wink entwickeln, verwenden Sie die Channel Manager API-Endpunkte:

- **Preisaktualisierungen** — Übermitteln Sie Preisänderungen an Wink.
- **Verfügbarkeitsaktualisierungen** — Übermitteln Sie Verfügbarkeitsänderungen an Wink.
- **Buchungsabruf** — Holen Sie Buchungen von Wink in Ihr System.

Die Channel Manager API folgt den OTA-Protokollstandards (SOAP/XML) für die Kompatibilität mit bestehenden Hospitality-Systemen.

## Webhook-Benachrichtigungen

Sie können Channel Manager Webhook-Events abonnieren, um Echtzeit-Benachrichtigungen zu erhalten:

- `channel-manager.update.rate` — Preisaktualisierung erhalten.
- `channel-manager.update.availability` — Verfügbarkeitsaktualisierung erhalten.
- `channel-manager.update` — Allgemeine Channel Manager Aktualisierung.

Details finden Sie im [Webhook Events Katalog](/de/developers/webhook-events#channel-manager-events).
