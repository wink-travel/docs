Für alle, die WordPress für ihr Unternehmen oder Blog nutzen, empfehlen wir dringend die Verwendung des [Wink Affiliate WordPress Plugins](https://wordpress.org/plugins/wink2travel/).
Das Plugin bietet einfachen Zugriff auf Ihre Wink-Karten, Raster, Karten usw. und macht es so einfach wie das Ziehen von Inhalten in Ihre Beiträge und Seiten mit Ihrem bevorzugten Page Builder oder unseren benutzerdefinierten Shortcodes.

##  Installation

So installieren Sie das `Wink Affiliate WordPress Plugin` auf Ihrer WordPress-Website.

<Steps>
1. Melden Sie sich als Administrator an und gehen Sie zum Dashboard.
2. Klicken Sie auf `Plugins > Neues Plugin hinzufügen`.
3. Suchen Sie nach `Wink Affiliate WordPress Plugin`.
4. Klicken Sie auf `Installieren`, um das Plugin herunterzuladen und auf Ihrer Seite zu installieren.
5. Klicken Sie auf `Aktivieren`, um das Plugin auf Ihrer Seite zu aktivieren.
6. Nach der Aktivierung werden Sie zur Seite `Wink Einstellungen` weitergeleitet. *Sie können auch dorthin gelangen, indem Sie zu `Design > Customizer > Wink Einstellungen` navigieren.*
7. Dort werden Sie nach Ihrer `Client ID` und `Client Secret` gefragt. Gehen Sie zu [Applications](/de/developers/applications), um zu erfahren, wie Sie Ihre Zugangsdaten erhalten.
8. Geben Sie Ihre Zugangsdaten ein und klicken Sie auf `Veröffentlichen`. Sie sind jetzt mit Ihrem Wink-Konto verbunden.
</Steps>

## Page Builder

Wir unterstützen die beliebtesten Page Builder auf WordPress, die es jedem erleichtern, Wink zu nutzen.

### Gutenberg

Gutenberg ist der native Editor von WordPress.

### Avada

[Avada](https://avada.com) ist bekannt für seine große Auswahl an schönen Vorlagen sowie einen robusten Page Builder mit einem erweiterten Funktionsumfang.

### Elementor

[Elementor](https://elementor.com) ist der [derzeit] beliebteste Page Builder mit einer kostenlosen und einer Pro-Version.

### WPBakery

[WPBakery](https://wpbakery.com) war der erste fortschrittliche Page Builder auf dem Markt.

## Shortcodes

### [winklookup]

Funktioniert mit dem gerankten Inhaltsraster. Geben Sie einen Ort ein, den Sie besuchen möchten, und lassen Sie sich die Verfügbarkeiten anzeigen.

### [winksearch]

Einfacher Button, um das Reiseplanungsformular zu öffnen.

### [winkaccount]

Account-Button, mit dem Sie sich authentifizieren können. Nach der Authentifizierung wird er zu einem Dropdown mit Kontooptionen.

### [winkitinerary]

Wie der Suchbutton, enthält jedoch Reiseplanungsinformationen als Button-Text.

### [winkcontent]

Ermöglicht es Ihnen, die von Ihnen auf Wink ausgewählte Verfügbarkeit direkt in eine Seite oder einen Beitrag einzubetten. Dieser Shortcode erwartet auch eine Layout-ID.

Das Layout

```
[winkcontent layoutid=""]
```
