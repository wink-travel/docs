Für alle, die WordPress für ihr Unternehmen oder ihren Blog verwenden, empfehlen wir die Verwendung von[Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
Das Plugin ermöglicht Ihnen einfachen Zugriff auf Ihre Wink-Karten, Raster, Karten usw. und macht es so einfach wie das Ziehen von Inhalten in Ihre Beiträge und Seiten mithilfe Ihres bevorzugten Seitenerstellers oder unserer benutzerdefinierten Shortcodes.

## Installieren

So installieren Sie die`Wink Affiliate WordPress Plugin`in Ihre WordPress-Website.

<Steps>
  1. Wenn Sie als Administrator angemeldet sind und sich im Dashboard befinden.
  2. Klicken `Plugins > Add New Plugin`.
  3. Suchen nach `Wink Affiliate WordPress Plugin`.
  4. Klicken `Install` um das Plugin herunterzuladen und auf Ihrer Site zu installieren.
  5. Klicken `Activate` um das Plugin auf Ihrer Site zu aktivieren.
  6. Durch die Aktivierung erhalten Sie die `Wink Settings` Seite. *Sie können auch dorthin gelangen, indem Sie zu navigieren `Appearance > Customize > Wink Settings`.*
  7. Sie werden nach Ihrem `Client ID` Und `Client Secret`. Gehe zu [Anwendungen](/developers/applications) um zu erfahren, wie Sie Ihre Anmeldeinformationen abrufen.
  8. Geben Sie Ihre Anmeldeinformationen ein und klicken Sie auf `Publish`. Sie sind jetzt mit Ihrem Wink-Konto verbunden.
</Steps>

## Seitenersteller

Wir unterstützen die beliebtesten Seitenersteller auf WordPress, die es jedem leicht machen, Wink zu verwenden.

### Gutenberg

Gutenberg ist der native Editor von WordPress.

### Avada

[Avada](https://avada.com) ist berühmt für seine große Auswahl an schönen Vorlagen sowie einen robusten Seitenersteller mit einem erweiterten Funktionsumfang.

### Elementor

[Elementor](https://elementor.com) ist der \[derzeit] beliebteste Seitenersteller, der sowohl in einer kostenlosen als auch in einer Pro-Version verfügbar ist.

### WPBakery

[WPBakery](https://wpbakery.com) war der erste erweiterte Seitenersteller auf dem Markt.

## Kurzwahlnummern

### \[zwinkerlookup]

Funktioniert mit einem geordneten Inhaltsraster. Geben Sie einen Ort ein, den Sie besuchen möchten, und lassen Sie sich das Inventar anzeigen.

### \[zwinkersuche]

Einfache Schaltfläche zum Öffnen des Reiseroutenformulars.

### \[zwinkeraccount]

Schaltfläche „Konto“ zur Authentifizierung. Nach der Authentifizierung wird ein Dropdown-Menü mit Kontooptionen angezeigt.

### \[zwinker]

Dasselbe wie die Suchschaltfläche, nur dass sie Reiserouteninformationen als Schaltflächentext enthält.

### \[zwinkerinhalt]

Ermöglicht das Einbetten des auf Wink ausgewählten Inventars direkt in eine Seite oder einen Beitrag. Dieser Shortcode erwartet auch eine Layout-ID.

Das Layout

```
[winkcontent layoutid=""]
```
