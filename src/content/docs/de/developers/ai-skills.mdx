KI-Fähigkeiten sind portable Anweisungspakete, die Coding-Agenten beibringen, wie sie mit Wink arbeiten. Anstatt sich auf Trainingsdaten oder veraltete Dokumentation zu verlassen, weist jede Fähigkeit den Agenten an, den [Wink MCP-Server](/de/developers/mcp) als aktuelle Quelle der Wahrheit für API-Verträge und Plattformdokumentationen zu verwenden.

Fähigkeiten funktionieren mit jedem Agenten-Client, der sie unterstützt — Claude Code, Codex, Cursor und andere.

## Verfügbare Fähigkeiten

### wink-api-builder

Implementieren, überprüfen oder debuggen Sie Wink API-Integrationen. Die Fähigkeit führt den Agenten durch einen strukturierten Workflow:

1. Suche nach der richtigen API-Operation über `api_search`
2. Öffnen der Live-Anfrage-/Antwortverträge vom MCP-Server
3. Abrufen der Authentifizierungseinrichtung aus der Dokumentation
4. Schreiben von Code basierend auf dem verifizierten Vertrag
5. Markieren von Abweichungen zwischen bestehendem Code und der aktuellen API

Nützlich für:
- Erstellen neuer Client-Methoden für jede Wink API
- Debuggen von Vertragsabweichungen (z. B. eine Anfrage, die HTTP 400 zurückgibt)
- Überprüfen bestehender Integrationen anhand der Live-Spezifikation

## Installation

Kopieren Sie den Fähigkeitsordner in das Fähigkeitsverzeichnis Ihres Agenten.

:::tip
Der Wink MCP-Server muss in derselben Umgebung verbunden sein. Siehe [MCP Server](/de/developers/mcp) für Installationsanweisungen.
:::

**Claude Code**

```bash
# repository-lokal
cp -r skills/wink-api-builder .claude/skills/

# oder benutzerweit
cp -r skills/wink-api-builder ~/.claude/skills/
```

Aufruf mit:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repository-lokal
cp -r skills/wink-api-builder .agents/skills/

# oder benutzerweit
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Aufruf mit:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Andere Agenten**

Platzieren Sie den Fähigkeitsordner dort, wo Ihr Agent Fähigkeiten auflöst. Der Einstiegspunkt ist `SKILL.md`.
