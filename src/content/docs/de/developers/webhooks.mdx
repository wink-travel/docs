Mithilfe von Webhooks können Sie Echtzeitereignisse von Wink und TripPay bezüglich Ihres Kontos empfangen und darauf reagieren.

## Beispiel

<Steps>
  1. Eine Buchung erfolgt über Wink.
  2. Das Wink-Event `booking.created` wird versendet.
  3. Ihr System überwacht dieses Ereignis aktiv.
  4. Ihr System verfolgt Ihren eigenen Satz von Analysen und verwendet dieses Ereignis, um seine Zeitreihendatenbank zu aktualisieren.
</Steps>

:::Hinweis
Ein Wink-Webhook unterscheidet sich von einem TripPay-Webhook.

* Wenn Sie Wink-Ereignisse abhören möchten, können Sie einen Webhook erstellen von[Extranet](https://extranet.wink.travel) oder [Studio](https://studio.wink.travel).
* Wenn Sie TripPay-Ereignisse abhören möchten, können Sie einen Webhook erstellen von[Zahlung](https://pay.wink.travel).
  ::

*Für dieses Beispiel authentifizieren wir uns über[Studio](https://studio.wink.travel).*

<Steps>
  1. Klicken Sie nach der Authentifizierung auf Ihr Profilsymbol in der oberen rechten Ecke. Ein Menü wird angezeigt.
  2. Klicken Sie auf das `Webhooks` Link.
  3. Sie werden zu Ihrer Webhook-Seite weitergeleitet.
</Steps>


## Webhook erstellen

:::Achtung
Sie müssen ein Partnerkonto bei Wink oder ein Konto bei TripPay erstellen, bevor Sie einen Webhook erstellen, da Sie Ihren Webhook an ein bestimmtes Konto binden.
:::

So erstellen Sie einen Webhook:

<Steps>
  1. Klicken Sie auf der Webhooks-Seite auf das `Create new webhook` Taste.
  2. ***Name*** Geben Sie Ihrem Webhook einen Namen. *zB Wink Stalker*
  3. ***Aktivieren*** Zum Aktivieren unverändert lassen.
  4. **Entitätsname** Verknüpfen Sie Ihren Webhook mit einem Ihrer bestehenden Konten. *zB Cool Account*
  5. **Veranstaltungen** Hören Sie sich die Ereignisse an, die Sie interessieren. *z.B. `booking.created`*
  6. **Webhook-URL** Geben Sie die URL ein, unter der Sie unsere Veranstaltungen anhören möchten. *z.B. `https://my.system.com/webhook/wink`*
  7. Klicken Sie auf das `Save` Schaltfläche, um fortzufahren.
</Steps>


Sie werden zurück zu Ihrer Liste mit Webhooks umgeleitet und Ihr neuer Webhook ist einsatzbereit.

## Webhook aktualisieren

Sie müssen Ihre App aktualisieren, wenn Sie:

* Deaktivieren Sie Ihren Webhook.
* Mehr/weniger Events anhören.
* Ändern Sie die URL Ihres Listening-Webhooks.

Um einen vorhandenen Webhook zu aktualisieren, gehen Sie wie folgt vor:

<Steps>
  1. Klicken Sie auf das `Actions` Link für den Webhook, den Sie aktualisieren möchten.
  2. Klicken Sie auf das `Update` Schaltfläche unter Aktionen.
  3. Nehmen Sie Ihre Änderungen am Webhook vor.
  4. Klicken Sie auf das `Save` Schaltfläche, um fortzufahren.
</Steps>


Sie werden zurück zu Ihrer Liste mit Webhooks weitergeleitet.

## Webhook entfernen

Wenn Sie den Webhook nicht mehr benötigen, können Sie ihn entfernen.

Gehen Sie wie folgt vor, um einen Webhook zu entfernen:

<Steps>
  1. Klicken Sie auf das `Actions` Link für den Webhook, den Sie aktualisieren möchten.
  2. Klicken Sie auf das `Remove` Schaltfläche unter Aktionen.
  3. Klicken `OK` Schaltfläche, um das Löschen zu bestätigen.
</Steps>


Sie werden zurück zu Ihrer Liste mit Webhooks weitergeleitet.
