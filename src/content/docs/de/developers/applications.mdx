Anwendungen bieten Ihnen vollen Zugriff auf die Funktionen von Wink und TripPay und ermöglichen Ihren Nutzern die Authentifizierung und Anmeldung auf Ihrer Website über unseren OAuth2-Autorisierungsserver – genau wie bei Facebook und Google.

Mit Wink können Sie Ihr eigenes OTA erstellen. Alles beginnt mit einer Anwendung.

*Standardmäßig erhalten Sie eine App, wenn Sie ein neues Wink-Partnerkonto oder TripPay-Konto registrieren. Wir verwenden die`client ID`für einige Szenarien.*

:::Hinweis
Eine auf Wink oder TripPay erstellte Anwendung kann für alle API-Endpunkte verwendet werden.
:::

*Für dieses Beispiel authentifizieren wir uns über[Wink Studio](https://studio.wink.travel).*

<Steps>
  1. Klicken Sie nach der Authentifizierung auf Ihr Profilsymbol in der oberen rechten Ecke. Ein Menü wird angezeigt.
  2. Klicken Sie auf das `Applications` Link.
  3. Sie werden auf Ihre App-Seite weitergeleitet.
</Steps>


## App erstellen

Da Sie keinen Zugriff auf die`secret key`der App, die wir für Sie erstellt haben, müssen Sie Ihre eigene erstellen.

:::Achtung
Sie müssen vor der App-Erstellung ein Partnerkonto bei Wink oder TripPay erstellen, da Ihre Anwendung an ein bestimmtes Konto gebunden ist.
:::

So erstellen Sie eine Anwendung:

<Steps>
  1. Klicken Sie auf der Apps-Seite auf das `Create new application` Taste.
  2. ***Name*** Geben Sie Ihrer App einen Namen. *zB Coole App*
  3. **Entitätsname** Verknüpfen Sie Ihre App mit einem Ihrer bestehenden Konten. *zB Cool Account*
  4. **Umleitungs-URIs** Es ist mindestens eine Weiterleitungs-URI erforderlich. Diese dient zur Validierung der Weiterleitungsdomäne nach erfolgreicher Authentifizierung. *z.B. https://www.cool-site.com*
  5. Klicken Sie auf das `Save` Schaltfläche, um fortzufahren.
</Steps>


Sie werden zurück zu Ihrer Anwendungsliste geleitet. Ihre neue App wird dort angezeigt und Sie haben nur dieses eine Mal Zugriff auf den geheimen App-Schlüssel. Kopieren Sie ihn und speichern Sie ihn an einem sicheren Ort.

## App aktualisieren

Sie müssen Ihre App aktualisieren, wenn sich eine Ihrer Umleitungs-URIs ändert oder Sie eine URI hinzufügen oder entfernen möchten.

Um eine vorhandene App zu aktualisieren, gehen Sie wie folgt vor:

<Steps>
  1. Klicken Sie auf das `Actions` Link für die App, die Sie aktualisieren möchten.
  2. Klicken Sie auf das `Update` Schaltfläche unter Aktionen.
  3. Nehmen Sie Ihre Änderungen an der App vor.
  4. Klicken Sie auf das `Save` Schaltfläche, um fortzufahren.
</Steps>


Sie werden zurück zu Ihrer Bewerbungsliste geleitet.

## App entfernen

:::Achtung
Bitte löschen Sie die automatisch erstellte App nicht, um sicherzustellen, dass alle Funktionen für Ihr Konto funktionieren.
:::

Wenn Sie eine App nicht mehr benötigen, können Sie sie einfach entfernen.

Um eine App zu entfernen, gehen Sie wie folgt vor:

<Steps>
  1. Klicken Sie auf das `Actions` Link für die App, die Sie aktualisieren möchten.
  2. Klicken Sie auf das `Remove` Schaltfläche unter Aktionen.
  3. Klicken `OK` Schaltfläche, um das Löschen zu bestätigen.
</Steps>


Sie werden zurück zu Ihrer Bewerbungsliste geleitet.
