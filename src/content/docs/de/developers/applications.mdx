Anwendungen geben Ihnen vollen Zugriff auf Funktionen von Wink und TripPay und ermöglichen es Ihren Nutzern, sich über unseren OAuth2-Autorisierungsserver auf Ihrer Website zu authentifizieren und anzumelden – genau wie bei Facebook und Google.
Sie können mit Wink Ihre eigene OTA erstellen. Alles beginnt mit einer Anwendung.

*Standardmäßig erhalten Sie eine App, wenn Sie ein neues Wink-Partnerkonto oder TripPay-Konto registrieren. Wir verwenden die `client ID` für einige Szenarien.*

Eine auf Wink oder TripPay generierte Anwendung kann für alle API-Endpunkte verwendet werden.

*Für dieses Beispiel authentifizieren wir uns über [Wink Studio](https://studio.wink.travel).*

<Steps>
1. Nach der Authentifizierung klicken Sie auf Ihr Profil-Symbol oben rechts, und ein Menü erscheint.
2. Klicken Sie auf den Link `Applications`.
3. Sie werden zu Ihrer App-Seite weitergeleitet.
</Steps>

## App erstellen

Da Sie keinen Zugriff auf den `secret key` der für Sie erstellten App haben, müssen Sie Ihre eigene erstellen.

:::caution
Sie müssen ein Partnerkonto bei Wink oder ein Konto bei TripPay erstellen, bevor Sie eine App anlegen, da Sie Ihre Anwendung an ein bestimmtes Konto binden.
:::

Hier sind die Schritte zur Erstellung einer Anwendung:

<Steps>
1. Klicken Sie auf der App-Seite auf die Schaltfläche `Create new application`.
2. ***Name*** Geben Sie Ihrer App einen Namen. *z. B. Cool App*
3. **Entity name** Verknüpfen Sie Ihre App mit einem Ihrer bestehenden Konten. *z. B. Cool Account*
4. **Redirect URIs** Mindestens eine Redirect-URI ist erforderlich. Sie wird zur Validierung der Redirect-Domain nach erfolgreicher Authentifizierung verwendet. *z. B. https://www.cool-site.com*
5. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
</Steps>

Sie werden zurück zu Ihrer Liste der Anwendungen geleitet. Sie sehen Ihre neue App in der Liste und haben nur dieses eine Mal Zugriff auf den geheimen Schlüssel der App. Stellen Sie sicher, dass Sie ihn kopieren und an einem sicheren Ort speichern.

## App aktualisieren

Sie müssen Ihre App aktualisieren, wenn sich eine Ihrer Redirect-URIs ändert oder Sie eine URI hinzufügen oder entfernen möchten.

So aktualisieren Sie eine bestehende App:

<Steps>
1. Klicken Sie auf den Link `Actions` für die App, die Sie aktualisieren möchten.
2. Klicken Sie unter Aktionen auf die Schaltfläche `Update`.
3. Nehmen Sie Ihre Änderungen an der App vor.
4. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
</Steps>

Sie werden zurück zu Ihrer Liste der Anwendungen geleitet.

## App entfernen

:::caution
Bitte löschen Sie nicht die App, die wir automatisch für Sie erstellt haben, um sicherzustellen, dass alle Funktionen für Ihr Konto funktionieren.
:::

Wenn Sie eine App nicht mehr benötigen, können Sie sie entfernen.

So entfernen Sie eine App:

<Steps>
1. Klicken Sie auf den Link `Actions` für die App, die Sie entfernen möchten.
2. Klicken Sie unter Aktionen auf die Schaltfläche `Remove`.
3. Klicken Sie auf die Schaltfläche `OK`, um die Löschung zu bestätigen.
</Steps>

Sie werden zurück zu Ihrer Liste der Anwendungen geleitet.
