Alle unsere REST-APIs sind sprachunabhängig und Open API-konform. Beginnen Sie mit dem Testen, ohne eine einzige Codezeile zu schreiben.

Wenn Sie noch kein Wink-Konto erstellt haben, [starten Sie hier](/user/register).

## Consume endpoints

Consume-Endpunkte sind für Entwickler, die bestehende Reisebestände finden und entweder buchen oder über eines ihrer Wink-Partnerkonten bewerben möchten.

### Configuration API

Ein einzelner Endpunkt, um Whitelabel- und Anpassungsinformationen für die Buchungsmaschine abzurufen.

<LinkButton href="https://developer.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
Mehr erfahren
</LinkButton>

### Lookup API

Eine programmiererfreundliche Möglichkeit, maßgeschneiderte Reisebestände über geografische Regionen hinweg zu suchen.

<LinkButton href="https://developer.wink.travel/lookup" variant="secondary" icon="external" target="_blank">
Mehr erfahren
</LinkButton>

### Inventory API

Eine programmiererfreundliche Möglichkeit, bekannte Reisebestände abzurufen, wie sie offiziell mit der Lookup API lokalisiert wurden.

<LinkButton href="https://developer.wink.travel/inventory" variant="secondary" icon="external" target="_blank">
Mehr erfahren
</LinkButton>

### Booking API

Eine programmiererfreundliche Möglichkeit, Bestände zu buchen, die auf unserer Plattform gefunden wurden.

<LinkButton href="https://developer.wink.travel/booking" variant="secondary" icon="external" target="_blank">

</LinkButton>

### Travel Agent API

Die Travel Agent API stellt Endpunkte zur Verwaltung von durch Agenten vermittelten Buchungen bereit.

## Payment API

Zahlungsendpunkte sind für Entwickler, die Reisebestände kaufen möchten. Dies kann über die API als registrierter Reisebüro-Agent oder in Verbindung mit unserem PCI-konformen Zahlungs-Widget für alle anderen Entitäten erfolgen.
Die Payment API stellt Endpunkte für Partner und Hotels bereit, um Buchungen, Analysen und Verfügbarkeiten von Mitteln zu verfolgen. Sie können auch verfügbare Mittel auf ihr Bankkonto auszahlen lassen.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Mehr erfahren
</LinkButton>
