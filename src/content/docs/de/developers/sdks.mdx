Beschleunigen Sie die Markteinführungszeit, indem Sie eines unserer offiziell unterstützten Software Development Kits für Sprachen verwenden.  
Falls wir Ihre bevorzugte Sprache derzeit nicht unterstützen, gehen Sie zur API, die Sie verwenden möchten, und laden Sie die OpenAPI-Spezifikation herunter, die wir für jede API oben auf der Seite bereitstellen. Sie können Ihr eigenes SDK generieren, indem Sie entweder eines unserer öffentlichen SDK-Repositories forken oder einen der OpenAPI-Generatoren verwenden.

## Java

Diese Bibliothek wurde für Java 24 kompiliert. Sie ist so konfiguriert, dass sie den reaktiven WebClient verwendet und für die Nutzung mit dem Spring Framework optimiert ist. Spring ist jedoch keine Voraussetzung, bietet aber OAuth2-Unterstützung direkt out of the box.
