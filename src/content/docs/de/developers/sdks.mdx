Beschleunigen Sie die Markteinführungszeit mit einem unserer offiziell unterstützten Software Development Kits (SDKs).

Sollten wir Ihre bevorzugte Sprache derzeit nicht unterstützen, rufen Sie die gewünschte API auf und laden Sie die OpenAPI-Spezifikation herunter, die wir oben auf der Seite für jede API bereitstellen. Sie können Ihr eigenes SDK generieren, indem Sie entweder eines unserer öffentlichen SDK-Repositories forken oder einen der OpenAPI-Generatoren verwenden.

## Java

Diese Bibliothek wurde für Java 24 kompiliert. Sie ist für die Verwendung des reaktiven WebClients konfiguriert und für die Verwendung mit dem Spring Framework optimiert. Spring ist zwar keine Voraussetzung, bietet aber standardmäßig OAuth2-Unterstützung.
