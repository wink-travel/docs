Teilbare Links sind eine Kernfunktion von Wink und entstanden aus der Notwendigkeit, Lagerbestände und Preise auf dynamische Weise und auf die einfachste Art und Weise im Internet mitzuteilen, nämlich über den Hyperlink.

<h4 align="center">[https://trvl.as/s/3xWCH](https://trvl.as/s/3xWCH)</h4>


Oben sehen Sie ein Beispiel für einen unserer Links. Er ist kurz und handlich – sogar kurz genug, um ihn sich zu merken. Obwohl er kurz ist, bietet dieser Link eine**explosiv**Menge an Informationen, wie zum Beispiel:

* Ihre App-ID.
* Ihre Partner-ID.
* Die Bilder und der Text, die dem Benutzer angezeigt werden sollen.
* Das Look-and-Feel (*d.h. Thema, Farben und mehr*).
* Die Reiseroute, die wir für unsere Preisabfrage verwenden möchten (*2 Erwachsene, 1 Nacht*).
* Die Währung, in der der Preis angezeigt werden soll.
* Die Sprache, in der der Text angezeigt werden soll.

Der Rest dieses Artikels führt Sie durch die Erstellung, Anpassung und Freigabe Ihrer Links.

## Erstellen eines Links

  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="Aktionen" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

Oben ist ein Bild aus[Suchen](/studio/search)und zeigt Ihnen einige der Dinge, die Sie mit Ihren Suchergebnissen tun können. Eine dieser Aktionen ist`Make a link`.
Klicken Sie auf diese Schaltfläche. Sie werden dann zu unserer Formularseite für teilbare Links weitergeleitet, wo Sie mit der Anpassung Ihres Links beginnen können.

## Link anpassen

Mit dem Formular können Sie Ihren Link auf folgende Weise anpassen:

<Steps>
  1. Geben Sie ihm einen Namen, damit Sie sich merken können, worum es bei dem Link geht.
  2. Wählen Sie die [Anpassung](/studio/customization) Sie möchten sich über diesen Link bewerben.
  3. Fügen Sie durch Kommas getrennte Schlüsselwörter hinzu, die verwendet werden von [Webcrawler](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
  4. Geben Sie optional Ihren X-Konto-Handle oder Ihre Facebook-App-ID ein.
  5. Fügen Sie Titel und Beschreibungen in den Sprachen hinzu, die der Benutzer sehen soll.
  6. Wählen Sie ein oder mehrere Bilder aus, die angezeigt werden, wenn Sie den Link auf Plattformen wie Facebook oder LinkedIn teilen.
  7. Wählen Sie ein Design/Overlay, das den ausgewählten Bildern ein Markendesign hinzufügt. *Hinweis: Auf diese Weise werden die Preise über dem Bild angezeigt.*
  8. Klicken Sie auf das `Save` Schaltfläche, um fortzufahren.
</Steps>

## Link teilen

<figure>
  <CloudinaryImage width="350" height="641" resize="scale" quality="best" alt="Aktionen" src="academy/shareable-link-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Aktionen für teilbare Links*</figcaption>
</figure>


Oben sehen Sie ein Bild mit allen verfügbaren Aktionen für Ihren Link:

* **Vorschau**Zeigt Ihnen einen schnellen Überblick darüber, wie der Link konfiguriert wurde und wie er für den Benutzer aussehen wird.
* **Aktualisieren**Aktualisiert Ihre Linkkonfiguration.
* **Zu WinkLinks hinzufügen**Fügt den Link zu Ihrem WinkLinks-Konto hinzu.
* **Kopie**Kopiert den Link in Ihre Zwischenablage, sodass Sie ihn problemlos überall teilen können (*einschließlich [WinkLinks](/link-manager/wink-links)*).
* **Aktie**Öffnet ein neues Fenster mit Optionen zum Teilen in vielen sozialen Netzwerken und Messaging-Plattformen.
* **QR-Code**Generiert einen QR-Code, den Sie auf Plattformen wie IG teilen können, die stärker auf Bilder ausgerichtet sind.
* **Einbetten**Zeigt Ihnen, wie Sie diesen Link als[Werbebanner](/developers/web-components/#content-loader)in Ihre Website.
* **Verwendung mit WordPress**Zeigt Ihnen, wie Sie unsere[WordPress-Plugin](/developers/wordpress)um diesen Link in Ihre Website einzubetten.

Einige dieser Optionen werden im Folgenden ausführlicher behandelt.

### Vorschau

  <CloudinaryImage width="700" height="373" resize="scale" quality="best" alt="Vorschau" src="academy/shareable-link-preview" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Vorschau des teilbaren Links*</figcaption>

Wenn Sie auf`Preview`, sehen Sie ein Bild mit Ihrem Design. Wenn Sie den Link in sozialen Medien wie Facebook teilen, wird dasselbe Bild zusammen mit Ihrem Titel und Ihrer Beschreibung in Ihrem Feed angezeigt.

### Soziales Teilen

<figure>
  <CloudinaryImage width="350" height="231" resize="scale" quality="best" alt="Soziales Teilen" src="academy/shareable-link-share" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Social-Share-Optionen*</figcaption>
</figure>


Wenn Sie auf das`Share`Klicken Sie auf die Schaltfläche. Ein neues Fenster öffnet sich und bietet Ihnen eine praktische Möglichkeit, den Link direkt in Ihren bevorzugten sozialen Netzwerken und Messaging-Apps zu teilen.
Klicken Sie auf eine der Symbolschaltflächen, um Ihren Link in diesem Netzwerk weiter zu veröffentlichen.*Hinweis: Einige Schaltflächen werden nur angezeigt, wenn Sie von Ihrem Mobilgerät aus teilen.*

### QR-Code

<figure>
  <CloudinaryImage width="350" height="440" resize="scale" quality="best" alt="Soziales Teilen" src="academy/shareable-link-qr-code" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Teilen per QR-Code*</figcaption>
</figure>


Wenn Sie anstelle eines Links einen QR-Code teilen möchten, klicken Sie auf das`QR code`Klicken Sie auf die Schaltfläche und es wird ein neues Fenster mit Ihrem QR-Code angezeigt.

So speichern Sie Ihren QR-Code:

* Wenn Sie auf Ihrem Laptop sind, klicken Sie mit der rechten Maustaste auf das Bild und wählen Sie`Save image`.
* Wenn Sie Ihr Mobiltelefon verwenden:
  * Drücken Sie Ihren Finger nach unten und halten Sie ihn oben auf dem Bild, bis eine Option zum Speichern des Bilds in Ihren Fotos angezeigt wird.
  * Machen Sie einen Screenshot des QR-Codes, schneiden Sie ihn passend zu und speichern Sie ihn in der Foto-App.

### Einbetten

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

So betten Sie Ihren Link als Werbebanner in Ihre Site ein:

* Zeile 3 zeigt Ihnen, wie Sie die Wink-Stile in Ihre Site einbetten.
* Die Zeilen 6 bis 9 zeigen Ihnen, wie Sie die[Wink-Inhaltslader](/developers/web-components/#content-loader)Webkomponente und weisen Sie sie an, ein Werbebanner für Ihren Code abzurufen.
* Zeile 11 zeigt Ihnen, wie Sie unser Javascript in Ihre Site einbetten.
* Zeile 13 zeigt Ihnen, wie Sie die[Wink-App-Loader](/developers/web-components/#application-loader)Webkomponente und weisen Sie sie an, Ihre Konfigurationseinstellungen auf Seitenebene abzurufen.
