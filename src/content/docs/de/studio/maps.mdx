## Erstellen einer Karte

Karten können auf zwei Arten erstellt werden:

* **Einzelmarkierungskarte**: Erstellen Sie eine Karte aus etwas, das Sie gefunden haben, während[Suche](/studio/search).
* **Karte mit mehreren Markierungen**: Erstellen Sie eine Karte aus einer[kuratierte Liste](/studio/curated-lists) oder [gespeicherte Suche](/studio/saved-searches).

### Einzelmarkierungskarte

  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="Aktionen" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

Oben ist ein Bild aus[Suchen](/studio/search)und zeigt Ihnen einige der Dinge, die Sie mit Ihren Suchergebnissen tun können. Eine dieser Aktionen ist`Make a map`.
Klicken Sie auf diese Schaltfläche. Sie werden dann zu unserer Kartenformularseite weitergeleitet, wo Sie mit der Anpassung Ihrer Karte beginnen können.

### Karte mit mehreren Markierungen

Beide[kuratierte Listen](/studio/curated-lists#list-actions) Und [gespeicherte Suchvorgänge](/studio/saved-searches#list-actions)Mit der Aktion „Karte erstellen“ können Sie eine Karte erstellen. Diese Karte enthält Markierungen für alle Bestände in diesen Listen.

## Karte anpassen

Mit dem Formular können Sie Ihre Karte auf folgende Weise anpassen:

<Steps>
  1. Geben Sie ihr einen Namen, damit Sie sich daran erinnern, worum es auf der Karte geht.
  2. Wählen Sie, ob der Benutzer sich auf der Karte bewegen darf.
  3. Legen Sie die gewünschte Höhe der Karte in Pixeln fest.
  4. Wählen Sie, ob der Benutzer die Karte vergrößern und verkleinern dürfen soll.
  5. Wählen Sie einen Kartenstil, der zu der Website passt, in die Sie die Karte einbetten möchten.
  6. Wählen Sie eine Markierungsfarbe.
  7. Wählen Sie die Kartenseite aus, die den Benutzern zuerst angezeigt werden soll. Standardmäßig ist dies die native Seite dieses Inventars.
  8. Sie können Ihre eigenen Markierungen, Kreise, Rechtecke und Polygone hinzufügen, um andere Orte auf der Karte anzuzeigen, die für Ihre Benutzer interessant sein könnten.
  9. Klicken Sie auf das `Save` Schaltfläche, um fortzufahren.
</Steps>


Nachdem Sie Ihre Karte gespeichert haben, werden Sie auf Ihre Kartenseite weitergeleitet und können sie nun mit der Welt teilen.

## Karte teilen

<figure>
  <CloudinaryImage width="350" height="946" resize="scale" quality="best" alt="Karte" src="academy/map" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Ein Karteneintrag mit eingeklapptem Aktionsmenü*</figcaption>
</figure>


Oben sehen Sie ein Bild mit allen verfügbaren Aktionen für Ihre Karte:

* **Aktualisieren**Aktualisiert Ihre Kartenkonfiguration.
* **Zu WinkLinks hinzufügen**Fügt die Karte Ihrem WinkLinks-Konto hinzu.
* **Einbetten**Zeigt Ihnen, wie Sie diese Karte einbetten als[Karte](/developers/web-components/#content-loader)in Ihre Website.
* **Verwendung mit WordPress**Zeigt Ihnen, wie Sie unsere[WordPress-Plugin](/developers/wordpress)um diese Karte in Ihre Website einzubetten.

Einige dieser Optionen werden im Folgenden ausführlicher behandelt.

### Einbetten

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

So betten Sie Ihre Karte in Ihre Site ein:

* Zeile 3 zeigt Ihnen, wie Sie die Wink-Stile in Ihre Site einbetten.
* Die Zeilen 6 bis 9 zeigen Ihnen, wie Sie die[Wink-Inhaltslader](/developers/web-components/#content-loader)Webkomponente und weisen Sie sie an, ein Werbebanner für Ihren Code abzurufen.
* Zeile 11 zeigt Ihnen, wie Sie unser Javascript in Ihre Site einbetten.
* Zeile 13 zeigt Ihnen, wie Sie die[Wink-App-Loader](/developers/web-components/#application-loader)Webkomponente und weisen Sie sie an, Ihre Konfigurationseinstellungen auf Seitenebene abzurufen.
