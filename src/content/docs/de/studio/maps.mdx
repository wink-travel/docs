## Karte erstellen

Karten können auf zwei Arten erstellt werden:

- **Einzelmarker-Karte**: Erstellen Sie eine Karte aus etwas, das Sie beim [Suchen](/de/studio/search) gefunden haben.
- **Mehrfachmarker-Karte**: Erstellen Sie eine Karte aus einer [kuratierten Liste](/de/studio/curated-lists) oder einer [gespeicherten Suche](/de/studio/saved-searches).

Klicken Sie auf diese Schaltfläche, und Sie werden zu unserer Kartenformularseite weitergeleitet, wo Sie mit der Anpassung Ihrer Karte beginnen können.

### Mehrfachmarker-Karte

Sowohl [kuratierte Listen](/de/studio/curated-lists#list-actions) als auch [gespeicherte Suchen](/de/studio/saved-searches#list-actions) verfügen über eine Aktion, mit der Sie eine Karte erstellen können. Diese Karte enthält Kartenmarker für alle Inventare in diesen Listen.

## Karte anpassen

 Das Formular ermöglicht Ihnen, Ihre Karte auf folgende Weise anzupassen:

<Steps>
 1. Geben Sie ihr einen Namen, damit Sie sich erinnern, worum es bei der Karte geht.
 2. Wählen Sie, ob der Benutzer die Karte verschieben darf.
 3. Legen Sie fest, wie hoch die Karte in Pixeln sein soll.
 4. Wählen Sie, ob der Benutzer in die Karte hinein- und herauszoomen darf.
 5. Wählen Sie einen Kartenstil, der zur Website passt, in die Sie die Karte einbetten.
 6. Wählen Sie eine Markerfarbe.
 7. Wählen Sie die anfängliche Kartenansicht, die Benutzer zuerst sehen sollen. Standardmäßig ist dies die native Ansicht des Inventars.
 8. Sie können eigene Marker, Kreise, Rechtecke und Polygone hinzufügen, um andere interessante Orte auf der Karte zu markieren.
 9. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
</Steps>

Nach dem Speichern Ihrer Karte werden Sie zu Ihrer Kartenübersicht weitergeleitet und Ihre Karte ist nun bereit, mit der Welt geteilt zu werden.

Oben sehen Sie ein Bild mit allen verfügbaren Aktionen für Ihre Karte:

- **Update** Aktualisiert Ihre Kartenkonfiguration.
- **Add to WinkLinks** Fügt die Karte Ihrem WinkLinks-Konto hinzu.
- **Embed** Zeigt Ihnen, wie Sie diese Karte als [Map](/de/developers/web-components/#content-loader) in Ihre Website einbetten.
- **Use with WordPress** Zeigt Ihnen, wie Sie unser [WordPress-Plugin](/de/developers/wordpress) verwenden, um diese Karte in Ihre Website einzubetten.

Einige dieser Optionen erläutern wir im Folgenden ausführlicher.

### Einbetten

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

So betten Sie Ihre Karte in Ihre Website ein:

- Zeile 3 zeigt, wie Sie die Wink-Styles in Ihre Website einbinden.
- Die Zeilen 6 bis 9 zeigen, wie Sie die [wink-content-loader](/de/developers/web-components/#content-loader) Web-Komponente verwenden und ihr mitteilen, ein Werbebanner für Ihren Code zu laden.
- Zeile 11 zeigt, wie Sie unser Javascript in Ihre Website einbinden.
- Zeile 13 zeigt, wie Sie die [wink-app-loader](/de/developers/web-components/#application-loader) Web-Komponente einbinden und ihr mitteilen, Ihre Seitenkonfigurationspräferenzen zu laden.
