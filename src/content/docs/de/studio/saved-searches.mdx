Gespeicherte Suchen machen, was der Name schon sagt. Während Sie nach Reiseangeboten suchen, haben Sie die Möglichkeit, Ihre Suchanfrage für die spätere Verwendung zu speichern.

## Speichern Ihrer Suchanfrage

Wenn Sie mit der Inventarsuche beginnen (siehe[Suchen](/studio/search)), können Sie Ihre Suchfilter verfeinern.
Am Ende des Abfrageformulars finden Sie die Option`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Suche speichern" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Zeigt den Link zum Speichern der Suchanfrage an*</figcaption>
</figure>


Wenn Sie auf diesen Link klicken, haben Sie die Möglichkeit, dieser Suche einen Namen zu geben (*z. B. Meine Bibliothek der magischen Hotels*) etwas, an das Sie sich später erinnern werden.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Geben Sie Ihrer Suche einen Namen" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Geben Sie Ihrer Suche einen Namen*</figcaption>
</figure>


Geben Sie ihm einen Namen und klicken Sie auf das`Save search`Schaltfläche. Anschließend können Sie zu`Inventory > Saves Searches`aus der Hauptnavigationsleiste, um Ihren neu erstellten Datensatz anzuzeigen.

## Listenaktionen

Ihre gespeicherte Suche kann verwendet werden, um Ihren Benutzern Reiseangebote in Form eines Rasters oder einer Karte zugänglich zu machen und Ihre Suchergebnisse nach Ihren Wünschen anzuzeigen.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Gespeicherte Suchaktionen" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Gespeicherte Suchanfrage ohne angewendete Filter*</figcaption>
</figure>


Für jede gespeicherte Suchanfrage gibt es Aktionen, die Sie ausführen können:

* Konvertieren Sie die Suchanfrage in eine[Karte](/studio/maps).
* Konvertieren Sie die Suchanfrage in eine[Netz](/studio/grids).
* Entfernen Sie die gespeicherte Suche.

## API

Entwickler, die gespeicherte Suchen verwalten möchten, können zu[Entwickler > API > Durchsuchen](/developers/apis/#browse-api).
