Gespeicherte Suchen machen genau das, was der Name sagt. Während Sie nach Reiseinventar suchen, haben Sie die Möglichkeit, Ihre Suchanfrage für die spätere Verwendung zu speichern.

## Speichern Ihrer Suchanfrage

Wenn Sie mit der Suche nach Inventar beginnen (siehe [Search](/de/studio/search) ), können Sie Ihre Suchfilter feinjustieren.
Am unteren Rand des Suchformulars befindet sich die Option `Save search`.

Wenn Sie auf diesen Link klicken, haben Sie die Möglichkeit, dieser Suche einen Namen zu geben (*z. B. Meine Bibliothek magischer Hotels*), den Sie sich später merken können.

<figure>
<CloudinaryImage

    <figcaption>*Geben Sie Ihrer Suche einen Namen*</figcaption>
</figure>

Geben Sie einen Namen ein und klicken Sie auf die Schaltfläche `Save search`. Danach können Sie im Hauptnavigationsmenü unter `Inventory > Saves Searches` Ihren neu erstellten Eintrag sehen.

## Listenaktionen

Ihre gespeicherte Suche kann verwendet werden, um Reiseinventar Ihren Nutzern in Form eines Rasters oder einer Karte anzuzeigen und Ihre Suchergebnisse so darzustellen, wie Sie es wünschen.

<figure>
<CloudinaryImage
      width="700"
      height="939"

    <figcaption>*Gespeicherte Suchanfrage ohne angewendete Filter*</figcaption>
</figure>

Jede gespeicherte Suchanfrage bietet Aktionen, die Sie ausführen können:

- Konvertieren Sie die Suchanfrage in eine [Karte](/de/studio/maps).
- Konvertieren Sie die Suchanfrage in ein [Raster](/de/studio/grids).
- Entfernen Sie die gespeicherte Suche.

## API

Entwickler, die gespeicherte Suchen verwalten möchten, können zu [Developers > API > Browse](/de/developers/apis/#browse-api) wechseln.
