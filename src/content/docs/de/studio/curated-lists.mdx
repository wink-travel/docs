Kuratierte Listen sind eine Funktion, mit der Sie ausgewählte Reiseangebote gruppieren und sortieren können.
Nach der Erstellung können Sie die Liste Ihren Nutzern zum Verkauf anbieten.[Netz](/studio/grids)oder ein[Karte](/studio/maps) durch [WinkLinks](/link-manager/wink-links), unser [WordPress-Plugin](/developers/wordpress/)oder durch Einbettung eines unserer[Webkomponenten](/developers/web-components)in Ihre Site.

Standardmäßig beginnt jedes Affiliate-Konto mit einem`Favorites`Liste, die Sie sofort verwenden können.`Favorites`ist speziell und kann nicht entfernt werden.
Alle kuratierten Listen ermöglichen Ihnen Folgendes:

## Erstellen einer Liste

<figure>
  <CloudinaryImage width="350" height="219" resize="scale" quality="best" alt="Kuratierte Liste erstellen" src="academy/create-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Erstellen Sie ein neues kuratiertes Listenformular*</figcaption>
</figure>


* Klicken Sie im Dashboard auf`Inventory > Curated Lists`in der Navigationsleiste.
* Erstellen Sie eine neue kuratierte Liste, indem Sie auf das`Make a new list` Taste.
* Sie werden aufgefordert, Ihrer Liste einen Namen zu geben.*Beispiel: Top 3 NYC Picks*.
* Klicken Sie auf das`Save` Taste.

## Hinzufügen eines Listenelements

Sie können aus den Suchergebnissen ein Listenelement zu jeder Liste hinzufügen.

* Gehe zu[Suchen](/studio/search)
* Suchen Sie nach etwas, das Sie Ihrer Liste hinzufügen möchten.
* Wenn Sie es gefunden haben, klicken Sie auf das`Actions`Link unten links auf der Suchergebniskarte.
* Klicken Sie auf`Add to...`und wählen Sie dann die kuratierte Liste aus, zu der Sie dieses Inventar hinzufügen möchten.
* Gehen Sie zurück zur Seite mit den kuratierten Listen, um zu überprüfen, ob der Artikel zu Ihrer Liste hinzugefügt wurde.

## Kopieren eines Listenelements

<figure>
  <CloudinaryImage width="450" height="178" resize="scale" quality="best" alt="In kuratiertes Listenelement kopieren" src="academy/copy-to-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

Sie können ein Listenelement kopieren, indem Sie auf das Kopiersymbol klicken und die Liste auswählen, in die Sie das Element kopieren möchten.

## Entfernen eines Listenelements

<figure>
  <CloudinaryImage width="350" height="216" resize="scale" quality="best" alt="Kuratiertes Listenelement entfernen" src="academy/remove-curated-list-item" errorImage="noimage_opaque_nyrtl0.png" />

Sie können ein Listenelement entfernen, indem Sie auf das Papierkorbsymbol klicken und den Löschvorgang bestätigen.

## Listenaktionen

<figure>
  <CloudinaryImage width="350" height="183" resize="scale" quality="best" alt="Kuratierte Listenaktionen" src="academy/curated-list-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Kuratierte Listenaktionen*</figcaption>
</figure>


Jede Liste verfügt über einige Aktionen, die Sie darauf ausführen können.

* Konvertieren Sie die Liste in eine[Karte](/studio/maps).
* Konvertieren Sie die Liste in eine[Netz](/studio/grids).
* Entfernen Sie die Liste (*außer Favoriten*).
