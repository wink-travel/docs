Kuratierte Listen ist eine Funktion, mit der Sie Inventar gruppieren und sortieren können, das Sie aus den Ergebnissen Ihrer Suche nach Reiseinventar ausgewählt haben.
Einmal erstellt, können Sie die Liste Ihren Nutzern zugänglich machen, indem Sie sie als [Grid](/de/studio/grids) oder [Karte](/de/studio/maps) über [WinkLinks](/de/link-manager/wink-links), unser [WordPress-Plugin](/de/integrations/wordpress/) oder durch Einbetten eines unserer [Web Components](/de/integrations/web-components) auf Ihrer Website zum Verkauf anbieten.

Standardmäßig beginnt jedes Affiliate-Konto mit einer `Favoriten`-Liste, die Sie sofort verwenden können. `Favoriten` ist speziell und kann nicht entfernt werden.
Alle kuratierten Listen ermöglichen Ihnen Folgendes:

## Eine Liste erstellen

- Klicken Sie im Dashboard in der Navigationsleiste auf `Inventar > Kuratierte Listen`.
- Erstellen Sie eine neue kuratierte Liste, indem Sie auf die Schaltfläche `Neue Liste erstellen` klicken.
- Sie werden aufgefordert, Ihrer Liste einen Namen zu geben. *Beispiel: Top 3 NYC Picks*.
- Klicken Sie auf die Schaltfläche `Speichern`.

## Ein Listenelement hinzufügen

Sie können ein Listenelement aus den Suchergebnissen zu jeder Liste hinzufügen.

- Gehen Sie zu [Suche](/de/studio/search)
- Suchen Sie nach etwas, das Sie zu Ihrer Liste hinzufügen möchten.
- Sobald Sie es gefunden haben, klicken Sie unten links auf der Suchergebnis-Karte auf den Link `Aktionen`.
- Klicken Sie auf `Hinzufügen zu...` und wählen Sie dann die kuratierte Liste aus, zu der Sie dieses Inventar hinzufügen möchten.
- Gehen Sie zurück zur Seite der kuratierten Listen, um zu überprüfen, ob das Element zu Ihrer Liste hinzugefügt wurde.

Jede Liste bietet einige Aktionen, die Sie ausführen können.

- Die Liste in eine [Karte](/de/studio/maps) umwandeln.
- Die Liste in ein [Grid](/de/studio/grids) umwandeln.
- Die Liste entfernen (*außer Favoriten*).

## API

Entwickler, die kuratierte Listen verwalten möchten, können zu [Entwickler > API > Durchsuchen](/de/developers/apis/#browse-api) gehen.
