Studio bietet mehrere einbettbare Widgets, die Sie auf Ihrer eigenen Website hinzufügen können, um Buchungen zu fördern.

Um auf die Embed-Widgets zuzugreifen, navigieren Sie im Seitenmenü zu `Tools > Miscellaneous`.

## Verfügbare Widgets

| Widget | Beschreibung |
| -- | -- |
| **Account button** | Ein gebrandeter Button, der zu Ihrer Affiliate-Seite verlinkt |
| **Itinerary button** | Ein Button, der den Reiseplan-Auswahl öffnet |
| **Itinerary form** | Ein Inline-Formular zur Auswahl von Datum und Gästen |
| **Search button** | Ein Button, der die Unterkunftssuche öffnet |
| **Lookup button** | Ein Button, um spezifische Inventare abzurufen |

## So betten Sie ein

Jedes Widget stellt einen Code-Snippet bereit, den Sie kopieren und in den HTML-Code Ihrer Website einfügen können. Der Snippet enthält Ihre **Client ID** und **Customization ID**, sodass das Widget automatisch mit Ihrem Affiliate-Konto verknüpft ist.

:::note
Wenn Sie WordPress verwenden, bieten wir auch ein WordPress-Plugin für eine einfachere Integration an.
:::
