Ein Raster stellt eine Liste von Lieferanten und Beständen dar und zeigt die Details über eine interaktive, buchbare Raster-Benutzeroberfläche an.
Das Raster unterstützt die Anzeige aller Bestände aus Ihrer kuratierten Liste oder gespeicherten Suche.
Ein Nutzer interagiert mit einem Raster genauso wie mit einem einzelnen[Karte](/studio/cards)mit einer zusätzlichen Funktion:

1. Blättern Sie durch das Raster, indem Sie auf das`Show more` Taste (*wenn mehr Artikel verfügbar sind*).

  <figcaption>*Raster mit mehreren Zimmertypen*</figcaption>

Oben sehen Sie ein Beispiel unseres Rasters mit einer Liste von Zimmertypkarten.

Der Rest dieses Artikels führt Sie durch die Erstellung, Anpassung und Bereitstellung des Rasters für Ihre Benutzer.

## Rastertypen

Es gibt drei Arten von Rastern:

1. Raster basierend auf einem[kuratierte Liste](/studio/curated-lists#list-actions).
2. Raster basierend auf einem[gespeicherte Suche](/studio/saved-searches#list-actions).
3. Raster basierend auf einem Standort und Sortierkriterien (*d. h. Ranglistenraster*).

## Kuratiertes Listenraster

Dies ist ein Raster, das den Bestand verwendet, den Sie in einer Ihrer kuratierten Listen gesammelt haben, und die Liste in buchbaren Reisebestand umwandelt, den Sie Ihren Benutzern zeigen können.

### Erstellen

Navigieren Sie zu`Inventory > Curated Lists`aus der Hauptnavigationsleiste. Für dieses Beispiel verwenden wir Ihre`Favorites`Liste.
Wenn Sie Ihrer Liste noch nichts hinzugefügt haben`Favorites`gehen Sie zu[Suchen](/studio/search)um zu lernen wie.

<figure>
  <CloudinaryImage width="350" height="394" resize="scale" quality="best" alt="Erstellen Sie ein kuratiertes Listenraster" src="academy/create-curated-list-grid" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Erstellen Sie ein kuratiertes Listenrasterformular*</figcaption>
</figure>


<Steps>
  1. Klicken Sie auf das `Actions` Schaltfläche auf der `Favorites` Liste.
  2. Klicken Sie auf das `Create a grid` Taste.
  3. Es wird ein neues Fenster angezeigt, in dem Sie Ihrem Raster einen Namen geben können. *Siehe unten*.
  4. Klicken `OK` Schaltfläche, um fortzufahren.
</Steps>


Ihr Raster wurde erstellt. Navigieren Sie zu`Tools > Grids`Klicken Sie in der Hauptnavigationsleiste auf das`Curated Grids`Registerkarte, um Ihr neues Raster anzuzeigen.

### Anpassen

Mit dem Formular können Sie Ihr Raster auf folgende Weise anpassen:

<Steps>
  1. Geben Sie ihm einen Namen, damit Sie sich daran erinnern, worum es in dem Raster geht.
  2. Wählen Sie die [Anpassung](/studio/customization) Sie auf dieses Raster anwenden möchten.
  3. Wählen Sie die Kartenseite aus, die den Benutzern zuerst angezeigt werden soll. Standardmäßig ist dies die native Seite dieses Inventars.
  4. Wählen Sie das Badge aus, das Sie auf der Karte verwenden möchten. Mit einem Badge können Benutzer den Bestand anhand einer aggregierten Metrik vergleichen, z. B. `eco-friendly`.
  5. Fügen Sie durch Kommas getrennte Schlüsselwörter hinzu, die verwendet werden von [Webcrawler](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
  6. Fügen Sie Titel und Beschreibungen in den Sprachen hinzu, die der Benutzer sehen soll.
  7. Klicken Sie auf das `Save` Schaltfläche, um fortzufahren.
</Steps>

Einige dieser Optionen werden im Folgenden ausführlicher behandelt.

```html {3, 6-9, 11, 13-16}
<html>

  </head>

       layout="LIST"
       id="9a212b5e-62a7-11ef-ac3f-42004e494300"
    ></wink-content-loader>

    <wink-app-loader

      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"

  </body>
</html>
```

So betten Sie Ihr Raster in Ihre Site ein:

* Zeile 3 zeigt Ihnen, wie Sie die Wink-Stile in Ihre Site einbetten.
* Die Zeilen 6 bis 9 zeigen Ihnen, wie Sie die[Wink-Inhaltslader](/developers/web-components/#content-loader)Webkomponente und weisen Sie sie an, ein Raster für Ihren Code abzurufen.
* Zeile 11 zeigt Ihnen, wie Sie unser Javascript in Ihre Site einbetten.
* Zeile 13 zeigt Ihnen, wie Sie die[Wink-App-Loader](/developers/web-components/#application-loader)Webkomponente und weisen Sie sie an, Ihre Konfigurationseinstellungen auf Seitenebene abzurufen.

## Gespeichertes Suchraster

Dies ist ein Raster, das den Bestand in Ihrer gespeicherten Suchanfrage verwendet und die Suchergebnisse in buchbaren Reisebestand umwandelt, den Sie Ihren Benutzern anzeigen können.

### Erstellen

Navigieren Sie zu`Inventory > Saved searches`aus der Hauptnavigationsleiste.
Wenn Sie noch keine gespeicherte Suche erstellt haben, gehen Sie zu[Suchen](/studio/search)um zu lernen wie.

<figure>
  <CloudinaryImage width="350" height="394" resize="scale" quality="best" alt="Gespeichertes Suchraster erstellen" src="academy/create-saved-search-grid" errorImage="noimage_opaque_nyrtl0.png" />

</figure>

</Steps>


4. Klicken Sie auf`OK`Schaltfläche, um fortzufahren.

Ihr Raster wurde erstellt. Navigieren Sie zu`Tools > Grids`Klicken Sie in der Hauptnavigationsleiste auf das`Saved Search Grids`Registerkarte, um Ihr neues Raster anzuzeigen.

## Rangliste

### Erstellen

Erstellen Sie ein Ranglistenraster, indem Sie zu`Tools > Grids`und klicken Sie auf das`Ranked Grid`Klicken Sie auf die`Create ranked grid` Taste.

### Anpassen

Mit dem Formular können Sie Ihr Rangraster auf folgende Weise anpassen:

<Steps>
  1. Wählen Sie ein Ziel. *Z. B. Tokio*.
  2. Geben Sie ihm einen Namen, damit Sie sich daran erinnern, worum es in dem Raster geht. *Z. B. Umweltfreundliche Hotels in Tokio*
  3. Wählen Sie die [Anpassung](/studio/customization) Sie auf dieses Raster anwenden möchten.
  4. Wählen Sie die Kartenseite aus, die den Benutzern zuerst angezeigt werden soll. Standardmäßig ist dies die native Seite dieses Inventars.
  5. Wählen Sie aus, nach welchem Merkmal die Eigenschaften sortiert werden sollen. *Z.B. Umweltfreundlichkeit*.
  6. Fügen Sie durch Kommas getrennte Schlüsselwörter hinzu, die verwendet werden von [Webcrawler](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
  7. Fügen Sie Titel und Beschreibungen in den Sprachen hinzu, die der Benutzer sehen soll.
  8. Klicken Sie auf das `Save` Schaltfläche, um fortzufahren.
</Steps>


Nachdem Sie Ihr Ranglistenraster gespeichert haben, werden Sie auf Ihre Ranglistenraster-Seite weitergeleitet und Ihr Raster kann nun mit der Welt geteilt werden.
