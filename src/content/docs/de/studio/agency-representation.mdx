Eine Agentur ist ein normales Wink-Konto, das Sie Ihrem Konto zu Vertretungszwecken zuweisen können. Nachfolgend finden Sie einige Gründe, warum Sie sich von einer Agentur vertreten lassen sollten:

* Vollständige/teilweise Verwaltung Ihres Wink-Kontos durch Dritte.
* Talent hat eine bestehende Vereinbarung mit seiner Agentur und möchte, dass Wink den finanziellen Aspekt der Abwicklung der über Wink eingehenden Gelder übernimmt.
* Die Talentagentur erledigt die gesamte Online-Arbeit und verwaltet alle Konten auf Wink.

## Agentur zuweisen

So weisen Sie Ihrem Konto eine Agentur zu.

<Steps>
  1. Melden Sie sich in Ihrem Benutzerkonto an unter [Wink Studio](https://studio.wink.travel).
  2. Wählen Sie das Partnerkonto aus, mit dem Sie arbeiten möchten.
  3. Klicken Sie auf `Account > Managers` aus der Hauptnavigationsleiste.
  4. Im `Agency` Geben Sie im Abschnitt den Namen der Agentur ein, die Sie vertreten soll.
  5. Wählen Sie die Agenturgebühr aus der Dropdown-Liste.
  6. Fügen Sie optional Ablaufregeln hinzu.
  7. Klicken Sie auf das `Set agency` Schaltfläche, um fortzufahren.
</Steps>


:::Hinweis
Um eine Agentur zuzuweisen, müssen Sie Inhaber des Kontos sein.
:::

:::Tipp \[Beispiel]
Die Agenturgebühr ergibt sich aus der Affiliate-Provision.

Buchung = 100 €, Provision = 10 %, Agenturgebühr = 10 % => 100 \* 0,1 \* 0,1 = 1 €
:::

## Agenturzuweisung aufheben

So heben Sie die Zuweisung einer Agentur zu Ihrem Konto auf.

<Steps>
  1. Melden Sie sich in Ihrem Benutzerkonto an unter [Wink Studio](https://studio.wink.travel).
  2. Wählen Sie das Partnerkonto aus, mit dem Sie arbeiten möchten.
  3. Klicken Sie auf `Account > Managers` aus der Hauptnavigationsleiste.
  4. Im `Agency` klicken Sie auf das <Badge text="x" variant="danger" size="small" /> Symbol auf der rechten Seite der Dropdown-Liste „Agentur“.
  5. Klicken Sie auf das `Set agency` Schaltfläche, um fortzufahren.
</Steps>
