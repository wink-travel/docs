Eine Agentur ist ein normales Wink-Konto, das Sie Ihrem Konto für Vertretungszwecke zuweisen können. Nachfolgend einige Gründe, warum Sie eine Agentur als Vertreter wünschen könnten:

- Vollständige / teilweise Verwaltung Ihres Wink-Kontos durch Dritte.
- Talent hat eine bestehende Vereinbarung mit seiner Agentur und möchte, dass Wink den finanziellen Aspekt der Abwicklung der über Wink eingehenden Gelder übernimmt.
- Die Talentagentur erledigt alle Online-Arbeiten und verwaltet alle Konten auf Wink.

## Agentur zuweisen

So weisen Sie eine Agentur Ihrem Konto zu.

<Steps>
 1. Melden Sie sich in Ihrem Benutzerkonto auf [Wink Studio](https://studio.wink.travel) an.
 2. Wählen Sie das Partnerkonto aus, mit dem Sie arbeiten möchten.
 3. Klicken Sie in der Hauptnavigationsleiste auf `Account > Managers`.
 4. Geben Sie im Abschnitt `Agency` den Namen der Agentur ein, die Sie vertreten soll.
 5. Wählen Sie die Agenturgebühr aus dem Dropdown-Menü.
 5. Optional können Sie Ablaufregeln hinzufügen.
 7. Klicken Sie auf die Schaltfläche `Set agency`, um fortzufahren.
 </Steps>

 :::note
Sie müssen Eigentümer des Kontos sein, um eine Agentur zuzuweisen.
 :::

 :::tip[Beispiel]
Die Agenturgebühr ergibt sich aus der Partnerprovision.

Buchung = 100 $, Provision = 10 %, Agenturgebühr = 10 % => 100 * 0,1 * 0,1 = 1 $
 :::

## Agentur entfernen

So entfernen Sie eine Agentur von Ihrem Konto.

<Steps>
 1. Melden Sie sich in Ihrem Benutzerkonto auf [Wink Studio](https://studio.wink.travel) an.
 2. Wählen Sie das Partnerkonto aus, mit dem Sie arbeiten möchten.
 3. Klicken Sie in der Hauptnavigationsleiste auf `Account > Managers`.
 4. Klicken Sie im Abschnitt `Agency` auf das <Badge text="x" variant="danger" size="small" /> Symbol rechts neben dem Agentur-Dropdown.
 5. Klicken Sie auf die Schaltfläche `Set agency`, um fortzufahren.
 </Steps>


## API

Entwickler, die die Agenturverwaltung über die API steuern möchten, können zu [Developers > API > Affiliate](/de/developers/apis/#affiliate-api) wechseln.
