Der **Sicherheit**-Tab unter [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) ist der Ort, an dem Sie **Passkeys** verwalten — WebAuthn-Anmeldeinformationen, mit denen Sie sich ohne Passwort anmelden können.

## Was ist ein Passkey?

Ein Passkey ist eine kryptografische Anmeldeinformation, die an Ihr Gerät (oder Ihren Passwortmanager) gebunden ist und beweist, dass Sie Sie sind. Passkeys sind phishing-resistent und schneller als das Eintippen eines Passworts.

## Registrierung eines Passkeys

1. Melden Sie sich mit Ihren bestehenden Zugangsdaten bei [iam.wink.travel](https://iam.wink.travel) an.
2. Gehen Sie zu **Mein Konto → Sicherheit**.
3. Klicken Sie auf `Passkey hinzufügen`.
4. Ihr Browser fordert Sie auf, sich mit Ihrem Gerät zu authentifizieren (Touch ID, Face ID, Windows Hello, einem Hardware-Schlüssel oder einem synchronisierten Passwortmanager).
5. Geben Sie dem Passkey einen freundlichen Namen (z. B. *Arbeits-MacBook*), damit Sie ihn später erkennen können.

## Anmeldung mit einem Passkey

Geben Sie auf dem Anmeldebildschirm von [iam.wink.travel](https://iam.wink.travel) Ihre E-Mail-Adresse ein — wenn für diese E-Mail ein Passkey registriert ist, werden Sie aufgefordert, sich damit zu authentifizieren, anstatt ein Passwort einzugeben.

## Entfernen eines Passkeys

Im selben Sicherheitstab hat jeder registrierte Passkey eine Schaltfläche `Entfernen`. Verwenden Sie diese, wenn Sie ein Gerät verlieren, einen Hardware-Schlüssel außer Betrieb nehmen oder nicht mehr genutzte Anmeldeinformationen bereinigen möchten.

:::caution
Wenn Sie alle Ihre Passkeys entfernen, bleibt das Passwort die einzige Anmeldemethode. Stellen Sie sicher, dass Sie Ihr Passwort kennen, bevor Sie den letzten Passkey entfernen.
:::
