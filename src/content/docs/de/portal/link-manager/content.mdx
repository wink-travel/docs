Ihre Inhalte finden Sie unter dem Reiter `Einträge`.

Wir unterstützen drei Haupttypen von Inhalten:

- Öffentlich zugängliche URLs, die auf Webseiten verlinken.
- Dateien, die über WinkLinks hochgeladen wurden.
- Textinhalte, die über unsere Funktion `Zitate` erstellt wurden.

## Inhalt hinzufügen

So fügen Sie Inhalte zu WinkLinks hinzu. Unter dem Reiter `Einträge` klicken Sie zunächst auf die Schaltfläche `Eintrag hinzufügen`.

### Normaler Link

:::tip
Der Link muss auf eine Webseite führen. Zum Beispiel können Sie nicht direkt auf eine PDF verlinken. Wenn Sie eine PDF teilen möchten, müssen Sie die Funktion `Datei hochladen` verwenden.
:::

Hier sind die Schritte, um einen normalen Link hinzuzufügen:

<Steps>
1. Geben Sie den Link ein, den Sie teilen möchten. *z. B. https://trvl.as/3xWCH*
2. Sobald Sie den Link eingegeben haben, sucht Wink nach Details zu dieser Webseite und füllt das folgende Formular aus.
3. Sie können einige der Daten ändern, wie z. B. den Seitennamen, Titel und Beschreibung.
4. Sie können auch ein benutzerdefiniertes Bild hochladen.
5. Klicken Sie auf die Schaltfläche `Speichern`, wenn Sie bereit sind, fortzufahren.
</Steps>

### Datei hochladen

:::caution[Anforderungen]
- Bilder dürfen 10 MB nicht überschreiten.
- Videos dürfen 50 MB nicht überschreiten.
- Wir unterstützen alle anerkannten Bild- und Videoformate, einschließlich PDF.
:::

:::note[Empfehlungen]
- 2560w x 1440h (16:9 Seitenverhältnis) für Ihr Titelbild.
- 2560w x 1920h (4:3 Seitenverhältnis) für alle anderen Bilder.
- Die Bildgröße sollte 5 MB nicht überschreiten. Bitte komprimieren Sie Ihre Bilder vor dem Hochladen auf Wink.
:::

Hier sind die Schritte zum Hochladen einer Datei:

<Steps>
1. Medien können hinzugefügt werden:
    - Indem Sie Bilder von Ihrem Computer in das Fenster ziehen oder auf `Durchsuchen` klicken.
    - Indem Sie eine externe URL einfügen, die auf ein Bild verweist.
    - Indem Sie Ihre Laptop-Kamera verwenden.
    - Von Ihrem Google Drive-Konto.
    - Von Ihrem Dropbox-Konto.
    - Von Shutterstock
    - Von gettyimages
    - Von iStock
    - Von Unsplash
2. Sobald der Upload abgeschlossen ist, können Sie weitere Metadaten hinzufügen, um Ihre Datei zu beschreiben, z. B. Titel und Beschreibung.
3. Klicken Sie auf die Schaltfläche `Speichern`, wenn Sie bereit sind, fortzufahren.
</Steps>

### Ein Zitat schreiben

:::tip
Ein Zitat ist eine großartige Möglichkeit, eine lange Liste zu unterbrechen [...und es lässt Sie auch klug und kenntnisreich wirken 😉].
:::

Hier sind die Schritte, um ein Zitat zu schreiben:

<Steps>
1. Geben Sie das Zitat ein *z. B. Rosen sind rot. Veilchen sind blau...*.
2. Geben Sie den Namen des ursprünglichen Autors ein. *z. B. Edmund Spenser*
3. Geben Sie den Namen der Seite ein, von der Sie das Zitat haben. *z. B. Daily Quotes*
4. Klicken Sie auf die Schaltfläche `Speichern`, wenn Sie bereit sind, fortzufahren.
</Steps>

## Inhalt aktualisieren

Alle Inhalte können einfach bearbeitet werden, indem Sie auf den Link `Aktionen` klicken und `Aktualisieren` auswählen. Inhalte können auf ähnliche Weise entfernt werden, indem Sie auf die Schaltfläche `Entfernen` klicken.

## Inhalt entfernen

Alle Inhalte können einfach bearbeitet werden, indem Sie `Aktualisieren` auswählen. Inhalte können auf ähnliche Weise entfernt werden, indem Sie zuerst auf den Link `Aktionen` klicken und dann die Schaltfläche `Entfernen` auswählen.

## Inhalte sortieren

Sie können Ihre Inhalte sortieren, indem Sie auf das Symbol in der unteren linken Ecke jedes Eintrags klicken und es an die gewünschte Position ziehen.

## Inhalte gruppieren

:::tip
Erstellen Sie Ihre Tags, bevor Sie Ihre Einträge erstellen. Andernfalls verlieren Sie Ihren Fortschritt und müssen von vorne beginnen.
:::

Wenn Sie einen Eintrag bearbeiten, haben Sie die Möglichkeit, einen oder mehrere Tags zur Gruppierung hinzuzufügen. Scrollen Sie zum Ende des Formulars, um das Formularfeld `Tags` zu finden.

Nutzer, die Ihre WinkLinks-Seite besuchen, können wählen, wie sie Ihre Inhalte konsumieren möchten (entweder Liste oder Raster) und können ein oder mehrere Tags auswählen, um Ihre Inhalte weiter zu filtern.

## API

Entwickler, die `WinkLinks` verwalten möchten, können zu [Developers > API > WinkLinks](/de/api) gehen.
