Sie können über jedes unserer Portale ein Benutzerkonto erstellen. Nach der Erstellung kann dieses Konto in allen Apps und Websites von Wink verwendet werden.

## Reisende

Gehen Sie folgendermaßen vor:

<Steps>
  1. Gehen Sie weiter zu [https://ota.wink.travel](https://ota.wink.travel)
  2. Klicken Sie auf `Sign-In / Register` in der oberen rechten Ecke
  3. Sie werden aufgefordert, grundlegende Benutzerdaten einzugeben. Füllen Sie alle erforderlichen Felder aus und klicken Sie auf `Save`.
  4. Sie können sich jetzt mit Ihrem Benutzernamen (E-Mail) und Passwort bei Wink anmelden.
  5. Sobald Sie angemeldet sind, leitet Sie das System dorthin zurück, wo Sie hergekommen sind.
</Steps>

## Anmeldung

Mit einem Wink-Benutzerkonto erhalten Sie Zugriff auf alle Funktionen der Plattform. Sie können unbegrenzt viele Immobilien und Partnerkonten erstellen.

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="Anmelden" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Anmeldeformular*</figcaption>
</figure>


<Steps>
  1. Wenn Sie zum ersten Mal zu unserem Autorisierungsserver weitergeleitet werden, werden Sie aufgefordert, Ihren Benutzernamen und Ihr Passwort einzugeben.
  2. Klicken Sie stattdessen auf `Create account` Taste.
  3. Füllen Sie alle erforderlichen Felder aus und klicken Sie auf `Register` , um fortzufahren. Dadurch gelangen Sie zurück zum Anmeldeformular.
  4. Geben Sie Ihre E-Mail-Adresse/Ihren Benutzernamen und Ihr Passwort ein, um sich anzumelden.
  5. Sie werden zurück zur Site umgeleitet, von der Sie gekommen sind.
</Steps>
