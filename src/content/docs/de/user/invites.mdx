Ein Wink-Benutzer kann Ihnen die Verwaltung eines seiner Immobilien-, Partner- oder Reisebürokonten überlassen. In diesem Fall erhalten Sie eine E-Mail mit einer Einladungsanfrage.

import { Steps } from '@astrojs/starlight/components';

Um Ihre aktuellen Wink-Einladungen anzuzeigen, folgen Sie diesen Schritten:

<Steps>
  1. Sie müssen auf einer unserer Websites angemeldet sein.
  2. Klicken Sie oben rechts auf Ihr Profilsymbol. Ein Menü wird angezeigt.
  3. Klicken Sie auf das `Invites` Link.
  4. Interagieren Sie mit Ihren Einladungen, indem Sie entweder `Accepting` oder `Rejecting` ihnen.
</Steps>
