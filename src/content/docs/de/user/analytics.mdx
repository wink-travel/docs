Wissen ist das Lebenselixier von Wink. Es zeigt Ihnen stundengenau, wie jeder und alles auf Wink abschneidet. Vor Wink mussten Hotels einen Drittanbieter abonnieren und viel Geld bezahlen, um auf das zuzugreifen, was das Revenue-Management-Team als … bezeichnet.`the comp set`; was übersetzt bedeutet:`How does my property compare with another property?`.

Mit Wink haben Sie kostenlosen Zugriff auf plattformweite Analysen. So können Sie die Leistung Ihrer Partner, Reisebüros oder Unterkünfte anhand beliebiger Kennzahlen verfolgen – und das so lange Sie möchten. Sie können auch regionale Trends einsehen und sehen, welche Produkte wann und wo verkauft werden. In den richtigen Händen können diese Erkenntnisse die Verkaufsfähigkeiten jedes Wink-Nutzers verbessern.

## Zugriff auf Diagramme

Sie können von jeder unserer Websites aus auf Ihre Diagramme zugreifen. Gehen Sie dazu folgendermaßen vor:

<Steps>
  1. Sie müssen auf einer unserer Websites angemeldet sein.
  2. Klicken Sie oben rechts auf Ihr Profilsymbol. Ein Menü wird angezeigt.
  3. Klicken Sie auf das `Analytics` Link.
  4. Sie werden zu Ihren Diagrammen weitergeleitet.
</Steps>


:::Hinweis
Standardmäßig erstellen wir einfache Diagramme für jedes von Ihnen erstellte Unterkunfts-, Reisebüro- und Partnerkonto, damit Sie die Leistung Ihrer eigenen Konten leicht verfolgen können.
:::

## Erstellen eines Diagramms

Wenn Sie mit der Erstellung Ihrer eigenen Diagramme beginnen möchten, gehen Sie wie oben beschrieben zu Ihren Diagrammen und klicken Sie auf`Create chart` Taste.

Nachfolgend finden Sie die einfachen Schritte zum Erstellen eines Analysediagramms, gefolgt von ausführlicheren Anweisungen.

<Steps>
  1. **Zeitreihen** Wählen Sie eine Zeitreihe für Ihr Diagramm. *Z. B. Täglich*.
  2. **Tracking-Typ** Wählen Sie, ob Sie einen festen Datumsbereich oder kontinuierlich verfolgen möchten. *Z.B. `Continuous`*.
     * Kontinuierlich: Wählen Sie Zeiteinheiten aus. *Beispiel: 7 Tage*
     * Behoben: Datums-/Zeitbereich eingeben *Z.B. 22. Sept. 2023 16:00 - 29. Sept. 2023 16:00*.
  3. **Anzeigewährung** Wenn Sie in Ihrem Diagramm mit Beträgen arbeiten, wählen Sie aus, in welchen Währungen Sie diese anzeigen möchten. *Z. B. USD*.
  4. **Track-Eigenschaft** Verfolgen Sie eine bestimmte Eigenschaft. *Optional*
  5. **Vertriebskanal verfolgen** Verfolgen Sie den Standort der Vertriebskanäle nach Kontinent, Land oder Stadt. *Optional*
  6. **Vertriebskanaltyp filtern**Filtern Sie nach einem bestimmten Typ von Vertriebskanal. *Z.B. Influencer*. *Optional*
  7. **Verfolgen Sie bestimmte Vertriebskanäle** Verfolgen Sie einen bestimmten Vertriebskanal. *Optional*
  8. **Track-Reisender** Verfolgen Sie den Standort von Reisenden nach Kontinent, Land oder Stadt. *Optional*
  9. **Datenpunkte auswählen** Wählen Sie die Datenpunkte aus, die Sie verfolgen möchten. *Z. B. Buchungen und Stornierungen*
  10. **Gruppendaten** Anstatt nach Eigenschaften oder Vertriebskanälen zu filtern, können Sie auch Gruppierungen vornehmen, um Datenpunkte zu aggregieren. *Z. B. Gruppierung nach Vertriebskanälen*
  11. **Diagrammname** Geben Sie Ihrem Diagramm einen aussagekräftigen Namen. *Z.B. Travelbug Buchungen auf der Stundenkarte*
  12. Klicken `Save` Taste.
</Steps>


## So funktioniert es

Die Diagrammverarbeitung erfolgt in vier Schritten. Diese Schritte werden im Folgenden beschrieben.

### Zeitreihentyp

Durch die Auswahl eines festen Datumsbereichs oder eines kontinuierlichen Zeitabschnitts können Sie sich auf den WANN konzentrieren, ab dem Sie Daten sehen möchten.

### Tracking / Filterung

Durch die Auswahl einer Immobilie, eines Vertriebskanals usw. konzentrieren Sie sich darauf, von wem Sie Daten sehen möchten. Sie filtern effektiv alles heraus, was Sie nicht sehen möchten.

### Datenpunkte

Durch die Auswahl von Datenpunkten konzentrieren Sie sich darauf, WOZU Sie Daten sehen möchten.

:::Hinweis
Die Datenpunkte sollten so synergistisch sein, dass sie in ein Diagramm passen. Beispielsweise sollten Sie „Seitenaufrufe“ und „Gesamtnettoumsatz“ nicht im selben Diagramm darstellen.
:::

### Gruppierung<Badge text="New" variant="note" />


Durch Gruppieren lassen sich Daten wirkungsvoll aggregieren und fokussieren.

:::Tipp
Wenn Sie beispielsweise eine Unterkunft betreiben und die Anzahl der Buchungen über jeden Ihrer Vertriebskanäle erfassen möchten, filtern Sie nach Ihrer Unterkunft (siehe oben), wählen Sie „Buchungen“ (siehe oben) und gruppieren Sie die Daten im Bereich „Gruppierung“ nach „Vertriebskanal“.
:::

## Entfernen eines Diagramms

Um ein Diagramm zu entfernen, gehen Sie folgendermaßen vor:

<Steps>
  1. Klicken Sie auf das `Actions` Link, der unter jedem Diagramm vorhanden ist.
  2. Klicken Sie auf das `Remove` Taste
</Steps>


Ihr Diagramm wurde gelöscht.
