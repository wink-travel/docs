Wissen ist das Lebenselixier von Wink. Es zeigt Ihnen, wie jeder und alles auf Wink performt, bis hin zur Stunde. Vor Wink haben Hotels einen Drittanbieter abonniert und viel Geld bezahlt, um Zugriff auf das zu erhalten, was das Revenue-Management-Team als `the comp set` bezeichnet; was übersetzt bedeutet: `Wie vergleicht sich meine Unterkunft mit einer anderen Unterkunft?`.

Auf Wink haben Sie kostenlosen Zugriff auf plattformweite Analytics. So können Sie die Leistung jedes Affiliates, Reisebüros oder jeder Unterkunft über jede gewünschte Metrik verfolgen... so viel Sie wollen. Sie können auch breitere Trends nach Region einsehen, um zu sehen, welche Art von Produkt wo und wann verkauft wird. In den richtigen Händen kann diese Einsicht die Verkaufskompetenz jedes Wink-Nutzers verbessern.

## Zugriff auf Diagramme

Sie können von jeder unserer Websites auf Ihre Diagramme zugreifen. Um auf Ihre analytischen Diagramme zuzugreifen, folgen Sie diesen Schritten:

<Steps>
1. Sie müssen auf einer unserer Websites eingeloggt sein.
2. Klicken Sie auf Ihr Profil-Symbol oben rechts, es erscheint ein Menü.
3. Klicken Sie auf den Link `Analytics`.
4. Sie werden zu Ihren Diagrammen weitergeleitet.
</Steps>

:::note
Standardmäßig erstellen wir einfache Diagramme für jede Unterkunft, jedes Reisebüro und jedes Affiliate-Konto, das Sie anlegen, damit Sie leicht verfolgen können, wie Ihre eigenen Konten performen.
:::

## Ein Diagramm erstellen

Wenn Sie eigene Diagramme erstellen möchten, gehen Sie wie oben beschrieben zu Ihren Diagrammen und klicken Sie auf die Schaltfläche `Create chart`.

Nachfolgend finden Sie die einfachen Schritte zur Erstellung eines Analytics-Diagramms, gefolgt von detaillierteren Anweisungen.

<Steps>
1. **Zeitreihe** Wählen Sie eine Zeitreihe für Ihr Diagramm. *z. B. Täglich*.
2. **Tracking-Typ** Wählen Sie, ob Sie einen festen Datumsbereich oder kontinuierlich verfolgen möchten. *z. B. `Continuous`*.
    - Kontinuierlich: Wählen Sie Zeiteinheiten. *z. B. 7 Tage*
    - Fest: Geben Sie einen Datums-/Zeitraum ein *z. B. 22. Sept. 2023 16:00 - 29. Sept. 2023 16:00*.
3. **Anzeigewährung** Wenn Sie Beträge in Ihrem Diagramm haben, wählen Sie, in welchen Währungen Sie diese sehen möchten. *z. B. USD*.
4. **Unterkunft verfolgen** Verfolgen Sie eine bestimmte Unterkunft. *Optional*
5. **Vertriebskanal verfolgen** Verfolgen Sie Vertriebskanäle nach Kontinent, Land oder Stadt. *Optional*
6. **Vertriebskanaltyp filtern** Filtern Sie nach einem bestimmten Vertriebskanaltyp. *z. B. Influencer*. *Optional*
7. **Bestimmten Vertriebskanal verfolgen** Verfolgen Sie einen bestimmten Vertriebskanal. *Optional*
8. **Reisenden verfolgen** Verfolgen Sie den Standort von Reisenden nach Kontinent, Land oder Stadt. *Optional*
9. **Datenpunkte auswählen** Wählen Sie die Datenpunkte, die Sie verfolgen möchten. *z. B. Buchungen & Stornierungen*
10. **Daten gruppieren** Statt nach Unterkünften oder Vertriebskanälen zu filtern, können Sie auch gruppieren, um Datenpunkte zu aggregieren. *z. B. Gruppieren nach Vertriebskanälen*
11. **Diagrammname** Geben Sie Ihrem Diagramm einen aussagekräftigen Namen. *z. B. Travelbug-Buchungen stündlich*
12. Klicken Sie auf die Schaltfläche `Save`.
</Steps>

## Funktionsweise

Diagramme werden in 4 Schritten verarbeitet. Diese Schritte werden im Folgenden beschrieben.

### Zeitreihentyp

Indem Sie einen festen Datumsbereich oder einen kontinuierlichen Zeitabschnitt wählen, konzentrieren Sie sich darauf, WANN Sie Daten sehen möchten.

### Tracking / Filterung

Indem Sie eine Unterkunft, einen Vertriebskanal usw. wählen, konzentrieren Sie sich darauf, VON WEM Sie Daten sehen möchten. Sie filtern effektiv alles heraus, was Sie nicht sehen wollen.

### Datenpunkte

Indem Sie Datenpunkte auswählen, konzentrieren Sie sich darauf, FÜR WAS Sie Daten sehen möchten.

:::note
Die Datenpunkte sollten „synergistisch“ sein, sodass sie zusammen auf ein Diagramm passen. Zum Beispiel würden Sie nicht „Seitenaufrufe“ und „Gesamtnettoumsatz“ im selben Diagramm anzeigen.
:::

### Gruppierung <Badge text="Neu" variant="note" />

Gruppierung ist eine leistungsstarke Methode, um Daten zu aggregieren und zu fokussieren.

:::tip
Wenn Sie beispielsweise eine Unterkunft sind und verfolgen möchten, wie viele Buchungen jeder Ihrer Vertriebskanäle gemacht hat, würden Sie oben auf Ihre Unterkunft filtern, „Buchungen“ auswählen und in der Gruppierungssektion nach „Vertriebskanal“ gruppieren.
:::

## Ein Diagramm entfernen

Um ein Diagramm zu entfernen, gehen Sie wie folgt vor:

<Steps>
1. Klicken Sie auf den Link `Actions`, der unter jedem Diagramm vorhanden ist.
2. Klicken Sie auf die Schaltfläche `Remove`.
</Steps>

Ihr Diagramm wurde gelöscht.
