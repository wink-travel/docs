Wenn Sie Ihr Passwort für Ihr Wink-Benutzerkonto ändern möchten, gehen Sie folgendermaßen vor:

## Kennwort aktualisieren

<Steps>
  1. Sie müssen auf einer unserer Websites angemeldet sein.
  2. Klicken Sie oben rechts auf Ihr Profilsymbol. Ein Menü wird angezeigt.
  3. Klicken Sie auf das `Change password` Link.
  4. Folgen Sie den Anweisungen, um Ihr Passwort zu ändern.
  5. Klicken `Save`.
</Steps>


Beim nächsten Login benötigen Sie Ihr neues Passwort.

## Passwort vergessen

Manchmal ist es notwendig, das Benutzerkennwort zurückzusetzen.

So setzen Sie Ihr Benutzerkennwort zurück:

<Steps>
  1. Klicken Sie auf das `Forgot password` Link von der Anmeldeseite.
  2. Geben Sie den Benutzernamen/die E-Mail-Adresse des Kontos ein, das Sie zurücksetzen möchten.
  3. Klicken `Forgot password` Taste. *Sie werden zum Formular zum Zurücksetzen des Passworts weitergeleitet.*
  4. Überprüfen Sie Ihre E-Mails.
</Steps>

  <CloudinaryImage width="350" height="354" resize="scale" quality="best" alt="Passwort vergessen" src="academy/forgot-password" errorImage="noimage_opaque_nyrtl0.png" />

<Steps>
  1. Fügen Sie den Code, den Sie in der E-Mail erhalten haben, zusammen mit Ihrem neuen Passwort in das neue Formular ein.
  2. Bestätigen Sie Ihr neues Passwort.
  3. Klicken `Reset password` Taste. *Sie werden zurück zur Anmeldeseite weitergeleitet.*
</Steps>


<figure>
  <CloudinaryImage width="350" height="467" resize="scale" quality="best" alt="Passwort zurücksetzen" src="academy/reset-password" errorImage="noimage_opaque_nyrtl0.png" />

Ihr Passwort wurde erfolgreich zurückgesetzt. Sie können sich mit Ihren neuen Anmeldedaten bei Wink anmelden.
