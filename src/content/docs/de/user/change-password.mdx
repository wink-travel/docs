Wenn Sie Ihr Passwort für Ihr Wink-Benutzerkonto ändern möchten, befolgen Sie diese Schritte:

## Passwort aktualisieren

<Steps>
1. Sie müssen auf einer unserer Websites eingeloggt sein.
2. Klicken Sie auf Ihr Profil-Symbol oben rechts, es erscheint ein Menü.
3. Klicken Sie auf den Link `Passwort ändern`.
4. Folgen Sie den Anweisungen, um Ihr Passwort zu ändern.
5. Klicken Sie auf `Speichern`.
</Steps>

Beim nächsten Login benötigen Sie Ihr neues Passwort.

## Passwort vergessen

Manchmal ist es notwendig, das Benutzerpasswort zurückzusetzen.

Hier sind die Schritte zum Zurücksetzen Ihres Benutzerpassworts:

<Steps>
1. Klicken Sie auf der Anmeldeseite auf den Link `Passwort vergessen`.
2. Geben Sie den Benutzernamen / die E-Mail-Adresse des Kontos ein, das Sie zurücksetzen möchten.
3. Klicken Sie auf die Schaltfläche `Passwort vergessen`. *Sie werden zum Formular zum Zurücksetzen des Passworts weitergeleitet.*
4. Prüfen Sie Ihre E-Mails.
</Steps>

<Steps>
1. Fügen Sie den Code, den Sie per E-Mail erhalten haben, zusammen mit Ihrem neuen Passwort in das neue Formular ein.
2. Bestätigen Sie Ihr neues Passwort.
3. Klicken Sie auf die Schaltfläche `Passwort zurücksetzen`. *Sie werden zurück zur Anmeldeseite weitergeleitet.*
</Steps>
