Wenn Sie Ihr Benutzerprofil für Ihr Wink-Benutzerkonto aktualisieren möchten, befolgen Sie diese Schritte:

<Steps>
1. Sie müssen auf einer unserer Websites eingeloggt sein.
2. Klicken Sie auf Ihr Profil-Symbol oben rechts, und es erscheint ein Menü.
3. Klicken Sie auf den Link `Profil`.
4. Folgen Sie den Anweisungen, um Ihre `E-Mail`, Ihren `Vor-` und `Nachnamen` zu aktualisieren.
5. Sie können auch ein neues `Profilbild` von sich hochladen.
6. Klicken Sie auf `Speichern`.
</Steps>

:::note
Ihr neues Profilbild wird beim nächsten vollständigen Ausloggen aus Wink angezeigt.
:::
