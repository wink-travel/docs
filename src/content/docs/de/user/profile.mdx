Wenn Sie Ihr Benutzerprofil für Ihr Wink-Benutzerkonto aktualisieren möchten, gehen Sie folgendermaßen vor:

<Steps>
  1. Sie müssen auf einer unserer Websites angemeldet sein.
  2. Klicken Sie oben rechts auf Ihr Profilsymbol. Ein Menü wird angezeigt.
  3. Klicken Sie auf das `Profile` Link.
  4. Folgen Sie den Anweisungen zum Aktualisieren Ihres `e-mail`, `first` Und `last name`.
  5. Sie können auch eine neue `profile picture` von dir selbst.
  6. Klicken `Save`.
</Steps>


:::Hinweis
Ihr neues Profilbild wird angezeigt, wenn Sie sich das nächste Mal vollständig von Wink abmelden.
:::
