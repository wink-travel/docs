Wenn Sie ein Reiseunternehmen sind und den Zahlungs- und Auszahlungsbereich Ihres Geschäfts an TripPay auslagern möchten, sind hier 6 grundlegende Schritte, um zu beginnen:

<Steps>
1. Erstellen Sie ein [Benutzerkonto](/de/user/register) bei Wink.
2. Erstellen Sie ein Konto bei [TripPay](https://pay.wink.travel).
3. Erstellen Sie eine [Anwendung](/de/developers/applications) bei TripPay, die Ihnen den Zugriff auf unsere API ermöglicht.
4. Erstellen und verknüpfen Sie Ihre Begünstigten mit TripPay. *z. B. bedeutet das für die meisten, Konten für die Unterkünfte auf Ihrer Plattform zu erstellen, die TripPay bezahlen soll.*
5. Integrieren Sie die [Zahlungs-Webkomponente](/de/developers/web-components/#payment) in Ihre Website.
6. Senden Sie TripPay den `Einkaufswagen` des Reisenden und starten Sie den Zahlungsprozess.
</Steps>

:::tip
Für eine vollständige Anleitung zur Nutzung von TripPay für Ihre Reise-Website lesen Sie den Leitfaden [Integration mit TripPay](/de/guides/integrating-with-trip-pay).
:::

## API

Entwickler, die `Zahlungen` verwalten möchten, gehen zu [Entwickler > API > Zahlungen](/de/developers/apis/#payment-api).
