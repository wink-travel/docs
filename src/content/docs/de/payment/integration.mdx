Wenn Sie ein Reiseunternehmen sind und den Zahlungs- und Auszahlungsteil Ihres Geschäfts an TripPay auslagern möchten, finden Sie hier 6 grundlegende Schritte für den Einstieg:

<Steps>
  1. Erstellen Sie ein [Benutzerkonto](/user/register) auf Wink.
  2. Erstellen Sie ein Konto mit [TripPay](https://pay.wink.travel).
  3. Erstellen Sie ein [Anwendung](/developers/applications) auf TripPay, das Ihnen die Kommunikation mit unserer API ermöglicht.
  4. Erstellen und ordnen Sie Ihre Begünstigten mit TripPay zu. *Für die meisten bedeutet das beispielsweise, dass Sie Konten für die Unterkünfte auf Ihrer Plattform erstellen müssen, die Sie über TripPay bezahlen möchten.*
  5. Integrieren Sie die [Zahlungs-Webkomponente](/developers/web-components/#payment) in Ihre Site.
  6. Senden Sie TripPay an den Reisenden `shopping cart` und leiten Sie den Zahlungsvorgang ein.
</Steps>


:::Tipp
Eine vollständige Anleitung zur Nutzung von TripPay für Ihre Reise-Website finden Sie im Leitfaden[Integration mit TripPay](/guides/integrating-with-trip-pay).
::
