Während Sie auf https://ota.wink.travel sind, können Sie Ihr `Reiseprofil` ausfüllen, das Hotels und Erlebnisanbietern mehr über Sie verrät.

Ein Reiseprofil enthält:

- Grundlegende Informationen.
- Spracheinstellungen.
- Notfallkontakte.
- Adresse.
- Persönliche Informationen.
- Haustiere.
- Körperliche Einschränkungen.
- Hotelpräferenzen.
- Ausstattungsvorlieben.
- Sicherheitspräferenzen.
- Firmenpräferenzen.
- Präferenzen bei Behinderungen.
- Zimmerpräferenzen.
- Zimmerausstattungspräferenzen.
- Persönliche Präferenzen.

Am Ende Ihres Profils werden Sie gefragt, ob Sie dieses Profil mit den Hotels, die Sie besuchen, teilen möchten. Wenn Sie sich für das Teilen entscheiden, sieht das Hotel das vollständige Profil in seiner Buchungsbestätigung.
