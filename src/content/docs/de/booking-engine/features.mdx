Die Buchungsmaschine gibt es in zwei Varianten, um die unterschiedlichen Bedürfnisse eines Affiliates, einer Property-Management-Firma und eines Hotels zu unterstützen.

## Online-Reisebüro

Für einen Affiliate macht es am meisten Sinn, die Online-Reisebüro-Version zu verwenden. Der Grund ist, dass die Seite so eingerichtet ist, dass Reisende mehrere Unterkünfte ansehen können.

Die Startseite dieser Anwendung finden Sie unten.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Internet-Buchungsmaschine

Für ein Hotel oder eine Property-Management-Firma macht es am meisten Sinn, die Internet-Buchungsmaschinen-Version zu verwenden. Dieser Nutzertyp möchte eine einzelne Unterkunft oder mehrere verwaltete Unterkünfte hervorheben.

Die Startseite dieser Anwendung finden Sie unten.

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE
</LinkButton>

## Konfiguration

Meistens ist es nicht notwendig, eine dieser Anwendungen direkt zu teilen. Wir haben verschiedene Möglichkeiten geschaffen, um Reiseinventar zu teilen, die Sie in [Wink Studio](/de/studio/what-is-studio) oder über [WinkLinks](/de/link-manager/wink-links) entdecken können.

Wenn Sie jedoch manuell auf eine dieser Websites verlinken möchten, stellen Sie sicher, dass Sie Ihre Client-ID und Anpassungs-ID als Parameter zur URL hinzufügen.

Ihre Client-ID finden Sie, indem Sie:

<Steps>
1. Melden Sie sich entweder bei [Wink Extranet](/de/extranet/what-is-extranet) oder [Wink Studio](/de/studio/what-is-studio) an.
2. Klicken Sie auf Ihr Profil-Symbol oben rechts und dann auf [Applications](/de/developers/applications).
3. Verwenden Sie die Anwendung, die mit dem Wink-Konto verknüpft ist, mit dem Sie arbeiten möchten.
4. Kopieren Sie die Client-ID aus dieser Anwendung und fügen Sie sie als Anfrageparameter in die URL ein. Beispiel: ?client-id=MEINE_CLIENT_ID
</Steps>

### Extranet

Ihre Anpassungs-ID als Unterkunft finden Sie, indem Sie:

<Steps>
1. Melden Sie sich im [Wink Extranet](/de/extranet/what-is-extranet) an.
2. Wählen Sie die Unterkunft aus, mit der Sie arbeiten möchten.
3. Klicken Sie in der Hauptnavigationsleiste auf `Account > Branding`.
4. Kopieren Sie die Anpassungs-ID und fügen Sie sie als Anfrageparameter in die URL ein. Beispiel: ?configuration-id=MEINE_ANPASSUNGS_ID
</Steps>

### Studio

Ihre Anpassungs-ID als Affiliate finden Sie, indem Sie:

<Steps>
1. Melden Sie sich im [Wink Studio](/de/studio/what-is-studio) an.
2. Wählen Sie das Konto aus, mit dem Sie arbeiten möchten.
3. Klicken Sie in der Hauptnavigationsleiste auf `Tools > Customization`.
4. Kopieren Sie die Anpassungs-ID der gewünschten Anpassung und fügen Sie sie als Anfrageparameter in die URL ein. Beispiel: ?configuration-id=MEINE_ANPASSUNGS_ID
</Steps>

## OTA

Am Ende erhalten Sie eine URL, die ungefähr so aussieht:

https://ota.wink.travel?client-id=MEINE_CLIENT_ID&configuration-id=MEINE_ANPASSUNGS_ID

Wenn Sie direkt zu einer Hotel-Seite gehen möchten, sieht die URL ungefähr so aus:

https://ota.wink.travel/hotel/hotel-x?client-id=MEINE_CLIENT_ID&configuration-id=MEINE_ANPASSUNGS_ID

:::note
Wenn Sie keine Client- und/oder Anpassungs-ID angeben, verwendet Wink OTA eine Standardkonfiguration.
:::

## IBE

Wenn Sie eine Property-Management-Firma sind, möchten Sie auf die Wink IBE-Startseite verlinken. Die Startseite zeigt Ihr festgelegtes Theme zusammen mit den [Unterkünften, die Sie vertreten](/de/studio/agency-representation).

Am Ende erhalten Sie eine URL, die ungefähr so aussieht:

https://book.wink.travel?client-id=MEINE_CLIENT_ID&configuration-id=MEINE_ANPASSUNGS_ID

Wenn Sie eine Unterkunft sind, möchten Sie direkt auf die Wink IBE-Unterkunfts-Landingpage verlinken.

Diese URL sieht ungefähr so aus:

https://book.wink.travel/hotel/hotel-x?client-id=MEINE_CLIENT_ID&configuration-id=MEINE_ANPASSUNGS_ID

:::note
Wenn Sie keine Client- und/oder Anpassungs-ID angeben, lädt Wink IBE NICHT. Diese Parameter sind erforderlich.
:::

## Beispiel: Unterkunftsseite

Jede Unterkunft bei Wink erhält ihre eigene einzigartige Landingpage auf einer der Websites. Alle externen Links führen Reisende zu einer dieser Landingpages.

Die Landingpage enthält alle Informationen, die ein Reisender benötigt, um eine fundierte Entscheidung zu treffen, ob er in dieser Unterkunft übernachten möchte.
