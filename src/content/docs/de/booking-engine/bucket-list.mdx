Überall, wo Sie Wink-Inventar sehen [Karten](/studio/cards), sehen Sie in der oberen rechten Ecke der Karte ein Herz, mit dem Sie dieses Inventar zu Ihrer Bucket List hinzufügen können.

Befolgen Sie diese Schritte, um Ihre Bucket List zu verwalten:

<Steps>
  1. Klicken Sie auf das ♡ `heart icon` für das Inventar, das Sie hinzufügen möchten.
  2. Wenn Sie angemeldet sind, wird das Herzsymbol rot.
  3. Wenn Sie nicht angemeldet sind, werden Sie umgeleitet, dies zu tun.
</Steps>


:::Tipp
Klicke auf das ♥️`heart icon`ein zweites Mal, um es aus deiner Liste zu entfernen.
::

Sie können alle Elemente Ihrer Bucket List verwalten, indem Sie oben rechts auf Ihr Profilsymbol und dann auf das`Bucket list` Taste.
