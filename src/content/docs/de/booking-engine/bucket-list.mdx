Überall, wo Sie Wink-Inventar-[Karten](/de/studio/cards) sehen, finden Sie oben rechts auf der Karte ein Herz, mit dem Sie dieses Inventar zu Ihrer Wunschliste hinzufügen können.

Folgen Sie diesen Schritten, um Ihre Wunschliste zu verwalten:

<Steps>
1. Klicken Sie auf das ♡ `Herzsymbol` für das Inventar, das Sie hinzufügen möchten.
2. Wenn Sie eingeloggt sind, wird das Herzsymbol rot.
3. Wenn Sie nicht eingeloggt sind, werden Sie zur Anmeldung weitergeleitet.
</Steps>

:::tip
Klicken Sie ein zweites Mal auf das ♥️ `Herzsymbol`, um es von Ihrer Liste zu entfernen.
:::

Sie können alle Ihre Wunschlisteneinträge verwalten, indem Sie auf Ihr Profil-Symbol oben rechts klicken und dann die Schaltfläche `Wunschliste` auswählen.
