Dieser Artikel behandelt den gesamten Buchungsvorgang auf Wink.

## Wie du buchen kannst

Sobald Sie die Unterkunft und das Zimmer gefunden haben, das Sie buchen möchten, klicken Sie auf`Book`Schaltfläche, um zu beginnen.

<figure>
  <CloudinaryImage width="350" height="819" resize="scale" quality="best" alt="Zubuchbare Zimmerkarte" src="academy/guest-room-card" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Beispiel einer buchbaren Gästezimmerkarte*</figcaption>
</figure>


<Steps>
  1. Klicken Sie auf das `Book` Taste.
  2. Das Fenster des Raumkonfigurators öffnet sich. *[Abbildung 1](#configurator)*
  3. Wählen Sie die für diesen Raum verfügbaren Optionen. *[Abbildung 2](#options)*
  4. Klicken `Log in to continue` wenn Sie bereit sind.
  5. Wenn Sie Wink zum ersten Mal nutzen, müssen Sie [ein Konto erstellen](/user/register#registration).
  6. Geben Sie Ihre Zahlungsdetails ein. *[Abbildung 3](#payment)*
  7. Klicken `Pay` Schaltfläche, um die Buchung abzuschließen.
</Steps>


### Konfigurator

Mit dem Zimmerkonfigurator können Sie Optionen für und neben dem Zimmer auswählen.
Halten Sie hier Ausschau nach tollen Angeboten von Hotels, indem Sie nach unten scrollen, um alle Angebote anzuzeigen.

### Optionen

Zu den gängigen Optionen gehören:

* Stornierungsbedingungen
* Mahlzeiten
* Bettenpräferenzen
* Sonderwünsche

Weitere Optionen sind:

* Zusatzausstattung im Zimmer.
* Andere Add-ons
* Sonderangebote für Einrichtungen und Erlebnisse:
  * Spa-Sitzungen.
  * Restaurantangebote.
  * Konferenzräume buchen.
  * Aktivitäten buchen.
  * ...und vieles mehr!!

### Zahlung

Jetzt ist es Zeit zu bezahlen. Wählen Sie die Zahlungsmethode, die für Sie geeignet ist, und klicken Sie auf`Pay`.

<figure>
  <CloudinaryImage width="700" height="1141" resize="scale" quality="best" alt="Zahlungsdetails" src="academy/payment-modal" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Abbildung 2. Zahlungsdetails*</figcaption>
</figure>


Nach der Bezahlung werden Sie auf eine Buchungsbestätigungsseite weitergeleitet, die Sie ausdrucken können.
Zusätzlich erhalten Sie eine E-Mail mit allen wichtigen Informationen zur Buchung.

## Buchungen verwalten

<figure>

Sie können Ihre vergangenen und zukünftigen Buchungen sehen, indem Sie auf Ihr Profilsymbol in der oberen rechten Ecke der Seite klicken und dann auf`Bookings`.
Es öffnet sich ein Fenster, in dem Ihre Buchungen in aufsteigender Reihenfolge angezeigt werden.

## Eine Buchung stornieren

:::Achtung
Sie können nur Buchungen stornieren, deren Stornierungsbedingungen eine Stornierung zulassen. Lesen Sie die Stornierungsbedingungen sorgfältig durch, bevor Sie ein Zimmer buchen, und wählen Sie eine Stornierungsrichtlinie, die für Sie geeignet ist.
:::

Wenn Sie eine Buchung stornieren möchten, gehen Sie folgendermaßen vor:

<Steps>
  1. Wählen Sie die Buchung aus, die Sie stornieren möchten. Sie können dies folgendermaßen tun:
     * Klicken Sie auf `Manage booking` Link aus der Bestätigungs-E-Mail, die Sie erhalten haben.
     * Gehen Sie zu „Buchungen verwalten“ auf https://ota.wink.travel und klicken Sie auf `Bookings` Link, der im vorherigen Abschnitt erwähnt wurde.
  2. Klicken Sie auf das `Cancel` Taste. *Die Schaltfläche „Abbrechen“ ist deaktiviert, wenn Sie die Buchung nicht mehr stornieren können.*
  3. Teilen Sie uns den Grund für Ihre Stornierung mit und klicken Sie auf `Submit`.
  4. Basierend auf den Stornierungsbedingungen wird Ihnen der volle oder teilweise Betrag zurückerstattet.
</Steps>


:::Hinweis
Die Bearbeitung einer Rückerstattung dauert ca. 1–3 Tage.
:::

## Erstattung beantragen

Unter bestimmten Umständen können Sie nach Ablauf der Widerrufsfrist eine Rückerstattung verlangen.

Bitte kontaktieren Sie das Hotel und schildern Sie den Sachverhalt. Das Hotel wird die Rückerstattung in Ihrem Namen beantragen.

## Schreiben Sie eine Rezension

Sobald die Reise zu Ende ist \[und Sie Urlaub vom Urlaub brauchen], senden wir Ihnen eine E-Mail mit der Bitte, eine Bewertung für die Unterkunft zu schreiben, in der Sie übernachtet haben.

So schreiben Sie eine Bewertung für die Unterkunft:

<Steps>
  1. Klicken Sie auf den Link in der E-Mail, die wir Ihnen gesendet haben.
  2. Möglicherweise müssen Sie sich erneut bei Wink anmelden.
  3. Über den Link gelangen Sie zur Buchungsbestätigungsseite.
  4. Klicken `Write review` Taste.
  5. Beantworten Sie die 5 Multiple-Choice-Fragen.
  6. Schreiben Sie einen oder mehrere kurze Sätze über Ihren Aufenthalt.
  7. Klicken Sie auf das `Submit` Taste.
</Steps>


Bewertungen sind eine großartige Möglichkeit, Unterkünfte bekannt zu machen, und zwar ohne Preisschild.

:::Tipp\[Profi-Tipp]
Wenn Ihr Aufenthalt nicht so toll war, verschwenden Sie nicht Ihre ganze Zeit damit, eine negative Bewertung zu schreiben.

Informieren Sie stattdessen das Hotel, während Sie noch auf dem Gelände sind, und geben Sie dem GM die Chance, es wiedergutzumachen.

\#SprechenSie#EsFunktioniert
:::
