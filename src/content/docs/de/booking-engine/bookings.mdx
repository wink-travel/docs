## Wie man bucht

Sobald Sie die Unterkunft und das Zimmer gefunden haben, das Sie buchen möchten, klicken Sie auf die Schaltfläche `Buchen`, um zu beginnen.

<Steps>
1. Klicken Sie auf die Schaltfläche `Buchen`.
2. Das Fenster des Raumkonfigurators öffnet sich. * [Abbildung 1](#configurator) *
3. Wählen Sie die für dieses Zimmer verfügbaren Optionen aus. * [Abbildung 2](#options) *
4. Klicken Sie auf `Anmelden, um fortzufahren`, wenn Sie bereit sind.
5. Wenn Sie zum ersten Mal auf Wink sind, müssen Sie ein [Konto erstellen](/de/user/register#registration).
6. Geben Sie Ihre Zahlungsdaten ein. * [Abbildung 3](#payment) *
7. Klicken Sie auf die Schaltfläche `Bezahlen`, um die Buchung abzuschließen.
</Steps>

### Konfigurator

Der Raumkonfigurator gibt Ihnen die Möglichkeit, Optionen für und neben dem Zimmer auszuwählen.
Achten Sie auf tolle Angebote von Hotels, indem Sie nach unten scrollen, um alles zu sehen, was sie anbieten.

### Optionen

Häufige Optionen sind:

- Stornierungsbedingungen
- Mahlzeiten
- Bettpräferenzen
- Sonderwünsche

Weitere Optionen umfassen:

- Zusatzausstattung im Zimmer.
- Weitere Extras
- Sonderangebote für Einrichtungen und Erlebnisse:
    - Spa-Sitzungen.
    - Restaurantangebote.
    - Tagungsräume buchen.
    - Aktivitäten buchen.
    - ...und vieles mehr!!

Ein Fenster öffnet sich und zeigt Ihre Buchungen in aufsteigender Reihenfolge an.

## Buchung stornieren

:::caution
Sie können nur eine Buchung stornieren, wenn die Stornierungsbedingungen dies erlauben. Lesen Sie die Stornierungsbedingungen sorgfältig durch, bevor Sie ein Zimmer buchen, und wählen Sie eine Stornierungsbedingung, die für Sie passt.
:::

Wenn Sie eine Buchung stornieren möchten, gehen Sie wie folgt vor:

<Steps>
1. Wählen Sie die Buchung aus, die Sie stornieren möchten. Das können Sie tun, indem Sie entweder:
    - Auf den Link `Buchung verwalten` in der Bestätigungs-E-Mail klicken, die Sie erhalten haben.
    - Zur Buchungsverwaltung auf https://ota.wink.travel gehen und auf den zuvor erwähnten Link `Buchungen` klicken.
2. Klicken Sie auf die Schaltfläche `Stornieren`. *Die Stornieren-Schaltfläche ist deaktiviert, wenn Sie die Buchung nicht mehr stornieren können.*
3. Geben Sie den Grund für die Stornierung an und klicken Sie auf `Absenden`.
4. Je nach Stornierungsbedingungen wird Ihnen der volle oder ein Teilbetrag erstattet.
</Steps>

:::note
Rechnen Sie mit etwa 1 - 3 Tagen, bis die Rückerstattung erfolgt ist.
:::

## Rückerstattung beantragen

Unter bestimmten Umständen können Sie eine Rückerstattung beantragen, nachdem die Stornierungsbedingungen abgelaufen sind.

Bitte kontaktieren Sie das Hotel und erklären Sie die Situation. Das Hotel wird die Rückerstattung in Ihrem Namen beantragen.

## Eine Bewertung schreiben

Sobald die Reise beendet ist [und Sie eine Auszeit vom Urlaub brauchen], senden wir Ihnen eine E-Mail mit der Bitte, eine Bewertung für die Unterkunft zu schreiben, in der Sie übernachtet haben.

So schreiben Sie eine Bewertung für die Unterkunft:

<Steps>
1. Klicken Sie auf den Link in der E-Mail, die wir Ihnen gesendet haben.
2. Möglicherweise müssen Sie sich erneut bei Wink anmelden.
3. Der Link führt Sie zur Buchungsbestätigungsseite.
4. Klicken Sie auf die Schaltfläche `Bewertung schreiben`.
5. Beantworten Sie die 5 Multiple-Choice-Fragen.
6. Schreiben Sie einen kurzen Satz oder mehrere Sätze über Ihren Aufenthalt.
7. Klicken Sie auf die Schaltfläche `Absenden`.
</Steps>

Bewertungen sind eine großartige Möglichkeit für Unterkünfte, Anerkennung zu erhalten, die keinen Preis hat.

:::tip[Profi-Tipp]
Wenn Ihr Aufenthalt nicht so gut war, verbringen Sie nicht Ihre ganze Zeit damit, eine negative Bewertung zu schreiben.

Teilen Sie dem Hotel stattdessen vor Ort Ihre Anliegen mit und geben Sie dem Geschäftsführer die Chance, es wieder gutzumachen.

:::
