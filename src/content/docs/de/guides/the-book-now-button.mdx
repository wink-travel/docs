Der **JETZT BUCHEN**„Button“ ist ein allgegenwärtiger Begriff in der Hotelbranche und bezeichnet die hoteleigene Integration mit einem Anbieter, der Verfügbarkeit und Preise der Unterkunft kennt und Reisenden die Buchung eines Zimmers ermöglicht. Der Button ist für Hotels die einfachste Möglichkeit, Zimmer auf ihrer eigenen Website und zu ihren eigenen Bedingungen zu verkaufen.

*Viele Hotels nutzen die gebündelte Buchungsmaschine ihres Channel Managers. Wir empfehlen Hotels, sich zu informieren und verschiedene Buchungsmaschinen zu vergleichen, bevor sie sich für eine bestimmte entscheiden.*

## Publikum

Dieser Artikel richtet sich in erster Linie an Hotels, die unsere kostenlose Buchungsmaschine mit hoher Konversionsrate auf ihrer Website installieren möchten.
Er ist aber auch eine nützliche Lektüre für alle, die unsere Technologien auf ihrer Website nutzen möchten.

## über Link

Für die einfachste Integration empfehlen wir die Verwendung eines[Teilbarer Link](/studio/shareable-links).

Gehen Sie dazu folgendermaßen vor:

<Steps>
  1. Gehen Sie zu https://extranet.wink.travel
  2. Klicken `Sign-In / Register` in der oberen rechten Ecke und melden Sie sich mit Ihrem Benutzerkonto an.
  3. Wählen Sie aus der Dropdown-Liste oben links die Eigenschaft aus, mit der Sie arbeiten möchten.
  4. Klicken Sie auf `Booking engine` in der Navigationsleiste.
  5. Auf der Seite wird Ihnen Ihr Link angezeigt.
  6. Geben Sie diesen Link an Ihren Webmaster weiter und nutzen Sie ihn für Ihre **JETZT BUCHEN** Taste.
  7. Du bist ✅.
</Steps>


### Fortschrittlich

Der Link enthält bereits eine Reiseroute, die in die URL integriert ist und von Ihrem gewählten`customization`Sie können Ihren Link weiter anpassen, indem Sie:

* Ändern der ausgewählten Anpassung ODER
* Überschreiben Sie die Reiseroute

In diesem Artikel konzentrieren wir uns auf Letzteres, da es Ihnen mehr Freiheit gibt, Reisenden die Auswahl einer Reiseroute auf Ihrer Website zu ermöglichen und diese an die Buchungsmaschine von Wink weiterzuleiten.

So geben Sie Ihre eigene Reiseroute zusammen mit der URL weiter.

Gültige Anfrageparameter:

* **SD**So fügen Sie ein Startdatum hinzu:*sd=2024-08-24*
* **N**So fügen Sie die Anzahl der Nächte hinzu:*n = 1* ODER
* **ed**Enddatum verwenden:*ed=2024-08-25*
* **rc**Gäste:*rc=a2*(Erwachsene: 2)
* **l**Fordern Sie eine Anzeigesprache an:*l=es*
* **C**Fordern Sie eine Anzeigewährung an:*c=EUR*
* **P**Fügen Sie eine Aktion hinzu:*p=ABC123*

Die URL würde folgendermaßen aussehen:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2\&l=es\&c=EUR

Für anspruchsvollere Anwendungsfälle können Sie auch mehrere Raumkonfigurationen anfordern, indem Sie`+` Zeichen:

Beispiel:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2-ca16q1-ca8q1+a2\&l=es\&c=EUR

* Zimmertyp 1 = Erwachsene: 2, Kinder: 2 = 16 Jahre + 8 Jahre.
* Zimmertyp 2 = Erwachsene: 2.

### Weitere Vorteile

Es gibt noch weitere Vorteile, wenn Sie einen unserer Links als Ihren`book now` Taste.

* **Sozialfreundlich**Der Link kann einfach in sozialen Medien geteilt werden und zeigt`dynamic pricing`über dem von Ihnen ausgewählten Bild.
* **Kurz**Die URL lässt sich problemlos teilen und Sie können sie beispielsweise in Ihrer Instagram-Bio-URL verwenden.

## über Embed

Manche Hotels wünschen sich mehr Funktionalität und eine tiefere Integration für ihre Website. Dieses Beispiel behandelt die Einbettung eines [Karte](/studio/cards) in ihre Website.

Gehen Sie dazu folgendermaßen vor:

<Steps>
  1. Gehen Sie zu https://studio.wink.travel
  2. Klicken `Sign-In / Register` in der oberen rechten Ecke und melden Sie sich mit Ihrem Benutzerkonto an.
  3. Wählen Sie oben links die Eigenschaft aus, mit der Sie arbeiten möchten.
  4. Klicken Sie auf `Search` aus der Hauptnavigationsleiste.
  5. Geben Sie den Namen Ihrer Immobilie ein und klicken Sie auf die Suchschaltfläche.
  6. Klicken Sie auf das `Actions` Link auf Ihrer Suchergebniskarte und wählen Sie `Create > Make a Card`.
  7. Dadurch entsteht eine `Card`.
  8. In der oberen rechten Ecke wird eine Meldung angezeigt, die Ihnen mitteilt, dass Ihre Karte fertig ist und ob Sie sie weiter anpassen möchten.
  9. Klicken Sie auf `Tools > Cards` aus der Hauptnavigationsleiste und Sie sehen Ihre neue Karte.
  10. Klicken Sie auf das `Actions` Link auf Ihrer Linkkarte und klicken Sie auf `Embed`.
  11. Dadurch wird ein neues Fenster mit Beispiel-HTML-Code geöffnet, den Sie kopieren können.
  12. Um Ihre `Client-ID`gehen Sie zu [Anwendungen](/developers/applications).
  13. Um Ihre `Configuration-ID`gehen Sie zu [Anpassung](/studio/customization)
  14. Du bist ✅.
</Steps>


Beispiel für eingebettetes HTML für eine Karte:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*Das obige HTML-Beispiel enthält nicht das erforderliche Wink-CSS und Javascript; nur die Karte.*
