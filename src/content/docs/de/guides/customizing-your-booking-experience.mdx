Anpassungen steuern, wie Ihr Inventar Reisenden angezeigt wird — die Themenfarben, Standardwährung, Sprache, Reiseroute und mehr. Sie gelten für teilbare Links, eingebettete Karten, Raster, Kartenansichten und die Buchungsmaschine.

## Was ist eine Anpassung?

Eine Anpassung ist ein Konfigurationsprofil, das definiert:

- **Anzeigewährung** — In welcher Währung Preise angezeigt werden.
- **Anzeigesprache** — Die Standardsprache für Inhalte.
- **Standard-Reiseroute** — Vorgefüllte Check-in/-out-Daten und Gästeanzahl.
- **Thema** — Farben und Branding, die auf eingebettete Komponenten angewendet werden.
- **Konfiguration** — Wie das Inventar präsentiert wird (Layout, Sortierung).

Sie können mehrere Anpassungen für verschiedene Märkte oder Anwendungsfälle erstellen.

## Erstellen einer Anpassung

<Steps>
1. Gehen Sie zu https://studio.wink.travel (oder Ihrem Konto auf https://my.wink.travel).
2. Navigieren Sie im Seitenmenü zu `Customization`.
3. Klicken Sie auf `Create a new customization`.
4. Geben Sie einen aussagekräftigen Namen ein (z. B. „Europe - EUR“, „Japan Market“, „Default“).
5. Legen Sie die Anzeigewährung, Sprache und Standard-Reiseroute fest.
6. Konfigurieren Sie die Themenfarben passend zu Ihrem Branding.
7. Klicken Sie auf `Save`.
</Steps>

## Wo Anpassungen verwendet werden

Nach der Erstellung werden Anpassungen plattformweit verwendet:

| Funktion | Wie die Anpassung angewendet wird |
| -- | -- |
| **Teilbare Links** | Der Link verwendet die Währung, Sprache und Reiseroute der Anpassung als Standard |
| **Eingebettete Karten** | Karten werden mit Ihren Themenfarben und Anzeigeeinstellungen dargestellt |
| **Raster und Kartenansichten** | Inventarraster und Kartenmarker folgen der Anpassung |
| **Buchungsmaschine** | Der gesamte Buchungsablauf nutzt die Einstellungen der Anpassung |
| **Soziale Beiträge** | KI-generierte Beiträge können eine bestimmte Anpassung für die Preisangabe verwenden |

## Mehrere Anpassungen

Ein gängiges Muster ist, für jeden Zielmarkt eine Anpassung zu erstellen:

- **„US Market“** — USD, Englisch, Standard 2-Nächte-Aufenthalt.
- **„Europe“** — EUR, Englisch, Standard 3-Nächte-Aufenthalt.
- **„Japan“** — JPY, Japanisch, Standard 1-Nacht-Aufenthalt.

Wenn Sie einen teilbaren Link erstellen oder eine Karte einbetten, wählen Sie aus, welche Anpassung verwendet werden soll. So kann dieselbe Unterkunft je nach Zielgruppe unterschiedlich präsentiert werden.

## Überschreiben über URL

Sie können Standardwerte der Anpassung auch über URL-Parameter bei teilbaren Links überschreiben:

- `l=es` — Überschreibt die Sprache auf Spanisch.
- `c=EUR` — Überschreibt die Währung auf EUR.
- `sd=2024-08-24&n=2` — Überschreibt die Reiseroute mit einem bestimmten Datum und Anzahl der Nächte.

Siehe [The Book Now Button](/de/guides/the-book-now-button) für die vollständige Liste der URL-Parameter.
