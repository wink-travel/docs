[Zwinkern](https://wink.travel)ist eine bahnbrechende neue Reiseplattform, die von Grund auf entwickelt wurde, um Hotels, Erlebnisanbieter, Influencer und Partner gleichermaßen zu unterstützen. Durch die nahtlose Integration innovativer Tools in eine benutzerfreundliche Oberfläche transformiert Wink die Hotelbranche und macht es einfacher denn je, sich zu vernetzen, zu wachsen und erfolgreich zu sein.

Wink besteht aus acht separaten Anwendungen mit Funktionen, die nach Nutzungstyp gruppiert sind.
Beginnen Sie Ihre Reise, indem Sie einen der folgenden Abschnitte auswählen.

## Kernkonzepte

> Sie sehen einen ... Sie haben sie alle gesehen

Wenn Sie sich mit allen Funktionen von Wink vertraut machen, werden Sie feststellen, dass sich die Funktionsweisen häufig überschneiden.
Das Verständnis der Kernkonzepte gibt Ihnen mehr Vertrautheit und mehr Sicherheit beim Durcharbeiten unserer Funktionen.

Auf geht's:

* Normalerweise gibt es in jedem Abschnitt eine Möglichkeit, Einträge aufzulisten, hinzuzufügen, zu aktualisieren und zu entfernen.
* Wenn Sie einen Eintrag bearbeiten, wird ein rotes Sternchen<Badge text="*" variant="danger" size="large" />
  in einem Formularfeld bedeutet, dass es erforderlich ist.
* Wenn ein bestimmtes Formularfeld fehlerhaft ist, wird darum ein roter Rahmen angezeigt. Unter dem Formularfeld wird in rotem Text das Problem erläutert.
* Sie können nur`Save`ein Formular, sobald das Formular gültig ist und einige Änderungen daran vorgenommen wurden.
* Manche Formulare umfassen mehrere Registerkarten, wie etwa das Gästezimmerformular.
* Alle Registerkarten müssen gültig sein, bevor Sie fortfahren können.
* Eine Registerkarte mit fehlenden oder ungültigen Formulardaten wird mit einem gelben Dreieck angezeigt.<Badge text="⚠️" variant="caution" size="large" />
  neben seinem Namen.
* Beim Absenden eines Formulars wird in den meisten Formularen ein`Confirmation`Sie werden gefragt, ob Sie fortfahren möchten.
