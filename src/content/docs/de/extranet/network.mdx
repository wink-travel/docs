Dieser Artikel zeigt Ihnen, wie Sie sich mit engagierten Partnern auf Wink verbinden und bestehende verwalten.

## Vertriebskanäle

Um Ihre Vertriebskanäle zu verwalten, klicken Sie in der Hauptnavigationsleiste auf `Distribution > Sales channels`.

Vertriebskanäle sind Ihre bestehenden Partnerbeziehungen, die mit dem Inventar, den Angeboten und Rabatten, auf die sie Zugriff haben, kodifiziert sind.

Um einen bestehenden Vertriebskanal zu aktualisieren, klicken Sie auf den Link `Actions` unter dem Vertriebskanal, den Sie aktualisieren möchten.

- **Mitgliederrabatt** Der Mitgliedspreis wird Reisenden gewährt, die über diesen Vertriebskanal buchen.
- **Provision** Die Provision wird dem Vertriebskanal für jede erfolgreiche Buchung zugeteilt.
- **Einzelne Aktionen** Fügen Sie die Aktionen hinzu, die dieser Vertriebskanal erhalten soll.
- **Bündelaktionen** Fügen Sie die Bündelaktionen hinzu, die dieser Vertriebskanal erhalten soll.
- **Aktiviert** Schalten Sie den Aktiviert-Schalter um, um den Vertriebskanal vollständig zu aktivieren/deaktivieren.
- **Schwarzgelistet** Wenn ein Vertriebskanal als schwarzgelistet markiert wird, ist es unmöglich, über diesen Vertriebskanal zu buchen.

Wenn Sie mit Ihren Einstellungen zufrieden sind, klicken Sie auf die Schaltfläche `Save`, um fortzufahren.

:::tip
Durch das Schwarzlisten eines Vertriebskanals verhindern Sie, dass dieser Vertriebskanal überhaupt irgendwelche Preise von Ihnen erhält; auch nicht Ihre Wink Network-Preise.
:::

## Netzwerk erkunden

Um neue Partner auf Wink zu finden, klicken Sie in der Hauptnavigationsleiste auf `Distribution > Explore network`.

In diesem Bereich sehen Sie oben auf der Seite ausstehende Anfragen von Partnern und darunter eine Liste von Partnern, mit denen Sie sich verbinden können.

### Auf Anfrage reagieren

Sie können die Partneranfrage ablehnen. Wenn Sie mit der Anfrage fortfahren möchten, werden Sie zur Seite `affiliate details` weitergeleitet, auf der Sie weitere Informationen über den Partner sowie vergangene Leistungskennzahlen auf Wink erhalten.

### Nach Partnern suchen

Im Partnerraster können Sie nach Partnern anhand von Name, Typ und Standort suchen. Wenn Sie einen passenden Partner finden, klicken Sie darauf, und Sie gelangen zur Seite `affiliate details`.

### Mit Partner verbinden

Wenn Sie eine direkte Beziehung mit einem Partner eingehen möchten, werden Sie aufgefordert, die Beziehung genauso zu kodifizieren, wie Sie oben einen Vertriebskanal verwalten würden.

Nachdem Sie den neuen Vertriebskanal gespeichert haben, werden Sie zurück zu Ihrer Liste der Vertriebskanäle geführt.

## API

Entwickler, die ihr `Network` verwalten möchten, können zu [Developers > APIs > Distribution](/de/developers/apis/#distribution-api) wechseln.
