:::note
Dies ist ein Teil der [Shareable Links](/de/studio/shareable-links)-Funktion, die in Wink Studio verfügbar ist.

Immobilien haben automatisch ihren gesamten Bestand als teilbare Links verfügbar, ohne dass Sie zu Wink Studio gehen müssen, um sie zu generieren.
:::

Teilbare Links sind eine Kernfunktion von Wink und entstanden aus dem Bedürfnis, Inventar und Preise auf dynamische Weise über den einfachsten Weg im Internet zu kommunizieren; nämlich den Hyperlink.

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

Oben sehen Sie ein Beispiel für einen unserer Links. Er ist kurz und handlich... sogar kurz genug, um ihn sich zu merken. Obwohl kurz, enthält dieser Link eine **explosive** Menge an Informationen, wie zum Beispiel:

- Ihre App-ID.
- Ihre Affiliate-ID.
- Die Bild(er) und den Text, die dem Nutzer angezeigt werden.
- Das Aussehen (*d.h. Thema, Farben und mehr*).
- Die Reiseroute, die für unsere Preisanfrage verwendet wird (*2 Erwachsene, 1 Nacht*).
- Die Währung, in der der Preis angezeigt wird.
- Die Sprache, in der der Text angezeigt wird.

Der Rest dieses Artikels führt Sie durch die Verwaltung der Links zu Ihren Beständen.

## Link anpassen

 Das Formular ermöglicht es Ihnen, Ihren Link auf folgende Weise anzupassen:

<Steps>
 1. Standardmäßig erhält der Link den Namen Ihres Bestands auf Englisch. Sie können ihm jeden Namen geben, der Ihnen hilft, sich zu merken, worum es bei dem Link geht.
 2. Standardmäßig wird Ihre primäre Markenanpassung auf den Link angewendet. Sie können andere [Anpassungen](/de/studio/customization) auswählen, die Sie in Wink Studio erstellt haben.
 3. Fügen Sie Schlüsselwörter hinzu, getrennt durch Kommas, die von [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler) verwendet werden.
 4. Optional können Sie Ihren X-Account-Handle oder Facebook App ID eingeben.
 5. Fügen Sie Titel und Beschreibungen in den Sprachen hinzu, die der Nutzer sehen soll.
 6. Wählen Sie ein oder mehrere Bilder aus, die angezeigt werden, wenn Sie den Link auf Plattformen wie Facebook oder LinkedIn teilen.
 7. Wählen Sie ein Thema / Overlay, das ein gebrandetes Design über das/die ausgewählte(n) Bild(er) legt. *Hinweis: So werden Preise über dem Bild angezeigt.*
 8. Klicken Sie auf die Schaltfläche `Save`, um fortzufahren.
 </Steps>

Oben sehen Sie ein Bild mit allen verfügbaren Aktionen für Ihren Link:

- **Vorschau** Zeigt Ihnen eine schnelle Übersicht, wie der Link konfiguriert wurde und wie er für den Nutzer aussehen wird.
- **Aktualisieren** Aktualisiert Ihre Link-Konfiguration.
- **Zu WinkLinks hinzufügen** Fügt den Link Ihrem WinkLinks-Konto hinzu.
- **Kopieren** Kopiert den Link in Ihre Zwischenablage, damit Sie ihn überall einfach teilen können (*einschließlich [WinkLinks](/de/link-manager/wink-links) *).
- **Teilen** Öffnet ein neues Fenster mit Optionen, um auf vielen sozialen Netzwerken und Messaging-Plattformen zu teilen.
- **QR-Code** Generiert einen QR-Code, den Sie auf Plattformen wie IG teilen können, die bildfokussierter sind.
- **Einbetten** Zeigt Ihnen, wie Sie diesen Link als [Ad Banner](/de/developers/web-components/#content-loader) in Ihre Website einbetten.
- **Mit WordPress verwenden** Zeigt Ihnen, wie Sie unser [WordPress-Plugin](/de/developers/wordpress) verwenden, um diesen Link in Ihre Website einzubetten.

Einige dieser Optionen werden im Folgenden ausführlicher behandelt.

Klicken Sie auf eine der Symbolschaltflächen, um Ihren Link in diesem Netzwerk zu posten. *Hinweis: Einige Schaltflächen werden nur angezeigt, wenn Sie von Ihrem Mobilgerät aus teilen.*

Wenn Sie lieber einen QR-Code statt eines Links teilen möchten, klicken Sie auf die Schaltfläche `QR code` und es öffnet sich ein neues Fenster mit Ihrem QR-Code.

So speichern Sie Ihren QR-Code:

- Wenn Sie an Ihrem Laptop sind, klicken Sie mit der rechten Maustaste auf das Bild und wählen Sie `Bild speichern`.
- Wenn Sie auf Ihrem Mobiltelefon sind:
    - Drücken und halten Sie Ihren Finger auf dem Bild, bis eine Option erscheint, das Bild in Ihren Fotos zu speichern.
    - Machen Sie einen Screenshot des QR-Codes, schneiden Sie ihn passend zu und speichern Sie ihn in Ihren Fotos.

### Einbetten

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

So betten Sie Ihren Link als Werbebanner in Ihre Website ein:

- Zeile 3 zeigt, wie Sie die Wink-Styles in Ihre Website einbinden.
- Die Zeilen 6 bis 9 zeigen, wie Sie die [wink-content-loader](/de/developers/web-components/#content-loader) Web-Komponente verwenden und ihr mitteilen, ein Werbebanner für Ihren Code abzurufen.
- Zeile 11 zeigt, wie Sie unser Javascript in Ihre Website einbinden.
- Zeile 13 zeigt, wie Sie die [wink-app-loader](/de/developers/web-components/#application-loader) Web-Komponente einbetten und ihr mitteilen, Ihre Seitenkonfigurationspräferenzen abzurufen.
