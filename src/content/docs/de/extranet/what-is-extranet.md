[Wink Extranet](https://extranet.wink.travel) ist unser Portal für Unterkünfte. Es richtet sich an Hotels, Ketten, Marken und Hotelmanagement-Unternehmen, um das digitale Profil ihrer Unterkünfte auf der Wink-Plattform zu erstellen und zu verwalten.

Unser Extranet wurde bewusst so einfach wie möglich gestaltet, unterstützt aber dennoch alle Funktionen, die Power-User von einem fortschrittlichen Unterkunftsportal erwarten. Hier ein paar Beispiele:

1. Einfachheit: Nutzen Sie unseren intelligenten Onboarding-Workflow, um Ihre Unterkunft anzumelden und in weniger als einer Stunde LIVE zu gehen.
2. Fortgeschritten: Wink unterstützt 8500 verschiedene Varianten von Stornierungsbedingungen.
3. Cool: Gewähren Sie einer einzelnen Person auf der Welt 10 % Rabatt, ohne dass sich der Nutzer einloggen oder einen Promo-Code eingeben muss.

Lassen Sie sich von unseren Assistenten durch den gesamten Prozess führen. Wenn Sie sich sicher fühlen... legen Sie selbst los.

Wink ist ein fortschrittliches Content-Management-System für Unterkünfte und Erlebnisanbieter. Alles, was Sie in einer unserer Apps tun können, können Sie auch über die API erledigen. So ist eine tiefe Integration mit Wink möglich, sodass Sie über Wink verkaufen können, ohne sich jemals in eine unserer Apps einloggen zu müssen.

:::tip[Profi-Tipp]
Zahlen Sie für teures Hosting von Medieninhalten über Leonardo International? Sie können genau dasselbe über Wink machen UND erhalten viele weitere Optionen, um Ihre Bilder und Videos zu bearbeiten und fein abzustimmen.
:::

All diese Funktionen sind für Sie kostenlos. Erst wenn Sie eine Buchung erhalten, erheben wir eine kleine Gebühr. Nichts zu verlieren – alles zu gewinnen.
