:::Hinweis
Ähnlich wie die Flugbranche benötigen Hotels viel Flexibilität bei Preisen und Rabatten.
Mit Sonderangeboten kann das Hotel einzigartige Konditionen schaffen, unter denen Reisende Anspruch auf einen zusätzlichen Preisnachlass \[oder einen Aufschlag] haben.
:::

Um Ihre Werbeaktionen zu verwalten, klicken Sie auf`Monetize > Single Promotions`aus der Hauptnavigationsleiste.

Um eine neue Aktion zu erstellen, klicken Sie auf das`Create single promotion` Taste.

## Konfiguration

In diesem Abschnitt wird der tatsächliche Geldwert der Aktion festgelegt.

* **Name**Geben Sie einen Namen für diese Aktion ein, den nur Sie sehen können.
* **Typ**Wählen Sie, ob diese Aktion einen Rabatt oder eine Prämie enthalten soll.
* **Preistyp**Wählen Sie, wie dieses Paket berechnet werden soll.
  * `Per stay`: Der Betrag wird auf die letzte Nacht angewendet.*zB 100 % Rabatt > Bleiben Sie 2 Nächte – die 3. Nacht ist gratis.*
  * `Per night`: Betrag multipliziert mit der Anzahl der Nächte.
  * `Per use`: Es erfolgt keine zusätzliche Berechnung.
  * `Per person`: Betrag multipliziert mit der Anzahl der Gäste.
  * `Per person per night`: Betrag multipliziert mit Anzahl der Nächte + Anzahl der Gäste.
* **Betragsart**Wählen Sie, ob der Betragsmodifikator ein fester Betrag oder ein Prozentsatz sein soll.
* **Status**Mit dem Schalter können Sie steuern, ob die Aktion über alle Vertriebskanäle hinweg verfügbar ist.

## Beschreibungen

:::Hinweis
Sie müssen mindestens eine Beschreibung in englischer Sprache verfassen.
:::

Klicken Sie auf das`Descriptions`Registerkarte, um fortzufahren.

Eine lokalisierte Beschreibung erklärt Reisenden in ihrer Sprache, worum es bei diesem Angebot geht.
Sie können in beliebig vielen Sprachen schreiben. Wir fügen Übersetzungen für alle gängigen Sprachen hinzu.
Dies ist der Text, der Reisenden beim Betrachten Ihres Angebots angezeigt wird.

* **Beschreibung**Geben Sie der Aktion einen Namen.*z. B. Frühbucherrabatt – 20 % Rabatt*

## Einschränkungen

:::Hinweis
Beachten Sie, dass es sich hierbei um Einschränkungen handelt. Aktivieren Sie diese nur, wenn Sie Ihre Werbung weniger zugänglich gestalten möchten.
:::

Klicken Sie auf das`Restrictions`Registerkarte, um fortzufahren.

Die Werbemaschine von Wink verfügt über nicht weniger als 20 einzigartige Möglichkeiten, eine Preisänderung auszulösen.

Wir unterstützen die folgenden Kriterien:

* Beschränken Sie die Werbung auf bestimmte Add-Ons.
* Beschränken Sie die Aktion auf bestimmte Haupttarife.
* Beschränken Sie die Aktion auf bestimmte Tarifpläne.
* Beschränken Sie die Aktion auf bestimmte Stornierungsbedingungen.
* Beschränken Sie die Aktion auf die Aufenthaltsdauer.
* Beschränken Sie die Aktion auf Vorausbuchungen.
* Beschränken Sie die Aktion auf die Anzahl der Zimmer.
* Beschränken Sie die Aktion auf einen Aktionscode.
* Beschränken Sie die Werbung auf den IP-Bereich des Reisenden.
* Beschränken Sie die Aktion auf die letzte Minute.
* Beschränken Sie die Promo auf die Stadt des Reisenden.
* Beschränken Sie die Aktion auf das Land des Reisenden.
* Beschränken Sie die Aktion auf den Kontinent der Reisenden.
* Beschränken Sie die Werbeaktion auf die Zeitzone des Reisenden.
* Beschränken Sie die Aktion auf das/die Verkaufsdatum(e).
* Beschränken Sie die Aktion auf das/die Aufenthaltsdatum(e).
* Beschränken Sie die Aktion auf Buchungstage in der Woche.
* Beschränken Sie die Aktion auf die Ankunftstage in der Woche.
* Beschränken Sie die Aktion auf die Abflugtage in der Woche.
* Beschränken Sie die Aktion auf die erforderlichen Wochentage.

Lesen Sie weiter, um zu erfahren, wie Sie die einzelnen Elemente verwenden.

### Hinzufügen Auf

:::Hinweis
Um diese Funktion nutzen zu können, müssen Sie mindestens ein Add-on erstellt haben.
:::

Damit eine Aktion nur auf die von Ihnen ausgewählten Add-Ons angewendet wird, führen Sie die folgenden Schritte aus:

<Steps>
  1. Aktivieren Sie die `Add-on restriction` schalten.
  2. Aktivieren Sie das Kontrollkästchen neben jedem Add-on, das Sie einschließen möchten.
</Steps>

<Steps>

  2. Aktivieren Sie das Kontrollkästchen neben jedem Master-Tarif, den Sie einschließen möchten.
</Steps>


### Tarifplan

Damit eine Aktion nur für die von Ihnen ausgewählten Tarifpläne gilt, gehen Sie folgendermaßen vor:

<Steps>

  2. Aktivieren Sie das Kontrollkästchen neben jedem Tarifplan, den Sie einschließen möchten.
</Steps>


### Rückerstattungsfähig

  2. Aktivieren Sie das Kontrollkästchen neben jeder Stornierungsrichtlinie, die Sie einschließen möchten.

### Aufenthaltsdauer

:::Hinweis
Diese Einschränkung verwenden Sie normalerweise als`long term` Förderung.
:::

Wenn Sie diese Einschränkung hinzufügen, müssen Ihre Gäste`stay at least` Und `no more than`eine bestimmte Anzahl von Tagen.

Befolgen Sie diese Schritte, um`Length of Stay`:

<Steps>
  1. Aktivieren Sie die `Length of stay restriction` schalten.
  2. Geben Sie die Mindestanzahl an Tagen ein, die ein Gast bleiben muss. *z.B. 7*
  3. Geben Sie optional die maximale Anzahl von Tagen ein, die ein Gast bleiben kann. *z.B. 90*
</Steps>


*Beispiel: Dieses Angebot gilt nur für Gäste, die länger als 10 Tage bleiben. Hinweis: Die maximale Aufenthaltsdauer ist optional.*

### Reservierung

:::Hinweis
Diese Einschränkung verwenden Sie normalerweise als`early bird` Förderung.
:::

Wenn Sie diese Einschränkung hinzufügen, müssen Ihre Gäste`book at least` Und `no more than`eine bestimmte Anzahl von Tagen im Voraus.

Befolgen Sie diese Schritte, um`Advance booking`:

<Steps>
  1. Aktivieren Sie die `Advance booking restriction` schalten.
  2. Geben Sie die Mindestanzahl von Tagen ein, die ein Gast im Voraus buchen muss. *z.B. 10*
  3. Geben Sie optional die maximale Anzahl Tage ein, die ein Gast im Voraus buchen muss. *z.B. 180*
</Steps>


*Beispiel: Dieses Angebot gilt nur für Gäste, die mehr als 10 Tage im Voraus buchen. Hinweis: Sie müssen außerdem eine maximale Vorausbuchung angeben.*

### Anzahl der Zimmer

:::Hinweis
Die Aktion ist nur verfügbar, wenn der Gast die Mindestanzahl an Zimmern bucht.
:::

Wenn Sie diese Einschränkung hinzufügen, müssen Ihre Gäste`book at least` Und `no more than`eine bestimmte Anzahl Zimmer als Teil ihrer Buchung.

Befolgen Sie diese Schritte, um`Number of rooms`:

<Steps>
  1. Aktivieren Sie die `Room restriction` schalten.
  2. Geben Sie die Mindestanzahl an Zimmern ein, die ein Gast buchen muss. *z.B. 3*
  3. Geben Sie die maximale Anzahl an Zimmern ein, die ein Gast buchen muss. *z.B. 8*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die mehr als 3 Zimmer buchen.*

### Aktionscode

:::Hinweis
Verwenden Sie diese Einschränkung, wenn der Gast einen Rabattcode eingeben muss, um diese Aktion freizuschalten.
:::

Befolgen Sie diese Schritte, um`Promo code`:

<Steps>
  1. Aktivieren Sie die `Promotion restriction` schalten.
  2. Klicken Sie auf das `Add promo code` Taste.
  3. Geben Sie den Aktionscode ein. *z.B. ABC123*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die den Aktionscode eingeben`ABC123`.*

### IP-Bereich

:::Hinweis
Diese Einschränkung ist sinnvoll, wenn Sie eine Cross-Promotion-Kampagne mit Mobilfunkanbietern oder größeren Unternehmen mit eigenen IP-Bereichen durchführen möchten.
:::

Befolgen Sie diese Schritte, um`IP range`:

<Steps>
  1. Aktivieren Sie die `IP range restriction` schalten.
  2. Klicken Sie auf das `Add IP range restriction` Taste.
  3. Geben Sie den anfänglichen \[global zugänglichen] IP-Bereich ein. *z.B. 203.0.113.50*
  4. Geben Sie den letzten \[global erreichbaren] IP-Bereich ein. *z. B. 203.0.113.59*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste mit IP-Nummern zwischen`203.0.113.50`-`203.0.113.59`.*

### Last Minute

:::Hinweis
Dies ist der Zeitraum des Last-Minute-Angebots, den Sie einschränken möchten.
:::

Befolgen Sie diese Schritte, um`Last minute`:

<Steps>
  1. Aktivieren Sie die `Last minute restriction` schalten.
  2. Klicken Sie auf das `Add IP range restriction` Taste.
  3. Geben Sie die Anzahl der Stunden vor dem Ankunftsdatum ein. *zB 24 Stunden*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die 24 Stunden vor dem Ankunftsdatum buchen.*

### Stadt

:::Hinweis
Ermöglicht die Einschränkung auf die Stadt, in der sich der Reisende aktuell befindet. Erstellen Sie für jede Stadt, die Sie einbeziehen möchten, eine Einschränkung.
:::

Befolgen Sie diese Schritte, um`City`:

<Steps>
  1. Aktivieren Sie die `City restriction` schalten.
  2. Beginnen Sie mit der Eingabe des Städtenamens.
  3. Wählen Sie die Stadt aus den Suchergebnissen aus. *z. B. New York City*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die buchen von`New York City`.*

### Land

:::Hinweis
Ermöglicht die Einschränkung auf das Land, in dem sich der Reisende aktuell aufhält. Erstellen Sie für jedes einzuschließende Land eine Einschränkung.
:::

Befolgen Sie diese Schritte, um`Country`:

<Steps>
  1. Aktivieren Sie die `Country restriction` schalten.
  2. Beginnen Sie mit der Eingabe des Ländernamens.
  3. Wählen Sie das Land aus den Suchergebnissen aus. *z. B. Vereinigte Staaten von Amerika*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die buchen von`United States of America`.*

### Kontinent

:::Hinweis
Hier können Sie den Preis auf den Kontinent beschränken, auf dem sich der Gast aktuell befindet. Erstellen Sie für jeden Kontinent, den Sie berücksichtigen möchten, eine Einschränkung.
:::

Befolgen Sie diese Schritte, um`Continent`:

<Steps>
  1. Aktivieren Sie die `Continent restriction` schalten.
  2. Beginnen Sie mit der Eingabe des Kontinentnamens.
  3. Wählen Sie den Kontinent aus den Suchergebnissen aus. *z.B. Nordamerika*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die buchen von`North America`.*

### Zeitzone

:::Hinweis
Hier können Sie den Preis auf die Zeitzone beschränken, in der sich der Gast aktuell befindet. Erstellen Sie für jede einzubeziehende Zeitzone eine Einschränkung.
:::

Befolgen Sie diese Schritte, um`Timezone`:

<Steps>
  1. Aktivieren Sie die `Timezone restriction` schalten.
  2. Beginnen Sie mit der Eingabe des Namens der Zeitzone.
  3. Wählen Sie die Zeitzone aus den Suchergebnissen aus. *z. B. America/New\_York*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die buchen von`America/New_York`.*

### Verkaufsdatum

:::Hinweis
Ermöglicht die Einschränkung auf einen bestimmten Zeitraum. Reisende sind berechtigt, ein Zimmer innerhalb dieses Zeitraums zu buchen.
:::

Befolgen Sie diese Schritte, um`Sell date`:

<Steps>
  1. Aktivieren Sie die `Sell date restriction` schalten.
  2. Klicken Sie auf das `Add sell date restriction` Taste.
  3. Geben Sie das Startdatum ein. *z.B. 19. September 2024*
  4. Geben Sie das Enddatum ein. *z. B. 1. Dezember 2024*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die buchen zwischen`Sept. 19th 2024` Und `Dec. 1st 2024`.*

### Aufenthaltsdatum

:::Hinweis
Ermöglicht die Einschränkung auf einen bestimmten Zeitraum. Reisende sind berechtigt, ein Zimmer für die angegebenen Daten innerhalb dieses Zeitraums zu buchen.
:::

Befolgen Sie diese Schritte, um`Stay date`:

<Steps>
  1. Aktivieren Sie die `Stay date restriction` schalten.
  2. Klicken Sie auf das `Add stay date restriction` Taste.
  3. Geben Sie das Startdatum ein. *z.B. 19. September 2024*
  4. Geben Sie das Enddatum ein. *z. B. 1. Dezember 2024*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die einen Aufenthalt buchen zwischen`Sept. 19th 2024` Und `Dec. 1st 2024`.*

### Buchungstage der Woche

:::Hinweis
Wenn Sie diese Einschränkung aktivieren, ist dieser Preis nur an dem angegebenen Wochentag verfügbar, an dem die Buchung erfolgt.

Diese Einschränkung passt gut zur „Verkaufsdatumsbeschränkung“, mit der Sie einen Datumsbereich einschränken und den Datumsbereich mithilfe dieser Einschränkung durch Wochentage weiter eingrenzen können.
:::

Befolgen Sie diese Schritte, um`Booking days`:

<Steps>
  1. Aktivieren Sie die `Booking days restriction` schalten.
  2. Aktivieren Sie einen der Wochentagsschalter. *z.B. Montag*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die einen Aufenthalt buchen auf einem`Monday`.*

:::Hinweis
Wenn Sie diese Einschränkung aktivieren, müssen Gäste an diesem Tag anreisen, um diesen Preis sehen zu können.

Diese Einschränkung passt gut zur „Einschränkung des Aufenthaltsdatums“, mit der Sie den Datumsbereich und die Wochentage weiter einschränken können.
:::

Befolgen Sie diese Schritte, um`Arrival days`:

<Steps>
  1. Aktivieren Sie die `Arrival days restriction` schalten.
  2. Aktivieren Sie einen der Wochentagsschalter. *z.B. Montag*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die einen Aufenthalt buchen, der beginnt an einem`Monday`.*

### Abreisetage der Woche

:::Hinweis
Wenn Sie diese Einschränkung aktivieren, müssen Gäste an diesem Tag abreisen, um diesen Preis sehen zu können.

Diese Einschränkung passt gut zur „Einschränkung des Aufenthaltsdatums“, mit der Sie den Datumsbereich und die Wochentage weiter einschränken können.

Befolgen Sie diese Schritte, um`Departure days`:

<Steps>

</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die einen Aufenthalt buchen, der endet an einem`Monday`.*

### Erforderliche Wochentage

:::Hinweis
Wenn Sie diese Einschränkung aktivieren, müssen Gäste an mindestens einem dieser Tage übernachten, um diesen Preis zu sehen.

Diese Einschränkung passt gut zur „Einschränkung des Aufenthaltsdatums“, mit der Sie den Datumsbereich und die Wochentage weiter einschränken können.
:::

Befolgen Sie diese Schritte, um`Required days`:

<Steps>
  1. Aktivieren Sie die `Required days restriction` schalten.
  2. Aktivieren Sie einen der Wochentagsschalter. *z.B. Montag*
</Steps>


*Beispiel: Diese Aktion ist nur für Gäste verfügbar, die einen Aufenthalt buchen, der einen`Monday`.*

## Sperrdaten

Klicken Sie auf das`Blackout dates`Registerkarte, um fortzufahren.

Kontrollieren Sie die Aktion weiter, indem Sie Daten angeben, an denen diese Aktion nicht verfügbar sein soll.

Um ein Sperrdatum hinzuzufügen, klicken Sie auf das`Add blackout date` Taste.

* **Startdatum**Wählen Sie das Startdatum Ihres Sperrdatums aus.
* **Enddatum**Wählen Sie das Enddatum Ihres Sperrdatums aus.

Wenn die Buchungsdaten diesen Datumsbereich berühren, wird die Aktion nicht wirksam.

## Pakete

Um Ihre Bundles zu verwalten, klicken Sie auf`Monetize > Bundled Promotions`aus der Hauptnavigationsleiste.

Ein Reisender kann gleichzeitig Anspruch auf mehrere Angebote haben, z. B. Langzeit- und Frühbucherangebot.
Ohne ein gebündeltes Angebot hätte der Reisende nur Anspruch auf eines davon: das beste Angebot.

Durch die Erstellung einer gebündelten Werbeaktion berücksichtigen Sie die Berechtigung des Reisenden zu beiden Werbeaktionen und können diese weiter anpassen.

Um eine gebündelte Promo zu erstellen, klicken Sie auf das`Create bundled promotion` Taste.

### Konfiguration

* **Name**Geben Sie dem Werbepaket einen Namen, den nur Sie sehen können.
* **Status**Mit dem Schalter können Sie steuern, ob das Paket über alle Vertriebskanäle verfügbar ist.
* **Aktionen**Wählen Sie mindestens 2 einzelne Promos aus, um ein Paket zusammenzustellen.

### Überschreibungsbetrag

Sie können den Gesamtrabatt überschreiben. Wenn Sie ihn weglassen, wird der Rabatt aller unten ausgewählten Aktionen zu einem Rabatt zusammengerechnet.

* **Überschreibungsbetrag**Ob Sie alle Rabattbeträge zusammenfassen oder mit Ihrem eigenen Betrag überschreiben möchten.
* **Typ**Wählen Sie aus, ob der Preis erhöht oder gesenkt werden soll.*z.B. Einstellen auf`Discount`für eine Preisminderung.*
* **Preistyp**Wählen Sie, wie dieses Paket berechnet werden soll.
  * `Per stay`: Der Betrag wird auf die letzte Nacht angewendet.*zB 100 % Rabatt > Bleiben Sie 2 Nächte – die 3. Nacht ist gratis.*
  * `Per night`: Betrag multipliziert mit der Anzahl der Nächte.
  * `Per use`: Es erfolgt keine zusätzliche Berechnung.
  * `Per person`: Betrag multipliziert mit der Anzahl der Gäste.
  * `Per person per night`: Betrag multipliziert mit Anzahl der Nächte + Anzahl der Gäste.
* **Betragsart**Wählen Sie, ob der Betragsmodifikator ein fester Betrag oder ein Prozentsatz sein soll.
