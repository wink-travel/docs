Eine Agentur ist ein normales Wink-Konto, das Sie Ihrem Konto für Vertretungszwecke zuweisen können. Nachfolgend einige Gründe, warum Sie eine Agentur als Vertreter wünschen könnten:

- Vollständige / teilweise Verwaltung Ihres Wink-Kontos durch Dritte.
- Die Immobilie hat eine bestehende Vereinbarung mit ihrer Verwaltungsgesellschaft und möchte, dass Wink den finanziellen Aspekt der Abwicklung der über Wink eingehenden Gelder übernimmt.
- Die Immobilienverwaltungsagentur erledigt alle Online-Arbeiten im Namen ihrer Immobilien.

## Agentur zuweisen

So weisen Sie eine Agentur Ihrem Konto zu.

<Steps>
 1. Melden Sie sich in Ihrem Benutzerkonto auf [Wink Extranet](https://extranet.wink.travel) an.
 2. Wählen Sie die Immobilie aus, mit der Sie arbeiten möchten.
 3. Klicken Sie in der Hauptnavigationsleiste auf `Konto > Manager`.
 4. Geben Sie im Abschnitt `Agentur` den Namen der Agentur ein, die Sie vertreten soll.
 5. Wählen Sie die Agenturgebühr aus dem Dropdown-Menü.
 5. Optional können Sie Ablaufregeln hinzufügen.
 7. Klicken Sie auf die Schaltfläche `Agentur festlegen`, um fortzufahren.
 </Steps>

 :::note
Sie müssen Eigentümer der Immobilie sein, um eine Agentur zuzuweisen.
 :::

 :::tip[Beispiel]
Die Agenturgebühr wird aus der Buchung berechnet.

Buchung = 100 $, Agenturgebühr = 1 % => 100 * 0,01 = 1 $
 :::

## Agentur entfernen

So entfernen Sie eine Agentur von Ihrem Konto.

<Steps>
 1. Melden Sie sich in Ihrem Benutzerkonto auf [Wink Extranet](https://extranet.wink.travel) an.
 2. Wählen Sie die Immobilie aus, mit der Sie arbeiten möchten.
 3. Klicken Sie in der Hauptnavigationsleiste auf `Konto > Manager`.
 4. Klicken Sie im Abschnitt `Agentur` auf das <Badge text="x" variant="danger" size="small" /> Symbol rechts neben dem Agentur-Dropdown.
 5. Klicken Sie auf die Schaltfläche `Agentur festlegen`, um fortzufahren.
 </Steps>


## API

Entwickler, die die Agenturverwaltung über die API steuern möchten, können zu [Entwickler > API > Affiliate](/developers/apis/#affiliate-api) wechseln.
