Die Erstellung Ihres ersten Tarifplans ist Teil des Onboarding-Assistenten.

Um Ihre Tarifpläne zu verwalten, klicken Sie auf`Monetize > Rate Plans`aus der Hauptnavigationsleiste.

Preispläne dienen als Filter, über den den Nutzern Gästezimmer angezeigt werden.
Sie enthalten außerdem Einstellungen, die die Preisstruktur in bestimmten Szenarien steuern.

Mit den Tarifplänen können Sie:

* Legen Sie eine Stornierungsrichtlinie \[und Richtlinienausnahmen] fest.
* Preis inkl. MwSt., zzgl. Kinder, Erwachsene und Einzelreisende.
* Feste Menüs.
* Fügen Sie Gebühren für frühes Einchecken und spätes Auschecken hinzu.
* Legen Sie zusätzliche Gebühren fest.
* Legen Sie Einschränkungen fest.

Um einen Tarifplan zu erstellen, klicken Sie auf das`Create a new rate plan` Taste.

## Konfiguration

Der `Configurations`Die Registerkarte ist beim Start standardmäßig ausgewählt.

In diesem Abschnitt werden die grundlegenden Optionen zur Konfiguration Ihres Tarifplans behandelt.

* **Name des Tarifplans**Der Name des Tarifplans, wie Sie ihn intern bezeichnen.*z. B. BAR RO*
* **Treuepunkte**Ob der Gast mit diesem Tarif Ihre Treuepunkte sammelt.*z.B. Nein*
* **Einzelbelegung**Legen Sie einen Preismodifikator für einen einzelnen Bewohner in einem Mehrbettzimmer fest.*z.B. 5% Rabatt auf den regulären Preis*
* **Zusätzlicher Erwachsener**Legen Sie einen Preismodifikator für einen zusätzlichen Erwachsenen im Zimmer fest.*z.B. +10 / Erwachsener*
* **Zusätzliches Kind**Legen Sie einen Preismodifikator für ein zusätzliches Kind im Zimmer fest.*z.B. +5 / Kind*

### Stornierungsbedingungen

* **Stornierungsbedingungen**Legen Sie die Stornierungsbedingungen für diesen Tarifplan fest.*z. B. Erstattungsfähig*
* **Ausnahmen**Legen Sie zusätzliche Stornierungsbedingungen fest, die für bestimmte Datumsbereiche gelten.*z.B. Für den Monat Dezember verwenden Sie die`Refundable with extra conditions` Politik.*

### Mahlzeiten

Legen Sie die Mahlzeiten fest, die zu diesem Tarifplan gehören.*z.B. Frühstück inklusive*

## Gebühren

Legen Sie zusätzliche Gebühren fest, die mit dem Tarifplan verbunden sind.

* **Frühes Einchecken**Legen Sie einen Aufpreis fest, wenn der Gast früher einchecken möchte.*Dies dient nur zu Informationszwecken*
* **Spätes Auschecken**Legen Sie eine zusätzliche Gebühr fest, wenn der Gast spät auschecken möchte.*Dies dient nur zu Informationszwecken*

### Zusätzliche Kosten

Legen Sie zusätzliche Gebühren mit dem Tarifplan fest.

Um eine zusätzliche Gebühr zu erstellen, klicken Sie auf das`Create extra charge` Taste.

* **Preistyp**Wählen Sie, wie diese Gebühr berechnet wird.*z.B. Pro Aufenthalt*
* **Menge**Legen Sie den festen Betrag der Gebühr fest.*z.B. 100*

Geben Sie den Namen und die Beschreibung dieser Gebühr ein.

:::Hinweis
Sie müssen mindestens eine Beschreibung in englischer Sprache verfassen.
:::

* **Name**Name der Gebühr.*zB Einmalige Reinigungsgebühr*
* **Beschreibung**Beschreiben Sie die Anklage in ein oder zwei Absätzen.*z.B.: Die Villa wird vor Ihrer Ankunft von unserem professionellen Reinigungsteam gereinigt.*

## Einschränkungen

Legen Sie Einschränkungen fest, die steuern, ob ein Gästezimmer/Preisplan verfügbar ist oder nicht.

Unterstützte Einschränkungen:

* Aufenthaltsdauer
* Reservierung
* Belegung
* Alter
* Verkaufen / Aufenthalt / Buchen / Anreise / Abreise / Wunschdaten

### Aufenthaltsdauer

Wenn Sie diese Einschränkung hinzufügen, verlangen Sie von Ihren Gästen, dass sie „mindestens“ und „nicht länger“ eine bestimmte Anzahl von Tagen bleiben.

*Beispiel: Dieser Zimmertyp ist nur für Gäste verfügbar, die länger als 10 Tage bleiben. Hinweis: Die maximale Aufenthaltsdauer ist optional.*

### Reservierung

Wenn Sie diese Einschränkung hinzufügen, verlangen Sie von Ihren Gästen, dass sie „mindestens“ und „nicht mehr als“ eine bestimmte Anzahl von Tagen im Voraus buchen.

*Beispiel: Dieses Angebot gilt nur für Gäste, die mehr als 10 Tage im Voraus buchen. Hinweis: Sie müssen außerdem eine maximale Vorausbuchung angeben.*

### Belegung

Wenn Sie diese Einschränkung hinzufügen, verlangen Sie, dass die ankommende Gruppe „mindestens“ und „nicht mehr als“ die angegebene Anzahl an Personen umfasst.

### Alter

Wenn Sie diese Einschränkung hinzufügen, verlangen Sie, dass die Gäste „mindestens“ und „nicht älter“ als das angegebene Alter sind.

### Verkaufsdaten

Mit dieser Einschränkung können Sie diesen Preis nur innerhalb eines bestimmten Zeitraums buchen. Ein Gast sieht diesen Preis nur, wenn er innerhalb dieses Zeitraums ein Zimmer am angegebenen Datum bucht.

### Aufenthaltsdaten

Mit dieser Einschränkung können Sie diesen Preis nur für Aufenthalte innerhalb eines bestimmten Zeitraums verfügbar machen. Ein Gast sieht diesen Preis nur, wenn er ein Zimmer für das angegebene Datum innerhalb dieses Zeitraums bucht.

### Buchungsdaten

:::Hinweis
Diese Einschränkung passt gut zur „Verkaufsdatumsbeschränkung“, mit der Sie einen Datumsbereich und den Datumsbereich mithilfe dieser Einschränkung durch Wochentage weiter einschränken können.
:::

Durch Aktivieren dieser Einschränkung erlauben Sie die Suche nach diesem Tarif NUR an den angegebenen Tagen.

### Anreisetermine

:::Hinweis
Diese Einschränkung passt gut zur „Einschränkung des Aufenthaltsdatums“, mit der Sie den Datumsbereich und die Wochentage zusätzlich einschränken können.
:::

Wenn Sie diese Einschränkung aktivieren, MÜSSEN Gäste an diesem Tag anreisen, um diesen Preis zu sehen.

*Beispiel: Die Aktivierung von Montag erfordert, dass der Gast an einem Montag ankommt.*

### Abreisedaten

:::Hinweis
Diese Einschränkung passt gut zur „Einschränkung des Aufenthaltsdatums“, mit der Sie den Datumsbereich und die Wochentage zusätzlich einschränken können.
:::

Wenn Sie diese Einschränkung aktivieren, MÜSSEN Gäste an diesem Tag abreisen, um diesen Preis zu sehen.

*Beispiel: Die Aktivierung von Montag erfordert, dass der Gast an einem Montag abreist.*

### Erforderliche Daten

:::Hinweis
Diese Einschränkung passt gut zur „Einschränkung des Aufenthaltsdatums“, mit der Sie den Datumsbereich und die Wochentage zusätzlich einschränken können.
:::

Wenn Sie diese Einschränkung aktivieren, MÜSSEN Gäste an „mindestens einem“ dieser Tage übernachten, um diesen Preis zu sehen.

*Beispiel: Wenn Sie „Montag“ aktivieren, müssen die Gäste am Montag übernachten.*
