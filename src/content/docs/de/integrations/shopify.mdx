Die **Wink Shopify App** ermöglicht es Reiseveranstaltern, Erlebnisanbietern und Gastgewerbe-Händlern, das Live-Wink-Inventar direkt in ihrem Shopify-Shop anzuzeigen – neben Produkten, auf Landingpages oder überall im Theme.

## Installation

<Steps>
1. Gehen Sie zum [Shopify App Store](https://apps.shopify.com) und suchen Sie nach **Wink Travel**.
2. Klicken Sie auf **App hinzufügen** und folgen Sie dem OAuth-Prozess zur Autorisierung.
3. Geben Sie in den Wink-App-Einstellungen Ihre **Client ID** und **Client Secret** ein.
   Rufen Sie Ihre Zugangsdaten unter [Applications](/de/portal/settings/applications) ab.
4. Klicken Sie auf **Überprüfen & Speichern**. Eine Erfolgsmeldung bestätigt die Verbindung.
</Steps>

## Aktivieren Sie den App Embed Block

Der App Embed Block fügt automatisch das Wink-Skript und Stylesheet auf jeder Seite ein – Sie müssen keinen Theme-Code bearbeiten.

<Steps>
1. Gehen Sie im Shopify-Adminbereich zu **Onlineshop → Themes → Anpassen**.
2. Klicken Sie im linken Bereich auf **App-Embed-Blocks**.
3. Schalten Sie **Wink Travel** ein.
4. Klicken Sie auf **Speichern**.
</Steps>

:::note
Der App Embed Block liest Ihre Client ID aus einem Shopify-Metafeld, das während der Installation gesetzt wurde. Ihr Client Secret wird niemals im Browser offengelegt.
:::

## Fügen Sie eine Komponente zu einer Seite hinzu

<Steps>
1. Öffnen Sie im Theme-Editor die Seite oder den Abschnitt, auf dem Sie Wink-Inhalte anzeigen möchten.
2. Klicken Sie auf **Block hinzufügen** und wählen Sie aus den Wink-Blöcken:
   - **Wink Content** — Hotelkarten, Raster oder Karten
   - **Wink Lookup** — Suchleiste für Reiseziele
   - **Wink Search** — Button zum Auswählen der Reiseroute
   - **Wink Account** — Anmeldebutton
   - **Wink Itinerary** — Button für den Reiseroutenstatus
   - **Wink Cart** — Warenkorb-Button
3. Wählen Sie für **Wink Content** einen **Layout-Typ** und geben Sie die **Layout-ID** aus Wink Studio ein.
4. Klicken Sie auf **Speichern**.
</Steps>

## Verfügbare Theme-Blöcke

| Block | Zweck |
|---|---|
| **Wink Content** | Zeigt Hotelkarten, gerankte Raster, Karten und mehr an |
| **Wink Lookup** | Suchleiste für Reiseziele und Hotels |
| **Wink Search** | Icon-Button, der den Reiserouten-Auswahldialog öffnet |
| **Wink Account** | Anmelde- / Benutzerkonto-Button |
| **Wink Itinerary** | Reiserouten-Button, der die aktuelle Reise anzeigt |
| **Wink Cart** | Zusammenfassungs-Button für den Warenkorb |
