Die **Wink TYPO3 Extension** integriert Wink Reiseinventar als native Inhaltselemente in TYPO3 13. Redakteure fügen Wink-Komponenten über den Standard-TYPO3-Seiteneditor zu Seiten hinzu – keine Template-Änderungen erforderlich.

## Anforderungen

- TYPO3 13.x
- PHP 8.2+
- Composer

## Installation

<Steps>
1. Installation via Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Extension aktivieren:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Im TYPO3 Backend zu **Admin Tools → Einstellungen → Extension-Konfiguration → wink** gehen.
4. Ihre **Client ID**, **Client Secret** eingeben und die Umgebung auswählen. Speichern.
   Gehen Sie zu [Applications](/de/portal/settings/applications), um Ihre Zugangsdaten abzurufen.
5. Das statische TypoScript-Template zum Root-Template hinzufügen:
   **Template → Bearbeiten → Includes → Verfügbare Elemente → Wink Travel**.
</Steps>

## Inhaltselement zu einer Seite hinzufügen

<Steps>
1. Öffnen Sie eine Seite im **Seite**-Modul.
2. Klicken Sie auf die **+**-Schaltfläche im Inhaltsbereich, um ein neues Element hinzuzufügen.
3. Wählen Sie eines der **Wink Travel** Inhaltselemente:
   - **Wink Content** — Hotelkarten, gerankte Raster, Karten
   - **Wink Lookup** — Ziel-Suchleiste
   - **Wink Search** — Reiseplan-Auswahlknopf
   - **Wink Account** — Anmeldeknopf
   - **Wink Itinerary** — Reiseplan-Statusknopf
   - **Wink Cart** — Warenkorb-Knopf
4. Für **Wink Content** verwenden Sie die FlexForm-Felder, um einen **Layout-Typ** auszuwählen und die **Layout-ID** einzugeben.
5. Speichern und veröffentlichen Sie die Seite.
</Steps>

## Verfügbare Inhaltselemente

| Element | Zweck |
|---|---|
| **Wink Content** | Hotelkarten, gerankte Raster, Karten und mehr |
| **Wink Lookup** | Ziel-Suchleiste |
| **Wink Search** | Icon-Knopf, der den Reiseplan-Auswähler öffnet |
| **Wink Account** | Anmelde- / Benutzerkonto-Knopf |
| **Wink Itinerary** | Reiseplan-Knopf, der die aktuelle Reise anzeigt |
| **Wink Cart** | Warenkorb-Zusammenfassungs-Knopf |
