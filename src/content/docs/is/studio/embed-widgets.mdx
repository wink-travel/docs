Studio býður upp á nokkur innbyggð viðmót sem þú getur bætt við á eigin vefsíðu til að auka bókanir.

Til að nálgast innbyggð viðmót skaltu fara í `Tools > Miscellaneous` í hliðarstikunni.

## Tiltæk viðmót

| Viðmót | Lýsing |
| -- | -- |
| **Account button** | Vörumerktur hnappur sem tengir á tengilsíðu þína |
| **Itinerary button** | Hnappur sem opnar ferðalýsivalkostinn |
| **Itinerary form** | Innbyggt eyðublað fyrir dagsetningar og gestaval |
| **Search button** | Hnappur sem opnar leit að gistingu |
| **Lookup button** | Hnappur til að leita að tilteknu birgðahaldi |

## Hvernig á að innbyggja

Hvert viðmót gefur þér kóðabút sem þú getur afritað og límt inn í HTML kóða vefsíðunnar þinnar. Kóðabúturinn inniheldur **Client ID** og **Customization ID** svo viðmótið tengist sjálfkrafa tengilsaðila reikningnum þínum.

:::note
Ef þú notar WordPress bjóðum við einnig upp á WordPress viðbót til að auðvelda samþættingu.
:::
