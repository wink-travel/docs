## Yfirlit
Hýsti Wink MCP þjónninn er aðgengilegur á:

```
https://docs.mcp.wink.travel/mcp
```

Bendið samhæfum AI viðskiptavinum (VS Code, Cursor, aðrir sem innleiða MCP HTTP flutning) á þennan enda til að leyfa þeim að:
- Skoða vörur & pallskjöl (`wink://docs/...`)
- Skoða lifandi OpenAPI forskriftir og aðgerðarskjöl (`wink://api/...`)
- Framkvæma létta merkingar- / lykilorðaleit yfir skjöl og lifandi API

## Flutningur
Notið aðal straumflutningshæfan HTTP enda:

```
https://docs.mcp.wink.travel/mcp
```

Ef viðskiptavinur styður aðeins SSE, notið:

```
https://docs.mcp.wink.travel/sse
```

### Claude Code
Opinber skjöl: [Tengdu Claude Code við verkfæri í gegnum MCP](https://code.claude.com/docs/en/mcp)

Anthropic mælir með fjarflutningi HTTP fyrir skýhýsta MCP þjóninn:
```bash
claude mcp add --transport http wink-docs https://docs.mcp.wink.travel/mcp
claude mcp list
```

Innan Claude Code, keyrðu `/mcp` til að skoða stöðu þjónsins. Ef þú vilt deila þjóninum með öllu verkefninu, bættu við `--scope project` svo Claude Code skrifi stillinguna í `.mcp.json`.

### Codex
Opinber skjöl: [Docs MCP fljótstart fyrir Codex](https://developers.openai.com/resources/docs-mcp)

OpenAI lýsir MCP uppsetningu í gegnum Codex CLI og bendir á að stillingarnar séu deilt með IDE viðbótinni:
```bash
codex mcp add wink-docs --url https://docs.mcp.wink.travel/mcp
codex mcp list
```

### Cursor
Cursor MCP skjöl: https://cursor.com/docs/context/mcp
Uppsetning: Bættu við fjarþjóni í gegnum MCP stillingar.
```json
{
  "mcpServers": {
    "wink-docs": {
      "url": "https://docs.mcp.wink.travel/mcp",
    }
  }
}
```

### VS Code
Opinber skjöl: https://code.visualstudio.com/docs/copilot/customization/mcp-servers
Lágmarks fjarstilling (notandi eða vinnusvæði `mcp.json`):
```json
{
  "servers": {
    "wink-docs": {
      "type": "http",
      "url": "https://docs.mcp.wink.travel/mcp"
    }
  }
}
```

Ábendingar:
- Notaðu Command Palette: „MCP: List Servers“ til að staðfesta tengingu & skoða verkfæri.
- Verkfæraval leyfir þér að virkja/óvirkja `api_search` og `docs_search`.
- Bættu við samhengi: Spjallgluggi → Bæta við samhengi → MCP auðlindir til að tengja skjalsíður.

## Kjarnauppsprettu URI

| URI | Lýsing |
|-----|-------------|
| `wink://docs` | Skjalaskrá (flokkað eftir flokkum) |
| `wink://docs/{path}` | Sérstakur skjalsgrein (Markdown / MDX) |
| `wink://api` | Lifandi OpenAPI skrá sem uppgötvuð er úr Wink swagger stillingum |
| `wink://api/{apiName}` | Þétt API yfirlit með auðkenningu, merkjum, hausum og dæmaaðgerðum |
| `wink://api/{apiName}/operations` | Fullt aðgerðarskrá með flokkun eftir merkjum |
| `wink://api/{apiName}/operation/{key}` | Upplýsingar um einstaka aðgerð (breytur, beiðnibolur, svör) |
| `wink://api/{apiName}/schema/{schemaName}` | Útfært hlutaform með innfelldum reitum, gerðum og dæmum |
| `wink://api/{apiName}/operation/{key}/request` | Skoðun á beiðni með nauðsynlegum inntökum og lágmarks dæmi um gagnapakka |
| `wink://api/{apiName}/operation/{key}/response/{status}` | Skoðun á svari fyrir ákveðinn stöðukóða |
| `wink://api/{apiName}/operation/{key}/recipe` | Verkferill fyrir rétta notkun aðgerðarinnar |

## Leitartól (Fljótleg yfirlit)
| Tól | Tilgangur | Dæmi um algengar fyrirspurnir |
|------|---------|-----------------------|
| `api_search` | Finna lifandi API eða aðgerð til að samþætta | `create affiliate`, `payment receipt`, `lookup inventory`, `invite manager` |
| `docs_search` | Finna hugmyndafræðileg skjöl / leiðbeiningar | `authentication`, `webhooks`, `wordPress` |

Láttu `query` vera tómt til að fá tillögur. Hvert niðurstaða inniheldur MCP URI sem hægt er að opna.
Fyrir aðgerðarniðurstöður skilar `api_search` einnig tengdum beiðni, svari, formi og uppskrift URI svo umboðsmenn geti hoppað beint í gagnlegustu undiraðlindirnar.

## Venjulegt notkunarmynstur
1. Keyrðu `api_search` með þinni fyrirætlan (t.d. *„create affiliate“* eða *„payment receipt“*).
2. Opnaðu skilaða `wink://api/...` URI, eða fylgdu tengdum `request_uri`, `response_uris`, `request_schema_uris`, `response_schema_uris` og `recipe_uri` reitum.
3. Notaðu `docs_search` fyrir stuðningshugmyndir (auðkenning, webhooks, vinnuflæði).

## AI Hæfileikar

Fyrirfram smíðaðir hæfileikar umboðsmanna nota þennan MCP þjón sem sannleiksgjafa. Hæfileikar gefa kóðunarumboðsmanni þínum uppbyggðan vinnuflæði til að byggja á Wink API — í stað þess að giska á enda, les umboðsmaðurinn lifandi samninga beint frá MCP þjóninum.

Sjá [AI Skills](/is/developers/ai-skills) fyrir tiltæka hæfileika og uppsetningarleiðbeiningar.

## Athugasemdir
- Hvert skilað URI er hægt að opna af umboðsmanni til að fá fullan innihald.
- `wink://api` tréð er lifandi og fylgir núverandi OpenAPI skrá frá uppsprettu í stað búins skjalaskrár.
- Tillögur birtast sjálfkrafa þegar fyrirspurnir eru tómar.
