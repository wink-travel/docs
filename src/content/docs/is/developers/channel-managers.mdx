Wink samþættir við leiðandi channel manager-a, eignastýringarkerfi (PMS) og miðlægar bókunarkerfi (CRS) til að halda verðlagi og framboði samstilltu yfir alla söluvegi þína.

## Hvernig það virkar

Samþættingar við channel manager-a gera kleift tvíhliða gagnaflæði milli eignastýringarkerfisins þíns og Wink:

1. **Verðuppfærslur** — Channel manager-inn þinn sendir verðuppfærslur til Wink. Þegar þú uppfærir verðlagningu í PMS-kerfinu þínu, birtast þær breytingar sjálfkrafa á Wink.
2. **Framboðssamstilling** — Channel manager-inn þinn sendir framboðsuppfærslur til Wink. Þegar herbergi eru bókuð eða losuð í PMS-kerfinu þínu, uppfærir Wink birgðastöðu í rauntíma.
3. **Bókunartilkynningar** — Þegar bókun er gerð á Wink, fær channel manager-inn þinn tilkynningu svo bókunin birtist í PMS-kerfinu þínu.

## Stuðningsaðilar channel manager-a

Wink samþættir nú við eftirfarandi kerfi:

| Channel Manager | Lýsing |
|---|---|
| **Channex** | Alheims dreifingar- og channel management vettvangur. |
| **TravelClick** | Hótellausnir og channel management (Amadeus). |
| **Omnibees** | Dreifing og tekjustýring fyrir hótel. |
| **D-Edge** | Tæknilausnir fyrir gististaði og miðlægar bókanir. |
| **Allotz** | Birgðastýring og channel management. |
| **Ezee / Yanolja** | Skýjabundið PMS og channel management með áherslu á Asíumarkaðinn. |
| **RateGain** | Verðgreind og dreifingartækni. |
| **RateTiger** | Dreifing verðlags og framboðs í rauntíma. |
| **CloudBeds** | Skýjabundið PMS, channel manager og bókunarvél. |
| **SiteMinder** | Leiðandi channel manager og dreifingarvettvangur fyrir hótel. |
| **Exely** | Samstilling verðlags og framboðs fyrir hótel. |

:::note
Ef channel manager-inn þinn er ekki á listanum, hafðu samband við Wink teymið. Nýjar samþættingar geta verið bættar við byggðar á OTA (Open Travel Alliance) samskiptareglum sem Wink styður.
:::

## Fyrir forritara

Ef þú ert að byggja samþættingu fyrir channel manager eða PMS við Wink, notaðu Channel Manager API endapunkta:

- **Verðuppfærslur** — Ýttu verðbreytingum til Wink.
- **Framboðsuppfærslur** — Ýttu framboðsbreytingum til Wink.
- **Bókunartöku** — Sækja bókanir frá Wink inn í kerfið þitt.

Channel Manager API fylgir OTA samskiptareglum (SOAP/XML) til að tryggja samhæfni við núverandi gistikerfi.

## Webhook tilkynningar

Þú getur gerst áskrifandi að webhook atburðum frá channel manager til að fá tilkynningar í rauntíma:

- `channel-manager.update.rate` — Verðuppfærsla móttekin.
- `channel-manager.update.availability` — Framboðsuppfærsla móttekin.
- `channel-manager.update` — Almenn uppfærsla frá channel manager.

Sjáðu [Webhook Events Catalog](/is/developers/webhook-events#channel-manager-events) fyrir nánari upplýsingar.
