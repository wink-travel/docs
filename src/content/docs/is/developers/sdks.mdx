Flýttu fyrir markaðssetningu með því að nota eitt af opinberlega studdu hugbúnaðarþróunarpökkunum okkar.
Ef við styðjum ekki uppáhaldstungumálið þitt núna skaltu fara í forritaskilið sem þú vilt nota og sækja OpenAPI forskriftina sem við gerum aðgengilega fyrir hvert forritaskil efst á síðunni. Þú getur búið til þitt eigið SDK með því að annað hvort nota eitt af opinberu SDK geymslunum okkar eða með því að nota einn af OpenAPI rafallunum.

## Java

Þetta bókasafn var þýtt fyrir Java 24. Það er stillt til að nota reactive WebClient og er fínstillt fyrir notkun með Spring Framework. Spring er þó ekki skilyrði en það veitir þér OAuth2 stuðning strax úr kassanum.
