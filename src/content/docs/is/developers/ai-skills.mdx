AI hæfileikar eru flytjanlegar leiðbeiningapakkar sem kenna kóðunaraðstoðarmönnum hvernig á að vinna með Wink. Í stað þess að treysta á þjálfunargögn eða úrelt skjöl, leiðbeinir hver hæfileiki aðstoðarmanninum að nota [Wink MCP þjóninn](/is/developers/mcp) sem lifandi sannleiksgjafa fyrir API samninga og pallskjöl.

Hæfileikar virka með hvaða aðstoðarmanni sem styður þá — Claude Code, Codex, Cursor og aðrir.

## Tiltækir hæfileikar

### wink-api-builder

Innleiða, yfirfara eða laga villur í Wink API samþættingum. Hæfileikinn leiðir aðstoðarmanninn í gegnum uppbyggðan vinnuflæði:

1. Leita að réttri API aðgerð með `api_search`
2. Opna lifandi beiðni/svör samninga frá MCP þjóninum
3. Sækja auðkenningaruppsetningu úr skjölunum
4. Skrifa kóða í samræmi við staðfestan samning
5. Merkja öll ósamræmi milli núverandi kóða og núverandi API

Gagnlegt fyrir:
- Að byggja nýjar viðskiptavinaraðferðir fyrir hvaða Wink API sem er
- Að laga ósamræmi í samningum (t.d. beiðni sem skilar HTTP 400)
- Að yfirfara núverandi samþættingar miðað við lifandi forskrift

## Uppsetning

Afritaðu hæfileikamöppuna í hæfileikamöppu aðstoðarmannsins þíns.

:::tip
Wink MCP þjónninn þarf að vera tengdur í sama umhverfi. Sjá [MCP Server](/is/developers/mcp) fyrir uppsetningarleiðbeiningar.
:::

**Claude Code**

```bash
# repo-local
cp -r skills/wink-api-builder .claude/skills/

# eða user-level
cp -r skills/wink-api-builder ~/.claude/skills/
```

Kallaðu á með:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repo-local
cp -r skills/wink-api-builder .agents/skills/

# eða user-level
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Kallaðu á með:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Aðrir aðstoðarmenn**

Settu hæfileikamöppuna hvar sem aðstoðarmaðurinn þinn sækir hæfileika. Aðalinnsláttur er `SKILL.md`.
