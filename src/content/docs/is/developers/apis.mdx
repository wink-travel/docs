## Greiðslur API

Greiðsluendapunktar eru fyrir forritara sem vilja kaupa ferðavöru. Þetta er hægt að gera í gegnum API sem skráður ferðaskrifstofumaður eða með því að nota API-ið okkar ásamt PCI-samhæfu greiðslutæki fyrir aðrar einingar.
Greiðslur API býður upp á endapunkta fyrir tengda aðila og hótel til að fylgjast með bókunum, greiningum og framboði fjármuna. Þeir geta einnig valið að taka út aðgengilega peninga á bankareikning sinn.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Lærðu meira
</LinkButton>
