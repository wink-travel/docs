## Payment API

Payment endapunktar eru fyrir forritara sem vilja kaupa ferðavörur. Þetta er hægt að gera í gegnum API sem skráður ferðaskipuleggjandi eða með því að nota API okkar ásamt PCI samhæfu greiðslutæki fyrir allar aðrar einingar.
Payment API opnar endapunkta fyrir tengda aðila og hótel til að fylgjast með bókunum, greiningum og framboði fjármuna. Þeir geta einnig valið að taka út aðgengilega fjármuni á bankareikning sinn.

Lærðu meira
</LinkButton>
