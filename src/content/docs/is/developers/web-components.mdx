[Vefhlutir](https://developer.mozilla.org/en-US/docs/Web/API/Web_components) eru staðall sem leyfir þér að samþætta heilar JavaScript-eiginleika inn á vefsíðuna þína án þess að þurfa að kunna að forrita.
Þökk sé þessari flottu tækni geturðu fellt inn Wink ferðavöruúrval með mjög litlu fyrirhöfn. Þessi grein leiðir þig í gegnum safn okkar af vefhlutum og sýnir þér hvernig á að nota þá.

## Fella inn Wink

Það eru þrjú atriði sem þú þarft að hafa með á hvaða síðu sem er þar sem þú vilt sýna einn af vefhlutum okkar.

1. Bættu við stílsniði okkar.
2. Bættu við JavaScript-inu okkar.
3. Bættu við forritshleðslunni okkar.

### Stílsnið

Bættu við CSS-stílum okkar í haus skjalsins þíns.

```html {3}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
</html>
```

### Javascript

Bættu við JavaScript-inu okkar neðst í skjalið þitt. (*Við mælum með að setja það beint fyrir ofan lok body-tagginn*).

```html {3}
<html>
  <body>
    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
  </body>
</html>
```

Síðan þín er nú tilbúin og getur sýnt vefhluti okkar.

## Hlutar

Lestu um safn okkar af hlutum hér að neðan.

### Forritshleðsla

Hleðsluvélin sér um að halda utan um stöðu og stjórna samskiptum milli hluta okkar. (*Settu hana inn undir JavaScript-inu okkar*).

```html {3-6}
<html>
  <body>
    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

### Innihaldshleðsla

Innihaldshleðslan er kjarninn í safni vefhluta okkar. Hún sér um að sýna vöruúrval þitt (kort, grindur, kort).

Fáanlegir eiginleikar:

- layout
- id
- sort

*Eiginleikinn `sort` er aðeins í boði þegar layout er `RANKED` og þú vilt ekki nota núverandi raðaða grind. Í þessum tilfellum skaltu skilja `id` tómt.*

Fáanlegir gerðir layout:

- `AD_BANNER`
- `MAP`
- `HOTEL`
- `GUEST_ROOM`
- `MEETING_ROOM`
- `SPA`
- `RESTAURANT`
- `ACTIVITY`
- `ATTRACTION`
- `PLACE`
- `ADD_ON`
- `LIST`
- `SEARCH`
- `RANKED`

Fáanlegar gerðir sort:

- `MEMBER`
- `PRICE_LOW_TO_HIGH`
- `PRICE_HIGH_TO_LOW`
- `PRICE`
- `POPULARITY`
- `ECO`
- `EXPERIENCE`
- `PERK`
- `LOYALTY`
- `PACKAGE`

```html {3-6}
<html>
  <body>
    <wink-content-loader
       layout="GUEST_ROOM"
       id="2de7ee9c-61c9-11ef-9722-42004e494300"
    ></wink-content-loader>
  </body>
</html>
```

Dæmi má finna fyrir [kort](/studio/cards), [grindur](/studio/grids) og [kort](/studio/maps).

    <wink-lookup></wink-lookup>
  </body>

## Hlutar

### Greiðsla

Greiðsluhluturinn leyfir þér að tilkynna TripPay að ferðalangur vilji kaupa eitthvað og reglur og verð fyrir þá hluti.

Það er einn nauðsynlegur eiginleiki fyrir viðmótið:

- `id` Auðkenni bókunarsamningsins sem þú vilt framkvæma.

```html {3}
<html>
  <body>
    <trip-pay id="<INSERT_BOOKING_CONTRACT_ID>"></trip-pay>
  </body>
</html>
```

:::note
Til að læra hvernig á að búa til bókunarsamnings-`id`, lestu leiðbeininguna: [Integrating with TripPay](/guides/integrating-with-trip-pay).
:::

Viðmótið undirbýr samninginn til framkvæmdar og sýnir greiðsluupplýsingar *(Mynd 1)* fyrir notandann til að ljúka bókuninni.

## Villur

Ef þú færð villuskilaboð þegar þú ert að fella inn einn af vefhlutum okkar, hvort sem það er á WinkLinks eða á eigin vefsíðu, gætu nokkur atriði hafa farið úrskeiðis:

### Framboð

Vöruúrval gæti verið ekki lengur í boði. Farðu á [Wink Studio](/studio/cards) og athugaðu hvort staða „kortsins“ sé í boði.
Ef það er ekki, mun það birtast sem rautt. Í því tilfelli geturðu beðið og kannski gerir birgirinn það aftur aðgengilegt eða fjarlægt það af listanum þínum.

:::note
`Framboð` þýðir ekki daglegt framboð vöru til bókunar. Það þýðir að varan er alveg óaðgengileg fyrir þig og/eða aðra.
:::

### Fjarlægt

Vöruúrval gæti hafa verið fjarlægt. Farðu á [Wink Studio](/studio/cards) og athugaðu hvort staða „kortsins“ sé í boði.
Ef það er ekki, mun það birtast sem rautt. Í því tilfelli geturðu beðið og kannski gerir birgirinn það aftur aðgengilegt eða fjarlægt það af listanum þínum.

### Sérsnið

Þú gætir óvart fjarlægt sérsnið sem þú hefur skilgreint til að fylgja vöruúrvalinu þínu. Gakktu úr skugga um að sérsniðið sé enn í boði og stilltu nýtt ef eitthvað vantar.

### Forrit

Þetta er aðeins fyrir vefþróunaraðila. Ef þú hefur óvart fjarlægt [Forritið](/developer/applications); þannig að client ID er ekki lengur í boði, vinsamlegast búðu til nýtt forrit og notaðu nýja client ID til að fella inn vefhluti okkar.
