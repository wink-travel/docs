[Vefþættir](https://developer.mozilla.org/en-US/docs/Web/API/Web_components)er staðall sem gerir þér kleift að samþætta alla Javascript eiginleika í vefsíðuna þína án þess að kunna að forrita.
Þökk sé þessari flottu tækni geturðu fellt inn ferðabirgðir frá Wink með mjög litlum fyrirhöfn. Þessi grein leiðir þig í gegnum safn okkar af vefíhlutum og sýnir þér hvernig á að nota þá.

## Fella inn Wink

Það eru þrír hlutir sem þú verður að hafa með á hverri síðu þar sem þú vilt birta einn af vefþáttum okkar.

1. Hafðu stílblaðið okkar með.
2. Innifalið Javascript okkar.
3. Hafðu með forritahleðslutækið okkar.

### Stílblað

Settu CSS stílana okkar inn í hausinn á skjalinu þínu.

```html {3}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
</html>
```

### Javascript

Settu Javascript-kóðann okkar neðst í skjalið þitt. (*Við mælum með að þú setjir strax fyrir ofan lokamerkinguna*).

```html {3}
<html>
  <body>
    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
  </body>
</html>
```

Síðan þín er nú tilbúin og getur birt vefþætti okkar.

## Íhlutir

Lestu um íhlutasafnið okkar hér að neðan.

### Forritshleðslutæki

Hleðslutækið ber ábyrgð á að viðhalda stöðu og stjórna samskiptum milli íhluta okkar. (*Settu það inn fyrir neðan Javascript-kóðann okkar*).

```html {3-6}
<html>
  <body>
    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

### Innihaldshleðslutæki

Innihaldshleðslutækið er kjarninn í vefþáttasafni okkar. Það sér um að birta birgðir þínar (kort, töflur, kort).

Tiltækir eiginleikar:

* skipulag
* auðkenni
* flokka

*Eiginleikinn`sort`er aðeins í boði þegar útlitið er`RANKED`og þú vilt ekki nota núverandi raðaða rist. Í þessum tilfellum skaltu skilja eftir`id`tómt.*

Tiltækar gerðir af útliti:

* `AD_BANNER`
* `MAP`
* `HOTEL`
* `GUEST_ROOM`
* `MEETING_ROOM`
* `SPA`
* `RESTAURANT`
* `ACTIVITY`
* `ATTRACTION`
* `PLACE`
* `ADD_ON`
* `LIST`
* `SEARCH`
* `RANKED`

Tiltækar flokkunartegundir:

* `MEMBER`
* `PRICE_LOW_TO_HIGH`
* `PRICE_HIGH_TO_LOW`
* `PRICE`
* `POPULARITY`
* `ECO`
* `EXPERIENCE`
* `PERK`
* `LOYALTY`
* `PACKAGE`

```html {3-6}
<html>
  <body>
    <wink-content-loader
       layout="GUEST_ROOM"
       id="2de7ee9c-61c9-11ef-9722-42004e494300"
    ></wink-content-loader>
  </body>
</html>
```

Hægt er að finna sýnishorn fyrir[spil](/studio/cards),[grindur](/studio/grids)og[kort](/studio/maps).

## Innfella TripPay

Það eru tveir hlutir sem þú þarft að hafa með á hverri síðu þar sem þú vilt nota vefþáttinn fyrir TripPay greiðslu.

1. Hafðu stílblaðið okkar með.
2. Innifalið Javascript okkar.

### Stílblað

Settu CSS stílana okkar inn í hausinn á skjalinu þínu.

```html {3}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.trippay.io/styles.css"></link>
  </head>
</html>
```

### Javascript

Settu Javascript-kóðann okkar neðst í skjalið þitt. (*Við mælum með að þú setjir strax fyrir ofan lokamerkinguna*).

```html {3}
<html>
  <body>
    <script src="https://elements.trippay.io/elements.js" type="module" defer></script>
  </body>
</html>
```

Síðan þín er nú tilbúin og getur birt vefþætti okkar.

## Íhlutir

### Greiðsla

Greiðsluþátturinn gerir þér kleift að láta TripPay vita að ferðalangur vilji kaupa eitthvað og reglurnar og verðið fyrir þessar vörur.

Það er einn nauðsynlegur eiginleiki fyrir búnaðinn:

* `id`Auðkenni bókunarsamningsins sem þú vilt að verði undirritaður.

```html {3}
<html>
  <body>
    <trip-pay id="<INSERT_BOOKING_CONTRACT_ID>"></trip-pay>
  </body>
</html>
```

:::athugið
Til að læra hvernig á að búa til bókunarsamninginn`id`, lestu leiðbeiningarnar:[Samþætting við TripPay](/guides/integrating-with-trip-pay).&#x20;
:::
