Þessi síða listar alla webhook-viðburði sem þú getur gerst áskrifandi að á Wink. Fyrir uppsetningarleiðbeiningar, sjá [Webhook Integration Guide](/is/guides/webhook-integration) eða [Webhooks management page](/is/my-account/webhooks).

## Hvernig á að gerast áskrifandi

1. Farðu í **Applications > Webhooks** í þínu aðgangi.
2. Búðu til webhook og veldu þá viðburði sem þú vilt fá.
3. Wink sendir HTTP POST á þinn endaþjón í hvert skipti sem viðburðurinn á sér stað.

Fyrir fullar uppsetningarleiðbeiningar og bestu vinnubrögð, sjá [Webhook Integration](/is/guides/webhook-integration).

## Viðburðir tengdir reikningi

Viðburðir tengdir lífsferli og stillingum TripPay reikningsins þíns.

| Viðburður | Lykill | Lýsing |
|---|---|---|
| Reikningur stofnaður | `account.create` | Nýr reikningur hefur verið stofnaður. |
| Reikningur í skoðun | `account.under_review` | Reikningur er í bið eftir skoðun. |
| Reikningur uppfærður | `account.updated` | Upplýsingar um reikning hafa breyst. |
| Staða reiknings uppfærð | `account.status.updated` | Staða reiknings hefur breyst (t.d. samþykktur, stöðvaður). |
| Heimilisfang reiknings uppfært | `account.address.updated` | Heimilisfang reiknings hefur verið breytt. |
| Upplýsingar reiknings uppfærðar | `account.details.updated` | Upplýsingareiningar reiknings hafa breyst. |
| Almennar upplýsingar reiknings uppfærðar | `account.information.update` | Almennar upplýsingar um reikning hafa verið uppfærðar. |
| Reikningsstjórar uppfærðir | `account.managers.update` | Listi yfir reikningsstjóra hefur breyst. |
| Próffílmynd reiknings uppfærð | `account.profile_picture.updated` | Próffílmynd reiknings hefur verið breytt. |
| Umboðsmaður reiknings uppfærður | `account.agent.updated` | Upplýsingar um umboðsmann reiknings hafa verið breyttar. |
| Vefslóðarnafn reiknings uppfært | `account.url-name.update` | Sérsniðið vefslóðarnafn reiknings hefur breyst. |
| Einstakt auðkenni reiknings uppfært | `account.unique-id.update` | Einstakt auðkenni reiknings hefur verið breytt. |

### Verkefni reiknings

| Viðburður | Lykill | Lýsing |
|---|---|---|
| Verkefni stofnað | `account.task.create` | Nýtt verkefni hefur verið bætt við reikninginn. |
| Verkefni uppfært | `account.task.update` | Verkefni reiknings hefur verið breytt. |
| Verkefni fjarlægt | `account.task.remove` | Verkefni reiknings hefur verið fjarlægt. |
| Verkefni lokið | `account.task.complete` | Verkefni reiknings hefur verið lokið. |
| Skattnúmeraverkefni lokið | `account.task.tax_id.complete` | Verkefni um staðfestingu skattnúmera hefur verið lokið. |

### Reikningskaupandi

| Viðburður | Lykill | Lýsing |
|---|---|---|
| Kaupandi stofnaður | `account.acquirer.created` | Nýr kaupandi hefur verið bætt við reikninginn. |
| Kaupandi uppfærður | `account.acquirer.updated` | Reikningskaupandi hefur verið breyttur. |
| Kaupandi fjarlægður | `account.acquirer.removed` | Kaupandi hefur verið fjarlægður af reikningnum. |

## Viðburðir tengdir greiðanda

Viðburðir tengdir stjórnun greiðanda (þ.e. móttakanda greiðslu).

| Viðburður | Lykill | Lýsing |
|---|---|---|
| Greiðandi stofnaður | `payee.create` | Nýr greiðandi hefur verið stofnaður með góðum árangri. |
| Mistókst að stofna greiðanda | `payee.create.failure` | Mistókst að stofna greiðanda. |
| Greiðandi uppfærður | `payee.update` | Greiðandi hefur verið uppfærður. |
| Mistókst að uppfæra greiðanda | `payee.update.failure` | Mistókst að uppfæra greiðanda. |

## Viðburðir bókunarsamninga

Viðburðir sem fylgjast með lífsferli bókunarsamnings í gegnum TripPay.

| Viðburður | Lykill | Lýsing |
|---|---|---|
| Beiðni um upphaf bókunarsamnings | `booking.contract.initialize.request` | Beiðni um að hefja bókunarsamning hefur verið send. |
| Upphaf bókunarsamnings lokið | `booking.contract.initialize.complete` | Upphaf bókunarsamnings hefur lokið. |
| Samningur stofnaður | `booking.contract.created` | Nýr bókunarsamningur hefur verið stofnaður. |
| Samningur í vinnslu | `booking.contract.processing` | Greiðsla bókunarsamnings er í vinnslu. |
| Samningur mistókst | `booking.contract.failed` | Greiðsla bókunarsamnings mistókst. |
| Samningur felldur niður | `booking.contract.cancelled` | Bókunarsamningur hefur verið felldur niður. |
| Hlutaendurgreiðsla | `booking.contract.refund.partial` | Hlutaendurgreiðsla hefur verið framkvæmd á samningnum. |
| Full endurgreiðsla | `booking.contract.refund.full` | Full endurgreiðsla hefur verið framkvæmd á samningnum. |
| Beiðni um endurgreiðslu | `booking.contract.refund.request` | Beiðni um endurgreiðslu hefur verið send. |
| Villa við endurgreiðslu | `booking.contract.refund.error` | Villa kom upp við vinnslu endurgreiðslu. |
| Úttekt stofnuð | `booking.contract.payout.create` | Úttekt hefur verið stofnuð fyrir samninginn. |
| Úttekt lokið | `booking.contract.payout.complete` | Úttekt hefur verið lokið með góðum árangri. |

## Viðburðir stjórnunaraðila

Viðburðir fyrir stjórnunaraðila (eignir, tengiliði, fyrirtæki) og stjórnun þeirra.

### Áskrift

| Viðburður | Lykill | Lýsing |
|---|---|---|
| Áskrift stofnuð | `managing_entity.subscription.provisioned` | Ný áskrift hefur verið stofnuð. |
| Áskrift breytt | `managing_entity.subscription.changed` | Áskriftaráætlun hefur verið breytt. |
| Reikningur áskriftar breyttur | `managing_entity.subscription.invoice.changed` | Reikningur áskriftar hefur verið uppfærður. |

### Staða og stjórnun

| Viðburður | Lykill | Lýsing |
|---|---|---|
| Staða uppfærð | `managing_entity.status.updated` | Staða aðila hefur breyst. |
| Aðili eytt | `managing_entity.deleted` | Stjórnunaraðili hefur verið eytt. |
| Umboðsmaður uppfærður | `managing_entity.agent.updated` | Umboðsmaður aðila hefur verið breyttur. |
| Tölvupósttilkynning til umboðsmanns | `managing_entity.agent.email.notification` | Tölvupósttilkynning hefur verið send til umboðsmanns. |
| Tölvupósttilkynning til stjórnanda | `managing_entity.manager.email.notification` | Tölvupósttilkynning hefur verið send til stjórnanda. |

### Lífsferill stjórnanda

| Viðburður | Lykill | Lýsing |
|---|---|---|
| Stjórnandi bætt við | `managing_entity.manager.added` | Nýr stjórnandi hefur verið bætt við. |
| Stjórnandi boðinn | `managing_entity.manager.invited` | Boð til stjórnanda hefur verið sent. |
| Stjórnandi fjarlægður | `managing_entity.manager.removed` | Stjórnandi hefur verið fjarlægður. |
| Stjórnandi samþykktur | `managing_entity.manager.accepted` | Boð til stjórnanda hefur verið samþykkt. |
| Stjórnandi hafnað | `managing_entity.manager.rejected` | Boð til stjórnanda hefur verið hafnað. |
| Eigandi skilgreindur | `managing_entity.manager.owner.set` | Eignarhald aðila hefur verið flutt. |

| Merki uppfært | `managing_entity.logo.updated` | Merki aðila hefur verið uppfært. |
| Vefslóðarnafn uppfært | `managing_entity.url_name.updated` | Sérsniðið vefslóðarnafn aðila hefur breyst. |
