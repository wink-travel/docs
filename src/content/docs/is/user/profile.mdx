import { Steps } from '@astrojs/starlight/components';

Ef þú vilt uppfæra notendaprófílinn þinn fyrir Wink notandareikninginn þinn, fylgdu þessum skrefum:

<Steps>
1. Þú verður að vera skráður inn á einni af vefsíðum okkar.
2. Smelltu á prófílmyndina þína, efst í hægra horninu og valmynd birtist.
3. Smelltu á `Profile` hlekkinn.
4. Fylgdu leiðbeiningunum til að uppfæra `netfangið`, `fornafn` og `eftirnafn`.
5. Þú getur einnig hlaðið upp nýrri `prófílmynd` af þér.
6. Smelltu á `Save`.
</Steps>

:::note
Nýja prófílmyndin þín mun birtast næst þegar þú skráir þig alveg út af Wink.
:::
