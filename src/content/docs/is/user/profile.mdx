Ef þú vilt uppfæra notandasniðið þitt fyrir Wink notandareikninginn þinn skaltu fylgja þessum skrefum:

<Steps>
  1. Þú verður að vera skráð(ur) inn á eina af vefsíðum okkar.
  2. Smelltu á prófíltáknið þitt, efst í hægra horninu, og þá birtist valmynd.
  3. Smelltu á `Profile` tengill.
  4. Fylgdu leiðbeiningunum til að uppfæra `e-mail`, `first` og `last name`.
  5. Þú getur líka hlaðið inn nýju `profile picture` af sjálfum þér.
  6. Smelltu `Save`.
</Steps>


:::athugið
Nýja prófílmyndin þín birtist næst þegar þú skráir þig alveg út úr Wink.
:::
