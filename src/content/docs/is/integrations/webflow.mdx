Webflow styður sérsniðinn HTML og JavaScript á þremur stöðum: í `<head>` síðunnar, í `</body>` síðunnar og í kóðainnbyggingum fyrir einstaka þætti. Þetta gerir samþættingu Wink vefþátta einfalda.

:::note
Sérsniðinn kóði er í boði á **Basic áætlun Webflow og uppúr**. Innspýting í `</body>` síðunnar krefst greiddrar hýsingaráætlunar.
:::

## Hlaða Wink skriptum fyrir alla síðuna

<Steps>
1. Opnaðu **Project Settings → Custom Code** í Webflow Designer.
2. Í **Head Code** hlutanum, bættu við stílsniði:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. Í **Footer Code** hlutanum, bættu við skriptinu og forritshleðslunni (skiptu út `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Smelltu á **Save Changes**.
</Steps>

:::tip
Client ID þitt er að finna í Wink reikningnum þínum undir [Applications](/is/portal/settings/applications).
:::

## Bæta við þætti í ákveðinn þátt

<Steps>
1. Veldu **Div** eða **Section** í Designer þar sem þú vilt hafa þáttinn.
2. Í hægri spjaldi, smelltu á **`</>` Embed** táknið (eða bættu við **HTML Embed** þætti úr Add spjaldinu).
3. Límdu inn HTML þáttinn, til dæmis:
   ```html
   <wink-content-loader
     layout="RANKED"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Smelltu á **Save & Close**.
</Steps>

## Sérsniðinn kóði fyrir hverja síðu

Ef þú þarft aðeins Wink á ákveðnum síðum, notaðu kóðainnspýtingu fyrir hverja síðu í staðinn fyrir alla síðuna:

<Steps>
1. Opnaðu **Page Settings** fyrir viðkomandi síðu (gír táknið í Pages spjaldinu).
2. Skrunaðu niður í **Custom Code → Before `</body>` tag**.
3. Límdu inn sama skript og forritshleðslu og áður.
4. Vistaðu síðustillingarnar.
</Steps>

## Tiltækir þættir

| Þáttur | HTML tag | Tilgangur |
|---|---|---|
| Content loader | `<wink-content-loader>` | Sýna hótelkort, grindur, kort |
| Lookup | `<wink-lookup>` | Leitargluggi fyrir áfangastaði |
| Search button | `<wink-search-button>` | Opna ferðalagsval |
| Account button | `<wink-account-button>` | Skrá inn / notendavalmynd |
| Itinerary button | `<wink-itinerary-button>` | Sýna núverandi ferðalýsingu |
| Itinerary picker | `<wink-itinerary-picker>` | Fullt ferðalýsingareyðublað |
| Shopping cart | `<wink-shopping-cart-button>` | Yfirlitshnappur körfu |

Sjá [Web Components](/is/integrations/web-components) fyrir fulla eiginleikalýsingu.
