Innbyggða **Code Injection** eiginleikinn í Ghost gerir þér kleift að bæta við skriftum í alþjóðlega `<head>` og `</body>` útgáfunnar þinnar. Allar Ghost áætlanir styðja code injection, sem gerir það auðvelt að innbyggja Wink íhluti á hvaða síðu eða færslu sem er.

## Hlaða Wink skriftum á öllum síðum

<Steps>
1. Farðu í Ghost Admin spjaldið, veldu **Settings → Code Injection**.
2. Í reitnum **Site Header**, bættu við stílsniði:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. Í reitnum **Site Footer**, bættu við skriftu og forritshleðslu (skiptu út `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Smelltu á **Save**.
</Steps>

:::tip
Þitt Client ID er að finna í Wink reikningnum þínum undir [Applications](/is/portal/settings/applications).
:::

## Bæta íhluti við færslu eða síðu

Ritill Ghost styður **HTML kort** sem gerir þér kleift að setja inn hráa HTML hvar sem er í efni þínu.

<Steps>
1. Opnaðu færsluna eða síðuna í Ghost ritlinum.
2. Sláðu inn `/` til að opna kortaval og veldu **HTML**.
3. Límdu inn HTML íhlutans, til dæmis:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Smelltu utan við kortið til að forskoða, síðan birtaðu.
</Steps>

## Code injection fyrir hverja síðu

Fyrir íhlut sem á aðeins að birtast á einni síðu styður Ghost einnig code injection fyrir hverja síðu:

<Steps>
1. Opnaðu stillingar færslu eða síðu (gír táknið í ritlinum).
2. Skrunaðu niður að **Code Injection**.
3. Bættu við íhlutartákni í reitinn **Footer** (alþjóðlegur app-loader þarf aðeins að vera í síðufótinum).
4. Vistaðu.
</Steps>

## Tiltækir íhlutir

| Íhlutur | HTML tag | Tilgangur |
|---|---|---|
| Content loader | `<wink-content-loader>` | Sýna hótelkort, grindur, kort |
| Lookup | `<wink-lookup>` | Leitargluggi fyrir áfangastaði |
| Search button | `<wink-search-button>` | Opna ferðalagsval |
| Account button | `<wink-account-button>` | Skrá inn / notendavalmynd |
| Itinerary button | `<wink-itinerary-button>` | Sýna núverandi ferðalagsáætlun |
| Itinerary picker | `<wink-itinerary-picker>` | Fullt ferðalagsform |
| Shopping cart | `<wink-shopping-cart-button>` | Yfirlitshnappur körfu |

Sjá [Web Components](/is/integrations/web-components) fyrir fulla eiginleikalýsingu.
