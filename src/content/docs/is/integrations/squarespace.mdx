:::note
Code Injection eiginleikinn í Squarespace er í boði á **Business áætluninni og uppúr**. Sérsniðnir HTML kubbar eru í boði á öllum greiddu áætlunum.
:::

Squarespace styður sérsniðna HTML og JavaScript, sem þýðir að þú getur innbyggt hvaða Wink vefhluta sem er beint inn á síðuna þína án viðbótar. Þessi leiðarvísir sýnir þér hvernig á að hlaða Wink skriptum á alla síðuna og síðan setja hluta á einstakar síður.

## Hlaða Wink skriptum á alla síðuna

Að bæta skriptunum við alhliða fótar- og haus svæði síðunnar þýðir að Wink er tiltækt á hverri síðu án þess að þurfa að endurtaka uppsetninguna.

<Steps>
1. Í Squarespace ritlinum, farðu í **Settings → Advanced → Code Injection**.
2. Í **Header** reitnum, bættu við stílsniði:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. Í **Footer** reitnum, bættu við skriptinu og forritshleðslunni (skiptu út `YOUR_CLIENT_ID` fyrir þitt Wink Client ID):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Smelltu á **Save**.
</Steps>

:::tip
Client ID þitt er að finna í Wink reikningnum þínum undir [Applications](/is/portal/settings/applications).
:::

## Bæta við hluta á síðu

Þegar skriptin eru hlaðin á alla síðuna geturðu sett hvaða Wink hluta sem er á hvaða síðu sem er með **Code Block**.

<Steps>
1. Opnaðu síðuna sem þú vilt breyta.
2. Smelltu á **Edit** og bættu við nýjum **Code** kubbi þar sem þú vilt að hlutinn birtist.
3. Límdu inn HTML hlutarins, til dæmis efnishleðslutæki:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Smelltu á **Apply** og síðan **Save**.
</Steps>

## Tiltækir hlutir

| Hlutur | HTML tag | Tilgangur |
|---|---|---|
| Efnishleðslutæki | `<wink-content-loader>` | Sýna hótelkort, grindur, kort |
| Leit | `<wink-lookup>` | Leitargluggi fyrir áfangastaði |
| Leitartakki | `<wink-search-button>` | Opna ferðalagsval |
| Reikningstakki | `<wink-account-button>` | Skrá inn / notendavalmynd |
| Ferðalagatakki | `<wink-itinerary-button>` | Sýna núverandi ferðalag |
| Ferðalagavalkostur | `<wink-itinerary-picker>` | Fullt ferðalagform |
| Innkaupakörfu takki | `<wink-shopping-cart-button>` | Yfirlitstakki innkaupakörfu |

Sjá [Web Components](/is/integrations/web-components) fyrir fulla eiginleikalýsingu hvers hlutar.
