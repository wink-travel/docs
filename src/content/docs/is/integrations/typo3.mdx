**Wink TYPO3 viðbótin** samþættir Wink ferðainventar í TYPO3 13 sem innfædd efnisþætti. Ritstjórar bæta við Wink íhlutum á síður með hefðbundnum TYPO3 síðu ritstjóra — engar sniðmátabreytingar nauðsynlegar.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"
  href="https://github.com/wink-travel/wink-typo3-extension"
  description="Uppsprettukóði, leiðbeiningar fyrir þróunaraðila og leiðbeiningar um framlag."
/>

## Kröfur

- TYPO3 13.x
- PHP 8.2+
- Composer

## Uppsetning

<Steps>
1. Settu upp með Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Virkjaðu viðbótina:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Farðu í TYPO3 bakenda, veldu **Admin Tools → Settings → Extension Configuration → wink**.
4. Sláðu inn **Client ID**, **Client Secret** og veldu umhverfi. Vistaðu.
   Farðu á [Applications](/is/portal/settings/applications) til að sækja auðkenni þitt.
5. Bættu við statíska TypoScript sniðmátinu í rótarsniðmátið þitt:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Bæta efnisþætti við síðu

<Steps>
1. Opnaðu síðu í **Page** einingunni.
2. Smelltu á **+** hnappinn í efnisrýminu til að bæta við nýjum þætti.
3. Veldu einn af **Wink Travel** efnisþáttunum:
   - **Wink Content** — hótelkort, raðaðar grindur, kort
   - **Wink Lookup** — leitarlína áfangastaða
   - **Wink Search** — hnappur til að velja ferðalag
   - **Wink Account** — innskráningarhnappur
   - **Wink Itinerary** — hnappur fyrir stöðu ferðalags
   - **Wink Cart** — innkaupakörfu hnappur
4. Fyrir **Wink Content**, notaðu FlexForm reitina til að velja **Layout Type** og slá inn **Layout ID**.
5. Vistaðu og birtaðu síðuna.
</Steps>

## Tiltækir efnisþættir

| Þáttur | Tilgangur |
|---|---|
| **Wink Content** | Hótelkort, raðaðar grindur, kort og fleira |
| **Wink Lookup** | Leitarlína áfangastaða |
| **Wink Search** | Táknhnappur sem opnar ferðalagaval |
| **Wink Account** | Innskráning / notendareikningshnappur |
| **Wink Itinerary** | Ferðalags hnappur sem sýnir núverandi ferð |
| **Wink Cart** | Yfirlitshnappur innkaupakörfu |
