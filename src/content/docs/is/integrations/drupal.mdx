**Wink Drupal Module** bætir við Wink ferðainventarhlutum á Drupal síðuna þína sem innfædda kubba. Ritstjórar geta sett þá hvar sem er í uppsetningunni með hefðbundnu kubbakerfi.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Upprunakóði, leiðbeiningar fyrir þróunaraðila og leiðbeiningar um framlag."
/>

## Kröfur

- Drupal 10 eða 11
- PHP 8.1+
- Composer

## Uppsetning

<Steps>
1. Settu upp með Composer:
   ```bash
   composer require drupal/wink
   ```
2. Virkjaðu viðbótina:
   ```bash
   drush en wink -y
   ```
3. Farðu í **Stjórnandi → Stillingar → Vefþjónustur → Wink Stillingar** (`/admin/config/wink`).
4. Sláðu inn **Client ID** og **Client Secret**, veldu umhverfi þitt og smelltu á **Vista**.
   Farðu á [Applications](/is/portal/settings/applications) til að sækja auðkenni þitt.
</Steps>

## Bæta kubbi við síðu

<Steps>
1. Farðu í **Stjórnandi → Uppbygging → Kubbalayout**.
2. Smelltu á **Setja kubb** í svæðinu þar sem þú vilt hafa Wink efni.
3. Leitaðu að **Wink** til að finna tiltæka kubba:
   - **Wink Content** — birtir hótelkort, grind eða kort
   - **Wink Lookup** — leitargluggi fyrir áfangastaði
   - **Wink Search** — hnappur til að velja ferðalag
   - **Wink Account** — innskráningarhnappur
   - **Wink Itinerary** — hnappur sem sýnir stöðu ferðalags
   - **Wink Cart** — innkaupakörfu hnappur
4. Fyrir **Wink Content**, stilltu **Layout Type** og **Layout ID** í kubbasetningum.
5. Vistaðu kubbasetningarnar.
</Steps>

## Tiltækir kubbar

| Kubbur | Tilgangur |
|---|---|
| **Wink Content** | Sýnir hótelkort, raðaðar grindur, kort og fleira |
| **Wink Lookup** | Leitargluggi fyrir áfangastaði og hótel |
| **Wink Search** | Táknhnappur sem opnar ferðalagsskjáinn |
| **Wink Account** | Innskráning / notendareikningshnappur |
| **Wink Itinerary** | Ferðalags hnappur sem sýnir núverandi ferðalag |
| **Wink Cart** | Yfirlitshnappur innkaupakörfu |
