Ef þú ert að keyra auglýsingaherferðir í gegnum Sojern geturðu tengt Sojern reikninginn þinn við Wink með því að slá inn Sojern GTM Container ID í Wink vefsvæðinu. Þegar tengt er, kveikir Sojern rekjapixlinn sjálfkrafa fyrir eignina þína — sem nær yfir allt bókunarferli gesta frá leit og þar til bókun er staðfest.

<Aside type="tip">
Þetta gerir ráð fyrir að þú eigir þegar virkan Sojern reikning og hafir fengið GTM Container ID frá reikningsstjóra þínum hjá Sojern. Ef þú átt ekki enn Sojern reikning, farðu á [sojern.com/contact](https://www.sojern.com/contact) til að byrja.
</Aside>

---

## Hvað þú þarft að gera

<Steps>

1. **Fáðu GTM Container ID frá Sojern**

   Hafðu samband við reikningsstjóra þinn hjá Sojern og biðjaðu um **GTM Container ID**. Það er á formi `GTM-XXXXXXX`. Þetta er ekki það sama og Sojern Pixel ID — vertu viss um að biðja sérstaklega um GTM Container ID.

2. **Sláðu það inn í Wink vefsvæðið**

   Skráðu þig inn á [my.wink.travel](https://my.wink.travel), veldu reikninginn þinn og farðu í **Customizations** flipann. Límdu Sojern GTM Container ID í Sojern reitinn og vistaðu. Samþættingin er nú virk.

3. **Staðfestu við reikningsstjóra þinn hjá Sojern**

   Láttu reikningsstjóra þinn vita að samþættingin sé virk. Hann getur staðfest að pixlinn kveiki rétt á `wink.travel` fyrir eignina þína og að gögn flæði inn í Sojern reikninginn þinn.

</Steps>

<Aside type="note">
Þú þarft ekki að setja neinn pixlakóða á eigin vefsíðu fyrir þessa samþættingu. Wink hleður Sojern gáminum þínum sjálfkrafa á öllum síðum þar sem eignin þín er virk.

Þessi stilling virkar fyrir hvaða sérsniðna GTM Container ID sem er. Við notuðum Sojern sem dæmi þar sem þetta er mjög algeng samþætting.
</Aside>

## Hvað Wink gerir á okkar vegum

Til gagnsæis, hér er hvað Wink stillir til að láta þessa samþættingu virka:

- **Reitur fyrir Sojern GTM Container ID** er aðgengilegur í Customizations flipanum í Wink vefsvæðinu þínu. Þegar þú slærð inn Container ID, sannprófar Wink sniðið og geymir það örugglega fyrir eignina þína.
- **Á hverri síðu þar sem eignin þín er virk** á Wink, er Sojern GTM Container ID þitt sett inn í síðuna og tagjastjórnunarkerfi Wink hleður gáminum þínum samhliða sínum eigin. Þetta gerist sjálfkrafa — engar handvirkar aðgerðir eru nauðsynlegar frá Wink teyminu fyrir hverja eign.
- **Sojern gáminn þinn er hlaðinn ósamstillt**, sem þýðir að hann hefur ekki áhrif á hleðsluhraða síðunnar fyrir gesti þína.
- **Wink eigin rekjapixl og Sojern pixl keyra samtímis**, lesa úr sama sameiginlega atburðagagnasafni. Hver bókunaratburður sem Wink skráir er samstundis aðgengilegur Sojern gáminum þínum.
- **Þegar eignin þín er ekki skoðuð** — til dæmis þegar gestur skoðar aðra hótel á Wink — er Sojern gáminn ekki hlaðinn. Þetta tryggir að Sojern fær aðeins gögn sem tengjast eigninni þinni.

## DataLayer tilvísun

Wink ýtir GA4-samhæfum atburðum í `dataLayer` í gegnum allt bókunarferlið. GTM gáminn þinn les úr sama sameiginlega `dataLayer`, svo hvaða kveikju eða breytu sem þú stillir hefur aðgang að öllum eiginleikum sem lýst er hér að neðan.

Hver atburður fylgir staðlaðri GA4 Enhanced Ecommerce uppbyggingu: efst er `event` strengur og `ecommerce` hlutur. Áður en ýtt er á atburð, hreinsar Wink fyrri ecommerce gagnaflutning til að koma í veg fyrir að gömul gögn blandist nýjum kveikjum.

<Aside type="note">
**dataLayer endurstillingarmynstur**

Wink endurstillir ecommerce hlutinn fyrir hvern atburð:

```js
dataLayer.push({ ecommerce: null });   // hreinsa fyrri atburð
dataLayer.push({ event: '...', ecommerce: { ... } });
```

Ekki treysta á að ecommerce eiginleikar haldist milli atburða. Lestu þá alltaf þegar kveikt er á atburði innan sama atburðar.
</Aside>

### Atburðir í bókunarferli

Þessir atburðir kveikja sjálfkrafa þegar gestur fer í gegnum bókunarferlið. Stilltu GTM kveikjur sem **Custom Event** kveikjur sem passa við atburðanafnið hér að neðan.

| Skref | Atburður | Hvenær hann kveikir |
|-------|----------|---------------------|
| 1 | `view_item_list` | Hótel birting í töflu eða lista hleðst |
| 2 | `view_item` | Gestur skoðar ákveðna eignarsíðu |
| 3 | `add_to_cart` | Gestur bætir herbergi í körfu |
| 4 | `begin_checkout` | Úttekt / greiðslusíða hleðst |
| 5 | `add_payment_info` | Gestur velur greiðslumáta |
| 6 | `login` | Gestur skráir sig inn með SSO |
| 7 | `purchase` | Bókun staðfest |
| 8 | `refund` | Bókun felld niður |

### Sérsniðnar víddir í GA4

GA4 hunsar sérsniðnar atriðabreytur í skýrslum þar til þú skráir þær sérstaklega sem sérsniðnar atriða víddir. Þú þarft aðeins að skrá þær breytur sem þú ætlar að nota í skýrslum — ekki þarf að skrá allar 20.

<Steps>

1. **Opnaðu Sérsniðnar skilgreiningar**

   Í GA4, farðu í **Admin** (gírinn neðst til vinstri) → veldu eignina þína → **Data display** → **Custom definitions**.

2. **Búðu til nýja sérsniðna vídd**

   Smelltu á **Create custom dimensions**.

3. **Fylltu út upplýsingar um víddina**

   - **Dimension name** — mannlegur lesanlegur titill sem birtist í GA4 skýrslum (sjá töflu hér að neðan)
   - **Scope** — veldu **Item**
   - **Event parameter** — nákvæm breytunafn eins og það birtist í dataLayer (sjá töflu hér að neðan)
   - **Description** — valfrjálst en mælt með

4. **Vistaðu og endurtaktu**

   Smelltu á **Save** og endurtaktu fyrir hverja breytu sem þú vilt fylgjast með.

</Steps>

<Aside type="note">
Sérsniðnar víddir geta tekið allt að 24 klukkustundir að birtast í GA4 skýrslum eftir skráningu. Sögu gögn sem safnað var fyrir skráningu eru ekki uppfærð aftur.

GA4 leyfir allt að 50 sérsniðnar atriða víddir á staðlaðar eignir (360 eignir leyfa fleiri).
</Aside>

Notaðu eftirfarandi töflu sem viðmið við að búa til sérsniðnar víddir. Gildi í dálkinum **Event parameter** verður að passa nákvæmlega — GA4 er háð háum og lágum stöfum.

| Víddarnafn | Event parameter | Tegund |
|------------|-----------------|--------|
| Property ID | `property_id` | Texti |
| Property name | `property_name` | Texti |
| Property brand | `property_brand` | Texti |
| Property chain | `property_chain` | Texti |
| Property URL name | `property_url_name` | Texti |
| City name | `city_name` | Texti |
| Country code | `country_code` | Texti |
| Check-in date | `start_date` | Texti |
| Check-out date | `end_date` | Texti |
| Number of adults | `num_adults` | Tala |
| Number of children | `num_children` | Tala |
| Cancellation policy | `cancellation_policy` | Texti |
| Rate plan ID | `rate_plan_id` | Texti |
| Room rate ID | `room_rate_id` | Texti |
| Room rate name | `room_rate_name` | Texti |
| Number of perks | `num_perks` | Tala |
| Room location | `room_location` | Texti |
| Room view | `room_view` | Texti |
| Room classification | `room_classification` | Texti |
| Bed type | `bed_type` | Texti |

### `virtualPageView`

Þetta er sérsniðinn Wink atburður — ekki hluti af staðlaða GA4 atburðasettinu. Hann er ýttur á við hverja síðuferð og ber með sér tæki- og lotusamhengi. Hann inniheldur ekki `ecommerce` hlut.

```json
{
  "event": "virtualPageView",
  "device": "Desktop",
  "deviceAgent": "Mozilla/5.0 ...",
  "os": "MacOS",
  "path": "/hotels/grand-resort",
  "title": "Grand Resort — Wink",
  "userId": "user_abc123",
  "startDate": "2026-06-01",
  "endDate": "2026-06-05",
  "guests": "2",
  "rooms": "1"
}
```

| Eiginleiki | Lýsing |
|------------|--------|
| `device` | Greind tækitegund — `Desktop`, `Android`, `iPhone`, `iPad` eða svipað |
| `deviceAgent` | Upprunalegur user agent strengur |
| `os` | Greint stýrikerfi — `Windows`, `MacOS`, `Linux`, `UNIX` |
| `path` | Núverandi slóð á síðunni |
| `title` | Núverandi titill síðunnar |
