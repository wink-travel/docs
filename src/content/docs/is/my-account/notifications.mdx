Stundum viljum við hafa samband við þig. Þegar það gerist sérðu bjölluna efst í hægra horninu breytast.
Til að lesa það sem við sendum þér:

<Steps>
1. Smelltu á `Reikningur > Tilkynningar` í aðalvalmyndinni.
2. Þú munt sjá nýjar og eldri tilkynningar.
3. Veldu að `Merkja sem lesið` eða `Eyða` tilkynningunum þegar þú hefur lesið þær.
</Steps>

## API

Forritarar sem vilja stjórna `Affiliates` geta farið á [Developers > API > Affiliate](/is/developers/apis/#affiliate-api).
