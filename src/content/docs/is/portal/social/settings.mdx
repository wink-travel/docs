# Stillingar

Þegar þú byrjar fyrst með Social munum við spyrja þig hvernig þú vilt að við búum til efnið þitt. Þessar valkostir eru alltaf tiltækir svo þú getir auðveldlega breytt þeim síðar.

Farðu í `Preferences` í hliðarstikunni til að stjórna stillingunum þínum.

Hluti Preferences inniheldur grunnstillingar fyrir efnisgerð:

1. **Preferred customization** Veldu hvaða sérsnið þú vilt nota við að búa til efni. Sérsnið stjórna þemalitum, sjálfgefnu ferðalagi, gjaldmiðli, tungumáli og fleiru. Þú býrð til sérsnið undir reikningnum þínum á [my.wink.travel](https://my.wink.travel).
2. **Auto-approve** Þegar þetta er virkjað eru búin til færslur birtar sjálfkrafa án þess að þurfa samþykki þitt. Virkjaðu þetta þegar þú ert viss um gæði efnisins sem við framleiðum fyrir þig.
3. **Approver name** Nafn þess eða teymsins sem fær beiðnir um samþykki í tölvupósti.
4. **Approver e-mail** Tölvupóstfangið þar sem beiðnir um samþykki fyrir komandi færslur verða sendar.

## Content creators

Social kemur með marga efnisframleiðendur til að velja úr. Hver framleiðandi einbeitir sér að einum sérstökum hætti til að búa til einstakt efni fyrir eign þína með AI. Hver efnisframleiðandi hefur **Weight** eiginleika — hærri þyngd eykur líkurnar á að sá framleiðandi verði valinn við að búa til færslur.

Hér eru allir tiltækir efnisframleiðendur:

| Framleiðandi | Lýsing |
| -- | -- |
| **Destination** | Notar staðsetningu þína til að búa til gagnlegan texta um svæðið |
| **Room type** | Býr til bókanlega færslu með einni herbergistegund þinni með AI-búnum texta og myndum af herbergistegundinni |
| **Activity** | Býr til bókanlega færslu með einni af þínum afþreyingum |
| **Add-On** | Býr til bókanlega færslu með einni af þínum viðbótum |
| **Attraction** | Býr til bókanlega færslu með einni af þínum aðdráttaraflum |
| **Place** | Býr til bókanlega færslu með einni af þínum stöðum |
| **Meeting room** | Býr til bókanlega færslu með einni af þínum fundarherbergjum |
| **Restaurant** | Býr til bókanlega færslu með einni af þínum veitingastöðum |
| **Spa** | Býr til bókanlega færslu með einni af þínum heilsulindum |
| **Property** | Býr til bókanlega færslu með best verðlagða herbergi þínu með AI-búnum texta og myndum af eigninni |
| **User sentiment** | Safnar saman gestagagnrýni frá vefnum í tilfinningadrifna færslu sem dregur fram hvað gestir elska við eignina þína |
| **Funny** | Býr til léttar, skemmtilegar færslur til að halda straumnum áhugaverðum |

:::tip[Pro tip]
Gakktu úr skugga um að þú hafir þann birgðutegund sem þarf fyrir valda efnisframleiðendur. Til dæmis, ef þú velur "Restaurant," þarftu að hafa að minnsta kosti einn veitingastað skráðan á eigninni þinni.
:::

## Tone of voice

<figure>
<CldImage
  src="academy/social_personas"

  <figcaption>*Mynd 3 - Tone of voice*</figcaption>

Tónn raddarinnar setur tungumálastílinn sem AI notar við að búa til efni. Veldu þá tóna sem passa við persónuleika vörumerkisins þíns. Eins og með efnisframleiðendur hefur hver tónn **Weight** til að stjórna hversu oft hann er notaður.

Hér eru tiltækir tónar:

| Tónn | Stíll |
| -- | -- |
| **Neutral** | Rólegur, áreiðanlegur, staðreyndamiðaður. Engin yfirlýsing, engar brandarar — bara áreiðanleg, vel uppbyggð þekking |
| **Friendly** | Hlýlegur, aðgengilegur, brosandi í gegnum texta með afslappaðri styttingu og stundum emoji |
| **Luxury** | Fínn, fágaður. Orð eins og „glæsilegur“ og „sérsmíðaður“ sem láta lesendur finna fyrir sérstöku |
| **Foodie** | Ástríðufullar, munnvatnandi lýsingar. Maturspornó málfar sem gerir lesendur svanga |
| **Adventurous** | Djarfur, óhræddur, könnuður. Lýsingar sem vekja skynjun og gera allt epískt |
| **Family** | Hreinlæti, barnvænn spenningur. Hlýleg foreldraorku um öruggar og skemmtilegar upplifanir |
| **Romantic** | Ljóðrænn, þráhyggjukenndur, hjartsláttar. Sólsetur, mjúkt málfar og ást |
| **Humorous** | Hnyttinn, snöggar einlínum, snjall orðaleikur. Lesst eins og stand-up sýning |
| **Eco** | Heiðarlegur og réttlátur. Fagnar grænum valkostum og sjálfbærum ákvörðunum |
| **Local insider** | Mjög sértækar staðbundnar vísanir, varar við ferðamannagildrum, notar staðbundið slangur |
| **Inclusive** | Vinalegur, hátíðlegur og staðfestandi. Lætur alla finna sig séða og metna |
| **Mystical** | Dularfullur, andlegur, alheimssannindi. Mjúk boð um að líta inn á við |
| **Futuristic** | Sléttur, framúrskarandi. Elskar hugtök eins og „paradigm shift“ og bjartsýnar framtíðarsýn |
| **Budget** | Hagrænn, úrræðagóður, stoltur af sparnaði. Fagnar hagkvæmum fundum og ókeypis upplifunum |

## Post types

Tegundir færslna ákvarða markmið og uppbyggingu efnisins sem búið er til. Hver tegund færslu hefur **Weight** til að stjórna hversu oft hún er notuð.

| Tegund færslu | Lýsing |
| -- | -- |
| **Transactional** | Skýr, hnitmiðuð, umbreytingamiðuð. Hvetur til bókana og beinna aðgerða með staðreyndum, verðlagi og næstu skrefum |
| **Awareness** | Vörumerkjabyggjandi efni sem er ætlað til að kynna eignina þína og auka sýnileika meðal mögulegra gesta |
| **Sentiment** | Tilfinningalega áhrifaríkt efni sem skapar jákvæð tengsl og byggir upp tilfinningalegt samband við áhorfendur þína |

Þú gætir viljað beina markaðsstarfi þínu að ákveðnu tungumáli og svæði. Segðu okkur hvaða tungumál þú vilt búa til efni fyrir og hvaða tímabelti þú hefur áhuga á og við sjáum um restina. Færslur eru birtar á áætluðum tíma í valda tímabeltinu þínu.

*Flestar eignir nota eitt tungumál og sjálfgefið tungumál þeirra er staðbundið tungumál.*
