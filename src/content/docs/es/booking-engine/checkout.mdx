El proceso de pago guía a los viajeros desde la selección de habitación hasta el pago en tres pasos.

## Paso 1 — Configurador de habitación

Después de seleccionar un tipo de habitación en la página del establecimiento, se abre el configurador de habitación. Aquí el viajero puede:

- Elegir la configuración de camas y las preferencias de la habitación.
- Añadir artículos adicionales (desayuno, spa, actividades, etc.).
- Ingresar la información del huésped.
- Revisar el desglose de precios (totales netos y brutos).
- Añadir la habitación a su carrito de compras.

## Paso 2 — Carrito de compras

El carrito de compras muestra todos los artículos listos para comprar:

- Tipo de habitación, fechas y número de huéspedes para cada artículo.
- Política de cancelación para cada reserva.
- Desglose de precios por artículo y total.

Los viajeros pueden eliminar artículos o proceder al pago.

## Paso 3 — Pago

El paso de pago incluye un widget de pago seguro donde el viajero ingresa los datos de la tarjeta y completa la transacción. Después de un pago exitoso, el viajero es redirigido a la página de confirmación.

Si el pago falla, se muestra un mensaje de error y el viajero puede intentarlo de nuevo.
