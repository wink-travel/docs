La **aplicación Wink Wix** añade componentes web de Wink a tu sitio Wix como widgets nativos que puedes arrastrar y soltar en el Editor de Wix. No se requiere programación para los propietarios del sitio.

<LinkCard

/>

## Instalar

<Steps>
1. Ve al [Wix App Market](https://www.wix.com/app-market) y busca **Wink Travel**.
2. Haz clic en **Agregar al sitio** y sigue las indicaciones para autorizar la aplicación.
3. Una vez instalada, abre **Panel → Configuración de Wink** e ingresa tu **Client ID** y **Client Secret**.
   Ve a [Aplicaciones](/es/portal/settings/applications) para obtener tus credenciales.
4. Haz clic en **Guardar** y luego en **Probar conexión** para confirmar que Wink está conectado.
</Steps>

## Añadir un componente a una página

<Steps>
1. Abre el **Editor de Wix** y ve a la página donde quieres añadir contenido de Wink.
2. Haz clic en **Agregar (+) → Aplicaciones → Wink Travel** para ver todos los widgets disponibles.
3. Arrastra un widget a la página — por ejemplo, **Wink Content** para mostrar una cuadrícula de hoteles.
4. En el panel de configuración del widget, selecciona el **Tipo de diseño** y pega el **ID de diseño** desde Wink Studio.
5. Haz clic en **Publicar** para que los cambios se apliquen en vivo.
</Steps>

## Widgets disponibles

| Widget | Propósito |
|---|---|
| **Wink Content** | Muestra tarjetas de hoteles, cuadrículas clasificadas, mapas y más |
| **Wink Lookup** | Barra de búsqueda para destinos y hoteles |
| **Wink Search** | Botón con ícono que abre el selector de itinerarios |
| **Wink Account** | Botón de inicio de sesión / cuenta de usuario |
| **Wink Itinerary** | Botón de itinerario que muestra el viaje actual |
| **Wink Cart** | Botón resumen del carrito de compras |
