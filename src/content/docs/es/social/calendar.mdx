La publicación funciona así:

<Steps>
1. El contenido se genera a partir de tus preferencias una semana antes del mes de publicación.
2. Recibes un correo electrónico indicándote que apruebes el contenido recién creado [o notificándote que se ha creado contenido para el mes siguiente y que no tienes que hacer nada más porque has elegido aprobar automáticamente el contenido].
3. Inicias sesión en [social.wink.travel](https://social.wink.travel) para aprobar las publicaciones.
4. Las publicaciones se programan para ser publicadas. (Aún puedes detener la tarea de publicación si lo haces antes de que se publique. De lo contrario, tendrás que ir a la red social y eliminar o modificar la publicación manualmente allí).
5. Las publicaciones se publican en el momento más oportuno para la zona horaria que seleccionaste.
</Steps>

Una vez que el contenido está aprobado para su publicación, se refleja en tu calendario.

:::note
Habrá una entrada para cada idioma y cada red social.

Por ejemplo, si tienes 3 redes sociales y 3 idiomas seleccionados, verás 9 entradas en el calendario.
:::
