## Payment API

Los endpoints de pago son para desarrolladores que quieren comprar inventario de viajes. Esto se puede hacer vía API como Agente de Viajes registrado o usando nuestra API junto con nuestro widget de pago compatible con PCI para todas las demás entidades.
La API de Pago expone endpoints para afiliados y hoteles para rastrear reservas, analíticas y disponibilidad de fondos. También pueden elegir retirar fondos disponibles a su cuenta bancaria.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Saber más
</LinkButton>
