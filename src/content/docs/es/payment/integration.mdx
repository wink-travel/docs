Si eres una empresa de viajes y quieres externalizar la parte de pagos y desembolsos de tu negocio a TripPay, aquí tienes 6 pasos básicos para comenzar:

<Steps>
1. Crea una [cuenta de usuario](/es/user/register) en Wink.
2. Crea una cuenta con [TripPay](https://pay.wink.travel).
3. Crea una [Aplicación](/es/developers/applications) en TripPay que te permita comunicarte con nuestra API.
4. Crea y asigna tus beneficiarios con TripPay. *por ejemplo, para la mayoría, eso significa crear cuentas para las propiedades en tu plataforma a las que quieres que TripPay pague.*
5. Integra el [componente web de pago](/es/developers/web-components/#payment) en tu sitio.
6. Envía a TripPay el `carrito de compras` del viajero e inicia el proceso de pago.
</Steps>

:::tip
Para una guía completa sobre cómo usar TripPay para tu sitio de viajes, lee la guía [Integración con TripPay](/es/guides/integrating-with-trip-pay).
:::

## API

Los desarrolladores que quieran gestionar `Pagos`, vayan a [Desarrolladores > API > Pagos](/es/developers/apis/#payment-api).
