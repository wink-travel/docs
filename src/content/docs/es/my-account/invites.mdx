Un usuario de Wink puede elegir permitirte gestionar una de sus cuentas de propiedad, afiliado o agente de viajes. Cuando eso sucede, recibirás un correo electrónico con una solicitud de invitación.

import { Steps } from '@astrojs/starlight/components';

Para ver tus invitaciones actuales de Wink, sigue estos pasos:

<Steps>
 1. Inicia sesión en [my.wink.travel](https://my.wink.travel).
 2. Haz clic en `Invites` en la barra lateral izquierda.
 3. Verás una lista de todas las invitaciones actuales.
 4. Esta sección mostrará todas tus solicitudes.
 5. Elige `Accept` o `Reject` la invitación.
 6. Si aceptaste la invitación, la cuenta aparecerá en la sección `Accounts`.
</Steps>
