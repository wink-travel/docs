הפתרון שלנו לסוכני נסיעות רשומים הוא [Wink Agent](https://agent.wink.travel). זהו פורטל שירות עצמי שבו סוכנים יכולים לחפש ולהזמין עבור לקוחותיהם ולהרוויח עמלה על כל הזמנה.

סוכני נסיעות מתקדמים יותר יכולים לבחור להשתלב ברמת ה-API לצורך טיפול בתשלום ובפירעונות מחוץ לפלטפורמת Wink.

לסוכני נסיעות יש תת-קבוצה של אותם תכונות פורטל כמו ב-[Wink Studio](/he/studio/what-is-studio) ולכן לא נרחיב עליהן כאן.

## חיפוש

גש ל-[Search](/he/studio/search) כדי ללמוד כיצד לחפש ולהזמין מלונות.

אחרת, המשך לסעיף הבא.

## API

מפתחים שרוצים לנהל `Travel Agent` יכולים לגשת ל-[Developers > API > Travel Agent](/he/developers/apis/#travel-agent-api).
