כאשר מתבצעת הזמנה ב-Wink, מתרחשים הדברים הבאים:

<Steps>
1. מנהל הערוץ שלך, CRS או PMS מקבל הודעה.
2. אתה מקבל מייל אישור *(פרטי יצירת קשר עם דלפק ההזמנות)* עם קישור לניהול ההזמנה החדשה שלך.
3. הנוסע מקבל מייל אישור הכולל גם כיצד ליצור קשר עם דלפק ההזמנות שלך.
4. סוכן הנסיעות מקבל את אותו מייל אישור כמו הנוסע.
4. מפתחים יכולים להאזין לאירוע ה-webhook `booking.created` ולקבל מידע מפורט על ההזמנה מאיתנו בזמן אמת.
</Steps>

ניתן לעקוב אחר הקישור במייל, או לגשת ל-`Account > Bookings` מהסרגל הראשי כדי להתחיל.

## רשת ההזמנות

רשת ההזמנות מציגה את כל ההזמנות הקודמות והעתידיות שלך, ממוינות לפי תאריך ההזמנה בסדר עולה.

### מסננים

ניתן להחיל מסננים כדי למצוא את ההזמנות שאתה מחפש.

- תאריך הזמנה [טווח]
- תאריך הגעה [טווח]
- תאריך עזיבה [טווח]
- קוד הזמנה
- שם משפחה של הנוסע
- שם פרטי של הנוסע
- שם תעריף ראשי
- שם הנכס

### ייצוא

ניתן לייצא הזמנות התואמות לקריטריוני המסנן הנוכחיים שלך לקובץ CSV.

כדי לייצא את ההזמנות שלך:

<Steps>
1. ודא שהמסננים הרצויים פעילים.
2. המתן לרענון רשת ההזמנות.
2. לחץ על כפתור האייקון 🗂️ (*ייצוא קובץ*) בפינה הימנית התחתונה של הרשת.
</Steps>

## הזמנה

בחר את ההזמנה שברצונך לעבוד איתה מרשת ההזמנות ותועבר לעמוד ה-`Booking`.
עמוד זה מכיל את כל הפרטים שעליך לדעת כדי לקבל את פני האורח כראוי ביום ההגעה.

העמוד כולל גם כמה פעולות חשובות שעליך לבצע כדי לשרת את האורחים כראוי. ניתן למצוא את כל הפעולות הזמינות בתפריט הנפתח `Actions` בראש העמוד.

### שלח מחדש מייל

כדי לשלוח מחדש את מייל האישור לנוסע, לחץ על `Actions > Resend confirmation email`.

### ביטול הזמנה

כדי לבטל את ההזמנה בתור הנכס, לחץ על `Actions > Cancel booking`.

תופיע חלונית קופצת שתאפשר לך:

<Steps>
1. לבחור את סוג הביטול. *למשל תאריכים שגויים*
2. להקליד סיבה לביטול.
</Steps>

*כפתור זה פועל בהתאם לכללי מדיניות הביטול שלך ולכן לא תמיד יהיה זמין.*

## API

מפתחים שרוצים לנהל `Bookings` כסוכן נסיעות, יכולים לגשת ל-[Developers > API > Travel Agent](/he/developers/apis/#travel-agent-api).
