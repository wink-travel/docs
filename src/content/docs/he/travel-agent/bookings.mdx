כאשר מתבצעת הזמנה ב-Wink, מתרחשים הדברים הבאים:

<Steps>
  1. מנהל הערוצים, שירותי ה-CRS או מערכת ה-PMS שלך מקבל הודעה.
  2. אתה מקבל אימייל אישור *(אנשי קשר בדלפק ההזמנות)* עם קישור לניהול ההזמנה החדשה שלך.
  3. הנוסע מקבל אימייל אישור הכולל גם מידע על אופן יצירת קשר עם דלפק ההזמנות.
  4. סוכן הנסיעות מקבל את אותו אישור הזמנה כמו הנוסע.
  5. מפתחים יכולים להאזין ל- `booking.created` אירוע webhook וקבל מאיתנו מידע מפורט על ההזמנה בזמן אמת.
</Steps>


או שתעקוב אחר הקישור באימייל, או שתעבור אל`Account > Bookings`מסרגל הניווט הראשי כדי להתחיל.

## רשת הזמנות

טבלת ההזמנות מציגה את כל ההזמנות הקודמות והעתידיות שלך, ממוינות לפי תאריך ההזמנה בסדר עולה.

### מסננים

ניתן להחיל פילטרים כדי למצוא את ההזמנות שאתם מחפשים.

* תאריך ספר \[טווח]
* תאריך הגעה \[טווח]
* תאריך יציאה \[טווח]
* קוד הזמנה
* שם משפחה של המטייל
* שם פרטי של המטייל
* שם שיעור ראשי
* שם הנכס

### יְצוּא

ניתן לייצא הזמנות התואמות את קריטריוני הסינון הנוכחיים שלך לקובץ CSV.

כדי לייצא את ההזמנות שלך:

<Steps>
  1. ודא שהמסננים הרצויים פעילים.
  2. המתן לרענון רשת ההזמנות.
  3. לחצו על ה-🗂️ (*ייצוא קבצים*כפתור הסמל ) בפינה הימנית התחתונה של הרשת.
</Steps>


## הזמנה

בחר את ההזמנה שברצונך לעבוד איתה מרשת ההזמנות ותנותב אל`Booking`דף. דף זה מכיל את כל הפרטים שאתם צריכים לדעת כדי לקבל את פני האורח שלכם כראוי ביום ההגעה.

הדף מכיל גם כמה פעולות חשובות שעליכם לבצע כדי לתת שירות ראוי לאורחים שלכם. תוכלו למצוא את כל הפעולות הזמינות ב`Actions`תפריט נפתח בראש העמוד.

### שלח שוב דוא"ל

כדי לשלוח שוב את דוא"ל האישור לנוסע, לחץ על`Actions > Resend confirmation email`.

### ביטול הזמנה

כדי לבטל את ההזמנה כמקום האירוח, לחצו`Actions > Cancel booking`.

יוצג חלון קופץ שיאפשר לך:

<Steps>
  1. בחר את סוג הביטול. *לדוגמה, תאריכים שגויים*
  2. הקלד סיבה לביטול.
</Steps>


*כפתור זה פועל בהתאם לכללי מדיניות הביטול שלכם ולכן לא יהיה זמין תמיד.*

## ממשק API

מפתחים שרוצים לנהל`Bookings`, כסוכן נסיעות, יכול לפנות אל[מפתחים > API > סוכן נסיעות](/developers/apis/#travel-agent-api).
