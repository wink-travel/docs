מנהל קישורים מאפשר לך לשמור ולארגן כתובות URL שאתה אוהב לשימוש מאוחר יותר. הסימניות בדפדפן שלך הן דוגמה למנהל קישורים. עם הופעת הרשתות החברתיות, נוצר צורך גדול יותר לחשוף ולהנגיש קישורים לחברים ולקהלים שלך. הפיד שלך בפייסבוק הוא דוגמה למנהל קישורים ידידותי לרשתות חברתיות.

בנינו את [WinkLinks](https://i.trvl.as/) כדי להיות מנהל הקישורים לתעשיית התיירות. הוא כאן כדי ליידע, להלהיב, לפרסם... ומאפשר לך להזמין דילים מעולים מבתי מלון ברחבי העולם.

## תכונות

- WinkLinks מאפשר לך להוסיף את כל קישורי השותפים הקיימים שלך, כמו Commission Junction ואמזון, בחינם.
- אתה מקבל כתובת URL ייחודית ומרשימה, כמו [https://i.trvl.as/bob](https://i.trvl.as/bob), לזמנים שבהם ניתן לשתף רק כתובת URL אחת.
- ניתן לארגן ולשתף קישורים ולהעלות קבצים.
- בתי מלון, רשתות ומותגים יכולים להשתמש ב-WinkLinks כמנוע הזמנות מתקדם:
    - תומך בריבוי נכסים.
    - תומך בריבוי יעדים.
    - תומך בריבוי מטבעות.
    - תומך בריבוי שפות.
- WinkLinks תומך ב-`תוכן עשיר` *(ראה למטה)*.
- מעבר מ-IG להזמנה ב-2 לחיצות בלבד 🚀

## תוכן עשיר

:::note
הגדר את `Intelligent` ל-🟢 ב-[הגדרות](/he/link-manager/settings) שלך כדי להשתמש בתכונה זו.
:::

אם אתה משתף קישור מאתר שתומך בתוכן `עשיר` / `oEmbed`, הקישור שלך מתעורר לחיים ב-WinkLinks.

אתרים נתמכים:

- Facebook - *אינטראקציה עם הפוסט וניגון וידאו*.
- Instagram - *אינטראקציה עם הפוסט וניגון וידאו*.
- YouTube - *ניגון וידאו*.
- X (לשעבר Twitter) - *אינטראקציה עם פוסט*.
- Spotify - *ניגון רשימות השמעה מועדפות*.
- Wink - *ראה `מתקדם`*.

## מתקדם

WinkLinks הוא גם דרך מצוינת לחשוף את מלאי התיירות שמצאת, דרך [Wink Studio](https://studio.wink.travel), ואתה רוצה לפרסם לקהל שלך.

השתמש ב-WinkLinks כדי למכור את:

- [קישורים לשיתוף](/he/studio/shareable-links) - בחר להציג כ:
    - [באנר פרסומי](/he/integrations/web-components/#content-loader).
    - קישור רגיל עם עדכוני מחירים בזמן אמת.
- [כרטיסים](/he/studio/cards) - כרטיסים להזמנה עם מחירים וזמינות בזמן אמת.
- [רשתות](/he/studio/grids) - רשת עם כרטיסים להזמנה.
- [מפות](/he/studio/maps) - מפה עם סימנים שנפתחים לכרטיסים להזמנה.

:::tip
כדי לראות חשבון בפעולה, בדוק אחד מהחשבונות שלנו: [i.trvl.as/starringmay](https://i.trvl.as/starringmay).
:::
