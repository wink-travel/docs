מנהל קישורים מאפשר לך לשמור ולארגן כתובות URL שאתה אוהב לשימוש מאוחר יותר. סימניות הדפדפן שלך הן דוגמה למנהל קישורים. עם הופעת הרשתות החברתיות, נוצר צורך גדול יותר לחשוף ולהנגיש קישורים לחברים ולקהלים שלך. הפיד שלך בפייסבוק הוא דוגמה למנהל קישורים ידידותי חברתית.

:::caution
רוב מנהלי הקישורים דורשים מנוי לכל מי שמפרסם קישורי שותפים.
:::

בנינו את [WinkLinks](https://i.trvl.as/) ; מנהל קישורים חינמי לחלוטין, המתמקד בתחום התיירות, כדי להלהיב, לפרסם ולמכור ביעילות באינטרנט.

## תכונות

- WinkLinks מאפשר לך להוסיף את כל קישורי השותפים הקיימים שלך, כמו Commission Junction ואמזון, בחינם.
- אתה מקבל כתובת URL ייחודית ומרשימה, כמו [https://i.trvl.as/bob](https://i.trvl.as/bob), לזמנים שבהם ניתן לשתף רק כתובת URL אחת.
- ניתן לארגן ולשתף קישורים ולהעלות קבצים.
- מלונות, רשתות ומותגים יכולים להשתמש ב-WinkLinks כמנוע הזמנות מתקדם:
    - תומך בריבוי נכסים.
    - תומך בריבוי יעדים.
    - תומך בריבוי מטבעות.
    - תומך בריבוי שפות.
- WinkLinks תומך ב-`rich content` *(ראה למטה)*.
- עבור מ-IG להזמנה ב-2 לחיצות בלבד 🚀

## תוכן עשיר

:::note
הפעל את `Intelligent` ל-🟢 ב-[הגדרות](/he/link-manager/settings) שלך כדי להשתמש בתכונה זו.
:::

אם אתה משתף קישור מאתר שתומך בתוכן `rich` / `oEmbed`, הקישור שלך מתעורר לחיים ב-WinkLinks.

אתרים נתמכים:

- Facebook - *אינטראקציה עם הפוסט וניגון וידאו*.
- Instagram - *אינטראקציה עם הפוסט וניגון וידאו*.
- YouTube - *ניגון וידאו*.
- X (לשעבר Twitter) - *אינטראקציה עם פוסט*.
- Spotify - *ניגון רשימות השמעה מועדפות*.
- Wink - *ראה `Advanced`*.

## מתקדם

WinkLinks הוא גם דרך מצוינת לחשוף את מלאי התיירות שמצאת, דרך [Wink Studio](https://studio.wink.travel), ואתה רוצה למכור לקהל שלך.

השתמש ב-WinkLinks כדי למכור את:

- [קישורים לשיתוף](/he/studio/shareable-links) - בחר להציג כ:
    - [באנר פרסומי](/he/developers/integrations/web-components/#content-loader).
    - קישור רגיל עם עדכוני מחירים בזמן אמת.
- [כרטיסים](/he/studio/cards) - כרטיסים להזמנה עם מחירים וזמינות בזמן אמת.
- [רשתות](/he/studio/grids) - רשת עם כרטיסים להזמנה.
- [מפות](/he/studio/maps) - מפה עם סימנים שנפתחים לכרטיסים להזמנה.

:::note
לניהול חשבון WinkLinks שלך, היכנס לחשבון השותף שלך ב-[Wink Studio](https://studio.wink.travel).
:::

:::tip
כדי לראות חשבון בפעולה, בדוק את חשבון WinkLinks של ביורן, שותף מייסד שלנו: [https://i.trvl.as/bjorn](https://i.trvl.as/bjorn).
:::
