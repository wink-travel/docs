משתמש Wink יכול לבחור לאפשר לך לנהל אחד מחשבונות הנכס, השותף או סוכן הנסיעות שלו. כאשר זה קורה, תקבל דואר אלקטרוני עם בקשת הזמנה.

import { Steps } from '@astrojs/starlight/components';

כדי לראות את ההזמנות הנוכחיות שלך ב-Wink, בצע את השלבים הבאים:

<Steps>
 1. היכנס ל-[my.wink.travel](https://my.wink.travel).
 2. לחץ על `Invites` בסרגל הצד השמאלי.
 3. תראה רשימה של כל ההזמנות הנוכחיות.
 4. חלק זה יציג את כל הבקשות שלך.
 5. בחר `Accept` או `Reject` את ההזמנה.
 6. אם קיבלת את ההזמנה, החשבון יופיע בקטגוריית `Accounts`.
</Steps>
