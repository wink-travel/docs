לפעמים, אנחנו רוצים ליצור איתך קשר. כשזה קורה, תראה שהפעמון בפינה הימנית העליונה משתנה.
כדי לקרוא את מה ששלחנו לך:

<Steps>
1. לחץ על `חשבון > התראות` בסרגל הניווט הראשי.
2. תראה התראות חדשות וישנות.
3. בחר ב-`סמן כנקרא` או `מחק` את ההתראות לאחר שקראת אותן.
</Steps>

## API

מפתחים שרוצים לנהל `שותפים` יכולים לגשת ל-[מפתחים > API > שותף](/he/developers/apis/#affiliate-api).
