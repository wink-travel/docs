לא כל יום מתומחר באותו אופן. אירועים, חגים, עונות שפל ופערים של הרגע האחרון דורשים תמחור שונה. מדריך זה מראה כיצד להשתמש בתעריפים מיוחדים ובהגבלות תוכניות תעריף כדי להתמודד עם תרחישים אלה.

## תמחור אירועים

כאשר כנס, פסטיבל או אירוע מקומי מגדילים את הביקוש, תרצו להעלות את התעריפים עבור אותם תאריכים.

<Steps>
1. עבור אל `Monetize > Special Rates` ב-Extranet שלך.
2. לחץ על `Create a new special rate`.
3. בחר את התעריף הראשי וסוג החדר שברצונך לעקוף.
4. הגדר את טווח התאריכים שיכסה את תקופת האירוע.
5. הזן את המחירים היומיים המתוקנים (גבוהים יותר).
6. לחץ על `Save`.
</Steps>

התעריף המיוחד עוקף את התעריף הראשי רק עבור אותם תאריכים. התמחור הרגיל שלך נשאר ללא שינוי מחוץ לחלון האירוע.

## התאמות עונתיות

לתקופות ארוכות יותר (עונת שיא, עונת מעבר, עונת שפל), ניתן:

- **להשתמש בתעריפים מיוחדים** — ליצור עקיפות לכל עונה עם תמחור מתוקן.
- **לעדכן את התעריפים הראשיים** — להתאים את התעריפים הבסיסיים ישירות עבור תקופות אלה.

תעריפים מיוחדים מתאימים יותר להתאמות זמניות, בעוד שעדכון התעריפים הראשיים הגיוני לדפוסים שנתיים צפויים.

## תעריפים של הרגע האחרון

כדי למלא חדרים לא נמכרים קרוב לתאריך הצ'ק-אין:

<Steps>
1. צור **תוכנית תעריף** חדשה עם הגבלת הזמנה מראש (למשל, חייב להזמין לא יותר מ-3 ימים מראש).
2. חבר אותה עם **תעריף מיוחד** שמציע הנחה עבור התאריכים המיועדים.
3. תוכנית תעריף זו תופיע רק למטיילים שמזמינים במסגרת חלון הרגע האחרון.
</Steps>

## תאריכי חסימה

כדי לחסום תאריכים מסוימים מתוכנית תעריף ספציפית (למשל, ללא תעריפים מבצעיים בעונת שיא):

<Steps>
1. עבור אל `Monetize > Rate Plans` ובחר את תוכנית התעריף.
2. עבור לקטגוריית **Restrictions**.
3. הוסף הגבלת **Sell date** כדי להגביל מתי ניתן להזמין את התעריף.
4. או השתמש בהגבלות **Stay date** כדי להגביל מתי התעריף תקף לשהייה.
</Steps>

לדוגמה, ניתן להגביל את תוכנית התעריף "Early Bird" כך שלא תהיה זמינה להזמנות במהלך דצמבר (עונת שיא), בעוד שתוכנית התעריף הרגילה שלך תישאר פתוחה.

## שילוב הגבלות

ניתן לשלב הגבלות לשליטה מדויקת:

| הגבלה | מה היא שולטת בו |
| -- | -- |
| **Sell dates** | מתי ניתן להזמין את התעריף |
| **Stay dates** | אילו תאריכי שהייה התעריף מכסה |
| **Available days** | באילו ימי שבוע התעריף מופיע |
| **Arrival days** | באילו ימים האורח חייב להגיע |
| **Departure days** | באילו ימים האורח חייב לעזוב |
| **Required days** | האורח חייב לשהות לפחות באחד מהימים האלה |
| **Advance booking** | כמה זמן מראש יש לבצע את ההזמנה |
| **Length of stay** | מינימום ומקסימום לילות |
