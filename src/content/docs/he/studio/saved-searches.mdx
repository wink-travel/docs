חיפושים שמורים עושים את מה ששמם מרמז. בזמן חיפוש אחר מלאי נסיעות, יש לך אפשרות לשמור את שאילתת החיפוש שלך לשימוש מאוחר יותר.

## שמירת שאילתת החיפוש שלך

כאשר אתם מתחילים לחפש מלאי (ראו[לְחַפֵּשׂ](/studio/search)), יש לך את היכולת לכוונן את מסנני החיפוש שלך. בתחתית טופס השאילתה יש את האפשרות`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="שמירת חיפוש" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*מציג קישור לשמירת שאילתת החיפוש*</figcaption>
</figure>


לחיצה על קישור זה תאפשר לך לתת שם לחיפוש זה (*לדוגמה, הספרייה שלי של מלונות קסומים*) משהו שתזכור לאחר כך.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="תן שם לחיפוש שלך" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*תן שם לחיפוש שלך*</figcaption>
</figure>


תן לזה שם ולחץ על`Save search`כפתור. לאחר מכן, תוכלו לנווט אל`Inventory > Saves Searches`מסרגל הניווט הראשי כדי לראות את הרשומה החדשה שיצרת.

## רשימת פעולות

ניתן להשתמש בחיפוש השמור שלך כדי לחשוף את מלאי הנסיעות למשתמשים שלך בצורת רשת או מפה ולגרום לתוצאות החיפוש שלך להופיע באופן הרצוי.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="פעולות חיפוש שנשמרו" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*שאילתת חיפוש שמורה ללא פילטרים*</figcaption>
</figure>


לכל שאילתת חיפוש שמורה יש פעולות שניתן לבצע עליה:

* המר את שאילתת החיפוש ל-[מַפָּה](/studio/maps).
* המר את שאילתת החיפוש ל-[רֶשֶׁת](/studio/grids).
* הסר את החיפוש השמור.

## ממשק API

מפתחים שרוצים לנהל חיפושים שמורים יכולים לעבור אל[מפתחים > API > עיון](/developers/apis/#browse-api).

## קריאה נוספת

* קראו על שלנו[מַפָּה](/studio/maps)תכונות.
* קראו על שלנו[רֶשֶׁת](/studio/grids)תכונות.
