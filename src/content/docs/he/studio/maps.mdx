מפה מייצגת ספק יחיד (למשל, מלון או ספק חוויות) או מלאי (למשל, סוג חדר, חדר ישיבות, ספא וכו') או אוסף של מלאי כגון רשימה אוצרה או חיפוש שמור, וחושפת פרטים אלה באמצעות מפה אינטראקטיבית המכילה סמני מפה הניתנים להזמנה.

שאר המאמר יסביר לכם כיצד ליצור, להתאים אישית ולשתף את המפות שלכם.

## צור מפה

ניתן ליצור מפות בשתי דרכים:

* **מפה עם סמן יחיד**צור מפה ממשהו שמצאת בזמן[חיפוש](/studio/search).
* **מפה מרובת סמנים**צור מפה מ-[רשימה אוצרת](/studio/curated-lists)או[חיפוש שמור](/studio/saved-searches).

### מפה עם סמן יחיד

<figure>
  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="פעולות" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*פעולות בתוצאות החיפוש*</figcaption>

למעלה תמונה שנלקחה מ[לְחַפֵּשׂ](/studio/search)ומראה לך כמה מהדברים שאתה יכול לעשות עם תוצאות החיפוש שלך. אחת מהפעולות הללו היא`Make a map`.
לחץ על כפתור זה והוא יפנה אותך לדף טופס המפה שלנו, שם תוכל להתחיל להתאים אישית את המפה שלך.

### מפה מרובת סמנים

שניהם[רשימות שנבחרו](/studio/curated-lists#list-actions)ו[חיפושים שמורים](/studio/saved-searches#list-actions)מגיע עם פעולה המאפשרת לך ליצור מפה. מפה זו תכיל סמני מפה עבור כל המלאי ברשימות אלה.

## התאמה אישית של המפה

הטופס מאפשר לך להתאים אישית את המפה שלך בדרכים הבאות:

<Steps>
  1. תן לזה שם כדי שתזכור על מה המפה.
  2. בחר אם ברצונך שהמשתמש יוכל לנוע במפה.
  3. הגדר את הגובה הרצוי, בפיקסלים, של המפה.
  4. בחר אם ברצונך שהמשתמש יוכל להגדיל ולהקטין את המפה.
  5. בחרו סגנון מפה שמתאים לאתר האינטרנט שבו תטמיעו את המפה.
  6. בחר צבע סמן.
  7. בחר את הצד הראשוני של הכרטיס שברצונך שהמשתמשים יראו תחילה. כברירת מחדל, זהו הצד המקורי של המלאי.
  8. באפשרותך להוסיף סמנים, עיגולים, מלבנים ופוליגונים משלך כדי לציין מיקומים אחרים במפה שעשויים לעניין את המשתמשים שלך.
  9. לחץ על ה `Save` כפתור כדי להמשיך.
</Steps>


לאחר שמירת המפה, תועברו לדף המפות שלכם והמפה שלכם מוכנה כעת לשיתוף עם העולם.

## שתף מפה

למעלה מוצגת תמונה עם כל הפעולות הזמינות עבור המפה שלך:

* **לְעַדְכֵּן**מעדכן את תצורת המפה שלך.
* **הוסף ל-WinkLinks**מוסיף את המפה לחשבון WinkLinks שלך.
* **לְשַׁבֵּץ**מראה לך כיצד להטמיע את המפה הזו כ-[מַפָּה](/developers/web-components/#content-loader)לתוך אתר האינטרנט שלך.
* **שימוש עם וורדפרס**מראה לכם כיצד להשתמש בשירותינו[תוסף וורדפרס](/developers/wordpress)כדי להטמיע את המפה הזו באתר האינטרנט שלך.

נכסה חלק מהאפשרויות הללו, ביתר פירוט, להלן.

### לְשַׁבֵּץ

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

כך תטמיעו את המפה שלכם באתר שלכם:

* שורה 3 מראה לך כיצד להטמיע את סגנונות ה-Wink באתר שלך.
* שורות 6 עד 9 מראות לכם כיצד להשתמש ב-[טוען תוכן-wink](/developers/web-components/#content-loader)רכיב אינטרנט והורה לו להביא באנר פרסומי עבור הקוד שלך.
* שורה 11 מראה לכם כיצד להטמיע את קוד ה-Javascript שלנו באתר שלכם.
* שורה 13 מראה לך כיצד להטמיע את[טוען אפליקציות wink](/developers/web-components/#application-loader)רכיב האינטרנט והורה לו לאחזר את העדפות התצורה שלך ברמת הדף.
