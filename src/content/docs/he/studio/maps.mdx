שאר המאמר מדריך אותך כיצד ליצור, להתאים אישית ולשתף את המפות שלך.

## צור מפה

ניתן ליצור מפות בשתי דרכים:

- **מפת סמן יחיד**: צור מפה ממשהו שמצאת בזמן [חיפוש](/he/studio/search).
- **מפת סמנים מרובה**: צור מפה מרשימה [נבחרת בקפידה](/he/studio/curated-lists) או [חיפוש שמור](/he/studio/saved-searches).

מעל מופיעה תמונה שנלקחה מ-[חיפוש](/he/studio/search) ומציגה כמה מהדברים שניתן לעשות עם תוצאות החיפוש שלך. אחת מהפעולות היא `צור מפה`.
לחץ על כפתור זה והוא יפנה אותך לדף טופס המפה שלנו שבו תוכל להתחיל להתאים אישית את המפה שלך.

### מפת סמנים מרובה

גם [רשימות נבחרות בקפידה](/he/studio/curated-lists#list-actions) וגם [חיפושים שמורים](/he/studio/saved-searches#list-actions) כוללים פעולה שמאפשרת לך ליצור מפה. מפה זו תכיל סמני מפה לכל המלאי בתוך הרשימות הללו.

## התאם אישית מפה

הטופס מאפשר לך להתאים אישית את המפה שלך בדרכים הבאות:

<Steps>
 1. תן לה שם כדי שתזכור על מה המפה.
 2. בחר אם ברצונך לאפשר למשתמש לנוע במפה.
 3. הגדר כמה גבוהה, בפיקסלים, תרצה שהמפה תהיה.
 4. בחר אם ברצונך לאפשר למשתמש להתקרב ולהתרחק מהמפה.
 5. בחר סגנון מפה שמתאים לאתר שבו תטמיע את המפה.
 6. בחר צבע לסמן.
 7. בחר את הפנים הראשוני של הכרטיס שברצונך שמשתמשים יראו תחילה. כברירת מחדל, זו הפנים הטבעית של אותו מלאי.
 8. תוכל להוסיף סמנים משלך, עיגולים, מלבנים ופוליגונים כדי לציין מיקומים אחרים במפה שעשויים לעניין את המשתמשים שלך.
 9. לחץ על כפתור `שמור` כדי להמשיך.
</Steps>

לאחר ששמרת את המפה שלך, תועבר לדף המפות שלך והמפה שלך מוכנה כעת לשיתוף עם העולם.

מעל מוצגת תמונה עם כל הפעולות הזמינות עבור המפה שלך:

- **עדכן** מעדכן את תצורת המפה שלך.
- **הוסף ל-WinkLinks** מוסיף את המפה לחשבון WinkLinks שלך.
- **הטמע** מראה לך כיצד להטמיע מפה זו כ-[מפה](/he/developers/web-components/#content-loader) באתר שלך.
- **שימוש עם WordPress** מראה לך כיצד להשתמש ב[תוסף WordPress](/he/developers/wordpress) שלנו כדי להטמיע מפה זו באתר שלך.

נכסה חלק מהאפשרויות הללו, בפירוט רב יותר, להלן.

### הטמע

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

כך תטמיע את המפה שלך באתר:

- שורה 3 מראה כיצד להטמיע את סגנונות Wink באתר שלך.
- שורות 6 עד 9 מראות כיצד להשתמש ברכיב ה-Web [wink-content-loader](/he/developers/web-components/#content-loader) ולהורות לו להביא באנר פרסומי עבור הקוד שלך.
- שורה 11 מראה כיצד להטמיע את קוד ה-Javascript שלנו באתר שלך.
- שורה 13 מראה כיצד להטמיע את רכיב ה-Web [wink-app-loader](/he/developers/web-components/#application-loader) ולהורות לו להביא את העדפות התצורה ברמת הדף שלך.
