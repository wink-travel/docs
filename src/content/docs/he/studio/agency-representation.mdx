סוכנות היא חשבון Wink רגיל שתוכל להקצות לחשבונך למטרות ייצוג. להלן מספר סיבות מדוע תרצו סוכנות שתייצג אתכם:

* ניהול מלא/חלקי של חשבון Wink שלך על ידי צד שלישי.
* ל-Talent יש הסכם קיים עם הסוכנות שלהם והיא רוצה ש-Wink תטפל בהיבט הפיננסי של סידור המימון שלהם דרך Wink.
* סוכנות כישרונות עושה את כל העבודה המקוונת ומנהלת את כל החשבונות ב-Wink.

## הקצאת סוכנות

כך תשימו סוכנות לחשבונכם.

<Steps>
  1. היכנס לחשבון המשתמש שלך ב [אולפן וינק](https://studio.wink.travel).
  2. בחר את חשבון השותפים שאיתו ברצונך לעבוד.
  3. לחץ על `Account > Managers` מסרגל הניווט הראשי.
  4. ב- `Agency` במקטע, הקלד את שם הסוכנות שברצונך שייצג אותך.
  5. בחר את עמלת הסוכנות מהתפריט הנפתח.
  6. ניתן להוסיף כללי תפוגה באופן אופציונלי.
  7. לחץ על ה `Set agency` כפתור כדי להמשיך.
</Steps>


:::הערה
עליך להיות בעל החשבון כדי להקצות סוכנות.
:::

:::tip\[דוגמה]
דמי האינטנסיביות נובעים מעמלת השותפים.

הזמנה = $100, עמלה = 10%, עמלת סוכנות = 10% => 100 \* 0.1 \* 0.1 = $1
:::

## ביטול הקצאת סוכנות

כך תבטלו הקצאה של סוכנות מחשבונכם.

<Steps>
  1. היכנס לחשבון המשתמש שלך ב [אולפן וינק](https://studio.wink.travel).
  2. בחר את חשבון השותפים שאיתו ברצונך לעבוד.
  3. לחץ על `Account > Managers` מסרגל הניווט הראשי.
  4. ב- `Agency` סעיף, לחץ על <Badge text="x" variant="danger" size="small" /> סמל בצד ימין של תפריט הנפתח של הסוכנות.
  5. לחץ על ה `Set agency` כפתור כדי להמשיך.
</Steps>


## ממשק API

מפתחים שרוצים לנהל סוכנות דרך API יכולים לגשת אל[מפתחים > API > שותפים](/developers/apis/#affiliate-api).
