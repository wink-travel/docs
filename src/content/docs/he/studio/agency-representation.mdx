סוכנות היא חשבון Wink רגיל שניתן להקצות לחשבונך למטרות ייצוג. להלן כמה סיבות לכך שתרצה שסוכנות תייצג אותך:

- ניהול מלא / חלקי של חשבון ה-Wink שלך על ידי צד שלישי.
- לכישרון יש הסכם קיים עם הסוכנות שלו והוא רוצה ש-Wink תטפל בהיבט הכספי של הסידור שלהם של כספים הנכנסים דרך Wink.
- סוכנות הכישרונות מבצעת את כל העבודה המקוונת ומנהלת את כל החשבונות ב-Wink.

## הקצאת סוכנות

כך מקצים סוכנות לחשבונך.

<Steps>
 1. היכנס לחשבון המשתמש שלך ב-[Wink Studio](https://studio.wink.travel).
 2. בחר את חשבון השותף שברצונך לעבוד איתו.
 3. לחץ על `Account > Managers` בסרגל הניווט הראשי.
 4. בקטגוריית `Agency`, הקלד את שם הסוכנות שברצונך שתייצג אותך.
 5. בחר את דמי הסוכנות מהרשימה הנפתחת.
 5. במידת הצורך, הוסף כללי תפוגה.
 7. לחץ על כפתור `Set agency` כדי להמשיך.
 </Steps>

 :::note
עליך להיות הבעלים של החשבון כדי להקצות סוכנות.
 :::

 :::tip[דוגמה]
דמי הסוכנות נגזרים מעמלת השותף.

הזמנה = $100, עמלה = 10%, דמי סוכנות = 10% => 100 * 0.1 * 0.1 = $1
 :::

## הסרת הקצאת סוכנות

כך מסירים הקצאת סוכנות מחשבונך.

<Steps>
 1. היכנס לחשבון המשתמש שלך ב-[Wink Studio](https://studio.wink.travel).
 2. בחר את חשבון השותף שברצונך לעבוד איתו.
 3. לחץ על `Account > Managers` בסרגל הניווט הראשי.
 4. בקטגוריית `Agency`, לחץ על סמל <Badge text="x" variant="danger" size="small" /> בצד ימין של הרשימה הנפתחת של הסוכנות.
 5. לחץ על כפתור `Set agency` כדי להמשיך.
 </Steps>


## API

מפתחים שרוצים לנהל סוכנות דרך API יכולים לגשת ל-[Developers > API > Affiliate](/he/developers/apis/#affiliate-api).
