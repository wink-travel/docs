Studio מספק מספר ווידג'טים להטמעה שניתן להוסיף לאתר שלך כדי להגדיל הזמנות.

כדי לגשת לווידג'טים להטמעה, עבור אל `Tools > Miscellaneous` מהסרגל הצדדי.

## ווידג'טים זמינים

| ווידג'ט | תיאור |
| -- | -- |
| **כפתור חשבון** | כפתור ממותג שמקשר לעמוד השותף שלך |
| **כפתור מסלול** | כפתור שפותח את בוחר המסלול |
| **טופס מסלול** | טופס פנימי לבחירת תאריכים ואורחים |
| **כפתור חיפוש** | כפתור שפותח חיפוש נכסים |
| **כפתור חיפוש מלאי** | כפתור לחיפוש מלאי ספציפי |

## איך להטמיע

כל ווידג'ט מספק קטע קוד שניתן להעתיק ולהדביק ב-HTML של האתר שלך. הקטע כולל את **Client ID** ו-**Customization ID** שלך כך שהווידג'ט מקושר אוטומטית לחשבון השותף שלך.

:::note
אם אתה משתמש ב-WordPress, אנו גם מספקים תוסף WordPress לשילוב קל יותר.
:::
