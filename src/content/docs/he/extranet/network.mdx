מאמר זה מראה לכם כיצד ליצור קשר עם שותפים נלהבים ב-Wink ולנהל שותפים קיימים.

## ערוצי מכירות

כדי לנהל את ערוצי המכירות שלך, לחץ על`Distribution > Sales channels`מסרגל הניווט הראשי.

ערוצי מכירות הם קשרי השותפים הקיימים שלך, המקודדים עם המלאי, המבצעים וההנחות שיש להם גישה אליהם.

כדי לעדכן ערוץ מכירות קיים, לחץ על`Actions`קישור מתחת לערוץ המכירות שברצונך לעדכן.

* **הנחת חברים**תעריף החברים ניתן למטיילים המזמינים דרך ערוץ מכירות זה.
* **עֲמָלָה**העמלה מוקצית לערוץ המכירות עבור כל הזמנה מוצלחת.
* **מבצעים בודדים**הוסף את הקידומים שברצונך שיהיו בערוץ המכירות הזה.
* **מבצעים מקובצים**הוסף את מבצעי המכירות המשולבים שברצונך שיכללו בערוץ המכירות הזה.
* **מופעל**הפעל/הפעל את מתג ההפעלה כדי להפעיל/להשבית את ערוץ המכירות לחלוטין.
* **ברשימה השחורה**סימון ערוץ מכירות כרשימה שחורה מקשה על ביצוע הזמנה דרך ערוץ מכירות זה.

לאחר שתהיו מרוצים מההגדרות שלכם, לחצו על הלחצן`Save`כפתור כדי להמשיך.

:::tip
על ידי הוספת ערוץ מכירות לרשימה שחורה, אתם מונעים מערוץ המכירות לקבל מכם תעריפים כלשהם; אפילו לא את תעריפי Wink Network שלכם.
:::

## חקור את הרשת

כדי למצוא שותפים חדשים ב-Wink, לחצו על`Distribution > Explore network`מסרגל הניווט הראשי.

בחלק זה, תוכלו לראות בקשות ממתינות משותפים בחלק העליון של הדף ורשימה של שותפים להתחבר אליהם למטה.

### תגובה לבקשה

באפשרותך לבחור לדחות את בקשת השותפים. אם ברצונך להמשיך בבקשה, תועבר אל`affiliate details`דף שבו תוכלו לקבל מידע נוסף על השותף וכן מדדי ביצועים קודמים ב-Wink.

### חיפוש שותפים

בטבלת השותפים, תוכלו לחפש שותפים לפי שם, סוג ומיקום. כשאתם רואים אחד שמוצא חן בעיניכם, לחצו עליו והוא יוביל אתכם ל...`affiliate details`עמוד.

### התחבר עם שותף

אם תבחרו ליצור קשר ישיר עם שותף, תתבקשו לקודד את הקשר באותו אופן שבו הייתם מנהלים ערוץ מכירות לעיל.

לאחר שתשמור את ערוץ המכירות החדש, תוחזר לרשימת ערוצי המכירות שלך.

## ממשק API

מפתחים שרוצים לנהל את שלהם`Network`יכול לעבור ל[מפתחים > ממשקי API > הפצה](/developers/apis/#distribution-api).
