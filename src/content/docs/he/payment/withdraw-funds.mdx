:::note
אלא אם כן אתה אינטגרטור, אין צורך שתדאג ליצירת חשבון ב-TripPay. האינטגרטור ייצור עבורך את החשבון בזמן יצירת החשבון איתו.
:::

מקבלי תשלום משתמשים ב-TripPay כדי:

- לעקוב כמה כסף הרוויחו.
- כמה כסף נמצא כרגע בחשבונם.
- ...ואיזה סכום זמין למשיכה.

## שימוש

מקבל תשלום רגיל בדרך כלל מתקשר עם TripPay כך:

<Steps>
1. TripPay מחזיק כספים מ-10 הזמנות, מ-3 פלטפורמות נסיעות שונות, עבור מלון X והכספים כעת זמינים.
2. מלון X מקבל מייל מ-TripPay שאומר `יש לך כספים זמינים` וכולל קישור לניהול החשבון.
3. מלון X נכנס ל-[TripPay](https://pay.wink.travel).
4. מלון X רואה את הזמנות החודש הקודם יחד עם ניתוחי חשבון.
5. מלון X בוחר למשוך כספים.
6. מלון X בוחר את שיטת ה-`Pay-out`.
7. מלון X עוקב אחר השלבים הנדרשים לעיבוד הכספים.
8. מלון X מגיש בקשת העברה ולוחץ על `OK`.
</Steps>

*הכספים בדרך כלל זמינים בתוך 24 שעות בכל האזורים.*

:::note
כאשר תגיע לנפח וסכום מסוימים, ייתכן וניצור איתך קשר לקבלת מסמכים נוספים המאמתים את זהותך ו/או את זהות החברה שלך.
:::
