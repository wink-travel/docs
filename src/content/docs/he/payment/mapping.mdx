אם אתה אינטגרטור, כמו Wink, ורוצה למפות את מקבלי התשלום החיצוניים שלך לשימוש עם TripPay, תוכל לעשות זאת באחת משתי דרכים:

- מיפוי ידני
- שימוש ב-[Payment API](/he/developers/apis/#payment-api) (*ראה `Account Mappings`*)

:::tip
אלא אם כן אתה מתמודד רק עם מספר קטן של אותם נכסים, אנו ממליצים להשתמש ב-API כדי לאוטומט את השלב הזה.
:::

להלן, נדריך אותך בדרך הידנית למיפוי מקבל תשלום חיצוני. *היא משתמשת ב-API מתחת לפני השטח.*

<Steps>
1. היכנס ל-[TripPay](https://pay.wink.travel).
2. בחר את חשבון האינטגרטור שלך.
3. לחץ על `Settings > Mappings` בסרגל הניווט הראשי.
4. לחץ על כפתור `Create mapping`. תראה את טופס המיפוי.
5. **Local account** התחל להקליד את שם חשבון TripPay הקיים שברצונך למפות.
6. **External identifier** הדבק את מזהה החיצוני שלך עבור ישות זו כפי שהוא קיים במערכת המרוחקת.
7. לחץ על `Save` כדי להמשיך. תועבר חזרה לרשימת המיפויים שלך.
</Steps>

עם המיפויים שלך במקום, TripPay כעת יודע אילו חשבונות לזכות כמוטבים של הזמנה כאשר אתה מתייחס אליהם באמצעות המזהים החיצוניים שלך.

## API

מפתחים שרוצים לנהל `Payments`, ילכו ל-[Developers > API > Payments](/he/developers/apis/#payment-api).
