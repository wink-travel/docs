אם אתם אינטגרטורים, כמו Wink, ורוצים למפות את מקבלי התשלום החיצוניים שלכם לשימוש עם TripPay, תוכלו לעשות זאת באחת משתי דרכים:

* מיפוי ידני
* שימוש ב- [ממשק API לתשלום](/developers/apis/#payment-api) (*ראה`Account Mappings`*)

:::tip
אלא אם כן אתם מתמודדים עם קומץ מאותם מאפיינים, אנו ממליצים להשתמש ב-API כדי להפוך שלב זה לאוטומטי.
:::

להלן, נסקור את הדרך הידנית למיפוי מקבל תשלום חיצוני.*הוא משתמש ב-API מתחת למכסה המנוע.*

<Steps>
  1. התחבר [טריפפיי](https://pay.wink.travel).
  2. בחר את חשבון האינטגרטור שלך.
  3. נְקִישָׁה `Settings > Mappings` מסרגל הניווט הראשי.
  4. לחץ על ה `Create mapping` כפתור. תראה את טופס המיפוי.
  5. **חשבון מקומי** התחל להקליד את שם ה-TripPay הקיים שברצונך למפות.
  6. **מזהה חיצוני** הדבק את המזהה החיצוני שלך עבור ישות זו כפי שהיא קיימת במערכת המרוחקת.
  7. נְקִישָׁה `Save` כדי להמשיך. אתה מועבר חזרה לרשימת המיפויים שלך.
</Steps>


לאחר שהמיפויים שלכם קיימים, TripPay יודעת כעת אילו חשבונות לזכות כמוטבים של הזמנה כשאתם מתייחסים אליהם באמצעות המזהים החיצוניים שלכם.

## ממשק API

מפתחים שרוצים לנהל`Payments`, לך אל[מפתחים > API > תשלומים](/developers/apis/#payment-api).
