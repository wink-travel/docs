אם אתה חברת תיירות ואתה רוצה להאציל את נושא התשלום וההעברה לעסק שלך ל-TripPay, הנה 6 שלבים בסיסיים להתחלה:

<Steps>
1. צור [חשבון משתמש](/he/user/register) ב-Wink.
2. צור חשבון ב-[TripPay](https://pay.wink.travel).
3. צור [אפליקציה](/he/developers/applications) ב-TripPay שתאפשר לך לתקשר עם ה-API שלנו.
4. צור ומפה את המוטבים שלך עם TripPay. *לדוגמה, עבור רוב המשתמשים, זה אומר ליצור חשבונות לנכסים בפלטפורמה שלך שברצונך ש-TripPay ישלם עבורם.*
5. שלב את [רכיב התשלום לאינטרנט](/he/developers/web-components/#payment) באתר שלך.
6. שלח ל-TripPay את `עגלת הקניות` של הנוסע והפעל את תהליך התשלום.
</Steps>

:::tip
לקבלת מדריך מלא לשימוש ב-TripPay עבור אתר התיירות שלך, קרא את המדריך [אינטגרציה עם TripPay](/he/guides/integrating-with-trip-pay).
:::

## API

מפתחים שרוצים לנהל `תשלומים`, פנו ל-[מפתחים > API > תשלומים](/he/developers/apis/#payment-api).
