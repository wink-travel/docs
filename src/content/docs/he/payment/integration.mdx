אם אתם חברת תיירות ואתם רוצים להאציל את ניהול התשלומים והתשלומים החוזרים לעסק שלכם ל-TripPay, הנה 6 שלבים בסיסיים להתחלה:

<Steps>
1. צרו [חשבון משתמש](/he/user/register) ב-Wink.
2. צרו חשבון ב-[TripPay](https://pay.wink.travel).
3. צרו [אפליקציה](/he/developers/applications) ב-TripPay שתאפשר לכם לתקשר עם ה-API שלנו.
4. צרו ומיפו את המוטבים שלכם עם TripPay. *לדוגמה, עבור רוב המשתמשים, זה אומר ליצור חשבונות לנכסים בפלטפורמה שלכם שברצונכם ש-TripPay ישלם עבורם.*
5. שלבו את [רכיב התשלום לאינטרנט](/he/integrations/web-components/#payment) באתר שלכם.
6. שלחו ל-TripPay את `עגלת הקניות` של הנוסע והתחילו את תהליך התשלום.
</Steps>

:::tip
לקבלת מדריך מלא לשימוש ב-TripPay באתר התיירות שלכם, קראו את המדריך [אינטגרציה עם TripPay](/he/guides/integrating-with-trip-pay).
:::

## API

מפתחים שרוצים לנהל `תשלומים`, ילכו ל-[מפתחים > API > תשלומים](/he/developers/apis/#payment-api).
