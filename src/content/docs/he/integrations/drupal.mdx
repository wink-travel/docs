ה**מודול Wink לדרופל** מוסיף רכיבי מלאי נסיעות של Wink לאתר הדרופל שלך כבלוקים מקומיים. עורכים יכולים למקם אותם בכל מקום בפריסה דרך מערכת הבלוקים הסטנדרטית.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="קוד מקור, מדריך התקנה למפתחים והוראות לתרומה."
/>

## דרישות

- דרופל 10 או 11
- PHP 8.1+
- Composer

## התקנה

<Steps>
1. התקן דרך Composer:
   ```bash
   composer require drupal/wink
   ```
2. הפעל את המודול:
   ```bash
   drush en wink -y
   ```
3. עבור אל **מנהל → הגדרות → שירותי רשת → הגדרות Wink** (`/admin/config/wink`).
4. הזן את **Client ID** ו-**Client Secret** שלך, בחר את הסביבה שלך ולחץ על **שמור**.
   עבור אל [Applications](/he/portal/settings/applications) כדי לקבל את האישורים שלך.
</Steps>

## הוסף בלוק לעמוד

<Steps>
1. עבור אל **מנהל → מבנה → פריסת בלוקים**.
2. לחץ על **הוסף בלוק** באזור שבו תרצה תוכן Wink.
3. חפש **Wink** כדי למצוא את הבלוקים הזמינים:
   - **Wink Content** — מציג כרטיס מלון, רשת או מפה
   - **Wink Lookup** — שורת חיפוש ליעדים
   - **Wink Search** — כפתור לבחירת מסלול
   - **Wink Account** — כפתור התחברות
   - **Wink Itinerary** — כפתור סטטוס מסלול
   - **Wink Cart** — כפתור עגלת קניות
4. עבור **Wink Content**, הגדר את **סוג הפריסה** ו-**מזהה הפריסה** בהגדרות הבלוק.
5. שמור את הגדרות הבלוק.
</Steps>

## בלוקים זמינים

| בלוק | מטרה |
|---|---|
| **Wink Content** | מציג כרטיסי מלון, רשתות מדורגות, מפות ועוד |
| **Wink Lookup** | שורת חיפוש ליעדים ומלונות |
| **Wink Search** | כפתור אייקון שפותח את בורר המסלול |
| **Wink Account** | כפתור התחברות / חשבון משתמש |
| **Wink Itinerary** | כפתור מסלול המציג את הטיול הנוכחי |
| **Wink Cart** | כפתור סיכום עגלת קניות |
