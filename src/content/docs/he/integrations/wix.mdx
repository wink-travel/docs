האפליקציה **Wink Wix App** מוסיפה רכיבי רשת של Wink לאתר ה-Wix שלך כווידג'טים מקומיים שניתן לגרור ולשחרר בעורך ה-Wix. אין צורך בקידוד לבעלי האתר.

<LinkCard
  title="wink-travel/wink-wix-app on GitHub"
  href="https://github.com/wink-travel/wink-wix-app"
  description="קוד מקור, מדריך התקנה למפתחים והוראות לתרומה."
/>

## התקנה

<Steps>
1. עבור ל-[Wix App Market](https://www.wix.com/app-market) וחפש את **Wink Travel**.
2. לחץ על **Add to Site** ופעל לפי ההנחיות לאישור האפליקציה.
3. לאחר ההתקנה, פתח את **Dashboard → Wink Settings** והזן את **Client ID** ו-**Client Secret** שלך.
   עבור ל-[Applications](/he/portal/settings/applications) כדי לקבל את האישורים שלך.
4. לחץ על **Save** ואז על **Test Connection** כדי לאשר ש-Wink מחובר.
</Steps>

## הוספת רכיב לעמוד

<Steps>
1. פתח את **Wix Editor** ועבור לעמוד שבו ברצונך להוסיף תוכן של Wink.
2. לחץ על **Add (+) → Apps → Wink Travel** כדי לראות את כל הווידג'טים הזמינים.
3. גרור ווידג'ט לעמוד — לדוגמה, **Wink Content** להצגת רשת מלונות.
4. בפאנל הגדרות הווידג'ט, בחר את **Layout Type** והדבק את **Layout ID** מ-Wink Studio.
5. לחץ על **Publish** כדי להפוך את השינויים לזמינים.
</Steps>

## ווידג'טים זמינים

| ווידג'ט | מטרה |
|---|---|
| **Wink Content** | מציג כרטיסי מלונות, רשתות מדורגות, מפות ועוד |
| **Wink Lookup** | שורת חיפוש ליעדים ומלונות |
| **Wink Search** | כפתור אייקון שפותח את בורר המסלול |
| **Wink Account** | כפתור התחברות / חשבון משתמש |
| **Wink Itinerary** | כפתור מסלול המציג את הטיול הנוכחי |
| **Wink Cart** | כפתור סיכום עגלת קניות |
