ה**תוסף Wink לג'ומלה** הוא חבילה שמתקינה פלאגין מערכת (טוען סקריפטים מ-CDN), פלאגין תוכן (מעבד תגי קוד קצר במאמרים), ורכיב (דף הגדרות מנהל).

/>

## דרישות

- ג'ומלה 4 או 5
- PHP 8.1+

## התקנה

<Steps>
1. הורד את הקובץ העדכני `pkg_wink-x.x.x.zip` מ[דף ההוצאות ב-GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. בממשק הניהול של ג'ומלה, עבור ל- **Extensions → Manage → Install**.
3. העלה את קובץ ה-zip ולחץ על **Upload & Install**.
4. שני הפלאגינים מופעלים אוטומטית. עבור ל- **Components → Wink Settings**.
5. הזן את **Client ID** ו-**Client Secret** שלך ולחץ על **Save**.
   עבור ל-[Applications](/he/portal/settings/applications) כדי לקבל את האישורים שלך.
</Steps>

## הטמעת רכיבים במאמרים

השתמש בתגי קוד קצר בכל מקום בתוכן המאמר:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

פלאגין התוכן מחליף תגיות אלו ב-HTML הנכון של `<wink-*>` בזמן הרינדור. פלאגין המערכת טוען אוטומטית את סקריפטי ה-CDN ומזריק את `<wink-app-loader>` לתחתית העמוד.

## קודי קיצור זמינים

| קוד קצר | פלט | תיאור |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | כרטיסי מלונות, רשתות, מפות |
| `{wink-lookup}` | `<wink-lookup>` | שורת חיפוש יעד |
| `{wink-search}` | `<wink-search-button>` | כפתור בחירת מסלול |
| `{wink-account}` | `<wink-account-button>` | כניסה / חשבון משתמש |
| `{wink-itinerary}` | `<wink-itinerary-button>` | כפתור סטטוס מסלול |
| `{wink-cart}` | `<wink-shopping-cart-button>` | כפתור עגלת קניות |
