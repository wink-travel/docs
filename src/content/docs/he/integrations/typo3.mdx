ה**תוסף Wink TYPO3** משלב את מלאי הנסיעות של Wink ב-TYPO3 13 כאלמנטים תוכן מקומיים. העורכים מוסיפים רכיבי Wink לעמודים באמצעות עורך העמודים הסטנדרטי של TYPO3 — ללא צורך בשינויים בתבנית.

## דרישות

- TYPO3 13.x
- PHP 8.2+
- Composer

## התקנה

<Steps>
1. התקינו באמצעות Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. הפעלת התוסף:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. ב-TYPO3 backend, עברו ל-**Admin Tools → Settings → Extension Configuration → wink**.
4. הזינו את **Client ID**, **Client Secret**, ובחרו את הסביבה שלכם. שמרו.
   עברו ל-[Applications](/he/portal/settings/applications) כדי לקבל את האישורים שלכם.
5. הוסיפו את תבנית TypoScript הסטטית לתבנית השורש שלכם:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## הוספת אלמנט תוכן לעמוד

<Steps>
1. פתחו עמוד במודול **Page**.
2. לחצו על כפתור ה-**+** באזור התוכן להוספת אלמנט חדש.
3. בחרו באחד מאלמנטים התוכן של **Wink Travel**:
   - **Wink Content** — כרטיסי מלונות, רשתות מדורגות, מפות
   - **Wink Lookup** — שורת חיפוש יעד
   - **Wink Search** — כפתור בחירת מסלול
   - **Wink Account** — כפתור התחברות
   - **Wink Itinerary** — כפתור סטטוס מסלול
   - **Wink Cart** — כפתור עגלת קניות
4. עבור **Wink Content**, השתמשו בשדות FlexForm לבחירת **Layout Type** והזנת **Layout ID**.
5. שמרו ופרסמו את העמוד.
</Steps>

## אלמנטים זמינים

| אלמנט | מטרה |
|---|---|
| **Wink Content** | כרטיסי מלונות, רשתות מדורגות, מפות ועוד |
| **Wink Lookup** | שורת חיפוש יעד |
| **Wink Search** | כפתור אייקון לפתיחת בחירת המסלול |
| **Wink Account** | כפתור התחברות / חשבון משתמש |
| **Wink Itinerary** | כפתור מסלול המציג את הטיול הנוכחי |
| **Wink Cart** | כפתור סיכום עגלת קניות |
