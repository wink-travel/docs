תכונת **הזרקת קוד** המובנית של Ghost מאפשרת לך להוסיף סקריפטים ל-`<head>` ו-`</body>` הגלובליים של הפרסום שלך. כל תוכניות Ghost תומכות בהזרקת קוד, מה שמקל על הטמעת רכיבי Wink בכל דף או פוסט.

## טעינת סקריפטים של Wink בכל האתר

<Steps>
1. בלוח הניהול של Ghost, עבור אל **Settings → Code Injection**.
2. בתיבת **Site Header**, הוסף את גיליון הסגנונות:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. בתיבת **Site Footer**, הוסף את הסקריפט וטוען האפליקציה (החלף את `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. לחץ על **Save**.
</Steps>

:::tip
מזהה הלקוח שלך זמין בחשבון Wink שלך תחת [Applications](/he/portal/settings/applications).
:::

## הוספת רכיב לפוסט או לדף

עורך Ghost תומך ב**כרטיסיית HTML** שמאפשרת להכניס HTML גולמי בכל מקום בתוכן שלך.

<Steps>
1. פתח את הפוסט או הדף בעורך Ghost.
2. הקלד `/` כדי לפתוח את בורר הכרטיסיות ובחר **HTML**.
3. הדבק את ה-HTML של הרכיב, לדוגמה:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. לחץ מחוץ לכרטיסייה לתצוגה מקדימה, ואז פרסם.
</Steps>

## הזרקת קוד לפי דף

לרכיב שצריך להופיע רק בדף אחד, Ghost תומך גם בהזרקת קוד לפי דף:

<Steps>
1. פתח את הגדרות הפוסט או הדף (סמל גלגל השיניים בעורך).
2. גלול אל **Code Injection**.
3. הוסף את תג הרכיב בשדה **Footer** (טוען האפליקציה הגלובלי צריך להיות רק בכותרת תחתונה של האתר).
4. שמור.
</Steps>

## רכיבים זמינים

| רכיב | תג HTML | מטרה |
|---|---|---|
| טוען תוכן | `<wink-content-loader>` | הצגת כרטיסי מלון, רשתות, מפות |
| חיפוש | `<wink-lookup>` | שורת חיפוש ליעדים |
| כפתור חיפוש | `<wink-search-button>` | פתיחת בורר מסלול |
| כפתור חשבון | `<wink-account-button>` | כניסה / תפריט משתמש |
| כפתור מסלול | `<wink-itinerary-button>` | הצגת המסלול הנוכחי |
| בורר מסלול | `<wink-itinerary-picker>` | טופס מסלול מלא |
| עגלת קניות | `<wink-shopping-cart-button>` | כפתור סיכום עגלה |

ראה [Web Components](/he/integrations/web-components) למידע מלא על התכונות.
