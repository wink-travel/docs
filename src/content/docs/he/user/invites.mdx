משתמש Wink יכול לבחור לאפשר לך לנהל אחד מחשבונות הנכס, השותפים או סוכנות הנסיעות שלו. כאשר זה יקרה, תקבל דוא"ל עם בקשת הזמנה.

import { Steps } from '@astrojs/starlight/components';

כדי לראות את ההזמנות הנוכחיות שלך ב-Wink, בצע את השלבים הבאים:

<Steps>
  1. עליך להיות מחובר לאחד מאתרי האינטרנט שלנו.
  2. לחצו על סמל הפרופיל שלכם, בפינה הימנית העליונה, ויופיע תפריט.
  3. לחץ על ה `Invites` לְקַשֵׁר.
  4. אינטראקציה עם ההזמנות שלך על ידי אחד מהם `Accepting` אוֹ `Rejecting` אוֹתָם.
</Steps>
