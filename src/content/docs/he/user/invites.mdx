משתמש Wink יכול לבחור לאפשר לך לנהל אחד מחשבונות הנכס, השותף או סוכן הנסיעות שלו. כאשר זה קורה, תקבל דוא"ל עם בקשת הזמנה.

import { Steps } from '@astrojs/starlight/components';

כדי לראות את ההזמנות הנוכחיות שלך ב-Wink, בצע את השלבים הבאים:

<Steps>
1. עליך להיות מחובר לאחד מהאתרים שלנו.
2. לחץ על סמל הפרופיל שלך, בפינה הימנית העליונה, ותופיע תפריט.
3. לחץ על הקישור `Invites`.
4. פעל עם ההזמנות שלך על ידי `קבלה` או `דחייה` שלהן.
</Steps>
