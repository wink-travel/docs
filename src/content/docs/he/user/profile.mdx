import { Steps } from '@astrojs/starlight/components';

אם ברצונך לעדכן את פרופיל המשתמש שלך עבור חשבון המשתמש שלך ב-Wink, עקוב אחר השלבים הבאים:

<Steps>
1. עליך להיות מחובר לאחד מהאתרים שלנו.
2. לחץ על סמל הפרופיל שלך, בפינה הימנית העליונה, ותופיע תפריט.
3. לחץ על הקישור `Profile`.
4. עקוב אחר ההוראות לעדכון ה-`e-mail`, ה-`first` וה-`last name`.
5. תוכל גם להעלות תמונת `profile picture` חדשה שלך.
6. לחץ על `Save`.
</Steps>

:::note
תמונת הפרופיל החדשה שלך תוצג בפעם הבאה שתתנתק לחלוטין מ-Wink.
:::
