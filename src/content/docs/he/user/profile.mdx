אם ברצונך לעדכן את פרופיל המשתמש שלך עבור חשבון המשתמש Wink, בצע את השלבים הבאים:

<Steps>
  1. עליך להיות מחובר לאחד מאתרי האינטרנט שלנו.
  2. לחצו על סמל הפרופיל שלכם, בפינה הימנית העליונה, ויופיע תפריט.
  3. לחץ על ה `Profile` לְקַשֵׁר.
  4. פעל לפי ההוראות כדי לעדכן את `e-mail`, `first` ו `last name`.
  5. ניתן גם להעלות קובץ חדש `profile picture` של עצמך.
  6. נְקִישָׁה `Save`.
</Steps>


:::הערה
תמונת הפרופיל החדשה שלך תוצג בפעם הבאה שתתנתק לחלוטין מ-Wink.
:::
