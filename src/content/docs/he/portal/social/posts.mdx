עורך הפוסטים מאפשר לך ליצור פוסטים ידניים ללא הגבלה והוא גם המקום שבו תוכל לנהל פוסטים שנוצרו עבורך אוטומטית.

העורך מחולק ל-4 חלקים:

1. חלק התוכן.
2. חלק המדיה.
3. חלק הקישור.
4. חלק ההגדרות.

## תוכן

חלק התוכן מאפשר לך לשנות את תוכן הטקסט לפני הפרסום. ניתן ליצור תוכן מותאם לשפות שונות — לכל שפה יש כותרת ותיאור משלה.

## מדיה

חלק המדיה מאפשר לך לשנות את התמונות והסרטונים המלווים את הטקסט.

## קישור

חלק הקישור מאפשר לך לקשר למקום שבו תרצה שהמשתמש יגיע לאחר קריאת הפוסט שלך. *לדוגמה, זה יכול להיות קישור למבצע מיוחד או למקום למידע נוסף.* ניתן להוסיף כתובת URL לקישור, כיתוב קישור ותיאור קישור.

## הגדרות

חלק ההגדרות מאפשר לך לשנות / לעקוף את כל ההגדרות המוגדרות כברירת מחדל עבור כל פוסט. ניתן להגדיר:

- **פלטפורמה** — בחר לאיזו רשת חברתית יפורסם הפוסט.
- **סוג הפוסט** — בחר בין טרנזקציוני, מודעות או סנטימנט.
- **טון הדיבור** — עקוף את טון הדיבור עבור הפוסט הספציפי הזה.
- **התאמה אישית** — השתמש בהתאמה אישית שונה למטבע תצוגה, שפה, מסלול טיול וכו'.
- **אזור זמן** — עקוף את אזור הזמן לפרסום.
- **תאריך מתוזמן** — פוסטים שנוצרים אוטומטית מתוזמנים אוטומטית בזמן האופטימלי. ניתן לעקוף את התאריך והשעה אם תרצה חלון זמן שונה.

## מחזור חיי סטטוס הפוסט

כל פוסט עובר מחזור חיי סטטוס:

| סטטוס | תיאור |
| -- | -- |
| **טיוטה** | הפוסט נוצר אך עדיין לא מוכן לסקירה |
| **ממתין לאישור** | הפוסט ממתין לאישור אנושי לפני התזמון |
| **מאושר** | הפוסט אושר ויתוזמן לפרסום |
| **נדחה** | הפוסט נבדק ונדחה — לא יתפרסם |
| **מתוזמן** | הפוסט בתור לפרסום בתאריך ובשעה המתוזמנים |
| **פורסם** | הפוסט פורסם בהצלחה ברשת החברתית |
| **מושהה** | פרסום הפוסט הושהה זמנית |
| **נכשל** | הפוסט לא הצליח להתפרסם עקב שגיאה |

## אישור פוסטים

פוסטים מאושרים ברשת הפוסטים שבה ניתן לבחור פוסט אחד או יותר ולבחור לאשר או לדחות אותם.

:::note
יהיה ערך אחד עבור כל שפה וכל רשת חברתית.

לדוגמה, אם יש לך 3 רשתות חברתיות ו-3 שפות נבחרות, תראה 9 ערכים בלוח השנה.
:::
