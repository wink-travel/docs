# הגדרות

כשאתה מתחיל להשתמש ב-Social, נשאל אותך כיצד תרצה שניצור את התוכן שלך. אפשרויות אלו זמינות עבורך בכל עת כדי שתוכל לשנות אותן בקלות מאוחר יותר.

נווט אל `Preferences` מהסרגל הצדדי כדי לנהל את ההגדרות שלך.

מדור ההעדפות מכיל את הגדרות יצירת התוכן הבסיסיות שלך:

1. **התאמה מועדפת** בחר איזו התאמה להשתמש בעת יצירת תוכן. התאמות שולטות בצבעי הנושא, מסלול ברירת המחדל, מטבע, שפה ועוד. אתה יוצר התאמות תחת החשבון שלך ב-[my.wink.travel](https://my.wink.travel).
2. **אישור אוטומטי** כאשר מופעל, הפוסטים שנוצרים מתפרסמים אוטומטית ללא צורך באישורך. הפעל זאת כאשר אתה בטוח באיכות התוכן שאנו מייצרים עבורך.
3. **שם המאשר** שם האדם או הצוות שיקבלו מיילים עם בקשות לאישור.
4. **אימייל המאשר** כתובת האימייל שאליה יישלחו בקשות האישור לפוסטים הקרובים.

Social מגיע עם מגוון יוצרי תוכן לבחירה. כל יוצר מתמקד בדרך ספציפית ליצירת תוכן ייחודי לנכס שלך באמצעות AI. לכל יוצר תוכן יש מאפיין **משקל** — משקל גבוה יותר מגדיל את הסיכוי שהיוצר ייבחר בעת יצירת פוסטים.

להלן כל יוצרי התוכן הזמינים:

| יוצר | תיאור |
| -- | -- |
| **יעד** | משתמש במיקומך ליצירת טקסט מועיל על האזור |
| **סוג חדר** | יוצר פוסט להזמנה הכולל אחד מסוגי החדרים שלך עם טקסט שנוצר ב-AI ותמונות של סוג החדר שלך |
| **פעילות** | יוצר פוסט להזמנה הכולל אחת מהפעילויות שלך |
| **תוספת** | יוצר פוסט להזמנה הכולל אחת מהתוספות שלך |
| **אטרקציה** | יוצר פוסט להזמנה הכולל אחת מהאטרקציות שלך |
| **מקום** | יוצר פוסט להזמנה הכולל אחד מהמקומות שלך |
| **חדר ישיבות** | יוצר פוסט להזמנה הכולל אחד מחדרי הישיבות שלך |
| **מסעדה** | יוצר פוסט להזמנה הכולל אחת מהמסעדות שלך |
| **ספא** | יוצר פוסט להזמנה הכולל אחד מהספאות שלך |
| **נכס** | יוצר פוסט להזמנה הכולל את החדר במחיר הטוב ביותר שלך עם טקסט שנוצר ב-AI ותמונות הנכס שלך |
| **תחושת משתמש** | מאגד ביקורות אורחים מרחבי האינטרנט לפוסט מבוסס תחושה המדגיש מה האורחים אוהבים בנכס שלך |
| **מצחיק** | יוצר פוסטים קלילים ומשעשעים לשמירה על מעורבות הפיד שלך |

:::tip[טיפ מקצועי]
ודא שיש לך את סוג המלאי הנדרש ליוצרי התוכן שבחרת. לדוגמה, אם בחרת "מסעדה", עליך שיהיה לפחות מסעדה אחת רשומה בנכס שלך.
:::

טון הדיבור מגדיר את סגנון השפה שה-AI משתמש בו בעת יצירת התוכן. בחר את הטונים שמתאימים לאישיות המותג שלך. כמו עם יוצרי התוכן, לכל טון יש **משקל** לשליטה בתדירות השימוש בו.

להלן הטונים הזמינים:

| טון | סגנון |
| -- | -- |
| **נייטרלי** | רגוע, סמכותי, עובדתי. ללא הגזמות, ללא בדיחות — רק ידע אמין ומסודר |
| **ידידותי** | חם, נגיש, אנרגיה מחייכת דרך הטקסט עם קיצורים שגרתיים ואימוג'ים מדי פעם |
| **יוקרתי** | מתוחכם, מעודן. מילים כמו "מעולה" ו"מותאם אישית" שגורמות לקוראים להרגיש בלעדיים |
| **חובב אוכל** | תיאורים חושניים ומגרי תיאבון. שפת פורנו אוכל שגורמת לקוראים להרגיש רעבים |
| **הרפתקני** | נועז, אמיץ, אנרגיית חוקר. תיאורים חושיים חיים שהופכים הכל לאפיים |
| **משפחתי** | התרגשות בריאה, ידידותית לילדים. אנרגיה חמה של הורה על חוויות בטוחות ומהנות |
| **רומנטי** | פיוטי, נוסטלגי, מלטף לב. שקיעות, שפה רכה ואהבה |
| **הומוריסטי** | סרקסטי, משפטים קצרים ומהירים, משחקי מילים חכמים. קורא כמו מופע סטנד-אפ |
| **אקולוגי** | רציני וצודק. חוגג אלטרנטיבות ירוקות ובחירות ברות קיימא |
| **מקומי פנימי** | הפניות מקומיות מאוד ספציפיות, מזהיר מלכודות תיירים, משתמש בסלנג מקומי |
| **כוללני** | מזמין, חוגג ומאשר. גורם לכולם להרגיש נראים ומוערכים |
| **מיסטי** | אתרי, רוחני, אמיתות קוסמיות. הזמנות עדינות להסתכל פנימה |
| **עתידני** | חלק, חושב קדימה. אוהב מונחים כמו "שינוי פרדיגמה" וחזונות עתיד אופטימיים |
| **תקציבי** | חסכני, משאבים, גאה בחסכנות. חוגג מציאות משתלמת וחוויות חינמיות |

## סוגי פוסטים

סוגי הפוסטים קובעים את המטרה והמבנה של התוכן שנוצר. לכל סוג פוסט יש **משקל** לשליטה בתדירות השימוש בו.

| סוג פוסט | תיאור |
| -- | -- |
| **עסקי** | ברור, תמציתי, ממוקד המרה. מניע הזמנות ופעולות ישירות עם עובדות, מחירים ושלבים הבאים |
| **מודעות** | תוכן לבניית מותג שנועד להציג את הנכס שלך ולהגביר את הנראות בקרב אורחים פוטנציאליים |
| **תחושה** | תוכן רגשי שיוצר אסוציאציות חיוביות ובונה קשר רגשי עם הקהל שלך |

*רוב הנכסים משתמשים בשפה אחת ומוגדרים לשפתם המקומית כברירת מחדל.*
