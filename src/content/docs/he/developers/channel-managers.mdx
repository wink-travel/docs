Wink משתלב עם מנהלי ערוצים מובילים, מערכות ניהול נכסים (PMS) ומערכות הזמנות מרכזיות (CRS) כדי לשמור על סנכרון המחירים והזמינות שלך בכל ערוצי המכירה.

## איך זה עובד

אינטגרציות עם מנהלי ערוצים מאפשרות זרימת נתונים דו-כיוונית בין מערכת ניהול הנכסים שלך ל-Wink:

1. **דחיפת מחירים** — מנהל הערוצים שלך שולח עדכוני מחירים ל-Wink. כאשר אתה מעדכן את התמחור במערכת ה-PMS שלך, השינויים משתקפים אוטומטית ב-Wink.
2. **סנכרון זמינות** — מנהל הערוצים שלך שולח עדכוני זמינות ל-Wink. כאשר חדרים מוזמנים או משתחררים במערכת ה-PMS שלך, המלאי של Wink מתעדכן בזמן אמת.
3. **הודעות הזמנה** — כאשר מתבצעת הזמנה ב-Wink, מנהל הערוצים שלך מקבל הודעה כך שההזמנה תופיע במערכת ה-PMS.

## מנהלי ערוצים נתמכים

Wink משתלב כיום עם המערכות הבאות:

| מנהל ערוצים | תיאור |
|---|---|
| **Channex** | פלטפורמת הפצה וניהול ערוצים גלובלית. |
| **TravelClick** | פתרונות למלונות וניהול ערוצים (Amadeus). |
| **Omnibees** | הפצה וניהול הכנסות למלונות. |
| **D-Edge** | טכנולוגיית אירוח ופתרונות הזמנות מרכזיות. |
| **Allotz** | הקצאת מלאי וניהול ערוצים. |
| **Ezee / Yanolja** | PMS מבוסס ענן וניהול ערוצים המתמקד בשוק האסייתי. |
| **RateGain** | טכנולוגיית מודיעין מחירים והפצה. |
| **RateTiger** | הפצה בזמן אמת של מחירים וזמינות. |
| **CloudBeds** | PMS מבוסס ענן, מנהל ערוצים ומנוע הזמנות. |
| **SiteMinder** | מנהל ערוצים מוביל ופלטפורמת הפצה למלונות. |
| **Exely** | סנכרון מחירים וזמינות למלונות. |

:::note
אם מנהל הערוצים שלך אינו מופיע ברשימה, פנה לצוות Wink. ניתן להוסיף אינטגרציות חדשות בהתבסס על תקני פרוטוקול OTA (Open Travel Alliance) ש-Wink תומך בהם.
:::

## למפתחים

אם אתה בונה אינטגרציה עם מנהל ערוצים או PMS עבור Wink, השתמש בנקודות הקצה של Channel Manager API:

- **עדכוני מחירים** — דחוף שינויים במחירים ל-Wink.
- **עדכוני זמינות** — דחוף שינויים בזמינות ל-Wink.
- **שליפת הזמנות** — משוך הזמנות מ-Wink למערכת שלך.

ה-Channel Manager API פועל לפי תקני פרוטוקול OTA (SOAP/XML) להתאמה עם מערכות אירוח קיימות.

## התראות Webhook

ניתן להירשם לאירועי webhook של מנהל הערוצים כדי לקבל התראות בזמן אמת:

- `channel-manager.update.rate` — התקבל עדכון מחירים.
- `channel-manager.update.availability` — התקבל עדכון זמינות.
- `channel-manager.update` — עדכון כללי של מנהל הערוצים.

עיין ב-[קטלוג אירועי Webhook](/he/developers/webhook-events#channel-manager-events) לפרטים.
