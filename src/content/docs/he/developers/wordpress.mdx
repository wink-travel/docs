לכל מי שמשתמש בוורדפרס לעסק או לבלוג שלו, אנו ממליצים בחום להשתמש ב-[תוסף וורדפרס Wink לשותפים](https://wordpress.org/plugins/wink2travel/).
התוסף מעניק לכם גישה נוחה לכרטיסי הקריצה שלכם, לרשתות, למפות וכו', ומקל עליכם לגרור תוכן לפוסטים ולעמודים שלכם באמצעות בונה העמודים המועדף עליכם או באמצעות קודי הקישור המותאמים אישית שלנו.

## לְהַתְקִין

כך מתקינים את`Wink Affiliate WordPress Plugin`לתוך אתר הוורדפרס שלך.

<Steps>
  1. כאשר אתה מחובר כמנהל מערכת ובלוח המחוונים.
  2. נְקִישָׁה `Plugins > Add New Plugin`.
  3. חפש `Wink Affiliate WordPress Plugin`.
  4. נְקִישָׁה `Install` כדי להוריד ולהתקין את התוסף באתר שלך.
  5. נְקִישָׁה `Activate` כדי להפעיל את התוסף באתר שלך.
  6. ההפעלה תעביר לך את `Wink Settings` עַמוּד. *ניתן גם להגיע לשם על ידי ניווט אל `Appearance > Customize > Wink Settings`.*
  7. זה יבקש ממך את שלך `Client ID` ו `Client Secret`עבור אל [יישומים](/developers/applications) כדי ללמוד כיצד לאחזר את האישורים שלך.
  8. הזן את פרטי הגישה שלך ולחץ `Publish`כעת אתה מחובר לחשבון Wink שלך.
</Steps>

## בוני דפים

אנו תומכים בבוני הדפים הפופולריים ביותר בוורדפרס, מה שמקל על כל מי שרוצה להשתמש ב-Wink.

### גוטנברג

גוטנברג הוא עורך התוכנות המקורי של וורדפרס.

### אבדה

[אבדה](https://avada.com)מפורסמת בזכות מבחר התבניות היפות הגדול שלה יחד עם בונה דפים חזק עם סט תכונות מתקדם.

### אלמנטור

[אלמנטור](https://elementor.com)הוא בונה הדפים הפופולרי ביותר \[כרגע] עם גרסת חינמית וגרסה מקצועית זמינה.

### מאפיית WP

[מאפיית WP](https://wpbakery.com)היה בונה הדפים המתקדם הראשון בשוק.

## קודים קצרים

### \[חיפוש ווינקלופ]

עובד עם רשת תוכן מדורג. הקלד מקום שברצונך לבקר בו והצג את המלאי.

### \[חיפוש קריצה]

כפתור פשוט לפתיחת טופס מסלול טיול.

### \[winkaccount]

כפתור חשבון שיאפשר לך לאמת. לאחר האימות, הוא הופך לתפריט נפתח עם אפשרויות חשבון.

### \[וינקיטינררי]

זהה לכפתור החיפוש, רק שהוא מכיל מידע על מסלול הטיול כטקסט של הכפתור.

### \[תוכן קריצה]

מאפשר לך להטמיע את המלאי שבחרת ב-Wink ישירות בדף או בפוסט. קוד קצר זה מצפה גם למזהה פריסה.

הפריסה

```
[winkcontent layoutid=""]
```

## קריאה נוספת
