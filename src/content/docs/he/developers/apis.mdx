## ממשק API לתשלום

נקודות קצה לתשלום מיועדות למפתחים המעוניינים לרכוש מלאי נסיעות. ניתן לעשות זאת דרך ה-API כסוכן נסיעות רשום או באמצעות ה-API שלנו בשילוב עם ווידג'ט התשלום התואם PCI שלנו עבור כל הישויות האחרות.
ממשק ה-API לתשלום חושף נקודות קצה עבור שותפים ומלונות כדי לעקוב אחר הזמנות, ניתוחים וזמינות כספים. הם יכולים גם לבחור למשוך כספים זמינים לחשבון הבנק שלהם.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  למידע נוסף
</LinkButton>
