האיצו את זמן ההגעה לשוק על ידי שימוש באחת מערכות פיתוח התוכנה הנתמכות באופן רשמי שלנו.
אם איננו תומכים כעת בשפה המועדפת עליכם, גשו ל-API שברצונכם להשתמש בו והורידו את מפרט OpenAPI שאנו מעמידים לרשותכם עבור כל API בראש העמוד. תוכלו ליצור SDK משלכם על ידי forking (פורק) לאחד ממאגרי SDK הציבוריים שלנו או על ידי שימוש באחד מגנרטורי OpenAPI.

## ג'אווה

ספרייה זו הורכבה עבור Java 24. היא מוגדרת להשתמש ב-WebClient הריאקטיבי והיא ממוטבת לשימוש עם Spring Framework. עם זאת, Spring אינו דרישה, אך הוא כן מספק תמיכה ב-OAuth2 ישירות מהקופסה.
