מיומנויות AI הן חבילות הוראות ניידות המלמדות סוכני קידוד כיצד לעבוד עם Wink. במקום להסתמך על נתוני אימון או תיעוד מיושן, כל מיומנות מכוונת את הסוכן להשתמש ב-[Wink MCP server](/he/developers/mcp) כמקור האמת החי עבור חוזי API ותיעוד הפלטפורמה.

מיומנויות פועלות עם כל לקוח סוכן שתומך בהן — Claude Code, Codex, Cursor ואחרים.

## מיומנויות זמינות

### wink-api-builder

מימוש, סקירה או איתור תקלות באינטגרציות Wink API. המיומנות מלווה את הסוכן בתהליך עבודה מובנה:

1. חיפוש הפעולה הנכונה ב-API באמצעות `api_search`
2. פתיחת חוזי הבקשה/תגובה החיוניים משרת MCP
3. משיכת הגדרות האימות מהתיעוד
4. כתיבת קוד בהתאם לחוזה המאומת
5. סימון כל אי התאמה בין הקוד הקיים ל-API הנוכחי

שימושי ל:
- בניית שיטות לקוח חדשות לכל Wink API
- איתור תקלות בחוסר התאמה בחוזים (למשל בקשה שמחזירה HTTP 400)
- סקירת אינטגרציות קיימות מול המפרט החי

## התקנה

העתק את תיקיית המיומנות לתיקיית המיומנויות של הסוכן שלך.

:::tip
יש לחבר את שרת Wink MCP באותו סביבה. ראה [MCP Server](/he/developers/mcp) להוראות התקנה.
:::

**Claude Code**

```bash
# ברמת הריפוזיטורי
cp -r skills/wink-api-builder .claude/skills/

# או ברמת המשתמש
cp -r skills/wink-api-builder ~/.claude/skills/
```

הפעל עם:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# ברמת הריפוזיטורי
cp -r skills/wink-api-builder .agents/skills/

# או ברמת המשתמש
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

הפעל עם:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**סוכנים אחרים**

מקם את תיקיית המיומנות בכל מקום שהסוכן שלך טוען ממנו מיומנויות. נקודת הכניסה היא `SKILL.md`.
