-- *סר אייזק ניוטון*

פלטפורמה כזו לא נבנית בבידוד. היא סכום של חלקים רבים בתנועה; ישנים וחדשים. אנו רוצים להמשיך לשרת אחרים שיבואו אחרינו.

> "אין צורך להמציא מחדש את הגלגל השבור."

טכנולוגיית התיירות מפוצלת ומבוססת על מערכות ישנות. ניסינו לקחת את הטוב מהישן שעובד. שילבנו מספר מנהלי ערוצים, מערכות ניהול נכסים ++. הוספנו כמה דברים חדשים. ועטפנו את הכל בסטנדרטים ופרוטוקולים בתעשייה שהם מוכרים וניתנים לצריכה בקלות... כדי שלא תצטרכו.

:::note[גישה מונחית API]
משמעותה בקירוב, שכל התכונות שנבנה לעצמנו יהיו:

- מובנות
- גמישות
- ניתנות להרחבה
- ניתנות לשימוש חוזר ו...
- ניתנות להארכה

...על ידי אחרים שרוצים גם להשתמש ב-APIs האלה.
:::

Wink נבנתה באמצעות גישה כזו של `API-driven approach`. כתוצאה מפילוסופיית העיצוב הזו, כל התכונות של Wink ניתנות לאינטראקציה כמפתח חיצוני כאילו המערכת היא שלך. יש גם היבטים נוספים בפלטפורמה שניתן לנצל בעת אינטגרציה איתנו.
