> "אם ראיתי רחוק יותר, זה על ידי עמידה על כתפי ענקים."

\--*סר אייזק ניוטון*

פלטפורמה כזו אינה נבנית בבידוד. אלא, היא סך של חלקים נעים רבים; ישנים וחדשים. אנחנו רוצים להמשיך לשרת אחרים שיבואו אחרינו.

> "אין צורך להמציא מחדש את הגלגל השבור."

טכנולוגיית התיירות מקוטעת ומורשת. ניסינו לקחת את הטוב ביותר מהישן שעובד. שילבנו חבורה של מנהלי ערוצים, מערכות ניהול נכסים ++. הוספנו כמה דברים חדשים. ועטפנו את הכל בסטנדרטים ופרוטוקולים בתעשייה שידועים וקלים לצריכה... כדי שאתם לא תצטרכו.

:::note\[גישה מונעת API]
באופן כללי, כל התכונות שאנו בונים לעצמנו צריכות להיות:

* מוּבָן
* גָמִישׁ
* ניתן להרחבה
* ניתן לשימוש חוזר ו...
* ניתן להארכה

...על ידי אחרים שרוצים גם הם להשתמש בממשקי API אלה.
:::

Wink נבנה באמצעות כזה`API-driven approach`כתוצאה מפילוסופיית עיצוב זו, ניתן לתקשר עם כל התכונות של Wink כמפתח חיצוני כאילו המערכת היא שלך. ישנם גם היבטים נוספים בפלטפורמה שתוכל למנף בעת האינטגרציה איתנו.
