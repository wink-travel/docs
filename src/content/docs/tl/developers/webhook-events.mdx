Ipinapakita ng pahinang ito ang bawat kaganapan sa webhook na maaari mong i-subscribe sa Wink. Para sa mga tagubilin sa pagsasaayos, tingnan ang [Webhook Integration Guide](/tl/guides/webhook-integration) o ang [Webhooks management page](/tl/my-account/webhooks).

## Paano mag-subscribe

1. Pumunta sa **Applications > Webhooks** sa iyong portal.
2. Gumawa ng webhook at piliin ang mga kaganapan na nais mong matanggap.
3. Magpapadala ang Wink ng HTTP POST sa iyong endpoint sa tuwing magaganap ang kaganapan.

Para sa buong mga hakbang sa pagsasaayos at mga pinakamahusay na kasanayan, tingnan ang [Webhook Integration](/tl/guides/webhook-integration).

## Mga kaganapan sa account

Mga kaganapan na may kaugnayan sa lifecycle at mga setting ng iyong TripPay account.

| Kaganapan | Key | Paglalarawan |
|---|---|---|
| Account created | `account.create` | Isang bagong account ang nalikha. |
| Account under review | `account.under_review` | Nasa pagsusuri ang account. |
| Account updated | `account.updated` | Nagbago ang mga detalye ng account. |
| Account status updated | `account.status.updated` | Nagbago ang status ng account (hal., aprubado, suspendido). |
| Account address updated | `account.address.updated` | Binago ang address ng account. |
| Account details updated | `account.details.updated` | Nagbago ang mga detalye ng account. |
| Account information updated | `account.information.update` | Na-update ang pangkalahatang impormasyon ng account. |
| Account managers updated | `account.managers.update` | Nagbago ang listahan ng mga manager ng account. |
| Profile picture updated | `account.profile_picture.updated` | Binago ang larawan ng profile ng account. |
| Agent updated | `account.agent.updated` | Binago ang impormasyon ng ahente sa account. |
| URL name updated | `account.url-name.update` | Nagbago ang vanity URL ng account. |
| Unique ID updated | `account.unique-id.update` | Binago ang natatanging identifier ng account. |

### Mga gawain sa account

| Kaganapan | Key | Paglalarawan |
|---|---|---|
| Task created | `account.task.create` | Isang bagong gawain ang idinagdag sa account. |
| Task updated | `account.task.update` | Binago ang isang gawain sa account. |
| Task removed | `account.task.remove` | Tinanggal ang isang gawain sa account. |
| Task completed | `account.task.complete` | Natapos ang isang gawain sa account. |
| Tax ID task completed | `account.task.tax_id.complete` | Natapos ang gawain sa beripikasyon ng Tax ID. |

### Account acquirer

| Kaganapan | Key | Paglalarawan |
|---|---|---|
| Acquirer created | `account.acquirer.created` | Isang bagong acquirer ang idinagdag sa account. |
| Acquirer updated | `account.acquirer.updated` | Binago ang isang acquirer sa account. |
| Acquirer removed | `account.acquirer.removed` | Tinanggal ang isang acquirer mula sa account. |

## Mga kaganapan sa booking contract

Mga kaganapan na sumusubaybay sa lifecycle ng booking contract sa TripPay.

| Kaganapan | Key | Paglalarawan |
|---|---|---|
| Initialization requested | `booking.contract.initialize.request` | Hiningi ang pagsisimula ng booking contract. |
| Initialization complete | `booking.contract.initialize.complete` | Natapos ang pagsisimula ng booking contract. |
| Contract created | `booking.contract.created` | Isang bagong booking contract ang nalikha. |
| Contract processing | `booking.contract.processing` | Pinoproseso ang bayad ng booking contract. |
| Contract failed | `booking.contract.failed` | Nabigo ang bayad ng booking contract. |
| Contract cancelled | `booking.contract.cancelled` | Kinansela ang booking contract. |
| Partial refund | `booking.contract.refund.partial` | Nag-isyu ng partial refund sa contract. |
| Full refund | `booking.contract.refund.full` | Nag-isyu ng buong refund sa contract. |
| Refund requested | `booking.contract.refund.request` | Isinumite ang kahilingan para sa refund. |
| Refund error | `booking.contract.refund.error` | Nagkaroon ng error habang pinoproseso ang refund. |
| Payout created | `booking.contract.payout.create` | Nalikha ang payout para sa contract. |
| Payout completed | `booking.contract.payout.complete` | Matagumpay na natapos ang payout. |

## Mga kaganapan sa managing entity

Mga kaganapan para sa mga managing entity (mga property, affiliate, kumpanya) at ang kanilang administrasyon.

### Subscription

| Kaganapan | Key | Paglalarawan |
|---|---|---|
| Subscription provisioned | `managing_entity.subscription.provisioned` | Naitakda ang isang bagong subscription. |
| Subscription changed | `managing_entity.subscription.changed` | Nagbago ang plano ng subscription. |
| Subscription invoice changed | `managing_entity.subscription.invoice.changed` | Na-update ang invoice ng subscription. |

### Status at administrasyon

| Kaganapan | Key | Paglalarawan |
|---|---|---|
| Status updated | `managing_entity.status.updated` | Nagbago ang status ng entity. |
| Entity deleted | `managing_entity.deleted` | Tinanggal ang isang managing entity. |
| Agent updated | `managing_entity.agent.updated` | Binago ang ahente ng entity. |
| Agent email notification | `managing_entity.agent.email.notification` | Nagpadala ng email notification sa ahente. |
| Manager email notification | `managing_entity.manager.email.notification` | Nagpadala ng email notification sa isang manager. |

### Lifecycle ng manager

| Kaganapan | Key | Paglalarawan |
|---|---|---|
| Manager added | `managing_entity.manager.added` | Isang bagong manager ang idinagdag. |
| Manager invited | `managing_entity.manager.invited` | Nagpadala ng imbitasyon sa manager. |
| Manager removed | `managing_entity.manager.removed` | Tinanggal ang isang manager. |
| Manager accepted | `managing_entity.manager.accepted` | Tinanggap ang imbitasyon ng manager. |
| Manager rejected | `managing_entity.manager.rejected` | Tinanggihan ang imbitasyon ng manager. |
| Owner set | `managing_entity.manager.owner.set` | Nailipat ang pagmamay-ari ng entity. |

### Impormasyon ng entity

| Kaganapan | Key | Paglalarawan |
|---|---|---|
| Information updated | `managing_entity.information.updated` | Na-update ang impormasyon ng entity. |
| Address updated | `managing_entity.city.updated` | Nagbago ang address/lungsod ng entity. |
| Account representative updated | `managing_entity.account_representative.updated` | Nagbago ang kinatawan ng account. |
| Profile picture updated | `managing_entity.profile_picture.updated` | Na-update ang larawan ng profile ng entity. |
| Logo updated | `managing_entity.logo.updated` | Na-update ang logo ng entity. |
| URL name updated | `managing_entity.url_name.updated` | Nagbago ang vanity URL ng entity. |
| Unique ID updated | `managing_entity.unique_id.updated` | Nagbago ang natatanging identifier ng entity. |
| Name updated | `managing_entity.name.updated` | Nagbago ang pangalan ng entity. |
