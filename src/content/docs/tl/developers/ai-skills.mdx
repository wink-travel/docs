Ang mga kasanayan ng AI ay mga portable na pakete ng instruksyon na nagtuturo sa mga coding agent kung paano gamitin ang Wink. Sa halip na umasa sa training data o lipas na dokumentasyon, bawat kasanayan ay nagtuturo sa ahente na gamitin ang [Wink MCP server](/tl/developers/mcp) bilang live na pinagmumulan ng katotohanan para sa mga API contract at dokumento ng platform.

Ang mga kasanayan ay gumagana sa anumang agent client na sumusuporta dito — Claude Code, Codex, Cursor, at iba pa.

## Mga Available na Kasanayan

### wink-api-builder

Mag-implementa, mag-review, o mag-debug ng mga integrasyon ng Wink API. Ginagabayan ng kasanayan ang ahente sa isang istrukturadong workflow:

1. Maghanap ng tamang API operation gamit ang `api_search`
2. Buksan ang live na request/response contracts mula sa MCP server
3. Kunin ang authentication setup mula sa mga dokumento
4. Sumulat ng code laban sa beripikadong contract
5. I-flag ang anumang hindi tugma sa pagitan ng umiiral na code at kasalukuyang API

Kapaki-pakinabang para sa:
- Paggawa ng mga bagong client method para sa anumang Wink API
- Pag-debug ng mga hindi tugmang contract (hal. isang request na nagbabalik ng HTTP 400)
- Pag-review ng mga umiiral na integrasyon laban sa live na spec

## Install

Kopyahin ang skill folder sa skill directory ng iyong ahente.

:::tip
Ang Wink MCP server ay dapat nakakonekta sa parehong environment. Tingnan ang [MCP Server](/tl/developers/mcp) para sa mga tagubilin sa setup.
:::

**Claude Code**

```bash
# repo-local
cp -r skills/wink-api-builder .claude/skills/

# o user-level
cp -r skills/wink-api-builder ~/.claude/skills/
```

I-invoke gamit ang:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repo-local
cp -r skills/wink-api-builder .agents/skills/

# o user-level
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

I-invoke gamit ang:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Ibang mga ahente**

Ilagay ang skill folder saan man kinukuha ng iyong ahente ang mga kasanayan. Ang entry point ay `SKILL.md`.

## Mga Kinakailangan

- Isang nakakonektang [Wink MCP server](/tl/developers/mcp)
- Isang agent client na may suporta sa MCP at kasanayan
