Ang Wink ay nag-iintegrate sa mga nangungunang channel manager, property management system (PMS), at central reservation system (CRS) upang mapanatiling naka-synchronize ang iyong mga rate at availability sa lahat ng iyong mga sales channel.

## Paano ito gumagana

Pinapagana ng mga integrasyon ng channel manager ang dalawang-daan na daloy ng data sa pagitan ng iyong property management system at Wink:

1. **Rate push** — Ipinapadala ng iyong channel manager ang mga update sa rate sa Wink. Kapag in-update mo ang iyong presyo sa iyong PMS, awtomatikong naipapakita ang mga pagbabagong iyon sa Wink.
2. **Availability sync** — Ipinapadala ng iyong channel manager ang mga update sa availability sa Wink. Kapag may mga kuwartong nare-reserba o naibabalik sa iyong PMS, ang imbentaryo ng Wink ay na-update nang real time.
3. **Booking notifications** — Kapag may booking na ginawa sa Wink, naipapaalam ito sa iyong channel manager upang lumitaw ang reserbasyon sa iyong PMS.

## Mga suportadong channel manager

Kasulukuyan nang iniintegrate ng Wink ang mga sumusunod na sistema:

| Channel Manager | Paglalarawan |
|---|---|
| **Channex** | Global distribution at channel management platform. |
| **TravelClick** | Mga solusyon sa hotel at channel management (Amadeus). |
| **Omnibees** | Distribution at revenue management para sa mga hotel. |
| **D-Edge** | Teknolohiya sa hospitality at mga solusyon sa central reservation. |
| **Allotz** | Inventory allocation at channel management. |
| **Ezee / Yanolja** | Cloud PMS at channel management na nakatuon sa merkado ng Asya. |
| **RateGain** | Rate intelligence at distribution technology. |
| **RateTiger** | Real-time na distribusyon ng rate at availability. |
| **CloudBeds** | Cloud-based PMS, channel manager, at booking engine. |
| **SiteMinder** | Nangungunang hotel channel manager at distribution platform. |
| **Exely** | Pag-sync ng rate at availability para sa mga hotel. |

:::note
Kung ang iyong channel manager ay hindi nakalista, makipag-ugnayan sa koponan ng Wink. Maaaring maidagdag ang mga bagong integrasyon batay sa mga pamantayan ng OTA (Open Travel Alliance) protocol na sinusuportahan ng Wink.
:::

## Para sa mga developer

Kung ikaw ay gumagawa ng integrasyon ng channel manager o PMS sa Wink, gamitin ang mga Channel Manager API endpoint:

- **Rate updates** — I-push ang mga pagbabago sa rate sa Wink.
- **Availability updates** — I-push ang mga pagbabago sa availability sa Wink.
- **Booking retrieval** — Kunin ang mga booking mula sa Wink papunta sa iyong sistema.

Ang Channel Manager API ay sumusunod sa mga pamantayan ng OTA protocol (SOAP/XML) para sa pagiging compatible sa mga umiiral na sistema ng hospitality.

## Mga notification sa webhook

Maaari kang mag-subscribe sa mga webhook event ng channel manager upang makatanggap ng mga real-time na notification:

- `channel-manager.update.rate` — Natanggap ang update sa rate.
- `channel-manager.update.availability` — Natanggap ang update sa availability.
- `channel-manager.update` — Pangkalahatang update ng channel manager.

Tingnan ang [Webhook Events Catalog](/tl/developers/webhook-events#channel-manager-events) para sa mga detalye.
