Pabilisin ang time-to-market sa pamamagitan ng paggamit ng isa sa aming opisyal na suportadong wika Software Development Kit.
Kung sa kasalukuyan ay hindi namin sinusuportahan ang iyong paboritong wika, pumunta sa API na gusto mong gamitin at i-download ang OpenAPI spec na ginagawa naming available para sa bawat API sa itaas ng page. Maaari kang bumuo ng sarili mong SDK sa pamamagitan ng pag-forking sa isa sa aming mga pampublikong SDK na repository o sa pamamagitan ng paggamit ng isa sa mga generator ng OpenAPI.

## Java

Ang library na ito ay pinagsama-sama para sa Java 24. Ito ay na-configure upang gamitin ang reaktibong WebClient at na-optimize para sa paggamit sa Spring Framework. Gayunpaman, hindi kinakailangan ang Spring ngunit binibigyan ka nito ng suporta sa OAuth2 nang wala sa kahon.

</LinkButton>
