Ang lahat ng aming REST API ay language agnostic at Open API compliant. Simulan ang pagsubok nang hindi nagsusulat ng isang linya ng code.

Kung hindi mo pa nagagawa ang iyong Wink account, [magsimula ka dito](/user/register).

Ang Ping API ay isang mabilis na endpoint ng pagsubok upang i-verify na gumagana ang iyong mga kredensyal.

</LinkButton>

## API ng Pagbabayad

Ang mga endpoint ng pagbabayad ay para sa mga developer na gustong bumili ng imbentaryo ng paglalakbay. Magagawa ito sa pamamagitan ng API bilang isang rehistradong Travel Agent o gamit ang aming API kasabay ng aming widget ng pagbabayad na sumusunod sa PCI para sa lahat ng iba pang entity.
Inilalantad ng Payment API ang mga endpoint para sa mga affiliate at hotel para subaybayan ang mga booking, analytics at availability ng mga pondo. Maaari din nilang piliing mag-withdraw ng mga magagamit na pondo sa kanilang bank account.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Matuto pa
</LinkButton>
