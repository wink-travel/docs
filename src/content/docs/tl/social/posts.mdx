Pinapayagan ka ng post editor na gumawa ng walang limitasyong manual na mga post at dito ka rin pupunta upang pamahalaan ang mga post na awtomatikong ginawa para sa iyo.

May 4 na seksyon ang editor:

1. Ang seksyon ng nilalaman.
2. Ang seksyon ng media.
3. Ang seksyon ng link.
4. Ang seksyon ng mga setting.

## Nilalaman

Pinapayagan ka ng seksyon ng nilalaman na baguhin ang teksto bago ito mailathala. Maaari kang gumawa ng nilalamang lokal sa maraming wika — bawat wika ay may sariling pamagat at paglalarawan.

## Media

Pinapayagan ka ng seksyon ng media na baguhin ang mga larawan at video na kasama ng teksto.

## Link

Pinapayagan ka ng seksyon ng link na maglagay ng link papunta sa lugar na nais mong puntahan ng gumagamit pagkatapos basahin ang iyong post. *Halimbawa, maaari itong maging link sa isang espesyal na alok o lugar para sa karagdagang impormasyon.* Maaari kang magdagdag ng URL ng link, caption ng link, at paglalarawan ng link.

## Mga Setting

Pinapayagan ka ng seksyon ng mga setting na baguhin / i-override ang lahat ng iyong default na mga setting para sa bawat post. Maaari mong itakda ang:

- **Platform** — Piliin kung saang social network ipo-publish ang post na ito.
- **Uri ng post** — Piliin sa pagitan ng Transactional, Awareness, o Sentiment.
- **Tono ng boses** — I-override ang tono ng boses para sa partikular na post na ito.
- **Pag-customize** — Gumamit ng ibang pag-customize para sa display currency, wika, itinerary, atbp.
- **Timezone** — I-override ang timezone para sa publikasyon.
- **Iskedyul na petsa** — Awtomatikong na-iiskedyul ang mga generated na post sa pinakamainam na oras. Maaari mong baguhin ang petsa at oras kung nais mo ng ibang slot.

## Siklo ng buhay ng status ng post

Ang bawat post ay sumusunod sa siklo ng buhay ng status:

| Status | Paglalarawan |
| -- | -- |
| **Draft** | Nalikha na ang post ngunit hindi pa handa para sa pagsusuri |
| **Pending approval** | Naghihintay ang post ng aprubal ng tao bago i-iskedyul |
| **Approved** | Naaprubahan na ang post at i-iskedyul na |
| **Rejected** | Nasuri at tinanggihan ang post — hindi ito ipo-publish |
| **Scheduled** | Nakapila ang post para sa publikasyon sa naka-iskedyul na petsa at oras |
| **Published** | Matagumpay na na-publish ang post sa social network |
| **Paused** | Pansamantalang na-pause ang publikasyon ng post |
| **Failed** | Hindi na-publish ang post dahil sa error |

## Pag-apruba ng mga post

Inaaprubahan ang mga post sa post grid kung saan maaari kang pumili ng isa o higit pang mga post at piliing i-apruba o tanggihan ang mga ito.

:::note
Magkakaroon ng isang entry para sa bawat wika at bawat social network.

Halimbawa, kung mayroon kang 3 social network at 3 wika na napili, makikita mo ang 9 na entry sa kalendaryo.
:::
