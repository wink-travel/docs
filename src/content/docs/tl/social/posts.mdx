Pinapayagan ka ng post editor na gumawa ng walang limitasyong manual na mga post at dito ka rin pupunta upang pamahalaan ang mga post na awtomatikong ginawa para sa iyo.

Ang editor ay may 4 na seksyon:

1. Ang seksyon ng nilalaman.
2. Ang seksyon ng media.
3. Ang seksyon ng link.
4. Ang seksyon ng mga setting.

## Nilalaman

Pinapayagan ka ng seksyon ng nilalaman na baguhin ang teksto bago ito mailathala.

## Media

Pinapayagan ka ng seksyon ng media na baguhin ang mga larawan at video na kasama ng teksto.

## Link

Pinapayagan ka ng seksyon ng link na maglagay ng link papunta sa lugar na nais mong puntahan ng gumagamit pagkatapos niyang basahin ang iyong post. *Halimbawa, maaari itong maging link sa isang espesyal na alok o lugar para sa karagdagang impormasyon.*

## Mga Setting

Pinapayagan ka ng seksyon ng mga setting na baguhin o i-override ang lahat ng iyong default na mga setting. Maaari itong makatulong kung nais mong lumabas ang post sa ibang oras. Kadalasan, hindi ginagamit ang seksyong ito.

## Pag-apruba ng mga post

Inaaprubahan ang mga post sa post grid kung saan maaari kang pumili ng isa o higit pang mga post at piliin kung i-aapruba o tatanggihan ang isang post.

:::note
Magkakaroon ng isang entry para sa bawat wika at bawat social network.

Halimbawa, kung mayroon kang 3 social network at 3 wika na napili, makikita mo ang 9 na entry sa kalendaryo.
:::
