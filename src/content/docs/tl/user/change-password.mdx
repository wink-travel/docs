Kung nais mong palitan ang iyong password para sa iyong Wink user account, sundin ang mga hakbang na ito:

## I-update ang password

<Steps>
1. Dapat naka-log in ka sa isa sa aming mga website.
2. I-click ang iyong profile icon, sa kanang itaas na bahagi at lalabas ang isang menu.
3. I-click ang link na `Change password`.
4. Sundin ang mga tagubilin upang palitan ang iyong password.
5. I-click ang `Save`.
</Steps>

Sa susunod na pag-login mo, kakailanganin mo ang iyong bagong password.

## Nakalimutan ang password

Minsan kinakailangan na i-reset ang password ng user.

Narito ang mga hakbang sa pag-reset ng iyong user password:

<Steps>
1. I-click ang link na `Forgot password` mula sa sign-in page.
2. I-type ang username / e-mail para sa account na nais mong i-reset.
3. I-click ang button na `Forgot password`. *Ikaw ay ire-redirect sa reset password form.*
4. Suriin ang iyong e-mail.
</Steps>

2. Kumpirmahin ang iyong bagong password.
3. I-click ang button na `Reset password`. *Ikaw ay ire-redirect pabalik sa sign-in page.*
</Steps>
