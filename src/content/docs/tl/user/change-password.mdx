Kung gusto mong baguhin ang iyong password para sa iyong Wink user account, sundin ang mga hakbang na ito:

## I-update ang password

<Steps>
  1. Dapat kang naka-log in sa isa sa aming mga website.
  2. I-click ang icon ng iyong profile, sa kanang sulok sa itaas at may lalabas na menu.
  3. I-click ang `Change password` link.
  4. Sundin ang mga tagubilin upang baguhin ang iyong password.
  5. I-click `Save`.
</Steps>


Sa susunod na mag-log in ka, kakailanganin mo ang iyong bagong password.

## Nakalimutan ang password

Minsan kinakailangan na i-reset ang password ng user.

Narito ang mga hakbang sa pag-reset ng iyong password ng user:

<Steps>
  1. Mag-click sa `Forgot password` link mula sa pahina ng pag-sign-in.
  2. I-type ang username / e-mail para sa account na gusto mong i-reset.
  3. I-click `Forgot password` pindutan. *Na-redirect ka sa form ng pag-reset ng password.*
  4. Suriin ang iyong e-mail.
</Steps>

<Steps>
  1. I-paste ang code na iyong natanggap sa e-mail sa bagong form kasama ng iyong bagong password.
  2. Kumpirmahin ang iyong bagong password.
  3. I-click `Reset password` pindutan. *Ire-redirect ka pabalik sa pahina ng pag-sign in.*
</Steps>

Ang iyong password ay matagumpay na na-reset. Maaari kang mag-log in sa Wink gamit ang iyong mga bagong kredensyal.
