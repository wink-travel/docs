Kung nais mong i-update ang iyong user profile para sa iyong Wink user account, sundin ang mga hakbang na ito:

<Steps>
1. Dapat naka-log in ka sa isa sa aming mga website.
2. I-click ang iyong profile icon, sa kanang itaas na sulok at lilitaw ang isang menu.
3. I-click ang link na `Profile`.
4. Sundin ang mga tagubilin upang i-update ang iyong `e-mail`, `unang` at `huling pangalan`.
5. Maaari ka ring mag-upload ng bagong `profile picture` ng iyong sarili.
6. I-click ang `Save`.
</Steps>

:::note
Ang iyong bagong profile picture ay ipapakita sa susunod na mag-log out ka nang ganap mula sa Wink.
:::
