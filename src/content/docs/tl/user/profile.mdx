Kung gusto mong i-update ang iyong profile ng user para sa iyong Wink user account, sundin ang mga hakbang na ito:

<Steps>
  1. Dapat kang naka-log in sa isa sa aming mga website.
  2. I-click ang icon ng iyong profile, sa kanang sulok sa itaas at may lalabas na menu.
  3. I-click ang `Profile` link.
  4. Sundin ang mga tagubilin upang i-update ang iyong `e-mail`, `first` at `last name`.
  5. Maaari ka ring mag-upload ng bago `profile picture` ng iyong sarili.
  6. I-click `Save`.
</Steps>


:::tandaan
Ipapakita ang iyong bagong larawan sa profile sa susunod na ganap kang mag-log out mula sa Wink.
:::
