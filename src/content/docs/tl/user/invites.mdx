Maaaring piliin ng user ng Wink na hayaan kang pamahalaan ang isa sa kanilang mga property, affiliate o travel agent account. Kapag nangyari iyon, makakatanggap ka ng isang e-mail na may kahilingan sa imbitasyon.

import { Steps } from '@astrojs/starlight/components';

Upang makita ang iyong kasalukuyang mga imbitasyon sa Wink, sundin ang mga hakbang na ito:

<Steps>
  1. Dapat kang naka-log in sa isa sa aming mga website.
  2. I-click ang icon ng iyong profile, sa kanang sulok sa itaas at may lalabas na menu.
  3. I-click ang `Invites` link.
  4. Makipag-ugnayan sa iyong mga imbitasyon sa pamamagitan ng alinman `Accepting` o `Rejecting` sila.
</Steps>
