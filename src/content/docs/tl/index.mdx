Ang site na ito ay naglalaman ng lahat ng impormasyon na kailangan mong malaman upang matagumpay na magamit ang lahat ng mga tampok at serbisyo ng Wink. Mag-click sa [Magsimula](/getting-started/how-to-use) kung hindi mo pa nagamit ang aming site dati o nag-navigate sa isang partikular na seksyon ng site sa ibaba.

## Saan mo gustong pumunta...

  <Card title="Pinamamahalaang Social" icon="seti:info">

  </Card>

  <Card title="Wink Agency" icon="seti:info">

  </Card>

  <Card title="WinkLinks" icon="seti:info">
    Gusto mo bang gamitin ang aming cool na link manager para sa iyong IG bio URL? Pumunta sa [Seksyon ng WinkLinks](/link-manager/wink-links/) ng site na ito.
  </Card>

  <Card title="Mga kaakibat" icon="seti:info">
    Kahit sinong gustong matuto kung paano kumita ng passive income sa Wink? Pumunta sa [Seksyon ng kaakibat](/studio/what-is-studio/) ng site na ito.
  </Card>

  <Card title="Booking Engine" icon="seti:info">
    Gustong matutunan ang mga pasikot-sikot ng aming naka-whitelabel na booking engine? Pumunta sa [Booking Engine](/booking-engine/in-a-nutshell) para malaman.
  </Card>

  <Card title="Mga korporasyon" icon="seti:info">

  </Card>

  <Card title="Mga Katangian" icon="seti:info">
    Ang mga property na gustong i-onboard ang kanilang mga property sa Wink, pumunta sa [Seksyon ng ari-arian](/extranet/what-is-extranet) ng site na ito.
  </Card>

  <Card title="Mga Ahente sa Paglalakbay" icon="seti:info">
    Ang mga ahente sa paglalakbay na gustong magkaroon ng mga espesyal na rate, pumunta sa [Seksyon ng Travel Agent](/travel-agent/what-is-agent) ng site na ito.
  </Card>

  <Card title="Mga pagbabayad" icon="seti:info">
    Sinumang gustong malaman kung paano pinoproseso ng Wink ang mga pagbabayad, pumunta sa [Seksyon ng mga pagbabayad](/payment/what-is-trip-pay).
  </Card>

  <Card title="Mga webinar" icon="seti:info">
