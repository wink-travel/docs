Ang [Wink Agent](https://agent.wink.travel) ay nag-uugnay sa iyo sa mga travel inventory sa buong mundo. Tinatahak ka ng artikulong ito sa paggawa at pamamahala ng iyong mga travel agent account sa Wink.

## Gumawa ng account

Kapag ikaw ay na-authenticate, ikaw ay ire-redirect sa iyong [accounts page](https://agent.wink.travel/company/list) na nagpapakita ng lahat ng iyong travel agent account.

I-click ang `Create new account` na button upang magpatuloy. Ikaw ay ire-redirect sa aming [account creation](https://agent.wink.travel/company/create) page.

Punan ang lahat ng kinakailangang mga patlang.

<Steps>
1. **Pangalan** Ang pangalan ng iyong account. *Hindi kailangang ito ang iyong totoong pangalan ngunit ito ang makikita ng mga user kapag tiningnan ang iyong WinkLinks page o kapag nais ng mga supplier na direktang makipag-ugnayan sa iyo.*
2. **Uri ng entidad** Kung ikaw ay isang kumpanya o indibidwal. *Kadalasang ginagamit upang matukoy kung paano ka karapat-dapat tumanggap ng pondo.*
3. **Uri** Pumili ng uri na pinakaangkop sa iyong account. *hal. Influencer*
4. **Deskripsyon** Sumulat ng deskripsyon para sa iyong account. *Makikita ito sa iyong WinkLinks page at kapag nais ng mga supplier na direktang makipag-ugnayan sa iyo.*
5. **Website** Ilagay ang iyong pangunahing website. *Maaaring ito ang URL ng iyong IG account o website ng kumpanya. Dito mo ginagawa ang karamihan ng iyong online na trabaho. Pupuntahan ng mga supplier ang mga link na ito kapag tinutukoy nila kung makikipag-ugnayan sa iyo.*
6. **Lungsod** Saan ka nakabase?
</Steps>

### Mga termino sa pagbabayad

:::note
Kung pipiliin mong ikaw ang mananagot sa pagbabayad, magkakaroon ka lamang ng access sa mga property na may direktang koneksyon sa iyo.
:::

:::caution
Hindi mananagot ang Wink sa pagbabayad ng mga hotel sa ganitong sitwasyon.
:::

:::note
Maaari mo lamang gamitin ang opsyong ito sa panahon ng paggawa ng account. Karamihan sa mga travel agent ay dapat iwanang naka-disable ðŸ›‘ ang setting na ito at gamitin ang portal na ito sa halip.
:::

Ang mga rehistradong travel agent na nag-iintegrate sa amin gamit ang aming API ay may opsyon na tumanggap ng bayad at ipamahagi ang mga pondong iyon sa aming mga property.

<Steps>
1. **Gusto kong tumanggap ng bayad** Ang travel agent ang mananagot sa pagkolekta ng pondo.
2. **Gusto kong magbayad sa mga hotel** Ang travel agent ang mananagot sa pagbabayad sa mga hotel.
</Steps>

Makikita ng mga hotel na nais direktang makipag-ugnayan sa iyo ang isang babala at kailangang i-click upang kilalanin na hindi mananagot ang Wink sa pagbabayad ng mga hotel, upang maitatag ang relasyon sa iyong account.

<Steps>
1. **Kasunduan** Tanggapin ang aming [affiliate agreement](/tl/travel-agent/terms-of-service) at [payment terms](/tl/travel-agent/payment-terms) upang magpatuloy.
2. I-click ang `Save` na button.
</Steps>

Kapag nagawa na, maaari mong dagdagan ang iyong account sa pamamagitan ng pagpuno ng iyong address at pagdagdag ng profile picture sa iyong account.

<Steps>
1. I-click ang iyong bagong travel agent account. *Dadala ka nito sa iyong account dashboard.*
2. I-click ang `Account > Profile` sa pangunahing nav bar.
3. Sa Profile page, maaari mong i-update ang iyong kasalukuyang profile at magdagdag ng karagdagang impormasyon dito.
</Steps>

:::note
Ang profile picture ng iyong travel agent account ay hindi dapat ipagkamali sa iyong Wink user profile picture. Ang profile picture ng iyong travel agent account ay ipapakita sa iyong WinkLinks page at makikita ng mga supplier na nais makipag-ugnayan sa iyo.

Ang iyong user profile picture ay kadalasang para sa panloob na gamit.
:::

## Mga Manager

Maaari kang mag-imbita ng ibang mga user upang tumulong sa pamamahala ng iyong travel agent account. Kapag tinanggap ng user ang iyong imbitasyon, magkakaroon sila ng buong access sa iyong account maliban sa pagbabayad. Lagi kang magiging tanging may hawak ng pondo na pag-aari mo.

<Steps>
1. Piliin ang travel agent account na nais mong trabahuhin mula sa [accounts page](https://agent.wink.travel/company/list).
2. I-click ang `Account > Managers` sa pangunahing nav bar.
3. Idagdag ang e-mail ng user na nais mong gawing account manager.
4. I-click ang `Invite` na link sa tabi ng email.
</Steps>

Magpapadala ang Wink ng email sa user upang tanggapin o tanggihan ang iyong imbitasyon.

## Mga Notification

Minsan, nais naming makipag-ugnayan sa iyo. Kapag nangyari iyon, makikita mong magbabago ang kampanilya sa itaas na kanang sulok.
Para basahin ang aming ipinadala sa iyo:

<Steps>
1. I-click ang `Account > Notifications` sa pangunahing nav bar.
2. Makikita mo ang mga bagong at lumang notification.
3. Piliin ang `Mark as read` o `Delete` ang mga notification kapag nabasa mo na ang mga ito.
</Steps>

## API

Ang mga travel agent ay itinuturing pa ring mga affiliate. Ang mga developer na nais pamahalaan ang `Affiliates` ay maaaring pumunta sa [Developers > API > Affiliate](/tl/developers/apis/#affiliate-api).
