Kapag ang isang booking ay ginawa sa Wink, ang mga sumusunod ay mangyayari:

<Steps>
  1. Inaabisuhan ang iyong channel manager, CRS o PMS.
  2. Nakatanggap ka ng email ng kumpirmasyon *(mga contact sa reservation desk)* na may link para pamahalaan ang iyong bagong booking.
  3. Nakatanggap ang manlalakbay ng email ng kumpirmasyon na kasama rin kung paano makipag-ugnayan sa iyong reservation desk.
  4. Ang travel agent ay tumatanggap ng parehong booking confirmation gaya ng traveler.
  5. Maaaring makinig ang mga developer sa `booking.created` webhook event at makatanggap ng detalyadong impormasyon sa booking mula sa amin sa realtime.
</Steps>


Alinman sa sundin ang link sa email, o pumunta sa `Account > Bookings` mula sa pangunahing nav bar upang makapagsimula.

## grid ng pag-book

Ipinapakita sa iyo ng grid ng booking ang lahat ng iyong mga nakaraan at hinaharap na booking, na pinagsunod-sunod ayon sa petsa ng libro sa pataas na pagkakasunud-sunod.

### Mga filter

Maaari kang maglapat ng mga filter upang mahanap ang mga booking na iyong hinahanap.

* Petsa ng libro \[saklaw]
* Petsa ng pagdating \[saklaw]
* Petsa ng pag-alis \[saklaw]
* Code ng booking
* Apelyido ng manlalakbay
* Pangalan ng manlalakbay
* Pangalan ng master rate
* Pangalan ng ari-arian

### I-export

Maaari kang mag-export ng mga booking, na tumutugma sa iyong kasalukuyang pamantayan sa filter, sa CSV.

Upang i-export ang iyong mga booking:

## Pagbu-book

Piliin ang booking na gusto mong gamitin mula sa booking grid at ire-redirect ka sa `Booking` pahina.
Ang pahinang ito ay naglalaman ng lahat ng mga detalye na kailangan mong malaman upang maayos na matanggap ang iyong bisita sa araw ng pagdating.

Naglalaman din ang page ng ilang mahahalagang aksyon na kailangan mo para maayos na maserbisyuhan ang iyong mga bisita. Mahahanap mo ang lahat ng magagamit na pagkilos sa `Actions` dropdown sa itaas ng page.

### Muling ipadala ang e-mail

Upang muling ipadala ang email ng kumpirmasyon sa manlalakbay, i-click `Actions > Resend confirmation email`.

### Kanselahin ang booking

Para kanselahin ang booking bilang property, i-click `Actions > Cancel booking`.

May lalabas na pop-up window na magbibigay-daan sa iyo:

<Steps>
  1. Piliin ang uri ng pagkansela. *hal. Mga maling petsa*
  2. Mag-type ng dahilan para sa pagkansela.
</Steps>


*Ang button na ito ay sumusunod sa iyong mga patakaran sa pagkansela at samakatuwid ay hindi palaging magagamit.*

## API

Mga developer na gustong pamahalaan `Bookings`, bilang isang ahente sa paglalakbay, ay maaaring magtungo sa [Mga Developer > API > Travel Agent](/developers/apis/#travel-agent-api).
