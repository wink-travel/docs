Para pamahalaan ang iyong mga bank account, i-click ang `Account > Bank Accounts` mula sa sidebar.

## Pagdaragdag ng bank account

I-click ang `Add bank account` para magrehistro ng bagong account. Ang form ay nag-aangkop base sa napili mong bansa:

- **Payment country** — Ang bansa kung saan matatagpuan ang iyong bangko.
- **Currency** — Ang pera na gusto mong matanggap ang mga payout.
- **Account holder name** — Ang pangalan sa bank account.
- **Routing details** — Mga patlang na partikular sa bansa tulad ng Routing number (US), Sort code (UK), SWIFT/BIC code, o IBAN.
- **Account number** — Ang numero ng iyong bank account.

## Pangunahing account

Maaari kang magtalaga ng isang bank account bilang **Primary**. Ang iyong pangunahing account ang tatanggap ng lahat ng awtomatikong payout. Ang mga karagdagang account ay maaaring gamitin para sa manu-manong pag-withdraw.

Para baguhin ang iyong pangunahing account, i-click ang `Make primary` sa anumang account sa iyong listahan.

## Pamamahala ng mga account

Mula sa listahan ng bank account maaari mong:

- I-edit ang mga detalye ng account.
- I-delete ang account na hindi mo na kailangan.
- Palitan kung alin ang pangunahing account.
