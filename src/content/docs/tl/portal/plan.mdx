Pinapayagan ka ng pahina ng subscription na pamahalaan ang iyong plano at ang mga property na lisensyado sa ilalim ng iyong account.

## Available plans

Mag-browse ng mga available na subscription tier na ipinapakita sa pamamagitan ng aming pricing table. Bawat plano ay may iba't ibang set ng mga tampok at limitasyon sa property. Piliin ang planong pinakaangkop sa iyong mga pangangailangan.

## Current subscription

Kung mayroon ka nang aktibong subscription, ipinapakita ng seksyong ito ang mga detalye at status ng iyong kasalukuyang plano.

## Licenses

Ipinapakita ng license table ang lahat ng mga property na na-provision sa ilalim ng iyong subscription. Maaari mong:

- Tingnan kung aling mga property ang kasalukuyang lisensyado.
- Magdagdag ng mga bagong lisensya ng property gamit ang `Add license` button.
- Pamahalaan ang billing sa pamamagitan ng link ng Stripe customer portal.

:::note
Ang pamamahala ng billing (mga paraan ng pagbabayad, mga invoice, pagbabago ng plano) ay hinahandle sa pamamagitan ng Stripe customer portal.
:::
