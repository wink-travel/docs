Pinapayagan ka ng Recognition na i-highlight ang mga parangal, sertipikasyon, at badge ng industriya ng iyong property upang makabuo ng tiwala sa mga potensyal na bisita.

Para pamahalaan ang pagkilala, i-click ang `Property > Recognition` mula sa pangunahing nav bar.

## Pagdaragdag ng pagkilala

I-click ang `Create` upang magdagdag ng bagong entry ng pagkilala. Bawat entry ay may kasamang:

- **Category** — Ang uri ng pagkilala (hal., award, sertipikasyon).
- **Type** — Ang partikular na uri ng pagkilala.
- **Provider** — Ang nagbigay na organisasyon (hal., Michelin, TripAdvisor, AAA).
- **Rating** — Numerikong halaga ng rating, kung naaangkop.
- **Rating symbol** — Ang icon o simbolo na ginagamit para ipakita ang rating.
- **Date** — Kailan nakuha ang pagkilala.
- **Official appointment** — Kung ito ay isang opisyal na pagtatalaga.

## Pamamahala ng mga entry

Mula sa listahan ng pagkilala, maaari mong i-edit o alisin ang mga umiiral na entry. Ang mga entry ng pagkilala ay ipinapakita sa profile ng iyong property sa booking engine, na tumutulong sa mga biyahero na makita ang iyong mga parangal nang mabilis.
