Pinapayagan ka ng pahina ng mga user na pamahalaan kung sino ang may access sa iyong property sa Extranet.

Para pamahalaan ang iyong team, i-click ang `Property > Users` mula sa pangunahing nav bar.

## Pag-anyaya sa mga miyembro ng team

Ilagay ang email address at i-click ang `Invite` para magpadala ng imbitasyon. Makakatanggap ang tatanggap ng email na may link para sumali sa iyong property team.

## Pamamahala ng mga role

Ang bawat miyembro ng team ay maaaring magkaroon ng isa sa mga sumusunod na role:

- **Owner** — Buong access sa lahat ng setting ng property, kabilang ang kakayahang pamahalaan ang ibang mga user.
- **Manager** — Access sa araw-araw na mga tampok ng pamamahala ng property.

Maaari mong itaas ang isang manager bilang owner gamit ang button na `Make owner`.

## Pagtanggal ng mga miyembro

I-click ang `Remove` sa tabi ng anumang miyembro ng team para bawiin ang kanilang access. Agad itong magkakabisa.

## Mga nakabinbing imbitasyon

Ipinapakita ng seksyon ng mga imbitasyon ang lahat ng mga outstanding na imbitasyon na hindi pa tinatanggap.
