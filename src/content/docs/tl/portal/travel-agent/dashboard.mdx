Nagbibigay ang Travel Agent dashboard ng mabilisang pangkalahatang-ideya ng iyong pagganap at aktibidad.

## Metrics

Ipinapakita ng dashboard ang mga pangunahing sukatan sa isang sulyap:

- **Bookings** — Kabuuang bilang ng mga booking na iyong na-facilitate.
- **Revenue** — Kabuuang kita na nalikha mula sa iyong mga booking.
- **Upcoming bookings** — Mga booking na may mga darating na petsa ng pag-check-in.

## Activity

Sinusubaybayan ng seksyon ng aktibidad ang iyong kamakailang kasaysayan ng booking at mga trend sa paglipas ng panahon, na tumutulong sa iyo na matukoy ang mga peak na panahon at mga property na may pinakamataas na pagganap.

## Account status

Ipinapakita ng dashboard ang kasalukuyang estado ng iyong travel agent profile — kung kumpleto na ang setup at aktibo na ang mga komisyon.
