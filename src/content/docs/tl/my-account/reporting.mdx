Nagbibigay ang Wink ng sistema ng pag-uulat na nagpapahintulot sa iyo na gumawa at mag-export ng data sa buong iyong account. Ang mga ulat ay available sa format na CSV para sa madaling pag-import sa mga spreadsheet, analytics tools, o sa iyong sariling mga sistema.

## Mga uri ng ulat

Maaari kang gumawa ng mga ulat para sa mga sumusunod na kategorya ng data:

| Uri ng ulat | Ano ang kasama |
|---|---|
| **Property** | Mga detalye ng property, status ng listing, buod ng imbentaryo. |
| **User** | Mga user account, mga papel, buod ng aktibidad. |
| **Booking** | Kasaysayan ng booking, mga status, kita, mga detalye ng bisita. |
| **Account** | Impormasyon ng account, balanse, kasaysayan ng transaksyon. |

## Paano gumawa ng ulat

1. Pumunta sa seksyon ng pag-uulat sa iyong dashboard ng account.
2. Piliin ang **uri ng ulat** na nais mong gawin.
3. I-configure ang anumang **mga filter** upang paliitin ang data (hal., saklaw ng petsa, status, property).
4. Piliin ang **mga field** na nais mong isama sa export. Bawat uri ng ulat ay may set ng mga available na field na maaari mong piliin.
5. I-click ang **Export** upang i-download ang ulat bilang isang CSV file.

## Mga custom na field

Sinusuportahan ng mga ulat ang napapasadyang pagpili ng mga field. Pinipili mo kung aling mga column ang isasama sa iyong export, kaya makukuha mo eksakto ang data na kailangan mo — walang sobra, walang kulang.

## Mga gamit

- **Pagkakasundo ng kita** — I-export ang mga ulat ng booking na may mga detalye sa pananalapi para i-reconcile sa iyong accounting system.
- **Pagsusuri ng performance** — I-export ang mga ulat ng property para ikumpara ang performance ng listing sa iyong portfolio.
- **Mga insight tungkol sa bisita** — I-export ang data ng booking para maunawaan ang demograpiko ng mga biyahero at mga pattern ng booking.
- **Audit ng account** — I-export ang data ng account para sa pagsunod o internal na pagsusuri.
