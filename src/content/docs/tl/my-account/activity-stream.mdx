Ang activity stream ay nagbibigay ng kronolohikal na tala ng mga kaganapan at aksyon sa iyong Wink account. Tinutulungan ka nitong subaybayan ang mga nangyayari sa iyong mga property, booking, at account sa isang lugar.

## Ano ang nasusubaybayan

Kinukuha ng activity stream ang mga pangunahing kaganapan kabilang ang:

- **Booking events** — Mga bagong booking, pagkansela, at pagbabago.
- **Review events** — Pagsusumite at tugon sa mga review ng bisita.
- **Account changes** — Mga update sa profile, pagdagdag ng manager, pagbabago ng status.
- **Content updates** — Mga pagbabago sa paglalarawan ng property, kuwarto, at pasilidad.
- **Media uploads** — Mga larawan at video na idinagdag sa iyong mga listahan.

Bawat entry ng aktibidad ay may kasamang timestamp, ang user na gumawa ng aksyon (kung naaangkop), at paglalarawan ng nangyari.

## Mga antas ng visibility

Ang mga entry ng aktibidad ay may iba't ibang antas ng visibility depende sa uri ng kaganapan:

| Level | Sino ang makakakita |
|---|---|
| **Everyone** | Pampublikong nakikitang aktibidad. |
| **Authenticated** | Nakikita ng sinumang naka-login na user. |
| **Owner** | Nakikita lamang ng user na gumawa ng entry. |
| **Admin** | Nakikita lamang ng mga administrador. |

Makikita mo lamang ang mga entry ng aktibidad na naaayon sa iyong antas ng access.

## Paano tingnan ang iyong activity stream

Pumunta sa iyong account dashboard upang makita ang iyong activity stream. Ipinapakita ang mga kaganapan sa reverse chronological order — ang pinakabagong aktibidad ang unang lumalabas.

## Mga gamit

- **Audit trail** — Tingnan kung sino ang gumawa ng mga pagbabago sa iyong property o account, at kailan.
- **Booking monitoring** — Subaybayan ang mga bagong booking at pagkansela nang real time.
- **Team coordination** — Kapag maraming manager ang nagtatrabaho sa isang property, ipinapakita ng activity stream ang mga aksyon ng lahat.
- **Review management** — Tingnan kung kailan dumating ang mga bagong review upang makapagbigay ka ng agarang tugon.
