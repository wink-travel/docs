Maaaring piliin ng isang Wink user na hayaan kang pamahalaan ang isa sa kanilang property, affiliate, o travel agent accounts. Kapag nangyari iyon, makakatanggap ka ng e-mail na may kahilingan ng imbitasyon.

import { Steps } from '@astrojs/starlight/components';

Upang makita ang iyong kasalukuyang mga imbitasyon sa Wink, sundin ang mga hakbang na ito:

<Steps>
 1. Mag-log in sa [my.wink.travel](https://my.wink.travel).
 2. I-click ang `Invites` mula sa kaliwang sidebar.
 3. Makikita mo ang listahan ng lahat ng kasalukuyang imbitasyon.
 4. Ililista ng seksyong ito ang lahat ng iyong mga aplikasyon.
 5. Piliin ang `Accept` o `Reject` ang imbitasyon.
 6. Kung tinanggap mo ang imbitasyon, ipapakita ang account sa seksyong `Accounts`.
</Steps>
