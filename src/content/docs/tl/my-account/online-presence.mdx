Dapat punan ng lahat ang kanilang presensya sa online. Pinapayagan ng presensya sa online ang mga user na mahanap ang mga hotel at ang mga hotel na mahanap ang mga affiliate.

<Steps>
1. **Type** I-type ang uri ng presensya sa online. *hal. Facebook*
2. **Account** Ipasok ang URL ng iyong account. *hal. https://facebook.com/wink2travel*
3. **Audience size** Ipasok ang laki ng iyong audience. *hal. Maaaring ito ang iyong mga tagasubaybay o bilang ng mga empleyado*
4. **Audience type** Ipasok ang uri ng audience na mayroon ka. *hal. Pamilya dahil ang iyong travel blog ay nakatuon sa paglalakbay ng pamilya*
</Steps>
