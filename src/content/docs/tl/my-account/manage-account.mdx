Kapag nakagawa ka na ng account, narito ang mga bagay na maaari mo pang gawin dito habang nasa [my.wink.travel](https://my.wink.travel) ka.

- I-update ang kinatawan ng account
- I-update ang impormasyon ng account
- I-update ang mga natatanging system ID *paraan para mahanap ang iyong account sa buong sistema*
- I-update ang address
- I-update ang larawan ng profile *[ipinapakita sa WinkLinks at kapag kumokonekta sa mga property at affiliate sa Wink]*
- I-update ang logo *[ipinapakita sa mga e-mail]*

Maaari mo ring...

- Pamahalaan ang iyong [online presence](/tl/my-account/online-presence).
- Magtalaga ng [dedikadong kinatawan ng ahensya](/tl/my-account/agency-representation) para pamahalaan ang iyong account [kapalit ng bayad].
- Magtalaga ng mga [manager](/tl/my-account/managers) na nais mong magkaroon ng access sa iyong account.
- Pamahalaan ang [customizations](/tl/my-account/customizations) upang matulungan kang kontrolin ang karanasan ng gumagamit kapag tinitingnan ang iyong imbentaryo.
- Pamahalaan ang [applications](/tl/my-account/applications) upang mabigyan ka ng access sa Wink bilang developer at upang magdagdag / palawakin ang iyong sariling mga tampok sa Wink.
- Pamahalaan ang [webhooks](/tl/my-account/webhooks) upang makatanggap ng mga abiso tungkol sa mga kaganapan sa platform ng Wink habang nangyayari ito nang real-time. *Halimbawa: Makatanggap ng abiso ng bagong booking*.
