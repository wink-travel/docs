Kung nais mong palitan ang iyong password, o nakalimutan mo ang iyong password, sundin ang mga hakbang na ito:

## I-update ang password

<Steps>
1. Pumunta sa [my.wink.travel](https://my.wink.travel).
1. Mag-log in sa pamamagitan ng pag-click sa `Sign-in / Register` sa kanang itaas na bahagi.
2. I-click ang iyong profile icon, sa kanang itaas na bahagi at lalabas ang isang menu.
3. I-click ang link na `Change password`.
4. Sundin ang mga tagubilin upang palitan ang iyong password.
5. I-click ang `Change password` upang magpatuloy.
</Steps>

Sa susunod na pag-log in mo, kakailanganin mo ang iyong bagong password.

## Nakalimutan ang password

Minsan kinakailangan na i-reset ang password ng user.

Narito ang mga hakbang para i-reset ang iyong password:

<Steps>
1. I-click ang link na `Forgot password` mula sa sign-in page.
2. I-type ang username / e-mail para sa account na nais mong i-reset.
3. I-click ang `Forgot password` button. *Ikaw ay ire-redirect sa reset password form.*
4. Suriin ang iyong e-mail.
</Steps>

2. Kumpirmahin ang iyong bagong password.
3. I-click ang `Reset password` button. *Ikaw ay ire-redirect pabalik sa sign-in page.*
</Steps>
