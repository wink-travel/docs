Ang booking engine ay may dalawang uri upang suportahan ang iba't ibang pangangailangan ng isang kaakibat, isang kumpanya ng pamamahala ng ari-arian at isang hotel.

## Online na ahensya ng paglalakbay

Para sa isang kaakibat, pinaka-makatuwirang gamitin ang bersyon ng online na ahensya sa paglalakbay. Ang dahilan ay ang site ay naka-set up upang mapaunlakan ang mga manlalakbay na gustong tumingin ng maraming property.

Maaari mong mahanap ang homepage para sa application na ito sa ibaba.

</LinkButton>

## Configuration

Kadalasan, hindi na kailangang direktang ibahagi ang alinman sa mga application na ito. Gumawa kami ng iba't ibang paraan para makapagbahagi ka ng imbentaryo ng paglalakbay, kung saan mo matutuklasan [Wink Studio](/studio/what-is-studio) o sa pamamagitan ng [WinkLinks](/link-manager/wink-links).

Gayunpaman, kung gusto mong manu-manong mag-link sa isa sa mga website na ito, tiyaking idagdag mo ang iyong client ID at customization ID sa URL bilang mga parameter.

Mahahanap mo ang iyong client ID sa pamamagitan ng:

<Steps>
  1. Mag-log in sa alinman [Wink Extranet](/extranet/what-is-extranet) o [Wink Studio](/studio/what-is-studio).
  2. Mag-click sa icon ng iyong profile sa kanang sulok sa itaas at mag-click sa [Mga aplikasyon](/developers/applications).
  3. Gamitin ang application na nakatali sa Wink account na gusto mong gamitin.
  4. Kopyahin ang Client ID mula sa application na iyon at i-paste ito sa URL bilang parameter ng kahilingan. Halimbawa: ?client-id=MY\_CLIENT\_ID
</Steps>


### Extranet

Mahahanap mo ang iyong customization ID bilang property sa pamamagitan ng:

<Steps>
  1. Mag-log in [Wink Extranet](/extranet/what-is-extranet).
  2. Piliin ang property na gusto mong magtrabaho.
  3. Mula sa pangunahing navigation bar, i-click `Account > Branding`
  4. Kopyahin ang Customization ID at i-paste ito sa URL bilang parameter ng kahilingan. Halimbawa: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Studio

Mahahanap mo ang iyong customization ID bilang isang affiliate sa pamamagitan ng:

<Steps>
  1. Mag-log in [Wink Studio](/studio/what-is-studio).
  2. Piliin ang account na gusto mong gamitin.
  3. Mula sa pangunahing navigation bar, i-click `Tools > Customization`
  4. Kopyahin ang Customization ID mula sa customization na gusto mong gamitin at i-paste ito sa URL bilang parameter ng kahilingan. Halimbawa: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

Ang napupunta sa iyo ay isang URL na mukhang ganito:

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Kung gusto mong direktang pumunta sa isang page ng hotel, magiging ganito ang hitsura ng URL:

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::tandaan
Kung hindi ka magsasama ng client at / o configuration ID, ang Wink OTA ay magiging default sa isang default na configuration.
:::

## IBE

Kung isa kang kumpanya sa pamamahala ng ari-arian, gusto mong mag-link sa homepage ng Wink IBE. Ipapakita ng homepage ang iyong itinalagang tema kasama ang [mga ari-arian na iyong kinakatawan](/studio/agency-representation).

Ang napupunta sa iyo ay isang URL na mukhang ganito:

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Kung isa kang property, gusto mong direktang mag-link sa landing page ng property ng Wink IBE.

Magiging ganito ang hitsura ng URL na iyon:

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::tandaan
Kung hindi ka magsasama ng client at / o configuration ID, HINDI maglo-load ang Wink IBE. Ang mga parameter na ito ay kinakailangan.
:::

## Halimbawa: Pahina ng ari-arian

Ang bawat property sa Wink ay nakakakuha ng sarili nilang natatanging landing page sa alinmang website. Ang lahat ng panlabas na link ay nagdidirekta sa mga manlalakbay sa isa sa mga landing page na ito.

Ang landing page ay naglalaman ng lahat ng impormasyong kailangan ng isang manlalakbay upang makagawa ng isang pinag-aralan na pagpipilian kung mananatili sa property na ito.
