Ang booking engine ay may dalawang uri upang suportahan ang iba't ibang pangangailangan ng isang affiliate, isang kumpanya ng pamamahala ng ari-arian, at isang hotel.

## Online travel agency

Para sa isang affiliate, mas makatuwiran na gamitin ang bersyon ng online travel agency. Ang dahilan ay ang site ay nakaayos upang mag-accommodate ng mga biyahero na nais tingnan ang maraming ari-arian.

Maaari mong makita ang homepage para sa aplikasyon na ito sa ibaba.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Internet booking engine

Para sa isang hotel o isang kumpanya ng pamamahala ng ari-arian, mas makatuwiran na gamitin ang bersyon ng internet booking engine. Ang ganitong uri ng gumagamit ay interesado sa pag-highlight ng isang ari-arian o maraming ari-arian sa ilalim ng pamamahala.

Maaari mong makita ang homepage para sa aplikasyon na ito sa ibaba.

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE
</LinkButton>

## Konfigurasyon

Kadalasan, hindi na kailangang direktang ibahagi ang alinman sa mga aplikasyon na ito. Nilikha namin ang iba't ibang paraan para maibahagi mo ang travel inventory, na maaari mong tuklasin sa [Wink Studio](/studio/what-is-studio) o sa pamamagitan ng [WinkLinks](/link-manager/wink-links).

Gayunpaman, kung nais mong manu-manong i-link ang isa sa mga website na ito, siguraduhing idagdag ang iyong client ID at customization ID sa URL bilang mga parameter.

Maaari mong makita ang iyong client ID sa pamamagitan ng:

<Steps>
1. Mag-log in sa alinman sa [Wink Extranet](/extranet/what-is-extranet) o [Wink Studio](/studio/what-is-studio).
2. I-click ang iyong profile icon sa kanang itaas na sulok at i-click ang [Applications](/developers/applications).
3. Gamitin ang aplikasyon na naka-link sa Wink account na nais mong gamitin.
4. Kopyahin ang Client ID mula sa aplikasyon na iyon at i-paste ito sa URL bilang request parameter. Halimbawa: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Maaari mong makita ang iyong customization ID bilang isang ari-arian sa pamamagitan ng:

<Steps>
1. Mag-log in sa [Wink Extranet](/extranet/what-is-extranet).
2. Piliin ang ari-arian na nais mong gamitin.
3. Mula sa pangunahing navigation bar, i-click ang `Account > Branding`
4. Kopyahin ang Customization ID at i-paste ito sa URL bilang request parameter. Halimbawa: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Maaari mong makita ang iyong customization ID bilang isang affiliate sa pamamagitan ng:

<Steps>
1. Mag-log in sa [Wink Studio](/studio/what-is-studio).
2. Piliin ang account na nais mong gamitin.
3. Mula sa pangunahing navigation bar, i-click ang `Tools > Customization`
4. Kopyahin ang Customization ID mula sa customization na nais mong gamitin at i-paste ito sa URL bilang request parameter. Halimbawa: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Ang makukuha mo ay isang URL na ganito ang hitsura:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Kung nais mong direktang pumunta sa isang pahina ng hotel, ang URL ay magiging ganito ang hitsura:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Kung hindi mo isasama ang client at/o configuration ID, ang Wink OTA ay magde-default sa isang default na konfigurasyon.
:::

## IBE

Kung ikaw ay isang kumpanya ng pamamahala ng ari-arian, nais mong i-link ang Wink IBE homepage. Ipapakita ng homepage ang iyong itinalagang tema kasama ang [mga ari-arian na iyong kinakatawan](/studio/agency-representation).

Ang makukuha mo ay isang URL na ganito ang hitsura:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Kung ikaw ay isang ari-arian, nais mong direktang i-link ang landing page ng ari-arian sa Wink IBE.

Ang URL na iyon ay magiging ganito ang hitsura:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
Kung hindi mo isasama ang client at/o configuration ID, HINDI maglo-load ang Wink IBE. Kinakailangan ang mga parameter na ito.
:::

## Halimbawa: Pahina ng ari-arian
