Ang pahina ng paghahanap ay kung saan nagba-browse ang mga manlalakbay ng mga property ayon sa lokasyon.

## Pagba-browse ayon sa lokasyon

Maaaring mag-browse ng mga property ayon sa uri ng lokasyon — mga lungsod, isla, bansa, at iba pa. Bawat pahina ng lokasyon ay nagpapakita ng grid ng mga available na hotel kasama ang presyo.

Bawat card ng property ay nagpapakita ng:

- Pangalan ng hotel at pangunahing larawan.
- Lokasyon.
- Pinakamababang available na presyo.
- Rating ng mga bisita.

## Tagapili ng itineraryo

Maaaring pinuhin ng mga manlalakbay ang kanilang paghahanap gamit ang tagapili ng itineraryo sa itaas ng pahina:

- **Mga petsa ng Check-in / Check-out** — Itakda ang mga petsa ng paglalakbay.
- **Mga bisita** — Bilang ng mga matatanda at bata.
- **Promo code** — Mag-apply ng promotional code para sa mga diskwento.

Ang grid ng property ay nag-a-update nang real time habang nagbabago ang itineraryo.

## Walang resulta

Kung walang property na tumutugma sa mga pamantayan ng paghahanap, isang kapaki-pakinabang na mensahe ang ipinapakita na nagmumungkahi sa manlalakbay na baguhin ang kanilang mga petsa o lokasyon.
