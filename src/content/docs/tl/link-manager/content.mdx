Ang iyong nilalaman ay magagamit sa ilalim ng `Entries` tab.

Sinusuportahan namin ang tatlong pangunahing uri ng nilalaman:

* Mga URL na available sa publiko na nagli-link sa mga web page.
* Mga file na na-upload sa pamamagitan ng WinkLinks.
* Ang nilalaman ng teksto na nilikha sa pamamagitan ng aming `Quotes` tampok.

## Magdagdag ng nilalaman

Narito kung paano ka magdagdag ng nilalaman sa WinkLinks. Sa ilalim ng `Entries` tab, magsimula sa pamamagitan ng pag-click sa `Add entry` pindutan.

### Normal na link

:::tip
Ang link ay kailangang pumunta sa isang web page. Halimbawa, hindi ka maaaring direktang mag-link sa isang PDF. Kung gusto mong magbahagi ng PDF, kailangan mong gamitin ang `Upload file` tampok.
:::

Narito ang mga hakbang upang magdagdag ng normal na link:

<Steps>
  1. Ilagay ang link na gusto mong ibahagi. *hal. https://trvl.as/3xWCH*
  2. Kapag naipasok mo na ang link, lalabas si Wink para hanapin ang mga detalye ng website na ito at ipo-populate ang form na makikita mo sa susunod.
  3. Maaari mong piliing baguhin ang ilan sa data, gaya ng pangalan ng site, pamagat at paglalarawan.
  4. Maaari mo ring piliing mag-upload ng custom na larawan.
  5. I-click ang `Save` button kapag handa ka nang magpatuloy.
</Steps>


### Mag-upload ng file

:::pag-iingat\[Requirements]

* Hindi maaaring lumampas sa 10Mb ang laki ng mga larawan.
* Hindi maaaring lumampas sa 50Mb ang laki ng mga video.
* Sinusuportahan namin ang lahat ng kinikilalang format ng larawan at video, kabilang ang PDF.
  :::

:::note\[Recommendations]

* 2560w x 1440h (16x9 aspect ratio) para sa iyong itinatampok na larawan.
* 2560w x 1920h (4:3 aspect ratio) para sa lahat ng iba pang larawan.
* Hindi dapat mas malaki sa 5Mb ang laki ng larawan. Paki-compress ang iyong mga larawan bago i-upload sa Wink.
  :::

Narito ang mga hakbang sa pagdaragdag ng file:

<Steps>
  1. Maaaring idagdag ang media:
     * Sa pamamagitan ng pag-drag ng mga larawan mula sa iyong computer papunta sa window o pag-click `Browse`.
     * Sa pamamagitan ng pag-paste ng panlabas na URL na tumuturo sa isang larawan.
     * Sa pamamagitan ng paggamit ng iyong laptop camera.
     * Mula sa iyong Google Drive account.
     * Mula sa iyong Dropbox account.
     * Mula sa Shutterstock
     * Mula sa gettyimages
     * Mula sa iStock
     * Mula sa Unsplash
  2. Kapag natapos na ang pag-upload ng file, maaari kang magdagdag ng higit pang metadata upang ilarawan ang iyong file gaya ng pamagat at paglalarawan.
  3. I-click ang `Save` button kapag handa ka nang magpatuloy.
</Steps>


### Sumulat ng isang quote

:::tip
Ang isang quote ay isang mahusay na paraan upang magdagdag ng isang delimiter upang masira ang isang mahabang listahan \[...at ito ay gumagawa sa iyo tunog matalino at alam masyadong ðŸ˜‰].
:::

Narito ang mga hakbang sa pagsulat ng isang quote:

<Steps>
  1. I-type ang quote *hal. Ang mga rosas ay pula. Ang mga violet ay asul...*.
  2. Ilagay ang pangalan ng orihinal na may-akda. *hal. Edmund Spenser*
  3. Ilagay ang pangalan ng site kung saan mo nakuha ang quote. *hal. Pang-araw-araw na Quote*
  4. I-click ang `Save` button kapag handa ka nang magpatuloy.
</Steps>


## I-update ang nilalaman

Ang lahat ng nilalaman ay madaling mabago sa pamamagitan ng pag-click sa `Actions` link at pagpili `Update`. Maaari mong alisin ang nilalaman sa katulad na paraan sa pamamagitan ng pag-click sa `Remove` pindutan.

## Alisin ang nilalaman

Ang lahat ng nilalaman ay madaling mabago sa pagpili `Update`. Maaari mong alisin ang nilalaman sa katulad na paraan sa pamamagitan ng unang pag-click sa `Actions` link at pagkatapos ay i-click ang `Remove` pindutan.

## Pag-uuri ng nilalaman

Maaari mong i-order ang iyong nilalaman sa pamamagitan ng pag-click sa icon na matatagpuan sa ibabang kaliwang sulok ng bawat entry at i-drag ito sa posisyon na gusto mo.

## Pagpapangkat ng nilalaman

:::tip
Lumikha ng iyong mga tag bago ka gumawa ng iyong mga entry. Kung hindi, mawawala ang iyong pag-unlad at kailangang magsimulang muli.
:::

Kapag nag-e-edit ka ng isang entry, mayroon kang opsyon na magdagdag ng isa o higit pang mga tag para sa mga layunin ng pagpapangkat. Mag-scroll sa ibaba ng form upang mahanap ang `Tags` field ng form.

Ang mga gumagamit, na tumitingin sa iyong pahina ng WinkLinks, ay makakapili kung paano nila gustong ubusin ang iyong nilalaman (alinman sa listahan o grid) at maaaring pumili, isa o higit pa, mga tag upang higit pang i-filter ang iyong nilalaman.
