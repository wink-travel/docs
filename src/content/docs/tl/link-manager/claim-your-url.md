Sa pamamagitan ng paggawa ng isang [affiliate account](/tl/studio/what-is-studio), mayroon ka nang WinkLinks vanity url. Ang mga tampok ng WinkLinks publisher ay matatagpuan sa aming Studio app; kasama ng iba pa naming mga affiliate features.

### Halimbawa

Kung ang pangalan ng iyong account ay Queen Bee, ang iyong WinkLinks vanity url ay magiging:

<h4 align="center">https://i.trvl.as/queen-bee</h4>

Ang mga pangalan ng account ay available sa `first come - first serve` na batayan - Kunin ang iyong natatanging URL habang maaari pa!
