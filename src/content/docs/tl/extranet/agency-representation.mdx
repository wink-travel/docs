Ang ahensya ay isang normal na Wink account na maaari mong italaga sa iyong account para sa mga layunin ng representasyon. Nasa ibaba ang ilang dahilan kung bakit gusto mo ng ahensyang kumakatawan sa iyo:

* Buong / Bahagyang pamamahala ng iyong Wink account ng 3rd party.
* May umiiral nang kasunduan ang Property sa kanilang kumpanya ng pamamahala at nais ni Wink na pangasiwaan ang aspetong pinansyal ng kanilang pag-aayos ng mga pondong pumapasok sa pamamagitan ng Wink.
* Ang ahensya sa pamamahala ng ari-arian ay gumagawa ng lahat ng online na trabaho sa ngalan ng mga ari-arian nito.

## Magtalaga ng ahensya

Narito kung paano ka magtatalaga ng ahensya sa iyong account.

<Steps>
  1. Mag-log in sa iyong user account sa [Wink Extranet](https://extranet.wink.travel).
  2. Piliin ang property na gusto mong magtrabaho.
  3. Mag-click sa `Account > Managers` mula sa pangunahing nav bar.
  4. Sa `Agency` seksyon, i-type ang pangalan ng ahensya na gusto mong katawanin.
  5. Piliin ang bayad sa ahensya mula sa dropdown.
  6. Opsyonal na magdagdag ng mga panuntunan sa pag-expire.
  7. I-click ang `Set agency` pindutan upang magpatuloy.
</Steps>


:::tandaan
Kailangang ikaw ang may-ari ng property para magtalaga ng ahensya.
:::

:::tip\[Halimbawa]
Ang bayad sa ahensya ay nakukuha mula sa booking.

Pag-book = $100, Bayad sa ahensya = 1% => 100 \* 0.01 = $1
:::

## Alisin ang pagkakatalaga ng ahensya

Narito kung paano mo aalisin ang pagkakatalaga ng ahensya sa iyong account.

<Steps>
  1. Mag-log in sa iyong user account sa [Wink Extranet](https://extranet.wink.travel).
  2. Piliin ang property na gusto mong magtrabaho.
  3. Mag-click sa `Account > Managers` mula sa pangunahing nav bar.
  4. Sa `Agency` seksyon, i-click ang <Badge text="x" variant="danger" size="small" /> icon sa kanang bahagi ng dropdown ng ahensya.
  5. I-click ang `Set agency` pindutan upang magpatuloy.
</Steps>


## API

Maaaring magtungo ang mga developer na gustong pamahalaan ang ahensya sa pamamagitan ng API [Mga Developer > API > Kaakibat](/developers/apis/#affiliate-api).
