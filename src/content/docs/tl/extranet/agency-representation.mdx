Ang ahensya ay isang normal na Wink account na maaari mong italaga sa iyong account para sa layunin ng representasyon. Narito ang ilang mga dahilan kung bakit nais mong magkaroon ng ahensya na nagrerepresenta sa iyo:

- Buo / Bahagyang pamamahala ng iyong Wink account ng isang 3rd party.
- Ang property ay may umiiral na kasunduan sa kanilang management company at nais na hawakan ng Wink ang pinansyal na aspeto ng kanilang kasunduan sa mga pondong pumapasok sa pamamagitan ng Wink.
- Ginagawa ng property management agency ang lahat ng online na gawain para sa kanilang mga property.

## Magtalaga ng ahensya

Narito kung paano mo itatalaga ang isang ahensya sa iyong account.

<Steps>
 1. Mag-log in sa iyong user account sa [Wink Extranet](https://extranet.wink.travel).
 2. Piliin ang property na nais mong trabahuhin.
 3. I-click ang `Account > Managers` mula sa pangunahing nav bar.
 4. Sa seksyong `Agency`, i-type ang pangalan ng ahensya na nais mong mag-representa sa iyo.
 5. Piliin ang bayad sa ahensya mula sa dropdown.
 5. Opsyonal na magdagdag ng mga patakaran sa pag-expire.
 7. I-click ang button na `Set agency` upang magpatuloy.
 </Steps>

 :::note
Kailangan mong maging may-ari ng property upang makapag-assign ng ahensya.
 :::

 :::tip[Halimbawa]
Ang bayad sa ahensya ay nagmumula sa booking.

Booking = $100, Bayad sa ahensya = 1% => 100 * 0.01 = $1
 :::

## Alisin ang ahensya

Narito kung paano mo aalisin ang ahensya mula sa iyong account.

<Steps>
 1. Mag-log in sa iyong user account sa [Wink Extranet](https://extranet.wink.travel).
 2. Piliin ang property na nais mong trabahuhin.
 3. I-click ang `Account > Managers` mula sa pangunahing nav bar.
 4. Sa seksyong `Agency`, i-click ang <Badge text="x" variant="danger" size="small" /> icon sa kanang bahagi ng dropdown ng ahensya.
 5. I-click ang button na `Set agency` upang magpatuloy.
 </Steps>


## API

Ang mga developer na nais pamahalaan ang ahensya gamit ang API ay maaaring pumunta sa [Developers > API > Affiliate](/tl/developers/apis/#affiliate-api).
