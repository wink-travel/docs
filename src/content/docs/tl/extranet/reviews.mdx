Ipinapakita ng pahina ng mga review ang lahat ng mga review ng bisita na isinumite para sa iyong property.

Upang makita ang iyong mga review, i-click ang `Distribution > Reviews` mula sa pangunahing nav bar.

## Pag-browse ng mga review

Ang listahan ng review ay maaaring i-filter upang matulungan kang hanapin ang partikular na feedback:

- **Saklaw ng petsa** — I-filter ayon sa petsa kung kailan naisumite ang review.
- **Pangalan ng bisita** — Maghanap gamit ang unang pangalan o apelyido.
- **Email ng bisita** — Hanapin ang mga review gamit ang email address.
- **Nasagot** — I-filter kung nasagot mo na ang review.

Ang mga resulta ay naka-pahina na may 25 review bawat pahina.

## Mga detalye ng review

I-click ang anumang review upang makita ang buong detalye, kabilang ang nakasulat na feedback at mga rating ng bisita. Gamitin ito upang maunawaan ang damdamin ng bisita at matukoy ang mga lugar na maaaring pagbutihin.
