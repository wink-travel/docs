Ang profile ng ari-arian ay isang kinakailangang hakbang sa aming onboarding wizard at nagbibigay sa amin ng pagkakataon na mas makilala ang iyong ari-arian gamit ang mga pamantayan, tulad ng:

- Bilang ng mga bituin.
- Uri ng hotel.
- Uri ng lokasyon.
- ...at iba pa

Upang pamahalaan ang iyong profile ng ari-arian, i-click ang `Property > Profile` sa pangunahing navigation tab.

I-update ang form at i-click ang `Save` na button upang magpatuloy.
