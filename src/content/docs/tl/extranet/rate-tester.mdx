Pinapayagan ka ng tagasubok ng rate na tiyakin na tama ang pagkakaayos ng iyong mga presyo ng kuwarto bago ito maging live sa mga biyahero.

Para ma-access ang tagasubok ng rate, i-click ang `Monetize > Rate Tester` mula sa pangunahing nav bar.

## Paano ito gumagana

Ilagay ang isang search scenario at ipapakita ng tagasubok ng rate kung paano eksaktong kinakalkula ang mga presyo:

- **Sales channel** — Piliin ang channel na susubukan (hal., iyong booking engine, isang affiliate, o isang partikular na integration).
- **Mga petsa ng Check-in / Check-out** — Ang mga petsa ng paglalakbay na susubukan.
- **Kuwarto na pagsasaayos** — Bilang ng mga matatanda at bata.

I-click ang `Search` para makita ang mga resulta.

## Mga Resulta

Ipinapakita ng pahina ng mga resulta ang mga kinakalkulang presyo para sa bawat uri ng kuwarto at rate plan na tumutugma sa iyong mga search criteria. Gamitin ito upang tiyakin na:

- Ang mga modifier sa presyo (single occupancy, dagdag na matatanda/bata) ay naiaaplay nang tama.
- Ang mga patakaran sa pagkansela ay ipinapakita ayon sa inaasahan.
- Ang mga restriksyon ay tama ang pagsasala.
- Ang mga promosyon at diskwento ay epektibo.
