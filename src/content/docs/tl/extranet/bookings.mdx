Kapag may nag-book sa Wink, ang mga sumusunod ang nangyayari:

<Steps>
1. Naabisuhan ang iyong channel manager, CRS o PMS.
2. Makakatanggap ka ng confirmation email *(mga contact ng reservation desk)* na may link para pamahalaan ang iyong bagong booking.
3. Makakatanggap ang biyahero ng confirmation email na may kasamang impormasyon kung paano makipag-ugnayan sa iyong reservation desk.
4. Maaaring makinig ang mga developer sa `booking.created` webhook event at makatanggap ng detalyadong impormasyon ng booking mula sa amin nang realtime.
</Steps>

Sundin ang link sa email, o pumunta sa `Account > Bookings` mula sa pangunahing nav bar upang magsimula.

## Booking grid

Ipinapakita ng booking grid ang lahat ng iyong mga nakaraang at paparating na booking, nakaayos ayon sa petsa ng booking mula sa pinakamaaga.

### Mga Filter

Maaari kang maglagay ng mga filter upang mahanap ang mga booking na iyong hinahanap.

- Petsa ng booking [saklaw]
- Petsa ng pagdating [saklaw]
- Petsa ng pag-alis [saklaw]
- Booking code
- Apelyido ng biyahero
- Pangalan ng biyahero
- Pangalan ng master rate
- Sales channel / Pinagmulan

### Pag-export

Maaari mong i-export ang mga booking na tumutugma sa iyong kasalukuyang filter criteria sa CSV.

## Booking

Piliin ang booking na nais mong trabahuhin mula sa booking grid at ikaw ay ire-redirect sa pahina ng `Booking`.
Nasa pahinang ito ang lahat ng detalye na kailangan mong malaman upang maayos na tanggapin ang iyong bisita sa araw ng pagdating.

Nasa pahina rin ang ilang mahahalagang aksyon na kailangan mo upang maayos na mapagsilbihan ang iyong mga bisita. Makikita mo ang lahat ng magagamit na aksyon sa `Actions` dropdown sa itaas ng pahina.

### Muling pagpapadala ng e-mail

Para muling ipadala ang confirmation email sa biyahero, i-click ang `Actions > Resend confirmation email`.

### Kanselahin ang booking

Para kanselahin ang booking bilang property, i-click ang `Actions > Cancel booking`.

Lilitaw ang isang pop-up window na magpapahintulot sa iyo na:

<Steps>
1. Piliin ang uri ng pagkansela. *hal. Mali ang mga petsa*
2. I-type ang dahilan ng pagkansela.
</Steps>

*Ang button na ito ay sumusunod sa iyong mga patakaran sa pagkansela kaya hindi ito palaging magagamit.*

### Humiling ng refund

Para humiling ng refund, sa anumang dahilan, i-click ang `Actions > Request refund`.

Lilitaw ang isang pop-up window na magpapahintulot sa iyo na:

<Steps>
1. Humiling ng buong o bahagi ng refund.
    - Kung pinili mo ang bahagi, hihingin nito ang halaga.
2. Piliin ang dahilan.
3. I-type ang isang detalyadong dahilan para sa kahilingan.
4. Sabihin sa amin kung nais mo ring kanselahin ang booking, kahit na aprubado o hindi ang kahilingan.
</Steps>

 *Magagamit ang aksyong ito hanggang sa maproseso ang pondo para sa booking na ito.*.

## Calendar sync

Para simulan ang pagsi-sync ng iyong mga booking sa Wink sa iyong paboritong calendar software, pumunta sa `Distribution > Sync bookings to your calendar` mula sa pangunahing nav bar upang magsimula.

Sa unang pagkakataon, hihilingin kang gumawa ng pass code na maaari mong gamitin sa iyong calendar software upang i-authenticate ang koneksyon.

I-click ang `Yes` na button para magpatuloy o 'No' kung nagbago ang isip mo.

Makikita mo ang 3 patlang na kailangan mong gamitin upang ikonekta ang Wink sa iyong calendar software:

1. Ang URL na gagamitin para mag-subscribe sa mga booking.
2. Ang username para sa authentication.
3. Ang pass key para sa authentication.

Kapag natapos mo na ang hakbang na ito at naidagdag ang CalDAV URL sa iyong calendar software, magsisimulang ipakita ng iyong calendar ang mga booking mula sa Wink.
