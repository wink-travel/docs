Hindi pareho ang presyo araw-araw. Ang mga kaganapan, pista opisyal, off-peak na mga panahon, at mga last-minute na puwang ay nangangailangan ng iba't ibang pagpepresyo. Ipinapakita ng gabay na ito kung paano gamitin ang mga espesyal na presyo at mga limitasyon sa rate plan upang hawakan ang mga sitwasyong ito.

## Pagpepresyo para sa Kaganapan

Kapag ang isang kumperensya, pista, o lokal na kaganapan ay nagtutulak ng demand, nais mong itaas ang mga presyo para sa mga petsang iyon.

<Steps>
1. Pumunta sa `Monetize > Special Rates` sa iyong Extranet.
2. I-click ang `Create a new special rate`.
3. Piliin ang master rate at uri ng kuwarto na papalitan.
4. Itakda ang saklaw ng petsa upang masakop ang panahon ng kaganapan.
5. Ipasok ang naayos (mas mataas) na pang-araw-araw na presyo.
6. I-click ang `Save`.
</Steps>

Pinapalitan ng espesyal na presyo ang master rate para lamang sa mga petsang iyon. Nanatiling hindi nagbabago ang iyong regular na pagpepresyo sa labas ng panahon ng kaganapan.

## Mga Pagbabago sa Panahon

Para sa mas mahabang panahon (mataas na season, shoulder season, mababang season), maaari kang:

- **Gumamit ng mga espesyal na presyo** — Gumawa ng mga override para sa bawat season na may naayos na pagpepresyo.
- **I-update ang mga master rate** — Ayusin nang direkta ang mga base rate para sa mga panahong iyon.

Mas mainam ang mga espesyal na presyo para sa pansamantalang mga pagbabago, habang ang pag-update ng mga master rate ay makatuwiran para sa mga predictable na taunang pattern.

## Mga Last-minute na Presyo

Para mapuno ang mga hindi nabentang kuwarto malapit sa petsa ng pag-check-in:

<Steps>
1. Gumawa ng bagong **rate plan** na may advance booking restriction (hal., dapat mag-book hindi lalampas sa 3 araw bago).
2. Ipares ito sa isang **special rate** na nag-aalok ng diskwento para sa mga target na petsa.
3. Ang rate plan na ito ay lalabas lamang sa mga biyahero na nagbu-book sa loob ng last-minute na window.
</Steps>

## Blackout Dates

Para i-block ang ilang mga petsa mula sa isang partikular na rate plan (hal., walang promotional rates sa panahon ng peak season):

<Steps>
1. Pumunta sa `Monetize > Rate Plans` at piliin ang rate plan.
2. Pumunta sa seksyong **Restrictions**.
3. Magdagdag ng **Sell date** restriction upang limitahan kung kailan maaaring i-book ang rate.
4. O gumamit ng **Stay date** restrictions upang limitahan kung kailan saklaw ng rate ang pananatili.
</Steps>

Halimbawa, maaari mong limitahan ang iyong "Early Bird" rate plan upang hindi ito magamit para sa mga booking sa Disyembre (peak season), habang nananatiling bukas ang iyong regular na rate plan.

## Pagsasama-sama ng mga Limitasyon

Maaaring pagsamahin ang mga limitasyon para sa mas tumpak na kontrol:

| Restriction | Ano ang kinokontrol nito |
| -- | -- |
| **Sell dates** | Kung kailan maaaring i-book ang rate |
| **Stay dates** | Aling mga petsa ng pananatili ang saklaw ng rate |
| **Available days** | Aling mga araw ng linggo lumalabas ang rate |
| **Arrival days** | Aling mga araw dapat dumating ang bisita |
| **Departure days** | Aling mga araw dapat umalis ang bisita |
| **Required days** | Dapat manatili ang bisita sa kahit isa sa mga araw na ito |
| **Advance booking** | Gaano katagal bago ang booking dapat gawin |
| **Length of stay** | Minimum at maximum na bilang ng gabi |
