Ang **MAG-BOOK NA** Ang button ay isang ubiqutous na termino sa industriya ng hospitality na tumutukoy sa sariling pagsasama ng hotel sa isang provider na nakakaalam tungkol sa availability at mga presyo ng property at maaaring hayaan ang mga manlalakbay na mag-book ng kuwarto. Ang button ay ang pinakasimpleng paraan para makapagbenta ang mga hotel ng mga kuwarto sa sarili nilang website at sa sarili nilang mga tuntunin.

*Maraming hotel ang gumagamit ng bundle na booking engine na inaalok ng kanilang channel manager. Inirerekomenda namin na ang mga hotel ay magsaliksik at maghambing ng mga native booking engine bago gumawa sa alinmang partikular.*

## Madla

Ang artikulong ito ay pangunahin para sa mga hotel na gustong mag-install ng aming libre, mataas na conversion, booking engine sa kanilang website.
Ngunit... ito rin ay kapaki-pakinabang na pagbabasa para sa sinumang gustong gamitin ang aming mga teknolohiya sa kanilang mga website.

## sa pamamagitan ng Link

Para sa pinakasimpleng pagsasama, inirerekomenda namin ang paggamit ng a [Naibabahaging Link](/studio/shareable-links).

Narito ang mga hakbang para gawin ito:

<Steps>
  1. Pumunta sa https://extranet.wink.travel
  2. I-click `Sign-In / Register` sa kanang sulok sa itaas at mag-log in gamit ang iyong user account.
  3. Piliin ang property na gusto mong gamitin mula sa itaas na kaliwang dropdown.
  4. Mag-click sa `Booking engine` sa navigation bar.
  5. Ipapakita sa iyo ng page ang iyong link.
  6. Ibigay ang link na ito sa iyong webmaster at gamitin ito para sa iyong **MAG-BOOK NA** pindutan.
  7. Ikaw ay ✅.
</Steps>


### Advanced

Ang link ay mayroon nang itinerary na naka-bake sa URL, na nagmula sa iyong napili `customization`.
Maaari mong i-customize pa ang iyong link sa pamamagitan ng:

* Pagbabago sa napiling pag-customize O
* I-override ang itinerary

Sa artikulong ito, tututukan namin ang huli dahil binibigyan ka nito ng higit na kalayaan na hayaan ang mga manlalakbay na pumili ng itinerary sa iyong website at ipasa ito sa booking engine ng Wink.

Narito kung paano mo ipapasa ang iyong sariling itineraryo kasama ang URL.

Mga wastong parameter ng kahilingan:

* **sd** Upang magdagdag ng petsa ng pagsisimula: *sd=2024-08-24*
* **n** Upang magdagdag ng bilang ng mga gabi: *n=1* O
* **ed** Gamitin ang petsa ng pagtatapos: *ed=2024-08-25*
* **rc** Mga panauhin: *rc=a2* (Matanda: 2)
* **l** Humiling ng ipinapakitang wika: *l=es*
* **c** Humiling ng display currency: *c=EUR*
* **p** Magdagdag ng promosyon: *p=ABC123*

Ang URL ay magiging ganito:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2\&l=es\&c=EUR

Para sa mas advanced na mga kaso ng paggamit, maaari ka ring humiling ng maraming configuration ng kwarto gamit ang `+` tanda:

Halimbawa:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2-ca16q1-ca8q1+a2\&l=es\&c=EUR

* Uri ng kuwarto 1 = Matanda: 2, Bata: 2 = 16 taong gulang + 8 taong gulang.
* Uri ng kuwarto 2 = Matanda: 2.

### Halimbawa

Sample **MAG-BOOK NA** button na HTML:

*Ipinapalagay ng sample na ito na ginagamit mo [Bootstrap](https://getbootstrap.com) bilang iyong pangunahing CSS library.*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Book a room"
>
BOOK NOW
</a>
```

### Iba pang mga benepisyo

Mayroong iba pang mga benepisyo sa paggamit ng isa sa aming mga link bilang iyong `book now` pindutan.

* **Social-friendly** Ang link ay madaling maibahagi sa social media at ito ay nagpapakita `dynamic pricing` sa ibabaw ng larawang iyong pinili.
* **Maikli** Napaka-shareable ng URL at magagamit mo ito sa mga lugar tulad ng iyong Instagram Bio URL.

## sa pamamagitan ng I-embed

Gusto ng ilang mga hotel ng kaunti pang functionality, at mas malalim na pagsasama, para sa kanilang website. Saklaw ng halimbawang ito ang pag-embed ng a [Card](/studio/cards) sa kanilang website.

Narito ang mga hakbang para gawin ito:

<Steps>
  1. Pumunta sa https://studio.wink.travel
  2. I-click `Sign-In / Register` sa kanang sulok sa itaas at mag-log in gamit ang iyong user account.
  3. Piliin ang property na gusto mong gamitin mula sa kaliwang sulok sa itaas.
  4. Mag-click sa `Search` mula sa pangunahing nav bar.
  5. I-type ang pangalan ng iyong property at i-click ang search button.
  6. I-click ang `Actions` link sa iyong card ng resulta ng paghahanap at piliin `Create > Make a Card`.
  7. Ito ay lilikha ng isang `Card`.
  8. May lalabas na mensahe sa kanang sulok sa itaas na nagsasabi sa iyong handa na ang iyong card at kung gusto mo pa itong i-customize.
  9. Mag-click sa `Tools > Cards` mula sa pangunahing nav bar at makikita mo ang iyong bagong card.
  10. Mag-click sa `Actions` link sa iyong link card at mag-click sa `Embed`.
  11. Magbubukas ito ng bagong window na may sample na HTML code na maaari mong kopyahin.
  12. Upang makuha ang iyong `Client-ID`, pumunta sa [Mga aplikasyon](/developers/applications).
  13. Upang makuha ang iyong `Configuration-ID`, pumunta sa [Pagpapasadya](/studio/customization)
  14. Ikaw ay ✅.
</Steps>


Halimbawang naka-embed na HTML ng card:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*Ang HTML sample sa itaas ay hindi kasama ang kinakailangang Wink CSS at Javascript; yung card lang.*
