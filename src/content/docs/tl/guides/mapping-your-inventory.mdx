Kapag nagdagdag ka ng imbentaryo sa iyong ari-arian sa Extranet — mga kuwarto, restawran, spa, mga silid-pulong, mga aktibidad, at iba pa — nagiging available ang imbentaryong iyon sa buong ekosistema ng Wink. Ipinaliwanag ng gabay na ito kung paano nagkakaugnay ang mga bahagi.

## Ang daloy ng imbentaryo

```
Extranet (ikaw ang lumikha) → Booking Engine (nakikita ng mga biyahero) → Studio (ibinibenta ng mga affiliate)
```

1. **Ikaw ang lumilikha ng imbentaryo** sa Extranet — mga uri ng kuwarto, pasilidad, at mga karanasan.
2. **Nagtatangkilik ang mga biyahero** sa imbentaryong iyon sa booking engine sa pamamagitan ng mga shareable na link, mga card, o direktang paghahanap.
3. **Nadidiskubre ng mga affiliate** ang iyong imbentaryo sa Studio, lumilikha ng mga card at grid, at ibinebenta ito sa kanilang mga audience.

## Mga uri ng kuwarto

Ang mga uri ng kuwarto ang puso ng iyong imbentaryo. Bawat uri ng kuwarto ay nangangailangan ng:

- Pangalan at paglalarawan.
- Mga konfigurasyon ng kama.
- Pinakamataas na kapasidad (mga adulto at bata).
- Mga amenidad na partikular sa kuwarto.
- Mga larawan.
- Hindi bababa sa isang rate plan na may master rates.

Kung walang mga rate, umiiral ang uri ng kuwarto ngunit hindi ito maaaring i-book.

## Mga pasilidad at karanasan

Bukod sa mga kuwarto, maaari kang maglista ng:

| Uri ng Imbentaryo | Ano ito | Halimbawa |
| -- | -- | -- |
| **Restawran** | Kainan sa lugar | "Rooftop Bar & Grill" |
| **Spa** | Mga serbisyong pangkalusugan | "Serenity Spa - 60min massage" |
| **Silid-pulong** | Lugar para sa mga kaganapan at kumperensya | "Boardroom A - 20 pax" |
| **Aktibidad** | Mga bagay na pwedeng gawin sa o malapit sa ari-arian | "Guided snorkeling tour" |
| **Atraksiyon** | Mga kilalang lugar malapit | "National Museum - 500m" |
| **Lugar** | Mga kawili-wiling lokasyon | "Local market" |
| **Add-on** | Mga dagdag na pwedeng bilhin kasama ng kuwarto | "Airport transfer" |

Bawat uri ng pasilidad ay may sariling seksyon sa Extranet. Maaaring ibenta ng mga affiliate ang alinman sa mga ito nang hiwalay — maaaring madiskubre ng isang biyahero ang iyong ari-arian sa pamamagitan ng listahan ng spa kaysa sa paghahanap ng kuwarto.

## Paano ginagamit ng mga affiliate ang iyong imbentaryo

Sa Studio, maaaring:

- **Maghanap** para sa iyong ari-arian o mga indibidwal na item ng imbentaryo.
- **Lumikha ng mga card** — Isang bookable na card para sa kuwarto, spa treatment, o aktibidad.
- **Lumikha ng mga grid** — Koleksyon ng mga card na ipinapakita nang sabay.
- **Lumikha ng mga mapa** — Mga marker na nakabase sa lokasyon na nagpapakita ng mga bookable na imbentaryo.
- **Magbahagi ng mga link** — Direktang URL sa iyong ari-arian o mga partikular na item.

Kapag mas kumpleto at iba-iba ang iyong imbentaryo, mas maraming paraan para maipromote ng mga affiliate ang iyong ari-arian.

## Mga tip

- **Maging masusi** — Ilahad ang lahat ng inaalok ng iyong ari-arian. Bawat item ay posibleng maging entry point para sa booking.
- **Gumamit ng magagandang larawan** — Dapat may hindi bababa sa isang mataas na kalidad na larawan ang bawat item ng imbentaryo.
- **Panatilihing sariwa ang mga paglalarawan** — I-update ang mga paglalarawan kapag nagbago ang mga inaalok o ayon sa panahon.
- **Presyohan ang lahat** — Ang imbentaryo na walang presyo ay maaaring tingnan ngunit hindi ma-book.
