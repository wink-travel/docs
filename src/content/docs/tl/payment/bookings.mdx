Upang makita ang iyong kasaysayan ng booking, i-click ang `Bookings` mula sa sidebar.

## Booking grid

Ipinapakita ng booking grid ang lahat ng iyong mga transaksyon sa isang filterable, sortable na talahanayan. Maaari kang:

- Mag-filter ayon sa **saklaw ng petsa** (default ay ang huling 30 araw).
- Mag-sort ayon sa petsa, halaga, o pangalan ng bisita.
- Mag-navigate sa mga resulta.

Ipinapakita ng bawat hilera ang pangalan ng manlalakbay, ari-arian, bilang ng mga bisita, mga petsa ng pananatili, at kabuuang halaga.

## Booking details

I-click ang anumang booking upang makita ang buong detalye nito:

### General information

- Booking ID, petsa ng paglikha, paraan ng pagbabayad, at status ng pagbabayad.

### Breakdown

- Uri ng kuwarto, okupasyon (mga adulto at bata), at mga petsa ng pananatili.
- Mga karagdagang item — anumang mga dagdag na binili kasama ng kuwarto.

### Fund breakdown

Ipinapakita kung paano eksaktong hinahati ang mga pondo sa lahat ng partido:

- **Commission** — Komisyon ng affiliate o ahente.
- **Platform fee** — Bayad sa pagproseso ng TripPay.
- **Hotel agency fee** / **Affiliate agency fee** — Kung naaangkop.
- **Sale amount** — Netong halaga para sa ari-arian.

### Payment information

- Mga Transaction ID (TripPay, Merchant, Trace ID).
- Mga link sa **guest-friendly TripPay receipt** at sa **payment gateway receipt**.
- External booking code at supplier identifier, kung mayroon.

### Refunds

Kung na-proseso na ang refund, makikita mo ang halaga ng refund, petsa, dahilan ng pagkansela, at kung sino ang nag-umpisa ng pagkansela.

### Payout

Paraan ng payout (hal., Virtual Credit Card) at petsa ng isyu.
