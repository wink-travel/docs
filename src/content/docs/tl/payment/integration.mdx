Kung isa kang kumpanya sa paglalakbay at gusto mong i-outsource sa TripPay ang bahagi ng pagbabayad at disbursement ng iyong negosyo, narito ang 6 na pangunahing hakbang upang makapagsimula:

<Steps>
  1. Lumikha ng a [account ng gumagamit](/user/register) sa Wink.
  2. Gumawa ng account gamit ang [TripPay](https://pay.wink.travel).
  3. Lumikha ng isang [Aplikasyon](/developers/applications) sa TripPay na hinahayaan kang makipag-usap sa aming API.
  4. Lumikha at imapa ang iyong mga benepisyaryo gamit ang TripPay. *hal. Para sa karamihan, nangangahulugan iyon ng paggawa ng mga account para sa mga property sa iyong platform na gusto mong bayaran ng TripPay.*
  5. Isama ang [bahagi ng web ng pagbabayad](/developers/web-components/#payment) sa iyong site.
  6. Ipadala ang TripPay sa manlalakbay `shopping cart` at simulan ang proseso ng pagbabayad.
</Steps>


:::tip
Para sa kumpletong gabay sa paggamit ng TripPay para sa iyong site sa paglalakbay, basahin ang gabay [Pagsasama sa TripPay](/guides/integrating-with-trip-pay).
:::

## API

Mga developer na gustong pamahalaan `Payments`, pumunta sa [Mga Developer > API > Mga Pagbabayad](/developers/apis/#payment-api).
