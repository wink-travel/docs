Ginagawa ng mga naka-save na paghahanap ang ipinahihiwatig ng pangalan nito. Habang naghahanap ng imbentaryo ng paglalakbay, mayroon kang opsyon na i-save ang iyong query sa paghahanap para magamit sa ibang pagkakataon.

## Sine-save ang iyong query sa paghahanap

Kapag nagsimula kang maghanap ng imbentaryo (tingnan [Maghanap](/studio/search)), mayroon kang kakayahang ayusin ang iyong mga filter sa paghahanap.
Sa ibaba ng form ng query ay ang opsyon na `Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Mag-save ng paghahanap" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Ipinapakita ang link sa I-save ang query sa paghahanap*</figcaption>
</figure>


Ang pag-click sa link na ito ay magbibigay sa iyo ng pagkakataong pangalanan ang paghahanap na ito (*hal. Aking Library of Magic Hotels*) isang bagay na maaalala mo mamaya.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Bigyan ng pangalan ang iyong paghahanap" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Bigyan ng pangalan ang iyong paghahanap*</figcaption>
</figure>


Bigyan ito ng pangalan at i-click ang `Save search` pindutan. Pagkatapos nito, maaari kang mag-navigate sa `Inventory > Saves Searches` mula sa pangunahing nav bar upang makita ang iyong bagong likhang tala.

## Maglista ng mga aksyon

Ang iyong naka-save na paghahanap ay maaaring gamitin upang ilantad ang imbentaryo ng paglalakbay sa iyong mga user sa anyo ng isang grid o mapa at ipakita ang iyong mga resulta ng paghahanap sa paraang gusto mo.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Naka-save na mga aksyon sa paghahanap" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Naka-save na query sa paghahanap na walang inilapat na mga filter*</figcaption>
</figure>


Ang bawat naka-save na query sa paghahanap ay may mga aksyon na maaari mong isagawa dito:

* I-convert ang query sa paghahanap sa a [mapa](/studio/maps).
* I-convert ang query sa paghahanap sa a [grid](/studio/grids).
* Alisin ang naka-save na paghahanap.

## API

Maaaring magtungo ang mga developer na gustong pamahalaan ang mga naka-save na paghahanap [Mga Developer > API > Mag-browse](/developers/apis/#browse-api).

## Karagdagang pagbabasa

* Basahin ang tungkol sa aming [Mapa](/studio/maps) mga tampok.
* Basahin ang tungkol sa aming [Grid](/studio/grids) mga tampok.
