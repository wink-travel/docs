Nagbibigay ang Studio ng ilang embeddable na widget na maaari mong idagdag sa iyong sariling website upang mapalakas ang mga booking.

Upang ma-access ang mga embed widget, pumunta sa `Tools > Miscellaneous` mula sa sidebar.

## Mga Available na Widget

| Widget | Paglalarawan |
| -- | -- |
| **Account button** | Isang branded na button na nagli-link sa iyong affiliate page |
| **Itinerary button** | Isang button na nagbubukas ng itinerary selector |
| **Itinerary form** | Isang inline na form para sa pagpili ng petsa at bisita |
| **Search button** | Isang button na nagbubukas ng paghahanap ng property |
| **Lookup button** | Isang button para maghanap ng partikular na imbentaryo |

## Paano Mag-embed

Bawat widget ay nagbibigay ng code snippet na maaari mong kopyahin at i-paste sa HTML ng iyong website. Kasama sa snippet ang iyong **Client ID** at **Customization ID** upang ang widget ay awtomatikong naka-link sa iyong affiliate account.

:::note
Kung gumagamit ka ng WordPress, nagbibigay din kami ng WordPress plugin para sa mas madaling integrasyon.
:::
