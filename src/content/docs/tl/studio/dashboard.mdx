Ang Studio dashboard ay nagbibigay sa iyo ng mabilis na pangkalahatang-ideya ng iyong affiliate performance.

## Activity

Ipinapakita ng seksyon ng activity ang iyong booking analytics — kung ilan ang mga booking na iyong naipasok, kinita na kita, at mga trend ng conversion sa paglipas ng panahon.

## Supplier search

Gamitin ang mabilisang paghahanap upang mahanap ang mga supplier ayon sa pangalan ng property. Ito ay isang shortcut sa pag-browse ng supplier network.

## Account status

Ipinapakita ng dashboard ang kasalukuyang status ng iyong account — kung kumpleto na ang iyong profile at handa nang kumita ng komisyon.

## Leaderboard

Tingnan kung paano ka nakapuwesto laban sa ibang mga affiliate base sa mga performance metrics.
