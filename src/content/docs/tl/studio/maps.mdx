Ang natitirang bahagi ng artikulong ito ay gagabay sa iyo kung paano gumawa, i-customize at ibahagi ang iyong mga mapa.

## Gumawa ng mapa

Maaaring gumawa ng mga mapa sa dalawang paraan:

- **Single marker map**: Gumawa ng mapa mula sa isang bagay na nakita mo habang [naghahanap](/tl/studio/search).
- **Multi marker map**: Gumawa ng mapa mula sa isang [curated list](/tl/studio/curated-lists) o [saved search](/tl/studio/saved-searches).

I-click ang button na iyon at dadalhin ka nito sa aming pahina ng form ng mapa kung saan maaari mong simulan ang pag-customize ng iyong mapa.

### Multi marker map

Parehong ang [curated lists](/tl/studio/curated-lists#list-actions) at [saved searches](/tl/studio/saved-searches#list-actions) ay may kasamang aksyon na nagpapahintulot sa iyo na gumawa ng mapa. Ang mapang ito ay maglalaman ng mga marker sa mapa para sa lahat ng imbentaryo sa loob ng mga listahang iyon.

## I-customize ang mapa

 Pinapayagan ka ng form na i-customize ang iyong mapa sa mga sumusunod na paraan:

<Steps>
 1. Bigyan ito ng pangalan upang maalala mo kung tungkol saan ang mapa.
 2. Piliin kung nais mong payagan ang user na gumalaw sa paligid ng mapa.
 3. Itakda kung gaano kataas, sa pixels, ang gusto mong maging taas ng mapa.
 4. Piliin kung nais mong payagan ang user na mag-zoom in at out sa mapa.
 5. Piliin ang istilo ng mapa na babagay sa website kung saan mo i-e-embed ang mapa.
 6. Piliin ang kulay ng marker.
 7. Piliin ang unang card face na nais mong makita ng mga user. Sa default, ito ay ang native face ng imbentaryong iyon.
 8. Maaari kang magdagdag ng sarili mong mga marker, bilog, parihaba at polygon upang ipakita ang iba pang mga lokasyon sa mapa na maaaring maging kawili-wili sa iyong mga user.
 9. I-click ang button na `Save` upang magpatuloy.
</Steps>

Pagkatapos i-save ang iyong mapa, ikaw ay ire-redirect sa iyong pahina ng mga mapa at ang iyong mapa ay handa nang ibahagi sa buong mundo.

Nasa itaas ang isang larawan na nagpapakita ng lahat ng magagamit na mga aksyon para sa iyong mapa:

- **Update** Ina-update ang iyong configuration ng mapa.
- **Add to WinkLinks** Idinadagdag ang mapa sa iyong WinkLinks account.
- **Embed** Ipinapakita kung paano i-embed ang mapang ito bilang isang [Map](/tl/developers/web-components/#content-loader) sa iyong website.
- **Use with WordPress** Ipinapakita kung paano gamitin ang aming [WordPress plugin](/tl/developers/wordpress) upang i-embed ang mapang ito sa iyong website.

Tinutukoy namin ang ilan sa mga opsyong ito nang mas detalyado sa ibaba.

### Embed

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Narito kung paano i-embed ang iyong mapa sa iyong site:

- Ipinapakita ng Linya 3 kung paano i-embed ang mga estilo ng Wink sa iyong site.
- Ipinapakita ng mga Linya 6 hanggang 9 kung paano gamitin ang [wink-content-loader](/tl/developers/web-components/#content-loader) Web Component at sabihin dito na kunin ang isang ad banner para sa iyong code.
- Ipinapakita ng Linya 11 kung paano i-embed ang aming Javascript sa iyong site.
- Ipinapakita ng Linya 13 kung paano i-embed ang [wink-app-loader](/tl/developers/web-components/#application-loader) Web Component at sabihin dito na kunin ang iyong mga page-level configuration preferences.

## API

Ang mga developer na nais mag-manage ng mga mapa ay maaaring pumunta sa [Developers > API > Inventory](/tl/developers/apis/#inventory-api).

## Karagdagang babasahin

- Matuto nang higit pa tungkol sa aming koleksyon ng [Web Components](/tl/developers/web-components).
- Matuto nang higit pa tungkol sa [Wink WordPress plugin](/tl/developers/wordpress).
