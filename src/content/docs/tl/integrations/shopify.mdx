Pinapayagan ng **Wink Shopify App** ang mga tour operator, nagbebenta ng karanasan, at mga negosyante sa hospitality na ipakita ang live na Wink inventory nang direkta sa kanilang Shopify storefront — kasabay ng mga produkto, sa mga landing page, o kahit saan sa tema.

## I-install

<Steps>
1. Pumunta sa [Shopify App Store](https://apps.shopify.com) at hanapin ang **Wink Travel**.
2. I-click ang **Add app** at sundin ang OAuth flow para mag-authorize.
3. Sa mga setting ng Wink app, ilagay ang iyong **Client ID** at **Client Secret**.
   Pumunta sa [Applications](/tl/portal/settings/applications) para kunin ang iyong mga kredensyal.
4. I-click ang **Verify & Save**. Magpapakita ng mensahe ng tagumpay bilang kumpirmasyon ng koneksyon.
</Steps>

## I-enable ang App Embed Block

Awtomatikong ini-inject ng App Embed Block ang Wink script at stylesheet sa bawat pahina — hindi mo na kailangang i-edit ang theme code.

<Steps>
1. Sa Shopify Admin, pumunta sa **Online Store → Themes → Customize**.
2. I-click ang **App embeds** sa kaliwang panel.
3. I-toggle ang **Wink Travel** sa on.
4. I-click ang **Save**.
</Steps>

:::note
Binabasa ng App Embed Block ang iyong Client ID mula sa isang Shopify metafield na na-set noong install. Hindi kailanman naipapakita ang iyong Client Secret sa browser.
:::

## Magdagdag ng component sa isang pahina

<Steps>
1. Sa theme editor, buksan ang pahina o seksyon kung saan mo gustong ilagay ang Wink content.
2. I-click ang **Add block** at pumili mula sa mga Wink blocks:
   - **Wink Content** — mga hotel card, grids, o mapa
   - **Wink Lookup** — search bar para sa destinasyon
   - **Wink Search** — button para sa itinerary picker
   - **Wink Account** — button para sa pag-sign in
   - **Wink Itinerary** — button para sa status ng itinerary
   - **Wink Cart** — button para sa shopping cart
3. Para sa **Wink Content**, piliin ang **Layout Type** at ilagay ang **Layout ID** mula sa Wink Studio.
4. I-click ang **Save**.
</Steps>

## Mga available na theme blocks

| Block | Layunin |
|---|---|
| **Wink Content** | Nagpapakita ng mga hotel card, ranked grids, mapa, at iba pa |
| **Wink Lookup** | Search bar para sa mga destinasyon at hotel |
| **Wink Search** | Icon button na nagbubukas ng itinerary picker |
| **Wink Account** | Button para sa pag-sign in / user account |
| **Wink Itinerary** | Button ng itinerary na nagpapakita ng kasalukuyang biyahe |
| **Wink Cart** | Button para sa buod ng shopping cart |
