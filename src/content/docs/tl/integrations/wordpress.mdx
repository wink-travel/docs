Para sa sinumang gumagamit ng WordPress para sa kanilang negosyo o blog, lubos naming inirerekomenda ang paggamit ng [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
Pinapadali ng plugin na ito ang pag-access sa iyong mga Wink card, grids, mapa, atbp., at ginagawang kasing dali ng paghila ng nilalaman papunta sa iyong mga post at pahina gamit ang iyong paboritong page builder o ang aming mga custom shortcode.

##  Install

Ganito ang pag-install ng `Wink Affiliate WordPress Plugin` sa iyong WordPress website.

<Steps>
1. Kapag naka-log in ka bilang administrator at nasa Dashboard.
2. I-click ang `Plugins > Add New Plugin`.
3. Hanapin ang `Wink Affiliate WordPress Plugin`.
4. I-click ang `Install` para i-download at i-install ang plugin sa iyong site.
5. I-click ang `Activate` para i-activate ang plugin sa iyong site.
6. Dadalhin ka ng activation sa `Wink Settings` page. *Maaari ka ring pumunta doon sa pamamagitan ng pag-navigate sa `Appearance > Customize > Wink Settings`.*
7. Hihingin nito ang iyong `Client ID` at `Client Secret`. Pumunta sa [Applications](/tl/portal/settings/applications) para matutunan kung paano makuha ang iyong mga kredensyal.
8. Ilagay ang iyong mga kredensyal at i-click ang `Publish`. Ikaw ay konektado na ngayon sa iyong Wink account.
</Steps>

## Mga page builder

Sinusuportahan namin ang mga pinakasikat na page builder sa WordPress na nagpapadali para sa sinumang gustong gumamit ng Wink.

### Gutenberg

Ang Gutenberg ang native editor mula sa WordPress.

### Avada

[Avada](https://avada.com) ay kilala sa malawak nitong pagpipilian ng magagandang template kasama ang isang matibay na page builder na may advanced na mga tampok.

### Elementor

[Elementor](https://elementor.com) ang [kasalukuyang] pinakasikat na page builder na may libreng bersyon at pro na bersyon.

### WPBakery

[WPBakery](https://wpbakery.com) ang unang advanced na page builder sa merkado.

## Mga shortcode

### [winklookup]

Gumagana sa ranked content grid. I-type ang lugar na nais mong bisitahin at ipapakita ang imbentaryo.

### [winksearch]

Simpleng button para buksan ang itinerary form.

### [winkaccount]

Button para sa account upang mag-authenticate. Kapag na-authenticate na, nagiging dropdown ito na may mga opsyon sa account.

### [winkitinerary]

Katulad ng search button, ngunit naglalaman ito ng impormasyon ng itinerary bilang teksto ng button.

### [winkcontent]

Pinapayagan kang i-embed ang imbentaryo na pinili mo sa Wink nang direkta sa isang pahina o post. Ang shortcode na ito ay nangangailangan din ng layout ID.

Ang layout

```
[winkcontent layoutid=""]
```

## Karagdagang babasahin

Makakahanap ka ng higit pang impormasyon tungkol sa aming WordPress plugin mula sa mga link sa ibaba.

- Alamin ang tungkol sa aming [Web Components](/tl/integrations/web-components).
- I-install ang [Wink Affiliate WordPress plugin](https://wordpress.org/plugins/wink2travel/).
- I-fork at bumuo sa ibabaw ng aming Github [Wink Affiliate WordPress repository](https://github.com/wink-travel/wink-wordpress-plugin).
