Ang **Wink Wix App** ay nagdadagdag ng Wink web components sa iyong Wix site bilang mga native widget na maaari mong i-drag at i-drop sa Wix Editor. Hindi kailangan ng coding para sa mga may-ari ng site.

  title="wink-travel/wink-wix-app on GitHub"

/>

## Install

<Steps>
1. Pumunta sa [Wix App Market](https://www.wix.com/app-market) at hanapin ang **Wink Travel**.
2. I-click ang **Add to Site** at sundin ang mga tagubilin para i-authorize ang app.
3. Kapag na-install na, buksan ang **Dashboard → Wink Settings** at ilagay ang iyong **Client ID** at **Client Secret**.
   Pumunta sa [Applications](/tl/portal/settings/applications) para kunin ang iyong mga kredensyal.
4. I-click ang **Save** at pagkatapos ay **Test Connection** upang kumpirmahin na nakakonekta ang Wink.
</Steps>

## Magdagdag ng component sa isang pahina

<Steps>
1. Buksan ang **Wix Editor** at pumunta sa pahina kung saan mo gustong magdagdag ng Wink content.
2. I-click ang **Add (+) → Apps → Wink Travel** upang makita ang lahat ng available na mga widget.
3. I-drag ang isang widget papunta sa pahina — halimbawa, ang **Wink Content** para magpakita ng hotel grid.
4. Sa widget settings panel, piliin ang **Layout Type** at i-paste ang **Layout ID** mula sa Wink Studio.
5. I-click ang **Publish** para maging live ang mga pagbabago.
</Steps>

## Mga available na widget

| Widget | Layunin |
|---|---|
| **Wink Content** | Nagpapakita ng mga hotel card, ranked grids, mapa, at iba pa |
| **Wink Lookup** | Search bar para sa mga destinasyon at hotel |
| **Wink Search** | Icon button na nagbubukas ng itinerary picker |
| **Wink Account** | Sign-in / user account button |
| **Wink Itinerary** | Itinerary button na nagpapakita ng kasalukuyang biyahe |
| **Wink Cart** | Shopping cart summary button |
