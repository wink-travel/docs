Ang **Wink Drupal Module** ay nagdaragdag ng mga Wink travel inventory component sa iyong Drupal site bilang mga native block. Maaaring ilagay ng mga editor ang mga ito kahit saan sa layout gamit ang standard block system.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Source code, developer setup guide, and contribution instructions."
/>

## Mga Kinakailangan

- Drupal 10 o 11
- PHP 8.1+
- Composer

## Pag-install

<Steps>
1. I-install gamit ang Composer:
   ```bash
   composer require drupal/wink
   ```
2. I-enable ang module:
   ```bash
   drush en wink -y
   ```
3. Pumunta sa **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Ilagay ang iyong **Client ID** at **Client Secret**, piliin ang iyong environment, at i-click ang **Save**.
   Pumunta sa [Applications](/tl/portal/settings/applications) para kunin ang iyong mga kredensyal.
</Steps>

## Magdagdag ng block sa isang pahina

<Steps>
1. Pumunta sa **Admin → Structure → Block Layout**.
2. I-click ang **Place block** sa rehiyon kung saan mo nais ilagay ang Wink content.
3. Hanapin ang **Wink** para makita ang mga available na block:
   - **Wink Content** — nagpapakita ng hotel card, grid, o mapa
   - **Wink Lookup** — search bar para sa destinasyon
   - **Wink Search** — button para sa itinerary picker
   - **Wink Account** — sign-in button
   - **Wink Itinerary** — button para sa status ng itinerary
   - **Wink Cart** — button para sa shopping cart
4. Para sa **Wink Content**, i-configure ang **Layout Type** at **Layout ID** sa block settings.
5. I-save ang block configuration.
</Steps>

## Mga Available na block

| Block | Layunin |
|---|---|
| **Wink Content** | Nagpapakita ng hotel cards, ranked grids, mga mapa, at iba pa |
| **Wink Lookup** | Search bar para sa mga destinasyon at hotel |
| **Wink Search** | Icon button na nagbubukas ng itinerary picker |
| **Wink Account** | Sign-in / user account button |
| **Wink Itinerary** | Itinerary button na nagpapakita ng kasalukuyang biyahe |
| **Wink Cart** | Button para sa buod ng shopping cart |
