Ang **Wink TYPO3 Extension** ay nag-iintegrate ng Wink travel inventory sa TYPO3 13 bilang mga native content element. Ang mga editor ay nagdadagdag ng Wink components sa mga pahina gamit ang karaniwang TYPO3 page editor — hindi kailangan ng pagbabago sa template.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"

/>

## Mga Kinakailangan

- TYPO3 13.x
- PHP 8.2+
- Composer

## Pag-install

<Steps>
1. I-install gamit ang Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. I-activate ang extension:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Sa TYPO3 backend, pumunta sa **Admin Tools → Settings → Extension Configuration → wink**.
4. Ilagay ang iyong **Client ID**, **Client Secret**, at piliin ang iyong environment. I-save.
   Pumunta sa [Applications](/tl/portal/settings/applications) para kunin ang iyong mga kredensyal.
5. Idagdag ang static TypoScript template sa iyong root template:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Magdagdag ng content element sa isang pahina

<Steps>
1. Buksan ang isang pahina sa **Page** module.
2. I-click ang **+** button sa content area para magdagdag ng bagong element.
3. Piliin ang isa sa mga **Wink Travel** content element:
   - **Wink Content** — hotel cards, ranked grids, maps
   - **Wink Lookup** — destination search bar
   - **Wink Search** — itinerary picker button
   - **Wink Account** — sign-in button
   - **Wink Itinerary** — itinerary status button
   - **Wink Cart** — shopping cart button
4. Para sa **Wink Content**, gamitin ang FlexForm fields para pumili ng **Layout Type** at ilagay ang **Layout ID**.
5. I-save at i-publish ang pahina.
</Steps>

## Mga available na content element

| Element | Layunin |
|---|---|
| **Wink Content** | Mga hotel card, ranked grids, mapa, at iba pa |
| **Wink Lookup** | Search bar para sa destinasyon |
| **Wink Search** | Icon button na nagbubukas ng itinerary picker |
| **Wink Account** | Sign-in / user account button |
| **Wink Itinerary** | Itinerary button na nagpapakita ng kasalukuyang biyahe |
| **Wink Cart** | Shopping cart summary button |
