Jei esate integratorius, pavyzdžiui, Wink, ir norite susieti savo išorinius gavėjus naudoti su TripPay, galite tai padaryti dviem būdais:

- Rankinis susiejimas
- Naudojant [Payment API](/lt/developers/apis/#payment-api) (*žr. `Account Mappings`*)

:::tip
Jei nesate susidūrę tik su keliomis tomis pačiomis savybėmis, rekomenduojame naudoti API, kad automatizuotumėte šį žingsnį.
:::

Žemiau pateikiame žingsnius, kaip rankiniu būdu susieti išorinį gavėją. *Iš tiesų tai naudoja API fone.*

<Steps>
1. Prisijunkite prie [TripPay](https://pay.wink.travel).
2. Pasirinkite savo integratoriaus paskyrą.
3. Pagrindinėje navigacijos juostoje spustelėkite `Settings > Mappings`.
4. Spustelėkite mygtuką `Create mapping`. Pamatysite susiejimo formą.
5. **Vietinė paskyra** Pradėkite rašyti esamos TripPay paskyros, kurią norite susieti, pavadinimą.
6. **Išorinis identifikatorius** Įklijuokite savo išorinį ID šiai įmonei, kaip jis yra nuotoliniame sistemoje.
7. Spustelėkite `Save`, kad tęstumėte. Būsite nukreipti atgal į savo susiejimų sąrašą.
</Steps>

Turėdama jūsų susiejimus, TripPay dabar žino, kurioms paskyroms priskirti lėšas kaip rezervacijos gavėjams, kai nurodote juos naudodami savo išorinius identifikatorius.

## API

Kūrėjai, norintys valdyti `Payments`, eikite į [Developers > API > Payments](/lt/developers/apis/#payment-api).
