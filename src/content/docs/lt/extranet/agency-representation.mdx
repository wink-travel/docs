Agentūra yra įprasta Wink paskyra, kurią galite priskirti savo paskyrai atstovavimo tikslais. Žemiau pateikiamos kelios priežastys, kodėl norėtumėte, kad jus atstovautų agentūra:

- Pilnas / dalinis jūsų Wink paskyros valdymas trečiosios šalies.
- Turtas turi esamą susitarimą su savo valdymo įmone ir nori, kad Wink tvarkytų finansinę jų susitarimo dalį dėl lėšų, gaunamų per Wink.
- Nekilnojamojo turto valdymo agentūra atlieka visus internetinius darbus savo turto vardu.

## Priskirti agentūrą

Štai kaip priskirti agentūrą savo paskyrai.

<Steps>
 1. Prisijunkite prie savo vartotojo paskyros [Wink Extranet](https://extranet.wink.travel).
 2. Pasirinkite turtą, su kuriuo norite dirbti.
 3. Spustelėkite `Account > Managers` pagrindinėje navigacijos juostoje.
 4. Skiltyje `Agency` įveskite agentūros, kurią norite, kad jus atstovautų, pavadinimą.
 5. Iš išskleidžiamojo sąrašo pasirinkite agentūros mokestį.
 5. Pasirinktinai pridėkite galiojimo taisykles.
 7. Spustelėkite mygtuką `Set agency`, kad tęstumėte.
 </Steps>

 :::note
Norint priskirti agentūrą, turite būti turto savininkas.
 :::

 :::tip[Pavyzdys]
Agentūros mokestis skaičiuojamas nuo užsakymo sumos.

Užsakymas = 100 $, Agentūros mokestis = 1 % => 100 * 0.01 = 1 $
 :::

## Atšaukti agentūros priskyrimą

Štai kaip atšaukti agentūros priskyrimą savo paskyroje.

<Steps>
 1. Prisijunkite prie savo vartotojo paskyros [Wink Extranet](https://extranet.wink.travel).
 2. Pasirinkite turtą, su kuriuo norite dirbti.
 3. Spustelėkite `Account > Managers` pagrindinėje navigacijos juostoje.
 4. Skiltyje `Agency` spustelėkite <Badge text="x" variant="danger" size="small" /> piktogramą dešinėje agentūros išskleidžiamojo sąrašo pusėje.
 5. Spustelėkite mygtuką `Set agency`, kad tęstumėte.
 </Steps>


## API

Kūrėjai, norintys valdyti agentūrą per API, gali apsilankyti [Developers > API > Affiliate](/lt/developers/apis/#affiliate-api).
