Šiame puslapyje pateikiami visi webhook įvykiai, kuriems galite užsiprenumeruoti Wink. Diegimo instrukcijoms žr. [Webhook integracijos vadovą](/lt/guides/webhook-integration) arba [Webhook valdymo puslapį](/lt/my-account/webhooks).

## Kaip užsiprenumeruoti

1. Eikite į **Applications > Webhooks** savo portale.
2. Sukurkite webhook ir pasirinkite įvykius, kuriuos norite gauti.
3. Kiekvieną kartą įvykus įvykiui, Wink siunčia HTTP POST į jūsų galinį tašką.

Pilni diegimo žingsniai ir geriausios praktikos pateiktos [Webhook integracijoje](/lt/guides/webhook-integration).

## Sąskaitos įvykiai

Įvykiai, susiję su jūsų TripPay sąskaitos gyvavimo ciklu ir nustatymais.

| Įvykis | Raktas | Aprašymas |
|---|---|---|
| Sąskaita sukurta | `account.create` | Sukurta nauja sąskaita. |
| Sąskaita peržiūrima | `account.under_review` | Sąskaita laukia peržiūros. |
| Sąskaita atnaujinta | `account.updated` | Sąskaitos duomenys pasikeitė. |
| Sąskaitos būsena atnaujinta | `account.status.updated` | Sąskaitos būsena pasikeitė (pvz., patvirtinta, sustabdyta). |
| Sąskaitos adresas atnaujintas | `account.address.updated` | Sąskaitos adresas pakeistas. |
| Sąskaitos duomenys atnaujinti | `account.details.updated` | Sąskaitos duomenų laukai pasikeitė. |
| Sąskaitos informacija atnaujinta | `account.information.update` | Bendroji sąskaitos informacija atnaujinta. |
| Sąskaitos vadybininkai atnaujinti | `account.managers.update` | Sąskaitos vadybininkų sąrašas pasikeitė. |
| Profilio nuotrauka atnaujinta | `account.profile_picture.updated` | Sąskaitos profilio nuotrauka pakeista. |
| Agentas atnaujintas | `account.agent.updated` | Sąskaitos agento informacija pakeista. |
| URL pavadinimas atnaujintas | `account.url-name.update` | Sąskaitos pritaikytas URL pasikeitė. |
| Unikalus ID atnaujintas | `account.unique-id.update` | Sąskaitos unikalus identifikatorius pakeistas. |

### Sąskaitos užduotys

| Įvykis | Raktas | Aprašymas |
|---|---|---|
| Užduotis sukurta | `account.task.create` | Sąskaitai pridėta nauja užduotis. |
| Užduotis atnaujinta | `account.task.update` | Sąskaitos užduotis pakeista. |
| Užduotis pašalinta | `account.task.remove` | Sąskaitos užduotis pašalinta. |
| Užduotis užbaigta | `account.task.complete` | Sąskaitos užduotis įvykdyta. |
| Mokesčių ID užduotis užbaigta | `account.task.tax_id.complete` | Mokesčių ID patvirtinimo užduotis įvykdyta. |

### Sąskaitos įsigytojas

| Įvykis | Raktas | Aprašymas |
|---|---|---|
| Įsigytojas sukurtas | `account.acquirer.created` | Sąskaitai pridėtas naujas įsigytojas. |
| Įsigytojas atnaujintas | `account.acquirer.updated` | Sąskaitos įsigytojas pakeistas. |
| Įsigytojas pašalintas | `account.acquirer.removed` | Sąskaitos įsigytojas pašalintas. |

## Rezervacijos sutarties įvykiai

Įvykiai, stebintys rezervacijos sutarties gyvavimo ciklą per TripPay.

| Įvykis | Raktas | Aprašymas |
|---|---|---|
| Inicializavimas prašytas | `booking.contract.initialize.request` | Prašyta rezervacijos sutarties inicializavimo. |
| Inicializavimas baigtas | `booking.contract.initialize.complete` | Rezervacijos sutarties inicializavimas baigtas. |
| Sutartis sukurta | `booking.contract.created` | Sukurta nauja rezervacijos sutartis. |
| Sutartis apdorojama | `booking.contract.processing` | Vykdomas rezervacijos sutarties mokėjimo apdorojimas. |
| Sutartis nepavyko | `booking.contract.failed` | Nepavyko apdoroti rezervacijos sutarties mokėjimo. |
| Sutartis atšaukta | `booking.contract.cancelled` | Rezervacijos sutartis atšaukta. |
| Dalinis grąžinimas | `booking.contract.refund.partial` | Sutarties dalinis grąžinimas atliktas. |
| Pilnas grąžinimas | `booking.contract.refund.full` | Sutarties pilnas grąžinimas atliktas. |
| Grąžinimo prašymas | `booking.contract.refund.request` | Pateiktas grąžinimo prašymas. |
| Grąžinimo klaida | `booking.contract.refund.error` | Įvyko klaida apdorojant grąžinimą. |
| Išmokėjimas sukurtas | `booking.contract.payout.create` | Sutarties išmokėjimas sukurtas. |
| Išmokėjimas baigtas | `booking.contract.payout.complete` | Sutarties išmokėjimas sėkmingai baigtas. |

## Valdančiosios įstaigos įvykiai

Įvykiai, susiję su valdančiomis įstaigomis (objektais, filialais, įmonėmis) ir jų administravimu.

### Prenumerata

| Įvykis | Raktas | Aprašymas |
|---|---|---|
| Prenumerata suteikta | `managing_entity.subscription.provisioned` | Suteikta nauja prenumerata. |
| Prenumerata pakeista | `managing_entity.subscription.changed` | Prenumeratos planas pakeistas. |
| Prenumeratos sąskaita pakeista | `managing_entity.subscription.invoice.changed` | Prenumeratos sąskaita atnaujinta. |

### Būsena ir administravimas

| Įvykis | Raktas | Aprašymas |
|---|---|---|
| Būsena atnaujinta | `managing_entity.status.updated` | Įstaigos būsena pasikeitė. |
| Įstaiga ištrinta | `managing_entity.deleted` | Valdanti įstaiga ištrinta. |
| Agentas atnaujintas | `managing_entity.agent.updated` | Įstaigos agentas pakeistas. |
| Agentui išsiųstas el. laiškas | `managing_entity.agent.email.notification` | Agentui išsiųstas el. laiškas. |
| Vadybininkui išsiųstas el. laiškas | `managing_entity.manager.email.notification` | Vadybininkui išsiųstas el. laiškas. |

### Vadybininko gyvavimo ciklas

| Įvykis | Raktas | Aprašymas |
|---|---|---|
| Vadybininkas pridėtas | `managing_entity.manager.added` | Pridėtas naujas vadybininkas. |
| Vadybininkas pakviestas | `managing_entity.manager.invited` | Išsiųstas vadybininko kvietimas. |
| Vadybininkas pašalintas | `managing_entity.manager.removed` | Vadybininkas pašalintas. |
| Vadybininkas priimtas | `managing_entity.manager.accepted` | Vadybininko kvietimas priimtas. |
| Vadybininkas atmestas | `managing_entity.manager.rejected` | Vadybininko kvietimas atmestas. |
| Savininkas nustatytas | `managing_entity.manager.owner.set` | Įstaigos nuosavybė perleista. |
