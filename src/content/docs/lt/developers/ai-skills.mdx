Dirbtinio intelekto įgūdžiai yra nešiojami instrukcijų paketai, kurie moko programavimo agentus dirbti su Wink. Vietoje to, kad remtųsi mokymo duomenimis ar pasenusia dokumentacija, kiekvienas įgūdis nukreipia agentą naudoti [Wink MCP serverį](/lt/developers/mcp) kaip tiesioginį patikimą šaltinį API sutartims ir platformos dokumentacijai.

Įgūdžiai veikia su bet kuriuo agento klientu, kuris juos palaiko — Claude Code, Codex, Cursor ir kitais.

## Galimi įgūdžiai

### wink-api-builder

Įgyvendinkite, peržiūrėkite arba derinkite Wink API integracijas. Įgūdis veda agentą per struktūruotą darbo eigą:

1. Ieškokite tinkamos API operacijos per `api_search`
2. Atidarykite tiesiogines užklausų/atsakymų sutartis iš MCP serverio
3. Iš dokumentacijos paimkite autentifikacijos nustatymus
4. Rašykite kodą pagal patvirtintą sutartį
5. Pažymėkite bet kokius neatitikimus tarp esamo kodo ir dabartinio API

Naudinga:
- Kuriant naujus klientų metodus bet kuriam Wink API
- Derinant sutarties neatitikimus (pvz., užklausa grąžina HTTP 400)
- Peržiūrint esamas integracijas pagal tiesioginę specifikaciją

## Įdiegimas

Kopijuokite įgūdžių aplanką į savo agento įgūdžių katalogą.

:::tip
Wink MCP serveris turi būti prijungtas toje pačioje aplinkoje. Žr. [MCP Server](/lt/developers/mcp) diegimo instrukcijas.
:::

**Claude Code**

```bash
# vietinis repozitorijuje
cp -r skills/wink-api-builder .claude/skills/

# arba vartotojo lygiu
cp -r skills/wink-api-builder ~/.claude/skills/
```

Iškvieskite taip:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# vietinis repozitorijuje
cp -r skills/wink-api-builder .agents/skills/

# arba vartotojo lygiu
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Iškvieskite taip:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Kiti agentai**

Įdėkite įgūdžių aplanką ten, kur jūsų agentas ieško įgūdžių. Įėjimo taškas yra `SKILL.md`.
