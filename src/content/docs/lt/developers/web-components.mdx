[Žiniatinklio komponentai](https://developer.mozilla.org/en-US/docs/Web/API/Web_components)yra standartas, leidžiantis integruoti visas „Javascript“ funkcijas į savo svetainę nemokant programuoti.
Dėl šios puikios technologijos galite labai lengvai įterpti „Wink“ kelionių inventorių. Šiame straipsnyje pateikiama išsami informacija apie mūsų žiniatinklio komponentų kolekciją ir parodyta, kaip juos naudoti.

## Įterpti mirktelėjimą

Į bet kurį puslapį, kuriame norite rodyti vieną iš mūsų žiniatinklio komponentų, turite įtraukti tris dalykus.

1. Įtraukite mūsų stiliaus lapą.
2. Įtraukite mūsų „Javascript“.
3. Įtraukite mūsų programų įkroviklį.

### Stiliaus lapas

Įtraukite mūsų CSS stilius į savo dokumento antraštę.

```html {3}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
</html>
```

### Javascript

Įtraukite mūsų „Javascript“ kodą dokumento apačioje. (*Rekomenduojame iš karto virš galinės teksto žymos*).

```html {3}
<html>
  <body>
    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
  </body>
</html>
```

Jūsų svetainė dabar paruošta ir gali rodyti mūsų žiniatinklio komponentus.

## Komponentai

Skaitykite apie mūsų komponentų biblioteką žemiau.

### Programų įkėlėjas

Įkroviklis yra atsakingas už būsenos palaikymą ir mūsų komponentų sąveikos valdymą.*Įtraukite jį po mūsų „Javascript“*).

```html {3-6}
<html>
  <body>
    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

### Turinio įkėlėjas

Turinio įkėlimo programa yra mūsų žiniatinklio komponentų bibliotekos pagrindas. Ji atsakinga už jūsų inventoriaus (kortelių, tinklelių, žemėlapių) rodymą.

Galimi atributai:

* išdėstymas
* id
* rūšiuoti

*Atributas`sort`galima tik tada, kai išdėstymas yra`RANKED`ir nenorite naudoti esamos reitinguotos lentelės. Tokiais atvejais palikite`id`tuščias.*

Galimi išdėstymo tipai:

* `AD_BANNER`
* `MAP`
* `HOTEL`
* `GUEST_ROOM`
* `MEETING_ROOM`
* `SPA`
* `RESTAURANT`
* `ACTIVITY`
* `ATTRACTION`
* `PLACE`
* `ADD_ON`
* `LIST`
* `SEARCH`
* `RANKED`

Galimi rūšiavimo tipai:

* `MEMBER`
* `PRICE_LOW_TO_HIGH`
* `PRICE_HIGH_TO_LOW`
* `PRICE`
* `POPULARITY`
* `ECO`
* `EXPERIENCE`
* `PERK`
* `LOYALTY`
* `PACKAGE`

```html {3-6}
<html>
  <body>
    <wink-content-loader
       layout="GUEST_ROOM"
       id="2de7ee9c-61c9-11ef-9722-42004e494300"
    ></wink-content-loader>
  </body>
</html>
```

Pavyzdžių galima rasti[kortelės](/studio/cards),[tinkleliai](/studio/grids)ir[žemėlapiai](/studio/maps).

1. Įtraukite mūsų stiliaus lapą.
2. Įtraukite mūsų „Javascript“.

### Stiliaus lapas

Įtraukite mūsų CSS stilius į savo dokumento antraštę.

```html {3}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.trippay.io/styles.css"></link>
  </head>
</html>
```

### Javascript

Įtraukite mūsų „Javascript“ kodą dokumento apačioje. (*Rekomenduojame iš karto virš galinės teksto žymos*).

```html {3}

```

Jūsų svetainė dabar paruošta ir gali rodyti mūsų žiniatinklio komponentus.

## Komponentai

### Mokėjimas

Mokėjimo komponentas leidžia informuoti „TripPay“, kad keliautojas nori ką nors įsigyti, ir sužinoti šių prekių taisykles bei kainas.

Valdikliui yra vienas privalomas atributas:

* `id`Norimos sudaryti rezervavimo sutarties identifikatorius.

```html {3}
<html>
  <body>
    <trip-pay id="<INSERT_BOOKING_CONTRACT_ID>"></trip-pay>
  </body>
</html>
```

:::note
Norėdami sužinoti, kaip sukurti rezervavimo sutartį`id`, perskaitykite vadovą:[Integracija su „TripPay“](/guides/integrating-with-trip-pay).
:::

Valdiklis parengia sutartį vykdymui ir rodo mokėjimo informacij&#x105;*(1 pav.)*&#x76;artotojui, kad užbaigtų užsakymą.
