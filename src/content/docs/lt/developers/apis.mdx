Vienas galinis taškas, skirtas gauti baltąją etiketę ir tinkinimo informaciją rezervavimo sistemai.

  Sužinokite daugiau
</LinkButton>


### Paieškos API

Programuotojams patogus būdas ieškoti individualiai pritaikyto kelionių inventoriaus įvairiuose geografiniuose regionuose.

  Sužinokite daugiau

## Mokėjimo API

Mokėjimo galiniai taškai skirti kūrėjams, norintiems įsigyti kelionių inventoriaus. Tai galima padaryti per API kaip registruotam kelionių agentui arba naudojant mūsų API kartu su mūsų PCI suderinamu mokėjimo valdikliu visiems kitiems subjektams.

Mokėjimo API suteikia galinius taškus filialams ir viešbučiams, kad jie galėtų stebėti užsakymus, analizę ir lėšų prieinamumą. Jie taip pat gali pasirinkti išsiimti turimas lėšas į savo banko sąskaitą.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Sužinokite daugiau
</LinkButton>
