## Mokėjimų API

Mokėjimų galiniai taškai skirti kūrėjams, norintiems įsigyti kelionių inventorių. Tai galima atlikti per API kaip registruotas kelionių agentas arba naudoti mūsų API kartu su PCI atitinkančiu mokėjimų valdikliu visoms kitoms organizacijoms.
Mokėjimų API suteikia filialams ir viešbučiams galimybę sekti užsakymus, analitiką ir lėšų prieinamumą. Jie taip pat gali pasirinkti išsiimti prieinamas lėšas į savo banko sąskaitą.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Sužinokite daugiau
</LinkButton>
