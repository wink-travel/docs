Mūsų sprendimas registruotiems kelionių agentams yra [Wink Agent](https://agent.wink.travel). Tai savitarnos portalas, kurį agentai gali naudoti ieškodami ir užsakydami klientų vardu bei uždirbdami komisinius už kiekvieną užsakymą.

Pažangesni kelionių agentai gali pasirinkti integruotis API lygiu, kad galėtų tvarkyti mokėjimus ir išmokas už Wink platformos ribų.

Kelionių agentai turi dalį tų pačių portalo funkcijų kaip [Wink Studio](/lt/studio/what-is-studio), todėl čia jų neaptarsime iš naujo.

## Paieška

Eikite į [Paieška](/lt/studio/search), kad sužinotumėte, kaip ieškoti ir užsakyti viešbučius.

Kitu atveju tęskite kitą skyrių.

## API

Kūrėjai, norintys valdyti `Travel Agent`, gali eiti į [Developers > API > Travel Agent](/lt/developers/apis/#travel-agent-api).
