Wink vartotojas gali pasirinkti leisti jums valdyti vieną iš jų turto, filialo ar kelionių agento paskyrų. Kai taip nutinka, gausite el. laišką su kvietimo užklausa.

import { Steps } from '@astrojs/starlight/components';

Norėdami peržiūrėti savo dabartinius Wink kvietimus, atlikite šiuos veiksmus:

<Steps>
1. Turite būti prisijungę prie vieno iš mūsų svetainių.
2. Spustelėkite savo profilio piktogramą viršutiniame dešiniajame kampe, ir pasirodys meniu.
3. Spustelėkite nuorodą `Invites`.
4. Sąveikaukite su savo kvietimais, juos arba `Priimdami`, arba `Atmesdami`.
</Steps>
