„Wink“ naudotojas gali pasirinkti leisti jums valdyti vieną iš jo nekilnojamojo turto, partnerio ar kelionių agento paskyrų. Kai tai įvyks, gausite el. laišką su kvietimo prašymu.

import { Steps } from '@astrojs/starlight/components';

Norėdami peržiūrėti dabartinius „Wink“ kvietimus, atlikite šiuos veiksmus:

<Steps>
  1. Turite būti prisijungę prie vienos iš mūsų svetainių.
  2. Viršutiniame dešiniajame kampe spustelėkite savo profilio piktogramą ir pasirodys meniu.
  3. Spustelėkite `Invites` nuoroda.
  4. Sąveikaukite su savo kvietimais vienu iš šių būdų: `Accepting` arba `Rejecting` juos.
</Steps>
