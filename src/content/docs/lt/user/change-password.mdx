Jei norite pakeisti slaptažodį savo Wink vartotojo paskyroje, atlikite šiuos veiksmus:

## Slaptažodžio atnaujinimas

<Steps>
1. Turite būti prisijungę prie vieno iš mūsų tinklalapių.
2. Spustelėkite savo profilio piktogramą viršutiniame dešiniajame kampe, atsiras meniu.
3. Spustelėkite nuorodą `Change password`.
4. Vadovaukitės nurodymais, kad pakeistumėte slaptažodį.
5. Spustelėkite `Save`.
</Steps>

Kitą kartą prisijungdami turėsite naudoti naują slaptažodį.

## Pamiršote slaptažodį

Kartais reikia atstatyti vartotojo slaptažodį.

Štai veiksmai, kaip atstatyti vartotojo slaptažodį:

<Steps>
1. Spustelėkite nuorodą `Forgot password` prisijungimo puslapyje.
2. Įveskite vartotojo vardą / el. paštą, susijusį su paskyra, kurią norite atstatyti.
3. Spustelėkite mygtuką `Forgot password`. *Būsite nukreipti į slaptažodžio atstatymo formą.*
4. Patikrinkite savo el. paštą.
</Steps>

2. Patvirtinkite naują slaptažodį.
3. Spustelėkite mygtuką `Reset password`. *Būsite nukreipti atgal į prisijungimo puslapį.*
</Steps>
