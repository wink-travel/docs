**Wink Drupal modulis** prideda Wink kelionių inventoriaus komponentus į jūsų Drupal svetainę kaip vietinius blokus. Redaktoriai gali juos talpinti bet kurioje išdėstymo vietoje per standartinę blokų sistemą.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Šaltinio kodas, kūrėjo diegimo vadovas ir prisidėjimo instrukcijos."
/>

## Reikalavimai

- Drupal 10 arba 11
- PHP 8.1+
- Composer

## Diegimas

<Steps>
1. Įdiekite per Composer:
   ```bash
   composer require drupal/wink
   ```
2. Įjunkite modulį:
   ```bash
   drush en wink -y
   ```
3. Eikite į **Administravimas → Konfigūracija → Tinklo paslaugos → Wink nustatymai** (`/admin/config/wink`).
4. Įveskite savo **Kliento ID** ir **Kliento slaptažodį**, pasirinkite aplinką ir spustelėkite **Išsaugoti**.
   Norėdami gauti savo kredencialus, eikite į [Programos](/lt/portal/settings/applications).
</Steps>

## Bloko pridėjimas puslapyje

<Steps>
1. Eikite į **Administravimas → Struktūra → Blokų išdėstymas**.
2. Spustelėkite **Įdėti bloką** regione, kuriame norite rodyti Wink turinį.
3. Ieškokite **Wink**, kad rastumėte galimus blokus:
   - **Wink Content** — rodo viešbučio kortelę, tinklelį arba žemėlapį
   - **Wink Lookup** — paieškos juosta paskirties vietoms
   - **Wink Search** — maršruto pasirinkimo mygtukas
   - **Wink Account** — prisijungimo mygtukas
   - **Wink Itinerary** — maršruto būsenos mygtukas
   - **Wink Cart** — pirkinių krepšelio mygtukas
4. **Wink Content** bloke sukonfigūruokite **Išdėstymo tipą** ir **Išdėstymo ID**.
5. Išsaugokite bloko konfigūraciją.
</Steps>

## Galimi blokai

| Blokas | Paskirtis |
|---|---|
| **Wink Content** | Rodo viešbučių korteles, reitinguotus tinklelius, žemėlapius ir kt. |
| **Wink Lookup** | Paieškos juosta paskirties vietoms ir viešbučiams |
| **Wink Search** | Piktogramos mygtukas, atveriantis maršruto pasirinkimo įrankį |
| **Wink Account** | Prisijungimo / vartotojo paskyros mygtukas |
| **Wink Itinerary** | Maršruto mygtukas, rodantis esamą kelionę |
| **Wink Cart** | Pirkinių krepšelio santraukos mygtukas |
