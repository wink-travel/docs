Qwik atnaujinamumo modelis reiškia, kad JavaScript vykdomas tingiai — tik kai vartotojas sąveikauja. Naudokite `useVisibleTask$`, kad vykdytumėte kodą, kuris turi būti paleistas naršyklėje, kai komponentas tampa matomas.

## Įdiegimas

```bash
npm install @wink/elements
```

## Įkėlimas su useVisibleTask$

`useVisibleTask$` yra Qwik gyvavimo ciklo kabliukas, skirtas tik naršyklėje vykdomam, aktyviai paleidžiamam kodui. Įdėkite jį į savo pagrindinį išdėstymą:

```tsx
// src/routes/layout.tsx
import { component$, useVisibleTask$ } from '@builder.io/qwik';
import { load } from '@wink/elements';

export default component$(() => {
  useVisibleTask$(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });

  return (
    <main>
      <Slot />
    </main>
  );
});
```

<Aside type="tip" title="useVisibleTask$ strategija">
Pagal numatytuosius nustatymus `useVisibleTask$` vykdomas, kai komponentas matomas vaizdo lauke. Pagrindiniam išdėstymui, kuris visada matomas, tai vyksta iš karto puslapio įkėlimo metu — tai atitinka `onMount` kitose sistemose.

Pateikite `{ strategy: 'document-ready' }`, jei norite, kad skriptas būtų įkeltas kuo anksčiau:
```ts
useVisibleTask$({ strategy: 'document-ready' }, () => {
  load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
});
```
</Aside>

## Naudojimas puslapyje

```tsx
// src/routes/hotels/index.tsx
import { component$ } from '@builder.io/qwik';

export default component$(() => {
  return (
    <main>
      <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
      <wink-lookup />
    </main>
  );
});
```

## TypeScript — deklaruokite JSX intrinsics

Qwik naudoja savo JSX vardų sritį. Išplėskite ją deklaracijos faile:

```ts
// src/wink-elements.d.ts
import type {
  WinkContentLoaderAttributes,
  WinkLookupAttributes,
  WinkSearchButtonAttributes,
  WinkAccountButtonAttributes,
  WinkItineraryButtonAttributes,
  WinkShoppingCartButtonAttributes,
  WinkAppLoaderAttributes,
} from '@wink/elements';

declare module '@builder.io/qwik' {
  namespace JSX {
    interface IntrinsicElements {
      'wink-content-loader': WinkContentLoaderAttributes;
      'wink-lookup': WinkLookupAttributes;
      'wink-search-button': WinkSearchButtonAttributes;
      'wink-account-button': WinkAccountButtonAttributes;
      'wink-itinerary-button': WinkItineraryButtonAttributes;
      'wink-shopping-cart-button': WinkShoppingCartButtonAttributes;
      'wink-app-loader': WinkAppLoaderAttributes;
    }
  }
}
```

## Aplinkos kintamieji

Qwik (pagrįstas Vite) naršyklei atskleidžia kintamuosius, prasidedančius `PUBLIC_`:

```
# .env
PUBLIC_WINK_CLIENT_ID=your-client-id
```

<Aside type="caution" title="Serverio pusės renderinimas">
Qwik atvaizduoja HTML serveryje. `<wink-*>` žymos išvedamos kaip statinis žymėjimas. CDN skriptas ir pasirinktinių elementų registracija vyksta kliento pusėje po `useVisibleTask$` paleidimo. Elementai trumpam bus neišspręstoje būsenoje, kol įkelsis CDN paketas — tai yra numatytas elgesys.
</Aside>
