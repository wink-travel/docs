Svelte perduoda atributus tiesiogiai į DOM be virtualaus DOM sluoksnio, todėl pasirinktinių elementų naudojimas yra natūralus. SvelteKit (SSR) atveju įkelkite CDN viduje `onMount` arba naudokite `browser` patikrinimą.

<LinkCard
  title="@wink/elements npm paketas"
  href="/integrations/npm"
  description="Įdiekite ir įkelkite CDN paketą su TypeScript tipais."
/>

## Įdiegimas

```bash
npm install @wink/elements
```

## SvelteKit — pagrindinis išdėstymas

```ts
// src/routes/+layout.ts
import { browser } from '$app/environment';
import { load as loadWink } from '@wink/elements';

export function load() {
  if (browser) {
    loadWink({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  }
}
```

Alternatyviai, įkelkite viduje `onMount` faile `+layout.svelte`:

```svelte
<!-- src/routes/+layout.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<slot />
```

## Naudojimas komponentuose

Svelte perduoda visus nežinomus atributus tiesiogiai į DOM, todėl papildoma konfigūracija nereikalinga:

```svelte
<script>
  export let layoutId = 'YOUR_LAYOUT_ID';
</script>

<main>
  <wink-content-loader layout="HOTEL" id={layoutId} />
  <wink-lookup />
</main>
```

## Svelte 5 (runes sintaksė)

```svelte
<script lang="ts">
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  let { layoutId }: { layoutId: string } = $props();

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id={layoutId} />
```

## Aplinkos kintamieji

SvelteKit naršyklei atskleidžia tik kintamuosius, prasidedančius `PUBLIC_`:

```
# .env
PUBLIC_WINK_CLIENT_ID=your-client-id
```

<Aside type="caution" title="SSR atvaizdavimas">
Niekada neatvaizduokite `<wink-*>` elementų serverio pusėje — pasirinktini elementai registruojami naršyklės `customElements` registre, kuris neegzistuoja serveryje. Apvyniokite `{#if browser}` arba kvieskite `load()` tik viduje `onMount`.
</Aside>

```svelte
<script>
  import { browser } from '$app/environment';
</script>

{#if browser}
  <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
{/if}
```

## Paprastas Svelte (Vite, be SvelteKit)

```svelte
<!-- src/App.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.VITE_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
```
