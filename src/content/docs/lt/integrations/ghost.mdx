Ghost įmontuota **Code Injection** funkcija leidžia pridėti scenarijus į globalų `<head>` ir `</body>` jūsų leidinyje. Visuose Ghost planuose palaikomas kodo įterpimas, todėl lengva įterpti Wink komponentus bet kuriame puslapyje ar įraše.

## Įkelkite Wink scenarijus visame tinklalapyje

<Steps>
1. Ghost administravimo skydelyje eikite į **Settings → Code Injection**.
2. Lauke **Site Header** pridėkite stiliaus lapą:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. Lauke **Site Footer** pridėkite scenarijų ir programos įkroviklį (pakeiskite `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Spauskite **Save**.
</Steps>

:::tip
Jūsų Kliento ID rasite savo Wink paskyroje skiltyje [Applications](/lt/portal/settings/applications).
:::

## Pridėkite komponentą į įrašą arba puslapį

Ghost redaktorius palaiko **HTML kortelę**, leidžiančią įterpti žalią HTML bet kur turinyje.

<Steps>
1. Atidarykite įrašą arba puslapį Ghost redaktoriuje.
2. Įveskite `/`, kad atidarytumėte kortelių pasirinkimą, ir pasirinkite **HTML**.
3. Įklijuokite komponento HTML, pavyzdžiui:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Spauskite už kortelės ribų peržiūrai, tada paskelbkite.
</Steps>

## Kodo įterpimas atskiram puslapiui

Jei komponentas turi būti rodomas tik viename puslapyje, Ghost taip pat palaiko kodo įterpimą atskiram puslapiui:

<Steps>
1. Atidarykite įrašo arba puslapio nustatymus (įrankių piktograma redaktoriuje).
2. Slinkite iki **Code Injection**.
3. Lauke **Footer** pridėkite komponento žymę (globalus app-loader turi būti tik svetainės pabaigoje).
4. Išsaugokite.
</Steps>

## Galimi komponentai

| Komponentas | HTML žyma | Paskirtis |
|---|---|---|
| Turinio įkroviklis | `<wink-content-loader>` | Rodo viešbučių korteles, tinklus, žemėlapius |
| Paieška | `<wink-lookup>` | Paieškos juosta kryptims |
| Paieškos mygtukas | `<wink-search-button>` | Atidaro maršruto pasirinkimą |
| Paskyros mygtukas | `<wink-account-button>` | Prisijungimas / vartotojo meniu |
| Maršruto mygtukas | `<wink-itinerary-button>` | Rodo esamą maršrutą |
| Maršruto pasirinkėjas | `<wink-itinerary-picker>` | Pilna maršruto forma |
| Prekių krepšelio mygtukas | `<wink-shopping-cart-button>` | Krepšelio santraukos mygtukas |

Pilną atributų sąrašą žr. [Web Components](/lt/integrations/web-components).
