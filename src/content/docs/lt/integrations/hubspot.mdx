HubSpot CMS palaiko pasirinktinius HTML modulius ir visos svetainės antraštės/poraštės kodo įterpimą, todėl lengva įterpti Wink komponentus į rinkodaros puslapius ir nukreipimo puslapius.

:::note
Visos svetainės kodo įterpimui reikalinga prieiga prie **HubSpot CMS Hub** (Starter arba aukštesnė versija). Pasirinktiniai HTML moduliai prieinami visuose CMS lygiuose.
:::

## Įkelkite Wink skriptus visoje svetainėje

<Steps>
1. HubSpote eikite į **Marketing → Files and Templates → Design Tools**.
2. Atidarykite savo aktyvios **temos** `main.css` arba naudokite globalias turinio nustatymų parinktis.
3. Arba eikite į **Settings → Website → Pages → Custom Code**.
4. Lauke **Head HTML** pridėkite stiliaus lapą:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
5. Lauke **Footer HTML** pridėkite skriptą ir programos įkroviklį (pakeiskite `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
6. Išsaugokite ir paskelbkite.
</Steps>

:::tip
Jūsų Kliento ID yra prieinamas jūsų Wink paskyroje skiltyje [Applications](/lt/portal/settings/applications).
:::

## Pridėkite komponentą per pasirinktinių HTML modulį

<Steps>
1. Design Tools sukurkite naują **modulį** tipo **Rich Text** arba **HTML**.
2. Modulio numatytame turinyje įklijuokite komponento HTML:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
3. Išsaugokite modulį ir nutempkite jį į bet kurį puslapio šabloną arba puslapį naudodami puslapio redaktorių.
</Steps>

## Pridėkite komponentą tiesiogiai puslapio redaktoriuje

Jei nenorite kurti modulio, galite naudoti **Custom HTML** sekciją vilkimo ir numetimo redaktoriuje:

<Steps>
1. Atidarykite puslapį HubSpot puslapio redaktoriuje.
2. Spustelėkite **Add** ir pasirinkite **Custom HTML** arba **Embed** sekciją.
3. Įklijuokite komponento žymę į HTML lauką.
4. Spustelėkite **Apply** ir paskelbkite puslapį.
</Steps>

## Galimi komponentai

| Komponentas | HTML žyma | Paskirtis |
|---|---|---|
| Turinio įkroviklis | `<wink-content-loader>` | Rodyti viešbučių korteles, tinklus, žemėlapius |
| Paieška | `<wink-lookup>` | Paieškos juosta kryptims |
| Paieškos mygtukas | `<wink-search-button>` | Atidaryti maršruto pasirinkimą |
| Paskyros mygtukas | `<wink-account-button>` | Prisijungimas / vartotojo meniu |
| Maršruto mygtukas | `<wink-itinerary-button>` | Rodyti esamą maršrutą |
| Maršruto pasirinkėjas | `<wink-itinerary-picker>` | Pilna maršruto forma |
| Prekių krepšelio mygtukas | `<wink-shopping-cart-button>` | Krepšelio santraukos mygtukas |

Pilną atributų sąrašą žr. [Web Components](/lt/integrations/web-components).
