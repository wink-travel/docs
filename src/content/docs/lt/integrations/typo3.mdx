**Wink TYPO3 Extension** integruoja Wink kelionių inventorių į TYPO3 13 kaip natūralius turinio elementus. Redaktoriai prideda Wink komponentus prie puslapių naudodami standartinį TYPO3 puslapio redaktorių — nereikia keisti šablonų.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"

  description="Šaltinio kodas, kūrėjo diegimo vadovas ir prisidėjimo instrukcijos."
/>

## Reikalavimai

- TYPO3 13.x
- PHP 8.2+
- Composer

## Diegimas

<Steps>
1. Įdiekite per Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktyvuokite plėtinį:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. TYPO3 administravimo skiltyje eikite į **Admin Tools → Settings → Extension Configuration → wink**.
4. Įveskite savo **Client ID**, **Client Secret** ir pasirinkite aplinką. Išsaugokite.
   Eikite į [Applications](/lt/portal/settings/applications), kad gautumėte savo prisijungimo duomenis.
5. Pridėkite statinį TypoScript šabloną prie savo pagrindinio šablono:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Pridėti turinio elementą prie puslapio

<Steps>
1. Atidarykite puslapį **Page** modulyje.
2. Spustelėkite mygtuką **+** turinio srityje, kad pridėtumėte naują elementą.
3. Pasirinkite vieną iš **Wink Travel** turinio elementų:
   - **Wink Content** — viešbučių kortelės, reitinguotos lentelės, žemėlapiai
   - **Wink Lookup** — kelionės tikslo paieškos juosta
   - **Wink Search** — maršruto pasirinkimo mygtukas
   - **Wink Account** — prisijungimo mygtukas
   - **Wink Itinerary** — maršruto būsenos mygtukas
   - **Wink Cart** — pirkinių krepšelio mygtukas
4. Naudodami **Wink Content** FlexForm laukus pasirinkite **Layout Type** ir įveskite **Layout ID**.
5. Išsaugokite ir paskelbkite puslapį.
</Steps>

## Galimi turinio elementai

| Elementas | Paskirtis |
|---|---|
| **Wink Content** | Viešbučių kortelės, reitinguotos lentelės, žemėlapiai ir daugiau |
| **Wink Lookup** | Kelionės tikslo paieškos juosta |
| **Wink Search** | Piktogramos mygtukas, atveriantis maršruto pasirinkiklį |
| **Wink Account** | Prisijungimo / vartotojo paskyros mygtukas |
| **Wink Itinerary** | Maršruto mygtukas, rodantis esamą kelionę |
| **Wink Cart** | Pirkinių krepšelio santraukos mygtukas |
