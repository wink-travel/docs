Studio siūlo keletą įterpiamų valdiklių, kuriuos galite pridėti prie savo svetainės, kad skatintumėte užsakymus.

Norėdami pasiekti įterpiamus valdiklius, eikite į `Tools > Miscellaneous` šoniniame meniu.

## Galimi valdikliai

| Valdiklis | Aprašymas |
| -- | -- |
| **Paskyros mygtukas** | Prekės ženklu pažymėtas mygtukas, vedantis į jūsų filialo puslapį |
| **Maršruto mygtukas** | Mygtukas, atveriantis maršruto pasirinkimo įrankį |
| **Maršruto forma** | Įterpta datos ir svečių pasirinkimo forma |
| **Paieškos mygtukas** | Mygtukas, atveriantis apgyvendinimo paiešką |
| **Paieškos mygtukas** | Mygtukas, skirtas ieškoti konkrečios inventoriaus dalies |

## Kaip įterpti

Kiekvienas valdiklis pateikia kodo fragmentą, kurį galite nukopijuoti ir įklijuoti į savo svetainės HTML. Fragmentas apima jūsų **Kliento ID** ir **Individualizavimo ID**, todėl valdiklis automatiškai susiejamas su jūsų filialo paskyra.

:::note
Jei naudojate WordPress, taip pat siūlome WordPress įskiepį, palengvinantį integraciją.
:::
