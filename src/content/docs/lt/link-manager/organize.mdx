Norėdami valdyti savo WinkLinks žymas, eikite į `WinkLinks` pagrindinėje navigacijos juostoje ir spustelėkite skirtuką `Tags`.

Kai jūsų turinys augs, norėsite jį pradėti organizuoti. Tam sukūrėme `tags`. Galite grupuoti turinį naudodami tiek žymų, kiek norite.

## Sukurti žymą

<Steps>
1. Sukurkite naują žymą spustelėdami mygtuką `Add a new tag`.
2. Įveskite naujos žymos pavadinimą. *pvz., Top coffee shops in Berlin*
3. Spustelėkite nuorodą `Save`.
</Steps>

## Atnaujinti žymą

<Steps>
1. Pradėkite spustelėdami nuorodą `Edit` šalia žymos, kurią norite redaguoti.
2. Atnaujinkite žymos pavadinimą. *pvz., Best coffee shops in Berlin*
3. Spustelėkite nuorodą `Save`.
</Steps>

## Pašalinti žymą

:::note
Pašalinus žymą, ji bus pašalinta ir iš viso susieto turinio.
:::

<Steps>
1. Spustelėkite nuorodą `Remove` šalia žymos, kurią norite pašalinti.
</Steps>

## API

Kūrėjai, norintys valdyti `WinkLinks`, gali eiti į [Developers > API > WinkLinks](/lt/developers/apis/#winklinks-api).

## Tolimesnis skaitymas

- Sužinokite daugiau apie [turinio grupavimą](/lt/link-manager/content#grouping-content).
