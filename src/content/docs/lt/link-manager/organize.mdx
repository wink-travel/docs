Norėdami tvarkyti „WinkLinks“ žymas, eikite į`WinkLinks`pagrindinėje naršymo juostoje ir spustelėkite`Tags`skirtukas.

Turiniui augant, norėsite pradėti jį tvarkyti. Mes sukūrėme`tags`šiuo tikslu. Galite grupuoti turinį naudodami tiek žymų, kiek norite.

## Sukurti žymą

<Steps>
  1. Sukurkite naują žymą spustelėdami `Add a new tag` mygtukas.
  2. Įveskite naujos žymos pavadinimą. *pvz., geriausios kavinės Berlyne*
  3. Spustelėkite `Save` nuoroda.
</Steps>


## Atnaujinti žymą

<Steps>
  1. Pradėkite spustelėdami `Edit` nuorodą šalia norimos modifikuoti žymos.
  2. Atnaujinkite žymos pavadinimą. *pvz., geriausios kavinės Berlyne*
  3. Spustelėkite `Save` nuoroda.
</Steps>


## Pašalinti žymą

:::note
Pašalinus žymę, ji bus pašalinta ir iš viso susijusio turinio.
:::

<Steps>
  1. Spustelėkite `Remove` nuorodą šalia žymos, kurią norite pašalinti.
</Steps>


## API

Kūrėjai, norintys valdyti`WinkLinks`gali nueiti į[Kūrėjai > API > „WinkLinks“](/developers/apis/#winklinks-api).
