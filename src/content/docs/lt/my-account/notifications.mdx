Kartais norime su jumis susisiekti. Kai taip nutinka, viršutiniame dešiniajame kampe esantis varpelis pasikeičia.
Norėdami perskaityti, ką jums atsiuntėme:

<Steps>
1. Spustelėkite `Paskyra > Pranešimai` pagrindinėje navigacijos juostoje.
2. Pamatysite naujus ir ankstesnius pranešimus.
3. Pasirinkite `Pažymėti kaip perskaitytą` arba `Ištrinti` pranešimus, kai juos perskaitysite.
</Steps>

## API

Kūrėjai, norintys valdyti `Partnerius`, gali eiti į [Kūrėjai > API > Partneris](/lt/developers/apis/#affiliate-api).

## Tolimesnis skaitymas

- Sužinokite daugiau apie tai, kaip [priimti / atmesti kvietimus](/lt/user/invites).
