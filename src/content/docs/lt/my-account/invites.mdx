Wink vartotojas gali pasirinkti leisti jums valdyti vieną iš jų turto, filialo ar kelionių agento paskyrų. Kai taip nutinka, gausite el. laišką su kvietimo prašymu.

import { Steps } from '@astrojs/starlight/components';

Norėdami pamatyti savo dabartinius Wink kvietimus, atlikite šiuos veiksmus:

<Steps>
 1. Prisijunkite prie [my.wink.travel](https://my.wink.travel).
 2. Kairėje šoninėje juostoje spustelėkite `Invites`.
 3. Pamatysite visų dabartinių kvietimų sąrašą.
 4. Šiame skyriuje bus išvardytos visos jūsų paraiškos.
 5. Pasirinkite `Accept` arba `Reject` kvietimą.
 6. Jei priėmėte kvietimą, paskyra bus rodoma skyriuje `Accounts`.
</Steps>
