Wink provides a reporting system that lets you generate and export data across your account. Reports are available in CSV format for easy import into spreadsheets, analytics tools, or your own systems.

## Report types

You can generate reports for the following data categories:

| Report type | What it includes |
|---|---|
| **Property** | Property details, listing status, inventory summary. |
| **User** | User accounts, roles, activity summary. |
| **Booking** | Booking history, statuses, revenue, guest details. |
| **Account** | Account information, balances, transaction history. |

## How to generate a report

1. Navigate to the reporting section in your account dashboard.
2. Select the **report type** you want to generate.
3. Configure any **filters** to narrow the data (e.g., date range, status, property).
4. Choose the **fields** you want included in the export. Each report type offers a set of available fields you can select.
5. Click **Export** to download the report as a CSV file.

## Custom fields

Reports support customizable field selection. You choose which columns to include in your export, so you get exactly the data you need — nothing more, nothing less.

## Use cases

- **Revenue reconciliation** — Export booking reports with financial details to reconcile with your accounting system.
- **Performance analysis** — Export property reports to compare listing performance across your portfolio.
- **Guest insights** — Export booking data to understand traveler demographics and booking patterns.
- **Account auditing** — Export account data for compliance or internal review.
