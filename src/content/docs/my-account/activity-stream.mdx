The activity stream provides a chronological log of events and actions across your Wink account. It helps you track what's happening with your properties, bookings, and account in one place.

## What gets tracked

The activity stream captures key events including:

- **Booking events** — New bookings, cancellations, and modifications.
- **Review events** — Guest review submissions and responses.
- **Account changes** — Profile updates, manager additions, status changes.
- **Content updates** — Property, room, and facility description changes.
- **Media uploads** — Photos and videos added to your listings.

Each activity entry includes a timestamp, the user who performed the action (when applicable), and a description of what happened.

## Visibility levels

Activity entries have different visibility levels depending on the type of event:

| Level | Who can see it |
|---|---|
| **Everyone** | Publicly visible activity. |
| **Authenticated** | Visible to any logged-in user. |
| **Owner** | Visible only to the user who created the entry. |
| **Admin** | Visible only to administrators. |

You only see activity entries that match your access level.

## Viewing your activity stream

Navigate to your account dashboard to see your activity stream. Events are displayed in reverse chronological order — the most recent activity appears first.

## Use cases

- **Audit trail** — See who made changes to your property or account, and when.
- **Booking monitoring** — Keep track of new bookings and cancellations in real time.
- **Team coordination** — When multiple managers work on a property, the activity stream shows everyone's actions.
- **Review management** — See when new reviews come in so you can respond promptly.
