Aby zarządzać tagami WinkLinks, przejdź do `WinkLinks` w głównym pasku nawigacyjnym i kliknij zakładkę `Tags`.

W miarę rozwoju Twojej zawartości, zechcesz zacząć ją organizować. Stworzyliśmy do tego celu `tagi`. Możesz grupować zawartość, używając dowolnej liczby tagów.

## Utwórz tag

<Steps>
1. Utwórz nowy tag, klikając przycisk `Add a new tag`.
2. Wprowadź nazwę nowego tagu. *np. Najlepsze kawiarnie w Berlinie*
3. Kliknij link `Save`.
</Steps>

## Aktualizuj tag

<Steps>
1. Zacznij od kliknięcia linku `Edit` obok tagu, który chcesz zmodyfikować.
2. Zaktualizuj nazwę tagu. *np. Najlepsze kawiarnie w Berlinie*
3. Kliknij link `Save`.
</Steps>

## Usuń tag

:::note
Usuwając tag, usuniesz go również ze wszystkich powiązanych treści.
:::

<Steps>
1. Kliknij link `Remove` obok tagu, który chcesz usunąć.
</Steps>

## API

Deweloperzy, którzy chcą zarządzać `WinkLinks`, mogą przejść do [Developers > API > WinkLinks](/pl/developers/apis/#winklinks-api).
