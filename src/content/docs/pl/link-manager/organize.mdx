Aby zarządzać tagami WinkLinks, przejdź do`WinkLinks`na głównym pasku nawigacyjnym i kliknij na`Tags`zakładka.

W miarę jak Twoja treść będzie rosła, będziesz chciał zacząć ją organizować. Stworzyliśmy`tags`w tym celu. Możesz grupować treści, używając tylu tagów, ile chcesz.

## Utwórz tag

<Steps>
  1. Utwórz nowy tag klikając `Add a new tag` przycisk.
  2. Wprowadź nazwę nowego tagu. *np. Najlepsze kawiarnie w Berlinie*
  3. Kliknij `Save` połączyć.
</Steps>


## Aktualizuj tag

<Steps>
  1. Zacznij od kliknięcia `Edit` link obok tagu, który chcesz zmodyfikować.
  2. Zaktualizuj nazwę swojego tagu. *np. Najlepsze kawiarnie w Berlinie*
  3. Kliknij `Save` połączyć.
</Steps>


## Usuń tag

:::uwaga
Usuwając tag, usuniesz go również ze wszystkich powiązanych treści.
:::

<Steps>
  1. Kliknij `Remove` link obok tagu, który chcesz usunąć.
</Steps>


## API

Programiści chcący zarządzać`WinkLinks`możesz udać się do[Deweloperzy > API > WinkLinks](/developers/apis/#winklinks-api).
