**Moduł Wink Drupal** dodaje komponenty inwentarza podróży Wink do Twojej strony Drupal jako natywne bloki. Redaktorzy mogą umieszczać je w dowolnym miejscu układu za pomocą standardowego systemu bloków.

  title="wink-travel/wink-drupal-module na GitHub"

## Wymagania

- Drupal 10 lub 11
- PHP 8.1+
- Composer

## Instalacja

<Steps>
1. Zainstaluj przez Composer:
   ```bash
   composer require drupal/wink
   ```
2. Włącz moduł:
   ```bash
   drush en wink -y
   ```
3. Przejdź do **Administracja → Konfiguracja → Usługi sieciowe → Ustawienia Wink** (`/admin/config/wink`).
4. Wprowadź swój **Client ID** i **Client Secret**, wybierz środowisko i kliknij **Zapisz**.
   Przejdź do [Aplikacje](/pl/portal/settings/applications), aby pobrać swoje dane uwierzytelniające.
</Steps>

## Dodanie bloku do strony

<Steps>
1. Przejdź do **Administracja → Struktura → Układ bloków**.
2. Kliknij **Umieść blok** w regionie, w którym chcesz dodać zawartość Wink.
3. Wyszukaj **Wink**, aby znaleźć dostępne bloki:
   - **Wink Content** — wyświetla kartę hotelu, siatkę lub mapę
   - **Wink Lookup** — pasek wyszukiwania destynacji
   - **Wink Search** — przycisk wyboru planu podróży
   - **Wink Account** — przycisk logowania
   - **Wink Itinerary** — przycisk statusu planu podróży
   - **Wink Cart** — przycisk koszyka zakupów
4. Dla **Wink Content** skonfiguruj **Typ układu** i **ID układu** w ustawieniach bloku.
5. Zapisz konfigurację bloku.
</Steps>

## Dostępne bloki

| Blok | Przeznaczenie |
|---|---|
| **Wink Content** | Wyświetla karty hoteli, rankingi, mapy i inne |
| **Wink Lookup** | Pasek wyszukiwania destynacji i hoteli |
| **Wink Search** | Ikona przycisku otwierającego wybór planu podróży |
| **Wink Account** | Przycisk logowania / konta użytkownika |
| **Wink Itinerary** | Przycisk planu podróży pokazujący aktualną wycieczkę |
| **Wink Cart** | Przycisk podsumowania koszyka zakupów |
