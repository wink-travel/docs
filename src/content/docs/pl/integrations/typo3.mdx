**Rozszerzenie Wink TYPO3** integruje inwentarz podróży Wink z TYPO3 13 jako natywne elementy treści. Redaktorzy dodają komponenty Wink do stron za pomocą standardowego edytora stron TYPO3 — nie są wymagane zmiany w szablonach.

## Wymagania

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instalacja

<Steps>
1. Zainstaluj przez Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktywuj rozszerzenie:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. W backendzie TYPO3 przejdź do **Narzędzia administracyjne → Ustawienia → Konfiguracja rozszerzeń → wink**.
4. Wprowadź swój **Client ID**, **Client Secret** i wybierz środowisko. Zapisz.
   Przejdź do [Aplikacje](/pl/portal/settings/applications), aby pobrać swoje dane uwierzytelniające.
5. Dodaj statyczny szablon TypoScript do swojego szablonu głównego:
   **Szablon → Edytuj → Zawiera → Dostępne elementy → Wink Travel**.
</Steps>

## Dodawanie elementu treści do strony

<Steps>
1. Otwórz stronę w module **Strona**.
2. Kliknij przycisk **+** w obszarze treści, aby dodać nowy element.
3. Wybierz jeden z elementów treści **Wink Travel**:
   - **Wink Content** — karty hoteli, siatki rankingowe, mapy
   - **Wink Lookup** — pasek wyszukiwania destynacji
   - **Wink Search** — przycisk wyboru planu podróży
   - **Wink Account** — przycisk logowania
   - **Wink Itinerary** — przycisk statusu planu podróży
   - **Wink Cart** — przycisk koszyka zakupów
4. Dla **Wink Content** użyj pól FlexForm, aby wybrać **Typ układu** i wprowadzić **ID układu**.
5. Zapisz i opublikuj stronę.
</Steps>

## Dostępne elementy treści

| Element | Przeznaczenie |
|---|---|
| **Wink Content** | Karty hoteli, siatki rankingowe, mapy i inne |
| **Wink Lookup** | Pasek wyszukiwania destynacji |
| **Wink Search** | Ikona przycisku otwierająca wybór planu podróży |
| **Wink Account** | Przycisk logowania / konta użytkownika |
| **Wink Itinerary** | Przycisk planu podróży pokazujący aktualną wycieczkę |
| **Wink Cart** | Przycisk podsumowania koszyka zakupów |
