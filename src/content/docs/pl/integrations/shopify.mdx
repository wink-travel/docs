**Wink Shopify App** pozwala organizatorom wycieczek, sprzedawcom doświadczeń i sprzedawcom z branży hotelarskiej wyświetlać na żywo inwentarz Wink bezpośrednio na witrynie Shopify — obok produktów, na stronach docelowych lub w dowolnym miejscu w motywie.

## Instalacja

<Steps>
1. Przejdź do [Shopify App Store](https://apps.shopify.com) i wyszukaj **Wink Travel**.
2. Kliknij **Add app** i postępuj zgodnie z procesem OAuth, aby autoryzować.
3. W ustawieniach aplikacji Wink wpisz swój **Client ID** i **Client Secret**.
   Przejdź do [Applications](/pl/portal/settings/applications), aby pobrać swoje dane uwierzytelniające.
4. Kliknij **Verify & Save**. Pojawi się komunikat potwierdzający połączenie.
</Steps>

## Włącz blok osadzania aplikacji

Blok osadzania aplikacji automatycznie wstrzykuje skrypt i arkusz stylów Wink na każdą stronę — nie musisz edytować kodu motywu.

<Steps>
1. W panelu Shopify Admin przejdź do **Online Store → Themes → Customize**.
2. Kliknij **App embeds** w lewym panelu.
3. Przełącz **Wink Travel** na włączony.
4. Kliknij **Save**.
</Steps>

:::note
Blok osadzania aplikacji odczytuje Twój Client ID z metafield Shopify ustawionego podczas instalacji. Twój Client Secret nigdy nie jest ujawniany w przeglądarce.
:::

## Dodaj komponent do strony

<Steps>
1. W edytorze motywu otwórz stronę lub sekcję, gdzie chcesz umieścić zawartość Wink.
2. Kliknij **Add block** i wybierz spośród bloków Wink:
   - **Wink Content** — karty hoteli, siatki lub mapy
   - **Wink Lookup** — pasek wyszukiwania destynacji
   - **Wink Search** — przycisk wyboru planu podróży
   - **Wink Account** — przycisk logowania
   - **Wink Itinerary** — przycisk statusu planu podróży
   - **Wink Cart** — przycisk koszyka zakupów
3. Dla **Wink Content** wybierz **Layout Type** i wpisz **Layout ID** z Wink Studio.
4. Kliknij **Save**.
</Steps>

## Dostępne bloki motywu

| Blok | Przeznaczenie |
|---|---|
| **Wink Content** | Wyświetla karty hoteli, siatki rankingowe, mapy i inne |
| **Wink Lookup** | Pasek wyszukiwania destynacji i hoteli |
| **Wink Search** | Ikona przycisku otwierającego wybór planu podróży |
| **Wink Account** | Przycisk logowania / konta użytkownika |
| **Wink Itinerary** | Przycisk planu podróży pokazujący aktualną wycieczkę |
| **Wink Cart** | Przycisk podsumowania koszyka zakupów |
