**Rozszerzenie Wink Joomla** to pakiet instalujący wtyczkę systemową (ładuje skrypty CDN), wtyczkę treści (przetwarza shortcode’y w artykułach) oraz komponent (strona ustawień administratora).

## Wymagania

- Joomla 4 lub 5
- PHP 8.1+

## Instalacja

<Steps>
1. Pobierz najnowszy plik `pkg_wink-x.x.x.zip` ze [strony wydań GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. W panelu Joomla przejdź do **Rozszerzenia → Zarządzaj → Instaluj**.
3. Prześlij plik zip i kliknij **Prześlij i zainstaluj**.
4. Obie wtyczki zostaną automatycznie włączone. Przejdź do **Komponenty → Ustawienia Wink**.
5. Wprowadź swój **Client ID** i **Client Secret**, a następnie kliknij **Zapisz**.
   Aby pobrać dane uwierzytelniające, przejdź do [Aplikacje](/pl/portal/settings/applications).
</Steps>

## Osadzanie komponentów w artykułach

Używaj tagów w stylu shortcode w dowolnym miejscu treści artykułu:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Wtyczka treści zastępuje te tagi odpowiednim kodem HTML `<wink-*>` podczas renderowania. Wtyczka systemowa automatycznie ładuje skrypty CDN i wstrzykuje `<wink-app-loader>` w stopce strony.

## Dostępne shortcode’y

| Shortcode | Wyjście | Opis |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Karty hoteli, siatki, mapy |
| `{wink-lookup}` | `<wink-lookup>` | Pasek wyszukiwania destynacji |
| `{wink-search}` | `<wink-search-button>` | Przycisk wyboru planu podróży |
| `{wink-account}` | `<wink-account-button>` | Logowanie / konto użytkownika |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Przycisk statusu planu podróży |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Przycisk koszyka |
