Studio oferuje kilka osadzalnych widżetów, które możesz dodać do własnej strony internetowej, aby zwiększyć liczbę rezerwacji.

Aby uzyskać dostęp do widżetów do osadzenia, przejdź do `Narzędzia > Różne` w pasku bocznym.

## Dostępne widżety

| Widżet | Opis |
| -- | -- |
| **Przycisk konta** | Markowy przycisk prowadzący do Twojej strony afiliacyjnej |
| **Przycisk planu podróży** | Przycisk otwierający selektor planu podróży |
| **Formularz planu podróży** | Formularz wyboru dat i gości w linii |
| **Przycisk wyszukiwania** | Przycisk otwierający wyszukiwanie obiektów |
| **Przycisk wyszukiwania w inwentarzu** | Przycisk do wyszukiwania konkretnego inwentarza |

## Jak osadzić

Każdy widżet dostarcza fragment kodu, który możesz skopiować i wkleić do HTML swojej strony internetowej. Fragment zawiera Twój **Client ID** i **Customization ID**, dzięki czemu widżet jest automatycznie powiązany z Twoim kontem afiliacyjnym.

:::note
Jeśli korzystasz z WordPressa, oferujemy również wtyczkę WordPress ułatwiającą integrację.
:::
