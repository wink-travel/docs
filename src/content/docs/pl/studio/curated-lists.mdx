Listy kuratorskie to funkcja, która umożliwia grupowanie i sortowanie asortymentu wybranego spośród rzeczy znalezionych podczas wyszukiwania asortymentu turystycznego.
Po utworzeniu możesz udostępnić listę swoim użytkownikom, udostępniając ją do sprzedaży jako[siatka](/studio/grids)lub[mapa](/studio/maps) Poprzez [Linki do mrugnięć](/link-manager/wink-links), nasz [Wtyczka WordPress](/developers/wordpress/)lub poprzez osadzenie jednego z naszych[Komponenty sieciowe](/developers/web-components)do Twojej witryny.

Domyślnie każde konto partnerskie zaczyna się od`Favorites`lista, z której możesz skorzystać od razu.`Favorites`jest specjalny i nie można go usunąć.
Wszystkie listy kuratorskie pozwalają na wykonanie następujących czynności:

## Utwórz listę

<figure>
  <CloudinaryImage width="350" height="219" resize="scale" quality="best" alt="Utwórz listę selekcjonowaną" src="academy/create-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Utwórz nowy formularz listy selekcjonowanej*</figcaption>
</figure>


* Z pulpitu nawigacyjnego kliknij`Inventory > Curated Lists`na pasku nawigacyjnym.
* Utwórz nową listę selekcji, klikając`Make a new list` przycisk.
* Zostaniesz poproszony o nadanie nazwy liście.*Przykład: 3 najlepsze typy z Nowego Jorku*.
* Kliknij`Save` przycisk.

## Dodaj element listy

Możesz dodać element listy do dowolnej listy z wyników wyszukiwania.

* Idź do[Szukaj](/studio/search)
* Wyszukaj element, który chcesz dodać do listy.
* Po znalezieniu kliknij`Actions`link w lewym dolnym rogu karty wyników wyszukiwania.
* Kliknij`Add to...`a następnie wybierz listę, do której chcesz dodać ten inwentarz.
* Wróć na stronę z listami selekcji, aby sprawdzić, czy element został dodany do Twojej listy.

  <figcaption>*Skopiuj element listy selekcjonowanej*</figcaption>

Możesz skopiować element listy, klikając ikonę kopiowania i wybierając listę, do której chcesz skopiować element.

Możesz przenieść element listy, klikając ikonę kopiowania i wybierając listę, do której chcesz przenieść element.

<figure>
  <CloudinaryImage width="350" height="216" resize="scale" quality="best" alt="Usuń element listy selekcjonowanej" src="academy/remove-curated-list-item" errorImage="noimage_opaque_nyrtl0.png" />

## Lista działań

<figure>

Każda lista ma pewne akcje, które można na niej wykonać.

* Przekonwertuj listę na[mapa](/studio/maps).
* Przekonwertuj listę na[siatka](/studio/grids).
* Usuń listę (*z wyjątkiem Ulubionych*).

## API

Programiści chcący zarządzać listami kuratorskimi mogą przejść do[Deweloperzy > API > Przeglądaj](/developers/apis/#browse-api).
