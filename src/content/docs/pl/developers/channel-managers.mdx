Wink integruje się z wiodącymi channel managerami, systemami zarządzania nieruchomościami (PMS) oraz centralnymi systemami rezerwacyjnymi (CRS), aby utrzymać synchronizację stawek i dostępności we wszystkich kanałach sprzedaży.

## Jak to działa

Integracje z channel managerami umożliwiają dwukierunkowy przepływ danych między Twoim systemem zarządzania nieruchomościami a Wink:

1. **Wysyłanie stawek** — Twój channel manager przesyła aktualizacje stawek do Wink. Gdy aktualizujesz ceny w swoim PMS, zmiany te są automatycznie odzwierciedlane w Wink.
2. **Synchronizacja dostępności** — Twój channel manager przesyła aktualizacje dostępności do Wink. Gdy pokoje są rezerwowane lub zwalniane w PMS, zapasy w Wink aktualizują się w czasie rzeczywistym.
3. **Powiadomienia o rezerwacjach** — Gdy rezerwacja zostanie dokonana w Wink, Twój channel manager otrzymuje powiadomienie, aby rezerwacja pojawiła się w PMS.

## Obsługiwane channel managery

Wink obecnie integruje się z następującymi systemami:

| Channel Manager | Opis |
|---|---|
| **Channex** | Globalna platforma dystrybucji i zarządzania kanałami. |
| **TravelClick** | Rozwiązania hotelowe i zarządzanie kanałami (Amadeus). |
| **Omnibees** | Dystrybucja i zarządzanie przychodami dla hoteli. |
| **D-Edge** | Technologia hotelarska i centralne rozwiązania rezerwacyjne. |
| **Allotz** | Alokacja zapasów i zarządzanie kanałami. |
| **Ezee / Yanolja** | Chmurowy PMS i zarządzanie kanałami skoncentrowane na rynku azjatyckim. |
| **RateGain** | Technologia inteligencji cenowej i dystrybucji. |
| **RateTiger** | Dystrybucja stawek i dostępności w czasie rzeczywistym. |
| **CloudBeds** | Chmurowy PMS, channel manager i silnik rezerwacji. |
| **SiteMinder** | Wiodący channel manager hotelowy i platforma dystrybucji. |
| **Exely** | Synchronizacja stawek i dostępności dla hoteli. |

:::note
Jeśli Twój channel manager nie jest na liście, skontaktuj się z zespołem Wink. Nowe integracje mogą być dodane na podstawie standardów protokołu OTA (Open Travel Alliance), które wspiera Wink.
:::

## Dla deweloperów

Jeśli tworzysz integrację channel managera lub PMS z Wink, użyj punktów końcowych Channel Manager API:

- **Aktualizacje stawek** — Wysyłaj zmiany stawek do Wink.
- **Aktualizacje dostępności** — Wysyłaj zmiany dostępności do Wink.
- **Pobieranie rezerwacji** — Pobieraj rezerwacje z Wink do swojego systemu.

Channel Manager API stosuje standardy protokołu OTA (SOAP/XML) dla kompatybilności z istniejącymi systemami hotelarskimi.

## Powiadomienia webhook

Możesz subskrybować zdarzenia webhook channel managera, aby otrzymywać powiadomienia w czasie rzeczywistym:

- `channel-manager.update.rate` — Otrzymano aktualizację stawek.
- `channel-manager.update.availability` — Otrzymano aktualizację dostępności.
- `channel-manager.update` — Ogólna aktualizacja channel managera.

Szczegóły znajdziesz w [Katalogu zdarzeń webhook](/pl/developers/webhook-events#channel-manager-events).
