Webhooki umożliwiają odbieranie i reagowanie na zdarzenia w czasie rzeczywistym z Wink i TripPay dotyczące Twojego konta.

## Przykład

<Steps>
  1. Rezerwacja zostanie dokonana na Wink.
  2. Wydarzenie Wink `booking.created` zostaje wysłany.
  3. Twój system aktywnie nasłuchuje tego zdarzenia.
  4. Twój system śledzi własny zestaw analiz i wykorzystuje to zdarzenie do aktualizowania swojej bazy danych szeregów czasowych.
</Steps>

:::uwaga
Webhook Wink różni się od webhooka TripPay.

* Jeśli chcesz słuchać zdarzeń Wink, możesz utworzyć webhook z[Ekstranet](https://extranet.wink.travel) Lub [Studio](https://studio.wink.travel).
* Jeśli chcesz słuchać wydarzeń TripPay, możesz utworzyć webhook z[Zapłata](https://pay.wink.travel).
  :::

*W tym przykładzie uwierzytelnimy się za pomocą[Studio](https://studio.wink.travel).*

<Steps>
  1. Po uwierzytelnieniu kliknij ikonę swojego profilu w prawym górnym rogu, a pojawi się menu.
  2. Kliknij `Webhooks` połączyć.
  3. Zostaniesz przekierowany na stronę webhooków.
</Steps>


## Utwórz webhook

:::ostrożność
Musisz utworzyć konto partnerskie w Wink lub konto w TripPay przed utworzeniem webhooka, ponieważ będziesz wiązać swój webhook z konkretnym kontem.
:::

Oto kroki tworzenia webhooka:

<Steps>
  1. Na stronie webhooków kliknij `Create new webhook` przycisk.
  2. ***Nazwa*** Nadaj nazwę swojemu webhookowi. *np. Wink Stalker*
  3. ***Włączać*** Pozostaw bez zmian, aby włączyć.
  4. **Nazwa jednostki** Połącz swój webhook z jednym ze swoich istniejących kont. *np. Fajne konto*
  5. **Wydarzenia** Posłuchaj wydarzeń, które Cię interesują. *np. `booking.created`*
  6. **Adres URL webhooka** Wprowadź adres URL, pod którym będziesz słuchał naszych wydarzeń. *np. `https://my.system.com/webhook/wink`*
  7. Kliknij `Save` przycisk, aby kontynuować.
</Steps>


Nastąpi przekierowanie z powrotem do listy webhooków, a nowy webhook będzie gotowy do użycia.

## Aktualizuj webhook

Musisz zaktualizować swoją aplikację, jeśli chcesz:

* Wyłącz swój webhook.
* Słuchaj więcej / mniej wydarzeń.
* Zmień adres URL webhooka nasłuchującego.

Aby zaktualizować istniejący webhook, wykonaj następujące czynności:

<Steps>
  1. Kliknij `Actions` link do webhooka, który chcesz zaktualizować.
  2. Kliknij `Update` przycisk pod Akcjami.
  3. Wprowadź zmiany w webhooku.
  4. Kliknij `Save` przycisk, aby kontynuować.
</Steps>


Zostaniesz przekierowany z powrotem do listy webhooków.

## Usuń webhook

Jeśli nie potrzebujesz już webhooka, możesz go usunąć.

Aby usunąć webhook, wykonaj następujące czynności:

<Steps>
  1. Kliknij `Actions` link do webhooka, który chcesz zaktualizować.
  2. Kliknij `Remove` przycisk pod Akcjami.
  3. Trzask `OK` przycisk potwierdzający usunięcie.
</Steps>


Zostaniesz przekierowany z powrotem do listy webhooków.
