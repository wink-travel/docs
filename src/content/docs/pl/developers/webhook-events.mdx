Ta strona zawiera listę wszystkich zdarzeń webhook, na które możesz się zapisać w Wink. Instrukcje konfiguracji znajdziesz w [Przewodniku integracji webhook](/pl/guides/webhook-integration) lub na [stronie zarządzania webhookami](/pl/my-account/webhooks).

## Jak się zapisać

1. Przejdź do **Aplikacje > Webhooki** w swoim portalu.
2. Utwórz webhook i wybierz zdarzenia, które chcesz otrzymywać.
3. Wink wysyła żądanie HTTP POST do Twojego endpointu za każdym razem, gdy zdarzenie wystąpi.

Pełne kroki konfiguracji i najlepsze praktyki znajdziesz w [Integracji webhook](/pl/guides/webhook-integration).

## Zdarzenia konta

Zdarzenia związane z cyklem życia i ustawieniami konta TripPay.

| Zdarzenie | Klucz | Opis |
|---|---|---|
| Konto utworzone | `account.create` | Utworzono nowe konto. |
| Konto w trakcie weryfikacji | `account.under_review` | Konto oczekuje na weryfikację. |
| Konto zaktualizowane | `account.updated` | Dane konta zostały zmienione. |
| Status konta zaktualizowany | `account.status.updated` | Status konta uległ zmianie (np. zatwierdzone, zawieszone). |
| Adres konta zaktualizowany | `account.address.updated` | Adres konta został zmodyfikowany. |
| Szczegóły konta zaktualizowane | `account.details.updated` | Pola szczegółów konta zostały zmienione. |
| Informacje o koncie zaktualizowane | `account.information.update` | Ogólne informacje o koncie zostały zaktualizowane. |
| Menedżerowie konta zaktualizowani | `account.managers.update` | Lista menedżerów konta uległa zmianie. |
| Zdjęcie profilowe zaktualizowane | `account.profile_picture.updated` | Zdjęcie profilowe konta zostało zmienione. |
| Agent zaktualizowany | `account.agent.updated` | Informacje o agencie na koncie zostały zmodyfikowane. |
| Nazwa URL zaktualizowana | `account.url-name.update` | Zmieniono spersonalizowany URL konta. |
| Unikalny identyfikator zaktualizowany | `account.unique-id.update` | Unikalny identyfikator konta został zmodyfikowany. |

| Zdarzenie | Klucz | Opis |

| Zadanie usunięte | `account.task.remove` | Zadanie na koncie zostało usunięte. |
| Zadanie ukończone | `account.task.complete` | Zadanie na koncie zostało ukończone. |

### Nabywca konta

## Zdarzenia umowy rezerwacji

Zdarzenia śledzące cykl życia umowy rezerwacji w TripPay.

| Zdarzenie | Klucz | Opis |
|---|---|---|
| Żądanie inicjalizacji | `booking.contract.initialize.request` | Zażądano inicjalizacji umowy rezerwacji. |
| Inicjalizacja zakończona | `booking.contract.initialize.complete` | Inicjalizacja umowy rezerwacji została zakończona. |
| Umowa utworzona | `booking.contract.created` | Utworzono nową umowę rezerwacji. |
| Przetwarzanie umowy | `booking.contract.processing` | Przetwarzana jest płatność za umowę rezerwacji. |
| Płatność umowy nie powiodła się | `booking.contract.failed` | Płatność za umowę rezerwacji nie powiodła się. |
| Umowa anulowana | `booking.contract.cancelled` | Umowa rezerwacji została anulowana. |
| Częściowy zwrot | `booking.contract.refund.partial` | Wydano częściowy zwrot na umowę. |
| Pełny zwrot | `booking.contract.refund.full` | Wydano pełny zwrot na umowę. |
| Żądanie zwrotu | `booking.contract.refund.request` | Złożono żądanie zwrotu. |
| Błąd zwrotu | `booking.contract.refund.error` | Wystąpił błąd podczas przetwarzania zwrotu. |
| Wypłata utworzona | `booking.contract.payout.create` | Utworzono wypłatę dla umowy. |
| Wypłata zakończona | `booking.contract.payout.complete` | Wypłata została pomyślnie zakończona. |

| Faktura subskrypcji zmieniona | `managing_entity.subscription.invoice.changed` | Faktura subskrypcji została zaktualizowana. |

| Zdarzenie | Klucz | Opis |
