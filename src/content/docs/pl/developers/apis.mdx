## Payment API

Punkty końcowe Payment są dla programistów, którzy chcą kupować zasoby podróżnicze. Można to zrobić za pomocą API jako zarejestrowany Agent Podróży lub korzystając z naszego API wraz z naszym zgodnym z PCI widgetem płatności dla wszystkich innych podmiotów.
Payment API udostępnia punkty końcowe dla afiliantów i hoteli do śledzenia rezerwacji, analiz i dostępności środków. Mogą oni również wybrać wypłatę dostępnych środków na swoje konto bankowe.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Dowiedz się więcej
</LinkButton>
