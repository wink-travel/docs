## API płatności

Punkty końcowe płatności są przeznaczone dla deweloperów, którzy chcą kupić zasoby turystyczne. Można to zrobić za pośrednictwem interfejsu API jako zarejestrowany agent podróży lub korzystając z naszego interfejsu API w połączeniu z naszym zgodnym z PCI widżetem płatności dla wszystkich innych podmiotów.
Interfejs API płatności udostępnia punkty końcowe dla podmiotów stowarzyszonych i hoteli w celu śledzenia rezerwacji, analiz i dostępności środków. Mogą oni również zdecydować się na wypłatę dostępnych środków na swoje konto bankowe.

  Dowiedz się więcej
