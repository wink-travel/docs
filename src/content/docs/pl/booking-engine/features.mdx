## Konfiguracja

W większości przypadków nie ma potrzeby udostępniania żadnej z tych aplikacji bezpośrednio. Stworzyliśmy wiele sposobów udostępniania inwentarza podróży, które możesz odkryć w[Studio Wink](/studio/what-is-studio)lub przez[Linki do mrugnięć](/link-manager/wink-links).

Jeśli jednak chcesz ręcznie połączyć się z jedną z tych witryn, pamiętaj o dodaniu identyfikatora klienta i identyfikatora personalizacji do adresu URL jako parametrów.

Możesz znaleźć swój identyfikator klienta:

<Steps>
  1. Zaloguj się do jednego z nich [Okienko Extranet](/extranet/what-is-extranet) Lub [Studio Wink](/studio/what-is-studio).
  2. Kliknij na ikonę swojego profilu w prawym górnym rogu i kliknij [Aplikacje](/developers/applications).
  3. Użyj aplikacji powiązanej z kontem Wink, z którym chcesz pracować.
  4. Skopiuj identyfikator klienta z tej aplikacji i wklej go w adresie URL jako parametr żądania. Przykład: ?client-id=MY\_CLIENT\_ID
</Steps>


### Ekstranet

Możesz znaleźć swój identyfikator personalizacji jako właściwość, korzystając z następujących opcji:

<Steps>
  1. Zaloguj się [Okienko Extranet](/extranet/what-is-extranet).
  2. Wybierz nieruchomość, z którą chcesz pracować.
  3. Na głównym pasku nawigacyjnym kliknij `Account > Branding`
  4. Skopiuj identyfikator dostosowywania i wklej go w adresie URL jako parametr żądania. Przykład: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Studio

Jako partner możesz znaleźć swój identyfikator personalizacji w następujący sposób:

<Steps>
  1. Zaloguj się [Studio Wink](/studio/what-is-studio).
  2. Wybierz konto, z którym chcesz pracować.
  3. Na głównym pasku nawigacyjnym kliknij `Tools > Customization`
  4. Skopiuj identyfikator dostosowania z dostosowania, którego chcesz użyć i wklej go w adresie URL jako parametr żądania. Przykład: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

W efekcie otrzymasz adres URL wyglądający mniej więcej tak:

https://ota.wink.travel?client-id=MOJE\_ID\_KLIENTA\&configuration-id=MOJE\_ID\_DOSTOSOWANIA

Jeśli chcesz przejść bezpośrednio na stronę hotelu, adres URL będzie wyglądał mniej więcej tak:

https://ota.wink.travel/hotel/hotel-x?client-id=MOJE\_ID\_KLIENTA\&configuration-id=MOJE\_ID\_DOSTOSOWANIA

:::uwaga
Jeśli nie podasz identyfikatora klienta i/lub konfiguracji, Wink OTA domyślnie użyje domyślnej konfiguracji.
:::

## IBE

Jeśli jesteś firmą zarządzającą nieruchomościami, chcesz połączyć się ze stroną główną Wink IBE. Strona główna pokaże Twój wybrany motyw wraz z[nieruchomości, które reprezentujesz](/studio/agency-representation).

W efekcie otrzymasz adres URL wyglądający mniej więcej tak:

https://book.wink.travel?client-id=MOJE\_ID\_KLIENTA\&configuration-id=MOJE\_ID\_DOSTOSOWANIA

Jeśli jesteś właścicielem nieruchomości, chcesz umieścić bezpośredni link do strony docelowej nieruchomości Wink IBE.

Ten adres URL wyglądałby mniej więcej tak:

https://book.wink.travel/hotel/hotel-x?client-id=MOJE\_ID\_KLIENTA\&configuration-id=MOJE\_ID\_DOSTOSOWANIA

:::uwaga
Jeśli nie uwzględnisz klienta i/lub identyfikatora konfiguracji, Wink IBE NIE zostanie załadowany. Te parametry są wymagane.
:::
