Edytor postów pozwala tworzyć nieograniczoną liczbę postów ręcznych i jest także miejscem, w którym zarządzasz postami tworzonymi automatycznie.

Edytor składa się z 4 sekcji:

1. Sekcja treści.
2. Sekcja mediów.
3. Sekcja linków.
4. Sekcja ustawień.

## Treść

Sekcja treści pozwala modyfikować tekst przed jego publikacją. Możesz tworzyć zlokalizowane treści w wielu językach — każdy język ma własny tytuł i opis.

## Media

Sekcja mediów pozwala modyfikować obrazy i filmy towarzyszące tekstowi.

## Link

Sekcja linków pozwala dodać odnośnik, do którego chcesz skierować użytkownika po przeczytaniu posta. *Np. może to być link do specjalnej oferty lub miejsce z dodatkowymi informacjami.* Możesz dodać URL linku, podpis linku oraz opis linku.

## Ustawienia

Sekcja ustawień pozwala modyfikować / nadpisywać wszystkie domyślne ustawienia dla danego posta. Możesz ustawić:

- **Platforma** — Wybierz, do której sieci społecznościowej zostanie opublikowany ten post.
- **Typ posta** — Wybierz między Transakcyjnym, Świadomościowym lub Sentymantalnym.
- **Ton głosu** — Nadpisz ton głosu dla tego konkretnego posta.
- **Personalizacja** — Użyj innej personalizacji dla waluty wyświetlania, języka, planu podróży itp.
- **Strefa czasowa** — Nadpisz strefę czasową publikacji.
- **Data zaplanowana** — Posty generowane automatycznie są planowane na optymalny czas. Możesz nadpisać datę i godzinę, jeśli wolisz inny termin.

## Cykl życia statusu posta

Każdy post przechodzi przez cykl życia statusu:

| Status | Opis |
| -- | -- |
| **Szkic** | Post został utworzony, ale nie jest jeszcze gotowy do przeglądu |
| **Oczekuje na zatwierdzenie** | Post czeka na zatwierdzenie przez człowieka przed zaplanowaniem |
| **Zatwierdzony** | Post został zatwierdzony i zostanie zaplanowany |
| **Odrzucony** | Post został przejrzany i odrzucony — nie zostanie opublikowany |
| **Zaplanowany** | Post jest w kolejce do publikacji w zaplanowanym terminie |
| **Opublikowany** | Post został pomyślnie opublikowany w sieci społecznościowej |
| **Wstrzymany** | Publikacja posta została tymczasowo wstrzymana |
| **Niepowodzenie** | Post nie mógł zostać opublikowany z powodu błędu |

## Zatwierdzanie postów

Posty zatwierdza się w siatce postów, gdzie możesz wybrać jeden lub więcej postów i zdecydować o ich zatwierdzeniu lub odrzuceniu.

:::note
Dla każdego języka i każdej sieci społecznościowej będzie osobny wpis.

Np. jeśli masz wybrane 3 sieci społecznościowe i 3 języki, zobaczysz 9 wpisów w kalendarzu.
:::
