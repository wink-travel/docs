Użytkownik Wink może zdecydować się na umożliwienie Ci zarządzania jednym ze swoich kont nieruchomości, afiliacji lub agenta turystycznego. W takim przypadku otrzymasz e-mail z prośbą o zaproszenie.

import { Steps } from '@astrojs/starlight/components';

Aby zobaczyć swoje aktualne zaproszenia Wink, wykonaj następujące kroki:

<Steps>
1. Musisz być zalogowany na jednej z naszych stron.
2. Kliknij ikonę swojego profilu w prawym górnym rogu, aby pojawiło się menu.
3. Kliknij link `Zaproszenia`.
4. Zarządzaj swoimi zaproszeniami, wybierając opcję `Akceptuj` lub `Odrzuć`.
</Steps>
