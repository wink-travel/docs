Użytkownik Wink może wybrać opcję zarządzania jednym z kont nieruchomości, kont partnerskich lub kont agentów podróży. Gdy to nastąpi, otrzymasz wiadomość e-mail z prośbą o zaproszenie.

import { Steps } from '@astrojs/starlight/components';

Aby zobaczyć swoje aktualne zaproszenia Wink, wykonaj następujące kroki:

<Steps>
  1. Musisz być zalogowany na jednej z naszych stron internetowych.
  2. Kliknij ikonę swojego profilu w prawym górnym rogu, a pojawi się menu.
  3. Kliknij `Invites` połączyć.
  4. Możesz wchodzić w interakcję ze swoimi zaproszeniami, `Accepting` Lub `Rejecting` ich.
</Steps>
