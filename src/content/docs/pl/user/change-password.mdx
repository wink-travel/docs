Jeśli chcesz zmienić hasło do swojego konta użytkownika Wink, wykonaj następujące kroki:

## Zaktualizuj hasło

<Steps>
1. Musisz być zalogowany na jednej z naszych stron.
2. Kliknij ikonę swojego profilu w prawym górnym rogu, aby pojawiło się menu.
3. Kliknij link `Zmień hasło`.
4. Postępuj zgodnie z instrukcjami, aby zmienić hasło.
5. Kliknij `Zapisz`.
</Steps>

Przy następnym logowaniu będziesz potrzebować nowego hasła.

## Zapomniałeś hasła

Czasami konieczne jest zresetowanie hasła użytkownika.

Oto kroki do zresetowania hasła użytkownika:

<Steps>
1. Kliknij link `Zapomniałeś hasła` na stronie logowania.
2. Wpisz nazwę użytkownika / e-mail konta, które chcesz zresetować.
3. Kliknij przycisk `Zapomniałeś hasła`. *Zostaniesz przekierowany do formularza resetowania hasła.*
4. Sprawdź swoją skrzynkę e-mail.
</Steps>

2. Potwierdź nowe hasło.
3. Kliknij przycisk `Resetuj hasło`. *Zostaniesz przekierowany z powrotem na stronę logowania.*
</Steps>
