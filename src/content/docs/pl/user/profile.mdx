Jeśli chcesz zaktualizować swój profil użytkownika dla konta Wink, wykonaj następujące kroki:

<Steps>
1. Musisz być zalogowany na jednej z naszych stron.
2. Kliknij ikonę swojego profilu w prawym górnym rogu, aby pojawiło się menu.
3. Kliknij link `Profil`.
4. Postępuj zgodnie z instrukcjami, aby zaktualizować swój `e-mail`, `imię` i `nazwisko`.
5. Możesz także przesłać nowe `zdjęcie profilowe`.
6. Kliknij `Zapisz`.
</Steps>

:::note
Twoje nowe zdjęcie profilowe pojawi się przy następnym całkowitym wylogowaniu się z Wink.
:::
