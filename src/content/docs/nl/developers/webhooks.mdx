Webhooks zijn een manier voor jou om realtime gebeurtenissen van Wink en TripPay met betrekking tot je account te ontvangen en erop te reageren.

## Voorbeeld

<Steps>
1. Er vindt een boeking plaats op Wink.
1. Het Wink-event `booking.created` wordt verzonden.
2. Je systeem luistert actief naar dit event.
3. Je systeem houdt zijn eigen set analytics bij en gebruikt dit event om de tijdreeksdatabase bij te werken.
</Steps>

:::note
Een Wink-webhook is anders dan een TripPay-webhook.

- Als je naar Wink-events wilt luisteren, kun je een webhook aanmaken via [Extranet](https://extranet.wink.travel) of [Studio](https://studio.wink.travel).
- Als je naar TripPay-events wilt luisteren, kun je een webhook aanmaken via [Payment](https://pay.wink.travel).
:::

*Voor dit voorbeeld authenticeren we via [Studio](https://studio.wink.travel).*

<Steps>
1. Zodra je bent ingelogd, klik je op je profielpictogram rechtsboven en verschijnt er een menu.
2. Klik op de link `Webhooks`.
3. Je wordt doorgestuurd naar je webhooks-pagina.
</Steps>

## Webhook aanmaken

:::caution
Je moet een affiliate-account bij Wink of een account bij TripPay hebben voordat je een webhook aanmaakt, omdat je de webhook aan een specifiek account koppelt.
:::

Hier zijn de stappen om een webhook aan te maken:

<Steps>
1. Klik op de webhooks-pagina op de knop `Create new webhook`.
2. ***Naam*** Geef je webhook een naam. *bijv. Wink Stalker*
3. ***Inschakelen*** Laat dit aanstaan om de webhook te activeren.
4. **Entiteitsnaam** Koppel je webhook aan een van je bestaande accounts. *bijv. Cool Account*
5. **Events** Luister naar de events die je interesseren. *bijv. `booking.created`*
6. **Webhook URL** Voer de URL in waar je onze events wilt ontvangen. *bijv. `https://my.system.com/webhook/wink`*
7. Klik op de knop `Save` om door te gaan.
</Steps>

Je wordt teruggeleid naar je lijst met webhooks en je nieuwe webhook is klaar voor gebruik.

## Webhook bijwerken

Je moet je app bijwerken als je wilt:

- Je webhook uitschakelen.
- Naar meer/minder events luisteren.
- De URL van je luister-webhook wijzigen.

Om een bestaande webhook bij te werken, doe je het volgende:

<Steps>
1. Klik op de link `Actions` bij de webhook die je wilt bijwerken.
2. Klik op de knop `Update` onder Actions.
3. Breng je wijzigingen aan in de webhook.
4. Klik op de knop `Save` om door te gaan.
</Steps>

Je wordt teruggeleid naar je lijst met webhooks.

## Webhook verwijderen

Als je de webhook niet meer nodig hebt, kun je deze verwijderen.

Om een webhook te verwijderen, doe je het volgende:

<Steps>
1. Klik op de link `Actions` bij de webhook die je wilt verwijderen.
2. Klik op de knop `Remove` onder Actions.
3. Klik op de knop `OK` om de verwijdering te bevestigen.
</Steps>

Je wordt teruggeleid naar je lijst met webhooks.
