Versnel de time-to-market door een van onze officieel ondersteunde taal Software Development Kits te gebruiken.
Als we jouw favoriete taal momenteel niet ondersteunen, ga dan naar de API die je wilt gebruiken en download de OpenAPI-specificatie die we voor elke API bovenaan de pagina beschikbaar stellen. Je kunt je eigen SDK genereren door een van onze openbare SDK-repositories te forken of door een van de OpenAPI-generators te gebruiken.

## Java

Deze bibliotheek is gecompileerd voor Java 24. Het is geconfigureerd om de reactieve WebClient te gebruiken en is geoptimaliseerd voor gebruik met het Spring Framework. Spring is echter geen vereiste, maar het biedt wel OAuth2-ondersteuning out of the box.
