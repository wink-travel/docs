Versnel de time-to-market door een van onze officieel ondersteunde Software Development Kits te gebruiken.
Als we uw favoriete taal momenteel niet ondersteunen, ga dan naar de API die u wilt gebruiken en download de OpenAPI-specificatie die we voor elke API bovenaan de pagina beschikbaar stellen. U kunt uw eigen SDK genereren door een van onze openbare SDK-repositories te forken of door een van de OpenAPI-generators te gebruiken.

## Java

Deze bibliotheek is gecompileerd voor Java 24. Hij is geconfigureerd voor de reactieve WebClient en geoptimaliseerd voor gebruik met het Spring Framework. Spring is echter geen vereiste, maar biedt wel standaard OAuth2-ondersteuning.
