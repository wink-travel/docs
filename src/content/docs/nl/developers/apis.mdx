## Payment API

Payment endpoints zijn voor ontwikkelaars die reisvoorraad willen aankopen. Dit kan via de API als geregistreerde Travel Agent of met onze API in combinatie met onze PCI-compliant betaalwidget voor alle andere entiteiten.
De Payment API biedt endpoints voor affiliates en hotels om boekingen, analyses en beschikbaarheid van fondsen te volgen. Ze kunnen er ook voor kiezen om beschikbare fondsen naar hun bankrekening over te maken.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Meer informatie
</LinkButton>
