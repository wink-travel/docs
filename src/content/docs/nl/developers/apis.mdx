## Taxonomie API

## Analytics-API

De Analytics API biedt eindpunten voor iedereen die inzicht wil in ranglijsten op het scorebord en diepgaand inzicht in tijdreeksen en gegevens op platformniveau. De API host een breed scala aan statistieken.

<LinkButton href="https://docs.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
  Meer informatie
</LinkButton>


## Betalings-API

Betalingseindpunten zijn bedoeld voor ontwikkelaars die reisinventaris willen kopen. Dit kan via de API als geregistreerd reisagent of via onze API in combinatie met onze PCI-conforme betalingswidget voor alle andere entiteiten.
De betalings-API stelt eindpunten beschikbaar voor affiliates en hotels om boekingen, analyses en de beschikbaarheid van fondsen bij te houden. Ze kunnen er ook voor kiezen om het beschikbare geld naar hun bankrekening over te maken.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Meer informatie
</LinkButton>
