Deze pagina toont elk webhook event waarop je je kunt abonneren op Wink. Voor installatie-instructies, zie de [Webhook Integratie Gids](/nl/guides/webhook-integration) of de [Webhooks beheerpagina](/nl/my-account/webhooks).

## Hoe te abonneren

1. Navigeer naar **Applications > Webhooks** in je portal.
2. Maak een webhook aan en selecteer de events die je wilt ontvangen.
3. Wink stuurt een HTTP POST naar je endpoint telkens wanneer het event plaatsvindt.

Voor volledige installatie-stappen en best practices, zie [Webhook Integratie](/nl/guides/webhook-integration).

## Account events

Events gerelateerd aan de levenscyclus en instellingen van je TripPay-account.

| Event | Key | Beschrijving |
|---|---|---|
| Account aangemaakt | `account.create` | Er is een nieuw account aangemaakt. |
| Account in beoordeling | `account.under_review` | Account is in afwachting van beoordeling. |
| Account bijgewerkt | `account.updated` | Accountgegevens zijn gewijzigd. |
| Accountstatus bijgewerkt | `account.status.updated` | Accountstatus is gewijzigd (bijv. goedgekeurd, geschorst). |
| Accountadres bijgewerkt | `account.address.updated` | Accountadres is aangepast. |
| Accountdetails bijgewerkt | `account.details.updated` | Accountdetailvelden zijn gewijzigd. |
| Accountinformatie bijgewerkt | `account.information.update` | Algemene accountinformatie is bijgewerkt. |
| Accountmanagers bijgewerkt | `account.managers.update` | Lijst met accountmanagers is gewijzigd. |
| Profielfoto bijgewerkt | `account.profile_picture.updated` | Profielfoto van het account is gewijzigd. |
| Agent bijgewerkt | `account.agent.updated` | Agentinformatie op het account is aangepast. |
| URL-naam bijgewerkt | `account.url-name.update` | Vanity URL van het account is gewijzigd. |
| Unieke ID bijgewerkt | `account.unique-id.update` | Unieke identificatie van het account is aangepast. |

### Account taken

| Event | Key | Beschrijving |
|---|---|---|
| Taak aangemaakt | `account.task.create` | Er is een nieuwe taak toegevoegd aan het account. |
| Taak bijgewerkt | `account.task.update` | Een accounttaak is aangepast. |
| Taak verwijderd | `account.task.remove` | Een accounttaak is verwijderd. |
| Taak voltooid | `account.task.complete` | Een accounttaak is afgerond. |
| Taak belasting-ID voltooid | `account.task.tax_id.complete` | Verificatie van belasting-ID taak is afgerond. |

### Account acquirer

| Event | Key | Beschrijving |
|---|---|---|
| Acquirer aangemaakt | `account.acquirer.created` | Er is een nieuwe acquirer toegevoegd aan het account. |
| Acquirer bijgewerkt | `account.acquirer.updated` | Een account acquirer is aangepast. |
| Acquirer verwijderd | `account.acquirer.removed` | Een acquirer is verwijderd van het account. |

## Booking contract events

Events die de levenscyclus van een boekingscontract via TripPay volgen.

| Event | Key | Beschrijving |
|---|---|---|
| Initialisatie aangevraagd | `booking.contract.initialize.request` | Initialisatie van een boekingscontract is aangevraagd. |

| Contract aangemaakt | `booking.contract.created` | Er is een nieuw boekingscontract aangemaakt. |

| Contract mislukt | `booking.contract.failed` | Betaling van het boekingscontract is mislukt. |
| Contract geannuleerd | `booking.contract.cancelled` | Boekingscontract is geannuleerd. |
| Gedeeltelijke terugbetaling | `booking.contract.refund.partial` | Er is een gedeeltelijke terugbetaling gedaan op het contract. |
| Volledige terugbetaling | `booking.contract.refund.full` | Er is een volledige terugbetaling gedaan op het contract. |
| Terugbetaling aangevraagd | `booking.contract.refund.request` | Er is een terugbetalingsverzoek ingediend. |
| Terugbetalingsfout | `booking.contract.refund.error` | Er is een fout opgetreden bij het verwerken van een terugbetaling. |
