AI-vaardigheden zijn draagbare instructiepakketten die codeeragenten leren hoe ze met Wink moeten werken. In plaats van te vertrouwen op trainingsgegevens of verouderde documentatie, stuurt elke vaardigheid de agent aan om de [Wink MCP-server](/nl/developers/mcp) te gebruiken als de live bron van waarheid voor API-contracten en platformdocumentatie.

Vaardigheden werken met elke agentclient die ze ondersteunt — Claude Code, Codex, Cursor en anderen.

## Beschikbare vaardigheden

### wink-api-builder

Implementeer, beoordeel of debug Wink API-integraties. De vaardigheid begeleidt de agent door een gestructureerde workflow:

1. Zoek de juiste API-operatie via `api_search`
2. Open de live request/response-contracten van de MCP-server
3. Haal de authenticatie-instellingen uit de documentatie
4. Schrijf code tegen het geverifieerde contract
5. Markeer eventuele afwijkingen tussen bestaande code en de huidige API

Nuttig voor:
- Het bouwen van nieuwe clientmethoden voor elke Wink API
- Debuggen van contractafwijkingen (bijv. een request dat HTTP 400 retourneert)
- Beoordelen van bestaande integraties aan de hand van de live specificatie

## Installatie

Kopieer de vaardigheidsmap naar de vaardigheidsdirectory van je agent.

:::tip
De Wink MCP-server moet in dezelfde omgeving verbonden zijn. Zie [MCP Server](/nl/developers/mcp) voor installatie-instructies.
:::

**Claude Code**

```bash
# repo-lokaal
cp -r skills/wink-api-builder .claude/skills/

# of gebruikersniveau
cp -r skills/wink-api-builder ~/.claude/skills/
```

Aanroepen met:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repo-lokaal
cp -r skills/wink-api-builder .agents/skills/

# of gebruikersniveau
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Aanroepen met:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Andere agents**

Plaats de vaardigheidsmap waar je agent vaardigheden laadt. Het startpunt is `SKILL.md`.
