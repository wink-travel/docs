De boekingsengine is er in twee varianten om te voldoen aan de verschillende behoeften van een affiliate, een property management bedrijf en een hotel.

## Online reisbureau

Voor een affiliate is het het meest logisch om de online reisbureau versie te gebruiken. De reden is dat de site is opgezet om reizigers die meerdere accommodaties willen bekijken te bedienen.

Je vindt de startpagina van deze applicatie hieronder.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Internet boekingsengine

Voor een hotel of een property management bedrijf is het het meest logisch om de internet boekingsengine versie te gebruiken. Dit type gebruiker is geïnteresseerd in het uitlichten van één accommodatie of meerdere accommodaties onder beheer.

Je vindt de startpagina van deze applicatie hieronder.

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE

## Configuratie

Meestal is het niet nodig om een van deze applicaties direct te delen. We hebben verschillende manieren gecreëerd om reisaanbod te delen, die je kunt ontdekken in [Wink Studio](/studio/what-is-studio) of via [WinkLinks](/link-manager/wink-links).

Als je echter handmatig naar een van deze websites wilt linken, zorg er dan voor dat je je client ID en aanpassings-ID als parameters aan de URL toevoegt.

Je vindt je client ID door:

<Steps>
1. In te loggen op [Wink Extranet](/extranet/what-is-extranet) of [Wink Studio](/studio/what-is-studio).
2. Klik op je profielicoon rechtsboven en klik op [Applications](/developers/applications).
3. Gebruik de applicatie die gekoppeld is aan het Wink-account waarmee je wilt werken.
4. Kopieer de Client ID van die applicatie en plak deze als request-parameter in de URL. Voorbeeld: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Je vindt je aanpassings-ID als accommodatie door:

<Steps>
1. In te loggen op [Wink Extranet](/extranet/what-is-extranet).
2. Selecteer de accommodatie waarmee je wilt werken.
3. Klik in de hoofd-navigatiebalk op `Account > Branding`
4. Kopieer de Customization ID en plak deze als request-parameter in de URL. Voorbeeld: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Je vindt je aanpassings-ID als affiliate door:

<Steps>
1. In te loggen op [Wink Studio](/studio/what-is-studio).
2. Selecteer het account waarmee je wilt werken.
3. Klik in de hoofd-navigatiebalk op `Tools > Customization`
4. Kopieer de Customization ID van de aanpassing die je wilt gebruiken en plak deze als request-parameter in de URL. Voorbeeld: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Wat je uiteindelijk krijgt is een URL die er ongeveer zo uitziet:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Als je direct naar een hotelpagina wilt gaan, zou de URL er ongeveer zo uitzien:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Als je geen client- en/of configuratie-ID opgeeft, zal Wink OTA standaard een standaardconfiguratie laden.
:::

## IBE

Als je een property management bedrijf bent, wil je linken naar de Wink IBE startpagina. De startpagina toont je aangewezen thema samen met de [accommodaties die je vertegenwoordigt](/studio/agency-representation).

Wat je uiteindelijk krijgt is een URL die er ongeveer zo uitziet:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Als je een accommodatie bent, wil je direct linken naar de Wink IBE accommodatie landingspagina.

Die URL zou er ongeveer zo uitzien:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Als je geen client- en/of configuratie-ID opgeeft, zal Wink IBE NIET laden. Deze parameters zijn verplicht.
:::

## Voorbeeld: Accommodatiepagina

Elke accommodatie op Wink krijgt een eigen unieke landingspagina in beide websites. Alle externe links leiden reizigers naar een van deze landingspagina’s.
