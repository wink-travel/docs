De **Wink Drupal Module** voegt Wink reisvoorraadcomponenten toe aan je Drupal-site als native blokken. Editors kunnen ze overal in de lay-out plaatsen via het standaard blokken-systeem.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Broncode, ontwikkelaarsinstallatiehandleiding en instructies voor bijdragen."
/>

## Vereisten

- Drupal 10 of 11
- PHP 8.1+
- Composer

## Installeren

<Steps>
1. Installeer via Composer:
   ```bash
   composer require drupal/wink
   ```
2. Activeer de module:
   ```bash
   drush en wink -y
   ```
3. Ga naar **Beheer → Configuratie → Webservices → Wink-instellingen** (`/admin/config/wink`).
4. Voer je **Client ID** en **Client Secret** in, selecteer je omgeving en klik op **Opslaan**.
   Ga naar [Applications](/nl/portal/settings/applications) om je inloggegevens op te halen.
</Steps>

## Voeg een blok toe aan een pagina

<Steps>
1. Ga naar **Beheer → Structuur → Blokindeling**.
2. Klik op **Blok plaatsen** in het gebied waar je Wink-inhoud wilt tonen.
3. Zoek op **Wink** om de beschikbare blokken te vinden:
   - **Wink Content** — toont een hotelkaart, raster of kaart
   - **Wink Lookup** — zoekbalk voor bestemmingen
   - **Wink Search** — knop voor het selecteren van een reisroute
   - **Wink Account** — inlogknop
   - **Wink Itinerary** — knop voor de status van de reisroute
   - **Wink Cart** — winkelwagenknop
4. Stel voor **Wink Content** het **Layout Type** en **Layout ID** in de blokinstellingen in.
5. Sla de blokconfiguratie op.
</Steps>

## Beschikbare blokken

| Blok | Doel |
|---|---|
| **Wink Content** | Toont hotelkaarten, gerangschikte rasters, kaarten en meer |
| **Wink Lookup** | Zoekbalk voor bestemmingen en hotels |
| **Wink Search** | Icoonknop die de reisroutekiezer opent |
| **Wink Account** | Inlog- / gebruikersaccountknop |
| **Wink Itinerary** | Reisrouteknop die de huidige reis toont |
| **Wink Cart** | Samenvattingsknop van de winkelwagen |
