De **Wink TYPO3 Extension** integreert Wink reisvoorraad in TYPO3 13 als native contentelementen. Editors voegen Wink-componenten toe aan pagina’s met de standaard TYPO3-pagina-editor — geen aanpassingen aan templates nodig.

## Vereisten

- TYPO3 13.x
- PHP 8.2+
- Composer

## Installeren

<Steps>
1. Installeer via Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Activeer de extensie:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Ga in de TYPO3-backend naar **Admin Tools → Settings → Extension Configuration → wink**.
4. Voer je **Client ID**, **Client Secret** in en selecteer je omgeving. Sla op.
   Ga naar [Applications](/nl/portal/settings/applications) om je gegevens op te halen.
5. Voeg de statische TypoScript-template toe aan je root-template:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Voeg een contentelement toe aan een pagina

<Steps>
1. Open een pagina in de **Page**-module.
2. Klik op de **+**-knop in het contentgebied om een nieuw element toe te voegen.
3. Kies een van de **Wink Travel**-contentelementen:
   - **Wink Content** — hotelkaarten, gerangschikte grids, kaarten
   - **Wink Lookup** — bestemmingszoekbalk
   - **Wink Search** — routekiezerknop
   - **Wink Account** — inlogknop
   - **Wink Itinerary** — route-statusknop
   - **Wink Cart** — winkelwagenknop
4. Voor **Wink Content** gebruik je de FlexForm-velden om een **Layout Type** te selecteren en de **Layout ID** in te voeren.
5. Sla de pagina op en publiceer.
</Steps>

## Beschikbare contentelementen

| Element | Doel |
|---|---|
| **Wink Content** | Hotelkaarten, gerangschikte grids, kaarten en meer |
| **Wink Lookup** | Bestemmingszoekbalk |
| **Wink Search** | Icoonknop die de routekiezer opent |
| **Wink Account** | Inlog- / gebruikersaccountknop |
| **Wink Itinerary** | Routeknop die de huidige reis toont |
| **Wink Cart** | Samenvattingsknop van de winkelwagen |
