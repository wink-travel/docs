Het Travel Agent-dashboard biedt een snel overzicht van je prestaties en activiteiten.

## Metrics

Het dashboard toont belangrijke statistieken in één oogopslag:

- **Boekingen** — Totaal aantal boekingen dat je hebt gefaciliteerd.
- **Omzet** — Totale omzet gegenereerd via jouw boekingen.
- **Aankomende boekingen** — Boekingen met toekomstige incheckdata.

## Activity

De activiteitensectie volgt je recente boekingsgeschiedenis en trends in de tijd, zodat je piekperiodes en best presterende accommodaties kunt identificeren.

## Account status

Het dashboard toont de huidige status van je reisagentprofiel — of de setup voltooid is en commissies actief zijn.
