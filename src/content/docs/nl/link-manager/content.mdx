## Inhoud toevoegen

Zo voeg je inhoud toe aan WinkLinks. Ga onder het tabblad `Invoeren` eerst naar de knop `Invoer toevoegen`.

### Normale link

:::tip
De link moet naar een webpagina gaan. Je kunt bijvoorbeeld niet direct linken naar een PDF. Als je een PDF wilt delen, moet je de functie `Bestand uploaden` gebruiken.
:::

Hier zijn de stappen om een normale link toe te voegen:

<Steps>
1. Voer de link in die je wilt delen. *bijv. https://trvl.as/3xWCH*
2. Zodra je de link invoert, zoekt Wink de details van deze website op en vult het formulier dat je daarna ziet in.
3. Je kunt ervoor kiezen om sommige gegevens aan te passen, zoals de sitenaam, titel en beschrijving.
4. Je kunt ook een aangepaste afbeelding uploaden.
5. Klik op de knop `Opslaan` wanneer je klaar bent om door te gaan.
</Steps>

### Bestand uploaden

:::caution[Vereisten]
- Afbeeldingen mogen niet groter zijn dan 10Mb.
- Video's mogen niet groter zijn dan 50Mb.
- We ondersteunen alle erkende beeld- en videoformaten, inclusief PDF.
:::

:::note[Aanbevelingen]
- 2560w x 1440h (16x9 beeldverhouding) voor je uitgelichte afbeelding.
- 2560w x 1920h (4:3 beeldverhouding) voor alle andere afbeeldingen.
- De afbeeldingsgrootte mag niet groter zijn dan 5Mb. Comprimeer je afbeeldingen voordat je ze uploadt naar Wink.
:::

Hier zijn de stappen om een bestand toe te voegen:

<Steps>
1. Media kan worden toegevoegd:
    - Door afbeeldingen van je computer naar het venster te slepen of klik op `Bladeren`.
    - Door een externe URL te plakken die naar een afbeelding verwijst.
    - Door je laptopcamera te gebruiken.
    - Vanuit je Google Drive-account.
    - Vanuit je Dropbox-account.
    - Vanuit Shutterstock
    - Vanuit gettyimages
    - Vanuit iStock
    - Vanuit Unsplash

3. Klik op de knop `Opslaan` wanneer je klaar bent om door te gaan.
</Steps>

## Inhoud bijwerken

Alle inhoud kan eenvoudig worden aangepast door te klikken op de link `Acties` en `Bijwerken` te selecteren. Je kunt inhoud op een vergelijkbare manier verwijderen door op de knop `Verwijderen` te klikken.

## Inhoud verwijderen

Alle inhoud kan eenvoudig worden aangepast door `Bijwerken` te selecteren. Je kunt inhoud op een vergelijkbare manier verwijderen door eerst op de link `Acties` te klikken en vervolgens op de knop `Verwijderen`.

## Inhoud sorteren

Je kunt je inhoud ordenen door te klikken op het pictogram linksonder in elke invoer en het naar de gewenste positie te slepen.

## Inhoud groeperen

:::tip
Maak je tags aan voordat je je invoeren maakt. Anders verlies je je voortgang en moet je opnieuw beginnen.
:::

Wanneer je een invoer bewerkt, heb je de optie om een of meer tags toe te voegen voor groeperingsdoeleinden. Scroll naar beneden in het formulier om het formulierveld `Tags` te vinden.

Gebruikers die je WinkLinks-pagina bekijken, kunnen kiezen hoe ze je inhoud willen consumeren (lijst of raster) en kunnen een of meer tags selecteren om je inhoud verder te filteren.

## API

Ontwikkelaars die `WinkLinks` willen beheren, kunnen terecht bij [Developers > API > WinkLinks](/nl/developers/apis/#winklinks-api).
