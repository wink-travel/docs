Om je WinkLinks-tags te beheren, navigeer je naar `WinkLinks` in de hoofd navigatiebalk en klik je op het tabblad `Tags`.

Naarmate je content groeit, wil je deze gaan organiseren. Hiervoor hebben we `tags` gemaakt. Je kunt content groeperen met zoveel tags als je wilt.

## Tag aanmaken

<Steps>
1. Maak een nieuwe tag aan door op de knop `Add a new tag` te klikken.
2. Voer de naam van je nieuwe tag in. *bijv. Top koffiehuizen in Berlijn*
3. Klik op de link `Save`.
</Steps>

## Tag bijwerken

<Steps>
1. Begin met het klikken op de link `Edit` naast de tag die je wilt aanpassen.
2. Werk de naam van je tag bij. *bijv. Beste koffiehuizen in Berlijn*
3. Klik op de link `Save`.
</Steps>

## Tag verwijderen

:::note
Wanneer je een tag verwijdert, wordt deze ook verwijderd uit alle gekoppelde content.
:::

<Steps>
1. Klik op de link `Remove` naast de tag die je wilt verwijderen.
</Steps>

## API

Ontwikkelaars die `WinkLinks` willen beheren, kunnen terecht bij [Developers > API > WinkLinks](/nl/developers/apis/#winklinks-api).
