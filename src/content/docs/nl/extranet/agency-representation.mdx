Een agentschap is een normaal Wink-account dat je aan je account kunt toewijzen voor vertegenwoordiging. Hieronder staan enkele redenen waarom je een agentschap zou willen dat jou vertegenwoordigt:

- Volledige / gedeeltelijke beheer van je Wink-account door een derde partij.
- Eigendom heeft een bestaande overeenkomst met hun managementbedrijf en wil dat Wink het financiÃ«le aspect van hun regeling van binnenkomende gelden via Wink afhandelt.
- Property management agentschap doet al het online werk namens zijn eigendommen.

## Agentschap toewijzen

Zo wijs je een agentschap aan je account toe.

<Steps>
 1. Log in op je gebruikersaccount op [Wink Extranet](https://extranet.wink.travel).
 2. Selecteer de eigendom waarmee je wilt werken.
 3. Klik in de hoofd navigatiebalk op `Account > Managers`.
 4. Typ in de sectie `Agentschap` de naam van het agentschap dat jou wil vertegenwoordigen.
 5. Kies de agentschapsvergoeding uit de dropdown.
 5. Voeg optioneel vervalregels toe.
 7. Klik op de knop `Agentschap instellen` om door te gaan.
 </Steps>

 :::note
Je moet de eigenaar van de eigendom zijn om een agentschap toe te wijzen.
 :::

 :::tip[Voorbeeld]
De agentschapsvergoeding is afgeleid van de boeking.

Boeking = $100, Agentschapsvergoeding = 1% => 100 * 0.01 = $1
 :::

## Agentschap verwijderen

Zo verwijder je een agentschap van je account.

<Steps>
 1. Log in op je gebruikersaccount op [Wink Extranet](https://extranet.wink.travel).
 2. Selecteer de eigendom waarmee je wilt werken.
 3. Klik in de hoofd navigatiebalk op `Account > Managers`.
 4. Klik in de sectie `Agentschap` op het <Badge text="x" variant="danger" size="small" />-icoon aan de rechterkant van de agentschap-dropdown.
 5. Klik op de knop `Agentschap instellen` om door te gaan.
 </Steps>


## API

Ontwikkelaars die het agentschap via API willen beheren, kunnen terecht bij [Ontwikkelaars > API > Affiliate](/developers/apis/#affiliate-api).
