Een agency is een normaal Wink-account dat je aan je account kunt toewijzen voor vertegenwoordiging. Hieronder staan enkele redenen waarom je een agency zou willen die jou vertegenwoordigt:

- Volledig / gedeeltelijk beheer van je Wink-account door een derde partij.
- De accommodatie heeft een bestaande overeenkomst met hun managementbedrijf en wil dat Wink het financiÃ«le aspect van hun regeling van binnenkomende gelden via Wink afhandelt.
- Property management agency doet al het online werk namens haar accommodaties.

## Agency toewijzen

Zo wijs je een agency aan je account toe.

<Steps>
 1. Log in op je gebruikersaccount op [Wink Extranet](https://extranet.wink.travel).
 2. Selecteer de accommodatie waarmee je wilt werken.
 3. Klik in de hoofd navigatiebalk op `Account > Managers`.
 4. Typ in het gedeelte `Agency` de naam van de agency die jou moet vertegenwoordigen.
 5. Kies de agency fee uit de dropdown.
 5. Voeg optioneel vervalregels toe.
 7. Klik op de knop `Set agency` om door te gaan.
 </Steps>

 :::note
Je moet eigenaar zijn van de accommodatie om een agency toe te wijzen.
 :::

 :::tip[Voorbeeld]
De agency fee is afgeleid van de boeking.

Boeking = $100, Agency fee = 1% => 100 * 0.01 = $1
 :::

## Agency verwijderen

Zo verwijder je een agency van je account.

<Steps>
 1. Log in op je gebruikersaccount op [Wink Extranet](https://extranet.wink.travel).
 2. Selecteer de accommodatie waarmee je wilt werken.
 3. Klik in de hoofd navigatiebalk op `Account > Managers`.
 4. Klik in het gedeelte `Agency` op het <Badge text="x" variant="danger" size="small" /> icoon aan de rechterkant van de agency dropdown.
 5. Klik op de knop `Set agency` om door te gaan.
 </Steps>


## API

Ontwikkelaars die agency via API willen beheren, kunnen terecht bij [Developers > API > Affiliate](/nl/developers/apis/#affiliate-api).
