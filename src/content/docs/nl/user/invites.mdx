Een Wink-gebruiker kan ervoor kiezen om jou het beheer te laten doen van een van hun accommodatie-, partner- of reisagentaccounts. Wanneer dat gebeurt, ontvang je een e-mail met een uitnodigingsverzoek.

import { Steps } from '@astrojs/starlight/components';

Om je huidige Wink-uitnodigingen te bekijken, volg je deze stappen:

<Steps>
1. Je moet ingelogd zijn op een van onze websites.
2. Klik op je profielpictogram, rechtsboven in het scherm, en er verschijnt een menu.
3. Klik op de link `Uitnodigingen`.
4. Reageer op je uitnodigingen door ze te `Accepteren` of te `Weigeren`.
</Steps>
