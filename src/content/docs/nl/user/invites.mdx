Een Wink-gebruiker kan ervoor kiezen om een van zijn/haar vastgoed-, affiliate- of reisagentaccounts door jou te laten beheren. Je ontvangt dan een e-mail met een uitnodigingsverzoek.

import { Steps } from '@astrojs/starlight/components';

Om uw huidige Wink-uitnodigingen te bekijken, volgt u deze stappen:

<Steps>
  1. U moet ingelogd zijn op een van onze websites.
  2. Klik op uw profielpictogram in de rechterbovenhoek. Er verschijnt dan een menu.
  3. Klik op de `Invites` link.
  4. Interactie met uw uitnodigingen door: `Accepting` of `Rejecting` hen.
</Steps>
