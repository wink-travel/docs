Als u uw gebruikersprofiel voor uw Wink-gebruikersaccount wilt bijwerken, volgt u deze stappen:

<Steps>
  1. U moet ingelogd zijn op een van onze websites.
  2. Klik op uw profielpictogram in de rechterbovenhoek. Er verschijnt dan een menu.
  3. Klik op de `Profile` link.
  4. Volg de instructies om uw `e-mail`, `first` En `last name`.
  5. Je kunt ook een nieuwe uploaden `profile picture` van jezelf.
  6. Klik `Save`.
</Steps>


:::note
Je nieuwe profielfoto wordt weergegeven wanneer je je de volgende keer volledig afmeldt bij Wink.
:::
