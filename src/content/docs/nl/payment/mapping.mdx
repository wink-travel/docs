Bent u een integrator, zoals Wink, en wilt u uw externe begunstigden toewijzen voor gebruik met TripPay, dan kunt u dit op twee manieren doen:

* Handmatige mapping
* Met behulp van de[Betalings-API](/developers/apis/#payment-api)(*Zien `Account Mappings`*)

:::tip
Tenzij u slechts met een handvol dezelfde eigenschappen werkt, raden we u aan de API te gebruiken om deze stap te automatiseren.
:::

Hieronder leggen we u stap voor stap uit hoe u handmatig een externe begunstigde kunt toewijzen.*Het maakt gebruik van de API onder de motorkap.*

<Steps>
  1. Inloggen [TripPay](https://pay.wink.travel).
  2. Selecteer uw integratoraccount.
  3. Klik `Settings > Mappings` vanuit de hoofdnavigatiebalk.
  4. Klik op de `Create mapping` knop. U ziet het toewijzingsformulier.
  5. **Lokaal account** Begin met het typen van de naam van de bestaande TripPay die u wilt toewijzen.
  6. **Externe identificatie** Plak uw externe ID voor deze entiteit zoals deze in het externe systeem bestaat.
  7. Klik `Save` om verder te gaan. U wordt teruggeleid naar uw lijst met toewijzingen.
</Steps>


Nu de toewijzingen zijn gemaakt, weet TripPay welke rekeningen moeten worden gecrediteerd als begunstigden van een boeking wanneer u ernaar verwijst met behulp van uw externe identificatiegegevens.

## API

Ontwikkelaars die willen beheren`Payments`, ga naar[Ontwikkelaars > API > Betalingen](/developers/apis/#payment-api).
