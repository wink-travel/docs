Als je een integrator bent, zoals Wink, en je wilt je externe begunstigden koppelen voor gebruik met TripPay, dan kan dat op twee manieren:

- Handmatige mapping
- Gebruik van de [Payment API](/nl/developers/apis/#payment-api) (*Zie `Account Mappings`*)

:::tip
Tenzij je slechts met een handvol van dezelfde accommodaties werkt, raden we aan om de API te gebruiken om deze stap te automatiseren.
:::

Hieronder leiden we je door de handmatige manier van het koppelen van een externe begunstigde. *Het maakt onder de motorkap gebruik van de API.*

<Steps>
1. Log in op [TripPay](https://pay.wink.travel).
2. Selecteer je integratoraccount.
3. Klik in de hoofd navigatiebalk op `Settings > Mappings`.
4. Klik op de knop `Create mapping`. Je ziet het mappingformulier.
5. **Local account** Begin met het typen van de naam van het bestaande TripPay-account dat je wilt koppelen.
6. **External identifier** Plak je externe ID voor deze entiteit zoals die bestaat in het externe systeem.
7. Klik op `Save` om door te gaan. Je wordt teruggeleid naar je lijst met mappings.
</Steps>

Met je mappings op hun plaats weet TripPay nu welke accounts gecrediteerd moeten worden als begunstigden van een boeking wanneer je naar ze verwijst met je externe identificaties.

## API

Ontwikkelaars die `Payments` willen beheren, gaan naar [Developers > API > Payments](/nl/developers/apis/#payment-api).
