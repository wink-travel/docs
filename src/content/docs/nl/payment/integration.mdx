Bent u een reisorganisatie en wilt u de betalingen en uitbetalingen uitbesteden aan TripPay? Dan zijn hier zes basisstappen om aan de slag te gaan:

<Steps>
  1. Maak een [gebruikersaccount](/user/register) op Wink.
  2. Maak een account aan met [TripPay](https://pay.wink.travel).
  3. Maak een [Sollicitatie](/developers/applications) op TripPay waarmee u met onze API kunt communiceren.
  4. Maak en breng uw begunstigden in kaart met TripPay. *Bijvoorbeeld: Voor de meesten betekent dit dat ze accounts aanmaken voor de accommodaties op hun platform die ze door TripPay willen laten betalen.*
  5. Integreer de [betalingswebcomponent](/developers/web-components/#payment) op uw site.
  6. Stuur TripPay naar de reiziger `shopping cart` en start het betalingsproces.
</Steps>


:::tip
Lees de handleiding voor een complete handleiding voor het gebruik van TripPay voor uw reiswebsite[Integratie met TripPay](/guides/integrating-with-trip-pay).
::

Ontwikkelaars die willen beheren`Payments`, ga naar[Ontwikkelaars > API > Betalingen](/developers/apis/#payment-api).
