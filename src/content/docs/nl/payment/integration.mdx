Als je een reisbedrijf bent en je wilt het betalings- en uitbetalingsgedeelte van je bedrijf uitbesteden aan TripPay, dan zijn hier 6 basisstappen om te beginnen:

<Steps>
1. Maak een [gebruikersaccount](/nl/user/register) aan op Wink.
2. Maak een account aan bij [TripPay](https://pay.wink.travel).
3. Maak een [Applicatie](/nl/developers/applications) aan op TripPay waarmee je met onze API kunt communiceren.
4. Maak en koppel je begunstigden met TripPay. *bijv. Voor de meesten betekent dit het aanmaken van accounts voor de accommodaties op je platform die je door TripPay wilt laten betalen.*
5. Integreer de [payment web component](/nl/developers/web-components/#payment) in je site.
6. Stuur TripPay de `winkelwagen` van de reiziger en start het betalingsproces.
</Steps>

:::tip
Voor een complete handleiding over het gebruik van TripPay voor je reissite, lees de gids [Integreren met TripPay](/nl/guides/integrating-with-trip-pay).
:::

## API

Ontwikkelaars die `Betalingen` willen beheren, gaan naar [Ontwikkelaars > API > Betalingen](/nl/developers/apis/#payment-api).
