Een agency is een normaal Wink-account dat je aan je account kunt toewijzen voor vertegenwoordiging. Hieronder staan enkele redenen waarom je een agency zou willen die jou vertegenwoordigt:

- Volledig / gedeeltelijk beheer van je Wink-account door een derde partij.
- Talent heeft een bestaande overeenkomst met hun agency en wil dat Wink het financiÃ«le aspect van hun regeling van binnenkomende gelden via Wink afhandelt.
- Talent agency doet al het online werk en beheert alle accounts op Wink.

## Agency toewijzen

Zo wijs je een agency aan je account toe.

<Steps>
 1. Log in op je gebruikersaccount op [Wink Studio](https://studio.wink.travel).
 2. Selecteer het affiliate-account waarmee je wilt werken.
 3. Klik in de hoofd navigatiebalk op `Account > Managers`.
 4. Typ in de sectie `Agency` de naam van de agency die jou moet vertegenwoordigen.
 5. Kies de agency fee uit de dropdown.
 5. Voeg optioneel vervalregels toe.
 7. Klik op de knop `Set agency` om door te gaan.
 </Steps>

 :::note
Je moet de eigenaar van het account zijn om een agency toe te wijzen.
 :::

 :::tip[Voorbeeld]
De agency fee is afgeleid van de affiliate commissie.

Boeking = $100, Commissie = 10%, Agency fee = 10% => 100 * 0.1 * 0.1 = $1
 :::

## Agency verwijderen

Zo verwijder je een agency van je account.

<Steps>
 1. Log in op je gebruikersaccount op [Wink Studio](https://studio.wink.travel).
 2. Selecteer het affiliate-account waarmee je wilt werken.
 3. Klik in de hoofd navigatiebalk op `Account > Managers`.
 4. Klik in de sectie `Agency` op het <Badge text="x" variant="danger" size="small" /> icoon aan de rechterkant van de agency dropdown.
 5. Klik op de knop `Set agency` om door te gaan.
 </Steps>


## API

Ontwikkelaars die agency via API willen beheren, kunnen terecht bij [Developers > API > Affiliate](/nl/developers/apis/#affiliate-api).
