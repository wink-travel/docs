Studio biedt verschillende insluitbare widgets die je aan je eigen website kunt toevoegen om boekingen te stimuleren.

Om toegang te krijgen tot insluitwidgets, navigeer je naar `Tools > Miscellaneous` in de zijbalk.

## Beschikbare widgets

| Widget | Beschrijving |
| -- | -- |
| **Account button** | Een merkgebonden knop die linkt naar je affiliatepagina |
| **Itinerary button** | Een knop die de reisschema-selector opent |
| **Itinerary form** | Een inline formulier voor het selecteren van datum en gasten |
| **Search button** | Een knop die de accommodatiezoeker opent |
| **Lookup button** | Een knop om specifieke inventaris op te zoeken |

## Hoe in te sluiten

Elke widget biedt een codefragment dat je kunt kopiëren en plakken in de HTML van je website. Het fragment bevat je **Client ID** en **Customization ID** zodat de widget automatisch gekoppeld is aan je affiliate-account.

:::note
Als je WordPress gebruikt, bieden we ook een WordPress-plugin voor eenvoudigere integratie.
:::
