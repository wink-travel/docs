Opgeslagen zoekopdrachten doen wat de naam al zegt. Tijdens het zoeken naar reisinventaris heb je de optie om je zoekopdracht op te slaan voor later gebruik.

## Je zoekopdracht opslaan

Wanneer je begint met zoeken naar inventaris (zie [Search](/nl/studio/search) ), kun je je zoekfilters verfijnen.
Onderaan het zoekformulier staat de optie `Save search`.

Door op deze link te klikken krijg je de mogelijkheid om deze zoekopdracht een naam te geven (*bijv. Mijn Bibliotheek van Magische Hotels*), iets wat je later zult onthouden.

<figure>

</figure>

Geef het een naam en klik op de knop `Save search`. Daarna kun je navigeren naar `Inventory > Saves Searches` in de hoofd navigatiebalk om je nieuw aangemaakte record te zien.

## Lijstacties

Je opgeslagen zoekopdracht kan worden gebruikt om reisinventaris aan je gebruikers te tonen in de vorm van een raster of een kaart en om je zoekresultaten weer te geven zoals jij dat wilt.

<figure>
<CloudinaryImage
      width="700"
      height="939"

    <figcaption>*Opgeslagen zoekopdracht zonder toegepaste filters*</figcaption>
</figure>

Elke opgeslagen zoekopdracht heeft acties die je erop kunt uitvoeren:

- Zet de zoekopdracht om in een [map](/nl/studio/maps).
- Zet de zoekopdracht om in een [grid](/nl/studio/grids).
- Verwijder de opgeslagen zoekopdracht.

## API

Ontwikkelaars die opgeslagen zoekopdrachten willen beheren kunnen terecht bij [Developers > API > Browse](/nl/developers/apis/#browse-api).
