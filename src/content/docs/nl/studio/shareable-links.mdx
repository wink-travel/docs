Deelbare links zijn een kernfunctie van Wink en zijn ontstaan uit de behoefte om voorraad en prijzen op een dynamische manier te communiceren via de eenvoudigste manier op het internet; namelijk de hyperlink.

<h4 align="center">[https://trvl.as/3xWCH](https://trvl.as/3xWCH)</h4>

Bovenstaand is een voorbeeld van een van onze links. Het is kort en draagbaar... zelfs kort genoeg om te onthouden. Hoewel kort, bevat deze link een **explosieve** hoeveelheid informatie, zoals:

- Je App ID.
- Je Affiliate ID.
- De afbeelding(en) en tekst die aan de gebruiker worden getoond.
- De look-en-feel (*d.w.z. thema, kleuren en meer*).
- De reisroute die wordt gebruikt voor onze prijsopvraging (*2 volwassenen, 1 nacht*).
- De valuta waarin de prijs wordt weergegeven.
- De taal waarin de tekst wordt weergegeven.

De rest van dit artikel leidt je door het maken, aanpassen en delen van je links.

Bovenstaand is een afbeelding afkomstig van [Zoeken](/studio/search) en toont je enkele van de dingen die je met je zoekresultaten kunt doen. Een van die acties is `Maak een link`.
Klik op die knop en je wordt doorgestuurd naar onze pagina met het formulier voor deelbare links waar je je link kunt gaan aanpassen.

## Link aanpassen

 Het formulier laat je je link op de volgende manieren aanpassen:

<Steps>
 1. Geef het een naam zodat je weet waar de link over gaat.
 2. Selecteer de [Aanpassing](/studio/customization) die je op deze link wilt toepassen.
 3. Voeg trefwoorden toe, gescheiden door een komma, die gebruikt zullen worden door [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Voer optioneel je X-accountnaam of Facebook App ID in.
 5. Voeg titels en beschrijvingen toe in de talen die je wilt dat de gebruiker ziet.
 6. Selecteer een of meer afbeeldingen die worden getoond wanneer je de link deelt op platforms zoals Facebook of LinkedIn.
 7. Kies een thema / overlay die een merkgebonden ontwerp toevoegt bovenop de afbeelding(en) die je hebt geselecteerd. *Opmerking: Dit is hoe je prijzen bovenop de afbeelding laat zien.*
 8. Klik op de knop `Opslaan` om door te gaan.
 </Steps>

Bovenstaand toont een afbeelding met alle beschikbare acties voor je link:

- **Voorbeeld** Toont je een snel overzicht van hoe de link is geconfigureerd en hoe deze eruit zal zien voor de gebruiker.
- **Bijwerken** Werkt je linkconfiguratie bij.
- **Toevoegen aan WinkLinks** Voegt de link toe aan je WinkLinks-account.
- **Kopiëren** Kopieert de link naar je klembord zodat je deze gemakkelijk overal kunt delen (*inclusief [WinkLinks](/link-manager/wink-links)*).
- **Delen** Opent een nieuw venster met opties om te delen op veel sociale netwerken en berichtplatforms.
- **QR-code** Genereert een QR-code die je kunt delen op platforms zoals IG die meer beeldgericht zijn.
- **Insluiten** Laat je zien hoe je deze link als een [Advertentiebanner](/developers/web-components/#content-loader) in je website kunt insluiten.
- **Gebruik met WordPress** Laat je zien hoe je onze [WordPress-plugin](/developers/wordpress) gebruikt om deze link in je website in te sluiten.

We behandelen enkele van deze opties hieronder uitgebreider.

Als je liever een QR-code deelt in plaats van een link, klik dan op de knop `QR-code` en er verschijnt een nieuw venster met je QR-code.

Hoe je je QR-code opslaat:

- Als je op je laptop bent, klik met de rechtermuisknop op de afbeelding en selecteer `Afbeelding opslaan`.
- Als je op je mobiele telefoon bent:
    - Houd je vinger op de afbeelding gedrukt totdat er een optie verschijnt om de afbeelding op te slaan in je Foto’s.
    - Maak een screenshot van de QR-code, snijd deze bij zodat hij past, en sla deze op in Foto’s.

### Insluiten

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Hier zie je hoe je je link als een advertentiebanner in je site insluit:

- Regel 3 laat zien hoe je de Wink-stijlen in je site insluit.
- Regels 6 tot en met 9 laten zien hoe je de [wink-content-loader](/developers/web-components/#content-loader) Web Component gebruikt en deze vertelt om een advertentiebanner voor je code op te halen.
- Regel 11 laat zien hoe je onze Javascript in je site insluit.
- Regel 13 laat zien hoe je de [wink-app-loader](/developers/web-components/#application-loader) Web Component insluit en deze vertelt om je pagina-voorkeuren op te halen.
