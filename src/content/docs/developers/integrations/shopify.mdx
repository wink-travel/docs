The **Wink Shopify App** lets tour operators, experience sellers, and hospitality merchants display live Wink inventory directly on their Shopify storefront — alongside products, on landing pages, or anywhere in the theme.

## Install

<Steps>
1. Go to the [Shopify App Store](https://apps.shopify.com) and search for **Wink Travel**.
2. Click **Add app** and follow the OAuth flow to authorize.
3. In the Wink app settings, enter your **Client ID** and **Client Secret**.
   Go to [Applications](/my-account/applications) to retrieve your credentials.
4. Click **Verify & Save**. A success message confirms the connection.
</Steps>

## Enable the App Embed Block

The App Embed Block automatically injects the Wink script and stylesheet into every page — you don't need to edit theme code.

<Steps>
1. In Shopify Admin, go to **Online Store → Themes → Customize**.
2. Click **App embeds** in the left panel.
3. Toggle **Wink Travel** on.
4. Click **Save**.
</Steps>

:::note
The App Embed Block reads your Client ID from a Shopify metafield set during install. Your Client Secret is never exposed to the browser.
:::

## Add a component to a page

<Steps>
1. In the theme editor, open the page or section where you want Wink content.
2. Click **Add block** and choose from the Wink blocks:
   - **Wink Content** — hotel cards, grids, or maps
   - **Wink Lookup** — destination search bar
   - **Wink Search** — itinerary picker button
   - **Wink Account** — sign-in button
   - **Wink Itinerary** — itinerary status button
   - **Wink Cart** — shopping cart button
3. For **Wink Content**, select a **Layout Type** and enter the **Layout ID** from Wink Studio.
4. Click **Save**.
</Steps>

## Available theme blocks

| Block | Purpose |
|---|---|
| **Wink Content** | Displays hotel cards, ranked grids, maps, and more |
| **Wink Lookup** | Search bar for destinations and hotels |
| **Wink Search** | Icon button that opens the itinerary picker |
| **Wink Account** | Sign-in / user account button |
| **Wink Itinerary** | Itinerary button showing the current trip |
| **Wink Cart** | Shopping cart summary button |
