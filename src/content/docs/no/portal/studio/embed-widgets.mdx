Studio tilbyr flere innebygde widgets du kan legge til på ditt eget nettsted for å øke bestillinger.

For å få tilgang til innebygde widgets, naviger til `Verktøy > Diverse` i sidemenyen.

## Tilgjengelige widgets

| Widget | Beskrivelse |
| -- | -- |
| **Konto-knapp** | En merkevareknapp som lenker til din affiliate-side |
| **Reiseplan-knapp** | En knapp som åpner reiseplanvelgeren |
| **Reiseplan-skjema** | Et innebygd skjema for dato- og gjestevalg |
| **Søk-knapp** | En knapp som åpner eiendomssøk |
| **Oppslag-knapp** | En knapp for å slå opp spesifikt inventar |

## Hvordan bygge inn

Hver widget gir et kodesnutt du kan kopiere og lime inn i nettstedets HTML. Snutten inkluderer din **Client ID** og **Customization ID** slik at widgeten automatisk kobles til din affiliate-konto.

:::note
Hvis du bruker WordPress, tilbyr vi også en WordPress-plugin for enklere integrasjon.
:::
