**Wink Wix App** legger til Wink webkomponenter på ditt Wix-nettsted som native widgets du kan dra og slippe i Wix Editor. Ingen koding kreves for nettstedseiere.

## Installer

<Steps>
1. Gå til [Wix App Market](https://www.wix.com/app-market) og søk etter **Wink Travel**.
2. Klikk **Legg til på nettsted** og følg instruksjonene for å autorisere appen.
3. Når den er installert, åpne **Dashboard → Wink Settings** og skriv inn din **Client ID** og **Client Secret**.
   Gå til [Applications](/no/portal/settings/applications) for å hente dine legitimasjoner.
4. Klikk **Lagre** og deretter **Test tilkobling** for å bekrefte at Wink er koblet til.
</Steps>

## Legg til en komponent på en side

<Steps>
1. Åpne **Wix Editor** og gå til siden der du vil legge til Wink-innhold.
2. Klikk **Legg til (+) → Apper → Wink Travel** for å se alle tilgjengelige widgets.
3. Dra en widget inn på siden — for eksempel **Wink Content** for å vise et hotellrutenett.
4. I widget-innstillingspanelet velger du **Layout Type** og limer inn **Layout ID** fra Wink Studio.
5. Klikk **Publiser** for å gjøre endringene synlige.
</Steps>

## Tilgjengelige widgets

| Widget | Formål |
|---|---|
| **Wink Content** | Viser hotellkort, rangerte rutenett, kart og mer |
| **Wink Lookup** | Søkelinje for destinasjoner og hoteller |
| **Wink Search** | Ikonknapp som åpner reiseplanleggeren |
| **Wink Account** | Påloggings- / brukerkontoknapp |
| **Wink Itinerary** | Reiseplan-knapp som viser gjeldende tur |
| **Wink Cart** | Handlekurvsammendrag-knapp |
