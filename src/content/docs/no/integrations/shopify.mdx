**Wink Shopify App** lar turoperatører, opplevelsesselgere og handelsbedrifter vise live Wink-inventar direkte på sin Shopify-butikk — sammen med produkter, på landingssider eller hvor som helst i temaet.

## Installer

<Steps>
1. Gå til [Shopify App Store](https://apps.shopify.com) og søk etter **Wink Travel**.
2. Klikk **Add app** og følg OAuth-flyten for å autorisere.
3. I Wink-appens innstillinger, skriv inn din **Client ID** og **Client Secret**.
   Gå til [Applications](/no/portal/settings/applications) for å hente dine legitimasjoner.
4. Klikk **Verify & Save**. En suksessmelding bekrefter tilkoblingen.
</Steps>

## Aktiver App Embed Block

App Embed Block injiserer automatisk Wink-skriptet og stilarket på hver side — du trenger ikke å redigere temakoden.

<Steps>
1. I Shopify Admin, gå til **Online Store → Themes → Customize**.
2. Klikk **App embeds** i venstre panel.
3. Slå på **Wink Travel**.
4. Klikk **Save**.
</Steps>

:::note
App Embed Block leser din Client ID fra et Shopify-metafelt som settes under installasjon. Din Client Secret blir aldri eksponert i nettleseren.
:::

## Legg til en komponent på en side

<Steps>
1. I temaredigereren, åpne siden eller seksjonen der du vil ha Wink-innhold.
2. Klikk **Add block** og velg blant Wink-blokkene:
   - **Wink Content** — hotellkort, rutenett eller kart
   - **Wink Lookup** — søkefelt for destinasjoner
   - **Wink Search** — knapp for reiseplanlegger
   - **Wink Account** — påloggingsknapp
   - **Wink Itinerary** — knapp for reiseplanstatus
   - **Wink Cart** — handlekurvknapp
3. For **Wink Content**, velg en **Layout Type** og skriv inn **Layout ID** fra Wink Studio.
4. Klikk **Save**.
</Steps>

## Tilgjengelige temablokker

| Blokk | Formål |
|---|---|
| **Wink Content** | Viser hotellkort, rangerte rutenett, kart og mer |
| **Wink Lookup** | Søkelinje for destinasjoner og hoteller |
| **Wink Search** | Ikonknapp som åpner reiseplanleggeren |
| **Wink Account** | Påloggings- / brukerkontoknapp |
| **Wink Itinerary** | Reiseplan-knapp som viser gjeldende tur |
| **Wink Cart** | Oppsummeringsknapp for handlekurv |
