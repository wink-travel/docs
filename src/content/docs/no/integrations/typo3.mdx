**Wink TYPO3 Extension** integrerer Wink reiseinventar i TYPO3 13 som native innholdselementer. Redaktører legger til Wink-komponenter på sider ved å bruke den vanlige TYPO3 sideeditoren — ingen malendringer kreves.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"

  description="Kildekode, utvikleroppsettguide og instruksjoner for bidrag."
/>

## Krav

- TYPO3 13.x
- PHP 8.2+
- Composer

## Installer

<Steps>
1. Installer via Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktiver utvidelsen:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. I TYPO3 backend, gå til **Admin Tools → Settings → Extension Configuration → wink**.
4. Skriv inn din **Client ID**, **Client Secret**, og velg miljø. Lagre.
   Gå til [Applications](/no/portal/settings/applications) for å hente dine legitimasjoner.
5. Legg til den statiske TypoScript-malen i din rotmal:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Legg til et innholdselement på en side

<Steps>
1. Åpne en side i **Page**-modulen.
2. Klikk på **+**-knappen i innholdsområdet for å legge til et nytt element.
3. Velg ett av **Wink Travel** innholdselementene:
   - **Wink Content** — hotellkort, rangerte rutenett, kart
   - **Wink Lookup** — destinasjonssøkefelt
   - **Wink Search** — knapp for reiseplanlegger
   - **Wink Account** — påloggingsknapp
   - **Wink Itinerary** — knapp for reiseplanstatus
   - **Wink Cart** — handlekurvknapp
4. For **Wink Content**, bruk FlexForm-feltene til å velge en **Layout Type** og skriv inn **Layout ID**.
5. Lagre og publiser siden.
</Steps>

## Tilgjengelige innholdselementer

| Element | Formål |
|---|---|
| **Wink Content** | Hotellkort, rangerte rutenett, kart og mer |
| **Wink Lookup** | Destinasjonssøkefelt |
| **Wink Search** | Ikonknapp som åpner reiseplanleggeren |
| **Wink Account** | Påloggings- / brukerkontoknapp |
| **Wink Itinerary** | Reiseplan-knapp som viser gjeldende tur |
| **Wink Cart** | Oppsummeringsknapp for handlekurv |
