**Wink Joomla Extension** er en pakke som installerer et systemplugin (laster CDN-skript), et innholdsplugin (behandler shortcode-tagger i artikler) og en komponent (admin-innstillingsside).

<LinkCard
  title="wink-travel/wink-joomla-extension on GitHub"
  href="https://github.com/wink-travel/wink-joomla-extension"
  description="Kildekode, utvikleroppsettguide og instruksjoner for bidrag."
/>

## Krav

- Joomla 4 eller 5
- PHP 8.1+

## Installer

<Steps>
1. Last ned siste `pkg_wink-x.x.x.zip` fra [GitHub releases-siden](https://github.com/wink-travel/wink-joomla-extension/releases).
2. I Joomla Admin, gå til **Extensions → Manage → Install**.
3. Last opp zip-filen og klikk **Upload & Install**.
4. Begge pluginene aktiveres automatisk. Gå til **Components → Wink Settings**.
5. Skriv inn din **Client ID** og **Client Secret** og klikk **Save**.
   Gå til [Applications](/no/portal/settings/applications) for å hente dine legitimasjoner.
</Steps>

## Legg inn komponenter i artikler

Bruk shortcode-lignende tagger hvor som helst i artikkelinnholdet:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Innholdspluginet erstatter disse taggene med korrekt `<wink-*>` HTML ved gjengivelse. Systempluginet laster automatisk CDN-skriptene og injiserer `<wink-app-loader>` i sidebunnen.

## Tilgjengelige shortcodes

| Shortcode | Output | Beskrivelse |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Hotellkort, rutenett, kart |
| `{wink-lookup}` | `<wink-lookup>` | Destinasjonssøkefelt |
| `{wink-search}` | `<wink-search-button>` | Knapp for reiseplanlegger |
| `{wink-account}` | `<wink-account-button>` | Pålogging / brukerkonto |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Knapp for reiseplanstatus |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Handlekurvknapp |
