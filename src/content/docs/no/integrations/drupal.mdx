**Wink Drupal-modulen** legger til Wink reiseinventarkomponenter på ditt Drupal-nettsted som native blokker. Redaktører kan plassere dem hvor som helst i oppsettet via det vanlige blokksystemet.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Kildekode, utvikleroppsettguide og instruksjoner for bidrag."
/>

## Krav

- Drupal 10 eller 11
- PHP 8.1+
- Composer

## Installer

<Steps>
1. Installer via Composer:
   ```bash
   composer require drupal/wink
   ```
2. Aktiver modulen:
   ```bash
   drush en wink -y
   ```
3. Gå til **Admin → Konfigurasjon → Webtjenester → Wink-innstillinger** (`/admin/config/wink`).
4. Skriv inn din **Client ID** og **Client Secret**, velg miljø, og klikk **Lagre**.
   Gå til [Applications](/no/portal/settings/applications) for å hente dine legitimasjoner.
</Steps>

## Legg til en blokk på en side

<Steps>
1. Gå til **Admin → Struktur → Blokkoppsett**.
2. Klikk **Plasser blokk** i regionen der du vil ha Wink-innhold.
3. Søk etter **Wink** for å finne tilgjengelige blokker:
   - **Wink Content** — viser hotellkort, rutenett eller kart
   - **Wink Lookup** — søkefelt for destinasjoner
   - **Wink Search** — knapp for reiseplanlegger
   - **Wink Account** — påloggingsknapp
   - **Wink Itinerary** — knapp for reiseplanstatus
   - **Wink Cart** — handlekurvknapp
4. For **Wink Content**, konfigurer **Layout Type** og **Layout ID** i blokkinnstillingene.
5. Lagre blokkinnstillingene.
</Steps>

## Tilgjengelige blokker

| Blokk | Formål |
|---|---|
| **Wink Content** | Viser hotellkort, rangerte rutenett, kart og mer |
| **Wink Lookup** | Søkelinje for destinasjoner og hoteller |
| **Wink Search** | Ikonknapp som åpner reiseplanleggeren |
| **Wink Account** | Påloggings- / brukerkontoknapp |
| **Wink Itinerary** | Reiseplan-knapp som viser gjeldende tur |
| **Wink Cart** | Oppsummeringsknapp for handlekurv |
