En Wink-bruker kan velge å la deg administrere en av sine eiendoms-, tilknyttede eller reisebyråkontoer. Når det skjer, vil du motta en e-post med en invitasjonsforespørsel.

import { Steps } from '@astrojs/starlight/components';

For å se dine nåværende Wink-invitasjoner, følg disse trinnene:

<Steps>
  1. Du må være logget inn på en av nettsidene våre.
  2. Klikk på profilikonet ditt øverst til høyre, så vises en meny.
  3. Klikk på `Invites` lenke.
  4. Samhandle med invitasjonene dine ved å enten `Accepting` eller `Rejecting` dem.
</Steps>
