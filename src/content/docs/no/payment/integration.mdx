Hvis du er et reiseselskap og ønsker å outsource betalings- og utbetalingsdelen av virksomheten din til TripPay, her er 6 grunnleggende steg for å komme i gang:

<Steps>
1. Opprett en [brukerkonto](/no/user/register) på Wink.
2. Opprett en konto hos [TripPay](https://pay.wink.travel).
3. Opprett en [Applikasjon](/no/developers/applications) på TripPay som lar deg kommunisere med vårt API.
4. Opprett og kartlegg dine mottakere hos TripPay. *f.eks. For de fleste betyr det å opprette kontoer for eiendommene på plattformen din som du ønsker at TripPay skal betale.*
5. Integrer [betalingsnettkomponenten](/no/developers/web-components/#payment) på nettstedet ditt.
6. Send TripPay reisendes `handlekurv` og start betalingsprosessen.
</Steps>

:::tip
For en komplett guide til bruk av TripPay for ditt reiseside, les guiden [Integrering med TripPay](/no/guides/integrating-with-trip-pay).
:::

## API

Utviklere som ønsker å administrere `Betalinger`, gå til [Utviklere > API > Betalinger](/no/developers/apis/#payment-api).
