## Payment API

Payment-endepunkter er for utviklere som ønsker å kjøpe reiseinventar. Dette kan gjøres via API som registrert reisebyrå eller ved å bruke vår API sammen med vår PCI-kompatible betalingswidget for alle andre enheter.
Payment API eksponerer endepunkter for affiliates og hoteller for å spore bookinger, analyser og tilgjengelighet av midler. De kan også velge å ta ut tilgjengelige midler til sin bankkonto.

</LinkButton>
