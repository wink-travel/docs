Wink integrerer med ledende channel managere, eiendomsstyringssystemer (PMS) og sentrale reservasjonsystemer (CRS) for å holde prisene og tilgjengeligheten synkronisert på tvers av alle dine salgskanaler.

## Slik fungerer det

Integrasjoner med channel manager muliggjør en toveis dataflyt mellom ditt eiendomsstyringssystem og Wink:

1. **Prisoppdatering** — Din channel manager sender prisoppdateringer til Wink. Når du oppdaterer prisene i ditt PMS, reflekteres disse endringene automatisk på Wink.
2. **Synkronisering av tilgjengelighet** — Din channel manager sender oppdateringer om tilgjengelighet til Wink. Når rom blir booket eller frigjort i ditt PMS, oppdateres Wink sitt lager i sanntid.
3. **Bookingvarsler** — Når en booking gjøres på Wink, blir din channel manager varslet slik at reservasjonen vises i ditt PMS.

## Støttede channel managere

Wink integrerer for øyeblikket med følgende systemer:

| Channel Manager | Beskrivelse |
|---|---|
| **Channex** | Global distribusjons- og channel management-plattform. |
| **TravelClick** | Hotelløsninger og channel management (Amadeus). |
| **Omnibees** | Distribusjon og inntektsstyring for hoteller. |
| **D-Edge** | Teknologi for gjestfrihet og sentrale reservasjonsløsninger. |
| **Allotz** | Lagerallokering og channel management. |
| **Ezee / Yanolja** | Skybasert PMS og channel management med fokus på det asiatiske markedet. |
| **RateGain** | Prisintelligens og distribusjonsteknologi. |
| **RateTiger** | Distribusjon av priser og tilgjengelighet i sanntid. |
| **CloudBeds** | Skybasert PMS, channel manager og bookingsystem. |
| **SiteMinder** | Ledende hotell channel manager og distribusjonsplattform. |
| **Exely** | Synkronisering av priser og tilgjengelighet for hoteller. |

:::note
Hvis din channel manager ikke er oppført, ta kontakt med Wink-teamet. Nye integrasjoner kan legges til basert på OTA (Open Travel Alliance) protokollstandarder som Wink støtter.
:::

## For utviklere

Hvis du bygger en channel manager- eller PMS-integrasjon med Wink, bruk Channel Manager API-endepunktene:

- **Prisoppdateringer** — Send prisendringer til Wink.
- **Oppdateringer av tilgjengelighet** — Send tilgjengelighetsendringer til Wink.
- **Henting av bookinger** — Hent bookinger fra Wink til ditt system.

Channel Manager API følger OTA-protokollstandarder (SOAP/XML) for kompatibilitet med eksisterende gjestfrihetssystemer.

## Webhook-varsler

Du kan abonnere på webhook-hendelser fra channel manager for å motta varsler i sanntid:

- `channel-manager.update.rate` — Prisoppdatering mottatt.
- `channel-manager.update.availability` — Tilgjengelighetsoppdatering mottatt.
- `channel-manager.update` — Generell oppdatering fra channel manager.

Se [Webhook Events Catalog](/no/developers/webhook-events#channel-manager-events) for detaljer.
