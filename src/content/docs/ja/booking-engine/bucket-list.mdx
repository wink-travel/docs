Winkの在庫[カード](/ja/studio/cards)をどこで見ても、カードの右上にハートが表示されており、そのハートを使ってその在庫をバケットリストに追加できます。

バケットリストを管理するには、以下の手順に従ってください：

<Steps>
1. 追加したい在庫の ♡ `ハートアイコン` をクリックします。
2. ログインしている場合、ハートアイコンが赤くなります。
3. ログインしていない場合は、ログイン画面にリダイレクトされます。
</Steps>

:::tip
♥️ `ハートアイコン` をもう一度クリックすると、リストから削除できます。
:::

右上のプロフィールアイコンをクリックし、`バケットリスト` ボタンをクリックすると、すべてのバケットリスト項目を管理できます。
