Winkの在庫が見られる場所ならどこでも[カード](/studio/cards)、カードの右上隅にハートが表示され、その在庫をバケット リストに追加できるようになります。

バケット リストを管理するには、次の手順に従います。

<Steps>
  1. ♡をクリックしてください `heart icon` 追加する在庫について。
  2. ログインしている場合は、ハートアイコンが赤に変わります。
  3. ログインしていない場合は、ログインするようにリダイレクトされます。
</Steps>


:::ヒント
♥️をクリックしてください`heart icon`もう一度クリックするとリストから削除されます。
:::

右上隅のプロフィールアイコンをクリックし、`Bucket list` ボタン。
