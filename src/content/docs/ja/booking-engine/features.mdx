予約エンジンには、アフィリエイト、不動産管理会社、ホテルのさまざまなニーズをサポートするために 2 種類あります。

## オンライン旅行代理店

アフィリエイトの場合、オンライン旅行代理店版を利用するのが最も理にかなっています。その理由は、複数の宿泊施設を閲覧したい旅行者に対応できるようにサイトが構築されているからです。

このアプリケーションのホームページは以下にあります。

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  ウィンクOTA
</LinkButton>


## インターネット予約エンジン

ホテルや不動産管理会社の場合、インターネット予約エンジン版のご利用が最も理にかなっています。このタイプのユーザーは、管理下にある単一の物件、または複数の物件をハイライト表示することに関心があります。

このアプリケーションのホームページは以下にあります。

  ウィンクIBE
</LinkButton>

## 構成

ほとんどの場合、これらのアプリケーションを直接共有する必要はありません。旅行の目録を共有するための様々な方法をご用意しており、以下でご覧いただけます。[ウィンクスタジオ](/studio/what-is-studio)または[ウィンクリンクス](/link-manager/wink-links)。

ただし、これらの Web サイトのいずれかに手動でリンクする場合は、クライアント ID とカスタマイズ ID をパラメータとして URL に追加してください。

クライアント ID は次の方法で確認できます。

<Steps>
  1. どちらかにログイン [ウィンクエクストラネット](/extranet/what-is-extranet) または [ウィンクスタジオ](/studio/what-is-studio)。
  2. 右上隅のプロフィールアイコンをクリックし、 [アプリケーション](/developers/applications)。
  3. 操作する Wink アカウントに関連付けられているアプリケーションを使用します。
  4. アプリケーションからクライアントIDをコピーし、リクエストパラメータとしてURLに貼り付けます。例：?client-id=MY\_CLIENT\_ID
</Steps>


### エクストラネット

カスタマイズ ID は、次の方法でプロパティとして見つけることができます。

<Steps>
  1. ログイン [ウィンクエクストラネット](/extranet/what-is-extranet)。
  2. 操作するプロパティを選択します。
  3. メインナビゲーションバーから、 `Account > Branding`
  4. カスタマイズIDをコピーし、リクエストパラメータとしてURLに貼り付けます。例：?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### スタジオ

アフィリエイトとしてのカスタマイズ ID は次の方法で見つけることができます。

<Steps>
  1. ログイン [ウィンクスタジオ](/studio/what-is-studio)。
  2. 操作するアカウントを選択します。
  3. メインナビゲーションバーから、 `Tools > Customization`
  4. 使用したいカスタマイズからカスタマイズIDをコピーし、リクエストパラメータとしてURLに貼り付けます。例：?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

最終的に次のような URL になります。

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

ホテルのページに直接アクセスしたい場合、URL は次のようになります。

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::note
クライアントIDや設定IDを指定しない場合、Wink OTAはデフォルトの設定を使用します。
:::

## IBE

不動産管理会社の場合は、Wink IBEのホームページにリンクを貼ります。ホームページには、ご指定のテーマと[あなたが代表する物件](/studio/agency-representation)。

最終的に次のような URL になります。

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

不動産業者の場合は、Wink IBE 不動産ランディング ページに直接リンクする必要があります。

その URL は次のようになります。

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::note
クライアントIDまたは設定IDを指定しないと、Wink IBEは読み込まれません。これらのパラメータは必須です。
:::

## 例: プロパティページ

Winkに掲載されている各宿泊施設には、それぞれのウェブサイトに専用のランディングページが用意されています。すべての外部リンクは、旅行者をこれらのランディングページのいずれかに誘導します。

ランディング ページには、旅行者がこの宿泊施設に宿泊するかどうかを十分な情報に基づいて決定するために必要なすべての情報が含まれています。
