予約エンジンは、アフィリエイト、プロパティ管理会社、ホテルのさまざまなニーズに対応するために2種類用意されています。

## オンライン旅行代理店

アフィリエイトの場合、オンライン旅行代理店バージョンを使用するのが最も理にかなっています。その理由は、複数の物件を閲覧したい旅行者に対応するようサイトが構築されているためです。

このアプリケーションのホームページは以下からご覧いただけます。

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## インターネット予約エンジン

ホテルやプロパティ管理会社の場合、インターネット予約エンジンバージョンを使用するのが最も理にかなっています。このタイプのユーザーは、単一の物件または管理下の複数物件を強調表示したいと考えています。

このアプリケーションのホームページは以下からご覧いただけます。

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE
</LinkButton>

## カスタマイズ

ホテルやアフィリエイトは、カラーパレットをカスタマイズすることで予約エンジンの`見た目と雰囲気`をカスタマイズできます。

以下の開発者サンドボックスでテーマのカスタマイズをお試しください。

<LinkButton href="https://developer.wink.travel/web-component" variant="secondary" icon="external" target="_blank">
  テーマ設定
</LinkButton>


## 設定

ほとんどの場合、これらのアプリケーションを直接共有する必要はありません。旅行在庫を共有するためのさまざまな方法を用意しており、[Wink Studio](/studio/what-is-studio)や[WinkLinks](/link-manager/wink-links)でご確認いただけます。

ただし、これらのウェブサイトのいずれかに手動でリンクしたい場合は、URLにクライアントIDとカスタマイズIDをパラメーターとして追加してください。

クライアントIDは以下の方法で確認できます：

<Steps>
1. [Wink Extranet](/extranet/what-is-extranet) または [Wink Studio](/studio/what-is-studio) にログインします。
2. 右上のプロフィールアイコンをクリックし、[Applications](/developers/applications) をクリックします。
3. 使用したいWinkアカウントに紐づくアプリケーションを選択します。
4. そのアプリケーションからクライアントIDをコピーし、URLのリクエストパラメーターとして貼り付けます。例：?client-id=MY_CLIENT_ID
</Steps>

### エクストラネット

プロパティとしてカスタマイズIDを確認するには：

<Steps>
1. [Wink Extranet](/extranet/what-is-extranet) にログインします。
2. 作業したいプロパティを選択します。
3. メインナビゲーションバーから `Account > Branding` をクリックします。
4. カスタマイズIDをコピーし、URLのリクエストパラメーターとして貼り付けます。例：?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### スタジオ

アフィリエイトとしてカスタマイズIDを確認するには：

<Steps>
1. [Wink Studio](/studio/what-is-studio) にログインします。
2. 作業したいアカウントを選択します。
3. メインナビゲーションバーから `Tools > Customization` をクリックします。
4. 使用したいカスタマイズのカスタマイズIDをコピーし、URLのリクエストパラメーターとして貼り付けます。例：?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

最終的に得られるURLは次のようになります：

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

ホテルページに直接アクセスしたい場合、URLは次のようになります：

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
クライアントIDおよび/またはカスタマイズIDを含めない場合、Wink OTAはデフォルトの設定を使用します。
:::

## IBE

プロパティ管理会社の場合、Wink IBEのホームページにリンクしたいでしょう。ホームページには指定されたテーマと[あなたが代表する物件](/studio/agency-representation)が表示されます。

最終的に得られるURLは次のようになります：

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

プロパティの場合は、Wink IBEのプロパティランディングページに直接リンクしたいでしょう。

そのURLは次のようになります：

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
クライアントIDおよび/またはカスタマイズIDを含めない場合、Wink IBEは起動しません。これらのパラメーターは必須です。
:::

## 例：プロパティページ

Winkのすべてのプロパティには、それぞれのウェブサイトで独自のランディングページがあります。すべての外部リンクは旅行者をこれらのランディングページのいずれかに誘導します。

ランディングページには、旅行者がこのプロパティに宿泊するかどうかを判断するために必要なすべての情報が含まれています。
