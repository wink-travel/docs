投稿エディターでは、無制限の手動投稿を作成できるほか、自動的に作成された投稿の管理も行えます。

エディターは4つのセクションで構成されています：

1. コンテンツセクション
2. メディアセクション
3. リンクセクション
4. 設定セクション

## コンテンツ

コンテンツセクションでは、公開前にテキスト内容を修正できます。複数言語でローカライズされたコンテンツを作成でき、それぞれの言語にタイトルと説明が設定されます。

## メディア

メディアセクションでは、テキストに添える画像や動画を修正できます。

## リンク

リンクセクションでは、投稿を読んだ後にユーザーを誘導したい場所へのリンクを設定できます。*例：特別な割引ページや詳細情報のページへのリンクなど*。リンクURL、リンクキャプション、リンク説明を追加可能です。

## 設定

設定セクションでは、投稿ごとにデフォルト設定を変更・上書きできます。設定できる項目は：

- **プラットフォーム** — この投稿を公開するソーシャルネットワークを選択します。
- **投稿タイプ** — トランザクショナル、認知度向上、感情表現のいずれかを選択します。
- **トーンオブボイス** — この投稿に特化したトーンを上書きします。
- **カスタマイズ** — 表示通貨、言語、旅程などのカスタマイズを別途設定します。
- **タイムゾーン** — 公開時のタイムゾーンを上書きします。
- **スケジュール日時** — 自動生成された投稿は最適な時間に自動スケジュールされますが、別の日時に変更可能です。

## 投稿のステータスライフサイクル

すべての投稿は以下のステータスライフサイクルに従います：

| ステータス | 説明 |
| -- | -- |
| **下書き** | 投稿は作成されたが、まだレビュー準備ができていない状態 |
| **承認待ち** | 投稿はスケジュール前の人間による承認を待っている状態 |
| **承認済み** | 投稿は承認され、スケジュールされる状態 |
| **却下** | 投稿はレビューされ却下されたため、公開されない状態 |
| **スケジュール済み** | 投稿は指定日時に公開されるためにキューに入っている状態 |
| **公開済み** | 投稿はソーシャルネットワークに正常に公開された状態 |
| **一時停止** | 投稿の公開が一時的に停止されている状態 |
| **失敗** | エラーにより投稿が公開できなかった状態 |

## 投稿の承認

投稿は投稿グリッドで承認されます。複数の投稿を選択して承認または却下を選べます。

:::note
言語ごと、ソーシャルネットワークごとに1件ずつエントリーがあります。

例：3つのソーシャルネットワークと3つの言語を選択している場合、カレンダーには9件のエントリーが表示されます。
:::
