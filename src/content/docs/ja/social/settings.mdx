# 設定

Socialを初めて使い始めるとき、どのようにコンテンツを作成するかをお尋ねします。これらのオプションはいつでも利用可能で、後から簡単に変更できます。

## 設定

Social設定の最初のタブには基本的な設定が含まれています。内容は以下の通りです：

1. **優先カスタマイズ** コンテンツ生成時に使用するカスタマイズを選択します。カスタマイズは[my.wink.travel](https://my.wink.travel)のアカウントで作成します。カスタマイズはテーマカラー、デフォルトの旅程などを制御し、当社の予約エンジンの文脈で機能します。
2. **自動承認** 毎月新しいコンテンツをすべて承認なしで公開してもよい場合は、このスイッチを有効にして、すべての作業をお任せください。
3. **承認者** ただし、新しく作成された投稿の通知を受け取り、承認したい場合は、その担当者の連絡先情報を入力してください。


## コンテンツクリエイター

<CldImage

Socialには多くのコンテンツクリエイターが用意されています。
コンテンツクリエイターは、物件のユニークなコンテンツを作成する特定の方法に焦点を当てています。各コンテンツクリエイターには`weight`属性があり、ランダムに選ばれる確率を示します。

### 例：

ホテルがAdd-OnとPropertyクリエイターを選択。Add-Onクリエイターの重みは5つ星、Propertyクリエイターの重みは10つ星です。Propertyコンテンツが作成される可能性はAdd-Onよりもはるかに高くなります。

生成するテキストは、ブランドにより合った異なる`声` / `ペルソナ`を使用できます。あなたに合うものを選んでください。コンテンツクリエイターと同様に、**Weight**を増やすことでペルソナが使用される確率を設定できます。

## Googleビジネス

<figure>
<CldImage
  src="academy/social_google_business"
  width="700"
  height="292"
  alt="Social Google Business"
/>
  <figcaption>*図4 - Googleビジネス*</figcaption>

Googleビジネスアカウントを追加することで、補足コンテンツとして利用できる多くの情報を抽出できます。一部のコンテンツクリエイターはGoogleビジネスアカウントの所有を**必須**としています。

:::tip[プロのヒント]
小さく始める

最初から野心的になりすぎないでください。物件のコンテンツを単一言語で作成することから始めることをお勧めします。
:::

:::tip[プロのヒント]
選択したコンテンツクリエイターに必要な在庫タイプを必ず用意してください。
:::
