# 接続

<figure>
<CldImage
  src="academy/social_connections"
  width="700"
  height="486"
  alt="Social connections"
/>
  <figcaption>*図1 - 接続*</figcaption>
</figure>

すでにリンク管理ツールであるWinkLinksに接続されています。`Connect a social network`をクリックして、さらに多くのソーシャルネットワークに接続してください。
このプロセスでは、`Wink Social`アプリがあなたに代わってコンテンツを投稿することを承認するために、ソーシャルネットワークにリダイレクトされます。

接続が完了すると、次の予定された月に向けてそのネットワーク用のコンテンツの生成を開始します。

:::note
各ソーシャルネットワークに最適化されたコンテンツを作成し、公開します。
:::
