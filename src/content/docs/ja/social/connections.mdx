あなたはすでにリンク管理ツールであるWinkLinksに接続しています。`Connect a social network`をクリックして、さらに多くのソーシャルネットワークに接続してください。
このプロセスでは、あなたのソーシャルネットワークにリダイレクトされ、`Wink Social`アプリがあなたに代わってコンテンツを投稿することを承認する必要があります。

接続が完了すると、次のスケジュールされた月に向けてそのネットワーク用のコンテンツの生成を開始します。

:::note
無料プランにはWinkLinksが含まれています。Facebook、Instagram、その他のプラットフォームは有料プランで利用可能です。
:::

## 対応プラットフォーム

| プラットフォーム | 説明 | 要件 |
| -- | -- | -- |
| **WinkLinks** | 高度なリンク管理ツールへの自動投稿 | すべてのプランに含まれています |
| **Facebook** | 投稿の自動化、コメントやDMへの返信 | Facebookビジネスページが必要です |
| **Instagram** | 投稿の自動化、コメントやDMへの返信 | FacebookアカウントにリンクされたInstagramビジネスアカウントが必要です |

各接続は現在のステータスを表示します：**アクティブ**、**切断済み**、または**停止中**。

:::note
各ソーシャルネットワークに最適化されたコンテンツを作成し、公開します。
:::
