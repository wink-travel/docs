# 概要

ソーシャルメディアはホテルにとって最も強力なマーケティングツールの一つですが、多くの施設は高品質で魅力的なコンテンツを継続的に投稿し、成果につなげるための時間、専門知識、リソースを持っていません。
私たちは、AI搭載の自動化を活用し、ホテル側の手動作業なしで動的かつリアルタイムなコンテンツを生成・投稿することで、この問題を解決します。

- AIによるホテルごとの提供内容に合わせたソーシャルメディアコンテンツ。
- リアルタイムの在庫、プロモーション、ユーザーアップロードコンテンツを活用。
- 完全な承認プロセスでホテルのブランドコントロールを維持。
- 手動作業ゼロで直接予約を最大化。

この機能により、ホテルは旅行者とのエンゲージメントを変革し、一貫性があり最適化された収益重視のソーシャルメディアプレゼンスを大規模に維持できます。

一般的なAIソーシャルメディアツールとは異なり、当システムはホテル専用に設計されており、リアルタイムの在庫、料金、ホテルコンテンツと直接連携しています。

## 仕組み

あらゆる規模の施設向けに開発された高度なコンテンツ管理プラットフォームにAIを組み込み、ホテルが存在するあらゆる場所にコンテンツを配信できるスケジューリング機能を搭載するのは自然な進化でした。

ソーシャルメディア自動化エンジンは、AIとWinkの無料機能（共有可能なリンク、リアルタイム在庫アクセス、ホテル提供コンテンツなど）を活用し、直接予約に最適化された投稿を作成します。

1. AIがWinkのCDNからコンテンツをスキャン。
2. AIがソーシャルメディア投稿を作成。
3. コンテンツはホテルに承認のため送信。
4. 投稿は希望言語に翻訳。
5. 選択されたソーシャルネットワークに投稿をスケジュール。

### コンテンツソース

AIは以下を使って動的にソーシャルメディア投稿を生成します：

- ホテルエクストラネットデータ：客室説明、画像、アメニティ、料金。
- リアルタイム在庫：空室状況、期間限定プロモーション、フラッシュセール。
- ユーザーアップロードコンテンツ：カスタム画像、動画、テキスト。
- 公開されているホテルコンテンツ。

### AI生成投稿

投稿は以下の手順で生成されます：

1. 利用可能なコンテンツに対して感情分析を実施。
2. 既存の画像を活用。
3. 事前定義されたペルソナを用いてキャプション、テキスト、ハッシュタグを強化。
4. リアルタイム価格オーバーレイ付きの画像プレビューを作成。

#### 投稿タイプ

生成された投稿は以下の3タイプのいずれかに分類されます：

- **トランザクショナル** — コンバージョン重視。事実、価格、次のステップで予約を促進。
- **認知向上** — ブランド構築。施設を紹介し認知度を高める。
- **感情共鳴** — 感情に訴える。オーディエンスにポジティブな印象を与える。

#### 例

- 「早期予約でお得に！💰」（トランザクショナル）
- 「お部屋と一緒に釣り🐟体験も予約しよう！」（トランザクショナル）
- 「10人中9人の旅行者が💕このホテルを評価」（感情共鳴）
- 「久しぶりに訪れた最高の場所。💥 --Dr. Shu」（感情共鳴）
- 「ディズニーランドからわずか200m💦」（認知向上）
- 「空港まで10分🛫」（認知向上）

### 承認プロセス

ホテルは常にコントロールを保持しつつ、ハンズフリーの自動化の恩恵を受けられます。すべてを自動化する「設定して忘れる」ポリシーを選択することも可能です。

ただし、ソーシャルチャネルの完全管理を任せる前に、翌月に投稿予定の各投稿の通知と承認を求めることもできます。

以下が可能です：

- 承認：投稿の翻訳言語と配信するソーシャルネットワークを選択。
- 修正：AIに代替バージョンを生成させ、手動で変更。
- 拒否：投稿を破棄。

### スケジューラー

投稿のタイムゾーンと言語を選択でき、投稿はリーチとエンゲージメントが最大化されるピークの曜日・時間にスケジュールされます。

### 分析

- すべてのチャネルでクリック数、コンバージョン、エンゲージメントをリアルタイムで追跡。
- 高パフォーマンス投稿はリサイクル・再利用されリーチを拡大。
- AIは過去のパフォーマンスから学習し、将来の投稿を最適化。

## 自動DM＆返信

自動DM＆返信機能は、ホテルがソーシャルメディア上のコメントやダイレクトメッセージを効率的に管理できるよう支援します。AIを活用し、施設内設備や近隣観光に関する問い合わせに迅速かつ正確に対応し、トラッカブルな予約リンクを提供。会話を予約に変換しやすくします。

### 例

1. ユーザーがソーシャル投稿に「美しい！！！ここにいたい😊」と返信。
2. 自動返信例：「いいですよね！！4月の夕日は本当に素晴らしいです。ちなみに早期予約で30%オフになります。こちらをチェック https://trvl.as/AB76g」

## AIコンシェルジュ

AIコンシェルジュは自動返信＆DMと同じ技術を使い、ホテルのウェブサイト向けにカスタムチャットボットを提供します。リアルタイムで潜在顧客と対話し、質問に答え、パーソナライズされたおすすめを提供し、予約完了をサポート。ウェブサイト訪問者をゲストに変えやすくします。

### 例

👤「こんにちは…質問があります。」

🤖「こんにちは！どうされましたか？」

👤「訪問に最適な時期はいつですか？」

🤖「良い質問です。12月から1月が最盛期です。もし2月に来られるなら、天気は同じですが料金が安く人も少ないです。」

👤「2月のプロモコードはありますか？」

🤖「ありませんが、30日前予約でツインルームの割引価格からさらに15%オフになります。」

🤖「こちらが部屋の写真です [Image]」

👤「それにします！」

🤖「素晴らしい、こちらのリンクをご利用ください https://trvl.as/AB76g」

🤖「他にお手伝いできることはありますか？」

## 料金

Socialはサブスクリプション型サービスで、各プランにはボリュームディスカウントが適用されます。

プランごとの主な違いは以下の通りです。

| 機能 | Starter | Growth | Elite | Enterprise |
| -- | -- | -- | -- | -- |
| AI生成投稿数 | 4 | 16 | 20 | 無制限 |
| 手動投稿数 | 無制限 | 無制限 | 無制限 | 無制限 |
| 画像投稿 | ✅ | ✅ | ✅ | ✅ |
| 動画投稿 | ❌ | ✅ | ✅ | ✅ |
| 多言語対応 | ✅ | ✅ | ✅ | ✅ |
| 自動DM＆返信 | ❌ | ✅ | ✅ | ✅ |
| AIコンシェルジュ | ❌ | ❌ | ✅ | ✅ |
| Wink Agency | ❌ | ❌ | ✅ | ✅ |
| 広告ブースト | ❌ | ❌ | ❌ | ✅ |
| エンゲージメント分析 | ✅ | ✅ | ✅ | ✅ |
| 予約分析 | ❌ | ✅ | ✅ | ✅ |
| サポート | メール | メール | プレミアム | アカウントマネージャー |

### どのプランを選ぶべきか

現在のソーシャルマーケティング戦略の状況によります。各プランの主な特徴を簡単にご紹介します。

#### Starter

ソーシャルでの認知が全くない場合、まずは`Starter`プランで試してみてください。
無料プランでは、プラットフォーム経由の予約は無制限で利用可能で、WinkLinksへの月次投稿も含まれます。Starterプランは月4件のAI生成投稿に制限されますが、手動投稿は無制限にスケジュール・投稿可能です。Facebook、Instagram、フルのバーチャルセールス＆マーケティングチームを利用したい場合は、有料プランをご検討ください。

#### Growth

`Growth`プランの大きな違いは、DM管理と投稿コメントへの返信を最適化したバーチャルセールスチームにアクセスできる点です。潜在顧客を確定予約に変えることを目的としています。

#### Elite

Eliteは機能が大幅に拡充されます。すぐにAIチャットボットを自社ウェブサイトに統合可能です。
`自動DM＆返信`と同じ技術をベースに、サイトに合わせたカスタマイズオプションが豊富です。
また、統合支援を行うプレミアムサポートも付帯します。

さらに、[Wink Agency](/ja/agency/what-is-agency)に初めてアクセスでき、インフルエンサーやアフィリエイトの大軍と連携して施設の販売を支援します。

#### Enterprise

Enterpriseは複数プランのグループ割引を希望するブランドやチェーン向けです。
対応プラットフォームでのコンテンツ`ブースト`機能も含まれます。複数のソーシャルネットワークでブーストしたい投稿を選択可能です。
3 / 6 / 12ヶ月分の広告費用を請求されます。

:::tip[プロのヒント]
プランを段階的に拡大しましょう

関係構築と信頼は時間がかかります。まずは手動で投稿を承認しながらSocialを運用してください。信頼が深まったら、1ヶ月間自動承認を試してみましょう。
:::
