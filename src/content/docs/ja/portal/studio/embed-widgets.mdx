Studioでは、予約促進のために自分のウェブサイトに追加できる複数の埋め込みウィジェットを提供しています。

埋め込みウィジェットにアクセスするには、サイドバーの `Tools > Miscellaneous` に移動してください。

## 利用可能なウィジェット

| ウィジェット | 説明 |
| -- | -- |
| **アカウントボタン** | アフィリエイトページへのリンクが付いたブランドボタン |
| **旅程ボタン** | 旅程セレクターを開くボタン |
| **旅程フォーム** | インラインの日付とゲスト選択フォーム |
| **検索ボタン** | 物件検索を開くボタン |
| **照会ボタン** | 特定の在庫を照会するボタン |

## 埋め込み方法

各ウィジェットは、ウェブサイトのHTMLにコピー＆ペーストできるコードスニペットを提供します。スニペットには**クライアントID**と**カスタマイズID**が含まれており、ウィジェットは自動的にあなたのアフィリエイトアカウントにリンクされます。

:::note
WordPressを使用している場合は、より簡単に統合できるWordPressプラグインも提供しています。
:::
