# 設定

Socialを初めて使い始めるとき、コンテンツをどのように作成するかをお尋ねします。これらのオプションはいつでも利用可能で、後から簡単に変更できます。

サイドバーの`Preferences`に移動して設定を管理してください。

Preferencesセクションには基本的なコンテンツ生成設定が含まれています：

1. **Preferred customization** コンテンツ生成時に使用するカスタマイズを選択します。カスタマイズはテーマカラー、デフォルトの旅程、通貨、言語などを制御します。カスタマイズは[my.wink.travel](https://my.wink.travel)のアカウントで作成します。
2. **Auto-approve** 有効にすると、生成された投稿が自動的に公開され、承認は不要になります。コンテンツの品質に自信がついたら有効にしてください。
3. **Approver name** 承認リクエストのメールを受け取る担当者またはチームの名前。
4. **Approver e-mail** 今後の投稿の承認リクエストが送信されるメールアドレス。

## Content creators

Socialには多くのコンテンツクリエイターが用意されています。各クリエイターはAIを使って物件のユニークなコンテンツを作成する特定の方法に特化しています。すべてのコンテンツクリエイターには**Weight**属性があり、重みが高いほど投稿生成時に選ばれる確率が高くなります。

利用可能なコンテンツクリエイターは以下の通りです：

| クリエイター | 説明 |
| -- | -- |
| **Destination** | あなたのロケーションを使って地域に関する役立つテキストを作成します |
| **Room type** | AI生成テキストと部屋タイプの画像を使った予約可能な投稿を作成します |
| **Activity** | あなたのアクティビティの予約可能な投稿を作成します |
| **Add-On** | あなたのアドオンの予約可能な投稿を作成します |
| **Attraction** | あなたのアトラクションの予約可能な投稿を作成します |
| **Place** | あなたの場所の予約可能な投稿を作成します |
| **Meeting room** | あなたの会議室の予約可能な投稿を作成します |
| **Restaurant** | あなたのレストランの予約可能な投稿を作成します |
| **Spa** | あなたのスパの予約可能な投稿を作成します |
| **Property** | AI生成テキストと物件の画像を使った最良価格の部屋の予約可能な投稿を作成します |
| **User sentiment** | ウェブ上のゲストレビューを集約し、ゲストが物件のどこを気に入っているかを強調した感情に基づく投稿を作成します |
| **Funny** | 軽快でエンターテインメント性のある投稿を作成し、フィードを魅力的に保ちます |

:::tip[プロのヒント]
選択したコンテンツクリエイターに必要な在庫タイプを必ず用意してください。例えば「Restaurant」を選ぶ場合は、物件に少なくとも1つのレストランが登録されている必要があります。
:::

## Tone of voice

トーンオブボイスは、AIがコンテンツ生成時に使用する言語スタイルを設定します。ブランドの個性に合ったトーンを選んでください。コンテンツクリエイターと同様に、各トーンには使用頻度を制御する**Weight**があります。

利用可能なトーンは以下の通りです：

| トーン | スタイル |
| -- | -- |
| **Neutral** | 落ち着いた、権威ある、事実に基づく。誇張や冗談なしで信頼できる構造化された知識 |
| **Friendly** | 温かく親しみやすい、テキストから笑顔が伝わるエネルギー。カジュアルな短縮形や時折の絵文字 |
| **Luxury** | 洗練されて上品。「exquisite」や「bespoke」のような言葉で読者に特別感を与える |
| **Foodie** | 官能的で食欲をそそる表現。食べ物の魅力を伝える言葉で読者をお腹が空く気分にさせる |
| **Adventurous** | 大胆で勇敢、探検家のエネルギー。鮮やかな感覚的描写で全てを壮大に感じさせる |
| **Family** | 健全で子供に優しい興奮。安全で楽しい体験についての温かい親のエネルギー |
| **Romantic** | 詩的で物思いにふける、胸がときめく。夕日や柔らかい言葉、愛を表現 |
| **Humorous** | 皮肉で機知に富んだ一言、巧みな言葉遊び。スタンドアップコメディのように読める |
| **Eco** | 真摯で正義感あふれる。環境に優しい選択や持続可能な選択を称賛 |
| **Local insider** | 非常に具体的な地元の言及、観光客向け罠を警告、地元のスラングを使用 |
| **Inclusive** | 受け入れられ、祝福され、肯定的。誰もが見られ、価値を感じられるようにする |
| **Mystical** | 霊的で宇宙的な真実。内面を見つめる優しい誘い |
| **Futuristic** | スマートで未来志向。「パラダイムシフト」や楽観的な未来像のような言葉を好む |
| **Budget** | 質素で工夫に富み、誇り高い節約家。手頃な発見や無料体験を称賛 |

## Post types

投稿タイプは生成されるコンテンツの目的と構造を決定します。各投稿タイプには使用頻度を制御する**Weight**があります。

| 投稿タイプ | 説明 |
| -- | -- |
| **Transactional** | 明確で簡潔、コンバージョン重視。事実、価格、次のステップで予約や直接行動を促進 |
| **Awareness** | ブランド構築のためのコンテンツで、物件を紹介し潜在的なゲストへの認知度を高める |
| **Sentiment** | 感情に響くコンテンツで、ポジティブな連想を生み、オーディエンスとの感情的なつながりを構築 |

特定の言語や地域にマーケティングを集中させたい場合があります。コンテンツを作成したい言語と関心のあるタイムゾーンを教えてください。投稿は選択したタイムゾーンのスケジュールされた時間に公開されます。

*ほとんどの物件は単一言語を使用し、現地の言語をデフォルトにしています。*

## Google Business

<figure>
<CldImage
  src="academy/social_google_business"
  width="700"
  height="292"
  alt="Social Google Business"
/>
  <figcaption>*図5 - Google Business*</figcaption>

Google Business Profileを連携することで、より豊富な物件情報を取得し、より効果的なコンテンツを作成できます。物件には認証済みのGoogle Business Profileが必要で、そこからコンテンツを生成します。一部のコンテンツクリエイターはGoogle Businessアカウントの所有を**必須**としています。

:::tip[プロのヒント]
小さく始める

最初から野心的になりすぎないでください。物件のコンテンツを単一言語で作成することから始めることをお勧めします。
:::
