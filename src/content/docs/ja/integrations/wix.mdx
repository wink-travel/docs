**Wink Wix App**は、WinkのウェブコンポーネントをWixサイトにネイティブウィジェットとして追加し、Wixエディターでドラッグ＆ドロップできます。サイトオーナーにコーディングは不要です。

## インストール

<Steps>
1. [Wix App Market](https://www.wix.com/app-market)にアクセスし、**Wink Travel**を検索します。
2. **サイトに追加**をクリックし、アプリの認証手順に従います。
3. インストール後、**ダッシュボード → Wink設定**を開き、**Client ID**と**Client Secret**を入力します。  
   資格情報は[Applications](/ja/portal/settings/applications)から取得してください。
4. **保存**をクリックし、**接続テスト**でWinkとの接続を確認します。
</Steps>

## ページにコンポーネントを追加する

<Steps>
1. **Wixエディター**を開き、Winkコンテンツを追加したいページに移動します。
2. **追加 (+) → アプリ → Wink Travel**をクリックして利用可能なウィジェットを表示します。
3. ウィジェットをページにドラッグします。例：ホテルグリッドを表示する**Wink Content**。
4. ウィジェット設定パネルで**レイアウトタイプ**を選択し、Wink Studioからコピーした**レイアウトID**を貼り付けます。
5. **公開**をクリックして変更を反映します。
</Steps>

## 利用可能なウィジェット

| ウィジェット | 用途 |
|---|---|
| **Wink Content** | ホテルカード、ランキンググリッド、地図などを表示 |
| **Wink Lookup** | 目的地やホテルの検索バー |
| **Wink Search** | 旅程ピッカーを開くアイコンボタン |
| **Wink Account** | サインイン／ユーザーアカウントボタン |
| **Wink Itinerary** | 現在の旅行を表示する旅程ボタン |
| **Wink Cart** | ショッピングカートの概要ボタン |
