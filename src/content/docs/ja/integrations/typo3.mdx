**Wink TYPO3 Extension**は、Winkの旅行在庫をTYPO3 13のネイティブコンテンツ要素として統合します。編集者は標準のTYPO3ページエディターを使ってページにWinkコンポーネントを追加できます。テンプレートの変更は不要です。

## 要件

- TYPO3 13.x
- PHP 8.2以上
- Composer

## インストール

<Steps>
1. Composerでインストール：
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. 拡張機能を有効化：
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. TYPO3バックエンドで、**管理ツール → 設定 → 拡張機能設定 → wink** に移動。
4. **Client ID**、**Client Secret**を入力し、環境を選択して保存。
   資格情報は[Applications](/ja/portal/settings/applications)から取得可能です。
5. ルートテンプレートに静的TypoScriptテンプレートを追加：
   **テンプレート → 編集 → インクルード → 利用可能な項目 → Wink Travel**。
</Steps>

## ページにコンテンツ要素を追加する

<Steps>
1. **ページ**モジュールでページを開く。
2. コンテンツエリアの**+**ボタンをクリックして新しい要素を追加。
3. **Wink Travel**のコンテンツ要素のいずれかを選択：
   - **Wink Content** — ホテルカード、ランキンググリッド、地図
   - **Wink Lookup** — 目的地検索バー
   - **Wink Search** — 旅程ピッカーボタン
   - **Wink Account** — サインインボタン
   - **Wink Itinerary** — 旅程ステータスボタン
   - **Wink Cart** — ショッピングカートボタン
4. **Wink Content**の場合、FlexFormフィールドで**レイアウトタイプ**を選択し、**レイアウトID**を入力。
5. ページを保存して公開。
</Steps>

## 利用可能なコンテンツ要素

| 要素 | 用途 |
|---|---|
| **Wink Content** | ホテルカード、ランキンググリッド、地図など |
| **Wink Lookup** | 目的地検索バー |
| **Wink Search** | 旅程ピッカーを開くアイコンボタン |
| **Wink Account** | サインイン／ユーザーアカウントボタン |
| **Wink Itinerary** | 現在の旅程を表示するボタン |
| **Wink Cart** | ショッピングカートの概要ボタン |
