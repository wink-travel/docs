ビジネスやブログでWordPressを使っている方には、[Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/)の使用を強くおすすめします。
このプラグインを使うと、Winkのカード、グリッド、マップなどに簡単にアクセスでき、お気に入りのページビルダーやカスタムショートコードを使って投稿やページにコンテンツをドラッグするだけで利用可能になります。

## インストール

`Wink Affiliate WordPress Plugin`をWordPressサイトにインストールする方法は以下の通りです。

<Steps>
1. 管理者としてログインし、ダッシュボードにいることを確認します。
2. `プラグイン > 新規追加`をクリックします。
3. `Wink Affiliate WordPress Plugin`を検索します。
4. `インストール`をクリックしてプラグインをダウンロードし、サイトにインストールします。
5. `有効化`をクリックしてプラグインをサイトで有効にします。
6. 有効化後、`Wink設定`ページにリダイレクトされます。*または`外観 > カスタマイズ > Wink設定`からもアクセス可能です。*
7. `Client ID`と`Client Secret`の入力を求められます。[Applications](/ja/developers/applications)で認証情報の取得方法を確認してください。
8. 認証情報を入力し、`公開`をクリックします。これでWinkアカウントと接続されました。
</Steps>

## ページビルダー

Winkを使いたい方にとって使いやすい、WordPressで最も人気のあるページビルダーをサポートしています。

### Gutenberg

GutenbergはWordPressのネイティブエディターです。

### Avada

[Avada](https://avada.com)は、美しいテンプレートの豊富なセレクションと高度な機能を備えた堅牢なページビルダーで有名です。

### Elementor

[Elementor](https://elementor.com)は、現在最も人気のあるページビルダーで、無料版とプロ版の両方が利用可能です。

### WPBakery

[WPBakery](https://wpbakery.com)は、市場で最初の高度なページビルダーでした。

## ショートコード

### [winklookup]

ランキングされたコンテンツグリッドで動作します。訪れたい場所を入力すると、在庫が表示されます。

### [winksearch]

旅程フォームを開くシンプルなボタンです。

### [winkaccount]

認証用のアカウントボタンです。認証後はアカウントオプションのドロップダウンに変わります。

### [winkitinerary]

検索ボタンと同じですが、ボタンテキストに旅程情報が含まれています。

### [winkcontent]

Winkで選択した在庫をページや投稿に直接埋め込むことができます。このショートコードはレイアウトIDも必要とします。

レイアウト

```
[winkcontent layoutid=""]
```
