ビジネスやブログにWordPressをご利用の方は、[WinkアフィリエイトWordPressプラグイン](https://wordpress.org/plugins/wink2travel/)このプラグインを使用すると、Wink カード、グリッド、マップなどに簡単にアクセスでき、お気に入りのページ ビルダーやカスタム ショートコードを使用してコンテンツを投稿やページにドラッグするだけで簡単に操作できます。

## インストール

インストール方法は次のとおりです`Wink Affiliate WordPress Plugin`WordPress ウェブサイトに。

<Steps>
  1. 管理者としてログインし、ダッシュボードにいるとき。
  2. クリック `Plugins > Add New Plugin`。
  3. 検索する `Wink Affiliate WordPress Plugin`。
  4. クリック `Install` プラグインをダウンロードしてサイトにインストールします。
  5. クリック `Activate` サイトでプラグインを有効にします。
  6. アクティベーションを行うと、 `Wink Settings` ページ。 *または、次のリンクからアクセスすることもできます。 `Appearance > Customize > Wink Settings`。*
  7. あなたに尋ねます `Client ID` そして `Client Secret`. へ移動 [アプリケーション](/developers/applications) 資格情報を取得する方法についてはこちらをご覧ください。
  8. 資格情報を入力してクリック `Publish`これで、Wink アカウントに接続されました。
</Steps>

## ページビルダー

私たちは WordPress で最も人気のあるページ ビルダーをサポートしており、Wink を使用したい人にとっては簡単です。

### グーテンベルク

Gutenberg は WordPress のネイティブ エディターです。

### アバダ

[アバダ](https://avada.com)豊富な美しいテンプレートと、高度な機能セットを備えた強力なページ ビルダーで有名です。

### エレメンター

[エレメンター](https://elementor.com)は、\[現在]最も人気のあるページ ビルダーであり、無料版とプロ版の両方が用意されています。

### WPBakery

[WPBakery](https://wpbakery.com)市場初の高度なページ ビルダーでした。

## ショートコード

### \[ウィンクルックアップ]

ランク付けされたコンテンツグリッドと連携します。訪問したい場所を入力すると、在庫が表示されます。

### \[ウィンクサーチ]

旅程フォームを開くためのシンプルなボタン。

### \[ウィンクアカウント]

アカウントボタンをクリックすると認証できます。認証が完了すると、アカウントオプションのドロップダウンが表示されます。

### \[ウィンキティネラリー]

検索ボタンと同じですが、ボタンテキストとして旅程情報が含まれています。

### \[ウィンクコンテンツ]

Winkで選択した在庫をページまたは投稿に直接埋め込むことができます。このショートコードにはレイアウトIDも必要です。

