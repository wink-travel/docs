当社のREST APIはすべて言語に依存せず、Open APIに準拠しています。コードを1行も書かずにテストを開始できます。

Winkアカウントをまだ作成していない場合は、[ここから始めましょう](/user/register)。

## エンドポイントを消費する

消費エンドポイントは、既存の旅行在庫を見つけて予約したり、Wink アフィリエイト アカウントを通じて広告を掲載したりしたい開発者向けです。

### 構成API

予約エンジンのホワイトラベル + カスタマイズ情報を取得するための単一のエンドポイント。

<LinkButton href="https://docs.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
  もっと詳しく知る
</LinkButton>

  もっと詳しく知る
</LinkButton>

Analytics API は、トラックのリーダーボードランキングに関する洞察だけでなく、時系列、プラットフォーム レベルのデータに関する詳細な洞察を求めているすべてのユーザー向けにエンドポイントを公開し、さまざまなメトリックをホストします。

## 決済API

決済エンドポイントは、旅行在庫を購入したい開発者向けです。登録旅行代理店としてAPI経由で購入することも、PCI準拠の決済ウィジェットと組み合わせたAPIを使用することで、他のすべての事業者向けにもご利用いただけます。
決済APIは、アフィリエイトやホテルが予約状況、分析、資金の空き状況を追跡するためのエンドポイントを提供します。また、利用可能な資金を銀行口座に引き出すことも可能です。

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  もっと詳しく知る
</LinkButton>
