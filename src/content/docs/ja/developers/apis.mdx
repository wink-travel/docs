当社のすべてのREST APIは言語に依存せず、Open API準拠です。コードを一行も書かずにテストを開始できます。

まだWinkアカウントを作成していない場合は、[こちらから始めてください](/ja/user/register)。

## Ping API

Ping APIは、認証情報が正しく機能しているかを素早く確認するためのテストエンドポイントです。

<LinkButton href="https://developer.wink.travel/ping" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>


## Notifications API

Notifications APIは、ユーザー、物件、またはアフィリエイトアカウントと連絡を取り合うための手段です。

<LinkButton href="https://developer.wink.travel/notifications" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

## User Settings API

User Settings APIは、サードパーティの統合者がWinkと通信するためのエンドポイントを公開しています。

<LinkButton href="https://developer.wink.travel/user-settings" variant="secondary" icon="external" target="_blank">

## Managing Entity API

Managing Entity APIは、ユーザーがプラットフォーム上でアクセスできるエンティティを簡単に確認できるエンドポイントです。

## Consume endpoints

Consumeエンドポイントは、既存の旅行在庫を見つけて予約したり、Winkのアフィリエイトアカウントを通じて広告に利用したい開発者向けです。

### Configuration API

予約エンジンのホワイトラベルおよびカスタマイズ情報を取得する単一のエンドポイントです。

<LinkButton href="https://developer.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

### Lookup API

地理的地域を横断して特注の旅行在庫を検索するプログラマー向けの方法です。

<LinkButton href="https://developer.wink.travel/lookup" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

### Inventory API

Lookup APIで正式に特定された既知の旅行在庫を取得するプログラマー向けの方法です。

<LinkButton href="https://developer.wink.travel/inventory" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

### Booking API

当社プラットフォームで見つけた在庫を予約するプログラマー向けの方法です。

<LinkButton href="https://developer.wink.travel/booking" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

### Travel Agent API

Travel Agent APIは、代理店が仲介する予約を管理するためのエンドポイントを公開しています。

<LinkButton href="https://developer.wink.travel/travel-agent" variant="secondary" icon="external" target="_blank">

## Produce endpoints

Produceエンドポイントは、旅行在庫を作成および管理したい開発者向けです。

### Registration API

プロデューサーとして、多くの場合ここから旅が始まります。これらのエンドポイントはWink上に物件を作成することを可能にします。

<LinkButton href="https://developer.wink.travel/extranet/property/register" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

### Property API

この物件エンドポイントのコレクションは主に管理用で、既存の物件の表示、ステータス変更などを行えます。

<LinkButton href="https://developer.wink.travel/extranet/property" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

### Facilities API

Facilities APIは、Wink上の体験（例：部屋タイプ）を管理するためのエンドポイントを公開しています。

<LinkButton href="https://developer.wink.travel/extranet/facilities" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

### Experiences API

Experiences APIは、Wink上の体験（例：アクティビティ）を管理するためのエンドポイントを公開しています。

<LinkButton href="https://developer.wink.travel/extranet/experiences" variant="secondary" icon="external" target="_blank">

### Monetize API

### Property Booking API

## Taxonomy API

Taxonomyエンドポイントは、旅行在庫の消費および生成を行う開発者向けで、在庫タイプ、クラス、ステータスなどの標準および非標準コードの分類体系を提供します。

<LinkButton href="https://developer.wink.travel/reference" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

## Analytics API

Analytics APIは、リーダーボードランキングの追跡や時系列、プラットフォームレベルのデータに関する深い洞察を求めるすべての人に向けて、多様な指標をホストするエンドポイントを公開しています。

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>

## Payment API

Paymentエンドポイントは、旅行在庫を購入したい開発者向けです。これは登録済みのTravel AgentとしてAPI経由で行うか、その他のエンティティ向けにはPCI準拠の決済ウィジェットと連携してAPIを利用できます。
Payment APIは、アフィリエイトやホテルが予約、分析、資金の利用可能状況を追跡し、利用可能な資金を銀行口座に出金することも可能にします。

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
詳細を見る
</LinkButton>
