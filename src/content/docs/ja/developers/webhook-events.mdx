このページでは、Winkで購読可能なすべてのWebhookイベントを一覧表示しています。設定手順については、[Webhook統合ガイド](/ja/guides/webhook-integration)または[Webhook管理ページ](/ja/my-account/webhooks)をご覧ください。

## 購読方法

1. ポータルの**Applications > Webhooks**に移動します。
2. Webhookを作成し、受信したいイベントを選択します。
3. イベントが発生するたびにWinkがHTTP POSTをエンドポイントに送信します。

完全な設定手順とベストプラクティスについては、[Webhook統合](/ja/guides/webhook-integration)をご覧ください。

## アカウントイベント

TripPayアカウントのライフサイクルおよび設定に関連するイベント。

| イベント | キー | 説明 |
|---|---|---|
| アカウント作成 | `account.create` | 新しいアカウントが作成されました。 |
| アカウント審査中 | `account.under_review` | アカウントが審査待ちです。 |
| アカウント更新 | `account.updated` | アカウントの詳細が変更されました。 |
| アカウントステータス更新 | `account.status.updated` | アカウントのステータスが変更されました（例：承認、停止）。 |
| アカウント住所更新 | `account.address.updated` | アカウントの住所が変更されました。 |
| アカウント詳細更新 | `account.details.updated` | アカウントの詳細フィールドが変更されました。 |
| アカウント情報更新 | `account.information.update` | 一般的なアカウント情報が更新されました。 |
| アカウントマネージャー更新 | `account.managers.update` | アカウントマネージャーのリストが変更されました。 |
| プロフィール画像更新 | `account.profile_picture.updated` | アカウントのプロフィール画像が変更されました。 |
| エージェント更新 | `account.agent.updated` | アカウントのエージェント情報が変更されました。 |
| URL名更新 | `account.url-name.update` | アカウントのバニティURLが変更されました。 |
| ユニークID更新 | `account.unique-id.update` | アカウントのユニーク識別子が変更されました。 |

### アカウントタスク

| イベント | キー | 説明 |
|---|---|---|
| タスク作成 | `account.task.create` | 新しいタスクがアカウントに追加されました。 |
| タスク更新 | `account.task.update` | アカウントのタスクが変更されました。 |
| タスク削除 | `account.task.remove` | アカウントのタスクが削除されました。 |

| 税IDタスク完了 | `account.task.tax_id.complete` | 税ID確認タスクが完了しました。 |

## 予約契約イベント

TripPayを通じた予約契約のライフサイクルを追跡するイベント。

| イベント | キー | 説明 |
|---|---|---|
| 初期化リクエスト | `booking.contract.initialize.request` | 予約契約の初期化がリクエストされました。 |
| 初期化完了 | `booking.contract.initialize.complete` | 予約契約の初期化が完了しました。 |
| 契約作成 | `booking.contract.created` | 新しい予約契約が作成されました。 |
| 契約処理中 | `booking.contract.processing` | 予約契約の支払いが処理中です。 |
| 契約失敗 | `booking.contract.failed` | 予約契約の支払いが失敗しました。 |
| 契約キャンセル | `booking.contract.cancelled` | 予約契約がキャンセルされました。 |
| 部分返金 | `booking.contract.refund.partial` | 契約に対して部分返金が行われました。 |
| 全額返金 | `booking.contract.refund.full` | 契約に対して全額返金が行われました。 |
| 返金リクエスト | `booking.contract.refund.request` | 返金リクエストが送信されました。 |
| 返金エラー | `booking.contract.refund.error` | 返金処理中にエラーが発生しました。 |
| 支払い作成 | `booking.contract.payout.create` | 契約に対する支払いが作成されました。 |
| 支払い完了 | `booking.contract.payout.complete` | 支払いが正常に完了しました。 |

:::
