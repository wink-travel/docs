公式にサポートされている言語のソフトウェア開発キットを使用して、マーケット投入までの時間を短縮しましょう。  
現在お使いの言語がサポートされていない場合は、使用したいAPIのページ上部にあるOpenAPI仕様をダウンロードしてください。公開されているSDKリポジトリをフォークするか、OpenAPIのジェネレーターを使って独自のSDKを生成できます。

## Java

このライブラリはJava 24向けにコンパイルされています。リアクティブなWebClientを使用するように設定されており、Spring Frameworkでの利用に最適化されています。ただし、Springは必須ではありませんが、OAuth2サポートが標準で利用可能になります。
