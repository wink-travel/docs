スパでは、施設内外でスパを紹介・販売できます。スパを作成するには、`Create spa` ボタンをクリックしてください。

以下のセクションでは、スパ管理の各ステップを順に説明します。

## 設定

開始時にデフォルトで選択されているのは `Configurations` タブです。

このセクションでは、スパの基本情報を設定します。

- **内部名** スパを内部で呼ぶ名前。*例：Santika Spa*

## スケジュール

このセクションでは、スパの利用可能時間を管理します。

### 営業時間

- **開始時間** スパが利用可能になる時間。*例：09:00*
- **終了時間** スパが利用不可になる時間。*例：21:00*

### 営業日

このセクションでは、スパが利用可能な曜日を設定します。

*スイッチを✅に切り替えて、その曜日が利用可能であることを示します。*

### 季節設定

:::note
ネイティブの「日/月」形式の入力フィールドはありません。この場合、日付セレクターから月と日だけを使用します。
:::

このセクションでは、年間を通じてスパが利用可能な日付を設定します。

- **開始日** スパが利用可能になる日付。*例：2024年9月11日*
- **終了日** スパが利用不可になる日付。*例：2024年7月11日*

## アメニティ

:::note
このセクションは任意です。
:::

`Amenities` タブをクリックして続行してください。

施設のアメニティと同様に、スパのアメニティは特定のスパにのみ適用されます。該当するアメニティにチェックを入れてください。

## 説明

:::note
少なくとも1つの説明文を英語で作成する必要があります。
:::

`Descriptions` タブをクリックして続行してください。

施設のウェルカムテキストと同様に、スパの説明文も多言語で作成できます。
人気のある言語すべての翻訳を追加します。
これは旅行者がスパを見る際に表示されるテキストです。

- **名前** スパの名前を付けてください。*例：Santika Spa*
- **説明** スパについて1〜2段落で説明してください。*例：これは世界で最高のスパです...*

## ソーシャルメディア

:::note
このセクションは任意です。
:::

`Social media` タブをクリックして続行してください。

「なぜスパに独自のソーシャルメディアセクションがあるのか？」と思うかもしれません。スパは施設の近くにありながら敷地外にあり、独自のスタッフとInstagramプロフィールを持つこともあります。

ソーシャルメディアは、新規および既存の旅行者と継続的に繋がる手段です。
プロフィールにソーシャルネットワークアカウントを追加して、ゲストが簡単に連絡を取り、このスパについて知ることができるようにしましょう。

<Steps>
1. 有効にしたいソーシャルネットワークのスイッチを切り替えます。
2. そのネットワーク上のアカウント名を入力します。
</Steps>

## 評判

:::note
このセクションは任意です。
:::

`Reputation` タブをクリックして続行してください。

`Social media` タブと同様に、このスパに別の評判スコアや特別な称賛がある場合にここで強調表示できます。

評判はあなたのデジタル通貨であり、どこへ行っても持ち運びたいものです。
評判エントリーは、第三者の評価やシェフが金賞を受賞したコンテストなど、施設が誇りに思うものなら何でも構いません。
このセクションでは、インターネット上のあらゆるスコアやランキングを追加できます。

`Add reputation score` ボタンをクリックしてください。

<Steps>
1. **カテゴリ** 追加したいスコアの種類を選択します。*例：第三者レビュー*
2. **提供者** スコアを授与した団体名を入力します。*例：BigOTA.com*
3. **評価タイプ** スコアの形式を選択します。*例：数値*
4. **日付** 任意でスコアを受け取った日付を入力します。
5. **評価** 受け取ったスコアを入力します。*例：8.5*
6. **最大評価** 任意でこの賞の最大スコアを入力します。*例：10*
</Steps>

## 属性ベース販売

:::note
このセクションは任意です。
:::

`Attribute Based Selling` タブをクリックして続行してください。

すべての施設には独自のABSセクションがあります。このセクションでは、メインの在庫にアップセル機会を作成したり、非取引型の在庫を取引型に変えたりできます。
予約プロセス中により高い価格帯で予約者とつながる可能性を高めるために、これらの機会を作成することを強くお勧めします。

:::tip
スパはABSを通じて販売可能にできます。販売方法はさまざまです：

- リフレクソロジー
- パンパーデイ
- 2人用パンパー
- 指圧スペシャル
- +++ 想像力を働かせて追加収入を得ましょう!!
:::

- **価格帯** 旅行者にこのスパの価格感を簡単に示します。
- **有効** このスパをすべての販売チャネルで予約不可にしたい場合はスイッチを🛑に切り替えます。
- **コミッション対象** アフィリエイトがこの在庫でコミッションを得るかどうかを示します。
- **注目** このスパをより目立つ位置に表示すべきかどうかを示します（該当オプションがある場合）。
- **属性ベース販売** この在庫でABSを有効にするにはスイッチを✅に切り替えます。*有効にしたら、少なくとも1つのABS商品を販売用に追加する必要があります*

### 付帯商品

ABSを有効にすると、付帯商品を追加して販売可能にできます。

*デフォルトで空の商品がすでに作成されています。`Add item for sale` ボタンをクリックしてさらに追加できます。*

- **名前** 付帯商品の名前を付けてください。*例：リフレクソロジー*
- **価格タイプ** 価格計算方法を示します。*例：`Per hour` に設定すると、レンタル時間数に基づく価格になります。*
- **基本価格** この商品の通常価格を入力します。*例：20 / 時間*
- **割引価格** Winkユーザーが予約時に得られる価格を入力します。*例：15 / 時間*

### 制限

数量の最小/最大制限を設定して、この商品の利用可能数を制限できます。

- **最小数量** 予約が必要な最小数量を制限します。*例：1名*
- **最大数量** 予約可能な最大数量を制限します。*例：4名*

### 説明

:::note
少なくとも1つの説明文を英語で作成する必要があります。
:::

- **名前** 付帯商品の名前を付けてください。*例：リフレクソロジー*
- **説明** 付帯商品について1〜2段落で説明してください。*例：プロのリフレクソロジストによる足マッサージの素晴らしさを体験してください。思い切り楽しんでください。*

## 近接情報

`Proximity` タブをクリックして続行してください。

すべての在庫が施設内にあるわけではありません。敷地外にある在庫がある場合は、ここで旅行者に知らせることができ、別の地図位置や任意の連絡先情報を設定できます。

*スパがホテルから数キロ離れている場合に、旅行者がタクシー運転手に行き方を説明するのに役立ちます。*

`Save` ボタンをクリックして続行してください。
