:::note
これは Wink Studio で利用可能な [共有可能なリンク](/ja/studio/shareable-links) 機能のサブセットです。

物件は Wink Studio に行って生成しなくても、自動的にすべての在庫が共有可能なリンクとして利用可能になります。
:::

共有可能なリンクは Wink のコア機能であり、インターネット上で最もシンプルな方法、すなわちハイパーリンクを通じて、在庫と価格情報を動的に伝える必要性から生まれました。

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

上記は当社のリンクの一例です。短くて持ち運びやすく…覚えられるほど短いです。短いながらも、このリンクには**膨大な**情報が詰まっています。例えば：

- あなたのアプリID。
- あなたのアフィリエイトID。
- ユーザーに表示する画像とテキスト。
- 見た目のスタイル（*テーマ、色など*）。
- 価格照会に使う旅程（*大人2名、1泊*）。
- 価格表示に使う通貨。
- テキスト表示に使う言語。

この記事の残りでは、在庫へのリンクの管理方法を説明します。

## リンクのカスタマイズ

フォームでは以下の方法でリンクをカスタマイズできます：

<Steps>
 1. デフォルトでは、リンクには英語の在庫名が付けられます。リンクの内容を思い出せる任意の名前を付けられます。
 2. デフォルトで、プライマリーブランディング／カスタマイズがリンクに適用されます。Wink Studio で生成した他の[カスタマイズ](/ja/studio/customization)を選択できます。
 3. カンマで区切ったキーワードを追加できます。これは[ウェブクローラー](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)に使用されます。
 4. 任意で X アカウントのハンドル名や Facebook アプリIDを入力できます。
 5. ユーザーに見せたい言語でタイトルと説明を追加します。
 6. Facebook や LinkedIn などのプラットフォームでリンクを共有したときに表示される画像を1枚以上選択します。
 7. 選択した画像の上にブランドデザインを追加するテーマ／オーバーレイを選びます。*注：これにより画像の上に価格が表示されます。*
 8. `保存` ボタンをクリックして続行します。
 </Steps>

上の画像はリンクに利用可能なすべてのアクションを示しています：

- **プレビュー** リンクの設定内容とユーザーにどのように見えるかを簡単に確認できます。
- **更新** リンクの設定を更新します。
- **WinkLinks に追加** リンクをあなたの WinkLinks アカウントに追加します。
- **コピー** リンクをクリップボードにコピーし、どこでも簡単に共有できます（*[WinkLinks](/ja/link-manager/wink-links) も含む*）。
- **共有** 新しいウィンドウが開き、多くのソーシャルネットワークやメッセージングプラットフォームで共有するオプションが表示されます。
- **QRコード** IGなど画像中心のプラットフォームで共有できるQRコードを生成します。
- **埋め込み** このリンクを[広告バナー](/ja/developers/web-components/#content-loader)としてウェブサイトに埋め込む方法を表示します。
- **WordPressで使用** 当社の[WordPressプラグイン](/ja/developers/wordpress)を使ってこのリンクをウェブサイトに埋め込む方法を表示します。

以下でこれらのオプションのいくつかを詳しく説明します。

アイコンボタンをクリックして、そのネットワークでリンクの投稿を続けてください。*注：一部のボタンはモバイルデバイスから共有するときのみ表示されます。*

リンクの代わりにQRコードで共有したい場合は、`QRコード` ボタンをクリックすると、新しいウィンドウにQRコードが表示されます。

QRコードの保存方法：

- ノートパソコンの場合は、画像を右クリックして「画像を保存」を選択します。
- 携帯電話の場合は：
    - 画像の上で指を長押しし、画像を写真に保存するオプションが表示されるまで待ちます。
    - QRコードのスクリーンショットを撮り、トリミングして写真に保存します。

### 埋め込み

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

以下は、広告バナーとしてリンクをサイトに埋め込む方法です：

- 3行目は Wink のスタイルをサイトに埋め込む方法を示しています。
- 6〜9行目は [wink-content-loader](/ja/developers/web-components/#content-loader) Webコンポーネントを使い、広告バナーをコードに基づいて取得する方法を示しています。
- 11行目は当社のJavascriptをサイトに埋め込む方法を示しています。
- 13行目は [wink-app-loader](/ja/developers/web-components/#application-loader) Webコンポーネントを使い、ページレベルの設定を取得する方法を示しています。
