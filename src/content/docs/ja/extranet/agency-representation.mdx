代理店とは、代理の目的であなたのアカウントに割り当てることができる通常のWinkアカウントです。以下は代理店に代理してもらう理由の一例です。

- 第三者によるWinkアカウントの全管理または部分管理。
- 物件が既に管理会社と契約を結んでおり、Winkを通じて入金される資金の財務面をWinkに任せたい場合。
- 物件管理代理店が物件に代わってすべてのオンライン作業を行う場合。

## 代理店の割り当て

代理店をアカウントに割り当てる方法は以下の通りです。

<Steps>
 1. [Wink Extranet](https://extranet.wink.travel)にユーザーアカウントでログインします。
 2. 作業したい物件を選択します。
 3. メインナビバーの`Account > Managers`をクリックします。
 4. `Agency`セクションで、代理店として割り当てたい代理店名を入力します。
 5. ドロップダウンから代理店手数料を選択します。
 5. 必要に応じて有効期限ルールを追加します。
 7. `Set agency`ボタンをクリックして続行します。
 </Steps>

 :::note
代理店を割り当てるには物件の所有者である必要があります。
 :::

 :::tip[例]
代理店手数料は予約金額から算出されます。

予約金額 = $100、代理店手数料 = 1% => 100 * 0.01 = $1
 :::

## 代理店の解除

代理店をアカウントから解除する方法は以下の通りです。

<Steps>
 1. [Wink Extranet](https://extranet.wink.travel)にユーザーアカウントでログインします。
 2. 作業したい物件を選択します。
 3. メインナビバーの`Account > Managers`をクリックします。
 4. `Agency`セクションで、代理店ドロップダウンの右側にある<Badge text="x" variant="danger" size="small" />アイコンをクリックします。
 5. `Set agency`ボタンをクリックして続行します。
 </Steps>


## API

APIを使って代理店を管理したい開発者は、[Developers > API > Affiliate](/developers/apis/#affiliate-api)をご覧ください。
