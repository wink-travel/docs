代理店とは、代理業務を目的としてご自身のアカウントに割り当てることができる通常のWinkアカウントです。代理店に代理を依頼する理由としては、以下のようなものが挙げられます。

* サードパーティによる Wink アカウントの完全 / 部分的な管理。
* 不動産会社は管理会社と既存の契約を結んでおり、Wink を通じて入ってくる資金の手配の財務面を Wink に任せたいと考えています。
* 不動産管理会社は、その不動産に代わってすべてのオンライン作業を実行します。

## 代理店の割り当て

アカウントに代理店を割り当てる方法は次のとおりです。

<Steps>
  1. ユーザーアカウントにログイン [ウィンクエクストラネット](https://extranet.wink.travel)。
  2. 操作するプロパティを選択します。
  3. クリック `Account > Managers` メインナビゲーションバーから。
  4. の中で `Agency` セクションで、代理を依頼する代理店の名前を入力します。
  5. ドロップダウンから代理店手数料を選択します。
  6. 必要に応じて有効期限ルールを追加します。
  7. クリック `Set agency` 続行するにはボタンをクリックしてください。
</Steps>


:::注
代理店を指定するには、物件の所有者である必要があります。
:::

:::tip\[例]
代理店手数料は予約から発生します。

予約 = $100、代理店手数料 = 1% => 100 \* 0.01 = $1
:::

## 代理店の割り当てを解除

アカウントから代理店の割り当てを解除する方法は次のとおりです。

<Steps>
  1. ユーザーアカウントにログイン [ウィンクエクストラネット](https://extranet.wink.travel)。
  2. 操作するプロパティを選択します。
  3. クリック `Account > Managers` メインナビゲーションバーから。
  4. の中で `Agency` セクションで、 <Badge text="x" variant="danger" size="small" /> 代理店ドロップダウンの右側にあるアイコンをクリックします。
  5. クリック `Set agency` 続行するにはボタンをクリックしてください。
</Steps>


## API

API経由で代理店を管理したい開発者は、[開発者 > API > アフィリエイト](/developers/apis/#affiliate-api)。
