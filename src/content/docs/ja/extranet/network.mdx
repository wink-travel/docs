この記事では、Winkで積極的なアフィリエイトとつながり、既存のアフィリエイトを管理する方法を説明します。

## 販売チャネル

販売チャネルを管理するには、メインナビバーの `Distribution > Sales channels` をクリックしてください。

販売チャネルは、既存のアフィリエイト関係であり、彼らがアクセスできる在庫、取引、割引が体系化されています。

既存の販売チャネルを更新するには、更新したい販売チャネルの下にある `Actions` リンクをクリックしてください。

- **会員割引** この販売チャネルを通じて予約した旅行者に適用される会員料金です。
- **コミッション** 成功した予約ごとに販売チャネルに割り当てられるコミッションです。
- **単一プロモーション** この販売チャネルに適用したいプロモーションを追加します。
- **バンドルプロモーション** この販売チャネルに適用したいバンドルプロモーションを追加します。
- **有効** 有効スイッチを切り替えて、販売チャネルを完全に有効化または無効化します。
- **ブラックリスト** 販売チャネルをブラックリストに登録すると、その販売チャネルを通じての予約が不可能になります。

設定に満足したら、`Save` ボタンをクリックして続行してください。

:::tip
販売チャネルをブラックリストに登録すると、その販売チャネルはあなたからいかなる料金も受け取れなくなります。Wink Networkの料金であっても同様です。
:::

## ネットワークを探索する

Winkで新しいアフィリエイトを見つけるには、メインナビバーの `Distribution > Explore network` をクリックしてください。

このセクションでは、ページ上部にアフィリエイトからの保留中のリクエストが表示され、下部には接続可能なアフィリエイトのリストが表示されます。

### リクエストに応答する

アフィリエイトのリクエストを拒否することができます。リクエストを続行したい場合は、`affiliate details` ページに移動し、アフィリエイトの詳細情報やWinkでの過去のパフォーマンス指標を確認できます。

### アフィリエイトを検索する

アフィリエイトグリッドでは、名前、タイプ、場所でアフィリエイトを検索できます。気に入ったアフィリエイトが見つかったら、それをクリックすると `affiliate details` ページに移動します。

### アフィリエイトと接続する

アフィリエイトと直接関係を築くことを選択した場合、上記の販売チャネル管理と同様に関係を体系化するよう求められます。

新しい販売チャネルを保存すると、販売チャネルのリストに戻ります。

## API

`Network` を管理したい開発者は、[Developers > APIs > Distribution](/ja/developers/apis/#distribution-api) にアクセスしてください。
