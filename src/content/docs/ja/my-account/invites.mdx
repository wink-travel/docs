Winkユーザーは、自分の物件、アフィリエイト、または旅行代理店アカウントの管理をあなたに任せることができます。その場合、招待リクエストのメールが届きます。

import { Steps } from '@astrojs/starlight/components';

現在のWink招待状を見るには、以下の手順に従ってください：

<Steps>
 1. [my.wink.travel](https://my.wink.travel) にログインします。
 2. 左のサイドバーから `Invites` をクリックします。
 3. 現在のすべての招待状のリストが表示されます。
 4. このセクションにはあなたのすべての申請が表示されます。
 5. 招待を `Accept` または `Reject` から選択します。
 6. 招待を承諾した場合、アカウントは `Accounts` セクションに表示されます。
</Steps>
