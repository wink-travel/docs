Winkユーザーアカウントのユーザープロフィールを更新したい場合は、以下の手順に従ってください。

<Steps>
1. [my.wink.travel](https://my.wink.travel) にアクセスします。
2. 右上の `Sign-in / Register` をクリックしてログインします。
3. ログイン後、右上のプロフィールアイコンをクリックするとメニューが表示されます。
4. `Profile` リンクをクリックします。
5. `e-mail`、`first`、`last name` を更新する指示に従います。
6. 新しい `profile picture` をアップロードすることもできます。
7. `Save` をクリックします。
</Steps>

<Steps>
1. 当社のいずれかのウェブサイトにログインしている必要があります。
2. 右上のプロフィールアイコンをクリックするとメニューが表示されます。
3. `Profile` リンクをクリックします。
4. `e-mail`、`first`、`last name` を更新する指示に従います。
5. 新しい `profile picture` をアップロードすることもできます。
6. `Update` をクリックします。
</Steps>

:::note
新しいプロフィール写真は、Winkから完全にログアウトした次回に表示されます。
:::
