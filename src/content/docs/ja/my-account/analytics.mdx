知識はWinkの生命線です。Wink上のすべての人やすべてのものが、時間単位でどのようにパフォーマンスを発揮しているかを教えてくれます。Wink以前は、ホテルはサードパーティのプロバイダーに加入し、多額の費用を支払って、収益管理チームが「コンプセット」と呼ぶもの、つまり「自分の施設は他の施設と比べてどうか？」を知るためのデータにアクセスしていました。

Winkでは、プラットフォームレベルのアナリティクスに無料でアクセスできます。これにより、任意のアフィリエイト、旅行代理店、施設のパフォーマンスを、任意の指標で、好きなだけ追跡できます。また、地域ごとの広範なトレンドにアクセスして、どのタイプの製品がどこでいつ販売されているかを確認できます。適切な手にかかれば、この洞察はどのWinkユーザーの販売スキルも向上させることができます。

## チャートへのアクセス

当社のいずれかのウェブサイトからチャートにアクセスできます。分析チャートにアクセスするには、以下の手順に従ってください。

<Steps>
1. 当社のウェブサイトのいずれかにログインしている必要があります。
2. 右上のプロフィールアイコンをクリックするとメニューが表示されます。
3. `Analytics`リンクをクリックします。
4. チャートにリダイレクトされます。
</Steps>

:::note
デフォルトで、作成したすべての施設、旅行代理店、アフィリエイトアカウントごとにシンプルなチャートを作成しているため、自分のアカウントのパフォーマンスを簡単に追跡できます。
:::

## チャートの作成

独自のチャートを作成したい場合は、上記の手順でチャートに移動し、`Create chart`ボタンをクリックしてください。

以下はアナリティクスチャートを作成する簡単な手順で、その後に詳細な説明があります。

<Steps>
1. **時系列** チャートの時系列を選択します。*例：日次*。
2. **追跡タイプ** 固定期間を追跡するか連続的に追跡するかを選択します。*例：`Continuous`*。
    - 連続：時間単位を選択します。*例：7日間*
    - 固定：日付/時間範囲を入力します。*例：2023年9月22日16:00 - 2023年9月29日16:00*。
3. **表示通貨** 金額を扱う場合、表示したい通貨を選択します。*例：USD*。
4. **施設の追跡** 特定の施設を追跡します。*任意*
5. **販売チャネルの追跡** 大陸、国、都市ごとに販売チャネルの場所を追跡します。*任意*
6. **販売チャネルタイプのフィルター** 特定の販売チャネルタイプでフィルターします。*例：インフルエンサー*。*任意*
7. **特定の販売チャネルの追跡** 特定の販売チャネルを追跡します。*任意*
8. **旅行者の追跡** 大陸、国、都市ごとに旅行者の場所を追跡します。*任意*
9. **データポイントの選択** 追跡したいデータポイントを選択します。*例：予約＆キャンセル*
10. **データのグループ化** 施設や販売チャネルでフィルターする代わりに、データポイントで集計するためにグループ化できます。*例：販売チャネルでグループ化*
11. **チャート名** チャートに説明的な名前を付けます。*例：Travelbugの時間別予約数*
12. `Save`ボタンをクリックします。
</Steps>

## 仕組み

チャートは4つのステップで処理されます。以下に説明します。

### 時系列タイプ

固定期間または連続時間スライスを選択することで、いつのデータを見たいかに焦点を当て始めます。

### 追跡 / フィルタリング

施設、販売チャネルなどを選択することで、誰のデータを見たいかに焦点を当て始めます。不要なデータは効果的に除外されます。

### データポイント

データポイントを選択することで、何のデータを見たいかに焦点を当て始めます。

:::note
データポイントは、グラフ上で一緒に適合するように「相乗効果的」であるべきです。例えば、「ページビュー」と「総純収益」を同じチャートに含めることはありません。
:::

### グループ化 <Badge text="New" variant="note" />

グループ化は、データを集計し、焦点を絞る強力な方法です。

:::tip
例えば、施設として各販売チャネルがどれだけ予約を獲得したかを追跡したい場合、上記で施設をフィルターし、上記で「予約」を追跡し、グループ化セクションで「販売チャネル」でグループ化を選択します。
:::

## チャートの削除

チャートを削除するには、以下の手順に従ってください。

<Steps>
1. 各チャートの下にある`Actions`リンクをクリックします。
2. `Remove`ボタンをクリックします。
</Steps>

チャートは削除されました。

## API

アナリティクスチャートを管理したい開発者は、[Developers > API > Analytics](/ja/developers/apis/#analytics-api)にアクセスしてください。
