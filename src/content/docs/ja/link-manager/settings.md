WinkLinksの設定を管理するには、メインナビゲーションバーの`WinkLinks`に移動し、`Settings`タブをクリックしてください。

`Settings`でWinkLinksの動作を設定する方法は以下の通りです：

- **インテリジェント** インテリジェンスを有効にすると、`Wink`、`Spotify`、`Instagram`、`Facebook`、`YouTube`、`X (旧Twitter)`などのサイトから利用可能な`リッチ`コンテンツを取得しようとします。*例：IGの投稿をWinkLinksで共有すると、IGで見るのと同じように、いいねやコメント、投稿とやり取りする方法が表示されます。*
- **初期表示** あなたのサイトを訪れた人が最初にどのようにコンテンツと対話し、表示されるかを制御します。以下の3つのオプションがあります：
    - `カラムベース`デザイン。これは推奨デザインで、あまり多くのコンテンツを表示しない場合はページ上で美しく見えます。ただし、多くのリッチコンテンツを共有すると`重く`なることがあります。
    - `行ベース`デザイン。ブラウザでの負荷を少し軽減する方法の一つですが、カードの整列を良くするためにテキスト量が似ているコンテンツを推奨します。
    - `リストベース`デザイン。ほとんどの他のリンクマネージャーがURLと最小限のコンテンツでシンプルなリストを表示します。大量のコンテンツをできるだけ速く表示したい場合はこちらを選択してください。
- **プロフィール写真** WinkLinksページの上部にはアカウントのプロフィール写真と説明が表示されます。プロフィール写真は以下の2つの方法で表示できます：
    - `円形` 自分自身の写真や丸いコンテナに収まるものには円形スタイルを推奨します。
    - `長方形` 会社のロゴがプロフィール写真の場合は、ほとんどの会社ロゴが長方形であるため長方形スタイルを推奨します。
- **カスタマイズ** WinkLinksページの見た目をパーソナライズできます。ブランドに完璧にマッチするようにカラーパレットを自由にカスタマイズ可能です。この設定はWinkのインベントリーカード、マップ、グリッドの表示にも影響します。  
  
追加のコントロールには以下が含まれます：  

- 投稿表示の動作（バナーモード、スマートバナー/インタラクティブ、レイアウト対応の自動調整）  
- カスタムブランドQRコード（形状、色、ドット、角）  
- タグによるコンテンツ整理（「ホテル」「旅行」「セール」などのリンクを分類）  
- リンクの配置、ファイルアップロード、コンテンツブロックの完全管理

## API

`WinkLinks`を管理したい開発者は、[Developers > API > WinkLinks](/ja/developers/apis/#winklinks-api)にアクセスしてください。
