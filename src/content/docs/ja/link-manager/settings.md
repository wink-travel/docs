WinkLinksの設定を管理するには、メインナビゲーションバーの`WinkLinks`に移動し、`Settings`タブをクリックしてください。

`Settings`でWinkLinksの動作を設定する方法は以下の通りです：

- **インテリジェント** インテリジェンスを有効にすると、`Wink`、`Spotify`、`Instagram`、`Facebook`、`YouTube`、`X (formerly Twitter)`などのサイトから利用可能な`リッチ`コンテンツを取得しようとします。*例：IGの投稿をWinkLinksで共有すると、その投稿はIGで見えるように、いいねやコメント、投稿とやり取りする方法と共に表示されます。*
- **初期表示** あなたのサイトを訪れた人が最初にどのようにコンテンツと対話し、表示されるかを制御します。以下の3つのオプションがあります：
    - `カラムベース`デザイン。これは推奨デザインで、あまり多くのコンテンツを表示しない場合はページ上で見栄えが良くなります。ただし、多くのリッチコンテンツを共有すると`重く`なることがあります。
    - `行ベース`デザイン。これはブラウザでの読み込みを少し軽くする方法の一つですが、カードの整列を良くするためにテキスト量が似ているコンテンツを推奨します。
    - `リストベース`デザイン。ほとんどの他のリンクマネージャーはURLと最小限のコンテンツを持つシンプルなリストを表示します。大量のコンテンツがあり、できるだけ速く表示したい場合はこちらを選択してください。
- **プロフィール写真** WinkLinksページの上部にはアカウントのプロフィール写真と説明が表示されます。プロフィール写真は以下の2つの方法で表示できます：
    - `円形` 自分自身の写真や丸いコンテナに収まるものには円形スタイルを推奨します。
    - `長方形` 会社のロゴがプロフィール写真の場合は、ほとんどの会社ロゴが長方形であるため長方形スタイルを推奨します。
- **カスタマイズ** WinkLinksページで使用するカスタマイズを選択します。これはテーマカラーやWinkの在庫カード、地図、グリッドを表示する際の旅程に影響します。

## API

`WinkLinks`を管理したい開発者は、[Developers > API > WinkLinks](/ja/developers/apis/#winklinks-api)にアクセスしてください。
