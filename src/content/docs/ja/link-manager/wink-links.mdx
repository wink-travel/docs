リンクマネージャーは、後で使いたいお気に入りのURLを保存・整理できるツールです。ブラウザのブックマークはリンクマネージャーの一例です。ソーシャルメディアの登場により、友人やオーディエンスにリンクを公開しやすくするニーズが高まりました。Facebookのフィードはソーシャルに適したリンクマネージャーの例です。

私たちは旅行業界向けのリンクマネージャーとして[WinkLinks](https://i.trvl.as/)を開発しました。情報提供、興奮、広告を目的とし、世界中のホテルの素晴らしいお得なプランを予約できるようにします。

## 機能

- WinkLinksでは、Commission JunctionやAmazonなど既存のアフィリエイトリンクを無料で追加できます。
- 例えば[https://i.trvl.as/bob](https://i.trvl.as/bob)のようなかっこいいバニティURLがもらえ、1つのURLだけを共有したい時に便利です。
- リンクの整理・共有やファイルのアップロードができます。
- ホテル、チェーン、ブランドはWinkLinksを高度な予約エンジンとして利用可能：
    - 複数の施設に対応。
    - 複数の目的地に対応。
    - 複数通貨に対応。
    - 複数言語に対応。
- WinkLinksは`リッチコンテンツ`*(下記参照)*に対応しています。
- IGからわずか2クリックで予約可能 🚀

## リッチコンテンツ

:::note
この機能を使うには、[設定](/ja/link-manager/settings)で`Intelligent`を🟢にしてください。
:::

`リッチ` / `oEmbed`コンテンツに対応したサイトのリンクを共有すると、WinkLinks上でリンクが動的に表示されます。

対応サイト：

- Facebook - *投稿に反応し動画を再生可能*。
- Instagram - *投稿に反応し動画を再生可能*。
- YouTube - *動画を再生可能*。
- X（旧Twitter） - *投稿に反応可能*。
- Spotify - *お気に入りのプレイリストを再生可能*。
- Wink - *`Advanced`を参照*。

## 高度な機能

WinkLinksは、[Wink Studio](https://studio.wink.travel)を通じて見つけた旅行在庫をオーディエンスに公開し、広告するのにも最適です。

WinkLinksを使って販売できるもの：

- [共有可能リンク](/ja/studio/shareable-links) - 表示方法を選択可能：
    - [広告バナー](/ja/developers/web-components/#content-loader)。
    - リアルタイム価格更新付きの通常リンク。
- [カード](/ja/studio/cards) - リアルタイム価格・空室情報付きの予約可能なカード。
- [グリッド](/ja/studio/grids) - 予約可能なカードを並べたグリッド。
- [マップ](/ja/studio/maps) - 予約可能なカードを開くマーカー付きマップ。
