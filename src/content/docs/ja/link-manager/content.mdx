あなたのコンテンツは、`Entries`タブ。

私たちは主に 3 種類のコンテンツをサポートしています。

* Web ページにリンクする公開 URL。
* WinkLinks を通じてアップロードされたファイル。
* 弊社を通じて作成されたテキストコンテンツ`Quotes` 特徴。

## コンテンツを追加する

WinkLinksにコンテンツを追加する方法は次のとおりです。`Entries`タブをクリックして開始します`Add entry` ボタン。

### 通常のリンク

:::ヒント
リンクはウェブページへのリンクである必要があります。例えば、PDFに直接リンクすることはできません。PDFを共有する場合は、`Upload file` 特徴。
：：：

通常のリンクを追加する手順は次のとおりです。

<Steps>
  1. 共有したいリンクを入力してください。 *例: https://trvl.as/s/3xWCH*
  2. リンクを入力すると、Wink はこの Web サイトの詳細を検索し、次に表示されるフォームに入力します。
  3. サイト名、タイトル、説明など、一部のデータの変更を選択できます。
  4. カスタム画像をアップロードすることもできます。
  5. クリック `Save` 続行する準備ができたら、ボタンをクリックします。
</Steps>


### ファイルをアップロード

:::注意\[要件]

* 画像のサイズは10Mbを超えることはできません。
* ビデオのサイズは 50 MB を超えることはできません。
* PDF を含む、認識されているすべての画像およびビデオ形式をサポートしています。
  :::

:::note\[おすすめ]

* 注目の画像は幅 2560 x 高さ 1440 (アスペクト比 16x9) です。
* その他の画像の場合は、幅 2560 x 高さ 1920 (アスペクト比 4:3)。
* 画像サイズは5MB以下にしてください。Winkにアップロードする前に画像を圧縮してください。
  :::

ファイルを追加する手順は次のとおりです。

<Steps>
  1. 追加できるメディア:
     * コンピュータから画像をウィンドウにドラッグするか、クリックして `Browse`。
     * 画像を指す外部 URL を貼り付けます。
     * ノートパソコンのカメラを使用します。
     * Google ドライブ アカウントから。
     * Dropbox アカウントから。
     * Shutterstockより
     * ゲッティイメージズより
     * iStockより
     * Unsplashより
  2. ファイルのアップロードが完了したら、タイトルや説明など、ファイルを説明するメタデータを追加できます。
  3. クリック `Save` 続行する準備ができたら、ボタンをクリックします。
</Steps>


### 引用を書く

:::ヒント
引用符は、長いリストを区切るのに最適な方法です \[...そして、賢くて知識豊富な印象を与えます 😉]。
:::

見積書を書く手順は次のとおりです。

<Steps>
  1. 引用文を入力してください *例えば、バラは赤いです。スミレは青いです。*。
  2. 原作者の名前を入力してください。 *例：エドマンド・スペンサー*
  3. 引用元のサイトの名前を入力します。 *例：日々の名言*
  4. クリック `Save` 続行する準備ができたら、ボタンをクリックします。
</Steps>


## コンテンツの更新

すべてのコンテンツは、`Actions`リンクと選択`Update`同様の方法でコンテンツを削除するには、`Remove` ボタン。

## コンテンツを削除する

すべてのコンテンツは、選択して簡単に変更できます。`Update`同様の方法でコンテンツを削除するには、まず`Actions`リンクをクリックして`Remove` ボタン。

## コンテンツの並べ替え

各エントリの左下隅にあるアイコンをクリックし、希望の位置にドラッグすることで、コンテンツを並べ替えることができます。

## コンテンツのグループ化

:::ヒント
エントリーを作成する前にタグを作成してください。そうしないと、これまでの作業が失われ、最初からやり直す必要があります。
:::

エントリを編集する際に、グループ化のために1つ以上のタグを追加することができます。フォームの一番下までスクロールすると、`Tags`フォームフィールド。

ユーザーは、WinkLinks ページを閲覧しながら、コンテンツの利用方法 (リストまたはグリッド) を選択でき、1 つ以上のタグを選択してコンテンツをさらにフィルタリングできます。
