コンテンツは`Entries`タブで確認できます。

サポートしている主なコンテンツタイプは3つです：

- ウェブページにリンクする公開URL。
- WinkLinksを通じてアップロードされたファイル。
- `Quotes`機能で作成されたテキストコンテンツ。

## コンテンツの追加

WinkLinksにコンテンツを追加する方法をご紹介します。`Entries`タブの下で、まず`Add entry`ボタンをクリックしてください。

### 通常のリンク

:::tip
リンク先はウェブページである必要があります。例えば、PDFに直接リンクすることはできません。PDFを共有したい場合は、`Upload file`機能を使用してください。
:::

通常のリンクを追加する手順は以下の通りです：

<Steps>
1. 共有したいリンクを入力します。*例: https://trvl.as/3xWCH*
2. リンクを入力すると、Winkがそのウェブサイトの詳細を取得し、次に表示されるフォームに自動で反映します。
3. サイト名、タイトル、説明などのデータを編集することも可能です。
4. カスタム画像をアップロードすることもできます。
5. 準備ができたら`Save`ボタンをクリックして続行します。
</Steps>

### ファイルのアップロード

:::caution[要件]
- 画像のサイズは10MBを超えてはいけません。
- 動画のサイズは50MBを超えてはいけません。
- PDFを含むすべての認識された画像および動画フォーマットをサポートしています。
:::

:::note[推奨事項]
- フィーチャー画像は2560w x 1440h（16:9のアスペクト比）を推奨します。
- その他の画像は2560w x 1920h（4:3のアスペクト比）を推奨します。
- 画像サイズは5MBを超えないようにしてください。アップロード前に画像を圧縮することをおすすめします。
:::

ファイルを追加する手順は以下の通りです：

<Steps>
1. メディアは以下の方法で追加できます：
    - コンピューターから画像をウィンドウにドラッグするか、`Browse`をクリック。
    - 画像を指す外部URLを貼り付ける。
    - ノートパソコンのカメラを使用する。
    - Google Driveアカウントから。
    - Dropboxアカウントから。
    - Shutterstockから。
    - gettyimagesから。
    - iStockから。
    - Unsplashから。
2. ファイルのアップロードが完了したら、タイトルや説明などのメタデータを追加できます。
3. 準備ができたら`Save`ボタンをクリックして続行します。
</Steps>

### 引用文を書く

:::tip
引用文は長いリストを区切るのに最適です[...そして賢く知識豊富に見せることもできます😉]。
:::

引用文を書く手順は以下の通りです：

<Steps>
1. 引用文を入力します。*例: Roses are red. Violets are blue...*。
2. 元の著者名を入力します。*例: Edmund Spenser*。
3. 引用元のサイト名を入力します。*例: Daily Quotes*。
4. 準備ができたら`Save`ボタンをクリックして続行します。
</Steps>

## コンテンツの更新

すべてのコンテンツは`Actions`リンクをクリックし、`Update`を選択することで簡単に編集できます。コンテンツの削除も同様に`Remove`ボタンをクリックして行えます。

## コンテンツの削除

すべてのコンテンツは`Update`を選択して簡単に編集できます。削除する場合は、まず`Actions`リンクをクリックし、その後`Remove`ボタンをクリックしてください。

## コンテンツの並べ替え

各エントリーの左下にあるアイコンをクリックしてドラッグすることで、コンテンツの順序を自由に変更できます。

## コンテンツのグループ化

:::tip
エントリーを作成する前にタグを作成してください。そうしないと進行状況が失われ、最初からやり直す必要があります。
:::

エントリー編集時に、グループ化のために1つ以上のタグを追加することができます。フォームの一番下にある`Tags`フィールドをスクロールして見つけてください。

WinkLinksページを閲覧するユーザーは、コンテンツの表示方法（リストまたはグリッド）を選択でき、さらに1つ以上のタグを選んでコンテンツを絞り込むことができます。

## API

`WinkLinks`を管理したい開発者は、[Developers > API > WinkLinks](/ja/developers/apis/#winklinks-api)にアクセスしてください。
