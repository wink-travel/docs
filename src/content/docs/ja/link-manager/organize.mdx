WinkLinksタグを管理するには、`WinkLinks`メインナビゲーションバーで`Tags`タブ。

コンテンツが増えていくにつれて、整理整頓が必要になります。そこで、`tags`この目的のために、コンテンツを必要なだけタグを使ってグループ化することができます。

## タグを作成

<Steps>
  1. をクリックして新しいタグを作成します `Add a new tag` ボタン。
  2. 新しいタグの名前を入力します。 *例：ベルリンのおすすめコーヒーショップ*
  3. クリック `Save` リンク。
</Steps>


## タグを更新

<Steps>
  1. まずはクリックして `Edit` 変更したいタグの横にあるリンクをクリックします。
  2. タグの名前を更新します。 *例：ベルリンの最高のコーヒーショップ*
  3. クリック `Save` リンク。
</Steps>


## タグを削除

:::注
タグを削除すると、関連するすべてのコンテンツからもタグが削除されます。
:::

<Steps>
  1. クリック `Remove` 削除したいタグの横にあるリンクをクリックします。
</Steps>


## API

管理したい開発者`WinkLinks`向かうことができます[開発者 > API > WinkLinks](/developers/apis/#winklinks-api)。
