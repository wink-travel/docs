WinkLinks のタグを管理するには、メインナビバーの `WinkLinks` に移動し、`Tags` タブをクリックしてください。

コンテンツが増えてくると、整理を始めたくなるでしょう。そのために `tags` を作成しました。タグはいくつでも使ってコンテンツをグループ化できます。

## タグを作成する

<Steps>
1. `Add a new tag` ボタンをクリックして新しいタグを作成します。
2. 新しいタグの名前を入力します。*例: ベルリンのトップコーヒーショップ*
3. `Save` リンクをクリックします。
</Steps>

## タグを更新する

<Steps>
1. 変更したいタグの横にある `Edit` リンクをクリックして開始します。
2. タグの名前を更新します。*例: ベルリンのベストコーヒーショップ*
3. `Save` リンクをクリックします。
</Steps>

## タグを削除する

:::note
タグを削除すると、そのタグが関連付けられているすべてのコンテンツからも削除されます。
:::

<Steps>
1. 削除したいタグの横にある `Remove` リンクをクリックします。
</Steps>

## API

`WinkLinks` を管理したい開発者は、[Developers > API > WinkLinks](/ja/developers/apis/#winklinks-api) にアクセスしてください。
