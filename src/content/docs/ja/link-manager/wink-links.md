リンクマネージャーは、後で使いたいお気に入りのURLを保存・整理するためのツールです。ブラウザのブックマークはリンクマネージャーの一例です。ソーシャルメディアの登場により、友人やオーディエンスにリンクを公開しやすくするニーズが高まりました。Facebookのフィードは、ソーシャルに適したリンクマネージャーの例です。

:::caution
ほとんどのリンクマネージャーは、アフィリエイトリンクを投稿する場合、サブスクリプションが必要です。 
:::

私たちは、旅行分野に特化し、オンラインで効果的に宣伝・販売できる完全無料のリンクマネージャーとして、[WinkLinks](https://i.trvl.as/) を開発しました。

## 機能

- WinkLinksでは、Commission JunctionやAmazonなど、既存のアフィリエイトリンクをすべて無料で追加できます。
- 共有できるURLが1つだけの場合に便利な、[https://i.trvl.as/bob](https://i.trvl.as/bob) のようなかっこいいバニティURLがもらえます。
- リンクの整理や共有、ファイルのアップロードが可能です。
- ホテル、チェーン、ブランドはWinkLinksを高度な予約エンジンとして利用できます：
    - 複数の施設に対応。
    - 複数の目的地に対応。
    - 複数の通貨に対応。
    - 複数の言語に対応。
- WinkLinksは`リッチコンテンツ` *(以下参照)* に対応しています。
- IGから予約までわずか2クリックで完了 🚀

## リッチコンテンツ

:::note
この機能を使うには、[設定](/ja/link-manager/settings)で`Intelligent`を🟢にしてください。
:::

`リッチ` / `oEmbed`コンテンツに対応したサイトのリンクを共有すると、WinkLinks上でリンクが動的に表示されます。

対応サイト：

- Facebook - *投稿に反応し、動画を再生*。
- Instagram - *投稿に反応し、動画を再生*。
- YouTube - *動画を再生*。
- X（旧Twitter） - *投稿に反応*。
- Spotify - *お気に入りのプレイリストを再生*。
- Wink - *`Advanced`を参照*。

## 高度な機能

WinkLinksは、[Wink Studio](https://studio.wink.travel)を通じて見つけた旅行在庫をオーディエンスに公開し、販売するのにも最適です。

WinkLinksを使って販売できるもの：

- [共有可能なリンク](/ja/studio/shareable-links) - 表示方法を選択可能：
    - [広告バナー](/ja/developers/web-components/#content-loader)。
    - リアルタイム価格更新付きの通常リンク。
- [カード](/ja/studio/cards) - リアルタイム価格・空室状況付きの予約可能なカード。
- [グリッド](/ja/studio/grids) - 予約可能なカードを並べたグリッド。
- [マップ](/ja/studio/maps) - 予約可能なカードを開くマーカー付きマップ。

:::note
WinkLinksアカウントの管理は、[Wink Studio](https://studio.wink.travel)のアフィリエイトアカウントにログインして行ってください。 
:::
