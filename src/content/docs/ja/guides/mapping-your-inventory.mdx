Extranetで施設に在庫（客室、レストラン、スパ、会議室、アクティビティなど）を追加すると、その在庫はWinkエコシステム全体で利用可能になります。このガイドでは、その仕組みを説明します。

## 在庫の流れ

```
Extranet (あなたが作成) → 予約エンジン (旅行者が閲覧) → Studio (アフィリエイトが販売)
```

1. **あなたが在庫を作成** します — 客室タイプ、施設、体験などをExtranetで設定。
2. **旅行者がその在庫を閲覧** します — 共有リンク、カード、直接検索を通じて予約エンジンで。
3. **アフィリエイトが在庫を発見** し、Studioでカードやグリッドを作成して自分の顧客に販売します。

## 客室タイプ

客室タイプは在庫の核となります。各客室タイプには以下が必要です：

- 名前と説明。
- ベッド構成。
- 最大収容人数（大人と子供）。
- 客室固有のアメニティ。
- 写真。
- マスターレートを含む少なくとも1つの料金プラン。

料金が設定されていない場合、客室タイプは存在しますが予約はできません。

## 施設と体験

客室以外にも以下を登録できます：

| 在庫タイプ | 内容 | 例 |
| -- | -- | -- |
| **レストラン** | 施設内の飲食店 | 「ルーフトップバー＆グリル」 |
| **スパ** | ウェルネスサービス | 「セレニティスパ - 60分マッサージ」 |
| **会議室** | イベント・会議スペース | 「ボードルームA - 20名」 |
| **アクティビティ** | 施設内または近隣の体験 | 「ガイド付きシュノーケリングツアー」 |
| **観光地** | 近隣の注目スポット | 「国立博物館 - 500m」 |
| **場所** | 興味深いロケーション | 「地元の市場」 |
| **アドオン** | 客室と一緒に購入可能な追加サービス | 「空港送迎」 |

各施設タイプはExtranet内に専用のセクションがあります。アフィリエイトはこれらを個別に販売可能で、旅行者はスパのリスティングから施設を知ることもあります。

## アフィリエイトの在庫利用方法

Studioではアフィリエイトが以下を行えます：

- **施設や個別在庫の検索**。
- **カードの作成** — 客室、スパトリートメント、アクティビティなど単一の予約可能カード。
- **グリッドの作成** — 複数のカードをまとめて表示。
- **マップの作成** — 予約可能な在庫を示す位置情報マーカー。
- **リンクの共有** — 施設や特定アイテムへの直接URL。

在庫が充実し多様であればあるほど、アフィリエイトはより多くの方法で施設をプロモーションできます。

## ヒント

- **漏れなく登録** — 施設が提供するすべてをリストアップしましょう。各アイテムが予約の入口になります。
- **良い写真を使う** — すべての在庫アイテムに高品質な画像を最低1枚用意しましょう。
- **説明文を最新に保つ** — 提供内容の変更や季節ごとに説明を更新しましょう。
- **価格を設定する** — 価格がない在庫は閲覧はできても予約はできません。
