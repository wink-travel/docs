Wink を使用すると、宿泊施設は世界中のインフルエンサー、コンテンツ クリエイター、ブロガーを簡単に見つけてつながることができます。

**ウィンク前**

Wink プラットフォームが登場する前は、インフルエンサーとのつながりは次のような感じでした。

* 不動産業者は影響力のある人物を見つけるのに時間を費やすことになります。
* 彼らはインフルエンサーに無料の宿泊一泊（またはそれに類似したもの）を提供します。
* 宿泊施設は🤞🏻して予約数の増加を期待するでしょう。インフルエンサーの投稿が実際に予約につながったかどうかは分かりません。

**ウィンクの後**

Wink プラットフォームは、上記のプロセスを簡素化し、価値を高めます。

* プロパティは Wink でインフルエンサーを検索し、つながることができます。
* インフルエンサーは Wink で物件を検索したり、物件とつながったりすることもできます。
* 双方ともこのチャネルを通じて行われた売上を追跡できます。
* Wink は各当事者に支払います。

不動産オーナーとして、インフルエンサーが貴社を訪問した際に、何らかの特典を交渉することも可能です。
しかし、将来的には、オンライン旅行代理店のように彼らと提携していくことが不可欠です。
マイケル・ジョーダンが本格的に収益を上げ始めたのは、ナイキのエアジョーダンの発売によるロイヤリティ収入を得た時でした。
同様に、当社のインフルエンサーも利益の一部を受け取る権利があり、1つのリンクから今後何年にもわたって不労所得を得ることができます。これは真の価値があります！

## 仕組み

### ウィンクネットワーク

インフルエンサーを探し始める前に、一般公開されている在庫は誰でも販売できることを覚えておいてください。
これは、 [ウィンクネットワーク](/extranet/network) 販売チャネル。
アフィリエイトが販売する在庫を検索すると表示されます。 [ウィンクスタジオ](/studio/what-is-studio) 。
また、在庫を販売するアフィリエイトを追跡することもできます。 [ウィンクネットワーク](/extranet/network) 。

それで...足を運びたくない場合は、次のようにします:

<Steps>
  1. Wink であなたの物件の素敵なプロフィールを作成しましょう。
  2. ゲストへのアップセルに使用できる他の多くのものを用意します。
  3. 設定する [ウィンクネットワーク](/extranet/network) 販売する在庫を探しているアフィリエイトにとって魅力的な販売チャネルになります。 *デフォルトでは、すでにかなり魅力的だと考えています*。
</Steps>


:::tip\[プロのヒント]
Wink を OTA 関係とは全く逆の扱い方で扱ってください。私たちは**ない**家の最後の一室を割引価格で販売するお手伝いをいたします。
アフィリエイトがあなたの代わりにストーリーテリングを行い、オーディエンスにとって忘れられない冒険を創造するために必要な在庫を提供します。
:::

他とは一線を画し、新しい世代の旅行者にアピールするには、Winkの宿泊施設ランディングページを`digitized version`あなたの所有地とその周辺地域について考えてみましょう。
あなたが所有している、あるいはあなたが促進できるものの中で、あなたの地域で何を売ったり、貸したり、予約したり、交換したり、共有したりできるでしょうか？今こそ、創造力を働かせて考えてみましょう。**枠にとらわれない**。

### 探検

このセクションは [エクストラネット > ネットワーク](/extranet/network/#explore-network) 。

このページでは、プラットフォーム上にすでに存在するさまざまな種類のアフィリエイトを検索し、それらの簡単な説明と過去の実績を確認できます。

### 招待する

すでにインフルエンサーと連携している場合**または**有名人と一緒に働きたい**そして**ROIを追跡したいと考えている方は、Winkでアカウントを作成するよう直接お伝えいただくか、弊社までご連絡いただければ招待状をお送りします。

### 直接行く

場合によっては、当社の関連会社と直接関係を築きたいこともあります。

これをするのは...

* 独占在庫を提供します。
* 彼らにユニークなコミッションを与えてください。
* 会員限定の特別料金を設定しましょう。
* ユニークなプロモーションを提供してください。

:::tip\[プロのヒント]
上記の理由以外に直接接続を作成する理由はありません。
:::

新しい直接販売チャネルを作成すると、在庫は [ウィンクスタジオ](/studio/what-is-studio) と`Direct ribbon`インベントリを重ね合わせます。
