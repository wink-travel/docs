その**今すぐ予約**「ボタン」とは、ホスピタリティ業界で広く使われている用語で、ホテルが、宿泊施設の空室状況と料金を把握し、旅行者が客室を予約できるプロバイダーと独自に連携することを指します。このボタンは、ホテルが自社ウェブサイトで、自社の条件で客室を販売できる最もシンプルな方法です。

*多くのホテルは、チャネルマネージャーが提供するバンドル予約エンジンを使用しています。特定の予約エンジンを導入する前に、ホテル側で調査を行い、ネイティブ予約エンジンを比較検討することをお勧めします。*

## 観客

この記事は主に、当社の無料かつ高コンバージョン率の予約エンジンを自社ウェブサイトに導入したいホテル向けです。
しかし、当社のテクノロジーを自社ウェブサイトで活用したいと考えている方にとっても、有益な情報となるでしょう。

## リンク経由

最もシンプルな統合には、[共有可能なリンク](/studio/shareable-links)。

これを行う手順は次のとおりです。

<Steps>
  1. https://extranet.wink.travel にアクセスしてください
  2. クリック `Sign-In / Register` 右上隅にある をクリックし、ユーザー アカウントでログインします。
  3. 左上のドロップダウンから操作するプロパティを選択します。
  4. クリック `Booking engine` ナビゲーション バーにあります。
  5. ページにリンクが表示されます。
  6. このリンクをウェブマスターに伝えて、 **今すぐ予約** ボタン。
  7. あなたは✅です。
</Steps>


### 高度な

リンクには、選択した場所から始まるURLに既に旅程が組み込まれています。`customization`以下の方法でリンクをさらにカスタマイズできます。

* 選択したカスタマイズを変更するか、
* 旅程を変更する

この記事では、旅行者がウェブサイトで旅程を選択し、それを Wink の予約エンジンに転送できる自由度が高まる後者に焦点を当てます。

独自の旅程を URL とともに渡す方法は次のとおりです。

有効なリクエストパラメータ:

* **sd**開始日を追加するには:*sd=2024-08-24*
* **n**泊数を追加するには:*1* または
* **エド**使用終了日:*ed=2024-08-25*
* **rc**ゲスト:*rc=a2*（大人：2名）
* **l**表示言語をリクエスト:*l=es*
* **c**表示通貨をリクエストする:*c=ユーロ*
* **p**プロモーションを追加する:*p=ABC123*

URL は次のようになります。

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2\&l=es\&c=EUR

より高度な使用例では、`+` サイン：

例：

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2-ca16q1-ca8q1+a2\&l=es\&c=EUR

* 部屋タイプ1＝大人：2名、子供：2名＝16歳+8歳。
* 部屋タイプ 2 = 大人: 2名。

### 例

サンプル**今すぐ予約**ボタンのHTML:

*このサンプルでは、[ブートストラップ](https://getbootstrap.com)主要な CSS ライブラリとして使用します。*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Book a room"
>
BOOK NOW
</a>
```

### その他の特典

当社のリンクをあなたの`book now` ボタン。

* **ソーシャルフレンドリー**リンクはソーシャルメディアで簡単に共有でき、`dynamic pricing`選択した画像の上に表示されます。
* **短い**URL は非常に共有しやすく、Instagram の Bio URL などの場所で使用できます。

## 埋め込み経由

ホテルによっては、ウェブサイトにもう少し機能を追加し、より深い統合を求めるところもあります。この例では、[カード](/studio/cards)彼らのウェブサイトに。

これを行う手順は次のとおりです。

<Steps>
  1. https://studio.wink.travel にアクセスしてください
  2. クリック `Sign-In / Register` 右上隅にある をクリックし、ユーザー アカウントでログインします。
  3. 左上隅から操作するプロパティを選択します。
  4. クリック `Search` メインナビゲーションバーから。
  5. 物件名を入力し、検索ボタンをクリックしてください。
  6. クリック `Actions` 検索結果カードのリンクをクリックして選択してください `Create > Make a Card`。
  7. これにより、 `Card`。
  8. 右上隅に、カードの準備ができたことを知らせるメッセージと、さらにカスタマイズするかどうかを知らせるメッセージが表示されます。
  9. クリック `Tools > Cards` メインナビゲーションバーから新しいカードが表示されます。
  10. クリックして `Actions` リンクカードのリンクをクリックして `Embed`。
  11. これにより、コピーできるサンプル HTML コードを含む新しいウィンドウが開きます。
  12. 取得するには `Client-ID`、へ [アプリケーション](/developers/applications)。
  13. 取得するには `Configuration-ID`、へ [カスタマイズ](/studio/customization)
  14. あなたは✅です。
</Steps>


埋め込みカードの HTML のサンプル:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*上記の HTML サンプルには、必要な Wink CSS と Javascript は含まれておらず、カードのみが含まれています。*
