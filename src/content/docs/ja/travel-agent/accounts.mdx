[ウィンクエージェント](https://agent.wink.travel)世界中の旅行代理店とつながることができます。この記事では、Winkで旅行代理店アカウントを作成し、管理する手順を説明します。

## アカウントを作成する

認証が完了すると、[アカウントページ](https://agent.wink.travel/company/list)旅行代理店のアカウントをすべて表示します。

クリック`Create new account`続行するにはボタンをクリックしてください。[アカウント作成](https://agent.wink.travel/company/create)ページ。

すべての必須フィールドに入力してください。

<Steps>
  1. **名前** アカウントの名前。 *これはあなたの本名である必要はありませんが、ユーザーがあなたの WinkLinks ページを閲覧するときやサプライヤーがあなたと直接連絡を取りたいときに表示されるものになります。*
  2. **エンティティタイプ** 企業であっても個人であっても構いません。 *主に、資金を受け取る資格があるかどうかを決定するために使用されます。*
  3. **タイプ** アカウントに最適なタイプを選択してください。 *例：インフルエンサー*
  4. **説明** アカウントの説明を記入してください。 *これは WinkLinks ページやサプライヤーが直接あなたと連絡を取りたいときに表示されます。*
  5. **Webサイト** メインのウェブサイトを入力します。 *IGアカウントのURLでも、会社のウェブサイトでも構いません。オンラインでの作業のほとんどをここで行うようにしましょう。サプライヤーは、あなたとのつながりを決める際に、これらのリンクを参照することになります。*
  6. **市** あなたの拠点はどこですか?
</Steps>


### 支払い条件

:::注
お支払いをご自身で負担される場合、直接関係のある物件のみにアクセスできます。
:::

:::注意
このような状況では、Wink はホテル代金の支払いについて一切責任を負いません。
:::

:::注
このオプションはアカウント作成時のみ有効です。ほとんどの旅行代理店様は、この設定を無効にして🛑、代わりにこちらのポータルをご利用ください。
:::

当社の API を使用して当社と統合する登録旅行代理店には、支払いを受け取り、その資金を当社の施設に分配するオプションがあります。

<Steps>
  1. **支払いを受けたい** 旅行代理店が資金回収の責任を負います。
  2. **ホテル代を支払いたい** ホテル代の支払いは旅行代理店が責任を負います。
</Steps>


お客様と直接つながりたいホテルには警告が表示され、お客様のアカウントとの関係を確立するために、Wink はホテルへの支払いに責任を負わないことを承認するためにクリックする必要があります。

### 合意

<Steps>
  1. **合意** 同意する [アフィリエイト契約](/travel-agent/terms-of-service) そして [支払い条件](/travel-agent/payment-terms) 続行します。
  2. クリック `Save` ボタン。
</Steps>


おめでとうございます 🎉 ... Wink で最初のアフィリエイト アカウントを作成しました。

## 管理

<figure>

  <figcaption>*旅行代理店アカウント入力*</figcaption>

作成したら、住所を入力したり、アカウントにプロフィール写真を追加したりして、アカウントをさらにカスタマイズできます。

<Steps>
  1. 新しい旅行代理店アカウントをクリックします。 *するとアカウントダッシュボードに移動します。*
  2. クリック `Account > Profile` メインナビゲーションバーにあります。
  3. プロフィール ページでは、既存のプロフィールを更新したり、追加情報を追加したりすることができます。
</Steps>


:::注意
旅行代理店アカウントのプロフィール写真は、Winkユーザーのプロフィール写真とは異なります。旅行代理店アカウントのプロフィール写真はWinkLinksページに表示され、あなたとのつながりを希望するサプライヤーに表示されます。

ユーザープロフィール写真は主に社内で使用されます。
:::

## マネージャー

旅行代理店アカウントの管理を手伝ってもらうために、他のユーザーを招待することができます。招待を承認したユーザーは、お支払いを除き、アカウントへのフルアクセスが可能になります。ご自身の資金の管理は、常にご自身のみとなります。

<Steps>
  1. 連携したい旅行代理店アカウントを [アカウントページ](https://agent.wink.travel/company/list)。
  2. クリック `Account > Managers` メインナビゲーションバーにあります。
  3. アカウント マネージャーにするユーザーの電子メールを追加します。
  4. クリック `Invite` メールの横にあるリンクをクリックします。
</Steps>


Wink はユーザーに招待を承認または拒否するためのメールを送信します。

## 通知

ご連絡させていただく場合がございます。その際は、右上のベルマークが変わります。
送信内容を確認するには：

<Steps>
  1. クリック `Account > Notifications` メインナビゲーションバーにあります。
  2. 新しい通知と過去の通知が表示されます。
  3. 選択する `Mark as read` または `Delete` 通知を読んだらすぐに閉じてください。
</Steps>
