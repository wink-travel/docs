[Wink Agent](https://agent.wink.travel)は、世界中の旅行在庫とあなたをつなげます。この記事では、Winkで旅行代理店アカウントを作成および管理する手順を説明します。

## アカウント作成

認証が完了すると、すべての旅行代理店アカウントが表示される[アカウントページ](https://agent.wink.travel/company/list)にリダイレクトされます。

`Create new account`ボタンをクリックして続行します。[アカウント作成](https://agent.wink.travel/company/create)ページにリダイレクトされます。

すべての必須項目を入力してください。

<Steps>
1. **名前** アカウントの名前。*これは実名である必要はなく、WinkLinksページでユーザーが見る名前や、サプライヤーが直接あなたとつながりたいときに表示される名前です。*
2. **法人種別** 会社か個人か。*主に資金受け取りの資格を判断するために使用されます。*
3. **タイプ** アカウントに最も適したタイプを選択してください。*例：インフルエンサー*
4. **説明** アカウントの説明を書いてください。*これはWinkLinksページやサプライヤーが直接あなたとつながりたいときに表示されます。*
5. **ウェブサイト** メインのウェブサイトを入力してください。*これはIGアカウントのURLや会社のウェブサイトでもかまいません。オンラインでの主な活動場所であるべきです。サプライヤーはこれらのリンクを見て、接続するかどうかを判断します。*
6. **都市** 拠点はどこですか？
</Steps>

### 支払い条件

:::note
支払い責任を負うことを選択した場合、直接接続している物件のみアクセス可能になります。
:::

:::caution
この場合、Winkはホテルへの支払いに関して一切の責任を負いません。
:::

:::note
このオプションはアカウント作成時のみ選択可能です。ほとんどの旅行代理店はこの設定を無効のままにし🛑、代わりにこのポータルを使用してください。
:::

APIを使って当社と連携している登録済み旅行代理店は、支払いを受け取り、当社の物件に資金を分配するオプションがあります。

<Steps>
1. **支払いを受け取りたい** 旅行代理店が資金回収の責任を負います。
2. **ホテルに支払いたい** 旅行代理店がホテルへの支払い責任を負います。
</Steps>

直接あなたと接続したいホテルは、警告を表示し、Winkがホテルへの支払い責任を負わないことを認識した上で、あなたのアカウントとの関係を確立するためにクリックして承認する必要があります。

### 同意

<Steps>
1. **同意** 続行するには、当社の[アフィリエイト契約](/ja/travel-agent/terms-of-service)および[支払い条件](/ja/travel-agent/payment-terms)に同意してください。
2. `Save`ボタンをクリックします。
</Steps>

おめでとうございます🎉 ...Winkで最初のアフィリエイトアカウントを作成しました。

作成後は、住所を入力したり、プロフィール写真を追加したりしてアカウントをさらにカスタマイズできます。

<Steps>
1. 新しく作成した旅行代理店アカウントをクリックします。*アカウントダッシュボードに移動します。*
2. メインナビゲーションバーの`Account > Profile`をクリックします。
3. プロフィールページで、既存のプロフィールを更新したり、追加情報を入力したりできます。
</Steps>

:::note
旅行代理店アカウントのプロフィール写真は、Winkユーザープロフィール写真と混同しないでください。旅行代理店アカウントのプロフィール写真はWinkLinksページに表示され、接続を希望するサプライヤーに見られます。

ユーザープロフィール写真は主に内部利用向けです。
:::

## マネージャー

他のユーザーを招待して旅行代理店アカウントの管理を手伝ってもらうことができます。ユーザーが招待を承諾すると、支払いを除くアカウントの全権限を持ちます。資金の管理は常にあなたが唯一の責任者です。

<Steps>
1. [アカウントページ](https://agent.wink.travel/company/list)から操作したい旅行代理店アカウントを選択します。
2. メインナビゲーションバーの`Account > Managers`をクリックします。
3. アカウントマネージャーにしたいユーザーのメールアドレスを追加します。
4. メールアドレスの横にある`Invite`リンクをクリックします。
</Steps>

Winkはユーザーに招待の承諾または拒否を促すメールを送信します。

## 通知

時折、私たちから連絡を差し上げることがあります。その際、右上のベルアイコンが変わります。
送信内容を読むには：

<Steps>
1. メインナビゲーションバーの`Account > Notifications`をクリックします。
2. 新着および過去の通知が表示されます。
3. 読んだ通知は`Mark as read`または`Delete`を選択してください。
</Steps>

## API

旅行代理店は引き続きアフィリエイトとみなされます。`Affiliates`を管理したい開発者は[Developers > API > Affiliate](/ja/developers/apis/#affiliate-api)にアクセスしてください。
