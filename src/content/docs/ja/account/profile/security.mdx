[iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) の**セキュリティ**タブでは、パスワードなしでサインインできるWebAuthn認証情報である**パスキー**を管理します。

## パスキーとは？

パスキーは、あなたのデバイス（またはパスワードマネージャー）に紐づいた暗号化認証情報で、あなた自身であることを証明します。パスキーはフィッシングに強く、パスワードを入力するよりも速く認証できます。

## パスキーの登録方法

1. 既存の認証情報で[iam.wink.travel](https://iam.wink.travel)にサインインします。
2. **マイアカウント → セキュリティ**に移動します。
3. `Add passkey`をクリックします。
4. ブラウザがデバイスでの認証を促します（Touch ID、Face ID、Windows Hello、ハードウェアキー、または同期されたパスワードマネージャー）。
5. パスキーにわかりやすい名前（例：*Work MacBook*）を付けて後で認識しやすくします。

## パスキーを使ったサインイン方法

[iam.wink.travel](https://iam.wink.travel)のサインイン画面でメールアドレスを入力すると、そのメールアドレスに登録されたパスキーがあれば、パスワード入力の代わりにパスキーでの認証を促されます。

## パスキーの削除方法

同じセキュリティタブで、登録済みの各パスキーに`Remove`ボタンがあります。デバイスを紛失した場合やハードウェアキーを廃止する場合、または使わなくなった認証情報を整理したい場合に使用してください。

:::caution
すべてのパスキーを削除すると、サインイン方法はパスワードのみになります。最後のパスキーを削除する前に、パスワードを必ず覚えていることを確認してください。
:::
