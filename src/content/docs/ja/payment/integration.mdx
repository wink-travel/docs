旅行会社で、ビジネスの支払いおよび支払い処理部分をTripPayにアウトソースしたい場合、始めるための基本的な6つのステップは以下の通りです：

<Steps>
1. Winkで[userアカウント](/ja/user/register)を作成します。
2. [TripPay](https://pay.wink.travel)でアカウントを作成します。
3. TripPayでAPIと連携するための[アプリケーション](/ja/developers/applications)を作成します。
4. TripPayで受取人を作成しマッピングします。*例：ほとんどの場合、TripPayに支払いをしてほしいプラットフォーム上の施設のアカウントを作成することを意味します。*
5. [支払い用ウェブコンポーネント](/ja/developers/web-components/#payment)をサイトに統合します。
6. 旅行者の`ショッピングカート`をTripPayに送信し、支払いプロセスを開始します。
</Steps>

:::tip
旅行サイトでTripPayを使用するための完全なガイドは、ガイド[Integrating with TripPay](/ja/guides/integrating-with-trip-pay)をご覧ください。
:::

## API

`Payments`を管理したい開発者は、[Developers > API > Payments](/ja/developers/apis/#payment-api)にアクセスしてください。
