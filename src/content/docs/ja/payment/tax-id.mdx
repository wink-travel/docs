一部の地域では、支払いを受け取る前に国の税務識別番号の提出が必要です。TripPayはアカウント設定の一環として、この手続きを促します。

## 税務IDの提出方法

求められたら、国の税務識別番号を入力してください：

- **米国の個人** — 社会保障番号（SSN）。*形式：111-11-1111*
- **米国の法人** — 雇用者識別番号（EIN）。*形式：11-111111*
- **その他の地域** — お住まいの国の同等の税務識別番号。

税務IDを入力後、送信前に情報が正しいことを確認するよう求められます。

:::note
これはKYC（顧客確認）コンプライアンスのための一度限りの手続きです。地域によっては追加の書類提出を求められる場合があります。
:::
