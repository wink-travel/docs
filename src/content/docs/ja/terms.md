# 利用規約

各ポータルには異なる利用規約がある場合があります。ご自身に該当するセクションに進み、「利用規約および支払い条件」というサブセクションをお探しください。

## アフィリエイトポータル

Winkを使ってソーシャルアカウントを収益化し、パッシブインカムを得る方法を知りたいアフィリエイト向けです。  
[Wink Studio](https://studio.wink.travel)には、プロのように販売するために必要なすべてのツールが揃っています。
