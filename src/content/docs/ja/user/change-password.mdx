Winkユーザーアカウントのパスワードを変更したい場合は、以下の手順に従ってください。

## パスワードの更新

<Steps>
1. 当社のいずれかのウェブサイトにログインしている必要があります。
2. 右上のプロフィールアイコンをクリックするとメニューが表示されます。
3. `パスワードを変更`リンクをクリックします。
4. パスワード変更の指示に従います。
5. `保存`をクリックします。
</Steps>

次回ログイン時には、新しいパスワードが必要になります。

## パスワードを忘れた場合

ユーザーパスワードをリセットする必要がある場合があります。

ユーザーパスワードをリセットする手順は以下の通りです。

<Steps>
1. サインインページの`パスワードを忘れた場合`リンクをクリックします。
2. リセットしたいアカウントのユーザー名またはメールアドレスを入力します。
3. `パスワードを忘れた場合`ボタンをクリックします。*パスワードリセットフォームにリダイレクトされます。*
4. メールを確認します。
</Steps>

2. 新しいパスワードを確認します。
3. `パスワードをリセット`ボタンをクリックします。*サインインページにリダイレクトされます。*
</Steps>
