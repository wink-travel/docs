Wink ユーザー アカウントのパスワードを変更する場合は、次の手順に従ってください。

## パスワードを更新する

<Steps>
  1. 弊社のウェブサイトのいずれかにログインする必要があります。
  2. 右上隅にあるプロフィールアイコンをクリックすると、メニューが表示されます。
  3. クリック `Change password` リンク。
  4. 指示に従ってパスワードを変更してください。
  5. クリック `Save`。
</Steps>


次回ログインするときには、新しいパスワードが必要になります。

## パスワードをお忘れですか

場合によっては、ユーザーパスワードをリセットする必要があります。

ユーザー パスワードをリセットする手順は次のとおりです。

<Steps>
  1. クリックして `Forgot password` サインインページからのリンク。
  2. リセットしたいアカウントのユーザー名/メールアドレスを入力します。
  3. クリック `Forgot password` ボタン。 *パスワードリセットフォームにリダイレクトされます。*
  4. メールを確認してください。
</Steps>

<Steps>
  1. 電子メールで受け取ったコードと新しいパスワードを新しいフォームに貼り付けます。
  2. 新しいパスワードを確認してください。
  3. クリック `Reset password` ボタン。 *サインインページにリダイレクトされます。*
</Steps>

パスワードのリセットに成功しました。新しい認証情報でWinkにログインできます。
