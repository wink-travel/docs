どのポータルからでもユーザーアカウントを作成できます。一度作成すると、このアカウントはWinkのすべてのアプリやウェブサイトで使用できます。

## 旅行者

以下の手順に従ってください：

<Steps>
1. [https://ota.wink.travel](https://ota.wink.travel) にアクセスします
2. 右上の `Sign-In / Register` をクリックします
3. 基本的なユーザー情報の入力を求められます。必須項目をすべて入力し、`Save` をクリックします。
4. これでユーザー名（メールアドレス）とパスワードでWinkにログインできます。
5. ログイン後、システムは元のページにリダイレクトします。
</Steps>

## 登録

Winkのユーザーアカウントを作成すると、プラットフォームのすべての機能にアクセスできます。物件やアフィリエイトアカウントを無制限に作成可能です。

<figure>
<CloudinaryImage
      width="350"
      height="553"
      resize="scale"
      quality="best"
      alt="サインイン"
      src="academy/sign-in"
      errorImage="noimage_opaque_nyrtl0.png"
    />
    <figcaption>*サインインフォーム*</figcaption>
</figure>

<Steps>
1. 初めて認証サーバーにリダイレクトされた際、ユーザー名とパスワードの入力を求められます。
2. 代わりに、`Create account` ボタンをクリックします。
3. 必須項目をすべて入力し、`Register` をクリックして続行します。これでサインインフォームに戻ります。
4. メールアドレス／ユーザー名とパスワードを入力してログインします。
5. 元のサイトにリダイレクトされます。
</Steps>
