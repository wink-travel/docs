いずれかのポータルからユーザーアカウントを作成できます。作成したアカウントは、Wink のすべてのアプリとウェブサイトでご利用いただけます。

## 旅行者

次の手順に従ってください。

<Steps>
  1. ぜひご覧ください [https://ota.wink.travel](https://ota.wink.travel)
  2. クリック `Sign-In / Register` 右上隅
  3. 基本的なユーザー情報を入力するよう求められます。必須項目をすべて入力し、クリックしてください。 `Save`。
  4. ユーザー名（電子メール）とパスワードを使用して Wink にログインできるようになりました。
  5. ログインすると、システムによって元の場所にリダイレクトされます。
</Steps>

## 登録

Winkユーザーアカウントを作成すると、プラットフォームのすべての機能にアクセスできるようになります。プロパティとアフィリエイトアカウントは無制限に作成できます。

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="サインイン" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*サインインフォーム*</figcaption>
</figure>


<Steps>
  1. 初めて認証サーバーにアクセスすると、ユーザー名とパスワードの入力を求められます。
  2. 代わりに、 `Create account` ボタン。
  3. 必須項目をすべて入力してクリック `Register` 続行するには、サインインフォームに戻ります。
  4. ログインするには、メールアドレス/ユーザー名とパスワードを入力してください。
  5. 元のサイトにリダイレクトされます。
</Steps>
