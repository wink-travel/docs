Winkユーザーアカウントのユーザープロフィールを更新したい場合は、以下の手順に従ってください。

<Steps>
1. 当社のいずれかのウェブサイトにログインしている必要があります。
2. 右上隅のプロフィールアイコンをクリックするとメニューが表示されます。
3. `プロフィール`リンクをクリックします。
4. `メールアドレス`、`名`、`姓`の更新手順に従います。
5. 新しい`プロフィール写真`をアップロードすることもできます。
6. `保存`をクリックします。
</Steps>

:::note
新しいプロフィール写真は、Winkから完全にログアウトした次回に表示されます。
:::
