Wink ユーザー アカウントのユーザー プロファイルを更新する場合は、次の手順に従います。

<Steps>
  1. 弊社のウェブサイトのいずれかにログインする必要があります。
  2. 右上隅にあるプロフィールアイコンをクリックすると、メニューが表示されます。
  3. クリック `Profile` リンク。
  4. 指示に従って更新してください `e-mail`、 `first` そして `last name`。
  5. 新しいものをアップロードすることもできます `profile picture` あなた自身の。
  6. クリック `Save`。
</Steps>


:::note
新しいプロフィール写真は、次回Winkから完全にログアウトした時に表示されます。
:::
