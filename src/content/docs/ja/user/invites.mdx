Winkユーザーは、自分の物件、アフィリエイト、または旅行代理店アカウントの管理をあなたに任せることができます。その場合、招待リクエストのメールが届きます。

import { Steps } from '@astrojs/starlight/components';

現在のWink招待を確認するには、以下の手順に従ってください。

<Steps>
1. 弊社のいずれかのウェブサイトにログインしている必要があります。
2. 右上のプロフィールアイコンをクリックするとメニューが表示されます。
3. `Invites`リンクをクリックします。
4. 招待を`Accepting`または`Rejecting`して対応します。
</Steps>
