Winkユーザーは、宿泊施設、アフィリエイト、または旅行代理店のアカウントのいずれかをあなたに管理してもらうことができます。その場合、招待リクエストが記載されたメールが届きます。

import { Steps } from '@astrojs/starlight/components';

現在の Wink 招待を確認するには、次の手順に従います。

<Steps>
  1. 弊社のウェブサイトのいずれかにログインする必要があります。
  2. 右上隅にあるプロフィールアイコンをクリックすると、メニューが表示されます。
  3. クリック `Invites` リンク。
  4. 招待された相手とやり取りするには、 `Accepting` または `Rejecting` 彼ら。
</Steps>
