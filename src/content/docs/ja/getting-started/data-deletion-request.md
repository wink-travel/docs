Winkでは、お客様のプライバシーを尊重し、個人データの保護に全力で取り組んでいます。私たちは**一般データ保護規則（GDPR）**および世界中のその他の適用されるデータ保護法を遵守しています。

このページでは、FacebookやInstagramなどのソーシャルログインプロバイダーを通じて取得したデータを含む、当社プラットフォームで収集した個人データの削除をリクエストする方法を説明します。

## データプライバシーに対する当社のコミットメント

当社はサービス提供に必要なデータのみを収集し、お客様の個人情報を販売することは決してありません。これには以下が含まれる場合があります：

- お名前とメールアドレス
- 公開プロフィール情報（例：FacebookやGoogleから）
- プラットフォーム上の利用状況やアクティビティデータ
- お客様が明示的に許可した権限

当社は適切な措置を講じてデータを保護し、GDPRおよびMetaのデータポリシーに従ってすべての削除リクエストを処理します。

## データ削除のリクエスト方法

お客様は、当社が保有するお客様の個人情報の削除をリクエストする権利があります。

データ削除をリクエストするには：

- 📧 **メールを送信してください**: [support@wink.travel](mailto:support@wink.travel)
- 📝 件名に「`Data Deletion Request`」と記載してください  
- 🔍 以下のような識別情報を含めてください：
  - 当社プラットフォームで使用しているメールアドレス
  - FacebookやGoogleのIDなど（該当する場合）

GDPRの要件により、リクエスト受領後**30日以内**に回答いたします。

## ご質問やご不明点は？

データの収集、利用、削除方法についてご質問がある場合は、プライバシーチームまでお気軽にお問い合わせください：[support@wink.travel](mailto:support@wink.travel)。
