Winkは人工知能を活用して、旅行在庫のあらゆる部分に対して洗練された多言語コンテンツの作成を支援します。新しい物件を登録する場合でも、既存のリスティングを更新する場合でも、AIにより市場投入までの時間を数週間から数分に短縮します。

## 仕組み

Winkのコンテンツエンジンは、堅牢なマルチプロバイダーAIアーキテクチャを基盤としています。AI生成コンテンツをリクエストすると、プラットフォームはOpenAI、Anthropic、Googleなどの主要なAIサービスのプールから最適なプロバイダーを自動的に選択します。もし一つのプロバイダーが一時的に利用できなくなっても、システムは次のプロバイダーにシームレスに切り替わり、ダウンタイムを防ぎます。

## AIが生成するもの

AIコンテンツ生成は**9つの在庫タイプ**に対応しています：

| 在庫タイプ | AIが生成する内容 |
|---|---|
| **物件** | ウェルカムテキスト、説明、ハイライト |
| **客室** | 部屋の説明、特徴、セールスポイント |
| **レストラン** | 食事の説明、料理のハイライト、雰囲気 |
| **スパ** | トリートメントの説明、ウェルネスのハイライト |
| **アクティビティ** | 体験の説明、旅程のナレーション |
| **観光名所** | 名所の概要、歴史的背景 |
| **会議室** | スペースの説明、イベント機能 |
| **場所** | 観光スポットの説明、地域の文脈 |
| **アドオン** | パッケージやバンドルの説明 |

## トーン

AI生成コンテンツのトーンはブランドに合わせてカスタマイズ可能です。利用可能なトーンは以下の通りです：

- **ニュートラル** — 率直で情報的。
- **フレンドリー** — 温かく親しみやすい。
- **ラグジュアリー** — 洗練され高級感のある。
- **フーディー** — 料理に焦点を当て味覚重視。
- **アドベンチャラス** — 大胆で行動的。
- **ファミリー** — 子連れ家族に優しい。
- **ロマンチック** — 親密でカップル向け。
- **ユーモラス** — 軽快で遊び心のある。
- **エコ** — 持続可能性と自然重視。
- **ローカルインサイダー** — 地元の専門家のようなアドバイス。
- **インクルーシブ** — 誰にでもアクセスしやすく歓迎的。
- **バジェット** — 価値重視でお得感のある。

物件の個性やターゲットに最適なトーンを選んでください。

## AI支援登録

WinkでのAIの最も強力な活用は物件登録時です。物件のウェブサイトURLを入力するだけで、Winkは以下を行います：

1. **ウェブサイトをスクレイピング**して物件を理解。
2. **説明文を生成** — ウェルカムテキスト、物件のハイライト、要約。
3. **アメニティを提案** — ウェブサイトの内容から業界標準のアメニティコードにマッピング。
4. **属性を抽出** — 位置情報、物件タイプ、対象顧客。

結果を確認・修正してから公開。数週間かかっていた作業が1時間以内で完了します。

## AIと翻訳

AIが主要言語でコンテンツを生成した後、Winkは統合された翻訳サービスを使って40以上の言語に自動翻訳できます。これにより、1回のコンテンツ生成で多言語リスティングをグローバルな顧客向けに作成可能です。
