Winkのアフィリエイトとして最初に始めるときは、プラットフォームで販売したい商品を見つけて、ソーシャルメディアや旅行ブログ、アプリなどで共有します。
ある時点で、ツールの使い方にかなり慣れてきて、いくつかの予約も獲得できているでしょう。
次に販売力をレベルアップさせるためにできることは、サプライヤーと直接の関係を求めることです。

:::note
アクティブおよび保留中の関係は、メインナビゲーションバーの`アカウント > ネットワーク`をクリックして操作できます。
:::

この記事では、サプライヤーに直接の関係を依頼するタイミングと方法について説明します。これは双方にとって有益な場合があります。

## 依頼するタイミング

ほとんどのアフィリエイトにとって、サプライヤーと直接つながることはあまり意味がありません。多くのサプライヤーはすでにプラットフォーム上のすべての人に最高の取引やアップセルを提供しています。
しかし、以下の条件のいずれかに当てはまる場合は、サプライヤーとの直接のつながりを試みることをお勧めします。

- 特定のサプライヤーに特に適した潜在的な予約者の大きなオーディエンスを持っている場合。
*例：あなたはスキー好きで、パウダーやバックカントリー、スキーに関するコンテンツを持ち、フォロワーも全員スキー好きです。スキー場に連絡して、平日限定の特別プランなどを依頼するのは賢い方法です。*
- 自分の旅行グループを目的地に連れて行く場合。
*例：ヨガリトリートを企画していて、20人のヨギが確定している場合。直接のつながりがあれば、リトリート側からグループ割引が適用されるユニークなプロモコードを提供してもらい、参加者に共有できます。*
- 旅行代理店や法人で大量の予約を扱っている場合。
*旅行代理店や法人もWinkのアフィリエイトとみなされますが、専用のサイトも持っています。*

## 要件

リクエストを開始するには、Winkで過去に少なくとも5件の予約実績が必要です。
この要件は、サプライヤーがメッセージで溢れないように設定しています。まずはWinkで実績を積むことに集中し、その後サプライヤーからアプローチが来ることをお勧めします。

## 仕組み

取引したいサプライヤーを見つけたら（[検索](/ja/studio/search/)を参照）、関係リクエストを開始する方法は以下の通りです。

<Steps>
1. サプライヤーの検索結果のいずれかで`直接のつながりをリクエスト`をクリックします。
2. サプライヤーに自己紹介するフォームが表示されます。礼儀正しく要点をまとめてください。自分が誰で、なぜ連絡しているのか、どのように役立てると思うのか、現在の「依頼内容」の具体的な点を伝えます。
3. フォームを送信し、返答を待ちます。返答には1日から1週間かかることがあります。サプライヤーは忙しいため、辛抱強くお待ちください。返答を待っている間は同じサプライヤーに再度リクエストを送ることはできません。
</Steps>

サプライヤーからの返答は以下のいずれかです：

- **承認** リクエストが承認され、あなたとサプライヤー間で直接の販売チャネルが作成されます。これはさまざまな方法で設定可能です。
- **拒否** リクエストが拒否され、これまでと同じ在庫にアクセスできます。拒否後、最短で2か月後に再度リクエストを送ることができます。
- **無応答** 何らかの理由でサプライヤーが全く返答しない場合は、別のサプライヤーを探すことを検討してください。

## 利点

承認されると、サプライヤーは新たに作成された販売チャネルに対して以下のことが可能になります。

- あなたのオーディエンス向けに異なる会員割引を設定する。
- あなたがより多くの報酬を得られるように異なるコミッション率を設定する。
- あなたのオーディエンスだけが利用できるユニークな取引を追加する。
- あなたのオーディエンスだけが利用できるユニークな在庫やアップセルを追加する。

## API

ネットワークを管理したい開発者は、[Developers > API > Sales Channel](/ja/developers/apis/#sales-channel-api)をご覧ください。
