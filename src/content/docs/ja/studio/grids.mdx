グリッドは、サプライヤーと在庫のリストを表し、インタラクティブで予約可能なグリッドUIを通じて詳細を表示します。
グリッドは、キュレーションリストや保存された検索からの任意の在庫を表示することができます。
ユーザーは、個別の[カード](/ja/studio/cards)と同じ方法でグリッドとやり取りしますが、1つの追加機能があります：

1. `もっと見る`ボタンをクリックしてグリッドをページネーションできます（*さらにアイテムがある場合*）。

## グリッドの種類

グリッドには3種類あります：

1. [キュレーションリスト](/ja/studio/curated-lists#list-actions)に基づくグリッド。
2. [保存された検索](/ja/studio/saved-searches#list-actions)に基づくグリッド。
3. ロケーションとソート基準に基づくグリッド（*ランキンググリッド*）。

## キュレーションリストグリッド

これは、キュレーションリストに収集した在庫を使用し、そのリストを予約可能な旅行在庫に変換してユーザーに表示するグリッドです。

### 作成

メインナビバーから `Inventory > Curated Lists` に移動します。この例では、`Favorites` リストを使用します。
まだ `Favorites` に何も追加していない場合は、[Search](/ja/studio/search)で方法を学んでください。

<figure>
<CloudinaryImage
      width="350"
      height="394"
      resize="scale"
      quality="best"
      alt="キュレーションリストグリッドの作成"
      src="academy/create-curated-list-grid"
      errorImage="noimage_opaque_nyrtl0.png"
    />
    <figcaption>*キュレーションリストグリッド作成フォーム*</figcaption>
</figure>

<Steps>
1. `Favorites` リストの `Actions` ボタンをクリックします。
2. `Create a grid` ボタンをクリックします。
3. 新しいウィンドウが表示され、グリッドの名前を入力できます。*以下を参照*。
4. `OK` ボタンをクリックして続行します。
</Steps>

グリッドが作成されました。メインナビバーから `Tools > Grids` に移動し、`Curated Grids` タブをクリックして新しいグリッドを確認してください。

### カスタマイズ

フォームでは、以下の方法でグリッドをカスタマイズできます：

<Steps>
 1. グリッドの内容を覚えやすい名前を付けます。
 2. このグリッドに適用したい[カスタマイズ](/ja/studio/customization)を選択します。
 3. ユーザーが最初に見るカードの表面を選択します。デフォルトはその在庫のネイティブフェイスです。
 4. カードに使用するバッジを選択します。バッジは、`エコフレンドリー`などの集約指標で在庫を比較するのに役立ちます。
 5. [Webクローラー](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)が使用するキーワードをカンマ区切りで追加します。
 6. ユーザーに表示したい言語でタイトルと説明を追加します。
 7. `Save` ボタンをクリックして続行します。
</Steps>

### 埋め込み

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="LIST"
       id="9a212b5e-62a7-11ef-ac3f-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

グリッドをサイトに埋め込む方法は以下の通りです：

- 3行目はWinkのスタイルをサイトに埋め込む方法を示しています。
- 6〜9行目は、[wink-content-loader](/ja/developers/web-components/#content-loader) Webコンポーネントを使い、コード用にグリッドを取得する方法を示しています。
- 11行目はJavascriptをサイトに埋め込む方法を示しています。
- 13行目は、[wink-app-loader](/ja/developers/web-components/#application-loader) Webコンポーネントを使い、ページレベルの設定を取得する方法を示しています。

## 保存された検索グリッド

これは、保存された検索クエリの在庫を使用し、検索結果を予約可能な旅行在庫に変換してユーザーに表示するグリッドです。

### 作成

メインナビバーから `Inventory > Saved searches` に移動します。
まだ保存された検索を作成していない場合は、[Search](/ja/studio/search)で方法を学んでください。

## ランキンググリッド

### 作成

`Tools > Grids` に移動し、`Ranked Grid` タブをクリックしてランキンググリッドを作成します。`Create ranked grid` ボタンをクリックしてください。

### カスタマイズ

フォームでは、ランキンググリッドを以下の方法でカスタマイズできます：

<Steps>
 1. 目的地を選択します。*例：東京*。
 1. グリッドの内容を覚えやすい名前を付けます。*例：東京のエコフレンドリーホテル*。
 2. このグリッドに適用したい[カスタマイズ](/ja/studio/customization)を選択します。
 3. ユーザーが最初に見るカードの表面を選択します。デフォルトはその在庫のネイティブフェイスです。
 4. 物件をソートする特性を選択します。*例：エコフレンドリー度*。
 5. [Webクローラー](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)が使用するキーワードをカンマ区切りで追加します。
 5. ユーザーに表示したい言語でタイトルと説明を追加します。
 7. `Save` ボタンをクリックして続行します。
</Steps>

ランキンググリッドを保存すると、ランキンググリッドページにリダイレクトされ、グリッドは世界に共有する準備が整います。
