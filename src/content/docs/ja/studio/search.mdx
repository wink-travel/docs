## 検索を開始

Wink でサプライヤーと在庫の検索を開始する方法は次のとおりです。

1. 連携したいアフィリエイトアカウントを選択してください[アカウントのリスト](https://studio.wink.travel/company/list)
2. 次の 2 つの方法のいずれかで検索ページに移動します。
   1. ダッシュボードから直接物件をすばやく検索したり、...
   2. メインナビゲーションバーから、在庫→検索をクリックします。
3. 検索ページには、Wink の最新のサプライヤーと在庫、および選択可能なカテゴリが表示されます。
4. この例では、カードをクリックしてください。`Hotels`。
5. 検索結果ページにリダイレクトされます。ここで、利用可能なサプライヤーをスクロールして確認できます。`Hotels`カテゴリを選択し、画面左側のフィルターを使用して検索フィルターを適用/変更します。

## 検索結果の調整

在庫とサプライヤーを絞り込む方法は以上です。検索結果の右上にあるフィルターアイコンをクリックしてください。

* プロパティ名（*例：センタラ*）
* 在庫名（*例：ヴィラ*）
* 在庫タイプ（*例：スパ*）
* 星評価（*例：5つ星ホテル*）
* ユーザーレビュー評価（*例：8 / 10*）
* 地理位置情報（*例：地図上の場所を選択*）
* 大陸（*例：北米*）
* 国（*例：ドイツやフランス*）
* 都市（*例：東京*）
* ライフスタイル（*例：LGBTQ*）
* 場所の種類（*例：空港*）
* セグメントタイプ（*例：中程度*）
* カテゴリの種類（*例：ロッジ*）
* スタイル（*例：アールデコ*）
* 直接関係のみ（*例：直接在庫の表示*）
* ペット同伴可（*例：ペットフレンドリーな在庫を表示する*）
* 環境に優しい（*例：環境に優しい在庫を表示する*）
* お子様連れに優しい（*例：子供向けの在庫を表示する*）
* 人気のある （*例：高価値の在庫を表示する*）

## アクション

アクションを使用すると、在庫やサプライヤーとやり取りできます。3つの主要なカテゴリがあり、それぞれ以下の操作が可能です。

1. **サプライヤーの詳細**サプライヤーについてさらに詳しく知るにはどうすればいいですか?
2. **作成する...**&#x3053;の在庫を販売できるものに変換するにはどうすればよいでしょうか?
3. **に追加...**&#x3053;の在庫を同時に販売できるリストに保存するにはどうすればよいでしょうか?

### サプライヤーの詳細

<figure>
  <CloudinaryImage width="350" height="190" resize="scale" quality="best" alt="サプライヤーの詳細" src="academy/supplier-details" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*サプライヤーの詳細アクション*</figcaption>
</figure>


クリックすると`Supplier details`、 あなたはできる：

* **本**宿泊施設に直接アクセスし、アフィリエイト ID を使用して予約すると、コミッションを獲得できます。
* **直接接続をリクエストする**サプライヤーに直接接続してもらえるかどうかを尋ねてください。
* **トラックパフォーマンス**分析チャートを作成して、サプライヤーのパフォーマンスを長期にわたって追跡します。

### 作成する...

<figure>
  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="アクション" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

クリックすると`Create...`、 あなたはできる：

* **リンクを作成する** [リンクを作成する](/studio/shareable-links)メール、ソーシャル メディア、メッセージ アプリで簡単に共有できます。
* **地図を作る** [マップを作成する](/studio/maps)WinkLinks、WordPress、埋め込みなどで使用できます。
* **カードを作る** [カードを作成する](/studio/cards)WinkLinks、WordPress、埋め込みなどで使用できます。

### に追加...

<figure>
  <CloudinaryImage width="350" height="203" resize="scale" quality="best" alt="アクション" src="academy/add-to" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*アクションに追加*</figcaption>

クリックすると`Add to...`既存の[キュレーションリスト](/studio/curated-lists)新しいリストを作成する方法もあります。以下のことが可能です。

* **既存のリストに追加**これにより、すでに作成したリストにサプライヤーまたは在庫が追加されます。
* **新しいキュレーションリスト**これにより、新しいリストを作成するように求められ、アイテムがそのリストに追加されます。

次のセクションでは、さまざまな方法で検索結果を収益化する方法を説明します。

## 空の結果

特定のサプライヤーを探しているのに見つからない場合は、Googleプレイスで検索するための入力欄が表示されます。Googleプレイスにサプライヤーが存在する場合は、`lead`Wink で。
これは、このサプライヤーに連絡して Wink への参加を依頼したいという依頼があることを示しています。サプライヤーに連絡して参加を依頼します。参加が決定次第、お知らせいたします。
