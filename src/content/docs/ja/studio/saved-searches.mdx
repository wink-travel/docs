保存された検索は、その名前が示す通りの機能です。旅行在庫を検索する際に、後で使うために検索クエリを保存するオプションがあります。

## 検索クエリの保存

在庫の検索を開始するとき（[Search](/ja/studio/search) を参照）、検索フィルターを細かく調整することができます。  
クエリフォームの下部に `Save search` のオプションがあります。

<figure>

このリンクをクリックすると、この検索に名前を付けることができます（*例：My Library of Magic Hotels*）ので、後で覚えておくことができます。

<figure>
<CloudinaryImage
      width="350"
      height="329"
      resize="scale"

      src="academy/name-save-search"
      errorImage="noimage_opaque_nyrtl0.png"
    />
    <figcaption>*検索に名前を付ける*</figcaption>
</figure>

名前を付けて `Save search` ボタンをクリックします。その後、メインナビゲーションバーの `Inventory > Saves Searches` に移動すると、新しく作成したレコードを確認できます。

## リストの操作

保存された検索は、グリッドやマップの形式で旅行在庫をユーザーに表示し、検索結果を希望の表示方法にするために使用できます。

<figure>
<CloudinaryImage
      width="700"
      height="939"
      resize="scale"
      quality="best"
      alt="保存された検索の操作"
      src="academy/saved-search-actions"
      errorImage="noimage_opaque_nyrtl0.png"
    />
    <figcaption>*フィルターが適用されていない保存された検索クエリ*</figcaption>
</figure>

保存された検索クエリには、実行できる操作があります：

- 検索クエリを[マップ](/ja/studio/maps)に変換する。
- 検索クエリを[グリッド](/ja/studio/grids)に変換する。
- 保存された検索を削除する。

## API

保存された検索を管理したい開発者は、[Developers > API > Browse](/ja/developers/apis/#browse-api) にアクセスしてください。
