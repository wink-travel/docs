キュレーションリストは、旅行在庫を検索して見つけた中から選択した在庫をグループ化し、並べ替える機能です。
作成後は、[WinkLinks](/ja/link-manager/wink-links)、当社の[WordPressプラグイン](/ja/integrations/wordpress/)、または当社の[Web Components](/ja/integrations/web-components)のいずれかを使って、[グリッド](/ja/studio/grids)や[マップ](/ja/studio/maps)として販売可能にし、ユーザーに公開できます。

すべてのアフィリエイトアカウントはデフォルトで `Favorites` リストが用意されており、すぐに使用可能です。`Favorites` は特別なリストで削除できません。
すべてのキュレーションリストでは以下の操作が可能です。

## リストを作成する

- ダッシュボードのナビゲーションバーで `Inventory > Curated Lists` をクリックします。
- `Make a new list` ボタンをクリックして新しいキュレーションリストを作成します。
- リストの名前を入力します。*例: Top 3 NYC Picks*。
- `Save` ボタンをクリックします。

## リストアイテムを追加する

検索結果から任意のリストにアイテムを追加できます。

- [Search](/ja/studio/search) に移動します。
- リストに追加したいものを検索します。
- 見つけたら、検索結果カードの左下にある `Actions` リンクをクリックします。
- `Add to...` をクリックし、追加したいキュレーションリストを選択します。
- キュレーションリストページに戻り、アイテムがリストに追加されていることを確認します。

## リストの操作

すべてのリストには実行可能な操作があります。

- リストを[マップ](/ja/studio/maps)に変換する。
- リストを[グリッド](/ja/studio/grids)に変換する。
- リストを削除する（*Favoritesを除く*）。

## API

キュレーションリストを管理したい開発者は、[Developers > API > Browse](/ja/developers/apis/#browse-api)にアクセスしてください。
