キュレーションリストは、旅行関連商品を検索した際に見つけた商品から選択した商品をグループ化して並べ替えることができる機能です。
作成したリストは、販売できるようにすることでユーザーに公開できます。[グリッド](/studio/grids)または[地図](/studio/maps) を通して [ウィンクリンクス](/link-manager/wink-links)、 私たちの [WordPressプラグイン](/developers/wordpress/)または、当社の[Webコンポーネント](/developers/web-components)あなたのサイトに。

デフォルトでは、すべてのアフィリエイトアカウントは`Favorites`すぐに使えるリストです。`Favorites`は特別な機能のため、削除できません。
すべてのキュレーションリストでは、以下の操作が可能です。

## リストを作成する

<figure>
  <CloudinaryImage width="350" height="219" resize="scale" quality="best" alt="厳選リストを作成する" src="academy/create-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*新しいキュレーションリストフォームを作成する*</figcaption>
</figure>


* ダッシュボードからクリック`Inventory > Curated Lists`ナビゲーション バーにあります。
* をクリックして新しいキュレーションリストを作成します`Make a new list` ボタン。
* リストに名前を付けるよう求められます。*例: ニューヨークのトップ3のおすすめ*。
* クリック`Save` ボタン。

## リスト項目を追加する

検索結果から任意のリストにリスト項目を追加できます。

* へ移動[検索](/studio/search)
* リストに追加したいものを検索します。
* 見つけたら、`Actions`検索結果カードの左下にあるリンク。
* クリック `Add to...`次に、この在庫を追加するキュレーション リストを選択します。
* キュレーション リスト ページに戻り、アイテムがリストに追加されたことを確認します。

## リスト項目の順序を変更する

<figure>
  <CloudinaryImage width="350" height="183" resize="scale" quality="best" alt="キュレーションされたリスト項目を並べ替える" src="academy/reorder-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*キュレーションされたリスト項目を並べ替える*</figcaption>
</figure>


リスト内の各項目の左側にある縦のドットをクリックしてドラッグすることで、リスト項目を並べ替えることができます。

## リスト項目をコピーする

<figure>
  <CloudinaryImage width="450" height="178" resize="scale" quality="best" alt="キュレーションリスト項目にコピー" src="academy/copy-to-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*キュレーションされたリスト項目をコピー*</figcaption>
</figure>


コピー アイコンをクリックし、アイテムをコピーするリストを選択すると、リスト アイテムをコピーできます。

## リスト項目を移動する

<figure>
  <CloudinaryImage width="450" height="173" resize="scale" quality="best" alt="キュレーションされたリスト項目を移動する" src="academy/move-to-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*キュレーションされたリスト項目を移動する*</figcaption>
</figure>


リスト項目を移動するには、コピー アイコンをクリックし、項目を移動先のリストを選択します。

## リスト項目を削除する

<figure>
  <CloudinaryImage width="350" height="216" resize="scale" quality="best" alt="キュレーションされたリスト項目を削除する" src="academy/remove-curated-list-item" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*キュレーションされたリスト項目を削除する*</figcaption>
</figure>


ゴミ箱アイコンをクリックして削除を確認すると、リスト項目を削除できます。

## リストアクション

<figure>
  <CloudinaryImage width="350" height="183" resize="scale" quality="best" alt="キュレーションリストアクション" src="academy/curated-list-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*キュレーションリストアクション*</figcaption>
</figure>


すべてのリストには、実行できるアクションがいくつかあります。

* リストを[地図](/studio/maps)。
* リストを[グリッド](/studio/grids)。
* リストを削除する（*お気に入りを除く*）。

## API

キュレーションリストを管理したい開発者は、[開発者 > API > 閲覧](/developers/apis/#browse-api)。
