共有可能なリンクはWinkのコア機能であり、インターネット上で最もシンプルな方法、すなわちハイパーリンクを通じて、在庫と価格情報を動的に伝える必要性から生まれました。

<h4 align="center">[https://trvl.as/3xWCH](https://trvl.as/3xWCH)</h4>

上記は当社のリンクの一例です。短くて持ち運びやすく…覚えられるほど短いです。この短いリンクには以下のような**膨大な**情報が詰まっています：

- あなたのアプリID。
- あなたのアフィリエイトID。
- ユーザーに表示する画像とテキスト。
- 見た目のスタイル（*テーマ、色など*）。
- 価格照会に使用する旅程（*大人2名、1泊*）。
- 価格表示に使う通貨。
- テキスト表示に使う言語。

この記事の残りの部分では、リンクの作成、カスタマイズ、共有方法を説明します。

そのボタンをクリックすると、共有可能なリンクのフォームページにリダイレクトされ、リンクのカスタマイズを開始できます。

## リンクをカスタマイズする

フォームでは以下の方法でリンクをカスタマイズできます：

<Steps>
 1. リンクの内容を覚えておくための名前を付けます。
 2. このリンクに適用したい[カスタマイズ](/studio/customization)を選択します。
 3. [ウェブクローラー](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)が使用するキーワードをカンマ区切りで追加します。
 4. 任意でXアカウントのハンドル名やFacebookアプリIDを入力します。
 5. ユーザーに表示したい言語でタイトルと説明を追加します。
 6. FacebookやLinkedInなどのプラットフォームでリンクを共有したときに表示される画像を1枚以上選択します。
 7. 選択した画像の上にブランドデザインを追加するテーマ／オーバーレイを選びます。*注：これにより画像の上に価格が表示されます。*
 8. `保存`ボタンをクリックして続行します。
 </Steps>

上記はリンクに利用可能なすべての操作を示す画像です：

- **プレビュー** リンクの設定内容とユーザーにどのように見えるかを簡単に確認できます。
- **更新** リンクの設定を更新します。
- **WinkLinksに追加** リンクをあなたのWinkLinksアカウントに追加します。
- **コピー** リンクをクリップボードにコピーし、どこでも簡単に共有できます（*[WinkLinks](/link-manager/wink-links)を含む*）。
- **共有** 新しいウィンドウが開き、多くのソーシャルネットワークやメッセージングプラットフォームで共有するオプションが表示されます。
- **QRコード** IGなど画像中心のプラットフォームで共有できるQRコードを生成します。
- **埋め込み** このリンクを[広告バナー](/developers/web-components/#content-loader)としてウェブサイトに埋め込む方法を表示します。
- **WordPressで使用** 当社の[WordPressプラグイン](/developers/wordpress)を使ってこのリンクをウェブサイトに埋め込む方法を表示します。

以下でこれらのオプションのいくつかを詳しく説明します。

リンクの代わりにQRコードを共有したい場合は、`QRコード`ボタンをクリックすると、新しいウィンドウにQRコードが表示されます。

QRコードの保存方法：

- ノートパソコンの場合は、画像を右クリックして`画像を保存`を選択します。
- 携帯電話の場合は：
    - 画像の上で指を長押しし、画像を写真に保存するオプションが表示されるまで待ちます。
    - QRコードのスクリーンショットを撮り、トリミングして写真に保存します。

### 埋め込み

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

以下は、広告バナーとしてリンクをサイトに埋め込む方法です：

- 3行目はWinkのスタイルをサイトに埋め込む方法を示しています。
- 6〜9行目は[ wink-content-loader ](/developers/web-components/#content-loader) Webコンポーネントを使い、コードに対応する広告バナーを取得する方法を示しています。
- 11行目は当社のJavascriptをサイトに埋め込む方法を示しています。
- 13行目は[ wink-app-loader ](/developers/web-components/#application-loader) Webコンポーネントを使い、ページレベルの設定を取得する方法を示しています。
