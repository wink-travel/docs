代理店とは、代表の目的であなたのアカウントに割り当てることができる通常のWinkアカウントです。以下は代理店に代表してもらう理由の一例です。

- 第三者によるあなたのWinkアカウントの全管理または部分管理。
- タレントが既に代理店と契約しており、Winkを通じて入金される資金の財務面をWinkに任せたい場合。
- タレント代理店がすべてのオンライン作業を行い、Wink上のすべてのアカウントを管理している場合。

## 代理店の割り当て

代理店をあなたのアカウントに割り当てる方法は以下の通りです。

<Steps>
 1. [Wink Studio](https://studio.wink.travel)でユーザーアカウントにログインします。
 2. 作業したいアフィリエイトアカウントを選択します。
 3. メインナビバーの`Account > Managers`をクリックします。
 4. `Agency`セクションで、あなたを代表してほしい代理店の名前を入力します。
 5. ドロップダウンから代理店手数料を選択します。
 5. 必要に応じて有効期限ルールを追加します。
 7. `Set agency`ボタンをクリックして続行します。
 </Steps>

 :::note
代理店を割り当てるには、そのアカウントの所有者である必要があります。
 :::

 :::tip[例]
代理店手数料はアフィリエイトコミッションから算出されます。

予約 = $100、コミッション = 10%、代理店手数料 = 10% => 100 * 0.1 * 0.1 = $1
 :::

## 代理店の解除

代理店をあなたのアカウントから解除する方法は以下の通りです。

<Steps>
 1. [Wink Studio](https://studio.wink.travel)でユーザーアカウントにログインします。
 2. 作業したいアフィリエイトアカウントを選択します。
 3. メインナビバーの`Account > Managers`をクリックします。
 4. `Agency`セクションで、代理店ドロップダウンの右側にある<Badge text="x" variant="danger" size="small" />アイコンをクリックします。
 5. `Set agency`ボタンをクリックして続行します。
 </Steps>


## API

APIを使って代理店を管理したい開発者は、[Developers > API > Affiliate](/ja/developers/apis/#affiliate-api)をご覧ください。
