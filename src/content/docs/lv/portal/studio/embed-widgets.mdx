Studio nodrošina vairākus iegultus logrīkus, kurus vari pievienot savai vietnei, lai veicinātu rezervācijas.

Lai piekļūtu iegultajiem logrīkiem, dodies uz `Rīki > Dažādi` no sānu izvēlnes.

## Pieejamie logrīki

| Logrīks | Apraksts |
| -- | -- |
| **Kontu poga** | Zīmola poga, kas saista ar tavu filiāles lapu |
| **Maršruta poga** | Poga, kas atver maršruta izvēles rīku |
| **Maršruta forma** | Iekļauta datuma un viesu izvēles forma |
| **Meklēšanas poga** | Poga, kas atver īpašumu meklēšanu |
| **Meklēšanas poga** | Poga, kas ļauj meklēt konkrētu inventāru |

## Kā iegult

Katrs logrīks nodrošina koda fragmentu, ko vari kopēt un ielīmēt savas vietnes HTML. Fragmentā ir iekļauts tavs **Klienta ID** un **Pielāgošanas ID**, lai logrīks automātiski būtu saistīts ar tavu filiāles kontu.

:::note
Ja izmanto WordPress, mēs piedāvājam arī WordPress spraudni ērtākai integrācijai.
:::
