Wink lietotājs var izvēlēties ļaut jums pārvaldīt vienu no viņa īpašuma, partnera vai ceļojumu aģenta kontiem. Kad tas notiks, jūs saņemsiet e-pastu ar uzaicinājuma pieprasījumu.

import { Steps } from '@astrojs/starlight/components';

Lai skatītu savus pašreizējos Wink ielūgumus, veiciet tālāk norādītās darbības.

<Steps>
  1. Jums jāpiesakās kādā no mūsu tīmekļa vietnēm.
  2. Noklikšķiniet uz sava profila ikonas augšējā labajā stūrī, un parādīsies izvēlne.
  3. Noklikšķiniet uz `Invites` saite.
  4. Mijiedarbojieties ar saviem ielūgumiem, izmantojot kādu no šiem veidiem: `Accepting` vai `Rejecting` viņiem.
</Steps>
