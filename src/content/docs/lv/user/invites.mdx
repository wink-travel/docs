Wink lietotājs var izvēlēties ļaut jums pārvaldīt vienu no viņa īpašuma, filiāles vai ceļojumu aģenta kontiem. Kad tas notiek, jūs saņemsiet e-pastu ar ielūguma pieprasījumu.

import { Steps } from '@astrojs/starlight/components';

Lai redzētu savus pašreizējos Wink ielūgumus, izpildiet šīs darbības:

<Steps>
1. Jums jābūt pieslēgtam vienā no mūsu vietnēm.
2. Noklikšķiniet uz sava profila ikonas augšējā labajā stūrī, un parādīsies izvēlne.
3. Noklikšķiniet uz saites `Ielūgumi`.
4. Darbojieties ar saviem ielūgumiem, tos vai nu `Pieņemot`, vai `Noraidot`.
</Steps>
