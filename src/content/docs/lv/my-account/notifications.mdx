Reizēm mēs vēlamies ar jums sazināties. Kad tas notiks, augšējā labajā stūrī esošā zvana ikona mainīsies.
Lai izlasītu, ko esam nosūtījuši:

<Steps>
1. Noklikšķiniet uz `Konts > Paziņojumi` galvenajā navigācijas joslā.
2. Jūs redzēsiet jaunus un iepriekšējos paziņojumus.
3. Izvēlieties `Atzīmēt kā izlasītu` vai `Dzēst` paziņojumus, kad tos būsiet izlasījis.
</Steps>

## API

Izstrādātāji, kuri vēlas pārvaldīt `Affiliate`, var doties uz [Izstrādātāji > API > Affiliate](/lv/developers/apis/#affiliate-api).
