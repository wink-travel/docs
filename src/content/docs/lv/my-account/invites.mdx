Wink lietotājs var izvēlēties ļaut jums pārvaldīt vienu no viņa īpašuma, filiāles vai ceļojumu aģenta kontiem. Kad tas notiek, jūs saņemsiet e-pastu ar ielūguma pieprasījumu.

import { Steps } from '@astrojs/starlight/components';

Lai redzētu savus pašreizējos Wink ielūgumus, izpildiet šīs darbības:

<Steps>
 1. Piesakieties [my.wink.travel](https://my.wink.travel).
 2. Kreisajā sānjoslā noklikšķiniet uz `Invites`.
 3. Jūs redzēsiet visu pašreizējo ielūgumu sarakstu.
 4. Šajā sadaļā tiks uzskaitītas visas jūsu pieteikšanās.
 5. Izvēlieties `Accept` vai `Reject` ielūgumu.
 6. Ja pieņēmāt ielūgumu, konts tiks parādīts sadaļā `Accounts`.
</Steps>
