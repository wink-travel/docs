Ghost iebūvētā **Code Injection** funkcija ļauj pievienot skriptus globālajā `<head>` un `</body>` jūsu publikācijā. Visi Ghost plāni atbalsta koda injekciju, padarot vienkāršu Wink komponentu iegulšanu jebkurā lapā vai ierakstā.

## Ielādēt Wink skriptus visā vietnē

<Steps>
1. Ghost Admin panelī dodieties uz **Settings → Code Injection**.
2. Laukā **Site Header** pievienojiet stila lapu:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. Laukā **Site Footer** pievienojiet skriptu un lietotnes ielādētāju (aizvietojiet `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Noklikšķiniet uz **Save**.
</Steps>

:::tip
Jūsu Client ID ir pieejams jūsu Wink kontā sadaļā [Applications](/lv/portal/settings/applications).
:::

## Pievienot komponentu ierakstam vai lapai

Ghost redaktors atbalsta **HTML karti**, kas ļauj ievietot neapstrādātu HTML jebkurā jūsu saturā.

<Steps>
1. Atveriet ierakstu vai lapu Ghost redaktorā.
2. Ierakstiet `/`, lai atvērtu karšu izvēlni, un izvēlieties **HTML**.
3. Ielīmējiet komponenta HTML, piemēram:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Noklikšķiniet ārpus kartes, lai priekšskatītu, pēc tam publicējiet.
</Steps>

## Koda injekcija pa lapām

Komponentam, kas jāparāda tikai vienā lapā, Ghost atbalsta arī koda injekciju pa lapām:

<Steps>
1. Atveriet ieraksta vai lapas iestatījumus (zobrata ikona redaktorā).
2. Ritiniet līdz **Code Injection**.
3. Pievienojiet komponenta tagu laukā **Footer** (globālais app-loader jābūt tikai vietnes kājenē).
4. Saglabājiet.
</Steps>

## Pieejamie komponenti

| Komponents | HTML tags | Mērķis |
|---|---|---|
| Saturs ielādētājs | `<wink-content-loader>` | Rāda viesnīcu kartītes, režģus, kartes |
| Meklēšana | `<wink-lookup>` | Meklēšanas josla galamērķiem |
| Meklēšanas poga | `<wink-search-button>` | Atver maršruta izvēlni |
| Konta poga | `<wink-account-button>` | Pieslēgšanās / lietotāja izvēlne |
| Maršruta poga | `<wink-itinerary-button>` | Rāda pašreizējo maršrutu |
| Maršruta izvēlne | `<wink-itinerary-picker>` | Pilna maršruta forma |
| Iepirkumu grozs | `<wink-shopping-cart-button>` | Groza kopsavilkuma poga |

Skatiet [Web Components](/lv/integrations/web-components) pilnu atribūtu sarakstu.
