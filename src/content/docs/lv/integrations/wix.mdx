The **Wink Wix App** adds Wink web components to your Wix site as native widgets you can drag and drop in the Wix Editor. No coding required for site owners.

<LinkCard
  title="wink-travel/wink-wix-app on GitHub"

/>

## Instalēšana

<Steps>
1. Dodieties uz [Wix App Market](https://www.wix.com/app-market) un meklējiet **Wink Travel**.
2. Noklikšķiniet uz **Pievienot vietnei** un sekojiet norādījumiem, lai autorizētu lietotni.
3. Kad instalēts, atveriet **Dashboard → Wink Settings** un ievadiet savu **Client ID** un **Client Secret**.
   Lai iegūtu savus akreditācijas datus, dodieties uz [Applications](/lv/portal/settings/applications).
4. Noklikšķiniet uz **Saglabāt** un pēc tam uz **Testēt savienojumu**, lai apstiprinātu, ka Wink ir savienots.
</Steps>

## Komponenta pievienošana lapai

<Steps>
1. Atveriet **Wix Editor** un dodieties uz lapu, kurā vēlaties pievienot Wink saturu.
2. Noklikšķiniet uz **Pievienot (+) → Lietotnes → Wink Travel**, lai redzētu visus pieejamos logrīkus.
3. Velciet logrīku uz lapu — piemēram, **Wink Content**, lai parādītu viesnīcu režģi.
4. Logrīka iestatījumu panelī izvēlieties **Izvietojuma veidu** un ielīmējiet **Izvietojuma ID** no Wink Studio.
5. Noklikšķiniet uz **Publicēt**, lai izmaiņas stātos spēkā.
</Steps>

## Pieejamie logrīki

| Logrīks | Mērķis |
|---|---|
| **Wink Content** | Parāda viesnīcu kartītes, rangotus režģus, kartes un vairāk |
| **Wink Lookup** | Meklēšanas josla galamērķiem un viesnīcām |
| **Wink Search** | Ikonas poga, kas atver maršruta izvēlētāju |
| **Wink Account** | Pieslēgšanās / lietotāja konta poga |
| **Wink Itinerary** | Maršruta poga, kas rāda pašreizējo ceļojumu |
| **Wink Cart** | Iepirkumu groza kopsavilkuma poga |
