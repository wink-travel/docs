The **Wink Joomla Extension** ir pakotne, kas instalē sistēmas spraudni (ielādē CDN skriptus), satura spraudni (apstrādā īskodus rakstos) un komponenti (administrēšanas iestatījumu lapa).

## Prasības

- Joomla 4 vai 5
- PHP 8.1+

## Instalēšana

<Steps>
1. Lejupielādējiet jaunāko `pkg_wink-x.x.x.zip` no [GitHub izlaidumu lapas](https://github.com/wink-travel/wink-joomla-extension/releases).
2. Joomla administrācijā dodieties uz **Extensions → Manage → Install**.
3. Augšupielādējiet zip failu un noklikšķiniet uz **Upload & Install**.
4. Abi spraudņi tiek automātiski aktivizēti. Dodieties uz **Components → Wink Settings**.
5. Ievadiet savu **Client ID** un **Client Secret** un noklikšķiniet uz **Save**.
   Lai iegūtu savus akreditācijas datus, dodieties uz [Applications](/lv/portal/settings/applications).
</Steps>

## Komponentu iegulšana rakstos

Izmantojiet īskodu stilā tagus jebkur raksta saturā:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Satura spraudnis šos tagus aizstāj ar pareizu `<wink-*>` HTML izkārtojumu renderēšanas laikā. Sistēmas spraudnis automātiski ielādē CDN skriptus un ievieto `<wink-app-loader>` lapas kājenē.

## Pieejamie īskodi

| Īskods | Izvade | Apraksts |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Viesnīcu kartītes, režģi, kartes |
| `{wink-lookup}` | `<wink-lookup>` | Galamērķa meklēšanas josla |
| `{wink-search}` | `<wink-search-button>` | Maršruta izvēles poga |
| `{wink-account}` | `<wink-account-button>` | Pieslēgšanās / lietotāja konts |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Maršruta statusa poga |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Groza poga |
