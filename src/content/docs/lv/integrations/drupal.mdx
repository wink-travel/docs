The **Wink Drupal Module** pievieno Wink ceļojumu inventāra komponentus jūsu Drupal vietnei kā vietējos blokus. Redaktori var tos novietot jebkurā vietā izkārtojumā, izmantojot standarta bloku sistēmu.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Avota kods, izstrādātāja uzstādīšanas ceļvedis un ieguldījuma norādījumi."

## Prasības

- Drupal 10 vai 11
- PHP 8.1+
- Composer

## Instalēšana

<Steps>
1. Instalējiet, izmantojot Composer:
   ```bash
   composer require drupal/wink
   ```
2. Ieslēdziet moduli:
   ```bash
   drush en wink -y
   ```
3. Dodieties uz **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Ievadiet savu **Client ID** un **Client Secret**, izvēlieties vidi un noklikšķiniet uz **Save**.
   Lai iegūtu savus akreditācijas datus, dodieties uz [Applications](/lv/portal/settings/applications).
</Steps>

## Bloka pievienošana lapai

<Steps>
1. Dodieties uz **Admin → Structure → Block Layout**.
2. Noklikšķiniet uz **Place block** reģionā, kur vēlaties ievietot Wink saturu.
3. Meklējiet **Wink**, lai atrastu pieejamos blokus:
   - **Wink Content** — attēlo viesnīcu karti, režģi vai karti
   - **Wink Lookup** — galamērķa meklēšanas josla
   - **Wink Search** — maršruta izvēles poga
   - **Wink Account** — pieslēgšanās poga
   - **Wink Itinerary** — maršruta statusa poga
   - **Wink Cart** — iepirkumu groza poga
4. Blokam **Wink Content** konfigurējiet **Layout Type** un **Layout ID** bloka iestatījumos.
5. Saglabājiet bloka konfigurāciju.
</Steps>

## Pieejamie bloki

| Bloks | Mērķis |
|---|---|
| **Wink Content** | Attēlo viesnīcu kartes, sakārtotus režģus, kartes un citus elementus |
| **Wink Lookup** | Meklēšanas josla galamērķiem un viesnīcām |
| **Wink Search** | Ikonas poga, kas atver maršruta izvēles rīku |
| **Wink Account** | Pieslēgšanās / lietotāja konta poga |
| **Wink Itinerary** | Maršruta poga, kas rāda pašreizējo ceļojumu |
| **Wink Cart** | Iepirkumu groza kopsavilkuma poga |
