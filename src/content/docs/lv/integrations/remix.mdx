Remix ir pilna kaudze ietvars, kas pēc noklusējuma veic servera renderēšanu. Ielādējiet Wink CDN tikai klienta pusē, izmantojot `useEffect` saknes komponentē.

<LinkCard
  title="@wink/elements npm package"
  href="/integrations/npm"
  description="Instalējiet un ielādējiet CDN pakotni ar TypeScript tipiem."
/>

## Instalēšana

```bash
npm install @wink/elements
```

## Ielāde app/root.tsx

`useEffect` darbojas tikai pārlūkprogrammā, nekad servera puses renderēšanas laikā:

```tsx
// app/root.tsx
import { useEffect } from 'react';
import {
  Links, Meta, Outlet, Scripts, ScrollRestoration,
} from '@remix-run/react';
import { load } from '@wink/elements';

export default function App() {
  useEffect(() => {
    load({ clientId: import.meta.env.VITE_WINK_CLIENT_ID });
  }, []);

  return (
    <html lang="en">
      <head>
        <Meta />
        <Links />
      </head>
      <body>
        <Outlet />
        <ScrollRestoration />
        <Scripts />
      </body>
    </html>
  );
}
```

## Vides mainīgie

Remix (balstīts uz Vite, v2.8+) atbalsta `import.meta.env` pārlūkprogrammā pieejamiem mainīgajiem. Lai padarītu mainīgo pieejamu pārlūkprogrammas paketē, pievienojiet prefiksu `VITE_`:

```
# .env
VITE_WINK_CLIENT_ID=your-client-id
```

Vecākām Remix konfigurācijām, kas izmanto Node.js adapteri, klienta ID atgrieziet no saknes ielādētāja un nolasiet to komponentē:

```tsx
// app/root.tsx
import { json } from '@remix-run/node';
import { useLoaderData } from '@remix-run/react';

export async function loader() {
  return json({ winkClientId: process.env.WINK_CLIENT_ID });
}

export default function App() {
  const { winkClientId } = useLoaderData<typeof loader>();

  useEffect(() => {
    if (winkClientId) {
      load({ clientId: winkClientId });
    }
  }, [winkClientId]);

  // ...
}
```

## Lietošana maršrutos

```tsx
// app/routes/hotels.tsx
export default function HotelsRoute() {
  return (
    <main>
      <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
    </main>
  );
}
```

<Aside type="caution" title="Hidratācijas neatbilstība">
Remix hidratē servera renderēto HTML klienta pusē. Ja `<wink-*>` elements tiek renderēts serverī, pārlūkprogramma īslaicīgi var parādīt neatpazītu pielāgotu elementu, pirms CDN skripts to reģistrē. Tas ir tikai vizuāls efekts — elements tiek pareizi atjaunināts, tiklīdz skripts ielādējas.
</Aside>
