**Wink Shopify App** ļauj tūrisma operatoriem, pieredzes pārdevējiem un viesmīlības tirgotājiem tieši savā Shopify veikalā rādīt tiešo Wink inventāru — blakus produktiem, galvenajās lapās vai jebkurā vietā tēmā.

## Instalēšana

<Steps>
1. Dodieties uz [Shopify App Store](https://apps.shopify.com) un meklējiet **Wink Travel**.
2. Noklikšķiniet uz **Add app** un sekojiet OAuth procesam, lai autorizētu.
3. Wink lietotnes iestatījumos ievadiet savu **Client ID** un **Client Secret**.
   Lai iegūtu savus akreditācijas datus, dodieties uz [Applications](/lv/portal/settings/applications).
4. Noklikšķiniet uz **Verify & Save**. Veiksmīga ziņa apstiprina savienojumu.
</Steps>

## Aktivizējiet App Embed Block

App Embed Block automātiski ievieto Wink skriptu un stilu lapā — nav nepieciešams rediģēt tēmas kodu.

<Steps>
1. Shopify administrācijā dodieties uz **Online Store → Themes → Customize**.
2. Kreisajā panelī noklikšķiniet uz **App embeds**.
3. Ieslēdziet **Wink Travel**.
4. Noklikšķiniet uz **Save**.
</Steps>

:::note
App Embed Block nolasa jūsu Client ID no Shopify metafield, kas iestatīts instalācijas laikā. Jūsu Client Secret nekad netiek atklāts pārlūkprogrammā.
:::

## Pievienojiet komponentu lapai

<Steps>
1. Tēmas redaktorā atveriet lapu vai sadaļu, kur vēlaties ievietot Wink saturu.
2. Noklikšķiniet uz **Add block** un izvēlieties no Wink blokiem:
   - **Wink Content** — viesnīcu kartītes, režģi vai kartes
   - **Wink Lookup** — galamērķu meklēšanas josla
   - **Wink Search** — maršruta izvēles poga
   - **Wink Account** — pieslēgšanās poga
   - **Wink Itinerary** — maršruta statusa poga
   - **Wink Cart** — iepirkumu groza poga
3. Pie **Wink Content** izvēlieties **Layout Type** un ievadiet **Layout ID** no Wink Studio.
4. Noklikšķiniet uz **Save**.
</Steps>

## Pieejamie tēmas bloki

| Bloks | Mērķis |
|---|---|
| **Wink Content** | Rāda viesnīcu kartītes, sakārtotus režģus, kartes un vairāk |
| **Wink Lookup** | Meklēšanas josla galamērķiem un viesnīcām |
| **Wink Search** | Ikonas poga, kas atver maršruta izvēli |
| **Wink Account** | Pieslēgšanās / lietotāja konta poga |
| **Wink Itinerary** | Maršruta poga, kas rāda pašreizējo ceļojumu |
| **Wink Cart** | Iepirkumu groza kopsavilkuma poga |
