**Wink TYPO3 Extension** integrē Wink ceļojumu inventāru TYPO3 13 kā vietējos satura elementus. Redaktori pievieno Wink komponentus lapām, izmantojot standarta TYPO3 lapu redaktoru — nav nepieciešamas veidņu izmaiņas.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"

  description="Avota kods, izstrādātāja uzstādīšanas ceļvedis un ieguldījumu norādījumi."
/>

## Prasības

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instalēšana

<Steps>
1. Instalē ar Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivizē paplašinājumu:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. TYPO3 aizmugurējā panelī dodies uz **Admin Tools → Settings → Extension Configuration → wink**.
4. Ievadi savu **Client ID**, **Client Secret** un izvēlies vidi. Saglabā.
   Dodies uz [Applications](/lv/portal/settings/applications), lai iegūtu savus akreditācijas datus.
5. Pievieno statisko TypoScript veidni savam saknes veidnim:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Pievienot satura elementu lapai

<Steps>
1. Atver lapu **Page** modulī.
2. Noklikšķini uz **+** pogas satura zonā, lai pievienotu jaunu elementu.
3. Izvēlies vienu no **Wink Travel** satura elementiem:
   - **Wink Content** — viesnīcu kartītes, sakārtotas režģi, kartes
   - **Wink Lookup** — galamērķa meklēšanas josla
   - **Wink Search** — maršruta izvēles poga
   - **Wink Account** — pieslēgšanās poga
   - **Wink Itinerary** — maršruta statusa poga
   - **Wink Cart** — iepirkumu groza poga
4. **Wink Content** gadījumā izmanto FlexForm laukus, lai izvēlētos **Layout Type** un ievadītu **Layout ID**.
5. Saglabā un publicē lapu.
</Steps>

## Pieejamie satura elementi

| Elements | Mērķis |
|---|---|
| **Wink Content** | Viesnīcu kartītes, sakārtoti režģi, kartes un citi |
| **Wink Lookup** | Galamērķa meklēšanas josla |
| **Wink Search** | Ikonas poga, kas atver maršruta izvēli |
| **Wink Account** | Pieslēgšanās / lietotāja konta poga |
| **Wink Itinerary** | Maršruta poga, kas rāda pašreizējo ceļojumu |
| **Wink Cart** | Iepirkumu groza kopsavilkuma poga |
