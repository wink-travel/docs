# Iestatījumi

Kad pirmo reizi sākat darbu ar Social, mēs jautāsim, kā vēlaties, lai mēs veidotu jūsu saturu. Šīs opcijas ir pieejamas jebkurā laikā, lai jūs varētu tās viegli mainīt vēlāk.

Dodieties uz `Preferences` no sānu izvēlnes, lai pārvaldītu savus iestatījumus.

Preferences sadaļā ir jūsu pamata satura ģenerēšanas iestatījumi:

1. **Preferred customization** Izvēlieties, kuru pielāgojumu izmantot satura ģenerēšanai. Pielāgojumi kontrolē tēmas krāsas, noklusējuma maršrutu, valūtu, valodu un daudz ko citu. Jūs varat izveidot pielāgojumus savā kontā vietnē [my.wink.travel](https://my.wink.travel).
2. **Auto-approve** Ja iespējots, ģenerētie ieraksti tiek publicēti automātiski bez jūsu apstiprinājuma. Ieslēdziet šo, kad esat pārliecināts par mūsu radītā satura kvalitāti.
3. **Approver name** Personas vai komandas vārds, kas saņems apstiprinājuma pieprasījuma e-pastus.
4. **Approver e-mail** E-pasta adrese, uz kuru tiks nosūtīti apstiprinājuma pieprasījumi par gaidāmajiem ierakstiem.

Social piedāvā daudz satura veidotāju izvēlei. Katrs veidotājs koncentrējas uz vienu konkrētu veidu, kā ar AI palīdzību radīt unikālu saturu jūsu īpašumam. Katram satura veidotājam ir **Weight** atribūts — lielāka vērtība palielina iespēju, ka šis veidotājs tiks izvēlēts, ģenerējot ierakstus.

Šeit ir visi pieejamie satura veidotāji:

| Veidotājs | Apraksts |
| -- | -- |
| **Destination** | Izmanto jūsu atrašanās vietu, lai radītu noderīgu tekstu par apkārtni |
| **Room type** | Izveido rezervējamu ierakstu ar vienu no jūsu istabu tipiem, izmantojot AI ģenerētu tekstu un jūsu istabu attēlus |
| **Activity** | Izveido rezervējamu ierakstu ar vienu no jūsu aktivitātēm |
| **Add-On** | Izveido rezervējamu ierakstu ar vienu no jūsu papildinājumiem |
| **Attraction** | Izveido rezervējamu ierakstu ar vienu no jūsu apskates objektiem |
| **Place** | Izveido rezervējamu ierakstu ar vienu no jūsu vietām |
| **Meeting room** | Izveido rezervējamu ierakstu ar vienu no jūsu sapulču telpām |
| **Restaurant** | Izveido rezervējamu ierakstu ar vienu no jūsu restorāniem |
| **Spa** | Izveido rezervējamu ierakstu ar vienu no jūsu spa |
| **Property** | Izveido rezervējamu ierakstu ar jūsu labākās cenas istabu, izmantojot AI ģenerētu tekstu un jūsu īpašuma attēlus |
| **User sentiment** | Apkopo viesu atsauksmes no interneta un veido emocionāli piesaistošu ierakstu, kas izceļ, kas viesiem patīk jūsu īpašumā |
| **Funny** | Izveido vieglus, izklaidējošus ierakstus, lai jūsu plūsma būtu interesanta |

:::tip[Pro padoms]
Pārliecinieties, ka jums ir nepieciešamais inventāra veids izvēlētajiem satura veidotājiem. Piemēram, ja izvēlaties "Restaurant", jūsu īpašumā jābūt vismaz vienam restorānam.
:::

Tona balss nosaka valodas stilu, ko AI izmanto satura ģenerēšanai. Izvēlieties toņus, kas atbilst jūsu zīmola personībai. Tāpat kā satura veidotājiem, katram tonim ir **Weight**, lai kontrolētu, cik bieži tas tiek izmantots.

Pieejamie toņi:

| Tona balss | Stils |
| -- | -- |
| **Neutral** | Mierīgs, autoritatīvs, faktisks. Bez pārspīlējumiem un jokiem — tikai uzticama, labi strukturēta informācija |
| **Friendly** | Silts, pieejams, ar smaidu tekstā, ar ikdienišķiem saīsinājumiem un reizēm emocijzīmēm |
| **Luxury** | Izsmalcināts, rafinēts. Vārdi kā "izcils" un "pēc pasūtījuma", kas lasītājam rada ekskluzivitātes sajūtu |
| **Foodie** | Sensuāls, mutē saskrienošs apraksts. "Ēdiena pornogrāfijas" valoda, kas liek lasītājiem izsalkt |
| **Adventurous** | Drosmīgs, nebaidīgs, pētnieka gars. Dzīvīgas maņu apraksti, kas padara visu episku |
| **Family** | Veselīgs, bērniem draudzīgs aizrautīgums. Silta vecāku enerģija par drošām un jautrām pieredzēm |
| **Romantic** | Poētisks, ilgoties rosinošs, sirdi aizkustinošs. Saulrieti, maiga valoda un mīlestība |
| **Humorous** | Sarkastisks, ātri joki, asprātīga vārdu spēle. Lasa kā stand-up šovs |
| **Eco** | Patiesīgs un taisnīgs. Svina zaļās alternatīvas un ilgtspējīgas izvēles |
| **Local insider** | Ļoti specifiski vietējie atsauces, brīdina par tūristu slazdiem, izmanto vietējo žargonu |
| **Inclusive** | Laipns, svinīgs un apstiprinošs. Padara ikvienu redzamu un novērtētu |
| **Mystical** | Eterisks, garīgs, kosmiski patiesības. Maigas aicinājums paskatīties sevī |
| **Futuristic** | Gluds, nākotnes domājošs. Mīl tādus terminus kā "paradigmas maiņa" un optimistiskas nākotnes vīzijas |
| **Budget** | Izdomas bagāts, taupīgs, lepni ekonomisks. Svina pieejamus atklājumus un bezmaksas pieredzes |

## Post types

Ierakstu veidi nosaka ģenerētā satura mērķi un struktūru. Katram ierakstu veidam ir **Weight**, lai kontrolētu, cik bieži tas tiek izmantots.

| Ieraksta veids | Apraksts |
| -- | -- |
| **Transactional** | Skaidrs, kodolīgs, uz konversiju orientēts. Veicina rezervācijas un tiešas darbības ar faktiem, cenām un nākamajiem soļiem |
| **Awareness** | Zīmola veidošanas saturs, kas paredzēts, lai iepazīstinātu ar jūsu īpašumu un palielinātu redzamību potenciālo viesu vidū |
| **Sentiment** | Emocionāli rezonējošs saturs, kas rada pozitīvas asociācijas un veido emocionālu saikni ar jūsu auditoriju |
