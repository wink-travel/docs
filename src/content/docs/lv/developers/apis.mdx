## Maksājumu API

Maksājumu galapunkti ir paredzēti izstrādātājiem, kuri vēlas iegādāties ceļojumu inventāru. To var izdarīt, izmantojot API kā reģistrēts ceļojumu aģents vai izmantojot mūsu API kopā ar mūsu PCI saderīgo maksājumu logrīku visām pārējām vienībām.
Maksājumu API nodrošina galapunktus filiālēm un viesnīcām, lai izsekotu rezervācijas, analītiku un līdzekļu pieejamību. Viņi var arī izvēlēties izņemt pieejamos līdzekļus uz savu bankas kontu.

  Uzziniet vairāk
</LinkButton>
