## Payment API

Maksājumu galapunkti ir paredzēti izstrādātājiem, kuri vēlas iegādāties ceļojumu inventāru. To var izdarīt, izmantojot API kā reģistrēts ceļojumu aģents vai izmantojot mūsu API kopā ar mūsu PCI atbilstošo maksājumu logrīku visām pārējām vienībām.
Payment API nodrošina galapunktus filiālēm un viesnīcām, lai izsekotu rezervācijas, analītiku un līdzekļu pieejamību. Viņi var arī izvēlēties izņemt pieejamos līdzekļus uz savu bankas kontu.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Uzzināt vairāk
</LinkButton>
