Wink integrējas ar vadošajiem kanālu pārvaldniekiem, īpašumu pārvaldības sistēmām (PMS) un centrālajām rezervāciju sistēmām (CRS), lai nodrošinātu jūsu cenu un pieejamības sinhronizāciju visos pārdošanas kanālos.

## Kā tas darbojas

Kanālu pārvaldnieku integrācijas nodrošina divvirzienu datu plūsmu starp jūsu īpašumu pārvaldības sistēmu un Wink:

1. **Cenu nosūtīšana** — Jūsu kanālu pārvaldnieks nosūta cenu atjauninājumus uz Wink. Kad jūs atjaunināt cenas savā PMS, šīs izmaiņas automātiski atspoguļojas Wink.
2. **Pieejamības sinhronizācija** — Jūsu kanālu pārvaldnieks nosūta pieejamības atjauninājumus uz Wink. Kad istabas tiek rezervētas vai atbrīvotas jūsu PMS, Wink krājumi tiek atjaunināti reāllaikā.
3. **Rezervāciju paziņojumi** — Kad rezervācija tiek veikta uz Wink, jūsu kanālu pārvaldnieks tiek informēts, lai rezervācija parādītos jūsu PMS.

## Atbalstītie kanālu pārvaldnieki

Pašlaik Wink integrējas ar šādām sistēmām:

| Kanālu pārvaldnieks | Apraksts |
|---|---|
| **Channex** | Globāla izplatīšanas un kanālu pārvaldības platforma. |
| **TravelClick** | Viesnīcu risinājumi un kanālu pārvaldība (Amadeus). |
| **Omnibees** | Izplatīšana un ieņēmumu pārvaldība viesnīcām. |
| **D-Edge** | Viesmīlības tehnoloģijas un centrālās rezervāciju risinājumi. |
| **Allotz** | Krājumu sadale un kanālu pārvaldība. |
| **Ezee / Yanolja** | Mākoņa PMS un kanālu pārvaldība, fokusējoties uz Āzijas tirgu. |
| **RateGain** | Cenu inteliģence un izplatīšanas tehnoloģija. |
| **RateTiger** | Reāllaika cenu un pieejamības izplatīšana. |
| **CloudBeds** | Mākoņa bāzēta PMS, kanālu pārvaldnieks un rezervāciju dzinējs. |
| **SiteMinder** | Vadošais viesnīcu kanālu pārvaldnieks un izplatīšanas platforma. |
| **Exely** | Cenu un pieejamības sinhronizācija viesnīcām. |

:::note
Ja jūsu kanālu pārvaldnieks nav sarakstā, sazinieties ar Wink komandu. Jaunas integrācijas var tikt pievienotas, balstoties uz OTA (Open Travel Alliance) protokola standartiem, ko atbalsta Wink.
:::

## Izstrādātājiem

Ja veidojat kanālu pārvaldnieka vai PMS integrāciju ar Wink, izmantojiet Channel Manager API galapunktus:

- **Cenu atjauninājumi** — Nosūtiet cenu izmaiņas uz Wink.
- **Pieejamības atjauninājumi** — Nosūtiet pieejamības izmaiņas uz Wink.
- **Rezervāciju iegūšana** — Iegūstiet rezervācijas no Wink savā sistēmā.

Channel Manager API seko OTA protokola standartiem (SOAP/XML), lai nodrošinātu saderību ar esošajām viesmīlības sistēmām.

## Webhook paziņojumi

Jūs varat abonēt kanālu pārvaldnieka webhook notikumus, lai saņemtu reāllaika paziņojumus:

- `channel-manager.update.rate` — Saņemts cenu atjauninājums.
- `channel-manager.update.availability` — Saņemts pieejamības atjauninājums.
- `channel-manager.update` — Vispārējs kanālu pārvaldnieka atjauninājums.

Skatiet [Webhook Events Catalog](/lv/developers/webhook-events#channel-manager-events) sīkākai informācijai.
