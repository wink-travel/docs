Pārējā šī raksta daļa soli pa solim parāda, kā izveidot, pielāgot un koplietot savas kartes.

## Izveidot karti

Kartes var izveidot divos veidos:

- **Viena marķiera karte**: Izveido karti no kaut kā, ko atradi, [meklējot](/lv/studio/search).
- **Daudzmarķieru karte**: Izveido karti no [kurēta saraksta](/lv/studio/curated-lists) vai [saglabātas meklēšanas](/lv/studio/saved-searches).

Augstāk redzama bilde no [Meklēšanas](/lv/studio/search), kas parāda dažas no darbībām, ko vari veikt ar saviem meklēšanas rezultātiem. Viena no šīm darbībām ir `Izveidot karti`.
Noklikšķini uz šīs pogas, un tā novirzīs tevi uz mūsu kartes veidlapas lapu, kur vari sākt pielāgot savu karti.

### Daudzmarķieru karte

Gan [kurētie saraksti](/lv/studio/curated-lists#list-actions), gan [saglabātās meklēšanas](/lv/studio/saved-searches#list-actions) satur darbību, kas ļauj izveidot karti. Šī karte saturēs karšu marķierus visam inventāram šajos sarakstos.

## Pielāgot karti

 Veidlapa ļauj pielāgot karti šādos veidos:

<Steps>
 1. Piešķir tai nosaukumu, lai atcerētos, par ko ir karte.
 2. Izvēlies, vai lietotājam atļaut pārvietoties pa karti.
 3. Iestatīt, cik augsta, pikseļos, vēlies, lai karte būtu.
 4. Izvēlies, vai lietotājam atļaut tuvināt un tālināt karti.
 5. Izvēlies kartes stilu, kas saskanēs ar vietni, kurā karte tiks iegultā.
 6. Izvēlies marķiera krāsu.
 7. Izvēlies sākotnējo kartes skatu, ko lietotāji redzēs vispirms. Pēc noklusējuma tas ir inventāra dzimtā seja.
 8. Vari pievienot savus marķierus, apļus, taisnstūrus un daudzstūrus, lai norādītu citas vietas kartē, kas varētu interesēt tavus lietotājus.
 9. Noklikšķini uz pogas `Saglabāt`, lai turpinātu.
</Steps>

Pēc kartes saglabāšanas tu tiks novirzīts uz savu karšu lapu, un tava karte tagad ir gatava dalīšanai ar pasauli.

Augstāk redzama bilde ar visām pieejamajām darbībām tavā kartē:

- **Atjaunināt** Atjaunina tavas kartes konfigurāciju.
- **Pievienot WinkLinks** Pievieno karti tavā WinkLinks kontā.
- **Iegult** Parāda, kā iegult šo karti kā [Karti](/lv/developers/web-components/#content-loader) savā vietnē.
- **Izmantot ar WordPress** Parāda, kā izmantot mūsu [WordPress spraudni](/lv/developers/wordpress), lai iegultu šo karti savā vietnē.

Tālāk mēs sīkāk apskatām dažas no šīm iespējām.

### Iegult

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Šeit ir, kā iegult savu karti savā vietnē:

- 3. rinda parāda, kā iegult Wink stilus savā vietnē.
- 6. līdz 9. rinda parāda, kā izmantot [wink-content-loader](/lv/developers/web-components/#content-loader) Web komponenti un norādīt tai ielādēt reklāmas baneri tavai kodam.
- 11. rinda parāda, kā iegult mūsu Javascript savā vietnē.
- 13. rinda parāda, kā iegult [wink-app-loader](/lv/developers/web-components/#application-loader) Web komponenti un norādīt tai ielādēt tavas lapas konfigurācijas preferences.

## API

Izstrādātāji, kas vēlas pārvaldīt kartes, var doties uz [Izstrādātāji > API > Inventārs](/lv/developers/apis/#inventory-api).
