Rezervēšanas dzinējs ir pieejams divās versijās, lai atbalstītu dažādās vajadzības, kas ir saistītas ar filiāli, īpašumu pārvaldības uzņēmumu un viesnīcu.

## Tiešsaistes ceļojumu aģentūra

Filiālei visjēdzīgāk ir izmantot tiešsaistes ceļojumu aģentūras versiju. Iemesls ir tāds, ka vietne ir izveidota, lai apmierinātu ceļotāju vajadzības, kuri vēlas apskatīt vairākus īpašumus.

Zemāk varat atrast šīs lietojumprogrammas sākumlapu.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Interneta rezervēšanas dzinējs

Viesnīcai vai īpašumu pārvaldības uzņēmumam visjēdzīgāk ir izmantot interneta rezervēšanas dzinēja versiju. Šāda veida lietotājs ir ieinteresēts izcelt vienu īpašumu vai vairākus pārvaldībā esošos īpašumus.

Zemāk varat atrast šīs lietojumprogrammas sākumlapu.

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE
</LinkButton>

## Pielāgošana

Viesnīcas un filiāles var pielāgot rezervēšanas dzinēja `izskatu un sajūtu`, pielāgojot krāsu paleti.

Izmēģiniet pielāgot savu tēmu mūsu izstrādātāju smilšu kastē zemāk.

## Konfigurācija

Lielākoties nav nepieciešams tieši koplietot nevienu no šīm lietojumprogrammām. Mēs esam izveidojuši dažādus veidus, kā koplietot ceļojumu inventāru, ko varat atklāt [Wink Studio](/studio/what-is-studio) vai caur [WinkLinks](/link-manager/wink-links).

Tomēr, ja vēlaties manuāli izveidot saiti uz kādu no šīm vietnēm, pārliecinieties, ka URL pievienojat savu klienta ID un pielāgošanas ID kā parametrus.

Savu klienta ID varat atrast šādi:

<Steps>
1. Piesakieties vai nu [Wink Extranet](/extranet/what-is-extranet), vai [Wink Studio](/studio/what-is-studio).
2. Noklikšķiniet uz sava profila ikonas augšējā labajā stūrī un izvēlieties [Applications](/developers/applications).
3. Izmantojiet lietojumprogrammu, kas saistīta ar Wink kontu, ar kuru vēlaties strādāt.
4. Nokopējiet klienta ID no šīs lietojumprogrammas un ielīmējiet to URL kā pieprasījuma parametru. Piemērs: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Savu pielāgošanas ID kā īpašumu varat atrast šādi:

<Steps>
1. Piesakieties [Wink Extranet](/extranet/what-is-extranet).
2. Izvēlieties īpašumu, ar kuru vēlaties strādāt.
3. No galvenās navigācijas joslas noklikšķiniet uz `Account > Branding`
4. Nokopējiet pielāgošanas ID un ielīmējiet to URL kā pieprasījuma parametru. Piemērs: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Savu pielāgošanas ID kā filiāli varat atrast šādi:

<Steps>
1. Piesakieties [Wink Studio](/studio/what-is-studio).
2. Izvēlieties kontu, ar kuru vēlaties strādāt.
3. No galvenās navigācijas joslas noklikšķiniet uz `Tools > Customization`
4. Nokopējiet pielāgošanas ID no pielāgošanas, ko vēlaties izmantot, un ielīmējiet to URL kā pieprasījuma parametru. Piemērs: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Rezultātā iegūstat URL, kas izskatās apmēram šādi:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Ja vēlaties tieši doties uz viesnīcas lapu, URL izskatītos apmēram šādi:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Ja neiekļaujat klienta un/vai pielāgošanas ID, Wink OTA izmantos noklusējuma konfigurāciju.
:::

## IBE

Ja esat īpašumu pārvaldības uzņēmums, vēlaties izveidot saiti uz Wink IBE sākumlapu. Sākumlapa parādīs jūsu norādīto tēmu kopā ar [īpašumiem, kurus pārstāvat](/studio/agency-representation).

Rezultātā iegūstat URL, kas izskatās apmēram šādi:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Ja esat īpašums, vēlaties izveidot saiti tieši uz Wink IBE īpašuma galveno lapu.

Šis URL izskatītos apmēram šādi:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Ja neiekļaujat klienta un/vai pielāgošanas ID, Wink IBE NEIELĀDĒSIES. Šie parametri ir obligāti.
:::
