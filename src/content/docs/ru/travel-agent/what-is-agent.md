Наше решение для зарегистрированных туристических агентов — это [Wink Agent](https://agent.wink.travel). Это портал самообслуживания, который агенты могут использовать для поиска и бронирования от имени своих клиентов, а также для получения комиссии с каждого бронирования.

Более продвинутые туристические агенты могут выбрать интеграцию на уровне API для обработки платежей и выплат вне платформы Wink.

Туристические агенты имеют подмножество тех же функций портала, что и [Wink Studio](/ru/studio/what-is-studio), поэтому мы не будем повторяться здесь.

## Поиск

Перейдите в раздел [Search](/ru/studio/search), чтобы узнать, как искать и бронировать отели.

В противном случае продолжайте к следующему разделу.

## API
