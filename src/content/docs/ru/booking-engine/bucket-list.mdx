Везде, где вы видите [карточки](/ru/studio/cards) инвентаря Wink, в правом верхнем углу карточки есть значок сердца, который позволяет добавить этот инвентарь в ваш список желаний.

Следуйте этим шагам, чтобы управлять своим списком желаний:

<Steps>
1. Нажмите на ♡ `значок сердца` для инвентаря, который хотите добавить.
2. Если вы вошли в систему, значок сердца станет красным.
3. Если вы не вошли в систему, вас перенаправят для входа.
</Steps>

:::tip
Нажмите на ♥️ `значок сердца` второй раз, чтобы удалить его из списка.
:::

Вы можете управлять всеми элементами вашего списка желаний, нажав на иконку профиля в правом верхнем углу, а затем на кнопку `Список желаний`.
