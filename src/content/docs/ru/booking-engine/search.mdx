Страница поиска — это место, где путешественники просматривают объекты размещения по локациям.

## Просмотр по локации

Объекты размещения можно просматривать по типу локации — города, острова, страны и другие. Каждая страница локации показывает сетку доступных отелей с ценами.

На каждой карточке объекта отображается:

- Название отеля и основное изображение.
- Локация.
- Самая низкая доступная цена.
- Рейтинг гостей.

## Селектор маршрута

Путешественники могут уточнять поиск с помощью селектора маршрута в верхней части страницы:

- **Даты заезда / выезда** — Установите даты путешествия.
- **Гости** — Количество взрослых и детей.
- **Промокод** — Примените промокод для скидок.

Сетка объектов обновляется в реальном времени при изменении маршрута.

## Нет результатов

Если ни один объект не соответствует критериям поиска, отображается полезное сообщение с предложением изменить даты или локацию.
