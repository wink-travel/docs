Находясь на https://ota.wink.travel, вы можете заполнить свою`travel profile`который позволяет отелям и поставщикам услуг узнать о вас больше.

Профиль путешествия содержит:

* Основная информация.
* Настройки локали.
* Контакты для экстренной связи.
* Адрес.
* Персональная информация.
* Домашние животные.
* Физические нагрузки.
* Предпочтения по отелям.
* Предпочтения по удобствам.
* Настройки безопасности.
* Корпоративные предпочтения.
* Предпочтения по инвалидности.
* Предпочтения по размещению.
* Предпочтения по удобствам в номере.
* Личные предпочтения.

Внизу вашего профиля вас спросят, хотите ли вы поделиться этим профилем с отелями, которые вы посещаете. Если вы выберете поделиться, отель увидит полный профиль в своем подтверждении бронирования.
