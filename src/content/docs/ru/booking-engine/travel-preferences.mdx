На сайте https://ota.wink.travel вы можете заполнить свой `профиль путешественника`, который позволяет отелям и поставщикам услуг лучше узнать вас.

Профиль путешественника содержит:

- Основную информацию.
- Предпочтения по языку и региону.
- Контакты для экстренной связи.
- Адрес.
- Личную информацию.
- Информацию о питомцах.
- Физические ограничения.
- Предпочтения по отелям.
- Предпочтения по удобствам.
- Предпочтения по безопасности.
- Корпоративные предпочтения.
- Предпочтения для людей с ограниченными возможностями.
- Предпочтения по номеру.
- Предпочтения по удобствам в номере.
- Личные предпочтения.

Внизу вашего профиля вас спросят, хотите ли вы поделиться этим профилем с отелями, которые вы посещаете. Если вы выберете поделиться, отель увидит полный профиль в подтверждении бронирования.
