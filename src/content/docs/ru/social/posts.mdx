Редактор постов позволяет создавать неограниченное количество ручных постов, а также управлять постами, которые были созданы автоматически.

Редактор состоит из 4 разделов:

1. Раздел контента.
2. Раздел медиа.
3. Раздел ссылок.
4. Раздел настроек.

## Контент

Раздел контента позволяет изменять текст перед публикацией. Вы можете создавать локализованный контент на нескольких языках — для каждого языка задается свой заголовок и описание.

## Медиа

Раздел медиа позволяет изменять изображения и видео, сопровождающие текст.

## Ссылка

Раздел ссылок позволяет указать, куда пользователь перейдет после прочтения поста. *Например, это может быть ссылка на специальное предложение или источник дополнительной информации.* Вы можете добавить URL ссылки, подпись и описание ссылки.

## Настройки

Раздел настроек позволяет изменять или переопределять все настройки по умолчанию для каждого поста. Вы можете задать:

- **Платформа** — Выберите социальную сеть, в которой будет опубликован этот пост.
- **Тип поста** — Выберите между Транзакционным, Осведомленностью или Настроением.
- **Тональность** — Переопределите тональность для этого конкретного поста.
- **Кастомизация** — Используйте другую кастомизацию для отображения валюты, языка, маршрута и т.д.
- **Часовой пояс** — Переопределите часовой пояс для публикации.
- **Запланированная дата** — Автоматически созданные посты планируются на оптимальное время. Вы можете изменить дату и время, если предпочитаете другой слот.

## Жизненный цикл статуса поста

Каждый пост проходит через жизненный цикл статуса:

| Статус | Описание |
| -- | -- |
| **Черновик** | Пост создан, но еще не готов к проверке |
| **Ожидает одобрения** | Пост ожидает одобрения человеком перед планированием |
| **Одобрен** | Пост одобрен и будет запланирован |
| **Отклонен** | Пост был проверен и отклонен — не будет опубликован |
| **Запланирован** | Пост поставлен в очередь на публикацию в запланированное время |
| **Опубликован** | Пост успешно опубликован в социальной сети |
| **Приостановлен** | Публикация поста временно приостановлена |
| **Ошибка** | Пост не удалось опубликовать из-за ошибки |

## Одобрение постов

Посты одобряются в сетке постов, где вы можете выбрать один или несколько постов и выбрать одобрить или отклонить их.

:::note
Для каждого языка и каждой социальной сети будет отдельная запись.

Например, если у вас выбраны 3 социальных сети и 3 языка, вы увидите 9 записей в календаре.
:::
