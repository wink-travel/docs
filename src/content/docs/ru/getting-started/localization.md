## На уровне платформы

Все наши приложения локализованы для поддержки всех перечисленных выше языков. В каждом приложении внизу страницы есть селектор языка, где пользователь может выбрать желаемый язык. Бронирование также поддерживает передачу желаемого языка через URL.

## Контент, создаваемый пользователями

Контент, создаваемый пользователями, локализован для поддержки всех перечисленных выше языков. Пользователи могут создать английскую версию контента и позволить Wink сделать остальное. Или они могут управлять конкретным набором языков для большего контроля и позволить Wink обработать остальные.

## Особенности

Язык по умолчанию — английский. Если не указано иное, мы сначала используем язык по умолчанию, когда пользователь использует любое из наших приложений. Мы не устанавливаем активный язык на основе локали браузера пользователя.
