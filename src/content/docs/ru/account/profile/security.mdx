Вкладка **Безопасность** на странице [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) — это место, где вы управляете **ключами доступа** — учетными данными WebAuthn, которые позволяют входить без пароля.

## Что такое ключ доступа?

Ключ доступа — это криптографический учетный данные, привязанные к вашему устройству (или менеджеру паролей), которые подтверждают вашу личность. Ключи доступа устойчивы к фишингу и работают быстрее, чем ввод пароля.

## Регистрация ключа доступа

1. Войдите на [iam.wink.travel](https://iam.wink.travel) с помощью существующих учетных данных.
2. Перейдите в **Мой аккаунт → Безопасность**.
3. Нажмите `Добавить ключ доступа`.
4. Ваш браузер предложит аутентифицироваться с помощью устройства (Touch ID, Face ID, Windows Hello, аппаратный ключ или синхронизированный менеджер паролей).
5. Дайте ключу доступа понятное имя (например, *Рабочий MacBook*), чтобы позже легко его узнать.

## Использование ключа доступа для входа

На экране входа [iam.wink.travel](https://iam.wink.travel) введите свой email — если для этого email зарегистрирован ключ доступа, вам предложат аутентифицироваться с его помощью вместо ввода пароля.

## Удаление ключа доступа

В той же вкладке Безопасность у каждого зарегистрированного ключа есть кнопка `Удалить`. Используйте ее, если вы потеряли устройство, вышли из эксплуатации аппаратный ключ или хотите очистить неиспользуемые учетные данные.

:::caution
Удаление всех ключей доступа оставляет пароли единственным способом входа. Убедитесь, что вы помните свой пароль, прежде чем удалять последний ключ доступа.
:::
