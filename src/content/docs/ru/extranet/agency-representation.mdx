Агентство — это обычный аккаунт Wink, который вы можете назначить своему аккаунту для целей представительства. Ниже приведены некоторые причины, по которым вам может понадобиться агентство, представляющее ваши интересы:

- Полное / частичное управление вашим аккаунтом Wink третьей стороной.
- Объект недвижимости имеет существующее соглашение со своей управляющей компанией и хочет, чтобы Wink занимался финансовой стороной их договорённости по поступающим через Wink средствам.
- Агентство по управлению недвижимостью выполняет всю онлайн-работу от имени своих объектов.

## Назначение агентства

Вот как назначить агентство вашему аккаунту.

<Steps>
 1. Войдите в свой пользовательский аккаунт на [Wink Extranet](https://extranet.wink.travel).
 2. Выберите объект недвижимости, с которым хотите работать.
 3. Нажмите `Account > Managers` в главном навигационном меню.
 4. В разделе `Agency` введите название агентства, которое вы хотите назначить представлять вас.
 5. Выберите комиссию агентства из выпадающего списка.
 5. При необходимости добавьте правила истечения срока.
 7. Нажмите кнопку `Set agency`, чтобы продолжить.
 </Steps>

 :::note
Вы должны быть владельцем объекта недвижимости, чтобы назначить агентство.
 :::

 :::tip[Пример]
Комиссия агентства рассчитывается от бронирования.

Бронирование = $100, комиссия агентства = 1% => 100 * 0.01 = $1
 :::

## Отмена назначения агентства

Вот как отменить назначение агентства вашему аккаунту.

<Steps>
 1. Войдите в свой пользовательский аккаунт на [Wink Extranet](https://extranet.wink.travel).
 2. Выберите объект недвижимости, с которым хотите работать.
 3. Нажмите `Account > Managers` в главном навигационном меню.
 4. В разделе `Agency` нажмите на иконку <Badge text="x" variant="danger" size="small" /> справа от выпадающего списка агентств.
 5. Нажмите кнопку `Set agency`, чтобы продолжить.
 </Steps>


## API

Разработчики, которые хотят управлять агентствами через API, могут перейти в раздел [Developers > API > Affiliate](/developers/apis/#affiliate-api).
