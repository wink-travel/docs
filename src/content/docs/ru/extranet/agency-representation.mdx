Агентство — это обычный аккаунт Wink, который вы можете назначить своему аккаунту для представительства. Ниже приведены некоторые причины, по которым вам может понадобиться агентство, представляющее вас:

* Полное/частичное управление вашей учетной записью Wink третьей стороной.
* У компании Property есть действующее соглашение с управляющей компанией, и она хочет, чтобы Wink занимался финансовым аспектом распределения средств, поступающих через Wink.
* Агентство по управлению недвижимостью выполняет всю работу в режиме онлайн от имени своих объектов.

## Назначить агентство

Вот как назначить агентство для своего аккаунта.

<Steps>
  1. Войдите в свою учетную запись пользователя на [Wink Экстранет](https://extranet.wink.travel).
  2. Выберите объект недвижимости, с которым вы хотите работать.
  3. Нажмите на `Account > Managers` из главной навигационной панели.
  4. В `Agency` введите название агентства, которое вы хотите представлять.
  5. Выберите размер агентского сбора из выпадающего списка.
  6. При желании можно добавить правила истечения срока действия.
  7. Нажмите на `Set agency` кнопку, чтобы продолжить.
</Steps>


:::note
Чтобы назначить агентство, вам нужно быть владельцем недвижимости.
:::

:::tip\[Пример]
Агентская комиссия взимается при бронировании.

Бронирование = $100, Комиссия агентства = 1% => 100 \* 0,01 = $1
:::

## Отменить назначение агентства

Вот как можно отменить назначение агентства вашему аккаунту.

<Steps>
  1. Войдите в свою учетную запись пользователя на [Wink Экстранет](https://extranet.wink.travel).
  2. Выберите объект недвижимости, с которым вы хотите работать.
  3. Нажмите на `Account > Managers` из главной навигационной панели.
  4. В `Agency` раздел, нажмите <Badge text="x" variant="danger" size="small" /> значок в правой части раскрывающегося списка агентств.
  5. Нажмите на `Set agency` кнопку, чтобы продолжить.
</Steps>
