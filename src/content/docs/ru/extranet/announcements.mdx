Объявления — это способ уведомить ваших гостей о важной информации таким образом, чтобы это привлекло внимание.
Если объявление `активировано`, оно появляется в виде нового окна поверх вашей целевой страницы.

Чтобы увидеть ваши объявления, нажмите на `Property > Announcement` в главном навигационном меню.

Нажмите кнопку `Create announcement`, чтобы создать новую запись.

<Steps>
1. Выберите, включать ли заголовок.
2. Объявления могут быть активированы одним из двух способов:
    - Всегда показывать при каждом посещении вашей страницы.
    - Показывать только когда маршрут совпадает с маршрутом объявления.
3. Если вы выбрали показывать объявление только в определённом диапазоне дат, добавьте даты в поля формы ниже.
4. Нажмите кнопку `Save`, и вы вернётесь к списку объявлений.
</Steps>
