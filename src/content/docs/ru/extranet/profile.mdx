Профиль объекта — это обязательный шаг в нашем мастере настройки, который позволяет нам лучше узнать ваш объект с помощью таких критериев, как:

- Количество звезд.
- Тип отеля.
- Тип расположения.
- ...и многое другое

Чтобы управлять профилем объекта, нажмите `Property > Profile` в главном навигационном меню.

Обновите форму и нажмите кнопку `Save`, чтобы продолжить.
