:::note
Создание вашего первого основного тарифа является частью мастера настройки.
:::

:::note
`Основной тариф` — это уникальная пара `гостевая комната / тарифный план`.

*Большинство поставщиков туристических тарифов используют только гостевую комнату и тарифный план в этом контексте. Когда вы видите свою гостевую комнату, выставленную на продажу на OTA или системах бронирования, путешественники сравнивают тарифные планы.*

Wink добавил термин `основной тариф`, чтобы придать уникальной комбинации большую гибкость.
:::

:::note
Основные тарифы создаются / удаляются автоматически при добавлении или удалении гостевой комнаты или тарифного плана.
:::

Для управления основными тарифами нажмите `Монетизация > Основной тариф` в главном навигационном меню.

У объекта размещения будет основной тариф для каждой комбинации гостевой комнаты и тарифного плана.

*Пример: если у вашего объекта размещения одна гостевая комната и два тарифных плана, путешественник увидит два доступных для продажи основных тарифа (если выполнены все условия).*

Запись основного тарифа имеет две опции для управления:

- Переключить доступность основного тарифа.
- Установить преимущества.

## Доступность

При включении / отключении основного тарифа это влияет на доступность во всех ваших каналах продаж. Этот переключатель — быстрый способ сразу отключить конкретный основной тариф везде.

## Преимущества

Когда путешественники ищут гостевую комнату для бронирования, они не взаимодействуют напрямую с гостевой комнатой. Вместо этого они взаимодействуют с основным тарифом этой гостевой комнаты.
Основной тариф может содержать уникальные преимущества, доступные только для конкретной комбинации гостевой комнаты и тарифного плана; таким образом, владельцы объектов размещения получают дополнительный способ стимулировать эту комбинацию.

Два момента, которые нужно учитывать при выборе преимущества:

- Гарантировано ли оно или зависит от доступности.
- Его числовое значение.

Гарантированное преимущество имеет более высокое числовое значение, так как представляет большую ценность для клиентов.
Каждый отель получает `оценку преимуществ`, основанную на суммарном числовом значении всех преимуществ по всем основным тарифам.
Путешественники, ищущие выгодные предложения, могут сортировать отели по высокой оценке преимуществ.

Преимущества позволяют:

- Отличить комбинацию гостевой комнаты и тарифного плана.
- Добиться большей видимости. *То есть получить более высокий балл по шкале преимуществ, чем другие отели в вашем регионе.*

:::tip
Преимущества появились из желания отелей выделиться и `обмануть систему`.
В прошлый раз, когда они это сделали, они буквально погубили себя, отдавая все свои прибыли OTA за лучшее размещение через более высокие комиссии.
Исторически комиссии были единственным способом для отелей играть с системой.

Преимущества — это способ для отелей вознаградить путешественников, не неся при этом реальных затрат.

Если у вас есть предложения по другим типам преимуществ, которые можно предложить, [свяжитесь с нами](mailto:hi@wink.travel).
:::

Существует два способа включить инвентарь для канала продаж:

- Переключить доступность инвентаря на карточке. *См. рисунок выше*
- Перейти в `Дистрибуция > Инвентарь` из главного навигационного меню, предназначенного для управления инвентарём во всех каналах продаж.

## API

Разработчики, желающие управлять `Основными тарифами`, могут перейти в [Разработчики > API > Монетизация](/ru/developers/apis/#monetize-api).
