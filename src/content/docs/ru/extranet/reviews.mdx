Страница отзывов показывает все отзывы гостей, оставленные о вашем объекте.

Чтобы просмотреть отзывы, нажмите `Distribution > Reviews` в главном навигационном меню.

## Просмотр отзывов

Список отзывов можно фильтровать, чтобы помочь найти конкретные отзывы:

- **Диапазон дат** — Фильтровать по дате отправки отзыва.
- **Имя гостя** — Поиск по имени или фамилии.
- **Email гостя** — Поиск отзывов по адресу электронной почты.
- **Ответы** — Фильтровать по наличию вашего ответа на отзыв.

Результаты разбиты на страницы по 25 отзывов на каждой.

## Детали отзыва

Нажмите на любой отзыв, чтобы увидеть полную информацию, включая письменный отзыв гостя и оценки. Используйте это, чтобы понять настроение гостей и выявить области для улучшения.
