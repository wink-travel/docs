Ваш зеленый индекс измеряется по воздействию вашего объекта на местную среду и по вашим решениям в области закупок.

Чтобы управлять своим рейтингом Зеленого индекса, нажмите в главном навигационном меню на `Property > Green Index`.

Пройдите по каждой вкладке и ответьте на вопросы анкеты максимально полно.

Нажмите кнопку `Save`, чтобы продолжить.
