Если вы хотите изменить пароль или забыли его, выполните следующие действия:

## Обновление пароля

<Steps>
1. Перейдите на [my.wink.travel](https://my.wink.travel).
1. Войдите в систему, нажав на `Sign-in / Register` в правом верхнем углу.
2. Нажмите на иконку профиля в правом верхнем углу, появится меню.
3. Нажмите на ссылку `Change password`.
4. Следуйте инструкциям для изменения пароля.
5. Нажмите `Change password`, чтобы продолжить.
</Steps>

При следующем входе в систему вам потребуется новый пароль.

## Забыли пароль

Иногда необходимо сбросить пароль пользователя.

Вот шаги для сброса пароля пользователя:

<Steps>
1. Нажмите на ссылку `Forgot password` на странице входа.
2. Введите имя пользователя / электронную почту для аккаунта, который хотите сбросить.
3. Нажмите кнопку `Forgot password`. *Вы будете перенаправлены на форму сброса пароля.*
4. Проверьте свою электронную почту.
</Steps>

2. Подтвердите новый пароль.
3. Нажмите кнопку `Reset password`. *Вы будете перенаправлены обратно на страницу входа.*
</Steps>
