Агентство — это обычный аккаунт Wink, который вы, как владелец, можете назначить своему аккаунту для целей представительства. Ниже приведены некоторые причины, по которым вы можете захотеть, чтобы агентство представляло вас:

- Полное / частичное управление вашим аккаунтом Wink третьей стороной.
- Объект недвижимости имеет существующее соглашение со своей управляющей компанией и хочет, чтобы Wink занимался финансовой стороной их договорённости по поступающим через Wink средствам.
- Агентство по управлению недвижимостью выполняет всю онлайн-работу от имени своих объектов.
- Талант-агентство представляет инфлюенсера / создателя контента.

## Назначение агентства

Вот как назначить агентство вашему аккаунту.

<Steps>
 1. Выберите аккаунт, с которым хотите работать, на странице [Accounts](https://my.wink.travel).
 2. Нажмите `Actions` внизу карточки аккаунта и выберите `Manage account`.
 3. На следующей странице перейдите на вкладку `Agency`.
 4. В разделе `Agency` введите название агентства, которое вы хотите назначить представлять вас.
 5. Выберите комиссию агентства из выпадающего списка.
 5. При необходимости добавьте правила истечения срока действия.
 7. Нажмите кнопку `Save`, чтобы продолжить.
 </Steps>

 :::note
Вы должны быть владельцем объекта, чтобы назначить агентство.
 :::

 :::tip[Пример]
Комиссия агентства рассчитывается от бронирования.

Бронирование = $100, комиссия агентства = 1% => 100 * 0.01 = $1
 :::

## Снятие назначения агентства

Вот как снять назначение агентства с вашего аккаунта.

<Steps>
 1. Выберите аккаунт, с которым хотите работать, на странице [Accounts](https://my.wink.travel).
 2. Нажмите `Actions` внизу карточки аккаунта и выберите `Manage account`.
 3. На следующей странице перейдите на вкладку `Agency`.
 4. В разделе `Agency` нажмите на иконку <Badge text="x" variant="danger" size="small" /> справа от выпадающего списка агентств.
 5. Нажмите кнопку `Save`, чтобы продолжить.
 </Steps>
