Wink предоставляет систему отчётов, которая позволяет создавать и экспортировать данные по вашему аккаунту. Отчёты доступны в формате CSV для удобного импорта в таблицы, аналитические инструменты или ваши собственные системы.

## Типы отчётов

Вы можете создавать отчёты по следующим категориям данных:

| Тип отчёта | Что включено |
|---|---|
| **Объект** | Информация об объекте, статус размещения, сводка по инвентарю. |
| **Пользователь** | Учётные записи пользователей, роли, сводка активности. |
| **Бронирование** | История бронирований, статусы, доходы, данные гостей. |
| **Аккаунт** | Информация об аккаунте, балансы, история транзакций. |

## Как создать отчёт

1. Перейдите в раздел отчётов на панели управления вашего аккаунта.
2. Выберите **тип отчёта**, который хотите создать.
3. Настройте **фильтры** для сужения данных (например, диапазон дат, статус, объект).
4. Выберите **поля**, которые хотите включить в экспорт. Для каждого типа отчёта доступен набор полей для выбора.
5. Нажмите **Экспорт** для скачивания отчёта в формате CSV.

## Пользовательские поля

Отчёты поддерживают выбор настраиваемых полей. Вы выбираете, какие столбцы включать в экспорт, чтобы получить именно те данные, которые вам нужны — ничего лишнего и ничего недостаточного.

## Сценарии использования

- **Сверка доходов** — Экспортируйте отчёты по бронированиям с финансовыми деталями для сверки с вашей бухгалтерской системой.
- **Анализ эффективности** — Экспортируйте отчёты по объектам, чтобы сравнить эффективность размещений в вашем портфолио.
- **Аналитика гостей** — Экспортируйте данные бронирований для понимания демографии путешественников и моделей бронирования.
- **Аудит аккаунта** — Экспортируйте данные аккаунта для соответствия требованиям или внутреннего контроля.
