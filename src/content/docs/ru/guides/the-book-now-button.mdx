Кнопка **ЗАБРОНИРОВАТЬ СЕЙЧАС** — это повсеместный термин в гостиничной индустрии, который обозначает собственную интеграцию отеля с провайдером, который знает о доступности и ценах объекта и позволяет путешественникам забронировать номер. Эта кнопка — самый простой способ для отелей продавать номера на собственном сайте и на своих условиях.

*Многие отели используют встроенный движок бронирования, предлагаемый их канал-менеджером. Мы рекомендуем отелям провести исследование и сравнить нативные движки бронирования перед тем, как выбрать какой-либо из них.*

## Аудитория

Эта статья предназначена в первую очередь для отелей, которые хотят установить наш бесплатный, с высокой конверсией, движок бронирования на свой сайт.  
Но... она также будет полезна всем, кто хочет использовать наши технологии на своих сайтах.

## через Ссылку

Для самой простой интеграции мы рекомендуем использовать [Shareable Link](/ru/studio/shareable-links).

Вот шаги для этого:

<Steps>
1. Перейдите на https://extranet.wink.travel  
2. Нажмите `Sign-In / Register` в правом верхнем углу и войдите в свою учетную запись.  
3. Выберите объект, с которым хотите работать, из выпадающего списка в левом верхнем углу.  
4. Нажмите на `Booking engine` в навигационной панели.  
5. Страница покажет вашу ссылку.  
6. Передайте эту ссылку вашему вебмастеру и используйте её для вашей кнопки **ЗАБРОНИРОВАТЬ СЕЙЧАС**.  
7. Вы ✅.
</Steps>

### Расширенные возможности

Ссылка уже содержит маршрут, встроенный в URL, который исходит из выбранной вами `customization`.  
Вы можете дополнительно настроить ссылку, изменив:

- Выбранную кастомизацию ИЛИ  
- Переопределив маршрут

В этой статье мы сосредоточимся на втором варианте, так как он дает больше свободы, позволяя путешественникам выбирать маршрут на вашем сайте и передавать его в движок бронирования Wink.

Вот как передать свой собственный маршрут вместе с URL.

Допустимые параметры запроса:

- **sd** Добавить дату начала: *sd=2024-08-24*  
- **n** Добавить количество ночей: *n=1* ИЛИ  
- **ed** Использовать дату окончания: *ed=2024-08-25*  
- **rc** Гости: *rc=a2* (Взрослые: 2)  
- **l** Запросить язык отображения: *l=es*  
- **c** Запросить валюту отображения: *c=EUR*  
- **p** Добавить промоакцию: *p=ABC123*

URL будет выглядеть так:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2&l=es&c=EUR

Для более сложных случаев вы также можете запросить несколько конфигураций номеров, используя знак `+`:

Пример:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2-ca16q1-ca8q1+a2&l=es&c=EUR

- Тип номера 1 = Взрослые: 2, Дети: 2 = 16 лет + 8 лет.  
- Тип номера 2 = Взрослые: 2.

### Другие преимущества

Есть и другие преимущества использования одной из наших ссылок в качестве вашей кнопки `book now`.

- **Социальная доступность** Ссылку легко поделиться в социальных сетях, и она отображает `динамическое ценообразование` поверх выбранного вами изображения.  
- **Короткая** URL очень удобно распространять, например, использовать в вашем Instagram Bio URL.

## через Встраивание

Некоторые отели хотят получить немного больше функциональности и более глубокую интеграцию для своего сайта. Этот пример покажет, как встроить [Card](/ru/studio/cards) на их сайт.

Вот шаги для этого:

<Steps>
1. Перейдите на https://studio.wink.travel  
2. Нажмите `Sign-In / Register` в правом верхнем углу и войдите в свою учетную запись.  
3. Выберите объект, с которым хотите работать, в левом верхнем углу.  
4. Нажмите на `Search` в главной навигационной панели.  
6. Введите название вашего объекта и нажмите кнопку поиска.  
7. Нажмите ссылку `Actions` на карточке результата поиска и выберите `Create > Make a Card`.  
8. Это создаст `Card`.  
9. В правом верхнем углу появится сообщение о том, что ваша карточка готова и вы можете её дополнительно настроить.  
10. Нажмите на `Tools > Cards` в главной навигационной панели, и вы увидите вашу новую карточку.  
11. Нажмите на ссылку `Actions` на вашей карточке и выберите `Embed`.  
12. Откроется новое окно с примером HTML-кода, который вы можете скопировать.  
13. Чтобы получить ваш `Client-ID`, перейдите в [Applications](/ru/developers/applications).  
14. Чтобы получить ваш `Configuration-ID`, перейдите в [Customization](/ru/studio/customization)  
14. Вы ✅.
</Steps>

Пример HTML для встроенной карточки:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*Приведенный выше пример HTML не включает необходимые CSS и Javascript Wink; только карточку.*

Пользователь теперь видит динамическое ценообразование, может изменить маршрут и нажать, чтобы забронировать напрямую на вашем сайте. Кнопка бронирования ведет на лендинг вашей собственности в Wink для завершения оплаты.
