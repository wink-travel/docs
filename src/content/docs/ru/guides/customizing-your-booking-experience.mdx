Настройки управляют тем, как ваш инвентарь отображается путешественникам — цвета темы, валюта по умолчанию, язык, маршрут и многое другое. Они применяются ко всем шарируемым ссылкам, встроенным карточкам, сеткам, картам и движку бронирования.

## Что такое настройка?

Настройка — это профиль конфигурации, который определяет:

- **Валюта отображения** — В какой валюте показываются цены.
- **Язык отображения** — Язык по умолчанию для контента.
- **Маршрут по умолчанию** — Предзаполненные даты заезда/выезда и количество гостей.
- **Тема** — Цвета и фирменный стиль, применяемые к встроенным компонентам.
- **Конфигурация** — Как представлен инвентарь (макет, сортировка).

Вы можете создавать несколько настроек для разных рынков или сценариев использования.

## Создание настройки

<Steps>
1. Перейдите на https://studio.wink.travel (или в ваш аккаунт на https://my.wink.travel).
2. В боковом меню выберите `Customization`.
3. Нажмите `Create a new customization`.
4. Дайте настройке описательное имя (например, "Europe - EUR", "Japan Market", "Default").
5. Установите валюту отображения, язык и маршрут по умолчанию.
6. Настройте цвета темы в соответствии с вашим фирменным стилем.
7. Нажмите `Save`.
</Steps>

## Где используются настройки

После создания настройки используются по всей платформе:

| Функция | Как применяется настройка |
| -- | -- |
| **Шарируемые ссылки** | Ссылка использует валюту, язык и маршрут настройки по умолчанию |
| **Встроенные карточки** | Карточки отображаются с вашими цветами темы и настройками отображения |
| **Сетки и карты** | Сетки инвентаря и маркеры на карте следуют настройке |
| **Движок бронирования** | Весь процесс бронирования использует настройки настройки |
| **Социальные публикации** | AI-сгенерированные посты могут использовать конкретную настройку для отображения цен |

## Несколько настроек

Распространённая практика — создавать по одной настройке для каждого целевого рынка:

- **"US Market"** — USD, английский, стандартное проживание 2 ночи.
- **"Europe"** — EUR, английский, стандартное проживание 3 ночи.
- **"Japan"** — JPY, японский, стандартное проживание 1 ночь.

При создании шарируемой ссылки или встраивании карточки вы выбираете, какую настройку использовать. Это значит, что один и тот же объект может быть представлен по-разному в зависимости от аудитории.

## Переопределение через URL

Вы также можете переопределять настройки по умолчанию с помощью параметров URL в шарируемых ссылках:

- `l=es` — Переопределить язык на испанский.
- `c=EUR` — Переопределить валюту на EUR.
- `sd=2024-08-24&n=2` — Переопределить маршрут на конкретную дату и количество ночей.

Смотрите [The Book Now Button](/ru/guides/the-book-now-button) для полного списка параметров URL.
