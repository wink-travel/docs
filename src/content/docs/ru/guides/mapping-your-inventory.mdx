Когда вы добавляете инвентарь на свой объект в Extranet — номера, рестораны, спа, конференц-залы, активности и другое — этот инвентарь становится доступен во всей экосистеме Wink. В этом руководстве объясняется, как все элементы связаны.

## Поток инвентаря

```
Extranet (вы создаёте) → Booking Engine (путешественники видят) → Studio (партнёры продают)
```

1. **Вы создаёте инвентарь** в Extranet — типы номеров, объекты и развлечения.
2. **Путешественники просматривают** этот инвентарь в системе бронирования через общие ссылки, карточки или прямой поиск.
3. **Партнёры находят** ваш инвентарь в Studio, создают карточки и сетки и продают его своей аудитории.

## Типы номеров

Типы номеров — основа вашего инвентаря. Каждый тип номера должен содержать:

- Название и описание.
- Конфигурацию кроватей.
- Максимальную вместимость (взрослые и дети).
- Удобства, специфичные для номера.
- Фотографии.
- По крайней мере один тарифный план с основными тарифами.

Без тарифов тип номера существует, но не может быть забронирован.

## Объекты и развлечения

Помимо номеров, вы можете добавить:

| Тип инвентаря | Что это | Пример |
| -- | -- | -- |
| **Ресторан** | Питание на территории | "Rooftop Bar & Grill" |
| **Спа** | Услуги для здоровья и отдыха | "Serenity Spa - 60min massage" |
| **Конференц-зал** | Пространство для мероприятий и конференций | "Boardroom A - 20 pax" |
| **Активность** | Чем заняться на территории или рядом | "Guided snorkeling tour" |
| **Достопримечательность** | Известные места поблизости | "National Museum - 500m" |
| **Место** | Интересные локации | "Local market" |
| **Дополнение** | Дополнительные услуги, приобретаемые с номером | "Airport transfer" |

Каждый тип объекта имеет свой раздел в Extranet. Партнёры могут продавать любой из них отдельно — путешественник может найти ваш объект через спа, а не через поиск номеров.

## Как партнёры используют ваш инвентарь

В Studio партнёры могут:

- **Искать** ваш объект или отдельные позиции инвентаря.
- **Создавать карточки** — отдельные бронируемые карточки для номера, спа-процедуры или активности.
- **Создавать сетки** — коллекции карточек, показанные вместе.
- **Создавать карты** — маркеры на карте с бронируемым инвентарём.
- **Делиться ссылками** — прямые URL на ваш объект или конкретные позиции.

Чем более полный и разнообразный ваш инвентарь, тем больше способов у партнёров продвигать ваш объект.

## Советы

- **Будьте тщательны** — укажите всё, что предлагает ваш объект. Каждый элемент — потенциальная точка входа для бронирования.
- **Используйте качественные фотографии** — у каждого элемента инвентаря должна быть хотя бы одна качественная фотография.
- **Обновляйте описания** — меняйте описания при изменении предложений или сезонно.
- **Цените всё** — инвентарь без цен можно просматривать, но нельзя забронировать.
