Чтобы управлять тегами WinkLinks, перейдите в `WinkLinks` в главном навигационном меню и нажмите на вкладку `Tags`.

По мере роста вашего контента вы захотите начать его организовывать. Для этой цели мы создали `tags`. Вы можете группировать контент, используя любое количество тегов.

## Создать тег

<Steps>
1. Создайте новый тег, нажав кнопку `Add a new tag`.
2. Введите название вашего нового тега. *например, Лучшие кофейни в Берлине*
3. Нажмите ссылку `Save`.
</Steps>

## Обновить тег

<Steps>
1. Начните с нажатия ссылки `Edit` рядом с тегом, который хотите изменить.
2. Обновите название вашего тега. *например, Лучшие кофейни в Берлине*
3. Нажмите ссылку `Save`.
</Steps>

## Удалить тег

:::note
При удалении тега он также будет удалён из всего связанного с ним контента.
:::

<Steps>
1. Нажмите ссылку `Remove` рядом с тегом, который хотите удалить.
</Steps>
