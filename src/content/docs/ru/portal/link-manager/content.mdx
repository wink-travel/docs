Ваш контент доступен во вкладке `Entries`.

Мы поддерживаем три основных типа контента:

- Публично доступные URL, которые ведут на веб-страницы.
- Файлы, загруженные через WinkLinks.
- Текстовый контент, созданный с помощью функции `Quotes`.

## Добавить контент

Вот как добавить контент в WinkLinks. Во вкладке `Entries` начните с нажатия кнопки `Add entry`.

### Обычная ссылка

:::tip
Ссылка должна вести на веб-страницу. Например, нельзя напрямую ссылаться на PDF. Если вы хотите поделиться PDF, используйте функцию `Upload file`.
:::

Вот шаги для добавления обычной ссылки:

<Steps>
1. Введите ссылку, которой хотите поделиться. *например, https://trvl.as/3xWCH*
2. После ввода ссылки Wink автоматически ищет данные о сайте и заполняет форму, которую вы видите далее.
3. Вы можете изменить некоторые данные, такие как название сайта, заголовок и описание.
4. Также можно загрузить собственное изображение.
5. Нажмите кнопку `Save`, когда будете готовы продолжить.
</Steps>

### Загрузка файла

:::caution[Требования]
- Размер изображений не должен превышать 10 Мб.
- Размер видео не должен превышать 50 Мб.
- Мы поддерживаем все распространённые форматы изображений и видео, включая PDF.
:::

:::note[Рекомендации]
- 2560w x 1440h (соотношение сторон 16:9) для вашего основного изображения.
- 2560w x 1920h (соотношение сторон 4:3) для всех остальных изображений.
- Размер изображения не должен превышать 5 Мб. Пожалуйста, сжимайте изображения перед загрузкой в Wink.
:::

Вот шаги для добавления файла:

<Steps>
1. Медиа можно добавить:
    - Перетащив изображения с компьютера в окно или нажав `Browse`.
    - Вставив внешний URL, указывающий на изображение.
    - Используя камеру вашего ноутбука.
    - Из вашего аккаунта Google Drive.
    - Из вашего аккаунта Dropbox.
    - С Shutterstock
    - С gettyimages
    - С iStock
    - С Unsplash
2. После загрузки файла вы можете добавить метаданные, такие как заголовок и описание.
3. Нажмите кнопку `Save`, когда будете готовы продолжить.
</Steps>

### Написать цитату

:::tip
Цитата — отличный способ добавить разделитель, чтобы разбить длинный список [...и при этом звучать умно и осведомлённо 😉].
:::

Вот шаги для написания цитаты:

<Steps>
1. Введите цитату *например, Roses are red. Violets are blue...*.
2. Введите имя оригинального автора. *например, Edmund Spenser*
3. Введите название сайта, откуда взята цитата. *например, Daily Quotes*
4. Нажмите кнопку `Save`, когда будете готовы продолжить.
</Steps>

## Обновление контента

Весь контент можно легко изменить, нажав на ссылку `Actions` и выбрав `Update`. Контент можно удалить аналогичным образом, нажав кнопку `Remove`.

## Удаление контента

Весь контент можно легко изменить, выбрав `Update`. Контент можно удалить аналогичным образом, сначала нажав на ссылку `Actions`, а затем на кнопку `Remove`.

## Сортировка контента

Вы можете упорядочить контент, нажав на иконку в левом нижнем углу каждой записи и перетащив её в нужное место.

## Группировка контента

:::tip
Создавайте теги до того, как начнёте создавать записи. Иначе вы потеряете прогресс и придётся начинать заново.
:::

При редактировании записи вы можете добавить один или несколько тегов для группировки. Прокрутите форму вниз, чтобы найти поле `Tags`.

Пользователи, просматривающие вашу страницу WinkLinks, могут выбирать, как им удобнее просматривать контент (списком или сеткой), а также могут выбрать один или несколько тегов для дополнительной фильтрации.
