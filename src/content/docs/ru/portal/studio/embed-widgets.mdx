Studio предоставляет несколько встраиваемых виджетов, которые вы можете добавить на свой сайт для увеличения бронирований.

Чтобы получить доступ к встраиваемым виджетам, перейдите в `Инструменты > Разное` в боковом меню.

## Доступные виджеты

| Виджет | Описание |
| -- | -- |
| **Кнопка аккаунта** | Фирменная кнопка, ведущая на вашу страницу партнёра |
| **Кнопка маршрута** | Кнопка, открывающая селектор маршрута |
| **Форма маршрута** | Встроенная форма выбора даты и гостей |
| **Кнопка поиска** | Кнопка, открывающая поиск объектов |
| **Кнопка поиска по инвентарю** | Кнопка для поиска конкретного инвентаря |

## Как встроить

Каждый виджет предоставляет фрагмент кода, который вы можете скопировать и вставить в HTML вашего сайта. В фрагменте содержатся ваши **Client ID** и **Customization ID**, чтобы виджет автоматически связывался с вашим партнёрским аккаунтом.

:::note
Если вы используете WordPress, мы также предоставляем плагин для WordPress для более простой интеграции.
:::
