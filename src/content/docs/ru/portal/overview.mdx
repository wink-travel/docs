[Portal](https://app.wink.travel) — это единое веб-приложение, в котором теперь сосредоточены все разделы экосистемы Wink. Ранее у каждой функциональной области (Extranet, Studio, Travel Agent, Payment, Social, Link Manager) был собственный URL. С этого релиза все они стали разделами единого Портала под `app.wink.travel`, организованными как подпути.

## Разделы

- **`/extranet`** — Функции для объектов размещения: отели, сети, бренды и управляющие компании.
- **`/studio`** — Функции для аффилиатов по продвижению туристического инвентаря и заработку комиссий.
- **`/travel-agent`** — Функции для зарегистрированных турагентов: поиск, бронирование и заработок.
- **`/payment`** — Функции для управления выплатами и выводом средств.
- **`/social`** — Автоматизированный Social для публикаций в соцсетях с помощью ИИ.
- **`/link-manager`** — Link Manager для создания страниц WinkLinks.
- **`/corporate`** — Корпоративный раздел (скоро).
- **`/settings`** — Настройки аккаунта: брендинг, менеджеры, приложения, вебхуки, выплаты и другое.
- **`/plan`** — Управление подпиской Wink.
- **`/invite`** — Принятие приглашений для управления другими аккаунтами.
- **`/analytics`** — Дашборды производительности по вашим аккаунтам.
- **`/leaderboard`** — Посмотрите, как ваш объект размещения ранжируется географически.
- **`/claims`** — Превращайте лиды объектов размещения в полные листинги.

## Portal и пользовательские настройки

Portal управляет всем, что вы делаете как организация или владелец аккаунта. Личное управление профилем пользователя (электронная почта, пароль, passkeys, согласия OAuth-приложений) осуществляется отдельно через сервис **IAM** по адресу `iam.wink.travel/my-account`. Подробнее смотрите в разделе [Пользовательские настройки](/ru/account/overview).

## Переключение аккаунтов

Вы можете управлять несколькими аккаунтами — объектами размещения, аффилиат-аккаунтами, аккаунтами турагентов, корпоративными аккаунтами — используя один вход. Для переключения между ними используйте переключатель аккаунтов в верхней навигации. Настройки, аналитика и большинство функций работают в контексте выбранного аккаунта.
