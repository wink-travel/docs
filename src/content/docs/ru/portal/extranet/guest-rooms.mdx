Каждое бронирование начинается с гостевой комнаты. Вокруг гостевых комнат много функций, и сначала это может показаться немного сложным.
В этой статье вы узнаете всё, что нужно, чтобы стать специалистом по гостевым комнатам в Wink.

Гостевые комнаты, или типы комнат, как говорят в `бизнесе`, позволяют вам демонстрировать и продавать ваши комнаты как на территории, так и за её пределами. Чтобы создать гостевую комнату, нажмите кнопку `Create a new guest room`.

В следующих разделах подробно описано управление гостевой комнатой.

## Конфигурация

Вкладка `Configurations` выбрана по умолчанию при начале работы.

Этот раздел охватывает основы вашей гостевой комнаты, чтобы мы знали, кому можем её продавать.

- **Название типа комнаты** Название комнаты, как вы её называете внутри компании. *например, Supreme deluxe*
- **Некомнатная зона для курения** Является ли комната для некурящих.
- **Базовая цена** Опубликованная цена за одну ночь без скидок и наценок. *например, 100. Обратите внимание, что это число используется только для отображения ожидаемой цены в каналах продаж или для предварительного заполнения некоторых полей при заполнении форм.*
- **Лучшая доступная цена** Лучшая доступная цена за одну ночь без скидок и наценок. *например, 75. Обратите внимание, что это число используется только для отображения ожидаемой цены в каналах продаж или для предварительного заполнения некоторых полей при заполнении форм.*
- **Количество** Количество единиц этого типа комнаты на любой день. *например, 40*

### Вместимость

:::caution
Вместимость играет важную роль в том, будет ли ваша гостевая комната отображаться как доступная для конкретного маршрута. Многие из этих полей будут заполнены автоматически при создании комнаты. Обязательно измените их, чтобы они соответствовали вашему типу комнаты.
:::

- **Мин. вместимость** Минимальное количество человек, разрешённых для проживания в этой комнате. *например, 1*
- **Включённая взрослые** Количество взрослых, включённых в цену за ночь. *например, 2. Примечание: если комната поддерживает большее количество, они рассчитываются с помощью модификатора `Extra adult` в вашем тарифном плане.*
- **Макс. вместимость** Максимальное количество человек (взрослых + детей), разрешённых для проживания в этой комнате. *например, 4*
- **Макс. взрослые** Максимальное количество взрослых, разрешённых для проживания в этой комнате. *например, 3*
- **Включённые дети** Количество детей, включённых в цену за ночь. *например, 1. Примечание: если комната поддерживает большее количество, они рассчитываются с помощью модификатора `Extra child` в вашем тарифном плане.*
- **Макс. дети** Максимальное количество детей, разрешённых для проживания в этой комнате. *например, 2*

### Прочее

Этот раздел предназначен только для информации. Путешественникам, ищущим большие комнаты и люксы, эта информация будет интересна.

- **Количество ванных комнат** Количество ванных комнат для этого типа комнаты. *например, 1*
- **Количество гостиных** Количество гостиных для этого типа комнаты. *например, 1*
- **Макс. раскладушек** Количество раскладушек, поддерживаемых этим типом комнаты. *например, 1*
- **Макс. детских кроваток** Количество детских кроваток, поддерживаемых этим типом комнаты. *например, 1*

### Классификация комнаты

- **Сегмент** Сегмент, к которому относится гостевая комната. *например, Budget*
- **Расположение** Тип расположения, к которому относится гостевая комната. *например, High floor*
- **Вид** Тип вида из этой гостевой комнаты. *например, City view*
- **Классификация** Классификация, к которой относится эта гостевая комната. *например, Economy*
- **Архитектура** Стиль здания вокруг этой гостевой комнаты. *например, Asian*
- **Этаж** Необязательный этаж(и), на котором находится эта гостевая комната *например, 20-й этаж и выше*

### Состав

Состав применим только если гостевая комната состоит из нескольких комнат.

- **Состав** Переключатель в положение ✅, если гостевая комната состоит из нескольких комнат. *например, Да*
- **Количество** Укажите, из скольких гостевых комнат состоит эта гостевая комната. *например, 2*

### Детали хостела

Этот раздел применим только если вы управляете общежитием, хостелом или другим сервисом с общими комнатами.

- **Общая комната** Переключатель в положение ✅, если гостевая комната является общей. *например, Да*
- **Пол** Укажите пол для этой гостевой комнаты. *например, Мужской*

## Спальни

:::note
Существует бесконечное количество возможных планировок спален. Мы постарались сделать это максимально настраиваемым, чтобы соответствовать вашему объекту.
:::

Нажмите на вкладку `Bedrooms`, чтобы продолжить.

Раздел спальни позволяет указать планировку вашей спальни(й) и указать, что в каждой из них.

Гостевые комнаты одного типа могут иметь несколько планировок, или `Bedding configurations`. Гостевая комната должна иметь как минимум одну планировку.

- **Название конфигурации** Назовите планировку так, чтобы вам было понятно *например, Main*

В одной планировке может быть несколько спален.

- **Тип спальни** Каждая спальня имеет тип. *Первая по умолчанию — `Master`, но вы можете изменить это в конфигурации.*

В спальне может быть несколько кроватей разных типов и количеств.

- **Тип кровати** Выберите тип кровати в вашей спальне. *например, King*
- **Количество** Выберите количество кроватей этого типа в спальне. *например, 1*

## Удобства

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Amenities`, чтобы продолжить.

Подобно удобствам объекта, удобства гостевой комнаты применяются только к конкретной гостевой комнате. Отметьте каждое удобство, которое применимо.

## Описания

:::note
Необходимо написать хотя бы одно описание, и оно должно быть на английском языке.
:::

Нажмите на вкладку `Descriptions`, чтобы продолжить.

Подобно приветственному тексту объекта, вы можете создавать локализованные описания для гостевой комнаты.
Вы можете писать на любом количестве языков. Мы добавим переводы для всех самых популярных языков.
Это текст, который увидят путешественники при просмотре ваших гостевых комнат.

- **Название** Дайте гостевой комнате имя. *например, Supreme deluxe*
- **Описание** Опишите гостевую комнату в одном-двух абзацах. *например, Это лучшая гостевая комната во всём мире...*

## Социальные сети

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Social media`, чтобы продолжить.

Возможно, вы задаётесь вопросом: «Зачем гостевой комнате отдельный раздел социальных сетей?». Гостевая комната может быть также виллой рядом с объектом, но вне его территории, с собственным персоналом и собственным профилем в Instagram.

Социальные сети помогают поддерживать связь с новыми и существующими путешественниками со временем.
Добавьте свои аккаунты в социальных сетях в профиль, чтобы гости могли легко связаться с вами и узнать больше о вашей гостевой комнате.

<Steps>
1. Включите переключатель социальной сети, которую хотите активировать.
2. Введите название вашего аккаунта в этой сети.
</Steps>

## Репутация

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Reputation`, чтобы продолжить.

Подобно вкладке `Social media`, вы можете использовать эту вкладку, если у этой гостевой комнаты есть отдельные оценки репутации или дополнительные награды, которые вы хотите выделить здесь.

Ваша репутация — это ваша цифровая валюта, и вы хотите иметь возможность брать её с собой куда угодно.
Запись о репутации может быть чем угодно, чем ваш объект хочет гордиться, например, рейтингом от третьей стороны или победой вашего шеф-повара на конкурсе.
В этом разделе вы можете добавить свои оценки и рейтинги со всего Интернета.

Нажмите кнопку `Add reputation score`.

<Steps>
1. **Категория** Выберите тип оценки, которую хотите добавить. *например, Отзыв третьей стороны*
2. **Поставщик** Введите, кто присвоил вам оценку. *например, BigOTA.com*
3. **Тип рейтинга** Выберите форму, в которой вы получили оценку. *например, Числовой*
4. **Дата** По желанию, введите дату получения оценки.
5. **Рейтинг** Введите полученную оценку. *например, 8.5*
6. **Максимальный рейтинг** По желанию, введите максимальный возможный балл для этого типа награды. *например, 10*
</Steps>

## Продажи на основе атрибутов

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Attribute Based Selling`, чтобы продолжить.

Все наши объекты имеют собственный раздел ABS. Этот раздел позволяет создавать возможности дополнительной продажи в основном инвентаре или делать не транзакционный тип инвентаря транзакционным.
Мы настоятельно рекомендуем создавать такие возможности, чтобы потенциально связаться с бронирующим по более высокой цене в процессе бронирования.

:::tip
Гостевая комната включает множество дополнительных опций:

- Улучшенный матрас.
- Делюкс набор для ванной.
- Bluetooth-колонка Marshal.
- Рабочий стол с дополнительным удобным креслом.
- Завтрак в номере
- Массаж в номере
- Упаковка из 12 бутылок пива
- Бутылка вина
- Премиальный шоколад
- +++ дайте волю своему воображению и заработайте дополнительные деньги!!
:::

- **Ценовой уровень** Дайте путешественнику быстрое представление о том, насколько дорогая эта гостевая комната. Именно гостевая комната, а не дополнительная опция.
- **Активна** Переключатель в положение 🛑, если вы хотите сделать эту гостевую комнату недоступной для бронирования во всех основных тарифах и каналах продаж.
- **Комиссионная** Укажите, получает ли партнёр комиссию с этого инвентаря.
- **Рекомендуемая** Укажите, должна ли эта гостевая комната занимать более заметное место, если такая возможность есть.
- **Продажи на основе атрибутов** Переключатель в положение ✅ для включения ABS на этом инвентаре. *После включения необходимо добавить как минимум 1 товар ABS для продажи*

### Дополнительный товар

При включении ABS вы можете начать добавлять дополнительные товары и делать их доступными для продажи.

*По умолчанию для вас уже создан пустой товар. Вы можете добавить больше, нажав кнопку `Add item for sale`.*

- **Название** Дайте вашему дополнительному товару имя. *например, Шоколадные трюфели с шампанским от Royce*
- **Тип ценообразования** Укажите, как вы хотите рассчитывать цену. *например, Аренда паддлборда. Установите `Per hour`, чтобы указать цену за час аренды.*
- **Базовая цена** Введите обычную цену этого товара. *например, 20 / час*
- **Скидочная цена** Введите цену, которую получают пользователи Wink при бронировании товара вместе с комнатой. *например, 15 / час*

### Ограничения

Вы можете ограничить доступность этого товара, установив минимальные и максимальные ограничения по количеству.
Если продолжить наш пример с арендой паддлборда по часам, но вы хотите, чтобы гость бронировал минимум 2 доски одновременно, то:

- **Минимальное количество** Ограничение на минимальное количество для бронирования. *например, 2 паддлборда*
- **Максимальное количество** Ограничение на максимальное количество для бронирования. *например, оставить пустым для отсутствия ограничения*

### Описания

:::note
Необходимо написать хотя бы одно описание, и оно должно быть на английском языке.
:::

- **Название** Дайте вашему дополнительному товару имя. *например, Stand Up Paddleboard*
- **Описание** Опишите дополнительный товар в одном-двух абзацах. *например, Не просто сидите — возьмите весло в руки!*

## Близость

Нажмите на вкладку `Proximity`, чтобы продолжить.

Не весь инвентарь находится на территории объекта. Если у вас есть инвентарь, расположенный вне территории, сообщите об этом путешественнику здесь, и вам будет предоставлена возможность установить отдельное местоположение на карте вместе с дополнительной контактной информацией.

*Это может быть полезно, например, если вилла принадлежит отелю, но находится в нескольких километрах от него, и путешественник хочет объяснить таксисту, как туда добраться.*

Нажмите кнопку `Save`, чтобы продолжить.
