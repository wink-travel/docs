Места позволяют вам демонстрировать и продавать ваши места, как на территории, так и вне её. Чтобы создать место, нажмите кнопку `Create place`.

В следующих разделах подробно описан каждый этап управления местом.

## Конфигурация

Вкладка `Configurations` выбрана по умолчанию при начале работы.

Этот раздел охватывает основные настройки вашего места.

- **Внутреннее имя** Название места, как вы его называете внутри системы. *например, Дом с привидениями*
- **Стиль жизни** По желанию выберите стиль жизни, который подходит этому месту. *например, Семейный*
- **Тип** По желанию выберите тип места, который больше всего подходит. *например, Музей*

## Расписание

Этот раздел позволяет управлять временем, когда место доступно.

### Часы работы

- **Открывается** Время суток, когда место становится доступным. *например, 09:00*
- **Закрывается** Время суток, когда место становится недоступным. *например, 21:00*

### Дни работы

Этот раздел позволяет установить дни недели, когда место доступно.

*Переключите тумблер в положение ✅, чтобы указать, что день доступен.*

### Сезонность

:::note
Нет удобного, нативного поля для выбора Дня / Месяца. В этом случае мы используем только месяц и день из этого селектора даты.
:::

Этот раздел позволяет установить даты, когда место доступно в течение года.

- **Дата начала** Дата, когда место становится доступным. *например, 11 сентября 2024*
- **Дата окончания** Дата, когда место становится недоступным. *например, 11 июля 2024*

## Удобства

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Amenities`, чтобы продолжить.

Подобно удобствам объекта, удобства места применяются только к конкретному месту. Отметьте каждое удобство, которое применимо.

## Описания

:::note
Необходимо написать хотя бы одно описание, и оно должно быть на английском языке.
:::

Нажмите на вкладку `Descriptions`, чтобы продолжить.

Подобно приветственному тексту объекта, вы можете создавать локализованные описания для места.
Вы можете писать на любом количестве языков. Мы добавим переводы для всех самых популярных языков.
Это текст, который увидят путешественники при просмотре ваших мест.

- **Название** Дайте месту название. *например, Дом с привидениями*
- **Описание** Опишите место в одном-двух абзацах. *например, Хотите узнать, существуют ли призраки?*

## Социальные сети

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Social media`, чтобы продолжить.

Возможно, вы задаётесь вопросом: «Почему у места есть собственный раздел социальных сетей?». Место может быть также объектом рядом с основным объектом, но вне его территории, с собственным персоналом и собственным профилем в Instagram.

Социальные сети помогают поддерживать связь с новыми и существующими путешественниками со временем.
Добавьте свои аккаунты в социальных сетях в профиль, чтобы гости могли легко связаться с вами и узнать об этом месте.

<Steps>
1. Включите тумблер социальной сети, которую хотите активировать.
2. Введите название вашего аккаунта в этой сети.
</Steps>

## Репутация

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Reputation`, чтобы продолжить.

Подобно вкладке `Social media`, вы можете использовать эту вкладку, если у этого места есть отдельные оценки репутации или дополнительные награды, которые вы хотите выделить здесь.

Ваша репутация — это ваша цифровая валюта, и вы хотите иметь возможность брать её с собой куда угодно.
Запись о репутации может быть чем угодно, за что ваш объект хочет получить признание, например, рейтинг от третьей стороны или победа вашего шефа на конкурсе.
В этом разделе вы можете добавить свои оценки и рейтинги со всего Интернета.

Нажмите кнопку `Add reputation score`.

<Steps>
1. **Категория** Выберите тип оценки, которую хотите добавить. *например, Отзыв третьей стороны*
2. **Поставщик** Введите, кто присвоил вам оценку. *например, BigOTA.com*
3. **Тип рейтинга** Выберите форму, в которой вы получили оценку. *например, Числовой*
4. **Дата** По желанию, введите дату получения оценки.
5. **Рейтинг** Введите полученную оценку. *например, 8.5*
6. **Максимальный рейтинг** По желанию, введите максимальный возможный балл для этого типа награды. *например, 10*
</Steps>

## Продажи на основе атрибутов

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Attribute Based Selling`, чтобы продолжить.

Все наши объекты имеют собственный раздел ABS. Этот раздел позволяет создавать возможности дополнительной продажи в основном инвентаре или делать не транзакционный тип инвентаря транзакционным.
Мы настоятельно рекомендуем создавать такие возможности, чтобы потенциально связаться с бронирующим по более высокой цене в процессе бронирования.

:::tip
Место может быть доступно для продажи через ABS. Его можно сделать доступным для продажи разными способами:

- Отдельные билеты.
- Групповая поездка.
- Встреча с призраком.
- +++ дайте волю своему воображению и заработайте дополнительные деньги!!
:::

- **Ценовой уровень** Дайте путешественнику быстрое представление о том, насколько дорогое это место.
- **Активно** Переключите тумблер в положение 🛑, если хотите, чтобы это место стало недоступным для бронирования во всех каналах продаж.
- **Комиссионное** Укажите, получает ли партнер комиссию с этого инвентаря.
- **Рекомендуемое** Укажите, должно ли это место занимать более заметное место, если такая возможность представится.
- **Продажи на основе атрибутов** Переключите тумблер в положение ✅, чтобы включить ABS для этого инвентаря. *После включения необходимо добавить как минимум 1 товар ABS для продажи*

### Дополнительный товар

При включении ABS вы можете начать добавлять дополнительные товары и делать их доступными для продажи.

*По умолчанию для вас уже создан пустой товар. Вы можете добавить больше, нажав кнопку `Add item for sale`.*

- **Название** Дайте вашему дополнительному товару название. *например, Билет в Дом с привидениями*
- **Тип ценообразования** Укажите, как вы хотите рассчитывать цену. *например, Установить `Per day`, чтобы цена зависела от количества дней.*
- **Базовая цена** Введите обычную цену этого товара. *например, 200 / день*
- **Скидочная цена** Введите цену, которую получают пользователи Wink при бронировании товара. *например, 150 / день*

### Лимиты

Вы можете ограничить доступность этого товара, установив минимальные и максимальные ограничения по количеству.

- **Минимальное количество** Ограничение на минимальное количество, которое нужно забронировать. *например, 1 участник*
- **Максимальное количество** Ограничение на максимальное количество, которое нужно забронировать. *например, 4 участника*

### Описания

:::note
Необходимо написать хотя бы одно описание, и оно должно быть на английском языке.
:::

- **Название** Дайте вашему дополнительному товару название. *например, Групповая поездка*
- **Описание** Опишите дополнительный товар в одном-двух абзацах. *например, Напугайте всех своих друзей до смерти.*

## Близость

Нажмите на вкладку `Proximity`, чтобы продолжить.

Не весь инвентарь находится на территории. Если у вас есть инвентарь, расположенный вне территории, сообщите об этом путешественнику здесь, и вам будет предоставлена возможность установить отдельное местоположение на карте вместе с дополнительной контактной информацией.

*Это может быть полезно, когда место находится в нескольких километрах от отеля, и путешественник хочет объяснить таксисту, как туда добраться.*

Нажмите кнопку `Save`, чтобы продолжить.
