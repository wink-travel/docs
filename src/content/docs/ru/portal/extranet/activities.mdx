Активности позволяют демонстрировать и продавать ваши активности как на территории, так и за её пределами. Чтобы создать активность, нажмите кнопку `Create activity`.

В следующих разделах подробно описан каждый этап управления активностью.

## Конфигурация

Вкладка `Configurations` выбрана по умолчанию при начале работы.

В этом разделе рассматриваются основные параметры вашей активности.

- **Внутреннее название** Название активности, под которым вы её знаете внутри системы. *например, Ловля нахлыстом*
- **Стиль жизни** По желанию выберите стиль жизни, который подходит для этой активности. *например, Спорт*
- **Возраст** По желанию выберите минимальную рекомендуемую возрастную группу для этой активности. *например, Взрослые*
- **Тип** По желанию выберите тип активности, который больше всего подходит. *например, Рыбалка*
- **Требуется бронирование** Нужно ли бронирование для этой активности. *например, Да*

## Расписание

Этот раздел позволяет управлять временем доступности активности.

### Часы работы

- **Открытие** Время суток, когда активность становится доступной. *например, 09:00*
- **Закрытие** Время суток, когда активность становится недоступной. *например, 21:00*

### Дни работы

Этот раздел позволяет установить дни недели, когда активность доступна.

*Переключите тумблер в положение ✅, чтобы указать, что день доступен.*

### Сезонность

:::note
Нет удобного, нативного поля для выбора Дня / Месяца. В этом случае используется только месяц и день из этого селектора даты.
:::

Этот раздел позволяет установить даты, когда активность доступна в течение года.

- **Дата начала** Дата, когда активность становится доступной. *например, 11 сентября 2024*
- **Дата окончания** Дата, когда активность становится недоступной. *например, 11 июля 2024*

## Удобства

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Amenities`, чтобы продолжить.

Подобно удобствам объекта размещения, удобства активности применяются только к конкретной активности. Отметьте каждое удобство, которое применимо.

## Описания

:::note
Необходимо написать хотя бы одно описание, и оно должно быть на английском языке.
:::

Нажмите на вкладку `Descriptions`, чтобы продолжить.

Подобно приветственному тексту объекта размещения, вы можете создавать локализованные описания для активности.
Вы можете писать на любом количестве языков. Мы добавим переводы для всех самых популярных языков.
Это текст, который увидят путешественники при просмотре ваших активностей.

- **Название** Дайте активности название. *например, Ловля нахлыстом*
- **Описание** Опишите активность в одном-двух абзацах. *например, Только вы... лес... река... Спокойствие в бутылке.*

## Социальные сети

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Social media`, чтобы продолжить.

Возможно, вы задаётесь вопросом: «Зачем активности собственный раздел социальных сетей?». Активность может быть также объектом рядом с отелем, но вне его территории, с собственным персоналом и собственным профилем в Instagram.

Социальные сети помогают поддерживать связь с новыми и существующими путешественниками со временем.
Добавьте свои аккаунты в социальных сетях в профиль, чтобы гости могли легко связаться с вами и узнать об этой активности.

<Steps>
1. Включите переключатель социальной сети, которую хотите активировать.
2. Введите название вашего аккаунта в этой сети.
</Steps>

## Репутация

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Reputation`, чтобы продолжить.

Подобно вкладке `Social media`, вы можете использовать эту вкладку, если у этой активности есть отдельные оценки репутации или дополнительные награды, которые вы хотите выделить здесь.

Ваша репутация — это ваша цифровая валюта, и вы хотите иметь возможность брать её с собой куда угодно.
Запись о репутации может быть чем угодно, за что ваш объект хочет получить признание, например, рейтинг от третьей стороны или победа вашего шефа на конкурсе.
В этом разделе вы можете добавить свои оценки и рейтинги со всего Интернета.

Нажмите кнопку `Add reputation score`.

<Steps>
1. **Категория** Выберите тип оценки, которую хотите добавить. *например, Отзыв третьей стороны*
2. **Поставщик** Введите, кто присвоил вам оценку. *например, BigOTA.com*
3. **Тип рейтинга** Выберите форму, в которой вы получили оценку. *например, Числовой*
4. **Дата** По желанию введите дату получения оценки.
5. **Рейтинг** Введите полученную оценку. *например, 8.5*
6. **Максимальный рейтинг** По желанию введите максимальный возможный балл для этого типа награды. *например, 10*
</Steps>

## Продажи на основе атрибутов

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Attribute Based Selling`, чтобы продолжить.

Все наши объекты имеют собственный раздел ABS. Этот раздел позволяет создавать возможности дополнительной продажи в основном инвентаре или делать не транзакционный тип инвентаря транзакционным.
Мы настоятельно рекомендуем создавать такие возможности, чтобы потенциально связаться с бронирующим по более высокой цене в процессе бронирования.

:::tip
Активность может быть доступна для продажи через ABS. Её можно продавать разными способами:

- Однодневная поездка.
- Трёхдневная поездка.
- +++ дайте волю своему воображению и заработайте дополнительные деньги!!
:::

- **Ценовой уровень** Дайте путешественнику быстрое представление о стоимости этой активности.
- **Активна** Переключите в положение 🛑, если хотите сделать эту активность недоступной для бронирования во всех каналах продаж.
- **Комиссионная** Укажите, получает ли партнёр комиссию с этого инвентаря.
- **Рекомендуемая** Укажите, должна ли эта активность занимать более заметное место, если такая возможность есть.
- **Продажи на основе атрибутов** Переключите в положение ✅, чтобы включить ABS для этого инвентаря. *После включения необходимо добавить как минимум 1 товар ABS для продажи*

### Дополнительный товар

При включении ABS вы можете начать добавлять дополнительные товары и делать их доступными для продажи.

*По умолчанию для вас уже создан пустой товар. Вы можете добавить больше, нажав кнопку `Add item for sale`.*

- **Название** Дайте вашему дополнительному товару название. *например, Однодневная поездка*
- **Тип ценообразования** Укажите, как вы хотите рассчитывать цену. *например, Установить на `Per day`, чтобы цена зависела от количества дней.*
- **Базовая цена** Введите обычную цену этого товара. *например, 200 / день*
- **Скидочная цена** Введите цену, которую получают пользователи Wink при бронировании товара. *например, 150 / день*

### Лимиты

Вы можете ограничить доступность этого товара, установив минимальные и максимальные ограничения по количеству.

- **Минимальное количество** Ограничение на минимальное количество, которое нужно забронировать. *например, 1 участник*
- **Максимальное количество** Ограничение на максимальное количество, которое нужно забронировать. *например, 4 участника*

### Описания

:::note
Необходимо написать хотя бы одно описание, и оно должно быть на английском языке.
:::

- **Название** Дайте вашему дополнительному товару название. *например, Однодневная поездка*
- **Описание** Опишите дополнительный товар в одном-двух абзацах. *например, Возьмите выходной и отправляйтесь к реке.*

## Близость

Нажмите на вкладку `Proximity`, чтобы продолжить.

Не весь инвентарь находится на территории объекта. Если у вас есть инвентарь, расположенный вне территории, сообщите об этом путешественнику здесь, и вам будет предоставлена возможность установить отдельное местоположение на карте вместе с дополнительной контактной информацией.

*Это может быть полезно, когда активность находится в нескольких километрах от отеля, и путешественник хочет объяснить таксисту, как туда добраться.*

Нажмите кнопку `Save`, чтобы продолжить.
