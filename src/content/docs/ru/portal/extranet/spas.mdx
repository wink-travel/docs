Спа позволяют демонстрировать и продавать ваши спа как на территории, так и за её пределами. Чтобы создать спа, нажмите кнопку `Create spa`.

В следующих разделах подробно описан каждый этап управления спа.

## Конфигурация

Вкладка `Configurations` выбрана по умолчанию при начале работы.

В этом разделе рассматриваются основные настройки вашего спа.

- **Внутреннее имя** Название спа, под которым вы его знаете внутри системы. *например, Santika Spa*

## Расписание

Этот раздел позволяет управлять временем доступности спа.

### Часы работы

- **Открытие** Время суток, когда спа становится доступным. *например, 09:00*
- **Закрытие** Время суток, когда спа становится недоступным. *например, 21:00*

### Дни работы

Этот раздел позволяет установить дни недели, когда спа доступен.

*Переключите тумблер в положение ✅, чтобы указать, что день доступен.*

### Сезонность

:::note
Нет удобного, нативного поля для выбора дня и месяца. В этом случае используется только месяц и день из этого селектора даты.
:::

Этот раздел позволяет установить даты, когда спа доступен в течение года.

- **Дата начала** Дата, с которой спа становится доступным. *например, 11 сентября 2024*
- **Дата окончания** Дата, с которой спа становится недоступным. *например, 11 июля 2024*

## Удобства

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Amenities`, чтобы продолжить.

Подобно удобствам объекта размещения, удобства спа применяются только к конкретному спа. Отметьте каждое удобство, которое применимо.

## Описания

:::note
Необходимо написать хотя бы одно описание, и оно должно быть на английском языке.
:::

Нажмите на вкладку `Descriptions`, чтобы продолжить.

Подобно приветственному тексту объекта размещения, вы можете создавать локализованные описания для спа.
Вы можете писать на любом количестве языков. Мы добавим переводы для всех самых популярных языков.
Это текст, который увидят путешественники при просмотре ваших спа.

- **Название** Дайте спа название. *например, Santika Spa*
- **Описание** Опишите спа в одном-двух абзацах. *например, Это лучший спа в мире...*

## Социальные сети

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Social media`, чтобы продолжить.

Возможно, вы задаётесь вопросом: «Почему у спа есть отдельный раздел социальных сетей?». Спа может быть объектом, расположенным рядом с отелем, но вне его территории, с собственным персоналом и отдельным профилем в Instagram.

Социальные сети помогают поддерживать связь с новыми и постоянными путешественниками со временем.
Добавьте свои аккаунты в социальных сетях в профиль, чтобы гости могли легко связаться с вами и узнать о спа.

<Steps>
1. Включите тумблер социальной сети, которую хотите активировать.
2. Введите название вашего аккаунта в этой сети.
</Steps>

## Репутация

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Reputation`, чтобы продолжить.

Подобно вкладке `Social media`, вы можете использовать эту вкладку, если у спа есть отдельные оценки репутации или дополнительные награды, которые вы хотите выделить здесь.

Ваша репутация — это ваша цифровая валюта, и вы хотите иметь возможность брать её с собой куда угодно.
Запись о репутации может включать всё, чем ваш объект хочет гордиться, например, рейтинг от третьей стороны или победу вашего шефа на конкурсе.
В этом разделе вы можете добавить свои оценки и рейтинги со всего Интернета.

Нажмите кнопку `Add reputation score`.

<Steps>
1. **Категория** Выберите тип оценки, которую хотите добавить. *например, Отзыв третьей стороны*
2. **Поставщик** Введите, кто присвоил вам оценку. *например, BigOTA.com*
3. **Тип рейтинга** Выберите форму, в которой вы получили оценку. *например, Числовой*
4. **Дата** По желанию, введите дату получения оценки.
5. **Рейтинг** Введите полученную оценку. *например, 8.5*
6. **Максимальный рейтинг** По желанию, введите максимальный возможный балл для этого типа награды. *например, 10*
</Steps>

## Продажи на основе атрибутов

:::note
Этот раздел необязателен.
:::

Нажмите на вкладку `Attribute Based Selling`, чтобы продолжить.

Все наши объекты имеют собственный раздел ABS. Этот раздел позволяет создавать возможности дополнительной продажи в основном инвентаре или делать не транзакционный тип инвентаря транзакционным.
Мы настоятельно рекомендуем создавать такие возможности, чтобы потенциально связаться с бронирующим по более высокой цене в процессе бронирования.

:::tip
Спа можно сделать доступным для продажи через ABS. Это можно сделать разными способами:

- Рефлексология.
- День ухода за собой.
- Уход для двоих.
- Специальный шиацу.
- +++ дайте волю своему воображению и заработайте дополнительные деньги!!
:::

- **Ценовой уровень** Дайте путешественнику быстрое представление о стоимости этого спа.
- **Активно** Переключите тумблер в положение 🛑, если хотите сделать спа недоступным для бронирования во всех каналах продаж.
- **Комиссионный** Укажите, получает ли партнёр комиссию с этого инвентаря.
- **Рекомендуемый** Укажите, должен ли этот спа занимать более заметное место, если такая возможность появится.
- **Продажи на основе атрибутов** Переключите тумблер в положение ✅, чтобы включить ABS для этого инвентаря. *После включения необходимо добавить как минимум 1 товар ABS для продажи*

### Дополнительный товар

При включении ABS вы можете начать добавлять дополнительные товары и делать их доступными для продажи.

*По умолчанию для вас уже создан пустой товар. Вы можете добавить больше, нажав кнопку `Add item for sale`.*

- **Название** Дайте вашему дополнительному товару название. *например, Рефлексология*
- **Тип ценообразования** Укажите, как вы хотите рассчитывать цену. *например, Установите `Per hour`, чтобы цена зависела от количества часов аренды.*
- **Базовая цена** Введите обычную цену этого товара. *например, 20 / час*
- **Скидочная цена** Введите цену, которую получают пользователи Wink при бронировании товара. *например, 15 / час*

### Лимиты

Вы можете ограничить доступность этого товара, установив минимальные и максимальные ограничения по количеству.

- **Минимальное количество** Ограничение на минимальное количество для бронирования. *например, 1 участник*
- **Максимальное количество** Ограничение на максимальное количество для бронирования. *например, 4 участника*

### Описания

:::note
Необходимо написать хотя бы одно описание, и оно должно быть на английском языке.
:::

- **Название** Дайте вашему дополнительному товару название. *например, Рефлексология*
- **Описание** Опишите дополнительный товар в одном-двух абзацах. *например, Узнайте, насколько великолепен массаж стоп, выполненный профессиональным рефлексологом. Вперед, дайте волю фантазии.*

## Близость

Нажмите на вкладку `Proximity`, чтобы продолжить.

Не весь инвентарь находится на территории. Если у вас есть инвентарь, расположенный вне территории, сообщите об этом путешественнику здесь, и вам будет предоставлена возможность установить отдельное местоположение на карте вместе с дополнительной контактной информацией.

*Это может быть полезно, если спа находится в нескольких километрах от отеля, и путешественник хочет объяснить таксисту, как туда добраться.*

Нажмите кнопку `Save`, чтобы продолжить.
