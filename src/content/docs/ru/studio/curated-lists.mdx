Курируемые списки — это функция, которая позволяет вам группировать и сортировать инвентарь, который вы выбрали из вещей, которые вы нашли при поиске инвентаря для путешествий.
После создания вы можете показать список своим пользователям, сделав его доступным для продажи в качестве[сетка](/studio/grids)или[карта](/studio/maps) через [WinkLinks](/link-manager/wink-links), наш [плагин WordPress](/developers/wordpress/)или путем встраивания одного из наших[Веб-компоненты](/developers/web-components)на ваш сайт.

По умолчанию каждая партнерская учетная запись начинается с`Favorites`список, который вы можете использовать прямо сейчас.`Favorites`является особенным и не может быть удален.
Все курируемые списки позволяют вам делать следующее:

## Создать список

<figure>
  <CloudinaryImage width="350" height="219" resize="scale" quality="best" alt="Создать список избранных" src="academy/create-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Создать новую форму кураторского списка*</figcaption>
</figure>


* На панели инструментов нажмите`Inventory > Curated Lists`на панели навигации.
* Создайте новый кураторский список, нажав кнопку`Make a new list` кнопка.
* Вам будет предложено дать название вашему списку.*Пример: Топ-3 выбора Нью-Йорка*.
* Нажмите на`Save` кнопка.

## Добавить элемент списка

Вы можете добавить элемент списка в любой список из результатов поиска.

* Перейти к[Поиск](/studio/search)
* Найдите то, что вы хотите добавить в свой список.
* Как только вы его найдете, нажмите кнопку`Actions`ссылка в левом нижнем углу на карточке результатов поиска.
* Нажмите`Add to...`а затем выберите список, в который вы хотите добавить этот инвентарь.
* Вернитесь на страницу отобранных списков, чтобы убедиться, что элемент добавлен в ваш список.

С каждым списком можно выполнять некоторые действия.

* Преобразовать список в[карта](/studio/maps).
* Преобразовать список в[сетка](/studio/grids).
* Удалить список (*за исключением избранного*).
