Курируемые списки — это функция, которая позволяет группировать и сортировать инвентарь, выбранный вами из найденного при поиске туристического инвентаря.  
После создания вы можете предоставить список своим пользователям, сделав его доступным для продажи в виде [сетки](/ru/studio/grids) или [карты](/ru/studio/maps) через [WinkLinks](/ru/link-manager/wink-links), наш [плагин для WordPress](/ru/developers/wordpress/) или встроив один из наших [веб-компонентов](/ru/developers/web-components) на ваш сайт.

По умолчанию у каждого партнёрского аккаунта есть список `Избранное`, который вы можете использовать сразу. `Избранное` особенный и не может быть удалён.  
Все курируемые списки позволяют выполнять следующие действия:

## Создать список

- В панели управления нажмите `Инвентарь > Курируемые списки` в навигационной панели.  
- Создайте новый курируемый список, нажав кнопку `Создать новый список`.  
- Вас попросят назвать список. *Пример: Топ 3 выбора в Нью-Йорке*.  
- Нажмите кнопку `Сохранить`.

## Добавить элемент в список

Вы можете добавить элемент в любой список из результатов поиска.

- Перейдите в [Поиск](/ru/studio/search)  
- Найдите то, что хотите добавить в список.  
- Когда найдёте, нажмите ссылку `Действия` в левом нижнем углу карточки результата поиска.  
- Нажмите `Добавить в...` и выберите курируемый список, в который хотите добавить этот инвентарь.  
- Вернитесь на страницу курируемых списков, чтобы убедиться, что элемент добавлен.

Для каждого списка доступны следующие действия:

- Преобразовать список в [карту](/ru/studio/maps).  
- Преобразовать список в [сеть](/ru/studio/grids).  
- Удалить список (*кроме Избранного*).
