Ссылки для совместного использования — это основная функция Wink, которая возникла из необходимости динамично передавать информацию об инвентаре и ценах самым простым способом в Интернете — через гиперссылку.

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

Выше приведён пример одной из наших ссылок. Она короткая и удобная для передачи... даже достаточно короткая, чтобы её запомнить. Несмотря на краткость, эта ссылка содержит **огромное** количество информации, такой как:

- Ваш App ID.
- Ваш Affiliate ID.
- Изображение(я) и текст, которые будут отображаться пользователю.
- Внешний вид (*тема, цвета и многое другое*).
- Маршрут для запроса цены (*2 взрослых, 1 ночь*).
- Валюта для отображения цены.
- Язык для отображения текста.

Далее в статье показано, как создавать, настраивать и делиться вашими ссылками.

## Настроить ссылку

Форма позволяет настроить вашу ссылку следующими способами:

<Steps>
 1. Дайте ей имя, чтобы помнить, о чём эта ссылка.
 2. Выберите [Customization](/ru/studio/customization), которую хотите применить к этой ссылке.
 3. Добавьте ключевые слова, разделённые запятыми, которые будут использоваться [веб-краулерами](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. По желанию введите ваш аккаунт в X или Facebook App ID.
 5. Добавьте заголовки и описания на языках, которые вы хотите показать пользователю.
 6. Выберите одно или несколько изображений, которые будут отображаться при публикации ссылки на платформах, таких как Facebook или LinkedIn.
 7. Выберите тему / оверлей, который добавит фирменный дизайн поверх выбранных изображений. *Примечание: именно так вы можете отображать цены поверх изображения.*
 8. Нажмите кнопку `Сохранить`, чтобы продолжить.
 </Steps>

На изображении выше показаны все доступные действия для вашей ссылки:

- **Предпросмотр** Показывает краткий обзор того, как настроена ссылка и как она будет выглядеть для пользователя.
- **Обновить** Обновляет конфигурацию вашей ссылки.
- **Добавить в WinkLinks** Добавляет ссылку в ваш аккаунт WinkLinks.
- **Копировать** Копирует ссылку в буфер обмена, чтобы вы могли легко поделиться ею где угодно (*включая [WinkLinks](/ru/link-manager/wink-links)*).
- **Поделиться** Открывает новое окно с опциями для публикации в различных социальных сетях и мессенджерах.
- **QR-код** Генерирует QR-код, который можно использовать на платформах, ориентированных на изображения, таких как IG.
- **Встроить** Показывает, как встроить эту ссылку в виде [рекламного баннера](/ru/developers/web-components/#content-loader) на ваш сайт.
- **Использовать с WordPress** Показывает, как использовать наш [плагин для WordPress](/ru/developers/wordpress) для встраивания этой ссылки на ваш сайт.

Некоторые из этих опций мы рассмотрим подробнее ниже.

Если вы хотите поделиться QR-кодом вместо ссылки, нажмите кнопку `QR-код`, и откроется новое окно с вашим QR-кодом.

Как сохранить QR-код:

- Если вы на ноутбуке, кликните правой кнопкой мыши по изображению и выберите `Сохранить изображение`.
- Если вы на мобильном телефоне:
    - Нажмите и удерживайте палец на изображении, пока не появится опция сохранить изображение в Фото.
    - Сделайте скриншот QR-кода, обрежьте его по размеру и сохраните в Фото.

### Встроить

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Вот как встроить вашу ссылку в виде рекламного баннера на ваш сайт:

- Строка 3 показывает, как подключить стили Wink на ваш сайт.
- Строки 6–9 показывают, как использовать Web Component [wink-content-loader](/ru/developers/web-components/#content-loader) и указать ему загрузить рекламный баннер для вашего кода.
- Строка 11 показывает, как встроить наш Javascript на ваш сайт.
- Строка 13 показывает, как встроить Web Component [wink-app-loader](/ru/developers/web-components/#application-loader) и указать ему загрузить ваши настройки конфигурации страницы.
