Панель управления Studio предоставляет быстрый обзор эффективности вашего аффилиата.

## Активность

Раздел активности показывает аналитику бронирований — сколько бронирований вы привели, сгенерированный доход и тенденции конверсии с течением времени.

## Поиск поставщиков

Используйте быстрый поиск, чтобы найти поставщиков по названию объекта. Это ярлык для просмотра сети поставщиков.

## Статус аккаунта

Панель управления показывает текущий статус вашего аккаунта — заполнен ли ваш профиль и готов ли он приносить комиссионные.

## Таблица лидеров

Узнайте, как вы ранжируетесь по сравнению с другими аффилиатами на основе показателей эффективности.
