Сохранённые поиски делают именно то, что подразумевает их название. Во время поиска туристического инвентаря у вас есть возможность сохранить запрос для последующего использования.

## Сохранение поискового запроса

Когда вы начинаете поиск инвентаря (см. [Search](/ru/studio/search)), у вас есть возможность точно настроить фильтры поиска.  
Внизу формы запроса есть опция `Save search`.

Нажав на эту ссылку, вы сможете дать имя этому поиску (*например, Моя библиотека волшебных отелей*), чтобы потом легко его вспомнить.

<figure>
<CloudinaryImage
      width="350"

      quality="best"
      alt="Дайте имя вашему поиску"
      src="academy/name-save-search"
      errorImage="noimage_opaque_nyrtl0.png"
    />
    <figcaption>*Дайте имя вашему поиску*</figcaption>
</figure>

Присвойте имя и нажмите кнопку `Save search`. После этого вы можете перейти в раздел `Inventory > Saves Searches` в главном меню, чтобы увидеть созданную запись.

## Действия со списком

Ваш сохранённый поиск можно использовать для отображения туристического инвентаря вашим пользователям в виде сетки или карты и настроить отображение результатов поиска так, как вам нужно.

Для каждого сохранённого поискового запроса доступны следующие действия:

- Преобразовать поисковый запрос в [карту](/ru/studio/maps).
- Преобразовать поисковый запрос в [сетки](/ru/studio/grids).
- Удалить сохранённый поиск.

## API

Разработчики, желающие управлять сохранёнными поисками, могут перейти в раздел [Developers > API > Browse](/ru/developers/apis/#browse-api).
