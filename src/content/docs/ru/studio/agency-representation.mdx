Агентство — это обычный аккаунт Wink, который вы можете назначить своему аккаунту для целей представительства. Ниже приведены некоторые причины, по которым вам может понадобиться агентство, представляющее вас:

- Полное / частичное управление вашим аккаунтом Wink третьей стороной.
- Талант имеет существующее соглашение со своим агентством и хочет, чтобы Wink занимался финансовой стороной их договорённости по поступающим через Wink средствам.
- Агентство талантов выполняет всю онлайн-работу и управляет всеми аккаунтами на Wink.

## Назначение агентства

Вот как назначить агентство вашему аккаунту.

<Steps>
 1. Войдите в свой пользовательский аккаунт на [Wink Studio](https://studio.wink.travel).
 2. Выберите аффилированный аккаунт, с которым хотите работать.
 3. Нажмите `Account > Managers` в главном навигационном меню.
 4. В разделе `Agency` введите название агентства, которое вы хотите назначить представлять вас.
 5. Выберите комиссию агентства из выпадающего списка.
 5. При необходимости добавьте правила истечения срока действия.
 7. Нажмите кнопку `Set agency`, чтобы продолжить.
 </Steps>

 :::note
Вы должны быть владельцем аккаунта, чтобы назначить агентство.
 :::

 :::tip[Пример]
Комиссия агентства исходит из комиссии аффилиата.

Бронирование = $100, Комиссия = 10%, Комиссия агентства = 10% => 100 * 0.1 * 0.1 = $1
 :::

## Снятие агентства с назначения

Вот как снять назначение агентства с вашего аккаунта.

<Steps>
 1. Войдите в свой пользовательский аккаунт на [Wink Studio](https://studio.wink.travel).
 2. Выберите аффилированный аккаунт, с которым хотите работать.
 3. Нажмите `Account > Managers` в главном навигационном меню.
 4. В разделе `Agency` нажмите на иконку <Badge text="x" variant="danger" size="small" /> справа от выпадающего списка агентства.
 5. Нажмите кнопку `Set agency`, чтобы продолжить.
 </Steps>


## API

Разработчики, которые хотят управлять агентством через API, могут перейти в раздел [Developers > API > Affiliate](/ru/developers/apis/#affiliate-api).
