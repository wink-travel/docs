**Расширение Wink для TYPO3** интегрирует инвентарь Wink для путешествий в TYPO3 13 в виде нативных элементов контента. Редакторы добавляют компоненты Wink на страницы с помощью стандартного редактора страниц TYPO3 — без необходимости менять шаблоны.

## Требования

- TYPO3 13.x
- PHP 8.2+
- Composer

## Установка

<Steps>
1. Установите через Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Активируйте расширение:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. В бэкенде TYPO3 перейдите в **Admin Tools → Settings → Extension Configuration → wink**.
4. Введите ваш **Client ID**, **Client Secret** и выберите окружение. Сохраните.
   Перейдите в [Applications](/ru/portal/settings/applications), чтобы получить ваши учетные данные.
5. Добавьте статический шаблон TypoScript в корневой шаблон:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Добавление элемента контента на страницу

<Steps>
1. Откройте страницу в модуле **Page**.
2. Нажмите кнопку **+** в области контента, чтобы добавить новый элемент.
3. Выберите один из элементов контента **Wink Travel**:
   - **Wink Content** — карточки отелей, ранжированные сетки, карты
   - **Wink Lookup** — строка поиска направления
   - **Wink Search** — кнопка выбора маршрута
   - **Wink Account** — кнопка входа в аккаунт
   - **Wink Itinerary** — кнопка статуса маршрута
   - **Wink Cart** — кнопка корзины покупок
4. Для **Wink Content** используйте поля FlexForm, чтобы выбрать **Layout Type** и ввести **Layout ID**.
5. Сохраните и опубликуйте страницу.
</Steps>

## Доступные элементы контента

| Элемент | Назначение |
|---|---|
| **Wink Content** | Карточки отелей, ранжированные сетки, карты и другое |
| **Wink Lookup** | Строка поиска направления |
| **Wink Search** | Иконка-кнопка, открывающая выбор маршрута |
| **Wink Account** | Кнопка входа / пользовательского аккаунта |
| **Wink Itinerary** | Кнопка маршрута, показывающая текущую поездку |
| **Wink Cart** | Кнопка с краткой информацией о корзине покупок |
