Приложение **Wink Wix App** добавляет веб-компоненты Wink на ваш сайт Wix в виде нативных виджетов, которые можно перетаскивать в редакторе Wix. Владельцам сайта не требуется программирование.

## Установка

<Steps>
1. Перейдите в [Wix App Market](https://www.wix.com/app-market) и найдите **Wink Travel**.
2. Нажмите **Добавить на сайт** и следуйте инструкциям для авторизации приложения.
3. После установки откройте **Панель управления → Настройки Wink** и введите ваш **Client ID** и **Client Secret**.
   Перейдите в [Applications](/ru/portal/settings/applications), чтобы получить ваши учетные данные.
4. Нажмите **Сохранить**, затем **Проверить соединение**, чтобы подтвердить подключение Wink.
</Steps>

## Добавление компонента на страницу

<Steps>
1. Откройте **Редактор Wix** и перейдите на страницу, куда хотите добавить контент Wink.
2. Нажмите **Добавить (+) → Приложения → Wink Travel**, чтобы увидеть все доступные виджеты.
3. Перетащите виджет на страницу — например, **Wink Content** для отображения сетки отелей.
4. В панели настроек виджета выберите **Тип макета** и вставьте **Layout ID** из Wink Studio.
5. Нажмите **Опубликовать**, чтобы применить изменения.
</Steps>

## Доступные виджеты

| Виджет | Назначение |
|---|---|
| **Wink Content** | Отображает карточки отелей, рейтинговые сетки, карты и другое |
| **Wink Lookup** | Строка поиска по направлениям и отелям |
| **Wink Search** | Кнопка-иконка, открывающая выбор маршрута |
| **Wink Account** | Кнопка входа / пользовательского аккаунта |
| **Wink Itinerary** | Кнопка маршрута, показывающая текущую поездку |
| **Wink Cart** | Кнопка с краткой информацией о корзине покупок |
