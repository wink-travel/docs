Приложение **Wink Shopify App** позволяет туроператорам, продавцам впечатлений и торговцам в сфере гостеприимства отображать актуальный инвентарь Wink прямо на витрине Shopify — рядом с товарами, на лендингах или в любом месте темы.

## Установка

<Steps>
1. Перейдите в [Shopify App Store](https://apps.shopify.com) и найдите **Wink Travel**.
2. Нажмите **Add app** и следуйте процессу OAuth для авторизации.
3. В настройках приложения Wink введите ваш **Client ID** и **Client Secret**.
   Перейдите в [Applications](/ru/portal/settings/applications), чтобы получить ваши учетные данные.
4. Нажмите **Verify & Save**. Появится сообщение об успешном подключении.
</Steps>

## Включите блок App Embed

Блок App Embed автоматически добавляет скрипт и таблицу стилей Wink на каждую страницу — редактировать код темы не нужно.

<Steps>
1. В админке Shopify перейдите в **Online Store → Themes → Customize**.
2. В левой панели нажмите **App embeds**.
3. Включите переключатель **Wink Travel**.
4. Нажмите **Save**.
</Steps>

:::note
Блок App Embed считывает ваш Client ID из метаполя Shopify, установленного при установке. Ваш Client Secret никогда не передается в браузер.
:::

## Добавление компонента на страницу

<Steps>
1. В редакторе темы откройте страницу или раздел, куда хотите добавить контент Wink.
2. Нажмите **Add block** и выберите один из блоков Wink:
   - **Wink Content** — карточки отелей, сетки или карты
   - **Wink Lookup** — строка поиска по направлениям
   - **Wink Search** — кнопка выбора маршрута
   - **Wink Account** — кнопка входа в аккаунт
   - **Wink Itinerary** — кнопка статуса маршрута
   - **Wink Cart** — кнопка корзины покупок
3. Для **Wink Content** выберите **Layout Type** и введите **Layout ID** из Wink Studio.
4. Нажмите **Save**.
</Steps>

## Доступные блоки темы

| Блок | Назначение |
|---|---|
| **Wink Content** | Отображает карточки отелей, ранжированные сетки, карты и другое |
| **Wink Lookup** | Строка поиска направлений и отелей |
| **Wink Search** | Иконка-кнопка, открывающая выбор маршрута |
| **Wink Account** | Кнопка входа / аккаунта пользователя |
| **Wink Itinerary** | Кнопка маршрута с текущей поездкой |
| **Wink Cart** | Кнопка с краткой информацией о корзине покупок |
