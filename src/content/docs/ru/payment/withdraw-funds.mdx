:::note
Если вы не являетесь интегратором, вам не нужно создавать аккаунт в TripPay. Интегратор сделает это за вас в момент создания аккаунта у них.
:::

Получатели платежей используют TripPay для:

- Отслеживания, сколько денег они заработали.
- Сколько средств сейчас на их счёте.
- ...и какую сумму можно вывести.

## Использование

Обычный получатель платежей обычно взаимодействует с TripPay следующим образом:

<Steps>
1. TripPay удерживает средства с 10 бронирований с 3 разных туристических платформ для отеля X, и средства теперь доступны.
2. Отель X получает письмо от TripPay с сообщением `У вас есть доступные средства` и ссылкой для управления аккаунтом.
3. Отель X входит в [TripPay](https://pay.wink.travel).
4. Отель X видит бронирования за прошлый месяц вместе с аналитикой аккаунта.
5. Отель X выбирает вывод средств.
6. Отель X выбирает метод `Выплата`.
7. Отель X выполняет необходимые шаги для обработки средств.
8. Отель X отправляет запрос на перевод и нажимает `OK`.
</Steps>

*Средства обычно становятся доступны в течение 24 часов во всех регионах.*

:::note
Когда вы достигнете определённого объёма и суммы, мы можем связаться с вами для предоставления дополнительных документов, подтверждающих вашу личность и/или личность вашей компании.
:::
