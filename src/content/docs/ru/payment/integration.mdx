Если вы туристическая компания и хотите передать на аутсорсинг часть вашего бизнеса, связанную с оплатой и выплатами, сервису TripPay, вот 6 основных шагов для начала:

<Steps>
1. Создайте [учетную запись пользователя](/ru/user/register) на Wink.
2. Создайте аккаунт в [TripPay](https://pay.wink.travel).
3. Создайте [приложение](/ru/developers/applications) в TripPay, которое позволит вам взаимодействовать с нашим API.
4. Создайте и сопоставьте ваших бенефициаров с TripPay. *например, для большинства это означает создание аккаунтов для объектов на вашей платформе, которым вы хотите, чтобы TripPay выплачивал средства.*
5. Интегрируйте [веб-компонент оплаты](/ru/developers/web-components/#payment) на ваш сайт.
6. Отправьте TripPay `корзину покупок` путешественника и инициируйте процесс оплаты.
</Steps>

:::tip
Для полного руководства по использованию TripPay на вашем туристическом сайте прочитайте руководство [Интеграция с TripPay](/ru/guides/integrating-with-trip-pay).
:::

## API

Разработчики, которые хотят управлять `платежами`, переходите в раздел [Разработчики > API > Платежи](/ru/developers/apis/#payment-api).
