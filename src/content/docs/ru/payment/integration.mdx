Если вы туристическая компания и хотите передать часть платежей и выплат вашего бизнеса на аутсорсинг TripPay, вот 6 основных шагов для начала работы:

<Steps>
  1. Создать [учетная запись пользователя](/user/register) на Wink.
  2. Создайте учетную запись с [TripPay](https://pay.wink.travel).
  3. Создайте [Приложение](/developers/applications) на TripPay, который позволяет вам взаимодействовать с нашим API.
  4. Создавайте и отображайте своих бенефициаров с помощью TripPay. *Например, для большинства это означает создание учетных записей для объектов недвижимости на вашей платформе, за которые вы хотите получать оплату через TripPay.*
  5. Интегрировать [веб-компонент оплаты](/developers/web-components/#payment) на ваш сайт.
  6. Отправьте TripPay путешественнику `shopping cart` и инициировать процесс оплаты.
</Steps>


:::tip
Полное руководство по использованию TripPay для вашего туристического сайта см. в руководстве[Интеграция с TripPay](/guides/integrating-with-trip-pay).
:::
