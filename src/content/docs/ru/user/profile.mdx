Если вы хотите обновить свой профиль пользователя для учетной записи Wink, выполните следующие шаги:

<Steps>
1. Вы должны быть вошли в одну из наших веб-платформ.
2. Нажмите на иконку профиля в правом верхнем углу, появится меню.
3. Нажмите на ссылку `Профиль`.
4. Следуйте инструкциям, чтобы обновить ваш `e-mail`, `имя` и `фамилию`.
5. Вы также можете загрузить новую `фотографию профиля`.
6. Нажмите `Сохранить`.
</Steps>

:::note
Ваша новая фотография профиля отобразится при следующем полном выходе из Wink.
:::
