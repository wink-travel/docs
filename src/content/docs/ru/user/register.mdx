Вы можете создать учетную запись пользователя через любой из наших порталов. После создания эта учетная запись может быть использована во всех приложениях и веб-сайтах Wink.

## Путешественники

Выполните следующие действия:

<Steps>
  1. Перейдите по ссылке [https://ota.wink.travel](https://ota.wink.travel)
  2. Нажмите на `Sign-In / Register` в правом верхнем углу
  3. Вам будет предложено ввести основные данные пользователя. Заполните все обязательные поля и нажмите `Save`.
  4. Теперь вы можете войти в Wink, используя свое имя пользователя (адрес электронной почты) и пароль.
  5. После входа в систему система перенаправит вас обратно туда, откуда вы пришли.
</Steps>

## Регистрация

Создание учетной записи пользователя Wink дает вам доступ ко всем функциям платформы. Вы можете создавать неограниченное количество объектов недвижимости и партнерских учетных записей.

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="Войти" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Форма входа*</figcaption>
</figure>


<Steps>
  1. При первом обращении на наш сервер авторизации вам будет предложено ввести имя пользователя и пароль.
  2. Вместо этого нажмите на `Create account` кнопка.
  3. Заполните все обязательные поля и нажмите `Register` для продолжения. Это вернет вас к форме входа.
  4. Введите свой адрес электронной почты / имя пользователя и пароль для входа в систему.
  5. Вы будете перенаправлены обратно на сайт, с которого пришли.
</Steps>
