Вы можете создать учетную запись пользователя через любой из наших порталов. После создания эта учетная запись может использоваться во всех приложениях и на сайтах Wink.

## Путешественники

Следуйте этим шагам:

<Steps>
1. Перейдите на [https://ota.wink.travel](https://ota.wink.travel)
2. Нажмите на `Sign-In / Register` в правом верхнем углу
3. Вас попросят ввести основные данные пользователя. Заполните все обязательные поля и нажмите `Save`.
4. Теперь вы можете войти в Wink, используя имя пользователя (электронную почту) и пароль.
5. После входа система перенаправит вас обратно туда, откуда вы пришли.
</Steps>

## Регистрация

Создание учетной записи пользователя Wink дает вам доступ ко всем функциям платформы. Вы можете создавать неограниченное количество объектов недвижимости и партнерских аккаунтов.

<figure>

      width="350"

    />
    <figcaption>*Форма входа*</figcaption>
</figure>

<Steps>
1. В первый раз, когда вы попадете на наш сервер авторизации, вас попросят ввести имя пользователя и пароль.
2. Вместо этого нажмите кнопку `Create account`.
3. Заполните все обязательные поля и нажмите `Register`, чтобы продолжить. Это вернет вас к форме входа.
4. Введите вашу электронную почту / имя пользователя и пароль для входа.
5. Вы будете перенаправлены обратно на сайт, с которого пришли.
</Steps>
