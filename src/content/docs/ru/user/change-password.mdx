Если вы хотите изменить пароль для своей учетной записи Wink, выполните следующие действия:

## Обновить пароль

<Steps>
  1. Вам необходимо войти на один из наших сайтов.
  2. Нажмите на значок своего профиля в правом верхнем углу, и появится меню.
  3. Нажмите на `Change password` связь.
  4. Следуйте инструкциям по смене пароля.
  5. Нажмите `Save`.
</Steps>


При следующем входе в систему вам понадобится новый пароль.

## Забыли пароль

Иногда необходимо сбросить пароль пользователя.

Вот шаги по сбросу пароля пользователя:

<Steps>
  1. Нажмите на `Forgot password` ссылка со страницы входа.
  2. Введите имя пользователя/адрес электронной почты для учетной записи, которую вы хотите сбросить.
  3. Нажмите `Forgot password` кнопка. *Вы будете перенаправлены на форму сброса пароля.*
  4. Проверьте свою электронную почту.
</Steps>

<Steps>
  1. Вставьте код, полученный вами по электронной почте, в новую форму вместе с вашим новым паролем.
  2. Подтвердите новый пароль.
  3. Нажмите `Reset password` кнопка. *Вы будете перенаправлены обратно на страницу входа.*
</Steps>

Ваш пароль успешно сброшен. Вы можете войти в Wink с вашими новыми учетными данными.
