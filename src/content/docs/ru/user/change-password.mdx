Если вы хотите изменить пароль для своей учетной записи пользователя Wink, выполните следующие действия:

## Обновление пароля

<Steps>
1. Вы должны быть вошли в одну из наших веб-страниц.
2. Нажмите на иконку профиля в правом верхнем углу, появится меню.
3. Нажмите на ссылку `Сменить пароль`.
4. Следуйте инструкциям для изменения пароля.
5. Нажмите `Сохранить`.
</Steps>

При следующем входе в систему вам потребуется новый пароль.

## Забыли пароль

Иногда необходимо сбросить пароль пользователя.

Вот шаги для сброса пароля пользователя:

<Steps>
1. Нажмите на ссылку `Забыли пароль` на странице входа.
2. Введите имя пользователя / электронную почту учетной записи, которую хотите сбросить.
3. Нажмите кнопку `Забыли пароль`. *Вы будете перенаправлены на форму сброса пароля.*
4. Проверьте свою электронную почту.
</Steps>

<Steps>
1. Вставьте код, который вы получили по электронной почте, в новую форму вместе с новым паролем.
2. Подтвердите новый пароль.
3. Нажмите кнопку `Сбросить пароль`. *Вы будете перенаправлены обратно на страницу входа.*
</Steps>
