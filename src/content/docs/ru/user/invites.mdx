Пользователь Wink может позволить вам управлять одним из своих аккаунтов собственности, филиала или турагента. Когда это произойдет, вы получите электронное письмо с запросом на приглашение.

import { Steps } from '@astrojs/starlight/components';

Чтобы просмотреть текущие приглашения Wink, выполните следующие действия:

<Steps>
  1. Вам необходимо войти на один из наших сайтов.
  2. Нажмите на значок своего профиля в правом верхнем углу, и появится меню.
  3. Нажмите на `Invites` связь.
  4. Взаимодействуйте с вашими приглашениями одним из следующих способов: `Accepting` или `Rejecting` их.
</Steps>
