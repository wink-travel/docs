Пользователь Wink может разрешить вам управлять одним из своих аккаунтов недвижимости, филиала или туристического агента. В этом случае вы получите электронное письмо с запросом на приглашение.

import { Steps } from '@astrojs/starlight/components';

Чтобы увидеть текущие приглашения Wink, выполните следующие действия:

<Steps>
1. Вы должны войти в один из наших сайтов.
2. Нажмите на значок профиля в правом верхнем углу, и появится меню.
3. Нажмите на ссылку `Invites`.
4. Взаимодействуйте с вашими приглашениями, либо `Приняв`, либо `Отклонив` их.
</Steps>
