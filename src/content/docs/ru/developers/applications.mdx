Приложения предоставляют полный доступ к функциям Wink и TripPay и позволяют вашим пользователям аутентифицироваться и входить на ваш сайт с помощью нашего сервера авторизации OAuth2; так же, как Facebook и Google.  
Вы можете создать собственное OTA с помощью Wink. Всё начинается с приложения.

*По умолчанию вы получаете приложение при регистрации нового партнёрского аккаунта Wink или аккаунта TripPay. Мы используем `client ID` в некоторых сценариях.*

:::note
Приложение, созданное в Wink или TripPay, можно использовать для всех API-эндпоинтов.
:::

*В этом примере мы будем аутентифицироваться через [Wink Studio](https://studio.wink.travel).*

<Steps>
1. После аутентификации нажмите на иконку профиля в правом верхнем углу, появится меню.
2. Нажмите на ссылку `Applications`.
3. Вы будете перенаправлены на страницу ваших приложений.
</Steps>

## Создать приложение

Поскольку у вас нет доступа к `секретному ключу` приложения, созданного для вас, вам нужно создать своё собственное.

:::caution
Вам необходимо создать партнёрский аккаунт в Wink или аккаунт в TripPay перед созданием приложения, так как вы будете привязывать приложение к конкретному аккаунту.
:::

Вот шаги для создания приложения:

<Steps>
1. На странице приложений нажмите кнопку `Create new application`.
2. ***Название*** Дайте вашему приложению имя. *например, Cool App*
3. **Название организации** Свяжите приложение с одним из ваших существующих аккаунтов. *например, Cool Account*
4. **Redirect URIs** Требуется как минимум один redirect URI. Он используется для проверки домена перенаправления после успешной аутентификации. *например, https://www.cool-site.com*
5. Нажмите кнопку `Save`, чтобы продолжить.
</Steps>

Вы будете перенаправлены обратно к списку приложений. В списке вы увидите новое приложение и получите доступ к секретному ключу приложения только один раз. Обязательно скопируйте его и сохраните в безопасном месте.

## Обновить приложение

Вам нужно обновить приложение, если изменился один из redirect URI или вы хотите добавить или удалить URI.

Чтобы обновить существующее приложение, выполните следующие действия:

<Steps>
1. Нажмите ссылку `Actions` для приложения, которое хотите обновить.
2. Нажмите кнопку `Update` в разделе Actions.
3. Внесите изменения в приложение.
4. Нажмите кнопку `Save`, чтобы продолжить.
</Steps>

Вы будете перенаправлены обратно к списку приложений.

## Удалить приложение

:::caution
Пожалуйста, не удаляйте приложение, созданное автоматически для вас, чтобы все функции вашего аккаунта работали корректно.
:::

Если приложение вам больше не нужно, вы можете удалить его.

Чтобы удалить приложение, выполните следующие действия:

<Steps>
1. Нажмите ссылку `Actions` для приложения, которое хотите удалить.
2. Нажмите кнопку `Remove` в разделе Actions.
3. Нажмите кнопку `OK`, чтобы подтвердить удаление.
</Steps>

Вы будете перенаправлены обратно к списку приложений.
