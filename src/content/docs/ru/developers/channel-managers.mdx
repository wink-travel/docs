Wink интегрируется с ведущими channel manager, системами управления недвижимостью (PMS) и центральными системами бронирования (CRS), чтобы поддерживать синхронизацию ваших тарифов и доступности во всех каналах продаж.

## Как это работает

Интеграции с channel manager обеспечивают двусторонний обмен данными между вашей системой управления недвижимостью и Wink:

1. **Отправка тарифов** — Ваш channel manager отправляет обновления тарифов в Wink. Когда вы обновляете цены в вашей PMS, эти изменения автоматически отражаются в Wink.
2. **Синхронизация доступности** — Ваш channel manager отправляет обновления доступности в Wink. Когда номера бронируются или освобождаются в вашей PMS, инвентарь Wink обновляется в реальном времени.
3. **Уведомления о бронированиях** — Когда бронирование сделано через Wink, ваш channel manager получает уведомление, чтобы резервирование появилось в вашей PMS.

## Поддерживаемые channel manager

В настоящее время Wink интегрируется со следующими системами:

| Channel Manager | Описание |
|---|---|
| **Channex** | Глобальная платформа дистрибуции и управления каналами. |
| **TravelClick** | Решения для отелей и управление каналами (Amadeus). |
| **Omnibees** | Дистрибуция и управление доходами для отелей. |
| **D-Edge** | Технологии для гостиничного бизнеса и решения для центрального бронирования. |
| **Allotz** | Распределение инвентаря и управление каналами. |
| **Ezee / Yanolja** | Облачная PMS и управление каналами, ориентированные на азиатский рынок. |
| **RateGain** | Технологии анализа тарифов и дистрибуции. |
| **RateTiger** | Распределение тарифов и доступности в реальном времени. |
| **CloudBeds** | Облачная PMS, channel manager и система бронирования. |
| **SiteMinder** | Ведущий channel manager и платформа дистрибуции для отелей. |
| **Exely** | Синхронизация тарифов и доступности для отелей. |

:::note
Если вашего channel manager нет в списке, свяжитесь с командой Wink. Новые интеграции могут быть добавлены на основе стандартов протокола OTA (Open Travel Alliance), которые поддерживает Wink.
:::

## Для разработчиков

Если вы создаёте интеграцию channel manager или PMS с Wink, используйте конечные точки Channel Manager API:

- **Обновления тарифов** — Отправляйте изменения тарифов в Wink.
- **Обновления доступности** — Отправляйте изменения доступности в Wink.
- **Получение бронирований** — Получайте бронирования из Wink в вашу систему.

Channel Manager API следует стандартам протокола OTA (SOAP/XML) для совместимости с существующими гостиничными системами.

## Уведомления через webhook

Вы можете подписаться на события webhook channel manager, чтобы получать уведомления в реальном времени:

- `channel-manager.update.rate` — Получено обновление тарифа.
- `channel-manager.update.availability` — Получено обновление доступности.
- `channel-manager.update` — Общее обновление channel manager.

Подробности смотрите в [Каталоге событий webhook](/ru/developers/webhook-events#channel-manager-events).
