Если вы еще не создали аккаунт Wink, [начните здесь](/ru/user/register).

## Payment API

Платежные эндпоинты предназначены для разработчиков, которые хотят приобретать туристический инвентарь. Это можно сделать через API как зарегистрированный туристический агент или используя наш API вместе с нашим PCI-совместимым платежным виджетом для всех остальных сущностей.
Payment API предоставляет эндпоинты для партнеров и отелей для отслеживания бронирований, аналитики и доступности средств. Они также могут выбрать вывод доступных средств на свой банковский счет.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Узнать больше
</LinkButton>
