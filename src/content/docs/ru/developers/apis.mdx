## API-интерфейс для платежей

Платежные конечные точки предназначены для разработчиков, которые хотят приобрести туристический инвентарь. Это можно сделать через API в качестве зарегистрированного турагента или с помощью нашего API в сочетании с нашим платежным виджетом, совместимым с PCI, для всех других субъектов.
Платежный API предоставляет конечные точки для аффилированных лиц и отелей для отслеживания бронирований, аналитики и доступности средств. Они также могут выбрать вывод доступных средств на свой банковский счет.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Узнать больше
</LinkButton>
