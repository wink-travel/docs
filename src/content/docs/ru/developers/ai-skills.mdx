Навыки ИИ — это переносимые пакеты инструкций, которые обучают агентов по программированию работать с Wink. Вместо того чтобы полагаться на обучающие данные или устаревшую документацию, каждый навык направляет агента использовать [Wink MCP server](/ru/developers/mcp) как актуальный источник правды для контрактов API и документации платформы.

Навыки работают с любым клиентом агента, который их поддерживает — Claude Code, Codex, Cursor и другими.

## Доступные навыки

### wink-api-builder

Реализуйте, проверяйте или отлаживайте интеграции Wink API. Навык проводит агента через структурированный рабочий процесс:

1. Поиск нужной операции API через `api_search`
2. Открытие актуальных контрактов запросов/ответов с сервера MCP
3. Получение настроек аутентификации из документации
4. Написание кода на основе проверенного контракта
5. Отметка любых несоответствий между существующим кодом и текущим API

Полезно для:
- Создания новых клиентских методов для любого Wink API
- Отладки несоответствий контрактов (например, запрос возвращает HTTP 400)
- Проверки существующих интеграций по актуальной спецификации

## Установка

Скопируйте папку навыка в директорию навыков вашего агента.

:::tip
Сервер Wink MCP должен быть подключен в той же среде. См. [MCP Server](/ru/developers/mcp) для инструкций по настройке.
:::

**Claude Code**

```bash
# локально в репозитории
cp -r skills/wink-api-builder .claude/skills/

# или на уровне пользователя
cp -r skills/wink-api-builder ~/.claude/skills/
```

Вызовите с помощью:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# локально в репозитории
cp -r skills/wink-api-builder .agents/skills/

# или на уровне пользователя
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Вызовите с помощью:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Другие агенты**

Разместите папку навыка там, где ваш агент ищет навыки. Точка входа — `SKILL.md`.

## Требования

- Клиент агента с поддержкой MCP и навыков
