На этой странице перечислены все события webhook, на которые вы можете подписаться в Wink. Для инструкций по настройке смотрите [Руководство по интеграции Webhook](/ru/guides/webhook-integration) или [страницу управления Webhook](/ru/my-account/webhooks).

## Как подписаться

1. Перейдите в **Applications > Webhooks** в вашем портале.
2. Создайте webhook и выберите события, которые хотите получать.
3. Wink отправляет HTTP POST на ваш endpoint каждый раз, когда происходит событие.

Для полного описания шагов настройки и лучших практик смотрите [Интеграция Webhook](/ru/guides/webhook-integration).

## События аккаунта

События, связанные с жизненным циклом и настройками вашего аккаунта TripPay.

| Событие | Ключ | Описание |
|---|---|---|
| Аккаунт создан | `account.create` | Создан новый аккаунт. |
| Аккаунт на проверке | `account.under_review` | Аккаунт находится на проверке. |
| Аккаунт обновлён | `account.updated` | Данные аккаунта изменены. |
| Статус аккаунта обновлён | `account.status.updated` | Статус аккаунта изменён (например, одобрен, приостановлен). |
| Адрес аккаунта обновлён | `account.address.updated` | Адрес аккаунта изменён. |
| Детали аккаунта обновлены | `account.details.updated` | Поля деталей аккаунта изменены. |
| Информация аккаунта обновлена | `account.information.update` | Общая информация аккаунта обновлена. |
| Менеджеры аккаунта обновлены | `account.managers.update` | Список менеджеров аккаунта изменён. |
| Фото профиля обновлено | `account.profile_picture.updated` | Фото профиля аккаунта изменено. |
| Агент обновлён | `account.agent.updated` | Информация об агенте аккаунта изменена. |
| URL имя обновлено | `account.url-name.update` | Персональный URL аккаунта изменён. |
| Уникальный ID обновлён | `account.unique-id.update` | Уникальный идентификатор аккаунта изменён. |

### Задачи аккаунта

| Событие | Ключ | Описание |
|---|---|---|
| Задача создана | `account.task.create` | В аккаунт добавлена новая задача. |
| Задача обновлена | `account.task.update` | Задача аккаунта изменена. |
| Задача удалена | `account.task.remove` | Задача аккаунта удалена. |
| Задача выполнена | `account.task.complete` | Задача аккаунта выполнена. |
| Задача проверки налогового ID выполнена | `account.task.tax_id.complete` | Задача проверки налогового ID выполнена. |

### Аквайер аккаунта

| Событие | Ключ | Описание |
|---|---|---|
| Аквайер создан | `account.acquirer.created` | В аккаунт добавлен новый аквайер. |
| Аквайер обновлён | `account.acquirer.updated` | Аквайер аккаунта изменён. |
| Аквайер удалён | `account.acquirer.removed` | Аквайер удалён из аккаунта. |
