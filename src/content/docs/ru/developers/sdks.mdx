Ускорьте вывод продукта на рынок, используя один из официально поддерживаемых языковых наборов для разработки программного обеспечения.
Если мы пока не поддерживаем ваш любимый язык, перейдите к нужному API и скачайте OpenAPI спецификацию, которую мы размещаем для каждого API вверху страницы. Вы можете сгенерировать собственный SDK, либо форкнув один из наших публичных репозиториев SDK, либо используя один из генераторов OpenAPI.

## Java

Эта библиотека скомпилирована для Java 24. Она настроена на использование реактивного WebClient и оптимизирована для работы с Spring Framework. Однако Spring не является обязательным, но предоставляет поддержку OAuth2 из коробки.
