Ускорьте время выхода на рынок, используя один из наших официально поддерживаемых языковых комплектов разработки программного обеспечения.
Если в настоящее время мы не поддерживаем ваш любимый язык, перейдите к API, который вы хотите использовать, и загрузите спецификацию OpenAPI, которую мы делаем доступной для каждого API в верхней части страницы. Вы можете создать свой собственный SDK, либо разветвив один из наших общедоступных репозиториев SDK, либо используя один из генераторов OpenAPI.

## Ява

Эта библиотека была скомпилирована для Java 24. Она настроена на использование реактивного WebClient и оптимизирована для использования с Spring Framework. Однако Spring не является обязательным требованием, но она предоставляет вам поддержку OAuth2 из коробки.
