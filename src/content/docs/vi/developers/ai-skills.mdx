Kỹ năng AI là các gói hướng dẫn di động giúp các tác nhân lập trình cách làm việc với Wink. Thay vì dựa vào dữ liệu đào tạo hoặc tài liệu lỗi thời, mỗi kỹ năng hướng dẫn tác nhân sử dụng [máy chủ MCP của Wink](/vi/developers/mcp) làm nguồn dữ liệu chính xác trực tiếp cho các hợp đồng API và tài liệu nền tảng.

Kỹ năng hoạt động với bất kỳ khách hàng tác nhân nào hỗ trợ chúng — Claude Code, Codex, Cursor và các tác nhân khác.

## Kỹ năng có sẵn

### wink-api-builder

Triển khai, xem xét hoặc gỡ lỗi các tích hợp API của Wink. Kỹ năng hướng dẫn tác nhân qua một quy trình làm việc có cấu trúc:

1. Tìm kiếm thao tác API phù hợp qua `api_search`
2. Mở các hợp đồng yêu cầu/phản hồi trực tiếp từ máy chủ MCP
3. Lấy thiết lập xác thực từ tài liệu
4. Viết mã dựa trên hợp đồng đã được xác minh
5. Đánh dấu bất kỳ sự không khớp nào giữa mã hiện có và API hiện tại

Hữu ích cho:
- Xây dựng các phương thức khách hàng mới cho bất kỳ API Wink nào
- Gỡ lỗi các sự không khớp hợp đồng (ví dụ: một yêu cầu trả về HTTP 400)
- Xem xét các tích hợp hiện có so với đặc tả trực tiếp

## Cài đặt

Sao chép thư mục kỹ năng vào thư mục kỹ năng của tác nhân của bạn.

:::tip
Máy chủ MCP của Wink phải được kết nối trong cùng môi trường. Xem [Máy chủ MCP](/vi/developers/mcp) để biết hướng dẫn cài đặt.
:::

**Claude Code**

```bash
# repo-local
cp -r skills/wink-api-builder .claude/skills/

# or user-level
cp -r skills/wink-api-builder ~/.claude/skills/
```

Gọi lệnh với:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repo-local
cp -r skills/wink-api-builder .agents/skills/

# or user-level
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Gọi lệnh với:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Các tác nhân khác**

Đặt thư mục kỹ năng ở bất kỳ đâu tác nhân của bạn giải quyết kỹ năng. Điểm vào là `SKILL.md`.
