Wink tích hợp với các channel manager hàng đầu, hệ thống quản lý bất động sản (PMS) và hệ thống đặt phòng trung tâm (CRS) để giữ cho giá và tình trạng phòng của bạn được đồng bộ trên tất cả các kênh bán hàng.

## Cách hoạt động

Tích hợp channel manager cho phép luồng dữ liệu hai chiều giữa hệ thống quản lý bất động sản của bạn và Wink:

1. **Đẩy giá** — Channel manager của bạn gửi cập nhật giá đến Wink. Khi bạn cập nhật giá trong PMS, những thay đổi đó sẽ tự động được phản ánh trên Wink.
2. **Đồng bộ tình trạng phòng** — Channel manager của bạn gửi cập nhật tình trạng phòng đến Wink. Khi phòng được đặt hoặc hủy trong PMS, tồn kho của Wink sẽ được cập nhật theo thời gian thực.
3. **Thông báo đặt phòng** — Khi có đặt phòng trên Wink, channel manager của bạn sẽ nhận được thông báo để đặt phòng xuất hiện trong PMS.

## Các channel manager được hỗ trợ

Hiện tại Wink tích hợp với các hệ thống sau:

| Channel Manager | Mô tả |
|---|---|
| **Channex** | Nền tảng phân phối toàn cầu và quản lý kênh. |
| **TravelClick** | Giải pháp khách sạn và quản lý kênh (Amadeus). |
| **Omnibees** | Phân phối và quản lý doanh thu cho khách sạn. |
| **D-Edge** | Công nghệ khách sạn và giải pháp đặt phòng trung tâm. |
| **Allotz** | Phân bổ tồn kho và quản lý kênh. |
| **Ezee / Yanolja** | PMS đám mây và quản lý kênh tập trung vào thị trường châu Á. |
| **RateGain** | Công nghệ phân tích giá và phân phối. |
| **RateTiger** | Phân phối giá và tình trạng phòng theo thời gian thực. |
| **CloudBeds** | PMS đám mây, channel manager và công cụ đặt phòng. |
| **SiteMinder** | Channel manager khách sạn hàng đầu và nền tảng phân phối. |
| **Exely** | Đồng bộ giá và tình trạng phòng cho khách sạn. |

:::note
Nếu channel manager của bạn không có trong danh sách, hãy liên hệ với đội ngũ Wink. Các tích hợp mới có thể được thêm dựa trên tiêu chuẩn giao thức OTA (Open Travel Alliance) mà Wink hỗ trợ.
:::

## Dành cho nhà phát triển

Nếu bạn đang xây dựng tích hợp channel manager hoặc PMS với Wink, hãy sử dụng các điểm cuối API Channel Manager:

- **Cập nhật giá** — Đẩy thay đổi giá đến Wink.
- **Cập nhật tình trạng phòng** — Đẩy thay đổi tình trạng phòng đến Wink.
- **Lấy đặt phòng** — Lấy đặt phòng từ Wink vào hệ thống của bạn.

API Channel Manager tuân theo tiêu chuẩn giao thức OTA (SOAP/XML) để tương thích với các hệ thống khách sạn hiện có.

## Thông báo webhook

Bạn có thể đăng ký sự kiện webhook của channel manager để nhận thông báo theo thời gian thực:

- `channel-manager.update.rate` — Nhận cập nhật giá.
- `channel-manager.update.availability` — Nhận cập nhật tình trạng phòng.
- `channel-manager.update` — Cập nhật chung của channel manager.

Xem [Danh mục sự kiện Webhook](/vi/developers/webhook-events#channel-manager-events) để biết chi tiết.
