## API thanh toán

Điểm cuối thanh toán dành cho các nhà phát triển muốn mua hàng tồn kho du lịch. Điều này có thể được thực hiện thông qua API với tư cách là Đại lý du lịch đã đăng ký hoặc sử dụng API của chúng tôi kết hợp với tiện ích thanh toán tuân thủ PCI của chúng tôi cho tất cả các thực thể khác.
API thanh toán cung cấp các điểm cuối cho các chi nhánh và khách sạn để theo dõi đặt phòng, phân tích và tình trạng sẵn có của quỹ. Họ cũng có thể chọn rút tiền khả dụng vào tài khoản ngân hàng của mình.
