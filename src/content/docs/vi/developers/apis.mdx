## Payment API

Các điểm cuối Payment dành cho các nhà phát triển muốn mua kho du lịch. Việc này có thể thực hiện qua API với tư cách là Đại lý Du lịch đã đăng ký hoặc sử dụng API của chúng tôi kết hợp với widget thanh toán tuân thủ PCI cho tất cả các thực thể khác.
Payment API cung cấp các điểm cuối cho các đối tác liên kết và khách sạn để theo dõi đặt chỗ, phân tích và tình trạng quỹ. Họ cũng có thể chọn rút tiền có sẵn về tài khoản ngân hàng của mình.
