**Module Wink Drupal** thêm các thành phần kho lưu trữ du lịch Wink vào trang Drupal của bạn dưới dạng các block gốc. Biên tập viên có thể đặt chúng ở bất kỳ đâu trong bố cục thông qua hệ thống block tiêu chuẩn.

<LinkCard
  title="wink-travel/wink-drupal-module trên GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Mã nguồn, hướng dẫn thiết lập cho nhà phát triển và hướng dẫn đóng góp."
/>

## Yêu cầu

- Drupal 10 hoặc 11
- PHP 8.1+
- Composer

## Cài đặt

<Steps>
1. Cài đặt qua Composer:
   ```bash
   composer require drupal/wink
   ```
2. Kích hoạt module:
   ```bash
   drush en wink -y
   ```
3. Vào **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Nhập **Client ID** và **Client Secret**, chọn môi trường của bạn, rồi nhấn **Save**.
   Vào [Applications](/vi/portal/settings/applications) để lấy thông tin đăng nhập.
</Steps>

## Thêm block vào trang

<Steps>
1. Vào **Admin → Structure → Block Layout**.
2. Nhấn **Place block** ở khu vực bạn muốn hiển thị nội dung Wink.
3. Tìm kiếm **Wink** để thấy các block có sẵn:
   - **Wink Content** — hiển thị thẻ khách sạn, lưới hoặc bản đồ
   - **Wink Lookup** — thanh tìm kiếm điểm đến
   - **Wink Search** — nút chọn hành trình
   - **Wink Account** — nút đăng nhập
   - **Wink Itinerary** — nút trạng thái hành trình
   - **Wink Cart** — nút giỏ hàng
4. Với **Wink Content**, cấu hình **Layout Type** và **Layout ID** trong cài đặt block.
5. Lưu cấu hình block.
</Steps>

## Các block có sẵn

| Block | Mục đích |
|---|---|
| **Wink Content** | Hiển thị thẻ khách sạn, lưới xếp hạng, bản đồ và nhiều hơn nữa |
| **Wink Lookup** | Thanh tìm kiếm điểm đến và khách sạn |
| **Wink Search** | Nút biểu tượng mở bộ chọn hành trình |
| **Wink Account** | Nút đăng nhập / tài khoản người dùng |
| **Wink Itinerary** | Nút hành trình hiển thị chuyến đi hiện tại |
| **Wink Cart** | Nút tóm tắt giỏ hàng |
