**Tiện ích mở rộng Wink Joomla** là một gói cài đặt plugin hệ thống (tải các script CDN), plugin nội dung (xử lý các thẻ shortcode trong bài viết), và một thành phần (trang cài đặt quản trị).

<LinkCard
  title="wink-travel/wink-joomla-extension trên GitHub"
  href="https://github.com/wink-travel/wink-joomla-extension"
  description="Mã nguồn, hướng dẫn thiết lập cho nhà phát triển và hướng dẫn đóng góp."
/>

## Yêu cầu

- Joomla 4 hoặc 5
- PHP 8.1+

## Cài đặt

<Steps>
1. Tải xuống phiên bản mới nhất `pkg_wink-x.x.x.zip` từ [trang phát hành GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. Trong quản trị Joomla, vào **Extensions → Manage → Install**.
3. Tải lên file zip và nhấn **Upload & Install**.
4. Cả hai plugin sẽ được kích hoạt tự động. Vào **Components → Wink Settings**.
5. Nhập **Client ID** và **Client Secret** của bạn rồi nhấn **Save**.
   Vào [Applications](/vi/portal/settings/applications) để lấy thông tin đăng nhập.
</Steps>

## Nhúng thành phần vào bài viết

Sử dụng các thẻ shortcode ở bất kỳ đâu trong nội dung bài viết:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Plugin nội dung sẽ thay thế các thẻ này bằng HTML `<wink-*>` chính xác khi hiển thị. Plugin hệ thống tự động tải các script CDN và chèn `<wink-app-loader>` vào chân trang.

## Các shortcode có sẵn

| Shortcode | Kết quả | Mô tả |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Thẻ khách sạn, lưới, bản đồ |
| `{wink-lookup}` | `<wink-lookup>` | Thanh tìm kiếm điểm đến |
| `{wink-search}` | `<wink-search-button>` | Nút chọn hành trình |
| `{wink-account}` | `<wink-account-button>` | Đăng nhập / tài khoản người dùng |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Nút trạng thái hành trình |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Nút giỏ hàng |
