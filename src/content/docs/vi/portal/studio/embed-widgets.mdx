Studio cung cấp một số widget có thể nhúng mà bạn có thể thêm vào trang web của mình để thúc đẩy đặt phòng.

Để truy cập các widget nhúng, điều hướng đến `Tools > Miscellaneous` từ thanh bên.

## Các widget có sẵn

| Widget | Mô tả |
| -- | -- |
| **Nút tài khoản** | Nút có thương hiệu liên kết đến trang liên kết của bạn |
| **Nút hành trình** | Nút mở bộ chọn hành trình |
| **Biểu mẫu hành trình** | Biểu mẫu chọn ngày và khách nội tuyến |
| **Nút tìm kiếm** | Nút mở tìm kiếm bất động sản |
| **Nút tra cứu** | Nút để tra cứu kho hàng cụ thể |

## Cách nhúng

Mỗi widget cung cấp một đoạn mã bạn có thể sao chép và dán vào HTML của trang web. Đoạn mã bao gồm **Client ID** và **Customization ID** của bạn để widget tự động liên kết với tài khoản liên kết của bạn.

:::note
Nếu bạn sử dụng WordPress, chúng tôi cũng cung cấp một plugin WordPress để tích hợp dễ dàng hơn.
:::
