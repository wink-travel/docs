Joskus haluamme ottaa sinuun yhteyttä. Kun näin tapahtuu, oikeassa yläkulmassa oleva kello muuttuu.
Lukemaan lähettämämme viestit:

<Steps>
1. Klikkaa päävalikosta `Tili > Ilmoitukset`.
2. Näet uudet ja aiemmat ilmoitukset.
3. Valitse `Merkitse luetuksi` tai `Poista` ilmoitukset, kun olet lukenut ne.
</Steps>

## API

Kehittäjät, jotka haluavat hallita `Affiliateja`, voivat siirtyä kohtaan [Kehittäjät > API > Affiliate](/fi/developers/apis/#affiliate-api).
