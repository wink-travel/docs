Wink-käyttäjä voi valita, että hallinnoit yhtä heidän kiinteistö-, yhteistyö- tai matkatoimistotileistään. Tällöin saat sähköpostin, jossa on kutsupyyntö.

import { Steps } from '@astrojs/starlight/components';

Näet nykyiset Wink-kutsusi seuraavasti:

<Steps>
 1. Kirjaudu sisään osoitteessa [my.wink.travel](https://my.wink.travel).
 2. Klikkaa vasemman sivupalkin kohdasta `Invites`.
 3. Näet listan kaikista nykyisistä kutsuista.
 4. Tässä osiossa näkyvät kaikki hakemuksesi.
 5. Valitse `Accept` tai `Reject` kutsu.
 6. Jos hyväksyt kutsun, tili näkyy kohdassa `Accounts`.
</Steps>
