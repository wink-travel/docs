Kun varaus tehdään Winkissä, seuraavaa tapahtuu:

<Steps>
1. Kanavahallintajärjestelmäsi, CRS tai PMS saa ilmoituksen.
2. Saat vahvistussähköpostin *(varauspisteen yhteystiedot)*, jossa on linkki uuden varauksesi hallintaan.
3. Matkustaja saa vahvistussähköpostin, jossa on myös ohjeet yhteydenottoon varauspisteeseesi.
4. Matkatoimisto saa saman varausvahvistuksen kuin matkustaja.
4. Kehittäjät voivat kuunnella `booking.created` webhook-tapahtumaa ja saada meiltä reaaliaikaista yksityiskohtaista varausinformaatiota.
</Steps>

Seuraa sähköpostin linkkiä tai siirry päävalikosta kohtaan `Tili > Varaukset` aloittaaksesi.

## Varausnäkymä

Varausnäkymä näyttää kaikki menneet ja tulevat varauksesi, lajiteltuna varauspäivän mukaan nousevassa järjestyksessä.

### Suodattimet

Voit käyttää suodattimia löytääksesi etsimäsi varaukset.

- Varauspäivä [aikaväli]
- Saapumispäivä [aikaväli]
- Lähtöpäivä [aikaväli]
- Varauskoodi
- Matkustajan sukunimi
- Matkustajan etunimi
- Master-hintaryhmän nimi
- Kohteen nimi

### Vienti

Voit viedä varaukset, jotka vastaavat nykyisiä suodatinvalintojasi, CSV-muodossa.

Viedäksesi varauksesi:

<Steps>
1. Varmista, että haluamasi suodattimet ovat aktiivisia.
2. Odota, että varausnäkymä päivittyy.
2. Klikkaa 🗂️ (*Tiedostovienti*) kuvaketta näkymän oikeassa alakulmassa.
</Steps>

## Varaus

Valitse varaus, jonka kanssa haluat työskennellä varausnäkymästä, ja sinut ohjataan `Varaus`-sivulle.
Tältä sivulta löydät kaikki tiedot, joita tarvitset vieraasi asianmukaiseen vastaanottamiseen saapumispäivänä.

Sivulla on myös muutama tärkeä toiminto, joita tarvitset vieraidesi palvelemiseen. Kaikki käytettävissä olevat toiminnot löytyvät `Toiminnot`-alasvetovalikosta sivun yläosasta.

### Lähetä sähköposti uudelleen

Lähettääksesi vahvistussähköpostin uudelleen matkustajalle, klikkaa `Toiminnot > Lähetä vahvistussähköposti uudelleen`.

### Peruuta varaus

Peruuttaaksesi varauksen kohteen puolesta, klikkaa `Toiminnot > Peruuta varaus`.

Näyttöön avautuu ponnahdusikkuna, jossa voit:

<Steps>
1. Valita peruutustyypin. *esim. Virheelliset päivämäärät*
2. Kirjoittaa peruutuksen syyn.
</Steps>

*Tämä painike noudattaa peruutusehtojasi, joten se ei aina ole käytettävissä.*

## API

Kehittäjät, jotka haluavat hallita `Varauksia` matkatoimistona, voivat siirtyä kohtaan [Developers > API > Travel Agent](/fi/api).
