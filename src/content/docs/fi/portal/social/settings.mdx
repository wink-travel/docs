# Asetukset

Kun aloitat Socialin käytön, kysymme, miten haluat meidän luovan sisältöäsi. Nämä vaihtoehdot ovat käytettävissäsi aina, joten voit helposti muuttaa niitä myöhemmin.

Siirry sivupalkista kohtaan `Preferences` hallitaksesi asetuksiasi.

## Preferences

<figure>
<CldImage
  src="academy/social_preferences"
  width="700"
  height="355"
  alt="Social preferences"
/>
  <figcaption>*Kuvio 1 - Preferences*</figcaption>
</figure>

Preferences-osio sisältää perusasetuksesi sisällön luomiseen:

1. **Preferred customization** Valitse, mitä räätälöintiä käytetään sisällön luomisessa. Räätälöinnit ohjaavat teeman värejä, oletusmatkaohjelmaa, valuuttaa, kieltä ja muuta. Luo räätälöintejä tilisi kautta osoitteessa [my.wink.travel](https://my.wink.travel).
2. **Auto-approve** Kun tämä on käytössä, luodut julkaisut julkaistaan automaattisesti ilman hyväksyntääsi. Ota tämä käyttöön, kun olet varma tuottamamme sisällön laadusta.
3. **Approver name** Henkilön tai tiimin nimi, joka saa hyväksyntäpyyntösähköpostit.
4. **Approver e-mail** Sähköpostiosoite, johon tulevat hyväksyntäpyynnöt tulevista julkaisuista lähetetään.

## Sisällöntuottajat

<figure>
<CldImage
  src="academy/social_content_creators"
  width="700"

  alt="Social content creators"
/>
  <figcaption>*Kuvio 2 - Sisällöntuottajat*</figcaption>
</figure>

Social sisältää monia valittavissa olevia sisällöntuottajia. Jokainen tuottaja keskittyy tiettyyn tapaan luoda ainutlaatuista sisältöä kiinteistöllesi tekoälyn avulla. Jokaisella sisällöntuottajalla on **Weight**-arvo — suurempi arvo lisää todennäköisyyttä, että kyseinen tuottaja valitaan julkaisua luotaessa.

Tässä ovat kaikki käytettävissä olevat sisällöntuottajat:

| Tuottaja | Kuvaus |
| -- | -- |
| **Destination** | Käyttää sijaintiasi luodakseen hyödyllistä tekstiä alueesta |
| **Room type** | Luo varattavan julkaisun, jossa esitellään yksi huonetyypeistäsi tekoälyn luomalla tekstillä ja huonetyyppisi kuvilla |
| **Activity** | Luo varattavan julkaisun, jossa esitellään yksi aktiviteeteistasi |
| **Add-On** | Luo varattavan julkaisun, jossa esitellään yksi lisäpalveluistasi |
| **Attraction** | Luo varattavan julkaisun, jossa esitellään yksi nähtävyyksistäsi |
| **Place** | Luo varattavan julkaisun, jossa esitellään yksi paikoistasi |
| **Meeting room** | Luo varattavan julkaisun, jossa esitellään yksi kokoustiloistasi |
| **Restaurant** | Luo varattavan julkaisun, jossa esitellään yksi ravintoloistasi |
| **Spa** | Luo varattavan julkaisun, jossa esitellään yksi kylpylöistäsi |
| **Property** | Luo varattavan julkaisun, jossa esitellään parhaiten hinnoiteltu huoneesi tekoälyn luomalla tekstillä ja kiinteistösi kuvilla |
| **User sentiment** | Kerää vierasarvosteluja verkosta ja luo tunteisiin vetoava julkaisu, joka korostaa, mitä vieraat rakastavat kiinteistössäsi |
| **Funny** | Luo kevyitä, viihdyttäviä julkaisuja, jotka pitävät syötteesi kiinnostavana |

:::tip[Pro vinkki]
Varmista, että sinulla on valitsemillesi sisällöntuottajille tarvittava varastotyyppi. Esimerkiksi jos valitset "Restaurant", kiinteistölläsi tulee olla vähintään yksi ravintola listattuna.
:::

## Äänensävy

<figure>
<CldImage
  src="academy/social_personas"

  height="396"
  alt="Social tone of voice"
/>
  <figcaption>*Kuvio 3 - Äänensävy*</figcaption>
</figure>

Äänensävy määrittää kielen tyylin, jota tekoäly käyttää sisällön luomisessa. Valitse äänensävy, joka vastaa brändisi persoonallisuutta. Kuten sisällöntuottajilla, jokaisella äänensävyllä on **Weight**, joka ohjaa sen käyttötiheyttä.

Tässä ovat käytettävissä olevat äänensävytyypit:

| Äänensävy | Tyyli |
| -- | -- |
| **Neutral** | Rauhallinen, auktoriteettinen, faktapohjainen. Ei liioittelua, ei vitsejä — vain luotettavaa, hyvin jäsenneltyä tietoa |
| **Friendly** | Lämmin, lähestyttävä, hymyilevä tekstin kautta, rentoja lyhennyksiä ja satunnaisia emojeita |
| **Luxury** | Hienostunut, tyylikäs. Sanat kuten "erinomainen" ja "räätälöity", jotka saavat lukijan tuntemaan itsensä erityiseksi |
| **Foodie** | Aistilliset, herkulliset kuvaukset. Ruokapornokieli, joka herättää nälän |
| **Adventurous** | Rohkea, peloton, seikkailijan energia. Elävät aistilliset kuvaukset, jotka tekevät kaikesta eeppistä |
| **Family** | Terveellinen, lapsiystävällinen innostus. Lämmin vanhempien energia turvallisista ja hauskoista kokemuksista |
| **Romantic** | Runollinen, haikea, sydäntä lämmittävä. Auringonlaskut, pehmeä kieli ja rakkaus |
| **Humorous** | Sarkastinen, nopeat vitsit, nokkela sanaleikki. Lukee kuin stand-up-esitys |
| **Eco** | Vilpitön ja oikeamielinen. Juhlistaa vihreitä vaihtoehtoja ja kestäviä valintoja |
| **Local insider** | Erittäin paikalliset viittaukset, varoittaa turistirysistä, käyttää paikallista slangia |
| **Inclusive** | Tervetullut, juhlistava ja vahvistava. Saa kaikki tuntemaan itsensä nähdyiksi ja arvostetuiksi |
| **Mystical** | Eteerinen, henkinen, kosmiset totuudet. Lempeitä kutsuja katsomaan sisäänpäin |
| **Futuristic** | Tyylikäs, tulevaisuuteen suuntautuva. Rakastaa termejä kuten "paradigman muutos" ja optimistiset tulevaisuuden visiot |
| **Budget** | Sinnikäs, kekseliäs, ylpeästi säästäväinen. Juhlistaa edullisia löytöjä ja ilmaisia kokemuksia |

## Julkaisutyyppit

Julkaisutyyppi määrittää luodun sisällön tavoitteen ja rakenteen. Jokaisella julkaisutyyppillä on **Weight**, joka ohjaa sen käyttötiheyttä.

| Julkaisutyyppi | Kuvaus |
| -- | -- |
| **Transactional** | Selkeä, ytimekäs, konversioon keskittyvä. Ohjaa varauksia ja suoria toimia faktojen, hintojen ja seuraavien vaiheiden avulla |
| **Awareness** | Brändin rakentamiseen suunnattu sisältö, joka esittelee kiinteistöäsi ja lisää näkyvyyttä potentiaalisten vieraiden keskuudessa |
| **Sentiment** | Tunnepohjainen sisältö, joka luo positiivisia assosiaatioita ja rakentaa emotionaalista yhteyttä yleisöösi |

## Aikavyöhykkeet

  <figcaption>*Kuvio 4 - Aikavyöhykkeet*</figcaption>

Saatat haluta kohdistaa markkinointiponnistelusi tiettyyn kieleen ja alueeseen. Kerro meille kielet, joille haluat luoda sisältöä, ja aikavyöhyke, josta olet kiinnostunut, niin me hoidamme loput. Julkaisut julkaistaan aikataulun mukaisesti valitsemassasi aikavyöhykkeessä.

*Useimmat kiinteistöt käyttävät yhtä kieltä ja oletuksena paikallista kieltään.*

## Google Business

<figure>
<CldImage
  src="academy/social_google_business"

Yhdistämällä Google Business -profiilisi voimme hakea rikkaampia kiinteistötietoja tehokkaamman sisällön luomiseksi. Kiinteistölläsi tulee olla vahvistettu Google Business -profiili, jotta voimme luoda sisältöä listauksestasi. Jotkut sisällöntuottajistamme **vaativat**, että sinulla on Google Business -tili.

:::tip[Pro vinkki]
Aloita pienestä

Älä ole liian kunnianhimoinen ensimmäisellä kerralla. Suosittelemme aloittamaan sisällön luomisen kiinteistöllesi yhdellä kielellä.
:::
