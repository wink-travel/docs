Studio tarjoaa useita upotettavia widgettejä, jotka voit lisätä omaan verkkosivustoosi lisätäksesi varauksia.

Pääset upotettaviin widgetteihin siirtymällä sivupalkista kohtaan `Työkalut > Sekalaiset`.

## Saatavilla olevat widgetit

| Widget | Kuvaus |
| -- | -- |
| **Tilikuvake** | Brändätty painike, joka linkittää kumppanisivullesi |
| **Matkaohjelman painike** | Painike, joka avaa matkaohjelman valitsimen |
| **Matkaohjelman lomake** | Rivillä oleva päivämäärä- ja vierasvalintalomake |
| **Hakupainike** | Painike, joka avaa majoitushakemiston |
| **Hakulinkki** | Painike tietyn varaston hakemiseen |

## Kuinka upottaa

Jokainen widget tarjoaa koodinpätkän, jonka voit kopioida ja liittää verkkosivustosi HTML-koodiin. Pätkä sisältää **Asiakas-ID**:si ja **Mukautus-ID**:si, joten widget linkittyy automaattisesti kumppanitiliisi.

:::note
Jos käytät WordPressiä, tarjoamme myös WordPress-laajennuksen helpompaa integrointia varten.
:::
