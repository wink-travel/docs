Wink-käyttäjä voi valita, että annat hallita yhtä heidän kiinteistö-, yhteistyö- tai matkanjärjestäjätiliään. Tällöin saat sähköpostin, jossa on kutsupyyntö.

import { Steps } from '@astrojs/starlight/components';

Näet nykyiset Wink-kutsusi seuraavasti:

<Steps>
1. Sinun on kirjauduttava sisään yhdelle verkkosivustostamme.
2. Klikkaa profiilikuvakettasi oikeassa yläkulmassa, jolloin valikko avautuu.
3. Klikkaa `Invites`-linkkiä.
4. Toimi kutsujesi kanssa joko `Hyväksymällä` tai `Hylkäämällä` ne.
</Steps>
