Wink-käyttäjä voi halutessaan antaa sinun hallita yhtä hänen kiinteistö-, kumppanuus- tai matkatoimistotiliään. Saat kutsupyynnön sähköpostitse.

import { Steps } from '@astrojs/starlight/components';

Näet nykyiset Wink-kutsusi seuraavasti:

<Steps>
  1. Sinun on oltava kirjautuneena sisään jollekin verkkosivustoistamme.
  2. Napsauta profiilikuvakettasi oikeassa yläkulmassa, niin näkyviin tulee valikko.
  3. Klikkaa `Invites` linkki.
  4. Ole vuorovaikutuksessa kutsujesi kanssa joko `Accepting` tai `Rejecting` niitä.
</Steps>
