Jos haluat vaihtaa salasanasi Wink-käyttäjätilillesi, noudata näitä ohjeita:

## Päivitä salasana

<Steps>
1. Sinun on oltava kirjautuneena yhdelle verkkosivustostamme.
2. Napsauta profiilikuvakettasi oikeassa yläkulmassa, jolloin valikko avautuu.
3. Napsauta `Vaihda salasana` -linkkiä.
4. Noudata ohjeita salasanan vaihtamiseksi.
5. Napsauta `Tallenna`.
</Steps>

Seuraavalla kirjautumiskerralla tarvitset uuden salasanasi.

## Unohditko salasanan

Joskus käyttäjän salasanan nollaaminen on tarpeen.

Tässä ovat ohjeet käyttäjän salasanan nollaamiseen:

<Steps>
1. Napsauta `Unohditko salasanan` -linkkiä kirjautumissivulla.
2. Kirjoita käyttäjätunnus / sähköpostiosoite tilille, jonka haluat nollata.
3. Napsauta `Unohditko salasanan` -painiketta. *Sinut ohjataan salasanan nollauslomakkeeseen.*
4. Tarkista sähköpostisi.
</Steps>

<Steps>
1. Liitä sähköpostissa saamasi koodi uuteen lomakkeeseen yhdessä uuden salasanasi kanssa.
2. Vahvista uusi salasanasi.
3. Napsauta `Nollaa salasana` -painiketta. *Sinut ohjataan takaisin kirjautumissivulle.*
</Steps>
