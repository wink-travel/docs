Jos haluat päivittää käyttäjäprofiiliasi Wink-käyttäjätililläsi, noudata näitä ohjeita:

<Steps>
1. Sinun on kirjauduttava sisään yhdelle verkkosivustostamme.
2. Napsauta profiilikuvakettasi oikeassa yläkulmassa, jolloin valikko avautuu.
3. Napsauta `Profiili`-linkkiä.
4. Seuraa ohjeita päivittääksesi `sähköpostiosoitteesi`, `etunimesi` ja `sukunimesi`.
5. Voit myös ladata uuden `profiilikuvan` itsestäsi.
6. Napsauta `Tallenna`.
</Steps>

:::note
Uusi profiilikuvasi näkyy seuraavan kerran, kun kirjaudut kokonaan ulos Winkistä.
:::
