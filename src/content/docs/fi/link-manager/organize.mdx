Hallinnoidaksesi WinkLinks-tagejasi, siirry päävalikosta kohtaan `WinkLinks` ja klikkaa `Tags`-välilehteä.

Sisällön kasvaessa haluat alkaa järjestää sitä. Olemme luoneet tätä varten `tagit`. Voit ryhmitellä sisältöä niin monella tagilla kuin haluat.

## Luo tagi

<Steps>
1. Luo uusi tagi klikkaamalla `Add a new tag` -painiketta.
2. Syötä uuden tagisi nimi. *esim. Top coffee shops in Berlin*
3. Klikkaa `Save`-linkkiä.
</Steps>

## Päivitä tagi

<Steps>
1. Aloita klikkaamalla `Edit`-linkkiä tagin vierestä, jota haluat muokata.
2. Päivitä tagisi nimi. *esim. Best coffee shops in Berlin*
3. Klikkaa `Save`-linkkiä.
</Steps>

## Poista tagi

:::note
Kun poistat tagin, poistat sen myös kaikesta siihen liitetyistä sisällöistä.
:::

<Steps>
1. Klikkaa `Remove`-linkkiä tagin vierestä, jonka haluat poistaa.
</Steps>

## API

Kehittäjät, jotka haluavat hallinnoida `WinkLinks`-tageja, voivat siirtyä kohtaan [Developers > API > WinkLinks](/fi/developers/apis/#winklinks-api).
