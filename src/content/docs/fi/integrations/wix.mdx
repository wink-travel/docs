**Wink Wix App** lisää Wink-verkkokomponentit Wix-sivustollesi natiivisina widgetteinä, joita voit vetää ja pudottaa Wix-editorissa. Sivuston omistajilta ei vaadita koodausta.

/>

## Asennus

<Steps>
1. Mene [Wix App Marketiin](https://www.wix.com/app-market) ja etsi **Wink Travel**.
2. Klikkaa **Lisää sivustolle** ja seuraa ohjeita sovelluksen valtuuttamiseksi.
3. Asennuksen jälkeen avaa **Hallintapaneeli → Wink-asetukset** ja syötä **Client ID** ja **Client Secret**.
   Hae tunnuksesi [Sovelluksista](/fi/portal/settings/applications).
4. Klikkaa **Tallenna** ja sitten **Testaa yhteys** varmistaaksesi, että Wink on yhdistetty.
</Steps>

## Lisää komponentti sivulle

<Steps>
1. Avaa **Wix Editor** ja siirry sivulle, jolle haluat lisätä Wink-sisältöä.
2. Klikkaa **Lisää (+) → Sovellukset → Wink Travel** nähdäksesi kaikki saatavilla olevat widgetit.
3. Vedä widget sivulle — esimerkiksi **Wink Content** hotelliruudukon näyttämiseen.
4. Widgetin asetuspaneelissa valitse **Asettelutyyppi** ja liitä **Layout ID** Wink Studiosta.
5. Klikkaa **Julkaise** tehdäksesi muutokset näkyviksi.
</Steps>

## Saatavilla olevat widgetit

| Widget | Tarkoitus |
|---|---|
| **Wink Content** | Näyttää hotellikortteja, järjestettyjä ruudukkoja, karttoja ja muuta |
| **Wink Lookup** | Hakupalkki kohteille ja hotelleille |
| **Wink Search** | Kuvakepainike, joka avaa matkaohjelman valitsimen |
| **Wink Account** | Kirjautumis- / käyttäjätilipainike |
| **Wink Itinerary** | Matkaohjelmapainike, joka näyttää nykyisen matkan |
| **Wink Cart** | Ostoskorin yhteenvetopainike |
