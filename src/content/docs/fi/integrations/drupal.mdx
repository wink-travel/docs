**Wink Drupal Module** lisää Wink-matkavaraston komponentit Drupal-sivustollesi natiiviblokkeina. Toimittajat voivat sijoittaa ne mihin tahansa asettelussa tavallisen blokki-järjestelmän kautta.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Lähdekoodi, kehittäjän asennusopas ja ohjeet osallistumiseen."
/>

## Vaatimukset

- Drupal 10 tai 11
- PHP 8.1+
- Composer

## Asennus

<Steps>
1. Asenna Composerilla:
   ```bash
   composer require drupal/wink
   ```
2. Ota moduuli käyttöön:
   ```bash
   drush en wink -y
   ```
3. Siirry kohtaan **Hallinta → Asetukset → Verkkopalvelut → Wink-asetukset** (`/admin/config/wink`).
4. Syötä **Client ID** ja **Client Secret**, valitse ympäristö ja klikkaa **Tallenna**.
   Hae tunnuksesi [Sovellukset]-sivulta (/portal/settings/applications).
</Steps>

## Lisää lohko sivulle

<Steps>
1. Siirry kohtaan **Hallinta → Rakenne → Lohkojen asettelu**.
2. Klikkaa **Aseta lohko** alueella, johon haluat Wink-sisältöä.
3. Etsi **Wink** löytääksesi saatavilla olevat lohkot:
   - **Wink Content** — näyttää hotellikortin, ruudukon tai kartan
   - **Wink Lookup** — matkakohteen hakupalkki
   - **Wink Search** — matkaohjelman valintapainike
   - **Wink Account** — kirjautumispainike
   - **Wink Itinerary** — matkaohjelman tilapainike
   - **Wink Cart** — ostoskoripainike
4. **Wink Content** -lohkolle määritä **Asettelutyyppi** ja **Asettelun tunnus** lohkon asetuksissa.
5. Tallenna lohkon asetukset.
</Steps>

## Saatavilla olevat lohkot

| Lohko | Tarkoitus |
|---|---|
| **Wink Content** | Näyttää hotellikortteja, järjestettyjä ruudukkoja, karttoja ja muuta |
| **Wink Lookup** | Hakupalkki matkakohteille ja hotelleille |
| **Wink Search** | Kuvakepainike, joka avaa matkaohjelman valitsimen |
| **Wink Account** | Kirjautumis- / käyttäjätilipainike |
| **Wink Itinerary** | Matkaohjelmapainike, joka näyttää nykyisen matkan |
| **Wink Cart** | Ostoskorin yhteenvetopainike |
