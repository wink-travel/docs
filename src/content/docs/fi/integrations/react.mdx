React 19+ tukee räätälöityjä elementtejä ensiluokkaisesti. React 18 ja sitä vanhemmat versiot välittävät vain merkkijonoattribuutteja — käytä alla olevaa tyypitettyä apumallia pysyäksesi yhteensopivana eri versioiden kanssa.

## Asennus

```bash
npm install @wink/elements
```

## Lataa kerran sovelluksen juuresta

Kutsu `load()` pääkomponentissasi. Se on idempotentti — turvallista kutsua useita kertoja.

```tsx
// src/App.tsx
import { useEffect } from 'react';
import { load } from '@wink/elements';

export default function App() {
  useEffect(() => {
    load({ clientId: import.meta.env.VITE_WINK_CLIENT_ID });
  }, []);

  return <YourRoutes />;
}
```

## Käytä komponentteja JSX:ssä

```tsx
export default function HotelPage() {
  return (
    <main>
      <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
      <wink-lookup />
    </main>
  );
}
```

## TypeScript — määrittele JSX-intrinsicit

React ei tunne räätälöityjen elementtien nimiä oletuksena. Lisää määrittelytiedosto kerran, jotta tyyppivirheet poistuvat:

```ts
// src/wink-elements.d.ts
import type {
  WinkContentLoaderAttributes,
  WinkLookupAttributes,
  WinkSearchButtonAttributes,
  WinkAccountButtonAttributes,
  WinkItineraryButtonAttributes,
  WinkShoppingCartButtonAttributes,
  WinkAppLoaderAttributes,
} from '@wink/elements';

declare module 'react' {
  namespace JSX {
    interface IntrinsicElements {
      'wink-content-loader': WinkContentLoaderAttributes & React.HTMLAttributes<HTMLElement>;
      'wink-lookup': WinkLookupAttributes & React.HTMLAttributes<HTMLElement>;
      'wink-search-button': WinkSearchButtonAttributes & React.HTMLAttributes<HTMLElement>;
      'wink-account-button': WinkAccountButtonAttributes & React.HTMLAttributes<HTMLElement>;
      'wink-itinerary-button': WinkItineraryButtonAttributes & React.HTMLAttributes<HTMLElement>;
      'wink-shopping-cart-button': WinkShoppingCartButtonAttributes & React.HTMLAttributes<HTMLElement>;
      'wink-app-loader': WinkAppLoaderAttributes & React.HTMLAttributes<HTMLElement>;
    }
  }
}
```

<Aside type="caution" title="React 18:n attribuuttirajoitus">
React 18 ja sitä vanhemmat versiot sarjoittavat kaikki JSX-prop-arvot merkkijonoiksi ennen niiden välittämistä DOM:iin. Boolean- ja objekti-propit täytyy muuntaa manuaalisesti. React 19 välittää ei-merkkijono-propit suoraan räätälöidyille elementeille — kiertotietä ei tarvita.
</Aside>

## Tyypitettyjen attribuuttien välittäminen (React 18 yhteensopiva)

```tsx
import type { WinkContentLoaderAttributes } from '@wink/elements';

const attrs: WinkContentLoaderAttributes = {
  layout: 'HOTEL',
  id: 'abc123',
  sort: 'POPULARITY',
};

// Spread toimii React 19:ssä; React 18:ssa kaikki arvot ovat joka tapauksessa merkkijonoja
export default function HotelCard() {
  return <wink-content-loader layout={attrs.layout} id={attrs.id} sort={attrs.sort} />;
}
```
