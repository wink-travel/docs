**Wink TYPO3 Extension** integroi Wink-matkavaraston TYPO3 13:een natiivina sisältöelementteinä. Toimittajat lisäävät Wink-komponentteja sivuille käyttämällä tavallista TYPO3-sivueditoria — ei vaadi mallipohjan muutoksia.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"
  href="https://github.com/wink-travel/wink-typo3-extension"
  description="Lähdekoodi, kehittäjän asennusopas ja ohjeet osallistumiseen."
/>

## Vaatimukset

- TYPO3 13.x
- PHP 8.2+
- Composer

## Asennus

<Steps>
1. Asenna Composerilla:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivoi laajennus:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Mene TYPO3-hallintapaneelissa kohtaan **Admin Tools → Settings → Extension Configuration → wink**.
4. Syötä **Client ID**, **Client Secret** ja valitse ympäristö. Tallenna.
   Hae tunnukset [Applications](/fi/portal/settings/applications) -sivulta.
5. Lisää staattinen TypoScript-malli juurimalliin:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Lisää sisältöelementti sivulle

<Steps>
1. Avaa sivu **Page**-moduulissa.
2. Klikkaa sisältöalueen **+**-painiketta lisätäksesi uuden elementin.
3. Valitse yksi **Wink Travel** -sisältöelementeistä:
   - **Wink Content** — hotellikortit, järjestetyt ruudukot, kartat
   - **Wink Lookup** — matkakohteen hakupalkki
   - **Wink Search** — matkaohjelman valintapainike
   - **Wink Account** — kirjautumispainike
   - **Wink Itinerary** — matkaohjelman tilapainike
   - **Wink Cart** — ostoskoripainike
4. **Wink Content** -elementille valitse FlexForm-kentistä **Layout Type** ja syötä **Layout ID**.
5. Tallenna ja julkaise sivu.
</Steps>

## Saatavilla olevat sisältöelementit

| Elementti | Tarkoitus |
|---|---|
| **Wink Content** | Hotellikortit, järjestetyt ruudukot, kartat ja muuta |
| **Wink Lookup** | Matkakohteen hakupalkki |
| **Wink Search** | Kuvakepainike, joka avaa matkaohjelman valitsimen |
| **Wink Account** | Kirjautumis- / käyttäjätilapainike |
| **Wink Itinerary** | Matkaohjelmapainike, joka näyttää nykyisen matkan |
| **Wink Cart** | Ostoskorin yhteenvetopainike |
