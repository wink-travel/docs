Tallennetut haut tekevät juuri sitä, mitä nimi lupaa. Matkavarastoa etsiessäsi sinulla on mahdollisuus tallentaa hakukysely myöhempää käyttöä varten.

## Hakukyselyn tallentaminen

Kun aloitat varaston etsimisen (katso [Search](/fi/studio/search) ), voit hienosäätää hakusuodattimiasi.
Hakulomakkeen alareunassa on vaihtoehto `Save search`.

Tämän linkin klikkaaminen antaa sinulle mahdollisuuden nimetä tämä haku (*esim. Taikahotellien kirjastoni*), jotta muistat sen myöhemmin.

<CloudinaryImage

Anna nimi ja klikkaa `Save search` -painiketta. Tämän jälkeen voit siirtyä päävalikosta kohtaan `Inventory > Saves Searches` nähdäksesi juuri luomasi tallenteen.

## Listatoiminnot

Tallennettua hakua voi käyttää matkavaraston näyttämiseen käyttäjillesi ruudukon tai kartan muodossa ja voit määrittää hakutulosten näkymisen haluamallasi tavalla.

Jokaisella tallennetulla hakukyselyllä on toimintoja, joita voit suorittaa:

- Muunna hakukysely [kartaksi](/fi/studio/maps).
- Muunna hakukysely [ruudukoksi](/fi/studio/grids).
- Poista tallennettu haku.

## API

Kehittäjät, jotka haluavat hallita tallennettuja hakuja, voivat siirtyä kohtaan [Developers > API > Browse](/fi/developers/apis/#browse-api).

## Lisälukemista

- Lue lisää [Map](/fi/studio/maps) -ominaisuuksistamme.
- Lue lisää [Grid](/fi/studio/grids) -ominaisuuksistamme.
