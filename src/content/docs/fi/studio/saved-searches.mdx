Tallennetut haut tekevät mitä nimensäkin kertoo. Kun etsit matkailutuotteita, voit tallentaa hakukyselysi myöhempää käyttöä varten.

## Hakukyselyn tallentaminen

Kun aloitat varaston etsimisen (katso[Haku](/studio/search)), voit hienosäätää hakusuodattimiasi.
Kyselylomakkeen alareunassa on vaihtoehto`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Tallenna haku" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Näyttää linkin hakukyselyn tallentamiseen*</figcaption>
</figure>


Klikkaamalla tätä linkkiä saat mahdollisuuden nimetä haun (*esim. Taikahotellien kirjastoni*) jotain, minkä muistat myöhemmin.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Anna haullesi nimi" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Anna haullesi nimi*</figcaption>
</figure>


Anna sille nimi ja napsauta`Save search`-painiketta. Sen jälkeen voit siirtyä kohtaan`Inventory > Saves Searches`päänavigointipalkista nähdäksesi juuri luomasi tietueen.

## Listatoiminnot

Tallennettua hakuasi voidaan käyttää matkailutuotteiden näyttämiseen käyttäjille ruudukon tai kartan muodossa ja hakutulosten näyttämiseen haluamallasi tavalla.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Tallennetut hakutoiminnot" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Tallennettu hakukysely ilman suodattimia*</figcaption>
</figure>


Jokaisella tallennetulla hakukyselyllä on toimintoja, joita voit suorittaa sille:

* Muunna hakukysely muotoon[kartta](/studio/maps).
* Muunna hakukysely muotoon[ruudukko](/studio/grids).
* Poista tallennettu haku.

## API

Kehittäjät, jotka haluavat hallita tallennettuja hakuja, voivat siirtyä osoitteeseen[Kehittäjät > API > Selaa](/developers/apis/#browse-api).

## Lisälukemista

* Lue lisää[Kartta](/studio/maps)ominaisuuksia.
* Lue lisää[Ruudukko](/studio/grids)ominaisuuksia.
