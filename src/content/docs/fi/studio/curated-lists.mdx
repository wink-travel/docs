Kuratoidut listat on ominaisuus, jonka avulla voit ryhmitellä ja järjestää varastoa, jonka olet valinnut matkavarastohauissa löytämistäsi kohteista.
Kun lista on luotu, voit tehdä sen käyttäjillesi näkyväksi myytäväksi [ruudukon](/fi/studio/grids) tai [kartan](/fi/studio/maps) kautta [WinkLinksin](/fi/link-manager/wink-links), WordPress-laajennuksemme (/developers/wordpress/) tai upottamalla jonkin [Web-komponenteistamme](/fi/developers/web-components) sivustollesi.

Oletuksena jokainen affiliate-tili alkaa `Suosikit`-listalla, jota voit käyttää heti. `Suosikit` on erityinen eikä sitä voi poistaa.
Kaikki kuratoidut listat mahdollistavat seuraavat toiminnot:

## Luo lista

- Hallintapaneelista valitse `Varasto > Kuratoidut listat` navigaatiopalkista.
- Luo uusi kuratoitu lista klikkaamalla `Luo uusi lista` -painiketta.
- Sinua pyydetään nimeämään listasi. *Esimerkki: Top 3 NYC valinnat*.
- Klikkaa `Tallenna`-painiketta.

## Lisää listan kohde

Voit lisätä kohteen mihin tahansa listaan hakutuloksista.

- Mene kohtaan [Haku](/fi/studio/search)
- Etsi jotain, jonka haluat lisätä listallesi.
- Kun löydät sen, klikkaa `Toiminnot`-linkkiä hakutuloksen kortin vasemmassa alakulmassa.
- Klikkaa `Lisää kohteeseen...` ja valitse kuratoitu lista, johon haluat lisätä tämän varaston.
- Palaa kuratoidut listat -sivulle varmistaaksesi, että kohde on lisätty listallesi.

Jokaisella listalla on joitakin toimintoja, joita voit suorittaa.

- Muunna lista [kartaksi](/fi/studio/maps).
- Muunna lista [ruudukoksi](/fi/studio/grids).
- Poista lista (*paitsi Suosikit*).

## API

Kehittäjät, jotka haluavat hallita kuratoituja listoja, voivat siirtyä kohtaan [Kehittäjät > API > Selaa](/fi/developers/apis/#browse-api).

## Lisälukemista
