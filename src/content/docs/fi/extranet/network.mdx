Tässä artikkelissa näytetään, miten voit yhdistää innokkaisiin kumppaneihin Winkissä ja hallita olemassa olevia.

## Myyntikanavat

Hallinnoidaksesi myyntikanaviasi, klikkaa päävalikosta `Distribution > Sales channels`.

Myyntikanavat ovat olemassa olevia kumppanisuhteitasi, jotka on määritelty varaston, tarjousten ja alennusten perusteella, joihin heillä on pääsy.

Päivittääksesi olemassa olevaa myyntikanavaa, klikkaa `Actions`-linkkiä sen myyntikanavan alla, jota haluat muokata.

- **Jäsenalennus** Jäsenhinta annetaan matkustajille, jotka varaavat tämän myyntikanavan kautta.
- **Komissio** Komissio kohdistetaan myyntikanavalle jokaisesta onnistuneesta varauksesta.
- **Yksittäiset kampanjat** Lisää kampanjat, jotka haluat tämän myyntikanavan saavan.
- **Paketoidut kampanjat** Lisää paketoidut kampanjat, jotka haluat tämän myyntikanavan saavan.
- **Ota käyttöön** Vaihda päälle/pois -kytkin ottaaksesi myyntikanavan kokonaan käyttöön tai pois käytöstä.
- **Mustalla listalla** Merkitsemällä myyntikanavan mustalle listalle, varaukset tämän kanavan kautta estetään kokonaan.

Kun olet tyytyväinen asetuksiisi, klikkaa `Save`-painiketta jatkaaksesi.

:::tip
Merkitsemällä myyntikanavan mustalle listalle, estät myyntikanavaa saamasta lainkaan hintoja sinulta; mukaan lukien Wink Network -hinnat.
:::

## Tutki verkostoa

Löytääksesi uusia kumppaneita Winkissä, klikkaa päävalikosta `Distribution > Explore network`.

Tässä osiossa näet odottavat pyynnöt kumppaneilta sivun yläosassa ja alla listan kumppaneista, joihin voit muodostaa yhteyden.

### Vastaa pyyntöön

Voit valita hylätä kumppanipyynnön. Jos haluat jatkaa pyynnön käsittelyä, sinut ohjataan `affiliate details` -sivulle, jossa voit saada lisätietoja kumppanista sekä Winkissä saavutetuista aiemmista tuloksista.

### Etsi kumppaneita

Kumppaniverkostossa voit hakea kumppaneita nimen, tyypin ja sijainnin mukaan. Kun löydät kiinnostavan, klikkaa sitä, jolloin siirryt `affiliate details` -sivulle.

### Yhdistä kumppaniin

Jos päätät solmia suoran suhteen kumppanin kanssa, sinua pyydetään määrittelemään suhde samalla tavalla kuin yllä myyntikanavaa hallinnoidessasi.

Kun tallennat uuden myyntikanavan, sinut ohjataan takaisin myyntikanavalistallesi.

## API

Kehittäjät, jotka haluavat hallita `Network`-verkostoaan, voivat siirtyä osoitteeseen [Developers > APIs > Distribution](/fi/developers/apis/#distribution-api).
