Wink integroituu johtaviin kanavahallintoihin, kiinteistöhallintajärjestelmiin (PMS) ja keskitettyihin varausjärjestelmiin (CRS) pitääkseen hintasi ja saatavuutesi synkronoituna kaikissa myyntikanavissasi.

## Miten se toimii

Kanavahallinnan integraatiot mahdollistavat kaksisuuntaisen tiedonsiirron kiinteistöhallintajärjestelmäsi ja Winkin välillä:

1. **Hintojen päivitys** — Kanavahallintasi lähettää hintapäivitykset Winkille. Kun päivität hinnoittelusi PMS:ssä, muutokset näkyvät automaattisesti Winkissä.
2. **Saatavuuden synkronointi** — Kanavahallintasi lähettää saatavuuspäivitykset Winkille. Kun huoneita varataan tai vapautetaan PMS:ssä, Winkin varastotiedot päivittyvät reaaliajassa.
3. **Varausilmoitukset** — Kun varaus tehdään Winkissä, kanavahallintasi saa ilmoituksen, jotta varaus näkyy PMS:ssäsi.

## Tuetut kanavahallinnat

Wink integroituu tällä hetkellä seuraaviin järjestelmiin:

| Kanavahallinta | Kuvaus |
|---|---|
| **Channex** | Globaali jakelu- ja kanavahallinta-alusta. |
| **TravelClick** | Hotelliratkaisut ja kanavahallinta (Amadeus). |
| **Omnibees** | Jakelu- ja tulosjohtamisen ratkaisut hotelleille. |
| **D-Edge** | Majoitusteknologia ja keskitetyt varausratkaisut. |
| **Allotz** | Varaston allokointi ja kanavahallinta. |
| **Ezee / Yanolja** | Pilvipohjainen PMS ja kanavahallinta, keskittyen Aasian markkinoihin. |
| **RateGain** | Hintatiedustelu ja jakeluteknologia. |
| **RateTiger** | Reaaliaikainen hintojen ja saatavuuden jakelu. |
| **CloudBeds** | Pilvipohjainen PMS, kanavahallinta ja varausmoottori. |
| **SiteMinder** | Johtava hotellien kanavahallinta- ja jakelualusta. |
| **Exely** | Hintojen ja saatavuuden synkronointi hotelleille. |

:::note
Jos kanavahallintaasi ei ole listattu, ota yhteyttä Wink-tiimiin. Uusia integraatioita voidaan lisätä OTA (Open Travel Alliance) -protokollastandardien mukaisesti, joita Wink tukee.
:::

## Kehittäjille

Jos rakennat kanavahallinnan tai PMS-integraation Winkin kanssa, käytä Channel Manager API -päätepisteitä:

- **Hintapäivitykset** — Lähetä hintamuutokset Winkille.
- **Saatavuuspäivitykset** — Lähetä saatavuusmuutokset Winkille.
- **Varausten haku** — Hae varauksia Winkistä järjestelmääsi.

Channel Manager API noudattaa OTA-protokollastandardeja (SOAP/XML) yhteensopivuuden varmistamiseksi olemassa olevien majoitusjärjestelmien kanssa.

## Webhook-ilmoitukset

Voit tilata kanavahallinnan webhook-tapahtumia saadaksesi reaaliaikaisia ilmoituksia:

- `channel-manager.update.rate` — Hintapäivitys vastaanotettu.
- `channel-manager.update.availability` — Saatavuuspäivitys vastaanotettu.
- `channel-manager.update` — Yleinen kanavahallinnan päivitys.

Katso [Webhook Events Catalog](/fi/developers/webhook-events#channel-manager-events) lisätietoja varten.
