## Payment API

Maksupäätepisteet on tarkoitettu kehittäjille, jotka haluavat ostaa matkailuvarastoa. Tämä voidaan tehdä API:n kautta rekisteröitynä matkatoimistona tai käyttämällä API:tämme yhdessä PCI-yhteensopivan maksuelementtimme kanssa kaikille muille tahoille.
Payment API tarjoaa päätepisteitä kumppaneille ja hotelleille seurata varauksia, analytiikkaa ja varojen saatavuutta. He voivat myös valita nostaa saatavilla olevat varat pankkitililleen.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Lisätietoja
</LinkButton>
