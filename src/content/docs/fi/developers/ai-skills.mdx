AI-taidot ovat siirrettäviä ohjepaketteja, jotka opettavat koodausagentteja työskentelemään Wink:n kanssa. Sen sijaan, että luotettaisiin koulutusdataan tai vanhentuneeseen dokumentaatioon, jokainen taito ohjaa agenttia käyttämään [Wink MCP -palvelinta](/fi/developers/mcp) API-sopimusten ja alustan dokumentaation reaaliaikaisena totuuden lähteenä.

Taidot toimivat minkä tahansa agenttiasiakkaan kanssa, joka tukee niitä — Claude Code, Codex, Cursor ja muut.

## Saatavilla olevat taidot

### wink-api-builder

Toteuta, tarkista tai debuggaa Wink API -integraatioita. Taito ohjaa agenttia rakenteellisen työnkulun läpi:

1. Etsi oikea API-toiminto `api_search`-komennolla
2. Avaa MCP-palvelimen reaaliaikaiset pyyntö-/vastaussopimukset
3. Hae todennuksen asetukset dokumentaatiosta
4. Kirjoita koodi vahvistetun sopimuksen mukaisesti
5. Merkitse mahdolliset ristiriidat olemassa olevan koodin ja nykyisen API:n välillä

Hyödyllinen:
- Uusien asiakasmetodien rakentamiseen mille tahansa Wink API:lle
- Sopimusristiriitojen debuggaamiseen (esim. pyyntö palauttaa HTTP 400)
- Olemassa olevien integraatioiden tarkistamiseen reaaliaikaista spesifikaatiota vasten

## Asennus

Kopioi taitokansio agenttisi taitokansioon.

:::tip
Wink MCP -palvelimen on oltava yhdistetty samassa ympäristössä. Katso [MCP Server](/fi/developers/mcp) asennusohjeet.
:::

**Claude Code**

```bash
# repossa paikallisesti
cp -r skills/wink-api-builder .claude/skills/

# tai käyttäjätasolla
cp -r skills/wink-api-builder ~/.claude/skills/
```

Käynnistä komennolla:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repossa paikallisesti
cp -r skills/wink-api-builder .agents/skills/

# tai käyttäjätasolla
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Käynnistä komennolla:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Muut agentit**

Sijoita taitokansio mihin tahansa, mistä agenttisi hakee taitoja. Sisäänkäyntipiste on `SKILL.md`.
