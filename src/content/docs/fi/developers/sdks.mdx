Nopeuta markkinoille pääsyä käyttämällä yhtä virallisesti tuetuista ohjelmointikielten Software Development Kit -paketeistamme.
Jos suosikkikieltäsi ei tällä hetkellä tueta, siirry haluamasi API:n sivulle ja lataa OpenAPI-spesifikaatio, jonka tarjoamme jokaisen API:n sivun yläosassa. Voit luoda oman SDK:si joko haarukoimalla jonkin julkisista SDK-repositorioistamme tai käyttämällä jotakin OpenAPI:n generaattoreista.

## Java

Tämä kirjasto on käännetty Java 24:lle. Se on konfiguroitu käyttämään reaktiivista WebClientiä ja optimoitu käytettäväksi Spring Frameworkin kanssa. Spring ei kuitenkaan ole pakollinen, mutta se tarjoaa OAuth2-tuen valmiiksi.
