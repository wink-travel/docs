Nopeuta markkinoilletuloaikaa käyttämällä jotakin virallisesti tuetuista kielistämme, kuten Software Development Kit.

Jos emme tällä hetkellä tue suosikkikieltäsi, siirry haluamaasi API:in ja lataa kullekin API:lle sivun yläreunasta saatavilla oleva OpenAPI-spesifikaatio. Voit luoda oman SDK:n joko haarautumalla johonkin julkisista SDK-arkistoistamme tai käyttämällä jotakin OpenAPI:n generaattoreista.

## Java

Tämä kirjasto on käännetty Java 24:lle. Se on konfiguroitu käyttämään reaktiivista WebClientiä ja optimoitu käytettäväksi Spring Frameworkin kanssa. Spring ei kuitenkaan ole vaatimus, mutta se tarjoaa OAuth2-tuen heti käyttövalmiina.
