## Yleiskatsaus
Isännöity Wink MCP -palvelin on saatavilla osoitteessa:

```
https://docs.mcp.wink.travel/mcp
```

Ohjaa yhteensopivat tekoälyasiakkaat (VS Code, Cursor, muut MCP HTTP -kuljetuksen toteuttavat) tähän päätepisteeseen, jotta ne voivat:
- Selailla tuote- ja alustan dokumentaatiota (`wink://docs/...`)
- Löytää SDK:t ja tutkia API-luokkia / malleja (`wink://sdk/...` & `wink://sdk/.../docs`)
- Suorittaa kevyitä semanttisia / avainsanahakuja dokumentaatiossa ja SDK-yhteenvedoissa

## Kuljetus
Käytä ensisijaista striimattavaa HTTP-päätepistettä:

```
https://docs.mcp.wink.travel/mcp
```

Jos asiakkaasi tukee vain SSE:tä, käytä:

```
https://docs.mcp.wink.travel/sse
```

### Cursor
Cursor MCP -dokumentaatio: https://cursor.com/docs/context/mcp
Asetus: Lisää etäpalvelin MCP-asetusten kautta.
```json
{
  "mcpServers": {
    "wink-docs": {
      "url": "https://docs.mcp.wink.travel/mcp",
    }
  }
}
```

### VS Code
Viralliset ohjeet: https://code.visualstudio.com/docs/copilot/customization/mcp-servers
Minimaalinen etäkonfiguraatio (käyttäjä- tai työtilan `mcp.json`):
```json
{
  "servers": {
    "wink-docs": {
      "type": "http",
      "url": "https://docs.mcp.wink.travel/mcp"
    }
  }
}
```

Vinkkejä:
- Käytä komentopalettia: “MCP: List Servers” vahvistaaksesi yhteyden ja selataksesi työkaluja.
- Työkalujen valitsin antaa mahdollisuuden ottaa käyttöön/pois päältä `sdk_search` ja `docs_search`.
- Lisää konteksti: Keskustelunäkymä → Lisää konteksti → MCP-resurssit liittääksesi dokumenttisivut.

## Ydint resurssi-URI:t

| URI | Kuvaus |
|-----|-------------|
| `wink://docs` | Dokumentaation indeksi (ryhmitelty kategorian mukaan) |
| `wink://docs/{path}` | Tietty dokumentaatioartikkeli (Markdown / MDX) |
| `wink://sdk` | SDK:n yleiskatsaus (jokainen SDK + dokumenttilaskuri) |
| `wink://sdk/{sdkPath}` | SDK:n README / moduulien dokumentaatiokansio / tietty moduulin dokumentaatio |

### Moduulidokumentaatio
SDK-dokumentaatiokansion avaaminen (esim. `wink://sdk/python/wink-sdk-booking/docs`) palauttaa ryhmiteltynä:
- **API-luokat** (metodien määrä + esimerkkimetodien nimet)
- **Mallit ja tyypit** (näiden API:en käyttämät tietorakenteet)

## Hakutyökalut (Pikaviite)
| Työkalu | Tarkoitus | Tyypillisiä hakuesimerkkejä |
|------|---------|-----------------------|
| `sdk_search` | Löytää käytettävän SDK:n / API-luokan | `peruuta varaus`, `listaa kohteet`, `kumppanivarasto` |
| `docs_search` | Löytää käsitteellisiä dokumentteja / oppaita | `todennus`, `webhookit`, `wordPress` |

Jätä `query` tyhjäksi saadaksesi ehdotuksia. Jokainen tulos sisältää MCP-URI:n, jonka voit avata.

## Tyypillinen käyttötapaus
1. Suorita `sdk_search` aikomuksellasi (esim. *"listaa kohteet"*).
2. Avaa palautettu SDK README tai sen `docs_uri` tarkastellaksesi API:ita / malleja.
3. Käytä `docs_search` tukevien käsitteiden (todennus, webhookit) löytämiseen.

## Huomautuksia
- Jokainen palautettu URI voidaan avata agentin toimesta koko sisällön saamiseksi.
- Ehdotukset ilmestyvät automaattisesti, kun haut ovat tyhjiä.
