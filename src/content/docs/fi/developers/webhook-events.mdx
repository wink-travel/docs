Tällä sivulla on lueteltu kaikki webhook-tapahtumat, joihin voit tilata Winkissä. Asennusohjeita varten katso [Webhook Integration Guide](/fi/guides/webhook-integration) tai [Webhooks management page](/fi/my-account/webhooks).

## Kuinka tilata

1. Siirry portaalissasi kohtaan **Applications > Webhooks**.
2. Luo webhook ja valitse vastaanotettavat tapahtumat.
3. Wink lähettää HTTP POST -pyynnön päätepisteeseesi aina, kun tapahtuma tapahtuu.

Täydelliset asennusohjeet ja parhaat käytännöt löydät osoitteesta [Webhook Integration](/fi/guides/webhook-integration).

## Tilitapahtumat

Tapahtumat, jotka liittyvät TripPay-tilisi elinkaareen ja asetuksiin.

| Tapahtuma | Avain | Kuvaus |
|---|---|---|
| Tili luotu | `account.create` | Uusi tili on luotu. |
| Tili tarkastettavana | `account.under_review` | Tili on tarkastettavana. |
| Tili päivitetty | `account.updated` | Tilin tiedot ovat muuttuneet. |
| Tilin tila päivitetty | `account.status.updated` | Tilin tila on muuttunut (esim. hyväksytty, keskeytetty). |
| Tilin osoite päivitetty | `account.address.updated` | Tilin osoitetta on muokattu. |
| Tilin tiedot päivitetty | `account.details.updated` | Tilin yksityiskohtaiset tiedot ovat muuttuneet. |
| Tilin tiedot päivitetty | `account.information.update` | Yleiset tilitiedot on päivitetty. |
| Tilin hallinnoijat päivitetty | `account.managers.update` | Tilin hallinnoijalista on muuttunut. |
| Profiilikuva päivitetty | `account.profile_picture.updated` | Tilin profiilikuva on vaihdettu. |
| Agentti päivitetty | `account.agent.updated` | Tilin agenttitiedot on muokattu. |
| URL-nimi päivitetty | `account.url-name.update` | Tilin mukautettu URL on muuttunut. |
| Yksilöllinen tunniste päivitetty | `account.unique-id.update` | Tilin yksilöllinen tunniste on muokattu. |

### Tilin tehtävät

| Tapahtuma | Avain | Kuvaus |
|---|---|---|
| Tehtävä luotu | `account.task.create` | Tilille on lisätty uusi tehtävä. |
| Tehtävä päivitetty | `account.task.update` | Tilin tehtävää on muokattu. |
| Tehtävä poistettu | `account.task.remove` | Tilin tehtävä on poistettu. |
| Tehtävä suoritettu | `account.task.complete` | Tilin tehtävä on suoritettu. |
| Verotunnisteen tehtävä suoritettu | `account.task.tax_id.complete` | Verotunnisteen vahvistustehtävä on suoritettu. |

### Tilin maksunvälittäjä

| Tapahtuma | Avain | Kuvaus |
|---|---|---|
| Maksunvälittäjä luotu | `account.acquirer.created` | Tilille on lisätty uusi maksunvälittäjä. |
| Maksunvälittäjä päivitetty | `account.acquirer.updated` | Tilin maksunvälittäjää on muokattu. |
| Maksunvälittäjä poistettu | `account.acquirer.removed` | Maksunvälittäjä on poistettu tililtä. |

## Hallinnoivan yksikön tapahtumat
