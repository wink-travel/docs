Jos olet matkailualan yritys ja haluat ulkoistaa maksujen ja maksujen jakamisen TripPaylle, tässä on 6 perusaskelta aloittamiseen:

<Steps>
1. Luo [käyttäjätili](/fi/user/register) Winkissä.
2. Luo tili [TripPayssä](https://pay.wink.travel).
3. Luo [Sovellus](/fi/developers/applications) TripPayssä, joka mahdollistaa yhteyden APIimme.
4. Luo ja yhdistä edunsaajasi TripPayhin. *esim. Useimmille tämä tarkoittaa tilien luomista alustan kohteille, joille haluat TripPayn maksavan.*
5. Integroi [maksun web-komponentti](/fi/developers/web-components/#payment) sivustollesi.
6. Lähetä TripPaylle matkustajan `ostoskori` ja käynnistä maksuprosessi.
</Steps>

:::tip
Täydellisen oppaan TripPayn käyttämiseen matkailusivustollasi löydät oppaasta [Integrointi TripPayn kanssa](/fi/guides/integrating-with-trip-pay).
:::

## API

Kehittäjät, jotka haluavat hallita `Maksuja`, siirtykää kohtaan [Kehittäjät > API > Maksut](/fi/developers/apis/#payment-api).
