Jos olet matkatoimisto ja haluat ulkoistaa yrityksesi maksu- ja kulukäsittelyn TripPaylle, tässä on kuusi perusvaihetta aloittamiseen:

<Steps>
  1. Luo [käyttäjätili](/user/register) Winkissä.
  2. Luo tili [TripPay](https://pay.wink.travel).
  3. Luo [Hakemus](/developers/applications) TripPayssa, jonka avulla voit kommunikoida API:mme kanssa.
  4. Luo ja kartoita edunsaajasi TripPaylla. *esim. Useimmille tämä tarkoittaa tilien luomista alustallasi oleville kiinteistöille, joille haluat TripPayn maksavan.*
  5. Integroi [maksuverkkokomponentti](/developers/web-components/#payment) sivustollesi.
  6. Lähetä TripPay matkailijalle `shopping cart` ja käynnistää maksuprosessin.
</Steps>


:::vihje
Saat täydellisen oppaan TripPayn käyttöön matkailusivustollasi lukemalla oppaan[Integrointi TripPayn kanssa](/guides/integrating-with-trip-pay).
:::

## API

Kehittäjät, jotka haluavat hallita`Payments`, mene osoitteeseen[Kehittäjät > API > Maksut](/developers/apis/#payment-api).
