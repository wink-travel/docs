Jos olet integraattori, kuten Wink, ja haluat kartoittaa ulkoiset maksunsaajasi TripPayn käyttöä varten, voit tehdä sen kahdella tavalla:

- Manuaalinen kartoitus
- Käyttämällä [Payment APIa](/fi/developers/apis/#payment-api) (*Katso `Account Mappings`*)

:::tip
Ellet käsittele vain pientä määrää samoja kohteita, suosittelemme käyttämään APIa tämän vaiheen automatisointiin.
:::

Alla käymme läpi manuaalisen tavan kartoittaa ulkoinen maksunsaaja. *Se käyttää APIa taustalla.*

<Steps>
1. Kirjaudu sisään [TripPayhin](https://pay.wink.travel).
2. Valitse integraattoritilisi.
3. Klikkaa päävalikosta `Settings > Mappings`.
4. Klikkaa `Create mapping` -painiketta. Näet kartoituslomakkeen.
5. **Local account** Ala kirjoittaa olemassa olevan TripPay-tilin nimeä, jonka haluat kartoittaa.
6. **External identifier** Liitä ulkoinen tunniste tälle taholle, kuten se on kaukaisessa järjestelmässä.
7. Klikkaa `Save` jatkaaksesi. Sinut ohjataan takaisin kartoituslistaan.
</Steps>

Kun kartoitukset ovat paikoillaan, TripPay tietää nyt, mitä tilejä hyvittää varauksen edunsaajina, kun viittaat niihin ulkoisilla tunnisteillasi.

## API

Kehittäjät, jotka haluavat hallita `Payments`-toimintoja, siirtyvät kohtaan [Developers > API > Payments](/fi/developers/apis/#payment-api).
