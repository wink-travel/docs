Jos olet integraattori, kuten Wink, ja haluat yhdistää ulkoiset maksunsaajasi TripPayn käyttöön, voit tehdä sen kahdella tavalla:

* Manuaalinen kartoitus
* Käyttämällä[Maksu-API](/developers/apis/#payment-api)(*Katso`Account Mappings`*)

:::vihje
Ellei kyseessä ole vain muutama samankaltainen ominaisuus, suosittelemme, että käytät API:a tämän vaiheen automatisointiin.
:::

Alla opastamme sinua ulkoisen maksunsaajan manuaalisessa yhdistämisessä.*Se käyttää API:a konepellin alla.*

<Steps>
  1. Kirjaudu sisään [TripPay](https://pay.wink.travel).
  2. Valitse integraattoritilisi.
  3. Klikkaa `Settings > Mappings` päänavigointipalkista.
  4. Klikkaa `Create mapping` -painiketta. Näet kartoituslomakkeen.
  5. **Paikallinen tili** Ala kirjoittaa sen olemassa olevan TripPay-tilin nimeä, jonka haluat yhdistää.
  6. **Ulkoinen tunniste** Liitä tämän yksikön ulkoinen tunnuksesi sellaisena kuin se on etäjärjestelmässä.
  7. Klikkaa `Save` jatkaaksesi. Sinut ohjataan takaisin kartoitusluetteloosi.
</Steps>


Kun yhdistämismäärityksesi ovat käytössä, TripPay tietää nyt, mille tileille hyvitetään varauksen edunsaajia, kun viittaat niihin ulkoisilla tunnisteillasi.

## API

Kehittäjät, jotka haluavat hallita`Payments`, mene osoitteeseen[Kehittäjät > API > Maksut](/developers/apis/#payment-api).
