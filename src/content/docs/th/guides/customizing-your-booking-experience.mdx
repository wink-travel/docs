การปรับแต่งควบคุมวิธีการแสดงสินค้าคงคลังของคุณให้กับนักเดินทาง — สีธีม สกุลเงินเริ่มต้น ภาษา แผนการเดินทาง และอื่นๆ อีกมากมาย ใช้ได้กับลิงก์ที่แชร์ได้ การ์ดฝัง กริด แผนที่ และเครื่องมือจอง

## การปรับแต่งคืออะไร?

การปรับแต่งคือโปรไฟล์การตั้งค่าที่กำหนด:

- **สกุลเงินที่แสดง** — สกุลเงินที่ราคาจะแสดง
- **ภาษาที่แสดง** — ภาษาปริยายสำหรับเนื้อหา
- **แผนการเดินทางเริ่มต้น** — วันที่เช็คอิน/เช็คเอาต์และจำนวนผู้เข้าพักที่กรอกไว้ล่วงหน้า
- **ธีม** — สีและแบรนด์ที่ใช้กับส่วนประกอบฝัง
- **การตั้งค่า** — วิธีการนำเสนอสินค้าคงคลัง (เลย์เอาต์ การจัดเรียง)

คุณสามารถสร้างการปรับแต่งหลายรายการสำหรับตลาดหรือกรณีการใช้งานที่แตกต่างกันได้

## การสร้างการปรับแต่ง

<Steps>
1. ไปที่ https://studio.wink.travel (หรือบัญชีของคุณที่ https://my.wink.travel)
2. ไปที่ `Customization` จากแถบด้านข้าง
3. คลิก `Create a new customization`
4. ตั้งชื่อที่อธิบายได้ (เช่น "Europe - EUR", "Japan Market", "Default")
5. ตั้งค่าสกุลเงินที่แสดง ภาษา และแผนการเดินทางเริ่มต้น
6. กำหนดสีธีมให้ตรงกับแบรนด์ของคุณ
7. คลิก `Save`
</Steps>

## การใช้งานการปรับแต่ง

เมื่อสร้างแล้ว การปรับแต่งจะถูกอ้างอิงทั่วทั้งแพลตฟอร์ม:

| ฟีเจอร์ | วิธีการใช้การปรับแต่ง |
| -- | -- |
| **ลิงก์ที่แชร์ได้** | ลิงก์ใช้สกุลเงิน ภาษา และแผนการเดินทางของการปรับแต่งเป็นค่าเริ่มต้น |
| **การ์ดฝัง** | การ์ดแสดงด้วยสีธีมและการตั้งค่าการแสดงผลของคุณ |
| **กริดและแผนที่** | กริดสินค้าคงคลังและเครื่องหมายแผนที่ตามการปรับแต่ง |
| **เครื่องมือจอง** | กระบวนการจองทั้งหมดใช้การตั้งค่าของการปรับแต่ง |
| **โพสต์โซเชียล** | โพสต์ที่สร้างโดย AI สามารถใช้การปรับแต่งเฉพาะสำหรับการแสดงราคาที่ต้องการ |

## การปรับแต่งหลายรายการ

รูปแบบที่พบบ่อยคือการสร้างการปรับแต่งหนึ่งรายการต่อแต่ละตลาดเป้าหมาย:

- **"US Market"** — USD, ภาษาอังกฤษ, พัก 2 คืนเป็นค่าเริ่มต้น
- **"Europe"** — EUR, ภาษาอังกฤษ, พัก 3 คืนเป็นค่าเริ่มต้น
- **"Japan"** — JPY, ภาษาญี่ปุ่น, พัก 1 คืนเป็นค่าเริ่มต้น

เมื่อคุณสร้างลิงก์ที่แชร์ได้หรือฝังการ์ด คุณจะเลือกการปรับแต่งที่จะใช้ ซึ่งหมายความว่าอสังหาริมทรัพย์เดียวกันสามารถนำเสนอแตกต่างกันตามกลุ่มเป้าหมาย

## การแทนที่ผ่าน URL

คุณยังสามารถแทนที่ค่าปริยายของการปรับแต่งโดยใช้พารามิเตอร์ URL บนลิงก์ที่แชร์ได้:

- `l=es` — แทนที่ภาษาเป็นภาษาสเปน
- `c=EUR` — แทนที่สกุลเงินเป็น EUR
- `sd=2024-08-24&n=2` — แทนที่แผนการเดินทางเป็นวันที่และจำนวนคืนที่ระบุ

ดู [The Book Now Button](/th/guides/the-book-now-button) สำหรับรายการพารามิเตอร์ URL ทั้งหมด
