ไม่ใช่ทุกวันที่มีราคาที่เหมือนกัน เหตุการณ์ วันหยุด ฤดูกาลที่ไม่ใช่ช่วงพีค และช่องว่างในนาทีสุดท้าย ล้วนต้องการการตั้งราคาที่แตกต่างกัน คู่มือนี้แสดงวิธีใช้ราคาพิเศษและข้อจำกัดของแผนอัตราเพื่อจัดการกับสถานการณ์เหล่านี้

## การตั้งราคาเหตุการณ์

เมื่อการประชุม เทศกาล หรือเหตุการณ์ท้องถิ่นเพิ่มความต้องการ คุณจะต้องการเพิ่มอัตราสำหรับวันที่เหล่านั้น

<Steps>
1. ไปที่ `Monetize > Special Rates` ใน Extranet ของคุณ
2. คลิก `Create a new special rate`
3. เลือกอัตราหลักและประเภทห้องที่ต้องการแทนที่
4. ตั้งช่วงวันที่ให้ครอบคลุมช่วงเหตุการณ์
5. กรอกราคาต่อวันที่ปรับแล้ว (สูงขึ้น)
6. คลิก `Save`
</Steps>

อัตราพิเศษจะแทนอัตราหลักเฉพาะในวันที่กำหนดเท่านั้น ราคาปกติของคุณจะไม่เปลี่ยนแปลงนอกช่วงเหตุการณ์

## การปรับตามฤดูกาล

สำหรับช่วงเวลาที่ยาวนานขึ้น (ฤดูสูงสุด ฤดูไหล่ ฤดูต่ำ) คุณสามารถ:

- **ใช้ราคาพิเศษ** — สร้างการแทนที่สำหรับแต่ละฤดูกาลโดยปรับราคา
- **อัปเดตอัตราหลัก** — ปรับอัตราฐานโดยตรงสำหรับช่วงเวลานั้น

ราคาพิเศษเหมาะสำหรับการปรับชั่วคราว ขณะที่การอัปเดตอัตราหลักเหมาะกับรูปแบบประจำปีที่คาดการณ์ได้

## อัตรานาทีสุดท้าย

เพื่อเติมห้องที่ยังไม่ขายใกล้วันเช็คอิน:

<Steps>
1. สร้าง **แผนอัตรา** ใหม่พร้อมข้อจำกัดการจองล่วงหน้า (เช่น ต้องจองไม่เกิน 3 วันล่วงหน้า)
2. จับคู่กับ **ราคาพิเศษ** ที่ให้ส่วนลดสำหรับวันที่เป้าหมาย
3. แผนอัตรานี้จะแสดงเฉพาะกับนักเดินทางที่จองในช่วงนาทีสุดท้าย
</Steps>

## วันที่ปิดขาย

เพื่อบล็อกวันที่บางวันจากแผนอัตราเฉพาะ (เช่น ไม่มีอัตราส่งเสริมการขายในช่วงฤดูพีค):

<Steps>
1. ไปที่ `Monetize > Rate Plans` และเลือกแผนอัตรา
2. ไปที่ส่วน **Restrictions**
3. เพิ่มข้อจำกัด **Sell date** เพื่อจำกัดช่วงเวลาที่สามารถจองอัตรานี้ได้
4. หรือใช้ข้อจำกัด **Stay date** เพื่อจำกัดช่วงเวลาที่อัตรานี้ครอบคลุม
</Steps>

ตัวอย่างเช่น คุณสามารถจำกัดแผนอัตรา "Early Bird" ไม่ให้จองได้ในเดือนธันวาคม (ฤดูพีค) ขณะที่แผนอัตราปกติยังเปิดให้จองอยู่

## การรวมข้อจำกัด

ข้อจำกัดสามารถซ้อนกันเพื่อควบคุมอย่างแม่นยำ:

| ข้อจำกัด | สิ่งที่ควบคุม |
| -- | -- |
| **Sell dates** | ช่วงเวลาที่สามารถจองอัตราได้ |
| **Stay dates** | วันที่เข้าพักที่อัตราครอบคลุม |
| **Available days** | วันในสัปดาห์ที่อัตราปรากฏ |
| **Arrival days** | วันที่ผู้เข้าพักต้องเช็คอิน |
| **Departure days** | วันที่ผู้เข้าพักต้องเช็คเอาท์ |
| **Required days** | ผู้เข้าพักต้องพักอย่างน้อยหนึ่งวันในช่วงนี้ |
| **Advance booking** | ระยะเวลาล่วงหน้าที่ต้องจอง |
| **Length of stay** | จำนวนคืนขั้นต่ำและสูงสุด |
