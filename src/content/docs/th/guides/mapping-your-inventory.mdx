เมื่อคุณเพิ่มสินค้าคงคลังให้กับที่พักของคุณบน Extranet — ห้องพัก ร้านอาหาร สปา ห้องประชุม กิจกรรม และอื่นๆ — สินค้าคงคลังนั้นจะพร้อมใช้งานทั่วทั้งระบบนิเวศของ Wink คู่มือนี้อธิบายว่าชิ้นส่วนเหล่านี้เชื่อมต่อกันอย่างไร

## การไหลของสินค้าคงคลัง

```
Extranet (คุณสร้าง) → Booking Engine (นักเดินทางเห็น) → Studio (พันธมิตรขาย)
```

1. **คุณสร้างสินค้าคงคลัง** บน Extranet — ประเภทห้อง สิ่งอำนวยความสะดวก และประสบการณ์
2. **นักเดินทางเรียกดู** สินค้าคงคลังนั้นบนระบบจองผ่านลิงก์ที่แชร์ได้ การ์ด หรือการค้นหาโดยตรง
3. **พันธมิตรค้นพบ** สินค้าคงคลังของคุณบน Studio สร้างการ์ดและกริด และขายให้กับผู้ชมของพวกเขา

## ประเภทห้องพัก

ประเภทห้องพักเป็นหัวใจหลักของสินค้าคงคลังของคุณ ประเภทห้องแต่ละประเภทต้องมี:

- ชื่อและคำอธิบาย
- การจัดเตียง
- จำนวนผู้เข้าพักสูงสุด (ผู้ใหญ่และเด็ก)
- สิ่งอำนวยความสะดวกเฉพาะห้อง
- รูปภาพ
- แผนราคาหนึ่งแผนขึ้นไปพร้อมอัตราหลัก

หากไม่มีราคา ประเภทห้องจะมีอยู่แต่ไม่สามารถจองได้

## สิ่งอำนวยความสะดวกและประสบการณ์

นอกเหนือจากห้องพัก คุณสามารถลงรายการ:

| ประเภทสินค้าคงคลัง | คืออะไร | ตัวอย่าง |
| -- | -- | -- |
| **ร้านอาหาร** | ร้านอาหารในสถานที่ | "Rooftop Bar & Grill" |
| **สปา** | บริการสุขภาพและความงาม | "Serenity Spa - นวด 60 นาที" |
| **ห้องประชุม** | พื้นที่จัดงานและประชุม | "Boardroom A - 20 คน" |
| **กิจกรรม** | สิ่งที่ทำได้ที่หรือใกล้ที่พัก | "ทัวร์ดำน้ำดูปะการังพร้อมไกด์" |
| **สถานที่ท่องเที่ยว** | สถานที่ใกล้เคียงที่น่าสนใจ | "พิพิธภัณฑ์แห่งชาติ - 500 ม." |
| **สถานที่** | สถานที่น่าสนใจ | "ตลาดท้องถิ่น" |
| **สิ่งเสริม** | สิ่งเพิ่มเติมที่ซื้อพร้อมห้องพัก | "บริการรับส่งสนามบิน" |

แต่ละประเภทสิ่งอำนวยความสะดวกมีส่วนของตัวเองใน Extranet พันธมิตรสามารถขายแต่ละรายการเหล่านี้แยกกัน — นักเดินทางอาจค้นพบที่พักของคุณผ่านรายการสปาแทนการค้นหาห้องพัก

## วิธีที่พันธมิตรใช้สินค้าคงคลังของคุณ

บน Studio พันธมิตรสามารถ:

- **ค้นหา** ที่พักของคุณหรือสินค้าคงคลังแต่ละรายการ
- **สร้างการ์ด** — การ์ดที่จองได้สำหรับห้องพัก การรักษาสปา หรือกิจกรรม
- **สร้างกริด** — คอลเลกชันของการ์ดที่แสดงพร้อมกัน
- **สร้างแผนที่** — ตัวชี้ตำแหน่งที่แสดงสินค้าคงคลังที่จองได้
- **แชร์ลิงก์** — URL โดยตรงไปยังที่พักของคุณหรือรายการเฉพาะ

สินค้าคงคลังที่ครบถ้วนและหลากหลายมากขึ้น จะมีวิธีที่พันธมิตรสามารถโปรโมตที่พักของคุณได้มากขึ้น

## เคล็ดลับ

- **ละเอียดถี่ถ้วน** — ลงรายการทุกอย่างที่ที่พักของคุณมี แต่ละรายการเป็นจุดเริ่มต้นที่เป็นไปได้สำหรับการจอง
- **ใช้รูปภาพคุณภาพดี** — สินค้าคงคลังแต่ละรายการควรมีภาพคุณภาพสูงอย่างน้อยหนึ่งภาพ
- **อัปเดตคำอธิบายเสมอ** — ปรับคำอธิบายเมื่อคุณเปลี่ยนแปลงข้อเสนอหรือเปลี่ยนตามฤดูกาล
- **ตั้งราคาทุกอย่าง** — สินค้าคงคลังที่ไม่มีราคา สามารถเรียกดูได้แต่ไม่สามารถจองได้
