ปุ่ม **BOOK NOW** เป็นคำที่ใช้กันทั่วไปในอุตสาหกรรมการบริการที่หมายถึงการผสานรวมของโรงแรมกับผู้ให้บริการที่ทราบข้อมูลเกี่ยวกับความพร้อมใช้งานและราคาของที่พัก และสามารถให้ผู้เดินทางจองห้องพักได้ ปุ่มนี้เป็นวิธีที่ง่ายที่สุดสำหรับโรงแรมในการขายห้องพักบนเว็บไซต์ของตนเองและตามเงื่อนไขของตนเอง

*โรงแรมหลายแห่งใช้ระบบจองที่รวมมาให้โดยผู้จัดการช่องทางของพวกเขา เราแนะนำให้โรงแรมทำการวิจัยและเปรียบเทียบระบบจองเนทีฟก่อนที่จะตัดสินใจเลือกใช้ระบบใดระบบหนึ่ง*

## กลุ่มเป้าหมาย

บทความนี้เหมาะสำหรับโรงแรมที่ต้องการติดตั้งระบบจองฟรีที่มีอัตราการแปลงสูงของเราในเว็บไซต์ของตนเอง  
แต่... ยังเป็นประโยชน์สำหรับผู้ที่ต้องการใช้เทคโนโลยีของเราบนเว็บไซต์ของตนเองด้วย

## ผ่านลิงก์

สำหรับการผสานรวมที่ง่ายที่สุด เราแนะนำให้ใช้ [Shareable Link](/th/studio/shareable-links)

ขั้นตอนมีดังนี้:

<Steps>
1. ไปที่ https://extranet.wink.travel  
2. คลิก `Sign-In / Register` ที่มุมขวาบน และเข้าสู่ระบบด้วยบัญชีผู้ใช้ของคุณ  
3. เลือกที่พักที่คุณต้องการทำงานด้วยจากเมนูแบบเลื่อนลงมุมซ้ายบน  
4. คลิกที่ `Booking engine` ในแถบนำทาง  
5. หน้าจะโชว์ลิงก์ของคุณ  
6. ส่งลิงก์นี้ให้กับเว็บมาสเตอร์ของคุณและใช้สำหรับปุ่ม **BOOK NOW** ของคุณ  
7. คุณเสร็จเรียบร้อย ✅  
</Steps>

### ขั้นสูง

ลิงก์นี้มาพร้อมกับแผนการเดินทางที่ฝังอยู่ใน URL ซึ่งมาจาก `customization` ที่คุณเลือก  
คุณสามารถปรับแต่งลิงก์ของคุณเพิ่มเติมโดย:

- เปลี่ยนการปรับแต่งที่เลือก หรือ  
- เขียนทับแผนการเดินทาง

ในบทความนี้ เราจะเน้นที่ข้อหลังเพราะให้ความยืดหยุ่นมากขึ้นในการให้ผู้เดินทางเลือกแผนการเดินทางบนเว็บไซต์ของคุณและส่งต่อไปยังระบบจองของ Wink

นี่คือวิธีการส่งแผนการเดินทางของคุณพร้อมกับ URL

พารามิเตอร์คำขอที่ถูกต้อง:

- **sd** เพื่อเพิ่มวันที่เริ่มต้น: *sd=2024-08-24*  
- **n** เพื่อเพิ่มจำนวนคืน: *n=1* หรือ  
- **ed** ใช้วันที่สิ้นสุด: *ed=2024-08-25*  
- **rc** จำนวนผู้เข้าพัก: *rc=a2* (ผู้ใหญ่: 2)  
- **l** ขอภาษาที่แสดงผล: *l=es*  
- **c** ขอสกุลเงินที่แสดงผล: *c=EUR*  
- **p** เพิ่มโปรโมชั่น: *p=ABC123*

URL จะมีลักษณะดังนี้:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2&l=es&c=EUR

สำหรับกรณีการใช้งานขั้นสูง คุณยังสามารถขอการกำหนดค่าห้องพักหลายแบบโดยใช้เครื่องหมาย `+`:

ตัวอย่าง:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2-ca16q1-ca8q1+a2&l=es&c=EUR

- ประเภทห้อง 1 = ผู้ใหญ่: 2, เด็ก: 2 = อายุ 16 ปี + 8 ปี  
- ประเภทห้อง 2 = ผู้ใหญ่: 2

### ตัวอย่าง

ตัวอย่าง HTML ปุ่ม **BOOK NOW**:

*ตัวอย่างนี้สมมติว่าคุณใช้ [Bootstrap](https://getbootstrap.com) เป็นไลบรารี CSS หลักของคุณ*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Book a room"
>
BOOK NOW
</a>
```

### ประโยชน์อื่น ๆ

ยังมีประโยชน์อื่น ๆ ในการใช้ลิงก์ของเราเป็นปุ่ม `book now` ของคุณ

- **เหมาะกับโซเชียล** ลิงก์สามารถแชร์ได้ง่ายบนโซเชียลมีเดียและแสดง `dynamic pricing` บนภาพที่คุณเลือก  
- **สั้น** URL สั้นและแชร์ได้ง่าย คุณสามารถใช้ในที่ต่าง ๆ เช่น URL ประวัติ Instagram ของคุณ

## ผ่านการฝัง

โรงแรมบางแห่งต้องการฟังก์ชันการทำงานที่มากขึ้นและการผสานรวมที่ลึกขึ้นสำหรับเว็บไซต์ของตน ตัวอย่างนี้จะครอบคลุมการฝัง [Card](/th/studio/cards) ลงในเว็บไซต์ของพวกเขา

ขั้นตอนมีดังนี้:

<Steps>
1. ไปที่ https://studio.wink.travel  
2. คลิก `Sign-In / Register` ที่มุมขวาบน และเข้าสู่ระบบด้วยบัญชีผู้ใช้ของคุณ  
3. เลือกที่พักที่คุณต้องการทำงานด้วยจากมุมซ้ายบน  
4. คลิกที่ `Search` จากแถบนำทางหลัก  
6. พิมพ์ชื่อที่พักของคุณและคลิกปุ่มค้นหา  
7. คลิกลิงก์ `Actions` บนการ์ดผลการค้นหาและเลือก `Create > Make a Card`  
8. ระบบจะสร้าง `Card` ให้  
9. ข้อความจะปรากฏที่มุมขวาบนแจ้งว่าการ์ดของคุณพร้อมใช้งานและหากคุณต้องการปรับแต่งเพิ่มเติม  
10. คลิกที่ `Tools > Cards` จากแถบนำทางหลักและคุณจะเห็นการ์ดใหม่ของคุณ  
11. คลิกลิงก์ `Actions` บนการ์ดของคุณและคลิกที่ `Embed`  
12. หน้าต่างใหม่จะเปิดขึ้นพร้อมโค้ด HTML ตัวอย่างที่คุณสามารถคัดลอกได้  
13. เพื่อดึง `Client-ID` ของคุณ ไปที่ [Applications](/th/developers/applications)  
14. เพื่อดึง `Configuration-ID` ของคุณ ไปที่ [Customization](/th/studio/customization)  
14. คุณเสร็จเรียบร้อย ✅  
</Steps>

ตัวอย่าง HTML การ์ดฝัง:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*ตัวอย่าง HTML ข้างต้นไม่รวม CSS และ Javascript ของ Wink ที่จำเป็น; มีเพียงการ์ดเท่านั้น*

ผู้จองจะเห็นราคาที่เปลี่ยนแปลงได้ สามารถเปลี่ยนแผนการเดินทางและคลิกจองได้โดยตรงบนเว็บไซต์ของคุณ ปุ่มจองจะพาคุณไปยังหน้าลงจอดที่พักของ Wink เพื่อดำเนินการชำระเงินให้เสร็จสมบูรณ์
