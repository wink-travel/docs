การ**จองเลยตอนนี้**ปุ่มเป็นคำศัพท์ที่ใช้กันอย่างแพร่หลายในอุตสาหกรรมการบริการ ซึ่งหมายถึงการบูรณาการของโรงแรมกับผู้ให้บริการที่ทราบเกี่ยวกับห้องว่างและราคาของโรงแรม และสามารถให้ผู้เดินทางจองห้องพักได้ ปุ่มเป็นวิธีที่ง่ายที่สุดสำหรับโรงแรมที่จะสามารถขายห้องพักบนเว็บไซต์ของตนเองและตามเงื่อนไขของตนเอง

*โรงแรมหลายแห่งใช้ระบบจองห้องพักแบบรวมที่จัดทำโดย Channel Manager ของโรงแรม เราขอแนะนำให้โรงแรมค้นคว้าข้อมูลและเปรียบเทียบระบบจองห้องพักแบบเนทีฟก่อนตัดสินใจเลือกระบบใดระบบหนึ่งโดยเฉพาะ*

## ผู้ชม

บทความนี้มีไว้สำหรับโรงแรมที่ต้องการติดตั้งระบบจองห้องพักฟรีที่มีอัตราการแปลงสูงบนเว็บไซต์ของตน
แต่... บทความนี้ยังเป็นประโยชน์สำหรับผู้ที่ต้องการใช้เทคโนโลยีของเราบนเว็บไซต์ของตนอีกด้วย

## ผ่านลิงค์

สำหรับการบูรณาการที่ง่ายที่สุด เราขอแนะนำให้ใช้[ลิงค์ที่สามารถแชร์ได้](/studio/shareable-links)-

ขั้นตอนในการทำมีดังต่อไปนี้:

<Steps>
  1. ไปที่ https://extranet.wink.travel
  2. คลิก `Sign-In / Register` ที่มุมขวาบนและเข้าสู่ระบบด้วยบัญชีผู้ใช้ของคุณ
  3. เลือกคุณสมบัติที่คุณต้องการทำงานด้วยจากดรอปดาวน์ด้านซ้ายบน
  4. คลิ๊กที่นี่ `Booking engine` ในแถบนำทาง
  5. หน้าจะแสดงลิงก์ของคุณ
  6. ส่งลิงค์นี้ให้เว็บมาสเตอร์ของคุณและใช้มัน **จองเลยตอนนี้** ปุ่ม.
  7. คุณเป็น ✅
</Steps>


### ขั้นสูง

ลิงก์มาพร้อมกับแผนการเดินทางที่รวมอยู่ใน URL ซึ่งมีต้นทางจากที่คุณเลือก`customization`.
คุณสามารถปรับแต่งลิงก์ของคุณเพิ่มเติมได้โดย:

* การเปลี่ยนแปลงการปรับแต่งที่เลือกหรือ
* การแทนที่กำหนดการเดินทาง

ในบทความนี้เราจะเน้นที่ส่วนหลังเนื่องจากช่วยให้คุณมีอิสระมากขึ้นในการให้ผู้เดินทางเลือกแผนการเดินทางบนเว็บไซต์ของคุณและส่งต่อไปยังระบบจองของ Wink

นี่คือวิธีส่งแผนการเดินทางของคุณพร้อมกับ URL

พารามิเตอร์คำขอที่ถูกต้อง:

* **เอสดี**ในการเพิ่มวันที่เริ่มต้น:*sd=2024-08-24*
* **น**การเพิ่มจำนวนคืน:*น=1* หรือ
* **เอ็ด**ใช้วันที่สิ้นสุด:*เรียบเรียง=2024-08-25*
* **อาร์ซี**แขก:*อาร์ซี=เอ2*(ผู้ใหญ่ : 2)
* **ล**ขอภาษาที่แสดง:*ล=เอส*
* **ซี**ขอแสดงสกุลเงิน:*c=ยูโร*
* **พี**เพิ่มโปรโมชั่น:*พี=ABC123*

URL จะมีลักษณะดังนี้:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2\&l=es\&c=EUR

สำหรับกรณีการใช้งานขั้นสูง คุณยังสามารถขอการกำหนดค่าห้องหลายห้องได้โดยใช้`+` เข้าสู่ระบบ:

ตัวอย่าง:

ภาษาไทย: https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2-ca16q1-ca8q1+a2\&l=es\&c=EUR

* ประเภทห้อง 1 = ผู้ใหญ่ : 2, เด็ก : 2 = 16 ปี + 8 ปี.
* ห้องประเภท 2 = ผู้ใหญ่: 2.

### ตัวอย่าง

ตัวอย่าง**จองเลยตอนนี้**ปุ่ม HTML:

*ตัวอย่างนี้ถือว่าคุณกำลังใช้[บูตสแตรป](https://getbootstrap.com)เป็นไลบรารี CSS หลักของคุณ*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Book a room"
>
BOOK NOW
</a>
```

### ผลประโยชน์อื่น ๆ

มีข้อดีอื่นๆ ในการใช้ลิงก์ของเราเป็นของคุณ`book now` ปุ่ม.

* **เป็นมิตรกับสังคม**สามารถแชร์ลิงก์บนโซเชียลมีเดียได้อย่างง่ายดายและแสดง`dynamic pricing`บนสุดของรูปภาพที่คุณเลือก
* **สั้น**URL นี้สามารถแชร์ได้ และคุณสามารถใช้ได้ในที่ต่างๆ เช่น URL Bio ของ Instagram ของคุณ

## ผ่านการฝัง

โรงแรมบางแห่งต้องการฟังก์ชันเพิ่มเติมและการบูรณาการที่ลึกซึ้งยิ่งขึ้นสำหรับเว็บไซต์ของตน ตัวอย่างนี้จะครอบคลุมการฝัง[การ์ด](/studio/cards)เข้าไปในเว็บไซต์ของพวกเขา

ขั้นตอนในการทำมีดังต่อไปนี้:

<Steps>
  1. ไปที่ https://studio.wink.travel
  2. คลิก `Sign-In / Register` ที่มุมขวาบนและเข้าสู่ระบบด้วยบัญชีผู้ใช้ของคุณ
  3. เลือกคุณสมบัติที่คุณต้องการทำงานด้วยจากมุมซ้ายบน
  4. คลิ๊กที่นี่ `Search` จากแถบนำทางหลัก
  5. พิมพ์ชื่อทรัพย์สินของคุณและคลิกปุ่มค้นหา
  6. คลิกที่ `Actions` ลิงก์บนบัตรผลลัพธ์การค้นหาของคุณและเลือก `Create > Make a Card`-
  7. สิ่งนี้จะสร้าง `Card`-
  8. ข้อความจะปรากฏที่มุมขวาบนเพื่อแจ้งให้คุณทราบว่าบัตรของคุณพร้อมแล้ว และคุณต้องการปรับแต่งเพิ่มเติมหรือไม่
  9. คลิ๊กที่นี่ `Tools > Cards` จากแถบนำทางหลัก และคุณจะเห็นบัตรใหม่ของคุณ
  10. คลิกที่ `Actions` ลิงค์บนการ์ดลิงค์ของคุณและคลิกที่ `Embed`-
  11. นี้จะเปิดหน้าต่างใหม่พร้อมตัวอย่างโค้ด HTML ที่คุณสามารถคัดลอกได้
  12. เพื่อดึงข้อมูลของคุณ `Client-ID`, ไปที่ [แอปพลิเคชั่น](/developers/applications)-
  13. เพื่อดึงข้อมูลของคุณ `Configuration-ID`ไปที่ [การปรับแต่ง](/studio/customization)
  14. คุณเป็น ✅
</Steps>


ตัวอย่างการ์ดฝัง HTML:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*ตัวอย่าง HTML ด้านบนไม่มี Wink CSS และ Javascript ที่จำเป็น มีเพียงการ์ดเท่านั้น*
