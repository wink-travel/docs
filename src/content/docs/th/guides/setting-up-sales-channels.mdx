ช่องทางการขายคือวิธีที่คุณควบคุมการกระจายสินค้าคงคลัง ช่องทางการขายกำหนดว่าใครสามารถขายห้องของคุณ ค่าคอมมิชชั่นที่พวกเขาได้รับ และแผนราคาที่มีให้เลือก

## ประเภทของช่องทางการขาย

มีสามวิธีหลักในการกระจายที่พักของคุณบน Wink:

- **Wink Network** — ที่พักของคุณจะมองเห็นได้โดยพันธมิตรทั้งหมดบนแพลตฟอร์ม นี่คือการกระจายที่กว้างที่สุด
- **ช่องทางตรง** — คุณสร้างช่องทางส่วนตัวสำหรับพันธมิตรหรือคู่ค้ารายใดรายหนึ่งโดยมีอัตราค่าคอมมิชชั่นที่กำหนดเอง
- **เว็บไซต์ของคุณเอง** — ใช้ลิงก์ที่แชร์ได้หรือการ์ดฝังเพื่อขายโดยตรงจากเว็บไซต์ของคุณโดยไม่มีคนกลาง

## การสร้างช่องทางการขาย

<Steps>
1. ไปที่ https://extranet.wink.travel และเลือกที่พักของคุณ
2. ไปที่ส่วนช่องทางการขาย
3. คลิก `Create a new sales channel`
4. ตั้งชื่อช่องทาง (เช่น "Website Direct", "Partner - Kim Travel")
5. กำหนด **เปอร์เซ็นต์ค่าคอมมิชชั่น** — จำนวนที่พันธมิตรได้รับต่อการจองแต่ละครั้ง
6. เลือก **แผนราคา** ที่มีให้ในช่องทางนี้
7. จำกัดเฉพาะ **ประเภทห้อง** หรือ **สินค้าคงคลัง** ได้ตามต้องการ
8. คลิก `Save`
</Steps>

## Wink Network

Wink Network คือช่องทางการกระจายเริ่มต้น เมื่อเปิดใช้งาน ที่พักของคุณจะปรากฏใน [Studio](/th/studio/search) ซึ่งพันธมิตรทุกคนสามารถค้นหาและขายได้

วิธีเปิดใช้งาน:

<Steps>
1. ไปที่การตั้งค่าเครือข่ายของที่พักของคุณ
2. เปิดใช้งาน `Wink Network`
3. กำหนดอัตราค่าคอมมิชชั่นเริ่มต้นสำหรับพันธมิตรในเครือข่าย
</Steps>

:::tip
Wink Network เป็นวิธีที่ง่ายที่สุดในการเพิ่มการมองเห็นอย่างกว้างขวาง คุณสามารถสร้างช่องทางตรงในภายหลังสำหรับพันธมิตรที่ทำผลงานดีที่สุดของคุณโดยมีอัตราค่าคอมมิชชั่นที่แตกต่างกันได้เสมอ
:::

## ช่องทางพันธมิตรตรง

สำหรับพันธมิตรที่คุณต้องการทำงานอย่างใกล้ชิด ให้สร้างช่องทางตรงพร้อมค่าคอมมิชชั่นที่กำหนดเอง

<Steps>
1. ค้นหาพันธมิตรผ่านส่วนเครือข่าย ดูที่ [Network](/th/extranet/network)
2. สร้างช่องทางการขายตรงสำหรับพันธมิตรนั้น
3. กำหนดอัตราค่าคอมมิชชั่นที่กำหนดเอง (ซึ่งอาจสูงหรือต่ำกว่าอัตราเริ่มต้นของเครือข่าย)
4. เลือกสินค้าคงคลังที่มีให้ในช่องทางนี้
</Steps>

สิ่งนี้มีประโยชน์เมื่อคุณต้องการเสนออัตราพิเศษหรือการเข้าถึงเฉพาะสำหรับอินฟลูเอนเซอร์หรือเอเจนซี่ท่องเที่ยวบางราย

## โครงสร้างค่าคอมมิชชั่น

ค่าคอมมิชชั่นจะแสดงเป็นเปอร์เซ็นต์ของยอดรวมการจอง ค่าคอมมิชชั่นจะแบ่งระหว่างฝ่ายที่เกี่ยวข้อง:

- **ค่าคอมมิชชั่นพันธมิตร** — สิ่งที่ผู้ขาย (พันธมิตร, เอเจนซี่ท่องเที่ยว, อินฟลูเอนเซอร์) ได้รับ
- **ค่าธรรมเนียมแพลตฟอร์ม** — ค่าธรรมเนียมการดำเนินการของ Wink
- **รายได้ของที่พัก** — ส่วนที่เหลือจะเป็นของคุณ

คุณควบคุมค่าคอมมิชชั่นพันธมิตรได้ ค่าธรรมเนียมแพลตฟอร์มจะถูกหักโดยอัตโนมัติ

## การติดตามผลการดำเนินงาน

เมื่อช่องทางของคุณเปิดใช้งานแล้ว ใช้ [Analytics](/th/my-account/analytics) เพื่อติดตามว่าช่องทางใดสร้างการจองและรายได้มากที่สุด ซึ่งช่วยให้คุณปรับอัตราค่าคอมมิชชั่นและตัดสินใจลงทุนในพันธมิตรใด
