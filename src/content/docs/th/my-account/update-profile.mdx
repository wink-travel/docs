import { Steps } from '@astrojs/starlight/components';

หากคุณต้องการอัปเดตโปรไฟล์ผู้ใช้สำหรับบัญชีผู้ใช้ Wink ของคุณ ให้ทำตามขั้นตอนเหล่านี้:

<Steps>
1. ไปที่ [my.wink.travel](https://my.wink.travel).
2. เข้าสู่ระบบโดยคลิกที่ `Sign-in / Register` ที่มุมขวาบน.
3. เมื่อคุณเข้าสู่ระบบแล้ว ให้คลิกที่ไอคอนโปรไฟล์ของคุณที่มุมขวาบน จะมีเมนูปรากฏขึ้น.
4. คลิกที่ลิงก์ `Profile`.
5. ทำตามคำแนะนำเพื่ออัปเดต `e-mail`, `ชื่อ` และ `นามสกุล`.
6. คุณยังสามารถอัปโหลด `รูปโปรไฟล์` ใหม่ของคุณได้ด้วย.
7. คลิก `Save`.
</Steps>

<Steps>
1. คุณต้องเข้าสู่ระบบในหนึ่งในเว็บไซต์ของเรา.
2. คลิกที่ไอคอนโปรไฟล์ของคุณที่มุมขวาบน จะมีเมนูปรากฏขึ้น.
3. คลิกที่ลิงก์ `Profile`.
4. ทำตามคำแนะนำเพื่ออัปเดต `e-mail`, `ชื่อ` และ `นามสกุล`.
5. คุณยังสามารถอัปโหลด `รูปโปรไฟล์` ใหม่ของคุณได้ด้วย.
6. คลิก `Update`.
</Steps>

:::note
รูปโปรไฟล์ใหม่ของคุณจะแสดงผลเมื่อคุณออกจากระบบ Wink อย่างสมบูรณ์ครั้งถัดไป
:::
