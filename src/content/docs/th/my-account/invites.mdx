ผู้ใช้ Wink สามารถเลือกให้คุณจัดการบัญชีที่พัก, บริษัทในเครือ หรือเอเจนซี่ท่องเที่ยวของพวกเขาได้ เมื่อเกิดเหตุการณ์นั้น คุณจะได้รับอีเมลพร้อมคำขอเชิญ

import { Steps } from '@astrojs/starlight/components';

ในการดูคำเชิญ Wink ปัจจุบันของคุณ ให้ทำตามขั้นตอนเหล่านี้:

<Steps>
 1. เข้าสู่ระบบที่ [my.wink.travel](https://my.wink.travel)
 2. คลิกที่ `Invites` จากแถบด้านข้างซ้าย
 3. คุณจะเห็นรายการคำเชิญทั้งหมดในปัจจุบัน
 4. ส่วนนี้จะแสดงรายการใบสมัครทั้งหมดของคุณ
 5. เลือก `Accept` หรือ `Reject` คำเชิญ
 6. หากคุณยอมรับคำเชิญ บัญชีจะปรากฏในส่วน `Accounts`
</Steps>
