แอปพลิเคชันช่วยให้คุณเข้าถึงฟีเจอร์ทั้งหมดของ Wink ในฐานะนักพัฒนา คุณสามารถเข้าถึงข้อมูลทั้งหมดเดียวกันและขยายฟีเจอร์ของ Wink ได้
คุณสามารถเลือกใช้เซิร์ฟเวอร์ OAuth2 ของ Wink เพื่อยืนยันตัวตนเว็บไซต์ท่องเที่ยวของคุณในแบบเดียวกับที่ใช้ Google หรือ Facebook
ด้วยวิธีนี้คุณสามารถขยายฟีเจอร์ของ Wink ไปยังเว็บไซต์หรือแอปพลิเคชันของคุณเองได้

:::note
แอปแรกของคุณถูกสร้างขึ้นแล้วเมื่อคุณ [สร้างบัญชีใหม่](/th/my-account/create-account) เราใช้แอปนี้ในบางสถานการณ์การจอง

คุณไม่สามารถใช้แอปนี้ซ้ำได้ กรุณาสร้างแอปใหม่ตามที่คุณต้องการ
:::

<Steps>
 1. เลือกบัญชีที่คุณต้องการทำงานด้วยจาก [Accounts](https://my.wink.travel)
 2. คลิกที่ `Actions` ที่ด้านล่างของการ์ดบัญชีแล้วเลือก `Manage account`
 3. คลิกแท็บ `Applications` ในหน้าถัดไป
 4. ส่วนนี้จะแสดงรายการแอปพลิเคชันทั้งหมดของคุณ
 5. เลือกแก้ไขแอปที่มีอยู่หรือสร้างแอปใหม่
 </Steps>

## สร้างแอป

เนื่องจากคุณไม่สามารถเข้าถึง `secret key` ของแอปที่เราสร้างให้คุณได้ คุณจึงต้องสร้างแอปของคุณเอง

:::caution
คุณต้องสร้างบัญชีพันธมิตรบน Wink หรือบัญชีบน TripPay ก่อนสร้างแอป เพราะคุณจะผูกแอปพลิเคชันกับบัญชีเฉพาะ
:::

ขั้นตอนการสร้างแอปพลิเคชันมีดังนี้:

<Steps>
1. จากหน้าของแอป คลิกปุ่ม `Create new application`
2. ***ชื่อ*** ตั้งชื่อแอปของคุณ *เช่น Cool App*
3. **ชื่อหน่วยงาน** ผูกแอปของคุณกับบัญชีที่มีอยู่ *เช่น Cool Account*
4. **Redirect URIs** ต้องมีอย่างน้อยหนึ่ง redirect URI ใช้สำหรับตรวจสอบโดเมน redirect หลังจากยืนยันตัวตนสำเร็จ *เช่น https://www.cool-site.com*
5. คลิกปุ่ม `Save` เพื่อดำเนินการต่อ
</Steps>

คุณจะถูกเปลี่ยนเส้นทางกลับไปยังรายการแอปพลิเคชันของคุณ คุณจะเห็นแอปใหม่ในรายการและสามารถเข้าถึง secret key ของแอปได้เพียงครั้งเดียวเท่านั้น โปรดคัดลอกและเก็บไว้ในที่ปลอดภัย

## อัปเดตแอป

คุณจะต้องอัปเดตแอปของคุณหาก redirect URI ใดเปลี่ยนแปลง หรือคุณต้องการเพิ่มหรือลบ URI

ในการอัปเดตแอปที่มีอยู่ ให้ทำดังนี้:

<Steps>
1. คลิกที่ลิงก์ `Actions` ของแอปที่คุณต้องการอัปเดต
2. คลิกปุ่ม `Update` ภายใต้ Actions
3. แก้ไขแอปตามต้องการ
4. คลิกปุ่ม `Save` เพื่อดำเนินการต่อ
</Steps>

คุณจะถูกเปลี่ยนเส้นทางกลับไปยังรายการแอปพลิเคชันของคุณ

## ลบแอป

:::caution
กรุณาอย่าลบแอปที่เราสร้างให้อัตโนมัติเพื่อให้แน่ใจว่าฟีเจอร์ทั้งหมดทำงานได้กับบัญชีของคุณ
:::

หากคุณไม่ต้องการใช้แอปใดอีกต่อไป คุณสามารถลบแอปนั้นได้

ในการลบแอป ให้ทำดังนี้:

<Steps>
1. คลิกที่ลิงก์ `Actions` ของแอปที่คุณต้องการลบ
2. คลิกปุ่ม `Remove` ภายใต้ Actions
3. คลิกปุ่ม `OK` เพื่อยืนยันการลบ
</Steps>

คุณจะถูกเปลี่ยนเส้นทางกลับไปยังรายการแอปพลิเคชันของคุณ
