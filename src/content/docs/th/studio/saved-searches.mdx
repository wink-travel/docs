การค้นหาที่บันทึกไว้จะทำหน้าที่ตามที่ชื่อบ่งบอก ขณะค้นหาสินค้าคงคลังสำหรับการเดินทาง คุณมีตัวเลือกในการบันทึกคำค้นหาของคุณเพื่อใช้ในภายหลัง

## บันทึกคำค้นหาของคุณ

เมื่อคุณเริ่มค้นหาสินค้าคงคลัง (ดู[ค้นหา](/studio/search)) คุณสามารถปรับแต่งตัวกรองการค้นหาของคุณได้
ที่ด้านล่างของแบบฟอร์มแบบสอบถามจะมีตัวเลือกให้`Save search`-

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="บันทึกการค้นหา" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*แสดงลิงก์เพื่อบันทึกคำค้นหา*</figcaption>
</figure>


การคลิกลิงก์นี้จะทำให้คุณมีโอกาสตั้งชื่อการค้นหานี้ (*เช่น ห้องสมุดโรงแรมมหัศจรรย์ของฉัน*) อะไรบางอย่างที่คุณจะจำไว้ในภายหลัง

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="ตั้งชื่อการค้นหาของคุณ" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*ตั้งชื่อการค้นหาของคุณ*</figcaption>
</figure>


ตั้งชื่อแล้วคลิกที่`Save search`ปุ่ม หลังจากนั้นคุณสามารถไปที่`Inventory > Saves Searches`จากแถบนำทางหลักเพื่อดูบันทึกที่คุณสร้างขึ้นใหม่

## รายการการกระทำ

คุณสามารถใช้การค้นหาที่บันทึกไว้เพื่อเปิดเผยรายการเดินทางให้แก่ผู้ใช้ของคุณในรูปแบบตารางหรือแผนที่ และแสดงผลการค้นหาตามที่คุณต้องการได้

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="การดำเนินการค้นหาที่บันทึกไว้" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*บันทึกคำค้นหาโดยไม่ใช้ตัวกรอง*</figcaption>
</figure>


แบบสอบถามการค้นหาที่บันทึกไว้ทุกรายการมีการดำเนินการที่คุณสามารถดำเนินการได้:

* แปลงคำค้นหาเป็น[แผนที่](/studio/maps)-
* แปลงคำค้นหาเป็น[กริด](/studio/grids)-
* ลบการค้นหาที่บันทึกไว้

## เอพีไอ

นักพัฒนาที่ต้องการจัดการการค้นหาที่บันทึกไว้สามารถไปที่[นักพัฒนา > API > เรียกดู](/developers/apis/#browse-api)-

## อ่านเพิ่มเติม

* อ่านเกี่ยวกับของเรา[แผนที่](/studio/maps) คุณสมบัติ.
* อ่านเกี่ยวกับเรา[กริด](/studio/grids) คุณสมบัติ.
