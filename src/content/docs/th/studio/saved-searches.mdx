การค้นหาที่บันทึกไว้ทำตามที่ชื่อบอกไว้ ขณะค้นหาสินค้าคงคลังการเดินทาง คุณมีตัวเลือกในการบันทึกคำค้นหาของคุณเพื่อใช้งานในภายหลัง

## การบันทึกคำค้นหาของคุณ

เมื่อคุณเริ่มค้นหาสินค้าคงคลัง (ดูที่ [Search](/th/studio/search) ) คุณสามารถปรับแต่งตัวกรองการค้นหาของคุณได้อย่างละเอียด
ที่ด้านล่างของแบบฟอร์มคำค้นหาจะมีตัวเลือก `Save search`

การคลิกที่ลิงก์นี้จะให้คุณตั้งชื่อคำค้นหานี้ (*เช่น ห้องสมุดโรงแรมมหัศจรรย์ของฉัน*) ซึ่งเป็นชื่อที่คุณจะจำได้ในภายหลัง

ตั้งชื่อแล้วคลิกปุ่ม `Save search` หลังจากนั้นคุณสามารถไปที่ `Inventory > Saves Searches` จากแถบนำทางหลักเพื่อดูบันทึกที่คุณสร้างใหม่

## การดำเนินการในรายการ

คำค้นหาที่บันทึกไว้ของคุณสามารถใช้เพื่อแสดงสินค้าคงคลังการเดินทางให้กับผู้ใช้ของคุณในรูปแบบกริดหรือแผนที่ และทำให้ผลการค้นหาของคุณแสดงในแบบที่คุณต้องการ

คำค้นหาที่บันทึกไว้แต่ละรายการมีการดำเนินการที่คุณสามารถทำได้:

- แปลงคำค้นหาเป็น [map](/th/studio/maps)
- แปลงคำค้นหาเป็น [grid](/th/studio/grids)
- ลบคำค้นหาที่บันทึกไว้

## API

นักพัฒนาที่ต้องการจัดการคำค้นหาที่บันทึกไว้สามารถไปที่ [Developers > API > Browse](/th/developers/apis/#browse-api)

## การอ่านเพิ่มเติม

- อ่านเกี่ยวกับฟีเจอร์ [Map](/th/studio/maps) ของเรา
- อ่านเกี่ยวกับฟีเจอร์ [Grid](/th/studio/grids) ของเรา
