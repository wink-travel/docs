ส่วนที่เหลือของบทความนี้จะแนะนำวิธีการสร้าง ปรับแต่ง และแชร์แผนที่ของคุณ

## สร้างแผนที่

แผนที่สามารถสร้างได้สองวิธี:

- **แผนที่เครื่องหมายเดียว**: สร้างแผนที่จากสิ่งที่คุณพบขณะ [ค้นหา](/th/studio/search)  
- **แผนที่หลายเครื่องหมาย**: สร้างแผนที่จาก [รายการที่คัดสรร](/th/studio/curated-lists) หรือ [การค้นหาที่บันทึกไว้](/th/studio/saved-searches)

ภาพด้านบนมาจาก [ค้นหา](/th/studio/search) และแสดงบางสิ่งที่คุณสามารถทำได้กับผลการค้นหาของคุณ หนึ่งในนั้นคือ `สร้างแผนที่`  
คลิกที่ปุ่มนั้นแล้วจะพาคุณไปยังหน้าฟอร์มแผนที่ของเราเพื่อเริ่มปรับแต่งแผนที่ของคุณ

### แผนที่หลายเครื่องหมาย

ทั้ง [รายการที่คัดสรร](/th/studio/curated-lists#list-actions) และ [การค้นหาที่บันทึกไว้](/th/studio/saved-searches#list-actions) มีการดำเนินการที่ให้คุณสร้างแผนที่ แผนที่จะมีเครื่องหมายแผนที่สำหรับสินค้าคงคลังทั้งหมดในรายการเหล่านั้น

## ปรับแต่งแผนที่

ฟอร์มช่วยให้คุณปรับแต่งแผนที่ของคุณได้ดังนี้:

<Steps>
 1. ตั้งชื่อเพื่อให้คุณจำได้ว่าแผนที่เกี่ยวกับอะไร  
 2. เลือกว่าคุณต้องการให้ผู้ใช้สามารถเคลื่อนที่ไปรอบๆ แผนที่ได้หรือไม่  
 3. กำหนดความสูงของแผนที่เป็นพิกเซล  
 4. เลือกว่าคุณต้องการให้ผู้ใช้สามารถซูมเข้าและออกจากแผนที่ได้หรือไม่  
 5. เลือกสไตล์แผนที่ที่เหมาะกับเว็บไซต์ที่คุณจะฝังแผนที่ลงไป  
 6. เลือกสีของเครื่องหมาย  
 7. เลือกหน้าการ์ดเริ่มต้นที่คุณต้องการให้ผู้ใช้เห็นก่อน โดยค่าเริ่มต้นคือหน้าการ์ดเนทีฟของสินค้าคงคลังนั้น  
 8. คุณสามารถเพิ่มเครื่องหมาย วงกลม สี่เหลี่ยม และรูปหลายเหลี่ยมของคุณเองเพื่อระบุสถานที่อื่นๆ บนแผนที่ที่ผู้ใช้อาจสนใจ  
 9. คลิกปุ่ม `บันทึก` เพื่อดำเนินการต่อ  
</Steps>

หลังจากบันทึกแผนที่ของคุณแล้ว คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าของแผนที่ของคุณ และแผนที่ของคุณก็พร้อมที่จะแชร์กับโลกแล้ว

ภาพด้านบนแสดงให้คุณเห็นภาพรวมของการดำเนินการทั้งหมดที่มีสำหรับแผนที่ของคุณ:

- **อัปเดต** อัปเดตการตั้งค่าแผนที่ของคุณ  
- **เพิ่มไปยัง WinkLinks** เพิ่มแผนที่ไปยังบัญชี WinkLinks ของคุณ  
- **ฝัง** แสดงวิธีฝังแผนที่นี้เป็น [Map](/th/developers/web-components/#content-loader) ลงในเว็บไซต์ของคุณ  
- **ใช้กับ WordPress** แสดงวิธีใช้ [ปลั๊กอิน WordPress](/th/developers/wordpress) ของเราเพื่อฝังแผนที่นี้ลงในเว็บไซต์ของคุณ  

เราจะอธิบายตัวเลือกเหล่านี้อย่างละเอียดเพิ่มเติมด้านล่าง

### ฝัง

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

นี่คือวิธีฝังแผนที่ของคุณลงในเว็บไซต์:

- บรรทัดที่ 3 แสดงวิธีฝังสไตล์ของ Wink ลงในเว็บไซต์ของคุณ  
- บรรทัดที่ 6 ถึง 9 แสดงวิธีใช้ [wink-content-loader](/th/developers/web-components/#content-loader) Web Component และบอกให้ดึงแบนเนอร์โฆษณาสำหรับโค้ดของคุณ  
- บรรทัดที่ 11 แสดงวิธีฝัง Javascript ของเราในเว็บไซต์ของคุณ  
- บรรทัดที่ 13 แสดงวิธีฝัง [wink-app-loader](/th/developers/web-components/#application-loader) Web Component และบอกให้ดึงการตั้งค่าระดับหน้าของคุณ  

## API

นักพัฒนาที่ต้องการจัดการแผนที่สามารถไปที่ [Developers > API > Inventory](/th/developers/apis/#inventory-api)
