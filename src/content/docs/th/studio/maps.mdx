แผนที่แสดงซัพพลายเออร์รายเดียว (เช่น โรงแรมหรือผู้ให้บริการประสบการณ์) หรือสินค้าคงคลัง (เช่น ประเภทห้อง ห้องประชุม สปา เป็นต้น) หรือคอลเลกชันสินค้าคงคลัง เช่น รายการที่คัดสรรหรือการค้นหาที่บันทึกไว้ และแสดงรายละเอียดเหล่านั้นผ่านแผนที่แบบโต้ตอบที่มีเครื่องหมายแผนที่ที่จองได้

บทความที่เหลือนี้จะแนะนำวิธีการสร้าง ปรับแต่ง และแชร์แผนที่ของคุณ

## สร้างแผนที่

แผนที่สามารถสร้างได้สองวิธี:

* **แผนที่เครื่องหมายเดี่ยว**: สร้างแผนที่จากสิ่งที่คุณพบขณะ[การค้นหา](/studio/search)-
* **แผนที่หลายเครื่องหมาย**: สร้างแผนที่จาก[รายการที่คัดสรร](/studio/curated-lists) หรือ [บันทึกการค้นหา](/studio/saved-searches)-

### แผนที่เครื่องหมายเดี่ยว

  <figcaption>*การดำเนินการผลการค้นหา*</figcaption>

ด้านบนเป็นภาพที่นำมาจาก[ค้นหา](/studio/search)และแสดงให้คุณเห็นบางสิ่งที่คุณสามารถทำได้กับผลการค้นหาของคุณ หนึ่งในการดำเนินการเหล่านั้นคือ`Make a map`.
คลิกที่ปุ่มนั้นแล้วมันจะพาคุณไปที่หน้าแบบฟอร์มแผนที่ซึ่งคุณสามารถเริ่มปรับแต่งแผนที่ของคุณได้

### แผนที่หลายเครื่องหมาย

ทั้งคู่ [รายการที่ได้รับการคัดสรร](/studio/curated-lists#list-actions) และ [การค้นหาที่บันทึกไว้](/studio/saved-searches#list-actions)มาพร้อมกับการดำเนินการที่ช่วยให้คุณสร้างแผนที่ แผนที่นี้จะมีเครื่องหมายแผนที่สำหรับสินค้าคงคลังทั้งหมดภายในรายการเหล่านั้น

## ปรับแต่งแผนที่

แบบฟอร์มนี้ช่วยให้คุณปรับแต่งแผนที่ของคุณได้ในรูปแบบต่อไปนี้:

<Steps>
  1. ตั้งชื่อให้เพื่อจะได้จำได้ว่าแผนที่นี้เป็นเกี่ยวกับอะไร
  2. เลือกว่าคุณต้องการให้ผู้ใช้สามารถเคลื่อนที่ไปรอบ ๆ แผนที่หรือไม่
  3. ตั้งค่าความสูงเป็นพิกเซลของแผนที่ที่คุณต้องการ
  4. เลือกว่าคุณต้องการให้ผู้ใช้ซูมเข้าหรือออกจากแผนที่หรือไม่
  5. เลือกสไตล์แผนที่ที่เข้ากับเว็บไซต์ที่คุณจะฝังแผนที่ลงไป
  6. เลือกสีปากกาเมจิก
  7. เลือกหน้าการ์ดเริ่มต้นที่คุณต้องการให้ผู้ใช้เห็นก่อน โดยค่าเริ่มต้น หน้าการ์ดจะเป็นหน้าดั้งเดิมของสินค้าคงคลังนั้น
  8. คุณสามารถเพิ่มเครื่องหมาย วงกลม สี่เหลี่ยมผืนผ้า และรูปหลายเหลี่ยมของคุณเองเพื่อระบุตำแหน่งอื่นๆ บนแผนที่ที่ผู้ใช้ของคุณอาจสนใจ
  9. คลิกที่ `Save` ปุ่มเพื่อดำเนินการต่อ
</Steps>


หลังจากบันทึกแผนที่ของคุณแล้ว คุณจะถูกส่งต่อไปยังหน้าแผนที่ และแผนที่ของคุณก็พร้อมที่จะแชร์กับคนทั่วโลกแล้ว

## แบ่งปันแผนที่

  <figcaption>*รายการแผนที่พร้อมเมนูการกระทำที่ยุบลง*</figcaption>

ด้านบนแสดงภาพพร้อมการดำเนินการทั้งหมดที่มีสำหรับแผนที่ของคุณ:

* **อัปเดต**อัพเดตการกำหนดค่าแผนที่ของคุณ
* **เพิ่มไปยัง WinkLinks**เพิ่มแผนที่ลงในบัญชี WinkLinks ของคุณ
* **ฝัง**แสดงวิธีการฝังแผนที่นี้เป็น[แผนที่](/developers/web-components/#content-loader)เข้าสู่เว็บไซต์ของคุณ
* **ใช้งานร่วมกับ WordPress**แสดงวิธีการใช้งานของเรา[ปลั๊กอิน WordPress](/developers/wordpress)เพื่อฝังแผนที่นี้ลงในเว็บไซต์ของคุณ

เราจะครอบคลุมตัวเลือกบางส่วนเหล่านี้ในรายละเอียดเพิ่มเติมด้านล่าง

### ฝัง

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

วิธีการฝังแผนที่ลงในไซต์ของคุณมีดังนี้:

* บรรทัดที่ 3 จะแสดงวิธีการฝังสไตล์ Wink ลงในไซต์ของคุณ
* บรรทัดที่ 6 ถึง 9 จะแสดงวิธีใช้[ตัวโหลดเนื้อหากระพริบตา](/developers/web-components/#content-loader)ส่วนประกอบเว็บและสั่งให้ดึงแบนเนอร์โฆษณามาให้กับโค้ดของคุณ
* บรรทัดที่ 11 จะแสดงวิธีการฝัง Javascript ของเราลงในไซต์ของคุณ
* บรรทัดที่ 13 แสดงวิธีการฝัง[โหลดแอพวิงค์](/developers/web-components/#application-loader)ส่วนประกอบเว็บและบอกให้ดึงค่ากำหนดการกำหนดค่าระดับหน้าของคุณ
