## เริ่มต้นการค้นหาของคุณ

นี่คือวิธีเริ่มต้นการค้นหาผู้ให้บริการและสินค้าคงคลังบน Wink

1. เลือกบัญชีพันธมิตรที่คุณต้องการทำงานด้วยจาก [รายการบัญชีของคุณ](https://studio.wink.travel/company/list)
2. ไปที่หน้าค้นหาได้สองวิธี:
    1. คุณสามารถค้นหาอย่างรวดเร็วสำหรับที่พักได้โดยตรงจากแดชบอร์ด หรือ...
    2. จากแถบนำทางหลัก คลิกที่ Inventory → Search
3. หน้าค้นหาจะแสดงผู้ให้บริการและสินค้าคงคลังล่าสุดบน Wink พร้อมหมวดหมู่ให้เลือก
4. ในตัวอย่างของเรา ให้คลิกที่การ์ดที่เขียนว่า `Hotels`
5. คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าผลลัพธ์การค้นหา ที่นี่คุณสามารถเลื่อนดูผู้ให้บริการที่มีในหมวดหมู่ `Hotels` และใช้/เปลี่ยนตัวกรองการค้นหาที่อยู่ด้านซ้ายของหน้าจอ

ผลลัพธ์การค้นหาประกอบด้วยข้อมูลรวมเกี่ยวกับผู้ให้บริการหรือสินค้าคงคลังพร้อมกับการดำเนินการที่คุณสามารถทำได้

## ปรับแต่งผลลัพธ์การค้นหา

นี่คือวิธีทั้งหมดที่คุณสามารถกรองสินค้าคงคลังและผู้ให้บริการได้ คลิกไอคอนตัวกรองที่มุมขวาบนของผลลัพธ์การค้นหา

- ชื่อที่พัก (*เช่น Centara*)
- ชื่อสินค้าคงคลัง (*เช่น Villa*)
- ประเภทสินค้าคงคลัง (*เช่น Spas*)
- คะแนนดาว (*เช่น ที่พักระดับ 5 ดาว*)
- คะแนนรีวิวผู้ใช้ (*เช่น 8 / 10*)
- ตำแหน่งทางภูมิศาสตร์ (*เช่น เลือกตำแหน่งบนแผนที่*)
- ทวีป (*เช่น อเมริกาเหนือ*)
- ประเทศ (*เช่น เยอรมนี หรือ ฝรั่งเศส*)
- เมือง (*เช่น โตเกียว*)
- ไลฟ์สไตล์ (*เช่น LGBTQ*)
- ประเภทสถานที่ (*เช่น สนามบิน*)
- ประเภทเซ็กเมนต์ (*เช่น ปานกลาง*)
- ประเภทหมวดหมู่ (*เช่น Lodge*)
- สไตล์ (*เช่น Art deco*)
- เฉพาะความสัมพันธ์โดยตรง (*เช่น แสดงสินค้าคงคลังโดยตรง*)
- เป็นมิตรกับสัตว์เลี้ยง (*เช่น แสดงสินค้าคงคลังที่เป็นมิตรกับสัตว์เลี้ยง*)
- เป็นมิตรกับสิ่งแวดล้อม (*เช่น แสดงสินค้าคงคลังที่เป็นมิตรกับสิ่งแวดล้อม*)
- เป็นมิตรกับเด็ก (*เช่น แสดงสินค้าคงคลังที่เป็นมิตรกับเด็ก*)
- เป็นที่นิยม (*เช่น แสดงสินค้าคงคลังที่มีมูลค่าสูง*)

## การดำเนินการ

การดำเนินการช่วยให้คุณโต้ตอบกับสินค้าคงคลังหรือผู้ให้บริการได้ เรามี 3 หมวดหมู่หลักที่ให้คุณทำสิ่งต่อไปนี้:

1. **รายละเอียดผู้ให้บริการ** ฉันจะรู้ข้อมูลเพิ่มเติมเกี่ยวกับผู้ให้บริการได้อย่างไร?
2. **สร้าง...** ฉันจะเปลี่ยนสินค้าคงคลังนี้ให้เป็นสิ่งที่ฉันสามารถขายได้อย่างไร?
3. **เพิ่มไปยัง...** ฉันจะบันทึกสินค้าคงคลังนี้ลงในรายการที่ฉันสามารถขายพร้อมกันได้อย่างไร?

เมื่อคุณคลิกที่ `รายละเอียดผู้ให้บริการ` คุณสามารถ:

- **จอง** ไปยังที่พักโดยตรงและจองโดยใช้รหัสพันธมิตรของคุณเพื่อรับค่าคอมมิชชั่น
- **ขอการเชื่อมต่อโดยตรง** ขอให้ผู้ให้บริการหากพวกเขาต้องการให้คุณเชื่อมต่อโดยตรง
- **ติดตามประสิทธิภาพ** สร้างแผนภูมิการวิเคราะห์เพื่อติดตามประสิทธิภาพของผู้ให้บริการตามเวลา

เมื่อคุณคลิกที่ `เพิ่มไปยัง...` เมนูจะเปิดขึ้นแสดง [รายการคัดสรร](/th/studio/curated-lists) ที่มีอยู่ของคุณและวิธีสร้างรายการใหม่ คุณสามารถ:

- **เพิ่มไปยังรายการที่มีอยู่** เพิ่มผู้ให้บริการหรือสินค้าคงคลังไปยังรายการที่คุณสร้างไว้แล้ว
- **รายการคัดสรรใหม่** ระบบจะขอให้คุณสร้างรายการใหม่และเพิ่มรายการนั้นเข้าไป

ส่วนถัดไปจะแสดงวิธีเริ่มสร้างรายได้จากผลลัพธ์การค้นหาในหลายวิธี

## ผลลัพธ์ว่างเปล่า

หากคุณกำลังมองหาผู้ให้บริการบางรายแต่ไม่พบ คุณจะเห็นช่องให้กรอกข้อมูลซึ่งช่วยให้คุณค้นหาสิ่งที่คุณต้องการบน Google Places หากมีอยู่บน Google Places คุณสามารถสร้าง `lead` บน Wink ได้
สิ่งนี้แจ้งให้เราทราบว่ามีคนต้องการให้เราติดต่อผู้ให้บริการนี้และเชิญพวกเขาเข้าร่วม Wink เราจะติดต่อผู้ให้บริการและขอให้พวกเขาเข้าร่วม คุณจะได้รับแจ้งเมื่อพวกเขาทำเช่นนั้น

## API

นักพัฒนาที่ต้องการ `เรียกดูสินค้าคงคลัง` สามารถไปที่ [Developers > API > Browse](/th/developers/apis/#browse-api)

## การอ่านเพิ่มเติม

- การจัดการ [รายการคัดสรร](/th/studio/curated-lists)
- การเปลี่ยนผลลัพธ์การค้นหาเป็น [ลิงก์ที่แชร์ได้](/th/studio/shareable-links)
- การเปลี่ยนผลลัพธ์การค้นหาเป็น [การ์ด](/th/studio/cards)
- การเปลี่ยนผลลัพธ์การค้นหาเป็น [แผนที่](/th/studio/maps)
