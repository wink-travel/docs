Studio มีวิดเจ็ตฝังหลายแบบที่คุณสามารถเพิ่มลงในเว็บไซต์ของคุณเองเพื่อกระตุ้นการจอง

ในการเข้าถึงวิดเจ็ตฝัง ให้ไปที่ `Tools > Miscellaneous` จากแถบด้านข้าง

## วิดเจ็ตที่มีให้ใช้งาน

| Widget | คำอธิบาย |
| -- | -- |
| **Account button** | ปุ่มแบรนด์ที่ลิงก์ไปยังหน้าพันธมิตรของคุณ |
| **Itinerary button** | ปุ่มที่เปิดตัวเลือกแผนการเดินทาง |
| **Itinerary form** | ฟอร์มเลือกวันที่และจำนวนผู้เข้าพักแบบฝังในหน้า |
| **Search button** | ปุ่มที่เปิดการค้นหาที่พัก |
| **Lookup button** | ปุ่มสำหรับค้นหาสินค้าคงคลังเฉพาะ |

## วิธีฝัง

แต่ละวิดเจ็ตจะมีโค้ดสั้นให้คุณคัดลอกและวางลงใน HTML ของเว็บไซต์คุณ โค้ดนี้จะรวม **Client ID** และ **Customization ID** ของคุณเพื่อให้วิดเจ็ตเชื่อมโยงกับบัญชีพันธมิตรของคุณโดยอัตโนมัติ

:::note
หากคุณใช้ WordPress เรายังมีปลั๊กอิน WordPress สำหรับการรวมที่ง่ายขึ้นด้วย
:::
