Agency คือบัญชี Wink ปกติที่คุณสามารถกำหนดให้กับบัญชีของคุณเพื่อวัตถุประสงค์ในการเป็นตัวแทน ด้านล่างนี้คือเหตุผลบางประการที่คุณอาจต้องการให้ agency เป็นตัวแทนของคุณ:

- การจัดการบัญชี Wink ของคุณแบบเต็มรูปแบบ / บางส่วนโดยบุคคลที่สาม
- Talent มีข้อตกลงที่มีอยู่กับ agency ของพวกเขาและต้องการให้ Wink ดูแลด้านการเงินของการจัดการเงินที่เข้ามาผ่าน Wink
- Talent agency ทำงานออนไลน์ทั้งหมดและจัดการบัญชีทั้งหมดบน Wink

## กำหนด agency

นี่คือวิธีการกำหนด agency ให้กับบัญชีของคุณ

<Steps>
 1. เข้าสู่ระบบบัญชีผู้ใช้ของคุณที่ [Wink Studio](https://studio.wink.travel)
 2. เลือกบัญชี affiliate ที่คุณต้องการทำงานด้วย
 3. คลิกที่ `Account > Managers` จากแถบนำทางหลัก
 4. ในส่วน `Agency` พิมพ์ชื่อของ agency ที่คุณต้องการให้เป็นตัวแทนของคุณ
 5. เลือกค่าธรรมเนียม agency จากเมนูแบบเลื่อนลง
 5. เพิ่มกฎการหมดอายุได้ตามต้องการ
 7. คลิกปุ่ม `Set agency` เพื่อดำเนินการต่อ
 </Steps>

 :::note
คุณต้องเป็นเจ้าของบัญชีเพื่อกำหนด agency
 :::

 :::tip[ตัวอย่าง]
ค่าธรรมเนียม agency มาจากค่าคอมมิชชั่นของ affiliate

การจอง = $100, ค่าคอมมิชชั่น = 10%, ค่าธรรมเนียม agency = 10% => 100 * 0.1 * 0.1 = $1
 :::

## ยกเลิกการกำหนด agency

นี่คือวิธีการยกเลิกการกำหนด agency จากบัญชีของคุณ

<Steps>
 1. เข้าสู่ระบบบัญชีผู้ใช้ของคุณที่ [Wink Studio](https://studio.wink.travel)
 2. เลือกบัญชี affiliate ที่คุณต้องการทำงานด้วย
 3. คลิกที่ `Account > Managers` จากแถบนำทางหลัก
 4. ในส่วน `Agency` คลิกที่ไอคอน <Badge text="x" variant="danger" size="small" /> ทางด้านขวาของเมนูแบบเลื่อนลงของ agency
 5. คลิกปุ่ม `Set agency` เพื่อดำเนินการต่อ
 </Steps>


## API

นักพัฒนาที่ต้องการจัดการ agency ผ่าน API สามารถไปที่ [Developers > API > Affiliate](/th/developers/apis/#affiliate-api) ได้เลย
