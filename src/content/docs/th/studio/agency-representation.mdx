เอเจนซี่คือบัญชี Wink ทั่วไปที่คุณสามารถกำหนดให้กับบัญชีของคุณเพื่อวัตถุประสงค์ในการเป็นตัวแทน ด้านล่างนี้คือเหตุผลบางประการว่าทำไมคุณจึงต้องการให้เอเจนซี่เป็นตัวแทนของคุณ:

* การจัดการบัญชี Wink ของคุณแบบเต็มรูปแบบ / บางส่วนโดยบุคคลที่สาม
* Talent มีข้อตกลงกับเอเจนซี่ของตนอยู่แล้ว และต้องการให้ Wink จัดการด้านการเงินในการจัดเตรียมเงินทุนที่เข้ามาผ่านทาง Wink
* เอเจนซี่จัดหางานทำด้านออนไลน์ทั้งหมดและจัดการบัญชีทั้งหมดบน Wink

## มอบหมายหน่วยงาน

นี่คือวิธีการกำหนดหน่วยงานให้กับบัญชีของคุณ

<Steps>
  1. เข้าสู่ระบบบัญชีผู้ใช้ของคุณบน [วิงค์สตูดิโอ](https://studio.wink.travel)-
  2. เลือกบัญชีพันธมิตรที่คุณต้องการใช้งาน
  3. คลิ๊กที่นี่ `Account > Managers` จากแถบนำทางหลัก
  4. ใน `Agency` ส่วนนี้ให้พิมพ์ชื่อหน่วยงานที่คุณต้องการให้เป็นตัวแทน
  5. เลือกค่าธรรมเนียมหน่วยงานจากรายการดรอปดาวน์
  6. เพิ่มกฎการหมดอายุได้ตามต้องการ
  7. คลิกที่ `Set agency` ปุ่มเพื่อดำเนินการต่อ
</Steps>


:::หมายเหตุ
คุณต้องเป็นเจ้าของบัญชีจึงจะกำหนดตัวแทนได้
:::

:::tip\[ตัวอย่าง]
ค่าธรรมเนียมตัวแทนจะมาจากค่าคอมมิชชั่นจากพันธมิตร

การจอง = 100 ดอลลาร์, ค่าคอมมิชชั่น = 10%, ค่าธรรมเนียมเอเจนซี่ = 10% => 100 \* 0.1 \* 0.1 = 1 ดอลลาร์
:::

## ยกเลิกการมอบหมายหน่วยงาน

นี่คือวิธียกเลิกการมอบหมายเอเจนซี่จากบัญชีของคุณ

<Steps>
  1. เข้าสู่ระบบบัญชีผู้ใช้ของคุณบน [วิงค์สตูดิโอ](https://studio.wink.travel)-
  2. เลือกบัญชีพันธมิตรที่คุณต้องการใช้งาน
  3. คลิ๊กที่นี่ `Account > Managers` จากแถบนำทางหลัก
  4. ใน `Agency` ส่วนให้คลิกปุ่ม <Badge text="x" variant="danger" size="small" /> ไอคอนทางด้านขวาของเมนูแบบดรอปดาวน์ของหน่วยงาน
  5. คลิกที่ `Set agency` ปุ่มเพื่อดำเนินการต่อ
</Steps>
