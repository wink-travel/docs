การปรับแต่งเป็นฟีเจอร์ที่ทรงพลังที่คุณสามารถใช้ควบคุมทุกแง่มุมของการรวมกับ Wink โดยไม่ต้องเขียนโค้ด ชิ้นส่วนของสินค้าคงคลังที่คุณขายสามารถปรับแต่งได้ทั้งหมด เพิ่มหรือลบฟีเจอร์ เปลี่ยนโทนสี และอื่นๆ อีกมากมาย

ขณะที่อยู่ในหน้า `Customization` ให้คลิกปุ่ม `Actions` ใต้การปรับแต่งที่คุณต้องการทำงานด้วย แล้วคลิก `Update`
*คุณจะมีรายการเริ่มต้นให้ใช้งานเมื่อสร้างบัญชีของคุณ รายการนี้ไม่สามารถลบได้*

สิ่งนี้จะพาคุณไปยังแบบฟอร์มการปรับแต่งที่คุณจะเห็นแท็บ 5 - 7 แท็บ (*ขึ้นอยู่กับประเภทบัญชีของคุณ*)

## ทั่วไป

นี่คือตัวเลือกที่มีให้คุณภายใต้แท็บ `General`

- **ชื่อ** ตั้งชื่อการปรับแต่งของคุณเพื่อให้ง่ายต่อการจดจำ *เช่น สำหรับบล็อกท่องเที่ยวของฉัน*
- **สกุลเงินที่แสดง** เลือกสกุลเงินเริ่มต้นที่ผู้ใช้จะเห็นแสดงถัดจากสินค้าคงคลังท่องเที่ยวที่จองได้ *เช่น USD*
- **ภาษาที่แสดง** เลือกภาษาที่ผู้ใช้จะพบเป็นครั้งแรก *เช่น ภาษาอังกฤษ*
- **ต่อท้ายชื่อหน้า** คุณสามารถควบคุมสิ่งที่จะถูกเพิ่มในชื่อหน้า (*ชื่อที่คุณเห็นในแท็บเบราว์เซอร์*) ซึ่งใช้หลักๆ กับเครื่องมือจองของเรา *เช่น หากคุณไปที่ https://ota.wink.travel โดยค่าเริ่มต้นคุณจะเห็น `Home | Traveliko - Hotels with Benefits`* คุณสามารถเปลี่ยนทุกอย่างหลังจาก '|' pipe ได้

## ธีม

แท็บ `Theme` ให้คุณปรับแต่งรูปลักษณ์และความรู้สึกของเว็บไซต์ [booking engine](/th/booking-engine/in-a-nutshell), เว็บไซต์ [WinkLinks](/th/link-manager/wink-links) และ [Web Components](/th/developers/web-components) ทั้งหมดของเรา

### การตั้งค่า

ส่วนนี้ให้คุณตั้งค่าการออกแบบและเลย์เอาต์ของการ์ด ปัจจุบันมีการออกแบบเพียงแบบเดียว คือ Wink

เลย์เอาต์ของการ์ดควบคุมว่าคุณต้องการให้ผู้ใช้เห็นการ์ดธุรกรรมจัดเรียงในแนวนอนหรือแนวตั้ง ข้อมูลจะเหมือนกัน

:::note
ไม่มีความแตกต่างของเลย์เอาต์การ์ดเมื่อดูการ์ดธุรกรรมของเราจากอุปกรณ์มือถือ อย่างไรก็ตาม คุณจะเห็นความแตกต่างบน iPad mini และความละเอียดที่ใหญ่กว่า
:::

:::tip[เคล็ดลับมือโปร]
ใช้เลย์เอาต์แนวนอนถ้าคุณมีสินค้าคงคลังท่องเที่ยว < 4 รายการ (เช่น ประเภทห้อง) ในรายการ... มันดูดีกว่า
:::

## แผนที่

เพื่อฝังแผนที่ของเราในเว็บไซต์ของคุณโดยใช้ [ปลั๊กอิน WordPress](/th/developers/wordpress) หรือ [Web Components](/th/developers/web-components) คุณต้องให้กุญแจ API Google Maps ของคุณเอง

:::note
  [สร้างกุญแจ Google Maps API ได้ที่นี่](https://console.developers.google.com/projectselector2/apis/credentials?supportedpurview=project)
:::

เมื่อคุณสร้างกุญแจและตั้งค่าสำหรับเว็บไซต์ของคุณแล้ว ให้กรอกในแท็บ `Analytics`

## แผนการเดินทาง

แท็บ `Itinerary` น่าจะเป็นแท็บที่น่าสนใจที่สุดในแง่ของการทำให้ราคาปรากฏสำหรับสินค้าคงคลังที่คุณเลือก
เนื่องจากการตั้งราคาสินค้าคงคลังท่องเที่ยวค่อนข้างซับซ้อน Wink จำเป็นต้องทราบแผนการเดินทางของผู้ใช้ก่อนจึงจะแสดงราคาที่ถูกต้องได้

ส่วนนี้ให้คุณตั้งค่าแผนการเดินทางเริ่มต้นเพื่อแสดงราคาสำหรับอสังหาริมทรัพย์ดิจิทัลทั้งหมดของเรา เช่น:

- ภาพที่เป็นมิตรกับโซเชียลของเราที่มีการซ้อนทับราคา
- กริด การ์ด และลิงก์ WinkLinks
- Web Components
- เครื่องมือจอง

### วันที่

คุณสามารถเลือกว่าจะตั้งค่าช่วงวันที่แบบคงที่หรือแบบเลื่อน

- เลือกช่วงวันที่แบบคงที่ถ้าคุณจะใช้การปรับแต่งสำหรับเหตุการณ์เฉพาะที่เกิดขึ้นในวันที่กำหนด
- เลือกช่วงวันที่แบบเลื่อนถ้าคุณต้องการ:
    1. ตั้งค่าเริ่มต้นเป็นวันนี้ + 1 คืน ซึ่ง OTA ส่วนใหญ่ทำ หรือ
    2. ตั้งค่าเริ่มต้นเป็นวันที่ในอนาคตที่ควรมีห้องว่างเสมอ

### การตั้งค่าห้องพัก

โดยค่าเริ่มต้น แผนการเดินทางตั้งไว้ที่ 2 ผู้ใหญ่ เนื่องจากเป็นการตั้งค่าที่พบบ่อยที่สุดสำหรับนักเดินทางส่วนใหญ่
คุณสามารถเปลี่ยนแปลงได้ง่ายและเพิ่มห้องหลายห้องพร้อมผู้เข้าพักหลายคน ทั้งผู้ใหญ่และเด็ก

### โปรโมชั่น

โปรโมชั่นเป็นฟีเจอร์ที่ช่วยประหยัดเวลาของผู้ใช้ หากที่พักให้รหัสโปรโมชั่นที่คุณสามารถแชร์กับเพื่อนๆ ได้ คุณสามารถกรอกที่นี่
ใครก็ตามที่เข้ามาผ่านลิงก์ของคุณ หรือคล้ายกัน จะได้รับรหัสโปรโมชั่นเพิ่มในแผนการเดินทางโดยอัตโนมัติ และจะเห็นราคาที่ลดแล้วแสดงโดยอัตโนมัติ

## การรวมระบบ

แท็บ `Integration` สำหรับโรงแรม ผู้รวมระบบ และตัวแทนท่องเที่ยวที่ต้องการควบคุมการกระจายการจองบน Wink มากขึ้น

### การแจ้งเตือน

ตัวเลือกนี้ให้โรงแรมและผู้รวมระบบแจ้ง Wink เพื่อแจ้งเตือนทุกฝ่ายที่เกี่ยวข้องเมื่อมีการจองเกิดขึ้น
รวมถึงการแจ้งเตือนผู้จัดการช่องทาง

โดยค่าเริ่มต้น ตัวเลือกเหล่านี้เปิดใช้งานทั้งหมด กรณีที่ผู้รวมระบบอาจต้องการปิดใช้งานหนึ่งหรือมากกว่านั้น ได้แก่:

- ตัวแทนท่องเที่ยวรวมระบบในระดับ API และต้องการส่งอีเมลยืนยันเอง
- โรงแรมกำลังทดสอบการรวมระบบและไม่ต้องการรบกวนพนักงานขาย
- โรงแรมไม่สนใจอีเมลและต้องการรับแจ้งผ่าน PMS เท่านั้น

### โลโก้อีเมล

สำหรับผู้รวมระบบที่ต้องการใช้บริการแจ้งเตือนอีเมลของเรา สามารถปรับแต่งอีเมลให้ดูเหมือนมาจากพวกเขาได้

:::note
  ขนาดที่แนะนำคือ กว้าง 300px x สูง 75px
:::

## เฉพาะโรงแรม

ภายใต้แท็บ `Hotel-specific` มีตัวเลือกให้โรงแรมควบคุมสิ่งที่แสดงบนหน้าแลนดิ้งเพจของพวกเขาบนเครื่องมือจองของเรา และสิ่งที่จะแสดงเมื่อไม่มีสินค้าคงคลังสำหรับแผนการเดินทางที่ร้องขอ

- **อันดับ**: ที่พักที่ยังไม่ได้รับอันดับที่มีความหมาย อาจเลือกปิดฟีเจอร์นี้ได้

:::note
  ที่พักไม่สามารถควบคุมวิธีที่พันธมิตรเลือกแสดงหน้าแลนดิ้งเพจของโรงแรม... นั่นเป็นการตั้งค่าที่แตกต่างกันสองแบบ
:::

- **ความพร้อมใช้งาน**: ให้ที่พักเลือกว่าจะยังคงแสดงสินค้าคงคลังแม้จะไม่มีความพร้อมใช้งานหรือไม่

:::note
  ในกรณีส่วนใหญ่ ที่พักจะต้องการให้ผู้ใช้สามารถคลิกปุ่ม `Book` เพื่อไปยังหน้าแลนดิ้งเพจของพวกเขา ที่ซึ่งผู้ใช้สามารถเปลี่ยนแผนการเดินทางและลองหาวันที่ว่างได้
  เปิดสวิตช์นี้เมื่อคุณต้องการบล็อกไม่ให้ผู้ใช้ดำเนินการต่อ
:::

## วิธีใช้

ในสถานการณ์ส่วนใหญ่ คุณจะเลือก `Customization` ที่จะใช้กับลิงก์ การ์ด แผนที่ และกริดของคุณ

นี่คือเวลาที่คุณอาจต้องการใช้การตั้งค่าของคุณด้วยตนเอง:

- **แทนที่ด้วยตนเอง** คุณต้องการแชร์ลิงก์ไปยังเครื่องมือจองของเราโดยตรง และไม่ใช้ฟีเจอร์ [Shareable Links](/th/studio/shareable-links) ของเรา โดยมีการตั้งค่าการปรับแต่งของคุณถูกนำไปใช้
- **ปลั๊กอิน WordPress** เพิ่มการตั้งค่าไปยัง [ปลั๊กอิน WordPress](/th/developers/wordpress) ของ Wink ใน `Settings`
- **การรวมระบบแบบกำหนดเอง** ใช้กับผู้รวมระบบของคุณเองด้วยการใช้ [wink-app-loader](/th/developers/web-components/#application-loader) Web Component

### แทนที่ด้วยตนเอง

นี่คือตัวอย่างง่ายๆ ของการใช้รหัสการตั้งค่าของคุณ พร้อมการตั้งค่าทั้งหมด ไปยัง URL หลักของเครื่องมือจองของเรา

<h4 align="center"> [https://ota.wink.travel?configurationId=MY-UNIQUE-ID](https://ota.wink.travel?configurationId=MY-UNIQUE-ID) </h4>

:::note
  เมื่อผู้ใช้คลิกที่ลิงก์นี้ คุณจะตั้งคุกกี้การอ้างอิงพันธมิตรของคุณด้วย
:::

### ปลั๊กอิน WordPress

ไปที่ [ปลั๊กอิน WordPress ของ Wink](/th/developers/wordpress) เพื่อเรียนรู้เพิ่มเติม

### การรวมระบบแบบกำหนดเอง

ควบคุม Web Components ของ Wink ที่ฝังอยู่ทั้งหมดบนหน้าเดียวด้วย [wink-app-loader](/th/developers/web-components/#application-loader) Web Component

ด้านล่างคือตัวอย่างที่ใช้ `MY-UNIQUE-ID` เป็นรหัสการตั้งค่า

```html {11}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="MY-UNIQUE-ID"
    ></wink-app-loader>
  </body>
</html>
```
