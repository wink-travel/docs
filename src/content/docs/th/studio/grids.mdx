กริดแสดงรายชื่อซัพพลายเออร์และสินค้าคงคลัง และแสดงรายละเอียดผ่าน UI แบบกริดที่จองได้แบบโต้ตอบได้
กริดรองรับการแสดงสินค้าคงคลังใดๆ จากรายการที่คุณคัดสรรหรือการค้นหาที่บันทึกไว้
ผู้ใช้โต้ตอบกับกริดในลักษณะเดียวกับที่โต้ตอบกับบุคคลอื่น[การ์ด](/studio/cards)พร้อมคุณสมบัติเพิ่มเติมอีกหนึ่งอย่าง:

1. แบ่งหน้าผ่านกริดโดยคลิกที่`Show more` ปุ่ม (*เมื่อมีสินค้าเพิ่มเติม*-

ด้านบนเป็นตัวอย่างกริดของเราซึ่งแสดงรายการบัตรประเภทห้อง

บทความที่เหลือนี้จะแนะนำวิธีการสร้าง ปรับแต่ง และเปิดเผยกริดแก่ผู้ใช้ของคุณ

## ประเภทกริด

มีกริดอยู่ 3 ประเภท:

1. กริดที่อิงจาก[รายการที่คัดสรร](/studio/curated-lists#list-actions)-
2. กริดที่อิงจาก[บันทึกการค้นหา](/studio/saved-searches#list-actions)-
3. กริดกริดตามตำแหน่งและเกณฑ์การเรียงลำดับ (*คือกริดอันดับ*-

## กริดรายการที่ได้รับการคัดสรร

นี่คือกริดที่ใช้สินค้าคงคลังที่คุณรวบรวมไว้ในหนึ่งในรายการที่คุณจัดทำขึ้น และแปลงรายการดังกล่าวเป็นสินค้าคงคลังการเดินทางที่สามารถจองได้ ซึ่งคุณสามารถแสดงให้ผู้ใช้ของคุณดูได้

### สร้าง

นำทางไปที่`Inventory > Curated Lists`จากแถบนำทางหลัก สำหรับตัวอย่างนี้ เราจะใช้`Favorites`รายการ
หากคุณยังไม่ได้เพิ่มอะไรลงในรายการของคุณ`Favorites`ไปที่[ค้นหา](/studio/search)เพื่อเรียนรู้วิธีการ

<figure>
  <CloudinaryImage width="350" height="394" resize="scale" quality="best" alt="สร้างกริดรายการดูแล" src="academy/create-curated-list-grid" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*สร้างแบบฟอร์มกริดรายการที่คัดสรร*</figcaption>
</figure>


<Steps>
  1. คลิกที่ `Actions` ปุ่มบน `Favorites` รายการ.
  2. คลิกที่ `Create a grid` ปุ่ม.
  3. หน้าต่างใหม่จะแสดงขึ้นเพื่อให้คุณสามารถตั้งชื่อกริดของคุณได้ *ดูด้านล่าง*-
  4. คลิก `OK` ปุ่มเพื่อดำเนินการต่อ
</Steps>


กริดของคุณถูกสร้างขึ้นแล้ว นำทางไปที่`Tools > Grids`จากแถบนำทางหลักและคลิก`Curated Grids`แท็บเพื่อดูกริดใหม่ของคุณ

### ปรับแต่ง

แบบฟอร์มนี้ช่วยให้คุณปรับแต่งกริดของคุณได้ในรูปแบบต่อไปนี้:

<Steps>
  1. ตั้งชื่อให้เพื่อจะได้จำได้ว่ากริดนี้เกี่ยวกับอะไร
  2. เลือก [การปรับแต่ง](/studio/customization) คุณต้องการสมัครเข้าระบบกริดนี้
  3. เลือกหน้าการ์ดเริ่มต้นที่คุณต้องการให้ผู้ใช้เห็นก่อน โดยค่าเริ่มต้น หน้าการ์ดจะเป็นหน้าดั้งเดิมของสินค้าคงคลังนั้น
  4. เลือกตราสัญลักษณ์ที่คุณต้องการใช้บนการ์ด ตราสัญลักษณ์ช่วยให้ผู้ใช้สามารถเปรียบเทียบสินค้าคงคลังในเมตริกรวม เช่น `eco-friendly`-
  5. เพิ่มคำหลักคั่นด้วยเครื่องหมายจุลภาคที่จะใช้โดย [โปรแกรมรวบรวมข้อมูลเว็บ](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)-
  6. เพิ่มชื่อเรื่องและคำอธิบายในภาษาที่คุณต้องการให้ผู้ใช้เห็น
  7. คลิกที่ `Save` ปุ่มเพื่อดำเนินการต่อ
</Steps>

ด้านบนแสดงภาพพร้อมการดำเนินการทั้งหมดที่มีสำหรับกริดของคุณ:

* **อัปเดต**อัปเดตการกำหนดค่ากริดของคุณ
* **เพิ่มไปยัง WinkLinks**เพิ่มกริดไปยังบัญชี WinkLinks ของคุณ
* **ฝัง**แสดงวิธีการฝังกริดนี้เป็น[กริด](/developers/web-components/#content-loader)เข้าสู่เว็บไซต์ของคุณ
* **ใช้งานร่วมกับ WordPress**แสดงวิธีการใช้งานของเรา[ปลั๊กอิน WordPress](/developers/wordpress)เพื่อฝังกริดนี้ลงในเว็บไซต์ของคุณ

เราจะครอบคลุมตัวเลือกบางส่วนเหล่านี้ในรายละเอียดเพิ่มเติมด้านล่าง

### ฝัง

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="LIST"
       id="9a212b5e-62a7-11ef-ac3f-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

วิธีการฝังกริดลงในไซต์ของคุณมีดังนี้:

* บรรทัดที่ 3 จะแสดงวิธีการฝังสไตล์ Wink ลงในไซต์ของคุณ
* บรรทัดที่ 6 ถึง 9 จะแสดงวิธีใช้[ตัวโหลดเนื้อหากระพริบตา](/developers/web-components/#content-loader)ส่วนประกอบเว็บและบอกให้ดึงตารางสำหรับโค้ดของคุณ
* บรรทัดที่ 11 จะแสดงวิธีการฝัง Javascript ของเราลงในไซต์ของคุณ
* บรรทัดที่ 13 แสดงวิธีการฝัง[โหลดแอพวิงค์](/developers/web-components/#application-loader)ส่วนประกอบเว็บและบอกให้ดึงค่ากำหนดการกำหนดค่าระดับหน้าของคุณ

## ตารางค้นหาที่บันทึกไว้

นี่คือกริดที่ใช้สินค้าคงคลังในแบบสอบถามการค้นหาที่บันทึกไว้ของคุณและแปลงผลการค้นหาเป็นสินค้าคงคลังการเดินทางที่สามารถจองได้ซึ่งคุณสามารถแสดงให้ผู้ใช้ของคุณได้

### สร้าง

นำทางไปที่`Inventory > Saved searches`จากแถบการนำทางหลัก
หากคุณยังไม่ได้สร้างการค้นหาที่บันทึกไว้ ให้ไปที่[ค้นหา](/studio/search)เพื่อเรียนรู้วิธีการ

  <CloudinaryImage width="350" height="394" resize="scale" quality="best" alt="สร้างกริดการค้นหาที่บันทึกไว้" src="academy/create-saved-search-grid" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*สร้างแบบฟอร์มกริดการค้นหาที่บันทึกไว้*</figcaption>
</figure>


<Steps>
  1. คลิกที่ `Actions` ปุ่มในการค้นหาที่คุณบันทึกไว้
  2. คลิกที่ `Create a grid` ปุ่ม.
  3. หน้าต่างใหม่จะแสดงขึ้นเพื่อให้คุณสามารถตั้งชื่อกริดของคุณได้ *ดูด้านล่าง*-
</Steps>


4. คลิก`OK`ปุ่มเพื่อดำเนินการต่อ

กริดของคุณถูกสร้างขึ้นแล้ว นำทางไปที่`Tools > Grids`จากแถบนำทางหลักและคลิก`Saved Search Grids`แท็บเพื่อดูกริดใหม่ของคุณ

## กริดอันดับ

### สร้าง

สร้างกริดอันดับโดยไปที่`Tools > Grids`และคลิกที่ปุ่ม`Ranked Grid`แท็บ คลิกที่`Create ranked grid` ปุ่ม.

### ปรับแต่ง

แบบฟอร์มนี้จะช่วยให้คุณปรับแต่งกริดอันดับของคุณได้ในรูปแบบต่อไปนี้:

<Steps>
  1. เลือกจุดหมายปลายทาง. *เช่น โตเกียว*-
  2. ตั้งชื่อให้เพื่อจะได้จำได้ว่ากริดนี้เกี่ยวกับอะไร *เช่น โรงแรมที่เป็นมิตรต่อสิ่งแวดล้อมในโตเกียว*
  3. เลือก [การปรับแต่ง](/studio/customization) คุณต้องการสมัครเข้าระบบกริดนี้
  4. เลือกหน้าการ์ดเริ่มต้นที่คุณต้องการให้ผู้ใช้เห็นก่อน โดยค่าเริ่มต้น หน้าการ์ดจะเป็นหน้าดั้งเดิมของสินค้าคงคลังนั้น
  5. เลือกคุณลักษณะที่ต้องการใช้ในการเรียงลำดับคุณสมบัติ *เช่น ความเป็นมิตรต่อสิ่งแวดล้อม*-
  6. เพิ่มคำหลักคั่นด้วยเครื่องหมายจุลภาคที่จะใช้โดย [โปรแกรมรวบรวมข้อมูลเว็บ](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)-
  7. เพิ่มชื่อเรื่องและคำอธิบายในภาษาที่คุณต้องการให้ผู้ใช้เห็น
  8. คลิกที่ `Save` ปุ่มเพื่อดำเนินการต่อ
</Steps>


หลังจากบันทึกกริดจัดอันดับของคุณแล้ว คุณจะถูกส่งต่อไปยังหน้ากริดจัดอันดับของคุณ และกริดของคุณก็พร้อมที่จะแชร์กับคนทั่วโลกแล้ว

* บรรทัดที่ 3 จะแสดงวิธีการฝังสไตล์ Wink ลงในไซต์ของคุณ
* บรรทัดที่ 6 ถึง 9 จะแสดงวิธีใช้[ตัวโหลดเนื้อหากระพริบตา](/developers/web-components/#content-loader)ส่วนประกอบเว็บและบอกให้ดึงกริดจัดอันดับสำหรับโค้ดของคุณ
* บรรทัดที่ 11 จะแสดงวิธีการฝัง Javascript ของเราลงในไซต์ของคุณ
* บรรทัดที่ 13 แสดงวิธีการฝัง[โหลดแอพวิงค์](/developers/web-components/#application-loader)ส่วนประกอบเว็บและบอกให้ดึงค่ากำหนดการกำหนดค่าระดับหน้าของคุณ
