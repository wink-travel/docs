โซลูชันของเราสำหรับตัวแทนท่องเที่ยวที่ลงทะเบียนคือ [Wink Agent](https://agent.wink.travel) ซึ่งเป็นพอร์ทัลแบบบริการตนเองที่ตัวแทนสามารถใช้ค้นหาและจองแทนลูกค้าของตน และรับค่าคอมมิชชั่นสำหรับตัวเองในแต่ละการจอง

ตัวแทนท่องเที่ยวที่มีความเชี่ยวชาญมากขึ้นสามารถเลือกผสานรวมในระดับ API เพื่อจัดการการชำระเงินและการจ่ายเงินนอกแพลตฟอร์ม Wink

ตัวแทนท่องเที่ยวมีฟีเจอร์บางส่วนของพอร์ทัลเดียวกับ [Wink Studio](/th/studio/what-is-studio) ดังนั้นเราจะไม่กล่าวถึงซ้ำที่นี่

## การค้นหา

ไปที่ [Search](/th/studio/search) เพื่อเรียนรู้วิธีการค้นหาและจองโรงแรม

หากไม่เช่นนั้น ให้ดำเนินการต่อไปยังส่วนถัดไป

## API

นักพัฒนาที่ต้องการจัดการ `Travel Agent` สามารถไปที่ [Developers > API > Travel Agent](/th/developers/apis/#travel-agent-api)
