**Wink Shopify App** ช่วยให้ผู้ประกอบการทัวร์ ผู้ขายประสบการณ์ และพ่อค้าในธุรกิจโรงแรม สามารถแสดงสินค้าคงคลัง Wink แบบสด ๆ บนหน้าร้าน Shopify ของตนได้โดยตรง — ร่วมกับสินค้า บนหน้าแลนดิ้งเพจ หรือที่ใดก็ได้ในธีม

## ติดตั้ง

<Steps>
1. ไปที่ [Shopify App Store](https://apps.shopify.com) และค้นหา **Wink Travel**
2. คลิก **Add app** และทำตามขั้นตอน OAuth เพื่ออนุญาต
3. ในการตั้งค่าแอป Wink ให้กรอก **Client ID** และ **Client Secret** ของคุณ
   ไปที่ [Applications](/th/portal/settings/applications) เพื่อดึงข้อมูลรับรองของคุณ
4. คลิก **Verify & Save** จะมีข้อความยืนยันการเชื่อมต่อสำเร็จ
</Steps>

## เปิดใช้งาน App Embed Block

App Embed Block จะฝังสคริปต์และสไตล์ชีตของ Wink อัตโนมัติในทุกหน้า — คุณไม่จำเป็นต้องแก้ไขโค้ดธีม

<Steps>
1. ใน Shopify Admin ไปที่ **Online Store → Themes → Customize**
2. คลิก **App embeds** ในแผงด้านซ้าย
3. เปิดสวิตช์ **Wink Travel**
4. คลิก **Save**
</Steps>

:::note
App Embed Block จะอ่าน Client ID ของคุณจาก Shopify metafield ที่ตั้งค่าไว้ในระหว่างการติดตั้ง ส่วน Client Secret ของคุณจะไม่ถูกเปิดเผยต่อเบราว์เซอร์
:::

## เพิ่มคอมโพเนนต์ลงในหน้า

<Steps>
1. ในตัวแก้ไขธีม เปิดหน้าหรือส่วนที่คุณต้องการแสดงเนื้อหา Wink
2. คลิก **Add block** และเลือกจากบล็อก Wink:
   - **Wink Content** — การ์ดโรงแรม ตารางจัดอันดับ หรือแผนที่
   - **Wink Lookup** — แถบค้นหาปลายทาง
   - **Wink Search** — ปุ่มเลือกแผนการเดินทาง
   - **Wink Account** — ปุ่มเข้าสู่ระบบ
   - **Wink Itinerary** — ปุ่มสถานะแผนการเดินทาง
   - **Wink Cart** — ปุ่มตะกร้าสินค้า
3. สำหรับ **Wink Content** เลือก **Layout Type** และกรอก **Layout ID** จาก Wink Studio
4. คลิก **Save**
</Steps>

## บล็อกธีมที่มีให้ใช้งาน

| บล็อก | จุดประสงค์ |
|---|---|
| **Wink Content** | แสดงการ์ดโรงแรม ตารางจัดอันดับ แผนที่ และอื่น ๆ |
| **Wink Lookup** | แถบค้นหาปลายทางและโรงแรม |
| **Wink Search** | ปุ่มไอคอนที่เปิดตัวเลือกแผนการเดินทาง |
| **Wink Account** | ปุ่มเข้าสู่ระบบ / บัญชีผู้ใช้ |
| **Wink Itinerary** | ปุ่มแผนการเดินทางที่แสดงทริปปัจจุบัน |
| **Wink Cart** | ปุ่มสรุปตะกร้าสินค้า |
