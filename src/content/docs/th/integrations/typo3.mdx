**Wink TYPO3 Extension** ผสานรวมสินค้าคงคลังการเดินทางของ Wink เข้ากับ TYPO3 13 ในรูปแบบองค์ประกอบเนื้อหาเนทีฟ บรรณาธิการสามารถเพิ่มส่วนประกอบ Wink ลงในหน้าโดยใช้ตัวแก้ไขหน้า TYPO3 มาตรฐาน — ไม่ต้องเปลี่ยนเทมเพลตใดๆ

## ความต้องการ

- TYPO3 13.x
- PHP 8.2+
- Composer

## การติดตั้ง

<Steps>
1. ติดตั้งผ่าน Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. เปิดใช้งานส่วนขยาย:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. ในส่วนหลังบ้าน TYPO3 ไปที่ **Admin Tools → Settings → Extension Configuration → wink**.
4. กรอก **Client ID**, **Client Secret** และเลือกสภาพแวดล้อมของคุณ จากนั้นบันทึก
   ไปที่ [Applications](/th/portal/settings/applications) เพื่อดึงข้อมูลรับรองของคุณ
5. เพิ่มเทมเพลต TypoScript แบบสแตติกในเทมเพลตรากของคุณ:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## การเพิ่มองค์ประกอบเนื้อหาลงในหน้า

<Steps>
1. เปิดหน้าในโมดูล **Page**.
2. คลิกปุ่ม **+** ในพื้นที่เนื้อหาเพื่อเพิ่มองค์ประกอบใหม่
3. เลือกหนึ่งในองค์ประกอบเนื้อหา **Wink Travel**:
   - **Wink Content** — การ์ดโรงแรม ตารางอันดับ แผนที่
   - **Wink Lookup** — แถบค้นหาปลายทาง
   - **Wink Search** — ปุ่มเลือกแผนการเดินทาง
   - **Wink Account** — ปุ่มเข้าสู่ระบบ
   - **Wink Itinerary** — ปุ่มสถานะแผนการเดินทาง
   - **Wink Cart** — ปุ่มตะกร้าสินค้า
4. สำหรับ **Wink Content** ใช้ฟิลด์ FlexForm เพื่อเลือก **Layout Type** และกรอก **Layout ID**
5. บันทึกและเผยแพร่หน้า
</Steps>

## องค์ประกอบเนื้อหาที่มีให้ใช้งาน

| องค์ประกอบ | จุดประสงค์ |
|---|---|
| **Wink Content** | การ์ดโรงแรม ตารางอันดับ แผนที่ และอื่นๆ |
| **Wink Lookup** | แถบค้นหาปลายทาง |
| **Wink Search** | ปุ่มไอคอนที่เปิดตัวเลือกแผนการเดินทาง |
| **Wink Account** | ปุ่มเข้าสู่ระบบ / บัญชีผู้ใช้ |
| **Wink Itinerary** | ปุ่มแผนการเดินทางที่แสดงทริปปัจจุบัน |
| **Wink Cart** | ปุ่มสรุปตะกร้าสินค้า |
