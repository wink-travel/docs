The **Wink Wix App** เพิ่มคอมโพเนนต์เว็บของ Wink ลงในไซต์ Wix ของคุณในรูปแบบวิดเจ็ตเนทีฟที่คุณสามารถลากและวางใน Wix Editor ได้โดยไม่ต้องเขียนโค้ดสำหรับเจ้าของไซต์

## ติดตั้ง

<Steps>
1. ไปที่ [Wix App Market](https://www.wix.com/app-market) และค้นหา **Wink Travel**
2. คลิก **Add to Site** และทำตามขั้นตอนเพื่ออนุญาตแอป
3. เมื่อติดตั้งเสร็จ เปิด **Dashboard → Wink Settings** และกรอก **Client ID** และ **Client Secret** ของคุณ
   ไปที่ [Applications](/th/portal/settings/applications) เพื่อดึงข้อมูลรับรองของคุณ
4. คลิก **Save** แล้วคลิก **Test Connection** เพื่อยืนยันว่าเชื่อมต่อกับ Wink เรียบร้อยแล้ว
</Steps>

## เพิ่มคอมโพเนนต์ลงในหน้า

<Steps>
1. เปิด **Wix Editor** และไปที่หน้าที่คุณต้องการเพิ่มเนื้อหา Wink
2. คลิก **Add (+) → Apps → Wink Travel** เพื่อดูวิดเจ็ตทั้งหมดที่มี
3. ลากวิดเจ็ตลงบนหน้า — เช่น **Wink Content** เพื่อแสดงกริดโรงแรม
4. ในแผงการตั้งค่าวิดเจ็ต เลือก **Layout Type** และวาง **Layout ID** จาก Wink Studio
5. คลิก **Publish** เพื่อเผยแพร่การเปลี่ยนแปลง
</Steps>

## วิดเจ็ตที่มีให้ใช้งาน

| วิดเจ็ต | จุดประสงค์ |
|---|---|
| **Wink Content** | แสดงการ์ดโรงแรม กริดอันดับ แผนที่ และอื่นๆ |
| **Wink Lookup** | แถบค้นหาสำหรับปลายทางและโรงแรม |
| **Wink Search** | ปุ่มไอคอนที่เปิดตัวเลือกแผนการเดินทาง |
| **Wink Account** | ปุ่มเข้าสู่ระบบ / บัญชีผู้ใช้ |
| **Wink Itinerary** | ปุ่มแผนการเดินทางที่แสดงทริปปัจจุบัน |
| **Wink Cart** | ปุ่มสรุปตะกร้าสินค้า |
