**Wink Joomla Extension** คือแพ็กเกจที่ติดตั้งปลั๊กอินระบบ (โหลดสคริปต์ CDN), ปลั๊กอินเนื้อหา (ประมวลผลแท็ก shortcode ในบทความ) และคอมโพเนนต์ (หน้าการตั้งค่าผู้ดูแลระบบ)

/>

## ความต้องการ

- Joomla 4 หรือ 5
- PHP 8.1+

## การติดตั้ง

<Steps>
1. ดาวน์โหลดไฟล์ `pkg_wink-x.x.x.zip` เวอร์ชันล่าสุดจาก [หน้าปล่อยเวอร์ชัน GitHub](https://github.com/wink-travel/wink-joomla-extension/releases)
2. ใน Joomla Admin ไปที่ **Extensions → Manage → Install**
3. อัปโหลดไฟล์ zip แล้วคลิก **Upload & Install**
4. ปลั๊กอินทั้งสองจะถูกเปิดใช้งานโดยอัตโนมัติ ไปที่ **Components → Wink Settings**
5. กรอก **Client ID** และ **Client Secret** ของคุณแล้วคลิก **Save**  
   ไปที่ [Applications](/th/portal/settings/applications) เพื่อดึงข้อมูลรับรองของคุณ
</Steps>

## ฝังคอมโพเนนต์ในบทความ

ใช้แท็กสไตล์ shortcode ที่ใดก็ได้ในเนื้อหาบทความ:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

ปลั๊กอินเนื้อหาจะเปลี่ยนแท็กเหล่านี้เป็น `<wink-*>` HTML ที่ถูกต้องในเวลาการแสดงผล ปลั๊กอินระบบจะโหลดสคริปต์ CDN โดยอัตโนมัติและแทรก `<wink-app-loader>` ในส่วนท้ายของหน้า

## Shortcodes ที่มีให้ใช้งาน

| Shortcode | ผลลัพธ์ | คำอธิบาย |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | การ์ดโรงแรม, ตาราง, แผนที่ |
| `{wink-lookup}` | `<wink-lookup>` | แถบค้นหาปลายทาง |
| `{wink-search}` | `<wink-search-button>` | ปุ่มเลือกแผนการเดินทาง |
| `{wink-account}` | `<wink-account-button>` | เข้าสู่ระบบ / บัญชีผู้ใช้ |
| `{wink-itinerary}` | `<wink-itinerary-button>` | ปุ่มสถานะแผนการเดินทาง |
| `{wink-cart}` | `<wink-shopping-cart-button>` | ปุ่มตะกร้าสินค้า |
