**Wink Drupal Module** เพิ่มส่วนประกอบสินค้าคงคลังการเดินทางของ Wink ลงในไซต์ Drupal ของคุณในรูปแบบบล็อกเนทีฟ บรรณาธิการสามารถวางบล็อกเหล่านี้ได้ทุกที่ในเลย์เอาต์ผ่านระบบบล็อกมาตรฐาน

## ความต้องการ

- Drupal 10 หรือ 11
- PHP 8.1+
- Composer

## การติดตั้ง

<Steps>
1. ติดตั้งผ่าน Composer:
   ```bash
   composer require drupal/wink
   ```
2. เปิดใช้งานโมดูล:
   ```bash
   drush en wink -y
   ```
3. ไปที่ **ผู้ดูแลระบบ → การตั้งค่า → บริการเว็บ → การตั้งค่า Wink** (`/admin/config/wink`)
4. กรอก **Client ID** และ **Client Secret** ของคุณ เลือกสภาพแวดล้อม แล้วคลิก **บันทึก**
   ไปที่ [Applications](/th/portal/settings/applications) เพื่อรับข้อมูลรับรองของคุณ
</Steps>

## เพิ่มบล็อกลงในหน้า

<Steps>
1. ไปที่ **ผู้ดูแลระบบ → โครงสร้าง → การจัดวางบล็อก**
2. คลิก **วางบล็อก** ในพื้นที่ที่คุณต้องการให้แสดงเนื้อหา Wink
3. ค้นหา **Wink** เพื่อหาบล็อกที่มีให้เลือก:
   - **Wink Content** — แสดงการ์ดโรงแรม ตาราง หรือแผนที่
   - **Wink Lookup** — แถบค้นหาปลายทาง
   - **Wink Search** — ปุ่มเลือกแผนการเดินทาง
   - **Wink Account** — ปุ่มเข้าสู่ระบบ
   - **Wink Itinerary** — ปุ่มสถานะแผนการเดินทาง
   - **Wink Cart** — ปุ่มตะกร้าสินค้า
4. สำหรับ **Wink Content** ให้กำหนดค่า **ประเภทเลย์เอาต์** และ **รหัสเลย์เอาต์** ในการตั้งค่าบล็อก
5. บันทึกการตั้งค่าบล็อก
</Steps>

## บล็อกที่มีให้ใช้งาน

| บล็อก | จุดประสงค์ |
|---|---|
| **Wink Content** | แสดงการ์ดโรงแรม ตารางอันดับ แผนที่ และอื่นๆ |
| **Wink Lookup** | แถบค้นหาสำหรับปลายทางและโรงแรม |
| **Wink Search** | ปุ่มไอคอนที่เปิดตัวเลือกแผนการเดินทาง |
| **Wink Account** | ปุ่มเข้าสู่ระบบ / บัญชีผู้ใช้ |
| **Wink Itinerary** | ปุ่มแผนการเดินทางที่แสดงทริปปัจจุบัน |
| **Wink Cart** | ปุ่มสรุปตะกร้าสินค้า |
