:::note
การสร้างอัตราหลักครั้งแรกของคุณเป็นส่วนหนึ่งของตัวช่วยเริ่มต้นใช้งาน
:::

:::note
`อัตราหลัก` คือคู่ที่ไม่ซ้ำกันของ `ห้องพัก / แผนราคา`

*ผู้ให้บริการอัตราการเดินทางส่วนใหญ่จะใช้เพียงห้องพักและแผนราคาในบริบทนี้ เมื่อคุณเห็นห้องพักของคุณถูกแสดงขายบน OTA หรือเครื่องมือจอง นักเดินทางจะเปรียบเทียบแผนราคา*

Wink เพิ่มคำว่า `อัตราหลัก` เพื่อให้สามารถให้ความยืดหยุ่นกับการรวมกันที่ไม่ซ้ำกันนี้ได้มากขึ้น
:::

:::note
อัตราหลักจะถูกสร้าง / ลบโดยอัตโนมัติเมื่อมีการเพิ่มหรือลบห้องพักหรือแผนราคา
:::

ในการจัดการอัตราหลักของคุณ ให้คลิกที่ `Monetize > Master Rate` จากแถบนำทางหลัก

ที่พักจะมีอัตราหลักสำหรับทุกการรวมกันของห้องพัก / แผนราคา

*ตัวอย่าง: หากที่พักของคุณมีห้องพักหนึ่งห้องและแผนราคาสองแผน ผู้เข้าพักจะเห็นอัตราหลักสองรายการที่พร้อมขาย (ถ้าผ่านเงื่อนไขทั้งหมด)*

รายการอัตราหลักมาพร้อมกับสองตัวเลือกในการจัดการ:

- สลับการเปิดใช้งานอัตราหลัก
- ตั้งค่าของแถม

## ความพร้อมใช้งาน

เมื่อคุณเปิด / ปิดใช้งานอัตราหลัก จะส่งผลต่อความพร้อมใช้งานในทุกช่องทางการขาย สวิตช์นี้เป็นวิธีที่รวดเร็วในการปิดอัตราหลักเฉพาะที่ทุกที่ทันที

## ของแถม

เมื่อผู้เดินทางกำลังมองหาการจองห้องพัก พวกเขาจะไม่โต้ตอบกับห้องพักโดยตรง แต่จะโต้ตอบกับอัตราหลักของห้องพักนั้น
อัตราหลักสามารถมีของแถมเฉพาะที่มีให้เฉพาะสำหรับการรวมกันของห้องพัก / แผนราคานั้น ๆ ซึ่งช่วยให้เจ้าของที่พักมีวิธีเพิ่มแรงจูงใจให้กับการรวมกันนั้นได้อีกทางหนึ่ง

สองสิ่งที่ควรจำเมื่อเลือกของแถม:

- ว่าของแถมนั้นรับประกันหรือขึ้นอยู่กับความพร้อมใช้งาน
- ค่าตัวเลขของมัน

ของแถมที่รับประกันจะมีค่าตัวเลขสูงกว่าเพราะแสดงถึงมูลค่าที่มากกว่าสำหรับลูกค้า
โรงแรมทุกแห่งจะได้รับ `คะแนนของแถม` ซึ่งคำนวณจากค่าตัวเลขรวมของของแถมทั้งหมดในทุกอัตราหลัก
ผู้เดินทางที่มองหาข้อเสนอสามารถจัดเรียงโรงแรมตามคะแนนของแถมที่สูงได้

ของแถมเป็นวิธีในการ:

- แยกความแตกต่างของการรวมกันห้องพัก / แผนราคา
- สร้างการมองเห็นที่มากขึ้น *เช่น คะแนนของแถมสูงกว่าระดับของโรงแรมอื่นในพื้นที่ของคุณ*

:::tip
ของแถมเกิดจากความต้องการของโรงแรมที่จะสร้างความแตกต่างและ `เล่นเกมระบบ`
ครั้งล่าสุดที่พวกเขาทำเช่นนั้น พวกเขาแทบจะทำลายตัวเองโดยการให้กำไรทั้งหมดแก่ OTA เพื่อแลกกับตำแหน่งที่ดีกว่าผ่านค่าคอมมิชชั่นที่สูงขึ้น
ในอดีต ค่าคอมมิชชั่นเป็นสิ่งเดียวที่โรงแรมใช้เล่นเกมได้

ของแถมเป็นวิธีที่โรงแรมให้รางวัลแก่ผู้เดินทางโดยที่ไม่จำเป็นต้องมีต้นทุนจริงสำหรับโรงแรม

หากคุณมีข้อเสนอแนะสำหรับของแถมประเภทอื่น ๆ ที่จะมอบให้ [ติดต่อเรา](mailto:hi@wink.travel)
:::

มีสองวิธีในการเปิดใช้งานสินค้าคงคลังสำหรับช่องทางการขาย:

- สลับการเปิดใช้งานสินค้าคงคลังบนการ์ด *ดูภาพด้านบน*
- ไปที่ `Distribution > Inventory` จากแถบนำทางหลักที่จัดการสินค้าคงคลังในทุกช่องทางการขาย

## API

นักพัฒนาที่ต้องการจัดการ `Master Rates` สามารถไปที่ [Developers > APIs > Monetize](/th/developers/apis/#monetize-api) ได้เลย
