บทความนี้แสดงวิธีการเชื่อมต่อกับพันธมิตรที่กระตือรือร้นบน Wink และจัดการพันธมิตรที่มีอยู่

## ช่องทางการขาย

ในการจัดการช่องทางการขายของคุณ ให้คลิกที่ `Distribution > Sales channels` จากแถบนำทางหลัก

ช่องทางการขายคือความสัมพันธ์พันธมิตรที่มีอยู่ของคุณ ซึ่งถูกกำหนดด้วยสินค้าคงคลัง ข้อเสนอ และส่วนลดที่พวกเขาสามารถเข้าถึงได้

ในการอัปเดตช่องทางการขายที่มีอยู่ ให้คลิกที่ลิงก์ `Actions` ใต้ช่องทางการขายที่คุณต้องการอัปเดต

- **ส่วนลดสมาชิก** อัตราสมาชิกจะมอบให้กับนักเดินทางที่จองผ่านช่องทางการขายนี้
- **ค่าคอมมิชชั่น** ค่าคอมมิชชั่นจะถูกจัดสรรให้กับช่องทางการขายสำหรับการจองที่สำเร็จแต่ละครั้ง
- **โปรโมชั่นเดี่ยว** เพิ่มโปรโมชั่นที่คุณต้องการให้ช่องทางการขายนี้มี
- **โปรโมชั่นแบบรวม** เพิ่มโปรโมชั่นแบบรวมที่คุณต้องการให้ช่องทางการขายนี้มี
- **เปิดใช้งาน** สลับสวิตช์เปิดใช้งานเพื่อเปิด/ปิดใช้งานช่องทางการขายอย่างสมบูรณ์
- **รายการดำ** การทำเครื่องหมายช่องทางการขายว่าเป็นรายการดำจะทำให้ไม่สามารถจองผ่านช่องทางการขายนี้ได้

เมื่อคุณพอใจกับการตั้งค่าแล้ว ให้คลิกปุ่ม `Save` เพื่อดำเนินการต่อ

:::tip
การทำรายการดำช่องทางการขาย จะทำให้ช่องทางการขายนั้นไม่สามารถรับอัตราใด ๆ จากคุณได้เลย แม้แต่ราคาจาก Wink Network ของคุณ
:::

## สำรวจเครือข่าย

ในการค้นหาพันธมิตรใหม่บน Wink ให้คลิกที่ `Distribution > Explore network` จากแถบนำทางหลัก

ในส่วนนี้ คุณจะเห็นคำขอที่รอดำเนินการจากพันธมิตรที่ด้านบนของหน้า และรายการพันธมิตรที่สามารถเชื่อมต่อได้ด้านล่าง

### ตอบกลับคำขอ

คุณสามารถเลือกที่จะปฏิเสธคำขอพันธมิตรได้ หากคุณต้องการดำเนินการต่อกับคำขอ คุณจะถูกนำไปยังหน้า `affiliate details` ซึ่งคุณสามารถดูข้อมูลเพิ่มเติมเกี่ยวกับพันธมิตร รวมถึงตัวชี้วัดประสิทธิภาพที่ผ่านมาใน Wink

### ค้นหาพันธมิตร

ในกริดพันธมิตร คุณสามารถค้นหาพันธมิตรตามชื่อ ประเภท และสถานที่ เมื่อคุณเจอพันธมิตรที่ถูกใจ ให้คลิกที่พันธมิตรนั้นเพื่อไปยังหน้า `affiliate details`

### เชื่อมต่อกับพันธมิตร

หากคุณเลือกที่จะสร้างความสัมพันธ์โดยตรงกับพันธมิตร คุณจะถูกขอให้กำหนดความสัมพันธ์ในลักษณะเดียวกับการจัดการช่องทางการขายข้างต้น

เมื่อคุณบันทึกช่องทางการขายใหม่แล้ว คุณจะถูกนำกลับไปยังรายการช่องทางการขายของคุณ

## API

นักพัฒนาที่ต้องการจัดการ `Network` สามารถไปที่ [Developers > APIs > Distribution](/th/developers/apis/#distribution-api) ได้เลย
