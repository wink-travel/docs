:::note
นี่คือส่วนย่อยของฟีเจอร์ [ลิงก์ที่แชร์ได้](/studio/shareable-links) ที่มีใน Wink Studio

ที่พักจะมีสินค้าคงคลังทั้งหมดของตนเองที่พร้อมใช้งานเป็นลิงก์ที่แชร์ได้โดยอัตโนมัติโดยไม่ต้องไปที่ Wink Studio เพื่อสร้างลิงก์เหล่านั้น
:::

ลิงก์ที่แชร์ได้เป็นฟีเจอร์หลักของ Wink และเกิดขึ้นจากความต้องการสื่อสารสินค้าคงคลังและราคาผ่านวิธีที่ง่ายที่สุดบนอินเทอร์เน็ต นั่นคือ ลิงก์ไฮเปอร์

<h4 align="center">[https://trvl.as/3xWCH](https://trvl.as/3xWCH)</h4>

ด้านบนคือตัวอย่างลิงก์ของเรา มันสั้นและพกพาง่าย... สั้นพอที่จะจำได้ แม้จะสั้น แต่ลิงก์นี้บรรจุข้อมูลจำนวนมากอย่าง **ทรงพลัง** เช่น:

- รหัสแอปของคุณ
- รหัสพันธมิตรของคุณ
- รูปภาพและข้อความที่จะแสดงให้ผู้ใช้เห็น
- รูปลักษณ์และความรู้สึก (*เช่น ธีม สี และอื่นๆ*)
- แผนการเดินทางที่ใช้สำหรับการค้นหาราคา (*ผู้ใหญ่ 2 คน 1 คืน*)
- สกุลเงินที่ใช้แสดงราคา
- ภาษาที่ใช้แสดงข้อความ

ส่วนที่เหลือของบทความนี้จะแนะนำวิธีจัดการลิงก์ไปยังสินค้าคงคลังของคุณ

## ปรับแต่งลิงก์

 แบบฟอร์มช่วยให้คุณปรับแต่งลิงก์ของคุณได้ดังนี้:

<Steps>
 1. โดยค่าเริ่มต้น ลิงก์จะถูกตั้งชื่อตามชื่อสินค้าคงคลังของคุณเป็นภาษาอังกฤษ คุณสามารถตั้งชื่อใดก็ได้ที่ช่วยให้คุณจำได้ว่าลิงก์นี้เกี่ยวกับอะไร
 2. โดยค่าเริ่มต้น การตั้งค่าแบรนด์หลัก / การปรับแต่งของคุณจะถูกนำไปใช้กับลิงก์ คุณสามารถเลือก [การปรับแต่ง](/studio/customization) อื่นๆ ที่คุณสร้างใน Wink Studio
 3. เพิ่มคำสำคัญ คั่นด้วยเครื่องหมายจุลภาค ซึ่งจะถูกใช้โดย [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)
 4. กรอกชื่อบัญชี X หรือ Facebook App ID ของคุณ (ถ้ามี)
 5. เพิ่มชื่อเรื่องและคำอธิบายในภาษาที่คุณต้องการให้ผู้ใช้เห็น
 6. เลือกรูปภาพหนึ่งภาพหรือมากกว่าที่จะแสดงเมื่อคุณแชร์ลิงก์บนแพลตฟอร์ม เช่น Facebook หรือ LinkedIn
 7. เลือกธีม / โอเวอร์เลย์ที่เพิ่มดีไซน์แบรนด์บนรูปภาพที่คุณเลือก *หมายเหตุ: นี่คือวิธีที่ทำให้ราคาปรากฏบนรูปภาพ*
 8. คลิกปุ่ม `Save` เพื่อดำเนินการต่อ
 </Steps>

ด้านบนแสดงภาพพร้อมการดำเนินการทั้งหมดที่มีสำหรับลิงก์ของคุณ:

- **ดูตัวอย่าง** แสดงภาพรวมอย่างรวดเร็วว่าลิงก์ถูกตั้งค่าอย่างไรและจะมีลักษณะอย่างไรกับผู้ใช้
- **อัปเดต** อัปเดตการตั้งค่าลิงก์ของคุณ
- **เพิ่มไปยัง WinkLinks** เพิ่มลิงก์ไปยังบัญชี WinkLinks ของคุณ
- **คัดลอก** คัดลอกลิงก์ไปยังคลิปบอร์ดของคุณเพื่อให้คุณแชร์ได้ง่ายทุกที่ (*รวมถึง [WinkLinks](/link-manager/wink-links)*)
- **แชร์** เปิดหน้าต่างใหม่พร้อมตัวเลือกแชร์บนโซเชียลเน็ตเวิร์กและแพลตฟอร์มส่งข้อความต่างๆ
- **รหัส QR** สร้างรหัส QR ที่คุณสามารถแชร์บนแพลตฟอร์มอย่าง IG ที่เน้นรูปภาพมากกว่า
- **ฝัง** แสดงวิธีฝังลิงก์นี้เป็น [แบนเนอร์โฆษณา](/developers/web-components/#content-loader) ลงในเว็บไซต์ของคุณ
- **ใช้กับ WordPress** แสดงวิธีใช้ [ปลั๊กอิน WordPress](/developers/wordpress) ของเราเพื่อฝังลิงก์นี้ลงในเว็บไซต์ของคุณ

เราจะอธิบายตัวเลือกเหล่านี้บางส่วนอย่างละเอียดด้านล่าง

ถ้าคุณต้องการแชร์รหัส QR แทนลิงก์ ให้คลิกปุ่ม `รหัส QR` แล้วหน้าต่างใหม่จะปรากฏพร้อมรหัส QR ของคุณ

วิธีบันทึกรหัส QR ของคุณ:

- หากคุณใช้แล็ปท็อป คลิกขวาที่ภาพแล้วเลือก `บันทึกรูปภาพ`
- หากคุณใช้โทรศัพท์มือถือ:
    - กดนิ้วค้างไว้บนภาพจนกว่าจะมีตัวเลือกให้บันทึกรูปภาพลงในแกลเลอรี
    - ถ่ายภาพหน้าจอของรหัส QR ตัดภาพให้พอดี แล้วบันทึกลงในแกลเลอรี

### ฝัง

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

นี่คือวิธีฝังลิงก์ของคุณในรูปแบบแบนเนอร์โฆษณาลงในเว็บไซต์ของคุณ:

- บรรทัดที่ 3 แสดงวิธีฝังสไตล์ของ Wink ลงในเว็บไซต์ของคุณ
- บรรทัดที่ 6 ถึง 9 แสดงวิธีใช้ [wink-content-loader](/developers/web-components/#content-loader) Web Component และบอกให้ดึงแบนเนอร์โฆษณาสำหรับรหัสของคุณ
- บรรทัดที่ 11 แสดงวิธีฝัง Javascript ของเราในเว็บไซต์ของคุณ
- บรรทัดที่ 13 แสดงวิธีฝัง [wink-app-loader](/developers/web-components/#application-loader) Web Component และบอกให้ดึงการตั้งค่าระดับหน้าของคุณ
