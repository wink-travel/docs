link manager ช่วยให้คุณบันทึกและจัดระเบียบ URL ที่คุณชอบไว้ใช้ในภายหลังได้ บุ๊กมาร์กของเบราว์เซอร์ของคุณเป็นตัวอย่างของ link manager เมื่อมีโซเชียลมีเดียเกิดขึ้น ความต้องการในการเผยแพร่และทำให้ลิงก์เข้าถึงได้สำหรับเพื่อนและผู้ชมของคุณก็เพิ่มมากขึ้น ฟีด Facebook ของคุณเป็นตัวอย่างของ link manager ที่เป็นมิตรกับโซเชียล

เราได้สร้าง [WinkLinks](https://i.trvl.as/) ให้เป็น link manager สำหรับอุตสาหกรรมการท่องเที่ยว ที่นี่เพื่อให้ข้อมูล สร้างความตื่นเต้น โฆษณา... และช่วยให้คุณจองดีลที่ยอดเยี่ยมจากโรงแรมทั่วโลกได้

## คุณสมบัติ

- WinkLinks ช่วยให้คุณเพิ่มลิงก์พันธมิตรที่มีอยู่ทั้งหมด เช่น Commission Junction และ Amazon ได้ฟรี
- คุณจะได้รับ URL สวยงาม เช่น [https://i.trvl.as/bob](https://i.trvl.as/bob) สำหรับเวลาที่คุณสามารถแชร์ได้แค่ URL เดียว
- คุณสามารถจัดระเบียบและแชร์ลิงก์ รวมถึงอัปโหลดไฟล์ได้
- โรงแรม เครือโรงแรม และแบรนด์ สามารถใช้ WinkLinks เป็นเครื่องมือจองขั้นสูง:
    - รองรับหลายสถานที่
    - รองรับหลายจุดหมายปลายทาง
    - รองรับหลายสกุลเงิน
    - รองรับหลายภาษา
- WinkLinks รองรับ `rich content` *(ดูด้านล่าง)*
- ไปจาก IG มาจองได้ในแค่ 2 คลิก 🚀

## เนื้อหาแบบ rich

:::note
ตั้งค่า `Intelligent` เป็น 🟢 ใน [Settings](/th/link-manager/settings) ของคุณเพื่อใช้ฟีเจอร์นี้
:::

ถ้าคุณแชร์ลิงก์จากเว็บไซต์ที่รองรับเนื้อหา `rich` / `oEmbed` ลิงก์ของคุณจะมีชีวิตชีวาบน WinkLinks

เว็บไซต์ที่รองรับ:

- Facebook - *โต้ตอบกับโพสต์และเล่นวิดีโอ*
- Instagram - *โต้ตอบกับโพสต์และเล่นวิดีโอ*
- YouTube - *เล่นวิดีโอ*
- X (เดิมชื่อ Twitter) - *โต้ตอบกับโพสต์*
- Spotify - *เล่นเพลย์ลิสต์โปรดของคุณ*
- Wink - *ดูที่ `Advanced`*

## ขั้นสูง

WinkLinks ยังเป็นวิธีที่ยอดเยี่ยมในการเผยแพร่สินค้าคงคลังการท่องเที่ยวที่คุณพบ ผ่าน [Wink Studio](https://studio.wink.travel) และที่คุณต้องการโฆษณาให้ผู้ชมของคุณ

ใช้ WinkLinks เพื่อขาย:

- [Shareable links](/th/studio/shareable-links) - เลือกแสดงเป็น:
    - [แบนเนอร์โฆษณา](/th/developers/integrations/web-components/#content-loader)
    - ลิงก์ปกติพร้อมอัปเดตราคาจริงแบบเรียลไทม์
- [Cards](/th/studio/cards) - การ์ดที่จองได้พร้อมราคาจริงและสถานะว่าง
- [Grids](/th/studio/grids) - ตารางที่มีการ์ดจองได้
- [Maps](/th/studio/maps) - แผนที่ที่มีเครื่องหมายซึ่งเปิดเป็นการ์ดจองได้
