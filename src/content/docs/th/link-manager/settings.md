ในการจัดการการตั้งค่า WinkLinks ของคุณ ให้ไปที่ `WinkLinks` ในแถบนำทางหลักและคลิกที่แท็บ `Settings`

นี่คือวิธีที่คุณสามารถกำหนดพฤติกรรมของ WinkLinks ของคุณด้วย `Settings`:

- **อัจฉริยะ** โดยการเปิดใช้งานความอัจฉริยะ เราจะพยายามดึงเนื้อหา `rich` ที่มีอยู่จากเว็บไซต์ต่างๆ เช่น `Wink`, `Spotify`, `Instagram`, `Facebook`, `YouTube`, `X (formerly Twitter)` และอื่นๆ *เช่น หากคุณแชร์โพสต์ IG บน WinkLinks โพสต์นั้นจะแสดงผลเหมือนที่คุณเห็นบน IG พร้อมยอดไลค์ ความคิดเห็น และวิธีการโต้ตอบกับโพสต์*
- **การแสดงผลเริ่มต้น** ควบคุมวิธีที่ผู้ชมของคุณโต้ตอบกับไซต์ของคุณครั้งแรกและเห็นเนื้อหาของคุณ มีตัวเลือกสามแบบให้คุณเลือก:
    - การออกแบบแบบ `Column-based` นี่คือการออกแบบที่เราแนะนำ และถ้าคุณไม่แสดงเนื้อหามากเกินไป มันจะดูดีบนหน้าของคุณ อย่างไรก็ตาม อาจทำให้หน้า `หนัก` หากคุณแชร์เนื้อหา rich จำนวนมาก
    - การออกแบบแบบ `Row-based` นี่เป็นวิธีหนึ่งในการทำให้เนื้อหาของคุณเบาลงสำหรับเบราว์เซอร์ แต่เราแนะนำให้เนื้อหาของคุณมีปริมาณข้อความที่ใกล้เคียงกันเพื่อให้การ์ดจัดเรียงได้ดีขึ้น
    - การออกแบบแบบ `List-based` ผู้จัดการลิงก์ส่วนใหญ่จะแสดงรายการง่ายๆ พร้อม URL และเนื้อหาขั้นต่ำ เลือกวิธีนี้หากคุณมีเนื้อหาจำนวนมากและต้องการแสดงผลอย่างรวดเร็วที่สุด
- **รูปโปรไฟล์** ที่ด้านบนของทุกหน้า WinkLinks คือรูปโปรไฟล์บัญชีของคุณพร้อมคำอธิบายบัญชี คุณสามารถแสดงรูปโปรไฟล์ของคุณได้สองแบบ:
    - `วงกลม` เราแนะนำให้ใช้สไตล์วงกลมหากรูปของคุณเป็นรูปตัวเองหรือสิ่งอื่นที่เหมาะกับภาชนะทรงกลม
    - `สี่เหลี่ยมผืนผ้า` เราแนะนำให้ใช้สไตล์สี่เหลี่ยมผืนผ้าหากโลโก้บริษัทของคุณเป็นรูปโปรไฟล์ เนื่องจากโลโก้บริษัทส่วนใหญ่มีลักษณะเป็นสี่เหลี่ยมผืนผ้า
- **การปรับแต่ง** เลือกการปรับแต่งที่คุณต้องการใช้สำหรับหน้า WinkLinks ของคุณ ซึ่งจะส่งผลต่อสีธีมของคุณ รวมถึงแผนการเดินทางเมื่อแสดงการ์ดสินค้าคงคลัง Wink แผนที่ หรือกริด
