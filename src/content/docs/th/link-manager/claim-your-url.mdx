โดยการสร้าง [บัญชีพันธมิตร](/th/studio/what-is-studio) คุณจะมี URL แบบ vanity ของ WinkLinks อยู่แล้ว ฟีเจอร์ผู้เผยแพร่ WinkLinks มีอยู่ในแอป Studio ของเรา พร้อมกับฟีเจอร์พันธมิตรอื่นๆ

### ตัวอย่าง

ถ้าชื่อบัญชีของคุณคือ Queen Bee URL แบบ vanity ของ WinkLinks จะเป็น:

<h4 align="center">https://i.trvl.as/queen-bee</h4>

ชื่อบัญชีจะถูกจัดสรรตามลำดับ `มาก่อนได้ก่อน` — รีบคว้า URL เฉพาะตัวของคุณก่อนใคร!
