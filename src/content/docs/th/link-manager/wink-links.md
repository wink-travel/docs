link manager ช่วยให้คุณบันทึกและจัดระเบียบ URL ที่คุณชอบไว้ใช้ในภายหลัง ตัวอย่างของ link manager คือบุ๊กมาร์กในเบราว์เซอร์ของคุณ ด้วยการมาของโซเชียลมีเดีย ทำให้เกิดความต้องการมากขึ้นในการเผยแพร่และทำให้ลิงก์พร้อมใช้งานสำหรับเพื่อนและผู้ชมของคุณ ฟีด Facebook ของคุณเป็นตัวอย่างของ link manager ที่เป็นมิตรกับโซเชียล

:::caution
link manager ส่วนใหญ่ต้องสมัครสมาชิกสำหรับผู้ที่โพสต์ลิงก์พันธมิตร
:::

เราสร้าง [WinkLinks](https://i.trvl.as/) ; link manager ที่ฟรีทั้งหมด มุ่งเน้นในด้านการท่องเที่ยว เพื่อกระตุ้น โฆษณา และขายอย่างมีประสิทธิภาพออนไลน์

## คุณสมบัติ

- WinkLinks ช่วยให้คุณเพิ่มลิงก์พันธมิตรที่มีอยู่ทั้งหมด เช่น Commission Junction และ Amazon ได้ฟรี
- คุณจะได้รับ URL สวยงาม เช่น [https://i.trvl.as/bob](https://i.trvl.as/bob) สำหรับเวลาที่คุณแชร์ได้แค่ URL เดียว
- คุณสามารถจัดระเบียบและแชร์ลิงก์ รวมถึงอัปโหลดไฟล์ได้
- โรงแรม เครือข่าย และแบรนด์ สามารถใช้ WinkLinks เป็นเครื่องมือจองขั้นสูง:
    - รองรับหลายสถานที่
    - รองรับหลายจุดหมายปลายทาง
    - รองรับหลายสกุลเงิน
    - รองรับหลายภาษา
- WinkLinks รองรับ `rich content` *(ดูด้านล่าง)*
- ไปจาก IG มาจองได้ในแค่ 2 คลิก 🚀

## เนื้อหาแบบ rich

:::note
ตั้งค่า `Intelligent` เป็น 🟢 ใน [Settings](/th/link-manager/settings) ของคุณเพื่อใช้ฟีเจอร์นี้
:::

ถ้าคุณแชร์ลิงก์จากเว็บไซต์ที่รองรับเนื้อหา `rich` / `oEmbed` ลิงก์ของคุณจะมีชีวิตชีวาบน WinkLinks

เว็บไซต์ที่รองรับ:

- Facebook - *โต้ตอบกับโพสต์และเล่นวิดีโอ*
- Instagram - *โต้ตอบกับโพสต์และเล่นวิดีโอ*
- YouTube - *เล่นวิดีโอ*
- X (เดิมชื่อ Twitter) - *โต้ตอบกับโพสต์*
- Spotify - *เล่นเพลย์ลิสต์โปรดของคุณ*
- Wink - *ดูที่ `Advanced`*

## ขั้นสูง

WinkLinks ยังเป็นวิธีที่ยอดเยี่ยมในการเผยแพร่สินค้าคงคลังการท่องเที่ยวที่คุณพบ ผ่าน [Wink Studio](https://studio.wink.travel) และที่คุณต้องการขายให้กับผู้ชมของคุณ

ใช้ WinkLinks เพื่อขาย:

- [Shareable links](/th/studio/shareable-links) - เลือกแสดงเป็น:
    - [แบนเนอร์โฆษณา](/th/developers/integrations/web-components/#content-loader)
    - ลิงก์ปกติพร้อมอัปเดตราคาจริงแบบเรียลไทม์
- [Cards](/th/studio/cards) - การ์ดที่จองได้พร้อมราคาจริงและสถานะว่าง
- [Grids](/th/studio/grids) - ตารางที่มีการ์ดจองได้
- [Maps](/th/studio/maps) - แผนที่พร้อมเครื่องหมายที่เปิดเป็นการ์ดจองได้

:::note
เพื่อจัดการบัญชี WinkLinks ของคุณ เข้าสู่ระบบบัญชีพันธมิตรของคุณที่ [Wink Studio](https://studio.wink.travel)
:::

:::tip
หากต้องการดูบัญชีที่ใช้งานจริง ลองดูบัญชี WinkLinks ของ Bjorn ผู้ร่วมก่อตั้งของเรา: [https://i.trvl.as/bjorn](https://i.trvl.as/bjorn)
:::
