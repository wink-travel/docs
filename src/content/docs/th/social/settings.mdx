# การตั้งค่า

เมื่อคุณเริ่มต้นใช้งาน Social เราจะถามคุณว่าต้องการให้เราสร้างเนื้อหาอย่างไร ตัวเลือกเหล่านี้พร้อมใช้งานตลอดเวลาเพื่อให้คุณสามารถเปลี่ยนแปลงได้อย่างง่ายดายในภายหลัง

ไปที่ `Preferences` จากแถบด้านข้างเพื่อจัดการการตั้งค่าของคุณ

ส่วน Preferences ประกอบด้วยการตั้งค่าพื้นฐานสำหรับการสร้างเนื้อหาของคุณ:

1. **Preferred customization** เลือกการปรับแต่งที่ต้องการใช้เมื่อสร้างเนื้อหา การปรับแต่งควบคุมสีธีม แผนการเดินทางเริ่มต้น สกุลเงิน ภาษา และอื่นๆ คุณสามารถสร้างการปรับแต่งได้ภายใต้บัญชีของคุณที่ [my.wink.travel](https://my.wink.travel)
2. **Auto-approve** เมื่อเปิดใช้งาน โพสต์ที่สร้างขึ้นจะถูกเผยแพร่อัตโนมัติโดยไม่ต้องรอการอนุมัติของคุณ เปิดใช้งานนี้เมื่อคุณมั่นใจในคุณภาพของเนื้อหาที่เราสร้างให้คุณ
3. **Approver name** ชื่อของบุคคลหรือทีมที่จะได้รับอีเมลคำขออนุมัติ
4. **Approver e-mail** ที่อยู่อีเมลที่คำขออนุมัติสำหรับโพสต์ที่จะมาถึงจะถูกส่งไป

Social มาพร้อมกับผู้สร้างเนื้อหาหลายแบบให้เลือก แต่ละผู้สร้างเน้นวิธีเฉพาะในการสร้างเนื้อหาที่ไม่ซ้ำใครสำหรับที่พักของคุณโดยใช้ AI ผู้สร้างเนื้อหาทุกคนมีคุณสมบัติ **Weight** — ค่าน้ำหนักที่สูงกว่าจะเพิ่มโอกาสที่ผู้สร้างคนนั้นจะถูกเลือกเมื่อสร้างโพสต์

นี่คือผู้สร้างเนื้อหาทั้งหมดที่มีให้:

| Creator | คำอธิบาย |
| -- | -- |
| **Destination** | ใช้ตำแหน่งของคุณเพื่อสร้างข้อความที่เป็นประโยชน์เกี่ยวกับพื้นที่ |
| **Room type** | สร้างโพสต์ที่จองได้โดยมีข้อความที่สร้างด้วย AI และภาพของประเภทห้องพักของคุณ |
| **Activity** | สร้างโพสต์ที่จองได้โดยมีหนึ่งในกิจกรรมของคุณ |
| **Add-On** | สร้างโพสต์ที่จองได้โดยมีหนึ่งในบริการเสริมของคุณ |
| **Attraction** | สร้างโพสต์ที่จองได้โดยมีหนึ่งในสถานที่ท่องเที่ยวของคุณ |
| **Place** | สร้างโพสต์ที่จองได้โดยมีหนึ่งในสถานที่ของคุณ |
| **Meeting room** | สร้างโพสต์ที่จองได้โดยมีหนึ่งในห้องประชุมของคุณ |
| **Restaurant** | สร้างโพสต์ที่จองได้โดยมีหนึ่งในร้านอาหารของคุณ |
| **Spa** | สร้างโพสต์ที่จองได้โดยมีหนึ่งในสปาของคุณ |
| **Property** | สร้างโพสต์ที่จองได้โดยมีห้องพักราคาดีที่สุดของคุณพร้อมข้อความที่สร้างด้วย AI และภาพของที่พัก |
| **User sentiment** | รวบรวมรีวิวจากแขกทั่วเว็บเป็นโพสต์ที่ขับเคลื่อนด้วยความรู้สึกที่เน้นสิ่งที่แขกชื่นชอบเกี่ยวกับที่พักของคุณ |
| **Funny** | สร้างโพสต์ที่เบาสมองและสนุกสนานเพื่อให้ฟีดของคุณน่าสนใจ |

:::tip[เคล็ดลับมือโปร]
ตรวจสอบให้แน่ใจว่าคุณมีประเภทสินค้าคงคลังที่จำเป็นสำหรับผู้สร้างเนื้อหาที่คุณเลือก เช่น หากคุณเลือก "Restaurant" คุณต้องมีร้านอาหารอย่างน้อยหนึ่งแห่งในที่พักของคุณ
:::

## Tone of voice

<figure>
<CldImage

โทนเสียงกำหนดสไตล์ภาษาที่ AI ใช้เมื่อสร้างเนื้อหา เลือกโทนเสียงที่ตรงกับบุคลิกแบรนด์ของคุณ เช่นเดียวกับผู้สร้างเนื้อหาแต่ละโทนมี **Weight** เพื่อควบคุมความถี่ในการใช้งาน

นี่คือโทนเสียงที่มีให้เลือก:

| Tone | สไตล์ |
| -- | -- |
| **Neutral** | สุขุม น่าเชื่อถือ และเป็นข้อเท็จจริง ไม่มีการโอ้อวดหรือเล่นมุก — เพียงความรู้ที่น่าเชื่อถือและมีโครงสร้างดี |
| **Friendly** | อบอุ่น เข้าถึงง่าย มีพลังบวกผ่านข้อความ ใช้คำย่อแบบไม่เป็นทางการและอีโมจิเป็นครั้งคราว |
| **Luxury** | หรูหรา และประณีต ใช้คำเช่น "exquisite" และ "bespoke" ที่ทำให้ผู้อ่านรู้สึกพิเศษ |
| **Foodie** | บรรยายอย่างเร้าใจและชวนหิว ใช้ภาษาที่ทำให้ผู้อ่านรู้สึกอยากกิน |
| **Adventurous** | กล้าหาญและเต็มไปด้วยพลังนักสำรวจ บรรยายประสาทสัมผัสอย่างชัดเจนทำให้ทุกอย่างดูยิ่งใหญ่ |
| **Family** | อบอุ่นและเหมาะสำหรับเด็ก มีพลังแบบพ่อแม่เกี่ยวกับประสบการณ์ที่ปลอดภัยและสนุกสนาน |
| **Romantic** | มีความเป็นกวี อ่อนหวาน และทำให้ใจเต้นแรง พระอาทิตย์ตก ภาษาอ่อนโยน และความรัก |
| **Humorous** | เสียดสี ตลกแบบรวดเร็ว และเล่นคำอย่างชาญฉลาด อ่านเหมือนการแสดงสแตนด์อัพ |
| **Eco** | จริงใจและถูกต้องทางศีลธรรม ชื่นชมทางเลือกสีเขียวและความยั่งยืน |
| **Local insider** | อ้างอิงท้องถิ่นเฉพาะเจาะจง เตือนเกี่ยวกับกับดักนักท่องเที่ยว ใช้สแลงท้องถิ่น |
| **Inclusive** | ต้อนรับและเฉลิมฉลอง ทำให้ทุกคนรู้สึกได้รับการยอมรับและมีคุณค่า |
| **Mystical** | ลึกลับ จิตวิญญาณ และความจริงในจักรวาล เชิญชวนให้มองเข้าไปข้างในอย่างอ่อนโยน |
| **Futuristic** | ทันสมัยและมองไปข้างหน้า ชอบคำเช่น "paradigm shift" และวิสัยทัศน์อนาคตที่มองโลกในแง่ดี |
| **Budget** | ประหยัดและมีไหวพริบ ภูมิใจในความประหยัด ชื่นชมการค้นหาที่คุ้มค่าและประสบการณ์ฟรี |

## Post types

ประเภทโพสต์กำหนดเป้าหมายและโครงสร้างของเนื้อหาที่สร้างขึ้น แต่ละประเภทโพสต์มี **Weight** เพื่อควบคุมความถี่ในการใช้งาน

| Post type | คำอธิบาย |
| -- | -- |
| **Transactional** | ชัดเจน กระชับ เน้นการแปลงยอด ขับเคลื่อนการจองและการดำเนินการโดยตรงด้วยข้อเท็จจริง ราคา และขั้นตอนถัดไป |
| **Awareness** | เนื้อหาสร้างแบรนด์ที่ออกแบบมาเพื่อแนะนำที่พักของคุณและเพิ่มการมองเห็นในกลุ่มแขกเป้าหมาย |
| **Sentiment** | เนื้อหาที่สร้างความรู้สึกเชิงบวกและสร้างความสัมพันธ์ทางอารมณ์กับผู้ชมของคุณ |
