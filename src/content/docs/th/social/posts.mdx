ตัวแก้ไขโพสต์ช่วยให้คุณสร้างโพสต์ด้วยตนเองได้ไม่จำกัด และยังเป็นที่ที่คุณจัดการโพสต์ที่ถูกสร้างให้อัตโนมัติด้วย

ตัวแก้ไขมี 4 ส่วน:

1. ส่วนเนื้อหา
2. ส่วนสื่อ
3. ส่วนลิงก์
4. ส่วนการตั้งค่า

## เนื้อหา

ส่วนเนื้อหาช่วยให้คุณแก้ไขข้อความก่อนที่จะเผยแพร่ คุณสามารถสร้างเนื้อหาที่แปลเป็นภาษาต่างๆ ได้ — แต่ละภาษาจะมีชื่อเรื่องและคำอธิบายของตัวเอง

## สื่อ

ส่วนสื่อช่วยให้คุณแก้ไขรูปภาพและวิดีโอที่แนบมากับข้อความ

## ลิงก์

ส่วนลิงก์ช่วยให้คุณเชื่อมโยงไปยังที่ที่คุณต้องการให้ผู้ใช้ไปหลังจากอ่านโพสต์ *เช่น อาจเป็นลิงก์ไปยังข้อเสนอพิเศษหรือสถานที่สำหรับข้อมูลเพิ่มเติม* คุณสามารถเพิ่ม URL ลิงก์ คำบรรยายลิงก์ และคำอธิบายลิงก์ได้

## การตั้งค่า

ส่วนการตั้งค่าช่วยให้คุณแก้ไข/แทนที่การตั้งค่าเริ่มต้นทั้งหมดสำหรับแต่ละโพสต์ คุณสามารถตั้งค่าได้ดังนี้:

- **แพลตฟอร์ม** — เลือกเครือข่ายโซเชียลที่โพสต์นี้จะถูกเผยแพร่
- **ประเภทโพสต์** — เลือกได้ระหว่าง Transactional, Awareness หรือ Sentiment
- **โทนเสียง** — แทนที่โทนเสียงสำหรับโพสต์นี้โดยเฉพาะ
- **การปรับแต่ง** — ใช้การปรับแต่งที่แตกต่างสำหรับสกุลเงินแสดงผล ภาษา แผนการเดินทาง ฯลฯ
- **เขตเวลา** — แทนที่เขตเวลาสำหรับการเผยแพร่
- **วันที่กำหนดเวลา** — โพสต์ที่สร้างขึ้นจะถูกกำหนดเวลาโดยอัตโนมัติในเวลาที่เหมาะสม คุณสามารถแทนที่วันที่และเวลาได้หากต้องการช่วงเวลาที่ต่างออกไป

## วงจรสถานะโพสต์

โพสต์แต่ละโพสต์จะมีวงจรสถานะดังนี้:

| สถานะ | คำอธิบาย |
| -- | -- |
| **ร่าง** | โพสต์ถูกสร้างขึ้นแต่ยังไม่พร้อมสำหรับการตรวจสอบ |
| **รอการอนุมัติ** | โพสต์รอการอนุมัติจากมนุษย์ก่อนกำหนดเวลาเผยแพร่ |
| **อนุมัติแล้ว** | โพสต์ได้รับการอนุมัติและจะถูกกำหนดเวลาเผยแพร่ |
| **ถูกปฏิเสธ** | โพสต์ถูกตรวจสอบและปฏิเสธ — จะไม่ถูกเผยแพร่ |
| **กำหนดเวลาแล้ว** | โพสต์ถูกจัดคิวสำหรับเผยแพร่ในวันที่และเวลาที่กำหนด |
| **เผยแพร่แล้ว** | โพสต์ถูกเผยแพร่สำเร็จไปยังเครือข่ายโซเชียล |
| **หยุดชั่วคราว** | การเผยแพร่โพสต์ถูกหยุดชั่วคราว |
| **ล้มเหลว** | โพสต์ไม่สามารถเผยแพร่ได้เนื่องจากเกิดข้อผิดพลาด |

## การอนุมัติโพสต์

โพสต์จะได้รับการอนุมัติในตารางโพสต์ที่คุณสามารถเลือกโพสต์หนึ่งหรือหลายโพสต์แล้วเลือกอนุมัติหรือปฏิเสธ

:::note
จะมีรายการหนึ่งรายการสำหรับทุกภาษาและทุกเครือข่ายโซเชียล

เช่น หากคุณมี 3 เครือข่ายโซเชียลและเลือก 3 ภาษา คุณจะเห็นรายการ 9 รายการในปฏิทิน
:::
