แท็บ **ความปลอดภัย** ที่ [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) คือที่ที่คุณจัดการ **รหัสผ่านแบบไม่ใช้รหัส** — ข้อมูลรับรอง WebAuthn ที่ช่วยให้คุณเข้าสู่ระบบโดยไม่ต้องใช้รหัสผ่าน

## รหัสผ่านแบบไม่ใช้รหัสคืออะไร?

รหัสผ่านแบบไม่ใช้รหัสคือข้อมูลรับรองเข้ารหัสที่ผูกกับอุปกรณ์ของคุณ (หรือผู้จัดการรหัสผ่าน) ซึ่งพิสูจน์ว่าคุณคือคุณ รหัสผ่านแบบไม่ใช้รหัสมีความต้านทานต่อการฟิชชิ่งและรวดเร็วกว่าการพิมพ์รหัสผ่าน

## การลงทะเบียนรหัสผ่านแบบไม่ใช้รหัส

1. เข้าสู่ระบบที่ [iam.wink.travel](https://iam.wink.travel) ด้วยข้อมูลรับรองที่มีอยู่ของคุณ
2. ไปที่ **บัญชีของฉัน → ความปลอดภัย**
3. คลิก `เพิ่มรหัสผ่านแบบไม่ใช้รหัส`
4. เบราว์เซอร์ของคุณจะขอให้คุณยืนยันตัวตนด้วยอุปกรณ์ของคุณ (Touch ID, Face ID, Windows Hello, กุญแจฮาร์ดแวร์ หรือผู้จัดการรหัสผ่านที่ซิงค์ไว้)
5. ตั้งชื่อรหัสผ่านแบบไม่ใช้รหัสให้จำง่าย (เช่น *Work MacBook*) เพื่อให้คุณจดจำได้ในภายหลัง

## การใช้รหัสผ่านแบบไม่ใช้รหัสเพื่อเข้าสู่ระบบ

ที่หน้าจอเข้าสู่ระบบของ [iam.wink.travel](https://iam.wink.travel) ให้กรอกอีเมลของคุณ — หากคุณมีรหัสผ่านแบบไม่ใช้รหัสที่ลงทะเบียนไว้สำหรับอีเมลนั้น คุณจะได้รับแจ้งให้ยืนยันตัวตนด้วยรหัสผ่านแบบไม่ใช้รหัสแทนการพิมพ์รหัสผ่าน

## การลบรหัสผ่านแบบไม่ใช้รหัส

ในแท็บความปลอดภัยเดียวกัน รหัสผ่านแบบไม่ใช้รหัสแต่ละรายการที่ลงทะเบียนไว้จะมีปุ่ม `ลบ` ใช้ปุ่มนี้หากคุณทำอุปกรณ์หาย เลิกใช้กุญแจฮาร์ดแวร์ หรืออยากล้างข้อมูลรับรองที่ไม่ได้ใช้งาน

:::caution
การลบรหัสผ่านแบบไม่ใช้รหัสทั้งหมดจะทำให้รหัสผ่านเป็นวิธีเดียวในการเข้าสู่ระบบ โปรดตรวจสอบให้แน่ใจว่าคุณจำรหัสผ่านได้ก่อนลบรหัสผ่านแบบไม่ใช้รหัสตัวสุดท้าย
:::
