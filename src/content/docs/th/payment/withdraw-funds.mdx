:::note
เว้นแต่คุณจะเป็นผู้รวมระบบ คุณไม่จำเป็นต้องกังวลเกี่ยวกับการสร้างบัญชีบน TripPay ผู้รวมระบบจะดำเนินการให้คุณในขณะที่คุณสร้างบัญชีกับพวกเขา
:::

ผู้รับเงินใช้ TripPay เพื่อ:

- ติดตามจำนวนเงินที่พวกเขาได้รับ
- จำนวนเงินที่มีอยู่ในบัญชีของพวกเขาในขณะนี้
- ...และจำนวนเงินที่สามารถถอนได้

## การใช้งาน

ผู้รับเงินทั่วไปมักจะโต้ตอบกับ TripPay ดังนี้:

<Steps>
1. TripPay เก็บเงินจากการจอง 10 รายการ จาก 3 แพลตฟอร์มการเดินทางที่แตกต่างกัน สำหรับโรงแรม X และเงินเหล่านั้นพร้อมให้ใช้แล้ว
2. โรงแรม X ได้รับอีเมลจาก TripPay แจ้งว่า `คุณมีเงินพร้อมใช้` พร้อมลิงก์เพื่อจัดการบัญชีของพวกเขา
3. โรงแรม X เข้าสู่ระบบที่ [TripPay](https://pay.wink.travel)
4. โรงแรม X เห็นการจองของเดือนที่แล้วพร้อมกับการวิเคราะห์บัญชี
5. โรงแรม X เลือกถอนเงิน
6. โรงแรม X เลือกวิธี `จ่ายเงิน`
7. โรงแรม X ทำตามขั้นตอนที่จำเป็นเพื่อดำเนินการเงิน
8. โรงแรม X ส่งคำขอโอนเงินและคลิก `ตกลง`
</Steps>

*เงินมักจะพร้อมใช้ภายใน 24 ชั่วโมงในทุกภูมิภาค*

:::note
เมื่อคุณถึงปริมาณและจำนวนเงินที่กำหนด เราอาจติดต่อคุณเพื่อขอเอกสารเพิ่มเติมที่ยืนยันตัวตนของคุณและ/หรือบริษัทของคุณ
:::
