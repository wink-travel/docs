:::หมายเหตุ
คุณไม่จำเป็นต้องกังวลเรื่องการสร้างบัญชีบน TripPay เว้นแต่คุณจะเป็นผู้บูรณาการ ผู้บูรณาการจะดำเนินการดังกล่าวให้กับคุณในเวลาที่คุณสร้างบัญชีกับพวกเขา
:::

ผู้รับเงินใช้ TripPay เพื่อ:

* ติดตามจำนวนเงินที่พวกเขาทำ
* ขณะนี้มีเงินอยู่ในบัญชีเท่าไร
* ...และมียอดเงินที่สามารถถอนได้เท่าไร

## การใช้งาน

ผู้รับเงินปกติมักจะโต้ตอบกับ TripPay ดังต่อไปนี้:

<Steps>
  1. TripPay เก็บเงินจากการจอง 10 รายการจากแพลตฟอร์มการท่องเที่ยว 3 แห่งที่แตกต่างกันสำหรับโรงแรม X และตอนนี้เงินทุนก็พร้อมให้ใช้แล้ว
  2. โรงแรม X ได้รับอีเมลจาก TripPay ว่า `You have funds available` และมีลิงค์สำหรับจัดการบัญชีของตนเอง
  3. โรงแรม X เข้าสู่ระบบ [ทริปเพย์](https://pay.wink.travel)-
  4. โรงแรม X ดูการจองห้องพักของเดือนที่แล้วพร้อมกับการวิเคราะห์บัญชี
  5. โรงแรม X เลือกที่จะถอนเงิน
  6. โรงแรม X เลือก `Pay-out` วิธี.
  7. โรงแรม X ปฏิบัติตามขั้นตอนที่จำเป็นเพื่อดำเนินการรับเงินทุน
  8. โรงแรม X ส่งคำขอโอนและคลิก `OK`-
</Steps>


*โดยปกติเงินทุนจะพร้อมใช้งานภายใน 24 ชั่วโมงในทุกภูมิภาค*
