ในการจัดการบัญชีธนาคารของคุณ ให้คลิกที่ `Account > Bank Accounts` จากแถบด้านข้าง

## การเพิ่มบัญชีธนาคาร

คลิก `Add bank account` เพื่อสมัครบัญชีใหม่ แบบฟอร์มจะปรับตามประเทศที่คุณเลือก:

- **ประเทศที่ชำระเงิน** — ประเทศที่ธนาคารของคุณตั้งอยู่
- **สกุลเงิน** — สกุลเงินที่คุณต้องการรับการจ่ายเงิน
- **ชื่อเจ้าของบัญชี** — ชื่อบนบัญชีธนาคาร
- **รายละเอียดเส้นทาง** — ช่องกรอกข้อมูลเฉพาะประเทศ เช่น Routing number (สหรัฐฯ), Sort code (สหราชอาณาจักร), รหัส SWIFT/BIC หรือ IBAN
- **หมายเลขบัญชี** — หมายเลขบัญชีธนาคารของคุณ

## บัญชีหลัก

คุณสามารถกำหนดบัญชีธนาคารหนึ่งบัญชีเป็น **บัญชีหลัก** บัญชีหลักของคุณจะได้รับการจ่ายเงินอัตโนมัติทั้งหมด บัญชีเพิ่มเติมสามารถใช้สำหรับการถอนเงินด้วยตนเอง

หากต้องการเปลี่ยนบัญชีหลัก ให้คลิก `Make primary` บนบัญชีใดก็ได้ในรายการของคุณ

## การจัดการบัญชี

จากรายการบัญชีธนาคาร คุณสามารถ:

- แก้ไขรายละเอียดบัญชี
- ลบบัญชีที่คุณไม่ต้องการใช้งานแล้ว
- สลับบัญชีที่เป็นบัญชีหลัก
