หากคุณเป็นบริษัทท่องเที่ยวและต้องการจ้างภายนอกในส่วนการชำระเงินและการจ่ายเงินของธุรกิจของคุณให้กับ TripPay นี่คือ 6 ขั้นตอนพื้นฐานสำหรับการเริ่มต้น:

<Steps>
1. สร้าง [บัญชีผู้ใช้](/th/user/register) บน Wink
2. สร้างบัญชีกับ [TripPay](https://pay.wink.travel)
3. สร้าง [Application](/th/developers/applications) บน TripPay ที่ช่วยให้คุณสื่อสารกับ API ของเราได้
4. สร้างและแมปผู้รับผลประโยชน์ของคุณกับ TripPay *เช่น สำหรับส่วนใหญ่ นั่นหมายถึงการสร้างบัญชีสำหรับที่พักบนแพลตฟอร์มของคุณที่คุณต้องการให้ TripPay ชำระเงิน*
5. ผสานรวม [คอมโพเนนต์เว็บการชำระเงิน](/th/developers/web-components/#payment) เข้ากับเว็บไซต์ของคุณ
6. ส่ง `shopping cart` ของนักท่องเที่ยวไปยัง TripPay และเริ่มกระบวนการชำระเงิน
</Steps>

:::tip
สำหรับคู่มือฉบับสมบูรณ์ในการใช้ TripPay สำหรับเว็บไซต์ท่องเที่ยวของคุณ โปรดอ่านคู่มือ [Integrating with TripPay](/th/guides/integrating-with-trip-pay)
:::
