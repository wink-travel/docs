หากคุณเป็นบริษัทการท่องเที่ยวและต้องการจ้างบุคคลภายนอกให้ทำหน้าที่ชำระเงินและเบิกจ่ายค่าใช้จ่ายทางธุรกิจของคุณให้กับ TripPay ต่อไปนี้เป็น 6 ขั้นตอนพื้นฐานในการเริ่มต้น:

<Steps>
  1. สร้าง [บัญชีผู้ใช้](/user/register) บน Wink
  2. สร้างบัญชีด้วย [ทริปเพย์](https://pay.wink.travel)-
  3. สร้าง [แอปพลิเคชัน](/developers/applications) บน TripPay ที่ให้คุณพูดคุยกับ API ของเราได้
  4. สร้างและจัดทำแผนที่ผู้รับผลประโยชน์ของคุณด้วย TripPay *เช่น สำหรับคนส่วนใหญ่ นั่นหมายถึงการสร้างบัญชีสำหรับคุณสมบัติบนแพลตฟอร์มของคุณที่คุณต้องการให้ TripPay ชำระเงิน*
  5. บูรณาการ [ส่วนประกอบเว็บการชำระเงิน](/developers/web-components/#payment) เข้าสู่เว็บไซต์ของคุณ
  6. ส่ง TripPay ให้กับนักเดินทาง `shopping cart` และเริ่มกระบวนการชำระเงิน
</Steps>


:::เคล็ดลับ
สำหรับคำแนะนำฉบับสมบูรณ์ในการใช้ TripPay สำหรับเว็บไซต์ท่องเที่ยวของคุณ โปรดอ่านคู่มือ[การบูรณาการกับ TripPay](/guides/integrating-with-trip-pay)-

## เอพีไอ

นักพัฒนาที่ต้องการบริหารจัดการ`Payments`ไปที่[นักพัฒนา > API > การชำระเงิน](/developers/apis/#payment-api)-
