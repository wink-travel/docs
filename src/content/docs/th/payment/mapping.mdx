หากคุณเป็นผู้บูรณาการ เช่น Wink และต้องการแมปผู้รับเงินภายนอกเพื่อใช้กับ TripPay คุณสามารถทำได้ 1 ใน 2 วิธีดังต่อไปนี้:

* การทำแผนที่ด้วยตนเอง
* การใช้[API การชำระเงิน](/developers/apis/#payment-api)-*ดู `Account Mappings`*-

:::เคล็ดลับ
เว้นแต่คุณจะจัดการกับคุณสมบัติเดียวกันเพียงไม่กี่อย่าง เราขอแนะนำให้คุณใช้ API เพื่อทำให้ขั้นตอนนี้เป็นแบบอัตโนมัติ
:::

ด้านล่างนี้ เราจะแนะนำคุณเกี่ยวกับวิธีการจัดทำแผนที่ผู้รับเงินภายนอกด้วยตนเอง*จะใช้ API ขั้นสูง*

<Steps>
  1. เข้าสู่ระบบ [ทริปเพย์](https://pay.wink.travel)-
  2. เลือกบัญชีผู้ผสานรวมของคุณ
  3. คลิก `Settings > Mappings` จากแถบนำทางหลัก
  4. คลิกที่ `Create mapping` ปุ่ม คุณจะเห็นฟอร์มการทำแผนที่
  5. **บัญชีท้องถิ่น** เริ่มพิมพ์ชื่อ TripPay ที่มีอยู่ที่คุณต้องการแมป
  6. **ตัวระบุภายนอก** วาง ID ภายนอกของคุณสำหรับเอนทิตีนี้เนื่องจากมีอยู่ในระบบระยะไกล
  7. คลิก `Save` เพื่อดำเนินการต่อ คุณจะถูกส่งกลับไปยังรายการการแมปของคุณ
</Steps>


เมื่อมีการแมปไว้แล้ว TripPay จะทราบทันทีว่าบัญชีใดที่จะเครดิตเป็นผู้รับผลประโยชน์จากการจองเมื่อคุณอ้างอิงถึงบัญชีเหล่านั้นโดยใช้ตัวระบุภายนอกของคุณ

## เอพีไอ

นักพัฒนาที่ต้องการบริหารจัดการ`Payments`, ไปที่[นักพัฒนา > API > การชำระเงิน](/developers/apis/#payment-api)-
