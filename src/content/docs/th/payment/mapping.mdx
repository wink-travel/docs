หากคุณเป็นผู้รวมระบบ เช่น Wink และต้องการทำแผนที่ผู้รับชำระเงินภายนอกของคุณเพื่อใช้กับ TripPay คุณสามารถทำได้สองวิธี:

- การทำแผนที่ด้วยตนเอง
- ใช้ [Payment API](/th/developers/apis/#payment-api) (*ดูที่ `Account Mappings`*)

:::tip
เว้นแต่คุณจะจัดการกับที่พักเพียงไม่กี่แห่งที่เหมือนกัน เราแนะนำให้คุณใช้ API เพื่อทำขั้นตอนนี้โดยอัตโนมัติ
:::

ด้านล่างนี้ เราจะแนะนำวิธีการทำแผนที่ผู้รับชำระเงินภายนอกด้วยตนเอง *ซึ่งใช้ API อยู่เบื้องหลัง*

<Steps>
1. เข้าสู่ระบบที่ [TripPay](https://pay.wink.travel)
2. เลือกบัญชีผู้รวมระบบของคุณ
3. คลิก `Settings > Mappings` จากแถบนำทางหลัก
4. คลิกปุ่ม `Create mapping` คุณจะเห็นแบบฟอร์มการทำแผนที่
5. **Local account** เริ่มพิมพ์ชื่อของ TripPay ที่มีอยู่ซึ่งคุณต้องการทำแผนที่
6. **External identifier** วาง ID ภายนอกของคุณสำหรับหน่วยงานนี้ตามที่มีอยู่ในระบบระยะไกล
7. คลิก `Save` เพื่อดำเนินการต่อ คุณจะถูกเปลี่ยนเส้นทางกลับไปยังรายการแผนที่ของคุณ
</Steps>

เมื่อคุณตั้งค่าการทำแผนที่แล้ว TripPay จะทราบว่าควรเครดิตบัญชีใดในฐานะผู้รับผลประโยชน์ของการจองเมื่อคุณอ้างอิงถึงพวกเขาโดยใช้ตัวระบุภายนอกของคุณ

## API

นักพัฒนาที่ต้องการจัดการ `Payments` ให้ไปที่ [Developers > API > Payments](/th/developers/apis/#payment-api)
