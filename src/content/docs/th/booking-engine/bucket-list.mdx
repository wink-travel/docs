ทุกที่ที่คุณเห็นสินค้าของ Wink[การ์ด](/studio/cards)คุณจะเห็นหัวใจที่มุมบนขวาของการ์ดซึ่งให้คุณเพิ่มสินค้าคงคลังลงในรายการสิ่งที่อยากทำก่อนตายของคุณได้

ปฏิบัติตามขั้นตอนเหล่านี้เพื่อจัดการรายการสิ่งที่ต้องการทำของคุณ:

<Steps>
  1. คลิกที่ ♡ `heart icon` สำหรับสินค้าคงคลังที่คุณต้องการเพิ่ม
  2. หากคุณล็อกอินแล้ว ไอคอนหัวใจจะเปลี่ยนเป็นสีแดง
  3. หากคุณไม่ได้ล็อกอิน คุณจะถูกส่งต่อเพื่อล็อกอินอีกครั้ง
</Steps>


:::เคล็ดลับ
คลิกที่ ♥️`heart icon`ครั้งที่สองเพื่อลบออกจากรายการของคุณ
:::

คุณสามารถจัดการรายการในรายการสิ่งที่อยากได้ทั้งหมดของคุณได้โดยคลิกไอคอนโปรไฟล์ของคุณที่มุมขวาบน จากนั้นคลิก`Bucket list` ปุ่ม.
