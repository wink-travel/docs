เครื่องมือจองห้องพักมี 2 ประเภทเพื่อรองรับความต้องการที่แตกต่างกันของบริษัทในเครือ ซึ่งก็คือบริษัทจัดการทรัพย์สินและโรงแรม

## บริษัททัวร์ออนไลน์

สำหรับพันธมิตร การใช้เวอร์ชันเอเจนซี่ท่องเที่ยวออนไลน์นั้นสมเหตุสมผลที่สุด เนื่องจากไซต์นี้จัดทำขึ้นเพื่อรองรับนักเดินทางที่ต้องการดูที่พักหลายแห่ง

คุณสามารถค้นหาหน้าแรกของแอปพลิเคชันนี้ได้ด้านล่าง

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  วิงค์โอตา
</LinkButton>


## ระบบจองผ่านอินเตอร์เน็ต

สำหรับโรงแรมหรือบริษัทจัดการทรัพย์สิน การใช้ระบบจองห้องพักทางอินเทอร์เน็ตถือเป็นทางเลือกที่เหมาะสมที่สุด ผู้ใช้ประเภทนี้มักสนใจที่จะเน้นย้ำถึงทรัพย์สินเดียวหรือหลายแห่งที่อยู่ภายใต้การจัดการ

คุณสามารถค้นหาหน้าแรกของแอปพลิเคชันนี้ได้ด้านล่าง

  วิงค์ ไอบีอี
</LinkButton>


## การปรับแต่ง

โรงแรมและบริษัทในเครือสามารถปรับแต่งได้`look and feel`ของเครื่องมือจองโดยการปรับแต่งจานสี

ทดสอบการปรับแต่งธีมของคุณในแซนด์บ็อกซ์ของนักพัฒนาของเราด้านล่าง

<LinkButton href="https://docs.wink.travel/web-component" variant="secondary" icon="external" target="_blank">
  ธีม
</LinkButton>


## การกำหนดค่า

ส่วนใหญ่แล้ว ไม่จำเป็นต้องแชร์แอปพลิเคชันเหล่านี้โดยตรง เราได้สร้างวิธีต่างๆ มากมายเพื่อให้คุณแชร์สินค้าคงคลังการเดินทางได้ ซึ่งคุณสามารถค้นพบได้ใน[วิงค์ สตูดิโอ](/studio/what-is-studio)หรือผ่านทาง[วิงค์ลิงค์](/link-manager/wink-links)-

อย่างไรก็ตาม หากคุณต้องการลิงก์ไปยังเว็บไซต์เหล่านี้ด้วยตนเอง โปรดแน่ใจว่าคุณได้เพิ่ม ID ไคลเอนต์และ ID การปรับแต่งลงใน URL เป็นพารามิเตอร์แล้ว

คุณสามารถค้นหารหัสลูกค้าของคุณได้โดย:

<Steps>
  1. เข้าสู่ระบบอย่างใดอย่างหนึ่ง [วิงค์เอ็กซ์ทราเน็ต](/extranet/what-is-extranet) หรือ [วิงค์ สตูดิโอ](/studio/what-is-studio)-
  2. คลิกที่ไอคอนโปรไฟล์ของคุณที่มุมขวาบนและคลิก [แอปพลิเคชั่น](/developers/applications)-
  3. ใช้แอปพลิเคชั่นที่เชื่อมโยงกับบัญชี Wink ที่คุณต้องการใช้งาน
  4. คัดลอกรหัสไคลเอนต์จากแอปพลิเคชันนั้นแล้ววางลงใน URL เป็นพารามิเตอร์คำขอ ตัวอย่าง: ?client-id=MY\_CLIENT\_ID
</Steps>


### ระบบเอ็กซ์ทราเน็ต

คุณสามารถค้นหา ID การปรับแต่งของคุณเป็นคุณสมบัติได้โดย:

<Steps>
  1. เข้าสู่ระบบ [วิงค์เอ็กซ์ทราเน็ต](/extranet/what-is-extranet)-
  2. เลือกคุณสมบัติที่คุณต้องการจะร่วมงานด้วย
  3. จากแถบนำทางหลัก คลิก `Account > Branding`
  4. คัดลอกรหัสการปรับแต่งและวางลงใน URL เป็นพารามิเตอร์คำขอ ตัวอย่าง: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### สตูดิโอ

คุณสามารถค้นหา ID การปรับแต่งของคุณในฐานะผู้ร่วมงานได้โดย:

<Steps>
  1. เข้าสู่ระบบ [วิงค์สตูดิโอ](/studio/what-is-studio)-
  2. เลือกบัญชีที่คุณต้องการใช้งาน
  3. จากแถบนำทางหลัก คลิก `Tools > Customization`
  4. คัดลอก ID การปรับแต่งจากการปรับแต่งที่คุณต้องการใช้และวางลงใน URL เป็นพารามิเตอร์คำขอ ตัวอย่าง: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## โอทีเอ

สิ่งที่คุณจะได้คือ URL ที่มีลักษณะประมาณนี้:

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

หากคุณต้องการไปที่หน้าโรงแรมโดยตรง URL จะมีลักษณะดังนี้:

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::หมายเหตุ
หากไม่ได้ระบุไคลเอนต์และ/หรือ ID การกำหนดค่า Wink OTA จะใช้การกำหนดค่าเริ่มต้นเป็นค่าเริ่มต้น
:::

## ไอบีอี

หากคุณเป็นบริษัทจัดการทรัพย์สิน คุณต้องการเชื่อมโยงไปยังหน้าแรกของ Wink IBE หน้าแรกจะแสดงธีมที่คุณกำหนดไว้พร้อมกับ[คุณสมบัติที่คุณเป็นตัวแทน](/studio/agency-representation)-

สิ่งที่คุณจะได้คือ URL ที่มีลักษณะประมาณนี้:

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

หากคุณเป็นทรัพย์สิน คุณอยากจะเชื่อมโยงโดยตรงไปยังหน้า Landing Page ของทรัพย์สิน Wink IBE

URL นั้นจะมีลักษณะประมาณนี้:

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::หมายเหตุ
หากคุณไม่ได้ระบุไคลเอนต์และ/หรือ ID การกำหนดค่า Wink IBE จะไม่โหลด จำเป็นต้องมีพารามิเตอร์เหล่านี้
:::

## ตัวอย่าง: หน้าทรัพย์สิน

ทรัพย์สินทุกแห่งใน Wink มีหน้า Landing Page ที่เป็นเอกลักษณ์เฉพาะของตนเองในเว็บไซต์ทั้งสองแห่ง ลิงก์ภายนอกทั้งหมดจะนำนักเดินทางไปยังหน้า Landing Page เหล่านี้

หน้า Landing Page ประกอบด้วยข้อมูลทั้งหมดที่นักเดินทางจำเป็นต้องทราบเพื่อตัดสินใจเลือกว่าจะเข้าพักที่โรงแรมแห่งนี้หรือไม่
