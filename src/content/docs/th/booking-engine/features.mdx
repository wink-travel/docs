เครื่องมือจองมีสองรูปแบบเพื่อรองรับความต้องการที่แตกต่างกันของพันธมิตร บริษัทบริหารจัดการที่พัก และโรงแรม

## ตัวแทนท่องเที่ยวออนไลน์

สำหรับพันธมิตร การใช้เวอร์ชันตัวแทนท่องเที่ยวออนไลน์จะเหมาะสมที่สุด เนื่องจากเว็บไซต์ถูกตั้งค่าให้รองรับนักเดินทางที่ต้องการดูที่พักหลายแห่ง

คุณสามารถดูหน้าแรกของแอปพลิเคชันนี้ได้ด้านล่าง

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## เครื่องมือจองผ่านอินเทอร์เน็ต

สำหรับโรงแรมหรือบริษัทบริหารจัดการที่พัก การใช้เวอร์ชันเครื่องมือจองผ่านอินเทอร์เน็ตจะเหมาะสมที่สุด ผู้ใช้ประเภทนี้สนใจเน้นที่พักเดียวหรือหลายที่พักภายใต้การบริหารจัดการ

คุณสามารถดูหน้าแรกของแอปพลิเคชันนี้ได้ด้านล่าง

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE
</LinkButton>

## การปรับแต่ง

โรงแรมและพันธมิตรสามารถปรับแต่ง *รูปลักษณ์และความรู้สึก* ของเครื่องมือจองได้โดยการปรับแต่งชุดสี

ทดลองปรับแต่งธีมของคุณในแซนด์บ็อกซ์สำหรับนักพัฒนาของเราด้านล่าง

<LinkButton href="https://developer.wink.travel/web-component" variant="secondary" icon="external" target="_blank">
  การตั้งธีม
</LinkButton>


## การตั้งค่า

ส่วนใหญ่แล้ว ไม่จำเป็นต้องแชร์แอปพลิเคชันเหล่านี้โดยตรง เราได้สร้างวิธีการหลากหลายให้คุณแชร์สินค้าท่องเที่ยว ซึ่งคุณสามารถค้นพบได้ใน [Wink Studio](/th/studio/what-is-studio) หรือผ่าน [WinkLinks](/th/link-manager/wink-links)

อย่างไรก็ตาม หากคุณต้องการลิงก์ไปยังเว็บไซต์เหล่านี้ด้วยตนเอง ให้แน่ใจว่าคุณเพิ่มรหัสลูกค้าและรหัสการปรับแต่งลงใน URL เป็นพารามิเตอร์

คุณสามารถค้นหารหัสลูกค้าของคุณได้โดย:

<Steps>
1. เข้าสู่ระบบที่ [Wink Extranet](/th/extranet/what-is-extranet) หรือ [Wink Studio](/th/studio/what-is-studio)
2. คลิกที่ไอคอนโปรไฟล์ของคุณที่มุมบนขวา แล้วคลิกที่ [Applications](/th/developers/applications)
3. ใช้แอปพลิเคชันที่เชื่อมโยงกับบัญชี Wink ที่คุณต้องการทำงานด้วย
4. คัดลอกรหัสลูกค้าจากแอปพลิเคชันนั้นและวางลงใน URL เป็นพารามิเตอร์คำขอ ตัวอย่าง: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

คุณสามารถค้นหารหัสการปรับแต่งในฐานะที่พักโดย:

<Steps>
1. เข้าสู่ระบบที่ [Wink Extranet](/th/extranet/what-is-extranet)
2. เลือกที่พักที่คุณต้องการทำงานด้วย
3. จากแถบนำทางหลัก คลิก `Account > Branding`
4. คัดลอกรหัสการปรับแต่งและวางลงใน URL เป็นพารามิเตอร์คำขอ ตัวอย่าง: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

คุณสามารถค้นหารหัสการปรับแต่งในฐานะพันธมิตรโดย:

<Steps>
1. เข้าสู่ระบบที่ [Wink Studio](/th/studio/what-is-studio)
2. เลือกบัญชีที่คุณต้องการทำงานด้วย
3. จากแถบนำทางหลัก คลิก `Tools > Customization`
4. คัดลอกรหัสการปรับแต่งจากการปรับแต่งที่คุณต้องการใช้และวางลงใน URL เป็นพารามิเตอร์คำขอ ตัวอย่าง: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

สิ่งที่คุณจะได้คือ URL ที่มีลักษณะดังนี้:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

หากคุณต้องการไปยังหน้าของโรงแรมโดยตรง URL จะมีลักษณะดังนี้:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
หากคุณไม่รวมรหัสลูกค้าและ/หรือรหัสการปรับแต่ง Wink OTA จะใช้การตั้งค่าเริ่มต้น
:::

## IBE

หากคุณเป็นบริษัทบริหารจัดการที่พัก คุณต้องการลิงก์ไปยังหน้าแรกของ Wink IBE หน้าแรกจะแสดงธีมที่กำหนดไว้ของคุณพร้อมกับ [ที่พักที่คุณเป็นตัวแทน](/th/studio/agency-representation)

สิ่งที่คุณจะได้คือ URL ที่มีลักษณะดังนี้:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

หากคุณเป็นที่พัก คุณต้องการลิงก์ไปยังหน้าลงจอดของที่พักใน Wink IBE โดยตรง

URL นั้นจะมีลักษณะดังนี้:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
หากคุณไม่รวมรหัสลูกค้าและ/หรือรหัสการปรับแต่ง Wink IBE จะไม่โหลด พารามิเตอร์เหล่านี้เป็นสิ่งจำเป็น
:::

## ตัวอย่าง: หน้าโรงแรม

ที่พักทุกแห่งใน Wink จะมีหน้าลงจอดเฉพาะของตนเองในแต่ละเว็บไซต์ ลิงก์ภายนอกทั้งหมดจะนำทางนักเดินทางไปยังหน้าลงจอดเหล่านี้

หน้าลงจอดประกอบด้วยข้อมูลทั้งหมดที่นักเดินทางต้องการเพื่อช่วยในการตัดสินใจอย่างมีข้อมูลว่าจะพักที่ที่พักนี้หรือไม่

  คลิกที่นี่เพื่อดูตัวอย่างหน้าเต็ม
</LinkButton>
