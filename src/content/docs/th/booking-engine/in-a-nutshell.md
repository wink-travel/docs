นี่คือวิธีที่เราจัดแบ่งความรับผิดชอบใน Wink:

- [Extranet](/th/extranet/what-is-extranet) ดูแลซัพพลายเออร์ที่จัดการสต็อกสินค้า
- [Studio](/th/studio/what-is-studio) ดูแลพันธมิตรที่จัดการเนื้อหา
- `Booking Engine` ดูแลการสร้างเนื้อหาที่สร้างใน `Studio` ซึ่งอิงจากสต็อกที่สร้างใน `Extranet` และทำให้สามารถจองได้
- [Payment](/th/payment/what-is-trip-pay) ดูแลการเรียกเก็บเงิน / คืนเงินให้กับนักเดินทางที่จอง [ผ่านแพลตฟอร์มอย่าง Wink] และจ่ายเงินให้ผู้ขาย

booking engine คือหัวใจหลักของแพลตฟอร์มของเรา

ความรับผิดชอบของมันรวมถึง:

- การจัดการสกุลเงินและอัปเดตอัตราแลกเปลี่ยนให้ทันสมัย
- ให้ผู้ให้บริการอัตราราคาส่งข้อมูลอัตราให้เรา
- การจัดการข้อมูลค้นหา *เช่น ภูมิภาค ประเทศ จุดหมายปลายทาง ที่พัก ฯลฯ*
- การจัดการภาษา
- การสร้างการวิเคราะห์ตามการโต้ตอบของผู้ใช้ เหตุการณ์ และการเรียก API
- การให้บริการสต็อกที่สามารถจองได้ในรูปแบบ ภาษา และสกุลเงินที่ต้องการ เช่น:
    - [ลิงก์ที่แชร์ได้](/th/studio/shareable-links) บน Facebook
    - หน้าแลนดิ้งของโรงแรมบน https://ota.wink.travel ที่มีสต็อกของโรงแรมนั้น
    - [การ์ด](/th/studio/cards) ที่มีสต็อกของพันธมิตร
    - [แผนที่](/th/studio/maps) บนบัญชี [WinkLinks](/th/link-manager/wink-links) ของคุณ
    - [การเรียก API](/th/developers/apis) จากระบบตัวแทนท่องเที่ยวระยะไกล
- การให้ทางเลือกแก่นักเดินทางเพื่อ:
    - ค้นหาสต็อกการเดินทาง
    - จองสต็อกที่มีอยู่
    - จัดการ [การจองและรีวิว](/th/booking-engine/bookings) ของตน
    - จัดการ [รายการสิ่งที่อยากจอง](/th/booking-engine/bucket-list) ในอนาคต
    - จัดการ [โปรไฟล์การเดินทาง](/th/booking-engine/travel-preferences) ของตน *เช่น ภูมิแพ้ ผู้ติดต่อฉุกเฉิน ฯลฯ*

booking engine คือชุดฟังก์ชันที่นำกลับมาใช้ใหม่ได้ซึ่งเราเปิดเผยผ่าน https://ota.wink.travel; เว็บไซต์ที่ปรับแต่งได้สูงสำหรับโรงแรมและพันธมิตรในการขายสินค้าให้กับนักเดินทาง

ผู้รวมระบบที่ต้องการอิสระมากขึ้น สามารถเข้าถึงฟีเจอร์ทั้งหมดผ่าน [ปลั๊กอิน WordPress](/th/developers/wordpress/), [Web Components](/th/developers/web-components) และ [APIs](/th/developers/apis) ของเรา

อ่านต่อในส่วนถัดไปเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับสิ่งที่ booking engine มีให้

## อ่านเพิ่มเติม

- อ่านเกี่ยวกับ [Web Components](/th/developers/web-components/)
- อ่านเกี่ยวกับ [APIs](/th/developers/apis/)
