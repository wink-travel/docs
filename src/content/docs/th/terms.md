# ข้อกำหนดและเงื่อนไข

แต่ละพอร์ทัลอาจมีข้อกำหนดและเงื่อนไขที่แตกต่างกัน กรุณาไปยังส่วนด้านล่างที่เกี่ยวข้องกับคุณและค้นหาย่อยที่ชื่อว่า ข้อกำหนดการให้บริการและเงื่อนไขการชำระเงิน

## พอร์ทัลพันธมิตร

สำหรับพันธมิตรที่ต้องการเรียนรู้วิธีสร้างรายได้แบบพาสซีฟโดยการสร้างรายได้จากบัญชีโซเชียลของคุณกับ Wink  
[Wink Studio](https://studio.wink.travel) มีเครื่องมือทั้งหมดที่คุณต้องการเพื่อขายอย่างมืออาชีพ

<LinkButton href="/th/studio/what-is-studio">
เรียนรู้เพิ่มเติมเกี่ยวกับ Wink Studio
</LinkButton>

## ระบบจอง

ทุกเส้นทางนำไปสู่ระบบจองของเรา! เรียนรู้ว่าคุณสามารถทำอะไรได้บ้างกับระบบจองที่ปรับแต่งได้มากที่สุดและเน้นการท่องเที่ยวในตลาด

<LinkButton href="/th/booking-engine/in-a-nutshell">
เรียนรู้เพิ่มเติมเกี่ยวกับระบบจองของเรา
</LinkButton>

## พอร์ทัลที่พัก

สำหรับที่พักที่ต้องการเริ่มขายบน Wink ใช้เครื่องมือของเราเพื่อขายและทำการตลาดออนไลน์อย่างมีประสิทธิภาพ และรับกองทัพพันธมิตรจากทั่วโลกมาช่วยคุณหาลูกค้าและการจอง

<LinkButton href="/th/extranet/what-is-extranet">
เรียนรู้เพิ่มเติมเกี่ยวกับ Wink Extranet
</LinkButton>

## พอร์ทัลตัวแทนท่องเที่ยว

สำหรับตัวแทนท่องเที่ยวที่ต้องการค้นหาว่า Wink สามารถเสนออะไรให้พวกเขาในแง่ของอัตราพิเศษ ข้อเสนอ และสินค้าคงคลังโดยตรงจากซัพพลายเออร์

<LinkButton href="/th/travel-agent/what-is-agent">
เรียนรู้เพิ่มเติมเกี่ยวกับ Wink Agent
</LinkButton>
