[Wink Agent](https://agent.wink.travel) เชื่อมต่อคุณกับสินค้าคงคลังการท่องเที่ยวทั่วโลก บทความนี้จะแนะนำขั้นตอนการสร้างและจัดการบัญชีตัวแทนท่องเที่ยวของคุณบน Wink

## สร้างบัญชี

เมื่อคุณเข้าสู่ระบบสำเร็จแล้ว คุณจะถูกเปลี่ยนเส้นทางไปยัง [หน้าบัญชีของคุณ](https://agent.wink.travel/company/list) ซึ่งแสดงบัญชีตัวแทนท่องเที่ยวทั้งหมดของคุณ

คลิกปุ่ม `Create new account` เพื่อดำเนินการต่อ คุณจะถูกเปลี่ยนเส้นทางไปยังหน้า [สร้างบัญชี](https://agent.wink.travel/company/create) ของเรา

กรอกข้อมูลในช่องที่จำเป็นทั้งหมด

<Steps>
1. **ชื่อ** ชื่อของบัญชีของคุณ *ไม่จำเป็นต้องเป็นชื่อจริงของคุณ แต่จะเป็นสิ่งที่ผู้ใช้เห็นเมื่อดูที่หน้าของ WinkLinks ของคุณ หรือเมื่อซัพพลายเออร์ต้องการเชื่อมต่อกับคุณโดยตรง*
2. **ประเภทนิติบุคคล** ว่าคุณเป็นบริษัทหรือบุคคล *ส่วนใหญ่ใช้เพื่อกำหนดว่าคุณมีสิทธิ์รับเงินอย่างไร*
3. **ประเภท** เลือกประเภทที่เหมาะสมกับบัญชีของคุณที่สุด *เช่น Influencer*
4. **คำอธิบาย** เขียนคำอธิบายสำหรับบัญชีของคุณ *จะปรากฏบนหน้าของ WinkLinks ของคุณและเมื่อซัพพลายเออร์ต้องการเชื่อมต่อกับคุณโดยตรง*
5. **เว็บไซต์** กรอกเว็บไซต์หลักของคุณ *อาจเป็น URL บัญชี IG หรือเว็บไซต์บริษัท ควรเป็นที่ที่คุณทำงานออนไลน์ส่วนใหญ่ ซัพพลายเออร์จะเข้าไปดูลิงก์เหล่านี้เมื่อตัดสินใจว่าจะเชื่อมต่อกับคุณหรือไม่*
6. **เมือง** คุณตั้งอยู่ที่ไหน?
</Steps>

### เงื่อนไขการชำระเงิน

:::note
หากคุณเลือกที่จะรับผิดชอบการชำระเงิน คุณจะสามารถเข้าถึงเฉพาะที่พักที่คุณมีการเชื่อมต่อโดยตรงเท่านั้น
:::

:::caution
Wink จะไม่รับผิดชอบในการชำระเงินให้โรงแรมในกรณีนี้
:::

:::note
คุณสามารถเลือกตัวเลือกนี้ได้เฉพาะในระหว่างการสร้างบัญชี ตัวแทนท่องเที่ยวส่วนใหญ่ควรปล่อยให้การตั้งค่านี้ปิดใช้งาน 🛑 และใช้พอร์ทัลนี้แทน
:::

ตัวแทนท่องเที่ยวที่ลงทะเบียนและเชื่อมต่อกับเราผ่าน API ของเรา มีตัวเลือกในการรับชำระเงินและจ่ายเงินให้กับที่พักของเรา

<Steps>
1. **ฉันต้องการรับชำระเงิน** ตัวแทนท่องเที่ยวรับผิดชอบในการเก็บเงิน
2. **ฉันต้องการจ่ายเงินให้โรงแรม** ตัวแทนท่องเที่ยวรับผิดชอบในการจ่ายเงินให้โรงแรม
</Steps>

โรงแรมที่ต้องการเชื่อมต่อกับคุณโดยตรงจะเห็นคำเตือนและต้องคลิกเพื่อยอมรับว่า Wink ไม่รับผิดชอบในการจ่ายเงินให้โรงแรม เพื่อสร้างความสัมพันธ์กับบัญชีของคุณ

### ข้อตกลง

<Steps>
1. **ข้อตกลง** ยอมรับ [ข้อตกลงพันธมิตร](/th/portal/travel-agent/terms-of-service) และ [เงื่อนไขการชำระเงิน](/th/portal/travel-agent/payment-terms) ของเราเพื่อดำเนินการต่อ
2. คลิกปุ่ม `Save`
</Steps>

ขอแสดงความยินดี 🎉 ...คุณได้สร้างบัญชีพันธมิตรแรกของคุณบน Wink แล้ว

เมื่อสร้างแล้ว คุณสามารถปรับแต่งบัญชีของคุณเพิ่มเติมโดยกรอกที่อยู่และเพิ่มรูปโปรไฟล์ให้กับบัญชีของคุณ

<Steps>
1. คลิกที่บัญชีตัวแทนท่องเที่ยวใหม่ของคุณ *จะพาคุณไปยังแดชบอร์ดบัญชีของคุณ*
2. คลิกที่ `Account > Profile` ในแถบนำทางหลัก
3. ในหน้าประวัติส่วนตัว คุณจะสามารถอัปเดตโปรไฟล์ที่มีอยู่และเพิ่มข้อมูลเพิ่มเติมได้
</Steps>

:::note
รูปโปรไฟล์บัญชีตัวแทนท่องเที่ยวของคุณไม่ควรสับสนกับรูปโปรไฟล์ผู้ใช้ Wink ของคุณ รูปโปรไฟล์บัญชีตัวแทนท่องเที่ยวจะแสดงบนหน้าของ WinkLinks ของคุณและจะเห็นโดยซัพพลายเออร์ที่ต้องการเชื่อมต่อกับคุณ

รูปโปรไฟล์ผู้ใช้ของคุณส่วนใหญ่ใช้สำหรับภายในเท่านั้น
:::

## ผู้จัดการ

คุณสามารถเชิญผู้ใช้คนอื่นมาช่วยจัดการบัญชีตัวแทนท่องเที่ยวของคุณได้ เมื่อผู้ใช้ยอมรับคำเชิญของคุณ พวกเขาจะสามารถเข้าถึงบัญชีของคุณได้เต็มรูปแบบ ยกเว้นเรื่องการชำระเงิน คุณจะเป็นผู้รับผิดชอบเงินทุนที่คุณเป็นเจ้าของเพียงผู้เดียวเสมอ

<Steps>
1. เลือกบัญชีตัวแทนท่องเที่ยวที่คุณต้องการทำงานด้วยจาก [หน้าบัญชี](https://agent.wink.travel/company/list)
2. คลิกที่ `Account > Managers` ในแถบนำทางหลัก
3. เพิ่มอีเมลของผู้ใช้ที่คุณต้องการแต่งตั้งเป็นผู้จัดการบัญชี
4. คลิกที่ลิงก์ `Invite` ข้างอีเมลนั้น
</Steps>

Wink จะส่งอีเมลไปยังผู้ใช้เพื่อให้ยอมรับหรือปฏิเสธคำเชิญของคุณ

## การแจ้งเตือน

บางครั้ง เราต้องการติดต่อคุณ เมื่อเกิดเหตุการณ์นั้น คุณจะเห็นไอคอนกระดิ่งที่มุมขวาบนเปลี่ยนแปลง
เพื่ออ่านสิ่งที่เราส่งให้คุณ:

<Steps>
1. คลิกที่ `Account > Notifications` ในแถบนำทางหลัก
2. คุณจะเห็นการแจ้งเตือนใหม่และการแจ้งเตือนที่ผ่านมา
3. เลือก `Mark as read` หรือ `Delete` การแจ้งเตือนหลังจากที่คุณอ่านแล้ว
</Steps>
