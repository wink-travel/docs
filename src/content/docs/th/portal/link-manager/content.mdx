เนื้อหาของคุณจะอยู่ภายใต้แท็บ `Entries`

เรารองรับเนื้อหาหลักสามประเภท:

- URL ที่เปิดให้สาธารณะเข้าถึงซึ่งลิงก์ไปยังหน้าเว็บ
- ไฟล์ที่อัปโหลดผ่าน WinkLinks
- เนื้อหาข้อความที่สร้างผ่านฟีเจอร์ `Quotes` ของเรา

## เพิ่มเนื้อหา

นี่คือวิธีเพิ่มเนื้อหาใน WinkLinks ภายใต้แท็บ `Entries` เริ่มต้นโดยคลิกปุ่ม `Add entry`

### ลิงก์ปกติ

:::tip
ลิงก์ต้องไปยังหน้าเว็บเท่านั้น เช่น คุณไม่สามารถลิงก์โดยตรงไปยังไฟล์ PDF ได้ หากคุณต้องการแชร์ PDF คุณต้องใช้ฟีเจอร์ `Upload file`
:::

ขั้นตอนการเพิ่มลิงก์ปกติ:

<Steps>
1. ป้อนลิงก์ที่คุณต้องการแชร์ *เช่น https://trvl.as/3xWCH*
2. เมื่อคุณป้อนลิงก์ Wink จะค้นหารายละเอียดของเว็บไซต์นี้และกรอกแบบฟอร์มที่คุณเห็นถัดไป
3. คุณสามารถเลือกแก้ไขข้อมูลบางส่วน เช่น ชื่อเว็บไซต์ ชื่อเรื่อง และคำอธิบาย
4. คุณยังสามารถเลือกอัปโหลดภาพที่กำหนดเองได้
5. คลิกปุ่ม `Save` เมื่อคุณพร้อมที่จะดำเนินการต่อ
</Steps>

### อัปโหลดไฟล์

:::caution[ข้อกำหนด]
- ขนาดภาพต้องไม่เกิน 10Mb
- ขนาดวิดีโอต้องไม่เกิน 50Mb
- เรารองรับทุกรูปแบบภาพและวิดีโอที่เป็นที่รู้จัก รวมถึง PDF
:::

:::note[คำแนะนำ]
- 2560w x 1440h (อัตราส่วน 16x9) สำหรับภาพเด่นของคุณ
- 2560w x 1920h (อัตราส่วน 4:3) สำหรับภาพอื่นๆ ทั้งหมด
- ขนาดภาพไม่ควรเกิน 5Mb กรุณาบีบอัดภาพก่อนอัปโหลดไปยัง Wink
:::

ขั้นตอนการเพิ่มไฟล์:

<Steps>
1. สามารถเพิ่มสื่อได้โดย:
    - ลากภาพจากคอมพิวเตอร์ของคุณลงในหน้าต่าง หรือคลิก `Browse`
    - วาง URL ภายนอกที่ชี้ไปยังภาพ
    - ใช้กล้องแล็ปท็อปของคุณ
    - จากบัญชี Google Drive ของคุณ
    - จากบัญชี Dropbox ของคุณ
    - จาก Shutterstock
    - จาก gettyimages
    - จาก iStock
    - จาก Unsplash
2. เมื่อไฟล์อัปโหลดเสร็จ คุณสามารถเพิ่มข้อมูลเมตาเพิ่มเติมเพื่ออธิบายไฟล์ เช่น ชื่อเรื่องและคำอธิบาย
3. คลิกปุ่ม `Save` เมื่อคุณพร้อมที่จะดำเนินการต่อ
</Steps>

### เขียนคำคม

:::tip
คำคมเป็นวิธีที่ดีในการเพิ่มตัวแบ่งเพื่อแยกรายการยาวๆ [...และยังทำให้คุณดูฉลาดและมีความรู้ด้วย 😉]
:::

ขั้นตอนการเขียนคำคม:

<Steps>
1. พิมพ์คำคม *เช่น Roses are red. Violets are blue...*
2. ป้อนชื่อผู้แต่งต้นฉบับ *เช่น Edmund Spenser*
3. ป้อนชื่อเว็บไซต์ที่คุณได้คำคมมา *เช่น Daily Quotes*
4. คลิกปุ่ม `Save` เมื่อคุณพร้อมที่จะดำเนินการต่อ
</Steps>

## อัปเดตเนื้อหา

เนื้อหาทั้งหมดสามารถแก้ไขได้ง่ายโดยคลิกที่ลิงก์ `Actions` และเลือก `Update` คุณสามารถลบเนื้อหาได้ในลักษณะเดียวกันโดยคลิกปุ่ม `Remove`

## ลบเนื้อหา

เนื้อหาทั้งหมดสามารถแก้ไขได้โดยเลือก `Update` คุณสามารถลบเนื้อหาได้ในลักษณะเดียวกันโดยคลิกที่ลิงก์ `Actions` ก่อนแล้วจึงคลิกปุ่ม `Remove`

## การจัดเรียงเนื้อหา

คุณสามารถจัดลำดับเนื้อหาของคุณโดยคลิกที่ไอคอนที่อยู่มุมล่างซ้ายของแต่ละรายการและลากไปยังตำแหน่งที่คุณต้องการ

## การจัดกลุ่มเนื้อหา

:::tip
สร้างแท็กของคุณก่อนสร้างรายการ มิฉะนั้นคุณจะสูญเสียความคืบหน้าและต้องเริ่มใหม่
:::

เมื่อคุณแก้ไขรายการ คุณสามารถเพิ่มแท็กหนึ่งหรือมากกว่าสำหรับการจัดกลุ่ม เลื่อนลงไปที่ด้านล่างของแบบฟอร์มเพื่อหาช่องกรอก `Tags`

ผู้ใช้ที่ดูหน้า WinkLinks ของคุณ สามารถเลือกวิธีการบริโภคเนื้อหาของคุณ (แบบรายการหรือแบบกริด) และสามารถเลือกแท็กหนึ่งหรือมากกว่าเพื่อกรองเนื้อหาเพิ่มเติม
