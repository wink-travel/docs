หากคุณต้องการเปลี่ยนรหัสผ่านสำหรับบัญชีผู้ใช้ Wink ของคุณ ให้ทำตามขั้นตอนเหล่านี้:

## อัพเดตรหัสผ่าน

<Steps>
  1. คุณจะต้องเข้าสู่ระบบในหนึ่งในเว็บไซต์ของเรา
  2. คลิกไอคอนโปรไฟล์ของคุณที่มุมขวาบนแล้วเมนูจะปรากฏขึ้น
  3. คลิกที่ `Change password` ลิงค์
  4. ปฏิบัติตามคำแนะนำเพื่อเปลี่ยนรหัสผ่านของคุณ
  5. คลิก `Save`-
</Steps>


เมื่อคุณเข้าสู่ระบบครั้งถัดไป คุณจะต้องใช้รหัสผ่านใหม่

## ลืมรหัสผ่าน

บางครั้งจำเป็นต้องรีเซ็ตรหัสผ่านผู้ใช้

นี่คือขั้นตอนในการรีเซ็ตรหัสผ่านผู้ใช้ของคุณ:

<Steps>
  1. คลิกที่ `Forgot password` ลิงก์จากหน้าลงชื่อเข้าใช้
  2. พิมพ์ชื่อผู้ใช้ / อีเมล์ สำหรับบัญชีที่คุณต้องการรีเซ็ต
  3. คลิก `Forgot password` ปุ่ม. *คุณจะถูกส่งต่อไปยังแบบฟอร์มการรีเซ็ตรหัสผ่าน*
  4. ตรวจสอบอีเมล์ของคุณ
</Steps>

<Steps>
  1. วางรหัสที่คุณได้รับจากอีเมลลงในแบบฟอร์มใหม่พร้อมรหัสผ่านใหม่ของคุณ
  2. ยืนยันรหัสผ่านใหม่ของคุณ
  3. คลิก `Reset password` ปุ่ม. *คุณจะถูกส่งกลับไปยังหน้าลงชื่อเข้าใช้*
</Steps>

  <figcaption>*แบบฟอร์มรีเซ็ตรหัสผ่าน*</figcaption>

รีเซ็ตรหัสผ่านของคุณเรียบร้อยแล้ว คุณสามารถเข้าสู่ระบบ Wink ด้วยข้อมูลประจำตัวใหม่ของคุณได้
