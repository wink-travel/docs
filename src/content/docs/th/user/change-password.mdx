หากคุณต้องการเปลี่ยนรหัสผ่านสำหรับบัญชีผู้ใช้ Wink ของคุณ ให้ทำตามขั้นตอนเหล่านี้:

## อัปเดตรหัสผ่าน

<Steps>
1. คุณต้องเข้าสู่ระบบในหนึ่งในเว็บไซต์ของเรา
2. คลิกที่ไอคอนโปรไฟล์ของคุณ ที่มุมขวาบน แล้วเมนูจะปรากฏขึ้น
3. คลิกที่ลิงก์ `Change password`
4. ทำตามคำแนะนำเพื่อเปลี่ยนรหัสผ่านของคุณ
5. คลิก `Save`
</Steps>

ครั้งถัดไปที่คุณเข้าสู่ระบบ คุณจะต้องใช้รหัสผ่านใหม่ของคุณ

## ลืมรหัสผ่าน

บางครั้งจำเป็นต้องรีเซ็ตรหัสผ่านผู้ใช้

นี่คือขั้นตอนในการรีเซ็ตรหัสผ่านผู้ใช้ของคุณ:

<Steps>
1. คลิกที่ลิงก์ `Forgot password` จากหน้าลงชื่อเข้าใช้
2. พิมพ์ชื่อผู้ใช้ / อีเมลของบัญชีที่คุณต้องการรีเซ็ต
3. คลิกปุ่ม `Forgot password` *คุณจะถูกเปลี่ยนเส้นทางไปยังแบบฟอร์มรีเซ็ตรหัสผ่าน*
4. ตรวจสอบอีเมลของคุณ
</Steps>

<Steps>
1. วางรหัสที่คุณได้รับในอีเมลลงในแบบฟอร์มใหม่พร้อมกับรหัสผ่านใหม่ของคุณ
2. ยืนยันรหัสผ่านใหม่ของคุณ
3. คลิกปุ่ม `Reset password` *คุณจะถูกเปลี่ยนเส้นทางกลับไปยังหน้าลงชื่อเข้าใช้*
</Steps>

รหัสผ่านของคุณถูกรีเซ็ตเรียบร้อยแล้ว คุณสามารถเข้าสู่ระบบ Wink ด้วยข้อมูลประจำตัวใหม่ของคุณได้ทันที
