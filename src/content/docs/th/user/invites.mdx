ผู้ใช้ Wink สามารถเลือกให้คุณจัดการบัญชีทรัพย์สิน บัญชีพันธมิตร หรือบัญชีตัวแทนท่องเที่ยว เมื่อเป็นเช่นนั้น คุณจะได้รับอีเมลพร้อมคำขอคำเชิญ

import { Steps } from '@astrojs/starlight/components';

หากต้องการดูคำเชิญ Wink ปัจจุบันของคุณ ให้ทำตามขั้นตอนเหล่านี้:

<Steps>
  1. คุณจะต้องเข้าสู่ระบบในหนึ่งในเว็บไซต์ของเรา
  2. คลิกไอคอนโปรไฟล์ของคุณที่มุมขวาบนแล้วเมนูจะปรากฏขึ้น
  3. คลิกที่ `Invites` ลิงค์
  4. โต้ตอบกับคำเชิญของคุณโดย `Accepting` หรือ `Rejecting` พวกเขา.
</Steps>
