ผู้ใช้ Wink สามารถเลือกให้คุณจัดการบัญชีที่พัก, บริษัทในเครือ หรือเอเจนซี่ท่องเที่ยวของพวกเขาได้ เมื่อเป็นเช่นนั้น คุณจะได้รับอีเมลพร้อมคำขอเชิญ

import { Steps } from '@astrojs/starlight/components';

เพื่อดูคำเชิญ Wink ปัจจุบันของคุณ ให้ทำตามขั้นตอนเหล่านี้:

<Steps>
1. คุณต้องเข้าสู่ระบบในหนึ่งในเว็บไซต์ของเรา
2. คลิกไอคอนโปรไฟล์ของคุณที่มุมขวาบน แล้วเมนูจะปรากฏขึ้น
3. คลิกที่ลิงก์ `Invites`
4. โต้ตอบกับคำเชิญของคุณโดยการ `Accepting` หรือ `Rejecting` พวกมัน
</Steps>
