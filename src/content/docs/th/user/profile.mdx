import { Steps } from '@astrojs/starlight/components';

หากคุณต้องการอัปเดตโปรไฟล์ผู้ใช้สำหรับบัญชีผู้ใช้ Wink ของคุณ ให้ทำตามขั้นตอนเหล่านี้:

<Steps>
1. คุณต้องเข้าสู่ระบบในหนึ่งในเว็บไซต์ของเรา
2. คลิกที่ไอคอนโปรไฟล์ของคุณ ที่มุมขวาบน แล้วเมนูจะปรากฏขึ้น
3. คลิกที่ลิงก์ `Profile`
4. ทำตามคำแนะนำเพื่ออัปเดต `อีเมล` `ชื่อ` และ `นามสกุล` ของคุณ
5. คุณยังสามารถอัปโหลด `รูปโปรไฟล์` ใหม่ของตัวเองได้ด้วย
6. คลิก `บันทึก`
</Steps>

:::note
รูปโปรไฟล์ใหม่ของคุณจะแสดงผลเมื่อคุณออกจากระบบ Wink อย่างสมบูรณ์ครั้งถัดไป
:::
