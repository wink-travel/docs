import { Steps } from '@astrojs/starlight/components';

หากคุณต้องการอัปเดตโปรไฟล์ผู้ใช้สำหรับบัญชีผู้ใช้ Wink ของคุณ ให้ทำตามขั้นตอนเหล่านี้:

<Steps>
  1. คุณจะต้องเข้าสู่ระบบในหนึ่งในเว็บไซต์ของเรา
  2. คลิกไอคอนโปรไฟล์ของคุณที่มุมขวาบนแล้วเมนูจะปรากฏขึ้น
  3. คลิกที่ `Profile` ลิงค์
  4. ปฏิบัติตามคำแนะนำเพื่ออัพเดตของคุณ `e-mail`- `first` และ `last name`-
  5. คุณยังสามารถอัพโหลดใหม่ได้ `profile picture` ของตัวคุณเอง
  6. คลิก `Save`-
</Steps>


:::หมายเหตุ
รูปโปรไฟล์ใหม่ของคุณจะปรากฏขึ้นในครั้งถัดไปที่คุณออกจากระบบ Wink โดยสมบูรณ์
:::
