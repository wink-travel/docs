คุณสามารถสร้างบัญชีผู้ใช้ผ่านพอร์ทัลใดก็ได้ของเรา เมื่อสร้างบัญชีแล้ว บัญชีนี้จะสามารถใช้งานได้กับแอปและเว็บไซต์ทั้งหมดของ Wink

## นักเดินทาง

ปฏิบัติตามขั้นตอนเหล่านี้:

<Steps>
  1. มุ่งหน้าไปที่ [https://ota.wink.travel](https://ota.wink.travel)
  2. คลิ๊กที่นี่ `Sign-In / Register` ที่มุมขวาบน
  3. คุณจะถูกขอให้ป้อนรายละเอียดผู้ใช้พื้นฐาน กรอกข้อมูลที่จำเป็นทั้งหมดแล้วคลิก `Save`-
  4. ตอนนี้คุณสามารถเข้าสู่ระบบ Wink ได้ด้วยชื่อผู้ใช้ (อีเมล) และรหัสผ่านของคุณ
  5. เมื่อเข้าสู่ระบบแล้ว ระบบจะนำคุณกลับไปยังจุดที่คุณมา
</Steps>

## การลงทะเบียน

การสร้างบัญชีผู้ใช้ Wink จะทำให้คุณเข้าถึงฟีเจอร์ทั้งหมดบนแพลตฟอร์มได้ คุณสามารถสร้างคุณสมบัติและบัญชีพันธมิตรได้ไม่จำกัด

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="เข้าสู่ระบบ" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*แบบฟอร์มลงชื่อเข้าใช้งาน*</figcaption>
</figure>


<Steps>
  1. ครั้งแรกที่คุณถูกส่งไปที่เซิร์ฟเวอร์การอนุญาตของเรา คุณจะถูกขอให้ป้อนชื่อผู้ใช้และรหัสผ่านของคุณ
  2. ให้คลิกที่แทน `Create account` ปุ่ม.
  3. กรอกข้อมูลที่จำเป็นให้ครบถ้วนแล้วคลิก `Register` เพื่อดำเนินการต่อ ซึ่งจะนำคุณกลับไปยังแบบฟอร์มลงชื่อเข้าใช้
  4. กรุณากรอกอีเมล์ / ชื่อผู้ใช้ และรหัสผ่าน เพื่อเข้าสู่ระบบ
  5. คุณจะถูกส่งกลับไปยังไซต์ที่คุณมา
</Steps>
