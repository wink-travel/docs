คุณสามารถสร้างบัญชีผู้ใช้ผ่านพอร์ทัลใดก็ได้ของเรา เมื่อสร้างแล้ว บัญชีนี้สามารถใช้ได้กับแอปและเว็บไซต์ทั้งหมดของ Wink

## นักเดินทาง

ทำตามขั้นตอนเหล่านี้:

<Steps>
1. ไปที่ [https://ota.wink.travel](https://ota.wink.travel)
2. คลิกที่ `Sign-In / Register` ที่มุมขวาบน
3. ระบบจะขอให้คุณกรอกรายละเอียดผู้ใช้พื้นฐาน กรอกข้อมูลในช่องที่จำเป็นทั้งหมดแล้วคลิก `Save`
4. ตอนนี้คุณสามารถเข้าสู่ระบบ Wink ด้วยชื่อผู้ใช้ (อีเมล) และรหัสผ่านของคุณ
5. เมื่อเข้าสู่ระบบแล้ว ระบบจะพาคุณกลับไปยังหน้าที่คุณมาจาก
</Steps>

## การลงทะเบียน

การสร้างบัญชีผู้ใช้ Wink จะทำให้คุณเข้าถึงฟีเจอร์ทั้งหมดบนแพลตฟอร์มได้ คุณสามารถสร้างที่พักและบัญชีผู้ร่วมงานได้ไม่จำกัด

    <figcaption>*แบบฟอร์มเข้าสู่ระบบ*</figcaption>
</figure>

<Steps>
1. ครั้งแรกที่คุณถูกนำไปยังเซิร์ฟเวอร์การอนุญาตของเรา ระบบจะขอให้คุณกรอกชื่อผู้ใช้และรหัสผ่าน
2. ให้คลิกที่ปุ่ม `Create account` แทน
3. กรอกข้อมูลในช่องที่จำเป็นทั้งหมดแล้วคลิก `Register` เพื่อดำเนินการต่อ ระบบจะพาคุณกลับไปยังแบบฟอร์มเข้าสู่ระบบ
4. กรอกอีเมล / ชื่อผู้ใช้ และรหัสผ่านเพื่อเข้าสู่ระบบ
5. คุณจะถูกพากลับไปยังเว็บไซต์ที่คุณมาจาก
</Steps>
