:::หมายเหตุ
เราห่อ WordPress ไว้รอบๆ[ส่วนประกอบของเว็บไซต์](/developers/web-components). ไปที่นั่นเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับสิ่งที่ส่วนประกอบต่างๆ สามารถทำได้ บทความนี้จะเน้นที่วิธีใช้ส่วนประกอบเหล่านี้กับ WordPress
:::

สำหรับใครก็ตามที่ใช้ WordPress สำหรับธุรกิจหรือบล็อกของตน เราขอแนะนำให้ใช้[ปลั๊กอิน Wink Affiliate WordPress](https://wordpress.org/plugins/wink2travel/).
ปลั๊กอินช่วยให้คุณเข้าถึงการ์ด Wink ตาราง แผนที่ ฯลฯ ได้อย่างง่ายดาย และทำให้ง่ายดายเหมือนกับการลากเนื้อหาลงในโพสต์และหน้าของคุณโดยใช้ตัวสร้างหน้าที่คุณชื่นชอบหรือรหัสย่อแบบกำหนดเองของเรา

## ติดตั้ง

นี่คือวิธีการติดตั้ง`Wink Affiliate WordPress Plugin`ลงในเว็บไซต์ WordPress ของคุณ

<Steps>
  1. เมื่อคุณเข้าสู่ระบบในฐานะผู้ดูแลระบบและบนแดชบอร์ด
  2. คลิก `Plugins > Add New Plugin`-
  3. ค้นหา `Wink Affiliate WordPress Plugin`-
  4. คลิก `Install` เพื่อดาวน์โหลดและติดตั้งปลั๊กอินลงในไซต์ของคุณ
  5. คลิก `Activate` เพื่อเปิดใช้งานปลั๊กอินบนเว็บไซต์ของคุณ
  6. การเปิดใช้งานจะส่งต่อให้คุณ `Wink Settings` หน้าหนังสือ. *คุณสามารถไปที่นั่นได้โดยไปที่ `Appearance > Customize > Wink Settings`-*
  7. มันจะถามคุณถึง `Client ID` และ `Client Secret`. ไปที่ [แอปพลิเคชั่น](/developers/applications) เพื่อเรียนรู้วิธีดึงข้อมูลประจำตัวของคุณ
  8. กรอกข้อมูลประจำตัวของคุณและคลิก `Publish`ตอนนี้คุณเชื่อมต่อกับบัญชี Wink ของคุณแล้ว
</Steps>

## โปรแกรมสร้างหน้าเพจ

เรารองรับโปรแกรมสร้างเพจที่ได้รับความนิยมสูงสุดใน WordPress ซึ่งทำให้ทุกคนที่ต้องการใช้ Wink ได้อย่างง่ายดาย

### กูเทนเบิร์ก

Gutenberg คือตัวแก้ไขดั้งเดิมของ WordPress

### อาวาดา

[อาวาดา](https://avada.com)มีชื่อเสียงในด้านการมีเทมเพลตสวยงามให้เลือกมากมาย พร้อมด้วยโปรแกรมสร้างหน้าอันแข็งแกร่งพร้อมชุดคุณสมบัติขั้นสูง

### เอเลเมนเตอร์

[เอเลเมนเตอร์](https://elementor.com)คือเครื่องมือสร้างเพจที่ได้รับความนิยมสูงสุด \[ในปัจจุบัน] โดยมีทั้งเวอร์ชันฟรีและเวอร์ชันโปรให้เลือกใช้

### ดับเบิ้ลยูพีเบเกอรี่

[ดับเบิ้ลยูพีเบเกอรี่](https://wpbakery.com)เป็นเครื่องมือสร้างหน้าขั้นสูงตัวแรกในตลาด

## รหัสสั้น

### \[กระพริบตาดู]

ทำงานร่วมกับกริดเนื้อหาแบบจัดอันดับ พิมพ์สถานที่ที่คุณต้องการไปเยี่ยมชมและแสดงสินค้าคงคลัง

### \[กระพริบตาค้นหา]

ปุ่มง่ายๆ เพื่อเปิดแบบฟอร์มแผนการเดินทาง

### \[บัญชีกระพริบตา]

ปุ่มบัญชีช่วยให้คุณตรวจสอบความถูกต้อง เมื่อตรวจสอบความถูกต้องแล้ว ปุ่มดังกล่าวจะเปลี่ยนเป็นเมนูแบบดรอปดาวน์พร้อมตัวเลือกบัญชี

### \[วิงค์กิตินารี]

เช่นเดียวกับปุ่มค้นหา เพียงแต่มีข้อมูลแผนการเดินทางเป็นข้อความบนปุ่ม

### \[กระพริบตาเนื้อหา]

ช่วยให้คุณฝังสินค้าคงคลังที่คุณเลือกบน Wink ลงในเพจหรือโพสต์ได้โดยตรง นอกจากนี้ ชอร์ตโค้ดนี้ยังต้องการ ID เค้าโครงด้วย

เค้าโครง

```
[winkcontent layoutid=""]
```

## อ่านเพิ่มเติม

คุณสามารถหาข้อมูลเพิ่มเติมเกี่ยวกับปลั๊กอิน WordPress ของเราได้จากลิงค์ด้านล่าง

* เรียนรู้เกี่ยวกับของเรา[ส่วนประกอบของเว็บไซต์](/developers/web-components)-
* ติดตั้ง[ปลั๊กอิน Wink Affiliate WordPress](https://wordpress.org/plugins/wink2travel/)-
* ฟอร์กและสร้างบน Github ของเรา[คลังข้อมูล Wink Affiliate WordPress](https://github.com/wink-travel/wink-wordpress-plugin)-
