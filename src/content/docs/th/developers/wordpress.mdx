สำหรับผู้ที่ใช้ WordPress สำหรับธุรกิจหรือบล็อก เราขอแนะนำให้ใช้ [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/)
ปลั๊กอินนี้ช่วยให้คุณเข้าถึงการ์ด Wink, กริด, แผนที่ ฯลฯ ได้อย่างง่ายดาย และทำให้การลากเนื้อหาไปยังโพสต์และเพจของคุณด้วย page builder ที่คุณชื่นชอบหรือ shortcodes ที่เราสร้างขึ้นเป็นเรื่องง่าย

##  การติดตั้ง

นี่คือวิธีการติดตั้ง `Wink Affiliate WordPress Plugin` ลงในเว็บไซต์ WordPress ของคุณ

<Steps>
1. เมื่อคุณเข้าสู่ระบบในฐานะผู้ดูแลระบบและอยู่ที่ Dashboard
2. คลิก `Plugins > Add New Plugin`
3. ค้นหา `Wink Affiliate WordPress Plugin`
4. คลิก `Install` เพื่อดาวน์โหลดและติดตั้งปลั๊กอินลงในเว็บไซต์ของคุณ
5. คลิก `Activate` เพื่อเปิดใช้งานปลั๊กอินในเว็บไซต์ของคุณ
6. การเปิดใช้งานจะพาคุณไปยังหน้า `Wink Settings` *คุณยังสามารถไปที่นั่นได้โดยไปที่ `Appearance > Customize > Wink Settings`*
7. ระบบจะขอ `Client ID` และ `Client Secret` ของคุณ ไปที่ [Applications](/th/developers/applications) เพื่อเรียนรู้วิธีดึงข้อมูลรับรองของคุณ
8. กรอกข้อมูลรับรองของคุณแล้วคลิก `Publish` ตอนนี้คุณเชื่อมต่อกับบัญชี Wink ของคุณแล้ว
</Steps>

## Page builders

เรารองรับ page builders ที่ได้รับความนิยมมากที่สุดบน WordPress ซึ่งทำให้ง่ายสำหรับทุกคนที่ต้องการใช้ Wink

### Gutenberg

Gutenberg คือเครื่องมือแก้ไขเนื้อหาเริ่มต้นของ WordPress

### Avada

[Avada](https://avada.com) มีชื่อเสียงในด้านเทมเพลตสวยงามหลากหลาย พร้อมกับ page builder ที่แข็งแกร่งและฟีเจอร์ขั้นสูง

### Elementor

[Elementor](https://elementor.com) เป็น page builder ที่ได้รับความนิยมสูงสุดในปัจจุบัน มีทั้งเวอร์ชันฟรีและโปร

### WPBakery

[WPBakery](https://wpbakery.com) เป็น page builder ขั้นสูงตัวแรกในตลาด

## Short codes

### [winklookup]

ทำงานร่วมกับ ranked content grid พิมพ์ชื่อสถานที่ที่คุณต้องการเยี่ยมชมและแสดงสินค้าคงคลัง

### [winksearch]

ปุ่มง่ายๆ สำหรับเปิดฟอร์มแผนการเดินทาง

### [winkaccount]

ปุ่มบัญชีเพื่อให้คุณยืนยันตัวตน เมื่อยืนยันแล้วจะเปลี่ยนเป็นเมนูแบบเลื่อนลงพร้อมตัวเลือกบัญชี

### [winkitinerary]

เหมือนกับปุ่มค้นหา แต่มีข้อมูลแผนการเดินทางเป็นข้อความบนปุ่ม

### [winkcontent]

ช่วยให้คุณฝังสินค้าคงคลังที่คุณเลือกใน Wink ลงในเพจหรือโพสต์โดยตรง shortcode นี้ต้องการ layout ID ด้วย

รูปแบบ

```
[winkcontent layoutid=""]
```

## การอ่านเพิ่มเติม

คุณสามารถหาข้อมูลเพิ่มเติมเกี่ยวกับปลั๊กอิน WordPress ของเราได้จากลิงก์ด้านล่าง

- เรียนรู้เกี่ยวกับ [Web Components](/th/developers/web-components)
- ติดตั้ง [Wink Affiliate WordPress plugin](https://wordpress.org/plugins/wink2travel/)
- Fork และพัฒนาต่อยอดจาก Github [Wink Affiliate WordPress repository](https://github.com/wink-travel/wink-wordpress-plugin)
