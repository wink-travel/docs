แอปพลิเคชันช่วยให้คุณเข้าถึงฟีเจอร์ต่างๆ บน Wink และ TripPay ได้อย่างเต็มที่ และให้ผู้ใช้ของคุณตรวจสอบและเข้าสู่ระบบไซต์ของคุณโดยใช้เซิร์ฟเวอร์การอนุญาต OAuth2 ของเรา เช่นเดียวกับ Facebook และ Google
คุณสามารถสร้าง OTA ของคุณเองได้ด้วย Wink โดยเริ่มต้นด้วยแอปพลิเคชัน

*โดยค่าเริ่มต้น คุณจะได้รับแอปเมื่อคุณลงทะเบียนบัญชีพันธมิตร Wink ใหม่หรือบัญชี TripPay เราใช้`client ID`สำหรับบางสถานการณ์*

แอปพลิเคชันที่สร้างบน Wink หรือ TripPay สามารถนำไปใช้กับจุดสิ้นสุด API ทั้งหมดได้
:::

*สำหรับตัวอย่างนี้ เราจะตรวจสอบความถูกต้องผ่าน[วิงค์ สตูดิโอ](https://studio.wink.travel)-*

<Steps>
  1. เมื่อผ่านการตรวจสอบแล้ว คลิกไอคอนโปรไฟล์ของคุณที่มุมขวาบน แล้วเมนูจะปรากฏขึ้น
  2. คลิกที่ `Applications` ลิงค์
  3. คุณจะถูกนำไปยังหน้าแอปของคุณ
</Steps>


## สร้างแอพ

เนื่องจากคุณไม่สามารถเข้าถึง`secret key`ในแอปที่เราสร้างให้คุณ คุณจะต้องสร้างแอปของคุณเอง

:::ข้อควรระวัง
คุณต้องสร้างบัญชีพันธมิตรบน Wink หรือบัญชีบน TripPay ก่อนที่จะสร้างแอป เนื่องจากคุณจะเชื่อมโยงแอปพลิเคชันของคุณกับบัญชีเฉพาะ
:::

ขั้นตอนการสร้างแอปพลิเคชันมีดังนี้:

<Steps>
  1. จากหน้าแอป ให้คลิก `Create new application` ปุ่ม.
  2. ***ชื่อ*** ตั้งชื่อให้กับแอปของคุณ *เช่น แอปเจ๋งๆ*
  3. **ชื่อนิติบุคคล** เชื่อมโยงแอปของคุณเข้ากับบัญชีที่มีอยู่ของคุณ *เช่น บัญชีสุดเจ๋ง*
  4. **เปลี่ยนเส้นทาง URIs** จำเป็นต้องมี URI การเปลี่ยนเส้นทางอย่างน้อยหนึ่งรายการ ซึ่งใช้สำหรับตรวจสอบโดเมนการเปลี่ยนเส้นทางหลังจากการตรวจสอบสิทธิ์สำเร็จ *เช่น https://www.cool-site.com*
  5. คลิกที่ `Save` ปุ่มเพื่อดำเนินการต่อ
</Steps>


คุณจะถูกส่งกลับไปยังรายการแอปพลิเคชันของคุณ คุณจะเห็นแอปพลิเคชันใหม่ของคุณในรายการและสามารถเข้าถึงรหัสลับของแอปพลิเคชันได้เพียงครั้งเดียวเท่านั้น โปรดคัดลอกและบันทึกไว้ในที่ปลอดภัย

## อัปเดตแอป

คุณจะต้องอัปเดตแอปของคุณหาก URI การเปลี่ยนเส้นทางของคุณมีการเปลี่ยนแปลง หรือคุณต้องการเพิ่มหรือลบ URI

หากต้องการอัปเดตแอปที่มีอยู่ ให้ทำดังต่อไปนี้:

<Steps>
  1. คลิกที่ `Actions` ลิงก์สำหรับแอปที่คุณต้องการอัปเดต
  2. คลิกที่ `Update` ปุ่มภายใต้การดำเนินการ
  3. ทำการเปลี่ยนแปลงของคุณในแอป
  4. คลิกที่ `Save` ปุ่มเพื่อดำเนินการต่อ
</Steps>


คุณจะถูกส่งกลับไปยังรายการแอปพลิเคชันของคุณ

## ลบแอปออก

:::ข้อควรระวัง
โปรดอย่าลบแอปที่เราสร้างให้คุณโดยอัตโนมัติ เพื่อให้แน่ใจว่าฟีเจอร์ทั้งหมดใช้งานได้กับบัญชีของคุณ
:::

หากคุณไม่ต้องการใช้แอปใดอีกต่อไป คุณสามารถลบแอปนั้นออกได้

หากต้องการลบแอป ให้ทำดังต่อไปนี้:

<Steps>
  1. คลิกที่ `Actions` ลิงก์สำหรับแอปที่คุณต้องการอัปเดต
  2. คลิกที่ `Remove` ปุ่มภายใต้การดำเนินการ
  3. คลิก `OK` ปุ่มเพื่อยืนยันการลบ
</Steps>


คุณจะถูกส่งกลับไปยังรายการแอปพลิเคชันของคุณ
