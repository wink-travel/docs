แอปพลิเคชันช่วยให้คุณเข้าถึงฟีเจอร์ต่างๆ บน Wink และ TripPay ได้อย่างเต็มที่ และช่วยให้ผู้ใช้ของคุณสามารถตรวจสอบสิทธิ์และเข้าสู่ระบบบนเว็บไซต์ของคุณโดยใช้เซิร์ฟเวอร์การอนุญาต OAuth2 ของเรา เช่นเดียวกับ Facebook และ Google  
คุณสามารถสร้าง OTA ของคุณเองด้วย Wink ได้ ซึ่งเริ่มต้นด้วยแอปพลิเคชัน

*โดยค่าเริ่มต้น คุณจะได้รับแอปเมื่อคุณลงทะเบียนบัญชีพันธมิตร Wink ใหม่หรือบัญชี TripPay เราใช้ `client ID` สำหรับบางสถานการณ์*

:::note
แอปพลิเคชันที่สร้างบน Wink หรือ TripPay สามารถใช้กับทุก API endpoint ได้
:::

*สำหรับตัวอย่างนี้ เราจะตรวจสอบสิทธิ์ผ่าน [Wink Studio](https://studio.wink.travel)*

<Steps>
1. เมื่อเข้าสู่ระบบแล้ว ให้คลิกไอคอนโปรไฟล์ของคุณที่มุมขวาบน แล้วเมนูจะปรากฏขึ้น
2. คลิกลิงก์ `Applications`
3. คุณจะถูกเปลี่ยนเส้นทางไปยังหน้ารายการแอปของคุณ
</Steps>

## สร้างแอป

เนื่องจากคุณไม่สามารถเข้าถึง `secret key` ของแอปที่เราสร้างให้คุณได้ คุณจึงต้องสร้างแอปของคุณเอง

:::caution
คุณต้องสร้างบัญชีพันธมิตรบน Wink หรือบัญชีบน TripPay ก่อนที่จะสร้างแอป เนื่องจากคุณจะผูกแอปพลิเคชันกับบัญชีเฉพาะ
:::

ขั้นตอนการสร้างแอปพลิเคชันมีดังนี้:

<Steps>
1. จากหน้ารายการแอป ให้คลิกปุ่ม `Create new application`
2. ***ชื่อ*** ตั้งชื่อแอปของคุณ *เช่น Cool App*
3. **ชื่อหน่วยงาน** ผูกแอปของคุณกับบัญชีที่มีอยู่ *เช่น Cool Account*
4. **Redirect URIs** ต้องมีอย่างน้อยหนึ่ง redirect URI ใช้สำหรับตรวจสอบโดเมน redirect หลังจากการตรวจสอบสิทธิ์สำเร็จ *เช่น https://www.cool-site.com*
5. คลิกปุ่ม `Save` เพื่อดำเนินการต่อ
</Steps>

คุณจะถูกเปลี่ยนเส้นทางกลับไปยังรายการแอปของคุณ คุณจะเห็นแอปใหม่ในรายการและสามารถเข้าถึง secret key ของแอปได้เพียงครั้งเดียวเท่านั้น อย่าลืมคัดลอกและเก็บไว้ในที่ปลอดภัย

## อัปเดตแอป

คุณจะต้องอัปเดตแอปของคุณหาก redirect URI ใดเปลี่ยนแปลง หรือคุณต้องการเพิ่มหรือลบ URI

ในการอัปเดตแอปที่มีอยู่ ให้ทำดังนี้:

<Steps>
1. คลิกลิงก์ `Actions` สำหรับแอปที่คุณต้องการอัปเดต
2. คลิกปุ่ม `Update` ภายใต้ Actions
3. แก้ไขแอปตามต้องการ
4. คลิกปุ่ม `Save` เพื่อดำเนินการต่อ
</Steps>

คุณจะถูกเปลี่ยนเส้นทางกลับไปยังรายการแอปของคุณ

## ลบแอป

:::caution
กรุณาอย่าลบแอปที่เราสร้างให้อัตโนมัติเพื่อให้แน่ใจว่าฟีเจอร์ทั้งหมดทำงานได้กับบัญชีของคุณ
:::

หากคุณไม่ต้องการใช้งานแอปใดอีกต่อไป คุณสามารถลบแอปนั้นได้

ในการลบแอป ให้ทำดังนี้:

<Steps>
1. คลิกลิงก์ `Actions` สำหรับแอปที่คุณต้องการลบ
2. คลิกปุ่ม `Remove` ภายใต้ Actions
3. คลิกปุ่ม `OK` เพื่อยืนยันการลบ
</Steps>

คุณจะถูกเปลี่ยนเส้นทางกลับไปยังรายการแอปของคุณ
