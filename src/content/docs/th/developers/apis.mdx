## Consume endpoints

Consume endpoints สำหรับนักพัฒนาที่ต้องการค้นหาสินค้าคงคลังการเดินทางที่มีอยู่และจองหรือใช้เพื่อโฆษณาผ่านบัญชีพันธมิตร Wink ของตน

### Configuration API

จุดเชื่อมต่อเดียวเพื่อดึงข้อมูล whitelabel และการปรับแต่งสำหรับเครื่องมือจอง

<LinkButton href="https://developer.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Lookup API

<LinkButton href="https://developer.wink.travel/lookup" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Inventory API

วิธีที่เป็นมิตรกับโปรแกรมเมอร์ในการดึงสินค้าคงคลังการเดินทางที่ทราบแล้วตามที่ระบุอย่างเป็นทางการโดยใช้ Lookup API

<LinkButton href="https://developer.wink.travel/inventory" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Booking API

## Produce endpoints

Produce endpoints สำหรับนักพัฒนาที่ต้องการสร้างและจัดการสินค้าคงคลังการเดินทาง

### Registration API

<LinkButton href="https://developer.wink.travel/extranet/property/register" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

Analytics API เปิดเผยจุดเชื่อมต่อสำหรับผู้ที่ต้องการข้อมูลเชิงลึกเกี่ยวกับการจัดอันดับกระดานผู้นำ รวมถึงข้อมูลเชิงลึกลึกเกี่ยวกับชุดข้อมูลตามช่วงเวลาในระดับแพลตฟอร์ม โดยมีเมตริกหลากหลายประเภท

## Payment API

Payment endpoints สำหรับนักพัฒนาที่ต้องการซื้อสินค้าคงคลังการเดินทาง สามารถทำได้ผ่าน API ในฐานะตัวแทนท่องเที่ยวที่ลงทะเบียน หรือใช้ API ร่วมกับวิดเจ็ตชำระเงินที่เป็นไปตามมาตรฐาน PCI สำหรับหน่วยงานอื่นๆ
Payment API เปิดเผยจุดเชื่อมต่อสำหรับพันธมิตรและโรงแรมในการติดตามการจอง การวิเคราะห์ และความพร้อมของเงินทุน พวกเขายังสามารถเลือกถอนเงินที่มีอยู่ไปยังบัญชีธนาคารของตนได้

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>
