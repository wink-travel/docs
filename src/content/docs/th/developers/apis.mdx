API REST ทั้งหมดของเราสามารถใช้งานได้กับทุกภาษาและเป็นไปตามมาตรฐาน Open API เริ่มทดสอบได้โดยไม่ต้องเขียนโค้ดแม้แต่บรรทัดเดียว

หากคุณยังไม่ได้สร้างบัญชี Wink ของคุณ, [เริ่มต้นที่นี่](/user/register)

Ping API เป็นจุดทดสอบอย่างรวดเร็วเพื่อยืนยันว่าข้อมูลรับรองของคุณใช้งานได้

</LinkButton>

<LinkButton href="https://developer.wink.travel/notifications" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

<LinkButton href="https://developer.wink.travel/user-settings" variant="secondary" icon="external" target="_blank">

</LinkButton>

## Consume endpoints

Consume endpoints สำหรับนักพัฒนาที่ต้องการค้นหาสินค้าท่องเที่ยวที่มีอยู่และจองหรือใช้เพื่อโฆษณาผ่านบัญชีพันธมิตร Wink ของตน

### Configuration API

จุดเชื่อมต่อเดียวเพื่อดึงข้อมูล whitelabel + การปรับแต่งสำหรับเครื่องมือจอง

<LinkButton href="https://developer.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Lookup API

วิธีที่เป็นมิตรกับโปรแกรมเมอร์ในการค้นหาสินค้าท่องเที่ยวเฉพาะทางในภูมิภาคต่าง ๆ

<LinkButton href="https://developer.wink.travel/lookup" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Inventory API

วิธีที่เป็นมิตรกับโปรแกรมเมอร์ในการดึงสินค้าท่องเที่ยวที่ทราบแล้วตามที่ระบุอย่างเป็นทางการโดยใช้ Lookup API

<LinkButton href="https://developer.wink.travel/inventory" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Booking API

วิธีที่เป็นมิตรกับโปรแกรมเมอร์ในการจองสินค้าที่พบในแพลตฟอร์มของเรา

<LinkButton href="https://developer.wink.travel/booking" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Travel Agent API

Travel Agent API เปิดเผยจุดเชื่อมต่อเพื่อจัดการการจองที่ดำเนินการโดยตัวแทนท่องเที่ยว

## Produce endpoints

Produce endpoints สำหรับนักพัฒนาที่ต้องการสร้างและจัดการสินค้าท่องเที่ยว

### Registration API

ในฐานะผู้ผลิต นี่มักเป็นจุดเริ่มต้นของการเดินทางของคุณ จุดเชื่อมต่อเหล่านี้ช่วยให้คุณสร้างที่พักบน Wink ได้

<LinkButton href="https://developer.wink.travel/extranet/property/register" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Property API

ชุดจุดเชื่อมต่อที่พักส่วนใหญ่เป็นจุดเชื่อมต่อสำหรับการจัดการที่ช่วยให้คุณแสดงผล, เปลี่ยนสถานะ และอื่น ๆ สำหรับที่พักที่มีอยู่ของคุณ

<LinkButton href="https://developer.wink.travel/extranet/property" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Facilities API

Facilities API เปิดเผยจุดเชื่อมต่อสำหรับการจัดการประสบการณ์บน Wink เช่น ประเภทห้องพัก

<LinkButton href="https://developer.wink.travel/extranet/facilities" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>

### Experiences API

Experiences API เปิดเผยจุดเชื่อมต่อสำหรับการจัดการประสบการณ์บน Wink เช่น กิจกรรมต่าง ๆ

<LinkButton href="https://developer.wink.travel/extranet/experiences" variant="secondary" icon="external" target="_blank">

</LinkButton>

Monetize API เปิดเผยจุดเชื่อมต่อสำหรับการจัดการนโยบายการยกเลิก, แผนราคา, โปรโมชั่น และอื่น ๆ บน Wink

<LinkButton href="https://developer.wink.travel/extranet/monetize" variant="secondary" icon="external" target="_blank">

<LinkButton href="https://developer.wink.travel/extranet/distribution" variant="secondary" icon="external" target="_blank">

## Analytics API

Analytics API เปิดเผยจุดเชื่อมต่อสำหรับผู้ที่ต้องการข้อมูลเชิงลึกเกี่ยวกับการจัดอันดับบนกระดานผู้นำ รวมถึงข้อมูลเชิงลึกลึกเกี่ยวกับชุดข้อมูลตามช่วงเวลาในระดับแพลตฟอร์ม โดยมีเมตริกหลากหลายประเภท

## Payment API

Payment endpoints สำหรับนักพัฒนาที่ต้องการซื้อสินค้าท่องเที่ยว สามารถทำได้ผ่าน API ในฐานะตัวแทนท่องเที่ยวที่ลงทะเบียน หรือใช้ API ร่วมกับวิดเจ็ตชำระเงินที่เป็นไปตามมาตรฐาน PCI สำหรับหน่วยงานอื่น ๆ
Payment API เปิดเผยจุดเชื่อมต่อสำหรับพันธมิตรและโรงแรมในการติดตามการจอง, การวิเคราะห์ และความพร้อมของเงินทุน พวกเขายังสามารถเลือกถอนเงินที่มีอยู่ไปยังบัญชีธนาคารของตนได้

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
เรียนรู้เพิ่มเติม
</LinkButton>
