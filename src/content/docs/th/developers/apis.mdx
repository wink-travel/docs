REST API ของเราทั้งหมดไม่ขึ้นกับภาษาและรองรับ Open API เริ่มทดสอบโดยไม่ต้องเขียนโค้ดแม้แต่บรรทัดเดียว

## ผลิตจุดสิ้นสุด

## API การชำระเงิน

จุดสิ้นสุดการชำระเงินมีไว้สำหรับนักพัฒนาที่ต้องการซื้อสินค้าคงคลังสำหรับการเดินทาง ซึ่งสามารถทำได้ผ่าน API ในฐานะตัวแทนการเดินทางที่ลงทะเบียนแล้ว หรือใช้ API ของเราร่วมกับวิดเจ็ตการชำระเงินที่สอดคล้องกับ PCI สำหรับหน่วยงานอื่นๆ ทั้งหมด
API การชำระเงินจะเปิดเผยจุดสิ้นสุดสำหรับบริษัทในเครือและโรงแรมเพื่อติดตามการจอง การวิเคราะห์ และความพร้อมของเงินทุน พวกเขายังสามารถเลือกที่จะถอนเงินที่มีอยู่ไปยังบัญชีธนาคารของพวกเขาได้อีกด้วย

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  เรียนรู้เพิ่มเติม
</LinkButton>
