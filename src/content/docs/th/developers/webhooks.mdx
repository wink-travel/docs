เว็บฮุกเป็นวิธีให้คุณรับและตอบสนองต่อเหตุการณ์เรียลไทม์จาก Wink และ TripPay ที่เกี่ยวข้องกับบัญชีของคุณ

## ตัวอย่าง

<Steps>
  1. การจองเกิดขึ้นบน Wink
  2. กิจกรรมเดอะวิงค์ `booking.created` ถูกส่งไปแล้ว
  3. ระบบของคุณกำลังรับฟังเหตุการณ์นี้อยู่
  4. ระบบของคุณติดตามชุดการวิเคราะห์ของคุณเอง และใช้เหตุการณ์นี้เพื่ออัปเดตฐานข้อมูลชุดเวลา
</Steps>

:::หมายเหตุ
เว็บฮุก Wink แตกต่างจากเว็บฮุก TripPay

* หากคุณต้องการฟังเหตุการณ์ Wink คุณสามารถสร้างเว็บฮุกได้จาก[ระบบเอ็กซ์ทราเน็ต](https://extranet.wink.travel) หรือ [สตูดิโอ](https://studio.wink.travel)-
* หากคุณต้องการฟังเหตุการณ์ TripPay คุณสามารถสร้างเว็บฮุกได้จาก[การชำระเงิน](https://pay.wink.travel)-

*สำหรับตัวอย่างนี้ เราจะตรวจสอบความถูกต้องผ่าน[สตูดิโอ](https://studio.wink.travel)-*

<Steps>
  1. เมื่อผ่านการตรวจสอบแล้ว คลิกไอคอนโปรไฟล์ของคุณที่มุมขวาบน แล้วเมนูจะปรากฏขึ้น
  2. คลิกที่ `Webhooks` ลิงค์
  3. คุณจะถูกนำไปยังหน้าเว็บฮุกของคุณ
</Steps>


## สร้างเว็บฮุก

:::ข้อควรระวัง
คุณต้องสร้างบัญชีพันธมิตรบน Wink หรือบัญชีบน TripPay ก่อนที่จะสร้างเว็บฮุก เนื่องจากคุณจะผูกเว็บฮุกของคุณกับบัญชีเฉพาะ
:::

ขั้นตอนในการสร้างเว็บฮุกมีดังต่อไปนี้:

<Steps>
  1. จากเพจเว็บฮุก ให้คลิก `Create new webhook` ปุ่ม.
  2. ***ชื่อ*** ตั้งชื่อให้กับเว็บฮุกของคุณ *เช่น Wink Stalker*
  3. ***เปิดใช้งาน*** ปล่อยไว้ตามสภาพจึงจะเปิดใช้งานได้
  4. **ชื่อนิติบุคคล** เชื่อมโยงเว็บฮุกของคุณเข้ากับบัญชีที่มีอยู่ของคุณ *เช่น บัญชีสุดเจ๋ง*
  5. **กิจกรรม** รับฟังเหตุการณ์ที่คุณสนใจ *เช่น `booking.created`*
  6. **URL เว็บฮุก** กรอก URL ที่คุณจะรับฟังกิจกรรมจากเรา *เช่น `https://my.system.com/webhook/wink`*
  7. คลิกที่ `Save` ปุ่มเพื่อดำเนินการต่อ
</Steps>


คุณจะถูกส่งกลับไปยังรายการเว็บฮุกของคุณ และเว็บฮุกใหม่ของคุณก็พร้อมใช้งานแล้ว

## อัพเดตเว็บฮุก

คุณจะต้องอัปเดตแอปของคุณหากคุณต้องการ:

* ปิดใช้งานเว็บฮุกของคุณ
* ฟังเหตุการณ์มากขึ้น/น้อยลง
* เปลี่ยน URL เว็บฮุกการฟังของคุณ

หากต้องการอัปเดตเว็บฮุกที่มีอยู่ ให้ทำดังต่อไปนี้:

<Steps>
  1. คลิกที่ `Actions` ลิงก์สำหรับเว็บฮุกที่คุณต้องการอัปเดต
  2. คลิกที่ `Update` ปุ่มภายใต้การดำเนินการ
  3. ทำการเปลี่ยนแปลงของคุณในเว็บฮุก
  4. คลิกที่ `Save` ปุ่มเพื่อดำเนินการต่อ
</Steps>


คุณจะถูกส่งกลับไปยังรายการเว็บฮุกของคุณ

## ลบเว็บฮุก

หากคุณไม่ต้องการใช้เว็บฮุกอีกต่อไป คุณสามารถลบออกได้

หากต้องการลบเว็บฮุก ให้ทำดังต่อไปนี้:

<Steps>
  1. คลิกที่ `Actions` ลิงก์สำหรับเว็บฮุกที่คุณต้องการอัปเดต
  2. คลิกที่ `Remove` ปุ่มภายใต้การดำเนินการ
  3. คลิก `OK` ปุ่มเพื่อยืนยันการลบ
</Steps>


คุณจะถูกส่งกลับไปยังรายการเว็บฮุกของคุณ
