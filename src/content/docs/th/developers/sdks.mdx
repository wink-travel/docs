เร่งเวลาออกสู่ตลาดด้วยการใช้ชุดพัฒนาซอฟต์แวร์ภาษาที่เราสนับสนุนอย่างเป็นทางการ
หากปัจจุบันเราไม่สนับสนุนภาษาที่คุณชอบ โปรดไปที่ API ที่คุณต้องการใช้และดาวน์โหลดข้อมูลจำเพาะของ OpenAPI ที่เราจัดเตรียมไว้สำหรับแต่ละ API ที่ด้านบนของหน้า คุณสามารถสร้าง SDK ของคุณเองได้โดยการฟอร์กคลัง SDK สาธารณะของเราหรือโดยใช้ตัวสร้าง OpenAPI

## ชวา

ไลบรารีนี้ได้รับการคอมไพล์สำหรับ Java 24 โดยได้รับการกำหนดค่าให้ใช้ WebClient แบบตอบสนอง และได้รับการเพิ่มประสิทธิภาพให้ใช้กับ Spring Framework อย่างไรก็ตาม Spring ไม่ใช่ข้อกำหนด แต่ช่วยให้คุณรองรับ OAuth2 ได้ทันที
