## Nível da plataforma

Todas as nossas aplicações são localizadas para suportar todos os idiomas mencionados acima. Cada aplicação possui um seletor de idioma na parte inferior da página onde o usuário pode escolher seu idioma desejado. O motor de reservas também suporta a passagem do idioma desejado via URL.

## Conteúdo gerado pelo usuário

O conteúdo gerado pelo usuário é localizado para suportar todos os idiomas mencionados acima. Os usuários podem optar por criar a versão em inglês do conteúdo e deixar o Wink cuidar do resto. Ou, podem gerenciar um conjunto específico de idiomas para maior controle e deixar o Wink lidar com o restante.

## Avisos

O idioma padrão é o inglês. A menos que especificado de outra forma, usaremos o idioma padrão primeiro quando um usuário utilizar qualquer uma de nossas aplicações. Não definimos o idioma ativo com base na localidade do navegador do usuário.
