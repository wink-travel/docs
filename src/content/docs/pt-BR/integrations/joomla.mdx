A **Extensão Wink para Joomla** é um pacote que instala um plugin de sistema (carrega scripts CDN), um plugin de conteúdo (processa tags shortcode em artigos) e um componente (página de configurações administrativas).

/>

## Requisitos

- Joomla 4 ou 5
- PHP 8.1+

## Instalação

<Steps>
1. Baixe o arquivo `pkg_wink-x.x.x.zip` mais recente na [página de releases do GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. No Joomla Admin, vá para **Extensões → Gerenciar → Instalar**.
3. Faça upload do arquivo zip e clique em **Upload & Instalar**.
4. Ambos os plugins são ativados automaticamente. Vá para **Componentes → Wink Settings**.
5. Insira seu **Client ID** e **Client Secret** e clique em **Salvar**.
   Acesse [Applications](/pt-BR/portal/settings/applications) para obter suas credenciais.
</Steps>

## Incorporar componentes em artigos

Use tags no estilo shortcode em qualquer lugar do conteúdo do artigo:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

O plugin de conteúdo substitui essas tags pelo HTML correto `<wink-*>` no momento da renderização. O plugin de sistema carrega automaticamente os scripts CDN e injeta `<wink-app-loader>` no rodapé da página.

## Shortcodes disponíveis

| Shortcode | Saída | Descrição |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Cartões de hotel, grades, mapas |
| `{wink-lookup}` | `<wink-lookup>` | Barra de busca de destinos |
| `{wink-search}` | `<wink-search-button>` | Botão seletor de itinerário |
| `{wink-account}` | `<wink-account-button>` | Login / conta do usuário |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Botão de status do itinerário |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Botão do carrinho |
