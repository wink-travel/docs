O **Wink Wix App** adiciona componentes web Wink ao seu site Wix como widgets nativos que você pode arrastar e soltar no Editor Wix. Não é necessário codificação para os proprietários do site.

<LinkCard
  title="wink-travel/wink-wix-app on GitHub"
  href="https://github.com/wink-travel/wink-wix-app"

/>

## Instalar

<Steps>
1. Vá para o [Wix App Market](https://www.wix.com/app-market) e pesquise por **Wink Travel**.
2. Clique em **Adicionar ao Site** e siga as instruções para autorizar o app.
3. Após a instalação, abra **Dashboard → Wink Settings** e insira seu **Client ID** e **Client Secret**.
   Vá para [Applications](/pt-BR/portal/settings/applications) para recuperar suas credenciais.
4. Clique em **Salvar** e depois em **Testar Conexão** para confirmar que o Wink está conectado.
</Steps>

## Adicionar um componente a uma página

<Steps>
1. Abra o **Editor Wix** e vá para a página onde deseja adicionar o conteúdo Wink.
2. Clique em **Adicionar (+) → Apps → Wink Travel** para ver todos os widgets disponíveis.
3. Arraste um widget para a página — por exemplo, **Wink Content** para exibir uma grade de hotéis.
4. No painel de configurações do widget, selecione o **Tipo de Layout** e cole o **Layout ID** do Wink Studio.
5. Clique em **Publicar** para tornar as alterações visíveis.
</Steps>

## Widgets disponíveis

| Widget | Propósito |
|---|---|
| **Wink Content** | Exibe cartões de hotéis, grades classificadas, mapas e mais |
| **Wink Lookup** | Barra de busca para destinos e hotéis |
| **Wink Search** | Botão ícone que abre o seletor de itinerário |
| **Wink Account** | Botão de login / conta do usuário |
| **Wink Itinerary** | Botão de itinerário mostrando a viagem atual |
| **Wink Cart** | Botão resumo do carrinho de compras |
