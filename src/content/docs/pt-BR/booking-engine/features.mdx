O mecanismo de reservas vem em duas variedades para dar suporte às diferentes necessidades de um afiliado, uma empresa de administração de propriedades e um hotel.

## Agência de viagens online

Para um afiliado, faz mais sentido usar a versão online para agências de viagens. Isso porque o site foi criado para acomodar viajantes que desejam visualizar várias propriedades.

Você pode encontrar a página inicial deste aplicativo abaixo.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>


## Motor de reservas pela internet

Para um hotel ou uma empresa de administração de imóveis, faz mais sentido usar a versão do mecanismo de reservas online. Esse tipo de usuário está interessado em destacar um único imóvel ou vários imóveis sob gestão.

## Configuração

Na maioria das vezes, não há necessidade de compartilhar nenhum desses aplicativos diretamente. Criamos diversas maneiras de compartilhar seu inventário de viagens, que você pode descobrir em[Estúdio Wink](/studio/what-is-studio)ou através[WinkLinks](/link-manager/wink-links).

No entanto, se você quiser criar um link manual para um desses sites, certifique-se de adicionar seu ID de cliente e ID de personalização ao URL como parâmetros.

Você pode encontrar seu ID de cliente por:

<Steps>
  1. Faça login em qualquer um dos [Extranet Wink](/extranet/what-is-extranet) ou [Estúdio Wink](/studio/what-is-studio).
  2. Clique no ícone do seu perfil no canto superior direito e clique em [Aplicações](/developers/applications).
  3. Use o aplicativo vinculado à conta Wink com a qual você deseja trabalhar.
  4. Copie o ID do Cliente desse aplicativo e cole-o na URL como um parâmetro de solicitação. Exemplo: ?client-id=MY\_CLIENT\_ID
</Steps>


### Extranet

Você pode encontrar seu ID de personalização como uma propriedade por:

<Steps>
  1. Entrar [Extranet Wink](/extranet/what-is-extranet).
  2. Selecione a propriedade com a qual deseja trabalhar.
  3. Na barra de navegação principal, clique em `Account > Branding`
  4. Copie o ID de personalização e cole-o na URL como parâmetro de solicitação. Exemplo: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Estúdio

Você pode encontrar seu ID de personalização como afiliado por:

<Steps>
  1. Entrar [Estúdio Wink](/studio/what-is-studio).
  2. Selecione a conta com a qual deseja trabalhar.
  3. Na barra de navegação principal, clique em `Tools > Customization`
  4. Copie o ID de personalização da personalização que deseja usar e cole-o na URL como um parâmetro de solicitação. Exemplo: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

O resultado é uma URL parecida com esta:

https://ota.wink.travel?client-id=MEU\_ID\_DO\_CLIENTE\&configuration-id=MEU\_ID\_DE\_PERSONALIZAÇÃO

Se você quisesse ir diretamente para a página de um hotel, o URL seria algo como isto:

https://ota.wink.travel/hotel/hotel-x?client-id=MEU\_ID\_DO\_CLIENTE\&configuration-id=MEU\_ID\_DE\_PERSONALIZAÇÃO

:::nota
Se você não incluir um ID de cliente e/ou de configuração, o Wink OTA usará a configuração padrão.
:::

## IBE

Se você é uma empresa de administração de imóveis, você deseja criar um link para a página inicial do Wink IBE. A página inicial exibirá o tema escolhido junto com o[propriedades que você representa](/studio/agency-representation).

O resultado é uma URL parecida com esta:

https://book.wink.travel?client-id=MEU\_ID\_DO\_CLIENTE\&configuration-id=MEU\_ID\_DE\_PERSONALIZAÇÃO

Se você for um proprietário de imóvel, você deseja criar um link direto para a página inicial do imóvel do Wink IBE.

Essa URL seria algo como isto:

https://book.wink.travel/hotel/hotel-x?client-id=MEU\_ID\_DO\_CLIENTE\&configuration-id=MEU\_ID\_DE\_PERSONALIZAÇÃO

:::nota
Se você não incluir um ID de cliente e/ou de configuração, o Wink IBE NÃO será carregado. Estes parâmetros são obrigatórios.
:::
