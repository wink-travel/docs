Para gerenciar suas tags do WinkLinks, navegue até `WinkLinks` na barra de navegação principal e clique na aba `Tags`.

À medida que seu conteúdo cresce, você vai querer começar a organizar parte dele. Criamos as `tags` para esse propósito. Você pode agrupar conteúdo usando quantas tags quiser.

## Criar tag

<Steps>
1. Crie uma nova tag clicando no botão `Adicionar uma nova tag`.
2. Insira o nome da sua nova tag. *ex: Melhores cafeterias em Berlim*
3. Clique no link `Salvar`.
</Steps>

## Atualizar tag

<Steps>
1. Comece clicando no link `Editar` ao lado da tag que deseja modificar.
2. Atualize o nome da sua tag. *ex: Melhores cafeterias em Berlim*
3. Clique no link `Salvar`.
</Steps>

## Remover tag

:::note
Ao remover uma tag, você também a removerá de todo o conteúdo associado.
:::

<Steps>
1. Clique no link `Remover` ao lado da tag que deseja excluir.
</Steps>

## API

Desenvolvedores que desejam gerenciar o `WinkLinks` podem acessar [Developers > API > WinkLinks](/pt-BR/developers/apis/#winklinks-api).
