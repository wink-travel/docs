O editor de publicações permite criar publicações manuais ilimitadas e também é o local onde você gerencia publicações que foram criadas automaticamente para você.

O editor possui 4 seções:

1. A seção de conteúdo.
2. A seção de mídia.
3. A seção de link.
4. A seção de configurações.

## Conteúdo

A seção de conteúdo permite modificar o texto antes de ser publicado.

## Mídia

A seção de mídia permite modificar as imagens e vídeos que acompanham o texto.

## Link

A seção de link permite adicionar um link para onde você deseja que o usuário vá após ler sua publicação. *Por exemplo, pode ser um link para uma oferta especial ou um local para obter mais informações.*

## Configurações

A seção de configurações permite modificar/substituir todas as suas configurações padrão. Pode ser útil quando você deseja que a publicação seja feita em um horário diferente. Na maioria das vezes, esta seção não será usada.

## Aprovação de publicações

As publicações são aprovadas na grade de publicações, onde você pode selecionar uma ou mais publicações e escolher aprovar ou rejeitar uma publicação.

:::note
Haverá uma entrada para cada idioma e para cada rede social.

Por exemplo, se você tiver 3 redes sociais e 3 idiomas selecionados, verá 9 entradas no calendário.
:::
