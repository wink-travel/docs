A publicação funciona assim:

<Steps>
1. O conteúdo é gerado a partir das suas preferências uma semana antes do mês de publicação.
2. Você recebe um e-mail solicitando a aprovação do conteúdo recém-criado [ou notificando que o conteúdo foi criado para o mês seguinte e que não há mais nada a fazer porque você escolheu aprovar automaticamente o conteúdo].
3. Você faz login em [social.wink.travel](https://social.wink.travel) para aprovar as publicações.
4. As publicações são agendadas para serem publicadas. (Você ainda pode interromper o trabalho de publicação se fizer isso antes da publicação. Caso contrário, terá que acessar a rede social e excluir ou modificar a publicação manualmente lá.)
5. As publicações são enviadas no momento mais oportuno para o seu fuso horário selecionado.
</Steps>

Uma vez que o conteúdo é aprovado para publicação, ele é enviado para o seu calendário.

Você pode visualizar suas publicações agendadas em três visualizações diferentes:

- **Mês** — Veja todas as publicações do mês inteiro de relance.
- **Semana** — Foque em uma única semana de publicações agendadas.
- **Dia** — Aprofunde-se na programação de um dia específico.

:::note
Haverá uma entrada para cada idioma e para cada rede social.

Por exemplo, se você tiver 3 redes sociais e 3 idiomas selecionados, verá 9 entradas no calendário.
:::
