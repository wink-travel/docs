# Configurações

Quando você começar a usar o Social, perguntaremos como deseja que criemos seu conteúdo. Essas opções estarão disponíveis a qualquer momento para que você possa alterá-las facilmente depois.

Navegue até `Preferências` no menu lateral para gerenciar suas configurações.

A seção de preferências contém suas configurações básicas de geração de conteúdo:

1. **Personalização preferida** Escolha qual personalização usar ao gerar conteúdo. As personalizações controlam cores do tema, itinerário padrão, moeda, idioma e mais. Você cria personalizações na sua conta em [my.wink.travel](https://my.wink.travel).
2. **Aprovação automática** Quando ativada, as postagens geradas são publicadas automaticamente sem precisar da sua aprovação. Ative isso quando estiver confiante na qualidade do conteúdo que produzimos para você.
3. **Nome do aprovador** O nome da pessoa ou equipe que receberá os e-mails de solicitação de aprovação.
4. **E-mail do aprovador** O endereço de e-mail para onde serão enviadas as solicitações de aprovação para as próximas postagens.

## Criadores de conteúdo

<CldImage

O Social oferece muitos criadores de conteúdo para escolher. Cada criador foca em uma forma específica de criar conteúdo único para sua propriedade usando IA. Cada criador de conteúdo tem um atributo **Peso** — um peso maior aumenta a chance de que esse criador seja selecionado ao gerar postagens.

Aqui estão todos os criadores de conteúdo disponíveis:

| Criador | Descrição |
| -- | -- |
| **Destino** | Usa sua localização para criar textos úteis sobre a região |
| **Tipo de quarto** | Cria uma postagem reservável com um dos seus tipos de quarto, com texto gerado por IA e imagens do tipo de quarto |
| **Atividade** | Cria uma postagem reservável com uma das suas atividades |
| **Complemento** | Cria uma postagem reservável com um dos seus complementos |
| **Atração** | Cria uma postagem reservável com uma das suas atrações |
| **Local** | Cria uma postagem reservável com um dos seus locais |
| **Sala de reunião** | Cria uma postagem reservável com uma das suas salas de reunião |
| **Restaurante** | Cria uma postagem reservável com um dos seus restaurantes |
| **Spa** | Cria uma postagem reservável com um dos seus spas |
| **Propriedade** | Cria uma postagem reservável com seu quarto com melhor preço, com texto gerado por IA e imagens da propriedade |
| **Sentimento do usuário** | Agrega avaliações de hóspedes da web em uma postagem baseada em sentimento que destaca o que os hóspedes amam na sua propriedade |
| **Engraçado** | Cria postagens leves e divertidas para manter seu feed envolvente |

:::tip[Dica profissional]
Certifique-se de ter o tipo de inventário necessário para os criadores de conteúdo escolhidos. Por exemplo, se selecionar "Restaurante", você precisa ter pelo menos um restaurante listado na sua propriedade.
:::

## Tom de voz

<CldImage
  src="academy/social_personas"

O tom de voz define o estilo de linguagem que a IA usa ao gerar conteúdo. Escolha os tons que combinam com a personalidade da sua marca. Assim como os criadores de conteúdo, cada tom tem um **Peso** para controlar a frequência de uso.

Aqui estão os tons disponíveis:

| Tom | Estilo |
| -- | -- |
| **Neutro** | Calmo, autoritário, factual. Sem exageros, sem piadas — apenas conhecimento confiável e bem estruturado |
| **Amigável** | Caloroso, acessível, energia sorridente no texto com contrações casuais e emojis ocasionais |
| **Luxo** | Sofisticado, refinado. Palavras como "exclusivo" e "sob medida" que fazem o leitor se sentir especial |
| **Gastronômico** | Descrições sensuais e de dar água na boca. Linguagem de food porn que deixa o leitor com fome |
| **Aventureiro** | Ousado, intrépido, energia exploradora. Descrições sensoriais vívidas que fazem tudo parecer épico |
| **Familiar** | Excitação saudável e amigável para crianças. Energia calorosa e parental sobre experiências seguras e divertidas |
| **Romântico** | Poético, nostálgico, que faz o coração bater mais forte. Pôr do sol, linguagem suave e amor |
| **Humorístico** | Sarcástico, frases rápidas, trocadilhos inteligentes. Lê como um show de stand-up |
| **Ecológico** | Sincero e justo. Celebra alternativas verdes e escolhas sustentáveis |
| **Insider local** | Referências locais hiperespecíficas, alerta contra armadilhas para turistas, usa gírias locais |
| **Inclusivo** | Acolhedor, celebrativo e afirmativo. Faz todos se sentirem vistos e valorizados |
| **Místico** | Etéreo, espiritual, verdades cósmicas. Convites suaves para olhar para dentro |
| **Futurista** | Elegante, visionário. Adora termos como "mudança de paradigma" e visões otimistas do futuro |
| **Econômico** | Prático, engenhoso, orgulhosamente econômico. Celebra achados acessíveis e experiências gratuitas |

## Tipos de postagem

Os tipos de postagem determinam o objetivo e a estrutura do conteúdo gerado. Cada tipo de postagem tem um **Peso** para controlar a frequência de uso.

| Tipo de postagem | Descrição |
| -- | -- |
| **Transacional** | Claro, conciso, focado em conversão. Impulsiona reservas e ações diretas com fatos, preços e próximos passos |
| **Conscientização** | Conteúdo para construção de marca, projetado para apresentar sua propriedade e aumentar a visibilidade entre potenciais hóspedes |
| **Sentimento** | Conteúdo emocionalmente ressonante que cria associações positivas e constrói uma conexão emocional com seu público |

Você pode querer focar seus esforços de marketing em um idioma e região específicos. Informe os idiomas para os quais deseja criar conteúdo e o fuso horário de interesse, e faremos o resto. As postagens são publicadas no horário agendado no fuso horário selecionado.

*A maioria das propriedades usa um único idioma e o padrão é o idioma local.*

## Google Business

<CldImage

  <figcaption>*Fig. 5 - Google Business*</figcaption>

Ao conectar seu Perfil Comercial do Google, podemos obter detalhes mais ricos da propriedade para um conteúdo mais eficaz. Sua propriedade precisa ter um Perfil Comercial do Google verificado para que possamos gerar conteúdo a partir do seu anúncio. Alguns dos nossos criadores de conteúdo **exigem** que você tenha uma conta Google Business.

:::tip[Dica profissional]
Comece pequeno

Não seja muito ambicioso na sua primeira tentativa. Recomendamos que você comece criando conteúdo para sua propriedade em um único idioma.
:::
