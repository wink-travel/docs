Você já está conectado ao WinkLinks, nossa ferramenta de gerenciamento de links. Conecte-se a mais redes sociais clicando em `Conectar uma rede social`.
O processo irá redirecioná-lo para sua rede social para aprovar o aplicativo `Wink Social` para publicar conteúdo em seu nome.

Uma vez conectado, começamos a gerar conteúdo para essa rede para o próximo mês agendado.

:::note
Criamos e publicamos conteúdo otimizado para cada rede social.
:::
