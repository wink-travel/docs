O Studio oferece vários widgets incorporáveis que você pode adicionar ao seu próprio site para aumentar as reservas.

Para acessar os widgets incorporáveis, navegue até `Tools > Miscellaneous` na barra lateral.

## Widgets disponíveis

| Widget | Descrição |
| -- | -- |
| **Account button** | Um botão com a marca que liga à sua página de afiliado |
| **Itinerary button** | Um botão que abre o seletor de itinerário |
| **Itinerary form** | Um formulário embutido para seleção de datas e hóspedes |
| **Search button** | Um botão que abre a busca de propriedades |
| **Lookup button** | Um botão para consultar inventário específico |

## Como incorporar

Cada widget fornece um trecho de código que você pode copiar e colar no HTML do seu site. O trecho inclui seu **Client ID** e **Customization ID** para que o widget seja automaticamente vinculado à sua conta de afiliado.

:::note
Se você usa WordPress, também oferecemos um plugin para WordPress para facilitar a integração.
:::
