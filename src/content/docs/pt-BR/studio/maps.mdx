O restante deste artigo explica como criar, personalizar e compartilhar seus mapas.

## Criar um mapa

Os mapas podem ser criados de duas maneiras:

* **Mapa de marcador único**: Crie um mapa a partir de algo que você encontrou enquanto[procurando](/studio/search).
* **Mapa de marcadores múltiplos**: Crie um mapa a partir de um[lista com curadoria](/studio/curated-lists) ou [pesquisa salva](/studio/saved-searches).

### Mapa de marcador único

Acima está uma imagem tirada de [Procurar](/studio/search) e mostra algumas coisas que você pode fazer com os resultados da sua pesquisa. Uma dessas ações é`Make a map`.
Clique nesse botão e você será redirecionado para nossa página de formulário de mapa, onde poderá começar a personalizá-lo.

### Mapa de marcadores múltiplos

Ambos [listas selecionadas](/studio/curated-lists#list-actions) e [pesquisas salvas](/studio/saved-searches#list-actions) vêm com uma ação que permite criar um mapa. Este mapa conterá marcadores para todo o inventário dentro dessas listas.

## Personalizar mapa

O formulário permite que você personalize seu mapa das seguintes maneiras:

<Steps>
  1. Dê um nome para que você possa lembrar do que se trata o mapa.
  2. Escolha se você deseja que o usuário tenha permissão para se movimentar pelo mapa.
  3. Defina a altura, em pixels, que você deseja que o mapa tenha.
  4. Escolha se você deseja que o usuário tenha permissão para aumentar ou diminuir o zoom no mapa.
  5. Escolha um estilo de mapa que combine com o site no qual você irá incorporar o mapa.
  6. Escolha uma cor de marcador.
  7. Selecione a face inicial da carta que você deseja que os usuários vejam primeiro. Por padrão, é a face nativa do inventário.
  8. Você pode adicionar seus próprios marcadores, círculos, retângulos e polígonos para indicar outros locais no mapa que seus usuários possam achar interessantes.
  9. Clique no `Save` botão para continuar.
</Steps>


Depois de salvar seu mapa, você será redirecionado para sua página de mapas e seu mapa estará pronto para ser compartilhado com o mundo.

Abordaremos algumas dessas opções com mais detalhes abaixo.

### Incorporar

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Veja como incorporar seu mapa em seu site:

* A linha 3 mostra como incorporar os estilos Wink no seu site.
* As linhas 6 a 9 mostram como usar o [carregador de conteúdo piscante](/developers/web-components/#content-loader) Componente Web e diga para ele buscar um banner de anúncio para seu código.
* A linha 11 mostra como incorporar nosso Javascript em seu site.
* A linha 13 mostra como incorporar o [carregador de aplicativo wink](/developers/web-components/#application-loader) Componente Web e peça para ele buscar suas preferências de configuração no nível da página.
