Listas selecionadas são um recurso que permite agrupar e classificar o inventário selecionado a partir de itens encontrados durante a busca por inventário de viagens.
Após a criação, você pode expor a lista aos seus usuários, disponibilizando-a para venda como um[grade](/studio/grids)ou um[mapa](/studio/maps) através [WinkLinks](/link-manager/wink-links), nosso [Plugin do WordPress](/developers/wordpress/)ou incorporando um dos nossos[Componentes da Web](/developers/web-components)no seu site.

Por padrão, cada conta de afiliado começa com um`Favorites`lista que você pode usar imediatamente.`Favorites`é especial e não pode ser removido.
Todas as listas selecionadas permitem que você faça o seguinte:

## Criar uma lista

<figure>
  <CloudinaryImage width="350" height="219" resize="scale" quality="best" alt="Criar lista com curadoria" src="academy/create-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Criar um novo formulário de lista com curadoria*</figcaption>
</figure>


* No painel, clique em`Inventory > Curated Lists`na barra de navegação.
* Crie uma nova lista selecionada clicando em`Make a new list` botão.
* Você será solicitado a nomear sua lista.*Exemplo: As 3 principais escolhas de Nova York*.
* Clique no`Save` botão.

## Adicionar um item de lista

Você pode adicionar um item de lista a qualquer lista dos resultados da pesquisa.

* Vá para [Procurar](/studio/search)
* Procure algo que você queira adicionar à sua lista.
* Depois de encontrá-lo, clique em`Actions`link no canto inferior esquerdo do cartão de resultados da pesquisa.
* Clique`Add to...`e então selecione a lista selecionada à qual você deseja adicionar esse inventário.
* Volte para a página de listas selecionadas para verificar se o item foi adicionado à sua lista.

## Listar ações

<figure>
  <CloudinaryImage width="350" height="183" resize="scale" quality="best" alt="Ações de lista com curadoria" src="academy/curated-list-actions" errorImage="noimage_opaque_nyrtl0.png" />

Cada lista tem algumas ações que você pode executar nela.

* Converta a lista em um[mapa](/studio/maps).
* Converta a lista em um[grade](/studio/grids).
* Remover a lista (*exceto para Favoritos*).

## API

Os desenvolvedores que desejam gerenciar listas selecionadas podem acessar[Desenvolvedores > API > Navegar](/developers/apis/#browse-api).
