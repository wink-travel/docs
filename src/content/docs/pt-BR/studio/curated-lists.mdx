Listas curadas é um recurso que permite agrupar e ordenar inventário que você selecionou a partir de itens encontrados durante a busca por inventário de viagem.
Uma vez criada, você pode expor a lista para seus usuários tornando-a disponível para venda como um [grid](/pt-BR/studio/grids) ou um [map](/pt-BR/studio/maps) através do [WinkLinks](/pt-BR/link-manager/wink-links), nosso [plugin para WordPress](/pt-BR/developers/wordpress/) ou incorporando um dos nossos [Web Components](/pt-BR/developers/web-components) em seu site.

Por padrão, toda conta afiliada começa com uma lista `Favoritos` que você pode usar imediatamente. `Favoritos` é especial e não pode ser removida.
Todas as listas curadas permitem que você faça o seguinte:

## Criar uma lista

- No painel, clique em `Inventário > Listas Curadas` na barra de navegação.
- Crie uma nova lista curada clicando no botão `Criar uma nova lista`.
- Você será solicitado a nomear sua lista. *Exemplo: Top 3 Escolhas em NYC*.
- Clique no botão `Salvar`.

## Adicionar um item à lista

Você pode adicionar um item a qualquer lista a partir dos resultados da busca.

- Vá para [Busca](/pt-BR/studio/search)
- Procure algo que deseja adicionar à sua lista.
- Quando encontrar, clique no link `Ações` no canto inferior esquerdo do cartão do resultado da busca.
- Clique em `Adicionar a...` e selecione a lista curada à qual deseja adicionar este inventário.
- Volte para a página de listas curadas para verificar se o item foi adicionado à sua lista.

Cada lista possui algumas ações que você pode executar nela.

- Converter a lista para um [mapa](/pt-BR/studio/maps).
- Converter a lista para um [grid](/pt-BR/studio/grids).
- Remover a lista (*exceto Favoritos*).

## API

Desenvolvedores que desejam gerenciar listas curadas podem acessar [Desenvolvedores > API > Navegar](/pt-BR/developers/apis/#browse-api).
