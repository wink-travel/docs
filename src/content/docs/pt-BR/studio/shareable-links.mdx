Links compartilháveis são um recurso central do Wink e surgiram da necessidade de comunicar inventário e preços de forma dinâmica pelo meio mais simples possível na Internet; ou seja, o hiperlink.

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

Acima está um exemplo de um dos nossos links. É curto e portátil... até curto o suficiente para lembrar. Embora curto, este link contém uma quantidade **explosiva** de informações, tais como:

- Seu App ID.
- Seu Affiliate ID.
- A(s) imagem(ns) e texto a serem exibidos para o usuário.
- O visual (*ou seja, tema, cores e mais*).
- O itinerário a ser usado para nossa consulta de preço (*2 adultos, 1 noite*).
- A moeda para exibir o preço.
- O idioma para exibir o texto.

O restante deste artigo guia você sobre como criar, personalizar e compartilhar seus links.

Clique nesse botão e você será redirecionado para a página do formulário de link compartilhável, onde poderá começar a personalizar seu link.

## Personalizar link

 O formulário permite que você personalize seu link das seguintes maneiras:

<Steps>
 1. Dê um nome para lembrar do que se trata o link.
 2. Selecione a [Personalização](/pt-BR/studio/customization) que deseja aplicar a este link.
 3. Adicione palavras-chave, separadas por vírgula, que serão usadas por [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Opcionalmente, insira seu identificador da conta X ou Facebook App ID.
 5. Adicione títulos e descrições nos idiomas que deseja que o usuário veja.
 6. Selecione uma ou mais imagens que serão exibidas quando você compartilhar o link em plataformas como Facebook ou LinkedIn.
 7. Escolha um tema / sobreposição que adiciona um design com a marca sobre a(s) imagem(ns) selecionada(s). *Nota: É assim que você faz os preços aparecerem sobre a imagem.*
 8. Clique no botão `Salvar` para continuar.
 </Steps>

Acima você vê uma imagem com todas as ações disponíveis para seu link:

- **Visualizar** Mostra uma visão rápida de como o link foi configurado e como ele aparecerá para o usuário.
- **Atualizar** Atualiza a configuração do seu link.
- **Adicionar ao WinkLinks** Adiciona o link à sua conta WinkLinks.
- **Copiar** Copia o link para sua área de transferência para que você possa compartilhá-lo facilmente onde quiser (*incluindo [WinkLinks](/pt-BR/link-manager/wink-links) *).
- **Compartilhar** Abre uma nova janela com opções para compartilhar em várias redes sociais e plataformas de mensagens.
- **Código QR** Gera um código QR que você pode compartilhar em plataformas como IG, que são mais focadas em imagens.
- **Incorporar** Mostra como incorporar este link como um [Banner de Anúncio](/pt-BR/developers/web-components/#content-loader) em seu site.
- **Usar com WordPress** Mostra como usar nosso [plugin do WordPress](/pt-BR/developers/wordpress) para incorporar este link em seu site.

Se preferir compartilhar um código QR em vez de um link, clique no botão `Código QR` e uma nova janela aparecerá com seu código QR.

Como salvar seu código QR:

- Se estiver no seu laptop, clique com o botão direito na imagem e selecione `Salvar imagem`.
- Se estiver no seu celular:
    - Pressione e segure o dedo sobre a imagem até aparecer a opção de salvar a imagem nas Fotos.
    - Tire uma captura de tela do código QR, recorte para ajustar e salve nas Fotos.

### Incorporar

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Aqui está como incorporar seu link, como um banner de anúncio, em seu site:

- A linha 3 mostra como incorporar os estilos do Wink em seu site.
- As linhas 6 a 9 mostram como usar o [wink-content-loader](/pt-BR/developers/web-components/#content-loader) Web Component e instruí-lo a buscar um banner de anúncio para seu código.
- A linha 11 mostra como incorporar nosso Javascript em seu site.
- A linha 13 mostra como incorporar o [wink-app-loader](/pt-BR/developers/web-components/#application-loader) Web Component e instruí-lo a buscar suas preferências de configuração em nível de página.
