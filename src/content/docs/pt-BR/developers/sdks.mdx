Acelere o tempo de lançamento no mercado usando um dos nossos Kits de Desenvolvimento de Software para a linguagem oficialmente suportada.
Se atualmente não oferecemos suporte à sua linguagem favorita, acesse a API que deseja usar e baixe a especificação OpenAPI que disponibilizamos para cada API no topo da página. Você pode gerar seu próprio SDK bifurcando um de nossos repositórios públicos de SDK ou usando um dos geradores OpenAPI.

## Java

Esta biblioteca foi compilada para Java 24. Ela está configurada para usar o WebClient reativo e otimizada para uso com o Spring Framework. No entanto, o Spring não é um requisito, mas oferece suporte OAuth2 pronto para uso.
