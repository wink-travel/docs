Acelere o time-to-market usando um dos nossos Kits de Desenvolvimento de Software oficialmente suportados para linguagens.
Se atualmente não suportamos sua linguagem favorita, acesse a API que deseja usar e baixe a especificação OpenAPI que disponibilizamos para cada API no topo da página. Você pode gerar seu próprio SDK seja fazendo fork de um dos nossos repositórios públicos de SDK ou usando um dos geradores do OpenAPI.

## Java

Esta biblioteca foi compilada para Java 24. Está configurada para usar o WebClient reativo e é otimizada para uso com o Spring Framework. No entanto, o Spring não é um requisito, mas oferece suporte a OAuth2 pronto para uso.
