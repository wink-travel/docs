## API de pagamento

Os endpoints de pagamento são para desenvolvedores que desejam comprar inventário de viagens. Isso pode ser feito por meio da API como um Agente de Viagens registrado ou usando nossa API em conjunto com nosso widget de pagamento compatível com PCI para todas as outras entidades.
A API de Pagamento expõe endpoints para que afiliados e hotéis acompanhem reservas, análises e disponibilidade de fundos. Eles também podem optar por sacar os fundos disponíveis para suas contas bancárias.

  Saber mais
</LinkButton>
