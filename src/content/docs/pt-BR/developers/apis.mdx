## Payment API

Os endpoints Payment são para desenvolvedores que querem comprar inventário de viagem. Isso pode ser feito via API como um Agente de Viagens registrado ou usando nossa API em conjunto com nosso widget de pagamento compatível com PCI para todas as outras entidades.
A Payment API expõe endpoints para afiliados e hotéis acompanharem reservas, análises e disponibilidade de fundos. Eles também podem optar por sacar fundos disponíveis para sua conta bancária.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Saiba mais
</LinkButton>
