Esta página lista todos os eventos webhook aos quais você pode se inscrever na Wink. Para instruções de configuração, veja o [Guia de Integração Webhook](/pt-BR/guides/webhook-integration) ou a [página de gerenciamento de Webhooks](/pt-BR/my-account/webhooks).

## Como se inscrever

1. Navegue até **Aplicações > Webhooks** no seu portal.
2. Crie um webhook e selecione os eventos que deseja receber.
3. A Wink envia um HTTP POST para seu endpoint toda vez que o evento ocorre.

Para passos completos de configuração e melhores práticas, veja [Integração Webhook](/pt-BR/guides/webhook-integration).

## Eventos de conta

Eventos relacionados ao ciclo de vida e configurações da sua conta TripPay.

| Evento | Chave | Descrição |
|---|---|---|
| Conta criada | `account.create` | Uma nova conta foi criada. |
| Conta em revisão | `account.under_review` | Conta está pendente de revisão. |
| Conta atualizada | `account.updated` | Detalhes da conta foram alterados. |
| Status da conta atualizado | `account.status.updated` | O status da conta foi alterado (ex: aprovado, suspenso). |
| Endereço da conta atualizado | `account.address.updated` | O endereço da conta foi modificado. |
| Detalhes da conta atualizados | `account.details.updated` | Campos de detalhes da conta foram alterados. |
| Informações da conta atualizadas | `account.information.update` | Informações gerais da conta foram atualizadas. |
| Gerentes da conta atualizados | `account.managers.update` | A lista de gerentes da conta foi alterada. |
| Foto de perfil atualizada | `account.profile_picture.updated` | A foto de perfil da conta foi alterada. |
| Agente atualizado | `account.agent.updated` | Informações do agente na conta foram modificadas. |
| Nome da URL atualizado | `account.url-name.update` | A URL personalizada da conta foi alterada. |
| ID único atualizado | `account.unique-id.update` | O identificador único da conta foi modificado. |
