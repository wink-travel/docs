Para qualquer pessoa que use o WordPress para seus negócios ou blogs, recomendamos fortemente o uso do[Plugin WordPress de afiliados Wink](https://wordpress.org/plugins/wink2travel/).
O plugin oferece acesso fácil aos seus cartões Wink, grades, mapas etc., e torna tudo tão simples quanto arrastar conteúdo para suas postagens e páginas usando seu construtor de páginas favorito ou nossos códigos de acesso personalizados.

## Instalar

É assim que você instala o`Wink Affiliate WordPress Plugin`no seu site WordPress.

<Steps>
  1. Quando você estiver conectado como administrador e no Painel.
  2. Clique `Plugins > Add New Plugin`.
  3. Procurar `Wink Affiliate WordPress Plugin`.
  4. Clique `Install` para baixar e instalar o plugin no seu site.
  5. Clique `Activate` para ativar o plugin no seu site.
  6. A ativação encaminhará para você o `Wink Settings` página. *Você também pode ir até lá navegando até `Appearance > Customize > Wink Settings`.*
  7. Ele irá pedir seu `Client ID` e `Client Secret`. Vá para [Aplicações](/developers/applications) para saber como recuperar suas credenciais.
  8. Insira suas credenciais e clique `Publish`. Agora você está conectado à sua conta Wink.
</Steps>

## Construtores de páginas

Oferecemos suporte aos criadores de páginas mais populares no WordPress, o que facilita o uso do Wink para qualquer pessoa que queira usá-lo.

### Gutenberg

Gutenberg é o editor nativo do WordPress.

### Avada

[Avada](https://avada.com)é famoso por sua grande seleção de belos modelos, além de um construtor de páginas robusto com um conjunto de recursos avançados.

### Elementor

[Elementor](https://elementor.com)é o construtor de páginas mais popular \[atualmente], com versões gratuitas e profissionais disponíveis.

### Padaria WP

[Padaria WP](https://wpbakery.com)foi o primeiro construtor de páginas avançado do mercado.

## Códigos curtos

### \[piscadela]

Funciona com grade de conteúdo classificado. Digite um lugar que você deseja visitar e exiba o inventário.

### \[pesquisa de piscadela]

Botão simples para abrir o formulário de itinerário.

### \[conta piscadinha]

Botão Conta para autenticação. Após a autenticação, o botão abre um menu suspenso com opções de conta.

### \[winkitinerário]

Igual ao botão de pesquisa, só que contém informações do itinerário como texto do botão.

### \[piscadela de conteúdo]

Permite incorporar o inventário selecionado no Wink diretamente em uma página ou publicação. Este shortcode também espera um ID de layout.

O layout

```
[winkcontent layoutid=""]
```
