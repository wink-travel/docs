Para quem usa WordPress para seu negócio ou blog, recomendamos fortemente o uso do [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
O plugin oferece acesso fácil aos seus cartões, grades, mapas Wink etc., e torna simples arrastar conteúdo para suas postagens e páginas usando seu construtor de páginas favorito ou nossos shortcodes personalizados.

##  Instalação

Veja como instalar o `Wink Affiliate WordPress Plugin` no seu site WordPress.

<Steps>
1. Quando estiver logado como administrador e no Painel.
2. Clique em `Plugins > Adicionar Novo Plugin`.
3. Procure por `Wink Affiliate WordPress Plugin`.
4. Clique em `Instalar` para baixar e instalar o plugin no seu site.
5. Clique em `Ativar` para ativar o plugin no seu site.
6. A ativação o levará para a página `Configurações do Wink`. *Você também pode acessar indo em `Aparência > Personalizar > Configurações do Wink`.*
7. Será solicitado seu `Client ID` e `Client Secret`. Vá para [Aplicações](/pt-BR/developers/applications) para aprender como obter suas credenciais.
8. Insira suas credenciais e clique em `Publicar`. Agora você está conectado à sua conta Wink.
</Steps>

## Construtores de páginas

Suportamos os construtores de páginas mais populares no WordPress, facilitando para qualquer pessoa que queira usar o Wink.

### Gutenberg

Gutenberg é o editor nativo do WordPress.

### Avada

[Avada](https://avada.com) é famoso pela grande seleção de templates bonitos junto com um construtor de páginas robusto e com um conjunto avançado de recursos.

### Elementor

[Elementor](https://elementor.com) é o construtor de páginas [atualmente] mais popular, com versões gratuita e profissional disponíveis.

### WPBakery

[WPBakery](https://wpbakery.com) foi o primeiro construtor de páginas avançado no mercado.

## Short codes

### [winklookup]

Funciona com grade de conteúdo ranqueado. Digite um lugar que deseja visitar e tenha o inventário exibido.

### [winksearch]

Botão simples para abrir o formulário de itinerário.

### [winkaccount]

Botão de conta para autenticação. Uma vez autenticado, ele se transforma em um menu suspenso com opções da conta.

### [winkitinerary]

Igual ao botão de busca, só que contém informações do itinerário como texto do botão.

### [winkcontent]

Permite incorporar o inventário que você selecionou no Wink diretamente em uma página ou postagem. Este shortcode também espera um ID de layout.

O layout

```
[winkcontent layoutid=""]
```
