O Wink integra-se com os principais channel managers, sistemas de gestão de propriedades (PMS) e sistemas centrais de reservas (CRS) para manter suas tarifas e disponibilidade sincronizadas em todos os seus canais de venda.

## Como funciona

As integrações com channel manager permitem um fluxo bidirecional de dados entre seu sistema de gestão de propriedades e o Wink:

1. **Envio de tarifas** — Seu channel manager envia atualizações de tarifas para o Wink. Quando você atualiza seus preços no PMS, essas mudanças são refletidas automaticamente no Wink.
2. **Sincronização de disponibilidade** — Seu channel manager envia atualizações de disponibilidade para o Wink. Quando quartos são reservados ou liberados no PMS, o inventário do Wink é atualizado em tempo real.
3. **Notificações de reservas** — Quando uma reserva é feita no Wink, seu channel manager é notificado para que a reserva apareça no seu PMS.

## Channel managers suportados

Atualmente, o Wink integra-se com os seguintes sistemas:

| Channel Manager | Descrição |
|---|---|
| **Channex** | Plataforma global de distribuição e gestão de canais. |
| **TravelClick** | Soluções para hotéis e gestão de canais (Amadeus). |
| **Omnibees** | Distribuição e gestão de receita para hotéis. |
| **D-Edge** | Tecnologia para hospitalidade e soluções de reservas centrais. |
| **Allotz** | Alocação de inventário e gestão de canais. |
| **Ezee / Yanolja** | PMS em nuvem e gestão de canais focados no mercado asiático. |
| **RateGain** | Inteligência de tarifas e tecnologia de distribuição. |
| **RateTiger** | Distribuição em tempo real de tarifas e disponibilidade. |
| **CloudBeds** | PMS baseado em nuvem, channel manager e motor de reservas. |
| **SiteMinder** | Principal channel manager e plataforma de distribuição para hotéis. |
| **Exely** | Sincronização de tarifas e disponibilidade para hotéis. |

:::note
Se o seu channel manager não estiver listado, entre em contato com a equipe do Wink. Novas integrações podem ser adicionadas com base nos padrões do protocolo OTA (Open Travel Alliance) que o Wink suporta.
:::

## Para desenvolvedores

Se você está construindo uma integração de channel manager ou PMS com o Wink, use os endpoints da Channel Manager API:

- **Atualizações de tarifas** — Envie alterações de tarifas para o Wink.
- **Atualizações de disponibilidade** — Envie alterações de disponibilidade para o Wink.
- **Recuperação de reservas** — Obtenha reservas do Wink para seu sistema.

A Channel Manager API segue os padrões do protocolo OTA (SOAP/XML) para compatibilidade com sistemas de hospitalidade existentes.

## Notificações via webhook

Você pode se inscrever em eventos webhook do channel manager para receber notificações em tempo real:

- `channel-manager.update.rate` — Atualização de tarifa recebida.
- `channel-manager.update.availability` — Atualização de disponibilidade recebida.
- `channel-manager.update` — Atualização geral do channel manager.

Consulte o [Catálogo de Eventos Webhook](/pt-BR/developers/webhook-events#channel-manager-events) para mais detalhes.
