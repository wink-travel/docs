As aplicações dão a você acesso total aos recursos do Wink e TripPay e permitem que seus usuários se autentiquem e façam login no seu site usando nosso servidor de autorização OAuth2; assim como Facebook e Google.
Você pode construir sua própria OTA com Wink. Tudo começa com uma aplicação.

*Por padrão, você recebe um app ao registrar uma nova conta afiliada Wink ou uma conta TripPay. Usamos o `client ID` para alguns cenários.*

:::note
Uma aplicação gerada no Wink ou TripPay pode ser usada para todos os endpoints da API.
:::

*Para este exemplo, vamos autenticar via [Wink Studio](https://studio.wink.travel).*

<Steps>
1. Uma vez autenticado, clique no ícone do seu perfil, no canto superior direito, e um menu aparecerá.
2. Clique no link `Applications`.
3. Você será redirecionado para a página dos seus apps.
</Steps>

## Criar app

Como você não pode acessar a `secret key` do app que criamos para você, será necessário criar o seu próprio.

:::caution
Você precisa criar uma conta afiliada no Wink ou uma conta no TripPay antes de criar um app, pois você irá vincular sua aplicação a uma conta específica.
:::

Aqui estão os passos para criar uma aplicação:

<Steps>
1. Na página de apps, clique no botão `Create new application`.
2. ***Nome*** Dê um nome para seu app. *ex: Cool App*
3. **Nome da entidade** Vincule seu app a uma das suas contas existentes. *ex: Cool Account*
4. **Redirect URIs** É necessário pelo menos um redirect URI. Ele é usado para validar o domínio de redirecionamento após uma autenticação bem-sucedida. *ex: https://www.cool-site.com*
5. Clique no botão `Save` para continuar.
</Steps>

Você será redirecionado de volta para sua lista de aplicações. Você verá seu novo app na lista e terá acesso à chave secreta do app apenas esta vez. Certifique-se de copiá-la e guardá-la em um local seguro.

## Atualizar app

Você precisará atualizar seu app se algum dos seus redirect URIs mudar ou se quiser adicionar ou remover um URI.

Para atualizar um app existente, faça o seguinte:

<Steps>
1. Clique no link `Actions` do app que deseja atualizar.
2. Clique no botão `Update` em Actions.
3. Faça as alterações no app.
4. Clique no botão `Save` para continuar.
</Steps>

Você será redirecionado de volta para sua lista de aplicações.

## Remover app

:::caution
Por favor, não exclua o app que criamos automaticamente para você para garantir que todos os recursos funcionem para sua conta.
:::

Se você não tiver mais uso para um app, pode removê-lo.

Para remover um app, faça o seguinte:

<Steps>
1. Clique no link `Actions` do app que deseja remover.
2. Clique no botão `Remove` em Actions.
3. Clique no botão `OK` para confirmar a exclusão.
</Steps>

Você será redirecionado de volta para sua lista de aplicações.
