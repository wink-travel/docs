Os aplicativos oferecem acesso total aos recursos do Wink e do TripPay e permitem que seus usuários se autentiquem e acessem seu site usando nosso servidor de autorização OAuth2, assim como o Facebook e o Google.
Você pode criar sua própria OTA com o Wink. Tudo começa com um aplicativo.

*Por padrão, você recebe um aplicativo ao registrar uma nova conta de afiliado Wink ou TripPay. Usamos o`client ID`para alguns cenários.*

Um aplicativo gerado no Wink ou TripPay pode ser usado para todos os endpoints da API.
:::

*Para este exemplo, iremos autenticar via[Estúdio Wink](https://studio.wink.travel).*

<Steps>
  1. Após a autenticação, clique no ícone do seu perfil, no canto superior direito, e um menu aparecerá.
  2. Clique no `Applications` link.
  3. Você será redirecionado para sua página de aplicativos.
</Steps>


## Criar aplicativo

Como você não pode acessar o`secret key`do aplicativo que criamos para você, você precisará criar o seu próprio.

:::Atenção
Você precisa criar uma conta de afiliado no Wink ou uma conta no TripPay antes de criar um aplicativo, pois você vinculará seu aplicativo a uma conta específica.
:::

Aqui estão as etapas para criar um aplicativo:

<Steps>
  1. Na página de aplicativos, clique em `Create new application` botão.
  2. ***Nome*** Dê um nome ao seu aplicativo. *por exemplo, Cool App*
  3. **Nome da entidade** Vincule seu aplicativo a uma de suas contas existentes. *por exemplo, Conta Legal*
  4. **URIs de redirecionamento** Pelo menos um URI de redirecionamento é necessário. Ele é usado para validar o domínio de redirecionamento após uma autenticação bem-sucedida. *por exemplo https://www.cool-site.com*
  5. Clique no `Save` botão para continuar.
</Steps>


Você será redirecionado para a sua lista de aplicativos. Você verá seu novo aplicativo na lista e terá acesso à chave secreta do aplicativo apenas uma vez. Certifique-se de copiá-la e salvá-la em um local seguro.

## Atualizar aplicativo

Você precisará atualizar seu aplicativo se um dos seus URIs de redirecionamento mudar ou se você quiser adicionar ou remover um URI.

Para atualizar um aplicativo existente, faça o seguinte:

<Steps>
  1. Clique no `Actions` link para o aplicativo que você deseja atualizar.
  2. Clique no `Update` botão em Ações.
  3. Faça suas alterações no aplicativo.
  4. Clique no `Save` botão para continuar.
</Steps>


Você será redirecionado de volta para sua lista de aplicativos.

## Remover aplicativo

:::Atenção
Não exclua o aplicativo que criamos automaticamente para você, para garantir que todos os recursos funcionem na sua conta.
:::

Se você não tiver mais utilidade para um aplicativo, você pode removê-lo.

Para remover um aplicativo, faça o seguinte:

<Steps>
  1. Clique no `Actions` link para o aplicativo que você deseja atualizar.
  2. Clique no `Remove` botão em Ações.
  3. Clique `OK` botão para confirmar a exclusão.
</Steps>


Você será redirecionado de volta para sua lista de aplicativos.
