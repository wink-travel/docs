As propriedades terão automaticamente todo o seu inventário disponível como links compartilháveis, sem precisar acessar o Wink Studio para gerá-los.
:::

Links compartilháveis são um recurso central do Wink e surgiram da necessidade de comunicar inventário e preços de forma dinâmica pelo meio mais simples possível na Internet; ou seja, o hiperlink.

<h4 align="center">[https://trvl.as/3xWCH](https://trvl.as/3xWCH)</h4>

Acima está um exemplo de um dos nossos links. É curto e portátil... até curto o suficiente para lembrar. Embora curto, este link contém uma quantidade **explosiva** de informações, como:

- Seu ID de App.
- Seu ID de Afiliado.
- A(s) imagem(ns) e texto a serem exibidos para o usuário.
- O visual (*ou seja, tema, cores e mais*).
- O itinerário a ser usado para nossa consulta de preço (*2 adultos, 1 noite*).
- A moeda para exibir o preço.
- O idioma para exibir o texto.

O restante deste artigo explica como gerenciar os links para seus inventários.

## Personalizar link

O formulário permite personalizar seu link das seguintes maneiras:

<Steps>
 1. Por padrão, o link recebe o nome do seu inventário em inglês. Você pode dar qualquer nome que o ajude a lembrar do que se trata o link.
 2. Por padrão, sua marcação / personalização principal é aplicada ao link. Você pode selecionar outras [Personalizações](/studio/customization) que você gerou no Wink Studio.
 3. Adicione palavras-chave, separadas por vírgula, que serão usadas por [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Opcionalmente, insira seu identificador da conta X ou ID do App do Facebook.
 5. Adicione títulos e descrições nos idiomas que deseja que o usuário veja.
 6. Selecione uma ou mais imagens que serão exibidas quando você compartilhar o link em plataformas como Facebook ou LinkedIn.
 7. Escolha um tema / sobreposição que adiciona um design com a marca sobre a(s) imagem(ns) selecionada(s). *Nota: É assim que você faz os preços aparecerem sobre a imagem.*
 8. Clique no botão `Salvar` para continuar.
 </Steps>

Acima você vê uma imagem com todas as ações disponíveis para seu link:

- **Visualizar** Mostra uma visão rápida de como o link foi configurado e como ele aparecerá para o usuário.
- **Atualizar** Atualiza a configuração do seu link.
- **Adicionar ao WinkLinks** Adiciona o link à sua conta WinkLinks.
- **Copiar** Copia o link para sua área de transferência para que você possa compartilhá-lo facilmente onde quiser (*incluindo [WinkLinks](/link-manager/wink-links)*).
- **Compartilhar** Abre uma nova janela com opções para compartilhar em várias redes sociais e plataformas de mensagens.
- **Código QR** Gera um código QR que você pode compartilhar em plataformas como o IG, que são mais focadas em imagens.
- **Incorporar** Mostra como incorporar este link como um [Banner de Anúncio](/developers/web-components/#content-loader) no seu site.
- **Usar com WordPress** Mostra como usar nosso [plugin do WordPress](/developers/wordpress) para incorporar este link no seu site.

Se você quiser compartilhar um código QR em vez de um link, clique no botão `Código QR` e uma nova janela aparecerá com seu código QR.

Como salvar seu código QR:

- Se estiver no seu laptop, clique com o botão direito na imagem e selecione `Salvar imagem`.
- Se estiver no seu celular:
    - Pressione e segure o dedo sobre a imagem até aparecer a opção de salvar a imagem nas Fotos.
    - Tire uma captura de tela do código QR, recorte para ajustar e salve nas Fotos.

### Incorporar

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Aqui está como incorporar seu link, como um banner de anúncio, no seu site:

- A linha 3 mostra como incorporar os estilos do Wink no seu site.
- As linhas 6 a 9 mostram como usar o [wink-content-loader](/developers/web-components/#content-loader) Web Component e instruí-lo a buscar um banner de anúncio para seu código.
- A linha 11 mostra como incorporar nosso Javascript no seu site.
- A linha 13 mostra como incorporar o [wink-app-loader](/developers/web-components/#application-loader) Web Component e instruí-lo a buscar suas preferências de configuração em nível de página.
