:::note
Semelhante à indústria aérea, os hotéis exigem muita flexibilidade em relação a preços e descontos.
Promoções são uma forma do hotel criar condições únicas sob as quais um viajante é elegível para uma redução adicional de preço [ou acréscimo].
:::

Para gerenciar suas Promoções, clique em `Monetize > Single Promotions` na barra de navegação principal.

Para criar uma nova promoção, clique no botão `Create single promotion`.

## Configuração

Esta seção é responsável por definir o valor monetário real da promoção.

- **Nome** Insira um nome para esta promoção que somente você poderá ver.
- **Tipo** Escolha se deseja que esta promoção contenha um desconto ou um acréscimo.
- **Tipo de precificação** Escolha como deseja que este pacote seja calculado.
    - `Por estadia`: O valor é aplicado na última noite. *ex: 100% de desconto > Fique 2 noites - Ganhe a 3ª noite grátis.*
    - `Por noite`: Valor multiplicado pelo número de noites.
    - `Por uso`: Nenhum cálculo adicional ocorre.
    - `Por pessoa`: Valor multiplicado pelo número de hóspedes.
    - `Por pessoa por noite`: Valor multiplicado pelo número de noites + número de hóspedes.
- **Tipo de valor** Escolha se deseja que o modificador de valor seja um valor fixo ou percentual.
- **Status** Ative o interruptor para controlar se a promoção está disponível em todos os canais de venda.

## Descrições

:::note
Você precisa escrever pelo menos uma descrição e ela precisa estar em inglês.
:::

Clique na aba `Descriptions` para continuar.

Uma descrição localizada descreve para o viajante, em seu idioma, o propósito desta promoção.
Você pode escrever em quantos idiomas desejar. Adicionaremos traduções para todos os idiomas mais populares.
Este é o texto que os viajantes verão ao visualizar sua promoção.

- **Descrição** Dê um nome para a promoção. *ex: Early bird - 20% OFF*

## Restrições

:::note
Lembre-se, estas são restrições... Ative-as somente se quiser tornar sua promoção menos acessível.
:::

Clique na aba `Restrictions` para continuar.

O motor de promoções do Wink possui nada menos que 20 formas únicas de disparar uma alteração de preço.

Suportamos os seguintes critérios:

- Restringir promo a certos complementos.
- Restringir promo a certas tarifas master.
- Restringir promo a certos planos tarifários.
- Restringir promo a certas políticas de cancelamento.
- Restringir promo ao comprimento da estadia.
- Restringir promo à reserva antecipada.
- Restringir promo ao número de quartos.
- Restringir promo ao código promocional.
- Restringir promo ao intervalo de IP do viajante.
- Restringir promo ao último minuto.
- Restringir promo à cidade do viajante.
- Restringir promo ao país do viajante.
- Restringir promo ao continente do viajante.
- Restringir promo ao fuso horário do viajante.
- Restringir promo às datas de venda.
- Restringir promo às datas de estadia.
- Restringir promo aos dias da semana da reserva.
- Restringir promo aos dias da semana da chegada.
- Restringir promo aos dias da semana da partida.
- Restringir promo aos dias da semana obrigatórios.

Continue lendo para ver como usar cada um.

### Complemento

:::note
Para usar este recurso, você precisa ter criado pelo menos um complemento.
:::

Para que uma promoção se aplique somente aos complementos que você selecionar, siga estes passos:

<Steps>
1. Ative o interruptor `Add-on restriction`.
2. Marque a caixa ao lado de cada complemento que deseja incluir.
</Steps>

### Tarifa master

:::note
Para usar este recurso, você precisa ter criado pelo menos uma tarifa master.
:::

Para que uma promoção se aplique somente às tarifas master que você selecionar, siga estes passos:

<Steps>
1. Ative o interruptor `Master rate restriction`.
2. Marque a caixa ao lado de cada tarifa master que deseja incluir.
</Steps>

### Plano tarifário

:::note
Para usar este recurso, você precisa ter criado pelo menos um plano tarifário.
:::

Para que uma promoção se aplique somente aos planos tarifários que você selecionar, siga estes passos:

<Steps>
1. Ative o interruptor `Rate plan restriction`.
2. Marque a caixa ao lado de cada plano tarifário que deseja incluir.
</Steps>

### Reembolsável

:::note
Para usar este recurso, você precisa ter criado pelo menos uma política de cancelamento.
:::

Para que uma promoção se aplique somente às políticas de cancelamento que você selecionar, siga estes passos:

<Steps>
1. Ative o interruptor `Cancellation policy restriction`.
2. Marque a caixa ao lado de cada política de cancelamento que deseja incluir.
</Steps>

### Comprimento da estadia

:::note
Esta restrição é o que você normalmente usa como sua promoção de `longo prazo`.
:::

Quando você adiciona esta restrição, está exigindo que os hóspedes `fiquem pelo menos` e `no máximo` um número especificado de dias.

Siga estes passos para ativar `Length of Stay`:

<Steps>
1. Ative o interruptor `Length of stay restriction`.
2. Insira o número mínimo de dias que o hóspede deve ficar. *ex: 7*
3. Opcionalmente, insira o número máximo de dias que o hóspede pode ficar. *ex: 90*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que ficam mais de 10 dias. Nota: O comprimento máximo da estadia é opcional.*

### Reserva antecipada

:::note
Esta restrição é o que você normalmente usa como sua promoção de `early bird`.
:::

Quando você adiciona esta restrição, está exigindo que os hóspedes `reservem pelo menos` e `no máximo` um número especificado de dias antes.

Siga estes passos para ativar `Advance booking`:

<Steps>
1. Ative o interruptor `Advance booking restriction`.
2. Insira o número mínimo de dias que o hóspede precisa reservar com antecedência. *ex: 10*
3. Opcionalmente, insira o número máximo de dias que o hóspede precisa reservar com antecedência. *ex: 180*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam com mais de 10 dias de antecedência. Nota: Também exigimos que você adicione uma reserva antecipada máxima.*

### Número de quartos

:::note
A promoção está disponível somente se o hóspede reservar o número mínimo de quartos.
:::

Quando você adiciona esta restrição, está exigindo que os hóspedes `reservem pelo menos` e `no máximo` um número especificado de quartos como parte da reserva.

Siga estes passos para ativar `Number of rooms`:

<Steps>
1. Ative o interruptor `Room restriction`.
2. Insira o número mínimo de quartos que o hóspede precisa reservar. *ex: 3*
3. Insira o número máximo de quartos que o hóspede precisa reservar. *ex: 8*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam mais de 3 quartos.*

### Código promocional

:::note
Use esta restrição se quiser que o hóspede insira um código de desconto para desbloquear esta promoção.
:::

Siga estes passos para ativar `Promo code`:

<Steps>
1. Ative o interruptor `Promotion restriction`.
2. Clique no botão `Add promo code`.
3. Insira o código promocional. *ex: ABC123*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que inserem o código promocional `ABC123`.*

### Intervalo de IP

:::note
Esta é uma boa restrição para usar quando você quer fazer uma campanha cruzada com operadoras móveis ou grandes empresas que possuem seus próprios intervalos de IP.
:::

Siga estes passos para ativar `IP range`:

<Steps>
1. Ative o interruptor `IP range restriction`.
2. Clique no botão `Add IP range restriction`.
3. Insira o intervalo inicial de IP [globalmente acessível]. *ex: 203.0.113.50*
4. Insira o intervalo final de IP [globalmente acessível]. *ex: 203.0.113.59*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes com números de IP entre `203.0.113.50` - `203.0.113.59`.*

### Último minuto

:::note
Este é o período do negócio de última hora que você quer restringir.
:::

Siga estes passos para ativar `Last minute`:

<Steps>
1. Ative o interruptor `Last minute restriction`.
2. Clique no botão `Add IP range restriction`.
3. Insira o número de horas antes da data de chegada. *ex: 24 horas*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam 24 horas antes da data de chegada.*

### Cidade

:::note
Permite restringir à cidade onde o viajante está atualmente localizado. Crie uma restrição para cada cidade que deseja incluir.
:::

Siga estes passos para ativar `City`:

<Steps>
1. Ative o interruptor `City restriction`.
2. Comece a digitar o nome da cidade.
3. Selecione a cidade nos resultados da busca. *ex: New York City*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam de `New York City`.*

### País

:::note
Permite restringir ao país onde o viajante está atualmente localizado. Crie uma restrição para cada país que deseja incluir.
:::

Siga estes passos para ativar `Country`:

<Steps>
1. Ative o interruptor `Country restriction`.
2. Comece a digitar o nome do país.
3. Selecione o país nos resultados da busca. *ex: United States of America*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam de `United States of America`.*

### Continente

:::note
Permite restringir a tarifa ao continente onde o hóspede está atualmente localizado. Crie uma restrição para cada continente que deseja incluir.
:::

Siga estes passos para ativar `Continent`:

<Steps>
1. Ative o interruptor `Continent restriction`.
2. Comece a digitar o nome do continente.
3. Selecione o continente nos resultados da busca. *ex: North America*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam de `North America`.*

### Fuso horário

:::note
Permite restringir a tarifa ao fuso horário onde o hóspede está atualmente localizado. Crie uma restrição para cada fuso horário que deseja incluir.
:::

Siga estes passos para ativar `Timezone`:

<Steps>
1. Ative o interruptor `Timezone restriction`.
2. Comece a digitar o nome do fuso horário.
3. Selecione o fuso horário nos resultados da busca. *ex: America/New_York*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam de `America/New_York`.*

### Data de venda

:::note
Permite restringir a um intervalo específico de datas. Um viajante será elegível ao reservar um quarto nas datas especificadas dentro deste intervalo.
:::

Siga estes passos para ativar `Sell date`:

<Steps>
1. Ative o interruptor `Sell date restriction`.
2. Clique no botão `Add sell date restriction`.
3. Insira a data de início. *ex: 19 de set. de 2024*
4. Insira a data de término. *ex: 1 de dez. de 2024*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam entre `19 de set. de 2024` e `1 de dez. de 2024`.*

### Data de estadia

:::note
Permite restringir a um intervalo específico de datas. Um viajante será elegível ao reservar um quarto para as datas especificadas dentro deste intervalo.
:::

Siga estes passos para ativar `Stay date`:

<Steps>
1. Ative o interruptor `Stay date restriction`.
2. Clique no botão `Add stay date restriction`.
3. Insira a data de início. *ex: 19 de set. de 2024*
4. Insira a data de término. *ex: 1 de dez. de 2024*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam uma estadia entre `19 de set. de 2024` e `1 de dez. de 2024`.*

### Dias da semana da reserva

:::note
Ao ativar esta restrição, você permite que esta tarifa esteja disponível somente no dia da semana especificado em que a reserva ocorre.

Esta restrição funciona bem com a "Restrição de data de venda", onde você pode limitar um intervalo de datas e restringir ainda mais o intervalo com dias da semana usando esta restrição.
:::

Siga estes passos para ativar `Booking days`:

<Steps>
1. Ative o interruptor `Booking days restriction`.
2. Ative um dos interruptores de dia da semana. *ex: Segunda-feira*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam uma estadia em uma `Segunda-feira`.*

### Dias da semana da chegada

:::note
Ao ativar esta restrição, os hóspedes devem chegar nesse dia para ver esta tarifa.

Esta restrição funciona bem com a "Restrição de data de estadia", onde você pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Siga estes passos para ativar `Arrival days`:

<Steps>
1. Ative o interruptor `Arrival days restriction`.
2. Ative um dos interruptores de dia da semana. *ex: Segunda-feira*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam uma estadia que começa em uma `Segunda-feira`.*

### Dias da semana da partida

:::note
Ao ativar esta restrição, os hóspedes devem partir nesse dia para ver esta tarifa.

Esta restrição funciona bem com a "Restrição de data de estadia", onde você pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Siga estes passos para ativar `Departure days`:

<Steps>
1. Ative o interruptor `Departure days restriction`.
2. Ative um dos interruptores de dia da semana. *ex: Segunda-feira*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam uma estadia que termina em uma `Segunda-feira`.*

### Dias da semana obrigatórios

:::note
Ao ativar esta restrição, os hóspedes devem permanecer em "pelo menos um" desses dias para ver esta tarifa.

Esta restrição funciona bem com a "Restrição de data de estadia", onde você pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Siga estes passos para ativar `Required days`:

<Steps>
1. Ative o interruptor `Required days restriction`.
2. Ative um dos interruptores de dia da semana. *ex: Segunda-feira*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservam uma estadia que inclui uma `Segunda-feira`.*

## Datas de bloqueio

Clique na aba `Blackout dates` para continuar.

Controle a promoção ainda mais especificando datas em que esta promoção não deve estar disponível.

Para adicionar uma data de bloqueio, clique no botão `Add blackout date`.

- **Data de início** Selecione a data de início da sua data de bloqueio.
- **Data de término** Selecione a data de término da sua data de bloqueio.

Se as datas da reserva tocarem este intervalo, a promoção não terá efeito.

Um viajante pode ser elegível para múltiplas promoções, como longo prazo + early bird, ao mesmo tempo.
Sem uma promoção agrupada, o viajante seria elegível apenas para uma delas; a melhor oferta.

Ao criar uma promoção agrupada, você honra a elegibilidade do viajante para ambas as promoções e pode personalizá-la ainda mais.

Para criar uma promoção agrupada, clique no botão `Create bundled promotion`.

### Configuração

- **Nome** Dê um nome para a promoção agrupada que somente você possa ver.
- **Status** Ative o interruptor para controlar se o pacote está disponível em todos os canais de venda.
- **Promoções** Selecione, no mínimo, 2 promoções simples para compor o pacote.

### Substituir valor

Você pode optar por substituir o desconto total aplicado. Quando deixar desativado, o desconto de cada promoção selecionada abaixo será calculado em um único desconto.

- **Substituir valor** Se deseja agregar todos os valores descontados ou substituir pelo seu próprio valor.
- **Tipo** Selecione se deseja que o preço aumente ou diminua. *ex: Defina como `Discount` para uma redução de preço.*
- **Tipo de precificação** Escolha como deseja que este pacote seja calculado.
    - `Por estadia`: O valor é aplicado na última noite. *ex: 100% de desconto > Fique 2 noites - Ganhe a 3ª noite grátis.*
    - `Por noite`: Valor multiplicado pelo número de noites.
    - `Por uso`: Nenhum cálculo adicional ocorre.
    - `Por pessoa`: Valor multiplicado pelo número de hóspedes.
    - `Por pessoa por noite`: Valor multiplicado pelo número de noites + número de hóspedes.
- **Tipo de valor** Escolha se deseja que o modificador de valor seja um valor fixo ou percentual.
