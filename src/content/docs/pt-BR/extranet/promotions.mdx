:::nota
Assim como no setor aéreo, os hotéis exigem bastante flexibilidade em relação a preços e descontos.
As promoções são uma forma de o hotel criar condições únicas sob as quais o viajante tem direito a uma redução de preço adicional \[ou prêmio].
:::

Para gerenciar suas promoções, clique em`Monetize > Single Promotions`na barra de navegação principal.

Para criar uma nova promoção, clique no`Create single promotion` botão.

## Configuração

Esta seção é responsável por definir o valor monetário real da promoção.

* **Nome**Digite um nome para esta promoção que somente você possa ver.
* **Tipo**Escolha se você quer que esta promoção contenha um desconto ou um prêmio.
* **Tipo de preço**Escolha como você deseja que este pacote seja calculado.
  * `Per stay`: O valor é aplicado à noite passada.*por exemplo, 100% de desconto > Fique 2 noites - Ganhe a 3ª noite grátis.*
  * `Per night`: Valor multiplicado pelo número de noites.
  * `Per use`: Nenhum cálculo adicional ocorre.
  * `Per person`: Valor multiplicado pelo número de convidados.
  * `Per person per night`: Valor multiplicado pelo número de noites + número de hóspedes.
* **Tipo de valor**Escolha se deseja que o modificador de valor seja um valor fixo ou uma porcentagem.
* **Status**Alterne o botão para controlar se a promoção está disponível em todos os canais de vendas.

## Descrições

:::nota
Você precisa escrever pelo menos uma descrição, que deve estar em inglês.
:::

Clique no`Descriptions`guia para continuar.

Uma descrição localizada descreve o propósito desta promoção para o viajante, no idioma dele.
Você pode escrever em quantos idiomas quiser. Adicionaremos traduções para todos os idiomas mais populares.
Este é o texto que os viajantes verão ao visualizar sua promoção.

* **Descrição**Dê um nome à promoção.*por exemplo, Madrugador - 20% DE DESCONTO*

## Restrições

:::nota
Lembre-se de que estas são restrições... Ative-as somente se quiser tornar sua promoção menos acessível.
:::

Clique no`Restrictions`guia para continuar.

O mecanismo de promoção do Wink tem, nada menos que, 20 maneiras exclusivas de acionar uma mudança de preço.

Apoiamos os seguintes critérios:

* Restringir a promoção a determinados complementos.
* Restringir a promoção a determinadas taxas principais.
* Restringir a promoção a determinados planos de tarifas.
* Restringir a promoção a determinadas políticas de cancelamento.
* Restringir a promoção à duração da estadia.
* Restringir a promoção à reserva antecipada.
* Restringir a promoção ao número de quartos.
* Restringir promoção ao código promocional.
* Restringir a promoção ao intervalo de IP do viajante.
* Restrinja a promoção ao último minuto.
* Restringir a promoção à cidade do viajante.
* Restringir a promoção ao país do viajante.
* Restringir a promoção ao continente viajante.
* Restringir a promoção ao fuso horário do viajante.
* Restringir a promoção às datas de venda.
* Restringir a promoção às datas de estadia.
* Restringir a promoção aos dias de reserva da semana.
* Restringir a promoção aos dias de chegada da semana.
* Restringir a promoção aos dias de partida da semana.
* Restringir a promoção aos dias da semana necessários.

Continue lendo para ver como usar cada um.

### Adicionar

:::nota
Para usar este recurso, você precisa ter criado pelo menos um complemento.
:::

Para que uma promoção seja aplicada somente aos complementos selecionados, siga estas etapas:

<Steps>
  1. Habilitar o `Add-on restriction` trocar.
  2. Marque a caixa de seleção ao lado de cada complemento que você deseja incluir.
</Steps>


### Taxa de mestre

:::nota
Para usar este recurso, você precisa ter criado pelo menos uma tarifa mestre.
:::

Para que uma promoção seja aplicada somente às tarifas master selecionadas, siga estas etapas:

<Steps>
  1. Habilitar o `Master rate restriction` trocar.
  2. Marque a caixa de seleção ao lado de cada tarifa mestre que você deseja incluir.
</Steps>


### Plano de tarifas

:::nota
Para usar este recurso, você precisa ter criado pelo menos um plano de tarifas.
:::

Para que uma promoção seja aplicada somente aos planos de tarifas selecionados, siga estas etapas:

<Steps>
  1. Habilitar o `Rate plan restriction` trocar.
  2. Marque a caixa de seleção ao lado de cada plano de tarifa que você deseja incluir.
</Steps>


### Reembolsável

:::Observação
Para usar este recurso, você precisa ter criado pelo menos uma política de cancelamento.
:::

Para que uma promoção seja aplicada somente às políticas de cancelamento selecionadas, siga estas etapas:

<Steps>
  1. Habilitar o `Cancellation policy restriction` trocar.
  2. Marque a caixa de seleção ao lado de cada política de cancelamento que você deseja incluir.
</Steps>


### Duração da estadia

:::nota
Essa restrição é o que você normalmente usa como sua`long term`promoção.
:::

Ao adicionar esta restrição, você está exigindo que os convidados`stay at least` e `no more than`um número específico de dias.

Siga estas etapas para habilitar`Length of Stay`:

<Steps>
  1. Habilitar o `Length of stay restriction` trocar.
  2. Insira o número mínimo de dias que um hóspede deve ficar. *por exemplo 7*
  3. Opcional: insira o número máximo de dias que um hóspede pode ficar. *por exemplo 90*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes com estadias superiores a 10 dias. Observação: a duração máxima da estadia é opcional.*

### Reserva antecipada

:::nota
Essa restrição é o que você normalmente usa como sua`early bird`promoção.
:::

Ao adicionar esta restrição, você está exigindo que os convidados`book at least` e `no more than`um número específico de dias de antecedência.

Siga estas etapas para habilitar`Advance booking`:

<Steps>
  1. Habilitar o `Advance booking restriction` trocar.
  2. Informe o número mínimo de dias que um hóspede precisa reservar com antecedência. *por exemplo 10*
  3. Opcionalmente, insira o número máximo de dias que um hóspede precisa reservar com antecedência. *por exemplo 180*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem com mais de 10 dias de antecedência. Observação: também é necessário adicionar uma antecedência máxima para a reserva.*

### Número de quartos

:::Observação
A promoção só é válida se o hóspede reservar o número mínimo de quartos.
:::

Ao adicionar esta restrição, você está exigindo que os convidados`book at least` e `no more than`um número específico de quartos como parte de sua reserva.

Siga estas etapas para habilitar`Number of rooms`:

<Steps>
  1. Habilitar o `Room restriction` trocar.
  2. Insira o número mínimo de quartos que um hóspede precisa reservar. *por exemplo 3*
  3. Insira o número máximo de quartos que um hóspede precisa reservar. *por exemplo 8*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem mais de 3 quartos.*

### Código promocional

:::nota
Use esta restrição se desejar que o hóspede insira um código de desconto para desbloquear esta promoção.
:::

Siga estas etapas para habilitar`Promo code`:

<Steps>
  1. Habilitar o `Promotion restriction` trocar.
  2. Clique no `Add promo code` botão.
  3. Digite o código promocional. *por exemplo ABC123*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que inserirem o código promocional`ABC123`.*

### Intervalo de IP

:::nota
Esta é uma boa restrição para usar quando você deseja fazer uma campanha de promoção cruzada com operadoras de celular ou empresas maiores que tenham seus próprios intervalos de IP.
:::

Siga estas etapas para habilitar`IP range`:

<Steps>
  1. Habilitar o `IP range restriction` trocar.
  2. Clique no `Add IP range restriction` botão.
  3. Insira o intervalo IP inicial \[globalmente acessível]. *por exemplo 203.0.113.50*
  4. Digite o intervalo IP final \[globalmente acessível]. *por exemplo 203.0.113.59*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes com números de IP entre`203.0.113.50`-`203.0.113.59`.*

### Último minuto

:::nota
Este é o período da oferta de última hora que você deseja restringir.
:::

Siga estas etapas para habilitar`Last minute`:

<Steps>
  1. Habilitar o `Last minute restriction` trocar.
  2. Clique no `Add IP range restriction` botão.
  3. Insira o número de horas antes da data de chegada. *por exemplo, 24 horas*
</Steps>


*Exemplo: Esta promoção só está disponível para hóspedes que reservarem 24 horas antes da data de chegada.*

### Cidade

:::nota
Permite restringir a cidade onde o viajante está atualmente. Crie uma restrição para cada cidade que você deseja incluir.
:::

Siga estas etapas para habilitar`City`:

<Steps>
  1. Habilitar o `City restriction` trocar.
  2. Comece a digitar o nome da cidade.
  3. Selecione a cidade nos resultados da pesquisa. *por exemplo, cidade de Nova York*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem a partir de`New York City`.*

### País

:::nota
Permite restringir o país onde o viajante está atualmente. Crie uma restrição para cada país que deseja incluir.
:::

Siga estas etapas para habilitar`Country`:

<Steps>
  1. Habilitar o `Country restriction` trocar.
  2. Comece a digitar o nome do país.
  3. Selecione o país nos resultados da pesquisa. *por exemplo Estados Unidos da América*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem a partir de`United States of America`.*

### Continente

:::nota
Permite restringir a tarifa no continente onde o hóspede está atualmente. Crie uma restrição para cada continente que deseja incluir.
:::

Siga estas etapas para habilitar`Continent`:

<Steps>
  1. Habilitar o `Continent restriction` trocar.
  2. Comece a digitar o nome do continente.
  3. Selecione o continente nos resultados da pesquisa. *por exemplo, América do Norte*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem a partir de`North America`.*

### Fuso horário

:::nota
Permite restringir a tarifa ao fuso horário em que o hóspede está localizado. Crie uma restrição para cada fuso horário que desejar incluir.
:::

Siga estas etapas para habilitar`Timezone`:

<Steps>
  1. Habilitar o `Timezone restriction` trocar.
  2. Comece a digitar o nome do fuso horário.
  3. Selecione o fuso horário nos resultados da pesquisa. *por exemplo América/Nova\_Iorque*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem a partir de`America/New_York`.*

### Data de venda

:::nota
Permite restringir a estadia em um período específico. Um viajante será elegível ao reservar um quarto nas datas especificadas dentro desse período.
:::

Siga estas etapas para habilitar`Sell date`:

<Steps>
  1. Habilitar o `Sell date restriction` trocar.
  2. Clique no `Add sell date restriction` botão.
  3. Insira a data de início. *por exemplo, 19 de setembro de 2024*
  4. Insira a data final. *por exemplo, 1º de dezembro de 2024*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem entre`Sept. 19th 2024` e `Dec. 1st 2024`.*

### Data da estadia

:::nota
Permite restringir a estadia em um período específico. Um viajante será elegível ao reservar um quarto para as datas especificadas dentro desse período.
:::

Siga estas etapas para habilitar`Stay date`:

<Steps>
  1. Habilitar o `Stay date restriction` trocar.
  2. Clique no `Add stay date restriction` botão.
  3. Insira a data de início. *por exemplo, 19 de setembro de 2024*
  4. Insira a data final. *por exemplo, 1º de dezembro de 2024*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem uma estadia entre`Sept. 19th 2024` e `Dec. 1st 2024`.*

### Dias da semana de reserva

:::nota
Ao habilitar essa restrição, você permite que essa tarifa esteja disponível apenas no dia da semana especificado em que a reserva for realizada.

Essa restrição combina bem com a "Restrição de data de venda", onde você pode limitar um intervalo de datas e limitar ainda mais o intervalo de datas com os dias da semana usando essa restrição.
:::

Siga estas etapas para habilitar`Booking days`:

<Steps>
  1. Habilitar o `Booking days restriction` trocar.
  2. Ative um dos interruptores do dia da semana. *por exemplo segunda-feira*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem uma estadia em um`Monday`.*

### Dias de chegada da semana

:::nota
Ao ativar esta restrição, os hóspedes devem chegar naquele dia para ver esta tarifa.

Essa restrição combina bem com a "Restrição de data de estadia", onde você pode limitar o intervalo de datas e limitar ainda mais os dias da semana usando essa restrição.
:::

Siga estas etapas para habilitar`Arrival days`:

<Steps>
  1. Habilitar o `Arrival days restriction` trocar.
  2. Ative um dos interruptores do dia da semana. *por exemplo segunda-feira*
</Steps>


*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem uma estadia que comece em um`Monday`.*

### Dias de partida da semana

:::nota
Ao habilitar esta restrição, os hóspedes devem sair naquele dia para ver esta tarifa.

Essa restrição combina bem com a "Restrição de data de estadia", onde você pode limitar o intervalo de datas e limitar ainda mais os dias da semana usando essa restrição.
:::

Siga estas etapas para habilitar`Departure days`:

<Steps>
  1. Habilitar o `Departure days restriction` trocar.
  2. Ative um dos interruptores do dia da semana. *por exemplo segunda-feira*

### Dias da semana obrigatórios

:::nota
Ao ativar esta restrição, os hóspedes devem pernoitar em "pelo menos um" destes dias para ver esta tarifa.

Essa restrição combina bem com a "Restrição de data de estadia", onde você pode limitar o intervalo de datas e limitar ainda mais os dias da semana usando essa restrição.

Siga estas etapas para habilitar`Required days`:

<Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservarem uma estadia que envolva um`Monday`.*

## Datas de blackout

Clique no`Blackout dates`guia para continuar.

Controle ainda mais a promoção especificando datas em que ela não deve estar disponível.

Para adicionar uma data de indisponibilidade, clique em`Add blackout date` botão.

* **Data de início**Selecione a data de início da sua data de indisponibilidade.
* **Data de término**Selecione a data final da sua data de indisponibilidade.

Se as datas da reserva atingirem esse intervalo de datas, a promoção não entrará em vigor.

## Pacotes

Para gerenciar seus Pacotes, clique em`Monetize > Bundled Promotions`na barra de navegação principal.

  <CloudinaryImage width="350" height="475" resize="scale" quality="best" alt="Pacote promocional" src="academy/promotion-bundle" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Exemplo de entrada para pacote promocional*</figcaption>
</figure>


Um viajante pode ser elegível para várias promoções, como longa duração + reserva antecipada, ao mesmo tempo.
Sem uma promoção combinada, o viajante só seria elegível para uma delas: a melhor oferta.

Ao criar uma promoção combinada, você honra a elegibilidade do viajante para ambas as promoções e pode personalizá-la ainda mais.

Para criar uma promoção em pacote, clique em`Create bundled promotion` botão.

### Configuração

* **Nome**Dê ao pacote promocional um nome que só você possa ver.
* **Status**Alterne a chave para controlar se o pacote estará disponível em todos os canais de vendas.
* **Promoções**Selecione pelo menos 2 promoções individuais para criar um pacote.

### Valor de substituição

Você pode optar por substituir o desconto total aplicado. Ao desativá-lo, o desconto de cada promoção selecionada abaixo será calculado como um desconto.

* **Valor de substituição**Se você deseja agregar todos os valores descontados ou substituir pelo seu próprio valor.
* **Tipo**Selecione se deseja que o preço aumente ou diminua.*por exemplo, definido para`Discount`para uma redução de preço.*
* **Tipo de preço**Escolha como você deseja que este pacote seja calculado.
  * `Per stay`: O valor é aplicado à noite passada.*por exemplo, 100% de desconto > Fique 2 noites - Ganhe a 3ª noite grátis.*
  * `Per night`: Valor multiplicado pelo número de noites.
  * `Per use`: Nenhum cálculo adicional ocorre.
  * `Per person`: Valor multiplicado pelo número de convidados.
  * `Per person per night`: Valor multiplicado pelo número de noites + número de hóspedes.
* **Tipo de valor**Escolha se deseja que o modificador de valor seja um valor fixo ou uma porcentagem.
