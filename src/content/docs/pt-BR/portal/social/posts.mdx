O editor de publicações permite criar publicações manuais ilimitadas e também é o local onde você gerencia publicações criadas automaticamente para você.

O editor possui 4 seções:

1. A seção de conteúdo.
2. A seção de mídia.
3. A seção de link.
4. A seção de configurações.

## Conteúdo

A seção de conteúdo permite modificar o texto antes de ser publicado. Você pode criar conteúdo localizado em vários idiomas — cada idioma tem seu próprio título e descrição.

## Mídia

A seção de mídia permite modificar as imagens e vídeos que acompanham o texto.

## Link

A seção de link permite adicionar um link para onde você deseja que o usuário vá após ler sua publicação. *Por exemplo, pode ser um link para uma oferta especial ou um local para obter mais informações.* Você pode adicionar URL do link, legenda do link e descrição do link.

## Configurações

A seção de configurações permite modificar / substituir todas as suas configurações padrão por publicação. Você pode definir:

- **Plataforma** — Escolha em qual rede social esta publicação será publicada.
- **Tipo de publicação** — Escolha entre Transacional, Conscientização ou Sentimento.
- **Tom de voz** — Substitua o tom de voz para esta publicação específica.
- **Personalização** — Use uma personalização diferente para moeda de exibição, idioma, itinerário, etc.
- **Fuso horário** — Substitua o fuso horário para a publicação.
- **Data agendada** — Publicações geradas são automaticamente agendadas no horário ideal. Você pode substituir a data e hora se preferir outro horário.

## Ciclo de vida do status da publicação

Cada publicação segue um ciclo de vida de status:

| Status | Descrição |
| -- | -- |
| **Rascunho** | A publicação foi criada, mas ainda não está pronta para revisão |
| **Aguardando aprovação** | A publicação está aguardando aprovação humana antes do agendamento |
| **Aprovada** | A publicação foi aprovada e será agendada |
| **Rejeitada** | A publicação foi revisada e rejeitada — não será publicada |
| **Agendada** | A publicação está na fila para publicação na data e hora agendadas |
| **Publicada** | A publicação foi publicada com sucesso na rede social |
| **Pausada** | A publicação foi temporariamente pausada |
| **Falha** | A publicação não pôde ser publicada devido a um erro |

## Aprovação de publicações

As publicações são aprovadas na grade de publicações, onde você pode selecionar uma ou mais publicações e escolher aprovar ou rejeitar.

:::note
Haverá uma entrada para cada idioma e cada rede social.

Por exemplo, se você tiver 3 redes sociais e 3 idiomas selecionados, verá 9 entradas no calendário.
:::
