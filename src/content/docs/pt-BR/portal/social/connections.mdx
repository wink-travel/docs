Você já está conectado ao WinkLinks, nossa ferramenta de gerenciamento de links. Conecte-se a mais redes sociais clicando em `Conectar uma rede social`.
O processo irá redirecioná-lo para sua rede social para aprovar o app `Wink Social` para publicar conteúdo em seu nome.

Uma vez conectado, começamos a gerar conteúdo para essa rede para o próximo mês agendado.

:::note
O plano gratuito inclui o WinkLinks. Facebook, Instagram e outras plataformas estão disponíveis nos planos pagos.
:::

## Plataformas suportadas

| Plataforma | Descrição | Requisitos |
| -- | -- | -- |
| **WinkLinks** | Publicação automática em nosso avançado gerenciador de links | Incluído em todos os planos |
| **Facebook** | Automatiza publicações, respostas a comentários e DMs | Requer uma Página Comercial do Facebook |
| **Instagram** | Automatiza publicações, respostas a comentários e DMs | Requer uma Conta Comercial do Instagram vinculada a uma conta do Facebook |

Cada conexão mostra seu status atual: **Ativa**, **Desconectada** ou **Suspensa**.

:::note
Criamos e publicamos conteúdo otimizado para cada rede social.
:::
