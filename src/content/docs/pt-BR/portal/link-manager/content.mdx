Seu conteúdo está disponível na aba `Entradas`.

Suportamos três tipos principais de conteúdo:

- URLs publicamente disponíveis que linkam para páginas web.
- Arquivos que foram enviados através do WinkLinks.
- Conteúdo de texto criado por meio do nosso recurso `Citações`.

## Adicionar conteúdo

Veja como adicionar conteúdo ao WinkLinks. Na aba `Entradas`, comece clicando no botão `Adicionar entrada`.

### Link normal

:::tip
O link deve direcionar para uma página web. Por exemplo, você não pode linkar diretamente para um PDF. Se quiser compartilhar um PDF, precisa usar o recurso `Enviar arquivo`.
:::

Aqui estão os passos para adicionar um link normal:

<Steps>
1. Insira o link que deseja compartilhar. *ex: https://trvl.as/3xWCH*
2. Depois de inserir o link, o Wink busca os detalhes deste site e preenche o formulário que você verá a seguir.
3. Você pode optar por modificar alguns dos dados, como nome do site, título e descrição.
4. Também pode escolher enviar uma imagem personalizada.
5. Clique no botão `Salvar` quando estiver pronto para continuar.
</Steps>

### Enviar arquivo

:::caution[Requisitos]
- Imagens não podem exceder 10Mb de tamanho.
- Vídeos não podem exceder 50Mb de tamanho.
- Suportamos todos os formatos reconhecidos de imagem e vídeo, incluindo PDF.
:::

:::note[Recomendações]
- 2560w x 1440h (proporção 16x9) para sua imagem em destaque.
- 2560w x 1920h (proporção 4:3) para todas as outras imagens.
- O tamanho da imagem não deve ser maior que 5Mb. Por favor, comprima suas imagens antes de enviá-las para o Wink.
:::

Aqui estão os passos para enviar um arquivo:

<Steps>
1. A mídia pode ser adicionada:
    - Arrastando imagens do seu computador para a janela ou clicando em `Procurar`.
    - Colando uma URL externa que aponte para uma imagem.
    - Usando a câmera do seu laptop.
    - Da sua conta do Google Drive.
    - Da sua conta do Dropbox.
    - Do Shutterstock
    - Do gettyimages
    - Do iStock
    - Do Unsplash
2. Depois que o arquivo terminar de enviar, você pode adicionar mais metadados para descrever seu arquivo, como título e descrição.
3. Clique no botão `Salvar` quando estiver pronto para continuar.
</Steps>

### Escrever uma citação

:::tip
Uma citação é uma ótima forma de adicionar um delimitador para dividir uma lista longa [...e também faz você parecer inteligente e conhecedor 😉].
:::

Aqui estão os passos para escrever uma citação:

<Steps>
1. Digite a citação *ex: Rosas são vermelhas. Violetas são azuis...*.
2. Insira o nome do autor original. *ex: Edmund Spenser*
3. Insira o nome do site de onde você obteve a citação. *ex: Daily Quotes*
4. Clique no botão `Salvar` quando estiver pronto para continuar.
</Steps>

## Atualizar conteúdo

Todo conteúdo pode ser facilmente modificado clicando no link `Ações` e selecionando `Atualizar`. Você pode remover conteúdo de forma semelhante clicando no botão `Remover`.

## Remover conteúdo

Todo conteúdo pode ser facilmente modificado selecionando `Atualizar`. Você pode remover conteúdo de forma semelhante clicando primeiro no link `Ações` e depois no botão `Remover`.

## Ordenar conteúdo

Você pode ordenar seu conteúdo clicando no ícone localizado no canto inferior esquerdo de cada entrada e arrastando para a posição desejada.

## Agrupar conteúdo

:::tip
Crie suas tags antes de criar suas entradas. Caso contrário, você perde seu progresso e precisa começar novamente.
:::

Quando estiver editando uma entrada, você tem a opção de adicionar uma ou mais tags para fins de agrupamento. Role até o final do formulário para encontrar o campo `Tags`.

Os usuários, ao visualizarem sua página WinkLinks, podem escolher como querem consumir seu conteúdo (lista ou grade) e podem selecionar uma ou mais tags para filtrar ainda mais seu conteúdo.

## API

Desenvolvedores que desejam gerenciar o `WinkLinks` podem acessar [Developers > API > WinkLinks](/pt-BR/api).
