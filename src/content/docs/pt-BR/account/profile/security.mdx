A aba **Segurança** em [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) é onde você gerencia **passkeys** — credenciais WebAuthn que permitem fazer login sem senha.

## O que é um passkey?

Um passkey é uma credencial criptográfica vinculada ao seu dispositivo (ou gerenciador de senhas) que prova que você é você. Passkeys são resistentes a phishing e mais rápidas do que digitar uma senha.

## Registrando um passkey

1. Faça login em [iam.wink.travel](https://iam.wink.travel) com suas credenciais existentes.
2. Vá para **Minha Conta → Segurança**.
3. Clique em `Adicionar passkey`.
4. Seu navegador solicitará que você se autentique com seu dispositivo (Touch ID, Face ID, Windows Hello, uma chave de hardware ou um gerenciador de senhas sincronizado).
5. Dê um nome amigável ao passkey (ex: *MacBook do Trabalho*) para que você possa reconhecê-lo depois.

## Usando um passkey para fazer login

Na tela de login do [iam.wink.travel](https://iam.wink.travel), insira seu e-mail — se você tiver um passkey registrado para esse e-mail, será solicitado que você se autentique com ele em vez de digitar uma senha.

## Removendo um passkey

Na mesma aba Segurança, cada passkey registrado tem um botão `Remover`. Use-o se você perder um dispositivo, aposentar uma chave de hardware ou quiser limpar credenciais não usadas.

:::caution
Remover todos os seus passkeys deixa a senha como o único método de login. Certifique-se de lembrar sua senha antes de remover o último passkey.
:::
