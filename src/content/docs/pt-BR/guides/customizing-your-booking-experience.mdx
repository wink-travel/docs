As personalizações controlam como seu inventário é exibido para os viajantes — as cores do tema, moeda padrão, idioma, itinerário e mais. Elas se aplicam a links compartilháveis, cartões incorporados, grades, mapas e o motor de reservas.

## O que é uma personalização?

Uma personalização é um perfil de configuração que define:

- **Moeda de exibição** — Em qual moeda os preços são mostrados.
- **Idioma de exibição** — O idioma padrão para o conteúdo.
- **Itinerário padrão** — Datas de check-in/checkout e número de hóspedes pré-preenchidos.
- **Tema** — Cores e branding aplicados aos componentes incorporados.
- **Configuração** — Como o inventário é apresentado (layout, ordenação).

Você pode criar múltiplas personalizações para diferentes mercados ou casos de uso.

## Criando uma personalização

<Steps>
1. Acesse https://studio.wink.travel (ou sua conta em https://my.wink.travel).
2. Navegue até `Customization` no menu lateral.
3. Clique em `Create a new customization`.
4. Dê um nome descritivo (ex.: "Europe - EUR", "Japan Market", "Default").
5. Defina a moeda de exibição, idioma e itinerário padrão.
6. Configure as cores do tema para combinar com seu branding.
7. Clique em `Save`.
</Steps>

## Onde as personalizações são usadas

Uma vez criadas, as personalizações são referenciadas em toda a plataforma:

| Recurso | Como a personalização se aplica |
| -- | -- |
| **Links compartilháveis** | O link usa a moeda, idioma e itinerário da personalização como padrão |
| **Cartões incorporados** | Os cartões são exibidos com as cores do seu tema e configurações de exibição |
| **Grades e mapas** | Grades de inventário e marcadores no mapa seguem a personalização |
| **Motor de reservas** | Todo o fluxo de reserva usa as configurações da personalização |
| **Publicações sociais** | Postagens geradas por IA podem usar uma personalização específica para exibição de preços |

## Múltiplas personalizações

Um padrão comum é criar uma personalização para cada mercado-alvo:

- **"US Market"** — USD, inglês, estadia padrão de 2 noites.
- **"Europe"** — EUR, inglês, estadia padrão de 3 noites.
- **"Japan"** — JPY, japonês, estadia padrão de 1 noite.

Quando você cria um link compartilhável ou incorpora um cartão, seleciona qual personalização usar. Isso significa que a mesma propriedade pode ser apresentada de forma diferente dependendo do público.

## Sobrescrevendo via URL

Você também pode sobrescrever os padrões da personalização usando parâmetros na URL dos links compartilháveis:

- `l=es` — Sobrescreve o idioma para espanhol.
- `c=EUR` — Sobrescreve a moeda para EUR.
- `sd=2024-08-24&n=2` — Sobrescreve o itinerário para uma data e número de noites específicos.

Veja [The Book Now Button](/pt-BR/guides/the-book-now-button) para a lista completa de parâmetros de URL.
