Se você é uma empresa de viagens e deseja terceirizar a parte de pagamento e desembolso do seu negócio para o TripPay, aqui estão 6 passos básicos para começar:

<Steps>
1. Crie uma [conta de usuário](/pt-BR/user/register) no Wink.
2. Crie uma conta no [TripPay](https://pay.wink.travel).
3. Crie uma [Aplicação](/pt-BR/developers/applications) no TripPay que permita que você se comunique com nossa API.
4. Crie e mapeie seus beneficiários com o TripPay. *por exemplo, para a maioria, isso significa criar contas para as propriedades na sua plataforma que você deseja que o TripPay pague.*
5. Integre o [componente web de pagamento](/pt-BR/developers/web-components/#payment) no seu site.
6. Envie ao TripPay o `carrinho de compras` do viajante e inicie o processo de pagamento.
</Steps>

:::tip
Para um guia completo sobre como usar o TripPay para seu site de viagens, leia o guia [Integrando com o TripPay](/pt-BR/guides/integrating-with-trip-pay).
:::

## API

Desenvolvedores que desejam gerenciar `Pagamentos`, acessem [Desenvolvedores > API > Pagamentos](/pt-BR/developers/apis/#payment-api).
