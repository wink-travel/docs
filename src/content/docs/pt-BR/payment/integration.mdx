Se você é uma empresa de viagens e deseja terceirizar a parte de pagamento e desembolso do seu negócio para o TripPay, aqui estão 6 etapas básicas para começar:

<Steps>
  1. Criar um [conta de usuário](/user/register) no Wink.
  2. Crie uma conta com [TripPay](https://pay.wink.travel).
  3. Criar um [Aplicativo](/developers/applications) no TripPay que permite que você converse com nossa API.
  4. Crie e mapeie seus beneficiários com o TripPay. *Por exemplo: para a maioria, isso significa criar contas para as propriedades na sua plataforma que você deseja que o TripPay pague.*
  5. Integrar o [componente web de pagamento](/developers/web-components/#payment) no seu site.
  6. Envie ao TripPay o viajante `shopping cart` e iniciar o processo de pagamento.
</Steps>


:::dica
Para um guia completo sobre como usar o TripPay no seu site de viagens, leia o guia[Integração com TripPay](/guides/integrating-with-trip-pay).
:::
