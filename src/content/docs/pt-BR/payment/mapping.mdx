Se você é um integrador, como a Wink, e deseja mapear seus pagadores externos para uso com o TripPay, pode fazer isso de duas maneiras:

- Mapeamento manual
- Usando a [Payment API](/pt-BR/developers/apis/#payment-api) (*Veja `Account Mappings`*)

:::tip
A menos que você esteja lidando com apenas algumas propriedades iguais, recomendamos que use a API para automatizar esta etapa.
:::

A seguir, mostramos o passo a passo do mapeamento manual de um pagador externo. *Ele usa a API nos bastidores.*

<Steps>
1. Faça login no [TripPay](https://pay.wink.travel).
2. Selecione sua conta de integrador.
3. Clique em `Settings > Mappings` na barra de navegação principal.
4. Clique no botão `Create mapping`. Você verá o formulário de mapeamento.
5. **Local account** Comece a digitar o nome da conta TripPay existente que deseja mapear.
6. **External identifier** Cole seu ID externo para esta entidade conforme existe no sistema remoto.
7. Clique em `Save` para continuar. Você será redirecionado de volta para sua lista de mapeamentos.
</Steps>

Com seus mapeamentos configurados, o TripPay agora sabe quais contas creditar como beneficiárias de uma reserva quando você as referencia usando seus identificadores externos.

## API

Desenvolvedores que desejam gerenciar `Payments`, acessem [Developers > API > Payments](/pt-BR/developers/apis/#payment-api).
