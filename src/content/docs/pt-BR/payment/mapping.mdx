Se você for um integrador, como o Wink, e quiser mapear seus beneficiários externos para uso com o TripPay, você pode fazer isso de duas maneiras:

* Mapeamento manual
* Usando o[API de pagamento](/developers/apis/#payment-api)(*Ver `Account Mappings`*)

:::dica
A menos que você esteja lidando apenas com algumas das mesmas propriedades, recomendamos que você use a API para automatizar esta etapa.
:::

Abaixo, mostraremos como mapear manualmente um beneficiário externo.*Ele usa a API por baixo dos panos.*

<Steps>
  1. Entrar [TripPay](https://pay.wink.travel).
  2. Selecione sua conta de integrador.
  3. Clique `Settings > Mappings` na barra de navegação principal.
  4. Clique no `Create mapping` botão. Você verá o formulário de mapeamento.
  5. **Conta local** Comece a digitar o nome do TripPay existente que você deseja mapear.
  6. **Identificador externo** Cole seu ID externo para esta entidade como ele existe no sistema remoto.
  7. Clique `Save` para continuar. Você será redirecionado de volta para sua lista de mapeamentos.
</Steps>


Com seus mapeamentos em vigor, o TripPay agora sabe quais contas creditar como beneficiárias de uma reserva quando você se refere a elas usando seus identificadores externos.
