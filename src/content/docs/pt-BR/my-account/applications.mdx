As aplicações dão a você acesso a todos os recursos do Wink como desenvolvedor. Você pode acessar todos os mesmos dados e até estender os próprios recursos do Wink.
Você pode optar por usar o servidor OAuth2 do Wink para autenticar seu site de viagens da mesma forma que usaria Google ou Facebook.
Dessa forma, você pode estender os recursos do Wink para seu próprio site ou aplicativo.

:::note
Seu primeiro app já está criado quando você [cria uma nova conta](/pt-BR/my-account/create-account). Usamos esse app para alguns cenários de reserva.

Você não pode reutilizar esse app. Por favor, crie novos apps conforme precisar.
:::

Uma aplicação gerada no Wink pode ser usada para todos os endpoints da API.
:::

<Steps>
 1. Selecione a conta com a qual deseja trabalhar em [Contas](https://my.wink.travel).
 2. Clique em `Ações` na parte inferior do cartão da conta e escolha `Gerenciar conta`.
 3. Clique na aba `Aplicações` na página seguinte.
 4. Esta seção listará todas as suas aplicações.
 5. Escolha editar uma existente ou criar uma nova.
 </Steps>

## Criar app

Como você não pode acessar a `chave secreta` do app que criamos para você, será necessário criar o seu próprio.

:::caution
Você precisa criar uma conta afiliada no Wink ou uma conta no TripPay antes de criar um app, pois você irá vincular sua aplicação a uma conta específica.
:::

Aqui estão os passos para criar uma aplicação:

<Steps>
1. Na página de apps, clique no botão `Criar nova aplicação`.
2. ***Nome*** Dê um nome ao seu app. *ex: App Legal*
3. **Nome da entidade** Vincule seu app a uma das suas contas existentes. *ex: Conta Legal*
4. **URIs de redirecionamento** É necessário pelo menos um URI de redirecionamento. Ele é usado para validar o domínio de redirecionamento após uma autenticação bem-sucedida. *ex: https://www.site-legal.com*
5. Clique no botão `Salvar` para continuar.
</Steps>

Você será redirecionado de volta para sua lista de aplicações. Você verá seu novo app na lista e terá acesso à chave secreta do app apenas esta vez. Certifique-se de copiá-la e guardá-la em um local seguro.

## Atualizar app

Você precisará atualizar seu app se algum dos seus URIs de redirecionamento mudar ou se quiser adicionar ou remover um URI.

Para atualizar um app existente, faça o seguinte:

<Steps>
1. Clique no link `Ações` do app que deseja atualizar.
2. Clique no botão `Atualizar` em Ações.
3. Faça as alterações no app.
4. Clique no botão `Salvar` para continuar.
</Steps>

Você será redirecionado de volta para sua lista de aplicações.

## Remover app

:::caution
Por favor, não exclua o app que criamos automaticamente para você para garantir que todos os recursos funcionem para sua conta.
:::

Se você não tiver mais uso para um app, pode removê-lo.

Para remover um app, faça o seguinte:

<Steps>
1. Clique no link `Ações` do app que deseja remover.
2. Clique no botão `Remover` em Ações.
3. Clique no botão `OK` para confirmar a exclusão.
</Steps>

Você será redirecionado de volta para sua lista de aplicações.
