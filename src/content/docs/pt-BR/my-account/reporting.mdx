O Wink oferece um sistema de relatórios que permite gerar e exportar dados da sua conta. Os relatórios estão disponíveis em formato CSV para fácil importação em planilhas, ferramentas de análise ou seus próprios sistemas.

## Tipos de relatório

Você pode gerar relatórios para as seguintes categorias de dados:

| Tipo de relatório | O que inclui |
|---|---|
| **Propriedade** | Detalhes da propriedade, status do anúncio, resumo do inventário. |
| **Usuário** | Contas de usuário, funções, resumo de atividades. |
| **Reserva** | Histórico de reservas, status, receita, detalhes dos hóspedes. |
| **Conta** | Informações da conta, saldos, histórico de transações. |

## Como gerar um relatório

1. Navegue até a seção de relatórios no painel da sua conta.
2. Selecione o **tipo de relatório** que deseja gerar.
3. Configure quaisquer **filtros** para restringir os dados (ex.: intervalo de datas, status, propriedade).
4. Escolha os **campos** que deseja incluir na exportação. Cada tipo de relatório oferece um conjunto de campos disponíveis para seleção.
5. Clique em **Exportar** para baixar o relatório em arquivo CSV.

## Campos personalizados

Os relatórios suportam seleção personalizada de campos. Você escolhe quais colunas incluir na exportação, para obter exatamente os dados que precisa — nada mais, nada menos.

## Casos de uso

- **Conciliação de receita** — Exporte relatórios de reservas com detalhes financeiros para conciliar com seu sistema contábil.
- **Análise de desempenho** — Exporte relatórios de propriedades para comparar o desempenho dos anúncios em seu portfólio.
- **Insights sobre hóspedes** — Exporte dados de reservas para entender a demografia dos viajantes e padrões de reserva.
- **Auditoria de contas** — Exporte dados de contas para conformidade ou revisão interna.
