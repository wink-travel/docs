Um usuário Wink pode escolher permitir que você gerencie uma de suas contas de propriedade, afiliado ou agente de viagens. Quando isso acontecer, você receberá um e-mail com uma solicitação de convite.

import { Steps } from '@astrojs/starlight/components';

Para ver seus convites Wink atuais, siga estes passos:

<Steps>
 1. Faça login em [my.wink.travel](https://my.wink.travel).
 2. Clique em `Convites` na barra lateral esquerda.
 3. Você verá uma lista de todos os convites atuais.
 4. Esta seção listará todas as suas aplicações.
 5. Escolha `Aceitar` ou `Rejeitar` o convite.
 6. Se você aceitou o convite, a conta aparecerá na seção `Contas`.
</Steps>
