Um usuário Wink pode escolher permitir que você gerencie uma de suas contas de propriedade, afiliado ou agente de viagens. Quando isso acontecer, você receberá um e-mail com uma solicitação de convite.

import { Steps } from '@astrojs/starlight/components';

Para ver seus convites Wink atuais, siga estes passos:

<Steps>
1. Você deve estar logado em um de nossos sites.
2. Clique no ícone do seu perfil, no canto superior direito, e um menu aparecerá.
3. Clique no link `Convites`.
4. Interaja com seus convites aceitando ou rejeitando-os.
</Steps>
