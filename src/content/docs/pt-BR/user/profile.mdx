Se você quiser atualizar seu perfil de usuário para sua conta de usuário Wink, siga estas etapas:

<Steps>
  1. Você deve estar logado em um de nossos sites.
  2. Clique no ícone do seu perfil, no canto superior direito, e um menu aparecerá.
  3. Clique no `Profile` link.
  4. Siga as instruções para atualizar seu `e-mail`, `first` e `last name`.
  5. Você também pode fazer upload de um novo `profile picture` de si mesmo.
  6. Clique `Save`.
</Steps>


:::nota
Sua nova foto de perfil será exibida na próxima vez que você sair completamente do Wink.
:::
