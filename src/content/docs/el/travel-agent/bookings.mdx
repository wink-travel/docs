Όταν γίνεται κράτηση στο Wink, συμβαίνουν τα εξής:

<Steps>
  1. Ο διαχειριστής καναλιών σας, το CRS ή το PMS ειδοποιείται.
  2. Λαμβάνετε ένα email επιβεβαίωσης *(επικοινωνία με το γραφείο κρατήσεων)* με έναν σύνδεσμο για τη διαχείριση της νέας σας κράτησης.
  3. Ο ταξιδιώτης λαμβάνει ένα email επιβεβαίωσης που περιλαμβάνει επίσης τον τρόπο επικοινωνίας με το γραφείο κρατήσεων.
  4. Ο ταξιδιωτικός πράκτορας λαμβάνει την ίδια επιβεβαίωση κράτησης με τον ταξιδιώτη.
  5. Οι προγραμματιστές μπορούν να ακούσουν το `booking.created` webhook event και λάβετε λεπτομερείς πληροφορίες κράτησης από εμάς σε πραγματικό χρόνο.
</Steps>


Ακολουθήστε τον σύνδεσμο στο email ή μεταβείτε στη διεύθυνση`Account > Bookings`από την κύρια γραμμή πλοήγησης για να ξεκινήσετε.

## Πλέγμα κρατήσεων

Το πλέγμα κρατήσεων σας δείχνει όλες τις προηγούμενες και μελλοντικές κρατήσεις σας, ταξινομημένες κατά ημερομηνία κράτησης σε αύξουσα σειρά.

### Φίλτρα

Μπορείτε να εφαρμόσετε φίλτρα για να βρείτε τις κρατήσεις που αναζητάτε.

* Ημερομηνία κράτησης \[εύρος]
* Ημερομηνία άφιξης \[εύρος]
* Ημερομηνία αναχώρησης \[εύρος]
* Κωδικός κράτησης
* Επώνυμο του ταξιδιώτη
* Όνομα ταξιδιώτη
* Όνομα κύριας τιμής
* Όνομα ακινήτου

### Εξαγωγή

Μπορείτε να εξαγάγετε κρατήσεις που ταιριάζουν με τα τρέχοντα κριτήρια φίλτρου σας σε CSV.

Για να εξαγάγετε τις κρατήσεις σας:

<Steps>
  1. Βεβαιωθείτε ότι τα φίλτρα που επιθυμείτε είναι ενεργά.
  2. Περιμένετε να ανανεωθεί το πλέγμα κρατήσεων.
  3. Κάντε κλικ στο 🗂️ (*Εξαγωγή αρχείων*) κουμπί εικονιδίου στην κάτω δεξιά γωνία του πλέγματος.
</Steps>


## Εγγραφή

Επιλέξτε την κράτηση με την οποία θέλετε να εργαστείτε από το πλέγμα κρατήσεων και θα μεταφερθείτε στο`Booking`σελίδα.
Αυτή η σελίδα περιέχει όλες τις λεπτομέρειες που πρέπει να γνωρίζετε για να καλωσορίσετε σωστά τον επισκέπτη σας την ημέρα της άφιξης.

Η σελίδα περιέχει επίσης μερικές σημαντικές ενέργειες που χρειάζεστε για να εξυπηρετήσετε σωστά τους καλεσμένους σας. Μπορείτε να βρείτε όλες τις διαθέσιμες ενέργειες στο`Actions`αναπτυσσόμενο μενού στο επάνω μέρος της σελίδας.

### Επανάληψη αποστολής ηλεκτρονικού ταχυδρομείου

Για να στείλετε ξανά το email επιβεβαίωσης στον ταξιδιώτη, κάντε κλικ στο`Actions > Resend confirmation email`.

### Ακύρωση κράτησης

Για να ακυρώσετε την κράτηση ως κατάλυμα, κάντε κλικ στο`Actions > Cancel booking`.

Θα εμφανιστεί ένα αναδυόμενο παράθυρο που σας επιτρέπει:

<Steps>
  1. Επιλέξτε τον τύπο ακύρωσης. *π.χ. Λανθασμένες ημερομηνίες*
  2. Πληκτρολογήστε έναν λόγο για την ακύρωση.
</Steps>


*Αυτό το κουμπί ακολουθεί τους κανόνες της πολιτικής ακύρωσης και επομένως δεν θα είναι πάντα διαθέσιμο.*

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Bookings`, ως ταξιδιωτικός πράκτορας, μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Ταξιδιωτικός πράκτορας](/developers/apis/#travel-agent-api).
