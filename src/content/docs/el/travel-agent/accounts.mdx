:::note
Αυτό το άρθρο προϋποθέτει ότι έχετε ήδη λογαριασμό χρήστη στο Wink.
:::

Το [Wink Agent](https://agent.wink.travel) σας συνδέει με ταξιδιωτικό απόθεμα παγκοσμίως. Αυτό το άρθρο σας καθοδηγεί στη δημιουργία και διαχείριση των λογαριασμών ταξιδιωτικού πράκτορα στο Wink.

## Δημιουργία λογαριασμού

Μόλις πιστοποιηθείτε, θα ανακατευθυνθείτε στη [σελίδα λογαριασμών](https://agent.wink.travel/company/list) που εμφανίζει όλους τους λογαριασμούς ταξιδιωτικού πράκτορα σας.

Κάντε κλικ στο κουμπί `Create new account` για να συνεχίσετε. Θα ανακατευθυνθείτε στη σελίδα [δημιουργίας λογαριασμού](https://agent.wink.travel/company/create).

Συμπληρώστε όλα τα απαιτούμενα πεδία.

<Steps>
1. **Όνομα** Το όνομα του λογαριασμού σας. *Δεν χρειάζεται να είναι το πραγματικό σας όνομα, αλλά κάτι που θα βλέπουν οι χρήστες όταν επισκέπτονται τη σελίδα σας στο WinkLinks ή όταν οι προμηθευτές θέλουν να συνδεθούν απευθείας μαζί σας.*
2. **Τύπος οντότητας** Αν είστε εταιρεία ή ιδιώτης. *Χρησιμοποιείται κυρίως για να καθοριστεί η επιλεξιμότητά σας για λήψη χρημάτων.*
3. **Τύπος** Επιλέξτε τον τύπο που ταιριάζει καλύτερα στον λογαριασμό σας. *π.χ. Influencer*
4. **Περιγραφή** Γράψτε μια περιγραφή για τον λογαριασμό σας. *Αυτή θα εμφανίζεται στη σελίδα σας στο WinkLinks και όταν οι προμηθευτές θέλουν να συνδεθούν απευθείας μαζί σας.*
5. **Ιστότοπος** Εισάγετε τον κύριο ιστότοπό σας. *Μπορεί να είναι το URL του λογαριασμού σας στο IG ή ο ιστότοπος της εταιρείας σας. Πρέπει να είναι το μέρος όπου κάνετε το μεγαλύτερο μέρος της διαδικτυακής σας εργασίας. Οι προμηθευτές θα επισκέπτονται αυτούς τους συνδέσμους για να αποφασίσουν αν θα συνδεθούν μαζί σας.*
6. **Πόλη** Πού εδρεύετε;
</Steps>

### Όροι πληρωμής

:::note
Αν επιλέξετε να είστε υπεύθυνοι για την πληρωμή, θα έχετε πρόσβαση μόνο σε καταλύματα με τα οποία έχετε άμεση σύνδεση.
:::

:::caution
Το Wink δεν αναλαμβάνει καμία ευθύνη για την πληρωμή των ξενοδοχείων σε αυτή την περίπτωση.
:::

:::note
Μπορείτε να αλληλεπιδράσετε με αυτήν την επιλογή μόνο κατά τη δημιουργία του λογαριασμού. Οι περισσότεροι ταξιδιωτικοί πράκτορες θα πρέπει να αφήσουν αυτήν τη ρύθμιση απενεργοποιημένη 🛑 και να χρησιμοποιούν αυτήν την πύλη.
:::

Οι εγγεγραμμένοι ταξιδιωτικοί πράκτορες που ενσωματώνονται μαζί μας μέσω του API μας, έχουν την επιλογή να λαμβάνουν πληρωμές και να διανέμουν αυτά τα κεφάλαια στα καταλύματά μας.

<Steps>
1. **Θέλω να λαμβάνω πληρωμές** Ο ταξιδιωτικός πράκτορας αναλαμβάνει την ευθύνη συλλογής των χρημάτων.
2. **Θέλω να πληρώνω τα ξενοδοχεία** Ο ταξιδιωτικός πράκτορας είναι υπεύθυνος για την πληρωμή των ξενοδοχείων.
</Steps>

Τα ξενοδοχεία που θέλουν να συνδεθούν απευθείας μαζί σας θα δουν μια προειδοποίηση και θα πρέπει να κάνουν κλικ για να αποδεχτούν ότι το Wink δεν είναι υπεύθυνο για την πληρωμή των ξενοδοχείων, προκειμένου να δημιουργηθεί η σχέση με τον λογαριασμό σας.

### Συμφωνία

<Steps>
1. **Συμφωνία** Αποδεχτείτε τη [συμφωνία συνεργάτη](/el/travel-agent/terms-of-service) και τους [όρους πληρωμής](/el/travel-agent/payment-terms) για να συνεχίσετε.
2. Κάντε κλικ στο κουμπί `Save`.
</Steps>

Συγχαρητήρια 🎉 ...δημιουργήσατε τον πρώτο σας λογαριασμό συνεργάτη στο Wink.

Μόλις δημιουργηθεί, μπορείτε να προσαρμόσετε περαιτέρω τον λογαριασμό σας συμπληρώνοντας τη διεύθυνσή σας και προσθέτοντας μια φωτογραφία προφίλ.

<Steps>
1. Κάντε κλικ στον νέο σας λογαριασμό ταξιδιωτικού πράκτορα. *Αυτό θα σας μεταφέρει στον πίνακα ελέγχου του λογαριασμού σας.*
2. Κάντε κλικ στο `Account > Profile` στη βασική γραμμή πλοήγησης.
3. Στη σελίδα Προφίλ, θα μπορείτε να ενημερώσετε το υπάρχον προφίλ σας και να προσθέσετε επιπλέον πληροφορίες.
</Steps>

:::note
Η φωτογραφία προφίλ του λογαριασμού ταξιδιωτικού πράκτορα δεν πρέπει να συγχέεται με τη φωτογραφία προφίλ χρήστη του Wink. Η φωτογραφία προφίλ του λογαριασμού ταξιδιωτικού πράκτορα θα εμφανίζεται στη σελίδα σας στο WinkLinks και θα τη βλέπουν οι προμηθευτές που θέλουν να συνδεθούν μαζί σας.

Η φωτογραφία προφίλ χρήστη προορίζεται κυρίως για εσωτερική χρήση.
:::

## Διαχειριστές

Μπορείτε να προσκαλέσετε άλλους χρήστες να βοηθήσουν στη διαχείριση του λογαριασμού ταξιδιωτικού πράκτορα σας. Μόλις ένας χρήστης αποδεχτεί την πρόσκλησή σας, θα έχει πλήρη πρόσβαση στον λογαριασμό σας εκτός από τις πληρωμές. Εσείς θα είστε πάντα ο μόνος υπεύθυνος για τα κεφάλαια που κατέχετε.

<Steps>
1. Επιλέξτε τον λογαριασμό ταξιδιωτικού πράκτορα με τον οποίο θέλετε να εργαστείτε από τη [σελίδα λογαριασμών](https://agent.wink.travel/company/list).
2. Κάντε κλικ στο `Account > Managers` στη βασική γραμμή πλοήγησης.
3. Προσθέστε το email του χρήστη που θέλετε να ορίσετε ως διαχειριστή λογαριασμού.
4. Κάντε κλικ στον σύνδεσμο `Invite` δίπλα στο email.
</Steps>

Το Wink θα στείλει στον χρήστη ένα email για να αποδεχτεί ή να απορρίψει την πρόσκλησή σας.

## Ειδοποιήσεις

Κατά καιρούς, θέλουμε να επικοινωνήσουμε μαζί σας. Όταν συμβεί αυτό, το εικονίδιο του κουδουνιού στην πάνω δεξιά γωνία θα αλλάξει.
Για να διαβάσετε τι σας στείλαμε:

<Steps>
1. Κάντε κλικ στο `Account > Notifications` στη βασική γραμμή πλοήγησης.
2. Θα δείτε νέες και παλιές ειδοποιήσεις.
3. Επιλέξτε `Mark as read` ή `Delete` τις ειδοποιήσεις μόλις τις διαβάσετε.
</Steps>

## API

Οι ταξιδιωτικοί πράκτορες θεωρούνται ακόμα συνεργάτες. Οι προγραμματιστές που θέλουν να διαχειριστούν `Affiliates` μπορούν να μεταβούν στο [Developers > API > Affiliate](/el/developers/apis/#affiliate-api).
