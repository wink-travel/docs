:::σημείωση
Αυτό το άρθρο προϋποθέτει ότι έχετε ήδη έναν λογαριασμό χρήστη Wink.
:::

[Πράκτορας Wink](https://agent.wink.travel)σας συνδέει με ταξιδιωτικά αποθέματα παγκοσμίως. Αυτό το άρθρο σας καθοδηγεί στη δημιουργία και τη διαχείριση των λογαριασμών σας ως ταξιδιωτικού πράκτορα στο Wink.

## Δημιουργία λογαριασμού

Μόλις ολοκληρωθεί η επαλήθευση, θα μεταφερθείτε στη διεύθυνση[σελίδα λογαριασμών](https://agent.wink.travel/company/list)που σας δείχνει όλους τους λογαριασμούς σας ως ταξιδιωτικός πράκτορας.

Κάντε κλικ στο`Create new account`κουμπί για να συνεχίσετε. Θα μεταφερθείτε στο[δημιουργία λογαριασμού](https://agent.wink.travel/company/create)σελίδα.

Συμπληρώστε όλα τα υποχρεωτικά πεδία.

<Steps>
  1. **Ονομα** Το όνομα του λογαριασμού σας. *Αυτό δεν χρειάζεται να είναι το πραγματικό σας όνομα, αλλά θα είναι κάτι που θα βλέπουν οι χρήστες όταν βλέπουν τη σελίδα σας στο WinkLinks ή όταν οι προμηθευτές θέλουν να επικοινωνήσουν απευθείας μαζί σας.*
  2. **Τύπος οντότητας** Είτε είστε εταιρεία είτε ιδιώτης. *Χρησιμοποιείται κυρίως για να προσδιοριστεί ο τρόπος με τον οποίο δικαιούστε να λάβετε κεφάλαια.*
  3. **Τύπος** Επιλέξτε έναν τύπο που ταιριάζει καλύτερα στον λογαριασμό σας. *π.χ. Influencer*
  4. **Περιγραφή** Γράψτε μια περιγραφή για τον λογαριασμό σας. *Αυτό θα εμφανίζεται στη σελίδα σας στο WinkLinks και όταν οι προμηθευτές θέλουν να επικοινωνήσουν απευθείας μαζί σας.*
  5. **Δικτυακός τόπος** Εισαγάγετε τον κύριο ιστότοπό σας. *Αυτή μπορεί να είναι η διεύθυνση URL του λογαριασμού σας στο IG ή ο ιστότοπος της εταιρείας. Θα πρέπει να είναι το μέρος όπου κάνετε το μεγαλύτερο μέρος της διαδικτυακής σας εργασίας. Οι προμηθευτές θα μεταβαίνουν σε αυτούς τους συνδέσμους όταν αποφασίζουν εάν θα συνδεθούν μαζί σας.*
  6. **Πόλη** Πού εδρεύετε;
</Steps>


### Όροι πληρωμής

:::σημείωση
Εάν επιλέξετε να είστε υπεύθυνοι για την πληρωμή, θα έχετε πρόσβαση μόνο σε ακίνητα με τα οποία έχετε άμεση σύνδεση.
:::

:::προσοχή
Η Wink δεν φέρει καμία ευθύνη για την πληρωμή ξενοδοχείων σε αυτό το σενάριο.
:::

:::σημείωση
Μπορείτε να αλληλεπιδράσετε με αυτήν την επιλογή μόνο κατά τη δημιουργία λογαριασμού. Οι περισσότεροι ταξιδιωτικοί πράκτορες θα πρέπει να αφήσουν αυτήν τη ρύθμιση απενεργοποιημένη 🛑 και να χρησιμοποιήσουν αυτήν την πύλη.
:::

Οι εγγεγραμμένοι ταξιδιωτικοί πράκτορες που ενσωματώνονται σε εμάς, χρησιμοποιώντας το API μας, έχουν την επιλογή να εισπράττουν πληρωμές και να καταβάλλουν αυτά τα χρήματα στα καταλύματά μας.

<Steps>
  1. **Θέλω να λάβω πληρωμή** Ο ταξιδιωτικός πράκτορας αναλαμβάνει την ευθύνη για την είσπραξη των χρημάτων.
  2. **Θέλω να πληρώσω ξενοδοχεία** Ο ταξιδιωτικός πράκτορας είναι υπεύθυνος για την πληρωμή των ξενοδοχείων.
</Steps>


Τα ξενοδοχεία που θέλουν να συνδεθούν απευθείας μαζί σας θα δουν μια προειδοποίηση και θα πρέπει να κάνουν κλικ για να αναγνωρίσουν ότι η Wink δεν φέρει ευθύνη για την πληρωμή των ξενοδοχείων, προκειμένου να δημιουργήσουν μια σχέση με τον λογαριασμό σας.

## Διαχειρίζομαι

<figure>
  <CloudinaryImage width="350" height="445" resize="scale" quality="best" alt="Λογαριασμός ταξιδιωτικού πράκτορα" src="academy/travel-agent" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Είσοδος λογαριασμού ταξιδιωτικού πράκτορα*</figcaption>
</figure>


Μόλις δημιουργηθεί, μπορείτε να προσαρμόσετε περαιτέρω τον λογαριασμό σας συμπληρώνοντας τη διεύθυνσή σας και προσθέτοντας μια εικόνα προφίλ στον λογαριασμό σας.

<Steps>
  1. Κάντε κλικ στον νέο σας λογαριασμό ταξιδιωτικού πράκτορα. *Αυτό θα σας μεταφέρει στον πίνακα ελέγχου του λογαριασμού σας.*
  2. Κάντε κλικ στο `Account > Profile` στην κύρια γραμμή πλοήγησης.
  3. Στη σελίδα Προφίλ, θα μπορείτε να ενημερώσετε το υπάρχον προφίλ σας και να προσθέσετε επιπλέον πληροφορίες σε αυτό.
</Steps>


:::σημείωση
Η εικόνα προφίλ του λογαριασμού σας στο ταξιδιωτικό πρακτορείο δεν πρέπει να συγχέεται με την εικόνα προφίλ χρήστη στο Wink. Η εικόνα προφίλ του λογαριασμού σας στο ταξιδιωτικό πρακτορείο θα εμφανίζεται στη σελίδα σας στο WinkLinks και θα είναι ορατή σε προμηθευτές που επιθυμούν να συνδεθούν μαζί σας.

Η εικόνα προφίλ χρήστη σας προορίζεται κυρίως για εσωτερική χρήση.
:::

## Διευθυντές

Μπορείτε να προσκαλέσετε άλλους χρήστες να σας βοηθήσουν στη διαχείριση του λογαριασμού σας ως ταξιδιωτικού πράκτορα. Μόλις ένας χρήστης αποδεχτεί την πρόσκλησή σας, θα έχει πλήρη πρόσβαση στον λογαριασμό σας, με εξαίρεση την πληρωμή. Θα είστε πάντα ο μόνος υπεύθυνος για τα χρήματα που σας ανήκουν.

<Steps>
  1. Επιλέξτε τον λογαριασμό ταξιδιωτικού πράκτορα με τον οποίο θέλετε να συνεργαστείτε από το [σελίδα λογαριασμών](https://agent.wink.travel/company/list).
  2. Κάντε κλικ στο `Account > Managers` στην κύρια γραμμή πλοήγησης.
  3. Προσθέστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του χρήστη που θέλετε να ορίσετε ως διαχειριστή λογαριασμού.
  4. Κάντε κλικ στο `Invite` σύνδεσμος δίπλα στο email.
</Steps>


Το Wink θα στείλει στον χρήστη ένα email για να αποδεχτεί ή να απορρίψει την πρόσκλησή σας.

## Ειδοποιήσεις

Κατά καιρούς, θέλουμε να επικοινωνήσουμε μαζί σας. Όταν συμβεί αυτό, θα δείτε το κουδουνάκι στην επάνω δεξιά γωνία να αλλάζει.
Για να διαβάσετε τι σας στείλαμε:

<Steps>
  1. Κάντε κλικ στο `Account > Notifications` στην κύρια γραμμή πλοήγησης.
  2. Θα δείτε νέες και παλαιότερες ειδοποιήσεις.
  3. Επιλέξτε να `Mark as read` ή `Delete` τις ειδοποιήσεις μόλις τις διαβάσετε.
</Steps>


## API

Οι ταξιδιωτικοί πράκτορες εξακολουθούν να θεωρούνται συνεργαζόμενες εταιρείες. Οι προγραμματιστές που θέλουν να διαχειρίζονται`Affiliates`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Συνεργάτης](/developers/apis/#affiliate-api).
