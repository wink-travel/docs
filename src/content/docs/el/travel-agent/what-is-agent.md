Η λύση μας για εγγεγραμμένους ταξιδιωτικούς πράκτορες είναι το [Wink Agent](https://agent.wink.travel). Είναι μια πλατφόρμα αυτοεξυπηρέτησης που μπορούν να χρησιμοποιούν οι πράκτορες για να αναζητούν και να κάνουν κρατήσεις εκ μέρους των πελατών τους και να κερδίζουν προμήθεια για κάθε κράτηση.

Οι πιο προχωρημένοι ταξιδιωτικοί πράκτορες μπορούν να επιλέξουν να ενσωματωθούν σε επίπεδο API για σκοπούς διαχείρισης πληρωμών και αποπληρωμών εκτός της πλατφόρμας Wink.

Οι ταξιδιωτικοί πράκτορες έχουν ένα υποσύνολο των ίδιων λειτουργιών της πλατφόρμας με το [Wink Studio](/el/studio/what-is-studio), οπότε δεν θα τις καλύψουμε ξανά εδώ.

## Αναζήτηση

Μεταβείτε στο [Search](/el/studio/search) για να μάθετε πώς να αναζητάτε και να κάνετε κρατήσεις ξενοδοχείων.

Αλλιώς, συνεχίστε στο επόμενο τμήμα.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν το `Travel Agent` μπορούν να μεταβούν στο [Developers > API > Travel Agent](/el/developers/apis/#travel-agent-api).
