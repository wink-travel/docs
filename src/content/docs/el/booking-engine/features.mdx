Η μηχανή κρατήσεων διατίθεται σε δύο εκδόσεις για να υποστηρίξει τις διαφορετικές ανάγκες ενός συνεργάτη, μιας εταιρείας διαχείρισης ακινήτων και ενός ξενοδοχείου.

## Online travel agency

Για έναν συνεργάτη, έχει περισσότερο νόημα να χρησιμοποιήσει την έκδοση της online ταξιδιωτικής πλατφόρμας. Ο λόγος είναι ότι ο ιστότοπος έχει διαμορφωθεί για να εξυπηρετεί ταξιδιώτες που θέλουν να δουν πολλαπλές ιδιοκτησίες.

Μπορείτε να βρείτε την αρχική σελίδα αυτής της εφαρμογής παρακάτω.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Internet booking engine

Για ένα ξενοδοχείο ή μια εταιρεία διαχείρισης ακινήτων, έχει περισσότερο νόημα να χρησιμοποιήσουν την έκδοση της μηχανής κρατήσεων στο διαδίκτυο. Αυτός ο τύπος χρήστη ενδιαφέρεται να αναδείξει μια μεμονωμένη ιδιοκτησία ή πολλαπλές ιδιοκτησίες υπό διαχείριση.

Μπορείτε να βρείτε την αρχική σελίδα αυτής της εφαρμογής παρακάτω.

## Διαμόρφωση

Τις περισσότερες φορές, δεν υπάρχει ανάγκη να μοιραστείτε απευθείας καμία από αυτές τις εφαρμογές. Έχουμε δημιουργήσει διάφορους τρόπους για να μοιραστείτε το ταξιδιωτικό απόθεμα, τους οποίους μπορείτε να ανακαλύψετε στο [Wink Studio](/el/studio/what-is-studio) ή μέσω των [WinkLinks](/el/link-manager/wink-links).

Ωστόσο, αν θέλετε να συνδέσετε χειροκίνητα σε έναν από αυτούς τους ιστότοπους, βεβαιωθείτε ότι προσθέτετε το client ID και το customization ID στη διεύθυνση URL ως παραμέτρους.

Μπορείτε να βρείτε το client ID σας ως εξής:

<Steps>
1. Συνδεθείτε είτε στο [Wink Extranet](/el/extranet/what-is-extranet) είτε στο [Wink Studio](/el/studio/what-is-studio).
2. Κάντε κλικ στο εικονίδιο του προφίλ σας στην πάνω δεξιά γωνία και επιλέξτε [Applications](/el/developers/applications).
3. Χρησιμοποιήστε την εφαρμογή που συνδέεται με τον λογαριασμό Wink που θέλετε να εργαστείτε.
4. Αντιγράψτε το Client ID από αυτήν την εφαρμογή και επικολλήστε το στη διεύθυνση URL ως παράμετρο αιτήματος. Παράδειγμα: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Μπορείτε να βρείτε το customization ID σας ως ιδιοκτησία ως εξής:

<Steps>
1. Συνδεθείτε στο [Wink Extranet](/el/extranet/what-is-extranet).
2. Επιλέξτε την ιδιοκτησία με την οποία θέλετε να εργαστείτε.
3. Από τη βασική γραμμή πλοήγησης, κάντε κλικ στο `Account > Branding`
4. Αντιγράψτε το Customization ID και επικολλήστε το στη διεύθυνση URL ως παράμετρο αιτήματος. Παράδειγμα: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Μπορείτε να βρείτε το customization ID σας ως συνεργάτης ως εξής:

<Steps>
1. Συνδεθείτε στο [Wink Studio](/el/studio/what-is-studio).
2. Επιλέξτε τον λογαριασμό με τον οποίο θέλετε να εργαστείτε.
3. Από τη βασική γραμμή πλοήγησης, κάντε κλικ στο `Tools > Customization`
4. Αντιγράψτε το Customization ID από την προσαρμογή που θέλετε να χρησιμοποιήσετε και επικολλήστε το στη διεύθυνση URL ως παράμετρο αιτήματος. Παράδειγμα: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Το αποτέλεσμα είναι μια διεύθυνση URL που μοιάζει κάπως έτσι:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Αν θέλετε να μεταβείτε απευθείας σε μια σελίδα ξενοδοχείου, η διεύθυνση URL θα μοιάζει κάπως έτσι:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Αν δεν συμπεριλάβετε client και / ή configuration ID, το Wink OTA θα χρησιμοποιήσει μια προεπιλεγμένη διαμόρφωση.
:::

## IBE

Αν είστε εταιρεία διαχείρισης ακινήτων, θέλετε να συνδέσετε στην αρχική σελίδα του Wink IBE. Η αρχική σελίδα θα εμφανίσει το καθορισμένο θέμα σας μαζί με τις [ιδιοκτησίες που εκπροσωπείτε](/el/studio/agency-representation).

Το αποτέλεσμα είναι μια διεύθυνση URL που μοιάζει κάπως έτσι:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Αν είστε ιδιοκτησία, θέλετε να συνδέσετε απευθείας στη σελίδα προορισμού της ιδιοκτησίας στο Wink IBE.

Αυτή η διεύθυνση URL θα μοιάζει κάπως έτσι:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
Αν δεν συμπεριλάβετε client και / ή configuration ID, το Wink IBE ΔΕΝ θα φορτώσει. Αυτές οι παράμετροι είναι απαραίτητες.
:::

## Παράδειγμα: Σελίδα ιδιοκτησίας

Κάθε ιδιοκτησία στο Wink έχει τη δική της μοναδική σελίδα προορισμού σε κάθε ιστότοπο. Όλοι οι εξωτερικοί σύνδεσμοι κατευθύνουν τους ταξιδιώτες σε μία από αυτές τις σελίδες προορισμού.

Η σελίδα προορισμού περιέχει όλες τις πληροφορίες που χρειάζεται ένας ταξιδιώτης για να κάνει μια ενημερωμένη επιλογή σχετικά με το αν θα μείνει σε αυτή την ιδιοκτησία.
