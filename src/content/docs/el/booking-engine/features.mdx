Η μηχανή κρατήσεων διατίθεται σε δύο τύπους για να υποστηρίξει τις διαφορετικές ανάγκες ενός συνεργάτη, μιας εταιρείας διαχείρισης ακινήτων και ενός ξενοδοχείου.

## Διαδικτυακό ταξιδιωτικό γραφείο

Για έναν συνεργάτη, είναι πιο λογικό να χρησιμοποιεί την έκδοση του διαδικτυακού ταξιδιωτικού γραφείου. Ο λόγος είναι ότι ο ιστότοπος έχει ρυθμιστεί για να εξυπηρετεί ταξιδιώτες που θέλουν να δουν πολλά ακίνητα.

Μπορείτε να βρείτε την αρχική σελίδα για αυτήν την εφαρμογή παρακάτω.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>


## Μηχανή κρατήσεων στο διαδίκτυο

Για ένα ξενοδοχείο ή μια εταιρεία διαχείρισης ακινήτων, είναι πιο λογικό να χρησιμοποιείται η έκδοση της μηχανής κρατήσεων μέσω διαδικτύου. Αυτός ο τύπος χρήστη ενδιαφέρεται να επισημάνει ένα μόνο ακίνητο ή πολλά ακίνητα υπό διαχείριση.

## Διαμόρφωση

Τις περισσότερες φορές, δεν χρειάζεται να κοινοποιήσετε απευθείας καμία από αυτές τις εφαρμογές. Έχουμε δημιουργήσει μια ποικιλία τρόπων για να κοινοποιείτε ταξιδιωτικό απόθεμα, τους οποίους μπορείτε να ανακαλύψετε στο[Στούντιο Wink](/studio/what-is-studio)ή μέσω[WinkLinks](/link-manager/wink-links).

Ωστόσο, εάν θέλετε να συνδεθείτε χειροκίνητα σε έναν από αυτούς τους ιστότοπους, βεβαιωθείτε ότι έχετε προσθέσει το αναγνωριστικό πελάτη και το αναγνωριστικό προσαρμογής στη διεύθυνση URL ως παραμέτρους.

Μπορείτε να βρείτε το αναγνωριστικό πελάτη σας ως εξής:

<Steps>
  1. Συνδεθείτε είτε [Wink Extranet](/extranet/what-is-extranet) ή [Στούντιο Wink](/studio/what-is-studio).
  2. Κάντε κλικ στο εικονίδιο του προφίλ σας στην επάνω δεξιά γωνία και κάντε κλικ στο [Εφαρμογές](/developers/applications).
  3. Χρησιμοποιήστε την εφαρμογή που είναι συνδεδεμένη με τον λογαριασμό Wink με τον οποίο θέλετε να εργαστείτε.
  4. Αντιγράψτε το Client ID από αυτήν την εφαρμογή και επικολλήστε το στη διεύθυνση URL ως παράμετρο αιτήματος. Παράδειγμα: ?client-id=MY\_CLIENT\_ID
</Steps>


### Εξωδίκτυο

Μπορείτε να βρείτε το αναγνωριστικό προσαρμογής σας ως ιδιότητα ως εξής:

<Steps>
  1. Συνδεθείτε [Wink Extranet](/extranet/what-is-extranet).
  2. Επιλέξτε την ιδιότητα με την οποία θέλετε να εργαστείτε.
  3. Από την κύρια γραμμή πλοήγησης, κάντε κλικ στο `Account > Branding`
  4. Αντιγράψτε το αναγνωριστικό προσαρμογής και επικολλήστε το στη διεύθυνση URL ως παράμετρο αιτήματος. Παράδειγμα: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Στούντιο

Μπορείτε να βρείτε το αναγνωριστικό προσαρμογής σας ως συνεργάτης ως εξής:

<Steps>
  1. Συνδεθείτε [Στούντιο Wink](/studio/what-is-studio).
  2. Επιλέξτε τον λογαριασμό με τον οποίο θέλετε να εργαστείτε.
  3. Από την κύρια γραμμή πλοήγησης, κάντε κλικ στο `Tools > Customization`
  4. Αντιγράψτε το αναγνωριστικό προσαρμογής από την προσαρμογή που θέλετε να χρησιμοποιήσετε και επικολλήστε το στη διεύθυνση URL ως παράμετρο αιτήματος. Παράδειγμα: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

Αυτό που καταλήγετε είναι μια διεύθυνση URL που μοιάζει κάπως έτσι:

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Αν θέλετε να μεταβείτε απευθείας στη σελίδα ενός ξενοδοχείου, η διεύθυνση URL θα μοιάζει κάπως έτσι:

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::σημείωση
Εάν δεν συμπεριλάβετε ένα αναγνωριστικό πελάτη ή/και διαμόρφωσης, το Wink OTA θα ορίσει από προεπιλογή μια προεπιλεγμένη διαμόρφωση.
:::

## ΙΒΕ

Εάν είστε εταιρεία διαχείρισης ακινήτων, θέλετε να συνδεθείτε με την αρχική σελίδα του Wink IBE. Η αρχική σελίδα θα εμφανίζει το θέμα που έχετε ορίσει μαζί με το[ιδιότητες που αντιπροσωπεύετε](/studio/agency-representation).

Αυτό που καταλήγετε είναι μια διεύθυνση URL που μοιάζει κάπως έτσι:

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Αν είστε ακίνητο, θέλετε να συνδεθείτε απευθείας με τη σελίδα προορισμού του ακινήτου Wink IBE.

Αυτή η διεύθυνση URL θα έμοιαζε κάπως έτσι:

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::σημείωση
Εάν δεν συμπεριλάβετε ένα αναγνωριστικό πελάτη ή/και διαμόρφωσης, το Wink IBE ΔΕΝ θα φορτωθεί. Αυτές οι παράμετροι είναι απαραίτητες.
:::

## Παράδειγμα: Σελίδα ιδιότητας

Κάθε κατάλυμα στο Wink διαθέτει τη δική του μοναδική σελίδα προορισμού σε οποιονδήποτε από τους δύο ιστότοπους. Όλοι οι εξωτερικοί σύνδεσμοι κατευθύνουν τους ταξιδιώτες σε μία από αυτές τις σελίδες προορισμού.

Η σελίδα προορισμού περιέχει όλες τις πληροφορίες που χρειάζεται ένας ταξιδιώτης για να κάνει μια ενημερωμένη επιλογή σχετικά με το αν θα μείνει σε αυτό το κατάλυμα.
