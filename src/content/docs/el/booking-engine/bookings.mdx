Αυτό το άρθρο καλύπτει την πλήρη διαδικασία κράτησης στο Wink.

## Πώς να κάνετε κράτηση

Μόλις βρείτε το κατάλυμα και το δωμάτιο που θέλετε να κάνετε κράτηση, κάντε κλικ στο`Book`κουμπί για να ξεκινήσετε.

<figure>
  <CloudinaryImage width="350" height="819" resize="scale" quality="best" alt="Κάρτα δωματίου επισκεπτών με δυνατότητα κράτησης" src="academy/guest-room-card" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Δείγμα κάρτας δωματίου επισκεπτών με δυνατότητα κράτησης*</figcaption>
</figure>


<Steps>
  1. Κάντε κλικ στο `Book` κουμπί.
  2. Ανοίγει το παράθυρο του διαμορφωτή δωματίου. * [Σχήμα 1](#configurator) *
  3. Επιλέξτε τις διαθέσιμες επιλογές για αυτό το δωμάτιο. * [Σχήμα 2](#options) *
  4. Κλικ `Log in to continue` όταν είναι έτοιμο.
  5. Αν αυτή είναι η πρώτη φορά που χρησιμοποιείτε το Wink, θα πρέπει να [δημιουργία λογαριασμού](/user/register#registration).
  6. Συμπληρώστε τα στοιχεία πληρωμής σας. * [Σχήμα 3](#payment) *
  7. Κλικ `Pay` κουμπί για να ολοκληρώσετε την κράτηση.
</Steps>


### Διαμορφωτής

Ο διαμορφωτής δωματίων σάς δίνει την ευκαιρία να επιλέξετε επιλογές για το δωμάτιο και παράλληλα με αυτό.
Αναζητήστε εξαιρετικές προσφορές από ξενοδοχεία εδώ, κάνοντας κύλιση προς τα κάτω για να δείτε όλα όσα προσφέρουν.

### Επιλογές

Οι συνήθεις επιλογές περιλαμβάνουν:

* Πολιτική ακύρωσης
* Γεύματα
* Προτιμήσεις κρεβατιού
* Ειδικά αιτήματα

Άλλες επιλογές περιλαμβάνουν:

* Βοηθητικά μέσα στο δωμάτιο.
* Άλλα πρόσθετα
* Ειδικές προσφορές για εγκαταστάσεις και εμπειρίες:
  * Συνεδρίες σπα.
  * Προσφορές εστιατορίων.
  * Κάντε κράτηση αιθουσών συσκέψεων.
  * Δραστηριότητες βιβλίου.
  * ...και πολλά άλλα!!

Μόλις πληρώσετε, θα μεταφερθείτε σε μια σελίδα επιβεβαίωσης κράτησης την οποία μπορείτε να εκτυπώσετε.

Θα λάβετε επίσης ένα email που θα περιέχει όλα όσα πρέπει να γνωρίζετε σχετικά με την κράτηση.

## Ακύρωση κράτησης

:::προσοχή
Μπορείτε να ακυρώσετε μόνο μια κράτηση που έχει πολιτική ακύρωσης που σας επιτρέπει να την ακυρώσετε. Διαβάστε προσεκτικά την πολιτική ακύρωσης πριν κάνετε κράτηση για οποιοδήποτε δωμάτιο και επιλέξτε μια πολιτική ακύρωσης που σας ταιριάζει.
:::

Αν θέλετε να ακυρώσετε μια κράτηση, ακολουθήστε τα παρακάτω βήματα:

<Steps>
  1. Επιλέξτε την κράτηση που θέλετε να ακυρώσετε. Μπορείτε να το κάνετε αυτό με έναν από τους εξής τρόπους:
     * Κάντε κλικ στο `Manage booking` σύνδεσμο από το email επιβεβαίωσης που λάβατε.
     * Μεταβείτε στη διαχείριση κρατήσεων στο https://ota.wink.travel και κάντε κλικ στο `Bookings` σύνδεσμο που αναφέρθηκε στην προηγούμενη ενότητα.
  2. Κάντε κλικ στο `Cancel` κουμπί. *Το κουμπί ακύρωσης είναι απενεργοποιημένο εάν δεν μπορείτε πλέον να ακυρώσετε την κράτηση.*
  3. Πείτε μας τον λόγο για τον οποίο πρέπει να ακυρώσετε και κάντε κλικ `Submit`.
  4. Με βάση την πολιτική ακύρωσης, σας επιστρέφεται ολόκληρο ή μερικό ποσό.
</Steps>


:::σημείωση
Υπολογίστε περίπου 1 - 3 ημέρες για να ολοκληρωθεί η επιστροφή χρημάτων.
:::

## Αίτημα επιστροφής χρημάτων

Υπό ορισμένες συνθήκες, μπορείτε να ζητήσετε επιστροφή χρημάτων, μετά τη λήξη της πολιτικής ακύρωσης.

Παρακαλούμε επικοινωνήστε με το ξενοδοχείο και εξηγήστε την κατάσταση. Το ξενοδοχείο θα ζητήσει την επιστροφή χρημάτων εκ μέρους σας.

## Γράψτε μια κριτική

Μόλις τελειώσει το ταξίδι \[και χρειάζεστε διακοπές από τις διακοπές], θα σας στείλουμε email ζητώντας σας να γράψετε μια κριτική για το κατάλυμα στο οποίο μείνατε.

Δείτε πώς μπορείτε να γράψετε μια κριτική για το ακίνητο:

<Steps>
  1. Κάντε κλικ στον σύνδεσμο στο email που σας στείλαμε.
  2. Ίσως χρειαστεί να συνδεθείτε ξανά στο Wink.
  3. Ο σύνδεσμος σας μεταφέρει στη σελίδα επιβεβαίωσης κράτησης.
  4. Κλικ `Write review` κουμπί.
  5. Συμπληρώστε τις 5 ερωτήσεις πολλαπλής επιλογής.
  6. Γράψτε μια σύντομη πρόταση/προτάσεις σχετικά με τη διαμονή σας.
  7. Κάντε κλικ στο `Submit` κουμπί.
</Steps>


Οι κριτικές είναι ένας εξαιρετικός τρόπος για να αναγνωριστούν τα ακίνητα, χωρίς να υπάρχει κάποιο κόστος.

:::tip\[Συμβουλή από επαγγελματίες]
Αν η διαμονή σας δεν ήταν τόσο καλή, μην ξοδεύετε όλο τον χρόνο σας γράφοντας μια αρνητική κριτική.

Αντ' αυτού, ενημερώστε το ξενοδοχείο, όσο βρίσκεστε ακόμα στις εγκαταστάσεις, και δώστε στον Γενικό Διευθυντή την ευκαιρία να σας το επανορθώσει.
