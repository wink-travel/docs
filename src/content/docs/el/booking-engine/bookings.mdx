Αυτό το άρθρο καλύπτει ολόκληρη τη διαδικασία κράτησης στο Wink.

## Πώς να κάνετε κράτηση

Μόλις βρείτε το κατάλυμα και το δωμάτιο που θέλετε να κλείσετε, κάντε κλικ στο κουμπί `Book` για να ξεκινήσετε.

<Steps>
1. Κάντε κλικ στο κουμπί `Book`.
2. Ανοίγει το παράθυρο ρυθμιστή δωματίου. * [Εικόνα 1](#configurator) *
3. Επιλέξτε τις διαθέσιμες επιλογές για το δωμάτιο. * [Εικόνα 2](#options) *
4. Κάντε κλικ στο `Log in to continue` όταν είστε έτοιμοι.
5. Αν είναι η πρώτη σας φορά στο Wink, πρέπει να [δημιουργήσετε λογαριασμό](/el/user/register#registration).
6. Συμπληρώστε τα στοιχεία πληρωμής σας. * [Εικόνα 3](#payment) *
7. Κάντε κλικ στο κουμπί `Pay` για να ολοκληρώσετε την κράτηση.
</Steps>

### Ρυθμιστής

Ο ρυθμιστής δωματίου σας δίνει τη δυνατότητα να επιλέξετε επιλογές για το δωμάτιο και παράλληλα με αυτό.
Ψάξτε για εξαιρετικές προσφορές από τα ξενοδοχεία εδώ, κάνοντας κύλιση προς τα κάτω για να δείτε όλα όσα προσφέρουν.

### Επιλογές

Συνηθισμένες επιλογές περιλαμβάνουν:

- Πολιτική ακύρωσης
- Γεύματα
- Προτιμήσεις κρεβατιού
- Ειδικά αιτήματα

Άλλες επιλογές περιλαμβάνουν:

- Παροχές εντός δωματίου.
- Άλλα πρόσθετα
- Ειδικές προσφορές για εγκαταστάσεις και εμπειρίες:
    - Συνεδρίες σπα.
    - Προσφορές εστιατορίου.
    - Κράτηση αιθουσών συνεδριάσεων.
    - Κράτηση δραστηριοτήτων.
    - ...και πολλά άλλα!!

## Ακύρωση κράτησης

:::caution
Μπορείτε να ακυρώσετε μόνο μια κράτηση που έχει πολιτική ακύρωσης που σας επιτρέπει να την ακυρώσετε. Διαβάστε προσεκτικά την πολιτική ακύρωσης πριν κάνετε κράτηση οποιουδήποτε δωματίου και επιλέξτε μια πολιτική ακύρωσης που σας ταιριάζει.
:::

Αν θέλετε να ακυρώσετε μια κράτηση, ακολουθήστε τα παρακάτω βήματα:

<Steps>
1. Επιλέξτε την κράτηση που θέλετε να ακυρώσετε. Μπορείτε να το κάνετε είτε:
    - Κάνοντας κλικ στο σύνδεσμο `Manage booking` από το email επιβεβαίωσης που λάβατε.
    - Πηγαίνοντας στη διαχείριση κρατήσεων στο https://ota.wink.travel και κάνοντας κλικ στο σύνδεσμο `Bookings` που αναφέρθηκε στην προηγούμενη ενότητα.
2. Κάντε κλικ στο κουμπί `Cancel`. *Το κουμπί ακύρωσης είναι απενεργοποιημένο αν δεν μπορείτε πλέον να ακυρώσετε την κράτηση.*
3. Πείτε μας τον λόγο για τον οποίο πρέπει να ακυρώσετε και κάντε κλικ στο `Submit`.
4. Βάσει της πολιτικής ακύρωσης, το πλήρες ή μερικό ποσό επιστρέφεται σε εσάς.
</Steps>

:::note
Αναμένετε περίπου 1 - 3 ημέρες για να ολοκληρωθεί η επιστροφή χρημάτων.
:::

## Αίτηση επιστροφής χρημάτων

Υπό ορισμένες συνθήκες, μπορείτε να ζητήσετε επιστροφή χρημάτων, μετά τη λήξη της πολιτικής ακύρωσης.

Παρακαλούμε επικοινωνήστε με το ξενοδοχείο και εξηγήστε την κατάσταση. Το ξενοδοχείο θα ζητήσει την επιστροφή χρημάτων εκ μέρους σας.

## Γράψτε μια κριτική

Μόλις τελειώσει το ταξίδι [και χρειάζεστε διακοπές από τις διακοπές], θα σας στείλουμε email ζητώντας σας να γράψετε μια κριτική για το κατάλυμα που μείνατε.

Ακολουθεί πώς να γράψετε μια κριτική για το κατάλυμα:

<Steps>
1. Κάντε κλικ στο σύνδεσμο στο email που σας στείλαμε.
2. Ίσως χρειαστεί να συνδεθείτε ξανά στο Wink.
3. Ο σύνδεσμος σας οδηγεί στη σελίδα επιβεβαίωσης κράτησης.
4. Κάντε κλικ στο κουμπί `Write review`.
5. Συμπληρώστε τις 5 ερωτήσεις πολλαπλής επιλογής.
6. Γράψτε μια σύντομη πρόταση ή προτάσεις για τη διαμονή σας.
7. Κάντε κλικ στο κουμπί `Submit`.
</Steps>

Οι κριτικές είναι ένας εξαιρετικός τρόπος για να αναγνωριστούν τα καταλύματα χωρίς κόστος.

:::tip[Επαγγελματική συμβουλή]
Αν η διαμονή σας δεν ήταν τόσο καλή, μην ξοδεύετε όλο τον χρόνο σας γράφοντας αρνητική κριτική.

Αντίθετα, ενημερώστε το ξενοδοχείο ενώ είστε ακόμα στον χώρο, δίνοντας στον διευθυντή την ευκαιρία να το διορθώσει.

:::
