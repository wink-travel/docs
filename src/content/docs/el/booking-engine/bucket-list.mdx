Οπουδήποτε βλέπετε το απόθεμα Wink[καρτέλλες](/studio/cards), θα δείτε μια καρδιά στην επάνω δεξιά γωνία της κάρτας που σας επιτρέπει να προσθέσετε αυτό το απόθεμα στη λίστα επιθυμιών σας.

Ακολουθήστε αυτά τα βήματα για να διαχειριστείτε τη λίστα επιθυμιών σας:

<Steps>
  1. Κάντε κλικ στο ♡ `heart icon` για το απόθεμα που θέλετε να προσθέσετε.
  2. Αν είστε συνδεδεμένοι, το εικονίδιο της καρδιάς θα γίνει κόκκινο.
  3. Εάν δεν είστε συνδεδεμένοι, θα ανακατευθυνθείτε για να το κάνετε.
</Steps>


:::συμβουλή
Κάντε κλικ στο ♥️`heart icon`μια δεύτερη φορά για να το αφαιρέσετε από τη λίστα σας.
:::

Μπορείτε να διαχειριστείτε όλα τα στοιχεία της λίστας επιθυμιών σας κάνοντας κλικ στο εικονίδιο του προφίλ σας στην επάνω δεξιά γωνία και, στη συνέχεια, κάνοντας κλικ στο`Bucket list`κουμπί.
