Οπουδήποτε δείτε κάρτες αποθέματος [cards](/el/studio/cards) του Wink, θα δείτε μια καρδιά στην πάνω δεξιά γωνία της κάρτας που σας επιτρέπει να προσθέσετε αυτό το απόθεμα στη λίστα επιθυμιών σας.

Ακολουθήστε αυτά τα βήματα για να διαχειριστείτε τη λίστα επιθυμιών σας:

<Steps>
1. Κάντε κλικ στο ♡ `εικονίδιο καρδιάς` για το απόθεμα που θέλετε να προσθέσετε.
2. Αν είστε συνδεδεμένοι, το εικονίδιο καρδιάς θα γίνει κόκκινο.
3. Αν δεν είστε συνδεδεμένοι, θα ανακατευθυνθείτε να συνδεθείτε.
</Steps>

:::tip
Κάντε κλικ στο ♥️ `εικονίδιο καρδιάς` μια δεύτερη φορά για να το αφαιρέσετε από τη λίστα σας.
:::

Μπορείτε να διαχειριστείτε όλα τα στοιχεία της λίστας επιθυμιών σας κάνοντας κλικ στο εικονίδιο προφίλ σας στην πάνω δεξιά γωνία και στη συνέχεια πατώντας το κουμπί `Bucket list`.
