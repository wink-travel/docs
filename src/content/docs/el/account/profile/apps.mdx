Η καρτέλα **Εφαρμογές** στο [iam.wink.travel/my-account/apps](https://iam.wink.travel/my-account/apps) εμφανίζει κάθε εφαρμογή OAuth2 στην οποία έχετε δώσει πρόσβαση στον λογαριασμό σας στο Wink, με δυνατότητα ανάκλησης αυτής της πρόσβασης.

## Πώς μια εφαρμογή εμφανίζεται εδώ

Όταν κάνετε κλικ στο `Authorize` σε ένα εργαλείο τρίτου μέρους που ενσωματώνεται με το Wink (ή σε μία από τις δικές του εφαρμογές), η ροή συγκατάθεσης καταγράφει την έγκριση. Από εκεί και πέρα, η εφαρμογή μπορεί να ενεργεί εκ μέρους σας εντός των πεδίων πρόσβασης που εγκρίνατε.

## Επισκόπηση πρόσβασης

Κάθε καταχώρηση εμφανίζει:

- **Όνομα εφαρμογής** και μια σύντομη περιγραφή.
- **Πεδία πρόσβασης** που μπορεί να χρησιμοποιήσει η εφαρμογή (π.χ. ανάγνωση αποθέματος, διαχείριση κρατήσεων).
- **Τελευταία χρήση** — πότε η εφαρμογή κάλεσε τελευταία φορά το Wink API εκ μέρους σας.

## Ανάκληση εφαρμογής

Κάντε κλικ στο `Revoke` δίπλα σε οποιαδήποτε καταχώρηση. Τα διακριτικά της εφαρμογής ακυρώνονται αμέσως και οποιεσδήποτε εργασίες στο παρασκήνιο που εκτελεί κατά του λογαριασμού σας θα αρχίσουν να αποτυγχάνουν με σφάλματα 401.

:::tip
Η ανάκληση δεν είναι καταστροφική — μπορείτε να εξουσιοδοτήσετε ξανά την εφαρμογή οποιαδήποτε στιγμή ακολουθώντας ξανά τη ροή συγκατάθεσης. Χρησιμοποιήστε το αν μια εφαρμογή παρουσιάζει προβλήματα, αν αλλάζετε διαπιστευτήρια ή αν δεν χρησιμοποιείτε πλέον την ενσωμάτωση.
:::

## Εφαρμογές OAuth που δημιουργείτε μόνοι σας

Αν θέλετε να *δημιουργήσετε* μια εφαρμογή OAuth (όχι απλώς να εξουσιοδοτήσετε μία), αυτή είναι μια λειτουργία λογαριασμού οργανισμού — δείτε [Ρυθμίσεις → Εφαρμογές](/el/portal/settings/applications).
