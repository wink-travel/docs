Η καρτέλα **Ασφάλεια** στο [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) είναι όπου διαχειρίζεστε τα **passkeys** — διαπιστευτήρια WebAuthn που σας επιτρέπουν να συνδεθείτε χωρίς κωδικό.

## Τι είναι ένα passkey;

Ένα passkey είναι ένα κρυπτογραφικό διαπιστευτήριο συνδεδεμένο με τη συσκευή σας (ή τον διαχειριστή κωδικών) που αποδεικνύει ότι είστε εσείς. Τα passkeys είναι ανθεκτικά σε phishing και πιο γρήγορα από το να πληκτρολογείτε κωδικό.

## Εγγραφή ενός passkey

1. Συνδεθείτε στο [iam.wink.travel](https://iam.wink.travel) με τα υπάρχοντα διαπιστευτήριά σας.
2. Μεταβείτε στο **Ο λογαριασμός μου → Ασφάλεια**.
3. Κάντε κλικ στο `Προσθήκη passkey`.
4. Ο περιηγητής σας θα ζητήσει να πιστοποιηθείτε με τη συσκευή σας (Touch ID, Face ID, Windows Hello, ένα υλικό κλειδί ή έναν συγχρονισμένο διαχειριστή κωδικών).
5. Δώστε στο passkey ένα φιλικό όνομα (π.χ. *Work MacBook*) ώστε να το αναγνωρίζετε αργότερα.

## Χρήση ενός passkey για σύνδεση

Στην οθόνη σύνδεσης του [iam.wink.travel](https://iam.wink.travel), εισάγετε το email σας — αν έχετε εγγεγραμμένο passkey για αυτό το email, θα σας ζητηθεί να πιστοποιηθείτε με αυτό αντί να πληκτρολογήσετε κωδικό.

## Αφαίρεση ενός passkey

Στην ίδια καρτέλα Ασφάλεια, κάθε εγγεγραμμένο passkey έχει ένα κουμπί `Αφαίρεση`. Χρησιμοποιήστε το αν χάσετε μια συσκευή, αποσύρετε ένα υλικό κλειδί ή θέλετε να καθαρίσετε αχρησιμοποίητα διαπιστευτήρια.

:::caution
Η αφαίρεση όλων των passkeys αφήνει τον κωδικό ως τη μόνη μέθοδο σύνδεσης. Βεβαιωθείτε ότι θυμάστε τον κωδικό σας πριν αφαιρέσετε το τελευταίο passkey.
:::
