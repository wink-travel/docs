Η **εφαρμογή Wink Wix** προσθέτει web components της Wink στην ιστοσελίδα σας Wix ως εγγενή widgets που μπορείτε να σύρετε και να αποθέσετε στον Wix Editor. Δεν απαιτείται κωδικοποίηση για τους ιδιοκτήτες ιστοσελίδων.

## Εγκατάσταση

<Steps>
1. Μεταβείτε στο [Wix App Market](https://www.wix.com/app-market) και αναζητήστε το **Wink Travel**.
2. Κάντε κλικ στο **Προσθήκη στην Ιστοσελίδα** και ακολουθήστε τις οδηγίες για να εξουσιοδοτήσετε την εφαρμογή.
3. Μόλις εγκατασταθεί, ανοίξτε **Πίνακας Ελέγχου → Ρυθμίσεις Wink** και εισάγετε το **Client ID** και το **Client Secret** σας.
   Μεταβείτε στις [Εφαρμογές](/el/portal/settings/applications) για να ανακτήσετε τα διαπιστευτήριά σας.
4. Κάντε κλικ στο **Αποθήκευση** και στη συνέχεια στο **Δοκιμή Σύνδεσης** για να επιβεβαιώσετε ότι η Wink είναι συνδεδεμένη.
</Steps>

## Προσθήκη component σε σελίδα

<Steps>
1. Ανοίξτε τον **Wix Editor** και μεταβείτε στη σελίδα όπου θέλετε να προσθέσετε περιεχόμενο Wink.
2. Κάντε κλικ στο **Προσθήκη (+) → Εφαρμογές → Wink Travel** για να δείτε όλα τα διαθέσιμα widgets.
3. Σύρετε ένα widget στη σελίδα — για παράδειγμα, το **Wink Content** για να εμφανίσετε ένα πλέγμα ξενοδοχείων.
4. Στο πάνελ ρυθμίσεων του widget, επιλέξτε τον **Τύπο Διάταξης** και επικολλήστε το **Layout ID** από το Wink Studio.
5. Κάντε κλικ στο **Δημοσίευση** για να κάνετε τις αλλαγές ενεργές.
</Steps>

## Διαθέσιμα widgets

| Widget | Σκοπός |
|---|---|
| **Wink Content** | Εμφανίζει κάρτες ξενοδοχείων, ταξινομημένα πλέγματα, χάρτες και άλλα |
| **Wink Lookup** | Μπάρα αναζήτησης για προορισμούς και ξενοδοχεία |
| **Wink Search** | Κουμπί εικονιδίου που ανοίγει τον επιλογέα δρομολογίου |
| **Wink Account** | Κουμπί σύνδεσης / λογαριασμού χρήστη |
| **Wink Itinerary** | Κουμπί δρομολογίου που εμφανίζει το τρέχον ταξίδι |
| **Wink Cart** | Κουμπί σύνοψης καλαθιού αγορών |
