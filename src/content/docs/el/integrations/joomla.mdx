Η **Wink Joomla Extension** είναι ένα πακέτο που εγκαθιστά ένα σύστημα plugin (φορτώνει CDN scripts), ένα content plugin (επεξεργάζεται shortcode tags σε άρθρα) και ένα component (σελίδα ρυθμίσεων διαχειριστή).

/>

## Απαιτήσεις

- Joomla 4 ή 5
- PHP 8.1+

## Εγκατάσταση

<Steps>
1. Κατεβάστε το πιο πρόσφατο `pkg_wink-x.x.x.zip` από τη [σελίδα εκδόσεων GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. Στο Joomla Admin, πηγαίνετε στο **Extensions → Manage → Install**.
3. Ανεβάστε το αρχείο zip και κάντε κλικ στο **Upload & Install**.
4. Και τα δύο plugins ενεργοποιούνται αυτόματα. Πηγαίνετε στο **Components → Wink Settings**.
5. Εισάγετε το **Client ID** και το **Client Secret** και κάντε κλικ στο **Save**.
   Πηγαίνετε στις [Applications](/el/portal/settings/applications) για να λάβετε τα διαπιστευτήριά σας.
</Steps>

## Ενσωμάτωση components σε άρθρα

Χρησιμοποιήστε tags τύπου shortcode οπουδήποτε στο περιεχόμενο του άρθρου:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Το content plugin αντικαθιστά αυτά τα tags με το σωστό HTML `<wink-*>` κατά το χρόνο απόδοσης. Το system plugin φορτώνει αυτόματα τα CDN scripts και εισάγει το `<wink-app-loader>` στο υποσέλιδο της σελίδας.

## Διαθέσιμα shortcodes

| Shortcode | Έξοδος | Περιγραφή |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Κάρτες ξενοδοχείων, πλέγματα, χάρτες |
| `{wink-lookup}` | `<wink-lookup>` | Μπάρα αναζήτησης προορισμού |
| `{wink-search}` | `<wink-search-button>` | Κουμπί επιλογής δρομολογίου |
| `{wink-account}` | `<wink-account-button>` | Σύνδεση / λογαριασμός χρήστη |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Κουμπί κατάστασης δρομολογίου |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Κουμπί καλαθιού |
