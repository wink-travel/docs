Η **εφαρμογή Wink Shopify** επιτρέπει σε τουριστικούς πράκτορες, πωλητές εμπειριών και εμπόρους φιλοξενίας να εμφανίζουν ζωντανό απόθεμα Wink απευθείας στην βιτρίνα τους στο Shopify — παράλληλα με προϊόντα, σε σελίδες προορισμού ή οπουδήποτε στο θέμα.

## Εγκατάσταση

<Steps>
1. Μεταβείτε στο [Shopify App Store](https://apps.shopify.com) και αναζητήστε το **Wink Travel**.
2. Κάντε κλικ στο **Add app** και ακολουθήστε τη ροή OAuth για εξουσιοδότηση.
3. Στις ρυθμίσεις της εφαρμογής Wink, εισάγετε το **Client ID** και το **Client Secret** σας.
   Μεταβείτε στις [Εφαρμογές](/el/portal/settings/applications) για να ανακτήσετε τα διαπιστευτήριά σας.
4. Κάντε κλικ στο **Verify & Save**. Ένα μήνυμα επιτυχίας επιβεβαιώνει τη σύνδεση.
</Steps>

## Ενεργοποίηση του App Embed Block

Το App Embed Block εισάγει αυτόματα το σενάριο και το φύλλο στυλ Wink σε κάθε σελίδα — δεν χρειάζεται να επεξεργαστείτε τον κώδικα του θέματος.

<Steps>
1. Στο Shopify Admin, μεταβείτε στο **Online Store → Themes → Customize**.
2. Κάντε κλικ στο **App embeds** στο αριστερό πάνελ.
3. Ενεργοποιήστε το **Wink Travel**.
4. Κάντε κλικ στο **Save**.
</Steps>

:::note
Το App Embed Block διαβάζει το Client ID σας από ένα metafield του Shopify που ορίζεται κατά την εγκατάσταση. Το Client Secret σας δεν εκτίθεται ποτέ στον περιηγητή.
:::

## Προσθήκη συστατικού σε μια σελίδα

<Steps>
1. Στον επεξεργαστή θέματος, ανοίξτε τη σελίδα ή την ενότητα όπου θέλετε περιεχόμενο Wink.
2. Κάντε κλικ στο **Add block** και επιλέξτε από τα μπλοκ Wink:
   - **Wink Content** — κάρτες ξενοδοχείων, πλέγματα ή χάρτες
   - **Wink Lookup** — γραμμή αναζήτησης προορισμών
   - **Wink Search** — κουμπί επιλογής δρομολογίου
   - **Wink Account** — κουμπί σύνδεσης
   - **Wink Itinerary** — κουμπί κατάστασης δρομολογίου
   - **Wink Cart** — κουμπί καλαθιού αγορών
3. Για το **Wink Content**, επιλέξτε **Layout Type** και εισάγετε το **Layout ID** από το Wink Studio.
4. Κάντε κλικ στο **Save**.
</Steps>

## Διαθέσιμα μπλοκ θέματος

| Μπλοκ | Σκοπός |
|---|---|
| **Wink Content** | Εμφανίζει κάρτες ξενοδοχείων, ταξινομημένα πλέγματα, χάρτες και άλλα |
| **Wink Lookup** | Γραμμή αναζήτησης για προορισμούς και ξενοδοχεία |
| **Wink Search** | Εικονίδιο κουμπιού που ανοίγει τον επιλογέα δρομολογίου |
| **Wink Account** | Κουμπί σύνδεσης / λογαριασμού χρήστη |
| **Wink Itinerary** | Κουμπί δρομολογίου που δείχνει το τρέχον ταξίδι |
| **Wink Cart** | Κουμπί σύνοψης καλαθιού αγορών |
