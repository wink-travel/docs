Η **Wink TYPO3 Extension** ενσωματώνει το απόθεμα ταξιδιών Wink στο TYPO3 13 ως εγγενή στοιχεία περιεχομένου. Οι συντάκτες προσθέτουν τα στοιχεία Wink στις σελίδες χρησιμοποιώντας τον τυπικό επεξεργαστή σελίδων TYPO3 — δεν απαιτούνται αλλαγές στα πρότυπα.

## Απαιτήσεις

- TYPO3 13.x
- PHP 8.2+
- Composer

## Εγκατάσταση

<Steps>
1. Εγκαταστήστε μέσω Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Ενεργοποιήστε την επέκταση:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Στο backend του TYPO3, μεταβείτε σε **Admin Tools → Settings → Extension Configuration → wink**.
4. Εισάγετε το **Client ID**, το **Client Secret** και επιλέξτε το περιβάλλον σας. Αποθηκεύστε.
   Μεταβείτε στα [Applications](/el/portal/settings/applications) για να λάβετε τα διαπιστευτήριά σας.
5. Προσθέστε το στατικό πρότυπο TypoScript στο ριζικό σας πρότυπο:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Προσθήκη στοιχείου περιεχομένου σε σελίδα

<Steps>
1. Ανοίξτε μια σελίδα στο module **Page**.
2. Κάντε κλικ στο κουμπί **+** στην περιοχή περιεχομένου για να προσθέσετε νέο στοιχείο.
3. Επιλέξτε ένα από τα στοιχεία περιεχομένου **Wink Travel**:
   - **Wink Content** — κάρτες ξενοδοχείων, ταξινομημένα πλέγματα, χάρτες
   - **Wink Lookup** — γραμμή αναζήτησης προορισμού
   - **Wink Search** — κουμπί επιλογής δρομολογίου
   - **Wink Account** — κουμπί σύνδεσης
   - **Wink Itinerary** — κουμπί κατάστασης δρομολογίου
   - **Wink Cart** — κουμπί καλαθιού αγορών
4. Για το **Wink Content**, χρησιμοποιήστε τα πεδία FlexForm για να επιλέξετε **Layout Type** και να εισάγετε το **Layout ID**.
5. Αποθηκεύστε και δημοσιεύστε τη σελίδα.
</Steps>

## Διαθέσιμα στοιχεία περιεχομένου

| Στοιχείο | Σκοπός |
|---|---|
| **Wink Content** | Κάρτες ξενοδοχείων, ταξινομημένα πλέγματα, χάρτες και άλλα |
| **Wink Lookup** | Γραμμή αναζήτησης προορισμού |
| **Wink Search** | Εικονίδιο κουμπιού που ανοίγει τον επιλογέα δρομολογίου |
| **Wink Account** | Κουμπί σύνδεσης / λογαριασμού χρήστη |
| **Wink Itinerary** | Κουμπί δρομολογίου που εμφανίζει το τρέχον ταξίδι |
| **Wink Cart** | Κουμπί σύνοψης καλαθιού αγορών |
