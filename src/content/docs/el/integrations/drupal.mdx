Το **Wink Drupal Module** προσθέτει στοιχεία αποθέματος ταξιδιών Wink στον ιστότοπό σας Drupal ως εγγενή blocks. Οι συντάκτες μπορούν να τα τοποθετήσουν οπουδήποτε στη διάταξη μέσω του τυπικού συστήματος blocks.

  title="wink-travel/wink-drupal-module on GitHub"

/>

## Απαιτήσεις

- Drupal 10 ή 11
- PHP 8.1+
- Composer

## Εγκατάσταση

<Steps>
1. Εγκαταστήστε μέσω Composer:
   ```bash
   composer require drupal/wink
   ```
2. Ενεργοποιήστε το module:
   ```bash
   drush en wink -y
   ```
3. Μεταβείτε στο **Διαχείριση → Διαμόρφωση → Υπηρεσίες Ιστού → Ρυθμίσεις Wink** (`/admin/config/wink`).
4. Εισάγετε το **Client ID** και το **Client Secret**, επιλέξτε το περιβάλλον σας και κάντε κλικ στο **Αποθήκευση**.
   Μεταβείτε στις [Εφαρμογές](/el/portal/settings/applications) για να λάβετε τα διαπιστευτήριά σας.
</Steps>

## Προσθήκη block σε σελίδα

<Steps>
1. Μεταβείτε στο **Διαχείριση → Δομή → Διάταξη Block**.
2. Κάντε κλικ στο **Τοποθέτηση block** στην περιοχή όπου θέλετε το περιεχόμενο Wink.
3. Αναζητήστε **Wink** για να βρείτε τα διαθέσιμα blocks:
   - **Wink Content** — εμφανίζει κάρτα ξενοδοχείου, πλέγμα ή χάρτη
   - **Wink Lookup** — γραμμή αναζήτησης προορισμών
   - **Wink Search** — κουμπί επιλογής δρομολογίου
   - **Wink Account** — κουμπί σύνδεσης
   - **Wink Itinerary** — κουμπί κατάστασης δρομολογίου
   - **Wink Cart** — κουμπί καλαθιού αγορών
4. Για το **Wink Content**, ρυθμίστε τον **Τύπο Διάταξης** και το **ID Διάταξης** στις ρυθμίσεις του block.
5. Αποθηκεύστε τη ρύθμιση του block.
</Steps>

## Διαθέσιμα blocks

| Block | Σκοπός |
|---|---|
| **Wink Content** | Εμφανίζει κάρτες ξενοδοχείων, ταξινομημένα πλέγματα, χάρτες και άλλα |
| **Wink Lookup** | Γραμμή αναζήτησης για προορισμούς και ξενοδοχεία |
| **Wink Search** | Εικονίδιο κουμπιού που ανοίγει τον επιλογέα δρομολογίου |
| **Wink Account** | Κουμπί σύνδεσης / λογαριασμού χρήστη |
| **Wink Itinerary** | Κουμπί δρομολογίου που εμφανίζει το τρέχον ταξίδι |
| **Wink Cart** | Κουμπί σύνοψης καλαθιού αγορών |
