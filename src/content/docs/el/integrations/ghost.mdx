Η ενσωματωμένη λειτουργία **Code Injection** του Ghost σας επιτρέπει να προσθέτετε σενάρια στο παγκόσμιο `<head>` και `</body>` της έκδοσής σας. Όλα τα πακέτα Ghost υποστηρίζουν το code injection, καθιστώντας εύκολη την ενσωμάτωση των στοιχείων Wink σε οποιαδήποτε σελίδα ή ανάρτηση.

## Φόρτωση των σεναρίων Wink σε όλο τον ιστότοπο

<Steps>
1. Στο πάνελ διαχείρισης του Ghost, μεταβείτε στις **Ρυθμίσεις → Code Injection**.
2. Στο πλαίσιο **Site Header**, προσθέστε το φύλλο στυλ:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. Στο πλαίσιο **Site Footer**, προσθέστε το σενάριο και τον φορτωτή εφαρμογής (αντικαταστήστε το `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Κάντε κλικ στο **Αποθήκευση**.
</Steps>

:::tip
Το Client ID σας είναι διαθέσιμο στον λογαριασμό σας Wink στην ενότητα [Applications](/el/portal/settings/applications).
:::

## Προσθήκη στοιχείου σε ανάρτηση ή σελίδα

Ο επεξεργαστής του Ghost υποστηρίζει μια **κάρτα HTML** που σας επιτρέπει να εισάγετε ακατέργαστο HTML οπουδήποτε στο περιεχόμενό σας.

<Steps>
1. Ανοίξτε την ανάρτηση ή τη σελίδα στον επεξεργαστή Ghost.
2. Πληκτρολογήστε `/` για να ανοίξετε τον επιλογέα καρτών και επιλέξτε **HTML**.
3. Επικολλήστε το HTML του στοιχείου, για παράδειγμα:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Κάντε κλικ έξω από την κάρτα για προεπισκόπηση και στη συνέχεια δημοσιεύστε.
</Steps>

## Code injection ανά σελίδα

Για ένα στοιχείο που πρέπει να εμφανίζεται μόνο σε μία σελίδα, το Ghost υποστηρίζει επίσης code injection ανά σελίδα:

<Steps>
1. Ανοίξτε τις ρυθμίσεις της ανάρτησης ή της σελίδας (εικονίδιο γραναζιού στον επεξεργαστή).
2. Μεταβείτε στο **Code Injection**.
3. Προσθέστε το tag του στοιχείου στο πεδίο **Footer** (ο παγκόσμιος app-loader χρειάζεται μόνο στο footer σε όλο τον ιστότοπο).
4. Αποθηκεύστε.
</Steps>

## Διαθέσιμα στοιχεία

| Στοιχείο | HTML tag | Σκοπός |
|---|---|---|
| Content loader | `<wink-content-loader>` | Εμφάνιση καρτών ξενοδοχείων, πλεγμάτων, χαρτών |
| Lookup | `<wink-lookup>` | Μπάρα αναζήτησης προορισμών |
| Search button | `<wink-search-button>` | Άνοιγμα επιλογέα δρομολογίου |
| Account button | `<wink-account-button>` | Σύνδεση / μενού χρήστη |
| Itinerary button | `<wink-itinerary-button>` | Εμφάνιση τρέχοντος δρομολογίου |
| Itinerary picker | `<wink-itinerary-picker>` | Πλήρης φόρμα δρομολογίου |
| Shopping cart | `<wink-shopping-cart-button>` | Κουμπί σύνοψης καλαθιού |

Δείτε τα [Web Components](/el/integrations/web-components) για την πλήρη αναφορά χαρακτηριστικών.
