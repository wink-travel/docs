[Wink Extranet](https://extranet.wink.travel) είναι η πύλη ιδιοκτησίας μας. Απευθύνεται σε ξενοδοχεία, αλυσίδες, μάρκες και εταιρείες διαχείρισης ξενοδοχείων για να δημιουργήσουν και να διαχειριστούν το ψηφιακό προφίλ των ιδιοκτησιών τους στην πλατφόρμα Wink.

Το Extranet μας έχει σχεδιαστεί με προσοχή ώστε να είναι όσο το δυνατόν πιο απλό, αλλά ταυτόχρονα να υποστηρίζει όλες τις λειτουργίες που οι προχωρημένοι χρήστες περιμένουν από μια προηγμένη πύλη ιδιοκτησίας. Για να σας δώσουμε μερικά παραδείγματα:

1. Απλότητα: Χρησιμοποιήστε τη έξυπνη ροή εισαγωγής μας για να εισάγετε την ιδιοκτησία σας και να πάτε LIVE σε λιγότερο από μία ώρα.
2. Προχωρημένο: Το Wink υποστηρίζει 8500 διαφορετικές παραλλαγές πολιτικών ακύρωσης.
3. Κουλ: Δώστε σε ένα μόνο άτομο στον πλανήτη 10% έκπτωση χωρίς να χρειάζεται να συνδεθεί ή να πληκτρολογήσει κωδικό προσφοράς.

Αφήστε τους οδηγούς μας να σας καθοδηγήσουν σε όλη τη διαδικασία. Όταν νιώσετε αρκετά άνετα... προχωρήστε μόνοι σας.

Το Wink είναι ένα προηγμένο σύστημα διαχείρισης περιεχομένου για ιδιοκτησίες και παρόχους εμπειριών. Ό,τι μπορείτε να κάνετε σε μία από τις εφαρμογές μας, μπορείτε να το κάνετε μέσω API. Αυτό σας επιτρέπει να κάνετε μια βαθιά ενσωμάτωση με το Wink σε τέτοιο βαθμό ώστε να μπορείτε να πουλάτε μέσω Wink χωρίς ποτέ να χρειαστεί να συνδεθείτε σε κάποια από τις εφαρμογές μας.

:::tip[Επαγγελματική Συμβουλή]
Πληρώνετε για ακριβή φιλοξενία μέσων μέσω της Leonardo International; Μπορείτε να κάνετε ακριβώς το ίδιο μέσω του Wink ΚΑΙ έχετε πολλές περισσότερες επιλογές για να χειριστείτε και να ρυθμίσετε τις εικόνες και τα βίντεό σας.
:::
