Θα σας ζητηθεί να ολοκληρώσετε τις πολιτικές του καταλύματός σας κατά τη ρύθμιση.

Για να διαχειριστείτε τις πολιτικές του καταλύματός σας, κάντε κλικ στο `Property > Policies` στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τους επισκέπτες σας σχετικά με:

- Τις ώρες check-in/check-out.
- Αν επιτρέπονται κατοικίδια και παιδιά στον χώρο.
- Αν προσφέρετε ίντερνετ και χώρο στάθμευσης στον χώρο.

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
