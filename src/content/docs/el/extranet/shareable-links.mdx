:::note
Αυτή είναι μια υποενότητα της λειτουργίας [Κοινόχρηστοι Σύνδεσμοι](/studio/shareable-links) που είναι διαθέσιμη στο Wink Studio.

Οι ιδιοκτησίες θα έχουν αυτόματα όλο το απόθεμά τους διαθέσιμο ως κοινόχρηστους συνδέσμους χωρίς να χρειάζεται να μεταβείτε στο Wink Studio για να τους δημιουργήσετε.
:::

Οι κοινόχρηστοι σύνδεσμοι είναι μια βασική λειτουργία του Wink και προέκυψε από την ανάγκη να επικοινωνείται το απόθεμα και οι τιμές με δυναμικό τρόπο μέσω του απλούστερου μέσου στο Διαδίκτυο· δηλαδή του υπερσυνδέσμου.

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

Παραπάνω είναι ένα παράδειγμα ενός από τους συνδέσμους μας. Είναι σύντομος και φορητός... ακόμη και αρκετά σύντομος για να τον θυμάστε. Παρότι σύντομος, αυτός ο σύνδεσμος περιέχει μια **εκρηκτική** ποσότητα πληροφοριών, όπως:

- Το App ID σας.
- Το Affiliate ID σας.
- Την εικόνα(ες) και το κείμενο που θα εμφανιστεί στον χρήστη.
- Την εμφάνιση και αίσθηση (*δηλαδή το θέμα, τα χρώματα και άλλα*).
- Το δρομολόγιο που θα χρησιμοποιηθεί για το ερώτημα τιμής (*2 ενήλικες, 1 διανυκτέρευση*).
- Το νόμισμα για την εμφάνιση της τιμής.
- Τη γλώσσα για την εμφάνιση του κειμένου.

Το υπόλοιπο του άρθρου σας καθοδηγεί πώς να διαχειριστείτε τους συνδέσμους προς τα αποθέματά σας.

## Προσαρμογή συνδέσμου

Η φόρμα σας επιτρέπει να προσαρμόσετε τον σύνδεσμό σας με τους εξής τρόπους:

<Steps>
 1. Από προεπιλογή, ο σύνδεσμος παίρνει το όνομα του αποθέματός σας στα Αγγλικά. Μπορείτε να του δώσετε οποιοδήποτε όνομα σας βοηθά να θυμάστε για τι αφορά ο σύνδεσμος.
 2. Από προεπιλογή, εφαρμόζεται η κύρια επωνυμία / προσαρμογή σας στον σύνδεσμο. Μπορείτε να επιλέξετε άλλες [Προσαρμογές](/studio/customization) που έχετε δημιουργήσει στο Wink Studio.
 3. Προσθέστε λέξεις-κλειδιά, διαχωρισμένες με κόμμα, που θα χρησιμοποιηθούν από [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Προαιρετικά εισάγετε το όνομα χρήστη του λογαριασμού σας στο X ή το Facebook App ID.
 5. Προσθέστε τίτλους και περιγραφές στις γλώσσες που θέλετε να δει ο χρήστης.
 6. Επιλέξτε μία ή περισσότερες εικόνες που θα εμφανίζονται όταν μοιράζεστε τον σύνδεσμο σε πλατφόρμες όπως το Facebook ή το LinkedIn.
 7. Επιλέξτε ένα θέμα / επικάλυψη που προσθέτει ένα σχεδιασμένο επώνυμο πάνω στην εικόνα(ες) που επιλέξατε. *Σημείωση: Έτσι εμφανίζονται οι τιμές πάνω στην εικόνα.*
 8. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
 </Steps>

Παραπάνω βλέπετε μια εικόνα με όλες τις διαθέσιμες ενέργειες για τον σύνδεσμο σας:

- **Προεπισκόπηση** Σας δείχνει μια γρήγορη επισκόπηση του πώς έχει ρυθμιστεί ο σύνδεσμος και πώς θα φαίνεται στον χρήστη.
- **Ενημέρωση** Ενημερώνει τη ρύθμιση του συνδέσμου σας.
- **Προσθήκη στα WinkLinks** Προσθέτει τον σύνδεσμο στον λογαριασμό σας στα WinkLinks.
- **Αντιγραφή** Αντιγράφει τον σύνδεσμο στο πρόχειρο σας ώστε να μπορείτε εύκολα να τον μοιραστείτε οπουδήποτε θέλετε (*συμπεριλαμβανομένων των [WinkLinks](/link-manager/wink-links) *).
- **Κοινή χρήση** Ανοίγει ένα νέο παράθυρο με επιλογές για να μοιραστείτε σε πολλά κοινωνικά δίκτυα και πλατφόρμες μηνυμάτων.
- **Κωδικός QR** Δημιουργεί έναν κωδικό QR που μπορείτε να μοιραστείτε σε πλατφόρμες όπως το IG που είναι πιο εστιασμένες σε εικόνες.
- **Ενσωμάτωση** Σας δείχνει πώς να ενσωματώσετε αυτόν τον σύνδεσμο ως [Διαφημιστικό Banner](/developers/web-components/#content-loader) στην ιστοσελίδα σας.
- **Χρήση με WordPress** Σας δείχνει πώς να χρησιμοποιήσετε το [plugin WordPress](/developers/wordpress) μας για να ενσωματώσετε αυτόν τον σύνδεσμο στην ιστοσελίδα σας.

Καλύπτουμε μερικές από αυτές τις επιλογές, πιο αναλυτικά, παρακάτω.

Αν θέλετε να μοιραστείτε έναν κωδικό QR αντί για σύνδεσμο, κάντε κλικ στο κουμπί `QR code` και θα εμφανιστεί νέο παράθυρο με τον κωδικό QR σας.

Πώς να αποθηκεύσετε τον κωδικό QR σας:

- Αν είστε σε φορητό υπολογιστή, κάντε δεξί κλικ στην εικόνα και επιλέξτε `Αποθήκευση εικόνας`.
- Αν είστε σε κινητό τηλέφωνο:
    - Πατήστε και κρατήστε το δάχτυλό σας πάνω στην εικόνα μέχρι να εμφανιστεί επιλογή για αποθήκευση της εικόνας στις Φωτογραφίες.
    - Κάντε στιγμιότυπο οθόνης του κωδικού QR, κόψτε το ώστε να ταιριάζει και αποθηκεύστε το στις Φωτογραφίες.

### Ενσωμάτωση

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Ακολουθεί πώς να ενσωματώσετε τον σύνδεσμο σας, ως διαφημιστικό banner, στην ιστοσελίδα σας:

- Η γραμμή 3 δείχνει πώς να ενσωματώσετε τα στυλ του Wink στην ιστοσελίδα σας.
- Οι γραμμές 6 έως 9 δείχνουν πώς να χρησιμοποιήσετε το [wink-content-loader](/developers/web-components/#content-loader) Web Component και να του πείτε να φορτώσει ένα διαφημιστικό banner για τον κωδικό σας.
- Η γραμμή 11 δείχνει πώς να ενσωματώσετε το Javascript μας στην ιστοσελίδα σας.
- Η γραμμή 13 δείχνει πώς να ενσωματώσετε το [wink-app-loader](/developers/web-components/#application-loader) Web Component και να του πείτε να φορτώσει τις προτιμήσεις ρύθμισης σε επίπεδο σελίδας.
