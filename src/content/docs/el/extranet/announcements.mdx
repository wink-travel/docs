Οι ανακοινώσεις είναι ένας τρόπος να ενημερώσετε τους επισκέπτες σας για σημαντικές πληροφορίες με τρόπο που τραβάει την προσοχή.
Εάν μια ανακοίνωση `ενεργοποιηθεί`, εμφανίζεται ως νέο παράθυρο πάνω από τη σελίδα προορισμού σας.

Για να δείτε τις ανακοινώσεις σας, κάντε κλικ στο `Property > Announcement` στη βασική γραμμή πλοήγησης.

Κάντε κλικ στο κουμπί `Create announcement` για να δημιουργήσετε μια νέα καταχώρηση.

<Steps>
1. Επιλέξτε αν θέλετε ή όχι να συμπεριλάβετε έναν τίτλο.
2. Οι ανακοινώσεις μπορούν να ενεργοποιηθούν με έναν από τους δύο τρόπους:
    - Να εμφανίζονται πάντα, κάθε φορά που κάποιος επισκέπτεται τη σελίδα σας.
    - Να εμφανίζονται μόνο όταν το δρομολόγιο ταιριάζει με το δρομολόγιο της ανακοίνωσης.
3. Εάν επιλέξατε να εμφανίζεται η ανακοίνωση μόνο για ένα συγκεκριμένο χρονικό διάστημα, προσθέστε τις ημερομηνίες στα πεδία της φόρμας παρακάτω.
4. Κάντε κλικ στο κουμπί `Save` και θα επιστρέψετε στις ανακοινώσεις σας.
</Steps>
