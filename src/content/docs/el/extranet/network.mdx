Αυτό το άρθρο σας δείχνει πώς να συνδεθείτε με ενθουσιώδεις συνεργάτες στο Wink και να διαχειριστείτε τους υπάρχοντες.

## Κανάλια πωλήσεων

Για να διαχειριστείτε τα Κανάλια πωλήσεών σας, κάντε κλικ στο `Distribution > Sales channels` από τη βασική γραμμή πλοήγησης.

Τα κανάλια πωλήσεων είναι οι υπάρχουσες συνεργασίες σας με συνεργάτες, κωδικοποιημένες με το απόθεμα, τις προσφορές και τις εκπτώσεις στις οποίες έχουν πρόσβαση.

Για να ενημερώσετε ένα υπάρχον κανάλι πωλήσεων, κάντε κλικ στον σύνδεσμο `Actions` κάτω από το κανάλι πωλήσεων που θέλετε να ενημερώσετε.

- **Έκπτωση μέλους** Το ποσοστό μέλους δίνεται στους ταξιδιώτες που κάνουν κράτηση μέσω αυτού του καναλιού πωλήσεων.
- **Προμήθεια** Η προμήθεια αποδίδεται στο κανάλι πωλήσεων για κάθε επιτυχημένη κράτηση.
- **Μονές προσφορές** Προσθέστε τις προσφορές που θέλετε να έχει αυτό το κανάλι πωλήσεων.
- **Συνδυασμένες προσφορές** Προσθέστε τις συνδυασμένες προσφορές που θέλετε να έχει αυτό το κανάλι πωλήσεων.
- **Ενεργοποιημένο** Εναλλάξτε τον διακόπτη ενεργοποίησης για να ενεργοποιήσετε / απενεργοποιήσετε πλήρως το κανάλι πωλήσεων.
- **Σε μαύρη λίστα** Η σηματοδότηση ενός καναλιού πωλήσεων ως σε μαύρη λίστα καθιστά αδύνατη την κράτηση μέσω αυτού του καναλιού.

Μόλις είστε ικανοποιημένοι με τις ρυθμίσεις σας, κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

:::tip
Με το να βάλετε ένα κανάλι πωλήσεων σε μαύρη λίστα, το καθιστάτε αδύνατο να λάβει οποιεσδήποτε τιμές από εσάς· ακόμη και τις τιμές του Wink Network.
:::

## Εξερεύνηση δικτύου

Για να βρείτε νέους συνεργάτες στο Wink, κάντε κλικ στο `Distribution > Explore network` από τη βασική γραμμή πλοήγησης.

Σε αυτή την ενότητα, μπορείτε να δείτε εκκρεμείς αιτήσεις από συνεργάτες στο πάνω μέρος της σελίδας και μια λίστα συνεργατών για σύνδεση παρακάτω.

### Απάντηση σε αίτηση

Μπορείτε να επιλέξετε να απορρίψετε την αίτηση συνεργάτη. Αν θέλετε να συνεχίσετε με την αίτηση, θα μεταφερθείτε στη σελίδα `affiliate details` όπου μπορείτε να λάβετε περισσότερες πληροφορίες για τον συνεργάτη καθώς και μετρήσεις προηγούμενης απόδοσης στο Wink.

### Αναζήτηση συνεργατών

Στο πλέγμα συνεργατών, μπορείτε να αναζητήσετε συνεργάτες με βάση το όνομα, τον τύπο και την τοποθεσία. Όταν δείτε κάποιον που σας αρέσει, κάντε κλικ σε αυτόν και θα μεταφερθείτε στη σελίδα `affiliate details`.

### Σύνδεση με συνεργάτη

Αν επιλέξετε να δημιουργήσετε άμεση σχέση με έναν συνεργάτη, θα σας ζητηθεί να κωδικοποιήσετε τη σχέση με τον ίδιο τρόπο που διαχειρίζεστε ένα κανάλι πωλήσεων παραπάνω.

Μόλις αποθηκεύσετε το νέο κανάλι πωλήσεων, θα επιστρέψετε στη λίστα των καναλιών πωλήσεών σας.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν το `Network` μπορούν να μεταβούν στο [Developers > APIs > Distribution](/el/developers/apis/#distribution-api).
