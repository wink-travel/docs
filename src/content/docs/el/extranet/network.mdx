Αυτό το άρθρο σάς δείχνει πώς να συνδεθείτε με ενθουσιώδεις συνεργάτες στο Wink και πώς να διαχειριστείτε υπάρχοντες.

## Κανάλια πωλήσεων

Για να διαχειριστείτε τα κανάλια πωλήσεών σας, κάντε κλικ στο`Distribution > Sales channels`από την κύρια γραμμή πλοήγησης.

Τα κανάλια πωλήσεων είναι οι υπάρχουσες σχέσεις συνεργατών σας, κωδικοποιημένες με το απόθεμα, τις προσφορές και τις εκπτώσεις στις οποίες έχουν πρόσβαση.

Για να ενημερώσετε ένα υπάρχον κανάλι πωλήσεων, κάντε κλικ στο`Actions`σύνδεσμο κάτω από το κανάλι πωλήσεων που θέλετε να ενημερώσετε.

* **Έκπτωση για μέλη**Η τιμή μέλους παρέχεται στους ταξιδιώτες που κάνουν κράτηση μέσω αυτού του καναλιού πωλήσεων.
* **Επιτροπή**Η προμήθεια κατανέμεται στο κανάλι πωλήσεων για κάθε επιτυχημένη κράτηση.
* **Μεμονωμένες προωθητικές ενέργειες**Προσθέστε τις προωθητικές ενέργειες που θέλετε να έχει αυτό το κανάλι πωλήσεων.
* **Ομαδοποιημένες προσφορές**Προσθέστε τις συνδυασμένες προωθητικές ενέργειες που θέλετε να έχει αυτό το κανάλι πωλήσεων.
* **Ενεργοποιημένο**Ενεργοποιήστε/απενεργοποιήστε πλήρως το κανάλι πωλήσεων με εναλλαγή του διακόπτη.
* **Μαύρη λίστα**Η επισήμανση ενός καναλιού πωλήσεων ως σε μαύρη λίστα καθιστά αδύνατη την κράτηση μέσω αυτού του καναλιού πωλήσεων.

Μόλις είστε ικανοποιημένοι με τις ρυθμίσεις σας, κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

:::συμβουλή
Με τη μαύρη λίστα ενός καναλιού πωλήσεων, καθίσταται αδύνατο για το κανάλι πωλήσεων να λάβει οποιεσδήποτε τιμές από εσάς· ακόμη και τις τιμές του Wink Network.
:::

## Εξερευνήστε το δίκτυο

Για να βρείτε νέους συνεργάτες στο Wink, κάντε κλικ στο`Distribution > Explore network`από την κύρια γραμμή πλοήγησης.

Σε αυτήν την ενότητα, μπορείτε να δείτε εκκρεμή αιτήματα από συνεργάτες στο επάνω μέρος της σελίδας και μια λίστα συνεργατών για σύνδεση παρακάτω.

### Απάντηση στο αίτημα

Μπορείτε να επιλέξετε να απορρίψετε το αίτημα συνεργασίας. Εάν θέλετε να συνεχίσετε με το αίτημα, θα μεταφερθείτε στο`affiliate details`σελίδα όπου μπορείτε να λάβετε περισσότερες πληροφορίες σχετικά με τον συνεργάτη καθώς και παλαιότερες μετρήσεις απόδοσης στο Wink.

### Αναζήτηση συνεργατών

Στο πλέγμα συνεργατών, μπορείτε να αναζητήσετε συνεργάτες με βάση το όνομα, τον τύπο και την τοποθεσία. Όταν δείτε κάποιον που σας αρέσει, κάντε κλικ πάνω του και θα μεταφερθείτε στο`affiliate details`σελίδα.

### Σύνδεση με συνεργάτη

Εάν επιλέξετε να δημιουργήσετε μια άμεση σχέση με έναν συνεργάτη, θα σας ζητηθεί να κωδικοποιήσετε τη σχέση με τον ίδιο τρόπο που θα διαχειριζόσασταν ένα κανάλι πωλήσεων παραπάνω.

Μόλις αποθηκεύσετε το νέο κανάλι πωλήσεων, θα επιστρέψετε στη λίστα καναλιών πωλήσεων.

## API

Προγραμματιστές που θέλουν να διαχειριστούν τα`Network`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Διανομή](/developers/apis/#distribution-api).
