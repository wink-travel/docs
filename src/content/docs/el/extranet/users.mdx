Η σελίδα χρηστών σας επιτρέπει να διαχειρίζεστε ποιος έχει πρόσβαση στο κατάλυμά σας στο Extranet.

Για να διαχειριστείτε την ομάδα σας, κάντε κλικ στο `Property > Users` από τη βασική γραμμή πλοήγησης.

## Πρόσκληση μελών της ομάδας

Εισάγετε μια διεύθυνση email και κάντε κλικ στο `Invite` για να στείλετε μια πρόσκληση. Ο παραλήπτης θα λάβει ένα email με έναν σύνδεσμο για να ενταχθεί στην ομάδα του καταλύματός σας.

## Διαχείριση ρόλων

Κάθε μέλος της ομάδας μπορεί να έχει έναν από τους παρακάτω ρόλους:

- **Owner** — Πλήρης πρόσβαση σε όλες τις ρυθμίσεις του καταλύματος, συμπεριλαμβανομένης της δυνατότητας διαχείρισης άλλων χρηστών.
- **Manager** — Πρόσβαση στις καθημερινές λειτουργίες διαχείρισης του καταλύματος.

Μπορείτε να προαγάγετε έναν manager σε owner χρησιμοποιώντας το κουμπί `Make owner`.

## Αφαίρεση μελών

Κάντε κλικ στο `Remove` δίπλα σε οποιοδήποτε μέλος της ομάδας για να ανακαλέσετε την πρόσβασή του. Η αλλαγή ισχύει άμεσα.

## Εκκρεμείς προσκλήσεις

Η ενότητα προσκλήσεων εμφανίζει όλες τις εκκρεμείς προσκλήσεις που δεν έχουν ακόμη γίνει αποδεκτές.
