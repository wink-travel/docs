## Δημιουργία ιδιότητας

Μόλις ολοκληρωθεί η επαλήθευση, θα μεταφερθείτε στη διεύθυνση[σελίδα ιδιοτήτων](https://extranet.wink.travel/property/list)που σας δείχνει όλες τις υπάρχουσες ιδιότητές σας.
Εάν έχετε πολλές, μπορείτε να φιλτράρετε τις ιδιότητές σας πληκτρολογώντας ένα όνομα.

Κάντε κλικ στο`Create a new property`κουμπί για να συνεχίσετε. Θα μεταφερθείτε στο[δημιουργία ιδιοκτησίας](https://extranet.wink.travel/property/create)σελίδα.

Υπάρχουν τρεις τρόποι που μπορείτε να ακολουθήσετε για να ξεκινήσετε την ενσωμάτωση του ακινήτου σας.

1. **Εγχειρίδιο**Η προσέγγιση του μάστορα`Do it all yourself!`
2. **Αξίωση**Αναλάβετε την ευθύνη ενός`lead`και να το διεκδικήσετε ως ιδιοκτησία σας.
3. **Εξυπνος**Αξιοποιήστε την Τεχνητή Νοημοσύνη για να αυτοματοποιήσετε ορισμένα βήματα της διαδικασίας ένταξης.

### Εγχειρίδιο

*Χρόνος ολοκλήρωσης κατά προσέγγιση: ~5 λεπτά*

Η μη αυτόματη διαδικασία ενσωμάτωσης σάς επιτρέπει να δημιουργήσετε μια ιδιότητα με τον ελάχιστο αριθμό πεδίων για να ξεκινήσετε.
Ο εύχρηστος οδηγός μας σάς καθοδηγεί σε όλα όσα απαιτούνται για την ενεργοποίηση του προφίλ σας και τη δημοσίευση.

<Steps>
  1. **Πόλη** Πείτε μας πού βρίσκεται το ακίνητό σας *Π.χ. Εστεπόνα, Ισπανία*.
  2. **Ονομα** Το όνομα του ακινήτου σας. *Π.χ. Santiva Resort*
  3. **Τοπική ονομασία** Εάν το ακίνητό σας βρίσκεται σε χώρα που δεν χρησιμοποιεί το λατινικό αλφάβητο, πληκτρολογήστε το τοπικό όνομα του ακινήτου σας. *Π.χ. ซานทิวา รีสอร์ท στα ταϊλανδικά*
  4. **Διεύθυνση** Συμπληρώστε τη διεύθυνση του ακινήτου
  5. **Γραφείο κρατήσεων** Συμπληρώστε τα στοιχεία του γραφείου κρατήσεων.
  6. **Συμφωνία** Αποδεχτείτε μας [όροι παροχής υπηρεσιών](/extranet/terms-of-service) και [όροι πληρωμής](/extranet/payment-terms) να συνεχίσω.
  7. Κάντε κλικ στο `Save` κουμπί.
</Steps>


Θα ανακατευθυνθείτε πίσω στη σελίδα των ιδιοτήτων σας.

Συγχαρητήρια 🎉 ​​...δημιουργήσατε την πρώτη σας ιδιοκτησία στο Wink με μη αυτόματο τρόπο.

### Αξίωση

*Χρόνος ολοκλήρωσης κατά προσέγγιση: ~2 λεπτά*

Άλλα μέλη μπορούν επίσης[σε προσκαλώ](/studio/search#empty-results)για να γίνετε μέλος του Wink. Θα λάβετε μια πρόσκληση μέσω email από εμάς με ένα`lead`αίτημα που μπορείτε να διεκδικήσετε και να χρησιμοποιήσετε για την οριστικοποίηση της δημιουργίας ακινήτου.

Για αυτές τις περιπτώσεις, θα:

<Steps>
  1. Κάντε κλικ στην καρτέλα Αξίωση από το [σελίδα ιδιοτήτων](https://extranet.wink.travel/property/list).
  2. Κάντε κλικ στο ακίνητο που θέλετε να διεκδικήσετε.
  3. Συνέχισε το [έξυπνος](#intelligent) ροή εργασίας.
</Steps>


Μπορείτε επίσης να καταργήσετε την αξίωση και να δημιουργήσετε την ιδιοκτησία σας από την αρχή, αν το επιθυμείτε.

### Εξυπνος

*Χρόνος ολοκλήρωσης κατά προσέγγιση: ~2 λεπτά*

:::σημείωση
Το ακίνητό σας πρέπει να είναι διαθέσιμο στο Google Places για να συνεχίσετε.
:::

Η έξυπνη ενσωμάτωση εξαλείφει ορισμένα από τα βήματα που θα έπρεπε να κάνετε χειροκίνητα. Λαμβάνετε:

* Γεωγραφική τοποθεσία
* Διεύθυνση
* Εικόνες
* Αυτόματα δημιουργημένο κείμενο καλωσορίσματος.
* ...και μερικά άλλα πράγματα δωρεάν

Κάντε κλικ στο`Get Started`κουμπί για έξυπνη ενσωμάτωση.

<Steps>
  1. **Ονομα** Ξεκινήστε να πληκτρολογείτε το όνομα του ακινήτου σας όπως είναι στην Google. *Π.χ. Santiva Resort*.
  2. **Διεύθυνση** Μερικές φορές, δεν μπορούμε να αντιστοιχίσουμε σωστά μια διεύθυνση Google και θα ζητήσουμε τη βοήθειά σας.
  3. Κάντε κλικ στο `Save` κουμπί.
</Steps>


Θα ανακατευθυνθείτε πίσω στη σελίδα των ιδιοτήτων σας.

## Ταμπλό

Μόλις δημιουργηθεί το ακίνητό σας, μπορείτε να ολοκληρώσετε τη δημιουργία του προφίλ σας ή να ξεκινήσετε τη διαχείριση του ακινήτου σας κάνοντας κλικ στο`Manage`σύνδεσμο στη λίστα ακινήτων σας.
Αυτό σας μεταφέρει στον πίνακα ελέγχου του ακινήτου.

Ο πίνακας ελέγχου αποτελεί τη βάση για το ακίνητό σας. Αποτελείται από ενότητες που σας βοηθούν να πλοηγηθείτε και να αξιοποιήσετε στο έπακρο τα εργαλεία μας:

### Πώς τα πάω;

Αυτή η ενότητα σάς δίνει μια γρήγορη εικόνα για την κατάσταση του ακινήτου σας. Σας ενημερώνει για τα εξής:

* Συνολικός αριθμός κρατήσεων
* Μέση αξία κράτησης
* Συνολικό ποσό πωλήσεων

Αυτοί οι αριθμοί περιλαμβάνουν όλες τις κρατήσεις που πραγματοποιούνται από οποιοδήποτε κανάλι πωλήσεων στο δίκτυο Wink.

### Πουλήστε το απόθεμά σας

Αυτή η ενότητα σάς ενημερώνει για το είδος των εργαλείων που έχετε στη διάθεσή σας όταν πουλάτε το απόθεμά σας μέσω του Wink.
Κάνοντας κλικ σε οποιοδήποτε από τα πλαίσια θα μεταφερθείτε στο[Στούντιο Wink](/studio/what-is-studio).
Είτε είστε ξενοδοχείο είτε συνεργαζόμενος οργανισμός, όλες οι πωλήσεις γίνονται εκεί και τα ξενοδοχεία μπορούν να είναι οι δικοί τους συνεργάτες, πουλώντας μέσω του δικού τους εγγενούς καναλιού πωλήσεων.

### Κλείσιμο αιτημάτων δικτύου

Μπορείτε να βρείτε εκκρεμή αιτήματα από συνεργάτες που θέλουν να επικοινωνήσουν μαζί σας σε αυτήν την ενότητα.
Υπάρχει επίσης ένας σύνδεσμος εκεί που σας μεταφέρει απευθείας στο`Distribution > Explore network`ώστε να μπορείτε να διαχειρίζεστε όλα τα τρέχοντα και τα εκκρεμή κανάλια πωλήσεών σας.

### Εργασίες

Η ενότητα εργασιών σας δείχνει προαιρετικά, αλλά προτεινόμενα, πράγματα που μπορείτε να κάνετε για να ξεχωρίσετε περισσότερο το προφίλ του καταλύματός σας.

*Σας συνιστούμε να προσθέσετε τις βαθμολογίες διαδικτυακής φήμης σας από οποιονδήποτε εξωτερικό ιστότοπο στον οποίο έχετε αξιολογηθεί, όπως ενημερωμένες αξιολογήσεις από μεγάλα OTA.*

### Αναλυτικά στοιχεία

Μια νέα ιδιοκτησία λαμβάνει αυτόματα τα δικά της βασικά αναλυτικά στοιχεία κατά τη δημιουργία της. Από προεπιλογή, θα δείτε:

* Καθαρά έσοδα για τους τελευταίους 12 μήνες.
* Μέση αξία κράτησης για τους τελευταίους 12 μήνες.
* Συνολικές προβολές σελίδας για τους τελευταίους 12 μήνες.
* Συνολικές κρατήσεις για τους τελευταίους 12 μήνες.
* Συνολικές ακυρώσεις τους τελευταίους 12 μήνες.

Αυτά τα γραφήματα μπορούν εύκολα να τροποποιηθούν. Ωστόσο, αυτά τα γραφήματα περιέχουν όλα τα αναλυτικά στοιχεία που θα χρειαστεί το ακίνητό σας για κάποιο χρονικό διάστημα.

## Ανακοινώσεις

Οι ανακοινώσεις είναι ένας τρόπος να ειδοποιήσετε τους επισκέπτες σας για σημαντικές πληροφορίες με τρόπο που να γίνεται αντιληπτός.
Εάν μια ανακοίνωση είναι`triggered`, εμφανίζεται ως νέο παράθυρο πάνω από τη σελίδα προορισμού σας.

Για να δείτε τις ανακοινώσεις σας, κάντε κλικ στο`Property > Announcement`στην κύρια γραμμή πλοήγησης.

Κάντε κλικ στο`Create announcement`κουμπί για να δημιουργήσετε μια νέα καταχώρηση.

<Steps>
  1. Επιλέξτε αν θα συμπεριλάβετε ή όχι έναν τίτλο.
  2. Οι ανακοινώσεις μπορούν να ενεργοποιηθούν με έναν από τους δύο τρόπους:
     * Να εμφανίζεται πάντα, κάθε φορά που κάποιος επισκέπτεται τη σελίδα σας.
     * Εμφάνιση μόνο όταν το δρομολόγιο ταιριάζει με το δρομολόγιο της ανακοίνωσης.
  3. Εάν επιλέξατε να εμφανίζετε ανακοινώσεις μόνο για ένα συγκεκριμένο εύρος ημερομηνιών, προσθέστε τις ημερομηνίες στα παρακάτω πεδία φόρμας.
  4. Κάντε κλικ στο `Save` και θα επιστρέψετε στις ανακοινώσεις σας.
</Steps>

## Προφίλ

Το προφίλ του ακινήτου είναι ένα υποχρεωτικό βήμα στον οδηγό ένταξης και μας δίνει την ευκαιρία να γνωρίσουμε καλύτερα το ακίνητό σας χρησιμοποιώντας κριτήρια, όπως:

* Αριθμός αστεριών.
* Τύπος ξενοδοχείου.
* Τύπος τοποθεσίας.
* ...και περισσότερα

Για να διαχειριστείτε το προφίλ του ακινήτου σας, κάντε κλικ στο`Property > Profile`στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τη φόρμα και κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Διεύθυνση

Θα σας ζητηθεί η διεύθυνση του ακινήτου σας κατά την εγκατάσταση.

Για να διαχειριστείτε τη διεύθυνσή σας, κάντε κλικ στο`Property > Address`στην κύρια καρτέλα πλοήγησης.

:::σημείωση
Σε ορισμένες χώρες, όπως η Ταϊλάνδη, συνιστάται να συμπληρώσετε και την τοπική διεύθυνση στα Ταϊλανδέζικα, καθώς μπορεί να είναι πολύ χρήσιμο για τους επισκέπτες να την δώσουν στον οδηγό ταξί.
:::

Ενημερώστε τη φόρμα και κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

:::σημείωση
Η λίστα μας με τις πόλεις προέρχεται απευθείας από[geonames.org](https://www.geonames.org)Εάν η πόλη ή το νησί σας δεν εμφανίζεται στην αναπτυσσόμενη λίστα πόλεων, ενημερώστε μας στη διεύθυνση<support@wink.travel>. :::

## Γραφείο κρατήσεων

Θα σας ζητηθούν τα στοιχεία επικοινωνίας του γραφείου κρατήσεων κατά την εγκατάσταση.

Για να διαχειριστείτε το γραφείο κρατήσεων, κάντε κλικ στο`Property > Reservation Desk`στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τη φόρμα και κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

:::σημείωση
Δεν είναι απαραίτητο να χρησιμοποιήσετε ένα συγκεκριμένο όνομα του κύριου ατόμου στη ρεσεψιόν. Απλώς προσθέτει στην οικειότητα του χώρου.
Οι επισκέπτες που έρχονται θα ήθελαν να επικοινωνήσουν με ένα πραγματικό άτομο κατά την άφιξή τους και να μπορούν να ρωτήσουν κάτι όπως "Γεια... είναι εδώ η Λίζα;"
:::

## Κείμενο καλωσορίσματος

Θα σας ζητηθεί να γράψετε μια σύντομη και εκτενή περιγραφή του ακινήτου σας κατά την εγκατάσταση.

Για να διαχειριστείτε το κείμενο καλωσορίσματος, κάντε κλικ στο`Property > Welcome Text`στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τη φόρμα και κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

:::σημείωση
Κατ' ελάχιστον, χρειαζόμαστε να συμπληρώσετε το κείμενο καλωσορίσματος στα Αγγλικά.

Δημιουργήστε κείμενο καλωσορίσματος για όλες τις άλλες γλώσσες που θέλετε να διαχειριστείτε.
Θα χρησιμοποιήσουμε το αγγλικό σας κείμενο για να το μεταφράσουμε σε όλες τις άλλες δημοφιλείς γλώσσες.
:::

## Γενικός διευθυντής

Η προσθήκη πληροφοριών για τον Γενικό Διευθυντή βοηθά στη δημιουργία μεγαλύτερης εξοικείωσης με τον τόπο.

Για να ενημερώσετε τα στοιχεία του Γενικού Διευθυντή, κάντε κλικ στο`Property > General Manager`στην κύρια καρτέλα πλοήγησης.

Μπορείτε να προσθέσετε το όνομα και τη φωτογραφία του τρέχοντος Γενικού Διευθυντή σας μαζί με ένα προσωπικό μήνυμα στους επισκέπτες.

Ενημερώστε τη φόρμα και κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Πολιτικές

Θα σας ζητηθεί να ολοκληρώσετε τις πολιτικές του ακινήτου σας κατά τη διάρκεια της ρύθμισης.

Για να διαχειριστείτε τις πολιτικές ακινήτων σας, κάντε κλικ στο`Property > Policies`στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τους καλεσμένους σας για:

* Ώρες άφιξης/αναχώρησης.
* Είτε επιτρέπετε κατοικίδια και παιδιά στις εγκαταστάσεις.
* Είτε προσφέρετε ίντερνετ και χώρο στάθμευσης στις εγκαταστάσεις.

Ενημερώστε τη φόρμα και κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Τοποθεσία στον χάρτη

Θα σας ζητηθεί να ορίσετε την τοποθεσία σας στον χάρτη κατά τη διάρκεια της ρύθμισης.

Για να διαχειριστείτε την τοποθεσία του ακινήτου σας στον χάρτη, κάντε κλικ στο`Property > Map location`στην κύρια καρτέλα πλοήγησης.

Μπορείτε να ορίσετε την τοποθεσία σας στον χάρτη με δύο τρόπους:

1. Αναζητήστε το ακίνητό σας με το όνομά του στο πεδίο αναζήτησης πάνω από τον χάρτη και επιλέξτε το.
2. Χρησιμοποιώντας τον χάρτη, κάντε ζουμ και μετατόπιση μέχρι την τοποθεσία του ακινήτου σας και κάντε κλικ στον χάρτη όταν το βρείτε.

Θα εμφανιστεί ένας δείκτης στον χάρτη με τις ενημερωμένες συντεταγμένες σας.

:::σημείωση
Το ακίνητό σας πρέπει να είναι καταχωρημένο στις Σελίδες Google για να εμφανίζεται στο πεδίο αναζήτησης.
:::

Ενημερώστε τη φόρμα και κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Ανεσεις

Οι παροχές του καταλύματος είναι υπηρεσίες και εγκαταστάσεις που είναι διαθέσιμες στο ακίνητό σας.

Για να διαχειριστείτε τις παροχές του καταλύματός σας, κάντε κλικ στο`Property > Amenities`στην κύρια καρτέλα πλοήγησης.

Επιλέξτε όλα τα πλαίσια που ισχύουν για το ακίνητό σας.

Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

### Μεταδεδομένα

Μπορείτε να προσθέσετε επιπλέον μεταδεδομένα στις εικόνες και τα βίντεό σας κάνοντας κλικ στο εικονίδιο ✏️ δίπλα στην εικόνα σας.

Τα μεταδεδομένα μπορούν να είναι:

* **Τρόπος ζωής**Ορίστε το εάν μια εικόνα αντιπροσωπεύει έναν συγκεκριμένο τρόπο ζωής.
* **Κατηγορία**Ορίστε το εάν ταιριάζει με μια συγκεκριμένη κατηγορία.*π.χ. Θέα στην πισίνα*
* **Υπότιτλοι**Περιγράψτε την εικόνα σε όποια γλώσσα θέλετε.

:::σημείωση\[Γιατί να χρησιμοποιήσουμε τον τρόπο ζωής;]
Εάν οι ταξιδιώτες φιλτράρουν έναν συγκεκριμένο τρόπο ζωής και εμφανιστεί το κατάλυμά σας, εμφανίζουμε την εικόνα που ταιριάζει με τον τρόπο ζωής που έχει επιλέξει ο ταξιδιώτης ως την επιλεγμένη εικόνα σας, καθιστώντας έτσι το κατάλυμά σας πιο σχετικό.
:::

Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Τρόποι ζωής

Θα σας ζητηθεί να συσχετίσετε ορισμένους τρόπους ζωής με το ακίνητό σας κατά τη διάρκεια της ρύθμισης.

Για να διαχειριστείτε τον τρόπο ζωής σας, κάντε κλικ στο`Property > Lifestyles`στην κύρια καρτέλα πλοήγησης.

Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Φήμη

Η φήμη σας είναι το ψηφιακό σας νόμισμα και θέλετε να μπορείτε να την παίρνετε μαζί σας όπου κι αν πάτε.

Μια καταχώριση φήμης μπορεί να είναι οτιδήποτε θέλει να πιστωθεί το κατάλυμά σας, όπως μια αξιολόγηση τρίτου μέρους ή για το bake-off όπου ο σεφ σας κέρδισε το χρυσό.
Σε αυτήν την ενότητα, μπορείτε να προσθέσετε τις βαθμολογίες και τις κατατάξεις σας από όλο το Διαδίκτυο.

Για να διαχειριστείτε τη φήμη σας, κάντε κλικ στο`Property > Reputation`στην κύρια καρτέλα πλοήγησης.

Κάντε κλικ στο`Add reputation score`κουμπί.

<Steps>
  1. **Κατηγορία** Επιλέξτε τον τύπο βαθμολογίας που θέλετε να προσθέσετε. *π.χ. αξιολόγηση τρίτου μέρους*
  2. **Προμηθευτής** Πληκτρολογήστε ποιος σας έδωσε τη βαθμολογία. *π.χ. BigOTA.com*
  3. **Τύπος αξιολόγησης** Επιλέξτε τη μορφή με την οποία λάβατε τη βαθμολογία. *π.χ. Αριθμητικό*
  4. **Ημερομηνία** Προαιρετικά, εισαγάγετε την ημερομηνία που λάβατε τη βαθμολογία.
  5. **Εκτίμηση** Εισαγάγετε τη βαθμολογία που λάβατε. *π.χ. 8,5*
  6. **Μέγιστη βαθμολογία** Προαιρετικά, εισαγάγετε τη μέγιστη δυνατή βαθμολογία για αυτόν τον τύπο βραβείου. *π.χ. 10*
</Steps>


Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Κοινωνικά μέσα

Τα μέσα κοινωνικής δικτύωσης σας κρατούν σε επαφή με νέους και υπάρχοντες ταξιδιώτες με την πάροδο του χρόνου.
Προσθέστε τους λογαριασμούς σας στα μέσα κοινωνικής δικτύωσης στο προφίλ σας, ώστε οι επισκέπτες να μπορούν εύκολα να επικοινωνήσουν μαζί σας και να μάθουν για το κατάλυμά σας.

Για να διαχειριστείτε τους λογαριασμούς σας στα μέσα κοινωνικής δικτύωσης, κάντε κλικ στο`Property > Social media`στην κύρια καρτέλα πλοήγησης.

<Steps>
  1. Ενεργοποιήστε/ενεργοποιήστε το διακόπτη ενός κοινωνικού δικτύου που θέλετε να ενεργοποιήσετε.
  2. Εισαγάγετε το όνομα του λογαριασμού σας σε αυτό το δίκτυο.
</Steps>


Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Πράσινος Δείκτης

Ο δείκτης πρασίνου σας μετριέται με βάση τον αντίκτυπο που έχει το ακίνητό σας στο τοπικό περιβάλλον και στις αγοραστικές σας αποφάσεις.

Για να διαχειριστείτε τη βαθμολογία σας στο Green Index, κάντε κλικ στο`Property > Green Index`στην κύρια καρτέλα πλοήγησης.

Μελετήστε κάθε καρτέλα και απαντήστε στο ερωτηματολόγιο όσο καλύτερα μπορείτε.

Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## Δημιουργία επωνυμίας

Μπορείτε να προσαρμόσετε την εμπειρία του ταξιδιώτη όταν βρίσκεται στη σελίδα προορισμού του Wink και όταν λαμβάνει email επιβεβαίωσης κράτησης.

Για να διαχειριστείτε την επωνυμία σας, κάντε κλικ στο`Account > Branding`στην κύρια καρτέλα πλοήγησης.

Για να μάθετε περισσότερα σχετικά με τις επιλογές που έχετε στη διάθεσή σας, μεταβείτε στη διεύθυνση[Προσαρμογή](/studio/customization).

## Διευθυντές

Μπορείτε να προσκαλέσετε άλλους χρήστες να σας βοηθήσουν στη διαχείριση του λογαριασμού συνεργατών σας. Μόλις ένας χρήστης αποδεχτεί την πρόσκλησή σας, θα έχει πλήρη πρόσβαση στον λογαριασμό σας, με εξαίρεση την πληρωμή. Θα είστε πάντα ο μόνος υπεύθυνος για τα χρήματα που σας ανήκουν.

<Steps>
  1. Κάντε κλικ στο `Account > Managers` στην κύρια γραμμή πλοήγησης.
  2. Προσθέστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του χρήστη που θέλετε να ορίσετε ως διαχειριστή λογαριασμού.
  3. Κάντε κλικ στο `Invite` σύνδεσμος δίπλα στο email.
</Steps>


## Ειδοποιήσεις

Κατά καιρούς, θέλουμε να επικοινωνήσουμε μαζί σας. Όταν συμβεί αυτό, θα δείτε το κουδουνάκι στην επάνω δεξιά γωνία να αλλάζει.
Για να διαβάσετε τι σας στείλαμε:

<Steps>
  1. Κάντε κλικ στο `Account > Notifications` στην κύρια γραμμή πλοήγησης.
  2. Θα δείτε νέες και παλαιότερες ειδοποιήσεις.
  3. Επιλέξτε να `Mark as read` ή `Delete` τις ειδοποιήσεις μόλις τις διαβάσετε.
</Steps>
