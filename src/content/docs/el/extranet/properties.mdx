## Δημιουργία ακινήτου

Μόλις πιστοποιηθείτε, θα ανακατευθυνθείτε στη [σελίδα ακινήτων](https://extranet.wink.travel/property/list) σας που εμφανίζει όλα τα υπάρχοντα ακίνητά σας.
Αν έχετε πολλά, μπορείτε να φιλτράρετε τα ακίνητά σας πληκτρολογώντας ένα όνομα.

Κάντε κλικ στο κουμπί `Create a new property` για να συνεχίσετε. Θα ανακατευθυνθείτε στη σελίδα [δημιουργίας ακινήτου](https://extranet.wink.travel/property/create).

Υπάρχουν τρεις τρόποι για να ξεκινήσετε την εισαγωγή του ακινήτου σας.

1. **Manual** Η προσέγγιση του «Κάντο μόνος σου!»
2. **Claim** Αναλάβετε την ιδιοκτησία ενός `lead` και διεκδικήστε το ως το ακίνητό σας.
3. **Intelligent** Αξιοποιήστε την τεχνητή νοημοσύνη για να αυτοματοποιήσετε κάποια βήματα της διαδικασίας εισαγωγής.

### Manual

*Περίπου χρόνος ολοκλήρωσης: ~5 λεπτά*

Η χειροκίνητη διαδικασία εισαγωγής σας επιτρέπει να δημιουργήσετε ένα ακίνητο με τον ελάχιστο αριθμό πεδίων για να ξεκινήσετε.
Ο εύχρηστος οδηγός μας σας καθοδηγεί σε όλα όσα απαιτούνται για να ενεργοποιήσετε το προφίλ σας και να το κάνετε διαθέσιμο.

<Steps>
1. **Πόλη** Πείτε μας πού βρίσκεται το ακίνητό σας *π.χ. Estepona, Ισπανία*.
2. **Όνομα** Το όνομα του ακινήτου σας. *π.χ. Santiva Resort*
3. **Τοπικό όνομα** Αν το ακίνητό σας βρίσκεται σε χώρα που δεν χρησιμοποιεί το λατινικό αλφάβητο, πληκτρολογήστε το τοπικό όνομα του ακινήτου σας. *π.χ. ซานทิวา รีสอร์ท στα Ταϊλανδικά*
4. **Διεύθυνση** Συμπληρώστε τη διεύθυνση του ακινήτου.
5. **Γραφείο κρατήσεων** Συμπληρώστε τα στοιχεία του γραφείου κρατήσεων.
6. **Συμφωνία** Αποδεχτείτε τους [όρους υπηρεσίας](/el/extranet/terms-of-service) και τους [όρους πληρωμής](/el/extranet/payment-terms) για να συνεχίσετε.
7. Κάντε κλικ στο κουμπί `Save`.
</Steps>

Θα ανακατευθυνθείτε ξανά στη σελίδα των ακινήτων σας.

Συγχαρητήρια 🎉 ...δημιουργήσατε το πρώτο σας ακίνητο στο Wink με τον χειροκίνητο τρόπο.

### Claim

*Περίπου χρόνος ολοκλήρωσης: ~2 λεπτά*

Άλλα μέλη μπορούν επίσης να σας [προσκαλέσουν](/el/studio/search#empty-results) να συμμετάσχετε στο Wink. Θα λάβετε ένα email πρόσκλησης από εμάς με ένα αίτημα `lead` που μπορείτε να διεκδικήσετε και να χρησιμοποιήσετε για να ολοκληρώσετε τη δημιουργία του ακινήτου.

Σε αυτές τις περιπτώσεις, θα:

<Steps>
1. Κάντε κλικ στην καρτέλα Claim από τη [σελίδα ακινήτων](https://extranet.wink.travel/property/list).
2. Κάντε κλικ στο ακίνητο που θέλετε να διεκδικήσετε.
3. Συνεχίστε τη ροή εργασίας [intelligent](#intelligent).
</Steps>

Μπορείτε επίσης να αφαιρέσετε τη διεκδίκηση και να δημιουργήσετε το ακίνητό σας από την αρχή, αν το επιθυμείτε.

### Intelligent

*Περίπου χρόνος ολοκλήρωσης: ~2 λεπτά*

:::note
Το ακίνητό σας πρέπει να είναι διαθέσιμο στο Google Places για να συνεχίσετε.
:::

Η έξυπνη εισαγωγή εξαλείφει κάποια από τα βήματα που θα κάνατε χειροκίνητα. Λαμβάνετε:

- Γεωτοποθεσία
- Διεύθυνση
- Εικόνες
- Αυτόματα δημιουργημένο κείμενο καλωσορίσματος.
- ...και μερικά ακόμα δωρεάν

Κάντε κλικ στο κουμπί `Get Started` για έξυπνη εισαγωγή.

<Steps>
1. **Όνομα** Ξεκινήστε να πληκτρολογείτε το όνομα του ακινήτου σας όπως είναι στο Google. *π.χ. Santiva Resort*.
2. **Διεύθυνση** Μερικές φορές, δεν μπορούμε να αντιστοιχίσουμε σωστά μια διεύθυνση Google και θα ζητήσουμε τη βοήθειά σας.
3. Κάντε κλικ στο κουμπί `Save`.
</Steps>

Θα ανακατευθυνθείτε ξανά στη σελίδα των ακινήτων σας.

## Πίνακας ελέγχου

Μόλις δημιουργηθεί το ακίνητό σας, μπορείτε να ολοκληρώσετε τη δημιουργία του προφίλ σας ή να ξεκινήσετε τη διαχείριση του ακινήτου σας κάνοντας κλικ στον σύνδεσμο `Manage` στη λίστα των ακινήτων σας.
Αυτό σας μεταφέρει στον πίνακα ελέγχου του ακινήτου.

Ο πίνακας ελέγχου είναι η βάση για το ακίνητό σας. Αποτελείται από ενότητες που σας βοηθούν να πλοηγηθείτε και να αξιοποιήσετε στο έπακρο τα εργαλεία μας:

### Πώς τα πάω;

Αυτή η ενότητα σας δίνει μια γρήγορη ματιά στην κατάσταση του ακινήτου σας. Σας ενημερώνει για:

- Συνολικό αριθμό κρατήσεων
- Μέση αξία κράτησης
- Συνολικό ποσό πωλήσεων

Αυτοί οι αριθμοί περιλαμβάνουν όλες τις κρατήσεις που έγιναν από οποιοδήποτε κανάλι πωλήσεων στο δίκτυο Wink.

### Πουλήστε το απόθεμά σας

Αυτή η ενότητα σας ενημερώνει για τον τύπο εργαλείων που είναι διαθέσιμα όταν πουλάτε το απόθεμά σας μέσω Wink.
Κάνοντας κλικ σε οποιοδήποτε από τα κουτιά, θα μεταφερθείτε στο [Wink Studio](/el/studio/what-is-studio).
Είτε είστε ξενοδοχείο είτε συνεργάτης, όλες οι πωλήσεις γίνονται εκεί και τα ξενοδοχεία μπορούν να είναι οι δικοί τους συνεργάτες· πουλώντας μέσω του δικού τους εγγενούς καναλιού πωλήσεων.

### Αιτήματα δικτύου Wink

Μπορείτε να βρείτε εκκρεμή αιτήματα από συνεργάτες που θέλουν να συνδεθούν μαζί σας σε αυτή την ενότητα.
Υπάρχει επίσης ένας σύνδεσμος που σας μεταφέρει απευθείας στο `Distribution > Explore network` ώστε να διαχειριστείτε όλα τα τρέχοντα και εκκρεμή κανάλια πωλήσεων.

### Εργασίες

Η ενότητα εργασιών σας δείχνει προαιρετικά, αλλά συνιστώμενα, πράγματα που μπορείτε να κάνετε για να κάνετε το προφίλ του ακινήτου σας να ξεχωρίσει περισσότερο.

*Συνιστούμε να προσθέσετε τους βαθμούς διαδικτυακής φήμης σας από οποιονδήποτε εξωτερικό ιστότοπο έχετε αξιολόγηση, όπως ενημερωμένες βαθμολογίες από μεγάλες OTA.*

### Αναλύσεις

Ένα νέο ακίνητο λαμβάνει αυτόματα τις βασικές του αναλύσεις κατά τη δημιουργία. Από προεπιλογή, θα δείτε:

- Καθαρά έσοδα για τους τελευταίους 12 μήνες.
- Μέση αξία κράτησης για τους τελευταίους 12 μήνες.
- Συνολικές προβολές σελίδας για τους τελευταίους 12 μήνες.
- Συνολικές κρατήσεις για τους τελευταίους 12 μήνες.
- Συνολικές ακυρώσεις για τους τελευταίους 12 μήνες.

Αυτά τα διαγράμματα μπορούν εύκολα να τροποποιηθούν. Ωστόσο, αυτά τα διαγράμματα περιέχουν όλες τις αναλύσεις που θα χρειαστεί το ακίνητό σας για ένα διάστημα.

## Ανακοινώσεις

Οι ανακοινώσεις είναι ένας τρόπος να ενημερώσετε τους επισκέπτες σας για σημαντικές πληροφορίες με τρόπο που να γίνεται αντιληπτός.
Αν μια ανακοίνωση `ενεργοποιηθεί`, εμφανίζεται ως νέο παράθυρο πάνω από τη σελίδα προορισμού σας.

Για να δείτε τις ανακοινώσεις σας, κάντε κλικ στο `Property > Announcement` στη βασική γραμμή πλοήγησης.

Κάντε κλικ στο κουμπί `Create announcement` για να δημιουργήσετε μια νέα καταχώρηση.

<Steps>
1. Επιλέξτε αν θα συμπεριλάβετε ή όχι έναν τίτλο.
2. Οι ανακοινώσεις μπορούν να ενεργοποιηθούν με δύο τρόπους:
    - Να εμφανίζονται πάντα, κάθε φορά που κάποιος επισκέπτεται τη σελίδα σας.
    - Να εμφανίζονται μόνο όταν το δρομολόγιο ταιριάζει με το δρομολόγιο της ανακοίνωσης.
3. Αν επιλέξατε να εμφανίζεται η ανακοίνωση μόνο για συγκεκριμένο χρονικό διάστημα, προσθέστε τις ημερομηνίες στα πεδία της φόρμας παρακάτω.
4. Κάντε κλικ στο κουμπί `Save` και θα επιστρέψετε στις ανακοινώσεις σας.
</Steps>

## Προφίλ

Το προφίλ του ακινήτου είναι ένα απαραίτητο βήμα στον οδηγό εισαγωγής μας και μας δίνει την ευκαιρία να γνωρίσουμε καλύτερα το ακίνητό σας χρησιμοποιώντας κριτήρια, όπως:

- Αριθμός αστέρων.
- Τύπος ξενοδοχείου.
- Τύπος τοποθεσίας.
- ...και άλλα

Για να διαχειριστείτε το προφίλ του ακινήτου σας, κάντε κλικ στο `Property > Profile` στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Διεύθυνση

Θα σας ζητηθεί η διεύθυνση του ακινήτου σας κατά τη ρύθμιση.

Για να διαχειριστείτε τη διεύθυνσή σας, κάντε κλικ στο `Property > Address` στην κύρια καρτέλα πλοήγησης.

:::note
Σε ορισμένες χώρες, όπως η Ταϊλάνδη, συνιστάται να συμπληρώσετε επίσης την τοπική διεύθυνση στα Ταϊλανδικά, καθώς μπορεί να είναι πολύ χρήσιμο για τους επισκέπτες να τη δώσουν στον οδηγό ταξί.
:::

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

:::note
Η λίστα των πόλεων μας προέρχεται απευθείας από το [geonames.org](https://www.geonames.org). Αν η πόλη ή το νησί σας δεν εμφανίζεται στο αναπτυσσόμενο μενού πόλεων, παρακαλούμε ενημερώστε μας στο support@wink.travel.
:::

## Γραφείο κρατήσεων

Θα σας ζητηθούν τα στοιχεία επικοινωνίας του γραφείου κρατήσεών σας κατά τη ρύθμιση.

Για να διαχειριστείτε το γραφείο κρατήσεών σας, κάντε κλικ στο `Property > Reservation Desk` στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

:::note
Δεν απαιτείται να χρησιμοποιήσετε συγκεκριμένο όνομα του υπεύθυνου στη ρεσεψιόν. Απλώς προσθέτει οικειότητα στον χώρο.
Οι επισκέπτες που φτάνουν θα ήθελαν να συνδεθούν με ένα πραγματικό άτομο κατά την άφιξή τους και να μπορούν να ρωτήσουν κάτι όπως "Γεια... είναι η Λίζα εδώ;"
:::

## Κείμενο καλωσορίσματος

Θα σας ζητηθεί να γράψετε μια μακρά και μια σύντομη περιγραφή του ακινήτου σας κατά τη ρύθμιση.

Για να διαχειριστείτε το κείμενο καλωσορίσματος, κάντε κλικ στο `Property > Welcome Text` στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

:::note
Τουλάχιστον, χρειάζεται να συμπληρώσετε το κείμενο καλωσορίσματος στα Αγγλικά.

Δημιουργήστε κείμενο καλωσορίσματος για όλες τις άλλες γλώσσες που θέλετε να διαχειριστείτε.
Θα χρησιμοποιήσουμε το αγγλικό κείμενό σας για να το μεταφράσουμε σε όλες τις άλλες δημοφιλείς γλώσσες.
:::

## Γενικός διευθυντής

Η προσθήκη πληροφοριών για τον Γενικό Διευθυντή βοηθά στη δημιουργία μεγαλύτερης οικειότητας με τον χώρο.

Για να ενημερώσετε τις πληροφορίες του Γενικού Διευθυντή, κάντε κλικ στο `Property > General Manager` στην κύρια καρτέλα πλοήγησης.

Μπορείτε να προσθέσετε το όνομα και τη φωτογραφία του τρέχοντος Γενικού Διευθυντή σας μαζί με ένα προσωπικό μήνυμα προς τους επισκέπτες.

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Πολιτικές

Θα σας ζητηθεί να συμπληρώσετε τις πολιτικές του ακινήτου σας κατά τη ρύθμιση.

Για να διαχειριστείτε τις πολιτικές του ακινήτου σας, κάντε κλικ στο `Property > Policies` στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τους επισκέπτες σας για:

- Ώρες check-in/out.
- Αν επιτρέπονται κατοικίδια και παιδιά στον χώρο.
- Αν προσφέρετε ίντερνετ και χώρο στάθμευσης στον χώρο.

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Τοποθεσία στο χάρτη

Θα σας ζητηθεί να ορίσετε την τοποθεσία σας στο χάρτη κατά τη ρύθμιση.

Για να διαχειριστείτε την τοποθεσία του ακινήτου σας στο χάρτη, κάντε κλικ στο `Property > Map location` στην κύρια καρτέλα πλοήγησης.

Μπορείτε να ορίσετε την τοποθεσία σας στο χάρτη με δύο τρόπους:

1. Αναζητήστε το ακίνητό σας με το όνομά του στο πεδίο αναζήτησης πάνω από το χάρτη και επιλέξτε το.
2. Χρησιμοποιώντας το χάρτη, κάντε ζουμ και μετακινήστε τον χάρτη στην τοποθεσία του ακινήτου σας και κάντε κλικ στο χάρτη όταν το βρείτε.

Θα εμφανιστεί ένας δείκτης χάρτη με τις ενημερωμένες συντεταγμένες σας.

:::note
Το ακίνητό σας πρέπει να είναι καταχωρημένο στο Google Pages για να εμφανίζεται στο πεδίο αναζήτησης.
:::

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Παροχές

Οι παροχές ακινήτου είναι υπηρεσίες και εγκαταστάσεις που είναι διαθέσιμες στο ακίνητό σας.

Για να διαχειριστείτε τις παροχές του ακινήτου σας, κάντε κλικ στο `Property > Amenities` στην κύρια καρτέλα πλοήγησης.

Τσεκάρετε όλα τα κουτιά που ισχύουν για το ακίνητό σας.

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Φωτογραφίες & Βίντεο

:::note
Αυτή η ενότητα αφορά μόνο εικόνες και βίντεο των εγκαταστάσεών σας, όπως λόμπι, εξωτερικοί χώροι κ.λπ. Όλο το άλλο απόθεμα έχει τη δική του ενότητα μέσων.
:::

Θα σας ζητηθεί να προσθέσετε φωτογραφίες του ακινήτου σας κατά τη ρύθμιση.

Για να διαχειριστείτε τα μέσα του ακινήτου σας, κάντε κλικ στο `Property > Photos & Videos` στην κύρια καρτέλα πλοήγησης.

Για να προσθέσετε μια νέα φωτογραφία ή βίντεο:

<PhotosAndVideosText/>

### Μεταδεδομένα

Μπορείτε να προσθέσετε επιπλέον μεταδεδομένα στις εικόνες και τα βίντεό σας κάνοντας κλικ στο εικονίδιο ✏️ δίπλα στην εικόνα σας.

Τα μεταδεδομένα μπορεί να είναι:

- **Lifestyle** Ορίστε το αν μια εικόνα αντιπροσωπεύει έναν συγκεκριμένο τρόπο ζωής.
- **Κατηγορία** Ορίστε το αν ταιριάζει σε μια συγκεκριμένη κατηγορία. *π.χ. Θέα στην πισίνα*
- **Λεζάντες** Περιγράψτε την εικόνα σε οποιαδήποτε γλώσσα θέλετε.

:::note[Γιατί να χρησιμοποιήσετε το lifestyle;]
Αν οι ταξιδιώτες φιλτράρουν με βάση έναν συγκεκριμένο τρόπο ζωής και το ακίνητό σας εμφανίζεται, εμφανίζουμε την εικόνα που ταιριάζει με τον επιλεγμένο τρόπο ζωής του ταξιδιώτη ως την κύρια εικόνα σας· κάνοντας έτσι το ακίνητό σας πιο σχετικό.
:::

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Τρόποι ζωής

Θα σας ζητηθεί να συσχετίσετε κάποιους τρόπους ζωής με το ακίνητό σας κατά τη ρύθμιση.

Για να διαχειριστείτε τους τρόπους ζωής σας, κάντε κλικ στο `Property > Lifestyles` στην κύρια καρτέλα πλοήγησης.

Ενεργοποιήστε τον διακόπτη δίπλα στους τρόπους ζωής που θέλετε να χρησιμοποιήσετε.

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

:::caution[Απαιτήσεις]
Μπορείτε να επιλέξετε έως 6 τρόπους ζωής. Επιλέξτε αυτούς που ταιριάζουν περισσότερο στο ακίνητό σας.
:::

## Φήμη

Η φήμη σας είναι το ψηφιακό σας νόμισμα και θέλετε να μπορείτε να την παίρνετε μαζί σας όπου κι αν πάτε.
Μια καταχώρηση φήμης μπορεί να είναι οτιδήποτε θέλει το ακίνητό σας να πιστωθεί, όπως μια αξιολόγηση τρίτου μέρους ή για τον διαγωνισμό όπου ο σεφ σας κέρδισε χρυσό.
Σε αυτή την ενότητα, μπορείτε να προσθέσετε τους βαθμούς και τις κατατάξεις σας από όλο το Διαδίκτυο.

Για να διαχειριστείτε τη φήμη σας, κάντε κλικ στο `Property > Reputation` στην κύρια καρτέλα πλοήγησης.

Κάντε κλικ στο κουμπί `Add reputation score`.

<Steps>
1. **Κατηγορία** Επιλέξτε τον τύπο βαθμολογίας που θέλετε να προσθέσετε. *π.χ. Αξιολόγηση τρίτου μέρους*
2. **Πάροχος** Πληκτρολογήστε ποιος σας έδωσε τη βαθμολογία. *π.χ. BigOTA.com*
3. **Τύπος αξιολόγησης** Επιλέξτε τη μορφή με την οποία λάβατε τη βαθμολογία. *π.χ. Αριθμητική*
4. **Ημερομηνία** Προαιρετικά, εισάγετε την ημερομηνία που λάβατε τη βαθμολογία.
5. **Βαθμολογία** Εισάγετε τη βαθμολογία που λάβατε. *π.χ. 8.5*
6. **Μέγιστη βαθμολογία** Προαιρετικά, εισάγετε τη μέγιστη δυνατή βαθμολογία για αυτόν τον τύπο βράβευσης. *π.χ. 10*
</Steps>

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Κοινωνικά δίκτυα

Τα κοινωνικά δίκτυα σας κρατούν σε επαφή με νέους και υπάρχοντες ταξιδιώτες με την πάροδο του χρόνου.
Προσθέστε τους λογαριασμούς των κοινωνικών δικτύων σας στο προφίλ σας ώστε οι επισκέπτες να μπορούν εύκολα να επικοινωνήσουν μαζί σας και να μάθουν για το ακίνητό σας.

Για να διαχειριστείτε τους λογαριασμούς κοινωνικών δικτύων, κάντε κλικ στο `Property > Social media` στην κύρια καρτέλα πλοήγησης.

<Steps>
1. Ενεργοποιήστε τον διακόπτη ενός κοινωνικού δικτύου που θέλετε να ενεργοποιήσετε.
2. Εισάγετε το όνομα του λογαριασμού σας σε αυτό το δίκτυο.
</Steps>

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Πράσινος Δείκτης

Ο πράσινος δείκτης σας μετράται με βάση τον αντίκτυπο που έχει το ακίνητό σας στο τοπικό περιβάλλον και στις αγοραστικές σας αποφάσεις.

Για να διαχειριστείτε το σκορ του Πράσινου Δείκτη, κάντε κλικ στο `Property > Green Index` στην κύρια καρτέλα πλοήγησης.

Περιηγηθείτε σε κάθε καρτέλα και απαντήστε στο ερωτηματολόγιο όσο καλύτερα μπορείτε.

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## Εταιρική ταυτότητα

Μπορείτε να προσαρμόσετε την εμπειρία του ταξιδιώτη όταν βρίσκεται στη σελίδα προορισμού σας στο Wink και όταν λαμβάνει email επιβεβαίωσης κράτησης.

Για να διαχειριστείτε την εταιρική ταυτότητα, κάντε κλικ στο `Account > Branding` στην κύρια καρτέλα πλοήγησης.

Για να μάθετε περισσότερα για τις διαθέσιμες επιλογές, μεταβείτε στο [Customization](/el/studio/customization).

## Διαχειριστές

Μπορείτε να προσκαλέσετε άλλους χρήστες να βοηθήσουν στη διαχείριση του λογαριασμού συνεργάτη σας. Μόλις ένας χρήστης αποδεχτεί την πρόσκλησή σας, θα έχει πλήρη πρόσβαση στον λογαριασμό σας με εξαίρεση τις πληρωμές. Εσείς θα είστε πάντα ο μόνος υπεύθυνος για τα κεφάλαια που κατέχετε.

<Steps>
1. Κάντε κλικ στο `Account > Managers` στη βασική γραμμή πλοήγησης.
2. Προσθέστε το email του χρήστη που θέλετε να ορίσετε ως διαχειριστή λογαριασμού.
3. Κάντε κλικ στο σύνδεσμο `Invite` δίπλα στο email.
</Steps>

## Ειδοποιήσεις

Κατά καιρούς, θέλουμε να επικοινωνήσουμε μαζί σας. Όταν συμβαίνει αυτό, θα δείτε το κουδούνι στην πάνω δεξιά γωνία να αλλάζει.
Για να διαβάσετε τι σας στείλαμε:

<Steps>
1. Κάντε κλικ στο `Account > Notifications` στη βασική γραμμή πλοήγησης.
2. Θα δείτε νέες και παλιές ειδοποιήσεις.
3. Επιλέξτε να `Mark as read` ή `Delete` τις ειδοποιήσεις μόλις τις διαβάσετε.
</Steps>
