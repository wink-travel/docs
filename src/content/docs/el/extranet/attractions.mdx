Τα αξιοθέατα σας επιτρέπουν να προβάλλετε και να πουλάτε τα αξιοθέατά σας, εντός και εκτός των εγκαταστάσεων. Για να δημιουργήσετε ένα αξιοθέατο, κάντε κλικ στο κουμπί `Create attraction`.

Οι παρακάτω ενότητες σας καθοδηγούν σε κάθε μέρος της διαχείρισης ενός αξιοθέατου.

## Διαμόρφωση

Η καρτέλα `Configurations` είναι επιλεγμένη από προεπιλογή όταν ξεκινάτε.

Αυτή η ενότητα καλύπτει τα βασικά του αξιοθέατου σας.

- **Εσωτερικό όνομα** Το όνομα του αξιοθέατου, όπως το αναφέρετε εσωτερικά. *π.χ. Fly fishing*
- **Lifestyle** Προαιρετικά επιλέξτε ένα lifestyle που ταιριάζει σε αυτό το αξιοθέατο. *π.χ. Sports*
- **Ηλικία** Προαιρετικά επιλέξτε την κατώτερη συνιστώμενη ηλικιακή ομάδα για αυτό το αξιοθέατο. *π.χ. Adult*
- **Τύπος** Προαιρετικά επιλέξτε τον τύπο αξιοθέατου που ταιριάζει περισσότερο. *π.χ. Fishing*
- **Απαιτείται κράτηση** Αν αυτό το αξιοθέατο χρειάζεται κράτηση. *π.χ. Yes*

## Πρόγραμμα

Αυτή η ενότητα σας επιτρέπει να διαχειριστείτε πότε το αξιοθέατο είναι διαθέσιμο.

### Ώρες λειτουργίας

- **Άνοιγμα** Ώρα της ημέρας που το αξιοθέατο γίνεται διαθέσιμο. *π.χ. 09:00*
- **Κλείσιμο** Ώρα της ημέρας που το αξιοθέατο γίνεται μη διαθέσιμο. *π.χ. 21:00*

### Ημέρες λειτουργίας

Αυτή η ενότητα σας επιτρέπει να ορίσετε τις ημέρες της εβδομάδας που το αξιοθέατο είναι διαθέσιμο.

*Ενεργοποιήστε τον διακόπτη σε ✅ για να υποδείξετε ότι η ημέρα είναι διαθέσιμη.*

### Εποχιακό

:::note
Δεν υπάρχει όμορφο, εγγενές πεδίο ημέρας / μήνα. Σε αυτή την περίπτωση, χρησιμοποιούμε μόνο τον μήνα και την ημέρα από αυτόν τον επιλογέα ημερομηνίας.
:::

Αυτή η ενότητα σας επιτρέπει να ορίσετε τις ημερομηνίες που το αξιοθέατο είναι διαθέσιμο καθ’ όλη τη διάρκεια του έτους.

- **Ημερομηνία έναρξης** Ημερομηνία που το αξιοθέατο γίνεται διαθέσιμο. *π.χ. 11 Σεπτεμβρίου 2024*
- **Ημερομηνία λήξης** Ημερομηνία που το αξιοθέατο γίνεται μη διαθέσιμο. *π.χ. 11 Ιουλίου 2024*

## Παροχές

:::note
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στην καρτέλα `Amenities` για να συνεχίσετε.

Παρόμοια με τις παροχές του καταλύματος, οι παροχές του αξιοθέατου ισχύουν μόνο για το συγκεκριμένο αξιοθέατο. Επιλέξτε κάθε παροχή που ισχύει.

## Περιγραφές

:::note
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και αυτή πρέπει να είναι στην αγγλική γλώσσα.
:::

Κάντε κλικ στην καρτέλα `Descriptions` για να συνεχίσετε.

Παρόμοια με το κείμενο καλωσορίσματος του καταλύματος, μπορείτε να δημιουργήσετε τοπικές περιγραφές για το αξιοθέατο επίσης.
Μπορείτε να γράψετε σε όσες γλώσσες θέλετε. Θα προσθέσουμε μεταφράσεις για όλες τις πιο δημοφιλείς γλώσσες.
Αυτό είναι το κείμενο που θα βλέπουν οι ταξιδιώτες όταν προβάλλουν τα αξιοθέατά σας.

- **Όνομα** Δώστε ένα όνομα στο αξιοθέατο. *π.χ. Fly fishing*
- **Περιγραφή** Περιγράψτε το αξιοθέατο σε μία ή δύο παραγράφους. *π.χ. Μόνο εσείς... το δάσος... ο ποταμός... Ηρεμία σε ένα μπουκάλι.*

## Κοινωνικά δίκτυα

:::note
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στην καρτέλα `Social media` για να συνεχίσετε.

Ίσως αναρωτιέστε, "Γιατί ένα αξιοθέατο έχει τη δική του ενότητα κοινωνικών δικτύων;". Ένα αξιοθέατο μπορεί επίσης να είναι μια εγκατάσταση κοντά στο κατάλυμα αλλά εκτός των εγκαταστάσεων με δικό του προσωπικό και δικό του προφίλ στο IG.

Τα κοινωνικά δίκτυα σας κρατούν σε επαφή με νέους και υπάρχοντες ταξιδιώτες με την πάροδο του χρόνου.
Προσθέστε τους λογαριασμούς των κοινωνικών σας δικτύων στο προφίλ σας ώστε οι επισκέπτες να μπορούν εύκολα να επικοινωνήσουν μαζί σας και να μάθουν για αυτό το αξιοθέατο.

<Steps>
1. Ενεργοποιήστε τον διακόπτη ενός κοινωνικού δικτύου που θέλετε να ενεργοποιήσετε.
2. Εισάγετε το όνομα του λογαριασμού σας σε αυτό το δίκτυο.
</Steps>

## Φήμη

:::note
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στην καρτέλα `Reputation` για να συνεχίσετε.

Παρόμοια με την καρτέλα `Social media`, θα χρησιμοποιήσετε αυτή την καρτέλα αν αυτό το αξιοθέατο έχει ξεχωριστή βαθμολογία φήμης ή επιπλέον διακρίσεις που θέλετε να αναδείξετε εδώ.

Η φήμη σας είναι το ψηφιακό σας νόμισμα και θέλετε να μπορείτε να την παίρνετε μαζί σας όπου κι αν πάτε.
Μια καταχώρηση φήμης μπορεί να είναι οτιδήποτε θέλει το κατάλυμά σας να πιστωθεί, όπως μια αξιολόγηση τρίτου μέρους ή για τον διαγωνισμό όπου ο σεφ σας κέρδισε χρυσό.
Σε αυτή την ενότητα, μπορείτε να προσθέσετε τις βαθμολογίες και τις κατατάξεις σας από όλο το Διαδίκτυο.

Κάντε κλικ στο κουμπί `Add reputation score`.

<Steps>
1. **Κατηγορία** Επιλέξτε τον τύπο βαθμολογίας που θέλετε να προσθέσετε. *π.χ. Αξιολόγηση τρίτου μέρους*
2. **Πάροχος** Πληκτρολογήστε ποιος σας έδωσε τη βαθμολογία. *π.χ. BigOTA.com*
3. **Τύπος βαθμολογίας** Επιλέξτε τη μορφή με την οποία λάβατε τη βαθμολογία. *π.χ. Αριθμητική*
4. **Ημερομηνία** Προαιρετικά, εισάγετε την ημερομηνία που λάβατε τη βαθμολογία.
5. **Βαθμολογία** Εισάγετε τη βαθμολογία που λάβατε. *π.χ. 8.5*
6. **Μέγιστη βαθμολογία** Προαιρετικά, εισάγετε τη μέγιστη δυνατή βαθμολογία για αυτόν τον τύπο βράβευσης. *π.χ. 10*
</Steps>

## Πωλήσεις βάσει χαρακτηριστικών

:::note
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στην καρτέλα `Attribute Based Selling` για να συνεχίσετε.

Όλες οι εγκαταστάσεις μας διαθέτουν τη δική τους ενότητα ABS. Αυτή η ενότητα σας επιτρέπει να δημιουργήσετε ευκαιρίες upsell στο κύριο απόθεμα ή να κάνετε έναν μη-συναλλακτικό τύπο αποθέματος συναλλακτικό.
Σας συνιστούμε έντονα να δημιουργήσετε αυτές τις ευκαιρίες για να συνδεθείτε πιθανώς με τον πελάτη σε υψηλότερη τιμή κατά τη διαδικασία κράτησης.

:::tip
Ένα αξιοθέατο μπορεί να διατεθεί προς πώληση μέσω ABS. Μπορεί να διατεθεί προς πώληση με διάφορους τρόπους:

- Ημερήσια εκδρομή.
- Εκδρομή 3 ημερών.
- +++ αφήστε τη φαντασία σας να τρέξει και κερδίστε επιπλέον χρήματα!!
:::

- **Τιμή** Δώστε στον ταξιδιώτη μια γρήγορη ένδειξη για το πόσο ακριβό είναι αυτό το αξιοθέατο.
- **Ενεργό** Απενεργοποιήστε τον διακόπτη σε 🛑 αν θέλετε αυτό το αξιοθέατο να γίνει μη διαθέσιμο για κράτηση σε όλα τα κανάλια πώλησης.
- **Προμήθεια** Υποδείξτε αν οι συνεργάτες λαμβάνουν προμήθεια για αυτό το απόθεμα.
- **Προεπιλεγμένο** Υποδείξτε αν αυτό το αξιοθέατο πρέπει να τοποθετηθεί πιο εμφανώς αν παρουσιαστεί η επιλογή.
- **Πωλήσεις βάσει χαρακτηριστικών** Ενεργοποιήστε τον διακόπτη σε ✅ για να ενεργοποιήσετε το ABS σε αυτό το απόθεμα. *Μόλις ενεργοποιηθεί, πρέπει να προσθέσετε τουλάχιστον 1 αντικείμενο ABS προς πώληση*

### Συμπληρωματικό αντικείμενο

Όταν ενεργοποιήσετε το ABS, μπορείτε να αρχίσετε να προσθέτετε συμπληρωματικά αντικείμενα και να τα κάνετε διαθέσιμα προς πώληση.

*Από προεπιλογή, έχει ήδη δημιουργηθεί ένα κενό αντικείμενο για εσάς. Μπορείτε να προσθέσετε περισσότερα κάνοντας κλικ στο κουμπί `Add item for sale`.*

- **Όνομα** Δώστε ένα όνομα στο συμπληρωματικό σας αντικείμενο. *π.χ. Ημερήσια εκδρομή*
- **Τύπος τιμολόγησης** Υποδείξτε πώς θέλετε να υπολογίζεται η τιμή. *π.χ. Ορίστε σε `Per day` για να υποδείξετε ότι η τιμή βασίζεται στον αριθμό ημερών.*
- **Βασική τιμή** Εισάγετε την κανονική τιμή αυτού του αντικειμένου. *π.χ. 200 / ημέρα*
- **Εκπτωτική τιμή** Εισάγετε την τιμή που λαμβάνουν οι χρήστες του Wink κατά την κράτηση του αντικειμένου. *π.χ. 150 / ημέρα*

### Όρια

Μπορείτε να περιορίσετε τη διαθεσιμότητα αυτού του αντικειμένου ορίζοντας ελάχιστα / μέγιστα όρια στην ποσότητα.

- **Ελάχιστη ποσότητα** Περιορίστε στην ελάχιστη ποσότητα που πρέπει να γίνει κράτηση. *π.χ. 1 συμμετέχων*
- **Μέγιστη ποσότητα** Περιορίστε στη μέγιστη ποσότητα που πρέπει να γίνει κράτηση. *π.χ. 4 συμμετέχοντες*

### Περιγραφές

:::note
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και αυτή πρέπει να είναι στην αγγλική γλώσσα.
:::

- **Όνομα** Δώστε ένα όνομα στο συμπληρωματικό σας αντικείμενο. *π.χ. Ημερήσια εκδρομή*
- **Περιγραφή** Περιγράψτε το συμπληρωματικό αντικείμενο σε μία ή δύο παραγράφους. *π.χ. Πάρτε μια μέρα άδεια και κατευθυνθείτε προς τον ποταμό.*

## Εγγύτητα

Κάντε κλικ στην καρτέλα `Proximity` για να συνεχίσετε.

Δεν βρίσκεται όλο το απόθεμα εντός των εγκαταστάσεων. Αν έχετε απόθεμα που βρίσκεται εκτός των εγκαταστάσεων, ενημερώστε τον ταξιδιώτη εδώ και θα σας δοθεί η δυνατότητα να ορίσετε ξεχωριστή τοποθεσία στο χάρτη μαζί με προαιρετικές πληροφορίες επικοινωνίας.

*Αυτό μπορεί να είναι χρήσιμο όταν ένα αξιοθέατο βρίσκεται μερικά χιλιόμετρα μακριά από το ξενοδοχείο και ο ταξιδιώτης θέλει να εξηγήσει στον οδηγό ταξί πώς να φτάσει εκεί.*

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

- Ενεργοποιήστε τη διαθεσιμότητα αποθέματος στην κάρτα. *Δείτε την εικόνα παραπάνω*
- Μεταβείτε στο `Distribution > Inventory` από την κύρια γραμμή πλοήγησης αφιερωμένη στη διαχείριση αποθέματος σε όλα τα κανάλια πώλησης.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν `Attractions` μπορούν να μεταβούν στο [Developers > APIs > Experiences](/el/developers/apis/#experiences-api).
