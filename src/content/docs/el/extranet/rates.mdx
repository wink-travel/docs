Δείτε τη λίστα με τους υποστηριζόμενους [παρόχους τιμών](/el/extranet/rate-providers).

Έχουμε ενσωματωθεί με:

- Channel Managers (CM)
- Central Reservation Systems (CRS)
- Property Management Systems (PMS)

Για να διαχειριστείτε τις Τιμές σας, κάντε κλικ στο `Distribution > Master rate calendar` από την κύρια γραμμή πλοήγησης.

Αυτό το άρθρο σας καθοδηγεί σε όσα πρέπει να γνωρίζετε για το πώς λειτουργούν οι τιμές στο Wink.

## Χαρακτηριστικά

:::note
Το Wink διαχειρίζεται τις τιμές σε επίπεδο κύριας τιμής και όχι σε επίπεδο δωματίου επισκέπτη. Αυτό σας δίνει επιπλέον ευελιξία στον τρόπο που επιλέγετε να κατανείμετε τα δωμάτια.
:::

Μια τιμή στο Wink υποστηρίζει τα ακόλουθα δεδομένα:

- **Ποσότητα** Αριθμός διαθέσιμων μονάδων.
- **Ποσό τιμής** Βασική τιμή.
- **Διαθέσιμο** Εάν η τιμή είναι διαθέσιμη για αυτή την ημερομηνία ή όχι.
- **Κλειστό για άφιξη (COA)** Εάν οι επισκέπτες μπορούν να φτάσουν σε αυτή την ημερομηνία.
- **Κλειστό για αναχώρηση (COD)** Εάν οι επισκέπτες μπορούν να αναχωρήσουν σε αυτή την ημερομηνία.
- **Ελάχιστη διάρκεια διαμονής (Min LoS)** Εάν υπάρχει απαίτηση ελάχιστης διάρκειας διαμονής.
- **Μέγιστη διάρκεια διαμονής (Max LoS)** Εάν υπάρχει απαίτηση μέγιστης διάρκειας διαμονής.
- **Ελάχιστη πληρότητα** Εάν υπάρχει απαίτηση ελάχιστης πληρότητας.
- **Μέγιστη πληρότητα** Εάν υπάρχει απαίτηση μέγιστης πληρότητας.

Τα παρακάτω δεδομένα είναι διαθέσιμα μόνο μέσω παρόχου τιμών:

*Εάν ο πάροχος τιμών σας δεν υποστηρίζει αυτά τα δεδομένα, μπορείτε να τα ελέγχετε μέσω των τιμοκαταλόγων σας στο Wink.*

- Τροποποιητής τιμής για μεμονωμένο επισκέπτη
- Τροποποιητής τιμής για επιπλέον ενήλικα
- Τροποποιητής τιμής για επιπλέον παιδί

Οι τιμές είναι ο πυρήνας του Wink. Οι ιδιοκτησίες μπορούν να προσθέτουν τιμές χειροκίνητα ή να συνδέουν τον πάροχο τιμών τους.

## Χειροκίνητη καταχώρηση

:::caution
Δημιουργείτε / ενημερώνετε τιμές χειροκίνητα μόνο αν δεν πουλάτε τους τύπους δωματίων σας σε άλλη πλατφόρμα εκτός από το Wink.
:::

:::caution
Εάν ενημερώνετε τιμές χειροκίνητα ενώ είστε συνδεδεμένοι με πάροχο τιμών, οι τιμές θα αλλάξουν ξανά τη στιγμή που ο πάροχος τιμών σας θα μας στείλει ενημέρωση για το ίδιο εύρος ημερομηνιών.
:::

## Μέσω παρόχου τιμών

Για να αρχίσετε να μας στέλνετε τιμές μέσω του παρόχου τιμών σας, ακολουθήστε τα εξής βήματα:

- Πρώτα βεβαιωθείτε ότι υποστηρίζουμε τον [πάροχο τιμών](/el/extranet/rate-providers) σας.
- Ορίστε τον πάροχο τιμών σας στο `Distribution > Channel manager`.
- Συνδεθείτε στον πάροχο τιμών σας.
- Προσθέστε το κανάλι `Wink` στον πίνακα ελέγχου του παρόχου τιμών σας.
- Αντιστοιχίστε τους τύπους δωματίων / τιμοκαταλόγους σας με αυτούς που δημιουργήσατε στο Wink.
- Ξεκινήστε μια πλήρη αποστολή τιμών προς το Wink [αν δεν έχει γίνει αυτόματα από τον πάροχο τιμών].

Περιμένετε περίπου 3-5 λεπτά για να δείτε τα ημερολόγια κύριας τιμής του Wink να γεμίζουν με τιμές.

## Επαληθευτής

Μπορείτε να δοκιμάσετε τη ρύθμιση των τιμών σας πηγαίνοντας στο `Distribution > Verifier` από την κύρια γραμμή πλοήγησης.

Σε αυτή τη σελίδα, μπορείτε να προσομοιώσετε οποιοδήποτε είδος αιτήματος τιμής που μπορεί να κάνει ένας ταξιδιώτης.

:::tip
Εάν δεν βρίσκετε τη ρύθμιση που ψάχνετε, κάντε κλικ στο κουμπί `Advanced search` στο κάτω μέρος της σελίδας.
:::

### Παράδειγμα

Παρακάτω είναι ένα δείγμα διαδρομής μέσω ενός από τα υπάρχοντα κανάλια πωλήσεών σας:

- **Κανάλι** Επιλέξτε το κανάλι πωλήσεων μέσω του οποίου θέλετε να πουλήσετε. *π.χ. Traveliko OTA*.
- **Νόμισμα** Επιλέξτε το νόμισμα εμφάνισης. *π.χ. USD*
- **Ημερομηνία έναρξης διαμονής** Εισάγετε την ημερομηνία άφιξης. *π.χ. 10 Σεπτεμβρίου 2024*
- **Νύχτες** Εισάγετε τον αριθμό των διανυκτερεύσεων. *π.χ. 1*
- **Επισκέπτες** Εισάγετε ενήλικες / παιδιά που επισκέπτονται. *π.χ. 2 ενήλικες*
- Κάντε κλικ στο κουμπί `Search` για να συνεχίσετε.

Παραπάνω είναι ένα δείγμα αποτελέσματος αναζήτησης με διαθεσιμότητα καθώς και έκπτωση καναλιού / μέλους και προωθητική έκπτωση.
Μπορείτε να επαληθεύσετε ότι οι τιμές εμφανίζονται όπως πρέπει για τις συνθήκες που δοκιμάζετε.

Μπορείτε να δοκιμάσετε ολόκληρη τη ροή εργασίας σας από την αρχή μέχρι το τέλος δημιουργώντας μια δοκιμαστική κράτηση.
Μια δοκιμαστική κράτηση χαρακτηρίζεται ως `test` και δεν εξασφαλίζει την κράτηση με πιστωτική κάρτα ή άλλη μέθοδο πληρωμής.

Οι επιλογές δοκιμαστικής κράτησης περιλαμβάνουν:

- **Ειδοποίηση ιδιοκτησίας μέσω email** Αυτό ενεργοποιεί την αποστολή επιβεβαίωσης μέσω email στην ιδιοκτησία.
- **Ειδοποίηση channel manager μέσω API** Αυτό ενεργοποιεί την αποστολή της κράτησης στον επιλεγμένο πάροχο τιμών σας.
- **Ειδοποίηση κρατούντα μέσω email** Αυτό ενεργοποιεί την αποστολή επιβεβαίωσης μέσω email στον κρατούντα.

Κατά προεπιλογή, ο κρατών σε αυτό το σενάριο είστε εσείς, ο πιστοποιημένος χρήστης. Μπορείτε επίσης να επιλέξετε να προσθέσετε προσαρμοσμένα στοιχεία κρατούντα ενεργοποιώντας το διακόπτη `Add custom booker details` και συμπληρώνοντας τη φόρμα.

*Εάν επιλέξατε να ειδοποιήσουμε τον πάροχο τιμών σας, ο πάροχος θα χειριστεί αυτή την κράτηση όπως κάθε άλλη και δεν θα γνωρίζει ότι είναι δοκιμαστική. Συνιστούμε να δοκιμάζετε με επιστρεφόμενες τιμές για εύκολη ακύρωση μετά την ολοκλήρωση της δοκιμής.*

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν τις `Τιμές` μπορούν να μεταβούν στο [Developers > APIs > Monetize](/el/developers/apis/#monetize-api).
