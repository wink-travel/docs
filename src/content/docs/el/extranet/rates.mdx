:::σημείωση
Δείτε τη λίστα με τα υποστηριζόμενα[πάροχοι τιμών](/extranet/rate-providers).

Είμαστε ενσωματωμένοι με:

* Διαχειριστές Καναλιών (CM)
* Κεντρικά Συστήματα Κρατήσεων (CRS)
* Συστήματα Διαχείρισης Ακινήτων (PMS)
  :::

Για να διαχειριστείτε τις τιμές σας, κάντε κλικ στο`Distribution > Master rate calendar`από την κύρια γραμμή πλοήγησης.

Αυτό το άρθρο σας καθοδηγεί σε όλα όσα πρέπει να γνωρίζετε σχετικά με τον τρόπο λειτουργίας των τιμών στο Wink.

## Χαρακτηριστικά

:::σημείωση
Το Wink διαχειρίζεται τις τιμές στο επίπεδο της κύριας τιμής και όχι στο επίπεδο του δωματίου. Αυτό σας δίνει πρόσθετη ευελιξία στον τρόπο με τον οποίο επιλέγετε να κατανείμετε τα δωμάτια.
:::

Μια τιμή στο Wink υποστηρίζει τα ακόλουθα σημεία δεδομένων:

* **Ποσότητα**Αριθμός διαθέσιμων μονάδων.
* **Ποσό τιμής**Βασική τιμή.
* **Διαθέσιμος**Εάν η τιμή είναι διαθέσιμη για αυτήν την ημερομηνία ή όχι.
* **Κλειστά κατά την άφιξη (COA)**&#x395;άν οι επισκέπτες μπορούν να φτάσουν αυτήν την ημερομηνία.
* **Κλειστά κατά την αναχώρηση (COD)**&#x395;άν οι επισκέπτες μπορούν να αναχωρήσουν αυτήν την ημερομηνία.
* **Ελάχιστη διάρκεια διαμονής (Min LoS)**&#x395;άν υπάρχει ελάχιστη απαίτηση LoS.
* **Μέγιστη διάρκεια διαμονής (Max LoS)**&#x395;άν υπάρχει μέγιστη απαίτηση LoS.
* **Ελάχιστη πληρότητα**Εάν υπάρχει ελάχιστη απαίτηση πληρότητας.
* **Μέγιστη πληρότητα**Εάν υπάρχει μέγιστη απαίτηση LoS.

Τα παρακάτω δεδομένα είναι διαθέσιμα μόνο μέσω ενός παρόχου τιμών:

*Εάν ο πάροχος τιμών σας δεν υποστηρίζει αυτά τα σημεία δεδομένων, μπορείτε να τα ελέγξετε μέσω των προγραμμάτων τιμών σας στο Wink.*

* Τροποποιητής τιμής για έναν μόνο κάτοικο
* Τροποποιητής τιμής για επιπλέον ενήλικες
* Τροποποιητής τιμής επιπλέον παιδιού

Οι τιμές βρίσκονται στον πυρήνα του Wink. Τα καταλύματα μπορούν να προσθέσουν τιμές χειροκίνητα ή να συνδέσουν τον πάροχο τιμών τους.

## Χειροκίνητη εισαγωγή

:::προσοχή
Δημιουργήστε/ενημερώστε τις τιμές χειροκίνητα μόνο εάν δεν πουλάτε τους τύπους δωματίων σας σε καμία άλλη πλατφόρμα εκτός από το Wink.
:::

:::προσοχή
Εάν ενημερώσετε τις τιμές χειροκίνητα, όταν είστε συνδεδεμένοι σε έναν πάροχο τιμών, οι τιμές θα αλλάξουν ξανά τη στιγμή που ο πάροχος τιμών μας στείλει μια ενημέρωση για το ίδιο εύρος ημερομηνιών.

## Μέσω παρόχου χρεώσεων

Για να ξεκινήσετε να μας στέλνετε τιμές μέσω του παρόχου τιμών σας, ακολουθήστε τα εξής βήματα:

* Αρχικά, βεβαιωθείτε ότι σας υποστηρίζουμε[πάροχος τιμών](/extranet/rate-providers).
* Ορίστε τον πάροχο χρεώσεων σας ως εξής`Distribution > Channel manager`.
* Συνδεθείτε στον πάροχο χρεώσεων.
* Προσθέστε το`Wink`κανάλι στον πίνακα ελέγχου του παρόχου χρεώσεων.
* Αντιστοιχίστε τους τύπους δωματίων / τις τιμές σας με αυτές που δημιουργήσατε στο Wink.
* Ξεκινήστε μια πλήρη ώθηση τιμής στο Wink \[εάν δεν γίνει αυτόματα από τον πάροχο τιμών].

Περιμένετε περίπου 3-5 λεπτά για να δείτε τα κύρια ημερολόγια τιμών του Wink να γεμίζουν με τιμές.

## Επιβεβαιωτής

Μπορείτε να δοκιμάσετε τη ρύθμιση τιμών σας μεταβαίνοντας στο`Distribution > Verifier`από την κύρια γραμμή πλοήγησης.

Σε αυτήν τη σελίδα, μπορείτε να προσομοιώσετε οποιοδήποτε είδος αιτήματος τιμής που μπορεί να κάνει ένας ταξιδιώτης.

:::συμβουλή
Εάν δεν μπορείτε να βρείτε τη ρύθμιση που ψάχνετε, κάντε κλικ στο`Advanced search`κουμπί στο κάτω μέρος της σελίδας.
:::

### Παράδειγμα

Παρακάτω είναι ένα δείγμα δρομολογίου σε ένα από τα υπάρχοντα κανάλια πωλήσεών σας:

* **Κανάλι**Επιλέξτε το κανάλι πωλήσεων μέσω του οποίου θέλετε να πουλήσετε.*π.χ. Traveliko OTA*.
* **Νόμισμα**Επιλέξτε το νόμισμα εμφάνισης.*π.χ. δολάρια ΗΠΑ*
* **Ημερομηνία έναρξης παραμονής**Εισαγάγετε την ημερομηνία άφιξης.*π.χ. 10 Σεπτεμβρίου 2024*
* **Νύχτες**Εισαγάγετε τον αριθμό των διανυκτερεύσεων.*π.χ. 1*
* **Επισκέπτες**Εισάγετε ενήλικες / παιδιά που επισκέπτονται.*π.χ. 2 ενήλικες*
* Κάντε κλικ στο`Search`κουμπί για να συνεχίσετε.

Θα δείτε όλα τα δωμάτια επισκεπτών σας να εμφανίζονται στα αποτελέσματα αναζήτησης.
Εάν δεν υπάρχει διαθεσιμότητα ή πρόσβαση, θα μπορείτε να δείτε σε ποιο σημείο της ροής εργασίας δεν ήταν διαθέσιμο.

*π.χ. Δεν έχετε διαθεσιμότητα επειδή ορίσατε την κύρια τιμή σας σε μη διαθέσιμη σε όλα τα κανάλια πωλήσεων.*

Παραπάνω είναι ένα δείγμα αποτελέσματος αναζήτησης με διαθεσιμότητα καθώς και μια έκπτωση καναλιού / μέλους και μια προωθητική έκπτωση.
Μπορείτε να επαληθεύσετε ότι οι τιμές εμφανίζονται όπως θα έπρεπε για τις συνθήκες που δοκιμάζετε.

### Δοκιμαστική κράτηση

<figure>
  <CloudinaryImage width="350" height="459" resize="scale" quality="best" alt="Δοκιμή τρόπου κράτησης" src="academy/test-booking" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Παράθυρο επιβεβαίωσης δοκιμαστικής κράτησης*</figcaption>
</figure>


Μπορείτε να δοκιμάσετε ολόκληρη τη ροή εργασίας σας από άκρο σε άκρο δημιουργώντας μια δοκιμαστική κράτηση.
Μια δοκιμαστική κράτηση επισημαίνεται ως`test`και δεν εξασφαλίζει την κράτηση με πιστωτική κάρτα ή άλλη μέθοδο πληρωμής.

Οι επιλογές κράτησης δοκιμών περιλαμβάνουν:

* **Ειδοποίηση ακινήτου μέσω email**Αυτό ενεργοποιεί την αποστολή ενός email επιβεβαίωσης στο κατάλυμα.
* **Ειδοποίηση διαχειριστή καναλιού μέσω API**Αυτό ενεργοποιεί την αποστολή της κράτησης στον πάροχο τιμών που έχετε επιλέξει.
* **Ειδοποιήστε τον υπεύθυνο κράτησης μέσω email**Αυτό ενεργοποιεί την αποστολή του email επιβεβαίωσης της κράτησης.

Από προεπιλογή, ο χρήστης που κάνει κράτηση σε αυτό το σενάριο είστε εσείς, ο χρήστης με έλεγχο ταυτότητας. Μπορείτε επίσης να επιλέξετε να προσθέσετε προσαρμοσμένα στοιχεία χρήστη ενεργοποιώντας την επιλογή`Add custom booker details`αλλαγή και συμπλήρωση της φόρμας.

*Εάν επιλέξατε να ειδοποιήσουμε τον πάροχο τιμών σας, ο πάροχος τιμών θα αντιμετωπίσει αυτήν την κράτηση ως οποιαδήποτε άλλη κράτηση και δεν θα γνωρίζει ότι πρόκειται για δοκιμαστική κράτηση. Συνιστούμε να κάνετε δοκιμές με επιστρέψιμες τιμές για εύκολη ακύρωση μετά την ολοκλήρωση της δοκιμής.*

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Rates`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Δημιουργία εσόδων](/developers/apis/#monetize-api).
