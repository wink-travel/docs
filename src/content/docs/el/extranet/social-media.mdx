Τα κοινωνικά δίκτυα σας κρατούν σε επαφή με νέους και υπάρχοντες ταξιδιώτες με την πάροδο του χρόνου.  
Προσθέστε τους λογαριασμούς των κοινωνικών σας δικτύων στο προφίλ σας ώστε οι επισκέπτες να μπορούν εύκολα να επικοινωνήσουν μαζί σας και να μάθουν για την ιδιοκτησία σας.

Για να διαχειριστείτε τους λογαριασμούς των κοινωνικών σας δικτύων, κάντε κλικ στο `Property > Social media` στην κύρια καρτέλα πλοήγησης.

<Steps>
1. Ενεργοποιήστε τον διακόπτη του κοινωνικού δικτύου που θέλετε να ενεργοποιήσετε.
2. Εισάγετε το όνομα του λογαριασμού σας σε αυτό το δίκτυο.
</Steps>

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
