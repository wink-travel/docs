Τα πρόσθετα είναι μια προηγμένη μορφή εγκαταστάσεων και εμπειριών. Τα πρόσθετα σάς επιτρέπουν να προβάλλετε και να πουλάτε τα πρόσθετά σας, εντός και εκτός των εγκαταστάσεων. Για να δημιουργήσετε ένα πρόσθετο, κάντε κλικ στο`Create add-on`κουμπί.

Οι ακόλουθες ενότητες σας καθοδηγούν σε κάθε στάδιο της διαχείρισης ενός πρόσθετου.

## Διαμόρφωση

Το`Configurations`Η καρτέλα είναι επιλεγμένη από προεπιλογή όταν ξεκινάτε.

Αυτή η ενότητα καλύπτει τα βασικά του πρόσθετου σας.

* **Εσωτερικό όνομα**Το όνομα του πρόσθετου, όπως το αναφέρετε εσωτερικά.*π.χ. στοιχειωμένο σπίτι*
* **Αριθμός μονάδων**Ορίστε τη διαθέσιμη ποσότητα.*π.χ. 10*
* **Επιτακτικός**Εάν αυτό το πρόσθετο είναι υποχρεωτικό με την κράτηση μαζί με ένα δωμάτιο.*π.χ. Όχι*
* **πρόγραμμα τιμών**Προαιρετικά, ορίστε ένα πρόγραμμα τιμών για να αξιοποιήσετε τους περιορισμούς του.

## Πρόγραμμα

Αυτή η ενότητα σάς επιτρέπει να διαχειριστείτε πότε το πρόσθετο είναι άμεσα διαθέσιμο.

### Ώρες λειτουργίας

* **Ανοίγει**Ώρα της ημέρας κατά την οποία το πρόσθετο καθίσταται διαθέσιμο.*π.χ. 09:00*
* **Κλείνει**Ώρα της ημέρας κατά την οποία το πρόσθετο δεν είναι διαθέσιμο.*π.χ. 21:00*

### Ημέρες λειτουργίας

Αυτή η ενότητα σάς επιτρέπει να ορίσετε τις ημέρες της εβδομάδας κατά τις οποίες το πρόσθετο είναι διαθέσιμο.

*Ενεργοποιήστε το διακόπτη στη θέση ✅ για να υποδείξετε ότι η ημέρα είναι διαθέσιμη.*

### Εποχής

:::σημείωση
Δεν υπάρχει όμορφο, εγγενές πεδίο φόρμας Ημέρα / Μήνας. Σε αυτό το σενάριο, χρησιμοποιούμε μόνο τον μήνα και την ημέρα από αυτόν τον επιλογέα ημερομηνίας.
::::

Αυτή η ενότητα σάς επιτρέπει να ορίσετε τις ημερομηνίες που το πρόσθετο θα είναι διαθέσιμο καθ' όλη τη διάρκεια του έτους.

* **Ημερομηνία έναρξης**Ημερομηνία κατά την οποία το πρόσθετο θα γίνει διαθέσιμο.*π.χ. 11 Σεπτεμβρίου 2024*
* **Ημερομηνία έναρξης**Ημερομηνία κατά την οποία το πρόσθετο δεν θα είναι διαθέσιμο.*π.χ. 11 Ιουλίου 2024*

## Ανεσεις

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Amenities`καρτέλα για να συνεχίσετε.

Όπως και με τις παροχές του καταλύματος, οι πρόσθετες παροχές ισχύουν μόνο για τη συγκεκριμένη πρόσθετη παροχή. Ελέγξτε κάθε παροχή που ισχύει.

## Περιγραφές

:::σημείωση
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και να είναι στα Αγγλικά.
:::

Κάντε κλικ στο`Descriptions`καρτέλα για να συνεχίσετε.

Όπως και με το κείμενο καλωσορίσματος του καταλύματος, μπορείτε να δημιουργήσετε και μεταφρασμένες περιγραφές για το πρόσθετο.
Μπορείτε να γράψετε σε όσες γλώσσες θέλετε. Θα προσθέσουμε μεταφράσεις για όλες τις πιο δημοφιλείς γλώσσες.
Αυτό είναι το κείμενο που θα βλέπουν οι ταξιδιώτες όταν βλέπουν τα πρόσθετά σας.

* **Ονομα**Δώστε ένα όνομα στο πρόσθετο.*π.χ. μεταφορά με υδροπλάνο*
* **Περιγραφή**Περιγράψτε το πρόσθετο σε μία ή δύο παραγράφους.*π.χ. Το νησί μας είναι προσβάσιμο μόνο με υδροπλάνο...*

## Κοινωνικά μέσα

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Social media`καρτέλα για να συνεχίσετε.

Μπορεί να αναρωτιέστε: «Γιατί ένα πρόσθετο έχει τη δική του ενότητα στα μέσα κοινωνικής δικτύωσης;». Ένα πρόσθετο μπορεί επίσης να είναι μια εγκατάσταση κοντά στο κατάλυμα αλλά εκτός του χώρου, με το δικό της προσωπικό και το δικό της προφίλ στο IG.

Τα μέσα κοινωνικής δικτύωσης σας κρατούν σε επαφή με νέους και υπάρχοντες ταξιδιώτες με την πάροδο του χρόνου.
Προσθέστε τους λογαριασμούς σας στα μέσα κοινωνικής δικτύωσης στο προφίλ σας, ώστε οι επισκέπτες να μπορούν εύκολα να επικοινωνήσουν μαζί σας και να μάθουν για αυτό το πρόσθετο.

<Steps>
  1. Ενεργοποιήστε/ενεργοποιήστε το διακόπτη ενός κοινωνικού δικτύου που θέλετε να ενεργοποιήσετε.
  2. Εισαγάγετε το όνομα του λογαριασμού σας σε αυτό το δίκτυο.
</Steps>


## Φήμη

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Reputation`καρτέλα για να συνεχίσετε.

Παρόμοια με το`Social media`καρτέλα, θα χρησιμοποιούσατε αυτήν την καρτέλα εάν αυτό το πρόσθετο έχει ξεχωριστή βαθμολογία/βαθμολογίες φήμης ή επιπλέον διακρίσεις που θέλετε να επισημάνετε εδώ.

Η φήμη σας είναι το ψηφιακό σας νόμισμα και θέλετε να μπορείτε να την παίρνετε μαζί σας όπου κι αν πάτε.

Μια καταχώριση φήμης μπορεί να είναι οτιδήποτε θέλει να πιστωθεί το κατάλυμά σας, όπως μια αξιολόγηση τρίτου μέρους ή για το bake-off όπου ο σεφ σας κέρδισε το χρυσό.
Σε αυτήν την ενότητα, μπορείτε να προσθέσετε τις βαθμολογίες και τις κατατάξεις σας από όλο το Διαδίκτυο.

Κάντε κλικ στο`Add reputation score`κουμπί.

<Steps>
  1. **Κατηγορία** Επιλέξτε τον τύπο βαθμολογίας που θέλετε να προσθέσετε. *π.χ. αξιολόγηση τρίτου μέρους*
  2. **Προμηθευτής** Πληκτρολογήστε ποιος σας έδωσε τη βαθμολογία. *π.χ. BigOTA.com*
  3. **Τύπος αξιολόγησης** Επιλέξτε τη μορφή με την οποία λάβατε τη βαθμολογία. *π.χ. Αριθμητικό*
  4. **Ημερομηνία** Προαιρετικά, εισαγάγετε την ημερομηνία που λάβατε τη βαθμολογία.
  5. **Εκτίμηση** Εισαγάγετε τη βαθμολογία που λάβατε. *π.χ. 8,5*
  6. **Μέγιστη βαθμολογία** Προαιρετικά, εισαγάγετε τη μέγιστη δυνατή βαθμολογία για αυτόν τον τύπο βραβείου. *π.χ. 10*
</Steps>


## Πώληση βάσει χαρακτηριστικών

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Attribute Based Selling`καρτέλα για να συνεχίσετε.

Όλες οι εγκαταστάσεις μας διαθέτουν τη δική τους ενότητα ABS. Αυτή η ενότητα σάς επιτρέπει να δημιουργήσετε ευκαιρίες upsell στο κύριο απόθεμα ή να μετατρέψετε ένα μη συναλλακτικό είδος αποθέματος σε συναλλακτικό.
Σας συνιστούμε ανεπιφύλακτα να δημιουργήσετε αυτές τις ευκαιρίες για να συνδεθείτε ενδεχομένως με τον χρήστη που κάνει την κράτηση σε υψηλότερη τιμή κατά τη διαδικασία κράτησης.

:::συμβουλή
Ένα πρόσθετο μπορεί να διατεθεί προς πώληση μέσω του ABS. Μπορεί να διατεθεί προς πώληση με διάφορους τρόπους:

* Αργό αλλά ελεύθερο.

* Γρήγορο αλλά ακριβό.

* +++ Αφήστε τη φαντασία σας ελεύθερη και κερδίστε μερικά επιπλέον χρήματα!!
  :::

* **Τιμή**Δώστε στον ταξιδιώτη μια γρήγορη ένδειξη για το πόσο ακριβό είναι αυτό το πρόσθετο.

* **Ενεργός**Ενεργοποιήστε/απενεργοποιήστε τον διακόπτη σε 🛑 εάν θέλετε αυτό το πρόσθετο να μην είναι διαθέσιμο για κράτηση σε όλα τα κανάλια πωλήσεων.

* **Επιλέξιμος**Υποδείξτε εάν οι θυγατρικές εταιρείες εισπράττουν προμήθεια από αυτό το απόθεμα.

* **Προτεινόμενα**Υποδείξτε εάν αυτό το πρόσθετο θα πρέπει να είναι πιο εμφανές ως πρόσθετο, εάν παρουσιαστεί η ίδια η επιλογή.

* **Πώληση βάσει χαρακτηριστικών**Εναλλαγή διακόπτη στη θέση ✅ για να ενεργοποιήσετε το ABS σε αυτό το απόθεμα.*Μόλις ενεργοποιηθεί, πρέπει να προσθέσετε τουλάχιστον 1 αντικείμενο ABS προς πώληση*

### Βοηθητικό στοιχείο

Όταν ενεργοποιήσετε το ABS, μπορείτε να ξεκινήσετε να προσθέτετε βοηθητικά είδη και να τα διαθέτετε προς πώληση.

*Από προεπιλογή, έχει ήδη δημιουργηθεί ένα κενό στοιχείο για εσάς. Μπορείτε να προσθέσετε περισσότερα κάνοντας κλικ στο`Add item for sale`κουμπί.*

* **Ονομα**Δώστε ένα όνομα στον βοηθό σας.*π.χ. Αργό αλλά ελεύθερο*
* **Τύπος τιμολόγησης**Υποδείξτε πώς θέλετε να υπολογίσετε την τιμή.*π.χ. Ορισμός σε`Per person`για να υποδείξει ότι η τιμή βασίζεται στον αριθμό των ατόμων.*
* **Βασική τιμή**Εισαγάγετε την κανονική τιμή αυτού του είδους.*π.χ. 200 / άτομο*
* **Τιμή με έκπτωση**Εισαγάγετε την τιμή που λαμβάνουν οι χρήστες του Wink κατά την κράτηση του αντικειμένου.*π.χ. 150 / άτομο*

### Όρια

Μπορείτε να περιορίσετε τη διαθεσιμότητα αυτού του προϊόντος ορίζοντας ελάχιστο/μέγιστο όριο στην ποσότητα.

* **Ελάχιστη ποσότητα**Περιορίστε τον ελάχιστο αριθμό που πρέπει να κρατηθεί.*π.χ. 1 συμμετέχων*
* **Μέγιστη ποσότητα**Περιορίστε τον μέγιστο αριθμό που πρέπει να κρατηθεί.*π.χ. 4 συμμετέχοντες*

### Περιγραφές

:::σημείωση
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και να είναι στα Αγγλικά.
:::

* **Ονομα**Δώστε ένα όνομα στον βοηθό σας.*π.χ. Αργό αλλά ελεύθερο*
* **Περιγραφή**Περιγράψτε το βοηθητικό στοιχείο σε μία ή δύο παραγράφους.*π.χ. Χρησιμοποιήστε το πολύ παλιό μας ελικοδρόμιο που σας μεταφέρει εκεί.*

## Εγγύτητα

Κάντε κλικ στο`Proximity`καρτέλα για να συνεχίσετε.

Δεν βρίσκεται όλο το απόθεμα εντός των εγκαταστάσεων. Εάν έχετε απόθεμα που βρίσκεται εκτός των εγκαταστάσεων, ενημερώστε τον ταξιδιώτη εδώ και θα σας δοθεί η ευκαιρία να ορίσετε μια ξεχωριστή τοποθεσία στον χάρτη μαζί με προαιρετικά στοιχεία επικοινωνίας.

*Αυτό μπορεί να είναι χρήσιμο σε περιπτώσεις που κάποιο πρόσθετο κατάλυμα βρίσκεται λίγα χιλιόμετρα μακριά από το ξενοδοχείο και ο ταξιδιώτης θέλει να εξηγήσει στον οδηγό του ταξί πώς να φτάσει εκεί.*

Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.
