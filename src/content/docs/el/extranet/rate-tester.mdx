Ο δοκιμαστής τιμών σας επιτρέπει να επαληθεύσετε ότι οι τιμές των δωματίων σας έχουν ρυθμιστεί σωστά πριν γίνουν διαθέσιμες στους ταξιδιώτες.

Για να αποκτήσετε πρόσβαση στον δοκιμαστή τιμών, κάντε κλικ στο `Monetize > Rate Tester` από την κύρια γραμμή πλοήγησης.

## Πώς λειτουργεί

Εισάγετε ένα σενάριο αναζήτησης και ο δοκιμαστής τιμών θα σας δείξει ακριβώς πώς υπολογίζονται οι τιμές:

- **Κανάλι πωλήσεων** — Επιλέξτε το κανάλι για δοκιμή (π.χ., η μηχανή κρατήσεών σας, ένας συνεργάτης ή μια συγκεκριμένη ενσωμάτωση).
- **Ημερομηνίες check-in / check-out** — Οι ημερομηνίες ταξιδιού για δοκιμή.
- **Διαμόρφωση δωματίου** — Αριθμός ενηλίκων και παιδιών.

Κάντε κλικ στο `Search` για να δείτε τα αποτελέσματα.

## Αποτελέσματα

Η σελίδα αποτελεσμάτων εμφανίζει τις υπολογισμένες τιμές για κάθε τύπο δωματίου και τιμολογιακό πλάνο που ταιριάζει με τα κριτήρια αναζήτησής σας. Χρησιμοποιήστε το για να επαληθεύσετε ότι:

- Οι τροποποιητές τιμής (μονή κατοίκηση, επιπλέον ενήλικες/παιδιά) εφαρμόζονται σωστά.
- Οι πολιτικές ακύρωσης εμφανίζονται όπως αναμένεται.
- Οι περιορισμοί φιλτράρονται σωστά.
- Οι προσφορές και οι εκπτώσεις εφαρμόζονται.
