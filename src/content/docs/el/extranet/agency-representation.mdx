Ένα πρακτορείο είναι ένας κανονικός λογαριασμός Wink που μπορείτε να αναθέσετε στον λογαριασμό σας για σκοπούς αντιπροσώπευσης. Παρακάτω αναφέρονται μερικοί λόγοι για τους οποίους θα θέλατε ένα πρακτορείο να σας αντιπροσωπεύει:

- Πλήρης / Μερική διαχείριση του λογαριασμού σας στο Wink από τρίτο μέρος.
- Το ακίνητο έχει υπάρχουσα συμφωνία με την εταιρεία διαχείρισής του και θέλει το Wink να χειριστεί την οικονομική πτυχή της ρύθμισης των χρημάτων που εισέρχονται μέσω Wink.
- Η εταιρεία διαχείρισης ακινήτων αναλαμβάνει όλη την ηλεκτρονική εργασία εκ μέρους των ακινήτων της.

## Ανάθεση πρακτορείου

Ακολουθεί πώς αναθέτετε ένα πρακτορείο στον λογαριασμό σας.

<Steps>
 1. Συνδεθείτε στον λογαριασμό χρήστη σας στο [Wink Extranet](https://extranet.wink.travel).
 2. Επιλέξτε το ακίνητο με το οποίο θέλετε να συνεργαστείτε.
 3. Κάντε κλικ στο `Account > Managers` από τη βασική γραμμή πλοήγησης.
 4. Στην ενότητα `Agency`, πληκτρολογήστε το όνομα του πρακτορείου που θέλετε να σας αντιπροσωπεύει.
 5. Επιλέξτε την αμοιβή πρακτορείου από το αναπτυσσόμενο μενού.
 5. Προαιρετικά, προσθέστε κανόνες λήξης.
 7. Κάντε κλικ στο κουμπί `Set agency` για να συνεχίσετε.
 </Steps>

 :::note
Πρέπει να είστε ο ιδιοκτήτης του ακινήτου για να αναθέσετε πρακτορείο.
 :::

 :::tip[Παράδειγμα]
Η αμοιβή πρακτορείου προκύπτει από την κράτηση.

Κράτηση = $100, Αμοιβή πρακτορείου = 1% => 100 * 0.01 = $1
 :::

## Αφαίρεση ανάθεσης πρακτορείου

Ακολουθεί πώς αφαιρείτε την ανάθεση ενός πρακτορείου από τον λογαριασμό σας.

<Steps>
 1. Συνδεθείτε στον λογαριασμό χρήστη σας στο [Wink Extranet](https://extranet.wink.travel).
 2. Επιλέξτε το ακίνητο με το οποίο θέλετε να συνεργαστείτε.
 3. Κάντε κλικ στο `Account > Managers` από τη βασική γραμμή πλοήγησης.
 4. Στην ενότητα `Agency`, κάντε κλικ στο εικονίδιο <Badge text="x" variant="danger" size="small" /> στη δεξιά πλευρά του αναπτυσσόμενου μενού πρακτορείου.
 5. Κάντε κλικ στο κουμπί `Set agency` για να συνεχίσετε.
 </Steps>


## API

Οι προγραμματιστές που θέλουν να διαχειριστούν πρακτορείο μέσω API μπορούν να μεταβούν στο [Developers > API > Affiliate](/developers/apis/#affiliate-api).
