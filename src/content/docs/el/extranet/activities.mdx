Οι δραστηριότητες σας επιτρέπουν να προβάλλετε και να πουλάτε τις δραστηριότητές σας, εντός και εκτός των εγκαταστάσεων. Για να δημιουργήσετε μια δραστηριότητα, κάντε κλικ στο κουμπί `Create activity`.

Οι παρακάτω ενότητες σας καθοδηγούν σε κάθε μέρος της διαχείρισης μιας δραστηριότητας.

## Διαμόρφωση

Η καρτέλα `Configurations` είναι επιλεγμένη από προεπιλογή όταν ξεκινάτε.

Αυτή η ενότητα καλύπτει τα βασικά της δραστηριότητάς σας.

- **Εσωτερικό όνομα** Το όνομα της δραστηριότητας, όπως αναφέρεστε σε αυτήν εσωτερικά. *π.χ. Fly fishing*
- **Lifestyle** Προαιρετικά επιλέξτε ένα lifestyle που ταιριάζει σε αυτή τη δραστηριότητα. *π.χ. Sports*
- **Ηλικία** Προαιρετικά επιλέξτε την κατώτερη συνιστώμενη ηλικιακή ομάδα για αυτή τη δραστηριότητα. *π.χ. Adult*
- **Τύπος** Προαιρετικά επιλέξτε τον τύπο δραστηριότητας που ταιριάζει περισσότερο. *π.χ. Fishing*
- **Απαιτείται κράτηση** Αν αυτή η δραστηριότητα χρειάζεται κράτηση. *π.χ. Yes*

## Πρόγραμμα

Αυτή η ενότητα σας επιτρέπει να διαχειριστείτε πότε η δραστηριότητα είναι διαθέσιμη.

### Ώρες λειτουργίας

- **Άνοιγμα** Ώρα της ημέρας που η δραστηριότητα γίνεται διαθέσιμη. *π.χ. 09:00*
- **Κλείσιμο** Ώρα της ημέρας που η δραστηριότητα γίνεται μη διαθέσιμη. *π.χ. 21:00*

### Ημέρες λειτουργίας

Αυτή η ενότητα σας επιτρέπει να ορίσετε τις ημέρες της εβδομάδας που η δραστηριότητα είναι διαθέσιμη.

*Ενεργοποιήστε τον διακόπτη σε ✅ για να υποδείξετε ότι η ημέρα είναι διαθέσιμη.*

### Εποχιακό

:::note
Δεν υπάρχει όμορφο, εγγενές πεδίο ημέρας / μήνα. Σε αυτή την περίπτωση, χρησιμοποιούμε μόνο τον μήνα και την ημέρα από αυτόν τον επιλογέα ημερομηνίας.
:::

Αυτή η ενότητα σας επιτρέπει να ορίσετε τις ημερομηνίες που η δραστηριότητα είναι διαθέσιμη καθ’ όλη τη διάρκεια του έτους.

- **Ημερομηνία έναρξης** Ημερομηνία που η δραστηριότητα γίνεται διαθέσιμη. *π.χ. 11 Σεπτεμβρίου 2024*
- **Ημερομηνία λήξης** Ημερομηνία που η δραστηριότητα γίνεται μη διαθέσιμη. *π.χ. 11 Ιουλίου 2024*

## Παροχές

:::note
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στην καρτέλα `Amenities` για να συνεχίσετε.

Παρόμοια με τις παροχές του καταλύματος, οι παροχές της δραστηριότητας ισχύουν μόνο για τη συγκεκριμένη δραστηριότητα. Επιλέξτε κάθε παροχή που ισχύει.

## Περιγραφές

:::note
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και αυτή πρέπει να είναι στην αγγλική γλώσσα.
:::

Κάντε κλικ στην καρτέλα `Descriptions` για να συνεχίσετε.

Παρόμοια με το κείμενο καλωσορίσματος του καταλύματος, μπορείτε να δημιουργήσετε τοπικές περιγραφές για τη δραστηριότητα.
Μπορείτε να γράψετε σε όσες γλώσσες θέλετε. Θα προσθέσουμε μεταφράσεις για όλες τις πιο δημοφιλείς γλώσσες.
Αυτό είναι το κείμενο που θα βλέπουν οι ταξιδιώτες όταν προβάλλουν τις δραστηριότητές σας.

- **Όνομα** Δώστε ένα όνομα στη δραστηριότητα. *π.χ. Fly fishing*
- **Περιγραφή** Περιγράψτε τη δραστηριότητα σε μία ή δύο παραγράφους. *π.χ. Μόνο εσείς... το δάσος... το ποτάμι... Ηρεμία σε ένα μπουκάλι.*

## Κοινωνικά δίκτυα

:::note
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στην καρτέλα `Social media` για να συνεχίσετε.

Ίσως αναρωτιέστε, "Γιατί μια δραστηριότητα έχει τη δική της ενότητα κοινωνικών μέσων;". Μια δραστηριότητα μπορεί επίσης να είναι μια εγκατάσταση κοντά στο κατάλυμα αλλά εκτός των εγκαταστάσεων με δικό της προσωπικό και δικό της προφίλ στο IG.

Τα κοινωνικά μέσα σας κρατούν σε επαφή με νέους και υπάρχοντες ταξιδιώτες με την πάροδο του χρόνου.
Προσθέστε τους λογαριασμούς των κοινωνικών δικτύων σας στο προφίλ σας ώστε οι επισκέπτες να μπορούν εύκολα να επικοινωνήσουν μαζί σας και να μάθουν για αυτή τη δραστηριότητα.

<Steps>
1. Ενεργοποιήστε τον διακόπτη ενός κοινωνικού δικτύου που θέλετε να ενεργοποιήσετε.
2. Εισάγετε το όνομα του λογαριασμού σας σε αυτό το δίκτυο.
</Steps>

## Φήμη

:::note
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στην καρτέλα `Reputation` για να συνεχίσετε.

Παρόμοια με την καρτέλα `Social media`, θα χρησιμοποιήσετε αυτή την καρτέλα αν αυτή η δραστηριότητα έχει ξεχωριστή βαθμολογία φήμης ή επιπλέον διακρίσεις που θέλετε να αναδείξετε εδώ.

Η φήμη σας είναι το ψηφιακό σας νόμισμα και θέλετε να μπορείτε να την παίρνετε μαζί σας όπου κι αν πάτε.
Μια καταχώρηση φήμης μπορεί να είναι οτιδήποτε θέλει το κατάλυμά σας να αναγνωριστεί, όπως μια αξιολόγηση από τρίτο μέρος ή για τον διαγωνισμό όπου ο σεφ σας κέρδισε χρυσό.
Σε αυτή την ενότητα, μπορείτε να προσθέσετε τις βαθμολογίες και τις κατατάξεις σας από όλο το Διαδίκτυο.

Κάντε κλικ στο κουμπί `Add reputation score`.

<Steps>
1. **Κατηγορία** Επιλέξτε τον τύπο βαθμολογίας που θέλετε να προσθέσετε. *π.χ. Third party review*
2. **Πάροχος** Πληκτρολογήστε ποιος σας έδωσε τη βαθμολογία. *π.χ. BigOTA.com*
3. **Τύπος αξιολόγησης** Επιλέξτε τη μορφή με την οποία λάβατε τη βαθμολογία. *π.χ. Numeric*
4. **Ημερομηνία** Προαιρετικά, εισάγετε την ημερομηνία που λάβατε τη βαθμολογία.
5. **Βαθμολογία** Εισάγετε τη βαθμολογία που λάβατε. *π.χ. 8.5*
6. **Μέγιστη βαθμολογία** Προαιρετικά, εισάγετε τη μέγιστη δυνατή βαθμολογία για αυτόν τον τύπο βράβευσης. *π.χ. 10*
</Steps>

## Πώληση βάσει χαρακτηριστικών

:::note
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στην καρτέλα `Attribute Based Selling` για να συνεχίσετε.

Όλες οι εγκαταστάσεις μας διαθέτουν τη δική τους ενότητα ABS. Αυτή η ενότητα σας επιτρέπει να δημιουργήσετε ευκαιρίες upsell στο κύριο απόθεμα ή να κάνετε έναν μη-συναλλακτικό τύπο αποθέματος συναλλακτικό.
Σας προτείνουμε έντονα να δημιουργήσετε αυτές τις ευκαιρίες για να συνδεθείτε πιθανώς με τον πελάτη σε υψηλότερη τιμή κατά τη διαδικασία κράτησης.

:::tip
Μια δραστηριότητα μπορεί να διατεθεί προς πώληση μέσω ABS. Μπορεί να διατεθεί προς πώληση με διάφορους τρόπους:

- Ημερήσια εκδρομή.
- Εκδρομή 3 ημερών.
- +++ αφήστε τη φαντασία σας να τρέξει και κερδίστε επιπλέον χρήματα!!
:::

- **Τιμή** Δώστε στον ταξιδιώτη μια γρήγορη ένδειξη για το πόσο ακριβή είναι αυτή η δραστηριότητα.
- **Ενεργό** Απενεργοποιήστε τον διακόπτη σε 🛑 αν θέλετε αυτή η δραστηριότητα να γίνει μη διαθέσιμη για κράτηση σε όλα τα κανάλια πώλησης.
- **Προμήθεια** Υποδείξτε αν οι συνεργάτες λαμβάνουν προμήθεια για αυτό το απόθεμα.
- **Προτεινόμενο** Υποδείξτε αν αυτή η δραστηριότητα πρέπει να τοποθετηθεί πιο εμφανώς αν παρουσιαστεί η επιλογή.
- **Πώληση βάσει χαρακτηριστικών** Ενεργοποιήστε τον διακόπτη σε ✅ για να ενεργοποιήσετε το ABS σε αυτό το απόθεμα. *Μόλις ενεργοποιηθεί, πρέπει να προσθέσετε τουλάχιστον 1 αντικείμενο ABS προς πώληση*

### Συμπληρωματικό αντικείμενο

Όταν ενεργοποιήσετε το ABS, μπορείτε να αρχίσετε να προσθέτετε συμπληρωματικά αντικείμενα και να τα κάνετε διαθέσιμα προς πώληση.

*Από προεπιλογή, έχει ήδη δημιουργηθεί ένα κενό αντικείμενο για εσάς. Μπορείτε να προσθέσετε περισσότερα κάνοντας κλικ στο κουμπί `Add item for sale`.*

- **Όνομα** Δώστε ένα όνομα στο συμπληρωματικό σας αντικείμενο. *π.χ. Day trip*
- **Τύπος τιμολόγησης** Υποδείξτε πώς θέλετε να υπολογίζεται η τιμή. *π.χ. Ορίστε σε `Per day` για να υποδείξετε ότι η τιμή βασίζεται στον αριθμό ημερών.*
- **Βασική τιμή** Εισάγετε την κανονική τιμή αυτού του αντικειμένου. *π.χ. 200 / ημέρα*
- **Εκπτωτική τιμή** Εισάγετε την τιμή που λαμβάνουν οι χρήστες Wink κατά την κράτηση του αντικειμένου. *π.χ. 150 / ημέρα*

### Όρια

Μπορείτε να περιορίσετε τη διαθεσιμότητα αυτού του αντικειμένου ορίζοντας ελάχιστα / μέγιστα όρια στην ποσότητα.

- **Ελάχιστη ποσότητα** Περιορίστε την ελάχιστη ποσότητα που πρέπει να γίνει κράτηση. *π.χ. 1 συμμετέχων*
- **Μέγιστη ποσότητα** Περιορίστε τη μέγιστη ποσότητα που πρέπει να γίνει κράτηση. *π.χ. 4 συμμετέχοντες*

### Περιγραφές

:::note
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και αυτή πρέπει να είναι στην αγγλική γλώσσα.
:::

- **Όνομα** Δώστε ένα όνομα στο συμπληρωματικό σας αντικείμενο. *π.χ. Day trip*
- **Περιγραφή** Περιγράψτε το συμπληρωματικό αντικείμενο σε μία ή δύο παραγράφους. *π.χ. Πάρτε μια μέρα άδεια και κατευθυνθείτε προς το ποτάμι.*

## Εγγύτητα

Κάντε κλικ στην καρτέλα `Proximity` για να συνεχίσετε.

Δεν βρίσκεται όλο το απόθεμα εντός των εγκαταστάσεων. Αν έχετε απόθεμα που βρίσκεται εκτός των εγκαταστάσεων, ενημερώστε τον ταξιδιώτη εδώ και θα σας δοθεί η δυνατότητα να ορίσετε ξεχωριστή τοποθεσία στο χάρτη μαζί με προαιρετικές πληροφορίες επικοινωνίας.

*Αυτό μπορεί να είναι χρήσιμο όταν μια δραστηριότητα βρίσκεται λίγα χιλιόμετρα μακριά από το ξενοδοχείο και ο ταξιδιώτης θέλει να εξηγήσει στον οδηγό ταξί πώς να φτάσει εκεί.*

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν τις `Activities` μπορούν να μεταβούν στο [Developers > APIs > Experiences](/el/developers/apis/#experiences-api).
