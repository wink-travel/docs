Κάθε διαδικασία κράτησης ξεκινά με ένα δωμάτιο επισκεπτών. Υπάρχουν πολλά χαρακτηριστικά γύρω από τα δωμάτια επισκεπτών και μπορεί να είναι λίγο κουραστικό στην αρχή.
Αυτό το άρθρο θα καλύψει όλα όσα πρέπει να γνωρίζετε για να γίνετε ειδικός σε δωμάτια επισκεπτών της Wink.

Δωμάτια επισκεπτών ή τύποι δωματίων όπως λένε`the biz`, σας επιτρέπει να προβάλλετε και να πουλάτε τα δωμάτιά σας, εντός και εκτός των εγκαταστάσεων. Για να δημιουργήσετε ένα δωμάτιο επισκεπτών, κάντε κλικ στο`Create a new guest room`κουμπί.

Οι ακόλουθες ενότητες σας καθοδηγούν σε κάθε στάδιο της διαχείρισης ενός δωματίου επισκεπτών.

## Διαμόρφωση

Το`Configurations`Η καρτέλα είναι επιλεγμένη από προεπιλογή όταν ξεκινάτε.

Αυτή η ενότητα καλύπτει τα βασικά στοιχεία του δωματίου σας, ώστε να γνωρίζουμε σε ποιον μπορούμε να το πουλήσουμε.

* **Όνομα τύπου δωματίου**Το όνομα του δωματίου, όπως το αναφέρετε εσωτερικά.*π.χ. Supreme deluxe*
* **Μη καπνίζοντες**Είτε το δωμάτιο είναι μη καπνιζόντων είτε όχι.
* **Τιμή ραφιού**Η δημοσιευμένη τιμή για μία διανυκτέρευση χωρίς εκπτώσεις ή ασφάλιστρα.*π.χ. 100. Σημειώστε ότι χρησιμοποιούμε αυτόν τον αριθμό μόνο για να εμφανίσουμε την αναμενόμενη τιμή στα κανάλια πωλήσεων ή για να προσυμπληρώσουμε ορισμένα πεδία όταν συμπληρώνετε φόρμες.*
* **Καλύτερη διαθέσιμη τιμή**Η καλύτερη διαθέσιμη τιμή για μία διανυκτέρευση χωρίς εκπτώσεις ή ασφάλιστρα.*π.χ. 75. Σημειώστε ότι χρησιμοποιούμε αυτόν τον αριθμό μόνο για να εμφανίσουμε την αναμενόμενη τιμή στα κανάλια πωλήσεων ή για να προσυμπληρώσουμε ορισμένα πεδία όταν συμπληρώνετε φόρμες.*
* **Ποσότητα**Ο αριθμός των μονάδων που έχετε από αυτόν τον τύπο δωματίου σε οποιαδήποτε δεδομένη ημέρα.*π.χ. 40*

### Κατοχή

:::προσοχή
Η πληρότητα παίζει μεγάλο ρόλο στο αν το δωμάτιό σας εμφανίζεται ως διαθέσιμο για ένα συγκεκριμένο δρομολόγιο. Πολλά από αυτά τα πεδία θα συμπληρωθούν αυτόματα όταν δημιουργήσετε για πρώτη φορά το δωμάτιο. Βεβαιωθείτε ότι τα τροποποιείτε ώστε να ταιριάζουν στον τύπο του δωματίου σας.
:::

* **Ελάχιστη πληρότητα**Ο ελάχιστος αριθμός ατόμων που επιτρέπεται να διαμείνουν σε αυτό το δωμάτιο.*π.χ. 1*
* **Συμπεριλαμβάνεται διαμονή ενηλίκων**Αριθμός ενηλίκων που περιλαμβάνεται στην τιμή ενός διανυκτερεύοντος δωματίου.*π.χ. 2. Σημείωση: Εάν το δωμάτιο υποστηρίζει περισσότερους από αυτόν τον αριθμό, υπολογίζονται χρησιμοποιώντας το`Extra adult`τροποποιητής στο πρόγραμμα τιμολόγησής σας.*
* **Μέγιστη πληρότητα**Ο μέγιστος αριθμός ατόμων (ενήλικες + παιδιά) που επιτρέπεται να διαμείνουν σε αυτό το δωμάτιο.*π.χ. 4*
* **Μέγιστη χωρητικότητα ενηλίκων**Ο μέγιστος αριθμός ενηλίκων που επιτρέπεται να διαμείνουν σε αυτό το δωμάτιο.*π.χ. 3*
* **Συμπεριλαμβάνεται η διαμονή παιδιών**Αριθμός παιδιών που περιλαμβάνεται στην τιμή ενός διανυκτερεύοντος δωματίου.*π.χ. 1. Σημείωση: Εάν το δωμάτιο υποστηρίζει περισσότερους από αυτόν τον αριθμό, υπολογίζονται χρησιμοποιώντας το`Extra child`τροποποιητής στο πρόγραμμα τιμολόγησής σας.*
* **Μέγιστη χωρητικότητα παιδιών**Ο μέγιστος αριθμός παιδιών που επιτρέπεται να διαμείνουν σε αυτό το δωμάτιο.*π.χ. 2*

### Αλλος

Αυτή η ενότητα προορίζεται μόνο για ενημερωτικούς σκοπούς. Οι ταξιδιώτες που αναζητούν μεγαλύτερα δωμάτια και σουίτες θα βρουν ενδιαφέρουσες αυτές τις πληροφορίες.

* **Αριθμός μπάνιων**Ο αριθμός των μπάνιων για αυτόν τον τύπο δωματίου.*π.χ. 1*
* **Αριθμός σαλονιών**Ο αριθμός των καθιστικών για αυτόν τον τύπο δωματίου.*π.χ. 1*
* **Μέγιστο rollaways**Ο αριθμός των πτυσσόμενων κρεβατιών που υποστηρίζονται από αυτόν τον τύπο δωματίου.*π.χ. 1*
* **Max κούνιες**Ο αριθμός των βρεφικών κρεβατιών που υποστηρίζονται από αυτόν τον τύπο δωματίου.*π.χ. 1*

### Ταξινόμηση δωματίου

* **Τμήμα**Το τμήμα στο οποίο κατηγοριοποιείται το δωμάτιο επισκεπτών.*π.χ. Προϋπολογισμός*
* **Τοποθεσία**Ο τύπος τοποθεσίας στον οποίο κατηγοριοποιείται το δωμάτιο επισκεπτών.*π.χ. Ψηλός όροφος*
* **Θέα**Η θέα που αναμένεται από αυτό το δωμάτιο.*π.χ. Θέα στην πόλη*
* **Ταξινόμηση**Η κατηγορία στην οποία υπάγεται αυτό το δωμάτιο.*π.χ. Οικονομική*
* **Αρχιτεκτονική**Το στυλ κτιρίου γύρω από αυτό το δωμάτιο επισκεπτών.*π.χ. Ασιάτης*
* **Πάτωμα**Ένας/Οι προαιρετικός/οί όροφος/οι όπου βρίσκεται αυτό το δωμάτιο επισκεπτών*π.χ. 20ός όροφος και άνω*

### Σύνθεση

Η σύνθεση ισχύει για εσάς μόνο εάν το δωμάτιο αποτελείται από πολλά δωμάτια επισκεπτών.

* **Σύνθεση**Εναλλαγή διακόπτη σε ✅ εάν το δωμάτιο επισκεπτών αποτελείται από πολλά δωμάτια.*π.χ. Ναι*
* **Κόμης**Υποδείξτε από πόσα δωμάτια αποτελείται αυτό το δωμάτιο.*π.χ. 2*

### Λεπτομέρειες ξενώνα

Αυτή η ενότητα ισχύει μόνο εάν διαχειρίζεστε κοιτώνα, ξενώνα ή άλλη υπηρεσία κοινόχρηστου δωματίου.

* **Κοινόχρηστο δωμάτιο**Εναλλαγή διακόπτη σε ✅ εάν το δωμάτιο επισκεπτών είναι κοινόχρηστο.*π.χ. Ναι*
* **Γένος**Υποδείξτε το φύλο για αυτό το δωμάτιο επισκεπτών.*π.χ. Άνδρας*

## Υπνοδωμάτια

:::σημείωση
Υπάρχει ένας άπειρος αριθμός πιθανών διαρρυθμίσεων υπνοδωματίων. Προσπαθήσαμε να το κάνουμε όσο το δυνατόν πιο προσαρμόσιμο ώστε να ταιριάζει στο ακίνητό σας.
:::

Κάντε κλικ στο`Bedrooms`καρτέλα για να συνεχίσετε.

Η ενότητα υπνοδωματίων σάς επιτρέπει να καθορίσετε τη διάταξη του/των υπνοδωματίου/ων σας και να υποδείξετε τι υπάρχει σε κάθε ένα.

Τα δωμάτια επισκεπτών του ίδιου τύπου μπορούν να έχουν πολλαπλές διαρρυθμίσεις ή`Bedding configurations`Ένα δωμάτιο επισκεπτών χρειάζεται τουλάχιστον μία διαρρύθμιση.

* **Όνομα διαμόρφωσης**Ονομάστε τη διάταξη με κάτι που θα καταλάβετε*π.χ. Κύριο*

Μπορείτε να έχετε πολλά υπνοδωμάτια σε μια διάταξη.

* **Τύπος υπνοδωματίου**Κάθε υπνοδωμάτιο διαθέτει έναν τύπο.*Το πρώτο είναι`Master`από προεπιλογή, αλλά μπορείτε να το αλλάξετε στη διαμόρφωσή σας.*

Μέσα σε ένα υπνοδωμάτιο, μπορείτε να έχετε πολλά κρεβάτια διαφορετικών τύπων και ποσοτήτων.

* **Τύπος κρεβατιού**Επιλέξτε τον τύπο κρεβατιού στην κρεβατοκάμαρά σας.*π.χ. Βασιλιάς*
* **Ποσότητα**Επιλέξτε τον αριθμό των κρεβατιών αυτού του τύπου σε αυτό το υπνοδωμάτιο.*π.χ. 1*

## Κοινωνικά μέσα

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Social media`καρτέλα για να συνεχίσετε.

Ίσως αναρωτιέστε: «Γιατί ένα δωμάτιο επισκεπτών έχει τη δική του ενότητα στα μέσα κοινωνικής δικτύωσης;». Ένα δωμάτιο επισκεπτών μπορεί επίσης να είναι μια βίλα κοντά στο κατάλυμα αλλά εκτός του χώρου, με το δικό της προσωπικό και το δικό του προφίλ στο IG.

Τα μέσα κοινωνικής δικτύωσης σας κρατούν σε επαφή με νέους και υπάρχοντες ταξιδιώτες με την πάροδο του χρόνου.
Προσθέστε τους λογαριασμούς σας στα μέσα κοινωνικής δικτύωσης στο προφίλ σας, ώστε οι επισκέπτες να μπορούν εύκολα να επικοινωνήσουν μαζί σας και να μάθουν περισσότερα για το δωμάτιό σας.

<Steps>
  1. Ενεργοποιήστε/ενεργοποιήστε το διακόπτη ενός κοινωνικού δικτύου που θέλετε να ενεργοποιήσετε.
  2. Εισαγάγετε το όνομα του λογαριασμού σας σε αυτό το δίκτυο.
</Steps>


## Φήμη

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Reputation`καρτέλα για να συνεχίσετε.

Παρόμοια με το`Social media`καρτέλα, θα χρησιμοποιούσατε αυτήν την καρτέλα εάν αυτό το δωμάτιο επισκεπτών έχει ξεχωριστή βαθμολογία/βαθμολογίες φήμης ή επιπλέον διακρίσεις που θέλετε να επισημάνετε εδώ.

Η φήμη σας είναι το ψηφιακό σας νόμισμα και θέλετε να μπορείτε να την παίρνετε μαζί σας όπου κι αν πάτε.

Μια καταχώριση φήμης μπορεί να είναι οτιδήποτε θέλει να πιστωθεί το κατάλυμά σας, όπως μια αξιολόγηση τρίτου μέρους ή για το bake-off όπου ο σεφ σας κέρδισε το χρυσό.
Σε αυτήν την ενότητα, μπορείτε να προσθέσετε τις βαθμολογίες και τις κατατάξεις σας από όλο το Διαδίκτυο.

Κάντε κλικ στο`Add reputation score`κουμπί.

<Steps>
  1. **Κατηγορία** Επιλέξτε τον τύπο βαθμολογίας που θέλετε να προσθέσετε. *π.χ. αξιολόγηση τρίτου μέρους*
  2. **Προμηθευτής** Πληκτρολογήστε ποιος σας έδωσε τη βαθμολογία. *π.χ. BigOTA.com*
  3. **Τύπος αξιολόγησης** Επιλέξτε τη μορφή με την οποία λάβατε τη βαθμολογία. *π.χ. Αριθμητικό*
  4. **Ημερομηνία** Προαιρετικά, εισαγάγετε την ημερομηνία που λάβατε τη βαθμολογία.
  5. **Εκτίμηση** Εισαγάγετε τη βαθμολογία που λάβατε. *π.χ. 8,5*
  6. **Μέγιστη βαθμολογία** Προαιρετικά, εισαγάγετε τη μέγιστη δυνατή βαθμολογία για αυτόν τον τύπο βραβείου. *π.χ. 10*
</Steps>


## Πώληση βάσει χαρακτηριστικών

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Attribute Based Selling`καρτέλα για να συνεχίσετε.

Όλες οι εγκαταστάσεις μας διαθέτουν τη δική τους ενότητα ABS. Αυτή η ενότητα σάς επιτρέπει να δημιουργήσετε ευκαιρίες upsell στο κύριο απόθεμα ή να μετατρέψετε ένα μη συναλλακτικό είδος αποθέματος σε συναλλακτική μορφή.
Σας συνιστούμε ανεπιφύλακτα να δημιουργήσετε αυτές τις ευκαιρίες για να συνδεθείτε ενδεχομένως με τον χρήστη που κάνει την κράτηση σε υψηλότερη τιμή κατά τη διαδικασία κράτησης.

:::συμβουλή
Ένα δωμάτιο επισκεπτών διαθέτει πολλά πράγματα που μπορείτε να συμπεριλάβετε ως βοηθητικά:

* Στρώμα ανώτερης ποιότητας.

* Πολυτελές κιτ μπάνιου.

* Ηχείο Bluetooth Marshal.

* Γραφείο εργασίας με επιπλέον άνετη καρέκλα.

* Πρωινό στο δωμάτιο

* Μασάζ στο δωμάτιο

* Συσκευασία 12 τεμαχίων μπύρας

* Μπουκάλι κρασί

* Σοκολάτα υψηλής ποιότητας

* +++ Αφήστε τη φαντασία σας ελεύθερη και κερδίστε μερικά επιπλέον χρήματα!!
  :::

* **Τιμή**Δώστε στον ταξιδιώτη μια γρήγορη ένδειξη για το πόσο ακριβό είναι αυτό το δωμάτιο. Σωστά... για το δωμάτιο. Όχι για το βοηθητικό.

* **Ενεργός**Ενεργοποιήστε/απενεργοποιήστε τον διακόπτη σε 🛑 εάν θέλετε αυτό το δωμάτιο να μην είναι διαθέσιμο για κράτηση σε όλες τις κύριες τιμές και τα κανάλια πωλήσεων.

* **Επιλέξιμος**Υποδείξτε εάν οι θυγατρικές εταιρείες εισπράττουν προμήθεια από αυτό το απόθεμα.

* **Προτεινόμενα**Υποδείξτε εάν αυτό το δωμάτιο επισκεπτών θα πρέπει να τοποθετηθεί σε πιο εμφανή θέση, εάν παρουσιαστεί η δυνατότητα.

* **Πώληση βάσει χαρακτηριστικών**Εναλλαγή διακόπτη στη θέση ✅ για να ενεργοποιήσετε το ABS σε αυτό το απόθεμα.*Μόλις ενεργοποιηθεί, πρέπει να προσθέσετε τουλάχιστον 1 αντικείμενο ABS προς πώληση*

### Βοηθητικό στοιχείο

Όταν ενεργοποιήσετε το ABS, μπορείτε να ξεκινήσετε να προσθέτετε βοηθητικά είδη και να τα διαθέσετε προς πώληση.

*Από προεπιλογή, έχει ήδη δημιουργηθεί ένα κενό στοιχείο για εσάς. Μπορείτε να προσθέσετε περισσότερα κάνοντας κλικ στο`Add item for sale`κουμπί.*

* **Ονομα**Δώστε ένα όνομα στον βοηθό σας.*π.χ. τρούφες σοκολάτας με σαμπάνια από την Royce*
* **Τύπος τιμολόγησης**Υποδείξτε πώς θέλετε να υπολογίσετε την τιμή.*π.χ. Ενοικίαση σανίδας paddleboard. Ορίστε σε`Per hour`για να υποδείξει ότι η τιμή βασίζεται στον αριθμό των ωρών ενοικίασης.*
* **Βασική τιμή**Εισαγάγετε την κανονική τιμή αυτού του είδους.*π.χ. 20 / ώρα*
* **Τιμή με έκπτωση**Εισαγάγετε την τιμή που λαμβάνουν οι χρήστες του Wink όταν κάνουν κράτηση του αντικειμένου μαζί με το δωμάτιο.*π.χ. 15 / ώρα*

### Όρια

Μπορείτε να περιορίσετε τη διαθεσιμότητα αυτού του προϊόντος ορίζοντας ελάχιστο/μέγιστο όριο στην ποσότητα.
Αν συνεχίσουμε το παράδειγμά μας ενοικίασης σανίδας paddle με την ώρα, αλλά θέλουμε να απαιτήσουμε από τον επισκέπτη να κάνει κράτηση τουλάχιστον 2 σανίδων ταυτόχρονα, θα:

* **Ελάχιστη ποσότητα**Περιορίστε τον ελάχιστο αριθμό που πρέπει να κρατηθεί.*π.χ. 2 σανίδες ρακέτας*
* **Μέγιστη ποσότητα**Περιορίστε τον μέγιστο αριθμό που πρέπει να κρατηθεί.*π.χ. αφήστε κενό για μηδενικό μέγιστο περιορισμό*

## Εγγύτητα

Κάντε κλικ στο`Proximity`καρτέλα για να συνεχίσετε.

Δεν βρίσκεται όλο το απόθεμα εντός των εγκαταστάσεων. Εάν έχετε απόθεμα που βρίσκεται εκτός των εγκαταστάσεων, ενημερώστε τον ταξιδιώτη εδώ και θα σας δοθεί η ευκαιρία να ορίσετε μια ξεχωριστή τοποθεσία στον χάρτη μαζί με προαιρετικά στοιχεία επικοινωνίας.

*Αυτό μπορεί να είναι χρήσιμο σε ορισμένες περιπτώσεις, όταν πρόκειται για μια βίλα που ανήκει στο ξενοδοχείο, αλλά βρίσκεται λίγα χιλιόμετρα μακριά από το ξενοδοχείο και ο ταξιδιώτης θέλει να εξηγήσει στον οδηγό του ταξί πώς να φτάσει εκεί.*

Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Guest Rooms`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Εγκαταστάσεις](/developers/apis/#facilities-api).
