Θα σας ζητηθεί να συσχετίσετε κάποιους τρόπους ζωής με το κατάλυμά σας κατά τη ρύθμιση.

Για να διαχειριστείτε τους τρόπους ζωής, κάντε κλικ στο `Property > Lifestyles` στην κύρια καρτέλα πλοήγησης.

Ενεργοποιήστε τον διακόπτη δίπλα στους τρόπους ζωής που θέλετε να χρησιμοποιήσετε.

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

:::caution[Απαιτήσεις]
Μπορείτε να επιλέξετε έως και 6 τρόπους ζωής. Επιλέξτε αυτούς που ταιριάζουν περισσότερο με το κατάλυμά σας.
:::
