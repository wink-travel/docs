Σε αυτή τη σελίδα, θα μάθετε όλα όσα χρειάζεται να ξέρετε για το πώς να δημιουργείτε και να διαχειρίζεστε ακίνητα στο Wink.

:::note
Αυτό προϋποθέτει ότι έχετε ήδη δημιουργήσει λογαριασμό ακινήτου στο [my.wink.travel](https://my.wink.travel).
:::

Επιλέξτε το ακίνητό σας από το πλέγμα `Accounts` και από το `Actions` επιλέξτε `Go to Extranet`. Αυτό θα σας μεταφέρει στον πίνακα ελέγχου Extranet με το ακίνητό σας επιλεγμένο.

## Wizard

Αν υπάρχουν απαιτούμενες ενότητες που δεν έχουν ακόμα ολοκληρωθεί, το `Wizard` θα εμφανιστεί στην αριστερή πλευρά και θα σας ενημερώσει ποιο/α βήμα/βήματα πρέπει να ολοκληρώσετε.

## Dashboard

Μόλις δημιουργηθεί το ακίνητό σας, μπορείτε να ολοκληρώσετε τη δημιουργία του προφίλ σας ή να ξεκινήσετε τη διαχείριση του ακινήτου κάνοντας κλικ στον σύνδεσμο `Manage` στη λίστα των ακινήτων σας.
Αυτό σας μεταφέρει στον πίνακα ελέγχου του ακινήτου.

Ο πίνακας ελέγχου είναι η βάση για το ακίνητό σας. Αποτελείται από ενότητες που σας βοηθούν να πλοηγηθείτε και να αξιοποιήσετε στο έπακρο τα εργαλεία μας:

### Πώς τα πάω;

Αυτή η ενότητα σας δίνει μια γρήγορη ματιά στην κατάσταση του ακινήτου σας. Σας ενημερώνει για:

- Συνολικό αριθμό κρατήσεων
- Μέση αξία κράτησης
- Συνολικό ποσό πωλήσεων

Αυτοί οι αριθμοί περιλαμβάνουν όλες τις κρατήσεις που έγιναν από οποιοδήποτε κανάλι πωλήσεων στο δίκτυο Wink.

### Αιτήματα δικτύου Wink

Μπορείτε να βρείτε εκκρεμή αιτήματα από συνεργάτες που θέλουν να συνδεθούν μαζί σας σε αυτή την ενότητα.
Υπάρχει επίσης ένας σύνδεσμος που σας μεταφέρει απευθείας στο `Distribution > Explore network` ώστε να διαχειριστείτε όλα τα τρέχοντα και εκκρεμή κανάλια πωλήσεών σας.

### Εργασίες

Η ενότητα εργασιών σας δείχνει προαιρετικά, αλλά συνιστώμενα, πράγματα που μπορείτε να κάνετε για να κάνετε το προφίλ του ακινήτου σας να ξεχωρίσει περισσότερο.

*Συνιστούμε να προσθέσετε τους βαθμούς διαδικτυακής φήμης σας από οποιαδήποτε εξωτερική ιστοσελίδα όπου έχετε αξιολόγηση, όπως ενημερωμένες βαθμολογίες από μεγάλες OTA.*

### Analytics

Ένα νέο ακίνητο λαμβάνει αυτόματα τα βασικά του analytics κατά τη δημιουργία. Από προεπιλογή, θα δείτε:

- Καθαρά έσοδα για τους τελευταίους 12 μήνες.
- Μέση αξία κράτησης για τους τελευταίους 12 μήνες.
- Συνολικές προβολές σελίδας για τους τελευταίους 12 μήνες.
- Συνολικές κρατήσεις για τους τελευταίους 12 μήνες.
- Συνολικές ακυρώσεις για τους τελευταίους 12 μήνες.

Αυτά τα γραφήματα μπορούν εύκολα να τροποποιηθούν. Ωστόσο, αυτά τα γραφήματα περιέχουν όλα τα analytics που θα χρειαστεί το ακίνητό σας για ένα διάστημα.
