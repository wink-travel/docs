## Διαμόρφωση

Αυτή η ενότητα είναι υπεύθυνη για τον καθορισμό της πραγματικής χρηματικής αξίας της προωθητικής ενέργειας.

* **Ονομα**Εισαγάγετε ένα όνομα για αυτήν την προσφορά που μόνο εσείς μπορείτε να δείτε.
* **Τύπος**Επιλέξτε αν θέλετε αυτή η προσφορά να περιλαμβάνει έκπτωση ή premium.
* **Τύπος τιμολόγησης**Επιλέξτε πώς θέλετε να υπολογιστεί αυτό το πακέτο.
  * `Per stay`: Το ποσό εφαρμόζεται στο χθεσινό βράδυ.*π.χ. 100% έκπτωση > Μείνετε 2 διανυκτερεύσεις - Πάρτε την 3η διανυκτέρευση δωρεάν.*
  * `Per night`: Ποσό πολλαπλασιασμένο με τον αριθμό των διανυκτερεύσεων.
  * `Per use`Δεν πραγματοποιείται επιπλέον υπολογισμός.
  * `Per person`: Ποσό πολλαπλασιασμένο επί τον αριθμό των επισκεπτών.
  * `Per person per night`: Ποσό πολλαπλασιασμένο επί τον αριθμό των διανυκτερεύσεων + τον αριθμό των επισκεπτών.
* **Τύπος ποσού**Επιλέξτε αν θέλετε ο τροποποιητής ποσού να είναι ένα σταθερό ποσό ή ένα ποσοστό.
* **Κατάσταση**Ενεργοποιήστε το διακόπτη για να ελέγξετε εάν η προσφορά είναι διαθέσιμη σε όλα τα κανάλια πωλήσεων.

## Περιγραφές

:::σημείωση
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και να είναι στα Αγγλικά.
:::

Κάντε κλικ στο`Descriptions`καρτέλα για να συνεχίσετε.

Μια μεταφρασμένη περιγραφή περιγράφει τον σκοπό αυτής της προσφοράς στον ταξιδιώτη στη γλώσσα του.

Μπορείτε να γράψετε σε όσες γλώσσες θέλετε. Θα προσθέσουμε μεταφράσεις για όλες τις πιο δημοφιλείς γλώσσες.
Αυτό είναι το κείμενο που θα βλέπουν οι ταξιδιώτες όταν βλέπουν την προώθησή σας.

* **Περιγραφή**Δώστε ένα όνομα στην προώθηση.*π.χ. Early bird - 20% ΕΚΠΤΩΣΗ*

## Περιορισμοί

:::σημείωση
Λάβετε υπόψη ότι αυτοί είναι περιορισμοί... Ενεργοποιήστε τους μόνο εάν θέλετε να κάνετε την προώθησή σας λιγότερο προσβάσιμη.
:::

Κάντε κλικ στο`Restrictions`καρτέλα για να συνεχίσετε.

Η μηχανή προώθησης του Wink διαθέτει, όχι λιγότερο από, 20 μοναδικούς τρόπους για να προκαλέσει μια αλλαγή τιμής.

Υποστηρίζουμε τα ακόλουθα κριτήρια:

* Περιορίστε την προσφορά σε συγκεκριμένα πρόσθετα.
* Περιορίστε την προσφορά σε συγκεκριμένες τιμές για κύριους πελάτες.
* Περιορίστε την προσφορά σε συγκεκριμένα προγράμματα τιμών.
* Περιορίστε την προσφορά σε συγκεκριμένες πολιτικές ακύρωσης.
* Περιορίστε την προσφορά στη διάρκεια της διαμονής.
* Περιορίστε την προσφορά σε κρατήσεις εκ των προτέρων.
* Περιορίστε την προσφορά σε αριθμό δωματίων.
* Περιορισμός προσφοράς σε κωδικό προσφοράς.
* Περιορισμός της προσφοράς στο εύρος IP ταξιδιωτών.
* Περιορίστε την προσφορά στην τελευταία στιγμή.
* Περιορισμός της προσφοράς στην πόλη των ταξιδιωτών.
* Περιορισμός της προσφοράς στη χώρα του ταξιδιώτη.
* Περιορίστε την προώθηση στην ήπειρο των ταξιδιωτών.
* Περιορισμός της προσφοράς στη ζώνη ώρας του ταξιδιώτη.
* Περιορίστε την προσφορά στις ημερομηνίες πώλησης.
* Περιορίστε την προσφορά στις ημερομηνίες διαμονής.
* Περιορίστε την προσφορά στις ημέρες κράτησης της εβδομάδας.
* Περιορίστε την προσφορά στις ημέρες άφιξης της εβδομάδας.
* Περιορίστε την προσφορά στις ημέρες αναχώρησης της εβδομάδας.
* Περιορίστε την προσφορά στις απαιτούμενες ημέρες της εβδομάδας.

Συνεχίστε να διαβάζετε για να δείτε πώς να χρησιμοποιείτε το καθένα.

### Πρόσθετο

:::σημείωση
Για να χρησιμοποιήσετε αυτήν τη λειτουργία, πρέπει να έχετε δημιουργήσει τουλάχιστον ένα πρόσθετο.
:::

Για να ισχύει μια προσφορά μόνο για τα πρόσθετα που επιλέγετε, ακολουθήστε τα εξής βήματα:

<Steps>
  1. Ενεργοποίηση του `Add-on restriction` διακόπτης.
  2. Επιλέξτε το πλαίσιο ελέγχου δίπλα σε κάθε πρόσθετο που θέλετε να συμπεριλάβετε.
</Steps>

:::

<Steps>

</Steps>


### πρόγραμμα τιμών

:::σημείωση

:::

Για να ισχύει μια προσφορά μόνο για τα προγράμματα τιμολόγησης που επιλέγετε, ακολουθήστε τα παρακάτω βήματα:

<Steps>

### Διάρκεια διαμονής

:::σημείωση
Αυτός ο περιορισμός είναι αυτός που χρησιμοποιείτε συνήθως ως`long term`προώθηση.
:::

Όταν προσθέτετε αυτόν τον περιορισμό, απαιτείτε από τους επισκέπτες να`stay at least`και`no more than`συγκεκριμένο αριθμό ημερών.

Ακολουθήστε αυτά τα βήματα για να ενεργοποιήσετε`Length of Stay`:

<Steps>
  1. Ενεργοποίηση του `Length of stay restriction` διακόπτης.
  2. Εισαγάγετε τον ελάχιστο αριθμό ημερών που πρέπει να διαμείνει ένας επισκέπτης. *π.χ. 7*
  3. Προαιρετικά, εισαγάγετε τον μέγιστο αριθμό ημερών που μπορεί να μείνει ένας επισκέπτης. *π.χ. 90*
</Steps>


*Παράδειγμα: Αυτή η προσφορά είναι διαθέσιμη μόνο σε επισκέπτες που διαμένουν περισσότερες από 10 ημέρες. Σημείωση: Η μέγιστη διάρκεια διαμονής είναι προαιρετική.*

### Προκράτηση

:::σημείωση
Αυτός ο περιορισμός είναι αυτός που χρησιμοποιείτε συνήθως ως`early bird`προώθηση.
:::

Όταν προσθέτετε αυτόν τον περιορισμό, απαιτείτε από τους επισκέπτες να`book at least`και`no more than`συγκεκριμένο αριθμό ημερών εκ των προτέρων.

Ακολουθήστε αυτά τα βήματα για να ενεργοποιήσετε`Advance booking`:

<Steps>
  1. Ενεργοποίηση του `Advance booking restriction` διακόπτης.
  2. Εισαγάγετε τον ελάχιστο αριθμό ημερών που χρειάζεται ένας επισκέπτης για να κάνει κράτηση εκ των προτέρων. *π.χ. 10*
  3. Προαιρετικά, εισαγάγετε τον μέγιστο αριθμό ημερών που ένας επισκέπτης χρειάζεται να κάνει κράτηση εκ των προτέρων. *π.χ. 180*
</Steps>


*Παράδειγμα: Αυτή η προσφορά είναι διαθέσιμη μόνο σε επισκέπτες που κάνουν κράτηση περισσότερες από 10 ημέρες νωρίτερα. Σημείωση: Απαιτείται επίσης να προσθέσετε ένα μέγιστο ποσό κράτησης εκ των προτέρων.*

### Αριθμός δωματίων

:::σημείωση
Η προσφορά είναι διαθέσιμη μόνο εάν ο επισκέπτης κάνει κράτηση για τον ελάχιστο αριθμό δωματίων.
:::

Όταν προσθέτετε αυτόν τον περιορισμό, απαιτείτε από τους επισκέπτες να`book at least`και`no more than`έναν συγκεκριμένο αριθμό δωματίων ως μέρος της κράτησής της.

Ακολουθήστε αυτά τα βήματα για να ενεργοποιήσετε`Number of rooms`:

<Steps>
  1. Ενεργοποίηση του `Room restriction` διακόπτης.
  2. Εισαγάγετε τον ελάχιστο αριθμό δωματίων που πρέπει να κάνει κράτηση ένας επισκέπτης. *π.χ. 3*
  3. Εισαγάγετε τον μέγιστο αριθμό δωματίων που χρειάζεται να κάνει κράτηση ένας επισκέπτης. *π.χ. 8*
</Steps>


*Παράδειγμα: Αυτή η προσφορά είναι διαθέσιμη μόνο σε επισκέπτες που κάνουν κράτηση για περισσότερα από 3 δωμάτια.*

### Εύρος IP

:::σημείωση
Αυτός είναι ένας καλός περιορισμός για χρήση όταν θέλετε να κάνετε μια διασταυρούμενη προωθητική καμπάνια με παρόχους κινητής τηλεφωνίας ή μεγαλύτερες εταιρείες που έχουν τις δικές τους περιοχές IP.
:::

Ακολουθήστε αυτά τα βήματα για να ενεργοποιήσετε`IP range`:

<Steps>
  1. Ενεργοποίηση του `IP range restriction` διακόπτης.
  2. Κάντε κλικ στο `Add IP range restriction` κουμπί.
  3. Εισαγάγετε το αρχικό εύρος IP \[καθολικά προσβάσιμο]. *π.χ. 203.0.113.50*
  4. Εισαγάγετε το τελικό εύρος IP \[παγκόσμια προσβάσιμο]. *π.χ. 203.0.113.59*
</Steps>


*Παράδειγμα: Αυτή η προσφορά είναι διαθέσιμη μόνο σε επισκέπτες με αριθμούς IP μεταξύ`203.0.113.50`-`203.0.113.59`.*

### Ημερομηνία πώλησης

:::σημείωση
Σας επιτρέπει να περιορίσετε σε ένα συγκεκριμένο εύρος ημερομηνιών. Ένας ταξιδιώτης θα είναι επιλέξιμος όταν κάνει κράτηση δωματίου στις ημερομηνίες που καθορίζονται εντός αυτού του εύρους ημερομηνιών.
:::

Ακολουθήστε αυτά τα βήματα για να ενεργοποιήσετε`Sell date`:

<Steps>
  1. Ενεργοποίηση του `Sell date restriction` διακόπτης.
  2. Κάντε κλικ στο `Add sell date restriction` κουμπί.
  3. Εισαγάγετε την ημερομηνία έναρξης. *π.χ. 19 Σεπτεμβρίου 2024*
  4. Εισαγάγετε την ημερομηνία λήξης. *π.χ. 1η Δεκεμβρίου 2024*
</Steps>


*Παράδειγμα: Αυτή η προσφορά είναι διαθέσιμη μόνο σε επισκέπτες που κάνουν κράτηση μεταξύ`Sept. 19th 2024`και`Dec. 1st 2024`.*

### Ημέρες κράτησης της εβδομάδας

:::σημείωση
Ενεργοποιώντας αυτόν τον περιορισμό, επιτρέπετε σε αυτήν την τιμή να είναι διαθέσιμη μόνο την καθορισμένη ημέρα της εβδομάδας που πραγματοποιείται η κράτηση.

Αυτός ο περιορισμός ταιριάζει καλά με τον "Περιορισμό ημερομηνίας πώλησης", όπου μπορείτε να περιορίσετε ένα εύρος ημερομηνιών και να περιορίσετε περαιτέρω το εύρος ημερομηνιών με ημέρες της εβδομάδας χρησιμοποιώντας αυτόν τον περιορισμό.
:::

Ακολουθήστε αυτά τα βήματα για να ενεργοποιήσετε`Booking days`:

<Steps>
  1. Ενεργοποίηση του `Booking days restriction` διακόπτης.
  2. Ενεργοποίηση διακοπτών για την έναρξη της ημέρας της εβδομάδας. *π.χ. Δευτέρα*
</Steps>

## Ημερομηνίες διακοπής λειτουργίας

Κάντε κλικ στο`Blackout dates`καρτέλα για να συνεχίσετε.

Ελέγξτε περαιτέρω την προσφορά καθορίζοντας ημερομηνίες για τις οποίες αυτή η προσφορά δεν θα πρέπει να είναι διαθέσιμη.

Για να προσθέσετε μια ημερομηνία αποκλεισμού, κάντε κλικ στο`Add blackout date`κουμπί.

* **Ημερομηνία έναρξης**Επιλέξτε την ημερομηνία έναρξης της ημερομηνίας διακοπής ρεύματος.
* **Ημερομηνία λήξης**Επιλέξτε την ημερομηνία λήξης της ημερομηνίας διακοπής ρεύματος.

Εάν οι ημερομηνίες κράτησης αγγίζουν αυτό το εύρος ημερομηνιών, η προσφορά δεν θα τεθεί σε ισχύ.

Ένας ταξιδιώτης θα μπορούσε να είναι επιλέξιμος για πολλαπλές προσφορές, όπως μακροπρόθεσμη προσφορά + early bird, ταυτόχρονα.
Χωρίς μια συνδυαστική προσφορά, ο ταξιδιώτης θα ήταν επιλέξιμος μόνο για μία από αυτές, δηλαδή την καλύτερη προσφορά.

Δημιουργώντας μια συνδυασμένη προσφορά, σέβεστε την επιλεξιμότητα του ταξιδιώτη και για τις δύο προσφορές και μπορείτε να την προσαρμόσετε περαιτέρω.

Για να δημιουργήσετε μια προσφορά πακέτου, κάντε κλικ στο`Create bundled promotion`κουμπί.

### Διαμόρφωση

* **Ονομα**Δώστε στην προσφορά πακέτου ένα όνομα που μόνο εσείς μπορείτε να δείτε.
* **Κατάσταση**Ενεργοποιήστε το διακόπτη για να ελέγξετε εάν το πακέτο είναι διαθέσιμο σε όλα τα κανάλια πωλήσεων.
* **Προσφορές**Επιλέξτε τουλάχιστον 2 μεμονωμένες προσφορές για να δημιουργήσετε ένα πακέτο.

### Ποσό παράκαμψης

Μπορείτε να επιλέξετε να παρακάμψετε τη συνολική έκπτωση που εφαρμόστηκε. Όταν την παραλείψετε, η έκπτωση κάθε προσφοράς που επιλέγετε παρακάτω υπολογίζεται σε μία έκπτωση.

* **Ποσό παράκαμψης**Είτε θέλετε να συγκεντρώσετε όλα τα ποσά με έκπτωση είτε να τα παρακάμψετε με το δικό σας ποσό.
* **Τύπος**Επιλέξτε αν θέλετε η τιμή να αυξηθεί ή να μειωθεί.*π.χ. Ορισμός σε`Discount`για μείωση τιμής.*
* **Τύπος τιμολόγησης**Επιλέξτε πώς θέλετε να υπολογιστεί αυτό το πακέτο.
  * `Per stay`: Το ποσό εφαρμόζεται στο χθεσινό βράδυ.*π.χ. 100% έκπτωση > Μείνετε 2 διανυκτερεύσεις - Πάρτε την 3η διανυκτέρευση δωρεάν.*
  * `Per night`: Ποσό πολλαπλασιασμένο με τον αριθμό των διανυκτερεύσεων.
  * `Per use`Δεν πραγματοποιείται επιπλέον υπολογισμός.
  * `Per person`: Ποσό πολλαπλασιασμένο επί τον αριθμό των επισκεπτών.
  * `Per person per night`: Ποσό πολλαπλασιασμένο επί τον αριθμό των διανυκτερεύσεων + τον αριθμό των επισκεπτών.
* **Τύπος ποσού**Επιλέξτε αν θέλετε ο τροποποιητής ποσού να είναι ένα σταθερό ποσό ή ένα ποσοστό.
