:::note
Η δημιουργία της πρώτης σας κύριας τιμής αποτελεί μέρος του οδηγού εισαγωγής.
:::

:::note
Μια `κύρια τιμή` είναι ένα μοναδικό ζευγάρι `δωμάτιο επισκέπτη / τιμοκατάλογος`.

*Οι περισσότεροι πάροχοι τιμών ταξιδιών χρησιμοποιούν μόνο το δωμάτιο επισκέπτη και τον τιμοκατάλογο σε αυτό το πλαίσιο. Όταν βλέπετε το δωμάτιό σας να προσφέρεται προς πώληση σε OTAs ή μηχανές κρατήσεων, οι ταξιδιώτες συγκρίνουν τους τιμοκαταλόγους.*

Το Wink πρόσθεσε τον όρο `κύρια τιμή` για να δώσει στη μοναδική αυτή συνδυαστική επιλογή περισσότερη ευελιξία.
:::

:::note
Οι κύριες τιμές δημιουργούνται / αφαιρούνται αυτόματα όταν προστίθεται ή αφαιρείται ένα δωμάτιο επισκέπτη ή τιμοκατάλογος.
:::

Για να διαχειριστείτε τις Κύριες Τιμές σας, κάντε κλικ στο `Monetize > Master Rate` από τη βασική γραμμή πλοήγησης.

Μια ιδιοκτησία θα έχει μια κύρια τιμή για κάθε συνδυασμό δωματίου επισκέπτη / τιμοκαταλόγου.

*Παράδειγμα: Αν η ιδιοκτησία σας έχει ένα δωμάτιο επισκέπτη και δύο τιμοκαταλόγους, ο επισκέπτης θα δει δύο κύριες τιμές διαθέσιμες προς πώληση (εφόσον πληρούνται όλες οι προϋποθέσεις).*

Μια καταχώρηση κύριας τιμής συνοδεύεται από δύο επιλογές διαχείρισης:

- Εναλλαγή διαθεσιμότητας κύριας τιμής.
- Ορισμός προνομίων.

## Διαθεσιμότητα

Όταν ενεργοποιείτε / απενεργοποιείτε μια κύρια τιμή, επηρεάζει τη διαθεσιμότητα σε όλα τα κανάλια πωλήσεών σας. Αυτός ο διακόπτης είναι ένας γρήγορος τρόπος να απενεργοποιήσετε μια συγκεκριμένη κύρια τιμή παντού, άμεσα.

## Προνόμια

Όταν οι ταξιδιώτες αναζητούν να κλείσουν ένα δωμάτιο επισκέπτη, δεν αλληλεπιδρούν απευθείας με το δωμάτιο. Αντίθετα, αλληλεπιδρούν με την κύρια τιμή αυτού του δωματίου.
Μια κύρια τιμή μπορεί να φορτωθεί με μοναδικά προνόμια που είναι διαθέσιμα μόνο για τον συγκεκριμένο συνδυασμό δωματίου επισκέπτη / τιμοκαταλόγου, δίνοντας έτσι στους ιδιοκτήτες ακινήτων έναν επιπλέον τρόπο να ενθαρρύνουν αυτόν τον συνδυασμό.

Δύο πράγματα που πρέπει να έχετε υπόψη όταν επιλέγετε ένα προνόμιο:

- Αν είναι εγγυημένο ή βασίζεται στη διαθεσιμότητα.
- Η αριθμητική του αξία.

Ένα εγγυημένο προνόμιο έχει υψηλότερη αριθμητική αξία καθώς αντιπροσωπεύει μεγαλύτερη αξία για τους πελάτες.
Κάθε ξενοδοχείο λαμβάνει ένα `σκορ προνομίων` που βασίζεται στη συνολική αριθμητική αξία όλων των προνομίων σε όλες τις κύριες τιμές.
Οι ταξιδιώτες που αναζητούν προσφορές μπορούν να ταξινομήσουν τα ξενοδοχεία με υψηλά σκορ προνομίων.

Τα προνόμια είναι ένας τρόπος για να:

- Διαφοροποιήσετε τον συνδυασμό δωματίου επισκέπτη / τιμοκαταλόγου.
- Επιτύχετε μεγαλύτερη ορατότητα. *δηλαδή, να έχετε υψηλότερο σκορ προνομίων από άλλα ξενοδοχεία στην περιοχή σας.*

:::tip
Τα προνόμια προέκυψαν από την επιθυμία των ξενοδοχείων να διαφοροποιηθούν και να `παίξουν το σύστημα`.
Την τελευταία φορά που το έκαναν, κυριολεκτικά αυτοκαταστράφηκαν δίνοντας όλα τα κέρδη τους στα OTAs για καλύτερη τοποθέτηση μέσω υψηλότερων προμηθειών.
Ιστορικά, οι προμήθειες ήταν το μόνο που μπορούσαν να εκμεταλλευτούν τα ξενοδοχεία.

Τα προνόμια είναι ένας τρόπος για τα ξενοδοχεία να ανταμείψουν τους ταξιδιώτες χωρίς να έχουν πραγματικό κόστος για τα ξενοδοχεία.

Αν έχετε προτάσεις για άλλους τύπους προνομίων που θα μπορούσαν να δοθούν, [επικοινωνήστε μαζί μας](mailto:hi@wink.travel).
:::

Υπάρχουν δύο τρόποι ενεργοποίησης αποθέματος για ένα κανάλι πωλήσεων:

- Εναλλαγή διαθεσιμότητας αποθέματος στην κάρτα. *Δείτε την εικόνα παραπάνω*
- Πλοήγηση στο `Distribution > Inventory` από τη βασική γραμμή πλοήγησης αφιερωμένη στη διαχείριση αποθέματος σε όλα τα κανάλια πωλήσεων.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν τις `Κύριες Τιμές` μπορούν να μεταβούν στο [Developers > APIs > Monetize](/el/developers/apis/#monetize-api).
