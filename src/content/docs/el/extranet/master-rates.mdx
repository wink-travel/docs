:::σημείωση
Η δημιουργία της πρώτης κύριας τιμής σας αποτελεί μέρος του οδηγού ενσωμάτωσης.
:::

:::σημείωση
A`master rate`είναι ένα μοναδικό`guest room / rate plan`ζευγάρι.

*Οι περισσότεροι πάροχοι τιμών ταξιδιών χρησιμοποιούν μόνο δωμάτιο επισκεπτών και πρόγραμμα τιμών σε αυτό το πλαίσιο. Όταν δείτε το δωμάτιό σας προς πώληση σε OTA ή μηχανές κρατήσεων, οι ταξιδιώτες θα συγκρίνουν προγράμματα τιμών.*

Ο Wink πρόσθεσε τον όρο`master rate`για να μπορέσει να δώσει στον μοναδικό συνδυασμό μεγαλύτερη ευελιξία.
:::

:::σημείωση
Οι κύριες τιμές δημιουργούνται / αφαιρούνται αυτόματα όταν προστίθεται ή αφαιρείται ένα δωμάτιο επισκεπτών ή ένας τιμοκατάλογος.
:::

Για να διαχειριστείτε τις κύριες τιμές σας, κάντε κλικ στο`Monetize > Master Rate`από την κύρια γραμμή πλοήγησης.

Ένα κατάλυμα θα έχει μια κύρια τιμή για κάθε συνδυασμό δωματίου επισκέπτη / τιμολογίου.

*Παράδειγμα: Εάν το κατάλυμά σας διαθέτει ένα δωμάτιο επισκεπτών και δύο τιμολογιακά προγράμματα, ο επισκέπτης θα δει δύο κύριες τιμές διαθέσιμες προς πώληση (εάν πληρούνται όλες οι προϋποθέσεις).*

Μια καταχώρηση κύριας τιμής συνοδεύεται από δύο επιλογές διαχείρισης:

* Εναλλαγή διαθεσιμότητας κύριας τιμής.
* Ορίστε προνόμια.

## Διαθεσιμότητα

Όταν ενεργοποιείτε/απενεργοποιείτε μια κύρια τιμή, επηρεάζει τη διαθεσιμότητα σε όλα τα κανάλια πωλήσεών σας. Αυτή η επιλογή είναι ένας γρήγορος τρόπος για να απενεργοποιήσετε μια συγκεκριμένη κύρια τιμή παντού, αμέσως.

## Προνόμια

Όταν οι ταξιδιώτες θέλουν να κάνουν κράτηση για ένα δωμάτιο, δεν αλληλεπιδρούν απευθείας με το δωμάτιο. Αντίθετα, αλληλεπιδρούν με την κύρια τιμή αυτού του δωματίου.

Μια κύρια τιμή μπορεί να φορτωθεί με μοναδικά προνόμια που είναι διαθέσιμα μόνο για τον συγκεκριμένο συνδυασμό δωματίου/τιμολογίου, δίνοντας έτσι στους ιδιοκτήτες καταλυμάτων έναν άλλο τρόπο να δώσουν κίνητρα για αυτόν τον συνδυασμό.

Δύο πράγματα που πρέπει να έχετε υπόψη σας όταν επιλέγετε ένα προνόμιο:

* Είτε είναι εγγυημένο είτε βάσει διαθεσιμότητας.
* Η αριθμητική του τιμή.

Ένα εγγυημένο προνόμιο έχει υψηλότερη αριθμητική αξία, καθώς αντιπροσωπεύει μεγαλύτερη αξία για τους πελάτες.
Κάθε ξενοδοχείο θα λάβει ένα`perk score`αυτό βασίζεται στη συνολική αριθμητική τιμή όλων των προνομίων σε όλες τις κύριες τιμές.
Οι ταξιδιώτες που αναζητούν μια προσφορά μπορούν να ταξινομήσουν ξενοδοχεία με υψηλές βαθμολογίες προνομίων.

Τα προνόμια είναι ένας τρόπος για να:

* Διαφοροποιήστε τον συνδυασμό δωματίου/τιμολογίου.
* Επιτύχετε μεγαλύτερη ορατότητα.*δηλαδή, να έχετε υψηλότερη βαθμολογία στην κλίμακα βαθμολογίας προνομίων από άλλα ξενοδοχεία στην περιοχή σας.*

:::συμβουλή
Τα προνόμια προήλθαν από την επιθυμία των ξενοδοχείων να διαφοροποιηθούν και`game the system`.
Την τελευταία φορά που το έκαναν αυτό, κυριολεκτικά αυτοκτόνησαν δίνοντας όλα τα κέρδη τους σε OTA για καλύτερη τοποθέτηση μέσω υψηλότερων προμηθειών.
Ιστορικά, οι προμήθειες ήταν το μόνο πράγμα στο οποίο τα ξενοδοχεία μπορούσαν να παίξουν παιχνίδια.

Τα προνόμια είναι ένας τρόπος με τον οποίο τα ξενοδοχεία ανταμείβουν τους ταξιδιώτες που δεν έχουν απαραίτητα πραγματικό κόστος για τα ξενοδοχεία.

Αν έχετε προτάσεις για άλλα είδη προνομίων που μπορείτε να χαρίσετε,[επικοινωνήστε μαζί μας](mailto:hi@wink.travel). :::

Υπάρχουν δύο τρόποι ενεργοποίησης αποθέματος για ένα κανάλι πωλήσεων:

* Ενεργοποιήστε/ενεργοποιήστε τη διαθεσιμότητα αποθέματος στην κάρτα.*Δείτε το παραπάνω σχήμα*
* Πλοήγηση σε`Distribution > Inventory`από την κύρια γραμμή πλοήγησης που είναι αφιερωμένη στη διαχείριση αποθέματος σε όλα τα κανάλια πωλήσεων.

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Master Rates`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Δημιουργία εσόδων](/developers/apis/#monetize-api).
