Η δημιουργία του πρώτου σας προγράμματος τιμών αποτελεί μέρος του οδηγού ενσωμάτωσης.
:::

Για να διαχειριστείτε τα προγράμματα τιμών σας, κάντε κλικ στο`Monetize > Rate Plans`από την κύρια γραμμή πλοήγησης.

Οι τιμοκατάλογοι είναι το φίλτρο μέσω του οποίου παρουσιάζονται τα δωμάτια στους χρήστες.
Περιέχει επίσης ρυθμίσεις που ελέγχουν τη δομή τιμολόγησης υπό ορισμένες συνθήκες.

Τα προγράμματα τιμολόγησης σάς επιτρέπουν:

* Ορίστε μια πολιτική ακύρωσης \[και εξαιρέσεις πολιτικής].
* Τιμή σε επιπλέον παιδιά, ενήλικες και μεμονωμένους ταξιδιώτες.
* Καθορισμένα γεύματα.
* Προσθέστε χρεώσεις για πρόωρο check-in και αργοπορημένο check-out.
* Ορίστε επιπλέον χρεώσεις.
* Ορίστε περιορισμούς.

Για να δημιουργήσετε έναν τιμοκατάλογο, κάντε κλικ στο`Create a new rate plan`κουμπί.

## Διαμόρφωση

Το`Configurations`Η καρτέλα είναι επιλεγμένη από προεπιλογή όταν ξεκινάτε.

Αυτή η ενότητα καλύπτει τις βασικές επιλογές για τη διαμόρφωση του τιμολογίου σας.

* **Όνομα προγράμματος τιμών**Το όνομα του τιμολογίου, όπως το αναφέρετε εσωτερικά.*π.χ. BAR RO*
* **Πόντοι πιστότητας**Εάν ο επισκέπτης συγκεντρώνει πόντους επιβράβευσης σε αυτό το πρόγραμμα τιμής.*π.χ. Όχι*
* **Μονοκατοικία**Ορίστε έναν τροποποιητή τιμής για έναν μόνο ένοικο σε ένα δωμάτιο πολλαπλών χρήσεων.*π.χ. 5% ΕΚΠΤΩΣΗ στην κανονική τιμή*
* **Επιπλέον ενήλικας**Ορίστε έναν τροποποιητή τιμής για έναν επιπλέον ενήλικα στο δωμάτιο.*π.χ. +10 / ενήλικας*
* **Επιπλέον παιδί**Ορίστε έναν τροποποιητή τιμής για ένα επιπλέον παιδί στο δωμάτιο.*π.χ. +5 / παιδί*

### Πολιτική ακύρωσης

* **Πολιτική ακύρωσης**Ορίστε την πολιτική ακύρωσης για αυτό το πρόγραμμα τιμών.*π.χ. Επιστρεφόμενο*
* **Εξαιρέσεις**Ορίστε πρόσθετες πολιτικές ακύρωσης που τίθενται σε ισχύ για συγκεκριμένα εύρη ημερομηνιών.*π.χ. Για τον μήνα Δεκέμβριο, χρησιμοποιήστε το`Refundable with extra conditions`πολιτική.*

### Γεύματα

Ορίστε τα γεύματα που συνοδεύουν αυτό το πρόγραμμα τιμών.*π.χ. Πρωινό συμπεριλαμβάνεται*

## Ταρίφα

Ορίστε πρόσθετες χρεώσεις που συνοδεύουν το πρόγραμμα τιμών.

* **Πρώιμο check-in**Ορίστε μια επιπλέον χρέωση όταν ο επισκέπτης θέλει να κάνει check-in νωρίς.*Αυτό προορίζεται μόνο για ενημερωτική χρήση*
* **Αργή αναχώρηση**Ορίστε μια επιπλέον χρέωση όταν ο επισκέπτης θέλει να κάνει check-out αργά.*Αυτό προορίζεται μόνο για ενημερωτική χρήση*

### Επιπλέον χρεώσεις

Ορίστε επιπλέον χρεώσεις με το πρόγραμμα τιμών.

Για να δημιουργήσετε μια επιπλέον χρέωση, κάντε κλικ στο`Create extra charge`κουμπί.

* **Τύπος τιμολόγησης**Επιλέξτε τον τρόπο υπολογισμού αυτής της χρέωσης.*π.χ. Ανά διαμονή*
* **Ποσό**Ορίστε το σταθερό ποσό της χρέωσης.*π.χ. 100*

Συμπληρώστε το όνομα και την περιγραφή αυτής της χρέωσης.

:::σημείωση
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και να είναι στα Αγγλικά.
:::

* **Ονομα**Όνομα χρέωσης.*π.χ. Εφάπαξ χρέωση καθαρισμού*
* **Περιγραφή**Περιγράψτε την κατηγορία σε μία ή δύο παραγράφους.*π.χ. Η βίλα θα καθαριστεί από την επαγγελματική ομάδα καθαριστών μας πριν από την άφιξή σας.*

## Περιορισμοί

Ορίστε περιορισμούς που ελέγχουν εάν ένα δωμάτιο επισκεπτών / ένα πρόγραμμα τιμών είναι διαθέσιμο ή όχι.

Υποστηριζόμενοι περιορισμοί:

* Διάρκεια διαμονής
* Προκράτηση
* Κατοχή
* Ηλικία
* Πώληση / Διαμονή / Κράτηση / Άφιξη / Αναχώρηση / Απαιτούμενες ημερομηνίες

### Διάρκεια διαμονής

Όταν προσθέτετε αυτόν τον περιορισμό, απαιτείτε από τους επισκέπτες να «διαμένουν τουλάχιστον» και «όχι περισσότερο από» έναν συγκεκριμένο αριθμό ημερών.

*Παράδειγμα: Αυτός ο τύπος δωματίου είναι διαθέσιμος μόνο σε επισκέπτες που διαμένουν περισσότερες από 10 ημέρες. Σημείωση: Η μέγιστη διάρκεια διαμονής είναι προαιρετική.*

### Προκράτηση

Όταν προσθέτετε αυτόν τον περιορισμό, απαιτείται από τους επισκέπτες να κάνουν κράτηση τουλάχιστον και όχι περισσότερο από έναν συγκεκριμένο αριθμό ημερών νωρίτερα.

*Παράδειγμα: Αυτή η προσφορά είναι διαθέσιμη μόνο σε επισκέπτες που κάνουν κράτηση περισσότερες από 10 ημέρες νωρίτερα. Σημείωση: Απαιτείται επίσης να προσθέσετε ένα μέγιστο ποσό κράτησης εκ των προτέρων.*

### Κατοχή

Όταν προσθέτετε αυτόν τον περιορισμό, απαιτείται η ομάδα που θα φτάσει να είναι «τουλάχιστον» και «όχι περισσότερο» από τον καθορισμένο αριθμό ατόμων.

### Ηλικία

Όταν προσθέτετε αυτόν τον περιορισμό, απαιτείτε από τους επισκέπτες να είναι «τουλάχιστον» και «όχι μεγαλύτεροι» από την καθορισμένη ηλικία.

### Ημερομηνίες πώλησης

Όταν προσθέτετε αυτόν τον περιορισμό, θέλετε να κάνετε αυτήν την τιμή διαθέσιμη μόνο για κρατήσεις εντός ενός συγκεκριμένου εύρους ημερομηνιών. Ένας επισκέπτης θα δει αυτήν την τιμή μόνο εάν κάνει κράτηση δωματίου ΤΗΝ ημερομηνία που καθορίζεται εντός αυτού του εύρους ημερομηνιών.

### Ημερομηνίες διαμονής

Όταν προσθέτετε αυτόν τον περιορισμό, θέλετε να κάνετε αυτήν την τιμή διαθέσιμη μόνο για διαμονή εντός ενός συγκεκριμένου εύρους ημερομηνιών. Ένας επισκέπτης θα βλέπει αυτήν την τιμή μόνο εάν κάνει κράτηση δωματίου ΓΙΑ την ημερομηνία που καθορίζεται εντός αυτού του εύρους ημερομηνιών.

### Ημερομηνίες κράτησης

:::σημείωση
Αυτός ο περιορισμός ταιριάζει απόλυτα με τον "Περιορισμό ημερομηνίας πώλησης", όπου μπορείτε να περιορίσετε ένα εύρος ημερομηνιών και να περιορίσετε περαιτέρω το εύρος ημερομηνιών με ημέρες της εβδομάδας χρησιμοποιώντας αυτόν τον περιορισμό.
:::

Ενεργοποιώντας αυτόν τον περιορισμό, επιτρέπετε την αναζήτηση αυτής της τιμής ΜΟΝΟ τις καθορισμένες ημέρες.

### Ημερομηνίες άφιξης

:::σημείωση
Αυτός ο περιορισμός ταιριάζει απόλυτα με τον "Περιορισμό ημερομηνίας διαμονής", όπου μπορείτε να περιορίσετε το εύρος ημερομηνιών και περαιτέρω τις ημέρες της εβδομάδας χρησιμοποιώντας αυτόν τον περιορισμό.
:::

Ενεργοποιώντας αυτόν τον περιορισμό, οι επισκέπτες ΠΡΕΠΕΙ να φτάσουν εκείνη την ημέρα για να δουν αυτήν την τιμή.

*Παράδειγμα: Για την ενεργοποίηση της Δευτέρας, ο επισκέπτης πρέπει να φτάσει Δευτέρα.*

### Ημερομηνίες αναχώρησης

:::σημείωση
Αυτός ο περιορισμός ταιριάζει απόλυτα με τον "Περιορισμό ημερομηνίας διαμονής", όπου μπορείτε να περιορίσετε το εύρος ημερομηνιών και περαιτέρω τις ημέρες της εβδομάδας χρησιμοποιώντας αυτόν τον περιορισμό.
:::

Ενεργοποιώντας αυτόν τον περιορισμό, οι επισκέπτες ΠΡΕΠΕΙ να αναχωρήσουν εκείνη την ημέρα για να δουν αυτήν την τιμή.

*Παράδειγμα: Για την ενεργοποίηση της Δευτέρας, ο επισκέπτης πρέπει να αναχωρήσει Δευτέρα.*

### Απαιτούμενες ημερομηνίες

:::σημείωση
Αυτός ο περιορισμός ταιριάζει απόλυτα με τον "Περιορισμό ημερομηνίας διαμονής", όπου μπορείτε να περιορίσετε το εύρος ημερομηνιών και περαιτέρω τις ημέρες της εβδομάδας χρησιμοποιώντας αυτόν τον περιορισμό.
:::

Ενεργοποιώντας αυτόν τον περιορισμό, οι επισκέπτες ΠΡΕΠΕΙ να παραμείνουν "τουλάχιστον μία" από αυτές τις ημέρες για να δουν αυτήν την τιμή.

*Παράδειγμα: Η ενεργοποίηση της Δευτέρας απαιτεί από τους επισκέπτες να διαμείνουν στο ξενοδοχείο τη Δευτέρα.*

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Rate Plans`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Δημιουργία εσόδων](/developers/apis/#monetize-api).
