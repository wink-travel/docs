Θα σας ζητηθεί να ορίσετε την τοποθεσία σας στον χάρτη κατά τη ρύθμιση.

Για να διαχειριστείτε την τοποθεσία του ακινήτου σας στον χάρτη, κάντε κλικ στο `Property > Map location` στην κύρια καρτέλα πλοήγησης.

Μπορείτε να ορίσετε την τοποθεσία σας στον χάρτη με δύο τρόπους:

1. Αναζητήστε το ακίνητό σας με το όνομά του στο πεδίο αναζήτησης πάνω από τον χάρτη και επιλέξτε το.
2. Χρησιμοποιώντας τον χάρτη, κάντε ζουμ και μετακινήστε τον μέχρι να βρείτε την τοποθεσία του ακινήτου σας και κάντε κλικ στον χάρτη όταν τη βρείτε.

Θα εμφανιστεί ένας δείκτης στον χάρτη με τις ενημερωμένες συντεταγμένες σας.

:::note
Το ακίνητό σας πρέπει να είναι καταχωρημένο στις Google Pages για να εμφανίζεται στο πεδίο αναζήτησης.
:::

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
