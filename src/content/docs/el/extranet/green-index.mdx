Ο πράσινος δείκτης σας μετράται με βάση τον αντίκτυπο που έχει το ακίνητό σας στο τοπικό περιβάλλον και στις αγοραστικές σας αποφάσεις.

Για να διαχειριστείτε τη βαθμολογία του Πράσινου Δείκτη, κάντε κλικ στο `Property > Green Index` στην κύρια καρτέλα πλοήγησης.

Περιηγηθείτε σε κάθε καρτέλα και απαντήστε στο ερωτηματολόγιο όσο καλύτερα μπορείτε.

Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
