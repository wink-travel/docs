Οι αίθουσες συσκέψεων σάς επιτρέπουν να προβάλλετε και να πουλάτε τις αίθουσες συσκέψεών σας, εντός και εκτός των εγκαταστάσεων. Για να δημιουργήσετε μια αίθουσα συσκέψεων, κάντε κλικ στο`Create meeting room`κουμπί.

Οι ακόλουθες ενότητες σας καθοδηγούν σε κάθε στάδιο της διαχείρισης μιας αίθουσας συσκέψεων.

## Διαμόρφωση

Το`Configurations`Η καρτέλα είναι επιλεγμένη από προεπιλογή όταν ξεκινάτε.

Αυτή η ενότητα καλύπτει τα βασικά της αίθουσας συσκέψεών σας.

* **Εσωτερικό όνομα**Το όνομα του δωματίου, όπως το αναφέρετε εσωτερικά.*π.χ. Αίθουσα Χορού 1*
* **Τύπος**Ο τύπος της αίθουσας συσκέψεων.*π.χ. Αίθουσα χορού*
* **Πρόσβαση**Ο τύπος πρόσβασης της αίθουσας συσκέψεων.*π.χ. Ιδιωτική Πρόσβαση*
* **Επίπεδο**Το επίπεδο στο οποίο βρίσκεται η αίθουσα συνεδριάσεων.*π.χ. Μεζανίν*
* **Αφιερωμένο**Είτε πρόκειται για ειδική αίθουσα συσκέψεων.*π.χ. Ναι*
* **Ακανόνιστος**Εάν αυτή η αίθουσα συσκέψεων έχει ακανόνιστο σχήμα.*π.χ. Όχι*
* **Ικανότητα**Ο αριθμός των PAX που υποστηρίζει η αίθουσα συσκέψεων.*π.χ. 10*

### Διάσταση

Αυτή η ενότητα σάς επιτρέπει να εισαγάγετε τις διαστάσεις της αίθουσας συσκέψεων.

* **Εκταση**Εμβαδόν σε τετραγωνικά μέτρα.*π.χ. 1000 τ.μ.*
* **Υψος**Ύψος δωματίου σε μέτρα.*π.χ. 12μ*
* **Πλάτος**Πλάτος δωματίου σε μέτρα.*π.χ. 100 μέτρα*
* **Μήκος**Μήκος δωματίου σε μέτρα.*π.χ. 100 μέτρα*

## Πρόγραμμα

Αυτή η ενότητα σάς επιτρέπει να διαχειρίζεστε πότε η αίθουσα συσκέψεων είναι άμεσα διαθέσιμη.

### Ώρες λειτουργίας

* **Ανοίγει**Ώρα της ημέρας κατά την οποία η αίθουσα συσκέψεων καθίσταται διαθέσιμη.*π.χ. 09:00*
* **Κλείνει**Ώρα της ημέρας κατά την οποία η αίθουσα συσκέψεων δεν είναι διαθέσιμη.*π.χ. 21:00*

### Ημέρες λειτουργίας

Αυτή η ενότητα σάς επιτρέπει να ορίσετε τις ημέρες της εβδομάδας κατά τις οποίες η αίθουσα συσκέψεων είναι διαθέσιμη.

*Ενεργοποιήστε το διακόπτη στη θέση ✅ για να υποδείξετε ότι η ημέρα είναι διαθέσιμη.*

### Εποχής

:::σημείωση
Δεν υπάρχει όμορφο, εγγενές πεδίο φόρμας Ημέρα / Μήνας. Σε αυτό το σενάριο, χρησιμοποιούμε μόνο τον μήνα και την ημέρα από αυτόν τον επιλογέα ημερομηνίας.
::::

Αυτή η ενότητα σάς επιτρέπει να ορίσετε τις ημερομηνίες κατά τις οποίες η αίθουσα συσκέψεων θα είναι διαθέσιμη καθ' όλη τη διάρκεια του έτους.

* **Ημερομηνία έναρξης**Ημερομηνία κατά την οποία η αίθουσα συσκέψεων θα είναι διαθέσιμη.*π.χ. 11 Σεπτεμβρίου 2024*
* **Ημερομηνία έναρξης**Ημερομηνία κατά την οποία η αίθουσα συσκέψεων δεν είναι διαθέσιμη.*π.χ. 11 Ιουλίου 2024*

## Ανεσεις

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Amenities`καρτέλα για να συνεχίσετε.

Όπως και με τις παροχές του καταλύματος, οι παροχές της αίθουσας συσκέψεων ισχύουν μόνο για τη συγκεκριμένη αίθουσα συσκέψεων. Ελέγξτε κάθε παροχή που ισχύει.

## Περιγραφές

:::σημείωση
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και να είναι στα Αγγλικά.
:::

Κάντε κλικ στο`Descriptions`καρτέλα για να συνεχίσετε.

Όπως και με το κείμενο καλωσορίσματος του καταλύματος, μπορείτε να δημιουργήσετε μεταφρασμένες περιγραφές και για την αίθουσα συσκέψεων.
Μπορείτε να γράψετε σε όσες γλώσσες θέλετε. Θα προσθέσουμε μεταφράσεις για όλες τις πιο δημοφιλείς γλώσσες.
Αυτό είναι το κείμενο που θα βλέπουν οι ταξιδιώτες όταν βλέπουν τις αίθουσες συσκέψεών σας.

* **Ονομα**Δώστε ένα όνομα στην αίθουσα συσκέψεων.*π.χ. Αίθουσα Χορού 1*
* **Περιγραφή**Περιγράψτε την αίθουσα συσκέψεων σε μία ή δύο παραγράφους.*π.χ. Αυτή είναι η καλύτερη αίθουσα χορού σε ολόκληρο τον κόσμο...*

## Κοινωνικά μέσα

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Social media`καρτέλα για να συνεχίσετε.

Ίσως αναρωτιέστε: «Γιατί μια αίθουσα συσκέψεων έχει τη δική της ενότητα κοινωνικής δικτύωσης;». Μια αίθουσα συσκέψεων μπορεί επίσης να είναι μια εγκατάσταση κοντά στο κατάλυμα αλλά εκτός των εγκαταστάσεων με το δικό της προσωπικό και το δικό της προφίλ στο IG.

Τα μέσα κοινωνικής δικτύωσης σας κρατούν σε επαφή με νέους και υπάρχοντες ταξιδιώτες με την πάροδο του χρόνου.
Προσθέστε τους λογαριασμούς σας στα μέσα κοινωνικής δικτύωσης στο προφίλ σας, ώστε οι επισκέπτες να μπορούν εύκολα να επικοινωνήσουν μαζί σας και να μάθουν για αυτήν την αίθουσα συσκέψεων.

<Steps>
  1. Ενεργοποιήστε/ενεργοποιήστε το διακόπτη ενός κοινωνικού δικτύου που θέλετε να ενεργοποιήσετε.
  2. Εισαγάγετε το όνομα του λογαριασμού σας σε αυτό το δίκτυο.
</Steps>


## Φήμη

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Reputation`καρτέλα για να συνεχίσετε.

Παρόμοια με το`Social media`καρτέλα, θα χρησιμοποιούσατε αυτήν την καρτέλα εάν αυτή η αίθουσα συσκέψεων έχει ξεχωριστή βαθμολογία/βαθμολογίες φήμης ή επιπλέον διακρίσεις που θέλετε να επισημάνετε εδώ.

Η φήμη σας είναι το ψηφιακό σας νόμισμα και θέλετε να μπορείτε να την παίρνετε μαζί σας όπου κι αν πάτε.

Μια καταχώριση φήμης μπορεί να είναι οτιδήποτε θέλει να πιστωθεί το κατάλυμά σας, όπως μια αξιολόγηση τρίτου μέρους ή για το bake-off όπου ο σεφ σας κέρδισε το χρυσό.
Σε αυτήν την ενότητα, μπορείτε να προσθέσετε τις βαθμολογίες και τις κατατάξεις σας από όλο το Διαδίκτυο.

Κάντε κλικ στο`Add reputation score`κουμπί.

<Steps>
  1. **Κατηγορία** Επιλέξτε τον τύπο βαθμολογίας που θέλετε να προσθέσετε. *π.χ. αξιολόγηση τρίτου μέρους*
  2. **Προμηθευτής** Πληκτρολογήστε ποιος σας έδωσε τη βαθμολογία. *π.χ. BigOTA.com*
  3. **Τύπος αξιολόγησης** Επιλέξτε τη μορφή με την οποία λάβατε τη βαθμολογία. *π.χ. Αριθμητικό*
  4. **Ημερομηνία** Προαιρετικά, εισαγάγετε την ημερομηνία που λάβατε τη βαθμολογία.
  5. **Εκτίμηση** Εισαγάγετε τη βαθμολογία που λάβατε. *π.χ. 8,5*
  6. **Μέγιστη βαθμολογία** Προαιρετικά, εισαγάγετε τη μέγιστη δυνατή βαθμολογία για αυτόν τον τύπο βραβείου. *π.χ. 10*
</Steps>


## Πώληση βάσει χαρακτηριστικών

:::σημείωση
Αυτή η ενότητα είναι προαιρετική.
:::

Κάντε κλικ στο`Attribute Based Selling`καρτέλα για να συνεχίσετε.

Όλες οι εγκαταστάσεις μας διαθέτουν τη δική τους ενότητα ABS. Αυτή η ενότητα σάς επιτρέπει να δημιουργήσετε ευκαιρίες upsell στο κύριο απόθεμα ή να μετατρέψετε ένα μη συναλλακτικό είδος αποθέματος σε συναλλακτικό.
Σας συνιστούμε ανεπιφύλακτα να δημιουργήσετε αυτές τις ευκαιρίες για να συνδεθείτε ενδεχομένως με τον χρήστη που κάνει την κράτηση σε υψηλότερη τιμή κατά τη διαδικασία κράτησης.

:::συμβουλή
Μια αίθουσα συσκέψεων μπορεί να διατεθεί προς πώληση μέσω της ABS. Μπορεί να διατεθεί προς πώληση με διάφορους τρόπους:

* Ωριαίος.

* Καθημερινά.

* Καθημερινά με catering.

* +++ Αφήστε τη φαντασία σας ελεύθερη και κερδίστε μερικά επιπλέον χρήματα!!
  :::

* **Τιμή**Δώστε στον ταξιδιώτη μια γρήγορη ένδειξη για το πόσο ακριβή είναι αυτή η αίθουσα συσκέψεων.

* **Ενεργός**Ενεργοποιήστε/απενεργοποιήστε τον διακόπτη σε 🛑 εάν θέλετε αυτή η αίθουσα συσκέψεων να μην είναι διαθέσιμη για κράτηση σε όλα τα κανάλια πωλήσεων.

* **Επιλέξιμος**Υποδείξτε εάν οι θυγατρικές εταιρείες εισπράττουν προμήθεια από αυτό το απόθεμα.

* **Προτεινόμενα**Υποδείξτε εάν αυτή η αίθουσα συσκέψεων θα πρέπει να τοποθετηθεί σε πιο εμφανές σημείο, εάν παρουσιαστεί η δυνατότητα.

* **Πώληση βάσει χαρακτηριστικών**Εναλλαγή διακόπτη στη θέση ✅ για να ενεργοποιήσετε το ABS σε αυτό το απόθεμα.*Μόλις ενεργοποιηθεί, πρέπει να προσθέσετε τουλάχιστον 1 αντικείμενο ABS προς πώληση*

### Βοηθητικό στοιχείο

Όταν ενεργοποιήσετε το ABS, μπορείτε να ξεκινήσετε να προσθέτετε βοηθητικά είδη και να τα διαθέτετε προς πώληση.

*Από προεπιλογή, έχει ήδη δημιουργηθεί ένα κενό στοιχείο για εσάς. Μπορείτε να προσθέσετε περισσότερα κάνοντας κλικ στο`Add item for sale`κουμπί.*

* **Ονομα**Δώστε ένα όνομα στον βοηθό σας.*π.χ. Αίθουσα χορού 1 ώρα*
* **Τύπος τιμολόγησης**Υποδείξτε πώς θέλετε να υπολογίσετε την τιμή.*π.χ. Ορισμός σε`Per hour`για να υποδείξει ότι η τιμή βασίζεται στον αριθμό των ωρών ενοικίασης.*
* **Βασική τιμή**Εισαγάγετε την κανονική τιμή αυτού του είδους.*π.χ. 20 / ώρα*
* **Τιμή με έκπτωση**Εισαγάγετε την τιμή που λαμβάνουν οι χρήστες του Wink κατά την κράτηση του αντικειμένου.*π.χ. 15 / ώρα*

### Όρια

Μπορείτε να περιορίσετε τη διαθεσιμότητα αυτού του προϊόντος ορίζοντας ελάχιστο/μέγιστο όριο στην ποσότητα.

* **Ελάχιστη ποσότητα**Περιορίστε τον ελάχιστο αριθμό που πρέπει να κρατηθεί.*π.χ. 20 συμμετέχοντες*
* **Μέγιστη ποσότητα**Περιορίστε τον μέγιστο αριθμό που πρέπει να κρατηθεί.*π.χ. 100 συμμετέχοντες*

### Περιγραφές

:::σημείωση
Πρέπει να γράψετε τουλάχιστον μία περιγραφή και να είναι στα Αγγλικά.
:::

* **Ονομα**Δώστε ένα όνομα στον βοηθό σας.*π.χ. Κράτηση με την ώρα*
* **Περιγραφή**Περιγράψτε το βοηθητικό στοιχείο σε μία ή δύο παραγράφους.*π.χ. Κράτηση Αίθουσας Χορού 1 ανά ώρα. Ελάχιστος αριθμός ατόμων: 20 - Μέγιστος αριθμός ατόμων: 100.*

## Εγγύτητα

Κάντε κλικ στο`Proximity`καρτέλα για να συνεχίσετε.

Δεν βρίσκεται όλο το απόθεμα εντός των εγκαταστάσεων. Εάν έχετε απόθεμα που βρίσκεται εκτός των εγκαταστάσεων, ενημερώστε τον ταξιδιώτη εδώ και θα σας δοθεί η ευκαιρία να ορίσετε μια ξεχωριστή τοποθεσία στον χάρτη μαζί με προαιρετικά στοιχεία επικοινωνίας.

*Αυτό μπορεί να είναι χρήσιμο σε περιπτώσεις που μια αίθουσα συσκέψεων βρίσκεται λίγα χιλιόμετρα μακριά από το ξενοδοχείο και ο ταξιδιώτης θέλει να εξηγήσει στον οδηγό ταξί πώς να φτάσει εκεί.*

Κάντε κλικ στο`Save`κουμπί για να συνεχίσετε.

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Meeting Rooms`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Εγκαταστάσεις](/developers/apis/#facilities-api).
