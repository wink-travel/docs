Οι παροχές του καταλύματος είναι υπηρεσίες και εγκαταστάσεις που είναι διαθέσιμες στο κατάλυμά σας.

Για να διαχειριστείτε τις παροχές του καταλύματός σας, κάντε κλικ στο `Κατάλυμα > Παροχές` στην κύρια καρτέλα πλοήγησης.

Επιλέξτε όλα τα κουτάκια που ισχύουν για το κατάλυμά σας.

Κάντε κλικ στο κουμπί `Αποθήκευση` για να συνεχίσετε.
