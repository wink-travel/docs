Όταν γίνεται μια κράτηση στο Wink, συμβαίνουν τα εξής:

<Steps>
1. Ενημερώνεται ο channel manager, το CRS ή το PMS σας.
2. Λαμβάνετε ένα email επιβεβαίωσης *(επαφές γραφείου κρατήσεων)* με έναν σύνδεσμο για να διαχειριστείτε τη νέα σας κράτηση.
3. Ο ταξιδιώτης λαμβάνει ένα email επιβεβαίωσης που περιλαμβάνει επίσης πληροφορίες για το πώς να επικοινωνήσει με το γραφείο κρατήσεών σας.
4. Οι προγραμματιστές μπορούν να ακούσουν το webhook event `booking.created` και να λαμβάνουν λεπτομερείς πληροφορίες κράτησης από εμάς σε πραγματικό χρόνο.
</Steps>

Ακολουθήστε είτε τον σύνδεσμο στο email, είτε μεταβείτε στο `Account > Bookings` από την κύρια γραμμή πλοήγησης για να ξεκινήσετε.

## Πλέγμα κρατήσεων

Το πλέγμα κρατήσεων σας δείχνει όλες τις παρελθούσες και μελλοντικές κρατήσεις σας, ταξινομημένες κατά ημερομηνία κράτησης σε αύξουσα σειρά.

### Φίλτρα

Μπορείτε να εφαρμόσετε φίλτρα για να βρείτε τις κρατήσεις που αναζητάτε.

- Ημερομηνία κράτησης [εύρος]
- Ημερομηνία άφιξης [εύρος]
- Ημερομηνία αναχώρησης [εύρος]
- Κωδικός κράτησης
- Επώνυμο ταξιδιώτη
- Όνομα ταξιδιώτη
- Όνομα βασικής τιμής
- Κανάλι πωλήσεων / Πηγή

### Εξαγωγή

Μπορείτε να εξάγετε τις κρατήσεις που ταιριάζουν με τα τρέχοντα κριτήρια φίλτρου σε CSV.

Για να εξάγετε τις κρατήσεις σας:

<Steps>
1. Βεβαιωθείτε ότι τα επιθυμητά φίλτρα είναι ενεργά.
2. Περιμένετε να ανανεωθεί το πλέγμα κρατήσεων.
2. Κάντε κλικ στο κουμπί εικονιδίου 🗂️ (*Εξαγωγή αρχείου*) στην κάτω δεξιά γωνία του πλέγματος.
</Steps>

## Κράτηση

Επιλέξτε την κράτηση με την οποία θέλετε να εργαστείτε από το πλέγμα κρατήσεων και θα ανακατευθυνθείτε στη σελίδα `Booking`.
Αυτή η σελίδα περιέχει όλες τις λεπτομέρειες που χρειάζεστε για να καλωσορίσετε σωστά τον επισκέπτη σας την ημέρα της άφιξης.

Η σελίδα περιέχει επίσης μερικές σημαντικές ενέργειες που χρειάζεστε για να εξυπηρετήσετε σωστά τους επισκέπτες σας. Μπορείτε να βρείτε όλες τις διαθέσιμες ενέργειες στο αναπτυσσόμενο μενού `Actions` στην κορυφή της σελίδας.

### Επανάληψη αποστολής email

Για να ξαναστείλετε το email επιβεβαίωσης στον ταξιδιώτη, κάντε κλικ στο `Actions > Resend confirmation email`.

### Ακύρωση κράτησης

Για να ακυρώσετε την κράτηση ως κατάλυμα, κάντε κλικ στο `Actions > Cancel booking`.

Θα εμφανιστεί ένα αναδυόμενο παράθυρο που σας επιτρέπει να:

<Steps>
1. Επιλέξετε τον τύπο ακύρωσης. *π.χ. Λανθασμένες ημερομηνίες*
2. Πληκτρολογήσετε έναν λόγο για την ακύρωση.
</Steps>

*Αυτό το κουμπί ακολουθεί τους κανόνες της πολιτικής ακύρωσής σας και επομένως δεν θα είναι πάντα διαθέσιμο.*

### Αίτηση επιστροφής χρημάτων

Για να ζητήσετε επιστροφή χρημάτων, για οποιονδήποτε λόγο, κάντε κλικ στο `Actions > Request refund`.

Θα εμφανιστεί ένα αναδυόμενο παράθυρο που σας επιτρέπει να:

<Steps>
1. Ζητήσετε πλήρη ή μερική επιστροφή χρημάτων.
    - Αν επιλέξετε μερική, θα σας ζητηθεί το ποσό.
2. Επιλέξετε έναν λόγο.
3. Πληκτρολογήσετε έναν περιγραφικό λόγο για το αίτημα.
4. Μας ενημερώσετε αν θέλετε επίσης να ακυρώσετε την κράτηση, ανεξάρτητα από το αν το αίτημα εγκριθεί ή όχι.
</Steps>

 *Αυτή η ενέργεια είναι διαθέσιμη μέχρι να έχουν επεξεργαστεί τα χρήματα για αυτήν την κράτηση.*.

## Συγχρονισμός ημερολογίου

Για να ξεκινήσετε το συγχρονισμό των κρατήσεών σας στο Wink με το αγαπημένο σας λογισμικό ημερολογίου, μεταβείτε στο `Distribution > Sync bookings to your calendar` από την κύρια γραμμή πλοήγησης για να ξεκινήσετε.

Την πρώτη φορά, θα σας ζητηθεί να δημιουργήσετε έναν κωδικό πρόσβασης που μπορείτε να χρησιμοποιήσετε με το λογισμικό ημερολογίου σας για να πιστοποιήσετε τη σύνδεση.

Κάντε κλικ στο κουμπί `Yes` για να συνεχίσετε ή στο 'No' αν αλλάξατε γνώμη.

Θα δείτε 3 πεδία που πρέπει να χρησιμοποιήσετε για να συνδέσετε το Wink με το λογισμικό ημερολογίου σας:

1. Το URL που θα χρησιμοποιήσετε για να εγγραφείτε στις κρατήσεις.
2. Το όνομα χρήστη για πιστοποίηση.
3. Το κλειδί πρόσβασης για πιστοποίηση.

Μόλις ολοκληρώσετε αυτό το βήμα και προσθέσετε το URL CalDAV στο λογισμικό ημερολογίου σας, το ημερολόγιό σας θα αρχίσει να εμφανίζει κρατήσεις από το Wink.

## Κριτικές

3 ημέρες μετά την αναχώρηση του επισκέπτη, ο επισκέπτης θα λάβει ένα email από εμάς ζητώντας να αξιολογήσει το κατάλυμά σας.
Η κριτική αποτελείται από απλές ερωτήσεις πολλαπλής επιλογής με τιμές από 1 έως 5. Οι κατηγορίες είναι κατάλυμα, υπηρεσίες και εγκαταστάσεις.

Οι επισκέπτες μπορούν επίσης να σας αφήσουν ένα μήνυμα στο οποίο μπορείτε να απαντήσετε και να επιλέξετε να το κάνετε δημόσιο για να το δουν άλλοι επισκέπτες.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν `Booking` μπορούν να μεταβούν στο [Developers > API > Property Booking](/el/developers/apis/#property-booking-api).
