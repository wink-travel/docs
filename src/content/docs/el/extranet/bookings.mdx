Όταν γίνεται κράτηση στο Wink, συμβαίνουν τα εξής:

<Steps>
  1. Ο διαχειριστής καναλιών σας, το CRS ή το PMS ειδοποιείται.
  2. Λαμβάνετε ένα email επιβεβαίωσης *(επικοινωνία με το γραφείο κρατήσεων)* με έναν σύνδεσμο για τη διαχείριση της νέας σας κράτησης.
  3. Ο ταξιδιώτης λαμβάνει ένα email επιβεβαίωσης που περιλαμβάνει επίσης τον τρόπο επικοινωνίας με το γραφείο κρατήσεων.
  4. Οι προγραμματιστές μπορούν να ακούσουν το `booking.created` webhook event και λάβετε λεπτομερείς πληροφορίες κράτησης από εμάς σε πραγματικό χρόνο.
</Steps>


Ακολουθήστε τον σύνδεσμο στο email ή μεταβείτε στη διεύθυνση`Account > Bookings`από την κύρια γραμμή πλοήγησης για να ξεκινήσετε.

## Πλέγμα κρατήσεων

Το πλέγμα κρατήσεων σας δείχνει όλες τις προηγούμενες και μελλοντικές κρατήσεις σας, ταξινομημένες κατά ημερομηνία κράτησης σε αύξουσα σειρά.

### Φίλτρα

Μπορείτε να εφαρμόσετε φίλτρα για να βρείτε τις κρατήσεις που αναζητάτε.

* Ημερομηνία κράτησης \[εύρος]
* Ημερομηνία άφιξης \[εύρος]
* Ημερομηνία αναχώρησης \[εύρος]
* Κωδικός κράτησης
* Επώνυμο του ταξιδιώτη
* Όνομα ταξιδιώτη
* Όνομα κύριας τιμής
* Κανάλι πωλήσεων / Πηγή

### Εξαγωγή

Μπορείτε να εξαγάγετε κρατήσεις που ταιριάζουν με τα τρέχοντα κριτήρια φίλτρου σας σε CSV.

Για να εξαγάγετε τις κρατήσεις σας:

<Steps>
  1. Βεβαιωθείτε ότι τα φίλτρα που επιθυμείτε είναι ενεργά.
  2. Περιμένετε να ανανεωθεί το πλέγμα κρατήσεων.
  3. Κάντε κλικ στο 🗂️ (*Εξαγωγή αρχείων*) κουμπί εικονιδίου στην κάτω δεξιά γωνία του πλέγματος.
</Steps>


## Εγγραφή

Επιλέξτε την κράτηση με την οποία θέλετε να εργαστείτε από το πλέγμα κρατήσεων και θα μεταφερθείτε στο`Booking`σελίδα.
Αυτή η σελίδα περιέχει όλες τις λεπτομέρειες που πρέπει να γνωρίζετε για να καλωσορίσετε σωστά τον επισκέπτη σας την ημέρα της άφιξης.

Η σελίδα περιέχει επίσης μερικές σημαντικές ενέργειες που χρειάζεστε για να εξυπηρετήσετε σωστά τους καλεσμένους σας. Μπορείτε να βρείτε όλες τις διαθέσιμες ενέργειες στο`Actions`αναπτυσσόμενο μενού στο επάνω μέρος της σελίδας.

### Επανάληψη αποστολής ηλεκτρονικού ταχυδρομείου

Για να στείλετε ξανά το email επιβεβαίωσης στον ταξιδιώτη, κάντε κλικ στο`Actions > Resend confirmation email`.

### Ακύρωση κράτησης

Για να ακυρώσετε την κράτηση ως κατάλυμα, κάντε κλικ στο`Actions > Cancel booking`.

Θα εμφανιστεί ένα αναδυόμενο παράθυρο που σας επιτρέπει:

<Steps>
  1. Επιλέξτε τον τύπο ακύρωσης. *π.χ. Λανθασμένες ημερομηνίες*
  2. Πληκτρολογήστε έναν λόγο για την ακύρωση.
</Steps>


*Αυτό το κουμπί ακολουθεί τους κανόνες της πολιτικής ακύρωσης και επομένως δεν θα είναι πάντα διαθέσιμο.*

### Αίτημα επιστροφής χρημάτων

Για να ζητήσετε επιστροφή χρημάτων, για οποιονδήποτε λόγο, κάντε κλικ`Actions > Request refund`.

Θα εμφανιστεί ένα αναδυόμενο παράθυρο που σας επιτρέπει:

<Steps>
  1. Ζητήστε πλήρη ή μερική επιστροφή χρημάτων.
     * Αν επιλέξετε μερικό, θα σας ζητηθεί το ποσό.
  2. Επιλέξτε έναν λόγο.
  3. Πληκτρολογήστε έναν περιγραφικό λόγο για το αίτημα.
  4. Ενημερώστε μας αν θέλετε επίσης να ακυρώσετε την κράτηση, ανεξάρτητα από το αν το αίτημα εγκριθεί ή όχι.
</Steps>


*Αυτή η ενέργεια είναι διαθέσιμη μέχρι να διεκπεραιωθούν τα χρήματα για αυτήν την κράτηση.*.

## Συγχρονισμός ημερολογίου

Για να ξεκινήσετε τον συγχρονισμό των κρατήσεών σας στο Wink με το αγαπημένο σας λογισμικό ημερολογίου, μεταβείτε στη διεύθυνση`Distribution > Sync bookings to your calendar`από την κύρια γραμμή πλοήγησης για να ξεκινήσετε.

Την πρώτη φορά, θα σας ζητηθεί να δημιουργήσετε έναν κωδικό πρόσβασης που μπορείτε να χρησιμοποιήσετε με το λογισμικό ημερολογίου σας για να επαληθεύσετε τη σύνδεση.

<figure>
  <CloudinaryImage width="350" height="188" resize="scale" quality="best" alt="Δημιουργία κωδικού πρόσβασης" src="academy/caldav-pass-code" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Σας ρωτάει αν θέλετε να δημιουργήσετε έναν κωδικό πρόσβασης CalDAV*</figcaption>
</figure>


Κάντε κλικ στο`Yes`κουμπί για να συνεχίσετε ή «Όχι» εάν αλλάξατε γνώμη.

Θα δείτε 3 πεδία που πρέπει να χρησιμοποιήσετε για να συνδέσετε το Wink με το λογισμικό ημερολογίου σας:

1. Η διεύθυνση URL που θα χρησιμοποιηθεί για την εγγραφή σε κρατήσεις.
2. Το όνομα χρήστη με το οποίο θα γίνει έλεγχος ταυτότητας.
3. Το κλειδί πρόσβασης για τον έλεγχο ταυτότητας.

Μόλις ολοκληρώσετε αυτό το βήμα και προσθέσετε τη διεύθυνση URL CalDAV στο λογισμικό ημερολογίου σας, το ημερολόγιό σας θα αρχίσει να εμφανίζει κρατήσεις από το Wink.

## Κριτικές

<figure>
  <CloudinaryImage width="700" height="151" resize="scale" quality="best" alt="Πλέγμα αξιολόγησης" src="academy/reviews" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Πλέγμα κριτικών*</figcaption>
</figure>


3 ημέρες μετά την αναχώρησή του, ο επισκέπτης θα λάβει ένα email από εμάς με το οποίο θα του ζητάμε να αξιολογήσει το κατάλυμά σας.
Η αξιολόγηση αποτελείται από απλές ερωτήσεις πολλαπλής επιλογής με τιμές που κυμαίνονται από 1 έως 5. Οι κατηγορίες είναι το κατάλυμα, οι υπηρεσίες και οι εγκαταστάσεις.

Οι επισκέπτες μπορούν επίσης να σας αφήσουν ένα μήνυμα στο οποίο μπορείτε να απαντήσετε και να επιλέξετε να το δημοσιοποιήσετε ώστε να το δουν και άλλοι επισκέπτες.

  <CloudinaryImage width="700" height="421" resize="scale" quality="best" alt="Έγκριση αξιολόγησης" src="academy/review-details" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Φόρμα ελέγχου λεπτομερειών*</figcaption>
</figure>


Θα λάβετε ένα email από εμάς όταν δημιουργηθεί μια κριτική και μπορείτε να συνδεθείτε στο Wink για να δείτε και να απαντήσετε σε αυτήν.

<figure>

</figure>


Οι εγκεκριμένες κριτικές θα εμφανίζονται στην ενότητα αξιολογήσεων στη σελίδα προορισμού σας.

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Booking`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > Κράτηση Ακινήτων](/developers/apis/#property-booking-api).
