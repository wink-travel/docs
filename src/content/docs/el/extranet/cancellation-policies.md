Για να διαχειριστείτε τις Πολιτικές Ακύρωσής σας, κάντε κλικ στο `Monetize > Cancellation policies` από την κύρια γραμμή πλοήγησης.

Οι πολιτικές ακύρωσης καθορίζουν πώς διαχειρίζονται τα χρήματα του πελάτη σε περίπτωση που ο επισκέπτης δεν μπορεί να έρθει. Οι πολιτικές ακύρωσης διακρίνονται σε δύο βασικές κατηγορίες:

1. Επιστρέψιμες
2. Μη επιστρέψιμες

Τις περισσότερες φορές, αυτές οι δύο επιλογές είναι αρκετές. Ωστόσο, και οι δύο σας δίνουν μεγάλη ευελιξία για να προσαρμόσετε περαιτέρω την πολιτική ακύρωσής σας.

:::tip[Διασκεδαστικό γεγονός]
Το Wink υποστηρίζει συνολικά 8500 παραλλαγές πολιτικών ακύρωσης.
:::

Για να δημιουργήσετε μια πολιτική ακύρωσης, κάντε κλικ στο κουμπί `Create a new cancellation policy`.

## Μη επιστρέψιμη

Για να δημιουργήσετε μια μη επιστρέψιμη πολιτική ακύρωσης, θέστε τον διακόπτη σε 🛑 απενεργοποιημένο όταν σας ρωτήσουν αν η πολιτική είναι επιστρέψιμη.

### Επιλογές

Μπορείτε προαιρετικά να διαμορφώσετε περαιτέρω την `μη επιστρέψιμη` πολιτική σας για μεγαλύτερο έλεγχο.

- **Πόσο χρεώνεται ο επισκέπτης για μια ακύρωση;** Από προεπιλογή, μια μη επιστρέψιμη πολιτική είναι 100% μη επιστρέψιμη. Μπορείτε να ρυθμίσετε το ποσό εδώ. *π.χ. 50% της συνολικής τιμής*
- **Υπάρχει προθεσμία για τη χρέωση;** Αν ρυθμίσατε το ποσό, μπορείτε να το προσαρμόσετε περαιτέρω ορίζοντας πόσο διαρκεί η ισχύς αυτού του ποσού. *π.χ. Μέχρι 3 ημέρες πριν την άφιξη*
- **Πόσο χρεώνεται ο επισκέπτης για ακύρωση μετά την προθεσμία;** Σε περίπτωση που ο επισκέπτης ξεπεράσει την παραπάνω προθεσμία, μπορείτε να ορίσετε το ποσό που πληρώνει μετά την προθεσμία. *π.χ. 70% της συνολικής τιμής*

## Επιστρέψιμη

Για να δημιουργήσετε μια επιστρέψιμη πολιτική ακύρωσης, θέστε τον διακόπτη σε ✅ ενεργοποιημένο όταν σας ρωτήσουν αν η πολιτική είναι επιστρέψιμη.

### Επιλογές

Μπορείτε προαιρετικά να διαμορφώσετε περαιτέρω την `επιστρέψιμη` πολιτική σας για μεγαλύτερο έλεγχο.

- **Πόσο νωρίτερα από την άφιξη μπορεί ο επισκέπτης να ακυρώσει χωρίς χρέωση;** Από προεπιλογή, μια επιστρέψιμη πολιτική είναι πάντα 100% επιστρέψιμη. Μπορείτε να ρυθμίσετε την προθεσμία εδώ. *π.χ. 3 ημέρες πριν την άφιξη*
- **Τι πρέπει να χρεωθεί ο επισκέπτης αν η ακύρωση γίνει μετά την παραπάνω προθεσμία;** Αν ρυθμίσατε την προθεσμία, μπορείτε να ορίσετε το ποσό που πληρώνει ο επισκέπτης μετά την προθεσμία. *π.χ. 50% της συνολικής τιμής*
- **Πόσο πρέπει να χρεωθεί ο επισκέπτης για 'Μη εμφάνιση';** Σε περίπτωση που ο επισκέπτης δεν εμφανιστεί, μπορείτε να ορίσετε το ποσό που χρεώνεται για `Μη εμφάνιση`. *π.χ. 100% της συνολικής τιμής*
