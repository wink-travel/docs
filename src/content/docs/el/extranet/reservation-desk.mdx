Κατά τη ρύθμιση θα σας ζητηθούν τα στοιχεία επικοινωνίας του γραφείου κρατήσεών σας.

Για να διαχειριστείτε το γραφείο κρατήσεών σας, κάντε κλικ στο `Property > Reservation Desk` στην κύρια καρτέλα πλοήγησης.

Ενημερώστε τη φόρμα και κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.

:::note
Δεν είναι απαραίτητο να χρησιμοποιήσετε συγκεκριμένο όνομα του βασικού ατόμου στη ρεσεψιόν σας. Απλώς προσθέτει οικειότητα στον χώρο.
Οι επισκέπτες που φτάνουν θα ήθελαν να συνδεθούν με ένα πραγματικό άτομο κατά την άφιξή τους και να μπορούν να ρωτήσουν κάτι όπως "Γεια... είναι εδώ η Lisa;"
:::
