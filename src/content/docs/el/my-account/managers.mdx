## Πρόσκληση χρήστη

Μπορείτε να προσκαλέσετε άλλους χρήστες να βοηθήσουν στη διαχείριση του λογαριασμού σας συνεργάτη. Μόλις ένας χρήστης αποδεχτεί την πρόσκλησή σας, θα έχει πλήρη πρόσβαση στον λογαριασμό σας με εξαίρεση τις πληρωμές. Εσείς θα είστε πάντα ο μόνος υπεύθυνος για τα κεφάλαια που κατέχετε.

<Steps>
1. Επιλέξτε τον λογαριασμό με τον οποίο θέλετε να εργαστείτε από τα [Accounts](https://my.wink.travel).
2. Κάντε κλικ στο `Actions` στο κάτω μέρος της κάρτας του λογαριασμού και επιλέξτε `Manage account`.
3. Κάντε κλικ στην καρτέλα `Managers` στην επόμενη σελίδα.
4. Εισάγετε το email του ατόμου που θέλετε να προσκαλέσετε να διαχειριστεί τον λογαριασμό σας.
5. Κάντε κλικ στο σύνδεσμο `Invite` δίπλα στο email.
</Steps>

Η Wink θα στείλει ένα email στο χρήστη που θέλετε να προσθέσετε στον λογαριασμό σας και το άτομο αυτό μπορεί να επιλέξει να αποδεχτεί ή να απορρίψει την πρόσκλησή σας.

## Μεταβίβαση ιδιοκτησίας

Αν είστε ο τρέχων ιδιοκτήτης του λογαριασμού, έχετε τη δυνατότητα να μεταβιβάσετε την ιδιοκτησία του λογαριασμού σε έναν υπάρχοντα διαχειριστή.

Ακολουθήστε τα παρακάτω βήματα για να αλλάξετε την ιδιοκτησία:

<Steps>
1. Επιλέξτε τον λογαριασμό με τον οποίο θέλετε να εργαστείτε από τα [Accounts](https://my.wink.travel).
2. Κάντε κλικ στο `Actions` στο κάτω μέρος της κάρτας του λογαριασμού και επιλέξτε `Manage account`.
3. Κάντε κλικ στην καρτέλα `Managers` στην επόμενη σελίδα.
4. Κάντε κλικ στο σύνδεσμο `Make owner` δίπλα σε έναν από τους υπάρχοντες διαχειριστές του λογαριασμού.
5. Τώρα είστε διαχειριστής του λογαριασμού αλλά όχι πλέον ο ιδιοκτήτης.
</Steps>
