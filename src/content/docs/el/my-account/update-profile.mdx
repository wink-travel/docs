Αν θέλετε να ενημερώσετε το προφίλ χρήστη για τον λογαριασμό σας στο Wink, ακολουθήστε αυτά τα βήματα:

<Steps>
1. Μεταβείτε στο [my.wink.travel](https://my.wink.travel).
2. Συνδεθείτε κάνοντας κλικ στο `Sign-in / Register` στην πάνω δεξιά γωνία.
3. Μόλις συνδεθείτε, κάντε κλικ στο εικονίδιο του προφίλ σας, στην πάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
4. Κάντε κλικ στον σύνδεσμο `Profile`.
5. Ακολουθήστε τις οδηγίες για να ενημερώσετε το `e-mail`, το `όνομα` και το `επώνυμό` σας.
6. Μπορείτε επίσης να ανεβάσετε μια νέα `φωτογραφία προφίλ` σας.
7. Κάντε κλικ στο `Save`.
</Steps>

<Steps>
1. Πρέπει να είστε συνδεδεμένοι σε μία από τις ιστοσελίδες μας.
2. Κάντε κλικ στο εικονίδιο του προφίλ σας, στην πάνω δεξιά γωνία και θα εμφανιστεί ένα μενού.
3. Κάντε κλικ στον σύνδεσμο `Profile`.
4. Ακολουθήστε τις οδηγίες για να ενημερώσετε το `e-mail`, το `όνομα` και το `επώνυμό` σας.
5. Μπορείτε επίσης να ανεβάσετε μια νέα `φωτογραφία προφίλ` σας.
6. Κάντε κλικ στο `Update`.
</Steps>

:::note
Η νέα σας φωτογραφία προφίλ θα εμφανιστεί την επόμενη φορά που θα αποσυνδεθείτε πλήρως από το Wink.
:::
