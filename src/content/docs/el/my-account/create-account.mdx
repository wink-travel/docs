Το Wink σας συνδέει με ταξιδιωτικό απόθεμα παγκοσμίως. Αυτό το άρθρο σας καθοδηγεί βήμα-βήμα για να δημιουργήσετε τον πρώτο σας λογαριασμό στο Wink.

<Steps>
1. Μεταβείτε στο [my.wink.travel](https://my.wink.travel).
2. Συνδεθείτε κάνοντας κλικ στο `Sign-in / Register` στην πάνω δεξιά γωνία.
3. Μόλις συνδεθείτε, κάντε κλικ στο `Accounts` στο αριστερό μενού.
4. Κάντε κλικ στο κουμπί `Create` για να ξεκινήσετε τη δημιουργία νέου λογαριασμού.
5. Πρώτα επιλέξτε τον τύπο λογαριασμού που θέλετε να δημιουργήσετε.
6. **Όνομα** Το όνομα του λογαριασμού σας. *Δεν χρειάζεται να είναι το πραγματικό σας όνομα, αλλά κάτι που θα βλέπουν οι χρήστες όταν επισκέπτονται τη σελίδα σας στο WinkLinks ή όταν οι προμηθευτές θέλουν να συνδεθούν απευθείας μαζί σας.*
7. **Τύπος οντότητας** Αν είστε εταιρεία ή ιδιώτης. *Χρησιμοποιείται κυρίως για να καθοριστεί η επιλεξιμότητά σας για λήψη χρημάτων.*
8. **Τύπος** Επιλέξτε τον τύπο που ταιριάζει καλύτερα στον λογαριασμό σας. *π.χ. Influencer*
9. **Περιγραφή** Γράψτε μια περιγραφή για τον λογαριασμό σας. *Αυτή θα εμφανίζεται στη σελίδα σας στο WinkLinks και όταν οι προμηθευτές θέλουν να συνδεθούν απευθείας μαζί σας.*
10. **Ιστοσελίδα** Εισάγετε την κύρια ιστοσελίδα σας. *Μπορεί να είναι το URL του λογαριασμού σας στο IG ή η ιστοσελίδα της εταιρείας σας. Πρέπει να είναι ο χώρος όπου κάνετε το μεγαλύτερο μέρος της διαδικτυακής σας δουλειάς. Οι προμηθευτές θα επισκέπτονται αυτούς τους συνδέσμους για να αποφασίσουν αν θα συνδεθούν μαζί σας.*
11. **Πόλη** Πού εδρεύετε;
12. Αποδεχτείτε τους Όρους & Προϋποθέσεις και τους Όρους Πληρωμής.
13. Κάντε κλικ στο κουμπί `Save`.
14. Θα ανακατευθυνθείτε στη σελίδα `Manage account` όπου μπορείτε να προσθέσετε περισσότερες λεπτομέρειες στον λογαριασμό σας.
</Steps>

Συγχαρητήρια 🎉 Μόλις δημιουργήσατε τον πρώτο σας λογαριασμό στο Wink.

:::tip[Επαγγελματική συμβουλή]
Ένας λογαριασμός ακινήτου λειτουργεί επίσης ως λογαριασμός συνεργάτη. Αν είστε ακίνητο, δεν χρειάζεται να δημιουργήσετε δύο λογαριασμούς...
Μπορείτε να διαχειρίζεστε και να πουλάτε το απόθεμά σας χρησιμοποιώντας τον ίδιο λογαριασμό.
:::
