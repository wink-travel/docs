Ένας χρήστης Wink μπορεί να επιλέξει να σας επιτρέψει να διαχειριστείτε έναν από τους λογαριασμούς της ιδιοκτησίας, του συνεργάτη ή του ταξιδιωτικού πράκτορα του. Όταν συμβεί αυτό, θα λάβετε ένα email με αίτημα πρόσκλησης.

import { Steps } from '@astrojs/starlight/components';

Για να δείτε τις τρέχουσες προσκλήσεις σας στο Wink, ακολουθήστε αυτά τα βήματα:

<Steps>
 1. Συνδεθείτε στο [my.wink.travel](https://my.wink.travel).
 2. Κάντε κλικ στο `Invites` από το αριστερό πλαϊνό μενού.
 3. Θα δείτε μια λίστα με όλες τις τρέχουσες προσκλήσεις.
 4. Αυτή η ενότητα θα εμφανίσει όλες τις αιτήσεις σας.
 5. Επιλέξτε να `Accept` ή `Reject` την πρόσκληση.
 6. Εάν αποδεχτείτε την πρόσκληση, ο λογαριασμός θα εμφανιστεί στην ενότητα `Accounts`.
</Steps>
