Το Wink παρέχει ένα σύστημα αναφορών που σας επιτρέπει να δημιουργείτε και να εξάγετε δεδομένα από τον λογαριασμό σας. Οι αναφορές είναι διαθέσιμες σε μορφή CSV για εύκολη εισαγωγή σε υπολογιστικά φύλλα, εργαλεία ανάλυσης ή τα δικά σας συστήματα.

## Τύποι αναφορών

Μπορείτε να δημιουργήσετε αναφορές για τις ακόλουθες κατηγορίες δεδομένων:

| Τύπος αναφοράς | Τι περιλαμβάνει |
|---|---|
| **Κατάλυμα** | Λεπτομέρειες καταλύματος, κατάσταση καταχώρησης, σύνοψη αποθεμάτων. |
| **Χρήστης** | Λογαριασμοί χρηστών, ρόλοι, σύνοψη δραστηριότητας. |
| **Κράτηση** | Ιστορικό κρατήσεων, καταστάσεις, έσοδα, λεπτομέρειες επισκεπτών. |
| **Λογαριασμός** | Πληροφορίες λογαριασμού, υπόλοιπα, ιστορικό συναλλαγών. |

## Πώς να δημιουργήσετε μια αναφορά

1. Μεταβείτε στην ενότητα αναφορών στον πίνακα ελέγχου του λογαριασμού σας.
2. Επιλέξτε τον **τύπο αναφοράς** που θέλετε να δημιουργήσετε.
3. Διαμορφώστε τυχόν **φίλτρα** για να περιορίσετε τα δεδομένα (π.χ. εύρος ημερομηνιών, κατάσταση, κατάλυμα).
4. Επιλέξτε τα **πεδία** που θέλετε να συμπεριληφθούν στην εξαγωγή. Κάθε τύπος αναφοράς προσφέρει ένα σύνολο διαθέσιμων πεδίων για επιλογή.
5. Κάντε κλικ στο **Εξαγωγή** για να κατεβάσετε την αναφορά ως αρχείο CSV.

## Προσαρμοσμένα πεδία

Οι αναφορές υποστηρίζουν προσαρμοσμένη επιλογή πεδίων. Επιλέγετε ποιες στήλες θα συμπεριληφθούν στην εξαγωγή σας, ώστε να λαμβάνετε ακριβώς τα δεδομένα που χρειάζεστε — τίποτα περισσότερο, τίποτα λιγότερο.

## Περιπτώσεις χρήσης

- **Συμφωνία εσόδων** — Εξάγετε αναφορές κρατήσεων με οικονομικές λεπτομέρειες για συμφωνία με το λογιστικό σας σύστημα.
- **Ανάλυση απόδοσης** — Εξάγετε αναφορές καταλυμάτων για να συγκρίνετε την απόδοση των καταχωρήσεων στο χαρτοφυλάκιό σας.
- **Επισκόπηση επισκεπτών** — Εξάγετε δεδομένα κρατήσεων για να κατανοήσετε τα δημογραφικά στοιχεία των ταξιδιωτών και τα πρότυπα κρατήσεων.
- **Έλεγχος λογαριασμού** — Εξάγετε δεδομένα λογαριασμού για συμμόρφωση ή εσωτερική ανασκόπηση.
