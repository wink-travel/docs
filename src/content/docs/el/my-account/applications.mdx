Οι εφαρμογές σας δίνουν πρόσβαση σε όλες τις λειτουργίες του Wink ως προγραμματιστής. Μπορείτε να έχετε πρόσβαση σε όλα τα ίδια δεδομένα και ακόμη και να επεκτείνετε τις λειτουργίες του Wink.
Μπορείτε να επιλέξετε να χρησιμοποιήσετε τον διακομιστή OAuth2 του Wink για να αυθεντικοποιήσετε τον ταξιδιωτικό σας ιστότοπο με τον ίδιο τρόπο που θα χρησιμοποιούσατε το Google ή το Facebook.
Με αυτόν τον τρόπο μπορείτε να επεκτείνετε τις λειτουργίες του Wink στον δικό σας ιστότοπο ή εφαρμογή.

:::note
Η πρώτη σας εφαρμογή έχει ήδη δημιουργηθεί όταν [δημιουργήσετε νέο λογαριασμό](/el/my-account/create-account). Χρησιμοποιούμε αυτήν την εφαρμογή για ορισμένα σενάρια κρατήσεων.

Δεν μπορείτε να επαναχρησιμοποιήσετε αυτήν την εφαρμογή. Παρακαλούμε δημιουργήστε νέες εφαρμογές ανάλογα με τις ανάγκες σας.
:::

Μια εφαρμογή που δημιουργείται στο Wink μπορεί να χρησιμοποιηθεί για όλα τα API endpoints.
:::

<Steps>
 1. Επιλέξτε τον λογαριασμό με τον οποίο θέλετε να εργαστείτε από το [Accounts](https://my.wink.travel).
 2. Κάντε κλικ στο `Actions` στο κάτω μέρος της κάρτας του λογαριασμού και επιλέξτε `Manage account`.
 3. Κάντε κλικ στην καρτέλα `Applications` στην επόμενη σελίδα.
 4. Αυτή η ενότητα θα εμφανίσει όλες τις εφαρμογές σας.
 5. Επιλέξτε να επεξεργαστείτε μια υπάρχουσα ή να δημιουργήσετε μια νέα.
 </Steps>

## Δημιουργία εφαρμογής

Καθώς δεν έχετε πρόσβαση στο `secret key` της εφαρμογής που δημιουργήσαμε για εσάς, θα χρειαστεί να δημιουργήσετε τη δική σας.

:::caution
Πρέπει να δημιουργήσετε έναν συνεργατικό λογαριασμό στο Wink ή έναν λογαριασμό στο TripPay πριν δημιουργήσετε μια εφαρμογή, καθώς θα συνδέσετε την εφαρμογή σας με συγκεκριμένο λογαριασμό.
:::

Ακολουθούν τα βήματα για τη δημιουργία μιας εφαρμογής:

<Steps>
1. Από τη σελίδα των εφαρμογών, κάντε κλικ στο κουμπί `Create new application`.
2. ***Όνομα*** Δώστε ένα όνομα στην εφαρμογή σας. *π.χ. Cool App*
3. **Όνομα οντότητας** Συνδέστε την εφαρμογή σας με έναν από τους υπάρχοντες λογαριασμούς σας. *π.χ. Cool Account*
4. **Redirect URIs** Απαιτείται τουλάχιστον ένα redirect URI. Χρησιμοποιείται για την επικύρωση του domain ανακατεύθυνσης μετά από επιτυχή αυθεντικοποίηση. *π.χ. https://www.cool-site.com*
5. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των εφαρμογών σας. Θα δείτε τη νέα σας εφαρμογή στη λίστα και θα έχετε πρόσβαση στο μυστικό κλειδί της εφαρμογής μόνο αυτή τη φορά. Φροντίστε να το αντιγράψετε και να το αποθηκεύσετε σε ασφαλές μέρος.

## Ενημέρωση εφαρμογής

Θα χρειαστεί να ενημερώσετε την εφαρμογή σας αν αλλάξει κάποιο από τα redirect URIs ή αν θέλετε να προσθέσετε ή να αφαιρέσετε κάποιο URI.

Για να ενημερώσετε μια υπάρχουσα εφαρμογή, κάντε τα εξής:

<Steps>
1. Κάντε κλικ στο σύνδεσμο `Actions` για την εφαρμογή που θέλετε να ενημερώσετε.
2. Κάντε κλικ στο κουμπί `Update` κάτω από το Actions.
3. Κάντε τις αλλαγές στην εφαρμογή.
4. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των εφαρμογών σας.

## Αφαίρεση εφαρμογής

:::caution
Παρακαλούμε μην διαγράψετε την εφαρμογή που δημιουργήσαμε αυτόματα για εσάς, ώστε να διασφαλιστεί ότι όλες οι λειτουργίες δουλεύουν για τον λογαριασμό σας.
:::

Αν δεν χρειάζεστε πλέον μια εφαρμογή, μπορείτε να την αφαιρέσετε.

Για να αφαιρέσετε μια εφαρμογή, κάντε τα εξής:

<Steps>
1. Κάντε κλικ στο σύνδεσμο `Actions` για την εφαρμογή που θέλετε να αφαιρέσετε.
2. Κάντε κλικ στο κουμπί `Remove` κάτω από το Actions.
3. Κάντε κλικ στο κουμπί `OK` για να επιβεβαιώσετε τη διαγραφή.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των εφαρμογών σας.
