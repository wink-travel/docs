Κάποιες φορές θέλουμε να επικοινωνήσουμε μαζί σας. Όταν συμβεί αυτό, θα δείτε το κουδούνι στην πάνω δεξιά γωνία να αλλάζει.
Για να διαβάσετε τι σας στείλαμε:

<Steps>
1. Κάντε κλικ στο `Λογαριασμός > Ειδοποιήσεις` στη βασική γραμμή πλοήγησης.
2. Θα δείτε νέες και παλιές ειδοποιήσεις.
3. Επιλέξτε να `Σημειώσετε ως αναγνωσμένο` ή να `Διαγράψετε` τις ειδοποιήσεις μόλις τις διαβάσετε.
</Steps>

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν `Affiliates` μπορούν να μεταβούν στο [Developers > API > Affiliate](/el/developers/apis/#affiliate-api).
