Αν θέλετε να αλλάξετε τον κωδικό σας, ή έχετε ξεχάσει τον κωδικό σας, ακολουθήστε αυτά τα βήματα:

## Ενημέρωση κωδικού

<Steps>
1. Μεταβείτε στο [my.wink.travel](https://my.wink.travel).
1. Συνδεθείτε κάνοντας κλικ στο `Sign-in / Register` στην πάνω δεξιά γωνία.
2. Κάντε κλικ στο εικονίδιο του προφίλ σας, στην πάνω δεξιά γωνία και θα εμφανιστεί ένα μενού.
3. Κάντε κλικ στον σύνδεσμο `Change password`.
4. Ακολουθήστε τις οδηγίες για να αλλάξετε τον κωδικό σας.
5. Κάντε κλικ στο `Change password` για να συνεχίσετε.
</Steps>

Την επόμενη φορά που θα συνδεθείτε, θα χρειαστείτε τον νέο σας κωδικό.

## Ξεχάσατε τον κωδικό

Μερικές φορές είναι απαραίτητο να επαναφέρετε τον κωδικό χρήστη.

Ακολουθούν τα βήματα για την επαναφορά του κωδικού χρήστη:

<Steps>
1. Κάντε κλικ στον σύνδεσμο `Forgot password` από τη σελίδα σύνδεσης.
2. Πληκτρολογήστε το όνομα χρήστη / το e-mail για τον λογαριασμό που θέλετε να επαναφέρετε.
3. Κάντε κλικ στο κουμπί `Forgot password`. *Θα ανακατευθυνθείτε στη φόρμα επαναφοράς κωδικού.*
4. Ελέγξτε το e-mail σας.
</Steps>

2. Επιβεβαιώστε τον νέο σας κωδικό.
3. Κάντε κλικ στο κουμπί `Reset password`. *Θα ανακατευθυνθείτε ξανά στη σελίδα σύνδεσης.*
</Steps>
