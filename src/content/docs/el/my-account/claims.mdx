*Περίπου χρόνος ολοκλήρωσης: ~2 λεπτά*

Άλλα μέλη μπορούν επίσης να [καλέσουν την καταχώρησή σας](/el/studio/search) να ενταχθεί στο Wink. Θα λάβετε από εμάς μια πρόσκληση μέσω email με ένα αίτημα `lead` που μπορείτε να διεκδικήσετε και να χρησιμοποιήσετε για να ολοκληρώσετε τη δημιουργία της καταχώρησης.

Για να δείτε τις τρέχουσες απαιτήσεις σας στο Wink, ακολουθήστε τα παρακάτω βήματα:

<Steps>
 1. Συνδεθείτε στο [my.wink.travel](https://my.wink.travel).
 2. Κάντε κλικ στο `Claims` από το αριστερό μενού.
 3. Θα δείτε μια λίστα με όλες τις τρέχουσες απαιτήσεις.
 4. Επιλέξτε την απαίτηση που θέλετε να επεξεργαστείτε και συμπληρώστε τα ελλιπή πεδία της φόρμας για να δημιουργήσετε τον λογαριασμό σας.
</Steps>
