Μόλις δημιουργήσετε έναν λογαριασμό, εδώ είναι τα πράγματα που μπορείτε να κάνετε περαιτέρω σε αυτόν ενώ βρίσκεστε στο [my.wink.travel](https://my.wink.travel).

- Ενημέρωση εκπροσώπου λογαριασμού
- Ενημέρωση πληροφοριών λογαριασμού
- Ενημέρωση μοναδικών συστημικών ID *τρόποι εύρεσης του λογαριασμού σας σε όλο το σύστημα*
- Ενημέρωση διεύθυνσης
- Ενημέρωση φωτογραφίας προφίλ *[εμφανίζεται στο WinkLinks και κατά τη σύνδεση με καταλύματα και συνεργάτες στο Wink]*
- Ενημέρωση λογότυπου *[εμφανίζεται σε e-mails]*

Μπορείτε επίσης...

- Να διαχειριστείτε την [ηλεκτρονική σας παρουσία](/el/my-account/online-presence).
- Να αναθέσετε έναν [αφιερωμένο εκπρόσωπο πρακτορείου](/el/my-account/agency-representation) για να διαχειρίζεται τον λογαριασμό σας [με αμοιβή].
- Να αναθέσετε [διαχειριστές](/el/my-account/managers) που θέλετε να έχουν πρόσβαση στον λογαριασμό σας.
- Να διαχειριστείτε [προσαρμογές](/el/my-account/customizations) για να ελέγχετε την εμπειρία χρήστη κατά την προβολή του αποθέματός σας.
- Να διαχειριστείτε [εφαρμογές](/el/my-account/applications) για να έχετε πρόσβαση στο Wink ως προγραμματιστής και να προσθέσετε / επεκτείνετε τις δικές σας λειτουργίες στο Wink.
- Να διαχειριστείτε [webhooks](/el/my-account/webhooks) για να λαμβάνετε ειδοποιήσεις για γεγονότα στην πλατφόρμα Wink σε πραγματικό χρόνο. *π.χ. Λάβετε ειδοποίηση για μια νέα κράτηση*.

## Ενημέρωση φωτογραφίας προφίλ


:::note
Η φωτογραφία προφίλ του λογαριασμού συνεργάτη σας δεν πρέπει να συγχέεται με τη φωτογραφία προφίλ χρήστη σας στο Wink. Η φωτογραφία προφίλ του λογαριασμού συνεργάτη θα εμφανίζεται στη σελίδα σας στο WinkLinks και θα τη βλέπουν προμηθευτές που επιθυμούν να συνδεθούν μαζί σας.

Η φωτογραφία προφίλ χρήστη είναι κυρίως για εσωτερική χρήση.
:::
