## Εύρεση πραγμάτων για πώληση

Όταν ξεκινάτε ως συνεργάτης στο Wink, το πρώτο που λέτε στον εαυτό σας είναι:

> Ας βγάλουμε χρήματα!.

Αμέσως μετά, ρωτάτε τον εαυτό σας:

> Πώς; Τι; Πότε; Πού;

Κάποιοι συνεργάτες ήδη ξέρουν τι θέλουν να πουλήσουν μέσω των καναλιών τους. Αν είστε νέος στο παιχνίδι και θέλετε να καταλάβετε ποιοι προμηθευτές να προωθήσετε... τότε οι αναλύσεις μπορούν να σας φανούν χρήσιμες.

Εδώ είναι ένα παράδειγμα για το πώς μπορεί να το κάνετε για πρώτη φορά:

<Steps>
1. Μεταβείτε στο [Wink Studio](/el/studio/search) για να αναζητήσετε προμηθευτές.
2. Δημιουργήστε ένα γράφημα που παρακολουθεί κρατήσεις σε πολλές χώρες. *Τώρα ξέρετε ποιες χώρες αποδίδουν καλύτερα.*
3. Δημιουργήστε άλλο γράφημα για μερικές από τις πόλεις σε αυτές τις χώρες που αποδίδουν καλά. *Τώρα ξέρετε ποιες πόλεις αποδίδουν καλύτερα.*
4. Αναζητήστε προμηθευτές σε αυτές τις πόλεις και βρείτε μερικούς που σας αρέσουν.
5. Παρακολουθήστε αυτούς τους προμηθευτές κάνοντας κλικ στο κουμπί `Actions > Supplier details > Track performance` στην κάρτα αποτελεσμάτων αναζήτησης *Δείτε την εικόνα παρακάτω*.
6. Αυτό δημιουργεί ένα νέο γράφημα για τον προμηθευτή που μπορείτε να παρακολουθείτε.
7. Συγκρίνετε την προηγούμενη απόδοση κάθε προμηθευτή μαζί με τη μέση τιμή και την προμήθεια που καταβάλλεται.
8. Αυτό θα σας δώσει μια χούφτα προμηθευτές για να ξεκινήσετε να παράγετε.
</Steps>

<Steps>

Με αυτή τη γνώση, έχετε μια αρκετά καλή κατανόηση του τι πουλάει, ποιος το αγοράζει και πότε.

Μπορείτε να γίνετε ακόμα πιο λεπτομερείς και να μάθετε ακριβώς τι κλείνεται επίσης. *Παράδειγμα: Οι συνεδρίες σπα πωλούνται καλά στο Βερολίνο, Γερμανία τα Σαββατοκύριακα.*

## Ανταγωνιστικό σύνολο

Στην ορολογία διαχείρισης ξενοδοχείων, ένα “*comp set*” (σύντμηση του competitive set) αναφέρεται σε μια ομάδα ξενοδοχείων που ένα συγκεκριμένο ξενοδοχείο θεωρεί ως τους κύριους ανταγωνιστές του.
Αυτά τα ξενοδοχεία επιλέγονται βάσει παραγόντων όπως η τοποθεσία, η αγορά-στόχος, οι παροχές και οι τιμές.
Το comp set χρησιμοποιείται για benchmarking και σύγκριση απόδοσης, βοηθώντας ένα ξενοδοχείο να αξιολογήσει τη θέση του στην αγορά.

Τα comp sets είναι διαθέσιμα σε όλα τα ξενοδοχεία στην πλατφόρμα Wink, χωρίς χρέωση. Μπορείτε να παρακολουθείτε την απόδοση οποιουδήποτε ξενοδοχείου στην πλατφόρμα μας χρησιμοποιώντας αυτά τα μετρικά:

- Επισκέψεις σε σελίδα
- Επισκέψεις σε κάρτα
- Επισκέψεις σε δείκτη χάρτη
- Κρατήσεις
- Διανυκτερεύσεις
- Αίθουσες συνεδριάσεων
- Εστιατόρια
- Σπα
- Δραστηριότητες
- Αξιοθέατα
- Τοποθεσίες
- Παροχές εντός δωματίου
- Επιπλέον υπηρεσίες
- Έσοδα δωματίων
- Μέσο έσοδο δωματίου
- Έσοδα αιθουσών συνεδριάσεων
- Έσοδα εστιατορίων
- Έσοδα σπα
- Έσοδα δραστηριοτήτων

Για να δημιουργήσετε ένα `comp set`, ακολουθήστε τα παρακάτω βήματα:

<Steps>
1. Μεταβείτε στο [Wink Studio](/el/studio/search) για να αναζητήσετε προμηθευτές.
2. Αναζητήστε άλλες ιδιοκτησίες με τις οποίες θέλετε να συγκριθείτε.
3. Παρακολουθήστε την ιδιοκτησία κάνοντας κλικ στο κουμπί `Actions > Supplier details > Track performance` στην κάρτα αποτελεσμάτων αναζήτησης *Δείτε την εικόνα παρακάτω*.
4. Αυτό δημιουργεί ένα νέο γράφημα για τον προμηθευτή που μπορείτε να παρακολουθείτε.
5. Μεταβείτε στα [Charts](https://extranet.wink.travel/analytics/list).
6. Για να ενημερώσετε το γράφημα ώστε να παρακολουθεί τα μετρικά που θέλετε, κάντε κλικ στο κουμπί `Actions > Update` για το νέο σας γράφημα.
6. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
</Steps>
