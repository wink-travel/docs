## Εύρεση πραγμάτων προς πώληση

Όταν ξεκινάτε ως συνεργάτης στο Wink, το πρώτο πράγμα που λέτε στον εαυτό σας είναι:

> Ας βγάλουμε λίγα χρήματα!

Αμέσως μετά, αναρωτιέσαι:

> Πώς; Τι; Πότε; Πού;

Μερικοί συνεργάτες γνωρίζουν ήδη τι θέλουν να πουλήσουν μέσω των καναλιών τους. Αν είστε νέοι στον χώρο και θέλετε να καταλάβετε ποιους προμηθευτές να προωθήσετε... τότε είναι που τα αναλυτικά στοιχεία μπορούν να σας φανούν χρήσιμα.

Ακολουθεί ένα παράδειγμα για το πώς μπορείτε να το κάνετε αυτό για πρώτη φορά:

<Steps>
  1. Μετάβαση σε [Στούντιο Wink](/studio/search) για αναζήτηση προμηθευτών.
  2. Δημιουργήστε ένα γράφημα που παρακολουθεί τις κρατήσεις σε πολλές χώρες. *Τώρα ξέρετε ποιες χώρες έχουν την καλύτερη απόδοση.*
  3. Δημιουργήστε ένα άλλο διάγραμμα για ορισμένες από τις πόλεις σε αυτές τις χώρες με τις καλύτερες επιδόσεις. *Τώρα ξέρετε ποιες πόλεις έχουν την καλύτερη απόδοση.*
  4. Αναζητήστε προμηθευτές σε αυτές τις πόλεις και βρείτε μερικούς που σας αρέσουν.
  5. Παρακολουθήστε αυτούς τους προμηθευτές κάνοντας κλικ στο `Actions > Supplier details > Track performance` κουμπί στην κάρτα αποτελεσμάτων αναζήτησης *Δείτε την εικόνα παρακάτω*.
  6. Αυτό δημιουργεί ένα νέο γράφημα για αυτόν τον προμηθευτή που μπορείτε να παρακολουθήσετε.
  7. Συγκρίνετε την προηγούμενη απόδοση για κάθε προμηθευτή μαζί με τη μέση τιμή και την προμήθεια που καταβάλλεται.
  8. Αυτό θα σας δώσει μια χούφτα προμηθευτών για τους οποίους μπορείτε να ξεκινήσετε την παραγωγή.
</Steps>

Τώρα, αλλάξτε περιβάλλον και ξεκινήστε να παρακολουθείτε τους ταξιδιώτες.

<Steps>
  1. Παρακολουθήστε ποιες χώρες και πόλεις κάνουν κράτηση για τα περισσότερα ταξίδια.
  2. Παρακολουθήστε τις ώρες που γίνονται κρατήσεις για αυτά τα ταξίδια.
</Steps>


Με αυτή τη γνώση, έχετε μια αρκετά καλή κατανόηση του τι πουλάει, ποιος το αγοράζει και πότε.

Μπορείτε να προχωρήσετε ακόμη πιο λεπτομερώς και να μάθετε ακριβώς τι έχει κρατηθεί.*Παράδειγμα: Οι συνεδρίες σπα πωλούνται καλά στο Βερολίνο της Γερμανίας κατά τη διάρκεια του Σαββατοκύριακου.*

## Ανταγωνιστικό σετ

Στην ορολογία της διοίκησης ξενοδοχείων, ένα «...*σύνολο συμπιεστών*«(συντομογραφία του competitive set) αναφέρεται σε μια ομάδα ξενοδοχείων που ένα συγκεκριμένο ξενοδοχείο θεωρεί ως τους κύριους ανταγωνιστές του.
Αυτά τα ξενοδοχεία επιλέγονται με βάση παράγοντες όπως η τοποθεσία, η αγορά-στόχος, οι παροχές και η τιμολόγηση.
Το comp set χρησιμοποιείται για συγκριτική αξιολόγηση και σύγκριση απόδοσης, βοηθώντας ένα ξενοδοχείο να αξιολογήσει τη δική του θέση στην αγορά.

Τα σύνολα αποδόσεων είναι διαθέσιμα σε όλα τα ξενοδοχεία, στην πλατφόρμα Wink, χωρίς χρέωση. Μπορείτε να παρακολουθείτε την απόδοση οποιουδήποτε ξενοδοχείου στην πλατφόρμα μας χρησιμοποιώντας αυτές τις μετρήσεις:

* Επισκέψεις σελίδας
* Επισκέψεις με κάρτα
* Επισκέψεις με δείκτες χάρτη
* Κρατήσεις
* Διανυκτερεύσεις σε δωμάτια
* Αίθουσες συσκέψεων
* Εστιατόρια
* Σπα
* Δραστηριότητες
* Θελγήτρα
* Μέρη
* Βοηθητικά μέσα στο δωμάτιο
* Πρόσθετα
* Έσοδα δωματίων
* Μέσο έσοδο δωματίου
* Έσοδα από αίθουσες συσκέψεων
* Έσοδα εστιατορίων
* Έσοδα από σπα
* Έσοδα από δραστηριότητες
* Έσοδα από αξιοθέατα
* Έσοδα από μέρη
* Έσοδα από βοηθητικά είδη εντός δωματίου

Για να δημιουργήσετε ένα`comp set`, ακολουθήστε τα εξής βήματα:

<Steps>
  1. Μετάβαση σε [Στούντιο Wink](/studio/search) για αναζήτηση προμηθευτών.
  2. Αναζητήστε άλλα ακίνητα με τα οποία θέλετε να συγκρίνετε τον εαυτό σας.
  3. Παρακολουθήστε την ιδιότητα κάνοντας κλικ στο `Actions > Supplier details > Track performance` κουμπί στην κάρτα αποτελεσμάτων αναζήτησης *Δείτε την εικόνα παρακάτω*.
  4. Αυτό δημιουργεί ένα νέο γράφημα για αυτόν τον προμηθευτή που μπορείτε να παρακολουθήσετε.
  5. Πήγαινε στο δικό σου [Γραφήματα](https://extranet.wink.travel/analytics/list).
  6. Για να ενημερώσετε το γράφημα ώστε να παρακολουθεί τις μετρήσεις που θέλετε, κάντε κλικ στο `Actions > Update` κουμπί για το νέο σας γράφημα.
  7. Κάντε κλικ στο `Save` κουμπί για να συνεχίσετε.
</Steps>
