Το**ΚΑΝΤΕ ΚΡΑΤΗΣΗ ΤΩΡΑ**Το κουμπί είναι ένας πανταχού παρών όρος στον κλάδο της φιλοξενίας που αναφέρεται στην ενσωμάτωση του ξενοδοχείου με έναν πάροχο που γνωρίζει τη διαθεσιμότητα και τις τιμές του καταλύματος και μπορεί να επιτρέψει στους ταξιδιώτες να κάνουν κράτηση δωματίου. Το κουμπί είναι ο απλούστερος τρόπος για να μπορούν τα ξενοδοχεία να πωλούν δωμάτια στον δικό τους ιστότοπο και με τους δικούς τους όρους.

*Πολλά ξενοδοχεία χρησιμοποιούν την πακέτα μηχανών κρατήσεων που προσφέρεται από τον channel manager τους. Συνιστούμε στα ξενοδοχεία να κάνουν κάποια έρευνα και να συγκρίνουν τις εγγενείς μηχανές κρατήσεων πριν δεσμευτούν σε οποιαδήποτε συγκεκριμένη.*

## Ακροατήριο

Αυτό το άρθρο απευθύνεται κυρίως σε ξενοδοχεία που θέλουν να εγκαταστήσουν τη δωρεάν μηχανή κρατήσεων υψηλής μετατροπής στον ιστότοπό τους.
Αλλά... είναι επίσης χρήσιμο ανάγνωσμα για όποιον θέλει να χρησιμοποιήσει τις τεχνολογίες μας στους ιστότοπούς του.

## μέσω συνδέσμου

Για την απλούστερη ενσωμάτωση, συνιστούμε τη χρήση ενός[Σύνδεσμος με δυνατότητα κοινής χρήσης](/studio/shareable-links).

Ακολουθούν τα βήματα για να το κάνετε αυτό:

<Steps>
  1. Μεταβείτε στη διεύθυνση https://extranet.wink.travel
  2. Κλικ `Sign-In / Register` στην επάνω δεξιά γωνία και συνδεθείτε με τον λογαριασμό χρήστη σας.
  3. Επιλέξτε την ιδιότητα με την οποία θέλετε να εργαστείτε από το αναπτυσσόμενο μενού επάνω αριστερά.
  4. Κάντε κλικ στο `Booking engine` στη γραμμή πλοήγησης.
  5. Η σελίδα θα σας δείξει τον σύνδεσμό σας.
  6. Δώστε αυτόν τον σύνδεσμο στον διαχειριστή σας και χρησιμοποιήστε τον για το δικό σας **ΚΑΝΤΕ ΚΡΑΤΗΣΗ ΤΩΡΑ** κουμπί.
  7. Είσαι ✅.
</Steps>


### Προχωρημένος

Ο σύνδεσμος συνοδεύεται ήδη από ένα δρομολόγιο ενσωματωμένο στη διεύθυνση URL, το οποίο προέρχεται από την επιλεγμένη σας διεύθυνση.`customization`.
Μπορείτε να προσαρμόσετε περαιτέρω τον σύνδεσμό σας ως εξής:

* Αλλαγή της επιλεγμένης προσαρμογής Ή
* Παράκαμψη του δρομολογίου

Σε αυτό το άρθρο, θα επικεντρωθούμε στο τελευταίο, καθώς σας δίνει μεγαλύτερη ελευθερία να επιτρέψετε στους ταξιδιώτες να επιλέξουν ένα δρομολόγιο στον ιστότοπό σας και να το προωθήσουν στη μηχανή κρατήσεων του Wink.

Δείτε πώς μπορείτε να μεταβιβάσετε το δικό σας δρομολόγιο μαζί με τη διεύθυνση URL.

Έγκυρες παράμετροι αιτήματος:

* **sd**Για να προσθέσετε μια ημερομηνία έναρξης:*sd=24-08-2024*
* **ν**Για να προσθέσετε τον αριθμό των διανυκτερεύσεων:*n=1*Ή
* **ed**Χρήση ημερομηνίας λήξης:*ed=2024-08-25*
* **rc**Επισκέπτες:*rc=a2*(Ενήλικες: 2)
* **μεγάλο**Αίτημα για γλώσσα εμφάνισης:*l=es*
* **ντο**Αίτημα για νόμισμα εμφάνισης:*c=EUR*
* **π**Προσθήκη προσφοράς:*p=ABC123*

Η διεύθυνση URL θα έχει την εξής μορφή:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2\&l=es\&c=EUR

Για πιο προηγμένες περιπτώσεις χρήσης, μπορείτε επίσης να ζητήσετε πολλαπλές διαμορφώσεις δωματίων χρησιμοποιώντας το`+`σημάδι:

Παράδειγμα:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2-ca16q1-ca8q1+a2\&l=es\&c=EUR

* Τύπος δωματίου 1 = Ενήλικες: 2, Παιδιά: 2 = 16 ετών + 8 ετών.
* Τύπος δωματίου 2 = Ενήλικες: 2.

### Παράδειγμα

*Αυτό το δείγμα υποθέτει ότι χρησιμοποιείτε[Bootstrap](https://getbootstrap.com)ως την κύρια βιβλιοθήκη CSS σας.*

```html
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Book a room"

</a>
```

### Άλλα οφέλη

Υπάρχουν και άλλα οφέλη από τη χρήση ενός από τους συνδέσμους μας ως`book now`κουμπί.

* **Φιλικό προς την κοινωνία**Ο σύνδεσμος μπορεί εύκολα να κοινοποιηθεί στα μέσα κοινωνικής δικτύωσης και εμφανίζεται`dynamic pricing`πάνω από την εικόνα που επιλέξατε.
* **Μικρός**Η διεύθυνση URL είναι εύκολα κοινοποιήσιμη και μπορείτε να τη χρησιμοποιήσετε σε μέρη όπως η διεύθυνση URL του βιογραφικού σας στο Instagram.

## μέσω Ενσωμάτωσης

Ορισμένα ξενοδοχεία θέλουν λίγο περισσότερη λειτουργικότητα και μια βαθύτερη ενσωμάτωση για τον ιστότοπό τους. Αυτό το παράδειγμα θα καλύψει την ενσωμάτωση ενός[Κάρτα](/studio/cards)στην ιστοσελίδα τους.

Ακολουθούν τα βήματα για να το κάνετε αυτό:

<Steps>
  1. Μεταβείτε στη διεύθυνση https://studio.wink.travel
  2. Κλικ `Sign-In / Register` στην επάνω δεξιά γωνία και συνδεθείτε με τον λογαριασμό χρήστη σας.
  3. Επιλέξτε την ιδιότητα με την οποία θέλετε να εργαστείτε από την επάνω αριστερή γωνία.
  4. Κάντε κλικ στο `Search` από την κύρια γραμμή πλοήγησης.
  5. Πληκτρολογήστε το όνομα του ακινήτου σας και κάντε κλικ στο κουμπί αναζήτησης.
  6. Κάντε κλικ στο `Actions` σύνδεσμο στην κάρτα αποτελεσμάτων αναζήτησης και επιλέξτε `Create > Make a Card`.
  7. Αυτό θα δημιουργήσει ένα `Card`.
  8. Θα εμφανιστεί ένα μήνυμα στην επάνω δεξιά γωνία που θα σας ενημερώνει ότι η κάρτα σας είναι έτοιμη και αν θέλετε να την προσαρμόσετε περαιτέρω.
  9. Κάντε κλικ στο `Tools > Cards` από την κύρια γραμμή πλοήγησης και θα δείτε τη νέα σας κάρτα.
  10. Κάντε κλικ στο `Actions` σύνδεσμο στην κάρτα συνδέσμου σας και κάντε κλικ στο `Embed`.
  11. Αυτό ανοίγει ένα νέο παράθυρο με δείγμα κώδικα HTML που μπορείτε να αντιγράψετε.
  12. Για να ανακτήσετε το δικό σας `Client-ID`, πηγαίνετε στο [Εφαρμογές](/developers/applications).
  13. Για να ανακτήσετε το δικό σας `Configuration-ID`, πηγαίνετε στο [Προσαρμογή](/studio/customization)
  14. Είσαι ✅.
</Steps>


Δείγμα HTML ενσωματωμένης κάρτας:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*Το παραπάνω δείγμα HTML δεν περιλαμβάνει τα απαιτούμενα Wink CSS και Javascript, μόνο την κάρτα.*
