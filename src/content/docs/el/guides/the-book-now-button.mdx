Το **ΚΟΥΜΠΙ ΚΡΑΤΗΣΗΣ ΤΩΡΑ** είναι ένας διαδεδομένος όρος στη βιομηχανία φιλοξενίας που αναφέρεται στην ενσωμάτωση του ξενοδοχείου με έναν πάροχο που γνωρίζει τη διαθεσιμότητα και τις τιμές του καταλύματος και μπορεί να επιτρέψει στους ταξιδιώτες να κάνουν κράτηση δωματίου. Το κουμπί είναι ο απλούστερος τρόπος για τα ξενοδοχεία να μπορούν να πουλάνε δωμάτια στην ιστοσελίδα τους και με τους δικούς τους όρους.

*Πολλά ξενοδοχεία χρησιμοποιούν τη συνοδευτική μηχανή κρατήσεων που προσφέρεται από τον διαχειριστή καναλιών τους. Συνιστούμε στα ξενοδοχεία να κάνουν κάποια έρευνα και να συγκρίνουν τις εγγενείς μηχανές κρατήσεων πριν δεσμευτούν σε κάποια συγκεκριμένη.*

## Κοινό

Αυτό το άρθρο απευθύνεται κυρίως σε ξενοδοχεία που θέλουν να εγκαταστήσουν τη δωρεάν, υψηλής μετατροπής, μηχανή κρατήσεων μας στην ιστοσελίδα τους.  
Αλλά... είναι επίσης χρήσιμο ανάγνωσμα για όποιον θέλει να χρησιμοποιήσει τις τεχνολογίες μας στις ιστοσελίδες του.

## μέσω Συνδέσμου

Για την απλούστερη ενσωμάτωση, προτείνουμε τη χρήση ενός [Shareable Link](/el/studio/shareable-links).

Ακολουθούν τα βήματα για να το κάνετε:

<Steps>
1. Μεταβείτε στο https://extranet.wink.travel  
2. Κάντε κλικ στο `Sign-In / Register` στην πάνω δεξιά γωνία και συνδεθείτε με τον λογαριασμό σας.  
3. Επιλέξτε το κατάλυμα με το οποίο θέλετε να εργαστείτε από το αναπτυσσόμενο μενού πάνω αριστερά.  
4. Κάντε κλικ στο `Booking engine` στη γραμμή πλοήγησης.  
5. Η σελίδα θα σας δείξει τον σύνδεσμό σας.  
6. Δώστε αυτόν τον σύνδεσμο στον webmaster σας και χρησιμοποιήστε τον για το κουμπί **ΚΡΑΤΗΣΗ ΤΩΡΑ**.  
7. Είστε ✅.
</Steps>

### Προχωρημένο

Ο σύνδεσμος ήδη περιλαμβάνει ένα δρομολόγιο ενσωματωμένο στο URL, που προέρχεται από την επιλεγμένη σας `customization`.  
Μπορείτε να προσαρμόσετε περαιτέρω τον σύνδεσμό σας με:

- Αλλαγή της επιλεγμένης παραμετροποίησης Ή  
- Παράκαμψη του δρομολογίου

Σε αυτό το άρθρο, θα εστιάσουμε στο δεύτερο καθώς σας δίνει μεγαλύτερη ελευθερία να αφήσετε τους ταξιδιώτες να επιλέξουν ένα δρομολόγιο στην ιστοσελίδα σας και να το προωθήσουν στη μηχανή κρατήσεων του Wink.

Δείτε πώς περνάτε το δικό σας δρομολόγιο μαζί με το URL.

Έγκυροι παράμετροι αιτήματος:

- **sd** Για να προσθέσετε ημερομηνία έναρξης: *sd=2024-08-24*  
- **n** Για να προσθέσετε αριθμό διανυκτερεύσεων: *n=1* Ή  
- **ed** Χρήση ημερομηνίας λήξης: *ed=2024-08-25*  
- **rc** Επισκέπτες: *rc=a2* (Ενήλικες: 2)  
- **l** Ζητήστε γλώσσα εμφάνισης: *l=es*  
- **c** Ζητήστε νόμισμα εμφάνισης: *c=EUR*  
- **p** Προσθέστε προσφορά: *p=ABC123*

Το URL θα μοιάζει ως εξής:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2&l=es&c=EUR

Για πιο προχωρημένες περιπτώσεις, μπορείτε επίσης να ζητήσετε πολλαπλές διαμορφώσεις δωματίων χρησιμοποιώντας το σύμβολο `+`:

Παράδειγμα:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2-ca16q1-ca8q1+a2&l=es&c=EUR

- Τύπος δωματίου 1 = Ενήλικες: 2, Παιδιά: 2 = 16 ετών + 8 ετών.  
- Τύπος δωματίου 2 = Ενήλικες: 2.

### Παράδειγμα

Δείγμα HTML κουμπιού **ΚΡΑΤΗΣΗ ΤΩΡΑ**:

*Αυτό το δείγμα υποθέτει ότι χρησιμοποιείτε το [Bootstrap](https://getbootstrap.com) ως την κύρια βιβλιοθήκη CSS σας.*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Κάντε κράτηση δωματίου"
>
ΚΡΑΤΗΣΗ ΤΩΡΑ
</a>
```

### Άλλα οφέλη

Υπάρχουν και άλλα οφέλη από τη χρήση ενός από τους συνδέσμους μας ως το κουμπί `book now`.

- **Φιλικό προς τα κοινωνικά δίκτυα** Ο σύνδεσμος μπορεί εύκολα να κοινοποιηθεί στα κοινωνικά δίκτυα και εμφανίζει `δυναμική τιμολόγηση` πάνω στην εικόνα που επιλέξατε.  
- **Σύντομο** Το URL είναι πολύ εύκολο στη διαμοίραση και μπορείτε να το χρησιμοποιήσετε σε μέρη όπως το Instagram Bio URL σας.

## μέσω Ενσωμάτωσης

Ορισμένα ξενοδοχεία θέλουν λίγη περισσότερη λειτουργικότητα και βαθύτερη ενσωμάτωση στην ιστοσελίδα τους. Αυτό το παράδειγμα καλύπτει την ενσωμάτωση μιας [Κάρτας](/el/studio/cards) στην ιστοσελίδα τους.

Ακολουθούν τα βήματα για να το κάνετε:

<Steps>
1. Μεταβείτε στο https://studio.wink.travel  
2. Κάντε κλικ στο `Sign-In / Register` στην πάνω δεξιά γωνία και συνδεθείτε με τον λογαριασμό σας.  
3. Επιλέξτε το κατάλυμα με το οποίο θέλετε να εργαστείτε από την πάνω αριστερή γωνία.  
4. Κάντε κλικ στο `Search` από τη βασική γραμμή πλοήγησης.  
6. Πληκτρολογήστε το όνομα του καταλύματός σας και πατήστε το κουμπί αναζήτησης.  
7. Κάντε κλικ στον σύνδεσμο `Actions` στην κάρτα αποτελεσμάτων αναζήτησης και επιλέξτε `Create > Make a Card`.  
8. Αυτό θα δημιουργήσει μια `Κάρτα`.  
9. Θα εμφανιστεί ένα μήνυμα στην πάνω δεξιά γωνία που θα σας ενημερώνει ότι η κάρτα σας είναι έτοιμη και αν θέλετε να την προσαρμόσετε περαιτέρω.  
10. Κάντε κλικ στο `Tools > Cards` από τη βασική γραμμή πλοήγησης και θα δείτε την καινούργια σας κάρτα.  
11. Κάντε κλικ στον σύνδεσμο `Actions` στην κάρτα σας και επιλέξτε `Embed`.  
12. Αυτό ανοίγει ένα νέο παράθυρο με δείγμα κώδικα HTML που μπορείτε να αντιγράψετε.  
13. Για να βρείτε το `Client-ID` σας, μεταβείτε στα [Applications](/el/developers/applications).  
14. Για να βρείτε το `Configuration-ID` σας, μεταβείτε στο [Customization](/el/studio/customization)  
14. Είστε ✅.
</Steps>

Δείγμα ενσωματωμένης κάρτας HTML:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*Το παραπάνω δείγμα HTML δεν περιλαμβάνει το απαιτούμενο CSS και Javascript του Wink· μόνο την κάρτα.*

Ο χρήστης τώρα βλέπει δυναμική τιμολόγηση, μπορεί να αλλάξει το δρομολόγιο και να κάνει κλικ για κράτηση απευθείας στην ιστοσελίδα σας. Το κουμπί κράτησης τον οδηγεί στη σελίδα προορισμού του καταλύματός σας στο Wink για να ολοκληρώσει την πληρωμή.
