Δεν τιμολογείται κάθε μέρα το ίδιο. Εκδηλώσεις, αργίες, εκτός αιχμής περίοδοι και κενά της τελευταίας στιγμής απαιτούν διαφορετική τιμολόγηση. Αυτός ο οδηγός σας δείχνει πώς να χρησιμοποιείτε ειδικές τιμές και περιορισμούς στα σχέδια τιμών για να διαχειριστείτε αυτές τις περιπτώσεις.

## Τιμολόγηση εκδηλώσεων

Όταν μια διάσκεψη, φεστιβάλ ή τοπική εκδήλωση αυξάνει τη ζήτηση, θα θελήσετε να αυξήσετε τις τιμές για αυτές τις ημερομηνίες.

<Steps>
1. Μεταβείτε στο `Monetize > Special Rates` στο Extranet σας.
2. Κάντε κλικ στο `Create a new special rate`.
3. Επιλέξτε την κύρια τιμή και τον τύπο δωματίου που θέλετε να αντικαταστήσετε.
4. Ορίστε το εύρος ημερομηνιών που καλύπτει την περίοδο της εκδήλωσης.
5. Εισάγετε τις προσαρμοσμένες (υψηλότερες) ημερήσιες τιμές.
6. Κάντε κλικ στο `Save`.
</Steps>

Η ειδική τιμή αντικαθιστά την κύρια τιμή μόνο για αυτές τις ημερομηνίες. Η κανονική τιμολόγησή σας παραμένει αμετάβλητη εκτός του παραθύρου της εκδήλωσης.

## Εποχιακές προσαρμογές

Για μεγαλύτερες περιόδους (υψηλή περίοδος, ενδιάμεση περίοδος, χαμηλή περίοδος), μπορείτε είτε:

- **Να χρησιμοποιήσετε ειδικές τιμές** — Δημιουργήστε αντικαταστάσεις για κάθε εποχή με προσαρμοσμένες τιμές.
- **Να ενημερώσετε τις κύριες τιμές** — Προσαρμόστε απευθείας τις βασικές τιμές για αυτές τις περιόδους.

Οι ειδικές τιμές είναι καλύτερες για προσωρινές προσαρμογές, ενώ η ενημέρωση των κύριων τιμών έχει νόημα για προβλέψιμα ετήσια μοτίβα.

## Τιμές τελευταίας στιγμής

Για να γεμίσετε μη πουλημένα δωμάτια κοντά στην ημερομηνία άφιξης:

<Steps>
1. Δημιουργήστε ένα νέο **σχέδιο τιμών** με περιορισμό προκράτησης (π.χ., πρέπει να γίνει κράτηση το πολύ 3 ημέρες νωρίτερα).
2. Συνδυάστε το με μια **ειδική τιμή** που προσφέρει έκπτωση για τις στοχευμένες ημερομηνίες.
3. Αυτό το σχέδιο τιμών θα εμφανίζεται μόνο σε ταξιδιώτες που κάνουν κράτηση μέσα στο παράθυρο της τελευταίας στιγμής.
</Steps>

## Ημερομηνίες αποκλεισμού

Για να αποκλείσετε συγκεκριμένες ημερομηνίες από ένα συγκεκριμένο σχέδιο τιμών (π.χ., να μην υπάρχουν προωθητικές τιμές κατά την περίοδο αιχμής):

<Steps>
1. Μεταβείτε στο `Monetize > Rate Plans` και επιλέξτε το σχέδιο τιμών.
2. Πλοηγηθείτε στην ενότητα **Restrictions**.
3. Προσθέστε έναν περιορισμό **Sell date** για να περιορίσετε πότε η τιμή είναι διαθέσιμη για κράτηση.
4. Ή χρησιμοποιήστε περιορισμούς **Stay date** για να περιορίσετε πότε καλύπτεται η τιμή.
</Steps>

Για παράδειγμα, μπορείτε να περιορίσετε το σχέδιο τιμών "Early Bird" ώστε να μην είναι διαθέσιμο για κρατήσεις κατά τον Δεκέμβριο (περίοδο αιχμής), ενώ το κανονικό σας σχέδιο τιμών παραμένει ανοιχτό.

## Συνδυασμός περιορισμών

Οι περιορισμοί μπορούν να συνδυαστούν για ακριβή έλεγχο:

| Περιορισμός | Τι ελέγχει |
| -- | -- |
| **Sell dates** | Πότε μπορεί να γίνει κράτηση της τιμής |
| **Stay dates** | Ποιες ημερομηνίες διαμονής καλύπτει η τιμή |
| **Available days** | Ποιες ημέρες της εβδομάδας εμφανίζεται η τιμή |
| **Arrival days** | Ποιες ημέρες πρέπει να φτάσει ο επισκέπτης |
| **Departure days** | Ποιες ημέρες πρέπει να αναχωρήσει ο επισκέπτης |
| **Required days** | Ο επισκέπτης πρέπει να μείνει τουλάχιστον μία από αυτές τις ημέρες |
| **Advance booking** | Πόσο νωρίτερα πρέπει να γίνει η κράτηση |
| **Length of stay** | Ελάχιστες και μέγιστες διανυκτερεύσεις |
