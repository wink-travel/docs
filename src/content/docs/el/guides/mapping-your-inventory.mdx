Όταν προσθέτετε απόθεμα στην ιδιοκτησία σας στο Extranet — δωμάτια, εστιατόρια, σπα, αίθουσες συσκέψεων, δραστηριότητες και άλλα — αυτό το απόθεμα γίνεται διαθέσιμο σε όλο το οικοσύστημα Wink. Αυτός ο οδηγός εξηγεί πώς συνδέονται τα μέρη.

## Η ροή του αποθέματος

```
Extranet (εσείς δημιουργείτε) → Μηχανή Κρατήσεων (οι ταξιδιώτες βλέπουν) → Studio (οι συνεργάτες πωλούν)
```

1. **Δημιουργείτε απόθεμα** στο Extranet — τύπους δωματίων, εγκαταστάσεις και εμπειρίες.
2. **Οι ταξιδιώτες περιηγούνται** σε αυτό το απόθεμα στη μηχανή κρατήσεων μέσω κοινόχρηστων συνδέσμων, καρτών ή άμεσων αναζητήσεων.
3. **Οι συνεργάτες ανακαλύπτουν** το απόθεμά σας στο Studio, δημιουργούν κάρτες και πλέγματα και το πωλούν στο κοινό τους.

## Τύποι δωματίων

Οι τύποι δωματίων είναι ο πυρήνας του αποθέματός σας. Κάθε τύπος δωματίου χρειάζεται:

- Όνομα και περιγραφή.
- Διαμόρφωση κρεβατιών.
- Μέγιστη χωρητικότητα (ενήλικες και παιδιά).
- Παροχές ειδικές για το δωμάτιο.
- Φωτογραφίες.
- Τουλάχιστον ένα σχέδιο τιμολόγησης με βασικές τιμές.

Χωρίς τιμές, ένας τύπος δωματίου υπάρχει αλλά δεν μπορεί να γίνει κράτηση.

## Εγκαταστάσεις και εμπειρίες

Πέρα από τα δωμάτια, μπορείτε να καταχωρήσετε:

| Τύπος αποθέματος | Τι είναι | Παράδειγμα |
| -- | -- | -- |
| **Εστιατόριο** | Φαγητό στον χώρο | "Rooftop Bar & Grill" |
| **Σπα** | Υπηρεσίες ευεξίας | "Serenity Spa - 60min μασάζ" |
| **Αίθουσα συσκέψεων** | Χώρος εκδηλώσεων και συνεδρίων | "Boardroom A - 20 άτομα" |
| **Δραστηριότητα** | Πράγματα να κάνετε στην ή κοντά στην ιδιοκτησία | "Ξενάγηση με καταδύσεις" |
| **Αξιοθέατο** | Σημαντικά κοντινά μέρη | "Εθνικό Μουσείο - 500μ" |
| **Τοποθεσία** | Ενδιαφέροντα σημεία | "Τοπική αγορά" |
| **Πρόσθετο** | Επιπλέον που αγοράζονται με δωμάτιο | "Μεταφορά από/προς αεροδρόμιο" |

Κάθε τύπος εγκατάστασης έχει τη δική του ενότητα στο Extranet. Οι συνεργάτες μπορούν να πουλήσουν οποιοδήποτε από αυτά ξεχωριστά — ένας ταξιδιώτης μπορεί να ανακαλύψει την ιδιοκτησία σας μέσω μιας καταχώρησης σπα αντί για αναζήτηση δωματίου.

## Πώς χρησιμοποιούν οι συνεργάτες το απόθεμά σας

Στο Studio, οι συνεργάτες μπορούν:

- **Να αναζητήσουν** την ιδιοκτησία σας ή μεμονωμένα αντικείμενα αποθέματος.
- **Να δημιουργήσουν κάρτες** — Μια μεμονωμένη κάρτα κρατήσιμη για δωμάτιο, θεραπεία σπα ή δραστηριότητα.
- **Να δημιουργήσουν πλέγματα** — Μια συλλογή καρτών που εμφανίζονται μαζί.
- **Να δημιουργήσουν χάρτες** — Σημάνσεις βάσει τοποθεσίας που δείχνουν κρατήσιμο απόθεμα.
- **Να μοιραστούν συνδέσμους** — Άμεσες διευθύνσεις URL για την ιδιοκτησία σας ή συγκεκριμένα αντικείμενα.

Όσο πιο πλήρες και ποικίλο είναι το απόθεμά σας, τόσους περισσότερους τρόπους έχουν οι συνεργάτες να προωθήσουν την ιδιοκτησία σας.

## Συμβουλές

- **Να είστε αναλυτικοί** — Καταχωρήστε τα πάντα που προσφέρει η ιδιοκτησία σας. Κάθε αντικείμενο είναι μια πιθανή είσοδος για κράτηση.
- **Χρησιμοποιήστε καλές φωτογραφίες** — Κάθε αντικείμενο αποθέματος πρέπει να έχει τουλάχιστον μια εικόνα υψηλής ποιότητας.
- **Διατηρήστε τις περιγραφές φρέσκες** — Ενημερώνετε τις περιγραφές όταν αλλάζετε προσφορές ή εποχιακά.
- **Τιμολογήστε τα πάντα** — Το απόθεμα χωρίς τιμές μπορεί να περιηγηθεί αλλά όχι να κλειστεί.
