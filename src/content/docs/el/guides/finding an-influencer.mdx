Το Wink διευκολύνει τις ιδιοκτησίες να βρουν και να συνδεθούν με influencers, δημιουργούς περιεχομένου και bloggers από όλο τον κόσμο.

**Πριν το Wink**

Έτσι ήταν η σύνδεση με influencers πριν εμφανιστεί η πλατφόρμα Wink:

- Μια ιδιοκτησία αφιέρωνε χρόνο προσπαθώντας να βρει τον influencer.
- Έδινε στον influencer μια δωρεάν διανυκτέρευση [ή κάτι παρόμοιο].
- Η ιδιοκτησία 🤞🏻 ελπίζε σε αύξηση των κρατήσεων. Δεν υπήρχε τρόπος να γνωρίζουν αν η ανάρτηση του influencer μετέτρεψε σε κρατήσεις.

**Μετά το Wink**

Η πλατφόρμα Wink απλοποιεί και προσθέτει αξία στη διαδικασία αυτή:

- Μια ιδιοκτησία μπορεί να αναζητήσει και να συνδεθεί με influencers στο Wink.
- Ένας influencer μπορεί επίσης να αναζητήσει και να συνδεθεί με ιδιοκτησίες στο Wink.
- Και τα δύο μέρη μπορούν να παρακολουθούν τις πωλήσεις που προήλθαν από αυτό το κανάλι.
- Το Wink πληρώνει κάθε μέρος.

Ως ιδιοκτησία, μπορείτε ακόμα να επιλέξετε να διαπραγματευτείτε κάποια οφέλη προς τον influencer όταν επισκεφθεί τις εγκαταστάσεις σας. 
Ωστόσο, το μέλλον είναι στη συνεργασία μαζί τους όπως θα κάνατε με ένα online ταξιδιωτικό πρακτορείο. 
Ο Michael Jordan άρχισε πραγματικά να βγάζει χρήματα όταν έλαβε δικαιώματα από την κυκλοφορία των Nike Air Jordans. 
Παρομοίως, οι influencers μας δικαιούνται ένα κομμάτι της πίτας και μπορούν να κερδίζουν παθητικό εισόδημα για χρόνια από έναν μόνο σύνδεσμο. Αυτό έχει πραγματική αξία!

## Πώς λειτουργεί

### Δίκτυο Wink

Πριν ξεκινήσετε να ψάχνετε για influencers, να θυμάστε ότι ο καθένας μπορεί να πουλήσει το γενικά διαθέσιμο απόθεμά σας. 
Αυτό είναι το απόθεμα που κάνετε διαθέσιμο μέσω του καναλιού πωλήσεων [Wink Network](/el/extranet/network). 
Εμφανίζεται όταν οι συνεργάτες αναζητούν απόθεμα για πώληση στο [Wink Studio](/el/studio/what-is-studio).
Μπορείτε επίσης να παρακολουθείτε οποιονδήποτε συνεργάτη που πουλά απόθεμα από το [Wink Network](/el/extranet/network).

Άρα... αν δεν θέλετε να κάνετε καμία επιπλέον δουλειά, κάντε το εξής:

<Steps>
1. Δημιουργήστε ένα ωραίο προφίλ για την ιδιοκτησία σας στο Wink.
2. Κάντε διαθέσιμα πολλά άλλα πράγματα που μπορούν να χρησιμοποιηθούν για upsell στους επισκέπτες.
3. Διαμορφώστε το κανάλι πωλήσεων [Wink Network](/el/extranet/network) ώστε να είναι ελκυστικό για συνεργάτες που αναζητούν απόθεμα για πώληση. *Κατά προεπιλογή, πιστεύουμε ότι είναι ήδη αρκετά ελκυστικό*.
</Steps>

:::tip[Συμβουλή ειδικού]
Αντιμετωπίστε το Wink ακριβώς αντίθετα από το πώς θα αντιμετωπίζατε μια σχέση με OTA. Δεν είμαστε **ΕΔΩ** για να σας βοηθήσουμε να πουλήσετε το τελευταίο δωμάτιο στο σπίτι σε μειωμένη τιμή.
Αφήστε τους συνεργάτες να διηγηθούν την ιστορία για εσάς και δώστε τους το απόθεμα που χρειάζονται για να δημιουργήσουν μια πραγματικά αξέχαστη εμπειρία για το κοινό τους.
:::

Για να ξεχωρίσετε και να προσελκύσετε τη νέα γενιά ταξιδιωτών, πρέπει να σκεφτείτε τη σελίδα προορισμού της ιδιοκτησίας σας στο Wink ως την `ψηφιοποιημένη έκδοση` της ιδιοκτησίας και των γύρω περιοχών της.
Τι μπορεί να πουληθεί, να νοικιαστεί, να κλειστεί, να ανταλλαχθεί, να μοιραστεί στην περιοχή σας που κατέχετε ή που μπορείτε να διευκολύνετε; Αυτή είναι η στιγμή σας να γίνετε δημιουργικοί και να σκεφτείτε **έξω από το κουτί**.

### Εξερεύνηση

Αυτή η ενότητα καλύπτεται στο [Extranet > Network](/el/extranet/network/#explore-network).

Η σελίδα σας επιτρέπει να αναζητήσετε όλους τους διαφορετικούς τύπους συνεργατών που είναι ήδη στην πλατφόρμα και σας δίνει μια σύντομη περιγραφή τους καθώς και την προηγούμενη απόδοσή τους.

### Πρόσκληση

Αν η ιδιοκτησία σας ήδη συνεργάζεται με influencers **Ή** θέλετε να συνεργαστείτε με γνωστές προσωπικότητες **ΚΑΙ** θέλετε να μπορείτε να παρακολουθείτε την απόδοση της επένδυσής σας. 
Πείτε τους να δημιουργήσουν απευθείας λογαριασμό στο Wink ή επικοινωνήστε μαζί μας και θα τους στείλουμε πρόσκληση.

### Άμεση Επικοινωνία

Μερικές φορές, θέλετε να δημιουργήσετε μια άμεση σχέση με έναν από τους συνεργάτες μας.

Το κάνετε για να...

- Τους προσφέρετε αποκλειστικό απόθεμα.
- Τους δώσετε μοναδική προμήθεια.
- Τους δώσετε μοναδική τιμή μέλους.
- Τους δώσετε μοναδική προσφορά.

:::tip[Συμβουλή ειδικού]
Δεν υπάρχουν άλλοι λόγοι για να δημιουργήσετε άμεση σύνδεση πέρα από τους παραπάνω.
:::

Μόλις δημιουργήσετε ένα νέο, άμεσο, κανάλι πωλήσεων για αυτούς, το απόθεμά σας θα εμφανιστεί στο [Wink Studio](/el/studio/what-is-studio) με μια `Ετικέτα Άμεσης` επάνω από το απόθεμά σας.
