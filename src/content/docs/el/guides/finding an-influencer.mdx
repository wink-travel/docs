Το Wink διευκολύνει τα ακίνητα να βρίσκουν και να συνδέονται με influencers, δημιουργούς περιεχομένου και bloggers από όλο τον κόσμο.

**Πριν το Wink**

Δείτε πώς ήταν η επικοινωνία με influencers πριν εμφανιστεί η πλατφόρμα Wink:

* Ένα ακίνητο θα ξόδευε χρόνο προσπαθώντας να βρει τον influencer.
* Θα έδιναν στον influencer ένα δωρεάν δωμάτιο τη νύχτα \[ή κάτι παρόμοιο].
* Το κατάλυμα θα 🤞🏻 και θα ήλπιζε σε αύξηση των κρατήσεων. Δεν θα υπήρχε τρόπος να καταλάβουμε αν η ανάρτηση του influencer μετατράπηκε όντως σε κρατήσεις.

**Μετά το Wink**

Η πλατφόρμα Wink απλοποιεί και προσθέτει αξία στην παραπάνω διαδικασία:

* Ένα κατάλυμα μπορεί να αναζητήσει και να συνδεθεί με influencers στο Wink.
* Ένας influencer μπορεί επίσης να αναζητήσει και να συνδεθεί με ακίνητα στο Wink.
* Και τα δύο μέρη μπορούν να παρακολουθούν τις πωλήσεις που προήλθαν μέσω αυτού του καναλιού.
* Ο Γουίνκ πληρώνει κάθε ομάδα.

Ως ιδιοκτήτης, μπορείτε ακόμα να επιλέξετε να διαπραγματευτείτε κάποιο είδος οφέλους για τον influencer όταν επισκέπτεται τις εγκαταστάσεις σας.
Ωστόσο, το μέλλον βρίσκεται στη συνεργασία μαζί του, όπως θα κάνατε με ένα διαδικτυακό ταξιδιωτικό γραφείο.
Ο Michael Jordan άρχισε να βγάζει χρήματα μόνο όταν έλαβε δικαιώματα από την κυκλοφορία των Nike Air Jordans.
Ομοίως, οι influencers μας δικαιούνται ένα κομμάτι της πίτας και μπορούν να κερδίσουν ένα παθητικό εισόδημα για τα επόμενα χρόνια για έναν μόνο σύνδεσμο. Αυτό έχει πραγματική αξία!

## Πώς λειτουργεί

### Δίκτυο Wink

Πριν ξεκινήσετε να αναζητάτε influencers, λάβετε υπόψη ότι ο καθένας μπορεί να πουλήσει το γενικά διαθέσιμο απόθεμά σας.
Πρόκειται για απόθεμα που διαθέτετε μέσω του[Δίκτυο Wink](/extranet/network)κανάλι πωλήσεων.
Εμφανίζεται όταν οι συνεργάτες αναζητούν απόθεμα για πώληση[Στούντιο Wink](/studio/what-is-studio).
Μπορείτε επίσης να παρακολουθείτε οποιαδήποτε συνεργαζόμενη εταιρεία που πουλάει απόθεμα από[Δίκτυο Wink](/extranet/network).

Λοιπόν... αν δεν θέλετε να κάνετε καμία προπόνηση με τα πόδια, δείτε τι μπορείτε να κάνετε:

<Steps>
  1. Δημιουργήστε ένα όμορφο προφίλ για το ακίνητό σας στο Wink.
  2. Διαθέστε πολλά άλλα πράγματα που μπορούν να χρησιμοποιηθούν για να προωθήσετε προϊόντα προς τους επισκέπτες.
  3. Διαμορφώστε το [Δίκτυο Wink](/extranet/network) κανάλι πωλήσεων που να είναι ελκυστικό για συνεργάτες που αναζητούν απόθεμα προς πώληση. *Από προεπιλογή, πιστεύουμε ότι είναι ήδη αρκετά ελκυστικό*.
</Steps>


:::tip\[Συμβουλή επαγγελματία]
Φερθείτε στον Wink ακριβώς το αντίθετο από αυτό που θα φερόσασταν σε μια σχέση OTA. Είμαστε**ΔΕΝ**Είμαστε εδώ για να σας βοηθήσουμε να πουλήσετε το τελευταίο δωμάτιο του σπιτιού σε μειωμένη τιμή.
Αφήστε τους συνεργάτες να αφηγηθούν την ιστορία για εσάς και να τους δώσουν τον κατάλογο που χρειάζονται για να δημιουργήσουν μια πραγματικά αξέχαστη περιπέτεια για το κοινό τους.
:::

Για να είστε διαφορετικοί και να προσελκύσετε τη νέα γενιά ταξιδιωτών, πρέπει να σκεφτείτε τη σελίδα προορισμού του καταλύματός σας στο Wink ως`digitized version`της ιδιοκτησίας σας και των περιχώρων της.
Τι μπορεί να πωληθεί, να ενοικιαστεί, να κρατηθεί, να ανταλλαχθεί, να μοιραστεί στην περιοχή σας που σας ανήκει ή που μπορείτε να κάνετε εσείς; Αυτή είναι η ώρα σας να γίνετε δημιουργικοί και να σκεφτείτε**έξω από τα συνηθισμένα**.

### Εξερευνώντας

Αυτή η ενότητα καλύπτεται από[Εξτρανέτ > Δίκτυο](/extranet/network/#explore-network).

Η σελίδα σάς επιτρέπει να αναζητήσετε όλους τους διαφορετικούς τύπους συνεργατών που βρίσκονται ήδη στην πλατφόρμα και σας παρέχει μια σύντομη περιγραφή τους, καθώς και την προηγούμενη απόδοσή τους.

### Ελκυστικός

Εάν το κατάλυμά σας συνεργάζεται ήδη με influencers**Ή**θέλω να συνεργαστώ με γνωστές προσωπικότητες**ΚΑΙ**θέλουν να μπορούν να παρακολουθούν την απόδοση της επένδυσής τους (ROI).
Πείτε τους απευθείας να δημιουργήσουν έναν λογαριασμό στο Wink ή επικοινωνήστε μαζί μας και θα τους στείλουμε μια πρόσκληση.

### Πηγαίνοντας απευθείας

Μερικές φορές, θέλετε να δημιουργήσετε μια άμεση σχέση με έναν από τους συνεργάτες μας.

Το κάνεις αυτό για να...

* Προσφέρετέ τους αποκλειστικό απόθεμα.
* Δώστε τους μια μοναδική προμήθεια.
* Δώστε τους μια μοναδική τιμή μέλους.
* Κάντε τους μια μοναδική προσφορά.

:::tip\[Επαγγελματική συμβουλή]
Δεν υπάρχουν άλλοι λόγοι για να δημιουργήσετε μια άμεση σύνδεση εκτός από τους λόγους που αναφέρονται παραπάνω.
:::

Μόλις δημιουργήσετε ένα νέο, άμεσο κανάλι πωλήσεων για αυτούς, το απόθεμά σας θα εμφανιστεί στο[Στούντιο Wink](/studio/what-is-studio)με ένα`Direct ribbon`επικαλύψτε το απόθεμά σας.
