Η πύλη συνεργατών εκθέτει ταξιδιωτικό απόθεμα από προμηθευτές σε όλο τον κόσμο. Ο στόχος σας είναι να βρείτε το απόθεμα που σας ταιριάζει. Σας προσφέρουμε έναν τρόπο να περιηγηθείτε σε ξενοδοχεία, εγκαταστάσεις και εμπειρίες. Αυτό το άρθρο αναφέρεται σε τρόπους για να επιλέξετε τι να πουλήσετε, βασισμένο σε δεδομένα και όχι μόνο σε όμορφες εικόνες.

Σας προτείνουμε να διαβάσετε πώς να[Ερευνα](/studio/search)και[Insight For Dummies](/guides/insight-for-dummies)εκ των προτέρων.

## Στρατηγικές

Υπάρχουν πολλά πράγματα που πρέπει να λάβετε υπόψη όταν αποφασίζετε για το ποια θα είναι η θέση σας για τη δημιουργία εσόδων. Θα καλύψουμε εδώ μερικά που πιστεύουμε ότι μπορούν να αποτελέσουν καλές παθητικές στρατηγικές για εσάς καθώς ξεκινάτε.

Καταρχάς, ακολουθούν μερικά πράγματα που πρέπει να σκεφτείτε πριν ξεκινήσετε να πουλάτε οποιοδήποτε είδος ταξιδιωτικού αποθέματος:

* Το γεγονός ότι η λίστα αναγράφει ότι είναι διαθέσιμη ενώ κάνετε αναζήτηση δεν σημαίνει ότι είναι διαθέσιμη για ένα συγκεκριμένο δρομολόγιο.
* Ως συνεργάτης, βλέπετε μόνο τη βασική τιμή κατά την αναζήτηση και όχι την υπολογισμένη / τελική τιμή. Αυτό συμβαίνει επειδή...
* Η τιμολόγηση του ταξιδιωτικού αποθέματος είναι περίπλοκη και μπορεί να συνοδεύεται από πολλούς κανόνες.

### FOMO / YOLO

:::σημείωση\[Με λίγα λόγια]
Ακριβό, πολυτελές απόθεμα υψηλής ποιότητας που παρέχει καλό περιθώριο κέρδους ακόμη και αν η προμήθεια είναι χαμηλότερη.
:::

<Steps>
  1. Αναζητήστε ακίνητα ή βίλες που έχουν εξωφρενικές τιμές \[που αποθαρρύνουν τους περισσότερους άλλους πωλητές].
  2. Βρείτε/Δημιουργήστε ένα κοινό που θα απολάμβανε το δικαίωμα να καυχιέται ή θα μπορούσε να θεωρήσει το ταξίδι ως επαγγελματικό έξοδο.
</Steps>


*Αυτός ο τύπος αγοραστή είναι συνήθως ένα επιμελημένο κοινό/ομάδα.*

### Η Συμπίεση

:::σημείωση\[Με λίγα λόγια]
Ξενοδοχεία με 1000+ κλειδιά που επισκέπτονται είτε εταιρικοί ταξιδιώτες είτε οικογένειες σε διακοπές.
:::

*Είναι χρήσιμο αν έχετε ήδη ως εξειδικευμένο πεδίο τα εταιρικά ή οικογενειακά ταξίδια. Αν δεν έχετε, θα πρέπει να επενδύσετε σε διαφημιστικά έξοδα σε μέρη όπως το Facebook, όπου εστιάζετε σε αυτό το είδος κοινού.*

<Steps>
  1. Βρείτε ένα μεγάλο ξενοδοχείο που θέλετε να εκπροσωπήσετε.
  2. Πουλήστε το μέσω των καναλιών σας που αναζητούν ενεργά αυτό το είδος αποθέματος.
  3. Λόγω του μεγέθους και της ζήτησης που υπερβαίνει το κανονικό, αυτή η στρατηγική μπορεί πραγματικά να αποδώσει.
</Steps>


### Εμμεσος

:::σημείωση\[Με λίγα λόγια]
Μην πουλάτε το δωμάτιο... Πουλήστε την εμπειρία!
:::

Λέμε στους προμηθευτές να αναβαθμίσουν το παιχνίδι πωλήσεών τους σκεπτόμενοι ευρύτερα και να πουλήσουν όλες τις εγκαταστάσεις και τις εμπειρίες τους \[και των άλλων]. Αυτό σας επιτρέπει να πουλήσετε...:

* Αίθουσες συσκέψεων για εταιρικούς ταξιδιώτες.
* Από εστιατόρια μέχρι λάτρεις του καλού φαγητού.
* Σπα για να περιποιηθείτε τους φανατικούς.
* ...και πολλά άλλα.

:::συμβουλή
Η διαδικασία κράτησης απαιτεί πάντα από τον χρήστη που κάνει την κράτηση να επιλέξει πρώτα ένα δωμάτιο. Μόλις επιλεγεί αυτό το δωμάτιο, βλέπει όλες τις επιπλέον παροχές που έχει επίσης διαθέσει το ξενοδοχείο.

ΑΛΛΑ... το μάρκετινγκ δεν χρειάζεται να ξεκινά με ένα δωμάτιο. Σας αφήνει ανοιχτούς για να εξερευνήσετε και να αναδείξετε οτιδήποτε έχει να προσφέρει το ξενοδοχείο.
:::

*Για αυτό το παράδειγμα, ας υποθέσουμε ότι το κοινό σας αποτελείται κυρίως από δύτες.*

<Steps>
  1. [Ερευνα](/studio/search) για μια εμπειρία κατάδυσης.
  2. Πουλήστε αυτήν την εμπειρία κατάδυσης στο κοινό σας αντί για το δωμάτιο του ξενοδοχείου.
  3. Κερδίζετε προμήθεια τόσο από το δωμάτιο όσο και από τυχόν επιπλέον παροχές που καταγράφει ο επισκέπτης.
</Steps>


### Με τους αριθμούς

:::note\[Με λίγα λόγια]
Αξιοποιήστε τα αναλυτικά στοιχεία για να προσδιορίσετε τι πουλάει πού, πότε και από ποιον.
:::

Αυτή η στρατηγική είναι η πιο ευέλικτη. Δεν έχει σημασία αν έχετε μια εξειδικευμένη αγορά ή όχι.

Ακολουθούν μερικά εύκολα βήματα για να το πετύχετε αυτό:

<Steps>
  1. Διαβάστε τον οδηγό [Insight For Dummies](/guides/insight-for-dummies).
  2. Με βάση τις πληροφορίες από το #1, βρείτε απόθεμα που ταιριάζει με την/τις τάση/ές που βρήκατε.
  3. Πουλήστε το ίδιο απόθεμα σε πιθανούς πελάτες σε περιοχές που παρατηρήσατε ότι είναι συχνοί αγοραστές.
</Steps>
