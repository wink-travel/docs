Συνδέοντας το Google Analytics 4 property σας με το Wink, τα γεγονότα κρατήσεων — από τις προβολές δωματίων μέχρι τις ολοκληρωμένες αγορές — προωθούνται αυτόματα στον δικό σας πίνακα ελέγχου GA4 παράλληλα με τα δικά του analytics του Wink. Χωρίς κώδικα, χωρίς τεχνική ρύθμιση από εσάς.

<Aside type="tip">
Το μόνο που χρειάζεστε είναι ένα GA4 Measurement ID. Μόλις το εισάγετε στην πύλη του Wink, τα analytics σας είναι ενεργά αμέσως.
</Aside>

---

## Πώς Λειτουργεί

```
Εισάγετε το G-XXXXXXXXXX στην πύλη του Wink
       ↓
Το Wink προωθεί τα γεγονότα κρατήσεων στο GA4 property σας
       ↓
Συνδέεστε στο analytics.google.com και βλέπετε τις κρατήσεις σας
```

---

## Τι Πρέπει να Κάνετε

<Steps>

1. **Δημιουργήστε ένα GA4 Property**

   Μεταβείτε στο [analytics.google.com](https://analytics.google.com): Admin → Create Property. Ορίστε τη ζώνη ώρας και το νόμισμα ώστε να ταιριάζουν με την αγορά σας.

2. **Δημιουργήστε ένα Web Data Stream**

   Admin → Data Streams → Add Stream → Web. Εισάγετε `wink.travel` ως URL. Αυτό δημιουργεί το **Measurement ID** σας (`G-XXXXXXXXXX`). Αντιγράψτε το.

3. **Εισάγετε το Measurement ID στην Πύλη του Wink**

   Συνδεθείτε στο [my.wink.travel](https://my.wink.travel), επιλέξτε τον λογαριασμό σας και μεταβείτε στην καρτέλα **Customizations**. Επικολλήστε εκεί το Measurement ID και αποθηκεύστε. Αυτό ήταν — η ενσωμάτωση GA4 είναι πλέον ενεργή.

4. **Ορίστε το `purchase` ως Βασικό Γεγονός**

   Επιστρέψτε στο GA4 property σας: Admin → Events → βρείτε το `purchase` → ενεργοποιήστε το **Mark as key event**. Αυτό κάνει τις ολοκληρωμένες κρατήσεις να εμφανίζονται στην αναφορά Μετατροπών.

5. **Δείτε τα Δεδομένα σας**

   Συνδεθείτε στο [analytics.google.com](https://analytics.google.com) και θα δείτε τις κρατήσεις, τα έσοδα και το funnel των επισκεπτών — όλα παρακολουθούνται αυτόματα μέσω του Wink.

</Steps>

<Aside type="note">
Δεν χρειάζεται να εγκαταστήσετε κανέναν κώδικα παρακολούθησης στην ιστοσελίδα σας για αυτή την ενσωμάτωση. Το Wink αναλαμβάνει τα πάντα. Το Measurement ID σας είναι απλώς ο προορισμός όπου αποστέλλονται τα δεδομένα κρατήσεων.
</Aside>

## Τι Μπορείτε να Παρακολουθείτε

Μόλις συνδεθεί, το GA4 property σας λαμβάνει το πλήρες σύνολο των γεγονότων Enhanced E-commerce για το ακίνητό σας:

### Ανακάλυψη

| Γεγονός | Τι Σημαίνει |
|---|---|
| `view_item_list` | Ένας επισκέπτης είδε το ακίνητό σας σε αναζήτηση ή λίστα κατηγορίας |
| `select_item` | Ένας επισκέπτης έκανε κλικ στο ακίνητό σας από λίστα |
| `view_item` | Ένας επισκέπτης είδε τη σελίδα λεπτομερειών του ξενοδοχείου σας |

### Εξέταση

| Γεγονός | Τι Σημαίνει |
|---|---|
| `view_promotion` | Ένας επισκέπτης είδε μια προωθητική προσφορά για το ακίνητό σας |
| `select_promotion` | Ένας επισκέπτης έκανε κλικ σε προωθητική προσφορά |
| `add_to_wishlist` | Ένας επισκέπτης αποθήκευσε το ακίνητό σας στη λίστα επιθυμιών του |

### Funnel Κράτησης

| Γεγονός | Τι Σημαίνει |
|---|---|
| `add_to_cart` | Ένας επισκέπτης επέλεξε δωμάτιο ή πακέτο |
| `view_cart` | Ένας επισκέπτης είδε την περίληψη της κράτησής του |
| `remove_from_cart` | Ένας επισκέπτης αφαίρεσε δωμάτιο ή πακέτο |
| `begin_checkout` | Ένας επισκέπτης ξεκίνησε τη διαδικασία ολοκλήρωσης αγοράς |
| `add_payment_info` | Ένας επισκέπτης εισήγαγε τα στοιχεία πληρωμής του |
| `add_shipping_info` | Ένας επισκέπτης εισήγαγε τα στοιχεία του επισκέπτη |
| `purchase` | Ένας επισκέπτης ολοκλήρωσε μια κράτηση |
| `refund` | Μια κράτηση ακυρώθηκε και επιστράφηκε |

Όλα τα γεγονότα περιλαμβάνουν σχετικές διαστάσεις όπως έσοδα, νόμισμα, τύπο δωματίου και αξία κράτησης — δίνοντάς σας πλήρη εικόνα του ταξιδιού του επισκέπτη από την πρώτη εντύπωση μέχρι την ολοκληρωμένη κράτηση.

## Παρακολούθηση Cross-Domain (Προχωρημένο)

Η βασική ενσωμάτωση παραπάνω παρακολουθεί ό,τι συμβαίνει μέσα στο Wink. Ωστόσο, αν θέλετε να παρακολουθείτε τους επισκέπτες από την ιστοσελίδα σας μέχρι την ολοκλήρωση κράτησης στο Wink — και να αποδίδετε σωστά την αρχική πηγή κίνησης (Google Ads, email καμπάνιες, οργανική αναζήτηση κ.ά.) — απαιτείται ένα επιπλέον βήμα σε κάθε πλευρά.

<Aside type="caution">
Αυτή η ενότητα ισχύει μόνο αν έχετε εγκατεστημένο GA4 στην ιστοσελίδα σας. Αν δεν τρέχετε GA4 στον ιστότοπό σας, η βασική ενσωμάτωση παραπάνω είναι ό,τι χρειάζεστε.
</Aside>

### Γιατί Είναι Σημαντικό

Χωρίς παρακολούθηση cross-domain, ένας επισκέπτης που επισκέπτεται πρώτα την ιστοσελίδα σας και μετά κάνει κράτηση στο Wink θα εμφανίζεται στο GA4 ως **Άμεση** συνεδρία χωρίς σύνδεση με την προηγούμενη επίσκεψή του. Κάθε κλικ από Google Ads, email καμπάνιες ή οργανική αναζήτηση που οδήγησε στην κράτηση δεν θα λαμβάνει πιστωτική αναγνώριση.

Με ρυθμισμένη την παρακολούθηση cross-domain, βλέπετε το πλήρες ταξίδι του επισκέπτη:

```
google.com (πληρωμένη διαφήμιση)
       ↓
η ιστοσελίδα σας (ο επισκέπτης περιηγείται σε δωμάτια)
       ↓
wink.travel (ο επισκέπτης επιλέγει δωμάτιο και κάνει κράτηση)
       ↓
η αγορά αποδίδεται σωστά στο αρχικό κλικ Google Ads
```

### Ενέργεια Απαιτείται: Εσείς

Ρυθμίστε το GA4 property σας ώστε να αναγνωρίζει το `wink.travel` ως μέρος του δικού σας domain, όχι ως εξωτερική παραπομπή. Αυτό λέει στο GA4 να συνδέσει τις δύο συνεδρίες σε ένα συνεχές ταξίδι επισκέπτη.

<Steps>
1. GA4 Admin → Data Streams → το stream σας
2. Configure tag settings → **Configure your domains**
3. Προσθέστε το `wink.travel` ως συνδεδεμένο domain
4. Αποθηκεύστε
</Steps>

### Ενέργεια Απαιτείται: Wink

Επικοινωνήστε με το [Wink Support](https://my.wink.travel) και δώστε το domain της ιστοσελίδας σας (π.χ. `yourhotel.com`). Το Wink θα προσθέσει το domain σας στη λίστα αποκλεισμού παραπομπών στο GA4 property του Wink. Αυτό αποτρέπει την καταγραφή της ιστοσελίδας σας ως πηγή παραπομπής, που διαφορετικά θα αντικαθιστούσε την αρχική απόδοση και θα ξεκινούσε νέα συνεδρία.

<Aside type="tip">
Η βασική ενσωμάτωση παραπάνω είναι επαρκής για τα περισσότερα ακίνητα και σας δίνει άμεση ορατότητα στις κρατήσεις και τα έσοδα στο Wink. Η παρακολούθηση cross-domain είναι μια προχωρημένη επιλογή που αξίζει να ρυθμίσετε όταν είστε άνετοι με τα δεδομένα GA4 και θέλετε βαθύτερη ανάλυση απόδοσης.
</Aside>
