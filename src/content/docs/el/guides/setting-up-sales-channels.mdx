Τα κανάλια πωλήσεων είναι ο τρόπος με τον οποίο ελέγχετε τη διανομή του αποθέματός σας. Ένα κανάλι πωλήσεων ορίζει ποιος μπορεί να πουλήσει τα δωμάτιά σας, ποια προμήθεια κερδίζει και ποια τιμοκαταλόγια είναι διαθέσιμα.

## Τύποι καναλιών πωλήσεων

Υπάρχουν τρεις βασικοί τρόποι διανομής του καταλύματός σας στο Wink:

- **Wink Network** — Το κατάλυμά σας είναι ορατό σε όλους τους συνεργάτες στην πλατφόρμα. Αυτή είναι η ευρύτερη διανομή.
- **Άμεσα κανάλια** — Δημιουργείτε ένα ιδιωτικό κανάλι για έναν συγκεκριμένο συνεργάτη ή εταίρο με προσαρμοσμένο ποσοστό προμήθειας.
- **Η δική σας ιστοσελίδα** — Χρησιμοποιήστε κοινόχρηστους συνδέσμους ή ενσωματωμένες κάρτες για να πουλήσετε απευθείας από τον ιστότοπό σας χωρίς ενδιάμεσο.

## Δημιουργία καναλιού πωλήσεων

<Steps>
1. Μεταβείτε στο https://extranet.wink.travel και επιλέξτε το κατάλυμά σας.
2. Πλοηγηθείτε στην ενότητα καναλιών πωλήσεων.
3. Κάντε κλικ στο `Δημιουργία νέου καναλιού πωλήσεων`.
4. Δώστε ένα όνομα στο κανάλι (π.χ., "Άμεση Ιστοσελίδα", "Συνεργάτης - Kim Travel").
5. Ορίστε το **ποσοστό προμήθειας** — πόσο κερδίζει ο συνεργάτης ανά κράτηση.
6. Επιλέξτε ποια **τιμοκαταλόγια** είναι διαθέσιμα σε αυτό το κανάλι.
7. Προαιρετικά περιορίστε σε συγκεκριμένους **τύπους δωματίων** ή **απόθεμα**.
8. Κάντε κλικ στο `Αποθήκευση`.
</Steps>

## Wink Network

Το Wink Network είναι το προεπιλεγμένο κανάλι διανομής. Όταν ενεργοποιηθεί, το κατάλυμά σας εμφανίζεται στο [Studio](/el/studio/search) όπου οποιοσδήποτε συνεργάτης μπορεί να το βρει και να το πουλήσει.

Για να το ενεργοποιήσετε:

<Steps>
1. Μεταβείτε στις ρυθμίσεις δικτύου του καταλύματός σας.
2. Ενεργοποιήστε το `Wink Network`.
3. Ορίστε ένα προεπιλεγμένο ποσοστό προμήθειας για τους συνεργάτες του δικτύου.
</Steps>

:::tip
Το Wink Network είναι ο πιο εύκολος τρόπος για να αποκτήσετε ευρεία προβολή. Μπορείτε πάντα να δημιουργήσετε άμεσα κανάλια αργότερα για τους κορυφαίους συνεργάτες σας με διαφορετικά ποσοστά προμήθειας.
:::

## Άμεσα κανάλια συνεργατών

Για συνεργάτες με τους οποίους θέλετε να συνεργαστείτε στενά, δημιουργήστε ένα άμεσο κανάλι με προσαρμοσμένη προμήθεια.

<Steps>
1. Βρείτε τον συνεργάτη μέσω της ενότητας Δικτύου. Δείτε [Network](/el/extranet/network).
2. Δημιουργήστε ένα άμεσο κανάλι πωλήσεων για αυτόν τον συνεργάτη.
3. Ορίστε ένα προσαρμοσμένο ποσοστό προμήθειας (που μπορεί να είναι υψηλότερο ή χαμηλότερο από το προεπιλεγμένο του δικτύου).
4. Επιλέξτε ποιο απόθεμα είναι διαθέσιμο σε αυτό το κανάλι.
</Steps>

Αυτό είναι χρήσιμο όταν θέλετε να προσφέρετε σε έναν influencer ή ταξιδιωτικό πράκτορα προνομιακή τιμή ή αποκλειστική πρόσβαση σε ορισμένα δωμάτια.

## Δομή προμήθειας

Οι προμήθειες εκφράζονται ως ποσοστό του συνολικού ποσού της κράτησης. Η προμήθεια διαμοιράζεται μεταξύ των εμπλεκόμενων μερών:

- **Προμήθεια συνεργάτη** — Αυτό που κερδίζει ο πωλητής (συνεργάτης, ταξιδιωτικός πράκτορας, influencer).
- **Προμήθεια πλατφόρμας** — Η χρέωση επεξεργασίας του Wink.
- **Έσοδα καταλύματος** — Το υπόλοιπο πηγαίνει σε εσάς.

Εσείς ελέγχετε την προμήθεια του συνεργάτη. Η προμήθεια της πλατφόρμας αφαιρείται αυτόματα.

## Παρακολούθηση απόδοσης

Μόλις τα κανάλια σας είναι ενεργά, χρησιμοποιήστε τα [Analytics](/el/my-account/analytics) για να παρακολουθείτε ποια κανάλια φέρνουν τις περισσότερες κρατήσεις και έσοδα. Αυτό σας βοηθά να βελτιστοποιήσετε τα ποσοστά προμήθειας και να αποφασίσετε σε ποιες συνεργασίες να επενδύσετε.
