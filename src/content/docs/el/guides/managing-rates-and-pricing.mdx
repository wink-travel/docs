Η διαχείριση τιμών είναι ένα από τα πιο σημαντικά — και πιο σύνθετα — μέρη της λειτουργίας ενός καταλύματος στο Wink. Αυτός ο οδηγός αναλύει την ιεραρχία τιμολόγησης ώστε να μπορείτε να τη ρυθμίσετε με αυτοπεποίθηση.

## Η ιεραρχία τιμών

Το Wink χρησιμοποιεί μια τριεπίπεδη δομή τιμολόγησης:

1. **Κύρια τιμή** — Η βασική ημερήσια τιμή για έναν τύπο δωματίου.
2. **Πλάνο τιμής** — Ένα σύνολο κανόνων που περιβάλλει την κύρια τιμή (πολιτική ακύρωσης, γεύματα, περιορισμοί, τροποποιητές).
3. **Ειδική τιμή** — Μια υπερισχύουσα ρύθμιση που προσαρμόζει την κύρια τιμή για συγκεκριμένες ημερομηνίες, προσφορές ή συνθήκες.

Η κατανόηση αυτής της ιεραρχίας είναι το κλειδί. Οι κύριες τιμές θέτουν τα θεμέλια, τα πλάνα τιμών διαμορφώνουν τον τρόπο παρουσίασης και πώλησης της τιμής, και οι ειδικές τιμές σας επιτρέπουν να τρέχετε στοχευμένες προσφορές χωρίς να επηρεάζετε την βασική τιμή.

## Βήμα 1 — Ρύθμιση κύριων τιμών

Οι κύριες τιμές είναι οι βασικές ημερήσιες τιμές ανά τύπο δωματίου.

<Steps>
1. Μεταβείτε στο https://extranet.wink.travel και επιλέξτε το κατάλυμά σας.
2. Κάντε κλικ στο `Monetize > Master Rates` από τη βασική γραμμή πλοήγησης.
3. Κάντε κλικ στο `Create a new master rate`.
4. Επιλέξτε τον τύπο δωματίου, το πλάνο τιμής και το εύρος ημερομηνιών που θέλετε να τιμολογήσετε.
5. Εισάγετε τις ημερήσιες τιμές για κάθε ημέρα στο εύρος.
6. Κάντε κλικ στο `Save`.
</Steps>

:::tip
Χρησιμοποιήστε την προβολή ημερολογίου για να δείτε γρήγορα κενά στην τιμολόγησή σας. Ένας τύπος δωματίου δεν θα είναι διαθέσιμος για κράτηση σε ημερομηνίες χωρίς κύρια τιμή.
:::

## Βήμα 2 — Διαμόρφωση πλάνων τιμών

Τα πλάνα τιμών ελέγχουν τις συνθήκες υπό τις οποίες πωλείται μια τιμή.

<Steps>
1. Κάντε κλικ στο `Monetize > Rate Plans` από τη βασική γραμμή πλοήγησης.
2. Δημιουργήστε ένα νέο πλάνο τιμής ή επεξεργαστείτε ένα υπάρχον.
3. Διαμορφώστε τις βασικές ρυθμίσεις:
   - **Πολιτική ακύρωσης** — Επιστρέψιμη, μη επιστρέψιμη ή υπό όρους.
   - **Γεύματα** — Πρωινό, all-inclusive, κ.ά.
   - **Τροποποιητές πληρότητας** — Έκπτωση για μονή κατοίκηση, επιπλέον χρεώσεις για ενήλικες/παιδιά.
   - **Περιορισμοί** — Διάρκεια διαμονής, προκρατήσεις, ημερομηνίες πώλησης, ημέρες άφιξης, κ.ά.
4. Κάντε κλικ στο `Save`.
</Steps>

Ένας τύπος δωματίου μπορεί να έχει πολλαπλά πλάνα τιμών. Για παράδειγμα, μπορεί να έχετε ένα πλάνο "Ευέλικτο" (επιστρέψιμο, με πρωινό) και ένα πλάνο "Οικονομικό" (μη επιστρέψιμο, μόνο δωμάτιο) — και τα δύο χρησιμοποιούν την ίδια κύρια τιμή αλλά παρουσιάζονται διαφορετικά στους ταξιδιώτες.

## Βήμα 3 — Δημιουργία ειδικών τιμών (προαιρετικό)

Οι ειδικές τιμές υπερισχύουν των κύριων τιμών για συγκεκριμένα σενάρια — προσφορές, εποχιακές προσαρμογές ή τιμολόγηση εκδηλώσεων.

<Steps>
1. Κάντε κλικ στο `Monetize > Special Rates` από τη βασική γραμμή πλοήγησης.
2. Κάντε κλικ στο `Create a new special rate`.
3. Επιλέξτε την κύρια τιμή που θέλετε να υπερισχύσετε και ορίστε το εύρος ημερομηνιών.
4. Εισάγετε τις προσαρμοσμένες ημερήσιες τιμές.
5. Κάντε κλικ στο `Save`.
</Steps>

Οι ειδικές τιμές έχουν προτεραιότητα έναντι των κύριων τιμών για τις ημερομηνίες που καλύπτουν.

## Βήμα 4 — Επαλήθευση με το εργαλείο δοκιμής τιμών

Πριν οι τιμές σας γίνουν ενεργές, χρησιμοποιήστε το [Rate Tester](/el/extranet/rate-tester) για να επαληθεύσετε ότι όλα υπολογίζονται σωστά.

<Steps>
1. Κάντε κλικ στο `Monetize > Rate Tester` από τη βασική γραμμή πλοήγησης.
2. Επιλέξτε κανάλι πωλήσεων, εύρος ημερομηνιών και διαμόρφωση δωματίου.
3. Κάντε κλικ στο `Search`.
4. Ελέγξτε τα αποτελέσματα — βεβαιωθείτε ότι η τιμολόγηση, οι περιορισμοί και οι πολιτικές ακύρωσης εμφανίζονται όπως αναμένεται.
</Steps>

## Συνηθισμένα μοτίβα

- **BAR (Best Available Rate)** — Ένα μόνο ευέλικτο πλάνο τιμής με την τυπική σας πολιτική ακύρωσης. Η πιο συνηθισμένη ρύθμιση.
- **Έκπτωση μη επιστρέψιμης τιμής** — Ένα δεύτερο πλάνο τιμής με χαμηλότερη τιμή και μη επιστρέψιμη ακύρωση. Ελκύει ταξιδιώτες ευαίσθητους στην τιμή.
- **Τιμολόγηση εκδηλώσεων** — Χρησιμοποιήστε ειδικές τιμές για να αυξήσετε τις τιμές σε περιόδους υψηλής ζήτησης (συνέδρια, αργίες, φεστιβάλ).
- **Έκπτωση για μακρά διαμονή** — Προσθέστε περιορισμό διάρκειας διαμονής (π.χ. ελάχιστο 7 νύχτες) σε ένα ειδικό πλάνο τιμής με χαμηλότερη βασική τιμή.
