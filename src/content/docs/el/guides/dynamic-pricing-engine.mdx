Η μηχανή τιμολόγησης του Wink ξεπερνά τις απλές νυχτερινές τιμές. Υπολογίζει δυναμικά την τελική τιμή που βλέπει ο ταξιδιώτης συνδυάζοντας πολλαπλά επίπεδα τιμολόγησης — βασικές τιμές, τιμοκαταλόγους, προσφορές και διαμορφώσεις δωματίων.

## Πώς λειτουργεί η τιμολόγηση

Η μηχανή τιμολόγησης αξιολογεί διάφορα επίπεδα για να υπολογίσει την τελική τιμή:

### 1. Βασικές τιμές

Η βάση της τιμολόγησής σας. Οι βασικές τιμές ορίζουν την αρχική τιμή για κάθε τύπο δωματίου ή είδος αποθέματος. Τις ορίζετε στο Extranet και χρησιμεύουν ως αφετηρία για όλους τους υπολογισμούς τιμών.

<LinkButton href="/el/extranet/master-rates">
  Ρύθμιση βασικών τιμών
</LinkButton>

### 2. Τιμοκατάλογοι

Οι τιμοκατάλογοι τροποποιούν τις βασικές τιμές για συγκεκριμένες συνθήκες. Ένας τιμοκατάλογος μπορεί να προσαρμόσει την τιμολόγηση βάσει:

- **Κατοίκησης** — Διαφορετικές τιμές για μονή, διπλή ή τριπλή κατοίκηση.
- **Διάρκειας διαμονής** — Εκπτώσεις για μεγαλύτερες διαμονές.
- **Συμπερίληψης γευμάτων** — Μόνο δωμάτιο, πρωινό, ημιδιατροφή ή πλήρης διατροφή.
- **Παραθύρου κράτησης** — Τιμές για πρώιμη κράτηση ή τελευταία στιγμή.

<LinkButton href="/el/extranet/rate-plans">
  Διαμόρφωση τιμοκαταλόγων
</LinkButton>

### 3. Ημερήσιες τιμές

Οι ημερήσιες τιμές σας επιτρέπουν να παρακάμψετε την τιμολόγηση για συγκεκριμένες ημερομηνίες. Χρησιμοποιήστε τις για:

- **Εποχιακή τιμολόγηση** — Υψηλότερες τιμές κατά την υψηλή περίοδο, χαμηλότερες στην εκτός εποχής.
- **Τιμολόγηση εκδηλώσεων** — Προσαυξήσεις κατά τη διάρκεια τοπικών εκδηλώσεων ή αργιών.
- **Ημερομηνίες αποκλεισμού** — Ημερομηνίες όπου συγκεκριμένες τιμές δεν είναι διαθέσιμες.

<LinkButton href="/el/extranet/rates">
  Διαχείριση ημερήσιων τιμών
</LinkButton>

### 4. Προσφορές

Οι προσφορές εφαρμόζουν εκπτώσεις ή ειδικές προσφορές πάνω στην υπολογισμένη τιμή:

- **Ποσοστιαίες εκπτώσεις** — π.χ. 20% έκπτωση για πρώιμη κράτηση.
- **Προσφορές με πρόσθετη αξία** — π.χ. δωρεάν πρωινό, πίστωση για σπα.
- **Προσφορές με χρονικό περιορισμό** — Προσφορές που ισχύουν μόνο για συγκεκριμένες ημερομηνίες ταξιδιού ή κράτησης.

<LinkButton href="/el/extranet/promotions">
  Δημιουργία προσφορών
</LinkButton>

## Υπολογισμός τιμής διαμονής

Όταν ένας ταξιδιώτης αναζητά διαμονή πολλών διανυκτερεύσεων, η μηχανή υπολογίζει το σύνολο ως εξής:

1. Αναζητά την **βασική τιμή** για τον τύπο δωματίου.
2. Εφαρμόζει τις σχετικές προσαρμογές του **τιμοκαταλόγου**.
3. Ελέγχει για **παρακάμψεις ημερήσιων τιμών** σε κάθε νύχτα της διαμονής.
4. Εφαρμόζει τυχόν **προσφορές** που ταιριάζουν με τις συνθήκες κράτησης.
5. Αθροίζει τις τιμές ανά νύχτα σε μια **συνολική τιμή διαμονής**.

Αυτό σημαίνει ότι κάθε νύχτα της διαμονής μπορεί να έχει διαφορετική αποτελεσματική τιμή, και ο ταξιδιώτης βλέπει ένα ακριβές σύνολο που αντανακλά όλους τους κανόνες τιμολόγησής σας.

## Επίδραση πολιτικής ακύρωσης

Οι πολιτικές ακύρωσης μπορούν επίσης να επηρεάσουν την εμφανιζόμενη τιμή. Οι ευέλικτες πολιτικές ακύρωσης μπορεί να έχουν μια μικρή προσαύξηση σε σχέση με τις μη επιστρέψιμες τιμές, δίνοντας στους ταξιδιώτες την επιλογή μεταξύ τιμής και ευελιξίας.

<LinkButton href="/el/extranet/cancellation-policies">
  Ρύθμιση πολιτικής ακύρωσης
</LinkButton>

## Συμβουλές

- **Χρησιμοποιήστε το Rate Tester.** Το Extranet περιλαμβάνει εργαλείο δοκιμής τιμών που σας επιτρέπει να προσομοιώσετε αναζητήσεις για να επαληθεύσετε ότι τα επίπεδα τιμολόγησής σας παράγουν τα αναμενόμενα αποτελέσματα.
- **Ξεκινήστε με τις βασικές τιμές, μετά προσθέστε επίπεδα.** Ορίστε πρώτα τις βασικές τιμές και μετά προσθέστε τιμοκαταλόγους και προσφορές. Αυτό διατηρεί τη λογική τιμολόγησής σας διαχειρίσιμη.
- **Επανεξετάζετε τις ημερήσιες τιμές εποχιακά.** Ενημερώστε τις παρακάμψεις ημερήσιων τιμών πριν από κάθε εποχή για να διασφαλίσετε ότι η τιμολόγησή σας παραμένει ανταγωνιστική.
