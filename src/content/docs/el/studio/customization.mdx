Η προσαρμογή είναι μια ισχυρή λειτουργία που μπορείτε να χρησιμοποιήσετε για να ελέγχετε κάθε πτυχή της ενσωμάτωσης με το Wink χωρίς να χρειάζεται να γράψετε κώδικα. Κάθε κομμάτι αποθέματος που πουλάτε μπορεί να προσαρμοστεί. Προσθέστε ή αφαιρέστε λειτουργίες, αλλάξτε συνδυασμό χρωμάτων και πολλά άλλα.

Ενώ βρίσκεστε στο`Customization`σελίδα, κάντε κλικ στο`Actions`κάτω από την προσαρμογή με την οποία θέλετε να εργαστείτε και κάντε κλικ`Update`.*Έχετε ήδη μια προεπιλεγμένη καταχώρηση διαθέσιμη όταν δημιουργείτε τον λογαριασμό σας. Αυτή η καταχώρηση δεν μπορεί να καταργηθεί.*

Αυτό θα σας μεταφέρει στη φόρμα προσαρμογής όπου μπορείτε να δείτε 5 - 7 καρτέλες (*ανάλογα με τον τύπο του λογαριασμού σας*).

## Γενικός

Ακολουθούν οι επιλογές που έχετε στη διάθεσή σας στο πλαίσιο του`General`καρτέλα.

* **Ονομα**Δώστε στην προσαρμογή σας ένα όνομα για να το θυμάστε εύκολα.*π.χ. Για το ταξιδιωτικό μου ιστολόγιο*
* **Εμφάνιση νομίσματος**Επιλέξτε το αρχικό νόμισμα που θα βλέπουν οι χρήστες δίπλα στο ταξιδιωτικό απόθεμα με δυνατότητα κράτησης.*π.χ. δολάρια ΗΠΑ*
* **Γλώσσα εμφάνισης**Επιλέξτε την αρχική γλώσσα που θα συναντήσουν οι χρήστες.*π.χ. Αγγλικά*
* **Προσθήκη στον τίτλο σελίδας**Μπορείτε να ελέγξετε τι προστίθεται στον τίτλο της σελίδας (*ο τίτλος που βλέπετε στην καρτέλα του προγράμματος περιήγησης*). Αυτό χρησιμοποιείται κυρίως για τη μηχανή κρατήσεών μας.*π.χ. Αν πλοηγηθείτε σε<https://ota.wink.travel>, από προεπιλογή, θα δείτε`Home | Traveliko - Hotels with Benefits`*&#x39C;πορείτε να αλλάξετε τα πάντα μετά τον αγωγό '|'.

## Θέμα

Το`Theme`Η καρτέλα σάς επιτρέπει να προσαρμόσετε την εμφάνιση και την αίσθηση του[μηχανή κρατήσεων](/booking-engine/in-a-nutshell)ιστοσελίδα,[WinkLinks](/link-manager/wink-links)ιστοσελίδα και όλα μας[Στοιχεία Ιστού](/developers/web-components).

### Διαμόρφωση

Αυτή η ενότητα σάς επιτρέπει να διαμορφώσετε το σχέδιο και τη διάταξη της κάρτας. Προς το παρόν, υπάρχει μόνο ένα σχέδιο: το Wink.

Η διάταξη της κάρτας ελέγχει εάν θέλετε οι χρήστες σας να βλέπουν οριζόντια ή κάθετα ευθυγραμμισμένες κάρτες συναλλαγών. Οι πληροφορίες είναι οι ίδιες.

:::σημείωση
Δεν υπάρχει διαφορά στη διάταξη της κάρτας κατά την προβολή των καρτών συναλλαγών μας από κινητή συσκευή. Ωστόσο, μπορείτε να δείτε τη διαφορά σε ένα iPad mini και σε μεγαλύτερες αναλύσεις.
:::

:::tip\[Συμβουλή επαγγελματία]
Χρησιμοποιήστε οριζόντια διάταξη εάν έχετε μόνο < 4 ταξιδιωτικά αποθέματα (δηλαδή τύπους δωματίων) σε μια λίστα... απλώς φαίνεται καλύτερα.
:::

### Χρωματιστά

Το χρώμα ελέγχει τον τρόπο με τον οποίο οι χρήστες σας βλέπουν οποιοδήποτε από τα στοιχεία του UI και τους ιστότοπούς μας. Χρησιμοποιήστε το για να ταιριάζει με το υπάρχον χρωματικό σας συνδυασμό \[εάν υπάρχει].

Έχουμε ήδη μια μεγάλη ποικιλία από προκαθορισμένα χρώματα με τα οποία μπορείτε να αρχίσετε να πειραματίζεστε. Τα προκαθορισμένα θέματα θα αλλάξουν τα χρώματα στα πεδία της φόρμας παρακάτω. Μπορείτε επίσης να χρησιμοποιήσετε εξ ολοκλήρου τα δικά σας χρώματα.

## Δρομολόγιο

Το`Itinerary`Η καρτέλα είναι ίσως η πιο ενδιαφέρουσα όσον αφορά τη διασφάλιση ότι οι τιμές εμφανίζονται για το απόθεμα που έχετε επιλέξει.

Καθώς η τιμολόγηση του αποθέματος ταξιδιών είναι μάλλον περίπλοκη, το Wink πρέπει να γνωρίζει το δρομολόγιο του χρήστη πριν μπορέσει να εμφανίσει μια ακριβή τιμή.

Αυτή η ενότητα σάς επιτρέπει να ορίσετε το προεπιλεγμένο δρομολόγιο για να εμφανίσετε τιμές για όλα τα ψηφιακά ακίνητά μας, όπως:

* Οι εικόνες μας φιλικές προς τα κοινωνικά δίκτυα με επικάλυψη τιμών.
* Πλέγματα, κάρτες και σύνδεσμοι WinkLinks.
* Στοιχεία Ιστού
* Μηχανή κρατήσεων

### Ημερομηνίες

Μπορείτε να επιλέξετε αν θέλετε να ορίσετε ένα στατικό ή ένα κυλιόμενο εύρος ημερομηνιών.

* Επιλέξτε ένα στατικό εύρος ημερομηνιών εάν θα χρησιμοποιήσετε την προσαρμογή για ένα συγκεκριμένο συμβάν που λαμβάνει χώρα σε μια δεδομένη ημερομηνία.
* Επιλέξτε ένα κυλιόμενο εύρος ημερομηνιών εάν θέλετε είτε:
  1. Προεπιλογή σε σήμερα + 1 διανυκτέρευση, κάτι που κάνουν οι περισσότερες OTA Ή.
  2. Προεπιλογή σε μια ημερομηνία στο μέλλον που θα πρέπει να είναι πάντα διαθέσιμη.

### Διαμόρφωση δωματίου

Από προεπιλογή, το δρομολόγιο έχει οριστεί σε 2 ενήλικες, καθώς είναι η πιο συνηθισμένη ρύθμιση για τους περισσότερους ταξιδιώτες.

Αυτό μπορεί εύκολα να αλλάξει και μπορείτε να προσθέσετε πολλά δωμάτια με πολλαπλούς επιβάτες, τόσο ενήλικες όσο και παιδιά.

### Προσφορές

Οι Προσφορές είναι μια έξυπνη λειτουργία που εξοικονομεί χρόνο στους χρήστες σας. Εάν ένα κατάλυμα σας έχει δώσει έναν κωδικό προσφοράς που μπορείτε να μοιραστείτε με τους φίλους σας, μπορείτε να επιλέξετε να τον εισαγάγετε εδώ.

Όποιος εισέρχεται μέσω του συνδέσμου σας ή παρόμοιου, θα λάβει αυτόματα τον κωδικό προσφοράς στο δρομολόγιό του και θα δει την μειωμένη τιμή να εφαρμόζεται αυτόματα.

## Ολοκλήρωση

Το`Integration`Η καρτέλα απευθύνεται σε ξενοδοχεία, ολοκληρωτές και ταξιδιωτικούς πράκτορες που επιθυμούν να έχουν μεγαλύτερο έλεγχο στον τρόπο διανομής των κρατήσεων στο Wink.

### Ειδοποιήσεις

Αυτό δίνει στο ξενοδοχείο και στον πάροχο ολοκληρωμένων υπηρεσιών την επιλογή να επιτρέψουν στην Wink να ειδοποιήσει όλα τα εμπλεκόμενα μέρη όταν πραγματοποιηθεί μια κράτηση.

Αυτό περιλαμβάνει την ενημέρωση του channel manager.

Από προεπιλογή, όλες αυτές οι επιλογές είναι ενεργοποιημένες. Οι περιπτώσεις όπου ένας ολοκληρωτής μπορεί να θέλει να απενεργοποιήσει μία ή περισσότερες από αυτές τις λειτουργίες είναι:

* Το ταξιδιωτικό πρακτορείο είναι ενσωματωμένο σε επίπεδο API και θέλει να στέλνει τα δικά του email επιβεβαίωσης.
* Το ξενοδοχείο δοκιμάζει την ενσωμάτωσή του και δεν θέλει να ενοχλήσει το προσωπικό πωλήσεών του.
* Το ξενοδοχείο δεν ενδιαφέρεται για τα ηλεκτρονικά μηνύματα και θέλει μόνο να ειδοποιείται μέσω του PMS.

### Λογότυπο ηλεκτρονικού ταχυδρομείου

Για τους integrators που επιθυμούν να χρησιμοποιήσουν την υπηρεσία ειδοποιήσεων μέσω email, μπορούν να προσαρμόσουν το email ώστε να φαίνεται ότι προέρχεται από αυτούς.

:::σημείωση
Οι προτεινόμενες διαστάσεις είναι 300px πλάτος x 75px ύψος
:::

## Ειδικά για ξενοδοχεία

Υπό το`Hotel-specific`Η καρτέλα είναι επιλογές που επιτρέπουν στο ξενοδοχείο να ελέγχει τι εμφανίζεται στη σελίδα προορισμού του στη μηχανή κρατήσεών μας και τι θα εμφανίζεται όταν δεν υπάρχει απόθεμα για το ζητούμενο δρομολόγιο.

* **Κατατάξεις**Ένα ακίνητο, το οποίο δεν έχει ακόμη επιτύχει καμία ουσιαστική κατάταξη, μπορεί να επιλέξει να απενεργοποιήσει αυτήν τη λειτουργία.

:::σημείωση
Το κατάλυμα δεν μπορεί να ελέγξει τον τρόπο με τον οποίο ένας συνεργάτης επιλέγει να εμφανίσει τη σελίδα προορισμού του ξενοδοχείου... Αυτές είναι δύο διαφορετικές διαμορφώσεις.
:::

* **Διαθεσιμότητα**: Επιτρέπει στην ιδιοκτησία να επιλέξει εάν θα συνεχίσει να εμφανίζει το απόθεμα ακόμα κι αν δεν υπάρχει διαθεσιμότητα.

:::σημείωση
Στις περισσότερες περιπτώσεις, η ιδιότητα θα ήθελε να δώσει στους χρήστες έναν τρόπο να κάνουν κλικ στο`Book`για να συνεχίσουν στη σελίδα προορισμού τους, όπου μπορούν να αλλάξουν το δρομολόγιό τους και να προσπαθήσουν να βρουν μια διαθέσιμη ημερομηνία.
Ενεργοποιήστε αυτόν τον διακόπτη όταν θέλετε να αποκλείσετε τη συνέχεια του χρήστη.
:::

## Πώς να χρησιμοποιήσετε

Στις περισσότερες περιπτώσεις, θα επιλέξετε ποιο`Customization`για εφαρμογή στους συνδέσμους, τις κάρτες, τους χάρτες και τα πλέγματά σας.

Ακολουθούν οι περιπτώσεις που ίσως θελήσετε να εφαρμόσετε χειροκίνητα τη διαμόρφωσή σας:

* **Χειροκίνητη παράκαμψη**Θέλετε να κοινοποιήσετε απευθείας έναν σύνδεσμο προς τη μηχανή κρατήσεών μας και όχι να χρησιμοποιήσετε την[Κοινόχρηστοι σύνδεσμοι](/studio/shareable-links)λειτουργία, με την εφαρμογή της προσαρμοσμένης διαμόρφωσής σας.
* **Πρόσθετο WordPress**Προσθέστε τη διαμόρφωση στο Wink σας[Πρόσθετο WordPress](/developers/wordpress) `Settings`.
* **Προσαρμοσμένη ενσωμάτωση**Εφαρμόστε το χειροκίνητα στον προσαρμοσμένο ολοκληρωτή σας χρησιμοποιώντας το δικό μας[wink-app-loader](/developers/web-components/#application-loader)Στοιχείο ιστού.

### Χειροκίνητη παράκαμψη

Ακολουθεί ένα απλό παράδειγμα για το πώς να εφαρμόσετε το αναγνωριστικό διαμόρφωσής σας, με όλες τις ρυθμίσεις σας, στις κύριες διευθύνσεις URL της μηχανής κρατήσεων.

<h4 align="center">[https://ota.wink.travel?configurationId=MY-UNIQUE-ID](https://ota.wink.travel?configurationId=MY-UNIQUE-ID)</h4>


:::σημείωση
Όταν ένας χρήστης κάνει κλικ σε αυτόν τον σύνδεσμο, θα ορίσετε επίσης το cookie απόδοσης συνεργατών.
:::

### Πρόσθετο WordPress

Πλοηγηθείτε στο[Πρόσθετο Wink WordPress](/developers/wordpress)για να μάθετε περισσότερα.

### Προσαρμοσμένη ενσωμάτωση

Ελέγξτε όλα τα ενσωματωμένα στοιχεία ιστού Wink σε μια σελίδα με το[wink-app-loader](/developers/web-components/#application-loader)Στοιχείο ιστού.

Παρακάτω είναι ένα δείγμα που χρησιμοποιεί`MY-UNIQUE-ID`ως αναγνωριστικό διαμόρφωσης.

```html {11}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    
    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="MY-UNIQUE-ID"
    ></wink-app-loader>
  </body>
</html>
```
