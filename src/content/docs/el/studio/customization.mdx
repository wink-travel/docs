Η προσαρμογή είναι μια ισχυρή λειτουργία που μπορείτε να χρησιμοποιήσετε για να ελέγξετε κάθε πτυχή της ενσωμάτωσης με το Wink χωρίς να χρειάζεται να γράψετε κώδικα. Κάθε κομμάτι αποθέματος που πουλάτε μπορεί να προσαρμοστεί. Προσθέστε ή αφαιρέστε λειτουργίες, αλλάξτε το χρωματικό σχήμα και πολλά άλλα.

Ενώ βρίσκεστε στη σελίδα `Customization`, κάντε κλικ στο κουμπί `Actions` κάτω από την προσαρμογή με την οποία θέλετε να εργαστείτε και επιλέξτε `Update`.
*Έχετε ήδη μια προεπιλεγμένη είσοδο διαθέσιμη όταν δημιουργείτε τον λογαριασμό σας. Αυτή η είσοδος δεν μπορεί να αφαιρεθεί.*

Αυτό θα σας μεταφέρει στη φόρμα προσαρμογής όπου μπορείτε να δείτε 5 - 7 καρτέλες (*ανάλογα με τον τύπο του λογαριασμού σας*).

## Γενικά

Εδώ είναι οι επιλογές που έχετε διαθέσιμες στην καρτέλα `General`.

- **Όνομα** Δώστε ένα όνομα στην προσαρμογή σας για να το θυμάστε εύκολα. *π.χ. Για το ταξιδιωτικό μου blog*
- **Νόμισμα εμφάνισης** Επιλέξτε το αρχικό νόμισμα που θα βλέπουν οι χρήστες δίπλα στο διαθέσιμο ταξιδιωτικό απόθεμα. *π.χ. USD*
- **Γλώσσα εμφάνισης** Επιλέξτε την αρχική γλώσσα που θα συναντήσουν οι χρήστες. *π.χ. Αγγλικά*
- **Προσθήκη στον τίτλο της σελίδας** Μπορείτε να ελέγξετε τι προστίθεται στον τίτλο της σελίδας (*τον τίτλο που βλέπετε στην καρτέλα του προγράμματος περιήγησης*). Αυτό χρησιμοποιείται κυρίως για το booking engine μας. *π.χ. Αν μεταβείτε στο https://ota.wink.travel, από προεπιλογή θα δείτε `Home | Traveliko - Hotels with Benefits`*. Μπορείτε να αλλάξετε ό,τι βρίσκεται μετά το σύμβολο '|'.

## Θέμα

Η καρτέλα `Theme` σας επιτρέπει να προσαρμόσετε την εμφάνιση και την αίσθηση του ιστότοπου του [booking engine](/el/booking-engine/in-a-nutshell), του ιστότοπου [WinkLinks](/el/link-manager/wink-links) και όλων των [Web Components](/el/developers/web-components).

### Διαμόρφωση

Αυτή η ενότητα σας επιτρέπει να διαμορφώσετε το σχεδιασμό και τη διάταξη των καρτών. Προς το παρόν, υπάρχει μόνο ένας σχεδιασμός· συγκεκριμένα ο Wink.

Η διάταξη της κάρτας ελέγχει αν θέλετε οι χρήστες σας να βλέπουν τις συναλλαγές σε οριζόντια ή κατακόρυφη διάταξη. Οι πληροφορίες είναι οι ίδιες.

:::note
Δεν υπάρχει διαφορά στη διάταξη της κάρτας όταν βλέπετε τις συναλλακτικές κάρτες από κινητή συσκευή. Ωστόσο, μπορείτε να δείτε τη διαφορά σε iPad mini και μεγαλύτερες αναλύσεις.
:::

:::tip[Συμβουλή επαγγελματία]
Χρησιμοποιήστε οριζόντια διάταξη αν έχετε λιγότερα από 4 ταξιδιωτικά αποθέματα (π.χ. τύπους δωματίων) σε μια λίστα... απλά φαίνεται καλύτερο.
:::

### Χρώματα

Το χρώμα ελέγχει πώς οι χρήστες σας βλέπουν οποιοδήποτε από τα UI components και τους ιστότοπους μας. Χρησιμοποιήστε το για να ταιριάξετε με το υπάρχον χρωματικό σας σχήμα [αν υπάρχει].

Έχουμε ήδη μια μεγάλη επιλογή προκαθορισμένων χρωμάτων με τα οποία μπορείτε να πειραματιστείτε. Τα προκαθορισμένα θέματα θα αλλάξουν τα χρώματα στα πεδία της φόρμας παρακάτω. Μπορείτε επίσης να χρησιμοποιήσετε εντελώς δικά σας χρώματα.

## Αναλύσεις

Υποστηρίζουμε αναλύσεις GA4 από προεπιλογή. Αν θέλετε να παρακολουθείτε ολόκληρη τη διαδρομή μετατροπής του βελτιωμένου ηλεκτρονικού εμπορίου, από την αρχή μέχρι το τέλος, έχετε έτοιμο το αναγνωριστικό μέτρησής σας και εισάγετέ το στο πεδίο κάτω από την καρτέλα `Analytics`.

<Aside type="caution" title="Υπενθύμιση">
  Μην ξεχάσετε να προσθέσετε τα `i.trvl.as`, `book.wink.travel` και `ota.wink.travel` στα streams της ιδιοκτησίας GA4 σας.
</Aside>

## Χάρτες

Για να ενσωματώσετε τους χάρτες μας στον ιστότοπό σας είτε χρησιμοποιώντας το [WordPress plugin](/el/developers/wordpress) είτε τα [Web Components](/el/developers/web-components), πρέπει να μας παρέχετε το δικό σας κλειδί API Google Maps.

:::note
  [Δημιουργήστε ένα κλειδί API Google Maps εδώ](https://console.developers.google.com/projectselector2/apis/credentials?supportedpurview=project)
:::

Μόλις δημιουργήσετε το κλειδί σας και το διαμορφώσετε για τον ιστότοπό σας, εισάγετέ το στην καρτέλα `Analytics`.

## Δρομολόγιο

Η καρτέλα `Itinerary` είναι πιθανώς η πιο ενδιαφέρουσα όσον αφορά τη διασφάλιση ότι οι τιμές εμφανίζονται για το απόθεμα που έχετε επιλέξει.
Καθώς η τιμολόγηση του ταξιδιωτικού αποθέματος είναι αρκετά πολύπλοκη, το Wink χρειάζεται να γνωρίζει το δρομολόγιο του χρήστη πριν μπορέσει να εμφανίσει ακριβή τιμή.

Αυτή η ενότητα σας επιτρέπει να ορίσετε το προεπιλεγμένο δρομολόγιο για να εμφανίζονται τιμές σε όλη μας την ψηφιακή παρουσία, όπως:

- Οι εικόνες φιλικές προς τα κοινωνικά δίκτυα με επικάλυψη τιμής.
- Πλέγματα, κάρτες και σύνδεσμοι WinkLinks.
- Web Components
- Booking engine

### Ημερομηνίες

Μπορείτε να επιλέξετε αν θέλετε να ορίσετε ένα στατικό εύρος ημερομηνιών ή ένα κυλιόμενο.

- Επιλέξτε στατικό εύρος ημερομηνιών αν θα χρησιμοποιήσετε την προσαρμογή για μια συγκεκριμένη εκδήλωση που λαμβάνει χώρα σε συγκεκριμένη ημερομηνία.
- Επιλέξτε κυλιόμενο εύρος ημερομηνιών αν θέλετε είτε:
    1. Προεπιλογή σε σήμερα + 1 διανυκτέρευση, όπως κάνουν οι περισσότερες OTA ή.
    2. Προεπιλογή σε μια μελλοντική ημερομηνία που θα πρέπει πάντα να έχει διαθεσιμότητα.

### Διαμόρφωση δωματίου

Από προεπιλογή, το δρομολόγιο ορίζεται σε 2 ενήλικες, καθώς είναι η πιο κοινή ρύθμιση για τους περισσότερους ταξιδιώτες.
Αυτό μπορεί εύκολα να αλλάξει και μπορείτε να προσθέσετε πολλαπλά δωμάτια με πολλούς επιβάτες· τόσο ενήλικες όσο και παιδιά.

### Προσφορές

Οι προσφορές είναι μια έξυπνη λειτουργία που εξοικονομεί χρόνο στους χρήστες σας. Αν ένα κατάλυμα σας έχει δώσει έναν κωδικό προσφοράς που μπορείτε να μοιραστείτε με τους φίλους σας, μπορείτε να επιλέξετε να τον εισάγετε εδώ.
Οποιοσδήποτε εισέρχεται μέσω του συνδέσμου σας ή παρόμοιου, θα έχει αυτόματα τον κωδικό προσφοράς προστιθέμενο στο δρομολόγιό του και θα βλέπει την εφαρμογή της έκπτωσης αυτόματα.

## Ενσωμάτωση

Η καρτέλα `Integration` απευθύνεται σε ξενοδοχεία, ενσωματωτές και ταξιδιωτικά πρακτορεία που επιθυμούν να έχουν μεγαλύτερο έλεγχο στον τρόπο διανομής των κρατήσεων στο Wink.

### Ειδοποιήσεις

Αυτό δίνει στο ξενοδοχείο και τον ενσωματωτή την επιλογή να επιτρέψουν στο Wink να ειδοποιεί όλα τα εμπλεκόμενα μέρη όταν γίνεται μια κράτηση.
Αυτό περιλαμβάνει την ειδοποίηση του channel manager.

Από προεπιλογή, αυτές οι επιλογές είναι όλες ενεργοποιημένες. Περιπτώσεις όπου ένας ενσωματωτής μπορεί να θέλει να απενεργοποιήσει μία ή περισσότερες από αυτές τις λειτουργίες είναι:

- Ο ταξιδιωτικός πράκτορας είναι ενσωματωμένος σε επίπεδο API και θέλει να στέλνει τα δικά του επιβεβαιωτικά email.
- Το ξενοδοχείο δοκιμάζει την ενσωμάτωσή του και δεν θέλει να ενοχλεί το προσωπικό πωλήσεών του.
- Το ξενοδοχείο δεν ενδιαφέρεται για τα email και θέλει να λαμβάνει ειδοποιήσεις μόνο μέσω του PMS του.

### Λογότυπο email

Για ενσωματωτές που θέλουν να χρησιμοποιήσουν την υπηρεσία ειδοποιήσεων μέσω email, μπορούν να προσαρμόσουν το email ώστε να φαίνεται ότι προέρχεται από αυτούς.

:::note
  Οι προτεινόμενες διαστάσεις είναι 300px πλάτος x 75px ύψος
:::

## Ειδικά για ξενοδοχεία

Στην καρτέλα `Hotel-specific` υπάρχουν επιλογές που επιτρέπουν στο ξενοδοχείο να ελέγχει τι εμφανίζεται στη σελίδα προορισμού τους στο booking engine μας και τι να εμφανίζεται όταν δεν υπάρχει διαθέσιμο απόθεμα για το ζητούμενο δρομολόγιο.

- **Κατατάξεις**: Ένα κατάλυμα που δεν έχει ακόμη αποκτήσει ουσιαστικές κατατάξεις μπορεί να επιλέξει να απενεργοποιήσει αυτή τη λειτουργία.

:::note
  Το κατάλυμα δεν μπορεί να ελέγξει πώς ένας συνεργάτης επιλέγει να εμφανίσει τη σελίδα προορισμού του ξενοδοχείου... Αυτές είναι δύο διαφορετικές διαμορφώσεις.
:::

- **Διαθεσιμότητα**: Επιτρέπει στο κατάλυμα να επιλέξει αν θα συνεχίσει να εμφανίζει το απόθεμα ακόμα και όταν δεν υπάρχει διαθεσιμότητα.

:::note
  Στις περισσότερες περιπτώσεις, το κατάλυμα θα θέλει να δώσει στους χρήστες τη δυνατότητα να πατήσουν το κουμπί `Book` για να συνεχίσουν στη σελίδα προορισμού τους όπου μπορούν να αλλάξουν το δρομολόγιό τους και να προσπαθήσουν να βρουν διαθέσιμη ημερομηνία.
  Ενεργοποιήστε αυτόν τον διακόπτη όταν θέλετε να εμποδίσετε τον χρήστη από το να συνεχίσει.
:::

## Πώς να χρησιμοποιήσετε

Στις περισσότερες περιπτώσεις, θα επιλέξετε ποια `Customization` θα εφαρμόσετε στους συνδέσμους, τις κάρτες, τους χάρτες και τα πλέγματά σας.

Εδώ είναι οι περιπτώσεις όπου μπορεί να θέλετε να εφαρμόσετε χειροκίνητα τη διαμόρφωσή σας:

- **Χειροκίνητη παράκαμψη** Θέλετε να μοιραστείτε έναν σύνδεσμο απευθείας στο booking engine μας, και να μην χρησιμοποιήσετε τη λειτουργία [Shareable Links](/el/studio/shareable-links), με την προσαρμογή σας να εφαρμόζεται.
- **WordPress plugin** Προσθέστε τη διαμόρφωση στις `Ρυθμίσεις` του Wink [WordPress Plugin](/el/developers/wordpress).
- **Προσαρμοσμένη ενσωμάτωση** Εφαρμόστε χειροκίνητα τη διαμόρφωση στον δικό σας ενσωματωτή με τη χρήση του [wink-app-loader](/el/developers/web-components/#application-loader) Web Component.

### Χειροκίνητη παράκαμψη

Εδώ είναι ένα απλό παράδειγμα για το πώς να εφαρμόσετε το ID διαμόρφωσης, με όλες τις ρυθμίσεις σας, στα κύρια URLs του booking engine μας.

<h4 align="center"> [https://ota.wink.travel?configurationId=MY-UNIQUE-ID](https://ota.wink.travel?configurationId=MY-UNIQUE-ID) </h4>

:::note
  Όταν ένας χρήστης κάνει κλικ σε αυτόν τον σύνδεσμο, θα ορίσετε επίσης το cookie απόδοσης συνεργάτη σας.
:::

### WordPress plugin

Πλοηγηθείτε στο [Wink WordPress plugin](/el/developers/wordpress) για να μάθετε περισσότερα.

### Προσαρμοσμένη ενσωμάτωση

Ελέγξτε όλα τα ενσωματωμένα Wink Web Components σε μια σελίδα με το [wink-app-loader](/el/developers/web-components/#application-loader) Web Component.

Παρακάτω είναι ένα παράδειγμα που χρησιμοποιεί το `MY-UNIQUE-ID` ως το ID διαμόρφωσης.

```html {11}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="MY-UNIQUE-ID"
    ></wink-app-loader>
  </body>
</html>
```
