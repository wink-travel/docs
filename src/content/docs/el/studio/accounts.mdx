Το [Wink Studio](https://studio.wink.travel) σας συνδέει με ταξιδιωτικό απόθεμα παγκοσμίως. Αυτό το άρθρο σας καθοδηγεί στη δημιουργία και διαχείριση των λογαριασμών συνεργατών σας στο Wink.

## Δημιουργία λογαριασμού

Μόλις πιστοποιηθείτε, θα ανακατευθυνθείτε στη [σελίδα λογαριασμών](https://studio.wink.travel/company/list) που εμφανίζει όλους τους λογαριασμούς συνεργατών σας.

Κάντε κλικ στο κουμπί `Create new account` για να συνεχίσετε. Θα ανακατευθυνθείτε στη σελίδα [δημιουργίας λογαριασμού](https://studio.wink.travel/company/create).

Συμπληρώστε όλα τα απαιτούμενα πεδία.

<Steps>
1. **Όνομα** Το όνομα του λογαριασμού σας. *Δεν χρειάζεται να είναι το πραγματικό σας όνομα, αλλά κάτι που θα βλέπουν οι χρήστες όταν επισκέπτονται τη σελίδα σας στο WinkLinks ή όταν οι προμηθευτές θέλουν να συνδεθούν απευθείας μαζί σας.*
2. **Τύπος οντότητας** Αν είστε εταιρεία ή ιδιώτης. *Χρησιμοποιείται κυρίως για να καθοριστεί η επιλεξιμότητά σας για λήψη χρημάτων.*
3. **Τύπος** Επιλέξτε τον τύπο που ταιριάζει καλύτερα στον λογαριασμό σας. *π.χ. Influencer*
4. **Περιγραφή** Γράψτε μια περιγραφή για τον λογαριασμό σας. *Αυτή θα εμφανίζεται στη σελίδα σας στο WinkLinks και όταν οι προμηθευτές θέλουν να συνδεθούν απευθείας μαζί σας.*
5. **Ιστοσελίδα** Εισάγετε την κύρια ιστοσελίδα σας. *Μπορεί να είναι το URL του λογαριασμού σας στο IG ή η ιστοσελίδα της εταιρείας σας. Πρέπει να είναι το μέρος όπου κάνετε το μεγαλύτερο μέρος της διαδικτυακής σας δουλειάς. Οι προμηθευτές θα επισκέπτονται αυτούς τους συνδέσμους για να αποφασίσουν αν θα συνδεθούν μαζί σας.*
6. **Πόλη** Πού εδρεύετε;
7. **Ετήσια δαπάνη ταξιδιών** Πόσα ξοδεύετε εσείς ή η εταιρεία σας σε ταξίδια κάθε χρόνο.
7. **Συμφωνία** Αποδεχτείτε τη [συμφωνία συνεργάτη](/el/studio/terms-of-service) και τους [όρους πληρωμής](/el/studio/payment-terms) για να συνεχίσετε.
8. Κάντε κλικ στο κουμπί `Save`.
</Steps>

Συγχαρητήρια 🎉 ...δημιουργήσατε τον πρώτο σας λογαριασμό συνεργάτη στο Wink.

## Διαδικτυακή παρουσία

Συνιστάται σε όλους τους συνεργάτες να συμπληρώσουν την διαδικτυακή τους παρουσία. Η διαδικτυακή παρουσία επιτρέπει στα ξενοδοχεία να βλέπουν πού εμφανίζεστε στους χρήστες και το επίπεδο της εμβέλειάς σας.

<Steps>
1. **Τύπος** Εισάγετε τον τύπο διαδικτυακής παρουσίας. *π.χ. Facebook*
2. **Λογαριασμός** Εισάγετε το URL του λογαριασμού σας. *π.χ. https://facebook.com/wink2travel*
3. **Μέγεθος κοινού** Εισάγετε το μέγεθος του κοινού σας. *π.χ. Μπορεί να είναι οι ακόλουθοί σας ή ο αριθμός των υπαλλήλων*
4. **Τύπος κοινού** Εισάγετε τον τύπο κοινού που έχετε. *π.χ. Οικογένεια επειδή το ταξιδιωτικό σας blog εστιάζει στα οικογενειακά ταξίδια*
</Steps>

Μόλις δημιουργηθεί, μπορείτε να προσαρμόσετε περαιτέρω τον λογαριασμό σας συμπληρώνοντας τη διεύθυνσή σας και προσθέτοντας μια φωτογραφία προφίλ.

<Steps>
1. Κάντε κλικ στον νέο σας λογαριασμό συνεργάτη. *Αυτό θα σας μεταφέρει στον πίνακα ελέγχου του λογαριασμού σας.*
2. Κάντε κλικ στο `Account > Profile` στην κύρια γραμμή πλοήγησης.
3. Στη σελίδα Προφίλ, θα μπορείτε να ενημερώσετε το υπάρχον προφίλ σας και να προσθέσετε επιπλέον πληροφορίες.
</Steps>

:::note
Η φωτογραφία προφίλ του λογαριασμού συνεργάτη σας δεν πρέπει να συγχέεται με τη φωτογραφία προφίλ χρήστη σας στο Wink. Η φωτογραφία προφίλ του λογαριασμού συνεργάτη θα εμφανίζεται στη σελίδα σας στο WinkLinks και θα τη βλέπουν οι προμηθευτές που θέλουν να συνδεθούν μαζί σας.

Η φωτογραφία προφίλ χρήστη προορίζεται κυρίως για εσωτερική χρήση.
:::

## Διαχειριστές

Μπορείτε να προσκαλέσετε άλλους χρήστες να βοηθήσουν στη διαχείριση του λογαριασμού συνεργάτη σας. Μόλις ένας χρήστης αποδεχτεί την πρόσκλησή σας, θα έχει πλήρη πρόσβαση στον λογαριασμό σας εκτός από τις πληρωμές. Εσείς θα είστε πάντα ο μόνος υπεύθυνος για τα κεφάλαια που κατέχετε.

<Steps>
1. Επιλέξτε τον λογαριασμό συνεργάτη με τον οποίο θέλετε να εργαστείτε από τη [σελίδα λογαριασμών](https://studio.wink.travel/company/list).
2. Κάντε κλικ στο `Account > Managers` στην κύρια γραμμή πλοήγησης.
3. Προσθέστε το email του χρήστη που θέλετε να ορίσετε ως διαχειριστή λογαριασμού.
4. Κάντε κλικ στον σύνδεσμο `Invite` δίπλα στο email.
</Steps>

Το Wink θα στείλει στον χρήστη ένα email για να αποδεχτεί ή να απορρίψει την πρόσκλησή σας.

## Ειδοποιήσεις

Κατά καιρούς, θέλουμε να επικοινωνήσουμε μαζί σας. Όταν συμβεί αυτό, θα δείτε το κουδούνι στην πάνω δεξιά γωνία να αλλάζει.
Για να διαβάσετε τι σας στείλαμε:

<Steps>
1. Κάντε κλικ στο `Account > Notifications` στην κύρια γραμμή πλοήγησης.
2. Θα δείτε νέες και παλιές ειδοποιήσεις.
3. Επιλέξτε να `Mark as read` ή `Delete` τις ειδοποιήσεις μόλις τις διαβάσετε.
</Steps>

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν `Affiliates` μπορούν να μεταβούν στο [Developers > API > Affiliate](/el/developers/apis/#affiliate-api).
