:::σημείωση
Αυτό το άρθρο προϋποθέτει ότι έχετε ήδη έναν λογαριασμό χρήστη Wink.
:::

[Στούντιο Wink](https://studio.wink.travel)σας συνδέει με ταξιδιωτικά αποθέματα παγκοσμίως. Αυτό το άρθρο σας καθοδηγεί στη δημιουργία και τη διαχείριση των λογαριασμών συνεργατών σας στο Wink.

## Δημιουργία λογαριασμού

Μόλις ολοκληρωθεί η επαλήθευση, θα μεταφερθείτε στη διεύθυνση[σελίδα λογαριασμών](https://studio.wink.travel/company/list)που σας δείχνει όλους τους λογαριασμούς συνεργατών σας.

Κάντε κλικ στο`Create new account`κουμπί για να συνεχίσετε. Θα μεταφερθείτε στο[δημιουργία λογαριασμού](https://studio.wink.travel/company/create)σελίδα.

Συμπληρώστε όλα τα υποχρεωτικά πεδία.

<Steps>
  1. **Ονομα** Το όνομα του λογαριασμού σας. *Αυτό δεν χρειάζεται να είναι το πραγματικό σας όνομα, αλλά θα είναι κάτι που θα βλέπουν οι χρήστες όταν βλέπουν τη σελίδα σας στο WinkLinks ή όταν οι προμηθευτές θέλουν να επικοινωνήσουν απευθείας μαζί σας.*
  2. **Τύπος οντότητας** Είτε είστε εταιρεία είτε ιδιώτης. *Χρησιμοποιείται κυρίως για να προσδιοριστεί ο τρόπος με τον οποίο δικαιούστε να λάβετε κεφάλαια.*
  3. **Τύπος** Επιλέξτε έναν τύπο που ταιριάζει καλύτερα στον λογαριασμό σας. *π.χ. Influencer*
  4. **Περιγραφή** Γράψτε μια περιγραφή για τον λογαριασμό σας. *Αυτό θα εμφανίζεται στη σελίδα σας στο WinkLinks και όταν οι προμηθευτές θέλουν να επικοινωνήσουν απευθείας μαζί σας.*
  5. **Δικτυακός τόπος** Εισαγάγετε τον κύριο ιστότοπό σας. *Αυτή μπορεί να είναι η διεύθυνση URL του λογαριασμού σας στο IG ή ο ιστότοπος της εταιρείας. Θα πρέπει να είναι το μέρος όπου κάνετε το μεγαλύτερο μέρος της διαδικτυακής σας εργασίας. Οι προμηθευτές θα μεταβαίνουν σε αυτούς τους συνδέσμους όταν αποφασίζουν εάν θα συνδεθούν μαζί σας.*
  6. **Πόλη** Πού εδρεύετε;
  7. **Ετήσιες δαπάνες ταξιδιού** Πόσα ξοδεύετε εσείς ή η εταιρεία σας σε ταξίδια κάθε χρόνο.
  8. **Συμφωνία** Αποδεχτείτε μας [συμφωνία συνεργασίας](/studio/terms-of-service) και [όροι πληρωμής](/studio/payment-terms) να συνεχίσω.
  9. Κάντε κλικ στο `Save` κουμπί.
</Steps>


Συγχαρητήρια 🎉 ​​...δημιουργήσατε τον πρώτο σας λογαριασμό συνεργάτη στο Wink.

## Διαδικτυακή παρουσία

Συνιστάται σε όλους τους συνεργάτες να συμπληρώσουν την ηλεκτρονική τους παρουσία. Η ηλεκτρονική παρουσία επιτρέπει στα ξενοδοχεία να βλέπουν πού σας βρίσκουν οι χρήστες και το επίπεδο εμβέλειάς σας.

<Steps>
  1. **Τύπος** Εισαγάγετε τον τύπο της διαδικτυακής παρουσίας. *π.χ. Facebook*
  2. **Λογαριασμός** Εισαγάγετε τη διεύθυνση URL του λογαριασμού σας. *π.χ. https://facebook.com/wink2travel*
  3. **Μέγεθος κοινού** Εισαγάγετε το μέγεθος του κοινού σας. *π.χ. Αυτός μπορεί να είναι οι ακόλουθοί σας ή ο αριθμός των υπαλλήλων*
  4. **Τύπος κοινού** Εισαγάγετε τον τύπο κοινού που έχετε. *π.χ. Οικογένεια επειδή το ταξιδιωτικό σας ιστολόγιο επικεντρώνεται στα οικογενειακά ταξίδια*
</Steps>

Μόλις δημιουργηθεί, μπορείτε να προσαρμόσετε περαιτέρω τον λογαριασμό σας συμπληρώνοντας τη διεύθυνσή σας και προσθέτοντας μια εικόνα προφίλ στον λογαριασμό σας.

<Steps>
  1. Κάντε κλικ στον νέο σας λογαριασμό συνεργάτη. *Αυτό θα σας μεταφέρει στον πίνακα ελέγχου του λογαριασμού σας.*
  2. Κάντε κλικ στο `Account > Profile` στην κύρια γραμμή πλοήγησης.
  3. Στη σελίδα Προφίλ, θα μπορείτε να ενημερώσετε το υπάρχον προφίλ σας και να προσθέσετε επιπλέον πληροφορίες σε αυτό.
</Steps>


:::σημείωση
Η εικόνα προφίλ του λογαριασμού συνεργατών σας δεν πρέπει να συγχέεται με την εικόνα προφίλ χρήστη Wink. Η εικόνα προφίλ του λογαριασμού συνεργατών σας θα εμφανίζεται στη σελίδα σας στο WinkLinks και θα είναι ορατή σε προμηθευτές που επιθυμούν να συνδεθούν μαζί σας.

Η εικόνα προφίλ χρήστη σας προορίζεται κυρίως για εσωτερική χρήση.
:::

## Διευθυντές

Μπορείτε να προσκαλέσετε άλλους χρήστες να σας βοηθήσουν στη διαχείριση του λογαριασμού συνεργατών σας. Μόλις ένας χρήστης αποδεχτεί την πρόσκλησή σας, θα έχει πλήρη πρόσβαση στον λογαριασμό σας, με εξαίρεση την πληρωμή. Θα είστε πάντα ο μόνος υπεύθυνος για τα χρήματα που σας ανήκουν.

<Steps>
  1. Επιλέξτε τον λογαριασμό συνεργάτη με τον οποίο θέλετε να συνεργαστείτε από το [σελίδα λογαριασμών](https://studio.wink.travel/company/list).
  2. Κάντε κλικ στο `Account > Managers` στην κύρια γραμμή πλοήγησης.
  3. Προσθέστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του χρήστη που θέλετε να ορίσετε ως διαχειριστή λογαριασμού.
  4. Κάντε κλικ στο `Invite` σύνδεσμος δίπλα στο email.
</Steps>


Το Wink θα στείλει στον χρήστη ένα email για να αποδεχτεί ή να απορρίψει την πρόσκλησή σας.

## Ειδοποιήσεις

Κατά καιρούς, θέλουμε να επικοινωνήσουμε μαζί σας. Όταν συμβεί αυτό, θα δείτε το κουδουνάκι στην επάνω δεξιά γωνία να αλλάζει.
Για να διαβάσετε τι σας στείλαμε:

<Steps>
  1. Κάντε κλικ στο `Account > Notifications` στην κύρια γραμμή πλοήγησης.
  2. Θα δείτε νέες και παλαιότερες ειδοποιήσεις.
  3. Επιλέξτε να `Mark as read` ή `Delete` τις ειδοποιήσεις μόλις τις διαβάσετε.
</Steps>
