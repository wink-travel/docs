Ένας χάρτης αντιπροσωπεύει είτε έναν μεμονωμένο προμηθευτή (π.χ. ξενοδοχείο ή πάροχο εμπειριών) είτε απόθεμα (δηλαδή τύπο δωματίου, αίθουσα συνεδριάσεων, σπα κ.ά.) είτε μια συλλογή αποθέματος όπως μια επιμελημένη λίστα ή αποθηκευμένη αναζήτηση και εμφανίζει αυτές τις λεπτομέρειες μέσω ενός διαδραστικού χάρτη που περιέχει επισημάνσεις που μπορούν να γίνουν κράτηση.
Ένας χρήστης μπορεί να αλληλεπιδράσει με έναν χάρτη με τους εξής τρόπους:

Το υπόλοιπο αυτού του άρθρου σας καθοδηγεί πώς να δημιουργήσετε, να προσαρμόσετε και να μοιραστείτε τους χάρτες σας.

## Δημιουργία χάρτη

Οι χάρτες μπορούν να δημιουργηθούν με δύο τρόπους:

- **Χάρτης με μία επισήμανση**: Δημιουργήστε έναν χάρτη από κάτι που βρήκατε κατά την [αναζήτηση](/el/studio/search).
- **Χάρτης με πολλές επισημάνσεις**: Δημιουργήστε έναν χάρτη από μια [επιμελημένη λίστα](/el/studio/curated-lists) ή [αποθηκευμένη αναζήτηση](/el/studio/saved-searches).

Κάντε κλικ σε αυτό το κουμπί και θα ανακατευθυνθείτε στη σελίδα της φόρμας χάρτη όπου μπορείτε να ξεκινήσετε να προσαρμόζετε τον χάρτη σας.

### Χάρτης με πολλές επισημάνσεις

Τόσο οι [επιμελημένες λίστες](/el/studio/curated-lists#list-actions) όσο και οι [αποθηκευμένες αναζητήσεις](/el/studio/saved-searches#list-actions) διαθέτουν μια ενέργεια που σας επιτρέπει να δημιουργήσετε έναν χάρτη. Αυτός ο χάρτης θα περιέχει επισημάνσεις για όλο το απόθεμα μέσα σε αυτές τις λίστες.

## Προσαρμογή χάρτη

Η φόρμα σας επιτρέπει να προσαρμόσετε τον χάρτη σας με τους εξής τρόπους:

<Steps>
 1. Δώστε του ένα όνομα για να θυμάστε τι αφορά ο χάρτης.
 2. Επιλέξτε αν θέλετε να επιτρέπεται στον χρήστη να μετακινείται στον χάρτη.
 3. Ορίστε το ύψος του χάρτη σε pixels.
 4. Επιλέξτε αν θέλετε να επιτρέπεται στον χρήστη να κάνει ζουμ μέσα και έξω από τον χάρτη.
 5. Επιλέξτε ένα στυλ χάρτη που ταιριάζει με την ιστοσελίδα στην οποία θα ενσωματώσετε τον χάρτη.
 6. Επιλέξτε χρώμα επισήμανσης.
 7. Επιλέξτε την αρχική όψη κάρτας που θέλετε να βλέπουν πρώτα οι χρήστες. Από προεπιλογή, είναι η εγγενής όψη αυτού του αποθέματος.
 8. Μπορείτε να προσθέσετε δικές σας επισημάνσεις, κύκλους, ορθογώνια και πολύγωνα για να υποδείξετε άλλες τοποθεσίες στον χάρτη που μπορεί να βρουν ενδιαφέρουσες οι χρήστες σας.
 9. Κάντε κλικ στο κουμπί `Αποθήκευση` για να συνεχίσετε.
</Steps>

Μετά την αποθήκευση του χάρτη σας, ανακατευθύνεστε στη σελίδα των χαρτών σας και ο χάρτης σας είναι πλέον έτοιμος να μοιραστεί με τον κόσμο.

Παραπάνω βλέπετε μια εικόνα με όλες τις διαθέσιμες ενέργειες για τον χάρτη σας:

- **Ενημέρωση** Ενημερώνει τη διαμόρφωση του χάρτη σας.
- **Προσθήκη στο WinkLinks** Προσθέτει τον χάρτη στον λογαριασμό σας στο WinkLinks.
- **Ενσωμάτωση** Σας δείχνει πώς να ενσωματώσετε αυτόν τον χάρτη ως [Χάρτη](/el/developers/web-components/#content-loader) στην ιστοσελίδα σας.
- **Χρήση με WordPress** Σας δείχνει πώς να χρησιμοποιήσετε το [plugin WordPress](/el/developers/wordpress) μας για να ενσωματώσετε αυτόν τον χάρτη στην ιστοσελίδα σας.

Καλύπτουμε μερικές από αυτές τις επιλογές, πιο αναλυτικά, παρακάτω.

### Ενσωμάτωση

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Ακολουθεί πώς να ενσωματώσετε τον χάρτη σας στην ιστοσελίδα σας:

- Η γραμμή 3 σας δείχνει πώς να ενσωματώσετε τα στυλ του Wink στην ιστοσελίδα σας.
- Οι γραμμές 6 έως 9 σας δείχνουν πώς να χρησιμοποιήσετε το [wink-content-loader](/el/developers/web-components/#content-loader) Web Component και να του πείτε να φορτώσει ένα διαφημιστικό banner για τον κωδικό σας.
- Η γραμμή 11 σας δείχνει πώς να ενσωματώσετε το Javascript μας στην ιστοσελίδα σας.
- Η γραμμή 13 σας δείχνει πώς να ενσωματώσετε το [wink-app-loader](/el/developers/web-components/#application-loader) Web Component και να του πείτε να φορτώσει τις προτιμήσεις διαμόρφωσης σε επίπεδο σελίδας.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν χάρτες μπορούν να μεταβούν στο [Developers > API > Inventory](/el/developers/apis/#inventory-api).
