Ένας χάρτης αντιπροσωπεύει είτε έναν μόνο προμηθευτή (π.χ. ξενοδοχείο ή πάροχο εμπειρίας) είτε ένα απόθεμα (π.χ. τύπο δωματίου, αίθουσα συσκέψεων, σπα κ.λπ.) είτε μια συλλογή αποθεμάτων, όπως μια επιμελημένη λίστα ή μια αποθηκευμένη αναζήτηση, και εμφανίζει αυτές τις λεπτομέρειες μέσω ενός διαδραστικού χάρτη που περιέχει δείκτες χάρτη με δυνατότητα κράτησης.
Ένας χρήστης μπορεί να αλληλεπιδράσει με έναν χάρτη με τους ακόλουθους τρόπους:

Το υπόλοιπο άρθρο σάς καθοδηγεί στον τρόπο δημιουργίας, προσαρμογής και κοινής χρήσης των χαρτών σας.

## Δημιουργήστε έναν χάρτη

Οι χάρτες μπορούν να δημιουργηθούν με δύο τρόπους:

* **Χάρτης με έναν μόνο δείκτη**: Δημιουργήστε έναν χάρτη από κάτι που βρήκατε ενώ[ερευνητικός](/studio/search).
* **Χάρτης πολλαπλών δεικτών**: Δημιουργήστε έναν χάρτη από ένα[επιμελημένη λίστα](/studio/curated-lists)ή[αποθηκευμένη αναζήτηση](/studio/saved-searches).

### Χάρτης με έναν μόνο δείκτη

  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="Ενέργειες" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

Παραπάνω είναι μια εικόνα που τραβήχτηκε από[Ερευνα](/studio/search)και σας δείχνει μερικά από τα πράγματα που μπορείτε να κάνετε με τα αποτελέσματα αναζήτησής σας. Μία από αυτές τις ενέργειες είναι`Make a map`.
Κάντε κλικ σε αυτό το κουμπί και θα σας ανακατευθύνει στη σελίδα φόρμας χάρτη όπου μπορείτε να ξεκινήσετε να προσαρμόζετε τον χάρτη σας.

### Χάρτης πολλαπλών δεικτών

Και τα δύο[επιμελημένες λίστες](/studio/curated-lists#list-actions)και[αποθηκευμένες αναζητήσεις](/studio/saved-searches#list-actions)συνοδεύεται από μια ενέργεια που σας επιτρέπει να δημιουργήσετε έναν χάρτη. Αυτός ο χάρτης θα περιέχει δείκτες χάρτη για όλο το απόθεμα εντός αυτών των λιστών.

## Προσαρμογή χάρτη

Η φόρμα σάς επιτρέπει να προσαρμόσετε τον χάρτη σας με τους εξής τρόπους:

<Steps>
  1. Δώστε του ένα όνομα για να θυμάστε περί τίνος πρόκειται ο χάρτης.
  2. Επιλέξτε αν θέλετε να επιτρέπεται στον χρήστη να κινείται στον χάρτη.
  3. Ορίστε το ύψος, σε pixel, που θέλετε να έχει ο χάρτης.
  4. Επιλέξτε εάν θέλετε ο χρήστης να έχει τη δυνατότητα μεγέθυνσης και σμίκρυνσης του χάρτη.
  5. Επιλέξτε ένα στυλ χάρτη που ταιριάζει με τον ιστότοπο στον οποίο θα ενσωματώσετε τον χάρτη.
  6. Επιλέξτε ένα χρώμα δείκτη.
  7. Επιλέξτε την αρχική όψη της κάρτας που θέλετε να βλέπουν πρώτα οι χρήστες. Από προεπιλογή, είναι η εγγενής όψη αυτού του αποθέματος.
  8. Μπορείτε να προσθέσετε τους δικούς σας δείκτες, κύκλους, ορθογώνια και πολύγωνα για να υποδείξετε άλλες τοποθεσίες στον χάρτη που μπορεί να βρουν ενδιαφέρουσες οι χρήστες σας.
  9. Κάντε κλικ στο `Save` κουμπί για να συνεχίσετε.
</Steps>


Αφού αποθηκεύσετε τον χάρτη σας, θα μεταφερθείτε στη σελίδα των χαρτών σας και ο χάρτης σας είναι πλέον έτοιμος για κοινή χρήση με τον κόσμο.

## Κοινή χρήση χάρτη

Παραπάνω εμφανίζεται μια εικόνα με όλες τις διαθέσιμες ενέργειες για τον χάρτη σας:

* **Εκσυγχρονίζω**Ενημερώνει τη διαμόρφωση του χάρτη σας.
* **Προσθήκη στο WinkLinks**Προσθέτει τον χάρτη στον λογαριασμό σας WinkLinks.
* **Ενθέτω**Σας δείχνει πώς να ενσωματώσετε αυτόν τον χάρτη ως[Χάρτης](/developers/web-components/#content-loader)στον ιστότοπό σας.
* **Χρήση με WordPress**Σας δείχνει πώς να χρησιμοποιήσετε το[Πρόσθετο WordPress](/developers/wordpress)για να ενσωματώσετε αυτόν τον χάρτη στον ιστότοπό σας.

Καλύπτουμε μερικές από αυτές τις επιλογές, με περισσότερες λεπτομέρειες, παρακάτω.

### Ενθέτω

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Δείτε πώς μπορείτε να ενσωματώσετε τον χάρτη σας στον ιστότοπό σας:

* Η γραμμή 3 σας δείχνει πώς να ενσωματώσετε τα στυλ Wink στον ιστότοπό σας.
* Οι γραμμές 6 έως 9 σας δείχνουν πώς να χρησιμοποιήσετε το[πρόγραμμα φόρτωσης περιεχομένου-wink](/developers/web-components/#content-loader)Web Component και πείτε του να ανακτήσει ένα διαφημιστικό banner για τον κώδικά σας.
* Η γραμμή 11 σας δείχνει πώς να ενσωματώσετε την Javascript μας στον ιστότοπό σας.
* Η γραμμή 13 σας δείχνει πώς να ενσωματώσετε το[wink-app-loader](/developers/web-components/#application-loader)Στοιχείο Web και πείτε του να ανακτήσει τις προτιμήσεις διαμόρφωσης σε επίπεδο σελίδας.
