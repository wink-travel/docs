Οι επιμελημένες λίστες είναι μια λειτουργία που σας επιτρέπει να ομαδοποιείτε και να ταξινομείτε το απόθεμα που έχετε επιλέξει από πράγματα που βρήκατε κατά την αναζήτηση αποθεμάτων ταξιδιών.
Μόλις δημιουργηθεί, μπορείτε να προβάλετε τη λίστα στους χρήστες σας, καθιστώντας την διαθέσιμη προς πώληση ως[πλέγμα](/studio/grids)ή ένα[χάρτης](/studio/maps)μέσα από[WinkLinks](/link-manager/wink-links), μας[Πρόσθετο WordPress](/developers/wordpress/)ή ενσωματώνοντας ένα από τα δικά μας[Στοιχεία Ιστού](/developers/web-components)στον ιστότοπό σας.

Από προεπιλογή, κάθε λογαριασμός συνεργάτη ξεκινά με ένα`Favorites`λίστα που μπορείτε να χρησιμοποιήσετε αμέσως.`Favorites`είναι ειδικό και δεν μπορεί να καταργηθεί.
Όλες οι επιμελημένες λίστες σάς επιτρέπουν να κάνετε τα εξής:

## Δημιουργήστε μια λίστα

<figure>
  <CloudinaryImage width="350" height="219" resize="scale" quality="best" alt="Δημιουργία επιμελημένης λίστας" src="academy/create-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Δημιουργήστε μια νέα φόρμα επιμελημένης λίστας*</figcaption>
</figure>


* Από τον πίνακα ελέγχου, κάντε κλικ στο`Inventory > Curated Lists`στη γραμμή πλοήγησης.
* Δημιουργήστε μια νέα επιμελημένη λίστα κάνοντας κλικ στο`Make a new list`κουμπί.
* Θα σας ζητηθεί να ονομάσετε τη λίστα σας.*Παράδειγμα: Οι 3 κορυφαίες επιλογές στη Νέα Υόρκη*.
* Κάντε κλικ στο`Save`κουμπί.

## Προσθήκη στοιχείου λίστας

Μπορείτε να προσθέσετε ένα στοιχείο λίστας σε οποιαδήποτε λίστα από τα αποτελέσματα αναζήτησης.

* Μετάβαση σε[Ερευνα](/studio/search)
* Αναζητήστε κάτι που θέλετε να προσθέσετε στη λίστα σας.
* Μόλις το βρείτε, κάντε κλικ στο`Actions`σύνδεσμος κάτω αριστερά στην κάρτα αποτελεσμάτων αναζήτησης.
* Κάντε κλικ`Add to...`και, στη συνέχεια, επιλέξτε την επιμελημένη λίστα στην οποία θέλετε να προσθέσετε αυτό το απόθεμα.
* Επιστρέψτε στη σελίδα με τις επιμελημένες λίστες για να επαληθεύσετε ότι το στοιχείο προστέθηκε στη λίστα σας.

Κάθε λίστα έχει κάποιες ενέργειες που μπορείτε να εκτελέσετε σε αυτήν.

* Μετατρέψτε τη λίστα σε[χάρτης](/studio/maps).
* Μετατρέψτε τη λίστα σε[πλέγμα](/studio/grids).
* Αφαίρεση της λίστας (*εκτός από τα Αγαπημένα*).
