Οι επιλεγμένες λίστες είναι μια λειτουργία που σας επιτρέπει να ομαδοποιείτε και να ταξινομείτε αποθέματα που έχετε επιλέξει από πράγματα που βρήκατε κατά την αναζήτηση αποθεμάτων ταξιδιών.  
Μόλις δημιουργηθεί, μπορείτε να κάνετε τη λίστα διαθέσιμη στους χρήστες σας προς πώληση ως [grid](/el/studio/grids) ή [map](/el/studio/maps) μέσω των [WinkLinks](/el/link-manager/wink-links), του [WordPress plugin](/el/developers/wordpress/) μας ή ενσωματώνοντας ένα από τα [Web Components](/el/developers/web-components) μας στην ιστοσελίδα σας.

Από προεπιλογή, κάθε λογαριασμός συνεργάτη ξεκινά με μια λίστα `Favorites` που μπορείτε να χρησιμοποιήσετε αμέσως. Η `Favorites` είναι ειδική και δεν μπορεί να αφαιρεθεί.  
Όλες οι επιλεγμένες λίστες σας επιτρέπουν να κάνετε τα εξής:

- Από τον πίνακα ελέγχου, κάντε κλικ στο `Inventory > Curated Lists` στη γραμμή πλοήγησης.  
- Δημιουργήστε μια νέα επιλεγμένη λίστα κάνοντας κλικ στο κουμπί `Make a new list`.  
- Θα σας ζητηθεί να ονομάσετε τη λίστα σας. *Παράδειγμα: Top 3 NYC Picks*.  
- Κάντε κλικ στο κουμπί `Save`.

## Προσθήκη στοιχείου στη λίστα

Μπορείτε να προσθέσετε ένα στοιχείο σε οποιαδήποτε λίστα από τα αποτελέσματα αναζήτησης.

- Μεταβείτε στο [Search](/el/studio/search)  
- Αναζητήστε κάτι που θέλετε να προσθέσετε στη λίστα σας.  
- Μόλις το βρείτε, κάντε κλικ στο σύνδεσμο `Actions` κάτω αριστερά στην κάρτα αποτελέσματος αναζήτησης.  
- Κάντε κλικ στο `Add to...` και επιλέξτε την επιλεγμένη λίστα στην οποία θέλετε να προσθέσετε αυτό το απόθεμα.  
- Επιστρέψτε στη σελίδα των επιλεγμένων λιστών για να επιβεβαιώσετε ότι το στοιχείο προστέθηκε στη λίστα σας.

Κάθε λίστα έχει κάποιες ενέργειες που μπορείτε να εκτελέσετε σε αυτήν.

- Μετατρέψτε τη λίστα σε [map](/el/studio/maps).  
- Μετατρέψτε τη λίστα σε [grid](/el/studio/grids).  
- Αφαιρέστε τη λίστα (*εκτός από την Favorites*).

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν επιλεγμένες λίστες μπορούν να μεταβούν στο [Developers > API > Browse](/el/developers/apis/#browse-api).

- Διαβάστε για τις δυνατότητες του [map](/el/studio/maps).
