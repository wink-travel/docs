Ένα πρακτορείο είναι ένας κανονικός λογαριασμός Wink που μπορείτε να αντιστοιχίσετε στον λογαριασμό σας για σκοπούς εκπροσώπησης. Παρακάτω αναφέρονται μερικοί λόγοι για τους οποίους θα θέλατε ένα πρακτορείο να σας εκπροσωπεί:

* Πλήρης / Μερική διαχείριση του λογαριασμού σας Wink από τρίτο μέρος.
* Η Talent έχει μια υπάρχουσα συμφωνία με το πρακτορείο της και θέλει ο Wink να χειριστεί την οικονομική πτυχή της διευθέτησης των κεφαλαίων που εισρέουν μέσω του Wink.
* Το πρακτορείο ταλέντων κάνει όλη την ηλεκτρονική δουλειά και διαχειρίζεται όλους τους λογαριασμούς στο Wink.

## Ανάθεση πρακτορείου

Δείτε πώς μπορείτε να αντιστοιχίσετε μια εταιρεία στον λογαριασμό σας.

<Steps>
  1. Συνδεθείτε στον λογαριασμό χρήστη σας στο [Στούντιο Wink](https://studio.wink.travel).
  2. Επιλέξτε τον λογαριασμό συνεργάτη με τον οποίο θέλετε να συνεργαστείτε.
  3. Κάντε κλικ στο `Account > Managers` από την κύρια γραμμή πλοήγησης.
  4. Στο `Agency` στην ενότητα, πληκτρολογήστε το όνομα του οργανισμού που θέλετε να σας εκπροσωπήσει.
  5. Επιλέξτε την αμοιβή πρακτορείου από το αναπτυσσόμενο μενού.
  6. Προαιρετικά, προσθέστε κανόνες λήξης.
  7. Κάντε κλικ στο `Set agency` κουμπί για να συνεχίσετε.
</Steps>


:::σημείωση
Πρέπει να είστε ο κάτοχος του λογαριασμού για να ορίσετε μια εταιρεία.
:::

:::tip\[Παράδειγμα]
Η αμοιβή διαχείρισης προέρχεται από την προμήθεια συνεργατών.

Κράτηση = 100 $, Προμήθεια = 10%, Αμοιβή πρακτορείου = 10% => 100 \* 0,1 \* 0,1 = 1 $
:::

## Κατάργηση ανάθεσης πρακτορείου

Δείτε πώς μπορείτε να καταργήσετε την ανάθεση μιας εταιρείας από τον λογαριασμό σας.

<Steps>
  1. Συνδεθείτε στον λογαριασμό χρήστη σας στο [Στούντιο Wink](https://studio.wink.travel).
  2. Επιλέξτε τον λογαριασμό συνεργάτη με τον οποίο θέλετε να συνεργαστείτε.
  3. Κάντε κλικ στο `Account > Managers` από την κύρια γραμμή πλοήγησης.
  4. Στο `Agency` ενότητα, κάντε κλικ στο <Badge text="x" variant="danger" size="small" /> εικονίδιο στη δεξιά πλευρά του αναπτυσσόμενου μενού πρακτορείου.
  5. Κάντε κλικ στο `Set agency` κουμπί για να συνεχίσετε.
</Steps>


## API

Οι προγραμματιστές που θέλουν να διαχειριστούν το πρακτορείο μέσω API μπορούν να επισκεφθούν τη διεύθυνση[Προγραμματιστές > API > Συνεργάτης](/developers/apis/#affiliate-api).
