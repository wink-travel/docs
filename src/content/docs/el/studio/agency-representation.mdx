Ένα πρακτορείο είναι ένας κανονικός λογαριασμός Wink που μπορείτε να αναθέσετε στον λογαριασμό σας για σκοπούς αντιπροσώπευσης. Παρακάτω αναφέρονται μερικοί λόγοι για τους οποίους θα θέλατε ένα πρακτορείο να σας αντιπροσωπεύει:

- Πλήρης / Μερική διαχείριση του λογαριασμού σας στο Wink από τρίτο μέρος.
- Ο ταλέντος έχει υπάρχουσα συμφωνία με το πρακτορείο του και θέλει το Wink να διαχειρίζεται το οικονομικό μέρος της συμφωνίας του για τα έσοδα που προέρχονται μέσω Wink.
- Το πρακτορείο ταλέντων αναλαμβάνει όλη την online εργασία και διαχειρίζεται όλους τους λογαριασμούς στο Wink.

## Ανάθεση πρακτορείου

Ακολουθεί πώς αναθέτετε ένα πρακτορείο στον λογαριασμό σας.

<Steps>
 1. Συνδεθείτε στον λογαριασμό χρήστη σας στο [Wink Studio](https://studio.wink.travel).
 2. Επιλέξτε τον συνεργαζόμενο λογαριασμό με τον οποίο θέλετε να εργαστείτε.
 3. Κάντε κλικ στο `Account > Managers` από την κύρια γραμμή πλοήγησης.
 4. Στην ενότητα `Agency`, πληκτρολογήστε το όνομα του πρακτορείου που θέλετε να σας αντιπροσωπεύει.
 5. Επιλέξτε την προμήθεια πρακτορείου από το αναπτυσσόμενο μενού.
 5. Προαιρετικά, προσθέστε κανόνες λήξης.
 7. Κάντε κλικ στο κουμπί `Set agency` για να συνεχίσετε.
 </Steps>

 :::note
Πρέπει να είστε ο ιδιοκτήτης του λογαριασμού για να αναθέσετε πρακτορείο.
 :::

 :::tip[Παράδειγμα]
Η προμήθεια πρακτορείου προκύπτει από την προμήθεια του συνεργάτη.

Κράτηση = $100, Προμήθεια = 10%, Προμήθεια πρακτορείου = 10% => 100 * 0.1 * 0.1 = $1
 :::

## Αφαίρεση πρακτορείου

Ακολουθεί πώς αφαιρείτε ένα πρακτορείο από τον λογαριασμό σας.

<Steps>
 1. Συνδεθείτε στον λογαριασμό χρήστη σας στο [Wink Studio](https://studio.wink.travel).
 2. Επιλέξτε τον συνεργαζόμενο λογαριασμό με τον οποίο θέλετε να εργαστείτε.
 3. Κάντε κλικ στο `Account > Managers` από την κύρια γραμμή πλοήγησης.
 4. Στην ενότητα `Agency`, κάντε κλικ στο εικονίδιο <Badge text="x" variant="danger" size="small" /> στη δεξιά πλευρά του αναπτυσσόμενου μενού πρακτορείου.
 5. Κάντε κλικ στο κουμπί `Set agency` για να συνεχίσετε.
 </Steps>


## API

Οι προγραμματιστές που θέλουν να διαχειριστούν πρακτορείο μέσω API μπορούν να μεταβούν στο [Developers > API > Affiliate](/el/developers/apis/#affiliate-api).
