Οι κοινόχρηστοι σύνδεσμοι είναι μια βασική λειτουργία του Wink και προέκυψαν από την ανάγκη να επικοινωνείται το απόθεμα και οι τιμές με δυναμικό τρόπο μέσω του απλούστερου μέσου στο Διαδίκτυο· δηλαδή του υπερσυνδέσμου.

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

Παραπάνω είναι ένα παράδειγμα ενός από τους συνδέσμους μας. Είναι σύντομος και φορητός... ακόμη και αρκετά σύντομος για να τον θυμάστε. Παρότι σύντομος, αυτός ο σύνδεσμος περιέχει μια **εκρηκτική** ποσότητα πληροφοριών, όπως:

- Το App ID σας.
- Το Affiliate ID σας.
- Την εικόνα(ες) και το κείμενο που θα εμφανιστεί στον χρήστη.
- Την εμφάνιση (*δηλαδή θέμα, χρώματα και άλλα*).
- Το δρομολόγιο που θα χρησιμοποιηθεί για το ερώτημα τιμής (*2 ενήλικες, 1 διανυκτέρευση*).
- Το νόμισμα για την εμφάνιση της τιμής.
- Τη γλώσσα για την εμφάνιση του κειμένου.

Το υπόλοιπο αυτού του άρθρου σας καθοδηγεί πώς να δημιουργήσετε, να προσαρμόσετε και να μοιραστείτε τους συνδέσμους σας.

## Προσαρμογή συνδέσμου

 Η φόρμα σας επιτρέπει να προσαρμόσετε τον σύνδεσμό σας με τους εξής τρόπους:

<Steps>
 1. Δώστε του ένα όνομα για να θυμάστε τι αφορά ο σύνδεσμος.
 2. Επιλέξτε την [Προσαρμογή](/el/studio/customization) που θέλετε να εφαρμόσετε σε αυτόν τον σύνδεσμο.
 3. Προσθέστε λέξεις-κλειδιά, χωρισμένες με κόμμα, που θα χρησιμοποιηθούν από [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Προαιρετικά εισάγετε το X account handle ή το Facebook App ID σας.
 5. Προσθέστε τίτλους και περιγραφές στις γλώσσες που θέλετε να δει ο χρήστης.
 6. Επιλέξτε μία ή περισσότερες εικόνες που θα εμφανίζονται όταν μοιράζεστε τον σύνδεσμο σε πλατφόρμες όπως το Facebook ή το LinkedIn.
 7. Επιλέξτε ένα θέμα / επικάλυψη που προσθέτει ένα σχεδιασμό με το brand πάνω στην εικόνα(ες) που επιλέξατε. *Σημείωση: Έτσι εμφανίζονται οι τιμές πάνω στην εικόνα.*
 8. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
 </Steps>

Παραπάνω βλέπετε μια εικόνα με όλες τις διαθέσιμες ενέργειες για τον σύνδεσμό σας:

- **Προεπισκόπηση** Σας δείχνει μια γρήγορη επισκόπηση του πώς έχει ρυθμιστεί ο σύνδεσμος και πώς θα φαίνεται στον χρήστη.
- **Ενημέρωση** Ενημερώνει τη ρύθμιση του συνδέσμου σας.
- **Προσθήκη στα WinkLinks** Προσθέτει τον σύνδεσμο στον λογαριασμό σας στα WinkLinks.
- **Αντιγραφή** Αντιγράφει τον σύνδεσμο στο πρόχειρο για να τον μοιραστείτε εύκολα όπου θέλετε (*συμπεριλαμβανομένων των [WinkLinks](/el/link-manager/wink-links) *).
- **Μοιραστείτε** Ανοίγει ένα νέο παράθυρο με επιλογές για να μοιραστείτε σε πολλά κοινωνικά δίκτυα και πλατφόρμες μηνυμάτων.
- **Κωδικός QR** Δημιουργεί έναν κωδικό QR που μπορείτε να μοιραστείτε σε πλατφόρμες όπως το IG που είναι πιο εστιασμένες σε εικόνες.
- **Ενσωμάτωση** Σας δείχνει πώς να ενσωματώσετε αυτόν τον σύνδεσμο ως [Ad Banner](/el/developers/web-components/#content-loader) στην ιστοσελίδα σας.
- **Χρήση με WordPress** Σας δείχνει πώς να χρησιμοποιήσετε το [WordPress plugin](/el/developers/wordpress) μας για να ενσωματώσετε αυτόν τον σύνδεσμο στην ιστοσελίδα σας.

Καλύπτουμε μερικές από αυτές τις επιλογές, πιο αναλυτικά, παρακάτω.

Αν θέλετε να μοιραστείτε έναν κωδικό QR αντί για σύνδεσμο, κάντε κλικ στο κουμπί `QR code` και θα εμφανιστεί νέο παράθυρο με τον κωδικό QR σας.

Πώς να αποθηκεύσετε τον κωδικό QR σας:

- Αν είστε σε φορητό υπολογιστή, κάντε δεξί κλικ στην εικόνα και επιλέξτε `Αποθήκευση εικόνας`.
- Αν είστε σε κινητό τηλέφωνο:
    - Πατήστε και κρατήστε το δάχτυλό σας πάνω στην εικόνα μέχρι να εμφανιστεί επιλογή για αποθήκευση της εικόνας στις Φωτογραφίες.
    - Κάντε στιγμιότυπο οθόνης του κωδικού QR, κόψτε το ώστε να ταιριάζει και αποθηκεύστε το στις Φωτογραφίες.

### Ενσωμάτωση

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Ακολουθεί πώς να ενσωματώσετε τον σύνδεσμό σας, ως διαφημιστικό banner, στην ιστοσελίδα σας:

- Η γραμμή 3 σας δείχνει πώς να ενσωματώσετε τα στυλ του Wink στην ιστοσελίδα σας.
- Οι γραμμές 6 έως 9 σας δείχνουν πώς να χρησιμοποιήσετε το [wink-content-loader](/el/developers/web-components/#content-loader) Web Component και να του πείτε να φέρει ένα διαφημιστικό banner για τον κωδικό σας.
- Η γραμμή 11 σας δείχνει πώς να ενσωματώσετε το Javascript μας στην ιστοσελίδα σας.
- Η γραμμή 13 σας δείχνει πώς να ενσωματώσετε το [wink-app-loader](/el/developers/web-components/#application-loader) Web Component και να του πείτε να φέρει τις ρυθμίσεις προτιμήσεων σε επίπεδο σελίδας.
