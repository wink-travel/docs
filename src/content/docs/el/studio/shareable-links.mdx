Οι κοινοποιήσιμοι σύνδεσμοι είναι ένα βασικό χαρακτηριστικό του Wink και προέκυψαν από την ανάγκη επικοινωνίας του αποθέματος και της τιμολόγησης με δυναμικό τρόπο μέσω του απλούστερου δυνατού τρόπου στο Διαδίκτυο, δηλαδή μέσω υπερσυνδέσμων.

<h4 align="center">[https://trvl.as/s/3xWCH](https://trvl.as/s/3xWCH)</h4>


Παραπάνω είναι ένα παράδειγμα ενός από τους συνδέσμους μας. Είναι σύντομος και φορητός... ακόμη και αρκετά σύντομος για να τον θυμάστε. Αν και σύντομος, αυτός ο σύνδεσμος διαθέτει ένα**εκρηκτικός**ποσότητα πληροφοριών, όπως:

* Το αναγνωριστικό εφαρμογής σας.
* Το αναγνωριστικό συνεργάτη σας.
* Η/Οι εικόνα/ες και το κείμενο που θα εμφανίζονται στον χρήστη.
* Η εμφάνιση και η αίσθηση (*δηλαδή θέμα, χρώματα και άλλα*).
* Το δρομολόγιο που θα χρησιμοποιηθεί για το ερώτημα τιμών μας (*2 ενήλικες, 1 διανυκτέρευση*).
* Το νόμισμα στο οποίο θα εμφανίζεται η τιμή.
* Η γλώσσα στην οποία θα εμφανίζεται το κείμενο.

Το υπόλοιπο άρθρο σάς καθοδηγεί στον τρόπο δημιουργίας, προσαρμογής και κοινής χρήσης των συνδέσμων σας.

Παραπάνω είναι μια εικόνα που τραβήχτηκε από[Ερευνα](/studio/search)και σας δείχνει μερικά από τα πράγματα που μπορείτε να κάνετε με τα αποτελέσματα αναζήτησής σας. Μία από αυτές τις ενέργειες είναι`Make a link`.
Κάντε κλικ σε αυτό το κουμπί και θα σας ανακατευθύνει στη σελίδα φόρμας σύνδεσης με δυνατότητα κοινής χρήσης, όπου μπορείτε να ξεκινήσετε να προσαρμόζετε τον σύνδεσμό σας.

## Προσαρμογή συνδέσμου

Η φόρμα σάς επιτρέπει να προσαρμόσετε τον σύνδεσμό σας με τους ακόλουθους τρόπους:

<Steps>
  1. Δώστε του ένα όνομα για να θυμάστε περί τίνος πρόκειται ο σύνδεσμος.
  2. Επιλέξτε το [Προσαρμογή](/studio/customization) θέλετε να κάνετε αίτηση σε αυτόν τον σύνδεσμο.
  3. Προσθέστε λέξεις-κλειδιά, διαχωρισμένες με κόμμα, που θα χρησιμοποιηθούν από [Ανιχνευτές ιστού](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
  4. Προαιρετικά, εισαγάγετε το όνομα χρήστη του λογαριασμού σας X ή το αναγνωριστικό εφαρμογής Facebook.
  5. Προσθέστε τίτλους και περιγραφές στις γλώσσες που θέλετε να βλέπει ο χρήστης.
  6. Επιλέξτε μία ή περισσότερες εικόνες που θα εμφανίζονται όταν κοινοποιείτε τον σύνδεσμο σε πλατφόρμες όπως το Facebook ή το LinkedIn.
  7. Επιλέξτε ένα θέμα/επικάλυψη που προσθέτει ένα σχέδιο επωνυμίας πάνω από την/τις εικόνα/ές που επιλέξατε. *Σημείωση: Έτσι εμφανίζονται οι τιμές στο πάνω μέρος της εικόνας.*
  8. Κάντε κλικ στο `Save` κουμπί για να συνεχίσετε.
</Steps>

## Κοινοποίηση συνδέσμου

<figure>
  <CloudinaryImage width="350" height="641" resize="scale" quality="best" alt="Ενέργειες" src="academy/shareable-link-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Ενέργειες σύνδεσης με δυνατότητα κοινής χρήσης*</figcaption>
</figure>


Παραπάνω εμφανίζεται μια εικόνα με όλες τις διαθέσιμες ενέργειες για τον σύνδεσμό σας:

* **Πρεμιέρα**Σας δείχνει μια γρήγορη επισκόπηση του τρόπου διαμόρφωσης του συνδέσμου και πώς θα φαίνεται στον χρήστη.
* **Εκσυγχρονίζω**Ενημερώνει τη διαμόρφωση του συνδέσμου σας.
* **Προσθήκη στο WinkLinks**Προσθέτει τον σύνδεσμο στον λογαριασμό σας WinkLinks.
* **Αντίγραφο**Αντιγράφει τον σύνδεσμο στο πρόχειρό σας, ώστε να μπορείτε εύκολα να τον μοιραστείτε οπουδήποτε θέλετε (*συμπεριλαμβανομένων[WinkLinks](/link-manager/wink-links)*).
* **Μερίδιο**Ανοίγει ένα νέο παράθυρο με επιλογές για κοινή χρήση σε πολλά κοινωνικά δίκτυα και πλατφόρμες ανταλλαγής μηνυμάτων.
* **Κωδικός QR**Δημιουργεί έναν κωδικό QR που μπορείτε να μοιραστείτε σε πλατφόρμες όπως το IG που εστιάζουν περισσότερο στην εικόνα.
* **Ενθέτω**Σας δείχνει πώς να ενσωματώσετε αυτόν τον σύνδεσμο ως[Διαφήμιση Banner](/developers/web-components/#content-loader)στον ιστότοπό σας.
* **Χρήση με WordPress**Σας δείχνει πώς να χρησιμοποιήσετε το[Πρόσθετο WordPress](/developers/wordpress)για να ενσωματώσετε αυτόν τον σύνδεσμο στον ιστότοπό σας.

Καλύπτουμε μερικές από αυτές τις επιλογές, με περισσότερες λεπτομέρειες, παρακάτω.

Κάντε κλικ σε οποιοδήποτε από τα κουμπιά εικονιδίων για να συνεχίσετε να δημοσιεύετε τον σύνδεσμό σας σε αυτό το δίκτυο.*Σημείωση: Ορισμένα κουμπιά θα εμφανίζονται μόνο όταν κάνετε κοινή χρήση από την κινητή σας συσκευή.*

### Κωδικός QR

<figure>
  <CloudinaryImage width="350" height="440" resize="scale" quality="best" alt="Κοινοποίηση στα μέσα κοινωνικής δικτύωσης" src="academy/shareable-link-qr-code" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Κοινοποίηση μέσω κωδικού QR*</figcaption>
</figure>


Αν θέλετε να κοινοποιήσετε έναν κωδικό QR αντί για έναν σύνδεσμο, κάντε κλικ στο`QR code`κουμπί και θα εμφανιστεί ένα νέο παράθυρο με τον κωδικό QR σας.

Πώς να αποθηκεύσετε τον κωδικό QR σας:

* Αν βρίσκεστε στον φορητό υπολογιστή σας, κάντε δεξί κλικ στην εικόνα και επιλέξτε να`Save image`.
* Αν είστε στο κινητό σας τηλέφωνο:
  * Πατήστε παρατεταμένα το δάχτυλό σας πάνω από την εικόνα μέχρι να εμφανιστεί μια επιλογή για να αποθηκεύσετε την εικόνα στις Φωτογραφίες σας.
  * Τραβήξτε ένα στιγμιότυπο οθόνης του κωδικού QR, περικόψτε το για να χωράει και αποθηκεύστε το στις Φωτογραφίες.

### Ενθέτω

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Δείτε πώς μπορείτε να ενσωματώσετε τον σύνδεσμό σας, ως διαφημιστικό banner, στον ιστότοπό σας:

* Η γραμμή 3 σας δείχνει πώς να ενσωματώσετε τα στυλ Wink στον ιστότοπό σας.
* Οι γραμμές 6 έως 9 σας δείχνουν πώς να χρησιμοποιήσετε το[πρόγραμμα φόρτωσης περιεχομένου-wink](/developers/web-components/#content-loader)Web Component και πείτε του να ανακτήσει ένα διαφημιστικό banner για τον κώδικά σας.
* Η γραμμή 11 σας δείχνει πώς να ενσωματώσετε την Javascript μας στον ιστότοπό σας.
* Η γραμμή 13 σας δείχνει πώς να ενσωματώσετε το[wink-app-loader](/developers/web-components/#application-loader)Στοιχείο Web και πείτε του να ανακτήσει τις προτιμήσεις διαμόρφωσης σε επίπεδο σελίδας.
