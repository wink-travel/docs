Όταν ξεκινάτε ως συνεργάτης στο Wink, βρίσκετε κάποια πράγματα που θέλετε να πουλήσετε στην πλατφόρμα και τα μοιράζεστε στα μέσα κοινωνικής δικτύωσης, στο ταξιδιωτικό σας blog, στην εφαρμογή κ.λπ.
Κάποια στιγμή, νιώθετε ότι έχετε γίνει αρκετά έμπειροι με τα εργαλεία μας και έχετε κάνει κάποιες κρατήσεις.
Το επόμενο πράγμα που μπορείτε να κάνετε για να βελτιώσετε τις πωλήσεις σας είναι να αναζητήσετε άμεσες σχέσεις με προμηθευτές.

:::σημείωση
Μπορείτε να αλληλεπιδράσετε με τις ενεργές και τις εκκρεμείς σχέσεις σας κάνοντας κλικ στο`Account > Network`από την κύρια γραμμή πλοήγησης.
:::

Αυτό το άρθρο καλύπτει πότε και πώς να ζητήσετε από έναν προμηθευτή μια άμεση σχέση που θα μπορούσε να είναι επωφελής και για τους δύο σας.

## Πότε να ρωτήσετε

Λάβετε υπόψη ότι για τους περισσότερους συνεργάτες, δεν έχει νόημα να συνδέονται απευθείας με έναν προμηθευτή. Οι περισσότεροι προμηθευτές προσφέρουν ήδη τις καλύτερες προσφορές και προσφορές τους σε όλους στην πλατφόρμα.
Ωστόσο, σας συνιστούμε να προσπαθήσετε να ξεκινήσετε μια άμεση σύνδεση με έναν από τους προμηθευτές μας όταν πληρούται οποιαδήποτε από τις ακόλουθες προϋποθέσεις:

* Έχετε ένα μεγάλο κοινό πιθανών κρατήσεων που ταιριάζουν ιδιαίτερα με έναν συγκεκριμένο προμηθευτή.*Π.χ. Είσαι λάτρης του σκι με λίγη τρελή όρεξη, περιεχόμενο για σκι στην ενδοχώρα, και όλοι οι ακόλουθοί σου είναι λάτρεις του σκι. Είναι έξυπνο να επικοινωνήσεις με ένα χιονοδρομικό κέντρο για να ζητήσεις μια μοναδική προσφορά στα μέσα της εβδομάδας ή κάτι παρόμοιο.*
* Φέρνεις μαζί σου τη δική σου ομάδα ταξιδιωτών σε έναν προορισμό.*Π.χ. Οργανώνετε κατασκηνώσεις γιόγκα και επιβεβαιώνεται η συμμετοχή μιας ομάδας 20 γιόγκι. Με μια άμεση σύνδεση, η κατασκηνωτική εκδήλωση θα μπορούσε να σας δώσει έναν μοναδικό κωδικό προσφοράς που μπορείτε να μοιραστείτε με τους συναδέλφους σας ασκούμενους, ο οποίος τους δίνει το δικαίωμα σε ομαδική έκπτωση.*
* Είστε ταξιδιωτικός πράκτορας ή εταιρεία που ασχολείται με μεγάλο όγκο κρατήσεων.*Τόσο οι ταξιδιωτικοί πράκτορες όσο και οι εταιρείες θεωρούνται συνεργάτες του Wink, αλλά έχουν και τους δικούς τους αποκλειστικούς ιστότοπους.*

## Απαιτήσεις

Πρέπει να έχετε τουλάχιστον 5 προηγούμενες κρατήσεις στο Wink πριν μπορέσετε να ξεκινήσετε ένα αίτημα.
Ορίσαμε αυτήν την απαίτηση για να αποφύγουμε την κατακλυσμό των προμηθευτών με μηνύματα. Σας συνιστούμε να επικεντρωθείτε πρώτα στη δημιουργία ενός ιστορικού επιδόσεων στο Wink. Στη συνέχεια, οι προμηθευτές θα έρθουν σε εσάς.

## Πώς λειτουργεί

Όταν βρείτε έναν προμηθευτή με τον οποίο θα θέλατε να συνεργαστείτε (βλ.[Ερευνα](/studio/search/)) και θέλετε να ξεκινήσετε ένα αίτημα σχέσης, δείτε τι συμβαίνει:

<Steps>
  1. Κάνεις κλικ στο `Request a direct connection` σε οποιοδήποτε από τα αποτελέσματα αναζήτησης προμηθευτή που εμφανίστηκαν για εσάς.
  2. Σας παρουσιάζεται μια φόρμα που σας επιτρέπει να συστηθείτε στον προμηθευτή. Σας παρακαλούμε να είστε ευγενικοί και να επιμείνετε στο θέμα. Πείτε τους ποιος είστε και γιατί επικοινωνείτε μαζί τους. Γιατί πιστεύετε ότι μπορείτε να τους εξυπηρετήσετε και τυχόν λεπτομέρειες σχετικά με αυτό. `the ask` που έχετε αυτή τη στιγμή.
  3. Υποβάλετε τη φόρμα και στη συνέχεια περιμένετε την απάντησή τους. Αυτό μπορεί να διαρκέσει από μία ημέρα έως μία εβδομάδα. Οι προμηθευτές είναι απασχολημένοι, επομένως παρακαλούμε να είστε υπομονετικοί. Δεν μπορείτε να υποβάλετε άλλο αίτημα στον ίδιο προμηθευτή όσο εξακολουθείτε να περιμένετε απάντηση.
</Steps>


Ο προμηθευτής θα απαντήσει με:

* **Αποδέχομαι**Το αίτημά σας έγινε δεκτό και δημιουργήθηκε ένα κανάλι απευθείας πωλήσεων μεταξύ εσάς και αυτών, το οποίο μπορεί να διαμορφωθεί με πολλούς διαφορετικούς τρόπους.
* **Απορρίπτω**Το αίτημά σας απορρίφθηκε και έχετε πρόσβαση στο ίδιο απόθεμα όπως και πριν. Μπορείτε να στείλετε ένα άλλο αίτημα, το νωρίτερο δύο μήνες μετά την απόρριψη.
* **Μη ανταποκρινόμενο**Για οποιονδήποτε λόγο, ο προμηθευτής δεν απαντά καθόλου. Σε αυτό το σημείο, θα πρέπει να εξετάσετε το ενδεχόμενο να αναζητήσετε άλλον προμηθευτή.

## Οφέλη

Μόλις γίνετε δεκτοί, ο προμηθευτής μπορεί να κάνει τα εξής στο νεοσύστατο κανάλι πωλήσεών σας.

* Ορίστε μια διαφορετική έκπτωση συνδρομής για το κοινό σας.
* Ορίστε ένα διαφορετικό επίπεδο προμήθειας για να κερδίσετε περισσότερα.
* Προσθέστε μοναδικές προσφορές που είναι διαθέσιμες μόνο στο κοινό σας.
* Προσθέστε μοναδικό απόθεμα και προσφορές upsells που είναι διαθέσιμες μόνο στο κοινό σας.
