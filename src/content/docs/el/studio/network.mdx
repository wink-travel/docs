Όταν ξεκινάτε ως συνεργάτης στο Wink, βρίσκετε κάποια προϊόντα που θέλετε να πουλήσετε στην πλατφόρμα και τα μοιράζεστε στα social media, το ταξιδιωτικό σας blog, την εφαρμογή σας κ.ά.
Κάποια στιγμή, νιώθετε ότι έχετε αποκτήσει αρκετή εμπειρία με τα εργαλεία μας και έχετε κάποιες κρατήσεις στο ενεργητικό σας.
Το επόμενο βήμα για να βελτιώσετε το επίπεδο των πωλήσεών σας είναι να αναζητήσετε άμεσες σχέσεις με προμηθευτές.

:::note
Μπορείτε να αλληλεπιδράσετε με τις ενεργές και εκκρεμείς σχέσεις σας κάνοντας κλικ στο `Account > Network` από την κύρια γραμμή πλοήγησης.
:::

Αυτό το άρθρο καλύπτει πότε και πώς να ζητήσετε από έναν προμηθευτή μια άμεση σχέση που θα μπορούσε να είναι επωφελής και για τους δύο.

## Πότε να ζητήσετε

Λάβετε υπόψη ότι για τους περισσότερους συνεργάτες, δεν έχει νόημα να συνδεθούν απευθείας με έναν προμηθευτή. Οι περισσότεροι προμηθευτές ήδη προσφέρουν τις καλύτερες προσφορές και upsells σε όλους στην πλατφόρμα.
Ωστόσο, συνιστούμε να προσπαθήσετε να ξεκινήσετε μια άμεση σύνδεση με έναν από τους προμηθευτές μας όταν πληρούνται οποιεσδήποτε από τις παρακάτω προϋποθέσεις:

- Έχετε μεγάλο κοινό πιθανών πελατών που ταιριάζει ιδιαίτερα καλά με έναν συγκεκριμένο προμηθευτή.
*Π.χ. Είστε λάτρης του σκι με εξαιρετικό περιεχόμενο για σκι, back country και όλοι οι ακόλουθοί σας είναι επίσης λάτρεις του σκι. Η σύνδεση με ένα χιονοδρομικό κέντρο για να ζητήσετε μια μοναδική προσφορά μέσα στην εβδομάδα ή παρόμοια είναι έξυπνη.*
- Φέρνετε τη δική σας ομάδα ταξιδιωτών σε έναν προορισμό.
*Π.χ. Οργανώνετε yoga retreats και μια ομάδα 20 γιόγκις έχει επιβεβαιωθεί. Με μια άμεση σύνδεση, το retreat θα μπορούσε να σας δώσει έναν μοναδικό κωδικό προσφοράς που μπορείτε να μοιραστείτε με τους συναδέλφους σας, ο οποίος τους δίνει δικαίωμα σε ομαδική έκπτωση.*
- Είστε ταξιδιωτικός πράκτορας ή εταιρεία που διαχειρίζεται μεγάλο όγκο κρατήσεων.
*Τόσο οι ταξιδιωτικοί πράκτορες όσο και οι εταιρείες θεωρούνται συνεργάτες στο Wink, αλλά έχουν και τις δικές τους αφιερωμένες ιστοσελίδες.*

## Απαιτήσεις

Πρέπει να έχετε τουλάχιστον 5 προηγούμενες κρατήσεις στο Wink πριν μπορέσετε να ξεκινήσετε ένα αίτημα.
Θέσαμε αυτή την απαίτηση για να αποφύγουμε να κατακλύζονται οι προμηθευτές από μηνύματα. Σας συνιστούμε πρώτα να εστιάσετε στο να δημιουργήσετε ένα ιστορικό στο Wink. Τότε οι προμηθευτές θα έρθουν σε εσάς.

## Πώς λειτουργεί

Όταν βρείτε έναν προμηθευτή με τον οποίο θέλετε να συνεργαστείτε (δείτε [Αναζήτηση](/el/studio/search/) ) και θέλετε να ξεκινήσετε ένα αίτημα σχέσης, τα εξής συμβαίνουν:

<Steps>
1. Κάνετε κλικ στο `Request a direct connection` σε οποιοδήποτε από τα αποτελέσματα αναζήτησης του προμηθευτή που εμφανίστηκαν για εσάς.
2. Σας παρουσιάζεται μια φόρμα που σας επιτρέπει να συστηθείτε στον προμηθευτή. Σας παρακαλούμε να είστε ευγενικοί και να πηγαίνετε κατευθείαν στο θέμα. Πείτε τους ποιοι είστε και γιατί επικοινωνείτε. Γιατί πιστεύετε ότι μπορείτε να τους φανείτε χρήσιμοι και οποιεσδήποτε λεπτομέρειες σχετικά με το `αίτημα` που έχετε αυτή τη στιγμή.
3. Υποβάλετε τη φόρμα και μετά περιμένετε την απάντησή τους. Αυτό μπορεί να διαρκέσει από μία μέρα έως και μία εβδομάδα. Οι προμηθευτές είναι απασχολημένοι, οπότε παρακαλούμε να έχετε υπομονή. Δεν μπορείτε να ξεκινήσετε άλλο αίτημα στον ίδιο προμηθευτή όσο περιμένετε απάντηση.
</Steps>

Ο προμηθευτής θα απαντήσει με:

- **Αποδοχή** Το αίτημά σας έγινε αποδεκτό και δημιουργείται ένα άμεσο κανάλι πωλήσεων μεταξύ αυτού και εσάς που μπορεί να διαμορφωθεί με πολλούς διαφορετικούς τρόπους.
- **Απόρριψη** Το αίτημά σας απορρίφθηκε και έχετε πρόσβαση στο ίδιο απόθεμα όπως πριν. Μπορείτε να στείλετε άλλο αίτημα, το νωρίτερο, δύο μήνες μετά την απόρριψη.
- **Μη ανταπόκριση** Για οποιονδήποτε λόγο, ο προμηθευτής δεν απαντά καθόλου. Σε αυτή την περίπτωση, θα πρέπει να σκεφτείτε να αναζητήσετε άλλον προμηθευτή.

## Οφέλη

Μόλις γίνετε αποδεκτοί, ο προμηθευτής μπορεί να κάνει τα εξής στο νέο, άμεσα δημιουργημένο, κανάλι πωλήσεών σας:

- Να ορίσει διαφορετική έκπτωση μέλους για να απολαμβάνει το κοινό σας.
- Να ορίσει διαφορετικό επίπεδο προμήθειας για να κερδίζετε περισσότερα.
- Να προσθέσει μοναδικές προσφορές που είναι διαθέσιμες μόνο στο κοινό σας.
- Να προσθέσει μοναδικό απόθεμα και upsells που είναι διαθέσιμα μόνο στο κοινό σας.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν το δίκτυό τους μπορούν να μεταβούν στο [Developers > API > Sales Channel](/el/developers/apis/#sales-channel-api).
