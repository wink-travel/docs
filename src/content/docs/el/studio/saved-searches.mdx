Οι αποθηκευμένες αναζητήσεις κάνουν αυτό που υποδηλώνει το όνομά τους. Κατά την αναζήτηση αποθεμάτων ταξιδιών, έχετε την επιλογή να αποθηκεύσετε το ερώτημα αναζήτησής σας για μελλοντική χρήση.

## Αποθήκευση του ερωτήματος αναζήτησής σας

Όταν ξεκινήσετε την αναζήτηση αποθέματος (βλ.[Ερευνα](/studio/search)), έχετε τη δυνατότητα να βελτιώσετε τα φίλτρα αναζήτησής σας.
Στο κάτω μέρος της φόρμας ερωτήματος βρίσκεται η επιλογή να`Save search`.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Αποθήκευση αναζήτησης" src="academy/save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Εμφανίζει τον σύνδεσμο για την Αποθήκευση ερωτήματος αναζήτησης*</figcaption>
</figure>


Κάνοντας κλικ σε αυτόν τον σύνδεσμο, θα έχετε την ευκαιρία να ονομάσετε αυτήν την αναζήτηση (*π.χ. Η Βιβλιοθήκη Μαγικών Ξενοδοχείων μου*) κάτι που θα θυμάστε για αργότερα.

<figure>
  <CloudinaryImage width="350" height="329" resize="scale" quality="best" alt="Δώστε ένα όνομα στην αναζήτησή σας" src="academy/name-save-search" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Δώστε ένα όνομα στην αναζήτησή σας*</figcaption>
</figure>


Δώστε του ένα όνομα και κάντε κλικ στο`Save search`κουμπί. Μετά από αυτό, μπορείτε να μεταβείτε στο`Inventory > Saves Searches`από την κύρια γραμμή πλοήγησης για να δείτε την εγγραφή που μόλις δημιουργήσατε.

## Λίστα ενεργειών

Η αποθηκευμένη αναζήτησή σας μπορεί να χρησιμοποιηθεί για να παρουσιάσετε το ταξιδιωτικό σας απόθεμα στους χρήστες σας με τη μορφή πλέγματος ή χάρτη και να κάνετε τα αποτελέσματα αναζήτησής σας να εμφανίζονται με τον τρόπο που θέλετε.

<figure>
  <CloudinaryImage width="700" height="939" resize="scale" quality="best" alt="Αποθηκευμένες ενέργειες αναζήτησης" src="academy/saved-search-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Αποθηκευμένο ερώτημα αναζήτησης χωρίς εφαρμογή φίλτρων*</figcaption>
</figure>


Κάθε αποθηκευμένο ερώτημα αναζήτησης έχει ενέργειες που μπορείτε να εκτελέσετε σε αυτό:

* Μετατρέψτε το ερώτημα αναζήτησης σε ένα[χάρτης](/studio/maps).
* Μετατρέψτε το ερώτημα αναζήτησης σε ένα[πλέγμα](/studio/grids).
* Αφαίρεση της αποθηκευμένης αναζήτησης.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν αποθηκευμένες αναζητήσεις μπορούν να μεταβούν στο[Προγραμματιστές > API > Περιήγηση](/developers/apis/#browse-api).

## Περαιτέρω ανάγνωση

* Διαβάστε για το[Χάρτης](/studio/maps)χαρακτηριστικά.
* Διαβάστε για το[Πλέγμα](/studio/grids)χαρακτηριστικά.
