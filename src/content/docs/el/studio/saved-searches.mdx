Οι αποθηκευμένες αναζητήσεις κάνουν ακριβώς αυτό που υποδηλώνει το όνομά τους. Κατά την αναζήτηση αποθέματος ταξιδιών, έχετε τη δυνατότητα να αποθηκεύσετε το ερώτημα αναζήτησής σας για μελλοντική χρήση.

## Αποθήκευση του ερωτήματος αναζήτησής σας

Όταν ξεκινάτε την αναζήτηση αποθέματος (δείτε [Search](/el/studio/search) ), έχετε τη δυνατότητα να ρυθμίσετε λεπτομερώς τα φίλτρα αναζήτησής σας.  
Στο κάτω μέρος της φόρμας ερωτήματος υπάρχει η επιλογή `Save search`.

Κάνοντας κλικ σε αυτόν τον σύνδεσμο, θα έχετε την ευκαιρία να δώσετε ένα όνομα σε αυτήν την αναζήτηση (*π.χ. Η Βιβλιοθήκη μου με Μαγικά Ξενοδοχεία*) κάτι που θα θυμάστε για αργότερα.

<figure>

Δώστε της ένα όνομα και κάντε κλικ στο κουμπί `Save search`. Μετά από αυτό, μπορείτε να μεταβείτε στο `Inventory > Saves Searches` από τη βασική γραμμή πλοήγησης για να δείτε την πρόσφατα δημιουργημένη εγγραφή σας.

## Ενέργειες λίστας

Η αποθηκευμένη αναζήτησή σας μπορεί να χρησιμοποιηθεί για να εμφανίσει το απόθεμα ταξιδιών στους χρήστες σας με τη μορφή πλέγματος ή χάρτη και να κάνει τα αποτελέσματα της αναζήτησής σας να εμφανίζονται με τον τρόπο που θέλετε.

Κάθε αποθηκευμένο ερώτημα αναζήτησης έχει ενέργειες που μπορείτε να εκτελέσετε:

- Μετατρέψτε το ερώτημα αναζήτησης σε [χάρτη](/el/studio/maps).
- Μετατρέψτε το ερώτημα αναζήτησης σε [πλέγμα](/el/studio/grids).
- Αφαιρέστε την αποθηκευμένη αναζήτηση.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν αποθηκευμένες αναζητήσεις μπορούν να μεταβούν στο [Developers > API > Browse](/el/developers/apis/#browse-api).
