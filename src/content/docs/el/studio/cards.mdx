Μια κάρτα αντιπροσωπεύει έναν μόνο προμηθευτή (π.χ. ξενοδοχείο ή πάροχο εμπειρίας) ή απόθεμα (π.χ. τύπο δωματίου, αίθουσα συσκέψεων, σπα κ.λπ.) και εμφανίζει τις λεπτομέρειες μέσω ενός διαδραστικού, κρατήσιμου περιβάλλοντος εργασίας χρήστη κάρτας.
Ένας χρήστης μπορεί να αλληλεπιδράσει με μια κάρτα με τους ακόλουθους τρόπους:

* Κάντε κλικ στο`Book`κουμπί.
* Κάντε κλικ στο`Gallery`σύνδεσμος για να δείτε όλες τις φωτογραφίες και τα βίντεο για αυτό το απόθεμα.
* Κάντε κλικ`Best deal`σύνδεσμος, εάν υπάρχει, για να δείτε το δωμάτιο με την καλύτερη τιμή.
* Κάντε κλικ`Hotel details`σύνδεσμος, εάν υπάρχει, για να δείτε πληροφορίες για το ακίνητο.

Μια κάρτα υποστηρίζει πολλαπλές όψεις για την ελαχιστοποίηση του μεγέθους της και την κατηγοριοποίηση των δεδομένων, καθιστώντας την εύκολα αναγνώσιμη από τον χρήστη και σας δίνει μεγαλύτερη πιθανότητα να μετατρέψετε τον χρήστη σε κράτηση.

Μια κάρτα έχει δύο ή τρεις όψεις:

* Μια κάρτα ξενοδοχείου εμφανίζει τα στοιχεία του καταλύματος στο μπροστινό μέρος και τον καλύτερο τύπο δωματίου στο πίσω μέρος. Η τιμή, και στις δύο πλευρές, εμφανίζει την καλύτερη τιμή για το ξενοδοχείο.
* Μια κάρτα τύπου δωματίου εμφανίζει τα στοιχεία του τύπου δωματίου στο μπροστινό μέρος και τα στοιχεία του καταλύματος στο πίσω μέρος. Η τιμή εμφανίζει την τιμή του τύπου δωματίου στο μπροστινό μέρος και την καλύτερη τιμή για το ξενοδοχείο στο πίσω μέρος.
* Όλες οι άλλες κάρτες (π.χ. εστιατόριο σπα κ.λπ.) έχουν 3 όψεις. Παράδειγμα: Τα δεδομένα σπα εμφανίζονται στην πρώτη σελίδα με την καλύτερη τιμή του σπα. Τα δεδομένα τύπου δωματίου για το δωμάτιο με την καλύτερη τιμή βρίσκονται στη δεύτερη όψη. Τα δεδομένα καταλύματος μαζί με το δωμάτιο με την καλύτερη τιμή βρίσκονται στην τρίτη όψη της κάρτας.

Παραπάνω είναι ένα παράδειγμα της κάρτας τύπου δωματίου μας. Περιέχει στοιχεία καταλύματος, αξιολόγησης και δωματίου, καθώς και πολιτική ακύρωσης και γεύματα.

Το υπόλοιπο άρθρο σάς καθοδηγεί στον τρόπο δημιουργίας, προσαρμογής και προβολής της κάρτας στους χρήστες σας.

## Δημιουργήστε μια κάρτα

  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="Ενέργειες" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

Παραπάνω είναι μια εικόνα που τραβήχτηκε από[Ερευνα](/studio/search)και σας δείχνει μερικά από τα πράγματα που μπορείτε να κάνετε με τα αποτελέσματα αναζήτησής σας. Μία από αυτές τις ενέργειες είναι`Make a card`.
Κάντε κλικ σε αυτό το κουμπί και θα σας ανακατευθύνει στη σελίδα φόρμας κάρτας μας, όπου μπορείτε να ξεκινήσετε να προσαρμόζετε την κάρτα σας.

## Προσαρμογή κάρτας

Η φόρμα σάς επιτρέπει να προσαρμόσετε την κάρτα σας με τους ακόλουθους τρόπους:

<Steps>
  1. Δώστε της ένα όνομα για να θυμάστε περί τίνος πρόκειται η κάρτα.
  2. Επιλέξτε το [Προσαρμογή](/studio/customization) θέλετε να υποβάλετε αίτηση για αυτήν την κάρτα.
  3. Επιλέξτε την αρχική όψη της κάρτας που θέλετε να βλέπουν πρώτα οι χρήστες. Από προεπιλογή, είναι η εγγενής όψη αυτού του αποθέματος.
  4. Επιλέξτε το <Badge text="badge" variant="tip" size="small" /> που θέλετε να χρησιμοποιήσετε στην κάρτα. Ένα σήμα επιτρέπει στους χρήστες να συγκρίνουν το απόθεμα με βάση μια συγκεντρωτική μέτρηση, όπως `eco-friendly`.
  5. Προσθέστε λέξεις-κλειδιά, διαχωρισμένες με κόμμα, που θα χρησιμοποιηθούν από [Ανιχνευτές ιστού](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
  6. Προσθέστε τίτλους και περιγραφές στις γλώσσες που θέλετε να βλέπει ο χρήστης.
  7. Επιλέξτε μία ή περισσότερες εικόνες που θα εμφανίζονται όταν οι χρήστες κάνουν κλικ στο `Gallery` σύνδεσμος.
  8. Κάντε κλικ στο `Save` κουμπί για να συνεχίσετε.
</Steps>


Αφού αποθηκεύσετε την κάρτα σας, θα μεταφερθείτε στη σελίδα των καρτών σας και η κάρτα σας είναι πλέον έτοιμη για κοινή χρήση με τον κόσμο.

Παραπάνω εμφανίζεται μια εικόνα με όλες τις διαθέσιμες ενέργειες για την κάρτα σας:

* **Εκσυγχρονίζω**Ενημερώνει τη διαμόρφωση της κάρτας σας.
* **Προσθήκη στο WinkLinks**Προσθέτει την κάρτα στον λογαριασμό σας WinkLinks.
* **Ενθέτω**Σας δείχνει πώς να ενσωματώσετε αυτήν την κάρτα ως[Κάρτα](/developers/web-components/#content-loader)στον ιστότοπό σας.
* **Χρήση με WordPress**Σας δείχνει πώς να χρησιμοποιήσετε το[Πρόσθετο WordPress](/developers/wordpress)για να ενσωματώσετε αυτήν την κάρτα στον ιστότοπό σας.

Καλύπτουμε μερικές από αυτές τις επιλογές, με περισσότερες λεπτομέρειες, παρακάτω.

### Ενθέτω

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="GUEST_ROOM"
       id="2de7ee9c-61c9-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Δείτε πώς μπορείτε να ενσωματώσετε την κάρτα σας στον ιστότοπό σας:

* Η γραμμή 3 σας δείχνει πώς να ενσωματώσετε τα στυλ Wink στον ιστότοπό σας.
* Οι γραμμές 6 έως 9 σας δείχνουν πώς να χρησιμοποιήσετε το[πρόγραμμα φόρτωσης περιεχομένου-wink](/developers/web-components/#content-loader)Στοιχείο Web και πείτε του να ανακτήσει μια κάρτα δωματίου επισκεπτών για τον κωδικό σας.
* Η γραμμή 11 σας δείχνει πώς να ενσωματώσετε την Javascript μας στον ιστότοπό σας.
* Η γραμμή 13 σας δείχνει πώς να ενσωματώσετε το[wink-app-loader](/developers/web-components/#application-loader)Στοιχείο Web και πείτε του να ανακτήσει τις προτιμήσεις διαμόρφωσης σε επίπεδο σελίδας.
