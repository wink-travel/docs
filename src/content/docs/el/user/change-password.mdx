Αν θέλετε να αλλάξετε τον κωδικό σας για τον λογαριασμό χρήστη Wink, ακολουθήστε αυτά τα βήματα:

## Ενημέρωση κωδικού

<Steps>
1. Πρέπει να είστε συνδεδεμένοι σε μία από τις ιστοσελίδες μας.
2. Κάντε κλικ στο εικονίδιο του προφίλ σας, στην πάνω δεξιά γωνία και θα εμφανιστεί ένα μενού.
3. Κάντε κλικ στον σύνδεσμο `Αλλαγή κωδικού`.
4. Ακολουθήστε τις οδηγίες για να αλλάξετε τον κωδικό σας.
5. Κάντε κλικ στο `Αποθήκευση`.
</Steps>

Την επόμενη φορά που θα συνδεθείτε, θα χρειαστείτε τον νέο σας κωδικό.

## Ξεχάσατε τον κωδικό

Μερικές φορές είναι απαραίτητο να επαναφέρετε τον κωδικό χρήστη.

Ακολουθούν τα βήματα για την επαναφορά του κωδικού χρήστη:

<Steps>
1. Κάντε κλικ στον σύνδεσμο `Ξεχάσατε τον κωδικό` από τη σελίδα σύνδεσης.
2. Πληκτρολογήστε το όνομα χρήστη / το e-mail για τον λογαριασμό που θέλετε να επαναφέρετε.
3. Κάντε κλικ στο κουμπί `Ξεχάσατε τον κωδικό`. *Θα ανακατευθυνθείτε στη φόρμα επαναφοράς κωδικού.*
4. Ελέγξτε το e-mail σας.
</Steps>

2. Επιβεβαιώστε τον νέο σας κωδικό.
3. Κάντε κλικ στο κουμπί `Επαναφορά κωδικού`. *Θα ανακατευθυνθείτε πίσω στη σελίδα σύνδεσης.*
</Steps>
