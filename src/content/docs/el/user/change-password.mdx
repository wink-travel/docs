Αν θέλετε να αλλάξετε τον κωδικό πρόσβασής σας για τον λογαριασμό χρήστη Wink, ακολουθήστε τα εξής βήματα:

## Ενημέρωση κωδικού πρόσβασης

<Steps>
  1. Πρέπει να είστε συνδεδεμένοι σε έναν από τους ιστότοπούς μας.
  2. Κάντε κλικ στο εικονίδιο του προφίλ σας, στην επάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
  3. Κάντε κλικ στο `Change password` σύνδεσμος.
  4. Ακολουθήστε τις οδηγίες για να αλλάξετε τον κωδικό πρόσβασής σας.
  5. Κλικ `Save`.
</Steps>


Την επόμενη φορά που θα συνδεθείτε, θα χρειαστείτε τον νέο σας κωδικό πρόσβασης.

## Ξέχασα τον κωδικό πρόσβασης

Μερικές φορές είναι απαραίτητο να επαναφέρετε τον κωδικό πρόσβασης χρήστη.

Ακολουθούν τα βήματα για την επαναφορά του κωδικού πρόσβασης χρήστη:

<Steps>
  1. Κάντε κλικ στο `Forgot password` σύνδεσμο από τη σελίδα σύνδεσης.
  2. Πληκτρολογήστε το όνομα χρήστη / email για τον λογαριασμό που θέλετε να επαναφέρετε.
  3. Κλικ `Forgot password` κουμπί. *Θα ανακατευθυνθείτε στη φόρμα επαναφοράς κωδικού πρόσβασης.*
  4. Ελέγξτε το ηλεκτρονικό σας ταχυδρομείο.
</Steps>

<Steps>
  1. Επικολλήστε τον κωδικό που λάβατε στο email στη νέα φόρμα μαζί με τον νέο σας κωδικό πρόσβασης.
  2. Επιβεβαιώστε τον νέο σας κωδικό πρόσβασης.
  3. Κλικ `Reset password` κουμπί. *Θα ανακατευθυνθείτε πίσω στη σελίδα σύνδεσης.*
</Steps>

Ο κωδικός πρόσβασής σας επαναφέρθηκε με επιτυχία. Μπορείτε να συνδεθείτε στο Wink με τα νέα σας διαπιστευτήρια.
