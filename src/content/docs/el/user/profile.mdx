Αν θέλετε να ενημερώσετε το προφίλ χρήστη σας για τον λογαριασμό χρήστη Wink, ακολουθήστε τα εξής βήματα:

<Steps>
  1. Πρέπει να είστε συνδεδεμένοι σε έναν από τους ιστότοπούς μας.
  2. Κάντε κλικ στο εικονίδιο του προφίλ σας, στην επάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
  3. Κάντε κλικ στο `Profile` σύνδεσμος.
  4. Ακολουθήστε τις οδηγίες για να ενημερώσετε το `e-mail`, `first` και `last name`.
  5. Μπορείτε επίσης να ανεβάσετε ένα νέο `profile picture` του εαυτού σου.
  6. Κλικ `Save`.
</Steps>


:::σημείωση
Η νέα σας εικόνα προφίλ θα εμφανιστεί την επόμενη φορά που θα αποσυνδεθείτε εντελώς από το Wink.
:::
