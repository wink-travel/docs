Ένας χρήστης του Wink μπορεί να επιλέξει να σας επιτρέψει να διαχειριστείτε έναν από τους λογαριασμούς του σε ακίνητο, συνεργάτη ή ταξιδιωτικό πράκτορα. Όταν συμβεί αυτό, θα λάβετε ένα email με ένα αίτημα πρόσκλησης.

import { Steps } from '@astrojs/starlight/components';

Για να δείτε τις τρέχουσες προσκλήσεις σας στο Wink, ακολουθήστε τα εξής βήματα:

<Steps>
  1. Πρέπει να είστε συνδεδεμένοι σε έναν από τους ιστότοπούς μας.
  2. Κάντε κλικ στο εικονίδιο του προφίλ σας, στην επάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
  3. Κάντε κλικ στο `Invites` σύνδεσμος.
  4. Αλληλεπιδράστε με τις προσκλήσεις σας είτε με `Accepting` ή `Rejecting` τους.
</Steps>
