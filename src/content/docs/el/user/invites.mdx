Ένας χρήστης Wink μπορεί να επιλέξει να σας επιτρέψει να διαχειριστείτε έναν από τους λογαριασμούς του για ακίνητα, συνεργάτες ή ταξιδιωτικά πρακτορεία. Όταν συμβεί αυτό, θα λάβετε ένα email με αίτημα πρόσκλησης.

import { Steps } from '@astrojs/starlight/components';

Για να δείτε τις τρέχουσες προσκλήσεις σας στο Wink, ακολουθήστε αυτά τα βήματα:

<Steps>
1. Πρέπει να έχετε συνδεθεί σε μία από τις ιστοσελίδες μας.
2. Κάντε κλικ στο εικονίδιο του προφίλ σας, στην επάνω δεξιά γωνία και θα εμφανιστεί ένα μενού.
3. Κάντε κλικ στον σύνδεσμο `Προσκλήσεις`.
4. Αλληλεπιδράστε με τις προσκλήσεις σας είτε `Αποδεχόμενοι` είτε `Απορρίπτοντάς` τες.
</Steps>
