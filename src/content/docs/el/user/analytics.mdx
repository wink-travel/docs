Η γνώση είναι η κινητήρια δύναμη του Wink. Σας λέει πώς αποδίδουν όλοι, και όλα, στο Wink, μέχρι και την ώρα. Πριν από το Wink, τα ξενοδοχεία εγγράφονταν σε έναν τρίτο πάροχο και τους πλήρωναν πολλά χρήματα για να έχουν πρόσβαση σε αυτό που η ομάδα διαχείρισης εσόδων αποκαλεί,`the comp set`; που μεταφράζεται ως:`How does my property compare with another property?`.

Στο Wink, έχετε πρόσβαση σε αναλυτικά στοιχεία σε επίπεδο πλατφόρμας δωρεάν. Αυτό σας επιτρέπει να παρακολουθείτε την απόδοση οποιουδήποτε συνεργάτη, ταξιδιωτικού πράκτορα ή καταλύματος σε οποιαδήποτε μέτρηση θέλετε... όσο θέλετε. Μπορείτε επίσης να έχετε πρόσβαση σε ευρύτερες τάσεις ανά περιοχή για να δείτε ποιος τύπος προϊόντος πωλείται πού και πότε. Στα σωστά χέρια, αυτή η πληροφορία μπορεί να βελτιώσει τις δεξιότητες πωλήσεων οποιουδήποτε χρήστη του Wink.

## Πρόσβαση σε γραφήματα

Μπορείτε να έχετε πρόσβαση στα διαγράμματά σας από οποιονδήποτε από τους ιστότοπούς μας. Για να αποκτήσετε πρόσβαση στα αναλυτικά σας διαγράμματα, ακολουθήστε τα εξής βήματα:

<Steps>
  1. Πρέπει να είστε συνδεδεμένοι σε έναν από τους ιστότοπούς μας.
  2. Κάντε κλικ στο εικονίδιο του προφίλ σας, στην επάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
  3. Κάντε κλικ στο `Analytics` σύνδεσμος.
  4. Θα ανακατευθυνθείτε στα γραφήματά σας.
</Steps>


:::σημείωση
Από προεπιλογή, δημιουργούμε απλά γραφήματα για κάθε κατάλυμα, ταξιδιωτικό πράκτορα και λογαριασμό συνεργάτη που δημιουργείτε, ώστε να μπορείτε εύκολα να παρακολουθείτε την απόδοση των δικών σας λογαριασμών.
:::

## Δημιουργήστε ένα γράφημα

Όταν θέλετε να ξεκινήσετε να δημιουργείτε τα δικά σας γραφήματα, μεταβείτε στα γραφήματά σας, όπως περιγράφεται παραπάνω, και κάντε κλικ στο`Create chart`κουμπί.

Παρακάτω παρατίθενται τα απλά βήματα για τη δημιουργία ενός αναλυτικού γραφήματος, ακολουθούμενα από πιο λεπτομερείς οδηγίες.

<Steps>
  1. **Χρονικές σειρές** Επιλέξτε μια χρονοσειρά για το γράφημά σας. *Π.χ. Καθημερινά*.
  2. **Τύπος παρακολούθησης** Επιλέξτε αν θα παρακολουθείτε ένα σταθερό εύρος ημερομηνιών ή συνεχόμενα. *Π.χ. `Continuous`*.
     * Συνεχής: Επιλογή μονάδων χρόνου. *Π.χ. 7 ημέρες*
     * Διορθώθηκε: Εισαγωγή εύρους ημερομηνίας / ώρας *Π.χ. 22 Σεπτεμβρίου 2023 16:00 - 29 Σεπτεμβρίου 2023 16:00*.
  3. **Εμφάνιση νομίσματος** Εάν έχετε να κάνετε με ποσά στο γράφημά σας, επιλέξτε σε ποια νομίσματα θέλετε να τα δείτε. *Π.χ. USD*.
  4. **Παρακολούθηση ιδιότητας** Παρακολουθήστε μια συγκεκριμένη ιδιότητα. *Προαιρετικός*
  5. **Παρακολούθηση καναλιού πωλήσεων** Παρακολουθήστε την τοποθεσία του καναλιού πωλήσεων ανά ήπειρο, χώρα ή πόλη. *Προαιρετικός*
  6. **Φιλτράρισμα τύπου καναλιού πωλήσεων**Φιλτράρετε με βάση έναν συγκεκριμένο τύπο καναλιού πωλήσεων. *Π.χ. Influencer*. *Προαιρετικός*
  7. **Παρακολούθηση συγκεκριμένου καναλιού πωλήσεων** Παρακολουθήστε ένα συγκεκριμένο κανάλι πωλήσεων. *Προαιρετικός*
  8. **Ταξιδιώτης που παρακολουθεί** Παρακολουθήστε την τοποθεσία του ταξιδιώτη ανά ήπειρο, χώρα ή πόλη. *Προαιρετικός*
  9. **Επιλογή σημείων δεδομένων** Επιλέξτε τα σημεία δεδομένων που θέλετε να παρακολουθήσετε. *Π.χ. Κρατήσεις & Ακυρώσεις*
  10. **Δεδομένα ομάδας** Αντί να φιλτράρετε με βάση τα ακίνητα ή τα κανάλια πωλήσεων, μπορείτε επίσης να ομαδοποιήσετε για να συγκεντρώσετε δεδομένα με βάση τα σημεία δεδομένων. *Π.χ. Ομάδα σε κανάλια πωλήσεων*
  11. **Όνομα γραφήματος** Δώστε στο γράφημά σας ένα περιγραφικό όνομα. *Π.χ. κρατήσεις Travelbug στο Hourly*
  12. Κλικ `Save` κουμπί.
</Steps>


## Πώς λειτουργεί

Τα γραφήματα υποβάλλονται σε επεξεργασία σε 4 βήματα. Αυτά τα βήματα περιγράφονται παρακάτω.

### Τύπος χρονοσειράς

Επιλέγοντας ένα σταθερό εύρος ημερομηνιών ή ένα συνεχές χρονικό διάστημα, αρχίζετε να εστιάζετε στο ΠΟΤΕ από το οποίο θέλετε να δείτε δεδομένα.

### Παρακολούθηση / Φιλτράρισμα

Επιλέγοντας ένα ακίνητο, ένα κανάλι πωλήσεων κ.λπ., αρχίζετε να εστιάζετε σε ΠΟΙΟΝ θέλετε να δείτε δεδομένα. Ουσιαστικά φιλτράρετε όλα τα στοιχεία που δεν θέλετε να δείτε.

### Σημεία δεδομένων

Επιλέγοντας σημεία δεδομένων, αρχίζετε να εστιάζετε σε ΤΙ θέλετε να δείτε δεδομένα.

:::σημείωση
Τα σημεία δεδομένων θα πρέπει να είναι «συνεργικά» με τρόπο που να ταιριάζουν μαζί σε ένα γράφημα. Για παράδειγμα, δεν θα συμπεριλάβετε τις «Προβολές σελίδας» και τα «Συνολικά καθαρά έσοδα» στο ίδιο γράφημα.
:::

### Ομαδοποίηση<Badge text="New" variant="note" />


Η ομαδοποίηση είναι ένας ισχυρός τρόπος για τη συγκέντρωση και την εστίαση δεδομένων.

:::συμβουλή
Για παράδειγμα, αν είστε ένα κατάλυμα και θέλετε να παρακολουθείτε πόσες κρατήσεις πραγματοποίησε κάθε ένα από τα κανάλια πωλήσεών σας, θα πρέπει να φιλτράρετε με βάση το κατάλυμά σας (παραπάνω), να επιλέξετε να παρακολουθείτε τις «Κρατήσεις» (παραπάνω) και να ομαδοποιήσετε το κανάλι με βάση το «Κανάλι πωλήσεων» στην ενότητα Ομαδοποίηση.
:::

## Αφαίρεση γραφήματος

Για να καταργήσετε ένα γράφημα, ακολουθήστε τα εξής βήματα:

<Steps>
  1. Κάντε κλικ στο `Actions` σύνδεσμος που υπάρχει κάτω από κάθε γράφημα.
  2. Κάντε κλικ στο `Remove` κουμπί
</Steps>


Το γράφημά σας έχει διαγραφεί.
