Η γνώση είναι η ζωτική δύναμη του Wink. Σας δείχνει πώς αποδίδει ο καθένας και τα πάντα στο Wink, μέχρι και την ώρα. Πριν το Wink, τα ξενοδοχεία συνδρομοποιούνταν σε τρίτους παρόχους και πλήρωναν πολλά χρήματα για να έχουν πρόσβαση σε αυτό που η ομάδα διαχείρισης εσόδων αναφέρεται ως `το comp set`, που μεταφράζεται σε: `Πώς συγκρίνεται το ακίνητό μου με ένα άλλο ακίνητο;`.

Στο Wink, έχετε δωρεάν πρόσβαση σε αναλύσεις σε επίπεδο πλατφόρμας. Αυτό σας επιτρέπει να παρακολουθείτε την απόδοση οποιουδήποτε συνεργάτη, ταξιδιωτικού πράκτορα ή ακινήτου σε οποιοδήποτε μετρικό θέλετε... όσο θέλετε. Μπορείτε επίσης να έχετε πρόσβαση σε ευρύτερες τάσεις ανά περιοχή για να δείτε τι είδους προϊόν πωλείται πού και πότε. Στα σωστά χέρια, αυτή η γνώση μπορεί να βελτιώσει τις πωλησιακές δεξιότητες κάθε χρήστη του Wink.

## Πρόσβαση στα διαγράμματα

Μπορείτε να έχετε πρόσβαση στα διαγράμματά σας από οποιαδήποτε από τις ιστοσελίδες μας. Για να αποκτήσετε πρόσβαση στα αναλυτικά σας διαγράμματα, ακολουθήστε τα εξής βήματα:

<Steps>
1. Πρέπει να έχετε συνδεθεί σε μία από τις ιστοσελίδες μας.
2. Κάντε κλικ στο εικονίδιο προφίλ σας, στην πάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
3. Κάντε κλικ στον σύνδεσμο `Analytics`.
4. Θα ανακατευθυνθείτε στα διαγράμματά σας.
</Steps>

:::note
Κατά προεπιλογή, δημιουργούμε απλά διαγράμματα για κάθε ακίνητο, ταξιδιωτικό πράκτορα και λογαριασμό συνεργάτη που δημιουργείτε, ώστε να μπορείτε εύκολα να παρακολουθείτε πώς αποδίδουν οι δικοί σας λογαριασμοί.
:::

## Δημιουργία διαγράμματος

Όταν θέλετε να ξεκινήσετε να δημιουργείτε τα δικά σας διαγράμματα, πηγαίνετε στα διαγράμματά σας, όπως περιγράφεται παραπάνω, και κάντε κλικ στο κουμπί `Create chart`.

Παρακάτω είναι τα απλά βήματα για να δημιουργήσετε ένα αναλυτικό διάγραμμα, ακολουθούμενα από πιο λεπτομερείς οδηγίες.

<Steps>
1. **Χρονική σειρά** Επιλέξτε μια χρονική σειρά για το διάγραμμά σας. *π.χ. Καθημερινά*.
2. **Τύπος παρακολούθησης** Επιλέξτε αν θα παρακολουθείτε ένα σταθερό εύρος ημερομηνιών ή συνεχώς. *π.χ. `Continuous`*.
    - Συνεχής: Επιλέξτε μονάδες χρόνου. *π.χ. 7 ημέρες*
    - Σταθερό: Εισάγετε εύρος ημερομηνιών / χρόνου *π.χ. 22 Σεπτεμβρίου 2023 16:00 - 29 Σεπτεμβρίου 2023 16:00*.
3. **Νόμισμα εμφάνισης** Αν χειρίζεστε ποσά στο διάγραμμά σας, επιλέξτε σε ποια νομίσματα θέλετε να τα βλέπετε. *π.χ. USD*.
4. **Παρακολούθηση ακινήτου** Παρακολουθήστε ένα συγκεκριμένο ακίνητο. *Προαιρετικό*
5. **Παρακολούθηση καναλιού πωλήσεων** Παρακολουθήστε την τοποθεσία καναλιού πωλήσεων ανά ήπειρο, χώρα ή πόλη. *Προαιρετικό*
6. **Φιλτράρισμα τύπου καναλιού πωλήσεων** Φιλτράρετε σε συγκεκριμένο τύπο καναλιού πωλήσεων. *π.χ. Influencer*. *Προαιρετικό*
7. **Παρακολούθηση συγκεκριμένου καναλιού πωλήσεων** Παρακολουθήστε ένα συγκεκριμένο κανάλι πωλήσεων. *Προαιρετικό*
8. **Παρακολούθηση ταξιδιώτη** Παρακολουθήστε την τοποθεσία ταξιδιώτη ανά ήπειρο, χώρα ή πόλη. *Προαιρετικό*
9. **Επιλογή δεδομένων** Επιλέξτε τα δεδομένα που θέλετε να παρακολουθείτε. *π.χ. Κρατήσεις & Ακυρώσεις*
10. **Ομαδοποίηση δεδομένων** Αντί να φιλτράρετε ακίνητα ή κανάλια πωλήσεων, μπορείτε επίσης να ομαδοποιήσετε για να συγκεντρώσετε δεδομένα. *π.χ. Ομαδοποίηση ανά κανάλια πωλήσεων*
11. **Όνομα διαγράμματος** Δώστε στο διάγραμμά σας ένα περιγραφικό όνομα. *π.χ. Κρατήσεις Travelbug ανά ώρα*
12. Κάντε κλικ στο κουμπί `Save`.
</Steps>

## Πώς λειτουργεί

Τα διαγράμματα επεξεργάζονται με 4 βήματα. Αυτά τα βήματα περιγράφονται παρακάτω.

### Τύπος χρονικής σειράς

Επιλέγοντας ένα σταθερό εύρος ημερομηνιών ή μια συνεχή χρονική περίοδο, αρχίζετε να εστιάζετε στο ΠΟΤΕ θέλετε να δείτε δεδομένα.

### Παρακολούθηση / Φιλτράρισμα

Επιλέγοντας ακίνητο, κανάλι πωλήσεων κ.ά., αρχίζετε να εστιάζετε στο ΠΟΙΟΣ θέλετε να δείτε δεδομένα. Ουσιαστικά φιλτράρετε όλα όσα δεν θέλετε να δείτε.

### Δεδομένα

Επιλέγοντας δεδομένα, αρχίζετε να εστιάζετε στο ΤΙ θέλετε να δείτε.

:::note
Τα δεδομένα πρέπει να είναι 'συνεργιστικά' με τέτοιο τρόπο ώστε να ταιριάζουν σε ένα γράφημα μαζί. Για παράδειγμα, δεν θα συμπεριλαμβάνατε 'Προβολές Σελίδας' και 'Συνολικά Καθαρά Έσοδα' στο ίδιο διάγραμμα.
:::

### Ομαδοποίηση <Badge text="Νέο" variant="note" />

Η ομαδοποίηση είναι ένας ισχυρός τρόπος για να συγκεντρώσετε και να εστιάσετε σε δεδομένα.

:::tip
Για παράδειγμα, αν είστε ένα ακίνητο και θέλετε να παρακολουθήσετε πόσες κρατήσεις έκανε κάθε κανάλι πωλήσεών σας, θα φιλτράρετε το ακίνητό σας (παραπάνω), θα επιλέξετε να παρακολουθείτε 'Κρατήσεις' (παραπάνω) και θα ομαδοποιήσετε ανά 'Κανάλι πωλήσεων' στην ενότητα Ομαδοποίηση.
:::

## Αφαίρεση διαγράμματος

Για να αφαιρέσετε ένα διάγραμμα, ακολουθήστε τα εξής βήματα:

<Steps>
1. Κάντε κλικ στον σύνδεσμο `Actions` που υπάρχει κάτω από κάθε διάγραμμα.
2. Κάντε κλικ στο κουμπί `Remove`.
</Steps>

Το διάγραμμά σας έχει διαγραφεί.
