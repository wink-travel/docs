Μπορείτε να δημιουργήσετε έναν λογαριασμό χρήστη μέσω οποιασδήποτε από τις πύλες μας. Μόλις δημιουργηθεί, αυτός ο λογαριασμός μπορεί να χρησιμοποιηθεί σε όλες τις εφαρμογές και τους ιστότοπους του Wink.

## Ταξιδιώτες

Ακολουθήστε τα παρακάτω βήματα:

<Steps>
  1. Κατευθυνθείτε προς [https://ota.wink.travel](https://ota.wink.travel)
  2. Κάντε κλικ στο `Sign-In / Register` στην επάνω δεξιά γωνία
  3. Θα σας ζητηθεί να εισαγάγετε βασικά στοιχεία χρήστη. Συμπληρώστε όλα τα υποχρεωτικά πεδία και κάντε κλικ στο `Save`.
  4. Μπορείτε τώρα να συνδεθείτε στο Wink με το όνομα χρήστη (email) και τον κωδικό πρόσβασής σας.
  5. Μόλις συνδεθείτε, το σύστημα σας ανακατευθύνει πίσω στο σημείο από όπου προήλθατε.
</Steps>

## Εγγραφή

Η δημιουργία ενός λογαριασμού χρήστη Wink σάς παρέχει πρόσβαση σε όλες τις λειτουργίες της πλατφόρμας. Μπορείτε να δημιουργήσετε απεριόριστες ιδιοκτησίες και λογαριασμούς συνεργατών.

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="Σύνδεση" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Φόρμα σύνδεσης*</figcaption>
</figure>


<Steps>
  1. Την πρώτη φορά που θα μεταφερθείτε στον διακομιστή εξουσιοδότησης, θα σας ζητηθεί να εισαγάγετε το όνομα χρήστη και τον κωδικό πρόσβασής σας.
  2. Αντ' αυτού, κάντε κλικ στο `Create account` κουμπί.
  3. Συμπληρώστε όλα τα υποχρεωτικά πεδία και κάντε κλικ `Register` για να συνεχίσετε. Αυτό σας μεταφέρει πίσω στη φόρμα σύνδεσης.
  4. Εισαγάγετε το email/όνομα χρήστη και τον κωδικό πρόσβασής σας για να συνδεθείτε.
  5. Θα ανακατευθυνθείτε πίσω στον ιστότοπο από τον οποίο ήρθατε.
</Steps>
