Μπορείτε να δημιουργήσετε έναν λογαριασμό χρήστη μέσω οποιασδήποτε από τις πύλες μας. Μόλις δημιουργηθεί, αυτός ο λογαριασμός μπορεί να χρησιμοποιηθεί σε όλες τις εφαρμογές και ιστοσελίδες του Wink.

## Ταξιδιώτες

Ακολουθήστε αυτά τα βήματα:

<Steps>
1. Μεταβείτε στο [https://ota.wink.travel](https://ota.wink.travel)
2. Κάντε κλικ στο `Sign-In / Register` στην πάνω δεξιά γωνία
3. Θα σας ζητηθεί να εισάγετε βασικά στοιχεία χρήστη. Συμπληρώστε όλα τα απαιτούμενα πεδία και κάντε κλικ στο `Save`.
4. Τώρα μπορείτε να συνδεθείτε στο Wink με το όνομα χρήστη (e-mail) και τον κωδικό σας.
5. Μόλις συνδεθείτε, το σύστημα σας ανακατευθύνει πίσω από όπου ήρθατε.
</Steps>

## Εγγραφή

Η δημιουργία λογαριασμού χρήστη στο Wink σας δίνει πρόσβαση σε όλες τις λειτουργίες της πλατφόρμας. Μπορείτε να δημιουργήσετε απεριόριστα καταλύματα και λογαριασμούς συνεργατών.

    <figcaption>*Φόρμα σύνδεσης*</figcaption>

<Steps>
1. Την πρώτη φορά που θα μεταβείτε στον διακομιστή εξουσιοδότησής μας, θα σας ζητηθεί να εισάγετε το όνομα χρήστη και τον κωδικό σας.
2. Αντί γι’ αυτό, κάντε κλικ στο κουμπί `Create account`.
3. Συμπληρώστε όλα τα απαιτούμενα πεδία και κάντε κλικ στο `Register` για να συνεχίσετε. Αυτό σας επιστρέφει στη φόρμα σύνδεσης.
4. Εισάγετε το e-mail / όνομα χρήστη και τον κωδικό για να συνδεθείτε.
5. Ανακατευθύνεστε πίσω στην ιστοσελίδα από όπου ήρθατε.
</Steps>
