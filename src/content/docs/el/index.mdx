Αυτός ο ιστότοπος περιέχει όλες τις πληροφορίες που χρειάζεστε για να χρησιμοποιήσετε με επιτυχία όλες τις λειτουργίες και τις υπηρεσίες του Wink. Κάντε κλικ στο[Ξεκινήστε](/getting-started/how-to-use)εάν δεν έχετε χρησιμοποιήσει ποτέ ξανά τον ιστότοπό μας ή δεν έχετε μεταβεί σε μια συγκεκριμένη ενότητα του παρακάτω ιστότοπου.

## Πού θέλεις να πας...

<CardGrid stagger>
  <Card title="Διαχειριζόμενα Κοινωνικά Δίκτυα" icon="seti:info">
    Τα καταλύματα που θέλουν να αξιοποιήσουν τον αυτοματισμό των μέσων κοινωνικής δικτύωσης ως μια στρατηγική μάρκετινγκ υψηλής απόδοσης, απευθύνονται στο [Διαχειριζόμενη ενότητα Κοινωνικών Δικτύων](/managed-social/what-is-managed-social/) αυτού του ιστότοπου.
  </Card>

  <Card title="Πρακτορείο Wink" icon="seti:info">
    Τα ακίνητα που θέλουν έναν στρατό από influencers να τα εκπροσωπούν και να τα πωλούν, πηγαίνουν στο [Τμήμα Πρακτορείου Wink](/agency/what-is-agency/) αυτού του ιστότοπου.
  </Card>

  <Card title="WinkLinks" icon="seti:info">
    Θέλετε να χρησιμοποιήσετε τον διαχειριστή συνδέσμων μας για τη διεύθυνση URL του βιογραφικού σας στο IG; Μεταβείτε στο [Ενότητα WinkLinks](/link-manager/wink-links/) αυτού του ιστότοπου.
  </Card>

  <Card title="Συνεργάτες" icon="seti:info">
    Κάποιος που θέλει να μάθει πώς να κερδίζει παθητικό εισόδημα στο Wink; Επισκεφθείτε το [Ενότητα συνεργατών](/studio/what-is-studio/) αυτού του ιστότοπου.
  </Card>

  <Card title="Μηχανή κρατήσεων" icon="seti:info">
    Θέλετε να μάθετε τα πάντα για τη μηχανή κρατήσεων με λευκή ετικέτα; Μεταβείτε στο [Μηχανή κρατήσεων](/booking-engine/in-a-nutshell) να ανακαλύψω.
  </Card>

  <Card title="Εταιρείες" icon="seti:info">
    Οι εταιρείες που αναζητούν καλύτερες προσφορές σε ξενοδοχεία, απευθύνονται στο [Εταιρικό τμήμα](/corporate/what-is-group) αυτού του ιστότοπου.
  </Card>

  <Card title="Σκηνικά θέατρου" icon="seti:info">
    Τα ακίνητα που θέλουν να ενσωματώσουν τα ακίνητά τους στο Wink, πηγαίνουν στο [Ενότητα ακινήτου](/extranet/what-is-extranet) αυτού του ιστότοπου.
  </Card>

  <Card title="Ταξιδιωτικοί πράκτορες" icon="seti:info">
    Ταξιδιωτικοί πράκτορες που επιθυμούν πρόσβαση σε ειδικές τιμές, μπορούν να επισκεφθούν το [Τμήμα ταξιδιωτικών πρακτόρων](/travel-agent/what-is-agent) αυτού του ιστότοπου.
  </Card>

  <Card title="Πληρωμές" icon="seti:info">
    Όποιος θέλει να μάθει πώς η Wink επεξεργάζεται τις πληρωμές, ας πάει στο [Ενότητα πληρωμών](/payment/what-is-trip-pay).
  </Card>

  <Card title="Διαδικτυακά σεμινάρια" icon="seti:info">
    Δείτε μας να σας καθοδηγούμε στο πώς να κάνετε συγκεκριμένα πράγματα στο Wink σε σύντομα, εύπεπτα βίντεο. Πηγαίνετε στο [Διαδικτυακά σεμινάρια](/webinars/how-tos).
  </Card>

  <Card title="Οδηγοί" icon="seti:info">
    Ή... θέλετε να αποκτήσετε τις γνώσεις που χρειάζεστε και να μάθετε από τους ειδικούς; Επισκεφθείτε το [Οδηγοί](/guides/a-quick-intro).
  </Card>
</CardGrid>
