Αυτός ο ιστότοπος περιέχει όλες τις πληροφορίες που χρειάζεστε για να χρησιμοποιήσετε με επιτυχία όλες τις λειτουργίες και υπηρεσίες του Wink. Κάντε κλικ στο [Ξεκινήστε](/el/getting-started/how-to-use) αν δεν έχετε χρησιμοποιήσει ποτέ τον ιστότοπό μας ή πλοηγηθείτε σε μια συγκεκριμένη ενότητα παρακάτω.

## Πού θέλετε να πάτε...

<CardGrid stagger>
	<Card title="Managed Social" icon="seti:info">
		Ακίνητα που θέλουν να αξιοποιήσουν την αυτοματοποίηση κοινωνικών μέσων ως μια υψηλής απόδοσης στρατηγική μάρκετινγκ, πηγαίνετε στην [ενότητα Managed Social](/el/managed-social/what-is-managed-social/) αυτού του ιστότοπου.
	</Card>
	<Card title="Wink Agency" icon="seti:info">
		Ακίνητα που θέλουν έναν στρατό από influencers να τους εκπροσωπούν και να πουλάνε για λογαριασμό τους, πηγαίνετε στην [ενότητα Wink Agency](/el/agency/what-is-agency/) αυτού του ιστότοπου.
	</Card>
	<Card title="WinkLinks" icon="seti:info">
		Θέλετε να χρησιμοποιήσετε τον δροσερό διαχειριστή συνδέσμων μας για το URL του βιογραφικού σας στο IG; Πηγαίνετε στην [ενότητα WinkLinks](/el/link-manager/wink-links/) αυτού του ιστότοπου.
	</Card>
	<Card title="Affiliates" icon="seti:info">
		Όποιος θέλει να μάθει πώς να κερδίζει παθητικό εισόδημα στο Wink; Πηγαίνετε στην [ενότητα Affiliate](/el/studio/what-is-studio/) αυτού του ιστότοπου.
	</Card>
	<Card title="Booking Engine" icon="seti:info">
		Θέλετε να μάθετε τα βασικά και τις λεπτομέρειες της λευκής ετικέτας μηχανής κρατήσεων μας; Πηγαίνετε στην [Μηχανή Κρατήσεων](/el/booking-engine/in-a-nutshell) για να μάθετε.
	</Card>
	<Card title="Corporations" icon="seti:info">
		Εταιρείες που αναζητούν καλύτερες προσφορές σε ξενοδοχεία, πηγαίνετε στην [εταιρική ενότητα](/el/corporate/what-is-group) αυτού του ιστότοπου.
	</Card>
	<Card title="Properties" icon="seti:info">
		Ακίνητα που θέλουν να εντάξουν τα ακίνητά τους στο Wink, πηγαίνετε στην [ενότητα Property](/el/extranet/what-is-extranet) αυτού του ιστότοπου.
	</Card>
	<Card title="Travel Agents" icon="seti:info">
		Ταξιδιωτικοί πράκτορες που θέλουν πρόσβαση σε ειδικές τιμές, πηγαίνετε στην [ενότητα Travel Agent](/el/travel-agent/what-is-agent) αυτού του ιστότοπου.
	</Card>
	<Card title="Payments" icon="seti:info">
		Όποιος θέλει να μάθει πώς το Wink επεξεργάζεται τις πληρωμές, πηγαίνετε στην [ενότητα Payments](/el/payment/what-is-trip-pay).
	</Card>
	<Card title="Webinars" icon="seti:info">
		Παρακολουθήστε μας να εξηγούμε πώς να κάνετε συγκεκριμένα πράγματα στο Wink σε σύντομα, εύπεπτα βίντεο. Πηγαίνετε στα [Webinars](/el/webinars/how-tos).
	</Card>
	<Card title="Guides" icon="seti:info">
		Ή... θέλετε να αποκτήσετε τις γνώσεις που χρειάζεστε και να μάθετε από τους ειδικούς; Κατευθυνθείτε στα [Guides](/el/guides/a-quick-intro).
	</Card>
</CardGrid>
