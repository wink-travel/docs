## Σε επίπεδο πλατφόρμας

Όλες οι εφαρμογές μας είναι τοπικοποιημένες για να υποστηρίζουν όλες τις παραπάνω γλώσσες. Κάθε εφαρμογή διαθέτει επιλογέα γλώσσας στο κάτω μέρος της σελίδας από όπου ο χρήστης μπορεί να επιλέξει την επιθυμητή γλώσσα. Η μηχανή κρατήσεων υποστηρίζει επίσης τη μετάδοση της επιθυμητής γλώσσας μέσω του URL.

## Περιεχόμενο που δημιουργείται από χρήστες

Το περιεχόμενο που δημιουργείται από χρήστες τοπικοποιείται για να υποστηρίζει όλες τις παραπάνω γλώσσες. Οι χρήστες μπορούν να επιλέξουν να δημιουργήσουν την αγγλική έκδοση του περιεχομένου και να αφήσουν το Wink να αναλάβει τα υπόλοιπα. Ή, μπορούν να διαχειριστούν ένα συγκεκριμένο σύνολο γλωσσών για μεγαλύτερο έλεγχο και να αφήσουν το Wink να χειριστεί τα υπόλοιπα.

## Προειδοποιήσεις

Η προεπιλεγμένη γλώσσα είναι τα Αγγλικά. Εκτός αν ορίζεται διαφορετικά, θα χρησιμοποιούμε πρώτα την προεπιλεγμένη γλώσσα όταν ένας χρήστης χρησιμοποιεί οποιαδήποτε από τις εφαρμογές μας. Δεν ορίζουμε την ενεργή γλώσσα βάσει της τοπικής ρύθμισης του προγράμματος περιήγησης του χρήστη.
