Προτού μπορέσετε να δεχτείτε πληρωμές, το TripPay απαιτεί να επαληθεύσετε την ταυτότητά σας. Αυτό αποτελεί μέρος της διαδικασίας συμμόρφωσης KYC (Γνωρίστε τον Πελάτη σας).

## Ολοκλήρωση επαλήθευσης

Θα σας ζητηθεί να παρέχετε τα εξής:

- **Τύπος οντότητας** — Εάν είστε **Ιδιώτης** ή **Εταιρεία**.
- **Αριθμός φορολογικού μητρώου (TIN)** — Ο εθνικός φορολογικός σας αριθμός. Για ιδιώτες στις ΗΠΑ, αυτός είναι ο SSN.
- **Αριθμός αναγνώρισης οντότητας (EIN)** — Για εταιρείες, ο εθνικά αναγνωρισμένος αριθμός εταιρείας.
- **Ημερομηνία γέννησης** — Απαιτείται μόνο για ιδιώτες.
- **Διεύθυνση** — Η νόμιμη διεύθυνσή σας που είναι καταχωρημένη.
- **Προτιμώμενος τύπος καταβολής** — Πώς επιθυμείτε να λαμβάνετε τις πληρωμές.

:::note
Ορισμένες περιοχές απαιτούν επιπλέον οικονομικές πληροφορίες. Σε αυτή την περίπτωση, θα λάβετε ένα επόμενο email με τα επόμενα βήματα.
:::

## Μετά την υποβολή

Μόλις υποβληθεί, η επαλήθευσή σας εξετάζεται. Μπορείτε να συνεχίσετε να χρησιμοποιείτε την πλατφόρμα ενώ η επαλήθευση εκκρεμεί. Εάν χρειαστούν επιπλέον έγγραφα, θα ειδοποιηθείτε μέσω email.
