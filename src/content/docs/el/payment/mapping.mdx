Αν είστε ενσωματωτής, όπως η Wink, και θέλετε να χαρτογραφήσετε τους εξωτερικούς δικαιούχους σας για χρήση με το TripPay, μπορείτε να το κάνετε με έναν από τους δύο τρόπους:

- Χειροκίνητη χαρτογράφηση
- Χρήση του [Payment API](/el/developers/apis/#payment-api) (*Δείτε `Account Mappings`*)

:::tip
Εκτός αν διαχειρίζεστε μόνο λίγες από τις ίδιες ιδιοκτησίες, προτείνουμε να χρησιμοποιήσετε το API για να αυτοματοποιήσετε αυτό το βήμα.
:::

Παρακάτω, σας καθοδηγούμε στη χειροκίνητη μέθοδο χαρτογράφησης ενός εξωτερικού δικαιούχου. *Χρησιμοποιεί το API στο παρασκήνιο.*

<Steps>
1. Συνδεθείτε στο [TripPay](https://pay.wink.travel).
2. Επιλέξτε τον λογαριασμό ενσωμάτωσής σας.
3. Κάντε κλικ στο `Settings > Mappings` από τη βασική γραμμή πλοήγησης.
4. Κάντε κλικ στο κουμπί `Create mapping`. Θα δείτε τη φόρμα χαρτογράφησης.
5. **Local account** Ξεκινήστε να πληκτρολογείτε το όνομα του υπάρχοντος TripPay που θέλετε να χαρτογραφήσετε.
6. **External identifier** Επικολλήστε το εξωτερικό σας ID για αυτήν την οντότητα όπως υπάρχει στο απομακρυσμένο σύστημα.
7. Κάντε κλικ στο `Save` για να συνεχίσετε. Θα ανακατευθυνθείτε πίσω στη λίστα των χαρτογραφήσεών σας.
</Steps>

Με τις χαρτογραφήσεις σας στη θέση τους, το TripPay γνωρίζει πλέον ποιοι λογαριασμοί πρέπει να πιστωθούν ως δικαιούχοι μιας κράτησης όταν τους αναφέρετε χρησιμοποιώντας τους εξωτερικούς σας αναγνωριστικούς.

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν `Payments`, πηγαίνουν στο [Developers > API > Payments](/el/developers/apis/#payment-api).
