Εάν είστε integrator, όπως ο Wink, και θέλετε να αντιστοιχίσετε τους εξωτερικούς δικαιούχους πληρωμής σας για χρήση με το TripPay, μπορείτε να το κάνετε με έναν από τους δύο τρόπους:

* Χειροκίνητη χαρτογράφηση
* Χρησιμοποιώντας το[API πληρωμών](/developers/apis/#payment-api)(*Δείτε`Account Mappings`*)

:::συμβουλή
Εκτός κι αν ασχολείστε μόνο με λίγες από τις ίδιες ιδιότητες, σας συνιστούμε να χρησιμοποιήσετε το API για να αυτοματοποιήσετε αυτό το βήμα.
:::

Παρακάτω, σας παρουσιάζουμε τον χειροκίνητο τρόπο αντιστοίχισης ενός εξωτερικού δικαιούχου πληρωμής.*Χρησιμοποιεί το API στο κάτω μέρος.*

<Steps>
  1. Συνδεθείτε [TripPay](https://pay.wink.travel).
  2. Επιλέξτε τον λογαριασμό σας ως ενοποιητή.
  3. Κλικ `Settings > Mappings` από την κύρια γραμμή πλοήγησης.
  4. Κάντε κλικ στο `Create mapping` κουμπί. Θα δείτε τη φόρμα αντιστοίχισης.
  5. **Τοπικός λογαριασμός** Ξεκινήστε να πληκτρολογείτε το όνομα του υπάρχοντος TripPay που θέλετε να αντιστοιχίσετε.
  6. **Εξωτερικό αναγνωριστικό** Επικολλήστε το εξωτερικό σας αναγνωριστικό για αυτήν την οντότητα όπως υπάρχει στο απομακρυσμένο σύστημα.
  7. Κλικ `Save` για να συνεχίσετε. Θα ανακατευθυνθείτε πίσω στη λίστα αντιστοιχίσεων.
</Steps>


Με τις αντιστοιχίσεις σας στη θέση τους, το TripPay γνωρίζει πλέον ποιοι λογαριασμοί πρέπει να πιστωθούν ως δικαιούχοι μιας κράτησης όταν τους αναφέρετε χρησιμοποιώντας τα εξωτερικά σας αναγνωριστικά.

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Payments`, πηγαίνετε στο[Προγραμματιστές > API > Πληρωμές](/developers/apis/#payment-api).
