Ορισμένες περιοχές απαιτούν έναν εθνικό φορολογικό αριθμό πριν μπορέσετε να λάβετε πληρωμές. Το TripPay θα σας ζητήσει να ολοκληρώσετε αυτό το βήμα ως μέρος της ρύθμισης του λογαριασμού σας.

## Υποβολή του αριθμού φορολογικού μητρώου σας

Όταν σας ζητηθεί, εισάγετε τον εθνικό φορολογικό σας αριθμό:

- **Ιδιώτες στις Η.Π.Α.** — Ο Αριθμός Κοινωνικής Ασφάλισης (SSN). *Μορφή: 111-11-1111*
- **Εταιρείες στις Η.Π.Α.** — Ο Αριθμός Αναγνώρισης Εργοδότη (EIN). *Μορφή: 11-111111*
- **Άλλες περιοχές** — Το αντίστοιχο εθνικό αναγνωριστικό της χώρας σας.

Μετά την εισαγωγή του αριθμού φορολογικού μητρώου, θα σας ζητηθεί να επιβεβαιώσετε ότι οι πληροφορίες είναι σωστές πριν την υποβολή.

:::note
Αυτό είναι ένα εφάπαξ βήμα που απαιτείται για τη συμμόρφωση με το KYC (Γνωρίστε τον Πελάτη σας). Ενδέχεται να επικοινωνήσουν μαζί σας για επιπλέον έγγραφα ανάλογα με την περιοχή σας.
:::
