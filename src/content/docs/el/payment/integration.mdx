Αν είστε μια ταξιδιωτική εταιρεία και θέλετε να αναθέσετε το κομμάτι των πληρωμών και των διανομών της επιχείρησής σας στο TripPay, εδώ είναι 6 βασικά βήματα για να ξεκινήσετε:

<Steps>
1. Δημιουργήστε έναν [λογαριασμό χρήστη](/el/user/register) στο Wink.
2. Δημιουργήστε έναν λογαριασμό με το [TripPay](https://pay.wink.travel).
3. Δημιουργήστε μια [Εφαρμογή](/el/developers/applications) στο TripPay που σας επιτρέπει να επικοινωνείτε με το API μας.
4. Δημιουργήστε και αντιστοιχίστε τους δικαιούχους σας με το TripPay. *π.χ. Για τους περισσότερους, αυτό σημαίνει τη δημιουργία λογαριασμών για τις ιδιοκτησίες στην πλατφόρμα σας που θέλετε να πληρώνει το TripPay.*
5. Ενσωματώστε το [web component πληρωμής](/el/developers/web-components/#payment) στην ιστοσελίδα σας.
6. Στείλτε στο TripPay το `καλάθι αγορών` του ταξιδιώτη και ξεκινήστε τη διαδικασία πληρωμής.
</Steps>

:::tip
Για έναν πλήρη οδηγό χρήσης του TripPay για την ταξιδιωτική σας ιστοσελίδα, διαβάστε τον οδηγό [Ενσωμάτωση με το TripPay](/el/guides/integrating-with-trip-pay).
:::

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν `Πληρωμές`, πηγαίνουν στο [Developers > API > Payments](/el/developers/apis/#payment-api).
