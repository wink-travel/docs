Αν είστε ταξιδιωτική εταιρεία και θέλετε να αναθέσετε το κομμάτι των πληρωμών και των εκταμιεύσεων της επιχείρησής σας στην TripPay, ακολουθούν 6 βασικά βήματα για να ξεκινήσετε:

<Steps>
  1. Δημιουργήστε ένα [λογαριασμός χρήστη](/user/register) στο Wink.
  2. Δημιουργήστε έναν λογαριασμό με [TripPay](https://pay.wink.travel).
  3. Δημιουργήστε ένα [Εφαρμογή](/developers/applications) στο TripPay που σας επιτρέπει να επικοινωνείτε με το API μας.
  4. Δημιουργήστε και χαρτογραφήστε τους δικαιούχους σας με το TripPay. *π.χ. Για τους περισσότερους, αυτό σημαίνει τη δημιουργία λογαριασμών για τα ακίνητα στην πλατφόρμα σας που θέλετε να πληρώνει το TripPay.*
  5. Ενσωματώστε το [στοιχείο ιστού πληρωμών](/developers/web-components/#payment) στον ιστότοπό σας.
  6. Στείλτε TripPay στον ταξιδιώτη `shopping cart` και να ξεκινήσει η διαδικασία πληρωμής.
</Steps>


:::συμβουλή
Για έναν πλήρη οδηγό χρήσης του TripPay για τον ταξιδιωτικό σας ιστότοπο, διαβάστε τον οδηγό[Ενσωμάτωση με το TripPay](/guides/integrating-with-trip-pay). :::

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`Payments`, πηγαίνετε στο[Προγραμματιστές > API > Πληρωμές](/developers/apis/#payment-api).
