Για να διαχειριστείτε τους τραπεζικούς σας λογαριασμούς, κάντε κλικ στο `Account > Bank Accounts` από το πλαϊνό μενού.

## Προσθήκη τραπεζικού λογαριασμού

Κάντε κλικ στο `Add bank account` για να καταχωρήσετε νέο λογαριασμό. Η φόρμα προσαρμόζεται ανάλογα με τη χώρα που επιλέγετε:

- **Χώρα πληρωμής** — Η χώρα όπου βρίσκεται η τράπεζά σας.
- **Νόμισμα** — Το νόμισμα στο οποίο θέλετε να λαμβάνετε τις πληρωμές.
- **Όνομα κατόχου λογαριασμού** — Το όνομα που αναγράφεται στον τραπεζικό λογαριασμό.
- **Στοιχεία δρομολόγησης** — Πεδία ειδικά για τη χώρα, όπως Αριθμός δρομολόγησης (ΗΠΑ), Sort code (ΗΒ), κωδικός SWIFT/BIC ή IBAN.
- **Αριθμός λογαριασμού** — Ο αριθμός του τραπεζικού σας λογαριασμού.

## Κύριος λογαριασμός

Μπορείτε να ορίσετε έναν τραπεζικό λογαριασμό ως **Κύριο**. Ο κύριος λογαριασμός λαμβάνει όλες τις αυτόματες πληρωμές. Οι επιπλέον λογαριασμοί μπορούν να χρησιμοποιηθούν για χειροκίνητες αναλήψεις.

Για να αλλάξετε τον κύριο λογαριασμό σας, κάντε κλικ στο `Make primary` σε οποιονδήποτε λογαριασμό στη λίστα σας.

## Διαχείριση λογαριασμών

Από τη λίστα τραπεζικών λογαριασμών μπορείτε να:

- Επεξεργαστείτε τα στοιχεία του λογαριασμού.
- Διαγράψετε λογαριασμό που δεν χρειάζεστε πλέον.
- Αλλάξετε ποιος λογαριασμός είναι ο κύριος.
