Ο επεξεργαστής ανάρτησης σας επιτρέπει να δημιουργείτε απεριόριστες χειροκίνητες αναρτήσεις και είναι επίσης το μέρος όπου διαχειρίζεστε τις αναρτήσεις που έχουν δημιουργηθεί αυτόματα για εσάς.

Ο επεξεργαστής έχει 4 ενότητες:

1. Την ενότητα περιεχομένου.
2. Την ενότητα μέσων.
3. Την ενότητα συνδέσμου.
4. Την ενότητα ρυθμίσεων.

## Περιεχόμενο

Η ενότητα περιεχομένου σας επιτρέπει να τροποποιήσετε το κείμενο πριν αυτό δημοσιευτεί.

## Μέσα

Η ενότητα μέσων σας επιτρέπει να τροποποιήσετε τις εικόνες και τα βίντεο που συνοδεύουν το κείμενο.

## Σύνδεσμος

Η ενότητα συνδέσμου σας επιτρέπει να συνδέσετε κάπου όπου θέλετε να πάει ο χρήστης μετά την ανάγνωση της ανάρτησής σας. *Π.χ. Αυτό μπορεί να είναι ένας σύνδεσμος σε μια ειδική προσφορά ή σε μια σελίδα για περισσότερες πληροφορίες.*

## Ρυθμίσεις

Η ενότητα ρυθμίσεων σας επιτρέπει να τροποποιήσετε / παρακάμψετε όλες τις προεπιλεγμένες ρυθμίσεις σας. Μπορεί να είναι χρήσιμη όταν θέλετε η ανάρτηση να δημοσιευτεί σε διαφορετική ώρα. Τις περισσότερες φορές, αυτή η ενότητα δεν θα χρησιμοποιηθεί.

## Έγκριση αναρτήσεων

Οι αναρτήσεις εγκρίνονται στο πλέγμα αναρτήσεων όπου μπορείτε να επιλέξετε μία ή περισσότερες αναρτήσεις και να επιλέξετε να εγκρίνετε ή να απορρίψετε μια ανάρτηση.

:::note
Θα υπάρχει μία εγγραφή για κάθε γλώσσα και κάθε κοινωνικό δίκτυο.

Π.χ. Αν έχετε 3 κοινωνικά δίκτυα και 3 γλώσσες επιλεγμένες, θα δείτε 9 εγγραφές στο ημερολόγιο.
:::
