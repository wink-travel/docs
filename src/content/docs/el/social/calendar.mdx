Η δημοσίευση λειτουργεί ως εξής:

<Steps>
1. Το περιεχόμενο δημιουργείται από τις προτιμήσεις σας μία εβδομάδα πριν από τον μήνα δημοσίευσης.
2. Λαμβάνετε ένα e-mail που σας ενημερώνει να εγκρίνετε το νέο περιεχόμενο [ή να σας ειδοποιεί ότι το περιεχόμενο έχει δημιουργηθεί για τον επόμενο μήνα και ότι δεν χρειάζεται να κάνετε κάτι άλλο επειδή έχετε επιλέξει την αυτόματη έγκριση του περιεχομένου].
3. Συνδέεστε στο [social.wink.travel](https://social.wink.travel) για να εγκρίνετε τις δημοσιεύσεις.
4. Οι δημοσιεύσεις προγραμματίζονται για δημοσίευση. (Μπορείτε ακόμα να σταματήσετε τη διαδικασία δημοσίευσης αν το κάνετε πριν δημοσιευτεί. Διαφορετικά, θα πρέπει να πάτε στο κοινωνικό δίκτυο και να διαγράψετε ή να τροποποιήσετε τη δημοσίευση χειροκίνητα εκεί.)
5. Οι δημοσιεύσεις αναρτώνται την πιο κατάλληλη στιγμή για τη ζώνη ώρας που έχετε επιλέξει.
</Steps>

Μόλις το περιεχόμενο εγκριθεί για δημοσίευση, εμφανίζεται στο ημερολόγιό σας.

:::note
Θα υπάρχει μία καταχώρηση για κάθε γλώσσα και κάθε κοινωνικό δίκτυο.

Π.χ. Αν έχετε επιλέξει 3 κοινωνικά δίκτυα και 3 γλώσσες, θα δείτε 9 καταχωρήσεις στο ημερολόγιο.
:::
