Για να διαχειριστείτε τις ετικέτες WinkLinks, μεταβείτε στο `WinkLinks` στη βασική γραμμή πλοήγησης και κάντε κλικ στην καρτέλα `Tags`.

Καθώς το περιεχόμενό σας αυξάνεται, θα θελήσετε να αρχίσετε να οργανώνετε μέρος του. Δημιουργήσαμε τις `tags` για αυτόν τον σκοπό. Μπορείτε να ομαδοποιήσετε περιεχόμενο χρησιμοποιώντας όσες ετικέτες θέλετε.

## Δημιουργία ετικέτας

<Steps>
1. Δημιουργήστε μια νέα ετικέτα κάνοντας κλικ στο κουμπί `Add a new tag`.
2. Εισάγετε το όνομα της νέας σας ετικέτας. *π.χ. Top coffee shops in Berlin*
3. Κάντε κλικ στον σύνδεσμο `Save`.
</Steps>

## Ενημέρωση ετικέτας

<Steps>
1. Ξεκινήστε κάνοντας κλικ στον σύνδεσμο `Edit` δίπλα στην ετικέτα που θέλετε να τροποποιήσετε.
2. Ενημερώστε το όνομα της ετικέτας σας. *π.χ. Best coffee shops in Berlin*
3. Κάντε κλικ στον σύνδεσμο `Save`.
</Steps>

## Αφαίρεση ετικέτας

:::note
Όταν αφαιρείτε μια ετικέτα, θα αφαιρεθεί επίσης από όλο το σχετικό περιεχόμενο.
:::

<Steps>
1. Κάντε κλικ στον σύνδεσμο `Remove` δίπλα στην ετικέτα που θέλετε να αφαιρέσετε.
</Steps>

## API

Οι προγραμματιστές που θέλουν να διαχειριστούν τα `WinkLinks` μπορούν να μεταβούν στο [Developers > API > WinkLinks](/el/developers/apis/#winklinks-api).
