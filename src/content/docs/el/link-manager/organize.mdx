Για να διαχειριστείτε τις ετικέτες WinkLinks, μεταβείτε στη διεύθυνση`WinkLinks`στην κύρια γραμμή πλοήγησης και κάντε κλικ στο`Tags`καρτέλα.

Καθώς το περιεχόμενό σας μεγαλώνει, θα θελήσετε να αρχίσετε να το οργανώνετε. Δημιουργήσαμε`tags`για αυτόν τον σκοπό. Μπορείτε να ομαδοποιήσετε περιεχόμενο χρησιμοποιώντας όσες ετικέτες θέλετε.

## Δημιουργία ετικέτας

<Steps>
  1. Δημιουργήστε μια νέα ετικέτα κάνοντας κλικ στο `Add a new tag` κουμπί.
  2. Εισαγάγετε το όνομα της νέας ετικέτας σας. *π.χ. Κορυφαίες καφετέριες στο Βερολίνο*
  3. Κάντε κλικ στο `Save` σύνδεσμος.
</Steps>


## Ενημέρωση ετικέτας

<Steps>
  1. Ξεκινήστε κάνοντας κλικ στο `Edit` σύνδεσμο δίπλα στην ετικέτα που θέλετε να τροποποιήσετε.
  2. Ενημερώστε το όνομα της ετικέτας σας. *π.χ. Τα καλύτερα καφέ στο Βερολίνο*
  3. Κάντε κλικ στο `Save` σύνδεσμος.
</Steps>


## Αφαίρεση ετικέτας

:::σημείωση
Όταν αφαιρείτε μια ετικέτα, την αφαιρείτε και από όλο το σχετικό περιεχόμενο.
:::

<Steps>
  1. Κάντε κλικ στο `Remove` σύνδεσμο δίπλα στην ετικέτα που θέλετε να καταργήσετε.
</Steps>


## API

Προγραμματιστές που θέλουν να διαχειρίζονται`WinkLinks`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > WinkLinks](/developers/apis/#winklinks-api).
