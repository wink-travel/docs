Για να διαχειριστείτε περιεχόμενο στο WinkLinks, βεβαιωθείτε ότι έχετε ήδη επιλέξει τον λογαριασμό συνεργάτη με τον οποίο θα συνεργαστείτε στο Wink Studio, μεταβείτε στη διεύθυνση`WinkLinks`στην κύρια γραμμή πλοήγησης.

Το περιεχόμενό σας είναι διαθέσιμο υπό την`Entries`καρτέλα.

Υποστηρίζουμε τρεις κύριους τύπους περιεχομένου:

* Δημόσια διαθέσιμες διευθύνσεις URL που συνδέονται με ιστοσελίδες.
* Αρχεία που μεταφορτώθηκαν μέσω του WinkLinks.
* Κειμενικό περιεχόμενο που δημιουργήθηκε μέσω του`Quotes`χαρακτηριστικό.

## Προσθήκη περιεχομένου

Δείτε πώς προσθέτετε περιεχόμενο στο WinkLinks. Σύμφωνα με το`Entries`καρτέλα, ξεκινήστε κάνοντας κλικ στην`Add entry`κουμπί.

### Κανονικός σύνδεσμος

:::συμβουλή
Ο σύνδεσμος πρέπει να οδηγεί σε μια ιστοσελίδα. Για παράδειγμα, δεν μπορείτε να συνδέσετε απευθείας ένα PDF. Εάν θέλετε να μοιραστείτε ένα PDF, πρέπει να χρησιμοποιήσετε το`Upload file`χαρακτηριστικό.
:::

Ακολουθούν τα βήματα για την προσθήκη ενός κανονικού συνδέσμου:

<Steps>
  1. Εισαγάγετε τον σύνδεσμο που θέλετε να κοινοποιήσετε. *π.χ. https://trvl.as/s/3xWCH*
  2. Μόλις εισαγάγετε τον σύνδεσμο, ο Wink αναζητά τις λεπτομέρειες αυτού του ιστότοπου και συμπληρώνει τη φόρμα που βλέπετε στη συνέχεια.
  3. Μπορείτε να επιλέξετε να τροποποιήσετε ορισμένα από τα δεδομένα, όπως το όνομα, τον τίτλο και την περιγραφή του ιστότοπου.
  4. Μπορείτε επίσης να επιλέξετε να ανεβάσετε μια προσαρμοσμένη εικόνα.
  5. Κάντε κλικ στο `Save` κουμπί όταν είστε έτοιμοι να συνεχίσετε.
</Steps>


### Μεταφόρτωση αρχείου

:::προσοχή\[Απαιτήσεις]

* Οι εικόνες δεν μπορούν να υπερβαίνουν τα 10Mb σε μέγεθος.
* Τα βίντεο δεν μπορούν να υπερβαίνουν τα 50Mb σε μέγεθος.
* Υποστηρίζουμε όλες τις αναγνωρισμένες μορφές εικόνας και βίντεο, συμπεριλαμβανομένων των PDF.
  ::::

:::σημείωση\[Προτάσεις]

* 2560π x 1440υ (αναλογία διαστάσεων 16x9) για την εικόνα που παρουσιάζετε.
* 2560π x 1920υ (λόγος διαστάσεων 4:3) για όλες τις άλλες εικόνες.
* Το μέγεθος της εικόνας δεν πρέπει να είναι μεγαλύτερο από 5Mb. Παρακαλούμε συμπιέστε τις εικόνες σας πριν τις ανεβάσετε στο Wink.
  :::

Ακολουθούν τα βήματα για την προσθήκη ενός αρχείου:

<Steps>
  1. Μπορούν να προστεθούν μέσα:
     * Σύροντας εικόνες από τον υπολογιστή σας στο παράθυρο ή κάνοντας κλικ `Browse`.
     * Επικολλώντας μια εξωτερική διεύθυνση URL που παραπέμπει σε μια εικόνα.
     * Χρησιμοποιώντας την κάμερα του φορητού υπολογιστή σας.
     * Από τον λογαριασμό σας στο Google Drive.
     * Από τον λογαριασμό σας στο Dropbox.
     * Από το Shutterstock
     * Από το gettyimages
     * Από το iStock
     * Από το Unsplash
  2. Μόλις ολοκληρωθεί η μεταφόρτωση του αρχείου, μπορείτε να προσθέσετε περισσότερα μεταδεδομένα για να περιγράψετε το αρχείο σας, όπως τίτλο και περιγραφή.
  3. Κάντε κλικ στο `Save` κουμπί όταν είστε έτοιμοι να συνεχίσετε.
</Steps>


### Γράψτε μια προσφορά

:::συμβουλή
Ένα απόσπασμα είναι ένας εξαιρετικός τρόπος για να προσθέσετε έναν οριοθέτη για να χωρίσετε μια μεγάλη λίστα \[...και σας κάνει να ακούγεστε έξυπνοι και έμπειροι επίσης 😉].
:::

Ακολουθούν τα βήματα για τη σύνταξη μιας προσφοράς:

<Steps>
  1. Πληκτρολογήστε την προσφορά *π.χ. Τα τριαντάφυλλα είναι κόκκινα. Οι βιολέτες είναι μπλε...*.
  2. Εισαγάγετε το όνομα του αρχικού συγγραφέα. *π.χ. Έντμουντ Σπένσερ*
  3. Εισαγάγετε το όνομα του ιστότοπου από τον οποίο λάβατε την προσφορά. *π.χ. Ημερήσιες Προσφορές*
  4. Κάντε κλικ στο `Save` κουμπί όταν είστε έτοιμοι να συνεχίσετε.
</Steps>


## Ενημέρωση περιεχομένου

Όλο το περιεχόμενο μπορεί εύκολα να τροποποιηθεί κάνοντας κλικ στο`Actions`σύνδεσμος και επιλογή`Update`Μπορείτε να καταργήσετε περιεχόμενο με παρόμοιο τρόπο κάνοντας κλικ στο`Remove`κουμπί.

## Αφαίρεση περιεχομένου

Όλο το περιεχόμενο μπορεί εύκολα να τροποποιηθεί επιλέγοντας`Update`Μπορείτε να καταργήσετε περιεχόμενο με παρόμοιο τρόπο κάνοντας πρώτα κλικ στο`Actions`σύνδεσμο και, στη συνέχεια, κάνοντας κλικ στο`Remove`κουμπί.

## Ταξινόμηση περιεχομένου

Μπορείτε να ταξινομήσετε το περιεχόμενό σας κάνοντας κλικ στο εικονίδιο που βρίσκεται στην κάτω αριστερή γωνία κάθε καταχώρησης και σύροντάς το στη θέση που θέλετε να βρίσκεται.

## Ομαδοποίηση περιεχομένου

:::συμβουλή
Δημιουργήστε τις ετικέτες σας πριν δημιουργήσετε τις καταχωρίσεις σας. Διαφορετικά, χάνετε την πρόοδό σας και θα πρέπει να ξεκινήσετε από την αρχή.
:::

Όταν επεξεργάζεστε μια καταχώρηση, έχετε την επιλογή να προσθέσετε μία ή περισσότερες ετικέτες για σκοπούς ομαδοποίησης. Μεταβείτε στο κάτω μέρος της φόρμας για να βρείτε την`Tags`πεδίο φόρμας.

Οι χρήστες, που βλέπουν τη σελίδα WinkLinks, μπορούν να επιλέξουν πώς θέλουν να καταναλώσουν το περιεχόμενό σας (είτε σε λίστα είτε σε πλέγμα) και μπορούν να επιλέξουν μία ή περισσότερες ετικέτες για να φιλτράρουν περαιτέρω το περιεχόμενό σας.

## API

Προγραμματιστές που θέλουν να διαχειρίζονται`WinkLinks`μπορεί να κατευθυνθεί προς[Προγραμματιστές > API > WinkLinks](/developers/apis/#winklinks-api).
