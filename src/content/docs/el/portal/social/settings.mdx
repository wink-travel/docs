# Ρυθμίσεις

Όταν ξεκινάτε με το Social, θα σας ρωτήσουμε πώς θέλετε να δημιουργούμε το περιεχόμενό σας. Αυτές οι επιλογές είναι διαθέσιμες ανά πάσα στιγμή ώστε να μπορείτε εύκολα να τις αλλάξετε αργότερα.

Πλοηγηθείτε στις `Προτιμήσεις` από το πλαϊνό μενού για να διαχειριστείτε τις ρυθμίσεις σας.

Η ενότητα προτιμήσεων περιέχει τις βασικές ρυθμίσεις δημιουργίας περιεχομένου:

1. **Προτιμώμενη προσαρμογή** Επιλέξτε ποια προσαρμογή θα χρησιμοποιείται κατά τη δημιουργία περιεχομένου. Οι προσαρμογές ελέγχουν τα χρώματα θέματος, το προεπιλεγμένο δρομολόγιο, το νόμισμα, τη γλώσσα και άλλα. Δημιουργείτε προσαρμογές στον λογαριασμό σας στο [my.wink.travel](https://my.wink.travel).
2. **Αυτόματη έγκριση** Όταν είναι ενεργοποιημένη, οι δημιουργημένες αναρτήσεις δημοσιεύονται αυτόματα χωρίς να απαιτείται η έγκρισή σας. Ενεργοποιήστε το όταν είστε βέβαιοι για την ποιότητα του περιεχομένου που παράγουμε για εσάς.
3. **Όνομα εγκριτή** Το όνομα του ατόμου ή της ομάδας που θα λαμβάνει τα αιτήματα έγκρισης μέσω email.
4. **Email εγκριτή** Η διεύθυνση email όπου θα αποστέλλονται τα αιτήματα έγκρισης για τις επερχόμενες αναρτήσεις.

## Δημιουργοί περιεχομένου

Το Social διαθέτει πολλούς δημιουργούς περιεχομένου για να επιλέξετε. Κάθε δημιουργός εστιάζει σε έναν συγκεκριμένο τρόπο δημιουργίας μοναδικού περιεχομένου για το κατάλυμά σας χρησιμοποιώντας AI. Κάθε δημιουργός περιεχομένου έχει μια ιδιότητα **Βάρος** — μεγαλύτερο βάρος αυξάνει την πιθανότητα να επιλεγεί κατά τη δημιουργία αναρτήσεων.

Ακολουθούν όλοι οι διαθέσιμοι δημιουργοί περιεχομένου:

| Δημιουργός | Περιγραφή |
| -- | -- |
| **Προορισμός** | Χρησιμοποιεί την τοποθεσία σας για να δημιουργήσει χρήσιμο κείμενο για την περιοχή |
| **Τύπος δωματίου** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με έναν από τους τύπους δωματίων σας με κείμενο που παράγεται από AI και εικόνες του τύπου δωματίου σας |
| **Δραστηριότητα** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με μία από τις δραστηριότητές σας |
| **Πρόσθετο** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με ένα από τα πρόσθετά σας |
| **Αξιοθέατο** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με ένα από τα αξιοθέατά σας |
| **Τοποθεσία** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με μία από τις τοποθεσίες σας |
| **Αίθουσα συνεδριάσεων** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με μία από τις αίθουσες συνεδριάσεών σας |
| **Εστιατόριο** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με ένα από τα εστιατόριά σας |
| **Σπα** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με ένα από τα σπα σας |
| **Κατάλυμα** | Δημιουργεί μια διαθέσιμη προς κράτηση ανάρτηση με το καλύτερης τιμής δωμάτιό σας με κείμενο που παράγεται από AI και εικόνες του καταλύματός σας |
| **Συναίσθημα χρήστη** | Συγκεντρώνει κριτικές επισκεπτών από όλο το διαδίκτυο σε μια ανάρτηση βασισμένη στα συναισθήματα που αναδεικνύει τι αγαπούν οι επισκέπτες στο κατάλυμά σας |
| **Αστείο** | Δημιουργεί ελαφριές, διασκεδαστικές αναρτήσεις για να κρατά το feed σας ενδιαφέρον |

:::tip[Επαγγελματική συμβουλή]
Βεβαιωθείτε ότι έχετε τον τύπο αποθέματος που απαιτείται για τους επιλεγμένους δημιουργούς περιεχομένου. Για παράδειγμα, αν επιλέξετε "Εστιατόριο", πρέπει να έχετε τουλάχιστον ένα εστιατόριο καταχωρημένο στο κατάλυμά σας.
:::

Ο τόνος φωνής καθορίζει το στυλ γλώσσας που χρησιμοποιεί το AI κατά τη δημιουργία περιεχομένου. Επιλέξτε τους τόνους που ταιριάζουν στην προσωπικότητα της μάρκας σας. Όπως και με τους δημιουργούς περιεχομένου, κάθε τόνος έχει ένα **Βάρος** για να ελέγχει πόσο συχνά χρησιμοποιείται.

Ακολουθούν οι διαθέσιμοι τόνοι:

| Τόνος | Στυλ |
| -- | -- |
| **Ουδέτερος** | Ήρεμος, αυθεντικός, πραγματολογικός. Χωρίς υπερβολές, χωρίς αστεία — απλώς αξιόπιστη, καλά δομημένη γνώση |
| **Φιλικός** | Ζεστός, προσιτός, με ενέργεια που χαμογελά μέσα από το κείμενο, με casual συντομεύσεις και περιστασιακά emojis |
| **Πολυτελείας** | Κομψός, εκλεπτυσμένος. Λέξεις όπως "εκλεκτός" και "κατά παραγγελία" που κάνουν τον αναγνώστη να νιώθει αποκλειστικός |
| **Γαστρονομικός** | Αισθησιακές, λαχταριστές περιγραφές. Γλώσσα που προκαλεί όρεξη |
| **Περιπετειώδης** | Τολμηρός, ατρόμητος, με ενέργεια εξερευνητή. Ζωντανές αισθητηριακές περιγραφές που κάνουν τα πάντα να ακούγονται επικά |
| **Οικογενειακός** | Υγιής, φιλικός προς τα παιδιά ενθουσιασμός. Ζεστή γονεϊκή ενέργεια για ασφαλείς και διασκεδαστικές εμπειρίες |
| **Ρομαντικός** | Ποιητικός, νοσταλγικός, που φτερουγίζει την καρδιά. Ηλιοβασιλέματα, απαλός λόγος και αγάπη |
| **Χιουμοριστικός** | Σαρκαστικός, γρήγορα αστεία, έξυπνο λογοπαίγνιο. Διαβάζεται σαν stand-up comedy |
| **Οικολογικός** | Ειλικρινής και δίκαιος. Γιορτάζει τις πράσινες εναλλακτικές και τις βιώσιμες επιλογές |
| **Τοπικός γνώστης** | Υπερ-συγκεκριμένες τοπικές αναφορές, προειδοποιεί για τουριστικές παγίδες, χρησιμοποιεί τοπική αργκό |
| **Συμπεριληπτικός** | Φιλόξενος, εορταστικός και επιβεβαιωτικός. Κάνει όλους να νιώθουν ορατοί και αξιόλογοι |
| **Μυστικιστικός** | Αιθέριος, πνευματικός, κοσμικές αλήθειες. Απαλές προσκλήσεις για εσωτερική αναζήτηση |
| **Φουτουριστικός** | Λείος, προοδευτικός. Λατρεύει όρους όπως "παραδειγματική αλλαγή" και αισιόδοξες μελλοντικές οράσεις |
| **Οικονομικός** | Επινοητικός, πρακτικός, περήφανα φειδωλός. Γιορτάζει τις προσιτές ευκαιρίες και τις δωρεάν εμπειρίες |

## Τύποι αναρτήσεων

Οι τύποι αναρτήσεων καθορίζουν τον στόχο και τη δομή του παραγόμενου περιεχομένου. Κάθε τύπος ανάρτησης έχει ένα **Βάρος** για να ελέγχει πόσο συχνά χρησιμοποιείται.

| Τύπος ανάρτησης | Περιγραφή |
| -- | -- |
| **Συναλλακτικός** | Σαφής, συνοπτικός, εστιασμένος στη μετατροπή. Οδηγεί σε κρατήσεις και άμεσες ενέργειες με γεγονότα, τιμές και επόμενα βήματα |
| **Ενημερωτικός** | Περιεχόμενο για την οικοδόμηση της μάρκας που έχει σχεδιαστεί να παρουσιάσει το κατάλυμά σας και να αυξήσει την ορατότητα σε πιθανούς επισκέπτες |
| **Συναισθηματικός** | Περιεχόμενο που προκαλεί συναισθηματική ανταπόκριση, δημιουργεί θετικές συσχετίσεις και χτίζει συναισθηματική σύνδεση με το κοινό σας |

  alt="Social time zones"

Ίσως θέλετε να εστιάσετε τις προσπάθειες μάρκετινγκ σε μια συγκεκριμένη γλώσσα και περιοχή. Πείτε μας τις γλώσσες για τις οποίες θέλετε να δημιουργήσετε περιεχόμενο και τη ζώνη ώρας που σας ενδιαφέρει και θα αναλάβουμε τα υπόλοιπα. Οι αναρτήσεις δημοσιεύονται την προγραμματισμένη ώρα στη ζώνη ώρας που έχετε επιλέξει.

*Τα περισσότερα καταλύματα χρησιμοποιούν μία γλώσσα και προεπιλέγουν τη γλώσσα της περιοχής τους.*

Συνδέοντας το Google Business Profile σας, μπορούμε να αντλήσουμε πλουσιότερες λεπτομέρειες για το κατάλυμά σας για πιο αποτελεσματικό περιεχόμενο. Το κατάλυμά σας χρειάζεται ένα επαληθευμένο Google Business Profile ώστε να μπορούμε να δημιουργήσουμε περιεχόμενο από την καταχώρισή σας. Ορισμένοι από τους δημιουργούς περιεχομένου μας **απαιτούν** να έχετε λογαριασμό Google Business.
