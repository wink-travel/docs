Το Studio παρέχει διάφορα ενσωματώσιμα widgets που μπορείτε να προσθέσετε στην ιστοσελίδα σας για να αυξήσετε τις κρατήσεις.

Για να αποκτήσετε πρόσβαση στα embed widgets, μεταβείτε στο `Tools > Miscellaneous` από το πλαϊνό μενού.

## Διαθέσιμα widgets

| Widget | Περιγραφή |
| -- | -- |
| **Account button** | Ένα κουμπί με το λογότυπό σας που συνδέει στη σελίδα συνεργάτη σας |
| **Itinerary button** | Ένα κουμπί που ανοίγει τον επιλογέα δρομολογίου |
| **Itinerary form** | Μια ενσωματωμένη φόρμα επιλογής ημερομηνιών και επισκεπτών |
| **Search button** | Ένα κουμπί που ανοίγει την αναζήτηση καταλυμάτων |
| **Lookup button** | Ένα κουμπί για αναζήτηση συγκεκριμένου αποθέματος |

## Πώς να ενσωματώσετε

Κάθε widget παρέχει ένα απόσπασμα κώδικα που μπορείτε να αντιγράψετε και να επικολλήσετε στο HTML της ιστοσελίδας σας. Το απόσπασμα περιλαμβάνει το **Client ID** και το **Customization ID** σας ώστε το widget να συνδέεται αυτόματα με τον λογαριασμό συνεργάτη σας.

:::note
Αν χρησιμοποιείτε WordPress, παρέχουμε επίσης ένα πρόσθετο WordPress για πιο εύκολη ενσωμάτωση.
:::
