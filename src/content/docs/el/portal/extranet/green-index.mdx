Ο πράσινος δείκτης σας μετράται με βάση τον αντίκτυπο που έχει το ακίνητό σας στο τοπικό περιβάλλον και τις αποφάσεις αγορών σας.

Για να διαχειριστείτε τη βαθμολογία του Πράσινου Δείκτη, κάντε κλικ στο `Property > Green Index` στην κύρια καρτέλα πλοήγησης.

Το ερωτηματολόγιο του Πράσινου Δείκτη παρουσιάζεται ως ένας οδηγός βήμα προς βήμα που σας καθοδηγεί σε κάθε κατηγορία ξεχωριστά. Ολοκληρώστε κάθε βήμα και κάντε κλικ στο `Next` για να προχωρήσετε στην επόμενη ενότητα.
