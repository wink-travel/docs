Η Αναγνώριση σας επιτρέπει να αναδείξετε τα βραβεία, τις πιστοποιήσεις και τα διακριτικά του κλάδου της ιδιοκτησίας σας για να χτίσετε εμπιστοσύνη με τους πιθανούς επισκέπτες.

Για να διαχειριστείτε την αναγνώριση, κάντε κλικ στο `Property > Recognition` από την κύρια γραμμή πλοήγησης.

## Προσθήκη αναγνώρισης

Κάντε κλικ στο `Create` για να προσθέσετε μια νέα εγγραφή αναγνώρισης. Κάθε εγγραφή περιλαμβάνει:

- **Κατηγορία** — Ο τύπος της αναγνώρισης (π.χ. βραβείο, πιστοποίηση).
- **Τύπος** — Ο συγκεκριμένος τύπος αναγνώρισης.
- **Πάροχος** — Ο οργανισμός που την εκδίδει (π.χ. Michelin, TripAdvisor, AAA).
- **Βαθμολογία** — Αριθμητική τιμή βαθμολογίας, αν ισχύει.
- **Σύμβολο βαθμολογίας** — Το εικονίδιο ή σύμβολο που χρησιμοποιείται για την εμφάνιση της βαθμολογίας.
- **Ημερομηνία** — Πότε αποκτήθηκε η αναγνώριση.
- **Επίσημος διορισμός** — Αν πρόκειται για επίσημο χαρακτηρισμό.

## Διαχείριση εγγραφών

Από τη λίστα αναγνώρισης, μπορείτε να επεξεργαστείτε ή να αφαιρέσετε υπάρχουσες εγγραφές. Οι εγγραφές αναγνώρισης εμφανίζονται στο προφίλ της ιδιοκτησίας σας στη μηχανή κρατήσεων, βοηθώντας τους ταξιδιώτες να δουν τα διακρίματά σας με μια ματιά.
