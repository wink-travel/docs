Αυτή η σελίδα απαριθμεί κάθε γεγονός webhook στο οποίο μπορείτε να εγγραφείτε στο Wink. Για οδηγίες ρύθμισης, δείτε τον [Οδηγό Ενσωμάτωσης Webhook](/el/guides/webhook-integration) ή τη [σελίδα διαχείρισης Webhooks](/el/my-account/webhooks).

## Πώς να εγγραφείτε

1. Μεταβείτε στο **Applications > Webhooks** στο portal σας.
2. Δημιουργήστε ένα webhook και επιλέξτε τα γεγονότα που θέλετε να λαμβάνετε.
3. Το Wink στέλνει ένα HTTP POST στο endpoint σας κάθε φορά που συμβαίνει το γεγονός.

Για πλήρη βήματα ρύθμισης και βέλτιστες πρακτικές, δείτε τον [Οδηγό Ενσωμάτωσης Webhook](/el/guides/webhook-integration).

## Γεγονότα λογαριασμού

Γεγονότα σχετικά με τον κύκλο ζωής και τις ρυθμίσεις του λογαριασμού TripPay σας.

| Γεγονός | Κλειδί | Περιγραφή |
|---|---|---|
| Δημιουργία λογαριασμού | `account.create` | Έχει δημιουργηθεί νέος λογαριασμός. |
| Λογαριασμός υπό εξέταση | `account.under_review` | Ο λογαριασμός βρίσκεται σε αναμονή για έλεγχο. |
| Ενημέρωση λογαριασμού | `account.updated` | Τα στοιχεία του λογαριασμού έχουν αλλάξει. |
| Ενημέρωση κατάστασης λογαριασμού | `account.status.updated` | Η κατάσταση του λογαριασμού έχει αλλάξει (π.χ. εγκεκριμένος, ανασταλμένος). |
| Ενημέρωση διεύθυνσης λογαριασμού | `account.address.updated` | Η διεύθυνση του λογαριασμού έχει τροποποιηθεί. |
| Ενημέρωση λεπτομερειών λογαριασμού | `account.details.updated` | Τα πεδία λεπτομερειών του λογαριασμού έχουν αλλάξει. |
| Ενημέρωση πληροφοριών λογαριασμού | `account.information.update` | Οι γενικές πληροφορίες του λογαριασμού έχουν ενημερωθεί. |
| Ενημέρωση διαχειριστών λογαριασμού | `account.managers.update` | Η λίστα διαχειριστών λογαριασμού έχει αλλάξει. |
| Ενημέρωση φωτογραφίας προφίλ | `account.profile_picture.updated` | Η φωτογραφία προφίλ του λογαριασμού έχει αλλάξει. |
| Ενημέρωση πράκτορα | `account.agent.updated` | Οι πληροφορίες του πράκτορα στο λογαριασμό έχουν τροποποιηθεί. |
| Ενημέρωση ονόματος URL | `account.url-name.update` | Το vanity URL του λογαριασμού έχει αλλάξει. |
| Ενημέρωση μοναδικού αναγνωριστικού | `account.unique-id.update` | Το μοναδικό αναγνωριστικό του λογαριασμού έχει τροποποιηθεί. |

### Εργασίες λογαριασμού

| Γεγονός | Κλειδί | Περιγραφή |
|---|---|---|
| Δημιουργία εργασίας | `account.task.create` | Προστέθηκε νέα εργασία στον λογαριασμό. |
| Ενημέρωση εργασίας | `account.task.update` | Μια εργασία λογαριασμού έχει τροποποιηθεί. |
| Αφαίρεση εργασίας | `account.task.remove` | Μια εργασία λογαριασμού έχει αφαιρεθεί. |
| Ολοκλήρωση εργασίας | `account.task.complete` | Μια εργασία λογαριασμού έχει ολοκληρωθεί. |
| Ολοκλήρωση εργασίας επαλήθευσης φορολογικού αριθμού | `account.task.tax_id.complete` | Η εργασία επαλήθευσης φορολογικού αριθμού έχει ολοκληρωθεί. |

### Αποκτώντας λογαριασμό

| Γεγονός | Κλειδί | Περιγραφή |
|---|---|---|
| Δημιουργία αποκτών | `account.acquirer.created` | Προστέθηκε νέος αποκτών στον λογαριασμό. |
| Ενημέρωση αποκτών | `account.acquirer.updated` | Ένας αποκτών λογαριασμού έχει τροποποιηθεί. |
| Αφαίρεση αποκτών | `account.acquirer.removed` | Ένας αποκτών έχει αφαιρεθεί από τον λογαριασμό. |

## Γεγονότα συμβολαίου κράτησης

Γεγονότα που παρακολουθούν τον κύκλο ζωής ενός συμβολαίου κράτησης μέσω TripPay.

| Γεγονός | Κλειδί | Περιγραφή |
|---|---|---|
| Αίτηση αρχικοποίησης | `booking.contract.initialize.request` | Έχει ζητηθεί η αρχικοποίηση συμβολαίου κράτησης. |
| Ολοκλήρωση αρχικοποίησης | `booking.contract.initialize.complete` | Η αρχικοποίηση συμβολαίου κράτησης έχει ολοκληρωθεί. |
| Δημιουργία συμβολαίου | `booking.contract.created` | Έχει δημιουργηθεί νέο συμβόλαιο κράτησης. |
| Επεξεργασία συμβολαίου | `booking.contract.processing` | Η πληρωμή του συμβολαίου κράτησης επεξεργάζεται. |
| Αποτυχία συμβολαίου | `booking.contract.failed` | Η πληρωμή του συμβολαίου κράτησης απέτυχε. |
| Ακύρωση συμβολαίου | `booking.contract.cancelled` | Το συμβόλαιο κράτησης έχει ακυρωθεί. |
| Μερική επιστροφή χρημάτων | `booking.contract.refund.partial` | Έχει εκδοθεί μερική επιστροφή χρημάτων για το συμβόλαιο. |
| Πλήρης επιστροφή χρημάτων | `booking.contract.refund.full` | Έχει εκδοθεί πλήρης επιστροφή χρημάτων για το συμβόλαιο. |
| Αίτηση επιστροφής χρημάτων | `booking.contract.refund.request` | Έχει υποβληθεί αίτηση επιστροφής χρημάτων. |
| Σφάλμα επιστροφής χρημάτων | `booking.contract.refund.error` | Παρουσιάστηκε σφάλμα κατά την επεξεργασία επιστροφής χρημάτων. |
| Δημιουργία πληρωμής | `booking.contract.payout.create` | Έχει δημιουργηθεί πληρωμή για το συμβόλαιο. |
| Ολοκλήρωση πληρωμής | `booking.contract.payout.complete` | Η πληρωμή έχει ολοκληρωθεί με επιτυχία. |

## Γεγονότα διαχειριζόμενης οντότητας

Γεγονότα για διαχειριζόμενες οντότητες (ιδιοκτησίες, θυγατρικές, εταιρείες) και τη διαχείρισή τους.

### Συνδρομή

| Γεγονός | Κλειδί | Περιγραφή |
|---|---|---|
| Παροχή συνδρομής | `managing_entity.subscription.provisioned` | Έχει παρασχεθεί νέα συνδρομή. |
| Αλλαγή συνδρομής | `managing_entity.subscription.changed` | Το πρόγραμμα συνδρομής έχει αλλάξει. |
| Αλλαγή τιμολογίου συνδρομής | `managing_entity.subscription.invoice.changed` | Το τιμολόγιο συνδρομής έχει ενημερωθεί. |

### Κατάσταση και διαχείριση

| Γεγονός | Κλειδί | Περιγραφή |
|---|---|---|
| Ενημέρωση κατάστασης | `managing_entity.status.updated` | Η κατάσταση της οντότητας έχει αλλάξει. |
| Διαγραφή οντότητας | `managing_entity.deleted` | Μια διαχειριζόμενη οντότητα έχει διαγραφεί. |
| Ενημέρωση πράκτορα | `managing_entity.agent.updated` | Ο πράκτορας της οντότητας έχει τροποποιηθεί. |
| Ειδοποίηση πράκτορα μέσω email | `managing_entity.agent.email.notification` | Έχει σταλεί ειδοποίηση μέσω email στον πράκτορα. |
| Ειδοποίηση διαχειριστή μέσω email | `managing_entity.manager.email.notification` | Έχει σταλεί ειδοποίηση μέσω email σε διαχειριστή. |

### Κύκλος ζωής διαχειριστή

| Γεγονός | Κλειδί | Περιγραφή |
|---|---|---|
| Προσθήκη διαχειριστή | `managing_entity.manager.added` | Έχει προστεθεί νέος διαχειριστής. |
| Πρόσκληση διαχειριστή | `managing_entity.manager.invited` | Έχει σταλεί πρόσκληση διαχειριστή. |
| Αφαίρεση διαχειριστή | `managing_entity.manager.removed` | Ένας διαχειριστής έχει αφαιρεθεί. |
| Αποδοχή διαχειριστή | `managing_entity.manager.accepted` | Η πρόσκληση διαχειριστή έχει γίνει αποδεκτή. |
| Απόρριψη διαχειριστή | `managing_entity.manager.rejected` | Η πρόσκληση διαχειριστή έχει απορριφθεί. |
| Ορισμός ιδιοκτήτη | `managing_entity.manager.owner.set` | Η ιδιοκτησία της οντότητας έχει μεταβιβαστεί. |

### Πληροφορίες οντότητας

| Γεγονός | Κλειδί | Περιγραφή |
|---|---|---|
| Ενημέρωση πληροφοριών | `managing_entity.information.updated` | Οι πληροφορίες της οντότητας έχουν ενημερωθεί. |
| Ενημέρωση διεύθυνσης | `managing_entity.city.updated` | Η διεύθυνση/πόλη της οντότητας έχει αλλάξει. |
| Ενημέρωση εκπροσώπου λογαριασμού | `managing_entity.account_representative.updated` | Ο εκπρόσωπος λογαριασμού έχει αλλάξει. |
| Ενημέρωση φωτογραφίας προφίλ | `managing_entity.profile_picture.updated` | Η φωτογραφία προφίλ της οντότητας έχει ενημερωθεί. |
| Ενημέρωση λογότυπου | `managing_entity.logo.updated` | Το λογότυπο της οντότητας έχει ενημερωθεί. |
| Ενημέρωση ονόματος URL | `managing_entity.url_name.updated` | Το vanity URL της οντότητας έχει αλλάξει. |
| Ενημέρωση μοναδικού αναγνωριστικού | `managing_entity.unique_id.updated` | Το μοναδικό αναγνωριστικό της οντότητας έχει αλλάξει. |
| Ενημέρωση ονόματος | `managing_entity.name.updated` | Το όνομα της οντότητας έχει αλλάξει. |
