Για όποιον χρησιμοποιεί WordPress για την επιχείρησή του ή το ιστολόγιό του, συνιστούμε ανεπιφύλακτα τη χρήση του[Πρόσθετο WordPress για συνεργάτες Wink](https://wordpress.org/plugins/wink2travel/).
Το πρόσθετο σάς παρέχει εύκολη πρόσβαση στις κάρτες Wink, τα πλέγματα, τους χάρτες κ.λπ. και το κάνει τόσο απλό όσο η μεταφορά περιεχομένου στις αναρτήσεις και τις σελίδες σας χρησιμοποιώντας το αγαπημένο σας εργαλείο δημιουργίας σελίδων ή τους προσαρμοσμένους κωδικούς μας.

## Εγκαθιστώ

Έτσι εγκαθιστάτε το`Wink Affiliate WordPress Plugin`στον ιστότοπό σας στο WordPress.

<Steps>
  1. Όταν είστε συνδεδεμένοι ως διαχειριστής και στον Πίνακα Ελέγχου.
  2. Κλικ `Plugins > Add New Plugin`.
  3. Αναζήτηση για `Wink Affiliate WordPress Plugin`.
  4. Κλικ `Install` για να κατεβάσετε και να εγκαταστήσετε το πρόσθετο (plugin) στον ιστότοπό σας.
  5. Κλικ `Activate` για να ενεργοποιήσετε το πρόσθετο στον ιστότοπό σας.
  6. Η ενεργοποίηση θα σας προωθήσει το `Wink Settings` σελίδα. *Μπορείτε επίσης να μεταβείτε εκεί πλοηγούμενοι προς `Appearance > Customize > Wink Settings`.*
  7. Θα σας ζητήσει τα δικά σας `Client ID` και `Client Secret`. Πηγαίνετε στο [Εφαρμογές](/developers/applications) για να μάθετε πώς να ανακτήσετε τα διαπιστευτήριά σας.
  8. Εισαγάγετε τα διαπιστευτήριά σας και κάντε κλικ `Publish`. Είστε πλέον συνδεδεμένοι με τον λογαριασμό σας Wink.
</Steps>

## Δημιουργοί σελίδων

Υποστηρίζουμε τα πιο δημοφιλή εργαλεία δημιουργίας σελίδων στο WordPress που διευκολύνουν όποιον θέλει να χρησιμοποιήσει το Wink.

### Γουτεμβέργιος

Ο Gutenberg είναι ο εγγενής επεξεργαστής από το WordPress.

### Αβάντα

[Αβάντα](https://avada.com)φημίζεται για τη μεγάλη ποικιλία από όμορφα πρότυπα, καθώς και για ένα ισχυρό εργαλείο δημιουργίας σελίδων με ένα προηγμένο σύνολο χαρακτηριστικών.

### Στοιχειωτής

[Στοιχειωτής](https://elementor.com)είναι το πιο δημοφιλές εργαλείο δημιουργίας σελίδων \[αυτήν τη στιγμή] με διαθέσιμη τόσο δωρεάν όσο και επαγγελματική έκδοση.

### WPBakery

[WPBakery](https://wpbakery.com)ήταν το πρώτο προηγμένο εργαλείο δημιουργίας σελίδων στην αγορά.

## Σύντομοι κωδικοί

### \[κλείσιμο ματιού]

Λειτουργεί με πλέγμα περιεχομένου με κατάταξη. Πληκτρολογήστε ένα μέρος που θέλετε να επισκεφθείτε και θα εμφανιστεί το απόθεμά σας.

### \[αναζήτηση ματιών]

Απλό κουμπί για άνοιγμα της φόρμας δρομολογίου.

### \[λογαριασμός ματιού]

Κουμπί λογαριασμού για έλεγχο ταυτότητας. Μόλις ολοκληρωθεί ο έλεγχος ταυτότητας, μετατρέπεται σε αναπτυσσόμενο μενού με επιλογές λογαριασμού.

### \[winkinitineary]

Το ίδιο με το κουμπί αναζήτησης, μόνο που περιέχει πληροφορίες δρομολογίου ως κείμενο κουμπιού.

### \[περιεχόμενο ματιού]

Σας επιτρέπει να ενσωματώσετε το απόθεμα που έχετε επιλέξει στο Wink απευθείας σε μια σελίδα ή ανάρτηση. Αυτός ο σύντομος κώδικας απαιτεί επίσης ένα αναγνωριστικό διάταξης.

Η διάταξη

```
[winkcontent layoutid=""]
```

## Περαιτέρω ανάγνωση

Μπορείτε να βρείτε περισσότερες πληροφορίες σχετικά με το πρόσθετο WordPress από τους παρακάτω συνδέσμους.

* Μάθετε για το[Στοιχεία Ιστού](/developers/web-components).
* Εγκαταστήστε το[Πρόσθετο WordPress για Συνεργάτες Wink](https://wordpress.org/plugins/wink2travel/).
* Διαχωρίστε και δημιουργήστε πάνω από το Github μας[Αποθετήριο WordPress για Συνεργάτες Wink](https://github.com/wink-travel/wink-wordpress-plugin).
