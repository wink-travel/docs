Για όσους χρησιμοποιούν το WordPress για την επιχείρηση ή το blog τους, προτείνουμε ανεπιφύλακτα τη χρήση του [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
Το plugin σας δίνει εύκολη πρόσβαση στις κάρτες, τα πλέγματα, τους χάρτες κ.ά. του Wink, και το κάνει τόσο απλό όσο να σύρετε περιεχόμενο στις αναρτήσεις και τις σελίδες σας χρησιμοποιώντας τον αγαπημένο σας κατασκευαστή σελίδων ή τα δικά μας προσαρμοσμένα shortcodes.

##  Εγκατάσταση

Αυτός είναι ο τρόπος που εγκαθιστάτε το `Wink Affiliate WordPress Plugin` στον ιστότοπό σας WordPress.

<Steps>
1. Όταν έχετε συνδεθεί ως διαχειριστής και βρίσκεστε στον Πίνακα Ελέγχου.
2. Κάντε κλικ στο `Plugins > Add New Plugin`.
3. Αναζητήστε το `Wink Affiliate WordPress Plugin`.
4. Κάντε κλικ στο `Install` για να κατεβάσετε και να εγκαταστήσετε το plugin στον ιστότοπό σας.
5. Κάντε κλικ στο `Activate` για να ενεργοποιήσετε το plugin στον ιστότοπό σας.
6. Η ενεργοποίηση θα σας προωθήσει στη σελίδα `Wink Settings`. *Μπορείτε επίσης να μεταβείτε εκεί πηγαίνοντας στο `Appearance > Customize > Wink Settings`.*
7. Θα σας ζητηθεί το `Client ID` και το `Client Secret`. Μεταβείτε στα [Applications](/el/developers/applications) για να μάθετε πώς να ανακτήσετε τα διαπιστευτήριά σας.
8. Εισάγετε τα διαπιστευτήριά σας και κάντε κλικ στο `Publish`. Τώρα είστε συνδεδεμένοι με τον λογαριασμό σας στο Wink.
</Steps>

## Κατασκευαστές σελίδων

Υποστηρίζουμε τους πιο δημοφιλείς κατασκευαστές σελίδων στο WordPress που κάνουν εύκολη τη χρήση του Wink για όλους.

### Gutenberg

Το Gutenberg είναι ο εγγενής επεξεργαστής του WordPress.

### Avada

Το [Avada](https://avada.com) είναι γνωστό για την μεγάλη επιλογή όμορφων προτύπων μαζί με έναν ισχυρό κατασκευαστή σελίδων με προηγμένο σύνολο λειτουργιών.

### Elementor

Το [Elementor](https://elementor.com) είναι ο [επί του παρόντος] πιο δημοφιλής κατασκευαστής σελίδων με διαθέσιμες δωρεάν και επαγγελματικές εκδόσεις.

### WPBakery

Το [WPBakery](https://wpbakery.com) ήταν ο πρώτος προηγμένος κατασκευαστής σελίδων στην αγορά.

## Short codes

### [winklookup]

Λειτουργεί με πλέγμα περιεχομένου κατάταξης. Πληκτρολογήστε έναν τόπο που θέλετε να επισκεφθείτε και εμφανίστε το διαθέσιμο απόθεμα.

### [winksearch]

Απλό κουμπί για να ανοίξετε τη φόρμα δρομολογίου.

### [winkaccount]

Κουμπί λογαριασμού για να κάνετε αυθεντικοποίηση. Μόλις αυθεντικοποιηθείτε, μετατρέπεται σε αναπτυσσόμενο με επιλογές λογαριασμού.

### [winkitinerary]

Ίδιο με το κουμπί αναζήτησης, μόνο που περιέχει πληροφορίες δρομολογίου ως κείμενο κουμπιού.

### [winkcontent]

Σας επιτρέπει να ενσωματώσετε το απόθεμα που έχετε επιλέξει στο Wink απευθείας σε μια σελίδα ή ανάρτηση. Αυτό το shortcode απαιτεί επίσης ένα layout ID.

Το layout

```
[winkcontent layoutid=""]
```

## Περισσότερη ανάγνωση

Μπορείτε να βρείτε περισσότερες πληροφορίες για το plugin μας για WordPress από τους παρακάτω συνδέσμους.

- Μάθετε για τα [Web Components](/el/developers/web-components).
- Εγκαταστήστε το [Wink Affiliate WordPress plugin](https://wordpress.org/plugins/wink2travel/).
- Κάντε fork και αναπτύξτε πάνω στο Github αποθετήριο μας [Wink Affiliate WordPress repository](https://github.com/wink-travel/wink-wordpress-plugin).
