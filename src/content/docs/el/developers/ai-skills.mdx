Οι δεξιότητες AI είναι φορητά πακέτα οδηγιών που διδάσκουν στους agents κωδικοποίησης πώς να συνεργάζονται με το Wink. Αντί να βασίζονται σε δεδομένα εκπαίδευσης ή ξεπερασμένη τεκμηρίωση, κάθε δεξιότητα κατευθύνει τον agent να χρησιμοποιεί τον [Wink MCP server](/el/developers/mcp) ως την ζωντανή πηγή αλήθειας για τα συμβόλαια API και τα έγγραφα της πλατφόρμας.

Οι δεξιότητες λειτουργούν με οποιονδήποτε client agent που τις υποστηρίζει — Claude Code, Codex, Cursor και άλλους.

## Διαθέσιμες Δεξιότητες

### wink-api-builder

Υλοποιήστε, ελέγξτε ή αποσφαλματώστε ενσωματώσεις Wink API. Η δεξιότητα καθοδηγεί τον agent μέσα από μια δομημένη ροή εργασίας:

1. Αναζήτηση της σωστής λειτουργίας API μέσω `api_search`
2. Άνοιγμα των ζωντανών συμβολαίων αιτήσεων/απαντήσεων από τον MCP server
3. Ανάκτηση ρυθμίσεων αυθεντικοποίησης από τα έγγραφα
4. Γράψιμο κώδικα βάσει του επαληθευμένου συμβολαίου
5. Επισήμανση τυχόν ασυμφωνιών μεταξύ υπάρχοντος κώδικα και του τρέχοντος API

Χρήσιμο για:
- Δημιουργία νέων μεθόδων client για οποιοδήποτε Wink API
- Αποσφαλμάτωση ασυμφωνιών συμβολαίων (π.χ. αίτηση που επιστρέφει HTTP 400)
- Έλεγχο υπαρχουσών ενσωματώσεων σε σχέση με το ζωντανό spec

## Εγκατάσταση

Αντιγράψτε τον φάκελο δεξιότητας στον κατάλογο δεξιοτήτων του agent σας.

:::tip
Ο Wink MCP server πρέπει να είναι συνδεδεμένος στο ίδιο περιβάλλον. Δείτε [MCP Server](/el/developers/mcp) για οδηγίες εγκατάστασης.
:::

**Claude Code**

```bash
# repo-local
cp -r skills/wink-api-builder .claude/skills/

# or user-level
cp -r skills/wink-api-builder ~/.claude/skills/
```

Εκτελέστε με:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repo-local
cp -r skills/wink-api-builder .agents/skills/

# or user-level
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Εκτελέστε με:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Άλλοι agents**

Τοποθετήστε τον φάκελο δεξιότητας όπου κι αν αναζητά δεξιότητες ο agent σας. Το σημείο εισόδου είναι το `SKILL.md`.
