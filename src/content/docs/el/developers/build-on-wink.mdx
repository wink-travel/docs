> «Αν έχω δει περισσότερα, είναι στέκοντας στους ώμους γιγάντων.»

\--*Σερ Ισαάκ Νεύτων*

Μια πλατφόρμα σαν κι αυτή δεν κατασκευάζεται μεμονωμένα. Αντίθετα, είναι το άθροισμα πολλών κινούμενων μερών, παλιών και νέων. Θέλουμε να συνεχίσουμε να υπηρετούμε και άλλους που θα έρθουν μετά από εμάς.

> «Δεν υπάρχει λόγος να επανεφεύρουμε τον σπασμένο τροχό».

Η τεχνολογία ταξιδιών είναι κατακερματισμένη και έχει περάσει στην ιστορία. Προσπαθήσαμε να αξιοποιήσουμε τα καλύτερα από τα παλιά που λειτουργούν. Έχουμε ενσωματώσει μια σειρά από διαχειριστές καναλιών, συστήματα διαχείρισης ακινήτων ++. Έχουμε προσθέσει μερικά νέα στοιχεία. Και τα έχουμε ενσωματώσει όλα σε βιομηχανικά πρότυπα και πρωτόκολλα που είναι γνωστά και εύκολα καταναλώσιμα... ώστε να μην χρειάζεται να το κάνετε εσείς.

:::note\[Προσέγγιση που βασίζεται σε API]
Σε γενικές γραμμές, όποιες λειτουργίες και αν δημιουργήσουμε για εμάς θα πρέπει να είναι:

* Κατανοητός
* Εύκαμπτος
* Ανάβατος
* Επαναχρησιμοποιήσιμο και...
* Επεκτάσιμο

...από άλλους που θέλουν επίσης να χρησιμοποιήσουν αυτά τα API.
:::

Το Wink κατασκευάστηκε χρησιμοποιώντας ένα τέτοιο`API-driven approach`Ως αποτέλεσμα αυτής της φιλοσοφίας σχεδιασμού, όλες οι λειτουργίες του Wink μπορούν να χρησιμοποιηθούν ως εξωτερικός προγραμματιστής σαν να ήταν δικό σας σύστημα. Υπάρχουν επίσης και άλλες πτυχές της πλατφόρμας που μπορείτε να αξιοποιήσετε κατά την ενσωμάτωσή τους μαζί μας.
