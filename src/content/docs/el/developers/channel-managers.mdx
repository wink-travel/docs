Το Wink ενσωματώνεται με κορυφαίους channel managers, συστήματα διαχείρισης ακινήτων (PMS) και κεντρικά συστήματα κρατήσεων (CRS) για να διατηρεί τις τιμές και τη διαθεσιμότητά σας συγχρονισμένες σε όλα τα κανάλια πωλήσεών σας.

## Πώς λειτουργεί

Οι ενσωματώσεις channel manager επιτρέπουν μια αμφίδρομη ροή δεδομένων μεταξύ του συστήματος διαχείρισης ακινήτων σας και του Wink:

1. **Αποστολή τιμών** — Ο channel manager σας στέλνει ενημερώσεις τιμών στο Wink. Όταν ενημερώνετε τις τιμές στο PMS σας, αυτές οι αλλαγές αντικατοπτρίζονται αυτόματα στο Wink.
2. **Συγχρονισμός διαθεσιμότητας** — Ο channel manager σας στέλνει ενημερώσεις διαθεσιμότητας στο Wink. Όταν δωμάτια κλείνονται ή απελευθερώνονται στο PMS σας, το απόθεμα του Wink ενημερώνεται σε πραγματικό χρόνο.
3. **Ειδοποιήσεις κρατήσεων** — Όταν γίνεται μια κράτηση στο Wink, ο channel manager σας ειδοποιείται ώστε η κράτηση να εμφανιστεί στο PMS σας.

## Υποστηριζόμενοι channel managers

Το Wink ενσωματώνεται αυτή τη στιγμή με τα ακόλουθα συστήματα:

| Channel Manager | Περιγραφή |
|---|---|
| **Channex** | Παγκόσμια πλατφόρμα διανομής και διαχείρισης καναλιών. |
| **TravelClick** | Λύσεις ξενοδοχείων και διαχείριση καναλιών (Amadeus). |
| **Omnibees** | Διανομή και διαχείριση εσόδων για ξενοδοχεία. |
| **D-Edge** | Τεχνολογία φιλοξενίας και λύσεις κεντρικών κρατήσεων. |
| **Allotz** | Κατανομή αποθέματος και διαχείριση καναλιών. |
| **Ezee / Yanolja** | Cloud PMS και διαχείριση καναλιών με έμφαση στην ασιατική αγορά. |
| **RateGain** | Τεχνολογία ανάλυσης τιμών και διανομής. |
| **RateTiger** | Διανομή τιμών και διαθεσιμότητας σε πραγματικό χρόνο. |
| **CloudBeds** | Cloud-based PMS, channel manager και μηχανή κρατήσεων. |
| **SiteMinder** | Κορυφαίος channel manager ξενοδοχείων και πλατφόρμα διανομής. |
| **Exely** | Συγχρονισμός τιμών και διαθεσιμότητας για ξενοδοχεία. |

:::note
Αν ο channel manager σας δεν περιλαμβάνεται στη λίστα, επικοινωνήστε με την ομάδα του Wink. Νέες ενσωματώσεις μπορούν να προστεθούν βάσει των προτύπων πρωτοκόλλου OTA (Open Travel Alliance) που υποστηρίζει το Wink.
:::

## Για προγραμματιστές

Αν αναπτύσσετε ενσωμάτωση channel manager ή PMS με το Wink, χρησιμοποιήστε τα API endpoints του Channel Manager:

- **Ενημερώσεις τιμών** — Στείλτε αλλαγές τιμών στο Wink.
- **Ενημερώσεις διαθεσιμότητας** — Στείλτε αλλαγές διαθεσιμότητας στο Wink.
- **Ανάκτηση κρατήσεων** — Λάβετε κρατήσεις από το Wink στο σύστημά σας.

Το Channel Manager API ακολουθεί τα πρότυπα πρωτοκόλλου OTA (SOAP/XML) για συμβατότητα με υπάρχοντα συστήματα φιλοξενίας.

## Ειδοποιήσεις Webhook

Μπορείτε να εγγραφείτε σε γεγονότα webhook του channel manager για να λαμβάνετε ειδοποιήσεις σε πραγματικό χρόνο:

- `channel-manager.update.rate` — Λήψη ενημέρωσης τιμής.
- `channel-manager.update.availability` — Λήψη ενημέρωσης διαθεσιμότητας.
- `channel-manager.update` — Γενική ενημέρωση channel manager.

Δείτε τον [Κατάλογο Γεγονότων Webhook](/el/developers/webhook-events#channel-manager-events) για λεπτομέρειες.
