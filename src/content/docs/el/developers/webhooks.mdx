Τα webhooks είναι ένας τρόπος για να λαμβάνετε και να αντιδράτε σε συμβάντα σε πραγματικό χρόνο από το Wink και το TripPay που αφορούν τον λογαριασμό σας.

## Παράδειγμα

<Steps>
  1. Μια κράτηση γίνεται στο Wink.
  2. Η εκδήλωση Wink `booking.created` αποστέλλεται.
  3. Το σύστημά σας ακούει ενεργά αυτό το συμβάν.
  4. Το σύστημά σας παρακολουθεί το δικό σας σύνολο αναλυτικών στοιχείων και χρησιμοποιεί αυτό το συμβάν για να ενημερώσει τη βάση δεδομένων χρονοσειρών του.
</Steps>

Ένα webhook Wink διαφέρει από ένα webhook TripPay.

* Αν θέλετε να ακούσετε τα συμβάντα Wink, μπορείτε να δημιουργήσετε ένα webhook από[Εξωδίκτυο](https://extranet.wink.travel)ή[Στούντιο](https://studio.wink.travel).
* Αν θέλετε να ακούσετε συμβάντα του TripPay, μπορείτε να δημιουργήσετε ένα webhook από[Πληρωμή](https://pay.wink.travel). :::

*Για αυτό το παράδειγμα, θα κάνουμε έλεγχο ταυτότητας μέσω[Στούντιο](https://studio.wink.travel).*

<Steps>
  1. Μόλις ολοκληρωθεί η επαλήθευση, κάντε κλικ στο εικονίδιο του προφίλ σας, στην επάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
  2. Κάντε κλικ στο `Webhooks` σύνδεσμος.
  3. Θα ανακατευθυνθείτε στη σελίδα των webhooks σας.
</Steps>


## Δημιουργία webhook

:::προσοχή
Πρέπει να δημιουργήσετε έναν λογαριασμό συνεργάτη στο Wink ή έναν λογαριασμό στο TripPay πριν δημιουργήσετε ένα webhook, καθώς θα συνδέσετε το webhook σας με έναν συγκεκριμένο λογαριασμό.
:::

Ακολουθούν τα βήματα για τη δημιουργία ενός webhook:

<Steps>
  1. Από τη σελίδα webhooks, κάντε κλικ στο `Create new webhook` κουμπί.
  2. ***Ονομα*** Δώστε ένα όνομα στο webhook σας. *π.χ. Wink Stalker*
  3. ***Καθιστώ ικανό*** Αφήστε την ως έχει για να ενεργοποιηθεί.
  4. **Όνομα οντότητας** Συνδέστε το webhook σας με έναν από τους υπάρχοντες λογαριασμούς σας. *π.χ. Λογαριασμός Cool*
  5. **Εκδηλώσεις** Ακούστε τα γεγονότα που σας ενδιαφέρουν. *π.χ. `booking.created`*
  6. **URL ιστού** Εισαγάγετε τη διεύθυνση URL όπου θα ακούτε εκδηλώσεις από εμάς. *π.χ. `https://my.system.com/webhook/wink`*
  7. Κάντε κλικ στο `Save` κουμπί για να συνεχίσετε.
</Steps>


Θα ανακατευθυνθείτε πίσω στη λίστα με τα webhooks σας και το νέο σας webhook είναι έτοιμο.

## Ενημέρωση webhook

Θα χρειαστεί να ενημερώσετε την εφαρμογή σας εάν θέλετε:

* Απενεργοποιήστε το webhook σας.
* Ακούστε περισσότερα / λιγότερα γεγονότα.
* Αλλάξτε τη διεύθυνση URL του webhook ακρόασης.

Για να ενημερώσετε ένα υπάρχον webhook, κάντε τα εξής:

<Steps>
  1. Κάντε κλικ στο `Actions` σύνδεσμο για το webhook που θέλετε να ενημερώσετε.
  2. Κάντε κλικ στο `Update` κουμπί στην ενότητα Ενέργειες.
  3. Κάντε τις αλλαγές σας στο webhook.
  4. Κάντε κλικ στο `Save` κουμπί για να συνεχίσετε.
</Steps>


Θα ανακατευθυνθείτε πίσω στη λίστα με τα webhooks σας.

## Κατάργηση webhook

Εάν δεν έχετε πλέον καμία χρήση για το webhook, μπορείτε να προχωρήσετε και να το αφαιρέσετε.

Για να καταργήσετε ένα webhook, κάντε τα εξής:

<Steps>
  1. Κάντε κλικ στο `Actions` σύνδεσμο για το webhook που θέλετε να ενημερώσετε.
  2. Κάντε κλικ στο `Remove` κουμπί στην ενότητα Ενέργειες.
  3. Κλικ `OK` κουμπί για επιβεβαίωση της διαγραφής.
</Steps>


Θα ανακατευθυνθείτε πίσω στη λίστα με τα webhooks σας.
