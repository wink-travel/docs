Τα Webhooks είναι ένας τρόπος για να λαμβάνετε και να αντιδράτε σε πραγματικό χρόνο σε γεγονότα από το Wink και το TripPay που αφορούν τον λογαριασμό σας.

## Παράδειγμα

<Steps>
1. Γίνεται μια κράτηση στο Wink.
1. Το γεγονός του Wink `booking.created` αποστέλλεται.
2. Το σύστημά σας ακούει ενεργά αυτό το γεγονός.
3. Το σύστημά σας παρακολουθεί το δικό του σύνολο αναλύσεων και χρησιμοποιεί αυτό το γεγονός για να ενημερώσει τη βάση δεδομένων χρονικών σειρών του.
</Steps>

:::note
Ένα webhook του Wink διαφέρει από ένα webhook του TripPay.

- Αν θέλετε να ακούτε γεγονότα του Wink, μπορείτε να δημιουργήσετε ένα webhook από το [Extranet](https://extranet.wink.travel) ή το [Studio](https://studio.wink.travel).
- Αν θέλετε να ακούτε γεγονότα του TripPay, μπορείτε να δημιουργήσετε ένα webhook από το [Payment](https://pay.wink.travel).
:::

*Για αυτό το παράδειγμα, θα αυθεντικοποιηθούμε μέσω του [Studio](https://studio.wink.travel).*

<Steps>
1. Μόλις αυθεντικοποιηθείτε, κάντε κλικ στο εικονίδιο του προφίλ σας, στην πάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
2. Κάντε κλικ στον σύνδεσμο `Webhooks`.
3. Θα ανακατευθυνθείτε στη σελίδα των webhooks σας.
</Steps>

## Δημιουργία webhook

:::caution
Πρέπει να δημιουργήσετε έναν λογαριασμό συνεργάτη στο Wink ή έναν λογαριασμό στο TripPay πριν δημιουργήσετε ένα webhook, καθώς θα συνδέσετε το webhook σας με έναν συγκεκριμένο λογαριασμό.
:::

Ακολουθούν τα βήματα για τη δημιουργία ενός webhook:

<Steps>
1. Από τη σελίδα των webhooks, κάντε κλικ στο κουμπί `Create new webhook`.
2. ***Όνομα*** Δώστε ένα όνομα στο webhook σας. *π.χ. Wink Stalker*
3. ***Ενεργοποίηση*** Αφήστε ως έχει για να είναι ενεργοποιημένο.
4. **Όνομα οντότητας** Συνδέστε το webhook σας με έναν από τους υπάρχοντες λογαριασμούς σας. *π.χ. Cool Account*
5. **Γεγονότα** Ακούστε τα γεγονότα που σας ενδιαφέρουν. *π.χ. `booking.created`*
6. **URL Webhook** Εισάγετε το URL όπου θα ακούτε τα γεγονότα από εμάς. *π.χ. `https://my.system.com/webhook/wink`*
7. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των webhooks σας και το νέο σας webhook είναι έτοιμο προς χρήση.

## Ενημέρωση webhook

Θα χρειαστεί να ενημερώσετε την εφαρμογή σας αν θέλετε να:

- Απενεργοποιήσετε το webhook σας.
- Ακούσετε περισσότερα ή λιγότερα γεγονότα.
- Αλλάξετε το URL ακρόασης του webhook.

Για να ενημερώσετε ένα υπάρχον webhook, κάντε τα εξής:

<Steps>
1. Κάντε κλικ στον σύνδεσμο `Actions` για το webhook που θέλετε να ενημερώσετε.
2. Κάντε κλικ στο κουμπί `Update` κάτω από το Actions.
3. Κάντε τις αλλαγές σας στο webhook.
4. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των webhooks σας.

## Αφαίρεση webhook

Αν δεν χρειάζεστε πλέον το webhook, μπορείτε να το αφαιρέσετε.

Για να αφαιρέσετε ένα webhook, κάντε τα εξής:

<Steps>
1. Κάντε κλικ στον σύνδεσμο `Actions` για το webhook που θέλετε να αφαιρέσετε.
2. Κάντε κλικ στο κουμπί `Remove` κάτω από το Actions.
3. Κάντε κλικ στο κουμπί `OK` για να επιβεβαιώσετε τη διαγραφή.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των webhooks σας.
