Όλα τα REST API μας δεν εξαρτώνται από κάποια γλώσσα και είναι συμβατά με Open API. Ξεκινήστε τις δοκιμές χωρίς να γράψετε ούτε μία γραμμή κώδικα.

Αν δεν έχετε ήδη δημιουργήσει τον λογαριασμό σας Wink,[ξεκινήστε εδώ](/user/register).

## API πληρωμών

Τα τελικά σημεία πληρωμής απευθύνονται σε προγραμματιστές που θέλουν να αγοράσουν ταξιδιωτικό απόθεμα. Αυτό μπορεί να γίνει μέσω του API ως εγγεγραμμένος ταξιδιωτικός πράκτορας ή χρησιμοποιώντας το API μας σε συνδυασμό με το γραφικό στοιχείο πληρωμής μας που είναι συμβατό με PCI για όλες τις άλλες οντότητες.
Το API πληρωμής εκθέτει τελικά σημεία για συνεργάτες και ξενοδοχεία για την παρακολούθηση κρατήσεων, αναλυτικών στοιχείων και διαθεσιμότητας χρημάτων. Μπορούν επίσης να επιλέξουν να αποσύρουν τα διαθέσιμα χρήματα στον τραπεζικό τους λογαριασμό.

  Μάθετε περισσότερα
</LinkButton>
