## Επισκόπηση
Ο φιλοξενούμενος διακομιστής MCP της Wink είναι διαθέσιμος στο:

```
https://docs.mcp.wink.travel/mcp
```

Κατευθύνετε συμβατούς πελάτες AI (VS Code, Cursor, άλλους που υλοποιούν MCP HTTP μεταφορά) σε αυτό το σημείο τερματισμού για να τους επιτρέψετε να:
- Περιηγηθούν στην τεκμηρίωση προϊόντων & πλατφόρμας (`wink://docs/...`)
- Ανακαλύψουν SDKs & εξερευνήσουν κλάσεις API / μοντέλα (`wink://sdk/...` & `wink://sdk/.../docs`)
- Εκτελέσουν ελαφριά σημασιολογική / αναζήτηση με λέξεις-κλειδιά πάνω σε τεκμηρίωση & περιλήψεις SDK

## Μεταφορά
Χρησιμοποιήστε το κύριο ροή HTTP endpoint:

```
https://docs.mcp.wink.travel/mcp
```

Αν ο πελάτης σας υποστηρίζει μόνο SSE, χρησιμοποιήστε:

```
https://docs.mcp.wink.travel/sse
```

### Cursor
Τεκμηρίωση MCP για Cursor: https://cursor.com/docs/context/mcp  
Ρύθμιση: Προσθέστε απομακρυσμένο διακομιστή μέσω ρυθμίσεων MCP.
```json
{
  "mcpServers": {
    "wink-docs": {
      "url": "https://docs.mcp.wink.travel/mcp",
    }
  }
}
```

### VS Code
Επίσημη τεκμηρίωση: https://code.visualstudio.com/docs/copilot/customization/mcp-servers  
Ελάχιστη απομακρυσμένη ρύθμιση (αρχείο `mcp.json` χρήστη ή χώρου εργασίας):
```json
{
  "servers": {
    "wink-docs": {
      "type": "http",
      "url": "https://docs.mcp.wink.travel/mcp"
    }
  }
}
```

Συμβουλές:
- Χρησιμοποιήστε την Command Palette: “MCP: List Servers” για να επιβεβαιώσετε τη σύνδεση & να περιηγηθείτε στα εργαλεία.
- Ο επιλογέας εργαλείων σας επιτρέπει να ενεργοποιήσετε/απενεργοποιήσετε `sdk_search` και `docs_search`.
- Προσθέστε πλαίσιο: Προβολή συνομιλίας → Προσθήκη Πλαισίου → Πόροι MCP για να επισυνάψετε σελίδες τεκμηρίωσης.

## Βασικά URI Πόρων

| URI | Περιγραφή |
|-----|-------------|
| `wink://docs` | Ευρετήριο τεκμηρίωσης (ομαδοποιημένο κατά κατηγορία) |
| `wink://docs/{path}` | Συγκεκριμένο άρθρο τεκμηρίωσης (Markdown / MDX) |
| `wink://sdk` | Επισκόπηση SDK (κάθε SDK + μετρήσεις τεκμηρίωσης) |
| `wink://sdk/{sdkPath}` | README SDK / φάκελος τεκμηρίωσης module / συγκεκριμένο έγγραφο module |

### Τεκμηρίωση Module
Το άνοιγμα φακέλου τεκμηρίωσης SDK (π.χ. `wink://sdk/python/wink-sdk-booking/docs`) επιστρέφει ομαδοποιημένα:
- **Κλάσεις API** (αριθμός μεθόδων + δείγματα ονομάτων μεθόδων)
- **Μοντέλα & Τύποι** (δομές δεδομένων που χρησιμοποιούνται από αυτά τα APIs)

## Εργαλεία Αναζήτησης (Σύντομη Αναφορά)
| Εργαλείο | Σκοπός | Τυπικά Παραδείγματα Ερωτημάτων |
|----------|---------|-------------------------------|
| `sdk_search` | Βρείτε ποιο SDK / κλάση API να χρησιμοποιήσετε | `cancel booking`, `list properties`, `affiliate inventory` |
| `docs_search` | Βρείτε εννοιολογική τεκμηρίωση / οδηγούς | `authentication`, `webhooks`, `wordPress` |

Αφήστε το `query` κενό για να λάβετε προτάσεις. Κάθε αποτέλεσμα περιλαμβάνει το URI MCP που μπορείτε να ανοίξετε.

## Τυπικό Πρότυπο Χρήσης
1. Εκτελέστε `sdk_search` με την πρόθεσή σας (π.χ. *"list properties"*).
2. Ανοίξτε το επιστρεφόμενο README SDK ή το `docs_uri` του για να εξετάσετε APIs / μοντέλα.
3. Χρησιμοποιήστε `docs_search` για υποστηρικτικές έννοιες (authentication, webhooks).

## Σημειώσεις
- Κάθε επιστρεφόμενο URI μπορεί να ανοιχτεί από τον πράκτορα για πλήρες περιεχόμενο.
- Οι προτάσεις εμφανίζονται αυτόματα όταν τα ερωτήματα είναι κενά.
