## Επισκόπηση
Ο φιλοξενούμενος MCP server της Wink είναι διαθέσιμος στο:

```
https://docs.mcp.wink.travel/mcp
```

Διευθύνετε συμβατούς AI clients (VS Code, Cursor, άλλους που υλοποιούν MCP HTTP transport) σε αυτό το endpoint για να τους επιτρέψετε να:
- Περιηγηθούν στην τεκμηρίωση προϊόντων & πλατφόρμας (`wink://docs/...`)
- Περιηγηθούν σε ζωντανά OpenAPI specs και τεκμηρίωση λειτουργιών (`wink://api/...`)
- Εκτελέσουν ελαφριά σημασιολογική / αναζήτηση με λέξεις-κλειδιά πάνω σε docs και ζωντανά APIs

## Μεταφορά
Χρησιμοποιήστε το κύριο streamable HTTP endpoint:

```
https://docs.mcp.wink.travel/mcp
```

Αν ο client σας υποστηρίζει μόνο SSE, χρησιμοποιήστε:

```
https://docs.mcp.wink.travel/sse
```

### Claude Code
Επίσημη τεκμηρίωση: [Connect Claude Code to tools via MCP](https://code.claude.com/docs/en/mcp)

Η Anthropic προτείνει τη μεταφορά HTTP για απομακρυσμένους MCP servers που φιλοξενούνται στο cloud:
```bash
claude mcp add --transport http wink-docs https://docs.mcp.wink.travel/mcp
claude mcp list
```

Μέσα στο Claude Code, εκτελέστε `/mcp` για να ελέγξετε την κατάσταση του server. Αν θέλετε να μοιραστείτε τον server με ολόκληρο το repo, προσθέστε `--scope project` ώστε το Claude Code να γράψει τη ρύθμιση στο `.mcp.json`.

### Codex
Επίσημη τεκμηρίωση: [Docs MCP quickstart for Codex](https://developers.openai.com/resources/docs-mcp)

Η OpenAI τεκμηριώνει τη ρύθμιση MCP μέσω του Codex CLI, και σημειώνει ότι η διαμόρφωση μοιράζεται με την επέκταση IDE:
```bash
codex mcp add wink-docs --url https://docs.mcp.wink.travel/mcp
codex mcp list
```

### Cursor
Cursor MCP docs: https://cursor.com/docs/context/mcp
Ρύθμιση: Προσθέστε απομακρυσμένο server μέσω των ρυθμίσεων MCP.
```json
{
  "mcpServers": {
    "wink-docs": {
      "url": "https://docs.mcp.wink.travel/mcp",
    }
  }
}
```

### VS Code
Επίσημη τεκμηρίωση: https://code.visualstudio.com/docs/copilot/customization/mcp-servers
Ελάχιστη απομακρυσμένη ρύθμιση (user ή workspace `mcp.json`):
```json
{
  "servers": {
    "wink-docs": {
      "type": "http",
      "url": "https://docs.mcp.wink.travel/mcp"
    }
  }
}
```

Συμβουλές:
- Χρησιμοποιήστε το Command Palette: “MCP: List Servers” για να επιβεβαιώσετε τη σύνδεση & να περιηγηθείτε στα εργαλεία.
- Ο επιλογέας εργαλείων σας επιτρέπει να ενεργοποιήσετε/απενεργοποιήσετε το `api_search` και το `docs_search`.
- Προσθέστε context: Προβολή συνομιλίας → Προσθήκη Context → MCP Resources για να επισυνάψετε σελίδες τεκμηρίωσης.

## Βασικά URIs Πόρων

| URI | Περιγραφή |
|-----|-------------|
| `wink://docs` | Ευρετήριο τεκμηρίωσης (ομαδοποιημένο κατά κατηγορία) |
| `wink://docs/{path}` | Συγκεκριμένο άρθρο τεκμηρίωσης (Markdown / MDX) |
| `wink://api` | Ζωντανός κατάλογος OpenAPI που ανακαλύπτεται από τη ρύθμιση swagger της Wink |
| `wink://api/{apiName}` | Συνοπτική περίληψη API με auth, tags, headers και δείγματα λειτουργιών |
| `wink://api/{apiName}/operations` | Πλήρης λίστα λειτουργιών ομαδοποιημένη κατά tag |
| `wink://api/{apiName}/operation/{key}` | Λεπτομέρειες μεμονωμένης λειτουργίας (παράμετροι, σώμα αιτήματος, απαντήσεις) |
| `wink://api/{apiName}/schema/{schemaName}` | Επίλυση σχήματος component με εμφωλευμένα πεδία, τύπους και παραδείγματα |
| `wink://api/{apiName}/operation/{key}/request` | Προβολή εστιασμένη στο αίτημα με απαιτούμενες εισόδους και ελάχιστο παράδειγμα payload |
| `wink://api/{apiName}/operation/{key}/response/{status}` | Προβολή εστιασμένη στην απάντηση για συγκεκριμένο κωδικό κατάστασης |
| `wink://api/{apiName}/operation/{key}/recipe` | Ροή εργασίας προσανατολισμένη σε εργασία για σωστή κλήση της λειτουργίας |

## Εργαλεία Αναζήτησης (Σύντομη Αναφορά)
| Εργαλείο | Σκοπός | Τυπικά Παραδείγματα Ερωτημάτων |
|----------|---------|-------------------------------|
| `api_search` | Εύρεση ζωντανού API ή λειτουργίας για ενσωμάτωση | `create affiliate`, `payment receipt`, `lookup inventory`, `invite manager` |
| `docs_search` | Εύρεση εννοιολογικών docs / οδηγών | `authentication`, `webhooks`, `wordPress` |

Αφήστε το `query` κενό για να λάβετε προτάσεις. Κάθε αποτέλεσμα περιλαμβάνει το MCP URI που μπορείτε να ανοίξετε.
Για αποτελέσματα λειτουργιών, το `api_search` επιστρέφει επίσης συνδεδεμένα URIs για αίτημα, απάντηση, σχήμα και συνταγή ώστε οι agents να πηγαίνουν απευθείας στο πιο χρήσιμο υπο-πόρο.

## Τυπικό Πρότυπο Χρήσης
1. Εκτελέστε `api_search` με την πρόθεσή σας (π.χ. *"create affiliate"* ή *"payment receipt"*).
2. Ανοίξτε το επιστρεφόμενο `wink://api/...` URI, ή ακολουθήστε τα συνδεδεμένα πεδία `request_uri`, `response_uris`, `request_schema_uris`, `response_schema_uris`, και `recipe_uri`.
3. Χρησιμοποιήστε `docs_search` για υποστηρικτικές έννοιες (authentication, webhooks, workflows).

## Δεξιότητες AI

Οι προ-κατασκευασμένες δεξιότητες agent χρησιμοποιούν αυτόν τον MCP server ως πηγή αλήθειας. Οι δεξιότητες δίνουν στον coding agent σας μια δομημένη ροή εργασίας για ανάπτυξη με τα Wink APIs — αντί να μαντεύει τα endpoints, ο agent διαβάζει απευθείας τα ζωντανά συμβόλαια από τον MCP server.

Δείτε [AI Skills](/el/developers/ai-skills) για διαθέσιμες δεξιότητες και οδηγίες εγκατάστασης.

## Σημειώσεις
- Κάθε επιστρεφόμενο URI μπορεί να ανοιχτεί από τον agent για πλήρες περιεχόμενο.
- Το δέντρο `wink://api` είναι ζωντανό και ακολουθεί τον τρέχοντα upstream κατάλογο OpenAPI αντί για ευρετήριο που παράγεται από repo.
- Οι προτάσεις εμφανίζονται αυτόματα όταν τα ερωτήματα είναι κενά.
