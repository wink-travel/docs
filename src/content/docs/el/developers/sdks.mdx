Επιταχύνετε τον χρόνο διάθεσης στην αγορά χρησιμοποιώντας ένα από τα επίσημα υποστηριζόμενα κιτ ανάπτυξης λογισμικού.
Εάν δεν υποστηρίζουμε αυτήν τη στιγμή την αγαπημένη σας γλώσσα, μεταβείτε στο API που θέλετε να χρησιμοποιήσετε και κατεβάστε την προδιαγραφή OpenAPI που διαθέτουμε για κάθε API στο επάνω μέρος της σελίδας. Μπορείτε να δημιουργήσετε το δικό σας SDK είτε διασυνδέοντας ένα από τα δημόσια αποθετήρια SDK μας είτε χρησιμοποιώντας μία από τις γεννήτριες του OpenAPI.

## Ιάβα

Αυτή η βιβλιοθήκη μεταγλωττίστηκε για Java 24. Έχει ρυθμιστεί ώστε να χρησιμοποιεί το reactive WebClient και είναι βελτιστοποιημένη για χρήση με το Spring Framework. Ωστόσο, το Spring δεν είναι απαιτούμενο, αλλά σας παρέχει υποστήριξη OAuth2 αμέσως.
