Επιταχύνετε τον χρόνο διάθεσης στην αγορά χρησιμοποιώντας ένα από τα επίσημα υποστηριζόμενα Software Development Kits για γλώσσες προγραμματισμού.
Αν δεν υποστηρίζουμε προς το παρόν την αγαπημένη σας γλώσσα, μεταβείτε στο API που θέλετε να χρησιμοποιήσετε και κατεβάστε το OpenAPI spec που διαθέτουμε για κάθε API στην κορυφή της σελίδας. Μπορείτε να δημιουργήσετε το δικό σας SDK είτε κάνοντας fork σε ένα από τα δημόσια αποθετήρια SDK μας είτε χρησιμοποιώντας έναν από τους γεννήτορες του OpenAPI.

## Java

Αυτή η βιβλιοθήκη έχει μεταγλωττιστεί για Java 24. Είναι ρυθμισμένη να χρησιμοποιεί τον αντιδραστικό WebClient και είναι βελτιστοποιημένη για χρήση με το Spring Framework. Ωστόσο, το Spring δεν είναι απαραίτητο, αλλά παρέχει υποστήριξη OAuth2 έτοιμη προς χρήση.
