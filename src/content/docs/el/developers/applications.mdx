Οι εφαρμογές σας δίνουν πλήρη πρόσβαση σε λειτουργίες στο Wink και το TripPay και επιτρέπουν στους χρήστες σας να αυθεντικοποιούνται και να συνδέονται στον ιστότοπό σας χρησιμοποιώντας τον διακομιστή εξουσιοδότησης OAuth2 μας· όπως ακριβώς το Facebook και η Google.  
Μπορείτε να δημιουργήσετε το δικό σας OTA με το Wink. Ξεκινά με μια εφαρμογή.

*Από προεπιλογή, λαμβάνετε μια εφαρμογή όταν εγγράφεστε σε νέο λογαριασμό συνεργάτη Wink ή λογαριασμό TripPay. Χρησιμοποιούμε το `client ID` για ορισμένα σενάρια.*

:::note
Μια εφαρμογή που δημιουργείται στο Wink ή το TripPay μπορεί να χρησιμοποιηθεί για όλα τα API endpoints.
:::

*Για αυτό το παράδειγμα, θα αυθεντικοποιηθούμε μέσω του [Wink Studio](https://studio.wink.travel).*

<Steps>
1. Μόλις αυθεντικοποιηθείτε, κάντε κλικ στο εικονίδιο προφίλ σας, στην επάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
2. Κάντε κλικ στον σύνδεσμο `Applications`.
3. Θα ανακατευθυνθείτε στη σελίδα των εφαρμογών σας.
</Steps>

## Δημιουργία εφαρμογής

Καθώς δεν μπορείτε να έχετε πρόσβαση στο `secret key` της εφαρμογής που δημιουργήσαμε για εσάς, θα χρειαστεί να δημιουργήσετε τη δική σας.

:::caution
Πρέπει να δημιουργήσετε λογαριασμό συνεργάτη στο Wink ή λογαριασμό στο TripPay πριν δημιουργήσετε μια εφαρμογή, καθώς θα συνδέσετε την εφαρμογή σας με συγκεκριμένο λογαριασμό.
:::

Ακολουθούν τα βήματα για τη δημιουργία μιας εφαρμογής:

<Steps>
1. Από τη σελίδα των εφαρμογών, κάντε κλικ στο κουμπί `Create new application`.
2. ***Όνομα*** Δώστε ένα όνομα στην εφαρμογή σας. *π.χ. Cool App*
3. **Όνομα οντότητας** Συνδέστε την εφαρμογή σας με έναν από τους υπάρχοντες λογαριασμούς σας. *π.χ. Cool Account*
4. **Redirect URIs** Απαιτείται τουλάχιστον ένα redirect URI. Χρησιμοποιείται για την επικύρωση του domain ανακατεύθυνσης μετά από επιτυχή αυθεντικοποίηση. *π.χ. https://www.cool-site.com*
5. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των εφαρμογών σας. Θα δείτε τη νέα σας εφαρμογή στη λίστα και θα έχετε πρόσβαση στο μυστικό κλειδί της εφαρμογής μόνο αυτή τη φορά. Φροντίστε να το αντιγράψετε και να το αποθηκεύσετε σε ασφαλές μέρος.

## Ενημέρωση εφαρμογής

Θα χρειαστεί να ενημερώσετε την εφαρμογή σας αν αλλάξει κάποιο από τα redirect URIs ή αν θέλετε να προσθέσετε ή να αφαιρέσετε κάποιο URI.

Για να ενημερώσετε μια υπάρχουσα εφαρμογή, κάντε τα εξής:

<Steps>
1. Κάντε κλικ στον σύνδεσμο `Actions` για την εφαρμογή που θέλετε να ενημερώσετε.
2. Κάντε κλικ στο κουμπί `Update` κάτω από το Actions.
3. Κάντε τις αλλαγές στην εφαρμογή.
4. Κάντε κλικ στο κουμπί `Save` για να συνεχίσετε.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των εφαρμογών σας.

## Αφαίρεση εφαρμογής

:::caution
Παρακαλούμε μην διαγράψετε την εφαρμογή που δημιουργήσαμε αυτόματα για εσάς, ώστε να διασφαλιστεί η λειτουργία όλων των δυνατοτήτων για τον λογαριασμό σας.
:::

Αν δεν χρειάζεστε πλέον μια εφαρμογή, μπορείτε να την αφαιρέσετε.

Για να αφαιρέσετε μια εφαρμογή, κάντε τα εξής:

<Steps>
1. Κάντε κλικ στον σύνδεσμο `Actions` για την εφαρμογή που θέλετε να αφαιρέσετε.
2. Κάντε κλικ στο κουμπί `Remove` κάτω από το Actions.
3. Κάντε κλικ στο κουμπί `OK` για να επιβεβαιώσετε τη διαγραφή.
</Steps>

Θα ανακατευθυνθείτε πίσω στη λίστα των εφαρμογών σας.
