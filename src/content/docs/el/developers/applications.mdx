Οι εφαρμογές σάς παρέχουν πλήρη πρόσβαση στις λειτουργίες του Wink και του TripPay και επιτρέπουν στους χρήστες σας να ελέγχουν την ταυτότητά τους και να συνδέονται στον ιστότοπό σας χρησιμοποιώντας τον διακομιστή εξουσιοδότησης OAuth2, όπως ακριβώς συμβαίνει στο Facebook και την Google.
Μπορείτε να δημιουργήσετε το δικό σας OTA με το Wink. Ξεκινά με μια εφαρμογή.

*Από προεπιλογή, λαμβάνετε μια εφαρμογή όταν εγγράφετε έναν νέο λογαριασμό συνεργάτη Wink ή έναν λογαριασμό TripPay. Χρησιμοποιούμε το`client ID`για ορισμένα σενάρια.*

:::σημείωση
Μια εφαρμογή που δημιουργείται στο Wink ή στο TripPay μπορεί να χρησιμοποιηθεί για όλα τα τελικά σημεία API.
:::

*Για αυτό το παράδειγμα, θα κάνουμε έλεγχο ταυτότητας μέσω[Στούντιο Wink](https://studio.wink.travel).*

<Steps>
  1. Μόλις ολοκληρωθεί η επαλήθευση, κάντε κλικ στο εικονίδιο του προφίλ σας, στην επάνω δεξιά γωνία, και θα εμφανιστεί ένα μενού.
  2. Κάντε κλικ στο `Applications` σύνδεσμος.
  3. Θα μεταφερθείτε στη σελίδα των εφαρμογών σας.
</Steps>


## Δημιουργία εφαρμογής

Καθώς δεν έχετε πρόσβαση στο`secret key`της εφαρμογής που δημιουργήσαμε για εσάς, θα χρειαστεί να δημιουργήσετε τη δική σας.

:::προσοχή
Πρέπει να δημιουργήσετε έναν λογαριασμό συνεργάτη στο Wink ή έναν λογαριασμό στο TripPay πριν δημιουργήσετε μια εφαρμογή, καθώς θα συνδέσετε την εφαρμογή σας με έναν συγκεκριμένο λογαριασμό.
:::

Ακολουθούν τα βήματα για τη δημιουργία μιας εφαρμογής:

<Steps>
  1. Από τη σελίδα εφαρμογών, κάντε κλικ στο `Create new application` κουμπί.
  2. ***Ονομα*** Δώστε ένα όνομα στην εφαρμογή σας. *π.χ. Ωραία εφαρμογή*
  3. **Όνομα οντότητας** Συνδέστε την εφαρμογή σας με έναν από τους υπάρχοντες λογαριασμούς σας. *π.χ. Λογαριασμός Cool*
  4. **Ανακατεύθυνση URI** Απαιτείται τουλάχιστον ένα URI ανακατεύθυνσης. Χρησιμοποιείται για την επικύρωση του τομέα ανακατεύθυνσης μετά από έναν επιτυχημένο έλεγχο ταυτότητας. *π.χ. https://www.cool-site.com*
  5. Κάντε κλικ στο `Save` κουμπί για να συνεχίσετε.
</Steps>


Θα ανακατευθυνθείτε πίσω στη λίστα εφαρμογών σας. Θα δείτε τη νέα σας εφαρμογή στη λίστα και θα έχετε πρόσβαση στο μυστικό κλειδί της εφαρμογής μόνο μία φορά. Βεβαιωθείτε ότι το έχετε αντιγράψει και αποθηκεύσει σε ασφαλές μέρος.

## Ενημέρωση εφαρμογής

Θα χρειαστεί να ενημερώσετε την εφαρμογή σας εάν αλλάξει ένα από τα URI ανακατεύθυνσης ή εάν θέλετε να προσθέσετε ή να καταργήσετε ένα URI.

Για να ενημερώσετε μια υπάρχουσα εφαρμογή, κάντε τα εξής:

<Steps>
  1. Κάντε κλικ στο `Actions` σύνδεσμο για την εφαρμογή που θέλετε να ενημερώσετε.
  2. Κάντε κλικ στο `Update` κουμπί στην ενότητα Ενέργειες.
  3. Κάντε τις αλλαγές σας στην εφαρμογή.
  4. Κάντε κλικ στο `Save` κουμπί για να συνεχίσετε.
</Steps>


Θα ανακατευθυνθείτε πίσω στη λίστα εφαρμογών σας.

## Κατάργηση εφαρμογής

:::προσοχή
Παρακαλούμε μην διαγράψετε την εφαρμογή που δημιουργήσαμε αυτόματα για εσάς, για να βεβαιωθούμε ότι όλες οι λειτουργίες λειτουργούν για τον λογαριασμό σας.
:::

Εάν δεν έχετε πλέον καμία χρησιμότητα για μια εφαρμογή, μπορείτε να προχωρήσετε και να την καταργήσετε.

Για να καταργήσετε μια εφαρμογή, κάντε τα εξής:

<Steps>
  1. Κάντε κλικ στο `Actions` σύνδεσμο για την εφαρμογή που θέλετε να ενημερώσετε.
  2. Κάντε κλικ στο `Remove` κουμπί στην ενότητα Ενέργειες.
  3. Κλικ `OK` κουμπί για επιβεβαίωση της διαγραφής.
</Steps>


Θα ανακατευθυνθείτε πίσω στη λίστα εφαρμογών σας.
