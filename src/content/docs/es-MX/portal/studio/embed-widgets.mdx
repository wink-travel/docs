Studio ofrece varios widgets para insertar que puedes agregar a tu propio sitio web para impulsar las reservas.

Para acceder a los widgets para insertar, navega a `Tools > Miscellaneous` desde la barra lateral.

## Widgets disponibles

| Widget | Descripción |
| -- | -- |
| **Account button** | Un botón con marca que enlaza a tu página de afiliado |
| **Itinerary button** | Un botón que abre el selector de itinerarios |
| **Itinerary form** | Un formulario en línea para seleccionar fechas y huéspedes |
| **Search button** | Un botón que abre la búsqueda de propiedades |
| **Lookup button** | Un botón para buscar inventario específico |

## Cómo insertar

Cada widget proporciona un fragmento de código que puedes copiar y pegar en el HTML de tu sitio web. El fragmento incluye tu **Client ID** y **Customization ID** para que el widget se vincule automáticamente a tu cuenta de afiliado.

:::note
Si usas WordPress, también ofrecemos un plugin de WordPress para una integración más sencilla.
:::
