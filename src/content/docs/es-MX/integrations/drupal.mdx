El **Módulo Wink para Drupal** añade componentes del inventario de viajes de Wink a tu sitio Drupal como bloques nativos. Los editores pueden colocarlos en cualquier parte del diseño mediante el sistema estándar de bloques.

<LinkCard
  title="wink-travel/wink-drupal-module en GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Código fuente, guía de configuración para desarrolladores e instrucciones para contribuir."
/>

## Requisitos

- Drupal 10 o 11
- PHP 8.1+
- Composer

## Instalación

<Steps>
1. Instala vía Composer:
   ```bash
   composer require drupal/wink
   ```
2. Activa el módulo:
   ```bash
   drush en wink -y
   ```
3. Ve a **Administración → Configuración → Servicios Web → Configuración de Wink** (`/admin/config/wink`).
4. Ingresa tu **Client ID** y **Client Secret**, selecciona tu entorno y haz clic en **Guardar**.
   Ve a [Aplicaciones](/es-MX/portal/settings/applications) para obtener tus credenciales.
</Steps>

## Agregar un bloque a una página

<Steps>
1. Ve a **Administración → Estructura → Diseño de Bloques**.
2. Haz clic en **Colocar bloque** en la región donde quieres el contenido de Wink.
3. Busca **Wink** para encontrar los bloques disponibles:
   - **Wink Content** — muestra una tarjeta de hotel, cuadrícula o mapa
   - **Wink Lookup** — barra de búsqueda de destinos
   - **Wink Search** — botón para seleccionar itinerarios
   - **Wink Account** — botón de inicio de sesión
   - **Wink Itinerary** — botón de estado del itinerario
   - **Wink Cart** — botón del carrito de compras
4. Para **Wink Content**, configura el **Tipo de diseño** y el **ID de diseño** en la configuración del bloque.
5. Guarda la configuración del bloque.
</Steps>

## Bloques disponibles

| Bloque | Propósito |
|---|---|
| **Wink Content** | Muestra tarjetas de hotel, cuadrículas clasificadas, mapas y más |
| **Wink Lookup** | Barra de búsqueda para destinos y hoteles |
| **Wink Search** | Botón ícono que abre el selector de itinerarios |
| **Wink Account** | Botón de inicio de sesión / cuenta de usuario |
| **Wink Itinerary** | Botón de itinerario que muestra el viaje actual |
| **Wink Cart** | Botón resumen del carrito de compras |
