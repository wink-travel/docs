La **Extensión Wink para Joomla** es un paquete que instala un plugin de sistema (carga scripts CDN), un plugin de contenido (procesa etiquetas shortcode en artículos) y un componente (página de configuración administrativa).

<LinkCard
  title="wink-travel/wink-joomla-extension en GitHub"
  href="https://github.com/wink-travel/wink-joomla-extension"
  description="Código fuente, guía de configuración para desarrolladores e instrucciones para contribuir."
/>

## Requisitos

- Joomla 4 o 5
- PHP 8.1+

## Instalación

<Steps>
1. Descarga el último `pkg_wink-x.x.x.zip` desde la [página de lanzamientos en GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. En el administrador de Joomla, ve a **Extensiones → Administrar → Instalar**.
3. Sube el archivo zip y haz clic en **Subir e instalar**.
4. Ambos plugins se habilitan automáticamente. Ve a **Componentes → Wink Settings**.
5. Ingresa tu **Client ID** y **Client Secret** y haz clic en **Guardar**.
   Ve a [Applications](/es-MX/portal/settings/applications) para obtener tus credenciales.
</Steps>

## Insertar componentes en artículos

Usa etiquetas estilo shortcode en cualquier parte del contenido del artículo:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

El plugin de contenido reemplaza estas etiquetas con el HTML correcto `<wink-*>` al momento de renderizar. El plugin de sistema carga automáticamente los scripts CDN e inyecta `<wink-app-loader>` en el pie de página.

## Shortcodes disponibles

| Shortcode | Salida | Descripción |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Tarjetas de hotel, rejillas, mapas |
| `{wink-lookup}` | `<wink-lookup>` | Barra de búsqueda de destinos |
| `{wink-search}` | `<wink-search-button>` | Botón para seleccionar itinerario |
| `{wink-account}` | `<wink-account-button>` | Iniciar sesión / cuenta de usuario |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Botón de estado del itinerario |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Botón del carrito |
