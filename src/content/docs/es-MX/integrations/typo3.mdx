La **Extensión Wink para TYPO3** integra el inventario de viajes de Wink en TYPO3 13 como elementos de contenido nativos. Los editores agregan componentes de Wink a las páginas usando el editor estándar de páginas de TYPO3 — no se requieren cambios en las plantillas.

## Requisitos

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instalación

<Steps>
1. Instala vía Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Activa la extensión:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. En el backend de TYPO3, ve a **Admin Tools → Settings → Extension Configuration → wink**.
4. Ingresa tu **Client ID**, **Client Secret** y selecciona tu entorno. Guarda.
   Ve a [Applications](/es-MX/portal/settings/applications) para obtener tus credenciales.
5. Agrega la plantilla estática TypoScript a tu plantilla raíz:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Agregar un elemento de contenido a una página

<Steps>
1. Abre una página en el módulo **Page**.
2. Haz clic en el botón **+** en el área de contenido para agregar un nuevo elemento.
3. Elige uno de los elementos de contenido **Wink Travel**:
   - **Wink Content** — tarjetas de hotel, cuadrículas clasificadas, mapas
   - **Wink Lookup** — barra de búsqueda de destinos
   - **Wink Search** — botón selector de itinerarios
   - **Wink Account** — botón de inicio de sesión
   - **Wink Itinerary** — botón de estado del itinerario
   - **Wink Cart** — botón del carrito de compras
4. Para **Wink Content**, usa los campos FlexForm para seleccionar un **Tipo de Diseño** e ingresar el **ID de Diseño**.
5. Guarda y publica la página.
</Steps>

## Elementos de contenido disponibles

| Elemento | Propósito |
|---|---|
| **Wink Content** | Tarjetas de hotel, cuadrículas clasificadas, mapas y más |
| **Wink Lookup** | Barra de búsqueda de destinos |
| **Wink Search** | Botón ícono que abre el selector de itinerarios |
| **Wink Account** | Botón de inicio de sesión / cuenta de usuario |
| **Wink Itinerary** | Botón de itinerario que muestra el viaje actual |
| **Wink Cart** | Botón resumen del carrito de compras |
