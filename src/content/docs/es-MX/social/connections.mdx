Ya estás conectado a WinkLinks, nuestra herramienta de gestión de enlaces. Conéctate a más de tus redes sociales haciendo clic en `Conectar una red social`.
El proceso te redirigirá a tu red social para aprobar la app `Wink Social` para publicar contenido en tu nombre.

Una vez conectado, comenzamos a generar contenido para esa red para el siguiente mes programado.

:::note
El plan gratuito incluye WinkLinks. Facebook, Instagram y otras plataformas están disponibles en planes de pago.
:::

## Plataformas compatibles

| Plataforma | Descripción | Requisitos |
| -- | -- | -- |
| **WinkLinks** | Publicación automática en nuestro avanzado gestor de enlaces | Incluido en todos los planes |
| **Facebook** | Automatiza publicaciones, responde comentarios y mensajes directos | Requiere una Página de Facebook para negocios |
| **Instagram** | Automatiza publicaciones, responde comentarios y mensajes directos | Requiere una cuenta de Instagram para negocios vinculada a una cuenta de Facebook |

Cada conexión muestra su estado actual: **Activa**, **Desconectada** o **Suspendida**.

:::note
Creamos y publicamos contenido optimizado para cada red social.
:::
