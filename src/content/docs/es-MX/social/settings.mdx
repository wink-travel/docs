# Configuración

Cuando comienzas con Social, te preguntaremos cómo quieres que creemos tu contenido. Estas opciones están disponibles para ti en todo momento para que puedas cambiarlas fácilmente más adelante.

Navega a `Preferencias` desde la barra lateral para administrar tus configuraciones.

## Preferencias

La sección de preferencias contiene tus configuraciones básicas para la generación de contenido:

1. **Personalización preferida** Elige qué personalización usar al generar contenido. Las personalizaciones controlan los colores del tema, itinerario predeterminado, moneda, idioma y más. Creas personalizaciones en tu cuenta en [my.wink.travel](https://my.wink.travel).
2. **Aprobación automática** Cuando está activada, las publicaciones generadas se publican automáticamente sin requerir tu aprobación. Actívala una vez que confíes en la calidad del contenido que producimos para ti.
3. **Nombre del aprobador** El nombre de la persona o equipo que recibirá los correos de solicitud de aprobación.
4. **Correo electrónico del aprobador** La dirección de correo donde se enviarán las solicitudes de aprobación para las próximas publicaciones.

## Creadores de contenido

Social incluye muchos creadores de contenido para elegir. Cada creador se enfoca en una forma específica de crear contenido único para tu propiedad usando IA. Cada creador tiene un atributo de **Peso** — un peso más alto aumenta la probabilidad de que ese creador sea seleccionado al generar publicaciones.

Aquí están todos los creadores de contenido disponibles:

| Creador | Descripción |
| -- | -- |
| **Destino** | Usa tu ubicación para crear texto útil sobre la zona |
| **Tipo de habitación** | Crea una publicación reservable con uno de tus tipos de habitación con texto generado por IA y tus imágenes de habitación |
| **Actividad** | Crea una publicación reservable con una de tus actividades |
| **Complemento** | Crea una publicación reservable con uno de tus complementos |
| **Atracción** | Crea una publicación reservable con una de tus atracciones |
| **Lugar** | Crea una publicación reservable con uno de tus lugares |
| **Sala de reuniones** | Crea una publicación reservable con una de tus salas de reuniones |
| **Restaurante** | Crea una publicación reservable con uno de tus restaurantes |
| **Spa** | Crea una publicación reservable con uno de tus spas |
| **Propiedad** | Crea una publicación reservable con tu habitación mejor precio con texto generado por IA y las imágenes de tu propiedad |
| **Sentimiento del usuario** | Agrega reseñas de huéspedes de toda la web en una publicación basada en sentimientos que destaca lo que los huéspedes aman de tu propiedad |
| **Divertido** | Crea publicaciones ligeras y entretenidas para mantener tu feed atractivo |

:::tip[Consejo profesional]
Asegúrate de tener el tipo de inventario requerido para los creadores de contenido que elijas. Por ejemplo, si seleccionas "Restaurante", necesitas tener al menos un restaurante listado en tu propiedad.
:::

El tono de voz establece el estilo de lenguaje que la IA usa al generar contenido. Elige los tonos que coincidan con la personalidad de tu marca. Al igual que con los creadores de contenido, cada tono tiene un **Peso** para controlar con qué frecuencia se usa.

Aquí están los tonos disponibles:

| Tono | Estilo |
| -- | -- |
| **Neutral** | Calmado, autoritario, factual. Sin exageraciones, sin bromas — solo conocimiento confiable y bien estructurado |
| **Amigable** | Cálido, accesible, energía sonriente a través del texto con contracciones casuales y emojis ocasionales |
| **Lujo** | Sofisticado, refinado. Palabras como "exquisito" y "a la medida" que hacen sentir exclusividad |
| **Gourmet** | Descripciones sensuales y apetitosas. Lenguaje de comida que hace que los lectores se antojen |
| **Aventurero** | Audaz, intrépido, energía exploradora. Descripciones sensoriales vívidas que hacen que todo suene épico |
| **Familiar** | Emoción sana y apta para niños. Energía cálida y parental sobre experiencias seguras y divertidas |
| **Romántico** | Poético, nostálgico, que hace latir el corazón. Atardeceres, lenguaje suave y amor |
| **Humorístico** | Sarcástico, frases rápidas, juegos de palabras ingeniosos. Lee como un show de stand-up |
| **Ecológico** | Sincero y justo. Celebra alternativas verdes y elecciones sostenibles |
| **Local experto** | Referencias locales muy específicas, advierte sobre trampas turísticas, usa jerga local |
| **Inclusivo** | Acogedor, celebratorio y afirmativo. Hace que todos se sientan vistos y valorados |
| **Místico** | Etéreo, espiritual, verdades cósmicas. Invitaciones suaves a mirar hacia adentro |
| **Futurista** | Elegante, visionario. Ama términos como "cambio de paradigma" y visiones optimistas del futuro |
| **Económico** | Ingenioso, ingenioso, orgullosamente frugal. Celebra hallazgos asequibles y experiencias gratuitas |

## Tipos de publicación

Los tipos de publicación determinan el objetivo y la estructura del contenido generado. Cada tipo de publicación tiene un **Peso** para controlar con qué frecuencia se usa.

| Tipo de publicación | Descripción |
| -- | -- |
| **Transaccional** | Claro, conciso, enfocado en la conversión. Impulsa reservas y acciones directas con hechos, precios y próximos pasos |
| **Conciencia** | Contenido para construir marca diseñado para presentar tu propiedad y aumentar la visibilidad entre posibles huéspedes |
| **Sentimiento** | Contenido emocionalmente resonante que crea asociaciones positivas y construye una conexión emocional con tu audiencia |

Quizá quieras enfocar tus esfuerzos de marketing en un idioma y región en particular. Dinos los idiomas para los que quieres crear contenido y la zona horaria que te interesa y nosotros haremos el resto. Las publicaciones se publican en el horario programado en la zona horaria seleccionada.

*La mayoría de las propiedades usan un solo idioma y predeterminan su idioma local.*

## Google Business

<CldImage

Al conectar tu Perfil de Google Business, podemos obtener detalles más completos de la propiedad para un contenido más efectivo. Tu propiedad necesita un Perfil de Google Business verificado para que podamos generar contenido desde tu listado. Algunos de nuestros creadores de contenido **requieren** que tengas una cuenta de Google Business.

:::tip[Consejo profesional]
Empieza pequeño

No seas demasiado ambicioso en tu primer intento. Recomendamos que comiences creando contenido para tu propiedad en un solo idioma.
:::
