El editor de publicaciones te permite crear publicaciones manuales ilimitadas y también es el lugar al que vas para administrar las publicaciones que se han creado automáticamente para ti.

El editor tiene 4 secciones:

1. La sección de contenido.
2. La sección de medios.
3. La sección de enlace.
4. La sección de configuración.

## Contenido

La sección de contenido te permite modificar el texto antes de que se publique. Puedes crear contenido localizado en varios idiomas; cada idioma tiene su propio título y descripción.

## Medios

La sección de medios te permite modificar las imágenes y videos que acompañan al texto.

## Enlace

La sección de enlace te permite vincular a algún lugar al que desees que el usuario vaya después de leer tu publicación. *Por ejemplo, puede ser un enlace a una oferta especial o un lugar para obtener más información.* Puedes agregar una URL de enlace, un título para el enlace y una descripción del enlace.

## Configuración

La sección de configuración te permite modificar o sobrescribir todas tus configuraciones predeterminadas por publicación. Puedes establecer:

- **Plataforma** — Elige a qué red social se publicará esta publicación.
- **Tipo de publicación** — Elige entre Transaccional, Conciencia o Sentimiento.
- **Tono de voz** — Sobrescribe el tono de voz para esta publicación específica.
- **Personalización** — Usa una personalización diferente para moneda de visualización, idioma, itinerario, etc.
- **Zona horaria** — Sobrescribe la zona horaria para la publicación.
- **Fecha programada** — Las publicaciones generadas se programan automáticamente en el momento óptimo. Puedes sobrescribir la fecha y hora si prefieres otro horario.

## Ciclo de vida del estado de la publicación

Cada publicación sigue un ciclo de vida de estado:

| Estado | Descripción |
| -- | -- |
| **Borrador** | La publicación ha sido creada pero aún no está lista para revisión |
| **Pendiente de aprobación** | La publicación está esperando aprobación humana antes de programarse |
| **Aprobada** | La publicación ha sido aprobada y será programada |
| **Rechazada** | La publicación fue revisada y rechazada — no se publicará |
| **Programada** | La publicación está en cola para publicarse en la fecha y hora programadas |
| **Publicada** | La publicación se ha publicado exitosamente en la red social |
| **Pausada** | La publicación ha sido pausada temporalmente |
| **Fallida** | La publicación no pudo publicarse debido a un error |

## Aprobación de publicaciones

Las publicaciones se aprueban en la cuadrícula de publicaciones donde puedes seleccionar una o más publicaciones y elegir aprobarlas o rechazarlas.

:::note
Habrá una entrada por cada idioma y cada red social.

Por ejemplo, si tienes 3 redes sociales y 3 idiomas seleccionados, verás 9 entradas en el calendario.
:::
