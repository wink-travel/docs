Un usuario de Wink puede optar por permitirle administrar una de sus cuentas de propiedad, afiliado o agencia de viajes. En ese caso, recibirá un correo electrónico con una solicitud de invitación.

import { Steps } from '@astrojs/starlight/components';

Para ver tus invitaciones de Wink actuales, sigue estos pasos:

<Steps>
  1. Debes iniciar sesión en uno de nuestros sitios web.
  2. Haz clic en el icono de tu perfil, en la esquina superior derecha, y aparecerá un menú.
  3. Haga clic en el `Invites` enlace.
  4. Interactúa con tus invitaciones ya sea `Accepting` o `Rejecting` a ellos.
</Steps>
