## A nivel de plataforma

Todas nuestras aplicaciones están localizadas para soportar todos los idiomas mencionados arriba. Cada aplicación tiene un selector de idioma en la parte inferior de la página desde donde el usuario puede seleccionar su idioma deseado. El motor de reservas también soporta pasar el idioma deseado a través de la URL.

## Contenido generado por el usuario

El contenido generado por el usuario está localizado para soportar todos los idiomas mencionados arriba. Los usuarios pueden elegir crear la versión en inglés del contenido y dejar que Wink haga el resto. O bien, pueden gestionar un conjunto específico de idiomas para tener más control y dejar que Wink maneje el resto.

## Advertencias

El idioma predeterminado es inglés. A menos que se especifique lo contrario, usaremos el idioma predeterminado primero cuando un usuario utilice cualquiera de nuestras aplicaciones. No establecemos el idioma activo basado en la configuración regional del navegador del usuario.
