Un administrador de enlaces te permite guardar y organizar URLs que te gustan para después. Los marcadores de tu navegador son un ejemplo de administrador de enlaces. Con la llegada de las redes sociales, surgió una mayor necesidad de exponer y hacer enlaces disponibles para tus amigos y audiencias. Tu feed de Facebook es un ejemplo de un administrador de enlaces amigable con las redes sociales.

Construimos [WinkLinks](https://i.trvl.as/) para ser el administrador de enlaces para la industria de viajes. Está aquí para informar, emocionar, publicitar... y te permite reservar grandes ofertas de hoteles en todo el mundo.

## Características

- WinkLinks te permite agregar todos tus enlaces de afiliados existentes, como Commission Junction y Amazon, de forma gratuita.
- Obtienes una URL personalizada genial, como [https://i.trvl.as/bob](https://i.trvl.as/bob), para esos momentos en que solo puedes compartir una URL.
- Puedes organizar y compartir enlaces y subir archivos.
- Hoteles, cadenas y marcas pueden usar WinkLinks como un motor de reservas avanzado:
    - Soporta múltiples propiedades.
    - Soporta múltiples destinos.
    - Soporta múltiples monedas.
    - Soporta múltiples idiomas.
- WinkLinks soporta `contenido enriquecido` *(ver abajo)*.
- Ve de IG a reservar en solo 2 clics 🚀

## Contenido enriquecido

:::note
Activa `Inteligente` a 🟢 en tus [Configuraciones](/es-MX/link-manager/settings) para usar esta función.
:::

Si compartes un enlace de un sitio que soporta contenido `enriquecido` / `oEmbed`, tu enlace cobra vida en WinkLinks.

Sitios soportados:

- Facebook - *Interactúa con la publicación y reproduce un video*.
- Instagram - *Interactúa con la publicación y reproduce un video*.
- YouTube - *Reproduce un video*.
- X (antes Twitter) - *Interactúa con una publicación*.
- Spotify - *Reproduce tus listas de reproducción favoritas*.
- Wink - *Ver `Avanzado`*.

## Avanzado

WinkLinks también es una excelente manera de exponer el inventario de viajes que encontraste, a través de [Wink Studio](https://studio.wink.travel), y que quieres publicitar a tu audiencia.

Usa WinkLinks para vender tus:

- [Enlaces compartibles](/es-MX/studio/shareable-links) - Elige mostrar como:
    - [Banner publicitario](/es-MX/integrations/web-components/#content-loader).
    - Enlace normal con actualizaciones de precios en tiempo real.
- [Tarjetas](/es-MX/studio/cards) - Tarjetas reservables con precios y disponibilidad en tiempo real.
- [Cuadrículas](/es-MX/studio/grids) - Cuadrícula con tarjetas reservables.
- [Mapas](/es-MX/studio/maps) - Mapa con marcadores que abren tarjetas reservables.
