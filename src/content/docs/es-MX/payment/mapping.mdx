Si eres un integrador, como Wink, y quieres mapear tus beneficiarios externos para usarlos con TripPay, puedes hacerlo de dos maneras:

- Mapeo manual
- Usando la [Payment API](/es-MX/developers/apis/#payment-api) (*Consulta `Account Mappings`*)

:::tip
A menos que solo manejes un pequeño número de las mismas propiedades, recomendamos usar la API para automatizar este paso.
:::

A continuación, te guiamos por la forma manual de mapear un beneficiario externo. *Utiliza la API en segundo plano.*

<Steps>
1. Inicia sesión en [TripPay](https://pay.wink.travel).
2. Selecciona tu cuenta de integrador.
3. Haz clic en `Settings > Mappings` desde la barra de navegación principal.
4. Haz clic en el botón `Create mapping`. Verás el formulario de mapeo.
5. **Local account** Comienza a escribir el nombre de la cuenta existente de TripPay que quieres mapear.
6. **External identifier** Pega tu ID externo para esta entidad tal como existe en el sistema remoto.
7. Haz clic en `Save` para continuar. Serás redirigido de vuelta a tu lista de mapeos.
</Steps>

Con tus mapeos configurados, TripPay ahora sabe qué cuentas acreditar como beneficiarios de una reserva cuando las refieres usando tus identificadores externos.

## API

Los desarrolladores que quieran gestionar `Payments`, vayan a [Developers > API > Payments](/es-MX/developers/apis/#payment-api).
