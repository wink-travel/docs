Studio tilbyder flere indlejrede widgets, som du kan tilføje til dit eget website for at øge antallet af bookinger.

For at få adgang til indlejrede widgets, gå til `Tools > Miscellaneous` i sidebaren.

## Tilgængelige widgets

| Widget | Beskrivelse |
| -- | -- |
| **Account button** | En brandet knap, der linker til din affiliate-side |
| **Itinerary button** | En knap, der åbner rejseplanvælgeren |
| **Itinerary form** | En inline formular til valg af dato og gæster |
| **Search button** | En knap, der åbner ejendomssøgning |
| **Lookup button** | En knap til at slå specifikt inventar op |

## Sådan indlejres

Hver widget leverer et kodeudsnit, som du kan kopiere og indsætte i dit websites HTML. Udsnittet inkluderer din **Client ID** og **Customization ID**, så widgetten automatisk forbindes til din affiliate-konto.

:::note
Hvis du bruger WordPress, tilbyder vi også en WordPress-plugin for nemmere integration.
:::
