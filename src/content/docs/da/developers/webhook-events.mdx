Denne side viser alle webhook-begivenheder, du kan abonnere på i Wink. For opsætningsinstruktioner, se [Webhook Integration Guide](/da/guides/webhook-integration) eller [Webhooks management page](/da/my-account/webhooks).

## Sådan abonnerer du

1. Gå til **Applications > Webhooks** i din portal.
2. Opret en webhook og vælg de begivenheder, du ønsker at modtage.
3. Wink sender en HTTP POST til dit endpoint hver gang begivenheden opstår.

For fulde opsætningsvejledninger og bedste praksis, se [Webhook Integration](/da/guides/webhook-integration).

## Konto-begivenheder

Begivenheder relateret til din TripPay-kontos livscyklus og indstillinger.

| Begivenhed | Nøgle | Beskrivelse |
|---|---|---|
| Konto oprettet | `account.create` | En ny konto er blevet oprettet. |
| Konto under gennemgang | `account.under_review` | Kontoen afventer gennemgang. |
| Konto opdateret | `account.updated` | Kontodetaljer er blevet ændret. |
| Konto status opdateret | `account.status.updated` | Kontostatus er ændret (f.eks. godkendt, suspenderet). |
| Kontoadresse opdateret | `account.address.updated` | Kontoens adresse er blevet ændret. |
| Kontodetaljer opdateret | `account.details.updated` | Kontodetaljefelter er ændret. |
| Kontooplysninger opdateret | `account.information.update` | Generelle konto oplysninger er blevet opdateret. |
| Kontoadministratorer opdateret | `account.managers.update` | Listen over kontoadministratorer er ændret. |
| Profilbillede opdateret | `account.profile_picture.updated` | Kontoens profilbillede er blevet ændret. |
| Agent opdateret | `account.agent.updated` | Agentoplysninger på kontoen er blevet ændret. |
| URL-navn opdateret | `account.url-name.update` | Kontoens vanity-URL er ændret. |
| Unikt ID opdateret | `account.unique-id.update` | Kontoens unikke identifikator er blevet ændret. |

| Begivenhed | Nøgle | Beskrivelse |

| Opgave fjernet | `account.task.remove` | En konto-opgave er blevet fjernet. |

| Skatte-ID-opgave fuldført | `account.task.tax_id.complete` | Skatte-ID-verifikationsopgave er blevet fuldført. |

### Kontoindløser
