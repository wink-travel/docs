## Payment API

Payment endpoints er til udviklere, der ønsker at købe rejseinventar. Dette kan gøres via API'et som registreret rejseagent eller ved at bruge vores API i kombination med vores PCI-kompatible betalingswidget for alle andre enheder.
Payment API'et eksponerer endpoints for affiliates og hoteller til at spore bookinger, analyser og tilgængelighed af midler. De kan også vælge at hæve tilgængelige midler til deres bankkonto.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Lær mere
</LinkButton>
