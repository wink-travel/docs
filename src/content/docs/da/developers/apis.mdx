## Betalings-API

Betalingsslutpunkter er for udviklere, der ønsker at købe rejseinventar. Dette kan gøres via API'en som registreret rejsebureau eller ved hjælp af vores API sammen med vores PCI-kompatible betalingswidget for alle andre enheder.
Betalings-API'en eksponerer slutpunkter for affilierede selskaber og hoteller, så de kan spore bookinger, analyser og tilgængelighed af midler. De kan også vælge at hæve tilgængelige midler til deres bankkonto.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Lær mere
</LinkButton>
