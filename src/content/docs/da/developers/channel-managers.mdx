Wink integrerer med førende channel managers, property management systems (PMS) og central reservationssystemer (CRS) for at holde dine priser og tilgængelighed synkroniseret på tværs af alle dine salgskanaler.

## Sådan fungerer det

Channel manager-integrationer muliggør en tovejskommunikation mellem dit property management system og Wink:

1. **Prisopdatering** — Din channel manager sender prisopdateringer til Wink. Når du opdaterer dine priser i dit PMS, afspejles disse ændringer automatisk på Wink.
2. **Synkronisering af tilgængelighed** — Din channel manager sender opdateringer om tilgængelighed til Wink. Når værelser bookes eller frigives i dit PMS, opdateres Winks lager i realtid.
3. **Bookingnotifikationer** — Når en booking foretages på Wink, bliver din channel manager underrettet, så reservationen vises i dit PMS.

## Understøttede channel managers

Wink integrerer i øjeblikket med følgende systemer:

| Channel Manager | Beskrivelse |
|---|---|
| **Channex** | Global distributions- og channel management-platform. |
| **TravelClick** | Hotelløsninger og channel management (Amadeus). |
| **Omnibees** | Distribution og revenue management for hoteller. |
| **D-Edge** | Hospitality-teknologi og central reservationsløsninger. |
| **Allotz** | Lagerallokering og channel management. |
| **Ezee / Yanolja** | Cloud PMS og channel management med fokus på det asiatiske marked. |
| **RateGain** | Prisintelligens og distributionsteknologi. |
| **RateTiger** | Distribution af priser og tilgængelighed i realtid. |
| **CloudBeds** | Cloud-baseret PMS, channel manager og bookingmotor. |
| **SiteMinder** | Ledende hotel channel manager og distributionsplatform. |
| **Exely** | Synkronisering af priser og tilgængelighed for hoteller. |

:::note
Hvis din channel manager ikke er på listen, kontakt Wink-teamet. Nye integrationer kan tilføjes baseret på OTA (Open Travel Alliance) protokolstandarder, som Wink understøtter.
:::

## For udviklere

Hvis du bygger en channel manager- eller PMS-integration med Wink, brug Channel Manager API-endpoints:

- **Prisopdateringer** — Push prisændringer til Wink.
- **Tilgængelighedsopdateringer** — Push ændringer i tilgængelighed til Wink.
- **Bookinghentning** — Hent bookinger fra Wink ind i dit system.

Channel Manager API følger OTA-protokolstandarder (SOAP/XML) for kompatibilitet med eksisterende hospitality-systemer.

## Webhook-notifikationer

Du kan abonnere på channel manager webhook-events for at modtage notifikationer i realtid:

- `channel-manager.update.rate` — Prisopdatering modtaget.
- `channel-manager.update.availability` — Tilgængelighedsopdatering modtaget.
- `channel-manager.update` — Generel channel manager-opdatering.

Se [Webhook Events Catalog](/da/developers/webhook-events#channel-manager-events) for detaljer.
