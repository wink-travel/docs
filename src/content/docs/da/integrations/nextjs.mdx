Wink-komponenter er browser-only custom elements — CDN scripts skal indlæses på klientsiden. Mønstrene nedenfor dækker både **App Router** (Next.js 13+) og den ældre **Pages Router**.

<LinkCard
  title="@wink/elements npm package"

  description="Installer og indlæs CDN-bundtet med TypeScript-typer."
/>

## Installér

```bash
npm install @wink/elements
```

## App Router (Next.js 13+)

Opret en client component, der kalder `load()` én gang, og render den inde i dit root layout:

```tsx
// components/WinkLoader.tsx
'use client';
import { useEffect } from 'react';
import { load } from '@wink/elements';

export function WinkLoader() {
  useEffect(() => {
    load({ clientId: process.env.NEXT_PUBLIC_WINK_CLIENT_ID! });
  }, []);
  return null;
}
```

```tsx
// app/layout.tsx
import { WinkLoader } from '@/components/WinkLoader';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <WinkLoader />
        {children}
      </body>
    </html>
  );
}
```

Brug Wink-komponenter i enhver **Client Component**:

```tsx
// app/hotels/page.tsx
'use client';

export default function HotelsPage() {
  return (
    <main>
      <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
    </main>
  );
}
```

<Aside type="caution" title="Server Components">
`<wink-*>` tags kan ikke rendres i Server Components, fordi custom element-registreringen kun findes i browseren. Tilføj altid `'use client'` til enhver komponent, der renderer Wink-elementer.
</Aside>

## Pages Router

Indlæs én gang i `_app.tsx`:

```tsx
// pages/_app.tsx
import type { AppProps } from 'next/app';
import { useEffect } from 'react';
import { load } from '@wink/elements';

export default function App({ Component, pageProps }: AppProps) {
  useEffect(() => {
    load({ clientId: process.env.NEXT_PUBLIC_WINK_CLIENT_ID! });
  }, []);
  return <Component {...pageProps} />;
}
```

Brug derefter Wink-komponenter på enhver side:

```tsx
// pages/hotels.tsx
export default function HotelsPage() {
  return <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />;
}
```

## Miljøvariabler

Tilføj til `.env.local`:

```
NEXT_PUBLIC_WINK_CLIENT_ID=your-client-id
```

`NEXT_PUBLIC_` præfikset er nødvendigt for at eksponere variablen til browser-bundtet.
