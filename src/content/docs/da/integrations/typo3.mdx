**Wink TYPO3 Extension** integrerer Wink rejseinventar i TYPO3 13 som native indholdselementer. Redaktører tilføjer Wink-komponenter til sider ved hjælp af den standard TYPO3 sideeditor — ingen skabelonændringer er nødvendige.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"

  description="Kildekode, udvikleropsætningsvejledning og bidragsinstruktioner."
/>

## Krav

- TYPO3 13.x
- PHP 8.2+
- Composer

## Installation

<Steps>
1. Installer via Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivér udvidelsen:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Gå i TYPO3 backend til **Admin Tools → Settings → Extension Configuration → wink**.
4. Indtast din **Client ID**, **Client Secret**, og vælg dit miljø. Gem.
   Gå til [Applications](/da/portal/settings/applications) for at hente dine legitimationsoplysninger.
5. Tilføj den statiske TypoScript-skabelon til din rodlayout:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Tilføj et indholdselement til en side

<Steps>
1. Åbn en side i **Page**-modulet.
2. Klik på **+**-knappen i indholdsområdet for at tilføje et nyt element.
3. Vælg et af **Wink Travel** indholdselementerne:
   - **Wink Content** — hotelkort, rangerede gitter, kort
   - **Wink Lookup** — destinationssøgefelt
   - **Wink Search** — rejseplanvælgerknap
   - **Wink Account** — log ind-knap
   - **Wink Itinerary** — rejseplanstatusknap
   - **Wink Cart** — indkøbskurvknap
4. For **Wink Content**, brug FlexForm-felterne til at vælge en **Layout Type** og indtaste **Layout ID**.
5. Gem og publicer siden.
</Steps>

## Tilgængelige indholdselementer

| Element | Formål |
|---|---|
| **Wink Content** | Hotelkort, rangerede gitter, kort og mere |
| **Wink Lookup** | Destinationssøgefelt |
| **Wink Search** | Ikonknap, der åbner rejseplanvælgeren |
| **Wink Account** | Log ind / brugerprofilknap |
| **Wink Itinerary** | Rejseplan-knap, der viser den aktuelle tur |
| **Wink Cart** | Oversigtsknap for indkøbskurv |
