**Wink Drupal-modulet** tilføjer Wink rejseinventarkomponenter til dit Drupal-site som native blokke. Redaktører kan placere dem hvor som helst i layoutet via det standard bloksystem.

<LinkCard
  title="wink-travel/wink-drupal-module på GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Kildekode, udvikleropsætningsvejledning og bidragsinstruktioner."
/>

## Krav

- Drupal 10 eller 11
- PHP 8.1+
- Composer

## Installation

<Steps>
1. Installer via Composer:
   ```bash
   composer require drupal/wink
   ```
2. Aktiver modulet:
   ```bash
   drush en wink -y
   ```
3. Gå til **Admin → Konfiguration → Web Services → Wink-indstillinger** (`/admin/config/wink`).
4. Indtast din **Client ID** og **Client Secret**, vælg dit miljø, og klik på **Gem**.
   Gå til [Applications](/da/portal/settings/applications) for at hente dine legitimationsoplysninger.
</Steps>

## Tilføj en blok til en side

<Steps>
1. Gå til **Admin → Struktur → Bloklayout**.
2. Klik på **Placer blok** i den region, hvor du vil have Wink-indhold.
3. Søg efter **Wink** for at finde de tilgængelige blokke:
   - **Wink Content** — viser et hotelkort, gitter eller kort
   - **Wink Lookup** — destinationssøgefelt
   - **Wink Search** — knap til valg af rejseplan
   - **Wink Account** — log ind-knap
   - **Wink Itinerary** — knap til rejseplanstatus
   - **Wink Cart** — indkøbskurvsknap
4. For **Wink Content**, konfigurer **Layout Type** og **Layout ID** i blokindstillingerne.
5. Gem blokkonfigurationen.
</Steps>

## Tilgængelige blokke

| Blok | Formål |
|---|---|
| **Wink Content** | Viser hotelkort, rangerede gitter, kort og mere |
| **Wink Lookup** | Søgefelt til destinationer og hoteller |
| **Wink Search** | Ikonknap, der åbner rejseplanvælgeren |
| **Wink Account** | Log ind / bruger konto-knap |
| **Wink Itinerary** | Rejseplanknap, der viser den aktuelle tur |
| **Wink Cart** | Oversigtsknap for indkøbskurv |
