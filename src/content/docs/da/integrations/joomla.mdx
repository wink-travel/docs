**Wink Joomla Extension** er en pakke, der installerer et systemplugin (indlæser CDN-scripts), et indholdsplugin (behandler shortcode-tags i artikler) og en komponent (admin-indstillingsside).

<LinkCard
  title="wink-travel/wink-joomla-extension on GitHub"
  href="https://github.com/wink-travel/wink-joomla-extension"
  description="Kildekode, udvikleropsætningsvejledning og bidragsinstruktioner."
/>

## Krav

- Joomla 4 eller 5
- PHP 8.1+

## Installation

<Steps>
1. Download den nyeste `pkg_wink-x.x.x.zip` fra [GitHub releases-siden](https://github.com/wink-travel/wink-joomla-extension/releases).
2. Gå i Joomla Admin til **Extensions → Manage → Install**.
3. Upload zip-filen og klik på **Upload & Install**.
4. Begge plugins aktiveres automatisk. Gå til **Components → Wink Settings**.
5. Indtast din **Client ID** og **Client Secret** og klik på **Save**.
   Gå til [Applications](/da/portal/settings/applications) for at hente dine legitimationsoplysninger.
</Steps>

## Indsæt komponenter i artikler

Brug shortcode-lignende tags hvor som helst i artikelindholdet:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Indholdspluginet erstatter disse tags med korrekt `<wink-*>` HTML ved gengivelsestidspunktet. Systempluginet indlæser automatisk CDN-scripts og indsætter `<wink-app-loader>` i sidefoden.

## Tilgængelige shortcodes

| Shortcode | Output | Beskrivelse |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Hotelkort, gitter, kort |
| `{wink-lookup}` | `<wink-lookup>` | Destinationssøgefelt |
| `{wink-search}` | `<wink-search-button>` | Rejseplanvælger-knap |
| `{wink-account}` | `<wink-account-button>` | Log ind / brugerprofil |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Rejseplansstatus-knap |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Indkøbskurv-knap |
