Svelte sender attributter direkte til DOM uden et virtuelt DOM-lag, hvilket gør brugerdefinerede elementer til en naturlig løsning. For SvelteKit (SSR) skal du indlæse CDN’en inde i `onMount` eller beskytte med `browser`-tjekket.

<LinkCard
  title="@wink/elements npm package"
  href="/integrations/npm"
  description="Installer og indlæs CDN-pakken med TypeScript-typer."
/>

## Installér

```bash
npm install @wink/elements
```

## SvelteKit — root layout

```ts
// src/routes/+layout.ts
import { browser } from '$app/environment';
import { load as loadWink } from '@wink/elements';

export function load() {
  if (browser) {
    loadWink({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  }
}
```

Alternativt, indlæs inde i `onMount` i `+layout.svelte`:

```svelte
<!-- src/routes/+layout.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<slot />
```

## Brug i komponenter

Svelte sender alle ukendte attributter direkte til DOM, så der kræves ingen ekstra konfiguration:

```svelte
<script>
  export let layoutId = 'YOUR_LAYOUT_ID';
</script>

<main>
  <wink-content-loader layout="HOTEL" id={layoutId} />
  <wink-lookup />
</main>
```

## Svelte 5 (runesyntaks)

```svelte
<script lang="ts">
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  let { layoutId }: { layoutId: string } = $props();

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id={layoutId} />
```

## Miljøvariabler

SvelteKit eksponerer kun variabler med præfikset `PUBLIC_` til browseren:

```
# .env
PUBLIC_WINK_CLIENT_ID=your-client-id
```

<Aside type="caution" title="SSR rendering">
Render aldrig `<wink-*>` elementer under server-side rendering — brugerdefinerede elementer registreres i browserens `customElements`-register, som ikke findes på serveren. Pak ind i `{#if browser}` eller kald `load()` kun inde i `onMount`.
</Aside>

```svelte
<script>
  import { browser } from '$app/environment';
</script>

{#if browser}
  <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
{/if}
```

## Ren Svelte (Vite, uden SvelteKit)

```svelte
<!-- src/App.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.VITE_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
```
