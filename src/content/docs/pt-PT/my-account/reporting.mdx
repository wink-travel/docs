O Wink disponibiliza um sistema de relatórios que lhe permite gerar e exportar dados da sua conta. Os relatórios estão disponíveis em formato CSV para fácil importação em folhas de cálculo, ferramentas de análise ou nos seus próprios sistemas.

## Tipos de relatório

Pode gerar relatórios para as seguintes categorias de dados:

| Tipo de relatório | O que inclui |
|---|---|
| **Propriedade** | Detalhes da propriedade, estado do anúncio, resumo do inventário. |
| **Utilizador** | Contas de utilizador, funções, resumo de atividade. |
| **Reserva** | Histórico de reservas, estados, receitas, detalhes dos hóspedes. |
| **Conta** | Informação da conta, saldos, histórico de transações. |

## Como gerar um relatório

1. Navegue até à secção de relatórios no painel da sua conta.
2. Selecione o **tipo de relatório** que pretende gerar.
3. Configure quaisquer **filtros** para restringir os dados (ex.: intervalo de datas, estado, propriedade).
4. Escolha os **campos** que quer incluir na exportação. Cada tipo de relatório oferece um conjunto de campos disponíveis para selecionar.
5. Clique em **Exportar** para descarregar o relatório em ficheiro CSV.

## Campos personalizados

Os relatórios suportam a seleção personalizada de campos. Pode escolher quais as colunas a incluir na sua exportação, para obter exatamente os dados que precisa — nem mais, nem menos.

## Casos de uso

- **Reconciliação de receitas** — Exporte relatórios de reservas com detalhes financeiros para conciliar com o seu sistema contabilístico.
- **Análise de desempenho** — Exporte relatórios de propriedades para comparar o desempenho dos anúncios na sua carteira.
- **Insights sobre hóspedes** — Exporte dados de reservas para compreender a demografia dos viajantes e padrões de reserva.
- **Auditoria de contas** — Exporte dados de contas para conformidade ou revisão interna.
