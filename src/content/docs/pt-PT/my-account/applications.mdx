As aplicações dão-lhe acesso a todas as funcionalidades do Wink como programador. Pode aceder a todos os mesmos dados e até estender as funcionalidades do próprio Wink.
Pode optar por usar o servidor OAuth2 do Wink para autenticar o seu site de viagens da mesma forma que usaria o Google ou o Facebook.
Desta forma, pode estender as funcionalidades do Wink para o seu próprio website ou aplicação.

:::note
A sua primeira app já está criada quando [cria uma nova conta](/pt-PT/my-account/create-account). Usamos essa app para alguns cenários de reserva.

Não pode reutilizar esta app. Por favor, crie novas apps conforme precisar.
:::

:::note
Uma aplicação gerada no Wink pode ser usada para todos os endpoints da API.
:::

<Steps>
 1. Selecione a conta com a qual quer trabalhar em [Contas](https://my.wink.travel).
 2. Clique em `Ações` na parte inferior do cartão da conta e escolha `Gerir conta`.
 3. Clique no separador `Aplicações` na página seguinte.
 4. Esta secção listará todas as suas aplicações.
 5. Escolha editar uma existente ou criar uma nova.
 </Steps>

## Criar app

Como não pode aceder à `chave secreta` da app que criámos para si, terá de criar a sua própria.

:::caution
Precisa de criar uma conta afiliada no Wink ou uma conta no TripPay antes de criar uma app, pois irá associar a sua aplicação a uma conta específica.
:::

Aqui estão os passos para criar uma aplicação:

<Steps>
1. Na página das apps, clique no botão `Criar nova aplicação`.
2. ***Nome*** Dê um nome à sua app. *ex. App Fixe*
3. **Nome da entidade** Associe a sua app a uma das suas contas existentes. *ex. Conta Fixe*
4. **URIs de redirecionamento** É necessário pelo menos um URI de redirecionamento. É usado para validar o domínio de redirecionamento após uma autenticação bem-sucedida. *ex. https://www.site-fixe.com*
5. Clique no botão `Guardar` para continuar.
</Steps>

Será redirecionado de volta para a sua lista de aplicações. Verá a sua nova app na lista e terá acesso à chave secreta da app apenas desta vez. Certifique-se de copiá-la e guardá-la num local seguro.

## Atualizar app

Terá de atualizar a sua app se algum dos seus URIs de redirecionamento mudar ou se quiser adicionar ou remover um URI.

Para atualizar uma app existente, faça o seguinte:

<Steps>
1. Clique no link `Ações` da app que quer atualizar.
2. Clique no botão `Atualizar` em Ações.
3. Faça as alterações na app.
4. Clique no botão `Guardar` para continuar.
</Steps>

Será redirecionado de volta para a sua lista de aplicações.

## Remover app

:::caution
Por favor, não elimine a app que criámos automaticamente para si para garantir que todas as funcionalidades funcionem para a sua conta.
:::

Se não precisar mais de uma app, pode removê-la.

Para remover uma app, faça o seguinte:

<Steps>
1. Clique no link `Ações` da app que quer remover.
2. Clique no botão `Remover` em Ações.
3. Clique no botão `OK` para confirmar a eliminação.
</Steps>

Será redirecionado de volta para a sua lista de aplicações.
