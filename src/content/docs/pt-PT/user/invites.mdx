Um utilizador do Wink pode optar por deixá-lo gerir uma das suas contas de propriedade, afiliado ou agente de viagens. Quando isso acontecer, receberá um e-mail com um pedido de convite.

import { Steps } from '@astrojs/starlight/components';

Para ver os seus convites atuais do Wink, siga os seguintes passos:

<Steps>
  1. Deve estar logado num de nossos sites.
  2. Clique no ícone do seu perfil, no canto superior direito, e irá aparecer um menu.
  3. Clique no `Invites` ligação.
  4. Interaja com os seus convites através de `Accepting` ou `Rejecting` eles.
</Steps>
