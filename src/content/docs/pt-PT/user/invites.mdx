Um utilizador Wink pode escolher permitir que você gere uma das suas contas de propriedade, afiliado ou agente de viagens. Quando isso acontecer, receberá um e-mail com um pedido de convite.

import { Steps } from '@astrojs/starlight/components';

Para ver os seus convites Wink atuais, siga estes passos:

<Steps>
1. Deve estar autenticado num dos nossos websites.
2. Clique no seu ícone de perfil, no canto superior direito, e aparecerá um menu.
3. Clique no link `Convites`.
4. Interaja com os seus convites, `Aceitando` ou `Rejeitando`-os.
</Steps>
