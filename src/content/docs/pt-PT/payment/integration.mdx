Se é uma empresa de viagens e quer terceirizar a parte de pagamento e desembolso do seu negócio para o TripPay, aqui estão 6 passos básicos para começar:

<Steps>
1. Crie uma [conta de utilizador](/pt-PT/user/register) na Wink.
2. Crie uma conta com o [TripPay](https://pay.wink.travel).
3. Crie uma [Aplicação](/pt-PT/developers/applications) no TripPay que lhe permita comunicar com a nossa API.
4. Crie e associe os seus beneficiários com o TripPay. *por exemplo, para a maioria, isso significa criar contas para as propriedades na sua plataforma que quer que o TripPay pague.*
5. Integre o [componente web de pagamento](/pt-PT/developers/web-components/#payment) no seu site.
6. Envie ao TripPay o `carrinho de compras` do viajante e inicie o processo de pagamento.
</Steps>

:::tip
Para um guia completo sobre como usar o TripPay no seu site de viagens, leia o guia [Integrando com o TripPay](/pt-PT/guides/integrating-with-trip-pay).
:::

## API

Desenvolvedores que querem gerir `Pagamentos`, acedam a [Desenvolvedores > API > Pagamentos](/pt-PT/developers/apis/#payment-api).
