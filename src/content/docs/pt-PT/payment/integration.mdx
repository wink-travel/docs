Se é uma empresa de viagens e pretende subcontratar a parte de pagamento e desembolso do seu negócio para o TripPay, aqui estão 6 passos básicos para começar:

<Steps>
  1. Criar um [conta de utilizador](/user/register) no Wink.
  2. Crie uma conta com [TripPay](https://pay.wink.travel).
  3. Criar um [Aplicação](/developers/applications) no TripPay que lhe permite conversar com a nossa API.
  4. Crie e mapeie os seus beneficiários com o TripPay. *por exemplo Para a maioria, isto significa criar contas para as propriedades na sua plataforma que pretende que o TripPay pague.*
  5. Integrar o [componente web de pagamento](/developers/web-components/#payment) no seu site.
  6. Enviar ao TripPay o viajante `shopping cart` e iniciar o processo de pagamento.
</Steps>


:::dica
Para um guia completo sobre como utilizar o TripPay no seu site de viagens, leia o guia [Integração com o TripPay](/guides/integrating-with-trip-pay).
:::
