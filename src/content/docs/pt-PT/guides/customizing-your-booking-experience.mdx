As personalizações controlam como o seu inventário é apresentado aos viajantes — as cores do tema, moeda predefinida, idioma, itinerário e mais. Aplicam-se a links partilháveis, cartões incorporados, grelhas, mapas e ao motor de reservas.

## O que é uma personalização?

Uma personalização é um perfil de configuração que define:

- **Moeda de exibição** — Em que moeda os preços são apresentados.
- **Idioma de exibição** — O idioma predefinido para o conteúdo.
- **Itinerário predefinido** — Datas de check-in/check-out e número de hóspedes pré-preenchidos.
- **Tema** — Cores e branding aplicados aos componentes incorporados.
- **Configuração** — Como o inventário é apresentado (layout, ordenação).

Pode criar várias personalizações para diferentes mercados ou casos de uso.

## Criar uma personalização

<Steps>
1. Aceda a https://studio.wink.travel (ou à sua conta em https://my.wink.travel).
2. Navegue até `Customization` no menu lateral.
3. Clique em `Create a new customization`.
4. Dê-lhe um nome descritivo (ex.: "Europe - EUR", "Japan Market", "Default").
5. Defina a moeda de exibição, idioma e itinerário predefinido.
6. Configure as cores do tema para corresponder ao seu branding.
7. Clique em `Save`.
</Steps>

## Onde as personalizações são usadas

Depois de criadas, as personalizações são referenciadas em toda a plataforma:

| Funcionalidade | Como a personalização se aplica |
| -- | -- |
| **Links partilháveis** | O link usa a moeda, idioma e itinerário da personalização como predefinições |
| **Cartões incorporados** | Os cartões são renderizados com as cores do seu tema e definições de exibição |
| **Grelhas e mapas** | As grelhas de inventário e marcadores de mapa seguem a personalização |
| **Motor de reservas** | Todo o fluxo de reservas utiliza as definições da personalização |
| **Publicações sociais** | Publicações geradas por IA podem usar uma personalização específica para exibição de preços |

## Múltiplas personalizações

Um padrão comum é criar uma personalização por mercado-alvo:

- **"US Market"** — USD, inglês, estadia predefinida de 2 noites.
- **"Europe"** — EUR, inglês, estadia predefinida de 3 noites.
- **"Japan"** — JPY, japonês, estadia predefinida de 1 noite.

Quando cria um link partilhável ou incorpora um cartão, seleciona qual personalização usar. Isto significa que a mesma propriedade pode ser apresentada de forma diferente consoante o público.

## Substituir via URL

Também pode substituir as predefinições da personalização usando parâmetros URL em links partilháveis:

- `l=es` — Substitui o idioma para espanhol.
- `c=EUR` — Substitui a moeda para EUR.
- `sd=2024-08-24&n=2` — Substitui o itinerário para uma data e número de noites específicos.

Consulte [The Book Now Button](/pt-PT/guides/the-book-now-button) para a lista completa de parâmetros URL.
