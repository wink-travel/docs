O botão **RESERVAR AGORA** é um termo ubíquo na indústria hoteleira que se refere à integração própria do hotel com um fornecedor que conhece a disponibilidade e os preços da propriedade e que permite aos viajantes reservar um quarto. O botão é a forma mais simples para os hotéis venderem quartos no seu próprio site e nos seus próprios termos.

*Muitos hotéis utilizam o motor de reservas incluído que é oferecido pelo seu gestor de canais. Recomendamos que os hotéis façam alguma pesquisa e comparem motores de reservas nativos antes de se comprometerem com algum em particular.*

## Público-alvo

Este artigo destina-se principalmente a hotéis que querem instalar o nosso motor de reservas gratuito e de alta conversão no seu website.  
Mas... também é uma leitura útil para qualquer pessoa que queira usar as nossas tecnologias nos seus websites.

## via Link

Para a integração mais simples, recomendamos usar um [Link Partilhável](/pt-PT/studio/shareable-links).

Aqui estão os passos para fazer isto:

<Steps>
1. Vá a https://extranet.wink.travel  
2. Clique em `Sign-In / Register` no canto superior direito e inicie sessão com a sua conta de utilizador.  
3. Selecione a propriedade com a qual quer trabalhar no menu suspenso no canto superior esquerdo.  
4. Clique em `Booking engine` na barra de navegação.  
5. A página mostrará o seu link.  
6. Dê este link ao seu webmaster e use-o para o seu botão **RESERVAR AGORA**.  
7. Está ✅.  
</Steps>

### Avançado

O link já vem com um itinerário incorporado na URL, que se origina da sua `customização` escolhida.  
Pode personalizar ainda mais o seu link ao:

- Alterar a customização selecionada OU  
- Substituir o itinerário

Neste artigo, vamos focar no último, pois dá-lhe mais liberdade para permitir que os viajantes selecionem um itinerário no seu website e o encaminhem para o motor de reservas do Wink.

Aqui está como passar o seu próprio itinerário juntamente com a URL.

Parâmetros de pedido válidos:

- **sd** Para adicionar uma data de início: *sd=2024-08-24*  
- **n** Para adicionar número de noites: *n=1* OU  
- **ed** Usar data de fim: *ed=2024-08-25*  
- **rc** Hóspedes: *rc=a2* (Adultos: 2)  
- **l** Pedir uma língua de exibição: *l=es*  
- **c** Pedir uma moeda de exibição: *c=EUR*  
- **p** Adicionar uma promoção: *p=ABC123*

A URL ficaria assim:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2&l=es&c=EUR

Para casos de uso mais avançados, pode também pedir múltiplas configurações de quartos usando o sinal `+`:

Exemplo:

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2-ca16q1-ca8q1+a2&l=es&c=EUR

- Tipo de quarto 1 = Adultos: 2, Crianças: 2 = 16 anos + 8 anos de idade.  
- Tipo de quarto 2 = Adultos: 2.

### Exemplo

Exemplo de HTML para botão **RESERVAR AGORA**:

*Este exemplo assume que está a usar [Bootstrap](https://getbootstrap.com) como a sua biblioteca CSS principal.*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Reservar um quarto"
>
RESERVAR AGORA
</a>
```

### Outros benefícios

Existem outros benefícios em usar um dos nossos links como o seu botão `reservar agora`.

- **Amigável para redes sociais** O link pode ser facilmente partilhado nas redes sociais e mostra `preços dinâmicos` sobre a imagem que selecionou.  
- **Curto** A URL é muito fácil de partilhar e pode usá-la em locais como o URL da sua Bio no Instagram.

## via Embed

Alguns hotéis querem um pouco mais de funcionalidade e uma integração mais profunda para o seu website. Este exemplo cobre a incorporação de um [Card](/pt-PT/studio/cards) no seu website.

Aqui estão os passos para fazer isto:

<Steps>
1. Vá a https://studio.wink.travel  
2. Clique em `Sign-In / Register` no canto superior direito e inicie sessão com a sua conta de utilizador.  
3. Selecione a propriedade com a qual quer trabalhar no canto superior esquerdo.  
4. Clique em `Search` na barra de navegação principal.  
6. Escreva o nome da sua propriedade e clique no botão de pesquisa.  
7. Clique no link `Actions` no cartão do resultado da pesquisa e selecione `Create > Make a Card`.  
8. Isto irá criar um `Card`.  
9. Aparecerá uma mensagem no canto superior direito a informar que o seu card está pronto e se quer personalizá-lo mais.  
10. Clique em `Tools > Cards` na barra de navegação principal e verá o seu novo card.  
11. Clique no link `Actions` no seu card e clique em `Embed`.  
12. Isto abre uma nova janela com o código HTML de exemplo que pode copiar.  
13. Para obter o seu `Client-ID`, vá a [Applications](/pt-PT/developers/applications).  
14. Para obter o seu `Configuration-ID`, vá a [Customization](/pt-PT/studio/customization)  
14. Está ✅.  
</Steps>

Exemplo de HTML para card incorporado:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*O exemplo de HTML acima não inclui o CSS e Javascript necessários do Wink; apenas o card.*
