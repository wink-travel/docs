O motor de reservas vem em duas variedades para apoiar as diferentes necessidades de um afiliado, uma empresa de gestão de propriedades e um hotel.

## Agência de viagens online

Para um afiliado, faz mais sentido utilizar a versão de agência de viagens online. A razão é que o site foi criado para acomodar viajantes que desejam visualizar diversas propriedades.

Pode encontrar a página inicial deste aplicativo abaixo.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>


## Motor de reservas pela internet

Para um hotel ou uma empresa de gestão de propriedades, faz mais sentido utilizar a versão do motor de reservas através da internet. Este tipo de utilizador está interessado em destacar uma única propriedade ou várias propriedades sob gestão.

## Configuração

Na maioria das vezes, não há necessidade de partilhar nenhum destes aplicativos diretamente. Criámos uma variedade de formas para que possa partilhar o inventário de viagens, que pode descobrir em [Estúdio Wink](/studio/what-is-studio) ou através [WinkLinks](/link-manager/wink-links).

No entanto, se quiser criar um link manual para um destes sites, certifique-se de que adiciona o seu ID de cliente e o ID de personalização ao URL como parâmetros.

Pode encontrar o seu ID de cliente por:

<Steps>
  1. Faça login em qualquer um dos [Extranet Wink](/extranet/what-is-extranet) ou [Estúdio Wink](/studio/what-is-studio).
  2. Clique no ícone do seu perfil no canto superior direito e clique em [Aplicações](/developers/applications).
  3. Utilize a aplicação ligada à conta Wink com a qual deseja trabalhar.
  4. Copie o ID do cliente dessa aplicação e cole-o no URL como parâmetro de pedido. Exemplo: ?client-id=MY\_CLIENT\_ID
</Steps>


### Extranet

Pode encontrar o seu ID de personalização como uma propriedade por:

<Steps>
  1. Entrar [Extranet Wink](/extranet/what-is-extranet).
  2. Selecione a propriedade com a qual pretende trabalhar.
  3. Na barra de navegação principal, clique em `Account > Branding`
  4. Copie o ID de personalização e cole-o no URL como parâmetro de pedido. Exemplo: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Estúdio

Pode encontrar o seu ID de personalização como afiliado por:

<Steps>
  1. Entrar [Estúdio Wink](/studio/what-is-studio).
  2. Selecione a conta com a qual pretende trabalhar.
  3. Na barra de navegação principal, clique em `Tools > Customization`
  4. Copie o ID de personalização da personalização que pretende utilizar e cole-o no URL como parâmetro de pedido. Exemplo: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

O resultado é um URL semelhante a este:

https://ota.wink.travel?client-id=MEU\_ID\_DO\_CLIENTE\&configuration-id=MEU\_ID\_DE\_PERSONALIZAÇÃO

Se quisesse ir diretamente para a página de um hotel, o URL seria algo como isto:

https://ota.wink.travel/hotel/hotel-x?client-id=MEU\_ID\_DO\_CLIENTE\&configuration-id=MEU\_ID\_DE\_PERSONALIZAÇÃO

:::observação
Se não incluir um ID de cliente e/ou uma configuração, o Wink OTA utiliza uma configuração predefinida.
:::

## IBE

Se é uma empresa de gestão imobiliária, quer criar um link para a página inicial do Wink IBE. A página inicial mostrará o tema designado juntamente com o [propriedades que representa](/studio/agency-representation).

O resultado é um URL semelhante a este:

https://book.wink.travel?client-id=MEU\_ID\_DO\_CLIENTE\&configuration-id=MEU\_ID\_DE\_PERSONALIZAÇÃO

Se for proprietário de um imóvel, pretende criar um link direto para a página inicial do imóvel Wink IBE.

Esse URL seria algo como isto:

https://book.wink.travel/hotel/hotel-x?client-id=MEU\_ID\_DO\_CLIENTE\&configuration-id=MEU\_ID\_DE\_PERSONALIZAÇÃO

:::observação
Se não incluir um ID de cliente e/ou configuração, o Wink IBE NÃO será carregado. Estes parâmetros são obrigatórios.
:::
