O Studio fornece vários widgets incorporáveis que pode adicionar ao seu próprio website para impulsionar reservas.

Para aceder aos widgets incorporados, navegue até `Tools > Miscellaneous` na barra lateral.

## Widgets disponíveis

| Widget | Descrição |
| -- | -- |
| **Account button** | Um botão com marca que liga à sua página de afiliado |
| **Itinerary button** | Um botão que abre o seletor de itinerário |
| **Itinerary form** | Um formulário inline para seleção de datas e hóspedes |
| **Search button** | Um botão que abre a pesquisa de propriedades |
| **Lookup button** | Um botão para consultar inventário específico |

## Como incorporar

Cada widget fornece um trecho de código que pode copiar e colar no HTML do seu website. O trecho inclui o seu **Client ID** e **Customization ID** para que o widget seja automaticamente ligado à sua conta de afiliado.

:::note
Se usar WordPress, também disponibilizamos um plugin para WordPress para uma integração mais fácil.
:::
