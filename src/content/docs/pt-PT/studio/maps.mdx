## Criar um mapa

Os mapas podem ser criados de duas formas:

* **Mapa de marcador único**: Crie um mapa a partir de algo que encontrou enquanto [procurando](/studio/search).
* **Mapa de marcadores múltiplos**: Crie um mapa a partir de um [lista com curadoria](/studio/curated-lists) ou [pesquisa salva](/studio/saved-searches).

Clique nesse botão e ele irá redirecioná-lo para a nossa página de formulário de mapa, onde poderá começar a personalizar o seu mapa.

### Mapa de marcadores múltiplos

Ambos [listas selecionadas](/studio/curated-lists#list-actions) e [pesquisas guardadas](/studio/saved-searches#list-actions) vem com uma ação que permite criar um mapa. Este mapa conterá marcadores de mapa para todo o inventário dentro destas listas.

## Personalizar mapa

O formulário permite personalizar o seu mapa das seguintes formas:

<Steps>
  1. Dê um nome para que se possa lembrar do que se trata o mapa.
  2. Escolha se pretende que o utilizador tenha permissão para se movimentar pelo mapa.
  3. Defina a altura, em pixéis, que pretende que o mapa tenha.
  4. Escolha se pretende que o utilizador tenha permissão para aumentar ou diminuir o zoom no mapa.
  5. Escolha um estilo de mapa que combine com o site no qual vai incorporar o mapa.
  6. Escolha uma cor de marcador.
  7. Selecione a face inicial do cartão que pretende que os utilizadores vejam primeiro. Por defeito, é a face nativa deste inventário.
  8. Pode adicionar os seus próprios marcadores, círculos, retângulos e polígonos para indicar outros locais no mapa que os seus utilizadores possam considerar interessantes.
  9. Clique no `Save` botão para continuar.
</Steps>


Depois de guardar o seu mapa, será redirecionado para a sua página de mapas e o seu mapa estará pronto para ser partilhado com o mundo.

Abordaremos algumas destas opções com mais detalhe abaixo.

### Incorporar

```html {3, 6-9, 11, 13-16}
<html>
  <head>

  </head>
  <body>

       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Veja como incorporar o seu mapa no seu site:

* A linha 3 mostra como incorporar os estilos Wink no seu website.
* As linhas 6 a 9 mostram como utilizar o [carregador de conteúdo intermitente](/developers/web-components/#content-loader) Componente Web e diga-lhe para procurar um banner de anúncio para o seu código.
* A linha 11 mostra como incorporar o nosso Javascript no seu site.
* A linha 13 mostra como incorporar o [carregador de aplicações wink](/developers/web-components/#application-loader) Componente Web e peça-lhe para procurar as suas preferências de configuração ao nível da página.
