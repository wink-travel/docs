## Criar um mapa

Os mapas podem ser criados de duas formas:

- **Mapa de marcador único**: Crie um mapa a partir de algo que encontrou enquanto [pesquisava](/pt-PT/studio/search).
- **Mapa de múltiplos marcadores**: Crie um mapa a partir de uma [lista selecionada](/pt-PT/studio/curated-lists) ou [pesquisa guardada](/pt-PT/studio/saved-searches).

Clique nesse botão e será redirecionado para a nossa página de formulário de mapas onde pode começar a personalizar o seu mapa.

### Mapa de múltiplos marcadores

Tanto as [listas selecionadas](/pt-PT/studio/curated-lists#list-actions) como as [pesquisas guardadas](/pt-PT/studio/saved-searches#list-actions) têm uma ação que lhe permite criar um mapa. Este mapa conterá marcadores para todo o inventário dentro dessas listas.

## Personalizar mapa

O formulário permite-lhe personalizar o seu mapa das seguintes formas:

<Steps>
 1. Dê-lhe um nome para se lembrar do que se trata o mapa.
 2. Escolha se quer permitir que o utilizador se mova pelo mapa.
 3. Defina a altura, em pixels, que quer que o mapa tenha.
 4. Escolha se quer permitir que o utilizador faça zoom dentro e fora do mapa.
 5. Escolha um estilo de mapa que combine com o site onde vai incorporar o mapa.
 6. Escolha uma cor para o marcador.
 7. Selecione a face inicial do cartão que quer que os utilizadores vejam primeiro. Por defeito, é a face nativa desse inventário.
 8. Pode adicionar os seus próprios marcadores, círculos, retângulos e polígonos para indicar outros locais no mapa que os seus utilizadores possam achar interessantes.
 9. Clique no botão `Guardar` para continuar.
</Steps>

Depois de guardar o seu mapa, será redirecionado para a sua página de mapas e o seu mapa estará agora pronto para ser partilhado com o mundo.

Cobrimos algumas destas opções, com mais detalhe, abaixo.

### Incorporar

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Aqui está como incorporar o seu mapa no seu site:

- A linha 3 mostra como incorporar os estilos Wink no seu site.
- As linhas 6 a 9 mostram como usar o [wink-content-loader](/pt-PT/developers/web-components/#content-loader) Web Component e indicar que deve buscar um banner publicitário para o seu código.
- A linha 11 mostra como incorporar o nosso Javascript no seu site.
- A linha 13 mostra como incorporar o [wink-app-loader](/pt-PT/developers/web-components/#application-loader) Web Component e indicar que deve buscar as suas preferências de configuração a nível de página.
