Links partilháveis são uma funcionalidade central do Wink e surgiram da necessidade de comunicar inventário e preços de forma dinâmica através da forma mais simples possível na Internet; nomeadamente o hiperligação.

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

Acima está um exemplo de um dos nossos links. É curto e portátil... até suficientemente curto para memorizar. Apesar de curto, este link contém uma quantidade **explosiva** de informação, como:

- O seu ID de App.
- O seu ID de Afiliado.
- A(s) imagem(ns) e texto a mostrar ao utilizador.
- O aspeto visual (*ou seja, tema, cores e mais*).
- O itinerário a usar para a nossa consulta de preços (*2 adultos, 1 noite*).
- A moeda para mostrar o preço.
- O idioma para mostrar o texto.

O resto deste artigo guia-o sobre como criar, personalizar e partilhar os seus links.

Clique nesse botão e será redirecionado para a página do formulário de link partilhável onde pode começar a personalizar o seu link.

## Personalizar link

 O formulário permite-lhe personalizar o seu link das seguintes formas:

<Steps>
 1. Dê-lhe um nome para se lembrar do que trata o link.
 2. Selecione a [Personalização](/studio/customization) que quer aplicar a este link.
 3. Adicione palavras-chave, separadas por vírgulas, que serão usadas por [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Opcionalmente, insira o seu identificador de conta X ou o ID da App do Facebook.
 5. Adicione títulos e descrições nos idiomas que quer que o utilizador veja.
 6. Selecione uma ou mais imagens que serão mostradas quando partilhar o link em plataformas como Facebook ou LinkedIn.
 7. Escolha um tema / sobreposição que adiciona um design de marca por cima da(s) imagem(ns) selecionada(s). *Nota: É assim que faz os preços aparecerem por cima da imagem.*
 8. Clique no botão `Guardar` para continuar.
 </Steps>

Acima vê uma imagem com todas as ações disponíveis para o seu link:

- **Pré-visualizar** Mostra-lhe uma visão rápida de como o link foi configurado e como irá parecer para o utilizador.
- **Atualizar** Atualiza a configuração do seu link.
- **Adicionar ao WinkLinks** Adiciona o link à sua conta WinkLinks.
- **Copiar** Copia o link para a sua área de transferência para que possa partilhá-lo facilmente onde quiser (*incluindo [WinkLinks](/link-manager/wink-links) *).
- **Partilhar** Abre uma nova janela com opções para partilhar em muitas redes sociais e plataformas de mensagens.
- **Código QR** Gera um código QR que pode partilhar em plataformas como o IG, que são mais focadas em imagens.
- **Incorporar** Mostra-lhe como incorporar este link como um [Banner Publicitário](/developers/web-components/#content-loader) no seu website.
- **Usar com WordPress** Mostra-lhe como usar o nosso [plugin WordPress](/developers/wordpress) para incorporar este link no seu website.

Se preferir partilhar um código QR em vez de um link, clique no botão `Código QR` e aparecerá uma nova janela com o seu código QR.

Como guardar o seu código QR:

- Se estiver no seu portátil, clique com o botão direito na imagem e selecione `Guardar imagem`.
- Se estiver no seu telemóvel:
    - Pressione e mantenha o dedo sobre a imagem até aparecer uma opção para guardar a imagem nas suas Fotos.
    - Tire um screenshot do código QR, recorte para ajustar e guarde nas Fotos.

### Incorporar

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Aqui está como incorporar o seu link, como um banner publicitário, no seu site:

- A linha 3 mostra como incorporar os estilos Wink no seu site.
- As linhas 6 a 9 mostram como usar o [wink-content-loader](/developers/web-components/#content-loader) Web Component e indicar para buscar um banner publicitário para o seu código.
- A linha 11 mostra como incorporar o nosso Javascript no seu site.
- A linha 13 mostra como incorporar o [wink-app-loader](/developers/web-components/#application-loader) Web Component e indicar para buscar as suas preferências de configuração a nível de página.
