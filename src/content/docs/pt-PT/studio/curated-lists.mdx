Listas curadas é uma funcionalidade que lhe permite agrupar e ordenar inventário que selecionou a partir de itens encontrados enquanto procurava inventário de viagens.
Uma vez criada, pode disponibilizar a lista aos seus utilizadores tornando-a disponível para venda como uma [grelha](/pt-PT/studio/grids) ou um [mapa](/pt-PT/studio/maps) através do [WinkLinks](/pt-PT/link-manager/wink-links), o nosso [plugin WordPress](/pt-PT/developers/wordpress/) ou incorporando um dos nossos [Web Components](/pt-PT/developers/web-components) no seu site.

Por defeito, todas as contas afiliadas começam com uma lista `Favoritos` que pode usar imediatamente. `Favoritos` é especial e não pode ser removida.
Todas as listas curadas permitem-lhe fazer o seguinte:

## Criar uma lista

- No painel, clique em `Inventário > Listas Curadas` na barra de navegação.
- Crie uma nova lista curada clicando no botão `Criar nova lista`.
- Será solicitado que nomeie a sua lista. *Exemplo: Top 3 Escolhas em NYC*.
- Clique no botão `Guardar`.

## Adicionar um item à lista

Pode adicionar um item a qualquer lista a partir dos resultados da pesquisa.

- Vá a [Pesquisa](/pt-PT/studio/search)
- Procure algo que queira adicionar à sua lista.
- Quando encontrar, clique no link `Ações` no canto inferior esquerdo do cartão do resultado da pesquisa.
- Clique em `Adicionar a...` e depois selecione a lista curada à qual quer adicionar este inventário.
- Volte à página das listas curadas para verificar se o item foi adicionado à sua lista.

Todas as listas têm algumas ações que pode executar.

- Converter a lista num [mapa](/pt-PT/studio/maps).
- Converter a lista numa [grelha](/pt-PT/studio/grids).
- Remover a lista (*exceto Favoritos*).

## API

Os desenvolvedores que quiserem gerir listas curadas podem aceder a [Developers > API > Browse](/pt-PT/developers/apis/#browse-api).
