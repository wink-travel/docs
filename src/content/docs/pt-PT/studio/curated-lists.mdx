As listas selecionadas são uma funcionalidade que permite agrupar e classificar o inventário selecionado entre os itens encontrados durante a pesquisa de inventário de viagem.
Depois de criada, pode expor a lista aos seus utilizadores, disponibilizando-a para venda como uma [rede](/studio/grids) ou um [mapa](/studio/maps) através [WinkLinks](/link-manager/wink-links), nosso [Plugin WordPress](/developers/wordpress/) ou incorporando um dos nossos [Componentes da Web](/developers/web-components) no seu site.

Por predefinição, cada conta de afiliado começa com um `Favorites` lista que pode utilizar imediatamente. `Favorites` é especial e não pode ser removido.
Todas as listas selecionadas permitem fazer o seguinte:

## Criar uma lista

<figure>
  <CloudinaryImage width="350" height="219" resize="scale" quality="best" alt="Criar lista com curadoria" src="academy/create-curated-list" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Criar um novo formulário de lista com curadoria*</figcaption>
</figure>


* No painel, clique em `Inventory > Curated Lists` na barra de navegação.
* Crie uma nova lista selecionada clicando em `Make a new list` botão.
* Ser-lhe-á solicitado que nomeie a sua lista. *Exemplo: As 3 principais escolhas de Nova Iorque*.
* Clique no `Save` botão.

## Adicionar um item de lista

Pode adicionar um item de lista a qualquer lista dos resultados da pesquisa.

* Vá para [Pesquisa](/studio/search)
* Procure algo que queira adicionar à sua lista.
* Depois de o encontrar, clique em `Actions` link no canto inferior esquerdo do cartão de resultados da pesquisa.
* Clique `Add to...` e, em seguida, selecione a lista selecionada à qual pretende adicionar esse inventário.
* Volte à página de listas selecionadas para verificar se o item foi adicionado à sua lista.

## Listar ações

<figure>
  <CloudinaryImage width="350" height="183" resize="scale" quality="best" alt="Ações de lista com curadoria" src="academy/curated-list-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Ações de lista com curadoria*</figcaption>

Cada lista tem algumas ações que pode executar nela.

* Converta a lista num [mapa](/studio/maps).
* Converta a lista num [rede](/studio/grids).
* Remover a lista (*exceto para Favoritos*).

## API

Os programadores que desejam gerir listas selecionadas podem aceder [Programadores > API > Navegar](/developers/apis/#browse-api).
