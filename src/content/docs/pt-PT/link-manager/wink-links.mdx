Um gestor de links permite guardar e organizar URLs que gosta para mais tarde. Os favoritos do seu navegador são um exemplo de gestor de links. Com o advento das redes sociais, surgiu uma maior necessidade de expor e disponibilizar links aos seus amigos e audiências. O seu feed do Facebook é um exemplo de gestor de links amigável para redes sociais.

Construímos o [WinkLinks](https://i.trvl.as/) para ser o gestor de links da indústria de viagens. Está aqui para informar, entusiasmar, publicitar... e permite-lhe reservar ótimas ofertas de hotéis em todo o mundo.

## Funcionalidades

- O WinkLinks permite adicionar todos os seus links de afiliados existentes, como Commission Junction e Amazon, gratuitamente.
- Recebe uma URL personalizada fixe, como [https://i.trvl.as/bob](https://i.trvl.as/bob), para aquelas ocasiões em que só pode partilhar uma única URL.
- Pode organizar e partilhar links e carregar ficheiros.
- Hotéis, cadeias e marcas podem usar o WinkLinks como um motor de reservas avançado:
    - Suporta múltiplas propriedades.
    - Suporta múltiplos destinos.
    - Suporta múltiplas moedas.
    - Suporta múltiplos idiomas.
- O WinkLinks suporta `rich content` *(veja abaixo)*.
- Vá do IG para reservar em apenas 2 cliques 🚀

## Conteúdo enriquecido

:::note
Defina `Intelligent` para 🟢 nas suas [Definições](/pt-PT/link-manager/settings) para usar esta funcionalidade.
:::

Se partilhar um link de um site que suporta conteúdo `rich` / `oEmbed`, o seu link ganha vida no WinkLinks.

Sites suportados:

- Facebook - *Interaja com a publicação e reproduza um vídeo*.
- Instagram - *Interaja com a publicação e reproduza um vídeo*.
- YouTube - *Reproduza um vídeo*.
- X (antigo Twitter) - *Interaja com uma publicação*.
- Spotify - *Reproduza as suas playlists favoritas*.
- Wink - *Veja `Advanced`*.
