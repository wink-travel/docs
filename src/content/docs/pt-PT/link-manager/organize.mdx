Para gerir as suas etiquetas do WinkLinks, navegue até `WinkLinks` na barra de navegação principal e clique no separador `Etiquetas`.

À medida que o seu conteúdo cresce, vai querer começar a organizar parte dele. Criámos `etiquetas` para este propósito. Pode agrupar conteúdo usando tantas etiquetas quanto desejar.

## Criar etiqueta

<Steps>
1. Crie uma nova etiqueta clicando no botão `Adicionar uma nova etiqueta`.
2. Introduza o nome da sua nova etiqueta. *ex. Melhores cafés em Berlim*
3. Clique no link `Guardar`.
</Steps>

## Atualizar etiqueta

<Steps>
1. Comece por clicar no link `Editar` ao lado da etiqueta que pretende modificar.
2. Atualize o nome da sua etiqueta. *ex. Melhores cafés em Berlim*
3. Clique no link `Guardar`.
</Steps>

## Remover etiqueta

:::note
Ao remover uma etiqueta, irá também removê-la de todo o conteúdo associado.
:::

<Steps>
1. Clique no link `Remover` ao lado da etiqueta que pretende eliminar.
</Steps>

## API

Os programadores que quiserem gerir o `WinkLinks` podem aceder a [Developers > API > WinkLinks](/pt-PT/developers/apis/#winklinks-api).
