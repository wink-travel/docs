Adicionar informações do Diretor Geral ajuda a criar mais familiaridade com o local.

Para atualizar as informações do Diretor Geral, clique em `Propriedade > Diretor Geral` no separador principal de navegação.

Pode adicionar o nome e a foto do seu Diretor Geral atual, juntamente com uma mensagem pessoal para os hóspedes.

Atualize o formulário e clique no botão `Guardar` para continuar.
