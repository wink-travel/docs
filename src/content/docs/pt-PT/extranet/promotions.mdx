:::note
Semelhante à indústria aérea, os hotéis necessitam de muita flexibilidade em relação a preços e descontos.
As promoções são uma forma do hotel criar condições únicas sob as quais um viajante é elegível para uma redução adicional de preço [ou prémio].
:::

Para gerir as suas Promoções, clique em `Monetize > Single Promotions` na barra de navegação principal.

Para criar uma nova promoção, clique no botão `Create single promotion`.

## Configuração

Esta secção é responsável por definir o valor monetário real da promoção.

- **Nome** Introduza um nome para esta promoção que só você pode ver.
- **Tipo** Escolha se quer que esta promoção contenha um desconto ou um prémio.
- **Tipo de preço** Escolha como quer que este pacote seja calculado.
    - `Por estadia`: O valor é aplicado à última noite. *ex. 100% de desconto > Fique 2 noites - Ganhe a 3ª noite grátis.*
    - `Por noite`: Valor multiplicado pelo número de noites.
    - `Por uso`: Não ocorre cálculo adicional.
    - `Por pessoa`: Valor multiplicado pelo número de hóspedes.
    - `Por pessoa por noite`: Valor multiplicado pelo número de noites + número de hóspedes.
- **Tipo de valor** Escolha se quer que o modificador de valor seja um montante fixo ou uma percentagem.
- **Estado** Ative o interruptor para controlar se a promoção está disponível em todos os canais de venda.

## Descrições

:::note
É necessário escrever pelo menos uma descrição e esta deve estar em inglês.
:::

Clique no separador `Descriptions` para continuar.

Uma descrição localizada descreve para o viajante, na sua língua, o propósito desta promoção.
Pode escrever em quantas línguas desejar. Iremos adicionar traduções para todas as línguas mais populares.
Este é o texto que os viajantes verão ao consultar a sua promoção.

- **Descrição** Dê um nome à promoção. *ex. Early bird - 20% OFF*

## Restrições

:::note
Tenha em mente que estas são restrições... Só as ative se quiser tornar a sua promoção menos acessível.
:::

Clique no separador `Restrictions` para continuar.

O motor de promoções da Wink tem, nada menos que, 20 formas únicas de ativar uma alteração de preço.

Suportamos os seguintes critérios:

- Restringir a promoção a certos complementos.
- Restringir a promoção a certas tarifas principais.
- Restringir a promoção a certos planos tarifários.
- Restringir a promoção a certas políticas de cancelamento.
- Restringir a promoção à duração da estadia.
- Restringir a promoção à reserva antecipada.
- Restringir a promoção ao número de quartos.
- Restringir a promoção a código promocional.
- Restringir a promoção ao intervalo de IP do viajante.
- Restringir a promoção a última hora.
- Restringir a promoção à cidade do viajante.
- Restringir a promoção ao país do viajante.
- Restringir a promoção ao continente do viajante.
- Restringir a promoção ao fuso horário do viajante.
- Restringir a promoção às datas de venda.
- Restringir a promoção às datas de estadia.
- Restringir a promoção aos dias da semana da reserva.
- Restringir a promoção aos dias da semana da chegada.
- Restringir a promoção aos dias da semana da partida.
- Restringir a promoção aos dias da semana obrigatórios.

Continue a ler para ver como usar cada uma.

### Complemento

:::note
Para usar esta funcionalidade, precisa de ter criado pelo menos um complemento.
:::

Para aplicar uma promoção apenas aos complementos que selecionar, siga estes passos:

<Steps>
1. Ative o interruptor `Add-on restriction`.
2. Marque a caixa ao lado de cada complemento que deseja incluir.
</Steps>

### Tarifa principal

:::note
Para usar esta funcionalidade, precisa de ter criado pelo menos uma tarifa principal.
:::

Para aplicar uma promoção apenas às tarifas principais que selecionar, siga estes passos:

<Steps>
1. Ative o interruptor `Master rate restriction`.
2. Marque a caixa ao lado de cada tarifa principal que deseja incluir.
</Steps>

### Plano tarifário

:::note
Para usar esta funcionalidade, precisa de ter criado pelo menos um plano tarifário.
:::

Para aplicar uma promoção apenas aos planos tarifários que selecionar, siga estes passos:

<Steps>
1. Ative o interruptor `Rate plan restriction`.
2. Marque a caixa ao lado de cada plano tarifário que deseja incluir.
</Steps>

### Reembolsável

:::note
Para usar esta funcionalidade, precisa de ter criado pelo menos uma política de cancelamento.
:::

Para aplicar uma promoção apenas às políticas de cancelamento que selecionar, siga estes passos:

<Steps>
1. Ative o interruptor `Cancellation policy restriction`.
2. Marque a caixa ao lado de cada política de cancelamento que deseja incluir.
</Steps>

### Duração da estadia

:::note
Esta restrição é normalmente usada como a sua promoção de `longo prazo`.
:::

Ao adicionar esta restrição, está a exigir que os hóspedes `fiquem pelo menos` e `no máximo` um número especificado de dias.

Siga estes passos para ativar `Duração da estadia`:

<Steps>
1. Ative o interruptor `Length of stay restriction`.
2. Introduza o número mínimo de dias que o hóspede deve ficar. *ex. 7*
3. Opcionalmente, introduza o número máximo de dias que o hóspede pode ficar. *ex. 90*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que fiquem mais de 10 dias. Nota: O máximo da duração da estadia é opcional.*

### Reserva antecipada

:::note
Esta restrição é normalmente usada como a sua promoção de `early bird`.
:::

Ao adicionar esta restrição, está a exigir que os hóspedes `reservem pelo menos` e `no máximo` um número especificado de dias antes.

Siga estes passos para ativar `Reserva antecipada`:

<Steps>
1. Ative o interruptor `Advance booking restriction`.
2. Introduza o número mínimo de dias que o hóspede precisa para reservar antecipadamente. *ex. 10*
3. Opcionalmente, introduza o número máximo de dias que o hóspede precisa para reservar antecipadamente. *ex. 180*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem com mais de 10 dias de antecedência. Nota: Também é necessário adicionar um máximo para a reserva antecipada.*

### Número de quartos

:::note
A promoção está disponível apenas se o hóspede reservar o número mínimo de quartos.
:::

Ao adicionar esta restrição, está a exigir que os hóspedes `reservem pelo menos` e `no máximo` um número especificado de quartos como parte da sua reserva.

Siga estes passos para ativar `Número de quartos`:

<Steps>
1. Ative o interruptor `Room restriction`.
2. Introduza o número mínimo de quartos que o hóspede precisa reservar. *ex. 3*
3. Introduza o número máximo de quartos que o hóspede precisa reservar. *ex. 8*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem mais de 3 quartos.*

### Código promocional

:::note
Use esta restrição se quiser que o hóspede insira um código de desconto para desbloquear esta promoção.
:::

Siga estes passos para ativar `Código promocional`:

<Steps>
1. Ative o interruptor `Promotion restriction`.
2. Clique no botão `Add promo code`.
3. Introduza o código promocional. *ex. ABC123*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que inserem o código promocional `ABC123`.*

### Intervalo de IP

:::note
Esta é uma boa restrição para usar quando quer fazer uma campanha cruzada com operadoras móveis ou grandes empresas que têm os seus próprios intervalos de IP.
:::

Siga estes passos para ativar `Intervalo de IP`:

<Steps>
1. Ative o interruptor `IP range restriction`.
2. Clique no botão `Add IP range restriction`.
3. Introduza o intervalo inicial de IP [globalmente acessível]. *ex. 203.0.113.50*
4. Introduza o intervalo final de IP [globalmente acessível]. *ex. 203.0.113.59*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes com números de IP entre `203.0.113.50` - `203.0.113.59`.*

### Última hora

:::note
Este é o intervalo de tempo do negócio de última hora que quer restringir.
:::

Siga estes passos para ativar `Última hora`:

<Steps>
1. Ative o interruptor `Last minute restriction`.
2. Clique no botão `Add IP range restriction`.
3. Introduza o número de horas antes da data de chegada. *ex. 24 horas*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem 24 horas antes da data de chegada.*

### Cidade

:::note
Permite restringir à cidade onde o viajante se encontra atualmente. Crie uma restrição para cada cidade que deseja incluir.
:::

Siga estes passos para ativar `Cidade`:

<Steps>
1. Ative o interruptor `City restriction`.
2. Comece a digitar o nome da cidade.
3. Selecione a cidade nos resultados da pesquisa. *ex. New York City*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem a partir de `New York City`.*

### País

:::note
Permite restringir ao país onde o viajante se encontra atualmente. Crie uma restrição para cada país que deseja incluir.
:::

Siga estes passos para ativar `País`:

<Steps>
1. Ative o interruptor `Country restriction`.
2. Comece a digitar o nome do país.
3. Selecione o país nos resultados da pesquisa. *ex. United States of America*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem a partir de `United States of America`.*

### Continente

:::note
Permite restringir a tarifa ao continente onde o hóspede se encontra atualmente. Crie uma restrição para cada continente que deseja incluir.
:::

Siga estes passos para ativar `Continente`:

<Steps>
1. Ative o interruptor `Continent restriction`.
2. Comece a digitar o nome do continente.
3. Selecione o continente nos resultados da pesquisa. *ex. North America*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem a partir de `North America`.*

### Fuso horário

:::note
Permite restringir a tarifa ao fuso horário onde o hóspede se encontra atualmente. Crie uma restrição para cada fuso horário que deseja incluir.
:::

Siga estes passos para ativar `Fuso horário`:

<Steps>
1. Ative o interruptor `Timezone restriction`.
2. Comece a digitar o nome do fuso horário.
3. Selecione o fuso horário nos resultados da pesquisa. *ex. America/New_York*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem a partir de `America/New_York`.*

### Datas de venda

:::note
Permite restringir a um intervalo específico de datas. Um viajante será elegível ao reservar um quarto nas datas especificadas dentro deste intervalo.
:::

Siga estes passos para ativar `Datas de venda`:

<Steps>
1. Ative o interruptor `Sell date restriction`.
2. Clique no botão `Add sell date restriction`.
3. Introduza a data de início. *ex. 19 de set. de 2024*
4. Introduza a data de fim. *ex. 1 de dez. de 2024*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem entre `19 de set. de 2024` e `1 de dez. de 2024`.*

### Datas de estadia

:::note
Permite restringir a um intervalo específico de datas. Um viajante será elegível ao reservar um quarto para as datas especificadas dentro deste intervalo.
:::

Siga estes passos para ativar `Datas de estadia`:

<Steps>
1. Ative o interruptor `Stay date restriction`.
2. Clique no botão `Add stay date restriction`.
3. Introduza a data de início. *ex. 19 de set. de 2024*
4. Introduza a data de fim. *ex. 1 de dez. de 2024*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem uma estadia entre `19 de set. de 2024` e `1 de dez. de 2024`.*

### Dias da semana da reserva

:::note
Ao ativar esta restrição, está a permitir que esta tarifa esteja disponível apenas no dia da semana especificado em que a reserva ocorre.

Esta restrição funciona bem com a "Restrição de datas de venda", onde pode limitar um intervalo de datas e restringir ainda mais o intervalo com dias da semana usando esta restrição.
:::

Siga estes passos para ativar `Dias da reserva`:

<Steps>
1. Ative o interruptor `Booking days restriction`.
2. Ative um dos interruptores do dia da semana. *ex. Segunda-feira*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem uma estadia numa `Segunda-feira`.*

### Dias da semana da chegada

:::note
Ao ativar esta restrição, os hóspedes devem chegar nesse dia para ver esta tarifa.

Esta restrição funciona bem com a "Restrição de datas de estadia", onde pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Siga estes passos para ativar `Dias da chegada`:

<Steps>
1. Ative o interruptor `Arrival days restriction`.
2. Ative um dos interruptores do dia da semana. *ex. Segunda-feira*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem uma estadia que comece numa `Segunda-feira`.*

### Dias da semana da partida

:::note
Ao ativar esta restrição, os hóspedes devem partir nesse dia para ver esta tarifa.

Esta restrição funciona bem com a "Restrição de datas de estadia", onde pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Siga estes passos para ativar `Dias da partida`:

<Steps>
1. Ative o interruptor `Departure days restriction`.
2. Ative um dos interruptores do dia da semana. *ex. Segunda-feira*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem uma estadia que termine numa `Segunda-feira`.*

### Dias da semana obrigatórios

:::note
Ao ativar esta restrição, os hóspedes devem ficar pelo menos numa destas datas para ver esta tarifa.

Esta restrição funciona bem com a "Restrição de datas de estadia", onde pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Siga estes passos para ativar `Dias obrigatórios`:

<Steps>
1. Ative o interruptor `Required days restriction`.
2. Ative um dos interruptores do dia da semana. *ex. Segunda-feira*
</Steps>

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem uma estadia que inclua uma `Segunda-feira`.*

## Datas de blackout

Clique no separador `Blackout dates` para continuar.

Controle a promoção especificando datas em que esta promoção não deve estar disponível.

Para adicionar uma data de blackout, clique no botão `Add blackout date`.

- **Data de início** Selecione a data de início da sua data de blackout.
- **Data de fim** Selecione a data de fim da sua data de blackout.

Se as datas da reserva tocarem este intervalo, a promoção não terá efeito.

Um viajante pode ser elegível para múltiplas promoções, como longo prazo + early bird, ao mesmo tempo.
Sem uma promoção agrupada, o viajante seria elegível apenas para uma delas; nomeadamente a melhor oferta.

Ao criar uma promoção agrupada, respeita a elegibilidade do viajante para ambas as promoções e pode personalizá-la ainda mais.

Para criar uma promoção agrupada, clique no botão `Create bundled promotion`.

### Configuração

- **Nome** Dê um nome ao pacote que só você pode ver.
- **Estado** Ative o interruptor para controlar se o pacote está disponível em todos os canais de venda.
- **Promoções** Selecione, pelo menos 2, promoções simples para compor o pacote.

### Substituir valor

Pode optar por substituir o desconto total aplicado. Se deixar desligado, o desconto de cada promoção selecionada abaixo é calculado num único desconto.

- **Substituir valor** Se quer agregar todos os valores descontados ou substituir pelo seu próprio valor.
- **Tipo** Selecione se quer que o preço aumente ou diminua. *ex. Defina para `Discount` para uma redução de preço.*
- **Tipo de preço** Escolha como quer que este pacote seja calculado.
    - `Por estadia`: O valor é aplicado à última noite. *ex. 100% de desconto > Fique 2 noites - Ganhe a 3ª noite grátis.*
    - `Por noite`: Valor multiplicado pelo número de noites.
    - `Por uso`: Não ocorre cálculo adicional.
    - `Por pessoa`: Valor multiplicado pelo número de hóspedes.
    - `Por pessoa por noite`: Valor multiplicado pelo número de noites + número de hóspedes.
- **Tipo de valor** Escolha se quer que o modificador de valor seja um montante fixo ou uma percentagem.
