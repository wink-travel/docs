:::note
Esta é uma subfunção da funcionalidade de [Links Partilháveis](/pt-PT/studio/shareable-links) disponível no Wink Studio.

As propriedades terão automaticamente todo o seu inventário disponível como links partilháveis sem necessidade de aceder ao Wink Studio para os gerar.
:::

Links partilháveis é uma funcionalidade central do Wink e surgiu da necessidade de comunicar inventário e preços de forma dinâmica através do método mais simples possível na Internet; nomeadamente o hiperligação.

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

Acima está um exemplo de um dos nossos links. É curto e portátil... até suficientemente curto para memorizar. Apesar de curto, este link contém uma quantidade **explosiva** de informação, como:

- O seu App ID.
- O seu Affiliate ID.
- A(s) imagem(ns) e texto a mostrar ao utilizador.
- O aspeto visual (*ou seja, tema, cores e mais*).
- O itinerário a usar para a nossa consulta de preços (*2 adultos, 1 noite*).
- A moeda para apresentar o preço.
- O idioma para apresentar o texto.

O resto deste artigo explica como gerir os links para os seus inventários.

## Personalizar link

 O formulário permite personalizar o seu link das seguintes formas:

<Steps>
 1. Por defeito, o link recebe o nome do seu inventário em inglês. Pode dar-lhe qualquer nome que o ajude a lembrar do que se trata o link.
 2. Por defeito, a sua personalização / branding principal é aplicada ao link. Pode selecionar outras [Personalizações](/pt-PT/studio/customization) que tenha criado no Wink Studio.
 3. Adicione palavras-chave, separadas por vírgulas, que serão usadas por [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Opcionalmente, insira o seu identificador da conta X ou o Facebook App ID.
 5. Adicione títulos e descrições nos idiomas que deseja que o utilizador veja.
 6. Selecione uma ou mais imagens que serão exibidas quando partilhar o link em plataformas como Facebook ou LinkedIn.
 7. Escolha um tema / sobreposição que adiciona um design com a sua marca por cima da(s) imagem(ns) selecionada(s). *Nota: É assim que faz os preços aparecerem por cima da imagem.*
 8. Clique no botão `Save` para continuar.
 </Steps>

Acima vê uma imagem com todas as ações disponíveis para o seu link:

- **Preview** Mostra-lhe uma visão rápida de como o link foi configurado e como aparecerá ao utilizador.
- **Update** Atualiza a configuração do seu link.
- **Add to WinkLinks** Adiciona o link à sua conta WinkLinks.
- **Copy** Copia o link para a sua área de transferência para que possa partilhá-lo facilmente onde quiser (*incluindo [WinkLinks](/pt-PT/link-manager/wink-links) *).
- **Share** Abre uma nova janela com opções para partilhar em várias redes sociais e plataformas de mensagens.
- **QR code** Gera um código QR que pode partilhar em plataformas como o IG, que são mais focadas em imagens.
- **Embed** Mostra-lhe como incorporar este link como um [Ad Banner](/pt-PT/developers/web-components/#content-loader) no seu website.
- **Use with WordPress** Mostra-lhe como usar o nosso [plugin WordPress](/pt-PT/developers/wordpress) para incorporar este link no seu website.

Se preferir partilhar um código QR em vez de um link, clique no botão `QR code` e aparecerá uma nova janela com o seu código QR.

Como guardar o seu código QR:

- Se estiver no seu portátil, clique com o botão direito na imagem e selecione `Guardar imagem`.
- Se estiver no seu telemóvel:
    - Pressione e mantenha o dedo sobre a imagem até aparecer uma opção para guardar a imagem nas suas Fotos.
    - Tire um screenshot do código QR, recorte para ajustar e guarde nas Fotos.

### Incorporar

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Aqui está como incorporar o seu link, como um banner publicitário, no seu site:

- A linha 3 mostra como incorporar os estilos do Wink no seu site.
- As linhas 6 a 9 mostram como usar o [wink-content-loader](/pt-PT/developers/web-components/#content-loader) Web Component e indicar que deve buscar um banner publicitário para o seu código.
- A linha 11 mostra como incorporar o nosso Javascript no seu site.
- A linha 13 mostra como incorporar o [wink-app-loader](/pt-PT/developers/web-components/#application-loader) Web Component e indicar que deve buscar as suas preferências de configuração a nível de página.
