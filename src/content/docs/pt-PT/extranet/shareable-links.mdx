:::note
Esta é uma subfunção da funcionalidade de [Links Partilháveis](/studio/shareable-links) disponível no Wink Studio.

As propriedades terão automaticamente todo o seu inventário disponível como links partilháveis sem necessidade de aceder ao Wink Studio para os gerar.
:::

Links partilháveis é uma funcionalidade central do Wink e surgiu da necessidade de comunicar inventário e preços de forma dinâmica através do método mais simples possível na Internet; nomeadamente o hiperlink.

<h4 align="center">[https://trvl.as/3xWCH](https://trvl.as/3xWCH)</h4>

Acima está um exemplo de um dos nossos links. É curto e portátil... até suficientemente curto para memorizar. Apesar de curto, este link contém uma quantidade **explosiva** de informação, como:

- O seu ID de Aplicação.
- O seu ID de Afiliado.
- A(s) imagem(ns) e texto a mostrar ao utilizador.
- O aspeto visual (*ou seja, tema, cores e mais*).
- O itinerário a usar para a nossa consulta de preços (*2 adultos, 1 noite*).
- A moeda para apresentar o preço.
- O idioma para apresentar o texto.

O resto deste artigo explica como gerir os links para os seus inventários.

## Personalizar link

 O formulário permite personalizar o seu link das seguintes formas:

<Steps>
 1. Por defeito, o link recebe o nome do seu inventário em inglês. Pode dar-lhe qualquer nome que o ajude a lembrar do que se trata o link.
 2. Por defeito, a sua personalização / marca principal é aplicada ao link. Pode selecionar outras [Personalizações](/studio/customization) que tenha criado no Wink Studio.
 3. Adicione palavras-chave, separadas por vírgulas, que serão usadas por [Web Crawlers](https://developer.mozilla.org/en-US/docs/Glossary/Crawler).
 4. Opcionalmente, insira o seu identificador da conta X ou o ID da App do Facebook.
 5. Adicione títulos e descrições nos idiomas que deseja que o utilizador veja.
 6. Selecione uma ou mais imagens que serão exibidas quando partilhar o link em plataformas como Facebook ou LinkedIn.
 7. Escolha um tema / sobreposição que adiciona um design com a sua marca por cima da(s) imagem(ns) selecionada(s). *Nota: É assim que os preços são exibidos por cima da imagem.*
 8. Clique no botão `Guardar` para continuar.
 </Steps>

Acima vê uma imagem com todas as ações disponíveis para o seu link:

- **Pré-visualizar** Mostra uma visão rápida de como o link foi configurado e como aparecerá ao utilizador.
- **Atualizar** Atualiza a configuração do seu link.
- **Adicionar ao WinkLinks** Adiciona o link à sua conta WinkLinks.
- **Copiar** Copia o link para a sua área de transferência para que possa partilhá-lo facilmente onde quiser (*incluindo [WinkLinks](/link-manager/wink-links)*).
- **Partilhar** Abre uma nova janela com opções para partilhar em várias redes sociais e plataformas de mensagens.
- **Código QR** Gera um código QR que pode partilhar em plataformas como o IG, que são mais focadas em imagens.
- **Incorporar** Mostra como incorporar este link como um [Banner Publicitário](/developers/web-components/#content-loader) no seu website.
- **Usar com WordPress** Mostra como usar o nosso [plugin WordPress](/developers/wordpress) para incorporar este link no seu website.

Abordamos algumas destas opções, com mais detalhe, abaixo.

Se preferir partilhar um código QR em vez de um link, clique no botão `Código QR` e abrir-se-á uma nova janela com o seu código QR.

Como guardar o seu código QR:

- Se estiver no seu portátil, clique com o botão direito na imagem e selecione `Guardar imagem`.
- Se estiver no seu telemóvel:
    - Pressione e mantenha o dedo sobre a imagem até aparecer a opção para guardar a imagem nas suas Fotos.
    - Tire um ecrã do código QR, recorte para ajustar e guarde nas Fotos.

### Incorporar

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Aqui está como incorporar o seu link, como um banner publicitário, no seu site:

- A linha 3 mostra como incorporar os estilos Wink no seu site.
- As linhas 6 a 9 mostram como usar o [wink-content-loader](/developers/web-components/#content-loader) Web Component e indicar que deve buscar um banner publicitário para o seu código.
- A linha 11 mostra como incorporar o nosso Javascript no seu site.
- A linha 13 mostra como incorporar o [wink-app-loader](/developers/web-components/#application-loader) Web Component e indicar que deve buscar as suas preferências de configuração a nível de página.
