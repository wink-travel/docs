Os planos tarifários são o filtro através do qual os quartos são apresentados aos utilizadores.
Contêm também definições que controlam a estrutura de preços em determinados cenários.

Os planos tarifários permitem-lhe:

- Definir uma política de cancelamento [e exceções à política].
- Preço para crianças extra, adultos e viajantes individuais.
- Definir refeições.
- Adicionar encargos por check-in antecipado e check-out tardio.
- Definir encargos adicionais.
- Definir restrições.

Para criar um plano tarifário, clique no botão `Create a new rate plan`.

## Configuração

A aba `Configurations` está selecionada por defeito quando começa.

Esta secção cobre as opções básicas para configurar o seu plano tarifário.

- **Nome do plano tarifário** O nome do plano tarifário, como o referencia internamente. *ex. BAR RO*
- **Ativado** Ative ou desative este plano tarifário. Quando desativado, o plano não estará disponível para reserva em nenhum canal de venda.
- **Pré-pago** Defina se este plano tarifário requer pré-pagamento no momento da reserva.
- **Pontos de fidelidade** Se o hóspede acumula os seus pontos de fidelidade neste plano tarifário. *ex. Não*
- **Modificador de tarifa para ocupação individual** Defina um modificador de preço para um ocupante único num quarto de ocupação múltipla. *ex. 5% DE DESCONTO no preço regular*
- **Modificador de tarifa para adulto extra** Defina um modificador de preço para um adulto extra no quarto. *ex. +10 / adulto*
- **Modificador de tarifa para criança extra** Defina um modificador de preço para uma criança extra no quarto. *ex. +5 / criança*

### Política de cancelamento

- **Política de cancelamento** Defina a política de cancelamento para este plano tarifário. *ex. Reembolsável*
- **Exceções** Defina políticas de cancelamento adicionais que entram em vigor para intervalos de datas específicos. *ex. Para o mês de dezembro, use a política `Reembolsável com condições extra`.*

### Refeições

Selecione os planos de refeições que acompanham este plano tarifário. As seguintes opções estão disponíveis:

- **Pequeno-almoço**
- **Brunch**
- **Almoço**
- **Jantar**
- **Tudo incluído**
- **Tudo incluído + Álcool**

## Encargos

Defina encargos adicionais que acompanham o plano tarifário.

- **Check-in antecipado** Defina um encargo adicional quando o hóspede quiser fazer check-in antecipado. *Isto é apenas para uso informativo*
- **Check-out tardio** Defina um encargo adicional quando o hóspede quiser fazer check-out tardio. *Isto é apenas para uso informativo*

### Encargos extra

Defina encargos adicionais com o plano tarifário.

Para criar um encargo extra, clique no botão `Create extra charge`.

- **Tipo de preço** Escolha como este encargo é calculado. Tipos disponíveis:
  - *Por estadia* — Cobrado uma vez por reserva.
  - *Por dia* — Cobrado uma vez por noite de estadia.
  - *Por pessoa / Por estadia* — Cobrado por hóspede por reserva.
  - *Por pessoa / Por dia* — Cobrado por hóspede por noite.
- **Montante** Defina o montante fixo do encargo. *ex. 100*

Preencha o nome e a descrição deste encargo.

:::note
É necessário escrever pelo menos uma descrição e esta deve estar em inglês.
:::

- **Nome** Nome do encargo. *ex. Encargo único de limpeza*
- **Descrição** Descreva o encargo num ou dois parágrafos. *ex. A vila será limpa pela nossa equipa profissional de limpeza antes da sua chegada.*

## Restrições

Defina restrições que controlam se um quarto / plano tarifário está disponível ou não.

Restrições suportadas:

- Duração da estadia
- Reserva antecipada
- Ocupação
- Idade
- Datas de venda / estadia / disponíveis / chegada / partida / datas obrigatórias

### Duração da estadia

Ao adicionar esta restrição, está a exigir que os hóspedes "fiquem pelo menos" e "no máximo" um número especificado de dias.

*Exemplo: Este tipo de quarto está disponível apenas para hóspedes que fiquem mais de 10 dias. Nota: O máximo da duração da estadia é opcional.*

### Reserva antecipada

Ao adicionar esta restrição, está a exigir que os hóspedes "reservem pelo menos" e "no máximo" um número especificado de dias com antecedência.

*Exemplo: Esta promoção está disponível apenas para hóspedes que reservem com mais de 10 dias de antecedência. Nota: Também é necessário adicionar uma reserva antecipada máxima.*

### Ocupação

Ao adicionar esta restrição, está a exigir que o grupo que chega tenha "pelo menos" e "no máximo" o número especificado de pessoas.

### Idade

Ao adicionar esta restrição, está a exigir que os hóspedes tenham "pelo menos" e "no máximo" a idade especificada.

### Datas de venda

Ao adicionar esta restrição, pretende tornar esta tarifa disponível para reserva apenas dentro de um intervalo de datas específico. Um hóspede só verá esta tarifa se estiver a reservar um quarto NA data especificada dentro deste intervalo.

### Datas de estadia

Ao adicionar esta restrição, pretende tornar esta tarifa disponível para estadia apenas dentro de um intervalo de datas específico. Um hóspede só verá esta tarifa se estiver a reservar um quarto PARA a data especificada dentro deste intervalo.

### Dias da semana disponíveis

:::note
Esta restrição funciona bem com a "restrição de datas de venda", onde pode limitar um intervalo de datas e restringir ainda mais o intervalo com dias da semana usando esta restrição.
:::

Ao ativar esta restrição, está a permitir que esta tarifa seja pesquisada APENAS nos dias da semana especificados.

### Datas de chegada

:::note
Esta restrição funciona bem com a "restrição de datas de estadia", onde pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Ao ativar esta restrição, os hóspedes DEVEM chegar nesse dia para ver esta tarifa.

*Exemplo: Ativar segunda-feira exige que o hóspede chegue numa segunda-feira.*

### Datas de partida

:::note
Esta restrição funciona bem com a "restrição de datas de estadia", onde pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Ao ativar esta restrição, os hóspedes DEVEM partir nesse dia para ver esta tarifa.

*Exemplo: Ativar segunda-feira exige que o hóspede parta numa segunda-feira.*

### Datas obrigatórias

:::note
Esta restrição funciona bem com a "restrição de datas de estadia", onde pode limitar o intervalo de datas e restringir ainda mais os dias da semana usando esta restrição.
:::

Ao ativar esta restrição, os hóspedes DEVEM permanecer "pelo menos num" destes dias para ver esta tarifa.

*Exemplo: Ativar segunda-feira exige que os hóspedes permaneçam numa segunda-feira.*
