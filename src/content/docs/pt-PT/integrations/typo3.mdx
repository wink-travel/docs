A **Extensão Wink para TYPO3** integra o inventário de viagens Wink no TYPO3 13 como elementos de conteúdo nativos. Os editores adicionam componentes Wink às páginas usando o editor de páginas padrão do TYPO3 — não são necessárias alterações no template.

## Requisitos

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instalar

<Steps>
1. Instale via Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Ative a extensão:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. No backend do TYPO3, vá a **Admin Tools → Settings → Extension Configuration → wink**.
4. Insira o seu **Client ID**, **Client Secret** e selecione o seu ambiente. Guarde.
   Vá a [Applications](/pt-PT/portal/settings/applications) para obter as suas credenciais.
5. Adicione o template TypoScript estático ao seu template raiz:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Adicionar um elemento de conteúdo a uma página

<Steps>
1. Abra uma página no módulo **Page**.
2. Clique no botão **+** na área de conteúdo para adicionar um novo elemento.
3. Escolha um dos elementos de conteúdo **Wink Travel**:
   - **Wink Content** — cartões de hotéis, grelhas classificadas, mapas
   - **Wink Lookup** — barra de pesquisa de destinos
   - **Wink Search** — botão seletor de itinerário
   - **Wink Account** — botão de início de sessão
   - **Wink Itinerary** — botão de estado do itinerário
   - **Wink Cart** — botão do carrinho de compras
4. Para **Wink Content**, use os campos FlexForm para selecionar um **Tipo de Layout** e inserir o **ID do Layout**.
5. Guarde e publique a página.
</Steps>

## Elementos de conteúdo disponíveis

| Elemento | Finalidade |
|---|---|
| **Wink Content** | Cartões de hotéis, grelhas classificadas, mapas e mais |
| **Wink Lookup** | Barra de pesquisa de destinos |
| **Wink Search** | Botão ícone que abre o seletor de itinerário |
| **Wink Account** | Botão de início de sessão / conta de utilizador |
| **Wink Itinerary** | Botão de itinerário que mostra a viagem atual |
| **Wink Cart** | Botão resumo do carrinho de compras |
