O pacote npm **`@wink/elements`** é a forma mais rápida de adicionar componentes web Wink a qualquer projeto JavaScript ou TypeScript. Fornece uma função `load()` que injeta os recursos do CDN e declarações completas de tipos TypeScript para todos os atributos dos componentes.

## Instalar

```bash
npm install @wink/elements
```

## Início rápido

Chame `load()` uma vez no arranque da aplicação. É idempotente — seguro para chamar várias vezes.

```ts
import { load } from '@wink/elements';

load({ clientId: 'SEU_CLIENT_ID' });
```

Depois use qualquer componente Wink no seu HTML ou JSX:

```html
<wink-content-loader layout="HOTEL" id="SEU_LAYOUT_ID"></wink-content-loader>
```

## Exemplos por framework

### Next.js (App Router)

```tsx
// app/layout.tsx
'use client';
import { useEffect } from 'react';
import { load } from '@wink/elements';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  useEffect(() => {
    load({ clientId: process.env.NEXT_PUBLIC_WINK_CLIENT_ID! });
  }, []);
  return <html><body>{children}</body></html>;
}
```

### Nuxt 3

```ts
// plugins/wink.client.ts
import { load } from '@wink/elements';

export default defineNuxtPlugin(() => {
  load({ clientId: useRuntimeConfig().public.winkClientId });
});
```

### Astro

```astro
---
// src/layouts/Layout.astro
import { load } from '@wink/elements';
---
<script>
  import { load } from '@wink/elements';
  load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
</script>
```

### SvelteKit

```ts
// src/routes/+layout.ts
import { load as loadWink } from '@wink/elements';
import { browser } from '$app/environment';

export function load() {
  if (browser) {
    loadWink({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  }
}
```

## Tipos TypeScript

Todas as interfaces de atributos dos componentes são exportadas para uso em JSX, template literals ou dicionários tipados:

```ts
import type { WinkContentLoaderAttributes, WinkLayout } from '@wink/elements';

const attrs: WinkContentLoaderAttributes = {
  layout: 'HOTEL',    // O TypeScript irá detectar valores inválidos para layout
  id: 'abc123',
  sort: 'POPULARITY',
};
```

## Utilizar um CDN de staging

```ts
load({
  clientId: 'SEU_CLIENT_ID',
  cdnBaseUrl: 'https://staging-elements.wink.travel',
});
```

## API

### `load(options)`

| Opção | Tipo | Obrigatório | Padrão | Descrição |
|---|---|---|---|---|
| `clientId` | `string` | Sim | — | O seu Client ID OAuth2 da Wink |
| `configurationId` | `string` | Não | — | ID de personalização opcional |
| `cdnBaseUrl` | `string` | Não | `https://elements.wink.travel` | Sobrescreve a URL do CDN |
