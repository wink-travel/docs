As aplicações oferecem acesso total às funcionalidades do Wink e do TripPay e permitem que os seus utilizadores se autentiquem e efetuem login no seu website utilizando o nosso servidor de autorização OAuth2; assim como o Facebook e a Google.
Pode criar a sua própria OTA com o Wink. Começa com uma aplicação.

*Por predefinição, recebe uma aplicação ao registar uma nova conta de afiliado Wink ou TripPay. Nós usamos o `client ID` para alguns cenários.*

:::

*Para este exemplo, iremos autenticar através [Estúdio Wink](https://studio.wink.travel).*

<Steps>
  1. Após a autenticação, clique no ícone do seu perfil, no canto superior direito, e surgirá um menu.
  2. Clique no `Applications` ligação.
  3. Será redirecionado para a sua página de aplicações.
</Steps>


## Criar aplicação

Como não pode aceder ao `secret key` da aplicação que criámos para si, precisará de criar o seu próprio.

:::Cuidado
É necessário criar uma conta de afiliado no Wink ou uma conta no TripPay antes de criar uma aplicação, pois irá ligar a sua aplicação a uma conta específica.
:::

Aqui estão os passos para criar um aplicativo:

<Steps>
  1. Na página de aplicações, clique em `Create new application` botão.
  2. ***Nome*** Dê um nome à sua aplicação. *por exemplo Aplicação legal*
  3. **Nome da entidade** Ligue a sua aplicação a uma das suas contas existentes. *por exemplo Conta Legal*
  4. **URIs de redireccionamento** Pelo menos um URI de redireccionamento é necessário. É utilizado para validar o domínio de redireccionamento após uma autenticação bem-sucedida. *por exemplo https://www.cool-site.com*
  5. Clique no `Save` botão para continuar.
</Steps>


Será redirecionado de volta para a sua lista de aplicações. Verá a sua nova aplicação na lista e terá acesso à chave secreta da aplicação apenas desta vez. Não se esqueça de o copiar e guardar em local seguro.

## Atualizar aplicação

Terá de atualizar a sua aplicação se um dos seus URIs de redireccionamento mudar ou se quiser adicionar ou remover um URI.

Para atualizar uma aplicação existente, faça o seguinte:

<Steps>
  1. Clique no `Actions` link para a aplicação que pretende atualizar.
  2. Clique no `Update` botão em Ações.
  3. Faça as suas alterações na aplicação.
  4. Clique no `Save` botão para continuar.
</Steps>


Será redirecionado de volta para a sua lista de aplicações.

## Remover aplicação

:::Cuidado
Não apague a aplicação que criámos automaticamente para si, para garantir que todos os recursos funcionam na sua conta.
:::

Se já não tiver utilidade para uma aplicação, pode removê-la.

Para remover uma aplicação, faça o seguinte:

<Steps>
  1. Clique no `Actions` link para a aplicação que pretende atualizar.
  2. Clique no `Remove` botão em Ações.
  3. Clique `OK` botão para confirmar a eliminação.
</Steps>


Será redirecionado de volta para a sua lista de aplicações.
