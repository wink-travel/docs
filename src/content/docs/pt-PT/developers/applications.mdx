As aplicações dão-lhe acesso total às funcionalidades do Wink e TripPay e permitem que os seus utilizadores se autentiquem e iniciem sessão no seu site usando o nosso servidor de autorização OAuth2; tal como o Facebook e o Google.
Pode construir a sua própria OTA com o Wink. Tudo começa com uma aplicação.

*Por predefinição, recebe uma app quando regista uma nova conta afiliada Wink ou uma conta TripPay. Usamos o `client ID` para alguns cenários.*

:::note
Uma aplicação gerada no Wink ou TripPay pode ser usada para todos os endpoints da API.
:::

*Para este exemplo, vamos autenticar via [Wink Studio](https://studio.wink.travel).*

<Steps>
1. Depois de autenticado, clique no ícone do seu perfil, no canto superior direito, e aparecerá um menu.
2. Clique no link `Applications`.
3. Será redirecionado para a sua página de apps.
</Steps>

## Criar app

Como não pode aceder à `secret key` da app que criámos para si, terá de criar a sua própria.

:::caution
Precisa de criar uma conta afiliada no Wink ou uma conta no TripPay antes de criar uma app, pois irá associar a sua aplicação a uma conta específica.
:::

Aqui estão os passos para criar uma aplicação:

<Steps>
1. Na página das apps, clique no botão `Create new application`.
2. ***Nome*** Dê um nome à sua app. *ex. Cool App*
3. **Nome da entidade** Associe a sua app a uma das suas contas existentes. *ex. Cool Account*
4. **Redirect URIs** É necessário pelo menos um URI de redirecionamento. É usado para validar o domínio de redirecionamento após uma autenticação bem-sucedida. *ex. https://www.cool-site.com*
5. Clique no botão `Save` para continuar.
</Steps>

Será redirecionado de volta para a sua lista de aplicações. Verá a sua nova app na lista e terá acesso à secret key da app apenas esta vez. Certifique-se de copiá-la e guardá-la num local seguro.

## Atualizar app

Terá de atualizar a sua app se algum dos seus redirect URIs mudar ou se quiser adicionar ou remover um URI.

Para atualizar uma app existente, faça o seguinte:

<Steps>
1. Clique no link `Actions` da app que pretende atualizar.
2. Clique no botão `Update` em Actions.
3. Faça as alterações na app.
4. Clique no botão `Save` para continuar.
</Steps>

Será redirecionado de volta para a sua lista de aplicações.

## Remover app

:::caution
Por favor, não elimine a app que criámos automaticamente para si para garantir que todas as funcionalidades funcionem para a sua conta.
:::

Se não precisar mais de uma app, pode removê-la.

Para remover uma app, faça o seguinte:

<Steps>
1. Clique no link `Actions` da app que pretende atualizar.
2. Clique no botão `Remove` em Actions.
3. Clique no botão `OK` para confirmar a eliminação.
</Steps>

Será redirecionado de volta para a sua lista de aplicações.
