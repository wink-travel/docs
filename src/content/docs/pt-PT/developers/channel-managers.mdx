A Wink integra-se com os principais channel managers, sistemas de gestão de propriedades (PMS) e sistemas centrais de reservas (CRS) para manter as suas tarifas e disponibilidade sincronizadas em todos os seus canais de venda.

## Como funciona

As integrações com channel managers permitem um fluxo bidirecional de dados entre o seu sistema de gestão de propriedades e a Wink:

1. **Envio de tarifas** — O seu channel manager envia atualizações de tarifas para a Wink. Quando atualiza os seus preços no PMS, essas alterações são automaticamente refletidas na Wink.
2. **Sincronização de disponibilidade** — O seu channel manager envia atualizações de disponibilidade para a Wink. Quando quartos são reservados ou libertados no seu PMS, o inventário da Wink atualiza em tempo real.
3. **Notificações de reservas** — Quando uma reserva é feita na Wink, o seu channel manager é notificado para que a reserva apareça no seu PMS.

## Channel managers suportados

A Wink integra atualmente com os seguintes sistemas:

| Channel Manager | Descrição |
|---|---|
| **Channex** | Plataforma global de distribuição e gestão de canais. |
| **TravelClick** | Soluções para hotéis e gestão de canais (Amadeus). |
| **Omnibees** | Distribuição e gestão de receitas para hotéis. |
| **D-Edge** | Tecnologia hoteleira e soluções de reservas centrais. |
| **Allotz** | Alocação de inventário e gestão de canais. |
| **Ezee / Yanolja** | PMS em cloud e gestão de canais focados no mercado asiático. |
| **RateGain** | Inteligência de tarifas e tecnologia de distribuição. |
| **RateTiger** | Distribuição em tempo real de tarifas e disponibilidade. |
| **CloudBeds** | PMS em cloud, channel manager e motor de reservas. |
| **SiteMinder** | Principal channel manager e plataforma de distribuição para hotéis. |
| **Exely** | Sincronização de tarifas e disponibilidade para hotéis. |

:::note
Se o seu channel manager não estiver listado, contacte a equipa da Wink. Novas integrações podem ser adicionadas com base nos padrões do protocolo OTA (Open Travel Alliance) que a Wink suporta.
:::

## Para desenvolvedores

Se está a construir uma integração de channel manager ou PMS com a Wink, utilize os endpoints da Channel Manager API:

- **Atualizações de tarifas** — Envie alterações de tarifas para a Wink.
- **Atualizações de disponibilidade** — Envie alterações de disponibilidade para a Wink.
- **Recuperação de reservas** — Obtenha reservas da Wink para o seu sistema.

A Channel Manager API segue os padrões do protocolo OTA (SOAP/XML) para compatibilidade com sistemas hoteleiros existentes.

## Notificações via webhook

Pode subscrever eventos webhook do channel manager para receber notificações em tempo real:

- `channel-manager.update.rate` — Atualização de tarifa recebida.
- `channel-manager.update.availability` — Atualização de disponibilidade recebida.
- `channel-manager.update` — Atualização geral do channel manager.

Consulte o [Catálogo de Eventos Webhook](/pt-PT/developers/webhook-events#channel-manager-events) para mais detalhes.
