## API de pagamento

Os pontos finais de pagamento são para os programadores que desejam comprar inventário de viagens. Isto pode ser feito através da API como agente de viagens registado ou utilizando a nossa API em conjunto com o nosso widget de pagamento compatível com PCI para todas as outras entidades.
A API de pagamento expõe endpoints para afiliados e hotéis rastrearem reservas, análises e disponibilidade de fundos. Podem também optar por levantar os fundos disponíveis para a sua conta bancária.

  Saber mais
</LinkButton>
