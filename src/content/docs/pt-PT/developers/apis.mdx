Se ainda não criou a sua conta Wink, [comece aqui](/pt-PT/user/register).

## Analytics API

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
Saiba mais
</LinkButton>

## Payment API

Os endpoints de Payment são para programadores que querem comprar inventário de viagens. Isto pode ser feito via API como Agente de Viagens registado ou usando a nossa API em conjunto com o nosso widget de pagamento compatível com PCI para todas as outras entidades.
A Payment API expõe endpoints para afiliados e hotéis acompanharem reservas, análises e disponibilidade de fundos. Também podem optar por levantar fundos disponíveis para a sua conta bancária.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Saiba mais
</LinkButton>
