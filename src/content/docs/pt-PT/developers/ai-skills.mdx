As competências de IA são pacotes de instruções portáteis que ensinam agentes de codificação a trabalhar com a Wink. Em vez de depender de dados de treino ou documentação desatualizada, cada competência orienta o agente a usar o [servidor MCP da Wink](/pt-PT/developers/mcp) como a fonte de verdade em tempo real para contratos de API e documentação da plataforma.

As competências funcionam com qualquer cliente de agente que as suporte — Claude Code, Codex, Cursor e outros.

## Competências Disponíveis

### wink-api-builder

Implemente, reveja ou depure integrações da API da Wink. A competência guia o agente através de um fluxo de trabalho estruturado:

1. Pesquisar a operação API correta via `api_search`
2. Abrir os contratos de pedido/resposta em tempo real do servidor MCP
3. Obter a configuração de autenticação a partir da documentação
4. Escrever código com base no contrato verificado
5. Assinalar quaisquer discrepâncias entre o código existente e a API atual

Útil para:
- Construir novos métodos cliente para qualquer API da Wink
- Depurar discrepâncias de contrato (ex. um pedido que retorna HTTP 400)
- Rever integrações existentes em relação à especificação em tempo real

## Instalar

Copie a pasta da competência para o diretório de competências do seu agente.

:::tip
O servidor MCP da Wink deve estar ligado no mesmo ambiente. Veja [Servidor MCP](/pt-PT/developers/mcp) para instruções de configuração.
:::

**Claude Code**

```bash
# local ao repositório
cp -r skills/wink-api-builder .claude/skills/

# ou a nível de utilizador
cp -r skills/wink-api-builder ~/.claude/skills/
```

Invoque com:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# local ao repositório
cp -r skills/wink-api-builder .agents/skills/

# ou a nível de utilizador
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Invoque com:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Outros agentes**

Coloque a pasta da competência onde quer que o seu agente resolva competências. O ponto de entrada é `SKILL.md`.

## Requisitos

- Um [servidor MCP da Wink](/pt-PT/developers/mcp) ligado
- Um cliente de agente com suporte a MCP e competências
