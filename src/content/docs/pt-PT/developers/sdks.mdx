Acelere o tempo de colocação no mercado utilizando um dos nossos Kits de Desenvolvimento de Software de Línguas oficialmente suportados.
Se atualmente não suportamos o seu idioma favorito, aceda à API que pretende utilizar e descarregue a especificação OpenAPI que disponibilizamos para cada API no topo da página. Pode gerar o seu próprio SDK bifurcando um dos nossos repositórios públicos de SDK ou utilizando um dos geradores OpenAPI.

## Java

Esta biblioteca foi compilada para Java 24. Está configurada para utilizar o WebClient reativo e está otimizada para utilização com o Spring Framework. No entanto, o Spring não é um requisito, mas suporta OAuth2 pronto a usar.
