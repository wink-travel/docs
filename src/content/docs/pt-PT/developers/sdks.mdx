Acelere o tempo de lançamento no mercado utilizando um dos nossos Kits de Desenvolvimento de Software oficialmente suportados para linguagens.
Se atualmente não suportamos a sua linguagem favorita, vá para a API que pretende usar e descarregue a especificação OpenAPI que disponibilizamos para cada API no topo da página. Pode gerar o seu próprio SDK quer fazendo fork de um dos nossos repositórios públicos de SDK ou utilizando um dos geradores OpenAPI.

## Java

Esta biblioteca foi compilada para Java 24. Está configurada para usar o WebClient reativo e está otimizada para uso com o Spring Framework. No entanto, o Spring não é um requisito, mas oferece suporte OAuth2 imediatamente.
