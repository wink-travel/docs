Para qualquer pessoa que utilize o WordPress para o seu negócio ou blogs, recomendamos vivamente o uso do [Plugin WordPress de afiliados Wink](https://wordpress.org/plugins/wink2travel/).
O plugin dá-lhe um acesso fácil aos seus cartões Wink, grelhas, mapas, etc., e torna tudo tão simples como arrastar conteúdo para os seus posts e páginas utilizando o seu construtor de páginas favorito ou os nossos códigos de acesso personalizados.

## Instalar

É assim que se instala o `Wink Affiliate WordPress Plugin` no seu site WordPress.

<Steps>
  1. Quando estiver ligado como administrador e no Painel.
  2. Clique `Plugins > Add New Plugin`.
  3. Procurar `Wink Affiliate WordPress Plugin`.
  4. Clique `Install` para descarregar e instalar o plugin no seu site.
  5. Clique `Activate` para ativar o plugin no seu website.
  6. A ativação encaminhar-lhe-á o `Wink Settings` página. *Também pode ir até lá navegando até `Appearance > Customize > Wink Settings`.*
  7. Ele irá pedir ao seu `Client ID` e `Client Secret`. Vá para [Aplicações](/developers/applications) para saber como recuperar as suas credenciais.
  8. Introduza as suas credenciais e clique `Publish`. Agora está ligado à sua conta Wink.
</Steps>

## Construtores de páginas

Suportamos os criadores de páginas mais populares no WordPress, o que facilita a utilização do Wink para qualquer pessoa que o queira utilizar.

### Gutenberg

Gutenberg é o editor nativo do WordPress.

### Avada

[Avada](https://avada.com) é famoso pela sua grande seleção de belos modelos, bem como por um construtor de páginas robusto com um conjunto de características avançadas.

### Elementor

[Elementor](https://elementor.com) é o construtor de páginas mais popular \[atualmente], com versões gratuitas e profissionais disponíveis.

### Padaria WP

[Padaria WP](https://wpbakery.com) foi o primeiro construtor de páginas avançado do mercado.

## Códigos curtos

### \[piscadela]

Funciona com grelha de conteúdo classificado. Introduza um local que deseja visitar e que o inventário seja apresentado.

### \[pesquisa de piscar de olhos]

Botão simples para abrir o formulário de itinerário.

### \[conta piscadela]

Botão Conta para permitir a autenticação. Uma vez autenticado, transforma-se num menu suspenso com opções de conta.

### \[winkitinerário]

Igual ao botão de pesquisa, só que contém informação do itinerário como texto do botão.

### \[piscadela de conteúdo]

Permite incorporar o inventário selecionado no Wink diretamente numa página ou publicação. Este shortcode também espera um ID de layout.

O layout

```
[winkcontent layoutid=""]
```
