Esta página lista todos os eventos webhook aos quais pode subscrever na Wink. Para instruções de configuração, consulte o [Guia de Integração Webhook](/pt-PT/guides/webhook-integration) ou a [página de gestão de Webhooks](/pt-PT/my-account/webhooks).

## Como subscrever

1. Navegue até **Aplicações > Webhooks** no seu portal.
2. Crie um webhook e selecione os eventos que deseja receber.
3. A Wink envia um HTTP POST para o seu endpoint cada vez que o evento ocorre.

Para passos completos de configuração e melhores práticas, consulte [Integração Webhook](/pt-PT/guides/webhook-integration).

## Eventos de conta

Eventos relacionados ao ciclo de vida e configurações da sua conta TripPay.

| Evento | Chave | Descrição |
|---|---|---|
| Conta criada | `account.create` | Uma nova conta foi criada. |
| Conta em revisão | `account.under_review` | Conta está pendente de revisão. |
| Conta atualizada | `account.updated` | Os detalhes da conta foram alterados. |
| Estado da conta atualizado | `account.status.updated` | O estado da conta foi alterado (ex.: aprovado, suspenso). |
| Morada da conta atualizada | `account.address.updated` | A morada da conta foi modificada. |
| Detalhes da conta atualizados | `account.details.updated` | Os campos de detalhes da conta foram alterados. |
| Informação da conta atualizada | `account.information.update` | Informação geral da conta foi atualizada. |
| Gestores da conta atualizados | `account.managers.update` | A lista de gestores da conta foi alterada. |
| Foto de perfil atualizada | `account.profile_picture.updated` | A foto de perfil da conta foi alterada. |
| Agente atualizado | `account.agent.updated` | Informação do agente na conta foi modificada. |
| Nome URL atualizado | `account.url-name.update` | A URL personalizada da conta foi alterada. |
| ID único atualizado | `account.unique-id.update` | O identificador único da conta foi modificado. |
