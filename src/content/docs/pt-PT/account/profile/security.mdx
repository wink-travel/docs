A aba **Segurança** em [iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security) é onde gere as **passkeys** — credenciais WebAuthn que lhe permitem iniciar sessão sem palavra-passe.

## O que é uma passkey?

Uma passkey é uma credencial criptográfica associada ao seu dispositivo (ou gestor de palavras-passe) que prova que é você. As passkeys são resistentes a phishing e mais rápidas do que digitar uma palavra-passe.

## Registar uma passkey

1. Inicie sessão em [iam.wink.travel](https://iam.wink.travel) com as suas credenciais existentes.
2. Vá a **A Minha Conta → Segurança**.
3. Clique em `Adicionar passkey`.
4. O seu navegador irá pedir-lhe para autenticar com o seu dispositivo (Touch ID, Face ID, Windows Hello, uma chave de hardware ou um gestor de palavras-passe sincronizado).
5. Dê um nome amigável à passkey (ex. *MacBook do Trabalho*) para a poder reconhecer mais tarde.

## Usar uma passkey para iniciar sessão

No ecrã de início de sessão do [iam.wink.travel](https://iam.wink.travel), introduza o seu email — se tiver uma passkey registada para esse email, será solicitado que se autentique com ela em vez de digitar uma palavra-passe.

## Remover uma passkey

Na mesma aba Segurança, cada passkey registada tem um botão `Remover`. Use-o se perder um dispositivo, deixar de usar uma chave de hardware ou quiser limpar credenciais não utilizadas.

:::caution
Remover todas as suas passkeys deixa a palavra-passe como o único método de início de sessão. Certifique-se de que se lembra da sua palavra-passe antes de remover a última passkey.
:::
