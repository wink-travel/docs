# Definições

Quando começar a usar o Social, iremos perguntar-lhe como pretende que criemos o seu conteúdo. Estas opções estão sempre disponíveis para que possa alterá-las facilmente mais tarde.

Navegue até `Preferências` na barra lateral para gerir as suas definições.

A secção de preferências contém as suas definições básicas de geração de conteúdo:

1. **Personalização preferida** Escolha qual personalização usar ao gerar conteúdo. As personalizações controlam as cores do tema, itinerário padrão, moeda, idioma e mais. Pode criar personalizações na sua conta em [my.wink.travel](https://my.wink.travel).
2. **Aprovação automática** Quando ativada, as publicações geradas são publicadas automaticamente sem necessidade da sua aprovação. Ative esta opção quando estiver confiante na qualidade do conteúdo que produzimos para si.
3. **Nome do aprovador** O nome da pessoa ou equipa que receberá os e-mails de pedido de aprovação.
4. **E-mail do aprovador** O endereço de e-mail para onde serão enviados os pedidos de aprovação para as publicações futuras.

## Criadores de conteúdo

<figure>
<CldImage
  src="academy/social_content_creators"

  <figcaption>*Fig. 2 - Criadores de conteúdo*</figcaption>

O Social oferece muitos criadores de conteúdo para escolher. Cada criador foca-se numa forma específica de criar conteúdo único para a sua propriedade usando IA. Cada criador de conteúdo tem um atributo **Peso** — um peso mais alto aumenta a probabilidade de esse criador ser selecionado ao gerar publicações.

Aqui estão todos os criadores de conteúdo disponíveis:

| Criador | Descrição |
| -- | -- |
| **Destino** | Usa a sua localização para criar texto útil sobre a área |
| **Tipo de quarto** | Cria uma publicação reservável com um dos seus tipos de quarto com texto gerado por IA e imagens do seu tipo de quarto |
| **Atividade** | Cria uma publicação reservável com uma das suas atividades |
| **Complemento** | Cria uma publicação reservável com um dos seus complementos |
| **Atração** | Cria uma publicação reservável com uma das suas atrações |
| **Local** | Cria uma publicação reservável com um dos seus locais |
| **Sala de reuniões** | Cria uma publicação reservável com uma das suas salas de reuniões |
| **Restaurante** | Cria uma publicação reservável com um dos seus restaurantes |
| **Spa** | Cria uma publicação reservável com um dos seus spas |
| **Propriedade** | Cria uma publicação reservável com o seu quarto com melhor preço, com texto gerado por IA e imagens da sua propriedade |
| **Sentimento do utilizador** | Agrega avaliações de hóspedes de toda a web numa publicação orientada por sentimentos que destaca o que os hóspedes adoram na sua propriedade |
| **Divertido** | Cria publicações leves e divertidas para manter o seu feed envolvente |

:::tip[Dica profissional]
Certifique-se de que tem o tipo de inventário necessário para os criadores de conteúdo escolhidos. Por exemplo, se selecionar "Restaurante", precisa de ter pelo menos um restaurante listado na sua propriedade.
:::

## Tom de voz

<figure>
<CldImage
  src="academy/social_personas"

  <figcaption>*Fig. 3 - Tom de voz*</figcaption>

O tom de voz define o estilo de linguagem que a IA usa ao gerar conteúdo. Escolha os tons que correspondem à personalidade da sua marca. Tal como nos criadores de conteúdo, cada tom tem um **Peso** para controlar a frequência com que é usado.

Aqui estão os tons disponíveis:

| Tom | Estilo |
| -- | -- |
| **Neutro** | Calmo, autoritário, factual. Sem exageros, sem piadas — apenas conhecimento fiável e bem estruturado |
| **Amigável** | Caloroso, acessível, energia sorridente através do texto com contrações casuais e emojis ocasionais |
| **Luxo** | Sofisticado, refinado. Palavras como "exclusivo" e "personalizado" que fazem os leitores sentirem-se especiais |
| **Gastronómico** | Descrições sensuais e de dar água na boca. Linguagem de food porn que faz os leitores ficarem com fome |
| **Aventureiro** | Ousado, intrépido, energia exploradora. Descrições sensoriais vívidas que tornam tudo épico |
| **Familiar** | Entusiasmo saudável e adequado para crianças. Energia calorosa e parental sobre experiências seguras e divertidas |
| **Romântico** | Poético, nostálgico, que faz o coração bater mais forte. Pôr do sol, linguagem suave e amor |
| **Humorístico** | Sarcástico, frases rápidas, jogos de palavras inteligentes. Lê-se como um espetáculo de stand-up |
| **Ecológico** | Sincero e justo. Celebra alternativas verdes e escolhas sustentáveis |
| **Insider local** | Referências locais muito específicas, alerta para armadilhas turísticas, usa gírias locais |
| **Inclusivo** | Acolhedor, celebratório e afirmativo. Faz todos sentirem-se vistos e valorizados |
| **Místico** | Etéreo, espiritual, verdades cósmicas. Convites suaves para olhar para dentro de si |
| **Futurista** | Elegante, visionário. Adora termos como "mudança de paradigma" e visões otimistas do futuro |
| **Económico** | Prático, engenhoso, orgulhosamente frugal. Celebra descobertas acessíveis e experiências gratuitas |

## Tipos de publicação

Os tipos de publicação determinam o objetivo e a estrutura do conteúdo gerado. Cada tipo de publicação tem um **Peso** para controlar a frequência com que é usado.

| Tipo de publicação | Descrição |
| -- | -- |
| **Transacional** | Claro, conciso, focado na conversão. Impulsiona reservas e ações diretas com factos, preços e próximos passos |
| **Consciencialização** | Conteúdo para construção de marca destinado a apresentar a sua propriedade e aumentar a visibilidade entre potenciais hóspedes |
| **Sentimento** | Conteúdo emocionalmente ressonante que cria associações positivas e constrói uma ligação emocional com o seu público |

Pode querer focar os seus esforços de marketing num idioma e região específicos. Diga-nos os idiomas para os quais quer criar conteúdo e o fuso horário que lhe interessa e nós tratamos do resto. As publicações são publicadas na hora agendada no fuso horário selecionado.

*A maioria das propriedades usa um único idioma e define o idioma local como padrão.*

## Google Business

<figure>
<CldImage
  src="academy/social_google_business"

  height="292"
  alt="Google Business do Social"
/>
  <figcaption>*Fig. 5 - Google Business*</figcaption>

Ao ligar o seu Perfil Google Business, podemos obter detalhes mais ricos da propriedade para um conteúdo mais eficaz. A sua propriedade precisa de ter um Perfil Google Business verificado para que possamos gerar conteúdo a partir da sua listagem. Alguns dos nossos criadores de conteúdo **exigem** que tenha uma conta Google Business.

:::tip[Dica profissional]
Comece pequeno

Não seja demasiado ambicioso na sua primeira tentativa. Recomendamos que comece por criar conteúdo para a sua propriedade num único idioma.
:::
