O editor de publicações permite-lhe criar publicações manuais ilimitadas e é também o local onde gere as publicações que foram criadas automaticamente para si.

O editor tem 4 secções:

1. A secção de conteúdo.
2. A secção de media.
3. A secção de ligação.
4. A secção de definições.

## Conteúdo

A secção de conteúdo permite-lhe modificar o texto antes de ser publicado. Pode criar conteúdo localizado em várias línguas — cada língua tem o seu próprio título e descrição.

## Media

A secção de media permite-lhe modificar as imagens e vídeos que acompanham o texto.

## Ligação

A secção de ligação permite-lhe adicionar um link para onde deseja que o utilizador vá após ler a sua publicação. *Por exemplo, pode ser um link para uma oferta especial ou um local para obter mais informações.* Pode adicionar uma URL de ligação, uma legenda para o link e uma descrição do link.

## Definições

A secção de definições permite-lhe modificar / substituir todas as suas definições predefinidas por publicação. Pode definir:

- **Plataforma** — Escolha a rede social onde esta publicação será publicada.
- **Tipo de publicação** — Escolha entre Transacional, Sensibilização ou Sentimento.
- **Tom de voz** — Substitua o tom de voz para esta publicação específica.
- **Personalização** — Utilize uma personalização diferente para moeda de exibição, idioma, itinerário, etc.
- **Fuso horário** — Substitua o fuso horário para a publicação.
- **Data agendada** — As publicações geradas são automaticamente agendadas para o momento ideal. Pode substituir a data e hora se preferir outro horário.

## Ciclo de vida do estado da publicação

Cada publicação segue um ciclo de vida de estado:

| Estado | Descrição |
| -- | -- |
| **Rascunho** | A publicação foi criada mas ainda não está pronta para revisão |
| **Pendente de aprovação** | A publicação está à espera de aprovação humana antes de ser agendada |
| **Aprovada** | A publicação foi aprovada e será agendada |
| **Rejeitada** | A publicação foi revista e rejeitada — não será publicada |
| **Agendada** | A publicação está na fila para publicação na data e hora agendadas |
| **Publicada** | A publicação foi publicada com sucesso na rede social |
| **Pausada** | A publicação foi temporariamente pausada |
| **Falhou** | A publicação não pôde ser publicada devido a um erro |

## Aprovar publicações

As publicações são aprovadas na grelha de publicações onde pode selecionar uma ou mais publicações e escolher aprovar ou rejeitar.

:::note
Haverá uma entrada para cada língua e cada rede social.

Por exemplo, se tiver 3 redes sociais e 3 línguas selecionadas, verá 9 entradas no calendário.
:::
