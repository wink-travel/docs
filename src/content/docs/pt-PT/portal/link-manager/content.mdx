O seu conteúdo está disponível no separador `Entradas`.

Suportamos três tipos principais de conteúdo:

- URLs publicamente disponíveis que ligam a páginas web.
- Ficheiros que foram carregados através do WinkLinks.
- Conteúdo de texto criado através da nossa funcionalidade `Citações`.

## Adicionar conteúdo

Aqui está como adicionar conteúdo ao WinkLinks. No separador `Entradas`, comece por clicar no botão `Adicionar entrada`.

### Ligação normal

:::tip
A ligação tem de direcionar para uma página web. Por exemplo, não pode ligar diretamente a um PDF. Se quiser partilhar um PDF, precisa de usar a funcionalidade `Carregar ficheiro`.
:::

Aqui estão os passos para adicionar uma ligação normal:

<Steps>
1. Introduza a ligação que quer partilhar. *ex. https://trvl.as/3xWCH*
2. Depois de introduzir a ligação, o Wink procura os detalhes deste website e preenche o formulário que vê a seguir.
3. Pode optar por modificar alguns dos dados, como o nome do site, título e descrição.
4. Pode também optar por carregar uma imagem personalizada.
5. Clique no botão `Guardar` quando estiver pronto para continuar.
</Steps>

### Carregar ficheiro

:::caution[Requisitos]
- As imagens não podem exceder 10Mb de tamanho.
- Os vídeos não podem exceder 50Mb de tamanho.
- Suportamos todos os formatos reconhecidos de imagem e vídeo, incluindo PDF.
:::

:::note[Recomendações]
- 2560w x 1440h (proporção 16x9) para a sua imagem em destaque.
- 2560w x 1920h (proporção 4:3) para todas as outras imagens.
- O tamanho da imagem não deve ser superior a 5Mb. Por favor, comprima as suas imagens antes de as carregar para o Wink.
:::

Aqui estão os passos para adicionar um ficheiro:

<Steps>
1. Pode adicionar media:
    - Arrastando imagens do seu computador para a janela ou clicando em `Procurar`.
    - Colando uma URL externa que aponte para uma imagem.
    - Usando a câmara do seu portátil.
    - A partir da sua conta Google Drive.
    - A partir da sua conta Dropbox.
    - A partir do Shutterstock
    - A partir do gettyimages
    - A partir do iStock
    - A partir do Unsplash
2. Depois de o ficheiro terminar de carregar, pode adicionar mais metadados para descrever o seu ficheiro, como título e descrição.
3. Clique no botão `Guardar` quando estiver pronto para continuar.
</Steps>

### Escrever uma citação

:::tip
Uma citação é uma ótima forma de adicionar um delimitador para dividir uma lista longa [...e também faz com que pareça inteligente e conhecedor 😉].
:::

Aqui estão os passos para escrever uma citação:

<Steps>
1. Escreva a citação *ex. Rosas são vermelhas. Violetas são azuis...*.
2. Introduza o nome do autor original. *ex. Edmund Spenser*
3. Introduza o nome do site onde obteve a citação. *ex. Daily Quotes*
4. Clique no botão `Guardar` quando estiver pronto para continuar.
</Steps>

## Atualizar conteúdo

Todo o conteúdo pode ser facilmente modificado clicando no link `Ações` e selecionando `Atualizar`. Pode remover conteúdo de forma semelhante clicando no botão `Remover`.

## Remover conteúdo

Todo o conteúdo pode ser facilmente modificado selecionando `Atualizar`. Pode remover conteúdo de forma semelhante clicando primeiro no link `Ações` e depois no botão `Remover`.

## Ordenar conteúdo

Pode ordenar o seu conteúdo clicando no ícone localizado no canto inferior esquerdo de cada entrada e arrastando-o para a posição que desejar.

## Agrupar conteúdo

:::tip
Crie as suas etiquetas antes de criar as suas entradas. Caso contrário, perde o progresso e tem de começar de novo.
:::

Quando está a editar uma entrada, tem a opção de adicionar uma ou mais etiquetas para fins de agrupamento. Desça até ao final do formulário para encontrar o campo de formulário `Etiquetas`.

Os utilizadores, ao verem a sua página WinkLinks, podem escolher como querem consumir o seu conteúdo (lista ou grelha) e podem selecionar uma ou mais etiquetas para filtrar ainda mais o seu conteúdo.

## API

Os desenvolvedores que querem gerir o `WinkLinks` podem aceder a [Developers > API > WinkLinks](/pt-PT/api).
