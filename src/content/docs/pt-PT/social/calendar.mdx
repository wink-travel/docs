A publicação funciona assim:

<Steps>
1. O conteúdo é gerado a partir das suas preferências uma semana antes do mês de publicação.
2. Recebe um e-mail a pedir para aprovar o conteúdo recém-criado [ou a informar que o conteúdo foi criado para o mês seguinte e que não precisa de fazer mais nada porque escolheu aprovar automaticamente o conteúdo].
3. Inicia sessão em [social.wink.travel](https://social.wink.travel) para aprovar as publicações.
4. As publicações são agendadas para serem publicadas. (Ainda pode parar o trabalho de publicação se o fizer antes de ser publicado. Caso contrário, terá de ir à rede social e eliminar ou modificar a publicação manualmente lá.)
5. As publicações são feitas no momento mais oportuno para o fuso horário selecionado.
</Steps>

Depois de o conteúdo ser aprovado para publicação, ele é adicionado ao seu calendário.

:::note
Haverá uma entrada para cada idioma e para cada rede social.

Por exemplo, se tiver 3 redes sociais e 3 idiomas selecionados, verá 9 entradas no calendário.
:::
