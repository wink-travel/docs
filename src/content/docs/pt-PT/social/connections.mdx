Já está ligado ao WinkLinks, a nossa ferramenta de gestão de links. Ligue-se a mais redes sociais clicando em `Ligar uma rede social`.
O processo irá redirecioná-lo para a sua rede social para aprovar a aplicação `Wink Social` para publicar conteúdo em seu nome.

Assim que estiver ligado, começamos a gerar conteúdo para essa rede para o próximo mês agendado.

:::note
O plano gratuito inclui o WinkLinks. Facebook, Instagram e outras plataformas estão disponíveis nos planos pagos.
:::

## Plataformas suportadas

| Plataforma | Descrição | Requisitos |
| -- | -- | -- |
| **WinkLinks** | Publicação automática para o nosso avançado gestor de links | Incluído em todos os planos |
| **Facebook** | Automatiza publicações, respostas a comentários e mensagens privadas | Requer uma Página de Negócios no Facebook |
| **Instagram** | Automatiza publicações, respostas a comentários e mensagens privadas | Requer uma Conta Comercial do Instagram ligada a uma conta do Facebook |

Cada ligação mostra o seu estado atual: **Ativa**, **Desligada** ou **Suspensa**.

:::note
Criamos e publicamos conteúdo otimizado para cada rede social.
:::
