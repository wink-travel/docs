You can create a user account on any of our portals. Once created, this account can be used across all of Wink's sites.

## Travelers

Follow these steps:

<Steps>
1. Head on over to [https://ota.wink.travel](https://ota.wink.travel)
2. Click on `Sign-In / Register` in the top-right corner
3. You will be asked to enter basic user details. Fill in all required fields and click `Save`.
4. You can now log into Wink with your username (e-mail) and password.
5. Once logged in, the system redirects you back to where you came from.
</Steps>
