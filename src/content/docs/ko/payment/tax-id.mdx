일부 지역에서는 지급금을 받기 전에 국가 세금 식별 번호가 필요합니다. TripPay는 계정 설정 과정에서 이 단계를 완료하도록 안내합니다.

## 세금 ID 제출하기

요청 시 국가 세금 식별 번호를 입력하세요:

- **미국 개인** — 사회보장번호(SSN). *형식: 111-11-1111*
- **미국 기업** — 고용주 식별 번호(EIN). *형식: 11-111111*
- **기타 지역** — 해당 국가의 동등한 국가 식별 번호.

세금 ID를 입력한 후, 제출 전에 정보가 정확한지 확인하라는 요청을 받게 됩니다.

:::note
이 단계는 KYC(고객 알기) 준수를 위해 한 번만 필요합니다. 지역에 따라 추가 서류 제출을 요청받을 수 있습니다.
:::
