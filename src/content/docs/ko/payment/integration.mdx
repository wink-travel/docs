여행사이고 결제 및 지급 업무를 TripPay에 아웃소싱하려는 경우, 시작하는 데 필요한 6가지 기본 단계는 다음과 같습니다:

<Steps>
1. Wink에서 [사용자 계정](/ko/user/register)을 만듭니다.
2. [TripPay](https://pay.wink.travel)에서 계정을 만듭니다.
3. TripPay에서 API와 통신할 수 있는 [애플리케이션](/ko/developers/applications)을 만듭니다.
4. TripPay와 수취인을 생성하고 매핑합니다. *예: 대부분의 경우, TripPay가 결제할 플랫폼 내 숙소 계정을 생성하는 것을 의미합니다.*
5. 사이트에 [결제 웹 컴포넌트](/ko/developers/web-components/#payment)를 통합합니다.
6. 여행자의 `쇼핑 카트`를 TripPay에 전송하고 결제 프로세스를 시작합니다.
</Steps>

:::tip
여행 사이트에서 TripPay를 사용하는 완전한 가이드는 [Integrating with TripPay](/ko/guides/integrating-with-trip-pay) 가이드를 참조하세요.
:::

## API

`Payments`를 관리하려는 개발자는 [Developers > API > Payments](/ko/developers/apis/#payment-api)로 이동하세요.
