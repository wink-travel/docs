Studio는 예약을 유도할 수 있도록 웹사이트에 추가할 수 있는 여러 임베드 위젯을 제공합니다.

임베드 위젯에 접근하려면 사이드바에서 `Tools > Miscellaneous`로 이동하세요.

## 사용 가능한 위젯

| 위젯 | 설명 |
| -- | -- |
| **Account button** | 제휴 페이지로 연결되는 브랜드 버튼 |
| **Itinerary button** | 일정 선택기를 여는 버튼 |
| **Itinerary form** | 인라인 날짜 및 게스트 선택 양식 |
| **Search button** | 숙소 검색을 여는 버튼 |
| **Lookup button** | 특정 인벤토리를 조회하는 버튼 |

## 임베드 방법

각 위젯은 웹사이트 HTML에 복사하여 붙여넣을 수 있는 코드 스니펫을 제공합니다. 스니펫에는 **Client ID**와 **Customization ID**가 포함되어 있어 위젯이 자동으로 제휴 계정과 연결됩니다.

:::note
WordPress를 사용하는 경우, 더 쉬운 통합을 위해 WordPress 플러그인도 제공합니다.
:::
