저장된 검색은 이름 그대로의 기능을 합니다. 여행 인벤토리를 검색하는 동안, 나중에 사용할 수 있도록 검색 쿼리를 저장할 수 있는 옵션이 있습니다.

## 검색 쿼리 저장하기

인벤토리 검색을 시작할 때(자세한 내용은 [Search](/ko/studio/search) 참조), 검색 필터를 세밀하게 조정할 수 있습니다.  
쿼리 폼 하단에는 `검색 저장` 옵션이 있습니다.

이 링크를 클릭하면 이 검색에 이름을 지정할 수 있는 기회가 주어집니다(*예: 나의 마법 호텔 라이브러리*), 나중에 기억하기 쉽도록 말이죠.

이름을 지정한 후 `검색 저장` 버튼을 클릭하세요. 그 후에는 메인 내비게이션 바에서 `인벤토리 > 저장된 검색`으로 이동하여 새로 생성된 기록을 확인할 수 있습니다.

## 목록 작업

저장된 검색은 여행 인벤토리를 그리드 또는 지도 형태로 사용자에게 노출하고, 원하는 방식으로 검색 결과를 표시하는 데 사용할 수 있습니다.

모든 저장된 검색 쿼리에는 실행할 수 있는 작업이 있습니다:

- 검색 쿼리를 [지도](/ko/studio/maps)로 변환합니다.
- 검색 쿼리를 [그리드](/ko/studio/grids)로 변환합니다.
- 저장된 검색을 삭제합니다.

## API

저장된 검색을 관리하려는 개발자는 [Developers > API > Browse](/ko/developers/apis/#browse-api)로 이동하세요.
