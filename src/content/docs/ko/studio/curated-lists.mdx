선별된 목록은 여행 재고를 검색하는 동안 찾은 항목 중에서 선택한 재고를 그룹화하고 정렬할 수 있는 기능입니다.
목록이 생성되면, [WinkLinks](/ko/link-manager/wink-links), 당사의 [WordPress 플러그인](/ko/developers/wordpress/), 또는 당사의 [웹 컴포넌트](/ko/developers/web-components) 중 하나를 사이트에 임베드하여 [그리드](/ko/studio/grids) 또는 [지도](/ko/studio/maps) 형태로 판매 가능하도록 사용자에게 노출할 수 있습니다.

기본적으로 모든 제휴사 계정은 즉시 사용할 수 있는 `Favorites` 목록을 가지고 시작합니다. `Favorites`는 특별하며 삭제할 수 없습니다.
모든 선별된 목록에서는 다음 작업을 할 수 있습니다:

## 목록 생성

- 대시보드에서 네비게이션 바의 `Inventory > Curated Lists`를 클릭합니다.
- `Make a new list` 버튼을 클릭하여 새 선별된 목록을 만듭니다.
- 목록 이름을 입력하라는 메시지가 표시됩니다. *예시: Top 3 NYC Picks*.
- `Save` 버튼을 클릭합니다.

## 목록 항목 추가

검색 결과에서 어떤 목록에도 항목을 추가할 수 있습니다.

- [Search](/ko/studio/search)로 이동합니다.
- 목록에 추가할 항목을 검색합니다.
- 찾으면, 검색 결과 카드 왼쪽 하단의 `Actions` 링크를 클릭합니다.
- `Add to...`를 클릭한 후 이 재고를 추가할 선별된 목록을 선택합니다.
- 목록에 항목이 추가되었는지 확인하려면 선별된 목록 페이지로 돌아갑니다.

모든 목록에는 실행할 수 있는 작업이 있습니다.

- 목록을 [지도](/ko/studio/maps)로 변환합니다.
- 목록을 [그리드](/ko/studio/grids)로 변환합니다.
- 목록을 제거합니다 (*Favorites 제외*).

## API

선별된 목록을 관리하려는 개발자는 [Developers > API > Browse](/ko/developers/apis/#browse-api)로 이동하세요.

## 추가 읽을거리

- 당사의 [그리드](/ko/studio/grids) 기능에 대해 읽어보세요.
- 당사의 [지도](/ko/studio/maps) 기능에 대해 읽어보세요.
