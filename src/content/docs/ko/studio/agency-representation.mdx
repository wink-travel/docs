대행사는 대리 목적으로 귀하의 계정에 할당할 수 있는 일반 Wink 계정입니다. 대리 대행사가 필요한 몇 가지 이유는 다음과 같습니다.

* 제3자가 귀하의 Wink 계정을 전체/일부 관리합니다.
* Talent는 자사 에이전시와 이미 계약을 맺은 상태이며, Wink가 Wink를 통해 들어오는 자금의 재정적인 측면을 처리해 주기를 원합니다.
* 인재 에이전시는 모든 온라인 업무를 처리하고 Wink의 모든 계정을 관리합니다.

## 대행사 배정

계정에 대행사를 지정하는 방법은 다음과 같습니다.

<Steps>
  1. 사용자 계정에 로그인하세요 [윙크 스튜디오](https://studio.wink.travel).
  2. 작업하려는 제휴 계정을 선택하세요.
  3. 클릭해주세요 `Account > Managers` 메인 탐색 바에서.
  4. 에서 `Agency` 섹션에 귀하를 대리해 줄 기관의 이름을 입력하세요.
  5. 드롭다운에서 중개 수수료를 선택하세요.
  6. 선택적으로 만료 규칙을 추가합니다.
  7. 클릭 `Set agency` 계속하려면 버튼을 클릭하세요.
</Steps>


:::참고
대행사를 지정하려면 계정 소유자여야 합니다.
:::

:::팁\[예시]
에이전시 수수료는 제휴 수수료에서 발생합니다.

예약 = 100달러, 수수료 = 10%, 중개 수수료 = 10% => 100 \* 0.1 \* 0.1 = 1달러
:::

## 대행사 할당 해제

계정에서 에이전시 할당을 해제하는 방법은 다음과 같습니다.

<Steps>
  1. 사용자 계정에 로그인하세요 [윙크 스튜디오](https://studio.wink.travel).
  2. 작업하려는 제휴 계정을 선택하세요.
  3. 클릭해주세요 `Account > Managers` 메인 탐색 바에서.
  4. 에서 `Agency` 섹션을 클릭하세요 <Badge text="x" variant="danger" size="small" /> 에이전시 드롭다운의 오른쪽에 있는 아이콘.
  5. 클릭 `Set agency` 계속하려면 버튼을 클릭하세요.
</Steps>
