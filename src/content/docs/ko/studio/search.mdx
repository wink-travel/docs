## 검색 시작하기

Wink에서 공급자와 인벤토리를 검색하는 방법은 다음과 같습니다.

1. 작업할 제휴 계정을 [계정 목록](https://studio.wink.travel/company/list)에서 선택하세요.
2. 검색 페이지로 이동하는 두 가지 방법이 있습니다:
    1. 대시보드에서 바로 숙소를 빠르게 검색하거나...
    2. 메인 내비게이션 바에서 Inventory → Search를 클릭하세요.
3. 검색 페이지에는 Wink의 최신 공급자와 인벤토리가 카테고리와 함께 표시됩니다.
4. 예시로, `Hotels`라고 적힌 카드를 클릭하세요.
5. 검색 결과 페이지로 리디렉션됩니다. 여기서 `Hotels` 카테고리 내 사용 가능한 공급자를 스크롤하며 왼쪽 필터를 사용해 검색 필터를 적용하거나 변경할 수 있습니다.

## 검색 결과 조정하기

인벤토리와 공급자를 필터링할 수 있는 모든 방법입니다. 검색 결과 오른쪽 상단의 필터 아이콘을 클릭하세요.

- 숙소 이름 (*예: Centara*)
- 인벤토리 이름 (*예: Villa*)
- 인벤토리 유형 (*예: Spas*)
- 별점 (*예: 5성급 숙소*)
- 사용자 리뷰 평점 (*예: 8 / 10*)
- 지리 위치 (*예: 지도 위치 선택*)
- 대륙 (*예: 북미*)
- 국가 (*예: 독일 또는 프랑스*)
- 도시 (*예: 도쿄*)
- 라이프스타일 (*예: LGBTQ*)
- 위치 유형 (*예: 공항*)
- 세그먼트 유형 (*예: 중간급*)
- 카테고리 유형 (*예: 롯지*)
- 스타일 (*예: 아르데코*)
- 직접 관계만 (*예: 직접 인벤토리만 표시*)
- 반려동물 친화적 (*예: 반려동물 친화적 인벤토리 표시*)
- 친환경 (*예: 친환경 인벤토리 표시*)
- 아동 친화적 (*예: 아동 친화적 인벤토리 표시*)
- 인기 (*예: 가치 높은 인벤토리 표시*)

## 작업

작업은 인벤토리 또는 공급자와 상호작용할 수 있게 해줍니다. 다음 세 가지 주요 카테고리가 있습니다:

1. **공급자 세부 정보** 공급자에 대해 더 알고 싶을 때
2. **생성...** 이 인벤토리를 판매 가능한 형태로 전환할 때
3. **추가...** 이 인벤토리를 동시에 판매할 수 있는 목록에 저장할 때

`Supplier details`를 클릭하면 다음을 할 수 있습니다:

- **예약하기** 제휴 ID를 사용해 숙소로 직접 이동하여 예약하고 수수료를 적립하세요.
- **직접 연결 요청** 공급자에게 직접 연결을 요청할 수 있습니다.
- **성과 추적** 공급자의 성과를 시간에 따라 추적하는 분석 차트를 만드세요.

## 결과 없음

특정 공급자를 찾고 있지만 찾을 수 없는 경우, Google Places에서 찾을 수 있도록 입력 필드가 표시됩니다. Google Places에 존재하면 Wink에서 `리드`를 생성할 수 있습니다.
이는 누군가가 이 공급자에게 연락해 Wink에 가입하도록 요청한다는 의미입니다. 저희가 공급자에게 연락하여 가입을 요청하며, 가입 시 알림을 받게 됩니다.
