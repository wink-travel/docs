## 검색을 시작하세요

Wink에서 공급업체와 재고를 검색하는 방법은 다음과 같습니다.

1. 작업하려는 제휴 계정을 선택하세요.[계정 목록](https://studio.wink.travel/company/list)
2. 다음 두 가지 방법 중 하나로 검색 페이지로 이동합니다.
   1. 대시보드에서 바로 부동산을 빠르게 검색할 수 있습니다. 또는...
   2. 메인 탐색 표시줄에서 인벤토리 → 검색을 클릭합니다.
3. 검색 페이지에는 Wink의 최신 공급업체와 재고가 표시되며 선택할 수 있는 카테고리도 제공됩니다.
4. 우리의 예에서, 계속해서 카드를 클릭하십시오.`Hotels`.
5. 검색 결과 페이지로 리디렉션됩니다. 여기에서 사용 가능한 공급업체를 스크롤할 수 있습니다.`Hotels`카테고리를 선택하고 화면 왼쪽에 있는 필터를 사용하여 검색 필터를 적용/변경하세요.

## 검색 결과 조정

재고와 공급업체를 필터링하는 방법은 다음과 같습니다. 검색 결과 오른쪽 상단에 있는 필터 아이콘을 클릭하세요.

* 속성 이름 (*예를 들어 센타라*)
* 재고 이름(*예를 들어 빌라*)
* 재고 유형(*예를 들어 스파*)
* 별점 (*예를 들어 5성급 호텔*)
* 사용자 리뷰 평가 (*예를 들어 8 / 10*)
* 지리적 위치(*예: 지도 위치 선택*)
* 대륙 (*예를 들어 북미*)
* 국가 (*예를 들어 독일이나 프랑스*)
* 도시들 (*예를 들어 도쿄*)
* 라이프스타일(*예를 들어 LGBTQ*)
* 위치 유형(*예를 들어 공항*)
* 세그먼트 유형(*예를 들어 중간*)
* 카테고리 유형(*예를 들어 롯지*)
* 스타일(*예를 들어 아르 데코*)
* 직접적인 관계만 (*예를 들어 직접 재고 표시*)
* 반려동물 동반 가능 (*예: 반려동물 친화적인 재고 표시*)
* 친환경적 (*예를 들어 친환경 재고 표시*)
* 어린이 친화적 (*예를 들어, 어린이에게 친화적인 재고를 표시합니다.*)
* 인기 있는 (*예를 들어, 고가치 재고 표시*)

## 행위

작업을 통해 재고 또는 공급업체와 상호 작용할 수 있습니다. 다음과 같은 세 가지 주요 범주를 통해 작업을 수행할 수 있습니다.

1. **공급업체 세부 정보**공급업체에 대해 더 자세히 알아보려면 어떻게 해야 하나요?
2. **만들다...**&#xC774; 재고를 판매 가능한 것으로 어떻게 전환할 수 있나요?
3. **...에 추가**이 재고를 동시에 판매할 수 있는 목록에 저장하려면 어떻게 해야 하나요?

클릭하면`Supplier details`, 다음을 수행할 수 있습니다.

* **책**제휴사 ID를 사용하여 부동산에 직접 가서 예약하고 수수료를 받으세요.
* **직접 연결을 요청하세요**공급업체에 직접 연결을 제공할 수 있는지 물어보세요.
* **트랙 성능**시간 경과에 따른 공급업체 성과를 추적하기 위해 분석 차트를 만듭니다.

* **기존 목록에 추가**이렇게 하면 이미 만든 목록에 공급업체나 재고가 추가됩니다.
* **새로 큐레이팅된 목록**이렇게 하면 새로운 목록을 만들라는 메시지가 표시되고 해당 항목이 목록에 추가됩니다.

## 빈 결과

특정 공급업체를 찾고 있지만 찾을 수 없는 경우, Google 지역 정보에서 원하는 공급업체를 찾을 수 있는 필드가 표시됩니다. Google 지역 정보에 해당 공급업체가 있는 경우,`lead`Wink에서.
이 메시지는 누군가 해당 공급업체에 연락하여 Wink에 가입하도록 요청하고 있음을 의미합니다. 저희는 공급업체에 연락하여 가입을 요청할 것입니다. 가입이 완료되면 알림을 보내드리겠습니다.
