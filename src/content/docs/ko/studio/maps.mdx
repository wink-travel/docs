지도는 단일 공급업체(예: 호텔 또는 체험 제공업체) 또는 재고(예: 객실 유형, 회의실, 스파 등) 또는 큐레이션된 목록이나 저장된 검색과 같은 재고 모음을 나타내며, 예약 가능한 지도 마커가 포함된 대화형 지도를 통해 이러한 세부 정보를 제공합니다.
사용자는 다음과 같은 방법으로 지도와 상호 작용할 수 있습니다.

이 기사의 나머지 부분에서는 지도를 만들고, 사용자 지정하고, 공유하는 방법을 안내합니다.

## 지도 만들기

지도는 두 가지 방법으로 만들 수 있습니다.

* **단일 마커 맵**: 당신이 발견한 것을 바탕으로 지도를 만들어 보세요.[수색](/studio/search).
* **다중 마커 맵**: 지도를 만듭니다[큐레이트된 목록](/studio/curated-lists) 또는 [저장된 검색](/studio/saved-searches).

### 단일 마커 맵

  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="행위" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*검색 결과 작업*</figcaption>

위는 에서 가져온 이미지입니다.[찾다](/studio/search)검색 결과를 활용하여 수행할 수 있는 몇 가지 작업을 보여드립니다. 이러한 작업 중 하나는 다음과 같습니다.`Make a map`.
해당 버튼을 클릭하면 지도 양식 페이지로 이동하여 지도를 맞춤 설정할 수 있습니다.

### 다중 마커 맵

둘 다 [큐레이트된 목록](/studio/curated-lists#list-actions) 그리고 [저장된 검색](/studio/saved-searches#list-actions)지도를 만들 수 있는 액션이 제공됩니다. 이 지도에는 해당 목록 내 모든 인벤토리에 대한 지도 마커가 포함됩니다.

## 지도 사용자 지정

이 양식을 사용하면 다음과 같은 방법으로 지도를 사용자 지정할 수 있습니다.

<Steps>
  1. 지도의 내용을 기억할 수 있도록 지도에 이름을 붙여주세요.
  2. 사용자가 지도를 돌아다닐 수 있도록 허용할지 여부를 선택하세요.
  3. 지도의 높이를 픽셀 단위로 설정합니다.
  4. 사용자가 지도를 확대/축소할 수 있는지 여부를 선택하세요.
  5. 지도를 삽입할 웹사이트에 맞는 지도 스타일을 선택하세요.
  6. 마커 색상을 선택하세요.
  7. 사용자에게 가장 먼저 보여줄 카드 페이스를 선택하세요. 기본적으로 해당 인벤토리의 기본 페이스입니다.
  8. 사용자가 흥미로워할 만한 다른 위치를 지도에 표시하기 위해 마커, 원, 사각형, 다각형을 추가할 수 있습니다.
  9. 클릭 `Save` 계속하려면 버튼을 클릭하세요.
</Steps>


지도를 저장한 후에는 지도 페이지로 이동하게 되며, 이제 지도를 전 세계와 공유할 준비가 되었습니다.

## 지도 공유

  <CloudinaryImage width="350" height="946" resize="scale" quality="best" alt="지도" src="academy/map" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*축소된 작업 메뉴가 있는 맵 항목*</figcaption>

위는 지도에서 사용 가능한 모든 작업을 보여주는 이미지입니다.

* **업데이트**지도 구성을 업데이트합니다.
* **WinkLinks에 추가**WinkLinks 계정에 지도를 추가합니다.
* **포함시키다**이 지도를 다음과 같이 포함하는 방법을 보여줍니다.[지도](/developers/web-components/#content-loader)귀하의 웹사이트에.
* **WordPress와 함께 사용**당사의 사용 방법을 보여줍니다[워드프레스 플러그인](/developers/wordpress)이 지도를 귀하의 웹사이트에 삽입하세요.

아래에서는 이러한 옵션 중 일부에 대해 더 자세히 살펴보겠습니다.

### 포함시키다

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

사이트에 지도를 삽입하는 방법은 다음과 같습니다.

* 3번째 줄에서는 Wink 스타일을 사이트에 내장하는 방법을 보여줍니다.
* 6~9번째 줄에서는 다음을 사용하는 방법을 보여줍니다.[윙크-컨텐츠-로더](/developers/web-components/#content-loader)웹 구성 요소를 가져와서 코드에 맞는 광고 배너를 가져오도록 합니다.
* 11번째 줄에서는 Javascript를 사이트에 삽입하는 방법을 보여줍니다.
* 13번째 줄에서는 다음을 포함하는 방법을 보여줍니다.[윙크 앱 로더](/developers/web-components/#application-loader)웹 구성 요소를 가져와서 페이지 수준 구성 기본 설정을 가져오도록 합니다.
