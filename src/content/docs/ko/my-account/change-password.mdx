비밀번호를 변경하거나 비밀번호를 잊어버린 경우, 다음 단계를 따르세요:

## 비밀번호 업데이트

<Steps>
1. [my.wink.travel](https://my.wink.travel)로 이동합니다.
1. 오른쪽 상단의 `Sign-in / Register`를 클릭하여 로그인합니다.
2. 오른쪽 상단의 프로필 아이콘을 클릭하면 메뉴가 나타납니다.
3. `Change password` 링크를 클릭합니다.
4. 비밀번호 변경 지침을 따릅니다.
5. 계속하려면 `Change password`를 클릭합니다.
</Steps>

다음 로그인 시에는 새 비밀번호가 필요합니다.

## 비밀번호를 잊으셨나요

때때로 사용자 비밀번호를 재설정해야 할 필요가 있습니다.

사용자 비밀번호 재설정 단계는 다음과 같습니다:

<Steps>
1. 로그인 페이지에서 `Forgot password` 링크를 클릭합니다.
2. 재설정하려는 계정의 사용자 이름 / 이메일을 입력합니다.
3. `Forgot password` 버튼을 클릭합니다. *비밀번호 재설정 폼으로 리디렉션됩니다.*
4. 이메일을 확인합니다.
</Steps>

<Steps>
1. 이메일로 받은 코드를 새 폼에 새 비밀번호와 함께 붙여넣습니다.
2. 새 비밀번호를 확인합니다.
3. `Reset password` 버튼을 클릭합니다. *로그인 페이지로 리디렉션됩니다.*
</Steps>
