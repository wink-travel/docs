Wink 사용자는 자신의 숙소, 제휴사 또는 여행사 계정을 귀하가 관리하도록 허용할 수 있습니다. 이 경우 초대 요청이 포함된 이메일을 받게 됩니다.

import { Steps } from '@astrojs/starlight/components';

현재 받은 Wink 초대를 확인하려면 다음 단계를 따르세요:

<Steps>
 1. [my.wink.travel](https://my.wink.travel)에 로그인합니다.
 2. 왼쪽 사이드바에서 `Invites`를 클릭합니다.
 3. 현재 받은 모든 초대 목록이 표시됩니다.
 4. 이 섹션에는 모든 신청 내역이 나열됩니다.
 5. 초대를 `Accept` 또는 `Reject` 중 선택합니다.
 6. 초대를 수락하면 해당 계정이 `Accounts` 섹션에 표시됩니다.
</Steps>
