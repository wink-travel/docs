계정을 생성한 후, [my.wink.travel](https://my.wink.travel)에서 다음과 같은 작업을 추가로 할 수 있습니다.

- 계정 담당자 업데이트
- 계정 정보 업데이트
- 고유 시스템 ID 업데이트 *시스템 전반에서 계정을 찾는 방법*
- 주소 업데이트
- 프로필 사진 업데이트 *[WinkLinks 및 Wink에서 숙소 및 제휴사와 연결할 때 표시됨]*
- 로고 업데이트 *[이메일에 표시됨]*

또한 다음을 할 수 있습니다...

- [온라인 존재감](/ko/my-account/online-presence) 관리.
- 계정을 관리할 [전담 에이전시 담당자](/ko/my-account/agency-representation)를 [유료로] 지정.
- 계정에 접근할 수 있는 [관리자](/ko/my-account/managers) 지정.
- 재고를 볼 때 사용자 경험을 제어할 수 있도록 [맞춤 설정](/ko/my-account/customizations) 관리.
- 개발자로서 Wink에 접근하고 Wink에 기능을 추가/확장할 수 있도록 [애플리케이션](/ko/my-account/applications) 관리.
- Wink 플랫폼 이벤트가 실시간으로 발생할 때 알림을 받을 수 있도록 [웹훅](/ko/my-account/webhooks) 관리. *예: 새 예약 알림 받기*.
