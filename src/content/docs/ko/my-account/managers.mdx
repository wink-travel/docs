## 사용자 초대

다른 사용자를 초대하여 제휴 계정을 관리하도록 도울 수 있습니다. 사용자가 초대를 수락하면 결제를 제외한 계정에 대한 전체 접근 권한을 갖게 됩니다. 자금을 관리하는 권한은 항상 본인만 갖습니다.

<Steps>
1. [Accounts](https://my.wink.travel)에서 작업할 계정을 선택합니다.
2. 계정 카드 하단의 `Actions`를 클릭하고 `Manage account`를 선택합니다.
3. 다음 페이지에서 `Managers` 탭을 클릭합니다.
4. 계정을 관리할 사람의 이메일을 입력합니다.
5. 이메일 옆에 있는 `Invite` 링크를 클릭합니다.
</Steps>

Wink가 계정에 추가하려는 사용자에게 이메일을 보내며, 해당 사용자는 초대를 수락하거나 거절할 수 있습니다.

## 소유권 이전

현재 계정 소유자인 경우, 기존 관리자에게 계정 소유권을 이전할 수 있습니다.

소유권을 변경하려면 아래 단계를 따르세요:

<Steps>
1. [Accounts](https://my.wink.travel)에서 작업할 계정을 선택합니다.
2. 계정 카드 하단의 `Actions`를 클릭하고 `Manage account`를 선택합니다.
3. 다음 페이지에서 `Managers` 탭을 클릭합니다.
4. 계정의 기존 관리자 중 한 명 옆에 있는 `Make owner` 링크를 클릭합니다.
5. 이제 계정의 관리자가 되었으며 더 이상 소유자는 아닙니다.
</Steps>
