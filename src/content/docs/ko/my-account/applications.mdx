애플리케이션은 개발자로서 Wink의 모든 기능에 접근할 수 있게 해줍니다. 동일한 데이터에 접근할 수 있을 뿐만 아니라 Wink의 기능을 확장할 수도 있습니다.
Wink의 OAuth2 서버를 사용하여 Google이나 Facebook을 사용하는 것과 같은 방식으로 여행 사이트를 인증할 수 있습니다.
이렇게 하면 Wink의 기능을 자체 웹사이트나 애플리케이션에 확장할 수 있습니다.

:::note
[새 계정 만들기](/ko/my-account/create-account) 시 첫 번째 앱이 이미 생성되어 있습니다. 일부 예약 시나리오에 이 앱을 사용합니다.

이 앱은 재사용할 수 없습니다. 필요에 따라 새 앱을 생성해 주세요.
:::

:::note
Wink에서 생성된 애플리케이션은 모든 API 엔드포인트에 사용할 수 있습니다.
:::

<Steps>
 1. [계정](https://my.wink.travel)에서 작업할 계정을 선택합니다.
 2. 계정 카드 하단의 `Actions`를 클릭하고 `계정 관리`를 선택합니다.
 3. 다음 페이지에서 `Applications` 탭을 클릭합니다.
 4. 이 섹션에 모든 애플리케이션이 나열됩니다.
 5. 기존 앱을 편집하거나 새 앱을 생성합니다.
 </Steps>

## 앱 생성

저희가 생성해 드린 앱의 `비밀 키`에 접근할 수 없으므로 직접 앱을 생성해야 합니다.

:::caution
애플리케이션을 특정 계정에 연결해야 하므로 앱을 생성하기 전에 Wink의 제휴 계정 또는 TripPay 계정을 생성해야 합니다.
:::

애플리케이션 생성 단계는 다음과 같습니다:

<Steps>
1. 앱 페이지에서 `새 애플리케이션 생성` 버튼을 클릭합니다.
2. ***이름*** 앱 이름을 지정합니다. *예: Cool App*
3. **엔터티 이름** 기존 계정 중 하나와 앱을 연결합니다. *예: Cool Account*
4. **리디렉션 URI** 최소 하나의 리디렉션 URI가 필요합니다. 인증 성공 후 리디렉션 도메인을 검증하는 데 사용됩니다. *예: https://www.cool-site.com*
5. 계속하려면 `저장` 버튼을 클릭합니다.
</Steps>

애플리케이션 목록으로 리디렉션됩니다. 새 앱이 목록에 표시되며 앱 비밀 키는 이때만 확인할 수 있습니다. 반드시 복사하여 안전한 곳에 보관하세요.

## 앱 업데이트

리디렉션 URI가 변경되었거나 URI를 추가 또는 제거하려면 앱을 업데이트해야 합니다.

기존 앱을 업데이트하려면 다음을 수행하세요:

<Steps>
1. 업데이트할 앱의 `Actions` 링크를 클릭합니다.
2. Actions 아래의 `업데이트` 버튼을 클릭합니다.
3. 앱을 수정합니다.
4. 계속하려면 `저장` 버튼을 클릭합니다.
</Steps>

애플리케이션 목록으로 리디렉션됩니다.

## 앱 삭제

:::caution
계정의 모든 기능이 정상 작동하도록 자동 생성된 앱은 삭제하지 마세요.
:::

더 이상 사용하지 않는 앱은 삭제할 수 있습니다.

앱을 삭제하려면 다음을 수행하세요:

<Steps>
1. 삭제할 앱의 `Actions` 링크를 클릭합니다.
2. Actions 아래의 `삭제` 버튼을 클릭합니다.
3. 삭제 확인을 위해 `확인` 버튼을 클릭합니다.
</Steps>

애플리케이션 목록으로 리디렉션됩니다.
