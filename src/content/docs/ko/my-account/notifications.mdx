때때로 저희가 연락을 드려야 할 때가 있습니다. 그럴 때는 오른쪽 상단의 종 모양 아이콘이 바뀌는 것을 보실 수 있습니다.  
저희가 보낸 내용을 확인하려면:

<Steps>
1. 메인 내비게이션 바에서 `계정 > 알림`을 클릭하세요.
2. 새 알림과 이전 알림을 확인할 수 있습니다.
3. 알림을 읽은 후에는 `읽음 표시` 또는 `삭제`를 선택하세요.
</Steps>

## API

`Affiliates`를 관리하려는 개발자는 [Developers > API > Affiliate](/ko/developers/apis/#affiliate-api)로 이동하세요.
