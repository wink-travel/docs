웹훅은 귀하의 계정에 대해 Wink에서 실시간 이벤트를 수신하고 반응할 수 있는 방법입니다.

<Steps>
 1. [Accounts](https://my.wink.travel)에서 작업할 계정을 선택합니다.
 2. 계정 카드 하단의 `Actions`를 클릭하고 `Manage account`를 선택합니다.
 3. 다음 페이지에서 `Webhooks` 탭을 클릭합니다.
 4. 이 섹션에 모든 웹훅이 나열됩니다.
 5. 기존 웹훅을 편집하거나 새로 만듭니다.
 </Steps>

## 예시

<Steps>
1. Wink에서 예약이 발생합니다.
1. Wink 이벤트 `booking.created`가 전송됩니다.
2. 귀하의 시스템이 이 이벤트를 적극적으로 수신합니다.
3. 귀하의 시스템은 자체 분석을 추적하고 이 이벤트를 사용하여 시계열 데이터베이스를 업데이트합니다.
</Steps>

## 웹훅 생성

:::caution
웹훅을 생성하기 전에 Wink에서 제휴 계정을 만들거나 TripPay에서 계정을 만들어야 합니다. 웹훅이 특정 계정에 연결되기 때문입니다.
:::

웹훅 생성 단계는 다음과 같습니다:

<Steps>
1. 웹훅 페이지에서 `Create new webhook` 버튼을 클릭합니다.
2. ***이름*** 웹훅 이름을 지정합니다. *예: Wink Stalker*
3. ***활성화*** 활성 상태로 두십시오.
4. **엔터티 이름** 기존 계정 중 하나와 웹훅을 연결합니다. *예: Cool Account*
5. **이벤트** 관심 있는 이벤트를 선택합니다. *예: `booking.created`*
6. **웹훅 URL** 이벤트를 수신할 URL을 입력합니다. *예: `https://my.system.com/webhook/wink`*
7. 계속하려면 `Save` 버튼을 클릭합니다.
</Steps>

웹훅 목록으로 리디렉션되며 새 웹훅이 준비됩니다.

## 웹훅 업데이트

다음과 같은 경우 앱을 업데이트해야 합니다:

- 웹훅을 비활성화하려는 경우.
- 더 많은/적은 이벤트를 수신하려는 경우.
- 수신 웹훅 URL을 변경하려는 경우.

기존 웹훅을 업데이트하려면 다음을 수행하세요:

<Steps>
1. 업데이트할 웹훅의 `Actions` 링크를 클릭합니다.
2. Actions 아래의 `Update` 버튼을 클릭합니다.
3. 웹훅을 수정합니다.
4. 계속하려면 `Save` 버튼을 클릭합니다.
</Steps>

웹훅 목록으로 리디렉션됩니다.

## 웹훅 제거

더 이상 웹훅이 필요하지 않으면 제거할 수 있습니다.

웹훅을 제거하려면 다음을 수행하세요:

<Steps>
1. 제거할 웹훅의 `Actions` 링크를 클릭합니다.
2. Actions 아래의 `Remove` 버튼을 클릭합니다.
3. 삭제 확인을 위해 `OK` 버튼을 클릭합니다.
</Steps>

웹훅 목록으로 리디렉션됩니다.
