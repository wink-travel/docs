게시물 편집기는 무제한 수동 게시물을 생성할 수 있으며, 자동으로 생성된 게시물을 관리하는 곳이기도 합니다.

편집기는 4개의 섹션으로 구성되어 있습니다:

1. 콘텐츠 섹션.
2. 미디어 섹션.
3. 링크 섹션.
4. 설정 섹션.

## 콘텐츠

콘텐츠 섹션에서는 게시되기 전에 텍스트 콘텐츠를 수정할 수 있습니다. 여러 언어로 현지화된 콘텐츠를 생성할 수 있으며, 각 언어마다 별도의 제목과 설명이 있습니다.

## 미디어

미디어 섹션에서는 텍스트와 함께 제공되는 이미지와 비디오를 수정할 수 있습니다.

## 링크

링크 섹션에서는 사용자가 게시물을 읽은 후 이동하기를 원하는 곳으로 연결할 수 있습니다. *예: 특별 할인 링크나 추가 정보를 얻을 수 있는 장소로 연결할 수 있습니다.* 링크 URL, 링크 캡션, 링크 설명을 추가할 수 있습니다.

## 설정

설정 섹션에서는 게시물별로 기본 설정을 수정하거나 재정의할 수 있습니다. 설정할 수 있는 항목은 다음과 같습니다:

- **플랫폼** — 이 게시물이 게시될 소셜 네트워크를 선택합니다.
- **게시물 유형** — 거래성, 인지도, 감성 중에서 선택합니다.
- **어조** — 이 특정 게시물에 대한 어조를 재정의합니다.
- **맞춤 설정** — 표시 통화, 언어, 일정 등 다른 맞춤 설정을 사용합니다.
- **시간대** — 게시 시간대 재정의.
- **예약 날짜** — 생성된 게시물은 최적의 시간에 자동 예약됩니다. 다른 시간대를 원하면 날짜와 시간을 재정의할 수 있습니다.

## 게시물 상태 수명 주기

모든 게시물은 상태 수명 주기를 따릅니다:

| 상태 | 설명 |
| -- | -- |
| **초안** | 게시물이 생성되었으나 아직 검토 준비가 되지 않음 |
| **승인 대기 중** | 게시물이 예약 전에 사람의 승인을 기다림 |
| **승인됨** | 게시물이 승인되어 예약될 예정 |
| **거부됨** | 게시물이 검토되어 거부됨 — 게시되지 않음 |
| **예약됨** | 게시물이 예약된 날짜와 시간에 게시 대기 중 |
| **게시됨** | 게시물이 소셜 네트워크에 성공적으로 게시됨 |
| **일시 중지됨** | 게시물 게시가 일시적으로 중지됨 |
| **실패** | 오류로 인해 게시할 수 없음 |

## 게시물 승인

게시물은 게시물 그리드에서 하나 이상의 게시물을 선택하여 승인하거나 거부할 수 있습니다.

:::note
언어별, 소셜 네트워크별로 각각 한 개의 항목이 있습니다.

예: 3개의 소셜 네트워크와 3개의 언어를 선택하면 캘린더에 9개의 항목이 표시됩니다.
:::
