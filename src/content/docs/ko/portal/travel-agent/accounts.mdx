[Wink Agent](https://agent.wink.travel)는 전 세계 여행 인벤토리와 연결해 줍니다. 이 문서에서는 Wink에서 여행사 계정을 생성하고 관리하는 방법을 안내합니다.

## 계정 생성

인증이 완료되면 모든 여행사 계정을 보여주는 [계정 페이지](https://agent.wink.travel/company/list)로 리디렉션됩니다.

`Create new account` 버튼을 클릭하여 계속 진행하세요. [계정 생성](https://agent.wink.travel/company/create) 페이지로 이동합니다.

모든 필수 항목을 작성하세요.

<Steps>
1. **이름** 계정의 이름입니다. *실명이 아니어도 되며, 사용자가 WinkLinks 페이지를 보거나 공급업체가 직접 연결을 원할 때 보게 되는 이름입니다.*
2. **법인 유형** 회사인지 개인인지 선택합니다. *주로 자금 수령 자격을 판단하는 데 사용됩니다.*
3. **유형** 계정에 가장 적합한 유형을 선택하세요. *예: 인플루언서*
4. **설명** 계정에 대한 설명을 작성하세요. *WinkLinks 페이지와 공급업체가 직접 연결을 원할 때 보게 됩니다.*
5. **웹사이트** 주요 웹사이트를 입력하세요. *IG 계정 URL이나 회사 웹사이트가 될 수 있습니다. 온라인 작업을 주로 하는 곳이어야 하며, 공급업체가 연결 여부를 판단할 때 이 링크를 방문합니다.*
6. **도시** 거주하는 도시를 입력하세요.
</Steps>

### 결제 조건

:::note
결제 책임을 선택하면 직접 연결된 숙소에만 접근할 수 있습니다.
:::

:::caution
이 경우 Wink는 호텔 결제에 대한 책임을 지지 않습니다.
:::

:::note
이 옵션은 계정 생성 시에만 선택할 수 있습니다. 대부분의 여행사는 이 설정을 비활성화 🛑 상태로 두고 이 포털을 사용하는 것이 좋습니다.
:::

API를 통해 통합된 등록된 여행사는 결제를 받고 해당 자금을 숙소에 분배할 수 있는 옵션이 있습니다.

<Steps>
1. **결제를 받고 싶습니다** 여행사가 자금 수령 책임을 집니다.
2. **호텔에 결제하고 싶습니다** 여행사가 호텔 결제 책임을 집니다.
</Steps>

직접 연결을 원하는 호텔은 경고 메시지를 보고 Wink가 호텔 결제 책임이 없음을 확인해야 계정과의 관계를 설정할 수 있습니다.

### 계약

<Steps>
1. **계약** 계속하려면 [제휴사 계약](/ko/portal/travel-agent/terms-of-service) 및 [결제 조건](/ko/portal/travel-agent/payment-terms)을 수락하세요.
2. `Save` 버튼을 클릭하세요.
</Steps>

축하합니다 🎉 ...Wink에서 첫 제휴사 계정을 생성했습니다.

계정 생성 후 주소를 입력하고 프로필 사진을 추가하여 계정을 더 맞춤 설정할 수 있습니다.

<Steps>
1. 새로 생성한 여행사 계정을 클릭하세요. *계정 대시보드로 이동합니다.*
2. 메인 내비게이션 바에서 `Account > Profile`을 클릭하세요.
3. 프로필 페이지에서 기존 프로필을 업데이트하고 추가 정보를 입력할 수 있습니다.
</Steps>

:::note
여행사 계정 프로필 사진은 Wink 사용자 프로필 사진과 다릅니다. 여행사 계정 프로필 사진은 WinkLinks 페이지에 표시되며, 공급업체가 연결을 원할 때 보게 됩니다.

사용자 프로필 사진은 주로 내부 용도입니다.
:::

## 관리자

다른 사용자를 초대하여 여행사 계정을 함께 관리할 수 있습니다. 사용자가 초대를 수락하면 결제를 제외한 계정 전체에 대한 완전한 접근 권한을 갖게 됩니다. 자금 관리는 항상 본인이 책임집니다.

<Steps>
1. [계정 페이지](https://agent.wink.travel/company/list)에서 작업할 여행사 계정을 선택하세요.
2. 메인 내비게이션 바에서 `Account > Managers`를 클릭하세요.
3. 계정 관리자로 만들고 싶은 사용자의 이메일을 입력하세요.
4. 이메일 옆의 `Invite` 링크를 클릭하세요.
</Steps>

Wink가 사용자에게 초대 수락 또는 거절 이메일을 발송합니다.

## 알림

때때로 저희가 연락을 드릴 때가 있습니다. 그럴 때는 오른쪽 상단의 벨 아이콘이 변경됩니다.
받은 내용을 확인하려면:

<Steps>
1. 메인 내비게이션 바에서 `Account > Notifications`를 클릭하세요.
2. 새 알림과 이전 알림을 확인할 수 있습니다.
3. 읽은 후에는 `Mark as read` 또는 `Delete`를 선택하세요.
</Steps>

## API

여행사는 여전히 제휴사로 간주됩니다. `Affiliates` 관리를 원하는 개발자는 [Developers > API > Affiliate](/ko/api)로 이동하세요.
