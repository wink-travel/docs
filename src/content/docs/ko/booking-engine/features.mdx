## 온라인 여행사

제휴사의 경우 온라인 여행사 버전을 사용하는 것이 가장 합리적입니다. 이 사이트는 여러 숙소를 보고 싶어 하는 여행객을 위해 만들어졌기 때문입니다.

## 구성

대부분의 경우, 두 애플리케이션을 직접 공유할 필요는 없습니다. 저희는 여러분이 여행 목록을 공유할 수 있는 다양한 방법을 마련했으며, 자세한 내용은 다음에서 확인하실 수 있습니다.[윙크 스튜디오](/studio/what-is-studio)또는 통해[윙크링크스](/link-manager/wink-links).

하지만 이러한 웹사이트 중 하나에 수동으로 링크하려면 클라이언트 ID와 사용자 정의 ID를 URL에 매개변수로 추가해야 합니다.

클라이언트 ID는 다음과 같은 방법으로 찾을 수 있습니다.

<Steps>
  1. 둘 중 하나에 로그인하세요 [윙크 엑스트라넷](/extranet/what-is-extranet) 또는 [윙크 스튜디오](/studio/what-is-studio).
  2. 오른쪽 상단 모서리에 있는 프로필 아이콘을 클릭하고 클릭하세요. [응용 프로그램](/developers/applications).
  3. 작업하려는 Wink 계정에 연결된 애플리케이션을 사용하세요.
  4. 해당 애플리케이션에서 클라이언트 ID를 복사하여 URL에 요청 매개변수로 붙여넣으세요. 예: ?client-id=MY\_CLIENT\_ID
</Steps>


### 엑스트라넷

사용자 정의 ID를 속성으로 찾는 방법은 다음과 같습니다.

<Steps>
  1. 로그인 [윙크 엑스트라넷](/extranet/what-is-extranet).
  2. 작업하려는 속성을 선택하세요.
  3. 메인 탐색 바에서 다음을 클릭하세요. `Account > Branding`
  4. 사용자 지정 ID를 복사하여 URL에 요청 매개변수로 붙여넣으세요. 예: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### 사진관

제휴사로서 귀하의 맞춤 ID를 찾는 방법은 다음과 같습니다.

<Steps>
  1. 로그인 [윙크 스튜디오](/studio/what-is-studio).
  2. 작업하려는 계정을 선택하세요.
  3. 메인 탐색 바에서 다음을 클릭하세요. `Tools > Customization`
  4. 사용하려는 사용자 지정 항목에서 사용자 지정 ID를 복사하여 URL에 요청 매개변수로 붙여넣으세요. 예: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## 오타

결과적으로 다음과 같은 URL이 생성됩니다.

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

호텔 페이지로 바로 이동하려면 URL이 다음과 같습니다.

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::참고
클라이언트 및/또는 구성 ID를 포함하지 않으면 Wink OTA는 기본 구성을 사용합니다.
:::

## 아이비

부동산 관리 회사라면 Wink IBE 홈페이지에 링크를 걸어보세요. 홈페이지에는 지정된 테마와 함께[당신이 대표하는 속성](/studio/agency-representation).

결과적으로 다음과 같은 URL이 생성됩니다.

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

부동산을 소유하고 있다면 Wink IBE 부동산 랜딩 페이지에 직접 링크를 걸고 싶을 것입니다.

해당 URL은 다음과 같습니다.

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::참고
클라이언트 및/또는 구성 ID를 포함하지 않으면 Wink IBE가 로드되지 않습니다. 이러한 매개변수는 필수입니다.
:::

## 예: 속성 페이지

Wink의 모든 숙박 시설은 각 웹사이트에 고유한 랜딩 페이지를 제공합니다. 모든 외부 링크는 여행객을 이러한 랜딩 페이지 중 하나로 안내합니다.

랜딩 페이지에는 여행자가 해당 숙박 시설에 머물 것인지에 대한 현명한 선택을 내리는 데 필요한 모든 정보가 포함되어 있습니다.
