Wink 인벤토리 [카드](/ko/studio/cards)를 볼 때마다 카드 오른쪽 상단에 하트 아이콘이 있어 해당 인벤토리를 버킷 리스트에 추가할 수 있습니다.

버킷 리스트를 관리하려면 다음 단계를 따르세요:

<Steps>
1. 추가하려는 인벤토리의 ♡ `하트 아이콘`을 클릭합니다.
2. 로그인한 경우 하트 아이콘이 빨간색으로 바뀝니다.
3. 로그인하지 않은 경우 로그인 페이지로 리디렉션됩니다.
</Steps>

:::tip
♥️ `하트 아이콘`을 한 번 더 클릭하면 리스트에서 제거할 수 있습니다.
:::

오른쪽 상단의 프로필 아이콘을 클릭한 다음 `버킷 리스트` 버튼을 클릭하여 모든 버킷 리스트 항목을 관리할 수 있습니다.
