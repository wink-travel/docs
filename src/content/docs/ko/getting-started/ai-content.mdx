Wink는 인공지능을 활용하여 여행 인벤토리의 모든 부분에 대해 다국어로 다듬어진 콘텐츠를 만드는 데 도움을 줍니다. 새 숙소를 등록하든 기존 목록을 새롭게 하든, AI는 출시 시간을 몇 주에서 몇 분으로 단축시켜 줍니다.

## 작동 방식

Wink의 콘텐츠 엔진은 견고한 다중 공급자 AI 아키텍처를 기반으로 구축되었습니다. AI 생성 콘텐츠를 요청하면 플랫폼은 OpenAI, Anthropic, Google 등 주요 AI 서비스 풀에서 가장 적합한 공급자를 자동으로 선택합니다. 한 공급자가 일시적으로 사용 불가능할 경우 시스템은 원활하게 다음 공급자로 전환하여 중단 없이 서비스를 제공합니다.

## AI가 생성하는 콘텐츠

AI 콘텐츠 생성은 **9가지 인벤토리 유형**에 걸쳐 제공됩니다:

| 인벤토리 유형 | AI가 생성하는 내용 |
|---|---|
| **숙소** | 환영 문구, 설명, 하이라이트 |
| **객실** | 객실 설명, 특징, 판매 포인트 |
| **레스토랑** | 식사 설명, 요리 하이라이트, 분위기 |
| **스파** | 트리트먼트 설명, 웰니스 하이라이트 |
| **액티비티** | 체험 설명, 일정 내러티브 |
| **명소** | 명소 개요, 역사적 배경 |
| **회의실** | 공간 설명, 행사 가능성 |
| **장소** | 관심 지점 설명, 지역 맥락 |
| **추가 서비스** | 패키지 및 번들 설명 |

## 톤 오브 보이스

AI가 생성하는 콘텐츠의 톤을 브랜드에 맞게 맞춤 설정할 수 있습니다. 사용 가능한 톤은 다음과 같습니다:

- **중립적** — 직설적이고 정보 중심적.
- **친근한** — 따뜻하고 다가가기 쉬운.
- **럭셔리** — 세련되고 고급스러운.
- **푸디** — 요리 중심적이고 맛에 집중한.
- **모험적인** — 대담하고 행동 지향적.
- **가족 친화적** — 아이가 있는 가족에게 환영받는.
- **로맨틱** — 친밀하고 커플 친화적.
- **유머러스한** — 가볍고 장난기 있는.
- **에코** — 지속 가능성과 자연 중심.
- **지역 전문가** — 현지 전문가의 조언 같은.
- **포용적인** — 모두에게 접근 가능하고 환영하는.
- **예산 지향적** — 가성비와 혜택 중심.

숙소의 정체성과 대상 고객에 가장 적합한 톤을 선택하세요.

## AI 지원 등록

Wink에서 AI를 가장 강력하게 활용하는 부분은 숙소 등록 과정입니다. 숙소 웹사이트 URL만 제공하면 Wink가:

1. **웹사이트를 스크랩**하여 숙소를 이해합니다.
2. **설명 생성** — 환영 문구, 숙소 하이라이트, 요약.
3. **편의 시설 제안** — 웹사이트 콘텐츠를 업계 표준 편의 시설 코드에 매핑.
4. **인구 통계 추출** — 위치 정보, 숙소 유형, 대상 고객.

결과를 검토하고 다듬은 후 게시하세요. 몇 주 걸리던 작업이 한 시간 이내에 완료됩니다.

## AI와 번역

AI가 기본 언어로 콘텐츠를 생성한 후, Wink는 통합 번역 서비스를 사용해 40개 이상의 언어로 자동 번역할 수 있습니다. 즉, 한 번의 콘텐츠 생성 세션으로 전 세계 고객을 위한 다국어 목록을 만들 수 있습니다.
