이 문서는 Wink에서 현지화를 어떻게 관리하는지 설명합니다.

- Dansk
- Deutsch
- Eesti

- Español (MX)
- Filipino

- Ελληνικά
- עברית
- हिन्दी
- Hrvatski
- Magyar
- Íslenska
- Italiano

- Latviski

- 简体中文
- 繁體中文

## 플랫폼 수준

모든 애플리케이션은 위에 언급된 모든 언어를 지원하도록 현지화되어 있습니다. 각 애플리케이션에는 페이지 하단에 언어 선택기가 있어 사용자가 원하는 언어를 선택할 수 있습니다. 예약 엔진은 URL을 통해 원하는 언어를 전달하는 것도 지원합니다.

## 사용자 생성 콘텐츠

사용자 생성 콘텐츠는 위에 언급된 모든 언어를 지원하도록 현지화됩니다. 사용자는 콘텐츠의 영어 버전을 생성하고 Wink가 나머지를 처리하도록 선택할 수 있습니다. 또는 특정 언어 집합을 직접 관리하여 더 세밀한 제어를 하면서 Wink가 나머지를 처리하도록 할 수도 있습니다.

## 주의사항

기본 언어는 영어입니다. 별도로 지정하지 않는 한, 사용자가 당사 애플리케이션을 사용할 때 기본 언어를 우선 사용합니다. 사용자의 브라우저 로케일을 기준으로 활성 언어를 설정하지 않습니다.
