Wink는 귀하의 프라이버시를 존중하며 귀하의 개인 데이터를 보호하는 데 전념하고 있습니다. 저희는 **일반 데이터 보호 규정(GDPR)** 및 전 세계의 기타 관련 데이터 보호 법규를 준수합니다.

이 페이지에서는 Facebook 또는 Instagram과 같은 소셜 로그인 제공자를 통해 수집된 데이터를 포함하되 이에 국한되지 않는, 당사 플랫폼을 통해 수집된 개인 데이터 삭제를 요청하는 방법을 안내합니다.

## 데이터 프라이버시에 대한 우리의 약속

저희는 서비스 제공에 필요한 데이터만 수집하며, 귀하의 개인 정보를 절대 판매하지 않습니다. 수집하는 데이터는 다음을 포함할 수 있습니다:

- 귀하의 이름과 이메일 주소
- 공개 프로필 정보(예: Facebook 또는 Google에서)
- 당사 플랫폼에서의 사용 또는 활동 데이터
- 귀하가 명시적으로 허용한 권한

저희는 귀하의 데이터를 보호하기 위해 적절한 조치를 취하며, GDPR 및 Meta의 데이터 정책에 따라 모든 삭제 요청을 처리합니다.

## 데이터 삭제 요청 방법

귀하는 당사가 보유한 귀하의 개인 정보를 삭제하도록 요청할 권리가 있습니다.

데이터 삭제를 요청하려면:

- 📧 **이메일 보내기**: [support@wink.travel](mailto:support@wink.travel)
- 📝 제목란에 다음을 기재: `Data Deletion Request`  
- 🔍 다음과 같은 식별 정보를 포함해 주세요:
  - 당사 플랫폼에서 사용한 이메일 주소
  - 귀하의 Facebook 또는 Google ID 또는 유사한 정보(해당되는 경우)

저희는 GDPR에 따라 요청 접수 후 **30일 이내**에 답변 드립니다.

## 질문이나 우려 사항이 있으신가요?

데이터 수집, 사용 또는 삭제 방법에 대해 궁금한 점이 있으면 언제든지 개인정보 보호팀에 [support@wink.travel](mailto:support@wink.travel)로 문의해 주세요.

저희를 신뢰해 주셔서 감사합니다. 저희는 데이터 보호를 매우 중요하게 여기며 귀하가 자신의 정보에 대해 완전한 통제권을 가질 수 있도록 최선을 다하고 있습니다.
