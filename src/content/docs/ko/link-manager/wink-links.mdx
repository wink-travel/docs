링크 관리자는 나중에 사용할 수 있도록 좋아하는 URL을 저장하고 정리할 수 있게 해줍니다. 브라우저 즐겨찾기가 링크 관리자의 한 예입니다. 소셜 미디어가 등장하면서 친구나 청중에게 링크를 노출하고 공유할 필요가 커졌습니다. 페이스북 피드는 소셜 친화적인 링크 관리자의 예입니다.

우리는 여행 산업을 위한 링크 관리자 [WinkLinks](https://i.trvl.as/)를 만들었습니다. 정보 제공, 흥미 유발, 광고를 위해 존재하며 전 세계 호텔의 훌륭한 상품을 예약할 수 있게 해줍니다.

## 기능

- WinkLinks는 Commission Junction, Amazon 같은 기존 제휴 링크를 무료로 추가할 수 있게 해줍니다.
- [https://i.trvl.as/bob](https://i.trvl.as/bob) 같은 멋진 맞춤 URL을 제공합니다. 단일 URL만 공유해야 할 때 유용합니다.
- 링크를 정리하고 공유하며 파일도 업로드할 수 있습니다.
- 호텔, 체인 및 브랜드는 WinkLinks를 고급 예약 엔진으로 사용할 수 있습니다:
    - 여러 숙소 지원.
    - 여러 목적지 지원.
    - 여러 통화 지원.
    - 여러 언어 지원.
- WinkLinks는 `rich content` *(아래 참조)*를 지원합니다.
- 인스타그램에서 단 2번의 클릭으로 예약까지 가능 🚀

## 리치 콘텐츠

:::note
이 기능을 사용하려면 [설정](/ko/link-manager/settings)에서 `Intelligent`를 🟢로 설정하세요.
:::

`rich` / `oEmbed` 콘텐츠를 지원하는 사이트에서 링크를 공유하면 WinkLinks에서 링크가 생동감 있게 표시됩니다.

지원 사이트:

- Facebook - *게시물과 상호작용하고 동영상 재생 가능*.
- Instagram - *게시물과 상호작용하고 동영상 재생 가능*.
- YouTube - *동영상 재생 가능*.
- X (구 Twitter) - *게시물과 상호작용 가능*.
- Spotify - *좋아하는 플레이리스트 재생 가능*.
- Wink - *`Advanced` 참조*.

## 고급 기능

WinkLinks는 [Wink Studio](https://studio.wink.travel)를 통해 찾은 여행 인벤토리를 노출하고 청중에게 광고하는 훌륭한 방법이기도 합니다.

WinkLinks를 사용해 다음을 판매하세요:

- [공유 가능한 링크](/ko/studio/shareable-links) - 다음 형식으로 표시 선택 가능:
    - [광고 배너](/ko/developers/web-components/#content-loader).
    - 실시간 가격 업데이트가 반영된 일반 링크.
- [카드](/ko/studio/cards) - 실시간 가격 및 예약 가능 카드.
- [그리드](/ko/studio/grids) - 예약 가능한 카드가 포함된 그리드.
- [지도](/ko/studio/maps) - 예약 가능한 카드가 열리는 마커가 있는 지도.
