WinkLinks 태그를 관리하려면 메인 내비게이션 바에서 `WinkLinks`로 이동한 후 `Tags` 탭을 클릭하세요.

콘텐츠가 많아지면 일부를 정리하고 싶어질 것입니다. 이를 위해 `tags`를 만들었습니다. 원하는 만큼 많은 태그를 사용하여 콘텐츠를 그룹화할 수 있습니다.

## 태그 만들기

<Steps>
1. `Add a new tag` 버튼을 클릭하여 새 태그를 만듭니다.
2. 새 태그의 이름을 입력합니다. *예: 베를린 최고의 커피숍*
3. `Save` 링크를 클릭합니다.
</Steps>

## 태그 업데이트

<Steps>
1. 수정하려는 태그 옆의 `Edit` 링크를 클릭합니다.
2. 태그 이름을 업데이트합니다. *예: 베를린 최고의 커피숍*
3. `Save` 링크를 클릭합니다.
</Steps>

## 태그 제거

:::note
태그를 제거하면 해당 태그가 연결된 모든 콘텐츠에서도 함께 제거됩니다.
:::

<Steps>
1. 제거하려는 태그 옆의 `Remove` 링크를 클릭합니다.
</Steps>
