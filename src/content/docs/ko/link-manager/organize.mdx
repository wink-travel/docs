WinkLinks 태그를 관리하려면 다음으로 이동하세요.`WinkLinks`메인 탐색 표시줄에서 다음을 클릭하세요.`Tags`탭.

콘텐츠가 늘어나면 일부 콘텐츠를 정리하고 싶을 것입니다.`tags`이러한 목적을 위해, 원하는 수의 태그를 사용하여 콘텐츠를 그룹화할 수 있습니다.

## 태그 생성

<Steps>
  1. 새 태그를 만들려면 다음을 클릭하세요. `Add a new tag` 단추.
  2. 새로운 태그의 이름을 입력하세요. *예를 들어 베를린의 인기 커피숍*
  3. 클릭 `Save` 링크.
</Steps>


## 태그 업데이트

<Steps>
  1. 클릭하여 시작하세요 `Edit` 수정하려는 태그 옆에 있는 링크를 클릭하세요.
  2. 태그 이름을 업데이트하세요. *예를 들어 베를린 최고의 커피숍*
  3. 클릭 `Save` 링크.
</Steps>


## 태그 제거

:::참고
태그를 제거하면 연결된 모든 콘텐츠에서도 해당 태그가 제거됩니다.
:::

<Steps>
  1. 클릭 `Remove` 제거하려는 태그 옆에 있는 링크를 클릭하세요.
</Steps>

관리하고 싶은 개발자`WinkLinks`로 향할 수 있습니다[개발자 > API > WinkLinks](/developers/apis/#winklinks-api).
