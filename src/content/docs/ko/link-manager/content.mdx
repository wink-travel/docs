콘텐츠는 `Entries` 탭에서 확인할 수 있습니다.

우리는 세 가지 주요 콘텐츠 유형을 지원합니다:

- 웹 페이지로 연결되는 공개 URL.
- WinkLinks를 통해 업로드된 파일.
- `Quotes` 기능을 통해 생성된 텍스트 콘텐츠.

## 콘텐츠 추가

WinkLinks에 콘텐츠를 추가하는 방법은 다음과 같습니다. `Entries` 탭에서 `Add entry` 버튼을 클릭하여 시작하세요.

### 일반 링크

:::tip
링크는 웹 페이지로 연결되어야 합니다. 예를 들어, PDF 파일로 직접 연결할 수 없습니다. PDF를 공유하려면 `Upload file` 기능을 사용해야 합니다.
:::

일반 링크를 추가하는 단계는 다음과 같습니다:

<Steps>
1. 공유하려는 링크를 입력하세요. *예: https://trvl.as/3xWCH*
2. 링크를 입력하면 Wink가 해당 웹사이트의 세부 정보를 찾아 다음에 표시되는 양식을 자동으로 채웁니다.
3. 사이트 이름, 제목, 설명 등 일부 데이터를 수정할 수 있습니다.
4. 맞춤 이미지를 업로드할 수도 있습니다.
5. 계속하려면 `Save` 버튼을 클릭하세요.
</Steps>

### 파일 업로드

:::caution[요구사항]
- 이미지 크기는 10MB를 초과할 수 없습니다.
- 비디오 크기는 50MB를 초과할 수 없습니다.
- PDF를 포함한 모든 인식 가능한 이미지 및 비디오 형식을 지원합니다.
:::

:::note[권장사항]
- 대표 이미지의 경우 2560w x 1440h (16:9 비율).
- 기타 모든 이미지의 경우 2560w x 1920h (4:3 비율).
- 이미지 크기는 5MB를 넘지 않아야 합니다. Wink에 업로드하기 전에 이미지를 압축하세요.
:::

파일을 추가하는 단계는 다음과 같습니다:

<Steps>
1. 미디어는 다음 방법으로 추가할 수 있습니다:
    - 컴퓨터에서 이미지를 창으로 드래그하거나 `Browse`를 클릭.
    - 이미지 URL을 붙여넣기.
    - 노트북 카메라 사용.
    - Google Drive 계정에서.
    - Dropbox 계정에서.
    - Shutterstock에서.
    - gettyimages에서.
    - iStock에서.
    - Unsplash에서.
2. 파일 업로드가 완료되면 제목 및 설명과 같은 메타데이터를 추가할 수 있습니다.
3. 계속하려면 `Save` 버튼을 클릭하세요.
</Steps>

### 인용문 작성

:::tip
인용문은 긴 목록을 구분하는 데 훌륭한 방법입니다 [...그리고 똑똑하고 지식 있는 사람처럼 보이게도 합니다 😉].
:::

인용문 작성 단계는 다음과 같습니다:

<Steps>
1. 인용문을 입력하세요. *예: Roses are red. Violets are blue...*.
2. 원저자의 이름을 입력하세요. *예: Edmund Spenser*
3. 인용문을 가져온 사이트 이름을 입력하세요. *예: Daily Quotes*
4. 계속하려면 `Save` 버튼을 클릭하세요.
</Steps>

## 콘텐츠 업데이트

모든 콘텐츠는 `Actions` 링크를 클릭하고 `Update`를 선택하여 쉽게 수정할 수 있습니다. 콘텐츠 삭제도 `Remove` 버튼을 클릭하여 비슷한 방식으로 할 수 있습니다.

## 콘텐츠 삭제

모든 콘텐츠는 `Update`를 선택하여 쉽게 수정할 수 있습니다. 삭제하려면 먼저 `Actions` 링크를 클릭한 후 `Remove` 버튼을 클릭하세요.

## 콘텐츠 정렬

각 항목의 왼쪽 하단에 있는 아이콘을 클릭하고 원하는 위치로 드래그하여 콘텐츠 순서를 변경할 수 있습니다.

## 콘텐츠 그룹화

:::tip
항목을 만들기 전에 태그를 먼저 생성하세요. 그렇지 않으면 진행 중인 작업이 사라지고 다시 시작해야 합니다.
:::

항목을 편집할 때 그룹화 목적으로 하나 이상의 태그를 추가할 수 있습니다. 양식 하단에서 `Tags` 입력란을 찾으세요.

사용자는 WinkLinks 페이지에서 콘텐츠를 리스트 또는 그리드 형식으로 소비할 수 있으며, 하나 이상의 태그를 선택하여 콘텐츠를 추가로 필터링할 수 있습니다.

## API

`WinkLinks`를 관리하려는 개발자는 [Developers > API > WinkLinks](/developers/apis/#winklinks-api) 로 이동하세요.
