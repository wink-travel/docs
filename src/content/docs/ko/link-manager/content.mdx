귀하의 콘텐츠는 다음에서 사용할 수 있습니다.`Entries`탭.

우리는 세 가지 주요 유형의 콘텐츠를 지원합니다.

* 웹 페이지에 링크되는 공개적으로 사용 가능한 URL입니다.
* WinkLinks를 통해 업로드된 파일입니다.
* 당사를 통해 생성된 텍스트 콘텐츠`Quotes` 특징.

## 콘텐츠 추가

WinkLinks에 콘텐츠를 추가하는 방법은 다음과 같습니다.`Entries`탭을 클릭하고 시작하세요`Add entry`버튼.

### 일반 링크

:::팁
링크는 웹 페이지로 연결되어야 합니다. 예를 들어 PDF에 직접 링크할 수는 없습니다. PDF를 공유하려면`Upload file` 특징.
:::

일반 링크를 추가하는 단계는 다음과 같습니다.

<Steps>
  1. 공유하고 싶은 링크를 입력하세요. *예: https://trvl.as/s/3xWCH*
  2. 링크를 입력하면 윙크가 해당 웹사이트의 세부 정보를 찾아 다음에 보이는 양식을 채웁니다.
  3. 사이트 이름, 제목, 설명 등 일부 데이터를 수정할 수 있습니다.
  4. 사용자 정의 이미지를 업로드할 수도 있습니다.
  5. 클릭 `Save` 계속하려면 버튼을 클릭하세요.
</Steps>


### 파일 업로드

:::주의\[요구사항]

* 이미지 크기는 10Mb를 초과할 수 없습니다.
* 비디오 크기는 50Mb를 초과할 수 없습니다.
* PDF를 포함한 모든 인식 가능한 이미지 및 비디오 형식을 지원합니다.
  :::

:::note\[추천]

* 추천 이미지의 가로 2560 x 세로 1440(종횡비 16x9)입니다.
* 다른 모든 이미지는 2560w x 1920h(종횡비 4:3)입니다.
* 이미지 크기는 5MB를 넘을 수 없습니다. Wink에 업로드하기 전에 이미지를 압축해 주세요.
  :::

파일을 추가하는 단계는 다음과 같습니다.

<Steps>
  1. 미디어를 추가할 수 있습니다:
     * 컴퓨터에서 창으로 이미지를 끌어다 놓거나 클릭하세요 `Browse`.
     * 이미지를 가리키는 외부 URL을 붙여넣습니다.
     * 노트북 카메라를 사용하세요.
     * Google Drive 계정에서.
     * Dropbox 계정에서.
     * Shutterstock에서
     * 게티이미지에서
     * iStock에서
     * Unsplash에서
  2. 파일 업로드가 완료되면 제목과 설명 등 파일을 설명하는 메타데이터를 추가할 수 있습니다.
  3. 클릭 `Save` 계속하려면 버튼을 클릭하세요.
</Steps>


### 견적을 작성하세요

:::팁
인용문은 긴 목록을 구분하는 데 좋은 방법입니다. \[...그리고 똑똑하고 박식해 보이게도 해줍니다 😉].
:::

견적을 작성하는 단계는 다음과 같습니다.

<Steps>
  1. 견적을 입력하세요 *예를 들어, 장미는 빨간색이고, 제비꽃은 파란색입니다.*.
  2. 원저자의 이름을 입력하세요. *예를 들어 에드먼드 스펜서*
  3. 견적을 받은 사이트의 이름을 입력하세요. *예: 일일 인용문*
  4. 클릭 `Save` 계속하려면 버튼을 클릭하세요.
</Steps>


## 콘텐츠 업데이트

모든 콘텐츠는 클릭만 하면 쉽게 수정할 수 있습니다.`Actions`링크 및 선택`Update`. 유사한 방식으로 콘텐츠를 제거할 수 있습니다.`Remove`버튼.

## 콘텐츠 제거

모든 콘텐츠는 쉽게 선택하여 수정할 수 있습니다.`Update`. 유사한 방식으로 콘텐츠를 제거하려면 먼저 다음을 클릭하세요.`Actions`링크를 클릭한 다음`Remove`버튼.

## 콘텐츠 정렬

각 항목의 왼쪽 하단에 있는 아이콘을 클릭하고 원하는 위치로 끌어서 콘텐츠를 정렬할 수 있습니다.

## 콘텐츠 그룹화

:::팁
참여작을 작성하기 전에 태그를 만드세요. 그렇지 않으면 진행 상황이 손실되어 처음부터 다시 시작해야 합니다.
:::

항목을 편집할 때 그룹화 목적으로 하나 이상의 태그를 추가할 수 있습니다. 양식 하단으로 스크롤하여`Tags`양식 필드.

WinkLinks 페이지를 보는 사용자는 콘텐츠를 소비하는 방식(목록 또는 그리드)을 선택할 수 있으며, 하나 이상의 태그를 선택하여 콘텐츠를 추가로 필터링할 수 있습니다.
