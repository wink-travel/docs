Wink 사용자는 자신의 숙소, 제휴사 또는 여행사 계정 중 하나를 귀하가 관리하도록 허용할 수 있습니다. 이 경우 초대 요청이 포함된 이메일을 받게 됩니다.

import { Steps } from '@astrojs/starlight/components';

현재 받은 Wink 초대를 확인하려면 다음 단계를 따르세요:

<Steps>
1. 당사 웹사이트 중 하나에 로그인해야 합니다.
2. 오른쪽 상단의 프로필 아이콘을 클릭하면 메뉴가 나타납니다.
3. `Invites` 링크를 클릭합니다.
4. 초대를 `수락`하거나 `거절`하여 응답합니다.
</Steps>
