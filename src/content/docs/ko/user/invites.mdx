Wink 사용자는 자신의 호텔, 제휴사 또는 여행사 계정 중 하나를 관리하도록 선택할 수 있습니다. 관리하도록 선택하면 초대 요청 이메일이 발송됩니다.

import { Steps } from '@astrojs/starlight/components';

현재 윙크 초대장을 보려면 다음 단계를 따르세요.

<Steps>
  1. 당사 웹사이트 중 하나에 로그인해야 합니다.
  2. 오른쪽 상단 모서리에 있는 프로필 아이콘을 클릭하면 메뉴가 나타납니다.
  3. 클릭 `Invites` 링크.
  4. 다음 중 하나를 통해 초대장과 상호 작용하세요. `Accepting` 또는 `Rejecting` 그들을.
</Steps>
