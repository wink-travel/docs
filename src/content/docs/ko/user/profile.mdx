Wink 사용자 계정의 사용자 프로필을 업데이트하려면 다음 단계를 따르세요.

<Steps>
  1. 당사 웹사이트 중 하나에 로그인해야 합니다.
  2. 오른쪽 상단 모서리에 있는 프로필 아이콘을 클릭하면 메뉴가 나타납니다.
  3. 클릭 `Profile` 링크.
  4. 업데이트 지침을 따르세요 `e-mail`, `first` 그리고 `last name`.
  5. 새로운 것을 업로드할 수도 있습니다 `profile picture` 자신에 대해서.
  6. 딸깍 하는 소리 `Save`.
</Steps>


:::참고
새로운 프로필 사진은 다음에 Wink에서 완전히 로그아웃할 때 표시됩니다.
:::
