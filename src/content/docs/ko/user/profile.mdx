Wink 사용자 계정의 사용자 프로필을 업데이트하려면 다음 단계를 따르세요:

<Steps>
1. 당사 웹사이트 중 하나에 로그인해야 합니다.
2. 오른쪽 상단에 있는 프로필 아이콘을 클릭하면 메뉴가 나타납니다.
3. `프로필` 링크를 클릭하세요.
4. `이메일`, `이름` 및 `성`을 업데이트하는 지침을 따르세요.
5. 새 `프로필 사진`도 업로드할 수 있습니다.
6. `저장`을 클릭하세요.
</Steps>

:::note
새 프로필 사진은 Wink에서 완전히 로그아웃한 다음 다시 로그인할 때 표시됩니다.
:::
