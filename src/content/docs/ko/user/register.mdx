모든 포털을 통해 사용자 계정을 생성하실 수 있습니다. 생성하신 계정은 Wink의 모든 앱과 웹사이트에서 사용하실 수 있습니다.

## 여행자들

다음 단계를 따르세요.

<Steps>
  1. 로 향하세요 [https://ota.wink.travel](https://ota.wink.travel)
  2. 클릭해주세요 `Sign-In / Register` 오른쪽 상단 모서리에
  3. 기본 사용자 정보를 입력하라는 메시지가 표시됩니다. 모든 필수 필드를 입력하고 다음을 클릭하세요. `Save`.
  4. 이제 사용자 이름(이메일)과 비밀번호를 사용하여 Wink에 로그인할 수 있습니다.
  5. 로그인하면 시스템이 원래 있던 페이지로 돌아가도록 도와줍니다.
</Steps>

## 등록

Wink 사용자 계정을 생성하면 플랫폼의 모든 기능을 이용할 수 있습니다. 무제한으로 부동산 및 제휴 계정을 생성할 수 있습니다.

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="로그인" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*로그인 양식*</figcaption>
</figure>


<Steps>
  1. 처음으로 인증 서버로 이동하면 사용자 이름과 비밀번호를 입력하라는 메시지가 표시됩니다.
  2. 대신, 클릭하세요 `Create account` 단추.
  3. 필수 입력란을 모두 채우고 클릭하세요. `Register` 계속하려면 를 클릭하세요. 로그인 양식으로 돌아갑니다.
  4. 로그인하려면 이메일 / 사용자 이름 및 비밀번호를 입력하세요.
  5. 귀하는 원래 사이트로 다시 이동하게 됩니다.
</Steps>
