어느 포털에서든 사용자 계정을 생성할 수 있습니다. 계정이 생성되면 Wink의 모든 앱과 웹사이트에서 사용할 수 있습니다.

## 여행자

다음 단계를 따르세요:

<Steps>
1. [https://ota.wink.travel](https://ota.wink.travel)로 이동합니다.
2. 오른쪽 상단의 `Sign-In / Register`를 클릭합니다.
3. 기본 사용자 정보를 입력하라는 메시지가 표시됩니다. 모든 필수 항목을 작성하고 `Save`를 클릭합니다.
4. 이제 사용자 이름(이메일)과 비밀번호로 Wink에 로그인할 수 있습니다.
5. 로그인하면 시스템이 이전에 있던 페이지로 다시 이동시켜 줍니다.
</Steps>

## 회원가입

Wink 사용자 계정을 생성하면 플랫폼의 모든 기능을 이용할 수 있습니다. 무제한으로 숙소와 제휴사 계정을 생성할 수 있습니다.

<Steps>
1. 처음으로 인증 서버에 접속하면 사용자 이름과 비밀번호를 입력하라는 메시지가 표시됩니다.
2. 대신 `Create account` 버튼을 클릭하세요.
3. 모든 필수 항목을 작성하고 계속하려면 `Register`를 클릭하세요. 로그인 폼으로 돌아갑니다.
4. 이메일/사용자 이름과 비밀번호를 입력하여 로그인합니다.
5. 이전에 있던 사이트로 다시 이동합니다.
</Steps>
