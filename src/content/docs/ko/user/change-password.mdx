Wink 사용자 계정의 비밀번호를 변경하려면 다음 단계를 따르세요:

## 비밀번호 업데이트

<Steps>
1. 당사 웹사이트 중 하나에 로그인되어 있어야 합니다.
2. 오른쪽 상단에 있는 프로필 아이콘을 클릭하면 메뉴가 나타납니다.
3. `비밀번호 변경` 링크를 클릭하세요.
4. 비밀번호 변경 지침을 따르세요.
5. `저장`을 클릭하세요.
</Steps>

다음 로그인 시에는 새 비밀번호가 필요합니다.

## 비밀번호를 잊으셨나요

때때로 사용자 비밀번호를 재설정해야 할 때가 있습니다.

사용자 비밀번호 재설정 단계는 다음과 같습니다:

<Steps>
1. 로그인 페이지에서 `비밀번호를 잊으셨나요` 링크를 클릭하세요.
2. 재설정하려는 계정의 사용자 이름 / 이메일을 입력하세요.
3. `비밀번호를 잊으셨나요` 버튼을 클릭하세요. *비밀번호 재설정 양식으로 리디렉션됩니다.*
4. 이메일을 확인하세요.
</Steps>

2. 새 비밀번호를 확인하세요.
3. `비밀번호 재설정` 버튼을 클릭하세요. *로그인 페이지로 리디렉션됩니다.*
</Steps>
