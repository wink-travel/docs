Wink 사용자 계정의 비밀번호를 변경하려면 다음 단계를 따르세요.

## 비밀번호 업데이트

<Steps>
  1. 당사 웹사이트 중 하나에 로그인해야 합니다.
  2. 오른쪽 상단 모서리에 있는 프로필 아이콘을 클릭하면 메뉴가 나타납니다.
  3. 클릭 `Change password` 링크.
  4. 지시에 따라 비밀번호를 변경하세요.
  5. 딸깍 하는 소리 `Save`.
</Steps>


다음에 로그인할 때 새로운 비밀번호가 필요합니다.

## 비밀번호를 잊으셨나요?

때로는 사용자 비밀번호를 재설정해야 할 때가 있습니다.

사용자 비밀번호를 재설정하는 단계는 다음과 같습니다.

<Steps>
  1. 클릭해주세요 `Forgot password` 로그인 페이지에서 링크를 클릭하세요.
  2. 재설정하려는 계정의 사용자 이름/이메일을 입력하세요.
  3. 딸깍 하는 소리 `Forgot password` 단추. *비밀번호 재설정 양식으로 이동됩니다.*
  4. 이메일을 확인하세요.
</Steps>

  <figcaption>*비밀번호를 잊어버리셨나요?*</figcaption>

<Steps>
  1. 이메일로 받은 코드를 새 비밀번호와 함께 새 양식에 붙여넣으세요.
  2. 새로운 비밀번호를 확인하세요.
  3. 딸깍 하는 소리 `Reset password` 단추. *로그인 페이지로 다시 리디렉션됩니다.*
</Steps>

비밀번호가 재설정되었습니다. 새 계정으로 Wink에 로그인할 수 있습니다.
