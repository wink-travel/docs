모든 날짜가 동일한 가격으로 책정되는 것은 아닙니다. 이벤트, 휴일, 비수기, 막바지 예약 공백 등은 모두 다른 가격 책정을 필요로 합니다. 이 가이드에서는 특별 요금과 요금제 제한을 사용하여 이러한 상황을 처리하는 방법을 보여줍니다.

## 이벤트 가격 책정

회의, 축제 또는 지역 이벤트로 수요가 증가할 때 해당 날짜의 요금을 인상하고 싶을 것입니다.

<Steps>
1. Extranet에서 `Monetize > Special Rates`로 이동합니다.
2. `Create a new special rate`를 클릭합니다.
3. 재정의할 마스터 요금과 객실 유형을 선택합니다.
4. 이벤트 기간을 포함하는 날짜 범위를 설정합니다.
5. 조정된 (더 높은) 일일 요금을 입력합니다.
6. `Save`를 클릭합니다.
</Steps>

특별 요금은 해당 날짜에만 마스터 요금을 재정의합니다. 이벤트 기간 외에는 일반 가격이 변경되지 않습니다.

## 계절별 조정

장기간(성수기, 중간 시즌, 비수기)의 경우 다음 중 하나를 선택할 수 있습니다:

- **특별 요금 사용** — 조정된 가격으로 각 시즌에 대한 재정의를 만듭니다.
- **마스터 요금 업데이트** — 해당 기간의 기본 요금을 직접 조정합니다.

특별 요금은 일시적인 조정에 적합하며, 마스터 요금 업데이트는 예측 가능한 연간 패턴에 적합합니다.

## 막바지 요금

체크인 날짜에 가까운 미판매 객실을 채우려면:

<Steps>
1. 예약 제한(예: 최대 3일 전까지 예약 가능)이 있는 새 **요금제**를 만듭니다.
2. 대상 날짜에 할인을 제공하는 **특별 요금**과 짝지어 사용합니다.
3. 이 요금제는 막바지 예약 기간 내에 예약하는 여행자에게만 표시됩니다.
</Steps>

## 블랙아웃 날짜

특정 요금제에서 특정 날짜를 차단하려면(예: 성수기 동안 프로모션 요금 사용 금지):

<Steps>
1. `Monetize > Rate Plans`로 이동하여 요금제를 선택합니다.
2. **Restrictions** 섹션으로 이동합니다.
3. 요금 예약 가능 기간을 제한하는 **Sell date** 제한을 추가합니다.
4. 또는 요금 적용 기간을 제한하는 **Stay date** 제한을 사용합니다.
</Steps>

예를 들어, "Early Bird" 요금제를 12월(성수기) 예약에 사용할 수 없도록 제한하는 반면, 일반 요금제는 계속 열어둘 수 있습니다.

## 제한 조건 결합

정확한 제어를 위해 제한 조건을 중첩할 수 있습니다:

| 제한 조건 | 제어 내용 |
| -- | -- |
| **Sell dates** | 요금을 예약할 수 있는 시기 |
| **Stay dates** | 요금이 적용되는 숙박 날짜 |
| **Available days** | 요금이 표시되는 요일 |
| **Arrival days** | 투숙객이 도착해야 하는 요일 |
| **Departure days** | 투숙객이 출발해야 하는 요일 |
| **Required days** | 투숙객이 반드시 포함해야 하는 숙박일 |
| **Advance booking** | 예약이 이루어져야 하는 사전 기간 |
| **Length of stay** | 최소 및 최대 숙박일수 |
