맞춤 설정은 여행자에게 인벤토리가 어떻게 표시되는지 제어합니다 — 테마 색상, 기본 통화, 언어, 일정 등. 공유 가능한 링크, 임베디드 카드, 그리드, 지도, 예약 엔진 전반에 적용됩니다.

## 맞춤 설정이란?

맞춤 설정은 다음을 정의하는 구성 프로필입니다:

- **표시 통화** — 가격이 표시되는 통화.
- **표시 언어** — 콘텐츠의 기본 언어.
- **기본 일정** — 미리 채워진 체크인/체크아웃 날짜와 투숙객 수.
- **테마** — 임베디드 구성 요소에 적용되는 색상과 브랜딩.
- **구성** — 인벤토리 표시 방식(레이아웃, 정렬).

여러 시장이나 사용 사례에 맞게 여러 맞춤 설정을 만들 수 있습니다.

## 맞춤 설정 만들기

<Steps>
1. https://studio.wink.travel (또는 https://my.wink.travel의 계정)로 이동합니다.
2. 사이드바에서 `Customization`으로 이동합니다.
3. `Create a new customization`을 클릭합니다.
4. 설명이 포함된 이름을 지정합니다(예: "Europe - EUR", "Japan Market", "Default").
5. 표시 통화, 언어, 기본 일정을 설정합니다.
6. 브랜드에 맞게 테마 색상을 구성합니다.
7. `Save`를 클릭합니다.
</Steps>

## 맞춤 설정이 사용되는 곳

만들어진 맞춤 설정은 플랫폼 전반에서 참조됩니다:

| 기능 | 맞춤 설정 적용 방식 |
| -- | -- |
| **공유 가능한 링크** | 링크는 맞춤 설정의 통화, 언어, 일정을 기본값으로 사용합니다 |
| **임베디드 카드** | 카드는 테마 색상과 표시 설정으로 렌더링됩니다 |
| **그리드 및 지도** | 인벤토리 그리드와 지도 마커가 맞춤 설정을 따릅니다 |
| **예약 엔진** | 전체 예약 흐름이 맞춤 설정의 설정을 사용합니다 |
| **소셜 게시물** | AI 생성 게시물이 가격 표시를 위해 특정 맞춤 설정을 사용할 수 있습니다 |

## 여러 맞춤 설정

일반적인 패턴은 대상 시장별로 하나의 맞춤 설정을 만드는 것입니다:

- **"US Market"** — USD, 영어, 기본 2박 숙박.
- **"Europe"** — EUR, 영어, 기본 3박 숙박.
- **"Japan"** — JPY, 일본어, 기본 1박 숙박.

공유 가능한 링크를 만들거나 카드를 임베드할 때 사용할 맞춤 설정을 선택합니다. 즉, 같은 숙소라도 대상에 따라 다르게 표시할 수 있습니다.

## URL을 통한 재정의

공유 가능한 링크의 URL 매개변수를 사용해 맞춤 설정 기본값을 재정의할 수도 있습니다:

- `l=es` — 언어를 스페인어로 재정의.
- `c=EUR` — 통화를 EUR로 재정의.
- `sd=2024-08-24&n=2` — 특정 날짜와 숙박 일수로 일정을 재정의.

전체 URL 매개변수 목록은 [The Book Now Button](/ko/guides/the-book-now-button)에서 확인하세요.
