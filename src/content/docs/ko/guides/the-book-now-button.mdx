그**지금 예약하세요**'버튼'은 호텔 업계에서 흔히 쓰이는 용어로, 호텔이 해당 숙박 시설의 예약 가능 여부와 가격을 알고 여행객이 객실을 예약할 수 있도록 하는 업체와 직접 연동되는 것을 의미합니다. 이 버튼은 호텔이 자체 웹사이트에서 원하는 조건으로 객실을 판매할 수 있는 가장 간단한 방법입니다.

*많은 호텔이 채널 매니저에서 제공하는 묶음 예약 엔진을 사용합니다. 호텔 측에서는 특정 예약 엔진을 선택하기 전에 여러 예약 엔진을 조사하고 비교해 보실 것을 권장합니다.*

## 청중

이 글은 주로 웹사이트에 저희의 무료 고전환율 예약 엔진을 설치하려는 호텔을 위한 것입니다.
하지만... 저희 기술을 웹사이트에 활용하려는 모든 분들께도 유익한 글입니다.

## 링크를 통해

가장 간단한 통합을 위해서는 다음을 사용하는 것이 좋습니다.[공유 가능한 링크](/studio/shareable-links).

이를 위한 단계는 다음과 같습니다.

<Steps>
  1. https://extranet.wink.travel로 이동하세요
  2. 딸깍 하는 소리 `Sign-In / Register` 오른쪽 상단 모서리에서 사용자 계정으로 로그인하세요.
  3. 왼쪽 상단 드롭다운에서 작업하려는 속성을 선택하세요.
  4. 클릭해주세요 `Booking engine` 탐색 표시줄에서.
  5. 해당 페이지에 링크가 표시됩니다.
  6. 이 링크를 웹마스터에게 제공하고 사용하십시오. **지금 예약하세요** 단추.
  7. 당신은 ✅입니다.
</Steps>


### 고급의

링크에는 이미 선택한 URL에서 시작되는 여정이 포함되어 있습니다.`customization`.
다음과 같은 방법으로 링크를 더욱 세부적으로 맞춤 설정할 수 있습니다.

* 선택한 사용자 정의 변경 또는
* 여정을 재정의하다

이 기사에서는 후자에 초점을 맞춰보겠습니다. 후자는 여행객이 귀하의 웹사이트에서 여행 일정을 선택하고 이를 Wink의 예약 엔진으로 전달할 수 있는 자유도를 높여주기 때문입니다.

URL과 함께 자신의 여행 일정을 전달하는 방법은 다음과 같습니다.

유효한 요청 매개변수:

* **SD**시작 날짜를 추가하려면:*sd=2024-08-24*
* **N**숙박 일수를 추가하려면:*n=1* 또는
* **에드**종료 날짜 사용:*편집=2024-08-25*
* **RC**손님:*rc=a2*(성인 : 2명)
* **엘**표시 언어 요청:*l=es*
* **기음**표시 통화 요청:*c=유로*
* **피**프로모션 추가:*p=ABC123*

URL은 다음과 같습니다.

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2\&l=es\&c=EUR

더욱 고급 사용 사례의 경우 다음을 사용하여 여러 개의 객실 구성을 요청할 수도 있습니다.`+` 징후:

예:

https://trvl.as/abc123?sd=2024-08-24\&n=1\&rc=a2-ca16q1-ca8q1+a2\&l=es\&c=EUR

* 객실 유형 1 = 성인: 2명, 어린이: 2명 = 16세 + 8세.
* 객실 유형 2 = 성인: 2명.

### 예

샘플**지금 예약하세요**버튼 HTML:

*이 샘플에서는 다음을 사용한다고 가정합니다.[부트스트랩](https://getbootstrap.com)기본 CSS 라이브러리로 사용합니다.*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="Book a room"
>
BOOK NOW
</a>
```

### 기타 혜택

당사 링크 중 하나를 사용하는 데에는 다른 이점이 있습니다.`book now`버튼.

* **사회적 친화적**링크는 소셜 미디어에서 쉽게 공유될 수 있으며 표시됩니다.`dynamic pricing`선택한 이미지 위에.
* **짧은**URL은 공유하기 매우 쉽고 Instagram Bio URL과 같은 곳에 사용할 수 있습니다.

## Embed를 통해

일부 호텔은 웹사이트에 더 많은 기능과 더욱 긴밀한 통합을 원합니다. 이 예시에서는[카드](/studio/cards)웹사이트에 등록했습니다.

이를 위한 단계는 다음과 같습니다.

<Steps>
  1. https://studio.wink.travel로 이동하세요
  2. 딸깍 하는 소리 `Sign-In / Register` 오른쪽 상단 모서리에서 사용자 계정으로 로그인하세요.
  3. 왼쪽 상단에서 작업하려는 속성을 선택하세요.
  4. 클릭해주세요 `Search` 메인 탐색 바에서.
  5. 귀하의 부동산 이름을 입력하고 검색 버튼을 클릭하세요.
  6. 클릭 `Actions` 검색 결과 카드에서 링크를 선택하고 `Create > Make a Card`.
  7. 이렇게 하면 생성됩니다 `Card`.
  8. 카드가 준비되었으며 추가로 사용자 지정할 수 있는지 여부를 알려주는 메시지가 오른쪽 상단에 나타납니다.
  9. 클릭해주세요 `Tools > Cards` 메인 탐색 바에서 새 카드를 확인하세요.
  10. 클릭해주세요 `Actions` 링크 카드에 링크를 클릭하고 `Embed`.
  11. 이렇게 하면 복사할 수 있는 샘플 HTML 코드가 있는 새 창이 열립니다.
  12. 귀하의 검색을 위해 `Client-ID`, 가다 [응용 프로그램](/developers/applications).
  13. 귀하의 검색을 위해 `Configuration-ID`, 가다 [사용자 정의](/studio/customization)
  14. 당신은 ✅입니다.
</Steps>


샘플 내장 카드 HTML:

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*위의 HTML 샘플에는 필수적인 Wink CSS와 Javascript가 포함되어 있지 않습니다. 카드만 포함되어 있습니다.*
