[iam.wink.travel/my-account/security](https://iam.wink.travel/my-account/security)의 **보안** 탭은 비밀번호 없이 로그인할 수 있게 해주는 WebAuthn 자격 증명인 **패스키**를 관리하는 곳입니다.

## 패스키란 무엇인가요?

패스키는 본인임을 증명하는 기기(또는 비밀번호 관리자)에 연결된 암호화 자격 증명입니다. 패스키는 피싱에 강하며 비밀번호를 입력하는 것보다 빠릅니다.

## 패스키 등록하기

1. 기존 자격 증명으로 [iam.wink.travel](https://iam.wink.travel)에 로그인합니다.
2. **내 계정 → 보안**으로 이동합니다.
3. `패스키 추가`를 클릭합니다.
4. 브라우저가 기기 인증을 요청합니다(Touch ID, Face ID, Windows Hello, 하드웨어 키 또는 동기화된 비밀번호 관리자).
5. 나중에 쉽게 알아볼 수 있도록 패스키에 친근한 이름(예: *업무용 MacBook*)을 지정합니다.

## 패스키로 로그인하기

[iam.wink.travel](https://iam.wink.travel) 로그인 화면에서 이메일을 입력하면, 해당 이메일에 등록된 패스키가 있을 경우 비밀번호 대신 패스키로 인증하라는 안내가 표시됩니다.

## 패스키 제거하기

같은 보안 탭에서 등록된 각 패스키 옆에 `제거` 버튼이 있습니다. 기기를 분실했거나 하드웨어 키를 더 이상 사용하지 않거나 사용하지 않는 자격 증명을 정리하고 싶을 때 사용하세요.

:::caution
모든 패스키를 제거하면 비밀번호만 로그인 방법으로 남게 됩니다. 마지막 패스키를 제거하기 전에 비밀번호를 꼭 기억하고 있는지 확인하세요.
:::
