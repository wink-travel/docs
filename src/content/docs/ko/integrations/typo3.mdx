**Wink TYPO3 Extension**은 Wink 여행 인벤토리를 TYPO3 13에 네이티브 콘텐츠 요소로 통합합니다. 편집자는 표준 TYPO3 페이지 편집기를 사용하여 페이지에 Wink 컴포넌트를 추가할 수 있으며, 템플릿 변경이 필요하지 않습니다.

## 요구 사항

- TYPO3 13.x
- PHP 8.2 이상
- Composer

## 설치

<Steps>
1. Composer로 설치:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. 확장 기능 활성화:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. TYPO3 백엔드에서 **Admin Tools → Settings → Extension Configuration → wink**로 이동합니다.
4. **Client ID**, **Client Secret**을 입력하고 환경을 선택한 후 저장합니다.
   자격 증명은 [Applications](/ko/portal/settings/applications)에서 확인할 수 있습니다.
5. 루트 템플릿에 정적 TypoScript 템플릿을 추가합니다:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## 페이지에 콘텐츠 요소 추가하기

<Steps>
1. **Page** 모듈에서 페이지를 엽니다.
2. 콘텐츠 영역에서 **+** 버튼을 클릭하여 새 요소를 추가합니다.
3. **Wink Travel** 콘텐츠 요소 중 하나를 선택합니다:
   - **Wink Content** — 호텔 카드, 순위 그리드, 지도
   - **Wink Lookup** — 목적지 검색 바
   - **Wink Search** — 여정 선택 버튼
   - **Wink Account** — 로그인 버튼
   - **Wink Itinerary** — 여정 상태 버튼
   - **Wink Cart** — 쇼핑 카트 버튼
4. **Wink Content**의 경우 FlexForm 필드를 사용하여 **Layout Type**을 선택하고 **Layout ID**를 입력합니다.
5. 페이지를 저장하고 게시합니다.
</Steps>

## 사용 가능한 콘텐츠 요소

| 요소 | 용도 |
|---|---|
| **Wink Content** | 호텔 카드, 순위 그리드, 지도 등 |
| **Wink Lookup** | 목적지 검색 바 |
| **Wink Search** | 여정 선택기를 여는 아이콘 버튼 |
| **Wink Account** | 로그인 / 사용자 계정 버튼 |
| **Wink Itinerary** | 현재 여행을 보여주는 여정 버튼 |
| **Wink Cart** | 쇼핑 카트 요약 버튼 |
