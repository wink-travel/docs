**Wink Shopify App**은 투어 운영자, 체험 판매자, 호스피탤러티 상인이 Shopify 스토어프론트에서 제품과 함께, 랜딩 페이지 또는 테마 내 어디서든 실시간 Wink 인벤토리를 직접 표시할 수 있게 해줍니다.

## 설치

<Steps>
1. [Shopify App Store](https://apps.shopify.com)로 이동하여 **Wink Travel**을 검색하세요.
2. **앱 추가**를 클릭하고 OAuth 절차를 따라 권한을 부여하세요.
3. Wink 앱 설정에서 **Client ID**와 **Client Secret**을 입력하세요.
   자격 증명은 [Applications](/ko/portal/settings/applications)에서 확인할 수 있습니다.
4. **확인 및 저장**을 클릭하세요. 성공 메시지가 연결을 확인합니다.
</Steps>

## 앱 임베드 블록 활성화

앱 임베드 블록은 Wink 스크립트와 스타일시트를 모든 페이지에 자동으로 삽입하므로 테마 코드를 수정할 필요가 없습니다.

<Steps>
1. Shopify 관리자에서 **온라인 스토어 → 테마 → 맞춤설정**으로 이동하세요.
2. 왼쪽 패널에서 **앱 임베드**를 클릭하세요.
3. **Wink Travel**을 켜세요.
4. **저장**을 클릭하세요.
</Steps>

:::note
앱 임베드 블록은 설치 시 설정된 Shopify 메타필드에서 Client ID를 읽어옵니다. Client Secret은 브라우저에 노출되지 않습니다.
:::

## 페이지에 컴포넌트 추가하기

<Steps>
1. 테마 편집기에서 Wink 콘텐츠를 추가할 페이지나 섹션을 엽니다.
2. **블록 추가**를 클릭하고 Wink 블록 중에서 선택하세요:
   - **Wink Content** — 호텔 카드, 그리드, 지도
   - **Wink Lookup** — 목적지 검색 바
   - **Wink Search** — 일정 선택 버튼
   - **Wink Account** — 로그인 버튼
   - **Wink Itinerary** — 일정 상태 버튼
   - **Wink Cart** — 쇼핑 카트 버튼
3. **Wink Content**의 경우, **레이아웃 유형**을 선택하고 Wink Studio에서 받은 **레이아웃 ID**를 입력하세요.
4. **저장**을 클릭하세요.
</Steps>

## 사용 가능한 테마 블록

| 블록 | 용도 |
|---|---|
| **Wink Content** | 호텔 카드, 순위 그리드, 지도 등 표시 |
| **Wink Lookup** | 목적지 및 호텔 검색 바 |
| **Wink Search** | 일정 선택기 열기 아이콘 버튼 |
| **Wink Account** | 로그인 / 사용자 계정 버튼 |
| **Wink Itinerary** | 현재 여행 일정 버튼 표시 |
| **Wink Cart** | 쇼핑 카트 요약 버튼 |
