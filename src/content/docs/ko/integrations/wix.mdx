**Wink Wix App**은 Wix 사이트에 Wink 웹 컴포넌트를 네이티브 위젯으로 추가하여 Wix 에디터에서 드래그 앤 드롭으로 배치할 수 있게 합니다. 사이트 소유자는 코딩이 필요 없습니다.

## 설치

<Steps>
1. [Wix 앱 마켓](https://www.wix.com/app-market)으로 이동하여 **Wink Travel**을 검색하세요.
2. **사이트에 추가**를 클릭하고 앱 권한 부여 안내에 따라 진행하세요.
3. 설치가 완료되면 **대시보드 → Wink 설정**을 열고 **클라이언트 ID**와 **클라이언트 시크릿**을 입력하세요.
   자격 증명은 [애플리케이션](/ko/portal/settings/applications)에서 확인할 수 있습니다.
4. **저장**을 클릭한 후 **연결 테스트**를 눌러 Wink 연결 상태를 확인하세요.
</Steps>

## 페이지에 컴포넌트 추가하기

<Steps>
1. **Wix 에디터**를 열고 Wink 콘텐츠를 추가할 페이지로 이동하세요.
2. **추가 (+) → 앱 → Wink Travel**을 클릭하여 사용 가능한 모든 위젯을 확인하세요.
3. 위젯을 페이지로 드래그하세요 — 예를 들어 호텔 그리드를 표시하는 **Wink Content**.
4. 위젯 설정 패널에서 **레이아웃 유형**을 선택하고 Wink Studio에서 복사한 **레이아웃 ID**를 붙여넣으세요.
5. **게시**를 클릭하여 변경 사항을 라이브로 만드세요.
</Steps>

## 사용 가능한 위젯

| 위젯 | 용도 |
|---|---|
| **Wink Content** | 호텔 카드, 순위 그리드, 지도 등 표시 |
| **Wink Lookup** | 목적지 및 호텔 검색 바 |
| **Wink Search** | 일정 선택기를 여는 아이콘 버튼 |
| **Wink Account** | 로그인 / 사용자 계정 버튼 |
| **Wink Itinerary** | 현재 여행 일정을 보여주는 버튼 |
| **Wink Cart** | 쇼핑 카트 요약 버튼 |
