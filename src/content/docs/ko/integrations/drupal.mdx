**Wink Drupal Module**은 Wink 여행 인벤토리 컴포넌트를 Drupal 사이트에 네이티브 블록으로 추가합니다. 편집자는 표준 블록 시스템을 통해 레이아웃 어디에나 배치할 수 있습니다.

## 요구 사항

- Drupal 10 또는 11
- PHP 8.1 이상
- Composer

## 설치

<Steps>
1. Composer로 설치:
   ```bash
   composer require drupal/wink
   ```
2. 모듈 활성화:
   ```bash
   drush en wink -y
   ```
3. **관리자 → 구성 → 웹 서비스 → Wink 설정** (`/admin/config/wink`)으로 이동합니다.
4. **Client ID**와 **Client Secret**을 입력하고, 환경을 선택한 후 **저장**을 클릭합니다.
   자격 증명은 [Applications](/ko/portal/settings/applications)에서 확인할 수 있습니다.
</Steps>

## 페이지에 블록 추가하기

<Steps>
1. **관리자 → 구조 → 블록 레이아웃**으로 이동합니다.
2. Wink 콘텐츠를 배치할 영역에서 **블록 배치**를 클릭합니다.
3. **Wink**를 검색하여 사용 가능한 블록을 찾습니다:
   - **Wink Content** — 호텔 카드, 그리드 또는 지도 렌더링
   - **Wink Lookup** — 목적지 검색 바
   - **Wink Search** — 여행 일정 선택 버튼
   - **Wink Account** — 로그인 버튼
   - **Wink Itinerary** — 여행 일정 상태 버튼
   - **Wink Cart** — 쇼핑 카트 버튼
4. **Wink Content**의 경우, 블록 설정에서 **레이아웃 유형**과 **레이아웃 ID**를 구성합니다.
5. 블록 구성을 저장합니다.
</Steps>

## 사용 가능한 블록

| 블록 | 용도 |
|---|---|
| **Wink Content** | 호텔 카드, 순위 그리드, 지도 등 표시 |
| **Wink Lookup** | 목적지 및 호텔 검색 바 |
| **Wink Search** | 여행 일정 선택기 열기 아이콘 버튼 |
| **Wink Account** | 로그인 / 사용자 계정 버튼 |
| **Wink Itinerary** | 현재 여행 일정 표시 버튼 |
| **Wink Cart** | 쇼핑 카트 요약 버튼 |
