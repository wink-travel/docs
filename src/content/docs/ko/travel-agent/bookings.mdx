Wink에서 예약이 이루어지면 다음과 같은 일이 발생합니다:

<Steps>
1. 귀하의 채널 매니저, CRS 또는 PMS에 알림이 전송됩니다.
2. 새 예약을 관리할 수 있는 링크가 포함된 확인 이메일 *(예약 데스크 연락처)*를 받습니다.
3. 여행자는 예약 데스크 연락처 정보가 포함된 확인 이메일을 받습니다.
4. 여행사도 여행자와 동일한 예약 확인 이메일을 받습니다.
4. 개발자는 `booking.created` 웹훅 이벤트를 수신하여 실시간으로 상세 예약 정보를 받을 수 있습니다.
</Steps>

이메일 내 링크를 클릭하거나, 메인 내비게이션 바에서 `Account > Bookings`로 이동하여 시작하세요.

## 예약 그리드

예약 그리드는 과거 및 미래의 모든 예약을 예약일 기준 오름차순으로 보여줍니다.

### 필터

필터를 적용하여 원하는 예약을 찾을 수 있습니다.

- 예약일 [범위]
- 도착일 [범위]
- 출발일 [범위]
- 예약 코드
- 여행자 성
- 여행자 이름
- 마스터 요금 이름
- 숙소 이름

### 내보내기

현재 필터 조건에 맞는 예약을 CSV로 내보낼 수 있습니다.

예약을 내보내려면:

<Steps>
1. 원하는 필터가 활성화되어 있는지 확인합니다.
2. 예약 그리드가 새로고침될 때까지 기다립니다.
2. 그리드 오른쪽 하단의 🗂️ (*파일 내보내기*) 아이콘 버튼을 클릭합니다.
</Steps>

## 예약

예약 그리드에서 작업할 예약을 선택하면 `Booking` 페이지로 리디렉션됩니다.
이 페이지에는 도착 당일 손님을 적절히 맞이하는 데 필요한 모든 세부 정보가 포함되어 있습니다.

또한 이 페이지에는 손님 서비스를 원활히 하기 위한 몇 가지 중요한 작업이 포함되어 있습니다. 페이지 상단의 `Actions` 드롭다운에서 모든 사용 가능한 작업을 확인할 수 있습니다.

### 이메일 재전송

여행자에게 확인 이메일을 다시 보내려면 `Actions > Resend confirmation email`을 클릭하세요.

### 예약 취소

숙소 측에서 예약을 취소하려면 `Actions > Cancel booking`을 클릭하세요.

팝업 창이 나타나며 다음을 수행할 수 있습니다:

<Steps>
1. 취소 유형을 선택합니다. *예: 날짜 오류*
2. 취소 사유를 입력합니다.
</Steps>

*이 버튼은 귀하의 취소 정책 규칙을 따르므로 항상 사용 가능하지 않을 수 있습니다.*

## API

여행사로서 `Bookings`를 관리하려는 개발자는 [Developers > API > Travel Agent](/ko/developers/apis/#travel-agent-api)로 이동하세요.
