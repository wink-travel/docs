등록된 여행사를 위한 당사의 솔루션은 [Wink Agent](https://agent.wink.travel)입니다. 에이전트가 고객을 대신해 검색하고 예약할 수 있는 셀프 서비스 포털이며, 각 예약마다 수수료를 받을 수 있습니다.

더 고급 여행사는 결제 및 지급 관리를 Wink 플랫폼 외부에서 처리하기 위해 API 수준에서 통합을 선택할 수 있습니다.

여행사는 [Wink Studio](/ko/studio/what-is-studio)와 동일한 포털 기능의 일부만 사용할 수 있으므로 여기서는 다시 다루지 않습니다.

## 검색

호텔을 검색하고 예약하는 방법은 [검색](/ko/studio/search)에서 확인하세요.

그렇지 않으면 다음 섹션으로 계속 진행하세요.

## API

`Travel Agent`를 관리하려는 개발자는 [Developers > API > Travel Agent](/ko/developers/apis/#travel-agent-api)로 이동하세요.
