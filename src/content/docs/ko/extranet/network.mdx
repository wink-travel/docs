## 판매 채널

판매 채널을 관리하려면 메인 내비게이션 바에서 `Distribution > Sales channels`를 클릭하세요.

판매 채널은 제휴사 관계를 나타내며, 이들이 접근할 수 있는 재고, 거래 및 할인 조건이 명시되어 있습니다.

기존 판매 채널을 업데이트하려면, 업데이트하려는 판매 채널 아래의 `Actions` 링크를 클릭하세요.

- **회원 할인** 이 판매 채널을 통해 예약하는 여행자에게 제공되는 회원 요금입니다.
- **커미션** 성공적인 예약 건마다 판매 채널에 할당되는 커미션입니다.
- **단일 프로모션** 이 판매 채널에 적용할 단일 프로모션을 추가합니다.
- **묶음 프로모션** 이 판매 채널에 적용할 묶음 프로모션을 추가합니다.
- **활성화됨** 활성화 스위치를 토글하여 판매 채널을 완전히 활성화하거나 비활성화합니다.
- **블랙리스트 등록됨** 판매 채널을 블랙리스트로 지정하면 해당 판매 채널을 통해 예약할 수 없게 됩니다.

설정이 완료되면 `Save` 버튼을 클릭하여 계속 진행하세요.

:::tip
판매 채널을 블랙리스트로 지정하면 해당 판매 채널이 귀하로부터 어떠한 요금도 받을 수 없게 됩니다; Wink Network 요금도 포함됩니다.
:::

## 네트워크 탐색

Wink에서 새로운 제휴사를 찾으려면 메인 내비게이션 바에서 `Distribution > Explore network`를 클릭하세요.

이 섹션에서는 페이지 상단에 제휴사의 대기 요청을, 하단에는 연결할 수 있는 제휴사 목록을 볼 수 있습니다.

### 요청에 응답하기

제휴사 요청을 거절할 수 있습니다. 요청을 계속 진행하려면 `affiliate details` 페이지로 이동하며, 여기서 제휴사에 대한 자세한 정보와 Wink에서의 과거 성과 지표를 확인할 수 있습니다.

### 제휴사 검색

제휴사 그리드에서 이름, 유형, 위치별로 제휴사를 검색할 수 있습니다. 원하는 제휴사를 찾으면 클릭하여 `affiliate details` 페이지로 이동합니다.

### 제휴사와 연결하기

제휴사와 직접 관계를 맺기로 선택하면, 위에서 판매 채널을 관리하는 것과 동일한 방식으로 관계를 명시하도록 요청받습니다.

새 판매 채널을 저장하면 판매 채널 목록으로 돌아갑니다.

## API

`Network`를 관리하려는 개발자는 [Developers > APIs > Distribution](/ko/developers/apis/#distribution-api)로 이동할 수 있습니다.
