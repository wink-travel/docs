설정 과정에서 지도 위치를 설정하라는 요청을 받게 됩니다.

숙소의 지도 위치를 관리하려면 메인 내비게이션 탭에서 `Property > Map location`을 클릭하세요.

지도 위치는 두 가지 방법으로 설정할 수 있습니다:

1. 지도 위 검색 필드에 숙소 이름을 입력하여 검색한 후 선택합니다.
2. 지도를 확대/축소하고 이동하여 숙소 위치를 찾은 다음, 해당 위치를 클릭합니다.

업데이트된 좌표와 함께 지도 마커가 표시됩니다.

:::note
숙소가 Google Pages에 등록되어 있어야 검색 필드에 표시됩니다.
:::

양식을 업데이트한 후 `Save` 버튼을 클릭하여 계속 진행하세요.
