에이전시는 대리 목적을 위해 귀하의 계정에 할당할 수 있는 일반 Wink 계정입니다. 에이전시가 귀하를 대리하는 이유는 다음과 같습니다:

- 제3자가 귀하의 Wink 계정을 전면 또는 부분적으로 관리.
- 부동산이 기존 관리 회사와 계약을 맺었으며 Wink를 통해 들어오는 자금의 재정적 측면을 Wink가 처리하기를 원함.
- 부동산 관리 에이전시가 해당 부동산을 대신해 모든 온라인 업무를 수행.

## 에이전시 할당

계정에 에이전시를 할당하는 방법은 다음과 같습니다.

<Steps>
 1. [Wink Extranet](https://extranet.wink.travel)에서 사용자 계정에 로그인합니다.
 2. 작업할 부동산을 선택합니다.
 3. 상단 내비게이션 바에서 `Account > Managers`를 클릭합니다.
 4. `Agency` 섹션에서 귀하를 대리할 에이전시 이름을 입력합니다.
 5. 드롭다운에서 에이전시 수수료를 선택합니다.
 5. 선택적으로 만료 규칙을 추가합니다.
 7. 계속하려면 `Set agency` 버튼을 클릭합니다.
 </Steps>

 :::note
에이전시를 할당하려면 부동산 소유자여야 합니다.
 :::

 :::tip[예시]
에이전시 수수료는 예약 금액에서 산출됩니다.

예약 = $100, 에이전시 수수료 = 1% => 100 * 0.01 = $1
 :::

## 에이전시 해제

계정에서 에이전시를 해제하는 방법은 다음과 같습니다.

<Steps>
 1. [Wink Extranet](https://extranet.wink.travel)에서 사용자 계정에 로그인합니다.
 2. 작업할 부동산을 선택합니다.
 3. 상단 내비게이션 바에서 `Account > Managers`를 클릭합니다.
 4. `Agency` 섹션에서 에이전시 드롭다운 오른쪽에 있는 <Badge text="x" variant="danger" size="small" /> 아이콘을 클릭합니다.
 5. 계속하려면 `Set agency` 버튼을 클릭합니다.
 </Steps>


## API

API를 통해 에이전시를 관리하려는 개발자는 [Developers > API > Affiliate](/developers/apis/#affiliate-api)로 이동할 수 있습니다.
