소셜 미디어는 새로운 여행자와 기존 여행자와 지속적으로 소통할 수 있게 해줍니다.
프로필에 소셜 네트워크 계정을 추가하여 게스트가 쉽게 연락하고 숙소에 대해 알 수 있도록 하세요.

소셜 네트워크 계정을 관리하려면 메인 내비게이션 탭에서 `Property > Social media`를 클릭하세요.

<Steps>
1. 활성화하려는 소셜 네트워크의 스위치를 토글하세요.
2. 해당 네트워크에서 사용하는 계정 이름을 입력하세요.
</Steps>

계속하려면 `Save` 버튼을 클릭하세요.
