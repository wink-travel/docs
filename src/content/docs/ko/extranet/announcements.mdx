공지사항은 중요한 정보를 게스트에게 눈에 띄게 알리는 방법입니다.
공지사항이 `트리거`되면, 랜딩 페이지 위에 새 창으로 표시됩니다.

공지사항을 보려면, 메인 내비게이션 바에서 `Property > Announcement`를 클릭하세요.

`Create announcement` 버튼을 클릭하여 새 항목을 만드세요.

<Steps>
1. 제목 포함 여부를 선택하세요.
2. 공지사항은 두 가지 방법 중 하나로 트리거할 수 있습니다:
    - 항상 표시, 방문할 때마다 공지사항이 나타납니다.
    - 일정이 공지사항 일정과 일치할 때만 표시됩니다.
3. 특정 날짜 범위에만 공지사항을 표시하도록 선택했다면, 아래 양식 필드에 날짜를 입력하세요.
4. `Save` 버튼을 클릭하면 공지사항 목록으로 돌아갑니다.
</Steps>
