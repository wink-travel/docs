Wink에서 예약이 이루어지면 다음과 같은 일이 발생합니다:

<Steps>
1. 귀하의 채널 매니저, CRS 또는 PMS에 알림이 전송됩니다.
2. 새 예약을 관리할 수 있는 링크가 포함된 확인 이메일 *(예약 데스크 연락처)* 을 받습니다.
3. 여행객은 예약 데스크 연락처 정보가 포함된 확인 이메일을 받습니다.
4. 개발자는 `booking.created` 웹훅 이벤트를 수신하여 실시간으로 상세 예약 정보를 받을 수 있습니다.
</Steps>

이메일 내 링크를 클릭하거나 메인 내비게이션 바에서 `Account > Bookings`로 이동하여 시작하세요.

## 예약 그리드

예약 그리드는 과거 및 미래의 모든 예약을 예약일 기준 오름차순으로 보여줍니다.

### 필터

필터를 적용하여 원하는 예약을 찾을 수 있습니다.

- 예약일 [범위]
- 도착일 [범위]
- 출발일 [범위]
- 예약 코드
- 여행객 성
- 여행객 이름
- 마스터 요금 이름
- 판매 채널 / 출처

### 내보내기

현재 필터 조건에 맞는 예약을 CSV로 내보낼 수 있습니다.

예약을 내보내려면:

<Steps>
1. 원하는 필터가 활성화되어 있는지 확인합니다.
2. 예약 그리드가 새로고침될 때까지 기다립니다.
3. 그리드 오른쪽 하단의 🗂️ (*파일 내보내기*) 아이콘 버튼을 클릭합니다.
</Steps>

## 예약

예약 그리드에서 작업할 예약을 선택하면 `Booking` 페이지로 리디렉션됩니다.
이 페이지에는 도착 당일 손님을 적절히 맞이하는 데 필요한 모든 세부 정보가 포함되어 있습니다.

또한 이 페이지에는 손님 서비스를 위해 필요한 몇 가지 중요한 작업이 포함되어 있습니다. 페이지 상단의 `Actions` 드롭다운에서 모든 사용 가능한 작업을 찾을 수 있습니다.

### 이메일 재전송

여행객에게 확인 이메일을 다시 보내려면 `Actions > Resend confirmation email`을 클릭하세요.

### 예약 취소

숙소 측에서 예약을 취소하려면 `Actions > Cancel booking`을 클릭하세요.

팝업 창이 나타나며 다음을 수행할 수 있습니다:

<Steps>
1. 취소 유형 선택. *예: 잘못된 날짜*
2. 취소 사유 입력.
</Steps>

*이 버튼은 귀하의 취소 정책 규칙을 따르므로 항상 사용 가능하지 않을 수 있습니다.*

### 환불 요청

어떤 이유로든 환불을 요청하려면 `Actions > Request refund`를 클릭하세요.

팝업 창이 나타나며 다음을 수행할 수 있습니다:

<Steps>
1. 전액 또는 부분 환불 요청.
    - 부분 환불을 선택하면 금액을 입력하라는 요청이 표시됩니다.
2. 사유 선택.
3. 요청 사유를 상세히 입력.
4. 요청 승인 여부와 관계없이 예약을 취소할지 여부를 알려주세요.
</Steps>

*이 작업은 해당 예약에 대한 자금 처리가 완료되기 전까지 가능합니다.*

## 캘린더 동기화

Wink 예약을 선호하는 캘린더 소프트웨어와 동기화하려면 메인 내비게이션 바에서 `Distribution > Sync bookings to your calendar`로 이동하여 시작하세요.

처음에는 캘린더 소프트웨어와 연결을 인증하는 데 사용할 수 있는 패스 코드를 생성하라는 요청을 받게 됩니다.

    <figcaption>*CalDAV 패스 코드 생성을 요청하는 화면*</figcaption>
</figure>

계속하려면 `Yes` 버튼을 클릭하거나 마음이 바뀌면 `No`를 클릭하세요.

Wink와 캘린더 소프트웨어를 연결하는 데 필요한 3가지 필드가 표시됩니다:

1. 예약 구독에 사용할 URL.
2. 인증에 사용할 사용자 이름.
3. 인증에 사용할 패스 키.

이 단계를 완료하고 CalDAV URL을 캘린더 소프트웨어에 추가하면 캘린더에 Wink 예약이 표시되기 시작합니다.
