Wink는 주요 채널 매니저, 부동산 관리 시스템(PMS), 중앙 예약 시스템(CRS)과 통합하여 모든 판매 채널에서 요금과 가용성을 동기화 상태로 유지합니다.

## 작동 방식

채널 매니저 통합은 부동산 관리 시스템과 Wink 간의 양방향 데이터 흐름을 가능하게 합니다:

1. **요금 푸시** — 채널 매니저가 요금 업데이트를 Wink에 전송합니다. PMS에서 가격을 업데이트하면 해당 변경 사항이 자동으로 Wink에 반영됩니다.
2. **가용성 동기화** — 채널 매니저가 가용성 업데이트를 Wink에 전송합니다. PMS에서 객실이 예약되거나 해제되면 Wink의 재고가 실시간으로 업데이트됩니다.
3. **예약 알림** — Wink에서 예약이 이루어지면 채널 매니저에 알림이 전송되어 예약이 PMS에 표시됩니다.

## 지원되는 채널 매니저

Wink는 현재 다음 시스템과 통합됩니다:

| 채널 매니저 | 설명 |
|---|---|
| **Channex** | 글로벌 유통 및 채널 관리 플랫폼. |
| **TravelClick** | 호텔 솔루션 및 채널 관리(Amadeus). |
| **Omnibees** | 호텔용 유통 및 수익 관리. |
| **D-Edge** | 호스피탈리티 기술 및 중앙 예약 솔루션. |
| **Allotz** | 재고 할당 및 채널 관리. |
| **Ezee / Yanolja** | 아시아 시장에 중점 둔 클라우드 PMS 및 채널 관리. |
| **RateGain** | 요금 인텔리전스 및 유통 기술. |
| **RateTiger** | 실시간 요금 및 가용성 유통. |
| **CloudBeds** | 클라우드 기반 PMS, 채널 매니저, 예약 엔진. |
| **SiteMinder** | 선도적인 호텔 채널 매니저 및 유통 플랫폼. |
| **Exely** | 호텔용 요금 및 가용성 동기화. |

:::note
채널 매니저가 목록에 없으면 Wink 팀에 문의하세요. Wink가 지원하는 OTA(Open Travel Alliance) 프로토콜 표준을 기반으로 새로운 통합을 추가할 수 있습니다.
:::

## 개발자를 위한 정보

Wink와 채널 매니저 또는 PMS 통합을 구축하는 경우, 채널 매니저 API 엔드포인트를 사용하세요:

- **요금 업데이트** — 요금 변경 사항을 Wink에 푸시합니다.
- **가용성 업데이트** — 가용성 변경 사항을 Wink에 푸시합니다.
- **예약 조회** — Wink에서 예약을 시스템으로 가져옵니다.

채널 매니저 API는 기존 호스피탈리티 시스템과의 호환성을 위해 OTA 프로토콜 표준(SOAP/XML)을 따릅니다.

## 웹훅 알림

채널 매니저 웹훅 이벤트를 구독하여 실시간 알림을 받을 수 있습니다:

- `channel-manager.update.rate` — 요금 업데이트 수신.
- `channel-manager.update.availability` — 가용성 업데이트 수신.
- `channel-manager.update` — 일반 채널 매니저 업데이트.

자세한 내용은 [Webhook Events Catalog](/ko/developers/webhook-events#channel-manager-events)를 참조하세요.
