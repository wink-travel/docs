저희의 모든 REST API는 언어에 구애받지 않으며 Open API 규격을 준수합니다. 한 줄의 코드도 작성하지 않고 바로 테스트를 시작하세요.

아직 Wink 계정을 생성하지 않으셨다면, [여기서 시작하세요](/user/register).

## Ping API

Ping API는 자격 증명이 정상 작동하는지 빠르게 확인할 수 있는 테스트 엔드포인트입니다.

<LinkButton href="https://developer.wink.travel/ping" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>


## Notifications API

Notifications API는 사용자, 숙소 또는 제휴 계정과 소통할 수 있는 방법입니다.

<LinkButton href="https://developer.wink.travel/notifications" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

## User Settings API

User Settings API는 서드파티 통합자가 Wink와 소통할 수 있도록 엔드포인트를 제공합니다.

<LinkButton href="https://developer.wink.travel/user-settings" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

## Managing Entity API

Managing Entity API는 사용자가 플랫폼에서 접근할 수 있는 엔티티를 빠르고 간단하게 확인할 수 있는 방법입니다.

<LinkButton href="https://developer.wink.travel/managing-entity" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

## Consume endpoints

Consume 엔드포인트는 기존 여행 인벤토리를 찾아 예약하거나 Wink 제휴 계정을 통해 광고에 활용하려는 개발자를 위한 것입니다.

### Configuration API

예약 엔진의 화이트라벨 및 맞춤화 정보를 가져오는 단일 엔드포인트입니다.

<LinkButton href="https://developer.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Lookup API

지리적 지역별 맞춤 여행 인벤토리를 검색할 수 있는 개발자 친화적인 방법입니다.

<LinkButton href="https://developer.wink.travel/lookup" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Inventory API

Lookup API를 통해 공식적으로 위치가 확인된 여행 인벤토리를 개발자 친화적으로 조회할 수 있는 방법입니다.

<LinkButton href="https://developer.wink.travel/inventory" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Booking API

플랫폼에서 찾은 인벤토리를 개발자 친화적으로 예약할 수 있는 방법입니다.

<LinkButton href="https://developer.wink.travel/booking" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Travel Agent API

Travel Agent API는 에이전트가 중개하는 예약을 관리할 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://developer.wink.travel/travel-agent" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>


## Produce endpoints

Produce 엔드포인트는 여행 인벤토리를 생성하고 관리하려는 개발자를 위한 것입니다.

### Registration API

프로듀서로서 여기가 여정을 시작하는 곳입니다. 이 엔드포인트를 통해 Wink에 숙소를 생성할 수 있습니다.

<LinkButton href="https://developer.wink.travel/extranet/property/register" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Property API

이 숙소 관련 엔드포인트 모음은 주로 기존 숙소를 표시, 상태 변경 등 관리할 수 있는 엔드포인트입니다.

<LinkButton href="https://developer.wink.travel/extranet/property" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Facilities API

Facilities API는 Wink에서 경험을 관리할 수 있는 엔드포인트를 제공합니다; 예를 들어 객실 유형 등이 있습니다.

<LinkButton href="https://developer.wink.travel/extranet/facilities" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Experiences API

Experiences API는 Wink에서 경험을 관리할 수 있는 엔드포인트를 제공합니다; 예를 들어 액티비티 등이 있습니다.

<LinkButton href="https://developer.wink.travel/extranet/experiences" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Monetize API

Monetize API는 Wink에서 취소 정책, 요금제, 프로모션 등을 관리할 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://developer.wink.travel/extranet/monetize" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Distribution API

Distribution API는 Wink에서 판매 채널, 제휴 연결, 요금 및 인벤토리 캘린더 관리 등을 할 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://developer.wink.travel/extranet/distribution" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Property Booking API

Property Booking API는 숙소 단위에서 예약 및 리뷰를 관리할 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://developer.wink.travel/extranet/booking" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

### Affiliate API

이 제휴 관련 엔드포인트 모음은 주로 기존 계정을 표시, 상태 변경 등 관리할 수 있는 엔드포인트입니다.

<LinkButton href="https://developer.wink.travel/affiliate" variant="secondary" icon="external" target="_blank">

</LinkButton>

### Browse API

Browse API는 제휴사가 판매할 공급자와 인벤토리를 찾을 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://developer.wink.travel/affiliate/browse" variant="secondary" icon="external" target="_blank">

### Inventory API

Lookup API를 통해 공식적으로 위치가 확인된 여행 인벤토리를 개발자 친화적으로 조회할 수 있는 방법입니다.

<LinkButton href="https://developer.wink.travel/affiliate/inventory" variant="secondary" icon="external" target="_blank">

<LinkButton href="https://developer.wink.travel/affiliate/sales-channel" variant="secondary" icon="external" target="_blank">

</LinkButton>

### WinkLinks API

WinkLinks API는 제휴사가 자신의 WinkLinks 페이지를 관리할 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://developer.wink.travel/affiliate/winklinks" variant="secondary" icon="external" target="_blank">

</LinkButton>

### Channel Manager API

Channel Manager API는 외부 채널 매니저 파트너가 외부 인벤토리를 Wink와 매핑하고 요금 및 가용성 정보를 전송하며, Wink 플랫폼에서 발생하는 예약 정보를 숙소별로 통보받을 수 있게 합니다.

<LinkButton href="https://developer.wink.travel/channel-manager" variant="secondary" icon="external" target="_blank">

</LinkButton>

## Taxonomy API

Taxonomy 엔드포인트는 여행 인벤토리를 소비하고 생성하려는 개발자를 위한 것으로, 인벤토리 유형, 클래스, 상태 등에 대한 표준 및 비표준 코드 분류 체계를 제공합니다.

<LinkButton href="https://developer.wink.travel/reference" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

## Analytics API

Analytics API는 리더보드 순위 추적과 시계열, 플랫폼 수준 데이터에 대한 심층 분석을 원하는 누구에게나 다양한 지표를 제공하는 엔드포인트를 제공합니다.

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>

## Payment API

Payment 엔드포인트는 여행 인벤토리를 구매하려는 개발자를 위한 것입니다. 등록된 여행사로서 API를 통해 구매하거나, PCI 준수 결제 위젯과 함께 API를 사용하여 다른 모든 엔티티가 구매할 수 있습니다.
Payment API는 제휴사와 호텔이 예약, 분석, 자금 가용성을 추적할 수 있는 엔드포인트를 제공합니다. 또한 가용 자금을 은행 계좌로 출금할 수도 있습니다.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
자세히 알아보기
</LinkButton>
