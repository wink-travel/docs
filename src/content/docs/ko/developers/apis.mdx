모든 REST API는 언어에 구애받지 않으며 Open API를 준수합니다. 코드 한 줄도 작성하지 않고도 테스트를 시작할 수 있습니다.

아직 Wink 계정을 생성하지 않은 경우[여기서 시작하세요](/user/register).

## 엔드포인트 사용

소비 엔드포인트는 기존 여행 상품 목록을 찾아 예약하거나 Wink 제휴 계정 중 하나를 통해 광고하는 데 사용하려는 개발자를 위한 것입니다.

### 구성 API

예약 엔진에 대한 화이트라벨 + 맞춤형 정보를 검색하는 단일 엔드포인트입니다.

<LinkButton href="https://docs.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 조회 API

지리적으로 다양한 지역에서 맞춤형 여행 상품 재고를 검색하는 프로그래머 친화적인 방법입니다.

<LinkButton href="https://docs.wink.travel/lookup" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 인벤토리 API

Lookup API를 사용하여 공식적으로 위치한 알려진 여행 재고를 검색하는 프로그래머 친화적인 방법입니다.

<LinkButton href="https://docs.wink.travel/inventory" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 예약 API

우리 플랫폼에서 발견된 프로그래머 친화적인 재고 예약 방법입니다.

<LinkButton href="https://docs.wink.travel/booking" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 여행사 API

여행사 API는 여행사가 진행하는 예약을 관리하기 위한 엔드포인트를 제공합니다.

<LinkButton href="https://docs.wink.travel/travel-agent" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


## 엔드포인트 생성

생산 엔드포인트는 여행 재고를 만들고 관리하려는 개발자를 위한 것입니다.

### 등록 API

프로듀서로서, 종종 이것이 여정의 시작점입니다. 이러한 엔드포인트를 통해 Wink에서 콘텐츠를 제작할 수 있습니다.

<LinkButton href="https://docs.wink.travel/extranet/property/register" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 속성 API

이러한 속성 엔드포인트 컬렉션은 대부분 기존 속성에 대한 표시, 상태 변경 등을 수행할 수 있는 관리 엔드포인트입니다.

<LinkButton href="https://docs.wink.travel/extranet/property" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 시설 API

시설 API는 Wink에서의 경험을 관리하기 위한 엔드포인트(예: 객실 유형)를 제공합니다.

<LinkButton href="https://docs.wink.travel/extranet/facilities" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 경험 API

Experiences API는 Wink에서 경험을 관리하기 위한 엔드포인트(활동 등)를 제공합니다.

<LinkButton href="https://docs.wink.travel/extranet/experiences" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 수익화 API

Monetize API는 Wink에서 취소 정책, 요금제, 프로모션 등을 관리하기 위한 엔드포인트를 제공합니다.

<LinkButton href="https://docs.wink.travel/extranet/monetize" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 배포 API

Distribution API는 Wink에서 판매 채널에 대한 엔드포인트를 제공하고, 제휴사와 연결하고, 요금 및 재고 일정을 관리하는 등의 작업을 수행합니다.

<LinkButton href="https://docs.wink.travel/extranet/distribution" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 부동산 예약 API

Property Booking API는 숙박 시설 수준에서 예약 및 리뷰를 관리하기 위한 엔드포인트를 제공합니다.

<LinkButton href="https://docs.wink.travel/extranet/booking" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 제휴 API

이러한 제휴 엔드포인트 컬렉션은 대부분 기존 계정에 대한 표시, 상태 변경 등을 할 수 있는 관리 엔드포인트입니다.

<LinkButton href="https://docs.wink.travel/affiliate" variant="secondary" icon="external" target="_blank">

### API 탐색

Browse API는 제휴사가 공급업체와 판매할 재고를 찾을 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://docs.wink.travel/affiliate/browse" variant="secondary" icon="external" target="_blank">
  자세히 알아보기

### 인벤토리 API

Lookup API를 사용하여 공식적으로 위치한 알려진 여행 재고를 검색하는 프로그래머 친화적인 방법입니다.

<LinkButton href="https://docs.wink.travel/affiliate/inventory" variant="secondary" icon="external" target="_blank">
  자세히 알아보기

### 판매 채널 API

판매 채널 API는 제휴사가 기존 판매 채널을 관리하고 새로운 판매 채널을 찾을 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://docs.wink.travel/affiliate/sales-channel" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 윙크링크 API

WinkLinks API는 제휴사가 WinkLinks 페이지를 관리할 수 있는 엔드포인트를 제공합니다.

<LinkButton href="https://docs.wink.travel/affiliate/winklinks" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


### 채널 관리자 API

채널 관리자 API를 사용하면 외부 채널 관리자 파트너가 Wink를 통해 외부 인벤토리를 매핑하고 요금 및 가용성 정보를 당사에 보낼 수 있으며, Wink 플랫폼에서 해당 숙박 시설에 대해 발생하는 예약에 대한 정보도 받을 수 있습니다.

  자세히 알아보기

## 분류 API

분류 엔드포인트는 여행 인벤토리를 소비하고 생산하고 인벤토리 유형, 클래스, 상태 등에 대한 표준 및 비표준 코드의 분류가 필요한 개발자를 위한 것입니다.

<LinkButton href="https://docs.wink.travel/reference" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


## 분석 API

Analytics API는 트랙 리더보드 순위에 대한 통찰력과 시계열, 플랫폼 수준 데이터에 대한 심층적인 통찰력을 원하는 모든 사람에게 엔드포인트를 제공하며, 다양한 지표를 호스팅합니다.

<LinkButton href="https://docs.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>


## 결제 API

결제 엔드포인트는 여행 상품 구매를 원하는 개발자를 위한 것입니다. 등록된 여행사로서 API를 통해 구매하거나, 다른 모든 사업체에 PCI 호환 결제 위젯과 함께 API를 사용하여 구매할 수 있습니다.
결제 API는 제휴사 및 호텔이 예약, 분석 및 자금 가용성을 추적할 수 있도록 엔드포인트를 제공합니다. 또한, 은행 계좌로 사용 가능한 자금을 인출할 수도 있습니다.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  자세히 알아보기
</LinkButton>
