WordPress를 우리 주변에 배치합니다.[웹 구성 요소](/developers/web-components). 해당 구성 요소의 기능에 대해 자세히 알아보려면 여기를 방문하세요. 이 글에서는 WordPress에서 구성 요소를 사용하는 방법에 대해 중점적으로 설명합니다.

비즈니스나 블로그에 WordPress를 사용하는 모든 사람들에게 다음을 사용하는 것이 좋습니다.[윙크 제휴 워드프레스 플러그인](https://wordpress.org/plugins/wink2travel/).
이 플러그인을 사용하면 Wink 카드, 그리드, 지도 등에 쉽게 액세스할 수 있으며, 선호하는 페이지 빌더나 맞춤형 단축 코드를 사용하여 게시물과 페이지에 콘텐츠를 끌어다 놓는 것만큼 간편하게 사용할 수 있습니다.

## 설치하다

이렇게 설치하세요`Wink Affiliate WordPress Plugin`WordPress 웹사이트에.

<Steps>
  1. 관리자로 로그인하고 대시보드에 접속한 경우
  2. 딸깍 하는 소리 `Plugins > Add New Plugin`.
  3. 검색 `Wink Affiliate WordPress Plugin`.
  4. 딸깍 하는 소리 `Install` 귀하의 사이트에 플러그인을 다운로드하여 설치하세요.
  5. 딸깍 하는 소리 `Activate` 귀하의 사이트에서 플러그인을 활성화하세요.
  6. 활성화하면 다음이 전달됩니다. `Wink Settings` 페이지. *또한 다음을 탐색하여 이동할 수도 있습니다. `Appearance > Customize > Wink Settings`.*
  7. 그것은 당신에게 묻습니다 `Client ID` 그리고 `Client Secret`. 이동 [응용 프로그램](/developers/applications) 자격 증명을 검색하는 방법을 알아보세요.
  8. 자격 증명을 입력하고 클릭하세요 `Publish`이제 Wink 계정에 연결되었습니다.
</Steps>

## 페이지 빌더

우리는 WordPress에서 가장 인기 있는 페이지 빌더를 지원하여 Wink를 사용하고자 하는 모든 사람이 쉽게 사용할 수 있도록 해줍니다.

### 구텐베르크

구텐베르크는 워드프레스의 기본 편집기입니다.

### 아바다

[아바다](https://avada.com) 고급 기능 세트를 갖춘 강력한 페이지 빌더와 함께 다양한 아름다운 템플릿을 제공하는 것으로 유명합니다.

### 엘리멘터

[엘리멘터](https://elementor.com) \[현재] 가장 인기 있는 페이지 빌더로 무료 버전과 프로 버전이 모두 제공됩니다.

### WP베이커리

[WP베이커리](https://wpbakery.com) 시장에 출시된 최초의 고급 페이지 빌더였습니다.

## 단축 코드

### \[윙크룩업]

순위가 매겨진 콘텐츠 그리드에서 작동합니다. 방문하려는 장소를 입력하면 인벤토리가 표시됩니다.

### \[윙크서치]

여행 일정 양식을 여는 간단한 버튼입니다.

### \[윙크계정]

계정 버튼을 클릭하면 인증을 할 수 있습니다. 인증이 완료되면 계정 옵션이 있는 드롭다운 메뉴로 전환됩니다.

### \[윙키티너러리]

검색 버튼과 동일하지만, 버튼 텍스트로 여정 정보를 포함합니다.

### \[윙크내용]

Wink에서 선택한 인벤토리를 페이지나 게시물에 직접 삽입할 수 있습니다. 이 단축 코드는 레이아웃 ID도 필요로 합니다.

레이아웃

```
[winkcontent layoutid=""]
```
