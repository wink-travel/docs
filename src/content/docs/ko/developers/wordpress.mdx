워드프레스는 당사의 [Web Components](/ko/developers/web-components)를 감싸는 형태로 제공됩니다. 컴포넌트가 할 수 있는 기능에 대해 더 알고 싶다면 해당 페이지를 방문하세요. 이 문서에서는 워드프레스에서 사용하는 방법에 중점을 둡니다.
:::

비즈니스나 블로그에 워드프레스를 사용하는 분들께는 [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/) 사용을 강력히 추천합니다.
이 플러그인은 Wink 카드, 그리드, 지도 등에 쉽게 접근할 수 있게 해주며, 좋아하는 페이지 빌더나 당사의 맞춤 숏코드를 사용해 게시물과 페이지에 콘텐츠를 끌어다 놓는 것만큼 간단하게 만듭니다.

##  설치

`Wink Affiliate WordPress Plugin`을 워드프레스 웹사이트에 설치하는 방법입니다.

<Steps>
1. 관리자 계정으로 로그인하여 대시보드에 접속합니다.
2. `플러그인 > 새 플러그인 추가`를 클릭합니다.
3. `Wink Affiliate WordPress Plugin`을 검색합니다.
4. `설치`를 클릭하여 플러그인을 다운로드하고 사이트에 설치합니다.
5. `활성화`를 클릭하여 사이트에서 플러그인을 활성화합니다.
6. 활성화 후 `Wink 설정` 페이지로 이동합니다. *또는 `외모 > 맞춤설정 > Wink 설정`으로 이동할 수도 있습니다.*
7. `클라이언트 ID`와 `클라이언트 시크릿`을 입력하라는 메시지가 표시됩니다. 자격 증명 획득 방법은 [Applications](/ko/developers/applications)를 참고하세요.
8. 자격 증명을 입력하고 `게시`를 클릭합니다. 이제 Wink 계정과 연결되었습니다.
</Steps>

## 페이지 빌더

Wink 사용을 원하는 누구나 쉽게 사용할 수 있도록 워드프레스에서 가장 인기 있는 페이지 빌더를 지원합니다.

### Gutenberg

Gutenberg는 워드프레스의 기본 편집기입니다.

### Avada

[Avada](https://avada.com)는 다양한 아름다운 템플릿과 고급 기능을 갖춘 강력한 페이지 빌더로 유명합니다.

### Elementor

[Elementor](https://elementor.com)는 현재 가장 인기 있는 페이지 빌더로, 무료 및 프로 버전이 모두 제공됩니다.

### WPBakery

[WPBakery](https://wpbakery.com)는 시장에 나온 최초의 고급 페이지 빌더입니다.

## 숏코드

### [winklookup]

랭킹된 콘텐츠 그리드와 함께 작동합니다. 방문하고 싶은 장소를 입력하면 재고가 표시됩니다.

### [winksearch]

일정 양식을 열기 위한 간단한 버튼입니다.

### [winkaccount]

인증을 위한 계정 버튼입니다. 인증되면 계정 옵션이 포함된 드롭다운으로 변합니다.

### [winkitinerary]

검색 버튼과 동일하지만 버튼 텍스트에 일정 정보가 포함되어 있습니다.

### [winkcontent]

Wink에서 선택한 재고를 페이지나 게시물에 직접 삽입할 수 있습니다. 이 숏코드는 레이아웃 ID도 필요로 합니다.

레이아웃

```
[winkcontent layoutid=""]
```
