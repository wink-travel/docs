웹훅은 귀하의 계정과 관련된 Wink와 TripPay의 실시간 이벤트를 수신하고 이에 반응할 수 있는 방법입니다.

## 예

<Steps>
  1. 예약은 Wink에서 진행됩니다.
  2. 윙크 이벤트 `booking.created` 파견됩니다.
  3. 귀하의 시스템은 이 이벤트를 적극적으로 듣고 있습니다.
  4. 귀하의 시스템은 귀하의 분석 세트를 추적하고 이 이벤트를 사용하여 시계열 데이터베이스를 업데이트합니다.
</Steps>

:::참고
Wink 웹훅은 TripPay 웹훅과 다릅니다.

* Wink 이벤트를 듣고 싶다면 웹훅을 생성할 수 있습니다.[엑스트라넷](https://extranet.wink.travel) 또는 [사진관](https://studio.wink.travel).
* TripPay 이벤트를 듣고 싶다면 웹훅을 생성할 수 있습니다.[지불](https://pay.wink.travel).
  :::

*이 예에서는 다음을 통해 인증합니다.[사진관](https://studio.wink.travel).*

<Steps>
  1. 인증이 완료되면 오른쪽 상단에 있는 프로필 아이콘을 클릭하면 메뉴가 나타납니다.
  2. 클릭 `Webhooks` 링크.
  3. 귀하의 웹훅 페이지로 리디렉션됩니다.
</Steps>


## 웹훅 만들기

:::주의
웹훅을 생성하기 전에 Wink에 제휴 계정이나 TripPay에 계정을 생성해야 합니다. 웹훅을 특정 계정에 연결하게 됩니다.
:::

웹훅을 만드는 단계는 다음과 같습니다.

<Steps>
  1. 웹후크 페이지에서 다음을 클릭하세요. `Create new webhook` 단추.
  2. ***이름*** 웹훅에 이름을 지정합니다. *예를 들어 윙크 스토커*
  3. ***할 수 있게 하다*** 그대로 두면 활성화됩니다.
  4. **엔터티 이름** 귀하의 웹훅을 기존 계정 중 하나와 연결하세요. *예: 쿨 계정*
  5. **이벤트** 관심 있는 이벤트를 들어보세요. *예를 들어 `booking.created`*
  6. **웹훅 URL** 당사의 이벤트를 청취할 URL을 입력하세요. *예를 들어 `https://my.system.com/webhook/wink`*
  7. 클릭 `Save` 계속하려면 버튼을 클릭하세요.
</Steps>


웹훅 목록으로 다시 리디렉션되고 새로운 웹훅을 사용할 준비가 되었습니다.

## 웹훅 업데이트

다음을 원하는 경우 앱을 업데이트해야 합니다.

* 웹훅을 비활성화하세요.
* 더 많은/더 적은 이벤트를 들어보세요.
* 수신 웹훅 URL을 변경하세요.

기존 웹훅을 업데이트하려면 다음을 수행하세요.

<Steps>
  1. 클릭 `Actions` 업데이트하려는 웹훅에 대한 링크입니다.
  2. 클릭 `Update` 작업 아래의 버튼을 클릭하세요.
  3. 웹후크를 변경하세요.
  4. 클릭 `Save` 계속하려면 버튼을 클릭하세요.
</Steps>


웹후크 목록으로 다시 리디렉션됩니다.

## 웹훅 제거

더 이상 웹훅을 사용할 필요가 없다면 제거해도 됩니다.

웹훅을 제거하려면 다음을 수행하세요.

<Steps>
  1. 클릭 `Actions` 업데이트하려는 웹훅에 대한 링크입니다.
  2. 클릭 `Remove` 작업 아래의 버튼을 클릭하세요.
  3. 딸깍 하는 소리 `OK` 삭제를 확인하려면 버튼을 클릭하세요.
</Steps>


웹후크 목록으로 다시 리디렉션됩니다.
