이 페이지는 Wink에서 구독할 수 있는 모든 웹훅 이벤트를 나열합니다. 설정 방법은 [Webhook Integration Guide](/ko/guides/webhook-integration) 또는 [Webhooks 관리 페이지](/ko/my-account/webhooks)를 참조하세요.

## 구독 방법

1. 포털에서 **Applications > Webhooks**로 이동합니다.
2. 웹훅을 생성하고 수신할 이벤트를 선택합니다.
3. 이벤트가 발생할 때마다 Wink가 HTTP POST를 귀하의 엔드포인트로 전송합니다.

전체 설정 단계와 모범 사례는 [Webhook Integration](/ko/guides/webhook-integration)을 참조하세요.

## 계정 이벤트

TripPay 계정 수명 주기 및 설정과 관련된 이벤트입니다.

| 이벤트 | 키 | 설명 |
|---|---|---|
| 계정 생성됨 | `account.create` | 새 계정이 생성되었습니다. |
| 계정 검토 중 | `account.under_review` | 계정이 검토 대기 중입니다. |
| 계정 업데이트됨 | `account.updated` | 계정 정보가 변경되었습니다. |
| 계정 상태 업데이트됨 | `account.status.updated` | 계정 상태가 변경되었습니다(예: 승인, 정지). |
| 계정 주소 업데이트됨 | `account.address.updated` | 계정 주소가 수정되었습니다. |
| 계정 세부 정보 업데이트됨 | `account.details.updated` | 계정 세부 정보 필드가 변경되었습니다. |
| 계정 정보 업데이트됨 | `account.information.update` | 일반 계정 정보가 업데이트되었습니다. |
| 계정 관리자 업데이트됨 | `account.managers.update` | 계정 관리자 목록이 변경되었습니다. |
| 프로필 사진 업데이트됨 | `account.profile_picture.updated` | 계정 프로필 사진이 변경되었습니다. |
| 에이전트 업데이트됨 | `account.agent.updated` | 계정의 에이전트 정보가 수정되었습니다. |
| URL 이름 업데이트됨 | `account.url-name.update` | 계정의 맞춤 URL이 변경되었습니다. |
| 고유 ID 업데이트됨 | `account.unique-id.update` | 계정 고유 식별자가 수정되었습니다. |

### 계정 작업

| 이벤트 | 키 | 설명 |
|---|---|---|
| 작업 생성됨 | `account.task.create` | 새 작업이 계정에 추가되었습니다. |
| 작업 업데이트됨 | `account.task.update` | 계정 작업이 수정되었습니다. |
| 작업 제거됨 | `account.task.remove` | 계정 작업이 제거되었습니다. |
| 작업 완료됨 | `account.task.complete` | 계정 작업이 완료되었습니다. |
| 세금 ID 작업 완료됨 | `account.task.tax_id.complete` | 세금 ID 확인 작업이 완료되었습니다. |

### 계정 인수자

| 이벤트 | 키 | 설명 |
|---|---|---|
| 인수자 생성됨 | `account.acquirer.created` | 새 인수자가 계정에 추가되었습니다. |
| 인수자 업데이트됨 | `account.acquirer.updated` | 계정 인수자가 수정되었습니다. |
| 인수자 제거됨 | `account.acquirer.removed` | 인수자가 계정에서 제거되었습니다. |

## 예약 계약 이벤트

TripPay를 통한 예약 계약 수명 주기를 추적하는 이벤트입니다.

| 이벤트 | 키 | 설명 |
|---|---|---|
| 초기화 요청됨 | `booking.contract.initialize.request` | 예약 계약 초기화가 요청되었습니다. |
| 초기화 완료됨 | `booking.contract.initialize.complete` | 예약 계약 초기화가 완료되었습니다. |
| 계약 생성됨 | `booking.contract.created` | 새 예약 계약이 생성되었습니다. |
| 계약 처리 중 | `booking.contract.processing` | 예약 계약 결제가 처리 중입니다. |
| 계약 실패 | `booking.contract.failed` | 예약 계약 결제가 실패했습니다. |
| 계약 취소됨 | `booking.contract.cancelled` | 예약 계약이 취소되었습니다. |
| 부분 환불됨 | `booking.contract.refund.partial` | 계약에 대해 부분 환불이 이루어졌습니다. |
| 전액 환불됨 | `booking.contract.refund.full` | 계약에 대해 전액 환불이 이루어졌습니다. |
| 환불 요청됨 | `booking.contract.refund.request` | 환불 요청이 제출되었습니다. |
| 환불 오류 발생 | `booking.contract.refund.error` | 환불 처리 중 오류가 발생했습니다. |
| 지급 생성됨 | `booking.contract.payout.create` | 계약에 대한 지급이 생성되었습니다. |
| 지급 완료됨 | `booking.contract.payout.complete` | 지급이 성공적으로 완료되었습니다. |

## 관리 엔터티 이벤트

관리 엔터티(숙소, 제휴사, 회사) 및 그 관리와 관련된 이벤트입니다.

### 구독

| 이벤트 | 키 | 설명 |
|---|---|---|
| 구독 제공됨 | `managing_entity.subscription.provisioned` | 새 구독이 제공되었습니다. |
| 구독 변경됨 | `managing_entity.subscription.changed` | 구독 플랜이 변경되었습니다. |
| 구독 청구서 변경됨 | `managing_entity.subscription.invoice.changed` | 구독 청구서가 업데이트되었습니다. |

### 상태 및 관리

| 이벤트 | 키 | 설명 |
|---|---|---|
| 상태 업데이트됨 | `managing_entity.status.updated` | 엔터티 상태가 변경되었습니다. |
| 엔터티 삭제됨 | `managing_entity.deleted` | 관리 엔터티가 삭제되었습니다. |
| 에이전트 업데이트됨 | `managing_entity.agent.updated` | 엔터티 에이전트가 수정되었습니다. |
| 에이전트 이메일 알림 | `managing_entity.agent.email.notification` | 에이전트에게 이메일 알림이 전송되었습니다. |
| 관리자 이메일 알림 | `managing_entity.manager.email.notification` | 관리자에게 이메일 알림이 전송되었습니다. |

### 관리자 수명 주기

| 이벤트 | 키 | 설명 |
|---|---|---|
| 관리자 추가됨 | `managing_entity.manager.added` | 새 관리자가 추가되었습니다. |
| 관리자 초대됨 | `managing_entity.manager.invited` | 관리자 초대장이 발송되었습니다. |
| 관리자 제거됨 | `managing_entity.manager.removed` | 관리자가 제거되었습니다. |
| 관리자 초대 수락됨 | `managing_entity.manager.accepted` | 관리자 초대가 수락되었습니다. |
| 관리자 초대 거절됨 | `managing_entity.manager.rejected` | 관리자 초대가 거절되었습니다. |
| 소유자 설정됨 | `managing_entity.manager.owner.set` | 엔터티 소유권이 이전되었습니다. |

### 엔터티 정보

| 이벤트 | 키 | 설명 |
|---|---|---|
| 정보 업데이트됨 | `managing_entity.information.updated` | 엔터티 정보가 업데이트되었습니다. |
| 주소 업데이트됨 | `managing_entity.city.updated` | 엔터티 주소/도시가 변경되었습니다. |
| 계정 담당자 업데이트됨 | `managing_entity.account_representative.updated` | 계정 담당자가 변경되었습니다. |
| 프로필 사진 업데이트됨 | `managing_entity.profile_picture.updated` | 엔터티 프로필 사진이 업데이트되었습니다. |
| 로고 업데이트됨 | `managing_entity.logo.updated` | 엔터티 로고가 업데이트되었습니다. |
| URL 이름 업데이트됨 | `managing_entity.url_name.updated` | 엔터티 맞춤 URL이 변경되었습니다. |
| 고유 ID 업데이트됨 | `managing_entity.unique_id.updated` | 엔터티 고유 식별자가 변경되었습니다. |
