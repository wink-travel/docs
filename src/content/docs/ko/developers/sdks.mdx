공식 지원되는 언어 소프트웨어 개발 키트(SDK) 중 하나를 사용하여 출시 기간을 단축하세요.
현재 원하시는 언어를 지원하지 않는 경우, 사용하려는 API 페이지로 이동하여 페이지 상단에서 각 API에 대해 제공되는 OpenAPI 사양을 다운로드하세요. 공개 SDK 저장소 중 하나를 포크하거나 OpenAPI 생성기를 사용하여 직접 SDK를 생성할 수 있습니다.

## 자바

이 라이브러리는 Java 24용으로 컴파일되었습니다. 반응형 웹 클라이언트를 사용하도록 구성되었으며 Spring 프레임워크와 함께 사용하도록 최적화되어 있습니다. Spring은 필수 사항은 아니지만, 기본적으로 OAuth2를 지원합니다.
