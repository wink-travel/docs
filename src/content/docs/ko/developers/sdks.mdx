공식 지원하는 언어용 소프트웨어 개발 키트 중 하나를 사용하여 출시 시간을 단축하세요.  
현재 선호하는 언어를 지원하지 않는 경우, 사용하려는 API 페이지 상단에서 제공하는 OpenAPI 사양을 다운로드하세요. 공개 SDK 저장소를 포크하거나 OpenAPI 생성기 중 하나를 사용하여 직접 SDK를 생성할 수 있습니다.

## Java

이 라이브러리는 Java 24용으로 컴파일되었습니다. 반응형 WebClient를 사용하도록 구성되어 있으며 Spring Framework와 함께 사용하기에 최적화되어 있습니다. 다만 Spring은 필수는 아니며, Spring을 사용하면 기본적으로 OAuth2 지원을 받을 수 있습니다.
