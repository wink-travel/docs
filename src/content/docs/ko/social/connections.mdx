# 연결

귀하는 이미 링크 관리자 도구인 WinkLinks에 연결되어 있습니다. `소셜 네트워크 연결`을 클릭하여 더 많은 소셜 네트워크에 연결하세요.  
이 과정은 귀하를 소셜 네트워크로 리디렉션하여 `Wink Social` 앱이 귀하를 대신해 콘텐츠를 게시하도록 승인하도록 합니다.

연결이 완료되면 다음 예정된 달에 해당 네트워크용 콘텐츠 생성을 시작합니다.

:::note
각 소셜 네트워크에 최적화된 콘텐츠를 생성하고 게시합니다.
:::
