귀하는 이미 링크 관리자 도구인 WinkLinks에 연결되어 있습니다. `소셜 네트워크 연결`을 클릭하여 더 많은 소셜 네트워크에 연결하세요.
이 과정은 귀하를 소셜 네트워크로 리디렉션하여 `Wink Social` 앱이 귀하를 대신해 콘텐츠를 게시하도록 승인하도록 합니다.

연결이 완료되면 다음 예약된 달부터 해당 네트워크용 콘텐츠 생성을 시작합니다.

:::note
무료 요금제에는 WinkLinks가 포함되어 있습니다. Facebook, Instagram 및 기타 플랫폼은 유료 요금제에서 이용 가능합니다.
:::

## 지원 플랫폼

| 플랫폼 | 설명 | 요구 사항 |
| -- | -- | -- |
| **WinkLinks** | 고급 링크 관리자에 자동 게시 | 모든 요금제에 포함 |
| **Facebook** | 게시물 자동화, 댓글 및 DM에 자동 응답 | Facebook 비즈니스 페이지 필요 |
| **Instagram** | 게시물 자동화, 댓글 및 DM에 자동 응답 | Facebook 계정과 연결된 Instagram 비즈니스 계정 필요 |

각 연결은 현재 상태를 표시합니다: **활성**, **연결 끊김**, 또는 **중단됨**.

:::note
각 소셜 네트워크에 최적화된 콘텐츠를 생성하고 게시합니다.
:::
