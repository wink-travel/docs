# 설정

Social을 처음 시작할 때, 콘텐츠를 어떻게 생성할지 선택하도록 요청합니다. 이 옵션들은 언제든지 변경할 수 있어 나중에 쉽게 조정할 수 있습니다.

Social 설정의 첫 번째 탭에는 기본 설정이 포함되어 있습니다. 다음과 같습니다:

1. **선호하는 맞춤 설정** 콘텐츠 생성 시 어떤 맞춤 설정을 사용할지 선택합니다. 맞춤 설정은 [my.wink.travel](https://my.wink.travel) 계정에서 생성할 수 있습니다. 맞춤 설정은 테마 색상, 기본 일정 등 다양한 요소를 제어하며, 예약 엔진과 연동됩니다.
2. **자동 승인** 매달 새 콘텐츠를 승인 없이 자동으로 게시하도록 신뢰하시나요? 이 스위치를 활성화하면 모든 작업을 대신 처리해 드립니다.
3. **승인자** 하지만... 새로 생성된 게시물에 대해 알림을 받고 승인하고 싶다면, 담당자의 연락처 정보를 입력하세요.

## 콘텐츠 제작자

<CldImage

Social에는 다양한 콘텐츠 제작자가 있습니다.  
콘텐츠 제작자는 숙소에 맞는 독특한 콘텐츠를 만드는 특정 방식을 담당합니다. 각 제작자는 `weight` 속성을 가지며, 이는 무작위 선택될 확률을 나타냅니다.

### 예시:

호텔이 Add-On과 Property 제작자를 선택했습니다. Add-On 제작자는 5점, Property 제작자는 10점의 가중치를 가집니다. 따라서 Property 콘텐츠가 생성될 확률이 Add-On보다 훨씬 높습니다.

생성하는 텍스트는 브랜드에 더 잘 맞는 다양한 `음성` / `페르소나`를 사용할 수 있습니다. 적합한 페르소나를 선택하세요. 콘텐츠 제작자와 마찬가지로, **Weight**를 높여 페르소나가 사용될 확률을 설정할 수 있습니다.

*대부분 숙소는 단일 언어를 사용하며 현지 언어를 기본값으로 설정합니다*
