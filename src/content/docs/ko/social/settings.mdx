# 설정

Social을 처음 시작할 때, 콘텐츠를 생성하는 방식을 선택하도록 요청합니다. 이 옵션들은 언제든지 변경할 수 있어 편리합니다.

사이드바에서 `Preferences`로 이동하여 설정을 관리하세요.

환경설정 섹션에는 기본 콘텐츠 생성 설정이 포함되어 있습니다:

1. **선호하는 맞춤 설정** 콘텐츠 생성 시 사용할 맞춤 설정을 선택하세요. 맞춤 설정은 테마 색상, 기본 일정, 통화, 언어 등을 제어합니다. 맞춤 설정은 [my.wink.travel](https://my.wink.travel)에서 계정 내에서 생성할 수 있습니다.
2. **자동 승인** 활성화하면 생성된 게시물이 승인 없이 자동으로 게시됩니다. 콘텐츠 품질에 자신이 생기면 이 기능을 켜세요.
3. **승인자 이름** 승인 요청 이메일을 받을 사람 또는 팀의 이름입니다.
4. **승인자 이메일** 다가오는 게시물에 대한 승인 요청이 전송될 이메일 주소입니다.

## 콘텐츠 제작자

<CldImage

Social에는 다양한 콘텐츠 제작자가 있습니다. 각 제작자는 AI를 활용해 숙소에 맞는 독특한 콘텐츠를 만드는 특정 방식을 전문으로 합니다. 모든 콘텐츠 제작자는 **가중치(Weight)** 속성을 가지며, 가중치가 높을수록 게시물 생성 시 해당 제작자가 선택될 확률이 높아집니다.

사용 가능한 모든 콘텐츠 제작자는 다음과 같습니다:

| 제작자 | 설명 |
| -- | -- |
| **Destination** | 위치 정보를 활용해 해당 지역에 관한 유용한 텍스트를 생성합니다 |
| **Room type** | AI가 생성한 텍스트와 객실 유형 이미지를 사용해 예약 가능한 게시물을 만듭니다 |
| **Activity** | 숙소의 액티비티를 소개하는 예약 가능한 게시물을 만듭니다 |
| **Add-On** | 숙소의 추가 서비스(애드온)를 소개하는 예약 가능한 게시물을 만듭니다 |
| **Attraction** | 숙소 주변 명소를 소개하는 예약 가능한 게시물을 만듭니다 |
| **Place** | 숙소 주변 장소를 소개하는 예약 가능한 게시물을 만듭니다 |
| **Meeting room** | 회의실을 소개하는 예약 가능한 게시물을 만듭니다 |
| **Restaurant** | 레스토랑을 소개하는 예약 가능한 게시물을 만듭니다 |
| **Spa** | 스파를 소개하는 예약 가능한 게시물을 만듭니다 |
| **Property** | AI가 생성한 텍스트와 숙소 이미지를 사용해 최적가 객실을 소개하는 예약 가능한 게시물을 만듭니다 |
| **User sentiment** | 웹 전반의 고객 리뷰를 집계해 숙소에 대한 긍정적인 감정을 강조하는 게시물을 만듭니다 |
| **Funny** | 가볍고 재미있는 게시물을 만들어 피드를 활기차게 유지합니다 |

:::tip[전문가 팁]
선택한 콘텐츠 제작자에 필요한 재고 유형이 있는지 확인하세요. 예를 들어 "Restaurant"를 선택했다면 숙소에 최소 한 개 이상의 레스토랑이 등록되어 있어야 합니다.
:::

어조는 AI가 콘텐츠를 생성할 때 사용하는 언어 스타일을 설정합니다. 브랜드 성격에 맞는 어조를 선택하세요. 콘텐츠 제작자와 마찬가지로 각 어조는 사용 빈도를 조절하는 **가중치**가 있습니다.

사용 가능한 어조는 다음과 같습니다:

| 어조 | 스타일 |
| -- | -- |
| **Neutral** | 차분하고 권위 있으며 사실적입니다. 과장이나 농담 없이 신뢰할 수 있고 잘 구조화된 지식 전달 |
| **Friendly** | 따뜻하고 친근하며 텍스트로 미소 짓는 듯한 에너지, 일상적인 축약형과 가끔 이모지 포함 |
| **Luxury** | 세련되고 정제된 스타일. 독자에게 특별함을 느끼게 하는 "exquisite", "bespoke" 같은 단어 사용 |
| **Foodie** | 감각적이고 군침 도는 묘사. 독자를 배고프게 만드는 음식 관련 언어 |
| **Adventurous** | 대담하고 용감하며 탐험가 같은 에너지. 생생한 감각 묘사로 모든 것을 장대하게 표현 |
| **Family** | 건강하고 아이 친화적인 흥분. 안전하고 즐거운 경험에 대한 따뜻한 부모 같은 에너지 |
| **Romantic** | 시적이고 애틋하며 가슴 설레는 느낌. 일몰, 부드러운 언어, 사랑을 표현 |
| **Humorous** | 빈정대는, 빠른 한 줄 개그, 영리한 말장난. 스탠드업 코미디처럼 읽힘 |
| **Eco** | 진지하고 올바른 태도. 친환경 대안과 지속 가능한 선택을 기념 |
| **Local insider** | 매우 구체적인 지역 참조, 관광객 함정 경고, 지역 속어 사용 |
| **Inclusive** | 환영하고 축하하며 긍정적. 모두가 인정받고 소중히 여겨지는 느낌을 줌 |
| **Mystical** | 천상적이고 영적이며 우주적 진리. 내면을 들여다보도록 부드럽게 초대 |
| **Futuristic** | 세련되고 미래지향적. "paradigm shift"와 낙관적인 미래 비전 같은 용어 선호 |
| **Budget** | 검소하고 자원 활용에 능하며 자랑스럽게 절약하는 스타일. 저렴한 발견과 무료 경험을 기념 |

## 게시물 유형

게시물 유형은 생성된 콘텐츠의 목표와 구조를 결정합니다. 각 게시물 유형은 사용 빈도를 조절하는 **가중치**가 있습니다.

| 게시물 유형 | 설명 |
| -- | -- |
| **Transactional** | 명확하고 간결하며 전환 중심. 사실, 가격, 다음 단계로 예약과 직접 행동 유도 |
| **Awareness** | 브랜드 구축 콘텐츠로 숙소를 소개하고 잠재 고객 사이에서 인지도를 높임 |
| **Sentiment** | 감정적으로 공감하는 콘텐츠로 긍정적 연상과 청중과의 감정적 연결 구축 |

특정 언어와 지역에 마케팅 노력을 집중하고 싶을 수 있습니다. 콘텐츠를 생성할 언어와 관심 있는 시간대를 알려주시면 나머지는 저희가 처리합니다. 게시물은 선택한 시간대의 예약된 시간에 게시됩니다.

*대부분 숙소는 단일 언어를 사용하며 현지 언어를 기본값으로 설정합니다.*

Google 비즈니스 프로필을 연결하면 더 풍부한 숙소 정보를 가져와 더 효과적인 콘텐츠를 생성할 수 있습니다. 숙소에 인증된 Google 비즈니스 프로필이 있어야 목록에서 콘텐츠를 생성할 수 있습니다. 일부 콘텐츠 제작자는 Google 비즈니스 계정이 **필수**입니다.

:::
