**Wink Drupal मॉड्यूल** आपके Drupal साइट में Wink ट्रैवल इन्वेंटरी कंपोनेंट्स को नेटिव ब्लॉक्स के रूप में जोड़ता है। संपादक इन्हें स्टैंडर्ड ब्लॉक सिस्टम के माध्यम से लेआउट में कहीं भी रख सकते हैं।

## आवश्यकताएँ

- Drupal 10 या 11
- PHP 8.1+
- Composer

## इंस्टॉल करें

<Steps>
1. Composer के माध्यम से इंस्टॉल करें:
   ```bash
   composer require drupal/wink
   ```
2. मॉड्यूल सक्षम करें:
   ```bash
   drush en wink -y
   ```
3. **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`) पर जाएं।
4. अपना **Client ID** और **Client Secret** दर्ज करें, अपना पर्यावरण चुनें, और **Save** पर क्लिक करें।
   अपनी क्रेडेंशियल्स प्राप्त करने के लिए [Applications](/hi/portal/settings/applications) पर जाएं।
</Steps>

## पेज पर ब्लॉक जोड़ें

<Steps>
1. **Admin → Structure → Block Layout** पर जाएं।
2. उस क्षेत्र में **Place block** पर क्लिक करें जहाँ आप Wink कंटेंट चाहते हैं।
3. उपलब्ध ब्लॉक्स खोजने के लिए **Wink** खोजें:
   - **Wink Content** — होटल कार्ड, ग्रिड, या मैप रेंडर करता है
   - **Wink Lookup** — डेस्टिनेशन सर्च बार
   - **Wink Search** — यात्रा चयनकर्ता बटन
   - **Wink Account** — साइन-इन बटन
   - **Wink Itinerary** — यात्रा स्थिति बटन
   - **Wink Cart** — शॉपिंग कार्ट बटन
4. **Wink Content** के लिए, ब्लॉक सेटिंग्स में **Layout Type** और **Layout ID** कॉन्फ़िगर करें।
5. ब्लॉक कॉन्फ़िगरेशन सहेजें।
</Steps>

## उपलब्ध ब्लॉक्स

| ब्लॉक | उद्देश्य |
|---|---|
| **Wink Content** | होटल कार्ड, रैंक्ड ग्रिड, मैप्स, और अधिक दिखाता है |
| **Wink Lookup** | डेस्टिनेशन और होटल के लिए सर्च बार |
| **Wink Search** | आइकन बटन जो यात्रा चयनकर्ता खोलता है |
| **Wink Account** | साइन-इन / उपयोगकर्ता खाता बटन |
| **Wink Itinerary** | वर्तमान यात्रा दिखाने वाला यात्रा बटन |
| **Wink Cart** | शॉपिंग कार्ट सारांश बटन |
