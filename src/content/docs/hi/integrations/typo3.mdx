**Wink TYPO3 एक्सटेंशन** Wink ट्रैवल इन्वेंट्री को TYPO3 13 में नेटिव कंटेंट एलिमेंट्स के रूप में इंटीग्रेट करता है। संपादक मानक TYPO3 पेज एडिटर का उपयोग करके पेजों में Wink कंपोनेंट्स जोड़ते हैं — किसी टेम्प्लेट परिवर्तन की आवश्यकता नहीं।

## आवश्यकताएँ

- TYPO3 13.x
- PHP 8.2+
- Composer

## इंस्टॉल करें

<Steps>
1. Composer के माध्यम से इंस्टॉल करें:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. एक्सटेंशन सक्रिय करें:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. TYPO3 बैकएंड में जाएं, **Admin Tools → Settings → Extension Configuration → wink** खोलें।
4. अपना **Client ID**, **Client Secret** दर्ज करें और अपना पर्यावरण चुनें। सेव करें।
   अपनी क्रेडेंशियल्स प्राप्त करने के लिए [Applications](/hi/portal/settings/applications) पर जाएं।
5. अपने रूट टेम्प्लेट में स्टैटिक TypoScript टेम्प्लेट जोड़ें:
   **Template → Edit → Includes → Available Items → Wink Travel**।
</Steps>

## पेज में कंटेंट एलिमेंट जोड़ें

<Steps>
1. **Page** मॉड्यूल में एक पेज खोलें।
2. कंटेंट क्षेत्र में नया एलिमेंट जोड़ने के लिए **+** बटन पर क्लिक करें।
3. **Wink Travel** कंटेंट एलिमेंट्स में से एक चुनें:
   - **Wink Content** — होटल कार्ड, रैंक्ड ग्रिड, मैप्स
   - **Wink Lookup** — डेस्टिनेशन सर्च बार
   - **Wink Search** — यात्रा चयनकर्ता बटन
   - **Wink Account** — साइन-इन बटन
   - **Wink Itinerary** — यात्रा स्थिति बटन
   - **Wink Cart** — शॉपिंग कार्ट बटन
4. **Wink Content** के लिए, FlexForm फील्ड्स का उपयोग करके **Layout Type** चुनें और **Layout ID** दर्ज करें।
5. पेज को सेव और प्रकाशित करें।
</Steps>

## उपलब्ध कंटेंट एलिमेंट्स

| एलिमेंट | उद्देश्य |
|---|---|
| **Wink Content** | होटल कार्ड, रैंक्ड ग्रिड, मैप्स, और अधिक |
| **Wink Lookup** | डेस्टिनेशन सर्च बार |
| **Wink Search** | आइकन बटन जो यात्रा चयनकर्ता खोलता है |
| **Wink Account** | साइन-इन / उपयोगकर्ता खाता बटन |
| **Wink Itinerary** | वर्तमान यात्रा दिखाने वाला यात्रा बटन |
| **Wink Cart** | शॉपिंग कार्ट सारांश बटन |
