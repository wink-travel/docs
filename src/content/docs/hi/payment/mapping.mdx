यदि आप एक इंटीग्रेटर हैं, जैसे Wink, और TripPay के साथ उपयोग के लिए अपने बाहरी पेयर्स को मैप करना चाहते हैं, तो आप इसे दो तरीकों में से एक से कर सकते हैं:

- मैनुअल मैपिंग
- [Payment API](/hi/developers/apis/#payment-api) का उपयोग करना (*देखें `Account Mappings`*)

:::tip
जब तक आप केवल कुछ ही समान प्रॉपर्टीज़ से निपट रहे हों, हम सुझाव देते हैं कि आप इस चरण को स्वचालित करने के लिए API का उपयोग करें।
:::

नीचे, हम आपको बाहरी पेयर को मैनुअल रूप से मैप करने का तरीका बताते हैं। *यह अंतर्निहित रूप से API का उपयोग करता है।*

<Steps>
1. [TripPay](https://pay.wink.travel) में लॉग इन करें।
2. अपना इंटीग्रेटर खाता चुनें।
3. मुख्य नेव बार से `Settings > Mappings` पर क्लिक करें।
4. `Create mapping` बटन पर क्लिक करें। आपको मैपिंग फॉर्म दिखाई देगा।
5. **Local account** उस मौजूदा TripPay का नाम टाइप करना शुरू करें जिसे आप मैप करना चाहते हैं।
6. **External identifier** इस इकाई के लिए अपना बाहरी ID पेस्ट करें जैसा कि वह रिमोट सिस्टम में मौजूद है।
7. जारी रखने के लिए `Save` पर क्लिक करें। आपको आपके मैपिंग की सूची पर वापस भेजा जाएगा।
</Steps>

अपने मैपिंग्स के साथ, TripPay अब जानता है कि जब आप अपने बाहरी पहचानकर्ताओं का उपयोग करके उन्हें संदर्भित करते हैं तो बुकिंग के लाभार्थियों के रूप में किन खातों को क्रेडिट करना है।

## API

डेवलपर्स जो `Payments` प्रबंधित करना चाहते हैं, वे [Developers > API > Payments](/hi/developers/apis/#payment-api) पर जाएं।
