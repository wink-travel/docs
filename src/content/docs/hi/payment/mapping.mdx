यदि आप विंक की तरह एक इंटीग्रेटर हैं, और ट्रिपपे के साथ उपयोग के लिए अपने बाहरी भुगतानकर्ताओं को मैप करना चाहते हैं, तो आप इसे दो तरीकों में से एक से कर सकते हैं:

* मैनुअल मैपिंग
* का उपयोग [भुगतान एपीआई](/developers/apis/#payment-api)(*देखना `Account Mappings`*)

:::टिप
जब तक कि आप केवल कुछ ही समान प्रॉपर्टीज़ के साथ काम नहीं कर रहे हों, हम अनुशंसा करते हैं कि आप इस चरण को स्वचालित करने के लिए API का उपयोग करें।
:::

नीचे, हम आपको बाहरी भुगतानकर्ता को मैप करने के मैनुअल तरीके के बारे में बता रहे हैं।*यह API का उपयोग करता है।*

<Steps>
  1. में प्रवेश करें [ट्रिपपे](https://pay.wink.travel).
  2. अपना इंटीग्रेटर खाता चुनें.
  3. क्लिक `Settings > Mappings` मुख्य नेविगेशन बार से.
  4. क्लिक करें `Create mapping` बटन पर क्लिक करें। आपको मैपिंग फॉर्म दिखाई देगा।
  5. **स्थानीय खाता** उस मौजूदा ट्रिपपे का नाम लिखना प्रारंभ करें जिसे आप मैप करना चाहते हैं।
  6. **बाह्य पहचानकर्ता** इस इकाई के लिए अपनी बाह्य आईडी चिपकाएँ क्योंकि यह दूरस्थ सिस्टम में मौजूद है।
  7. क्लिक `Save` जारी रखने के लिए। आपको मैपिंग की अपनी सूची पर वापस भेज दिया जाता है।
</Steps>


आपकी मैपिंग के साथ, अब ट्रिपपे को पता है कि जब आप अपने बाहरी पहचानकर्ताओं का उपयोग करके किसी बुकिंग का उल्लेख करते हैं तो किस खाते को लाभार्थी के रूप में क्रेडिट करना है।

## एपीआई

डेवलपर्स जो प्रबंधन करना चाहते हैं`Payments`, जाओ [डेवलपर्स > API > भुगतान](/developers/apis/#payment-api).
