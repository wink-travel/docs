यदि आप एक यात्रा कंपनी हैं और अपने व्यवसाय के भुगतान और वितरण भाग को TripPay को आउटसोर्स करना चाहते हैं, तो शुरू करने के लिए यहां 6 मूलभूत कदम दिए गए हैं:

<Steps>
1. Wink पर एक [उपयोगकर्ता खाता](/hi/user/register) बनाएं।
2. [TripPay](https://pay.wink.travel) के साथ एक खाता बनाएं।
3. TripPay पर एक [एप्लिकेशन](/hi/developers/applications) बनाएं जो आपको हमारे API से बात करने देता है।
4. अपने लाभार्थियों को TripPay के साथ बनाएं और मैप करें। *जैसे, अधिकांश के लिए, इसका मतलब है उन संपत्तियों के लिए खाते बनाना जिन्हें आप चाहते हैं कि TripPay भुगतान करे।*
5. अपने साइट में [भुगतान वेब घटक](/hi/developers/web-components/#payment) को एकीकृत करें।
6. यात्री की `शॉपिंग कार्ट` TripPay को भेजें और भुगतान प्रक्रिया शुरू करें।
</Steps>

:::tip
अपने यात्रा साइट के लिए TripPay का उपयोग करने के लिए एक पूर्ण मार्गदर्शिका के लिए, गाइड [TripPay के साथ एकीकरण](/hi/guides/integrating-with-trip-pay) पढ़ें।
:::

## API

डेवलपर्स जो `Payments` प्रबंधित करना चाहते हैं, वे [Developers > API > Payments](/hi/developers/apis/#payment-api) पर जाएं।
