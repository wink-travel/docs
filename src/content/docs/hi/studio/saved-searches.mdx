सहेजे गए खोज नाम के अनुसार काम करते हैं। यात्रा इन्वेंटरी खोजते समय, आपके पास अपनी खोज क्वेरी को बाद में उपयोग के लिए सहेजने का विकल्प होता है।

## अपनी खोज क्वेरी सहेजना

जब आप इन्वेंटरी खोजना शुरू करते हैं (देखें [Search](/hi/studio/search) ), तो आप अपनी खोज फ़िल्टर को ठीक से समायोजित कर सकते हैं।  
क्वेरी फॉर्म के नीचे `Save search` का विकल्प होता है।

इसे एक नाम दें और `Save search` बटन पर क्लिक करें। इसके बाद, आप मुख्य नेविगेशन बार से `Inventory > Saves Searches` पर जाकर अपनी नई बनाई गई रिकॉर्ड देख सकते हैं।

## सूची क्रियाएँ

आपकी सहेजी गई खोज का उपयोग अपने उपयोगकर्ताओं को ग्रिड या मानचित्र के रूप में यात्रा इन्वेंटरी दिखाने के लिए किया जा सकता है और अपनी खोज परिणामों को अपनी पसंद के अनुसार प्रदर्शित कर सकते हैं।

हर सहेजी गई खोज क्वेरी पर आप निम्नलिखित क्रियाएँ कर सकते हैं:

- खोज क्वेरी को [map](/hi/studio/maps) में बदलें।
- खोज क्वेरी को [grid](/hi/studio/grids) में बदलें।
- सहेजी गई खोज को हटाएं।

## API

डेवलपर्स जो सहेजी गई खोजों का प्रबंधन करना चाहते हैं, वे [Developers > API > Browse](/hi/developers/apis/#browse-api) पर जा सकते हैं।
