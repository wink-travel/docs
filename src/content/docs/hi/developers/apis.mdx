हमारे सभी REST API भाषा-स्वतंत्र और Open API अनुरूप हैं। बिना एक भी लाइन कोड लिखे परीक्षण शुरू करें।

यदि आपने अभी तक अपना Wink खाता नहीं बनाया है, तो [यहाँ से शुरू करें](/hi/user/register)।

## उपभोग एंडपॉइंट

उपभोग एंडपॉइंट उन डेवलपर्स के लिए हैं जो मौजूदा यात्रा इन्वेंटरी खोजकर उसे बुक करना चाहते हैं या अपने Wink संबद्ध खातों के माध्यम से विज्ञापन के लिए उपयोग करना चाहते हैं।

### कॉन्फ़िगरेशन API

बुकिंग इंजन के लिए व्हाइटलेबल + कस्टमाइज़ेशन जानकारी प्राप्त करने के लिए एकल एंडपॉइंट।

<LinkButton href="https://developer.wink.travel/engine-client" variant="secondary" icon="external" target="_blank">
अधिक जानें
</LinkButton>

### लुकअप API

भूगोलिक क्षेत्रों में विशेष यात्रा इन्वेंटरी खोजने का प्रोग्रामर-अनुकूल तरीका।

<LinkButton href="https://developer.wink.travel/lookup" variant="secondary" icon="external" target="_blank">
अधिक जानें
</LinkButton>

### इन्वेंटरी API

लुकअप API का उपयोग करके आधिकारिक रूप से स्थित ज्ञात यात्रा इन्वेंटरी प्राप्त करने का प्रोग्रामर-अनुकूल तरीका।

<LinkButton href="https://developer.wink.travel/inventory" variant="secondary" icon="external" target="_blank">
अधिक जानें
</LinkButton>

### बुकिंग API

हमारे प्लेटफ़ॉर्म पर पाए गए इन्वेंटरी को बुक करने का प्रोग्रामर-अनुकूल तरीका।

<LinkButton href="https://developer.wink.travel/booking" variant="secondary" icon="external" target="_blank">

## उत्पादन एंडपॉइंट

## एनालिटिक्स API

एनालिटिक्स API उन सभी के लिए एंडपॉइंट प्रदान करता है जो ट्रैक लीडरबोर्ड रैंकिंग के साथ-साथ समय श्रृंखला, प्लेटफ़ॉर्म-स्तर, डेटा में गहरी अंतर्दृष्टि चाहते हैं; जो विभिन्न प्रकार के मेट्रिक्स होस्ट करता है।

## भुगतान API

भुगतान एंडपॉइंट उन डेवलपर्स के लिए हैं जो यात्रा इन्वेंटरी खरीदना चाहते हैं। यह API के माध्यम से एक पंजीकृत ट्रैवल एजेंट के रूप में किया जा सकता है या हमारे PCI अनुरूप भुगतान विजेट के साथ अन्य सभी इकाइयों के लिए हमारे API का उपयोग करके किया जा सकता है।
भुगतान API संबद्धों और होटलों को बुकिंग, एनालिटिक्स और फंड उपलब्धता ट्रैक करने के लिए एंडपॉइंट प्रदान करता है। वे उपलब्ध फंड को अपने बैंक खाते में निकालने का विकल्प भी चुन सकते हैं।

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
अधिक जानें
</LinkButton>
