## एंडपॉइंट का उपभोग करें

कंज्यूम एंडपॉइंट उन डेवलपर्स के लिए हैं जो मौजूदा यात्रा इन्वेंट्री ढूंढना चाहते हैं और या तो इसे बुक करना चाहते हैं या अपने विंक सहबद्ध खातों में से किसी एक के माध्यम से विज्ञापन देने के लिए इसका उपयोग करना चाहते हैं।

## समापन बिंदु बनाएं

## भुगतान एपीआई

भुगतान समापन बिंदु उन डेवलपर्स के लिए हैं जो ट्रैवल इन्वेंट्री खरीदना चाहते हैं। यह पंजीकृत ट्रैवल एजेंट के रूप में API के माध्यम से या अन्य सभी संस्थाओं के लिए हमारे PCI अनुपालन भुगतान विजेट के साथ हमारे API का उपयोग करके किया जा सकता है।
भुगतान API सहबद्धों और होटलों के लिए बुकिंग, विश्लेषण और धन की उपलब्धता को ट्रैक करने के लिए समापन बिंदु प्रदर्शित करता है। वे अपने बैंक खाते में उपलब्ध धन को निकालने का विकल्प भी चुन सकते हैं।
