हमारे आधिकारिक रूप से समर्थित भाषा सॉफ़्टवेयर डेवलपमेंट किट में से किसी एक का उपयोग करके बाज़ार में आने के समय को तेज़ करें।
यदि हम वर्तमान में आपकी पसंदीदा भाषा का समर्थन नहीं करते हैं, तो उस API पर जाएँ जिसका आप उपयोग करना चाहते हैं और OpenAPI विनिर्देश डाउनलोड करें जिसे हम पृष्ठ के शीर्ष पर प्रत्येक API के लिए उपलब्ध कराते हैं। आप हमारे किसी सार्वजनिक SDK रिपॉजिटरी को फ़ॉर्क करके या OpenAPI के किसी जनरेटर का उपयोग करके अपना स्वयं का SDK जेनरेट कर सकते हैं।

## जावा

यह लाइब्रेरी जावा 24 के लिए संकलित की गई थी। इसे रिएक्टिव वेबक्लाइंट का उपयोग करने के लिए कॉन्फ़िगर किया गया है और इसे स्प्रिंग फ्रेमवर्क के साथ उपयोग के लिए अनुकूलित किया गया है। हालाँकि, स्प्रिंग एक आवश्यकता नहीं है, लेकिन यह आपको OAuth2 समर्थन देता है।
