Wink प्रमुख चैनल मैनेजरों, प्रॉपर्टी मैनेजमेंट सिस्टम्स (PMS), और सेंट्रल रिजर्वेशन सिस्टम्स (CRS) के साथ इंटीग्रेट होता है ताकि आपकी दरें और उपलब्धता आपके सभी बिक्री चैनलों में सिंक्रनाइज़्ड रहें।

## यह कैसे काम करता है

चैनल मैनेजर इंटीग्रेशन आपके प्रॉपर्टी मैनेजमेंट सिस्टम और Wink के बीच डेटा का दो-तरफा प्रवाह सक्षम करता है:

1. **दर पुश** — आपका चैनल मैनेजर दर अपडेट्स Wink को भेजता है। जब आप अपने PMS में अपनी कीमतें अपडेट करते हैं, तो ये बदलाव स्वचालित रूप से Wink पर प्रतिबिंबित होते हैं।
2. **उपलब्धता सिंक** — आपका चैनल मैनेजर उपलब्धता अपडेट्स Wink को भेजता है। जब कमरे बुक या रिलीज़ होते हैं, तो Wink की इन्वेंटरी रियल टाइम में अपडेट होती है।
3. **बुकिंग सूचनाएं** — जब Wink पर कोई बुकिंग होती है, तो आपका चैनल मैनेजर सूचित किया जाता है ताकि आरक्षण आपके PMS में दिखाई दे।

## समर्थित चैनल मैनेजर

Wink वर्तमान में निम्नलिखित सिस्टम्स के साथ इंटीग्रेट होता है:

| चैनल मैनेजर | विवरण |
|---|---|
| **Channex** | वैश्विक वितरण और चैनल प्रबंधन प्लेटफ़ॉर्म। |
| **TravelClick** | होटल समाधान और चैनल प्रबंधन (Amadeus)। |
| **Omnibees** | होटलों के लिए वितरण और राजस्व प्रबंधन। |
| **D-Edge** | हॉस्पिटैलिटी तकनीक और सेंट्रल रिजर्वेशन समाधान। |
| **Allotz** | इन्वेंटरी आवंटन और चैनल प्रबंधन। |
| **Ezee / Yanolja** | एशियाई बाजार पर केंद्रित क्लाउड PMS और चैनल प्रबंधन। |
| **RateGain** | दर बुद्धिमत्ता और वितरण तकनीक। |
| **RateTiger** | रियल-टाइम दर और उपलब्धता वितरण। |
| **CloudBeds** | क्लाउड-आधारित PMS, चैनल मैनेजर, और बुकिंग इंजन। |
| **SiteMinder** | प्रमुख होटल चैनल मैनेजर और वितरण प्लेटफ़ॉर्म। |
| **Exely** | होटलों के लिए दर और उपलब्धता सिंक्रनाइज़ेशन। |

:::note
यदि आपका चैनल मैनेजर सूचीबद्ध नहीं है, तो Wink टीम से संपर्क करें। Wink द्वारा समर्थित OTA (Open Travel Alliance) प्रोटोकॉल मानकों के आधार पर नए इंटीग्रेशन जोड़े जा सकते हैं।
:::

## डेवलपर्स के लिए

यदि आप Wink के साथ चैनल मैनेजर या PMS इंटीग्रेशन बना रहे हैं, तो Channel Manager API endpoints का उपयोग करें:

- **दर अपडेट्स** — Wink को दर परिवर्तनों को पुश करें।
- **उपलब्धता अपडेट्स** — Wink को उपलब्धता परिवर्तनों को पुश करें।
- **बुकिंग पुनः प्राप्ति** — Wink से अपनी प्रणाली में बुकिंग्स खींचें।

Channel Manager API OTA प्रोटोकॉल मानकों (SOAP/XML) का पालन करता है ताकि मौजूदा हॉस्पिटैलिटी सिस्टम्स के साथ संगतता बनी रहे।

## वेबहुक सूचनाएं

आप चैनल मैनेजर वेबहुक इवेंट्स की सदस्यता लेकर रियल-टाइम सूचनाएं प्राप्त कर सकते हैं:

- `channel-manager.update.rate` — दर अपडेट प्राप्त हुआ।
- `channel-manager.update.availability` — उपलब्धता अपडेट प्राप्त हुआ।
- `channel-manager.update` — सामान्य चैनल मैनेजर अपडेट।

विवरण के लिए [Webhook Events Catalog](/hi/developers/webhook-events#channel-manager-events) देखें।
