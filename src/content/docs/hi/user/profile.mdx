import { Steps } from '@astrojs/starlight/components';

यदि आप अपने Wink उपयोगकर्ता खाते के लिए अपनी उपयोगकर्ता प्रोफ़ाइल अपडेट करना चाहते हैं, तो इन चरणों का पालन करें:

<Steps>
1. आपको हमारी किसी एक वेबसाइट में लॉग इन होना चाहिए।
2. ऊपर-दाएँ कोने में अपने प्रोफ़ाइल आइकन पर क्लिक करें, एक मेनू दिखाई देगा।
3. `Profile` लिंक पर क्लिक करें।
4. अपने `e-mail`, `first` और `last name` को अपडेट करने के निर्देशों का पालन करें।
5. आप अपने नए `profile picture` भी अपलोड कर सकते हैं।
6. `Save` पर क्लिक करें।
</Steps>

:::note
आपकी नई प्रोफ़ाइल तस्वीर अगली बार जब आप Wink से पूरी तरह लॉग आउट करेंगे, तब प्रदर्शित होगी।
:::
