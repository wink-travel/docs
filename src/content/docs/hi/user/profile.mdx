import { Steps } from '@astrojs/starlight/components';

यदि आप अपने विंक उपयोगकर्ता खाते के लिए अपना उपयोगकर्ता प्रोफ़ाइल अपडेट करना चाहते हैं, तो इन चरणों का पालन करें:

<Steps>
  1. आपको हमारी किसी एक वेबसाइट पर लॉग इन होना होगा।
  2. ऊपरी दाएं कोने में अपने प्रोफ़ाइल आइकन पर क्लिक करें और एक मेनू दिखाई देगा।
  3. क्लिक करें `Profile` जोड़ना।
  4. अपना अपडेट करने के लिए निर्देशों का पालन करें `e-mail`, `first` और `last name`.
  5. आप एक नया भी अपलोड कर सकते हैं `profile picture` तुम्हारा खुद का।
  6. क्लिक `Save`.
</Steps>


:::नोट
अगली बार जब आप विंक से पूरी तरह लॉग आउट करेंगे तो आपकी नई प्रोफ़ाइल तस्वीर प्रदर्शित होगी।
:::
