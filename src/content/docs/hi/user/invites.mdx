विंक उपयोगकर्ता आपको अपनी संपत्ति, सहबद्ध या ट्रैवल एजेंट खातों में से किसी एक को प्रबंधित करने की अनुमति दे सकता है। जब ऐसा होता है, तो आपको एक आमंत्रण अनुरोध के साथ एक ई-मेल प्राप्त होगा।

import { Steps } from '@astrojs/starlight/components';

अपने वर्तमान विंक आमंत्रण देखने के लिए, इन चरणों का पालन करें:

<Steps>
  1. आपको हमारी किसी एक वेबसाइट पर लॉग इन होना होगा।
  2. ऊपरी दाएं कोने में अपने प्रोफ़ाइल आइकन पर क्लिक करें और एक मेनू दिखाई देगा।
  3. क्लिक करें `Invites` जोड़ना।
  4. अपने आमंत्रणों के साथ बातचीत करें `Accepting` या `Rejecting` उन्हें।
</Steps>
