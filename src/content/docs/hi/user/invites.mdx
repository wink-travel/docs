एक Wink उपयोगकर्ता आपको उनके किसी संपत्ति, संबद्ध या यात्रा एजेंट खाते का प्रबंधन करने का विकल्प दे सकता है। जब ऐसा होता है, तो आपको एक निमंत्रण अनुरोध के साथ ई-मेल प्राप्त होगा।

import { Steps } from '@astrojs/starlight/components';

अपने वर्तमान Wink निमंत्रण देखने के लिए, इन चरणों का पालन करें:

<Steps>
1. आपको हमारी किसी वेबसाइट में लॉग इन होना चाहिए।
2. ऊपर-दाएँ कोने में अपने प्रोफ़ाइल आइकन पर क्लिक करें और एक मेनू दिखाई देगा।
3. `Invites` लिंक पर क्लिक करें।
4. अपने निमंत्रणों के साथ `स्वीकार` या `अस्वीकार` करके इंटरैक्ट करें।
</Steps>
