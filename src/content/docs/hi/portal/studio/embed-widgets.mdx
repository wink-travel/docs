Studio कई एम्बेड करने योग्य विजेट प्रदान करता है जिन्हें आप अपनी वेबसाइट पर जोड़ सकते हैं ताकि बुकिंग बढ़ाई जा सके।

एम्बेड विजेट्स तक पहुँचने के लिए, साइडबार से `Tools > Miscellaneous` पर जाएं।

## उपलब्ध विजेट्स

| विजेट | विवरण |
| -- | -- |
| **Account button** | एक ब्रांडेड बटन जो आपके एफिलिएट पेज से लिंक करता है |
| **Itinerary button** | एक बटन जो यात्रा चयनकर्ता खोलता है |
| **Itinerary form** | एक इनलाइन तारीख और अतिथि चयन फॉर्म |
| **Search button** | एक बटन जो प्रॉपर्टी खोज खोलता है |
| **Lookup button** | एक बटन जो विशिष्ट इन्वेंटरी खोजता है |

## एम्बेड कैसे करें

प्रत्येक विजेट एक कोड स्निपेट प्रदान करता है जिसे आप अपनी वेबसाइट के HTML में कॉपी और पेस्ट कर सकते हैं। इस स्निपेट में आपका **Client ID** और **Customization ID** शामिल होता है ताकि विजेट स्वचालित रूप से आपके एफिलिएट खाते से जुड़ा रहे।

:::note
यदि आप WordPress का उपयोग करते हैं, तो हम आसान एकीकरण के लिए एक WordPress प्लगइन भी प्रदान करते हैं।
:::
