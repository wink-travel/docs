हमारा समाधान पंजीकृत ट्रैवल एजेंट्स के लिए [Wink Agent](https://agent.wink.travel) है। यह एक सेल्फ-सर्व पोर्टल है जिसका उपयोग एजेंट अपने ग्राहकों की ओर से खोज करने और बुकिंग करने के लिए कर सकते हैं और प्रत्येक बुकिंग पर अपने लिए कमीशन कमा सकते हैं।

अधिक उन्नत ट्रैवल एजेंट भुगतान और भुगतान प्रसंस्करण के लिए Wink प्लेटफ़ॉर्म के बाहर API स्तर पर एकीकरण चुन सकते हैं।

ट्रैवल एजेंट्स के पास [Wink Studio](/hi/studio/what-is-studio) के समान पोर्टल सुविधाओं का एक उपसमूह होता है, इसलिए हम उन्हें यहां फिर से कवर नहीं करेंगे।

## खोज

होटल खोजने और बुक करने के लिए [Search](/hi/studio/search) पर जाएं।

अन्यथा, अगले अनुभाग पर जारी रखें।

## API

डेवलपर्स जो `Travel Agent` को प्रबंधित करना चाहते हैं, वे [Developers > API > Travel Agent](/hi/developers/apis/#travel-agent-api) पर जा सकते हैं।
