# सेटिंग्स

जब आप Social के साथ पहली बार शुरू करते हैं, तो हम आपसे पूछेंगे कि आप चाहते हैं कि हम आपकी सामग्री कैसे बनाएं। ये विकल्प हमेशा आपके लिए उपलब्ध होते हैं ताकि आप बाद में आसानी से उन्हें बदल सकें।

अपनी सेटिंग्स प्रबंधित करने के लिए साइडबार से `Preferences` पर जाएं।

Preferences सेक्शन में आपकी मूल सामग्री निर्माण सेटिंग्स होती हैं:

1. **Preferred customization** चुनें कि सामग्री बनाते समय कौन सी कस्टमाइज़ेशन का उपयोग करना है। कस्टमाइज़ेशन थीम रंग, डिफ़ॉल्ट यात्रा कार्यक्रम, मुद्रा, भाषा और अधिक को नियंत्रित करती हैं। आप [my.wink.travel](https://my.wink.travel) पर अपने खाते के तहत कस्टमाइज़ेशन बनाते हैं।
2. **Auto-approve** सक्षम होने पर, उत्पन्न पोस्ट बिना आपकी मंजूरी के स्वचालित रूप से प्रकाशित हो जाते हैं। जब आप हमारे द्वारा बनाई गई सामग्री की गुणवत्ता पर भरोसा कर लें, तो इसे सक्षम करें।
3. **Approver name** वह व्यक्ति या टीम का नाम जो अनुमोदन अनुरोध ईमेल प्राप्त करेगा।
4. **Approver e-mail** वह ईमेल पता जहां आगामी पोस्ट के अनुमोदन अनुरोध भेजे जाएंगे।

Social के साथ चुनने के लिए कई कंटेंट क्रिएटर्स आते हैं। प्रत्येक क्रिएटर आपके प्रॉपर्टी के लिए AI का उपयोग करके अनूठी सामग्री बनाने के एक विशिष्ट तरीके पर केंद्रित होता है। हर कंटेंट क्रिएटर के पास एक **Weight** गुण होता है — अधिक वेट होने पर उस क्रिएटर के चुने जाने की संभावना बढ़ जाती है।

यहाँ सभी उपलब्ध कंटेंट क्रिएटर्स हैं:

| Creator | विवरण |
| -- | -- |
| **Destination** | आपके स्थान का उपयोग करके क्षेत्र के बारे में सहायक टेक्स्ट बनाता है |
| **Room type** | आपके कमरे के प्रकारों में से एक को AI-जनित टेक्स्ट और आपके कमरे के प्रकार की छवियों के साथ बुक करने योग्य पोस्ट बनाता है |
| **Activity** | आपके किसी गतिविधि को प्रदर्शित करने वाला बुक करने योग्य पोस्ट बनाता है |
| **Add-On** | आपके किसी ऐड-ऑन को प्रदर्शित करने वाला बुक करने योग्य पोस्ट बनाता है |
| **Attraction** | आपके किसी आकर्षण को प्रदर्शित करने वाला बुक करने योग्य पोस्ट बनाता है |
| **Place** | आपके किसी स्थान को प्रदर्शित करने वाला बुक करने योग्य पोस्ट बनाता है |
| **Meeting room** | आपके किसी मीटिंग रूम को प्रदर्शित करने वाला बुक करने योग्य पोस्ट बनाता है |
| **Restaurant** | आपके किसी रेस्टोरेंट को प्रदर्शित करने वाला बुक करने योग्य पोस्ट बनाता है |
| **Spa** | आपके किसी स्पा को प्रदर्शित करने वाला बुक करने योग्य पोस्ट बनाता है |
| **Property** | आपके सबसे अच्छी कीमत वाले कमरे को AI-जनित टेक्स्ट और आपकी प्रॉपर्टी की छवियों के साथ बुक करने योग्य पोस्ट बनाता है |
| **User sentiment** | वेब पर मेहमानों की समीक्षाओं को एकत्रित करके एक भावना-आधारित पोस्ट बनाता है जो दर्शाता है कि मेहमान आपकी प्रॉपर्टी के बारे में क्या पसंद करते हैं |
| **Funny** | हल्के-फुल्के, मनोरंजक पोस्ट बनाता है जो आपके फीड को आकर्षक बनाए रखता है |

:::tip[Pro tip]
सुनिश्चित करें कि आपके चुने हुए कंटेंट क्रिएटर्स के लिए आवश्यक इन्वेंटरी प्रकार आपके पास उपलब्ध हो। उदाहरण के लिए, यदि आप "Restaurant" चुनते हैं, तो आपके प्रॉपर्टी पर कम से कम एक रेस्टोरेंट सूचीबद्ध होना चाहिए।
:::

Tone of voice वह भाषा शैली सेट करता है जिसका AI सामग्री बनाते समय उपयोग करता है। उन टोन को चुनें जो आपके ब्रांड की व्यक्तित्व से मेल खाते हों। कंटेंट क्रिएटर्स की तरह, प्रत्येक टोन का एक **Weight** होता है जो नियंत्रित करता है कि इसे कितनी बार उपयोग किया जाता है।

यहाँ उपलब्ध टोन हैं:

| Tone | शैली |
| -- | -- |
| **Neutral** | शांत, अधिकारपूर्ण, तथ्यात्मक। कोई अतिशयोक्ति नहीं, कोई मज़ाक नहीं — बस विश्वसनीय, सुव्यवस्थित ज्ञान |
| **Friendly** | गर्मजोशी भरा, सुलभ, टेक्स्ट में मुस्कुराहट के साथ, आकस्मिक संक्षेप और कभी-कभी इमोजी के साथ |
| **Luxury** | परिष्कृत, निखरा हुआ। "exquisite" और "bespoke" जैसे शब्द जो पाठकों को विशेष महसूस कराते हैं |
| **Foodie** | कामुक, मुँह में पानी लाने वाले विवरण। खाद्य प्रेमी भाषा जो पाठकों को भूखा कर देती है |
| **Adventurous** | साहसी, निडर, खोजी ऊर्जा। जीवंत संवेदी विवरण जो सब कुछ महाकाव्य बनाते हैं |
| **Family** | स्वस्थ, बच्चों के अनुकूल उत्साह। सुरक्षित और मजेदार अनुभवों के बारे में गर्म माता-पिता की ऊर्जा |
| **Romantic** | काव्यात्मक, उदासीन, दिल को छू लेने वाला। सूर्यास्त, कोमल भाषा, और प्रेम |
| **Humorous** | व्यंग्यात्मक, तेज़ एक-लाइनर, चतुर शब्दों का खेल। स्टैंड-अप सेट की तरह पढ़ता है |
| **Eco** | गंभीर और धार्मिक। हरित विकल्पों और टिकाऊ विकल्पों का जश्न मनाता है |
| **Local insider** | अत्यंत विशिष्ट स्थानीय संदर्भ, पर्यटक जाल से सावधान करता है, स्थानीय स्लैंग का उपयोग करता है |
| **Inclusive** | स्वागतयोग्य, उत्सवपूर्ण, और पुष्टि करने वाला। सभी को देखा और मूल्यवान महसूस कराता है |
| **Mystical** | अलौकिक, आध्यात्मिक, ब्रह्मांडीय सत्य। भीतर देखने के लिए कोमल निमंत्रण |
| **Futuristic** | चिकना, भविष्य-दृष्टि। "paradigm shift" और आशावादी भविष्य की दृष्टि जैसे शब्द पसंद करता है |
| **Budget** | जुगाड़ू, संसाधनपूर्ण, गर्व से किफायती। किफायती खोजों और मुफ्त अनुभवों का जश्न मनाता है |

## Post types

पोस्ट प्रकार निर्धारित करते हैं कि उत्पन्न सामग्री का लक्ष्य और संरचना क्या होगी। प्रत्येक पोस्ट प्रकार का एक **Weight** होता है जो नियंत्रित करता है कि इसे कितनी बार उपयोग किया जाता है।

| Post type | विवरण |
| -- | -- |
| **Transactional** | स्पष्ट, संक्षिप्त, रूपांतरण-केंद्रित। बुकिंग और सीधे क्रियाओं को तथ्यों, कीमतों, और अगले कदमों के साथ प्रेरित करता है |
| **Awareness** | ब्रांड-निर्माण सामग्री जो आपकी प्रॉपर्टी का परिचय देती है और संभावित मेहमानों के बीच दृश्यता बढ़ाती है |
| **Sentiment** | भावनात्मक रूप से प्रतिध्वनित सामग्री जो सकारात्मक संबंध बनाती है और आपके दर्शकों के साथ भावनात्मक जुड़ाव बनाती है |

अपने Google Business Profile को कनेक्ट करके, हम अधिक प्रभावी सामग्री के लिए समृद्ध प्रॉपर्टी विवरण प्राप्त कर सकते हैं। आपकी प्रॉपर्टी के पास एक सत्यापित Google Business Profile होना चाहिए ताकि हम आपकी लिस्टिंग से सामग्री उत्पन्न कर सकें। हमारे कुछ कंटेंट क्रिएटर्स के लिए यह **आवश्यक** है कि आपके पास Google Business खाता हो।
