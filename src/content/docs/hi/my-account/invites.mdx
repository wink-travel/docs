एक Wink उपयोगकर्ता आपको उनके किसी संपत्ति, सहयोगी या यात्रा एजेंट खाते का प्रबंधन करने का विकल्प दे सकता है। जब ऐसा होता है, तो आपको निमंत्रण अनुरोध के साथ एक ई-मेल प्राप्त होगा।

import { Steps } from '@astrojs/starlight/components';

अपने वर्तमान Wink निमंत्रण देखने के लिए, इन चरणों का पालन करें:

<Steps>
 1. [my.wink.travel](https://my.wink.travel) में लॉग इन करें।
 2. बाएं साइडबार से `Invites` पर क्लिक करें।
 3. आपको सभी वर्तमान निमंत्रणों की सूची दिखाई देगी।
 4. यह अनुभाग आपके सभी आवेदन सूचीबद्ध करेगा।
 5. निमंत्रण को `Accept` या `Reject` करने का विकल्प चुनें।
 6. यदि आपने निमंत्रण स्वीकार किया, तो खाता `Accounts` अनुभाग में दिखेगा।
</Steps>
