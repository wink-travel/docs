कभी-कभी, हम आपसे संपर्क करना चाहते हैं। जब ऐसा होगा, तो आप ऊपर दाईं ओर घंटी के बदलते हुए देखेंगे।  
जो हमने आपको भेजा है उसे पढ़ने के लिए:

<Steps>
1. मुख्य नेविगेशन बार में `Account > Notifications` पर क्लिक करें।  
2. आपको नई और पिछली सूचनाएं दिखेंगी।  
3. सूचनाएं पढ़ने के बाद उन्हें `Mark as read` या `Delete` करने का विकल्प चुनें।  
</Steps>

## API

डेवलपर्स जो `Affiliates` प्रबंधित करना चाहते हैं, वे [Developers > API > Affiliate](/hi/developers/apis/#affiliate-api) पर जा सकते हैं।
