यदि आप अपने Wink उपयोगकर्ता खाते के लिए अपनी उपयोगकर्ता प्रोफ़ाइल अपडेट करना चाहते हैं, तो इन चरणों का पालन करें:

<Steps>
1. [my.wink.travel](https://my.wink.travel) पर जाएं।
2. ऊपर-दाएँ कोने में `Sign-in / Register` पर क्लिक करके लॉग इन करें।
3. एक बार जब आप लॉग इन हो जाएं, तो ऊपर-दाएँ कोने में अपने प्रोफ़ाइल आइकन पर क्लिक करें, और एक मेनू दिखाई देगा।
4. `Profile` लिंक पर क्लिक करें।
5. अपने `e-mail`, `first` और `last name` को अपडेट करने के निर्देशों का पालन करें।
6. आप अपने नए `profile picture` भी अपलोड कर सकते हैं।
7. `Save` पर क्लिक करें।
</Steps>

<Steps>
1. आपको हमारी किसी वेबसाइट पर लॉग इन होना चाहिए।
2. ऊपर-दाएँ कोने में अपने प्रोफ़ाइल आइकन पर क्लिक करें और एक मेनू दिखाई देगा।
3. `Profile` लिंक पर क्लिक करें।
4. अपने `e-mail`, `first` और `last name` को अपडेट करने के निर्देशों का पालन करें।
5. आप अपने नए `profile picture` भी अपलोड कर सकते हैं।
6. `Update` पर क्लिक करें।
</Steps>

:::note
आपकी नई प्रोफ़ाइल तस्वीर अगली बार जब आप Wink से पूरी तरह लॉग आउट करेंगे, तब दिखाई देगी।
:::
