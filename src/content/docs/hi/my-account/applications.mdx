एप्लिकेशन आपको एक डेवलपर के रूप में Wink की सभी सुविधाओं तक पहुंच प्रदान करते हैं। आप सभी समान डेटा तक पहुंच सकते हैं और यहां तक कि Wink की अपनी सुविधाओं का विस्तार भी कर सकते हैं।  
आप अपने ट्रैवल साइट को प्रमाणित करने के लिए Wink के OAuth2 सर्वर का उपयोग कर सकते हैं, ठीक उसी तरह जैसे आप Google या Facebook का उपयोग करते हैं।  
इस तरह आप Wink की सुविधाओं को अपनी वेबसाइट या एप्लिकेशन में विस्तारित कर सकते हैं।

:::note
जब आप [नया खाता बनाते हैं](/hi/my-account/create-account), तो आपका पहला ऐप पहले से ही बनाया जाता है। हम कुछ बुकिंग परिदृश्यों के लिए इस ऐप का उपयोग करते हैं।

आप इस ऐप का पुनः उपयोग नहीं कर सकते। कृपया आवश्यकतानुसार नए ऐप बनाएं।
:::

:::note
Wink पर जनरेट किया गया एक एप्लिकेशन सभी API एंडपॉइंट्स के लिए उपयोग किया जा सकता है।
:::

<Steps>
 1. [Accounts](https://my.wink.travel) से उस खाते का चयन करें जिसके साथ आप काम करना चाहते हैं।
 2. खाते के कार्ड के नीचे `Actions` पर क्लिक करें और `Manage account` चुनें।
 3. अगली पृष्ठ पर `Applications` टैब पर क्लिक करें।
 4. इस सेक्शन में आपके सभी एप्लिकेशन सूचीबद्ध होंगे।
 5. किसी मौजूदा ऐप को संपादित करने या नया ऐप बनाने का विकल्प चुनें।
 </Steps>

## ऐप बनाएं

चूंकि आप हमारे द्वारा बनाए गए ऐप की `secret key` तक पहुंच नहीं सकते, इसलिए आपको अपना खुद का ऐप बनाना होगा।

:::caution
ऐप बनाने से पहले आपको Wink पर एक एफिलिएट खाता या TripPay पर एक खाता बनाना होगा क्योंकि आप अपने एप्लिकेशन को एक विशिष्ट खाते से जोड़ेंगे।
:::

यहाँ एप्लिकेशन बनाने के चरण दिए गए हैं:

<Steps>
1. ऐप्स पेज से, `Create new application` बटन पर क्लिक करें।
2. ***नाम*** अपने ऐप को एक नाम दें। *जैसे Cool App*
3. **Entity name** अपने ऐप को अपने किसी मौजूदा खाते से लिंक करें। *जैसे Cool Account*
4. **Redirect URIs** कम से कम एक रीडायरेक्ट URI आवश्यक है। यह सफल प्रमाणीकरण के बाद रीडायरेक्ट डोमेन को मान्य करने के लिए उपयोग किया जाता है। *जैसे https://www.cool-site.com*
5. जारी रखने के लिए `Save` बटन पर क्लिक करें।
</Steps>

आपको आपके एप्लिकेशन की सूची पर वापस रीडायरेक्ट किया जाएगा। आप अपनी नई ऐप सूची में देखेंगे और केवल एक बार ऐप का secret key प्राप्त कर पाएंगे। इसे कॉपी करके सुरक्षित स्थान पर सहेजना सुनिश्चित करें।

## ऐप अपडेट करें

यदि आपके किसी रीडायरेक्ट URI में बदलाव होता है या आप कोई URI जोड़ना या हटाना चाहते हैं, तो आपको अपना ऐप अपडेट करना होगा।

मौजूदा ऐप को अपडेट करने के लिए, निम्न करें:

<Steps>
1. उस ऐप के लिए `Actions` लिंक पर क्लिक करें जिसे आप अपडेट करना चाहते हैं।
2. Actions के अंतर्गत `Update` बटन पर क्लिक करें।
3. ऐप में आवश्यक परिवर्तन करें।
4. जारी रखने के लिए `Save` बटन पर क्लिक करें।
</Steps>

आपको आपके एप्लिकेशन की सूची पर वापस रीडायरेक्ट किया जाएगा।

## ऐप हटाएं

:::caution
कृपया वह ऐप न हटाएं जिसे हमने आपके लिए स्वचालित रूप से बनाया है ताकि आपके खाते के लिए सभी सुविधाएं सही ढंग से काम करें।
:::

यदि आपको किसी ऐप की अब आवश्यकता नहीं है, तो आप उसे हटा सकते हैं।

ऐप हटाने के लिए, निम्न करें:

<Steps>
1. उस ऐप के लिए `Actions` लिंक पर क्लिक करें जिसे आप हटाना चाहते हैं।
2. Actions के अंतर्गत `Remove` बटन पर क्लिक करें।
3. हटाने की पुष्टि के लिए `OK` बटन पर क्लिक करें।
</Steps>

आपको आपके एप्लिकेशन की सूची पर वापस रीडायरेक्ट किया जाएगा।
