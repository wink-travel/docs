Un utilizator Wink poate alege să îți permită să gestionezi unul dintre conturile lor de proprietate, afiliat sau agent de turism. Când se întâmplă acest lucru, vei primi un e-mail cu o cerere de invitație.

import { Steps } from '@astrojs/starlight/components';

Pentru a vedea invitațiile tale Wink curente, urmează acești pași:

<Steps>
1. Trebuie să fii autentificat pe unul dintre site-urile noastre.
2. Fă clic pe pictograma profilului tău, în colțul din dreapta sus, și va apărea un meniu.
3. Fă clic pe linkul `Invites`.
4. Interacționează cu invitațiile tale fie prin `Acceptarea`, fie prin `Respingerea` lor.
</Steps>
