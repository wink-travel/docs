Un utilizator Wink poate alege să vă permită să gestionați unul dintre conturile sale de proprietate, afiliat sau agent de turism. Când se întâmplă acest lucru, veți primi un e-mail cu o solicitare de invitație.

import { Steps } from '@astrojs/starlight/components';

Pentru a vedea invitațiile tale Wink actuale, urmează acești pași:

<Steps>
  1. Trebuie să fii autentificat pe unul dintre site-urile noastre web.
  2. Faceți clic pe pictograma profilului dvs., în colțul din dreapta sus, și va apărea un meniu.
  3. Faceți clic pe `Invites` legătură.
  4. Interacționează cu invitațiile tale fie prin `Accepting` sau `Rejecting` ei.
</Steps>
