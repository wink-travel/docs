Această pagină listează toate evenimentele webhook la care te poți abona pe Wink. Pentru instrucțiuni de configurare, vezi [Ghidul de integrare Webhook](/ro/guides/webhook-integration) sau [pagina de gestionare a Webhook-urilor](/ro/my-account/webhooks).

## Cum să te abonezi

1. Navighează la **Applications > Webhooks** în portalul tău.
2. Creează un webhook și selectează evenimentele pe care dorești să le primești.
3. Wink trimite un HTTP POST către endpoint-ul tău de fiecare dată când evenimentul are loc.

Pentru pașii compleți de configurare și bune practici, vezi [Integrarea Webhook](/ro/guides/webhook-integration).

## Evenimente cont

Evenimente legate de ciclul de viață și setările contului tău TripPay.

| Eveniment | Cheie | Descriere |
|---|---|---|
| Cont creat | `account.create` | Un cont nou a fost creat. |
| Cont în curs de revizuire | `account.under_review` | Contul este în așteptare pentru revizuire. |
| Cont actualizat | `account.updated` | Detaliile contului s-au schimbat. |
| Stare cont actualizată | `account.status.updated` | Starea contului s-a schimbat (ex: aprobat, suspendat). |
| Adresă cont actualizată | `account.address.updated` | Adresa contului a fost modificată. |
| Detalii cont actualizate | `account.details.updated` | Câmpurile de detalii ale contului s-au schimbat. |
| Informații cont actualizate | `account.information.update` | Informațiile generale ale contului au fost actualizate. |
| Manageri cont actualizați | `account.managers.update` | Lista managerilor contului s-a schimbat. |
| Poză profil actualizată | `account.profile_picture.updated` | Poză de profil a contului a fost schimbată. |
| Agent actualizat | `account.agent.updated` | Informațiile agentului din cont au fost modificate. |
| Nume URL actualizat | `account.url-name.update` | URL-ul personalizat al contului s-a schimbat. |
| ID unic actualizat | `account.unique-id.update` | Identificatorul unic al contului a fost modificat. |
