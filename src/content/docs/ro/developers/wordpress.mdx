Pentru oricine folosește WordPress pentru afacerea sau blogul său, vă recomandăm cu căldură să utilizați [Pluginul WordPress Wink Affiliate](https://wordpress.org/plugins/wink2travel/) Pluginul vă oferă acces facil la cardurile, grilele, hărțile Wink etc. și simplifică procesul de a introduce conținut în postări și pagini folosind constructorul de pagini preferat sau shortcode-urile noastre personalizate.

## Instala

Așa se instalează`Wink Affiliate WordPress Plugin`în site-ul tău WordPress.

<Steps>
  1. Când ești autentificat ca administrator și te afli în Panoul de control.
  2. Clic `Plugins > Add New Plugin`.
  3. Caută `Wink Affiliate WordPress Plugin`.
  4. Clic `Install` pentru a descărca și instala pluginul pe site-ul dvs.
  5. Clic `Activate` pentru a activa pluginul pe site-ul dvs.
  6. Activarea vă va transmite mai departe `Wink Settings` pagină. *De asemenea, puteți ajunge acolo navigând către `Appearance > Customize > Wink Settings`.*
  7. Vă va cere `Client ID` şi `Client Secret`Accesați [Aplicații](/developers/applications) pentru a afla cum să vă recuperați acreditările.
  8. Introduceți acreditările și faceți clic pe `Publish`Acum ești conectat la contul tău Wink.
</Steps>

## Constructori de pagini

Oferim suport pentru cei mai populari constructori de pagini pe WordPress, ceea ce facilitează utilizarea Wink pentru oricine dorește.

### Gutenberg

Gutenberg este editorul nativ din WordPress.

### Avada

[Avada](https://avada.com) este renumit pentru selecția sa largă de șabloane frumoase, împreună cu un constructor de pagini robust cu un set avansat de funcții.

### Elementor

[Elementor](https://elementor.com) este cel mai popular constructor de pagini \[în prezent], cu o versiune gratuită și una pro disponibilă.

### WPBakery

[WPBakery](https://wpbakery.com) a fost primul constructor de pagini avansat de pe piață.

## Coduri scurte

### \[căutare cu clinchet]

Funcționează cu o grilă de conținut clasat. Introduceți un loc pe care doriți să îl vizitați și să afișați inventarul.

### \[winksearch]

Buton simplu pentru a deschide formularul de itinerariu.

### \[contwink]

Butonul Cont pentru a vă permite să vă autentificați. După autentificare, se transformă într-un meniu derulant cu opțiuni pentru cont.

### \[winkitinery]

La fel ca butonul de căutare, doar că conține informații despre itinerariu sub formă de text.

### \[winkcontent]

Îți permite să încorporezi inventarul selectat pe Wink direct într-o pagină sau postare. Acest shortcode așteaptă și un ID de layout.

Aspectul

```
[winkcontent layoutid=""]
```
