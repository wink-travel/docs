Abilitățile AI sunt pachete portabile de instrucțiuni care învață agenții de programare cum să lucreze cu Wink. În loc să se bazeze pe date de antrenament sau documentație învechită, fiecare abilitate direcționează agentul să folosească [serverul Wink MCP](/ro/developers/mcp) ca sursă live de adevăr pentru contractele API și documentația platformei.

Abilitățile funcționează cu orice client agent care le suportă — Claude Code, Codex, Cursor și alții.

## Abilități disponibile

### wink-api-builder

Implementează, revizuiește sau depanează integrări API Wink. Abilitatea ghidează agentul printr-un flux de lucru structurat:

1. Caută operația API potrivită prin `api_search`
2. Deschide contractele live de cerere/răspuns de pe serverul MCP
3. Preia configurarea autentificării din documentație
4. Scrie cod conform contractului verificat
5. Semnalează orice nepotriviri între codul existent și API-ul curent

Util pentru:
- Construirea de noi metode client pentru orice API Wink
- Depanarea nepotrivirilor de contract (ex. o cerere care returnează HTTP 400)
- Revizuirea integrărilor existente față de specificația live

## Instalare

Copiază folderul cu abilitatea în directorul de abilități al agentului tău.

:::tip
Serverul Wink MCP trebuie să fie conectat în același mediu. Vezi [Server MCP](/ro/developers/mcp) pentru instrucțiuni de configurare.
:::

**Claude Code**

```bash
# la nivel de repo
cp -r skills/wink-api-builder .claude/skills/

# sau la nivel de utilizator
cp -r skills/wink-api-builder ~/.claude/skills/
```

Invocă cu:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# la nivel de repo
cp -r skills/wink-api-builder .agents/skills/

# sau la nivel de utilizator
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Invocă cu:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Alți agenți**

Plasează folderul cu abilitatea oriunde rezolvă agentul tău abilitățile. Punctul de intrare este `SKILL.md`.

## Cerințe

- Un [server Wink MCP](/ro/developers/mcp) conectat
- Un client agent cu suport MCP și abilități
