Accelerează timpul de lansare pe piață utilizând unul dintre kiturile noastre de dezvoltare software pentru limbaje acceptate oficial.

Dacă în prezent nu acceptăm limbajul tău preferat, accesează API-ul pe care dorești să îl utilizezi și descarcă specificațiile OpenAPI pe care le punem la dispoziție pentru fiecare API în partea de sus a paginii. Poți genera propriul SDK fie prin crearea uneia dintre depozitele noastre publice de SDK, fie utilizând unul dintre generatoarele OpenAPI.

## Java

Această bibliotecă a fost compilată pentru Java 24. Este configurată să utilizeze WebClient reactiv și este optimizată pentru utilizarea cu Spring Framework. Cu toate acestea, Spring nu este o cerință, dar oferă suport OAuth2 imediat.
