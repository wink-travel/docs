Accelerează timpul de lansare pe piață folosind unul dintre Software Development Kit-urile noastre oficial suportate pentru limbaje.
Dacă în prezent nu suportăm limbajul tău preferat, accesează API-ul pe care vrei să-l folosești și descarcă specificația OpenAPI pe care o punem la dispoziție pentru fiecare API în partea de sus a paginii. Poți genera propriul SDK fie prin forkarea unuia dintre depozitele noastre publice de SDK-uri, fie folosind unul dintre generatoarele OpenAPI.

## Java

Această bibliotecă a fost compilată pentru Java 24. Este configurată să folosească WebClient reactiv și este optimizată pentru utilizarea cu Spring Framework. Totuși, Spring nu este o cerință, dar îți oferă suport OAuth2 din cutie.
