## Produce endpoints

Endpoint-urile Produce sunt pentru dezvoltatorii care doresc să creeze și să gestioneze inventar de călătorie.

### Registration API

Ca producător, acesta este adesea punctul de plecare al călătoriei dvs. Aceste endpoint-uri vă permit să creați proprietăți pe Wink.

<LinkButton href="https://developer.wink.travel/extranet/property/register" variant="secondary" icon="external" target="_blank">
Aflați mai multe
</LinkButton>

## Analytics API

Analytics API expune endpoint-uri pentru oricine dorește să obțină informații despre clasamentele din leaderboard, precum și o perspectivă detaliată asupra seriilor temporale, la nivel de platformă; găzduind o varietate largă de metrici.

<LinkButton href="https://developer.wink.travel/analytics" variant="secondary" icon="external" target="_blank">
Aflați mai multe
</LinkButton>

## Payment API

Endpoint-urile Payment sunt pentru dezvoltatorii care doresc să achiziționeze inventar de călătorie. Acest lucru se poate face prin API ca agent de turism înregistrat sau folosind API-ul nostru împreună cu widget-ul nostru de plată conform PCI pentru toate celelalte entități.
Payment API expune endpoint-uri pentru afiliați și hoteluri pentru a urmări rezervările, analizele și disponibilitatea fondurilor. De asemenea, pot alege să retragă fondurile disponibile în contul lor bancar.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Aflați mai multe
</LinkButton>
