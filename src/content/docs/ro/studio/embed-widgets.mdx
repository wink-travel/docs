Studio oferă mai multe widgeturi încorporabile pe care le poți adăuga pe propriul site pentru a stimula rezervările.

Pentru a accesa widgeturile încorporate, navighează la `Tools > Miscellaneous` din bara laterală.

## Widgeturi disponibile

| Widget | Descriere |
| -- | -- |
| **Buton cont** | Un buton branduit care face legătura către pagina ta de afiliat |
| **Buton itinerariu** | Un buton care deschide selectorul de itinerariu |
| **Formular itinerariu** | Un formular în linie pentru selectarea datei și a oaspeților |
| **Buton căutare** | Un buton care deschide căutarea proprietăților |
| **Buton căutare inventar** | Un buton pentru a căuta inventar specific |

## Cum să încorporezi

Fiecare widget oferă un fragment de cod pe care îl poți copia și lipi în HTML-ul site-ului tău. Fragmentul include **Client ID** și **Customization ID** astfel încât widgetul să fie automat legat de contul tău de afiliat.

:::note
Dacă folosești WordPress, oferim și un plugin WordPress pentru o integrare mai ușoară.
:::
