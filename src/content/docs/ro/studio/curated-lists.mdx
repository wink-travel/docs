Listele selectate sunt o funcționalitate care îți permite să grupezi și să sortezi inventarul pe care l-ai selectat din lucrurile găsite în timpul căutării de inventar de călătorie.
Odată create, poți expune lista utilizatorilor tăi făcând-o disponibilă spre vânzare ca [grilă](/ro/studio/grids) sau [hartă](/ro/studio/maps) prin [WinkLinks](/ro/link-manager/wink-links), pluginul nostru pentru [WordPress](/ro/developers/wordpress/) sau prin încorporarea unuia dintre [Web Components](/ro/developers/web-components) în site-ul tău.

Implicit, fiecare cont afiliat începe cu o listă `Favorite` pe care o poți folosi imediat. `Favorite` este specială și nu poate fi ștearsă.
Toate listele selectate îți permit să faci următoarele:

## Creează o listă

- Din tabloul de bord, dă clic pe `Inventar > Liste selectate` în bara de navigare.
- Creează o listă selectată nouă făcând clic pe butonul `Creează o listă nouă`.
- Ți se va cere să denumești lista. *Exemplu: Top 3 alegeri NYC*.
- Dă clic pe butonul `Salvează`.

## Adaugă un element în listă

Poți adăuga un element în orice listă din rezultatele căutării.

- Mergi la [Căutare](/ro/studio/search)
- Caută ceva ce vrei să adaugi în lista ta.
- Odată ce îl găsești, dă clic pe linkul `Acțiuni` din colțul stânga-jos al cardului rezultatului căutării.
- Dă clic pe `Adaugă la...` și apoi selectează lista selectată în care vrei să adaugi acest inventar.
- Revino la pagina listelor selectate pentru a verifica dacă elementul a fost adăugat în listă.

Fiecare listă are câteva acțiuni pe care le poți executa asupra ei.

- Convertește lista într-o [hartă](/ro/studio/maps).
- Convertește lista într-o [grilă](/ro/studio/grids).
- Elimină lista (*cu excepția Favorite*).

## API

Dezvoltatorii care doresc să gestioneze liste selectate pot accesa [Dezvoltatori > API > Browse](/ro/developers/apis/#browse-api).
