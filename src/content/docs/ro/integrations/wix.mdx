Aplicația **Wink Wix App** adaugă componente web Wink pe site-ul tău Wix ca widgeturi native pe care le poți trage și plasa în Editorul Wix. Nu este necesară programare pentru proprietarii de site-uri.

## Instalare

<Steps>
1. Accesează [Wix App Market](https://www.wix.com/app-market) și caută **Wink Travel**.
2. Apasă pe **Add to Site** și urmează pașii pentru a autoriza aplicația.
3. După instalare, deschide **Dashboard → Wink Settings** și introdu **Client ID** și **Client Secret**.
   Accesează [Applications](/ro/portal/settings/applications) pentru a-ți recupera acreditările.
4. Apasă **Save** și apoi **Test Connection** pentru a confirma că Wink este conectat.
</Steps>

## Adaugă un component pe o pagină

<Steps>
1. Deschide **Wix Editor** și accesează pagina unde vrei să adaugi conținut Wink.
2. Apasă **Add (+) → Apps → Wink Travel** pentru a vedea toate widgeturile disponibile.
3. Trage un widget pe pagină — de exemplu, **Wink Content** pentru a afișa o grilă de hoteluri.
4. În panoul de setări al widgetului, selectează **Layout Type** și lipește **Layout ID** din Wink Studio.
5. Apasă **Publish** pentru a face modificările vizibile.
</Steps>

## Widgeturi disponibile

| Widget | Scop |
|---|---|
| **Wink Content** | Afișează carduri de hoteluri, grile ordonate, hărți și altele |
| **Wink Lookup** | Bară de căutare pentru destinații și hoteluri |
| **Wink Search** | Buton icon care deschide selectorul de itinerariu |
| **Wink Account** | Buton de autentificare / cont utilizator |
| **Wink Itinerary** | Buton itinerariu care arată călătoria curentă |
| **Wink Cart** | Buton sumar coș de cumpărături |
