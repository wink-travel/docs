Extensia **Wink TYPO3** integrează inventarul de călătorii Wink în TYPO3 13 ca elemente native de conținut. Editorii adaugă componente Wink pe pagini folosind editorul standard de pagini TYPO3 — nu sunt necesare modificări ale șablonului.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"
  href="https://github.com/wink-travel/wink-typo3-extension"
  description="Cod sursă, ghid de configurare pentru dezvoltatori și instrucțiuni de contribuție."
/>

## Cerințe

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instalare

<Steps>
1. Instalează prin Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Activează extensia:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. În backend-ul TYPO3, accesează **Admin Tools → Settings → Extension Configuration → wink**.
4. Introdu **Client ID**, **Client Secret** și selectează mediul. Salvează.
   Accesează [Applications](/ro/portal/settings/applications) pentru a obține acreditările.
5. Adaugă șablonul static TypoScript în șablonul rădăcină:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Adaugă un element de conținut pe o pagină

<Steps>
1. Deschide o pagină în modulul **Page**.
2. Apasă butonul **+** în zona de conținut pentru a adăuga un element nou.
3. Alege unul dintre elementele de conținut **Wink Travel**:
   - **Wink Content** — carduri hotel, grile ordonate, hărți
   - **Wink Lookup** — bara de căutare destinații
   - **Wink Search** — buton selector itinerariu
   - **Wink Account** — buton de autentificare
   - **Wink Itinerary** — buton status itinerariu
   - **Wink Cart** — buton coș de cumpărături
4. Pentru **Wink Content**, folosește câmpurile FlexForm pentru a selecta un **Tip Layout** și a introduce **ID-ul Layoutului**.
5. Salvează și publică pagina.
</Steps>

## Elemente de conținut disponibile

| Element | Scop |
|---|---|
| **Wink Content** | Carduri hotel, grile ordonate, hărți și altele |
| **Wink Lookup** | Bara de căutare destinații |
| **Wink Search** | Buton icon care deschide selectorul de itinerariu |
| **Wink Account** | Buton de autentificare / cont utilizator |
| **Wink Itinerary** | Buton itinerariu care afișează călătoria curentă |
| **Wink Cart** | Buton sumar coș de cumpărături |
