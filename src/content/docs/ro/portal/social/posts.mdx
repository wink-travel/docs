Editorul de postări îți permite să creezi un număr nelimitat de postări manuale și este, de asemenea, locul unde gestionezi postările create automat pentru tine.

Editorul are 4 secțiuni:

1. Secțiunea de conținut.
2. Secțiunea de media.
3. Secțiunea de link.
4. Secțiunea de setări.

## Conținut

Secțiunea de conținut îți permite să modifici textul înainte de publicare. Poți crea conținut localizat în mai multe limbi — fiecare limbă are propriul titlu și descriere.

## Media

Secțiunea de media îți permite să modifici imaginile și videoclipurile care însoțesc textul.

## Link

Secțiunea de link îți permite să adaugi un link către o destinație unde dorești ca utilizatorul să ajungă după ce citește postarea ta. *De exemplu, acesta poate fi un link către o ofertă specială sau o pagină cu mai multe informații.* Poți adăuga URL-ul linkului, o legendă pentru link și o descriere a linkului.

## Setări

Secțiunea de setări îți permite să modifici / suprascrii toate setările implicite pentru fiecare postare. Poți seta:

- **Platforma** — Alege pe ce rețea socială va fi publicată această postare.
- **Tipul postării** — Alege între Tranzacțional, Conștientizare sau Sentiment.
- **Tonul vocii** — Suprascrie tonul vocii pentru această postare specifică.
- **Personalizare** — Folosește o personalizare diferită pentru moneda afișată, limbă, itinerariu etc.
- **Fus orar** — Suprascrie fusul orar pentru publicare.
- **Data programată** — Postările generate sunt programate automat la momentul optim. Poți suprascrie data și ora dacă preferi un alt interval.

## Ciclu de viață al stării postării

Fiecare postare urmează un ciclu de viață al stării:

| Stare | Descriere |
| -- | -- |
| **Ciornă** | Postarea a fost creată, dar nu este încă gata pentru revizuire |
| **În așteptare de aprobare** | Postarea așteaptă aprobarea umană înainte de programare |
| **Aprobată** | Postarea a fost aprobată și va fi programată |
| **Respinsă** | Postarea a fost revizuită și respinsă — nu va fi publicată |
| **Programată** | Postarea este în coadă pentru publicare la data și ora programate |
| **Publicată** | Postarea a fost publicată cu succes pe rețeaua socială |
| **Pauză** | Publicarea postării a fost temporar întreruptă |
| **Eșuată** | Postarea nu a putut fi publicată din cauza unei erori |

## Aprobarea postărilor

Postările sunt aprobate în grila de postări, unde poți selecta una sau mai multe postări și alege să le aprobi sau să le respingi.

:::note
Va exista o intrare pentru fiecare limbă și fiecare rețea socială.

De exemplu, dacă ai 3 rețele sociale și 3 limbi selectate, vei vedea 9 intrări în calendar.
:::
