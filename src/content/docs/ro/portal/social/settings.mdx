# Setări

Când începi să folosești Social, îți vom cere cum dorești să creăm conținutul tău. Aceste opțiuni sunt disponibile oricând, astfel încât să le poți schimba ușor mai târziu.

Navighează la `Preferences` din bara laterală pentru a-ți gestiona setările.

Secțiunea de preferințe conține setările tale de bază pentru generarea conținutului:

1. **Personalizare preferată** Alege ce personalizare să folosești la generarea conținutului. Personalizările controlează culorile temei, itinerariul implicit, moneda, limba și altele. Creezi personalizări în contul tău pe [my.wink.travel](https://my.wink.travel).
2. **Aprobare automată** Când este activată, postările generate sunt publicate automat fără a necesita aprobarea ta. Activează această opțiune doar când ești sigur de calitatea conținutului pe care îl producem pentru tine.
3. **Nume aprobator** Numele persoanei sau echipei care va primi e-mailurile cu cereri de aprobare.
4. **E-mail aprobator** Adresa de e-mail unde vor fi trimise cererile de aprobare pentru postările viitoare.

## Creatori de conținut

<figure>

Social vine cu mulți creatori de conținut din care poți alege. Fiecare creator se concentrează pe o modalitate specifică de a crea conținut unic pentru proprietatea ta folosind AI. Fiecare creator de conținut are un atribut **Weight** — o valoare mai mare crește șansa ca acel creator să fie selectat la generarea postărilor.

Iată toți creatorii de conținut disponibili:

| Creator | Descriere |
| -- | -- |
| **Destinație** | Folosește locația ta pentru a crea texte utile despre zonă |
| **Tip cameră** | Creează o postare rezervabilă cu unul dintre tipurile tale de camere, cu text generat de AI și imagini ale tipului de cameră |
| **Activitate** | Creează o postare rezervabilă cu una dintre activitățile tale |
| **Add-On** | Creează o postare rezervabilă cu unul dintre suplimentele tale |
| **Atracție** | Creează o postare rezervabilă cu una dintre atracțiile tale |
| **Loc** | Creează o postare rezervabilă cu unul dintre locurile tale |
| **Sală de întâlniri** | Creează o postare rezervabilă cu una dintre sălile tale de întâlniri |
| **Restaurant** | Creează o postare rezervabilă cu unul dintre restaurantele tale |
| **Spa** | Creează o postare rezervabilă cu unul dintre spa-urile tale |
| **Proprietate** | Creează o postare rezervabilă cu camera ta cea mai bine cotată, cu text generat de AI și imagini ale proprietății |
| **Sentiment utilizator** | Agregă recenzii ale oaspeților de pe internet într-o postare bazată pe sentimente care evidențiază ce le place oaspeților la proprietatea ta |
| **Amuzant** | Creează postări ușoare și distractive pentru a menține feed-ul captivant |

:::tip[Pro tip]
Asigură-te că ai tipul de inventar necesar pentru creatorii de conținut aleși. De exemplu, dacă selectezi „Restaurant”, trebuie să ai cel puțin un restaurant listat în proprietatea ta.
:::

Tonul vocii stabilește stilul de limbaj pe care AI îl folosește la generarea conținutului. Alege tonurile care se potrivesc personalității brandului tău. Ca și în cazul creatorilor de conținut, fiecare ton are un **Weight** pentru a controla cât de des este folosit.

Iată tonurile disponibile:

| Ton | Stil |
| -- | -- |
| **Neutru** | Calm, autoritar, factual. Fără exagerări, fără glume — doar cunoștințe fiabile și bine structurate |
| **Prietenos** | Cald, accesibil, cu energie zâmbitoare prin text, cu contracții casual și emoji ocazionale |
| **Lux** | Sofisticat, rafinat. Cuvinte precum „exquisit” și „personalizat” care fac cititorii să se simtă exclusivi |
| **Gurmand** | Descrieri senzuale, care îți fac gura apă. Limbaj de food porn care stârnește pofta |
| **Aventuros** | Îndrăzneț, neînfricat, cu energie de explorator. Descrieri senzoriale vii care fac totul să sune epic |
| **Familial** | Entuziasm sănătos, potrivit pentru copii. Energie caldă parentală despre experiențe sigure și distractive |
| **Romantic** | Poetic, visător, care emoționează. Apusuri, limbaj blând și dragoste |
| **Umoristic** | Sarcastic, replici scurte, jocuri de cuvinte inteligente. Se citește ca un show de stand-up |
| **Eco** | Sincer și drept. Celebrează alternativele verzi și alegerile sustenabile |
| **Local insider** | Referințe locale foarte specifice, avertizează asupra capcanelor pentru turiști, folosește argou local |
| **Incluziv** | Primitor, festiv și afirmativ. Face pe toată lumea să se simtă văzută și apreciată |
| **Mistic** | Eteric, spiritual, adevăruri cosmice. Invitații blânde de a privi în interior |
| **Futurist** | Elegant, orientat spre viitor. Iubește termeni ca „schimbare de paradigmă” și viziuni optimiste despre viitor |
| **Buget** | Ingenios, resursiv, mândru de frugalitate. Celebrează descoperirile accesibile și experiențele gratuite |

## Tipuri de postări

Tipurile de postări determină scopul și structura conținutului generat. Fiecare tip de postare are un **Weight** pentru a controla cât de frecvent este folosit.

| Tip postare | Descriere |
| -- | -- |
| **Tranzacțional** | Clar, concis, orientat spre conversie. Stimulează rezervările și acțiunile directe cu fapte, prețuri și pași următori |
| **Conștientizare** | Conținut de construire a brandului conceput să introducă proprietatea ta și să crească vizibilitatea în rândul potențialilor oaspeți |
| **Sentiment** | Conținut emoțional care creează asocieri pozitive și construiește o conexiune emoțională cu audiența ta |

Poate dorești să îți concentrezi eforturile de marketing pe o anumită limbă și regiune. Spune-ne limbile pentru care vrei să creezi conținut și fusul orar care te interesează, iar noi vom face restul. Postările sunt publicate la ora programată în fusul orar selectat.

*Majoritatea proprietăților folosesc o singură limbă și implicit limba locală.*

Prin conectarea profilului tău Google Business, putem prelua detalii mai bogate despre proprietate pentru un conținut mai eficient. Proprietatea ta trebuie să aibă un profil Google Business verificat pentru a putea genera conținut din listarea ta. Unii dintre creatorii noștri de conținut **cer** să ai un cont Google Business.

:::tip[Pro tip]
Începe cu pași mici

Nu fi prea ambițios la prima încercare. Îți recomandăm să începi prin a crea conținut pentru proprietatea ta într-o singură limbă.
:::
