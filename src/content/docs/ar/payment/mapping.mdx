إذا كنت مدمجًا، مثل Wink، وترغب في تعيين المستفيدين الخارجيين لاستخدامهم مع TripPay، يمكنك القيام بذلك بإحدى الطريقتين:

- التعيين اليدوي
- استخدام [Payment API](/ar/developers/apis/#payment-api) (*انظر `Account Mappings`*)

:::tip
ما لم تكن تتعامل مع عدد قليل فقط من نفس العقارات، نوصي باستخدام API لأتمتة هذه الخطوة.
:::

فيما يلي، نرشدك خلال الطريقة اليدوية لتعيين مستفيد خارجي. *تستخدم API في الخلفية.*

<Steps>
1. سجّل الدخول إلى [TripPay](https://pay.wink.travel).
2. اختر حساب المدمج الخاص بك.
3. انقر على `Settings > Mappings` من شريط التنقل الرئيسي.
4. انقر على زر `Create mapping`. سترى نموذج التعيين.
5. **Local account** ابدأ بكتابة اسم حساب TripPay الموجود الذي تريد تعيينه.
6. **External identifier** الصق معرفك الخارجي لهذا الكيان كما هو موجود في النظام البعيد.
7. انقر على `Save` للمتابعة. سيتم إعادة توجيهك إلى قائمة التعيينات الخاصة بك.
</Steps>

مع وجود التعيينات الخاصة بك، يعرف TripPay الآن الحسابات التي يجب اعتمادها كمستفيدين من الحجز عند الإشارة إليها باستخدام معرفاتك الخارجية.

## API

للمطورين الذين يرغبون في إدارة `Payments`، انتقل إلى [Developers > API > Payments](/ar/developers/apis/#payment-api).
