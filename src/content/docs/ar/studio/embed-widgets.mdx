يوفر Studio عدة أدوات مصغرة قابلة للتضمين يمكنك إضافتها إلى موقعك الإلكتروني لزيادة الحجوزات.

للوصول إلى الأدوات المصغرة القابلة للتضمين، انتقل إلى `Tools > Miscellaneous` من الشريط الجانبي.

## الأدوات المصغرة المتاحة

| الأداة المصغرة | الوصف |
| -- | -- |
| **زر الحساب** | زر يحمل العلامة التجارية ويرتبط بصفحة الشريك الخاصة بك |
| **زر خط سير الرحلة** | زر يفتح محدد خط سير الرحلة |
| **نموذج خط سير الرحلة** | نموذج مدمج لاختيار التاريخ وعدد الضيوف |
| **زر البحث** | زر يفتح بحث العقارات |
| **زر البحث عن المخزون** | زر للبحث عن مخزون محدد |

## كيفية التضمين

كل أداة مصغرة توفر مقطع كود يمكنك نسخه ولصقه في HTML الخاص بموقعك الإلكتروني. يتضمن المقطع **معرّف العميل** و**معرّف التخصيص** الخاصين بك بحيث يتم ربط الأداة المصغرة تلقائيًا بحساب الشريك الخاص بك.

:::note
إذا كنت تستخدم WordPress، نوفر أيضًا إضافة WordPress لتسهيل التكامل.
:::
