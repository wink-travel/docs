تدمج **إضافة Wink TYPO3** مخزون السفر من Wink في TYPO3 13 كعناصر محتوى أصلية. يقوم المحررون بإضافة مكونات Wink إلى الصفحات باستخدام محرر صفحات TYPO3 القياسي — دون الحاجة إلى تغييرات في القالب.

## المتطلبات

- TYPO3 13.x
- PHP 8.2+
- Composer

## التثبيت

<Steps>
1. التثبيت عبر Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. تفعيل الإضافة:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. في لوحة تحكم TYPO3، اذهب إلى **أدوات الإدارة → الإعدادات → تكوين الإضافة → wink**.
4. أدخل **معرف العميل**، **سر العميل**، واختر بيئتك. احفظ.
   اذهب إلى [التطبيقات](/ar/portal/settings/applications) لاسترجاع بيانات الاعتماد الخاصة بك.
5. أضف قالب TypoScript الثابت إلى قالب الجذر الخاص بك:
   **القالب → تحرير → التضمينات → العناصر المتاحة → Wink Travel**.
</Steps>

## إضافة عنصر محتوى إلى صفحة

<Steps>
1. افتح صفحة في وحدة **الصفحة**.
2. انقر على زر **+** في منطقة المحتوى لإضافة عنصر جديد.
3. اختر أحد عناصر محتوى **Wink Travel**:
   - **Wink Content** — بطاقات الفنادق، الشبكات المرتبة، الخرائط
   - **Wink Lookup** — شريط بحث الوجهة
   - **Wink Search** — زر اختيار خط الرحلة
   - **Wink Account** — زر تسجيل الدخول
   - **Wink Itinerary** — زر حالة خط الرحلة
   - **Wink Cart** — زر عربة التسوق
4. بالنسبة لـ **Wink Content**، استخدم حقول FlexForm لاختيار **نوع التخطيط** وأدخل **معرف التخطيط**.
5. احفظ وانشر الصفحة.
</Steps>

## عناصر المحتوى المتاحة

| العنصر | الغرض |
|---|---|
| **Wink Content** | بطاقات الفنادق، الشبكات المرتبة، الخرائط، وأكثر |
| **Wink Lookup** | شريط بحث الوجهة |
| **Wink Search** | زر أيقونة يفتح منتقي خط الرحلة |
| **Wink Account** | زر تسجيل الدخول / حساب المستخدم |
| **Wink Itinerary** | زر خط الرحلة يعرض الرحلة الحالية |
| **Wink Cart** | زر ملخص عربة التسوق |
