تضيف **وحدة Wink لدروبال** مكونات مخزون السفر من Wink إلى موقع دروبال الخاص بك ككتل أصلية. يمكن للمحررين وضعها في أي مكان في التخطيط عبر نظام الكتل القياسي.

<LinkCard
  title="wink-travel/wink-drupal-module على GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="كود المصدر، دليل إعداد المطور، وتعليمات المساهمة."
/>

## المتطلبات

- دروبال 10 أو 11
- PHP 8.1+
- Composer

## التثبيت

<Steps>
1. التثبيت عبر Composer:
   ```bash
   composer require drupal/wink
   ```
2. تفعيل الوحدة:
   ```bash
   drush en wink -y
   ```
3. اذهب إلى **الإدارة → التكوين → خدمات الويب → إعدادات Wink** (`/admin/config/wink`).
4. أدخل **معرف العميل** و**السر السري للعميل**، اختر بيئتك، ثم اضغط **حفظ**.
   اذهب إلى [التطبيقات](/ar/portal/settings/applications) لاسترجاع بيانات الاعتماد الخاصة بك.
</Steps>

## إضافة كتلة إلى صفحة

<Steps>
1. اذهب إلى **الإدارة → الهيكل → تخطيط الكتل**.
2. اضغط على **وضع كتلة** في المنطقة التي تريد محتوى Wink فيها.
3. ابحث عن **Wink** للعثور على الكتل المتاحة:
   - **محتوى Wink** — يعرض بطاقة فندق، شبكة، أو خريطة
   - **بحث Wink** — شريط بحث الوجهات
   - **بحث Wink** — زر اختيار خط سير الرحلة
   - **حساب Wink** — زر تسجيل الدخول
   - **خط سير Wink** — زر حالة خط الرحلة
   - **عربة Wink** — زر عربة التسوق
4. بالنسبة لـ **محتوى Wink**، قم بتكوين **نوع التخطيط** و**معرف التخطيط** في إعدادات الكتلة.
5. احفظ إعدادات الكتلة.
</Steps>

## الكتل المتاحة

| الكتلة | الغرض |
|---|---|
| **محتوى Wink** | يعرض بطاقات الفنادق، الشبكات المرتبة، الخرائط، والمزيد |
| **بحث Wink** | شريط بحث للوجهات والفنادق |
| **بحث Wink** | زر أيقونة يفتح منتقي خط سير الرحلة |
| **حساب Wink** | زر تسجيل الدخول / حساب المستخدم |
| **خط سير Wink** | زر خط سير الرحلة يعرض الرحلة الحالية |
| **عربة Wink** | زر ملخص عربة التسوق |
