:::note
ميزة حقن الكود في سكوير سبيس متاحة في **خطة الأعمال وما فوق**. كتل HTML المخصصة متاحة في جميع الخطط المدفوعة.
:::

يدعم سكوير سبيس HTML وJavaScript المخصصين، مما يعني أنه يمكنك تضمين أي مكون ويب من Wink مباشرة في موقعك بدون إضافة. يوضح هذا الدليل كيفية تحميل سكريبتات Wink على مستوى الموقع ثم وضع المكونات في صفحات فردية.

## تحميل سكريبتات Wink على مستوى الموقع

إضافة السكريبتات إلى تذييل ورأس الموقع يعني أن Wink متاح في كل صفحة دون الحاجة لتكرار الإعداد.

<Steps>
1. في محرر سكوير سبيس، اذهب إلى **الإعدادات → متقدم → حقن الكود**.
2. في حقل **الرأس**، أضف ورقة الأنماط:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. في حقل **التذييل**، أضف السكريبت وموّلد التطبيق (استبدل `YOUR_CLIENT_ID` بمعرف العميل الخاص بك في Wink):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. اضغط **حفظ**.
</Steps>

:::tip
معرف العميل الخاص بك متاح في حساب Wink ضمن [التطبيقات](/ar/portal/settings/applications).
:::

## إضافة مكون إلى صفحة

بمجرد تحميل السكريبتات على مستوى الموقع، يمكنك وضع أي مكون من Wink في أي صفحة باستخدام **كتلة كود**.

<Steps>
1. افتح الصفحة التي تريد تعديلها.
2. اضغط **تعديل** وأضف كتلة **كود** جديدة في المكان الذي تريد ظهور المكون فيه.
3. الصق HTML الخاص بالمكون، مثل موّلد المحتوى:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. اضغط **تطبيق** ثم **حفظ**.
</Steps>

## المكونات المتاحة

| المكون | وسم HTML | الغرض |
|---|---|---|
| موّلد المحتوى | `<wink-content-loader>` | عرض بطاقات الفنادق، الشبكات، الخرائط |
| البحث | `<wink-lookup>` | شريط بحث عن الوجهات |
| زر البحث | `<wink-search-button>` | فتح منتقي خط الرحلة |
| زر الحساب | `<wink-account-button>` | تسجيل الدخول / قائمة المستخدم |
| زر خط الرحلة | `<wink-itinerary-button>` | عرض خط الرحلة الحالي |
| منتقي خط الرحلة | `<wink-itinerary-picker>` | نموذج خط الرحلة الكامل |
| عربة التسوق | `<wink-shopping-cart-button>` | زر ملخص العربة |

راجع [مكونات الويب](/ar/integrations/web-components) للمرجع الكامل للسمات لكل مكون.
