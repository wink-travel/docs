تتيح لك ميزة **حقن الشيفرة** المدمجة في جوست إضافة سكريبتات إلى `<head>` و`</body>` العام لمنشورك. تدعم جميع خطط جوست حقن الشيفرة، مما يسهل تضمين مكونات Wink في أي صفحة أو منشور.

## تحميل سكريبتات Wink على مستوى الموقع

<Steps>
1. في لوحة إدارة جوست، اذهب إلى **الإعدادات → حقن الشيفرة**.
2. في مربع **رأس الموقع**، أضف ورقة الأنماط:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. في مربع **تذييل الموقع**، أضف السكريبت ومحمل التطبيق (استبدل `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. اضغط **حفظ**.
</Steps>

:::tip
معرّف العميل الخاص بك متوفر في حساب Wink تحت [التطبيقات](/ar/portal/settings/applications).
:::

## إضافة مكون إلى منشور أو صفحة

يدعم محرر جوست **بطاقة HTML** تتيح لك إدراج HTML خام في أي مكان في المحتوى.

<Steps>
1. افتح المنشور أو الصفحة في محرر جوست.
2. اكتب `/` لفتح منتقي البطاقات واختر **HTML**.
3. الصق كود المكون، على سبيل المثال:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. اضغط خارج البطاقة للمعاينة، ثم انشر.
</Steps>

## حقن الشيفرة لكل صفحة

للمكون الذي يجب أن يظهر في صفحة واحدة فقط، يدعم جوست أيضًا حقن الشيفرة لكل صفحة:

<Steps>
1. افتح إعدادات المنشور أو الصفحة (رمز الترس في المحرر).
2. انتقل إلى **حقن الشيفرة**.
3. أضف وسم المكون في حقل **التذييل** (محمل التطبيق العام يجب أن يكون فقط في تذييل الموقع العام).
4. احفظ.
</Steps>

## المكونات المتاحة

| المكون | وسم HTML | الغرض |
|---|---|---|
| محمل المحتوى | `<wink-content-loader>` | عرض بطاقات الفنادق، الشبكات، الخرائط |
| البحث | `<wink-lookup>` | شريط بحث عن الوجهات |
| زر البحث | `<wink-search-button>` | فتح منتقي خط الرحلة |
| زر الحساب | `<wink-account-button>` | تسجيل الدخول / قائمة المستخدم |
| زر خط الرحلة | `<wink-itinerary-button>` | عرض خط الرحلة الحالي |
| منتقي خط الرحلة | `<wink-itinerary-picker>` | نموذج خط الرحلة الكامل |
| عربة التسوق | `<wink-shopping-cart-button>` | زر ملخص العربة |

راجع [مكونات الويب](/ar/integrations/web-components) للمرجع الكامل للسمات.
