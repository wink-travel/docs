يتكامل Wink مع مديري القنوات الرائدين، وأنظمة إدارة الممتلكات (PMS)، وأنظمة الحجز المركزية (CRS) للحفاظ على تزامن أسعارك وتوافر غرفك عبر جميع قنوات البيع الخاصة بك.

## كيف يعمل

تمكّن تكاملات مدير القنوات تدفقًا ثنائي الاتجاه للبيانات بين نظام إدارة الممتلكات الخاص بك وWink:

1. **دفع الأسعار** — يرسل مدير القنوات تحديثات الأسعار إلى Wink. عند تحديث التسعير في نظام إدارة الممتلكات الخاص بك، تنعكس هذه التغييرات تلقائيًا على Wink.
2. **مزامنة التوافر** — يرسل مدير القنوات تحديثات التوافر إلى Wink. عند حجز أو تحرير الغرف في نظام إدارة الممتلكات الخاص بك، يتم تحديث المخزون في Wink في الوقت الفعلي.
3. **إشعارات الحجز** — عند إجراء حجز على Wink، يتم إعلام مدير القنوات ليظهر الحجز في نظام إدارة الممتلكات الخاص بك.

## مديري القنوات المدعومين

يتكامل Wink حاليًا مع الأنظمة التالية:

| مدير القنوات | الوصف |
|---|---|
| **Channex** | منصة التوزيع وإدارة القنوات العالمية. |
| **TravelClick** | حلول الفنادق وإدارة القنوات (Amadeus). |
| **Omnibees** | التوزيع وإدارة الإيرادات للفنادق. |
| **D-Edge** | تكنولوجيا الضيافة وحلول الحجز المركزي. |
| **Allotz** | تخصيص المخزون وإدارة القنوات. |
| **Ezee / Yanolja** | نظام إدارة ممتلكات سحابي وإدارة القنوات يركز على السوق الآسيوي. |
| **RateGain** | تكنولوجيا استخبارات الأسعار والتوزيع. |
| **RateTiger** | توزيع الأسعار والتوافر في الوقت الحقيقي. |
| **CloudBeds** | نظام إدارة ممتلكات سحابي، مدير قنوات، ومحرك حجز. |
| **SiteMinder** | مدير قنوات الفنادق الرائد ومنصة التوزيع. |
| **Exely** | مزامنة الأسعار والتوافر للفنادق. |

:::note
إذا لم يكن مدير القنوات الخاص بك مدرجًا، تواصل مع فريق Wink. يمكن إضافة تكاملات جديدة بناءً على معايير بروتوكول OTA (Open Travel Alliance) التي يدعمها Wink.
:::

## للمطورين

إذا كنت تبني تكاملًا مع مدير قنوات أو نظام إدارة ممتلكات مع Wink، استخدم نقاط نهاية API الخاصة بمدير القنوات:

- **تحديثات الأسعار** — دفع تغييرات الأسعار إلى Wink.
- **تحديثات التوافر** — دفع تغييرات التوافر إلى Wink.
- **استرجاع الحجوزات** — سحب الحجوزات من Wink إلى نظامك.

يتبع API مدير القنوات معايير بروتوكول OTA (SOAP/XML) لضمان التوافق مع أنظمة الضيافة القائمة.

## إشعارات Webhook

يمكنك الاشتراك في أحداث Webhook الخاصة بمدير القنوات لتلقي الإشعارات في الوقت الحقيقي:

- `channel-manager.update.rate` — تم استلام تحديث السعر.
- `channel-manager.update.availability` — تم استلام تحديث التوافر.
- `channel-manager.update` — تحديث عام لمدير القنوات.

راجع [كتالوج أحداث Webhook](/ar/developers/webhook-events#channel-manager-events) للتفاصيل.
