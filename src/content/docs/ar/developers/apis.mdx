## واجهة برمجة تطبيقات الدفع

نقاط نهاية الدفع مخصصة للمطورين الذين يرغبون في شراء مخزون السفر. يمكن القيام بذلك عبر واجهة برمجة التطبيقات كوكيل سفر مسجل أو باستخدام واجهتنا البرمجية جنبًا إلى جنب مع أداة الدفع المتوافقة مع PCI لجميع الكيانات الأخرى.
واجهة برمجة تطبيقات الدفع تعرض نقاط نهاية للتابعين والفنادق لتتبع الحجوزات، التحليلات وتوفر الأموال. يمكنهم أيضًا اختيار سحب الأموال المتاحة إلى حسابهم المصرفي.

</LinkButton>
