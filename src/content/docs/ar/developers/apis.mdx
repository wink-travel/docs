جميع واجهات برمجة تطبيقات REST لدينا لا تعتمد على لغة محددة، وهي متوافقة مع معايير Open API. ابدأ الاختبار دون الحاجة لكتابة سطر واحد من التعليمات البرمجية.

إذا لم تقم بإنشاء حساب Wink الخاص بك بالفعل،[ابدأ هنا](/user/register).

## واجهة برمجة تطبيقات الدفع

نقاط الدفع متاحة للمطورين الراغبين بشراء مخزون السفر. يمكن القيام بذلك عبر واجهة برمجة التطبيقات (API) كوكيل سفر مسجل، أو باستخدام واجهة برمجة التطبيقات الخاصة بنا مع أداة الدفع المتوافقة مع معايير PCI لجميع الجهات الأخرى. تتيح واجهة برمجة تطبيقات الدفع نقاط دفع للشركاء والفنادق لتتبع الحجوزات والتحليلات وتوافر الأموال. كما يمكنهم اختيار سحب الأموال المتاحة إلى حساباتهم المصرفية.

<LinkButton href="https://docs.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  يتعلم أكثر
</LinkButton>
