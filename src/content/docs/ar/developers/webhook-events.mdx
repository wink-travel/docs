تدرج هذه الصفحة كل حدث webhook يمكنك الاشتراك فيه على Wink. للحصول على تعليمات الإعداد، راجع [دليل تكامل Webhook](/ar/guides/webhook-integration) أو [صفحة إدارة Webhooks](/ar/my-account/webhooks).

## كيفية الاشتراك

1. انتقل إلى **التطبيقات > Webhooks** في بوابتك.
2. أنشئ webhook وحدد الأحداث التي تريد استلامها.
3. ترسل Wink طلب HTTP POST إلى نقطة النهاية الخاصة بك في كل مرة يحدث فيها الحدث.

للحصول على خطوات الإعداد الكاملة وأفضل الممارسات، راجع [تكامل Webhook](/ar/guides/webhook-integration).

## أحداث الحساب

الأحداث المتعلقة بدورة حياة حساب TripPay والإعدادات.

| الحدث | المفتاح | الوصف |
|---|---|---|
| تم إنشاء الحساب | `account.create` | تم إنشاء حساب جديد. |
| الحساب قيد المراجعة | `account.under_review` | الحساب قيد المراجعة. |
| تم تحديث الحساب | `account.updated` | تم تغيير تفاصيل الحساب. |
| تم تحديث حالة الحساب | `account.status.updated` | تغيرت حالة الحساب (مثلًا، تمت الموافقة، تم التعليق). |
| تم تحديث عنوان الحساب | `account.address.updated` | تم تعديل عنوان الحساب. |
| تم تحديث تفاصيل الحساب | `account.details.updated` | تغيرت حقول تفاصيل الحساب. |
| تم تحديث معلومات الحساب | `account.information.update` | تم تحديث المعلومات العامة للحساب. |
| تم تحديث مديري الحساب | `account.managers.update` | تغيرت قائمة مديري الحساب. |
| تم تحديث صورة الملف الشخصي | `account.profile_picture.updated` | تم تغيير صورة الملف الشخصي للحساب. |
| تم تحديث الوكيل | `account.agent.updated` | تم تعديل معلومات الوكيل في الحساب. |
| تم تحديث اسم URL | `account.url-name.update` | تغير عنوان URL المخصص للحساب. |
| تم تحديث المعرف الفريد | `account.unique-id.update` | تم تعديل المعرف الفريد للحساب. |

### مهام الحساب

| الحدث | المفتاح | الوصف |
|---|---|---|
| تم إنشاء مهمة | `account.task.create` | تمت إضافة مهمة جديدة إلى الحساب. |
| تم تحديث مهمة | `account.task.update` | تم تعديل مهمة في الحساب. |
| تم إزالة مهمة | `account.task.remove` | تمت إزالة مهمة من الحساب. |
| تم إكمال مهمة | `account.task.complete` | تم إكمال مهمة في الحساب. |
| تم إكمال مهمة التحقق من رقم التعريف الضريبي | `account.task.tax_id.complete` | تم إكمال مهمة التحقق من رقم التعريف الضريبي. |

### مستحوذ الحساب

| الحدث | المفتاح | الوصف |
|---|---|---|
| تم إنشاء مستحوذ | `account.acquirer.created` | تمت إضافة مستحوذ جديد إلى الحساب. |
| تم تحديث مستحوذ | `account.acquirer.updated` | تم تعديل مستحوذ في الحساب. |
| تم إزالة مستحوذ | `account.acquirer.removed` | تمت إزالة مستحوذ من الحساب. |

## أحداث عقد الحجز

الأحداث التي تتتبع دورة حياة عقد الحجز عبر TripPay.

| الحدث | المفتاح | الوصف |
|---|---|---|
| طلب تهيئة | `booking.contract.initialize.request` | تم طلب تهيئة عقد الحجز. |
| اكتمال التهيئة | `booking.contract.initialize.complete` | تم الانتهاء من تهيئة عقد الحجز. |
| تم إنشاء العقد | `booking.contract.created` | تم إنشاء عقد حجز جديد. |
| معالجة العقد | `booking.contract.processing` | يتم معالجة دفع عقد الحجز. |
| فشل العقد | `booking.contract.failed` | فشل دفع عقد الحجز. |
| تم إلغاء العقد | `booking.contract.cancelled` | تم إلغاء عقد الحجز. |
| استرداد جزئي | `booking.contract.refund.partial` | تم إصدار استرداد جزئي على العقد. |
| استرداد كامل | `booking.contract.refund.full` | تم إصدار استرداد كامل على العقد. |
| طلب استرداد | `booking.contract.refund.request` | تم تقديم طلب استرداد. |
| خطأ في الاسترداد | `booking.contract.refund.error` | حدث خطأ أثناء معالجة الاسترداد. |
| تم إنشاء دفعة | `booking.contract.payout.create` | تم إنشاء دفعة للعقد. |
| اكتمال الدفعة | `booking.contract.payout.complete` | تم إكمال الدفعة بنجاح. |

## أحداث الكيان الإداري

الأحداث الخاصة بالكيانات الإدارية (العقارات، الشركات التابعة، الشركات) وإدارتها.

### الاشتراك

| الحدث | المفتاح | الوصف |
|---|---|---|
| تم توفير الاشتراك | `managing_entity.subscription.provisioned` | تم توفير اشتراك جديد. |
| تم تغيير الاشتراك | `managing_entity.subscription.changed` | تم تغيير خطة الاشتراك. |
| تم تغيير فاتورة الاشتراك | `managing_entity.subscription.invoice.changed` | تم تحديث فاتورة الاشتراك. |

### الحالة والإدارة

| الحدث | المفتاح | الوصف |
|---|---|---|
| تم تحديث الحالة | `managing_entity.status.updated` | تغيرت حالة الكيان. |
| تم حذف الكيان | `managing_entity.deleted` | تم حذف كيان إداري. |
| تم تحديث الوكيل | `managing_entity.agent.updated` | تم تعديل وكيل الكيان. |
| إشعار بريد إلكتروني للوكيل | `managing_entity.agent.email.notification` | تم إرسال إشعار عبر البريد الإلكتروني إلى الوكيل. |
| إشعار بريد إلكتروني للمدير | `managing_entity.manager.email.notification` | تم إرسال إشعار عبر البريد الإلكتروني إلى مدير. |

### دورة حياة المدير

| الحدث | المفتاح | الوصف |
|---|---|---|
| تم إضافة مدير | `managing_entity.manager.added` | تم إضافة مدير جديد. |
| تم دعوة المدير | `managing_entity.manager.invited` | تم إرسال دعوة لمدير. |
| تم إزالة المدير | `managing_entity.manager.removed` | تم إزالة مدير. |
| تم قبول المدير | `managing_entity.manager.accepted` | تم قبول دعوة المدير. |
| تم رفض المدير | `managing_entity.manager.rejected` | تم رفض دعوة المدير. |
| تم تعيين المالك | `managing_entity.manager.owner.set` | تم نقل ملكية الكيان. |

### معلومات الكيان

|---|---|---|
| تم تحديث المعلومات | `managing_entity.information.updated` | تم تحديث معلومات الكيان. |
| تم تحديث العنوان | `managing_entity.city.updated` | تم تغيير عنوان/مدينة الكيان. |
| تم تحديث ممثل الحساب | `managing_entity.account_representative.updated` | تم تغيير ممثل الحساب. |
| تم تحديث صورة الملف الشخصي | `managing_entity.profile_picture.updated` | تم تحديث صورة الملف الشخصي للكيان. |
| تم تحديث الشعار | `managing_entity.logo.updated` | تم تحديث شعار الكيان. |
| تم تحديث اسم URL | `managing_entity.url_name.updated` | تغير عنوان URL المخصص للكيان. |
