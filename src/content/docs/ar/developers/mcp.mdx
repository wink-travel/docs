## نظرة عامة
خادم MCP المستضاف من Wink متاح على:

```
https://docs.mcp.wink.travel/mcp
```

وجه عملاء الذكاء الاصطناعي المتوافقين (VS Code، Cursor، وغيرهم الذين ينفذون نقل MCP عبر HTTP) إلى هذه النقطة النهائية للسماح لهم بـ:
- تصفح وثائق المنتج والمنصة (`wink://docs/...`)
- اكتشاف SDKs والتعمق في فئات / نماذج API (`wink://sdk/...` و `wink://sdk/.../docs`)
- إجراء بحث دلالي / بالكلمات المفتاحية خفيف الوزن عبر ملخصات الوثائق وSDKs

## النقل
استخدم نقطة النهاية الأساسية القابلة للبث عبر HTTP:

```
https://docs.mcp.wink.travel/mcp
```

إذا كان عميلك يدعم SSE فقط، استخدم:

```
https://docs.mcp.wink.travel/sse
```

### Cursor
وثائق MCP لـ Cursor: https://cursor.com/docs/context/mcp
الإعداد: أضف الخادم البعيد عبر إعدادات MCP.
```json
{
  "mcpServers": {
    "wink-docs": {
      "url": "https://docs.mcp.wink.travel/mcp",
    }
  }
}
```

### VS Code
الوثائق الرسمية: https://code.visualstudio.com/docs/copilot/customization/mcp-servers
تكوين بعيد بسيط (ملف `mcp.json` للمستخدم أو مساحة العمل):
```json
{
  "servers": {
    "wink-docs": {
      "type": "http",
      "url": "https://docs.mcp.wink.travel/mcp"
    }
  }
}
```

نصائح:
- استخدم لوحة الأوامر: "MCP: List Servers" لتأكيد الاتصال وتصفح الأدوات.
- يتيح لك محدد الأدوات تمكين/تعطيل `sdk_search` و `docs_search`.
- أضف السياق: عرض الدردشة → إضافة سياق → موارد MCP لإرفاق صفحات الوثائق.

## عناوين URI الأساسية للموارد

| URI | الوصف |
|-----|-------------|
| `wink://docs` | فهرس الوثائق (مجمعة حسب الفئة) |
| `wink://docs/{path}` | مقالة وثائق محددة (Markdown / MDX) |
| `wink://sdk` | نظرة عامة على SDK (كل SDK + عدد الوثائق) |
| `wink://sdk/{sdkPath}` | ملف README الخاص بـ SDK / مجلد وثائق الوحدة / وثيقة وحدة محددة |

### وثائق الوحدة
فتح مجلد وثائق SDK (مثلاً `wink://sdk/python/wink-sdk-booking/docs`) يعرض مجموعات:
- **فئات API** (عدد الطرق + أسماء طرق نموذجية)
- **النماذج والأنواع** (هياكل البيانات المستخدمة من قبل تلك APIs)

## أدوات البحث (مرجع سريع)
| الأداة | الغرض | أمثلة استعلام نموذجية |
|------|---------|-----------------------|
| `sdk_search` | العثور على SDK / فئة API لاستخدامها | `إلغاء الحجز`، `قائمة العقارات`، `مخزون الشركاء` |
| `docs_search` | العثور على الوثائق المفاهيمية / الأدلة | `المصادقة`، `webhooks`، `ووردبريس` |

اترك `query` فارغًا لتلقي اقتراحات. كل نتيجة تتضمن URI MCP يمكنك فتحه.

## نمط الاستخدام النموذجي
1. شغّل `sdk_search` مع نيتك (مثلاً *"قائمة العقارات"*).
2. افتح ملف README الخاص بـ SDK أو `docs_uri` الخاص به لفحص APIs / النماذج.
3. استخدم `docs_search` للمفاهيم الداعمة (المصادقة، webhooks).

## ملاحظات
- يمكن للوكيل فتح كل URI معاد للحصول على المحتوى الكامل.
- تظهر الاقتراحات تلقائيًا عندما تكون الاستعلامات فارغة.
