تستخدم Wink تحديد الموقع الجغرافي بشكل واسع لتشغيل البحث بناءً على الموقع، واكتشاف العملة واللغة تلقائيًا، والتصفية الجغرافية للمخزون.

## اكتشاف GeoIP

عندما يزور المستخدم المنصة، تقوم Wink تلقائيًا باكتشاف موقعه بناءً على عنوان IP الخاص به. هذا يمكّن من:

- **اختيار العملة تلقائيًا** — عرض الأسعار بعملة المستخدم المحلية.
- **تفضيل اللغة** — اقتراح المحتوى باللغة التي يحتمل أن يستخدمها المستخدم.
- **النتائج القريبة** — إعطاء أولوية للمخزون القريب من موقع المستخدم.

## البحث المكاني

تدعم Wink البحث الجغرافي باستخدام أربعة أنواع من الاستعلامات المكانية. هذه تدعم البحث المعتمد على الخريطة، والبحث بنطاق نصف القطر، والتصفية بناءً على المنطقة عبر المنصة.

### نقطة

إحداثية جغرافية واحدة (خط العرض، خط الطول). تُستخدم لتحديد موقع معين — على سبيل المثال، عنوان عقار أو موقع المستخدم الحالي.

```json
{
  "point": {
    "x": -73.9857,
    "y": 40.7484
  }
}
```

### دائرة

نقطة مركزية مع نصف قطر. تُستخدم لاستعلامات "البحث بالقرب مني" أو استعلامات نصف القطر — على سبيل المثال، "الفنادق ضمن 10 كم من تايمز سكوير."

```json
{
  "point": {
    "x": -73.9857,
    "y": 40.7484
  },
  "radius": 10000
}
```

### مستطيل

مربع حدودي يُحدد بواسطة الزوايا الجنوبية الغربية والشمالية الشرقية. يُستخدم للبحث المعتمد على الخريطة — عندما يقوم المستخدم بتحريك وتكبير الخريطة، تحدد المنطقة المرئية استعلام مستطيل.

```json
{
  "southWest": {
    "x": -74.0060,
    "y": 40.7128
  },
  "northEast": {
    "x": -73.9350,
    "y": 40.7831
  }
}
```

### مضلع

شكل مخصص يُحدد بواسطة سلسلة من النقاط. يُستخدم للبحث ضمن حدود جغرافية غير منتظمة — على سبيل المثال، حي، منطقة مدينة، أو منطقة مرسومة مخصصة على الخريطة.

```json
{
  "polygon": {
    "points": [
      { "x": -74.0060, "y": 40.7128 },
      { "x": -73.9350, "y": 40.7128 },
      { "x": -73.9350, "y": 40.7831 },
      { "x": -74.0060, "y": 40.7831 },
      { "x": -74.0060, "y": 40.7128 }
    ]
  }
}
```

:::note
يجب أن تكون المضلعات مغلقة — يجب أن تتطابق النقطة الأخيرة مع النقطة الأولى.
:::

## حالات الاستخدام

| حالة الاستخدام | نوع الشكل | مثال |
|---|---|---|
| البحث بالقرب مني | دائرة | الفنادق ضمن 5 كم من موقعي |
| البحث في إطار عرض الخريطة | مستطيل | العقارات الظاهرة في عرض الخريطة الحالي |
| البحث في الحي | مضلع | الفنادق في منطقة مدينة محددة |
| موقع العقار | نقطة | الموقع الدقيق لقائمة العقار |

## للمطورين

يتم قبول معلمات الاستعلام المكاني من خلال نقاط نهاية Lookup API وInventory API. مرر الشكل المناسب GeoJSON في طلب البحث الخاص بك لتصفية النتائج حسب الموقع.

راجع [توثيق APIs](/ar/developers/apis) للحصول على تفاصيل خاصة بكل نقطة نهاية.
