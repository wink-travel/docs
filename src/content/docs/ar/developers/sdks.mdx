سرّع وقت الوصول إلى السوق باستخدام إحدى مجموعات تطوير البرمجيات الرسمية المدعومة لدينا.
إذا لم ندعم لغتك المفضلة حاليًا، انتقل إلى واجهة برمجة التطبيقات التي تريد استخدامها وقم بتنزيل مواصفة OpenAPI التي نوفرها لكل واجهة برمجة تطبيقات في أعلى الصفحة. يمكنك إنشاء مجموعة تطوير البرمجيات الخاصة بك إما عن طريق تفرع أحد مستودعات SDK العامة الخاصة بنا أو باستخدام أحد مولدات OpenAPI.

## جافا

تم تجميع هذه المكتبة لـ Java 24. وهي مهيأة لاستخدام WebClient التفاعلي ومحسّنة للاستخدام مع إطار عمل Spring. ومع ذلك، فإن Spring ليست مطلبًا لكنها توفر لك دعم OAuth2 مباشرة.
