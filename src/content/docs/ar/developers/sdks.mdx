سرّع وقت طرح منتجك في السوق باستخدام إحدى مجموعات تطوير البرامج المعتمدة رسميًا لدينا.
إذا لم نكن ندعم لغتك المفضلة حاليًا، فانتقل إلى واجهة برمجة التطبيقات التي ترغب في استخدامها، وحمّل مواصفات OpenAPI المتاحة لكل واجهة برمجة تطبيقات في أعلى الصفحة. يمكنك إنشاء حزمة تطوير البرامج الخاصة بك إما عن طريق نسخ أحد مستودعات حزم تطوير البرامج العامة لدينا أو باستخدام أحد مولدات OpenAPI.

## جافا

جُمعت هذه المكتبة لإصدار Java 24. وهي مُهيأة لاستخدام WebClient التفاعلي، ومُحسّنة للاستخدام مع إطار عمل Spring. مع ذلك، لا يُعد Spring شرطًا أساسيًا، ولكنه يوفر دعم OAuth2 جاهزًا للاستخدام.
