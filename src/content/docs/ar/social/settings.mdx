# الإعدادات

عند البدء مع Social، سنسألك كيف تريد منا إنشاء المحتوى الخاص بك. هذه الخيارات متاحة لك في جميع الأوقات حتى تتمكن من تغييرها بسهولة لاحقًا.

انتقل إلى `Preferences` من الشريط الجانبي لإدارة إعداداتك.

يحتوي قسم التفضيلات على إعدادات إنشاء المحتوى الأساسية الخاصة بك:

1. **التخصيص المفضل** اختر التخصيص الذي تريد استخدامه عند إنشاء المحتوى. تتحكم التخصيصات في ألوان السمة، وخطة الرحلة الافتراضية، والعملة، واللغة، والمزيد. يمكنك إنشاء تخصيصات تحت حسابك على [my.wink.travel](https://my.wink.travel).
2. **الموافقة التلقائية** عند التمكين، يتم نشر المنشورات التي تم إنشاؤها تلقائيًا دون الحاجة إلى موافقتك. قم بتمكين هذا بمجرد أن تكون واثقًا من جودة المحتوى الذي ننتجه لك.
3. **اسم الموافق** اسم الشخص أو الفريق الذي سيتلقى رسائل طلب الموافقة.
4. **البريد الإلكتروني للموافق** عنوان البريد الإلكتروني الذي سترسل إليه طلبات الموافقة على المنشورات القادمة.

## منشئو المحتوى

يأتي Social مع العديد من منشئي المحتوى للاختيار من بينهم. يركز كل منشئ على طريقة محددة لإنشاء محتوى فريد لممتلكاتك باستخدام الذكاء الاصطناعي. لكل منشئ محتوى خاصية **الوزن** — كلما زاد الوزن زادت فرصة اختيار هذا المنشئ عند إنشاء المنشورات.

فيما يلي جميع منشئي المحتوى المتاحين:

| المنشئ | الوصف |
| -- | -- |
| **الوجهة** | يستخدم موقعك لإنشاء نص مفيد عن المنطقة |
| **نوع الغرفة** | ينشئ منشورًا قابلاً للحجز يعرض أحد أنواع غرفك مع نص مولد بالذكاء الاصطناعي وصور نوع الغرفة الخاصة بك |
| **النشاط** | ينشئ منشورًا قابلاً للحجز يعرض أحد أنشطتك |
| **الإضافة** | ينشئ منشورًا قابلاً للحجز يعرض أحد الإضافات الخاصة بك |
| **الجذب السياحي** | ينشئ منشورًا قابلاً للحجز يعرض أحد معالم الجذب الخاصة بك |
| **المكان** | ينشئ منشورًا قابلاً للحجز يعرض أحد أماكنك |
| **غرفة الاجتماعات** | ينشئ منشورًا قابلاً للحجز يعرض إحدى غرف الاجتماعات الخاصة بك |
| **المطعم** | ينشئ منشورًا قابلاً للحجز يعرض أحد مطاعمك |
| **السبا** | ينشئ منشورًا قابلاً للحجز يعرض أحد مراكز السبا الخاصة بك |
| **الممتلكات** | ينشئ منشورًا قابلاً للحجز يعرض أفضل غرفة بسعر مناسب مع نص مولد بالذكاء الاصطناعي وصور الممتلكات الخاصة بك |
| **مشاعر المستخدم** | يجمع تقييمات الضيوف من مختلف أنحاء الويب في منشور قائم على المشاعر يبرز ما يحبه الضيوف في ممتلكاتك |
| **المرح** | ينشئ منشورات خفيفة الظل وترفيهية للحفاظ على تفاعل متابعيك |

:::tip[نصيحة احترافية]
تأكد من أن لديك نوع المخزون المطلوب لمنشئي المحتوى الذين اخترتهم. على سبيل المثال، إذا اخترت "المطعم"، يجب أن يكون لديك على الأقل مطعم واحد مدرج في ممتلكاتك.
:::

## نبرة الصوت

تحدد نبرة الصوت أسلوب اللغة التي يستخدمها الذكاء الاصطناعي عند إنشاء المحتوى. اختر النبرات التي تتناسب مع شخصية علامتك التجارية. كما هو الحال مع منشئي المحتوى، لكل نبرة خاصية **الوزن** للتحكم في مدى تكرار استخدامها.

فيما يلي النبرات المتاحة:

| النبرة | الأسلوب |
| -- | -- |
| **محايد** | هادئ، موثوق، واقعي. بدون مبالغة أو نكات — مجرد معرفة موثوقة ومنظمة جيدًا |
| **ودود** | دافئ، ودود، طاقة مبتسمة عبر النص مع اختصارات غير رسمية ووجوه تعبيرية أحيانًا |
| **فاخر** | راقٍ، متقن. كلمات مثل "رائع" و"مصمم خصيصًا" تجعل القارئ يشعر بالتميز |
| **محب للطعام** | أوصاف حسية تثير الشهية. لغة تصويرية للطعام تجعل القارئ يشعر بالجوع |
| **مغامر** | جريء، شجاع، طاقة مستكشف. أوصاف حسية حية تجعل كل شيء يبدو ملحميًا |
| **عائلي** | حماسة صحية مناسبة للأطفال. طاقة دافئة أبوية حول تجارب آمنة وممتعة |
| **رومانسي** | شعري، حالم، يثير القلب. غروب الشمس، لغة ناعمة، وحب |
| **فكاهي** | ساخر، نكات سريعة، ألعاب كلمات ذكية. يقرأ كعرض كوميدي |
| **بيئي** | جاد وصادق. يحتفي بالبدائل الخضراء والخيارات المستدامة |
| **محلي مطلع** | إشارات محلية محددة للغاية، يحذر من الفخاخ السياحية، يستخدم العامية المحلية |
| **شامل** | مرحب، احتفالي، ومؤكد. يجعل الجميع يشعرون بأنهم مرئيون ومقدرون |
| **غامض** | أثيري، روحي، حقائق كونية. دعوات لطيفة للنظر إلى الداخل |
| **مستقبلي** | أنيق، متقدم التفكير. يحب مصطلحات مثل "تحول النموذج" ورؤى مستقبلية متفائلة |
| **اقتصادي** | عملي، مبدع، مقتصد بفخر. يحتفي بالعثور على الصفقات والتجارب المجانية |

## أنواع المنشورات

تحدد أنواع المنشورات الهدف وبنية المحتوى الذي يتم إنشاؤه. لكل نوع منشور خاصية **الوزن** للتحكم في مدى تكرار استخدامه.

| نوع المنشور | الوصف |
| -- | -- |
| **تجاري** | واضح، موجز، يركز على التحويل. يدفع الحجوزات والإجراءات المباشرة مع الحقائق والأسعار والخطوات التالية |
| **توعوي** | محتوى بناء للعلامة التجارية مصمم لتعريف ممتلكاتك وزيادة الظهور بين الضيوف المحتملين |
| **عاطفي** | محتوى يثير المشاعر يخلق ارتباطات إيجابية ويبني علاقة عاطفية مع جمهورك |

*تستخدم معظم الممتلكات لغة واحدة وتفترض اللغة المحلية الافتراضية.*
