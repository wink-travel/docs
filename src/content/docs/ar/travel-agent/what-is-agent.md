حلنا للوكلاء المسجلين في مجال السفر هو [Wink Agent](https://agent.wink.travel). إنه بوابة خدمة ذاتية يمكن للوكلاء استخدامها للبحث والحجز نيابةً عن عملائهم وكسب عمولة لأنفسهم على كل حجز.

يمكن للوكلاء الأكثر تقدمًا اختيار التكامل على مستوى API لأغراض معالجة الدفع والمدفوعات خارج منصة Wink.

يمتلك وكلاء السفر مجموعة فرعية من نفس ميزات البوابة الموجودة في [Wink Studio](/ar/studio/what-is-studio) لذلك لن نغطيها مرة أخرى هنا.

## البحث

اذهب إلى [Search](/ar/studio/search) لتتعلم كيفية البحث عن الفنادق وحجزها.

وإلا، تابع إلى القسم التالي.

## API

يمكن للمطورين الذين يرغبون في إدارة `Travel Agent` التوجه إلى [Developers > API > Travel Agent](/ar/developers/apis/#travel-agent-api).
