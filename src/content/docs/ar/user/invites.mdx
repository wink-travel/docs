يمكن لمستخدمي Wink اختيار إدارة أحد حساباتهم العقارية أو حسابات الشركاء أو وكلاء السفر. عندها، ستتلقى بريدًا إلكترونيًا يتضمن طلب دعوة.

import { Steps } from '@astrojs/starlight/components';

لرؤية دعوات Wink الحالية لديك، اتبع الخطوات التالية:

<Steps>
  1. يجب عليك تسجيل الدخول إلى أحد مواقعنا الإلكترونية.
  2. انقر على أيقونة ملفك الشخصي في الزاوية اليمنى العليا وستظهر قائمة.
  3. انقر على `Invites` وصلة.
  4. تفاعل مع دعواتك عن طريق أي من الطرق التالية: `Accepting` أو `Rejecting` هم.
</Steps>
