يمكن لمستخدم Wink أن يختار السماح لك بإدارة أحد حسابات ممتلكاته أو حسابات الشركات التابعة أو وكلاء السفر الخاصة به. عند حدوث ذلك، ستتلقى بريدًا إلكترونيًا يحتوي على طلب دعوة.

import { Steps } from '@astrojs/starlight/components';

لمعرفة دعوات Wink الحالية الخاصة بك، اتبع هذه الخطوات:

<Steps>
1. يجب أن تكون مسجلاً الدخول إلى أحد مواقعنا.
2. انقر على أيقونة ملفك الشخصي في الزاوية العلوية اليمنى وسيظهر قائمة.
3. انقر على رابط `الدعوات`.
4. تفاعل مع دعواتك إما عن طريق `قبولها` أو `رفضها`.
</Steps>
