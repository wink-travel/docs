Občas vás chceme kontaktovat. Když k tomu dojde, ikona zvonku v pravém horním rohu se změní.  
Pro přečtení zprávy, kterou jsme vám poslali:

<Steps>
1. Klikněte na `Účet > Upozornění` v hlavním navigačním panelu.
2. Uvidíte nová i starší upozornění.
3. Po přečtení upozornění zvolte `Označit jako přečtené` nebo `Smazat`.
</Steps>

## API

Vývojáři, kteří chtějí spravovat `Affiliates`, mohou přejít na [Developers > API > Affiliate](/cs/developers/apis/#affiliate-api).
