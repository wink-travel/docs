# Nastavení

Když začnete používat Social, zeptáme se vás, jak chcete, abychom vytvářeli váš obsah. Tyto možnosti máte k dispozici kdykoli, takže je můžete snadno později změnit.

Přejděte do `Preferences` v postranním panelu pro správu nastavení.

Sekce preferences obsahuje vaše základní nastavení generování obsahu:

1. **Preferred customization** Vyberte, kterou přizpůsobení chcete použít při generování obsahu. Přizpůsobení ovlivňují barvy tématu, výchozí itinerář, měnu, jazyk a další. Přizpůsobení vytváříte ve svém účtu na [my.wink.travel](https://my.wink.travel).
2. **Auto-approve** Pokud je povoleno, generované příspěvky jsou automaticky publikovány bez nutnosti vašeho schválení. Aktivujte to, až budete mít jistotu v kvalitě obsahu, který pro vás vytváříme.
3. **Approver name** Jméno osoby nebo týmu, který bude dostávat e-maily se žádostmi o schválení.
4. **Approver e-mail** E-mailová adresa, kam budou zasílány žádosti o schválení nadcházejících příspěvků.

## Tvůrci obsahu

<figure>
<CldImage
  src="academy/social_content_creators"

  <figcaption>*Obr. 2 - Tvůrci obsahu*</figcaption>

Social nabízí mnoho tvůrců obsahu, ze kterých si můžete vybrat. Každý tvůrce se zaměřuje na specifický způsob vytváření unikátního obsahu pro vaši nemovitost pomocí AI. Každý tvůrce má atribut **Weight** — vyšší hodnota zvyšuje pravděpodobnost, že bude vybrán při generování příspěvků.

Zde jsou všichni dostupní tvůrci obsahu:

| Tvůrce | Popis |
| -- | -- |
| **Destination** | Využívá vaši lokalitu k vytvoření užitečného textu o okolí |
| **Room type** | Vytváří rezervovatelný příspěvek s jedním z vašich typů pokojů s AI generovaným textem a obrázky pokojů |
| **Activity** | Vytváří rezervovatelný příspěvek s jednou z vašich aktivit |
| **Add-On** | Vytváří rezervovatelný příspěvek s jedním z vašich doplňkových služeb |
| **Attraction** | Vytváří rezervovatelný příspěvek s jednou z vašich atrakcí |
| **Place** | Vytváří rezervovatelný příspěvek s jedním z vašich míst |
| **Meeting room** | Vytváří rezervovatelný příspěvek s jednou z vašich zasedacích místností |
| **Restaurant** | Vytváří rezervovatelný příspěvek s jednou z vašich restaurací |
| **Spa** | Vytváří rezervovatelný příspěvek s jedním z vašich spa |
| **Property** | Vytváří rezervovatelný příspěvek s vaším nejvýhodnějším pokojem s AI generovaným textem a obrázky nemovitosti |
| **User sentiment** | Shromažďuje recenze hostů z internetu do příspěvku založeného na sentimentu, který zdůrazňuje, co hosté na vaší nemovitosti milují |
| **Funny** | Vytváří lehké, zábavné příspěvky, které udržují váš feed zajímavý |

:::tip[Pro tip]
Ujistěte se, že máte typ inventáře potřebný pro vybrané tvůrce obsahu. Například pokud vyberete "Restaurant", musíte mít na své nemovitosti alespoň jednu restauraci.
:::

## Tón hlasu

Tón hlasu určuje jazykový styl, který AI používá při generování obsahu. Vyberte tóny, které odpovídají osobnosti vaší značky. Stejně jako u tvůrců obsahu má každý tón atribut **Weight**, který určuje, jak často se používá.

Zde jsou dostupné tóny:

| Tón | Styl |
| -- | -- |
| **Neutral** | Klidný, autoritativní, faktický. Žádný přehnaný nadšení, žádné vtipy — jen spolehlivé, dobře strukturované informace |
| **Friendly** | Teplý, přístupný, s úsměvem v textu, s neformálními zkratkami a občasnými emoji |
| **Luxury** | Sofistikovaný, vytříbený. Slova jako "exkluzivní" a "na míru", která čtenáře činí výjimečnými |
| **Foodie** | Smyslné, chutné popisy. Jazyk, který vyvolává chuť k jídlu |
| **Adventurous** | Odvážný, neohrožený, průzkumnický. Živé smyslové popisy, které vše znějí epicky |
| **Family** | Rodinný, přátelský k dětem. Teplá rodičovská energie o bezpečných a zábavných zážitcích |
| **Romantic** | Poetický, snivý, srdce rozechvívající. Západy slunce, jemný jazyk a láska |
| **Humorous** | Sarkastický, rychlé hlášky, chytré slovní hříčky. Čte se jako stand-up vystoupení |
| **Eco** | Upřímný a spravedlivý. Oslavuje zelené alternativy a udržitelné volby |
| **Local insider** | Velmi specifické místní odkazy, varuje před turistickými pastmi, používá místní slang |
| **Inclusive** | Vítající, oslavující a potvrzující. Dává všem pocit, že jsou viděni a ceněni |
| **Mystical** | Éterický, duchovní, kosmické pravdy. Jemné pozvání k nahlédnutí dovnitř sebe |
| **Futuristic** | Elegantní, zaměřený na budoucnost. Miluje termíny jako "paradigmatický posun" a optimistické vize budoucnosti |
| **Budget** | Šikovný, vynalézavý, hrdě spořivý. Oslavuje dostupné objevy a bezplatné zážitky |

## Typy příspěvků

Typy příspěvků určují cíl a strukturu generovaného obsahu. Každý typ příspěvku má atribut **Weight**, který určuje, jak často se používá.

| Typ příspěvku | Popis |
| -- | -- |
| **Transactional** | Jasný, stručný, zaměřený na konverzi. Podporuje rezervace a přímé akce s fakty, cenami a dalšími kroky |
| **Awareness** | Obsah budující značku, navržený k představení vaší nemovitosti a zvýšení viditelnosti mezi potenciálními hosty |
| **Sentiment** | Emočně rezonující obsah, který vytváří pozitivní asociace a buduje emocionální spojení s publikem |

## Časová pásma

  <figcaption>*Obr. 4 - Časová pásma*</figcaption>

Možná budete chtít zaměřit své marketingové úsilí na konkrétní jazyk a region. Řekněte nám jazyky, pro které chcete vytvářet obsah, a časové pásmo, které vás zajímá, a my uděláme zbytek. Příspěvky jsou publikovány ve stanovený čas ve vámi vybraném časovém pásmu.

*Většina nemovitostí používá jeden jazyk a výchozí je jejich místní jazyk.*

:::tip[Pro tip]
Začněte pomalu

Nebuďte příliš ambiciózní při prvním pokusu. Doporučujeme začít vytvářet obsah pro vaši nemovitost v jednom jazyce.
:::
