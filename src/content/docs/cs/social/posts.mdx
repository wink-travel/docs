Editor příspěvků vám umožňuje vytvářet neomezené množství manuálních příspěvků a je také místem, kde spravujete příspěvky, které byly vytvořeny automaticky.

Editor má 4 sekce:

1. Sekce obsahu.
2. Sekce médií.
3. Sekce odkazu.
4. Sekce nastavení.

## Obsah

Sekce obsahu vám umožňuje upravit textový obsah před jeho publikací. Můžete vytvářet lokalizovaný obsah ve více jazycích — každý jazyk má svůj vlastní název a popis.

## Média

Sekce médií vám umožňuje upravit obrázky a videa, která doprovázejí text.

## Odkaz

Sekce odkazu vám umožňuje přidat odkaz, kam chcete, aby uživatel po přečtení příspěvku zamířil. *Např. může jít o odkaz na speciální nabídku nebo místo, kde získat více informací.* Můžete přidat URL odkazu, popisek odkazu a popis odkazu.

## Nastavení

Sekce nastavení vám umožňuje upravit / přepsat všechna výchozí nastavení pro každý příspěvek. Můžete nastavit:

- **Platforma** — Vyberte sociální síť, na kterou bude tento příspěvek publikován.
- **Typ příspěvku** — Vyberte mezi Transakčním, Povšimnutím nebo Sentimentem.
- **Tón hlasu** — Přepište tón hlasu pro tento konkrétní příspěvek.
- **Přizpůsobení** — Použijte jiné přizpůsobení pro zobrazovanou měnu, jazyk, itinerář atd.
- **Časové pásmo** — Přepište časové pásmo pro publikaci.
- **Naplánované datum** — Generované příspěvky jsou automaticky naplánovány na optimální čas. Můžete přepsat datum a čas, pokud preferujete jiný termín.

## Životní cyklus stavu příspěvku

Každý příspěvek prochází životním cyklem stavu:

| Stav | Popis |
| -- | -- |
| **Koncept** | Příspěvek byl vytvořen, ale ještě není připraven k revizi |
| **Čeká na schválení** | Příspěvek čeká na lidské schválení před naplánováním |
| **Schváleno** | Příspěvek byl schválen a bude naplánován |
| **Zamítnuto** | Příspěvek byl zkontrolován a zamítnut — nebude publikován |
| **Naplánováno** | Příspěvek je zařazen do fronty k publikaci v naplánovaný čas a datum |
| **Publikováno** | Příspěvek byl úspěšně publikován na sociální síti |
| **Pozastaveno** | Publikace příspěvku byla dočasně pozastavena |
| **Selhalo** | Příspěvek nemohl být publikován kvůli chybě |

## Schvalování příspěvků

Příspěvky se schvalují v mřížce příspěvků, kde můžete vybrat jeden nebo více příspěvků a zvolit jejich schválení nebo zamítnutí.

:::note
Pro každý jazyk a každou sociální síť bude jeden záznam.

Např. pokud máte vybrané 3 sociální sítě a 3 jazyky, uvidíte v kalendáři 9 záznamů.
:::
