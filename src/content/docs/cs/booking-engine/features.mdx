Rezervační systém je k dispozici ve dvou variantách, aby vyhověl různým potřebám affiliate partnera, správcovské společnosti a hotelu.

## Online cestovní agentura

Pro affiliate partnera je nejvhodnější použít verzi online cestovní agentury. Důvodem je, že stránka je nastavena tak, aby vyhovovala cestovatelům, kteří chtějí prohlížet více nemovitostí.

Domovskou stránku této aplikace najdete níže.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Internetový rezervační systém

Pro hotel nebo správcovskou společnost je nejvhodnější použít verzi internetového rezervačního systému. Tento typ uživatele má zájem zvýraznit jednu nemovitost nebo více nemovitostí ve správě.

Domovskou stránku této aplikace najdete níže.

  Wink IBE
</LinkButton>

## Konfigurace

Většinou není potřeba tyto aplikace sdílet přímo. Vytvořili jsme různé způsoby, jak sdílet cestovní inventář, které můžete objevit v [Wink Studio](/studio/what-is-studio) nebo prostřednictvím [WinkLinks](/link-manager/wink-links).

Pokud byste však chtěli ručně odkazovat na některou z těchto webových stránek, ujistěte se, že do URL přidáte své ID klienta a ID přizpůsobení jako parametry.

Své ID klienta najdete takto:

<Steps>
1. Přihlaste se do [Wink Extranet](/extranet/what-is-extranet) nebo [Wink Studio](/studio/what-is-studio).
2. Klikněte na ikonu svého profilu v pravém horním rohu a vyberte [Aplikace](/developers/applications).
3. Použijte aplikaci, která je propojena s Wink účtem, se kterým chcete pracovat.
4. Zkopírujte Client ID z této aplikace a vložte ho do URL jako parametr požadavku. Příklad: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Své ID přizpůsobení jako nemovitosti najdete takto:

<Steps>
1. Přihlaste se do [Wink Extranet](/extranet/what-is-extranet).
2. Vyberte nemovitost, se kterou chcete pracovat.
3. V hlavním navigačním panelu klikněte na `Účet > Branding`
4. Zkopírujte Customization ID a vložte ho do URL jako parametr požadavku. Příklad: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Své ID přizpůsobení jako affiliate partner najdete takto:

<Steps>
1. Přihlaste se do [Wink Studio](/studio/what-is-studio).
2. Vyberte účet, se kterým chcete pracovat.
3. V hlavním navigačním panelu klikněte na `Nástroje > Přizpůsobení`
4. Zkopírujte Customization ID z přizpůsobení, které chcete použít, a vložte ho do URL jako parametr požadavku. Příklad: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Výsledkem je URL, které vypadá přibližně takto:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Pokud chcete jít přímo na stránku hotelu, URL by vypadala přibližně takto:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Pokud nezadáte ID klienta a/nebo ID přizpůsobení, Wink OTA použije výchozí konfiguraci.
:::

## IBE

Pokud jste správcovská společnost, chcete odkazovat na domovskou stránku Wink IBE. Domovská stránka zobrazí váš určený motiv spolu s [nemovitostmi, které zastupujete](/studio/agency-representation).

Výsledkem je URL, které vypadá přibližně takto:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Pokud jste nemovitost, chcete odkazovat přímo na vstupní stránku nemovitosti Wink IBE.

Tato URL by vypadala přibližně takto:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
Pokud nezadáte ID klienta a/nebo ID přizpůsobení, Wink IBE se NENAČTE. Tyto parametry jsou povinné.
:::
