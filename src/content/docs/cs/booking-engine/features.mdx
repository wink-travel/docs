Rezervační systém je k dispozici ve dvou variantách, aby podporoval různé potřeby affiliate partnerů, správcovských společností a hotelů.

## Online cestovní kancelář

Pro affiliate partnery je nejrozumnější používat online verzi cestovní kanceláře. Důvodem je, že web je nastaven tak, aby vyhovoval cestovatelům, kteří si chtějí prohlédnout více nemovitostí.

Domovskou stránku této aplikace naleznete níže.

  Mrknutí OTA
</LinkButton>

## Konfigurace

Většinou není nutné sdílet žádnou z těchto aplikací přímo. Vytvořili jsme pro vás řadu způsobů, jak sdílet cestovní inventář, které můžete objevit v[Wink Studio](/studio/what-is-studio)nebo prostřednictvím[WinkLinks](/link-manager/wink-links).

Pokud byste však chtěli ručně propojit jeden z těchto webů, nezapomeňte do adresy URL přidat své ID klienta a ID přizpůsobení jako parametry.

Své ID klienta najdete takto:

<Steps>
  1. Přihlaste se do kteréhokoli z nich [Wink Extranet](/extranet/what-is-extranet) nebo [Wink Studio](/studio/what-is-studio).
  2. Klikněte na ikonu svého profilu v pravém horním rohu a klikněte na [Aplikace](/developers/applications).
  3. Použijte aplikaci, která je propojena s účtem Wink, se kterým chcete pracovat.
  4. Zkopírujte ID klienta z dané aplikace a vložte ho do adresy URL jako parametr požadavku. Příklad: ?client-id=MY\_CLIENT\_ID
</Steps>


### Extranet

ID přizpůsobení jako vlastnost najdete takto:

<Steps>
  1. Přihlásit se [Wink Extranet](/extranet/what-is-extranet).
  2. Vyberte vlastnost, se kterou chcete pracovat.
  3. V hlavním navigačním panelu klikněte na `Account > Branding`
  4. Zkopírujte ID přizpůsobení a vložte ho do adresy URL jako parametr požadavku. Příklad: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Studio

Své ID přizpůsobení jako affiliate partner můžete najít takto:

<Steps>
  1. Přihlásit se [Wink Studio](/studio/what-is-studio).
  2. Vyberte účet, se kterým chcete pracovat.
  3. V hlavním navigačním panelu klikněte na `Tools > Customization`
  4. Zkopírujte ID přizpůsobení z přizpůsobení, které chcete použít, a vložte ho do adresy URL jako parametr požadavku. Příklad: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

Nakonec získáte URL adresu, která vypadá nějak takto:

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Pokud byste chtěli přejít přímo na stránku hotelu, URL adresa by vypadala nějak takto:

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::note
Pokud neuvedete ID klienta a/nebo konfigurace, Wink OTA se automaticky vrátí k výchozí konfiguraci.
:::

## IBE

Pokud jste správcovská společnost, chcete odkazovat na domovskou stránku Wink IBE. Na domovské stránce se zobrazí vámi zvolené téma spolu s[nemovitosti, které zastupujete](/studio/agency-representation).

Nakonec získáte URL adresu, která vypadá nějak takto:

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Pokud jste nemovitost, chcete odkazovat přímo na vstupní stránku nemovitosti Wink IBE.

Ta URL adresa by vypadala nějak takto:

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::note
Pokud neuvedete ID klienta a/nebo konfigurace, Wink IBE se NENAČTE. Tyto parametry jsou povinné. :::
