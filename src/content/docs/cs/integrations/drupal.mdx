Modul **Wink Drupal Module** přidává komponenty inventáře cestování Wink na váš web Drupal jako nativní bloky. Editoři je mohou umístit kamkoli do rozvržení pomocí standardního systému bloků.

<LinkCard
  title="wink-travel/wink-drupal-module na GitHubu"
  href="https://github.com/wink-travel/wink-drupal-module"

/>

## Požadavky

- Drupal 10 nebo 11
- PHP 8.1+
- Composer

## Instalace

<Steps>
1. Nainstalujte přes Composer:
   ```bash
   composer require drupal/wink
   ```
2. Povolit modul:
   ```bash
   drush en wink -y
   ```
3. Přejděte do **Admin → Konfigurace → Webové služby → Nastavení Wink** (`/admin/config/wink`).
4. Zadejte svůj **Client ID** a **Client Secret**, vyberte své prostředí a klikněte na **Uložit**.
   Pro získání přihlašovacích údajů přejděte na [Applications](/cs/portal/settings/applications).
</Steps>

## Přidání bloku na stránku

<Steps>
1. Přejděte do **Admin → Struktura → Rozvržení bloků**.
2. Klikněte na **Umístit blok** v oblasti, kde chcete obsah Wink.
3. Vyhledejte **Wink** pro nalezení dostupných bloků:
   - **Wink Content** — zobrazuje kartu hotelu, mřížku nebo mapu
   - **Wink Lookup** — vyhledávací lišta destinací
   - **Wink Search** — tlačítko pro výběr itineráře
   - **Wink Account** — tlačítko pro přihlášení
   - **Wink Itinerary** — tlačítko stavu itineráře
   - **Wink Cart** — tlačítko nákupního košíku
4. U **Wink Content** nastavte **Typ rozvržení** a **ID rozvržení** v nastavení bloku.
5. Uložte konfiguraci bloku.
</Steps>

## Dostupné bloky

| Blok | Účel |
|---|---|
| **Wink Content** | Zobrazuje karty hotelů, řazené mřížky, mapy a další |
| **Wink Lookup** | Vyhledávací lišta pro destinace a hotely |
| **Wink Search** | Ikonové tlačítko otevírající výběr itineráře |
| **Wink Account** | Tlačítko pro přihlášení / uživatelský účet |
| **Wink Itinerary** | Tlačítko itineráře zobrazující aktuální cestu |
| **Wink Cart** | Tlačítko shrnutí nákupního košíku |
