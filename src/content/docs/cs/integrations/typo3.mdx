**Wink TYPO3 Extension** integruje inventář cestování Wink do TYPO3 13 jako nativní obsahové prvky. Editoři přidávají komponenty Wink na stránky pomocí standardního editoru stránek TYPO3 — není potřeba měnit šablony.

## Požadavky

- TYPO3 13.x
- PHP 8.2+
- Composer

## Instalace

<Steps>
1. Nainstalujte přes Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktivujte rozšíření:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. V backendu TYPO3 přejděte do **Admin Tools → Settings → Extension Configuration → wink**.
4. Zadejte svůj **Client ID**, **Client Secret** a vyberte prostředí. Uložte.
   Pro získání přihlašovacích údajů přejděte na [Applications](/cs/portal/settings/applications).
5. Přidejte statickou šablonu TypoScript do kořenové šablony:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Přidání obsahového prvku na stránku

<Steps>
1. Otevřete stránku v modulu **Page**.
2. Klikněte na tlačítko **+** v oblasti obsahu pro přidání nového prvku.
3. Vyberte jeden z obsahových prvků **Wink Travel**:
   - **Wink Content** — hotelové karty, řazené mřížky, mapy
   - **Wink Lookup** — vyhledávací lišta destinací
   - **Wink Search** — tlačítko pro výběr itineráře
   - **Wink Account** — tlačítko pro přihlášení
   - **Wink Itinerary** — tlačítko stavu itineráře
   - **Wink Cart** — tlačítko nákupního košíku
4. U **Wink Content** použijte pole FlexForm pro výběr **Layout Type** a zadejte **Layout ID**.
5. Uložte a publikujte stránku.
</Steps>

## Dostupné obsahové prvky

| Prvek | Účel |
|---|---|
| **Wink Content** | Hotelové karty, řazené mřížky, mapy a další |
| **Wink Lookup** | Vyhledávací lišta destinací |
| **Wink Search** | Ikonové tlačítko otevírající výběr itineráře |
| **Wink Account** | Tlačítko pro přihlášení / uživatelský účet |
| **Wink Itinerary** | Tlačítko itineráře zobrazující aktuální cestu |
| **Wink Cart** | Tlačítko shrnutí nákupního košíku |
