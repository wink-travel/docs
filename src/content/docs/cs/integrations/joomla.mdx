**Wink Joomla Extension** je balíček, který instaluje systémový plugin (načítá CDN skripty), obsahový plugin (zpracovává shortcode tagy v článcích) a komponentu (stránka nastavení v administraci).

## Požadavky

- Joomla 4 nebo 5
- PHP 8.1+

## Instalace

<Steps>
1. Stáhněte si nejnovější `pkg_wink-x.x.x.zip` z [GitHub releases stránky](https://github.com/wink-travel/wink-joomla-extension/releases).
2. V administraci Joomla přejděte na **Extensions → Manage → Install**.
3. Nahrajte zip soubor a klikněte na **Upload & Install**.
4. Oba pluginy jsou automaticky povoleny. Přejděte na **Components → Wink Settings**.
5. Zadejte svůj **Client ID** a **Client Secret** a klikněte na **Save**.
   Pro získání přihlašovacích údajů navštivte [Applications](/cs/portal/settings/applications).
</Steps>

## Vkládání komponent do článků

Použijte shortcode tagy kdekoli v obsahu článku:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Obsahový plugin při vykreslení nahradí tyto tagy správným HTML `<wink-*>`. Systémový plugin automaticky načte CDN skripty a vloží `<wink-app-loader>` do patičky stránky.

## Dostupné shortcody

| Shortcode | Výstup | Popis |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Karty hotelů, mřížky, mapy |
| `{wink-lookup}` | `<wink-lookup>` | Vyhledávací lišta destinací |
| `{wink-search}` | `<wink-search-button>` | Tlačítko výběru itineráře |
| `{wink-account}` | `<wink-account-button>` | Přihlášení / uživatelský účet |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Tlačítko stavu itineráře |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Tlačítko košíku |
