HubSpot CMS podporuje vlastní HTML moduly a kód hlavičky/patičky pro celý web, což usnadňuje vložení komponent Wink na marketingové stránky a vstupní stránky.

:::note
Vkládání kódu pro celý web vyžaduje přístup k **HubSpot CMS Hub** (Starter nebo vyšší). Vlastní HTML moduly jsou dostupné ve všech úrovních CMS.
:::

## Načtení skriptů Wink pro celý web

<Steps>
1. V HubSpotu přejděte na **Marketing → Soubory a šablony → Nástroje pro návrh**.
2. Otevřete aktivní **téma** `main.css` nebo použijte globální nastavení obsahu.
3. Alternativně přejděte na **Nastavení → Web → Stránky → Vlastní kód**.
4. Do pole **Head HTML** přidejte stylopis:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
5. Do pole **Footer HTML** přidejte skript a načítač aplikace (nahraďte `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
6. Uložte a publikujte.
</Steps>

:::tip
Vaše Client ID najdete ve svém účtu Wink v sekci [Applications](/cs/portal/settings/applications).
:::

## Přidání komponenty pomocí vlastního HTML modulu

<Steps>
1. V Nástrojích pro návrh vytvořte nový **modul** typu **Rich Text** nebo **HTML**.
2. Do výchozího obsahu modulu vložte HTML komponenty:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
3. Uložte modul a přetáhněte ho do libovolné šablony stránky nebo stránky pomocí editoru stránek.
</Steps>

## Přidání komponenty přímo v editoru stránek

Pokud nechcete vytvářet modul, můžete použít sekci **Custom HTML** v editoru drag-and-drop:

<Steps>
1. Otevřete stránku v editoru HubSpot.
2. Klikněte na **Přidat** a vyberte sekci **Custom HTML** nebo **Embed**.
3. Vložte značku komponenty do pole HTML.
4. Klikněte na **Použít** a publikujte stránku.
</Steps>

## Dostupné komponenty

| Komponenta | HTML značka | Účel |
|---|---|---|
| Content loader | `<wink-content-loader>` | Zobrazení hotelových karet, mřížek, map |
| Lookup | `<wink-lookup>` | Vyhledávací lišta pro destinace |
| Search button | `<wink-search-button>` | Otevření výběru itineráře |
| Account button | `<wink-account-button>` | Přihlášení / uživatelské menu |
| Itinerary button | `<wink-itinerary-button>` | Zobrazení aktuálního itineráře |
| Itinerary picker | `<wink-itinerary-picker>` | Kompletní formulář itineráře |
| Shopping cart | `<wink-shopping-cart-button>` | Tlačítko souhrnu košíku |

Podívejte se na [Web Components](/cs/integrations/web-components) pro úplný přehled atributů.
