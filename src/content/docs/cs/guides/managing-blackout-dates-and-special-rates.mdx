Ne každý den má stejnou cenu. Události, svátky, mimosezóna a poslední volné termíny vyžadují různé ceny. Tento průvodce vám ukáže, jak používat speciální sazby a omezení sazebních plánů k řešení těchto situací.

## Ceny při událostech

Když konference, festival nebo místní událost zvýší poptávku, budete chtít zvýšit ceny pro tato data.

<Steps>
1. Přejděte do `Monetize > Special Rates` ve vašem Extranetu.
2. Klikněte na `Create a new special rate`.
3. Vyberte hlavní sazbu a typ pokoje, které chcete přepsat.
4. Nastavte časové období pokrývající událost.
5. Zadejte upravené (vyšší) denní ceny.
6. Klikněte na `Save`.
</Steps>

Speciální sazba přepíše hlavní sazbu pouze pro tato data. Vaše běžné ceny zůstanou nezměněné mimo období události.

## Sezónní úpravy

Pro delší období (vysoká sezóna, přechodná sezóna, nízká sezóna) můžete:

- **Použít speciální sazby** — Vytvořit přepisy pro každou sezónu s upravenými cenami.
- **Aktualizovat hlavní sazby** — Přímo upravit základní sazby pro daná období.

Speciální sazby jsou vhodnější pro dočasné úpravy, zatímco aktualizace hlavních sazeb dává smysl u předvídatelných ročních vzorců.

## Last-minute sazby

Pro zaplnění neprodaných pokojů blízko data příjezdu:

<Steps>
1. Vytvořte nový **sazební plán** s omezením na rezervaci předem (např. rezervace nejdříve 3 dny předem).
2. Spojte ho se **speciální sazbou** nabízející slevu pro cílová data.
3. Tento sazební plán se zobrazí pouze cestovatelům rezervujícím v last-minute okně.
</Steps>

## Blokovaná data

Pro zablokování určitých dat v konkrétním sazebním plánu (např. žádné promo sazby během hlavní sezóny):

<Steps>
1. Přejděte do `Monetize > Rate Plans` a vyberte sazební plán.
2. Přejděte do sekce **Restrictions**.
3. Přidejte omezení **Sell date** pro omezení, kdy je sazba rezervovatelná.
4. Nebo použijte omezení **Stay date** pro omezení, kdy sazba platí.
</Steps>

Například můžete omezit svůj sazební plán "Early Bird", aby nebyl dostupný pro rezervace během prosince (hlavní sezóna), zatímco váš běžný sazební plán zůstane otevřený.

## Kombinace omezení

Omezení lze kombinovat pro přesnou kontrolu:

| Omezení | Co ovládá |
| -- | -- |
| **Sell dates** | Kdy lze sazbu rezervovat |
| **Stay dates** | Které dny pobytu sazba pokrývá |
| **Available days** | Které dny v týdnu se sazba zobrazuje |
| **Arrival days** | Které dny musí host přijet |
| **Departure days** | Které dny musí host odjet |
| **Required days** | Host musí zůstat alespoň jeden z těchto dnů |
| **Advance booking** | Jak daleko předem musí být rezervace provedena |
| **Length of stay** | Minimální a maximální počet nocí |
