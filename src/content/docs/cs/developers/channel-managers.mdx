Wink se integruje s předními channel managery, systémy pro správu nemovitostí (PMS) a centrálními rezervačními systémy (CRS), aby udržel vaše ceny a dostupnost synchronizované napříč všemi vašimi prodejními kanály.

## Jak to funguje

Integrace channel managerů umožňují obousměrný tok dat mezi vaším systémem pro správu nemovitostí a Winkem:

1. **Odesílání cen** — Váš channel manager posílá aktualizace cen do Winku. Když změníte ceny ve svém PMS, tyto změny se automaticky projeví ve Winku.
2. **Synchronizace dostupnosti** — Váš channel manager posílá aktualizace dostupnosti do Winku. Když jsou pokoje rezervovány nebo uvolněny ve vašem PMS, inventář ve Winku se aktualizuje v reálném čase.
3. **Notifikace o rezervacích** — Když je provedena rezervace přes Wink, váš channel manager je o tom informován, aby se rezervace zobrazila ve vašem PMS.

## Podporovaní channel manageři

Wink aktuálně integruje následující systémy:

| Channel Manager | Popis |
|---|---|
| **Channex** | Globální platforma pro distribuci a správu kanálů. |
| **TravelClick** | Hotelová řešení a správa kanálů (Amadeus). |
| **Omnibees** | Distribuce a řízení příjmů pro hotely. |
| **D-Edge** | Technologie pro pohostinství a centrální rezervační řešení. |
| **Allotz** | Alokace inventáře a správa kanálů. |
| **Ezee / Yanolja** | Cloudové PMS a správa kanálů zaměřená na asijský trh. |
| **RateGain** | Technologie pro inteligenci cen a distribuci. |
| **RateTiger** | Distribuce cen a dostupnosti v reálném čase. |
| **CloudBeds** | Cloudové PMS, channel manager a rezervační engine. |
| **SiteMinder** | Přední hotelový channel manager a distribuční platforma. |
| **Exely** | Synchronizace cen a dostupnosti pro hotely. |

:::note
Pokud váš channel manager není na seznamu, kontaktujte tým Wink. Nové integrace mohou být přidány na základě standardů protokolu OTA (Open Travel Alliance), které Wink podporuje.
:::

## Pro vývojáře

Pokud vytváříte integraci channel managera nebo PMS s Winkem, použijte API endpointy Channel Managera:

- **Aktualizace cen** — Odesílejte změny cen do Winku.
- **Aktualizace dostupnosti** — Odesílejte změny dostupnosti do Winku.
- **Získávání rezervací** — Stahujte rezervace z Winku do svého systému.

API Channel Managera dodržuje standardy protokolu OTA (SOAP/XML) pro kompatibilitu s existujícími systémy v pohostinství.

## Notifikace přes webhooky

Můžete se přihlásit k odběru webhook událostí channel managera a dostávat notifikace v reálném čase:

- `channel-manager.update.rate` — Přijata aktualizace cen.
- `channel-manager.update.availability` — Přijata aktualizace dostupnosti.
- `channel-manager.update` — Obecná aktualizace channel managera.

Podrobnosti najdete v [Katalogu webhook událostí](/cs/developers/webhook-events#channel-manager-events).
