[Web Components](https://developer.mozilla.org/en-US/docs/Web/API/Web_components) je standard, který vám umožňuje integrovat celé JavaScript funkce do vašeho webu, aniž byste museli umět programovat.
Díky této skvělé technologii můžete snadno vložit Wink cestovní inventář. Tento článek vás provede naší kolekcí webových komponent a ukáže vám, jak je používat.

## Vložení Wink

Existují tři věci, které musíte zahrnout na jakoukoli stránku, kde chcete zobrazit jednu z našich webových komponent.

1. Zahrňte náš stylopis.
2. Zahrňte náš Javascript.
3. Zahrňte náš aplikační loader.

### Stylopis

Zahrňte náš CSS stylopis do hlavičky vašeho dokumentu.

```html {3}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
</html>
```

### Javascript

Zahrňte náš Javascript na konec vašeho dokumentu. (*Doporučujeme těsně nad koncovým tagem &lt;/body&gt;*).

```html {3}
<html>
  <body>
    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
  </body>
</html>
```

Váš web je nyní připraven a schopen zobrazovat naše webové komponenty.

## Komponenty

Níže si přečtěte o naší knihovně komponent.

### Aplikační loader

Loader je zodpovědný za udržování stavu a správu interakce mezi našimi komponentami. (*Zahrňte jej pod náš Javascript*).

```html {3-6}
<html>
  <body>
    <wink-app-loader
      client-id="ZDE VLOŽTE SVÉ CLIENT ID"
      configuration-id="ZDE VLOŽTE SVÉ KONFIGURAČNÍ ID"
    ></wink-app-loader>
  </body>
</html>
```

### Content loader

Content loader je srdcem naší knihovny webových komponent. Je zodpovědný za zobrazování vašeho inventáře (karty, mřížky, mapy).

Dostupné atributy:

- layout
- id
- sort

*Atribut `sort` je dostupný pouze pokud je layout `RANKED` a nechcete použít existující řazenou mřížku. V těchto případech nechte `id` prázdné.*

Dostupné typy layoutu:

- `AD_BANNER`
- `MAP`
- `HOTEL`
- `GUEST_ROOM`
- `MEETING_ROOM`
- `SPA`
- `RESTAURANT`
- `ACTIVITY`
- `ATTRACTION`
- `PLACE`
- `ADD_ON`
- `LIST`
- `SEARCH`
- `RANKED`

Dostupné typy řazení:

- `MEMBER`
- `PRICE_LOW_TO_HIGH`
- `PRICE_HIGH_TO_LOW`
- `PRICE`
- `POPULARITY`
- `ECO`
- `EXPERIENCE`
- `PERK`
- `LOYALTY`
- `PACKAGE`

```html {3-6}
<html>
  <body>
    <wink-content-loader
       layout="GUEST_ROOM"
       id="2de7ee9c-61c9-11ef-9722-42004e494300"
    ></wink-content-loader>
  </body>
</html>
```

Ukázky najdete pro [karty](/cs/studio/cards), [mřížky](/cs/studio/grids) a [mapy](/cs/studio/maps).

### Lookup

Lookup skvěle funguje společně s jednou z vašich řazených mřížek. Umožňuje uživatelům vyhledávat hotel a destinaci a mřížka se aktualizuje podle výsledků vyhledávání.

```html {3}
<html>
  <body>
    <wink-lookup></wink-lookup>
  </body>
</html>
```

## Komponenty

### Platba

Platební komponenta umožňuje informovat TripPay, že cestovatel chce něco zakoupit, a nastavuje pravidla a ceny těchto položek.

Widget má jeden povinný atribut:

- `id` Identifikátor rezervační smlouvy, kterou chcete provést.

```html {3}
<html>
  <body>
    <trip-pay id="<INSERT_BOOKING_CONTRACT_ID>"></trip-pay>
  </body>
</html>
```

:::note
Chcete-li se naučit, jak generovat `id` rezervační smlouvy, přečtěte si průvodce: [Integrace s TripPay](/cs/guides/integrating-with-trip-pay).
:::

Widget připraví smlouvu k provedení a zobrazí uživateli platební údaje *(Obrázek 1)* pro dokončení rezervace.

## Chyby

Pokud se při vkládání jedné z našich webových komponent setkáte s chybovou zprávou, ať už na WinkLinks nebo na vašem vlastním webu, může být několik důvodů:

### Dostupnost

Inventář nemusí být již dostupný. Přejděte do [Wink Studio](/cs/studio/cards) a zkontrolujte, zda je stav "karty" dostupný.
Pokud není, zobrazí se červeně. V takovém případě můžete počkat, zda dodavatel znovu zpřístupní, nebo ji odeberte ze svého seznamu.

:::note
`Dostupnost` neznamená denní dostupnost inventáře k rezervaci. Znamená to, že inventář je pro vás a/nebo kohokoli úplně nedostupný.
:::

### Odebráno

Inventář mohl být odstraněn. Přejděte do [Wink Studio](/cs/studio/cards) a zkontrolujte, zda je stav "karty" dostupný.
Pokud není, zobrazí se červeně. V takovém případě můžete počkat, zda dodavatel znovu zpřístupní, nebo ji odeberte ze svého seznamu.

### Přizpůsobení

Možná jste omylem odstranili přizpůsobení, které jste definovali pro svůj inventář. Ujistěte se, že přizpůsobení je stále dostupné, a pokud chybí, nastavte nové.

### Aplikace

Toto je určeno pouze pro webové vývojáře. Pokud jste omylem odstranili [Aplikaci](/cs/developer/applications), čímž se klientské ID stalo nedostupným, vytvořte novou aplikaci a použijte nové klientské ID pro vložení našich Web Components.
