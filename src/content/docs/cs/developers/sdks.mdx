Zrychlete uvedení na trh použitím jednoho z našich oficiálně podporovaných jazykových Software Development Kitů.
Pokud momentálně nepodporujeme váš oblíbený jazyk, přejděte na API, které chcete použít, a stáhněte si OpenAPI specifikaci, kterou zpřístupňujeme pro každé API v horní části stránky. Můžete si vytvořit vlastní SDK buď forkováním jednoho z našich veřejných SDK repozitářů, nebo použitím jednoho z OpenAPI generátorů.

## Java

Tato knihovna byla zkompilována pro Java 24. Je nakonfigurována pro použití reaktivního WebClienta a je optimalizována pro použití se Spring Frameworkem. Spring však není povinností, ale poskytuje vám podporu OAuth2 ihned po vybalení.
