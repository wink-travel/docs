Zrychlete uvedení produktu na trh pomocí jedné z našich oficiálně podporovaných sad pro vývoj softwaru.
Pokud váš oblíbený jazyk momentálně nepodporujeme, přejděte k požadovanému API a stáhněte si specifikaci OpenAPI, kterou pro každé API zpřístupňujeme v horní části stránky. Vlastní SDK si můžete vygenerovat buď vytvořením forku v jednom z našich veřejných repozitářů SDK, nebo použitím jednoho z generátorů OpenAPI.

## Jáva

Tato knihovna byla zkompilována pro Javu 24. Je konfigurována pro použití reaktivního webového klienta a je optimalizována pro použití se Spring Frameworkem. Spring sice není vyžadován, ale poskytuje podporu OAuth2 ihned po instalaci.
