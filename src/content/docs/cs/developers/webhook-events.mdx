Tato stránka uvádí všechny webhook události, na které se můžete na Wink přihlásit. Pro instrukce k nastavení si přečtěte [Průvodce integrací webhooků](/cs/guides/webhook-integration) nebo [stránku správy webhooků](/cs/my-account/webhooks).

## Jak se přihlásit k odběru

1. Přejděte v portálu do **Aplikace > Webhooky**.
2. Vytvořte webhook a vyberte události, které chcete přijímat.
3. Wink pošle HTTP POST na váš endpoint pokaždé, když nastane daná událost.

Pro kompletní kroky nastavení a doporučené postupy si přečtěte [Integraci webhooků](/cs/guides/webhook-integration).

## Události účtu

Události související s životním cyklem a nastavením vašeho TripPay účtu.

| Událost | Klíč | Popis |
|---|---|---|
| Účet vytvořen | `account.create` | Byl vytvořen nový účet. |
| Účet v revizi | `account.under_review` | Účet čeká na schválení. |
| Účet aktualizován | `account.updated` | Detaily účtu byly změněny. |
| Stav účtu aktualizován | `account.status.updated` | Stav účtu se změnil (např. schválen, pozastaven). |
| Adresa účtu aktualizována | `account.address.updated` | Adresa účtu byla upravena. |
| Detaily účtu aktualizovány | `account.details.updated` | Pole s detaily účtu byla změněna. |
| Informace o účtu aktualizovány | `account.information.update` | Obecné informace o účtu byly aktualizovány. |
| Správci účtu aktualizováni | `account.managers.update` | Seznam správců účtu se změnil. |
| Profilový obrázek aktualizován | `account.profile_picture.updated` | Profilový obrázek účtu byl změněn. |
| Agent aktualizován | `account.agent.updated` | Informace o agentovi na účtu byly upraveny. |
| URL jméno aktualizováno | `account.url-name.update` | Změnila se vanity URL účtu. |
| Unikátní ID aktualizováno | `account.unique-id.update` | Unikátní identifikátor účtu byl změněn. |
