Pro všechny, kdo používají WordPress pro své podnikání nebo blog, důrazně doporučujeme používat [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/).
Plugin vám umožní snadný přístup k vašim Wink kartám, mřížkám, mapám atd. a usnadní vkládání obsahu do vašich příspěvků a stránek pomocí oblíbeného page builderu nebo našich vlastních shortcode.

## Instalace

Takto nainstalujete `Wink Affiliate WordPress Plugin` do vašeho WordPress webu.

<Steps>
1. Přihlaste se jako administrátor a buďte na Dashboardu.
2. Klikněte na `Plugins > Add New Plugin`.
3. Vyhledejte `Wink Affiliate WordPress Plugin`.
4. Klikněte na `Install` pro stažení a instalaci pluginu na váš web.
5. Klikněte na `Activate` pro aktivaci pluginu na vašem webu.
6. Po aktivaci budete přesměrováni na stránku `Wink Settings`. *Můžete tam také přejít přes `Appearance > Customize > Wink Settings`.*
7. Budete vyzváni k zadání `Client ID` a `Client Secret`. Navštivte [Applications](/cs/developers/applications) a zjistěte, jak získat své přihlašovací údaje.
8. Zadejte své údaje a klikněte na `Publish`. Nyní jste připojeni ke svému Wink účtu.
</Steps>

## Page buildery

Podporujeme nejpopulárnější page buildery na WordPress, které usnadňují používání Wink komukoliv.

### Gutenberg

Gutenberg je nativní editor WordPressu.

### Avada

[Avada](https://avada.com) je známá pro svůj široký výběr krásných šablon spolu s robustním page builderem s pokročilou sadou funkcí.

### Elementor

[Elementor](https://elementor.com) je [aktuálně] nejpopulárnější page builder s dostupnou bezplatnou i pro verzí.

### WPBakery

[WPBakery](https://wpbakery.com) byl první pokročilý page builder na trhu.

## Shortcody

### [winklookup]

Funguje s řazenou obsahovou mřížkou. Zadejte místo, které chcete navštívit, a zobrazí se dostupnost.

### [winksearch]

Jednoduché tlačítko pro otevření formuláře itineráře.

### [winkaccount]

Tlačítko účtu pro autentizaci. Po přihlášení se změní na rozbalovací nabídku s možnostmi účtu.

### [winkitinerary]

Stejné jako tlačítko hledání, ale obsahuje informace o itineráři jako text tlačítka.

### [winkcontent]

Umožňuje vložit vybraný inventář z Wink přímo do stránky nebo příspěvku. Tento shortcode také očekává ID rozvržení.

Rozvržení

```
[winkcontent layoutid=""]
```
