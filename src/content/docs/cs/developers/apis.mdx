## Payment API

Payment endpointy jsou určeny pro vývojáře, kteří chtějí nakupovat cestovní inventář. To lze provést přes API jako registrovaný cestovní agent nebo pomocí našeho API ve spojení s naším PCI kompatibilním platebním widgetem pro všechny ostatní entity.
Payment API zpřístupňuje endpointy pro partnery a hotely ke sledování rezervací, analytiky a dostupnosti finančních prostředků. Mohou si také zvolit výběr dostupných prostředků na svůj bankovní účet.

</LinkButton>
