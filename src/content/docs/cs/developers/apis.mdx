## Platební API

Platební koncové body jsou určeny pro vývojáře, kteří si chtějí zakoupit cestovní inventář. To lze provést prostřednictvím API jako registrovaný cestovní agent nebo pomocí našeho API ve spojení s naším platebním widgetem kompatibilním s PCI pro všechny ostatní subjekty.
Platební API zpřístupňuje koncové body pro affiliate partnery a hotely pro sledování rezervací, analýz a dostupnosti finančních prostředků. Mohou se také rozhodnout vybrat si dostupné finanční prostředky na svůj bankovní účet.
