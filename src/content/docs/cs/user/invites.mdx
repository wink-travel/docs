Uživatel Winku si může zvolit, že vám umožní spravovat jeden ze svých účtů nemovitostí, partnerských společností nebo cestovních kanceláří. V takovém případě obdržíte e-mail s žádostí o pozvánku.

import { Steps } from '@astrojs/starlight/components';

Chcete-li zobrazit své aktuální pozvánky Wink, postupujte takto:

<Steps>
  1. Musíte být přihlášeni na jednu z našich webových stránek.
  2. Klikněte na ikonu svého profilu v pravém horním rohu a zobrazí se vám nabídka.
  3. Klikněte na `Invites` odkaz.
  4. Reagujte s pozvánkami buď `Accepting` nebo `Rejecting` jim.
</Steps>
