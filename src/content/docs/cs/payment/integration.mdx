Pokud jste cestovní společnost a chcete outsourcovat část svého podnikání týkající se plateb a výplat TripPay, zde je 6 základních kroků, jak začít:

<Steps>
1. Vytvořte si [uživatelský účet](/cs/user/register) na Wink.
2. Vytvořte si účet u [TripPay](https://pay.wink.travel).
3. Vytvořte [Aplikaci](/cs/developers/applications) na TripPay, která vám umožní komunikovat s naším API.
4. Vytvořte a namapujte své příjemce u TripPay. *např. Pro většinu to znamená vytvoření účtů pro objekty na vaší platformě, kterým chcete, aby TripPay platil.*
5. Integrujte [platební webovou komponentu](/cs/developers/web-components/#payment) do svého webu.
6. Pošlete TripPay `nákupní košík` cestovatele a zahajte platební proces.
</Steps>

:::tip
Pro kompletní průvodce používáním TripPay pro váš cestovní web si přečtěte průvodce [Integrace s TripPay](/cs/guides/integrating-with-trip-pay).
:::

## API

Vývojáři, kteří chtějí spravovat `Platby`, přejděte na [Vývojáři > API > Platby](/cs/developers/apis/#payment-api).
