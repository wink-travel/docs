Pokud jste cestovní kancelář a chcete outsourcovat platby a výplaty vaší firmy společnosti TripPay, zde je 6 základních kroků, jak začít:

<Steps>
  1. Vytvořte [uživatelský účet](/user/register) na Winkovi.
  2. Vytvořte si účet u [TripPay](https://pay.wink.travel).
  3. Vytvořte [Aplikace](/developers/applications) na TripPay, který vám umožňuje komunikovat s naším API.
  4. Vytvořte a namapujte své příjemce pomocí TripPay. *např. pro většinu to znamená vytvoření účtů pro nemovitosti na vaší platformě, za které chcete, aby TripPay platil.*
  5. Integrujte [webová komponenta pro platby](/developers/web-components/#payment) na váš web.
  6. Pošlete cestovateli platbu TripPay `shopping cart` a zahájit proces platby.
</Steps>


:::tip
Úplný návod k používání TripPay pro váš cestovní web najdete v tomto průvodci[Integrace s TripPay](/guides/integrating-with-trip-pay).
:::

## API

Vývojáři, kteří chtějí spravovat`Payments`, jít na[Vývojáři > API > Platby](/developers/apis/#payment-api).
