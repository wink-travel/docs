Pokud jste integrátor, jako je Wink, a chcete namapovat své externí příjemce plateb pro použití s ​​TripPay, můžete to udělat jedním ze dvou způsobů:

* Ruční mapování
* Použití[Platební API](/developers/apis/#payment-api)(*Viz`Account Mappings`*)

:::tip
Pokud nepracujete pouze s několika stejnými vlastnostmi, doporučujeme k automatizaci tohoto kroku použít API. :::

Níže vás provedeme manuálním postupem mapování externího příjemce platby.*Používá API pod kapotou.*

<Steps>
  1. Přihlásit se [TripPay](https://pay.wink.travel).
  2. Vyberte svůj účet integrátora.
  3. Klikněte `Settings > Mappings` z hlavního navigačního panelu.
  4. Klikněte na `Create mapping` tlačítko. Zobrazí se vám mapovací formulář.
  5. **Místní účet** Začněte psát název existujícího TripPay, který chcete namapovat.
  6. **Externí identifikátor** Vložte externí ID této entity tak, jak existuje ve vzdáleném systému.
  7. Klikněte `Save` pokračovat. Budete přesměrováni zpět na seznam mapování.
</Steps>


Díky nastavenému mapování nyní TripPay ví, které účty má připsat jako příjemce rezervace, když se na ně odvoláte pomocí svých externích identifikátorů.

## API

Vývojáři, kteří chtějí spravovat`Payments`, jít na[Vývojáři > API > Platby](/developers/apis/#payment-api).
