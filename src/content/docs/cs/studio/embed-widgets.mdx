Studio nabízí několik vkládatelných widgetů, které můžete přidat na své webové stránky a zvýšit tak počet rezervací.

Pro přístup k vkládaným widgetům přejděte v postranním panelu do `Nástroje > Různé`.

## Dostupné widgety

| Widget | Popis |
| -- | -- |
| **Tlačítko účtu** | Značkové tlačítko, které odkazuje na vaši partnerskou stránku |
| **Tlačítko itineráře** | Tlačítko, které otevře výběr itineráře |
| **Formulář itineráře** | Inline formulář pro výběr data a hostů |
| **Tlačítko vyhledávání** | Tlačítko, které otevře vyhledávání ubytování |
| **Tlačítko vyhledávání v inventáři** | Tlačítko pro vyhledání konkrétního inventáře |

## Jak vložit

Každý widget poskytuje kódový úryvek, který můžete zkopírovat a vložit do HTML svých webových stránek. Úryvek obsahuje vaše **Client ID** a **Customization ID**, takže je widget automaticky propojen s vaším partnerským účtem.

:::note
Pokud používáte WordPress, nabízíme také WordPress plugin pro snadnější integraci.
:::
