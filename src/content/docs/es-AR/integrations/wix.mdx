La **Wink Wix App** agrega componentes web de Wink a su sitio Wix como widgets nativos que puede arrastrar y soltar en el Editor de Wix. No se requiere codificación para los propietarios del sitio.

<LinkCard

## Instalar

<Steps>
1. Vaya al [Wix App Market](https://www.wix.com/app-market) y busque **Wink Travel**.
2. Haga clic en **Agregar al sitio** y siga las indicaciones para autorizar la aplicación.
3. Una vez instalado, abra **Panel → Configuración de Wink** e ingrese su **Client ID** y **Client Secret**.
   Vaya a [Aplicaciones](/es-AR/portal/settings/applications) para obtener sus credenciales.
4. Haga clic en **Guardar** y luego en **Probar conexión** para confirmar que Wink está conectado.
</Steps>

## Agregar un componente a una página

<Steps>
1. Abra el **Editor de Wix** y vaya a la página donde desea agregar contenido de Wink.
2. Haga clic en **Agregar (+) → Aplicaciones → Wink Travel** para ver todos los widgets disponibles.
3. Arrastre un widget a la página — por ejemplo, **Wink Content** para mostrar una cuadrícula de hoteles.
4. En el panel de configuración del widget, seleccione el **Tipo de diseño** y pegue el **ID de diseño** desde Wink Studio.
5. Haga clic en **Publicar** para que los cambios estén activos.
</Steps>

## Widgets disponibles

| Widget | Propósito |
|---|---|
| **Wink Content** | Muestra tarjetas de hoteles, cuadrículas clasificadas, mapas y más |
| **Wink Lookup** | Barra de búsqueda para destinos y hoteles |
| **Wink Search** | Botón ícono que abre el selector de itinerarios |
| **Wink Account** | Botón de inicio de sesión / cuenta de usuario |
| **Wink Itinerary** | Botón de itinerario que muestra el viaje actual |
| **Wink Cart** | Botón resumen del carrito de compras |
