El **Módulo Wink para Drupal** agrega componentes del inventario de viajes de Wink a su sitio Drupal como bloques nativos. Los editores pueden ubicarlos en cualquier parte del diseño mediante el sistema estándar de bloques.

<LinkCard
  title="wink-travel/wink-drupal-module en GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"

## Requisitos

- Drupal 10 o 11
- PHP 8.1+
- Composer

## Instalación

<Steps>
1. Instale vía Composer:
   ```bash
   composer require drupal/wink
   ```
2. Active el módulo:
   ```bash
   drush en wink -y
   ```
3. Vaya a **Admin → Configuración → Servicios Web → Configuración de Wink** (`/admin/config/wink`).
4. Ingrese su **Client ID** y **Client Secret**, seleccione su entorno y haga clic en **Guardar**.
   Vaya a [Aplicaciones](/es-AR/portal/settings/applications) para obtener sus credenciales.
</Steps>

## Agregar un bloque a una página

<Steps>
1. Vaya a **Admin → Estructura → Diseño de Bloques**.
2. Haga clic en **Colocar bloque** en la región donde desea contenido de Wink.
3. Busque **Wink** para encontrar los bloques disponibles:
   - **Wink Content** — muestra una tarjeta de hotel, cuadrícula o mapa
   - **Wink Lookup** — barra de búsqueda de destinos
   - **Wink Search** — botón para seleccionar itinerarios
   - **Wink Account** — botón de inicio de sesión
   - **Wink Itinerary** — botón de estado del itinerario
   - **Wink Cart** — botón del carrito de compras
4. Para **Wink Content**, configure el **Tipo de Diseño** y el **ID de Diseño** en la configuración del bloque.
5. Guarde la configuración del bloque.
</Steps>

## Bloques disponibles

| Bloque | Propósito |
|---|---|
| **Wink Content** | Muestra tarjetas de hoteles, cuadrículas clasificadas, mapas y más |
| **Wink Lookup** | Barra de búsqueda para destinos y hoteles |
| **Wink Search** | Botón ícono que abre el selector de itinerarios |
| **Wink Account** | Botón de inicio de sesión / cuenta de usuario |
| **Wink Itinerary** | Botón de itinerario que muestra el viaje actual |
| **Wink Cart** | Botón resumen del carrito de compras |
