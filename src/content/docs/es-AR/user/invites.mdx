Un usuario de Wink puede elegir que vos gestiones una de sus cuentas de propiedad, afiliado o agente de viajes. Cuando eso sucede, recibirás un correo electrónico con una solicitud de invitación.

import { Steps } from '@astrojs/starlight/components';

Para ver tus invitaciones actuales de Wink, seguí estos pasos:

<Steps>
1. Debés iniciar sesión en uno de nuestros sitios web.
2. Hacé clic en tu ícono de perfil, en la esquina superior derecha, y aparecerá un menú.
3. Hacé clic en el enlace `Invites`.
4. Interactuá con tus invitaciones aceptándolas o rechazándolas.
</Steps>
