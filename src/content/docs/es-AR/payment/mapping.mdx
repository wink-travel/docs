Si sos un integrador, como Wink, y querés mapear tus beneficiarios externos para usar con TripPay, podés hacerlo de dos maneras:

- Mapeo manual
- Usando la [Payment API](/es-AR/developers/apis/#payment-api) (*Ver `Account Mappings`*)

:::tip
A menos que solo manejes un puñado de las mismas propiedades, recomendamos usar la API para automatizar este paso.
:::

A continuación, te guiamos por la forma manual de mapear un beneficiario externo. *Usa la API en segundo plano.*

<Steps>
1. Iniciá sesión en [TripPay](https://pay.wink.travel).
2. Seleccioná tu cuenta de integrador.
3. Hacé clic en `Settings > Mappings` desde la barra de navegación principal.
4. Hacé clic en el botón `Create mapping`. Verás el formulario de mapeo.
5. **Local account** Empezá a escribir el nombre de la cuenta existente de TripPay que querés mapear.
6. **External identifier** Pegá tu ID externo para esta entidad tal como existe en el sistema remoto.
7. Hacé clic en `Save` para continuar. Serás redirigido de vuelta a tu lista de mapeos.
</Steps>

Con tus mapeos configurados, TripPay ahora sabe qué cuentas acreditar como beneficiarios de una reserva cuando las referís usando tus identificadores externos.

## API

Los desarrolladores que quieran gestionar `Payments`, vayan a [Developers > API > Payments](/es-AR/developers/apis/#payment-api).
