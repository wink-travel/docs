Si eres un integrador, como Wink, y quieres mapear a tus beneficiarios externos para usarlos con TripPay, puedes hacerlo de dos maneras:

* Mapeo manual
* Usando el [API de pago](/developers/apis/#payment-api) (*Ver `Account Mappings`*)

:::tip
A menos que solo trabaje con unas pocas propiedades iguales, le recomendamos usar la API para automatizar este paso.
:::

A continuación, le explicaremos paso a paso el proceso manual para asignar un beneficiario externo.*Utiliza la API subyacente.*

<Steps>
  1. Iniciar sesión [TripPay](https://pay.wink.travel).
  2. Seleccione su cuenta de integrador.
  3. Hacer clic `Settings > Mappings` desde la barra de navegación principal.
  4. Haga clic en el `Create mapping` Botón. Verá el formulario de mapeo.
  5. **Cuenta local** Comience a escribir el nombre del TripPay existente que desea asignar.
  6. **Identificador externo** Pegue su ID externa para esta entidad tal como existe en el sistema remoto.
  7. Hacer clic `Save` Para continuar, serás redirigido a tu lista de asignaciones.
</Steps>


Con sus asignaciones establecidas, TripPay ahora sabe qué cuentas acreditar como beneficiarios de una reserva cuando usted se refiere a ellas utilizando sus identificadores externos.
