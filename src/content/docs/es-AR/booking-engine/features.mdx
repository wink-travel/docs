El motor de reservas viene en dos variedades para satisfacer las diferentes necesidades de un afiliado, una empresa de administración de propiedades y un hotel.

## Agencia de viajes online

Para un afiliado, lo más lógico es usar la versión de agencia de viajes en línea. Esto se debe a que el sitio está diseñado para viajeros que desean ver varias propiedades.

Puede encontrar la página de inicio de esta aplicación a continuación.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Guiño OTA
</LinkButton>


## Motor de reservas por Internet

Para un hotel o una empresa de gestión de propiedades, lo más lógico es utilizar la versión del motor de reservas online. Este tipo de usuario está interesado en destacar una o varias propiedades bajo gestión.

## Configuración

La mayoría de las veces, no es necesario compartir ninguna de estas aplicaciones directamente. Hemos creado diversas maneras de compartir tu inventario de viajes, que puedes descubrir en[Estudio Wink](/studio/what-is-studio)o a través de[Enlaces de guiño](/link-manager/wink-links).

Sin embargo, si desea vincularse manualmente a uno de estos sitios web, asegúrese de agregar su ID de cliente y su ID de personalización a la URL como parámetros.

Puede encontrar su ID de cliente mediante:

<Steps>
  1. Inicie sesión en cualquiera de los dos [Extranet de Wink](/extranet/what-is-extranet) o [Estudio Wink](/studio/what-is-studio).
  2. Haz clic en el icono de tu perfil en la esquina superior derecha y haz clic en [Aplicaciones](/developers/applications).
  3. Utilice la aplicación que está vinculada a la cuenta Wink con la que desea trabajar.
  4. Copia el ID de cliente de esa aplicación y pégalo en la URL como parámetro de solicitud. Ejemplo: ?client-id=MY\_CLIENT\_ID
</Steps>


### Extranet

Puede encontrar su ID de personalización como propiedad mediante:

<Steps>
  1. Iniciar sesión [Extranet de Wink](/extranet/what-is-extranet).
  2. Seleccione la propiedad con la que desea trabajar.
  3. Desde la barra de navegación principal, haga clic en `Account > Branding`
  4. Copia el ID de personalización y pégalo en la URL como parámetro de solicitud. Ejemplo: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Estudio

Puede encontrar su ID de personalización como afiliado mediante:

<Steps>
  1. Iniciar sesión [Estudio Wink](/studio/what-is-studio).
  2. Seleccione la cuenta con la que desea trabajar.
  3. Desde la barra de navegación principal, haga clic en `Tools > Customization`
  4. Copia el ID de personalización de la personalización que quieras usar y pégalo en la URL como parámetro de solicitud. Ejemplo: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

Lo que obtendrás es una URL que se parece a esto:

https://ota.wink.travel?client-id=MI\_ID\_DE\_CLIENTE\&configuration-id=MI\_ID\_DE\_PERSONALIZACIÓN

Si quisieras ir directamente a la página de un hotel, la URL se vería así:

https://ota.wink.travel/hotel/hotel-x?client-id=MI\_ID\_DE\_CLIENTE\&configuration-id=MI\_ID\_DE\_PERSONALIZACIÓN

:::nota
Si no incluye un ID de cliente o de configuración, Wink OTA usará la configuración predeterminada.
:::

## IBE

Si tiene una empresa de administración de propiedades, querrá vincular a la página de inicio de Wink IBE. La página de inicio mostrará el tema que usted designó junto con...[propiedades que representas](/studio/agency-representation).

Lo que obtendrás es una URL que se parece a esto:

https://book.wink.travel?client-id=MI\_ID\_DE\_CLIENTE\&configuration-id=MI\_ID\_DE\_PERSONALIZACIÓN

Si eres una propiedad, deseas vincularte directamente a la página de destino de la propiedad de Wink IBE.

Esa URL se vería así:

https://book.wink.travel/hotel/hotel-x?client-id=MI\_ID\_DE\_CLIENTE\&configuration-id=MI\_ID\_DE\_PERSONALIZACIÓN

:::nota
Si no incluye un ID de cliente o de configuración, Wink IBE no se cargará. Estos parámetros son obligatorios.
:::
