## API de pago

Los puntos de pago están destinados a desarrolladores que desean adquirir inventario de viajes. Esto se puede realizar a través de la API como agente de viajes registrado o utilizando nuestra API junto con nuestro widget de pago compatible con PCI para todas las demás entidades.
La API de pago expone puntos de pago para que afiliados y hoteles puedan realizar un seguimiento de las reservas, los análisis y la disponibilidad de fondos. También pueden optar por retirar los fondos disponibles a su cuenta bancaria.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
  Más información
</LinkButton>
