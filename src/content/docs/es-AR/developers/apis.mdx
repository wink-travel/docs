## Payment API

Los endpoints de Payment son para desarrolladores que quieren comprar inventario de viajes. Esto puede hacerse vía API como Agente de Viajes registrado o usando nuestra API junto con nuestro widget de pago compatible con PCI para todas las demás entidades.
La Payment API expone endpoints para afiliados y hoteles para rastrear reservas, análisis y disponibilidad de fondos. También pueden elegir retirar fondos disponibles a su cuenta bancaria.

<LinkButton href="https://developer.wink.travel/payment" variant="secondary" icon="external" target="_blank">
Aprendé más
</LinkButton>
