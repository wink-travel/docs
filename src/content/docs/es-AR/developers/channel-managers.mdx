Wink se integra con los principales channel managers, sistemas de gestión de propiedades (PMS) y sistemas centrales de reservas (CRS) para mantener tus tarifas y disponibilidad sincronizadas en todos tus canales de venta.

## Cómo funciona

Las integraciones con channel managers permiten un flujo bidireccional de datos entre tu sistema de gestión de propiedades y Wink:

1. **Envío de tarifas** — Tu channel manager envía actualizaciones de tarifas a Wink. Cuando actualizas tus precios en tu PMS, esos cambios se reflejan automáticamente en Wink.
2. **Sincronización de disponibilidad** — Tu channel manager envía actualizaciones de disponibilidad a Wink. Cuando se reservan o liberan habitaciones en tu PMS, el inventario de Wink se actualiza en tiempo real.
3. **Notificaciones de reservas** — Cuando se realiza una reserva en Wink, tu channel manager es notificado para que la reserva aparezca en tu PMS.

## Channel managers compatibles

Actualmente Wink se integra con los siguientes sistemas:

| Channel Manager | Descripción |
|---|---|
| **Channex** | Plataforma global de distribución y gestión de canales. |
| **TravelClick** | Soluciones hoteleras y gestión de canales (Amadeus). |
| **Omnibees** | Distribución y gestión de ingresos para hoteles. |
| **D-Edge** | Tecnología hotelera y soluciones centrales de reservas. |
| **Allotz** | Asignación de inventario y gestión de canales. |
| **Ezee / Yanolja** | PMS en la nube y gestión de canales enfocado en el mercado asiático. |
| **RateGain** | Inteligencia de tarifas y tecnología de distribución. |
| **RateTiger** | Distribución en tiempo real de tarifas y disponibilidad. |
| **CloudBeds** | PMS en la nube, channel manager y motor de reservas. |
| **SiteMinder** | Principal channel manager hotelero y plataforma de distribución. |
| **Exely** | Sincronización de tarifas y disponibilidad para hoteles. |

:::note
Si tu channel manager no está en la lista, contacta al equipo de Wink. Se pueden agregar nuevas integraciones basadas en los estándares del protocolo OTA (Open Travel Alliance) que Wink soporta.
:::

## Para desarrolladores

Si estás desarrollando una integración de channel manager o PMS con Wink, utiliza los endpoints de la API de Channel Manager:

- **Actualizaciones de tarifas** — Envía cambios de tarifas a Wink.
- **Actualizaciones de disponibilidad** — Envía cambios de disponibilidad a Wink.
- **Obtención de reservas** — Extrae reservas de Wink hacia tu sistema.

La API de Channel Manager sigue los estándares del protocolo OTA (SOAP/XML) para compatibilidad con sistemas hoteleros existentes.

## Notificaciones por webhook

Puedes suscribirte a eventos webhook de channel manager para recibir notificaciones en tiempo real:

- `channel-manager.update.rate` — Actualización de tarifa recibida.
- `channel-manager.update.availability` — Actualización de disponibilidad recibida.
- `channel-manager.update` — Actualización general del channel manager.

Consulta el [Catálogo de Eventos Webhook](/es-AR/developers/webhook-events#channel-manager-events) para más detalles.
