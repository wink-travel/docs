Las habilidades de IA son paquetes portátiles de instrucciones que enseñan a los agentes de codificación cómo trabajar con Wink. En lugar de depender de datos de entrenamiento o documentación desactualizada, cada habilidad dirige al agente a usar el [servidor MCP de Wink](/es-AR/developers/mcp) como la fuente de verdad en vivo para los contratos de API y la documentación de la plataforma.

Las habilidades funcionan con cualquier cliente agente que las soporte — Claude Code, Codex, Cursor y otros.

## Habilidades disponibles

### wink-api-builder

Implementa, revisa o depura integraciones de la API de Wink. La habilidad guía al agente a través de un flujo de trabajo estructurado:

1. Buscar la operación API correcta mediante `api_search`
2. Abrir los contratos de solicitud/respuesta en vivo desde el servidor MCP
3. Extraer la configuración de autenticación de la documentación
4. Escribir código basado en el contrato verificado
5. Señalar cualquier discrepancia entre el código existente y la API actual

Útil para:
- Construir nuevos métodos cliente para cualquier API de Wink
- Depurar discrepancias en contratos (por ejemplo, una solicitud que devuelve HTTP 400)
- Revisar integraciones existentes contra la especificación en vivo

## Instalación

Copia la carpeta de la habilidad en el directorio de habilidades de tu agente.

:::tip
El servidor MCP de Wink debe estar conectado en el mismo entorno. Consulta [Servidor MCP](/es-AR/developers/mcp) para instrucciones de configuración.
:::

**Claude Code**

```bash
# a nivel de repositorio
cp -r skills/wink-api-builder .claude/skills/

# o a nivel de usuario
cp -r skills/wink-api-builder ~/.claude/skills/
```

Invocar con:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# a nivel de repositorio
cp -r skills/wink-api-builder .agents/skills/

# o a nivel de usuario
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Invocar con:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Otros agentes**

Coloca la carpeta de la habilidad donde tu agente resuelva las habilidades. El punto de entrada es `SKILL.md`.

## Requisitos

- Un [servidor MCP de Wink](/es-AR/developers/mcp) conectado
- Un cliente agente con soporte para MCP y habilidades
