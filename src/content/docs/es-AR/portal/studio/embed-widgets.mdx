Studio ofrece varios widgets para insertar que podés agregar a tu propio sitio web para impulsar reservas.

Para acceder a los widgets para insertar, navegá a `Tools > Miscellaneous` desde la barra lateral.

## Widgets disponibles

| Widget | Descripción |
| -- | -- |
| **Account button** | Un botón con marca que enlaza a tu página de afiliado |
| **Itinerary button** | Un botón que abre el selector de itinerarios |
| **Itinerary form** | Un formulario en línea para seleccionar fechas y huéspedes |
| **Search button** | Un botón que abre la búsqueda de propiedades |
| **Lookup button** | Un botón para buscar inventario específico |

## Cómo insertar

Cada widget proporciona un fragmento de código que podés copiar y pegar en el HTML de tu sitio web. El fragmento incluye tu **Client ID** y **Customization ID** para que el widget se vincule automáticamente a tu cuenta de afiliado.

:::note
Si usás WordPress, también ofrecemos un plugin para WordPress que facilita la integración.
:::
