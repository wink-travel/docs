Cada viaje de reserva comienza con una habitación para huéspedes. Hay muchas funciones relacionadas con las habitaciones para huéspedes y puede ser un poco abrumador al principio.
Este artículo cubrirá todo lo que necesitas saber para convertirte en un especialista en habitaciones para huéspedes de Wink.

Las habitaciones para huéspedes, o tipos de habitación como se dice en `el negocio`, te permiten mostrar y vender tus habitaciones, dentro y fuera del establecimiento. Para crear una habitación para huéspedes, haz clic en el botón `Crear una nueva habitación para huéspedes`.

Las siguientes secciones te guían por cada parte de la gestión de una habitación para huéspedes.

## Configuración

La pestaña `Configuraciones` está seleccionada por defecto cuando comienzas.

Esta sección cubre lo básico de tu habitación para huéspedes para que sepamos a quién podemos vender esta habitación.

- **Nombre del tipo de habitación** El nombre de la habitación, como la llamas internamente. *ej. Supreme deluxe*
- **No fumadores** Si la habitación es para no fumadores o no.
- **Tarifa estándar** La tarifa publicada por una noche sin descuentos ni recargos incluidos. *ej. 100. Ten en cuenta que solo usamos este número para mostrar el precio esperado a los canales de venta o para prellenar ciertos campos cuando completas formularios.*
- **Mejor tarifa disponible** La mejor tarifa disponible por una noche sin descuentos ni recargos incluidos. *ej. 75. Ten en cuenta que solo usamos este número para mostrar el precio esperado a los canales de venta o para prellenar ciertos campos cuando completas formularios.*
- **Cantidad** La cantidad de unidades que tienes de este tipo de habitación en cualquier día dado. *ej. 40*

### Ocupación

:::caution
La ocupación juega un papel importante en si tu habitación para huéspedes aparece como disponible para un itinerario específico. Muchos de estos campos se completarán automáticamente cuando crees la habitación por primera vez. Asegúrate de modificarlos para que se ajusten a tu tipo de habitación.
:::

- **Ocupación mínima** El número mínimo de personas permitidas para alojarse en esta habitación. *ej. 1*
- **Ocupación adulta incluida** Número de adultos incluidos en el precio de una noche de habitación. *ej. 2. Nota: Si la habitación admite más que este número, se calculan usando el modificador `Adulto extra` en tu plan tarifario.*
- **Ocupación máxima** El número máximo de personas (adultos + niños) permitidas para alojarse en esta habitación. *ej. 4*
- **Ocupación adulta máxima** El número máximo de adultos permitidos para alojarse en esta habitación. *ej. 3*
- **Ocupación infantil incluida** Número de niños incluidos en el precio de una noche de habitación. *ej. 1. Nota: Si la habitación admite más que este número, se calculan usando el modificador `Niño extra` en tu plan tarifario.*
- **Ocupación infantil máxima** El número máximo de niños permitidos para alojarse en esta habitación. *ej. 2*

### Otros

Esta sección es solo para fines informativos. Los viajeros que buscan habitaciones más grandes y suites encontrarán esta información interesante.

- **Cantidad de baños** La cantidad de baños para este tipo de habitación. *ej. 1*
- **Cantidad de salas de estar** La cantidad de salas de estar para este tipo de habitación. *ej. 1*
- **Máximo de camas plegables** La cantidad de camas plegables que admite este tipo de habitación. *ej. 1*
- **Máximo de cunas** La cantidad de cunas que admite este tipo de habitación. *ej. 1*

### Clasificación de la habitación

- **Segmento** El segmento bajo el cual se categoriza la habitación para huéspedes. *ej. Económico*
- **Ubicación** El tipo de ubicación bajo el cual se categoriza la habitación para huéspedes. *ej. Piso alto*
- **Vista** El tipo de vista que se espera desde esta habitación para huéspedes. *ej. Vista a la ciudad*
- **Clasificación** La clasificación bajo la cual se ubica esta habitación para huéspedes. *ej. Económica*
- **Arquitectura** El estilo arquitectónico alrededor de esta habitación para huéspedes. *ej. Asiático*
- **Piso** Un piso(s) opcional donde se puede encontrar esta habitación para huéspedes *ej. Piso 20 en adelante*

### Composición

La composición se aplica solo si la habitación para huéspedes está compuesta por múltiples habitaciones para huéspedes.

- **Composición** Interruptor para activar ✅ si la habitación para huéspedes está compuesta por varias habitaciones. *ej. Sí*
- **Cantidad** Indica cuántas habitaciones para huéspedes componen esta habitación para huéspedes. *ej. 2*

### Detalles del hostal

Esta sección se aplica solo si administras un dormitorio, hostal u otro servicio de habitación compartida.

- **Habitación compartida** Interruptor para activar ✅ si la habitación para huéspedes es compartida. *ej. Sí*
- **Género** Indica el género para esta habitación para huéspedes. *ej. Masculino*

## Dormitorios

:::note
Hay un número infinito de posibles distribuciones de dormitorios. Hemos intentado hacer esto lo más personalizable posible para adaptarse a tu propiedad.
:::

Haz clic en la pestaña `Dormitorios` para continuar.

La sección de dormitorios te permite especificar la distribución de tu(s) dormitorio(s) e indicar qué hay en cada uno.

Las habitaciones para huéspedes del mismo tipo pueden tener múltiples distribuciones, o `Configuraciones de camas`. Una habitación para huéspedes necesita al menos una distribución.

- **Nombre de la configuración** Nombra la distribución con algo que entiendas *ej. Principal*

Puedes tener múltiples dormitorios dentro de una distribución.

- **Tipo de dormitorio** Cada dormitorio tiene un tipo. *El primero es `Principal` por defecto, pero puedes cambiarlo en tu configuración.*

Dentro de un dormitorio, puedes tener múltiples camas de diferentes tipos y cantidades.

- **Tipo de cama** Elige el tipo de cama en tu dormitorio. *ej. King*
- **Cantidad** Elige la cantidad de camas de este tipo en este dormitorio. *ej. 1*

## Servicios

:::note
Esta sección es opcional.
:::

Haz clic en la pestaña `Servicios` para continuar.

Similar a los servicios de la propiedad, los servicios de la habitación para huéspedes aplican solo a la habitación específica. Marca cada servicio que corresponda.

## Descripciones

:::note
Necesitas escribir al menos una descripción y debe estar en inglés.
:::

Haz clic en la pestaña `Descripciones` para continuar.

Similar al texto de bienvenida de la propiedad, también puedes crear descripciones localizadas para la habitación para huéspedes.
Puedes escribir en tantos idiomas como quieras. Añadiremos traducciones para todos los idiomas más populares.
Este es el texto que los viajeros verán al ver tus habitaciones para huéspedes.

- **Nombre** Dale un nombre a la habitación para huéspedes. *ej. Supreme deluxe*
- **Descripción** Describe la habitación para huéspedes en uno o dos párrafos. *ej. Esta es la mejor habitación para huéspedes del mundo...*

## Redes sociales

:::note
Esta sección es opcional.
:::

Haz clic en la pestaña `Redes sociales` para continuar.

Quizás te preguntes, "¿Por qué una habitación para huéspedes tiene su propia sección de redes sociales?". Una habitación para huéspedes también puede ser una villa cerca de la propiedad pero fuera del establecimiento con su propio personal y su propio perfil de IG.

Las redes sociales te mantienen en contacto con viajeros nuevos y existentes a lo largo del tiempo.
Agrega tus cuentas de redes sociales a tu perfil para que los huéspedes puedan contactarte fácilmente y conocer más sobre tu habitación para huéspedes.

<Steps>
1. Activa el interruptor de la red social que deseas habilitar.
2. Ingresa el nombre de tu cuenta en esa red.
</Steps>

## Reputación

:::note
Esta sección es opcional.
:::

Haz clic en la pestaña `Reputación` para continuar.

Similar a la pestaña `Redes sociales`, usarías esta pestaña si esta habitación para huéspedes tiene una puntuación de reputación separada o reconocimientos adicionales que quieras destacar aquí.

Tu reputación es tu moneda digital y quieres poder llevarla contigo dondequiera que vayas.
Una entrada de reputación puede ser cualquier cosa que tu propiedad quiera destacar, como una calificación de terceros o el concurso donde tu chef ganó oro.
En esta sección, puedes agregar tus puntuaciones y rankings de toda la Internet.

Haz clic en el botón `Agregar puntuación de reputación`.

<Steps>
1. **Categoría** Elige el tipo de puntuación que quieres agregar. *ej. Reseña de terceros*
2. **Proveedor** Escribe quién te otorgó la puntuación. *ej. BigOTA.com*
3. **Tipo de calificación** Selecciona la forma en que recibiste la puntuación. *ej. Numérica*
4. **Fecha** Opcionalmente, ingresa la fecha en que recibiste la puntuación.
5. **Calificación** Ingresa la puntuación que recibiste. *ej. 8.5*
6. **Calificación máxima** Opcionalmente, ingresa la puntuación máxima posible para este tipo de premio. *ej. 10*
</Steps>

## Venta basada en atributos

:::note
Esta sección es opcional.
:::

Haz clic en la pestaña `Venta basada en atributos` para continuar.

Todas nuestras instalaciones tienen su propia sección ABS. Esta sección te permite crear oportunidades de venta adicional en el inventario principal o hacer que un tipo de inventario no transaccional sea transaccional.
Recomendamos encarecidamente crear estas oportunidades para potencialmente conectar con el reservador a un precio más alto durante el proceso de reserva.

:::tip
Una habitación para huéspedes viene con muchas cosas para incluir como servicios adicionales:

- Colchón superior.
- Kit de baño deluxe.
- Parlante Bluetooth Marshal.
- Escritorio de trabajo con silla extra cómoda.
- Desayuno en la habitación
- Masaje en la habitación
- Pack de 12 cervezas
- Botella de vino
- Chocolate premium
- +++ deja volar tu imaginación y gana dinero extra!!
:::

- **Punto de precio** Dale al viajero una indicación rápida de qué tan cara es esta habitación para huéspedes. Así es... para la habitación para huéspedes. No para el servicio adicional.
- **Activo** Interruptor para 🛑 si quieres que esta habitación para huéspedes quede indisponible para reservar en todas las tarifas maestras y canales de venta.
- **Comisionable** Indica si los afiliados ganan comisión sobre este inventario.
- **Destacado** Indica si esta habitación para huéspedes debería tener una ubicación más prominente si la opción se presenta.
- **Venta basada en atributos** Interruptor para activar ✅ para habilitar ABS en este inventario. *Una vez habilitado, necesitas agregar al menos 1 ítem ABS para la venta*

### Ítem adicional

Cuando habilitas ABS, puedes comenzar a agregar ítems adicionales y ponerlos a la venta.

*Por defecto, ya se creó un ítem vacío para ti. Puedes agregar más haciendo clic en el botón `Agregar ítem para la venta`.*

- **Nombre** Dale un nombre a tu ítem adicional. *ej. Trufas de chocolate con champán de Royce*
- **Tipo de precio** Indica cómo quieres calcular el precio. *ej. Alquiler de paddleboard. Configura en `Por hora` para indicar que el precio se basa en la cantidad de horas alquiladas.*
- **Precio base** Ingresa el precio regular de este ítem. *ej. 20 / hora*
- **Precio con descuento** Ingresa el precio que los usuarios de Wink obtienen al reservar el ítem junto con la habitación. *ej. 15 / hora*

### Límites

Puedes restringir la disponibilidad de este ítem estableciendo límites mínimos/máximos en la cantidad.
Si continuamos con nuestro ejemplo de alquilar un paddleboard por hora pero queremos exigir que el huésped reserve al menos 2 tablas al mismo tiempo, haríamos:

- **Cantidad mínima** Restringe la cantidad mínima que debe reservarse. *ej. 2 paddleboards*
- **Cantidad máxima** Restringe la cantidad máxima que debe reservarse. *ej. dejar vacío para no tener restricción máxima*

### Descripciones

:::note
Necesitas escribir al menos una descripción y debe estar en inglés.
:::

- **Nombre** Dale un nombre a tu ítem adicional. *ej. Stand Up Paddleboard*
- **Descripción** Describe el ítem adicional en uno o dos párrafos. *ej. No te quedes sentado - ¡Ponte a remar!*

## Proximidad

Haz clic en la pestaña `Proximidad` para continuar.

No todo el inventario está ubicado en el establecimiento. Si tienes inventario que está fuera del establecimiento, informa al viajero aquí y se te dará la oportunidad de establecer una ubicación de mapa separada junto con información de contacto opcional.

*Esto puede ser útil a veces cuando una villa que es propiedad del hotel está ubicada a unos kilómetros del hotel y el viajero quiere explicar cómo llegar al taxista.*

Haz clic en el botón `Guardar` para continuar.
