[Portal](https://app.wink.travel) es la aplicación web unificada donde ahora conviven todas las secciones del ecosistema Wink. Anteriormente, cada área funcional (Extranet, Studio, Travel Agent, Payment, Social, Link Manager) tenía su propia URL. A partir de esta versión, todas son secciones de un único Portal bajo `app.wink.travel`, organizadas como subrutas.

## Secciones

- **`/extranet`** — Funciones para propiedades de hoteles, cadenas, marcas y empresas de gestión.
- **`/studio`** — Funciones para afiliados para promocionar inventario de viajes y ganar comisiones.
- **`/travel-agent`** — Funciones para agentes de viajes registrados para buscar, reservar y ganar.
- **`/payment`** — Funciones de pago para gestionar pagos y retiros.
- **`/social`** — Social automatizado para publicaciones en redes sociales impulsadas por IA.
- **`/link-manager`** — Link Manager para crear páginas WinkLinks.
- **`/corporate`** — Corporativo (próximamente).
- **`/settings`** — Configuraciones por cuenta: branding, administradores, aplicaciones, webhooks, pagos y más.
- **`/plan`** — Gestiona tu suscripción a Wink.
- **`/invite`** — Acepta invitaciones para administrar otras cuentas.
- **`/analytics`** — Paneles de rendimiento para todas tus cuentas.
- **`/leaderboard`** — Ve cómo se posiciona tu propiedad geográficamente.
- **`/claims`** — Convierte leads de propiedades en listados completos.

## Portal vs. Configuración de usuario

El Portal gestiona todo lo que haces como organización o titular de cuenta. La gestión personal del perfil de usuario (email, contraseña, passkeys, consentimientos de apps OAuth) se realiza por separado en el servicio **IAM** en `iam.wink.travel/my-account`. Consulta [Configuración de usuario](/es-AR/account/overview) para más información.

## Cambio de cuentas

Puedes administrar múltiples cuentas — propiedades, cuentas de afiliados, cuentas de agentes de viajes, cuentas corporativas — desde un único inicio de sesión. Usa el selector de cuentas en la navegación superior para cambiar entre ellas. Las configuraciones, analíticas y la mayoría de las funciones se aplican por cuenta.
