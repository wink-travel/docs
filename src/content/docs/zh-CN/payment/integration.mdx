如果您是一家旅游公司，并且希望将业务的付款和支出部分外包给 TripPay，请按照以下 6 个基本步骤开始操作：

<Steps>
  1. 创建一个 [用户帐户](/user/register) 在 Wink 上。
  2. 创建账户 [旅费](https://pay.wink.travel)。
  3. 创建一个 [应用](/developers/applications) 在 TripPay 上您可以与我们的 API 对话。
  4. 使用 TripPay 创建并映射您的受益人。 *例如，对于大多数人来说，这意味着在您的平台上为您希望 TripPay 支付的房产创建账户。*
  5. 整合 [支付网络组件](/developers/web-components/#payment) 进入您的网站。
  6. 向 TripPay 发送旅行者的 `shopping cart` 并启动付款流程。
</Steps>


:::提示
如需了解在您的旅游网站上使用 TripPay 的完整指南，请阅读指南[与 TripPay 集成](/guides/integrating-with-trip-pay).
:::
