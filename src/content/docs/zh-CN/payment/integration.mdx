如果您是一家旅游公司，想将业务中的支付和结算部分外包给 TripPay，以下是开始的 6 个基本步骤：

<Steps>
1. 在 Wink 上创建一个 [用户账户](/zh-CN/user/register)。
2. 在 [TripPay](https://pay.wink.travel) 创建一个账户。
3. 在 TripPay 上创建一个 [应用程序](/zh-CN/developers/applications)，以便您可以调用我们的 API。
4. 在 TripPay 上创建并映射您的受益人。*例如，对于大多数人来说，这意味着为您平台上希望 TripPay 支付的物业创建账户。*
5. 将 [支付网页组件](/zh-CN/developers/web-components/#payment) 集成到您的网站中。
6. 向 TripPay 发送旅客的 `购物车` 并启动支付流程。
</Steps>

:::tip
有关如何在您的旅游网站上使用 TripPay 的完整指南，请阅读指南 [与 TripPay 集成](/zh-CN/guides/integrating-with-trip-pay)。
:::

## API

想要管理 `支付` 的开发者，请访问 [开发者 > API > 支付](/zh-CN/developers/apis/#payment-api)。
