:::note

收款人使用 TripPay 来：

- 跟踪他们赚了多少钱。
- 账户中当前有多少钱。
- …以及可提现的金额。

## 使用方法

普通收款人通常这样使用 TripPay：

<Steps>
1. TripPay 从 3 个不同旅游平台的 10 笔预订中保留了资金，针对酒店 X，资金现已可用。
2. 酒店 X 收到 TripPay 的电子邮件，内容为 `您有可用资金`，并附有管理账户的链接。
3. 酒店 X 登录 [TripPay](https://pay.wink.travel)。
4. 酒店 X 查看上个月的预订及账户分析。
5. 酒店 X 选择提现资金。
6. 酒店 X 选择 `Pay-out` 方式。
7. 酒店 X 按照要求完成资金处理步骤。
8. 酒店 X 提交转账请求并点击 `确定`。
</Steps>

*资金通常在所有地区 24 小时内到账。*

:::note
一旦达到一定的交易量和金额，我们可能会联系您，要求提供更多验证您本人及/或公司身份的文件。
:::
