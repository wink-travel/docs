如果您是像 Wink 这样的集成商，并且想要映射您的外部收款人以便与 TripPay 一起使用，您可以通过以下两种方式之一来实现：

* 手动映射
* 使用[支付API](/developers/apis/#payment-api)(*看 `Account Mappings`*)

:::提示
除非您只处理少量相同的属性，否则我们建议您使用 API 来自动执行此步骤。
:::

下面，我们将引导您完成手动映射外部收款人的方法。*它使用底层的 API。*

<Steps>
  1. 登录 [旅费](https://pay.wink.travel)。
  2. 选择您的集成商帐户。
  3. 点击 `Settings > Mappings` 从主导航栏。
  4. 点击 `Create mapping` 按钮。您将看到映射表单。
  5. **本地帐户** 开始输入您想要映射的现有 TripPay 的名称。
  6. **外部标识符** 粘贴此实体在远程系统中的外部 ID。
  7. 点击 `Save` 继续。您将被重定向回映射列表。
</Steps>


通过您的映射，当您使用外部标识符引用哪些帐户时，TripPay 现在知道将哪些帐户记为预订的受益人。

## API

想要管理的开发人员`Payments`，前往[开发者 > API > 付款](/developers/apis/#payment-api)。
