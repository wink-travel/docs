如果您是集成商，比如 Wink，并且想要映射您的外部收款人以便与 TripPay 一起使用，您可以通过以下两种方式之一完成：

- 手动映射
- 使用 [Payment API](/zh-CN/developers/apis/#payment-api) (*参见 `Account Mappings`*)

:::tip
除非您只处理少量相同的物业，否则我们建议您使用 API 来自动化此步骤。
:::

下面，我们将引导您完成手动映射外部收款人的方法。*它实际上是通过 API 实现的。*

<Steps>
1. 登录 [TripPay](https://pay.wink.travel)。
2. 选择您的集成商账户。
3. 从主导航栏点击 `Settings > Mappings`。
4. 点击 `Create mapping` 按钮。您将看到映射表单。
5. **本地账户** 开始输入您想映射的现有 TripPay 账户名称。
6. **外部标识符** 粘贴该实体在远程系统中的外部 ID。
7. 点击 `Save` 继续。您将被重定向回映射列表。
</Steps>

完成映射后，TripPay 就知道在您使用外部标识符引用时，应将预订的受益人记入哪些账户。

## API

想要管理 `Payments` 的开发者，请访问 [Developers > API > Payments](/zh-CN/developers/apis/#payment-api)。
