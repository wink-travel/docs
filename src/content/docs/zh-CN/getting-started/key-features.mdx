Wink 旅游平台的功能源于实际需求。人们不知道如何：

- 快速开始在线销售旅游库存。
- 轻松通过社交媒体或线上销售设施和体验。
- 在预订过程中向旅客进行追加销售。
- 找到专注于旅游且能赚取直接佣金的联盟网络。
- 利用联盟网络无额外努力地通过数千个销售渠道销售，从而降低成本和风险。
- 保留客户所有权以便重新定位。
- 利用他们的社交影响力实现变现。
- 集成一个不需要冗长保密协议和长时间等待的旅游平台。
- 在与联盟合作时正确跟踪投资回报率。
- ……以及更多

随着时间推移，新增了功能，扩展了现有功能，也移除了一些功能。如果你从事旅游行业，或者只是喜欢/需要旅行，我们有充分的理由让你加入 Wink。继续阅读，了解原因。

以下功能构成了 Wink 的核心。

## 预订引擎

Wink 的核心是一款强大、可白标且高度可定制的预订引擎。它具备你对预订引擎的所有期望功能，如实时定价、关于物业的信息、设施、位置等。它支持不同的排名类别，如环保、生活方式和评价。它允许旅客创建详细的旅客档案并分享，告知酒店他们的偏好、过敏情况、紧急联系方式等。

我们的预订引擎通过精美设计的界面让每个物业和目的地看起来都很出色。但它真正的独特之处在于能够向旅客推销任何与物业相关的服务。就像廉价航空鼓励乘客升级到更宽敞的座位、增加行李额度和餐食一样，Wink 让物业清晰了解追加销售的样貌。从客房附加服务、预订会议室或当地餐厅座位，到博物馆、自然徒步和水上乐园……Wink 赋能物业，让整个社区通过优惠和体验焕发生机，令旅客惊喜。

我们的重点是让旅客体验更丰富的旅程，并在抵达前以更高的价格将物业与客人连接起来。

## 社交分享

我们的社交分享功能旨在让制作、分享和消费旅游优惠在社交媒体或其他线上渠道变得轻松。最广泛使用的在线分享方式是 `链接`。我们的链接带有动态链接预览，显示价格叠加在你选择的美丽图片上。此功能加快了旅客发现和预订的速度，提升了从社交动态到预订的转化路径​​。

## 链接管理器

链接管理器让你保存和整理喜欢的网址以备后用。浏览器书签就是一种链接管理器。随着社交媒体的兴起，暴露和分享链接给朋友和观众的需求更大。你的 Facebook 动态就是一个社交友好的链接管理器示例。

我们的链接管理工具 WinkLinks 更进一步。

- 它为你提供一个个性化网址，如 https://i.trvl.as/bob，适用于 IG 等只允许分享单一 URL 的场景，并让你管理来自全网的链接。此外，如果网站支持 `丰富内容`，如 Spotify，WinkLinks 会让播放列表可播放，或让 Facebook 视频可观看。
- WinkLinks 允许你免费添加所有现有的联盟链接，如 CJ 和 Amazon。
- WinkLinks 也是任何人只需两次点击即可销售旅游库存的最简便方式。它是一个精简版的预订引擎，专注于将你的观众转化为预订。

WinkLinks 高度可定制，支持你的品牌需求。

## 联盟网络

预订引擎是我们的核心产品，但联盟网络让我们的平台充满活力。Wink 让物业与联盟互联。双方可以协商交易、佣金和折扣。物业无需猜测合作是否合理，可以查看联盟的过往表现及其带来的旅客细分。即使没有直接连接，联盟也能访问一般可用的价格和优惠。

Wink 以多样化方式向联盟提供库存。我们跟踪每笔预订的分析数据、你的投资回报率，并负责支付所有相关方。你可以专注于你最擅长的事情；即向你的观众销售。

## 物业门户

Wink 的 Extranet 是最先进的内容管理系统，供物业在线销售房型、设施、体验和各种附加服务。我们设计此系统时，首先考虑易用性，其次是酒店比以往任何时候都更需要销售更多。我们在整个用户旅程中利用 AI，帮助并加速物业的“上市时间”。过去需要数周的工作现在不到一小时即可完成。

我们的 Extranet 是一个自助系统，帮助物业连接联盟，触达全球旅客。
