在 Wink，我们尊重您的隐私，并全力保护您的个人数据。我们遵守**通用数据保护条例（GDPR）**及全球其他适用的数据保护法律。

本页概述了您如何请求删除我们通过平台收集的任何个人数据——包括但不限于通过 Facebook 或 Instagram 等社交登录提供商获得的数据。

## 我们对数据隐私的承诺

我们仅收集提供服务所必需的数据，且绝不出售您的个人信息。可能包括：

- 您的姓名和电子邮件地址
- 公开的个人资料信息（例如来自 Facebook 或 Google）
- 您在我们平台上的使用或活动数据
- 您明确授予的任何权限

我们采取适当措施保护您的数据，并根据 GDPR 及 Meta 的数据政策处理所有删除请求。

## 请求删除数据

您有权请求我们删除持有的任何您的个人信息。

请求删除数据的方法：

- 📧 **发送电子邮件至**：[support@wink.travel](mailto:support@wink.travel)
- 📝 邮件主题请写：`Data Deletion Request`  
- 🔍 请提供身份识别信息，例如：
  - 您在我们平台使用的电子邮件地址
  - 您的 Facebook 或 Google ID 或类似信息（如适用）

我们将在收到请求后**30天内**回复，符合 GDPR 要求。

## 有疑问或顾虑？

如果您对我们如何收集、使用或删除您的数据有任何疑问，欢迎随时联系隐私团队：[support@wink.travel](mailto:support@wink.travel)。

---

感谢您的信任。我们严肃对待数据保护，致力于让您完全掌控您的信息。
