# 连接

您已经连接了 WinkLinks，我们的链接管理工具。点击 `Connect a social network` 连接更多您的社交网络。
该过程将重定向您到您的社交网络，批准 `Wink Social` 应用代表您发布内容。

连接成功后，我们将开始为该网络生成下一个计划月份的内容。

:::note
我们为每个社交网络创建并发布优化的内容。
:::
