销售渠道是您控制库存分销的方式。销售渠道定义了谁可以销售您的房间，他们获得的佣金，以及可用的价格计划。

## 销售渠道类型

在 Wink 上分销您的物业主要有三种方式：

- **Wink Network** — 您的物业对平台上的所有合作伙伴可见。这是最广泛的分销方式。
- **直接渠道** — 您为特定合作伙伴创建私有渠道，并设定自定义佣金比例。
- **您自己的网站** — 使用可分享链接或嵌入卡片，直接从您的网站销售，无需中介。

## 创建销售渠道

<Steps>
1. 访问 https://extranet.wink.travel 并选择您的物业。
2. 导航到销售渠道部分。
3. 点击 `创建新的销售渠道`。
4. 为渠道命名（例如，“网站直销”，“合作伙伴 - Kim Travel”）。
5. 设置**佣金百分比** — 合作伙伴每笔预订获得的佣金比例。
6. 选择该渠道可用的**价格计划**。
7. 可选地限制特定的**房型**或**库存**。
8. 点击 `保存`。
</Steps>

## Wink Network

Wink Network 是默认的分销渠道。启用后，您的物业会出现在 [Studio](/zh-CN/studio/search)，任何合作伙伴都可以找到并销售它。

启用步骤：

<Steps>
1. 进入您的物业网络设置。
2. 启用 `Wink Network`。
3. 为网络合作伙伴设置默认佣金比例。
</Steps>

:::tip
Wink Network 是获得广泛曝光的最简单方式。您也可以稍后为表现优异的合作伙伴创建不同佣金率的直接渠道。
:::

## 直接合作伙伴渠道

对于您希望密切合作的合作伙伴，创建带有自定义佣金的直接渠道。

<Steps>
1. 通过网络部分找到该合作伙伴。参见 [Network](/zh-CN/extranet/network)。
2. 为该合作伙伴创建直接销售渠道。
3. 设置自定义佣金比例（可高于或低于网络默认）。
4. 选择该渠道可用的库存。
</Steps>

当您想为网红或旅行代理提供优惠价格或独家房间访问时，这非常有用。

## 佣金结构

佣金以预订总额的百分比表示。佣金在相关方之间分配：

- **合作伙伴佣金** — 销售方（合作伙伴、旅行代理、网红）获得的部分。
- **平台费用** — Wink 的处理费用。
- **物业收入** — 剩余部分归您所有。

您控制合作伙伴佣金。平台费用会自动扣除。

## 绩效监控

渠道上线后，使用 [Analytics](/zh-CN/my-account/analytics) 跟踪哪些渠道带来最多预订和收入。这有助于您优化佣金比例并决定投资哪些合作关系。
