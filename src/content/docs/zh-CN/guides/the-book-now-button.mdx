**立即预订**按钮是酒店行业中一个普遍使用的术语，指的是酒店与能够了解酒店房态和价格并允许旅客预订房间的供应商的集成。该按钮是酒店能够在自己网站上以自己的方式销售房间的最简单方式。

*许多酒店使用其渠道管理器提供的捆绑预订引擎。我们建议酒店在选择任何特定预订引擎之前，进行一些调研并比较本地预订引擎。*

## 受众

本文主要面向希望在其网站上安装我们免费、高转化率预订引擎的酒店。
但……对于任何想在其网站上使用我们技术的人来说，也具有参考价值。

## 通过链接

为了实现最简单的集成，我们推荐使用[可分享链接](/zh-CN/studio/shareable-links)。

操作步骤如下：

<Steps>
1. 访问 https://extranet.wink.travel
2. 点击右上角的 `登录 / 注册` 并使用您的用户账户登录。
3. 从左上角下拉菜单中选择您要操作的酒店。
4. 点击导航栏中的 `预订引擎`。
5. 页面将显示您的链接。
6. 将此链接提供给您的网站管理员，并用于您的 **立即预订** 按钮。
7. 您已完成 ✅。
</Steps>

### 高级用法

该链接已包含源自您所选 `定制` 的行程信息。
您可以通过以下方式进一步定制您的链接：

- 更改所选定制，或
- 覆盖行程

本文将重点介绍后者，因为它允许您让旅客在您的网站上选择行程并将其转发到 Wink 的预订引擎。

以下是如何通过 URL 传递您自己的行程。

有效的请求参数：

- **sd** 添加开始日期：*sd=2024-08-24*
- **n** 添加入住晚数：*n=1* 或
- **ed** 使用结束日期：*ed=2024-08-25*
- **rc** 客人数量：*rc=a2*（成人：2）
- **l** 请求显示语言：*l=es*
- **c** 请求显示货币：*c=EUR*
- **p** 添加促销代码：*p=ABC123*

URL 示例：

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2&l=es&c=EUR

对于更高级的用例，您还可以使用 `+` 符号请求多种房型配置：

示例：

https://trvl.as/abc123?sd=2024-08-24&n=1&rc=a2-ca16q1-ca8q1+a2&l=es&c=EUR

- 房型1 = 成人：2，儿童：2（16岁 + 8岁）
- 房型2 = 成人：2

### 示例

示例 **立即预订** 按钮 HTML：

*此示例假设您使用 [Bootstrap](https://getbootstrap.com) 作为主要 CSS 库。*

```html
<a
  href="https://trvl.as/abc123"
  target="_blank"
  class="btn btn-lg btn-primary"
  title="预订房间"

立即预订
</a>
```

### 其他优势

使用我们的链接作为您的 `立即预订` 按钮还有其他优势。

- **社交友好** 链接可轻松分享到社交媒体，并在您选择的图片上显示 `动态定价`。
- **简短** URL 非常便于分享，您可以在 Instagram 个人简介 URL 等位置使用。

## 通过嵌入

一些酒店希望在其网站上实现更多功能和更深层的集成。此示例将介绍如何将[卡片](/zh-CN/studio/cards)嵌入其网站。

操作步骤如下：

<Steps>
1. 访问 https://studio.wink.travel
2. 点击右上角的 `登录 / 注册` 并使用您的用户账户登录。
3. 从左上角选择您要操作的酒店。
4. 点击主导航栏中的 `搜索`。
6. 输入您的酒店名称并点击搜索按钮。
7. 点击搜索结果卡片上的 `操作` 链接，选择 `创建 > 制作卡片`。
8. 这将创建一个 `卡片`。
9. 右上角会出现消息，告知您的卡片已准备好，是否需要进一步定制。
10. 点击主导航栏中的 `工具 > 卡片`，您将看到新卡片。
11. 点击卡片上的 `操作` 链接，选择 `嵌入`。
12. 这将打开一个新窗口，您可以复制示例 HTML 代码。
13. 要获取您的 `Client-ID`，请访问 [应用](/zh-CN/developers/applications)。
14. 要获取您的 `Configuration-ID`，请访问 [定制](/zh-CN/studio/customization)。
14. 您已完成 ✅。
</Steps>

嵌入卡片示例 HTML：

```html
<wink-content-loader
  layout="GUEST_ROOM"
  id="09d7cca4-6ff6-11ef-949b-42004e494300"
></wink-content-loader>
```

*上述 HTML 示例不包含必需的 Wink CSS 和 Javascript；仅为卡片代码。*
