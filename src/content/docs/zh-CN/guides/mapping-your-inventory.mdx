当您在Extranet中为您的物业添加库存——客房、餐厅、水疗、会议室、活动等——这些库存将可在Wink生态系统中使用。本指南解释了各部分如何连接。

## 库存流程

```
Extranet（您创建）→ 预订引擎（旅客查看）→ Studio（联盟销售）
```

1. **您在Extranet创建库存**——房型、设施和体验。
2. **旅客通过可分享链接、卡片或直接搜索**在预订引擎浏览这些库存。
3. **联盟在Studio发现您的库存**，创建卡片和网格，并向其受众销售。

## 房型

房型是您库存的核心。每个房型需要：

- 名称和描述。
- 床型配置。
- 最大入住人数（成人和儿童）。
- 房间专属设施。
- 照片。
- 至少一个包含主价的价格计划。

没有价格，房型存在但无法预订。

## 设施和体验

除了客房，您还可以列出：

| 库存类型 | 定义 | 示例 |
| -- | -- | -- |
| **餐厅** | 现场用餐 | “屋顶酒吧与烧烤” |
| **水疗** | 健康服务 | “宁静水疗 - 60分钟按摩” |
| **会议室** | 活动和会议空间 | “董事会议室A - 20人” |
| **活动** | 物业内或附近的活动 | “导游浮潜之旅” |
| **景点** | 附近著名地点 | “国家博物馆 - 500米” |
| **地点** | 有趣的位置 | “当地市场” |
| **附加服务** | 可与房间一起购买的额外服务 | “机场接送” |

每种设施类型在Extranet中有独立版块。联盟可以单独销售这些设施——旅客可能通过水疗列表而非房型搜索发现您的物业。

## 联盟如何使用您的库存

在Studio，联盟可以：

- **搜索**您的物业或单个库存项目。
- **创建卡片**——单个可预订的房间、水疗或活动卡片。
- **创建网格**——一组卡片的集合展示。
- **创建地图**——基于位置的标记，显示可预订库存。
- **分享链接**——指向您的物业或特定项目的直接URL。

您的库存越完整多样，联盟推广您物业的方式就越多。

## 小贴士

- **详尽列出**——列出您物业提供的所有内容。每个项目都是潜在的预订入口。
- **使用优质照片**——每个库存项目应至少有一张高质量图片。
- **保持描述更新**——在更改服务或季节性调整时更新描述。
- **为所有项目定价**——无价格的库存可浏览但无法预订。
