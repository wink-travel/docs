对于任何使用 WordPress 开展业务或博客的人，我们强烈建议使用 [Wink Affiliate WordPress插件](https://wordpress.org/plugins/wink2travel/) 。
该插件可让您轻松访问 Wink 卡片、网格、地图等，并可让您使用您最喜欢的页面构建器或我们的自定义短代码将内容拖放到您的帖子和页面中。

## 安装

这是安装方法`Wink Affiliate WordPress Plugin`进入您的 WordPress 网站。

<Steps>
  1. 当您以管理员身份登录到仪表板时。
  2. 点击 `Plugins > Add New Plugin`。
  3. 搜索 `Wink Affiliate WordPress Plugin`。
  4. 点击 `Install` 下载插件并将其安装到您的网站。
  5. 点击 `Activate` 在您的网站上激活该插件。
  6. 激活后会转发 `Wink Settings` 页。 *您也可以通过导航前往 `Appearance > Customize > Wink Settings`。*
  7. 它会询问你 `Client ID` 和 `Client Secret`. 前往 [应用](/developers/applications) 了解如何检索您的凭证。
  8. 输入您的凭据并点击 `Publish`。您现在已连接到您的 Wink 帐户。
</Steps>

## 页面构建器

我们支持 WordPress 上最流行的页面构建器，让任何想要使用 Wink 的人都可以轻松使用。

### 古腾堡

Gutenberg 是 WordPress 的原生编辑器。

### 阿瓦达

[阿瓦达](https://avada.com) 以其大量精美的模板以及具有高级功能集的强大页面构建器而闻名。

### Elementor

[Elementor](https://elementor.com) 是\[当前]最受欢迎的页面构建器，有免费版和专业版可供选择。

### WPBakery

[WPBakery](https://wpbakery.com) 是市场上第一个先进的页面构建器。

## 短代码

### \[眨眼查找]

适用于排名内容网格。输入您想去的地方，即可显示库存。

### \[眨眼搜索]

简单的按钮即可打开行程表。

### \[winkaccount]

帐户按钮用于进行身份验证。身份验证完成后，它会变成一个包含帐户选项的下拉菜单。

### \[眨眼日记]

与搜索按钮相同，只是它包含行程信息作为按钮文本。

### \[眨眼内容]

允许您将 Wink 上选择的库存直接嵌入到页面或帖子中。此短代码也需要布局 ID。

布局

```
[winkcontent layoutid=""]
```
