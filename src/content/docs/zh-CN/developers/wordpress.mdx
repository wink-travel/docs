对于使用 WordPress 进行业务或博客运营的用户，我们强烈推荐使用 [Wink Affiliate WordPress Plugin](https://wordpress.org/plugins/wink2travel/)。
该插件让您轻松访问 Wink 卡片、网格、地图等内容，并且只需通过您喜欢的页面构建器或我们的自定义短代码，将内容拖放到您的文章和页面中即可。

## 安装

以下是将 `Wink Affiliate WordPress Plugin` 安装到您的 WordPress 网站的步骤。

<Steps>
1. 以管理员身份登录并进入仪表盘。
2. 点击 `插件 > 安装插件`。
3. 搜索 `Wink Affiliate WordPress Plugin`。
4. 点击 `安装`，将插件下载并安装到您的网站。
5. 点击 `启用`，激活插件。
6. 激活后会跳转到 `Wink 设置` 页面。*您也可以通过导航到 `外观 > 自定义 > Wink 设置` 进入该页面。*
7. 系统会要求您输入 `Client ID` 和 `Client Secret`。请访问 [Applications](/zh-CN/developers/applications) 了解如何获取您的凭证。
8. 输入凭证后点击 `发布`。您现在已连接到您的 Wink 账户。
</Steps>

## 页面构建器

我们支持 WordPress 上最流行的页面构建器，方便任何想使用 Wink 的用户。

### Gutenberg

Gutenberg 是 WordPress 的原生编辑器。

### Avada

[Avada](https://avada.com) 以其丰富的精美模板和功能强大的页面构建器著称。

### Elementor

[Elementor](https://elementor.com) 是目前最受欢迎的页面构建器，提供免费和专业版。

### WPBakery

[WPBakery](https://wpbakery.com) 是市场上首个高级页面构建器。

## 短代码

### [winklookup]

适用于排名内容网格。输入您想访问的地点，显示相关库存。

### [winksearch]

简单按钮，用于打开行程表单。

### [winkaccount]

账户按钮，允许您进行身份验证。认证后会变成带有账户选项的下拉菜单。

### [winkitinerary]

与搜索按钮相同，但按钮文本包含行程信息。

### [winkcontent]

允许您将选定的 Wink 库存直接嵌入页面或文章中。此短代码还需要一个布局 ID。

布局格式

```
[winkcontent layoutid=""]
```
