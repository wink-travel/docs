我们所有的 REST API 均与语言无关，并符合开放 API 标准。无需编写任何代码即可开始测试。

## 支付API

支付端点适用于希望购买旅游库存的开发者。注册旅行社可以通过 API 进行购买，或者结合使用我们的 API 和符合 PCI 标准的支付小部件，为所有其他实体提供服务。
支付 API 为联盟会员和酒店提供端点，用于跟踪预订、分析和资金可用性。他们还可以选择将可用资金提取到自己的银行账户。

  了解更多
</LinkButton>
