## Payment API

Payment端点适用于希望购买旅游库存的开发者。注册的旅行代理可以通过API完成购买，其他实体则可结合我们的PCI合规支付组件使用API。
Payment API提供端点，供联盟和酒店跟踪预订、分析和资金可用性。他们也可以选择将可用资金提现到银行账户。
