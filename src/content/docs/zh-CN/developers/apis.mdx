## Payment API

Payment 端点适用于希望购买旅游库存的开发者。注册的旅行代理可以通过 API 完成购买，其他实体则可结合我们的 PCI 合规支付组件使用 API。
Payment API 提供端点，供联盟和酒店跟踪预订、分析和资金可用性，也可选择将可用资金提现到银行账户。
