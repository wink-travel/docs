> “如果我看得更远，那是因为我站在巨人的肩膀上。”

-- *艾萨克·牛顿爵士*

这样的平台不是孤立构建的。它是许多动态部分的总和；既有旧的也有新的。我们希望继续为后来者服务。

> “没有必要重新发明破损的轮子。”

旅游科技分散且遗留。我们尝试采用有效的旧技术。我们集成了许多渠道管理器、物业管理系统等。我们还添加了一些新功能。并且，我们将所有这些包裹在业内公认且易于使用的标准和协议中……这样您就不必自己去做。

:::note[API 驱动的方法]
大致意味着，我们为自己构建的任何功能都应该是：

- 易于理解
- 灵活的
- 可扩展的
- 可重用的，且……
- 可扩展的

……供其他也想使用这些 API 的人使用。
:::

Wink 是采用这种 `API 驱动的方法` 构建的。基于这种设计理念，作为外部开发者，您可以像操作自己的系统一样与 Wink 的所有功能交互。平台还有其他方面，您在集成时也可以利用。
