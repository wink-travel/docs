> “如果说我看得更远，那是因为我站在巨人的肩膀上。”

\--*艾萨克·牛顿爵士*

这样的平台并非孤立构建，而是众多新旧组件共同作用的结果。我们希望继续服务于我们的后代。

> “没有必要重新发明破旧的轮子。”

旅游科技碎片化且历史悠久。我们努力汲取旧有技术的精髓。我们整合了一系列渠道管理工具、酒店管理系统等，并添加了一些新功能。此外，我们还将所有内容都融入了行业标准和协议中，这些标准和协议众所周知且易于使用……这样您就无需再费心了。

:::note\[API 驱动方法]
粗略地说，我们为自己构建的任何功能都应该：

* 可以理解
* 灵活的
* 可扩展
* 可重复使用且...
* 可扩展

...其他也想使用这些 API 的人。
:::

Wink 是使用这样的`API-driven approach`正是基于这一设计理念，Wink 的所有功能都可以让外部开发者像使用自己的系统一样使用。与我们集成时，您还可以利用该平台的其他功能。
