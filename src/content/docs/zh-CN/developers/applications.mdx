应用程序可让您完全访问 Wink 和 TripPay 的功能，并允许您的用户使用我们的 OAuth2 授权服务器进行身份验证并登录您的网站；就像 Facebook 和 Google 一样。
您可以使用 Wink 构建自己的 OTA。一切从应用程序开始。

*默认情况下，当您注册新的 Wink 联盟账户或 TripPay 账户时，您会获得一个应用程序。我们使用`client ID`对于某些场景。*

在 Wink 或 TripPay 上生成的应用程序可用于所有 API 端点。
:::

*对于此示例，我们将通过以下方式进行身份验证[Wink工作室](https://studio.wink.travel)。*

<Steps>
  1. 一旦通过身份验证，请单击右上角的个人资料图标，然后会出现一个菜单。
  2. 点击 `Applications` 关联。
  3. 您将被重定向到您的应用程序页面。
</Steps>


## 创建应用程序

由于您无法访问`secret key`我们为您创建的应用程序，您需要创建自己的应用程序。

:::注意
您需要在 Wink 或 TripPay 上创建一个联盟账户，然后再创建应用，因为您将把应用绑定到特定账户。
:::

以下是创建应用程序的步骤：

<Steps>
  1. 在应用程序页面上，单击 `Create new application` 按钮。
  2. ***姓名*** 给你的应用程序命名。 *例如 Cool App*
  3. **实体名称** 将您的应用与您的一个现有帐户关联。 *例如 Cool Account*
  4. **重定向 URI** 至少需要一个重定向 URI。它用于在身份验证成功后验证重定向域。 *例如https://www.cool-site.com*
  5. 点击 `Save` 按钮继续。
</Steps>


您将被重定向回应用列表。您将在列表中看到您的新应用，并且您只能访问一次应用密钥。请务必复制该密钥并将其保存在安全的地方。

## 更新应用程序

如果您的某个重定向 URI 发生更改或者您想要添加或删除 URI，则需要更新您的应用。

要更新现有应用程序，请执行以下操作：

<Steps>
  1. 点击 `Actions` 您要更新的应用程序的链接。
  2. 点击 `Update` 操作下的按钮。
  3. 对应用程序进行更改。
  4. 点击 `Save` 按钮继续。
</Steps>


您将被重定向回您的应用程序列表。

## 删除应用程序

:::注意
请不要删除我们为您自动创建的应用，以确保所有功能在您的帐户下正常使用。
:::

如果您不再使用某个应用程序，您可以继续将其删除。

要删除应用程序，请执行以下操作：

<Steps>
  1. 点击 `Actions` 您要更新的应用程序的链接。
  2. 点击 `Remove` 操作下的按钮。
  3. 点击 `OK` 按钮确认删除。
</Steps>


您将被重定向回您的应用程序列表。
