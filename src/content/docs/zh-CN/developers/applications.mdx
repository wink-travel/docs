应用程序让您可以完全访问 Wink 和 TripPay 的功能，并允许您的用户使用我们的 OAuth2 授权服务器在您的网站上进行身份验证和登录；就像 Facebook 和 Google 一样。
您可以使用 Wink 构建自己的 OTA。它从一个应用程序开始。

*默认情况下，当您注册新的 Wink 联盟账户或 TripPay 账户时，会获得一个应用程序。我们在某些场景中使用 `client ID`。*

:::note
在 Wink 或 TripPay 上生成的应用程序可用于所有 API 端点。
:::

*在本示例中，我们将通过 [Wink Studio](https://studio.wink.travel) 进行身份验证。*

<Steps>
1. 身份验证后，点击右上角的个人资料图标，会出现一个菜单。
2. 点击 `Applications` 链接。
3. 您将被重定向到您的应用程序页面。
</Steps>

## 创建应用

由于您无法访问我们为您创建的应用的 `secret key`，您需要自己创建一个。

:::caution
在创建应用之前，您需要先在 Wink 创建一个联盟账户或在 TripPay 创建一个账户，因为您将把应用绑定到特定账户。
:::

创建应用程序的步骤如下：

<Steps>
1. 在应用页面，点击 `Create new application` 按钮。
2. ***名称*** 给您的应用命名。*例如 Cool App*
3. **实体名称** 将您的应用与您现有的某个账户关联。*例如 Cool Account*
4. **重定向 URI** 至少需要一个重定向 URI。它用于在身份验证成功后验证重定向域。*例如 https://www.cool-site.com*
5. 点击 `Save` 按钮继续。
</Steps>

您将被重定向回应用列表。您会在列表中看到新应用，并且仅此一次可以访问应用的 secret key。请务必复制并妥善保存。

## 更新应用

如果您的某个重定向 URI 发生变化，或者您想添加或删除 URI，则需要更新应用。

更新现有应用的步骤如下：

<Steps>
1. 点击您想更新的应用的 `Actions` 链接。
2. 点击 Actions 下的 `Update` 按钮。
3. 对应用进行修改。
4. 点击 `Save` 按钮继续。
</Steps>

您将被重定向回应用列表。

## 删除应用

:::caution
请不要删除我们自动为您创建的应用，以确保所有功能正常为您的账户服务。
:::

如果您不再需要某个应用，可以将其删除。

删除应用的步骤如下：

<Steps>
1. 点击您想删除的应用的 `Actions` 链接。
2. 点击 Actions 下的 `Remove` 按钮。
3. 点击 `OK` 按钮确认删除。
</Steps>

您将被重定向回应用列表。
