本页列出了您可以在 Wink 上订阅的所有 webhook 事件。有关设置说明，请参阅[Webhook 集成指南](/zh-CN/guides/webhook-integration)或[Webhook 管理页面](/zh-CN/my-account/webhooks)。

## 如何订阅

1. 在您的门户中导航至 **应用 > Webhooks**。
2. 创建一个 webhook 并选择您想接收的事件。
3. 每当事件发生时，Wink 会向您的端点发送 HTTP POST 请求。

有关完整的设置步骤和最佳实践，请参阅[Webhook 集成](/zh-CN/guides/webhook-integration)。

| 头像更新 | `account.profile_picture.updated` | 账户头像已更改。 |

| URL 名称更新 | `account.url-name.update` | 账户自定义 URL 已更改。 |
| 唯一 ID 更新 | `account.unique-id.update` | 账户唯一标识符已修改。 |
