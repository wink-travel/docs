作为开发者，应用程序让您可以访问 Wink 的所有功能。您可以访问所有相同的数据，甚至扩展 Wink 自身的功能。  
您可以选择使用 Wink 的 OAuth2 服务器来验证您的旅游网站，就像使用 Google 或 Facebook 一样。  
这样，您就可以将 Wink 的功能扩展到您自己的网站或应用程序中。

:::note
当您[创建新账户](/zh-CN/my-account/create-account)时，您的第一个应用程序已经创建。我们在某些预订场景中使用该应用程序。

您不能重复使用此应用程序。请根据需要创建新的应用程序。
:::

<Steps>
 1. 从[账户](https://my.wink.travel)中选择您想要操作的账户。
 2. 点击账户卡底部的 `操作`，选择 `管理账户`。
 3. 在接下来的页面中点击 `应用程序` 标签。
 4. 此部分将列出您所有的应用程序。
 5. 选择编辑现有应用程序或创建新的应用程序。
 </Steps>

## 创建应用程序

由于您无法访问我们为您创建的应用程序的 `密钥`，您需要自己创建一个。

:::caution
在创建应用程序之前，您需要先在 Wink 创建一个联盟账户或在 TripPay 创建一个账户，因为您将把应用程序绑定到特定账户。
:::

创建应用程序的步骤如下：

<Steps>
1. 在应用程序页面，点击 `创建新应用程序` 按钮。
2. ***名称*** 给您的应用程序命名。*例如 Cool App*
3. **实体名称** 将您的应用程序与您现有的某个账户关联。*例如 Cool Account*
4. **重定向 URI** 至少需要一个重定向 URI。它用于在成功认证后验证重定向域。*例如 https://www.cool-site.com*
5. 点击 `保存` 按钮继续。
</Steps>

您将被重定向回应用程序列表。您会在列表中看到新应用程序，并且只会这一次看到应用程序的密钥。请务必复制并妥善保存。

## 更新应用程序

如果您的某个重定向 URI 发生变化，或者您想添加或删除 URI，您需要更新应用程序。

更新现有应用程序的步骤如下：

<Steps>
1. 点击您想更新的应用程序的 `操作` 链接。
2. 点击操作下的 `更新` 按钮。
3. 对应用程序进行修改。
4. 点击 `保存` 按钮继续。
</Steps>

您将被重定向回应用程序列表。

## 删除应用程序

:::caution
请勿删除我们自动为您创建的应用程序，以确保您账户的所有功能正常工作。
:::

如果您不再需要某个应用程序，可以将其删除。

删除应用程序的步骤如下：

<Steps>
1. 点击您想删除的应用程序的 `操作` 链接。
2. 点击操作下的 `删除` 按钮。
3. 点击 `确定` 按钮确认删除。
</Steps>

您将被重定向回应用程序列表。
