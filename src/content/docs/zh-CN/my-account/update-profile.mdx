如果您想更新 Wink 用户账户的个人资料，请按照以下步骤操作：

<Steps>
1. 访问 [my.wink.travel](https://my.wink.travel)。
2. 点击右上角的 `登录 / 注册` 进行登录。
3. 登录后，点击右上角的个人资料图标，会出现一个菜单。
4. 点击 `个人资料` 链接。
5. 按照指示更新您的 `电子邮件`、`名字` 和 `姓氏`。
6. 您还可以上传新的 `个人头像`。
7. 点击 `保存`。
</Steps>

<Steps>
1. 您必须已登录我们的网站之一。
2. 点击右上角的个人资料图标，会出现一个菜单。
3. 点击 `个人资料` 链接。
4. 按照指示更新您的 `电子邮件`、`名字` 和 `姓氏`。
5. 您还可以上传新的 `个人头像`。
6. 点击 `更新`。
</Steps>

:::note
您新的个人头像将在您完全退出 Wink 后下次登录时显示。
:::
