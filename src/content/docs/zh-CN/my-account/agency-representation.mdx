代理机构是一个普通的 Wink 账户，您作为业主可以将其分配给您的账户以进行代表。以下是您希望代理机构代表您的几个原因：

- 由第三方对您的 Wink 账户进行全部或部分管理。
- 物业与其管理公司已有协议，且希望 Wink 处理通过 Wink 进来的资金的财务方面。
- 物业管理代理机构代表其物业处理所有在线事务。
- 代表网红/内容创作者的人才代理机构。

## 分配代理机构

以下是如何将代理机构分配给您的账户。

<Steps>
 1. 从 [Accounts](https://my.wink.travel) 中选择您想操作的账户。
 2. 点击账户卡底部的 `Actions`，选择 `Manage account`。
 3. 在接下来的页面点击 `Agency` 标签。
 4. 在 `Agency` 部分，输入您希望代表您的代理机构名称。
 5. 从下拉菜单中选择代理费。
 5. 可选地添加过期规则。
 7. 点击 `Save` 按钮继续。
 </Steps>

 :::note
您必须是物业所有者才能分配代理机构。
 :::

 :::tip[示例]
代理费来源于预订。

预订 = $100，代理费 = 1% => 100 * 0.01 = $1
 :::

## 取消分配代理机构

以下是如何从您的账户取消分配代理机构。

<Steps>
 1. 从 [Accounts](https://my.wink.travel) 中选择您想操作的账户。
 2. 点击账户卡底部的 `Actions`，选择 `Manage account`。
 3. 在接下来的页面点击 `Agency` 标签。
 4. 在 `Agency` 部分，点击代理机构下拉菜单右侧的 <Badge text="x" variant="danger" size="small" /> 图标。
 5. 点击 `Save` 按钮继续。
 </Steps>
