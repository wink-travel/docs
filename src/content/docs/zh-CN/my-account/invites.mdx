Wink 用户可以选择让您管理他们的物业、加盟或旅行社账户。当发生这种情况时，您将收到一封包含邀请请求的电子邮件。

import { Steps } from '@astrojs/starlight/components';

要查看您当前的 Wink 邀请，请按照以下步骤操作：

<Steps>
 1. 登录 [my.wink.travel](https://my.wink.travel)。
 2. 点击左侧边栏中的 `Invites`。
 3. 您将看到所有当前邀请的列表。
 4. 本部分将列出您所有的申请。
 5. 选择 `Accept` 或 `Reject` 邀请。
 6. 如果您接受了邀请，该账户将显示在 `Accounts` 部分。
</Steps>
