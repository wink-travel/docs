您已经连接了 WinkLinks，我们的链接管理工具。点击 `连接社交网络` 以连接更多社交网络。
该过程会重定向您到社交网络，批准 `Wink Social` 应用代表您发布内容。

连接成功后，我们将开始为该网络生成下一个预定月份的内容。

:::note
免费套餐包含 WinkLinks。Facebook、Instagram 及其他平台仅在付费套餐中提供。
:::

## 支持的平台

| 平台 | 描述 | 要求 |
| -- | -- | -- |
| **WinkLinks** | 自动发布到我们的高级链接管理器 | 所有套餐均包含 |
| **Facebook** | 自动发布帖子，回复评论和私信 | 需要 Facebook 商业主页 |
| **Instagram** | 自动发布帖子，回复评论和私信 | 需要与 Facebook 账户关联的 Instagram 商业账户 |

每个连接都会显示当前状态：**激活**、**断开连接** 或 **暂停**。

:::note
我们会为每个社交网络创建并发布优化的内容。
:::
