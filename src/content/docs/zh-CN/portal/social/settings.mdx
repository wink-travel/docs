# 设置

当您首次使用 Social 时，我们会询问您希望我们如何创建内容。这些选项始终可用，您可以随时轻松更改。

从侧边栏导航到 `Preferences` 来管理您的设置。

偏好设置部分包含您的基本内容生成设置：

1. **首选定制** 选择生成内容时使用的定制方案。定制方案控制主题颜色、默认行程、货币、语言等。您可以在您的账户 [my.wink.travel](https://my.wink.travel) 下创建定制方案。
2. **自动批准** 启用后，生成的帖子将自动发布，无需您的批准。当您对我们为您生成的内容质量有信心时，请启用此功能。
3. **审批人姓名** 将收到审批请求邮件的人员或团队名称。
4. **审批人邮箱** 将发送即将发布帖子的审批请求的电子邮件地址。

## 内容创作者

<CldImage

/>
  <figcaption>*图 2 - 内容创作者*</figcaption>

Social 提供了多种内容创作者供您选择。每个创作者专注于使用 AI 为您的物业创建独特内容的特定方式。每个内容创作者都有一个 **权重** 属性——权重越高，生成帖子时被选中的概率越大。

以下是所有可用的内容创作者：

| 创作者 | 描述 |
| -- | -- |
| **目的地** | 利用您的位置创建有关该地区的实用文本 |
| **房型** | 创建包含 AI 生成文本和您的房型图片的可预订帖子 |
| **活动** | 创建包含您某项活动的可预订帖子 |
| **附加服务** | 创建包含您某项附加服务的可预订帖子 |
| **景点** | 创建包含您某个景点的可预订帖子 |
| **地点** | 创建包含您某个地点的可预订帖子 |
| **会议室** | 创建包含您某个会议室的可预订帖子 |
| **餐厅** | 创建包含您某个餐厅的可预订帖子 |
| **水疗** | 创建包含您某个水疗的可预订帖子 |
| **物业** | 创建包含您价格最优房型、AI 生成文本和物业图片的可预订帖子 |
| **用户情感** | 汇总网络上的客人评论，生成情感驱动的帖子，突出客人对您物业的喜爱 |
| **幽默** | 创建轻松有趣的帖子，保持您的动态吸引力 |

:::tip[专业提示]
确保您拥有所选内容创作者所需的库存类型。例如，如果您选择“餐厅”，则需要在您的物业中至少有一家餐厅。
:::

## 语气风格

<CldImage

  <figcaption>*图 3 - 语气风格*</figcaption>

语气风格设定了 AI 生成内容时使用的语言风格。选择与您的品牌个性相匹配的语气。与内容创作者一样，每种语气都有一个 **权重**，用于控制其使用频率。

以下是可用的语气：

| 语气 | 风格 |
| -- | -- |
| **中性** | 冷静、权威、事实性。无夸张，无笑话——只是可靠、结构良好的知识 |
| **友好** | 温暖、亲切、文字中带笑意，使用随意缩写和偶尔的表情符号 |
| **奢华** | 精致、优雅。使用“精美”、“定制”等词汇，让读者感受尊贵 |
| **美食家** | 感官丰富、令人垂涎的描述。美食诱惑语言让读者食欲大开 |
| **冒险** | 大胆、无畏、探险者气质。生动的感官描述让一切听起来史诗般 |
| **家庭** | 健康、适合儿童的兴奋感。温暖的父母气息，讲述安全有趣的体验 |
| **浪漫** | 诗意、怀旧、心动。夕阳、柔和语言和爱情 |
| **幽默** | 讽刺、快速的单句笑话、巧妙的文字游戏。读起来像脱口秀 |
| **环保** | 认真且正义。赞美绿色替代方案和可持续选择 |
| **本地内行** | 极具地方特色的本地参考，提醒避开游客陷阱，使用本地俚语 |
| **包容** | 欢迎、庆祝和肯定。让每个人都感到被看见和重视 |
| **神秘** | 空灵、精神、宇宙真理。温柔地邀请内省 |
| **未来感** | 时尚、前瞻。喜欢“范式转变”和乐观未来愿景等词汇 |
| **经济实惠** | 精明、足智多谋、自豪地节俭。赞美实惠发现和免费体验 |

## 帖子类型

帖子类型决定生成内容的目标和结构。每种帖子类型都有一个 **权重**，用于控制其使用频率。

| 帖子类型 | 描述 |
| -- | -- |
| **交易型** | 清晰、简洁、注重转化。通过事实、价格和后续步骤推动预订和直接行动 |
| **认知型** | 品牌建设内容，旨在介绍您的物业并提高潜在客人的知名度 |
| **情感型** | 情感共鸣内容，创造积极联想并与您的受众建立情感联系 |

您可能希望将营销重点放在特定语言和地区。告诉我们您想为哪些语言创建内容以及您感兴趣的时区，我们将完成其余工作。帖子将在您选择的时区的预定时间发布。

*大多数物业使用单一语言，并默认使用其本地语言。*

## Google 商家

<figure>
<CldImage
  src="academy/social_google_business"

通过连接您的 Google 商家资料，我们可以提取更丰富的物业详情，以生成更有效的内容。您的物业需要有经过验证的 Google 商家资料，才能从您的列表中生成内容。部分内容创作者**要求**您拥有 Google 商家账户。

:::tip[专业提示]
从小做起

首次尝试时不要过于激进。我们建议您先为您的物业创建单一语言的内容。
:::
