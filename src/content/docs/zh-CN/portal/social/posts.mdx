帖子编辑器允许您创建无限制的手动帖子，同时也是管理自动为您创建的帖子的地方。

编辑器有4个部分：

1. 内容部分。
2. 媒体部分。
3. 链接部分。
4. 设置部分。

## 内容

内容部分允许您在发布前修改文本内容。您可以创建多语言的本地化内容——每种语言都有自己的标题和描述。

## 媒体

媒体部分允许您修改与文本配套的图片和视频。

## 链接

链接部分允许您设置用户阅读帖子后希望跳转的链接。*例如，这可以是指向特价优惠或获取更多信息的链接。*您可以添加链接 URL、链接标题和链接描述。

## 设置

设置部分允许您针对每个帖子修改/覆盖所有默认设置。您可以设置：

- **平台** — 选择此帖子将发布到哪个社交网络。
- **帖子类型** — 选择交易型、认知型或情感型。
- **语气** — 覆盖此特定帖子的语气。
- **定制** — 使用不同的定制设置，如显示货币、语言、行程等。
- **时区** — 覆盖发布时间的时区。
- **计划日期** — 自动生成的帖子会在最佳时间自动安排发布。您可以覆盖日期和时间以选择不同的时间段。

## 帖子状态生命周期

每个帖子遵循状态生命周期：

| 状态 | 描述 |
| -- | -- |
| **草稿** | 帖子已创建但尚未准备好审核 |
| **待审批** | 帖子等待人工审批后安排发布 |
| **已批准** | 帖子已获批准，将被安排发布 |
| **已拒绝** | 帖子已审核并被拒绝——不会发布 |
| **已安排** | 帖子已排队，等待在计划的日期和时间发布 |
| **已发布** | 帖子已成功发布到社交网络 |
| **已暂停** | 帖子发布已被暂时暂停 |
| **失败** | 由于错误，帖子无法发布 |

## 审批帖子

帖子在帖子列表中审批，您可以选择一个或多个帖子并选择批准或拒绝。

:::note
每种语言和每个社交网络都会有一条记录。

例如，如果您选择了3个社交网络和3种语言，日历上将显示9条记录。
:::
