订阅页面让您管理您的计划以及账户下授权的物业。

## 可用计划

浏览通过我们的价格表展示的可用订阅等级。每个计划包含不同的功能集和物业限制。选择最适合您需求的计划。

## 当前订阅

如果您已有有效订阅，本部分将显示您当前计划的详细信息和状态。

## 许可证

许可证表显示您订阅下所有已配置的物业。您可以：

- 查看当前已授权的物业。
- 使用 `Add license` 按钮添加新的物业许可证。
- 通过 Stripe 客户门户链接管理账单。

:::note
账单管理（支付方式、发票、计划变更）通过 Stripe 客户门户处理。
:::
