要管理您的 WinkLinks 标签，请导航到主导航栏中的 `WinkLinks`，然后点击 `Tags` 选项卡。

随着内容的增长，您会想开始对部分内容进行组织。我们为此创建了 `tags`。您可以使用任意数量的标签对内容进行分组。

## 创建标签

<Steps>
1. 点击 `Add a new tag` 按钮创建一个新标签。
2. 输入新标签的名称。*例如：柏林顶级咖啡馆*
3. 点击 `Save` 链接。
</Steps>

## 更新标签

<Steps>
1. 首先点击您想修改的标签旁边的 `Edit` 链接。
2. 更新标签名称。*例如：柏林最佳咖啡馆*
3. 点击 `Save` 链接。
</Steps>

## 删除标签

:::note
删除标签时，该标签也会从所有关联内容中移除。
:::

<Steps>
1. 点击您想删除的标签旁边的 `Remove` 链接。
</Steps>

## API

想要管理 `WinkLinks` 的开发者可以前往 [Developers > API > WinkLinks](/zh-CN/developers/apis/#winklinks-api)。
