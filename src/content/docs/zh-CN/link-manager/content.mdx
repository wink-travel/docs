您的内容可在`Entries`选项卡。

我们支持三种主要类型的内容：

* 链接到网页的公开 URL。
* 通过 WinkLinks 上传的文件。
* 通过我们的创建的文本内容`Quotes` 特征。

## 添加内容

以下是向 WinkLinks 添加内容的方法。在`Entries`选项卡，首先单击`Add entry` 按钮。

### 正常链接

:::提示
链接必须指向网页。例如，您不能直接链接到 PDF 文件。如果您想分享 PDF 文件，您需要使用`Upload file` 特征。
：：：

以下是添加普通链接的步骤：

<Steps>
  1. 输入您想要分享的链接。 *例如https://trvl.as/s/3xWCH*
  2. 一旦您输入链接，Wink 就会寻找该网站的详细信息并填充您接下来看到的表单。
  3. 您可以选择修改一些数据，例如站点名称、标题和描述。
  4. 您还可以选择上传自定义图像。
  5. 点击 `Save` 准备好继续时，请按下按钮。
</Steps>


### 上传文件

:::警告\[要求]

* 图像大小不能超过 10Mb。
* 视频大小不能超过 50Mb。
* 我们支持所有公认的图像和视频格式，包括 PDF。
  :::

:::note\[建议]

* 您的特色图片尺寸为 2560w x 1440h（16x9 宽高比）。
* 所有其他图像均为 2560w x 1920h（4:3 宽高比）。
* 图片大小不得超过 5Mb。上传至 Wink 前请先压缩图片。
  :::

以下是添加文件的步骤：

<Steps>
  1. 可以添加媒体：
     * 通过将图像从计算机拖到窗口或单击 `Browse`。
     * 通过粘贴指向图像的外部 URL。
     * 通过使用笔记本电脑摄像头。
     * 来自您的 Google Drive 帐户。
     * 来自您的 Dropbox 帐户。
     * 来自 Shutterstock
     * 图片来源：Gettyimages
     * 来自 iStock
     * 来自 Unsplash
  2. 文件上传完成后，您可以添加更多元数据来描述您的文件，例如标题和描述。
  3. 点击 `Save` 准备好继续时，请按下按钮。
</Steps>


### 撰写报价

:::提示
引用是分隔长列表的好方法 \[……而且它还能让你听起来更聪明、更博学 😉]。
:::

以下是编写报价单的步骤：

<Steps>
  1. 输入引文 *例如，玫瑰是红色的。紫罗兰是蓝色的……*。
  2. 输入原作者的姓名。 *例如埃德蒙·斯宾塞*
  3. 输入您获得报价的网站名称。 *例如每日行情*
  4. 点击 `Save` 准备好继续时，请按下按钮。
</Steps>


## 更新内容

所有内容都可以通过点击`Actions`链接并选择`Update`。您可以通过类似的方式删除内容，只需点击`Remove` 按钮。

## 删除内容

所有内容都可以轻松修改选择`Update`。您可以用类似的方式删除内容，首先点击`Actions`链接，然后点击`Remove` 按钮。

## 内容排序

您可以通过单击每个条目左下角的图标并将其拖动到您想要的位置来对内容进行排序。

## 分组内容

:::提示
在创建条目之前先创建标签。否则，您将丢失进度并需要重新开始。
:::

编辑条目时，您可以选择添加一个或多个标签以便分组。滚动到表单底部，找到`Tags`表单字段。

用户在查看您的 WinkLinks 页面时，可以选择如何使用您的内容（列表或网格），并可以选择一个或多个标签来进一步过滤您的内容。

## API

想要管理的开发人员`WinkLinks`可以前往[开发者 > API > WinkLinks](/developers/apis/#winklinks-api)。
