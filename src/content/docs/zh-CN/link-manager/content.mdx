您的内容可在 `Entries` 标签下查看。

我们支持三种主要内容类型：

- 公开可用的链接，指向网页。
- 通过 WinkLinks 上传的文件。
- 通过我们的 `Quotes` 功能创建的文本内容。

## 添加内容

以下是向 WinkLinks 添加内容的方法。在 `Entries` 标签下，首先点击 `Add entry` 按钮。

### 普通链接

:::tip
链接必须指向网页。例如，您不能直接链接到 PDF。如果您想分享 PDF，需要使用 `Upload file` 功能。
:::

添加普通链接的步骤如下：

<Steps>
1. 输入您想分享的链接。*例如 https://trvl.as/3xWCH*
2. 输入链接后，Wink 会自动获取该网站的详细信息并填充您接下来看到的表单。
3. 您可以选择修改部分数据，如网站名称、标题和描述。
4. 您也可以选择上传自定义图片。
5. 准备好后，点击 `Save` 按钮继续。
</Steps>

### 上传文件

:::caution[要求]
- 图片大小不得超过 10Mb。
- 视频大小不得超过 50Mb。
- 我们支持所有常见的图片和视频格式，包括 PDF。
:::

:::note[建议]
- 特色图片建议尺寸为 2560w x 1440h（16:9 长宽比）。
- 其他图片建议尺寸为 2560w x 1920h（4:3 长宽比）。
- 图片大小不应超过 5Mb。请在上传到 Wink 前压缩图片。
:::

添加文件的步骤如下：

<Steps>
1. 媒体可以通过以下方式添加：
    - 将电脑中的图片拖拽到窗口，或点击 `Browse`。
    - 粘贴指向图片的外部 URL。
    - 使用笔记本电脑摄像头。
    - 从您的 Google Drive 账户。
    - 从您的 Dropbox 账户。
    - 从 Shutterstock。
    - 从 gettyimages。
    - 从 iStock。
    - 从 Unsplash。
2. 文件上传完成后，您可以添加更多元数据来描述文件，如标题和描述。
3. 准备好后，点击 `Save` 按钮继续。
</Steps>

### 撰写引用

:::tip
引用是分隔长列表的好方法[...而且还能让你显得聪明且博学 😉]。
:::

撰写引用的步骤如下：

<Steps>
1. 输入引用内容 *例如 Roses are red. Violets are blue...*。
2. 输入原作者姓名。*例如 Edmund Spenser*。
3. 输入引用来源网站名称。*例如 Daily Quotes*。
4. 准备好后，点击 `Save` 按钮继续。
</Steps>

## 更新内容

所有内容都可以通过点击 `Actions` 链接并选择 `Update` 轻松修改。您也可以通过点击 `Remove` 按钮以类似方式删除内容。

## 删除内容

所有内容都可以通过选择 `Update` 轻松修改。您也可以先点击 `Actions` 链接，然后点击 `Remove` 按钮以类似方式删除内容。

## 内容排序

您可以通过点击每个条目左下角的图标并拖动到您想要的位置来排序内容。

## 内容分组

:::tip
请先创建标签，再创建条目。否则，您将丢失进度，需要重新开始。
:::

编辑条目时，您可以选择添加一个或多个标签以便分组。滚动到表单底部找到 `Tags` 表单字段。

访问您 WinkLinks 页面时，用户可以选择以列表或网格方式浏览内容，并可选择一个或多个标签进一步筛选内容。

## API

想要管理 `WinkLinks` 的开发者可以前往 [Developers > API > WinkLinks](/developers/apis/#winklinks-api)。
