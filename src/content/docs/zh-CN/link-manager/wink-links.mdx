链接管理器让你保存和整理喜欢的网址以备后用。浏览器书签就是一种链接管理器。随着社交媒体的兴起，人们更需要向朋友和观众展示和分享链接。你的 Facebook 动态就是一个社交友好的链接管理器示例。

我们打造了 [WinkLinks](https://i.trvl.as/) 作为旅游行业的链接管理器。它用于传递信息、激发兴趣、做广告……并让你预订全球各地酒店的超值优惠。

## 功能

- WinkLinks 允许你免费添加所有现有的联盟链接，如 Commission Junction 和 Amazon。
- 你将获得一个酷炫的个性化网址，例如 [https://i.trvl.as/bob](https://i.trvl.as/bob)，适合只能分享单一链接的场合。
- 你可以整理和分享链接，还能上传文件。
- 酒店、连锁和品牌可以使用 WinkLinks 作为高级预订引擎：
    - 支持多家物业。
    - 支持多个目的地。
    - 支持多种货币。
    - 支持多种语言。
- WinkLinks 支持 `丰富内容` *(见下文)*。
- 从 IG 到预订，只需 2 次点击 🚀

## 丰富内容

:::note
在你的[设置](/zh-CN/link-manager/settings)中将 `智能` 设为 🟢 以使用此功能。
:::

如果你分享来自支持 `丰富` / `oEmbed` 内容的网站的链接，WinkLinks 会让你的链接变得生动。

支持的网站：

- Facebook - *可互动帖子并播放视频*。
- Instagram - *可互动帖子并播放视频*。
- YouTube - *播放视频*。
- X（前 Twitter）- *可互动帖子*。
- Spotify - *播放你喜欢的播放列表*。
- Wink - *见 `高级`*。

## 高级

WinkLinks 也是展示你通过 [Wink Studio](https://studio.wink.travel) 发现并想向观众推广的旅游库存的绝佳方式。

使用 WinkLinks 销售你的：

- [可分享链接](/zh-CN/studio/shareable-links) - 可选择展示为：
    - [广告横幅](/zh-CN/developers/web-components/#content-loader)。
    - 带实时价格更新的普通链接。
- [卡片](/zh-CN/studio/cards) - 可预订的卡片，带实时价格和可用性。
- [网格](/zh-CN/studio/grids) - 带可预订卡片的网格。
- [地图](/zh-CN/studio/maps) - 带标记的地图，标记可打开可预订卡片。
