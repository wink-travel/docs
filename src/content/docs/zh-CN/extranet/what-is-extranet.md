[Wink Extranet](https://extranet.wink.travel) 是我们的物业门户。它面向酒店、连锁、品牌和酒店管理公司，用于创建和管理其在 Wink 平台上的数字物业档案。

我们的 Extranet 经过精心设计，尽可能简洁，同时支持高级用户期望的所有功能。举几个例子：

1. 简单：使用我们的智能入驻流程，在不到一小时内完成物业入驻并上线。
2. 高级：Wink 支持 8500 种不同的取消政策变体。
3. 酷炫：让地球上某个人享受 10% 折扣，无需用户登录或输入促销代码。

让我们的向导引导您完成整个流程。当您觉得足够熟悉时……就可以独立操作了。

Wink 是一个面向物业和体验提供商的高级内容管理系统。您在我们的任何应用中能做的事情，都可以通过 API 实现。这使您能够与 Wink 深度集成，甚至无需登录任何我们的应用即可通过 Wink 销售。

:::tip[专业提示]
您是否正在通过 Leonardo International 支付昂贵的媒体资产托管费用？您可以通过 Wink 完全实现相同功能，并且获得更多选项来操作和微调您的图片和视频。
:::

所有这些功能对您都是免费的。只有在您完成预订时，我们才会收取少量费用。没有风险——尽享收益。
