**住宿提供商通用条款和条件**

通过注册并加入 Wink 计划成为住宿提供商，住宿提供商特此已审阅、理解、确认并接受本住宿提供商协议（“协议”）的条款和条件。

**双方：**

**TRAVELIKO SINGAPORE PTE. LTD.**，一家根据新加坡法律注册成立的公司，注册地址为 \#03-01 Wilkie Edge, 8 Wilkie Road, Singapore 228095，增值税注册号为 201437335D（“Wink”），以及

**住宿提供商**，其详细信息载于住宿提供商注册表格或已在线提交（“住宿提供商”）。

Wink 与住宿提供商各为本协议的“当事方”，合称“双方”。

本文件确立以下条款和条件：

1. 通过 Wink 或住宿提供商用于分销其产品的任何其他方式提供住宿分销服务（以下简称“住宿提供商”），其详细信息载于付款条款中，价格、条件和可用性已达成一致；及  
2. 住宿提供商向最终消费者/客人提供的住宿服务，该服务通过 Wink 预订。

Wink 不拥有、控制、提供或管理任何房源。Wink 不是住宿提供商与客人之间直接签订合同的当事方。除付款服务条款（“付款条款”）中规定外，Wink 不以任何身份作为住宿提供商的代理人。

若本条款和付款条款之间存在不一致，以付款条款为准。

因此，双方达成如下协议：

**1. 定义**

除本协议其他部分定义外，以下定义适用于本协议，除非另有相反意图：

**“住宿提供商”** 指任何在 Wink 创建账户，意图通过 Wink 平台销售其自有房间及附属服务库存的当事方。

**“协议”** 指本协议。

**“最佳可用价”或“BAR”** 指参与酒店、住宿提供商或其代表的任何第三方分销商向公众提供的含增值税的最低未折扣、可佣金计费的房价。为避免疑义，促销价、节假日价及任何其他公开无限制价格均包含在最佳可用价内。

**“预订”** 指通过 Wink 或 Wink 客户发出的房间预订请求，并被住宿提供商接受。

**“预订费”** 指 Wink 从预订金额中扣除的 1.5% 作为处理费。

**“预订金额”** 指支付服务商从客人处收取的预订总金额。

**“缺房”** 指住宿提供商因参与酒店或其代表的住宿提供商或任何第三方分销商无房可用而无法接待客人的情况。

**“佣金”** 指根据本协议，因每笔实际交易应付给合作伙伴的金额。

**“设施与服务”** 指参与酒店提供的任何设施、餐饮、便利设施及/或其他服务。

**“客人”** 指因直接或间接通过 Wink 预订而使用、正在使用或最终将使用房间（及适用的其他设施与服务）的最终用户。

**“酒店”** 指通过 Wink 网站提供的任何住宿。

**“Wink 平台”** 指双方为安全共享信息而使用的私有网络或技术解决方案。

**“知识产权”** 指全球范围内所有性质的知识产权（无论是否以文档形式记录，或存储于任何磁性或光学介质或存储器中），包括专利、实用新型、商标、注册设计和域名、上述任何申请、商号、商誉、版权及类似版权的权利、设计权、数据库权利、精神权利、专有技术及其他知识产权，涵盖计算机软件、程序、网站、文件、信息、技术、商业方法、图纸、标识、使用手册、客户名单、营销方法及广告资料，包括任何网站的“外观与感觉”。

**“实际交易”** 指客人与住宿提供商的预订，且该预订已导致实际提供住宿，住宿提供商已向 Wink 确认。实际交易将根据修改（如缩短住宿）、拒付、信用卡欺诈、坏账等进行调整。为避免疑义，取消、未入住等不视为实际交易。

**“净价”** 指双方书面同意 Wink 向每个参与酒店支付的房价。

**“未入住”** 指客人未按预订及时抵达参与酒店的任何情况。

**“合作伙伴”** 指任何连接或使用 Wink 平台向其受众和/或客户推广并销售住宿提供商库存以获取佣金的企业或个人。

**“支付服务商”** 指 Traveliko Singapore Pte. Ltd. 的全资子公司（“TripPay”），负责支付服务，从客人处收款（“支付入账”），通过其购买关联的支付方式（如信用卡、借记卡、银行转账、加密货币或 PayPal 等）扣款，并向住宿提供商支付应付资金（“净价”）。

**“支付入账”** 指支付服务商从客人处收取的付款。

**“支付出账”** 指支付服务商向合作伙伴支付净佣金。

**“支付服务费”** 指支付服务商从合作伙伴佣金中扣除的 4% 支付收取费。

**“付款条款”** 指双方就房价、可用性、优惠、促销、付款条款及任何其他与房间分销相关的规则或条件达成的协议。

**“潜在欺诈预订”** 指 (i) 预订时向 Wink 提供无效或错误信息，或因信用卡争议或未经授权收费报告导致的预订；或 (ii) 可能与先前高风险或欺诈交易相关的任何预订。

**“价格”** 指住宿提供商向 Wink 提供的适用于任何房型或价格类型的酒店房价，用于在所有 Wink 渠道分销。

**“房间”** 指位于任何参与酒店或住宿处的住宿。

**“税费”** 指所有地方、州、联邦及国家税费及/或服务费，包括但不限于增值税（VAT）、销售税、使用税、消费税、住宿税、过境税、租赁税、市政税、度假区税及其他类似税费、政府费用或收费。

**“TripPay”** 指 Wink 的全资子公司，负责收款（支付入账）和付款（支付出账），即支付服务商。

**“销售渠道”** 指 (i) Wink 运营、管理或拥有的供其他企业使用以进行预订且仅凭 Wink 发放密码访问的网站；(ii) Wink 与其客户旅游网站之间的 API 连接；或 (iii) Wink 向客户供应房间以供其进一步分销或销售的任何其他分销方式，无论是网站还是其他形式。

1.1.2 除非 Wink 书面同意或本协议另有规定，住宿提供商不得在其网站上发布任何明示或暗示该网站为 Wink 部分、获 Wink 认可或为 Wink 官方网站的声明。

当您通过 Wink 平台收到预订确认时，即表示您与客人直接签订合同，并负责按照预订确认中指定的条款和价格提供服务。您还同意在适用时支付付款条款中定义的相关费用。

**3. 住宿提供商的独立性**

您与 Wink 的关系为独立法律实体，除 TripPay 作为付款服务商外。Wink 不指挥或控制您提供的服务，您同意完全自主决定是否、何时提供服务，以及以何种价格和条款提供。

**4. 管理您的房源**

作为住宿提供商，Wink 为您提供通过我们专有销售渠道在线销售库存所需的工具。

住宿提供商负责随时保持其房源信息和内容的准确和最新。住宿提供商应遵守本协议约定的可用性、价格及其他承诺。住宿提供商独自负责通过 Wink 后台更新并保持可用性、价格及适用的地方税费（如有）最新。住宿提供商特此允许 Wink 在所有市场推广房间。Wink 强烈建议住宿提供商始终提供最具竞争力的价格、促销和优惠。

**5. 法律义务**

5.1 住宿提供商负责了解并遵守适用于其房源的所有法律、规则、法规及与第三方签订的合同。

5.2 住宿提供商负责按照适用隐私法律及本条款处理和使用客人及其他人的个人数据。

Wink 为住宿提供商提供数字工具，通过 5 个专有渠道分销和销售库存。通过仪表盘上的自助激活功能激活您的物业后，您的物业将自动可在 Traveliko.com 及网络中预订。您也可在后台的“分销 - 销售渠道”菜单中手动停用这些渠道。

任何确认预订适用以下预订费、佣金及支付费用：

* Traveliko.com：支付服务商费用（付款条款中描述，4%）+ Wink 预订费（1.5%）  
* WinkLinks：支付服务商费用（4%）+ Wink 预订费（1.5%）  
* 社交分享：支付服务商费用（4%）+ Wink 预订费（1.5%）  
* 预订引擎：支付服务商费用（4%）+ Wink 预订费（1.5%）

**7. 税费**

7.1 住宿提供商负责确保价格包含所有适用税费。住宿提供商应确保价格中包含的税费准确且最新。若住宿提供商未指明需支付的城市税、旅游费或其他地方费用，则视为已包含在价格内。

7.2 每个住宿提供商负责向相关政府机构缴纳税费。

7.3 住宿提供商应对未准确包含税费及未及时通知 Wink 相关税费所导致的任何损失、费用、罚款及/或赔偿承担责任，并同意赔偿 Wink。

7.4 Wink 向合作伙伴提供房间时应包含所有适用税费（受本条款第(a)款限制）。除非另有说明，本协议项下所有应付金额均含增值税或其他适用税费（公司税或利润税除外）。若相关税务机关对提供服务所在国征收增值税，Wink 在收到住宿提供商有效税务发票后，将支付含增值税金额。

7.5 相关机关因延迟缴纳税费或发票增值税产生的罚款或利息应由住宿提供商承担。若相关机关根据自我申报机制征收增值税，Wink 将根据当地增值税法规在其增值税申报中申报该税。因延迟缴纳自我申报增值税产生的罚款或利息由 Wink 承担。

**8. 银行信息及住宿提供商指定代表**

住宿提供商应确保向 Wink 提供的银行账户信息始终准确，并应及时通知任何变更。

仅签署本协议的人为住宿提供商唯一指定代表，有权请求更改银行账户及/或收款人信息。其他任何人无此权限。指定代表变更须书面通知 Wink，且仅通过双方书面协议修订本协议后生效。

**9. 预订修改**

9.1 住宿提供商对超出预订确认中取消政策范围的任何预订修改负责，需与客人直接沟通。

9.2 若住宿提供商上传错误价格导致预订以错误价格完成，住宿提供商应按该错误价格履约。

9.3 Wink 对住宿提供商的价格错误不承担任何责任。

**10. 预订识别 / 反欺诈合作**

10.1 住宿提供商应确保其系统准确录入 Wink 客户的完整正确公司信息，以便每笔预订均可明确识别为本协议项下预订。

10.2 每个参与住宿提供商应尽商业合理努力确保客人入住时所出示身份证明与预订信息一致。若预订为潜在欺诈预订，或客人提供数据无法由 Wink 验证，住宿提供商与 Wink 应合作处理，包括随时取消该预订。住宿提供商同意全面配合 Wink 并提供所需信息。

10.3 若住宿提供商未遵守本条款，且预订被认定为潜在欺诈预订，Wink 不承担责任。

**11. 无房 / 未履约**

11.1 若因超额预订或其他原因导致无房，住宿提供商应尽最大努力维持客人及预订，使客人尽可能留在原住宿。若必须将客人调至其他住宿，住宿提供商同意：

* 事先立即通知 Wink；  
* 将客人调至同区域内同等或更高等级的替代物业，条件与原预订相同，且调房费用由住宿提供商承担，且不向客人额外收费，包括但不限于交通及其他调房相关费用；  
* 若 Wink 因无房或调房产生任何财务罚款、赔偿或费用，Wink 有权从住宿提供商应收款中扣除所有直接费用。

11.2 若住宿提供商未履行本协议任何条件，包括场所条件不可接受及设施与服务不足，Wink 有权停止付款并要求正确履行预订条款。住宿提供商应承担因未履约向受影响客人支付的全部赔偿费用，Wink 不承担因住宿提供商未履约产生的任何责任。

**12. 参与酒店物业状况**

住宿提供商应尽早通过 Wink 后台“公告区”通知任何施工、装修、翻新、更新或其他可能影响提供房间、设施与服务能力或影响客人住宿的工作。若无房或未履约因上述情况导致，适用本条款第 11 节（“无房 / 未履约”），住宿提供商承担全部赔偿费用，Wink 不承担责任。

**13. 修改、期限及终止**

13.1 修改。除适用法律另有规定外，Wink 可随时修改本条款。若有重大变更，Wink 将在 Wink 平台发布修订条款并更新“最后更新”日期。受影响方将至少提前三十（30）天收到变更通知。若您未在修订条款生效前终止协议，继续使用服务即视为接受修订条款。

13.2 期限。除非另有约定，本协议自签署日起生效，期限不定。

13.3 本协议持续有效，直至双方按以下方式终止：

* Wink 可随时无因提前三十（30）日书面通知对方终止；  
* 住宿提供商可随时无因通过后台停用物业终止，无需通知；  
* 本协议其他明确规定的方式。

13.4 Wink 保留自行决定权，若发生以下情况，可立即书面通知住宿提供商终止协议：

* 住宿提供商进入破产、清算、司法管理等程序，且六十（60）天内未获驳回或解决；  
* 债权人对参与酒店执行或尝试执行抵押；  
* 住宿提供商停止正常经营；  
* 住宿提供商失去参与酒店的租赁权或品牌运营权；  
* 住宿性质发生变化，包括但不限于施工、装修；  
* 因不可抗力事件。

13.5 协议提前终止或期满时，住宿提供商应：

* 履行终止或期满前所有预订，按原预订价格，包括终止或期满后入住的预订，或提供合适替代服务；  
* 结清相关账目。
