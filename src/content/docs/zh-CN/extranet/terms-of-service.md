**住宿提供商通用条款和条件**

通过注册并加入 Wink 计划成为住宿提供商，住宿提供商特此已审阅、理解、确认并接受本住宿提供商协议（“协议”）的条款和条件。

**双方：**

**TRAVELIKO SINGAPORE PTE. LTD.**，一家根据新加坡法律注册成立的公司，注册地址为 \#03-01 Wilkie Edge, 8 Wilkie Road, Singapore 228095，增值税注册号为 201437335D（“Wink”），以及

**住宿提供商**，其详细信息载于住宿提供商注册表或已在线提交（“住宿提供商”）。

Wink 与住宿提供商各为本协议的“当事方”，合称“双方”。

本文件确立以下条款和条件：

1. 通过 Wink 或住宿提供商用于分销其产品的任何其他方式提供住宿分销服务（以下简称“住宿提供商”），其详细信息载于付款条款，价格、条件和可用性已达成一致；及  
2. 住宿提供商向最终消费者/客人提供的住宿服务，该服务通过 Wink 预订。

Wink 不拥有、控制、提供或管理任何房源。Wink 不是住宿提供商与客人之间直接签订合同的当事方。除付款服务条款（“付款条款”）中规定外，Wink 不以任何身份作为住宿提供商的代理人。

若本条款和付款条款（以下简称“条款”）存在不一致，以付款条款为准。

**1. 定义**

除本协议其他部分定义外，以下定义适用于本协议，除非另有相反意图：

**“住宿提供商”** 指任何在 Wink 创建账户，意图通过 Wink 平台销售其自有房间及附属服务库存的当事方。

**“协议”** 指本协议。

**“最佳可用价”或“BAR”** 指参与酒店、住宿提供商或其代表的任何第三方分销商向公众提供的含增值税的最低未折扣、可佣金计费的房价。为避免疑义，促销价、节假日价及其他任何公开无限制价格均包含在最佳可用价内。

**“预订”** 指通过 Wink 或 Wink 客户发出的房间预订请求，并被住宿提供商接受。

**“预订费”** 指 Wink 从预订金额中扣除的 1.5% 处理费。

**“预订金额”** 指支付服务商从客人处收取的预订总金额。

**“缺房”** 指住宿提供商因房间无可用性等原因未能接待客人。

**“佣金”** 指根据本协议应付给合作伙伴的每笔实际交易金额。

**“设施与服务”** 指参与酒店提供的任何设施、餐饮、便利设施及/或其他服务。

**“客人”** 指因直接或间接通过 Wink 预订而使用、正在使用或最终将使用房间（及相关设施与服务）的终端用户。

**“酒店”** 指通过 Wink 网站提供的任何住宿。

**“Wink 平台”** 指双方为安全共享信息而使用的私有网络或技术解决方案。

**“知识产权”** 指全球范围内所有形式的知识产权，包括专利、实用新型、商标、注册设计、域名、商业名称、商誉、版权及类似权利、设计权、数据库权、道德权、专有技术及其他知识产权，无论是否以文档形式记录或存储于任何介质。

**“实际交易”** 指客人与住宿提供商的预订，且该预订已实际提供住宿服务，并由住宿提供商确认给 Wink。实际交易会根据修改（如缩短住宿）、拒付、信用卡欺诈、坏账等进行调整。取消、未入住等不视为实际交易。

**“净价”** 指双方书面同意 Wink 向参与酒店支付的房价。

**“未入住”** 指客人未按预订及时抵达参与酒店。

**“合作伙伴”** 指通过 Wink 平台推广并销售住宿提供商库存给其受众和/或客户以赚取佣金的企业或个人。

**“支付服务商”** 指 Traveliko Singapore Pte. Ltd. 的全资子公司（“TripPay”），负责支付服务，从客人处收款（“支付入账”），通过信用卡、借记卡、银行转账、加密货币或 PayPal 等方式扣款，并向住宿提供商支付应付金额（“净价”）。

**“支付入账”** 指支付服务商从客人处收取的付款。

**“支付出账”** 指支付服务商向合作伙伴支付净佣金。

**“支付服务费”** 指支付服务商从合作伙伴佣金中扣除的 4% 支付获取费。

**“付款条款”** 指双方就房价、可用性、优惠、促销、付款条款及其他分销房间相关规则达成的协议。

**“潜在欺诈预订”** 指 (i) 预订时向 Wink 提供无效或错误信息，或因信用卡争议、未经授权收费报告导致的预订；或 (ii) 可能与之前高风险或欺诈交易相关的预订。

**“房价”** 指住宿提供商向 Wink 提供的适用于任何房型或房价类型的酒店房价，用于在所有 Wink 渠道分销。

**“房间”** 指位于任何参与酒店的住宿。

**“税费”** 指所有地方、州、联邦及国家税费及/或服务费，包括但不限于增值税（VAT）、销售税、使用税、消费税、住宿税、过境税、租赁税、市政税、度假区税及其他类似税费、政府费用或收费。

**“TripPay”** 指 Wink 的全资子公司，负责收款（支付入账）和付款（支付出账），即支付服务商。

**“销售渠道”** 指 (i) Wink 运营、管理或拥有的供其他企业使用的预订网站，仅限 Wink 发放密码访问；(ii) Wink 与客户旅游网站之间的 API 连接；或 (iii) Wink 向客户提供房间以供其进一步分销或销售的任何其他分销方式，无论是网站还是其他形式。

**1.1 无合伙关系**

1.1.1 本协议不旨在且不应被解释为在双方之间创建合资企业、合伙关系或委托代理关系。除非双方书面同意，任何一方不得 (i) 以另一方代理身份与第三方签订合同或承诺，(ii) 自称或以任何方式表现为该代理，或 (iii) 代表另一方行事或作任何形式的代理。

1.1.2 除非 Wink 书面同意或本协议另有规定，住宿提供商不得在其网站上发布任何明示或暗示该网站为 Wink 官方网站或受 Wink 认可的声明。

**2. 与客人签订合同**

当您通过 Wink 平台收到预订确认时，即表示您与客人直接签订合同，并负责按照预订确认中指定的条款和价格提供服务。您还同意在适用时支付付款条款中定义的相关费用。

**3. 住宿提供商的独立性**

您与 Wink 的关系为独立法律实体，除 TripPay 作为支付服务商外。Wink 不指挥或控制您提供的服务，您同意完全自主决定是否、何时提供服务，以及以何种价格和条款提供服务。

**4. 管理您的房源**

作为住宿提供商，Wink 为您提供通过专有销售渠道在线销售库存所需的工具。

住宿提供商负责随时保持房源信息和内容的准确和最新。住宿提供商应遵守本协议约定的可用性、价格及其他承诺。住宿提供商独自负责通过 Wink 后台更新并保持可用性、价格及适用的地方税费（如有）最新。住宿提供商特此授权 Wink 在所有市场推广房间。Wink 强烈建议住宿提供商始终提供最具竞争力的价格、促销和优惠。

**5. 法律义务**

5.1 住宿提供商负责了解并遵守适用于其房源的所有法律、规则、法规及与第三方签订的合同。

5.2 住宿提供商负责按照适用隐私法律及本条款处理和使用客人及其他人的个人数据。

**6. 预订费与佣金**

Wink 为住宿提供商提供数字工具，通过 5 个专有渠道分销和销售库存。通过仪表盘上的自助激活功能激活您的物业后，您的物业将自动在 Traveliko.com 及网络中可预订。您也可在后台的“分销 - 销售渠道”菜单中手动停用这些渠道。

* Traveliko.com – 0% 佣金 OTA  
* WinkLinks – 用于 Instagram 等社交媒体渠道的链接功能  
* 社交分享 – 可在任何线上分享的快捷链接  
* 预订引擎 – 酒店网站及更多的互联网预订引擎  
* Wink 网络 – 连接酒店与合作伙伴的联盟网络

任何确认预订适用以下预订费、佣金及支付费用：

* Traveliko.com：支付服务商费用（付款条款中描述，4%）+ Wink 预订费（1.5%）  
* WinkLinks：支付服务商费用（4%）+ Wink 预订费（1.5%）  
* 社交分享：支付服务商费用（4%）+ Wink 预订费（1.5%）  
* 预订引擎：支付服务商费用（4%）+ Wink 预订费（1.5%）  
* Wink 网络：支付服务商费用（4%）+ Wink 预订费（1.5%）+ 联盟佣金（酒店自行协商佣金）

通过 Traveliko、社交分享、预订引擎或 WinkLinks 进行的 100 美元预订计算示例：

100 - 4% = 96  
96 - 1.5% = 94.56  
应付酒店（“净价”）= 94.56 美元

通过佣金为 10% 的联盟进行的 100 美元预订计算示例：

100 - 4% = 96  
96 - 1.5% = 94.56  
94.56 - 10% = 85.1  
应付酒店（“净价”）= 85.1 美元

**7. 税费**

7.1 住宿提供商负责确保房价包含所有适用税费。住宿提供商应确保所含税费准确且最新。若住宿提供商未指明需支付的城市税、旅游费或其他地方费用，则视为已包含在房价内。

7.2 住宿提供商负责向相关政府机构和/或机构缴纳税费。

7.3 住宿提供商应对未准确包含税费及未及时通知 Wink 相关税费所导致的任何损失、费用、罚款及/或损害赔偿承担责任，并同意赔偿 Wink。

7.4 Wink 向合作伙伴提供房间时应包含所有适用税费（受本条款第(a)款限制）。除非另有说明，本协议项下所有应付金额均含增值税或其他适用税费（不含企业所得税或利润税）。若相关税务机关对提供服务所在国征收增值税，Wink 在收到住宿提供商有效税务发票后，将支付含税金额中的增值税部分。

7.5 相关税务机关因延迟缴纳或发票增值税产生的罚款或利息应由住宿提供商承担。此外，若相关税务机关根据自我申报机制征收增值税，Wink 将根据该国增值税法规在其增值税申报中申报该增值税。因延迟缴纳自我申报增值税产生的罚款或利息由 Wink 承担。

**8. 银行信息及住宿提供商指定代表**

住宿提供商应确保向 Wink 提供的银行账户信息始终准确，并应及时通知任何变更。

仅签署本协议的人为住宿提供商唯一指定代表，有权请求更改银行账户及/或收款人信息。住宿提供商其他人员无此权限。指定代表变更须书面通知 Wink，且仅通过双方书面协议修订本协议后方可生效。

**9. 预订修改**

9.1 住宿提供商对超出预订确认中取消政策范围的任何预订修改承担责任，需与客人直接沟通。

9.2 若住宿提供商上传错误房价导致预订以错误房价完成，住宿提供商应按该错误房价履约。

9.3 Wink 对住宿提供商的房价错误不承担任何责任。

**10. 预订识别 / 反欺诈合作**

10.1 住宿提供商应确保其系统准确录入 Wink 客户的完整正确公司信息，以便每笔预订均可明确识别为本协议项下预订。

10.2 每个参与住宿提供商应尽商业合理努力确保客人入住时所出示身份证明与预订信息一致。若预订为潜在欺诈预订，或客人提供的某些数据 Wink 无法验证，住宿提供商与 Wink 应合作处理该预订，包括随时取消该预订。住宿提供商同意全力配合 Wink，并提供 Wink 可能要求的任何信息。

10.3 若住宿提供商未遵守本条款，且预订被认定为潜在欺诈预订，Wink 不承担责任。

**11. 无房 / 未履约**

11.1 若因超额预订或其他原因导致无房，住宿提供商应尽最大努力维持客人及预订，尽可能让客人留在原住宿。若必须将客人调至其他住宿，住宿提供商同意：

* 在调房前立即通知 Wink；  
* 将客人调至同区域内同等或更高等级的替代物业，按原预订条件，且调房费用由住宿提供商承担，且不向客人额外收费，包括但不限于  
* 交通及其他调房相关费用；  
* 若 Wink 因无房或调房产生任何财务罚款、赔偿或费用，Wink 有权从住宿提供商应收款中扣除所有直接费用。

11.2 若住宿提供商未履行本协议任何条件，包括场所条件不可接受及设施与服务不足，Wink 有权停止付款并要求住宿提供商正确履行预订条款。住宿提供商应承担因未履约向受影响客人支付的全部赔偿费用，Wink 不承担因住宿提供商未履约产生的任何责任。

**12. 参与酒店物业状况**

住宿提供商应尽早通过 Wink 后台“公告”栏目通知任何可能影响提供房间、设施与服务或影响客人入住体验的施工、装修、更新等工作。若无房或未履约因上述原因导致，适用本条款第 11 节“无房 / 未履约”，住宿提供商承担全部赔偿费用，Wink 不承担责任。

**13. 修改、期限及终止**

13.1 修改。除适用法律另有规定外，Wink 可随时修改本条款。若有重大变更，Wink 将在 Wink 平台发布修订条款并更新“最后更新”日期。受影响方将至少提前三十（30）天收到变更通知。若您未在修订生效前终止协议，继续使用服务即视为接受修订条款。

13.2 期限。除非另有约定，本协议自签署日起生效，期限不定。

13.3 本协议持续有效，直至双方按以下方式终止：

* Wink 可随时无因提前三十（30）日书面通知对方终止；  
* 住宿提供商可随时无因通过后台停用物业终止，无需通知；  
* 本协议其他明确规定的方式。

13.4 Wink 保留自行决定权，若住宿提供商发生以下情况，可立即书面通知终止协议：

* 住宿提供商进入破产、清算、司法管理等程序，且六十（60）天内未获驳回或解决；  
* 债权人对参与酒店执行或尝试执行抵押；  
* 住宿提供商停止正常经营；  
* 住宿提供商失去租赁权或品牌运营权；  
* 住宿性质发生变化，包括施工、装修等；  
* 因不可抗力事件。

13.5 协议提前终止或期满时，住宿提供商应：

* 履行终止或期满前所有预订，按原预订价格，包括终止后入住的预订，或提供合适替代服务；  
* 结清相关账目。

**15. 客人事件与投诉**

15.1 若客人发生事件，住宿提供商应尽快通知 Wink 并提供详细信息，配合处理。

15.2 住宿提供商同意配合 Wink 处理所有客人索赔或投诉，并在收到 Wink 查询后七（7）日内或服务水平协议约定期限内详细回复。

15.3 若因住宿提供商行为导致 Wink 需向客人赔偿，住宿提供商应及时全额赔偿 Wink，符合本协议第 33 条“赔偿”规定。

15.4 住宿提供商同意 Wink 可从其应收款中扣除因客人事件和投诉产生的所有费用。

**16. 审计权**

16.1 协议期间，Wink 可合理审计住宿提供商及参与酒店相关记录，以核实预订履行、收付款项等。
