餐厅让您展示并销售您的餐厅，无论是在场内还是场外。要创建餐厅，请点击 `Create restaurant` 按钮。

以下各节将引导您完成管理餐厅的每个部分。

## 配置

开始时默认选中 `Configurations` 选项卡。

本节涵盖餐厅的基本信息。

- **内部名称** 餐厅的内部称呼。*例如 Tsunami Sushi*
- **是否需要预订** 该餐厅是否需要预订。*例如 是*

### 容量

本节允许您输入餐厅容量。

- **最大座位数** 总可用座位数。*例如 100*
- **最大单桌人数** 最大桌位容量。*例如 20*

### 餐食

本节涵盖餐厅提供的餐食类型。

*切换开关至 ✅ 以告知用户支持该餐食类型。*

## 营业时间

本节允许您管理餐厅的开放时间。

### 营业时间

- **开门时间** 餐厅开始营业的时间。*例如 09:00*
- **关门时间** 餐厅停止营业的时间。*例如 21:00*

### 营业日

本节允许您设置餐厅每周营业的日期。

*切换开关至 ✅ 表示该天营业。*

### 季节性

:::note
没有漂亮的原生“日/月”表单字段。在此情况下，我们仅使用日期选择器中的月份和日期。
:::

本节允许您设置餐厅全年可用的日期。

- **开始日期** 餐厅开始营业的日期。*例如 2024年9月11日*
- **结束日期** 餐厅停止营业的日期。*例如 2024年7月11日*

## 设施

:::note
本节为可选项。
:::

点击 `Amenities` 选项卡继续。

类似于物业设施，餐厅设施仅适用于特定餐厅。勾选适用的设施。

## 描述

:::note
您需要至少撰写一条描述，且必须为英文。
:::

点击 `Descriptions` 选项卡继续。

类似于物业欢迎文本，您也可以为餐厅创建本地化描述。
您可以用任意多种语言撰写。我们将为所有最流行的语言添加翻译。
这是旅客查看您的餐厅时看到的文本。

- **名称** 给餐厅命名。*例如 Tsunami Sushi*
- **描述** 用一两段文字描述餐厅。*例如 这是世界上最棒的舞厅...*

## 社交媒体

:::note
本节为可选项。
:::

点击 `Social media` 选项卡继续。

您可能会想，“为什么餐厅有自己的社交媒体部分？”餐厅也可以是物业附近但场外的设施，拥有自己的员工和 IG 账号。

社交媒体让您与新老旅客保持联系。
将您的社交网络账号添加到您的资料中，方便客人轻松联系您并了解该餐厅。

<Steps>
1. 切换您希望启用的社交网络开关。
2. 输入您在该网络上的账号名称。
</Steps>

## 口碑

:::note
本节为可选项。
:::

点击 `Reputation` 选项卡继续。

类似于 `Social media` 选项卡，如果该餐厅有独立的口碑评分或额外荣誉，您可以在此突出显示。

您的口碑是您的数字货币，您希望随时携带它。
口碑条目可以是物业想要展示的任何内容，如第三方评分或厨师赢得金牌的烘焙比赛。
在本节中，您可以添加来自互联网各处的评分和排名。

点击 `Add reputation score` 按钮。

<Steps>
1. **类别** 选择您要添加的评分类型。*例如 第三方评价*
2. **提供者** 输入授予您评分的机构。*例如 BigOTA.com*
3. **评分类型** 选择您获得评分的形式。*例如 数值型*
4. **日期** 可选，输入您获得评分的日期。
5. **评分** 输入您获得的分数。*例如 8.5*
6. **最高评分** 可选，输入该奖项的最高分。*例如 10*
</Steps>

## 基于属性的销售

:::note
本节为可选项。
:::

点击 `Attribute Based Selling` 选项卡继续。

我们所有设施都配有自己的 ABS 部分。本节允许您在主库存上创建增销机会，或将非交易型库存转为交易型。
我们强烈建议您创建这些机会，以便在预订过程中以更高价格与预订者建立联系。

:::tip
餐厅可以通过 ABS 进行销售。销售方式多样：

- 固定价格套餐。
- 团体自助餐。
- 吃到饱。
- 一只鸭子，一整只鸭子，只有鸭子。
- +++ 让您的想象力飞扬，赚取额外收入！！
:::

- **价格点** 快速向旅客展示该餐厅的价格水平。
- **激活** 如果您希望该餐厅在所有销售渠道中不可预订，请切换开关至 🛑。
- **可佣金** 表示该库存是否允许分销商获得佣金。
- **精选** 表示如果有机会，该餐厅是否应更显著展示。
- **基于属性的销售** 切换开关至 ✅ 以启用该库存的 ABS。*启用后，您需要至少添加 1 个 ABS 销售项目*

### 附加项目

启用 ABS 后，您可以开始添加附加项目并使其可售。

*默认情况下，系统已为您创建一个空项目。您可以点击 `Add item for sale` 按钮添加更多。*

- **名称** 给您的附加项目命名。*例如 吃到饱*
- **定价类型** 指明您如何计算价格。*例如 设为 `Per hour` 表示价格按租用小时数计算。*
- **基础价格** 输入该项目的常规价格。*例如 20 / 小时*
- **折扣价** 输入 Wink 用户预订该项目时享受的价格。*例如 15 / 小时*

### 限制

您可以通过设置最小/最大数量限制该项目的可用性。

- **最小数量** 限制必须预订的最小数量。*例如 1 位参与者*
- **最大数量** 限制必须预订的最大数量。*例如 4 位参与者*

### 描述

:::note
您需要至少撰写一条描述，且必须为英文。
:::

- **名称** 给您的附加项目命名。*例如 吃到饱*
- **描述** 用一两段文字描述该附加项目。*例如 吃到饱，持续 1 小时。尽情享用。*

## 附近位置

点击 `Proximity` 选项卡继续。

并非所有库存都位于场内。如果您有位于场外的库存，请在此告知旅客，您将有机会设置单独的地图位置及可选联系信息。

*当餐厅距离酒店几公里远，旅客想向出租车司机说明如何到达时，这非常有用。*

点击 `Save` 按钮继续。
