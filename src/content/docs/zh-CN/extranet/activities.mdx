活动让您展示并销售您的活动，无论是在场内还是场外。要创建活动，请点击 `Create activity` 按钮。

以下各节将逐步指导您管理活动的每个部分。

## 配置

默认情况下，开始时会选中 `Configurations` 选项卡。

本节涵盖活动的基础信息。

- **内部名称** 活动的名称，供内部使用。*例如：飞钓*
- **生活方式** 可选，选择适合此活动的生活方式。*例如：运动*
- **年龄** 可选，选择此活动推荐的最低年龄组。*例如：成人*
- **类型** 可选，选择最符合的活动类型。*例如：钓鱼*
- **是否需要预订** 此活动是否需要预订。*例如：是*

## 时间安排

本节让您管理活动的可用时间。

### 营业时间

- **开始时间** 活动开始可用的时间。*例如：09:00*
- **结束时间** 活动结束不可用的时间。*例如：21:00*

### 营业日

本节让您设置活动每周可用的日期。

*切换开关至 ✅ 表示该天可用。*

### 季节性

:::note
没有漂亮的原生“日/月”表单字段。在此情况下，我们仅使用此日期选择器中的月份和日期。
:::

本节让您设置活动全年可用的日期。

- **开始日期** 活动开始可用的日期。*例如：2024年9月11日*
- **结束日期** 活动结束不可用的日期。*例如：2024年7月11日*

## 设施

:::note
本节为可选项。
:::

点击 `Amenities` 选项卡继续。

类似于物业设施，活动设施仅适用于特定活动。勾选适用的设施。

## 描述

:::note
您需要至少撰写一条描述，且必须为英文。
:::

点击 `Descriptions` 选项卡继续。

类似于物业欢迎文本，您也可以为活动创建本地化描述。
您可以用任意多种语言撰写。我们将为所有最流行的语言添加翻译。
这是旅行者查看您的活动时看到的文本。

- **名称** 给活动命名。*例如：飞钓*
- **描述** 用一两段文字描述活动。*例如：只有你……森林……河流……瓶中宁静。*

## 社交媒体

:::note
本节为可选项。
:::

点击 `Social media` 选项卡继续。

您可能会想，“为什么活动有自己的社交媒体部分？”活动也可以是物业附近但场外的设施，拥有自己的员工和独立的 IG 账号。

社交媒体让您随时与新老旅行者保持联系。
将您的社交网络账号添加到您的资料中，方便客人轻松联系您并了解此活动。

<Steps>
1. 切换您希望启用的社交网络开关。
2. 输入您在该网络上的账号名称。
</Steps>

## 信誉

:::note
本节为可选项。
:::

点击 `Reputation` 选项卡继续。

类似于 `Social media` 选项卡，如果此活动有独立的信誉评分或额外荣誉，您可以在此突出显示。

您的信誉是您的数字货币，您希望随时携带。
信誉条目可以是物业想要展示的任何内容，如第三方评分或厨师赢得金奖的烘焙比赛。
在本节中，您可以添加来自互联网各处的评分和排名。

点击 `Add reputation score` 按钮。

<Steps>
1. **类别** 选择您要添加的评分类型。*例如：第三方评价*
2. **提供者** 输入授予您评分的机构。*例如：BigOTA.com*
3. **评分类型** 选择您获得评分的形式。*例如：数字*
4. **日期** 可选，输入您获得评分的日期。
5. **评分** 输入您获得的分数。*例如：8.5*
6. **最高评分** 可选，输入此类奖项的最高分。*例如：10*
</Steps>

## 基于属性的销售

:::note
本节为可选项。
:::

点击 `Attribute Based Selling` 选项卡继续。

我们所有设施都配有自己的 ABS 部分。本节让您在主库存上创建增销机会，或将非交易型库存转为交易型。
我们强烈建议您创建这些机会，以便在预订过程中以更高价格与预订者建立联系。

:::tip
活动可以通过 ABS 进行销售。它可以通过不同方式销售：

- 一日游。
- 三日游。
- +++ 让您的想象力自由发挥，赚取额外收入！！
:::

- **价格点** 快速向旅行者展示此活动的价格水平。
- **激活** 如果您希望此活动在所有销售渠道中不可预订，请切换开关至 🛑。
- **可佣金** 表示此库存是否向联盟支付佣金。
- **精选** 表示如果有机会，此活动是否应更显著展示。
- **基于属性的销售** 切换开关至 ✅ 以启用此库存的 ABS。*启用后，您需要至少添加 1 个 ABS 销售项目*

### 附加项目

启用 ABS 后，您可以开始添加附加项目并使其可销售。

*默认情况下，系统已为您创建一个空项目。您可以点击 `Add item for sale` 按钮添加更多。*

- **名称** 给您的附加项目命名。*例如：一日游*
- **定价类型** 指明您如何计算价格。*例如：设置为 `Per day` 表示价格按天数计算。*
- **基础价格** 输入此项目的常规价格。*例如：200 / 天*
- **折扣价** 输入 Wink 用户预订此项目时的价格。*例如：150 / 天*

### 限制

您可以通过设置最小/最大数量限制此项目的可用性。

- **最小数量** 限制必须预订的最小数量。*例如：1 位参与者*
- **最大数量** 限制必须预订的最大数量。*例如：4 位参与者*

### 描述

:::note
您需要至少撰写一条描述，且必须为英文。
:::

- **名称** 给您的附加项目命名。*例如：一日游*
- **描述** 用一两段文字描述附加项目。*例如：休息一天，前往河边。*

## 附近位置

点击 `Proximity` 选项卡继续。

并非所有库存都位于场内。如果您有位于场外的库存，请在此告知旅行者，您将有机会设置单独的地图位置及可选联系信息。

*当活动距离酒店几公里远，旅行者想向出租车司机说明如何到达时，这会很有帮助。*

点击 `Save` 按钮继续。
