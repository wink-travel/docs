:::note
创建您的第一个主费率是入职向导的一部分。
:::

:::note
`主费率` 是唯一的 `客房 / 费率计划` 组合。

*大多数旅游费率提供商在此上下文中仅使用客房和费率计划。当您看到您的客房在 OTA 或预订引擎上出售时，旅客将比较费率计划。*

Wink 添加了“主费率”一词，以便为唯一组合提供更多灵活性。
:::

:::note
当添加或删除客房或费率计划时，主费率会自动创建或移除。
:::

要管理您的主费率，请从主导航栏点击 `Monetize > Master Rate`。

每个物业都会为每个客房 / 费率计划组合拥有一个主费率。

*示例：如果您的物业有一个客房和两个费率计划，旅客将看到两个可供销售的主费率（如果满足所有条件）。*

主费率条目有两个管理选项：

- 切换主费率的可用性。
- 设置优惠。

## 可用性

启用/禁用主费率时，会影响所有销售渠道的可用性。此开关是快速关闭特定主费率的快捷方式，立即生效。

## 优惠

当旅客预订客房时，他们不会直接与客房互动，而是与该客房的主费率互动。
主费率可以附带仅适用于特定客房 / 费率计划组合的独特优惠，从而为物业所有者提供另一种激励该组合的方式。

选择优惠时需注意两点：

- 是保证提供还是基于可用性。
- 其数值大小。

保证提供的优惠具有更高的数值，因为它代表对客户更大的价值。
每家酒店都会获得一个基于所有主费率中所有优惠数值总和的 `优惠分数`。
寻找优惠的旅客可以根据优惠分数对酒店进行排序。

优惠的作用是：

- 区分客房 / 费率计划组合。
- 获得更高的曝光度。*即在您所在地区的其他酒店中获得更高的优惠分数排名。*

:::tip
优惠源于酒店希望区分自身并“玩转系统”的愿望。
上一次他们这样做时，实际上是通过为更好的排名支付更高佣金而把所有利润都送给了 OTA，结果自毁前程。
历史上，佣金是酒店唯一可以“玩”的项目。

优惠是酒店奖励旅客的一种方式，且通常不会给酒店带来实际成本。

如果您有其他类型优惠的建议，[请联系我们](mailto:hi@wink.travel)。
:::

- 从主导航栏进入 `Distribution > Inventory`，专门管理所有销售渠道的库存。

## API

想要管理 `主费率` 的开发者可以前往 [Developers > APIs > Monetize](/zh-CN/developers/apis/#monetize-api)。
