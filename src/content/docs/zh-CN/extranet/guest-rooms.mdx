每一次预订旅程都从客房开始。客房相关功能众多，初次使用时可能会有些不知所措。
本文将涵盖您需要了解的所有内容，助您成为 Wink 客房专家。

客房，或称为 `the biz` 中的房型，让您能够展示并销售您的房间，无论是在场内还是场外。要创建客房，请点击 `Create a new guest room` 按钮。

以下章节将逐步引导您完成客房管理的各个部分。

## 配置

默认情况下，您开始时会选中 `Configurations` 标签。

本节涵盖客房的基础信息，以便我们了解可以向谁销售此房间。

- **房型名称** 您内部称呼该房间的名称。*例如 Supreme deluxe*
- **禁烟** 该房间是否为禁烟房。
- **挂牌价** 未包含任何折扣或附加费的一晚住宿公布价格。*例如 100。请注意，我们仅使用此数字向销售渠道展示预期价格，或在您填写表单时预填某些字段。*
- **最佳可用价** 未包含任何折扣或附加费的一晚住宿最佳可用价格。*例如 75。请注意，我们仅使用此数字向销售渠道展示预期价格，或在您填写表单时预填某些字段。*
- **数量** 您在任意一天拥有的此房型单位数量。*例如 40*

### 入住人数

:::caution
入住人数在您的客房是否显示为特定行程可用中起着重要作用。许多字段在您首次创建房间时会自动填充。请确保根据您的房型进行修改。
:::

- **最少入住人数** 允许入住此房间的最少人数。*例如 1*
- **包含成人入住人数** 房价中包含的成人数量。*例如 2。注意：如果房间支持超过此人数，超出部分将根据您的价格计划中的 `Extra adult` 调整计算。*
- **最大入住人数** 允许入住此房间的最大人数（成人+儿童）。*例如 4*
- **最大成人入住人数** 允许入住此房间的最大成人数量。*例如 3*
- **包含儿童入住人数** 房价中包含的儿童数量。*例如 1。注意：如果房间支持超过此人数，超出部分将根据您的价格计划中的 `Extra child` 调整计算。*
- **最大儿童入住人数** 允许入住此房间的最大儿童数量。*例如 2*

### 其他

本节仅供参考。寻找更大房间和套房的旅客会对这些信息感兴趣。

- **浴室数量** 此房型的浴室数量。*例如 1*
- **客厅数量** 此房型的客厅数量。*例如 1*
- **最大折叠床数** 此房型支持的折叠床数量。*例如 1*
- **最大婴儿床数** 此房型支持的婴儿床数量。*例如 1*

### 房间分类

- **细分市场** 客房所属的细分市场。*例如 Budget*
- **位置** 客房所属的位置类型。*例如 高楼层*
- **景观** 该客房可期待的景观类型。*例如 城市景观*
- **分类** 该客房所属的分类。*例如 经济型*
- **建筑风格** 该客房周围的建筑风格。*例如 亚洲风格*
- **楼层** 可选的楼层（或楼层范围），该客房所在位置。*例如 20层及以上*

### 组成

仅当客房由多个客房组成时适用。

- **组成** 如果客房由多个房间组成，请切换开关为 ✅。*例如 是*
- **数量** 指明该客房由多少个客房组成。*例如 2*

### 青年旅社详情

仅当您经营宿舍、青年旅社或其他共享房间服务时适用。

- **共享房间** 如果该客房为共享房间，请切换开关为 ✅。*例如 是*
- **性别** 指明该客房的性别限制。*例如 男性*

## 卧室

:::note
卧室布局可能无限多样。我们尽力使其尽可能可定制，以适应您的物业。
:::

点击 `Bedrooms` 标签继续。

卧室部分允许您指定卧室的布局，并指明每个卧室内的配置。

同一房型的客房可以有多种布局，或称为 `Bedding configurations`。一个客房至少需要一个布局。

- **布局名称** 给布局命名，便于您理解。*例如 Main*

一个布局中可以包含多个卧室。

- **卧室类型** 每个卧室都有一个类型。*默认第一个为 `Master`，但您可以在配置中更改。*

在卧室内，您可以有多张不同类型和数量的床。

- **床型** 选择卧室中的床型。*例如 King*
- **数量** 选择该床型在卧室中的数量。*例如 1*

## 设施

:::note
本节为可选项。
:::

点击 `Amenities` 标签继续。

类似于物业设施，客房设施仅适用于特定客房。勾选适用的设施。

## 描述

:::note
您需要至少撰写一条描述，且必须为英文。
:::

点击 `Descriptions` 标签继续。

类似于物业欢迎语，您也可以为客房创建本地化描述。
您可以用多种语言撰写。我们将为所有最流行的语言添加翻译。
这是旅客查看您的客房时看到的文本。

- **名称** 给客房命名。*例如 Supreme deluxe*
- **描述** 用一两段描述客房。*例如 这是全世界最好的客房...*

## 社交媒体

:::note
本节为可选项。
:::

点击 `Social media` 标签继续。

您可能会想，“为什么客房会有自己的社交媒体部分？”客房也可以是物业附近但场外的别墅，拥有自己的员工和独立的 IG 账号。

社交媒体让您与新老旅客保持联系。
将您的社交网络账号添加到您的资料中，方便客人轻松联系您并了解更多关于您的客房信息。

<Steps>
1. 切换您希望启用的社交网络开关。
2. 输入您在该网络上的账号名称。
</Steps>

## 声誉

:::note
本节为可选项。
:::

点击 `Reputation` 标签继续。

类似于 `Social media` 标签，如果该客房有独立的声誉评分或额外荣誉，您可以在此突出显示。

您的声誉是您的数字货币，您希望无论走到哪里都能携带它。
声誉条目可以是您的物业想要展示的任何内容，例如第三方评分或厨师赢得金牌的烘焙比赛。
在本节中，您可以添加来自互联网各处的评分和排名。

点击 `Add reputation score` 按钮。

<Steps>
1. **类别** 选择您想添加的评分类型。*例如 第三方评价*
2. **提供者** 输入授予您评分的机构。*例如 BigOTA.com*
3. **评分类型** 选择您获得评分的形式。*例如 数值型*
4. **日期** 可选，输入您获得评分的日期。
5. **评分** 输入您获得的评分。*例如 8.5*
6. **最高评分** 可选，输入该奖项的最高可能分数。*例如 10*
</Steps>

## 基于属性的销售

:::note
本节为可选项。
:::

点击 `Attribute Based Selling` 标签继续。

我们所有设施都有自己的 ABS 部分。此部分让您在主库存上创建增销机会，或将非交易型库存转为交易型。
我们强烈建议您创建这些机会，以便在预订过程中有可能以更高价格与预订者建立联系。

:::tip
客房附带许多可作为附加服务的项目：

- 高级床垫。
- 豪华浴室套装。
- Marshal 蓝牙音箱。
- 配备超舒适椅子的工作桌。
- 房内早餐
- 房内按摩
- 12瓶装啤酒
- 一瓶葡萄酒
- 高级巧克力
- +++ 让您的想象力自由发挥，赚取额外收入！！
:::

- **价格点** 快速向旅客指示此客房的价格水平。没错……是客房的价格点，不是附加服务的。
- **激活** 如果您希望此客房在所有主价和销售渠道中变为不可预订，请切换开关为 🛑。
- **可佣金** 指明该库存是否为联盟带来佣金。
- **特色** 指明如果有机会，是否应更突出地展示此客房。
- **基于属性的销售** 切换开关为 ✅ 以启用此库存的 ABS。*启用后，您需要至少添加 1 个 ABS 销售项目*

### 附加项目

启用 ABS 后，您可以开始添加附加项目并使其可供销售。

*默认情况下，系统已为您创建了一个空项目。您可以点击 `Add item for sale` 按钮添加更多。*

- **名称** 给您的附加项目命名。*例如 Royce 的香槟巧克力松露*
- **定价类型** 指明您如何计算价格。*例如 桨板租赁。设置为 `Per hour` 表示价格基于租赁小时数。*
- **基础价格** 输入该项目的常规价格。*例如 20 / 小时*
- **折扣价** 输入 Wink 用户与房间一起预订该项目时享受的价格。*例如 15 / 小时*

### 限制

您可以通过设置最小/最大数量限制来限制该项目的可用性。
如果我们继续以按小时租赁桨板为例，但希望客人至少同时预订 2 块桨板，则：

- **最小数量** 限制必须预订的最小数量。*例如 2 块桨板*
- **最大数量** 限制必须预订的最大数量。*例如 留空表示无限制*

### 描述

:::note
您需要至少撰写一条描述，且必须为英文。
:::

- **名称** 给您的附加项目命名。*例如 立式桨板*
- **描述** 用一两段描述该附加项目。*例如 不要只是坐着——拿起桨板出发吧！*

## 附近环境

点击 `Proximity` 标签继续。

并非所有库存都位于物业内。如果您有位于物业外的库存，请在此告知旅客，您将有机会设置单独的地图位置及可选联系信息。

*当酒店拥有的别墅位于酒店几公里外，旅客想向出租车司机说明如何到达时，这会非常有用。*

点击 `Save` 按钮继续。
