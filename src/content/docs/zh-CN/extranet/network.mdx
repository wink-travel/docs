本文介绍如何在 Wink 上与积极的合作伙伴建立联系并管理现有合作伙伴。

## 销售渠道

要管理您的销售渠道，请点击主导航栏中的 `Distribution > Sales channels`。

销售渠道是您现有的合作伙伴关系，具体包括他们可访问的库存、优惠和折扣。

要更新现有销售渠道，请点击您想更新的销售渠道下方的 `Actions` 链接。

- **会员折扣** 通过此销售渠道预订的旅客可享受的会员价格。
- **佣金** 每笔成功预订后分配给该销售渠道的佣金。
- **单独促销** 添加您希望该销售渠道拥有的促销活动。
- **捆绑促销** 添加您希望该销售渠道拥有的捆绑促销活动。
- **启用** 切换启用开关以完全启用或禁用该销售渠道。
- **黑名单** 将销售渠道标记为黑名单后，该渠道将无法进行预订。

设置完成后，点击 `Save` 按钮继续。

:::tip
将销售渠道列入黑名单后，该渠道将无法从您这里获得任何价格；甚至包括您的 Wink Network 价格。
:::

## 探索网络

要在 Wink 上寻找新的合作伙伴，请点击主导航栏中的 `Distribution > Explore network`。

在此部分，页面顶部显示合作伙伴的待处理请求，下面列出可连接的合作伙伴列表。

### 回复请求

您可以选择拒绝合作伙伴请求。如果想继续处理请求，系统将带您进入 `affiliate details` 页面，您可以在此查看合作伙伴的详细信息及其在 Wink 上的历史表现指标。

### 搜索合作伙伴

在合作伙伴网格中，您可以按名称、类型和位置搜索合作伙伴。看到感兴趣的合作伙伴时，点击即可进入 `affiliate details` 页面。

### 与合作伙伴建立联系

如果您选择与某个合作伙伴建立直接关系，系统会要求您以管理销售渠道的方式对该关系进行编码。

保存新的销售渠道后，您将返回销售渠道列表。

## API

希望管理其 `Network` 的开发者可以前往 [Developers > APIs > Distribution](/zh-CN/developers/apis/#distribution-api)。
