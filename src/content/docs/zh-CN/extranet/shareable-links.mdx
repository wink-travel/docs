:::note
这是 Wink Studio 中可用的 [可分享链接](/studio/shareable-links) 功能的一个子集。

无需进入 Wink Studio 生成，物业将自动使其所有库存可作为可分享链接使用。
:::

可分享链接是 Wink 的核心功能之一，诞生于通过互联网最简单的方式——超链接，以动态方式传达库存和价格的需求。

<h4 align="center"> [https://trvl.as/3xWCH](https://trvl.as/3xWCH) </h4>

上面是我们链接的一个示例。它简短且便于携带……甚至足够短，便于记忆。虽然简短，但此链接包含了**丰富**的信息，例如：

- 您的应用 ID。
- 您的联盟 ID。
- 向用户展示的图片和文本。
- 外观风格（*即主题、颜色等*）。
- 用于价格查询的行程（*2 位成人，1 晚*）。
- 显示价格的货币。
- 显示文本的语言。

本文余下部分将指导您如何管理指向您库存的链接。

## 自定义链接

该表单允许您通过以下方式自定义链接：

<Steps>
 1. 默认情况下，链接以您的库存英文名称命名。您可以为其命名任何便于您记忆链接内容的名称。
 2. 默认情况下，链接应用您的主要品牌/自定义。您可以选择 Wink Studio 中生成的其他 [自定义](/studio/customization) 。
 3. 添加用逗号分隔的关键词，这些关键词将被 [网络爬虫](https://developer.mozilla.org/en-US/docs/Glossary/Crawler) 使用。
 4. 可选地输入您的 X 账号或 Facebook 应用 ID。
 5. 添加用户希望看到的语言的标题和描述。
 6. 选择一个或多个图片，当您在 Facebook 或 LinkedIn 等平台分享链接时将显示这些图片。
 7. 选择一个主题/叠加层，为您选择的图片添加品牌设计。*注意：这就是如何让价格显示在图片上的方式。*
 8. 点击 `保存` 按钮继续。
 </Steps>

上图展示了链接的所有可用操作：

- **预览** 显示链接配置的快速概览及其在用户端的展示效果。
- **更新** 更新您的链接配置。
- **添加到 WinkLinks** 将链接添加到您的 WinkLinks 账户。
- **复制** 将链接复制到剪贴板，方便您在任何地方分享（*包括 [WinkLinks](/link-manager/wink-links) *）。
- **分享** 打开新窗口，提供在多个社交网络和消息平台分享的选项。
- **二维码** 生成二维码，方便您在如 IG 这类更注重图片的平台分享。
- **嵌入** 展示如何将此链接作为 [广告横幅](/developers/web-components/#content-loader) 嵌入您的网站。
- **WordPress 使用** 展示如何使用我们的 [WordPress 插件](/developers/wordpress) 将此链接嵌入您的网站。

如果您想分享二维码而非链接，点击 `二维码` 按钮，将弹出新窗口显示您的二维码。

保存二维码的方法：

- 如果您使用笔记本电脑，右键点击图片并选择 `保存图片`。
- 如果您使用手机：
    - 长按图片，直到出现保存图片到相册的选项。
    - 截屏二维码，裁剪合适后保存到相册。

### 嵌入

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

以下是如何将您的链接作为广告横幅嵌入网站：

- 第 3 行展示如何将 Wink 样式嵌入您的网站。
- 第 6 至 9 行展示如何使用 [wink-content-loader](/developers/web-components/#content-loader) Web 组件，并告诉它为您的代码获取广告横幅。
- 第 11 行展示如何将我们的 Javascript 嵌入您的网站。
- 第 13 行展示如何嵌入 [wink-app-loader](/developers/web-components/#application-loader) Web 组件，并告诉它获取您的页面级配置偏好。
