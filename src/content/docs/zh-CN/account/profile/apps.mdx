**Apps** 标签页位于 [iam.wink.travel/my-account/apps](https://iam.wink.travel/my-account/apps)，列出了您授予访问 Wink 账户权限的所有 OAuth2 应用，并提供撤销访问的方式。

## 应用如何出现在这里

当您在集成了 Wink 的第三方工具（或 Wink 自有应用）上点击 `Authorize` 时，授权流程会记录该授权。从此，该应用可以在您批准的权限范围内代表您操作。

## 审查访问权限

每条记录显示：

- **应用名称**及简短描述。
- 应用可访问的**权限范围**（例如读取库存、管理预订）。
- **最后使用时间** — 应用最近一次代表您调用 Wink API 的时间。

## 撤销应用

点击任意条目旁的 `Revoke`。该应用的令牌将立即失效，任何针对您账户运行的后台任务将开始返回 401 错误。

:::tip
撤销操作是非破坏性的 — 您可以随时通过重新完成授权流程来重新授权该应用。若应用行为异常、您需要更换凭证，或不再使用该集成，均可使用此功能。
:::

## 您自己开发的 OAuth 应用

如果您想*创建*一个 OAuth 应用（而不仅仅是授权一个），这是组织账户的功能 — 请参见 [设置 → 应用](/zh-CN/portal/settings/applications)。
