[Wink Agent](https://agent.wink.travel) 连接您与全球的旅游库存。本文将引导您创建和管理您的 Wink 旅游代理账户。

## 创建账户

认证成功后，您将被重定向到您的[账户页面](https://agent.wink.travel/company/list)，该页面显示您所有的旅游代理账户。

点击 `Create new account` 按钮继续。您将被重定向到我们的[账户创建](https://agent.wink.travel/company/create)页面。

填写所有必填字段。

<Steps>
1. **名称** 您账户的名称。*这不必是您的真实姓名，但将是用户在查看您的 WinkLinks 页面或供应商想要直接与您联系时看到的名称。*
2. **实体类型** 您是公司还是个人。*主要用于确定您如何有资格接收资金。*
3. **类型** 选择最适合您账户的类型。*例如：影响者*
4. **描述** 为您的账户写一段描述。*这将在您的 WinkLinks 页面以及供应商想要直接与您联系时显示。*
5. **网站** 输入您的主要网站。*这可以是您的 IG 账号 URL 或公司网站。它应该是您大部分在线工作的地方。供应商会访问这些链接来决定是否与您建立联系。*
6. **城市** 您的所在地？
</Steps>

### 付款条款

:::note
如果您选择自行负责付款，您将只能访问与您有直接连接的物业。
:::

:::caution
在此情况下，Wink 不承担支付酒店费用的责任。
:::

:::note
您只能在创建账户时选择此选项。大多数旅游代理应保持此设置关闭 🛑，并使用此门户。
:::

通过我们的 API 集成的注册旅游代理，可以选择收款并将资金支付给我们的物业。

<Steps>
1. **我想收款** 旅游代理负责收取资金。
2. **我想支付酒店** 旅游代理负责支付酒店费用。
</Steps>

想要直接与您建立联系的酒店会看到警告，并必须点击确认 Wink 不负责支付酒店费用，才能与您的账户建立关系。

### 协议

<Steps>
1. **协议** 接受我们的[分销商协议](/zh-CN/travel-agent/terms-of-service)和[付款条款](/zh-CN/travel-agent/payment-terms)以继续。
2. 点击 `Save` 按钮。
</Steps>

恭喜 🎉 …您已在 Wink 上创建了您的第一个分销商账户。

创建后，您可以通过填写地址和添加头像进一步自定义您的账户。

<Steps>
1. 点击您的新旅游代理账户。*这将带您进入账户仪表盘。*
2. 在主导航栏点击 `Account > Profile`。
3. 在个人资料页面，您可以更新现有资料并添加更多信息。
</Steps>

:::note
您的旅游代理账户头像不应与您的 Wink 用户头像混淆。旅游代理账户头像会显示在您的 WinkLinks 页面，并被想要与您建立联系的供应商看到。

您的用户头像主要供内部使用。
:::

## 管理员

您可以邀请其他用户协助管理您的旅游代理账户。用户接受邀请后，将拥有账户的全部访问权限，但不包括付款权限。您始终是您资金的唯一负责人。

<Steps>
1. 从[账户页面](https://agent.wink.travel/company/list)选择您想操作的旅游代理账户。
2. 在主导航栏点击 `Account > Managers`。
3. 添加您想设为账户管理员的用户邮箱。
4. 点击邮箱旁的 `Invite` 链接。
</Steps>

Wink 会向该用户发送邮件，邀请其接受或拒绝您的邀请。

## 通知

有时我们需要与您联系。届时，右上角的铃铛图标会发生变化。
查看我们发送的内容：

<Steps>
1. 在主导航栏点击 `Account > Notifications`。
2. 您将看到新的和过去的通知。
3. 阅读后，您可以选择 `标记为已读` 或 `删除` 通知。
</Steps>

## API

旅游代理仍被视为分销商。想要管理 `Affiliates` 的开发者可以前往 [Developers > API > Affiliate](/zh-CN/developers/apis/#affiliate-api)。
