[眨眼代理](https://agent.wink.travel)为您提供全球旅行库存。本文将指导您在 Wink 上创建和管理旅行社帐户。

## 创建账户

一旦通过身份验证，您将被重定向到您的[账户页面](https://agent.wink.travel/company/list)它会显示您所有的旅行社账户。

点击`Create new account`按钮继续。您将被重定向到我们的[创建帐户](https://agent.wink.travel/company/create) 页。

填写所有必填字段。

<Steps>
  1. **姓名** 您的帐户名称。 *这不需要是您的真实姓名，但当用户查看您的 WinkLinks 页面或供应商想要直接与您联系时，就会看到它。*
  2. **实体类型** 无论您是公司还是个人。 *主要用于确定您如何有资格获得资金。*
  3. **类型** 选择最适合您帐户的类型。 *例如影响者*
  4. **描述** 为您的帐户写一个描述。 *这将在您的 WinkLinks 页面上显示，当供应商想要直接与您联系时也会显示。*
  5. **网站** 输入您的主网站。 *这可以是您的IG帐户网址或公司网站。它应该是您进行大部分在线工作的地方。供应商在决定是否与您联系时会访问这些链接。*
  6. **城市** 你们在哪里？
</Steps>


### 付款条款

:::注意
如果您选择自行付款，您将只能访问与您有直接联系的房产。
:::

:::警告
Wink 不承担此类情况下酒店费用的责任。
:::

:::注意
您只能在创建帐户时使用此选项。大多数旅行社应禁用此设置🛑，并改用此门户。
:::

使用我们的 API 与我们合作的注册旅行社可以选择收取付款并将这些资金支付给我们的酒店。

<Steps>
  1. **我想要付款** 旅行社负责收取资金。
  2. **我想支付酒店费用** 旅行社负责支付酒店费用。
</Steps>


想要与您直接联系的酒店将看到一条警告，并且必须点击确认，Wink 不负责向酒店付款，以便与您的帐户建立关系。

### 协议

<Steps>
  1. **协议** 接受我们的 [联营协议](/travel-agent/terms-of-service) 和 [付款条件](/travel-agent/payment-terms) 继续。
  2. 点击 `Save` 按钮。
</Steps>


恭喜🎉...您已经在 Wink 上创建了您的第一个联盟帐户。

## 管理

<figure>
  <CloudinaryImage width="350" height="445" resize="scale" quality="best" alt="旅行社账户" src="academy/travel-agent" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*旅行社帐户录入*</figcaption>
</figure>


创建后，您可以通过填写地址和添加个人资料图片来进一步定制您的帐户。

<Steps>
  1. 单击您的新旅行社帐户。 *这将带您进入您的帐户仪表板。*
  2. 点击 `Account > Profile` 在主导航栏中。
  3. 在个人资料页面上，您将能够更新您现有的个人资料并添加其他信息。
</Steps>


:::注意
请勿将旅行社账户头像与 Wink 用户头像混淆。您的旅行社账户头像将显示在您的 WinkLinks 页面上，希望与您建立联系的供应商将看到它。

您的用户个人资料图片主要供内部使用。
:::

## 经理

您可以邀请其他用户协助管理您的旅行社账户。一旦用户接受您的邀请，他们将拥有您账户的所有访问权限（付款除外）。您的资金将始终由您本人掌控。

<Steps>
  1. 从中选择您想要使用的旅行社帐户 [账户页面](https://agent.wink.travel/company/list)。
  2. 点击 `Account > Managers` 在主导航栏中。
  3. 将您想要设为帐户管理员的用户的电子邮件添加进去。
  4. 点击 `Invite` 电子邮件旁边的链接。
</Steps>


Wink 将向用户发送一封电子邮件，以接受或拒绝您的邀请。

## 通知

有时，我们想与您联系。当这种情况发生时，您会看到右上角的铃声发生变化。
要阅读我们发送给您的内容：

<Steps>
  1. 点击 `Account > Notifications` 在主导航栏中。
  2. 您将看到新的和过去的通知。
  3. 选择 `Mark as read` 或者 `Delete` 阅读完通知后，请立即通知。
</Steps>
