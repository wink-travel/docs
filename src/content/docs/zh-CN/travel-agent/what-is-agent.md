我们为注册旅行社提供的解决方案是 [眨眼代理](https://agent.wink.travel) 。这是一个自助服务门户，代理商可以使用它来代表客户搜索和预订，并在每次预订中赚取佣金。

更高级的旅行社可以选择在 API 级别进行集成，以便在 Wink 平台之外处理付款和支出。

旅行社拥有与 [Wink工作室](/studio/what-is-studio) 所以我们不会在这里再次介绍它们。
