可共享链接是 Wink 的核心功能，它源于通过互联网上最简单的方式（即超链接）以动态方式传达库存和定价的需求。

<h4 align="center">[https://trvl.as/s/3xWCH](https://trvl.as/s/3xWCH)</h4>


以上是我们的一个链接示例。它简短易记……甚至短到让人容易记住。虽然很短，但这个链接却包含**爆炸物**信息量，例如：

* 您的应用程序 ID。
* 您的会员 ID。
* 向用户显示的图像和文本。
* 外观和感觉（*例如主题、颜色等等*）。
* 用于价格查询的行程（*2 位成人，1 晚*）。
* 显示价格的货币。
* 显示文本的语言。

本文的其余部分将引导您了解如何创建、自定义和共享您的链接。

## 创建链接

  <CloudinaryImage width="350" height="238" resize="scale" quality="best" alt="行动" src="academy/inventory-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*搜索结果操作*</figcaption>

上面的图片取自[搜索](/studio/search)并向您展示您可以对搜索结果执行的一些操作。其中一项操作是`Make a link`。
点击该按钮，它会将您重定向到我们的可共享链接表单页面，您可以在其中开始自定义链接。

## 自定义链接

该表单允许您通过以下方式自定义链接：

<Steps>
  1. 给它起一个名字，以便您记住该链接的内容。
  2. 选择 [定制](/studio/customization) 您想申请此链接。
  3. 添加关键字，以逗号分隔，用于 [网络爬虫](https://developer.mozilla.org/en-US/docs/Glossary/Crawler)。
  4. 可选择输入您的 X 帐户句柄或 Facebook App ID。
  5. 使用您希望用户看到的语言添加标题和描述。
  6. 选择一张或多张将在 Facebook 或 LinkedIn 等平台上分享链接时显示的图片。
  7. 选择一个主题/覆盖，在您选择的图像上添加品牌设计。 *注意：这是让价格显示在图像顶部的方法。*
  8. 点击 `Save` 按钮继续。
</Steps>

保存链接后，您将被重定向到可共享链接页面，现在可以与全世界共享您的链接了。

## 分享链接

<figure>
  <CloudinaryImage width="350" height="641" resize="scale" quality="best" alt="行动" src="academy/shareable-link-actions" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*可共享链接操作*</figcaption>
</figure>


上面显示了您的链接的所有可用操作的图像：

* **预览**向您展示链接的配置方式以及链接在用户看来的样子的简要概述。
* **更新**更新您的链接配置。
* **添加到 WinkLinks**将链接添加到您的 WinkLinks 帐户。
* **复制**将链接复制到剪贴板，以便您可以轻松地在任何您想要的地方分享它（*包括 [WinkLinks](/link-manager/wink-links)*）。
* **分享**打开一个新窗口，其中提供在许多社交网络和消息平台上共享的选项。
* **二维码**生成一个二维码，您可以在 IG 等更注重图像的平台上分享。
* **嵌入**向您展示如何将此链接嵌入为[广告横幅](/developers/web-components/#content-loader)进入您的网站。
* **与 WordPress 一起使用**向您展示如何使用我们的[WordPress 插件](/developers/wordpress)将此链接嵌入到您的网站。

下面我们将更详细地介绍其中一些选项。

### 预览

<figure>
  <CloudinaryImage width="700" height="373" resize="scale" quality="best" alt="预览" src="academy/shareable-link-preview" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*可共享链接预览*</figcaption>
</figure>


当你点击`Preview`，您将看到一张应用了主题的图片。当您在 Facebook 等社交媒体上分享链接时，您会看到相同的内容以及标题和描述出现在您的动态中。

### 社交分享

<figure>
  <CloudinaryImage width="350" height="231" resize="scale" quality="best" alt="社交分享" src="academy/shareable-link-share" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*社交分享选项*</figcaption>
</figure>


如果您点击`Share`按钮后，会打开一个新窗口，方便您直接分享到您喜爱的社交网络和即时通讯应用。
点击任意图标按钮即可继续在该社交网络上发布您的链接。*注意：某些按钮仅当您从移动设备共享时才会显示。*

### 二维码

<figure>
  <CloudinaryImage width="350" height="440" resize="scale" quality="best" alt="社交分享" src="academy/shareable-link-qr-code" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*通过二维码分享*</figcaption>
</figure>


如果您想分享二维码而不是链接，请点击`QR code`按钮和新窗口将会出现您的二维码。

如何保存您的二维码：

* 如果您使用的是笔记本电脑，请右键单击图像并选择`Save image`。
* 如果您正在使用手机：
  * 按下并按住图像上的手指，直到出现将图像保存到照片的选项。
  * 截取二维码的屏幕截图，裁剪以适应大小，然后保存到照片中。

### 嵌入

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="AD_BANNER"
       id="3xWCH"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

以下是如何将您的链接作为广告横幅嵌入到您的网站：

* 第 3 行向您展示如何将 Wink 样式嵌入到您的网站。
* 第 6 至 9 行展示了如何使用[wink-内容加载器](/developers/web-components/#content-loader)Web 组件并告诉它为您的代码获取广告横幅。
* 第 11 行向您展示如何将我们的 Javascript 嵌入到您的网站。
* 第 13 行展示了如何嵌入[wink-app-loader](/developers/web-components/#application-loader)Web 组件并告诉它获取您的页面级配置首选项。
