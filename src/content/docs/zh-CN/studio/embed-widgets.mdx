Studio 提供了多个可嵌入的小部件，您可以将它们添加到自己的网站以促进预订。

要访问嵌入小部件，请从侧边栏导航到 `Tools > Miscellaneous`。

## 可用小部件

| 小部件 | 描述 |
| -- | -- |
| **Account button** | 一个链接到您的联盟页面的品牌按钮 |
| **Itinerary button** | 一个打开行程选择器的按钮 |
| **Itinerary form** | 一个内联日期和客人选择表单 |
| **Search button** | 一个打开物业搜索的按钮 |
| **Lookup button** | 一个用于查找特定库存的按钮 |

## 如何嵌入

每个小部件都提供一个代码片段，您可以复制并粘贴到网站的 HTML 中。该代码片段包含您的 **Client ID** 和 **Customization ID**，因此小部件会自动链接到您的联盟账户。

:::note
如果您使用 WordPress，我们还提供了一个 WordPress 插件以便更轻松集成。
:::
