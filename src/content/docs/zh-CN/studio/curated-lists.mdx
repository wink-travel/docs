精选列表是一项功能，允许您将从搜索旅游库存时发现的内容中挑选的库存进行分组和排序。
创建后，您可以通过将列表作为[网格](/zh-CN/studio/grids)或[地图](/zh-CN/studio/maps)通过[WinkLinks](/zh-CN/link-manager/wink-links)、我们的[WordPress插件](/zh-CN/developers/wordpress/)或将我们的[Web组件](/zh-CN/developers/web-components)嵌入您的网站，向用户展示该列表以供销售。

默认情况下，每个联盟账户都会有一个`Favorites`列表，您可以立即使用。`Favorites`是特殊的，无法删除。
所有精选列表都允许您执行以下操作：

## 创建列表

<CloudinaryImage
      width="350"
      height="219"

    <figcaption>*创建新精选列表表单*</figcaption>
</figure>

- 在仪表盘中，点击导航栏的`Inventory > Curated Lists`。
- 点击`Make a new list`按钮创建新的精选列表。
- 系统会要求您为列表命名。*示例：Top 3 NYC Picks*。
- 点击`Save`按钮。

## 添加列表项

您可以从搜索结果中向任何列表添加列表项。

- 访问[搜索](/zh-CN/studio/search)
- 搜索您想添加到列表的内容。
- 找到后，点击搜索结果卡片左下角的`Actions`链接。
- 点击`Add to...`，然后选择您想添加该库存的精选列表。
- 返回精选列表页面，确认该项已添加到您的列表中。

每个列表都有一些可执行的操作。

- 将列表转换为[地图](/zh-CN/studio/maps)。
- 将列表转换为[网格](/zh-CN/studio/grids)。
- 删除列表（*Favorites除外*）。

## API

想要管理精选列表的开发者可以访问[Developers > API > Browse](/zh-CN/developers/apis/#browse-api)。
