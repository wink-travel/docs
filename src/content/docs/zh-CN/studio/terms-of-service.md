**WINK 联盟合作伙伴通用条款和条件**

通过注册并加入 Wink 合作伙伴计划成为联盟合作伙伴，联盟合作伙伴特此已审阅、理解、确认并接受本联盟协议（“协议”）的条款和条件。

2. **联盟合作伙伴**，其详细信息载于联盟合作伙伴注册表或已在线提交（以下简称“联盟”）。

Wink 与联盟为本协议的“当事方”，合称“双方”。

**鉴于：**  
(i) Wink 运营一个在线系统（“系统”），参与的住宿提供商（统称“住宿提供商”，单称“住宿提供商”）可通过该系统提供其库存供预订，客人可通过该系统预订此类住宿提供商的服务（“服务”）；  
(ii) Wink 不拥有、控制、提供或管理任何房源列表。Wink 不是住宿提供商与客人之间直接签订合同的当事方。Wink 不以任何身份作为住宿提供商的代理；  
(iii) Wink 维护并运营其自有网站（“Wink 网站”），并在第三方网站上提供服务及服务链接；  
(iv) 联盟拥有、控制、托管和/或运营一个或多个互联网域名、网站和应用程序；  
(v) 联盟与 Wink 希望联盟能（直接或间接）向其客户及联盟网站和应用的访问者提供服务，且以本协议所述的形式和条款（以下简称“条款”）提供。

**1. 定义**

**“协议”** 指本协议。  
**“预订”** 指 Wink 记录的客人成功预订住宿提供商的交易。  
**“预订费”** 指 Wink 从联盟佣金中扣除的1.5%作为处理费。  
**“预订价值”** 指支付服务商从客人处收取的预订总金额。  
**“条款”** 指本协议的条款。  
**“佣金”** 指根据本协议，对于每笔已实现交易，扣除支付服务费和预订费前应付给联盟的金额。

**“控制”** 指拥有（直接或间接，单独或与他人共同，无论通过持有投票权证券或其他所有权权益、合伙关系或其他方式）(i) 在公司股东大会上行使或促使行使超过半数投票权，(ii) 任命超过半数（非）执行董事或监事，或 (iii) 指挥或促使指挥公司管理。  
**“客户数据”** 指客人的个人身份信息（“PII”），包括但不限于客人姓名、地址（含电子邮件地址）、信用卡信息及其他机密私人信息。  
**“双重投放”** 指在搜索引擎同一结果页上投放多个广告，目的是引导流量至内容相似的网站或页面。  
**“客人”** 指通过服务完成住宿提供商预订的网站或应用访问者。

**“链接”** 指嵌入网页或电子邮件中的图标、对象、图形或文本，包含指向联盟网站 Wink URL 的超文本指针。  
**“商户交易”** 指支付服务商作为商户记录的已实现交易，支付服务商直接向客人收取预订款项。

**1.2 无合伙关系**

1.2.1 本协议不旨在且任何条款或安排均不应被解释为在双方之间创建合资企业、合伙关系或委托代理关系。

1.2.2 除非 Wink 书面同意或本协议另有规定，联盟不得在联盟网站上发布任何明示或暗示该网站为 Wink 官方网站、受 Wink 认可或为 Wink 一部分的声明。

1.2.3 联盟可直接与住宿提供商协商确定独家或优惠价格、条款、促销、套餐附加服务等。

**2. 协议范围**

**2.1 非排他性**  
在本协议条款和条件约束下，联盟作为 Wink 的非排他性分销商（联盟）运营。

**2.2 服务**

2.2.1 在本协议期限内，双方同意 Wink 按联盟合作伙伴注册表中规定的方式（如链接或微站）及联盟合作伙伴注册表中指定的网站（即联盟网站）向联盟提供服务。

2.2.2 当访问者通过联盟网站通过系统完成预订时，Wink 负责将相关预订详情（如到达日期、住宿晚数、房型、客人姓名）从完成预订的访问者传送给住宿提供商，并向客人发送后续的（电子邮件）确认及/或确认凭证。

2.2.3 服务包括为客人提供客户服务。联盟应及时将所有与服务、预订（包括修改或取消）、住宿提供商及其他相关（支付）问题、投诉和疑问相关的客户服务事项直接转交给 Wink（客户服务中心），不得提供其他服务。

**2.3 链接或微站**

2.3.1 若服务通过链接提供，联盟应自费将链接集成并在联盟网站的显著位置、网页及 Wink 指示或双方协商一致的地点、大小和形式展示。

2.3.2 若服务通过微站提供，联盟应自费将连接和/或微站集成并在联盟网站的显著位置、网页及 Wink 指示或双方协商一致的地点、大小和形式展示。

**3. 许可**

**3.1 互授许可**

3.1.1 在第4.4条约束下，Wink 授予联盟非排他性、可撤销、有限制、免版税、全球范围内的权利和许可：

1) 在联盟网站展示 Wink 数据及 Wink 提供的住宿提供商相关信息；  
2) 根据本协议条款推广和营销服务。

3.1.2 联盟授予 Wink 免版税、全球范围内的权利和许可：

1) 在联盟网站整合、集成、包含并展示链接、微站和/或连接（视情况而定）；  
2) 在联盟网站提供服务。

**3.2 无转授权及保密**

3.2.1 除非 Wink 书面同意，联盟不得 (i) 转授权第3.1.1条授予的权利，(ii) 向第三方转授权链接或连接，或 (iii) 通过联盟集团内公司和/或第三方网站与 Wink 网站建立超链接。

3.2.2 除非 Wink 书面同意或本协议另有规定，联盟不得直接或间接出售、使用、转让、（转）授权、传播、披露、提供访问、泄露或以其他方式传播 Wink 数据或内容给 (i) 任何第三方，(ii) 用于价格/可用性比较、网站、评论或调查，(iii) 除通过服务生成住宿预订外的任何其他目的，或 (iv) 其他方式。

**4. 承诺与义务**

**4.1 一般承诺、义务**

4.1.1 在本协议条款约束下，联盟同意商业上合理地努力 (i) 定制联盟网站并集成链接、连接和/或微站，以最大化引流至 Wink 网站或联盟网站，(ii) 在其商业及内部网络内推广住宿提供商及预订选项，并为此提供其分销网络和渠道（如互联网和内联网）。

4.1.2 联盟同意不采取或遗漏任何可能影响 Wink 与 Wink 网站上住宿提供商关系的行为。联盟同意不做任何可能导致 Wink 被排除在任何住宿提供商预订流程之外的行为。

4.1.3 除第1.2.3条外，联盟同意不与任何住宿提供商就通过系统完成的预订或相关客户服务进行沟通。

4.1.4 联盟应勤勉维护并调整联盟网站内容，保持网站信息准确及时。发现错误或收到 Wink 通知后应及时更正。

4.1.5 联盟不得 (a) 以程序化方式评估或提取 Wink 网站任何部分信息（如客人评价），或尝试如此操作；不得以任何方式复制、修改、反编译、抓取 Wink 网站内容或链接源码；(b) 上传任何病毒、木马、蠕虫、定时炸弹、机器人命令或其他破坏性程序；(c) 向访问者或公众作出与公司、服务、链接或 Wink 网站相关的不实陈述；(d) 通过虚假陈述、欺诈行为或其他欺骗手段获取 Wink 佣金；(e) 其他损害 Wink 及 Wink 网站的行为。违反上述规定将导致联盟被立即终止合作并没收未付佣金。

4.1.6 联盟不得制作内容或 Wink 网站任何部分（包括客人评价）的静态副本。

4.1.7 联盟不得除 Wink 佣金外从预订中获利。联盟（包括其员工及授权人员）不得为转售目的在 Wink 网站或联盟网站上向住宿提供商预订房间。Wink 有权认定任何非预订人本人支付的预订为转售。转售构成本协议的重大违约。Wink 可拒绝或取消转售预订，且有权立即终止联盟参与计划，没收佣金并要求赔偿。

4.1.8 Wink 向联盟提供指定网站（“联盟门户”）、用户ID及密码，供联盟监控通过联盟网站预订及相关管理信息。联盟应妥善保管账号信息，未经授权不得泄露。发现安全漏洞应立即通知 Wink。

4.1.9 联盟在 Wink 网站或联盟网站上的预订受 Wink 使用条款约束。联盟不得将 Wink 网站或内容用于除收取佣金外的商业用途。

4.1.10 联盟同意本条款中限制性承诺对 Wink 具有重大意义，特别是为 Wink 提供服务、内容及知识产权的意愿及保护 Wink 商誉、产品、服务和市场声誉。联盟应确保其及其集团内人员遵守本条款。

**4.2 商誉、品牌保护及流量**

4.2.1 为保护 Wink 产品、服务、品牌及商誉，联盟承诺其网站（包括集团内其他网站，除微站外）与 Wink 网站在外观、色彩、字体、设计、布局、按钮、标志等方面明显不同。联盟不得模仿 Wink 网站。联盟应自费及时配合 Wink 合理要求修改任何可能引起混淆的内容。

**4.3 知识产权**

4.3.1 联盟承认 Wink 及其许可方拥有 Wink 知识产权及 Wink 网站体现的所有权利，包括 Wink 标志、内容及 Wink 数据。协议不转让任何权利给联盟。

4.3.2 联盟不得将内容和 Wink 数据与自身内容或 Wink 竞争对手内容合并、使用或提供给 Wink 竞争对手，除非本协议明确允许。联盟不得修改内容或包含指向 Wink 竞争对手网站的链接。

4.3.3 联盟及其集团不得注册、使用与“Wink”相同或极易混淆的互联网域名。

4.3.4 签署本协议不视为 Wink 放弃其对 Wink 知识产权的任何权利。

**4.4 推广与营销**

4.4.1 协议期间，联盟及其集团不得进行 (a) 付费搜索、SEM或SEO活动，(b) 不正当影响第三方平台结果的活动，或 (c) 任何其他形式的在线定向广告，涉及以下内容：

1. 服务；  
2. Wink 网站；  
3. 内容；  
4. Wink 数据；  
5. Wink 品牌；  
6. 住宿提供商品牌（除非获得该品牌所有者书面同意）；  
7. 联盟网站，若付费搜索、SEM、SEO或其他在线广告与住宿预订或住宿信息相关。

本条款中第(i)至(v)项在协议终止后仍有效。

**5. 佣金**

5.1 Wink 向联盟支付由住宿提供商设定的已实现交易预订价值的佣金百分比。支付条款另行约定。

5.2 联盟佣金百分比由住宿提供商设定，默认10%，双方可另行约定。

5.3 佣金计算公式：

佣金 = 预订价值 × 佣金百分比  
净佣金（支付）= 佣金 - 支付服务费 - 预订费

**“预订价值”** 指预订金额  
**“佣金百分比”** 指联盟佣金百分比  
**“预订费”** 指1.5%  
**“支付服务费”** 指4%

5.4 Wink 跟踪所有交易并合理努力确保已实现交易被正确记录和支付。Wink 记录为最终记录，除非 Wink 存在重大不当行为或欺诈，联盟无权就未准确跟踪提出赔偿。
