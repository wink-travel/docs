## 开始搜索

以下是您如何在 Wink 上开始搜索供应商和库存。

1. 从您的[账户列表](https://studio.wink.travel/company/list)中选择您想要合作的加盟账户
2. 通过以下两种方式之一进入搜索页面：
    1. 您可以直接从仪表盘快速搜索物业，或者...
    2. 从主导航栏点击库存 → 搜索
3. 搜索页面将显示 Wink 上最新的供应商和库存以及可供选择的类别。
4. 在我们的示例中，点击标有 `Hotels` 的卡片。
5. 您将被重定向到搜索结果页面。在这里，您可以浏览 `Hotels` 类别中可用的供应商，并使用屏幕左侧的筛选器应用或更改搜索过滤条件。

## 调整搜索结果

以下是您可以用来筛选库存和供应商的所有方式。点击搜索结果右上角的筛选图标。

- 物业名称（*例如 Centara*）
- 库存名称（*例如 Villa*）
- 库存类型（*例如 Spa*）
- 星级评分（*例如 5 星级物业*）
- 用户评价评分（*例如 8 / 10*）
- 地理位置（*例如选择地图位置*）
- 大洲（*例如北美*）
- 国家（*例如德国或法国*）
- 城市（*例如东京*）
- 生活方式（*例如 LGBTQ*）
- 位置类型（*例如机场*）
- 细分类型（*例如中档*）
- 类别类型（*例如旅馆*）
- 风格（*例如装饰艺术*）
- 仅显示直接关系（*例如显示直接库存*）
- 允许携带宠物（*例如显示允许携带宠物的库存*）
- 环保（*例如显示环保库存*）
- 适合儿童（*例如显示适合儿童的库存*）
- 热门（*例如显示高价值库存*）

## 操作

操作允许您与库存或供应商互动。我们有三大类操作，分别允许您执行以下操作：

1. **供应商详情** 我如何了解更多供应商信息？
2. **创建...** 我如何将此库存转换为可销售的产品？
3. **添加到...** 我如何将此库存保存到可同时销售的列表中？

- **请求直接连接** 询问供应商是否愿意为您提供直接连接。
- **跟踪绩效** 创建分析图表以跟踪供应商的绩效变化。

- **添加到现有列表** 将供应商或库存添加到您已创建的列表中。
- **新建精选列表** 系统会提示您创建新列表，并将该项目添加进去。

接下来的章节将向您展示如何通过多种方式开始将搜索结果变现。

## 空结果

如果您正在寻找某个供应商但找不到，系统会显示一个填写字段，帮助您在 Google Places 上查找。如果该供应商存在于 Google Places，您可以在 Wink 上生成一个 `线索`。
这表示有人希望我们联系该供应商并邀请他们加入 Wink。我们会联系供应商并请求他们加入。供应商加入后，您将收到通知。

## API

想要 `浏览库存` 的开发者可以前往 [开发者 > API > 浏览](/zh-CN/developers/apis/#browse-api)。
