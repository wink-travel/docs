网格代表供应商和库存列表，并通过可预订的交互式网格 UI 显示详细信息。
网格支持显示您精选列表或已保存搜索中的任何库存。
用户与网格的交互方式与与个人的交互方式相同 [卡片](/studio/cards) 还有一个附加功能：

1. 通过点击`Show more` 按钮 （*当有更多商品可用时*）。

## 网格类型

网格有三种类型：

1. 基于网格 [精选列表](/studio/curated-lists#list-actions) 。
2. 基于网格 [已保存的搜索](/studio/saved-searches#list-actions) 。
3. 基于位置和排序标准的网格（*即排名网格*）。

## 精选列表网格

这是一个网格，它使用您在其中一个精选列表中收集的库存，并将该列表转换为可向用户显示的可预订旅行库存。

### 创造

导航至`Inventory > Curated Lists`从主导航栏。在本例中，我们将使用您的`Favorites`列表。
如果您尚未添加任何内容到您的`Favorites`，前往 [搜索](/studio/search) 学习如何。

<figure>
  <CloudinaryImage width="350" height="394" resize="scale" quality="best" alt="创建策展列表网格" src="academy/create-curated-list-grid" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*创建精选列表网格表单*</figcaption>
</figure>


<Steps>
  1. 点击 `Actions` 按钮 `Favorites` 列表。
  2. 点击 `Create a grid` 按钮。
  3. 将显示一个新窗口，允许您命名网格。 *见下文*。
  4. 点击 `OK` 按钮继续。
</Steps>


您的网格已创建。导航至`Tools > Grids`从主导航栏点击`Curated Grids`选项卡来查看您的新网格。

### 定制

该表单允许您通过以下方式自定义网格：

<Steps>
  1. 给它起一个名字，以便您记住网格的含义。
  2. 选择 [定制](/studio/customization) 您想要应用到这个网格。
  3. 选择您希望用户首先看到的初始卡面。默认情况下，它是该库存的原生卡面。
  4. 选择您想在卡片上使用的徽章。徽章允许用户根据汇总指标比较库存，例如 `eco-friendly`。
  5. 添加关键字，以逗号分隔，用于 [网络爬虫](https://developer.mozilla.org/en-US/docs/Glossary/Crawler) 。
  6. 使用您希望用户看到的语言添加标题和描述。
  7. 点击 `Save` 按钮继续。
</Steps>

## 排名网格

### 创造

通过导航到创建排名网格`Tools > Grids`然后点击`Ranked Grid`选项卡。点击`Create ranked grid` 按钮。

### 定制

该表单允许您通过以下方式自定义排名网格：

<Steps>
  1. 选择目的地。 *例如东京*。
  2. 给它起一个名字，以便您记住网格的含义。 *例如东京的环保酒店*
  3. 选择 [定制](/studio/customization) 您想要应用到这个网格。
  4. 选择您希望用户首先看到的初始卡面。默认情况下，它是该库存的原生卡面。
  5. 选择按什么特征对属性进行排序。 *例如环保*。
  6. 添加关键字，以逗号分隔，用于 [网络爬虫](https://developer.mozilla.org/en-US/docs/Glossary/Crawler) 。
  7. 使用您希望用户看到的语言添加标题和描述。
  8. 点击 `Save` 按钮继续。
</Steps>


保存排名网格后，您将被重定向到排名网格页面，现在可以与全世界共享您的网格了。
