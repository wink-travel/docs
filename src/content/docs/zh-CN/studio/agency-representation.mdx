代理机构是一个普通的 Wink 账户，您可以将其分配给您的账户以进行代表。以下是您希望代理机构代表您的几个原因：

- 第三方对您的 Wink 账户进行全部或部分管理。
- 人才与其代理机构已有协议，并希望 Wink 处理通过 Wink 进来的资金安排的财务方面。
- 人才代理机构负责所有在线工作并管理 Wink 上的所有账户。

## 分配代理机构

以下是如何将代理机构分配给您的账户。

<Steps>
 1. 登录您的用户账户，访问 [Wink Studio](https://studio.wink.travel)。
 2. 选择您想合作的附属账户。
 3. 点击主导航栏中的 `Account > Managers`。
 4. 在 `Agency` 部分，输入您希望代表您的代理机构名称。
 5. 从下拉菜单中选择代理费率。
 5. 可选地添加过期规则。
 7. 点击 `Set agency` 按钮继续。
 </Steps>

 :::note
您需要是账户所有者才能分配代理机构。
 :::

 :::tip[示例]
代理费率来源于附属佣金。

预订金额 = $100，佣金 = 10%，代理费 = 10% => 100 * 0.1 * 0.1 = $1
 :::

## 取消分配代理机构

以下是如何从您的账户中取消分配代理机构。

<Steps>
 1. 登录您的用户账户，访问 [Wink Studio](https://studio.wink.travel)。
 2. 选择您想合作的附属账户。
 3. 点击主导航栏中的 `Account > Managers`。
 4. 在 `Agency` 部分，点击代理机构下拉菜单右侧的 <Badge text="x" variant="danger" size="small" /> 图标。
 5. 点击 `Set agency` 按钮继续。
 </Steps>
