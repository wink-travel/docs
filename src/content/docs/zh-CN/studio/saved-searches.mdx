已保存的搜索顾名思义。搜索旅游库存时，您可以选择保存搜索查询以备后用。

## 保存您的搜索查询

当您开始搜索库存（参见 [搜索](/zh-CN/studio/search)）时，您可以微调搜索筛选条件。
查询表单底部有一个选项 `保存搜索`。

点击此链接，您可以为此搜索命名（*例如：我的魔法酒店库*），以便以后记忆。

<figure>

    <figcaption>*为您的搜索命名*</figcaption>
</figure>

输入名称后，点击 `保存搜索` 按钮。之后，您可以从主导航栏进入 `库存 > 已保存的搜索` 查看新创建的记录。

## 列表操作

您的已保存搜索可以用来以网格或地图形式向用户展示旅游库存，并让您的搜索结果以您想要的方式显示。

<figure>
<CloudinaryImage
      width="700"
      height="939"

      src="academy/saved-search-actions"

    />
    <figcaption>*未应用筛选器的已保存搜索查询*</figcaption>
</figure>

每个已保存的搜索查询都有可执行的操作：

- 将搜索查询转换为 [地图](/zh-CN/studio/maps)。
- 将搜索查询转换为 [网格](/zh-CN/studio/grids)。
- 删除已保存的搜索。

## API

想要管理已保存搜索的开发者可以前往 [开发者 > API > 浏览](/zh-CN/developers/apis/#browse-api)。
