## 配置

大多数情况下，无需直接分享这些应用程序。我们创建了多种分享旅行清单的方式，您可以在[Wink工作室](/studio/what-is-studio)或通过[WinkLinks](/link-manager/wink-links)。

但是，如果您想手动链接到其中一个网站，请确保将您的客户端 ID 和自定义 ID 作为参数添加到 URL。

您可以通过以下方式找到您的客户端 ID：

<Steps>
  1. 登录 [Wink 外联网](/extranet/what-is-extranet) 或者 [Wink工作室](/studio/what-is-studio)。
  2. 点击右上角的个人资料图标，然后点击 [应用](/developers/applications)。
  3. 使用与您想要使用的 Wink 帐户绑定的应用程序。
  4. 从该应用程序复制客户端 ID，并将其粘贴到 URL 上作为请求参数。示例：?client-id=MY\_CLIENT\_ID
</Steps>


### 外联网

您可以通过以下方式找到您的自定义 ID 作为属性：

<Steps>
  1. 登录 [Wink 外联网](/extranet/what-is-extranet)。
  2. 选择您想要处理的属性。
  3. 在主导航栏中，单击 `Account > Branding`
  4. 复制自定义 ID 并将其粘贴到 URL 上作为请求参数。示例：?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### 工作室

您可以通过以下方式找到您的附属定制 ID：

<Steps>
  1. 登录 [Wink工作室](/studio/what-is-studio)。
  2. 选择您想要使用的帐户。
  3. 在主导航栏中，单击 `Tools > Customization`
  4. 从您要使用的自定义中复制自定义 ID，并将其粘贴到 URL 上作为请求参数。示例：?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## 空中交通管制

最终得到的 URL 类似于以下内容：

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

如果您想直接进入酒店页面，URL 将如下所示：

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::注意
如果您未添加客户端和/或配置 ID，Wink OTA 将默认使用默认配置。
:::

## 国际教育局

如果您是一家物业管理公司，您希望链接到 Wink IBE 主页。主页将显示您指定的主题以及[您所代表的财产](/studio/agency-representation)。

最终得到的 URL 类似于以下内容：

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

如果您是一家房地产公司，您希望直接链接到 Wink IBE 房地产登陆页面。

该 URL 看起来类似于以下内容：

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::注意
如果您未添加客户端和/或配置 ID，Wink IBE 将无法加载。这些参数是必需的。
:::
