预订引擎有两种版本，以支持加盟商、物业管理公司和酒店的不同需求。

## 在线旅行社

对于加盟商来说，使用在线旅行社版本最为合适。原因是该网站设置为满足希望查看多个物业的旅行者需求。

您可以在下面找到此应用的主页。

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## 网络预订引擎

对于酒店或物业管理公司来说，使用网络预订引擎版本最为合适。这类用户希望突出展示单个物业或管理下的多个物业。

您可以在下面找到此应用的主页。

<LinkButton href="https://book.wink.travel/home?client-id=client-1&mp;configuration-id=d906a163-5cee-4198-a170-2bedf2407f5f" variant="secondary" icon="external" target="_blank">
  Wink IBE
</LinkButton>

## 配置

大多数情况下，无需直接分享这两个应用中的任何一个。我们创建了多种方式供您分享旅游库存，您可以在 [Wink Studio](/zh-CN/studio/what-is-studio) 或通过 [WinkLinks](/zh-CN/link-manager/wink-links) 发现这些方式。

但是，如果您想手动链接到这些网站之一，请确保在 URL 中添加您的客户端 ID 和自定义 ID 作为参数。

您可以通过以下方式找到您的客户端 ID：

<Steps>
1. 登录 [Wink Extranet](/zh-CN/extranet/what-is-extranet) 或 [Wink Studio](/zh-CN/studio/what-is-studio)。
2. 点击右上角的个人资料图标，然后点击 [Applications](/zh-CN/developers/applications)。
3. 使用与您想要操作的 Wink 账户关联的应用程序。
4. 复制该应用程序中的客户端 ID，并将其作为请求参数粘贴到 URL 中。例如：?client-id=MY_CLIENT_ID
</Steps>

### Extranet

您可以作为物业找到您的自定义 ID：

<Steps>
1. 登录 [Wink Extranet](/zh-CN/extranet/what-is-extranet)。
2. 选择您想要操作的物业。
3. 在主导航栏中，点击 `Account > Branding`
4. 复制自定义 ID，并将其作为请求参数粘贴到 URL 中。例如：?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

您可以作为加盟商找到您的自定义 ID：

<Steps>
1. 登录 [Wink Studio](/zh-CN/studio/what-is-studio)。
2. 选择您想要操作的账户。
3. 在主导航栏中，点击 `Tools > Customization`
4. 复制您想使用的自定义的自定义 ID，并将其作为请求参数粘贴到 URL 中。例如：?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

最终您得到的 URL 看起来类似于：

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

如果您想直接进入某个酒店页面，URL 看起来类似于：

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
如果您未包含客户端和/或自定义 ID，Wink OTA 将默认使用默认配置。
:::

## IBE

如果您是物业管理公司，您需要链接到 Wink IBE 主页。主页将显示您指定的主题以及您所代表的[物业](/zh-CN/studio/agency-representation)。

最终您得到的 URL 看起来类似于：

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

如果您是物业，您需要直接链接到 Wink IBE 物业登录页面。

该 URL 看起来类似于：

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
如果您未包含客户端和/或自定义 ID，Wink IBE 将无法加载。这些参数是必需的。
:::

登录页面包含旅行者做出是否入住该物业的明智选择所需的所有信息。
