**Wink Drupal 模块** 将 Wink 旅游库存组件作为原生区块添加到您的 Drupal 站点。编辑者可以通过标准区块系统将它们放置在布局中的任何位置。

## 要求

- Drupal 10 或 11
- PHP 8.1+
- Composer

## 安装

<Steps>
1. 通过 Composer 安装：
   ```bash
   composer require drupal/wink
   ```
2. 启用模块：
   ```bash
   drush en wink -y
   ```
3. 进入 **管理 → 配置 → Web 服务 → Wink 设置** (`/admin/config/wink`)。
4. 输入您的 **Client ID** 和 **Client Secret**，选择您的环境，然后点击 **保存**。
   访问 [应用程序](/zh-CN/portal/settings/applications) 获取您的凭证。
</Steps>

## 向页面添加区块

<Steps>
1. 进入 **管理 → 结构 → 区块布局**。
2. 在您想放置 Wink 内容的区域点击 **放置区块**。
3. 搜索 **Wink** 以查找可用区块：
   - **Wink Content** — 渲染酒店卡片、网格或地图
   - **Wink Lookup** — 目的地搜索栏
   - **Wink Search** — 行程选择按钮
   - **Wink Account** — 登录按钮
   - **Wink Itinerary** — 行程状态按钮
   - **Wink Cart** — 购物车按钮
4. 对于 **Wink Content**，在区块设置中配置 **布局类型** 和 **布局 ID**。
5. 保存区块配置。
</Steps>

## 可用区块

| 区块 | 作用 |
|---|---|
| **Wink Content** | 显示酒店卡片、排名网格、地图等 |
| **Wink Lookup** | 目的地和酒店搜索栏 |
| **Wink Search** | 打开行程选择器的图标按钮 |
| **Wink Account** | 登录 / 用户账户按钮 |
| **Wink Itinerary** | 显示当前行程的行程按钮 |
| **Wink Cart** | 购物车摘要按钮 |
