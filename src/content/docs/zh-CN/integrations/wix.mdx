**Wink Wix 应用** 将 Wink Web 组件作为原生小部件添加到您的 Wix 站点，您可以在 Wix 编辑器中拖放使用。站点所有者无需编码。

## 安装

<Steps>
1. 访问 [Wix 应用市场](https://www.wix.com/app-market)，搜索 **Wink Travel**。
2. 点击 **添加到站点**，并按照提示授权应用。
3. 安装完成后，打开 **仪表盘 → Wink 设置**，输入您的 **Client ID** 和 **Client Secret**。
   访问 [应用](/zh-CN/portal/settings/applications) 获取您的凭证。
4. 点击 **保存**，然后点击 **测试连接** 以确认 Wink 已连接。
</Steps>

## 向页面添加组件

<Steps>
1. 打开 **Wix 编辑器**，进入您想添加 Wink 内容的页面。
2. 点击 **添加 (+) → 应用 → Wink Travel**，查看所有可用小部件。
3. 将小部件拖放到页面上——例如，使用 **Wink Content** 显示酒店网格。
4. 在小部件设置面板中，选择 **布局类型**，并粘贴来自 Wink Studio 的 **布局 ID**。
5. 点击 **发布** 使更改生效。
</Steps>

## 可用小部件

| 小部件 | 作用 |
|---|---|
| **Wink Content** | 显示酒店卡片、排名网格、地图等 |
| **Wink Lookup** | 目的地和酒店搜索栏 |
| **Wink Search** | 打开行程选择器的图标按钮 |
| **Wink Account** | 登录 / 用户账户按钮 |
| **Wink Itinerary** | 显示当前行程的行程按钮 |
| **Wink Cart** | 购物车摘要按钮 |
