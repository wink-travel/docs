**Wink Shopify 应用** 让旅游运营商、体验销售商和酒店商家能够直接在其 Shopify 店面展示实时 Wink 库存——可与产品并列展示，放在登录页或主题中的任何位置。

## 安装

<Steps>
1. 访问 [Shopify 应用商店](https://apps.shopify.com)，搜索 **Wink Travel**。
2. 点击 **添加应用** 并按照 OAuth 流程授权。
3. 在 Wink 应用设置中，输入您的 **Client ID** 和 **Client Secret**。
   访问 [Applications](/zh-CN/portal/settings/applications) 获取您的凭证。
4. 点击 **验证并保存**。成功消息确认连接。
</Steps>

## 启用应用嵌入块

应用嵌入块会自动将 Wink 脚本和样式表注入每个页面——无需编辑主题代码。

<Steps>
1. 在 Shopify 管理后台，进入 **在线商店 → 主题 → 自定义**。
2. 点击左侧面板的 **应用嵌入**。
3. 切换开启 **Wink Travel**。
4. 点击 **保存**。
</Steps>

:::note
应用嵌入块从安装时设置的 Shopify 元字段读取您的 Client ID。您的 Client Secret 永远不会暴露给浏览器。
:::

## 向页面添加组件

<Steps>
1. 在主题编辑器中，打开您想放置 Wink 内容的页面或部分。
2. 点击 **添加区块**，选择以下 Wink 区块：
   - **Wink Content** — 酒店卡片、网格或地图
   - **Wink Lookup** — 目的地搜索栏
   - **Wink Search** — 行程选择按钮
   - **Wink Account** — 登录按钮
   - **Wink Itinerary** — 行程状态按钮
   - **Wink Cart** — 购物车按钮
3. 对于 **Wink Content**，选择一个 **布局类型** 并输入来自 Wink Studio 的 **布局 ID**。
4. 点击 **保存**。
</Steps>

## 可用主题区块

| 区块 | 作用 |
|---|---|
| **Wink Content** | 显示酒店卡片、排名网格、地图等 |
| **Wink Lookup** | 目的地和酒店搜索栏 |
| **Wink Search** | 打开行程选择器的图标按钮 |
| **Wink Account** | 登录 / 用户账户按钮 |
| **Wink Itinerary** | 显示当前行程的按钮 |
| **Wink Cart** | 购物车摘要按钮 |
