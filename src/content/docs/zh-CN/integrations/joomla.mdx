**Wink Joomla 扩展** 是一个包含系统插件（加载 CDN 脚本）、内容插件（处理文章中的短代码标签）和组件（管理设置页面）的安装包。

  description="源代码、开发者安装指南及贡献说明。"
/>

## 要求

- Joomla 4 或 5
- PHP 8.1+

## 安装

<Steps>
1. 从[GitHub 发布页面](https://github.com/wink-travel/wink-joomla-extension/releases)下载最新的 `pkg_wink-x.x.x.zip`。
2. 在 Joomla 管理后台，进入 **扩展 → 管理 → 安装**。
3. 上传 zip 文件并点击 **上传并安装**。
4. 两个插件会自动启用。进入 **组件 → Wink 设置**。
5. 输入您的 **Client ID** 和 **Client Secret**，然后点击 **保存**。
   访问 [应用](/zh-CN/portal/settings/applications) 获取您的凭证。
</Steps>

## 在文章中嵌入组件

在文章内容中任意位置使用短代码样式标签：

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

内容插件会在渲染时将这些标签替换为正确的 `<wink-*>` HTML。系统插件会自动加载 CDN 脚本并在页面底部注入 `<wink-app-loader>`。

## 可用短代码

| 短代码 | 输出 | 描述 |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | 酒店卡片、网格、地图 |
| `{wink-lookup}` | `<wink-lookup>` | 目的地搜索栏 |
| `{wink-search}` | `<wink-search-button>` | 行程选择按钮 |
| `{wink-account}` | `<wink-account-button>` | 登录 / 用户账户 |
| `{wink-itinerary}` | `<wink-itinerary-button>` | 行程状态按钮 |
| `{wink-cart}` | `<wink-shopping-cart-button>` | 购物车按钮 |
