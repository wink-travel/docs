**Wink TYPO3 扩展** 将 Wink 旅游库存作为原生内容元素集成到 TYPO3 13 中。编辑者使用标准的 TYPO3 页面编辑器将 Wink 组件添加到页面，无需更改模板。

## 要求

- TYPO3 13.x
- PHP 8.2+
- Composer

## 安装

<Steps>
1. 通过 Composer 安装：
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. 激活扩展：
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. 在 TYPO3 后台，进入 **管理工具 → 设置 → 扩展配置 → wink**。
4. 输入您的 **Client ID**、**Client Secret**，并选择您的环境。保存。
   访问 [Applications](/zh-CN/portal/settings/applications) 获取您的凭证。
5. 将静态 TypoScript 模板添加到您的根模板：
   **模板 → 编辑 → 包含 → 可用项目 → Wink Travel**。
</Steps>

## 向页面添加内容元素

<Steps>
1. 在 **页面** 模块中打开一个页面。
2. 点击内容区域的 **+** 按钮添加新元素。
3. 选择一个 **Wink Travel** 内容元素：
   - **Wink Content** — 酒店卡片、排名网格、地图
   - **Wink Lookup** — 目的地搜索栏
   - **Wink Search** — 行程选择按钮
   - **Wink Account** — 登录按钮
   - **Wink Itinerary** — 行程状态按钮
   - **Wink Cart** — 购物车按钮
4. 对于 **Wink Content**，使用 FlexForm 字段选择 **布局类型** 并输入 **布局 ID**。
5. 保存并发布页面。
</Steps>

## 可用内容元素

| 元素 | 作用 |
|---|---|
| **Wink Content** | 酒店卡片、排名网格、地图等 |
| **Wink Lookup** | 目的地搜索栏 |
| **Wink Search** | 打开行程选择器的图标按钮 |
| **Wink Account** | 登录 / 用户账户按钮 |
| **Wink Itinerary** | 显示当前行程的行程按钮 |
| **Wink Cart** | 购物车摘要按钮 |
