如果您想更新 Wink 用户帐户的用户个人资料，请按照以下步骤操作：

<Steps>
  1. 您必须登录我们的某个网站。
  2. 单击右上角的个人资料图标，将出现一个菜单。
  3. 点击 `Profile` 关联。
  4. 按照说明更新您的 `e-mail`， `first` 和 `last name`。
  5. 您还可以上传新的 `profile picture` 你自己。
  6. 点击 `Save`。
</Steps>


:::注意
您的新头像将在下次您完全退出 Wink 时显示。
:::
