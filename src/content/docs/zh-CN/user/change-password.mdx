如果您想更改您的 Wink 用户账户密码，请按照以下步骤操作：

## 更新密码

<Steps>
1. 您必须已登录我们的网站之一。
2. 点击右上角的个人资料图标，会出现一个菜单。
3. 点击 `更改密码` 链接。
4. 按照指示更改您的密码。
5. 点击 `保存`。
</Steps>

下次登录时，您需要使用新密码。

## 忘记密码

有时需要重置用户密码。

以下是重置用户密码的步骤：

<Steps>
1. 在登录页面点击 `忘记密码` 链接。
2. 输入您要重置的账户的用户名/电子邮件。
3. 点击 `忘记密码` 按钮。*您将被重定向到重置密码表单。*
4. 检查您的电子邮件。
</Steps>

2. 确认您的新密码。
3. 点击 `重置密码` 按钮。*您将被重定向回登录页面。*
</Steps>
