如果您想更改 Wink 用户帐户的密码，请按照以下步骤操作：

## 更新密码

<Steps>
  1. 您必须登录我们的某个网站。
  2. 单击右上角的个人资料图标，将出现一个菜单。
  3. 点击 `Change password` 关联。
  4. 按照说明更改您的密码。
  5. 点击 `Save`。
</Steps>


下次登录时，您将需要新密码。

## 忘记密码

有时需要重置用户密码。

以下是重置用户密码的步骤：

<Steps>
  1. 点击 `Forgot password` 来自登录页面的链接。
  2. 输入您想要重置的帐户的用户名/电子邮件。
  3. 点击 `Forgot password` 按钮。 *您将被重定向到重置密码表单。*
  4. 检查你的电子邮件。
</Steps>

  <CloudinaryImage width="350" height="354" resize="scale" quality="best" alt="忘记密码" src="academy/forgot-password" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*忘记密码表格*</figcaption>

<Steps>
  1. 将您在电子邮件中收到的代码与新密码一起粘贴到新表单中。
  2. 确认您的新密码。
  3. 点击 `Reset password` 按钮。 *您将被重定向回登录页面。*
</Steps>

您的密码已成功重置。您可以使用新的凭证登录 Wink。
