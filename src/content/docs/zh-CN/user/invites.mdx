Wink 用户可以选择让您管理他们的酒店、关联公司或旅行社账户。届时，您将收到一封包含邀请请求的电子邮件。

import { Steps } from '@astrojs/starlight/components';

要查看您当前的 Wink 邀请，请按照以下步骤操作：

<Steps>
  1. 您必须登录我们的某个网站。
  2. 单击右上角的个人资料图标，将出现一个菜单。
  3. 点击 `Invites` 关联。
  4. 通过以下方式与您的邀请互动 `Accepting` 或者 `Rejecting` 他们。
</Steps>
