Wink 用户可以选择让您管理他们的某个物业、加盟或旅行社账户。当发生这种情况时，您将收到一封包含邀请请求的电子邮件。

import { Steps } from '@astrojs/starlight/components';

要查看您当前的 Wink 邀请，请按照以下步骤操作：

<Steps>
1. 您必须登录我们的网站之一。
2. 点击右上角的个人资料图标，菜单将会出现。
3. 点击 `Invites` 链接。
4. 通过“接受”或“拒绝”来处理您的邀请。
</Steps>
