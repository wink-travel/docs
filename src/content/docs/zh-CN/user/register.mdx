您可以通过我们的任何门户创建用户账户。账户创建后，可在所有 Wink 应用和网站中使用。

## 旅客

请按照以下步骤操作：

<Steps>
1. 访问 [https://ota.wink.travel](https://ota.wink.travel)
2. 点击右上角的 `Sign-In / Register`
3. 系统会要求您填写基本用户信息。填写所有必填项后点击 `Save`。
4. 现在您可以使用用户名（电子邮件）和密码登录 Wink。
5. 登录后，系统会将您重定向回您之前所在的位置。
</Steps>

## 注册

创建 Wink 用户账户可让您访问平台上的所有功能。您可以创建无限数量的物业和合作伙伴账户。

<figure>
<CloudinaryImage
      width="350"
      height="553"
      resize="scale"
      quality="best"
      alt="登录"
      src="academy/sign-in"
      errorImage="noimage_opaque_nyrtl0.png"
    />
    <figcaption>*登录表单*</figcaption>
</figure>

<Steps>
1. 第一次被引导到我们的授权服务器时，系统会要求您输入用户名和密码。
2. 请点击 `Create account` 按钮。
3. 填写所有必填项后点击 `Register` 继续。此操作会带您回到登录表单。
4. 输入您的电子邮件/用户名和密码登录。
5. 您将被重定向回您之前所在的网站。
</Steps>
