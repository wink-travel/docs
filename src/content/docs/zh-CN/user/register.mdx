您可以通过我们的任何门户创建用户帐户。创建后，此帐户可在 Wink 的所有应用和网站上使用。

## 旅行者

请按照以下步骤操作：

<Steps>
  1. 前往 [https://ota.wink.travel](https://ota.wink.travel)
  2. 点击 `Sign-In / Register` 在右上角
  3. 系统将要求您输入基本用户详细信息。填写所有必填字段，然后点击 `Save`。
  4. 您现在可以使用您的用户名（电子邮件）和密码登录 Wink。
  5. 一旦登录，系统就会将您重定向回原来的位置。
</Steps>

## 登记

创建 Wink 用户帐户即可访问平台上的所有功能。您可以创建无限数量的房产和联盟账户。

<figure>
  <CloudinaryImage width="350" height="553" resize="scale" quality="best" alt="登入" src="academy/sign-in" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*登录表单*</figcaption>
</figure>


<Steps>
  1. 第一次将您引导至我们的授权服务器时，系统会要求您输入用户名和密码。
  2. 相反，点击 `Create account` 按钮。
  3. 填写所有必填字段并点击 `Register` 继续。这将带您返回登录表单。
  4. 输入您的电子邮件/用户名和密码登录。
  5. 您将被重定向回原来的站点。
</Steps>
