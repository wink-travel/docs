Jika anda ingin mengemas kini profil pengguna untuk akaun pengguna Wink anda, ikut langkah-langkah berikut:

<Steps>
1. Anda mesti log masuk ke salah satu laman web kami.
2. Klik ikon profil anda, di penjuru kanan atas dan menu akan muncul.
3. Klik pautan `Profil`.
4. Ikuti arahan untuk mengemas kini `e-mel`, `nama pertama` dan `nama terakhir`.
5. Anda juga boleh memuat naik `gambar profil` baru diri anda.
6. Klik `Simpan`.
</Steps>

:::note
Gambar profil baru anda akan dipaparkan pada kali seterusnya anda log keluar sepenuhnya dari Wink.
:::
