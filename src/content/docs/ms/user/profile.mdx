Jika anda ingin mengemas kini profil pengguna anda untuk akaun pengguna Wink anda, ikut langkah ini:

<Steps>
  1. Anda mesti log masuk ke salah satu laman web kami.
  2. Klik ikon profil anda, di penjuru kanan sebelah atas dan menu akan muncul.
  3. Klik pada `Profile` pautan.
  4. Ikut arahan untuk mengemas kini anda `e-mail`, `first` dan `last name`.
  5. Anda juga boleh memuat naik yang baharu `profile picture` diri sendiri.
  6. klik `Save`.
</Steps>


:::nota
Gambar profil baharu anda akan dipaparkan pada kali seterusnya anda log keluar sepenuhnya daripada Wink.
:::
