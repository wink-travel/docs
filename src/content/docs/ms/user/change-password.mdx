Jika anda ingin menukar kata laluan untuk akaun pengguna Wink anda, ikut langkah-langkah berikut:

## Kemas kini kata laluan

<Steps>
1. Anda mesti log masuk ke salah satu laman web kami.
2. Klik ikon profil anda, di penjuru kanan atas dan menu akan muncul.
3. Klik pautan `Tukar kata laluan`.
4. Ikuti arahan untuk menukar kata laluan anda.
5. Klik `Simpan`.
</Steps>

Kali seterusnya anda log masuk, anda perlu menggunakan kata laluan baru anda.

## Lupa kata laluan

Kadang-kadang perlu untuk menetapkan semula kata laluan pengguna.

Berikut adalah langkah-langkah untuk menetapkan semula kata laluan pengguna anda:

<Steps>
1. Klik pada pautan `Lupa kata laluan` dari halaman log masuk.
2. Taipkan nama pengguna / e-mel untuk akaun yang anda ingin tetapkan semula.
3. Klik butang `Lupa kata laluan`. *Anda akan dialihkan ke borang tetapan semula kata laluan.*
4. Semak e-mel anda.
</Steps>

<Steps>
1. Tampal kod yang anda terima dalam e-mel ke dalam borang baru bersama dengan kata laluan baru anda.
2. Sahkan kata laluan baru anda.
3. Klik butang `Tetapkan semula kata laluan`. *Anda akan dialihkan kembali ke halaman log masuk.*
</Steps>
