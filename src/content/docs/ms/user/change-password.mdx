Jika anda ingin menukar kata laluan anda untuk akaun pengguna Wink anda, ikuti langkah berikut:

## Kemas kini kata laluan

<Steps>
  1. Anda mesti log masuk ke salah satu laman web kami.
  2. Klik ikon profil anda, di penjuru kanan sebelah atas dan menu akan muncul.
  3. Klik pada `Change password` pautan.
  4. Ikut arahan untuk menukar kata laluan anda.
  5. klik `Save`.
</Steps>


Kali seterusnya anda log masuk, anda memerlukan kata laluan baharu anda.

## Terlupa kata laluan

Kadangkala perlu menetapkan semula kata laluan pengguna.

Berikut ialah langkah untuk menetapkan semula kata laluan pengguna anda:

<Steps>
  1. Klik pada `Forgot password` pautan dari halaman log masuk.
  2. Taipkan nama pengguna / e-mel untuk akaun yang ingin anda tetapkan semula.
  3. klik `Forgot password` butang. *Anda dialihkan ke borang tetapan semula kata laluan.*
  4. Semak e-mel anda.
</Steps>

<Steps>
  1. Tampalkan kod yang anda terima dalam e-mel ke dalam borang baharu bersama kata laluan baharu anda.
  2. Sahkan kata laluan baharu anda.
  3. klik `Reset password` butang. *Anda dialihkan semula ke halaman log masuk.*
</Steps>


<figure>
  <CloudinaryImage width="350" height="467" resize="scale" quality="best" alt="Tetapkan semula kata laluan" src="academy/reset-password" errorImage="noimage_opaque_nyrtl0.png" />

Kata laluan anda telah berjaya ditetapkan semula. Anda boleh log masuk ke Wink dengan kelayakan baharu anda.
