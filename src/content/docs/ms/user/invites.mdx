Seorang pengguna Wink boleh memilih untuk membenarkan anda mengurus salah satu akaun hartanah, gabungan atau ejen pelancongan mereka. Apabila itu berlaku, anda akan menerima e-mel dengan permintaan jemputan.

import { Steps } from '@astrojs/starlight/components';

Untuk melihat jemputan Wink anda yang terkini, ikut langkah-langkah berikut:

<Steps>
1. Anda mesti log masuk ke salah satu laman web kami.
2. Klik ikon profil anda, di penjuru kanan atas dan menu akan muncul.
3. Klik pautan `Invites`.
4. Berinteraksi dengan jemputan anda dengan sama ada `Menerima` atau `Menolak` mereka.
</Steps>
