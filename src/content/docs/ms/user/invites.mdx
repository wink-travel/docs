Pengguna Wink boleh memilih untuk membenarkan anda mengurus salah satu akaun hartanah, ahli gabungan atau ejen pelancongan mereka. Apabila itu berlaku, anda akan menerima e-mel dengan permintaan jemputan.

import { Steps } from '@astrojs/starlight/components';

Untuk melihat jemputan Wink semasa anda, ikuti langkah berikut:

<Steps>
  1. Anda mesti log masuk ke salah satu laman web kami.
  2. Klik ikon profil anda, di penjuru kanan sebelah atas dan menu akan muncul.
  3. Klik pada `Invites` pautan.
  4. Berinteraksi dengan jemputan anda dengan sama ada `Accepting` atau `Rejecting` mereka.
</Steps>
