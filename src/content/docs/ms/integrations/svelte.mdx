Svelte menghantar atribut terus ke DOM tanpa lapisan DOM maya, menjadikan elemen tersuai sangat sesuai. Untuk SvelteKit (SSR), muatkan CDN di dalam `onMount` atau lindungi dengan pemeriksaan `browser`.

<LinkCard
  title="@wink/elements npm package"
  href="/integrations/npm"
  description="Pasang dan muatkan bundel CDN dengan jenis TypeScript."
/>

## Pasang

```bash
npm install @wink/elements
```

## SvelteKit — susun atur root

```ts
// src/routes/+layout.ts
import { browser } from '$app/environment';
import { load as loadWink } from '@wink/elements';

export function load() {
  if (browser) {
    loadWink({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  }
}
```

Sebagai alternatif, muatkan di dalam `onMount` dalam `+layout.svelte`:

```svelte
<!-- src/routes/+layout.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<slot />
```

## Gunakan dalam komponen

Svelte menghantar semua atribut yang tidak dikenali terus ke DOM, jadi tiada konfigurasi tambahan diperlukan:

```svelte
<script>
  export let layoutId = 'YOUR_LAYOUT_ID';
</script>

<main>
  <wink-content-loader layout="HOTEL" id={layoutId} />
  <wink-lookup />
</main>
```

## Svelte 5 (sintaks runes)

```svelte
<script lang="ts">
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  let { layoutId }: { layoutId: string } = $props();

  onMount(() => {
    load({ clientId: import.meta.env.PUBLIC_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id={layoutId} />
```

## Pembolehubah persekitaran

SvelteKit hanya mendedahkan pembolehubah yang diawali dengan `PUBLIC_` kepada pelayar:

```
# .env
PUBLIC_WINK_CLIENT_ID=your-client-id
```

<Aside type="caution" title="Rendering SSR">
Jangan sekali-kali render elemen `<wink-*>` semasa rendering sisi pelayan — elemen tersuai didaftarkan dalam daftar `customElements` pelayar yang tidak wujud di pelayan. Bungkus dalam `{#if browser}` atau panggil `load()` hanya di dalam `onMount`.
</Aside>

```svelte
<script>
  import { browser } from '$app/environment';
</script>

{#if browser}
  <wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
{/if}
```

## Bare Svelte (Vite, tanpa SvelteKit)

```svelte
<!-- src/App.svelte -->
<script>
  import { onMount } from 'svelte';
  import { load } from '@wink/elements';

  onMount(() => {
    load({ clientId: import.meta.env.VITE_WINK_CLIENT_ID });
  });
</script>

<wink-content-loader layout="HOTEL" id="YOUR_LAYOUT_ID" />
```
