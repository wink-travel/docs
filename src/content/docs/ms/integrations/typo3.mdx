**Sambungan Wink TYPO3** mengintegrasikan inventori perjalanan Wink ke dalam TYPO3 13 sebagai elemen kandungan asli. Penyunting menambah komponen Wink ke halaman menggunakan penyunting halaman TYPO3 standard — tiada perubahan templat diperlukan.

<LinkCard
  title="wink-travel/wink-typo3-extension on GitHub"
  href="https://github.com/wink-travel/wink-typo3-extension"
  description="Kod sumber, panduan penyediaan pembangun, dan arahan sumbangan."
/>

## Keperluan

- TYPO3 13.x
- PHP 8.2+
- Composer

## Pasang

<Steps>
1. Pasang melalui Composer:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktifkan sambungan:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. Dalam backend TYPO3, pergi ke **Admin Tools → Settings → Extension Configuration → wink**.
4. Masukkan **Client ID**, **Client Secret**, dan pilih persekitaran anda. Simpan.
   Pergi ke [Applications](/ms/portal/settings/applications) untuk mendapatkan kelayakan anda.
5. Tambah templat TypoScript statik ke templat akar anda:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Tambah elemen kandungan ke halaman

<Steps>
1. Buka halaman dalam modul **Page**.
2. Klik butang **+** di kawasan kandungan untuk menambah elemen baru.
3. Pilih salah satu elemen kandungan **Wink Travel**:
   - **Wink Content** — kad hotel, grid berperingkat, peta
   - **Wink Lookup** — bar carian destinasi
   - **Wink Search** — butang pemilih jadual perjalanan
   - **Wink Account** — butang log masuk
   - **Wink Itinerary** — butang status jadual perjalanan
   - **Wink Cart** — butang troli beli-belah
4. Untuk **Wink Content**, gunakan medan FlexForm untuk memilih **Jenis Susun Atur** dan masukkan **ID Susun Atur**.
5. Simpan dan terbitkan halaman.
</Steps>

## Elemen kandungan tersedia

| Elemen | Tujuan |
|---|---|
| **Wink Content** | Kad hotel, grid berperingkat, peta, dan lain-lain |
| **Wink Lookup** | Bar carian destinasi |
| **Wink Search** | Butang ikon yang membuka pemilih jadual perjalanan |
| **Wink Account** | Butang log masuk / akaun pengguna |
| **Wink Itinerary** | Butang jadual perjalanan yang menunjukkan perjalanan semasa |
| **Wink Cart** | Butang ringkasan troli beli-belah |
