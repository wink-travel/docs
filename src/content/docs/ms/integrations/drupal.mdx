**Modul Wink Drupal** menambah komponen inventori perjalanan Wink ke laman Drupal anda sebagai blok asli. Penyunting boleh meletakkannya di mana-mana dalam susun atur melalui sistem blok standard.

<LinkCard
  title="wink-travel/wink-drupal-module on GitHub"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Kod sumber, panduan penyediaan pembangun, dan arahan sumbangan."
/>

## Keperluan

- Drupal 10 atau 11
- PHP 8.1+
- Composer

## Pasang

<Steps>
1. Pasang melalui Composer:
   ```bash
   composer require drupal/wink
   ```
2. Aktifkan modul:
   ```bash
   drush en wink -y
   ```
3. Pergi ke **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Masukkan **Client ID** dan **Client Secret** anda, pilih persekitaran anda, dan klik **Save**.
   Pergi ke [Applications](/ms/portal/settings/applications) untuk mendapatkan kelayakan anda.
</Steps>

## Tambah blok ke halaman

<Steps>
1. Pergi ke **Admin → Structure → Block Layout**.
2. Klik **Place block** di kawasan di mana anda mahu kandungan Wink.
3. Cari **Wink** untuk mencari blok yang tersedia:
   - **Wink Content** — memaparkan kad hotel, grid, atau peta
   - **Wink Lookup** — bar carian destinasi
   - **Wink Search** — butang pemilih jadual perjalanan
   - **Wink Account** — butang log masuk
   - **Wink Itinerary** — butang status jadual perjalanan
   - **Wink Cart** — butang troli beli-belah
4. Untuk **Wink Content**, konfigurasikan **Layout Type** dan **Layout ID** dalam tetapan blok.
5. Simpan konfigurasi blok.
</Steps>

## Blok tersedia

| Blok | Tujuan |
|---|---|
| **Wink Content** | Memaparkan kad hotel, grid berperingkat, peta, dan lain-lain |
| **Wink Lookup** | Bar carian untuk destinasi dan hotel |
| **Wink Search** | Butang ikon yang membuka pemilih jadual perjalanan |
| **Wink Account** | Butang log masuk / akaun pengguna |
| **Wink Itinerary** | Butang jadual perjalanan yang menunjukkan perjalanan semasa |
| **Wink Cart** | Butang ringkasan troli beli-belah |
