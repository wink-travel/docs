**Sambungan Wink Joomla** adalah satu pakej yang memasang plugin sistem (memuatkan skrip CDN), plugin kandungan (memproses tag shortcode dalam artikel), dan komponen (halaman tetapan admin).

<LinkCard
  title="wink-travel/wink-joomla-extension on GitHub"
  href="https://github.com/wink-travel/wink-joomla-extension"
  description="Kod sumber, panduan penyediaan pembangun, dan arahan sumbangan."
/>

## Keperluan

- Joomla 4 atau 5
- PHP 8.1+

## Pasang

<Steps>
1. Muat turun `pkg_wink-x.x.x.zip` terkini dari [halaman pelepasan GitHub](https://github.com/wink-travel/wink-joomla-extension/releases).
2. Dalam Joomla Admin, pergi ke **Extensions → Manage → Install**.
3. Muat naik fail zip dan klik **Upload & Install**.
4. Kedua-dua plugin diaktifkan secara automatik. Pergi ke **Components → Wink Settings**.
5. Masukkan **Client ID** dan **Client Secret** anda dan klik **Save**.
   Pergi ke [Applications](/ms/portal/settings/applications) untuk mendapatkan kelayakan anda.
</Steps>

## Selitkan komponen dalam artikel

Gunakan tag gaya shortcode di mana-mana dalam kandungan artikel:

```
{wink-content layout="HOTEL" id="YOUR_LAYOUT_ID"}

{wink-lookup}

{wink-search}

{wink-account}

{wink-itinerary}

{wink-cart}
```

Plugin kandungan menggantikan tag ini dengan HTML `<wink-*>` yang betul semasa masa render. Plugin sistem secara automatik memuatkan skrip CDN dan menyuntik `<wink-app-loader>` di footer halaman.

## Shortcode tersedia

| Shortcode | Output | Penerangan |
|---|---|---|
| `{wink-content layout="..." id="..."}` | `<wink-content-loader>` | Kad hotel, grid, peta |
| `{wink-lookup}` | `<wink-lookup>` | Bar carian destinasi |
| `{wink-search}` | `<wink-search-button>` | Butang pemilih jadual perjalanan |
| `{wink-account}` | `<wink-account-button>` | Log masuk / akaun pengguna |
| `{wink-itinerary}` | `<wink-itinerary-button>` | Butang status jadual perjalanan |
| `{wink-cart}` | `<wink-shopping-cart-button>` | Butang troli |
