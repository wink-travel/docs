Untuk mengurus teg WinkLinks anda, navigasi ke `WinkLinks` dalam bar navigasi utama dan klik pada `Tags` tab.

Apabila kandungan anda berkembang, anda akan mahu mula menyusunnya. Kami mencipta `tags` untuk tujuan ini. Anda boleh mengumpulkan kandungan menggunakan seberapa banyak tag yang anda mahu.

## Buat tag

<Steps>
  1. Buat teg baharu dengan mengklik butang `Add a new tag` butang.
  2. Masukkan nama teg baharu anda. *cth. Kedai kopi popular di Berlin*
  3. Klik pada `Save` pautan.
</Steps>


## Kemas kini teg

<Steps>
  1. Mulakan dengan mengklik pada `Edit` pautan di sebelah teg yang anda ingin ubah suai.
  2. Kemas kini nama tag anda. *cth. Kedai kopi terbaik di Berlin*
  3. Klik pada `Save` pautan.
</Steps>


## Alih keluar tag

:::nota
Apabila mengalih keluar teg, anda juga akan mengalih keluarnya daripada semua kandungan yang berkaitan.
:::

<Steps>
  1. Klik pada `Remove` pautan di sebelah teg yang ingin anda alih keluar.
</Steps>


## API

Pemaju yang ingin mengurus `WinkLinks` boleh menuju ke [Pembangun > API > WinkLinks](/developers/apis/#winklinks-api).
