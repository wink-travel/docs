Untuk mengurus tag WinkLinks anda, navigasi ke `WinkLinks` di bar navigasi utama dan klik pada tab `Tags`.

Apabila kandungan anda berkembang, anda akan mahu mula menyusunnya. Kami mencipta `tags` untuk tujuan ini. Anda boleh mengelompokkan kandungan menggunakan sebanyak mana tag yang anda mahu.

## Cipta tag

<Steps>
1. Cipta tag baru dengan mengklik butang `Add a new tag`.
2. Masukkan nama tag baru anda. *contoh: Kedai kopi terbaik di Berlin*
3. Klik pautan `Save`.
</Steps>

## Kemas kini tag

<Steps>
1. Mula dengan mengklik pautan `Edit` di sebelah tag yang anda mahu ubah.
2. Kemas kini nama tag anda. *contoh: Kedai kopi terbaik di Berlin*
3. Klik pautan `Save`.
</Steps>

## Buang tag

:::note
Apabila membuang tag, anda juga akan membuangnya dari semua kandungan yang berkaitan.
:::

<Steps>
1. Klik pautan `Remove` di sebelah tag yang anda mahu buang.
</Steps>

## API

Pembangun yang mahu mengurus `WinkLinks` boleh pergi ke [Developers > API > WinkLinks](/ms/developers/apis/#winklinks-api).
