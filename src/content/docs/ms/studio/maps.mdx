## Buat peta

Peta boleh dibuat dengan dua cara:

- **Peta penanda tunggal**: Buat peta dari sesuatu yang anda temui semasa [mencari](/ms/studio/search).
- **Peta penanda berganda**: Buat peta dari [senarai terpilih](/ms/studio/curated-lists) atau [carian yang disimpan](/ms/studio/saved-searches).

Klik pada butang itu dan ia akan mengarahkan anda ke halaman borang peta kami di mana anda boleh mula menyesuaikan peta anda.

### Peta penanda berganda

Kedua-dua [senarai terpilih](/ms/studio/curated-lists#list-actions) dan [carian yang disimpan](/ms/studio/saved-searches#list-actions) mempunyai tindakan yang membolehkan anda membuat peta. Peta ini akan mengandungi penanda peta untuk semua inventori dalam senarai tersebut.

## Sesuaikan peta

Borang membolehkan anda menyesuaikan peta anda dengan cara berikut:

<Steps>
 1. Berikan nama supaya anda ingat apa yang peta itu tentang.
 2. Pilih sama ada anda mahu pengguna dibenarkan bergerak di sekitar peta.
 3. Tetapkan ketinggian peta dalam piksel.
 4. Pilih sama ada anda mahu pengguna dibenarkan untuk zum masuk dan keluar peta.
 5. Pilih gaya peta yang sesuai dengan laman web yang anda akan sematkan peta tersebut.
 6. Pilih warna penanda.
 7. Pilih muka kad awal yang anda mahu pengguna lihat terlebih dahulu. Secara lalai, ia adalah muka asli inventori tersebut.
 8. Anda boleh menambah penanda, bulatan, segi empat tepat dan poligon anda sendiri untuk menunjukkan lokasi lain pada peta yang mungkin menarik bagi pengguna anda.
 9. Klik butang `Simpan` untuk meneruskan.
</Steps>

Selepas menyimpan peta anda, anda akan diarahkan ke halaman peta anda dan peta anda kini sedia untuk dikongsi dengan dunia.

Di atas menunjukkan imej dengan semua tindakan yang tersedia untuk peta anda:

- **Kemas kini** Mengemas kini konfigurasi peta anda.
- **Tambah ke WinkLinks** Menambah peta ke akaun WinkLinks anda.
- **Sematkan** Menunjukkan cara untuk menyematkan peta ini sebagai [Peta](/ms/developers/web-components/#content-loader) ke dalam laman web anda.
- **Gunakan dengan WordPress** Menunjukkan cara menggunakan [plugin WordPress](/ms/developers/wordpress) kami untuk menyematkan peta ini ke dalam laman web anda.

Kami akan terangkan beberapa pilihan ini dengan lebih terperinci di bawah.

### Sematkan

```html {3, 6-9, 11, 13-16}
<html>
  <head>
    <link rel="stylesheet" href="https://elements.wink.travel/styles.css"></link>
  </head>
  <body>
    <wink-content-loader
       layout="MAP"
       id="64d7cbc8-61df-11ef-9722-42004e494300"
    ></wink-content-loader>

    <script src="https://elements.wink.travel/elements.js" type="module" defer></script>

    <wink-app-loader
      client-id="YOUR CLIENT ID GOES HERE"
      configuration-id="YOUR CUSTOMIZATION ID GOES HERE"
    ></wink-app-loader>
  </body>
</html>
```

Berikut adalah cara untuk menyematkan peta anda ke dalam laman web anda:

- Baris 3 menunjukkan cara menyematkan gaya Wink ke dalam laman web anda.
- Baris 6 hingga 9 menunjukkan cara menggunakan Komponen Web [wink-content-loader](/ms/developers/web-components/#content-loader) dan memberitahunya untuk mengambil banner iklan untuk kod anda.
- Baris 11 menunjukkan cara menyematkan Javascript kami ke dalam laman web anda.
- Baris 13 menunjukkan cara menyematkan Komponen Web [wink-app-loader](/ms/developers/web-components/#application-loader) dan memberitahunya untuk mengambil keutamaan konfigurasi peringkat halaman anda.

## API

Pembangun yang ingin mengurus peta boleh pergi ke [Pembangun > API > Inventori](/ms/developers/apis/#inventory-api).

## Bacaan lanjut

- Ketahui lebih lanjut tentang koleksi [Komponen Web](/ms/developers/web-components).
- Ketahui lebih lanjut tentang [plugin WordPress Wink](/ms/developers/wordpress).
