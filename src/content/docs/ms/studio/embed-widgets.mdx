Studio menyediakan beberapa widget yang boleh disematkan yang boleh anda tambah ke laman web anda sendiri untuk meningkatkan tempahan.

Untuk mengakses widget sematan, navigasi ke `Tools > Miscellaneous` dari bar sisi.

## Widget yang tersedia

| Widget | Penerangan |
| -- | -- |
| **Butang Akaun** | Butang berjenama yang pautkan ke halaman afiliasi anda |
| **Butang Jadual Perjalanan** | Butang yang membuka pemilih jadual perjalanan |
| **Borang Jadual Perjalanan** | Borang pemilih tarikh dan tetamu secara sebaris |
| **Butang Carian** | Butang yang membuka carian hartanah |
| **Butang Carian Inventori** | Butang untuk mencari inventori tertentu |

## Cara menyematkan

Setiap widget menyediakan petikan kod yang boleh anda salin dan tampal ke dalam HTML laman web anda. Petikan tersebut termasuk **Client ID** dan **Customization ID** anda supaya widget secara automatik dipautkan ke akaun afiliasi anda.

:::note
Jika anda menggunakan WordPress, kami juga menyediakan plugin WordPress untuk integrasi yang lebih mudah.
:::
