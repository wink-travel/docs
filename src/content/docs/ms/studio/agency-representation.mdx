Agensi adalah akaun Wink biasa yang boleh anda tetapkan pada akaun anda untuk tujuan perwakilan. Berikut adalah beberapa sebab mengapa anda ingin agensi mewakili anda:

- Pengurusan penuh / sebahagian akaun Wink anda oleh pihak ketiga.
- Bakat mempunyai perjanjian sedia ada dengan agensi mereka dan mahu Wink mengendalikan aspek kewangan susunan dana mereka yang masuk melalui Wink.
- Agensi bakat melakukan semua kerja dalam talian dan mengurus semua akaun di Wink.

## Tetapkan agensi

Berikut adalah cara anda menetapkan agensi pada akaun anda.

<Steps>
 1. Log masuk ke akaun pengguna anda di [Wink Studio](https://studio.wink.travel).
 2. Pilih akaun afiliasi yang anda ingin bekerjasama.
 3. Klik pada `Account > Managers` dari bar navigasi utama.
 4. Dalam bahagian `Agency`, taip nama agensi yang anda mahu wakilkan anda.
 5. Pilih yuran agensi dari dropdown.
 5. Pilihan tambah peraturan tamat tempoh.
 7. Klik butang `Set agency` untuk meneruskan.
 </Steps>

 :::note
Anda perlu menjadi pemilik akaun untuk menetapkan agensi.
 :::

 :::tip[Contoh]
Yuran agensi berasal dari komisen afiliasi.

Tempahan = $100, Komisen = 10%, Yuran agensi = 10% => 100 * 0.1 * 0.1 = $1
 :::

## Nyah tetapkan agensi

Berikut adalah cara anda nyah tetapkan agensi dari akaun anda.

<Steps>
 1. Log masuk ke akaun pengguna anda di [Wink Studio](https://studio.wink.travel).
 2. Pilih akaun afiliasi yang anda ingin bekerjasama.
 3. Klik pada `Account > Managers` dari bar navigasi utama.
 4. Dalam bahagian `Agency`, klik ikon <Badge text="x" variant="danger" size="small" /> di sebelah kanan dropdown agensi.
 5. Klik butang `Set agency` untuk meneruskan.
 </Steps>


## API

Pembangun yang ingin mengurus agensi melalui API boleh ke [Developers > API > Affiliate](/ms/developers/apis/#affiliate-api).
