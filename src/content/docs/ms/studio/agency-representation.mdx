Agensi ialah akaun Wink biasa yang anda boleh berikan kepada akaun anda untuk tujuan perwakilan. Berikut ialah beberapa sebab mengapa anda mahukan agensi yang mewakili anda:

* Pengurusan penuh / separa akaun Wink anda oleh pihak ke-3.
* Talent mempunyai perjanjian sedia ada dengan agensi mereka dan mahu Wink mengendalikan aspek kewangan susunan dana mereka yang masuk melalui Wink.
* Agensi bakat melakukan semua kerja dalam talian dan mengurus semua akaun di Wink.

## Tugaskan agensi

Begini cara anda menetapkan agensi pada akaun anda.

<Steps>
  1. Log masuk ke akaun pengguna anda pada [Studio Wink](https://studio.wink.travel).
  2. Pilih akaun ahli gabungan yang anda ingin bekerjasama.
  3. klik pada `Account > Managers` daripada bar navigasi utama.
  4. Dalam `Agency` bahagian, taipkan nama agensi yang anda mahu wakili anda.
  5. Pilih yuran agensi daripada menu lungsur.
  6. Tambah peraturan tamat tempoh secara pilihan.
  7. Klik pada `Set agency` butang untuk meneruskan.
</Steps>


:::nota
Anda perlu menjadi pemilik akaun untuk menetapkan agensi.
:::

:::tip\[Contoh]
Yuran aency berasal daripada komisen ahli gabungan.

Tempahan = $100, Komisen = 10%, Yuran agensi = 10% => 100 \* 0.1 \* 0.1 = $1
:::

## Nyahtugaskan agensi

Begini cara anda membatalkan penugasan agensi daripada akaun anda.

<Steps>
  1. Log masuk ke akaun pengguna anda pada [Studio Wink](https://studio.wink.travel).
  2. Pilih akaun ahli gabungan yang anda ingin bekerjasama.
  3. klik pada `Account > Managers` daripada bar navigasi utama.
  4. Dalam `Agency` bahagian, klik pada <Badge text="x" variant="danger" size="small" /> ikon di sebelah kanan menu lungsur agensi.
  5. Klik pada `Set agency` butang untuk meneruskan.
</Steps>


## API

Pembangun yang ingin mengurus agensi melalui API boleh pergi ke [Pembangun > API > Ahli Gabungan](/developers/apis/#affiliate-api).
