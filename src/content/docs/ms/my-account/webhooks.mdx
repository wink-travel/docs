Webhooks adalah cara untuk anda menerima, dan bertindak balas terhadap, acara masa nyata dari Wink untuk akaun anda.

<Steps>
 1. Pilih akaun yang anda ingin gunakan dari [Accounts](https://my.wink.travel).
 2. Klik pada `Actions` di bahagian bawah kad akaun dan pilih `Manage account`.
 3. Klik tab `Webhooks` pada halaman berikutnya.
 4. Bahagian ini akan menyenaraikan semua webhook anda.
 5. Pilih untuk mengedit yang sedia ada atau buat yang baru.
 </Steps>

## Contoh

<Steps>
1. Tempahan berlaku di Wink.
1. Acara Wink `booking.created` dihantar.
2. Sistem anda sedang mendengar acara ini secara aktif.
3. Sistem anda menjejak set analitik sendiri dan menggunakan acara ini untuk mengemas kini pangkalan data siri masa.
</Steps>

## Cipta webhook

:::caution
Anda perlu mencipta akaun affiliate di Wink atau akaun di TripPay sebelum anda mencipta webhook kerana anda akan mengikat webhook anda kepada akaun tertentu.
:::

Berikut adalah langkah-langkah untuk mencipta webhook:

<Steps>
1. Dari halaman webhooks, klik butang `Create new webhook`.
2. ***Nama*** Beri nama kepada webhook anda. *contoh: Wink Stalker*
3. ***Aktifkan*** Biarkan seperti sedia ada untuk diaktifkan.
4. **Nama entiti** Pautkan webhook anda dengan salah satu akaun sedia ada anda. *contoh: Cool Account*
5. **Acara** Dengar acara yang anda minati. *contoh: `booking.created`*
6. **URL Webhook** Masukkan URL di mana anda akan mendengar acara dari kami. *contoh: `https://my.system.com/webhook/wink`*
7. Klik butang `Save` untuk meneruskan.
</Steps>

Anda akan dialihkan kembali ke senarai webhook anda dan webhook baru anda sudah sedia digunakan.

## Kemas kini webhook

Anda perlu mengemas kini aplikasi anda jika anda ingin:

- Nyahaktifkan webhook anda.
- Mendengar lebih / kurang acara.
- Tukar URL webhook yang didengar.

Untuk mengemas kini webhook sedia ada, lakukan perkara berikut:

<Steps>
1. Klik pautan `Actions` untuk webhook yang anda ingin kemas kini.
2. Klik butang `Update` di bawah Actions.
3. Buat perubahan pada webhook.
4. Klik butang `Save` untuk meneruskan.
</Steps>

Anda akan dialihkan kembali ke senarai webhook anda.

## Buang webhook

Jika anda tidak lagi memerlukan webhook, anda boleh terus membuangnya.

Untuk membuang webhook, lakukan perkara berikut:

<Steps>
1. Klik pautan `Actions` untuk webhook yang anda ingin kemas kini.
2. Klik butang `Remove` di bawah Actions.
3. Klik butang `OK` untuk mengesahkan pemadaman.
</Steps>

Anda akan dialihkan kembali ke senarai webhook anda.
