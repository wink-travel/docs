Seorang pengguna Wink boleh memilih untuk membenarkan anda mengurus salah satu akaun hartanah, gabungan atau ejen pelancongan mereka. Apabila itu berlaku, anda akan menerima e-mel dengan permintaan jemputan.

import { Steps } from '@astrojs/starlight/components';

Untuk melihat jemputan Wink anda yang terkini, ikut langkah-langkah berikut:

<Steps>
 1. Log masuk ke [my.wink.travel](https://my.wink.travel).
 2. Klik pada `Invites` dari bar sisi kiri.
 3. Anda akan melihat senarai semua jemputan terkini.
 4. Bahagian ini akan menyenaraikan semua permohonan anda.
 5. Pilih untuk `Accept` atau `Reject` jemputan tersebut.
 6. Jika anda menerima jemputan, akaun tersebut akan dipaparkan dalam bahagian `Accounts`.
</Steps>
