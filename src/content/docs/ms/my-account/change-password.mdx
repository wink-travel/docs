Jika anda ingin menukar kata laluan anda, atau anda terlupa kata laluan anda, ikut langkah-langkah berikut:

## Kemas kini kata laluan

<Steps>
1. Pergi ke [my.wink.travel](https://my.wink.travel).
1. Log masuk dengan mengklik `Sign-in / Register` di penjuru kanan atas.
2. Klik ikon profil anda, di penjuru kanan atas dan menu akan muncul.
3. Klik pautan `Change password`.
4. Ikuti arahan untuk menukar kata laluan anda.
5. Klik `Change password` untuk meneruskan.
</Steps>

Kali seterusnya anda log masuk, anda perlu menggunakan kata laluan baru anda.

## Lupa kata laluan

Kadang-kadang perlu untuk menetapkan semula kata laluan pengguna.

Berikut adalah langkah-langkah untuk menetapkan semula kata laluan pengguna anda:

<Steps>
1. Klik pada pautan `Forgot password` dari halaman log masuk.
2. Taipkan nama pengguna / e-mel untuk akaun yang anda ingin tetapkan semula.
3. Klik butang `Forgot password`. *Anda akan dialihkan ke borang tetapan semula kata laluan.*
4. Semak e-mel anda.
</Steps>

<Steps>
1. Tampal kod yang anda terima dalam e-mel ke dalam borang baru bersama kata laluan baru anda.
2. Sahkan kata laluan baru anda.
3. Klik butang `Reset password`. *Anda akan dialihkan kembali ke halaman log masuk.*
</Steps>
