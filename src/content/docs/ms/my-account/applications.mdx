Aplikasi memberi anda akses kepada semua ciri Wink sebagai pembangun. Anda boleh mengakses semua data yang sama dan malah mengembangkan ciri Wink sendiri.
Anda boleh memilih untuk menggunakan pelayan OAuth2 Wink untuk mengesahkan laman perjalanan anda sama seperti anda menggunakan Google atau Facebook.
Dengan cara itu anda boleh mengembangkan ciri Wink ke dalam laman web atau aplikasi anda sendiri.

:::note
Aplikasi pertama anda sudah dicipta apabila anda [mencipta akaun baru](/ms/my-account/create-account). Kami menggunakan aplikasi itu untuk beberapa senario tempahan.

Anda tidak boleh menggunakan semula aplikasi ini. Sila cipta aplikasi baru mengikut keperluan anda.
:::

:::note
Aplikasi yang dijana di Wink boleh digunakan untuk semua titik akhir API.
:::

<Steps>
 1. Pilih akaun yang anda ingin gunakan dari [Accounts](https://my.wink.travel).
 2. Klik pada `Actions` di bahagian bawah kad akaun dan pilih `Manage account`.
 3. Klik tab `Applications` pada halaman berikutnya.
 4. Bahagian ini akan menyenaraikan semua aplikasi anda.
 5. Pilih untuk mengedit aplikasi sedia ada atau cipta yang baru.
 </Steps>

## Cipta aplikasi

Oleh kerana anda tidak boleh mengakses `secret key` aplikasi yang kami cipta untuk anda, anda perlu mencipta aplikasi anda sendiri.

:::caution
Anda perlu mencipta akaun affiliate di Wink atau akaun di TripPay sebelum mencipta aplikasi kerana anda akan mengikat aplikasi anda kepada akaun tertentu.
:::

Berikut adalah langkah-langkah untuk mencipta aplikasi:

<Steps>
1. Dari halaman aplikasi, klik butang `Create new application`.
2. ***Nama*** Beri nama kepada aplikasi anda. *contoh Cool App*
3. **Nama entiti** Pautkan aplikasi anda dengan salah satu akaun sedia ada anda. *contoh Cool Account*
4. **Redirect URIs** Sekurang-kurangnya satu redirect URI diperlukan. Ia digunakan untuk mengesahkan domain redirect selepas pengesahan berjaya. *contoh https://www.cool-site.com*
5. Klik butang `Save` untuk meneruskan.
</Steps>

Anda akan dialihkan kembali ke senarai aplikasi anda. Anda akan melihat aplikasi baru anda dalam senarai dan hanya sekali ini anda boleh mengakses kunci rahsia aplikasi. Pastikan anda menyalinnya dan menyimpannya di tempat yang selamat.

## Kemas kini aplikasi

Anda perlu mengemas kini aplikasi anda jika salah satu redirect URI anda berubah atau anda ingin menambah atau membuang URI.

Untuk mengemas kini aplikasi sedia ada, lakukan perkara berikut:

<Steps>
1. Klik pautan `Actions` untuk aplikasi yang anda ingin kemas kini.
2. Klik butang `Update` di bawah Actions.
3. Buat perubahan pada aplikasi.
4. Klik butang `Save` untuk meneruskan.
</Steps>

Anda akan dialihkan kembali ke senarai aplikasi anda.

## Buang aplikasi

:::caution
Sila jangan padam aplikasi yang kami cipta secara automatik untuk anda supaya semua ciri berfungsi untuk akaun anda.
:::

Jika anda tidak lagi memerlukan aplikasi, anda boleh terus membuangnya.

Untuk membuang aplikasi, lakukan perkara berikut:

<Steps>
1. Klik pautan `Actions` untuk aplikasi yang anda ingin kemas kini.
2. Klik butang `Remove` di bawah Actions.
3. Klik butang `OK` untuk mengesahkan pemadaman.
</Steps>

Anda akan dialihkan kembali ke senarai aplikasi anda.
