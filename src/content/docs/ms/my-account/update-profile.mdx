Jika anda ingin mengemas kini profil pengguna untuk akaun pengguna Wink anda, ikut langkah-langkah berikut:

<Steps>
1. Pergi ke [my.wink.travel](https://my.wink.travel).
2. Log masuk dengan mengklik `Sign-in / Register` di penjuru kanan atas.
3. Setelah anda log masuk, klik ikon profil anda di penjuru kanan atas, dan menu akan muncul.
4. Klik pautan `Profile`.
5. Ikuti arahan untuk mengemas kini `e-mel`, `nama pertama` dan `nama terakhir`.
6. Anda juga boleh memuat naik `gambar profil` baru anda.
7. Klik `Save`.
</Steps>

<Steps>
1. Anda mesti log masuk ke salah satu laman web kami.
2. Klik ikon profil anda di penjuru kanan atas dan menu akan muncul.
3. Klik pautan `Profile`.
4. Ikuti arahan untuk mengemas kini `e-mel`, `nama pertama` dan `nama terakhir`.
5. Anda juga boleh memuat naik `gambar profil` baru anda.
6. Klik `Update`.
</Steps>

:::note
Gambar profil baru anda akan dipaparkan pada kali seterusnya anda log keluar sepenuhnya dari Wink.
:::
