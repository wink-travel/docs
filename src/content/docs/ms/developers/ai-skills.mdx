Kemahiran AI adalah pakej arahan mudah alih yang mengajar ejen pengekodan cara bekerja dengan Wink. Daripada bergantung pada data latihan atau dokumentasi yang lapuk, setiap kemahiran mengarahkan ejen untuk menggunakan [pelayan MCP Wink](/ms/developers/mcp) sebagai sumber kebenaran langsung untuk kontrak API dan dokumen platform.

Kemahiran berfungsi dengan mana-mana klien ejen yang menyokongnya — Claude Code, Codex, Cursor, dan lain-lain.

## Kemahiran Tersedia

### wink-api-builder

Laksanakan, semak, atau debug integrasi API Wink. Kemahiran ini membimbing ejen melalui aliran kerja berstruktur:

1. Cari operasi API yang betul melalui `api_search`
2. Buka kontrak permintaan/respons langsung dari pelayan MCP
3. Tarik tetapan pengesahan dari dokumen
4. Tulis kod berdasarkan kontrak yang disahkan
5. Tandakan sebarang ketidakpadanan antara kod sedia ada dan API semasa

Berguna untuk:
- Membina kaedah klien baru untuk mana-mana API Wink
- Debug ketidakpadanan kontrak (contoh: permintaan yang mengembalikan HTTP 400)
- Menyemak integrasi sedia ada berbanding spesifikasi langsung

## Pasang

Salin folder kemahiran ke dalam direktori kemahiran ejen anda.

:::tip
Pelayan MCP Wink mesti disambungkan dalam persekitaran yang sama. Lihat [Pelayan MCP](/ms/developers/mcp) untuk arahan pemasangan.
:::

**Claude Code**

```bash
# repo-lokal
cp -r skills/wink-api-builder .claude/skills/

# atau peringkat pengguna
cp -r skills/wink-api-builder ~/.claude/skills/
```

Panggil dengan:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repo-lokal
cp -r skills/wink-api-builder .agents/skills/

# atau peringkat pengguna
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Panggil dengan:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Ejen lain**

Letakkan folder kemahiran di mana-mana sahaja ejen anda menyelesaikan kemahiran. Titik masuk adalah `SKILL.md`.
