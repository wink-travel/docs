Percepatkan masa ke pasaran dengan menggunakan salah satu Kit Pembangunan Perisian bahasa yang disokong secara rasmi oleh kami.
Jika kami tidak menyokong bahasa kegemaran anda buat masa ini, pergi ke API yang anda ingin gunakan dan muat turun spesifikasi OpenAPI yang kami sediakan untuk setiap API di bahagian atas halaman. Anda boleh menjana SDK anda sendiri sama ada dengan memfork salah satu repositori SDK awam kami atau dengan menggunakan salah satu penjana OpenAPI.

## Java

Perpustakaan ini disusun untuk Java 24. Ia dikonfigurasikan untuk menggunakan WebClient reaktif dan dioptimumkan untuk digunakan dengan Spring Framework. Walau bagaimanapun, Spring bukanlah keperluan tetapi ia memberikan anda sokongan OAuth2 secara terus.
