Percepatkan masa ke pasaran dengan menggunakan salah satu Kit Pembangunan Perisian bahasa kami yang disokong secara rasmi.
Jika pada masa ini kami tidak menyokong bahasa kegemaran anda, pergi ke API yang anda mahu gunakan dan muat turun spesifikasi OpenAPI yang kami sediakan untuk setiap API di bahagian atas halaman. Anda boleh menjana SDK anda sendiri dengan sama ada memotong salah satu repositori SDK awam kami atau dengan menggunakan salah satu penjana OpenAPI.

## Jawa

Pustaka ini telah disusun untuk Java 24. Ia dikonfigurasikan untuk menggunakan WebClient reaktif dan dioptimumkan untuk digunakan dengan Rangka Kerja Spring. Walau bagaimanapun, Spring bukanlah satu keperluan tetapi ia memberi anda sokongan OAuth2 di luar kotak.
