## Payment API

Titik akhir Pembayaran adalah untuk pembangun yang ingin membeli inventori perjalanan. Ini boleh dilakukan melalui API sebagai Ejen Pelancongan berdaftar atau menggunakan API kami bersama widget pembayaran yang mematuhi PCI untuk semua entiti lain.
Payment API mendedahkan titik akhir untuk afiliasi dan hotel menjejak tempahan, analitik dan ketersediaan dana. Mereka juga boleh memilih untuk mengeluarkan dana yang tersedia ke akaun bank mereka.
