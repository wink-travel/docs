## API Pembayaran

Titik akhir pembayaran adalah untuk pembangun yang ingin membeli inventori perjalanan. Ini boleh dilakukan melalui API sebagai Ejen Pelancongan berdaftar atau menggunakan API kami bersama-sama dengan widget pembayaran patuh PCI kami untuk semua entiti lain.
API Pembayaran mendedahkan titik akhir untuk ahli gabungan dan hotel untuk menjejaki tempahan, analitik dan ketersediaan dana. Mereka juga boleh memilih untuk mengeluarkan dana yang tersedia ke akaun bank mereka.
