Aplikasi memberi anda akses penuh kepada ciri-ciri di Wink dan TripPay serta membolehkan pengguna anda mengesahkan dan log masuk ke laman web anda menggunakan pelayan kebenaran OAuth2 kami; sama seperti Facebook dan Google.
Anda boleh membina OTA anda sendiri dengan Wink. Ia bermula dengan sebuah aplikasi.

*Secara lalai, anda akan mendapat aplikasi apabila anda mendaftar akaun afiliasi Wink baru atau akaun TripPay. Kami menggunakan `client ID` untuk beberapa senario.*

:::note
Aplikasi yang dijana di Wink atau TripPay boleh digunakan untuk semua titik akhir API.
:::

*Untuk contoh ini, kami akan mengesahkan melalui [Wink Studio](https://studio.wink.travel).*

<Steps>
1. Setelah disahkan, klik ikon profil anda di penjuru kanan atas, dan menu akan muncul.
2. Klik pautan `Applications`.
3. Anda akan dialihkan ke halaman aplikasi anda.
</Steps>

## Buat aplikasi

Oleh kerana anda tidak boleh mengakses `secret key` aplikasi yang kami buat untuk anda, anda perlu membuat aplikasi anda sendiri.

:::caution
Anda perlu membuat akaun afiliasi di Wink atau akaun di TripPay sebelum membuat aplikasi kerana anda akan mengikat aplikasi anda kepada akaun tertentu.
:::

Berikut adalah langkah-langkah untuk membuat aplikasi:

<Steps>
1. Dari halaman aplikasi, klik butang `Create new application`.
2. ***Nama*** Beri nama kepada aplikasi anda. *contoh: Cool App*
3. **Nama entiti** Pautkan aplikasi anda dengan salah satu akaun sedia ada anda. *contoh: Cool Account*
4. **Redirect URIs** Sekurang-kurangnya satu redirect URI diperlukan. Ia digunakan untuk mengesahkan domain redirect selepas pengesahan berjaya. *contoh: https://www.cool-site.com*
5. Klik butang `Save` untuk meneruskan.
</Steps>

Anda akan dialihkan kembali ke senarai aplikasi anda. Anda akan melihat aplikasi baru anda dalam senarai dan hanya sekali ini anda boleh mengakses kunci rahsia aplikasi. Pastikan anda menyalinnya dan menyimpannya di tempat yang selamat.

## Kemas kini aplikasi

Anda perlu mengemas kini aplikasi anda jika salah satu redirect URI anda berubah atau anda ingin menambah atau membuang URI.

Untuk mengemas kini aplikasi sedia ada, lakukan perkara berikut:

<Steps>
1. Klik pautan `Actions` untuk aplikasi yang ingin anda kemas kini.
2. Klik butang `Update` di bawah Actions.
3. Buat perubahan pada aplikasi.
4. Klik butang `Save` untuk meneruskan.
</Steps>

Anda akan dialihkan kembali ke senarai aplikasi anda.

## Buang aplikasi

:::caution
Sila jangan padam aplikasi yang kami buat secara automatik untuk anda supaya semua ciri berfungsi untuk akaun anda.
:::

Jika anda tidak lagi memerlukan aplikasi, anda boleh terus membuangnya.

Untuk membuang aplikasi, lakukan perkara berikut:

<Steps>
1. Klik pautan `Actions` untuk aplikasi yang ingin anda kemas kini.
2. Klik butang `Remove` di bawah Actions.
3. Klik butang `OK` untuk mengesahkan pemadaman.
</Steps>

Anda akan dialihkan kembali ke senarai aplikasi anda.
