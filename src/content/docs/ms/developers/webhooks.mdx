Webhooks ialah cara untuk anda menerima, dan bertindak balas kepada, acara masa nyata daripada Wink dan TripPay berkaitan akaun anda.

## Contoh

<Steps>
  1. Tempahan berlaku pada Wink.
  2. Acara Wink `booking.created` dihantar.
  3. Sistem anda sedang aktif mendengar acara ini.
  4. Sistem anda menjejaki set analitik anda sendiri dan menggunakan acara ini untuk mengemas kini pangkalan data siri masanya.
</Steps>

:::nota
Webhook Wink berbeza daripada webhook TripPay.

* Jika anda ingin mendengar acara Wink, anda boleh membuat webhook daripada [Ekstranet](https://extranet.wink.travel) atau [Studio](https://studio.wink.travel).
* Jika anda ingin mendengar acara TripPay, anda boleh membuat webhook daripada [Bayaran](https://pay.wink.travel).
  :::

*Untuk contoh ini, kami akan mengesahkan melalui [Studio](https://studio.wink.travel).*

<Steps>
  1. Setelah disahkan, klik ikon profil anda, di penjuru kanan sebelah atas, dan menu akan muncul.
  2. Klik pada `Webhooks` pautan.
  3. Anda akan dialihkan ke halaman webhooks anda.
</Steps>


## Buat webhook

:::berhati-hati
Anda perlu membuat akaun ahli gabungan di Wink atau akaun di TripPay sebelum anda membuat webhook kerana anda akan mengikat webhook anda pada akaun tertentu.
:::

Berikut ialah langkah-langkah untuk membuat webhook:

<Steps>
  1. Dari halaman webhooks, klik pada `Create new webhook` butang.
  2. ***Nama*** Beri nama webhook anda. *cth. Stalker kejap*
  3. ***Dayakan*** Biarkan apa adanya untuk didayakan.
  4. **Nama entiti** Pautkan webhook anda dengan salah satu akaun sedia ada anda. *cth. Akaun Cool*
  5. **Peristiwa** Dengar peristiwa yang menarik minat anda. *cth. `booking.created`*
  6. **URL Webhook** Masukkan URL tempat anda akan mendengar acara daripada kami. *cth. `https://my.system.com/webhook/wink`*
  7. Klik pada `Save` butang untuk meneruskan.
</Steps>


Anda dialihkan semula ke senarai webhooks anda dan webhook baharu anda sedia untuk digunakan.

## Kemas kini webhook

Anda perlu mengemas kini apl anda jika anda mahu:

* Lumpuhkan webhook anda.
* Dengar lebih banyak / kurang acara.
* Tukar URL webhook pendengaran anda.

Untuk mengemas kini webhook sedia ada, lakukan perkara berikut:

<Steps>
  1. Klik pada `Actions` pautan untuk webhook yang ingin anda kemas kini.
  2. Klik pada `Update` butang di bawah Tindakan.
  3. Buat perubahan anda pada webhook.
  4. Klik pada `Save` butang untuk meneruskan.
</Steps>


Anda dialihkan semula ke senarai webhooks anda.

## Alih keluar webhook

Jika anda tiada lagi penggunaan webhook, anda boleh meneruskan dan mengalih keluarnya.

Untuk mengalih keluar webhook, lakukan perkara berikut:

<Steps>
  1. Klik pada `Actions` pautan untuk webhook yang ingin anda kemas kini.
  2. Klik pada `Remove` butang di bawah Tindakan.
  3. klik `OK` butang untuk mengesahkan pemadaman.
</Steps>


Anda dialihkan semula ke senarai webhooks anda.
