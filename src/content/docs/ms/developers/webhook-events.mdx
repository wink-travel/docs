Halaman ini menyenaraikan setiap acara webhook yang anda boleh langgani di Wink. Untuk arahan penyediaan, lihat [Panduan Integrasi Webhook](/ms/guides/webhook-integration) atau [halaman pengurusan Webhooks](/ms/my-account/webhooks).

## Cara melanggan

1. Navigasi ke **Applications > Webhooks** dalam portal anda.
2. Cipta webhook dan pilih acara yang anda ingin terima.
3. Wink akan menghantar HTTP POST ke titik akhir anda setiap kali acara berlaku.

Untuk langkah penyediaan penuh dan amalan terbaik, lihat [Integrasi Webhook](/ms/guides/webhook-integration).

## Acara akaun

Acara berkaitan kitar hayat dan tetapan akaun TripPay anda.

| Acara | Kekunci | Penerangan |
|---|---|---|
| Akaun dicipta | `account.create` | Akaun baru telah dicipta. |
| Akaun dalam semakan | `account.under_review` | Akaun sedang menunggu semakan. |
| Akaun dikemas kini | `account.updated` | Butiran akaun telah berubah. |
| Status akaun dikemas kini | `account.status.updated` | Status akaun telah berubah (contoh: diluluskan, digantung). |
| Alamat akaun dikemas kini | `account.address.updated` | Alamat akaun telah diubah. |
| Butiran akaun dikemas kini | `account.details.updated` | Medan butiran akaun telah berubah. |
| Maklumat akaun dikemas kini | `account.information.update` | Maklumat umum akaun telah dikemas kini. |
| Pengurus akaun dikemas kini | `account.managers.update` | Senarai pengurus akaun telah berubah. |
| Gambar profil dikemas kini | `account.profile_picture.updated` | Gambar profil akaun telah diubah. |
| Ejen dikemas kini | `account.agent.updated` | Maklumat ejen pada akaun telah diubah. |
| Nama URL dikemas kini | `account.url-name.update` | URL vanity akaun telah berubah. |
| ID unik dikemas kini | `account.unique-id.update` | Pengenal unik akaun telah diubah. |

### Tugas akaun

| Acara | Kekunci | Penerangan |
|---|---|---|
| Tugas dicipta | `account.task.create` | Tugas baru telah ditambah ke akaun. |
| Tugas dikemas kini | `account.task.update` | Tugas akaun telah diubah. |
| Tugas dipadam | `account.task.remove` | Tugas akaun telah dipadam. |
| Tugas diselesaikan | `account.task.complete` | Tugas akaun telah diselesaikan. |
| Tugas pengesahan ID cukai selesai | `account.task.tax_id.complete` | Tugas pengesahan ID cukai telah selesai. |

### Pengakuisisi akaun

| Acara | Kekunci | Penerangan |
|---|---|---|
| Pengakuisisi dicipta | `account.acquirer.created` | Pengakuisisi baru telah ditambah ke akaun. |
| Pengakuisisi dikemas kini | `account.acquirer.updated` | Pengakuisisi akaun telah diubah. |
| Pengakuisisi dipadam | `account.acquirer.removed` | Pengakuisisi telah dipadam dari akaun. |

## Acara kontrak tempahan

Acara yang menjejak kitar hayat kontrak tempahan melalui TripPay.

| Acara | Kekunci | Penerangan |
|---|---|---|
| Permintaan inisialisasi | `booking.contract.initialize.request` | Permintaan inisialisasi kontrak tempahan telah dibuat. |
| Inisialisasi selesai | `booking.contract.initialize.complete` | Inisialisasi kontrak tempahan telah selesai. |
| Kontrak dicipta | `booking.contract.created` | Kontrak tempahan baru telah dicipta. |
| Kontrak diproses | `booking.contract.processing` | Pembayaran kontrak tempahan sedang diproses. |
| Kontrak gagal | `booking.contract.failed` | Pembayaran kontrak tempahan gagal. |
| Kontrak dibatalkan | `booking.contract.cancelled` | Kontrak tempahan telah dibatalkan. |
| Bayaran balik separa | `booking.contract.refund.partial` | Bayaran balik separa telah dikeluarkan untuk kontrak. |
| Bayaran balik penuh | `booking.contract.refund.full` | Bayaran balik penuh telah dikeluarkan untuk kontrak. |
| Permintaan bayaran balik | `booking.contract.refund.request` | Permintaan bayaran balik telah dihantar. |
| Ralat bayaran balik | `booking.contract.refund.error` | Ralat berlaku semasa memproses bayaran balik. |
| Pembayaran dicipta | `booking.contract.payout.create` | Pembayaran telah dicipta untuk kontrak. |
| Pembayaran selesai | `booking.contract.payout.complete` | Pembayaran telah berjaya diselesaikan. |

## Acara entiti pengurusan

Acara untuk entiti pengurusan (harta, afiliasi, syarikat) dan pentadbirannya.

### Langganan

| Acara | Kekunci | Penerangan |
|---|---|---|
| Langganan disediakan | `managing_entity.subscription.provisioned` | Langganan baru telah disediakan. |
| Langganan diubah | `managing_entity.subscription.changed` | Pelan langganan telah diubah. |
| Invois langganan diubah | `managing_entity.subscription.invoice.changed` | Invois langganan telah dikemas kini. |

### Status dan pentadbiran

| Acara | Kekunci | Penerangan |
|---|---|---|
| Status dikemas kini | `managing_entity.status.updated` | Status entiti telah berubah. |
| Entiti dipadam | `managing_entity.deleted` | Entiti pengurusan telah dipadam. |
| Ejen dikemas kini | `managing_entity.agent.updated` | Ejen entiti telah diubah. |
| Notifikasi e-mel ejen | `managing_entity.agent.email.notification` | Notifikasi e-mel telah dihantar kepada ejen. |
| Notifikasi e-mel pengurus | `managing_entity.manager.email.notification` | Notifikasi e-mel telah dihantar kepada pengurus. |

### Kitar hayat pengurus

| Acara | Kekunci | Penerangan |
|---|---|---|
| Pengurus ditambah | `managing_entity.manager.added` | Pengurus baru telah ditambah. |
| Pengurus dijemput | `managing_entity.manager.invited` | Jemputan pengurus telah dihantar. |
| Pengurus dipadam | `managing_entity.manager.removed` | Pengurus telah dipadam. |
| Pengurus diterima | `managing_entity.manager.accepted` | Jemputan pengurus telah diterima. |
| Pengurus ditolak | `managing_entity.manager.rejected` | Jemputan pengurus telah ditolak. |
| Pemilik ditetapkan | `managing_entity.manager.owner.set` | Pemilikan entiti telah dipindahkan. |

### Maklumat entiti

| Acara | Kekunci | Penerangan |
|---|---|---|
| Maklumat dikemas kini | `managing_entity.information.updated` | Maklumat entiti telah dikemas kini. |
| Alamat dikemas kini | `managing_entity.city.updated` | Alamat/bandar entiti telah diubah. |
| Wakil akaun dikemas kini | `managing_entity.account_representative.updated` | Wakil akaun telah diubah. |
| Gambar profil dikemas kini | `managing_entity.profile_picture.updated` | Gambar profil entiti telah dikemas kini. |
| Logo dikemas kini | `managing_entity.logo.updated` | Logo entiti telah dikemas kini. |
| Nama URL dikemas kini | `managing_entity.url_name.updated` | URL vanity entiti telah berubah. |
| ID unik dikemas kini | `managing_entity.unique_id.updated` | Pengenal unik entiti telah berubah. |
| Nama dikemas kini | `managing_entity.name.updated` | Nama entiti telah diubah. |
