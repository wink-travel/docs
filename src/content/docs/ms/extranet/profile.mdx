Profil hartanah adalah langkah wajib dalam wizard onboarding kami dan memberi peluang untuk kami mengenali hartanah anda dengan lebih baik menggunakan kriteria, seperti:

- Bilangan bintang.
- Jenis hotel.
- Jenis lokasi.
- ...dan banyak lagi

Untuk mengurus profil hartanah anda, klik pada `Property > Profile` dalam tab navigasi utama.

Kemas kini borang dan klik butang `Save` untuk meneruskan.
