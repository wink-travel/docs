Di mana-mana sahaja anda melihat inventori Wink [kad](/studio/cards), anda akan melihat hati di penjuru kanan sebelah atas kad yang membolehkan anda menambahkan inventori itu pada senarai baldi anda.

Ikuti langkah ini untuk mengurus senarai baldi anda:

<Steps>
  1. Klik pada ♡ `heart icon` untuk inventori yang ingin anda tambahkan.
  2. Jika anda log masuk, ikon hati akan bertukar merah.
  3. Jika anda tidak log masuk, anda akan diubah hala untuk berbuat demikian.
</Steps>


:::petua
Klik ♥️ `heart icon` kali kedua untuk mengalih keluarnya daripada senarai anda.
:::

Anda boleh mengurus semua item senarai baldi anda dengan mengklik ikon profil anda di penjuru kanan sebelah atas dan kemudian klik butang `Bucket list` butang.
