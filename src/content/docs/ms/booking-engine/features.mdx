Enjin tempahan hadir dalam dua jenis untuk menyokong keperluan berbeza bagi affiliate, syarikat pengurusan hartanah dan hotel.

## Agensi pelancongan dalam talian

Bagi affiliate, adalah lebih masuk akal untuk menggunakan versi agensi pelancongan dalam talian. Sebabnya ialah laman web disediakan untuk memenuhi keperluan pelancong yang ingin melihat pelbagai hartanah.

Anda boleh dapati halaman utama untuk aplikasi ini di bawah.

<LinkButton href="https://ota.wink.travel/home" variant="secondary" icon="external" target="_blank">
  Wink OTA
</LinkButton>

## Enjin tempahan internet

Bagi hotel atau syarikat pengurusan hartanah, adalah lebih masuk akal untuk menggunakan versi enjin tempahan internet. Jenis pengguna ini berminat untuk menonjolkan satu hartanah atau beberapa hartanah di bawah pengurusan.

Anda boleh dapati halaman utama untuk aplikasi ini di bawah.

## Konfigurasi

Kebiasaannya, tiada keperluan untuk berkongsi mana-mana aplikasi ini secara langsung. Kami telah mencipta pelbagai cara untuk anda berkongsi inventori pelancongan, yang boleh anda temui di [Wink Studio](/ms/studio/what-is-studio) atau melalui [WinkLinks](/ms/link-manager/wink-links).

Walau bagaimanapun, jika anda ingin pautkan secara manual ke salah satu laman web ini, pastikan anda tambah ID klien dan ID penyesuaian anda ke URL sebagai parameter.

Anda boleh dapati ID klien anda dengan:

<Steps>
1. Log masuk ke [Wink Extranet](/ms/extranet/what-is-extranet) atau [Wink Studio](/ms/studio/what-is-studio).
2. Klik pada ikon profil anda di penjuru kanan atas dan klik pada [Applications](/ms/developers/applications).
3. Gunakan aplikasi yang berkaitan dengan akaun Wink yang anda ingin gunakan.
4. Salin Client ID dari aplikasi tersebut dan tampalkannya pada URL sebagai parameter permintaan. Contoh: ?client-id=MY_CLIENT_ID
</Steps>

### Extranet

Anda boleh dapati ID penyesuaian anda sebagai hartanah dengan:

<Steps>
1. Log masuk ke [Wink Extranet](/ms/extranet/what-is-extranet).
2. Pilih hartanah yang anda ingin uruskan.
3. Dari bar navigasi utama, klik `Account > Branding`
4. Salin Customization ID dan tampalkannya pada URL sebagai parameter permintaan. Contoh: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

### Studio

Anda boleh dapati ID penyesuaian anda sebagai affiliate dengan:

<Steps>
1. Log masuk ke [Wink Studio](/ms/studio/what-is-studio).
2. Pilih akaun yang anda ingin uruskan.
3. Dari bar navigasi utama, klik `Tools > Customization`
4. Salin Customization ID dari penyesuaian yang anda ingin gunakan dan tampalkannya pada URL sebagai parameter permintaan. Contoh: ?configuration-id=MY_CUSTOMIZATION_ID
</Steps>

## OTA

Apa yang anda akan dapat adalah URL yang kelihatan seperti ini:

https://ota.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Jika anda ingin terus ke halaman hotel, URL akan kelihatan seperti ini:

https://ota.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

:::note
Jika anda tidak sertakan ID klien dan / atau ID konfigurasi, Wink OTA akan menggunakan konfigurasi lalai.
:::

## IBE

Jika anda adalah syarikat pengurusan hartanah, anda ingin pautkan ke halaman utama Wink IBE. Halaman utama akan menunjukkan tema yang ditetapkan bersama [hartanah yang anda wakili](/ms/studio/agency-representation).

Apa yang anda akan dapat adalah URL yang kelihatan seperti ini:

https://book.wink.travel?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID

Jika anda adalah hartanah, anda ingin pautkan terus ke halaman pendaratan hartanah Wink IBE.

URL itu akan kelihatan seperti ini:

https://book.wink.travel/hotel/hotel-x?client-id=MY_CLIENT_ID&configuration-id=MY_CUSTOMIZATION_ID


:::note
Jika anda tidak sertakan ID klien dan / atau ID konfigurasi, Wink IBE TIDAK akan dimuatkan. Parameter ini diperlukan.
:::

## Contoh: Halaman hartanah

Setiap hartanah di Wink mendapat halaman pendaratan unik mereka sendiri dalam mana-mana laman web. Semua pautan luaran mengarahkan pelancong ke salah satu halaman pendaratan ini.

Halaman pendaratan mengandungi semua maklumat yang diperlukan pelancong untuk membuat pilihan yang bijak sama ada untuk menginap di hartanah ini.
