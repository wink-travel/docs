## Konfigurasi

Selalunya, tidak perlu berkongsi salah satu daripada aplikasi ini secara langsung. Kami telah mencipta pelbagai cara untuk anda berkongsi inventori perjalanan, yang boleh anda temui [Studio Wink](/studio/what-is-studio) atau melalui [WinkLinks](/link-manager/wink-links).

Walau bagaimanapun, jika anda ingin memaut secara manual ke salah satu tapak web ini, pastikan anda menambah ID pelanggan dan ID penyesuaian anda pada URL sebagai parameter.

Anda boleh mencari ID pelanggan anda melalui:

<Steps>
  1. Log masuk sama ada [Kejap Extranet](/extranet/what-is-extranet) atau [Studio Wink](/studio/what-is-studio).
  2. Klik pada ikon profil anda di penjuru kanan sebelah atas dan klik pada [Aplikasi](/developers/applications).
  3. Gunakan aplikasi yang terikat dengan akaun Wink yang ingin anda gunakan.
  4. Salin ID Pelanggan daripada aplikasi itu dan tampalkannya pada URL sebagai parameter permintaan. Contoh: ?client-id=MY\_CLIENT\_ID
</Steps>


### Ekstranet

Anda boleh mencari ID penyesuaian anda sebagai harta dengan:

<Steps>
  1. Log masuk [Kejap Ekstranet](/extranet/what-is-extranet).
  2. Pilih harta yang anda ingin bekerjasama.
  3. Dari bar navigasi utama, klik `Account > Branding`
  4. Salin ID Penyesuaian dan tampalkannya pada URL sebagai parameter permintaan. Contoh: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


### Studio

Anda boleh mencari ID penyesuaian anda sebagai ahli gabungan dengan:

<Steps>
  1. Log masuk [Studio Wink](/studio/what-is-studio).
  2. Pilih akaun yang anda ingin bekerjasama.
  3. Dari bar navigasi utama, klik `Tools > Customization`
  4. Salin ID Penyesuaian daripada penyesuaian yang anda mahu gunakan dan tampalkannya pada URL sebagai parameter permintaan. Contoh: ?configuration-id=MY\_CUSTOMIZATION\_ID
</Steps>


## OTA

Perkara yang anda dapati ialah URL yang kelihatan seperti ini:

https://ota.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Jika anda mahu pergi terus ke halaman hotel, URL akan kelihatan seperti ini:

https://ota.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::nota
Jika anda tidak menyertakan klien dan / atau ID konfigurasi, Wink OTA akan lalai kepada konfigurasi lalai.
:::

## IBE

Jika anda sebuah syarikat pengurusan hartanah, anda ingin memaut ke halaman utama Wink IBE. Halaman utama akan menunjukkan tema yang anda tetapkan bersama-sama dengan [hartanah yang anda wakili](/studio/agency-representation).

Perkara yang anda dapati ialah URL yang kelihatan seperti ini:

https://book.wink.travel?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

Jika anda sebuah hartanah, anda ingin memautkan terus ke halaman pendaratan harta Wink IBE.

URL itu akan kelihatan seperti ini:

https://book.wink.travel/hotel/hotel-x?client-id=MY\_CLIENT\_ID\&configuration-id=MY\_CUSTOMIZATION\_ID

:::nota
Jika anda tidak menyertakan klien dan / atau ID konfigurasi, Wink IBE TIDAK akan dimuatkan. Parameter ini diperlukan.
:::

## Contoh: Halaman harta

Setiap harta di Wink mendapat halaman pendaratan unik mereka sendiri dalam mana-mana tapak web. Semua pautan luaran mengarahkan pengembara ke salah satu halaman pendaratan ini.
