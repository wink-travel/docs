Jika anda sebuah syarikat pelancongan dan anda ingin menyumber luar bahagian pembayaran dan pengeluaran perniagaan anda kepada TripPay, berikut ialah 6 langkah asas untuk bermula:

<Steps>
  1. Buat a [akaun pengguna](/user/register) pada Wink.
  2. Buat akaun dengan [TripPay](https://pay.wink.travel).
  3. Buat satu [Permohonan](/developers/applications) pada TripPay yang membolehkan anda bercakap dengan API kami.
  4. Cipta dan petakan benefisiari anda dengan TripPay. *cth. Bagi kebanyakan, ini bermakna membuat akaun untuk hartanah pada platform anda yang anda mahu TripPay bayar.*
  5. Mengintegrasikan [komponen web pembayaran](/developers/web-components/#payment) ke dalam laman web anda.
  6. Hantar TripPay kepada pengembara `shopping cart` dan memulakan proses pembayaran.
</Steps>


:::petua
Untuk panduan lengkap menggunakan TripPay untuk tapak perjalanan anda, baca panduan tersebut [Mengintegrasikan dengan TripPay](/guides/integrating-with-trip-pay).
:::

## API

Pemaju yang ingin mengurus `Payments`, pergi ke [Pembangun > API > Pembayaran](/developers/apis/#payment-api).
