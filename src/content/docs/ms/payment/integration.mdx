Jika anda sebuah syarikat pelancongan dan ingin mengalih keluar bahagian pembayaran dan pengagihan perniagaan anda kepada TripPay, berikut adalah 6 langkah asas untuk memulakan:

<Steps>
1. Cipta [akaun pengguna](/ms/user/register) di Wink.
2. Cipta akaun dengan [TripPay](https://pay.wink.travel).
3. Cipta [Aplikasi](/ms/developers/applications) di TripPay yang membolehkan anda berkomunikasi dengan API kami.
4. Cipta dan peta penerima manfaat anda dengan TripPay. *contohnya, bagi kebanyakan, itu bermakna mencipta akaun untuk hartanah di platform anda yang anda mahu TripPay bayar.*
5. Integrasikan [komponen web pembayaran](/ms/developers/web-components/#payment) ke dalam laman anda.
6. Hantar `troli beli` pengembara kepada TripPay dan mulakan proses pembayaran.
</Steps>

:::tip
Untuk panduan lengkap menggunakan TripPay untuk laman pelancongan anda, baca panduan [Integrasi dengan TripPay](/ms/guides/integrating-with-trip-pay).
:::

## API

Pembangun yang ingin mengurus `Pembayaran`, pergi ke [Pembangun > API > Pembayaran](/ms/developers/apis/#payment-api).
