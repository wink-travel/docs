Jika anda seorang penyepadu, seperti Wink, dan ingin memetakan penerima luar anda untuk digunakan dengan TripPay, anda boleh melakukannya satu daripada dua cara:

* Pemetaan manual
* Menggunakan [API Pembayaran](/developers/apis/#payment-api) (*Lihat `Account Mappings`*)

:::petua
Melainkan anda hanya berurusan dengan segelintir sifat yang sama, kami mengesyorkan agar anda menggunakan API untuk mengautomasikan langkah ini.
:::

Di bawah, kami memandu anda melalui cara manual untuk memetakan penerima luar. *Ia menggunakan API di bawah hud.*

<Steps>
  1. Log masuk [TripPay](https://pay.wink.travel).
  2. Pilih akaun penyepadu anda.
  3. klik `Settings > Mappings` daripada bar navigasi utama.
  4. Klik pada `Create mapping` butang. Anda akan melihat borang pemetaan.
  5. **Akaun tempatan** Mula menaip nama TripPay sedia ada yang ingin anda petakan.
  6. **Pengecam luaran** Tampalkan ID luaran anda untuk entiti ini kerana ia wujud dalam sistem jauh.
  7. klik `Save` untuk meneruskan. Anda dialihkan semula ke senarai pemetaan anda.
</Steps>


Dengan pemetaan anda disediakan, TripPay kini mengetahui akaun yang akan dikreditkan sebagai penerima tempahan apabila anda merujuk kepada mereka menggunakan pengecam luaran anda.

## API

Pemaju yang ingin mengurus `Payments`, pergi ke [Pembangun > API > Pembayaran](/developers/apis/#payment-api).
