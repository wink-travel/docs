Jika anda seorang integrator, seperti Wink, dan ingin memetakan penerima bayaran luaran anda untuk digunakan dengan TripPay, anda boleh melakukannya dengan dua cara:

- Pemetaan manual
- Menggunakan [Payment API](/ms/developers/apis/#payment-api) (*Lihat `Account Mappings`*)

:::tip
Kecuali anda hanya mengendalikan beberapa hartanah yang sama, kami mengesyorkan anda menggunakan API untuk mengautomasikan langkah ini.
:::

Di bawah, kami akan membimbing anda melalui cara manual memetakan penerima bayaran luaran. *Ia menggunakan API secara tersembunyi.*

<Steps>
1. Log masuk ke [TripPay](https://pay.wink.travel).
2. Pilih akaun integrator anda.
3. Klik `Settings > Mappings` dari bar navigasi utama.
4. Klik butang `Create mapping`. Anda akan melihat borang pemetaan.
5. **Local account** Mula taip nama TripPay sedia ada yang anda ingin peta.
6. **External identifier** Tampal ID luaran anda untuk entiti ini seperti yang wujud dalam sistem jauh.
7. Klik `Save` untuk meneruskan. Anda akan dialihkan kembali ke senarai pemetaan anda.
</Steps>

Dengan pemetaan anda yang telah disediakan, TripPay kini tahu akaun mana yang perlu dikreditkan sebagai penerima manfaat tempahan apabila anda merujuk mereka menggunakan pengecam luaran anda.

## API

Pembangun yang ingin mengurus `Payments`, pergi ke [Developers > API > Payments](/ms/developers/apis/#payment-api).
