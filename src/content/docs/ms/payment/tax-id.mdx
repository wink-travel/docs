Sesetengah kawasan memerlukan pengecam cukai nasional sebelum anda boleh menerima pembayaran. TripPay akan menggesa anda untuk melengkapkan langkah ini sebagai sebahagian daripada penyediaan akaun anda.

## Menghantar ID cukai anda

Apabila diminta, masukkan pengecam cukai nasional anda:

- **Individu di A.S.** — Nombor Keselamatan Sosial (SSN) anda. *Format: 111-11-1111*
- **Syarikat di A.S.** — Nombor Pengenalan Majikan (EIN) anda. *Format: 11-111111*
- **Kawasan lain** — Pengecam nasional setara negara anda.

Selepas memasukkan ID cukai anda, anda akan diminta untuk mengesahkan bahawa maklumat tersebut adalah betul sebelum menghantar.

:::note
Ini adalah langkah sekali sahaja yang diperlukan untuk pematuhan KYC (Kenali Pelanggan Anda). Anda mungkin akan dihubungi untuk dokumentasi tambahan bergantung pada kawasan anda.
:::
