# Seaded

Kui alustate Socialiga, küsime teilt, kuidas soovite, et me teie sisu looksime. Need valikud on teile alati saadaval, nii et saate neid hiljem lihtsalt muuta.

Seadete haldamiseks minge küljeribalt `Preferences`-i.

## Eelistused

Eelistuste sektsioon sisaldab teie põhilisi sisu loomise seadeid:

1. **Eelistatud kohandamine** Valige, millist kohandamist kasutada sisu genereerimisel. Kohandused kontrollivad teema värve, vaikimisi marsruuti, valuutat, keelt ja muud. Kohandusi loote oma kontol aadressil [my.wink.travel](https://my.wink.travel).
2. **Automaatne kinnitamine** Kui see on lubatud, avaldatakse genereeritud postitused automaatselt ilma teie kinnitust nõudmata. Lülitage see sisse, kui olete kindel meie loodava sisu kvaliteedis.
3. **Kinnitaja nimi** Isiku või meeskonna nimi, kes saab kinnitustaotluste e-kirju.
4. **Kinnitaja e-post** E-posti aadress, kuhu saadetakse kinnitustaotlused tulevaste postituste jaoks.

## Sisuloojad

Social pakub palju sisuloojaid, kelle seast valida. Iga looja keskendub ühele kindlale viisile unikaalse sisu loomiseks teie kinnisvara jaoks, kasutades tehisintellekti. Igal sisuloojal on **Kaal** atribuut — suurem kaal suurendab tõenäosust, et see looja valitakse postituste genereerimisel.

Siin on kõik saadaolevad sisuloojad:

| Looja | Kirjeldus |
| -- | -- |
| **Sihtkoht** | Kasutab teie asukohta, et luua kasulikku teksti piirkonna kohta |
| **Toatüüp** | Loob broneeritava postituse, mis tutvustab üht teie toatüüpi AI genereeritud teksti ja teie toatüübi piltidega |
| **Tegevus** | Loob broneeritava postituse, mis tutvustab üht teie tegevust |
| **Lisateenus** | Loob broneeritava postituse, mis tutvustab üht teie lisateenust |
| **Atraktsioon** | Loob broneeritava postituse, mis tutvustab üht teie atraktsiooni |
| **Paik** | Loob broneeritava postituse, mis tutvustab üht teie paika |
| **Koosolekuruum** | Loob broneeritava postituse, mis tutvustab üht teie koosolekuruumi |
| **Restoran** | Loob broneeritava postituse, mis tutvustab üht teie restorani |
| **Spa** | Loob broneeritava postituse, mis tutvustab üht teie spaad |
| **Kinnisvara** | Loob broneeritava postituse, mis tutvustab teie parima hinnaga tuba AI genereeritud teksti ja teie kinnisvara piltidega |
| **Kasutajate tunded** | Kogub külaliste arvustusi üle kogu veebipõhise tundeajastatud postituse loomiseks, mis toob esile, mida külalised teie kinnisvaras armastavad |
| **Naljakas** | Loob kergeid, meelelahutuslikke postitusi, et hoida teie voog kaasahaaravana |

:::tip[Pro nõuanne]
Veenduge, et teil oleks olemas teie valitud sisuloojate jaoks vajalik inventuuritüüp. Näiteks kui valite "Restoran", peab teie kinnisvaral olema vähemalt üks restoran.
:::

## Hääletoon

Hääletoon määrab keele stiili, mida AI kasutab sisu genereerimisel. Valige toonid, mis sobivad teie brändi isikupäraga. Nagu sisuloojate puhul, on igal toonil **Kaal**, mis kontrollib, kui tihti seda kasutatakse.

Siin on saadaolevad toonid:

| Toon | Stiil |
| -- | -- |
| **Neutraalne** | Rahulik, autoriteetne, faktipõhine. Ilma liialdusteta, naljadeta — lihtsalt usaldusväärne, hästi struktureeritud teadmine |
| **Sõbralik** | Soe, ligipääsetav, tekstist läbi naeratav energia koos juhuslike lühendite ja aeg-ajalt emotikonidega |
| **Luksuslik** | Peen, rafineeritud. Sõnad nagu "imeline" ja "kohandatud", mis tekitavad lugejas eksklusiivsuse tunde |
| **Toiduhuviline** | Sensuaalne, suu vett jooksma panev kirjeldus. Toiduporno keel, mis paneb lugejad nälga tundma |
| **Seikluslik** | Julge, kartmatu, avastaja energia. Elavad sensoorsed kirjeldused, mis muudavad kõik kõlavat eepiliselt |
| **Pere** | Terviklik, lastele sobiv elevus. Soe vanemlik energia turvaliste ja lõbusate kogemuste kohta |
| **Romantiline** | Luuleline, unistav, südantliigutav. Päikeseloojangud, pehme keel ja armastus |
| **Humoorikas** | Sarkastiline, kiirete ütlustega, nutikas sõnamäng. Loeb nagu stand-up etteaste |
| **Ökoloogiline** | Siiras ja õiglane. Tähistab rohelisi alternatiive ja jätkusuutlikke valikuid |
| **Kohalik sisering** | Hüper-spetsiifilised kohalikud viited, hoiatab turismilõksude eest, kasutab kohalikku slängi |
| **Kaasaegne** | Kutsuv, pidulik ja kinnitav. Paneb igaühe tundma nähtuna ja hinnatuna |
| **Müstiline** | Eeterlik, vaimne, kosmilised tõed. Õrnad kutseid vaadata sissepoole |
| **Tulevikku vaatav** | Sile, tulevikku suunatud. Armastab termineid nagu "paradigma muutus" ja optimistlikud tulevikunägemused |
| **Eelarveline** | Nutikas, leidlik, uhkelt kokkuhoidlik. Tähistab taskukohaseid leide ja tasuta kogemusi |

## Postituste tüübid

Postituste tüübid määravad genereeritud sisu eesmärgi ja struktuuri. Igal postituse tüübil on **Kaal**, mis kontrollib, kui tihti seda kasutatakse.

| Postituse tüüp | Kirjeldus |
| -- | -- |
| **Tehinguline** | Selge, konkreetne, konversioonile suunatud. Suunab broneeringuid ja otseseid tegevusi faktide, hindade ja järgmiste sammudega |
| **Teadlikkus** | Brändi loomise sisu, mis on mõeldud teie kinnisvara tutvustamiseks ja nähtavuse suurendamiseks potentsiaalsete külaliste seas |
| **Tunded** | Emotsionaalselt kõnetav sisu, mis loob positiivseid seoseid ja tugevdab emotsionaalset sidet teie publikuga |

Võite soovida keskenduda oma turundustegevuses kindlale keelele ja piirkonnale. Öelge meile keeled, mille jaoks soovite sisu luua, ja ajavöönd, mis teid huvitab, ning meie teeme ülejäänu. Postitused avaldatakse teie valitud ajavööndi ajakava järgi.

*Enamik kinnisvarasid kasutab ühte keelt ja vaikimisi oma kohalikku keelt.*
