See leht loetleb kõik webhook sündmused, millele saate Winkis tellida. Seadistamise juhiste saamiseks vaadake [Webhook Integration Guide](/et/guides/webhook-integration) või [Webhooks management page](/et/my-account/webhooks).

## Kuidas tellida

1. Minge oma portaalis menüüsse **Applications > Webhooks**.
2. Looge webhook ja valige sündmused, mida soovite vastu võtta.
3. Wink saadab iga sündmuse toimumisel teie lõpp-punktile HTTP POST päringu.

Täielike seadistamisjuhiste ja parimate tavade jaoks vaadake [Webhook Integration](/et/guides/webhook-integration).

## Konto sündmused

Sündmused, mis on seotud teie TripPay konto elutsükli ja seadistustega.

| Sündmus | Võti | Kirjeldus |
|---|---|---|
| Konto loodud | `account.create` | Uus konto on loodud. |
| Konto ülevaatamisel | `account.under_review` | Konto on ülevaatamisel. |
| Konto uuendatud | `account.updated` | Konto andmed on muutunud. |
| Konto staatuse uuendus | `account.status.updated` | Konto staatus on muutunud (nt kinnitatud, peatatud). |
| Konto aadress uuendatud | `account.address.updated` | Konto aadress on muudetud. |
| Konto detailide uuendus | `account.details.updated` | Konto detailväljad on muutunud. |
| Konto info uuendatud | `account.information.update` | Üldine konto info on uuendatud. |
| Konto haldajad uuendatud | `account.managers.update` | Konto haldajate nimekiri on muutunud. |
| Profiilipilt uuendatud | `account.profile_picture.updated` | Konto profiilipilt on muudetud. |
| Agent uuendatud | `account.agent.updated` | Konto agendi info on muudetud. |
| URL nimi uuendatud | `account.url-name.update` | Konto kohandatud URL on muutunud. |
| Unikaalne ID uuendatud | `account.unique-id.update` | Konto unikaalne identifikaator on muudetud. |

### Konto ülesanded

| Sündmus | Võti | Kirjeldus |
|---|---|---|
| Ülesanne loodud | `account.task.create` | Kontole on lisatud uus ülesanne. |
| Ülesanne uuendatud | `account.task.update` | Konto ülesanne on muudetud. |
| Ülesanne eemaldatud | `account.task.remove` | Konto ülesanne on eemaldatud. |
| Ülesanne lõpetatud | `account.task.complete` | Konto ülesanne on lõpetatud. |
| Maksu-ID ülesanne lõpetatud | `account.task.tax_id.complete` | Maksu-ID kontrolli ülesanne on lõpetatud. |

### Konto hankija

| Sündmus | Võti | Kirjeldus |
|---|---|---|
| Hankija loodud | `account.acquirer.created` | Kontole on lisatud uus hankija. |
| Hankija uuendatud | `account.acquirer.updated` | Konto hankija on muudetud. |
| Hankija eemaldatud | `account.acquirer.removed` | Hankija on kontolt eemaldatud. |

## Broneeringu lepingu sündmused

Sündmused, mis jälgivad broneeringu lepingu elutsüklit TripPay kaudu.

| Sündmus | Võti | Kirjeldus |
|---|---|---|
| Algatuse päring | `booking.contract.initialize.request` | Broneeringu lepingu algatuse päring on esitatud. |
| Algatus lõpetatud | `booking.contract.initialize.complete` | Broneeringu lepingu algatus on lõpetatud. |
| Leping loodud | `booking.contract.created` | Uus broneeringu leping on loodud. |
| Lepingu töötlemine | `booking.contract.processing` | Broneeringu lepingu makset töödeldakse. |
| Lepingu makse ebaõnnestus | `booking.contract.failed` | Broneeringu lepingu makse ebaõnnestus. |
| Leping tühistatud | `booking.contract.cancelled` | Broneeringu leping on tühistatud. |
| Osaline tagasimakse | `booking.contract.refund.partial` | Lepingule on tehtud osaline tagasimakse. |
| Täielik tagasimakse | `booking.contract.refund.full` | Lepingule on tehtud täielik tagasimakse. |
| Tagasimakse päring | `booking.contract.refund.request` | Tagasimakse päring on esitatud. |
| Tagasimakse viga | `booking.contract.refund.error` | Tagasimakse töötlemisel tekkis viga. |
| Väljamakse loodud | `booking.contract.payout.create` | Lepingule on loodud väljamakse. |
| Väljamakse lõpetatud | `booking.contract.payout.complete` | Väljamakse on edukalt lõpetatud. |

## Halduse üksuse sündmused

Sündmused, mis on seotud haldusüksuste (kinnistud, sidusettevõtted, ettevõtted) ja nende haldusega.

### Tellimus

| Sündmus | Võti | Kirjeldus |
|---|---|---|
| Tellimus aktiveeritud | `managing_entity.subscription.provisioned` | Uus tellimus on aktiveeritud. |
| Tellimus muudetud | `managing_entity.subscription.changed` | Tellimuse plaan on muudetud. |
| Tellimuse arve muudetud | `managing_entity.subscription.invoice.changed` | Tellimuse arve on uuendatud. |

### Staatus ja haldus

| Sündmus | Võti | Kirjeldus |
|---|---|---|
| Staatus uuendatud | `managing_entity.status.updated` | Üksuse staatus on muutunud. |
| Üksus kustutatud | `managing_entity.deleted` | Haldusüksus on kustutatud. |
| Agent uuendatud | `managing_entity.agent.updated` | Üksuse agent on muudetud. |
| Agendi e-posti teavitus | `managing_entity.agent.email.notification` | Agendile on saadetud e-posti teavitus. |
| Halduri e-posti teavitus | `managing_entity.manager.email.notification` | Haldurile on saadetud e-posti teavitus. |

### Halduri elutsükkel

| Sündmus | Võti | Kirjeldus |
|---|---|---|
| Haldur lisatud | `managing_entity.manager.added` | Uus haldur on lisatud. |
| Haldur kutsutud | `managing_entity.manager.invited` | Haldurile on saadetud kutse. |
| Haldur eemaldatud | `managing_entity.manager.removed` | Haldur on eemaldatud. |
| Haldur aktsepteeritud | `managing_entity.manager.accepted` | Halduri kutse on aktsepteeritud. |
| Haldur tagasi lükatud | `managing_entity.manager.rejected` | Halduri kutse on tagasi lükatud. |
| Omanik määratud | `managing_entity.manager.owner.set` | Üksuse omandiõigus on üle antud. |
