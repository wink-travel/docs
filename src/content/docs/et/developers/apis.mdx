## Payment API

Makse lõpp-punktid on mõeldud arendajatele, kes soovivad osta reisivarustust. Seda saab teha API kaudu registreeritud reisibüroona või kasutada meie API-d koos PCI nõuetele vastava maksevahendiga kõigi teiste üksuste jaoks.
Payment API pakub lõpp-punkte sidusettevõtetele ja hotellidele broneeringute, analüütika ja rahaliste vahendite jälgimiseks. Nad saavad ka valida saadaolevate vahendite väljavõtmise oma pangakontole.

Loe lisaks
</LinkButton>
