## Makse API

Makseotsingupunktid on mõeldud arendajatele, kes soovivad osta reisivarustust. Seda saab teha API kaudu registreeritud reisibüroos või meie API abil koos meie PCI-ga ühilduva maksevidinaga kõigi teiste üksuste jaoks.
Makseotsingu API pakub lõpp-punkte sidusettevõtetele ja hotellidele broneeringute, analüütika ja rahaliste vahendite kättesaadavuse jälgimiseks. Samuti saavad nad valida saadaolevate vahendite oma pangakontole kandmise.

  Lisateave
</LinkButton>
