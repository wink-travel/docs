## Maksete API

Maksete lõpp-punktid on mõeldud arendajatele, kes soovivad osta reisivarustust. Seda saab teha API kaudu registreeritud reisibüroona või kasutada meie API-d koos PCI nõuetele vastava maksevahendiga kõigi teiste üksuste jaoks.
Maksete API pakub lõpp-punkte sidusettevõtetele ja hotellidele broneeringute, analüütika ja rahaliste vahendite jälgimiseks. Nad saavad ka valida saadaolevate vahendite väljavõtmise oma pangakontole.

</LinkButton>
