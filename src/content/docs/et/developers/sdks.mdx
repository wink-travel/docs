Kiirendage turule jõudmise aega, kasutades üht meie ametlikult toetatud keele tarkvaraarenduskomplektidest.
Kui me praegu teie lemmikkeelt ei toeta, minge API-le, mida soovite kasutada, ja laadige alla OpenAPI spetsifikatsioon, mida me iga API lehe ülaosas pakume. Saate genereerida oma SDK kas meie avalike SDK hoidlate forkimise teel või kasutades mõnda OpenAPI generaatorit.

## Java

See teek on kompileeritud Java 24 jaoks. See on konfigureeritud kasutama reaktiivset WebClienti ja optimeeritud kasutamiseks Spring Frameworkiga. Kuid Spring ei ole nõutav, kuid see annab teile OAuth2 toe kohe välja pakendist.
