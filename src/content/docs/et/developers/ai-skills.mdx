Tehisintellekti oskused on kaasaskantavad juhendpaketid, mis õpetavad koodiagente töötama Winkiga. Selle asemel, et tugineda treeningandmetele või aegunud dokumentatsioonile, juhib iga oskus agenti kasutama [Wink MCP serverit](/et/developers/mcp) kui elavat tõeallikat API lepingute ja platvormi dokumentide jaoks.

Oskused töötavad iga agendikliendiga, mis neid toetab — Claude Code, Codex, Cursor ja teised.

## Saadaval olevad oskused

### wink-api-builder

Rakenda, vaata üle või silu Wink API integratsioone. Oskus juhendab agenti läbi struktureeritud töövoo:

1. Otsi õiget API toimingut `api_search` abil
2. Ava MCP serverist elavad päringu/vastuse lepingud
3. Võta autentimisseadistus dokumentidest
4. Kirjuta kood kinnitatud lepingu alusel
5. Märgi üles kõik lahknevused olemasoleva koodi ja praeguse API vahel

Kasulik:
- Uute kliendimeetodite loomisel mis tahes Wink API jaoks
- Lepingute lahknevuste silumiseks (nt päring, mis tagastab HTTP 400)
- Olemasolevate integratsioonide ülevaatamiseks elava spetsifikatsiooni alusel

## Paigaldus

Kopeeri oskuse kaust oma agendi oskuste kataloogi.

:::tip
Wink MCP server peab olema ühendatud samas keskkonnas. Vaata [MCP Server](/et/developers/mcp) seadistamise juhiseid.
:::

**Claude Code**

```bash
# repopõhine
cp -r skills/wink-api-builder .claude/skills/

# või kasutajapõhine
cp -r skills/wink-api-builder ~/.claude/skills/
```

Käivita käsuga:
```
/wink-api-builder add a TypeScript client for the Wink affiliate create flow
```

**Codex**

```bash
# repopõhine
cp -r skills/wink-api-builder .agents/skills/

# või kasutajapõhine
cp -r skills/wink-api-builder $CODEX_HOME/skills/
```

Käivita käsuga:
```
Use $wink-api-builder to add a TypeScript client for the Wink affiliate create flow.
```

**Teised agendid**

Paiguta oskuse kaust sinna, kust su agent oskusi laadib. Sisenemispunkt on `SKILL.md`.
