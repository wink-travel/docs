Wink integreerub juhtivate kanalihaldurite, kinnisvarahaldussüsteemide (PMS) ja tsentraalsete broneerimissüsteemidega (CRS), et hoida teie hinnad ja saadavus kõigis müügikanalites sünkroonis.

## Kuidas see töötab

Kanalihalduri integratsioonid võimaldavad kahepoolset andmevoogu teie kinnisvarahaldussüsteemi ja Winki vahel:

1. **Hindade edastamine** — Teie kanalihaldur saadab hinnauuendused Winkile. Kui muudate oma PMS-is hindu, kajastuvad need muudatused automaatselt Winkis.
2. **Saadavuse sünkroonimine** — Teie kanalihaldur saadab saadavuse uuendused Winkile. Kui toad broneeritakse või vabastatakse teie PMS-is, uuendab Wink reaalajas oma inventari.
3. **Broneeringute teavitused** — Kui Winkis tehakse broneering, teavitatakse teie kanalihaldurit, et broneering ilmuks teie PMS-is.

## Toetatud kanalihaldurid

Wink integreerub praegu järgmiste süsteemidega:

| Kanalihaldur | Kirjeldus |
|---|---|
| **Channex** | Globaalne levituse ja kanalihalduse platvorm. |
| **TravelClick** | Hotellilahendused ja kanalihaldus (Amadeus). |
| **Omnibees** | Hotellide levituse ja tulude haldus. |
| **D-Edge** | Majutustehnoloogia ja tsentraalse broneerimissüsteemi lahendused. |
| **Allotz** | Inventari jaotamine ja kanalihaldus. |
| **Ezee / Yanolja** | Pilvepõhine PMS ja kanalihaldus, keskendudes Aasia turule. |
| **RateGain** | Hinnateadlikkus ja levitustehnoloogia. |
| **RateTiger** | Reaalajas hindade ja saadavuse levitamine. |
| **CloudBeds** | Pilvepõhine PMS, kanalihaldur ja broneerimismootor. |
| **SiteMinder** | Juhtiv hotelli kanalihaldur ja levituse platvorm. |
| **Exely** | Hindade ja saadavuse sünkroonimine hotellidele. |

:::note
Kui teie kanalihaldur ei ole nimekirjas, võtke ühendust Wink meeskonnaga. Uued integratsioonid saab lisada vastavalt OTA (Open Travel Alliance) protokolli standarditele, mida Wink toetab.
:::

## Arendajatele

Kui ehitate kanalihalduri või PMS integratsiooni Winkiga, kasutage Channel Manager API lõpp-punkte:

- **Hinnauuendused** — Saada hinnamuudatused Winkile.
- **Saadavuse uuendused** — Saada saadavuse muudatused Winkile.
- **Broneeringute päring** — Tõmba broneeringud Winkist oma süsteemi.

Channel Manager API järgib OTA protokolli standardeid (SOAP/XML), et tagada ühilduvus olemasolevate majutussüsteemidega.

## Webhook teavitused

Võite tellida kanalihalduri webhook sündmusi, et saada reaalajas teavitusi:

- `channel-manager.update.rate` — Saadud hinnauuendus.
- `channel-manager.update.availability` — Saadud saadavuse uuendus.
- `channel-manager.update` — Üldine kanalihalduri uuendus.

Vaadake üksikasju [Webhook Events Catalog](/et/developers/webhook-events#channel-manager-events).
