Iga päev ei ole hinnastatud ühtemoodi. Üritused, pühad, madalhooajad ja viimasel hetkel tekkinud vabad ajad nõuavad erinevat hinnastamist. See juhend näitab, kuidas kasutada erihindu ja hinnaplaani piiranguid nende olukordade haldamiseks.

## Ürituste hinnastamine

Kui konverents, festival või kohalik üritus suurendab nõudlust, soovite nende kuupäevade jaoks hindu tõsta.

<Steps>
1. Minge oma Extraneti menüüs `Monetize > Special Rates`.
2. Klõpsake `Create a new special rate`.
3. Valige peamine hind ja tubatüüp, mida soovite muuta.
4. Määrake kuupäevavahemik, mis katab ürituse perioodi.
5. Sisestage kohandatud (kõrgemad) päevahinnad.
6. Klõpsake `Save`.
</Steps>

Erihind asendab peamist hinda ainult nende kuupäevade jaoks. Teie tavahinnad jäävad väljaspool ürituse perioodi muutumatuks.

## Hooajalised kohandused

Pikemate perioodide (kõrghooaeg, vahehooaeg, madalhooaeg) jaoks saate kas:

- **Kasutada erihindu** — luua iga hooaja jaoks kohandatud hinnad.
- **Uuendada peamisi hindu** — muuta baasihindu otse nende perioodide jaoks.

Erihinnad sobivad paremini ajutisteks muudatusteks, samas kui peamiste hindade uuendamine on mõistlik prognoositavate aastaste mustrite puhul.

## Viimasel hetkel hinnad

Müümata tubade täitmiseks vahetult enne saabumiskuupäeva:

<Steps>
1. Looge uus **hinnaplaan** koos ettetellimise piiranguga (nt broneering peab toimuma mitte rohkem kui 3 päeva ette).
2. Siduge see **erihinnaga**, mis pakub soodustust sihtkuupäevadel.
3. See hinnaplaan kuvatakse ainult reisijatele, kes broneerivad viimasel hetkel.
</Steps>

## Mustad kuupäevad

Teatud kuupäevade blokeerimiseks kindla hinnaplaani jaoks (nt kampaaniahinnad ei kehti kõrghooajal):

<Steps>
1. Minge `Monetize > Rate Plans` ja valige hinnaplaan.
2. Avage jaotis **Restrictions**.
3. Lisage **Sell date** piirang, et piirata, millal hind on broneeritav.
4. Või kasutage **Stay date** piiranguid, et piirata, millal hind kehtib.
</Steps>

Näiteks saate piirata oma "Early Bird" hinnaplaani nii, et see ei ole saadaval detsembris (kõrghooajal), samal ajal kui teie tavahinnaplaan jääb avatuks.

## Piirangute kombineerimine

Piiranguid saab kombineerida täpseks kontrolliks:

| Piirang | Mida see kontrollib |
| -- | -- |
| **Sell dates** | Millal hind on broneeritav |
| **Stay dates** | Millistel ööbimiskuupäevadel hind kehtib |
| **Available days** | Millistel nädalapäevadel hind kuvatakse |
| **Arrival days** | Millistel päevadel külaline peab saabuma |
| **Departure days** | Millistel päevadel külaline peab lahkuma |
| **Required days** | Külaline peab viibima vähemalt ühel neist päevadest |
| **Advance booking** | Kui palju ette broneering peab olema tehtud |
| **Length of stay** | Minimaalne ja maksimaalne ööde arv |
