Mõned piirkonnad nõuavad riiklikku maksutuvastuse numbrit enne, kui saate makseid vastu võtta. TripPay palub teil selle sammu täita oma konto seadistamise osana.

## Maksu ID esitamine

Kui seda küsitakse, sisestage oma riiklik maksutuvastuse number:

- **Isikud Ameerika Ühendriikides** — Teie sotsiaalkindlustuse number (SSN). *Vorming: 111-11-1111*
- **Ettevõtted Ameerika Ühendriikides** — Teie tööandja identifitseerimisnumber (EIN). *Vorming: 11-111111*
- **Muud piirkonnad** — Teie riigi vastav riiklik identifikaator.

Pärast maksu ID sisestamist palutakse teil kinnitada, et teave on õige, enne selle esitamist.

:::note
See on ühekordne samm, mis on vajalik KYC (Tunne oma klienti) nõuetele vastamiseks. Sõltuvalt teie piirkonnast võidakse teilt nõuda täiendavaid dokumente.
:::
