Kui oled reisifirma ja soovid oma äri maksete ja väljamaksete osa usaldada TripPayle, siis siin on 6 põhilist sammu alustamiseks:

<Steps>
1. Loo Winkis [kasutajakonto](/et/user/register).
2. Loo konto [TripPay](https://pay.wink.travel) juures.
3. Loo TripPays [rakendus](/et/developers/applications), mis võimaldab sul meie API-ga suhelda.
4. Loo ja kaardista oma kasusaajad TripPayga. *nt enamiku jaoks tähendab see kontode loomist platvormil olevatele majutusasutustele, kellele soovid, et TripPay maksaks.*
5. Integreeri oma saidile [maksete veebikomponent](/et/developers/web-components/#payment).
6. Saada TripPayle reisija `ostukorv` ja alusta makseprotsessi.
</Steps>

:::tip
TripPay kasutamise täieliku juhendi oma reisisaidil leiad juhendist [Integrating with TripPay](/et/guides/integrating-with-trip-pay).
:::

## API

Arendajad, kes soovivad hallata `makseid`, minge [Developers > API > Payments](/et/developers/apis/#payment-api).
