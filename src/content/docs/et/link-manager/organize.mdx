WinkLinksi siltide haldamiseks liigu põhimenüüs `WinkLinks` ja klõpsa vahekaardil `Tags`.

Kui su sisu kasvab, tahad hakata seda organiseerima. Selleks lõime `sildid`. Saad grupeerida sisu nii paljude siltidega kui soovid.

## Loo silt

<Steps>
1. Loo uus silt, klõpsates nupul `Add a new tag`.
2. Sisesta oma uue sildi nimi. *nt Top kohvikud Berliinis*
3. Klõpsa lingil `Save`.
</Steps>

## Uuenda silti

<Steps>
1. Alusta, klõpsates lingil `Edit` selle sildi kõrval, mida soovid muuta.
2. Uuenda oma sildi nime. *nt Parimad kohvikud Berliinis*
3. Klõpsa lingil `Save`.
</Steps>

## Eemalda silt

:::note
Sildi eemaldamisel eemaldad selle ka kogu sellega seotud sisust.
:::

<Steps>
1. Klõpsa lingil `Remove` selle sildi kõrval, mida soovid eemaldada.
</Steps>

## API

Arendajad, kes soovivad hallata `WinkLinks`i, saavad minna [Developers > API > WinkLinks](/et/developers/apis/#winklinks-api).
