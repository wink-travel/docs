Kui soovite oma Winki kasutajakonto kasutajaprofiili värskendada, toimige järgmiselt.

<Steps>
  1. Peate olema sisse logitud ühele meie veebisaitidest.
  2. Klõpsake oma profiiliikoonil paremas ülanurgas ja kuvatakse menüü.
  3. Klõpsake nuppu `Profile` link.
  4. Järgige juhiseid oma seadme värskendamiseks `e-mail`, `first` ja `last name`.
  5. Samuti saate üles laadida uue `profile picture` iseendast.
  6. Klõps `Save`.
</Steps>


:::note
Teie uus profiilipilt kuvatakse järgmisel korral, kui logite Winkist täielikult välja.
:::
