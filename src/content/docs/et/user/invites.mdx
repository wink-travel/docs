Wink kasutaja võib valida, et annate teile õiguse hallata ühte nende kinnisvara-, sidusettevõtte- või reisibüroo kontodest. Kui see juhtub, saate kutse taotlusega e-kirja.

import { Steps } from '@astrojs/starlight/components';

Oma praeguste Wink kutsete nägemiseks järgige neid samme:

<Steps>
1. Peate olema sisse logitud ühele meie veebisaitidest.
2. Klõpsake oma profiiliikooni paremas ülanurgas ja ilmub menüü.
3. Klõpsake lingil `Invites`.
4. Suhelge oma kutsetega, kas neid `Nõustudes` või `Keeldudes`.
</Steps>
