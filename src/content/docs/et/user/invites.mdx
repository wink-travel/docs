Winki kasutaja saab valida, kas lubada teil hallata ühte oma kinnisvara-, partner- või reisibüroo kontodest. Sellisel juhul saate kutsetaotlusega e-kirja.

import { Steps } from '@astrojs/starlight/components';

Oma praeguste Wink-kutsete nägemiseks toimige järgmiselt.

<Steps>
  1. Peate olema sisse logitud ühele meie veebisaitidest.
  2. Klõpsake oma profiiliikoonil paremas ülanurgas ja kuvatakse menüü.
  3. Klõpsake nuppu `Invites` link.
  4. Suhtle oma kutsetega ühel või teisel viisil `Accepting` või `Rejecting` neid.
</Steps>
