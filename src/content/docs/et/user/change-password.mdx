Kui soovite oma Winki kasutajakonto parooli muuta, toimige järgmiselt.

## Parooli uuendamine

<Steps>
  1. Peate olema sisse logitud ühele meie veebisaitidest.
  2. Klõpsake oma profiiliikoonil paremas ülanurgas ja kuvatakse menüü.
  3. Klõpsake nuppu `Change password` link.
  4. Parooli muutmiseks järgige juhiseid.
  5. Klõps `Save`.
</Steps>


Järgmisel sisselogimisel vajate oma uut parooli.

## Unustasin parooli

Mõnikord on vaja kasutaja parool lähtestada.

Kasutajaparooli lähtestamiseks toimige järgmiselt.

<Steps>
  1. Klõpsake nuppu `Forgot password` link sisselogimislehelt.
  2. Sisestage lähtestatava konto kasutajanimi/e-posti aadress.
  3. Klõps `Forgot password` nupp. *Teid suunatakse parooli lähtestamise vormile.*
  4. Kontrolli oma e-posti.
</Steps>

  <CloudinaryImage width="350" height="354" resize="scale" quality="best" alt="Unustasin parooli" src="academy/forgot-password" errorImage="noimage_opaque_nyrtl0.png" />

  <figcaption>*Unustasin parooli vormi*</figcaption>

<Steps>
  1. Kleepige e-kirjaga saadud kood koos uue parooliga uude vormi.
  2. Kinnitage oma uus parool.
  3. Klõps `Reset password` nupp. *Teid suunatakse tagasi sisselogimislehele.*
</Steps>


<figure>
  <CloudinaryImage width="350" height="467" resize="scale" quality="best" alt="Parooli lähtestamine" src="academy/reset-password" errorImage="noimage_opaque_nyrtl0.png" />

</figure>


Teie parool on edukalt lähtestatud. Saate Winki sisse logida oma uute sisselogimisandmetega.
