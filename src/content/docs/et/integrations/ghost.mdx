Ghosti sisseehitatud **Code Injection** funktsioon võimaldab teil lisada skripte oma väljaande globaalsesse `<head>` ja `</body>`. Kõik Ghosti paketid toetavad koodi manustamist, muutes Wink komponentide lisamise mis tahes lehele või postitusele lihtsaks.

## Laadige Wink skriptid kogu saidil

<Steps>
1. Ghost Admin paneelis minge **Settings → Code Injection**.
2. Lisage **Site Header** lahtrisse stiilileht:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
3. Lisage **Site Footer** lahtrisse skript ja rakenduse laadija (asendage `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
4. Klõpsake **Save**.
</Steps>

:::tip
Teie Client ID on saadaval teie Wink kontol jaotises [Applications](/et/portal/settings/applications).
:::

## Lisage komponent postitusele või lehele

Ghosti redaktor toetab **HTML kaarti**, mis võimaldab teil sisestada toorest HTML-i ükskõik kuhu oma sisus.

<Steps>
1. Avage postitus või leht Ghosti redaktoris.
2. Tippige `/`, et avada kaardivalija, ja valige **HTML**.
3. Kleepige komponendi HTML, näiteks:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
4. Klõpsake kaardist väljaspool, et eelvaadet näha, seejärel avaldage.
</Steps>

## Lehekohane koodi manustamine

Komponendi jaoks, mis peaks ilmuma ainult ühel lehel, toetab Ghost ka lehekohast koodi manustamist:

<Steps>
1. Avage postituse või lehe seaded (hammasratta ikoon redaktoris).
2. Kerige alla **Code Injection** sektsiooni.
3. Lisage komponendi silt **Footer** väljale (globaalne app-loader peab olema ainult saidi jaluses).
4. Salvestage.
</Steps>

## Saadaval komponendid

| Komponent | HTML silt | Eesmärk |
|---|---|---|
| Sisu laadija | `<wink-content-loader>` | Kuvab hotellikaarte, võrgustikke, kaarte |
| Otsing | `<wink-lookup>` | Sihtkohtade otsinguriba |
| Otsingunupp | `<wink-search-button>` | Avab marsruudi valija |
| Konto nupp | `<wink-account-button>` | Sisselogimine / kasutajamenüü |
| Marsruudi nupp | `<wink-itinerary-button>` | Kuvab praeguse marsruudi |
| Marsruudi valija | `<wink-itinerary-picker>` | Täielik marsruudi vorm |
| Ostukorvi nupp | `<wink-shopping-cart-button>` | Ostukorvi kokkuvõtte nupp |

Vaadake täielikku atribuudi viidet [Web Components](/et/integrations/web-components) lehel.
