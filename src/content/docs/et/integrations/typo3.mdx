**Wink TYPO3 Extension** integreerib Wink reisinventari TYPO3 13 natiivsete sisuelementidena. Toimetajad lisavad Wink komponendid lehtedele, kasutades standardset TYPO3 lehetoimetajat — mallimuudatusi pole vaja.

<LinkCard
  title="wink-travel/wink-typo3-extension GitHubis"
  href="https://github.com/wink-travel/wink-typo3-extension"
  description="Allikakood, arendaja seadistusjuhend ja panustamise juhised."
/>

## Nõuded

- TYPO3 13.x
- PHP 8.2+
- Composer

## Paigaldus

<Steps>
1. Paigalda Composeriga:
   ```bash
   composer require wink2travel/typo3-wink
   ```
2. Aktiveeri laiendus:
   ```bash
   vendor/bin/typo3 extension:activate wink
   ```
3. TYPO3 tagapoolses liideses mine **Admin Tools → Settings → Extension Configuration → wink**.
4. Sisesta oma **Client ID**, **Client Secret** ja vali keskkond. Salvesta.
   Oma mandaadid saad kätte [Applications](/et/portal/settings/applications) lehelt.
5. Lisa staatiline TypoScript mall oma juurmalli:
   **Template → Edit → Includes → Available Items → Wink Travel**.
</Steps>

## Lisa sisuelement lehele

<Steps>
1. Ava leht **Page** moodulis.
2. Klõpsa sisuala **+** nupul, et lisada uus element.
3. Vali üks **Wink Travel** sisuelementidest:
   - **Wink Content** — hotellikaarte, järjestatud ruudustikke, kaarte
   - **Wink Lookup** — sihtkoha otsinguriba
   - **Wink Search** — marsruudi valiku nupp
   - **Wink Account** — sisselogimise nupp
   - **Wink Itinerary** — marsruudi oleku nupp
   - **Wink Cart** — ostukorvi nupp
4. **Wink Content** puhul kasuta FlexForm välju, et valida **Layout Type** ja sisesta **Layout ID**.
5. Salvesta ja avalda leht.
</Steps>

## Saadaval olevad sisuelemendid

| Element | Eesmärk |
|---|---|
| **Wink Content** | Hotellikaarte, järjestatud ruudustikke, kaarte ja muud |
| **Wink Lookup** | Sihtkoha otsinguriba |
| **Wink Search** | Ikooninupp, mis avab marsruudi valiku |
| **Wink Account** | Sisselogimise / kasutajakonto nupp |
| **Wink Itinerary** | Marsruudi nupp, mis näitab praegust reisi |
| **Wink Cart** | Ostukorvi kokkuvõtte nupp |
