**Wink Drupal moodul** lisab Wink reisiinventari komponendid teie Drupal saidile natiivsete plokkidena. Toimetajad saavad neid paigutada ükskõik kuhu paigutuses, kasutades standardset plokisüsteemi.

<LinkCard
  title="wink-travel/wink-drupal-module GitHubis"
  href="https://github.com/wink-travel/wink-drupal-module"
  description="Allikakood, arendaja seadistusjuhend ja panustamise juhised."
/>

## Nõuded

- Drupal 10 või 11
- PHP 8.1+
- Composer

## Paigaldamine

<Steps>
1. Paigaldage Composeriga:
   ```bash
   composer require drupal/wink
   ```
2. Lülitage moodul sisse:
   ```bash
   drush en wink -y
   ```
3. Minge **Admin → Configuration → Web Services → Wink Settings** (`/admin/config/wink`).
4. Sisestage oma **Client ID** ja **Client Secret**, valige oma keskkond ning klõpsake **Save**.
   Oma mandaadid saate kätte [Applications](/et/portal/settings/applications) lehelt.
</Steps>

## Ploki lisamine lehele

<Steps>
1. Minge **Admin → Structure → Block Layout**.
2. Klõpsake piirkonnas, kuhu soovite Wink sisu lisada, nuppu **Place block**.
3. Otsige **Wink**, et leida saadaolevad plokid:
   - **Wink Content** — kuvab hotelli kaardi, ruudustiku või kaardi
   - **Wink Lookup** — sihtkoha otsinguriba
   - **Wink Search** — marsruudi valiku nupp
   - **Wink Account** — sisselogimisnupp
   - **Wink Itinerary** — marsruudi oleku nupp
   - **Wink Cart** — ostukorvi nupp
4. **Wink Content** puhul seadistage ploki sätetes **Layout Type** ja **Layout ID**.
5. Salvestage ploki konfiguratsioon.
</Steps>

## Saadaval plokid

| Plokk | Eesmärk |
|---|---|
| **Wink Content** | Kuvab hotellikaarte, järjestatud ruudustikke, kaarte ja muud |
| **Wink Lookup** | Otsinguriba sihtkohtade ja hotellide jaoks |
| **Wink Search** | Ikooninupp, mis avab marsruudi valiku |
| **Wink Account** | Sisselogimise / kasutajakonto nupp |
| **Wink Itinerary** | Marsruudi nupp, mis näitab praegust reisi |
| **Wink Cart** | Ostukorvi kokkuvõtte nupp |
