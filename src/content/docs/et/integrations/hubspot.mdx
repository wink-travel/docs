HubSpot CMS toetab kohandatud HTML-mooduleid ja saidiülest päise/jaluse koodi, mis teeb Wink komponentide manustamise turunduslehtedele ja maandumislehtedele lihtsaks.

:::note
Saiditasandi koodi manustamiseks on vaja juurdepääsu **HubSpot CMS Hub** (Starter või kõrgem). Kohandatud HTML-moodulid on saadaval kõigis CMS tasemetes.
:::

## Laadige Wink skriptid saidiülest

<Steps>
1. HubSpotis minge **Marketing → Files and Templates → Design Tools**.
2. Avage oma aktiivse **teema** `main.css` või kasutage globaalseid siseseadeid.
3. Või minge **Settings → Website → Pages → Custom Code**.
4. Lisage **Head HTML** väljale stiilileht:
   ```html
   <link rel="stylesheet" href="https://elements.wink.travel/styles.css">
   ```
5. Lisage **Footer HTML** väljale skript ja rakenduse laadija (asendage `YOUR_CLIENT_ID`):
   ```html
   <script src="https://elements.wink.travel/elements.js" type="module" defer></script>
   <wink-app-loader client-id="YOUR_CLIENT_ID"></wink-app-loader>
   ```
6. Salvestage ja avaldage.
</Steps>

:::tip
Teie kliendi ID on saadaval teie Wink kontol jaotises [Applications](/et/portal/settings/applications).
:::

## Lisage komponent kohandatud HTML-mooduli kaudu

<Steps>
1. Design Toolsis looge uus **moodul** tüüpi **Rich Text** või **HTML**.
2. Kleepige mooduli vaikesisusse komponendi HTML:
   ```html
   <wink-content-loader
     layout="HOTEL"
     id="YOUR_LAYOUT_ID"
   ></wink-content-loader>
   ```
3. Salvestage moodul ja lohistage see mis tahes lehe mallile või lehele, kasutades lehe redaktorit.
</Steps>

## Lisage komponent otse lehe redaktoris

Kui te ei soovi moodulit luua, saate kasutada **Custom HTML** sektsiooni lohistamis- ja kukutamisredaktoris:

<Steps>
1. Avage leht HubSpoti lehe redaktoris.
2. Klõpsake **Add** ja valige **Custom HTML** või **Embed** sektsioon.
3. Kleepige komponendi silt HTML väljale.
4. Klõpsake **Apply** ja avaldage leht.
</Steps>

## Saadaval olevad komponendid

| Komponent | HTML silt | Eesmärk |
|---|---|---|
| Sisu laadija | `<wink-content-loader>` | Kuvab hotellikaarte, võrgustikke, kaarte |
| Otsing | `<wink-lookup>` | Sihtkohtade otsinguriba |
| Otsingunupp | `<wink-search-button>` | Avab marsruudi valija |
| Konto nupp | `<wink-account-button>` | Sisselogimine / kasutajamenüü |
| Marsruudi nupp | `<wink-itinerary-button>` | Kuvab praeguse marsruudi |
| Marsruudi valija | `<wink-itinerary-picker>` | Täielik marsruudi vorm |
| Ostukorvi nupp | `<wink-shopping-cart-button>` | Ostukorvi kokkuvõtte nupp |

Vaadake täielikku atribuudi viidet [Web Components](/et/integrations/web-components) lehel.
