Wink kasutaja võib valida, et annate teile õiguse hallata ühte nende kinnisvara-, sidusettevõtte- või reisibürookontot. Kui see juhtub, saate kutse taotlusega e-kirja.

import { Steps } from '@astrojs/starlight/components';

Oma praeguste Wink kutsete nägemiseks järgige neid samme:

<Steps>
 1. Logige sisse aadressil [my.wink.travel](https://my.wink.travel).
 2. Klõpsake vasakpoolses külgribas valikul `Invites`.
 3. Näete kõigi praeguste kutsete nimekirja.
 4. See jaotis kuvab kõik teie taotlused.
 5. Valige, kas `Accept` või `Reject` kutse.
 6. Kui kutsest on nõusolek antud, kuvatakse konto jaotises `Accounts`.
</Steps>
